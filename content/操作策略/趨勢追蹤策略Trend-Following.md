# 趨勢追蹤策略 Trend Following

> 不預測市場，只跟隨趨勢——小虧多次、偶爾大賺，讓利潤奔跑、及早停損。

## 核心概念

趨勢追蹤（Trend Following）是所有順勢交易策略的母框架，核心信念是**市場趨勢有持續性，且大趨勢的利潤足以覆蓋多次小虧損**。

與「預測市場」完全不同：趨勢追蹤者不猜頭摸底，而是在趨勢已經啟動後才進場，在趨勢結束後才出場。這種方法的盈利分布呈現典型的**偏態分布**：少數大賺的交易貢獻絕大多數利潤，勝率通常只有30-40%。

### 為什麼趨勢追蹤有效？

1. **市場並非完美有效**：資訊的傳遞、理解和回應總有延遲，這讓價格沿著某個方向前進形成可持續的趨勢
2. **群體心理的正回饋**：價格上漲吸引更多買盤進場，形成良性循環，趨勢追蹤者正是抓住這種集體行為
3. **風險管理是護城河**：多數人知道規則卻無法堅持，機械化執行本身就是競爭優勢
4. **市場永遠會有趨勢**：極端行情（2008金融危機、2014原油崩盤、2020疫情）是趨勢追蹤的超級收割期

### 趨勢追蹤的盈虧結構

| 特徵 | 數據 |
|------|------|
| 典型勝率 | 30-40% |
| 獲利來源 | 少數大趨勢（5-10%的交易貢獻80%+利潤） |
| 單筆最大虧損 | 帳戶1-2%（嚴格風控） |
| 最大敵人 | 盤整市場（假突破反覆打停損） |
| 心理挑戰 | 連續小虧 + 大賺時忍住不跑 |

## 實戰應用

### 三種趨勢追蹤進場方法

**1. 突破策略（唐奇安通道/海龜系統）**
- 突破20日最高價做多、跌破20日最低價做空
- 確認：帶量突破、收盤站穩、3天確認
- 適合：趨勢明確的市場

**2. 均線系統策略**
- 多頭排列做多、空頭排列做空
- 葛蘭碧八大法則找進出場時機
- 價格回檔碰中短期均線獲撐是加碼良機
- 適合：中長期波段操作

**3. 動量追蹤策略**
- 用ROC篩選加速股，RS Rating排序強弱
- 漲勢中ROC持續上升作為進場或加碼依據
- 讓獲利自由發展但設好出場機制
- 適合：高成長產業爆發期

### 風險管理核心

**1%至2%法則**：單筆交易最大虧損不超過帳戶總資金的1-2%。10萬資金，單筆限虧1000-2000元。

**ATR動態停損**：根據市場波動性設定停損距離，波動大停損大、波動小停損小。比固定百分比更合理。

**金字塔加碼**：在獲利部位加碼，每上漲0.5倍ATR加碼一單位，最多4次加碼。絕不在虧損部位攤平。

**移動停利**：趨勢持續時將停損位跟隨價格上移，確保利潤最大化。核心心法：讓利潤奔跑。

### 趨勢追蹤的心理修煉

Curtis Faith（海龜最成功交易員）認為交易成功80%取決於心理素質：

1. **接受虧損為交易成本**：有正期望值的系統必然經歷虧損序列，是統計噪音不是系統失效
2. **專注過程而非結果**：交易如工廠生產流程，不因單一產品不合格改變整個流程
3. **機械化執行**：即使短期表現不佳，只要系統有統計優勢就持續遵循
4. **量化虧損預期**：量化drawdown幅度和持續時間，心理壓力大幅減輕
5. **交易日誌**：記錄每次交易的決策、情緒與結果，回顧改進模式

## 注意事項

### 八大失敗原因與應對

1. **盤整市場頻繁交易** → 布林窄縮或ADX<20辨識盤整，減交易等趨勢現形
2. **不設停損或停損過大** → 執行2%法則，進場前定技術停損
3. **過度槓桿與資金配置不當** → 控單筆風險，分散組合，慎用槓桿
4. **趨勢反轉未能及時出場** → 留意K線反轉、MACD背離，移動停利分批鎖盈
5. **心理素質不足** → 建量化系統嚴守，寫交易日誌
6. **追漲殺跌變盲目追高** → 辨真假突破（看量與回測），避乖離過高入場
7. **指標鈍化與誤用** → 多指標綜合（趨勢+動量），知各工具適用情境
8. **缺乏宏觀經濟與產業認知** → 融基本分析，技術與基本互補

### 歷史失敗案例

- **1987年10月股災**：美聯儲突然降息，利率期貨瞬間掉頭，海龜多頭倉位一日虧損20-40%。Richard Dennis本人基金被迫清盤。
- **1980年代後期**：商品和貨幣陷入區間波動，海龜交易員數月連續虧損。
- **盤整期**：勝率30-40%意味著在沒有趨勢的市況中，系統不斷發出假突破訊號又被打停損。

### 台股適用性

- **台積電(2330)實戰**：2020疫情後晶片需求爆發，破歷史高+60日線多頭排列進場，移動停利獲利豐厚。2022庫存壓力轉跌時出場。
- **20日突破太短**：台股淺碟市場假突破多，建議55日系統或加帶量確認
- **漲跌幅限制影響**：10%漲停限制，突破後可能買不到
- **手續費侵蝕**：系統一進出頻繁，建議以系統二為主
