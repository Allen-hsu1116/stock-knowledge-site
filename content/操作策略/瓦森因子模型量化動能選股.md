# 瓦森因子模型量化動能選股

> 鎖定高速營收成長企業，要求股價與成交量同步展現強勢——「營收動能 + 股價動能」雙重確認的機械式選股系統

## 核心概念

賈布利·瓦森（Gabriel Watson）是美國知名的成長動能型投資組合經理人，早年任職於摩根添惠與威廉歐尼爾公司。自1998年加入黑玫瑰資本管理公司後，發展出以「營收動能與股價強勢」為核心的機械式選股方法——「The Machine」。

瓦森策略的核心精神：鎖定具備高速營收成長的企業，並要求其股價與成交量同步展現強勢，反映基本面成長已獲市場認同。這套方法曾在1998年至1999年期間成功選中Netbank，不到一年股價漲幅超過700%。

不同於價值型投資的低估進場邏輯，瓦森的方法重視「成長中的企業」，以數據為依據進行全程量化操作，風格明確、進出彈性且強調即時性。

## 實戰應用

### 六大選股條件

| 條件 | 篩選標準 | 核心邏輯 |
|------|---------|---------|
| 1. 市值門檻 | 市值 > 市場第80百分位 | 排除規模過小、資訊不透明、流動性風險高的公司 |
| 2. 近12個月營收成長率 | 排名前20%（> 第80百分位） | 營收快速成長 = 基本面改善、產業需求回升或市占率擴張 |
| 3. 近三年營收成長率 | 排名前40%（> 第60百分位） | 短期成長與長期趨勢一致，篩除僅短期爆發的企業 |
| 4. 近12個月平均月成交量週轉率 | > 市場第60百分位 | 確保流動性，排除交易冷清難以進出的標的 |
| 5. 近12個月股價漲幅 | 排名前20%（> 第80百分位） | 股價強勢代表市場對營收與成長前景的積極評價 |
| 6. 近三月累計營收成長率 | 排名前40%（> 第60百分位） | 補足年度營收指標對當前季度趨勢反應遲緩的問題 |

### 操作方式

- 篩選出的股票以**等權重**方式買入
- 持有至下一次再平衡日
- **再平衡週期**：每21天（約1個月）進行換股
- 選出**前20檔**股票進行投資
- 以**夏普比率**作為排序依據，選出風險調整後報酬最佳的標的

### 回測績效（台股 2017-2025）

- 累計報酬：**508.783%**
- 同期大盤累計報酬：194.032%
- 2020～2024年間展現強勁上漲趨勢
- **弱點**：2025年空頭行情缺乏停損機制，導致大幅回吐獲利

### 瓦森因子模型 vs 價值投資

| 維度 | 瓦森因子模型 | 價值投資 |
|------|------------|---------|
| 選股邏輯 | 營收成長 + 股價強勢 | 低本益比 + 高股息 |
| 進場時機 | 動能確認後追進 | 價格低於內在價值時 |
| 持有週期 | 短中期（每月換股） | 長期（數年持有） |
| 風險控制 | 定期換股汰弱留強 | 安全邊際 |
| 適合環境 | 趨勢明確的多頭市場 | 市場恐慌、低估值環境 |

## 注意事項

- **營收成長是必要條件**：沒有營收成長支撐的股價上漲只是炒作，不是動能
- **市值門檻很重要**：小市值股票流動性差，可能無法在換股時順利進出
- **成交量週轉率確保可交易性**：再好的策略如果無法成交也沒用
- **21天換股週期是中短期策略**：與月營收公布節奏配合，及時捕捉營收動能變化
- **缺乏停損機制是最大弱點**：2025年回測顯示空頭行情中大幅回吐，需要加入趨勢反轉或大盤濾網條件
- **排名百分位 vs 固定數值**：瓦森使用排名百分位而非固定閾值，適應不同市場環境
- **等權重配置**：避免單一標的過度集中，但換股時交易成本較高

### 改良建議

1. **加入大盤濾網**：大盤均線空頭排列時降低持股比例或空倉
2. **加入停損機制**：個股跌破進場價8%或跌破60日均線即出場
3. **基本面安全網**：排除Z-Score < 1.81的財務危機股
4. **動能衰退監測**：股價跌破20日均線或RSI從70以上回落至50以下，提前減碼
5. **產業分散**：限制單一產業最多佔投資組合30%
