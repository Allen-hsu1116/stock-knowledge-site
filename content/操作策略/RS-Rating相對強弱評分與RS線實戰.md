# RS Rating相對強弱評分與RS線實戰

> RS Rating是股市的PR值——量化個股在全市場中的強弱排名，RS 80以上才是合格的領漲股候選，RS線突破新高是最強看漲訊號

## 核心概念

### RS Rating（相對強弱評分）

RS Rating（Relative Strength Rating）是CAN SLIM選股法核心指標，量化個股在過去12個月內相對於全市場股票的價格表現，以1-99的百分位排名表示：

- **RS 96** = 表現優於市場中96%的股票，屬於前4%
- **RS 80** = 歐奈爾建議的最低合格門檻
- **RS 50** = 表現剛好在中位數

用白話說，RS Rating就是「股市PR值」。考試不只看分數，更要看班排名——全班平均90你考80是退步，全班平均40你考60是進步。

### RS線（相對強度線）

RS線是動態圖形指標，將個股價格除以基準指數價格，直觀顯示個股相對大盤的強弱：

- **RS線上升** = 個股表現優於大盤（即使股價下跌，RS線仍可能上升）
- **RS線下降** = 個股落後大盤
- **RS線創新高** = 最強看漲訊號，尤其股價尚未創新高時

### RS Rating vs RSI：不要搞混

| 比較 | RS Rating / RS線 | RSI |
|------|------------------|-----|
| 比較對象 | 個股 vs 大盤/全市場 | 個股自身漲跌幅度 |
| 本質 | 相對強弱排名 | 動能擺盪指標 |
| 用途 | 找領漲股、強勢股 | 判斷超買超賣 |
| 數值範圍 | 1-99 | 0-100 |

## 實戰應用

### 一、找帶頭領漲股（核心用法）

1. **篩選RS ≥ 80的股票**，淘汰弱勢股
2. **觀察RS線走勢**：大盤下跌時RS線仍上升的股票最值得關注
3. **RS線率先創新高**：股價尚未突破，但RS線已創新高 = 潛在領漲股
4. **用技術模板二次過濾**：高RS只是濾網，不是進場訊號

### 二、研究市場健康度

- 高RS股集體突破順利 → 市場偏多
- 高RS股突破失敗、走勢集體變差 → 大盤可能轉壞
- 高RS名單中大型股比例 → 指數行情持續性判斷

### 三、挑選放空標的（進階）

- **不要隨便空RS高的股票**，高RS代表真實強度
- **空點在頭部確認後**：明確頭部型態出現、最後一波突破失敗
- **衰敗的高RS股**是最佳放空標的——強勢股反轉力道最猛

### 四、持股管理

- **賣出優先順序**：先賣RS下滑的股票，保留RS上升的
- **RS從高位滑落**：強勢股失去動能，考慮減碼

### Mark Minervini實證數據

- 「我絕大多數的績效來自於RS 90以上的股票，買進的股票RS最低不能小於70」
- 最強飆股在突破第一個整理區域時，**平均RS Rating為87分**
- Minervini 2021年以+334%績效奪得全美投資大賽冠軍，RS是他策略核心

## 注意事項

- **高RS ≠ 買進訊號**：RS Rating是一道濾網，不是進場指令，後續仍需技術面、基本面、籌碼面判斷
- **不適合抄底**：RS指標本質是找強勢股，逆向操作和抄底策略與RS邏輯衝突
- **不適合長投存股**：除非一路強到變長期投資，否則RS策略偏中短線
- **要搭配風險管理**：好策略配壞紀律 = 跑車配醉駕，下場只有一個
- **注意RS線與股價背離**：股價創新高但RS線下滑 = 假強勢，大盤更強你只是跟著漲
- **基準指數選擇**：台股用加權指數、美股用S&P 500、港股用恆生指數，基準不同RS值也不同
- **量化方法差異**：不同系統計算RS的方式不同（有的跟大盤比、有的跟同產業比），分數可能不一致
