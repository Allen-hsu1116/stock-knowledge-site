# VCP波動收縮型態 Volatility Contraction Pattern

> 波動收縮越來越小、成交量萎縮到極限時，就是最佳進場點 — Mark Minervini的「聖盃」型態

## 核心概念

VCP（Volatility Contraction Pattern，波動收縮型態）由美國投資冠軍 **Mark Minervini** 提出。他在1997年和2021年參加全美投資競賽，年化報酬率分別高達 **155%** 和 **334.8%**。

VCP的核心邏輯：**股價在上升趨勢的中段整理中，波動幅度一次比一次小，成交量也逐步萎縮，代表供給賣壓逐步消失，籌碼從弱者轉移到強者手中，最後出現突破點（樞紐點），就是最佳買進時機。**

### 比喻：擰毛巾

- 第一次擰：流出大量水（第一次回檔跌幅最大，通常不超過50%）
- 第二次擰：流出少量水（第二次回檔約為前一次的一半）
- 第三次擰：流出些許水（波動越來越小）
- 最後：一滴水都擰不出來（供給賣壓完全消失，少量買盤就能推升股價）

### VCP型態特徵

1. 股價經過 **2-6次** 的波浪收縮
2. 每次回檔幅度約為前一次的 **一半**（如25%→15%→12%→6%→3%）
3. 成交量同步萎縮
4. 最後收縮到極限，出現樞紐點（Pivot Buy Point）

## 實戰應用

### 選股三條件

**條件1：股票處於第二階段上升趨勢**

Minervini歸納股票四大階段：整理→承接→做頭→下降。幾乎所有強勢股的大漲都發生在第二階段。判斷標準：
- 股價高於200MA
- 200MA處於上升趨勢
- 150MA高於200MA
- 股價呈現階梯狀走勢（高點低點持續墊高）
- 短線均線位於長線之上
- 價漲量增、價跌量縮

**條件2：價格波動呈現收縮**

每次回檔幅度為前一次的一半，從大到小遞減：
- 第一次回檔：最大（如25%）
- 第二次回檔：約一半（如15%）
- 越收越小，最後可能只剩1-3%

**條件3：時間壓縮**

健全的整理期間需要 **3-65週**，讓籌碼從弱者轉到強者手中。短期內沒完成消化籌碼就急漲，即使型態符合，風險也較高。

### 進場：樞紐點（Pivot Buy Point）

VCP整理完成後出現樞紐點，是賣壓最小的臨界點：
- 價格行為呈現「價漲量增、價跌量縮」
- 只要有少量買盤就能推升股價
- 這是 **右側交易** 的進場點

### 突破後的確認訊號

1. **跟進走勢**：連續幾天大量漲勢（機構買盤vs散戶的區別）
2. **收高多於收低**：突破後第一二週，上漲天數多於下跌天數
3. **網球行為**：回檔後快速反彈（像網球彈跳，不是像雞蛋摔破）
4. **價格收在上半部**：線形上半部收盤次數多於下半部

### 停損設定

- 馬克建議停損 **7-8%** 或更低
- 原則：停損不超過期望獲利的一半
- 進場前制訂三件事：
  1. 起始停損點
  2. 重新進場條件（再滿足買進條件即可）
  3. 獲利了結（上移停損點）
- **極力反對向下攤平**：「只有輸家才會向下攤平虧損部位」

### 台股實戰回測（丹尼爾）

- 純VCP型態不夠，需搭配 **客觀量化條件**
- 結合 **基本面因子**（營收趨勢向上）+ **相對大盤季漲幅**（量價分析）
- 回測結果：**盈虧比>2，勝率5成多**，穩定獲利
- 紀律執行比選股更重要

### PowerSqueeze 指標搭配 VCP

PowerSqueeze 由 John Carter 的 Squeeze 概念改良而來，透過 **布林通道 + 肯特納通道** 組成，用於快速篩選波動收縮的股票，與 VCP 型態高度契合。

**三個能量累積等級**：
- 🔴 紅色：強力能量累積，爆發後動能較強，可延續較久時間
- 🟠 橘色：中等能量累積，可能發展成紅色強力訊號
- 🩷 粉色：一般能量累積，動能爆發後可延續的時間較短

**能量爆發（Squeeze Fired）**：當 PowerSqueeze 從紅/橘/粉色點轉為第一個灰色點時，代表累積能量開始釋放。透過動能 bar (Momentum Bar) 的方向判斷往上或往下。

**實戰搭配流程**：
1. 用 PowerSqueeze 篩選出正在累積能量的股票
2. 人工掃描是否符合 VCP 型態
3. 確認第二階段上升趨勢
4. 等待樞紐點突破進場

**實戰案例**：
- NVDA 日線：1~4步驟符合 VCP，股價放量突破後透過 PowerSqueeze 確認買入訊號
- TSLA 週線（2024年底）：爆發前有明顯 VCP 型態，數次收縮後賣壓減少，PowerSqueeze 能量累積訊號確認
- TDG 週線：兩個波幅的 VCP 型態 + PowerSqueeze 能量累積，放量突破後展開多頭趨勢

### VCP 選股量化條件（Minervini SEPA）

Mark Minervini 的 SEPA 策略對 VCP 選股有明確的量化條件：

1. 股價高於200天移動平均線，不低於30週及40週移動平均線，30週線高於40週線
2. 150天移動平均線位於200天移動平均線上方
3. 200天移動平均線向上至少1個月（最好4-5個月以上）
4. 股價向上突破底部時，位於50天移動平均之上
5. 相對強度排序（RS Ranking）不低於70
6. 一波比一波減少成交量，調整時間越來越少
7. 成交量要有一定程度的減少

### VCP 選股5條件速記（theinvestmedia整理）

1. **第二階段**：股票處於上升趨勢（Stan Weinstein 四階段理論）
2. **VCP型態選股**：均線排列正確、RS排序≥70、量縮收斂
3. **價格與成交量收縮**：波動幅度遞減、成交量顯著萎縮
4. **突破買入**：股價向上突破柄狀排列高點
5. **避開跳空下跌股**：會連續跳空下跌的股票不碰

## 注意事項

### 常見誤解
1. **不是所有波動收縮都是VCP**：合格VCP需要搭配SEPA策略的長期趨勢判斷，不是肉眼覺得「看起來像」就算
2. **VCP必須在上升趨勢中**：Minervini強調，只有第二階段（上升趨勢中段整理）的VCP才有效，下跌趨勢中的收縮不算
3. **VCP≠100%勝率**：出現VCP型態不一定能突破成功，突破失敗就要認賠

### 三大風險
1. **突破失敗**：VCP型態最後未必能突破，必須設好停損
2. **主觀判斷陷阱**：肉眼看起來像VCP不代表就是，需要客觀量化條件
3. **急漲盤整**：整理期間太短（未完成籌碼消化）就急漲，風險較高

### VCP策略的三個核心優勢（市場先生歸納）
1. **邏輯清晰**：背後有籌碼面理由（供給減少→價格易漲）
2. **風控容易**：波動收縮→停損點小→盈虧比好
3. **周轉率高**：右側交易，對錯快速揭曉，資金使用效率高
