# 三重濾網交易系統

> Elder的經典多時間框架交易架構：潮汐定方向、波浪找時機、漣漪確認進場。改良版雙重濾網簡化進場條件，保留順勢核心精神。

## 核心概念

三重濾網交易系統 (Triple Screen Trading System) 是 Alexander Elder 在《Trading for a Living》（《操作生涯不是夢》）中提出的交易方法。

Elder 用海洋的三種波動來比喻市場：

| 比喻 | 市場對應 | 時間框架 | 指標類型 |
|------|---------|---------|---------|
| 潮汐 Tide | 主趨勢 | 週線 | 趨勢型（MA、MACD） |
| 波浪 Wave | 中期趨勢 | 日線 | 震盪型（KD、RSI） |
| 漣漪 Ripple | 短期波動 | 盤中（60分K） | 突破型 |

**潮汐**是長線主趨勢，漲潮時海水愈來愈近海岸，退潮時愈來愈遠。**波浪**是潮汐中的一波波起伏。**漣漪**是波浪碰到地形的浪花，漂亮但對大方向無關緊要。

三重濾網的核心邏輯：不要逆著潮汐操作，在波浪中找時機，從漣漪確認進場。

口訣：**順大、逆小、抓突破**

## 實戰應用

### 第一重濾網：掌握主要趨勢（週線）
- **用趨勢型指標**：MA多頭排列、MACD黃金交叉
- **目的**：決定只做多還是只做空
- 週線MA多頭排列 → 只找做多機會，不做空
- 週線MACD斜率往上 → 做多方向確認

### 第二重濾網：找進場時機（日線）
- **用震盪指標**：KD、RSI、威廉指標、CCI
- **目的**：在主趨勢方向的拉回中找進場點
- 主趨勢多方 + KD超賣區（K<20, D<30）= 拉回買點
- 主趨勢空方 + KD超買區 = 反彈空點
- 也可用 [[Elder-Ray牛熊力量指標]] 的空頭力竭訊號

### 第三重濾網：確認進場點（盤中/60分K）
- **用突破型策略**：突破近期新高、帶量突破
- **目的**：從短期波動確認進場時機
- 價格突破60分K近期高點 → 確認進場

### 在XQ系統上建構

**選股腳本1 - 週線移動平均線多頭排列**（應用於週線）：
```
input:S1(6,"天期");
value1=average(close,s1);
if close>value1
then ret=1;
```

**選股腳本2 - 日線KD進入超賣區**（應用於日線）：
```
input: Length(9), RSVt(3), Kt(3);
variable: rsv(0), k(0), _d(0);
Stochastic(Length, RSVt, Kt, rsv, k, _d);
IF k<20 and _d<30
then ret=1;
```

**策略雷達3 - 60分鐘線創新高**：
```
input: Price(close);
input: Length(10);
if Price > highest(high[1],Length) then ret=1;
```

執行流程：前兩重用選股腳本每日自動執行，第三重用策略雷達盤中即時警示。

## 三重濾網的缺點與改良

### 缺點

三重濾網的第二重和第三重條件需要同時達成，這是非常嚴苛的條件：

1. **強勢行情中，日K的震盪指標很難回到低檔**：行情偏強時，RSI幾乎都在50以上跑動，50以上就代表偏強，不算低檔
2. **分K不一定出現盤整突破**：有時候行情直接V轉，不等盤整
3. **條件越多，觸發機率越低**：等不到訊號就無法進場
4. **大跳空時完全失效**：程式沒有放空邏輯，跳空時條件不滿足就不觸發

### 雙重濾網改良版

將三重濾網改良為雙重濾網，保留順勢核心精神，簡化進場條件：

**做多條件**：
1. **週K MACD斜率往上**（同三重濾網第一重）
2. **30分K的RSI從低檔轉強**（將第二、三重合併到分K層級）

**做空條件**：
1. **週K MACD斜率往下**
2. **30分K的RSI從高檔轉弱**（跌破70）

**為什麼這樣改？**
- 在強勢行情中，日K的RSI很難回到低檔
- 但30分K的RSI因為波動更大，更容易回到低檔
- 原本三重濾網的「日K震盪指標低檔 + 分K突破盤整」合併為「30分K的RSI低檔轉強」
- 抓住三重濾網的精神：順勢 + 拉回買進，但條件更實用

### RSI溫度計

將RSI視覺化為四色溫度計，直接顯示在K棒上：
- 🔴 亮紅色：最強（RSI高檔）
- 🔴 暗紅色：第二強
- 🟢 暗綠色：偏弱
- 🟢 亮綠色：非常弱（RSI低檔）

從顏色變化直接辨識轉強或轉弱：亮綠→暗綠 = 做多進場時機。

## 雙重濾網交易策略細節

### 長線策略
- **風險管理**：每筆損失控制在總資金的2%以內
- **進場**：雙重濾網（週K MACD + 30分K RSI）
- **停損**：跌破當日低點，或參考ATR設5%
- **移動停利**：最近3天K棒高低點基準，跌破6%出場
- **加碼**：永遠只在賺錢時加碼！拉回2%且RSI<40做多（做空反之RSI>60）
- **賺賠比**：有機會達到1:40甚至1:100

### 短線策略（期貨5分K）
- **停損**：50點
- **停利**：3根K棒 + 100點
- **加碼**：RSI<40做多，RSI>60做空
- **賺賠比**：有機會達到1:40

### 加碼的核心原則
- **永遠只在賺錢的部位上加碼**
- 每次加碼都要符合雙重濾網的進場條件
- 加碼位置：RSI拉回後重新轉強的位置，不是創高位置
- 加碼是拉開賺賠比的關鍵

## 注意事項

- **三重濾網不等於三個指標**：是三個「層級」的判斷，每個層級用不同類型的指標
- **不要跳過第一重**：不看大方向直接用KD做交易是散戶常見錯誤
- **指標類型要匹配**：第一重趨勢型、第二重震盪型、第三重突破型，不要混淆
- **三重都通過才進場**：任何一重不通過就不做，寧可錯過不要做錯
- **雙重濾網更實用**：解決三重濾網條件過於嚴苛的問題，但核心精神不變——順勢操作
- **挑有趨勢的商品做**：不管是當沖或波段，挑有趨勢盤的標的，不要在盤整盤進場
