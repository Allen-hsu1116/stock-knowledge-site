# 台指期現貨價差套利

> 當台指期逆價差超過0.3%，法人火力全開：賣出一籃子現貨、買入期貨，結算日收斂至零穩穳收下價差利潤——散戶做不了，但必須看懂法人在做什麼。

## 核心概念

### 台指期長期逆價差現象

台指期長期處於逆價差（折價）狀態，均值約0.2%，這不是偶然，而是台股制度面造成：

1. **融借券放空不易**：台股融券有定期強制回補（股東會、除權息），無法長期做空，放空需求轉到期貨
2. **期貨主要作為避險工具**：大股東和法人用現貨做多、用期貨做空避險，期貨市場空方較為積極
3. **反向ETF規模龐大**：如00632R等反向ETF轉倉時放空大量期貨部位，正向ETF幾乎全持股，而反向ETF的AUM更大

### 套利門檻

| 指標 | 數值 |
|------|------|
| 長期折價均值 | 約0.2% |
| 交易成本門檻 | 0.3%起 |
| 套利啟動區 | 0.3%-0.5% |
| 法人火力全開區 | >0.5% |
| 近期極值 | 約0.8% |

當折價超過0.3%-0.5%，價差收斂的利潤空間足以覆蓋交易成本，套利機會出現。

## 實戰應用

### 進場邏輯

當台指期逆價差超過門檻（>0.3%）：

**基本作法**：
- 將持股一籃子按台指權重賣出現貨
- 同時買入對應部位的期貨
- 等價差收斂獲利

**積極作法**：
- 除了賣出自有持股
- 更在全市場進行借券/融券放空一籃子股票
- 買入更大部位的期貨
- 借券成本較高，但價差夠大仍有利潤

### 出場方式

**方式一：價差收斂**
- 等台指期與現貨價差回到0.2%左右
- 反向平倉出場

**方式二：期貨結算**
- 市場空單強勁時價差可能不快速收斂
- 等到期貨結算日
- 台指期結算根據加權指數最後半小時簡單加權平均價
- 在最後半小時分批平倉，確保收斂至零出場

### 看懂法人動向

即使散戶做不了套利，理解法人行為很重要：
- 逆價差突然擴大 → 法人開始大舉賣現貨買期貨 → 市場可能有支撐
- 正價差異常擴大 → 法人可能賣期貨買現貨 → 注意避險需求

## 注意事項

- **散戶資金門檻高**：同時放空50檔以上股票+對齊權重，操作難度極高
- **借券成本不確定**：積極作法的借券成本較高，侵蝕套利利潤
- **交易成本計算**：現貨賣出+期貨買入的雙邊成本0.3%起，價差不夠大反而虧
- **價差可能先擴大**：進場後逆價差可能繼續擴大才收斂，需有承受能力
- **結算日是安全網**：即使價差不收斂，結算日一定收斂至零，但中間的保證金追繳風險要控管
- **反向ETF持續壓力**：反向ETF轉倉持續放空期貨，是逆價差的結構性因素
