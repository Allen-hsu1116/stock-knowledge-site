# 配對交易進階實戰

> 一多一空對沖系統風險，賺的是價差收斂的利潤——但歷史相關不代表未來相關

## 核心概念

配對交易（Pairs Trading）又稱價差交易或統計套利，核心邏輯是：兩種高度相關的資產，短期價差偏離歷史均值時，買進被低估的、賣出被高估的，等價差收斂時獲利。

這是一種**市場中性策略**：不預測市場方向，做多一檔同時做空另一檔，消除大部分系統性風險。

### 配對交易三要素

1. **配對選擇**：兩檔股票必須存在長期均衡關係（共整合關係）
2. **進場時機**：價差偏離均值到一定程度時進場
3. **出場時機**：價差回歸均值時平倉獲利，偏離過大時停損

### 共整合檢定（Cointegration Test）

配對交易的前提是兩檔股票存在**共整合關係**，不是單純的相關性：

- **相關性**：兩檔股票同漲同跌（相關係數高）
- **共整合**：兩檔股票的價差在長期會回歸均值（定態性）

檢定步驟：
1. 確認兩檔股票的日報酬率序列具有定態性
2. 用ADF檢定（Augmented Dickey-Fuller Test）確認價差序列是否定態
3. 若1%顯著性水平下拒絕虛無假設，確認共整合關係存在

### 形成期與交易期

- **形成期**：用歷史數據計算價差的μ均值和σ標準差
- **交易期**：用形成期的參數設定進出場條件
- 關鍵：避免前視偏誤，交易期的參數必須來自形成期

## 實戰應用

### TEJ實證：長榮與陽明配對交易

以長榮（2603）與陽明（2609）為例：
- 形成期：計算價差序列的α和β係數
- 交易期：用α和β計算即時價差

### 進出場規則

用形成期的μ和σ設定門檻：

| 條件 | 動作 |
|------|------|
| 價差 > μ+1.5σ | 做空配對（賣陽明、買長榮） |
| 價差回落至 μ+0.2σ | 反向平倉獲利 |
| 價差 < μ-1.5σ | 做多配對（買陽明、賣長榮） |
| 價差回升至 μ-0.2σ | 反向平倉獲利 |
| 價差突破 μ±2.5σ | 即時停損平倉 |

### 改善方向

1. **調降停損標準**：2.5σ可能太寬，考慮2σ
2. **Rolling參數計算**：用滾動方式計算μ和σ，避免形成期參數在交易期失靈
3. **注意波動性差異**：形成期低波動但交易期高波動時，參數會失效
4. **加入ADX過濾**：盤整期價差波動小，趨勢期波動大，用ADX過濾

## 注意事項

### 配對交易五大風險

1. **相關性崩潰**：歷史相關不代表未來相關，結構性變化會使配對失效
2. **價差不收斂**：價差可能持續擴大而不回歸均值（如海運股2021年）
3. **需要大量本金**：利潤通常很小，需要大本金才能獲得有意義的報酬
4. **手續費侵蝕**：一多一空雙邊交易，手續費和滑價成本較高（台股0.1425%×2 + 證交稅0.3%）
5. **報酬率逐年下降**：市場越效率、機構法人競爭增加，套利空間越來越小

### 台股配對交易的特殊考量

- **同產業配對**：同產業股票共整合關係較穩定（如長榮/陽明、國泰金/富邦金）
- **流動性風險**：做空台股需要借券，流動性可能不足
- **平盤以下不得放空**：台股限制平盤以下不能融券賣出
- **融券成本**：借券費可能侵蝕獲利
- **配對選擇**：需用統計方法（ADF檢定、相關係數篩選）而非直覺

### 實證研究結論

以台灣50及中型100成分股為樣本（2005-2014），配對交易在多頭時期表現較佳，但報酬率有明顯逐期下降趨勢，反映市場效率提升與機構競爭加劇。
