# 動能交易策略

> 買入價格走勢強勣的資產，在趨勢反轉或動能減弱時賣出；核心假設是「強者恆強，弱者恆弱」，學術基礎來自Jegadeesh & Titman (1993) 動能效應

## 核心概念

動能交易（Momentum Trading / Momentum Investing）基於「趨勢將持續」的核心假設。它不挖掘資產內在價值，而是專注於市場價格行為本身。假設市場短期內並非完全效率，投資者的群體心理（追漲殺跌、FOMO）會導致價格趨勢在一定時間內自我強化，創造出可利用的交易機會。

可以比喻為衝浪：不挑微波細浪，而是等待最高、最有力的浪潮，順勢而行，直到浪頭力量減弱才退場。

### 動能效應的學術基礎

1993年Jegadeesh和Titman的經典研究證實：根據過去3到12個月表現篩選出的**贏家投資組合**（Winners Portfolio），在未來數月內表現顯著優於**輸家投資組合**（Losers Portfolio）。這一發現在全球多個市場和不同資產類別中都得到反覆驗證。

行為經濟學解釋：投資人在牛市中過度樂觀、熊市中過度悲觀，導致價格持續擴張，形成「強者恆強、弱者恆弱」的動能效應。

### 動能投資 vs 價值投資

| 維度 | 動能投資 | 價值投資 |
|------|---------|----------|
| 核心哲學 | 強者恆強，順勢而為 | 物超所值，等待回歸 |
| 買入信號 | 價格突破新高、量能放大、技術強勢 | 股價低於內在價值、安全邊際大 |
| 賣出信號 | 趨勢反轉、跌破支撐、動能衰退 | 股價達到內在價值、基本面惡化 |
| 持股週期 | 數週到數月 | 數年到數十年 |
| 關注焦點 | 價格、成交量、技術指標 | 財報、護城河、本益比、PBR |
| 性格要求 | 紀律強、抗波動、果斷 | 耐心、獨立思考、長期視角 |

**價值動能（Valuentum）策略**：在價值被低估的股票中，尋找開始展現上升動能的標的，結合兩者優勢。

## 實戰應用

### 第一步：篩選具備動能的標的（四大選股標準）

- **價格接近52週新高**：設定「股價在52週高點的90%以上」，這是最直觀的強勢信號
- **相對強弱（Relative Strength, RS）**：注意RS不是RSI——RS是個股在過去6-12個月漲幅與市場指數的比較，RS排名高＝市場「領頭羊」
- **成交量顯著放大**：在突破關鍵價位時，成交量是平日均量**2倍以上**，代表趨勢有買盤支撐
- **優於預期的盈利報告**：良好基本面是動能的催化劑，盈餘超預期吸引大量資金湧入

### 動能投資操作四法

1. **回測績效挑選法**：挑選近3-6個月報酬率表現最佳的股票，以前10%為篩選標的
2. **尋找均線強勢股**：多條均線多頭排列（5日>20日>60日），均線向上發散
3. **設定進出場原則**：跌破20日或60日均線即停利或停損
4. **固定週期汰換**：每月/每季重新檢視投資組合，淘汰績效轉弱標的，**汰弱換強**

### 第二步：技術指標確認進場

| 指標 | 動能交易應用 |
|------|------------|
| MA | 黃金交叉買入；價格在均線之上且均線向上確認多頭 |
| RSI | RSI持續維持70以上＝極強動能，是強勢趨勢確認（非超買賣出） |
| MACD | DIF上穿MACD + 柱狀體0軸上放大＝多頭動能強勁 |
| ADX | ADX>25甚至40，代表趨勢非常強烈，適合動能進場 |

### 第三步：嚴格的出場與風控

- **固定停損**：進場時設定百分比（如-8%）或關鍵技術位
- **移動停損（Trailing Stop）**：停損點隨股價上價上漲提高，鎖定獲利同時讓利潤奔跑
- **停利**：預期報酬目標或動能衰退信號（RSI回落、價量背離）時分批出場

### 動能突破選股實戰條件

- 股價創20日/60日/120日新高
- 成交量>近5日均量3倍
- 突破上關價、盤中由黑翻紅、一紅吃三黑
- 突破糾結均線、均線多頭排列
- OBV突破20日均量
- 籌碼加碼：外資/投信近1日買超、大戶持股增加、主力密集收貨

### 進場兩種風格

1. **積極型**：突破後直接追價，突破多發生在開盤30分鐘至1小時內
2. **保守型**：突破後等拉回至均價線或前波高點，確認不破再進場

## 注意事項

- **假突破（Whipsaw）**：追高買入後市場急轉直下，是動能交易最大威脅
- **交易成本高昂**：頻繁短線操作的手續費與交易稅會侵蝕利潤
- **心理壓力**：FOMO讓人追在最高點，恐慌讓人砍在最低點
- **盤整市失效**：無明確趨勢時動能策略頻繁發出錯誤信號，導致連續虧損
- RSI>70在動能交易中是「強勢確認」而非「超買賣出」，但趨勢反轉時RSI也最快回落，要搭配其他指標
