# 固定時間出場 Time-Based Exit

> 每個策略都有有效期限，用時間控制出場跟用價格控制出場一樣重要——固定時間出場既是風險控管工具，也是策略有效性測試工具。

## 核心概念

固定時間出場（Time-Based Exit），就是進場後設定一段時間（例如7天、1個月、3個月），無論賺賠都要出場，結束持有的部位。

這不是一種「懶惰」的出場方式，而是有明確邏輯支撐的策略工具。

### 固定時間出場 vs 價格出場

| 比較 | 價格出場（停損停利） | 固定時間出場 |
|------|---------------------|-------------|
| 控制維度 | 價格 | 時間 |
| 核心目的 | 限制單筆虧損幅度 | 限制策略有效期限 |
| 適用場景 | 所有策略 | 有明確進場訊號的策略 |
| 最經典例子 | 停損單 | 當沖（日內出場） |

當沖就是最典型的固定時間出場：把波動風險限制在一天以內。差別只是時間長短——7天、1個月、3個月，意義跟停損一樣，只是用不同維度控制風險。

## 三個核心觀點

### 1. 固定時間出場附帶風險控管效果

每個策略都有它的「有效期限」。時間出場的重要程度甚至高於價格出場，因為：

- 價格出場只控制「虧多少」，不控制「虧多久」
- 時間出場控制「風險持續多久」，限制不確定性
- 兩者搭配使用才是完整的風險管理

### 2. 每個策略都有合理的有效期限

「合理」不應該由最佳化決定，而是由邏輯決定：

- 用短周期技術指標的策略，平均2天出現一次進場訊號，有效時間最長約10天
- 如果設定2個月出場，顯然不合理
- 最適合的數字本來就是一個範圍，不存在「最好的數字」
- 如果差1天就從賺錢變賠錢，策略本身一定有問題

**不知道固定出場應該取幾天？** 代表你對自己的策略還不夠了解，應該先停下來多想想。

### 3. 固定時間出場可以測試進場策略的有效性

固定時間出場等於將出場結果「隨機化」，可以測試「進場策略」夠不夠強：

- 加上停損停利等出場策略後，績效好，到底是進場策略的功勞還是出場策略的功勞？
- 固定時間出場消除了出場策略的影響，純粹測試進場品質

獲利可以只靠進場，也可以只靠出場策略：
- 只靠出場：定期定額買0050（進場不重要，選股+部位+出場才是關鍵）
- 只靠進場：短線脈衝策略（進場點位太差就註定賠錢，面臨頻繁停損和不足的波動性）

## 實戰應用

### 如何設定固定時間出場

1. **根據策略邏輯設定時間**：
   - 短線技術指標策略：5-10天
   - 波段策略：20-30天
   - 基本面策略：60-90天
   - 長線投資：180-365天

2. **固定時間出場≠不能長期持有**：
   - 長線投資 = 短線不斷確認「繼續持有」的結果
   - 到期重新評估：如果此時再買一次你仍願意進場 → 繼續持有
   - 到期重新評估：如果不願再買 → 出場

3. **每次測試新策略，先測固定時間出場**：
   - 了解策略的特質
   - 了解最危險的狀況
   - 了解如何承受或避開風險

### 固定時間出場的判斷邏輯

```
進場後設定時間T
↓
時間到 → 出場（不管盈虧）
↓
重新評估是否進場
↓
願意 → 繼續持有（重新計算時間T）
不願意 → 空手等待下一次訊號
```

這個邏輯把「持有」變成「不斷確認願意再買一次」的過程，而不是被動等待。

## 注意事項

1. **固定時間出場不能單獨使用**——如同大多數工具一樣，它需要搭配停損停利等策略
2. **小心最佳化陷阱**——不要不斷嘗試3天、5天、7天...去找「最好的天數」，合理範圍內績效差異不大
3. **策略的有效期限由邏輯決定，不是由回測決定**——短周期指標的時間限制短，長期基本面策略的時間限制長
4. **時間出場重要程度甚至高於價格出場**——因為每個策略都有有效期限
5. **如果策略的固定時間出場績效很差**——問題出在進場策略，不是出場設定
