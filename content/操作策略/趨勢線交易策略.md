# 趨勢線交易策略

> 趨勢線是所有技術分析中最直觀的工具——一條線看穿支撐壓力，但畫對趨勢線比你想的難得多

## 核心概念

### 什麼是趨勢線

趨勢線是透過連結價格的高點或低點所畫出的直線，用來判斷支撐與壓力、預測行情方向。當價格還在趨勢線的軌道裡時，會繼續沿著趨勢線前進。

### 上升趨勢線

- **畫法**：連結上升趨勢中的兩個低點（低點與次低點），形成向上傾斜的直線
- **意義**：上升趨勢線就是支撐線，價格回檔到趨勢線附近常會止跌
- **越多的低點落在趨勢線上，這條線越可靠**

### 下降趨勢線

- **畫法**：連結下降趨勢中的兩個高點（高點與次高點），形成向下傾斜的直線
- **意義**：下降趨勢線就是壓力線，價格反彈到趨勢線附近常會遇壓
- **越多的低點落在趨勢線上，這條線越有效**

### 畫趨勢線的七大規則

1. **一致性**：趨勢線可以連結K線實體高低點，也可以連結影線高低點，但不能一個用實體、一個用影線
2. **不可穿破K線實體**：趨勢線不能穿破任何K線實體，若被破了需重新畫
3. **斜度30-70度**：太陡或太平的趨勢線都不好用
4. **表示趨勢，不等於確認趨勢**：趨勢線是觀察工具，不是確認工具
5. **高點低點需配合**：畫上升趨勢線時高點也要上移；畫下降趨勢線時低點也要下移，否則是假趨勢線
6. **不必從最高/最低點開始**：選擇最能反映趨勢的點位即可
7. **多條線選被驗證最多次的**：滿足條件的趨勢線可能很多條，用被觸及最多次的那條

### 三點確認法

「兩點成線，三點確認」：連接任意兩個價格點只是「試探性」趨勢線；必須等到第三個價格點觸及該線並反彈，這條趨勢線才被視為有效。以極值（影線的最高或最低點）為基準，因為這代表了該時段內價格探索的極限。

## 實戰應用

### 趨勢通道（軌道線）

趨勢線的進階應用是畫出趨勢通道：

**上升通道**：
1. 先畫出上升趨勢線（連結兩個低點）
2. 將線平行上移至波段高點，形成上軌道
3. **下軌道 = 支撐，上軌道 = 壓力**

**下降通道**：
1. 先畫出下降趨勢線（連結兩個高點）
2. 將線平行下移至波段低點，形成下軌道
3. **上軌道 = 壓力，下軌道 = 支撐**

### 通道操作四法則

**法則1：多方趨勢跟通道操作**
- 第1次打到下軌道支撐 → 進場試單
- 第2次打到下軌道支撐 → 加碼
- 第3次打到下軌道支撐 → 再加碼
- 股價打到上軌道 → 減碼停利
- 第4次回測下軌道跌破 → 全數出場

**法則2：空方趨勢跟通道操作**
- 第1次打到上軌道壓力 → 進場試單
- 第2次打到上軌道壓力 → 加碼
- 第3次打到上軌道壓力 → 再加碼
- 股價打到下軌道 → 減碼停利
- 第4次回測上軌道突破 → 全數出場

**法則3：多方突破平移**
- 股價帶量突破上軌道 → 將軌道平行上移，移動距離與原軌道對稱
- 用新軌道線找新的停利點

**法則4：空方跌破平移**
- 股價帶量跌破下軌道 → 將軌道平行下移，移動距離與原軌道對稱
- 用新軌道線找新的停利點

### 趨勢線突破判斷

- **跌破上升趨勢線**：上漲可能結束，準備向下修正
- **突破下降趨勢線**：下跌可能結束，價格將「真正」起漲
- **確認突破**：幅度大於3%、連續站穩3天、成交量放大
- 通道被突破後，可配合[[趨勢線交易策略]]判斷趨勢是否真的改變

### 停損設置

- 進場後停損設在鄰近K棒的最低點（多方）或最高點（空方）
- 跌破/突破軌道線就是停損訊號，不用猶豫

## 注意事項

### 趨勢線是主觀的

- 同一張圖，10個人畫出10條不同的趨勢線
- 需要滾動式調整，直到找到最有效的一條
- 一條趨勢線如果在某個位置反覆產生支撐/壓力，才是真的有效

### 不要單獨使用

- 趨勢線是輔助工具，不是唯一依據
- 搭配 MACD、KD、RSI、成交量等指標綜合判斷
- 震盪盤中趨勢線無用，不如用[[均值回歸策略]]

### 盤整行情不畫趨勢線

- 趨勢線的前提是有趨勢，盤整行情畫了也沒用
- 盤整時改用[[箱型理論Darvas-Box]]或[[震盪整理交易策略]]

### 趨勢線被突破不等於趨勢反轉

- 可能只是趨勢加速，需要重新畫平行線
- 用123法則確認趨勢是否真正改變
- 平移通道找新目標點位

### 觸及次數的陷阱

- 趨勢線被觸及的次數越多，越可靠——但同時也代表支撐/壓力正在被消耗
- 第4次觸及的支撐/壓力，被突破的機率比前3次高得多
- 「支撐測試越多，支撐力越弱」
