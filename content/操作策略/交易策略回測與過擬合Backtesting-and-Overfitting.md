# 交易策略回測與過擬合

> 回測是交易策略的試金石，過擬合是回測的最大敵人——歷史績效再完美，樣本外失效就等於零

## 核心概念

### 回測的本質

回測（Backtesting）是將交易策略套用於歷史資料，驗證策略是否有效。核心邏輯是：如果策略在過去表現差，未來也不太可能好；但過去好不等於未來好。

回測不是證明策略有效，而是**排除無效策略**。通過回測的策略只是候選者，不是保證書。

### 回測的四大偏誤

#### 1. 過擬合（Overfitting）— 最大敵人

過度調整參數讓策略完美匹配歷史資料，但對未來資料毫無預測力。

**辨識過擬合的方法**：
- 參數微調1-2%，績效就劇烈變化 → 過擬合
- 樣本內 Sharpe > 2，樣本外 < 0.5 → 嚴重過擬合
- 策略規則太複雜、條件太多 → 高過擬合風險

**防範方法**：
- 樣本內/樣本外分離（60-70%訓練，30-40%測試）
- 參數精簡原則（Occam's Razor）——能用3個參數就不要用10個
- Walk-Forward Analysis（滾動前進分析）
- 蒙地卡羅模擬壓力測試
- 參數敏感度分析：微調參數看績效是否劇烈變化

#### 2. 前瞻偏差（Look-Ahead Bias）

在回測中使用了當時無法取得的資訊。例如：
- 年報數據假設12/31就能取得，但實際要到2月才公布
- 用當日收盤價做當日進場決策（應用次日開盤價）
- 使用經過回補調整的歷史價格（股票拆分後才調整，回測時卻當作已知）

**防範**：嚴格確認每筆資料的可用時間點。

#### 3. 生存者偏差（Survivorship Bias）

只用至今仍存在的公司做回測，忽略已下市的公司。結果讓策略績效虛高。

**實際影響**：如果只用台灣50成分股回測，等於自動排除了所有被剔除的成分股，績效自然好看。

**防範**：使用包含已下市公司的完整資料庫（如TEJ）。

#### 4. 選擇偏差（Selection Bias）

挑選特定股票和時間區間讓策略看起來很好。

**防範**：使用多個不同時間區間和無偏見的代表性樣本。

## 實戰應用

### 回測績效七大指標

| 指標 | 計算方式 | 實戰意義 |
|------|----------|----------|
| **CAGR** | 複合年化成長率 | 策略平均每年賺多少 |
| **Sharpe Ratio** | (報酬-無風險利率)/標準差 | 每承擔一單位風險換來的超額報酬 |
| **Maximum Drawdown** | (最低點-最高點)/最高點 | 最慘會虧多少 |
| **Calmar Ratio** | CAGR / MDD | 報酬與最大回撤的比值 |
| **Win Rate** | 獲利交易次數/總交易次數 | 勝率多高 |
| **Profit Factor** | 總獲利/總虧損 | 賺的錢是虧的幾倍 |
| **Max Consecutive Losses** | 最大連續虧損次數 | 心理壓力測試 |

### Sharpe Ratio 實戰判讀

Sharpe Ratio = (Rx - Rf) / StdDev Rx

| 夏普比率 | 評價 |
|----------|------|
| < 1 | 不佳，風險調整後報酬不夠 |
| 1 - 1.99 | 適當/良好 |
| 2 - 2.99 | 很好 |
| > 3 | 優秀（但小心過擬合） |

**關鍵理解**：
- 高報酬 ≠ 好投資，可能是承擔過高風險換來的
- Sharpe 把報酬除以風險，衡量每一單位風險換來的超額報酬
- 基金A報酬20%、Sharpe 2.0 vs 基金B報酬30%、Sharpe 0.5 → A是更好的風險調整後投資

### Maximum Drawdown 實戰判讀

MDD = (Trough - Peak) / Peak

- MDD -20%：需要 25% 報酬才能回本
- MDD -50%：需要 100% 報酬才能回本
- MDD -90%：需要 900% 報酬才能回本

**回撤恢復難度是對數增長的**——虧越多，回本所需的報酬率越大。

### 回撤最佳化投資組合

用 MDD 替代波動率建構投資組合：
- 追求低 MDD 的資產組合
- 確保回撤後恢復新高的平均時間在兩年以內
- 投資人真正關心的是：虧損後多久能回本？

### Walk-Forward Analysis（滾動前進分析）

解決過擬合的最佳實務方法：

1. 用 2015-2020 資料訓練參數
2. 用 2021 資料測試績效
3. 滾動：用 2016-2021 訓練，2022 測試
4. 再滾動：用 2017-2022 訓練，2023 測試
5. 看每次測試期績效是否穩定

如果每次 Out-of-Sample 績效都穩定，策略才有參考價值。

### 參數敏感度分析

微調每個參數 ±5%~±20%，觀察績效變化：
- 績效平穩變化 → 參數穩健，策略可靠
- 績效劇烈跳動 → 參數脆弱，過擬合風險高

**實務原則**：參數越少越好。3個參數的策略比10個參數的策略更可信。

## 注意事項

### 回測的六大陷阱

1. **過擬合** — 歷史績效完美，實盤一塌糊塗。永遠做 Out-of-Sample 測試
2. **前瞻偏差** — 用了未來資訊不自知。嚴格檢查每筆資料的可用時間點
3. **生存者偏差** — 只回測活著的公司，忽略下市的。績效虛高
4. **交易成本低估** — 手續費、滑價、衝擊成本會吃掉大量利潤。回測中務必加入合理交易成本
5. **流動性假設** — 回測假設任何數量都能以收盤價成交，實際大單會造成滑價
6. **市場制度變遷** — 歷史上的交易規則可能不同（漲跌幅限制、交易稅率等），直接套用會失真

### 過擬合的警訊

- 策略在樣本內 Sharpe > 3 但樣本外 < 1
- 參數微調就讓績效天翻地覆
- 規則超過5條、參數超過5個
- 只在特定市場環境有效
- 歷史績效曲線太完美（像直線上升）

### 好的回測長什麼樣子

- **Sharpe > 1**（樣本外）
- **MDD < 20%**
- **Profit Factor > 1.5**
- **參數穩健**（微調 ±20% 不會讓策略失效）
- **跨市場有效**（不只台股，美股、日股也能用）
- **跨時間有效**（牛市熊市震盪市都有合理表現）
