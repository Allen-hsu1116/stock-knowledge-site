# 因子投資與量化危機

> 因子投資長期有效不代表短期不會大幅落後市場，2018-2020年量化危機是半個世紀來首次其他因子補不了價值因子的虧損

## 核心概念

因子投資（Factor Investing）是透過系統化曝險於特定因子（價值、動能、規模、獲利、投資等）來獲取超額報酬的投資方法。Fama-French五因子模型是學術界公認的基礎框架：

- **價值因子**：低估值股票長期勝出（本益比低、淨值比低）
- **動能因子**：過去漲的股票短期繼續漲
- **規模因子**：小型股長期勝出
- **獲利因子**：高獲利能力公司勝出（ROE高、營業利潤率高）
- **投資因子**：保守投資（總資產增速低）的公司勝出

Smart Beta ETF就是將這些因子具體化的產品，如0056納入殖利率因子、00878加入ESG因子。

### 量化危機（2018-2020）

2018年中至2020年中爆發「量化危機」，多因子策略經歷嚴重績效下跌：

- 價值因子超大幅下跌，規模、投資因子也表現悽慘
- 100個投資組合中超過85個跑輸市場，70個落後10%以上
- 超大型成長股（FAANG+微軟+Tesla）的超額報酬53.8%
- 只有動能和獲利因子表現較好，但不足以彌補價值因子虧損

### 歷史上的價值因子大跌

從1963年以來，價值因子共有五次大跌：

| 期間 | 特徵 | 動能因子表現 |
|------|------|-------------|
| 1970-1972 (Nifty Fifty) | 超大型成長股主宰 | 其他市值分組動能正報酬 |
| 1979-1980 | 動能大復甦 | 所有市值分組動能正報酬 |
| 1989-1991 | 小型股弱勢 | 動能因子仍有效 |
| 1998-2000 (網路泡沫) | 動能復甦 | 動能報酬遠超價值虧損 |
| **2018-2020 (量化危機)** | **成長因子復甦** | **動能僅部分補償** |

前四次動能因子都能彌補價值因子虧損，2018-2020是半世紀來第一次其他因子補不了。

### 2009年量化危機

另一場量化危機發生在2009年初：
- 動能因子突然崩潰，2008年大跌的股票出現趨勢反轉（特別是銀行保險股）
- 但持續僅半年，多因子趨勢隨後反轉向上
- 與2018年量化危機下跌幅度相當，但時間短得多

## 實戰應用

### 因子投資的關鍵認知

1. **績效偏離是必然的**：因子投資的績效會跟市場大盤很不一樣，別人爛的時候你好，你爛的時候別人好
2. **低賣高買才是真正的風險**：在2018-2020年看到因子策略落後市場就放棄因子投資，等於低賣，轉買熱門科技股等於高買
3. **價值因子2020年後年化25%**：長期均值回歸仍然有效，大型成長股的超額報酬反而是反常
4. **多因子分散是唯一免費午餐**：但分散不等於不會虧損，只是在某一個因子崩潰時減少傷害

### 價值因子的反面是成長股，不是動能

從因子定義來說，價值股和成長股才是硬幣的正反兩面。當價值因子強勁時，遭殃的是成長股，不是動能因子。

### 價值與動能因子的互補關係

AQR研究（Asness, Moskowitz, Pedersen 2013）證實：
- 動能因子捕捉短期股價的Upwards extrapolation（向上過度反應）
- 價值因子捕捉Downwards extrapolated stocks（向下過度反應）
- 兩者結合才能完全捕捉市場的所有錯誤定價來源
- 動能因子是不可複製因子（irreplicable factor），即使與價值因子呈現負相關也不代表要失效

### 低波動異象

低波動因子（Low Volatility Factor）是另一個重要的因子：
- 低波動股票在風險調整後的回報率高於高波動股票
- 存股族的超額報酬來源之一就是低波動異象
- 越不效率的市場（如台灣），某些因子更加有效

## 注意事項

1. **因子不是萬靈丹**：任何因子都有長期落後市場的時候，不能只看回測就下結論
2. **績效偏離的心理考驗**：親朋好友績效比你好的時候，能否堅持？這是行為面的挑戰
3. **小型股因子流動性風險**：小型價值股雖然因子溢酬高，但流動性差，大資金需考慮滑價
4. **量化危機的教訓**：分散投資不一定每次都能對沖，2018-2020年就是反面教材
5. **因子擇時極度困難**：知道價值因子長期有效，不代表你能忍受中間的巨大回撤
