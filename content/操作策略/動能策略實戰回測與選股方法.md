# 動能策略實戰回測與選股方法

> TEJ回測顯示台股動能策略年化報酬28.26%但波動37%，玉山證券提供帶量突破即時選股條件；關鍵在於「持續性篩選」提升勝率、「市值前200大」過濾操控風險

## 核心概念

動能策略的實戰效果如何？TEJ台灣經濟新報用TQuant Lab回測了台股動能策略，提供了量化績效數據。而玉山證券的E-Trader策略選股工具則提供了盤中即時動能選股的實戰方法。兩者結合，形成從選股到進出場的完整動能策略框架。

### TEJ回測核心設定

- **股票池**：每年篩選市值前200大公司（避免選到成交量太小、容易被操控的股票）
- **動能計算**：過去20個交易日收盤價變化率
- **再平衡**：每月第一個交易日買入動能前5高的股票
- **配置**：等權配置（每檔20%）
- **持有規則**：連續入選者續抱，落選者賣出
- **回測期間**：2019/01/01 ~ 2025/04/10
- **初始資金**：1,000萬元

### TEJ回測績效

| 指標 | 數值 |
|------|------|
| 年化報酬率 | **28.26%** |
| 年化波動度 | **37.25%** |
| 夏普值 | 偏低（高波動侵蝕風險調整報酬） |
| 最大獲利來源 | 2021年捕捉陽明海運飆漲 |
| 2025關稅戰表現 | 比大盤差 |

**結論**：動能策略高報酬伴隨高風險，關鍵獲利來自少數幾次大的趨勢性機會。2025年關稅戰期間表現不佳，說明動能策略在系統性風險下會失效。

## 實戰應用

### 一、TEJ動能策略的改良方向

1. **加入止盈止損**：原始策略無止盈止損，波動極大，可自行設定停損-8%、停利+15%等條件
2. **調整再平衡天數**：程式提供20天再平衡版本，可依市場狀態調整
3. **加入市值過濾**：每年更新市值前200大股票池，避免流動性不足的個股
4. **多因子結合**：純動能策略容易選到短期飆股，可加入基本面或籌碼面濾網

### 二、玉山證券帶量突破選股法

這是盤中即時選股方法，補足了TEJ月度再平衡無法捕捉日內機會的不足。

#### 核心條件

| 條件 | 參數 | 原因 |
|------|------|------|
| 股價創新高 | 近20日/60日/120日新高 | 市場資金持續買進，吸引動能交易者進場 |
| 成交爆大量 | >近5日均量3倍 | 篩選出明顯不同於日常交易的個股 |

#### 進階篩選條件

**價量指標**：
- 突破上關價：三關價突破後容易形成趨勢
- 盤中由黑翻紅：買賣力道反轉訊號
- 一紅吃三黑：強勢反轉型態
- 突破糾結均線：漲幅>5%突破盤整區間
- 均線多頭排列：多頭趨勢確認
- OBV突破：量能趨勢啟動

**成交排行指標**：
- 週轉率排行前20名（波動大但風險也高）
- 昨量比排行前50名（量比放大=資金進駐）
- 成交值排行前100名（市場資金焦點）

**籌碼指標**：
- 外資近1日買超前30名
- 投信近1日買超前30名
- 1000張以上大戶持股增加>1%
- 近5日Top1券商買超佔比>20%（主力密集收貨）
- 近1日融資增加>10%（短線資金進駐）

#### 進場策略

**積極型**：
- 突破後直接追價
- 最佳時機：開盤後30分鐘至1小時
- 觀察成交量是否持續放大

**保守型**：
- 突破後等回測均價線或前波高點
- 確認均價線不破再掛單買進
- 不宜提前掛單等待（跌破風險）

#### 出場策略

**停損**：
- 追價進場：停損設在前波高點下方0.5%~1%（4~8 tick）
- 回測進場：跌破均價線即停損

**停利**：
- tick流/一波流：5~15分鐘內未繼續上漲即出場；獲利>5~10 tick後用智慧單保本
- 日內波/短波段：抱至中盤（11點~12點）或收盤前（13點~13:30）分批減碼
- 鎖漲停：改以波段操作

### 三、TEJ回測 + 玉山選股 結合實戰框架

| 層級 | 方法 | 用途 |
|------|------|------|
| 月度選股 | TEJ動能前5大 | 中長期趨勢股 |
| 日內選股 | 玉山帶量突破 | 短線飆股 |
| 持續性篩選 | 連續兩期動能前30% | 提升勝率（參考[[台灣持續性動能策略Momentum-Persistency]]） |
| 市值過濾 | 前200大 | 避免操控風險 |
| 籌碼確認 | 外資/投信/大戶買超 | 資金面確認 |

## 注意事項

- **動能策略的高波動**：TEJ回測年化波動37%，比大盤高很多，資金控管是關鍵
- **少數大贏撐起績效**：2021年陽明海運一次飆漲就撐起大部分報酬，過度依賴少數機會
- **系統性風險下失效**：2025年關稅戰期間動能策略比大盤差，趨勢反轉時動能策略受傷最重
- **盤中即時選股的限制**：玉山的條件是選股工具，不是完整策略，仍需搭配進出場紀律
- **漲跌幅限制影響**：台股±10%漲跌幅限制讓資訊無法一日內充分反映，影響動能效果（見[[台灣持續性動能策略Momentum-Persistency]]）
- **週轉率是核心問題**：台股約一半的贏家/輸家股票下一期就會反轉，純動能策略需要持續性濾網
