# 均值回歸策略

> 價格像橡皮筋，拉太遠終究會彈回來；但小心橡皮筋斷掉的那一刻

## 核心概念

均值回歸（Mean Reversion）策略的核心假設：成熟、流動性佳的金融資產，價格最終會圍繞歷史平均值上下波動。當價格因突發事件出現極端偏離，就像被拉扯的橡皮筋，拉得越遠（乖離越大），未來反彈回歸中心的力量越強。

### 兩種均值回歸

1. **時間軸均值回歸**：變數價值反彈回歷史平均
   - 例：CAPE達27.27倍偏貴，因歷史平均接近16倍

2. **跨類別均值回歸**：變數價值回歸跨類別平均
   - 例：本益比30倍的石油股偏貴，因石油類股平均PE接近15倍

## 實戰應用

### 三大進場訊號

**1. RSI極端值**
- 14日RSI跌破30 → 超賣區，觸發買進
- ⚠️ 調整參數時要避免過度優化（Over-optimization）

**2. 布林通道乖離邊界**
- 根據常態分配，價格95%機率在±2標準差內運行
- 日收盤價跌破布林通道下軌 → 極端乖離，觸發買進
- 布林通道本身就是移動平均線+標準差，與均值回歸概念高度契合

**3. 歷史均線乖離率（Bias Ratio）**
- Bias = (Close - MA) / MA × 100%
- 20日均線乖離率達到-5% → 自動買進
- 可配合不同週期均線計算乖離率

### 出場機制

**停利 — 回歸中值**：
- 目標不是抱著等翻倍，而是等價格回到中心平衡點
- 例：跌破布林下軌買進 → 觸碰中軌（20日均線）時賣出
- 勝率通常60%-70%

**停損 — 橡皮筋斷裂保護**：
- 公司基本面出現毀滅性打擊時，股價跌破後不反彈反而一路向下
- 強制停損：買進後繼續下跌超過總資金2%，無條件市價停損
- 這是均值回歸策略最容易遭遇滅頂之災的地方

### 趨勢追蹤搭配均值回歸

- 均值回歸策略勝率高，但單邊大趨勢時會連續停損（橡皮筋斷裂）
- 趨勢追蹤策略賠率高，但盤整時會連續假突破
- 成熟的系統化交易者同時納入兩者，在不同市場環境下截長補短
- 區間盤整 → 均值回歸策略表現好
- 趨勢行情 → 趨勢追蹤策略表現好

## 注意事項

### 均值回歸的三大陷阱

1. **「均值」的定義問題**：不同時間或跨類別樣本，計算出的均值不同
2. **「回歸」的時機不確定**：何時發生回歸？通過什麼方式回歸？（股價下跌 vs 獲利成長）
3. **結構性改變摧毀均值回歸**：
   - 企業老化：成長率自然下降，不再朝歷史常態移動
   - 科技與產業結構變化：如亞馬遜改變零售業常態
   - 投資人偏好改變：指數型基金量激增改變股價結構

### CAPE回測的教訓

- 1917-1966年：CAPE市場時機策略勝過買入持有
- 1967-2016年：CAPE策略輸給買入持有
- 買進訊號的報酬優於賣出訊號
- 統計顯著性 ≠ 投資報酬：相關性不是因果關係

### 避免過度優化

- 調整RSI參數（如把30改成28.5迎合歷史高勝率）要極度小心
- 過度優化導致策略在未來失效
- 必須做樣本外測試驗證策略穩健性

## 進階：統計檢定與配對交易

### 定常性（Stationarity）判斷

均值回歸的數學基礎是時間序列的「定常性」。定常序列的統計特性（均值、變異數）不隨時間改變，本質上就具備均值回歸特性。

透過自回歸模型 AR(1)：X(t) = φ × X(t-1) + ε(t)

- |φ| < 1 → 定常，具均值回歸特性
- |φ| = 1 → 隨機漫步，非定常
- |φ| > 1 → 發散

### ADF 檢定（Augmented Dickey-Fuller Test）

判斷單一資產是否適用均值回歸的統計工具：

- **虛無假設 H0**：序列存在單位根（隨機漫步，無均值回歸）
- **對立假設 H1**：序列不存在單位根（定常，有均值回歸）
- p-value < 0.05 → 拒絕 H0，序列具均值回歸特性

⚠️ 大部分單一金融資產的價格序列都是非定常的，直接應用均值回歸策略風險極高。

### 共整合與配對交易（Pairs Trading）

既然單一資產很難找到均值回歸，可以自己「創造」——找兩個非定常序列，其線性組合（價差）是定常的，即存在共整合關係。

**常見配對組合**：
- 同產業競爭對手：台積電 vs 聯電、可口可樂 vs 百事可樂
- 相似經濟驅動因素：AUD/JPY vs CAD/JPY（同為商品出口國）
- 不同級別公司債：AAA vs A

**共整合 ADF 檢定步驟（Engle-Granger）**：
1. 對兩個資產價格序列進行線性迴歸，得到殘差（價差）
2. 對殘差序列進行 ADF 檢定
3. 殘差定常（p < 0.05）→ 存在共整合，可進行配對交易

### Z-Score 判斷進出場

Z-Score = (當前價差 – 價差均數) / 價差標準差

- Z > 2.0 → 價差過度偏離，做空離，做空價差（賣A買B）
- Z < -2.0 → 價差過度偏離，做多價差（買A賣B）
- Z 回歸 0 附近 → 平倉

⚠️ 共整合關係可能因結構性變化而消失，歷史共整合 ≠ 未來共整合。必須持續監控價差定常性。
