# 開盤八法-開盤15分鐘判盤勢

> 開盤15分鐘的多空力道判斷，9:05/9:10/9:15三個時段的漲跌組合，預判當日行情走向

## 核心概念

開盤八法是根據歷史資料統計出的開盤判盤方法，透過觀察開盤後3個5分鐘時段（9:05、9:10、9:15）的漲跌組合，預判當日多空力道。

### 三個關鍵時段

- **(A)** 9:05開盤價 vs 前日收盤價
- **(B)** 9:10收盤 vs 9:05收盤
- **(C)** 9:15收盤 vs 9:10收盤

每個時段比較是漲還是跌，組合成8種型態。整天較多的成交量集中在開盤後15分鐘與收盤前15分鐘，因此開盤15分鐘就能大致判斷當日強弱。

### 為什麼成交量集中在開盤與收盤？

1. 許多投資人掛單後就上班去了，開盤不久即成交
2. 歐美收盤影響隔日開盤，超跌超漲造成開盤進出
3. 收盤前15分鐘是法人外資大單交易時間，當沖多空回補也在此時
4. 11:30前的數據較能接近當日高低點的真實位置

## 實戰應用

### 第一法：漲漲漲 — 多頭起漲或假突破

開盤後連續上攻，注意強勢拉抬後短線賣壓出籠。以9:10的收為不能跌破之基準點，拉回不破可作多。

- **開大漲**：利多消息刺激，開高震盪盤堅在高檔
- **開小漲**：無利多卻連續15分鐘拉抬，10點前跌破9:15未守穩 → 開高走低

### 第二法：漲漲跌 — 多頭上攻中段

10分鐘連漲後拉回，多頭短衝後暫時無力。觀察量能是否異常，以9:05的收為不破基準。

- **開大漲**：9:15跌的量不應大於9:10漲的量，否則跌破9:05機率高 → 佈空
- **開小漲**：日K多頭格局時盤中整理，9:15為今日多空點 → 佈多

### 第三法：漲跌跌 — 上檔有賣壓

5分鐘上漲後轉折向下，開盤上漲點數越大，拉回空間越大。

- **開小漲**：三個時段都是壓力區，突破作多，跌破做空
- **開大漲**：盤中無法突破9:15 → 尾盤多為小漲

### 第四法：跌漲漲 — 波段跌幅末端，有低接買盤

開低後轉折拉高，買方追價力道增加，底部有承接。

- **開小跌**：11:30前不能突破9:05 → 尾盤拉高機會大，隔日行情不差
- **開小跌**：10:00前突破9:15也突破開盤價 → 盤中持續拉抬機率高

### 第五法：跌跌漲 — 波段跌幅中段

10分鐘殺盤後有支撐買盤進場，以9:10的收為不破基準做低接。

- **開小跌**：反彈力道，短多操作
- **開大跌**：11:30前未能突破守穩昨收 → 尾盤收黑機率高

### 第六法：跌跌跌 — 波段跌幅尾端

開盤15分鐘連續下跌，通常是利空或巨大賣壓。反彈不見站上9:15 → 後市不樂觀。

- **開小跌**：反彈宜減碼，9:15低點不破可短多
- **開大跌**：隨即反彈且突破9:05以上 → 有買盤，明日可能還有高點

### 第七法：跌漲跌 — 高檔震盪

15分鐘內跌漲跌，多空膠著，狹幅震盪整理，短打為原則。

- **開小跌**：以9:15為基準多空線，突破前高短多、跌破前低短空
- **開大跌**：以9:10為基準做多線，破9:05反向作空

### 第八法：漲跌漲 — 多空短兵相接

15分鐘內漲跌漲，多空都有力道但振幅不大，抓價差不易，觀望或短多。

- **開大漲**：帶量突破前高 → 作多
- **開小漲**：帶量跌破9:10 → 作空，量平觀望

## 注意事項

- 開盤八法是統計機率，不是百分之百準確，需搭配其他技術分析
- 開盤15分鐘的多空力道只能做初步判斷，盤中任何消息都會影響盤勢
- 11:30前的數據比開盤15分鐘更接近當日高低點的真實位置
- 通常當日最高與最低多數發生在10:00~11:30之間
- 開盤大漲或大跌要區分是有利多利空消息還是自然走勢，判讀不同
