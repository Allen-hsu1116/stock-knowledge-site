# 台股大循環操作術

> 用GDP和景氣領先指標判斷台股多空，10年一個大循環，每個階段有不同的選股策略

## 核心概念

台股10年走一個大循環，背後推動力就是景氣循環。景氣循環終究是影響股市的最重要因素，可以用循環來判斷股市位於上升軌道或下降趨勢。

**關鍵邏輯**：不要見樹不見林。建議用「月K線」觀察加權指數，可看出七、八分的正確性。景氣循環不會一覺醒來就豬羊變色，一個小的循環週期動輒3-5年。

## 實戰應用

### 第一步：用GDP判斷多空年

- 年初利用GDP觀測未來一整年經濟走勢，為來年投資定調
- 每季跟隨主計處的預估數字修正
- **台股高點多半同步或領先GDP一季作頭**，落底也有同步或提早落底的特性

**判斷原則**：
- GDP逐季上揚（5%→6%→7%）→ 台股走高
- GDP從高點向下掉（8%→7%→6%）→ 即使絕對值仍好，台股可能提前反轉
- GDP變動幅度越大，台股上漲點數越多

**歷史驗證**：
- 2015-2017年GDP逐年增加（1.47%→2.17%→3.31%），台股上漲915→1,389點
- 2019-2021年GDP變動0.27%→0.3%→3.09%，台股上漲2,270→2,735→3,486點
- 2022年GDP預測值向下修正 → 空頭年

### 第二步：用景氣燈號確認循環階段

- 連續3個月藍燈 → 指數離低點不遠，站在買方
- 黃紅燈或紅燈 → 警戒行情將從高峰反轉，站在賣方
- 過去30年已獲得驗證

### 第三步：三大過熱指標確認高點

1. **大盤股價淨值比（P/B）**：
   - 大於1.9倍 → 賣出7成持股
   - 超過2倍 → 全部賣出
   - 跌至1.6倍 → 7成資金投入
   - 跌至1.3倍以下 → 持股100%

2. **巴菲特指標（市值/GDP）**：
   - 大於150% → 過熱
   - 120-140% → 合理價
   - 90-120% → 便宜價
   - 低於90% → 絕佳買點

3. **景氣燈號**：藍燈買、紅燈賣

### 第四步：階段性選股

| 循環階段 | 選股策略 | 具體方法 |
|---------|---------|---------|
| 復甦期 | 價值股優先 | 股價淨值比<1的龍頭股、高股息殖利率股、轉機股 |
| 繁榮期 | 成長股攻擊 | 營收持續3年成長3成、透視本益比 |
| 泡沫期 | 快狠準出場 | 寧可少賺不要貪、避開地雷股 |
| 空頭期 | 題材股反彈 | 短線題材、不戀戰 |

### 資金配置方法

月薪族把資金拆成12份（或每月薪水固定留下一筆）：
- 行情相對低檔時按月進場，持續下跌可加碼20%
- 行情衝高時只能買進6個月就要停

### 房地產循環的影響

- 房地產循環是景氣循環之母，約18年一個循環
- 房地產循環高峰過後，股市約落後18-24個月伴隨反轉向下
- 2022-2023年美國房地產循環高峰 → 2024年後要小心

## 注意事項

- **不要只看日K線**：用日K線看加權指數容易見樹不見林，月K線才是看大方向的正確工具
- **GDP方向比絕對值重要**：6%、7%的GDP數據看起來很好，但如果從8%往下掉，台股可能已經提前反映
- **政治利多只是暫時的**：兩岸關係、政治事件激情過後，一切都回歸基本面
- **2012年例外**：GDP數值低於前一年但台股卻是多頭，原因是GDP修正幅度已趨緩
- **巴菲特指標極端值**：2021年台股市值占GDP比重達264%，遠超巴菲特200%的「玩火」警告線
- **精準預測仍困難**：這些指標能幫你判斷相對高低，但不是精確的轉折點預測工具
