# Theta時間衰減實戰

> 時間是選擇權買方的敵人、賣方的朋友，Theta量化每天流失多少價值，價平選擇權最後30天衰減如雪崩，21DTE Rule是賣方的生存法則

## 核心概念

Theta（Θ）衡量選擇權價格對時間變化的敏感度，即「每過一天，選擇權價值減少多少」。在其他條件不變的情況下，隨到期日臨近，選擇權時間價值逐漸歸零，這個過程就是時間衰減（Time Decay）。

公式：Θ = ∂V/∂t

- V = 選擇權價格
- t = 時間

### Theta 的三大特性

1. **Theta 永遠為負（對買方）**：時間只會流逝不會倒流，選擇權的時間價值只會減少
2. **價平選擇權 Theta 最大**：因為價平的時間價值占比最高，每天流失的金額也最大
3. **衰減速度非線性**：越接近到期日，Theta絕對值越大，最後幾天如雪崩

### Theta 衰減的時間軸

| 剩餘天數 | 衰減速度 | 特徵 |
|---------|---------|------|
| 90天+ | 緩慢 | 每天流失很少，買方壓力小 |
| 60-90天 | 開始加速 | 時間價值穩定減少 |
| 30-60天 | 明顯加速 | 賣方甜蜜帶，Theta加速賺 |
| 15-30天 | 急速衰減 | 價平選擇權每天扣血明顯 |
| 0-15天 | 雪崩式歸零 | Theta最大，時間價值快速蒸發 |

## 實戰應用

### 買方的 Theta 生存法則

1. **速戰速決**：買方持有時間越長，Theta累計虧損越大，預期方向未兌現就要提前出場
2. **選擇較長到期日**：60天以上到期的選擇權Theta較小，給預期更多時間兌現
3. **避開最後10天**：價平選擇權最後10天的Theta衰減可能吃掉一半以上時間價值
4. **提前平倉**：當方向已兌現且獲利足夠，不要等到最後一刻讓Theta吃掉利潤

### 賣方的 Theta 武器

1. **賣30-45天到期**：這個區間Theta衰減開始加速，賣方每天坐收時間價值
2. **21DTE Rule**：在到期前21天平倉或Rolling，因為最後三週Gamma風險急速放大
3. **價平賣方Theta最大但Gamma也最大**：需要平衡Theta收入與Gamma風險
4. **鐵兀鷹/鐵蝴蝶**：用組合策略同時收割多個履約價的Theta

### 21DTE Rule 詳解

**為什麼是21天？**
- 接近到期時Gamma急速放大，賣方的「最大敵人」是Gamma不是Theta
- 21天是Gamma開始明顯放大的轉折點
- 繼續持有的Theta收益不足以補償Gamma風險

**執行方式**：
- 到期前21天：平倉獲利了結
- 如想繼續：Rolling到下個月相同履約價
- 避免「貪最後一塊錢」心態：最後幾天的Theta看似甜，但一次Gamma事件可吃掉整月利潤

### Theta 與其他 Greeks 的關係

| 關係 | 說明 |
|------|------|
| Theta vs Gamma | 一體兩面，高Theta伴隨高Gamma，賣方不能只看Theta |
| Theta vs Vega | IV擴張時Theta被壓縮，IV收縮時Theta加速收割 |
| Theta vs Delta | 深價內/深價外Theta小，價平Theta最大 |

### 實例計算

台指期履約價5200的買權，Theta = -4.0：
- 每靠近到期日一天，選擇權價格減少4點
- 7天不動 = 流失28點
- 最後一週如果大盤原地踏步，買方權利金蒸發驚人

價平買權價格5元，Theta = -0.05：
- 每天減少0.05元
- 30天後（若價格不動）= 減少1.5元（30%時間價值蒸發）
- 最後7天可能加速到每天-0.15元以上

## 注意事項

1. **Theta 只在其他條件不變時成立**：實際交易中標的物價格、IV都在變，Theta只是理論值
2. **週末Theta陷阱**：市場休市2天但選擇權時間價值仍在衰減，週五收盤時Theta效應被提前反映
3. **Theta不是線性**：不能簡單用每日Theta × 天數來算多天衰減，因為Theta本身每天都在變
4. **賣方不要被Theta誘惑**：高Theta = 高Gamma，看到Theta很大的賣方機會要先問「Gamma風險我承受得起嗎？」
5. **事件日Theta失效**：財報公布、FOMC會議前後，IV變動遠大於Theta衰減，不能單靠Theta判斷
6. **價平選擇權最後一天**：Theta衰減最劇烈，但同時Gamma也最大，賣方持有到最後一天等於裸奔
