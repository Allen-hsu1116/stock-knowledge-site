# 箱型理論 Darvas Box

> 股價超過一半時間在箱子裡震盪，找出箱頂箱底，突破箱頂買進、跌破箱底停損，是達華斯18個月從1萬變200萬的經典順勢策略

> 🔄 2026-05-14 進階更新：新增達華斯選股原則、階梯式操作法、假突破三道過濾、順勢vs逆勢箱型操作對比、末升段/末跌段注意事項

## 核心概念

箱型理論（Darvas Box Theory）由尼可拉斯·達華斯（Nicolas Darvas）提出，他原是舞蹈家，因客戶用股票支付酬勞而踏入股市，經歷大賠後自行研究出這套方法，18個月內將1萬美元滾成200萬美元。

**箱型是什麼**：股價在一段區間內上下震盪，相對高點形成賣壓（箱頂），相對低點形成支撐（箱底），價格在箱頂與箱底之間來回，就形成一個「箱子」。

**為什麼會形成箱型**：
- 箱頂賣壓來源：前波買在相對高點的投資人被套牢，當價格回到成本區想解套賣出
- 箱底支撐來源：主力護盤 + 技術分析者看到止跌跟進

**箱型確認條件**：
- 股價至少觸碰箱頂與箱底各2～3次
- 連續3個交易日不再突破前高 → 箱頂確立
- 連續3個交易日不再跌破前低 → 箱底確立
- 箱型持續時間越長，參考價值越高

**關鍵特性**：股價超過一半時間處於箱型整理，而非單邊趨勢。掌握箱型，就能掌握大部分時間的走勢。

## 實戰應用

### 一、達華斯原始操作法（順勢突破型）

1. **選股**：高成交量、有成長動能、處於上升趨勢的大型成長股
2. **找箱型**：在上升趨勢中找出箱型整理區間
3. **進場**：股價帶量突破箱頂時買進（預先掛單在箱頂上方一點）
4. **停損**：進場同時設停損在箱底下方，單次最大損失＝箱頂到箱底的距離
5. **移動停損**：股價進入下一個箱子後，停損點上移至新箱底
6. **加碼**：新箱頂突破時可加碼
7. **出場**：股價跌破當前箱底時出場

### 二、區間操作法（箱內低買高賣型）

適用條件：箱頂與箱底之間價差超過20%

- 接近箱底時逢低買入
- 接近箱頂時逢高賣出
- 停損設在箱底下方一點

### 三、確認真假突破

- **真突破**：帶量突破、收盤價站穩箱頂上方、出現多根中長紅K線
- **假突破**：盤中短暫突破但收盤跌回箱內、量能不足、突破後隔天立刻回落
- **應對**：假突破後果斷停損，不凹單；若再次突破可重新追回

### 四、階梯式上漲

上升趨勢中的箱型會呈現階梯狀：一個箱子突破後進入下一個更高的箱子，舊箱頂變成新箱底。只要股價不跌破前一個箱底就續抱，跌破則停損。

以台積電為例：566-593 → 突破 → 614-660 → 突破 → 700+ → 738-812 → 突破 → 818-873

### 五、支撐壓力互換原則

- 向上突破後：原箱頂（壓力）變成新箱底（支撐）
- 向下跌破後：原箱底（支撐）變成新箱頂（壓力）

## 注意事項

1. **只適用於上升趨勢**：在盤整或下跌趨勢中，箱型突破訊號模糊，誤判風險極高。達華斯是在牛市中開發此法，熊市能否複製存疑
2. **勝率低但期望值可為正**：準確度可能僅30-40%，但順勢策略不需高勝率，10次對3-4次即可有正期望值
3. **停損必須嚴格**：不嚴格停損，一次大虧可能吃掉所有小賺。「多次小虧損、一次大獲利」是核心邏輯
4. **不能小賺就跑**：順勢策略低勝率，小賺就出場會變成「小賺小賠低勝率」，長期必定虧損。必須讓利潤奔跑
5. **箱型範圍判斷主觀**：上下緣沒有絕對精確標準，可多抓0.3%誤差範圍減少假突破洗刷
6. **箱型越窄越有利**：停損範圍小，風險控制更容易
7. **冷門股不適用**：成交量過低的股票，主力可自行畫線，箱型理論失效
8. **滑價成本高**：順勢追價策略在突破瞬間常有滑價，需納入成本計算
9. **末升段/末跌段注意**：箱型出現在3波段走勢末端時，後續可能反轉為頭部或底部型態
10. **不好回測**：太多人為判斷因素（標的選擇、箱型範圍認定、假突破處理），難以量化驗證

## 達華斯選股原則

1. **有一定成交量**：冷門股和低流動性股票走勢易被主力操縱，箱型理論失效
2. **成長股或熱門產業股優先**：有成長動能的股票更容易出現階梯式箱型上漲
3. **股價接近歷史高點**：處於歷史新高或近兩三年相對高點的股票，箱型整理後放量突破的潛力最大
4. **只做多頭市場**：箱型理論在上升趨勢中勝率最高，熊市中不操作
5. **搭配基本面**：選擇財務狀況正在改善的公司，技術面與基本面共振

## 階梯式操作法（核心實戰）

達華斯的核心不是單一箱型，而是「階梯式上漲」：

### 初始進場
- 在箱頂上方一檔設定限價買單（預先掛單）
- 買進同時設定停損在箱底下方一檔
- 單次最大損失＝箱頂到箱底的距離

### 階梯追蹤
- 股價進入新箱型後，停損點上移至新箱底下方一檔（移動停損）
- 舊箱頂變成新箱底（支撐壓力互換原則）
- 只要股價不跌破前一個箱底就續抱

### 加碼原則
- 新箱頂帶量突破時可加碼
- 加碼後停損設在當前箱底下方

### 出場條件
- 股價跌破當前箱底 → 出場
- 箱型向下堆疊 → 停損出場

## 假突破過濾三道防線

### 1. 量能確認
- 向上突破必須伴隨成交量放大，否則為假突破機率高
- 無量突破不進場，多觀望

### 2. 收盤確認
- 盤中短暫突破不算數，必須收盤價站穩箱頂上方
- 建議連續3天確認

### 3. K線型態確認
- 突破日出現多根中長紅K線 → 真突破機率高
- 單根十字星或上影線長 → 可能為假突破

### 假突破後的處理
- 突破後隔天立刻回落 → 假突破，果斷停損
- 假突破被掃出後再次突破 → 可以重新追回，不預設立場

## 順勢 vs 逆勢箱型操作

| 面向 | 順勢操作（達華斯原始法） | 逆勢操作（箱內交易） |
|------|-------------------------|---------------------|
| 進場時機 | 帶量突破箱頂 | 箱底附近買入 |
| 停損位置 | 箱底下方 | 箱底下方2-3% |
| 獲利目標 | 下一個箱頂以上 | 箱頂附近 |
| 適合者 | 波段交易者 | 短線交易者 |
| 勝率 | 低（30-40%）但期望值正 | 較高但單次利潤小 |
| 風險 | 假突破洗刷 | 區間被突破 |

## 達華斯交易心法

1. **不要預測方向**：先觀察1～2個箱型區間的方向，再順勢操作
2. **不要主觀判斷好壞**：好與壞由市場決定
3. **抱牢上漲股，勇敢賣出下跌股**：多次小虧損、一次大獲利
4. **停損絕對嚴格**：不嚴格停損一次大虧可能吃掉所有小賺
5. **不要小賺就跑**：讓利潤奔跑，否則變成「小賺小賠低勝率」

## 末升段/末跌段注意

箱型出現在3波段走勢末端時，後續可能反轉為頭部或底部型態，需搭配其他指標確認。
