# 交易系統體制適應策略Adaptive-Trading-System

> 好的交易系統不是一套策略打天下，而是能根據市場體制動態切換策略——體制適應力決定長期存活率

## 核心概念

大多數交易系統只在特定市場體制下有效。趨勢追蹤系統在震盪盤中虧損連連，均值回歸系統在趨勢盤中被一路打穿。**體制適應策略（Regime-Adaptive Strategy）** 的核心思想是：用一套規則化的框架自動識別市場體制，然後切換到對應的最優策略。

這不是「預測市場方向」，而是「辨識當前環境，選擇正確工具」。

## 體制適應框架三層結構

### 第一層：體制偵測器（Regime Detector）

用多個指標共振判斷當前體制：

| 指標 | 趨勢訊號 | 震盪訊號 | 不明訊號 |
|------|---------|---------|---------|
| ADX | > 25 | < 20 | 20-25 |
| 均線排列 | 多頭/空頭排列 | 糾結 | 部分排列 |
| 布林帶寬 | 擴張 | 收縮 | 中性 |
| 價格位置 | 均線上方/下方 | 均線之間 | 接近均線 |
| 成交量 | 突破量 | 萎縮 | 正常 |

**判定規則**：
- 3個以上指標指向趨勢 → 趨勢體制
- 3個以上指標指向震盪 → 震盪體制
- 指標分歧 → 不明體制，減倉或觀望

### 第二層：策略切換器（Strategy Switcher）

| 體制 | 啟用策略 | 停用策略 | 部位大小 |
|------|---------|---------|---------|
| 趨勢上升 | 均線回測買進、VCP突破、順勢交易 | 區間交易、均值回歸 | 100%正常部位 |
| 趨勢下降 | 放空、反彈做空、順勢賣出 | 區間交易、均值回歸 | 100%正常部位 |
| 震盪 | 區間交易、均值回歸、網格交易 | 順勢交易、突破策略 | 70%正常部位 |
| 高波動混亂 | 選擇權賣方（僅低波動時）、極短線 | 中長線策略、大部位 | 30-50%正常部位 |
| 不明 | 觀望或小部位試探 | 所有完整部位策略 | 0-30%正常部位 |

### 第三層：風控過濾器（Risk Filter）

無論體制為何，風控規則永不關閉：
- 單筆最大虧損：帳戶的 1-2%
- 日最大虧損：帳戶的 3-5%
- 連續虧損3筆 → 自動減倉50%
- 連續虧損5筆 → 自動停止交易，重新評估體制

## 體制切換的實戰時機

### 從震盪轉趨勢的觸發條件

1. **價格突破震盪區間上緣或下緣**（帶量確認）
2. **ADX 從 20 以下突破 25**
3. **均線從糾結開始發散**
4. **布林通道 Squeeze 後帶寬急速擴張**

→ 切換動作：關閉區間策略，啟動順勢策略

### 從趨勢轉震盪的觸發條件

1. **價格連續3次突破失敗**
2. **ADX 從 25 以上跌破 20**
3. **均線開始糾結**
4. **布林帶寬收縮至近3個月最低**

→ 切換動作：關閉順勢策略，啟動區間策略

### 從任何體制轉高波動的觸發條件

1. **ATR 突然放大至 20 日均值的 2 倍以上**
2. **VIX 跳升超過 30%**
3. **連續出現跳空缺口**
4. **出現重大事件（央行會議、戰爭、黑天鵝）**

→ 切換動作：所有策略部位減半，停損設緊，準備應急方案

## 台股體制判斷實戰範例

### 範例：加權指數體制判斷

**Step 1：看大方向（週線）**
- 週KD位置：高位鈍化還是低位交叉？
- 20週均線方向：上揚、走平、下彎？
- 週成交量：是否持續放大？

**Step 2：確認中節奏（日線）**
- 日ADX讀數與方向
- 5日/20日/60日均線排列
- 日K線型態：趨勢型還是震盪型？

**Step 3：找進場點（60分鐘）**
- 細部支撐壓力
- 進場時機與停損位置
- 部位大小根據體制確認度調整

**綜合判斷**：
- 週線趨勢 + 日線趨勢 → 信心高，正常部位
- 週線趨勢 + 日線震盪 → 拉回找買點，部位略減
- 週線震盪 + 日線震盪 → 觀望或極短線
- 週線趨勢 + 日線反轉 → 可能是體制轉換，減倉觀察

## 體制適應的常見陷阱

1. **過度切換**：每天換策略等於沒有策略，體制判斷至少以週為單位
2. **延遲認帳**：體制已經轉變卻不願承認，繼續用舊策略虧損
3. **指標過多**：超過5個指標判斷體制會產生更多衝突訊號
4. **忽略時間框架**：日線趨勢和週線震盪可以同時存在，操作哪個就聽哪個的
5. **完美主義**：沒有100%確定的體制判斷，70%信心就足夠行動
6. **策略不完整**：切換到新策略但沒有完整的進出場規則和停損機制

## 體制適應策略的回測注意事項

1. **必須包含體制切換成本**：換策略的滑價和手續費
2. **不能只在趨勢期回測趨勢策略**：要包含震盪期的表現
3. **體制判斷本身也有延遲**：通常落後1-3根K線
4. **參數過度優化的風險**：體制判斷的閾值（如ADX 20/25）不應過度調整
5. **要測試錯誤判斷的影響**：假設體制判斷錯誤時最大虧損是多少
