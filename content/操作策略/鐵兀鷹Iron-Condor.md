# 鐵兀鷹 Iron Condor 中性策略

> 鐵兀鷹是選擇權賣方的經典中性策略，在盤整盤中收權利金，最大獲利 = 收到的權利金。

## 核心概念

### 組成
同時賣出一組看漲價差 + 一組看跌價差：
- 賣出 OTM Call + 買入更 OTM Call（看漲價差）
- 賣出 OTM Put + 買入更 OTM Put（看跌價差）

### 四條腿
1. 買 Call（最高履約價）— 限制上方風險
2. 賣 Call（次高履約價）— 收權利金
3. 賣 Put（次低履約價）— 收權利金
4. 買 Put（最低履約價）— 限制下方風險

### 風險報酬
- **最大獲利**：收到的淨權利金
- **最大虧損**：履約價間距 - 淨權利金
- **損益兩平點**：上損益兩平 = 賣 Call 履約價 + 淨權利金；下損益兩平 = 賣 Put 履約價 - 淨權利金

## 實戰應用

1. **進場時機**：IV 高時賣（IV > 30%），預期盤整
2. **Delta Neutral**：調整履約價間距使 Delta 接近 0
3. **調整策略**：價格接近某一邊時，可以買回那一邊的價差（Roll）
4. **時間衰減**：Theta 是最大盟友，越接近到期日獲利越快

## 注意事項

- 鐵兀鷹勝率高（80%+）但單次獲利有限
- 一次大虧損可能吃掉數月獲利
- 必須設停損：虧損達 2 倍權利金就出場
- 財報週、FOMC 會議等事件前不要做
- 搭配 IV 判讀：高 IV 賣、低 IV 不做
