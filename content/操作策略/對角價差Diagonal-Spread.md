# 對角價差Diagonal-Spread

> 垂直價差+時間價差的混合體，同時賺方向和時間——但維度越多，管理越難

## 核心概念

對角價差（Diagonal Spread）是選擇權價差策略中「履約價不同+到期日不同」的組合，兼具垂直價差的方向性和時間價差的時間衰減獲利。

### 與其他價差的定位

| 策略 | 履約價 | 到期日 | 獲利引擎 |
|------|--------|--------|----------|
| 垂直價差 | 不同 | 相同 | 方向價差 |
| 時間價差 | 相同 | 不同 | Theta差+Vega差 |
| **對角價差** | **不同** | **不同** | **方向+時間雙重** |

### 兩種基本型態

1. **Long Diagonal Call Spread**（看多對角買權價差）
   - Buy 遠月低履約價 Call + Sell 近月高履約價 Call
   - 溫和看多 + 賺近月時間價值消逝
   - 風險有限，最大虧損＝淨權利金

2. **Long Diagonal Put Spread**（看空對角賣權價差）
   - Buy 遠月高履約價 Put + Sell 近月低履約價 Put
   - 溫和看空 + 賺近月時間價值消逝
   - 風險有限，最大虧損＝淨權利金

3. **Short Diagonal**（反向操作）
   - 裸賣遠月風險大，散戶不建議

### 核心獲利邏輯

對角價差同時捕獲兩個維度的獲利：
- **方向維度**：遠月內含價值隨行情走有利的方向增加
- **時間維度**：近月OTM部位的時間價值消逝比遠月快，賣方收取權利金

## 實戰應用

### 進場時機
1. 溫和看多（或看空），預期行情緩步往某方向走
2. 近月預期盤整或小波動，遠月有方向性看法
3. IV在中等偏低時進場（避免IV驟降傷遠月Vega）

### 履約價選擇
- **遠月（買方）**：ATM或輕微ITM，保留內含價值
- **近月（賣方）**：OTM 1-2檔，時間價值消逝快且歸零機率高

### Rolling策略——對角價差的殺手鐧

近月結算後，遠月部位仍在：
1. **繼續賣下一期近月OTM Call/Put** → 變成持續收取權利金的機器
2. 類似Covered Call的升級版：用遠月長期Call替代持有股票，成本更低
3. 每次Rolling收權利金，逐月降低成本基礎

### 台指選實戰配置
- 遠月：月選或季選（流動性相對好）
- 近月：周選或雙週選（Theta衰減快，收權利金效率高）
- 履約價間距：至少200-300點（台指選），確保有方向性獲利空間

### Delta中性操作
- 調整買賣口數比例，使整體Delta接近0
- 變成純波動率交易：IV上升時遠月Vega大漲獲利，近月時間衰逝獲利
- 適合不確定方向但預期IV會上升的情境

## 注意事項

1. **波動率驟降風險**：遠月Vega大，IV壓縮時遠月貶值速度快，可能吃掉所有Theta獲利
2. **方向判斷錯誤**：近月OTM保護有限，行情大幅反向時遠月內含價值大減
3. **流動性風險**：遠月OTM合約流動性差，滑點成本高，尤其台指遠月價差大
4. **複雜度高**：需同時管理方向、時間、波動率三維度，不適合選擇權新手
5. **正逆價差影響**：跨合約部位受期貨正逆價差變化影響（與時間價差相同問題）
6. **保證金計算**：台指選以組合部位計算，但仍需準備充足保證金
7. **結算管理**：近月結算前必須決定平倉或Rolling，不能放任不管

### 與Covered Call的比較

| 項目 | Covered Call | Diagonal Spread |
|------|-------------|-----------------|
| 標的 | 持有股票+賣Call | 買遠月Call+賣近月Call |
| 資金需求 | 高（需買股票） | 低（只付權利金差） |
| 上方獲利 | 有限 | 有限但可Rolling |
| 下方風險 | 股票跌價風險 | 限於淨權利金 |
| 靈活度 | 低 | 高（可隨時調整） |
