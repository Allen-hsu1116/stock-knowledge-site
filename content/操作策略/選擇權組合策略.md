# 選擇權組合策略

> 單一策略的進階版，透過多個選擇權部位的組合，精準對應不同市場看法——價差、跨式、勒式、時間價差各有適用場景

## 核心概念

選擇權組合策略是將多個單一部位組合，創造出單一策略無法達到的風險報酬結構。主要分三大類：

1. **價差策略(Spreads)**：同標的同到期日，不同履約價
2. **時間價差(Calendar Spreads)**：同標的同履約價，不同到期日
3. **混合策略(Combinations)**：同標的同到期日，Call+Put組合

## 實戰應用

### 執行價格價差(Price Spreads)

同時買賣到期日相同但履約價不同的Call或Put。

**看多價差(Bull Call Spread)**：
- 買進低履約價Call + 賣出高履約價Call
- 適用：溫和看多，想降低權利金成本
- 最大獲利 = 高履約價 - 低履約價 - 淨權利金
- 最大虧損 = 淨權利金

**看空價差(Bear Put Spread)**：
- 買進高履約價Put + 賣出低履約價Put
- 適用：溫和看空，想降低權利金成本
- 風險報酬比優於單純Buy Put

**範例**：
- 買進十月份5100點Put(支付170點) + 賣出十月份5300點Put(收取260點)
- 淨收權利金90點(50×90=4500元)
- 最大獲利：4500元
- 最大虧損：50×110=5500元
- 保證金：50×(5300-5100)=10000元

### 時間價差(Calendar Spreads)

同時買賣履約價相同但到期日不同的選擇權。

**操作方式**：
- 買進6月5000 Call + 賣出7月5000 Call

**優點**：
- 合成部位起始投資金額較少
- 具財務槓桿效果
- 適合盤整格局

**6月到期時損益**：
- 大盤>5000：執行5000 Call買進標的，7月合約獲得時間價值
- 大盤<5000：5000 Call不被執行，7月合約依舊獲得時間價值

### 跨式組合(Straddles)

同時買進(或賣出)相同到期日與履約價的Call+Put。

**買進跨式(Long Straddle)**：
- 買進Call + 買進Put，同履約價同到期日
- 適用：預期大波動但方向不明
- 成本高(雙邊權利金)，需要大幅波動才能獲利
- 最大虧損 = 雙邊權利金

**賣出跨式(Short Straddle)**：
- 賣出Call + 賣出Put，同履約價同到期日
- 適用：預期盤整，波動縮小
- 最大獲利 = 雙邊權利金
- 風險無限，需要嚴格停損

**範例**：賣出十月份5300點Call(收125點) + 賣出十月份5300點Put(收100點)
- 收取225點權利金(50×225=11250元)
- 保證金 = Max(Call保證金, Put保證金) + 較少方權利金市值

### 勒式組合(Strangles)

同時買進(或賣出)相同到期日但不同履約價的Call+Put。

**與跨式的差異**：
- 跨式：同履約價(價平)
- 勒式：不同履約價(通常都選價外)

**買進勒式(Long Strangle)**：
- 買進價外Call + 買進價外Put
- 權利金成本比跨式低
- 需要更大波動才能獲利
- 適用：預期超大波動

**賣出勒式(Short Strangle)**：
- 賣出價外Call + 賣出價外Put
- 獲利空間比跨式大(區間更寬)
- 但風險也更大

## 注意事項

1. **組合策略不是免費午餐**：價差策略降低成本但也限制獲利，沒有「穩賺不賠」的組合
2. **流動性風險**：組合策略涉及多個部位，流動性差的合約會增加滑價
3. **保證金計算**：組合部位的保證金不一定等於各部位保證金加總，有組合保證金減免
4. **平倉順序**：組合策略平倉時要注意流動性，先平流動性差的部位
5. **時間衰減不同步**：不同履約價和到期日的Theta不同，組合的時間衰減不是簡單加總
6. **波動率微笑**：不同履約價的隱含波動率不同，影響組合策略的定價和損益
