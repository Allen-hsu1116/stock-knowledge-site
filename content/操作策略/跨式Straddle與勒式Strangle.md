# 跨式 Straddle 與勒式 Strangle

> 跨式和勒式是選擇權波動率策略，預期大波動但方向不確定時使用。

## 核心概念

### 跨式 Straddle
- 同時買入相同履約價的 Call + Put
- 履約價 = ATM（價平）
- 成本較高，但只要大波動就獲利
- 損益兩平：上方 = 履約價 + 總權利金；下方 = 履約價 - 總權利金

### 勒式 Strangle
- 同時買入 OTM Call + OTM Put
- 成本較低，但需要更大的波動才能獲利
- 風險較小但勝率也較低

### 買方 vs 賣方

| | 買跨式/勒式 | 賣跨式/勒式 |
|---|---|---|
| 預期 | 大波動 | 盤整理 |
| 最大獲利 | 無限（理論） | 權利金收入 |
| 最大虧損 | 權利金支出 | 無限（理論） |
| 適合時機 | 財報前、重大事件前 | IV 高、預期盤整 |

## 實戰應用

1. **買跨式時機**：IV 低 + 預期大波動（財報、法會、選舉）
2. **賣跨式時機**：IV 高 + 預期盤整（事件結束後 IV 回落）
3. **IV Crush 風險**：事件前買入，事件後 IV 崩跌可能賠錢
4. **調整策略**：獲利後可以只平一邊（Call 或 Put），讓另一邊奔跑

## 注意事項

- 買跨式最大敵人是時間衰減（Theta）
- 賣跨式最大敵人是黑天鵝（無限虧損風險）
- 財報前 IV 高，買跨式成本高，可能 IV Crush 後賠錢
- 實務上賣勒式（鐵兀鷹）比賣跨式更安全
