# 台股交易制度與漲跌幅限制

> 台股個股每日漲跌幅10%，新上市首5日無限制；交易制度從開盤到收盤有集合競價、逐筆交易、瞬間價格穩定措施三層機制，搞懂規則才不會在關鍵時刻犯錯。

## 核心概念

### 漲跌幅限制

台股每日漲跌幅以**前一日收盤價（開盤競價基準）±10%**為限：

- 計算方式：漲停價 = 基準價 × 110%，跌停價 = 基準價 × 90%
- 升降單位須符合規定，未滿一升降單位以一單位計
- 例：基準價40.60元，漲停價44.65元（非44.66因不符合0.05升降單位）

**特殊漲跌幅規定**：

| 證券種類 | 漲跌幅 |
|---------|--------|
| 一般股票、ETF | ±10% |
| 槓桿反向ETF（2倍型） | ±20% |
| 公司債 | ±5% |
| 新上市普通股首5日 | 無限制 |
| 外國股票第二上市 | 無限制 |
| 追蹤國外指數ETF | 無限制 |
| 權證 | 依標的漲跌幅×行使比例 |

### 漲停鎖死與跌停鎖死

- **漲停鎖死**：大量買單掛在漲停價，幾乎沒有賣單 → 買得到但成本已貴10%
- **跌停鎖死**：大量賣單掛在跌停價，幾乎沒有買單 → 想賣賣不掉，流動性喪失

漲跌停 ≠ 停止交易，價格被凍住但仍可下單，只是流動性極差。

### 各國漲跌幅/熔斷比較

| 國家 | 機制 |
|------|------|
| 台灣 | 個股±10%，新上市首5日無限制 |
| 美國 | S&P500跌7%/13%暫停15分鐘，跌20%停止交易；個股無限制 |
| 中國 | 個股±10%，科創板±20% |
| 韓國 | KOSPI跌10%暫停10分鐘，個股±15% |
| 日本 | 波動超一定幅度停10分鐘（依股價不同） |
| 香港 | 5分鐘內漲跌10%觸發5分鐘冷靜期 |

## 實戰應用

### 升降單位（跳動單位）與掛價技巧

升降單位決定了每一「檔」的價格間距，直接影響掛單策略：

| 股價區間 | 升降單位 |
|---------|---------|
| <5元 | 0.01元 |
| 5-10元 | 0.05元 |
| 10-50元 | 0.05元 |
| 50-100元 | 0.10元 |
| 100-150元 | 0.50元 |
| 150-500元 | 1.00元 |
| 500-1000元 | 1.00元 |
| >1000元 | 5.00元 |

**掛價技巧**：高價股升降單位大（1000元以上每檔5元），掛單要考慮滑價風險；低價股每檔0.01-0.05元，精確掛單相對容易。

### 六種委託方式與使用時機

| 委託方式 | 說明 | 適用時段 | 實戰用途 |
|---------|------|---------|---------|
| 限價ROD | 指定價格，當日有效 | 開盤/盤中/收盤 | 最常用，一般下單 |
| 限價IOC | 指定價格，立即成交否則取消 | 盤中 | 不想掛單排隊 |
| 限價FOK | 指定價格，全部成交否則取消 | 盤中 | 大單不想部分成交 |
| 市價ROD | 市價，當日有效 | 盤中 | 急著成交不管價格 |
| 市價IOC | 市價，立即成交否則取消 | 盤中 | 急成交+不排隊 |
| 市價FOK | 市價，全部成交否則取消 | 盤中 | 急成交+全部或沒有 |

**注意**：開盤和收盤只接受限價ROD，市價單和IOC/FOK會被退單。

### 瞬間價格穩定措施（盤中熔斷）

- **觸發條件**：限價ROD試算成交價超過參考價±3.5%
- **9:00-9:05**：以開盤價±3.5%
- **9:05-13:25**：前5分鐘加權平均價±3.5%
- **觸發後**：暫緩撮合2分鐘，之後以集合競價撮合
- **例外**：權證、新上市首5日、低價股（基準<1元）不適用

### 漲跌停時的實戰判斷

**手上有漲停股**：不建議急賣，隔天續漲機率較高，但要觀察量能是否持續。

**手上有跌停股**：想賣也賣不掉，冷靜判斷跌停原因——是個股基本面惡化還是市場恐慌？買進理由是否消失？

**想追漲停股**：成本已貴10%，且可能買不到，除非有明確連續漲停邏輯。

**想接跌停股**：只有當跌停原因非長期因素、營運本質未受影響、且你對標的足夠了解時才考慮。

**漲跌停鎖死的替代方案**：
1. **盤後定價交易**：以收盤價成交，但流動性有限
2. **海外ADR**：如台積電ADR在美股交易，無漲跌幅限制但價格可能超漲
3. **衍生性商品**：個股期貨、選擇權、權證可能仍有流動性
4. **關聯標的**：找走勢高度相關的替代股

### 零股交易制度

| 項目 | 盤中零股 | 盤後零股 |
|------|---------|---------|
| 交易時間 | 9:00-13:30 | 13:40-14:30送單，14:30撮合 |
| 撮合方式 | 每5秒撮合一次 | 集合競價一次撮合 |
| 成交機會 | 多次 | 一次 |
| 委託保留 | 未成交不保留至盤後 | — |
| 升降單位 | 比照普通交易 | 比照普通交易 |

## 注意事項

1. **新上市首5日無漲跌幅**：IPO首5天價格可能劇烈波動，風險極高，瞬間價格穩定措施也不適用
2. **市價單風險**：市價單優先於限價單成交，但成交價可能遠超預期，尤其在流動性差的股票上
3. **開收盤只能用限價ROD**：市價單、IOC、FOK在開盤和收盤時段會被退單
4. **暫緩開盤/收盤機制**：開盤前1分鐘（8:59-9:00）和收盤前1分鐘（13:29-13:30）如果模擬價格漲跌超3.5%，會暫緩2分鐘
5. **升降單位影響掛單精度**：高價股每檔5元，滑價風險大；零股交易升降單位與整股相同
6. **漲跌停≠停止交易**：價格被鎖住但可以繼續掛單，只是流動性極差
7. **權證漲跌幅特殊**：權證漲跌幅依標的證券計算，可能遠超10%，但認售權證跌停=標的漲停×行使比例
8. **盤後定價交易以收盤價成交**：無法指定價格，全數以當日收盤價撮合
