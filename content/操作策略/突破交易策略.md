# 突破交易策略

> 避開盤整、專抓趨勢起漲點，核心是判斷真假突破，量級分層+籌碼驗證+分批建倉是完整框架

## 核心概念

突破交易是避開盤整區間，等到股價突破壓力區間後再進場的策略。基於統計學與心理學原理：價格區間由交易者共識形成，新利多吸引新買盤突破共識，進入新價格帶。

### 盤整時間與波段目標的對應

| 波段目標 | 所需盤整時間 |
|----------|-------------|
| 1個月波段 | ≥ 3個月 |
| 半年波段 | 1～2年 |
| 1年以上大波段 | ≥ 5年 |

**邏輯**：冷門股沉寂越久，一旦爆發力道越強。長期無利多的標的突然出現基本面變化（虧轉盈、業績暴增），突破的爆發力最強。

### 適合的標的特徵

- 平常無人關注的冷門股
- 一有利多就能吸引市場目光
- 原本處於景氣循環低點的傳產股
- 例：疫情時的航運股，從冷門到全民追逐

### 成交量創新高的4個量級（2026實戰升級）

「量先價而行」，成交量創新高是主力表態的身分證。不同量級對應不同操作強度：

| 量能層級 | 週期對應 | 訊號強度 | 實戰解讀 |
|---------|---------|---------|----------|
| 20日新高量 | 月線級 | 中等 | 雜訊多，多屬隔日沖或避險資金，先觀察不急出手 |
| 60日新高量 | 季線級 | 較強 | 一季套牢區被消化，季線轉折向上可進場1/3部位 |
| 120日新高量 | 半年線級 | 高 | 強烈突破，通常伴隨產業景氣循環轉折，進場1/2部位 |
| 一年新高量 | 年線級 | 極強 | 歷史級別表態，多為主升段啟動點，可建立滿檔部位 |

⚠️ **反向訊號**：高位階出現「年新高量」卻收長上影線或長黑K，不是攻擊而是頭部訊號，主力趁亂大量出貨，必須立即出清。同樣的「量創高」，在底部是發動機，在頭部是斷頭台。

## 實戰應用

### 真突破四大確認條件

1. **帶量突破**：成交量至少是近期平均量的1.5倍以上（有效標準），量能達均量3倍以上為強訊號
2. **收盤站上壓力線**：盤中突破不算，必須收盤確認
3. **連續3天確認**：突破後連續3個交易日收在壓力線之上
4. **回測不破**：拉回測試壓力線（已變支撐）不跌破

### 有效突破的價量雙重標準（財訊方天龍）

- **價格標準**：穿越頸線、趨勢線或箱型上方達**3%以上**，才算是「有效突破」
- **量能標準**：當天預估量是成交量均量的**1.5倍以上**
- **向上突破**4原則：突破必有量 → 回測量縮 → 阻力變支撐站穩 → 量再度擴張時介入
- **向下突破**特點：可以無量下跌、可以爆量下跌、回測也可無量

### 籌碼4步驗證：過濾假突破（2026實戰升級）

看到量創高後，透過4步籌碼篩檢過濾主力對敲的虛假量能：

1. **分點券商前15大買賣超**：前5大買超合計 > 賣超總和的1.5倍，集中在內資主力分點（凱基台北、富邦發行等），代表資金延續性
2. **外資+投信是否同買**：法人同買勝率比單一法人高出22%。理想狀態：外資買+投信買+自營商避險性買入。僅自營商權證避險而外資倒貨 = 虛火
3. **融資增加vs融券回補**：最健康「資減券增」或「資增券減」動態平衡。量大+融資暴增+股價收不上去 = 散戶接火。券減空方回補 = 軋空助漲
4. **董監持股+集保戶數變化**：總股東人數連續3週減少 + 千張大戶持股比上升 = 籌碼由散戶流向大戶，真突破機率85%以上

**判讀準則**：4條全中 = 真突破（重倉參與）；中2條 = 弱突破（僅短線價差）；少於2條 = 雜訊（刪除自選）

### 假突破四大警訊

1. **量能不足**：突破時量能沒有明顯放大
2. **一日行情**：突破後隔天立刻跌回壓力線之下
3. **長上影線**：日K線留長上影線，上方賣壓沉重
4. **消息面薄弱**：缺乏基本面利多支撐

**假突破的成因**：主力高檔出不掉貨，利用震盪期間「量滾量」出貨，散戶以為股價又要動了盲目跟進，次日衝高回落跌破盤整平台。

### 動能延續性過濾（FinLab回測驗證）

單日創新高容易是假突破，**連續創新高才是真動能**：

- **基本策略**：股價創近200日新高 → 年化17.7%、夏普率約0.6
- **優化策略**：近5日內有3日以上創200日新高 → 年化25.9%、夏普率1.1

關鍵洞察：連續挑戰新高的次數越多，動能越強。適度放寬時間條件（不只看當天），反而能捕捉更穩定的突破訊號。

### 4階段波段建倉SOP

量創高訊號出現後，分批建倉，不一次All in：

| 階段 | 條件 | 動作 | 防守 |
|------|------|------|------|
| 1. 年量創高突破前高 | 量能≥3倍均量，籌碼4步驗證齊備 | 進場1/3部位 | 停損：突破點下緣-5% |
| 2. 量縮回測月線不破 | 突破後5-10天洗盤，量極度萎縮 | 股價碰月線站穩，加碼1/3 | — |
| 3. 再次量增續創高 | 主力再點火，突破第一階段高點 | 最後補倉1/3 | 停損上移至回測低點 |
| 4. 趨勢反轉 | 跌破月線量增 or 漲幅>50%後爆量長黑 | 全數出清 | 落袋為安 |

### 持續突破 vs 反轉判斷

**持續突破訊號**：
- 均線多頭排列（5MA > 20MA > 60MA）
- 回檔量縮，賣壓減輕
- 法人持續買超

**反轉警訊**：
- 爆大量但漲幅有限（量價背離）
- 高檔十字線或長上影線
- 均線糾結或下彎
- 法人轉賣超
- 爆量長黑 + 隔日沖分點為主 = 假突破

### 進場SOP（完整版）

1. 找出盤整區間（高點壓力、低點支撐）
2. 等待帶量突破壓力線（量≥1.5倍均量，穿越3%以上）
3. 確認收盤站上壓力線
4. 觀察2-3天是否持續站穩
5. 回測壓力線不破 → 階段1進場1/3
6. 量縮回測月線不破 → 階段2加碼1/3
7. 再度帶量創高 → 階段3補倉1/3
8. 停損設在壓力線下方（突破失敗即出場）

## 注意事項

- **不要追單日突破**：單日創新高很容易是假突破，至少觀察動能延續性
- **盤整時間不夠不要碰**：盤整時間短代表籌碼沉澱不足，突破後容易回落
- **量能是關鍵**：沒量的突破等於沒燃料的火箭，飛不起來
- **時間條件越死越吃運氣**：只看某一天是否突破，容易受到選股日隨機性影響
- **突破失敗要認賠**：假突破的特徵出現就該出場，不要硬凹
- **突破後的回測是正常的**：不要因為回測就恐慌出場，但回測跌破支撐就要停損
- **高位階量創高要警覺**：同樣的量創高，底部是發動機，頭部是斷頭台
- **籌碼驗證不可省略**：只看價量不驗籌碼，容易被主力對敲的虛假量能騙
- **技術面+籌碼面雙重確認**：現代技術分析遠不如「技術面+籌碼面」綜合判斷可靠
