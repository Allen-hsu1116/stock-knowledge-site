# 突破交易策略

> 避開盤整、專抓趨勢起漲點，核心是判斷真假突破

## 核心概念

突破交易是避開盤整區間，等到股價突破壓力區間後再進場的策略。基於統計學與心理學原理：價格區間由交易者共識形成，新利多吸引新買盤突破共識，進入新價格帶。

### 盤整時間與波段目標的對應

| 波段目標 | 所需盤整時間 |
|----------|-------------|
| 1個月波段 | ≥ 3個月 |
| 半年波段 | 1～2年 |
| 1年以上大波段 | ≥ 5年 |

**邏輯**：冷門股沉寂越久，一旦爆發力道越強。長期無利多的標的突然出現基本面變化（虧轉盈、業績暴增），突破的爆發力最強。

### 適合的標的特徵

- 平常無人關注的冷門股
- 一有利多就能吸引市場目光
- 原本處於景氣循環低點的傳產股
- 例：疫情時的航運股，從冷門到全民追逐

## 實戰應用

### 真突破四大確認條件

1. **帶量突破**：成交量至少是近期平均量的1.5倍以上
2. **收盤站上壓力線**：盤中突破不算，必須收盤確認
3. **連續3天確認**：突破後連續3個交易日收在壓力線之上
4. **回測不破**：拉回測試壓力線（已變支撐）不跌破

### 假突破四大警訊

1. **量能不足**：突破時量能沒有明顯放大
2. **一日行情**：突破後隔天立刻跌回壓力線之下
3. **長上影線**：日K線留長上影線，上方賣壓沉重
4. **消息面薄弱**：缺乏基本面利多支撐

### 動能延續性過濾（FinLab回測驗測驗證）

單日創新高容易是假突破，**連續創新高才是真動能**：

- **基本策略**：股價創近200日新高 → 年化17.7%、夏普率約0.6
- **優化策略**：近5日內有3日以上創200日新高 → 年化25.9%、夏普率1.1

關鍵洞察：連續挑戰新高的次數越多，動能越強。適度放寬時間條件（不只看當天），反而能捕捉更穩定的突破訊號，也不會錯過「前幾天突破、近期小拉回」的強勢股。

### 持續突破 vs 反轉判斷

**持續突破訊號**：
- 均線多頭排列（5MA > 20MA > 60MA）
- 回檔量縮，賣壓減輕
- 法人持續買超

**反轉警訊**：
- 爆大量但漲幅有限（量價背離）
- 高檔十字線或長上影線
- 均線糾結或下彎
- 法人轉賣超

### 進場SOP

1. 找出盤整區間（高點壓力、低點支撐）
2. 等待帶量突破壓力線
3. 確認收盤站上壓力線
4. 觀察2-3天是否持續站穩
5. 回測壓力線不破 → 進場
6. 停損設在壓力線下方（突破失敗即出場）

## 注意事項

- **不要追單日突破**：單日創新高很容易是假突破，至少觀察動能延續性
- **盤整時間不夠不要碰**：盤整時間短代表籌碼沉澱不足，突破後容易回落
- **量能是關鍵**：沒量的突破等於沒燃料的火箭，飛不起來
- **時間條件越死越吃運氣**：只看某一天是否突破，容易受到選股日隨機性影響
- **突破失敗要認賠**：假突破的特徵出現就該出場，不要硬凹
- **突破後的回測是正常的**：不要因為回測就恐慌出場，但回測跌破支撐就要停損
