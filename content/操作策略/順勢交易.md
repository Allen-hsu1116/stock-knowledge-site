# 順勢交易 Trend Trading

> 不預測市場方向，只跟著趨勢走。突破壓力做多、跌破支撐做空，核心是紀律和停損。

## 核心概念

順勢交易的核心信念：**市場趨勢有持續性**。上升趨勢中價格傾向繼續上升，下降趨勢中傾向繼續下降。交易者不需要預測市場走向，只需要辨識趨勢然後跟著走。

Michael W. Covel（TrendFollowing.com 創辦人）：「趨勢跟蹤不能預測市場走向，而是強調自律與精準的交易規則，還有嚴密的風控以應對市場波動性。」

### 四步操作流程

1. **辨識趨勢**：使用趨勢線、均線排列、技術指標判斷多空方向
2. **確定進出場**：突破支撐壓力進場，回到均線或達目標出場
3. **設定停損**：進場就設，不得事後修改
4. **追蹤調整**：移動停損、加減碼、依市況調整

### 三種順勢操作方法

**均線順勢**：最直覺的順勢方法。多頭排列做多、空頭排列做空。均線本身就是順勢指標，跟著均線的方向交易。

**KD/RSI 順勢用法**：打破傳統超買超賣思維，改用50作為多空分水嶺。KD/RSI > 50 做多，< 50 做空。出場用黃金/死亡交叉或超買超賣區。

**亞當理論**：看到上漲就做多，看到下跌就做空。十大原則的核心：賠錢不加碼、進場就設停損、停損只往有利方向移動、單日虧損不超過10%、不猜頭摸底。

## 實戰應用

### 台股趨勢跟蹤回測實證（TEJ）

TEJ 以 Cole Wilcox & Eric Crittenden (2005) 的研究改編，在台股進行5年回測：

| 項目 | 數據 |
|------|------|
| 入場條件 | 收盤價 > 一年來最高收盤價 |
| 出場條件 | 價格 < 當前價 - ATR × 1.25 |
| 年化報酬 | 31.1% |
| 累積報酬 | 266.5% |
| β值 | 0.79 |
| Sortino Ratio | 1.17 |

**關鍵設計**：
- 股價 ≥ 10元 + 流動性篩選
- 最多持有10檔，等權重配置
- 每日再平衡（注意：週轉率偏高）
- 除2022空頭年外，每年正報酬

### 順勢交易遇到支撐壓力的應對

- **突破壓力做多**：價格向上突破壓力區→進場做多→停損設在壓力區下方
- **跌破支撐做空**：價格向下突破支撐區→進場做空→停損設在支撐區上方
- **假突破防禦**：突破後觀察是否帶量、收盤站穩，避免追到假突破

## 注意事項

### 三大困難

1. **假突破頻繁**：台指期淺碟市場，突破後快速反轉是常態。需要帶量確認、收盤確認、3天站穩等過濾機制
2. **盤整期煎熬**：趨勢形成需要時間，盤整期頻繁進出會被兩面挨耳光。交易不順時縮手休息
3. **過度依賴技術面**：台股淺碟市場容易受消息面影響，應結合籌碼面和基本面判斷

### 順勢 vs 逆勢思維差異

| 順勢交易 | 逆勢交易 |
|---------|---------|
| 趨勢發動才進場，進場價較高 | 等回檔才買，進場價較低 |
| 趨勢在就不賣 | 覺得貴了就賣 |
| 賠錢停損小賠，賺錢加碼大賺 | 賺一點就跑，賠了擺著不理 |
| 適合趨勢明確的市場 | 適合盤整區間市場 |

### 週轉率陷阱

TEJ 回測發現每日再平衡導致週轉率偏高，交易手續費會侵蝕獲利。實戰建議：
- 降低再平衡頻率（週再平衡或月再平衡）
- 設定最小換股門檻（如新持股需優於現有持股5%以上才換）
- 手續費打折帳戶優先使用
