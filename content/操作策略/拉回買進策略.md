# 拉回買進策略

> 不追高，等價格拉回到支撐再進場，停損小、風險小、獲利空間大

## 核心概念

拉回買進（Pullback Buying）是趨勢交易中最實用的進場方式。核心邏輯是：

1. **先確認方向**：用均線排列判斷多空趨勢
2. **等待拉回**：價格拉回到支撐區（均線、前波高點、趨勢線）
3. **確認止跌**：出現紅K或止跌跡象再進場
4. **設定小停損**：因為進場在支撐附近，停損不需要太大

### 為什麼拉回買優於突破追高？

- 追高進場的問題：
  - 停損設太小（如40點）容易被掃掉
  - 即使停損設很大，仍可能被拉回掃掉
  - 波動幅度大時動輒3-4百點，很難承受
- 拉回買進的優勢：
  - 進場位置更好，停損不需要太大
  - 比較不容易被掃出場
  - 順著均線方向做交易，後續行情有機會往上走

### 均線判斷進場時機

**3日均線與5日均線搭配**：
- 多頭排列（3日在5日之上）→ 拉回買進
- 空頭排列（3日在5日之下）→ 反彈放空
- 黃金交叉 → 多空轉換點

**短線操作要領**：
- 價格沿5日均線往上爬 → 行情持續上漲
- 價格回落到均線附近出現反彈 → 均線具支撐力
- 乖離過大時 → 價格會回到均線附近（均值回歸效應）

**水母線輔助判斷**：
- 參數設定短均3、長均5，觀察多空排列狀態
- 柱狀體呈現多頭排列時行情往上，空頭排列時行情往下
- 可更直觀判斷短線方向

## 實戰應用

### 突破買進後的拉回進場

**箱型突破後的支撐確認**：
- 當股價有效突破箱頂（壓力轉為支撐），且收盤價在季線之上
- 每波行情突破新高後，必然伴隨群眾懼高心理，容易發生短線拉回修正
- 前一波箱型壓力轉為支撐 → 市場短線賣壓輕、籌碼有效換手

**五日線之上拉回買進**：
- K棒脫離5日線後 → 進入回測支撐階段
- 等待2個訊號：
  1. 守穩支撐收紅K棒
  2. 等待股價出量站穩5日線

**窒息量後的長紅K進場**：
- 股價橫盤超過1個月 → 意志不堅的散戶拋出
- 成交量進入量縮階段 → 出現「窒息量」
- 窒息量代表短線賣壓浮額清洗完畢
- 等待「窒息量後的下一根表態紅K棒」出現再進場

### 均線拉回買進操作流程

1. 確認均線排列方向（多頭排列做多是基本前提）
2. 等待價格拉回到3日或5日均線附近
3. 觀察是否出現止跌訊號（紅K、下影線）
4. 在支撐附近進場

### 停損設定 — 參考ATR

- 近期4000ATR平均435點，合理小停損約ATR的20-25%
- 20% → 約87點；25% → 約109點
- **原則**：波動大設大一點，波動小可以跟著縮小
- 拉回買進的好處：進場在支撐附近，停損50-80點就夠
- 追高進場壞處：停損再大都可能被掃掉

### 分批停利策略

- **第一口單**：設固定停利點（如125點，停利略大於停損）
- **第二口單**：採用移動停利（6根K棒高低點+100點）
- 先停利一半鎖定部分獲利，剩餘部位跟單續航
- 當行情觸及移動停利線時，完成整體出場

### 做空也適用

- 空頭排列時，等價格反彈到均線附近再放空
- 設定停損在均線上方
- 同樣使用分批停利和移動停利
- 方向對了 + 進場點好 = 勝率大幅提升

## 注意事項

- **不要追高**：寧可錯過也不要追高進場，拉回買才是穩健做法
- **方向優先**：先確認多空方向，再做對應操作，方向錯了進場點再好也沒用
- **停損要合理**：參考ATR設定停損，波動大停損大，波動小停損小
- **耐心等待**：好的進場點可能一天只出現一次，甚至兩天才有一次
- **均線是動態的**：不能以固定價格作為進場條件，要搭配即時均線位置判斷
- **看不懂就不出手**：如果走勢亂七八糟，不確定漲跌方向，就先別動
