# Black-Scholes定價模型

> 1997年諾貝爾獎公式，用五個變數算出選擇權理論價格，是所有Greeks的源頭，但假設世界完美所以永遠有誤差——理解模型限制比背公式更重要

## 核心概念

Black-Scholes Model（BSM）是由Fisher Black、Myron Scholes和Robert Merton在1973年提出的選擇權定價模型，1997年獲諾貝爾經濟學獎。它用五個輸入變數計算歐式選擇權的理論價格，至今仍是金融機構最常用的定價基準。

### 五個輸入變數

| 變數 | 符號 | 說明 |
|------|------|------|
| 標的物價格 | S | 目前股價或指數價格 |
| 履約價格 | K | 選擇權的執行價格 |
| 剩餘到期時間 | T-t | 距離到期的天數（年化） |
| 波動率 | σ | 歷史報酬波動度（年化標準差） |
| 無風險利率 | r | 通常用國庫券利率 |

### 公式結構

**買權（Call）**：
C = S × N(d1) - K × e^(-rT) × N(d2)

**賣權（Put）**：
P = K × e^(-rT) × N(-d2) - S × N(-d1)

其中：
- d1 = [ln(S/K) + (r + σ²/2)T] / (σ√T)
- d2 = d1 - σ√T
- N() = 標準常態分配累積機率密度函數

### 五個變數對價格的影響

| 變數 | 買權價格 | 賣權價格 |
|------|---------|---------|
| 標的物價格 ↑ | 上漲 | 下跌 |
| 履約價格 ↑ | 下跌 | 上漲 |
| 到期時間 ↑ | 上漲 | 上漲 |
| 波動率 ↑ | 上漲 | 上漲 |
| 無風險利率 ↑ | 上漲 | 下跌 |

## 實戰應用

### 用BS模型計算理論價格

Python實作核心：
```python
from scipy.stats import norm
import numpy as np

class BS_formula:
    def __init__(self, s0, k, r, sigma, T):
        self.d1 = (np.log(s0/k) + (r + sigma**2/2)*T) / (sigma*np.sqrt(T))
        self.d2 = self.d1 - sigma*np.sqrt(T)
    
    def call_price(self, s0, k, r, T):
        return s0*norm.cdf(self.d1) - k*np.exp(-r*T)*norm.cdf(self.d2)
    
    def put_price(self, k, r, T):
        return k*np.exp(-r*T)*norm.cdf(-self.d2) - s0*norm.cdf(-self.d1)
```

### 從BS模型推導Greeks

BS模型的偉大之處在於：每個Greeks都有解析解（closed-form solution），不需要用數值方法逼近。

| Greeks | 數學意義 | 買權公式 | 賣權公式 |
|--------|---------|---------|---------|
| Delta | ∂C/∂S | N(d1) | N(d1)-1 |
| Gamma | ∂²C/∂S² | N'(d1)/(Sσ√T) | 同買權 |
| Vega | ∂C/∂σ | S√T × N'(d1) | 同買權 |
| Theta | ∂C/∂t | -SN'(d1)σ/(2√T) - rKe^(-rT)N(d2) | -SN'(d1)σ/(2√T) + rKe^(-rT)N(-d2) |
| Rho | ∂C/∂r | KTe^(-rT)N(d2) | -KTe^(-rT)N(-d2) |

其中N'(d1)是標準常態分配的機率密度函數值（PDF）。

### 理論價 vs 實際價

BS模型算出的是「理論價格」，市場上的「實際價格」永遠有差異，原因：

1. **波動率微笑**：BS假設波動率是常數，但現實中不同履約價的IV不同
2. **連續交易假設**：BS假設可以連續調整部位，實際有交易成本和流動性限制
3. **歐式假設**：BS只適用歐式選擇權，美式選擇權有提前執行的溢價
4. **無股息假設**：原始BS不考慮股息，需用修正版
5. **常態分配假設**：實際報酬率有肥尾（fat tail），極端事件發生頻率高於常態分配預測

### 反推隱含波動率（IV）

實務上最常見的用法不是算價格，而是從市場價格反推IV：
- 輸入：市場價格 + S, K, T, r
- 輸出：隱含波動率 IV
- 用牛頓法迭代求解，因為C = f(σ)沒有反函數解析解

### 台指選擇權實例

TEJ用台指買權TXO202304C15500驗證：
- 履約價：15500
- 到期日：4月19日
- 計算日：4月10日（剩6個交易日）
- 用252天移動報酬波動度算歷史波動度
- 理論價與實際價有差異，差異來源就是波動率微笑/偏態

## 注意事項

1. **BS模型不是真理，是估計工具**：理論價永遠不等於市場價，差異本身就是交易機會
2. **波動率是唯一不可觀察的輸入**：S, K, T, r都已知，只有σ需要估計，這是模型最大的不確定性來源
3. **假設違反的後果**：
   - 假設波動率常數 → 忽略波動率微笑 → 定價誤差在深價外/深價內最大
   - 假設連續交易 → 低估避險成本 → 實際Greeks偏離理論值
   - 假設常態分配 → 低估尾部風險 → 極端行情下模型完全失效
4. **不要用歷史波動度代IV**：HV是過去、IV是未來預期，兩者經常差異很大
5. **Rho通常可忽略**：利率對短期選擇權影響極小，除非是LEAPS（長期選擇權）
6. **Greeks是瞬時偏微分**：只在那個瞬間成立，下一秒就變了，需要持續監控和調整
