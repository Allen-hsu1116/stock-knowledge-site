# 類股輪動策略

> 資金不會同時湧入所有產業——看懂輪動順序，提前部署下一個主流類股

## 核心概念

**類股輪動（Sector Rotation）** 是指市場資金在不同產業類別之間輪流移動的現象。A產業漲到一定程度後，資金撤出轉進B產業，形成「類股輪漲」的節奏。

### 為什麼會有類股輪動？

1. **資金效率**：台股規模越來越大，外資法人進入後，主力結構改變，無法同時拉抬所有類股，於是出現輪動
2. **景氣循環**：不同產業在不同景氣階段表現不同（參見[[美林投資時鐘景氣循環四階段|美林投資時鐘]]）
3. **短線資金操作**：資金湧入某類股推高股價 → 實現獲利後撤出 → 尋找下一個還沒漲的類股 → 重複
4. **消息面驅動**：政策、數據、事件驅動資金在不同產業間移動

### 類股 vs 概念股

- **類股**：以產業類別作為分類基準，有統一的客觀定義（如金融類股、電子類股、半導體類股）
- **概念股**：較偏向主觀個人認定的題材分類（如AI概念股、降息受惠股）

### 權值股是輪動的核心

類股輪動也可以叫做「權值股輪漲」。每個類股都有領頭羊（權值股），它們的表現決定了整個類股的方向。

## 實戰應用

### 辨識類股輪動的方法

1. **觀察類股指數強弱**：比較各產業類股指數的相對強度
2. **資金流向追蹤**：觀察法人資金在各類股間的流動
3. **RSI產業指標**：用各產業的RSI數值判斷相對強弱
   - RSI > 65：產業處於相對高點，可能開始循環
   - RSI 上升15以上：產業循環可能結束
4. **景氣信號燈搭配**：
   - 牛市（景氣信號上升）→ 觀察產業輪動信號，將0050 ETF短期置換成輪動類股
   - 熊市（景氣信號下降） → 平倉股票，持有短天期債券

### 航運領先半導體的輪動模式

TEJ實證發現：航運指數的轉強經常領先於半導體的上漲。
- 航運 RSI > 65 → 買入半導體相關類股
- 半導體 RSI 相對買入時上升15 → 出場賣出半導體回到0050 ETF
- 此策略搭配景氣燈號可優化單純持有大盤的績效

### 類股輪動操作SOP

1. **辨識當前主流類股**：用類股指數漲跌幅、法人買超、成交量判斷
2. **判斷主流是否接近尾聲**：法人開始減碼、成交量暴增但股價不漲、散戶大量進入
3. **尋找下一個輪動類股**：觀察RSI低、法人開始加碼、有題材但尚未發動的類股
4. **提前部署**：在主流類股尚未結束前就開始觀察下一個標的
5. **確認後進場**：新主流類股帶量突破、法人持續買超時進場

### 搭配美林投資時鐘

| 景氣階段 | 表現較好的類股 |
|----------|---------------|
| 復甦期 | 科技、非必需消費 |
| 繁榮期 | 原物料、工業 |
| 滯脹期 | 能源、必需消費 |
| 衰退期 | 公用事業、醫療、防禦股 |

## 注意事項

### 類股輪動的陷阱

- **散戶反向指標**：散戶開始狂買某類股時，可能就是輪動即將結束的訊號
- **輪動速度加快**：在震盪市中輪動速度極快，追高殺低風險大
- **假輪動**：某類股一天大漲不代表資金真的在輪動，可能是消息面驅動的短線脈衝
- **不要所有類股同時漲**：如果大盤全面上漲，就不是輪動而是牛市

### 實戰提醒

- 類股輪動在大型市場（如台股）才有意義，小型市場容易全面漲跌
- 輪動需要時間，不要因為一天的上漲就判斷輪動開始
- 法人資金流向是輪動最可靠的領先指標
