# 低波動因子選股策略

> 低波動異象是金融學中最反直覺的發現——波動越低的股票長期風險調整後報酬越高，NATR60是預測力最強的低波動因子

## 核心概念

低波動因子（Low Volatility Factor）是一種量化指標，用來衡量股票在給定期間內的價格波動程度。選擇波動相對較小的股票構建投資組合，歷史數據顯示在風險調整後能提供更穩定的超額報酬。

### 為什麼低波動異象存在？

1. **投資者行為偏差**：投資者過度追求高風險高回報的股票（彩票偏好），導致高波動股票被高估、低波動股票被低估
2. **基準偏誤（Benchmark Bias）**：基金經理人不願持有低波動股票，因為在牛市中會落後基準指數
3. **槓桿限制**：機構投資者受限於槓桿限制，傾向買高Beta股票來追求更高報酬

### 五種低波動因子衡量方式

| 因子 | 計算方式 | 特點 |
|------|---------|------|
| **NATR** | ATR / 收盤價 × 100 | 標準化真實波幅，不受股價高低影響 |
| **標準差** | 報酬率的標準差 | 最常見的波動衡量指標 |
| **最低/最高比** | 期間最低價 / 期間最高價 | 衡量價格壓縮程度 |
| **最大回撤** | 期間內從峰值到谷底的最大跌幅 | 衡量最壞情況 |
| **1/夏普比率** | 報酬率標準差 / 報酬率均值 | 單位風險下的報酬波動 |

### IC值（Information Coefficient）

IC值衡量因子預測未來收益率的能力：
- **正IC值**：因子值越高，未來收益率越高（順序因子）
- **負IC值**：因子值越高，未來收益率越低（逆序因子，低波動因子是這類）
- **IC接近0**：因子無預測能力

FinLab回測結果：**NATR60對於預測120天後的價格有最顯著的優勢**。

## 實戰應用

### 低波動因子選股策略

1. **選擇IC值最高的因子**：NATR60
2. **均線回歸過濾**：股價/120日均線 > 1（只選多頭排列的股票）
3. **因子排序篩選**：NATR60百分位 < 50%（只選低波動前半段）
4. **選最強30檔**：在低波動股票中選價格相對均線最強的30檔
5. **季度換股**：每季重新計算因子、調整持倉

### 多因子搭配

低波動因子可以與其他因子結合：
- **低波動 + 動能因子**：低波動篩選穩定標的，動能因子確認趨勢
- **低波動 + 價值因子**：低波動篩選穩定標的，價值因子確認估值合理
- **低波動 + 獲利因子**：低波動篩選穩定標的，獲利因子確認品質

### Beta因子的選股應用

FinLab實測結果：
- Beta小的組別以小型股（冷門股）居多
- 低Beta策略長期勝出，證實「冒著較高的風險，不一定報酬較高」
- 反而冒著較低的波動風險，長期能獲得較好的績效
- 中長期投組可納入風險因子作為篩選條件，低波動進場

## 注意事項

1. **低波動不是無波動**：低波動股票也會下跌，只是跌幅通常較小
2. **牛市可能落後**：低波動策略在牛市中可能顯著落後高Beta策略
3. **流動性問題**：低波動股票通常是小股本冷門股，大資金需注意滑價
4. **因子擇時困難**：低波動因子何時有效何時無效很難預測
5. **回測過度擬合**：IC值計算需要樣本外驗證，避免過度擬合
