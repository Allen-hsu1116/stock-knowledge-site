## 如何使用

- 每個題材有獨立的 JSON 檔案，路徑：`stock-picker/industry-map/{topic_id}.json`
- 題材面評分模組（`scoring/theme.py`）會自動讀取 JSON 來判斷股票的角色定位
- 新增題材：建立 JSON 檔案到 `industry-map/` 目錄，格式參考現有檔案
