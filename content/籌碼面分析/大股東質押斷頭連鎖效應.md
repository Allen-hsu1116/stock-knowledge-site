# 大股東質押斷頭連鎖效應

> 大股東質押比過高是地雷股前兆，股價下跌觸發斷頭會產生連鎖賣壓

## 核心概念

股票質押是大股東將持股抵押給金融機構借款的操作。當質押比過高且股價大跌時，維持率跌破130%會觸發斷頭，金融機構強制賣出質押股票，形成「股價下跌→斷頭→更多賣壓→更多斷頭」的惡性循環。

### 質押維持率計算

- **公式**：維持率 = 股票市值 ÷ 借款金額 × 100%
- **安全線**：130%（低於此線就會被追繳）
- **起始維持率**：約166%（借6成時）

### 斷頭觸發機制

| 借款成數 | 起始維持率 | 可承受跌幅 | 斷頭市值 |
|----------|-----------|------------|----------|
| 6成（60%）| 166% | 約22% | 借款×1.3 |
| 5成（50%）| 200% | 約35% | 借款×1.3 |
| 4成（40%）| 250% | 約48% | 借款×1.3 |

**關鍵**：借款成數越高，可承受跌幅越小，斷頭風險越大。

### 融資維持率 vs 質押維持率

| 項目 | 融資維持率 | 質押維持率 |
|------|-----------|-----------|
| 公式 | 股票市值÷融資金額 | 股票市值÷(借款+利息) |
| 斷頭線 | 130% | 130% |
| 追繳期限 | 2個營業日 | 2個營業日 |
| 借款目的 | 買股票 | 周轉用途不限 |
| 主體 | 一般投資人 | 大股東/法人為主 |

## 實戰應用

### 一、大股東質押比判讀四象限

| | 質押比低（<30%） | 質押比高（>50%） |
|---|---|---|
| **股價上漲** | ✅ 安心持有 | ⚠️ 質押借錢加碼？觀察用途 |
| **股價下跌** | 🔍 觀察是否開始增加質押 | 🚨 斷頭風險升高，連鎖賣壓 |

### 二、質押斷頭連鎖反應流程

```
股價下跌
  → 質押維持率跌破130%
    → 金融機構發出追繳通知（2個營業日內補足）
      → 大股東無力補繳
        → 金融機構強制賣出質押股票（斷頭）
          → 大量股票湧入市場
            → 供過於求，股價續跌
              → 更多質押維持率跌破130%
                → 連鎖斷頭效應
```

### 三、質押比高的大股東三種情境

1. **大型投資計畫**（如郭台銘質押買夏普）
   - 有明確用途
   - 公司基本面穩健
   - 不需過度擔心

2. **財務困難周轉**
   - 大股東需要錢應急
   - 高風險，股價下跌引發斷頭賣壓
   - 公司可能有隱藏負債

3. **槓桿加碼炒股**（最危險）
   - 質押借錢→買股→再質押→再借
   - 正反饋循環，一旦反轉就是災難
   - 可能掏空公司資產

### 四、質押斷頭的市場衝擊識別

1. **個股層面**：
   - 質押比>50%的個股，股價大跌時特別危險
   - 查詢管道：公開資訊觀測站 → 董監持股質押

2. **大盤層面**：
   - 2024年8月5日大盤融資維持率從166%降至139.42%
   - 接近130%斷頭線，引發恐慌性賣壓
   - 大盤融資維持率是系統性風險指標

3. **產業層面**：
   - 某產業大股東普遍高質押 → 產業性斷頭風險
   - 營建股、金融股常見高質押比

### 五、自保策略

1. **選股避雷**：避開董監質押比>50%的個股
2. **持股監控**：定期查詢持股質押比變化
3. **斷頭預警**：股價跌破質押成本×1.3時提高警覺
4. **產業風險**：同產業多家高質押，注意連鎖效應
5. **大盤訊號**：大盤融資維持率接近140%時，減少融資操作

## 注意事項

- **質押比高≠必死**：要了解質押用途，大型投資計畫的質押是合理的
- **斷頭有時間差**：2個營業日追繳期，不是立刻斷頭
- **補繳方式**：可以還現金或補質押股票，兩種方式都能拉回維持率
- **借6成只承受22%跌幅**：風險極高，借5成可承受35%，安全水位差很多
- **大盤融資維持率**是全系統指標，接近130%時代表系統性風險升溫
- **券商vs銀行差異**：券商每18個月需先還清再借，銀行無此限制
- **質押比突然暴增是最危險訊號**：趨勢比絕對值重要
