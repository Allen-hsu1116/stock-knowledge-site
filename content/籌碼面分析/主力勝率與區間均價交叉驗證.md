# 主力勝率與區間均價交叉驗證

> 用主力勝率判斷券商的操作質量、用區間均價定位主力成本，兩者交叉驗證才能分辨真主力與短線客

## 核心概念

### 兩個關鍵指標

1. **主力勝率**：該券商在某檔股票上的歷史操作結果（勝率+報酬率），反映「歷史操作質量」
2. **區間均價**：主力在特定期間的實際成本帶，反映「目前持倉位置」

單獨看任何一個都不夠，必須交叉驗證：
- 勝率高 + 現價站在區間均價之上 + 分點未大量賣出 → 主力仍願意維持多方布局
- 勝率高 + 跌破區間均價 + 持續賣超 → 主力可能正處於出場或調整階段

### 為什麼需要交叉驗證

如果只看勝率，容易被一次性暴利誤導。如果只看區間均價，無法判斷該分點是否真的「懂這檔股票」。兩者合用才能過濾掉短線績效的雜訊。

## 實戰應用

### 拆解「真主力」vs「短線客」

| 特徵 | 真主力 | 短線客 |
|------|--------|--------|
| 勝率穩定度 | 跨多個行情循環都維持穩定 | 單一次急漲行情拉出來 |
| 區間均價位置 | 與股價關鍵支撐帶重疊 | 集中在極短時間高位 |
| 回測行為 | 回測主力均價附近即出現承接 | 之後立刻大賣壓 |
| 分點動態 | 不斷分批加碼 | 短進短出 |
| 週期 | 長期熟悉該檔個股 | 隔日沖或題材炒作 |

### 交叉驗證三步驟

1. **篩選**：用主力勝率與區間均價篩選潛在標的（勝率穩定+現價在均價之上）
2. **驗證**：對照量價結構、整體籌碼、基本面與題材變化，檢查是否一致
3. **決策**：用自己的風險承受度、停損規則與持有期規劃決定是否參與

### 風險警訊判讀

以下組合出現時要提高警覺：
- 股價跌破主力區間均價 + 分點持續賣出 + 成交量放大卻無力反彈
- 過去再高的勝率也不足以構成「安全牌」
- 不要把「巧合」當成「主力護盤」

## 注意事項

1. **勝率要看樣本數和穩定度**：不是看數字大小，而是跨多個週期是否穩定
2. **跌破區間均價≠轉空**：但風險明顯升高，需搭配其他指標判斷
3. **分點K線是輔助濾網**：不應作為單一買進依據，需搭配產業、財報、技術面
4. **不要假設主力一定護盤**：看到高勝率不等於主力會撐住股價
5. **新手用法**：把分點K線當篩選工具，不是進出場依據
