# 庫藏股執行率與股價影響回測

> 庫藏股宣告不等於利多；執行率差=口水護盤，宣告後不漲=急救無效，後續更糟

## 核心概念

庫藏股（Treasury Stock）是公司用自有資金從市場買回自家股票。買回後股票不再流通，流通在外股數減少，EPS可能因此虛胖。

### 庫藏股三大實施目的（台股）

| 目的 | 真正意圖 | 評價 |
|------|---------|------|
| 轉讓予員工 | 員工激勵/認股權 | 偏正面 |
| 維護信用及股東權益 | 護盤/大股東自救 | 偏負面 |
| 股權轉換 | 可轉債/認股權證 | 中性 |

### 大股東4大真實動機

1. **硬撐股價** — 保護大股東質押部位，避免斷頭
2. **一買一賣出貨** — 用公司資金提供流動性，幫特定人士出貨
3. **提升股權比例** — 防止惡意併購
4. **節稅** — 減資代替配息，大股東股利所得稅率可能高於資本利得稅率

## 實戰應用

### 執行率：判斷庫藏股真假的關鍵

**執行率 = 實際買回股數 ÷ 預定買回股數**

| 執行率 | 解讀 |
|--------|------|
| > 80% | 認真執行，護盤有決心 |
| 50%~80% | 普普通通，可能中途放棄 |
| < 50% | 口水護盤，只是做樣子 |
| < 30% | 嚴重不及格，欺騙投資人 |

**查詢方式**：公開資訊觀測站 → 庫藏股統計彙總表（證交所按季發布）

### 回測數據：庫藏股對股價的實際影響

**2009~2012年台股歷史回測結果**（市場先生）：

#### 1. 執行庫藏股期間買進，平均不賺錢
- 執行期間1個月報酬率：平均不顯著，勝率低於50%
- 獲利集中在2009年（多頭市場），虧損集中在2011年（空頭市場）
- **結論：庫藏股效果受大盤行情影響很大，本身不具獨立選股價值**

#### 2. 股價大漲時的庫藏股（轉讓予員工）表現較好
- 過去半年漲幅 >10%，轉讓予員工目的的庫藏股，1個月後漲幅較高
- 2年後績效：虧損>50% 有3家，獲利>100% 有15家
- 原因：股價上漲時執行轉讓予員工，代表經營層看好公司未來

#### 3. 股價大跌時的庫藏股（維護股東權益）有反彈機會
- 過去半年跌幅 >40%，維護權益目的的庫藏股，1個月後漲幅通常較高
- 近40%的股票後續上漲超過100%
- 原因：股價過度低估時買回，對股東有利

#### 4. 急救無效訊號（最重要）
- **執行庫藏股後1個月仍下跌5%以上** → 急救無效
- 未來2年上漲機率 <50%
- 倒賠20%機率高達25%
- 1年內虧損機率可能超過60%
- **當庫藏股宣告後股價不漲反跌，就是最強烈的賣出訊號之一**

### 庫藏股操作判斷流程

```
公司宣告庫藏股
  ↓
查看實施目的：轉讓員工 vs 維護權益
  ↓
查看過去半年漲跌：
  → 大漲 + 轉讓員工 → 偏多，可觀察
  → 大跌 + 維護權益 → 可能反彈
  → 大跌 + 轉讓員工 → 員工拿到也不開心，偏空
  → 大漲 + 維護權益 → 可能出貨，偏空
  ↓
追蹤執行率：
  → 高執行率 → 有誠意
  → 低執行率 → 口水護盤
  ↓
觀察宣告後1個月股價：
  → 上漲 → 可考慮續抱
  → 下跌5%以上 → 急救無效，離場
```

### 2%和3億元公告門檻

買回期間內，累積買回股數達公司已發行股份總數2%或金額達3億元時必須公告。再次累積達同一標準也需再次公告。這是追蹤執行進度的重要工具。

## 注意事項

- **庫藏股不是萬靈丹**：本業不好的公司，實施庫藏股對股價只是短期現象
- **資訊疲乏效應**：多次買回庫藏股的公司（如宏達電12次），宣告效果隨次數遞減
- **護盤≠看好**：很多公司實施庫藏股是為了保護大股東質押，不是真的看好
- **EPS虛胖**：流通股數減少使EPS上升，但本業獲利能力未改善
- **台股vs美股差異**：美股庫藏股常是替代配息的節稅手段；台股庫藏股多是護盤，性質不同
- **宣告後不漲是最強烈賣出訊號**：急救無效的股票，未來2年表現大概率很差
