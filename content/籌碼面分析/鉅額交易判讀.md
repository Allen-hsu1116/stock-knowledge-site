# 鉅額交易判讀

> 盤後大單是主力的真實聲音，溢價搶貨看多、折價拋貨看空，價差方向比成交本身更重要

## 核心概念

鉅額交易是台股特有的制度，讓大額買賣可以在集中市場之外完成，避免對盤中價格造成劇烈衝擊。

**申報門檻**：
- 單一證券：500交易單位（50萬股）或金額1,500萬元以上
- 股票組合：5種股票且金額合計1,500萬元以上

**兩種交易方式**：

| | 配對交易 | 逐筆交易 |
|---|---------|---------|
| 時間 | 08:00-08:30, 09:00-17:00 | 09:00-17:00 |
| 方式 | 買賣雙方事先協商 | 分拆小單在公開市場執行 |
| 價格 | 協商價格，可偏離收盤價 | 公開市場價格發現 |
| 揭露 | 完成後才揭露 | 即時揭露 |
| 適用 | 機構大額移轉、大股東轉讓 | 機構分批進出 |

## 實戰應用

### 1. 溢價/折價判讀

**溢價成交**（成交價 > 當日收盤價）：
- 買方願意用更高價搶貨 → 看多訊號
- 可能原因：掌握未公开利多、急於建倉、法人配置需求
- 案例：2026/3/2 台積電收盤1,975元，鉅額交易2,190元（溢價+10.9%）

**折價成交**（成交價 < 當日收盤價）：
- 賣方願意用更低價出貨 → 看空訊號
- 可能原因：大股東急於變現、內部人看淡後市、機構降部位
- 折價幅度越大，賣壓越急

### 2. 鉅額交易追蹤方式

- **證交所/櫃買中心官網**：每日公布鉅額交易明細
- **TEJ PRO**：整理好的歷史與當日鉅額交易資料庫
- **券商APP**：部分券商提供鉅額交易即時通知

### 3. 鉅額交易後的股價行為

- 溢價鉅額交易後，短期（3-5日）上漲機率較高
- 折價鉅額交易後，短期下跌壓力大
- 但須注意：配對交易可能是大股東的資產規劃（贈與、信託），非看多看空
- 連續多日同方向鉅額交易，訊號強度更高

### 4. 鉅額分點追蹤

搭配分點進出資料，可以進一步判斷：
- 總公司vs分公司：總公司大單偏向法人操作，分公司偏向大戶
- 百大券商買超股：交易量最大的100家券商分點，買超數百張以上的個股

## 注意事項

- **配對交易可能是資產移轉**：大股東贈與、信託規劃也走鉅額交易，不一定是看多
- **單次鉅額交易不構成訊號**：要觀察趨勢，連續多日同方向才有效
- **流動性低的股票鉅額交易影響更大**：小型股一筆鉅額交易可能等於好幾天成交量
- **資訊揭露有延遲**：配對交易完成後才揭露，散戶永遠慢一步
- **避免過度解讀**：機構有時只是調整部位，不是方向性交易
- **注意交易者身分**：內部人申報轉讓走鉅額交易，和法人進出意義完全不同
