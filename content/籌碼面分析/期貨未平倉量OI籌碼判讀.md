# 期貨未平倉量OI籌碼判讀

> 透過觀察期貨未平倉量的變化，判斷市場資金流向與多空趨勢強弱，是籌碼面分析的核心工具。

## 核心概念

**未平倉量（Open Interest, OI）** 是期貨與選擇權市場中，已成交但尚未結算的合約總數量。一份合約包含一個買方和一個賣方，但只計算為1口（不是2口）。

OI 與成交量的根本區別：
- **成交量**：當日成交的合約數量，反映交易熱絡程度，隔日歸零重算
- **未平倉量**：市場上尚未結清的合約總量，持續累計直到合約到期結算歸零

三種交易對 OI 的影響：
1. 新買方+新賣方建倉 → OI +1（新資金進場）
2. 原有持倉者之間換手 → OI 不變（資金未增減）
3. 原有持倉者雙方平倉 → OI -1（資金離場）

OI 基本特性：
- 沒有數量上限（不像股票有流通股數限制）
- 通常隨時間緩慢增加，到期日歸零
- 近月 OI > 遠月 OI

## 實戰應用

### OI + 價格判讀表（核心）

| OI 變化 | 價格變化 | 判讀 | 操作意涵 |
|---------|---------|------|---------|
| OI 上升 | 價格上漲 | 純多頭 | 新資金看好，漲勢具延續性 |
| OI 上升 | 價格下跌 | 強空頭 | 新資金看壞，跌勢可能加劇 |
| OI 下降 | 價格上漲 | 動能衰退 | 贏家獲利了結，趨勢可能反轉 |
| OI 下降 | 價格下跌 | 動能衰退 | 輸家認賠出場，賣壓漸消 |
| OI 持平 | 價格震盪 | 缺乏方向 | 多空均衡，等待突破訊號 |

口訣：**OI 上升趨勢續，OI 下降趨勢弱**

### 外資台指期 OI 判讀（最重要）

外資是台指期最大的參與者，其 OI 變化最具參考價值：

- 外資淨多單持續增加 → 偏多訊號，外資看好後市
- 外資淨空單持續增加 → 偏空訊號，外資看壞後市
- **外資淨空單超過1萬口** → 小心近日行情回檔機率升高
- 外資操作週期以月為單位，短期虧損並非他們在意重點

自營商 OI 變動極快、影響短期，通常不提供方向性參考；投信部位小，參考價值有限。OI 分析判讀以外資動向最重要。

### OI 判斷真假突破

價格突破整理區時，可藉由 OI 增減判斷真假：
- 突破伴隨 OI 擴大 → 高機率真突破（新資金進場追價）
- 突破但 OI 減少 → 高機率假突破（舊持倉者平倉）

### 結算日前動能概念

接近結算日前，若 OI 仍然相對高：
- 外資法人獲利平倉或轉倉造成的指數波動越大
- 若價格長期震盪但 OI 持續增加，一旦突破將引發大量追價盤

### 每日淨口數變化追蹤

觀察 OI 由淨空倉轉為淨多倉時，有兩種可能：
1. 大幅建立新多單（偏多訊號）
2. 空單部位結清平倉（可能是獲利單或停損單）

需進一步分析是新建倉還是平倉，搭配價格走勢判斷。

### 估算法人進場成本

搭配每日外資多空單 OI 增減，計算不同價位區間累積的 OI，得出法人進場成本參考價，評估多方還是空方勝出，比單純看 OI 總數更準確。

### OI + 成交量雙確認

- 📈 OI 上升 + 成交量上升 = 趨勢強勁，市場活躍
- ⚠️ OI 上升 + 成交量下降 = 交易熱情減弱，小心趨勢陷阱
- 🔄 OI 下降 + 成交量下降 = 市場冷卻，可能進入震盪

### 三大法人 OI 觀察

台灣期交所每日15:00公布三大法人期貨未平倉數據：

- **外資 OI**：引領大盤趨勢，外資淨多單持續增加 = 偏多訊號
- **投信 OI**：相對穩定，變化較小
- **自營商 OI**：偏避險操作

**多空淨額判讀**：
- 淨額 > 0（正值）→ 偏多
- 淨額 < 0（負值）→ 偏空

### 散戶多空比（反向指標）

計算方式：
- 散戶多單 = 小台OI總量 - 三大法人小台多方OI
- 散戶空單 = 小台OI總量 - 三大法人小台空方OI
- 散戶多空比 = (散戶多單 - 散戶空單) / 小台OI總量

散戶極度看多時 → 警惕見頂回落
散戶極度看空時 → 可能反而是底部

### 高低 OI 的市場意義

**高 OI**：流動性好、趨勢延續性高、但潛在波動也大（關鍵位突破時連鎖反應）

**低 OI**：趨勢可能轉弱、波動性低、流動性差、進出場困難

### OI 與關鍵技術位

當 OI 長期處於高位，而價格接近關鍵支撐/壓力位時，容易出現劇烈波動（大量持倉可能遭遇強制平倉或觸發止損），應適時調整風險管理策略。

## 注意事項

1. **OI 不能單獨使用**：必須搭配價格和成交量綜合判斷，單看 OI 變化意義有限
2. **散戶 OI 反向指標不是每次都準**：有超額報酬的參考價值，但不能盲信
3. **結算日效應**：近月合約結算前 OI 會下降（轉倉至次月），這是正常現象不代表趨勢反轉
4. **三大法人是加總數字**：期交所公布的是三大法人加總互抵的統計結果，非特定機構的策略
5. **避險 vs 投機**：企業避險交易會使 OI 大但成交量小，需區分
