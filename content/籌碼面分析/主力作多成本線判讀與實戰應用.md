# 主力作多成本線判讀與實戰應用

> 透過計算主力買超的加權平均成本價，研判主力持股成本位置，作為止跌判斷、反彈滿足點與主力是否續抱的關鍵參考。

## 核心概念

主力作多成本線是將一段期間內「主力買超金額」除以「主力買超張數」，計算出主力持有該檔股票的平均成本價位。與一般均線不同，成本線加入了**身分別（誰在買）**與**交易量（買多少）**的考量，更能真實反映特定買方群體的持倉成本。

### 成本線的三大分類

| 類型 | 計算基礎 | 適用投資人 | 時間框架 |
|------|----------|------------|----------|
| 融資成本線 | 融資買超加權均價 | 短線投資人 | 日～週 |
| 投信成本線 | 投信買超加權均價 | 中線投資人 | 週～月 |
| 外資成本線 | 外資買超加權均價 | 長線投資人 | 月～年 |
| **主力作多成本線** | 主力（前15大券商分點）買超加權均價 | 中短線投資人 | 日～週 |

### 主力作多成本線的計算邏輯

```
主力作多成本線 = Σ(每日主力買張 × 每日均價) / Σ(每日主力買張)
```

其中「每日均價」可取 (開盤+最高+最低+收盤)/4。

XQ 平台的預設參數為 40 天，也可以依操作習慣調整天期。

## 實戰應用

### 1. 止跌判斷：股價拉回主力成本附近

當股價從高點拉回到主力作多成本線附近時：
- **主力持續買超** → 主力仍在加碼，成本線是有效支撐，可考慮逢低進場
- **主力轉為賣超** → 主力可能在停損出場，成本線支撐無效，不應接刀

### 2. 反彈滿足點判斷

大跌後反彈時，主力成本線可以作為**反彈的滿足點**。當股價反彈接近主力成本線，代表主力解套或接近損益兩平，此時要觀察主力是否開始減碼。

### 3. 三線交叉判讀

將融資成本線、投信成本線、外資成本線同時觀察：
- **三線收斂** → 不同法人成本趨於一致，容易出現方向性突破
- **三線發散** → 法人成本差異大，股價震盪機率高
- **股價在所有成本線之上** → 多方全面掌控
- **股價在所有成本線之下** → 空方全面掌控

### 4. 乖離過大的警訊

當股價與主力成本線乖離過大（距離太遠）：
- 不應再追高加碼
- 隨時可能出現拉回修正
- 可等股價回到成本線附近再考慮進場

## 注意事項

- **只適用有量的熱門股**：主力作多成本線對有特定主力著墨的熱門股才有參考價值，對沒什麼量、沒有特定主導勢力的冷門股不適用
- **成本線不等於均線**：成本線考慮了身分別與交易量，是均線的嚴謹版
- **單一指標不夠**：必須搭配主力買賣超方向一起看，成本線位置 + 主力進出方向才有意義
- **注意主力分倉操作**：主力可能分散在多個券商分點，單一成本線可能無法完全反映所有主力成本
- **融資成本線過於貼近股價**：短線參考價值較高，但容易被短線雜訊干擾
