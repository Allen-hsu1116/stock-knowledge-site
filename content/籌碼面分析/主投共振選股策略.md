# 主投共振選股策略

> 當主力與投信同時買超，等於兩大籌碼力量共振，是波段飆股最強發動訊號；投量比＞10%且股價在低檔時勝率最高

## 核心概念

「主投買」是指主力與投信同時買超的選股策略。當兩大籌碼力量方向一致，代表資金面與基本面共振，股價發動攻擊的機率大幅提升。

### 為什麼主力+投信共振這麼重要？

- **主力**：資金雄厚、操作精準，對股價有直接推升力
- **投信**：研究團隊把關、法規限制把關，買進代表基本面通過篩選
- **兩者同買**＝資金面+基本面雙重確認，比單一訊號可靠得多

### 主力的定義

主力是一群擁有資金且有豐富投資經驗的投資者，對股價走勢有一定程度的影響。常見組成：政府資金、三大法人、市場派、公司派。觀察方式：前15名券商分點買賣超。

## 實戰應用

### 追蹤主投買的8點注意事項

1. **投信第一天大買**：隔天股價容易強勢跳空開高，值得留意觀察
2. **投信持續性買超**：如果投信持續買超，這檔股票容易走波段
3. **投量比＞10%**：投量比＝投信買賣超張數÷成交量張數×100%，連續2天更好。投信買超占成交量高且股價在低檔，很有爆發力
4. **進場時機**：投信連買2天 + 投量比＞10% + 股價低檔 → 可考慮進場；投信持股超過股權15% → 準備出場
5. **買超張數太少不跟**：投信連續買但張數極少，沒有實質影響力
6. **投信小賣時看主力**：如果主力持續買超，股價可能有支撐；主力也轉賣時，行情可能結束
7. **主投買＝攻擊訊號**：主力買+投信買的股票容易發動攻勢
8. **主投賣＝空頭訊號**：主力賣+投信賣的股票偏空，高檔時投信下車+主力轉賣 → 跟著下車

### 選股SOP

1. 用軟體篩選「投信買超%高＋主力買超」的股票
2. 確認投量比＞10%（連續2天更好）
3. 確認股價在相對低檔（非已大漲個股）
4. 確認主力買超與投信買超方向一致
5. 設定停損：投信持股超過15%時留意出場

### 案例參考

- 緯創（3231）：3月3日主力和投信買進（32.65元），4月12日45.15元，漲幅38%
- 宜鼎（5289）：3月15日主力和投信買進（236.5元），4月12日322元，漲幅36%
- 華城（1519）：4月6日主力和投信買進（73.4元），4月12日96.8元，漲幅32%

### 最怕的情況

股價在低檔，投信買、主力也買 → 勝率高；最怕的是跟著投信買，但主力拼命在倒貨 → 不妙！記住：下單買賣要有多個理由，勝算才會高。

## 注意事項

### 陷阱一：只看投信不看主力

投信買超不等於股價必漲。如果主力在倒貨，投信買進只是「接刀」。必須確認兩者方向一致。

### 陷阱二：追高漲多個股

就算投信有介入，漲多的股票不一定會繼續飆漲。投信也可能買到高點，甚至認賠殺出或短線獲利了結。

### 陷阱三：忽略投量比

投信買超張數的絕對值不夠看，要相對於成交量才有意義。投量比＜10%的買超影響力有限。

### 陷阱四：投信持股過高仍追

投信持股超過15%時，已接近法規上限，投信只能只出不進。這時追進風險大，因為投信的買盤即將消失。

### 陷阱五：忽略出場訊號

當主力+投信同時轉賣（主投賣），是最強烈的空頭訊號。高檔出現主投賣 → 果斷下車。
