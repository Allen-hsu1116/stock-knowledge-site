# 黑嘴薦股騙局防範

> 識別非法證券諮詢的四步套路，避免成為抬轎接貨的犧牲品

## 核心概念

「黑嘴」是指非法證券諮詢機構或個人，利用互聯網、手機短信等媒介發展會員，收取薦股費用非法獲利，或推銷「包賺不賠」的薦股軟體。黑嘴擾亂市場定價機制，造成「劣幣驅逐良幣」的惡性循環，使守法諮詢機構難以正常開展業務。

## 實戰應用

### 黑嘴四步套路

| 步驟 | 操作 | 本質 |
|------|------|------|
| 第一步：高級會員建倉 | 合作夥伴先建倉 → 高級會員建倉抬轎 | 莊家先行布局 |
| 第二步：低級會員抬轎 | 短信群發通知買入，股價迅速推高 | 散戶集體接貨 |
| 第三步：免費送飆股誘餌 | 吸引更多散戶追高，同時獲取聯繫方式 | 補充流失的抬轎軍 |
| 第四步：公開點評推薦 | 場外資金接最後一棒 | T+1追高者全部套死 |

**最惡劣的黑嘴**：第二三四步同時進行，追高者當天無法賣出，全部套牢。

### 五招識別黑嘴

1. **是否掛牌經營**：合法機構會公示全稱及業務資格證書號碼、執業人員姓名及執業資格號碼
2. **是否招收異地會員**：法律禁止，異地會員投訴困難，黑嘴以此鑽漏洞
3. **合同是否合規**：統一印製、首頁載明投訴電話及收費帳戶、風險提示「不承諾投資收益」
4. **是否推薦黑馬/自吹自擂**：禁止明示或暗示一定獲利、「免費贈股」招攬業務
5. **處理投訴態度**：推諉不處理 = 黑嘴，合法機構須於次月月底前處理完畢

### 防範策略

1. **拋卻一夜暴富幻想**：黑嘴利用貪念引誘，越保證獲利越可疑
2. **獨立思考**：任何薦股都自行驗證，不盲從
3. **不追高接盤**：被推薦的股票往往已在高位，追進就是接貨
4. **建立自己的投資邏輯**：基本面+技術面+籌碼面綜合判斷
5. **驗證機構資格**：到證券監管部門官網查詢機構是否合法

### 台灣常見的薦股騙局模式

- **LINE群組報明牌**：先給甜頭（小賺），再推高價課程或代操
- **YouTube/Telegram喊單**：免費直播吸引，付費群組坑殺
- **假外資/假法人訊號**：偽造法人進出資料，引誘跟單
- **代操保證獲利**：捲款跑路，或用你的錢抬轎

## 注意事項

1. **保證獲利 = 騙局**：合法投顧不會承諾收益，任何「包賺不賠」都是紅旗
2. **T+1是黑嘴武器**：追高者當天無法賣出，黑嘴利用這個制度套殺
3. **先甜後苦是套路**：初期讓你小賺培養信任，後期一次坑殺
4. **群體行為抬轎**：群發通知 = 同時買入 = 股價瞬間推高 = 合作夥伴出貨
5. **驗證比信任重要**：任何薦股都自己查基本面、技術面、籌碼面
