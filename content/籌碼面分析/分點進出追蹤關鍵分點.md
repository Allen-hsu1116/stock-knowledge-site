# 分點進出追蹤關鍵分點

> 每日盤後券商分點買賣超數據是追蹤主力行蹤的利器，透過關鍵分點和地緣券商辨識大戶動向

## 核心概念

### 什麼是分點進出

分點進出是指各券商分公司每日在每檔股票的買賣紀錄。台灣證券交易所在每日盤後公布每一間券商分公司的買賣超數據，資料詳細到哪一天、哪一檔股票、哪一個價位、買進或賣出幾張。

每一筆交易都會歸屬到開戶的券商分點，因此可以從分點進出數據推測特定大戶或主力的進出場時機。

### 分點進出的三層意義

1. **追蹤已知主力** — 已辨識的關鍵分點，持續追蹤其動向
2. **發現未知主力** — 突然出現大量買超的陌生分點，可能是新主力進場
3. **確認出貨訊號** — 已知主力分點突然轉為大量賣超，出貨警訊

## 實戰應用

### 關鍵分點辨識

關鍵券商 = 一段區間內買超最多的券商分點。辨識標準：

1. **績效導向** — 能在低檔大買、高點大賣的分點
2. **連續性** — 連續數日或數週在同一檔股票買超
3. **量級突出** — 買超張數佔當日成交量顯著比例

**實例**：網龍（3083）2019/8/21，「兆豐-景美」突然買超 557 張，之後股價明顯上漲。在相對高點同一分點開始出脫持股，精準掌握波段。9月初再度進場，後續又有一波漲幅。

### 地緣券商判讀

公司附近券商出現大量買超 → 可能是公司派或內部人士進場：

1. 找出公司所在地（公開資訊觀測站可查公司地址）
2. 搜尋公司附近 5 公里內的券商分點
3. 監控這些「地緣券商」的買賣超變化
4. 大量買超 + 公司有利多傳聞 → 高機率是內部人布局

**實例**：為升（2231）位於彰化縣福興鄉，鹿港市區的券商分點若出現大量買超，可能是內部人士進場。

### 隔日沖分點辨識

已知隔日沖券商分點（如凱基-台北、元大-新竹等），遇到這些分點大量買超時要小心：
- 隔日沖分點買超 → 隔天大概率賣出
- 佔當日成交量超過 10% → 特別警戒
- 搭配股價位置判斷：低檔隔日沖影響小，高檔隔日沖危險

### 分點進出查詢工具

| 工具 | 特色 |

| 證交所買賣日報表 | 官方原始資料 |
| 籌碼K線 APP | 有「高手券商」排行，依勝率、報酬率排序 |

## 注意事項

1. **分點進出不是聖杯** — 大戶買進不代表一定漲，大戶也可能看錯
2. **分點可以換** — 主力可以開多個分點帳戶，看到的可能只是冰山一角
3. **法人分點混淆** — 外資透過複委託會歸在特定分點，不等於本土主力
4. **隔日沖陷阱** — 隔日沖分點大量買超不代表看好，隔天就會賣
5. **適合短線參考** — 分點進出偏向短線工具，長線投資參考價值較低
6. **資訊遞延** — 盤後才公布，無法即時掌握
7. **搭配其他指標** — 分點進出要搭配集保股權、法人買賣超、技術面等綜合判斷
