# 券資比判讀

> 券資比是融券餘額除以融資餘額的比值，30%為多空分水嶺，高券資比暗示軋空行情可能啟動

## 核心概念

券資比 = 融券餘額 ÷ 融資餘額 × 100%

這個比率反映市場多空力量對比：
- **券資比 > 30%**：看空的人較多，散戶放空意願強
- **券資比 < 30%**：看多的人較多，散戶偏多
- **以往70%以上**才算高券資比，近年**40%以上**就會引起注目

**背景知識**：
- **融資**：向券商借錢買股票，自備40%，券商出60%，通常視為散戶動向指標
- **融券**：向券商借股票賣出，自備90%保證金，看空工具
- 兩者都是信用交易，具有槓桿風險

## 實戰應用

### 0. 券資比三步實戰框架

**第一步：查詢與對照**

在籌碼工具查看資券頁，評估：
- 券資比水平與近10–20日變化率
- 融資與融券的同步性
- 疊合K線、成交量和法人買賣超

若券資比連續上升、融券餘額增加，同時出現爆量突破，代表空方回補風險升溫。
若券資比下滑但融資與量能同步放大，留意一致性樂觀下的回檔壓力。

**第二步：情境門檻與共振**

- **軋空候選**：券資比>30%且加速上升 + 外資買超由負轉正 + 量能大於近20日均量
- **去槓桰風險**：券資比<20% + 融資五日連增 + 價量背離（價滯量增或價跌量增）

**第三步：風險控管與時間窗**

- 關注公告、法說、指標題材的前後兩週，回補常在事件臨近加速
- 設好停損與分批進出
- 搭配借券賣出、當沖比率、主力與外資買超趨勢，辨識「誰在積極加碼」與「誰被迫回補」

把券資比當作「部位鏡子」：數據極端（如80%以上）且價格仍強，空方潛在買盤很大；但一旦量縮跌破關鍵均線、法人轉賣，結構易迅速反轉。

### 1. 高券資比的軋空行情邏輯

高券資比出現的典型流程：
1. 股價已上漲數日，散戶覺得到高點
2. 散戶開始融券放空，券資比攀升
3. 主力可能刻意賣出造成下跌，誘使更多散戶加碼做空
4. 券資比提高到一定程度
5. 主力大量買進或釋出利多，散戶被迫停損回補
6. 股價易漲難跌，形成軋空行情

**操作策略**：當券資比偏高且法人持續買超時，站在主力同一邊做多，等待軋空。

### 2. 券資比搭配其他指標

| 搭配指標 | 判斷方式 |
|----------|----------|
| 外資買賣超 | 高券資比 + 外資持續買超 = 強力軋空訊號 |
| 投信買賣超 | 高券資比 + 投信加碼 = 中線軋空潛力 |
| 融資餘額變化 | 融資減 + 融券增 = 散戶多翻空，留意反轉 |
| 股價位置 | 低檔高券資比比高檔更有軋空力道 |

### 4. 實實戰案例

穎漢(4562)案例：
- 股價從5月開始上攻，從十幾元翻倍大漲
- 券資比在6月暴增，散戶認為到了高點開始放空
- 外資持續買超
- 融券做空的散戶被迫回補
- 股價具備易漲難跌特性，上攻至80.1元

## 注意事項

- **主力也會使用融資/融券**：不要假設融資一定是散戶、融券一定是散戶
- **單一指標不足**：券資比需搭配基本面、法人籌碼、技術面綜合判斷
- **高券資比≠一定軋空**：可能主力也看空，需確認法人動向
- **券資比會變化**：需追蹤趨勢，不是看單一時點
- **信用交易槓桿風險**：融資融券都有放大風險，停損要嚴格
- **不是高券資比必漲、低券資比必跌**：勝率來自多訊號共振與風險控管
- **事件驅動是催化劑**：法說會、季報、題材消息前後，券資比變化加速度最具參考價值
