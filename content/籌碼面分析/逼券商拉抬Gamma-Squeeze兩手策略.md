# 逼券商拉抬Gamma Squeeze兩手策略

> 大戶一手買現貨一手掃權證，利用券商必須維持Delta中性的避險義務「逼」券商追價買進，形成正回饋循環推升股價——這就是台股最獨特的「兩手策略」

## 核心概念

### 什麼是逼券商拉抬

自營商發行權證後，法規要求必須維持避險部位。大戶利用這個「不得不買」的機制，刻意大量買進認購權證，迫使自營商在現貨市場追價買進避險，間接推升股價。

### 兩手策略的操作邏輯

1. **一手持有現貨**：大戶先建立現股多頭部位
2. **另一手狂掃權證**：大舉買進認購權證，消耗券商權證庫存
3. **券商被迫抬轎**：自營商為避險而買進現股，推高股價
4. **大戶兩頭獲利**：現股漲價賺價差 + 權證也漲價

### 為什麼權證被「秒殺」

權證發行股數一般僅5,000~1萬張。當大戶集中掃貨，權證可能掛牌不到3天就被收光。券商必須增發新權證或加碼避險部位，進一步放大對現貨的影響。

## 實戰應用

### Gamma Squeeze正回饋循環

股價上漲時的連鎖反應：

```
大戶買權證 → 自營商避險買現股 → 股價上漲
→ Delta上升（接近1）→ 自營商須加買更多現股
→ Gamma加速 → 股價漲更多 → Delta繼續上升 → 循環加劇
```

**關鍵**：價平附近的權證Gamma最大，Delta變化最快。大戶若選擇價平權證操作，逼券商效果最強。

### 負回饋循環：下跌時的連鎖反應

```
股價下跌 → Delta下降 → 自營商須減少避險部位（賣股）
→ 賣壓加重 → 股價跌更多 → Delta持續下降 → 自營商繼續賣
```

**結果**：股價越跌，券商越賣，賣壓越大，形成下跌加速器。

### 2017年實戰案例

外資三招齊發，其中第三招「狂掃認購權證」最為特殊：

- 自營商連6日買超台股達104億元，其中89億元是避險部位
- 10月權證成交值551億元，11月未過完已達565億元
- 晶電盤中拉到快漲停，交易員追漲搶買；瞬間又往下殺，交易員又跟著賣
- 交易員在盤中被軋來軋去，直呼「好辛苦」
- 這筆資金推估來自最熟悉權證操作的香港市場，「因為傳統長線外資不會買權證」

### 2026年自營商避險數據補充

- 豐雲學堂2026年2月整理：融資維持率低於150%的個股名單，提醒投資人注意斷頭風險
- 自營商資金規模是三大法人中最小，約佔大盤日成交量1-2%（約20-40億），但對個股影響力仍大
- 自營商風控嚴格：總虧損不超過10%、單筆虧損有上限，虧到上限輕則停止交易，重則交易員丟飯碗
- 這嚴格風控是自營商短線操作的根本原因——行情一有風吹草動就砍部位

### 辨識兩手策略的四個訊號

| 訊號 | 觀察方法 | 意義 |
|------|----------|------|
| 1. 權證成交暴量 | 單日權證成交值異常放大 | 大戶在掃權證 |
| 2. 自營商避險買超激增 | 避險買超遠大於自行買賣 | 券商被逼著買 |
| 3. 權證被秒殺 | 掛牌不到3天就被收光 | 大戶有意為之 |
| 4. 個股劇烈震盪 | 盤中急漲急跌，振幅大 | 交易員追漲殺跌 |

### 散戶應對策略

**做多者**：
1. 確認是「真買盤」還是「避險買盤」——避險買盤隔天大概率賣出
2. 有隔日沖/避險券商在場時，等9:30過後再進場
3. 9:30前賣壓消化完後，觀察是否有其他主力承接

**放空者**：
1. 等開盤漲勢停滯時，跟著隔日沖賣壓方向放空
2. 注意：兩手策略拉抬階段放空風險極大，必須等正回饋循環結束

### 大戶出場時機

大戶同時賣出權證和現貨時：
- 券商避險部位同步減少，也跟著賣出
- 形成「三重賣壓」：大戶現貨 + 大戶權證 → 券商避險
- 當天長紅K棒的隔天，可能開高走低

## 注意事項

### 陷阱一：以為避險買超是看好

避險買超是被動的，不代表自營商看好。事實上，避險買超代表散戶大量買進權證，散戶看多是反向指標。

### 陷阱二：低估下跌時的殺傷力

正回饋循環推升股價，負回饋循環同樣會加速下跌。2017年案例中，交易員「想買不敢買，結果該賣又不敢賣」，說明下跌時券商被迫賣股的壓力同樣驚人。

### 陷阱三：只看自營商總買超

自營商總買超包含「自行買進」和「避險買進」，兩者意義完全不同。必須拆開看，才能判斷避險的影響。

### 陷阱四：忽視權證到期效應

權證到期時，避險部位自動解除。如果大戶在到期日前大量賣出權證，券商也會同步減少避險部位，造成股價急跌。

### 陷阱五：以為所有券商都一樣

大型權證發行商（元大、凱基、群益、國票、富邦等）的避險動態影響力更大，小型券商的權證發行量少，避險影響有限。
