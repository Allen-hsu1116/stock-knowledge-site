# 選擇權籌碼綜合判讀框架Option-Chip-Integrated-Framework

> 系統化整合選擇權五大籌碼工具（P/C Ratio、支撐壓力表、法人OI、交易人區隔、OI增減流動），建立每日判讀SOP與多空訊號交叉確認機制。

## 核心概念

選擇權籌碼分析不應單獨看一個指標，必須將**五大工具**系統化整合，才能得到可靠的市場多空判斷。每個工具看的是不同維度：

| 工具 | 看什麼 | 時間尺度 | 核心問題 |
|------|--------|----------|----------|
| P/C Ratio | 整體多空氛圍 | 中長期 | 莊家偏多還是偏空？ |
| 支撐壓力表 | 關鍵價位 | 短中期 | 支撐壓力在哪？ |
| 法人OI | 法人方向 | 中期 | 外資自營商怎麼看？ |
| 交易人區隔 | 大戶vs散戶 | 中期 | 大戶散戶誰在對做？ |
| OI增減流動 | 籌碼動態 | 短期 | 籌碼正在往哪裡移？ |

### 前提：以賣方角度解讀

所有選擇權籌碼分析的基礎前提：

- **賣方留倉成本高**（保證金3-5萬），不會輕易留倉
- **賣方是長線思考**，希望收權利金歸零，代表莊家立場
- **買方是短線投機**，今天進明天出，留倉意義不大
- **解讀口訣**：Sell Put = 看多（訂地板）、Sell Call = 看空（訂天花板）

## 實戰應用

### 第一層：P/C Ratio 看全貌

**數值定義**：Sell Put OI ÷ Sell Call OI × 100%

| P/C Ratio | 意義 | 判讀 |
|-----------|------|------|
| > 120% | Sell Put遠大於Sell Call | 偏多，支撐力道強 |
| 100-120% | Sell Put略大於Sell Call | 溫和偏多 |
| 80-100% | Sell Call略大於Sell Put | 溫和偏空 |
| < 80% | Sell Call遠大於Sell Put | 偏空，壓力較大 |

**P/C Ratio 與大盤走勢的關係**：
- P/C Ratio 持續增加 → 大盤上漲機率高（正向關聯）
- P/C Ratio 持續減少 → 大盤下跌機率高
- **多空翻轉點**：P/C Ratio 從 < 100% 翻到 > 100%（空翻多），後續上漲機率約 8 成

⚠️ P/C Ratio 是看整體全貌的工具，細節要搭配支撐壓力表。

### 第二層：支撐壓力表看細節

**找出關鍵價位**：
1. Call 最大 OI 的履約價 → 上檔壓力區
2. Put 最大 OI 的履約價 → 下檔支撐區
3. **損平點才是真正點位**：Call損平點 = 履約價 + 權利金；Put損平點 = 履約價 - 權利金

**支撐壓力位移判斷趨勢**：
- 賣權支撐逐漸上移 → 市場信心增強，偏多
- 買權壓力逐漸下移 → 市場信心減弱，偏空
- 兩者同時上移 → 多頭格局確認
- 兩者同時下移 → 空頭格局確認

**最大增量比最大OI更即時**：
- 最大增量往價內移動 → 追隨趨勢（正常）
- 最大增量往價外移動 → 異常訊號，可能有特定買賣方在佈局

### 第三層：三大法人選擇權OI看方向

**外資**：選擇權市場最重要的方向性指標
- 外資交易是出於需要：幫現貨避險、方向性布局
- 外資大買 Call → 看多
- 外資大買 Put → 看空
- 外資賣 Call + 買 Put → 強烈看空
- ⚠️ 外資可能用選擇權做避險（非方向性），需搭配期貨部位判斷

**自營商**：造市者，部位是被動產生的
- 自營商賣 Call 不代表看空，是造市需求
- **關鍵是看避險方式**：賣 Call 後用期貨空單避險 → 中立；反而加期貨多單 → 偏多
- 有時保持中立，單純賺手續費與點差

**投信**：選擇權部位極小，參考性有限

**推估散戶部位**：
利用外資 + 自營商的選擇權資訊，反向推算散戶的未平倉部位。散戶多做逆勢單，容易形成外資跟散戶兩邊對做的印象。

### 第四層：交易人區隔看對作

期交所每週公布「期貨大額交易人未沖銷部位結構表」，分為：
- **自營商**：造市角色
- **外資**：方向性+避險
- **大戶**（特定資金）：大額交易人
- **散戶**：三方扣除即為散戶

**對作訊號**：
- 外資多 + 大戶多 + 散戶空 → 強烈偏多（主力同方向，散戶反向）
- 外資空 + 大戶空 + 散戶多 → 強烈偏空
- 法人之間方向分歧 → 盤整機率高

### 第五層：OI增減流動看動態

**未平倉金額是進階判讀工具**：
- 某方每日交易為賣、留倉為賣，但留倉金額為正且上升 → 可能在大力買價平合約
- 某方每天都買、留倉上升，但留倉金額沒增加 → 可能在賣價平合約

**成交量 + OI 方向判斷反轉**：
- 成交量增 + OI 減 + 價漲 → 多頭力量降低 → 尋找放空機會
- 成交量增 + OI 減 + 價跌 → 空頭力量降低 → 尋找做多機會
- 成交量與 OI 同向 → 行情延續
- 成交量與 OI 反向 → 行情反轉

## 每日判讀 SOP

### 盤後 15:00-16:00

1. **查 P/C Ratio**：確認 > 100% 或 < 100%，看連續變化趨勢
2. **查支撐壓力表**：記錄 Call/Put 最大 OI 層約價，計算損平點
3. **查三大法人選擇權 OI**：外資 Call/Put 淨部位，自營商避險方式
4. **比較位移**：支撐壓力區是否較前日移動？方向為何？
5. **查最大增量**：哪個履約價增量最大？往價內還是價外？

### 週三結算日注意

- 結算日 OI 大量移轉，支撐壓力表失真
- 結算前 1-2 天避免依賴支撐壓力表
- 雙週選每週三結算，月選每月第三週三結算
- 結算價看加權指數，不是看期貨

### 交叉確認機制

| 訊號強度 | 條件 | 行動建議 |
|----------|------|----------|
| ⭐⭐⭐⭐⭐ | 五層全部同向 | 信心最高，可加碼 |
| ⭐⭐⭐⭐ | 四層同向 | 信心高，正常部位 |
| ⭐⭐⭐ | 三層同向 | 信心中等，標準部位 |
| ⭐⭐ | 兩層同向 | 信心偏低，小部位或觀望 |
| ⭐ | 一層獨行 | 不可靠，不動作 |

**特別訊號**：
- P/C Ratio 多空翻轉 + 支撐壓力同向位移 → 強訊號
- 外資期貨 + 選擇權部位一致 → 強訊號
- 最大增量往價外移動 + OI金額異常 → 可能有主力佈局

## 注意事項

1. **不要硬套過去籌碼**：莊家會隨行情調整部位，昨天偏空今天就可能翻多
2. **選擇權籌碼是輔助指標**：不能單獨做為進出場依據，需搭配技術面和基本面
3. **結算日效應**：接近結算日，所有選擇權籌碼數據都會失真
4. **自營商要看避險方式**：部位本身不重要，避險方式才是自營商的真實看法
5. **極端值才有效**：P/C Ratio 在常態區間（80-100%）參考性有限
6. **VIX搭配**：多頭行情 VIX 緩降（賣方有利），空頭行情 VIX 急升
7. **自動化交易影響**：選擇權大多是電腦單在跑，籌碼是計算結果，可從數學角度推測意圖
8. **散戶是逆勢指標**：散戶多做逆勢單，散戶極度偏空反而是多頭訊號
