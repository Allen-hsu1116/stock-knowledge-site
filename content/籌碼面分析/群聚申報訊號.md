# 群聚申報訊號

> 多位內部人同時或連續申報轉讓，是市場最強烈的看空訊號之一。

## 核心概念

「群聚申報」指同一公司多位內部人（董事、監察人、經理人及其關係人）在短期內同時或接連申報轉讓持股。

單一內部人賣股可能是個人理財需求，但**多人同時申報**幾乎排除個人因素，代表公司內部對未來前景的共識偏向悲觀。

## 實戰應用

### 群聚申號的三種型態

1. **同步申報**：同一日或同一週內，2 位以上內部人同時申報轉讓
2. **連鎖申報**：首位內部人申報後，1-2 週內其他內部人跟進
3. **跨代際申報**：董監事本人 + 配偶 + 子女同時申報，通常是家族出脫

### 判斷嚴重程度的框架

| 維度 | 低嚴重 | 高嚴重 |
|------|--------|--------|
| 人數 | 1-2 人 | 3 人以上 |
| 方式 | 贈與/信託/洽特定人 | 一般交易/鉅額交易 |
| 位置 | 股價低檔 | 股價高檔或剛創新高 |
| 比例 | 占個人持股 < 10% | 占個人持股 > 30% 或清空 |
| 時間 | 分散申報 | 同一週內集中申報 |
| 身份 | 一般經理人 | 董事長/總經理及其配偶 |

### 操作策略

- **出場訊號**：高檔出現 3 人以上同步一般交易/鉅額交易申報 → 優先減碼
- **觀察訊號**：2 人申報 + 轉讓方式為一般交易 → 觀察後續是否有人跟進
- **忽略訊號**：1 人贈與/信託 → 個人理財，不影響操作

## 注意事項

- 必須排除「限制員工權利新股解禁」造成的群聚賣出，這類屬於正常兌現
- 上市櫃公司每月 15 日公告內部人持股變動，可作為定期檢查時間點
- 鉅額逐筆交易（500 張以上）若由多位內部人同時採用，殺傷力最大
- 交叉驗證營收、獲利是否同步衰退，若基本面也惡化則應果斷出場
- 勿以「長期信仰」對抗群聚申報訊號，國巨案例就是慘痛教訓
