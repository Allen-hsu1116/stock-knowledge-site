# 融券回補行情真假辨別

> 券資比超過20%~30%時，股價上漲可能含軋空虛灌成分；比較股價漲幅與盈餘增速可判別真假

## 核心概念

每年3~4月（股東會前）和5~7月（除權息前）是融券回補高峰期。融券放空者必須在停止過戶日前6個營業日回補，形成強制買回壓力。

但股價在這段期間的上漲，可能混合兩種力量：
1. **基本面改善** — 公司獲利成長、產業前景看好
2. **軋空虛灌** — 融券強制回補的機制性買盤推升股價

兩者並存時難以分辨，但判斷錯誤可能買在虛灌的高點，回補潮結束後股價瞬間回落。

## 實戰應用

### 方法一：券資比篩選

| 券資比 | 意義 |
|--------|------|
| < 10% | 軋空壓力極低，上漲偏基本面 |
| 10%~20% | 輕微回補壓力，需搭配其他指標 |
| 20%~30% | 軋空成分可能顯著，需警惕 |
| > 30% | 高機率含軋空虛灌，回補結束後回落風險高 |

實例：信立(4303)券資比高達76%，華泰(2329)券資比21.6%（融券餘額6,874張）。

### 方法二：股價漲幅 vs 盈餘增速比較

**核心邏輯**：如果股價漲幅遠超過盈餘成長率，代表漲幅中含大量非基本面成分。

**步驟**：
1. 計算近幾年盈餘年複合成長率（CAGR）
2. 計算同期股價年複合成長率
3. 比較兩者差距

**案例對照**：
- 中砂(1560)：盈餘CAGR 22.22%，股價年漲 >40% → 漲幅灌水明顯
- 台積電(2330)：盈餘CAGR 16.98%，股價年漲 6.78% → 漲幅較貼合基本面

**結論**：同樣面對AI產業趨勢，中砂股價灌水程度遠高於台積電。

### 方法三：回補潮時間表追蹤

1. 從證交所停券預告表查詢個股最後回補日
2. 追蹤融券餘額變化，觀察是否在回補日前急速下降
3. 若股價在回補日前1~2週急漲，但融券餘額同步大減 → 軋空行情
4. 回補結束後，觀察3~5日走勢：若回落，確認為虛灌

### 多頭格局下的軋空更危險

大盤氛圍偏樂觀時，軋空行情更加劇烈：
- 指數緩漲不需大漲，融券者就可能被逼回補
- 低檔融券激增股（融券增加但股價未過前高）最容易觸發軋空
- 基本面好的個股（營收創新高、本益比低）放空風險更高

## 注意事項

- **基本面vs籌碼面並存難分**：有些股票確實同時具備基本面和軋空動能，不要假設只能二選一
- **軋空結束=買盤消失**：被迫回補的買盤不是看好公司，回補潮一過，買盤瞬間消失
- **不要參與軋空行情的末段**：多數人不應參與短期軋空行情，但每個人都該辨別手上持股是否含軋空成分
- **券資比高不代表一定軋空**：需搭配法人籌碼方向和個股基本面綜合判斷
- **流動性差的股票軋空更極端**：融券餘額雖不大，但成交量更小，更容易被逼到漲停
