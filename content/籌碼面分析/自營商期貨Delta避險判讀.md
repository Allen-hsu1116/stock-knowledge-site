# 自營商期貨Delta避險判讀

> 自營商透過期貨進行 Delta 中性避險，當期貨避險部位出現大幅調整，常暗示市場波動即將加劇

## 核心概念

### Delta 避險的本質

Delta Hedge 是一種動態風險管理技術，透過持有與衍生性商品部位方向相反的標的資產，使投資組合的總 Delta 值為零。目的是抵銷標的資產價格波動對投資組合的影響。

Delta 避險的核心精神來自「無套利定價」：在不使用自身資金且不承擔價格風險的前提下，建構一個 Delta 中性（Delta Neutral）的投資組合，使標的資產價格發生微小變動時，組合的整體價值保持不變。

### 期貨避險 vs 權證避險

自營商有兩條主要的避險管道，機制和判讀邏輯不同：

| 比較 | 權證避險 | 期貨避險 |
|------|---------|---------|
| 觸發原因 | 散戶買進權證，法規強制避險 | 自營商主動調整部位風險 |
| 避險標的 | 現股（個股買賣） | 期貨（台指期、個股期） |
| 避險方向 | Delta > 0 買現股、Delta < 0 賣現股 | 買期貨多單或賣期貨空單 |
| 影響範圍 | 個股為主 | 大盤為主（台指期） |
| 時間特性 | 1-2天短期影響 | 可能持續數週至數月 |
| 判讀重點 | 避險買超是否異常大 | 期貨未平倉量變化 + Delta 值調整 |

### Delta 的定義與計算

Delta (Δ) 衡量標的資產價格每變動一個單位時，衍生性商品價格預期變動的幅度：

- **Call 的 Delta**：0 到 1（正數），股價漲買權就值錢
- **Put 的 Delta**：-1 到 0（負數），股價跌賣權才值錢
- **價平 (ATM)**：Delta 約 ±0.5
- **深價內 (ITM)**：Delta 接近 ±1
- **深價外 (OTM)**：Delta 接近 0

**Delta 的雙重含義**：
1. 避險比率：1 份 Call 的風險等效於 0.5952 股股票
2. 機率近似：Delta 0.5952 ≈ 59.5% 的機會在到期時處於價內

### 避險比例計算

核心公式：Hedge Ratio = 1/Δ

實例：持有 100,000 股股票要保護，Call Delta = 0.5952
→ 需要的期權數量 = 100,000 / 0.5952 = 168,010 份

### 期貨在 Delta 避險中的角色

期貨的 Delta 恆為 1（每口小台指 Delta = 1），計算相對簡單：
- 持有 100,000 股現股 → Delta = 100,000
- 每口小台 Delta = 1 × 200（點數）= 需要賣出 500 口小台達到 Delta 中性
- 實務上用大台（每口 Delta = 1 × 200 點 × 5）約 100 口

## 實戰應用

### 1. 判讀自營商期貨避險部位

自營商期貨避險的觀察重點：

- **期貨未平倉量 (OI) 大幅變動**：自營商避險部位調整
- **自營商期貨買賣超異常**：大額買超可能是在建立多頭避險
- **選擇權 Delta 值公布**（期交所每日公布）：對照自營商期貨部位變化
- **VIX 上升 + 自營商期貨空單增加**：避險需求升溫

### 2. Delta 中性策略的應用場景

- **集中部位避險**：財富經理利用指數期貨規避高集中度部位風險
- **市場中性策略**：預期波動率變化但不預測方向，透過 Delta 避險消除方向風險
- **動態資產配置**：機構法人利用期貨作為覆蓋工具，調整曝險程度

### 3. 從 Delta 避險推斷市場情緒

當自營商大規模調整 Delta 避險部位：
- **大幅增加期貨空單避險**：預期市場下跌風險升高
- **減少期貨避險部位**：風險預期降低，可能轉為偏多
- **頻繁調整避險比例**：市場進入高波動期

### 4. 散戶可觀察的指標

- 期交所「選擇權每日 Delta 值」公布
- 三大法人期貨買賣超（觀察自營商部分）
- 自營商避險買超金額（個股層面）
- VIX 恐慌指數與避險需求連動

## 注意事項

### 1. Delta 避險不是一勞永逸

Delta 會隨股價變動而改變（Gamma 效應），需要持續動態調整。若忽略 Gamma，避險組合可能在價格大幅變動時失效。

### 2. Gamma 風險

當股價在價平附近波動時，Delta 變化最快（Gamma 最大），自營商必須頻繁調整避險部位。這時期貨交易量會明顯放大，但不一定代表方向性看法。

### 3. Theta 衰減

時間流逝影響選擇權價值，間接影響 Delta 值。接近到期日，避險比例需要更頻繁調整。

### 4. Delta-Gamma-Theta 三維平衡

完整掌握衍生性商品風險管理需要同時追蹤 Delta、Gamma、Theta 三個 Greeks，而非只看 Delta。

### 5. 避險調整的落後性

自營商避險調整通常在盤後計算，隔天才執行。因此觀察到的避險變化是「昨天」的決策，有落後性。
