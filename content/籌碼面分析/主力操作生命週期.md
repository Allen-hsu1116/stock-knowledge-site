# 主力操作生命週期

> 主力操作個股的四階段：吸貨→洗盤→拉升→出貨，量先價行是核心判斷原則

## 核心概念

任何個股的生命週期不外乎經歷四個階段：
1. **低位吸貨**：主力在底部默默收集籌碼
2. **洗盤換手**：清除浮動籌碼，提高持股成本差距
3. **拉升**：量能發動，股價進入主升段
4. **高位出貨**：利用利好消息和市場情緒派發籌碼

這四個階段構成了一檔股票從興起到幻滅的完整循環，對應[[籌碼九階段]]中的「接→護→拉→換→養→軋→出→套→殺」。

## 實戰應用

### 階段一：吸貨（底部收集）

**量能特徵**：
- 5日均量 < 20日均量，成交量萎縮到窒息量
- 偶爾出現攻擊量（5MV向上交叉20MV），但尚未連續

**籌碼特徵**：
- 融資餘額持續下降（散戶退場）
- 法人開始連續買進3天以上
- 千張大戶持股比例緩步增加

**辨識要點**：
- 底部沒有人注意，消息面偏空
- 利空不跌是重要訊號
- 攻擊量出現後，月均量轉折向上是起漲確認

**主力吸籌的9種K線圖形**：

| 圖形 | 特徵 | 判讀要點 |
|------|------|----------|
| 牛長熊短 | 慢牛走高後快速打壓再建倉，N形K線 | 主力控制建倉價格區間 |
| 紅肥綠瘦 | 低開高走收陽線，陽線為主陰線少 | 主動性買盤推高股價 |
| 窄幅橫盤 | 底部15%幅度內橫盤箱體 | 跌勢轉橫盤 = 主力進場 |
| 年線/半年線橫盤 | 長期盤整在年線附近，量縮 | 後期一陽破多線+量增突破 |
| 下降楔形 | 下降通道末期量能放大 | 突破下降趨勢線確認 |
| 圓弧底 | 底部圓弧，右側放量上衝 | 太標準反而可能騙線；QFII最愛 |
| 頭肩底 | 三重底，右肩放量/不放量 | 不放量後市空間大 = 長莊 |
| 次低位窄幅震盪 | 前期已有倉位才能控住 | 莊家已有底倉 |
| 股票箱體 | 短期股價多次重覆箱型 | 做高低差不超過3次，否則吞餌 |

**吸籌發現時機**：大盤下跌時找逆勢飄紅股，牛市中主力放量吸籌，熊市中底部靜默吸籌。

### 階段二：洗盤（清除浮動籌碼）

**量能特徵**：
- 洗盤量能由大到小，呈縮量過程
- 量能萎縮後再放量突破

**籌碼特徵**：
- 融資繼續減少
- 主力利用震盪嚇出信心不足的投資者

**洗盤型態**：詳見[[主力洗盤甩轎手法]]
- 橫向洗盤、短線拉回洗盤、K線洗盤、甩轎等

### 階段三：拉升（主升段）

**量能特徵**：
- 多頭量能型態：5日均量 > 20日均量，且20日均量向上
- 量增價揚，形成良性互動
- 攻擊量連續出現，推升股價

**籌碼特徵**：
- 法人持續買超
- 融資可能開始增加（散戶追價）
- 大戶持股穩定或微增

**操作要點**：
- 守住第一防守線（長紅K最低點）續抱
- 跌破第一防線但不破第二防線 → 較長整理
- 跌破第二防線 → 考慮減碼

### 階段四：出貨（高位派發）

**量能特徵**：
- 高檔放量但不創新高
- 量價背離：價漲量縮

**籌碼特徵**：
- 大單賣出小單買入（拆單出貨）
- 利好消息頻繁但股價不漲
- 法人轉為賣超

**辨識要點**：詳見[[主力出貨辨識]]

### 散戶視角：養→套→殺完整循環

從散戶的角度看主力操作，可歸納為「養、套、殺」三步：

1. **養**：主力讓散戶嚐到甜頭。初期小漲讓跟單者獲利，培養散戶的「主力會護盤」信心，使其放鬆警戒、加碼追價。對應生命週期的「拉升」階段後段。

2. **套**：股價開始橫盤或小跌，散戶以為是正常回檔而不願停損。主力持續出貨，散戶越套越深。對應「出貨」階段中期，量價開始背離，利多不漲。

3. **殺**：主力出貨完畢後停止護盤，股價急殺跌破支撐，散戶恐慌性拋售。對應出貨末期，籌碼完全從集中轉為分散。

**散戶防禦核心**：
- 不要有「主力會護盤」的預設心態
- 設定明確停損點，虧損10%無條件出場
- 利多不漲 = 最高級別出貨訊號
- 籌碼分析是輔助工具，不是唯一依據

### 多頭量能型態判斷

多頭的量能型態兩個條件：
1. 短期5日均量 > 20日均量
2. 20日的月均量必須向上

如同股價均線：長均線趨勢向上 + 短均線 > 長均線 = 多頭格局

### 量價背離的兩種例外

**1. 中段整理的量價背離**
- 多頭維持中段整理，並非由多翻空
- 股價在區間整理但量能轉為空方型態
- 以季線為支撐關卡，略微跌破又快速回復可接受

**2. 高價股量價背離**
- 高價股飆漲時追價意願轉弱，出現價漲量縮
- 這並非主力出貨，而是籌碼相對安定
- 建議搭配法人進出指標判斷

## 注意事項

1. **量先價行**：成交量是領先指標，必須先有攻擊量才有起漲
2. **吸貨階段最難辨識**：底部沒人注意，需要籌碼追蹤經驗
3. **洗盤和變盤容易混淆**：守住防線是洗盤，跌破防線是變盤
4. **出貨是過程不是事件**：主力不會一天出完，觀察持續性訊號
5. **搭順風車的時機**：攻擊量出現+法人連續買進+融資下降 = 最佳進場點
