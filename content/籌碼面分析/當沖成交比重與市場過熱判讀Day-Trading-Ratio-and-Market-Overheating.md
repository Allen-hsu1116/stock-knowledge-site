# 當沖成交比重與市場過熱判讀Day-Trading-Ratio-and-Market-Overheating

> 當沖成交比重是衡量市場短線交易熱度與籌碼穩定度的關鍵指標，比重過高常伴隨市場過熱與反轉風險。

## 核心概念

### 什麼是當沖成交比重？

當沖成交比重（Day Trading Ratio），又稱日沖比率，是指「現股當沖成交金額（或張數）」佔「市場總成交金額（或張數）」的比率。這個數字直接反映了市場中有多少交易行為是當天買進當天賣出的短線進出。

**計算方式**：
- 當沖成交比重 = 現股當沖成交金額 / 市場總成交金額
- 當沖成交股數比重 = 現股當沖成交股數 / 市場總成交股數

**2026年5月實況**：台股當沖總成交股數占整體比重約25%，成交金額占比約38%，散戶當沖比重占53%，勝過法人。這意味著近四成的市場交易金額來自當天買賣的短線行為。

### 短線交易比例指標

XQ開發的「短線交易比例」指標，將當沖與融資行為合併觀察：

**公式**：短線交易比例 = (融資買進張數 + 現股當沖張數 + 資券互抵張數) / 成交張數

這個指標把三種短線籌碼行為合在一起，更全面地反映短線資金佔比：
- **融資買進**：槓桿做多
- **現股當沖**：日內進出
- **資券互抵**：當沖的一種形式

### 當沖比重與市場過熱的關係

當沖比重升高代表什麼？
1. **短線資金主導**：長期投資者退場，投機資金接管
2. **籌碼不穩定**：當沖者不留倉過夜，收盤前賣壓增加
3. **市場過熱訊號**：歷史經驗顯示，個股短線交易比例接近60%時常是反轉點
4. **散戶占比提升**：當沖自然人占比53%超過法人，反映散戶情緒高漲

## 實戰應用

### 大盤層級：當沖比重判斷市場溫度

| 當沖成交金額占比 | 市場狀態 | 操作建議 |
|:---:|:---:|:---:|
| < 20% | 冷門盤 | 量能不足，觀望為主 |
| 20%~30% | 正常盤 | 正常交易，依策略操作 |
| 30%~38% | 活躍盤 | 注意短線過熱跡象 |
| > 38% | 過熱盤 | 居高思危，減碼或避開追高 |

**關鍵閾值**：
- 大盤當沖比重超過35%：短線資金過度活躍，需警戒
- 個股短線交易比例達59%-60%：強烈反轉訊號
- 當沖比重創新高時搭配量增：高機率是波段高點

### 個股層級：短線過熱篩選

XQ的短線過熱選股條件：
1. 成交量比前一天多（可設5%以上）
2. 短線交易比例 ≥ 0.59（接近60%）
3. 近20日漲幅超過15%

當三個條件同時滿足，代表個股短線資金擁擠、籌碼鬆動，追高風險極大。

### 搭配其他散戶指標交叉判讀

**小台散戶多空比**：
- 公式：小台散戶留倉量 / 小台全體未平倉量
- 正值越大 → 散戶做多越多 → 反向思考偏空
- 負值越大 → 散戶做空越多 → 反向思考偏多
- 通常被視為反向指標

**大盤融資維持率**：
- 低於140% → 散戶斷頭壓力增加 → 波段低點可能接近
- 高於180% → 散戶槓桿偏高 → 市場可能過熱

**CNN恐懼貪婪指數**（美股參考）：
- 75以上：極度貪婪，短線可能回檔
- 25以下：極度恐懼，可能是低點

### 當沖比重的歷史經驗

1. **量增+當沖比飆升**：通常出現在波段高點附近，短線資金最活躍的時候就是最危險的時候
2. **個股案例**：雄獅（旅遊股）急漲後出現量增且短線交易比例接近0.6，隨即反轉
3. **2026年5月**：台股連續破兆成交量，當沖占比38%，散戶當沖比重53%，呈現明顯過熱跡象

## 注意事項

### 陷阱與誤區

1. **當沖比重低不等於看多**：低當沖比重可能是因為市場低迷、流動性差，不是進場訊號
2. **產業特性差異**：金融股、ETF當沖比重本來就偏高，不適合用同一標準判斷
3. **當沖比重是結果不是原因**：當沖比重升高是市場過熱的「症狀」，不是造成過熱的原因
4. **需搭配趨勢判斷**：多頭市場高當沖比可能持續很久，不能單一指標就反向操作
5. **回補效應**：當沖者收盤前強制回補可能造成尾盤波動加大，不是真正的多空力量

### 實戰限制

- 當沖比重數據為盤後公布，有延遲性
- 大盤當沖比重與個股當沖比重意義不同，不可混用
- 隔日沖（T+1賣出）不在當沖統計內，但同屬短線行為
- 當沖比重的歷史數據受制度變化影響（如當沖稅率調整）
