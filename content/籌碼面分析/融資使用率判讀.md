# 融資使用率判讀

> 融資使用率是散戶看多情緒的溫度計，過高代表追價力道耗盡，反轉風險升高

## 核心概念

### 什麼是融資使用率？

融資使用率 = 融資餘額 ÷ 融資限額

- **融資餘額**：目前市場上用融資買進尚未償還的金額
- **融資限額**：證交所規定融資餘額不得超過該公司股票發行數量的25%

**重要**：融資使用率不是「所有流通股數的百分比」，而是以25%融資限額為基準。例如融資使用率80%，表示在最大可融資額度（流通股25%）中，已使用了80%。

### 融資使用率代表什麼？

融資使用率代表散戶看多情緒：
- **使用率高** → 散戶看多的人多，但訊息可能已傳遞到末端
- **使用率低** → 散戶參與度低，籌碼面相對乾淨

## 實戰應用

### 1. 融資使用率過高的風險（反轉訊號）

當融資使用率過高時：
- **追價力道耗盡**：欲追價的散戶多已進場，後勢將有力道缺乏的可能
- **斷頭賣壓**：股價下跌時，融資使用者被迫賣出，產生連鎖賣壓
- **訊息末端**：散戶得到的資訊常落後於公司派及法人，融資使用率高代表訊息已傳遞到末端，為股價反轉的開始

**經驗值**：個股融資使用率超過60-70%就要警覺，超過80%是高風險區。

### 2. 融資使用率過低（機會訊號）

當融資使用率過低時：
- 散戶參與度低，籌碼集中在法人/大戶手中
- 若搭配法人買超、技術面轉強，可能是進場時機
- 低融資使用率代表下跌時不會有融資斷頭的連鎖賣壓

### 3. 融資使用率趨勢判讀

- **持續攀升**：散戶不斷進場，若股價也漲，注意何時力道耗盡
- **急速攀升**：短時間融資暴增，通常伴隨股價急漲，風險最大
- **持續下降**：散戶退場或被斷頭，若伴隨法人接手，可能是底部
- **急速下降**：融資斷頭潮，若股價止跌，可能是落底訊號

### 4. 大盤融資使用率 vs 個股融資使用率

- 大盤融資使用率看整體散戶情緒
- 個股融資使用率看該股籌碼穩定度
- 兩者要分開判讀，大盤融資高不代表個股危險

### 5. 搭配融券使用率

- 融資使用率高 + 融券使用率低 = 散戶一面倒看多，最危險
- 融資使用率低 + 融券使用率高 = 散戶一面倒看空，可能軋空
- 融資使用率 + 融券使用率都適中 = 籌碼相對穩定

## 注意事項

### 陷阱一：融資使用率高不等於一定會跌
融資使用率高是風險升高的訊號，不代表立即反轉。股價可能在高融資使用率下繼續上漲一段，特別是在多頭市場。

### 陷阱二：你該怕的是融資維持率，不是融資使用率
融資使用率高只是說「很多人用融資」，真正決定會不會被迫賣出的是融資維持率。當融資維持率跌破追繳線（120%），才會有斷頭賣壓。

### 陷阱三：不同產業融資使用率基準不同
- 熱門股、小型股天然融資使用率就高
- 大型權值股、冷門股融資使用率通常較低
- 不要跨產業直接比較數字高低

### 陷阱四：融資使用率是落後指標
融資使用率是日資料，反映的是「昨天」的散戶動向，不是即時的。搭配技術面和法人買賣超一起看才完整。
