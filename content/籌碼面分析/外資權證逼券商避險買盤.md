# 外資權證逼券商避險買盤

> 外資狂掃認購權證，自營商被迫買現貨避險，形成被動買盤推升股價的罕見操作模式

## 核心概念

外資權證逼券商避險是一種「借刀殺人」的籌碼操作手法：外資大量買進認購權證，迫使發行券商（自營商）必須買進等值的現股來避險（Delta Hedge），自營商的避險買盤就成了推升股價的免費助力。

### 運作機制

1. **外資掃權證**：外資大量買進某檔個股的認購權證，權證掛牌不到3天就被收光
2. **自營商被迫避險**：權證發行券商必須買進等值現股來對沖風險（Delta Neutral）
3. **避險買盤推升股價**：自營商的避險買盤不是因為看好，而是被動回補
4. **正回饋循環**：股價上漲→權證價值上升→自營商風險增加→更多避險買盤→股價再漲

### 關鍵數據辨識

| 指標 | 判斷標準 |
|------|----------|
| 自營商避險買超 | 連續數日大量買超，金額遠超自有買超 |
| 權證成交值 | 異常放大，掛牌後迅速被收光 |
| 權證發行量 | 5000~10000張的正常發行量，短時間被掃空 |
| 自營商避險vs自有比例 | 避險佔比超過85%代表被動買盤為主 |

### 2017年經典案例

2017年11月，外資採三種策略齊頭並進湧進台股：
1. **加碼金融股**：外資買超16億元中87%布局金融股
2. **布局反向ETF避險**：同時作多台指期+買台灣50反1
3. **狂掃認購權證**：最罕見的模式，讓自營商幫外資買現貨

結果：自營商連6日買超104.8億元，其中89億元是避險部位，創下當年最長累積買超紀錄。晶電、旺宏等個股權證掛牌3天就被掃光。

## 實戰應用

### 如何偵測外資權證逼券商

1. **查看自營商買超明細**：區分「自行買進」vs「避險買進」，避險佔比異常高是警訊
2. **觀察權證成交值**：當月權證成交值異常放大（如2017年10月551億→11月未過半已565億）
3. **追蹤特定個股權證**：權證掛牌後迅速被掃光是外資進場訊號
4. **注意自營商的交易員困境**：盤中「追漲殺跌」式的進出代表避險操作

### 跟單策略

- **順勢跟單**：確認外資正在掃權證+自營商避險買超，搭順風車做多
- **時間窗**：避險買盤通常持續數日到一週，不是長期買盤
- **出場訊號**：自營商避險買超縮小或轉賣，代表外資停止掃權證

### 避險買盤vs真實買盤的區別

| 特徵 | 避險買盤 | 真實買盤 |
|------|----------|----------|
| 動機 | 被動，因為權證風險 | 主動，因為看好基本面 |
| 持有時間 | 極短，通常隔日沖 | 中長期 |
| 賣出時機 | 股價下跌就賣 | 有停損機制 |
| 對股價影響 | 短期推升但不穩定 | 長期支撐 |

## 注意事項

1. **極為罕見的操作模式**：傳統長線外資不會買權證，這通常是來自香港市場的短線資金
2. **避險買盤是隔日沖**：自營商避險部位隔天極高機率賣出，不要把紅K棒當作真實買盤
3. **正回饋也可能反轉**：股價一旦反轉，自營商會反向避險（賣出），加速下跌
4. **交易員的困境**：盤中追漲殺跌式的避險操作會放大波動，散戶容易被洗
5. **需搭配其他訊號**：外資權證操作只是籌碼面一環，需搭配量價、法人進出、題材事件共同判斷
