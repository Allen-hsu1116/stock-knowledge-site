# 主力買賣超定義與計算方式

> 主力買賣超 = 前15大買超券商張數 − 前15大賣超券商張數，是觀察大戶多空力道最直接的指標，與股價正相關但不等於必漲

## 核心概念

### 主力是誰

「主力」不是特定身分，而是指有能力影響股價的資金力量。只要資金夠雄厚，散戶、三大法人、市場大戶都可能成為主力。

關鍵在於**相對規模**：A股票日均量只有500張，持有100張就能影響股價，你就是主力；B股票日均量10萬張，持有100張根本無足輕重。

### 主力買賣超怎麼算

**主力買賣超 = 買超前15大券商的買超總張數 − 賣超前15大券商的賣超總張數**

- 結果為正數 → 買方主力力道較強，偏多訊號
- 結果為負數 → 賣方主力力道較強，偏空訊號

這個數字每天盤後更新一次，在看盤軟體的「主力進出」頁面可以查到。

### 為什麼看前15大

台股有上百家券商，但真正影響股價的只有少數大戶。取前15大買超和前15大賣超，過濾掉散戶的零星交易，留下真正有影響力的主力動向。

15大是市場慣例，不是硬性規定。有些工具提供前10大、前20大的統計，概念相同。

## 實戰應用

### 主力買賣超與股價的關係

**一般來說，主力買賣超和股價呈正相關**，因為主力手上的資金和籌碼很多，不管是買進或賣出，對於股價的影響都很大。

| 主力買賣超 | 股價表現 | 解讀 |
|-----------|---------|------|
| 連續買超增加 | 股價上漲 | 主力持續加碼，多頭趨勢 |
| 買超減少 | 股價盤整或小跌 | 主力開始縮手，可能轉弱 |
| 轉為賣超 | 股價下跌 | 主力出貨，空頭訊號 |
| 連續賣超但股價不跌 | 抗跌 | 可能有新主力接手或主力洗盤 |

### 搭上主力順風車三步驟

**步驟一：尋找有題材的股票**
主力炒作的前提是有題材可以發揮。市場趨勢、利多消息、產業題材都是主力選股的依據。沒有題材的股票，主力也不會碰。

**步驟二：鎖定無消息但量增的股票**
主力比散戶更早知道利多消息，會在股價低檔默默吃貨。當一檔股票沒有明顯消息但成交量突然大增，可以預測不久後出現利多的可能性很高。

**步驟三：追蹤高手券商動向**
某些券商分點進出後，股價容易出現一波漲勢。但要留意：
- 當沖高手分點 → 當天買當天賣，不適合波段參考
- 隔日沖分點 → 隔天賣出，追價危險
- 波段操作分點 → 適合波段跟單

**選擇與自己操作策略相符的券商分點作為參考，而不是什麼都跟。**

### 主力買賣超的實戰陷阱

1. **主力也會看錯** — 主力買超不代表一定漲，主力也可能虧損出場
2. **主力可以分散下單** — 單一看前15大可能只看到冰山一角，主力可以開多個分點帳戶分散
3. **隔日沖混淆** — 隔日沖分點大買不代表看好，隔天就會賣出
4. **法人分點混入** — 外資複委託會歸在特定分點，不等於本土主力
5. **資訊遞延** — 盤後才公布，看到數據時股價可能已經墊高

### 主力買賣超 vs 三大法人買賣超

| 比較項目 | 三大法人買賣超 | 主力買賣超 |
|---------|-------------|-----------|
| 定義 | 外資+投信+自營商的買賣超 | 前15大券商分點的買賣超 |
| 包含對象 | 法人機構 | 法人+大戶+中實戶 |
| 更新頻率 | 每日盤後 | 每日盤後 |
| 特色 | 身分明確 | 包含非法人的大資金 |
| 用途 | 追蹤法人動向 | 追蹤整體大戶動向 |

兩者可以交叉參考：如果三大法人買超且主力也買超，多頭訊號更強；如果三大法人買超但主力賣超，可能法人在左手換右手。

## 注意事項

1. **主力買賣超是結果不是原因** — 主力買超是因為看好才買，不是因為買了所以會漲
2. **不要100%相信** — 主力買超只是參考指標，必須搭配技術面、基本面綜合判斷
3. **大型股分點多** — 台積電有上千個分點參與，前15大只佔一部分，訊號較弱
4. **小型股分點少** — 小型股參與分點少，前15大幾乎等於全部，訊號更清晰
5. **連續性比單日重要** — 單日買超可能是巧合，連續3-5天買超才有意義
6. **搭配股價位置** — 低檔買超是佈局訊號，高檔買超可能是誘多
