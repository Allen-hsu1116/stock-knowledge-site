# 自營商買賣超完整教學

> 自營商是券商自有資金投資部門，資金量三大法人最小，操作以短線和權證避險為主，避險部位比自行買賣更值得關注——避險買盤隔天大概率賣出，是短線反向訊號。

## 核心概念

### 自營商的定義與角色

自營商是證券公司自己的投資部門，用公司自有資金進行交易，而非代客操作。國內多家券商設有自營部門（統一、元富、康和、群益、宏遠等），每個交易員管理2,000萬至數億元資金部位。

### 自營商三大特徵

| 特徵 | 說明 |
|------|------|
| 資金規模 | 三大法人中最小，通常佔大盤日成交1%~2%，約20~40億元 |
| 操作策略 | 多空皆可，以短線價差為主，部分為權證避險 |
| 投資標的 | 不限大中小型股，但以中小型股居多，也有交易期貨選擇權 |

### 自營商買賣超兩大部位

自營商的買賣超數據分為兩個部位（2014年12月起分開揭露）：

1. **自行買賣部位**：自營商主動買賣，代表自身看法
2. **避險部位**（Hedge）：因發行權證而進行的避險買賣，不代表看好

**關鍵區別**：避險買進≠看好，隔天大概率賣出。

### 自營商避險機制

自營商發行權證給投資人，當投資人買進認購權證（看漲），自營商面臨標的上漲的風險，因此需要買進現股避險：

```
投資人買進認購權證 → 自營商賣出權證（收權利金）
→ 自營商同時買進現股避險（避險部位）
→ 隔天自營商賣出避險部位（回復中性）
```

**結果**：自營商避險買進當天的紅K棒，隔天大概率被賣出，股價容易開高走低。

## 實戰應用

### 自營商自行買賣部位的判讀

- **連續買超**：自營商連續買超某檔股票，短線偏多訊號（但持續性不如外資/投信）
- **連續賣超**：自營商連續賣超，短線偏空訊號
- **參考價值有限**：因為自營商可能今天買明天就賣，散戶得到的資訊永遠是落後的

### 自營商避險部位的實戰用法

1. **反向訊號**：大量避險買進的個股，隔天容易出現賣壓
2. **權證確認**：盤中看到大單湧進，收盤後到權證明細確認是否有人大量買進權證
3. **隔日沖辨識**：避險買進常與隔日沖券商行為重疊（見[[隔日沖券商辨識名單]]）

### 自營商vs外資vs投信 比較

| 項目 | 外資 | 投信 | 自營商 |
|------|------|------|--------|
| 資金規模 | 最大（千億級） | 中等（百億級） | 最小（數十億級） |
| 操作週期 | 中長期 | 中期 | 短線為主 |
| 選股邏輯 | 基本面、產業趨勢 | 基本面+主題性 | 短線價差、權證避險 |
| 持股影響 | 大型權值股為主 | 中大型股 | 中小型股 |
| 參考價值 | ★★★★★ | ★★★★ | ★★ |

### 如何查詢自營商買賣超

- 證交所三大法人買賣超統計表：https://www.twse.com.tw/zh/page/trading/fund/BFI82U.html
- 自營商買賣超彙總表：可查個股別的買賣超資訊
- 看盤軟體：籌碼K線等工具可查分日別的自營商買賣超明細

## 注意事項

### 自營商避險的六大盲點

1. **看到長紅K棒不能直接買進**：如果主要買進來源是自營商避險，隔天容易下跌
2. **技術線型可能被騙線**：高檔時主力可能正在出貨，籌碼已從主力流向散戶
3. **自營商買進不等於看好**：避險部位的買進是被迫的，不代表自營商看好該股
4. **多家券商加總掩蓋方向**：買賣超是多家券商合計，可能有幾家買、幾家賣
5. **避險部位隔天大概率賣出**：這是短線反向訊號，不是跟進訊號
6. **自營商風控嚴格導致短線操作**：虧損達上限即砍倉，行情一有風吹草動就出場

### 自營商跟單的風險

- **資訊落後**：散戶收盤後才看到買賣超數據，自營商可能當天就反手
- **策略不明**：自營商可能同時有多個交易員操作不同策略，整體方向難以判斷
- **避險干擾**：合併看自營商數據時，避險部位會掩蓋自行買賣的真實方向
