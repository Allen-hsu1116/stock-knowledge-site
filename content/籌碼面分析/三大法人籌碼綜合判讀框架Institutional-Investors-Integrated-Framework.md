# 三大法人籌碼綜合判讀框架

> 外資、投信、自營商三種法人各有不同操作慣性，看懂三者的差異與共振訊號，才是籌碼面分析的核心能力

## 核心概念

三大法人（外資、投信、自營商）是台股最重要的機構投資者，長期佔據市場約30%成交比重。三者的操作習性截然不同，資金規模差異巨大，若只看「三大法人合計買賣超」這個數字，等於把三種不同意義的訊號混在一起，判讀價值大打折扣。

### 三大法人操作慣性對比

| 特性 | 外資 | 投信 | 自營商 |
|------|------|------|--------|
| 資金規模 | 最大（佔大盤成交15-20%） | 中等 | 最小 |
| 操作標的 | 大型權值股 | 中小型成長股 | 股性活潑個股 |
| 持股時間 | 中長期 | 短中期 | 極短線 |
| 報酬目標 | 絕對報酬 | 相對報酬（贏大盤） | 絕對報酬 |
| 基本面重視 | 非常重視 | 重視 | 較不重視 |
| 連續性 | 連續買/賣超天數長 | 有連續性 | 進出快速難判方向 |
| 特殊行為 | 假外資隔日沖 | 季底作帳/結帳 | 避險買賣（權證附屬品） |

### 外資深度判讀

外資是三大法人中資金最雄厚者。根據Cmoney數據，自2003年以來，若外資當天為淨買進，大盤上漲機率高達70%，外資隔天續買機率更高達74%。

**外資的構成很複雜**：
- 海外共同基金、ETF：長線操作，重視基本面
- 避險基金：中短線，可能使用衍生性商品
- 假外資：本土資金透過外資券商下單，動機不明

**假外資辨識方法**：
- 中小型股（股本50億以下）出現外資買超，要留意
- 過去外資在該股有隔日沖紀錄（一天大買、一天大賣）
- 外資買超張數佔成交量比例異常高

**外資與大盤關係**：2022年外資在台股提款逾1兆，加權指數下跌22.4%，可見外資對大盤多空的舉足輕重。

### 投信深度判讀

投信是國內基金公司，雖然資金規模不如外資，但基金經理人有地緣優勢，對公司熟悉度高。

**投信內部制度限制**：
- 3日內不能反向操作
- 須維持一定比例持股（通常70%）
- 買財報不好的公司要額外寫報告
- 基金經理人季度績效排名連續落後會被撤換

這些限制使投信必須追求「相對績效」，偏好短期有機會大漲的中小型股。投信的買超標的通常同時具備基本面依據和技術面動能，參考價值高。

**季底作帳/結帳**：
- 作帳：利用現有資金部位拉抬持股，使帳面損益更漂亮
- 結帳：持股水位過高或已達目標價，出脫部位實現獲利

### 自營商深度判讀

自營商是券商用自有資金投資的部門，追求絕對報酬，不管大盤強弱都要賺錢。

**自營商買賣超要拆開看**：
- 自行買賣：交易員的進出紀錄，反映對個股的短期看法
- 避險買賣：發行權證的附屬品，認購權證→買現貨避險，認售權證→放空避險

**自營商的參考性有限**：
- 進出極快速果決，不適合跟單
- 避險部位占比高，數字意義被稀釋
- 但若偏好極短線，自營商異常大動作可作為選題材的參考

## 實戰應用

### 三大法人共振訊號

**三大法人同時買超**是籌碼面最強的偏多訊號之一，代表不同類型、不同操作邏輯的機構投資者都看好同一檔股票。

共振訊號的判讀優先順序：
1. **外資+投信同時買超**：權值股（外資主導）+中小型股（投信主導）都可能出現，訊號最強
2. **投信連續買超5日以上**：投信有「3日內不能反向操作」限制，連續買超代表中期看好
3. **外資忽然大幅買進**：留意成交量是否配合放大，可能是波段攻擊的起點

### 實戰SOP：三大法人籌碼判讀五步驟

1. **看大盤**：先看三大法人合計買賣超，判斷整體多空方向
2. **拆法人格**：分別看外資、投信、自營商的個別買賣超，找訊號方向
3. **找共振股**：外資+投信同時買超的個股，是重點觀察名單
4. **過濾假訊號**：
   - 自營商避險買超→剔除
   - 假外資隔日沖→剔除
   - 法人買超但股價不漲→可能是避險或對沖部位
5. **搭配技術面確認**：法人買超+均線黃金交叉+帶量突破，進場勝率最高

### 實戰情境對照表

| 情境 | 判讀 | 操作建議 |
|------|------|----------|
| 外資連續買超5日+股價放量上漲 | 外資趨勢買盤，偏多 | 回測均線支撐時進場 |
| 投信連續買超+中小型股 | 投信佈局波段行情 | 搭配均線支撐操作 |
| 融資暴增+股價橫盤 | 可能誘多 | 觀察主力出貨跡象 |
| 大戶持股穩步增加 | 籌碼集中 | 逢低佈局，設好停損 |
| 法人買超+股價不漲 | 可能為避險或對沖 | 不急進場，多觀察 |
| 外資大買超+中小型股 | 可能是假外資 | 留意隔日沖風險 |

### 歷史實戰案例

**外資連續買超台積電2年（2016-2018）**：隨著AI、HPC需求提高，外資連續買超台積電整整2年，股價緩步爬升走出超級大多頭。這是外資長線買盤的經典案例——基本面質變驅動外資持續進場。

**投信買超國巨（2017-2018）**：被動元件漲價潮中，投信一路買進國巨，股價突破1000元。投信的地緣優勢讓他們比外資更早發現中小型股的機會。

## 注意事項

1. **法人買超≠股價必漲**：需觀察是否為避險佈局，自營商避險買超不代表看多
2. **主力連買≠沒有風險**：有時為誘多吸籌，須留意基本面與量價配合
3. **三大法人合計數字是懶人指標**：外資賣超100億+投信買超10億+自營商買超5億=合計賣超85億，但投信買超的10億可能是中小型股的強烈偏多訊號
4. **籌碼面是盤後資料**：無法了解當天漲跌的籌碼來源，有時間落差
5. **假外資風險**：中小型股的外資買超要特別小心，可能是隔日沖假外資
6. **只觀察籌碼不看技術面**：籌碼與技術走勢若背離（主力連買但K線破底），應保持警覺
7. **高融券不一定看空**：高融券+股價不跌反漲=潛在軋空行情
