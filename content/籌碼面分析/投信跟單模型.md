# 投信跟單模型

> 投信連續買超2-5天、買超市占率高、K線平台整理——三條件命中飆股機率最高的籌碼策略

## 核心概念

### 為什麼投信最值得跟？

從股市資訊食物鏈來看：

1. **最上層**：公司派（掌握內部消息）
2. **次層**：與公司往來密切的主力及上下游協力廠商
3. **第三層**：法人、媒體
4. **最底層**：一般散戶

股價會不會漲，主要看最上面三層的臉色。三大法人中：

- **自營商**：追求絕對報酬，常短線進出，且進出明細包含權證避險部位，參考性不高
- **外資**：買超有時不單純看好公司，更多是配合指數期貨的拉抬或摜壓
- **投信**：最具有波段進出的參考性，因為投信經理人的選股有基本面支撐，建倉期長

投信的進出，是籌碼分析的核心。

### 投信選股邏輯

投信經理人的選股邏輯，若要大幅加碼，基本面一定要有所本。投信養了幾百位研究員，他們篩選出來的標的，又能讓經理人認同，背後一定有故事。

投信偏好選擇**平台整理**的股票，不太願意替漲太多的股票抬轎。

### 投信內規「二五八」建倉限制

即使再看好標的：

| 階段 | 持倉上限 | 說明 |
|------|---------|------|
| 初始建倉 | 2% | 先試水溫 |
| 獲利加碼 | 5% | 賺錢才加碼 |
| 再漲加碼 | 8% | 差不多滿檔 |

因此投信非常看好的標的，建倉過程一定是延續一段期間。當你發現投信連續買超2到5天的標的，就可以鎖定。

## 實戰應用

### 投信跟單三條件

陳家豐的「投信跟單」模型，三個條件同時滿足時命中飆股機率最高：

1. **投信連續買超 2～5 天**
   - 不是單日買超，而是連續性的買超
   - 代表投信有計畫地建倉，不是隨便買買

2. **投信買超張數市占率高**
   - 根據投信內規，單一基金不得買進超過當天成交量2～3成
   - 如果投信當日買超市占率超過3成，可能是一家以上投信同時買進
   - 多家投信共識 → 命中飆股的機率更高

3. **K線呈現平台整理**
   - 投信不愛追高，偏好低檔盤整區進場
   - 平台整理 + 投信開始連續買超 → 突破機率高

### 實戰案例：台表科（6138）

5月13日開始，投信連續7個交易日買超，股價從75元附近緩步上攻。5月26日投信一口氣大買近3千張，後面連續每天以千張買超的規模，一路將股價推到6月10日的92.7元。

後來台表科果然傳出變頻PCBA新事業明年獲利爆發力強，以及取得中國「十二五」節能家電訂單等利多消息。

**關鍵**：投信的買超領先了利多消息公布，說明投信研究員的選股確實有其價值。

### 買超市占率計算

投信買超市占率 = 投信買超張數 ÷ 當日總成交量

| 市占率 | 解讀 |
|--------|------|
| < 10% | 投信參與度低，訊號不明確 |
| 10%-20% | 正常建倉水位 |
| 20%-30% | 積極建倉 |
| > 30% | 多家投信共識，或單一投信大量買超，飆股機率高 |

### 投信跟單操作節奏

1. **發現期（Day 1-2）**：投信開始連續買超，觀察但不急著進場
2. **確認期（Day 3-5）**：投信連續買超 + 市占率提高 + 股價隨之走高 → 進場時機
3. **加速期**：投信加碼（市占率>30%），股價突破平台整理區
4. **減碼期**：投信買超減少或轉賣 → 考慮出場

### 投信跟單的風險控制

- **停損**：買進後股價跌破平台整理區下緣 → 出場
- **追蹤**：投信轉為賣超連續3天以上 → 出場
- **獲利了結**：投信買超趨緩 + 股價大漲後量縮 → 部分獲利了結

## 注意事項

1. **投信也會看錯** — 投信買超不代表保證上漲，基本面可能發生變化
2. **投信內規限制** — 單一基金買進不超過當日成交量2-3成，過度集中反而可能有流動性風險
3. **投信績效壓力** — 月底、季底投信有作帳壓力，買超可能不是看好而是作帳
4. **投信換股** — 投信可能從A檔賣出轉買B檔，買超B不代表全面看多
5. **多家投信 vs 一家投信** — 多家投信同時買超的訊號強度遠大於單一投信
6. **搭配其他指標** — 投信跟單要搭配基本面（營收成長、獲利改善）和技術面（型態突破）綜合判斷
7. **停損紀律** — 投信買超後股價反而下跌，可能是投信判斷錯誤或基本面惡化，必須設停損
