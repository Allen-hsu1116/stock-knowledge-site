# 融資追繳與斷頭連鎖效應實戰判讀

> 融資維持率跌破130%觸發追繳，未補繳T+3斷頭；多殺多連鎖效應是股災加速器，整戶擔保維持率讓一檔斷頭拖垮全帳戶

## 核心概念

### 融資追繳三階段

融資維持率是券商保護借出資金的機制，跌破130%即觸發追繳：

| 階段 | 維持率 | 時間 | 動作 |
|------|--------|------|------|
| 安全 | >166% | — | 安心持有 |
| 追繳 | 130%~166% | T日收到通知 | T+2收盤前補繳 |
| 斷頭 | <130%且未補 | T+3日 | 券商強制賣出 |

### 追繳補繳的三種結果

1. **補到166%以上**：追繳令完全撤銷，最安全
2. **補到130%~166%**：追繳令不撤銷，之後再跌破需「當日」補繳（沒有2天緩衝）
3. **未補或補不到130%**：T+3強制賣出 = 斷頭

### 「每日盯盤」陷阱

只補到130%~166%的投資人最常忽略的風險：追繳令不撤銷，每天收盤都重算維持率。只要盤中又低於130%，就要「當日下午」補錢，不再有2天緩衝。很多人以為補了錢就安全，結果隔天再跌又被迫斷頭。

### 整戶擔保維持率

券商不只看單一股票維持率，而是看整個帳戶的「整戶擔保維持率」：

**整戶擔保維持率 = 帳戶內所有擔保品市值 ÷ 整戶融資金額 × 100%**

- A股票賺錢可以掩護B股票虧損，但整戶跌破130%全部追繳
- 一檔股票大跌可能拖垮整個帳戶，被迫賣掉獲利股來救虧損股

## 實戰應用

### 斷頭連鎖效應（多殺多）

連鎖反應流程：
1. 股價下跌 → 融資維持率跌破130%
2. 券商發出追繳通知 → 融資戶無力補繳
3. T+3強制賣出（斷頭） → 大量賣壓湧入市場
4. 供過於求股價續跌 → 更多融資維持率跌破130%
5. 更多斷頭、更多賣壓 → 惡性循環

**數學案例**：100元融資買進（維持率166%），連續3根跌停板：
- 1根跌停→90元，維持率150%（安全）
- 2根跌停→81元，維持率135%（接近追繳）
- 3根跌停→72.9元，維持率121.5%（追繳啟動）

融資使用率>60%的個股，追繳潮一啟動很容易打到無量跌停。

### 股災三階段

1. **點火**：外部因素（政策、國際股市、消息）引發下跌
2. **融資斷頭潮**：維持率跌破130%，追繳→斷頭→更多賣壓的死亡螺旋
3. **流動性風險**：無量跌停，想賣賣不掉，市場信心崩潰

只要流動性風險未解，股市很難見底。等融資清洗完畢、市場呈現死水一段時間、信心回穩又開始量增，才是底部訊號。

### 2025年4月實戰案例

台股單日下跌2,065點（跌幅9.6%），大盤融資維持率降至133.8%，低於130%的個股高達589檔，形成連環追繳潮。融資餘額大幅清洗後，量縮紅K出現才是真正底部。

### 從外部觀察斷頭訊號

**個股層面**：
- 融資使用率>60%：高風險警區
- 融資維持率接近130%：追繳壓力浮現
- 融資餘額突然大減：斷頭已在發生
- 連續跌停+融資高：多殺多最危險組合

**大盤層面**：
- 大盤維持率<140%：系統性風險升溫
- 大盤維持率<130%：大規模追繳潮啟動
- 融資餘額急降+量暴增：恐慌性出清（可能是底部）
- 融資餘額穩定+維持率低：散戶撐著沒跑，反彈機率較高

### 反轉訊號辨識

融資斷頭潮底部不是V型反轉，而是四步確認：
1. 融資餘額大幅清洗（減少30%~50%）
2. 成交量從暴增轉為量縮
3. 出現紅K線（收盤>開盤）
4. 法人開始買超承接

**心法**：不要在斷頭潮正烈時接刀。等融資清洗完畢、量縮紅K出現，才是進場時機。

### 融資操作安全守則

1. 融資使用率控制在30%以下，不要把額度用到滿
2. 設定強制停損線：維持率低於140%就減碼，不要等追繳
3. 分散標的：不要全壓單一股票，一檔斷頭拖垮整戶
4. 避開高融資使用率個股：使用率>40%的股票少碰
5. 注意除權息假追繳：除息日前算好維持率變化

### 觀察他人斷頭的反向操作

- 大盤維持率驟降至130% → 全市場恐慌 → 等量縮紅K是買點
- 個股融資餘額暴減40%以上 → 斷頭清洗完成 → 觀察法人是否承接
- 融資使用率從高位急降 → 散戶出場 → 可能是底部區

## 注意事項

- **追繳不是斷頭**：有2天緩衝期可補救，但只補到130%~166%不等於安全
- **整戶維持率比個股維持率重要**：一檔股票斷頭會拖垮整個帳戶
- **斷頭賣出不計成本**：券商只管收回借款，不會替你挑好價格賣
- **融資利率6~7%偏高**：長期持有利息會吃掉利潤，融資只適合短中期操作
- **融資餘額是散戶反向指標**：融資暴增是頭部警訊、融資暴減是底部訊號
- **無量跌停是最可怕的**：連跑都跑不掉，只能等開板
- **除權息會影響維持率**：股價調降使維持率下降，需提前準備
