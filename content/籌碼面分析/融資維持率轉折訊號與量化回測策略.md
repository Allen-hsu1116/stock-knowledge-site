# 融資維持率轉折訊號與量化回測策略

> 融資維持率異常低檔是反轉訊號，搭配紅K、量縮、融資餘額穩定三重濾網，回測年化18.39%

## 核心概念

融資維持率（Maintenance Margin Ratio）是衡量融資帳戶風險狀況的指標，也能作為投資選股與避險的工具。TEJ提供的融資維持率是針對每檔股票所有融資戶的統計，反映整體平均帳戶的風險狀況。

**計算公式**：
融資維持率 = 名目股價 / (每股融資成本 × 融資乘數) × 100%

- 上市公司融資乘數：60%
- 上櫃公司融資乘數：50%

**範例**：100元買進上市股票
- 原始融資維持率 = 100 / (100 × 60%) = 166%
- 股價跌至78元 = 78 / 60 = 130%（觸發追繳）
- 股價跌至60元 = 60 / 60 = 100%（高風險斷頭區）

### 為何出現極低維持率？

1. **高檔融資進場**：散戶常在股價創高時追高，平均融資成本偏高
2. **股價急跌**：如AI、航運、重電類股修正，帳戶價值迅速縮水
3. **未及時補繳保證金**：投資人選擇觀望或無力補繳
4. **市場恐慌**：引發連鎖斷頭效應，融資餘額與維持率惡化

## 實戰應用

### TEJ回測策略：融資維持率異常低檔反轉

**進場條件（四重濾網）**：

1. **融資維持率異常低檔**：當日TEJ融資維持率 < 過去10日移動平均值
2. **成交量確認**：當日成交量 < 過去10日平均量（籌碼面趨於穩定，融資散戶已出場）
3. **紅K反轉訊號**：當日收盤價 > 開盤價（出現反轉時機）
4. **融資餘額穩定**：當日融資餘額 > 前5日平均融資餘額 × 0.95（剩餘融資戶仍撐著，反彈機率較高）

**出場條件**：
- 持有15個交易日
- 停利 +40%
- 停損 -10%
- 若15日內再次觸發進場訊號，以最新訊號日重新計算出場條件

### 回測結果（2020-2025）

| 績效指標 | 融資維持率策略 | 大盤Benchmark |
|---------|-------------|-------------|
| 年化報酬率 | 18.39% | 19.04% |
| 累積報酬率 | 110.51% | 115.6% |
| 年化波動度 | 17.67% | 16.897% |
| 夏普值 | 1.04 | 1.12 |
| 卡瑪比率 | 0.65 | 1.14 |
| 最大回撤 | -28.47% | -16.68% |

### 策略評估

- **最大回撤控制優於大盤**：有效降低極端波動對資金部位的衝擊
- **年化報酬率略高於大盤**：具備一定超額表現
- **累積報酬長期優於大盤**：良好的資產增值能力
- **年化波動度低於大盤**：承擔較低風險仍能提供不錯報酬
- **卡瑪比率較低**：最大回撤期間的策略表現不如大盤

### 2025年4月實戰案例

- 單日下跌2,065點（跌幅9.6%），台股史上第三大單日跌點
- 融資維持率降至133.8%
- 低於130%的個股達589檔
- **緯創(3231)**：融資維持率下探至91%，絕大多數融資戶浮虧
- **技嘉(2376)**：維持率跌破100%，引發多日追繳賣壓
- 這些個股後續反彈明顯弱於大盤，因籌碼凌亂、浮虧戶未清洗完全

## 注意事項

- **融資維持率是個股風險的溫度計**：不是大盤方向指標，而是個股壓力訊號
- **維持率低不代表一定會反彈**：需要搭配紅K確認、量縮穩定、融資餘額未大幅流出
- **回測績效不代表未來**：策略在特定市場環境有效，不同市況需要調整參數
- **個股維持率數據有落後性**：T日交易，T+1才公布
- **融資維持率極低時的斷頭風險**：可能形成多殺多惡性循環
