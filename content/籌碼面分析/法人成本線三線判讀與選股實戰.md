# 法人成本線三線判讀與選股實戰

> 法人成本線是加權均線的嚴謹版，加入身分別與交易量考量，精確反映外資、投信、融資的持倉成本，站上成本線做多、跌破做空，搭配持股比例10%門檻篩選外資認養股

## 核心概念

### 什麼是法人成本線？

法人成本線與一般均線（MA）不同，均線只統計全市場的平均價格，不考慮成交量與身分別；而成本線還加入了**身分別（外資/投信/融資）**與**交易量**的考量，更能忠實反映特定法人群體的持倉成本。

**三條成本線對應三種法人：**

| 成本線 | 對應法人 | 適用投資週期 | 特性 |
|--------|---------|-------------|------|
| 融資成本線 | 散戶（融資買進） | 短線 | 貼近股價走勢，波動大 |
| 投信成本線 | 投信法人 | 中線（3個月~半年） | 波段操作，有風控停損線 |
| 外資成本線 | 外資法人 | 長線（1年以上） | 資金量大，波段進出 |

### 計算方式

**精確區間計算法（投信成本線推算）：**

1. 找出投信連續買超的起點日期（對照股價與投信買賣超圖）
2. 選取該區間的三大法人買賣超資料
3. 計算區間內的加權平均成本：`Σ(每日買超張數 × 收盤價) / Σ(每日買超張數)`

**注意**：不能用固定天數（如5日、20日、60日均線）來推算法人成本，必須先用買賣超籌碼決定區間再計算，才能精確推算持有成本。

**滾動窗口法（Python量化）：**

```python
# 外資買進成本（60日滾動窗口）
外資買進成本 = (外資買進 * 收盤價).rolling(60, min_periods=1).sum() / 外資買進.rolling(60, min_periods=1).sum()

# 外資賣出成本（60日滾動窗口）
外資賣出成本 = (外資賣出 * 收盤價).rolling(60, min_periods=1).sum() / 外資賣出.rolling(60, min_periods=1).sum()
```

## 實戰應用

### 外資成本線選股三步驟

1. **篩選外資認養股**：外資持股比例 > 10%（外資成本線才有參考價值）
2. **突破外資成本線**：股價從外資成本線之下突破
3. **確認非假突破**：觀察3~5天是否站穩外資成本線之上

### 外資成本線多空判斷

| 情境 | 判斷 | 操作 |
|------|------|------|
| 股價站上外資成本線 | 外資幾乎都賺錢 → 繼續加碼 | 拉回到成本線附近是買點 |
| 股價跌破外資成本線 | 外資幾乎都套牢 → 停損賣出 | 反彈到成本線附近是空點 |
| 第一次突破（由空翻多） | 可能假突破 | 投入1/3資金試單 |
| 第二次突破 | 解套賣壓消化完畢 | 較佳買點，加碼進場 |
| 第一次跌破（由多翻空） | 可能假跌破 | 觀察3~5天確認 |

### 投信成本線風控停損應用

投信有嚴格的風控制度：

- 投信買超持有成本推算後，若股價跌破成本**超過10%**，可能觸發風控停損
- 觸發停損後，投信會連續賣超2~3天
- 因此投信成本線下方約**-10%**是重要的觀察價位

### 買進成本 vs 賣出成本交叉策略（FinLab量化）

- **當買進成本 > 賣出成本**：外資買在更高價，偏多訊號 → 可考慮買進
- **當買進成本 < 賣出成本**：外資賣在更高價，偏空訊號 → 觀望或減碼
- **股價拉回到買進成本線**：與外資成本接近 → 絕佳買點

## 注意事項

1. **外資持股比例門檻**：外資持股比例低於10%的個股，外資成本線參考性不足，因為外資影響力有限
2. **乖離過大不追高**：股價和成本線拉開一段距離後，不要再加碼進場
3. **假突破風險**：第一次突破不追，等3~5天確認站穩再進場；第二次突破是更好的買點
4. **投信成本推算不用固定天數**：必須用買賣超籌碼決定區間，再計算加權平均成本
5. **投信風控停損**：投信虧損超過10%可能觸發風控停損，連續賣超2~3天
6. **三條線不要混用**：短線看融資成本線、中線看投信成本線、長線看外資成本線，不要用錯週期
