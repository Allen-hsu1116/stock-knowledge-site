# 選擇權Put-Call-Ratio籌碼判讀

> 透過選擇權 Put/Call Ratio 判讀市場多空情緒與支撐壓力區，是籌碼面分析的重要工具，更是恐懼與貪婪指數的核心因子。

## 核心概念

**Put/Call Ratio（PCR）** 是選擇權市場中最常用的多空情緒指標，分為兩種：

- **成交量 PCR**：Put 成交量 / Call 成交量，反映短期（當日）交易情緒，最常用
- **未平倉量 PCR**：Put OI / Call OI，反映中長期留倉心態，觀察持倉結構性變化

**選擇權 OI 以「賣方」角度解讀**：
選擇權賣方需要負擔較高保證金，且承擔低機率大行情損失的風險，因此賣方代表更有資金實力的莊家觀點。

- Call OI 增加 → Sell Call 增多 → 不看漲、看有壓力
- Put OI 增加 → Sell Put 增多 → 不看跌、看有支撐

### 成交量 PCR vs 未平倉量 PCR 實戰差異

兩種 PCR 反映的市場意涵截然不同：

| 類型 | 計算方式 | 反映內容 | 實戰用法 |
|------|----------|----------|----------|
| 成交量 PCR | Put 成交量 / Call 成交量 | 當日交易情緒，短期即時 | 判斷短線多空氣氛 |
| 未平倉量 PCR | Put OI / Call OI | 中長期留倉結構，持倉心態 | 判斷趨勢方向與支撐壓力 |

**關鍵差異**：未平倉量 PCR 下降 = 多方部位增加 / 空方停損；未平倉量 PCR 上升 = 空方部位增加 / 多方停損。這與成交量 PCR 的解讀不同，成交量只看當天交易，未平倉看的是留在場內的資金。

### 成交量與未平倉量反向趨勢 = 行情反轉訊號

這是一個非常重要但常被忽略的實戰技巧：

- **成交量增加 + 未平倉量減少 + 股價上漲** → 空頭停損出場 + 多頭獲利了結 → 多頭力量降低 → **尋找放空機會**
- **成交量增加 + 未平倉量減少 + 股價下跌** → 多頭停損出場 + 空頭獲利了結 → 空頭力量降低 → **尋找做多機會**
- **成交量與未平倉量同向** → 行情延續機率高
- **成交量與未平倉量反向** → 行情反轉機率高

### Equity PCR vs Index PCR

進一步區分可以窺探散戶與大戶的不同動向：

| 類型 | 代表對象 | 特性 | 實戰意義 |
|------|----------|------|----------|
| Equity PCR（個股選擇權） | 散戶情緒 | 散戶偏好交易熱門個股選擇權投機 | 極端值是可靠的散戶逆向指標 |
| Index PCR（指數選擇權） | 機構避險 | 法人用指數選擇權做系統性避險 | 數值偏高可能是常態避險，非看空 |

**關鍵差異**：Index PCR 通常高於 Equity PCR，因為機構避險需求是常態。Index PCR 飆升可能只是常規風險管理，作為逆向指標的敏感度較低。

## 實戰應用

### PCR 判讀兩大派別

**第一派：直接解讀**
- PCR > 1：Put OI > Call OI，市場氣氛偏空
- PCR < 1：Call OI > Put OI，市場氣氛偏多

**第二派：反向指標（散戶指標）**
選擇權市場多為散戶參與，PCR 被解讀為散戶情緒指標：
- PCR 數值越高 → 散戶偏空 → **反向看多**
- PCR 數值越低 → 散戶偏多 → **反向看空**

實務上，第二派（反向指標）在台股指數選擇權更常被使用。

### 逆向指標的極端值門檻

PCR 作為逆向指標，極端值比中間值更有參考意義：

- **PCR 飆高（>1.2 或 >歷史 90 分位）**：市場極度恐慌，幾乎所有人都買 Put 避險，拋售壓力接近耗盡 → 潛在買入信號
- **PCR 極低（<0.5 或 <歷史 10 分位）**：市場極度樂觀，投資者瘋狂買 Call，忽視下跌風險 → 潛在賣出信號
- **常態區間（0.7-1.0）**：美股長期均值在此區間，因為多頭時間長於空頭，買 Call 投機需求略高於買 Put 避險

⚠️ **重要提醒**：「極端」沒有絕對數字標準，需對比歷史數據判斷。台股的基準值與美股不同，應觀察台股 PCR 歷史波動區間建立自己的標準。

### 選擇權未平倉大量區（支撐壓力判斷）

1. 找出 **Call 未平倉量最大的履約價** = 上檔壓力區
2. 找出 **Put 未平倉量最大的履約價** = 下檔支撐區
3. 兩個區間形成一個支撐壓力帶

**進階強化判斷**：
- 參考鄰近履約價的 OI 分布狀況（不只看最大量那檔）
- 觀察單日 OI 增減變化
- 大量區 + OI 明顯增加 = 支撐/壓力效果更顯著

### 三大法人選擇權 OI 觀察

期交所每日公布外資、投信、自營商的選擇權 Call/Put 未平倉部位：
- 觀察法人留倉狀況，衡量其對後市的樂觀/悲觀程度
- 外資在選擇權的部位可與期貨部位交叉驗證

### PCR + 期貨 OI 交叉確認

將 PCR 與期貨未平倉量搭配使用：
1. PCR 偏高（散戶偏空）+ 外資期貨淨多單增加 → 多頭訊號增強
2. PCR 偏低（散戶偏多）+ 外資期貨淨空單增加 → 空頭訊號增強
3. PCR 與期貨 OI 方向矛盾 → 需謹慎，可能處於整理階段

### PCR 回測實戰策略

根據台股回測，以下策略長期有不錯效果：

**基礎策略**：PCR 連續大於閾值 3 天 → 做多；連續小於閾值 3 天 → 做空

**進階策略**：
1. 進場後 3 天如果仍未獲利創新高 → 平倉出場
2. 搭配 K 線型態判斷，提升策略品質穩定度
3. 將 PCR 當作方向濾網，配合趨勢追隨或均值回歸策略使用

**結論**：PCR 適合判斷大盤方向多空，但更適合作為濾網而非獨立進出場訊號。

## 注意事項

1. **PCR 是散戶指標但有極端值**：極端值（>1.5 或 <0.5）更有參考價值，中間區間意義較弱
2. **結算日效應**：結算前 OI 會大量轉倉，PCR 數值會失真，應避開結算前1-2天判讀
3. **賣方不等於大戶**：選擇權賣方主要是造市商和法人，但散戶也會賣選擇權
4. **大量區會移動**：支撐壓力區不是固定的，每日 OI 變化會改變支撐壓力位置
5. **PCR 需搭配趨勢**：不能單獨使用，必須搭配趨勢方向和其他指標綜合判斷
6. **PCR 是同步/微滯後指標**：反映「當下」市場情緒，不是預測未來走勢的工具
7. **不同市場基準不同**：美股 0.7-1.0 基準不直接套用台股，需建立自己的歷史區間
8. **避險需求扭曲數據**：大量機構避險盤推高 Index PCR，不代表市場看空

### P/C Ratio 多空區間判讀表（GoOptions 實戰版）

台股 P/C Ratio 長期均值約 85-95，不宜死守 100 作為分界：

| P/C Ratio 區間 | 市場意涵 | 賣方解讀 | 實務建議 |
|---------------|---------|---------|----------|
| 大於 120 | 極度偏多 | Sell Put 遠大於 Sell Call | 留意散戶過度樂觀，搭配法人籌碼驗證 |
| 100-120 | 溫和偏多 | Put 未平倉略多於 Call | 順勢偏多操作，注意 P/C Ratio 是否持續下滑 |
| 80-100 | 溫和偏空 | Call 未平倉略多於 Put | 短線偏保守，留意是否為洗盤或真正轉空 |
| 小於 80 | 極度偏空 | Sell Call 遠大於 Sell Put | 反向思考：極端值可能是反轉訊號 |

### P/C Ratio 極端值反轉統計

- P/C Ratio 突破 130 後 5 個交易日內，加權指數回檔機率約 65%
- P/C Ratio 低於 70 後 5 個交易日內，指數反彈機率約 60%

### P/C Ratio 每日操作六步驟

1. **每日盤後查看 P/C Ratio**：記錄當日數值與前一日變化方向
2. **判斷整體多空傾向**：連續 3 日以上高於長期均值（約 90）且持續上升=偏多格局
3. **檢查法人籌碼一致性**：確認外資 Sell Put/Sell Call 方向是否與 P/C Ratio 走勢一致
4. **定位支撐壓力點**：找出 Put 與 Call 未平倉最大量的履約價
5. **設定極端值警示**：P/C Ratio 超過 130 或低於 70 啟動反向思考
6. **每週回顧修正**：比對判讀與實際走勢，紀錄準確率

### P/C Ratio + 法人籌碼一致性判斷

- P/C Ratio 升高但外資反而在 Sell Call → 矛盾訊號，值得警惕
- P/C Ratio 升高且外資 Sell Put → 多方訊號可信度高
- 散戶主導的 P/C Ratio 變動訊號可信度較低
