# 小台指散戶多空比反向指標判讀

> 從小台指期貨未平倉量反推散戶多空部位，散戶是標準反指標——做多越多行情越容易跌，做空越多行情越容易漲，極端值是反轉訊號

## 核心概念

### 什麼是小台指散戶多空比

小型台指期貨（MTX）是散戶最常交易的期貨商品，保證金僅為大台指的1/4（約46,000元 vs 184,000元），門檻低因此散戶參與比例高。透過反推法，從小台指未平倉量扣除三大法人部位，就能算出散戶的多空留倉狀況。

### 計算公式

```
散戶小台多單數量 = 小台指期未平倉總量 - 三大法人小台多方未平倉量
散戶小台空單數量 = 小台指期未平倉總量 - 三大法人小台空方未平倉量
台指期散戶多空比 = (散戶小台多單數量 - 散戶小台空單數量) / 小台指期未平倉總量
```

簡化公式：**散戶多空比 = -1 × 三大法人小台淨未平倉量 / 小台指期未平倉總量**

### 為什麼散戶是反指標

1. **人性弱點**：散戶看對不敢下大、賺一點就跑；看錯反而凹單攤平、不肯停損
2. **資金劣勢**：小資金遇到波動就容易恐慌，被迫出場或停損
3. **行為模式**：
   - 散戶多單留倉越多 → 散戶多單套牢 → 行情往往下跌
   - 散戶空單留倉越多 → 散戶空單套牢 → 行情往往上漲
4. **零和本質**：期貨是零和遊戲，散戶的對手就是三大法人，法人長期勝率遠高於散戶

## 實戰應用

### 判讀方向

| 散戶多空比 | 散戶部位 | 預期行情 | 操作方向 |
|-----------|---------|---------|---------|
| 正值越大 | 散戶偏多 | 容易下跌 | 偏空或減碼 |
| 正值適中 | 散戶偏多 | 觀察 | 謹慎 |
| 接近0 | 多空均衡 | 中性 | 搭配其他指標 |
| 負值適中 | 散戶偏空 | 觀察 | 謹慎 |
| 負值越大 | 散戶偏空 | 容易上漲 | 偏多或加碼 |

### 極端值反轉策略

散戶多空比極端值是最核心的交易訊號：

1. **散戶多空比正極端**（散戶過度看多）→ 行情反轉下跌風險大
   - 歷史經驗：加權指數2024年7月創24416新高後回檔18%，期間散戶多空比多日為正值
2. **散戶多空比負極端**（散戶過度看空）→ 行情反轉上漲機率高
   - 歷史經驗：2019年1月跌破9300時散戶積極放空，結果3個月上漲至11000

### 搭配其他指標

散戶多空比不應單獨使用，需搭配：

- **大盤技術面**：趨勢方向確認
- **三大法人現貨買賣超**：法人是否同步佈局
- **融資維持率**：散戶另一面溫度計
- **選擇權PCR**：散戶看空看多的另一驗證

### 資料查詢方式

1. **期交所官網**：交易資訊 → 三大法人 → 區分各期貨契約 → 選擇小型台指期貨
2. **玩股網**：即時小台指散戶多空比圖表
3. **MacroMicro**：歷史散戶多空比走勢圖
4. **永豐金FinLab**：回測功能可驗證策略

## 注意事項

1. **反反指標風險**：當很多人利用散戶多空比當反指標時，可能會有人專門反著這個反指標操作，沒有任何指標永遠100%準確
2. **非即時數據**：散戶多空比是盤後公布的數據，有時間落差
3. **小型台指vs大型台指**：小台指保證金低，散戶比例高，但大台指法人比重更高，兩者要分開看
4. **趨勢市vs盤整市**：趨勢明確時散戶反指標效果更好，盤整市容易出現反覆假訊號
5. **極端值定義因市況而異**：沒有絕對的門檻值，需根據歷史區間判斷當時的極端程度
6. **搭配停損使用**：反指標策略也會失靈，必須搭配停損紀律
7. **轉倉期間數據失真**：期貨轉倉時未平倉量變動大，散戶多空比會短暫失真
