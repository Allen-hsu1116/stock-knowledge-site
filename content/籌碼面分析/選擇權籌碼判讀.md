# 選擇權籌碼判讀

> 選擇權OI是市場燃料，以賣方（莊家）視角解讀支撐壓力，外資選擇權部位是方向性指標，自營商是造市+避險角色

## 核心概念

### 選擇權未平倉量（OI）

選擇權OI是市場上尚未結算的合約口數，代表多空雙方仍在場內對賭的資金量。OI越大，代表參與者越多、潛在動能越大。

**OI增減的意義**：
- OI增加 → 新的多空對決形成，趨勢燃料充足，行情可延續
- OI減少 → 輸家認賠出場、贏家獲利了結，原有趨勢可能結束
- OI不變但價格上漲 → 只是舊買方換手，燃料不增，行情趨緩

### 以賣方視角解讀

選擇權OI應以**賣方角度**解讀，因為：
1. 賣方需要繳保證金（資金門檻高），代表大戶/法人
2. 賣方承擔無限風險，必須對行情有明確判斷才敢進場
3. 賣方OI代表莊家的「看不漲/看不跌」立場

**解讀方式**：
- Call OI大量區 → 賣方認為不會漲過 → **壓力區**
- Put OI大量區 → 賣方認為不會跌破 → **支撐區**

### Put/Call Ratio

$$P/C\ Ratio = \frac{總Put\ OI}{總Call\ OI} \times 100\%$$

| P/C Ratio | 意義 |
|-----------|------|
| > 100% | Sell Put > Sell Call，偏多 |
| < 100% | Sell Call > Sell Put，偏空 |
| > 115% | 多頭趨勢明確 |
| < 75% | 空頭趨勢明確 |

P/C Ratio與大盤指數呈**正向關係**：P/C Ratio持續上升，大盤上漲機率高。

### 三大法人選擇權籌碼

**外資**：選擇權市場最重要的方向性指標
- 外資大買Call → 看多
- 外資大買Put → 看空
- 外資賣Call + 買Put → 強烈看空

**投信**：選擇權部位小，參考性有限

**自營商**：造市者（Market Maker），部位是被動產生的
- 自營商賣Call → 不是主動看空，是造市需求
- 關鍵是看自營商如何避險：賣Call後買期貨多單避險
- 透過自營商期貨部位反推外資選擇權動向

## 實戰應用

### 1. 選擇權支撐壓力判斷

找出Call和Put最大OI的履約價：
1. Call最大OI的履約價 = 上檔壓力區
2. Put最大OI的履約價 = 下檔支撐區
3. 觀察鄰近履約價的OI分布和增減變化
4. OI明顯增加的區域，支撐壓力效果更顯著

**倒莊效應**：當價格突破壓力區，賣方被迫回補，產生加速上漲效應。

### 2. OI + 成交量 + P/C Ratio 綜合判讀

| 指數 | OI | 成交量 | 買方 | 賣方 | 判斷 |
|------|----|--------|------|------|------|
| 上漲 | 增加 | 增加 | 獲利加碼 | 虧損攤平 | 強勢軋空 |
| 上漲 | 增加 | 減少 | 獲利加碼 | 虧損攤平 | 多頭持續 |
| 上漲 | 減少 | 增加 | 獲利了結 | 認賠出場 | 多頭反轉 |
| 下跌 | 增加 | 增加 | 虧損攤平 | 獲利加碼 | 強勢走空 |
| 下跌 | 減少 | 增加 | 認賠出場 | 獲利了結 | 空頭反轉 |

### 3. 外資選擇權 + 期貨交叉驗證

外資在選擇權和期貨的部位應一起看：
- 外資期貨多單 + 選擇權買Call → 一致看多，訊號強
- 外資期貨多單 + 選擇權買Put → 避險配置，可能偏多但有保護
- 外資期貨空單 + 選擇權賣Call → 一致看空，訊號強

### 4. VIX（恐慌指數）搭配

- 多頭行情：VIX緩步下降，賣方有利
- 空頭行情：VIX急速飆高
- 大事件前：VIX上升（選舉、央行會議等）
- 賣方策略在VIX下降時勝率最高

## 注意事項

1. **OI是莊家看法，不是保證**：壓力區可以被突破（倒莊），支撐區可以被跌破
2. **自營商不是方向性指標**：自營商是造市者，其選擇權部位是被動產生的，不要直接解讀為看多看空
3. **P/C Ratio極端值才有效**：常態範圍內（80-100%）參考性有限，要看到極端值（>115%或<75%）才具方向意義
4. **結算日效應**：接近結算日，OI會快速減少，支撐壓力效果減弱
5. **週選 vs 月選**：週選OI變化快，反映短期看法；月選OI較穩定，反映中期看法
6. **OI變化量比絕對值重要**：觀察OI是增加還是減少，比看OI絕對數字更有意義

### 5. 法人避險方式判讀（進階）

法人選擇權部位不能只看表面數字，必須看**避險方式**才能判斷真正方向：

**自營商避險方式判斷表**：

| 自營商選擇權部位 | 避險方式 | 解讀 |
|---|---|---|
| 賣Put + 買期貨空單對沖 | 中性造市 | 只是造市賺手續費，無方向性 |
| 賣Put + 增加期貨多單 | 偏多 | 避險方向跟選擇權部位同向 |
| 賣Call + 買期貨多單 | 偏多 | 不尋常，可能看好後市 |
| 賣Call + 賣期貨空單對沖 | 中性造市 | 標準造市避險 |
| 賣Call + 增加期貨空單 | 偏空 | 避險方向跟選擇權部位同向 |

**從自營商避險反推外資動向**：
- 自營商是「被動造市」，有人要買就必須賣
- 外資大買1億Call → 自營商被迫賣1億Call
- 此時看自營商的避險方式：
  - 如果自營商買期貨多單避險 → 正常中性造市
  - 如果自營商反而加空期貨 → 可能是另一個偏空訊號

**關鍵原則**：自營商的部位本身不重要，**避險方式才代表對後市的看法**。

### 6. 期交所資料進階解讀

- **負數代表賣方部位**：如 PUT未平倉口數 -10000 = 淨賣Put 10000口
- **未平倉金額正負**：金額為正代表買方金額>賣方金額
- **口數 vs 金額不一致的隱藏訊號**：
  - 每日交易為賣、留倉金額上升 → 可能大力買價平合約
  - 每天都買、留倉部位上升但金額沒增加 → 可能賣價平合約
- **散戶部位推算**：全市場 - 外資 - 自營商 = 散戶
- 散戶通常做逆勢單，與外資對做

### 7. 週選 vs 月選的判讀差異

| 比較項目 | 週選（雙週選） | 月選 |
|----------|---------------|------|
| 關鍵均線 | 5日均線 | 20日均線 |
| 時間框架 | 短線（1-2週） | 中期（1個月） |
| OI變化速度 | 快，每日大幅變動 | 較穩定 |
| 適用策略 | 短線賣方價差 | 波段方向判斷 |
| 結算頻率 | 每週三 | 當月第3週三 |
| Max Pain參考 | 短期收斂 | 中期趨勢 |
