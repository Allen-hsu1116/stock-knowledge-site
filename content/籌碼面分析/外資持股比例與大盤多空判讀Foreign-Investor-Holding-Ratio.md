# 外資持股比例與大盤多空判讀

> 外資是台股最大買家群體，持股比例變化是判斷個股和整體市場多空方向的核心籌碼指標——持股水位比絕對值重要，變化速度比水位更重要

## 核心概念

外資持股比例指外資（外國券商、投資信託、政府基金等境外機構）持有某檔股票佔流通在外股數的百分比。外資佔台股每日成交量約25-35%，是市場最大的單一買家群體，其持股變化對股價和大盤走勢有決定性影響。

核心假設：大型外資擁有全球視角和專業研究團隊，其買賣行為反映對標的基本面的深度判斷。外資持續買超會推升股價、持續賣超會壓抑股價，形成自我強化的趨勢。

### 四大持股水位

- **低水位區（<10%）**：外資不感興趣，多為中小型股或傳產股。若突然從低檔快速提升，可能是題材被發掘
- **中水位區（10%-30%）**：外資開始認同公司價值，持股比例從低檔上升至此區間通常為股價起漲初期
- **高水位區（30%-50%）**：外資高度認同，多為大型權值股。但高位持股意味著潛在賣壓風險——基本面轉差時外資退出會造成巨大賣壓
- **極高位區（>50%）**：如台積電外資持股長期70%以上。極高位代表買盤力道已充分消化，後續買超空間有限，一旦轉向賣超股價可能急跌

### 持股變化速度的四大模式

變化速度比絕對水位更重要：

- **快速上升（月增幅>5%）**：外資強烈看好，通常伴隨股價大漲。但需排除假外資
- **緩步上升（月增幅1%-3%）**：長線資金持續進場，最健康的買超型態，股價走勢穩定向上
- **快速下降（月減幅>5%）**：外資看空大舉撤出，通常有重大利空或基本面惡化
- **緩步下降（月減幅1%-3%）**：外資溫和調節，可能是獲利了結或基本面微調，不一定代表趨勢反轉

## 實戰應用

### 三種持股比例判讀模式

**模式一：持股比例持續上升 → 多頭訊號**
- 外資連續5個交易日以上買超，持股比例持續上升
- 長線資金進場，股價通常有支撐，回檔幅度有限
- 需排除MSCI季度調整的被動買超和假外資

**模式二：持股比例持續下降 → 空頭訊號**
- 外資連續10個交易日以上賣超，持股比例持續下降
- 需區分「基本面賣超」vs「避險賣超」——避險賣超可能是暫時的
- 外資持股高位（>40%）+開始連續賣超 = 最危險組合

**模式三：持股比例橫盤 → 觀望訊號**
- 外資買賣超金額不大，持股比例變動有限
- 外資對後市沒有明確方向，市場可能進入整理期
- 此時應關注投信、自營商等其他法人動向

### 選股策略一：外資持股持續上升股
1. 篩選：外資連續買超5日以上，持股比例月增幅>2%
2. 搭配：本業獲利穩定（營業利益率>10%）、股價在月線之上
3. 進場：持股比例從低檔開始加速上升時
4. 停損：跌破外資成本線（外資買超均價）

### 選股策略二：外資持股低檔反轉股
1. 篩選：外資持股比例從歷史低檔開始回升
2. 搭配：營收年增率由負轉正、股價帶量突破月線
3. 進場：外資持股比例連續3日回升且股價收紅
4. 停損：跌破外資開始買進時的價格

### 外資持股比例與大盤多空

- 外資買超期間，大盤上漲機率約70%
- 外資賣超期間，大盤下跌機率約65%
- 外資年度買超>5000億時，當年度大盤報酬率平均>15%
- 關鍵原則：持股比例趨勢比絕對值重要，變化速度判斷力道，搭配期貨部位驗證一致性

### 背離訊號

- **外資買超但大盤不漲**：可能外資只買特定權值股，非全面看多
- **外資賣超但大盤不跌**：可能有投信、主力等其他買盤承接
- 背離出現時需搭配[[籌碼面分析/三大法人籌碼綜合判讀框架Institutional-Investors-Integrated-Framework|三大法人籌碼綜合判讀框架]]交叉驗證

## 注意事項

- **排除MSCI被動買超**：MSCI季度調整會產生被動買賣超，不代表外資主動看好或看壞。搭配[[籌碼面分析/MSCI季度調整效應|MSCI季度調整效應]]判斷
- **排除假外資**：本土資金繞道海外再回買，持股比例上升但非真外資。搭配[[籌碼面分析/假外資辨識三指標與券商查詢法Fake-Foreign-Investor-Identification|假外資辨識三指標]]過濾
- **持股比例高位≠好**：外資持股>50%的個股，後續買超空間有限，賣超風險更大
- **不要只看單日數據**：單日買賣超波動大，應看5日、20日移動平均趨勢
- **搭配期貨部位驗證**：現貨買超+期貨多單增加=一致看多；現貨買超+期貨空單增加=可能避險。參考[[籌碼面分析/外資期現貨背離判讀實戰|外資期現貨背離判讀]]和[[籌碼面分析/外資現貨期貨部位綜合判讀|外資現貨期貨部位綜合判讀]]
- **大外資vs小外資行為不同**：大外資看長線方向、小外資看短線靈敏度。參考[[籌碼面分析/大外資與小外資判讀|大外資與小外資判讀]]
