# 法人三市場籌碼交叉判讀

> 只看現貨買賣超已經不夠——法人可以在現貨買超的同時在期貨放空避險，三個市場數據必須交叉比對才能看出法人真實意圖

## 核心概念

麥克連在《追籌碼賺1億②》中指出：全球熱錢氾濫後，外資在台灣期貨市場大舉布局，現貨成交量萎縮，外資期貨未平倉口數屢創新高。**現在解讀籌碼變化，期貨市場的重要性已高過現貨市場**。

單一市場的數據無法呈現全貌：
- 外資現貨買超 → 可能只是為了避險賣權的Delta對沖
- 外資期貨空單 → 可能是保護現貨多頭的避險部位
- 選擇權賣方 → 很多法人收權利金而非表達方向看法

三個市場的數據必須交叉比對，才能拆解法人真正的多空意圖。

## 三市場觀察重點

### 現貨市場：看方向
- **外資現貨買賣超**：外資買超偏多、賣超偏空
- **投信連續買超**：中期看好的訊號
- **自營商買超**：短期偏多，但要扣除權證避險買超
- **限制**：現貨數據只看得到方向，看不到避險部位

### 期貨市場：確認力道
- **外資台指期淨部位**：確認外資是否用槓桿放大看法
- **外資期貨空單超過3萬口**：歷史級警訊
- **期貨OI變化**：
  - OI增加 + 價格上漲 → 多方進場，趨勢續航
  - OI減少 + 價格上漲 → 空方回補，趨勢可能結束
- **前五大/前十大交易人部位**：大額交易人的多空比是更精準的風向球

### 選擇權市場：看防線與壓力
- **未平倉量以賣方角度解讀**：
  - Call未平倉量增加 → 看不漲、上檔有壓力
  - Put未平倉量增加 → 看不跌、下檔有支撐
- **Put Call Ratio（PCR）**：
  - PCR > 1 → 賣Put比賣Call多，莊家偏樂觀
  - PCR < 1 → 賣Call比賣Put多，莊家偏悲觀
- **未平倉大量區**：
  - Call最大量履約價 → 上檔壓力區
  - Put最大量履約價 → 下檔支撐區
- **三大法人選擇權部位**：外資Call淨空單+Put淨多單看空，反之看多

## 實戰應用

### 五步交叉判讀框架

1. **現貨看方向**：三大法人現貨買賣超，初步判斷法人多空態度
2. **期貨確認力道**：外資台指期淨部位，確認法人是否用槓桿放大看法
3. **選擇權看防線**：未平倉大量區找支撐壓力，PCR判斷整體多空氛圍
4. **三市場共振確認**：三個市場訊號是否一致，找出矛盾點
5. **結算日效應加權**：結算日前後籌碼變化最具參考價值

### 三市場共振判讀表

| 現貨 | 期貨 | 選擇權 | 判讀 |
|------|------|--------|------|
| 外資買超 | 期貨淨多單 | Sell Put偏多 | **強烈看多** |
| 外資買超 | 期貨淨多單 | Sell Call偏空 | 溫和看多（收權利金） |
| 外資買超 | 期貨淨空單 | Sell Call偏空 | **假看多，避險為主** |
| 外資賣超 | 期貨淨空單 | Sell Call偏空 | **強烈看空** |
| 外資賣超 | 期貨淨空單 | Sell Put偏多 | 溫和看空（收權利金） |
| 外資賣超 | 期貨淨多單 | Sell Put偏多 | **假看空，避險平倉** |

### 結算日操作
- 結算日前3-5天：法人調整期貨和選擇權部位，觀察調整方向
- 結算日當天：拉結算或壓結算力道最明顯
- 月選擇權結算日（每月第三個週三）前後籌碼最具參考價值
- 結算日後新倉位建立：觀察法人是否延續原來方向

## 注意事項

- **避險部位不等於看空**：外資期貨空單可能是保護現貨多頭，必須看淨部位
- **選擇權賣方不等於方向看法**：收權利金和看方向是兩件事
- **三大法人數據有盲點**：外資可透過複委託、借券繞過公開揭露
- **數據有時間差**：三大法人數據盤後公布，盤中籌碼已變動
- **大外資和小外資不同**：五大特法vs小外資，看法可能分歧
- **只看單一市場容易被騙**：現貨買超+期貨空單是假看多的經典騙局
