# 籌碼面選股實戰SOP

> 從零散指標到可執行框架——七步確認主力進場、籌碼七指標共振提高勝率

## 核心概念

籌碼面選股的核心不是看單一指標，而是回答三個問題：

1. **主力進場了嗎？** → 看分點、法人、融資券
2. **主力還在嗎？** → 看持股集中度、庫存量變化
3. **主力何時出場？** → 看出貨訊號、量價變化

單一指標容易被騙，多指標共振才是籌碼面選股的精髓。三大法人買超可能是避險、融資增加可能是散戶追價、千張大戶減少可能是出貨——只有交叉驗證才能過濾噪音。

## 實戰應用

### 七步選股SOP

**Step 1：三大法人方向確認**
- 外資連續買超5日以上偏多，連續賣超5日以上偏空
- 投信連續買超3日以上開始關注，10日以上強烈偏多
- 自營商避險買超要扣掉，自行買超才有意義
- 三大法人全買 = 最強共振訊號

⚠️ 三大法人訊號可能因換倉、避險、結算而失真，需交叉判讀期貨和選擇權部位

**Step 2：主力分點追蹤**
- 買超集中度 > 50%：主力明確
- 地緣券商大量買超：可能是內部人進場
- 神秘券商連續出現：主力分單
- 隔日沖券商（凱基、元大、國票、富國）：短線出貨風險高

**Step 3：融資券動態判讀**

八種組合判讀表：

| 融資 | 融券 | 價格 | 判讀 |
|------|------|------|------|
| 增 | 增 | 漲 | 主力與散戶同步看多，偏多 |
| 增 | 減 | 漲 | 多方強勢，空方認輸回補 |
| 減 | 增 | 漲 | 主力吸貨散戶看空，**最偏多** |
| 減 | 減 | 漲 | 縮量上漲，籌碼穩定 |
| 增 | 增 | 跌 | 多空對決激烈，方向未明 |
| 增 | 減 | 跌 | 散戶多單停損，偏空 |
| 減 | 增 | 跌 | 主力出貨散戶看空，**最偏空** |
| 減 | 減 | 跌 | 縮量下跌，賣壓漸歇 |

**Step 4：股權分散度確認**
- 千張大戶持股比例增加 = 籌碼集中，偏多
- 千張大戶持股比例下降 + 股東人數上升 = 主力出貨，避開
- 千張大戶比例上升 + 股東人數下降 = 最佳籌碼組合

**Step 5：借券與外資期貨交叉驗證**
- 現貨買超 + 期貨多單 = 真正看多
- 現貨買超 + 期貨空單 = 避險或套利，不是真看多
- 現貨賣超 + 借券增加 = 雙重看空

**Step 6：籌碼七指標共振確認**

| # | 指標 | 偏多條件 |
|---|------|----------|
| 1 | 三大法人連續買超 | 外資+投信連續3日以上 |
| 2 | 主力買超 | 前5大分點買超占總買超>50% |
| 3 | 融資使用率下降 | 散戶退場主力進場 |
| 4 | 券資比上升 | 融券增加有軋空潛力 |
| 5 | 千張大戶持股增加 | 大戶加碼 |
| 6 | 集保庫存量減少 | 籌碼集中 |
| 7 | 內部人未申報轉讓 | 公司派沒有賣股 |

共振標準：
- **3項以下**：訊號弱，不進場
- **4-5項**：訊號中等，可小量進場
- **6-7項**：訊號強烈，可正常部位進場

**Step 7：進場時機與停損設定**

進場條件（全部滿足才進場）：
1. 籌碼七指標≥4項共振
2. 技術面有突破或支撐確認
3. 量價配合（帶量突破或量縮回測）
4. 大盤不在系統性風險區（VIX<25、外資未大量賣超）

四重停損：
- **籌碼停損**：主力買超連續3日轉為賣超
- **技術停損**：跌破進場支撐或關鍵均線
- **比例停損**：虧損達7-10%出場
- **時間停損**：進場後5日未如預期走勢出場

### 三種實戰組合

**多頭籌碼組合**：外資連買 + 投信連買 + 主力集中度>50% + 融資減少 + 千張大戶增加 + 借券低 + PCR偏多

**空頭籌碼組合**：外資連賣 + 投信連賣 + 主力賣超集中 + 融資增加 + 千張大戶減少 + 借券增加 + PCR偏空

**軋空行情組合**：券資比>50% + 融券持續增加 + 接近停止過戶日 + 主力鎖碼 + 量縮不跌

## 注意事項

### 五大常見錯誤

1. **單一指標判斷**：只看三大法人買超就進場，忽略避險部位和換倉影響。修正：至少三個以上指標交叉驗證
2. **忽略量價配合**：籌碼面偏多但量價背離，主力可能在出貨。修正：籌碼+量價+技術三面合一
3. **追高隔日沖**：看到隔日沖券商買超就追，隔天被倒貨。修正：4%分水嶺原則，辨識隔日沖券商
4. **融資增加就偏多**：高檔融資增加是散戶追價訊號，反而偏空。修正：看股價位置判斷融資意義
5. **忽略大盤環境**：個股籌碼再好，大盤崩跌也會跟跌。修正：先判斷大盤多空，再選個股

### 每日必查清單

1. 三大法人買賣超（現貨+期貨+選擇權）
2. 主力分點買賣超（前5大）
3. 融資券餘額變化
4. 集保股權分散表（每週）
5. 借券賣出餘額
6. 選擇權PCR

### 快速篩選條件

- 三大法人連續買超 ≥ 3日
- 主力買超集中度 ≥ 50%
- 融資使用率 < 30%
- 券資比 > 20%（有軋空潛力）
- 千張大戶持股比例上升
