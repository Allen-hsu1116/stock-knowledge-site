# 流動比率與速動比率實戰判讀

> 流動比率看還債能力，速動比率排除存貨驗證品質，兩者差距越大存貨風險越高

## 核心概念

**流動比率**（Current Ratio）和**速動比率**（Quick Ratio / Acid-Test Ratio）是衡量企業短期償債能力的雙保險指標。

**流動比率** = 流動資產 / 流動負債（一般標準 > 2）

**速動比率** =（流動資產 − 存貨）/ 流動負債（一般標準 > 1）

核心差異：**存貨是否算進去**。存貨變現需要時間且可能貶值。速動比率假設最壞情況——存貨全部賣不掉，企業還能不能還債。

### 流動資產包含什麼

流動資產 = 現金 + 約當現金 + 短期投資 + 應收帳款 + 應收票據 + 存貨 + 預付費用

速動資產 = 流動資產 − 存貨 − 預付費用（變現能力最差的兩項）

## 實戰應用

### 五層判讀法

**第一層：絕對值判讀**

| 指標 | 安全 | 注意 | 危險 |
|------|------|------|------|
| 流動比率 | > 2 | 1.5-2 | < 1.5 |
| 速動比率 | > 1 | 0.5-1 | < 0.5 |
| 現金比率 | > 0.5 | 0.2-0.5 | < 0.2 |

流動比率 < 1 代表流動資產不足以覆蓋流動負債，短期有違約風險。

**第二層：趨勢判讀**
- 流動比率持續下降：現金被吃掉、短期借款增加、或存貨暴增
- 流動比率持續上升：現金堆積、應收帳款增加、或存貨增加
- 重點是變動的原因，不只看數字方向

**第三層：產業特性修正**

不同產業的合理流動比率差異極大：

- **科技/軟體業**：3-5很常見（現金多、存貨少）
- **製造業**：1.5-2.5合理（存貨和應收帳款多）
- **營建業**：< 1很常見（預收款計入流動負債）
- **金融業**：不適用（本質是高槓桿）
- **零售業**：1-1.5即可（存貨周轉快）

**第四層：流動比率與速動比率的差距**

差距越大 = 存貨占比越高：
- 差距小（< 0.5）：存貨少，變現壓力小，通常是服務業、軟體業
- 差距大（> 1）：存貨多，要警惕存貨跌價風險，通常是製造業、零售業
- **流動比率好但速動比率差**：問題出在存貨，要交叉看 [[存貨週轉率進階判讀與存貨陷阱識別]]

**第五層：交叉驗證三大指標**

- **流動比率 + 應收帳款周轉率**：流動資產好看但應收帳款收不回來是假的。參考 [[應收帳款周轉率與應收帳款周轉天數]]
- **流動比率 + 存貨周轉率**：存貨周轉快，流動比率可以更低。參考 [[存貨周轉率與存貨周轉天數]]
- **現金比率** =（現金 + 約當現金）/ 流動負債，是最嚴格的償債測試

### 實戰步驟

1. 看流動比率絕對值：是否在合理範圍
2. 看速動比率：排除存貨後是否仍安全
3. 看趨勢：連續8季是否惡化
4. 拆解流動資產結構：現金多少、應收多少、存貨多少
5. 對比同業：產業特性決定合理範圍
6. 計算現金比率：最保守的償債測試

### 台股實戰案例

**台積電**：流動比率2.5-3.5，速動比率2-3，現金極充裕。高科技業典型特徵，不代表資金運用效率差。

**鴻海**：流動比率約1.1-1.3，速動比率約0.5-0.7。表面看似危險，但鴻海存貨周轉極快，短期償債無虞。這就是為什麼要看 [[存貨周轉率與存貨周轉天數]] 交叉驗證。

**營建業**：流動比率常 < 1，是因為預收款入流動負債但對應的在建工程變現時間長。不代表要倒閉，是行業特性。參考 [[營運效率指標與CCC實戰判讀]]。

## 注意事項

1. **流動比率高但品質差**：流動資產都是賣不掉的存貨和收不回的應收帳款，數字好看但變現能力差
2. **窗飾效應**：結算日前臨時還借款降低流動負債，美化流動比率。參考 [[財報窗飾辨識Window-Dressing-Detection]]
3. **預收款扭曲**：營建業預收款讓流動比率看起來很差，但實際沒問題
4. **季節性波動**：零售業旺季前進貨讓流動比率驟降
5. **應收帳款呆帳**：應收帳款看起來是資產但其實收不回來
6. **存貨跌價**：存貨可能價值遠低於帳面數字，流動比率虛高。參考 [[存貨計價方法與財報影響Inventory-Valuation-Methods]]
7. **流動比率過高也不好**：> 5 可能代表現金閒置、投資不足、經營保守

### 速動比率的盲區

速動比率排除了存貨，但**沒有排除應收帳款**。如果應收帳款周轉天數很長，速動比率也會虛高。真正的「即時償債能力」要看現金比率。

### 營運資金陷阱

營運資金 = 流動資產 − 流動負債。營運資金為正是基本要求，但數字大不代表好——如果營運資金都卡在存貨和應收帳款，品質其實很差。參考 [[營運資金週轉率判讀Working-Capital-Turnover]]。
