# 固定資產週轉率判讀Fixed-Asset-Turnover

> 固定資產每1元能創造多少營收——資本密集產業的效率溫度計，太高可能投資不足，太低可能資產閒置

## 核心概念

### 固定資產週轉率（Fixed Asset Turnover Ratio, FAT）

固定資產週轉率 = 營業收入淨額 ÷ 固定資產平均淨值

固定資產平均淨值 = (期初淨值 + 期末淨值) ÷ 2

衡量公司利用現有固定資產（廠房、機器設備、土地等）創造營收的效率。

### 與資產週轉率的差異

| 指標 | 分母 | 特色 |
|------|------|------|
| 資產週轉率 | 總資產（流動+固定） | 包含現金、存貨、應收帳款等，流動資產波動大 |
| 固定資產週轉率 | 固定資產淨值 | 只看PP&E，排除流動資產的干擾 |

對於營收成長依賴固定資產擴張的產業（製造業、半導體、能源），固定資產週轉率比資產週轉率更精確。

### 計算案例

某製造公司：
- 年營業收入：2億美元
- 年初固定資產：5000萬美元
- 年末固定資產：6000萬美元

平均固定資產 = (5000+6000)/2 = 5500萬
固定資產週轉率 = 2億/5500萬 = 3.64

→ 每投入1美元固定資產，創造3.64美元營業收入

## 實戰應用

### 產業特性與基準值

| 產業類型 | 固定資產週轉率 | 特徵 |
|---------|--------------|------|
| 軟體/服務業 | 極高(10+) | 輕資產，幾乎不需要固定資產 |
| 零售業 | 高(3-8) | 店面固定資產少，營收周轉快 |
| 製造業 | 中等(1-3) | 需要廠房設備，但營收規模支撐 |
| 半導體 | 低(0.5-1.5) | 重資產投資，靠高毛利補償 |
| 能源/公用事業 | 極低(0.3-0.8) | 巨額基礎設施投資 |

**關鍵原則**：同行業比較才有意義，跨產業週轉率差異極大。

### 固定資產週轉率與成長階段

| 階段 | 週轉率表現 | 原因 |
|------|----------|------|
| 擴張投資期 | 下降 | 大量新設備投入，營收尚未跟上 |
| 產能爬坡期 | 上升 | 新產能開始貢獻營收 |
| 成熟穩定期 | 穩定 | 營收與資產同步成長 |
| 衰退期 | 可能上升 | 固定資產減損讓分母變小，不是效率提升 |

### 週轉率異常判讀

1. **週轉率突然上升**：
   - 可能是固定資產減損（分母變小），不是效率提升
   - 可能是出售大量資產
   - 可能是營收突然爆發

2. **週轉率持續下降**：
   - 可能是過度投資，營收跟不上資產擴張
   - 可能是設備老化效率低
   - 可能是擴廠產能尚未開出

3. **週轉率過高**：
   - 可能代表投資不足，機台廠房不夠應付訂單
   - 長期投資跟不上會讓競爭者趁虛而入

4. **週轉率過低**：
   - 固定資產閒置，效率差
   - 可能處於重資產投資期（不一定壞事）

### 搭配其他指標綜合判讀

| 搭配指標 | 判讀重點 |
|---------|---------|
| ROA | 固定資產週轉率高 + ROA高 = 資產效率佳且獲利佳 |
| 營業利益率 | 週轉率高 + 利益率低 = 薄利多銷型 |
| 固定資產/總資產比 | 比率高代表重資產公司，FAT更有參考價值 |
| 折舊/固定資產比 | 比率高代表資產老化，可能需要大量資本支出更新 |

### 不適用固定資產週轉率的產業

- **軟體業**：營收不靠固定資產驅動（如PayPal）
- **金融業**：固定資產佔比極低，週轉率失真
- **服務業**：人力資本為主，固定資產非獲利關鍵

這些產業應改用資產週轉率、ROA或ROE評估。

## 注意事項

1. **折舊方法影響**：分母用固定資產「淨值」，折舊方法（直線法vs加速折舊）會影響分母大小，同業比較時注意折舊政策一致性
2. **併購與資產處分**：大量購買資產會壓低週轉率，出售資產會提高週轉率，需排除非常規因素
3. **只看固定資產**：不反映流動資產管理效率，需搭配其他週轉率指標
4. **產業差異極大**：跨產業比較會誤判，必須同業比較
5. **成長期週轉率下降正常**：大量資本支出期間週轉率下降是投資期的自然現象，不代表效率變差
