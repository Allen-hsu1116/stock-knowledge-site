# 三率三費分析與營業利益率判讀

**來源**：AI整理（基於訓練數據中的台股費用率分析知識）
**日期**：2026-05-19
**主題**：基本面分析

---

## 為什麼費用率比毛利率更重要？

毛利率告訴你「產品好不好賺」，費用率告訴你「公司會不會花錢」。一家毛利率50%但三費佔40%的公司，營業利益率只有10%；另一家毛利率30%但三費只佔10%的公司，營業利益率也有20%。**最終決定公司獲利能力的是營業利益率，不是毛利率。**

---

## 三費是什麼？

### 銷售費用（銷費）
- 員工酬勞（業務人員薪資獎金）
- 廣告促銷費
- 運費與包裝費
- 佣金支出
- **重點觀察**：銷售費用率越高代表公司越依賴銷售推動成長

### 管理費用（管費）
- 行政人員薪資
- 辦公室租金
- 折舊攤銷（管理用）
- 法務與會計費用
- **重點觀察**：管理費用率應相對穩定，暴增代表管理效率下降

### 研發費用（研費）
- 研發人員薪資
- 實驗室設備
- 專利權費用
- 試驗材料費
- **重點觀察**：研發費用是「投資」而非「費用」，高科技公司研發費用率越高代表未來競爭力越強

## 三費用率的計算與判讀

### 三費用率 = 三費合計 ÷ 營收 × 100%
- 毛利率 - 三費用率 = 營業利益率
- 三費用率越低，營業利益率越高
- 不同產業的三費用率差異巨大

### 各產業三費用率參考範圍

| 產業 | 研發費用率 | 銷管費用率 | 三費用率合計 |
|------|-----------|-----------|-------------|
| 半導體設計 | 15-30% | 5-10% | 20-40% |
| 電子代工 | 1-3% | 3-5% | 4-8% |
| 軟體服務 | 15-25% | 15-25% | 30-50% |
| 零售通路 | 0-1% | 15-25% | 15-26% |
| 金融保險 | 0% | 20-40% | 20-40% |
| 傳產製造 | 1-3% | 5-10% | 6-13% |

### 判讀三原則

1. **同業比較才有意義**：電子代工的三費用率6%跟軟體公司的30%不能直接比
2. **趨勢比絕對值重要**：三費用率逐年上升代表經營效率惡化
3. **看結構不看總數**：研發費用率上升可能是好事，管理費用率上升通常是壞事

## 營業利益率：公司獲利能力的核心指標

### 營業利益率 = 營業利益 ÷ 營收 × 100%

營業利益率反映了公司「本業」的賺錢能力，排除了業外收支的干擾，是衡量公司經營效率最純粹的指標。

### 營業利益率四大判讀法

#### 1. 絕對值判讀
- >20%：極強的定價力與成本控制（如台積電、聯發科）
- 10-20%：健康穩定的獲利能力
- 5-10%：勉強及格，景氣下滑時可能虧損
- <5%：本業獲利能力薄弱，微利生存
- <0%：本業虧損，靠業外撐場面

#### 2. 趨勢判讀
- 營業利益率連續上升：規模效應發酵、產品組合優化
- 營業利益率連續下降：競爭加劃、費用失控、降價搶單
- 營業利益率穩定：經營效率穩定（大部分成熟型公司）

#### 3. 與同業比較判讀
- 同產業營業利益率最高的公司 = 產業領導者
- 營業利益率差距代表護城河寬度
- 龍頭公司營業利益率通常是同業的1.5-3倍

#### 4. 與毛利率關聯判讀
- 毛利率上升+營業利益率上升 = 真正好（產品力+管理效率雙升）
- 毛利率上升+營業利益率下降 = 警訊（費用失控吃掉毛利）
- 毛利率下降+營業利益率上升 = 好現象（費用控制抵消了毛利壓力）
- 毛利率下降+營業利益率下降 = 惡性循環（雙降代表公司體質惡化）

## 三率三升的進階判讀

「三率三升」是基本面選股的黃金訊號：

### 三率是哪三率？
- 毛利率 ↑
- 營業利益率 ↑
- 淨利率 ↑

### 三率齊升代表什麼？
- 毛利率上升：產品定價力增強或成本下降
- 營業利益率上升：經營效率提升（費用控制良好）
- 淨利率上升：整體獲利能力增強

### 三率三升的股票通常具備特徵
1. 產品ASP（平均售價）上升
2. 產品組合往高毛利產品移動
3. 營收規模擴大帶來規模經濟
4. 費用率因營收增加而自然稀釋

### 三率三升的選股應用
- 三率連續兩季以上齊升 = 進場觀察
- 三率連續四季齊升 = 高度確認的優質股
- 三率齊升但股價還沒反映 = 最佳買點

## 費用率異常的五種紅旗

### 紅旗1：管理費用率持續上升
- 可能原因：管理層膨脹、浪費、無效投資
- 判讀：管理費用率超過同業30%以上就該警覺
- 實例：管理费用率從3%連續升至6%，但營收沒有同步成長

### 紅旗2：銷售費用率遠高於同業
- 可能原因：產品吸引力不足，必須靠行銷推動
- 判讀：銷售費用上升但營收不成比例增加
- 惡性循環：越依賴行銷→毛利越薄→越需要更多行銷

### 紅旗3：研發費用率驟降
- 可能原因：裁減研發團隊、削減研發預算
- 判讀：短期利潤改善但長期競爭力受損
- 例外：研發階段完成的軟體公司，研發費用率自然下降

### 紅旗4：業外費用佔比過高
- 利息費用過高 = 舉債太多
- 處分損失頻繁 = 資產管理不善
- 匯兌損失巨大 = 外幣曝險管理不佳

### 紅旗5：三費用率與毛利率背離
- 毛利率上升但三費用率也大幅上升 = 增收不增利
- 成本結構改變可能讓公司「看起來」成長但實際獲利能力惡化

## 損益表三層利潤解析

```
營收
 └─ 減：營業成本 → 毛利
      └─ 減：銷售費用
      └─ 減：管理費用
      └─ 減：研發費用
      → 營業利益
           └─ 加/減：業外收支
           → 稅前淨利
                └─ 減：所得稅
                → 稅後淨利
```

**三層利潤率代表三層能力**：
- 毛利率 → 產品力（定價力+成本控制）
- 營業利益率 → 經營力（產品力+管理效率）
- 淨利率 → 綜合力（經營力+業外+稅務）
