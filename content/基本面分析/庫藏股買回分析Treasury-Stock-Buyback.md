# 庫藏股買回分析Treasury-Stock-Buyback

> 公司用自有資金買回自家股票，看似利多但動機才是關鍵——註銷提升EPS、轉讓員工激勵、護盤撐股價，三種目的三種解讀，執行率和買回區間才是辨真偽的照妖鏡

## 核心概念

### 什麼是庫藏股？

庫藏股（Treasury Stock）是公司用自有資金從公開市場買回已發行的股票，買回後這些股票不再對外流通。

**庫藏股特性**：
- 沒有盈餘分派權利（不配息）
- 沒有表決權（不計入投票）
- 公司解散時不能變現
- 庫藏股總價值不能超過公司資本的5%
- 買回加尚未註銷或轉讓的庫藏股，不能超過已發行股份總數10%

**基本等式**：
> 總已發行股數 = 流通在外股票 + 庫藏股

### 三種法定買回目的（台灣證交法）

1. **轉讓股份給員工**：激勵員工、留任人才，避免員工認股權稀釋股權
2. **作為特別股或公司債的轉換來源**：附認股權證公司債、可轉換公司債
3. **維護公司信用及股東權益**：買回並可銷除股份（俗稱護盤）

### 庫藏股兩種處置方式

**1. 註銷（庫藏股減資）**
- 減少股本，提升每股盈餘（EPS分母變小）
- 提高每股價值，向市場傳遞「股價被低估」的信號
- 本質等同於現金減資

**2. 重新發行/轉讓**
- 不稀釋現有股權（因為不是新股）
- 常見於員工認股權行使、可轉債轉換

## 實戰應用

### 買回區間價格怎麼看？

買回區間價格 = 董事會決議前10個或30個營業日平均收盤價（取高者）的70%～150%

**解讀**：
- 下限 = 決議日收盤價 × 70%
- 上限 = 10日或30日均價 × 150%（取高）
- 如果市價遠低於下限，公司根本不想買（口水護盤）
- 如果市價在上限附近，買回意願可能較高

### 庫藏股對股價的影響：回測實證

市場先生用2009-2012年台股資料回測：

| 情境 | 1個月報酬 | 結論 |
|------|----------|------|
| 執行庫藏股期間買進 | 平均不賺錢，勝率低於50% | 不能當穩定獲利方法 |
| 股價上漲 + 轉讓員工 | 短期偏多 | 管理階層看好公司 |
| 股價大跌 + 維護股東權益 | 1個月後漲幅較高 | 公司認為股價過低 |
| 執行庫藏股後仍跌5%以上 | 未來2年上漲機率低於50% | 急救無效，後續也很難好 |

**關鍵結論**：
1. 開始執行庫藏股後，短期1個月漲幅不明顯
2. 股價上漲時，「轉讓員工」類型效果較好
3. 股價大跌時，「維護股東權益」類型效果較好
4. **執行庫藏股後不漲 = 急救無效**，後續股價也很難有起色

### 大股東4大真實動機

表面上說「維護股東權益」，實際上可能：

1. **硬撐股價**：防止大股東質押股票斷頭
2. **一買一賣出貨**：用公司錢提供流動性，幫特定人士出貨
3. **提升大股東股權**：防止併購、拉高股價
4. **節稅替代配息**：庫藏股減資代替現金股利，大股東可節稅

### 庫藏股對財務報表的影響

**資產負債表**：
- 資產端：現金減少（拿去買股票）
- 股東權益：庫藏股為減項

**EPS影響**：
- 買回後不論是否註銷，流通在外股數減少
- EPS分母變小 → EPS上升
- 但如果本業沒成長，EPS上升只是數字遊戲

### 正面案例 vs 負面案例

| | 正面（可口可樂） | 負面（宏達電） |
|---|---|---|
| 背景 | 本業持續成長 | 本業持續衰退 |
| 動機 | 看好公司長期價值 | 護盤撐股價 |
| 執行率 | 高 | 低（12次宣告，多次執行率差） |
| 結果 | 營收+庫藏股雙引擎推升股價 | 護盤只是短期效果 |

## 注意事項

### 庫藏股5大陷阱

1. **口水護盤**：宣告買回但執行率極低，只是做樣子
2. **美化EPS**：本業沒成長，靠減少股數讓EPS好看
3. **質押護盤**：大股東質押比率過高，買庫藏股只是為了自救
4. **時機不對**：在高點買回，浪費公司現金
5. **資訊疲乏**：多次宣告庫藏股，市場反應遞減

### 辨識真偽庫藏股的3個問題

1. **執行率多少？** 低於50%要懷疑
2. **買回目的？** 「維護股東權益」 vs 「轉讓員工」意義不同
3. **公司本業是否健康？** 本業衰退的庫藏股只是護盤

### 庫藏股 vs 現金減資 vs 現金股利

| 比較 | 庫藏股註銷 | 現金減資 | 現金股利 |
|------|-----------|---------|---------|
| 對股東的影響 | 每股價值提升 | 拿回現金 | 拿到現金 |
| 對EPS的影響 | EPS上升 | EPS上升 | EPS不變 |
| 稅務考量 | 免稅（股價上漲才課稅） | 免稅（退還股款） | 併入綜所稅 |
| 對現金流的影響 | 一次性流出 | 一次性流出 | 每年流出 |
