# 財報窗飾辨識Window-Dressing-Detection

> 財報窗飾是企業在會計準則邊緣「化妝」的行為，投資人應抱持「健康的存疑態度」，用周轉期間分析和現金流量分析兩大利器看穿化妝術

## 核心概念

**財報窗飾（Window Dressing）** 是企業在財務報表編製完成前，運用各種方法改善或美化財務報表經營成果及財務狀況的行為。

### 窗飾 vs 舞弊 vs 盈餘管理

| | 財報窗飾 | 盈餘管理 | 財報舞弊 |
|---|---|---|---|
| 界線 | 在會計準則邊緣 | 在會計準則框架內 | 超出會計準則 |
| 性質 | 化妝美化 | 判斷空間運用 | 造假欺騙 |
| 合法性 | 灰色地帶 | 通常合法 | 違法 |
| 嚴重度 | 降低報表可信度 | 降低盈餘品質 | 完全不可信 |
| 識別難度 | 中等 | 較難（需對比分析） | 需深入調查 |

### 企業窗飾的五大誘因

1. **維持股價**：上市企業窗飾最主要目的，避免財報虧損直接反映股價
2. **爭取/維持銀行貸款**：業績不佳可能被斷絕融資
3. **節省稅負**：高獲利繳高稅，可能調降盈餘
4. **薪酬連動**：高階經理人薪酬與盈餘掛鉤
5. **避免被併購**：掩飾高額獲利，避免引來掠食者

## 實戰應用

### 兩大窗飾手法

#### 手法一：銷售額灌水

**①循環交易（Round-Trip Transaction）**
同一商品在多家公司間反覆交易，價格逐步墊高（10萬→20萬→30萬），虛增營收和利益。一旦某家公司出問題，就產生巨額呆帳。

**②提前認列營收**
- 未來仍須提供勞務卻提前認列（未賺得即先認列）
- 貨物運出前即認列（未實現即先認列）
- 完工比率法灌水：高估完工比例

**③虛構營收**
- 塞貨給關係企業或經銷商
- 集團公司間相互購買
- 虛設人頭公司充當客戶

**④三角貿易虛增營收**
同一產品認列兩次營收。例如：買進原料1億元→交由中國子公司代工→先認列1億營收→成品1.5億元賣到美國→再認列1.5億營收。會計師簽核時才會沖銷，但營收公布後股價已反映。

**⑤將非營收項目列入營收**
- 投資收益列為營收
- 借款取得的現金列為營收
- 資產處置收入列為營收

#### 手法二：費用少算

**①期末存貨灌水**
銷貨成本 = 期初庫存 + 期中添購 - 期末庫存。期末庫存灌水 → 銷貨成本降低 → 利益增加。但庫存會像滾雪球累積，資產負債表會很不自然。

**②遞延費用認列**
- 將營業成本資本化（原本應費用化的支出改為資產分攤）
- 折舊/攤銷速度放慢
- 過時存貨延遲報廢
- 設備維修延後

**③改變會計估計**
- 延長折舊年限
- 降低壞帳提列比例
- 減少退休基金提撥

### 四大辨識利器

#### 利器一：周轉期間分析（最有效）

**應收帳款周轉期間**
- 正常範圍因產業而異
- 資訊服務業：2-3個月
- 製造業：2-4個月
- **異常訊號**：周轉期間突然拉長，或遠高於同業平均

**存貨周轉期間**
- 突然異常增加 → 可能在窗飾過程中存貨大幅增加
- 應收帳款轉列存貨 → 可能是應收帳款過度膨脹後的掩飾

#### 利器二：現金流量分析（最可靠）

虛假交易不會產生真正的現金流量，且竄改銀行存款非常困難。

**關鍵判斷：**
- 營業現金流連續多期赤字，但損益表營業利益始終黑字 → **嚴重紅旗**
- 現金盈餘比 = 營業現金流 ÷ 稅後淨利 < 0.8 → 盈餘品質堪慮

**Media Links案例：**
- 損益表：營業利益穩步上升（黑字）
- 現金流量表：營業活動現金流連續三期赤字
- 應收帳款周轉期間：從5個月拉長到7個月（同業平均2-3月）
- 結果：2004年東窗事發，社長被判違反證券交易法

#### 利器三：博達案七步辨識法

2004年博達（2398）弊案手法：
1. 設立海外子公司作為「代理商」
2. 假銷貨給子公司，虛增營收與應收帳款
3. 將應收帳款賣斷給銀行並回存（受限用途存款，不能自由動用）
4. 虛增帳面現金、提高流動比率
5. 將應收帳款設計成CLN衍生性金融商品賣出取現

**財報效果：** 營收↑、利潤↑、流動資產↑、總資產↑

#### 利器四：CFRA七大騙局檢查表

1. ☐ 營收入帳時間或品質有問題？
2. ☐ 列記虛擬營收？
3. ☐ 利用一次性利得虛灌盈餘？
4. ☐ 本期費用移轉至後期/前期？
5. ☐ 未列計負債或短列負債？
6. ☐ 本期營收移轉至後期？
7. ☐ 未來費用以特別支出移轉至本期？

### 其他常見危險信號

- 透過外商銀行進行財務操作
- 經常發行海外公司債，由海外轉投資企業轉換回普通股賣出
- 以假外資真炒股拉抬自家股價
- 董監事股權質押比率 > 50%且持續增加
- 年年轉投資年年虧損又年年增資
- 關係人資金往來或背書保證過多
- 頻頻更換會計師或財務主管
- 企業重要幹部均為親戚（家族化）
- 高風險行業卻高負債比率經營

## 注意事項

### 誤區一：抱著抓賊心態看所有報表

應抱持「健康的存疑態度」，合理存疑是為了避免被扭曲資訊誤導，而非為否定而存疑。不是所有報表都有問題。

### 誤區二：只看損益表

窗飾手法同時影響資產負債表和現金流量表，必須三表一起看：
- 損益表：營收和利潤被美化
- 資產負債表：應收帳款、存貨、現金可能異常
- 現金流量表：虛假交易不會有真正的現金流

### 誤區三：忽略行業特性

不同行業的周轉期間差異很大，必須同業比較才有意義：
- 營建業存貨周轉期間本來就長
- B2B企業應收帳款周轉期間較長
- 要與同業平均比較

### 誤區四：忽略「所得平坦化」和「沖大澡」

- **所得平坦化（Income Smoothing）**：好年多提費用、壞年少提費用，讓盈餘看起來平穩
- **沖大澡（Big Bath）**：壞年一次認列所有未來費用，讓後續年度盈餘好看
- 兩者都扭曲了盈餘的真實性
