# PMI採購經理人指數實戰判讀

> PMI是製造業的健康檢查報告，50是榮枯分水嶺，趨勢比絕對值重要，新訂單指數是最具前瞻性的細項

## 核心概念

PMI（Purchasing Managers' Index）採購經理人指數，是衡量製造業和服務業景氣狀況的領先指標。透過對企業採購經理的問卷調查，匯總新訂單、生產、就業、供應商交貨時間和庫存等數據編製而成。

### PMI vs NMI

| 項目 | 製造業 PMI | 非製造業 NMI |
|------|-----------|-------------|
| 衡量對象 | 製造業景氣 | 服務業景氣 |
| 包含產業 | 電子光學、基礎原物料、電力機械、化學生技、食品紡織、交通工具 | 營造不動產、批發零售、金融保險、資訊通訊等 |
| 編制指標 | 新訂單20%+生產20%+僱用20%+交貨20%+存貨20% | 商業活動25%+新訂單25%+僱用25%+交貨25% |

### 為什麼PMI是領先指標？

企業的採購經理人身處生產第一線，對市場需求和供應鏈的變化最為敏感。製造業的原料需求、機具設備訂單與生產的週期較長，需在數個月甚至一兩年前便先做好預定。因此PMI中的「新訂單指數」尤其具有前瞻性。

### 計算方式

1. **計算擴散指數**：受訪經理人勾選「上升」、「持平」或「下降」
   - 擴散指數 = 上升%×1 + 持平%×0.5 + 下降%×0
2. **計算各產業別指數**：依各指標權重加權計算
3. **合成整體PMI/NMI**：依照各產業占名目GDP的比重權數加權平均

## 實戰應用

### 榮枯線50的判讀

- PMI > 50：景氣擴張期（Expansion）
- PMI = 50：與上月持平
- PMI < 50：景氣收縮期（Contraction）

**但50不是絕對分界線**：
- PMI跌到50以下但下個月馬上回升，不代表經濟真的衰退了
- 必須輔以PMI的趨勢變化和細項數據來判讀
- 連續3個月低於50才較有說服力

### 趨勢比絕對值重要

- PMI從55降到54 → 仍處於強勁擴張，不需恐慌
- PMI從51連續三個月降至48 → 值得警惕的下降趨勢
- 用3個月或6個月移動平均線平滑短期波動，看趨勢更清楚

### 細項比總指數更重要

| 細項 | 前瞻性 | 實戰意義 |
|------|--------|---------|
| **新訂單指數** | ⭐⭐⭐⭐⭐ | 最具前瞻性，反映未來需求 |
| **六個月展望指數** | ⭐⭐⭐⭐⭐ | 直接反映企業信心 |
| **供應商交貨時間** | ⭐⭐⭐⭐ | 下降=供貨速度提升=訂單不再擁擠=景氣轉弱 |
| **存貨指數** | ⭐⭐⭐ | 存貨上升可能是需求減弱警訊 |
| **生產指數** | ⭐⭐⭐ | 確認當期產能利用狀況 |
| **僱用指數** | ⭐⭐ | 就業是落後指標，確認趨勢用 |

### PMI與股市的關係

PMI和股市呈現正相關，但：
- PMI公布的是上個月的調查，股市可能已經提前反映
- PMI是領先指標，股市也是領先指標，兩者可能出現滯後效果
- 市場反應看的是「預期差」而非絕對數字

### ISM vs IHS Markit PMI（美國）

| 項目 | ISM PMI | IHS Markit PMI |
|------|---------|----------------|
| 權重 | 五項各20%等權重 | 新訂單30%、產出25%、聘僱20%、交貨15%、存貨10% |
| 問卷邏輯 | 主觀意見 | 客觀事實 |
| 調查對象 | 大型跨國公司 | 分散的代表性公司 |
| 產業分類 | NAICS | SIC |

### 美國PMI與台股的連動

美國是全球最大經濟體，美元是全球主要儲備貨幣。美國PMI的變化會透過貿易、資金流動影響台灣：
- 美國PMI擴張 → 台灣出口需求增加 → 科技股受惠
- 美國PMI收縮 → 外資可能從台股撤出 → 加權指數承壓
- 台灣PMI可追蹤電子暨光學產業細項，直接反映半導體景氣

## 台股實戰判讀SOP

### 步驟一：看總指數方向
- PMI > 50且趨勢向上 → 偏多操作
- PMI > 50但趨勢下滑 → 保守觀望
- PMI < 50且持續下滑 → 偏空防禦
- PMI < 50但止跌回升 → 尋找抄底機會

### 步驟二：看細項結構
- 新訂單上升+存貨下降 = 需求強勁，補庫存行情可期
- 新訂單下降+存貨上升 = 需求減弱，去庫存壓力大
- 六個月展望指數 > 60 = 企業高度樂觀

### 步驟三：看預期差
- 公布值 > 市場預期 → 利多反應
- 公布值 < 市場預期 → 利空反應
- 連續超預期或低於預期 → 趨勢確認

### 步驟四：搭配其他指標
- PMI + 景氣對策信號 → 確認景氣位置
- PMI + M1B年增率 → 確認資金動能
- PMI + 外資買賣超 → 確認法人態度

## 查詢管道

- **台灣**：國發會景氣指標查詢系統 (index.ndc.gov.tw)、中華經濟研究院
- **美國**：ISM官網、S&P Global (IHS Markit)
- **視覺化圖表**：MacroMicro財經M平方

## 注意事項

- PMI是問卷調查，反映的是採購經理人的「主觀判斷」，與實際數據可能存在落差
- 單月數據容易受季節性因素或突發事件干擾，至少觀察3個月趨勢
- PMI對股市的領先時間不是固定的，有時領先3個月有時長達12個月
- 數據公布後常會修正初值，避免基於不準確的初步數據做重大決策
- 台灣PMI要特別關注電子暨光學產業細項，因為這直接反映半導體產業景氣
