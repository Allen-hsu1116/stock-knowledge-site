# 現金流量表間接法調整項目判讀 Indirect Method Adjustments

> 從稅後淨利調整回營業現金流的每一個項目都是盈餘品質的照妖鏡——折舊加回不是現金、應收帳款暴增是紅旗、營運資金變動方向比金額更重要

## 核心概念

### 間接法的基本邏輯

台灣上市櫃公司超過 95% 採用間接法編制現金流量表。核心邏輯是從「稅後淨利」（會計利潤）出發，逐步調整回「營業活動現金流量」（實際現金進出）：

**稅後淨利** → 加回非現金支出 → 調整非營業項目 → 調整營運資金變動 → **營業活動現金流量**

為什麼需要調整？因為損益表認列的收入不等於當期收到的現金，認列的費用也不等於當期支付的現金。

### 調整項目分類

**非現金支出（加回）**：
- 折舊：帳面費用但無現金流出
- 攤銷：帳面費用但無現金流出
- 資產減損：帳面損失但無現金流出

**非現金收入（扣除）**：
- 權益法投資收益：帳面收益但可能無現金流入

**非營業項目**：
- 出售資產利益：扣除（歸屬投資活動）
- 出售資產損失：加回（歸屬投資活動）

**營運資金變動**：
- 應收帳款增加：扣除（錢沒收到）
- 應收帳款減少：加回（收到前期應收）
- 存貨增加：扣除（錢花在買存貨）
- 存貨減少：加回（賣掉前期存貨）
- 應付帳款增加：加回（錢還沒付）
- 應付帳款減少：扣除（付掉了前期應付）

## 實戰應用

### 非現金支出的深度判讀

**折舊與攤銷加回**：
- 折舊加回 ≠ 公司有等額現金可用
- 加回是為了還原「折舊已從淨利中扣除」的事實
- 折舊金額接近資本維持支出的近似值
- 資本支出 >> 折舊表示公司正在擴張
- 資本支出 ≈ 折舊表示只做維持性投資

**資產減損加回**：
- 減損不影響現金流（純帳面調整）
- 間接法必須加回還原淨利
- 投資者應看扣除減損後的正常化現金流

**權益法投資收益扣除**：
- 帳面上有收益但可能沒有收到現金
- 權益法投資收益佔淨利比 > 30% 需警惕盈餘品質

### 營運資金變動的三維判讀

**應收帳款變動判讀**：
- 良性增加：營收成長帶動應收帳款自然增加
- 惡性增加：收款天數拉長、客戶付款能力下降
- 判斷標準：應收帳款增加率 > 營收增加率 → 🚩紅旗
- DSO 上升 = 收款變慢 = 🚩紅旗

**存貨變動判讀**：
- 備貨型增加：預期銷售成長，提前備貨
- 滯銷型增加：賣不出去堆積
- 策略型增加：原物料低價時大量進貨
- 判斷標準：存貨增加率 >> 營收增加率 → 可能滯銷🚩
- 成品存貨增加 > 原物料增加 → 滯銷風險高🚩

**應付帳款變動判讀**：
- 良性增加：議價能力強，延後付款是免費融資
- 惡性增加：現金不足，付不出錢
- DPO 上升 + 現金充足 = 供應商融資（好）
- DPO 上升 + 現金壓力 = 支付困難（壞）

### 間接法異常偵測三大紅旗

**紅旗一：折舊攤銷異常偏低**
- 可能變更折舊政策延長耐用年限
- 可能採用過於樂觀的殘值估計
- 比較同行業公司折舊率判斷合理性

**紅旗二：營運資金變動方向矛盾**
- 營收成長但應收帳款減少 → 可能轉讓應收帳款
- 營收衰退但存貨增加 → 滯銷堆積🚩
- 應付帳款突然暴增 → 可能付不出錢🚩

**紅旗三：非營業項目金額過大**
- 處分資產利益/損失佔淨利 > 20% → 本業獲利能力存疑
- 權益法投資收益佔淨利 > 30% → 現金可能沒進來🚩

### 現金流品質三維檢查

| 維度 | 健康訊號 | 危險訊號 |
|------|----------|----------|
| 非現金調整 | 折舊攤銷穩定 | 減損大幅波動 |
| 營運資金 | 應收帳款與營收同步成長 | 應收帳款增速遠超營收 |
| 整體品質 | 營業現金流 > 淨利 | 營業現金流 < 淨利 |

## 注意事項

### 五個常見誤區

**誤區一：折舊加回 = 公司有現金可用**
- 錯！折舊加回只是還原帳面扣除
- 實際現金早在購買資產時已付出
- 加回折舊的意義是「這筆費用沒有實際現金流出」

**誤區二：營業現金流 > 淨利 = 盈餘品質好**
- 不一定！要看調整項目結構
- 靠應付帳款大幅增加（拖欠供應商）→ 不可持續
- 靠折舊攤銷加回 → 這是正常的

**誤區三：間接法數字和直接法一樣**
- 營業現金流總額一樣，但分類可能不同
- 間接法從淨利調整，看不到現金流源頭
- 直接法列出現金收入和支出，資訊更透明

**誤區四：營運資金變動不重要**
- 營運資金變動直接影響現金流
- 應收帳款暴增 = 收不到錢 = 🚩紅旗
- 存貨暴增 = 賣不出去 = 🚩紅旗

**誤區五：減損加回後就不用管減損**
- 減損雖不影響現金流，但影響資產品質
- 減損意味資產不值原先認列的價值
- 連續減損的公司資產負債表可能嚴重失真
