# 股價淨值比進階判讀 PBR Advanced

> PBR 不只是「比1大或比1小」，進階用法要懂 PBR×ROE 恆等式、景氣循環股歷史區間法、大盤 PBR 判多空、Tobin's Q 與有形淨值比，才能真正避開價值陷阱

## 核心概念

### PBR × ROE 恆等式

這是 PBR 進階判讀最重要的公式：

**PBR = PE × ROE**

推導：
- PBR = 股價 ÷ 每股淨值
- PE = 股價 ÷ 每股盈餘
- ROE = 每股盈餘 ÷ 每股淨值
- 因此：PBR = (股價 ÷ 每股淨值) = (股價 ÷ 每股盈餘) × (每股盈餘 ÷ 每股淨值) = PE × ROE

**這個恆等式告訴我們三件事**：

1. **高 ROE 公司 PBR 高是合理的**：ROE 15% 的公司，PBR = 2~3 倍可能還算便宜；ROE 5% 的公司，PBR = 1 倍都可能太貴
2. **PBR 不能獨立判斷**：同樣 PBR = 1.5，ROE 20% 的公司遠比 ROE 5% 的公司值得買
3. **PE 與 PBR 的選擇邏輯**：獲利穩定看 PE，獲利不穩定看 PBR，但兩者要透過 ROE 串起來

### 四象限判斷矩陣

|  | PBR 低 | PBR 高 |
|---|--------|--------|
| **ROE 高** | 🟢 真便宜（市場錯殺） | 🟡 合理溢價（成長股） |
| **ROE 低** | 🔴 價值陷阱（經營差） | 🔴 貴且爛（泡沫） |

- 🟢 真便宜：ROE > 8%、PBR < 1，市場低估，是最佳買點
- 🟡 合理溢價：ROE 高、PBR 也高，反映市場對成長性的期待，不一定是貴
- 🔴 價值陷阱：PBR < 1 但 ROE 低或為負，公司不賺錢，PBR 低只是反映經營事實
- 🔴 貴且爛：ROE 低、PBR 高，是最糟糕的組合

### Tobin's Q：市場價值 vs 重置成本

Tobin's Q 是 PBR 的學術升級版：

**Tobin's Q = 企業市場價值 ÷ 資產重置成本**

- Q > 1：市場價值高於重置成本，企業有超額利潤，新競爭者會想進入
- Q < 1：市場價值低於重置成本，企業利潤不足，長期資本會退出
- Q = 1：市場價值等於重置成本，長期均衡狀態

**與 PBR 的差異**：
- PBR 用「帳面價值」當分母，Tobin's Q 用「重置成本」當分母
- 重置成本 = 重新建造同樣企業需要花多少錢（考慮通膨、技術更新）
- 理論上比 PBR 更精準，但重置成本很難估算，實務上仍以 PBR 為主

**實戰意義**：當整個產業的 Tobin's Q 長期低於 1，代表產業過度競爭，資本報酬率低，不適合長期投資。

### 有形資產淨值比（Price-to-Tangible-Book, PTB）

**PTB = 股價 ÷ 每股有形淨值**
**有形淨值 = 總資產 – 總負債 – 特別股 – 無形資產（含商譽）**

PTB 排除了無形資產和商譽的干擾，特別適用於：
1. **商譽占比高的公司**：併購型公司商譽可能佔淨值30%以上，PBR 嚴重低估真實估值
2. **品牌價值不反映在帳面的公司**：科技業、服務業的無形資產（專利、商標、客戶關係）在會計上可能低估或未入帳
3. **金融業**：金融資產的公允價值與帳面價值差異大，PTB 更能反映真實情況

**PTB vs PBR 判斷**：
- PTB > PBR 很多 → 無形資產占比高，公司價值不在帳面上
- PTB ≈ PBR → 無形資產占比低，公司價值主要在有形資產

## 實戰應用

### 景氣循環股 PBR 歷史區間法

景氣循環股是 PBR 最適用的領域，但用法不是看 PBR < 1 就買：

**步驟一：建立歷史 PBR 區間**
- 收集至少5年的 PBR 數據
- 標記出 PBR 區間的上緣和下緣
- 區間中值作為合理價基準

**步驟二：判斷目前位置**
- PBR 在歷史區間下緣 → 景氣谷底，潛在買點
- PBR 在歷史區間中值 → 合理價位
- PBR 在歷史區間上緣 → 景氣高峰，潛在賣點

**步驟三：確認趨勢**
- PBR 區間逐年上移 → 產業成長中，可考慮買進
- PBR 區間逐年下移 → 產業衰退中，低點只會更低
- PBR 區間穩定 → 產業成熟，區間操作

**台泥(1101)案例**：
- 2010~2019年 PBR 區間：1.1倍~1.5倍
- PBR 跌到 1.1 以下 → 可能超跌
- PBR 漲到 1.5 以上 → 可能超漲
- 相關係數 0.67，股價與 PBR 中高度相關

**群創(3481)案例**：
- PBR 一波比一波低
- 2010~2011年大幅下降
- 2013年反彈至約 0.9 倍後持續下跌
- 2019年 PBR 低於 0.3 倍
- 相關係數 0.93，高度相關但趨勢向下
- **這類股票不能用 PBR 歷史區間法，因為區間持續下移**

### 大盤 PBR 判斷多空

台股大盤 PBR 是判斷整體市場多空的重要指標：

**國安基金進場時的大盤 PBR**：
- 2000/10月：PBR 約 1.8
- 2004/5月：PBR 約 1.72
- 2008/9月：PBR 約 1.37
- 2011/12月：PBR 約 1.54
- 2015/8月：PBR 約 1.49
- 2020/3月：PBR 約 1.43

**大盤 PBR 實戰判斷**：
- 大盤 PBR < 1.4：歷史上的大底部區域，長線買點
- 大盤 PBR 1.5~2.0：合理區間
- 大盤 PBR > 2.0：偏貴，開始注意風險
- 大盤 PBR > 2.5：歷史高檔，提高現金比例

**注意**：大盤 PBR 的合理區間會隨台股結構變化（科技股占比提高，合理 PBR 也會提高）。

### PBR 河流圖（PB Band）

PBR 河流圖和 [[本益比河流圖PE-Band|PE Band]] 概念相同，用歷史 PBR 區間畫出「河流」：

- 中間線 = 歷史 PBR 中位數對應的股價
- 上下軌道 = 歷史 PBR 區間對應的股價範圍
- 股價碰到下軌 → PBR 歷史低點，可能便宜
- 股價碰到上軌 → PBR 歷史高點，可能偏貴

**PBR Band vs PE Band 的選擇**：
- 獲利穩定 → 用 PE Band 更準確
- 獲利波動大（景氣循環股）→ 用 PBR Band 更穩定
- 兩者都看 → 互相驗證，結論一致時勝率最高

### 五種 PBR 陷阱識別

**陷阱一：虧損公司的 PBR < 1**
- 虧損公司淨值年年下降，PBR < 1 只是反映經營事實
- 錸德(2349)案例：PBR 0.45，但連續虧損4年，不是便宜是陷阱
- 解法：先看 ROE 是否 > 0

**陷阱二：商譽減損風險**
- 併購型公司商譽佔淨值比高，一旦減損，淨值驟降，PBR 突然變高
- 2018~2020年 A股多家公司因商譽減損導致 PBR 從 2 倍跳到 5 倍
- 解法：改用 PTB（有形淨值比），排除商譽干擾

**陷阱三：產業衰退導致 PBR 持續下滑**
- PBR 越來越低不是越來越便宜，而是產業衰退
- 面板業、DRAM業的 PBR 一波比一波低
- 解法：看 PBR 趨勢方向，而非絕對數值

**陷阱四：無形資產占比高的公司 PBR 失真**
- 科技業、品牌公司的價值在無形資產（專利、商標、客戶關係），這些不入帳
- 用 PBR 會嚴重高估這類公司的「貴」
- Google、台積電的 PBR 長期偏高，但不代表貴
- 解法：科技業、品牌股改用 PE 或 PEG，不要用 PBR

**陷阱五：資產重估增值未反映**
- 部分公司持有大量土地、房產，帳面價值遠低於市價
- PBR 看起來高，但實際資產價值更高
- 解法：查看公司不動產的市價 vs 帳面價值差異

### PBR × ROE 組合選股策略

**策略一：低 PBR + 高 ROE = 價值股**
- 篩選條件：PBR < 1.2、ROE > 10%
- 預期報酬：市場重新評價後 PBR 回升帶來的價差 + 持續獲利
- 適合標的：被錯殺的績優股、暫時受挫的景氣循環股

**策略二：歷史 PBR 低點買進景氣循環股**
- 篩選條件：PBR 在歷史區間下緣 + ROE 轉正
- 操作方式：在景氣谷底買進，景氣高峰（PBR 到歷史區間上緣）賣出
- 不適合長期持有：景氣循環股高 PBR 要賣，不是長抱

**策略三：金融股 PBR 選股**
- 金融股因為資產以金融資產為主，淨值與市價差異較小
- 金控 PBR < 1 是常態，但要區分：
  - PBR 0.8~1.0 且 ROE > 8% → 合理偏低
  - PBR < 0.7 且 ROE < 5% → 可能是價值陷阱
- 壽險股因為投資部位波動大，PBR 更不穩定

## 注意事項

### PBR 的產業特性差異

不同產業的合理 PBR 天差地別：

| 產業類型 | 合理 PBR 範圍 | 原因 |
|---------|-------------|------|
| 金融業（銀行、金控） | 0.8~1.5 | 資產以金融資產為主，淨值接近市價 |
| 景氣循環（鋼鐵、航運） | 0.8~2.0 | 獲利波動大，PBR 區間寬 |
| 傳產（水泥、紡織） | 1.0~1.8 | 資產扎實，PBR 相對穩定 |
| 科技業（半導體、IC設計） | 2.0~5.0+ | 無形資產和成長性未反映在淨值 |
| 軟體、品牌 | 5.0~20+ | 價值完全在無形資產，PBR 失去意義 |

**不要跨產業比較 PBR**：IC設計股 PBR 3倍可能很便宜，金融股 PBR 1.5倍可能已經偏貴。

### PBR 的時間維度

- **短期**（1~3個月）：PBR 波動主要反映股價變動，參考意義有限
- **中期**（6~12個月）：PBR 回歸歷史均值是可觀察的現象
- **長期**（3~5年）：PBR 趨勢方向反映產業結構變化

### 大盤 PBR 的歷史經驗

台股大盤 PBR 的歷史區間大約在 1.3~2.5 倍之間：
- 2008金融海嘯：大盤 PBR 跌到 1.37
- 2020疫情：大盤 PBR 跌到 1.43
- 國安基金進場時 PBR 通常在 1.4~1.8 之間
- 大盤 PBR > 2.0 時要提高警覺
