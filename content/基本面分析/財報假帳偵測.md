# 財報假帳偵測

> 先看老闆質押，再看應收帳款天數，這兩個是最大的警訊。

## 核心概念

### 假帳五大動機

| 動機 | 說明 |
|------|------|
| 順利上市上櫃 | 為了符合上市櫃標準，美化財報 |
| 掩飾本業衰退 | 怕財報難看導致股價下跌或銀行抽銀根 |
| 避免大股東破產 | 大股東股票質押成數高（>60%），股價不能跌 |
| 操縱股價 | 先低價買進，再假帳拉抬股價，最後高賣 |
| 掏空公司 | 把公司資產搬進自己口袋 |

### 財報假帳六大特徵

| 特徵 | 判斷標準 | 原理 |
|------|----------|------|
| 應收帳款天數過高 | > 90天 | 假交易利潤藏在應收帳款銷不掉 |
| 存貨天數過低 | < 10天且異於同業 | 虛增營收但存貨沒增加 |
| 不動產過高 | 週轉率低於同業 | 應收帳款轉入不動產消化 |
| 長期投資過高 | 未上市櫃公司投資多 | 難驗證的資產藏假 |
| 現金過高 | 現金高但借款也高 | 現金可能被限制（博達案） |
| 雜項資產過高 | 其他應收款、預付款高 | 雜項科目藏假 |

## 實戰應用

### 假帳偵測心法

1. 先看老闆質押：> 80% = 小心
2. 再看應收帳款天數：> 90天 = 警訊
3. 看存貨天數：< 10天且與同業差距大 = 警訊
4. 看現金與借款：雙高 = 現金可能被限制
5. 看營業現金流：營收成長但現金流為負 = 怪
6. 看利多頻率：澄清「不予置評」太多 = 放消息炒股
7. 看借殼歷史：借殼後 3-4 年內營收暴增 = 警訊
8. 研究產業：產業差卻賺錢 = 可能有假

### 非財務警訊

| 警訊 | 判斷方式 |
|------|----------|
| 老闆誠信被質疑 | 質押成數 > 80% = 很缺錢 |
| 經營階層變動 | 財務長、獨董、董事接連離職 |
| 股價異常飆漲 | 漲幅遠超同業且無基本面支撐 |
| 利多消息不斷 | 頻繁發布利多、澄清「不予置評」太多 |
