# 長期股權投資判讀成本法權益法子公司合營聯營

> 長期股權投資是企業持有子公司、合營企業、聯營企業的權益性投資，依持股比例與影響力分別採用成本法或權益法核算，判讀財報時必須理解兩種方法對資產、收益的呈現差異。

## 核心概念

### 三種影響力層級

| 影響力 | 持股比例 | 被投資單位稱呼 | 核算方法 |
|--------|----------|----------------|----------|
| 控制 | >50%（或實質控制） | 子公司 | 成本法（個別報表） |
| 共同控制 | 合同約定共享 | 合營企業 | 權益法 |
| 重大影響 | 20%~50% | 聯營企業 | 權益法 |
| 無重大影響 | <20% | — | 依IFRS 9金融工具處理 |

**控制判斷不只看持股比例**：即使未達50%，若符合以下條件仍視為控制：
1. 與其他投資者協議取得50%以上表決權
2. 章程或協議有權控制財務與經營政策
3. 有權任免董事會多數成員
4. 在董事會有半數以上投票權

**重大影響判斷**：即使持股<20%，若有以下情形仍視為重大影響：
1. 在被投資單位董事會派有代表
2. 參與政策制定過程
3. 與被投資單位有重要交易
4. 派出管理人員
5. 提供關鍵技術資料

### 成本法 vs 權益法

**成本法**（子公司投資，個別報表）：
- 投資帳面價值 = 初始投資成本（不隨被投資單位盈虧變動）
- 僅在收到現金股利時認列投資收益
- 優點：簡便、穩健、與稅法一致
- 缺點：無法反映投資企業在被投資單位的實際權益

**權益法**（合營企業、聯營企業投資）：
- 投資帳面價值隨被投資單位所有者權益變動而調整
- 被投資單位賺錢 → 投資帳面價值增加、認列投資收益
- 被投資單位虧損 → 投資帳面價值減少、認列投資損失
- 收到現金股利 → 沖減投資帳面價值（不是收益）

### 權益法的核心調整

#### 1. 初始投資成本調整
- **成本 > 可辨認淨資產公允價值份額**：差額視同商譽，不調整
- **成本 < 可辨認淨資產公允價值份額**：差額計入營業外收入，調增投資帳面價值（代表買到便宜貨）

#### 2. 投資損益確認（公允價值調整）
不能直接用被投資單位的帳面淨利，必須按取得投資時的公允價值調整：
- 固定資產公允價值 > 帳面價值 → 多提折舊 → 調減淨利
- 無形資產公允價值 > 帳面價值 → 多提攤銷 → 調減淨利
- 會計政策不一致 → 按投資企業政策調整

#### 3. 超額虧損處理順序
當被投資單位嚴重虧損時，投資企業認列損失的順序：
1. 沖減長期股權投資帳面價值
2. 沖減長期應收款（構成淨投資的部分）
3. 有額外承擔義務 → 認列預計負債

#### 4. 其他權益變動
被投資單位除淨損益外的權益變動（如其他綜合損益），按持股比例調整投資帳面價值，計入資本公積。

## 實戰應用

### 判讀長期股權投資品質

**1. 權益法投資收益 vs 現金股利**
- 權益法認列的投資收益是「紙上富貴」，不是真實現金流入
- 關鍵指標：**現金股利 / 權益法投資收益** 比率
  - > 80%：投資收益品質佳，有真實現金回流
  - < 50%：帳面賺但沒拿到錢，可能是被投資單位不配息或投資收益被高估

**2. 長期股權投資佔總資產比率**
- > 40%：公司本業可能被邊緣化，靠投資收益撐場面
- 比率逐年增加：可能透過投資進行盈餘管理

**3. 權益法投資損益波動度**
- 被投資單位獲利不穩定 → 母公司盈餘也跟著震盪
- 台股常見：轉投資眾多的集團股（如台塑集團、遠東集團），獲利受轉投資影響甚大

**4. 減值測試**
- 長期股權投資需定期做減值測試
- 若被投資單位持續虧損、股價低於帳面價值，可能需認列減值損失
- 減值一經認列不可迴轉（IFRS規定）

### 台股集團股判讀實戰

**集團股特徵**：
- 母公司持有大量子公司、聯營企業股權
- 個別報表看起來「長期股權投資」占比高
- 合併報表才能看到集團全貌

**判讀重點**：
1. 個別報表的長期股權投資金額 vs 合併報表的差異
2. 權益法投資收益的現金含量
3. 子公司之間的內部交易是否虛增營收
4. 參見 [[合併財報與非控制權益判讀]] 了解合併報表判讀

### 成本法轉權益法的時機判讀

當公司增持某被投資單位股權從<20%增至>20%時：
- 原本用成本法（或金融資產）→ 改用權益法
- 需追溯調整（舊準則）或以帳面價值直接轉換（新準則）
- 實戰意義：持股跨越20%門檻後，盈餘認列方式改變，可能造成帳面盈餘跳增

### 權益法投資收益的盈餘管理風險

- 母公司可透過被投資單位的配息政策調節自身盈餘
- 被投資單位若在年底前大量配息 → 母公司當期盈餘增加
- 檢查被投資單位的配息時間是否異常

## 注意事項

1. **權益法收益 ≠ 現金**：權益法下認列的投資收益未必有對應的現金流入，這是最常見的誤區。被投資單位賺錢但保留盈餘不配息時，母公司帳面有收益但沒有現金。

2. **個別報表 vs 合併報表**：個別報表中長期股權投資是單一行項目，合併報表才會展開子公司的全部資產負債。只看個別報表會嚴重低估公司的真實槓桿。

3. **持股比例陷阱**：20%和50%只是「一般原則」，實質影響力才是關鍵。低持股但實質控制的案例在台股不罕見（如透過董事會控制）。

4. **減值不可迴轉**：IFRS下長期股權投資一旦認列減值損失，不可迴轉。這與存貨跌價準備可迴轉不同。

5. **商譽隱含在成本中**：權益法下，初始投資成本大於可辨認淨資產公允價值份額的差額（商譽）不單獨列示，而是隱含在投資帳面價值中。參見 [[商譽減損與併購風險]]。

6. **公允價值調整的影響**：權益法確認投資收益時需做公允價值調整，但投資人通常只能拿到被投資單位的帳面數字，難以自行計算調整後的淨利，這增加了判讀難度。
