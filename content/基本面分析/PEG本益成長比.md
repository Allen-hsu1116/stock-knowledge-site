# PEG本益成長比

> 成長股估值的核心指標，將本益比與盈餘成長率結合，判斷成長股是否被低估或高估

## 核心概念

PEG（Price-to-Earning Growth Ratio，本益成長比）是衡量「成長型股票是否昂貴或便宜」的指標，由英國投資大師吉姆·史萊特（Jim Slater）發明，後由彼得·林區（Peter Lynch）推廣。

**公式**：
```
PEG = 本益比（PE）÷ 盈餘成長率（稅後淨利成長率）
```

**為什麼需要PEG？** 本益比只考慮公司當前獲利，沒有考慮未來成長性。當一間公司快速成長時，PE偏高是合理的——但單看PE會誤判為「太貴」而錯失機會。PEG將成長率納入考量，是本益比的進階版。

**計算範例**：
- 股價100元，預估EPS=5元，預估盈餘成長率=10%
- PEG = (100÷5) ÷ 10 = **2**

## 實戰應用

### FinLab PEG 策略回測：季換股是最佳解

FinLab 2026年回測發現反直覺的結論：**降低換股頻率反而降低風險**。基礎月換股 PEG 策略年化報酬 9.7%、MDD 50.5%，經4輪迭代優化後：

| 換股頻率 | 年化報酬 | 夏普 | MDD |
|---------|---------|------|------|
| 週換股 + RSI | 28.8% | 0.83 | 54.8% |
| 雙週 | 24.9% | 0.83 | 48.4% |
| 月換股（基礎） | 9.7% | 0.40 | 50.5% |
| **季換股** | **25.6%** | **0.93** | **37.0%** |

**季換股 MDD 最低的原因**：
1. 避免追漲殺跌：高頻換股容易在股價高點買入、低點賣出
2. 減少交易成本：頻繁換股累積手續費和滑價侵蝕報酬
3. 讓贏家充分發揮：好股票需要時間發酵，太早賣掉錯過大波段
4. 過濾短期雜訊：季度層級只看中期趨勢，不被日常波動干擾

**關鍵發現：傳統停損對成長股策略無效**。成長股在系統性下跌時同步下跌，停損只會鎖定虧損。8%停損反而讓MDD從54.8%升至58.3%，5%移動停損更讓年化報酬暴跌至14.1%。必須從策略結構（換股頻率）著手降低風險。

### PEG 依公司特性選擇交易策略

富果2025年回測發現，PEG適用性取決於公司類型：

- **穩定成長股（台積電、台達電）**：PEG適合做「第一次買進」的判斷，但成長放緩時不急於賣出，買入並長期持有獲利更高。用PEG只抓到部分上漲期，反而錯過長期上升坡段
- **景氣循環股（景碩、威剛、鴻海）**：PEG優勢明顯，因股價長期在區間波動，PEG能抓到上漲區段，獲利優於買進持有策略
- **結論**：先用PEG判斷買進時機，再依公司特性決定持有策略

### PEG 判讀標準

| PEG區間 | 意義 | 動作 |
|---------|------|------|
| > 2.0 | 過熱 | 避開 |
| > 1.2 | 高估 | 賣出/觀望 |
| = 1.0 | 合理 | 觀望 |
| < 0.75 | 低估 | 買進 |
| < 0.66 | 嚴重低估 | 果斷買進 |
| < 0.5 | 瘋狂低估 | 果斷進場 |

### PEG 計算4步驟SOP

1. **取得Trailing EPS**：近4季EPS加總
2. **取得Forward EPS**：參考至少3家券商的平均預估值（Consensus）
3. **計算EPS年成長率**：G = (Forward EPS - Trailing EPS) / Trailing EPS × 100%
4. **計算PEG**：PEG = Current PE / G

### 產業別PEG與PE參考

成長股看PE會以為很貴，看PEG反而合理。不同產業有不同估值邏輯：

| 產業 | 合理PE區間 | PEG應用 |
|------|-----------|---------|
| 半導體晶圓代工 | 20-28倍 | PEG<0.75低估 |
| IC設計 | 18-25倍 | PEG<0.75嚴重低估 |
| AI伺服器 | 15-20倍 | PEG判讀需配合營收基數 |
| 散熱族群 | 22-32倍 | 液冷派估值重新掛鉤 |

## 注意事項

### 五大陷阱

1. **PEG不是越小越好**：極小值可能代表短期獲利起伏大、股價波動劇烈的公司，而非真正被低估。基期低時短期成長率飆高，PEG驟降是假象
2. **只適用於成長股**：前提是公司本益比為正、EPS為正成長。虧損公司或衰退公司不適用
3. **業外收入造成假性成長**：單純PEG選股會把業外收入認列導致EPS暴增的公司納入，這是「假性」成長。FinLab回測證實這是PEG選股最大的問題
4. **台股PEG<0.75時機少**：台股許多績優股PEG低於0.75的時機很少，富果回測五年間只有1~2次進場時機，堅持PEG門檻可能錯失投資機會
5. **預估成長率是最大難點**：PEG假定未來長期成長率不變，但現實中企業成長率會變化。預估時應盡可能保守。計算涉及較多對未來的預估，forward PE + 預估EPS成長率比歷史數據更合理

### 停損對成長股策略無效

FinLab 2026年回測證實，傳統停損/移停對成長股策略不僅無效，反而惡化績效：

| 風控方式 | 年化報酬 | MDD | 結論 |
|---------|---------|-----|------|
| 10%停損 | 28.8% | 54.8% | 基準 |
| 8%停損 | 27.9% | 58.3% | MDD反而上升 |
| 5%移停 | 14.1% | 63.2% | 大幅惡化 |
| 20%止盈 | 28.5% | 53.2% | 僅小幅改善 |

原因：成長股在系統性下跌時同步下跌，停損會鎖定虧損後無法參與反彈。正確做法是從策略結構（季換股）降低MDD，而非依賴停損。

### 搭配使用建議

- **同時看PE和PEG**：不要單看PEG，PE是基礎、PEG是進階
- **搭配Forward PE**：法人看的是Forward PE（預估本益比），不是Trailing PE
- **排除業外收入**：用PEG選股時，先過濾掉業外收入占比過高的公司
- **參考產業PE中位數**：與同產業歷史中位數對比，而非大盤
- **動態調整**：每月檢查營收數據，確認Forward EPS成長邏輯是否消失

### 估值工具選擇

| 公司類型 | 適用指標 |
|---------|---------|
| 穩定獲利（金融、食品、公用事業） | PE本益比 |
| 高成長股（AI半導體、IC設計） | PEG |
| 資產型（金控、營建） | PB股價淨值比 |
| 資本密集（晶圓代工、電信） | EV/EBITDA |
