# 存貨計價方法與財報影響Inventory-Valuation-Methods

> 不同存貨計價方法（FIFO、加權平均、個別計價）直接影響毛利率、所得稅、流動比率，IFRS 下已禁用 LIFO，台股投資人必須理解存貨計價對盈餘的操縱空間。

## 核心概念

存貨計價方法決定了「賣出的存貨算多少成本」和「留在倉庫的存貨算多少價值」。同一批進貨，用不同方法算，利潤可以差很多。

### 五種主要方法

| 方法 | 邏輯 | 通膨時影響 | 期末存貨價值 |
|------|------|-----------|------------|
| FIFO（先進先出）| 先買的先出 | 利潤虛增、稅負較重 | 接近市價（較高）|
| 加權平均法 | 全部加權平均 | 利潤居中、稅負居中 | 居中 |
| 移動加權平均法 | 每次進貨重算 | 同加權平均 | 居中 |
| 個別計價法 | 逐批追蹤實際成本 | 最真實 | 最真實 |
| LIFO（後進先出）| 後買的先出 | 利潤較低、稅負較輕 | 偏低（遠離市價）|

### IFRS 規定：LIFO 已被禁用

台灣自 2013 年全面採用 IFRS，**LIFO 不再允許使用**。台股上市櫃公司只能用：
- FIFO
- 加權平均法
- 個別計價法
- 成本與淨變現價值孰低法（期末評價）

## 實戰應用

### 1. 判讀存貨計價方法的影響

**通膨環境下**（最常見）：
- FIFO → 銷貨成本用舊的低價算 → 毛利虛高 → EPS 偏高 → 本益比偏低（假便宜）
- 加權平均 → 自動平滑 → 毛利較真實
- 公司如果突然從加權平均改成 FIFO → 小心，可能是在美化利潤

**通縮環境下**：
- FIFO → 銷貨成本用舊的高價算 → 毛利偏低
- 結論反轉

### 2. 存貨計價方法變更 = 盈餘操縱紅旗

- 會計政策變更必須在財報附註揭露
- 如果公司無正當理由變更計價方法 → 大概率在操縱盈餘
- 特別注意：接近增資、減資、或 CEO 考績年度的變更

### 3. 存貨跌價損失（成本與淨變現價值孰低法）

- 期末存貨必須跟市價比較，取低的
- 存貨跌價損失列入當期費用 → 直接壓低利潤
- **反轉訊號**：如果公司大量提列存貨跌價損失後，次年又迴轉 → 可能是「大洗澡」(Big Bath) 操作

### 4. 產業特性決定存貨計價的影響程度

- **高存貨周轉率產業**（零售、食品）：計價方法差異小（存貨很快就賣掉了）
- **低存貨周轉率產業**（營建、半導體設備）：計價方法差異大（存貨放很久，成本落差大）
- **原物料波動大的產業**（鋼鐵、石化）：FIFO vs 加權平均差異非常顯著

### 5. 毛利率異常時查存貨

毛利率突然大幅提升，先查：
1. 是否變更存貨計價方法
2. 原物料價格是否下跌
3. 存貨跌價損失是否迴轉
4. 是否有虛增存貨（假進貨、假銷售）

## 注意事項

- **LIFO 雖然在台灣不能用，但看美股時要懂**：美國 GAAP 仍允許 LIFO，看美國公司財報時要注意
- **加權平均法最常用但最無感**：平滑效果讓你看不出成本波動，需搭配原物料價格走勢判斷
- **個別計價法最真實但最罕見**：只用在珠寶、古董、特定大單等可辨識存貨
- **存貨計價 × 存貨周轉率交叉判讀**：計價方法影響期末存貨價值，直接影響周轉率計算
- **不要只看利潤，看現金流**：FIFO 利潤高但多繳稅 → 現金流較差；LIFO 利潤低但少繳稅 → 現金流較好
