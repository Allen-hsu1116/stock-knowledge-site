# 殘餘收益模型Residual Income Valuation進階判讀

> 股票內在價值 = 淨值 + 未來殘餘收益折現總和，殘餘收益 = 淨利 - 股東資本成本，ROE > 資本成本才創造價值

## 核心概念

### 什麼是殘餘收益模型

殘餘收益模型（Residual Income Valuation Model, RIVM），又稱EBO模型或Ohlson模型，是一種基於會計資訊的股票估值方法。核心思想是：**企業只有在賺取超過股東要求報酬率的利潤時，才算真正創造價值**。

殘餘收益（Residual Income, RI）定義：
> **RIₜ = NIₜ - r × BVₜ₋₁ = (ROEₜ - r) × BVₜ₋₁**

其中：
- NIₜ = 第t期淨利
- r = 股東要求報酬率（權益資本成本）
- BVₜ₋₁ = 第t-1期每股淨值
- ROEₜ = 第t期淨資產報酬率

### 估值公式

**基本公式**：
> **V₀ = B₀ + Σ RIₜ / (1+r)ᵗ**

展開三種等價形式：
1. **V₀ = B₀ + Σ (NIₜ - r × BVₜ₋₁) / (1+r)ᵗ**
2. **V₀ = B₀ + Σ (ROEₜ - r) × BVₜ₋₁ / (1+r)ᵗ**
3. **V₀ = B₀ + Σ (Eₜ - r × BVₜ₋₁) / (1+r)ᵗ**

含義：**股票內在價值 = 當期每股淨值 + 未來殘餘收益的折現總和**

### 乾淨盈餘關係（Clean Surplus Relation）

這是殘餘收益模型的理論基礎：
> **BVₜ = BVₜ₋₁ + NIₜ - Dₜ**

期末淨值 = 期初淨值 + 淨利 - 股利。這保證了所有影響淨值的損益都經過損益表，不允許直接跳過損益表進入資產負債表的項目（即要求「綜合收益」）。

### 歷史發展

- **1961年**：Edwards & Bell 首次提出殘餘收益概念
- **1995年**：Ohlson 在《Earnings, Book Values, and Dividends in Equity Valuation》中系統化闡述，建立會計變數與公司價值的理論連結
- **1995年**：Feltham & Ohlson 提出線性資訊動態模型（LID），讓殘餘收益預測具可操作性
- 此後成為CFA考試核心估值方法之一

### Ohlson均值回歸特性

Beaver、Freeman、Ohlson和Penman的研究發現：
- 公司的投資收益率（ROE）服從**均值回歸**分佈，長期趨向資本成本
- 殘餘收益時間序列服從**自回歸分佈**，逐漸趨於零
- 市淨率（P/B）長期趨於1

這意味著：**超常報酬不可能永久維持**，這與企業生命週期理論和長期競爭均衡理論一致。

## 實戰應用

### 簡化單期模型（洪瑞泰風格）

最簡單的應用方式，假設殘餘收益永續：

> **Fair Value = BV × (1 + (ROE - r) / r)**
> = BV + (ROE - r) / r × BV

**實例：Wells Fargo（WFC）**
- ROE = 14.26%
- Book Value = $14.15
- re（要求報酬率）= 8%

Fair Value = 14.15 × (1 + (14.26% - 8%) / 8%)
           = 14.15 × (1 + 0.7825)
           = 14.15 × 1.7825
           = **$25.22**

2009/1/12收盤價 $25.14，模型估價非常接近！

### 兩階段模型（實務最常用）

將估值分為明確預測期（通常5-10年）和後續期：

> **V₀ = B₀ + Σ₁ᵀ RIₜ/(1+r)ᵗ + (Pₜ - Bₜ)/(1+r)ᵀ**

後續期假設：
- **永續零成長**：殘餘收益維持在最後一期水準不變
- **逐漸衰減**：殘餘收益以自回歸係數ω衰減（Ohlson模型）
- **完全收斂**：ROE → r，殘餘收益 → 0，Pₜ = Bₜ

### 三步實戰操作SOP

**Step 1：收集基礎數據**
- 從財報取得 BV₀（每股淨值）、ROE趨勢
- 估算 r（用CAPM或產業平均）
- 預測未來5年ROE（參考分析師預估、歷史趨勢、產業週期）

**Step 2：計算殘餘收益序列**
- 每期 RIₜ = (ROEₜ - r) × BVₜ₋₁
- 更新 BVₜ = BVₜ₋₁ + NIₜ - Dₜ（乾淨盈餘）
- 注意配息率影響BV成長速度

**Step 3：折現加總**
- 預測期殘餘收益逐期折現
- 後續期用終值假設
- V₀ = B₀ + 折現殘餘收益 + 終值折現

### 搭配杜邦分析預測ROE

透過杜邦拆解預測ROE各因子：
- 淨利率趨勢（成本控制、定價力）
- 資產週轉率趨勢（效率改善或惡化）
- 權益乘數趨勢（槓桿變化）

參考 [[ROE杜邦分析淨利率資產週轉率權益乘數]]

### 市淨率變形公式

用期末P/B取代後續期殘餘收益預測：
> **V₀ = B₀ + Σ₁ₙ RIₜ/(1+r)ᵗ + [(Pₙ/BVₙ - 1) × BVₙ] / (1+r)ⁿ**

P/B判斷：
- P/B > 1：後續期殘餘收益 > 0，公司持續創造價值
- P/B = 1：後續期殘餘收益 = 0，公司僅達正常收益
- P/B < 1：後續期殘餘收益 < 0，公司毀損價值

### 與其他估值模型的關係

**與DDM（股利折現）**：
- 數學等價：在乾淨盈餘假設下，RIVM與DDM估出相同價值
- 但RIVM不需要預測股利政策，更適合不配息公司
- RIVM著眼價值創造而非價值分配

**與DCF（現金流量折現）**：
- DCF不直接使用財報數字，需轉換
- RIVM直接用會計資訊，實用性更強
- RIVM的價值錨定在淨值（B₀），預測誤差只影響溢價部分
- 參考 [[現金流量折現法DCF估值]]

**與EVA**：
- EVA是殘餘收益的商業化版本
- EVA = NOPAT - WACC × TC（從公司整體角度）
- RIVM的RI = NI - r × BV（從股東角度）
- 參考 [[EVA經濟附加價值]]

## 注意事項

### 五大誤區與陷阱

1. **忽略乾淨盈餘關係**：直接用會計淨利而不調整為綜合收益（OCI未計入），違反模型前提。台灣IFRS 16租賃、金融工具未實現損益等可能跳過損益表
2. **資本成本估算偏差**：r差1%結果天差地遠，用歷史β估CAPM可能不準，建議用敏感度分析測試r的影響
3. **過度樂觀的ROE預測**：高ROE均值回歸是鐵律，永續假設ROE>r是不現實的。護城河公司可以維持較久，但不會永遠
4. **忽略配息率對BV的影響**：高配息率→BV成長慢→殘餘收益基礎小。低配息率→BV成長快但再投資風險大
5. **後續期假設過於簡化**：零成長假設低估、固定成長假設可能高估，建議用衰減模型（ω係數0.3-0.8）

### 適用範圍

**最適合**：
- 穩定配息的成熟企業（BV穩定可測）
- 金融業（資產負債皆以公允價值衡量，BV最接近真實價值）
- ROE穩定且可預測的公司
- 不配息或低配息的成長股（優於DDM）

**不適合**：
- 淨值為負的公司（BV₀是負的，模型基礎崩壞）
- ROE劇烈波動的景氣循環股
- 資產重估差異大的公司（BV與真實價值差距大）
- 新創公司（歷史ROE無參考價值）

### RIVM的獨特優勢

1. **價值錨定效應**：大部分價值來自B₀（通常佔60-80%），只有溢價部分需要預測，預測誤差對估值的影響比DCF小得多
2. **直接使用財報**：不需將會計數字轉換為現金流，減少轉換誤差
3. **價值創造視角**：明確指出只有ROE > r才創造價值，比單看EPS或PE更有經濟意義
4. **會計自我調整**：複式記帳的對沖效應讓會計方法選擇差異同時反映在BV和NI中，最終估值不受影響

### 關鍵參數設定建議

| 參數 | 建議 | 說明 |
|------|------|------|
| r（要求報酬率） | 8-12% | 台股可用8-9%，高波動產業用10-12% |
| 預測期 | 5-10年 | 5年為主，高度可測公司可到10年 |
| 衰減係數ω | 0.3-0.8 | 護城河強用0.7-0.8，一般公司0.3-0.5 |
| 後續期P/B | 1.0-1.5 | 成熟公司1.0，持續創造價值1.2-1.5 |
