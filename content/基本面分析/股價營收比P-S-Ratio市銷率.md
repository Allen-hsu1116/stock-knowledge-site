# 股價營收比P/S-Ratio市銷率

> 投資者願意為公司每1元營收支付多少價格，是本益比無法使用時的替代估值指標，最適合成長初期尚未獲利的公司。

## 核心概念

### 什麼是股價營收比？

股價營收比（Price-to-Sales Ratio，簡稱 P/S 或 PSR），又稱市銷率、市值營收比、市營比。它將公司的股價（或市值）與營業收入進行比較，顯示投資人願意花多少錢去買該公司每1元的營收。

由成長股投資大師 Philip Fisher 的兒子 **Kenneth L. Fisher** 於1984年在《非常潛力股（Super Stocks）》一書中提出，用來識別價值被低估的股票。

### 計算公式

**公式一（市值法）**：
```
P/S = 公司市值 ÷ 年營業收入
```

**公式二（每股法）**：
```
P/S = 股價 ÷ 每股營業收入
```

兩種計算方式結果相同，可依手邊資料選擇使用。

### 營收數據選擇

- **去年營收**：最常見，但時效性較差
- **近四季營收合計（TTM）**：更接近當前狀況，建議優先使用
- **預估營收**：最理想但最不確定

### P/S 與本益比的關鍵差異

| 比較項目 | 本益比 (P/E) | 股價營收比 (P/S) |
|----------|-------------|-----------------|
| 分母 | 稅後淨利（EPS） | 營業收入 |
| 適用情境 | 獲利穩定的成熟公司 | 成長初期、尚未獲利的公司 |
| 虧損公司 | 無法計算 | 仍可計算 |
| 操縱風險 | 較高（盈餘易被操控） | 較低（營收較難操縱） |
| 資訊深度 | 反映獲利能力 | 只反映營收規模 |

**核心原則：本益比能用時優先用本益比，P/S 是替代方案。**

## 實戰應用

### 一、P/S 適用的三種情境

1. **尚未獲利的新創公司**：初期大量投入研發與擴張，帳面虧損但營收成長中，P/E 無意義，P/S 可評估營收潛力
2. **景氣循環股**：獲利隨經濟週期大幅波動，P/E 不穩定，P/S 只看營收，提供更穩定的估值參考
3. **營收穩定但獲利不穩定的公司**：一次性支出或特殊會計處理導致獲利驟降，P/S 可排除干擾

### 二、P/S 判讀四步驟

1. **看歷史區間**：先用公司自身歷史 P/S 範圍當基準，而非用絕對數字
2. **同業比較**：與同產業、同淨利率水準的公司比較，不同產業不能直接比
3. **搭配成長性**：P/S 高 + 營收高成長 = 合理；P/S 高 + 營收放緩 = 危險
4. **搭配其他指標**：結合 P/E、P/B、FCF 等做綜合判斷

### 三、P/S 與其他指標的交叉判讀

| P/S 狀態 | P/E 狀態 | 可能含義 |
|----------|----------|---------|
| 低 | 合理 | 公司可能被低估 ✅ |
| 低 | 高 | 營收好但盈利差，成本控制有問題 ⚠️ |
| 高 | 合理 | 市場期待未來成長 🔍 |
| 高 | 低 | 獲利能力強但營收成長預期低 🤔 |

| P/S 狀態 | FCF 狀態 | 可能含義 |
|----------|----------|---------|
| 低 | 充裕 | 高投資價值 ✅ |
| 低 | 匱乏 | 營收無法轉化為現金 ⚠️ |

### 四、P/S 選股三技巧

1. **關注高成長產業**：新興產業中尚未獲利的公司，P/S 可提供估值參考
2. **尋找營收快速增長的公司**：營收增速驚人 + P/S 合理 = 潛力股
3. **結合行業分析**：P/S 低於同業平均但市佔率在擴大 = 可能被低估

### 五、Amazon 實戰案例

- 1997年 IPO 後7年沒獲利，P/E 完全失靈
- 1998-2000年 P/S 曾高達 40倍（網路泡沫極端值）
- 2003年 P/S 降至 1.8，之後長期落在 1.5~3倍區間
- **關鍵啟示**：P/S 在成長股上比 P/E 更有參考價值，但極端值仍需警惕

## 注意事項

### 六大誤區

1. **沒有絕對標準**：P/S < 1 不一定便宜，P/S > 10 不一定貴，要看行業利潤率
2. **忽略盈利能力**：P/S 只看營收，不考慮公司能不能賺錢。營收高但成本更高 = 虧錢機器
3. **忽略債務**：低 P/S + 高負債 = 價值陷阱，不是便宜
4. **跨行業比較無意義**：軟體業 P/S 正常 10~30倍，製造業可能 0.5~2倍，利潤率決定 P/S 水準
5. **營收結構變化**：併購、出售業務造成營收大變，P/S 會失真
6. **低 P/S ≠ 會漲**：可能是市場正確地看壞公司前景；高 P/S ≠ 會跌，可能是市場正確地期待成長

### 成長性的雙面刃

- **衰退公司**：P/S 便宜但營收下滑，未來 P/S 反而上升（因分母變小）
- **成長公司**：P/S 昂貴但營收成長，未來 P/S 反而下降（因分母變大）
- **關鍵**：P/S 是靜態指標，必須動態思考營收趨勢

### 何時不該用 P/S？

- 公司獲利穩定 → 用 [[本益比進階判讀PE-Ratio-Advanced]] 更好
- 利潤率波動劇烈或犧牲利潤率換營收（如電商）→ 用 [[股價現金流量比P-CF與現金流估值指標]]
- 不同產業間比較 → 無意義
