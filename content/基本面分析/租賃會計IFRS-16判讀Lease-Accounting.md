# 租賃會計 IFRS 16 判讀

> IFRS 16 讓所有超過12個月的租賃必須資本化入帳，結束了表外融資時代，投資人需重新理解資產負債表上的使用權資產與租賃負債。

## 核心概念

IFRS 16「租賃」於2019年實施，最大變革是承租人不再區分營業租賃與融資租賃，統一認列「使用權資產」與「租賃負債」。

### 新舊制差異

**舊制 IAS 17**：
- 融資租賃資本化，營業租賃不入帳（表外融資）
- 大量租賃隱藏在資產負債表外
- 低估負債、高估資產報酬率

**新制 IFRS 16**：
- 承租人單一模型：使用權資產 + 租賃負債
- 例外：短期租賃（≤12月）和低價值資產可不資本化
- 出租人仍維持雙模型

### 會計處理

**初始認列**：
- 使用權資產 = 租賃給付現值 + 初次直接成本 + 預付租金 - 租賃誘因
- 租賃負債 = 租賃給付現值（按增額借款利率折現）

**後續衡量**：
- 使用權資產：成本模式折舊
- 租賃負債：有效利率法攤銷
- 損益表：折舊費用（業內）+ 利息費用（業外），合計 > 舊制直線法租金

## 實戰應用

### 對財務報表的影響

- **資產負債表**：資產增（使用權資產）、負債增（租賃負債）、負債比率上升、ROA 下降
- **損益表**：EBITDA 大幅增加（租金費用→折舊+利息）、營業利益增加、利息費用增加
- **現金流量表**：營業活動 CF 減少（本金償還移至融資活動）、融資活動 CF 增加

### 台股高度受影響產業

1. **航空業**（長榮航、星宇航、虎航）：飛機租賃是最大表外項目，使用權資產可增逾千億
2. **零售業**（統一超、全家、寶雅）：門市租賃，使用權資產佔總資產可達 30-50%
3. **餐飲業**（王品、瓦城、漢來美食）：門市租約3-5年，IFRS 16 讓負債透明化
4. **物流業**（嘉里大榮、新竹物流）：倉庫與車輛租賃
5. **電信業**（中華電、台灣大、遠傳）：基地台、機房租賃

### 關鍵判讀指標

1. **使用權資產佔總資產比**：>30% 代表高度租賃依賴
2. **租賃負債佔總負債比**：追蹤趨勢，異常增加代表擴張或現金流壓力
3. **EBITDA 變化率**：IFRS 16 後 EBITDA 虛增，跨期與同業比較需調整
4. **租賃相關現金流**：本金償還在融資活動，利息在營業活動，計算 FCF 需注意
5. **租賃負債到期結構**：判斷短期流動性壓力與長期承諾風險

### 實戰步驟

1. 看**附註揭露**：租賃會計政策、使用權資產組成、到期分析
2. 計算**調整後財務比率**：納入使用權資產與租賃負債
3. 比較**同業租賃結構**：租賃 vs 自有的差異
4. 追蹤**租約到期時間表**：判斷續約風險
5. 分析**EBITDA 水分**：拆解增量來源是會計變動還是實質改善

## 注意事項

- **EBITDA 虛增誤導**：IFRS 16 讓 EBITDA 自動加回折舊，看似獲利提升實為會計變動
- **忽略租賃負債利息**：利息費用在業外，容易被忽略但影響淨利
- **負債比率突增恐慌**：首次適用時負債比率跳升，不代表財務惡化
- **費用型態不一致**：折舊用直線法、負債用有效利率法，導致前高後低的費用型態
- **短期租賃操縱**：公司可能拆分租約規避入帳，需查附註
