# 應計項目異象Accrual-Anomaly

> 高應計項目的公司未來報酬系統性偏低——市場高估了「沒有現金支撐的盈餘」的持續性，做多低應計、做空高應計可獲取超額報酬

## 核心概念

### 什麼是應計項目異象

Sloan (1996) 首次發現：**會計應計項目與後續股票報酬呈負相關**。這是基本面選股中最重要的市場異象之一。

**盈餘 = 現金流量 + 應計項目**

- **現金流量**：實際收到的錢，持續性高，是未來盈餘的良好預測指標
- **應計項目**：會計調整（應收帳款增加、存貨增加、折舊等），持續性低，容易被管理層操控

### 為什麼會產生異象

**盈餘固化假說（Earnings Fixation Hypothesis）**：

1. 投資人盯著EPS看，但無法區分「現金盈餘」和「應計盈餘」
2. 對高應計項目公司過度樂觀（以為今年多賺的明年還會賺）
3. 對低應計項目公司過度悲觀
4. 當預期修正時：高應計項目公司股價下跌，低應計項目公司股價上漲

### 應計項目計算（資產負債表法）

**BS_ACC = (∆CA - ∆Cash) - (∆CL - ∆STD - ∆ITP) - Dep**

| 符號 | 含義 |
|------|------|
| ∆CA | 流動資產年度變動 |
| ∆Cash | 現金及約當現金變動 |
| ∆CL | 流動負債變動 |
| ∆STD | 流動負債中的短期借款變動 |
| ∆ITP | 應付所得稅變動 |
| Dep | 年度折舊與攤銷費用 |

**簡化版**：應計項目比率 = 營業現金流 ÷ 淨利
- > 1 或接近 1：盈餘品質佳
- < 0.5：盈餘品質差
- < 0：公司虧現金卻報盈餘，高度警示

### 全球實證

LaFond (2006) 研究了17個國家 1989-2003 年數據：
- **應計項目異象是全球性現象**
- 營運資金應計項目的錯誤定價最為顯著
- 沒有單一因素能解釋各國的異象

## 實戰應用

### 基本策略

1. 計算所有股票的應計項目
2. 將股票按應計項目分成十等分（deciles）
3. **做多**：最低應計項目的股票（高現金品質）
4. **做空/避開**：最高應計項目的股票（低現金品質）
5. 每年5月再平衡（所有公司公布年報後）

### 回測績效（美股市場）

| 指標 | 數值 |
|------|------|
| 年化報酬 | 7.5% |
| 波動率 | 10.26% |
| 最大回撤 | -35.58% |
| Sharpe Ratio | 0.34 |

### 兩種不同的應計項目異象

Detzel, Schabel, Strauss (2018) 發現異象實際上是**兩個獨立現象**：

1. **投資相關應計項目**：對應風險溢價，報酬被市場情緒負向預測
   - 反映的是系統性風險，不是錯誤定價
   - 市場情緒高時投資應計項目高，後續報酬低

2. **非投資相關應計項目**：對應錯誤定價
   - 現金獲利能力只能解釋長期非投資應計項目
   - 這部分才是真正的「市場犯錯」

**實戰意義**：區分投資相關 vs 非投資相關應計項目，能更精準捕捉超額報酬。

### 異象的演變與有效性

| 時期 | 狀態 | 原因 |
|------|------|------|
| 1966-2002 | 持續有效 | 四十年穩定超額報酬 |
| 2002-2008 | 顯著減弱 | 分析師現金流量預測增加；市場學習 |
| 2008至今 | 部分反彈 | Bender & Nielsen：盈餘品質信號2008年底後反彈 |

**為什麼異象持續存在**（Lev & Nissim）：
- 極端應計項目公司常是小市值、流動性差、高交易成本
- 機構投資人難以套利（不願持有這類股票）
- 散戶無法有效執行策略（資訊成本高、交易成本高）
- **結論**：異象可能長期持續

### 台股實戰策略

1. **過濾法**：排除應計項目比率最高的30%公司，年化報酬提升2%+
2. **簡化版**：排除營業現金流÷淨利 < 0.5 的公司
3. **嚴格版**：排除營業現金流÷淨利 < 0 的公司（虧現金卻報盈餘）
4. **搭配七指標**：先用Sloan過濾，再用[[盈餘品質分析]]七項指標深入掃描

### 進階增強策略

- **多因子組合**：應計項目 + 動能 + 價值，三因子共振
- **產業內排序**：同產業內比較應計項目，消除產業差異
- **季度滾動**：改用季度數據，加快反應速度
- **搭配 [[Beneish-M-Score財報舞弊偵測]]**：高應計 + 高M-Score = 雙重紅旗

## 注意事項

1. **信心度已減弱**：OOS回測顯著負表現，原始研究可能存在數據挖掘
2. **交易成本**：高應計項目公司常伴隨小市值、低流動性，實際執行成本高
3. **異象並非穩定**：2002-2008年期間幾乎消失，需搭配其他因子使用
4. **做空的限制**：借券成本、強制回補風險，純做多低應計項目更可行
5. **投資 vs 非投資應計項目需區分**：混合計算可能稀釋信號強度
6. **金融股不適用**：應計項目結構完全不同
