# 本益比河流圖 PE Band / PE River Chart

> 把歷史本益比區間畫成河流，一眼看出股價現在是便宜還是昂貴——但只適用獲利穩定的公司

## 核心概念

本益比河流圖（PE Band / PE River Chart）是將不同倍數的本益比畫成線條，與股價走勢圖疊加呈現的視覺化工具。

**計算原理**：

```
股價 = EPS × 本益比
```

- 河流上緣 = EPS × 歷史最高本益比
- 河流下緣 = EPS × 歷史最低本益比
- 中間線條 = EPS × 各個本益比倍數

以台積電（2330）為例：2025年第20週 EPS = 45.25元，歷史本益比區間 13~23倍：

| 本益比倍數 | 對應股價 |
|-----------|---------|
| 13X | 588.2元 |
| 15X | 678.8元 |
| 17X | 769.2元 |
| 19X | 859.8元 |
| 21X | 950.2元 |
| 23X | 1,041元 |

當股價 999 元時，本益比 ≈ 22倍，落在 21X~23X 之間，偏河流上緣，代表相對歷史偏高。

## 實戰應用

### 三步判讀法

1. **看位置**：股價落在河流哪裡？
   - 偏下緣（低本益比區）→ 潛在報酬較高，可能是買點
   - 偏上緣（高本益比區）→ 潛在報酬較低，可能是賣點
   - 中間區 → 合理估值

2. **看方向**：河流往哪裡走？
   - 河流上升 → EPS 成長中，公司獲利成長
   - 河流持平 → EPS 停滯，獲利持穩或衰退
   - 河流下降 → EPS 衰退中，公司基本面惡化

3. **看結構**：河流是否規則？
   - 規則波浪狀 → 適合用河流圖判斷
   - 「斷水流」（突然跳動）→ 業外收益或一次性事件，不適合用河流圖

### 搭配其他指標

- 搭配 [[PEG本益成長比|PEG]]：河流圖看絕對本益比位置，PEG 看成長是否配得上本益比
- 搭配 [[經濟護城河Economic-Moat|經濟護城河]]：有護城河的公司本益比河流圖更穩定可靠
- 搭配 [[盈餘品質分析]]：先確認盈餘品質，河流圖才有參考價值

### 選股實戰

用本益比河流圖篩選股票的步驟：

1. 篩選 EPS 連續5年以上穩定成長的公司
2. 確認河流圖呈規則上升趨勢（非斷水流）
3. 找出目前股價落在河流下緣到中間區間的標的
4. 確認產業非景氣循環股
5. 搭配基本面分析確認公司體質

## 注意事項

### 適用範圍

**適用的股票**：
- 獲利穩定、穩定成長的股票（如台積電、中華電）
- 產業領導型公司
- EPS 連續5年以上穩定的公司

**不適用的股票**：
- 景氣循環股（本益比高低跟股價高低相反）
- 獲利不穩定的股票（河流圖不規則）
- 快速成長股（本益比失去參考意義）
- 轉機股、虧損股（EPS 為負或極度不穩定）

### 三大陷阱

1. **河流會移動**：歷史最高最低本益比會隨時間改變，今天的河流跟上個月的不一樣。本益比創新高時，整個河流圖會重新計算，不能把固定區間當絕對買賣規則

2. **斷水流假象**：業外收益、併購、一次性獲利會讓 EPS 突然跳動，河流圖出現斷層，這時河流圖失去參考價值

3. **景氣循環陷阱**：景氣循環股本益比最低時反而是景氣高峰（EPS 最高），此時股價高點本益比反而低；景氣谷底 EPS 低迷，本益比反而高。跟一般認知完全相反

### 平台差異

不同平台（Goodinfo、財報狗、玉山富果）的河流圖：
- EPS 計算方式可能不同（近四季 vs 年度預估）
- 本益比區間劃分方式不同
- 顯示期間不同（週線、月線）
- 結論方向一致但數值會有差異，不要跨平台比較數字
