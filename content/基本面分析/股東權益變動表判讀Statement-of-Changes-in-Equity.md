# 股東權益變動表判讀Statement-of-Changes-in-Equity

> 財報第四張表，記錄股東權益各項目的期初到期末變動過程，看出公司如何對待股東的錢——保留盈餘占比是判斷權益品質的關鍵

## 核心概念

股東權益變動表是資產負債表的延伸，呈現上期權益金額變動到本期權益金額的過程。回歸會計恆等式：資產＝負債＋權益，移項後權益是資產減負債後的淨額，又稱「淨值」。

### 權益六大組成項目

1. **法定資本（股本）**：已發行股數×面額。台灣面額通常10元。股本透露股票規模大小。
2. **資本公積**：投入資本中不屬於股本的部分，主要是「超額」投入資本（超過面額）。常見項目：股本溢價、資產重估增值、處分固定資產利益、企業合併利益、受領捐贈所得。資本公積可用來彌補虧損或發放股利，但動用它意味營運面臨狀況。
3. **保留盈餘**：每年淨利累積、尚未分配給股東的盈餘。期初保留盈餘＋本期淨利（或－本期淨損）－分派股利＝期末保留盈餘。分為已指撥（有限制）與未指撥（未限制）：
   - 法定盈餘公積：公司繳稅後分派稅後盈餘時應提10%
   - 特別盈餘公積：基於契約或特殊需求（如擴廠）自行提列
   - 累積虧損：持續虧損時呈現負盈餘
4. **庫藏股**：公司從市場買回股票，買回成本為權益的減項，是資本的暫時性減少。
5. **累計其他綜合損益**：匯率變動、金融資產評價等未實現損益造成的權益暫時性增減。
6. **非控制權益**：合併報表中母公司未100%持有子公司時，其他人持有的少數股權。

### 重要概念：流通在外股數

已發行股數扣除庫藏股股數＝流通在外股數。這是每股盈餘、每股淨值等評價比率的分母基礎。

## 實戰應用

### 1. 保留盈餘占股東權益比率——判斷權益品質

- 保留盈餘占比高（>50%）：股東帳上財富因公司經營有成而累積，是「有料的披薩」
- 保留盈餘占比低：股東帳上財富由新股東挹注，分食利潤的分母單位變多

### 2. 影響權益變動的常見事件判讀

| 事件 | 對權益的影響 | 判讀重點 |
|------|-------------|---------|
| 淨利 | 保留盈餘↑，權益↑ | 本業獲利能力 |
| 淨損 | 保留盈餘↓，權益↓ | 是否持續虧損 |
| 發放現金股利 | 保留盈餘↓，權益↓ | 配息率是否合理 |
| 現金增資 | 股本↑，資本公積↑，權益↑ | 是否稀釋原股東權益 |
| 現金減資 | 股本↓，權益↓ | 退還資本或提升ROE |
| 購買庫藏股 | 權益↓（減項） | 是護盤還是註銷減資 |
| 股票股利 | 保留盈餘↓/資本公積↓，股本↑ | 整體權益不變，但流通股數增加，EPS可能下降 |
| 匯率等因素 | 其他綜合損益變動 | 未實現損益的暫時性影響 |

### 3. 解讀經營階層意圖

公司會給出採取這些作法的說明，有些是真的，有些不是。投資人需要揣摩隱藏背後的真實原因、判斷公司真實意圖。例如：
- 頻繁現金增資：可能代表本業現金產生能力不足
- 保留盈餘持續減少：要查明是虧損還是高額配息
- 動用資本公積彌補虧損或發放股利：多半意味營運面臨狀況

## 注意事項

- **股票股利的稀釋效應**：盈餘轉增資或資本公積轉增資使整體股東權益不變，但流通在外股數增加，若盈餘未顯著成長，每股盈餘通常下降
- **金融股的特殊性**：IFRS新制下，金融股的其他綜合損益項目變動大，需特別關注
- **非控制權益的影響**：合併報表中少數股東權益增加，可能稀釋母公司股東的實際權益
- **累計其他綜合損益的暫時性**：匯率、金融資產評價的未實現損益可能大幅波動但非永久性
- **保留盈餘的配息基礎**：現金股利配發以盈餘為基礎，配息率過高（>80%）可能影響未來成長
