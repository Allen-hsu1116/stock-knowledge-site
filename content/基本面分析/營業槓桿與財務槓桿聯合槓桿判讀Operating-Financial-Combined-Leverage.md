# 營業槓桿與財務槓桿聯合槓桿判讀

> 固定成本放大營收對利潤的影響是營業槓桿、負債利息放大利潤對每股收益的影響是財務槓桿、兩者相乘就是聯合槓桿——三層放大效應，看懂企業風險結構的 X 光機

## 核心概念

### 營業槓桿（Operating Leverage / DOL）

由於企業存在**固定成本**，導致息稅前利潤（EBIT）變動率大於產銷量變動率的現象。

- 固定成本越高 → DOL 越大 → 營收增長時利潤暴增、營收下降時利潤暴跌
- 變動成本占比越高 → DOL 越小 → 利潤走勢較平穩

**計算公式**：
```
DOL = 邊際貢獻 / EBIT = Q(P-V) / [Q(P-V) - F]
```

- Q = 銷售數量、P = 單價、V = 單位變動成本、F = 固定成本總額
- DOL = 1 → 無固定成本，無營業槓桿效應
- DOL = 2 → 營收變動1%，EBIT變動2%

**影響DOL的四個因素**：
- 銷售數量↑ → DOL↓（反方向）
- 銷售單價↑ → DOL↓（反方向）
- 單位變動成本↑ → DOL↑（同方向）
- 固定成本↑ → DOL↑（同方向）

### 財務槓桿（Financial Leverage / DFL）

由於企業存在**固定利息支出**，導致每股收益（EPS）變動率大於息稅前利潤（EBIT）變動率的現象。

- 負債比率越高 → DFL 越大 → EBIT增長時EPS暴漲、EBIT下降時EPS暴跌
- 投資利潤率 > 負債利息率 → 正財務槓桿（股東受益）
- 投資利潤率 < 負債利息率 → 負財務槓桿（股東受損）

**計算公式**：
```
DFL = EBIT / (EBIT - I)
```

- I = 債務利息
- DFL = 1 → 無負債，無財務槓桿效應
- DFL = 1.67 → EBIT變動1%，EPS變動1.67%

**三公司比較案例**（資本總額均為200萬）：

| 項目 | A公司（零負債） | B公司（25%負債） | C公司（50%負債） |
|------|----------------|-----------------|-----------------|
| DFL | 1 | 1.25 | 1.67 |
| 基期EPS | 6.7 | 7.15 | 8.04 |
| EBIT倍增後EPS | 13.4 | 16.08 | 21.44 |
| EPS增幅 | 100% | 125% | 167% |

### 聯合槓桿（Combined Leverage / DCL）

營業槓桿與財務槓桿的綜合效應，反映**營收變動對每股收益的總放大效果**。

**計算公式**：
```
DCL = DOL × DFL = 邊際貢獻 / (EBIT - I)
```

- DCL 代表營收變動1%時，EPS變動多少%
- 兩層槓桿疊加，風險被大幅放大

## 實戰應用

### 1. 判斷企業風險結構

| 企業類型 | DOL | DFL | DCL | 風險特徵 |
|----------|-----|-----|-----|----------|
| 重資產製造業 | 高 | 中 | 高 | 營收波動直接放大到EPS |
| 輕資產服務業 | 低 | 中 | 中 | 營收波動影響較溫和 |
| 高負債企業 | 中 | 高 | 高 | 利息負擔重，EBIT波動劇烈放大 |
| 低負債穩健企業 | 中 | 低 | 中低 | EPS走勢相對平穩 |

### 2. 選股時的槓桿判斷

- **高DOL + 高DFL = 高風險高報酬**：景氣好時EPS爆發力強，景氣差時虧損也可能爆發
- **低DOL + 低DFL = 低風險低報酬**：獲利穩定但缺乏彈性，適合存股
- **高DOL + 低DFL = 中度風險**：營業風險大但財務穩健，衰退時不會被利息壓垮
- **低DOL + 高DFL = 中度風險**：營業穩定但負債重，利率上升時壓力大

### 3. 企業生命週期搭配槓桿策略

| 生命週期 | DOL建議 | DFL建議 | 理由 |
|----------|---------|---------|------|
| 引入期 | 高（固定投資多） | 低（權益融資為主） | 經營風險已高，不宜再加重財務風險 |
| 成長期 | 高 | 中 | 需要擴產但也要控制負債 |
| 成熟期 | 中 | 可適度提高 | 現金流穩定可承受較多負債 |
| 衰退期 | 降低 | 降低 | 雙重風險都要壓低 |

### 4. 景氣循環股的槓桿陷阱

景氣循環股（如鋼鐵、航運、記憶體）通常具有高DOL特徵：
- 景氣向上：營收增加 → EBIT因高DOL大幅成長 → EPS因高DFL再放大 → 獲利爆發
- 景氣向下：營收減少 → EBIT因高DOL大幅衰退 → EPS因高DFL再放大 → 虧損驚人
- **陷阱**：在景氣高峰看到高EPS就以為公司很賺，其實是雙重槓桿放大的結果，景氣反轉時EPS崩跌速度遠超預期

### 5. 盈虧臨界點的危險地帶

當企業接近盈虧臨界點（損益兩平）時，DOL趨近無窮大：
- 營收略增 → 利潤大幅跳升
- 營收略減 → 立刻由盈轉虧
- 投資這類公司等於在玩俄羅輪盤

## 注意事項

1. **DOL不是越高越好也不是越低越好**：高DOL在景氣好時是助力，景氣差時是致命傷
2. **DFL的雙面刃**：正財務槓桿讓股東受益，但負財務槓桿一旦啟動，虧損速度驚人
3. **兩種槓桿互逆原則**：經營風險高時應降低財務風險（低DFL），反之亦然，不可兩者同時拉高
4. **2008金融危機教訓**：貝爾斯登總資產/股東權益比達16倍，高槓桿在危機時讓公司瞬間倒閉
5. **DOL和DFL是動態的**：隨營收和EBIT變化，同一公司不同時期的槓桿係數也不同
6. **無息負債不計入DFL**：應付帳款等商業信用不產生固定利息，不在財務槓桿計算範圍內
