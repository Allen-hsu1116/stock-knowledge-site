# 或有負債與承諾事項判讀 Contingent Liabilities & Commitments

> 財報附註裡的隱形炸彈——不看附註，等於矇眼投資

## 核心概念

或有負債（Contingent Liability）是因過去事件產生的義務，履行與否取決於未來不確定事項。承諾事項（Commitments）是公司已承諾但尚未執行的未來支出義務。兩者都不會直接出現在資產負債表的主表中，而是隱藏在財報附註，是投資人最容易忽略的風險來源。

### IAS 37 三種分類

| 分類 | 發生機率 | 會計處理 |
|------|---------|---------|
| 很有可能（Probable） | > 50% | 認列負債準備於資產負債表 |
| 有可能（Possible） | 5%~50% | 不認列，但附註揭露 |
| 極少可能（Remote） | < 5% | 不認列也不揭露 |

### 常見的或有負債類型

1. **訴訟賠償**：侵權、違約、環境污染等判決未定
2. **產品保固**：售後保固責任
3. **連帶保證**：為關聯企業或他人提供的保證
4. **稅務爭議**：稅務機關對申報有異議
5. **合約爭議**：與客戶或供應商的糾紛
6. **環境負債**：可能被要求負擔的環境修復成本

### 常見的承諾事項類型

1. **資本承諾**：已簽約但尚未支付的固定資產購買
2. **經營租賃承諾**：未來需支付的租賃費用
3. **採購承諾**：長期合約的最低採購量
4. **授信承諾**：銀行的授信額度承諾

## 實戰應用

### 判讀五步法

1. **找附註位置**：台灣公開資訊觀測站財報附註中的「或有負債及承諾事項」
2. **逐項閱讀訴訟案件**：標的金額 vs 淨值比例、訴訟進度、公司評估（⚠️ 公司通常傾向樂觀）
3. **計算潛在損失上限**：加總最大損失金額，與淨值比較。> 20%為高風險
4. **追蹤歷年變化**：金額逐年增加？舊案長期未結？新增案件暴增？
5. **交叉驗證**：搭配董監質押比、關聯交易、營業現金流綜合判斷

### 紅旗判讀

- 🚨 或有負債金額 > 淨值 20%
- 🚨 訴訟案件長期未結（超過2年）
- 🚨 對關聯企業的大量連帶保證
- 🚨 或有負債金額逐年攀升
- 🚨 公司評估明顯偏離同業判例
- 🚨 稅務爭議金額 > 當期所得稅費用 50%

### 搭配指標交叉驗證

- **高質押比 + 高或有負債** → 極度危險，一旦觸發可能連鎖崩塌
- **高關聯交易 + 大量保證** → 集團風險溢出
- **低營業現金流 + 高承諾事項** → 未來周轉壓力

## 注意事項

1. **附註不是裝飾**：或有負債不進主表，不看附註就是矇眼投資
2. **輕信公司評估**：公司說「勝訴機率極高」但判決前的評估都是主觀的，應保守解讀
3. **忽略承諾事項**：資本承諾代表未來必須花的錢，直接影響現金流
4. **忘記合併考量**：子公司的或有負債也要併入評估
5. **只看數字不看趨勢**：單一金額不大但逐年增加代表持續暴露風險
