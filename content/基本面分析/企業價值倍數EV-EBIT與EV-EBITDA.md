# 企業價值倍數 EV/EBIT與EV/EBITDA

> 本益比的進階版——把負債和現金也算進去，用企業價值而非市值來衡量，比P/E更真實地反映收購一家公司的代價；外資常用選股指標，特別適合高折舊、重資本支出產業

## 核心概念

### 企業價值（Enterprise Value, EV）

**EV = 市值 + 總負債 + 少數股東權益 + 優先股價值 – 總現金與約當現金**

簡化版：**EV = 市值 + 淨負債**（淨負債 = 總負債 – 總現金）

EV是收購一家公司的真實代價：用市價買下所有股權、承擔所有負債、獲得帳上現金。跟市值相比，EV把債務和現金都考慮進來，避免高槓桿企業的評估誤差。簡單來說，EV就是你「無負債買下整家企業」的成本。

### EV/EBIT

**EV/EBIT = 企業價值(EV) ÷ 稅前息前利潤(EBIT)**

由價值投資者Joel Greenblatt（葛林布雷）提出。EBIT是未扣除所得稅和利息費用的盈餘，包含折舊和攤銷。

- 倍數越低代表越便宜
- 一般EV/EBIT < 10被視為可能低估
- 適合分析資本密集型企業（折舊是真實經濟成本）

### EV/EBITDA

**EV/EBITDA = 企業價值(EV) ÷ EBITDA**

**EBITDA = 稅後淨利 + 利息 + 稅 + 折舊與攤銷**

EBITDA排除了利息、稅、折舊、攤銷的影響，可以減少過去高度資本支出對獲利判斷的遮掩。概念跟本益比類似，但比本益比更適合跨產業比較。

## 實戰應用

### 估值比較選擇指南

| 指標 | 適用對象 | 特點 |
|------|---------|------|
| P/E 本益比 | 獲利穩定/成長型企業 | 最簡單，但忽略負債結構 |
| P/B 股價淨值比 | 虧損/景氣循環型企業 | 適用範圍不同 |
| EV/EBIT | 資本密集型企業 | 含折舊攤銷，較精確 |
| EV/EBITDA | 跨產業比較 | 排除折舊攤銷，泛用性最高 |

嫌麻煩用P/E，不嫌麻煩用EV/EBITDA——能得到更有參考性的數據。

### 查詢方式

美股可用 koyfin.com 查詢個股EV/EBITDA，需選擇：
- **LTM（Last Twelve Month）**：過去12個月財報資料
- **NTM（Next Twelve Months）**：未來12個月預測資料

### EV/EBITDA 的歷史由來

EV/EBITDA 的核心概念可追溯到 1980 年代，由 John Malone 在 Tele-Communications, Inc.（TCI）率先推廣。有線電視產業資本密集，需大量鋪設線路，但基礎設施完成後能穩定產生現金流。傳統淨利因折舊和利息費用被大幅侵蝕，無法反映真實現金流，EBITDA 因此應運而生。隨後被廣泛運用於鋼鐵業、無線通訊等資本密集產業。

### 台股實戰選股策略

財訊研究提出三條件挑「便宜的好公司」：
1. **本益比低於歷史平均與產業平均**
2. **EV/EBITDA 低於同業**
3. **高自由現金流 + 高殖利率**

三者共振時，代表股價被低估。年均獲利可達 18%。

### 國信策略研究結論

1. 全部個股口徑下，EV/EBITDA變化對漲跌幅的解釋力度強於PE
2. 成熟期、衰退期、增長期和復蘇期的股票，EV/EBITDA表現優於PE
3. 僅在淘汰期股票適用PE指標
4. EBITDA預測增速高於中位數的企業估值表現更好

## 注意事項

1. **沒有絕對標準**：跟本益比一樣，沒有絕對數字高低標準，無法把未來成長性考量進去（高成長反映在市值上）
2. **跨產業比較要謹慎**：不同行業EV/EBITDA差異大，高成長行業倍數高、成長緩慢行業倍數低
3. **只看便宜不看體質**：單純考慮便宜角度可以用，但公司好壞體質等因素需要搭配其他指標
4. **EBITDA的爭議**：巴菲特曾批評：「使用 EBITDA 的人不是在騙你，就是在騙自己」——因為 EBITDA 排除折舊攤銷，可能高估企業真實價值。折舊是真實支出，設備終究要維修替換。但對於高資本支出成長期企業，EBITDA確實更能反映營運能力
5. **忽略資本支出**：EBITDA未考慮設備維修與替換等未來現金支出，對資本密集產業尤為重要。必須搭配資本支出（CapEx）一起看，才能真正了解企業的自由現金流
6. **未反映稅負現實**：對高負債企業，若實際產生盈利仍需繳所得稅，會進一步削弱現金流量
7. **不適用虧損公司**：EBITDA為負時無法計算倍數

### EV/EBITDA vs P/E 選擇決策樹

```
要不要用EV/EBITDA取代P/E？
├── 公司有大量負債？ → 是 → 用EV/EBITDA
├── 公司折舊攤銷很高？ → 是 → 用EV/EBITDA  
├── 需要跨產業比較？ → 是 → 用EV/EBITDA
├── 公司在虧損？ → 是 → EV/EBITDA（EBITDA為正時）
└── 以上都不是 → P/E即可
```

### 企業價值驅動四因子

企業價值由四個核心驅動因子決定：

1. **ROIC（投入資本回報率）**：越大企業價值越大。ROIC > WACC是創造價值的前提
2. **WACC（加權平均資本成本）**：越低企業價值越大。折現率越低現值越高
3. **成長率**：越大企業價值越大。但只有ROIC > WACC時成長才創造價值
4. **存續期**：越長企業價值越大。護城河延長超額收益存續期

**關鍵公式**：企業價值 = FCF折現值，而FCF = NOPLAT × (1 - 再投資率)

當ROIC > WACC時，成長創造價值；當ROIC < WACC時，成長反而毀損價值。
