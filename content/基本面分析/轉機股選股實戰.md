# 轉機股選股實戰

> 轉機股是長期虧損或低獲利公司突然轉好，股價爆發力動輒1-3倍；選股四大重點：本業轉虧為盈、月營收創新高、三率三升、技術面多頭

## 核心概念

轉機股通常指長期虧損的公司突然某一季開始賺錢，或者平均獲利EPS都在0.2元以內突然連續大幅增加至0.5元或1元以上。操作轉機股之所以吸引人，是股價爆發力大，動輒上漲1倍至3倍，一年中只要挑到一兩支，當年績效就立於不敗之地。

個股因財報短期虧損或獲利不佳，股價容易長期在低檔區整理。一旦轉虧為盈，就會有人開始注意並進場吸收籌碼，但若要變成長線多頭股，還需要其他條件配合。

## 實戰應用

### 轉機股四大選股重點

#### 一、季報本業轉虧為盈

從季報挑選上一季虧損、本季開始賺錢的公司。**最關鍵的是獲利必須來自本業**，不是業外收益。

**業外收益判定**：
- 業外收益不宜超過本季獲利3成，越低越好
- 常見業外收益：賣地、賣廠房、賣轉投資股票 → 一次性收入，不列入觀察
- 許多個股為維持淨值5元以上而處理資產，這種不是真轉機

**另一種轉機型態**：原本是好公司突然上一季虧錢（例如賣掉虧損轉投資），本季恢復正常，若股價未大漲也可注意。

#### 二、月營收成長創新高

月營收成長是最重要的觀察指標：
- 轉機股營收通常會**創一年新高**
- 下一個月能否持續創新高或維持相對高檔才是觀察重點
- 季報EPS雖然賠錢，但最近2~3個月營收突然成長50%以上的個股也值得注意

#### 三、毛利率、營益率、淨利率三率三升

三率同步成長是轉機的重要確認訊號：

- **毛利率上升** → 新產品效益出現或規模經濟顯現，公司體質好轉
- **營業利益率上升** → 本業獲利能力提升（已扣除所有成本）
- **EPS上升** → 公司獲利能力最終結果，本業每股獲利成長，股價上漲機率越高

#### 四、技術面呈現多頭結構

基本面轉好後，還要注意股價位置和結構：
- **多頭結構或橫盤整理數月** → 可以考慮進場
- **空頭結構** → 不要進場
- 買的位置不佳很容易套牢

### 營收年增率由負轉正的股價反應

營收年增率由負轉正是最強的買進訊號之一，代表公司：
- 轉型成功
- 取得新訂單來源
- 產品逐漸受到市場青睞

但需注意：
- 由負轉正的初期仍需確認是否持續（至少觀察2-3個月）
- 若僅是基期過低造成的數字好轉，不具實質意義
- 搭配毛利率變化一起看，營收成長+毛利率提升才是真轉機

## 注意事項

### 五大陷阱

1. **業外收益假轉機**：賣資產、賣土地、賣投資的一次性收入，本業沒好轉
2. **營收成長但毛利率下滑**：可能只是降價搶單，越做越虧
3. **單月好轉不等於趨勢反轉**：至少連續2-3個月營收成長才能確認
4. **基期過低的假成長**：去年基期太低，YoY暴增只是數字好看
5. **技術面空頭不碰**：基本面再好，技術面空頭不碰，避免接刀

### 轉機股 vs 成長股

| 特徵 | 轉機股 | 成長股 |
|------|--------|--------|
| 獲利趨勢 | 由虧轉盈或由低轉高 | 持續穩定成長 |
| 股價位置 | 通常在低檔 | 通常已反映成長 |
| 風險 | 較高（可能假轉機） | 較低（已有成長紀錄） |
| 爆發力 | 動輒1-3倍 | 穩定但不誇張 |
| 觀察重點 | 營收由負轉正+三率三升 | 營收持續成長+估值合理 |

### 投資像開車，別只看後照鏡

財務報表公布的是過去的營收，但股價反映的是公司的未來。必須關注：
- 新產能的擴增
- 新技術的突破
- 新產品的開發
- 新競爭者的介入
- 消費者習慣的改變

這些因素會導致未來營運模式與過去大不相同，分析過去的營收數據來預測股價可能沒有意義。
