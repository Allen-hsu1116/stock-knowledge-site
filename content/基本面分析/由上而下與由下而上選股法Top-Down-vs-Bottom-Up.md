# 由上而下與由下而上選股法Top-Down-vs-Bottom-Up

> 兩種基本面選股流派：由上而下先看總經再看個股（重擇時），由下而上先看公司再看環境（重選股），理解差異才能建立適合自己的分析框架。

## 核心概念

基本面投資有兩大分析流派，決定了你從哪個方向切入選股決策：

### 由上而下（Top-Down Approach）

從宏觀到微觀，分析順序：總體經濟 → 產業趨勢 → 個股選擇

**核心邏輯**：企業會受整體景氣影響，因此先判斷當下整體景氣狀況是關鍵。

**分析步驟**：
1. 觀察全球經濟狀況（GDP、利率、通膨、匯率）
2. 判斷適合投資的國家/地區
3. 選擇當時有利的產業
4. 從產業中挑選潛力個股

**代表人物**：投信、大型基金。投信先選地點、再選產業後做投資決定。

### 由下而上（Bottom-Up Approach）

從微觀到宏觀，分析順序：個股基本面 → 產業競爭 → 總經環境

**核心邏輯**：企業的好壞是關鍵，只要找到好企業，就不用太在意景氣與環境影響。

**分析步驟**：
1. 閱讀財報、檢查個股基本面
2. 挑選低本益比、具盈利增長潛力的個股
3. 了解該產業的商業模式與競爭優勢
4. 總經環境用於判斷進出時機

**代表人物**：巴菲特。主張價值投資法，認為選股比擇時更重要。

### 兩者比較

| 維度 | 由上而下 | 由下而上 |
|------|----------|----------|
| 關注焦點 | 擇時 | 選股 |
| 分析起點 | 總體經濟 | 個股財報 |
| 投資期間 | 偏短中期 | 偏長期 |
| 分散程度 | 國家/產業多樣化 | 集中少數好公司 |
| 學習難度 | 高（需追蹤大量總經資訊） | 相對低（財報閱讀為主） |
| 最大風險 | 擇時錯誤錯過行情 | 選股失誤重壓單一標的 |

## 實戰應用

### 建議：先學由下而上，再補由上而下

市場先生的建議是先從由下而上開始學習，原因是：
- 參考資料較多，邏輯概念較完整
- 即使找不到很強的企業，排除有問題的企業是做得到的
- 能做好這點，至少能保護好自己的財富

由上而下的價值不在短期擇時，而在理解長期趨勢。整體經濟環境不會常常劇烈變動，如果學到一套總經預測天天在改變的作法，那肯定有問題。

### 兩者結合的實戰框架

1. **先用由上而下做大方向判斷**：目前景氣位置、哪些產業處於順風
2. **再用由下而上選具體標的**：從順風產業中找財報最強的公司
3. **最後用由上而下定進出時機**：總經環境決定加碼或減碼

### 台股實戰注意事項

- **由上而下**：台股電子類股通常引領大盤走向，外資動向（期貨未平倉）是重要的總經風向指標
- **由下而上**：台股電子業代工為主，國外抽單風險高，財報要看客戶集中度
- **匯率**：台股電子業以出口為主，匯率波動直接影響獲利，由上而下必須納入考量

## 注意事項

1. **不要二選一**：兩種方法互補而非對立，純由上而下容易錯過好公司，純由下而上容易逆勢操作
2. **由上而下的陷阱**：總經預測極難，判斷錯誤會錯過反彈收益
3. **由下而上的陷阱**：好公司也可能因總經逆風而下跌，需要有耐心和資金承受力
4. **基金績效壓力**：投信追求短期績效所以偏向由上而下，但個人投資者不需要追逐短期排名
