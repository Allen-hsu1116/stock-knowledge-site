# 損益表結構與獲利比率判讀

> 損益表是公司的「收入支出流水帳」，從營收到淨利像漏斗一層層扣除——看懂結構和三大獲利率，就能判斷公司「怎麼賺、賺得穩不穩、能不能持續賺」

## 核心概念

損益表（Income Statement）記錄公司在一段期間內的收入、支出與最終獲利，回答最核心的問題：「這家公司，真的有在賺錢嗎？」

損益表像漏斗形金字塔：
1. **營業收入** → 賣了多少
2. **營業成本** → 做或買這些東西花了多少
3. **營業毛利** = 營收 − 成本 → 產品本身賺多少
4. **營業費用**（管銷研） → 日常營運花了多少
5. **營業利益** = 毛利 − 營業費用 → 本業真正賺多少
6. **業外收支** → 非本業的收支
7. **稅前淨利** = 營業利益 ± 業外收支
8. **所得稅**
9. **稅後淨利** = 稅前淨利 − 所得稅 → 最終留在口袋的

## 實戰應用

### 三大獲利率判讀

**毛利率** = 毛利 ÷ 營收 × 100%
- 毛利率高 → 產品具溢價空間、品牌或技術門檻高（台積電58.6%）
- 毛利率低 → 競爭激烈、價格壓力大（鴻海6.3%）
- 毛利率逐年提升 → 競爭力增強

**營業利益率** = 營業利益 ÷ 營收 × 100%
- 觀察企業「體質健康與否」的關鍵指標
- 營收成長但營業利益沒跟上 → 成本或費用膨脹吃掉獲利
- 台積電營業利益率49.6% vs 鴻海3.2%——商業模式決定利潤天花板

**稅後淨利率** = 稅後淨利 ÷ 營收 × 100%
- 最終留在公司口袋裡的比例
- 觀察趨勢比單一數字更重要
- 高淨利率 → 獲利能力強；低淨利率 → 靠業外撐場面

### 營業費用率判讀

營業費用包含管理費用、銷售費用、研發費用三大部分：

- **研發費用率**：台積電約6.6%，鴻海約1.7%——高研發換取高毛利是正向經營槓桿
- **管理+銷售費用率**：鴻海毛利薄，營業費用就吃掉一半以上毛利
- 費用佔比太高會吃掉毛利 → 營業利益率下滑的隱形殺手

### 營業利益率vs淨利率的缺口

- 營業利益率與淨利率差距大 → 業外收支佔比高
- 靠業外收益撐起獲利 → 不穩定，可能是「假性成長」
- 理想情況：本業賺錢，業外加分

### 經營槓桿效果

- **正向經營槓桿**：營收成長39.5% → 營業利益成長61.8%（台積電）
- 固定費用佔比下降/產能利用率提升 → 利潤成長大於營收成長
- **溫和經營槓桿**：營收成長18.4% → 營業利益成長26.0%（鴻海）

## 注意事項

1. **只看營收不看利潤**：營收成長不代表真的賺得多，降價促銷營收好看但毛利下滑
2. **忽略業外損益**：某季獲利大增可能是賣資產或匯率收益，不代表本業變強
3. **迷信EPS**：公司可能透過減資或庫藏股提高EPS，要搭配整體獲利成長率
4. **沒對照現金流**：帳面賺錢不等於現金真的流進來，應收帳款太多會出事
5. **用單一年度評價公司**：要看3-5年長期趨勢，景氣循環產業波動大
6. **毛利率營業利益率趨勢背離**：毛利率上升但營業利益率下降 = 費用失控
7. **同業比較才有意義**：同業毛利率10%，這家公司20%才有定價優勢的結論
