# 財報三表綜合判讀與勾稽關係

> 單看一張報表只能看到片面真相，三表交叉驗證才能揪出假性獲利、盈餘品質問題和潛在地雷

## 核心概念

財報三大報表各自獨立又互相勾稽：

- **損益表**：一段時間內的賺賠成績單（流量）
- **資產負債表**：某一時點的家底快照（存量）
- **現金流量表**：一段時間內現金進出的真相（流量）

三張表不是孤立存在的，它們之間有嚴格的會計勾稽關係：

1. **損益表淨利 → 保留盈餘**：稅後淨利計入資產負債表的保留盈餘，影響股東權益
2. **資產負債表變動 → 現金流量表**：應收帳款增加→營業現金流減少；固定資產增加→投資現金流減少
3. **現金流量表 → 解釋現金增減**：期初現金 + 淨現金流 = 期末現金，對應資產負債表的現金科目

三表一起看才能看到全貌，單看一張報表容易被騙。

## 實戰應用

### 三表閱讀順序

1. **先看損益表**：公司賺不賺錢？毛利率、營利率、淨利率趨勢如何？
2. **再看資產負債表**：公司穩不穩？負債比率、流動比率、應收帳款/存貨變化
3. **最後看現金流量表**：錢真的進來了嗎？營業現金流是否為正？能否支撐投資和還債？
4. **交叉驗證**：三表數字是否一致？有沒有矛盾？

### 假性獲利辨識

最致命的組合：**損益表賺錢 + 現金流量表失血**

| 矛盾現象 | 可能問題 | 危險程度 |
|---------|---------|---------|
| 淨利成長但營業現金流下降 | 應收帳款堆積、獲利品質差 | ⚠️⚠️⚠️ |
| 營收成長但毛利率下降 | 成本失控或低價搶市 | ⚠️⚠️ |
| 淨利成長但營業利益下降 | 靠業外收入支撐 | ⚠️⚠️ |
| 保留盈餘增加但現金減少 | 盈餘品質差或大量投資 | ⚠️⚠️⚠️ |
| 資產增加但負債增加更快 | 槓桿過度 | ⚠️⚠️ |
| 營業現金流持續為負 | 即便損益表好看也終將倒閉 | 🔴🔴🔴 |

### 盈餘品質驗證

**營業現金流 / 稅後淨利 > 1** = 獲利品質好（現金真的收進來了）

- 比值 < 0.7：嚴重警示，獲利可能是紙上富貴
- 比值 0.7-1.0：需要進一步查原因
- 比值 > 1.0：獲利品質佳

### 償債能力判讀

不要只看資產負債表的比率，要搭配現金流量表：

- **流動比率 > 1.5** 是基本門檻
- **負債比率 < 50%** 較安全
- 但更重要的是：**營業現金流能否覆蓋短期負債**？
- 如果公司還債靠的是「借新還舊」（融資現金流借入→融資現金流還出），體質就不健康

### 營收成長的品質檢驗

營收成長要搭配三個指標驗證：

1. **毛利率**是否同步上升或至少持平？如果下降代表成本失控或低價衝量
2. **應收帳款週轉天數**是否增加？如果增加代表客戶付款變慢，營收品質差
3. **存貨週轉天數**是否增加？如果增加代表銷售速度跟不上進貨速度

三個指標同時惡化 = 營收成長是假的

## 注意事項

- **損益表可以美化，現金流量表很難造假**：折舊攤提、應收帳款認列時機都可以影響損益表，但現金流是實收實付
- **三表勾稽是驗證工具不是預測工具**：勾稽關係能幫你發現問題，但不能告訴你未來會怎樣
- **財報是歷史不是未來**：過去財務數字只是參考，不能線性推估未來
- **業外收入要排除**：判斷本業獲利能力時，業外收支（處分資產、匯兌損益等）應排除
- **不同產業特性不同**：資本密集產業（半導體）投資現金流自然大量為負，不代表不好
- **季節性影響**：單季財報可能失真，至少看四季或年報
