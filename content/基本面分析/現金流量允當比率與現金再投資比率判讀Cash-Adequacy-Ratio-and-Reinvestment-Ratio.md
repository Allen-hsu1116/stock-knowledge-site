# 現金流量允當比率與現金再投資比率判讀Cash-Adequacy-Ratio-and-Reinvestment-Ratio

> 現金流量允當比率衡量公司靠自身現金流支應支出的能力，現金再投資比率衡量扣除股利後可用於再投資的現金佔比，二者搭配債務保障比率構成現金流品質的三大核心指標

## 核心概念

現金流量表不只是看「現金進出」，還可以計算三個關鍵比率來量化公司的現金流健康度：

1. **現金流量允當比率**（Cash Adequacy Ratio）：公司能不能靠自己賺的錢支應資本支出、存貨增加和現金股利？
2. **現金再投資比率**（Cash Re-investment Ratio）：扣掉股利後，還有多少現金可以再投入業務？
3. **債務保障比率**（Debt Coverage Ratio）：營業現金流能覆蓋多少倍總負債？

三個比率共同回答一個核心問題：**這家公司是靠自己活，還是靠借錢活？**

## 現金流量允當比率

### 公式

現金流量允當比率 = 近5年平均營業活動淨現金流量 / 近5年（平均資本支出 + 平均存貨增加 + 平均現金股利）

### 為什麼要用5年平均？

- 單一年度可能因一次性事件（大型資本支出、處分利益）嚴重扭曲結果
- 5年平均消除景氣循環影響，更能反映公司持續性現金產生能力
- 資本支出通常有周期性（半導體建廠、電信基站升級），平均後更真實反映長期需求

### 判讀標準

- **> 1.0**：公司可以靠自身營運現金流支應主要支出，體質健康
- **0.8 ~ 1.0**：勉強自給自足，需持續監控趨勢
- **< 0.8**：長期依賴外部資金（借款、增資），體質堪憂
- **< 0.5**：高度警訊，深入檢查融資活動現金流是否持續流入

### 五大陷阱

1. **存貨減少讓比率虛增**：分母中的「存貨增加」若為負數（存貨減少），會讓分母變小、比率虛增，需檢查存貨減少原因
2. **資本支出分維持與成長**：維持性資本支出是必要支出，成長性資本支出是選擇性支出，不能一概而論
3. **產業特性差異極大**：高資本支出產業（半導體、電信）比率天然偏低但仍健康；輕資本產業（軟體、顧問）比率通常偏高
4. **比例剛好等於1不代表完美**：需要保留安全邊際，1.0 只是「剛好打平」
5. **忽略品質只看數字**：允當比率 > 1 但營業現金流品質差（大量應收帳款貢獻），反而更危險

## 現金再投資比率

### 公式

現金再投資比率 = （營業活動淨現金流量 – 現金股利）/（固定資產 + 長期投資 + 其他資產 + 營運資金）

### 分母解析

分母 = 公司總淨資產規模，代表「維持目前規模需要的資產投入」：

- **固定資產**：廠房、設備等生產性資產
- **長期投資**：轉投資、持有其他公司股票
- **其他資產**：無形資產、遞延費用等
- **營運資金** = 流動資產 – 流動負債

### 判讀標準

| 區間 | 解讀 |
|------|------|
| **7~11%** | 一般認為理想範圍，適度再投資 |
| **> 20%** | 再投資力度極大，可能是高速成長期或資本效率偏低 |
| **< 5%** | 再投資不足，可能面臨成長停滯或設備老化 |
| **負數** | 🔴 非常危險——營業現金流不足以支應現金股利，公司正在吃老本 |

### 產業差異

- **半導體**（如台積電）：資本密集，再投資比率通常 15~25%，長期維持高檔是成長訊號
- **軟體/IC設計**：輕資本，再投資比率可能 < 5%，但仍可持續成長
- **傳產/鋼鐵**：景氣循環產業，再投資比率波動大，看趨勢而非絕對值
- **金融業**：不適用此比率，資產結構完全不同

### 三大陷阱

1. **營運資金為負讓比率失真**：營運資金為負的公司（沃爾瑪模式），分母被低估，比率虛增
2. **異常高配息壓低比率**：一次性特別股利會讓比率驟降，需排除非經常性配息
3. **只看單年度不看趨勢**：比率逐年下降代表成長力道減弱，是比絕對值更重要的訊號

## 債務保障比率

### 公式

債務保障比率 = 營業活動淨現金流量 /（短期負債 + 長期負債）

### 判讀標準

- **> 0.5**：現金流對總債務覆蓋良好，財務彈性佳
- **0.2 ~ 0.5**：尚可接受，需持續監控
- **< 0.2**：年度現金流僅能覆蓋20%總債務，需關注短期償債壓力
- **負數**：🔴 公司營業現金流為負，完全無力償債

### 與利息保障倍數的區別

- 利息保障倍數看「EBIT能付多少利息」→ 利潤角度
- 債務保障比率看「現金流能覆蓋多少本金」→ 現金角度
- 利潤為正但現金流為負的公司（應收帳款暴增），利息保障倍數OK但債務保障比率極差

## 三率綜合判讀框架

### 四種典型情境

- **現金牛型**：允當比率 > 1.2、再投資比率 7~11%、債務保障 > 0.5 → 自給自足、適度再投資、償債無虞
- **成長型**：允當比率 ≈ 1.0、再投資比率 > 15%、債務保障 0.2~0.5 → 大量再投資、需監控現金流
- **吃老本型**：允當比率 < 0.8、再投資比率 < 5%、債務保障 < 0.2 → 靠外部資金、成長停滯、償債壓力大
- **擴張期型**：允當比率 < 1.0、再投資比率 > 25%、債務保障 0.1~0.3 → 暫時性低允當比率、需確認投資效益

### 五步判讀SOP

1. **計算三率**：直接從現金流量表與資產負債表取得數據，5年平均
2. **看趨勢**：單一年度可能被扭曲，趨勢比絕對值重要
3. **同業比較**：產業特性決定合理區間，跨產業比較無意義
4. **交叉驗證**：搭配 OCF/NI、利息保障倍數、流動比率等
5. **紅旗清單**：
   - ⚠️ 允當比率連續3年 < 0.8
   - ⚠️ 再投資比率連續3年 < 3%
   - ⚠️ 債務保障比率 < 0.15
   - 🔴 再投資比率為負數（立即深入檢查）

## 實戰應用

### 台積電案例分析

台積電2024年全年現金及約當現金餘額高達2.12兆元，佔總資產約40%，規模相當於中型銀行。其高資本支出特性使得：
- 允當比率可能長期在0.9~1.2之間波動（資本支出大但營業現金流也大）
- 再投資比率長期維持15~25%（大量建廠投資）
- 債務保障比率>0.5（現金流充沛覆蓋債務）

這組合是典型的「成長型」現金流型態——大量再投資但靠自身現金流支應。

### 高配息公司陷阱

一檔高殖利率股票可能：
- 現金股利高 → 再投資比率被壓低
- 但允當比率仍 > 1 → 公司確實賺得到錢
- 這種公司是「現金牛但成長有限」，不是「有問題的公司」

反之：
- 允當比率 < 0.8 且高配息 → 借錢配息的紅旗
- 再投資比率為負 → 配息超過營業現金流，正在吃老本

## 注意事項

- **三率是判斷工具不是預測工具**：告訴你公司現在的現金流體質，不能預測股價
- **產業特性是最大的變數**：跨產業比較絕對值毫無意義，一定要跟同業比
- **5年數據有時間落差**：用的是過去5年的平均，公司如果正在轉型，平均數據可能跟不上變化
- **分母的「存貨增加」最容易被忽略**：存貨減少會虛增允當比率，一定要檢查
