# 盈餘品質分析

> 帳面EPS好看不等於公司真的賺到錢——盈餘品質衡量帳面盈餘有多少是「真實現金」，有多少只是會計調整。

## 核心概念

### 盈餘品質 Earnings Quality 是什麼

衡量公司帳面盈餘有多少是「真實現金」、有多少是「會計調整（應計項目）」。品質越高代表盈餘越能反映真實經營績效、越能預測未來。

**為什麼重要**：
- 帳面EPS好看不等於公司真的賺到錢
- 應計項目（應收帳款、存貨變動等）可能隱藏財務造假
- 博達案、康友案都是盈餘品質差的公司
- Sloan (1996) 研究證實：高應計項目公司未來報酬顯著較低（應計項目異象 Accrual Anomaly）

---

### 七項盈餘品質指標（薛兆亨體系）

**所有指標 > 0 為佳**（與CMoney五指標相反，因為計算方式不同）

分析步驟：
1. 確認公司使用的會計方法是否適當
2. 查閱會計師查核報告
3. 確認公司之前是否有重編財報
4. 計算公司相關營運風險
5. 用以下七項指標進行量化分析

#### 指標一：存貨指標

**公式**：存貨指標 = 營收增加百分比 - 存貨增加百分比

**計算方式**：
1. 營收增加百分比 = (本年度營收 - 前兩年平均營收) ÷ 前兩年平均營收
2. 存貨增加百分比 = (本年度存貨 - 前兩年平均存貨) ÷ 前兩年平均存貨
3. 兩者相減即為存貨指標

**判讀**：
- > 0：存貨控管適當，盈餘品質好
- < 0：存貨堆積速度超過銷售成長，可能有滞銷風險

#### 指標二：應收款項指標

**公式**：應收款項指標 = 營收增加百分比 - 應收款項增加百分比

**判讀**：
- > 0：應收款項催收良好，產品銷售佳
- < 0：可能用賒銷墊高營收，應收帳款回收困難

#### 指標三：毛利指標

**公式**：毛利指標 = 毛利增加百分比 - 營收增加百分比

**判讀**：
- > 0：獲利能力超過營收成長，成本控制良好
- < 0：營收成長但毛利跟不上，可能降價搶單或成本失控

#### 指標四：銷管費用指標

**公式**：銷管費用指標 = 營收增加百分比 - 銷管費用增加百分比

**判讀**：
- > 0：銷管費用未因營收增加而同比例增加，經營效率提升
- < 0：費用成長超過營收成長，可能資源浪費或管理效率差

#### 指標五：備抵壞帳指標

**公式**：備抵壞帳指標 = 備抵壞帳增加百分比 - 應收款項增加百分比

**判讀**：
- > 0：備抵壞帳提列充足，財務報表可信
- < 0：備抵壞帳提列不足，有修飾財報嫌疑

#### 指標六：研發費用指標

**公式**：研發費用指標 = 研發費用增加百分比 - 同業研發費用增加百分比

**判讀**：
- > 0：研發投入高於同業，長期競爭力可能較強（但不宜過高）
- < 0：研發投入低於同業，可能犧牲長期發展換短期盈餘

**適用時機**：科技業、生技業等研發密集產業

#### 指標七：員工生產力指標

**公式**：員工生產力指標 = (本期員工平均營收 - 前期員工平均營收) ÷ 前期員工平均營收

**判讀**：
- > 0：員工生產力成長，人力素質上升
- < 0：員工生產力下降，可能人員膨脹或效率低落

---

### CMoney 五項盈餘品質指標（簡化版）

**差異說明**：CMoney的計算方向相反，**所有指標 < 0 為佳**

| 指標 | 判讀（< 0為佳） | 含義 |
|------|----------------|------|
| 存貨指標 | 存貨增加% > 營收增加% | 存貨管理不佳 |
| 應收帳款指標 | 應收增加% > 營收增加% | 催收狀況差或賒銷衝量 |
| 銷貨毛利指標 | 營收增加% > 毛利增加% | 成本控制不佳 |
| 銷管費用指標 | 銷管費用增加% > 營收增加% | 費用浪費 |
| 備抵壞帳指標 | 應收增加% > 備抵壞帳增加% | 備提不足 |

**適用時機**：快速篩選用，五項指標中負值越多越好

---

### Sloan 應計項目比率（學術版，最強）

**公式**：
應計項目 = [(Δ流動資產 - Δ現金) - (Δ流動負債 - Δ短期借款) - 折舊費用] ÷ 去年同季資產

**是什麼**：衡量公司盈餘中有多少是「沒有現金支撐的應計項目」。應計項目越高，盈餘品質越差。

**簡化版（財報狗版）**：營業現金流 ÷ 淨利
- > 1 或接近1：盈餘品質佳
- < 0.5：盈餘品質差，大量盈餘沒有現金支撐
- < 0：公司虧現金卻報盈餘，高度警示

---

### 應計項目異象 Accrual Anomaly

Sloan (1996) 發現的市場異象——高應計項目的公司未來報酬系統性偏低。

**背後邏輯**：
- 應計項目（應收帳款增加、存貨增加等）容易受管理層操控
- 市場傾向高估應計項目的持續性（以為今年多賺的明年還會賺）
- 實際上應計項目的持續性遠低於現金流
- 當市場修正預期時，高應計項目股價下跌

**實戰策略**：
- 做多：低應計項目（高現金流品質）的公司
- 做空/避開：高應計項目的公司
- 台股回測：排除高應計項目後，年化報酬提升2%+

---

## 實戰應用

### 盈餘品質分析實戰SOP

1. **快速過濾**：用營業現金流÷淨利 < 0.5 或為負的公司，直接剔除
2. **七指標掃描**：對通過第一步的公司，計算七項盈餘品質指標
3. **紅旗判斷**：
   - 3項以上指標為負 → 盈餘品質有疑慮
   - 備抵壞帳指標 + 應收款項指標同時為負 → 高度造假風險
   - 存貨指標 + 毛利指標同時為負 → 可能虛增存貨虛增盈餘
4. **Sloan過濾**：排除應計項目最高的30%公司
5. **產業校正**：金融業、營建業不適用存貨指標；研發費用指標只用於科技業

### 實例

**情境A**：某電子公司營收成長15%，但存貨增加25%
- 存貨指標 = 15% - 25% = -10%（負值，存貨失控）
- 判斷：可能出貨不如預期，存貨跌價風險高

**情境B**：某公司營收成長10%，應收帳款增加30%
- 應收款項指標 = 10% - 30% = -20%（嚴重負值）
- 判斷：高機率放寬信用政策衝營收，現金回收有問題

**情境C**：A公司淨利10億，但營業現金流只有3億
- 簡化Sloan = 3÷10 = 0.3（遠低於0.5）
- 判斷：高應計項目，70%盈餘沒有現金支撐，盈餘品質差

---

## 注意事項

- **薛兆亨七指標 > 0 為佳** vs **CMoney五指標 < 0 為佳**，計算方向相反，使用前確認是哪個體系
- 金融股不適用存貨指標、應收款項指標（無存貨、應收帳款結構不同）
- 營建股不適用存貨指標（存貨特性完全不同）
- 研發費用指標只用於科技業、生技業
- 前兩年平均的基期選擇會影響結果，建議用同產業比較
- 單一指標為負不一定有問題，要看整體組合判斷
