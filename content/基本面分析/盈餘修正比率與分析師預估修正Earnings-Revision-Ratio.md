# 盈餘修正比率與分析師預估修正 Earnings Revision Ratio

> 盈餘修正是盈餘動能的先行指標，分析師上修比下修多代表趨勢正在加速，連續上修比單次上修更有參考價值，搭配SUE和盈餘品質三因子共振是基本面選股最強訊號

## 核心概念

盈餘修正比率（Earnings Revision Ratio, ERR）衡量分析師對公司盈餘預估的上修/下修趨勢。當分析師紛紛上修EPS預估時，代表公司基本面正在改善；反之，下修趨勢代表基本面惡化。

### 為什麼盈餘修正重要？

- 分析師是有資訊優勢的專業投資者，他們的修正行為反映產業脈動
- 修正方向比修正幅度更重要——趨勢比數值關鍵
- 修正具有動量效應——連續上修的公司後續表現優於單次上修
- 修正是盈餘動能四大構面之一，與SUE、盈餘成長、盈餘品質共振效果最強

## 關鍵指標計算

### 上修比率（Upward Revision Ratio）

```
上修比率 = 上修分析師人數 / (上修人數 + 下修人數)
```

- 上修比率 > 0.7：強烈看多，分析師共識正在上修
- 上修比率 0.5 ~ 0.7：溫和看多
- 上修比率 < 0.3：強烈看空，分析師共識正在下修

### 盈餘修正動能（Earnings Revision Momentum）

```
ERM = (本期共識EPS - 上期共識EPS) / 上期共識EPS
```

- ERM > 0：共識正在上修，盈餘動能正向
- ERM < 0：共識正在下修，盈餘動能負向
- ERM 持續上升：動能加速中

### 修正幅度（Magnitude of Revision）

```
修正幅度 = (最新共識EPS - 原始預估EPS) / 原始預估EPS
```

- 正值代表上修幅度
- 負值代表下修幅度
- 搭配修正方向使用更有意義

## 三種修正型態判讀

### 型態一：持續上修（最強訊號）
- 連續2-3季分析師持續上修EPS
- 代表基本面持續改善，市場預期不斷上調
- 學術實證：持續上修的股票後續3-6個月超額報酬最顯著

### 型態二：單次大幅上修（觀察訊號）
- 一次大幅上修後沒有後續修正
- 需要觀察是否為一次性事件驅動
- 搭配SUE確認盈餘驚喜方向

### 型態三：由下修轉上修（反轉訊號）
- 連續下修後首次上修
- 這是盈餘動能反轉的強烈訊號
- 搭配轉機股策略效果最佳

## 實戰應用

### 策略一：修正動能選股
1. 篩選上修比率 > 0.7 的股票
2. ERM 連續2季為正（持續上修趨勢）
3. 搭配 SUE > 1.0（盈餘超預期）
4. 進場時機：修正趨勢確認後2-5天
5. 持有期間：3-6個月享受PEAD漂移

### 策略二：修正反轉策略
1. 篩選上修比率由 < 0.3 反轉至 > 0.5
2. ERM 由負轉正（修正方向反轉）
3. 搭配股價動量確認（價格已開始反映）
4. 停損：ERM 再次轉負

### 策略三：修正+品質雙因子
1. 上修比率 > 0.6 且 ERM > 0
2. 盈餘品質確認（OCF/NI > 0.8）
3. 應計項目占比 < 20%（排除低品質盈餘）
4. 雙因子確認降低踩雷機率

### 台股盈餘修正觀察重點

1. **投信研究報告**：台股分析師覆蓋以投信為主，上修/下修動作即時性高
2. **外資研究報告**：外資對大型股的預估權重較高
3. **月營收預估追蹤**：每月10號前公布營收，是修正EPS的依據
4. **法說會後修正**：法說會後1-2週是分析師密集修正期

## 注意事項

1. **分析師偏樂觀**：分析師有產業利益衝突，共識預估通常偏樂觀，需要打折
2. **覆蓋率偏差**：小型股分析師覆蓋少，上修比率參考性降低
3. **修正滯後性**：分析師修正通常滯後1-2個月，不是即時訊號
4. **群聚效應**：分析師傾向集體修正（羊群效應），需判斷是獨立研究還是跟風
5. **一次性修正≠趨勢**：單次大幅修正可能是事件驅動，不代表趨勢改變
6. **產業差異**：景氣循環股的修正訊號反向——下修到極致反而是低點
