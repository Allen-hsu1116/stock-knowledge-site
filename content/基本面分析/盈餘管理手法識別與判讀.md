# 盈餘管理手法識別與判讀

> 盈餘管理是在會計準則框架內「化妝」而非「整容」，但化妝品用太多一樣會遮住真面目——六大識別手法看穿管理層的盈餘美化術

## 核心概念

**盈餘管理（Earnings Management）** 是管理層利用會計判斷空間和交易安排，使財報盈餘達到特定目標的行為。

### 盈餘管理 vs 財報舞弊

| | 盈餘管理 | 財報舞弊 |
|---|---|---|
| 性質 | 在GAAP/IFRS框架內 | 超出會計準則 |
| 合法性 | 合法但可能誤導 | 違法造假 |
| 手法 | 會計判斷、交易時點安排 | 虛構交易、偽造文件 |
| 嚴重度 | 降低盈餘品質 | 完全不可信 |
| 識別 | 應計項目分析、趨勢觀察 | [[Beneish-M-Score財報舞弊偵測]] |

### 兩大類型

**1. 應計基礎盈餘管理（Accrual-Based EM）**
利用會計判斷空間調整盈餘，不改變實際經濟活動：

- 提前認列營收
- 遞延費用認列（資本化）
- 改變會計估計（折舊年限、壞帳率）
- 操控裁決性應計項目（備抵呆帳、保修準備）

**2. 真實盈餘管理（Real Activities Manipulation）**
操控實際經濟活動，改變現金流：

- 削減可裁量費用（研發、廣告、維修）
- 降價促銷衝量（季底衝營收）
- 出售資產認列收益
- 延遲投資計畫
- 股票買回時機操控

## 實戰應用

### 六大識別手法

#### 手法一：應計項目異常

最核心的識別方法。Sloan（1996）實證：**應計盈餘占比越高，盈餘持續性越低**。

**計算方式：**
應計項目 = 淨利 - 營業現金流
應計項目比率 = 應計項目 ÷ 平均總資產

**紅旗訊號：**
- 應計項目佔盈餘比例 > 50%
- 應收帳款成長率遠高於營收成長率
- 存貨周轉天數突然增加
- 連續多季應計盈餘 > 現金盈餘

#### 手法二：會計政策變更

**典型手法：**
- 折舊年限突然延長 → 降費用、增利潤
- 壞帳提列比例降低 → 降費用、增利潤
- 收入認列方式改變 → 提前認列收入
- 存貨計價方法變更 → 影響毛利率

**判斷原則：**
- 會計政策變更恰好在業績下滑時 → 高度可疑
- 變更原因「更允當表達」但業績剛好因此轉好 → 紅旗
- 一年內多次變更會計政策 → 嚴重紅旗

#### 手法三：可裁量支出異常

管理層最容易操控的三項支出：

| 支出類型 | 正常情況 | 盈餘管理特徵 |
|---|---|---|
| 研發費用 | 穩定或成長 | 季底突然大幅下降 |
| 廣告費用 | 按計畫投入 | 季底暴增後急縮 |
| 維修支出 | 定期維護 | 季度末異常減少 |

**判斷原則：**
- 研發費用率連續下降但營收成長 → 可能在削減未來投資美化當期
- 可裁量支出佔營收比突然下降 → 紅旗

#### 手法四：業外收入美化

**典型手法：**
- 季底出售資產實現利得
- 業外收入佔比突然上升
- 每年Q4都有「一次性」利得
- 處分投資收益填補本業虧損

**判斷原則：**
- 業外收入佔淨利 > 30% → 嚴重紅旗
- 「一次性」收入每年都出現 → 不是一次性
- 出售資產時機恰好在業績下滑時 → 可疑

#### 手法五：現金流與盈餘背離

**Sloan應計項目品質評估：**
- 營業現金流 > 稅後淨利 → 盈餘品質佳
- 營業現金流 < 稅後淨利 → 應計盈餘偏高，品質差
- 連續多季現金盈餘比 < 0.8 → 持續性堪憂

**現金盈餘比 = 營業現金流 ÷ 稅後淨利**
- > 1.2 → 盈餘品質優良
- 0.8-1.2 → 正常範圍
- < 0.8 → 盈餘品質堪憂
- < 0.5 → 嚴重紅旗

#### 手法六：季節性模式

**異常模式：**
- Q4盈餘異常高於前三季（年度目標壓力）
- 每年Q4都有「一次性」利得
- 每季末可裁量支出突然下降
- 季底營收暴衝但季初衰退

### 盈餘品質評估框架

| 指標 | 品質佳 | 品質差 |
|---|---|---|
| 現金盈餘比 | > 1.2 | < 0.8 |
| 應計項目比率 | < 5% | > 10% |
| 應收帳款成長 vs 營收成長 | 應收 ≈ 營收 | 應收 >> 營收 |
| 可裁量支出穩定性 | 穩定或成長 | 突然下降 |
| 業外收入佔比 | < 10% | > 30% |
| 會計政策變更頻率 | 很少 | 經常 |

## 注意事項

### 誤區一：所有盈餘管理都是壞的

適度的盈餘管理可能是合理的會計判斷，例如：
- 壞帳提列比例調整反映真實回收率變化
- 折舊年限延長反映資產使用壽命延長
- 關鍵是「動機」和「幅度」

### 誤區二：應計項目高一定有問題

有些產業應計項目本來就高：
- 營建業：存貨應計項目大
- 金融業：放款損失準備
- 要看行業特性，同業比較才有意義

### 誤區三：只看單一指標

單一指標可能誤判，應該多指標交叉確認：
- 現金盈餘比 + 應計項目比率 + 應收帳款變化
- 搭配 [[Beneish-M-Score財報舞弊偵測]] 綜合判斷
- 搭配 [[盈餘品質分析]] 系統性評估

### 誤區四：忽略趨勢

單一季度的異常可能是正當原因，連續2-3季出現同樣模式才是警訊。
