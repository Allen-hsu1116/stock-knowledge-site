# 研發費用資本化與費用化判讀R&D-Capitalization-vs-Expensing

> 研發費用「費用化」壓低當期利潤但最保守，「資本化」美化當期利潤但潛藏未來攤銷風險。判讀公司研發資本化比率是辨識盈餘管理的關鍵。

## 核心概念

### 研究階段 vs 開發階段

現行 IFRS/台灣準則採「有條件資本化」：

| 階段 | 定義 | 會計處理 |
|------|------|---------|
| 研究階段 | 原創性、計畫性調查，不確定性高 | **全部費用化** → 計入當期損益 |
| 開發階段 | 將研究結果應用於特定產品，具可行性 | **符合五條件則資本化** |

### 開發階段資本化的五個條件

1. 技術上完成該無形資產具有可行性
2. 具有完成並使用或出售的意圖
3. 能證明產生未來經濟利益（存在市場或內部使用價值）
4. 有足夠的技術、財務資源支持完成
5. 支出能夠可靠計量

**五條件全滿足 → 資本化；任一不滿足 → 費用化**

無法區分研究與開發 → 全部費用化。

### 三種處理方式比較

| 方式 | 優點 | 缺點 | 使用者 |
|------|------|------|--------|
| 全部費用化 | 最保守、無操縱空間 | 壓低利潤、不利研發投入 | 舊準則、部分稅法 |
| 全部資本化 | 反映長期投資價值 | 風險高、可能虛增資產 | 早期會計制度 |
| 有條件資本化 | 平衡真實性與謹慎性 | 劃分主觀、盈餘管理空間 | 現行 IFRS |

## 實戰應用

### 1. 研發資本化比率 = 盈餘管理溫度計

**研發資本化比率 = 資本化研發支出 / 總研發支出**

- **< 20%**：保守穩健，利潤品質高
- **20%-50%**：合理範圍，需看產業特性
- **> 50%**：偏激進，盈餘管理風險高
- **> 70%**：重大紅旗，利潤可能嚴重虛增

**跨公司比較**：同產業的研發資本化比率差異大 → 會計政策不同，利潤不可直接比較。

### 2. 台股高研發公司判讀

- **半導體設計**（聯發科、聯詠）：研發支出大，資本化比率通常 15-30%
- **軟體業**（趨勢、中華精測）：開發階段明確，資本化比率可能較高
- **製藥業**（中天生技、基亞）：臨床試驗費用是最大爭議點

### 3. 資本化的「時間炸彈」

資本化的研發支出變成無形資產後：
- 未來每年要攤銷 → 變成固定的利潤減項
- 如果研發失敗 → 要一次認列減值損失
- **高資本化比率的利潤「品質」較差**：當期利潤被美化，但未來要還債

### 4. 費用化是「免費的盈餘保護」

- 費用化把研發全扣掉 → 利潤基數低 → 未來沒有攤銷負擔
- 如果研發成功 → 利潤自然爆發（沒有攤銷吃掉）
- **巴菲特偏好費用化的公司**：利潤更真實，安全邊際更高

### 5. 稅法加計扣除的實戰意義

- 未形成無形資計：加計扣除 50%（扣 150%）
- 形成無形資產：按成本 150% 攤銷
- **稅法鼓勵研發** → 公司有動機多費用化（省稅）
- 但費用化壓低利潤 → 管理層又有動機資本化（美化報表）
- **兩股力量拉鋸** → 觀察最終比率可推測管理層傾向

### 6. 財報附註必查項目

- 研發支出總額（費用化 + 資本化）
- 資本化比率及變動
- 資本化無形資產的攤銷年限和方法
- 研發失敗的減值損失

## 注意事項

- **研究與開發階段劃分高度主觀**：這是盈餘管理的最大漏洞，公司可以調整「什麼時候算進入開發階段」
- **同產業不同公司的比率差異可能很大**：不代表高比率一定有問題，但要查原因
- **研發資本化比率高 + 利潤剛好達標**：最危險的組合，公司可能為了達標而資本化
- **新創公司通常全額費用化**：因為不確定性太高，難以證明「技術可行性」
- **稅法和會計的差異**：稅法鼓勵費用化（加計扣除），會計允許資本化（美化報表），方向相反
