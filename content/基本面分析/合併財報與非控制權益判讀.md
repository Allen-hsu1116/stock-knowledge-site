# 合併財報與非控制權益判讀

> 讀懂合併報表，區分「歸屬母公司」與「非控制權益」，才能看穿集團真實獲利。

## 核心概念

### 什麼是合併財務報表？

合併財務報表（Consolidated Financial Statements）是將母公司與所有子公司視為一個經濟主體，綜合反映整個企業集團的財務狀況、經營成果和現金流量。台股上市櫃公司幾乎都以合併報表為主要揭露文件。

**合併報表包含**：
- 合併資產負債表
- 合併綜合損益表
- 合併現金流量表
- 合併權益變動表

### 為什麼要讀合併報表？

母公司的個體報表只反映對子公司的投資金額，看不出子公司實際經營狀況。合併報表將集團視為一體，揭露：
- 集團整體的營收、獲利
- 集團整體的資產、負債
- 內部交易的抵消結果

### 非控制權益（NCI）是什麼？

**非控制權益**（Non-controlling Interest，舊稱少數股東權益/少數股權）是指子公司股東權益中，不屬於母公司的那部分。

**計算公式**：
```
NCI = 子公司股東權益 × (1 - 母公司持股比例)
```

**範例**：母公司持有子公司 80% 股權，子公司股東權益 10 億元
- 歸屬母公司：10 億 × 80% = 8 億
- NCI：10 億 × 20% = 2 億

### IFRS 下的重大改變

IFRS 採用「經濟主體理論」，NCI 的地位有根本性改變：

| 項目 | 舊準則（母公司理論） | 新準則（IFRS 經濟主體理論） |
|------|---------------------|--------------------------|
| NCI 在資產負債表 | 列在負債與權益之間 | 列在權益項下 |
| NCI 性質 | 視為外部負債 | 視為權益的一部分 |
| 損益表 NCI | 從淨利中扣除 | 在淨利下單獨列示 |
| 子公司資產評價 | 母公司持股按公允價值，NCI 按帳面價值 | 全部按公允價值 |

## 實戰應用

### 1. 歸屬母公司淨利 vs. 合併淨利 — 你賺的是哪個？

台股財報中最重要的區分：
```
合併淨利 = 歸屬母公司淨利 + 歸屬非控制權益淨利
```

**投資人真正要關注的是「歸屬母公司淨利」**，因為：
- 你持有的是母公司股票
- NCI 的利潤不歸你
- EPS 計算也是用歸屬母公司淨利 ÷ 母公司流通股數

**陷阱案例**：
- 某集團合併淨利 100 億，看似亮眼
- 但 NCI 淨利 40 億，歸屬母公司只有 60 億
- EPS 可能比預期低很多

### 2. NCI 比重過高的警訊

**判斷標準**：
- NCI 佔合併淨利比重 < 10%：正常範圍
- NCI 佔合併淨利比重 10-30%：需注意，了解子公司結構
- NCI 佔合併淨利比重 > 30%：高風險警訊

**高 NCI 比重意味什麼**：
1. 母公司對子公司持股偏低，大量利潤流向外部股東
2. 可能透過低持股控制多家子公司（槓桿效應）
3. 集團利潤成長可能不等於母公司股東可分配利潤成長

### 3. NCI 變化的趨勢判讀

**NCI 逐季增加**：
- 可能正在收購新子公司（但持股未達 100%）
- 留意收購溢價是否合理

**NCI 逐季減少**：
- 可能正在提高子公司持股（收購少數股東股份）
- 留意收購價格是否過高

**NCI 突然暴增**：
- 可能新併入了大型子公司
- 要看收購價格和商譽認列

### 4. 合併報表 vs. 個體報表對照

**實戰做法**：
1. 先看合併報表了解集團全貌
2. 再看母公司個體報表了解母公司自身狀況
3. 兩者差異 = 子公司貢獻

**關鍵差異項目**：
- 營收差異 → 子公司營收規模
- 淨利差異 → 子公司獲利貢獻
- 資產差異 → 子公司資產規模
- 長期投資差異 → 對子投資金額

### 5. 內部交易抵消的意義

合併報表會抵消以下內部交易：
- 母子公司之間的銷貨與進貨
- 母子公司之間的應收應付
- 母子公司之間的投資收益與股利

**看懂抵消的意義**：
- 合併營收可能比母+子個體營收加總少很多（內部銷貨被抵消）
- 這才是集團對外的真實營收

### 6. 商譽的認列

當母公司以高於子公司淨資產公允價值的價格收購子公司時，差額認列為商譽：
```
商譽 = 收購價格 - 子公司可辨認淨資產公允價值 × 持股比例
```

**IFRS 下的商譽**：經濟主體理論下，商譽包含 NCI 按比例分享的部分（全額商譽法）。

## 注意事項

### ⚠️ 常見陷阱

1. **只看合併淨利，忽略 NCI 扣除**：合併 EPS 與母公司 EPS 是不同的，投資人應以歸屬母公司淨利為準
2. **NCI 損益為負**：當子公司虧損時，NCI 承擔部分虧損，反而讓歸屬母公司淨利比合併淨利好
3. **商譽減損風險**：高商譽代表收購溢價高，若子公司績效不如預期，可能面臨商譽減損
4. **合併報表的債權人保護有限**：子公司的債權人只能對子公司追償，不能對母公司追償（除非有連帶保證）
5. **合併報表不等於公司可分配利潤**：利潤分配以個體報表為依據

### 🔍 實戰檢查清單

讀合併報表時，必查項目：
- [ ] 歸屬母公司淨利 vs. 合併淨利（NCI 佔比多少？）
- [ ] NCI 趨勢（逐季增加或減少？）
- [ ] 商譽金額（佔資產比例？是否有減損風險？）
- [ ] 內部銷貨抵消金額（對外真實營收多少？）
- [ ] 重要子公司揭露（名稱、持股、營收、淨利）
