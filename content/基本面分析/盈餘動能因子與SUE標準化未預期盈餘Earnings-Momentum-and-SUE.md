# 盈餘動能因子與SUE標準化未預期盈餘 Earnings Momentum & SUE

> 盈餘動能是獨立於價值、規模、價格動量的第四因子，SUE量化盈餘驚喜程度，正驚喜後續3-6個月仍持續上漲（PEAD），是基本面選股中最被低估的超額報酬來源

## 核心概念

盈餘動能（Earnings Momentum）是指公司盈餘持續成長或改善的趨勢力道，不同於單次盈餘數字，它衡量的是盈餘的「方向與加速度」。SUE（Standardized Unexpected Earnings）是量化盈餘動能最有效的指標。

**為什麼盈餘動能重要？**
- Ball & Brown (1968) 首次發現盈餘公告後價格漂移（PEAD）
- Bernard & Thomas (1989, 1990) 證實 PEAD 效應持續3-6個月
- 盈餘動能因子是獨立於價值、規模、動量因子的第四因子
- 市場對盈餘資訊反應不足（閱讀理解問題），這是超額報酬的來源

## SUE 標準化未預期盈餘

### 計算公式

```
SUE = (實際EPS - 預期EPS) / 歷史盈餘驚喜的標準差
```

### 三種預估方法

**方法一：分析師共識預估**
- 預期EPS = 分析師平均預估
- 優點：最即時、反映市場最新資訊
- 缺點：依賴分析師覆蓋率，小型股可能無預估

**方法二：隨機漫步模型（Random Walk）**
- 預期EPS = 上一季EPS
- 最簡單的模型，假設盈餘是隨機漫步

**方法三：季節性隨機漫步模型**
- 預期EPS = 去年同季EPS
- 考慮季節性因素，更準確
- 台股適用此模型因為有明顯的季節性

### SUE 判讀閾值

- **SUE > 2.0**：極度正驚喜，強烈看多訊號
- **SUE 1.0 ~ 2.0**：溫和正驚喜，偏多訊號
- **SUE -1.0 ~ 1.0**：符合預期，中性
- **SUE -2.0 ~ -1.0**：溫和負驚喜，偏空訊號
- **SUE < -2.0**：極度負驚喜，強烈看空訊號

## 盈餘動能的四大構面

### 1. 盈餘驚喜（Earnings Surprise）
- SUE 是量化驚喜的最佳指標
- 驚喜方向比驚喜幅度更重要
- 正驚喜後續表現 > 負驚喜，但市場反應不足是超額報酬來源

### 2. 盈餘修正（Earnings Revision）
- 分析師上修/下修EPS預估的趨勢
- 上修比率 = 上修人數 / (上修人數 + 下修人數)
- 連續上修 > 單次上修（方向持續性更重要）
- 台股觀察重點：投信研究報告的盈餘修正

### 3. 盈餘成長（Earnings Growth）
- EPS 年增率（YoY）持續為正且加速成長
- 季增率（QoQ）注意淡旺季調整
- 成長加速 > 成長持平 > 成長減速
- 成長減速是最容易被忽略的警訊

### 4. 盈餘品質（Earnings Quality）
- 營業現金流 / 淨利 > 1 確認盈餘真實性
- 應計項目占比低 = 盈餘品質好
- 盈餘品質差的股票，盈餘動能效應會打折

## 實戰應用

### 策略一：SUE 多頭策略
1. 篩選 SUE > 1.5 的股票（盈餘超預期）
2. 搭配盈餘品質過濾（OCF/NI > 0.8）
3. 等待財報公告後2-5天進場（避免公告日波動）
4. 持有3-6個月享受 PEAD 漂移

### 策略二：盈餘動能+價格動能雙因子
1. SUE > 1.0 且 SUE 持續上升
2. 過去6個月價格動能為正
3. 兩因子共振時超額報酬最高
4. 學術實證雙因子年化超額報酬達12-15%

### 策略三：盈餘反轉策略
1. 上一季 SUE < -1.5（嚴重低於預期）
2. 本季盈餘由負驚喜轉正驚喜
3. 轉折點是盈餘動能最強的訊號
4. 搭配轉機股選股使用

### 台股盈餘動能特殊性

1. **月營收領先性**：台股每月公布營收，是盈餘動能的領先指標
   - 月營收YoY由負轉正是最強訊號

2. **法人預估差異**：投信與外資預估可能不同

3. **季報公告效應**：台股季報法定公告期限內，盈餘驚喜效應更明顯

4. **成長股特性**：台股電子股盈餘動能效應比傳產更顯著
   - 景氣循環股反而呈現反向效應

## 注意事項

1. **只看驚喜方向不看品質**：低品質盈餘的正驚喜效果持續時間短
2. **忽略盈餘增速的變化**：成長減速是比盈餘絕對值更重要的訊號
3. **過度依賴單一季度**：至少看2-3季盈餘動能趨勢
4. **忽視預期調整**：分析師可能集體偏樂觀，共識預估需要打折
5. **忘記產業特性**：景氣循環股的盈餘動能邏輯與成長股相反
6. **忘了結合價格動量**：盈餘動能單獨使用效果有限，搭配價格動量效果最佳
