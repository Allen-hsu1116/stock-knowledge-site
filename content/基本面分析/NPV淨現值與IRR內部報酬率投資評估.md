# NPV淨現值與IRR內部報酬率投資評估

> NPV告訴你投資能「賺多少錢」，IRR告訴你投資的「報酬率多高」，兩者搭配使用才是完整的投資決策工具

## 核心概念

### NPV（Net Present Value，淨現值）

把投資案未來所有現金流按照折現率折算到現在，減去初始投資成本後的淨額。

**公式**：NPV = -C₀ + Σ CFₜ / (1+r)ᵗ

- C₀ = 初始投資（負值）
- CFₜ = 第 t 期現金流
- r = 折現率（通常用 [[WACC加權平均資本成本]]）
- t = 期數

**判斷標準**：
- NPV > 0 → 投資創造價值，值得進行
- NPV = 0 → 剛好回本
- NPV < 0 → 投資毀損價值，不建議

### IRR（Internal Rate of Return，內部報酬率）

讓 NPV 剛好等於零的那個折現率。代表投資案本身的年化報酬率。

**公式**：找到讓 Σ CFₜ / (1+IRR)ᵗ = 0 的 IRR 值

**判斷標準**：
- IRR > 資金成本 → 值得投資
- IRR = 資金成本 → 剛好回本
- IRR < 資金成本 → 不值得投資

## 實戰應用

### NPV 與 IRR 的搭配決策矩陣

| 條件 | 解讀 | 操作 |
|------|------|------|
| IRR > 資金成本，NPV > 0 | 報酬率達標且創造正價值 | ✅ 建議進行 |
| IRR > 資金成本，NPV ≈ 0 | 報酬率勉強達標 | ⚠️ 邊際投資，需觀察風險 |
| IRR < 資金成本，NPV < 0 | 報酬率不達標且毀損價值 | ❌ 應避免 |
| IRR 高但 NPV 小 | 小金額高報酬率但總價值有限 | 視資金規模決定 |

### 股票投資的 IRR 應用

1. **估算買進價的預期報酬率**：已知未來配息與賣出價，反推 IRR 判斷當前價格是否便宜
2. **比較不同標的**：同樣的預期現金流，IRR 越高代表買進價越便宜
3. **定期定額 vs 單筆投入**：定期定額的 IRR 通常較高（因為後期投入的資金時間價值不同）
4. **DCF 估值的倒推**：[[現金流量折現法DCF估值]] 本質就是找 NPV > 0 的買點，IRR 就是讓 NPV = 0 的折現率

### Excel 計算方式

- **IRR 函數**：`=IRR(values, [guess])`，values 須至少一正一負
- **NPV 函數**：`=NPV(rate, value1, value2, ...)`
- **XIRR 函數**：現金流發生時間不規則時使用 `=XIRR(values, dates, [guess])`
- **MIRR 函數**：`=MIRR(values, finance_rate, reinvest_rate)`

### 洪瑞泰 IRR 選股法（巴菲特班）

1. 選股三條件：ROE 穩定、配息率 > 40%、不會變的龍頭股
2. 便宜價 = IRR > 15% 的買進價（定存利率 6.7% 的 2 倍以上）
3. 用 ROE 和買進價反推預期 IRR
4. 常利 = 淨利 - 一次性利益（賣土地、賣股票等）
5. 預期常利 = 近 2 年平均 × 0.7 + 近 5 年中位數 × 0.3

## 注意事項

### NPV 的五大誤區

1. **折現率差 1%，結果天差地遠**：折現率的選擇是 NPV 最大的不確定性來源，建議用 [[WACC加權平均資本成本]] 並做敏感度分析
2. **現金流預測的垃圾進垃圾出**：NPV 精確到小數點是假精確，輸入假設才是關鍵
3. **忽略選擇權價值**：NPV 是靜態評估，不考慮未來決策彈性（如擴張、放棄的選擇權）
4. **不同規模投資不能只看 NPV**：NPV 10 億的投資案可能比 NPV 1 億的更糟（投入可能 100 億 vs 2 億）
5. **NPV 與 IRR 矛盾時以 NPV 為主**：規模差異大的投資案之間，IRR 可能誤導

### IRR 的四大陷阱

1. **多重 IRR 問題**：現金流正負交替時，數學上可能存在多個 IRR 讓 NPV = 0，此時應改用 [[MIRR修正內部報酬率]] 或直接看 NPV
2. **再投資假設不切實際**：IRR 隱含假設中間收回的現金流可以 IRR 再投資，實際上很難達到
3. **規模忽略問題**：IRR 是百分比，小金額高報酬率的 IRR 可能好看但總獲利有限
4. **短期偏誤**：短期投資的 IRR 通常較高，但長期投資的總價值可能更大

### IRR vs 年化報酬率（CAGR）

| 比較項目 | IRR | CAGR |
|----------|-----|------|
| 適用情境 | 多期現金流（定期定額、配息再投入） | 單筆投入（買入→持有→賣出） |
| 現金流時間 | 考慮每筆現金流的時間 | 只看起點和終點 |
| 計算方式 | 迭代求解折現率 | (終值/初值)^(1/n) - 1 |
| 股票應用 | 評估配息+賣出的綜合報酬 | 評估純資本利得的成長率 |
