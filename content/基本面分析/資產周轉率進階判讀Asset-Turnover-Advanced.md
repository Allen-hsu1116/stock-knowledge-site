# 資產周轉率進階判讀Asset-Turnover-Advanced

> 資產周轉率是杜邦分析三因子中的「效率因子」，同樣的淨利率下，周轉率越高ROE越高——看懂周轉率才能分辨公司是靠利潤還是靠效率賺錢

## 核心概念

資產周轉率 = 銷售收入淨額 / 平均資產總額

衡量企業每一塊錢資產能創造多少營收，是杜邦分析三因子中的「營運能力」因子：

**ROE = 淨利率 × 資產周轉率 × 權益乘數**

### 三層次拆解

資產周轉率可以進一步拆解：

- **總資產周轉率** = 營收 / 平均總資產
- **流動資產周轉率** = 營收 / 平均流動資產
- **固定資產周轉率** = 營收 / 平均固定資產
- **應收帳款周轉率** = 營收 / 平均應收帳款
- **存貨周轉率** = 營業成本 / 平均存貨

### 周轉率與周轉天數的換算

周轉天數 = 365 / 周轉率

周轉率越高 → 周轉天數越短 → 資金效率越好

## 實戰應用

### 產業特性決定基準值

| 產業類型 | 總資產周轉率 | 特徵 |
|---------|------------|------|
| 超商零售 | 2.0-4.0+ | 薄利多銷，周轉極快 |
| 電子製造 | 0.8-1.5 | 中等周轉，靠毛利率獲利 |
| 半導體 | 0.3-0.8 | 重資產，靠高淨利率獲利 |
| 金融保險 | 0.03-0.08 | 極低周轉，靠槓桿獲利 |

**關鍵原則**：同行業比較才有意義，跨產業周轉率差異極大。

### 杜邦分析的七種公司類型（周轉率視角）

| 類型 | 淨利率 | 周轉率 | 獲利模式 |
|------|--------|--------|---------|
| 奢侈品/軟體 | 高 | 低 | 靠利潤，不在乎效率 |
| 超商/量販 | 低 | 高 | 靠效率，薄利多銷 |
| 優質企業 | 高 | 高 | 利潤效率雙高，稀有 |
| 銀行/保險 | 中 | 極低 | 靠槓桿，周轉率失真 |
| 價值陷阱 | 低 | 低 | 什麼都不行 |

### 周轉率異常判讀

1. **周轉率突然上升但營收沒變**：可能是報廢大量固定資產導致分母變小，不是效率提升
2. **周轉率長期偏低**：公司有閒置資產，應處置或提高營收
3. **周轉率下降但營收成長**：資產擴張速度超過營收成長，需要觀察擴張是否帶來未來營收
4. **分子分母口徑不一致**：總資產包含投資性資產，但營收不包含投資收益，跨公司比較時需注意

### 周轉率趨勢分析

- **周轉率持續上升**：公司營運效率持續改善，或輕資產轉型中
- **周轉率持續下降**：營收成長跟不上資產擴張，可能過度投資
- **周轉率穩定**：產業成熟期，營收與資產同步增長

### 台股實戰：周轉率與ROE的搭配判讀

- 台積電：低周轉率（0.4-0.5）+ 高淨利率（40%+）+ 中等槓桿 → 靠利潤驅動
- 統一超：高周轉率（2.5+）+ 低淨利率（3-4%）+ 低槓桿 → 靠效率驅動
- 兆豐金：極低周轉率（0.03-0.05）+ 中等淨利率 + 高槓桿 → 靠槓桿驅動

## 注意事項

- **跨產業比較無意義**：金融業周轉率天生極低，不代表效率差
- **分子分母口徑問題**：總資產含對外投資，但營收不含投資收益，直接比會失真
- **季節性波動**：營收有淡旺季的公司，用年均資產計算比季末資產更準確
- **資產減損影響**：大幅減損會讓分母突然變小，周轉率虛高
- **合併報表的少數股權**：少數股權影響總資產但不影響母公司營收，需用歸屬母公司視角
