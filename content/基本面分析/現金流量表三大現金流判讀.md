# 現金流量表三大現金流判讀

> 現金流量表是企業的血液循環系統——損益表看帳面盈虧，現金流量表看真實現金進出，三大現金流組合模式直接反映公司生命週期與財務體質

## 核心概念

現金流量表（Statement of Cash Flows）記錄企業在特定期間內所有現金的來龍去脈。與損益表的「權責發生制」不同，現金流量表採用「現金收付制」，只記錄實際的現金進出。

### 為什麼現金流量表是投資者的「照妖鏡」？

損益表上的淨利可能是靠應收帳款堆出來的——貨賣出去了就認列收入，但現金可能還沒收到。現金流量表揭露的是「口袋裡實際能花的錢」。

**最危險的訊號**：淨利為正，但營運現金流為負 → 賺的都是收不回來的帳

### 編制方法：間接法

我國大部分企業採用「間接法」編制現金流量表：
- 從「稅前淨利」出發
- 加回沒有實際現金流出的項目（折舊、攤銷）
- 調整營運資金變動（存貨、應收帳款、應付帳款）
- 回推出實際現金流量

投資者不需要搞懂每個會計項目的回推計算，重點是理解三大活動現金流的正負涵義。

### 三大現金流

| 活動 | 代號 | 代表意義 | 正數涵義 | 負數涵義 |
|------|------|----------|----------|----------|
| 營業活動 | CFO | 本業賺錢能力 | 本業現金流入，體質健康 | 本業燒錢，需外部融資 |
| 投資活動 | CFI | 投資與擴張 | 可能變賣資產 | 投資擴廠研發（通常為負） |
| 融資活動 | CFF | 外部資金往來 | 借款或募資 | 還債或發股利（穩健表現） |

### 八大現金流組合模式

| CFO | CFI | CFF | 公司狀態 | 實戰判斷 |
|-----|-----|-----|----------|----------|
| + | - | - | 成熟穩健型 | ✅ 最理想：本業賺錢、投資未來、回饋股東 |
| + | - | + | 成長擴張型 | ✅ 本業賺錢、借錢加碼投資 |
| + | + | - | 收縮整頓型 | ⚠️ 本業賺錢但賣資產還債，停止擴張 |
| + | + | + | 積極籌資型 | ⚠️ 三路現金流入，錢從哪來？ |
| - | - | + | 新創燒錢型 | ⚠️ 本業虧錢、投資擴張、靠外部融資 |
| - | + | + | 求生變賣型 | ❌ 本業虧錢、賣資產、借錢度日 |
| - | - | - | 全面失血型 | ❌ 三路現金流出，即將斷氣 |
| - | + | - | 收縮還債型 | ❌ 本業虧錢、賣資產還債 |

## 實戰應用

### 1. 三步驟快速檢驗法

1. **看營運現金流**：是否長期為正？最基本的生存條件
2. **比較營運現金流與淨利**：理想狀況下CFO ≥ 淨利潤。CFO遠小於淨利 → 應收帳款過多
3. **看自由現金流**：是否長期為正？代表扣除必要支出後還有餘裕

### 2. 間接法調整項目判讀

| 調整項目 | 增加的意義 | 減少的意義 |
|----------|-----------|-----------|
| 折舊/攤銷 | 加回來：非實際現金支出 | — |
| 應收帳款增加 | — | 錢還沒收到，現金流出 |
| 應收帳款減少 | 錢收回來了，現金流入 | — |
| 存貨增加 | — | 買了更多存貨，現金流出 |
| 存貨減少 | 存貨賣掉變現，現金流入 | — |
| 應付帳款增加 | 延遲付款，保留現金 | — |
| 應付帳款減少 | — | 付清欠款，現金流出 |

### 3. 產業差異——不能用一套標準衡量

| 產業 | 現金流特徵 | 判讀重點 |
|------|-----------|----------|
| 製造/零售 | 高存貨、高資本支出 | 需穩定CFO支撐 |
| 軟體/服務 | 資本支出低 | FCF轉換率可能非常高 |
| 營建業 | 波動性大 | 需看長期平均，工程款入帳時間點劇烈變化 |
| 生技業 | 研發期無CFO | 完全依賴融資活動，風險極高 |
| 科技成長 | FCF可能為負 | 只要CFO成長，負FCF是投資未來 |
| 公用事業 | 穩定高FCF | 高FCF是優點，持續回饋股東 |

### 4. 實戰案例：得力(1464) — 損益表有獲利但現金流為負

得力Q3營業利益獲利2億多，但營運現金流卻流出3.1億元：
- 存貨增減：購買存貨讓現金流出2.42億元
- 應收帳款：6.1億元款項尚未收到現金

→ 追蹤重點：應收帳款何時收回？存貨是否滞銷？

### 5. 實戰案例：儒鴻(1476) — 損益表獲利低但現金流健康

儒鴻Q3稅前淨利22億元，但營業現金流入26.7億元：
- 折舊費用2.25億元加回（非實際現金流出）
- 應付帳款增加2億元（延遲付款保留現金）

→ 現金流優於帳面數字，財務體質比損益表顯示的更健康

## 關鍵現金流量指標

### 自由現金流量（FCF）

FCF = 營運活動現金流量 – 資本支出

持續產生正向且成長的FCF，代表商業模式強勁、具有護城河。

### 營運現金流量比率

營運現金流量 / 流動負債

衡量短期償債能力，比率越高越安全。

### 現金流量允當比率

近5年營運現金流量 / (近5年資本支出 + 存貨增加額 + 現金股利)

> 1 代表可自給自足；< 1 代表長期靠外部資金過活

### 債務保障比率（DCR）

營運現金流量 / (短期負債 + 長期負債)

比率越高，償債能力越強。

### 現金再投資比率

(營運現金流量 – 現金股利) / (固定資產 + 長期投資 + 其他資產 + 營運資金)

7~11%較理想，成長期偏低、成熟期逐步提升。

## 注意事項

- **單一指標不定生死**：分析現金流量要綜合評估，結合產業特性與公司發展階段
- **FCF為負不一定壞**：高速成長期的科技公司負FCF可能是投資未來，關鍵是CFO是否成長
- **CFO為正不是萬靈丹**：還需看現金如何運用，投資不相關業務或償還高利債都不健康
- **同業比較才有意義**：不同產業現金流特徵完全不同，不能跨業套用同一標準
- **間接法的陷阱**：折舊攤銷加回不代表公司有等額現金可用，要看實際現金變化
- **季節性波動**：營建業、零售業現金流有明顯季節性，需看年度而非單季
- **合並vs個體報表**：合併現金流量表包含子公司，與母公司個體報表可能差異很大
