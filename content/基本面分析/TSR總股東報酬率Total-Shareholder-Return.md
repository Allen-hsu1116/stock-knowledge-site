# TSR總股東報酬率Total-Shareholder-Return

> 衡量股東真實總報酬的黃金指標，同時納入資本利得與股利收益，是證交所「提升企業價值計畫2.0」新增的核心評估指標

## 核心概念

TSR（Total Shareholder Return，總股東報酬率）衡量投資者在特定期間內從持有某公司股票所獲得的「總體報酬」，同時納入資本利得（股價上漲）與股利收益兩部分。

與單看股價漲跌幅或僅看股利收益率不同，TSR完整呈現股東在持有期間所獲得的每一分報酬，因此特別適合作為長期投資績效的參考指標。

### TSR計算公式

**基本公式**：
TSR = [期末股價 × (1+配股率) - 期初股價 + 期間總現金股利] / 期初股價

**實務公式（使用除權息調整後股價）**：
TSR = (期末除權息調整後股價 - 期初除權息調整後股價) / 期初除權息調整後股價

後向調整（Backward-Adjusted Price）：以最新股價為錨點，由後往前回推，當公司配發股票或股利時，將歷史股價按比例調整，消除除權息缺口。

### Morgan Stanley拆解公式

**TSR ≈ EPS成長率 + PER變動率 + 現金股利殖利率**

三個驅動因子：
- **EPS成長率**：企業獲利成長的基本面驅動
- **PER變動率**：市場評價（估值）的變化
- **現金股利殖利率**：股利回饋的直接貢獻

案例拆解：若公司2025年TSR = -1%
- EPS成長率：5.5%（基本面穩健）
- PER變動率：-13%（市場評價壓縮）
- 現金股利殖利率：6.5%（穩定股利）

→ 基本面依然穩健，但市場評價壓縮導致TSR為負。

## 實戰應用

### 1. 股東報酬績效評估

衡量特定期間（一年、三年、五年）的總股東報酬，並與大盤指數或同業公司群組比較。

例：一家公司三年TSR為150%，同業平均為80%，則可視為在該期間為股東創造了相對優異的價值。

### 2. TSR驅動因子分析

將TSR拆解為「股價成長貢獻」與「股利貢獻」：
- 股價成長：(期末股價 - 期初股價) / 期初股價
- 股利貢獻：期間總股利 / 期初股價

不同產業的TSR驅動來源：
- **成熟產業**（電信、民生工業）→ 以股利為主要回報來源
- **高成長科技股**（AI供應鏈）→ 以股價上升為主

### 3. 跨公司比較與投資組合管理

將TSR與ROIC、成長率、波動率等指標並列，建構多維度標的評估矩陣，識別兼具「價值創造」與「穩定回報」的優質企業。

### 4. 累積總股東報酬率（Cumulative TSR）

衡量投資人在特定持有期間內結合股價增值與股利再投資後的總財富成長。使用除權息調整後股價推算，有效消除除權息與股票分割造成的股價不連續性，讓不同規模與股利政策的公司可標準化比較。

### 5. TSR與其他指標的關聯

企業價值創造的完整鏈條：
ROIC > WACC → 企業創造價值 → 自由現金流增加 → 市場評價提高（PER上升）+ 股利增加 → TSR提升

## 注意事項

- TSR是回顧性指標，過去TSR不代表未來表現
- TSR不適用於部門層級或私人公司
- 單一年度TSR可能受市場情緒影響，應看3-5年長期趨勢
- 高TSR可能來自估值膨脹（PER變動）而非基本面改善
- 股利再投資的假設會影響累積TSR計算結果
- 除權息調整後股價的正確性是TSR計算的基礎
