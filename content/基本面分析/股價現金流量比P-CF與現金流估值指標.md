# 股價現金流量比P-CF與現金流估值指標

> 用現金流代替盈餘來估值——P/CF比本益比更難被會計手法扭曲，是看穿「紙面獲利」的利器

## 核心概念

### 股價現金流量比（P/CF）是什麼

股價現金流量比（Price to Cash Flow Ratio，簡稱P/CF）衡量股票價格相對於每股營業現金流的價值。

**公式：**
- P/CF = 市值 ÷ 最近一年營業現金流
- P/CF = 股價 ÷ 每股營業現金流

**核心邏輯**：比起本益比使用稅後淨利（容易受折舊、攤銷等非現金支出影響），P/CF使用營業現金流，反映公司「真正收到口袋的錢」。

### 現金流計算方式差異

| 計算基礎 | 公式 | 特點 |
|----------|------|------|
| 營業現金流 | 股價 ÷ 每股營業現金流 | 最常用，標準算法 |
| 淨利+折舊攤銷 | (淨利+折舊+攤銷) ÷ 股數 | 營業現金流的簡化算法 |
| 自由現金流 | 股價 ÷ 每股自由現金流 | 最嚴格，扣除CapEx後的現金 |
| 三大現金流合計 | 營業+融資+投資現金流 | 較少使用 |

⚠️ 比較不同公司時，**必須使用一致的現金流定義**。

### P/CF vs 本益比（P/E）

| 比較項目 | P/E本益比 | P/CF現金流量比 |
|----------|-----------|---------------|
| 分母 | 稅後淨利/EPS | 營業現金流 |
| 受會計影響 | 高（折舊、攤銷、應收帳款） | 低（實際現金流入流出） |
| 適用場景 | 獲利穩定的成熟公司 | 高資本支出、高折舊公司 |
| 造假難度 | 較易（盈餘管理） | 較難（現金流難造假） |
| 缺點 | 虧損公司無法計算 | 現金流波動大時不穩定 |

**P/CF最適合的場景**：
- 成長型公司（大量再投資，EPS極低或為負）
- 高資本支出產業（半導體、面板、通訊）
- 折舊攤銷占比高的公司
- 需要驗證盈餘品質時

## 實戰應用

### P/CF選股策略

1. **同業比較法**：P/CF越低越可能被低估，但必須同產業比較
2. **歷史區間法**：觀察個股P/CF的歷史高低點，判斷目前相對位置
3. **P/CF + P/E雙重確認**：P/CF低但P/E高 → 可能有大量非現金支出；P/E低但P/CF高 → 可能有盈餘品質問題

### 範例計算

A公司：股價$25，每股現金流$5 → P/CF = 5
B公司：股價$25，每股現金流$10 → P/CF = 2.5

其他條件相同，B公司P/CF較低，可能被低估。

### 自由現金流收益率（FCF Yield）

自由現金流收益率是P/CF的倒數概念，用自由現金流而非營業現金流：

**FCF Yield = 每股自由現金流 ÷ 股價**

或以公司整體計算：

**FCF Yield = 自由現金流 ÷ 企業價值（EV）** → 即EV/FCF的倒數

台灣碩士論文實證（2018，政大）：
- FCFF Yield可有效預測未來個股報酬
- 零投資組合平均月報酬0.92%（年化約11.6%）
- 通過Fama-MacBeth檢定與穩定性測試
- FCFF Yield能捕捉系統風險與三因子模型以外的資訊

### FCF Margin（自由現金流利潤率）

FCF Margin = 自由現金流 ÷ 營收

判斷標準：
- **5%以上**：值得留意
- **10-15%**：相當優秀
- **逐年穩步成長**：具備大幅成長潛力
- **同業比較異常下降**：財務狀況惡化警訊

### 現金流覆蓋比率（Cash Flow Coverage Ratio）

**公式**：營業現金流淨額 ÷ 總負債

這是衡量償債能力的終極指標，比用淨利更可靠：

| 比率 | 判斷 |
|------|------|
| < 0.2 | 危險——嚴重依賴債務 |
| 0.2-0.5 | 適當——可管理 |
| > 0.5 | 健康——現金流輕鬆覆蓋債務 |
| ≥ 1.0 | 極強——一年可還清全部負債 |

**與DSCR的區別**：
- DSCR = 息稅前利潤 ÷ 年度債務支付（利息+本金）
- 現金流覆蓋比率 = 營業現金流 ÷ 總負債
- 現金流覆蓋比率更全面，看的是整個債務負擔而非單年支付

**實戰應用**：
1. 過濾價值陷阱：低P/E + 低現金流覆蓋比率 = 價值陷阱
2. 看3-5年趨勢：現金流成長快於負債 = 信用改善故事
3. 更保守版本：從營業現金流扣除CapEx再除以總負債

**產業差異**：
- 資本密集產業（公用事業、製造、電信）：比率偏低可接受，因負債分攤30-40年且現金流可預測
- 輕資產產業（軟體、顧問）：應有更高比率，Big Tech比率0.3是比水務公司0.3更大的紅旗

## 注意事項

### P/CF的陷阱

1. **現金流波動大**：單一年度的現金流可能受一次性事件影響（出售資產、大額收款），建議看3-5年平均
2. **產業差異巨大**：不同產業的P/CF不可直接比較，必須同業對比
3. **負現金流無法計算**：現金流為負的公司P/CF無意義
4. **資本支出未扣除**：營業現金流版本的P/CF未考慮維持營運必要的資本支出，自由現金流版本更準確

### 現金流覆蓋比率的陷阱

1. **負比率**：營業現金流為負時比率為負，早期新創公司常見但成熟公司是高危訊號
2. **行業特性**：不能跨行業直接比較
3. **高比率不等於高成長**：高比率證明安全但不保證成長
4. **季節性波動**：零售業Q4現金流大增，Q1驟降，看全年更準確

### 「燒錢慘業」警示

有些產業資本支出巨大但無法創造對應報酬：
- 面板業、LED、太陽能、DRAM（台灣四大燒錢慘業）
- 特徵：EPS為正但FCF為負、ROE逐年下降、資本支出遠高於折舊
- 這類公司P/CF可能偏低，但不是被低估而是結構性問題
