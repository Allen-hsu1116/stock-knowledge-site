# 存貨週轉率進階判讀與存貨陷阱識別

> 存貨週轉率只是起點，真正要判斷的是：存貨是黃金還是垃圾？是策略性備貨還是滯銷堆積？搭配報價趨勢、存貨結構、毛利率三重確認才能避開陷阱

## 核心概念

存貨週轉率的基礎判讀（越高越快越好）只是第一層，進階判讀需要回答三個問題：

1. **存貨是黃金還是垃圾？** → 報價趨勢向上 = 黃金，報價趨勢向下 = 垃圾
2. **存貨結構健康嗎？** → 原物料增加 vs 成品暴增，意義完全不同
3. **週轉率變化的真正原因是什麼？** → 策略性備貨 vs 滯銷堆積，需要交叉驗證

### 存貨價值的會計本質

IFRS 制度下，存貨依「成本與淨變現價值孰低」評價：
- **跌價損失**認列在營業成本中，直接壓低毛利率
- **回升利益**可迴轉（IFRS 允許），但回升幅度不得超過原跌價金額

這意味著：**存貨跌價 → 營業成本增加 → 毛利率下降**，三者有連動關係。

## 實戰應用

### 存貨五維進階判讀框架

| 維度 | 看什麼 | 判斷方法 |
|------|--------|---------|
| 1. 報價趨勢 | 存貨所屬產品的報價走勢 | 向上 = 黃金庫存；向下 = 垃圾庫存 |
| 2. 存貨結構 | 成品 vs 在製品 vs 原物料比例變化 | 成品暴增 = 滯銷風險；原物料增加 = 備貨或看漲 |
| 3. 毛利率連動 | 存貨跌價損失是否壓低毛利率 | 毛利率驟降 + 存貨增加 = 雙重警訊 |
| 4. 同業比較 | 同產業週轉率排名 | 偏離同業平均值太多 = 異常訊號 |
| 5. 營業現金流 | 獲利是否轉為現金 | 淨利增 + 營業現金流負 = 紙面富貴，存貨可能虛增 |

### 四種存貨陷阱

**陷阱一：降價清倉的假週轉率**
- 現象：存貨週轉率突然飆高，但毛利率同步下降
- 本質：公司打骨折賣庫存，看似「賣得快」，實則虧錢
- 對策：**週轉率 + 毛利率必須一起看**，週轉率上升但毛利率下降 = 紅旗

**陷阱二：DIO 驟降的假改善**
- 現象：DIO 從 50 天驟降到 20 天
- 本質：公司預期景氣變差，主動降低備貨量，存貨基數變小
- 對策：搭配營收成長率，如果營收也下降 = 保守應對，不是效率改善

**陷阱三：成品暴增的結構性風險**
- 現象：存貨總額上升，但全都是成品增加
- 本質：原物料 → 成品的轉換正常，但成品賣不出去 = 終端需求出問題
- 對策：查看存貨明細的組成變化，成品佔比超過 70% 要警惕

**陷阱四：原物料備貨的假滯銷**
- 現象：存貨週轉率下降，看起來像滯銷
- 本質：公司預期原物料漲價，提前囤貨（如群聯囤 NAND Flash 低價庫存）
- 對策：**看原物料報價走勢**，報價向上 + 存貨中原物料增加 = 策略性備貨，偏利多

### 群聯（8299）實戰案例

群聯是 NAND Flash 控制器IC 設計公司，其存貨特性：
- 當 NAND Flash 報價低谷時，群聯趁低價大量備貨
- 後續 NAND 報價逐季上漲，低價庫存成為毛利率改善的動力
- 同時華邦（2344）、威剛（3260）也有類似效益
- **關鍵判斷**：存貨週轉率暫時下降是「備貨策略」而非「滯銷風險」，需看報價趨勢確認

### 不適合用存貨週轉率的產業

| 產業 | 原因 | 替代指標 |
|------|------|---------|
| 金融業 | 存貨就是錢，週轉率無意義 | 放款周轉率、利息覆蓋率 |
| 營建業 | 土地即存貨，周轉天數天生長 | 推案去化率、合約負債轉收入 |
| 純服務業/SaaS | 幾乎無實體存貨 | 客戶成長率、續約率、MRR |
| 文創/精品 | 存貨有時效性，過季即貶值 | 庫齡分析、存貨跌價準備率 |

## 注意事項

1. **永遠搭配毛利率看**：週轉率上升 + 毛利率下降 = 降價清倉，不是真改善
2. **存貨結構比總量重要**：成品暴增才是真風險，原物料增加可能是策略性備貨
3. **營業現金流是最終驗證**：獲利增加但營業現金流為負 = 紙面富貴，存貨可能是虛的
4. **用年資料或四季滾動**：避免季節性扭曲，至少看 4 季平均
5. **原物料報價是關鍵變數**：存貨是黃金還是垃圾，取決於報價趨勢
6. **降價促銷 vs 需求暢旺**：前者毛利率下降，後者毛利率持平或上升，這是核心區別
