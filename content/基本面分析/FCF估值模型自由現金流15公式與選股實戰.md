# FCF估值模型自由現金流15公式與選股實戰

> 自由現金流是企業支付所有必要支出後真正能自由運用的現金——巴菲特也偏愛「產生現金」而非「消化現金」的公司。

## 核心概念

### 自由現金流（FCF）是什麼

自由現金流（Free Cash Flow, FCF）代表企業在支付營運費用和資本支出後，真正能留給投資人的「淨現金」。比起EPS或淨利，FCF更貼近企業的真實現金狀況：

- EPS包含非現金項目（折舊、攤銷），不代表真正收到的現金
- 應收帳款增加→帳面有盈餘但現金沒進來
- FCF只看實際的現金流入與流出，不受會計調整影響

### FCFF vs FCFE

| 類型 | 歸屬 | 比喻 |
|------|------|------|
| FCFF（公司自由現金流） | 債權人+股東共有 | 整塊還沒分的蛋糕 |
| FCFE（股權自由現金流） | 僅屬於股東 | 分給債主後剩下的蛋糕 |

### 15個核心公式

**一、基礎估值與WACC**

1. Firm Value = Σ FCFF_t / (1 + WACC)^t
2. Equity Value = Σ FCFE_t / (1 + r)^t
3. WACC = [MV(Debt)/總資本] × rd × (1-T) + [MV(Equity)/總資本] × r
4. Equity Value = Firm Value - Market Value of Debt

**二、固定增長模型**

5. FCFF_t = FCFF_{t-1} × (1 + g)
6. Firm Value = FCFF_1 / (WACC - g)（高登模型變體）

**三、從財報推導現金流**

7. FCFF = NI + NCC + Int(1-T) - FCInv - WCInv
8. FCFF = CFO + Int(1-T) - FCInv
9. FCFE = FCFF - Int(1-T) + Net borrowing
10. FCFE = NI + NCC - FCInv - WCInv + Net borrowing
11. FCFE = CFO - FCInv + Net borrowing

**四、稅前項目與預測應用**

12. FCFF = EBIT(1-T) + Dep - FCInv - WCInv
13. FCFF = EBITDA(1-T) + Dep×T - FCInv - WCInv
14. FCFE = NI - (FCInv - Dep) - WCInv + Net borrowing
15. FCFE = NI - (1-DR)(FCInv-Dep) - (1-DR)(WCInv)

**名詞**：NCC=非現金費用、FCInv=資本支出、WCInv=營運資金變動、DR=目標負債比率

## 實戰應用

### 三種FCF計算方式（從簡到深）

1. **現金流量表法**（最常用）：FCF = 營業現金流 - 資本支出
2. **損益表+資產負債表法**：FCF = EBIT×(1-稅率) + 折舊攤銷 - 營運資金變動 - 資本支出
3. **三大報表綜合法**：FCF = 淨利 + 利息×(1-稅率) + 折舊攤銷 - 營運資金變動 - 資本支出

### FCF選股兩項準則（夏綠蒂選股法）

1. **數據以「年」計算**——不以季報金額計算（擴廠需時間，季報波動大）
2. **近5年FCF中3年為正，且總額為正**——給2年醞釀期，之後必須帶回收益

### FCF正負判讀

| FCF狀態 | 好的情境 | 壞的情境 |
|---------|---------|---------|
| 正數 | 營業現金流入>投資支出，健康 | 營業現金流虧損但變賣資產湊正數 |
| 負數 | 擴廠投資，短期燒錢長期回報 | 長期為負，持續燒錢靠借貸維生 |

### EPS成長但FCF下降的四大危險信號

1. **瘋狂投資擴張**：CapEx大增→看FCF+投資活動現金流是否仍在成長
2. **庫存堆積**：現金卡在存貨→看存貨vs銷量是否脫鉤
3. **應收帳款問題**：客戶先買後付→看應收帳款是否異常增加
4. **供應商要求更快付款**：現金流出加速→看應付帳款變化

### FCF估值進階用法

- **不配息公司**：無法用DDM，必須用FCF折現模型
- **收購評估**：FCF是評估整家公司價值的基礎
- **收益品質驗證**：FCF vs 淨利差距大→會計操縱風險
- **高登模型**：穩定成長公司，Firm Value = FCFF_1 / (WACC - g)

## 注意事項

- 資本支出波動大，單一年度FCF可能失真→用5年趨勢判斷
- 成長型公司FCF常為負不代表不好→要區分「投資型負數」vs「燒錢型負數」
- FCF為正但來自變賣資產→非長遠之計，要檢查營業現金流是否健康
- 產業特性差異大→跨產業比較FCF需謹慎，同產業內比較較合理
- FCFF用WACC折現、FCFE用股權成本折現，不要混用
- 高登模型只適用穩定成長公司，g必須<WACC
- 台灣財報不常用「資本支出」一詞→用投資活動現金流替代
