# Altman Z-Score 破產預測模型

> 用五個財務比率加權計算出單一分數，量化公司在兩年內破產的機率。

## 核心概念

Altman Z-Score 由紐約大學 Edward Altman 教授於 1968 年提出，是全球最知名的破產預測模型。它透過**多重區別分析（MDA）**，將五個財務比率以不同權重組合成一個綜合分數，用來判斷公司是否可能在未来兩年內破產。

原始模型針對美國公開上市製造業公司設計，後來陸續推出適用於私有企業（Z'-Score）、非製造業（Z"-Score）和新興市場的變體版本。

### 為什麼 Z-Score 重要？

- 破產前一年預測準確率高達 **95%**
- 破產前兩年預測準確率達 **83%**
- 單一分數就能快速判斷公司財務健康，是信用分析、投資篩選的標準工具

## 三種公式版本

### 1. 原始 Z-Score（公開上市製造業）

Z = 1.2 × X₁ + 1.4 × X₂ + 3.3 × X₃ + 0.6 × X₄ + 1.0 × X₅

| 變數 | 公式 | 衡量維度 |
|------|------|----------|
| X₁ | 營運資金 / 總資產 | 流動性 |
| X₂ | 保留盈餘 / 總資產 | 累積獲利能力 |
| X₃ | EBIT / 總資產 | 營業獲利能力 |
| X₄ | 股本市值 / 總負債 | 槓桿與市場信心 |
| X₅ | 營收 / 總資產 | 資產使用效率 |

### 2. Z'-Score（私有製造業）

Z' = 0.717 × X₁ + 0.847 × X₂ + 3.107 × X₃ + 0.420 × X₄ + 0.998 × X₅

差異：X₄ 改用**帳面淨值**（Book Value of Equity）而非市值。

### 3. Z"-Score（非製造業 / 新興市場）

Z" = 6.56 × X₁ + 3.26 × X₂ + 6.72 × X₃ + 1.05 × X₄

差異：**移除 X₅（營收/總資產）**，因為資產週轉率在不同產業差異太大。新興市場版本加上常數項 3.25。

## 五個變數詳細判讀

### X₁：營運資金 / 總資產（WC/TA）

- **高值**：流動性好，短期償債能力強。但過高可能表示存貨堆積或現金未有效運用
- **低值/負值**：短期財務壓力大，但高存貨周轉率的企業（如量販店）負營運資金是正常現象
- Altman 發現此比率在三個流動性指標中最具鑑別力，優於流動比率與速動比率

### X₂：保留盈餘 / 總資產（RE/TA）

- **高值**：歷史獲利能力強，靠自身盈餘而非借款支撐資產
- **低值**：依賴外部融資，年輕公司天然偏低（尚未累積盈餘）
- **陷阱**：準改組（quasi-reorganization）和股票股利宣告會扭曲此比率
- Dun & Bradstreet 研究顯示 50% 的破產企業成立不到五年

### X₃：EBIT / 總資產（EBIT/TA）

- **高值**：資產生產力強，獲利能力佳
- **低值**：資產效率低，可能陷入財務困境
- 類似 ROA 但用 EBIT 而非淨利，排除稅負與槓桿影響
- Altman 發現此比率**持續優於現金流量等其他獲利指標**

### X₄：股本市值 / 總負債（MVE/TL）

- **高值**：市場信心強，資產價值可大幅下跌仍不致破產
- **低值**：市場不看好，市值稍跌就可能資不抵債
- **缺點**：股價波動會影響此比率，高股價可能虛增分數
- Altman 發現此比率比傳統的淨值/總負債（帳面值）更具預測力

### X₅：營收 / 總資產（S/TA）

- **高值**：資產使用效率高，管理能力強
- **低值**：維持市占率困難，需更多投資才能成長
- 單獨看最不顯著，但在模型中貢獻度排第二（因與其他變數的獨特關係）
- 不同產業差異極大，因此非製造業版本移除了此變數

## 判讀區間

### 公開上市製造業

| Z-Score | 區間 | 判讀 |
|---------|------|------|
| > 2.99 | 安全區 | 破產機率極低 |
| 1.81 – 2.99 | 灰色地帶 | 需密切監控 |
| < 1.81 | 危險區 | 破產機率高 |

### 私有非製造業

| Z-Score | 區間 | 判讀 |
|---------|------|------|
| > 2.60 | 安全區 | 破產機率極低 |
| 1.10 – 2.60 | 灰色地帶 | 需密切監控 |
| < 1.10 | 危險區 | 破產機率高 |

## 實戰應用

### 1. 投資篩選：排除破產風險股

在選股流程中加入 Z-Score 過濾：
- Z < 1.81 的公司直接剔除，或列為觀察名單
- 處於灰色地帶的公司需更深入分析籌碼面與產業趨勢
- **趨勢比單一數值更重要**：Z-Score 持續惡化比單次低分更危險

### 2. 信用風險評估

銀行和供應商可使用 Z-Score 作為授信參考：
- 安全區客戶可給予較寬鬆的信用條件
- 灰色地帶客戶需加強監控
- 危險區客戶應收緊信用額度或要求擔保

### 3. 持倉風險監控

投資組合中定期計算持股的 Z-Score：
- 每季計算一次，追蹤趨勢變化
- Z-Score 從安全區跌入灰色地帶 → 減碼
- Z-Score 跌入危險區 → 考慮出清

### 4. 搭配 Beneish M-Score 交叉驗證

Z-Score 偏低可能是財報操縱的結果：
- 如果 Z-Score 低 + M-Score 高 → 可能是假帳造成，真實狀況更差
- 如果 Z-Score 低 + M-Score 正常 → 真實的財務困境

### 5. 計算實例

某製造業公司財務數據（百萬）：
- 流動資產 60、流動負債 40、固定資產 100
- 淨利 10、股利 2、營收 60、營業費用 40
- 本益比 8x、總負債 120

計算：
- X₁ = (60-40) / (60+100) = 0.13
- X₂ = (10-2) / 160 = 0.05
- X₃ = (60-40) / 160 = 0.13
- X₄ = (8×10) / 120 = 0.67
- X₅ = 60 / 160 = 0.38

Z = 1.2×0.13 + 1.4×0.05 + 3.3×0.13 + 0.6×0.67 + 1.0×0.38 = **1.40**

→ 危險區，破產風險高。

## 注意事項

### 六大限制

1. **產業適用性有限**：金融業、新創公司、資本結構特殊的公司不適用原始模型
2. **歷史資料落後**：基於會計數據，可能無法及時反映急速惡化
3. **會計操縱風險**：盈餘管理或一次性交易可能扭曲分數
4. **負營運資金誤判**：餐飲業等高周轉率產業，負營運資金反而是健康的
5. **年輕公司偏見**：保留盈餘低不代表破產風險高，只是公司年輕
6. **高股價虛增分數**：X₄ 用市值，泡沫股可能得到偏高的 Z-Score

### 正確使用原則

- **作為早期預警，不是最終判決**：必須搭配其他指標和質性分析
- **追蹤趨勢，而非只看單一時點**：持續惡化的 Z-Score 比單次低分更危險
- **選對模型版本**：根據公司類型（公開/私有、製造/非製造）選擇對應公式
- **排除無形資產**：計算總資產時建議排除無形資產
- **同業比較**：與產業中位數比較，不同產業的 Z-Score 基準不同

### 其他破產預測模型比較

| 模型 | 特點 | 適用場景 |
|------|------|----------|
| Ohlson O-Score | 邏輯迴歸，變數更多 | 跨產業適用 |
| Springate S-Score | 簡化版 Z-Score | 特定產業 |
| Moody's KMV | 市場基礎（股價波動率） | 大型上市公司 |
| Beneish M-Score | 偵測財報舞弊 | 搭配 Z-Score 交叉驗證 |
