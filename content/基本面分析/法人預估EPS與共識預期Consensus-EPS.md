# 法人預估EPS與共識預期Consensus-EPS

> 共識EPS是市場對未來的集體預期——超預期大漲、不如預期大跌，關鍵是預期差而非絕對值

## 核心概念

### 什麼是共識EPS（Consensus EPS）

共識EPS是多家法人（外資、投信、自營商、研究機構）對同一檔股票未來每股盈餘的預估值，取**算術平均數**或**加權平均數**。

- **來源**：各家法人研究報告的EPS預估
- **彙整平台**：Bloomberg、路透社、台灣證券交易所（公開資訊觀測站）、CMoney、Goodinfo
- **更新頻率**：法人報告更新時同步更新，通常每季財報公布後大幅修正

### 共識預期的三層意義

1. **絕對值**：共識EPS本身代表市場對公司獲利的「基準預期」
2. **預期差**：實際EPS vs 共識EPS → 股價反應的真正驅動力
3. **趨勢變化**：共識EPS上修或下修的方向比絕對值更重要

### 預期差的三種情境

| 情境 | 實際 vs 共識 | 股價反應 |
|------|-------------|----------|
| 超預期（Beat） | 實際 > 共識 | 大漲（尤其連續超預期） |
| 符合預期（Meet） | 實際 ≈ 共識 | 小漲或持平（已反映） |
| 不如預期（Miss） | 實際 < 共識 | 大跌（尤其連續不如預期） |

## 實戰應用

### 如何查詢共識EPS

**免費管道**：
1. **公開資訊觀測站**（mops.twse.com.tw）：法人預估EPS彙整
2. **Goodinfo**（goodinfo.tw）：個股→法人預估
3. **CMoney**：法人預估EPS、目標價彙整
4. **Yahoo奇摩股市**：簡易版預估數據

**付費管道**：
1. **Bloomberg Terminal**：最完整共識數據
2. **路透社 Eikon**：法人預估歷史趨勢
3. **TEJ**：台股法人預估資料庫

### 共識EPS實戰判讀五步驟

**Step 1：看共識EPS絕對值**
- 判斷公司獲利水準
- 計算預估本益比：目前股價÷共識EPS

**Step 2：看共識EPS趨勢**
- 上修趨勢 → 法人越來越樂觀 → 偏多
- 下修趨勢 → 法人越來越悲觀 → 偏空
- 橫盤趨勢 → 市場看法分歧 → 觀望

**Step 3：看預估分散度（Dispersion）**
- 分散度低 → 法人看法一致 → 超預期或不如預期反應更大
- 分散度高 → 法人看法分歧 → 預期差反應較小

**Step 4：對比實際EPS**
- 財報公布後立刻對比共識
- 超預期1個標準差以上 → 強力買訊
- 不如預期1個標準差以上 → 強力賣訊

**Step 5：追蹤法人修正動向**
- 外資修正方向通常領先
- 連續3次以上上修 → 趨勢確認
- 首次下修 → 警訊，連續下修 → 賣訊

### 共識EPS與股價的四大關係

1. **本益比錨定效應**：股價圍繞共識EPS×合理PE波動
   - 共識EPS上修10% + PE不變 → 股價理論上漲10%
   - 實際漲幅取決於PE是否同步擴張或收縮

2. **預期差驅動短線**：
   - 財報公布前：市場已反映共識預期
   - 財報公布後：實際 vs 共識的差異驅動1-3日走勢
   - PEAD（盈餘公告後價格漂移）：超預期的股票後續1-3個月仍傾向上漲

3. **趨勢比數字重要**：
   - EPS預估從5元上修到7元 → 比EPS穩定在7元更偏多
   - 法人持續上修代表「持續超預期」的機率高

4. **本益比擴張/收縮**：
   - 共識EPS上修 + PE擴張 → 雙重利多，大漲
   - 共識EPS上修 + PE收縮 → 抵消，可能只小漲
   - 共識EPS下修 + PE收縮 → 雙重利空，大跌

### 預估EPS選股實戰

**條件篩選**：
- 共識EPS年增率>20%（成長股）
- 近3個月共識上修次數>下修次數
- 法人給予目標價距離>15%
- 預估本益比<同業平均

**避開的股票**：
- 共識EPS連續下修3個月以上
- 法人目標價持續下調
- 預估分散度極高（法人看不懂）
- 歷史超預期次數<30%（容易不如預期）

## 注意事項

### 共識EPS五大盲點

1. **法人群體偏誤**：法人傾向過度樂觀，尤其是有承銷利益的券商
2. **共識延遲性**：共識數據更新通常落後實際1-2週
3. **錨定效應**：法人不願大幅偏離共識，調整速度慢
4. **樣本偏差**：只有2-3家法人追蹤的小型股，共識不具代表性
5. **季節性操控**：公司可能調整認列時序讓單季超預期，但全年不變

### 使用共識EPS的正確心態

- 共識是**起點不是終點**——用它來判斷市場預期，再找出市場沒看到的東西
- 連續超預期的公司比偶爾超預期的更值得追蹤
- 當所有法人都在上修，反而要警惕「過度共識」的風險
- 共識EPS只能看方向，不能只看絕對值——要搭配趨勢和分散度

### 與本益比的搭配

共識EPS本身不決定股價，要搭配PE才完整：
- 預估PE = 當前股價 ÷ 共識EPS（未來一年）
- 歷史PE區間判斷目前PE是否偏高或偏低
- 同業PE比較判斷相對合理性
