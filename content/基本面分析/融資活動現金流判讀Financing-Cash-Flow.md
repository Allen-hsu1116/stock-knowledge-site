# 融資活動現金流判讀 Financing Cash Flow Analysis

> 看懂公司的金脈來源與去向——借錢、還錢、配息、增資，融資現金流揭穿公司怎麼找錢又怎麼花錢

## 核心概念

融資活動現金流（Cash Flow from Financing Activities, CFF）是現金流量表三大現金流之一，記錄公司與資金提供者（股東、債權人）之間的現金往來。

三大現金流定位：
- **營業現金流（OCF）**：本業賺的錢 → 最重要
- **投資現金流（ICF）**：投資花的錢 → 通常為負
- **融資現金流（CFF）**：借錢、還錢、發股利 → 看金脈來源

融資現金流只記錄**實際現金收付**，賒帳或應計項目不會出現在這裡。

### 融資現金流的組成項目

**現金流入（向外部取錢）**：
- 現金增資發行新股
- 銀行借款（短期、長期）
- 發行公司債
- 庫藏股出售

**現金流出（把錢還給外部）**：
- 發放現金股利（最常見的流出）
- 償還借款本金
- 買回庫藏股
- 少數股東取得股利

⚠️ IFRS vs 台灣實務差異：利息支出在IFRS下可歸入營業或融資，台灣多數公司歸入營業現金流，閱讀時要注意分類差異。

## 實戰應用

### 四大判讀模式

**模式一：CFF為負值（最常見的健康型態）**
- 主要因為配息和還債
- OCF穩定正數 + CFF為負（因配息） → ✅ 健康成熟公司
- 例：台積電2024年融資現金流淨流出2,000多億，主要因為發放現金股利

**模式二：CFF為正值（公司正在找錢）**
- 主動融資：有成長機會，OCF為正但仍需額外資金擴張 → ✅ 成長期公司
- 被動融資：OCF為負，靠借錢度日 → ⚠️ 危險

**模式三：CFF持續大額正數（持續舉債）**
- OCF長期為負 + CFF長期為正 → ❌ 借錢度日
- 利息支出占OCF>30% → 還款壓力大
- 資產負債比飆升 → 負債越滾越大

**模式四：CFF持續大額負數但不是配息**
- 大量償還借款（OCF能支應） → ✅ 降槓桿好事
- 大量買回庫藏股 → 看股價是否低估而定
- 發放特別股股利 → 注意特別股條款

### 三大現金流組合判讀

| OCF | ICF | CFF | 型態 | 判讀 |
|-----|-----|-----|------|------|
| + | - | - | 健康 | 本業賺錢、投資未來、配息還債 |
| + | - | + | 成長期 | 本業賺錢但投資需求大，需外部融資 |
| - | - | + | 危險 | 本業虧損、積極投資、借錢度日 |
| + | + | - | 收縮 | 本業賺錢、變賣資產、還債配息 |

### 進階判讀指標

**股利支付率（從現金流角度）**：
- 現金股利 / 營業現金流
- <30%：保守，保留現金投入成長
- 30%-70%：穩健
- >70%：高配息，可能壓縮投資
- >100%：⚠️ 借錢配息或吃老本

**FCF股利覆蓋率**：
- 自由現金流 / 現金股利
- >2倍：配息安全
- 1-2倍：尚可但需監控
- <1倍：⚠️ 不可持續

**借款淨變化**：
- 借款增加 > OCF成長 → 槓桿上升
- 借款減少 + OCF穩定 → 降槓桿好事
- 短期借款增 > 長期借款增 → ⚠️ 短債長投危險

## 注意事項

### 四大常見陷阱

1. **借錢配息**：表面配息率很高，但OCF不足以支應。辨識：CFF中借款增加≈現金股利、OCF/現金股利<1
2. **短債長投**：用短期借款支應長期投資，到期無法續借會周轉危機。辨識：流動比率<1.5、短債占比上升
3. **庫藏股護盤**：管理層宣稱「股價低估」買回，實際只是護盤。辨識：買回均價高於期間均價、買回後減資vs轉讓員工比例
4. **持續融資稀釋**：連續數年辦理現金增資，EPS持續被稀釋。辨識：增資用途模糊、增資價<每股淨值

### 五步判讀SOP

1. **看CFF正負**：正=找錢，負=還錢/配息
2. **拆解組成**：配息？還債？增資？庫藏股？各占多少？
3. **看OCF支應能力**：OCF能否支應CFF流出？
4. **追蹤趨勢**：連續3-5年CFF變化趨勢
5. **交叉驗證**：搭配[[基本面分析/負債比率與財務結構分析Debt-Ratio-and-Financial-Structure|負債比率]]、[[基本面分析/利息保障倍數實戰判讀Interest-Coverage-Ratio|利息保障倍數]]、股利支付率綜合判斷
