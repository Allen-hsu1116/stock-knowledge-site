# 投資組合理論與分散投資的局限 Portfolio Theory and Diversification Limits

> 分散投資降低風險——直到危機來臨，所有資產一起跌

## 核心概念

馬柯維茨投資組合理論（Markowitz Portfolio Theory, 1952）是現代金融學的基石，用**均值-方差模型**刻畫風險與報酬的關係，證明分散投資可以降低組合風險。但這套理論有致命假設：資產報酬服從常態分佈、相關性穩定、投資者理性——而現實中這三個假設都不成立。

### 投資組合理論核心

**1. 均值-方差模型**

投資組合的期望報酬率 = 各資產期望報酬率的加權平均
投資組合的風險（變異數）≠ 各資產風險的加權平均，而是：

```
σ²(p) = Σ wi²σ²i + 2ΣΣwiwjCov(ri,rj)
```

關鍵洞察：只要資產之間的相關係數 < 1，組合風險就低於個別資產風險的加權平均。相關係數越低，分散效果越好。

**2. 有效邊界（Efficient Frontier）**

在波動率-報酬率二維平面上，所有帕雷托最優投資組合構成的曲線。理性投資者只需在有效邊界上選擇：
- 最小方差點（MVP）：風險最低的組合
- 無風險資產加入後：資本市場線（CML）與有效邊界的切點即「市場組合」

**3. 分離定理（Tobin, 1958）**

每一種有效證券組合都是無風險資產與一種特殊風險資產的組合。投資者只需決定兩件事：
- 風險資產的比例（取決於風險偏好）
- 風險資產的組成（所有人相同）

### 分散投資降低風險的數學基礎

| 組合資產數 | 非系統性風險佔比 | 系統性風險佔比 |
|-----------|---------------|--------------|
| 1 | 100% | 0% |
| 5 | ~40% | ~60% |
| 10 | ~20% | ~80% |
| 20 | ~10% | ~90% |
| 50+ | ~5% | ~95% |

分散投資消除的是**非系統性風險**（個別公司的風險），但無法消除**系統性風險**（市場整體風險）。

### CAPM：從投資組合理論到資產定價

Sharpe（1964）、Lintner（1965）、Mossin（1966）在馬柯維茨基礎上提出資本資產定價模型（CAPM）：

```
E(Ri) = Rf + βi × [E(Rm) - Rf]
```

- β = 1：個股與大盤同漲同跌
- β < 1：個股波動小於大盤（防禦型）
- β > 1：個股波動大於大盤（攻擊型）

CAPM的結論：只有系統性風險（β）有溢價，非系統性風險沒有溢價——因為可以被分散掉。

## 實戰應用

### 分散投資七法的局限

雖然理論上分散投資降低風險，但實戰中有六大局限：

**1. 相關性在危機時趨向1**

這是最致命的局限。2008年金融海嘯、2020年COVID暴跌中，全球資產相關性趨向1，分散投資完全失效。詳見 [[相關性崩潰Correlation-Breakdown]]

**2. 過度分散降低報酬**

持有50檔股票的平均報酬趨近大盤，扣除交易成本和管理費用後可能跑輸指數。巴菲特：「分散投資是對無知的保護，如果你知道自己在做什麼，分散就沒有意義。」

**3. 隱藏相關性**

看似不相關的資產可能受同一因子驅動。2008年，房地產、MBS、銀行股、新興市場債券——看似分散，實際上都受「信用風險」這個單一因子影響。

**4. 肥尾部讓方差低估風險**

馬柯維茨用方差衡量風險，假設報酬服從常態分佈。但金融市場的報酬分佈是肥尾的——極端事件發生頻率遠高於常態分佈預測。方差嚴重低估了真正的風險。詳見 [[黑天鵝事件與尾部風險基礎Black-Swan-and-Tail-Risk-Fundamentals]]

**5. 交易成本與稅負侵蝕分散效果**

頻繁調整組合權重的再平衡操作，交易成本和稅負可能吃掉分散投資帶來的好處。

**6. 流動性風險**

危機時想賣的時候賣不掉。小盤股、新興市場債券的流動性在危機中急劇下降，再平衡操作根本無法執行。詳見 [[流動性風險Liquidity-Risk]]

### 實戰分散策略

| 策略 | 方法 | 優勢 | 局限 |
|------|------|------|------|
| 資產類別分散 | 股/債/商品/REITs | 最基礎的分散 | 危機時相關性趨1 |
| 地理分散 | 美股/歐股/新興市場 | 降低單一國家風險 | 全球化下相關性上升 |
| 因子分散 | 價值/動能/低波動/品質 | 分散不同風險溢價 | 因子也有相關性 |
| 時間分散 | 定期定額/分批進場 | 降低進場時機風險 | 不改變持有風險 |
| 策略分散 | 順勢+逆勢+套利 | 不同市況有表現 | 策略容量有限 |
| 現金分散 | 永遠保留10-30%現金 | 危機時有子彈 | 現金拖累報酬 |

### 從理論到實戰的修正

1. **不要只看歷史相關性**：使用壓力情景下的條件相關性
2. **加入真正的避險資產**：現金和短期國債在危機中提供真保護
3. **控制尾部風險**：價外賣權、做多VIX，代價是持續的「保費」
4. **動態調整**：根據市場狀態調整風險部位，而非固定比例
5. **承認模型局限**：馬柯維茨模型是地圖不是地形，不要把地圖當成地形

## 注意事項

- **方差不是好的風險度量**：方差把上漲和下跌都算風險，但投資者只怕下跌。半方差或CVaR更合理
- **20檔股票不夠分散**：學術研究顯示，要有效分散非系統性風險需要50-100檔，但超過30檔的邊際效益遞減明顯
- **分散 ≠ 絕對安全**：系統性風險無法分散，黑天鵝事件下所有資產同時下跌
- **再平衡的兩面刃**：定期再平衡維持目標權重，但可能產生交易成本和稅負
- **CAPM的β不穩定**：歷史β不代表未來β，β在市場壓力下會劇烈變化
- **單一因子模型的局限**：Fama-French三因子、五因子模型比CAPM更準確，但仍無法解釋所有異象
- **投資組合理論假設投資者理性**：行為金融學已證明投資者不理性，詳見 [[過度自信偏誤Overconfidence-Bias]]
