# 相關性崩潰 Correlation Breakdown

> 當你以為分散投資能保護你，市場告訴你：危機時所有資產一起跌——相關性在你最需要保護的時刻崩潰到1

## 核心概念

相關性崩潰（Correlation Breakdown）是指市場在壓力或危機期間，原本低相關或負相關的資產之間相關性急劇上升趨向1的現象。這不是「意外」，而是金融市場的**常態性特徵**——平時相關性穩定是假象，危機中相關性飆升才是真相。

與[[相關性風險Correlation-Risk]]不同，相關性風險是廣義概念（涵蓋隱藏相關性、非線性相關性等），而相關性崩潰特指**相關性結構在壓力下劇變**的機制、量化方法與防禦策略。

### 為什麼會崩潰？

**1. 共同流動性衝擊**
危機中所有投資者同時想賣，流動性蒸發。不論資產基本面如何，被迫賣出者按「能賣什麼就賣什麼」的原則拋售，導致所有資產同步下跌。

**2. 風險趨避統一行為**
恐慌情緒下，投資者不再區分「好資產」和「壞資產」，只分「風險資產」和「安全資產」。所有風險資產被歸為一類，相關性趨向1。

**3. 槓桿強制平倉**
當槓桿交易者被迫平倉時，他們賣出的是所有有流動性的部位，而非「想賣的」部位。這種非選擇性拋售讓所有資產同步下跌。

**4. 程式交易與演算法同步**
現代市場中超過70%的交易由演算法執行。當市場觸發特定訊號（如VIX突破某閾值），大量演算法同時執行類似策略，強化相關性趨向1的趨勢。

**5. 隱藏因子暴露**
看似不相關的資產可能受同一個隱藏因子驅動（如全球流動性、風險偏好）。平時因子波動小，相關性低；危機時因子劇烈波動，相關性突然顯現。

### 歷史案例：相關性崩潰的實證

| 事件 | 正常相關性 | 危機相關性 | 崩潰倍數 |
|------|-----------|-----------|---------|
| 2008金融海嘯 | 美股-美債 -0.3 | 美股-美債 +0.6 | 反轉 |
| 2020 COVID | S&P500-原油 +0.2 | S&P500-原油 +0.9 | 4.5x |
| 2022升息 | 美股-美債 -0.4 | 美股-美債 +0.7 | 反轉 |
| 1998 LTCM | 新興市場-已開發 +0.3 | 新興市場-已開發 +0.95 | 3.2x |

**關鍵觀察**：美股與美債的正常相關性約-0.3（股跌債漲，分散有效），但在2008和2022兩次危機中都反轉為正相關（股債同跌），分散投資在最需要的時刻完全失效。

## 實戰應用

### 1. 量化偵測相關性崩潰

**A. 滾動相關性監控**
計算60日和250日滾動相關性，當短期滾動相關性大幅超過長期值時，崩潰正在發生。

```
相關性崩潰訊號 = 60日滾動相關性 - 250日滾動相關性 > 0.3
```

**B. 條件相關性分析**
將市場分為「上漲日」和「下跌日」，分別計算相關性。如果下跌日的相關性顯著高於上漲日，代表崩潰風險存在。

```
條件相關性 = Corr(A, B | 市場下跌)
vs
正常相關性 = Corr(A, B)
```

**C. 尾部相關性（Tail Dependence）**
用Copula模型計算極端事件下的相關性。Pearson相關係數只衡量線性關係，無法捕捉尾部相關性。

```
上尾部相關性 λ_U = P(X > F_X^(-1)(0.95) | Y > F_Y^(-1)(0.95))
下尾部相關性 λ_L = P(X < F_X^(-1)(0.05) | Y < F_Y^(-1)(0.05))
```

下尾部相關性是崩潰的核心指標——當一個資產極端下跌時，另一個也極端下跌的機率。

**D. VIX作為相關性代理**
VIX飆升時，個股之間的相關性急劇上升。這是因為VIX反映了市場不確定性，不確定性越高，個股差異化越小。

實戰經驗法則：
- VIX < 15：個股分化大，選股有效
- VIX 15-25：相關性開始上升，選股效果降低
- VIX 25-35：相關性偏高，大盤方向為主
- VIX > 35：相關性崩潰，幾乎所有股票同漲同跌

### 2. 防禦相關性崩潰的五層架構

**第一層：因子多樣化（預防）**
不要只看資產類別，看因子暴露。一個「股債平衡」組合可能在以下因子上集中：
- 時間因子（都受利率影響）
- 流動性因子（都受市場流動性影響）
- 風險偏好因子（都受投資者情緒影響）

真正多樣化需要：價值因子、動能因子、防禦因子、流動性因子、低波動因子等多因子暴露。

**第二層：現金緩衝（生存）**
相關性崩潰時唯一不相關的是現金。建議保留：
- 保守型：20-30%現金
- 積極型：10-15%現金
- 極端防禦模式：50%現金（VIX > 35時）

**第三層：尾部對沖（保險）**
用深度價外賣權或VIX看漲期權作為保險。每年付出1-2%的「保費」，換取崩潰時數倍回報。

**第四層：動態降低部位（應變）**
當偵測到相關性崩潰訊號時：
1. 降低整體部位至正常的50-70%
2. 停止新增任何風險部位
3. 收緊停損至正常的一半

**第五層：壓力測試常態化（驗證）**
每季執行「相關性→1」壓力測試：
- 假設所有資產相關性=0.9
- 計算投資組合最大損失
- 確認最大損失在可承受範圍內
- 如果超過可承受範圍，降低部位

### 3. 台股相關性崩潰特徵

台股市場的相關性崩潰有特殊模式：

**A. 電子股一榮俱榮**
台股電子股佔比超過60%，平時電子股之間相關性就偏高（0.5-0.7），危機時進一步飆升到0.8-0.95。做10檔電子股的「分散」幾乎沒有保護效果。

**B. 外資同步撤出**
外資佔台股交易量30-40%，當全球風險偏好改變時，外資對所有台股同步賣超。這是台股相關性崩潰的最大驅動力。

**C. 股匯雙殺**
台幣貶值+台股下跌經常同步發生，外資在匯率和股市上雙重受創，加劇拋售力道。

**D. 櫃買指數崩潰更快**
中小型股流動性較差，崩潰時跌更深、反彈更慢。櫃買指數在大盤下跌時的相關性更高（平時0.6→危機0.95）。

**台股防禦配置建議**：
- 電子股部位不超過總部位的40%
- 至少配置15-20%在金融+傳產+高殖利率
- 保留10-15%現金部位
- VIX > 25時啟動防禦模式

### 4. 相關性崩潰的誤區

**誤區1：「我持有不同產業就夠分散了」**
錯。不同產業可能受同一因子驅動。2008年金融股、科技股、原物料股同時暴跌，因為都受「信用緊縮」和「風險偏好下降」兩個因子影響。

**誤區2：「債券永遠是避風港」**
錯。2022年聯準會激進升息，股債同時下跌。債券的保護效果取決於利率環境，在通膨驅動的危機中債券不提供保護。

**誤區3：「相關性崩潰是暫時的」**
對，但不完整。相關性崩潰可能只持續數週，但造成的損失可能需要數年才能恢復。2008年的相關性崩潰只持續了約3個月，但投資者花了5年才回到崩潰前水準。

**誤區4：「黃金一定抗崩潰」**
不一定。2020年3月COVID恐慌初期，黃金和股票同時下跌（流動性衝擊）。黃金在流動性危機中也可能被拋售。

## 注意事項

1. **相關性崩潰是常態而非異常**：歷史數據顯示，每3-5年就會發生一次顯著的相關性崩潰。這不是黑天鵝，而是白天鵝。

2. **VaR模型在崩潰中完全失靈**：VaR假設相關性穩定，但崩潰時相關性劇變。2008年許多金融機構的VaR模型嚴重低估風險，這是相關性崩潰最致命的後果之一。

3. **分散投資不是萬能藥**：分散投資在常態下有效，在崩潰中失效。這不是分散投資的錯，而是你需要**額外的保護層**（現金、尾部對沖、動態部位管理）。

4. **崩潰時不要加碼「不相關」的部位**：崩潰時所有風險資產的相關性趨向1，你以為在分散，其實在集中。

5. **回測中看不到崩潰的嚴重性**：回測用歷史相關性，低估了崩潰時的相關性。需要用壓力測試補充。
