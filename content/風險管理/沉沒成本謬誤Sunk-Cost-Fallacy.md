# 沉沒成本謬誤 Sunk Cost Fallacy

> 已經虧的錢不該影響未來決策——但你的大腦做不到，越套越深越不甘心，最後把牢底坐穿

## 核心概念

沉沒成本謬誤（Sunk Cost Fallacy），又稱協和謬誤，是指人們在做決策時，過度考慮已經投入且不可收回的成本（時間、金錢、精力），導致繼續投入資源到不值得的事情上，越陷越深。

### 關鍵原則

**理性決策只看未來成本和未來收益，不考慮已經沉沒的過去成本。** 但人類心理做不到——已經投入越多，越不甘心放棄，形成「欲罷不能」的困局。

### 為什麼沉沒成本影響交易者

1. **不甘心**：已經虧了30%，現在賣掉等於認輸，所以繼續抱
2. **損失厭惡**：賣出等於把「浮虧」變成「實虧」，心理痛苦太大
3. **自我認同**：承認買錯等於承認自己判斷失誤，傷害自尊
4. **希望偏誤**：相信「再等一下就會漲回來」，但等下去通常只會虧更多
5. **社會壓力**：跟別人說了看好這檔股票，現在賣掉面子掛不住

### 協和飛機效應

1960年代英法聯合開發協和超音速客機，中途發現成本暴增、市場不確定，但因為前期投入太大無法放棄。最後飛機做出來了，卻因為運營成本太高而血本無歸——這就是沉沒成本謬誤的命名由來。

## 實戰應用

### 在交易中的表現

- **加碼攤平**：股票從100跌到70，覺得「已經虧了30%，再加碼攤平就好」——但股價可能跌到50
- **不設停損**：到了停損點不執行，覺得「已經虧這麼多了，現在賣太可惜」
- **死抱套牢股**：套了兩年不放，理由是「已經等這麼久了」——但等待本身不會讓股票上漲
- **過度研究**：已經花了一週研究某檔股票，即使發現基本面有問題也硬買——因為不買等於浪費一週時間
- **策略不肯放棄**：開發了三個月的交易策略回測績效差，但捨不得丟掉——「我都花了這麼多時間了」

### 對抗沉沒成本謬誤的方法

1. **停損是唯一解藥**：到了停損點就執行，不找理由。停損是「認賠服輸」的具體行動
2. **零基決策**：每天重新問自己「如果我現在手上沒有這檔股票，我會買嗎？」如果答案是「不會」，就該賣
3. **分離決策與成本**：只看未來的預期收益和風險，不看已經投入多少
4. **記錄進場理由**：交易日誌記下進場理由，定期檢視——如果理由已經不存在，就該出場
5. **設定最大虧損**：在進場前就決定最多虧多少，到了就出場，事後不調整

### 跟其他偏誤的關聯

- 沉沒成本謬誤 + 損失厭惡 = 越套越深
- 沉沒成本謬誤 + 確認偏誤 = 只找支持繼續持有的理由
- 沉沒成本謬誤 + 希望偏誤 = 一定會漲回來的（不會的）

## 注意事項

- **沉沒成本不是只有金錢**：時間、精力、自尊心都是沉沒成本
- **停損不是認輸，是保命**：活下去比面子重要
- **不要因為怕沉沒成本而過早放棄**：這是另一種極端——有些策略需要時間驗證
- **區分「沉沒成本」和「長期投資」**：價值投資的耐心持有≠沉沒成本的死抱
- **團隊更容易沉沒**：組織的沉沒成本比個人更大，因為還有面子、政治、責任歸屬問題
