# VIX恐慌指數系統性風險監控

> VIX是市場恐慌的溫度計，低於15過熱、超過30恐慌、超過40可能是抄底機會。

## 核心概念

VIX（Volatility Index）由芝加哥期權交易所（CBOE）於1993年推出，衡量S&P 500指數期權的隱含波動率，反映市場對未來30天波動的預期。

- **計算基礎**：S&P 500指數選擇權的隱含波動率
- **單位**：年化百分比。VIX=15 代表未來30天預期年化波動率15%
- **與股價關係**：**負相關** — VIX升 → 股市跌，VIX降 → 股市漲
- **台股版本**：VIXTWN（臺指選擇權波動率指數），由期交所編製

### VIX數值解讀

| VIX區間 | 市場狀態 | 實戰判斷 |
|---------|----------|----------|
| < 15 | 極度樂觀 | 警惕虛假繁榮，隨時可能轉向 |
| 15-20 | 樂觀穩定 | 正常多頭環境 |
| 20-30 | 溫和擔憂 | 開始注意風險 |
| 30-40 | 恐慌 | 空頭信號，保守操作 |
| > 40 | 極度恐慌 | 可能是非理性殺盤，反彈在即 |

### VIX的領先性

- VIX具**領先性**，預測未來市場波動變化
- 與「實際波動率」不同 — 實際波動率是已發生的，VIX是預期的
- 空頭循環中VIX較大機率上升；多頭環境中VIX較大機率下降

## 實戰應用

### 系統性風險監控SOP

1. **每日查看VIX**：收盤後確認VIX水位
2. **VIX突破30**：啟動防禦模式
   - 減碼至5成以下
   - 停損設緊（原停損×0.7）
   - 禁止新開倉位
3. **VIX突破40**：極端防禦
   - 清空高槓桿部位
   - 只留核心持股
   - 準備抄底清單
4. **VIX回落至20以下**：恢復正常操作

### VIX作為部位管理工具

| VIX水位 | 建議部位 | 操作策略 |
|---------|----------|----------|
| < 15 | 80-100% | 滿倉操作，但注意過熱 |
| 15-20 | 70-80% | 正常操作 |
| 20-25 | 50-70% | 保守操作 |
| 25-30 | 30-50% | 大幅減碼 |
| 30-40 | 10-30% | 極度保守 |
| > 40 | 0-10% | 只留核心持股 |

### VIX避險策略

- 買入VIX期貨或ETF作為對沖
- 買入價外Put作為保險
- 注意：VIX相關商品有轉倉成本，長期持有會損耗

### 與其他指標搭配

- **VIX + TED利差**：雙重確認系統性風險
- **VIX + 信用利差**：信用市場收緊時VIX通常同步上升
- **VIX + 台指期外資淨部位**：外資大賣+VIX飆升 = 最危險組合

## 注意事項

- **VIX不是直接交易標的**：不能直接買賣VIX指數，只能交易VIX期貨/選擇權/ETF
- **VIX期貨型ETF有轉倉成本**：長期持有會因正價差結構而虧損
- **VIX是結果不是原因**：VIX上升是因為市場已經在跌，不要等VIX飆高才行動
- **台股對應**：VIXTWN波動通常比CBOE VIX小，但方向一致
- **均值回歸特性**：VIX長期均值約20，極端值終將回歸
