# 回測績效評估完整體系Backtest-Performance-Evaluation

> 策略好不好不是看報酬率就好——獲利性、虧損性、交易類型三大面向缺一不可，風險調整報酬指標（Sharpe/Sortino/Calmar/Profit Factor/Tail Ratio）是交叉驗證策略品質的必備工具

## 核心概念

回測績效評估是程式交易的核心基本功。很多新手只看報酬率就決定策略好壞，但這跟只看薪水不看工時一樣荒謬。完整的評估體系包含三大面向：

### 1. 獲利性評估

衡量策略「賺得多不多、效率好不好」：

| 指標 | 公式/定義 | 判讀標準 |
|------|-----------|---------|
| 淨利 | 毛利 - 毛損 | 正值是基本要求 |
| 報酬率 | 淨利 ÷ 初始資金 | 搭配年化比較 |
| 年化報酬率 | 跨年度比較基礎 | 要包含空頭年度 |
| 獲利因子(PF) | 毛利 ÷ 毛損 | >1.5 及格，>2 優秀 |
| Sharpe Ratio | (報酬率 - 無風險利率) ÷ 標準差 | >1 不錯，>1.5 優秀 |
| Sortino Ratio | (報酬率 - MAR) ÷ 下行標準差 | >2 非常優秀 |
| Buy & Hold 比較 | 策略 vs 買入持有 | 策略要打敗大盤才有意義 |

**獲利因子（Profit Factor）的重要性**：
- PF = 毛利 ÷ 毛損，每虧損1元能獲利多少
- PF > 1.5 才算及格，> 2 代表策略盈利能力強
- 但**平均單筆交易賺賠比**比 PF 更有實戰意義
- PF 1.3~2.0 是合理範圍，太高反而要懷疑過度最佳化

### 2. 虧損性評估

讓你對策略可能產生的虧損有所預期，是「定心丸」也是「警報器」：

| 指標 | 定義 | 實戰意義 |
|------|------|---------|
| MDD | 資金淨值從高峰滑落的最大幅度 | 投入資金應大於 MDD |
| 單筆最大虧損 | 所有虧損交易中最大的一筆 | 設定停損的參考 |
| 最大連續虧損筆數 | 連續虧損的最長序列 | 心理準備與資金規劃 |
| Calmar Ratio | 年化報酬 ÷ MDD | >0.5 優秀，<0.2 警告 |

**Calmar Ratio 實戰判讀**：
- Calmar > 0.5：年化報酬是 MDD 的兩倍以上，優秀
- Calmar 0.2~0.5：風險與回報平衡，合理
- Calmar < 0.2：承擔太大回撤只換來微薄報酬，需謹慎

**虧損性評估的雙重功能**：
- **定心丸**：預期內的虧損→保持信心持續執行
- **警報器**：預期外的虧損→重新檢視策略是否有問題

### 3. 交易類型評估

了解策略的「行為樣貌」：

| 指標 | 實戰意義 |
|------|---------|
| 交易次數/頻率 | 樣本數>30才有統計意義；高頻交易成本是隱形殺手 |
| 勝率 | 波段策略約40%，>70%要懷疑過度最佳化 |
| 賺賠比 | 平均獲利 ÷ 平均虧損，搭配勝率算期望值 |
| 平均持倉時間 | 影響資金使用效率 |
| 多空績效分開看 | 多空走勢本質不同，應考慮獨立發展 |

**勝率與報酬率的權衡**：
- 套利策略：高勝率低報酬率（系統性風險已被抵銷）
- 趨勢跟隨策略：低勝率高報酬率（趨勢不易捕捉）
- 展望理論解釋：多數人偏好高勝率（損失規避），但客觀上高報酬率策略可能更好

## 風險調整報酬指標進階

### Sharpe Ratio（夏普比率）

最基礎的風險調整報酬指標：
- 公式：(報酬率 - 無風險利率) ÷ 總標準差
- 看總波動（含上行+下行）
- **適用場景**：低波動投資組合、一般資產配置

### Sortino Ratio（索提諾比率）

夏普比率的改良版：
- 公式：(平均報酬 - MAR) ÷ 下行標準差
- 只看負報酬的波動，正報酬的波動視為加分項
- **適用場景**：高波動組合、盈虧不對稱策略
  - 勝率低、大賺小賠的策略
  - 勝率高、小賺大賠的策略

**判讀標準**：
- >2：非常優秀
- 1~2：良好
- <1：風險過高或回報不足

### Calmar Ratio（卡爾馬比率）

聚焦極端回撤風險：
- 公式：年化報酬率 ÷ MDD
- 只看最壞情況下的表現
- **優點**：比 Sharpe 更能反映「最壞能多慘」
- Calmar > Sortino > Sharpe（從嚴格程度排列）
- **適用場景**：重視資本保護、關注極端風險

### Tail Ratio（尾部比率）

評估極端情境的韌性：
- 公式：95%分位數 ÷ 5%分位數
- 衡量報酬分布兩端的比值
- **判讀**：
  - >1：極端上行情況大於極端下行情況，偏向有利
  - ≈1：上行和下行風險平衡
  - <1：極端下行情況更嚴重，風險較高

### Profit Factor（盈利因子）

最直觀的策略盈利能力指標：
- 公式：總盈利交易金額 ÷ 總虧損交易金額
- **判讀標準**：
  - >2：策略盈利能力非常強
  - 1.5~2：穩健策略
  - 1~1.5：具盈利潛力但需優化風險控制
  - <1：策略虧損，不具投資價值
- PF過高（>3）反而要懷疑過度最佳化

## 指標交叉驗證

單一指標無法完整評估策略品質，必須交叉驗證：

| 組合 | 交叉驗證目的 |
|------|-------------|
| Sharpe + Sortino | 區分總波動 vs 下行風險的影響 |
| Sortino + Calmar | 下行風險 vs 極端回撤的差異 |
| Profit Factor + 賺賠比 | PF高但賺賠比低=靠勝率堆疊 |
| MDD + Calmar | 回撤絕對值 vs 風險效率 |
| Tail Ratio + MDD | 尾部風險 vs 最大回撤的相互印證 |

**五指標體檢表**：
1. **Sharpe > 1**：策略整體風險效率可接受
2. **Sortino > 2**：下行風險控制優秀
3. **Calmar > 0.5**：極端回撤下仍有效
4. **Profit Factor > 1.5**：每虧1元至少賺1.5元
5. **Tail Ratio > 1**：極端好行情大於極端壞行情

## 回測常見陷阱

### 1. 預視資料（Look-ahead Bias）

在交易訊號中使用了當時尚無法取得的資料：
- 當日最低價當買價（收盤才知）
- 月初用上月營收（10號才公布）
- 盤中用三大法人買超（3點後才公布）

### 2. 交易成本忽視

手續費+證交稅+滑價，高頻策略可能被吃掉7%以上淨利。

### 3. 胃納量不足

大資金策略操作流動性不足的標的，實際成交價偏離回測價。

### 4. 過度最佳化

- 權益曲線過於完美（「螢光毛毛蟲」）
- 勝率>70%要警覺
- 用 WFO前向優化 + 參數高原選擇 防範

### 5. 樣本不足

- 交易次數<30次缺乏統計嚴謹度
- 樣本期間要包含至少一個完整空頭循環

## 實戰應用

### 策略選擇決策框架

1. **先用 Buy & Hold 當基準**：策略打不贏大盤就沒意義
2. **五指標體檢**：Sharpe、Sortino、Calmar、PF、Tail Ratio 全過
3. **看權益曲線**：是否平穩上升，還是靠少數幾筆大賺
4. **多空績效分開看**：多空走勢本質不同，分開研究
5. **交易成本壓力測試**：成本提高20%後策略是否仍有效

### 策略類型與適用指標

| 策略類型 | 核心指標 | 輔助指標 |
|---------|---------|---------|
| 趨勢追蹤 | Sortino + Calmar | Tail Ratio |
| 均值回歸 | Sharpe + Profit Factor | 勝率 |
| 當沖策略 | Profit Factor + 勝率 | 滑價影響 |
| 套利策略 | Sharpe + 勝率 | 胃納量 |

## 注意事項

1. **數據期間要夠長**：至少包含一個完整空頭循環，多頭年度的數據會讓指標失真
2. **流動性假象**：低流動性資產看起來波動小，但實際上是交易不頻繁，會讓 Sharpe 和 Sortino 虛高
3. **歷史數據是後照鏡**：指標高不代表未來一定好，只代表過去的風險報酬效率較佳
4. **不要過度比較小數點差異**：0.75 vs 0.80 的 Sharpe 在統計上不顯著
5. **多空分開看**：多單和空單績效差異可能很大，應獨立評估
6. **Rolling Test**：取不同時間段計算指標，如果各段差異極大，代表策略穩定性差
