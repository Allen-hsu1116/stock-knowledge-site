# 選擇權Greeks風險判讀

> Greeks是選擇權的X光機——Delta看方向、Gamma看加速度、Theta看時間衰變、Vega看波動率，搞懂四個維度才知道你到底在承擔什麼風險

## 核心概念

**Greeks** 是選擇權定價模型（Black-Scholes）中，衡量選擇權價格對各因素敏感度的指標。每個Greek代表一個風險維度：

| Greek | 衡量什麼 | 買方 | 賣方 |
|-------|---------|------|------|
| **Delta（δ）** | 方向風險 | 方向一致 | 方向相反 |
| **Gamma（γ）** | 加速度風險 | 正（有利） | 負（不利） |
| **Theta（θ）** | 時間衰變 | 負（不利） | 正（有利） |
| **Vega（ν）** | 波動率風險 | 正（賭IV升） | 負（賭IV降） |

### Delta — 方向風險

標的價格每變動1元，選擇權價格變動多少。

- 買權Delta：0到+1（平值≈0.5，深價內→1，深價外→0）
- 賣權Delta：-1到0（平值≈-0.5，深價內→-1，深價外→0）

**三層實戰意義**：

1. **部位等同性**：Delta=0.5的買權，相當於持有0.5張股票的方向暴露
2. **機率近似值**：Delta≈選擇權到期進入價內的機率。Delta=0.3約有30%機率
3. **避險比例**：持有1張股票，買Delta=-1的賣權可完全對沖

**Delta中性策略**：讓組合總Delta=0，不賺方向只賺波動率或時間價值。

### Gamma — 加速度風險

標的價格每變動1元，Delta變動多少。Gamma是Delta的導數。

- **買方有正Gamma**：價格往有利方向走時Delta加速變大——買方的優勢
- **賣方有負Gamma**：價格往不利方向走時虧損加速——賣方的風險

**Gamma最大的時刻**：
- 平值附近（Delta從0.5快速變化）
- 接近到期（短時間內Delta可能從0.1跳到0.9）
- 這就是[[逼券商拉抬Gamma-Squeeze兩手策略|Gamma Squeeze]]的原理——做市商被迫追價避險

### Theta — 時間衰變風險

每過一天，選擇權時間價值減少多少。對買方而言Theta是負數。

**Theta衰變曲線**（平值買權）：

| 距到期 | 每日時間價值衰變 |
|--------|----------------|
| 90天 | 緩慢，約0.3%/天 |
| 60天 | 中等，約0.5%/天 |
| 30天 | 加速，約1%/天 |
| 7天 | 急劇，約3-5%/天 |

時間價值像冰塊融化，越到最後融化越快。**買方必須在時間衰變吃掉利潤之前讓波動實現。**

**Theta與Gamma的取捨**是一體兩面：
- 買方：正Gamma（賺加速）+ 負Theta（賠時間）→ 需要波動夠大來彌補
- 賣方：負Gamma（賠加速）+ 正Theta（賺時間）→ 需要波動夠小來保住收入

核心公式：**Theta損失 ≈ 0.5 × Gamma × σ² × S²**

### Vega — 波動率風險

隱含波動率（IV）每變動1個百分點，選擇權價格變動多少。

- 買方有正Vega：IV上升時獲利
- 賣方有負Vega：IV下降時獲利
- 平值Vega最大、遠月Vega更大

**IV Crush**：財報公布、重大事件結束後IV急劇下降，即使方向對也可能虧錢。這是選擇權買方最常遇到的陷阱——看對方向但輸在波動率。

## 實戰應用

### 四象限策略選擇

| 預期 | 策略 | Delta | Vega | Theta |
|------|------|-------|------|-------|
| 看大漲 | 買Call | + | + | - |
| 看小漲 | 賣Put | + | - | + |
| 看大跌 | 買Put | - | + | - |
| 看小跌 | 賣Call | - | - | + |
| 看盤整 | [[鐵兀鷹Iron-Condor|Iron Condor]] | ≈0 | - | + |
| 看大波 | 買Straddle | ≈0 | + | - |

### 風險管理矩陣

| 風險類型 | 對應Greek | 管理方法 |
|----------|----------|---------|
| 方向風險 | Delta | Delta中性避險 |
| 加速度風險 | Gamma | 限制單一部位Gamma暴露 |
| 時間衰變 | Theta | 避免最後一週買方部位 |
| 波動率風險 | Vega | 分散到期日，控制IV暴露 |
| 利率風險 | Rho | 長期LEAPS需考慮，短線可忽略 |

### 三種Theta策略

1. **[[鐵兀鷹Iron-Condor|Iron Condor]]**：穩定Theta收入，但Gamma風險大
2. **[[時間價差Calendar-Spread|日曆價差]]**：近月Theta衰變快、遠月慢，賺時間差
3. **Covered Call（掩護性買權）**：持有股票+賣出買權，降低持倉成本但限制上漲空間

### Greeks動態管理

Greeks是瞬時指標，標的價格一動Greeks就跟著變：

1. **開盤前**：看整體Delta和Vega暴露，設定最大虧損
2. **交易中**：每30分鐘檢查Delta偏移，超過門檻就調整
3. **收盤前**：確認隔日Theta和Gamma風險，必要時調倉

## 注意事項

### 五大Greeks陷阱

1. **Greeks是瞬時指標**：只反映當前參數下的敏感度，大波動時Greeks變化劇烈，不能假設不變

2. **到期日效應**：最後一週Theta和Gamma急劇放大，短天期選擇權是雙面刃

3. **IV≠HV**：Vega衡量的是IV變動的影響，IV包含市場預期溢價，可能偏離[[隱含波動率IV與歷史波動率HV實戰判讀|歷史波動率]]很遠

4. **IV Crush是買方殺手**：財報前買選擇權，即使方向對也可能因IV下降而虧損。解法：用[[選擇權四大基本策略|價差策略]]降低Vega暴露

5. **模型風險**：Black-Scholes假設波動率恆定、報酬率常態分配，這些假設在極端市況下失效。參考[[模型風險Model Risk|模型風險]]

### 新手常犯錯誤

- **只看Delta不看其他Greeks**：方向對但Theta和Vega可能讓你虧錢
- **買最後一週的選擇權**：Theta衰變最劇烈，時間價值歸零速度超乎想像
- **賣出深價外選擇權**：看似穩定收入，但Gamma風險一旦爆發就是災難性虧損
- **忽略Gamma**：賣方部位Delta中性不代表安全，Gamma才是那顆未爆彈
