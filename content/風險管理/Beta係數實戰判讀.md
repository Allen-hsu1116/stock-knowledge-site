# Beta係數實戰判讀

> Beta是系統性風險的量化指標——β>1的股票在大盤漲時賺更多、跌時賠更多，β<1的股票波動較小但報酬也較低，選股前先看Beta才知道自己在承擔什麼風險

## 核心概念

**Beta係數（β）** 衡量個股相對於大盤的波動程度，是CAPM（資本資產定價模型）的核心參數。

公式：**β = Cov(Ri, Rm) / Var(Rm)**

Beta值的解讀：

| Beta值 | 意義 | 實例 |
|--------|------|------|
| β = 1 | 與大盤同步 | 0050、大型權值股 |
| β > 1 | 比大盤更波動 | 半導體股、成長股、小型股 |
| β < 1 | 比大盤穩定 | 公用事業、電信股、高股息股 |
| β < 0 | 與大盤反向 | 黃金、反向ETF |
| β = 0 | 與大盤無相關 | 現金、避險資產 |

### Beta來自CAPM迴歸

Ri - Rf = α + β(Rm - Rf) + ε

- **α** = 超額報酬（選股能力）
- **β** = 系統性風險暴露
- **ε** = 殘差（非系統性風險）

### 計算期間選擇

| 期間 | 適用場景 | 注意 |
|------|---------|------|
| 1年（250日） | 短期Beta，反映近期特徵 | 波動大，不穩定 |
| 2年（500日） | 平衡短期與穩定性 | 較常用 |
| 5年（1250日） | 長期結構性Beta | 最穩定，但可能落後 |

## 實戰應用

### 1. 選股時的Beta考量

- **牛市**：選β>1的股票，放大市場上漲報酬
- **熊市**：選β<1或β<0的標的，降低下檔風險
- **震盪市**：β≈0的標的，不隨大盤起舞

### 2. 投資組合的整體Beta

組合Beta = Σ（個股Beta × 個股權重）

範例：50% 0050（β=1）+ 30% 中華電（β=0.5）+ 20% 期貨空單（β=-1）
→ 組合β = 0.5×1 + 0.3×0.5 + 0.2×(-1) = **0.45**
→ 大盤漲10%，組合約漲4.5%

### 3. 調整投資組合Beta

**降低風險**：賣出高Beta股票、加入反向ETF或放空期貨、增加現金部位（β=0）

**提高報酬**：增加高Beta成長股、使用槓桿ETF、融資買入（風險最高）

### 4. Beta與停損設定

高Beta股票需要更大的停損空間：
- β=1.5的股票，大盤跌2%時可能跌3%
- 停損要考慮個股波動性（[[ATR平均真實波幅-Average-True-Range|ATR]]），不能只看百分比

### 5. 台股產業Beta特性

| 產業 | 典型Beta範圍 | 特性 |
|------|-------------|------|
| 半導體 | 1.2-1.8 | 高度景氣循環 |
| 金融 | 0.8-1.2 | 與利率週期相關 |
| 電信 | 0.4-0.7 | 防禦型，穩定現金流 |
| 營建 | 1.3-1.8 | 高Beta週期股 |
| 食品 | 0.5-0.8 | 消費必需品，低Beta |
| 生技 | 1.5-2.5 | 高波動，消息面驅動 |

### 6. DCF估值中的Beta

CAPM公式：預期報酬 = Rf + β(Rm - Rf)

Beta直接影響折現率和估值：
- β越高 → 要求報酬率越高 → 估值越低
- β越低 → 要求報酬率越低 → 估值越高
- 在[[現金流量折現法DCF估值|DCF估值]]中，Beta是計算WACC的關鍵輸入

## 注意事項

### Beta的五大陷阱

1. **後視鏡陷阱**：過去的Beta不代表未來的Beta。公司業務轉型、併購後Beta可能完全改變

2. **不對稱Beta（Downside Beta）**：牛市時β=0.8的股票，熊市時β可能變成1.3。「穩定」的股票在崩盤時可能比大盤跌更多——分別計算上漲日和下跌日的Beta

3. **R²太低無意義**：R²<0.3時個股走勢跟大盤幾乎無關，Beta沒有預測力。小型股、冷門股常見

4. **Beta不衡量非系統性風險**：β很低的公司可能非系統性風險極高（公司治理問題、單一產品風險）

5. **槓桿影響Beta**：有負債的公司權益Beta比資產Beta高。財務槓桿越高，Beta越高——這就是[[營業槓桿與財務槓桿聯合槓桿判讀Operating-Financial-Combined-Leverage|聯合槓桿]]的概念

### 承擔風險不一定是壞事

風險跟報酬相依，想要好報酬就要承擔合理的系統性風險。真正的風險來自超過承受能力和不知道自己在做什麼。
