# 停損方法

> 停損是交易系統的保險絲，沒有停損就等於沒有煞車在開車。

## 核心概念

七種主要停損方法：

| 方法 | 說明 | 適用情境 |
|---|---|---|
| K線停損 | 跌破關鍵K線低點 | 短線、型態交易 |
| 趨勢線停損 | 跌破趨勢線 | 波段操作 |
| 均線停損 | 跌破重要均線 | 中長線 |
| **理由消失停損** | 進場理由消失時出場 | 有明確進場邏輯時 |
| 固定比例停損 | 虧損達 X% 出場 | 通用、新手首選 |
| 支撐停損 | 跌破支撐區 | 型態交易 |
| 移動停損 | 停損只往獲利方向移 | 趨勢追蹤 |
| ATR動態停損 | 停損 = 進場價 - N×ATR | 波動大的股票 |

### 絕對停損原則
- 單筆虧損 > 10% 必須出場
- 跌破盤整區必須出場
- 趨勢反轉必須出場
- 逆勢單反轉必須出場

### 回本難度表
- 虧 10% → 需漲 11% 回本
- 虧 20% → 需漲 25% 回本
- 虧 50% → 需漲 100% 回本

## 實戰應用

1. **進場前設好停損**：任何進場都要有明確停損點
2. **停損不往下移**：停損只能往獲利方向移動（移動停損）
3. **ATR停損最科學**：考慮了波動度，不會被正常波動洗掉
4. **固定比例新手最安全**：5-7% 固定停損，簡單好執行

## 注意事項

- 停損的價值在「控制情緒」而非「提升績效」
- 不要因為被停損幾次就不用停損
- 停損太緊容易被洗，太寬虧損太大
- 回測證明：移動停損比固定停損的長期績效更好
