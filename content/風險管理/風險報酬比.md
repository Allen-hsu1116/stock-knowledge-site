# 風險報酬比

> 每筆交易的潛在獲利與潛在虧損的比例，是交易系統的生存基石。

## 核心概念

風險報酬比（Risk/Reward Ratio）衡量的是「為了賺 1 塊錢，你願意冒多少風險」：

- **公式**：風險報酬比 = 預期獲利 / 預期虧損
- **1:2 最低標準**：預期獲利至少是停損的 2 倍
- **1:3 理想目標**：長期穩定獲利的交易者多數使用 1:3 以上

### 數學邏輯

| 風險報酬比 | 所需勝率（損益兩平） |
|---|---|
| 1:1 | 50% |
| 1:2 | 33% |
| 1:3 | 25% |
| 1:5 | 17% |

勝率 33% + 風險報酬比 1:2 = 長期不虧不賺。

## 實戰應用

1. **進場前計算**：任何進場前先算好停損停利點位，風險報酬比 < 1:2 不進場
2. **移動停損**：獲利後將停損移到成本價，風險報酬比動態提升
3. **分批進場**：第一口風險報酬比最好，加碼口的風險報酬比遞減
4. **盈虧比 vs 勝率**：低勝率 + 高盈虧比 = 長期穩定（海龜交易法）；高勝率 + 低盈虧比 = 高頻當沖

## 注意事項

- 風險報酬比是「預期」的，實際獲利不一定達到目標
- 過度追求高風險報酬比會錯過很多機會
- 搭配勝率一起看才有意義：1:3 風險報酬比 + 30% 勝率 = 期望值為正
- 移動停損不能太緊，否則容易被震洗出場
