# 通膨風險與實質報酬 Inflation Risk and Real Return

> 通膨是投資的隱形稅——你的名義報酬看起來很美，扣掉通膨才知道真正賺了多少

## 核心概念

通貨膨脹風險又稱購買力風險（Purchasing Power Risk），是指因通貨膨脹導致投資實際購買力下降的風險。這是最容易被忽略的風險，因為投資者常常被「名義報酬」迷惑，忘記計算通膨侵蝕。

### 核心公式

```
實質報酬 = 名義報酬 - 通貨膨脹率
```

| 情境 | 名義報酬 | 通膨率 | 實質報酬 | 你真正的感覺 |
|------|----------|--------|----------|-------------|
| 溫和通膨 | 8% | 2% | 6% | 真的變有錢 |
| 高通膨 | 8% | 7% | 1% | 幾乎沒賺 |
| 惡性通膨 | 8% | 10% | -2% | 越存越窮 |
| 通縮 | 3% | -1% | 4% | 比你想的有錢 |

### 貨幣幻覺

通膨風險最危險的地方在於「貨幣幻覺」（Money Illusion）：投資者看到名義數字增長就以為自己賺錢了。帳面上從100萬變110萬（+10%），但通膨8%，實質只增加2萬。更糟的是，很多人會因為名義數字增加而增加消費，實質購買力反而下降。

### 台灣歷史通膨數據

台灣過去20年平均通膨率約1.5%，但個別年份差異大：
- 2008年：3.53%（原物料飆漲）
- 2009年：-0.87%（金融海嘯通縮）
- 2022年：2.95%（疫情後供應鏈+烏俄戰爭）
- 2023年：3.45%（通膨高峰）

即使平均1.5-2%，20年複利效果下，100萬的購買力只剩67-74萬——**通膨是慢性的隱形稅**。

## 實戰應用

### 1. 不同資產類別的通膨防護力

| 資產類別 | 通膨防護力 | 原因 | 台股實例 |
|----------|-----------|------|----------|
| 固定收益（債券、定存） | ❌ 最差 | 名義收益固定，通膨上升實質收益直降 | 10年公債殖利率1.5%，通膨3%就是-1.5%實質 |
| 高殖利率股 | ⚠️ 中等 | 股息可能成長但追不上通膨 | 金融股殖利率4-5%，但獲利成長受限 |
| 成長型股票 | ✅ 較好 | 企業可轉嫁成本，獲利隨通膨成長 | 台積電營收隨美元計價產品漲價 |
| 不動產 | ✅ 好 | 租金和價值隨通膨上升 | 台北市房價長期跑贏通膨 |
| 原物料股 | ✅✅ 最好 | 產品本身就是通膨的來源 | 鋼鐵、塑化股在通膨期通常表現好 |
| TIPS/通膨連結債 | ✅✅ 最佳 | 本金隨CPI調整 | 台灣無直接TIPS，可用元大台灣50反1對沖 |

### 2. 費雪方程式與通膨預期

```
名義利率 = 實質利率 + 預期通膨率 + 通膨風險溢價
```

當市場預期通膨上升時：
- 央行升息 → 名義利率上升
- 債券價格下跌（現有債券殖利率不夠吸引）
- 股市估值承壓（折現率上升）
- 原物料、不動產抗通膨資產受追捧

### 3. 通膨對不同股票的差異影響

**受益股特徵**：
- 定價能力強（品牌、壟斷），能轉嫁成本
- 輕資產、低資本支出
- 營收隨價格指數連動

**受害股特徵**：
- 競爭激烈無法漲價
- 重資本支出，原料成本占比高
- 合約固定價格無法調整

### 4. 台股通膨期間操作策略

1. **減持固定收益部位**：通膨上升期債券是最差的資產
2. **加碼定價權股**：台積電、聯發科等科技龍頭
3. **配置原物料股**：中鋼、台塑化在通膨期有超額報酬
4. **關注實質利率轉折**：實質利率由正轉負是資產重新定價的關鍵時刻
5. **不動產REITs替代**：台灣缺乏通膨連結債，可用REITs部分替代

### 5. 通膨率計算與監控

台灣主要通膨指標：
- **CPI年增率**：主計處每月公布，最核心指標
- **核心CPI**：扣除蔬果能源，更穩定
- **WPI躉售物價指數**：生產端通膨，領先CPI約3-6個月
- **進出口物價指數**：觀察輸入型通膨壓力

## 注意事項

### 1. 通膨數據落後性
CPI是落後指標，公布的數據反映1-2個月前的物價。用落後數據做交易決策容易追高殺低。WPI和進口物價指數更具領先性。

### 2. 通膨預期比實際通膨更重要
市場反應的是「預期通膨的變化」而非「通膨本身」。當通膨數據公布後如果符合預期，市場反應反而有限；超預期才是真正的衝擊。

### 3. 通膨對長期投資的複利侵蝕
2%通膨看似無害，但30年後100萬只值55萬。長期投資者必須確保實質報酬為正，否則「赚了利率賠了購買力」。

### 4. 通縮也非好事
通縮（物價下跌）下，實質利率上升，債務人負擔加重，消費延遲（為什麼今天買？明天更便宜），經濟陷入惡性循環。日本失落的30年就是通縮陷阱。

### 5. 不同通膨來源的投資含義
- **需求拉動型**：經濟過熱 → 央行升息 → 短空長多
- **成本推動型**：原物料飆漲 → 企業利潤受壓 → 原物料股受益但整體股市受傷
- **輸入型通膨**：匯率貶值+進口漲價 → 台灣高度依賴進口能源，特別脆弱
