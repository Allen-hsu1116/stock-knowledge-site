# 程式交易與主觀交易優劣比較 Systematic vs Discretionary Trading

> 程式交易的核心優勢不是取代人腦，而是克服人性「賺小賠大」的本能——但好的主觀交易者才可能發展出成功的程式交易。

## 核心概念

程式交易（Systematic Trading）和主觀交易（Discretionary Trading）是兩種截然不同的交易方式，各有優劣。成功的交易者不是二選一，而是兩者兼備——用程式交易的紀律克服人性弱點，用主觀交易的經驗提供策略靈感。

### 主觀交易的兩大致命弱點

1. **資訊過載**：快速又大量的資訊造成人腦難以應付，尤其在盤中高壓環境
2. **情緒干擾**：盈虧情境下心理因素和情緒（自信爆棚或壓力干擾）嚴重影響判斷力

### 行為財務學證實：人性內建「賺小賠大」

**測驗一**（賺錢情境）：
- A：確定贏得$250
- B：25%機率贏得$1000，75%機率什麼也沒得到

**測驗二**（賠錢情境）：
- C：確定損失$750
- D：75%機率損失$1000，25%機率沒有任何損失

多數人選A+D——賺錢時先落袋為安，賠錢時不停損拗單。但期望值完全相同：A=B=$250，C=D=-$750。

這就是**前景理論（Prospect Theory）**的經典證明：人類天生在獲利區間風險趨避（早點收工），在虧損區間風險愛好（凹單拚看看）。程式交易用機械化執行打破這個本能。

## 實戰應用

### 程式交易五大優勢

**1. 克服人性弱點的決策不理性**
- 該停損就無血無淚地停損
- 該抱單就牢牢抱著
- 不受恐懼和貪婪影響執行

**2. 績效回測驗證策略有效性**
- 隨機交易是新手最大毛病：憑感覺下單、殺進殺出
- 回測提供客觀數據：勝率、盈虧比、最大回撤、Sharpe Ratio
- 不用真金白銀去市場測試，大幅降低學習成本

**3. 程式碼解讀加速學習**
- 主觀交易的「盤感」無法邏輯化、無法快速複製
- 程式碼清楚記錄策略邏輯，可複製、可修改、可驗證
- 修正→回測→再修正的快速迭代循環

**4. 輔助工具角色**
- 不一定要全自動下單，程式交易概念可作為輔助
- 統計研究市場現象（例如週一到週五哪天振幅最大）
- 自定義技術指標（買賣力道、紅買綠賣等）
- 存股搭配程式空單避險策略

**5. 投資組合管理**
- 多商品多策略組合：提高勝率、降低風險
- 最佳下注比例與槓桿大小的科學計算
- 期權交易速度快，多商品同時管理需要程式輔助

### 主觀交易不可替代的價值

程式交易策略靈感從何而來？**不是憑空想像**。

- 沒有第一線面對市場、深入觀察行情，就不可能歸納出交易規則
- 好的主觀交易者更有可能發展出成功的程式交易
- 程式交易是工具，注入什麼靈魂才是成敗關鍵

### 程式交易與主觀交易對照表

| 面向 | 程式交易 | 主觀交易 |
|------|---------|---------|
| 執行 | 機械化、無情緒 | 受情緒影響 |
| 回測 | 可量化驗證 | 難以回測 |
| 適應性 | 固定規則，需要人為調整 | 盤中靈活應變 |
| 學習門檻 | 需要程式能力 | 需要市場經驗 |
| 黑天鵝 | 可能全盤崩潰 | 可緊急應變 |
| 策略靈感 | 來自主觀觀察 | 來自盤感與經驗 |
| 紀律 | 天然優勢 | 最大挑戰 |
| 過渡期 | 回測→模擬→小資金→正式 | 直接實戰 |

### 實務建議：兩者兼備

**初學者路線**：
1. 先用主觀交易累積市場經驗（至少1年）
2. 記錄交易日誌，找出可量化的規律
3. 將規律轉化為程式碼回測驗證
4. 回測通過後，用小資金程式交易
5. 逐步放大資金，但仍保持主觀交易能力

**進階者路線**：
1. 程式交易處理紀律要求高的部分（停損、部位控制）
2. 主觀判斷處理需要經驗的部分（市場體制辨識、風險事件應對）
3. 兩者互相驗證：主觀判斷與程式訊號一致時信心加倍

## 注意事項

- **程式交易不是聖盃**：回測績效≠實際績效，過度擬合（Overfitting）是最大陷阱
- **主觀交易不是藉口**：「靠感覺」不是主觀交易，有明確規則但保留判斷空間才是
- **過渡期最危險**：從主觀轉程式的過渡期，容易兩邊都不信任
- **黑天鵝事件**：程式交易遇到歷史未見情境可能全盤崩潰，需要人工監控
- **策略衰減**：所有程式策略都有生命週期，市場效率提升後策略會逐漸失效
- **回測陷阱**：生存者偏誤、前視偏誤、過度擬合是回測三大陷阱
