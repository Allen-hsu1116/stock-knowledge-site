# 保證金追繳與融資斷頭 Margin Call and Forced Liquidation

> 融資維持率低於130%觸發追繳令，T+2不補就T+3強制斷頭——槓桿的代價是失去控制權

## 核心概念

### 什麼是保證金追繳（Margin Call）？

融資買股票時，券商借你60%的資金，你自備40%。當股價下跌，你的自備款被虧損吃掉，融資維持率跌破130%時，券商就會發出追繳令（Margin Call），要求你在2天內補錢。

**融資維持率公式**：
```
融資維持率 = 股票現值 ÷ 融資金額 × 100%
```

**整戶擔保維持率**（含融資+融券）：
```
整戶擔保維持率 = (融資擔保證券市值 + 融券擔保價款及保證金) ÷ (原融資金額 + 融券證券市值) × 100%
```

### 三種保證金層次

| 類型 | 定義 | 實務意義 |
|------|------|----------|
| 原始保證金 | 進場前必須準備的最低金額 | 台股上市40%、上櫃50% |
| 維持保證金 | 持倉期間帳戶必須維持的最低金額 | 融資維持率130% |
| 追繳保證金 | 原始保證金與帳戶餘額的差額 | 需補至維持率166%以上 |

### 斷頭（Forced Liquidation）

收到追繳令後，T+2日內未補足，券商在T+3日開盤強制賣出你的股票。這不是「賣在低點」的問題——是你根本沒有選擇權。

## 實戰應用

### 融資維持率計算實例

**基本案例**：以100元融資買進一張
- 融資金額：100 × 1,000 × 60% = 60,000元
- 自備款：100 × 1,000 × 40% = 40,000元
- 初始維持率：100,000 ÷ 60,000 = 166%

| 股價 | 維持率 | 狀態 |
|------|--------|------|
| 100元 | 166% | ✅ 安全 |
| 78元 | 130% | ⚠️ 追繳門檻 |
| 60元 | 100% | 🔴 斷頭危險區 |

**關鍵數字**：股價跌至買入價的78%，維持率就觸及130%追繳門檻。

### 避免斷頭三階段處理

1. **T+2日內補至166%以上** → 券商撤銷追繳令，恢復正常
2. **T+2日內補至130%～166%** → 券商暫不強制處分，但追繳令仍存在；若再跌破130%，當日下午須立即補錢
3. **T+3日仍未補至130%以上** → 券商強制賣出（斷頭）

### 惡性循環機制

斷頭最可怕的不是虧損本身，而是連鎖反應：

```
股價大跌 → 維持率跌破130% → 追繳令 → 
投資人無力補繳 → 券商強制賣出（低於市價） → 
賣壓進一步打壓股價 → 更多投資人維持率跌破130% → 
更多追繳令 → 更多斷頭 → 惡性循環
```

### 融資維持率作為市場反轉指標

TEJ回測發現，當個股融資維持率異常偏低時，存在短期反轉機會：

**進場條件**：
- 當日融資維持率 < 過去10日移動平均（異常低檔）
- 當日收紅K（close > open）
- 當日融資餘額 > 前5日平均 × 0.95（籌碼未潰散）

**回測結果**（2020-2025）：
- 年化報酬率：18.39%
- 夏普值：1.04
- 最大回撤：-28.47%

### 2025年4月實戰案例

台股單日下跌2,065點（跌幅9.6%），融資維持率降至133.8%，589檔個股維持率低於130%。

- **緯創（3231）**：融資維持率下探至91%，代表絕大多數融資戶處於浮虧
- **技嘉（2376）**：維持率跌破100%，引發多日追繳賣壓
- 後續即使短線反彈，因籌碼凌亂、浮虧戶未清洗，表現明顯弱於大盤

## 注意事項

### 五大斷頭陷阱

1. **高檔融資進場**：散戶最常在股價創高時融資追漲，成本偏高，稍有回檔就觸及追繳
2. **只放原始保證金**：40%自備款是「最低標準」，在最低標準操作等於在危險邊緣試探
3. **忽略融資維持率的整戶計算**：融資+融券合併計算，一個部位的虧損可能拉低整戶維持率
4. **除權息陷阱**：除息後股價下降，維持率跟著下降，融資戶需特別注意
5. **惡性循環的系統性風險**：大盤暴跌時，大量融資戶同時被追繳，形成系統性賣壓

### 安全防護建議

- **融資維持率 > 166%** 才是真正安全區，130%只是最低門檻
- **自備款建議 > 50%**（而非最低40%），保留安全邊際
- **融資只用於短期交易**，長期持有不適合融資
- **收到追繳令時**：先計算補繳金額，評估是否值得補繳，不補就果斷出場
- **大盤融資維持率低於140%** 時，注意系統性風險

### 期貨 vs 融資保證金差異

| 項目 | 台股融資 | 台灣期貨 |
|------|----------|----------|
| 原始保證金 | 股價40%-50% | 期交所規定 |
| 維持門檻 | 維持率130% | 期交所規定 |
| 追繳通知 | 維持率<130% | 盤中/盤後低於維持保證金 |
| 補繳門檻 | 補至166%以上 | 補至原始保證金 |
| 斷頭門檻 | T+3強制賣出 | 盤中風險指標<25%立即強平 |
| 加收保證金 | 無 | 單一商品>20%部位加收20% |
