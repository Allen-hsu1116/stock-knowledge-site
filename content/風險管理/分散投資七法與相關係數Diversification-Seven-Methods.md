# 分散投資七法與相關係數 Diversification Seven Methods and Correlation Coefficient

> 相關係數越低，分散效果越好——但危機時一切相關性趨向1

## 核心概念

### 分散投資的定義

分散投資（Diversification）是將資金分散投入不同行業、企業、市場等，以降低投資風險、追求收益最大化的策略。

### 分散投資七法

| 方法 | 說明 | 實例 |
|------|------|------|
| **分散行業** | 不集中購買同一行業股票 | 電子+金融+傳產各佔1/3 |
| **分散企業** | 投入多家企業，避免單一企業風險 | 同產業選2-3檔 |
| **分散品種** | 股票與債券等不同資產搭配 | 股60%債40% |
| **分散地域** | 投資不同國家/地區 | 台股+美股+新興市場 |
| **分散時間** | 不同時間投入資金 | 定期定額、分批進場 |
| **分散板塊** | 同時持有不同特性板塊 | 大型股+小型股+成長+價值 |
| **分散市場** | 不同市場升跌不同步 | 股市+債市+商品+REITs |

### 相關係數基礎

相關係數（Correlation Coefficient, ρ）是度量兩變數線性關係密切程度的統計指標，由卡爾·皮爾遜設計。

**取值範圍**：-1 ≤ ρ ≤ +1

| ρ 值範圍 | 關係強度 | 分散效果 |
|----------|----------|----------|
| 0.8 ~ 1.0 | 高度正相關 | 幾乎無分散效果 |
| 0.3 ~ 0.8 | 中度正相關 | 部分分散效果 |
| -0.3 ~ 0.3 | 弱相關 | 良好分散效果 |
| -1.0 ~ -0.3 | 負相關 | 最佳分散效果 |

### 投資組合風險公式

兩資產組合的變異數：

**σ²p = w²A·σ²A + w²B·σ²B + 2·wA·wB·σA·σB·ρAB**

當 ρAB < 1 時，組合風險低於加權平均風險——這就是分散投資的數學基礎。

- ρAB = 1：組合風險 = 加權平均風險，分散無效
- ρAB = 0：組合風險 < 加權平均風險，分散有效
- ρAB = -1：理論上可完全消除風險

### 相關係數的陷阱

1. **不相關 ≠ 獨立**：ρ = 0 只表示沒有線性關係，可能存在非線性關係
2. **樣本量影響**：n 較小時相關係數易產生假象（n=2時 |ρ| 恆為1）
3. **時間變異**：過去的相關性不代表未來的相關性
4. **狀態依賴**：正常時期和危機時期的相關性結構完全不同

## 實戰應用

### 1. 分散投資的實際建構

**台股投資組合分散範例**：
- 行業分散：半導體(30%) + 金融(20%) + 光電(15%) + 傳產(15%) + 其他(20%)
- 時間分散：分3個月進場，每月投入1/3
- 品種分散：股票(70%) + ETF(20%) + 現金(10%)

### 2. 相關性檢查步驟

1. 計算持股間的相關係數矩陣
2. 找出相關性 > 0.7 的配對
3. 高相關配對視為同一風險來源，合併計算部位
4. 確保整體組合平均相關性 < 0.5

### 3. 危機時的相關性趨向1

這是分散投資最大的陷阱：平時分散有效的資產，在黑天鵝事件中全部一起跌。必須搭配尾部風險對沖。

詳見 [[相關性風險Correlation-Risk]] 和 [[黑天鵝事件與尾部風險基礎Black-Swan-and-Tail-Risk-Fundamentals]]

### 4. 分散的限度

- 分散到 15-20 檔股票已消除大部分非系統性風險
- 超過 30 檔的邊際效益極低
- 過度分散 = 過度稀釋報酬

## 注意事項

- **分散不等於避險**：分散降低的是非系統性風險，系統性風險需要對沖工具
- **相關性是動態的**：不能用靜態的相關係數做長期配置決策
- **危機時相關性趨向1**：多樣化在你最需要它的時候失效
- **過度分散稀釋報酬**：15-20檔已足夠，30檔以上意義不大
- **樣本量陷阱**：小樣本的相關係數不可靠，需做顯著性檢驗
- **非線性關係被遺漏**：相關係數只捕捉線性關係
