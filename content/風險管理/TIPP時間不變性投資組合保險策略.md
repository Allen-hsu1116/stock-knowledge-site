# TIPP時間不變性投資組合保險策略

> CPPI的改良版，保本底線會隨資產增值而上升，鎖定獲利、落袋為安。

## 核心概念

TIPP（Time-Invariant Portfolio Protection）是對CPPI策略的改進，核心差異在於**保本底線的動態調整**：

- **CPPI**：保本底線 F 是固定的，由初始保本額度貼現決定
- **TIPP**：保本底線取 max(當期保本額度, 上期保本額度)，會隨資產增值而上升

公式差異：
- CPPI：F_t = 固定值（初始設定的保本額度貼現值）
- TIPP：F_t = max(當期保本額度, 前期保本底線)

**效果**：
- 投資組合價值增加時 → 保本底線也提高 → 自動鎖定獲利
- 投資風格更穩健
- 避免前期收益在後續市場變動中損失殆盡

## 實戰應用

### 舉例

初始資金100萬，100%保本，無風險利率3%，乘數2，保本期3年：

**T=0**：
- CPPI底線 = 100/(1.03)^3 = 91.52萬
- TIPP底線 = 91.52萬（同CPPI）
- 安全墊 = 100 - 91.52 = 8.48萬
- 風險部位 = 8.48 × 2 = 16.96萬

**T=1（權益+20%，總資產增加至約105.86萬）**：
- CPPI底線 = 94.26萬（固定貼現，持續上升）
- TIPP底線 = max(94.26, 91.52) = 94.26萬（但若資產大漲，TIPP底線會更高）
- **關鍵差異**：當資產增長後，TIPP會將部分獲利納入新的保本底線

### TIPP vs CPPI 比較

| 特性 | CPPI | TIPP |
|------|------|------|
| 保本底線 | 固定 | 動態上升 |
| 獲利處理 | 不鎖定，全部暴露在風險中 | 部分鎖定，落袋為安 |
| 風格 | 較積極 | 較穩健 |
| 多頭表現 | 較好（更多資金在風險資產） | 較保守 |
| 空頭保護 | 固定底線 | 上升底線，保護更多 |

### OBPI（基於期權的投資組合保險）

第三種保本策略，原理相同但用期權替代風險資產：
- 大部分資金投資國債 → 到期保本
- 剩餘資金購買股指看漲期權 → 參與上漲
- 最大損失 = 權利金（期權費）

## 注意事項

- **TIPP上漲時保守**：鎖定獲利意味著少參與後續漲幅，多頭市場表現不如CPPI
- **保本不保證**：極端行情（如1987黑色星期一，道瓊單日跌22.6%）仍可能跌破底線
- **交易成本**：頻繁調整的摩擦成本
- **保本基金新規限制**：穩健資產 ≥ 80%，權益 ≤ 安全墊3倍
- **贖回風險**：市場劇烈波動引發大量贖回，被迫在低點調整
