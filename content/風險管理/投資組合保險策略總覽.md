# 投資組合保險策略總覽

> 用無風險資產保底、風險資產博收益，漲時加碼跌時減碼——保本與獲利之間的動態平衡

## 核心概念

投資組合保險策略（Portfolio Insurance Strategy）是一類動態資產配置策略的總稱：

- **核心思想**：把資金分為無風險資產（保底）和風險資產（博收益），隨市場變化動態調整比例
- **目標**：保證投資組合的最低價值，同時不放棄上漲的潛力
- **本質**：紀律化的「漲時加碼、跌時減碼」

### 三大類投資組合保險策略

| 類型 | 代表策略 | 核心機制 | 特點 |
|------|----------|----------|------|
| 期權型 | OBPI | 買賣權保護或複製賣權 | 需要估算波動率 |
| 參數型 | CPPI / TIPP | 安全墊×乘數動態調整 | 不需估算波動率 |
| 混合型 | VGPI | 結合價值增長的CPPI改良 | 交易成本最低 |

## 實戰應用

### OBPI策略實戰

**歐式保護性賣權**：持有股票 + 買入賣權（Put）
- 股價上漲：賣權歸零，享受上漲利潤
- 股價下跌：賣權獲利，彌補股票虧損
- 成本：賣權權利金 = 「保險費」

**複製性賣權**：用股票+無風險資產複製賣權的payoff
- 不需要選擇權市場存在
- 需要連續調整Delta
- 交易成本較高

### TIPP策略實戰

TIPP（Time-Invariant Portfolio Protection）是CPPI的改良版：

- **關鍵差異**：TIPP的保本底線不是固定的，而是隨著資產上漲而提高
- **鎖利機制**：當資產上漲到新高峰，新的高點成為新的保本底線
- **更保守**：永遠保護「當前最高的資產價值」，而不是只保護期初設定

範例：
```
期初：總資產100萬，保本底線80萬，安全墊20萬
股價上漲 → 總資產漲到120萬
CPPI：保本底線仍為80萬，安全墊40萬
TIPP：保本底線提高到96萬（120萬×80%），安全墊24萬
```

TIPP在牛市中更保守（更多資金鎖定在無風險資產），但在熊市中更安全（鎖住了已實現的利潤）。

### VGPI策略

價值增長型組合保險策略（Value Growth Portfolio Insurance）：
- 結合CPPI的動態調整與價值增長概念
- 交易成本相對最低
- 在大多數市況下表現穩定

## 五大策略特點

### 1. 追漲殺跌的本質

所有投資組合保險策略都是下跌減倉、上漲加倉，這是保本的代價。

- 單邊趨勢市 → 表現好
- 震盪市 → 反覆買高賣低

### 2. 投資目標的絕對性

保本是無條件的目標，不是「盡量」保本，而是「必須」保本。

### 3. 紀律性極強

必須嚴格執行再平衡，不執行就等於沒有保險。

### 4. 終值偏態分佈

切除下跌風險、保留上漲獲利 → 報酬分佈少了左尾 → 正偏態

### 5. 交易成本和機會成本

- **交易成本**：頻繁再平衡產生交易費用
- **機會成本**：大部分資金放在低收益資產，放棄了部分上漲機會（「保費」）
- **流動性要求**：急跌時需要賣出，流動性不足會失效

## 注意事項

### 策略選擇指南

| 市況 | 推薦策略 | 原因 |
|------|----------|------|
| 大多頭 | CPPI | 追漲效果最大化 |
| 大空頭 | TIPP | 鎖住利潤，保本能力最強 |
| 震盪市 | 固定比例 | 低買高賣，避免追漲殺跌 |
| 不確定 | TIPP | 進可攻退可守 |
| 極端波動 | OBPI | 賣權提供精確保護 |

### 散戶常見錯誤

1. **保本底線設太低**：幾乎等於沒保本，風險資產比例過高
2. **乘數設太大**：Gap Risk暴增，急跌時來不及減碼
3. **不執行再平衡**：策略等於白設
4. **風險資產選冷門股**：流動性不足，跌停鎖死無法減碼
5. **忽略交易成本**：頻繁調整在台股的手續費+交易稅很可觀

### 實證結論（台股）

- CPPI、TIPP、VGPI都能保證本金安全同時參與市場上漲
- CPPI在牛市獲利最高，但熊市保本能力略低
- TIPP在熊市表現最好，鎖住利潤能力最強
- CPPI交易成本最高，TIPP次之，VGPI最低
- 配合濾嘴法則（1%-9%比率）的權變策略，在台股近20年實證中確實可取得超額報酬
