# 交易日誌與交易紀律養成

> 交易日誌是交易者的成長DNA，記錄每一筆交易的心理動機與客觀條件；交易紀律是成功的基石，80%靠心理只有20%靠技巧

## 核心概念

交易日誌不只是記帳，而是一套結構化的自我回饋機制。透過系統化記錄每一筆交易的進場理由、市場條件、心理狀態與結果分析，找出反覆犯的錯誤並針對性改進。

**成功的交易80%靠心理，只有20%靠技巧**——馬克·道格拉斯（Mark Douglas）在《紀律的交易者》中指出，紀律是基數，策略與技巧是乘數。良好的紀律與資金控管，才能讓獲利持續奔跑。

## 實戰應用

### 一、交易日誌四步驟完整教學

#### 步驟1：買進後的紀錄

完成一筆交易時，紀錄以下事項：

- **買進日期、張數、股票代碼**
- **買進理由**：是不是符合你的策略？
  - ❌「我覺得股票會上漲」→ 主觀猜測，容易賠錢
  - ✅「股票已經到達買點...所以我才買」→ 符合策略的客觀判斷
  - **核心思維**：符合自己的買點才買，訓練自己成為機械化交易者
- **買進的技術分析圖型截圖**（超級重要）
  - 上面應該寫上停損
  - 畫出買點及支撐與壓力
  - 個人的技術分析判斷
  - 作為日後檢討依據

#### 步驟2：持有期間的規則設定（超過90%的人沒做到）

持有期間應該為自己寫下一連串規則（rules），並在下單完後就訂下：

- **時間停損點**：在x月x號，如果價格沒到x元，我就賣出
- **空間停損點**：百分比停損法、均線停損法等
- **最大可能虧損**：通常設定比停損的虧損更高，因為大盤可能突然跌
- **何時看盤，決定要不要賣出**
  - 開盤9:00-9:30？
  - 尾盤13:00-13:30？
  - 盤中逢高賣出？
- **停利點**：股價帳面獲利超過x元則停利、股價到了x價位則停利
- **加碼點**：股價穿越xx價格才再買一張、帳面獲利x元再買一張

**極端狀況準備**：
- 如果遇到大盤狂跌千點，導致個股瞬間穿過停損價格，此時該賣嗎？還是靜待反彈？
- **永遠都要為任何狀況做好準備，否則你的交易就沒有永遠**

**紀錄心情**：寫得像日記，隨興發揮，買進後的每天都要寫，不用批判自己，只要記錄即可。

#### 步驟3：賣出時的紀錄

- 賣出時間
- 賣出股價×張數
- 獲利/虧損多少
- **賣出的理由**（重要！可以和買進理由做對照）
- 截圖技術線型（把股價賣出點位截圖，在K棒上做紀錄）

**常見問題**：如果盤中確信明天會上漲，但是規則訂今天該賣出，可以不賣嗎？
- 問題不在於要不要賣，而是為什麼當初規則會把賣出日期訂得如此短
- 不該在盤中做判斷，寧願錯過或少賺，都不應該讓自己成為主觀交易者
- 下次交易可以加上但書，比如看到xx時就可以不賣

**這就是規則的力量**——讓我們盤中輕鬆面對股市的任何情況，只要根據開盤前設定的規則照著操作即可。

#### 步驟4：最多人忽略的一步——檢討

紀錄完後需要檢討，但**盡量不要馬上檢討**，應該過一個月後再進行檢討：

| 檢討項目 | 問題方向 |
|---------|---------|
| 時機 | 賣的太早？買的太晚？ |
| 策略 | 策略哪裡需要改進？ |
| 停損 | 停損時間該提早或延後？停損方式調整更寬或更窄？ |
| 一致性 | 買進與賣出理由一致嗎？ |
| 資金 | 調整每次下的資金？ |
| 勝率 | 自己的勝率多少？ |
| 個性 | 觀察自己的個性，調整買的方式 |
| 效率 | 持股夠有效率嗎？ |
| 複製 | 賺錢了，如何複製賺錢的方式？行為做對了嗎？（賺錢≠行為做對，可能是運氣） |
| 避免 | 賠錢了，如何避免下次再賠？行為做對了嗎？（賠錢≠行為做錯，需具體分析） |

**進階技巧**：再截圖一次，觀察買進、賣出、一個月後的股價變化。

### 二、交易旅程框架：交易前、交易中、交易後

投資人在交易的過程，依據時間框架不同可分為三個階段：

#### 交易前（Pre-Trading Stage）
- 分析市場，找到支撐和阻力
- 研究進場模式，確認趨勢行情或橫盤整理特徵
- 使用工具畫趨勢線、預設支撐價位
- 在股價旁寫關鍵註記（如「股價突破60日新高」、「周轉率上升至5%」）
- **預估停損價位**

#### 交易中（Trading Stage）
- 建立部位、管理部位、平倉
- 衡量標準：能否遵循前一天晚上制訂的操作計劃
- 保持心態靈活性——計劃不是死的，需要時可以調整
- **不要在盤中想上一筆交易的檢討**（那是交易後的事）
- 專注於尋找良好的交易機會

#### 交易後（Post-Trading Stage）
- 堅持對所做和錯失的交易做回顧
- 寫交易日誌
- 評估交易表現
- 找安靜的地方客觀思考

### 三、交易日誌九大必填欄位

每條交易紀錄至少要包含：

1. **交易日期、星期和時間**——確認是否在某些特定時間做交易
2. **交易帳戶**——不同策略的專用帳戶
3. **交易策略**——判定哪種方法獲利最佳
4. **市場資訊**——交易進行時的詳細市場資訊
5. **股票或商品代碼**
6. **多空方向**——區分多頭和空頭交易
7. **交易起因**——每筆交易一開始的構想
8. **交易設定**——簡單說明為何做這筆交易（如「第二號波段操作策略：90%」，不能只寫「我覺得不錯」）
9. **盈餘公布日期**——設定提醒，決定跨越盈餘公告時的處理方式

### 四、紀律交易者三心法（馬克·道格拉斯）

#### 心法一：建立交易結構
市場的混沌無法想像，幾萬個交易者每個人都有不同的經驗、個性、資金體量和價值觀。建立交易結構相當重要——將混沌的環境透過合理的分類、分別、分析，剝除擋住視線的眼罩，逐漸看清眼前的狀況。

無論從什麼流派進入，選擇一個去深入了解、復盤、做單，漸漸地會對市場有朦朧卻重要的基礎認知。

#### 心法二：正視虧損
**永遠記得：虧損無法避免。**

- 設置風險管理：一單只承受總資金2%的虧損
- **人性中，當能預期一個可能發生的損失，真的成立了就比較沒那麼痛苦**——預期損失是多少，吃筍時就不會痛得那麼誇張
- 獲利時需特別留意潛在的加倉風險——貪婪與死亡並行，見好就收或移動止損

#### 心法三：只關注當下
過去的一單無論是吃筍還是止盈，都是過去的體驗。在這個當下只有錢的總數是真的，盤面的跳動是真的。培養強健的身心，專注面對下一場比賽。

### 五、心理情緒——峰值與終值理論

體驗的記憶由兩個核心因素決定：
1. **峰值**：體驗最高峰時的感覺（正向或負向最高峰）
2. **終值**：結束時的感覺

IKEA效應：整個採購流程並非一直愉快，但峰值是坐在樣品屋的體驗，終值是出口處10元冰淇淋，所以多數人對IKEA評價不錯。

交易中的應用：
- 拿住獲利中的部位或許不愉快，但可能是有益的（低的峰值）
- 持股可能獲利而感到興奮（高的峰值），但同時提醒我們也有虧損風險
- 不愉快的感覺可以視為訊號，認真觀察市場

加強對情緒的覺察，有效運用情緒，可以更好地進行交易，並持續自我進化。

### 六、月回顧與年度檢討機制

#### 月回顧
每月底花1小時：
- 計算當月交易勝率
- 分析盈虧比
- 找出3筆「表現最差」的交易，核對是否違反停損規則
- 重點在「找錯」而非「找對」

#### 年度檢討
每年12月：
- 找一兩天假期做年終檢討
- 特別注意年度最高點和最低點
- 檢討犯下的錯誗和學習到的教訓
- 匯整學到的知識，把重點寫在年度紀錄的最後一頁
- 回顧這些重點（還有前幾年記下的重點），看看是否還在繼續學習和吸收

### 七、誠實記錄的原則

- 交易日記的主要目的：捕捉那些當下影響我們、但以後不會在線圖上顯現的事情
- 對自己誠實地記錄一切——誰也看不到這份日記
- 如果對自己做的事情感到尷尬而不想記錄下來，那就是一個很好的訊號，表示知道自己做錯了什麼
- 如果一點感覺也沒有，永遠無法從錯誤中學到經驗和教訓
- **真正的問題通常不是交易本身好或壞，而是交易背後的思考和動機**

### 八、語言的力量

長期暗示自己是「韭菜」、「散戶」，對交易成長沒有幫助。言語的力量無時無刻都在影響內心，只能依靠自己的引導，避免前往不想前往的地方。

## 注意事項

- 沒有記錄就沒有成長——交易日誌是從散戶進化為專業投資人的基礎工具
- 重點不是記錄正確的交易，而是從錯誤中學習
- 至少累積30筆交易記錄後才能評估SOP的有效性
- 檢討不要馬上做，至少過一個月再進行——善用馬後炮的方式看出自己的盲點
- **成長無捷徑，唯有紀錄與檢討**
