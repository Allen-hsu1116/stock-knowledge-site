# 自我服務偏差 Self-Serving Bias

> 賺了是我厲害，賠了是市場爛——自我服務偏差讓你永遠學不到教訓

## 核心概念

自我服務偏差（Self-Serving Bias）是指人們傾向於把成功歸因於自己的能力，把失敗歸因於外部因素。這是人類最普遍的認知偏誤之一，在交易中尤其致命。

典型表現：
- 「這筆賺了，我的分析真準」→ 歸因於內部（能力+努力）
- 「這筆賠了，主力太陰險」→ 歸因於外部（運氣+環境）
- 結果：**贏了自大，輸了不學**

### 歸因理論三維度（Weiner）

心理學家韋納提出歸因傾向三個維度：

| 維度 | 成功歸因 | 失敗歸因 | 交易影響 |
|------|---------|---------|---------|
| 內因vs外因 | 歸於能力（我強） | 歸於運氣（市場爛） | 贏了自大、輸了不反省 |
| 穩定vs易變 | 歸於穩定（我一直強） | 歸於不穩定（這次運氣差） | 過度自信持續、教訓不被記取 |
| 可控vs不可控 | 歸於可控（我的方法） | 歸於不可控（黑天鵝） | 忽略可控的風險管理失誤 |

### 最危險的歸因模式

將失敗歸因於「內部+穩定+不可控」→產生**習得性無助感**：
- 「我就是不會交易」
- 「股市就是騙人的」
- 這不是謙虛，是放棄

## 實戰應用

### 1. 交易日誌的歸因校正

強迫自己對每筆交易做雙向歸因：

**賺錢的交易**：
- ✅ 正確歸因：「我的系統在這種市況有效」
- ❌ 偏差歸因：「我太強了」
- 🔍 追問：「運氣佔多少？市場整體上漲我也跟著賺？」

**賠錢的交易**：
- ✅ 正確歸因：「我的停損設太緊，被洗出場」
- ❌ 偏差歸因：「主力洗盤太陰險」
- 🔍 追問：「我可控的部分是什麼？進場時機？部位大小？」

### 2. 歸因平衡表

每週做歸因平衡表，計算你的歸因偏差程度：

```
本週獲利交易：
  歸因於自己：___次
  歸因於運氣：___次

本週虧損交易：
  歸因於自己：___次
  歸因於外部：___次

偏差指標 = (獲利自歸因次數/總獲利次數) - (虧損自歸因次數/總虧損次數)
偏差指數 > 0.3 → 嚴重自我服務偏差
```

### 3. 反自我服務偏差清單

交易前問自己：
- [ ] 如果這筆交易虧了，我會怪什麼？（提前識別外部歸因藉口）
- [ ] 如果這筆交易賺了，我會覺得自己多厲害？（提前識別內部歸因衝動）

交易後問自己：
- [ ] 「如果我做相反的操作，結果會如何？」（測試運氣vs能力的比例）
- [ ] 「同樣的策略在過去10次中成功了幾次？」（用數據打破自我膨脹）
- [ ] 「這筆虧損中，哪部分是我可控的？」（找到可改善的而非全部歸咎外在）

### 4. 團隊交易與自我服務偏差

在投資社群或跟單群中：
- 共同獲利→每人覺得自己貢獻最大
- 共同虧損→每人覺得是別人的錯
- **這是人際衝突的主要來源**

### 5. 自我服務偏差的四層市場影響

1. **過度交易**：覺得自己比市場聰明→頻繁操作
2. **忽視風險管理**：覺得虧損是意外不是系統問題→不加強防護
3. **學習停滯**：失敗歸因外部→不檢討不改善
4. **報復性交易**：虧損不認帳→想「證明」自己是對的

## 注意事項

1. **自我服務偏差不是「裝謙虛」就能解決**——你需要在結構上強制自己對稱歸因
2. **適度的自我服務偏差有正面作用**——完全歸因於自己（連失敗也是）會導致習得性無助
3. **認知模型vs動機模型**：偏差來自認知（期望成功所以合理化）和動機（保護自尊），兩者同時作用
4. **與[[過度自信偏誤Overconfidence-Bias]]互為表裡**——自我服務偏差是歸因方式，過度自信是結果
5. **最健康的心態**：「我的系統有效，但不是萬能；虧損是系統成本，不是我的失敗」
