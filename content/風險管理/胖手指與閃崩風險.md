# 胖手指與閃崩風險

> 一個零的差距就是100%的虧損——人為操作失誤是金融史上最古老也最持續的風險

## 核心概念

**胖手指（Fat Finger Error）**：使用電腦時的人為操作失誤，如按錯錯鍵、輸入錯誤的金額或數量、多一兩個零，造成錯誤交易。

**閃崩（Flash Crash）**：在極短時間內（幾分鐘到幾小時內），某市場或股票迅速崩跌，然後快速反彈回到原來的價位。

兩者關係：胖手指是觸發閃崩的開關之一。胖手指引發的錯誤交易會觸發大量自動停損單，停損賣單進一步壓低價格，觸發更多停損，形成連鎖崩潰。

### 閃崩 vs 崩盤

| | 閃崩 | 崩盤 |
|---|------|------|
| 持續時間 | 幾分鐘到幾小時 | 數天到數月 |
| 反彈 | 快速反彈回原價 | 不一定反彈 |
| 主因 | 人為失誤、程式交易異常 | 經濟基本面惡化 |
| 典型案例 | 2010年5月6日美股閃崩 | 2008年金融海嘯 |

## 實戰應用

### 七大歷史血案

| 事件 | 年份 | 原因 | 損失 |
|------|------|------|------|
| **瑞穗證券** | 2005 | 「61萬×1股」打成「1×61萬股」 | 400億日圓（≈110億台幣） |
| **富邦證券** | 2005 | 8000萬買單打成80億 | 台股急漲50點，200檔漲停 |
| **中裕公司** | 2015 | 105元打成1.05元 | 興櫃無漲跌幅限制，差100倍 |
| **第一金證券** | 2016 | 選擇權系統異常 | Put從1.4飆到895，虧1300萬 |
| **法國巴黎證券** | 2018 | 胖手指 | 台塑市值蒸發≈30億美元 |
| **宿霧航空** | 2019 | 券商輸入錯誤 | 股價暴跌37% |
| **三星證券** | 2018 | 股息1000韓元→1000股 | 名義價值超公司市值30倍 |

### 散戶防胖手指五道防線

1. **交易限制設定**：券商APP設定單筆金額上限，避免一次按錯全部虧光
2. **下單前最後檢查**：養成「三查」習慣——查股數、查價格、查方向
3. **限價單優先**：避免都市價單，限價單至少鎖住最壞的成交價
4. **疲勞不交易**：壓力和眼睛疲勞是胖手指的溫床
5. **模擬單先行**：大單先用小單試水溫，確認流程沒問題再加碼

### 從閃崩中獲利？

有些程式交易者專門搜尋不合理的閃崩價格，在極短時間反向進場賺取利潤。但這是高技術性行為，需要：
- 極快的交易速度
- 完善的閃崩偵測演算法
- 嚴格的風險控制
- 同樣有很多程式在搶這種機會

## 注意事項

- **興櫃無漲跌幅限制**：胖手指在興櫃市場的殺傷力遠大於集中市場
- **台股一張=1000股**：美股一單位是1股，下單前確認市場否則差1000倍
- **程式交易是雙面刃**：自動化減少人為失誤，但程式BUG可能比胖手指更恐怖（騎士資本案例）
- **閃崩不可預測**：你能做的是讓自己的策略在極端波動中不被掃出場
- **熔斷機制不是萬能**：美國SEC的5分鐘10%暫停規則，但很多閃崩在5分鐘內已經完成
- **分散投資是唯一免費保險**：胖手指和閃崩影響的是特定標的或市場，分散可降低衝擊
