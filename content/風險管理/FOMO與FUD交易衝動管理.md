# FOMO與FUD交易衝動管理

> FOMO讓你在貪婪時追高，FUD讓你在恐懼時殺低——按設計交易，而非按情緒交易

## 核心概念

FOMO（Fear of Missing Out，錯失恐懼症）和FUD（Fear, Uncertainty, Doubt，恐懼不確定懷疑）是交易中最具破壞力的兩大情緒敵人。兩者互為鏡像：FOMO驅使你在高點追入，FUD驅使你在低點恐慌賣出。

### FOMO的本質

- 看到別人獲利→大腦將他人利潤轉化為自己「錯失的利潤」→不計成本追高
- 社群媒體曬單文化加劇FOMO：只看得到獲利，看不到虧損
- 當FOMO衝動來襲時，資產通常已被過度拉伸

### FUD的本質

- 負面消息或市場下跌→恐懼情緒主導→不計價格拋售
- 損失帶來的痛苦是等額獲利快樂的2-2.5倍（損失厭惡），FUD放大了這個效應
- 恐慌性拋售鎖定本可避免的虧損，暫時緩解恐懼但造成永久損失

### FOMO與FUD的循環

```
FOMO追高 → 買在頂部 → 價格回落 → 虧損 → FUD恐慌殺低 → 賣在底部
  ↑                                                            ↓
  └──────────────── 看到反彈又FOMO追高 ←──────────────────────────┘
```

## 實戰應用

### FOMO來襲時的應對策略

1. **回歸交易計畫**：問自己「目前價格是否符合我計畫中的進場點？」如果不是，堅決不動
2. **物理隔離**：暫時關掉交易軟體，或將圖表縮小到看不清細節
3. **機會成本思考**：市場永遠有下一個機會，錯過這次並不可惜，追高被套的虧損才是真實的
4. **30天冷靜期**：看到想追的標的，記錄下來等30天再看（波段交易適用）
5. **JOMO心態**（Joy of Missing Out）：選擇不抓住每一個機會帶來平靜和專注感

### FUD來襲時的應對策略

1. **事實查核**：辨別消息來源的可靠性，是真實基本面惡化還是市場情緒渲染？
2. **重溫持倉理由**：當初買入的根本原因是否改變？沒改變就沒有理由因為噪音賣出
3. **檢視止損位**：止損位是否被觸發？沒有就讓計畫保護你
4. **反轉思考**：如果今天空手，你會在這個價格買入嗎？

### 冷靜期SOP（情緒急救五步驟）

1. **立即離開螢幕**，深呼吸三次
2. **散步5-10分鐘**，做與交易無關的事
3. **回來後先看交易計畫**，不是看價格
4. **只有完全冷靜後才允許操作**
5. **記錄這次情緒觸發事件**，寫入交易日誌

### Mark Douglas紀律交易七步驟

1. **聚焦學習**：先選定一個領域深入鑽研，不要什麼都想學
2. **處理虧損**：接受虧損是交易的一部分，是成本不是失敗
3. **成為專家**：選擇一種市場行為深入研究，成為這方面的專家
4. **完美執行**：找到適合自己的交易系統，反覆練習到熟練
5. **機率思考**：每一筆交易都是獨立事件，不被上一筆結果影響
6. **學習客觀**：市場不針對你，上漲下跌都是中性事件
7. **監督自己**：觀察自己的情緒反應模式，找出弱點並建立防禦機制

### 三大交易心理陷阱

1. **從眾心理**：社群論壇都在討論就盲目跟風 → 建立獨立思考能力，任何決策必須基於自己的研究
2. **過度交易**：沒有明確信號時頻繁進出 → 設定每日/每周交易次數上限
3. **報復性交易**：虧損後立刻想「把錢賺回來」 → 設定當日虧損上限，觸及即關機

### 過程導向vs結果導向

- **結果導向**（錯誤）：賺錢=好交易，虧錢=壞交易
- **過程導向**（正確）：遵守計畫=好交易，衝動進出=壞交易
- 關鍵轉念：遵守計畫的虧損是「好」交易，靠運氣的獲利是「壞」交易

## 注意事項

- FOMO和FUD是人性，不可能完全消除，只能設法管理和降低影響
- 社群媒體是FOMO的放大器，交易時建議關閉社群
- 台股當沖便利性增加衝動交易機會，更需要紀律SOP
- 交易日誌是對抗所有情緒偏誤的最基礎工具，必須持續記錄
- 克服FOMO/FUD是穩定獲利的必要條件，但非充分條件——策略本身也必須是正期望值
