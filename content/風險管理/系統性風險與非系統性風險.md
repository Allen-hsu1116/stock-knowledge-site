# 系統性風險與非系統性風險

> 投資總風險的兩大支柱——系統性風險無法分散只能靠資產配置管理，非系統性風險可以透過分散投資消除，搞清楚虧損來自哪一種才能對症下藥

## 核心概念

### 投資風險的分解公式

**總風險 = 系統性風險 + 非系統性風險**

這是現代投資組合理論的基礎。理解這個分解，是所有風險管理的起點。

### 系統性風險（Systematic Risk）

又稱**市場風險**、**不可分散風險**。影響整個金融市場或大部分資產類別的宏觀風險。

**三大來源**：

1. **市場風險（Market Risk）**：整體市場情緒或預期轉變，如全球經濟衰退恐懼觸發大規模拋售
2. **利率風險（Interest Rate Risk）**：央行利率調整直接影響企業借貸成本、債券價格及股票估值。加息週期對高負債企業及成長股構成壓力
3. **通脹風險（Inflation Risk）**：通貨膨脹蠶食實質回報，可能促使央行收緊貨幣政策，引發利率風險

**關鍵特徵**：
- 無法透過分散投資消除
- 2008年金融海嘯是經典案例——幾乎所有股票都下跌，只有跌多跌少的差異
- COVID-19全球大流行也是系統性風險
- 用**Beta值**衡量

### 非系統性風險（Unsystematic Risk）

又稱**特定風險**、**可分散風險**。僅影響單一公司、特定行業或個別資產的風險。

**四種常見類型**：

1. **經營風險**：管理層決策失誤、新產品失敗、供應鏈中斷
2. **財務風險**：過度依賴債務融資，資金鏈緊張或違約
3. **行業風險**：新技術顛覆行業（如電動車對傳統燃油車）
4. **法律及監管風險**：政府針對特定行業頒布新法規

**關鍵特徵**：
- 可以透過分散投資顯著降低甚至消除
- 持有20-30檔不同行業股票就能大幅稀釋個股風險
- 用**基本面分析**衡量

## 實戰應用

### 衡量風險的三個指標

| 指標 | 衡量什麼 | 解讀 |
|------|---------|------|
| **夏普率** | 總風險（系統+非系統） | 承受1%風險得到多少報酬，股市長期約0.4-0.6 |
| **Beta值(β)** | 系統性風險 | β=1與市場同步，β>1比市場更波動，β<1比市場穩健 |
| **Alpha值(α)** | 非系統性風險的超額報酬 | Alpha越高越好，代表你在系統風險外額外創造的報酬 |

### Beta值實戰判讀

- **β > 1.5**：高波動成長股，牛市大賺熊市大賠
- **β ≈ 1**：與大盤同步，如0050
- **β < 1**：防禦型標的，如公用事業、電信股
- **β < 0**：與大盤反向，如黃金、反向ETF

### 管理系統性風險的兩大方法

**方法一：資產配置**

將資金分配到低相關性的不同資產類別：

| 資產 | 系統性風險特性 | 配置角色 |
|------|---------------|---------|
| 股票 | 高Beta，系統風險最大 | 報酬引擎 |
| 公債 | 負相關股票，避風港 | 風險緩衝 |
| 黃金 | 通脹對沖，低相關 | 系統風險保險 |
| REITs | 抗通脹，穩定現金流 | 分散來源 |
| 現金 | 零Beta，隨時可部署 | 選項價值 |

**方法二：對沖工具**

- 買入Put Option：支付權利金換取下跌保護
- 買入反向ETF：大盤下跌時反向ETF上漲
- 沽出指數期貨：鎖定目前部位價值

### 管理非系統性風險的方法

1. **跨行業分散**：至少配置5個以上不同行業
2. **跨地域分散**：台股+美股+其他市場
3. **深入基本面研究**：避開地雷股比分散投資更有效

### 系統性 vs 非系統性：虧損時怎麼判斷？

| 虧損情境 | 風險來源 | 應對方式 |
|----------|---------|---------|
| 大盤跌5%，你的股票跌8% | 系統性為主+非系統性 | 檢查Beta，看是否超額下跌 |
| 大盤漲2%，你的股票跌10% | 純非系統性 | 檢查個股基本面，可能是選股問題 |
| 大盤跌20%，你分散持股也跌18% | 純系統性 | 這不是錯誤，是必然要承受的 |
| 大盤跌3%，你的股票跌15% | 非系統性為主 | 檢查個股消息面和籌碼面 |

## 注意事項

1. **分散投資不能消除系統性風險**：最常見的迷思是「買很多檔股票就不會虧」，但金融海嘯時所有股票都跌，分散投資只能降低非系統性風險

2. **承擔風險不一定是壞事**：風險跟報酬相依，想要好報酬就要承擔合理的風險。真正的風險來自超過承受能力和不知道自己在做什麼

3. **Beta值的陷阱**：Beta是後視鏡指標，過去的Beta不代表未來的Beta。不同市場環境下Beta會改變

4. **Alpha可能只是運氣**：短期Alpha可能是運氣而非選股能力，至少需要3-5年數據才能判斷是否有真正的超額報酬能力

5. **資產配置的相關性會變**：危機時所有資產相關性趨向1，原本低相關的資產可能同時下跌。2008年股債雙跌就是案例

6. **對沖有成本**：買Put要付權利金，反向ETF有追蹤誤差，期貨有保證金成本。對沖不是免費的保險

7. **私營信貸市場是新興系統性風險觀察點**：低利率時期急速膨脹的私營信貸市場，信貸質素和流動性風險值得關注
