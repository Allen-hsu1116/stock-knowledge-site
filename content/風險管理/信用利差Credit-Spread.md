# 信用利差Credit Spread

> 不同信用等級債券之間的殖利率差額，是市場對違約風險的即時定價，也是股市投資人的系統性風險領先指標。

## 核心概念

信用利差（Credit Spread / Yield Spread）是指兩個不同信用品質但到期日相近的投資工具之間的殖利率差額。這個差額本質上是風險溢價——投資人承擔較高違約風險所要求的額外補償。

### 計算方式

- **基本公式**：Credit Spread = YTM(風險債券) - YTM(無風險債券)
- 例：10年期公債殖利率 2%，同天期BBB級公司債殖利率 4%，信用利差 = 2%

### 衡量方法

| 方法 | 說明 |
|------|------|
| I-Spread（插值利差） | 債券殖利率減去插值後的公債殖利率曲線 |
| Z-Spread（零波動利差） | 加在整條零息曲線上使債券現值等於市價的固定利差 |
| OAS（選擇權調整利差） | 扣除內含選擇權價值後的利差，適用含權債券 |
| TED Spread | 3月期歐洲美元利率 - 3月期國庫券利率，銀行間信用風險指標 |
| Baa-Treasury Spread | Baa級公司債 - 10年期公債，整體市場信用風險指標 |

### 信用利差曲線

信用利差與到期日的關係構成「信用利差曲線」，通常正斜率（長天期利差較大），因為長期不確定性更高。曲線形狀變化也蘊含訊息：

- **陡峭化**：市場預期長期經濟不確定性上升
- **平坦化/反轉**：短期壓力急劇上升，可能是危機前兆
- **全面擴大**：系統性恐慌信號

## 實戰應用

### 信用利差作為股市領先指標

信用利差是股市投資人最應關注的宏觀指標之一，因為它反映市場對未來違約的預期：

1. **利差收窄 → 風險偏好上升 → 股市偏多**
   - 市場願意承擔更多風險，資金流向高收益資產
   - 股市通常處於上升趨勢

2. **利差擴大 → 違約預期上升 → 股市偏空**
   - 資金逃向安全資產（Flight to Quality）
   - 股市可能面臨壓力或修正

3. **利差急速飆升 → 系統性風險爆發**
   - 2008年9月：TED Spread從1%飆至4.6%，雷曼兄弟破產
   - 2020年3月：COVID恐慌，投資級利差從1.1%飆至4.1%
   - 這種急升通常是股市底部附近的恐慌訊號

### 台股投資人的信用利差實戰

1. **追蹤TED Spread**
   - 正常範圍：0.2%-0.5%
   - 超過1%：銀行間信任下降，關注
   - 超過2%：系統性壓力，考慮減碼
   - 超過3%：危機模式，全面避險

2. **追蹤美國高收益債利差（HY Spread）**
   - 正常範圍：3%-5%
   - 超過7%：信用環境惡化
   - 超過10%：恐慌性拋售，往往是逢低布局的時機

3. **台灣50與信用利差的關係**
   - 外資買賣超與全球信用利差高度相關
   - 利差擴大 → 外資撤離台股 → 台幣貶值
   - 利差收窄 → 外資回流 → 台幣升值

4. **信用利差 vs VIX雙重確認**
   - 兩者同時上升：系統性恐慌確認
   - VIX升但利差穩定：短期波動，非系統性
   - 利差升但VIX穩定：信用風險正在累積，未來可能爆發

### 信用利差的產業判讀

不同產業的信用利差反映其產業特定風險：

- **金融業**：信用利差直接反映銀行資產品質，利差擴大意味壞帳風險
- **營建業**：利差反映房市景氣，擴大意味市場預期降價
- **科技業**：利差反映訂單能見度，擴大意味市場質疑成長持續性

## 注意事項

1. **信用利差反映的是「預期」，不是「現狀」**：利差擴大時，往往實際違約尚未發生。等違約真的爆了，利差可能已經見頂。

2. **流動性溢價混入信用利差**：信用利差不純粹是違約風險的定價，還包含流動性溢價。恐慌時流動性溢價暴增，使信用利差誇大實際違約風險。

3. **中央銀行扭曲信用利差**：QE直接購買公司債壓縮了信用利差，使利差信號失真。2020年Fed購買公司債後，利差快速收窄並非因為經濟好轉，而是央行直接干預。

4. **信用利差的均值回歸特性**：極端擴大的利差最終會回歸正常，但時間可能很長（數月到數年），不宜盲目抄底。

5. **低利差不等於低風險**：利差極度收窄時，市場可能過度樂觀（Minsky時刻的醞釀期），風險定價不足反而危險。

6. **台股缺乏本土信用利差基準**：台灣公司債市場規模小、流動性低，難以建立可靠的本土信用利差曲線。建議追蹤美國市場指標（TED、HY Spread）作為全球風險偏好代理。
