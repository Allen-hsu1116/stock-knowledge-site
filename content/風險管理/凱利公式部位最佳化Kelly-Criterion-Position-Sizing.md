# 凱利公式部位最佳化Kelly-Criterion-Position-Sizing

> 計算最佳下注比例的數學公式——贏時加碼、輸時減碼，但交易不是賭場

## 核心概念

凱利公式（Kelly Criterion）由物理學家 John Larry Kelly 於1956年在 AT&T 貝爾實驗室提出，最初是為了解決信息傳輸中的噪音問題，後來被 Ed Thorp 應用到賭局和投資理論聲名大噪。核心目標是**最大化長期財富的幾何成長率**，即最大化 E[log(W)]。

### 基本公式（全有或全無）

當虧損意味著損失全部下注金額時：

**f* = (bp - q) / b = p - q/b**

- f* = 下注金額占總資金的比例
- p = 勝率
- q = 虧損機率（= 1 - p）
- b = 賠率（獲利金額 ÷ 虧損金額，即盈虧比）

### 一般形式（部分損失）

交易中更常見的情況是虧損不是全賠，而是部分損失：

**f* = (p × b - q × a) / (b × a)**

或等價地：

**f* = p/a - q/b**

- a = 虧損時的損失率（例如下跌10%，a = 0.10）
- b = 獲利時的報酬率（例如上漲10%，b = 0.10）

這讓凱利公式可以直接應用於股票和期貨交易。

### 連續形式（股票市場應用）

對於連續報酬率的投資組合：

**f* = (μ - r) / σ²**

- μ = 期望報酬率
- r = 無風險利率
- σ² = 報酬率變異數

Ed Thorp 估計 S&P 500 的 Kelly fraction 約為 117%，但實務建議使用 Half Kelly（約58%），因為尾部風險和估計誤差。

### 計算範例

**範例1：簡單賭局**
勝率60%，賠率1:1（b=1）
f = (0.6 × 2 - 1) / 1 = 0.2 = **20%**

**範例2：期貨交易策略**
回測100次交易，勝率45%，平均獲利200元、平均虧損100元，盈虧比2
f = (0.45 × 3 - 1) / 2 = 0.175 = **17.5%**

**範例3：股票部位計算（一般形式）**
勝率55%，上漲時報酬15%（b=0.15），下跌時損失8%（a=0.08）
f* = (0.55 × 0.15 - 0.45 × 0.08) / (0.15 × 0.08) = (0.0825 - 0.036) / 0.012 = 3.875
→ Kelly 建議 387.5% 的資金，理論上要用槓桿。Half Kelly = 193.75%

**範例4：求最低盈虧比**
勝率45%，f=0
0 = (0.45(b+1) - 1) / b → b = 1.22
**盈虧比至少要1.22才不會長期虧損**

## 實戰應用

### 從凱利公式到部位控制

凱利公式告訴你理論上最理想的下注比例，但實戰中必須大幅打折：

| 方式 | 下注比例 | 成長率 | 波動 |
|------|---------|--------|------|
| Full Kelly | f × 100% | 100%（理論最大） | 極大 |
| Half Kelly | f × 50% | 75% | 降低約75% |
| Quarter Kelly | f × 25% | 50% | 大幅降低 |
| 固定比例法 | 帳戶1-2% | 視策略而定 | 最低 |

### Half Kelly 為何是最佳選擇

- Full Kelly 的波動太大，帳戶回撤可達50%以上
- Half Kelly 只犧牲25%的長期成長率，卻大幅降低波動
- Quarter Kelly 更保守，但長期成長率只剩 Full Kelly 的50%
- **成長率是凹函數**：下注比例從 Kelly 減少，成長率下降的速度比線性慢

### 反向求最低勝率或盈虧比

固定盈虧比求最低勝率：
**p > 1 / (b+1)**

固定勝率求最低盈虧比：
**b > (1-p) / p**

這兩個公式可以快速判斷一個策略是否值得交易。

### 應用於期貨交易

1. 先回測策略取得勝率和盈虧比
2. 代入凱利公式計算理論部位比例
3. 打折至 Half Kelly 或更低
4. 結合[[交易期望值Trading-Expectancy]]驗證策略正期望值
5. 用[[蒙地卡羅模擬交易驗證Monte-Carlo-Simulation]]測試不同部位的結果

### Kelly vs Optimal f

Kelly 公式和 Ralph Vince 的 Optimal f 看起來相似但有根本差異：

| 比較項目 | Kelly Criterion | Optimal f (Vince) |
|---------|---------------|-------------------|
| 損失定義 | 固定比例損失 | 歷史最大單筆損失 |
| 計算基礎 | 機率分布已知 | 歷史交易資料 |
| 輸入參數 | 勝率+盈虧比 | 交易淨利淨損序列 |
| 適用場景 | 理論最適比例 | 實際交易最適比例 |
| 破產考量 | 不直接考慮 | 以最大虧損為基礎 |

**Optimal f** 考慮了歷史最壞情況，比 Kelly 更保守。實務上兩者都應打折使用。

### Kelly 在投資組合的應用

凱利公式可以擴展到多資產組合：
- 單一資產 Kelly：f = (μ - r) / σ²
- 多資產 Kelly：需要計算共變異數矩陣
- 實務問題：估計誤差會被放大（Garbage in, garbage out）
- 解決方案：使用收縮估計（shrinkage estimation）降低極端權重

## 注意事項

### 凱利公式的四大限制

1. **假設資金可無限分割**：實際有最小交易單位（一口期貨、一張股票），連續虧損可能讓本金低到無法交易
2. **勝率和盈虧比不是固定值**：交易不是賭場，每次的參數都不同，回測績效不代表未來
3. **策略穩定性決定可用性**：如果不同回測區間的參數標準差很大，凱利公式求出的比例差異極大，參考價值有限
4. **參數估計誤差被放大**：Kelly 對勝率極度敏感，勝率估計差1%可能導致部位差5-10%

### 實驗證據的警醒

在一個受控實驗中（60%勝率的硬幣，even-money odds）：
- 28% 的受試者破產
- 平均報酬僅 $91（理論可達 $250）
- 18/61 人在某次全部押注
- Kelly 最佳策略：每次下注20%，幾何成長率2.034%/round

**人類天生不擅長部位控制**，這是凱利公式最有價值的教訓。

### 實戰守則

- **永遠不要用 Full Kelly**：理論最佳但實務致命
- **Half Kelly 是起點，不是終點**：根據策略穩定性再打折
- **勝率來自回測，回測不等於未來**：要加上安全邊際
- **虧損時自動縮小部位**：這才是凱利公式的核心精神
- **搭配蒙地卡羅模擬**：確認不同部位的破產機率
- **交易策略穩定性比精確計算更重要**：不穩定的策略用什麼公式都沒用
