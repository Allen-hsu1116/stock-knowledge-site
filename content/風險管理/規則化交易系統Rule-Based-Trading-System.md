# 規則化交易系統 Rule-Based Trading System

> 用明確規則取代主觀判斷，把交易從「感覺」變成「系統」——五步建立可重複執行的交易計畫。

## 核心概念

規則化交易系統（Rule-Based Trading System）是對所有交易環節都有嚴格定義規則的系統，交易者不允許偏離規則。基本規則涵蓋：進場條件、部位大小、最大交易風險、交易管理、最大組合風險、出場條件。

三種類型的規則化系統：
- **全自動策略**：程式交易，零人為介入
- **手動管理策略**：人為執行但規則明確
- **混合型**：部分自動部分手動（增量自動化）

### 為什麼需要規則化？

主觀交易最大的敵人不是市場，是**自己**。沒有明確規則時：
- 進場靠感覺 → 容易FOMO追高
- 出場靠心情 → 虧了不捨得出、賺了太早跑
- 部位靠衝動 → 連虧後加倍攤平
- 風控靠意志力 → 意志力會耗盡

規則化交易不是消除所有情緒，而是**建立讓犯規變困難的機制**。

## 實戰應用

### Step 1：了解什麼系統適合你的個性

**目標設定**：
- 你想翻倍小帳戶？還是想在較大帳戶上穩定獲利？
- 目標決定系統類型。30%年報酬率加上合理風險，比追求10,000%更實際。

**回撤承受度**：
- 40%回撤在模擬帳戶看起來沒什麼，但真金白銀時心理完全不同。
- 用小額真實帳戶測試你的實際承受度，你會發現自己能承受的比預期高或低。

**交易頻率**：
- 有些策略一個月只交易幾次但勝率高。有些人會無聊到犯規。
- 有些策略一天交易多次。有些人會壓力大到崩潰。
- 找到適合自己的頻率，不是每個人都一樣。

**勝率偏好**：
- 高勝率低賠率（75%勝率，1R獲利）vs 低勝率高賠率（20%勝率，100R獲利）
- 兩者都能獲利，選你能**心理承受**的那個。連續8次虧損你能忍受嗎？

### Step 2：建立書面交易計畫

交易計畫必備要素：

| 項目 | 說明 |
|------|------|
| 系統名稱 | 給你的系統取個名字 |
| 版本號 | 追蹤修改歷史 |
| 指標與設定 | 用的指標、參數值 |
| 進場條件 | 明確的進場規則 |
| 交易管理規則 | 持倉中的管理方式 |
| 風險規則 | 單筆風險、日虧損上限 |
| 重新進場規則 | 出場後何時可以再進場 |
| 組合風險規則 | 總體風險限制 |
| 執行方式 | 全自動/手動/混合 |

### Step 3：回測交易計畫

回測提供關鍵數據：
- 最大回撤（MDD）
- 勝率
- 最長連勝/連虧
- 最佳/最差交易日
- 最佳/最差交易時段

**關鍵原則**：每次只改一個參數，這樣才知道哪個改動有效。不要一次改多個。

### Step 4：前進測試（Beta Test）

回測通過後，不要直接上線。用模擬帳戶在目標券商環境測試幾個月：
- 券商的點差可能影響策略表現
- 伺服器延遲可能導致滑價
- 模擬環境和真實環境的差異可能很顯著

### Step 5：上線交易

- 從小帳戶或較小風險開始
- 用交易日誌追蹤每筆交易，每週檢討
- **至少100筆交易後再考慮修改系統**
- 初期虧損可能是正常回撤，給系統時間展現邊際優勢

## 注意事項

- **規則化≠消除情緒**：非常嚴格的系統可能反而讓你錯過一些機會，這是取捨。
- **個性匹配很重要**：不適合你個性的系統，再賺錢你也會放棄。低頻率策略對需要行動感的人是折磨。
- **回測≠未來保證**：回測績效通常有30-50%的衰退率，做好心理準備。
- **100筆交易原則**：上線後至少100筆交易再評估是否修改，避免過早放棄正常回撤中的系統。
- **漸進自動化**：不一定要一步到位全自動。可以從手動執行開始，逐步將可自動化的部分交給程式。
