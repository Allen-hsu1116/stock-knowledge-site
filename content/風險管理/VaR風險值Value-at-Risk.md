# VaR風險值Value-at-Risk

> 在給定置信水平和持有期間下，資產或投資組合可能遭受的最大損失——風險管理的通用語言

## 核心概念

**VaR（Value at Risk，風險值）** 是國際金融界測量市場風險的主流方法，1993年由G30集團在《衍生產品的實踐和規則》報告中正式提出，隨後J.P. Morgan推出Risk Metrics風險控制模型，被全球超過1000家金融機構採用。

### 定義

在給定置信水平（如95%）和持有期間（如1天）下，某一金融資產或投資組合的最大可能損失金額。

**公式**：P(ΔP ≤ VaR) = α

| 符號 | 含義 |
|------|------|
| ΔP | 持有期內的價值損失額 |
| VaR | 在險價值（可能損失上限） |
| α | 置信水平（如95%、99%） |

**實例**：某投資組合1天期、95%置信水平的VaR為520萬元
→ 有95%的把握明天損失不超過520萬元
→ 只有5%的機率損失超過520萬元
→ 平均每20個交易日出現一次超過VaR的損失

### VaR三要素

1. **持有期（Δt）**：衡量風險的時間跨度
   - 日內交易 → 1天
   - 波段交易 → 1週或1個月
   - 巴塞爾委員會 → 10個營業日
   
2. **置信水平（α）**：風險厭惡程度的體現
   - 保守（99%）→ 巴塞爾委員會要求
   - 中性（95%）→ J.P. Morgan、美洲銀行
   - 積極（90%）→ 某些避險基金
   
3. **觀察期間**：歷史數據的時間範圍
   - 太短 → 數據不足
   - 太長 → 市場結構已變
   - 巴塞爾要求 → 1年

## 實戰應用

### 1. VaR計算三種方法

| 方法 | 原理 | 優點 | 缺點 |
|------|------|------|------|
| **變異數-共變異數法** | 假設報酬率服從常態分配 | 計算快速、直觀 | 忽略肥尾效應、低估極端風險 |
| **歷史模擬法** | 用歷史報酬率排序取分位數 | 不需假設分配型態 | 歷史未必重複、需要大量數據 |
| **蒙地卡羅模擬法** | 隨機模擬數千種情境 | 最靈活、可處理非線性 | 計算量大、模型假設影響結果 |

**台股實戰建議**：個人投資者使用歷史模擬法最實際——取過去250個交易日的日報酬率，排序後取第5百分位（95% VaR）或第1百分位（99% VaR）。

### 2. VaR用於部位控制

**單一股票VaR計算**：
```
VaR = 持股金額 × 日報酬率標準差 × Z分數
例：持有台積電100萬元
    日標準差 = 2%
    95% VaR = 100萬 × 2% × 1.645 = 3.29萬
```

**投資組合VaR**：
- 不能簡單相加（因為相關性）
- 需要考慮股票間的相關係數
- 分散投資降低組合VaR

### 3. VaR用於停損設定

- 95%日VaR → 常規停損參考
- 99%日VaR → 緊急停損參考
- 超過99% VaR的損失 → 黑天鵝事件，需要額外保護機制

### 4. VaR用於期貨保證金

台灣期貨交易所保證金計算：
```
結算保證金 = 指數 × 每點價值 × 風險價格係數
風險價格係數 = Max{VaRmax, 5%}
```
其中VaRmax取30/60/90/180個交易日四個樣本群中最大者。

### 5. 條件VaR（CVaR / Expected Shortfall）

VaR只告訴你「損失超過這個值的機率是5%」，但沒說超過之後會虧多少。

**CVaR = 超過VaR的所有損失的平均值**

- 95% VaR = 3.29萬 → 「95%的機率虧不超過3.29萬」
- 95% CVaR = 5.8萬 → 「如果真的虧超過3.29萬，平均會虧5.8萬」

CVaR比VaR更保守，對尾部風險的衡量更準確。

## 注意事項

### VaR的四大陷阱

1. **常態分配假設陷阱**：金融市場報酬率有肥尾（fat tails）和偏態（skewness），常態分配嚴重低估極端事件。2008年金融海嘯時，許多銀行的VaR模型完全失靈。

2. **歷史不一定重複**：VaR基於歷史數據計算，但「過去的風險」≠「未來的風險」。市場結構性變化（法規改、新產品上市）會讓歷史VaR失效。

3. **VaR不告訴你極端損失有多大**：95% VaR = 3.29萬，但那5%的極端情況可能虧30萬。這就是為什麼需要搭配CVaR和[[策略壓力測試Stress-Testing]]。

4. **相關性不穩定**：危機時所有資產相關性趨近1（全部一起跌），分散投資的VaR計算在危機中失準。這和[[相關性風險Correlation-Risk]]密切相關。

### VaR的Nassim Taleb批判

《黑天鵝》作者Taleb稱VaR的常態分配假設為「騙術」（charlatanism），認為：
- VaR給人虛假的安全感
- 真正的風險來自VaR無法衡量的尾部事件
- 應該用壓力測試和情境分析取代單純的VaR

### 台股使用VaR的實際建議

- **不要只看一個數字**：95%和99% VaR都算，搭配CVaR
- **定期更新參數**：至少每季重新計算，重大事件後立即更新
- **搭配其他風險指標**：[[風險調整報酬指標夏普比率與索提諾比率]]、[[MDD最大回撤計算與恢復難度]]、[[策略壓力測試Stress-Testing]]
- **注意流動性風險**：台股中小型股流動性差，VaR假設可以即時出場可能不成立

### 變異數-共變異數法計算實作（TEJ實例）

以台股四檔（儒鴻1476、台積電2330、國泰金2882、長榮2603）組成投資組合，使用變異數-共變異數法計算99% VaR：

**Step 1：計算各標的每日盈餘風險（DEAR）**
- 相對VaR = |−2.33| × 日標準差 × 持股價值
- 絕對VaR = (|−2.33| × 日標準差 − 日均值) × 持股價值
- 使用 abs(−2.33) 是因為風險值考量的是下跌風險，用負數取絕對值表達正數

**Step 2：計算標的間相關係數**
- 必須使用報酬率資料，不能用價格資料（否則假性迴歸導致錯誤相關係數）

**Step 3：計算投資組合VaR**
- part1 = Σ(個別標的VaR²)
- part2 = 2 × ρ_ij × VaR_i × VaR_j（所有配對交叉項）
- 組合VaR = √(part1 + part2)

**結果**：99%信心水準，相對VaR = 52,206元，絕對VaR = 50,648元

**厚尾驗證**：四檔標的報酬率分配圖顯示，長榮厚尾最嚴重，台積電與儒鴻也有明顯厚尾，國泰金較不明顯——印證變異數-共變異數法會低估航運股風險。
