# CVaR條件風險價值 Conditional Value at Risk

> VaR只告訴你最壞情況的門檻，CVaR告訴你跨過門檻後會賠多少——真正衡量尾部風險的一致性風險測度

## 核心概念

CVaR（Conditional Value at Risk），又稱Expected Shortfall（ES），是VaR的進階版本。VaR告訴你「在95%信心水準下，最大損失不超過X」，但對於那剩餘5%的極端情境，VaR完全沉默。CVaR回答的是：**如果損失真的超過VaR，平均會賠多少？**

數學定義：

```
CVaRβ = E(-X | -X ≥ VaRβ)
```

即：在損失超過VaRβ的條件下，損失的條件期望值。

### VaR vs CVaR 核心差異

| 比較項目 | VaR | CVaR |
|----------|-----|------|
| 衡量內容 | 損失分佈的百分位數 | 超過百分位數後的平均損失 |
| 尾部風險 | 完全忽略 | 完整捕捉 |
| 一致性測度 | ❌ 不滿足次可加性 | ✅ 滿足一致性公理四大性質 |
| 凸性 | ❌ 非凸，優化可能無解 | ✅ 凸性，優化保證有解 |
| 可操縱性 | 可被操縱（改變尾部形狀） | 不易被操縱 |
| 分佈假設 | 依賴常態分佈假設 | 不依賴特定分佈 |

### 一致性風險測度四大性質

CVaR滿足Artzner等人（1999）提出的一致性公理：

1. **平移不變性**：Cβ(Y + c) = Cβ(Y) + c，加入無風險資產c，風險等量變化
2. **正齊次性**：Cβ(cY) = cCβ(Y)，部位放大c倍，風險放大c倍
3. **單調性**：Y1 ≤ Y2 → Cβ(Y1) ≤ Cβ(Y2)，損失越小風險越小
4. **次可加性**：Cβ(Y1 + Y2) ≤ Cβ(Y1) + Cβ(Y2)，**分散投資一定降低風險**

VaR不滿足次可加性，這意味著用VaR計量時，投資組合的風險可能大於各資產風險之和——違背了分散投資的基本邏輯。

## 實戰應用

### 1. 投資組合優化
經典馬柯威茨模型以最小方差為目標，但標準差對稱衡量波動，不區分上行和下行。用CVaR替代方差做約束，可以直接控制「最壞5%情境下平均賠多少」：

- 收益固定 → 最小化CVaR
- CVaR固定 → 最大化收益
- 兩者同時優化 → 有效邊界的CVaR版本

### 2. 設定風險限額
金融機構用CVaR設定交易員或部門的風險限額，優點：
- CVaR限額考慮了組合的風險分散效果
- 可設置層次性限額結構（總限額→部門限額→交易員限額）
- 不會像VaR那樣被交易員「鑽空子」操縱尾部

### 3. 風險資本計提
CVaR直接衡量極端損失的平均金額，比VaR更適合用來決定需要多少風險資本來覆蓋潛在虧損。Basel III已逐步轉向ES（即CVaR）框架。

### 4. 台股實務計算範例

假設台積電過去250個交易日的日報酬率排序後：
- 95% VaR = -2.5%（倒數第13個值）
- 95% CVaR = 最差12個日報酬率的平均值 = -4.2%

這意味著：如果今天真的跌超過2.5%，平均會跌4.2%——CVaR告訴你VaR沒說的那段故事。

## 注意事項

### 1. CVaR依賴尾部估計的穩定性
CVaR的值高度依賴損失分佈尾部的估計，而尾部數據本來就稀少。歷史模擬法中，最極端的幾個觀測值對CVaR影響很大，更換樣本期間可能導致CVaR大幅變動。建議搭配極值理論（EVT）的POT模型提高尾部估計穩定性。

### 2. 不與高階隨機占優一致
CVaR不滿足三階及以上的隨機占優，這意味著對於偏好偏態的投資者，CVaR可能無法完全捕捉風險特徵。

### 3. 事後檢驗困難
VaR的事後檢驗只需觀察超過VaR的次數是否接近預期（如95% VaR應約5%時間被突破），但CVaR的事後檢驗需要評估「突破後的平均損失」，數據更少、統計檢定力更低。

### 4. 計算方法選擇
三種主要計算方法各有優劣：
- **歷史模擬法**：簡單直觀，但受樣本期間影響大
- **參數法**：假設特定分佈（常態、t分佈），尾部形狀可能不符
- **蒙地卡羅模擬**：最靈活但計算成本最高

### 5. VaR不滿足次可加性的實際後果
當兩個資產的損失分佈具有厚尾特徵且相關性在極端情境下增強時，VaR可能出現：VaR(A+B) > VaR(A) + VaR(B)，這意味著分散投資反而「增加風險」——這顯然違背常識，是VaR的根本缺陷。
