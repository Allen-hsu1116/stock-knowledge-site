# 模型風險 Model Risk

> 你以為模型是真理，模型只是你對真相最好的猜測——而猜錯的代價可能比你想的大得多

## 核心概念

模型風險是指因使用不正確或不當的模型進行決策，而導致預期結果與實際結果產生重大偏差的風險。在交易中，任何依賴量化模型、回測結果、估值公式或風險指標的決策，都面臨模型風險。

**Rebonato的定義**：模型風險是「模型輸出與真實世界結果之間出現顯著偏差的可能性」。

### 模型風險的七大來源

1. **模型假設錯誤**：所有模型都是對現實的簡化。BSM期權定價模型假設波動率恆定、無交易成本、利率為常數——這些在現實中全都不成立
2. **數據錯誤**：垃圾進垃圾出，輸入數據有誤模型再好也沒用
3. **過度依賴模型**：把模型當聖盃，忽略模型的局限性
4. **模型外推**：在模型適用範圍之外使用（如用常態分佈預測尾部事件）
5. **參數估計誤差**：參數不穩定，歷史估計值不代表未來
6. **實施錯誤**：程式碼有bug、計算有誤
7. **模型淘汰**：市場結構變化導致原本有效的模型失效

## 實戰應用

### 散戶交易中的模型風險

散戶不用CDO定價模型，但一樣面臨模型風險：

1. **回測過度擬合**：你的交易策略回測年化30%，實盤卻虧損——這就是最常見的模型風險
2. **指標迷信**：KD黃金交叉就買、RSI超賣就進——把技術指標當模型，忘了指標只是統計
3. **均線模型失效**：均線糾結突破法在趨勢市場有效，在盤整市場連纉虧損
4. **VaR盲點**：你的「單日最大虧損1%」可能在黑天鵝事件中變成10%

### 模型風險管理五步驟

1. **模型驗模型驗證**：獨立驗證模型假設是否合理，不在適用範圍外使用
2. **樣本外測試**：回測用一半數據，另一半驗證——過度擬合的模型在樣本外必崩
3. **壓力測試**：假設最壞情境（如2008金融海嘯等級），看模型輸出是否合理
4. **模型備案**：永遠準備Plan B，當主模型失效時知道怎麼應對
5. **人為覆核**：模型是工具不是決策者，最終判斷必須有人類介入

### Rebonato的模型風險三原則

1. **所有模型都是錯的，但有些是有用的**——George Box
2. **知道模型哪裡錯，比知道模型哪裡對更重要**
3. **模型的價值不在於精確預測，而在於結構化思考**

### 回測模型風險自檢表

- [ ] 回測績效衰退率是否在30-50%以內？
- [ ] 是否做了樣本外測試？
- [ ] 是否做了前進分析（Walk Forward）？
- [ ] 參數是否在高原而非孤島？
- [ ] 是否測試過不同市況（趨勢/盤整/危機）？
- [ ] 是否考慮了滑價和交易成本？

## 注意事項

1. **模型風險≠模型無用**：不用模型的人照樣有模型風險——他們的模型叫「直覺」
2. **最危險的是你不知道自己在用模型**：任何交易決策背後都有一個隱含模型，沒意識到它才是最大的風險
3. **模型會集體失效**：相關性風險不只存在資產之間，也存在模型之間——很多量化策略同時失效
4. **簡單模型勝過複雜模型**：參數越少的模型，過度擬合風險越低
5. **歷史數據≠未來**：回測100年數據也不保證明天有效，市場結構會變化
6. **LTCM教訓**：諾貝爾獎得主+最先進模型+超高槓桿=災難，聰明人死於對自己模型的過度自信
