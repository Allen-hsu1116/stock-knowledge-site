# 交易紀律量化系統 Rule Adherence Score

> 把紀律從「靠意志力」變成「可量化的數字」——每週計算規則遵循率，用數據證明守規則比不守規則賺更多。

## 核心概念

交易紀律不是意志力問題，而是系統問題。多數交易教育告訴你「要更有紀律」，但這跟告訴減肥的人「少吃一點」一樣無用——它依賴意志力，而意志力是會耗盡的資源。

**決策疲勞（Decision Fatigue）**是真實存在的：一個早上紀律完美的交易者，到下午兩點經歷三筆停損後，犯規的機率大幅增加。這不是性格缺陷，是大腦運作方式。

Rule Adherence Score 系統的核心邏輯：**用系統取代意志力**。就像節食不是「少吃」，而是「廚房裡不放垃圾食物」；交易紀律不是「要守規則」，而是「建立讓犯規變困難的機制」。

### 五大組成元件

**1. 五條不可妥協的規則（5 Non-Negotiable Rules）**

每個類別一條規則，規則必須具體到另一個交易者看圖表就能判斷是否符合：

| 類別 | 範例 | 常見錯誤 |
|------|------|----------|
| 進場規則 | 價格拉回VWAP且量能確認，09:30-11:00 | 「找好setup」→ 太模糊 |
| 風險規則 | 單筆最大1%風險，日虧損上限3% | 沒有明確數字 |
| 出場規則 | 停損設在進場價-1ATR，停利2R目標 | 到時候再決定 |
| 部位規則 | 依停損距離和1%帳戶風險計算，不加碼虧損部位 | 虧了加倍攤平 |
| 行為規則 | 不在開盤前5分鐘交易，日虧損達限即停機，每日最多6筆 | 沒有行為煞車 |

**2. 進場前清單（Pre-Trade Checklist）**

30秒跑完五題檢查，任何一題答案是「否」或「情緒」就不進場：
- 這個setup符合我的進場條件嗎？
- 我已經根據風險規則計算好部位大小了嗎？
- 我的停損在下單前就設好了嗎？
- 我還在日虧損上限內嗎？
- 我進場是因為計畫還是情緒？

**3. 每筆交易標記（Tag Every Trade）**

交易結束後5分鐘內標記：「遵守規則」或「違反規則」。二元選擇，沒有灰色地帶。早30秒進場也是違規，停損移動1tick也是違規。

**4. 每週計算遵循率（Weekly Review）**

```
Rule Adherence Score = (遵守規則的交易數 / 總交易數) × 100
```

然後比較兩組的績效。典型發現：
- 遵守規則組：勝率58%，profit factor 1.7
- 違反規則組：勝率35%，profit factor 0.6

**用錢來量化犯規的代價**，比任何勵志建議都有用。

**5. 識別犯規觸發器（Discipline Triggers）**

2-3週標記後，找出犯規的共通性：
- **時間觸發**：下午1點後決策疲勞嚴重 → 設硬性收盤時間
- **序列觸發**：連續兩筆虧損後衝動進場 → 設「連虧2次強制暫停」規則
- **情緒觸發**：挫折、興奮、無聊、恐懼 → 標記情緒標籤找模式

### 遵循率評分表

| 分數 | 評級 | 意義 | 行動 |
|------|------|------|------|
| <60% | 系統問題 | 規則太複雜、太模糊、或不符合你實際交易方式 | 精簡到3條規則，達到75%再加回 |
| 60-75% | 穩步進步 | 多數時間守規則，但有明顯改善空間 | 找出最嚴重的觸發器，建一道防禦 |
| 75-85% | 強紀律 | 每個百分點提升都反映在P&L | 針對剩餘觸發器精進 |
| 85%+ | 精英紀律 | 維持比進步更重要，最大風險是自滿 | 持續每週追蹤，注意長期下滑 |

## 實戰應用

### 四週導入計畫

**第1週**：定義並標記。寫下5條規則，建立標記系統，每筆交易標記。這週不要改行為，只觀察和收集數據。

**第2週**：計算基線。週末計算第一次遵循率，比較兩組P&L。45%的起始分數很正常。

**第3週**：修一個觸發器。看前兩週「違反規則」的交易，找最常見的觸發器（時間、序列、情緒），建一道具體防禦。

**第4週**：比較。計算第3-4週的分數，和第1-2週比較。修好最嚴重觸發器後，通常能看到10-15個百分點的提升。

### 紀律是一切問題的根源

所有常見交易問題其實都是紀律問題：
- **報復性交易** = 進場規則違規（追價翻本）
- **FOMO交易** = 進場規則違規（不符合條件也進場）
- **過度交易** = 行為規則違規（超過每日交易上限）

不需要分開解決這三個問題——**提升遵循率就同時解決了全部**。

## 注意事項

- **二元誠實是關鍵**：標記「大部分遵守」的灰色地帶會摧毀整個系統。早30秒進場就是違規，停損移動1tick就是違規。
- **起始分數低很正常**：第一次追蹤的人45-60%很常見，不要因此放棄。
- **規則本身可能有问题**：如果遵循率持續低於60%，不是你不夠紀律，是規則太複雜或不符合你的交易方式。先精簡再重建。
- **85%以上要防自滿**：精英交易者最大的風險不是犯規增加，而是不知不覺慢慢下滑。
- **系統取代意志力**：不是「我要更有紀律」，而是「我的系統讓犯規變困難」。
