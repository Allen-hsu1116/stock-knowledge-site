# 稟賦效應（Endowment Effect）

> 擁有就高估——持有股票讓你覺得它比實際更值錢，這就是為什麼捨不得賣。

## 核心概念

稟賦效應（Endowment Effect）由2017年諾貝爾經濟學獎得主理查德·塞勒（Richard Thaler）於1980年提出，指**一旦擁有某項物品，人們對該物品的價值評估會顯著高於未擁有時**。

這個效應根源於**損失厭惡**：一定量的損失帶來的痛苦，遠大於等量收益帶來的快樂。因此人們對「避害」的考慮遠大於「趨利」，持有商品時索要的價格遠高於購買同樣商品時願意支付的價格。

### 經典實驗

**Thaler 咖啡杯實驗（1990）**：
- 獲得咖啡杯的學生（賣方）要價中位數：$5.25
- 沒有咖啡杯的學生（買方）出價中位數：$2.25-$2.75
- 賣方要價是買方出價的**2倍以上**
- 杯子市場 V/V' 率僅 0.20，說明交易嚴重不足

**關鍵發現**：代幣券（價值確定）市場沒有稟賦效應，V/V'=1.0。只有價值含糊的消費品才出現稟賦效應。這意味著——**投資人對不確定價值的資產，特別容易受稟賦效應影響。**

### 稟賦效應對科斯定理的挑戰

科斯定理認為交易成本為零時，產權初始分配不影響最終效率。但稟賦效應證明：即使無交易成本，產權擁有方也會高估價值，導致**交易量縮小、市場效率降低**。初始產權分配對最終結果有決定性影響。

## 實戰應用

### 1. 持股不賣的心理陷阱

投資人持有股票時會高估其價值，導致：
- **虧損股捨不得賣**：寧願承擔更大虧損也不認賠，因為「賣出」代表確認損失
- **獲利股過早賣出**：因為買入價容易錨定，稍微獲利就覺得「夠了」
- **股價越低成交量越低**：與需求曲線相悖的異常現象

### 2. 反直覺的風險偏好

稟賦效應讓人在虧損時變成**風險偏好者**：為了避免損失，反而願意承擔更大風險繼續持有。這就是為什麼：
- 散戶套牢後不認賠，反而加碼攤平
- 房市下跌時賣方不願降價，寧可空置
- 股市底部成交量最低（大家都在「等反彈」）

### 3. 克服稟賦效應的策略

| 策略 | 做法 |
|------|------|
| **假設自己沒有持股** | 問自己：「如果我現金在手，會買這檔股票嗎？」如果不會，就該賣 |
| **預設停損紀律** | 進場前設好停損，避免持有後才決定 |
| **定期再平衡** | 每月/季檢視持股比例，強制賣出超配部位 |
| **反向思考** | 把「我要賣掉這檔股」改成「我要把資金從這檔股移到更好的機會」 |
| **記錄買入理由** | 當初買的原因是否仍成立？不成立就該走 |

### 4. 賣方 vs 買方心態差異

- 當你持有股票時，你是**賣方心態**：高估價值，期待更高價格
- 當你空手時，你是**買方心態**：低估價值，想等更低價格
- 兩者之間的差距就是**稟賦效應造成的價差**
- 解法：**每次決策時，切換到觀察者視角**，想像自己既非買方也非賣方

## 注意事項

1. **稟賦效應不會隨經驗消除**：Thaler 實驗中，反覆交易也無法消除稟賦效應
2. **價值越模糊，效應越強**：股票的「正確價值」本質上就不確定，所以稟賦效應在投資中特別嚴重
3. **與處置效應互為表裡**：稟賦效應是「持有高估」，處置效應是「賺的太早賣、虧的太晚賣」，根源都是損失厭惡
4. **安於現狀偏誤**：稟賦效應的變體，人們傾向維持現狀不願改變，導致錯過更好的投資機會
5. **不要把稟賦效應跟合理持有混為一談**：如果基本面真的看好，持有是對的。問題是你需要誠實區分「理性看好」和「稟賦偏誤」
