# 策略失效偵測與策略健康監控Strategy-Failure-Detection

> 策略不是上線就永遠有效，量化偵測失效時機比猜測市場方向更重要

## 核心概念

任何交易策略都有壽命週期。策略失效不是瞬間發生的——它是一個漸進過程，從「超額報酬遞減」到「趨近隨機」到「虧損」。關鍵是：**在你虧掉大部分資金之前，用量化方法偵測到失效訊號**。

璞格交易團隊提出的 **Strategy over Strategy** 概念是核心框架：後者 Strategy 是判斷多空的指標，前者 Strategy 是管理指標的工具（Position Sizing、濾網、權益曲線管理、風險平衡）。很多交易者花所有心力在 A（指標），卻在 B（管理）用直覺執行，使得精心設計的 A 毫無用武之地。

### 策略失效的三層次

1. **方法失效**：策略本身的 Alpha 遞減（市場效率化、競爭者增加）
2. **標的物失效**：策略沒問題，但不適用於特定標的（主力操弄、流動性不足）
3. **市場失效**：黑天鵝或結構性改變讓過去統計完全失靈

## 實戰應用

### 方法一：權益曲線標準差法

在大樣本週期下，好的策略能在各個中期以上時間區段呈現類似特徵（勝率、獲利因子等）。權益曲線應呈現某個「不至於變化太大」的角度向右上方發展。

**判斷規則**：
- 計算權益曲線的均值與 1 個標準差
- 理論上僅約 16% 的日權益應在均值 - 1σ 之下
- 若短期權益數在此線下比例明顯提高 → **策略獲利能力遭壓縮或失效的特徵**

**實務操作**：
- 滾動計算過去 N 日的權益曲線均值與標準差
- 設定警戒線：均值 - 1σ（黃燈）、均值 - 2σ（紅燈）
- 跌破警戒線時降低部位，跌破紅燈時暫停策略

### 方法二：近期/長期最佳化比值法

**公式**：近期指標最佳化數值 / 長期指標最佳化數值（相同參數）

**判斷規則**：
- 比值接近 1.0 → 策略維持有效
- 比值偏離 1.0 越遠 → 策略正在退化

**多策略資金分配**：
- 三個策略比值分別為 1.1、1.2、1.3
- 前者評級最佳
- 資金分配：用偏離值的倒數加權
  - (0.1, 0.2, 0.3) → 倒數加權 → 6:3:2 比例配置

### 方法三：權益曲線斜率法（套利模型適用）

套利模型失效最容易辨別——權益曲線斜率變化。

**判斷規則**：
- 中期以上斜率呈現水平或角度降低 → 策略正在失效
- 套利模型勝率通常 60%+，失效時勝率明顯下降
- 無需留戀，轉換商品組合或增加多邊套利系統

### 方法四：夏普比率監控法

從策略評估的角度，持續監控策略的 Sharpe Ratio 變化：

- **滾動 Sharpe**：計算過去 3/6/12 個月的滾動夏普比率
- **衰退訊號**：近期 Sharpe 顯著低於歷史均值（超過 2 個標準差）
- **Deflated Sharpe Ratio (DSR)**：修正多重測試與過度搜尋偏誤，避免被回測中的虛假高 Sharpe 騙到

**業界基準**：
| Sharpe | 評價 |
|--------|------|
| < 0.5 | 策略可能無效 |
| 0.5-1.0 | 勉強可用 |
| 1.0-2.0 | 不錯 |
| 2.0-3.0 | 非常好（回測中罕見） |
| > 3.0 | 極度罕見，可能過擬合 |

### 方法五：貝氏機率法

使用貝氏方法計算「策略劣於基準的機率」：

- 設定先驗機率（策略有效的信心程度）
- 觀察實際交易結果（勝率、期望值）
- 更新後驗機率
- 當 P(策略失效 | 觀察數據) > 某閾值（如 80%），判定策略失效

### 方法六：回撤超出歷史分佈

- 計算策略歷史回撤的分位數（25th、50th、75th、90th、95th）
- 當前回撤超過 95th 分位數 → 策略可能在失效
- 區分「正常回撤」和「異常回撤」：
  - 正常回撤：在歷史分佈範圍內，繼續執行
  - 異常回撤：超出歷史經驗，啟動應急機制

## 注意事項

- **失效 ≠ 賠錢**：策略失效的結果是超額報酬趨近零，不是穩定虧損。有時賺有時賠但沒有優勢
- **緩慢失效 vs 快速失效**：多數策略失效是緩慢的，但市場結構性改變可以造成快速失效
- **不要反覆上架下架**：用直覺觀察損益就反覆修改策略，等於引入了不量化的參數
- **失效後的選擇**：
  - 暫停策略，等待市場回歸正常
  - 降低部位，用更小資金測試
  - 轉換到其他有效的策略（多策略組合的優勢）
  - 完全淘汰，開發新策略
- **多策略組合是最佳防禦**：不要只依賴單一策略，多策略 + 動態資金分配才能在策略失效時平滑過渡
- **監控頻率**：日頻監控權益曲線、週頻檢查 Sharpe、月頻回顧整體表現
- **確認是失效還是正常波動**：任何策略都有好壞時期，不要把正常回撤誤判為失效
