# 羊群效應（Herding Effect）

> 別人買你就買、別人賣你就賣——不獨立思考的代價，就是成為最後接棒的人。

## 核心概念

羊群效應（Herding Effect / Herd Behavior），又稱**從眾心理**，指個體在群體壓力下放棄獨立判斷，跟隨多數人行動的心理現象。就像羊群盲目跟隨領頭羊，全然不顧前方可能有利空或風險。

### 羊群效應的三種驅動力

1. **信息不對稱**：投資者無法獲得完全信息，傾向於觀察他人行為作為信息來源
   - 看到法人買超就跟單
   - 看到新聞推薦就進場
   - 忽略自己的研究和判斷

2. **名譽考慮**：基金經理為了維護名譽，傾向跟隨市場共識
   - 「與其與眾不同而犯錯，不如與眾相同而犯錯」
   - 這解釋了為什麼法人也會出現群體性誤判

3. **報酬結構**：委託代理問題使基金經理的報酬結構鼓勵從眾
   - 績效比較基準驅動（benchmark hugging）
   - 基金排名機制懲罰偏離共識者

### 信息瀑布（Information Cascade）

當前面足夠多的人做出相同選擇時，後續者會**忽略自己的私人信息**而跟隨前面的選擇。這就是信息瀑布——每個人都以為自己在做理性決策，但其實只是在跟風。

結果：**最初的少數幾個人的隨機選擇，可能決定了整個群體的方向。**

## 實戰應用

### 1. 識別羊群行為的五大訊號

| 訊號 | 表現 | 含義 |
|------|------|------|
| **散戶融資餘額暴增** | 融資餘額創歷史新高 | 散戶集體追漲，市場可能見頂 |
| **成交量異常放大** | 單日成交量數倍於均量 | 羊群湧入，往往是最後一波 |
| **新聞密集報導** | 媒體大幅報導某檔股票 | 大眾注意力已聚焦，利好可能已反映 |
| **親友推薦潮** | 周圍不投資的人都在聊股票 | 市場過熱的經典訊號 |
| **法人同向操作** | 三大法人同步買超或賣超 | 可能是信息瀑布，也可能是共同的基本面判斷 |

### 2. 台股常見的羊群行為模式

- **融資餘額見高點→散戶集體套牢**：2008年、2011年、2020年的融資高峰都對應市場高點
- **IPO申購熱潮**：新股上市時散戶瘋搶，上市後高機率套牢
- **當沖熱門股湧入**：當沖比例過高代表散戶羊群效應發酵
- **法人連續買超跟單**：外資連續買超5天以上，散戶開始跟進
- **社群平台股票推薦**：PTT、Dcard上的熱門推薦，常是散戶羊群行為的溫床

### 3. 反羊群策略

1. **逆向指標思維**：當散戶瘋狂買進時，考慮減碼；當散戶恐慌賣出時，考慮進場
2. **獨立研究清單**：每次交易前填寫自己的研究筆記，不要看別人的結論
3. **資金管理隔離**：預設部位大小，不要因為別人加碼就跟著加碼
4. **冷靜期制度**：看到群體行為後，強制等待24小時再做決定
5. **反向壓力測試**：假設市場共識是錯的，最壞情況會怎樣？

### 4. 羊群效應的市場影響

- **正反饋循環**：跟風買入→價格上漲→更多人跟風→價格泡沫→崩盤
- **市場效率降低**：價格偏離基本面，資源錯配
- **系統性風險增加**：集體拋售導致流動性枯竭
- **泡沫與崩盤的催化劑**：2000年網路泡沫、2008年金融危機都有羊群效應的影子

## 注意事項

1. **並非所有跟隨都是羊群行為**：法人同向操作可能是基於相同的基本面分析，不是盲目跟風
2. **逆向操作不等於正確**：反羊群≠反市場。市場共識有時是對的，不要為了逆向而逆向
3. **羊群效應有時間尺度**：短期羊群行為可能在幾天內反轉，長期趨勢則不一定
4. **信息不對稱是真實問題**：有時跟隨專業投資者是理性的，但需要驗證信息來源
5. **社群媒體放大羊群效應**：PTT、Discord、Telegram 股票群組加速了信息傳播和羊群行為
