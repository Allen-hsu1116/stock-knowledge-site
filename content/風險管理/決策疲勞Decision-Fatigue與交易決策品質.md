# 決策疲勞Decision-Fatigue與交易決策品質

> 決策疲勞是持續做決定讓意志力耗損、決策品質下降的心理現象——交易者在盤中不斷決策，正是決策疲勞的高危險群

## 核心概念

### 什麼是決策疲勞

決策疲勞（Decision Fatigue）是指持續地做決定讓人的生理和心理產生的不舒適感。一個人做的決定越多，損失的意志力、自律和自控力也就越多。

做決定的過程是一個消耗腦力的認知過程，需要不停地進行資訊比對來做出合理判斷。當可供選擇的項目太多，要比對、平衡的資訊就越多，要消耗的認知資源就越多。在腦力不足、意志力下降的情況下，人會傾向於做出折中選擇或保持現狀。

### 決策疲勞的兩種表現

1. **本能的魯莽行動**：衝動做出決定，不再權衡利弊
   - 在交易中表現為：盤中突然「看到就買」、不設停損就進場
   - 婚禮採購最後階段直接說「幫我推薦最好的」
   
2. **避而不作決定**：抵制一切涉及改變和風險的行動
   - 在交易中表現為：該停損時不動作、該進場時猶豫不決
   - 法官在傍晚做判決時傾向維持現狀（繼續關押）

### 經典實驗：法官假釋裁決

以色列研究人員分析一整年超過1100份假釋裁決，發現：
- 上午較早時間出席聽證的囚犯，**70%獲得假釋**
- 同一天晚些時候出席聽證的囚犯，**不到10%獲得假釋**
- 午休後第一個案件的核准率回到**65%**

這個規律與囚犯的族裔背景、犯了什麼罪及刑期長短都無關，**只和時間有關**——法官的決策品質隨著時間和決策次數而下降。

### 意志力耗損實驗（Baumeister）

心理學教授 Roy F. Baumeister 的經典實驗：
- 第一組：可自由選擇吃巧克力餅乾及紅蘿蔔
- 第二組：只能吃紅蘿蔔（需要意志力抵抗餅乾誘惑）
- 第三組：沒有準備任何食物

接著讓三組學生做不可能拚好的拼圖：
- 第一組和第三組：堅持約20分鐘
- 第二組：只堅持8分鐘就放棄

**結論**：第二組學生因為要耗費更多意志力去抵抗餅乾誘惑，導致後續缺乏意志力去完成拼圖任務。

### 決策疲勞的跡象

| 行為層面 | 認知層面 | 生理層面 |
|---------|---------|---------|
| 衝動購物/交易 | 無法清晰思考 | 疲勞 |
| 拖延決策 | 無法集中注意力 | 睡眠不佳 |
| 無意識避免決策 | 花大量時間做簡單決定 | 頭痛 |
| 易怒、脾氣暴躁 | 對任何選擇都不滿意 | 胃部不適 |
| 對自己感到挫敗 | | |

## 實戰應用

### 交易者的決策疲勞防禦

#### 一、交易前：減少不必要的決策

- **建立交易SOP和檢查清單**：將進場、出場條件標準化，減少盤中決策量
- **前一天晚上制訂操作計劃**：隔天只需要「執行」而非「決定」
- **固定穿著、固定流程**：如 Zuckerberg 和 Steve Jobs 穿同款衣服，把決策精力留給最重要的事
- **限制每天的重大交易決策不超過3-4個**

#### 二、交易中：高價值活動優先

- **盤前30分鐘最清醒**：用來做最重要的進場判斷
- **部位管理 > 尋找新機會**：已經有部位時，管理部位是高價值活動
- **不要在盤中想上一筆交易的檢討**：那是交易後的事
- **感覺決策疲勞時暫停**：離開電腦5-10分鐘，補充水分和血糖

#### 三、交易後：建設性自我評估

- **高素質預備 → 高效執行 → 建設性自我評估**：形成良好循環
- **不要馬上檢討**：至少過一小時再回顧
- **解離行為**：將每次交易視為獨立事件，問自己「如果現在是空手狀態，我會做這筆交易嗎？」
- **交易日誌記錄決策時的心理狀態**：是否感到疲勞？是否衝動？

#### 四、正念交易SOP

書中提供的正念SOP（STOP）：
- **S — Slow down 放慢**：在決策前刻意放慢速度
- **T — Take note 留意**：留意當下的情緒和身體狀態
- **O — Open up 開放**：以開放心態面對各種可能
- **P — Pursue High-value Actions 實踐高價值活動**：專注在真正重要的事

#### 五、決策自動化策略

| 日常決策 | 自動化做法 |
|---------|-----------|
| 穿什麼 | 固定穿搭，減少選擇 |
| 吃什麼 | 預先規劃一週菜單 |
| 何時交易 | 固定交易時段 |
| 進場條件 | 量化SOP，條件到就執行 |
| 停損停利 | 預掛條件單，不自動干預 |
| 觀察名單 | 盤前設定，盤中不另找標的 |

### 自我耗損與交易行為

心理學研究指出，自我耗損（Ego Depletion）會導致：
- 偏好即時利益和目標，忽略長期利益
- 規劃和執行能力下降
- 更容易被眼前獲利吸引，忽視風險

**交易中的具體表現**：
- 連續虧損後報復性交易（放棄長期紀律，追求即時回本）
- 盤中追高殺低（意志力耗盡後衝動決策）
- 該停損時不動作（決策逃避）
- 盤末亂買（大腦疲勞後的魯莽行動）

## 注意事項

- 決策疲勞與一般身體疲勞不同——你**不會意識到自己累了**，但心理能量已被耗損
- 現代人一天平均做出約35,000次決策，交易者盤中決策密度更高
- 決策疲勞不會因為「意志力強」就免疫——它是生理現象，只能用策略減輕
- 銀行核貸研究發現：午休前、下班前的一兩個小時，核貸率明顯下降——決策品質隨時間劣化
- 解離行為是對抗決策疲勞的重要技巧：每次問自己「如果現在空手，我會做這筆交易嗎？」
- 正面期望值的交易系統在連續虧損時，時近效應（recency effect）會讓你懷疑系統、放棄紀律——這正是決策疲勞與心理偏誤的交互作用
