# 試單策略與由小而大

> 贏時加碼、輸時減碼——用試單確認方向，用獲利加碼部位

## 核心概念

### 試單策略的核心思想

李佛摩（Jesse Livermore）的名言：「很多投機客在買進或賣出時都太衝動，幾乎所有部位都是在同一價位取得，而那是錯誤且危險的作法。」

試單策略的邏輯很簡單：**先用小部位試水溫，確認方向正確後再加碼。**

這是反馬丁格爾思維的實踐：
- **馬丁格爾**：賠了加碼（攤平）→ 贏時減碼、輸時加碼 → **破產風險極高**
- **反馬丁格爾**：贏了加碼 → 輸時減碼、贏時加碼 → **讓利潤奔跑、截斷虧損**

### 由小而大（Mark Minervini）

Minervini 的部位管理方法：把資金分為三個等級（25%、50%、100%），每個等級最大虧損25%。

| 部位等級 | 金額（以$4000為例） | 最大虧損(25%) |
|---------|-------------------|-------------|
| A (25%) | $1,000 | $250 |
| B (50%) | $2,000 | $500 |
| C (100%) | $4,000 | $1,000 |

**規則一**：總獲利 ≥ 下一級的最大虧損 → 升級部位
**規則二**：總獲利 < 當前級的最大虧損 → 降級部位

**範例流程**：
1. 起始A($1,000)，賺$500 → 升級到B
2. B($2,000)，賺$1,000 → 累計$1,500，升級到C
3. C($4,000)，賺$2,000 → 累計$3,500
4. C($4,000)，虧$1,000 → 累計$2,500，維持C
5. C($4,000)，虧$1,000 → 累計$1,500，維持C
6. C($4,000)，虧$1,000 → 累計$500，降級到B

**核心精神**：部位如果呈現虧損，實在沒有理由擴大交易規模。反之，交易順手時，可以用帳面獲利融通部位，不增添風險。

### 三一理論（陳富村）

贏三倍以上才可以加碼一倍：
- 基本$10 → 贏$30以上 → 加碼到$20
- 加碼後輸 → 立刻回到$10
- 贏$60 → 考慮加碼到$40
- 輸絕不加碼，贏了才加碼
- 每次最大下注 ≤ 全部資金的15%

### 幽靈規則二

「毫無例外並且正確的對你的獲利部位加碼。正確的交易本身實際上並不能產生豐厚的利潤。如果你真的希望自己擁有以交易謀生或賺外快的能力，你就應該在你的獲利倉位上增加籌碼。否則的話，你最多只能保本。」

## 實戰應用

### 奇正2的三刀流試單策略

- **初始買入**：先買1批
- **加碼條件**：自上次買進點漲3%加碼一次，共加碼2次（三刀流）
- **出場點**：1批時-12%，2批時-6%，3批時沒獲利就出場
- **為什麼三刀流不是六刀流**：12年回測期間，三刀流比四刀流穩定，只犧牲不到3%的獲利

### 一日二刀流原則

- 最多「一日二刀流」，第二刀必須在尾盤才出手
- 這樣不可能加碼後變成跌停（因為已經是尾盤）
- 一日漲幅超過6%不要一次加2碼，大漲後回檔機率大

### 期貨當沖高手的加碼策略

「我賠錢時永遠只賠1口，賺錢時卻是5口、20口在賺！」

1. 沒有明確趨勢前，先用一口單試單，嚴守停損
2. 趨勢明確出現，有獲利代表做對邊，開始加碼
3. 一整個波段抱下來，都是10口、20口在賺

### 程式交易高手S先生的加碼策略

「永遠把風險值賺到了，才允許進場訊號下單」

1. 總資金100萬，每筆風險1萬(1%)
2. 在8500買進一口期貨，風控50點(1萬元)
3. 行情漲到8550，未平倉獲利1萬元 = 風險值 → 允許加碼第二口
4. 第二口單幾乎等同於新單（因為口袋已有1萬獲利）
5. 停損跟著上移到8500

### Victor Sperandeo 的一致性獲利

- 一季開始，任何新部位規模都很小（相對風險資本）
- 第一筆虧損 → 新部位按比例縮小
- 第一筆獲利2,000 → 存入銀行1,000，部位增至6,000
- 這樣即使下一筆虧損，整體仍有獲利
- 12年間沒有一年虧損，平均年報酬70.7%

## 注意事項

1. **先有停損點才有進場點**（點石Ego）：如果進場前沒先有經過驗證的停損點，你算出來的部位大小毫無意義。「隨便買，賠1萬就賣」只是讓破產慢一點，不是在做正確的事。
2. **買太多不行，買太少也不行**：買太多會超過風控上限；買太少則是浪費機會。100萬只買3張（9萬），即使報酬率70%，本金只獲利6.3%。但按資控買9張（27萬），本金獲利18.9%。
3. **每筆交易金額要一致**：如果每筆金額不同，即使系統有正期望值，績效也會被運氣主宰。
4. **非對稱槓桿的殘酷**：虧10%需要賺11%回本，虧50%需要賺100%回本，虧30%需要賺43%回本。30%大約是多數人「勉強能回本」的邊際。
5. **絕對不要在虧損時加碼**：這是馬丁格爾的陷阱。贏家是「賠1口、賺5口」，不是「賠1口、加碼2口」。
