# 交易計畫書Trading-Plan

> 沒有計畫的交易是賭博——交易計畫書是你和市場之間的書面契約

## 核心概念

交易計畫書（Trading Plan）是一份書面的交易規則和流程文件，涵蓋從動機到記錄的完整框架。所有交易計畫都是個性化和獨一無二的，但都應包含七個核心要素。

### 為什麼需要交易計畫書

- 沒有計畫的交易 = 情緒驅動的賭博
- 計畫書讓你在市場波動時有依據，不靠直覺
- 交易日記是計畫書的驗證工具，持續修正才能進步

## 實戰應用

### 交易計畫書七大要素

#### 1. 動機（Why）

你為什麼要交易？
- 為了賺錢？尋求刺激？挑戰自我？
- 把動機寫下來，它是你堅持紀律的基礎
- 動機決定你的交易風格和風險承受度

#### 2. 時間投入（When）

你能投入多少時間？
- 當沖需要整天盯盤，波段可能每天30分鐘
- 包含學習、演練策略、分析市場行情的時間
- 時間不足就不要做當沖

#### 3. 交易目標（What）

使用 **SMART原則**設定目標：
- **S**pecific（具體）：明確的數字和時間
- **M**easurable（可衡量）：有客觀標準判斷是否達成
- **A**chievable（可實現）：根據歷史績效和市場條件
- **R**elevant（相關）：與你的動機和能力匹配
- **T**ime-bound（有時限）：設定截止日期

❌ 「我想快速致富」
✅ 「未來12個月將投資組合價值提高10%」

#### 4. 風險態度（How Much Risk）

- 每筆交易最大虧損：帳戶的1-2%
- 帳戶最大回撤：15-25%
- 單日最大虧損：帳戶的3-5%
- 風險態度與交易目標密切相關

#### 5. 交易資本（How Much）

- 只用可承受損失的資金
- 確保能承受每筆交易的最大潛在損失
- 資金不足先用模擬帳戶練習
- 區分生活資金和交易資金，永遠不要混用

#### 6. 交易市場（Where）

選擇市場時考慮三個因素：
- **交易時間**：市場何時開放？是否配合你的作息？
- **波動性**：平均每小時/天/周的波動幅度
- **合約規格**：每跳多少錢？最小交易單位？

選定市場後保持專注，深入了解比廣泛涉獵更重要。

#### 7. 交易日記（Record）

交易日記是計畫書的靈魂：

**記錄內容**：
- 入場原因（為什麼買？）
- 預期利潤和最大風險
- 入場點位和退出點位
- 交易期間的市場表現
- 偏離計畫的原因
- 情緒狀態

**追蹤指標**：
- 總損益
- 勝率
- 平均獲利 vs 平均虧損
- 盈虧比
- 最大連續虧損
- 特定市場/時段/策略的績效差異

### 交易計畫書範本

```
=== 我的交易計畫書 ===

【動機】
（寫下你為什麼交易）

【時間】
（每天可投入的交易和分析時間）

【目標】
（SMART格式的具體目標）

【風險規則】
- 單筆最大虧損：帳戶 ___%
- 帳戶最大回撤：___%
- 單日最大虧損：帳戶 ___%
- 連虧 ___ 筆停止交易

【資金】
- 交易資本：$___
- 最大部位：$___

【市場】
- 主要市場：___
- 交易時段：___
- 交易風格：當沖/波段/長期

【進場規則】
（明確寫下什麼條件進場）

【出場規則】
- 停損：___
- 停利：___
- 移動停利：___

【交易日記】
（記錄每筆交易的細節）
```

## 注意事項

- **計畫書必須寫下來**：腦袋裡的計畫不算計畫
- **計畫書不是裝飾品**：必須嚴格遵守，偏離時記錄原因
- **定期檢討修正**：每月檢視計畫是否需要調整
- **情緒是最大敵人**：計畫書就是幫你抵抗情緒的工具
- **模擬交易先行**：在模擬環境驗證策略至少3個月
- **從小部位開始**：實際交易先從最小部位開始
- 計畫書不是靈丹妙藥，有計畫也不保證獲利，但沒計畫幾乎保證虧損
