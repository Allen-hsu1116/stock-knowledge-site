# 認知偏誤與交易心理 Cognitive Biases in Trading

> 你的大腦是交易最大的敵人——15種認知偏誤讓你在不知不覺中做出錯誤決策，認識它們是克服的第一步

## 核心概念

行為經濟學（Behavioral Economics）認為，投資人並非完全理性，容易受到偏見影響而出現認知錯誤。行為金融（Behavioral Finance）是行為經濟學的分支，專門研究投資人不理性的金融行為。

CME知名交易者 Tom Basso 說：「如果一切重來，我會把整個順序完全顛倒過來。投資心理是最重要的一環，其次是風險管控，最後才是市場選擇。」

認知偏誤是系統性的思考錯誤，不是偶發的失誤——它會一再重複，直到你認出它為止。

## 15種交易認知偏誤

### 1. 確認偏誤 (Confirmation Bias)
選擇性收集支持自己觀點的資訊，忽略矛盾資訊。

- 持有A公司股票，只看利多消息，忽略利空
- 只追蹤看多的分析師，不看空方觀點
- **對策**：主動尋找反方論點，假設自己空手會怎麼做

### 2. 損失規避 (Loss Aversion)
失去100元的痛苦是獲得100元喜悅的2倍以上。

- 太早停利、太晚停損（處分效應）
- 繼續持有虧損部位，不願認賠
- **對策**：交易前設好停損，機械式執行；虧損是交易成本不是失敗

### 3. 處分效應 (Disposition Effect)
損失規避的直接表現：太早獲利了結、太晚停損出場。

- 賺5%就跑，虧20%還抱
- **對策**：讓利潤奔跑、截斷虧損——這是交易的核心原則

### 4. 沉沒成本 (Sunk Cost Fallacy)
已經發生無法改變的成本，不該納入決策。

- 「都已經虧這麼多了，不能現在賣」
- **對策**：永遠假設自己空手，問「如果空手會買嗎？」

### 5. 錨定偏見 (Anchoring Bias)
過度依賴第一個資訊做判斷。

- 看到今天股價，以此為基準判斷未來
- 看到前高前低就覺得是壓力支撐
- **對策**：多設幾個參考錨點，用量化數據取代感覺

### 6. 從眾行為 (Herd Mentality)
模仿多數人的行動，認為大群體不太可能出錯。

- 看到大家搶買就跟著買
- 看到分析師推薦就跟單
- **對策**：人多的地方不要去；獨立思考，逆向檢驗

### 7. 過度自信 (Overconfidence)
過於相信自己的判斷能力。

- 研究越深越相信自己是對的
- 連續幾次獲利就覺得自己找到聖盃
- **對策**：拿自己的選股績效和大盤ETF比較，保持謙遜

### 8. 控制的錯覺 (Illusion of Control)
高估自己控制事件的能力。

- 用幸運數字選股、看盤時覺得自己能影響走勢
- 基本面研究越深入越覺得自己能預測
- **對策**：市場不會因為你做了什麼就改變走勢

### 9. 賭徒謬誤 (Gambler's Fallacy)
根據過去事件結果，認為隨機事件發生的機率會改變。

- 連漲5天覺得「應該要跌了」
- 連跌3天覺得「反彈在即」
- **對策**：每個交易日相對獨立，不要用猜的；養成說「我不知道」的習慣

### 10. 近因偏誤 (Recency Bias)
偏重最近的數據或經驗，忽略長期趨勢。

- 剛經歷大跌就覺得市場永遠不會好
- 最近一筆交易大虧，從此不敢進場
- **對策**：看長期績效，不要被最近幾筆交易影響判斷

### 11. 心理帳戶 (Mental Accounting)
把客觀等價的金錢歸入不同帳戶，用不同態度對待。→ 詳見 [[心理賬戶Mental-Accounting]]

- 年終獎金比薪水更容易花掉
- 信用卡比現金更容易超支
- 交易獲利比本金更容易冒險
- **對策**：所有錢都是錢，獲利了結的錢和本金等價

### 12. 後見之明 (Hindsight Bias)
事後諸葛，誤以為事前可預測。

- 「我就知道會漲！」
- 事後看K線覺得訊號很明顯
- **對策**：交易日誌記錄當時的思考過程，事後對照

### 13. 近鄉偏誤 (Home Bias)
過度偏好本國市場，忽略全球分散投資。

- 只買台股，不碰海外市場
- **對策**：全球分散配置降低單一市場風險

### 14. 結果偏誤 (Outcome Bias)
用結果評斷決策好壞，而非評估決策過程。

- 賺了就是好決策，賠了就是壞決策
- **對策**：評估決策過程是否合理，而非只看結果

### 15. 選擇悖論 (Paradox of Choice)
選擇太多反而無法做決策。

- 1800檔股票不知道買哪檔
- 最後隨便選或乾脆不選
- **對策**：縮小選股範圍，只看自己理解的標的

## 偏誤之間的關聯

這些偏誤不是獨立存在的，它們會互相強化形成惡性循環：

```
確認偏誤 → 只看利多 → 過度自信 → 加碼
    ↓
虧損發生 → 損失規避 → 不願停損 → 沉沒成本
    ↓
從眾行為 → 跟著大家賣 → 後見之明 → 「我就知道」
```

## 實戰應用

### 偏誤辨識SOP
1. **交易前**：問自己「我現在可能受到哪種偏誤影響？」
2. **交易中**：感覺情緒波動時暫停，辨識是哪種偏誤在作祟
3. **交易後**：交易日誌記錄決策過程，事後檢驗有無偏誤

### 偏誤對抗工具箱

| 偏誤 | 對抗工具 |
|------|----------|
| 確認偏誤 | 主動找反方論點，設「反對自己的角色」 |
| 損失規避 | 交易前設停損，機械式執行 |
| 錨定偏見 | 多設參考錨點，量化數據取代感覺 |
| 從眾行為 | 獨立思考清單，逆向檢驗 |
| 過度自信 | 選股績效 vs 大盤ETF，保持謙遜 |
| 賭徒謬誤 | 不猜漲跌，接受「我不知道」 |
| 沉沒成本 | 永遠假設空手，看未來不看過去 |
| 後見之明 | 交易日誌記錄當時思考 |

### Tom Basso 交易優先順序
1. **投資心理** — 最重要的基礎
2. **風險管控** — 第二優先
3. **市場選擇** — 最後才需要操心

## 注意事項

- 認知偏誤是**系統性**的思考錯誤，不是偶發失誤，會一再重複
- 沒有人能完全消除偏誤，只能**減少**它的影響
- 辨識偏誤的第一步是**覺察**：感覺情緒波動時先暫停
- 偏誤之間會互相強化，需要整體對抗而非單一解決
- 優秀交易者10次中4次成功就算頂尖，不需要每次都賺
