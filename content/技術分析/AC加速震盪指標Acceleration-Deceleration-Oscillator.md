# AC加速震盪指標 Acceleration/Deceleration Oscillator

> AO的「加速度計」——動量改變先於價格改變，加速度改變又先於動量改變，AC是Bill Williams五維系統中比AO更領先的早期預警指標。

## 核心概念

AC（Acceleration/Deceleration Oscillator）是Bill Williams混沌交易法的第三維指標，設計邏輯基於一個簡單的物理學原理：

**價格改變方向之前，動量會先減速；動量改變之前，加速度會先改變。**

用物理學比喻：
- 價格 = 位置
- AO = 速度（位置的一階導數）
- AC = 加速度（速度的一階導數）

AC偵測的是「動量的變化率」，也就是動量正在加速還是減速。當動量開始減速，即使價格仍在創新高，AC會先轉向——這就是早期預警。

### 計算公式

```
中位價 = (最高價 + 最低價) / 2

AO = SMA(中位價, 5) - SMA(中位價, 34)

AC = AO - SMA(AO, 5)
```

- AO衡量動量方向和強弱（5期SMA vs 34期SMA的差值）
- AC = AO值減去AO的5期移動平均 = AO偏離自身均值的程度
- AC在零軸上方 = 動量正在加速上升
- AC在零軸下方 = 動量正在加速下降
- **零軸穿越本身不是交易訊號**，只是加速度平衡點

### 柱狀圖顏色規則

- 綠色柱：當前AC值 > 前一根AC值（動量正在增加）
- 紅色柱：當前AC值 < 前一根AC值（動量正在減少）

## 實戰應用

### 四種狀態判讀

| AC位置 | 柱狀圖方向 | 市場狀態 | 操作意義 |
|--------|-----------|---------|---------|
| 零軸上方 | 綠色上升 | 多頭加速 | 買方更強，順勢做多 |
| 零軸上方 | 紅色下降 | 多頭減速 | 上升力道減弱，警惕反轉 |
| 零軸下方 | 紅色下降 | 空頭加速 | 賣方更強，順勢做空 |
| 零軸下方 | 綠色上升 | 空頭減速 | 下降力道減弱，警惕反轉 |

### 交易訊號系統

#### 基本規則
- 當前柱為紅色 → **禁止買進**，只能賣出或不交易
- 當前柱為綠色 → **禁止賣出**，只能買進或不交易

#### 順勢訊號（與加速度方向一致）— 較強

**買進（AC在零軸上方）**：
1. 零軸上方出現1根紅柱 → 2根連續綠柱
2. 在第2根綠柱最高價上方1-3 tick設BuyStop
3. 若BuyStop觸發前柱狀圖轉紅 → 訊號作廢

**賣出（AC在零軸下方）**：
1. 零軸下方出現1根綠柱 → 2根連續紅柱
2. 在第2根紅柱最低價下方1-3 tick設SellStop
3. 若SellStop觸發前柱狀圖轉綠 → 訊號作廢

#### 逆勢訊號（與加速度方向相反）— 較弱，需額外確認

**買進（AC在零軸下方）**：
1. 零軸下方需要連續3根綠柱（而非2根）
2. 若前2根綠柱中有1根穿越零軸 → 第3根即可進場
3. 在訊號柱最高價上方1-3 tick設BuyStop

**賣出（AC在零軸上方）**：
1. 零軸上方需要連續3根紅柱（而非2根）
2. 若前2根紅柱中有1根穿越零軸 → 第3根即可進場
3. 在訊號柱最低價下方1-3 tick設SellStop

### 背離訊號

AC的背離出現頻率比其他震盪指標低，但一旦出現可靠性較高：
- 價格創新高 + AC未創新高 → 上升加速度減速，警惕反轉
- 價格創新低 + AC未創新低 → 下降加速度減速，警惕反轉
- 背離出現時，操作方向朝零軸突破方向，停損設在下一個局部極值後方

### 盤整辨識

AC也可辨識盤整市場：
- 柱狀圖在零軸附近小幅來回擺動（綠紅交替頻繁、幅度極小）
- 低波動期AC振幅壓縮 = 市場缺乏方向性驅動力
- 盤整期不應使用AC的趨勢訊號

### AO + AC 聯動實戰

AC是AO的衍生指標，兩者聯動是Bill Williams系統的核心應用：

1. **AO先轉向** → 動量開始改變（第一預警）
2. **AC跟進同向** → 動量加速度確認（第二確認）
3. **兩者同向** = 高勝率進場點
4. **AO轉向但AC未跟進** = 假訊號機率高，觀望

#### AO/AC加碼判讀
- AO為正 + AC連續2根綠柱 → 積極加碼做多
- AO為負 + AC連續2根紅柱 → 積極加碼做空
- 即使AC仍在零軸下方，只要出現綠柱就代表上漲加速度已啟動

### 與鱷魚線和碎形的配合

在Bill Williams五維系統中，AC的定位是**確認性質的輔助指標**，不是主要進場依據：

1. **鱷魚線**判斷趨勢方向（鱷魚睡覺 = 不交易）
2. **碎形突破**是主要進場訊號
3. **AC確認**碎形訊號的有效性：碎形突破時AC同向 = 有效訊號

### 參數調整建議

Williams原始參數(5, 34)為20多年前設計：
- 第一個參數（5）不建議改，與Elliott波浪結構和Mandelbrot碎形相關
- 第二個參數（34）可調整：活躍波動工具可嘗試(5, 20)或用EMA替代SMA
- 低於H1時間框架假訊號太多，建議H1以上使用
- 部分實戰者使用(5, 20, 3, EMA)參數組合處理高波動標的

## 注意事項

1. **AC不適合單獨使用**：必須搭配趨勢指標（鱷魚線、均線、布林通道），否則盤整市假訊號滿天飛
2. **零軸穿越不是訊號**：只影響需要幾根柱來確認（零軸同側2根，零軸異側3根）
3. **訊號取消紀律**：掛單觸發前柱狀圖變色 = 訊號作廢，沒有例外
4. **短週期假訊號多**：4小時以下時間框架假訊號比例極高，不建議使用
5. **背離稀少但可靠**：AC背離出現頻率低，但出現時勝率高，值得等待
6. **固定參數的雙面刃**：原始參數(5,34)經過最佳化但年代久遠，現代市場波動結構已不同，需依標的調整
7. **與MACD的區別**：MACD是價格的動量指標，AC是動量的動量指標（二階導數），兩者不可互相替代
