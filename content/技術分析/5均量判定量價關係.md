# 5均量判定量價關係

> 成交量＞5均量才有價格波動，成交量＜5均量就陷入停滯——用5日均量當基準，量化「有量」與「無量」，讓量價關係不再模糊

## 核心概念

### 為什麼是5均量？

每檔股票股本不同、價格不同，直接看成交張數無法互相比較。**5日均量（5MA Volume）** 提供了一致性的基準：

- **成交量 > 5均量** = 有量 = 價格會波動
- **成交量 < 5均量** = 無量 = 價格陷入停滯

這是最簡潔有力的量價定義：**有量就有價，無量就無價**。

### 核心觀念：用量去解型態，而非用型態去解量

所有價格型態（頭肩頂、W底、三角收斂等）都是**由成交量堆疊出來的**。如果只看價格型態而忽略成交量，就會迷失在型態的虛幻之中。

用量去解型態的思維：
1. 先看量的變化 → 判斷動能方向
2. 再看價格型態 → 確認走勢
3. 量先價行 → 量的訊號通常領先價格

## 實戰應用

### 1. 有量與無量的六大情境

| 區段 | 量能狀態 | 價格表現 | 實戰意義 |
|------|---------|---------|---------|
| 上升段有量 | 量 > 5均量 | 大幅上漲 | 多方發動，趨勢確認 |
| 上升段無量 | 量 < 5均量 | 小幅波動 | 動能不足，可能假突破 |
| 下跌段無量 | 量 < 5均量 | 盤整/小幅下跌 | 賣壓有限，可能是洗盤 |
| 下跌段有量 | 量 > 5均量 | 大幅下跌 | 空方發動，破勢下跌 |
| 量增有量 | 量 > 前段5均量 | 波動加大 | 進入趨勢段 |
| 量縮無量 | 量 < 前段5均量 | 波動縮小 | 進入整理或停滯 |

### 2. 爆大量的三種類型與隱藏含義

**量縮大漲股**：
- 利多消息 → 賣方惜售 → 漲停鎖死 → 買不到
- 市場看好後勢，必須用更高價格才能買進
- 解讀：真正看好者惜售，量縮鎖死是強勢

**爆量大漲股**：
- 利多消息 → 開盤爆大量 → 有人大量賣出
- 公司釋放利多，大股東/主力理應惜售，為何大量賣出？
- 解讀：**有出貨之嫌，爆量要先跑**

**爆量下跌股**：
- 長黑爆量 = 主力高檔倒貨
- 最危險的訊號，先賣再講

### 3. 高檔爆量操作法則

**核心法則：「高檔爆量，先賣再講，別問為什麼！」**

高檔爆量是主力透過市場傳達訊息，但你不知道是好消息還是壞消息，所以先賣再講。

**爆量判斷標準**：成交量 > 5均量的 2 倍

**大量K棒的相對位置判讀**：
| K棒型態 | 後續走勢 | 操作 |
|---------|---------|------|
| 長黑爆量 | 下跌 | 立即賣出 |
| 高檔爆量收上影線 | 盤整或下跌 | 先賣一趟 |
| 中低檔爆量長紅 | 上漲 | 可能是攻擊量，觀察 |
| 底部窒息量後帶量長紅 | 反轉向上 | 可能是起漲點 |

### 4. 5均量選股實戰

**選股條件**：
- 今日成交量 > 5均量（有量）
- 股價站上月線（趨勢偏多）
- 量能溫和放大（非爆量，避免追高）

**避開條件**：
- 成交量長期 < 5均量（無量殭屍股）
- 高檔爆量 > 5均量 2 倍以上（先跑再說）

### 5. 量的操作紀律

1. **用量先判斷有無趨勢**：5均量以下無趨勢，不交易
2. **量的趨勢比價的趨勢更可靠**：量先價行
3. **高檔爆量一律先出**：不管利多利空，先賣再講
4. **量縮大漲比爆量大漲更健康**：惜售代表看好
5. **下跌有量比下跌無量更危險**：有賣壓才有量

## 注意事項

- **5均量是相對基準**：不同股票的5均量絕對值不同，只能跟自己比
- **爆量判斷要搭配位置**：低檔爆量可能是攻擊量，高檔爆量一定是危險信號
- **量縮不等於止跌**：量縮只代表賣壓減弱，還需要確認買盤進場
- **不要忽略K棒型態**：量是燃料，K棒是方向，兩者缺一不可
- **主力利用爆量吸引注意**：爆量是主力讓散戶「不請自來」的手段
