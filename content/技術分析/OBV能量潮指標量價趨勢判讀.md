# OBV 能量潮指標量價趨勢判讀

> 用累計成交量判斷資金流向——價漲加量、價跌減量，OBV 領先價格反應是「量先價行」的具體指標，背離是核心訊號

## 核心概念

OBV（On Balance Volume，能量潮指標）由葛蘭維（Joe Granville）於1960年代發明，是最經典的量價指標之一。核心邏輯極簡：**價漲就把今日成交量加進去，價跌就把今日成交量扣掉，平盤就不算。** 累計起來就是 OBV 線。

### 計算公式

```
今日 OBV = 前日 OBV + 今日成交量 × sign(今日收盤價 - 昨日收盤價)
```

- 今日收盤 > 昨日收盤 → 加上今日成交量
- 今日收盤 < 昨日收盤 → 扣除今日成交量
- 今日收盤 = 昨日收盤 → OBV 不變

### 三大理論基礎

1. **分歧產生量**：投資者對股價看法越不一致，成交量越大。可用成交量判斷市場人氣和多空力量。
2. **重力原理**：股價上升需要能量（大量成交），下跌不需耗費很大能量（量可萎縮）。上升初期的放量是必要的。
3. **慣性原則**：熱門股成交量和波幅大，冷門股成交量和波幅小。動則恆動、靜則恆靜。

### OBV 與其他量價指標的差異

| 特徵 | OBV | AD（累積分配） | CMF（佳慶資金流量） |
|------|-----|----------------|---------------------|
| 漲跌判定 | 只看收盤價漲跌 | 看收盤價在日內位置 | 看收盤價在日內位置 |
| 量加權 | 全加或全扣（二元） | CLV加權（連續） | CLV加權後取移動平均 |
| 資訊量 | 最低（只有漲跌） | 中等（有位置資訊） | 最高（位置+移動平均） |
| 優勢 | 簡單直觀 | 較精細 | 最精細，有超買超賣 |

**OBV 的核心優勢**：簡單到不可能算錯，信號直觀——OBV 線的方向就是資金流向的方向。

## 實戰應用

### 一、背離訊號（最核心用法）

OBV 最強大的用法就是背離判斷：

- **頂背離**：股價創新高，OBV 未創新高 → 多頭力量耗盡，賣出訊號
- **底背離**：股價創新低，OBV 未創新低 → 空頭力量耗盡，買入訊號

背離出現時，不管當時股價是漲是跌，都隨時有反轉的可能。這是 OBV 的核心價值——**量先價行的具體體現**。

### 二、趨勢確認

- OBV 線緩慢上升 + 股價同步上漲 → 行情穩步向上，持股待漲
- OBV 線緩慢下降 + 股價同步下跌 → 行情盤跌，賣出或觀望
- OBV 線上升 + 股價下跌 → 人氣旺盛，下檔承接強，股價即將止跌回升（搶反彈機會）
- OBV 線下降 + 股價上漲 → 上升能量不足，股價隨時下跌（賣出訊號）

### 三、N 字形波動判斷

OBV 線呈 N 字形波動，Granville 提出以下規則：

1. OBV 線超越前一次 N 字形高點 → 向上箭頭（多方突破）
2. OBV 線跌破前一次 N 字形低點 → 向下箭頭（空方突破）
3. OBV 線連續小 N 字形上漲後出現大 N 字形 → 行情隨時反轉
4. OBV 線連續 N 字形上漲 → 上漲趨勢即將反轉

### 四、急速變化判讀

- **OBV 急速上升**：大量買盤湧入，但爆發不可能持續太久 → 行情將回檔，逢高賣出
- **OBV 急速下跌**：大量賣盤湧出，行情轉為跌勢 → 持幣觀望，不要搶反彈
- 急跌後 OBV 在底部形成鋸齒狀曲線 → 才可以考慮進場做短期反彈

### 五、長期累積高低點

- OBV 長期累積的波段高點 → 未來上升的大阻力區，突破後漲勢更猛
- OBV 長期累積的波段低點 → 未來下跌的大支撐區，跌破後跌勢更猛

### 六、橫盤突破

OBV 線持續一個月以上橫向移動後突然上沖 → 預示大行情隨時發生。這是主力資金從靜默轉為活躍的信號。

### 七、正負轉換

- **OBV 從負轉正**（對次新股特別有效）：上市就下跌的股票，OBV 長期為負值，一旦轉正代表買方取得決定性優勢，是最佳長線買進信號
- **OBV 從正轉負**：多方優勢喪失，賣出訊號

### 八、搭配其他指標

- OBV + [[RSI相對強弱指標]]：OBV 確認量能方向，RSI 確認超買超賣
- OBV + [[MACD指標實戰判讀]]：OBV 背離 + MACD 死亡交叉 = 強烈賣出
- OBV + [[布林通道Bollinger-Bands三軌八型態]]：布林帶突破 + OBV 確認 = 有效突破
- OBV + [[KD指標隨機指標]]：KD 鈍化區 + OBV 背離 = 趨勢反轉確認
- OBV + [[支撐壓力]]：價格突破支撐壓力 + OBV 同方向突破 = 真突破

## 注意事項

### OBV 的五大盲點

1. **漲跌二元判定過於簡化**：只看收盤價漲跌，不管漲多少跌多少。收盤在最高價附近漲1元和收盤在最低價附近漲1元，OBV 處理方式完全相同，但多方力量差很大
2. **平盤日資訊丟失**：平盤日 OBV 不變，但平盤日也可能有大量換手（主力對倒），這些資訊被忽略
3. **長期累積失真**：OBV 是累積值，時間越長數字越大，早期的異常值影響被稀釋但也可能扭曲
4. **不適合中長期研判**：周 OBV 和月 OBV 基本失去研判功能
5. **冷門股效果差**：成交量過小時 OBV 的變化缺乏意義

### 使用建議

- OBV 適用於**短期操作**，不要用來判斷中長期趨勢
- 優先使用**日 OBV**，避免使用周 OBV 和月 OBV
- OBV 的方向比數值重要，不要執著於 OBV 絕對值
- **背離是 OBV 最大的價值**，單獨看 OBV 走勢意義有限
- 上市兩年以上的老股，OBV 正負轉換的研判功能大幅降低
- 一定要搭配其他指標確認，OBV 不適合單獨使用

### 九、OBV 領先突破訊號

OBV 最有價值的用法之一——**領先價格突破**：

- **OBV 領先向上突破**：股價仍在盤整區間，但 OBV 已率先創新高 → 買方力量正在積累，股價可能即將跟進突破，潛在買入訊號
- **OBV 領先向下跌破**：股價仍在支撐區間，但 OBV 已跌破前低 → 賣方壓力巨大，資金正在撤出，股價跌破支撐的風險很高，潛在賣出訊號

這種「OBV 先行、價格跟進」的現象，正是「量先價行」最直觀的體現。

### 十、OBV 搭配均線實戰

使用 OBV 搭配移動平均線（建議 20MA 或 60MA）：

| OBV 方向 | 價格與均線位置 | 判讀 |
|----------|--------------|------|
| OBV 上升 | 價格在均線之上 | 多方力道強勢，持股續抱或加碼 |
| OBV 下降 | 價格在均線之下 | 空方力道強勢，持幣觀望或減碼 |
| OBV 上升 | 均線黃金交叉 | 更可靠的買入訊號（多重驗證） |
| OBV 下降 | MACD 死亡交叉 | 更可靠的賣出訊號（多重驗證） |

### 十一、背離判讀實戰步驟

1. 在價格圖上找出明顯的上升或下跌趨勢，標示出兩個連續的波段高點（頂背離）或低點（底背離）
2. 在 OBV 指標圖上，找到對應時間的 OBV 高低點
3. 分別連接價格和 OBV 的兩個高/低點，形成趨勢線
4. 觀察兩條趨勢線的斜率是否相反：
   - 價格向上 + OBV 向下 → 頂背離
   - 價格向下 + OBV 向上 → 底背離
5. **確認背離**：務必確認股價真的創了新高/新低，而指標沒有；如果價格只是反彈，就不算背離
6. **不要單打獨鬥**：搭配其他指標一起看，例如 RSI（確認是否超買超賣）、成交量（看有沒有大量資金進出）、均線或趨勢線（看價格有沒有突破）

### 十二、底背離確認進場時機

底背離出現後，不要急著買進，需等待確認訊號：
- 以長紅 K 線收盤確認底部
- K 線出現錘子線、看漲吞噬等反轉型態
- 股價站穩關鍵支撐位
- KD 指標從低檔區黃金交叉向上
- 成交量伴隨放大（有買盤進場承接）
