# FVG 公允價值缺口 (Fair Value Gap)

> 三根K線之間的價格真空帶，機構快速進場的證據，價格傾向回測失衡區重新平衡

## 核心概念

FVG（Fair Value Gap，公允價值缺口/失衡區）是SMC/ICT框架中辨識機構快速進場的核心工具。當買賣力量在短時間內嚴重失衡，一方大幅推動價格而對方來不及反應，就會形成一個**未被充分交易的價格區域**——這就是FVG。

### 如何辨識FVG？

至少需要**三根K線**：

- **看漲FVG**：中間大陽線的高點 > 前一根K線的低點，但中間K線的低點 > 第三根K線的高點 → 第三根高點到第一根低點之間的空隙
- **看跌FVG**：中間大陰線的低點 < 前一根K線的高點，但中間K線的高點 < 第三根K線的低點 → 第一根低點到第三根高點之間的空隙

簡單說：**中間K線與前後K線高低點不重疊的部分**就是FVG。

### FVG的原理

1. **機構行為證據**：聰明錢大量建倉時，價格快速移動形成失衡，這就是「機構在這裡大舉進場」的腳印
2. **價格填補傾向**：機構有未成交的殘餘訂單需要回補，市場需要足夠流動性完成交易，價格傾向回到失衡區重新平衡
3. **流動性驅動**：價格往流動性充足的地方移動，FVG區域就是流動性不足的地方

### BISI 與 SIBI — 失衡的兩面

ICT用更精確的術語描述失衡的供需結構：

- **BISI（Buyside Imbalance Sellside Inefficiency）**：看漲FVG的本質——買方過多、賣方不足。該區域留有大量未成交的買方訂單，價格再次觸及時積壓買單被執行，產生支撐效果
- **SIBI（Sellside Imbalance Buyside Inefficiency）**：看跌FVG的本質——賣方過多、買方不足。該區域留有大量未成交的賣方訂單，價格再次觸及時積壓賣單被執行，產生壓力效果

理解BISI/SIBI有助於判斷FVG的支撐/壓力角色：殘餘的是買單就支撐，殘餘的是賣單就壓力。

## 實戰應用

### 如何判斷關鍵FVG？

不是所有FVG都同等重要，四種高價值FVG：

1. **折/溢價區的FVG**
   - 用斐波那契回撤工具判斷
   - 折價區（0-0.382）出現的看漲FVG更有價值
   - 溢價區（0.618-1.0）出現的看跌FVG更有價值

2. **最靠近OB的FVG**
   - FVG出現在[[訂單塊Order-Block|訂單塊]]附近 = 「火龍果策略」
   - 機構成本區 + 失衡區雙重共振，勝率大幅提升

3. **流動性獵取後出現的FVG**
   - 機構完成獵取散戶止損後快速進場
   - 此時出現的FVG最可靠，代表機構已建立新部位

4. **與破壞塊一起出現的FVG**
   - 破壞塊 (Breaker Block) = 失效訂單塊翻轉後的角色
   - FVG + 破壞塊同時出現 = 支撐/阻力角色已轉換

### FVG進場實戰流程

1. 在趨勢行情中找到**尚未回測且明顯的FVG**
2. 往左找到需求區（[[訂單塊Order-Block|OB]]、FVG、Breaker Block），在需求區進場提高勝率
3. 等待價格止跌/止漲 → 流動性獵取 → [[BOS與CHoCH市場結構轉變|CHoCH]] → 出現FVG時進場
4. 止損設在需求區外，止盈設在目標FVG

### FVG作為出場目標

FVG也可以當作止盈目標：
- 進場後，價格朝未回測的FVG移動
- 到達FVG後部分或全部出場
- 特別是下跌趨勢中，下方的未回測FVG是自然的目標位

### iFVG = 逆轉FVG (Inversion Fair Value Gap)

當FVG被突破而失去原本作用時，角色反轉：

| 原本 | 填補後 |
|------|--------|
| 看漲FVG（支撐） | 看跌iFVG（阻力） |
| 看跌FVG（阻力） | 看漲iFVG（支撐） |

iFVG代表供需平衡已重新調整，角色轉換後仍是有效的交易區域。

**iFVG實戰進場三條件**（大時間框架iFVG → 小時間框架進場）：

1. **流動性獵取**：聰明錢進場前需要充足流動性，高機率先發生流動性獵取（突破高/低點後迅速回落）
2. **小框架出現FVG**：FVG是聰明錢進場線索，小時間框架出現FVG作為進場依據
3. **發生mss/CHoCH**：趨勢改變時一定發生市場結構轉變（前一個低/高點被突破）

三條件同時出現 → 價格回到小框架FVG時進場，止損守在FVG前一個低/高點，止盈設在大框架最近的流動性位置。

## 注意事項

- **FVG不是獨立策略**：必須搭配市場結構、[[訂單塊Order-Block|OB]]、流動性一起使用，單獨用FVG進場勝率不高
- **小FVG vs 大FVG**：越大的FVG代表失衡越嚴重，回測機率越高，但填補時間也越長
- **已填補的FVG失去意義**：一旦被完全填補，支撐/阻力角色消失（除非轉為iFVG）
- **多個FVG堆疊**：同一方向有多個未回測FVG，代表趨勢力道強，但回測時可能依序填補
- **FVG在盤整中失效**：盤整行情中的FVG缺乏機構行為支撐，容易被隨意穿越
- **時間框架選擇**：較大時間框架的FVG更可靠，日線FVG > 1小時FVG > 5分鐘FVG
