# 均線糾結選股與操作實戰

> 均線糾結是多條均線聚攏的現象，代表多空成本趨於一致，是大行情的前兆；盤整越久突破越大，帶量確認才是真突破。

## 核心概念

### 什麼是均線糾結

當5日、10日、20日、60日等多條不同週期的均線聚攏在一起，差距極小甚至幾乎重疊，稱為均線糾結。這表示不論是哪個週期交易的投資人，持股成本都差不多，市場正處於多空力量平衡的盤整狀態。

### 均線糾結的市場心理

- **所有參與者成本趨於一致**：短期、中期、長期投資人的平均成本幾乎相同
- **浮額被洗出**：沒有信心的投資人在盤整中耐不住性子出場，留下來的籌碼都有共識
- **主力吸籌階段**：盤整期間主力慢慢吸收籌碼，準備發動
- **能量蓄積**：就像彈簧壓得越緊放開後跳得越高，糾結時間越長突破後行情越大

### 三層濾網辨識

1. **均線聚攏度**：短期均線（5/10/20日）差距 < 2%
2. **成交量縮減**：均線糾結期間量能萎縮至20日均量50%以下
3. **籌碼集中**：千張大戶持股穩定或增加

### 糾結後的四种走勢

1. **向上突破（多頭行情啟動）**：股價帶量突破均線糾結區，站穩所有均線之上，均線開始向上發散形成多頭排列
2. **向下破位（空頭行情確立）**：股價帶量跌破均線糾結區，持續在所有均線之下運行，均線向下發散形成空頭排列
3. **橫向盤整（續盤待變）**：股價在均結區內持續小幅波動，多空力量依然平衡，觀望是最好的策略
4. **假突破/假破位（誘多誘空）**：股價短暫突破或跌破後迅速反轉，通常伴隨量能配合不佳

## 實戰應用

### 進階選股三招（多方策略）

#### 1. 避免KD、MACD背離股票

當出現KD或MACD背離時，股價突破均線糾結就容易產生騙線：
- 股價帶量創高，但KD與MACD沒有創高 → 後續容易回檔
- 技術指標有跟著股價創高 → 突破後會有一波漲幅

#### 2. 挑選長期均線走多的個股

- 中長期均線（月線、季線）維持多方趨勢 → 突破後容易形成均線多頭排列
- 月線仍下彎 → 股價不夠強勢，中期趨勢仍往下，突破容易失敗
- 用扣抵值確認長均線方向：扣抵值下降均線自然上揚

#### 3. 搭配籌碼面確認

- 投信連續買超 + 均線糾結突破 → 後續一波多頭行情機率高
- 千張大戶增加 + 均線糾結 → 主力吸籌完成，突破後行情更大
- 法人買超 > 1000張/週 → 籌碼面支撐突破

### 量化選股條件

**基本條件**：
1. 股價大於週線、月線、季線
2. 股價距離週線、月線、季線10%之內
3. 當日漲幅大於2%

**進階過濾**：
1. 排除地雷股：Z-Score < 1.81
2. 排除虧損股：近四季盈餘 < 0
3. 排除無量股：週均量 < 200張
4. 排除低價股：股價 < 10元
5. 量能暴增：當日成交量 > 月均量3倍
6. 大戶狂買：千張大戶佔總集保比率較上月成長 > 1%
7. 法人狂買：近1週法人買賣超 > 1000張
8. 中小型股：股本50億以下

### 突破均線糾結三大確認要點

1. **打底**：用2個低點劃上升趨勢線，股價在趨勢線之上才有打底現象
2. **突破**：用2個高點劃下降趨勢線，股價突破下降趨勢線才確認
3. **帶量**：突破K線必須帶量，無量突破極可能是假突破

### 操作SOP

#### 進場

1. **等待**：在糾結時不要動作（因為不知道會往上還是往下）
2. **確認**：等待一根「帶量長紅K」突破所有均線 = 主力表態做多
3. **進場點**：帶量突破盤整區時進場，雖然成本較高但安全性和成功率都較高
4. **停損點**：設在糾結區下緣或突破K線的最低點

#### 出場

- **暴量出場**：股價有一大段漲幅後暴出大量，可能是主力出脫持股
- **暴量標準**：參考個股過往最大成交量，或當日週轉率 > 10%
- **移動停利**：股價沿5日均線（短線）或10日均線（波段）墊升

#### 停損

- 帶量紅K最低點為停損點
- 轉強個股又回跌到發動點 → 假突破機會大，先退場觀察

### 假突破判斷四要素

1. **量能是關鍵**：真突破成交量明顯放大好幾倍，假突破量能不足
2. **K線型態**：真突破出現有力的長紅K線，假突破常有長上影線
3. **回測確認**：真突破後回測能守穩不破，假突破直接貫破
4. **時間要素**：真突破行情持續好幾天甚至幾週，假突破1-2天就跌回來

## 搭配指標確認

| 指標 | 糾結時特徵 | 突破時訊號 |
|------|-----------|-----------|
| 成交量 | 縮量至20日均量50%以下 | 帶量突破，量能明顯放大 |
| KD | 低檔或高檔鈍化，K/D來回交叉 | 金叉或死叉快速發散 |
| MACD | 柱狀圖趨近零軸，長度很短 | 柱狀圖從零軸迅速延伸，DIF/DEA交叉 |
| 布林通道 | 上下軌明顯收縮，通道狹窄 | 突破上下軌，通道開始擴張 |
| 均線角度 | 長均線走平或微彎 | 突破後長均線上彎確認 |

## 注意事項

- 均線糾結是「等待訊號」不是「進場訊號」——要等突破確認才動作
- 糾結時間越長，突破後的行情越大
- 大盤弱勢時個股均線糾結突破失敗率高
- 假突破後會快速回落均線束
- 不是萬靈丹——結合其他指標綜合判斷提高準確度
- 長週期糾結預示大行情，短週期糾結只是小行情
- 新手常見錯誤：過早判斷、忽視量能、沒有等待確認
- 搭配多時間框架分析：日K線糾結時切換到週K線或月K線看更大趨勢
