# Schaff Trend Cycle 沙夫趨勢週期指標

> MACD 的「加速版」——用隨機指標對 MACD 做二次處理，比 MACD 更快反應趨勢轉折，產生更少假訊號

## 核心概念

Schaff Trend Cycle（STC）由 Doug Schaff 在 1990 年代開發，核心觀察：**市場價格遵循循環模式——加速期與減速期交替出現**。STC 結合 MACD 和隨機震盪指標（Stochastic）的優點，比傳統 MACD 更快辨識趨勢反轉。

### 三階段計算邏輯

1. **計算 MACD**：EMA(23) − EMA(50)
2. **對 MACD 值套用隨機指標**：識別循環波動
   - Stochastic MACD = ((MACD − MinMACD) / (MaxMACD − MinMACD)) × 100
3. **額外平滑**：EMA(10) 消除雜訊

### STC 值域

- 範圍：0 ~ 100
- **> 75**：超買區
- **< 25**：超賣區
- 25 和 75 是關鍵水位線

### STC vs MACD 對比

| 特性 | STC | MACD |
|------|-----|------|
| 指標類型 | 震盪指標 | 趨勢指標 |
| 計算方式 | EMA差值 + 隨機處理 | EMA差值 |
| 敏感度 | 高 | 低 |
| 假訊號 | 較少 | 較多 |
| 適用策略 | 短線交易 | 中線交易 |
| 反應速度 | 快 | 慢 |
| 循環分析 | 有 | 無 |

## 實戰應用

### 1. 超買超賣交易

**做多**：
1. STC 從下方穿越 25 水位
2. STC 線上升表示下降趨勢結束
3. 搭配成交量放大或阻力突破確認

**做空**：
1. STC 從上方跌破 75 水位
2. STC 線下降表示超買即將反轉
3. 搭配支撐跌破或型態確認

### 2. STC + RSI 策略

- STC 穿越 25 + RSI < 30 → 雙重超賣確認，做多訊號更可靠
- STC 穿越 75 + RSI > 70 → 雙重超買確認，做空訊號更可靠
- 兩個指標同時發出訊號，勝率顯著高於單一指標

### 3. STC + 支撐壓力策略

- STC 穿越 25 且價格接近支撐位 → 在支撐位附近做多
- STC 穿越 75 且價格接近壓力位 → 在壓力位附近做空
- 關鍵：**只做順大趨勢方向的訊號**

### 4. STC + 均線策略

- 用長期均線（如 60MA）判斷主要趨勢方向
- 價格在 60MA 上方 → 只接受 STC 做多訊號
- 價格在 60MA 下方 → 只接受 STC 做空訊號
- STC 負責精確的進出場時機

### 5. STC 搭配循環分析

- STC 完成一個 0→100→0 的循環代表一次完整的多空循環
- 觀察 STC 循環週期長度，預判下一個轉折時機
- 循環週期縮短 → 趨勢加速
- 循環週期拉長 → 趨勢減速

## 注意事項

- **盤整市假訊號**：STC 在區間盤整時同樣會頻繁穿越 25/75 水位，必須搭配趨勢指標過濾
- **過度優化風險**：參數調整過度容易造成曲線套入（curve fitting），回渤好看但實盤失效
- **劇烈反轉仍可能落後**：V型反轉等極端走勢，STC 反應可能不夠即時
- **必須搭配確認指標**：RSI、支撐壓力、成交量等至少選一個搭配
- **預設參數（23/50/10）適合外匯**：台股使用可能需要微調，建議先回測驗證
- **0~100 的限制**：與 MACD 無上下限不同，STC 被鎖定在 0~100 區間，極端行情可能「壓縮」訊號
