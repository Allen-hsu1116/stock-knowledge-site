# VSA成交量價差分析 Volume Spread Analysis

> 從成交量、價差、收盤位置三維判讀聰明錢的真實意圖，是威科夫方法的量化實戰版，六種核心型態辨識主力吸籌與出貨

## 核心概念

VSA（Volume Spread Analysis，成交量價差分析）由 Tom Williams 基於 Richard Wyckoff 的市場理論發展而來，是一種透過分析**成交量（Volume）、價差（Spread）和收盤價位置**三者關係，來推斷「聰明錢」（Smart Money，即主力、法人）真實意圖的分析方法。

### 三大定律基礎

VSA 建立在 Wyckoff 三大定律之上：

1. **供需定律（Law of Supply and Demand）**：價格漲因需求大於供給、跌因供給大於需求。VSA 透過量與價差的組合判斷供需誰占主導——寬幅上漲帶大量代表需求強勁，窄幅下跌帶大量可能是隱藏買進（吸收行為）。

2. **因果定律（Law of Cause and Effect）**：吸籌（Accumulation）或派發（Distribution）是因，後續的趨勢是果。長時間的橫向震盪配合遞增的成交量，可能是主力在吸籌，為大行情蓄力。

3. **努力與結果定律（Law of Effort vs. Result）**：成交量是努力，價格變動是結果。**高努力低結果**（大量但價格不動）代表主力在對抗趨勢，反轉可能；**低努力高結果**（量不大但價格大幅移動）代表趨勢得到確認。

### VSA 關鍵定義

| 定義 | 說明 |
|------|------|
| **價差（Spread）** | 最高價 - 最低價 |
| **寬幅K線（WSB）** | 價差 > 1.8 倍平均價差 |
| **窄幅K線（NSB）** | 價差 < 0.8 倍平均價差 |
| **Up Bar** | 收盤高於前一根收盤 |
| **Down Bar** | 收盤低於前一根收盤 |
| **Up Close** | 收在價差 70% 以上 |
| **Mid Close** | 收在價差 30%~70% 之間 |
| **Down Close** | 收在價差 30% 以下 |

## 實戰應用

### 六種核心型態

#### 1. 無需求訊號（No Demand）
- **特徵**：上漲K線 + 窄幅 + 量低於前兩根
- **解讀**：聰明錢不願在更高價位追買，上漲缺乏真實需求
- **操作**：避免追多；若後續走弱確認空頭

#### 2. 無供給訊號（No Supply）
- **特徵**：下跌K線 + 窄幅 + 量低於前兩根
- **解讀**：賣壓枯竭，賣方失去興趣
- **操作**：等待下一根上漲K線確認，可能是反轉區

#### 3. 買進高潮（Buying Climax）
- **特徵**：價格飆升 + 極大量 + 收盤遠離最高點
- **解讀**：散戶搶進，主力趁機出貨
- **操作**：經典派發訊號，觀察後續走弱

#### 4. 賣出高潮（Selling Climax）
- **特徵**：急跌 + 極大量 + 收盤遠離最低點
- **解讀**：散戶恐慌拋售，主力暗中吸籌
- **操作**：潛在底部反轉訊號

#### 5. 上沖與彈簧（Upthrust & Spring）
- **上沖（Upthrust）**：快速突破阻力後收低——假突破誘多，主力派發
- **彈簧（Spring）**：跌破支撐後收高——假跌破誘空，主力吸籌
- **操作**：兩者都是主力操縱區，常出現在重大反轉前

#### 6. 測試（Test）
- **測試供給**：低量下跌K線測試前支撐——若量低且反彈，供給已枯竭
- **測試需求**：低量上漲K線測試前阻力——若量低且回落，需求不足
- **操作**：成功的測試（低量 + 下一根反轉）確認對手已耗盡

### 三步實戰框架

1. **判斷市場階段**：先確認目前處於吸籌、拉升、派發還是打壓階段（參考[[威科夫方法進階吸籌派發五階段判讀]]）
2. **逐根比對VSA訊號**：用六種型態逐根判讀，**不知道右邊K線時比對效果最好**——這是 VSA 最核心的練習方式
3. **與其他工具共振確認**：
   - **均線**：確認大趨勢方向，逆趨勢 VSA 訊號需謹慎
   - **RSI**：RSI 超買 + VSA 無需求 = 做空共振
   - **支撐壓力**：VSA 訊號出現在關鍵價位更可靠
   - **[[OBV能量潮指標量價趨勢判讀]]**：OBV 背離 + VSA 高潮訊號 = 強反轉確認

### VSA 與傳統技術分析的差異

| 維度 | 傳統技術分析 | VSA |
|------|-------------|-----|
| 核心關注 | 指標數值（RSI、MACD等） | 量、價差、收盤位置的關係 |
| 訊號類型 | 落後指標反應已發生的變化 | 領先判讀主力當下行為意圖 |
| 分析哲學 | 「發生了什麼」 | 「為什麼發生」 |
| 學習門檻 | 較低，公式化 | 較高，需內化經驗 |
| 適用範圍 | 公式適用所有商品 | 需依商品特性調整判讀 |

## 注意事項

### VSA 的主觀性陷阱
VSA 最大的風險是**主觀判讀**——同一根K線不同人可能解讀不同。解決方案：
- 用量化定義（WSB/NSB/Up Close 等）建立客觀標準
- 永遠在市場階段框架下判讀，不要孤立看單根K線
- 至少兩個 VSA 訊號共振才進場

### 量化困難
VSA 難以像 RSI/KD 一樣完全指標化——不同股票的平均價差不同、流動性不同，0.8 倍和 1.8 倍的門檻需要根據個股調整。**這是特色不是缺陷**：VSA 要求你真正理解手上的標的，而不是套公式。

### 市場適用性
- **最適用**：集中交易市場（股票、期貨），成交量數據準確
- **需謹慎**：外匯、加密貨幣等去中心化市場，成交量數據不完整，可用 Tick Volume 替代但準確度下降
- **冷門股**：成交量太低時 VSA 訊號不可靠

### 學習曲線
VSA 的學習曲線陡峭，建議：
1. 先熟讀 Wyckoff 三大定律（參考[[威科夫方法]]）
2. 用歷史K線遮住右邊逐根比對，培養內化能力
3. 從 No Demand/No Supply 兩種最簡單型態開始
4. 再進階到高潮訊號和 Upthrust/Spring
5. 最後整合測試訊號和多重確認
