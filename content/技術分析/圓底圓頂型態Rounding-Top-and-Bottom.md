# 圓底圓頂型態Rounding-Top-and-Bottom

> 圓底（圓弧底/碟形底）是主力耐心吸籌的底部反轉型態；圓頂（圓弧頂）是多頭力竭的頂部反轉訊號。兩者皆需帶量突破/跌破頸線確認。

## 核心概念

### 圓底型態（Rounding Bottom / Saucer Bottom）

又稱圓弧底、碟形底、鍋底形。股價經過長期下跌後，賣壓逐步減弱、買盤漸漸增強，形成圓弧狀的底部反轉型態。

**市場心理**：
- 下跌末期：賣方力量漸弱，投資者不計價搶低
- 底部區：主力耐心限價吸籌，成交量極度萎縮
- 回升期：買方逐步掌控，成交量緩步遞增
- 突破時：帶量突破頸線，主力拉升股價

### 圓頂型態（Rounding Top / Saucer Top）

與圓底相反，出現在長期上升後的頂部反轉型態。

**市場心理**：
- 上升末期：買方力量漸弱
- 頂部區：成交量減少，觀望氣氛濃
- 回落期：賣方逐步掌控，成交量緩步增加
- 跌破時：跌破頸線確認反轉

### 兩者比較

| 特徵 | 圓底 | 圓頂 |
|------|------|------|
| 出現位置 | 長期下跌底部 | 長期上升頂部 |
| 型態方向 | 由下而上緩慢攀升 | 由上而下緩慢下跌 |
| 成交量 | 左縮右增 | 左增右縮→跌破放量 |
| 訊號方向 | 看漲 | 看跌 |
| 形成時間 | 數週到數月（甚至數年） | 數週到數月 |

## 實戰應用

### 圓底進場策略

1. **保守進場**：突破頸線3%以上確認後買進
2. **積極進場**：圓底右側量能逐步放大時提前布局
3. **加碼時機**：突破頸線後回測頸線獲支撐

**停損位置**：
- 保守：跌破頸線
- 積極：跌破圓底最低點

### 圓頂出場策略

1. **跌破頸線**：立即賣出或做空
2. **頸線回測**：跌破後反彈至頸線是最後出場機會
3. **停損**：設在圓頂最高點上方

### 目標價計算

```
圓底目標價 = 頸線 +（頸線 - 最低點）
圓頂目標價 = 頸線 -（最高點 - 頸線）
```

例：圓底頸線 $100、最低點 $70 → 目標 $130

### 量價確認三步驟

1. **左側**：量能萎縮（賣壓減退或買壓減退）
2. **底部/頂部**：量能極度低迷（市場觀望）
3. **右側突破**：**必須顯著放量**（否則型態不可信）

### 實戰案例

- **鈊象(3293)**：2024年大漲後4月進入盤整，下探980元低點後量增回升，突破頸線後大漲至1520元（+55%）
- **圓底往往是機構大戶吸籌**：資金充足但不能一次買太多，耐心限價收集籌碼

## 注意事項

1. **成交量必須配合**：如果圓底右側量能沒有逐步放大，型態不可靠，應等待進一步確認
2. **碟柄（Saucer Handle）**：圓底完成後可能形成一個小平台整理，類似杯柄的柄部，突破時必須帶量
3. **形成時間越長越可靠**：數月形成的圓底比數天形成的更有參考價值
4. **假突破風險**：突破頸線後必須觀察3%以上確認，防止假突破
5. **搭配其他指標**：MACD金叉、RSI底背離等可輔助確認圓底
6. **底部波幅極小**：真正的圓底底部區間波動很小，若波動劇烈則不典型
