# VWAP成交量加權均價

> 結合價格與成交量的當日平均成本線，是日內交易判斷多空與進出場的核心指標

## 核心概念

VWAP（Volume-Weighted Average Price，成交量加權平均價格）是將「成交量」納入計算的平均價格線，代表當日所有參與者（含主力與散戶）的平均持倉成本。

### 計算公式

```
VWAP(t) = Σ(Pi × Vi) / Σ(Vi)
```

- Pi = 第 i 筆成交價格
- Vi = 第 i 筆成交量
- 典型價格 = (最高價 + 最低價 + 收盤價) / 3

**關鍵特性**：
- 每日開盤歸零重算，是純日內指標
- 成交量大的價位對 VWAP 影響更大（加權效果）
- 本質是「流動性結構」，而非一般振盪指標（如 MACD、RSI）

### VWAP vs 均線（MA）

| 項目 | VWAP | 均線 MA |
|------|------|---------|
| 計算基礎 | 價格 × 成交量 | 僅價格 |
| 核心意義 | 當日真實市場平均成本 | 特定期間算術平均 |
| 重設週期 | 每日歸零 | 連續計算 |
| 適用場景 | 當沖、日內多空判斷 | 波段、長線趨勢 |

## 實戰應用

### 策略一：日內多空判斷

- **價格在 VWAP 之上**：多方強勢，偏多操作
- **價格在 VWAP 之下**：空方強勢，偏空操作
- VWAP 是日內多空的「楚河漢界」

### 策略二：趨勢跟隨法（回測支撐）

1. 開盤後股價強勢上漲，拉開與 VWAP 距離
2. 不追高，等待股價回落修正
3. 股價回測 VWAP 未跌破（或跌破後迅速站回），伴隨下影線或紅 K
4. 邏輯：主力不願讓股價低於今日平均成本

### 策略三：乖離反轉法（抓轉折）

1. 股價急噴，距 VWAP 線極遠（乖離率過大）
2. 出現爆量長黑 K 或上影線，量能萎縮
3. 可嘗試短空，目標價 VWAP 附近
4. **風險較高，需嚴設停損**

### 策略四：動態支撐壓力

- **上升趨勢**：價格回測 VWAP 吸引買盤 → 支撐
- **下降趨勢**：價格反彈至 VWAP 觸發解套賣壓 → 壓力

### 策略五：穿越訊號

- **黃金交叉**：價格帶量從下方突破 VWAP → 看漲
- **死亡交叉**：價格放量跌破 VWAP → 看跌
- 穿越後回踩確認更可靠

## 進階：VWAP 標準差通道

類似布林通道，在 VWAP 上下加一倍/兩倍標準差：
- **68%** 的成交價落在一倍標準差內
- **95%** 的成交價落在兩倍標準差內

實戰應用：
- 突破 VWAP 但受阻於二倍標準差 → 多頭力道極限 → **假突破**
- 回測 VWAP 同時重疊下方支撐 → **共振點**，提高勝率
- 法人自動化交易以 VWAP 為基準分批執行 → VWAP 本身具有法人行為支撐

## 進階：Anchored VWAP

傳統 VWAP 固定從開盤計算，Anchored VWAP 可指定起始時間點：
- 以「重要事件發生點」為起點重新累積
- 範例：CPI 公布時、FOMC 會議聲明時
- 能追蹤事件驅動後的資金真實平均進場成本
- 補足傳統 VWAP 的缺點

### 多日 VWAP

- 計算過去 3 天的成交量加權平均
- 捕捉中期資金平均成本層
- 適合盤整區間做「中期流動性中心」推估

## 注意事項

### 適用時機

- ✅ 開盤第一小時（08:45～09:45）：VWAP 迅速拉出早盤資金中心區
- ✅ 盤整震盪盤：VWAP 成為主力回補/反壓分界線
- ✅ 多空轉折時刻：穿越 VWAP 伴隨趨勢切換

### 不適用時機

- ❌ 趨勢爆發盤（單邊 2%+）：VWAP 失效延遲，被遠遠甩開
- ❌ 超低成交量窒息盤：VWAP 跳動極小，無指標意義
- ❌ 開盤前 15-30 分鐘：數據不足，VWAP 不穩定
- ❌ 隔夜/盤後：VWAP 每日歸零，無意義

### 法人執行邏輯與 VWAP 的關係

管理大筆資金的法人操盤手，目標不是「買在最低點」，而是「不買得比市場平均貴」。因此許多法人的自動化交易演算法會設定以 VWAP 為基準進行分批執行：

- **股價 < VWAP**：法人買盤相對積極，以優於當日平均成本的價格建倉
- **股價 > VWAP**：買進力道通常減弱
- **這種「機構行為」正是 VWAP 形成強大支撐與壓力的根本原因**

理解法人以 VWAP 為基準的分批執行邏輯，可以幫助散戶解讀盤中主力行為，避免在波動過大的極端位置盲目追高。

### 常見誤區

1. **開盤初期過度依賴**：前 15-30 分鐘 VWAP 不穩定
2. **忽略大趨勢**：站上 VWAP 不代表可逆大勢做多
3. **把 VWAP 當一般均線**：VWAP 是流動性結構，非振盪指標
4. **無量穿越當真突破**：穿越 VWAP 時量能萎縮 = 假突破
