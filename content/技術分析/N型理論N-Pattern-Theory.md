# N型理論N-Pattern-Theory

> 股價走勢的基本結構單元——上漲N型代表攻擊、下跌N型代表殺盤，1/2法則判斷多空強弱，失敗N型是反轉前兆

## 核心概念

N型理論是波動力學的基礎，將股價走勢拆解為最基本的N字結構。一個N型由四個節點(a, b, c, d)組成，代表價格從起漲到回測再到攻擊（或從下跌到反彈再到續跌）的完整循環。

### 上漲N型（正N型）

```
    d
   / \
  b   c
 /
a
```

- **ab段**：突破段（逆轉段）——從原點a向上攻擊到b
- **bc段**：回測段（修正段）——從b回落到c
- **cd段**：攻擊段（趨勢段）——從c再次上漲突破b到d

看到正N型代表有攻擊力道，cd段是趨勢延續的確認。

### 下跌N型（倒N型）

```
a
 \
  b   c
   \ /
    d
```

- **ab段**：下跌段——從原點a向下殺到b
- **bc段**：反彈段——從b反彈到c（不過a）
- **cd段**：續跌段——從c跌破b到d

### 起漲n型與測幅n型

- **起漲n型**：從原點a開始的第一個小n型（abcd四個節點）
- **測幅n型**：回測守住1/2後再突破新高形成的第二個n型
- **初始N型** = 起漲n型 + 測幅n型

## 實戰應用

### 1/2法則——判斷多空強弱的核心

1/2法則是N型理論最重要的實戰工具：

| 回測位置 | 意義 | 後續走勢預期 |
|----------|------|-------------|
| 守住1/2以上 | 多頭強勢 | 再創新高，形成擴大N型 |
| 跌破1/2但反彈突破下跌段1/2 | 多頭弱但不死 | 盤漲或盤整 |
| 跌破1/2且反彈無力突破下跌段1/2 | 空頭反轉 | 形成n上+n下，跌回起漲原點 |

**1/2計算公式**：
- 上漲N型的1/2 = (a點低價 + d點高價) / 2
- 下跌N型的1/2 = (a點高價 + d點低價) / 2

### 關鍵K棒搭配N型理論

關鍵K棒（個股3%以上實體）與N型理論搭配是更強大的觀盤工具：

**關鍵紅K棒頸線計算**：
- 上頸線（壓力）= 收盤價 + 實體1/2
- 下頸線（支撐）= 實體1/2位置
- 頸線 = 收盤價

**關鍵黑K棒頸線計算**：
- 上頸線（壓力）= 實體1/2處
- 下頸線（支撐）= 收盤價 - 實體1/2

1根關鍵K棒可管控3~4根K棒的走勢。突破上頸線需帶量或跳空，測試下頸線需量縮。

### 型態開盤法——日內操作

1. 以前日收盤價為a點（原點）
2. 開盤後30分鐘確定b點和c點
3. 根據N型結構計算支撐壓力
4. 判斷盤態（震盪盤、趨勢盤等）
5. 做出兩種假設並隨走勢調整

### 失敗N型——反轉前兆

- **多頭失敗N型**：d點不過b點，多頭力道減弱，可能反轉向下
- **空頭失敗N型（收腳N型）**：該跌不跌，未破前低即反轉，是跌勢末段訊號

收腳N型確認方式：小W底 + 帶量中長紅突破頸線

### 基本面+技術面雙重確認

- 只懂技術面：在N型完成時先獲利了結，回測守住1/2再買回
- 同時懂基本面：做多業績佳的股票，放空業績衰退超漲的股票，達到多空雙贏
- 基本面成長的股票回測守住1/2機率高
- 基本面衰退的股票回測容易跌破1/2

## 注意事項

1. **N型不是預測工具**：N型理論是分類工具，幫助你理解當前走勢結構，不是預測未來
2. **1/2法則是概率不是保證**：守住1/2不保證一定再創新高，跌破1/2也不一定跌回原點
3. **變異型態常見**：變異正N型、破底穿頭的倒N型都不正常，需要額外確認
4. **必須搭配量能**：突破需帶量、支撐需量縮，沒有量能配合的N型不可信
5. **多時間框架**：日線N型、週線N型、月線N型要一起看，大週期主導方向
6. **假突破風險**：d點突破b點後可能假突破回落，需要量價確認
7. **關鍵K棒級距**：個股3%以上才算關鍵K棒，小K棒的N型結構意義不大
