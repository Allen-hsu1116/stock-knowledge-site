# 流動性獵取 Liquidity Sweep

> 聰明錢推動價格突破散戶止損密集區——你的止損就是機構進場的燃料，被掃之後才是真正的方向

## 核心概念

流動性獵取（Liquidity Sweep / Liquidity Grab / Liquidity Raid）是 SMC 聰明錢概念的核心機制。聰明錢（大型機構、對沖基金）需要大量流動性才能進場，而散戶的止損單就是最大的流動性來源。

**核心邏輯**：
1. 散戶在明顯的高低點設止損 → 形成流動性池
2. 聰明錢推動價格短暫突破這些位置 → 觸發止損單
3. 止損單 = 對手盤，聰明錢拿到足夠的對手盤後反向操作
4. 這就是為什麼你的止損總是被打掉然後價格就往你預期的方向走

## 流動性池的位置

流動性在哪裡聚集？

- **明顯波段高低點 (Swing Highs/Lows)** — 散戶最常設止損的位置
- **等高點/等低點 (Equal Highs/Lows)** — 雙重頂、雙重底累積大量止損
- **趨勢線** — 很多人沿趨勢線設止損或突破入場單
- **整數關卡** — 心理價位的止損密集區
- **前日高低點** — 日內交易者的止損區

## 流動性獵取的兩種型態

### 1. Buy Side Liquidity Sweep（掃上方流動性）
- 價格突破明顯高點或等高點 → 觸發買方止損（空單回補）+ 突破追多單
- 聰明錢在高位拿到空單流動性後 → 反手做空
- 後續常見：MSS/CHoCH + 看跌訂單塊或破壞塊

### 2. Sell Side Liquidity Sweep（掃下方流動性）
- 價格跌破明顯低點或等低點 → 觸發賣方止損（多單停損）+ 突破追空單
- 聰明錢在低位拿到多單流動性後 → 反手做多
- 後續常見：MSS/CHoCH + 看漲訂單塊或破壞塊

## 實戰判斷：真正的流動性獵取 vs 普通突破

### 有效流動性獵取的確認要素

1. **清晰的流動性池**：被測試多次都無法突破的高低點，流動性充足
2. **Strong Displacement**：獵取後的價格移動必須快速有力，伴隨 FVG 形成
3. **MSS/CHoCH 確認**：流動性被獲取後，市場結構必須發生轉變
4. **沒有流動性被 Raid 的 MSS 是無效的**：如果結構轉變沒有伴隨流動性獵取，可能只是普通波動

### ICT 進階判斷法則

ICT 強調：**Liquidity Raid 是第一要素，Strong Displacement 是第二要素。**

- 沒有流動性獵取的結構轉變 = 不可靠
- 沒有 FVG 的移動 = 不夠「明顯」，不是有效的獵取
- 真正有效的 MSS，需要有非常清晰的 Liquidity → 被 Raid → 然後結構轉變

## 實戰應用

### 流動性獵取交易框架

**步驟一：高時間框架找流動性池**
- 在日線或4小時圖，找出被多次測試的高低點
- 這些是聰明錢可能會去「打獵」的位置

**步驟二：觀察流動性是否被獵取**
- 價格突破流動性池 → 觸發止損 → 快速反向
- 關鍵：突破後是否出現 Strong Displacement + FVG

**步驟三：小時間框架找進場**
- 切換到5分鐘或15分鐘圖
- 等待 CHoCH 確認結構轉變
- 回調至訂單塊或破壞塊進場

**步驟四：止損止盈**
- 止損：設在流動性獵取的極值之外
- 止盈：下一個流動性池（波段高低點）

### 避免「被獵取」的方法

- 不要把止損設在明顯的高低點正下方/正上方
- 止損留一些緩衝區域，避免被短暫突破掃掉
- 如果止損被掃掉後價格快速反向 → 這就是流動性獵取的特徵
- 不要在被掃止損後情緒化反手——冷靜觀察是否有 CHoCH 確認

## 注意事項

- **流動性獵取不是每次都會發生**：不是所有突破都是獵取，有些是真突破
- **區分獵取和真突破**：獵取後快速反向 + FVG + CHoCH = 獵取特徵；真突破會持續移動
- **大時間框架的流動性更重要**：日線高點的流動性比5分鐘高點重要得多
- **獵取後不一定馬上反向**：有時聰明錢會在高位/低位停留一段時間再反向
- **不要預測獵取**：不要假設「一定會被打止損」，而是觀察獵取已經發生後再行動
- **多時間框架配合**：1小時的流動性獵取 + 5分鐘的 CHoCH = 高機率進場
