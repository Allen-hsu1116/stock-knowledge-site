# 一目均衡表進階判讀時間論波動論值幅觀測論

> 一目均衡表的三大隱藏理論——時間論預測轉折日期、波動論判斷趨勢結構、值幅觀測論計算目標價——這是一目山人花了30年研究的精華，也是讓一目均衡表從「五條線+雲帶」升級為「完整交易系統」的關鍵

## 核心概念

基礎版的一目均衡表教你看五條線和雲帶判斷趨勢，但一目山人花了30年研究的真正精華是**三大理論**：時間論、波動論、值幅觀測論。這三個理論讓你不再只看「現在的訊號」，而是能預測「何時轉折」和「漲到哪裡」。

大多數人只用三役好轉/逆轉做進出場，卻忽略了時間論告訴你「什麼時候要注意」、波動論告訴你「現在是哪種結構」、值幅觀測論告訴你「目標價在哪裡」。三個理論聯合判讀，勝率比單用三役好轉高出許多。

---

## 一、時間論：預測轉折日期

時間論是一目均衡表最獨特的理論——它不預測「漲多少」，而是預測「什麼時候會轉折」。

### 基本數值

一目山人從統計中發現，9、17、26是市場最容易出現轉折的時間間隔：

| 數值 | 名稱 | 計算方式 | 意義 |
|------|------|----------|------|
| 9 | 一節 | 原始數值 | 短期轉換點（對應轉換線） |
| 17 | 二節 | 9×2-1=17 | 中期轉換點 |
| 26 | 三節 | 9×3-2=26 | 長期轉換點（對應基準線） |

為什麼要減1？因為高點和低點的計算中，起始日和終止日重複計算了1天，要扣除。

### 複合數值

基本數值可以組合成更大的轉折時間窗口：

| 複合數值 | 計算方式 | 意義 |
|----------|----------|------|
| 33 | 17×2-1 | 上升趨勢第一波動常見天數 |
| 42 | — | 中期轉折 |
| 52 | 26×2 | 先行帶B的計算基礎 |
| 65 | 33×2-1 | 大型轉折 |
| 76 | 三期一巡 | 極大型轉折 |

更長週期：
- 三期一巡 = 76日
- 三巡一環 = 226日
- 三環一巡環 = 676日

### 對等數值（時間論的精髓）

對等數值是時間論中最實戰的部分：**過去的變化日之間經過的日數，就是未來變化日的參考**。

例如：
- 前一個高點到另一個高點之間經過了31天
- 那麼從最近一個高點往後推31天，就是下一個可能的轉折日

**實戰案例**（味之素 2802 週足）：
- 高點B到高點D之間經過31天
- 高點D到高點F之間經過30天 → **對等！**
- 高點D到低點G之間經過33天
- 低點G到低點I之間經過34天 → **幾乎對等！**

> ⚠️ 對等數值不需要是9、17、26的倍數，而是從實際走勢中尋找對應的時間間隔。一目山人的弟子曾說：「一目均衡表最重要的是時間的思考方式。」

### 時間論實戰步驟

1. **標記過去的轉折日**：找出近期所有重要高點和低點
2. **計算時間間隔**：量測高點到高點、低點到低點的日數
3. **尋找對等**：如果出現相近的間隔（允許±1~2日誤差），就找到對等數值
4. **推算未來轉折日**：從最近的轉折日加上對等數值，預測下一個轉折日
5. **用基本數值驗證**：如果對等數值接近9、17、26、33等基本數值，可信度更高

## 二、波動論：判斷趨勢結構

波動論將市場的價格運動歸納為三種基本形態，並告訴你「趨勢何時會結束」。

### 三種基本波動

| 波動類型 | 形態 | 意義 |
|----------|------|------|
| I型波動 | 單純上漲或單純下跌 | 趨勢最明確，力量最強 |
| V型波動 | 上漲→下跌 或 下跌→上漲 | 反轉急劇，代表方向改變 |
| N型波動 | 上漲→下跌→上漲 | 最基本的完整波動形態 |

### 關鍵法則：N型波動完成前不會反轉

這是波動論最核心的實戰法則：

> **上漲開始的價格，在N型波動完成之前，不會轉為下跌；下跌開始的價格，在N型波動完成之前，不會轉為上漲。**

翻譯成白話：如果你確認目前是上漲的N型波動（上漲→回調→再上漲），那麼在第二個上漲波完成之前，趨勢不會反轉。

### 波動演化過程

```
I型 → I型 → V型 → I型 → V型 → N型
（單純趨勢）  （反轉）   （反轉）  （完整結構）
```

I型和V型是市場的不完整狀態，最終都會演化成N型。N型波動對應艾略特波浪理論的5-3結構，但一目均衡表的分類更簡潔。

### 波動論實戰判斷

1. **辨識目前是哪種波動**：I型（持續方向）、V型（反轉）、N型（完整結構即將結束）
2. **I型波動時**：趨勢最強，可以加碼追隨
3. **V型波動時**：方向已改變，但要確認是否會演化成N型
4. **N型波動完成時**：原趨勢可能結束，準備防守或反手

## 三、值幅觀測論（水平論）：計算目標價

值幅觀測論是用過去的價格幅度，計算未來的目標價。這是一目均衡表最實用的出場工具。

### 四種計算方法

設定：A是起漲點（低點），B是第一波高點，C是回調低點

#### 1. N計算值（等幅測量）

**公式**：目標價 = C + (B - A)

**邏輯**：第3波的上漲幅度 = 第1波的上漲幅度

**實例**：
- A = 776（起漲點）
- B = 955（第一波高點）
- C = 755（回調低點）
- N計算值 = 755 + (955 - 776) = 755 + 179 = **934** → 實際高點F = 953 ✓

#### 2. E計算值（倍幅測量）

**公式**：目標價 = B + (B - A)

**邏輯**：第3波的上漲幅度 = 第1波上漲幅度的2倍

**實例**：
- D = 871（起漲點）
- E = 753（回調低點，這裡E是第二波回調）
- 實際用B和C計算：E計算值 = 871 + (871 - 755) = 871 + 116 = **987** → 實際高點H ≈ 1014 ✓

#### 3. V計算值（調整測量）

**公式**：目標價 = B + (B - C)

**邏輯**：第3波的上漲幅度 = 第2波回調幅度的2倍

**實例**：
- D = 871（高點）
- E = 753（回調低點）
- V計算值 = 871 + (871 - 753) = 871 + 118 = **989** → 實際高點H ≈ 1014 ✓

#### 4. NT計算值（差幅測量）

**公式**：目標價 = C + (C - A)

**邏輯**：第3波的上漲幅度 = 第1波幅度與第2波調整幅度的差

### 四種計算值比較表

| 計算法 | 公式 | 邏輯 | 適用情境 |
|--------|------|------|----------|
| N計算值 | C+(B-A) | 第3波=第1波 | 最常見，預設目標 |
| E計算值 | B+(B-A) | 第3波=第1波×2 | 強勢趨勢 |
| V計算值 | B+(B-C) | 第3波=回調×2 | V型反轉 |
| NT計算值 | C+(C-A) | 第3波=第1波-回調 | 弱勢反彈 |

### 值幅觀測論實戰步驟

1. **標記A、B、C三點**：找出最近的起漲點、第一波高點、回調低點
2. **同時計算四種目標價**：N、E、V、NT
3. **設定多個目標區間**：如果2~3種計算法指向相近的價位，可信度最高
4. **搭配時間論驗證**：在目標價附近是否有時間論的轉折日？
5. **搭配雲帶驗證**：目標價是否在雲帶附近？雲帶會形成支撐/壓力

> 💡 **值幅觀測論最強用法**：當N計算值和E計算值指向相近的價位，且該價位附近有雲帶壓力，就是最可靠的目標價判斷。

## 四、三大理論聯合判讀（最核心）

### 三理論共振進場法

單獨使用三役好轉的勝率約50-55%，但加上時間論和值幅觀測論可以提高至65-70%：

**進場確認清單**：
1. ✅ 三役好轉成立（趨勢確認）
2. ✅ 時間論轉折日附近（時機確認）
3. ✅ 波動論確認N型波動進行中（結構確認）
4. ✅ 值幅觀測論目標價有足夠空間（利潤確認）

**出場確認清單**：
1. 價格接近N計算值或E計算值目標價
2. 接近時間論的轉折日
3. 波動論判斷N型波動即將完成
4. 遇到雲帶壓力

### 實戰案例：味之素（2802）

**時間論驗證**：
- 高點B→高點D：31天
- 高點D→高點F：30天 → **對等！預測下一個高點也在31天左右**
- 高點D→低點G：33天
- 低點G→低點I：34天 → **對等！預測下一個低點也在33天左右**

**值幅觀測論驗證**：
- E計算值：871+(871-755) = 987 → 實際H=1014（誤差2.7%）
- N計算值：755+(955-776) = 934 → 實際F=953（誤差2.0%）
- V計算值：871+(871-753) = 989 → 實際H=1014（誤差2.5%）

三種計算法都指向934-989的目標區間，與實際高點955-1014相當吻合！

## 五、進階：遲行線與時間論的結合

遲行線是一目均衡表最被低估的組成部分，與時間論結合使用威力加倍：

1. **遲行線+對等數值**：遲行線穿過K線的日期，如果接近時間論的轉折日，反轉訊號的可信度大幅提升
2. **遲行線確認**：遲行線只是輕觸K線就折回 ≠ 轉勢，反而可能預示更強趨勢
3. **26日對稱**：遲行線本身用26日參數，這正是時間論的三節（26日），兩者互相驗證

## 六、雲帶交叉點的時間論判讀

先行帶A與先行帶B交叉的地方稱為「交叉點」，是市場劇烈波動的預兆：

- **交叉點附近的時間**：如果交叉點出現在時間論的轉折日附近，變盤機率極高
- **雲帶變薄**：交叉前雲帶會變薄，代表多空力量趨於均衡
- **穿越雲帶的時間**：從進入雲帶到穿越雲帶的時間，如果接近基本數值（9、17、26），穿越後趨勢更可靠

## 注意事項

### 時間論的誤區
- 基本數值不是精確到日的預測，而是「附近幾天容易出現轉折」的時間窗口
- 對等數值比基本數值更實戰，但要允許±1~2日的誤差
- 時間論只能告訴你「何時注意」，不能告訴你「方向是什麼」——方向要靠波動論和值幅觀測論判斷

### 波動論的誤區
- N型波動「完成前不會反轉」是大方向原則，不是絕對法則
- 大N型波動中可能包含小N型波動（嵌套結構），需要多時間框架判斷
- 波動論和艾略特波浪理論相似但更簡潔，不要混用兩者的命名

### 值幅觀測論的誤區
- 四種計算法同時計算，不要只選最接近實際價格的那一種（那是事後選擇偏誤）
- 計算出來的目標價是「參考區間」不是「精確價位」
- 目標價附近有雲帶時才是最可靠的目標，否則可能只是中途站

### 一目均衡表建議用日足
- 一目山人本人說過「一目均衡表應該只在日足使用」
- 時間論的基本數值以日足為基礎
- 短時間框架（5分鐘、15分鐘）假訊號太多
- 如果要用週足或月足，可以搭配對應的時間數值
