# Volume Profile 成交量分佈分析

> 把成交量從時間軸轉到價格軸，看見市場真正的成本結構——POC是公認價格、VA是多數人認同的價格區、LVN是價格快速穿越的不認同區

## 核心概念

Volume Profile（簡稱VP）是一種將成交量以橫條形式顯示在特定價格區域上的技術指標。傳統成交量柱狀圖按「時間」排列，VP 則按「價格」排列，讓你一眼看出哪個價位成交最多、哪個價位無人問津。

### 四個核心組成

| 組成 | 說明 |
|------|------|
| **POC（Point of Control）** | 成交量最高的價格，橫條最長的那一條，代表市場最認同的公允價格 |
| **VA（Value Area）** | 以 POC 為中心，累積成交量達 70% 的價格範圍，代表多數人認同的價格區 |
| **VAH / VAL** | Value Area 的高點和低點，盤整盤的箱型上下緣 |
| **LVN（Low Volume Node）** | 成交量低於平均的價位，價格快速穿越的「不認同區域」 |

### HVN 與 LVN

- **HVN（High Volume Node）**：高成交量區，代表市場認同的價格帶，價格在此停留時間長
- **LVN（Low Volume Node）**：低成交量區，代表市場不認同的價格帶，價格在此快速穿越

HVN 像磁鐵，價格傾向回歸；LVN 像真空，價格快速通過。

### 三種顯示方式

1. **Fixed Range**：框選特定時間範圍計算，適合回顧歷史
2. **Visible Range**：根據可視範圍計算，最常使用
3. **Session Volume**：每個交易日獨立計算，適合日內交易

## 實戰應用

### 三大市場定律

VP 揭示了三種基於交易情緒的市場定律，這些定律來自「人對盈虧的反應」，是恆久不變的：

#### 1. 回歸均值
- 價格偏離 POC 過遠時，通常會回歸到該區域
- VA 內的價格會持續震盪，像被磁鐵吸引
- VP 橫條最終會形成常態分佈圖形

#### 2. 回檔支撐
- 上升趨勢中，價格回到 VA 或 POC 位置時出現支撐
- 大多數人處於獲利狀態，認為「打折了」而買入
- 錨定效應：交易者對先前走勢的印象產生支撐反應

#### 3. 反彈阻力
- 下跌趨勢中，價格反彈至 VA 或 POC 位置時形成阻力
- 大多數人處於虧損狀態，反彈是「減少損失」的機會
- 賣盤出籠形成阻力

### 趨勢中的 VP 形狀

#### 上漲趨勢：正三角形
- 下方 HVN 大於上方 HVN
- 股價正在向上攀升，尋找更高的認同價
- 強勢股回測時不會回到下方 POC，應在兩個高原區中間就會反彈

#### 上漲結束：倒三角形
- 上方 HVN 大於下方 HVN
- 代表上漲可能結束，進入盤整
- 最近的高原區比舊的高原區更重要

#### 多個 HVN 的解讀
- 一個可視範圍可以出現多個 VA 高原區
- 最近的高原區比舊的更重要
- 強勢股票回測時應在兩個高原區中間反彈，不會回到下面 POC

### 應用一：找底部型態 Pivot Point

股價在低檔修正時，底部型態（頭肩底、杯帶柄、VCP等）的頸線可以用 VP 更精確定位：

- **VA 高原區與低谷區的界線**通常在比頸線更低的價格
- Pivot point 就在高原與低谷區的界線
- Mark Minervini 稱為 3C Pattern（Cup-completion-cheat）pivot point
- 突破 Pivot point 時，大部分人都已經賺錢，動能已偏多

**突破確認四要件**：
1. 突破當下要出量（超過平均量 50%）
2. 突破前最後一次下跌（洗盤）很快失敗，上漲量縮
3. 向上突破前的出發位置靠近 VAH
4. 突破後價格持續向上，讓短空單停損

**第一次回測**：從底部剛啟動的股票，常會回測頸線，第一個回測能站穩頸線之上，都是好的買入點。

### 應用二：區間逆勢操作

在盤整盤中使用 VAH 和 VAL 進行逆勢交易：

- **VAH 當箱型上緣**：向上突破失敗時做空
- **VAL 當箱型下緣**：向下突破失敗時做多
- 失敗的突破有很大機會回到 POC（第一目標）
- 第二目標是跌到 VAL（箱型下緣）
- 逆勢單勝率高但利潤低

### 應用三：選擇權履約價挑選

用 VP 的支撐區來選擇選擇權履約價：

- **Sell Put**：選擇 Pivot point 附近的履約價，即使被履約也在支撐區
- **Buy Call**（抄底暴跌股）：履約價設在上方的 VAL 到 POC 之間，因為股價上漲最可能在此受阻力

### VP 與其他指標搭配

VP 是**確認性指標**，不預測趨勢，需要與其他工具搭配：

1. **趨勢分析**（均線、趨勢線）→ 判斷方向
2. **型態分析**（VCP、杯帶柄）→ 找進場點
3. **VP** → 確認支撐阻力位置

實戰範例：
- 使用 ATR 突破策略時，結合 VP 確認價格在 POC 之上且無重大阻力區
- 使用 1 小時線 ATR 策略，搭配 20 日 VP 確認
- 搭配 VCP 型態，用 VP 抓取 Pivot point

### 適當的時間範圍選擇

| 交易週期 | VP 時間範圍 | 用途 |
|----------|------------|------|
| 日內交易 | Session Volume | 日內重要價位 |
| 波段交易 | 20 日 Fixed Range | 確認突破與支撐 |
| 中期修正 | 6 個月～1 年 Fixed Range | 觀察盤整型態，抓 VCP Pivot point |
| 漲幅 100%+ | 不宜太長 | 低價時交易量遠大於高價，會失真 |

## 注意事項

### VP 的限制
1. **無法預測趨勢**：VP 是確認指標，不是預測指標，必須搭配趨勢分析
2. **解讀因人而異**：VP 的圖形解讀沒有精確定義，很吃經驗
3. **無法程式回測**：TradingView 的 Pine Script 沒有 VP function，無法寫成程式回測
4. **時間範圍影響大**：不同時間範圍的 VP 圖形差異很大，需要根據交易週期選擇
5. **漲幅大的股票會失真**：低價時的成交量遠大於高價時，時間範圍不宜太長

### 常見誤區
1. **把 POC 當支撐/壓力的唯一判斷**：POC 只是成本中心，不代表一定撐住
2. **忽略 VA 的寬度**：寬 VA 代表共識強、窄 VA 代表分歧大
3. **忘記 LVN 的意義**：LVN 不是「不重要」，而是「價格快速通過」，突破 LVN 需要強大量
4. **VP 形狀的過度解讀**：正三角/倒三角只是參考，需要搭配其他分析確認

### 與 VWAP 的區別
- **VWAP**：當日成交量的加權均價，是「流動性基準線」
- **VP**：跨時間的成交量價格分佈，是「成本結構圖」
- VWAP 偏日內交易，VP 偏跨日分析，兩者互補
