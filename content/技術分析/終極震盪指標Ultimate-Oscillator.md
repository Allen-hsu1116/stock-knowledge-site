# 終極震盪指標 Ultimate Oscillator (UO)

> Larry Williams 的三重週期加權震盪指標，超買超賣不是用來撈頂撈底，背離+突破確認才是正確用法

## 核心概念

終極震盪指標（Ultimate Oscillator, UO/UOS）由 Larry Williams 創立，是少數將三個不同時間週期（7/14/28）的動量資訊整合為單一指標的設計。

**設計理念**：
Larry Williams 認為所有震盪指標都對參數選擇極度敏感——不同市況配不同參數，結果截然不同。他的解法是：**不要選一個參數，把三個週期全部用上，然後加權**。

**核心計算 — 買入壓力（Buying Pressure）**：
- 今日真正低點 TL = min(今日最低價, 昨日收盤價)
- BP = 今日收盤價 - TL（衡量從當日真正低點到收盤的買入力量）

這個設計比傳統收盤價減開盤價更精準——如果開高走低，傳統方法會誤判為賣壓，但 UO 能識別出「從真正低點反彈」的買入力量。

**三重週期加權（4:2:1）**：
- 7期 × 4 = 最重視（最貼市）
- 14期 × 2 = 中等權重
- 28期 × 1 = 最輕視（最遠期）

結果歸一化到 0-100 區間：UOS = (RawUO / 7) × 100

## 實戰應用

### ⚠️ UO 的超買超賣不是傳統用法！

**這是 UO 最容易被誤解的地方**：
- Larry Williams 明確表示：**不要用 70 超買區賣出、30 超賣區買入**
- 單純用超買超賣水平入市，準確度低，長炒一定輸多贏少
- 超買超賣區是用來「篩選訊號」，不是用來「直接交易」

### 正確用法：三步背離交易法

**做多（買升）三步驟**：
1. **背離形成**：價格一底比一底低，但 UO 一底比一底高（多頭背離）
2. **超賣確認**：UO 的第一個底部必須曾經跌到 30 以下（確保超賣條件滿足）
3. **突破確認**：UO 升穿兩個底部之間的高點（頸線），正式進場買升

**做空（買跌）三步驟**：
1. **背離形成**：價格一頂比一頂高，但 UO 一頂比一頂低（空頭背離）
2. **超買確認**：UO 的第一個頂部必須曾經漲到 70 以上（確保超買條件滿足）
3. **突破確認**：UO 跌穿兩個頂部之間的低點，正式進場買跌

### 進階訊號

| 訊號 | 條件 | 意義 |
|------|------|------|
| 短線買進 | UO 跌破35後回升突破35 | 超賣區反彈 |
| 強勢確認 | UO 突破65 | 氣勢極強，延長波開始 |
| 短線賣出 | UO 從50-70區間跌破50 | 多頭動能減弱 |
| 中線賣出 | UO 從70以上跌破70 | 多頭趨勢結束 |
| 空頭背離 | 價格創新高 + UO未創高 + UO曾<35 | 趨勢反轉警告 |
| 多頭背離 | 價格創新低 + UO未創新低 + UO曾>50 | 底部反轉訊號 |

### 出場策略
- UO 突破 70 後，等 UO 再跌破 70 時出場
- 或等 UO 與價格產生背離時出場

## 注意事項

1. **背離必須搭配超買超賣確認**：空頭背離的 UO 必須曾經到過 35 以下，多頭背離的 UO 必須曾經在 50 以上，否則背離不可信
2. **不要只看超買超賣**：這是最常見的錯誤，UO 的 70/30 是篩選條件不是交易條件
3. **三重週期設計的優勢**：比單一週期的 RSI 或 KD 更穩定，不會因為參數選擇而失準
4. **4:2:1 權重的邏輯**：近期數據佔比大（4/7 ≈ 57%），遠期數據佔比小（1/7 ≈ 14%），這是 Larry Williams 測試後的最佳加權比例
5. **盤整市仍會有假訊號**：背離後的突破確認是過濾盤整假訊號的關鍵，不要在背離形成時就急著進場
