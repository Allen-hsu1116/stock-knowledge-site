# MFI資金流量指標 Money Flow Index

> RSI的量價升級版，加入成交量判斷資金流向，背離訊號比RSI更可靠，但盤整盤一樣會被洗

## 核心概念

MFI（Money Flow Index，資金流量指標）又稱資金流向指標，是技術分析中用來衡量買賣壓力的震盪指標。它與RSI的計算邏輯相似，但關鍵差異是**加入了成交量**，因此也被稱為「成交量加權RSI」（Volume RSI, VRSI）。

MFI的核心思想：**量大的一天在計算中權重更大**。如果某天成交量是另一天的2倍，那天的價格數據對MFI的影響也是2倍。這讓MFI比純價格指標更能反映真實的資金動向。

### 計算公式

1. **典型價格** Typical Price =（最高價 + 最低價 + 收盤價）/ 3
2. **資金流量** Money Flow = 典型價格 × 成交量
3. 當天典型價格 > 前一天 → 正資金流入；反之 → 負資金流入
4. **資金流量比率** MF Ratio = 14天內正資金流入 / 14天內負資金流入
5. **MFI** = 100 − 100 /（1 + MF Ratio）

MFI值介於0～100之間，可以理解為「買方佔買賣雙方總力道的比例」。

### MFI vs RSI 關鍵差異

| 比較項目 | RSI | MFI |
|---------|-----|-----|
| 計算基礎 | 價格漲跌 | 價格漲跌 + 成交量 |
| 量價關係 | 只看漲跌方向 | 大量日的權重更高 |
| 背離可靠度 | 一般 | **比RSI更可靠**（MBA智庫長期測試結論） |
| 適用場景 | 一般趨勢判斷 | 資金流向判斷、放量背離 |

## 實戰應用

### 一、超買超賣判斷

- **MFI > 80**：超買區，資金短期過熱，留意反轉
- **MFI < 20**：超賣區，資金短期冷卻，留意反彈
- **MFI > 90 或 < 10**：極端值，很少出現

⚠️ **重要陷阱**：強勢趨勢中MFI可能長期停留在超買或超賣區，超買不代表立即下跌、超賣不代表立即上漲。必須結合K線型態確認。

### 二、背離訊號（最核心用法）

MFI背離比RSI背離更可靠，因為加入了成交量因素：

**看漲背離（底部背離）**：股價創新低，MFI卻出現較高的低點
- 含義：拋售壓力正在減少，買方即將接管
- 操作：留意買入機會

**看跌背離（頂部背離）**：股價創新高，MFI卻出現較低的高點
- 含義：購買壓力正在減少，賣家即將接管
- 操作：留意賣出或減倉機會

💡 **背離在4小時或日線圖上更容易找到有效訊號**，短週期假訊號太多。

### 三、失敗波動（Failure Swing）

純看MFI數據、不需要看價格的反轉訊號：

**看漲失敗波動（買入訊號）**：
1. MFI跌破20（超賣）
2. MFI反彈至20以上
3. MFI回調，但仍高於20
4. MFI突破前高 → **買入**

**看跌失敗波動（賣出訊號）**：
1. MFI升至80以上（超買）
2. MFI跌回80以下
3. MFI小幅反彈，但仍低於80
4. MFI跌破前低 → **賣出**

### 四、MFI + 均線搭配策略

將MFI與30期SMA結合：
- MFI上穿30 SMA + MFI < 50 → 做多訊號
- MFI下穿30 SMA + MFI > 50 → 做空訊號

### 五、MFI 50 分水嶺

MFI值在50附近時，表示多空力道接近均衡：
- MFI > 50：買方力道佔優
- MFI < 50：賣方力道佔優
- 可作為多空判斷的輔助濾網

## 注意事項

### 六大陷阱

1. **盤整盤失效**：MFI在盤整市中容易頻繁進出超買超賣區，假訊號大量出現
2. **不考慮漲跌幅度**：只看方向和成交量，大漲1%和小漲0.1%對MFI的影響一樣（方向相同）
3. **參數週期不宜太短**：太短會產生過多假訊號，建議用14日預設值，4小時或日線以上週期
4. **不能單獨使用**：MFI是輔助指標，必須搭配趨勢指標（均線、支撐壓力）和K線型態綜合判斷
5. **極端行情延遲**：突發大利多或利空時，MFI可能來不及反應
6. **順勢vs逆勢解讀**：同一個MFI值，有人順勢解讀（MFI>80趨勢強），有人逆勢解讀（MFI>80即將反轉），不存在標準答案

### MFI vs OBV 差異

| 指標 | 計算方式 | 特點 |
|------|---------|------|
| MFI | 典型價格 × 成交量，比值計算 | 震盪指標0-100，有超買超賣區間 |
| OBV | 累加成交量，上漲加下跌減 | 累積值無上限，看趨勢方向 |
| AD | 收盤位置加權成交量 | 介於兩者之間 |

MFI適合判斷超買超賣和背離，OBV適合判斷長期資金流向趨勢。

### 最佳搭配組合

- **MFI + 均線**：均線判趨勢方向，MFI判資金動能
- **MFI + 布林通道**：布林通道判波動和支撐壓力，MFI判資金進出
- **MFI + KD/RSI**：量價共振確認，多指標背離同時出現勝率最高
- **MFI + 成交量**：MFI背離 + 量價背離 = 最強反轉訊號
