# SuperTrend超級趨勢指標

> 基於ATR的趨勢追蹤指標——以階梯式線條明確顯示方向，既是趨勢判斷工具，也是移動停利系統

## 核心概念

SuperTrend（超級趨勢指標）是由 Olivier Seban 開發的趨勢追蹤指標，基於 ATR（平均真實波幅）計算。它在價格圖上以階梯式線條呈現，明確顯示當前趨勢方向，同時可作為移動停利和反手操作的依據。

### 計算公式

**三個參數**：ATR 期間（預設14）、倍數（預設2或3）、中心點

1. **TR（真實波幅）** = max(當日最高-當日最低, |當日最高-前日收盤|, |前日收盤-當日最低|)
2. **ATR** = TR 的 N 日移動平均（預設14日）
3. **中心點** = (最高價 + 最低價) / 2
4. **上軌** = 中心點 + 倍數 × ATR
5. **下軌** = 中心點 - 倍數 × ATR

### 翻轉邏輯

- 收盤價突破上軌 → **翻多**（顯示紫色/綠色線在上軌位置）
- 收盤價跌破下軌 → **翻空**（顯示線在下軌位置）
- 未突破未跌破 → **維持前一方向**
- **永遠在場**：不是多單就是空單

## 實戰應用

### 1. 趨勢判斷

- 紫色線往上 = 多頭趨勢
- 綠色線往下 = 空頭趨勢
- 翻轉 = 方向改變，考慮反手

### 2. 移動停利

SuperTrend 的階梯式特性非常適合作為移動停利：
- 做多時：停利設在 SuperTrend 線下方，不跌破就繼續抱
- 做空時：停利設在 SuperTrend 線上方，不突破就繼續抱
- 獲利自動隨趨勢延伸，賺賠比自然拉開

### 3. 股票操作策略

- **選股只做趨勢**：進場後用 SuperTrend 當停損停利
- 不跌破 SuperTrend 的就是好股票，跌破就是假上漲
- **加碼策略**：看對方向後逐步加碼，停損小、獲利大
- 賺賠比可達 10:1 以上（小停損換大波段）

### 4. 期貨操作策略

- 先判斷趨勢盤或盤整盤
- 趨勢盤用 SuperTrend，盤整盤用 KDJ/RSI
- 進場後用移動停利持續抱住部位
- 停損設定：可根據 ATR 百分比（如 20% ATR）或區間K棒高低點

### 5. 大週期保護小週期

設定兩組不同參數的 SuperTrend：
- **短線參數**（如 1倍 ATR）：反應快，進出頻繁
- **長線參數**（如 3倍 ATR）：過濾盤整，方向明確
- **策略**：長短同方向才進場，跌破短線先出一半，短線再翻多可補回

## 參數調整

| 參數倍數 | 特性 | 適用場景 |
|----------|------|----------|
| 1倍ATR | 貼近K線，反應極快 | 短線交易、當沖 |
| 2倍ATR | 平衡靈敏與穩定（預設） | 波段操作 |
| 3倍ATR | 距離較遠，過濾雜訊 | 中長線趨勢 |
| 4倍ATR | 距離很遠，看大方向 | 長線趨勢確認 |

**原則**：倍數越小 → 越靈敏但假訊號多；倍數越大 → 越穩定但反應慢

## 注意事項

### 盤整市三大問題

1. **頻繁假突破**：SuperTrend 在盤整區會被來回掃，進出多次都虧損
2. **無法判斷盤整**：指標本身不會告訴你現在是盤整盤
3. **虧損累積**：多次小虧會侵蝕一次大賺的利潤

### 解決方案

1. **搭配逆勢指標**：KDJ、RSI 等，盤整時切換使用
2. **只做趨勢盤**：錯的砍掉，不浪費時間在盤整股
3. **大週期保護小週期**：長短兩組參數同方向才進場
4. **搭配 ADX 過濾**：ADX > 25 表示有趨勢，ADX < 20 表示盤整

### 其他限制

- 均線型指標的通病：落後指標，翻轉訊號有延遲
- 不同市場需要調整參數，沒有一組參數通吃
- 反手操作需要承擔兩次手續費成本
- ATR 在跳空開盤時可能產生異常值

## 與其他趨勢指標比較

| 指標 | 優勢 | 劣勢 |
|------|------|------|
| SuperTrend | 階梯式、移動停利、反手操作 | 盤整被掃 |
| [[SAR拋物線指標Parabolic-SAR]] | 參數少、視覺清晰 | 加速因子可能過快 |
| [[均線判斷]] | 最常用、參數多元 | 交叉訊號落後 |
| [[鱷魚線Alligator-Indicator]] | 三線判斷方向明確 | 睡覺期（盤整）無訊號 |

SuperTrend 的核心優勢是**同時提供方向判斷和移動停利**，這是 SAR 和均線不容易做到的。
