# CCI商品通道指標 Commodity Channel Index

> 衡量價格與統計均值的偏離程度，CCI>100超買、CCI<-100超賣，但核心用法是0軸突破做順勢交易，不是猜頭抄底

## 核心概念

CCI（Commodity Channel Index）由Donald Lambert於1980年創立，最早用於期貨市場。CCI屬於震盪指標，但與一般震盪指標不同——它**沒有固定的上下限**（不像KD或RSI在0-100之間），數值可以超過±200甚至更高。

### CCI的本質：價格與均線的乖離關係

CCI的核心其實是描述**K線與均線的關係**：
- **CCI > 0**：K線在均線上方（多方區域）
- **CCI < 0**：K線在均線下方（空方區域）
- **CCI = 0**：K線等於均線（多空分界）
- CCI參數是多少，就代表K線跟多少MA的關係（CCI14 = K線與14MA的關係）

### 計算公式

```
CCI = (TP - MA) / (0.015 × MD)

TP（典型價格）= (最高價 + 最低價 + 收盤價) / 3
MA = n日的TP移動平均
MD = TP與MA的平均偏差
```

常數0.015是Lambert設定的縮放因子，讓大多數CCI值落在±100範圍內。使用Typical Price而非單純收盤價，使CCI更能敏感反應市場價格變動。

### 與乖離率的差異

類似的指標有移動平均乖離率（BIAS），但CCI是根據**平均差原則**計算，因此能反應價格的**波動性（volatility）**，而不僅僅是偏離百分比。

## 實戰應用

### 一、0軸突破做順勢交易（最重要）

CCI最重要的用法不是超買超賣，而是**0軸突破**：

| 訊號 | 意義 | 操作 |
|------|------|------|
| CCI由下往上突破0軸 | K線突破均線，轉強 | 做多進場 |
| CCI由上往下跌破0軸 | K線跌破均線，轉弱 | 做空進場 |

**順勢交易策略**：
1. 用雙EMA（12EMA+26EMA）判斷方向
2. 多方：K線在短均及長均之上 → 等價格拉回均線 → CCI突破0軸進場做多
3. 空方：K線在短均及長均之下 → 等價格反彈均線 → CCI跌破0軸進場做空
4. 停損設在轉折低（多）或轉折高（空）

### 二、超買超賣區的逆勢用法

CCI常見的±100閾值：
- **CCI > +100**：超買區
- **CCI < -100**：超賣區

⚠️ **重大誤區**：CCI跌破-100就買進是危險的！空方盤才會跌破-100，貿然進場容易被繼續下跌傷害。

**正確做法**：
- 超賣區買點：等CCI**向上突破-100**，開始往0軸靠攏時才買進
- 超買區賣點：等CCI**跌破+100**，開始往0軸靠攏時才賣出
- 在趨勢盤中CCI可以長期停留在±100以外，不要猜頭抄底

### 三、CCI背離

動量通常在價格變化之前率先變化：

| 背離類型 | 條件 | 意義 |
|---------|------|------|
| **看漲背離** | 價格創較低低點 + CCI創較高低點 | 下跌動能減弱，可能反轉向上 |
| **看跌背離** | 價格創較高高點 + CCI創較低高點 | 上漲動能減弱，可能反轉向下 |

### 四、順勢操作法（OANDA方法）

- CCI由下往上突破+100 → 買進
- CCI由上往下跌破-100 → 賣出
- 平倉時機：CCI重新回到±100內側

⚠️ 當市場劇烈變動，CCI同時超過±100和±200時，**不進場交易**。

### 五、參數調整

| 參數 | 適用場景 | 特性 |
|------|---------|------|
| CCI(14) | 日線波段（預設） | 平衡靈敏度與穩定性 |
| CCI(20) | 中長線 | 較為和緩，假訊號較少 |
| CCI(5) | 短線當沖 | 靈敏但假訊號多 |

期間較短容易反應細微價格變動，期間較長則呈現較為和緩的變化。

## 注意事項

### 五大盲點

1. **盤整市假訊號多**：CCI是趨勢追蹤指標，對平靜和震盪市場適應性差，盤整時容易頻繁突破0軸產生假訊號
2. **超買超賣不代表反轉**：在趨勢盤中CCI可以長期停留在±100以上，超買可能代表趨勢加強而非反轉
3. **單一指標不足**：CCI無法給出具體的交易訊號，必須搭配均線、支撐壓力等其他指標
4. **波動小的市場不適用**：CCI適合價格波動明顯的市場，波動小的市場容易產生誤差
5. **不同資產閾值不同**：波動較大的資產超買超賣閾值可能為±200，不要死守±100

### 搭配建議

- **CCI + 雙EMA**：用均線判斷趨勢方向，CCI找進場時機
- **CCI + ADX**：ADX過濾盤整，CCI在趨勢明確時找0軸突破點
- **CCI + 布林通道**：布林通道判斷波動收縮/擴張，CCI確認方向
- **CCI + KD/RSI**：震盪指標組合共振確認超買超賣

### 核心心法

> **CCI順勢交易的核心：不是在超買區放空、超賣區買進，而是用0軸突破確認趨勢轉強/轉弱，順著方向做。超買超賣是趨勢的燃料，不是轉折的訊號。**
