# 威科夫方法 Wyckoff Method

> 威科夫方法是看懂主力從吸籌到出貨全生命週期的經典方法，市場結構的鼻祖，四大法則解讀供需、因果與量價關係。

## 核心概念

理查·威科夫（Richard Wyckoff）在 1930 年代提出的市場分析框架，核心是觀察「複合主力」（Composite Operator, CO）的行為。威科夫與道氏、江恩、艾略特並列技術分析五大巨人，是少數真正在市場中致富的交易大師。

### 市場四階段

1. **吸籌（Accumulation）**：主力悄悄買進，量縮價穩
   - PS（Preliminary Support）：初步支撐
   - SC（Selling Climax）：賣出高潮，大量換手
   - AR（Automatic Rally）：自動反彈
   - ST（Secondary Test）：二次測試底部
   - Spring：假跌破吸最後浮動籌碼
   - SOS（Sign of Strength）：力道訊號
   - LPS（Last Point of Support）：最後支撐點 → 買點

2. **上漲（Markup）**：趨勢確立，價格上漲
   - 價格突破積累區上方阻力
   - 成交量顯著放大，回調幅度小
   - 高點和低點不斷抬高

3. **分發（Distribution）**：主力悄悄出貨
   - PSY（Preliminary Supply）：初步供給
   - BC（Buying Climax）：買入高潮
   - AR：自動回落
   - ST：二次測試高點
   - UTAD（Upthrust After Distribution）：假突破吸引買盤 → 賣點
   - SOW（Sign of Weakness）：弱勢訊號
   - LPSY：最後供給點

4. **下跌（Markdown）**：趨勢反轉向下
   - 價格跌破分配區下方支撐
   - 成交量放大，賣壓強
   - 反彈幅度小

### 威科夫四大法則

#### 1. 供需法則

價格變化由供需關係決定：
- 需求 > 供給 → 價格上漲（量價齊升）
- 供給 > 需求 → 價格下跌（量價齊跌）
- 供需平衡 → 價格橫盤（量增價穩）
- 價升量縮 → 上漲動力不足
- 價跌量縮 → 下跌動力不足
- 量增價穩 → 主力可能在吸籌或派發

#### 2. 因果法則

價格的運動幅度取決於之前的積累或分配的量：
- **「橫有多長，豎有多高」** — 底部長時間橫盤的股票一旦爆發漲幅驚人
- 短時間積累 → 突破後容易回落（假突破的根本原因）
- 假突破的本質：積累不足，持續上漲困難

#### 3. 努力與結果法則

大的成交量應對應大的價格波動：
- 量價協調：大量對應大K線 → 趨勢健康
- 量價不協調：量大但價格變動小 → 預示趨勢轉變
- **高位大量成交但價格不漲 = 主力出貨的警訊**

#### 4. 市場結構

- 支撐位：主力吸籌的關鍵區域
- 阻力位：主力派發的關鍵區域
- 積累範圍：低位橫盤整理，主力悄然吸籌
- 分配範圍：高位橫盤整理，主力逐步派發

### 假突破 vs 有效突破

- **假突破**：短暫突破後迅速回落，主力製造假象誤導散戶
  - 特徵：突破後快速回落，成交量異常
  - 應對：假突破後的回落是逆向操作的機會
- **有效突破**：突破後持續運行，成交量顯著放大
  - 特徵：突破後持續運行，量增
  - 應對：有效突破後的回調是順勢操作的機會

## 從主力視角看市場

### 主力操作四階段

1. **吸籌建倉**
   - 壓低建倉法：製造利空、大單賣出打壓，低價吸納
   - 橫盤建倉法：維持窄幅波動，隱蔽吸籌
   - 拉高建倉法：大幅買入推高股價，快速建倉

2. **洗盤**
   - 挖坑洗盤：製造下跌迫使不堅定者賣出
   - 橫盤洗盤：消磨耐心，清理浮籌
   - 高開低走洗盤：開盤拉高後迅速回落

3. **拉升**
   - 震盪拉升：拉升和調整交替進行
   - 快速拉升：短期連續大單買入

4. **派發出貨**
   - 高位震盪派發：反覆小幅拉升和回調
   - 快速派發：短時間大單賣出
   - 定向派發：大宗交易或協議轉讓

### 散戶策略建議

- 不要模仿大莊家的左側抄底思維
- 選擇順勢右側交易，借莊家的勢
- 威科夫方法至少需要週線級別的時間維度
- 培養耐心和紀律性

## 實戰應用

1. **辨識主力階段**：用成交量+價格行為判斷主力在吸籌還是出貨
2. **Spring 買點**：假跌破後快速回升 = 主力最後吸籌
3. **UTAD 賣點**：假突破後快速回落 = 主力最後出貨
4. **搭配量價**：吸籌期量縮、分發期放量
5. **供需法則判讀**：量大價不漲 → 供過於求 → 出貨警訊
6. **因果法則應用**：長時間橫盤 → 積累充足 → 突破後爆發力強
7. **努力結果法則**：高位大量但價格不動 → 主力出貨

## 注意事項

- 威科夫方法需要大量經驗判斷階段
- Spring 和 UTAD 不一定每次都出現
- 需搭配籌碼面（法人買賣超）確認
- 現代 SMC（Smart Money Concepts）是威科夫方法的簡化版
- 主觀性強，不同人對因果關係的理解各不相同
- 至少需要週線級別時間維度，短期交易不適用
- 需要搭配其他技術指標和籌碼面分析確認
