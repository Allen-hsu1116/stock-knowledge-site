# MACD指標實戰判讀 Moving Average Convergence Divergence

> MACD是動能趨勢指標之王，DIF快線看動能方向、DEA慢線確認訊號、柱狀圖視覺化強弱，零軸分多空、金叉死叉找時機、背離抓反轉，但盤整時完全失效

## 核心概念

### MACD是什麼

MACD（Moving Average Convergence / Divergence，指數平滑異同移動平均線），1970年代由Gerald Appel提出，是研判價格變化強度、方向、動能及趨勢週期的指標。

有別於RSI及KD指標的短期性質，MACD常作為**中期波段技術指標**，較適合中長線投資人使用。

### 三大組成元件

MACD由三個相互關聯的元素組成：

1. **DIF線（快線）**：短期EMA - 長期EMA
   - 公式：DIF = EMA(close, 12) - EMA(close, 26)
   - 反應最快，對市場變化最敏感
   - DIF > 0 代表短期動能強於長期，偏多
   - DIF < 0 代表短期動能弱於長期，偏空

2. **DEA線（慢線/訊號線）**：DIF的9日EMA
   - 公式：DEA = EMA(DIF, 9)
   - 是DIF的平滑處理，過濾雜訊
   - 比DIF更穩定，用來確認訊號

3. **柱狀圖（Histogram / OSC）**：DIF與DEA的差值
   - 公式：OSC = DIF - DEA（部分平台會乘以2）
   - 柱狀圖長度 = 快慢線距離
   - 最直觀的動能視覺化工具
   - 柱子變長 → 動能增強
   - 柱子變短 → 動能減弱（趨勢反轉預警）

### 參數設定

預設參數（12, 26, 9）：
- 12日EMA：短期均線
- 26日EMA：長期均線
- 9日EMA：訊號線平滑天數

這組參數經過幾十年市場驗證，**一般不需調整**，直接使用即可。

## 實戰應用

### 零軸判斷多空

**零軸是多空分界線**，這是MACD最基礎也最重要的判斷：

| 位置 | 含義 | 操作方向 |
|------|------|----------|
| DIF和DEA都在零軸上方 | 多頭市場 | 主要考慮做多 |
| DIF和DEA都在零軸下方 | 空頭市場 | 主要考慮做空 |
| DIF和DEA在零軸附近 | 方向不明 | 觀望為主 |

⚠️ **時效性重要**：MACD值需在零軸之上維持一段時間才算是偏多訊號，「曇花一現」式的攀上零軸不算多頭訊號。

### 黃金交叉與死亡交叉

**黃金交叉（買進訊號）**：
- DIF線從下方上穿DEA線
- 柱狀圖從負值變為正值
- 代表短期動能超越長期動能

**死亡交叉（賣出訊號）**：
- DIF線從上方下穿DEA線
- 柱狀圖從正值變為負值
- 代表短期動能弱於長期動能

⚠️ **不是所有金叉死叉都值得交易**：
- 零軸上方的黃金交叉（靠近零軸）→ 有效買進訊號
- 零軸下方的死亡交叉（靠近零軸）→ 有效賣出訊號
- 零軸上方遠離零軸的死亡交叉 → 往往只是回檔修正
- 零軸下方遠離零軸的黃金交叉 → 往往只是反彈修正

### 柱狀圖判讀技巧

柱狀圖是MACD最直觀的部分，比金叉死叉更**領先**：

1. **柱狀圖伸長** → 動能增強，趨勢加速
2. **柱狀圖縮短** → 動能減弱，趨勢可能反轉（早期預警）
3. **柱狀圖由負翻正** → 黃金交叉確認
4. **柱狀圖由正翻負** → 死亡交叉確認
5. **柱狀圖維持在零軸附近無法伸長** → 盤整格局

💡 **實戰重點**：柱狀圖縮短的時候就要提高警覺，不要等到金叉死叉出現才行動——那時候已經慢了。

### MACD背離判讀

背離是MACD最強大的功能之一，但也是最容易誤用的：

**多頭背離（底背離）**：
- 股價創新低，但DIF線沒有創新低（底部墊高）
- 代表空方力量衰退，可能出現底部
- 確認訊號：價格突破左邊長黑K的最高點

**空頭背離（頂背離）**：
- 股價創新高，但DIF線沒有創新高（高點降低）
- 代表多方力量衰退，可能出現頭部
- 確認訊號：價格跌破左邊長紅K的最低點

⚠️ **背離的三大限制**：
1. 不提示反轉的時間
2. 不提示反轉的位置
3. 必須搭配其他指標確認（如K線型態、支撐壓力）

### MACD抄底技巧

1. **DIF線底部墊高**：即使股價沒有背離，只要DIF線的底部墊高，就代表空方力量衰退
2. **背離 + 突破確認**：出現底背離後，等價格突破左邊長黑K最高點才進場
3. **零軸上方的黃金交叉**：最可靠的買進訊號

## 注意事項

### MACD的五大盲點

1. **盤整完全失效**：區間盤整時頻繁出現金叉死叉，依指標交易會反覆虧損手續費
2. **滯後性**：MACD從均線衍生，繼承了均線的滯後性。金叉死叉出現時，行情已經跑了一段
3. **背離不保證反轉**：背離只是警示，不是確定性訊號，需要其他確認
4. **鈍化問題**：強趨勢中MACD可能長期停留在超買或超賣區，失去參考價值
5. **短線不適用**：日內交易盈利空間小，MACD信號滯後會吃掉大量利潤

### 搭配建議

- **MACD + ADX**：用ADX過濾盤整，ADX > 25才用MACD訊號
- **MACD + RSI/KD**：用擺盪指標確認超買超賣，降低假訊號
- **MACD + 均線**：用均線判斷趨勢方向，MACD找進出場時機
- **MACD + 量價**：MACD金叉搭配放量突破更可靠
- **MACD + 支撐壓力**：在支撐壓力位附近的MACD訊號更有效

### 參數調整建議

- 日線：預設（12, 26, 9）最通用
- 週線：可以用（12, 26, 9），中長線趨勢判斷更準確
- 短線（60分K）：可考慮（5, 34, 5）或（8, 17, 9），但短線MACD效果本來就有限
- 搭配ADX使用時，參數調整意義不大
