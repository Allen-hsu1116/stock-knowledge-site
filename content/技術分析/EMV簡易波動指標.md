# EMV 簡易波動指標（Ease of Movement Value）

> 量價合成指標，衡量「價格移動的容易程度」：少量成交就能推動價格 → EMV 升高；大成交但價格不動 → EMV 趨近零。零軸穿越是主要交易訊號。

## 核心概念

EMV（Ease of Movement Value）簡易波動指標，由 Richard W. Arms, Jr. 根據等量圖（Equivolume Chart）和壓縮圖（Kagi Chart）的原理設計。它將價格變動與成交量結合，衡量價格移動的「容易程度」。

### 公式

```
MidPointMove = (今日最高 + 今日最低) / 2 - (昨日最高 + 昨日最低) / 2
BoxRatio = (成交量 / 10000) / (8 × (今日最高 - 今日最低))
EMV = MidPointMove / BoxRatio
```

**核心邏輯**：
- 價格移動幅度大 + 成交量小 → EMV 高（移動容易）
- 價格移動幅度小 + 成交量大 → EMV 低（移動困難）
- 價格不動或量大價不動 → EMV 趨近零

### EMV 的獨特哲學

EMV 採取與一般投資人**相反**的見解：
- 一般人追逐大成交量上漲
- **EMV 認為少量成交的漲勢才是健康的**，能保存元氣使漲勢更長遠
- EMV 曲線大部分時間在零軸下方，這是 EMV 的特色
- 過度急躁冒進的行情不被 EMV 看好

## 實戰應用

### 1. 零軸穿越（主要交易訊號）

| 訊號 | 意義 | 操作 |
|------|------|------|
| EMV 由下往上穿越零軸 | 價格向上移動容易，多頭力量增強 | 買進 |
| EMV 由上往下穿越零軸 | 價格向下移動容易，空頭力量增強 | 賣出 |

### 2. EMV 均線訊號

- **重要**：EMV 和 EMV 均線的交叉**沒有意義**
- 建議以 **EMV 均線穿越零軸** 為訊號（而非 EMV 本身）
- 均線穿越零軸比 EMV 本身穿越更穩定，過濾雜訊

### 3. EMV 與價格循環

下跌階段：
1. 股價下跌 → 買氣萎縮 → 成交量減少 → EMV 下降
2. 股價跌到支撐區 → 逢低買盤進入 → 成交量活躍 → EMV 向上攀升
3. EMV 由負值趨近零 → 信心資金成功扭轉跌勢

上漲階段：
1. EMV 由負值轉正值 → 買進訊號
2. 上漲過程中成交量緩慢遞增 → EMV 持續上升
3. 頭部成交量集中 → EMV 提前反應下降，趨近零
4. EMV 由正值轉負值 → 賣出訊號，行情正式反轉

### 4. EMV 的選股邏輯

- EMV 重視「移動長久且能產生足夠利潤的行情」
- 少量成交穩定上漲 → EMV 看好
- 大量急漲 → EMV 不看好（消耗力氣）
- **避免在人氣聚集且成交熱絡時買進**
- **在成交量逐漸展現無力感而多頭尚未察覺時賣出**

## 注意事項

1. **盤整市無效**：EMV 適合價格波動明顯的趨勢行情，牛皮整理時應放棄使用
2. **零軸下方居多**：EMV 大部分時間在零軸下方是正常現象，不代表一定偏空
3. **均線交叉無意義**：不要用 EMV 和均線交叉做訊號，要用均線穿越零軸
4. **不適合短線**：EMV 重視「緩慢穩定的趨勢」，短線急漲急跌的判讀效果差
5. **與一般認知相反**：EMV 不看好大成交量上漲，這與散戶「追量」的直覺相反，需要適應
