# 成交量確認原則Volume-Confirmation

> 沒有成交量支持的突破都是耍流氓——帶量突破才算真突破，量縮回測才算有效支撐，量價背離才是反轉確認

## 核心概念

成交量是價格的「燃料」和「確認器」。價格可以被短期消息或少數大戶操控，但成交量代表市場參與者的集體共識與資金動能。一個沒有成交量支持的漲跌，往往只是曇花一現的假象。

### 三大確認原則

| 原則 | 說明 | 實戰意義 |
|------|------|----------|
| 帶量突破 | 突破關鍵價位必須伴隨成交量顯著放大（≥5日均量1.5倍） | 確認突破真實性，過濾假突破 |
| 量縮回測 | 突破後回測關鍵價位時成交量萎縮 | 確認支撐有效，是理想第二買點 |
| 量價背離 | 價格創新高但成交量遞減，或價格破底但量能萎縮 | 確認趨勢即將反轉 |

### 成交量的三重角色

1. **趨勢確認器**：健康上漲趨勢伴隨量能溫和放大，下跌趨勢出現放量下殺或無量陰跌
2. **主力動向放大鏡**：法人、大戶的進出貨必然留下大量交易紀錄，突然鉅額成交量是主力操作信號
3. **市場情緒溫度計**：量放大=關注度高、多空分歧大；量萎縮=觀望、人氣低迷

## 實戰應用

### 帶量突破有效性判斷

**真突破三原則**：
1. **帶量**：突破當日成交量 ≥ 5日均量的1.5倍，最佳為2倍以上
2. **收盤站上**：收盤價站穩突破價位之上，不是盤中穿越就回落
3. **持續3天**：連續3個交易日收盤站在突破價位之上

**帶量突破均線**：
- 股價以帶量紅K站上月線/季線 = 強烈翻多信號，成交量確認突破有效性
- 突破後回測均線量縮 = 賣壓不大、籌碼穩定 = 理想第二買入點
- 帶量跌破重要均線 = 趨勢轉弱，考慮減碼或停損

**高檔爆量的陷阱**：
- 長期飆漲後高檔爆出歷史天量（天量天價）= 筹碼大量換手，主力可能出貨
- 高檔巨量長黑/長上影線（避雷針）= 主力高檔倒貨，強烈賣出信號

### 底部反轉的成交量訊號

1. **窒息量/凹洞量**：
   - 經過長期下跌後成交量萎縮到極致，形成像山谷般的凹洞
   - 窒息量：成交量不到20日內最大成交量的10%以下
   - 凹洞量：量縮到極低後接著量又回復
   - 代表賣壓竭盡、籌碼穩定，是底部即將完成的可靠信號
   - 窒息量/凹洞量後出現的第一根帶量長紅K棒 = 絕佳切入點

2. **爆量長紅（底部突破）**：
   - 底部盤整後突然出現巨大成交量 + 長紅K棒向上突破盤整區間
   - 主力資金發動攻擊的信號 = 新一輪漲勢的開始

3. **底部量價背離**：
   - 股價持續破底但成交量卻萎縮（價跌量縮）
   - 下殺動能衰竭、空方力量耗盡 = 趨勢即將反轉的潛在信號

### 頭部反轉的成交量訊號

1. **巨量長黑/避雷針**：高檔突然爆天量但股價開高走低 + 長上影線 = 主力高檔倒貨
2. **頭部量價背離**：股價持續創新高但成交量一波比一波低 = 追價買盤無力
3. **高檔出貨量**：高檔區間震盪看似強勢整理但成交量頻繁不尋常大量 = 主力分批出貨

### 成交量搭配指標確認

**成交量 + KD/RSI**：
- 低檔背離+放量：股價破底但KD/RSI未破底 + 爆量長紅K = 反轉可能性極高
- 高檔背離+爆量：股價創新高但KD/RSI未創新高 + 巨量長黑K = 頭部反轉強烈警訊

**成交量 + 均線**：
- 帶量突破均線 = 翻多確認
- 量縮回測均線 = 第二買點
- 帶量跌破均線 = 翻空確認

### 成交量選股實用技巧

1. **篩選溫和放量股**：今日成交量 ≥ 20日均量1.5倍 + 股價站上月線
2. **避開無量殭屍股**：成交量長期低迷 = 流動性差、人氣渙散
3. **關注突破平台股**：長期盤整（價平量縮）後突然帶量向上突破 = 後續力道最強

### 洗盤 vs 真正下跌的成交量判斷

| 情境 | 洗盤（主力洗籌） | 真正下跌（趨勢反轉） |
|------|-----------------|-------------------|
| 下跌時成交量 | 量縮（拋售壓力有限） | 量增（恐慌性賣壓） |
| 上漲時成交量 | 量增（有資金承接） | 量縮（無人追價） |
| 均線關係 | 長期均線仍多頭排列向上 | 長期均線轉為下彎空頭排列 |
| 跌破均線後 | 2-5日內快速站回 | 無法站回，均線持續下壓 |
| 成交量趨勢 | 遞減（浮額減少） | 維持或放大 |
| 洗盤結束訊號 | 帶量向上突破 | 無此訊號 |

## 注意事項

- **「大量」是相對概念**：沒有絕對標準，必須與個股平均成交量（VMA 5日/20日量均線）比較，超過1.5-2倍才算放量
- **成交量柱顏色只看當日開盤vs收盤**：與前一日收盤無關，開高走低即使大漲也是綠色量柱
- **成交量分析對中大型股最有效**：股本大、流動性好的股票成交量訊號更可靠
- **盤中預估量極有價值**：早盤10點前預估量已超過前日總量 = 市場交投熱絡，可能有大行情
- **高檔爆量不一定是好事**：低檔放量突破是好事，高檔天量天價是危險信號
- **量大不漲是壞事**：成交量增加但股價不漲 = 通常不是好現象，「量大而不漲，股價準回頭」
