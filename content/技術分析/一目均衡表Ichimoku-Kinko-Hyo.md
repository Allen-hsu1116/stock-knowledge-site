# 一目均衡表Ichimoku Kinko Hyo

> 一眼看穿多空——五條線+一片雲組成的完整趨勢判斷系統，雲帶是動態支撐壓力，三役好轉是最高勝率進場訊號，遲行線是唯一確認指標

## 核心概念

一目均衡表（Ichimoku Kinko Hyo）由日本記者細田悟一（筆名：一目山人）於1930年代發明，是全球技術分析的鼻祖級指標。「一目」即「一眼看穿」，「Kinko」為時空平衡點，「Hyo」為圖表——整個名字的意思是「一眼就能看出時空平衡的圖表」。

它是一套自成一體的技術分析系統，不需要搭配其他指標即可判斷趨勢方向、支撐壓力、進出場時機。分為「看得見的部分」（五條線+雲帶）和「看不見的部分」（時間論、波動論、值幅觀測論）。

### 五條線計算公式

| 線名稱 | 日文名 | 計算方式 | 期間 |
|--------|--------|----------|------|
| 轉換線 | Tenkan-sen | (9日內最高+9日內最低)/2 | 短期9日 |
| 基準線 | Kijun-sen | (26日內最高+26日內最低)/2 | 中期26日 |
| 遲行線 | Chikou Span | 今日收盤價後移26日 | 確認用 |
| 先行帶A | Senkou Span A | (轉換線+基準線)/2，前移26日 | 預測用 |
| 先行帶B | Senkou Span B | (52日內最高+52日內最低)/2，前移26日 | 預測用 |

**雲帶（Kumo）**= 先行帶A與先行帶B之間的區域

> ⚠️ **參數爭議**：原始參數9-26-52基於1930年代日本每週6天工作制。現代每週5天，有學者建議改為7-22-44。但實務上多數人仍用預設值，改參數不一定提高勝率。

### 與均線的關鍵差異

| 特徵 | 移動平均線 | 一目均衡表 |
|------|-----------|-----------|
| 計算基礎 | 收盤價平均 | 最高價與最低價的中值 |
| 方向性 | 只看過去 | 同時看過去（遲行線）、現在（轉換線/基準線）、未來（雲帶前移26日） |
| 支撐壓力 | 一條線一個點 | 雲帶是一個區域，有厚度概念 |
| 判斷方式 | 需要搭配其他指標 | 自成一體系統 |

## 實戰應用

### 一、趨勢判斷：價格與雲帶的位置關係

這是一目均衡表最基本也最實用的用法：

- **價格在雲帶上方** → 多頭優勢，找買點
- **價格在雲帶下方** → 空頭優勢，找賣點或觀望
- **價格在雲帶內部** → 方向不明，盤整整理，不進場

飛機比喻：飛機在雲層上方平穩飛行=多頭；飛入雲層顛簸=盤整；降到雲層下方=空頭

### 二、雲帶厚度判讀

- **雲帶厚** → 支撐/壓力強，突破不易
- **雲帶薄** → 支撐/壓力弱，容易突破
- **雲帶變薄** → 動能增強，當前趨勢可能延續
- **雲帶變厚** → 動能減弱，當前趨勢可能反轉

### 三、三役好轉（最強買進訊號）

三個條件同時滿足才是真正的三役好轉：

1. ✅ 轉換線在基準線上方（黃金交叉）
2. ✅ 遲行線在K線上方（確認多頭）
3. ✅ 價格在雲帶上方（趨勢確認）

**三役逆轉**（最強賣出訊號）：三個條件反轉
1. 轉換線在基準線下方
2. 遲行線在K線下方
3. 價格在雲帶下方

### 四、四道防線概念

從多翻空，價格需要依序跌破：
1. **轉換線** → 第一道防線（最弱）
2. **基準線** → 第二道防線（重要，跌破基準線是大調整信號）
3. **先行帶A** → 第三道防線
4. **先行帶B** → 最後防線（跌破=空頭確認）

反過來，從空翻多，價格需要依序站上這四道防線。

### 五、遲行線的確認角色

一目山人將遲行線稱為一目均衡表最重要的組成部分：

- **遲行線上穿K線** → 買入訊號
- **遲行線下穿K線** → 賣出訊號
- 遲行線只是輕微碰到K線又折回 → 走勢沒有轉換，且常常預示更強的走勢

**三點轉強**（底部反轉確認）：
1. 轉換線上穿基準線
2. 遲行線上穿K線
3. K線上穿雲帶

### 六、雲帶扭轉（趨勢反轉信號）

- 先行帶A跌破先行帶B → 雲帶由綠翻紅（多翻空信號）
- 先行帶A突破先行帶B → 雲帶由紅翻綠（空翻多信號）
- 雲帶扭轉處價格容易出現劇烈波動

### 七、調整期判斷

**多翻空的調整信號**：
- K線從雲帶上方進入雲層
- K線下穿基準線
- 轉換線在基準線之下停留
- 遲行線開始下穿K線

**空翻多的調整信號**：
- K線開始站在基準線上方
- 轉換線上穿基準線
- 遲行線上穿K線
- K線由下方進入雲帶，最終上穿雲帶

### 八、一目均衡表三大隱藏理論

**時間論**：9、17、26為基本數值，在這些時間點市場容易出現轉換。對照轉換線(9日)和基準線(26日)的參數，可見時間論已內建在指標中。

**波動論**：三種基本波動形態：
- I型波動：單純上漲/下跌
- V型波動：上漲→下跌 或 下跌→上漲
- N型波動：上漲→下跌→上漲（最基本形態）

**值幅觀測論**（水平論）：計算價格可能漲跌的幅度：
- N計算值：第3波幅度 = 第1波幅度
- E計算值：第3波幅度 = 第1波幅度的2倍
- V計算值：第3波幅度 = 第2波調整幅度的2倍
- NT計算值：第3波幅度 = 第1波幅度 - 第2波調整幅度

## 進階：一目均衡表三大隱藏理論詳解

### 時間論

9、17、26是一目均衡表的三個基本時間數值，在這些時間點市場容易出現轉換：
- 9日：短期轉換點（對應轉換線）
- 17日：中期轉換點（9+8）
- 26日：長期轉換點（對應基準線）

這不是隨意設定——轉換線用9日、基準線用26日、先行帶B用52日（26×2），時間論已經內建在指標參數中。

### 波動論

三種基本波動形態判斷市場結構：
- **I型波動**：單純上漲或單純下跌，趨勢最明確
- **V型波動**：上漲→下跌或下跌→上漲，反轉急劇
- **N型波動**：上漲→下跌→上漲（最基本型態），I型和V型最終都會演化成N型

> 實戰提示：N型波動是一目均衡表最核心的波動形態，對應艾略特波浪理論的5-3結構

### 值幅觀測論（水平論）

計算價格可能漲跌的目標幅度：
- **N計算值**：第3波幅度 ≈ 第1波幅度（等幅測量）
- **E計算值**：第3波幅度 ≈ 第1波幅度的2倍（倍幅測量）
- **V計算值**：第3波幅度 ≈ 第2波調整幅度的2倍（調整測量）
- **NT計算值**：第3波幅度 ≈ 第1波幅度 - 第2波調整幅度（差幅測量）

> 這四種計算方法可以預測目標價，在出場決策時特別有用

## 注意事項

### 雲帶內盤整的陷阱
- 價格在雲帶內部時方向不明確，容易出現假訊號
- 雲帶扭轉時也是方向不明的高風險區
- 短時間週期（5分鐘、15分鐘）假訊號更多
- **建議**：雲帶內不進場，等價格明確站上或跌破雲帶再操作

### 遲行線的誤解
- 遲行線不是預測線，是確認線
- 遲行線上穿K線確認多頭、下穿確認空頭——但訊號有延遲
- 遲行線只是輕觸K線就折回 ≠ 轉勢，反而可能預示更強的趨勢

### 轉換線的誤用
- 單獨跌破轉換線 ≠ 轉勢，可能只是短期調整
- 轉換線只是短期支撐/壓力，跌破轉換線先看基準線
- **只有在基準線也跌破時，才考慮趨勢轉變**

### 參數調整的爭議
- 原始參數9-26-52基於每週6天工作制
- 現代每週5天，理論上應改為7-22-44
- 但改參數不一定提高勝率，多數人仍用預設值
- **建議**：先用預設值熟悉指標，不要一開始就調參數

### 遲行線的进阶用法

- **强勁上升市**：遲行線與K線平行上升，距離穩定
- **遲行線與K線接近**：市勢轉弱警訊，進入調整期可能增大
- **遲行線不下穿基準線**是維持上升勢頭的必要條件
- **下降市中遲行線反彈至K線位置**：極佳賣點
- **遲行線輕觸K線就折回**：走勢未轉換，反而可能預示更強趨勢

### 雲帶交叉點

先行帶A與先行帶B交叉的地方稱為「交叉點」，價格經過此點時通常會出現較大波動。雲帶變薄的地方代表買賣力量趨於均衡，市場易出現劇烈波動。

### 與其他指標搭配
- 一目均衡表已經是完整系統，不需要過多其他指標
- 但可以搭配成交量確認突破真偽
- 也可以搭配RSI或KD做超買超賣過濾
- 搭配[[ADX趨勢強度過濾盤整]]過濾雲帶內盤整

### 多時間框架
- 一目均衡表建議按月→週→日順序分析
- 月圖看大方向、週圖看中期趨勢、日圖找進出場點
- 三個時間框架雲帶方向一致時勝率最高
