# 考夫曼自適應移動平均線 KAMA

> 盤整時像蝸牛不動、趨勢時像獵豹狂奔——根據市場效率自動調整靈敏度的「有潔癖」均線，專治假突破和盤整洗盤

## 核心概念

考夫曼自適應移動平均線（Kaufman's Adaptive Moving Average, KAMA），由量化大師 Perry Kaufman 在《Smarter Trading》中提出，是傳統均線的進化版。

傳統均線的痛點：短期均線太靈敏，盤整時被來回打臉；長期均線太遲鈍，趨勢都走一半了才跟上。KAMA 的解法是引入**效率係數（Efficiency Ratio, ER）**，讓均線自己判斷現在該快還是該慢。

### 效率係數 ER：KAMA 的大腦

ER = 方向位移 ÷ 總路徑

| 市場狀態 | 位移 | 路徑 | ER | KAMA 行為 |
|----------|------|------|-----|-----------|
| 趨勢盤（穩步上漲） | 大 | 小 | → 1 | 靈敏追蹤（像短期均線） |
| 震盪盤（來回刷洗） | 小 | 大 | → 0 | 停滯水平（像長期均線） |

ER 的核心邏輯：**價格走了多遠 vs 價格晃了多遠**。比值越高代表趨勢越有效率，比值越低代表只是噪音。

## 計算公式

### Step 1：效率係數 ER

```
Direction = |Close - Close[n]|          （淨位移：直線距離）
Volatility = Σ|Close - Close[1]|, n     （總路徑：每日波動加總）
ER = Direction / Volatility             （效率係數：0~1）
```

- 預設 n = 10（計算期間）

### Step 2：平滑常數 SC

```
FastSC = 2 / (FastEnd + 1)              （預設 FastEnd = 2）
SlowSC = 2 / (SlowEnd + 1)             （預設 SlowEnd = 30）
SC = [ER × (FastSC - SlowSC) + SlowSC]²
```

**平方運算是關鍵**：壓抑低 ER 時的靈敏度，讓盤整時 SC 趨近 SlowSC（慢），趨勢時 SC 趨近 FastSC（快）。

### Step 3：KAMA 遞迴計算

```
KAMA = KAMA[1] + SC × (Close - KAMA[1])
```

第一根 Bar 用收盤價初始化。

### 參數預設值

| 參數 | 預設值 | 說明 |
|------|--------|------|
| Period (n) | 10 | ER 計算期間 |
| FastEnd | 2 | 最快 EMA 週期 |
| SlowEnd | 30 | 最慢 EMA 週期 |

## 實戰應用

### 1. 過濾假突破——KAMA 最強大的能力

**假突破情境**：
- 股價瞬間大漲突破前波高點
- 傳統 5MA 立刻勾頭向上，發出買進訊號 → 買在最高點，隔天大跌
- KAMA 的反應：雖然位移（分子）變大，但路徑（分母）更大，ER 反而下降
- **KAMA 保持水平不動，拒絕跟隨噪音**

**真趨勢情境**：
- 股價穩步推升，每天漲一點點，回檔很淺
- 位移大、路徑短，ER 接近 1
- **KAMA 加速追趕，緊貼價格上揚**

### 2. 趨勢判斷

- KAMA 向上 → 多頭趨勢，考慮買入
- KAMA 向下 → 空頭趨勢，考慮賣出
- KAMA 走平 → 盤整，不追價

### 3. 交叉策略

- 短期均線向上穿越 KAMA → 買入訊號
- 短期均線向下穿越 KAMA → 賣出訊號
- KAMA 本身走平時的交叉無效

### 4. 動態支撐壓力

- 價格接近 KAMA 反彈 → 支撐
- 價格跌破 KAMA → 壓力
- 盤整時 KAMA 走平，支撐壓力功能增強

### 5. 搭配其他指標

- **KAMA + ADX**：ADX 確認趨勢存在，KAMA 確認進場時機
- **KAMA + 均線糾結**：KAMA 走平時等於均線糾結，等待方向突破
- **KAMA + 成交量**：KAMA 轉向配合帶量突破，勝率更高

## 注意事項

### 優勢

- **抗噪能力強**：盤整時自動走平，過濾假訊號
- **反應靈敏**：趨勢明確時迅速追蹤，不漏關鍵行情
- **自適應性**：一條線搞定所有市況，不需要切換參數

### 局限性

- **計算複雜**：ER → SC → KAMA 三層遞迴，手算不現實
- **參數敏感**：預設 10/2/30 適合日線，短線需調整（如 7/2/20）
- **滯後性**：終究是均線，對急速反轉反應較慢
- **不提供方向**：KAMA 只告訴你「該不該動」，不告訴你「往哪動」
- **盤整末期可能太慢**：從走平到轉向有延遲，可能錯過起漲第一根K線

### 常見誤區

| 誤區 | 正確理解 |
|------|----------|
| KAMA 不會產生假訊號 | KAMA 大幅減少假訊號，但不是零 |
| KAMA 可以單獨使用 | 仍需搭配方向性指標（+DI/-DI、均線排列） |
| 參數越靈敏越好 | 太靈敏會失去過濾雜訊的意義 |
| KAMA 走平就一定不交易 | 走平是過濾趨勢策略的訊號，區間策略仍可操作 |

## KAMA vs 其他均線比較

| 特性 | SMA | EMA | KAMA |
|------|-----|-----|------|
| 權重 | 等權 | 指數衰減 | 效率自適應 |
| 盤整表現 | 頻繁假訊號 | 較少但仍有 | 最佳（走平） |
| 趨勢反應 | 慢 | 中 | 快（ER高時） |
| 參數調整 | 固定週期 | 固定週期 | 自動適應 |
| 計算複雜度 | 低 | 中 | 高 |
