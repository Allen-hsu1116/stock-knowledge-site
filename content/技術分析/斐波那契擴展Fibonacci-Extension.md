# 斐波那契擴展 Fibonacci Extension

> 用61.8%、100%、161.8%三個比例預測趨勢目標價，是停利設定和諧波形態的核心工具

## 核心概念

斐波那契擴展（Fibonacci Extension）與回撤不同，它不是找回測位置，而是**預測趨勢的目標價**。在一段趨勢和一段回撤之後，用擴展比例計算價格可能到達的位置。

### 核心比例
- **61.8%**：保守目標
- **100%**：等幅目標（CD = AB）
- **127.2%**：√φ擴展（Butterfly型態D點）
- **161.8%**：φ擴展（Crab型態D點，強勢目標）
- **200%**：雙倍擴展
- **261.8%**：超強趨勢目標

### 實戰原則
1. 回撤結束後，用擴展計算趨勢延續的目標價
2. 161.8%是最常用的強勢目標
3. 搭配結構性支撐壓力確認目標有效性
4. 多個擴展比例重疊的區域是最強目標

## 實戰應用

### 斐波那契擴展的畫法

1. 先確定一段完整的趨勢（第1浪：起點到高點）
2. 再確定回撤的終點（第2浪低點）
3. 用擴展工具從第1浪起點拉到第1浪終點，再拉回第2浪低點
4. 畫面會自動顯示目標價位

### 常用目標價組合

| 比例 | 意義 | 實戰場景 |
|------|------|----------|
| 0.618 | 保守目標 | 弱勢趨勢中第一個停利點 |
| 1.000 | 等幅目標 | AB=CD 型態的D點，或N字攻擊的滿足點 |
| 1.272 | √φ擴展 | Butterfly 型態的D點目標 |
| 1.618 | φ擴展 | 最常用的強勢目標，波浪理論第3浪的典型目標 |
| 2.000 | 雙倍擴展 | 超強趨勢的第二目標 |
| 2.618 | φ²擴展 | 極端強勢的最終目標，波浪理論第3浪延伸目標 |

### 搭配回撤的完整操作 SOP

1. 在第2浪低點用回撤找到支撐（0.5-0.618區間），等待反轉訊號進場
2. 進場後，用擴展設定目標價：
   - 第一目標：1.000（等幅）
   - 第二目標：1.618（黃金比例）
   - 第三目標：2.618（超強趨勢）
3. 在各目標價附近觀察K線型態，決定是否分批停利

### 0.618 回撤 + 1.618 擴展的經典組合

這是斐波那契交易最經典的策略：
- 在 0.618 回撤位找到買點（盈虧比最佳）
- 設定 1.618 擴展為目標價（主力預設的滿足點）
- 停損設在跌破 0.618 下方一點點
- 這個組合的風險報酬比通常大於 1:2

## 注意事項

### 🚨 擴展不是精確預測

擴展目標是「價格可能到達的區域」，不是精確的轉折點。實戰中要觀察目標價附近的K線型態和成交量變化。

### 🚨 多個擴展重疊區域最有效

當不同波段的擴展目標重疊在同一區域時（confluence zone），這個區域的有效性大幅提升。

### 🚨 搭配波浪理論使用

斐波那契擴展與 [[艾略特波浪理論Elliott-Wave-Theory]] 天然搭配：第3浪用1.618倍數預測，第5浪用等長或0.618倍數預測。單獨使用擴展而不知道波浪結構，容易在錯誤的波段計算。
