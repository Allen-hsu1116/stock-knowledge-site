# 移動平均線種類比較 SMA EMA WMA

> SMA等權重平滑但落後、EMA指數加權反應快、WMA線性加權最靈敏，三種均線各有適用場景，了解差異才能選對工具。

## 核心概念

移動平均線（Moving Average, MA）是技術分析最基礎也最重要的指標，但不同計算方式產生的均線特性差異很大。了解 SMA、EMA、WMA 的計算邏輯和特性差異，是選擇正確均線工具的關鍵。

### 簡單移動平均線 SMA（Simple Moving Average）

**計算方式**：將N日收盤價加總後除以N

```
SMA = (P1 + P2 + ... + PN) / N
```

**特性**：
- 所有數據等權重，最公平但最遲鈍
- 對價格變動反應最慢，落後性最強
- 適合長期趨勢判斷，雜訊過濾能力最強
- 常用週期：20日、60日、120日、240日

**優點**：計算簡單、平滑度高、不易被單日極端值干擾
**缺點**：落後性強、對趨勢轉折反應慢

### 指數平滑移動平均線 EMA（Exponential Moving Average）

**計算方式**：以前一日EMA為基礎，加上平滑因子加權當日收盤價

```
EMAt = EMAt-1 + α × (Pt - EMAt-1)
α = 2 / (N + 1)  （平滑因子）
```

初次計算時，通常以SMA作為EMA的初始值。

**特性**：
- 所有歷史數據都納入計算，但權重呈指數衰減
- 近期數據權重最大，但舊數據不完全排除
- 反應速度介於SMA和WMA之間
- MACD的核心計算基礎（12EMA - 26EMA）

**優點**：反應比SMA快、舊數據仍保有微弱影響、最常被使用的均線類型
**缺點**：不同數據起點可能產生不同的EMA值

### 加權移動平均線 WMA（Weighted Moving Average）

**計算方式**：為每個週期賦予線性遞增的權重

```
WMA = (1×P1 + 2×P2 + ... + N×PN) / (1 + 2 + ... + N)
```

以5日WMA為例：
| 週期 | 權重 | 價格 | 加權值 |
|------|------|------|--------|
| 1 | 1 | 5 | 5 |
| 2 | 2 | 6 | 12 |
| 3 | 3 | 7 | 21 |
| 4 | 4 | 8 | 32 |
| 5 | 5 | 9 | 45 |
| **合計** | **15** | **35** | **115** |

WMA = 115 / 15 = 7.67

**特性**：
- 最近數據權重最大，線性遞減
- 對趨勢追隨性與感應度最高
- 超過計算週期的數據完全不被納入
- 在三種均線中反應最快

**優點**：對價格變動反應最靈敏、趨勢追隨性最強
**缺點**：雜訊最多、容易被單日極端值影響、計算較複雜

### 三種均線比較總覽

| 特性 | SMA | EMA | WMA |
|------|-----|-----|-----|
| 計算數據 | 最近N日 | 所有歷史數據 | 最近N日 |
| 權重分配 | 等權重 | 指數衰減 | 線性遞增 |
| 反應速度 | 最慢 | 中等 | 最快 |
| 平滑度 | 最高 | 中等 | 最低 |
| 雜訊過濾 | 最強 | 中等 | 最弱 |
| 落後性 | 最強 | 中等 | 最弱 |
| 適用場景 | 長期趨勢 | 通用/中期趨勢 | 短線/當沖 |

**追隨性與感應度排序**：WMA ＞ EMA ＞ SMA

## 實戰應用

### 1. 依交易週期選擇均線類型

| 交易風格 | 建議均線 | 理由 |
|---------|---------|------|
| 長線投資 | SMA 60/120/240 | 過濾雜訊，抓大趨勢 |
| 波段操作 | EMA 20/60 | 兼顧反應速度和穩定度 |
| 短線交易 | EMA 5/10 或 WMA 5/10 | 快速反應價格變動 |
| 當沖 | WMA 或 EMA 3/5 | 靈敏度優先 |

### 2. EMA 在 MACD 中的應用

MACD 的核心是 EMA 的差值：
- MACD 線 = 12EMA - 26EMA
- 訊號線 = MACD 的 9EMA
- 使用 EMA 而非 SMA，因為 EMA 對近期價格變動更敏感

參考 [[MACD指標實戰判讀]]

### 3. SMA vs EMA 交叉訊號比較

- SMA 交叉：訊號少但可靠度高，適合波段
- EMA 交叉：訊號多但假訊號也多，適合短線
- 實戰技巧：用 EMA 找進場點，用 SMA 確認趨勢

### 4. 多種均線搭配使用

常見搭配方式：
- **EMA 5 + SMA 20**：短線方向 + 中期趨勢
- **EMA 12 + EMA 26**：MACD 標準參數
- **SMA 20 + SMA 60 + SMA 120**：短中長期趨勢判斷

參考 [[均線判斷]]、[[均線多頭排列與空頭排列]]

### 5. WMA 在當沖的應用

WMA 反應最快，適合需要及時反應的場景：
- 當沖判斷短線方向
- 突破確認（WMA 先於 SMA 轉向）
- 搭配 [[VWAP成交量加權均價]] 做法人成本線判斷

## 注意事項

### 均線選擇的常見誤區

1. **不是越靈敏越好**：WMA 反應最快但也最容易產生假訊號，短線不一定用 WMA
2. **不同軟體的 EMA 值可能不同**：因為 EMA 納入所有歷史數據，起始點不同結果不同
3. **均線週期比類型重要**：20SMA 和 20EMA 的差異，遠小於 20EMA 和 60EMA 的差異
4. **不要頻繁更換均線類型**：固定使用一種，才能建立對該均線的盤感

### 實戰建議

- **新手**：從 SMA 開始，最直覺最好理解
- **進階**：切換到 EMA，對價格變動更敏感
- **當沖**：可嘗試 WMA 或超短週期 EMA
- **最重要的是一致性**：固定使用同一種均線，比不斷切換更重要

### 與其他指標的搭配

- 均線 + [[KD指標隨機指標]]：均線判方向，KD 找進出場點
- 均線 + [[布林通道Bollinger-Bands三軌八型態]]：均線為中軌，布林通道為波動範圍
- 均線 + [[ATR平均真實波幅-Average-True-Range]]：均線判趨勢，ATR 設停損
