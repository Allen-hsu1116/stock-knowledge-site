# 相對活力指標 RVI (Relative Vigor Index)

> 用收盤價與開盤價的關係衡量趨勢「活力」的震盪指標，多頭收高、空頭收低是基本假設，信號線交叉與背離是兩大操作核心

## 核心概念

**相對活力指標 (Relative Vigor Index, RVI)** 由 John Ehlers 開發，核心理論非常直覺：多頭趨勢中收盤價傾向高於開盤價，空頭趨勢中收盤價傾向低於開盤價。RVI 把這個「收盤 vs 開盤」的活力除以當日振幅（最高-最低），量化出趨勢的真實力道。

### 計算方式

**簡化版（IFC Markets）**：
- RVI(1) = (Close - Open) / (High - Low)
- RVI(10) = RVI(1) 的 10 期 SMA

**標準版（Fidelity）**：
- 分子：[a + 2b + 2c + d] / 6，其中 a/b/c/d 是近四根 K 線的 (Close - Open)
- 分母：[e + 2f + 2g + h] / 6，其中 e/f/g/h 是近四根 K 線的 (High - Low)
- RVI = 分子的 SMA / 分母的 SMA
- 信號線 = [RVI + 2i + 2j + k] / 6（i/j/k 為前三期 RVI 值）

兩種算法邏輯相同——標準版用加權平滑讓訊號更穩定，簡化版更直覺適合快速理解。

| 指標 | 比較 |

## 實戰應用

### 1. 信號線交叉——基本進出場

- **RVI 從下方穿越信號線** → 買進訊號（多頭活力增強）
- **RVI 從上方穿越信號線** → 賣出訊號（空頭活力增強）
- 在**上升趨勢**中只做買進交叉，在**下降趨勢**中只做賣出交叉——順勢操作

### 2. 背離判讀——反轉預警

- **多頭背離**：價格創新低，RVI 低點卻比前一次高 → 下跌力道減弱，反轉上漲
- **空頭背離**：價格創新高，RVI 高點卻比前一次低 → 上漲力道減弱，反轉下跌
- 背離是 RVI **最可靠**的訊號，比交叉更有參考價值

### 3. 零軸判斷多空

- RVI > 0 → 多方活力佔優
- RVI < 0 → 空方活力佔優
- RVI 在零軸附近徘徊 → 盤整，不適合用交叉訊號

### 4. 搭配策略

- **RVI + 均線**：均線判趨勢方向，RVI 信號線交叉找進場點
- **RVI + [[技術分析/背離Divergence進階實戰|背離確認]]**：RVI 背離 + 價格型態確認，提高勝率
- **RVI + [[技術分析/ADX趨勢強度過濾盤整|ADX]]**：ADX > 25 確認有趨勢，再用 RVI 找進出場，避免盤整假訊號

## 注意事項

- **盤整市是最大剋星**：RVI 在零軸附近來回震盪，信號線交叉頻繁產生假訊號，必須搭配趨勢過濾器（ADX、均線方向）
- **與 Stochastic RSI 類似但不同**：StochRSI 是對 RSI 做隨機化處理，RVI 是用收盤/開盤比值，不要混淆
- **信號線是 4 期 SMA**：非常短週期，反應快但雜訊多，可自行調長至 10 期降低假訊號
- **不提供超買超賣區間**：RVI 沒有像 KD 的 80/20 固定區間，判斷極端值需靠歷史經驗
- **多時間框架驗證**：日線 RVI 背離 + 週線趨勢方向一致，勝率更高
