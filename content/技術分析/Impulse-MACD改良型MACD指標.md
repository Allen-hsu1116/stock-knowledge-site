# Impulse-MACD改良型MACD指標

> LazyBear 開發的改良型 MACD，用零延遲 DEMA 替代 EMA，搭配 SMMA 高低價通道判斷趨勢衝量，過濾盤整雜訊

## 核心概念

Impulse MACD 是 TradingView 上知名指標開發者 LazyBear 的代表作。它並非傳統 MACD 的簡單變體，而是從底層計算邏輯就做了根本性的改良：

### 與傳統 MACD 的差異

| 項目 | 傳統 MACD | Impulse MACD |
|------|----------|--------------|
| 快線計算 | EMA 差值 | DEMA 與 SMMA 通道的偏離量 |
| 基準線 | EMA（慢速） | SMMA 的高低價通道 |
| 延遲性 | 中等（EMA 本身有延遲） | 低（DEMA 零延遲 + SMMA 平滑） |
| 盤整過濾 | 弱（盤整時假信號多） | 強（動量須與趨勢方向一致才出信號） |
| 適用市場 | 股票為主 | 期貨、趨勢明確市場效果佳 |

### 核心思想

只有當**價格動能與趨勢方向一致**時，產生的交易信號才更具價值。當價格在 SMMA 高低價通道之間遊走時，動量指標（md）為零，自動過濾掉盤整行情的雜訊。

## 實戰應用

### 指標計算步驟

1. **HLC/3** = (最高價 + 最低價 + 收盤價) / 3
2. **DEMA（零延遲雙指數均線）**= DEMA(HLC/3, ma_len)
3. **SMMA 通道** = SMMA(最高價, ma_len) 和 SMMA(最低價, ma_len)
4. **快線 md**（衝量）：
   - DEMA > SMMA(高) → md = DEMA - SMMA(高)（正衝量，價格突破通道上緣）
   - DEMA < SMMA(低) → md = DEMA - SMMA(低)（負衝量，價格跌破通道下緣）
   - 通道內 → md = 0（盤整區，無信號）
5. **慢線 sb** = SMA(md, sig_len)

### 參數建議（台指期實證）

- ma_len = 30（SMMA 通道和 DEMA 週期）
- sig_len = 8（信號線週期）
- atr_len = 20（ATR 停損週期）
- atr_multiplier = 3.25（ATR 停損倍數）

### 進出場信號

- **買進**：快線 md 由下往上穿越慢線 sb（黃金交叉）
- **賣出**：快線 md 由上往下穿越慢線 sb（死亡交叉）
- **盤整**：md = 0 時不出信號，這是 Impulse MACD 最核心的過濾機制

### ATR 移動停損

搭配 ATR 移動停損出場，而非反向信號出場：
- **多頭停損**：當前價 - 3.25 × ATR(20)，停損只往上調（鎖定利潤）
- **空頭停損**：當前價 + 3.25 × ATR(20)，停損只往下調
- 透過 StopOrder 掛單執行，確保停損即時生效

### 期貨轉倉處理

- 到期前 5 天自動偵測並轉倉
- 取消舊合約所有掛單
- 平舊倉、開新倉（同方向同口數）
- 為新倉位重新掛 ATR 停損單

## 注意事項

### 優勢
- 零延遲 DEMA 比傳統 EMA 更快反應價格變化
- SMMA 通道的「區間內歸零」機制有效過濾盤整假信號
- 適合趨勢明顯的期貨市場

### 限制
- 趨勢不明確時 md 容易在零附近震盪，產生反覆信號
- SMMA 的平滑效果可能延遲轉折信號
- 只用 1 口部位進出，在台指期可能獲利有限
- 單一指標策略，建議搭配其他確認工具

### SMMA vs SMA 的差異

SMMA（Smoothed Moving Average）的計算讓它比一般 SMA 更平滑，對突發波動的反應較慢但也較穩定。這在高波動的期貨市場中是優點（減少假信號），但也可能在趨勢初期反應較慢。
