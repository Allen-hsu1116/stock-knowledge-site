# Market Profile 市場輪廓 TPO

> 用時間密度描繪市場拍賣過程——TPO告訴你市場在哪個價位「花時間討論」，價值區告訴你多數人認同的範圍，IB破位告訴你趨勢是否啟動

## 核心概念

Market Profile 不是另一個技術指標，而是一套用**時間**去描繪價格拍賣過程的框架。它回答三個核心問題：

1. **市場現在在誰的控制下**（平衡還是失衡）
2. **價值正在往哪裡移動**（價值區位移方向）
3. **哪裡是風險最小而回報合理的介入點**（價值區邊緣與IB破位）

### TPO（Time Price Opportunity）

TPO是Market Profile的基本單位。當一個價格在一個時間單元被觸及，就留下一個TPO。TPO常以字母標記不同時間片（A、B、C...），逐步堆疊成分布。分布的形狀就是市場結構——**你看見的不是單根K線的戲劇性，而是整日拍賣的輪廓**。

### 關鍵構件

| 構件 | 定義 | 實戰意義 |
|------|------|---------|
| **POC（Point of Control）** | TPO最密集的價位 | 市場認同度最高的價格，隔日的磁石與分水嶺 |
| **價值區（Value Area）** | 包含全日約70% TPO的區間 | 市場一致認可的範圍 |
| **VAH** | 價值區上緣 | 多空分界線上方 |
| **VAL** | 價值區下緣 | 多空分界線下方 |
| **單印跡（Single Print）** | 某段價格僅出現一行TPO | 當下快速拒絕，強勢方向訊號 |
| **尾巴（Tail）** | 分布最外側的單印跡群 | 強勢的一次性掃單，保留與否判斷延續或反轉 |
| **IB（Initial Balance）** | 前兩個時間片的高低 | 開盤區間，IB破位代表開始尋找新價值 |

### 與 Volume Profile 的差異

| | Market Profile（TPO） | Volume Profile |
|--|----------------------|---------------|
| 衡量維度 | 時間密度 | 成交量 |
| 回答的問題 | 市場在哪花時間討論 | 市場在哪花力氣交易 |
| 優勢 | 反映市場建立共識的位置 | 反映真實資金投入的位置 |
| 弱點 | 瞬間掃單可能TPO稀少但意義重大 | 大量不一定是共識，可能是對沖 |

**互補原則**：價位同時擁有高TPO密度與高成交量 → 支撐阻力最具體。量能高但TPO稀少 → 主導性掃單，容易留下脆弱區。

## 實戰應用

### 1. 拍賣市場理論——平衡與失衡

市場在平衡與失衡之間來回：
- **平衡**：分布呈鐘形，價格圍繞POC擺動 → 用均值回歸策略
- **失衡**：分布傾斜，單印跡增多，市場主動離開舊價值區 → 用趨勢跟隨策略

核心心法：**先判斷環境是平衡還是失衡，再決定用哪種策略**。

### 2. 日內型態判讀

#### 趨勢日
- 開盤後迅速脫離IB，很少回測
- 分布呈現單邊延伸，尾巴完整保留
- POC持續單向位移
- **策略**：順勢分批加倉，止損放在尾巴之外

#### 正常日
- 多在前一日價值區內運動
- 分布接近對稱，POC位於中部
- **策略**：價值區邊緣的均值回歸

#### 正常變異日
- 某一側出現延伸但仍回到價值區內收
- **策略**：隔日易沿延伸方向測試未完成的拍賣

#### 中性日 / 雙向中性日
- 雙向測試IB並回到中值，買賣雙方拉鋸
- 雙向中性日兩端都留長尾
- **策略**：次日觀察哪一側的尾巴先被侵蝕，被侵蝕側代表主導力量回流

### 3. 開盤定位三場景

| 開盤位置 | 判斷 | 對應策略 |
|---------|------|---------|
| 開在價值區內 | 均值回歸機率高 | 價值區邊緣做回歸 |
| 跳空至價值區外且快速接受 | 價值轉移 | 順勢交易 |
| 跳空至價值區外但被拒絕 | 假突破 | 回補缺口，反向應對 |

### 4. IB破位質量判斷

**有效破位**（延伸具信度）：
- 成交量擴張
- 單印跡連續
- 回測不入IB
- POC持續上移/下移

**無效破位**（假突破）：
- 破位後POC不上移/下移
- 多次回到IB邊緣
- **應對**：減倉或放棄追價

### 5. 多週期結構整合

- 當日上攻挑戰週度VAH → 延伸空間有限，快進快出
- 上級別價值區破舊立新 → 單日回撤都是順勢機會
- 雙峰結構：中間低TPO區是薄弱帶，常被測試；站上薄弱帶並被接受 → 目標轉向對側峰值

### 6. 策略框架範例

**趨勢策略**：
1. 開盤在價值區外並接受
2. 等待IB上緣被突破後的首個回測進場
3. 止損放在回測低點之下
4. 第一目標：當日等幅延伸
5. 第二目標：上級別薄弱帶

**均值回歸策略**：
1. 開盤在價值區內，分布逐步對稱
2. 於VAH附近觀察拒絕訊號與單印跡形成
3. 進場做回歸，止損放在尾巴外
4. 目標先看POC，再看VAL

## 注意事項

- **不要把Market Profile當形狀辨識**：分布是否漂亮不重要，重點是接受與拒絕的過程、價值是否轉移、薄弱帶是否被快速回補
- **樣本不足就別武斷歸因**：單日型態需要與上級別結構一起評估，若矛盾則降低倉位
- **沒有成交的區域不是永遠的阻力**：只是暫時未被討論的價位
- **失真撤退訊號**：尾巴被吞噬、POC快速回撤 → 立即撤退
- **需要學習曲線**：TPO判讀需要時間累積經驗，初學者容易過度解讀單一型態
- **數據取得門檻**：TradingView有TPO指標，專業級需用Sierra Chart、CQG等平台
