# ICT交易法實戰三步驟

> 從大方向到小細節——確認趨勢→找聰明錢有興趣的價位→小時間框架精確進場，三步完成一筆高勝率交易

## 核心概念

ICT（Inner Circle Trader）交易法是 SMC 聰明錢概念的實戰執行框架。核心思路：**不要預測市場，而是觀察聰明錢已經在做什麼，然後跟著做。**

三步驟環環相扣：
1. **確認大方向** — 順勢不逆勢
2. **找聰明錢有興趣的價位** — 訂單塊、FVG、流動性池
3. **小時間框架精確進場** — CHoCH 確認 + 訂單塊回測

## 步驟一：確認大方向（HTF 趨勢判斷）

在高時間框架（1H、4H 或日線）確定市場結構：

- **多頭趨勢**：HH + HL 持續出現 + BOS 確認 → 找做多機會
- **空頭趨勢**：LH + LL 持續出現 + BOS 確認 → 找做空機會
- **盤整區間**：高低點交替，無明確趨勢 → 用溢價/折價區判斷操作方向

**關鍵原則**：不要和趨勢對做。如果目前是空頭趨勢，趨勢繼續的可能性比較高，找做空機會。

### 盤整區間的處理

- **溢價區 (Premium Zone)**：區間上半部 → 不做多
- **折價區 (Discount Zone)**：區間下半部 → 不做空
- 就像買菜：太貴不買，便宜才出手
- 操作方法：先判斷目前價格在折價區還是溢價區，決定做多或做空方向

## 步驟二：找聰明錢有興趣的價位（關鍵區域識別）

確認大方向後，觀察這波趨勢段中的關鍵區域：

### 訂單塊 (Order Block)
- 聰明錢的成本區
- 大幅單邊運動前最後一根反向K線
- 如果聰明錢要繼續操作，很可能回測這個區域再進場

### FVG（公允價值缺口）
- 價格快速移動留下的未測試區域
- 聰明錢可能有未成交訂單在這裡
- 回測 FVG = 再次進場的機會

### 流動性池 (Liquidity Pools)
- 被測試多次的高低點
- 聰明錢可能去獵取的區域
- 獵取後的結構轉變 = 最佳進場訊號

**組合判斷**：訂單塊 + FVG + 流動性池 三者重疊的區域，是最高機率的進場區。

## 步驟三：小時間框架精確進場（LTF 進場確認）

價格回到關鍵區域後，切換到小時間框架（3M、5M 或 15M）：

1. **等待流動性獵取**：價格突破流動性池 → 觸發止損
2. **確認 CHoCH/MSS**：小時間框架出現結構轉變，方向與大時間框架一致
3. **找訂單塊/破壞塊**：在 CHoCH 波段中找訂單塊或破壞塊
4. **等待回測進場**：價格回到訂單塊區域 + 出現反應（引線、拒絕型K線）
5. **設定止損止盈**：
   - 止損：設在訂單塊外側
   - 止盈：下一個流動性區域（波段高低點）

### 實戰案例

**情境**：1H 空頭趨勢

1. HTF 確認空頭趨勢（LH + LL + BOS）
2. 觀察下跌段，發現明顯的看跌訂單塊 + 巨大 FVG
3. 價格反彈至訂單塊區域 → 切換到 3M
4. 3M 圖出現：流動性池被獵取 → CHoCH 確認 → 看跌訂單塊回測
5. 在訂單塊區域進場做空
6. 止損設在訂單塊上方，止盈設在前低（流動性池）
7. 結果：2.7R 賺賠比

## 三步驟的常見錯誤

1. **跳過步驟一**：不看大方向就直接在小時間框架找訊號 → 逆勢操作被洗
2. **步驟二找太多區域**：每個看起來像訂單塊的地方都當作進場點 → 品質太差
3. **步驟三不等確認**：看到價格到區域就進場，不等 CHoCH → 太早進場被掃
4. **忽略流動性獵取**：CHoCH 前沒有 Liquidity Raid → 結構轉變不可靠
5. **止損設太近**：設在訂單塊正邊緣 → 被正常波動掃掉

## 與其他交易法的比較

| 特徵 | ICT 三步驟 | 傳統技術分析 | 純指標交易 |
|------|-----------|------------|-----------|
| 基礎 | 市場結構 + 流動性 | 型態 + 指標 | 數學公式 |
| 方向判斷 | HTF 市場結構 | 趨勢線/均線 | 指標方向 |
| 進場時機 | CHoCH + 訂單塊回測 | 型態完成 | 指標交叉 |
| 優勢 | 邏輯清晰，理解市場本質 | 直觀易懂 | 客觀量化 |
| 缺點 | 主觀性高，學習曲線陡 | 落後指標 | 忽略市場結構 |

## 注意事項

- **三步驟順序不可顛倒**：HTF → 關鍵區域 → LTF，這個順序是鐵律
- **步驟之間的銜接**：不是機械執行，需要理解每一步的「為什麼」
- **耐心是核心**：等不到好的關鍵區域就不進場，寧可錯過不要硬做
- **多時間框架一致性**：HTF 方向 + LTF 確認方向必須一致
- **風險管理永遠第一**：再好的三步驟分析，停損設不好一樣虧
- **需要大量看盤練習**：SMC 的主觀性意味著需要足夠的實戰經驗培養判斷力
- **溢價區不做多、折價區不做空**：這是盤整區間的鐵律
