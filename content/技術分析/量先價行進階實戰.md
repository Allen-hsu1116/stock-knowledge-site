# 量先價行進階實戰

> 成交量領先股價反應，遞增量是起漲訊號、換手大量是攻擊發令槍，週線量先價行是噴出行情的前奏

## 核心概念

量先價行是技術分析的核心原則：**成交量變化會領先股價變化**。股價要上漲或下跌前，成交量通常會先出現異常。

### 三大量先價行型態

#### 型態一：標準量先價行（量縮後大漲）

特徵：
1. 底部區間整理，出現明顯的**遞增量**
2. 均線翻揚
3. 出現關鍵的**換手大量**
4. 量縮整理後強勢大漲

背後邏輯：特定買盤在底部區間開始建立持股，吸納浮動籌碼（俗稱「吃貨」），遞增量就是主力進場的痕跡。

#### 型態二：先吃貨再低檔整理（下降楔形）

特徵：
1. 先出現量先價行訊號
2. 隨後進入量縮下跌走勢（下降楔形）
3. 下降楔形的特點：時間長、跌幅小
4. 反覆震盪打底後形成新底部
5. 遞增量出現、均線翻揚，啟動另一次量先價行

背後邏輯：長線主力為了特殊目的進行洗盤，徹底洗清浮動籌碼。

#### 型態三：週線量先價行（噴出行情）

特徵：
1. 週線量縮震盪打底，均線糾結
2. 出現連續遞增量的攻擊型態
3. 壓回進行三角收斂整理
4. 週線放量實體紅K確認整理結束
5. 週成交量領先突破前波大量區
6. 股價突破前波高點，伴隨量能擴增，強勢噴出

關鍵判斷：週成交量領先突破前波大量區，代表前波高點必將被突破。

## 實戰應用

### 遞增量判斷法則

1. **連續遞增量**：起漲前3-5日出現量能逐步遞增，是主力吃貨的明確訊號
2. **相對大量**：單日出現3-5倍均量的相對大量，是換手或攻擊的發令槍
3. **量縮不破低**：遞增量後的量縮整理，股價不跌破前低，是健康的換手

### 量先價行進場SOP

1. **觀察底部遞增量**：連續3-5日量能遞增，均線開始翻揚
2. **等待換手大量**：出現關鍵大量（通常伴隨長紅K或長下影線）
3. **確認量縮不破低**：大量後的量縮整理，股價不破前低
4. **進場時機**：量縮整理後再度放量，拉出紅K棒時進場
5. **停損設定**：設在量先價行最低點下方

### 下降楔形量先價行

這種型態的關鍵在於**耐心**：
- 量先價行訊號出現後，可能還要等3-6個月的下降楔形整理
- 下降楔形的特點：時間長、跌幅小
- 真正的起漲點是在第二次量先價行確認時

### 週線量先價行的威力

週線的量先價行比日線更可靠、更具爆發力：
- 18個月的三重底部格局 + 週線量先價行 = 強勢噴出
- 週成交量領先突破前波大量區 = 前波高點必將突破
- 突破後伴隨量能放大 + 均線多頭排列 = 一路揚長而去

## 注意事項

1. **量先價行不是萬能**：量先價行只提供方向判斷，進場點仍需型態確認
2. **假量先價行**：如果遞增量後出現帶上影線的長黑K，可能是主力出貨而非吃貨
3. **量能衰退的警訊**：遞增量後如果量能快速萎縮且股價跌破前低，代表吃貨失敗
4. **下降楔形的陷阱**：不是所有量縮下跌都是下降楔形，要確認跌幅不大且時間夠長
5. **週線優先於日線**：週線的量先價行訊號更可靠，日線的訊號可以是被週線趨勢修正的
6. **搭配均線判斷**：量先價行+均線翻揚=雙重確認，勝率更高
