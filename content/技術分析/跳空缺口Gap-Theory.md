# 跳空缺口 Gap Theory

> K線圖中沒有交易的價格真空帶，四種缺口判斷趨勢起始、延續與反轉

## 核心概念

跳空缺口是指兩根相鄰K線之間出現的價格真空區間——這個區間內完全沒有交易發生。通常因市場出現重大利多或利空，投資人在開盤前就大量掛單，導致價格直接跳過某段區間開高或開低。

**本質**：跳空缺口是市場主力的強勢表態。向上跳空代表多頭強勢，向下跳空代表空頭強勢。

### 四種缺口類型

| 缺口類型 | 發生位置 | 特徵 | 是否回補 |
|---------|---------|------|---------|
| **普通缺口** | 盤整區間內 | 價格真空區小，無方向性 | 1-2天內快速回補 |
| **突破缺口** | 突破盤整區時 | 伴隨成交量放大，確立趨勢方向 | 短期不回補=有效突破 |
| **逃逸缺口** | 趨勢進行中 | 趨勢加速段再次跳空 | 不容易被回補 |
| **竭盡缺口** | 趨勢末升/末跌段 | 成交量暴增但缺口快速回補 | 2-3天內回補=反轉訊號 |

### 缺口序列

一個完整的多頭行情通常依序出現：**突破缺口 → 逃逸缺口（可能複數）→ 竭盡缺口**

這就是經典的「三缺口理論」：
- **第一缺口買**（突破缺口 = 起漲點）
- **第二缺口抱**（逃逸缺口 = 趨勢延續）
- **第三缺口賣**（竭盡缺口 = 反轉前兆）

## 實戰應用

### 1. 突破缺口進場法
- **確認條件**：股價跳空突破盤整區 + 成交量明顯放大 + 2-3天內缺口未回補
- **進場時機**：缺口確認後，拉回缺口上緣附近買進
- **停損位置**：缺口被回補時出場
- **關鍵**：成交量越大，缺口被回補的機率越低

### 2. 缺口作為支撐壓力
- **向上跳空缺口**：未回補前 = 支撐區，回補後支撐失效
- **向下跳空缺口**：未回補前 = 壓力區，回補後壓力解除
- 向下跳空的力道比長黑K更強，因為代表整個價格區間都沒有人願意接手

### 3. 假突破辨識
- 向上跳空 2-3 天內被回補 = **假突破**，偏空訊號
- 向下跳空 2-3 天內被回補 = **假跌破**，偏多訊號
- 搭配 RSI、KD、MACD 死亡交叉確認假突破

### 4. 竭盡缺口反轉交易
- **辨識**：趨勢末端 + 成交量暴增 + 缺口快速（2-3天）回補
- **操作**：竭盡缺口回補後反向進場
- **注意**：逃逸缺口和竭盡缺口外觀相似，區別在於——竭盡缺口會「快速回補」

### 5. 島狀反轉
- **型態**：向上跳空後高檔無法續創新高 + 成交量萎縮 → 再向下跳空
- **圖形**：左右兩邊都有缺口，中間像海上島嶼
- **意義**：罕見但強烈的反轉訊號，通常出現在大跌初期
- **操作**：看到島狀反轉可積極做空

### 6. 突破缺口確認兩要點
1. 缺口越大越好，且能有效突破壓力線
2. 需有持續放量，表明籌碼換手積極

## 注意事項

### 除權息與減資缺口 — 無趨勢意義
- **除權息缺口**：因股本改變產生，非交易行為造成，無法預測趨勢
  - 觀察缺口時應使用**還原股價**圖表排除除權息影響
  - 填權息（回補除權息缺口）= 基本面強勁的訊號，越早填權息越好
- **減資缺口**：因流通股數減少產生，同樣無法單由技術面判斷
  - 需看減資目的（現金減資 vs 減資彌補虧損），意義完全不同

### 逃逸缺口 vs 竭盡缺口 — 最容易搞混
- **逃逸缺口**：趨勢延續中，缺口不回補，趨勢繼續
- **竭盡缺口**：趨勢末端，缺口快速回補，趨勢反轉
- **實戰判斷**：等 2-3 天！缺口被回補就是竭盡，沒回補就是逃逸

### 高檔突破缺口的陷阱
- 已經漲了一大段後出現的突破缺口，要留意是否為最後一波
- 需搭配位階判斷：低檔突破缺口勝率遠高於高檔

### 缺口回補的多重技術確認
- 向上跳空回補 + RSI/KD/MACD 死亡交叉 = 看空
- 向下跳空回補 + RSI/KD/MACD 黃金交叉 = 看多
- 不要單憑缺口回補就做決定，要搭配其他指標

### 普通缺口的處理
- 盤整中的普通缺口無參考價值，不要過度解讀
- 1-2天就會回補，不用特別操作
