# CDP三關價與Pivot-Point樞紐點

> 用昨日價格計算今日支撐壓力，當沖極短線的日內地圖。

## 核心概念

CDP（Cumulative Depth Index）與 Pivot Point 都是由技術分析大師 **Wells Wilder** 所開發的極短線操作工具，利用前一交易日的 OHLC（開高低收）計算出當日的支撐與壓力關卡，將當天價格震盪範圍劃分為多個等級。

兩者差異：
- **CDP**：五等級劃分（AH/NH/CDP/NL/AL），重視收盤價（權重×2）
- **Pivot Point**：七條線（R3/R2/R1/P/S1/S2/S3），標準版只取 HLC 三值，另有多種變形（Fibonacci、Camarilla 等）
- **三關價**：台灣本土化版本，利用斐波那契 1.382 比例計算上下關

### CDP 計算公式

```
CDP = (H + L + C × 2) / 4
AH（最高值）= CDP + (H - L)
NH（近高值）= CDP × 2 - L
NL（近低值）= CDP × 2 - H
AL（最低值）= CDP - (H - L)

排序：AH > NH > CDP > NL > AL
```

H = 前一日最高價，L = 前一日最低價，C = 前一日收盤價

### Pivot Point 計算公式（標準版）

```
P  = (H + L + C) / 3
R1 = P × 2 - L
R2 = P + (H - L)
R3（HBOP）= R1 + (H - L)
S1 = P × 2 - H
S2 = P - (H - L)
S3（LBOP）= S1 - (H - L)

排序：HBOP > R2 > R1 > P > S1 > S2 > LBOP
```

### 三關價計算公式

```
上關價 = L + (H - L) × 1.382
中關價 = (H + L) / 2
下關價 = H - (H - L) × 1.382
```

### Pivot Point 變形版本

| 版本 | 樞紐點 P 的計算 | 特色 |
|------|-----------------|------|
| **Standard** | (H+L+C)/3 | 最常用 |
| **Fibonacci** | 同 Standard，R/S 用 Fib 比例 | 0.382/0.618 比例 |
| **Camarilla** | (H+L+C)/3，R/S 用 1.0833~1.1680 乘數 | 8 個關卡更密集 |
| **Woodie** | (H+L+O×2)/4 | 重視開盤價 |
| **DeMark** | 依 C 與 O 關係有不同公式 | Tom DeMark 設計 |

## 實戰應用

### CDP 當沖操作策略

| 策略 | 進場條件 | 邏輯 |
|------|---------|------|
| **順勢做多** | 突破 AH 或 NH | 價格突破上檔，動能強 |
| **順勢做空** | 跌破 NL 或 AL | 價格跌破下檔，動能弱 |
| **逆勢做多** | 價格從下穿越 AL | AL 是極度超賣區 |
| **逆勢做空** | 價格從上穿越 AH | AH 是極度超買區 |

### Pivot Point 雙模式操作

**逆勢模式（Reaction）**：
- 價格到 S1 → 逢低買進，停利目標 R1
- 價格到 R1 → 逢高賣出，停利目標 S1
- 突破後可在 S2/R2 採相同策略

**順勢模式（Trend）**：
- 價格到達 HBOP（R3）→ 買進，趨勢啟動
- 價格到達 LBOP（S3）→ 賣出，趨勢啟動
- BOP = Break Out Point，突破點

### 三關價操作策略

| 價格位置 | 操作方向 | 邏輯 |
|----------|---------|------|
| 中關價～上關價 | 做多 | 偏多格局 |
| 中關價～下關價 | 做空 | 偏空格局 |
| 觸及上關價 | 留意壓力 | 可能反轉向下 |
| 觸及下關價 | 留意支撐 | 可能反轉向上 |

### Pivot 時間框架選擇

| 週期 | 適用 | 趨勢類型 |
|------|------|---------|
| 每日 Pivot | 日線 | 短期趨勢（當沖、極短線） |
| 每週 Pivot | 週線 | 中長期趨勢 |
| 每月 Pivot | 月線 | 中長期趨勢 |

### 實戰搭配組合

1. **CDP + 開盤價**：開盤位置決定當日格局（開在 CDP 之上偏多，之下偏空）
2. **Pivot Point + 成交量**：突破關卡需帶量確認
3. **三關價 + 均線**：均線方向過濾，順大逆小
4. **CDP + KD/RSI**：到達關卡時用指標確認超買超賣

## 注意事項

- **僅限當日有效**：每日 Pivot 隔天歸零重算，不可沿用
- **開盤價是關鍵**：開盤位置決定 CDP 有效性，錯過開盤訊號後指標作用降低
- **假訊號風險**：市場偶有假突破，不建議單獨使用
- **主觀性低**：Pivot Point 根據價格算出，所有人看到同一條線 → 容易成為自我實現的轉換點
- **大行情失效**：強趨勢盤中支撐壓力容易被穿越，逆勢操作風險大
- **三關價 1.382 來源**：斐波那契延伸比例，適用於一般波動範圍
