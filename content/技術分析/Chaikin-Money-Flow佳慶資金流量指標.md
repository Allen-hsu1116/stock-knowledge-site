# Chaikin Money Flow 佳慶資金流量指標

> 用收盤價在日內位置×成交量衡量資金流向——CMF > 0 買方主導、CMF < 0 賣方主導，背離是轉勢最強訊號，±0.05 緩衝區過濾洗盤

## 核心概念

Chaikin Money Flow (CMF) 由 Marc Chaikin 發明，基於一個核心假設：**強勢市場的收盤價通常在日內高點附近且伴隨大量，弱勢市場的收盤價通常在日內低點附近且伴隨大量**。它是 Accumulation/Distribution Line 的改良版，原理與 MFI 類似但計算方式不同。

CMF 本質上就是「價漲量增、價跌量縮」的量化表達——把收盤價在日內的位置（CLV）乘上成交量，然後除以總成交量，得到一個-1到+1之間的數值。

### 計算公式

**步驟 1：CLV（Close Location Value）**

CLV = ((收盤價 - 最低價) - (最高價 - 收盤價)) / (最高價 - 最低價)
簡化：CLV = (2 × 收盤價 - 最高價 - 最低價) / (最高價 - 最低價)

| 收盤價位置 | CLV 值 | 意義 |
|-----------|--------|------|
| 等於最高價 | +1 | 多方完全主導 |
| 在最高最低中間 | 0 | 多空平衡 |
| 等於最低價 | -1 | 空方完全主導 |

**步驟 2：CMF**

CMF = Σ(Vol × CLV) / Σ(Vol)

預設回溯期 20 日。CMF 值介於 -1 到 +1，實際上很少超過 ±0.3。

## 實戰應用

### 零軸分水嶺

- **CMF > 0**：買方主導，市場處於收集（Accumulation）階段
- **CMF < 0**：賣方主導，市場處於派發（Distribution）階段
- 停留時間越長，趨勢越可靠

### ±0.05 緩衝區（過濾洗盤）

零軸交叉最容易出現假訊號（洗盤）。實戰建議：
- CMF > +0.05 才視為偏多訊號
- CMF < -0.05 才視為偏空訊號
- 在 ±0.05 之間的波動忽略不計

### 超買超賣

- CMF 達 +0.3：超買區，股價可能需要回調
- CMF 達 -0.3：超賣區，股價可能反彈

### 背離訊號（最強訊號）

**看空背離**：價格創新高，CMF 卻未創新高或下降 → 買方力道減弱，趨勢可能反轉
**看多背離**：價格創新低，CMF 卻未創新低或上升 → 賣方力道減弱，趨勢可能反轉

背離是 CMF 最核心的應用，訊號比單純零軸交叉可靠得多。

### 長期收集/派發判讀

- **牛市調整**：股價下跌但 CMF 在零軸上方止跌回升 → 正常調整，可加碼
- **熊市假反彈**：CMF 每次升至零軸就回落 → 購買力弱，只是曇花一現的反彈

### CMF 搭配趨勢線

CMF 可以結合趨勢線及支撐壓力線突破進行分析，在突破關鍵位置時用 CMF 確認量能支持。

## 注意事項

1. **零軸洗盤嚴重**：CMF 頻繁穿過零軉是常態，必須用 ±0.05 緩衝區過濾
2. **短週期雜訊多**：20 日回溯期偏短，可用更長週期（如 50 日）降低雜訊
3. **收盤價位置偏誤**：CLV 只看收盤價在日內位置，不考慮跳空缺口，可能誤判
4. **不適合盤整**：盤整時 CMF 在零軸附近震盪，訊號無效
5. **與 MFI 類似但不相同**：MFI 用典型價格（H+L+C）/3 加上漲跌判斷，CMF 用 CLV 乘成交量，邏輯不同
6. **背離需要時間確認**：背離出現不代表立即反轉，可能持續多次背離才真正轉向
