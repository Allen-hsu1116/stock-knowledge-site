# Renko磚形圖

> 不看時間只看價格的階梯式圖表，用固定大小磚塊過濾噪音讓趨勢一目了然，磚塊大小是唯一最重要的參數

## 核心概念

Renko（練行）來自日文「煉瓦」（renga），意為「磚塊」。與K線圖不同，Renko完全忽略時間維度，只在價格移動超過設定的磚塊大小時才畫新磚。

**磚塊形成規則：**
- 收盤價 ≥ 前磚收盤價 + 磚塊大小 → 畫新綠磚（上漲）
- 收盤價 ≤ 前磚收盤價 - 磚塊大小 → 畫新紅磚（下跌）
- 不滿足條件 → 不畫磚（不管過了多久）

**與K線圖的本質差異：**
- K線每個時間段都畫一根，不管有沒有趨勢
- Renko只在有「意義的價格變動」時才畫一塊磚
- 一塊磚可能1分鐘形成，也可能數小時才形成
- Renko沒有影線，支撐壓力水平更乾淨

**與點數圖的比較：**
- 兩者都不看時間，只看價格
- 點數圖用X和O，Renko用固定大小磚塊
- 點數圖用1格單位和3格反轉，Renko用1塊磚等值反轉
- 點數圖的支撐壓力更精確（水平計算），Renko的趨勢視覺化更直觀

## 實戰應用

### 磚塊大小設定（最關鍵參數）

**三種設定方法：**

1. **固定值（Traditional）**
   - 手動設定數值（如10點、$1、50 pips）
   - 優點：簡單直觀、回測一致
   - 缺點：無法適應波動變化

2. **ATR自適應（推薦）**
   - 磚塊大小 = ATR(14) × 1.0～1.5
   - 低波動市場用ATR×1.0，高波動用ATR×1.5
   - 自動適應不同市場和波動環境
   - 切換市場不用手動調整

3. **Walk-Forward回測法**
   - 從ATR×1.2開始，回測後±10%微調
   - 直到淨利/最大回撤比合理、權益曲線平滑
   - 每月或重大波動事件後重新測試

### 趨勢辨識

Renko最大的優勢——趨勢辨識超清晰：
- **連續綠磚** = 上涨趨勢（階梯向上）
- **連續紅磚** = 下跌趨勢（階梯向下）
- **綠紅交替** = 盤整區間
- **反轉磚** = 趨勢可能轉向（2塊反轉磚確認）

### Renko + 均線策略

用20MA搭配Renko是經典組合：
- 價格在MA上方 → 做多
- 價格在MA下方 → 做空
- 盤整中綠磚不突破MA → 續持空單
- 反轉磚穿越MA → 出場訊號

**核心優勢：**讓你持有趨勢更久，不會被正常回調洗出場。

### Renko + Supertrend

Renko過濾噪音 + Supertrend確認趨勢方向：
- Supertrend翻綠 + Renko綠磚 → 做多
- Supertrend翻紅 + Renko紅磚 → 做空
- 兩者共振過濾假訊號

### 雙圖表策略

1. Renko圖辨識趨勢和支撐壓力
2. 切回K線圖找精確進場型態
3. Renko決定方向，K線決定時機

### 反轉確認

Renko的反轉需要「2塊磚」確認（設定磚塊大小×2的價格移動）：
- 不像K線一根就反轉
- 過濾掉大部分假突破
- 但也代表進場時機比K線晚

## 注意事項

### 五大常見錯誤

1. **磚塊太小** → 圖表像K線圖，訊號太頻繁，等於沒過濾
2. **對抗大時間框架趨勢** → 永遠先看大週期方向再操作
3. **忽略風險管理** → Renko過濾噪音不過濾虧損
4. **指標疊加過多** → 一兩個確認工具就夠（EMA+Supertrend）
5. **波動變化後沒調整磚塊大小** → 財報季、FOMC後必須重新設定

### Renko 的限制

1. **突破進場較晚**：需要完整磚塊確認，可能錯過突破第一段
2. **不顯示時間維度**：無法判斷盤整持續多久
3. **不顯示成交量**：需額外搭配OBV等指標
4. **不顯示缺口**：隔夜缺口變成一系列磚塊
5. **盤整市仍可能被洗**：磚塊太小沒過濾效果，太大則遲鈍
6. **反轉需要2倍磚塊確認**：進場比K線晚，好處是過濾假突破，壞處是少賺第一段

### 適用與不適用場景

**適用：**
- 趨勢追蹤和波段交易（最擅長）
- 日內交易（5分鐘+ATR磚塊）
- 加密貨幣（波動大，ATR自適應）
- 型態辨識（頭肩頂/底、雙頂/雙底更清晰）

**不適用：**
- 剝頭皮（需要時間精度）
- 成交量分析（Renko不顯示成交量）
- 時間相關策略（開盤區間、收盤策略）

### 時間框架選擇

- 波段交易：日線或4小時
- 日內交易：5分鐘
- 長期趨勢：週線
- 切換時間框架時ATR會自動調整磚塊大小
