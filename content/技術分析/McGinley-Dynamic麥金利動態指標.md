# McGinley Dynamic 麥金利動態指標

> 均線的自動變速箱——市場快它就快、市場慢它就慢，自動貼合價格減少假訊號，被稱為「你從未聽說過的最可靠指標」

## 核心概念

McGinley Dynamic（麥金利動態指標）是由特許市場技術分析師 John R. McGinley 在1990年代開發、1997年發表於《技術分析期刊》的技術指標。它本質上是一種**自適應移動平均線**，解決了傳統均線（SMA、EMA、WMA）的根本缺陷：**無法根據市場速度自動調整**。

### 傳統均線的三大問題

1. **不當應用**：10日、20日、50日均線在不同市場速度下表現差異巨大，交易者不知道該用哪一條
2. **價格分離**：均線經常與價格產生大幅偏離，落後價格行動
3. **鞭子假訊號**：短期均線在盤整市中頻繁穿越價格，產生大量假突破

McGinley 設計動態指標的目標就是讓指標線**緊貼（hug）價格**，自動消除價格分離和鞭子假訊號。

### 計算公式

```
MDi = MDi-1 + [Close - MDi-1] / [N × (Close / MDi-1)⁴]
```

另一版本含常數 k：
```
MDi = MDi-1 + [Close - MDi-1] / [k × N × (Close / MDi-1)⁴]
```

其中：
- **MDi**：當前 McGinley Dynamic 值
- **MDi-1**：前一個 McGinley Dynamic 值
- **Close**：收盤價
- **N**：移動平均週期（預設14，若要模擬20日均線則設N=10）
- **k**：常數，為選定週期的60%（k = 0.6）
- **(Close / MDi-1)⁴**：自適應因子，價格偏離越大調整越快

### 自適應機制的精妙之處

公式的核心是 `(Close / MDi-1)⁴` 這個自適應因子：

- **價格偏離越大** → 分母越大 → 調整速度越快 → 指標快速跟上
- **價格接近均線** → 分母接近1 → 調整速度正常 → 指標平穩移動

這造就了 McGinley Dynamic 最獨特的特性：**下跌時加速、上漲時減速**。這符合交易者心理——空頭想快跑、多頭想久留。

## 實戰應用

### 1. 趨勢判斷（基本用法）

| 價格位置 | MD 方向 | 判斷 |
|---------|---------|------|
| 價格在 MD 上方 | MD 向上 | 多頭趨勢 |
| 價格在 MD 下方 | MD 向下 | 空頭趨勢 |
| 價格頻繁穿越 MD | MD 水平 | 盤整，避免過度交易 |

### 2. 進出場訊號

**做多條件**：
- 價格從下方穿越 MD 並站穩在上方
- MD 線本身持續向上

**做空條件**：
- 價格從上方跌破 MD 並維持在下方
- MD 線本身持續向下

**反轉警訊**：
- 價格與 MD 線出現過度偏離 → 可能趨勢反轉

### 3. McGinley Dynamic + MACD 組合

- MD 負責平滑價格線、確認長期趨勢方向
- MACD 負責確認動能轉折
- **進場條件**：MACD 交叉信號線 + K線收盤連續兩根站在 MD 同側
- 適合波段交易和長線操作

### 4. McGinley Dynamic + RSI 組合

不看 RSI 的30/70超買超賣，改看**RSI 50 分水嶺**：

| 價格 vs MD | RSI | 訊號 |
|-----------|-----|------|
| 價格在 MD 上方 | RSI > 50 | 買進 |
| 價格在 MD 下方 | RSI < 50 | 賣出 |

停損設在前一波的 swing low/high。

### 5. McGinley Dynamic + Majority Rule 組合（進階）

Majority Rule 是動量振盪指標，衡量多個價格行為組成的共識方向：

**趨勢追蹤策略**：
- 做多：價格在上升 MD 上方 + Majority Rule 從下方穿越50
- 做空：價格在下降 MD 下方 + Majority Rule 從上方跌破50
- 退出：任一條件不滿足即減倉或出場

**背離偵測策略**：
- 看空背離：價格創高但 Majority Rule 未創高 → 等價格跌破 MD 再做空
- 看多背離：價格創低但 Majority Rule 未創低 → 等價格突破 MD 再做多

**極端值均值回歸**：
- Majority Rule > 75 且價格大幅偏離 MD → 考慮反向操作
- MD 作為護欄，永遠不逆 MD 定義的趨勢方向

### 6. 參數設定

| 欲模擬的均線 | MD 的 N 值 | 說明 |
|-------------|-----------|------|
| 20日均線 | 10 | N 設為目標均線的一半 |
| 14日均線 | 14 | 預設值 |
| 長線趨勢 | 20-30 | 波動較大的市場 |

N 值越小，MD 越貼合價格、越靈敏；N 值越大，MD 越平滑、越落後。

## 注意事項

### 優勢

1. **自動變速**：市場快就快、市場慢就慢，無需手動調整參數
2. **大幅減少鞭子假訊號**：比 SMA/EMA 少很多盤整假突破
3. **緊貼價格**：價格分離問題遠比傳統均線小
4. **下跌加速**：空頭市場中 MD 跟上價格更快，及早警告風險
5. **上漲減速**：多頭市場中 MD 移動更慢，讓交易者安心持有

### 限制

1. **仍然是落後指標**：所有趨勢追蹤工具都有落後性，快速反轉時會回吐部分利潤
2. **過渡期仍有假訊號**：趨勢轉換期間可能出現多次小虧損
3. **不適合盤整市**：MD 水平移動時訊號不明確
4. **McGinley 本人建議**：將 MD 視為平滑機制而非交易信號產生器
5. **知名度低**：多數交易平台有提供但文獻和討論較少

### 與其他自適應均線比較

| 指標 | 自適應機制 | 優勢 | 劣勢 |
|------|-----------|------|------|
| McGinley Dynamic | (Price/MD)⁴ 四次方因子 | 簡潔優雅、自動變速 | 文獻少、平台支援不一 |
| KAMA | 效率係數ER | 盤整幾乎不動、趨勢靈敏 | 計算較複雜 |
| SuperTrend | ATR 乘數 | 階梯式明確方向 | 盤整時頻繁翻轉 |
| EMA | 固定加權 | 通用性高 | 無自適應能力 |
