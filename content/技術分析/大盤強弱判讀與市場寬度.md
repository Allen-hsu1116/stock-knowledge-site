# 大盤強弱判讀與市場寬度

> 大盤不是一個數字，是整個市場的體檢報告——指數漲不等於市場健康，看懂寬度才知道能不能做

## 核心概念

**大盤強弱判讀**不只是看加權指數漲跌，更重要的是判斷整體市場的參與度和健康程度。指數可能因為少數大型股而上漲，但大多數股票下跌——這種「假強勢」是最危險的做多環境。

**市場寬度**（Market Breadth）衡量市場參與的廣度：上漲股票數量 vs 下跌股票數量、創新高 vs 創新低、均線以上 vs 均線以下。寬度擴張代表資金全面進場，寬度收縮代表只有少數股票在撐場面。

## 大盤強弱五維判斷

### 第一維：加權指數 vs 櫃買指數

| 情境 | 解讀 | 操作方向 |
|------|------|----------|
| 加權漲、櫃買漲 | 全面多頭 | 積極做多 |
| 加權漲、櫃買跌 | 大型股掩護出貨 | 謹慎，偏空 |
| 加權跌、櫃買漲 | 大型股調整、中小型仍強 | 選股做多 |
| 加權跌、櫃買跌 | 全面空頭 | 偏空或觀望 |

**關鍵原則**：櫃買指數是市場溫度計。櫃買強 = 資金願意冒險；櫃買弱 = 資金保守避險。

### 第二維：市場寬度指標

**漲跌家數比（Advance/Decline Ratio）**：
- 漲多跌少（A/D > 2:1）= 多頭健康
- 漲少跌多（A/D < 1:2）= 空頭掌控
- 漲跌各半 = 盤整待變

**漲停跌停家數**：
- 漲停家數 > 跌停家數 = 多頭氣盛
- 跌停家數 > 漲停家數 = 空頭氣盛
- 漲停0家 + 跌停多家 = 極度弱勢

**創新高創新低**：
- 創新高 > 創新低 = 多頭市場
- 創新低 > 創新高 = 空頭市場
- 兩者都少 = 盤整市場

### 第三維：成交量判斷

**量大漲**：資金積極進場，多頭確認
**量縮漲**：漲勢不穩，隨時可能回跌
**量大跌**：恐慌性賣壓或法人調節，觀察後續是否量縮止穩
**量縮跌**：賣壓減輕但買盤未進場，可能即將變盤

**5日均量 vs 20日均量**：
- 5均量 > 20均量 = 短期資金進場
- 5均量 < 20均量 = 資金退潮

### 第四維：均線與趨勢

**均線多頭排列**：5MA > 10MA > 20MA > 60MA
- 指數在所有均線之上 = 多頭市場
- 均線糾結 = 即將變盤

**關鍵均線位置**：
- 月線（20MA）是多空分界線
- 季線（60MA）是中長期趨勢線
- 年線（240MA）是多空生命線

**指數與均線的距離**：
- 離均線太遠 = 乖離過大，有回測壓力
- 貼著均線 = 均線支撐/壓力有效

### 第五維：法人籌碼

**三大法人同步買超** = 市場最強多頭訊號
**三大法人同步賣超** = 市場最強空頭訊號
**外資大買 + 投信賣** = 外資行情，看外資持續力
**投信大買 + 外資賣** = 中小型股行情

## 市場寬度進階指標

### 騰落指標（ADL, Advance-Decline Line）

計算：每日上漲家數 - 下跌家數 的累計值

**判讀**：
- ADL與指數同創新高 = 多頭健康
- 指數創新高但ADL未創新高 = **頂背離**，多頭力道不足
- ADL創新高但指數未創新高 = **底背離**，多頭蓄勢待發

### 漲跌比率（ADR, Advance-Decline Ratio）

計算：N日內上漲家數合計 ÷ N日內下跌家數合計

**常用參數**：10日ADR

**判讀**：
- ADR > 1.5 = 多頭過熱，可能回調
- ADR < 0.5 = 空頭過度，可能反彈
- ADR 在 0.5-1.5 之間 = 正常範圍

### 麥克連指標（McClellan Oscillator）

計算：19日EMA（淨漲跌家數）- 39日EMA（淨漲跌家數）

**判讀**：
- 正值擴大 = 多頭動能增強
- 負值擴大 = 空頭動能增強
- 正負反轉 = 短期趨勢轉折

### 騰落比率（Arms Index / TRIN）

計算：（上漲家數/下跌家數）÷（上漲成交量/下跌成交量）

**判讀**：
- TRIN < 1 = 多方占優（漲的家數比例高於漲的量比例）
- TRIN > 1 = 空方占優
- TRIN > 2 = 極度恐慌，可能反轉
- TRIN < 0.5 = 極度樂觀，可能回調

## 大盤強弱實戰SOP

### 開盤前（8:30-9:00）
1. 看隔夜美股漲跌和類股表現
2. 看台指期夜盤走勢
3. 看日圓和韓元匯率
4. 看期貨未平倉量變化

### 開盤後（9:00-9:30）
1. 看開盤漲跌家數比
2. 看量能是否放大
3. 看加權 vs 櫃買是否同步
4. 看法人期貨動向

### 盤中（9:30-13:00）
1. 每30分鐘檢查漲跌家數比變化
2. 觀察量能是否遞增
3. 注意漲停/跌停家數變化
4. 關注法人動態

### 收盤後
1. 計算當日ADL變化
2. 確認三大法人買賣超
3. 檢查市場寬度是否與指數背離
4. 更新大盤強弱判斷

## 大盤強弱與個股操作對應

| 大盤狀態 | 個股策略 | 部位比例 |
|----------|----------|----------|
| 全面多頭（寬度擴張+法人買超+均線多頭） | 積極做多，選強勢股 | 80-100% |
| 結構性多頭（指數強但寬度收縮） | 做多大型股，避開中小型 | 50-70% |
| 盤整（漲跌家數各半） | 區間操作或觀望 | 30-50% |
| 結構性空頭（指數弱但偶有反彈） | 減碼或做空弱勢股 | 10-30% |
| 全面空頭（寬度收縮+法人賣超+均線空頭） | 空手或做空 | 0-20% |

## 注意事項

- 大盤強弱判斷不要只看一個維度——指數漲不代表市場好，必須結合寬度、量能、籌碼
- 櫃買指數落後加權指數是常態——櫃買先跌通常預警，櫃買先漲通常確認多頭
- ADL背離是最可靠的大盤反轉訊號之一，但背離可能持續1-2個月才反轉
- 漲停家數是市場情緒的極端指標——連續3天漲停0家 = 市場極度低迷
- 法人籌碼要連續看，單日買賣超可能是調節不是趨勢
