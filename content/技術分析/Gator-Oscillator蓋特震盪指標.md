# Gator Oscillator 蓋特震盪指標

> 鱷魚線的「體溫計」——量化趨勢強弱而不判斷方向，綠柱擴張代表動能增強、紅柱縮減代表動能減弱，與鱷魚線搭配勝率可從68%提升至82%

## 核心概念

Gator Oscillator（蓋特震盪指標）由 Bill Williams 開發，是鱷魚線（Alligator）的配套指標。鱷魚線判斷趨勢方向，Gator 量化趨勢強弱——兩者合用才是完整的 Bill Williams 趨勢追蹤系統。

### 為什麼需要 Gator？

鱷魚線告訴你「趨勢往哪走」，但無法量化「趨勢有多強」。Gator 解決了這個問題：

| 問題 | 鱷魚線的局限 | Gator 的補充 |
|------|-------------|-------------|
| 趨勢強弱？ | 只能目測線距 | 柱狀圖直接量化 |
| 強弱變化？ | 線距變化慢 | 柱狀圖顏色即時反映 |
| 趨勢衰竭？ | 三線收斂才確認 | 紅柱出現提前預警 |
| 假突破？ | 單看排列可能誤判 | 柱狀圖不擴張=假訊號 |

### 計算方式

Gator 的柱狀圖分為零軸上方和下方兩部分：

**零軸上方（上方柱）**：藍線（Jaw）與紅線（Teeth）的距離
- 上方柱 = |Jaw - Teeth|

**零軸下方（下方柱）**：綠線（Lips）與紅線（Teeth）的距離
- 下方柱 = |Lips - Teeth|（取負值顯示）

**顏色規則**：
- 綠色：當前柱 > 前一根柱 → 趨勢動能正在增強
- 紅色：當前柱 < 前一根柱 → 趨勢動能正在減弱

參數與鱷魚線完全相同（5/8/13, 偏移3/5/8），一般無需修改。

## 實戰應用

### 四種狀態判斷

**① 鱷魚沉睡（震盪盤整）**
- 上方柱和下方柱都很短，接近零軸
- 紅色柱為主，偶有微弱綠色
- 含義：無趨勢，不交易

**② 鱷魚甦醒（趨勢萌芽）**
- 上方柱或下方柱開始出現綠色
- 柱狀圖從零軸附近逐漸拉長
- 含義：動能正在積聚，準備進場

**③ 鱷魚獵食（趨勢強勁）**
- 兩側柱狀圖均大幅擴張，綠色為主
- 上方柱和下方柱同時增長
- 含義：趨勢確立且動能強勁，順勢持倉

**④ 鱷魚吃飽（趨勢衰竭）**
- 兩側柱狀圖開始縮短，紅色柱增加
- 柱狀圖從高位回落
- 含義：動能衰退，考慮減倉或離場

### 多空判斷搭配

Gator 本身不判斷多空方向，需要搭配鱷魚線：

| 鱷魚線排列 | Gator 狀態 | 判斷 |
|-----------|-----------|------|
| 綠>紅>藍（多頭） | 上方綠柱 > 下方綠柱 | 短期動能強，多頭加速 |
| 綠>紅>藍（多頭） | 上方紅柱增加 | 短期動能減弱，多頭趨緩 |
| 藍>紅>綠（空頭） | 下方綠柱 > 上方綠柱 | 短期動能強，空頭加速 |
| 藍>紅>綠（空頭） | 下方紅柱增加 | 短期動能減弱，空頭趨緩 |

### 進階：Gator + 鱷魚線 + 碎形 三指標聯動

這是 Bill Williams 混沌交易法的核心組合：

**進場流程**：
1. 鱷魚線：確認趨勢方向（綠>紅>藍 或 藍>紅>綠）
2. Gator：確認動能正在增強（綠柱擴張）
3. 碎形：找到精確進場點（有效碎形突破）

**出場流程**：
1. Gator 紅柱開始增加 → 動能減弱，準備減倉
2. 鱷魚線三線開始收斂 → 趨勢可能結束，全部平倉
3. 碎形反向突破 → 確認趨勢反轉

### 實戰勝率數據

| 策略組合 | 勝率（日線） |
|---------|------------|
| 鱷魚線單獨使用 | 55%-60% |
| 鱷魚線 + 碎形 | 68%-72% |
| 鱷魚線 + 碎形 + Gator | 78%-85% |

## 注意事項

### 常見誤區

| 迷思 | 正確理解 |
|------|---------|
| Gator 可以判斷多空 | Gator 只量化強弱，方向看鱷魚線 |
| 柱狀圖越大越好 | 過度擴張可能是趨勢尾段，注意背離 |
| 紅柱出現就反手 | 紅柱只是動能減弱，不等於反轉 |
| 不需要搭配鱷魚線 | Gator 脫離鱷魚線無法判斷方向 |
| 週期越小越好 | 小週期噪音太多，建議≥1小時 |

### 指標限制

- **Gator 是滯後指標**：和鱷魚線一樣，訊號晚於價格實際轉折
- **盤整期訊號不明確**：柱狀圖在零軸附近波動，判讀困難
- **不適合極短線**：建議使用≥1小時週期
- **顏色判斷有延遲**：紅綠柱的顏色變化需等K線收盤確認
- **多空轉換期容易誤判**：趨勢反轉初期，Gator 訊號可能仍顯示舊趨勢動能

### 建議搭配

- **鱷魚線 Alligator**：判斷趨勢方向（必備搭配）
- **碎形 Fractal**：提供精確進出場訊號
- **AO Awesome Oscillator**：動能方向確認
- **成交量**：量能配合確認突破有效性
- **MACD**：動能背離確認趨勢衰竭
