# EMA指數平滑移動平均線

> EMA 對近期價格賦予更高權重，比 SMA 更靈敏地反應趨勢變化，是短線交易者和 MACD 指標的核心工具。

## 核心概念

### 什麼是 EMA

EMA（Exponential Moving Average，指數平滑移動平均線）由統計學家 Robert Goodell Brown 於 1960 年代提出指數平滑法，後應用於金融市場分析。

與 SMA（簡單移動平均線）最大的不同：EMA 採用**加權計算**，近期價格權重較高、遠期價格權重指數遞減，因此對價格變動更靈敏。

### EMA 計算公式

```
EMA = 前一日EMA × (K - 1) / (K + 1) + 今日收盤價 × 2 / (K + 1)
```

其中 K = 時間週期（如 12 日、26 日）

首次計算 EMA 時，通常以 SMA 作為初始值。

### SMA vs EMA vs WMA 對比

| 特性 | SMA | EMA | WMA |
|------|-----|-----|-----|
| **權重分配** | 均等 | 近期指數遞增 | 近期線性遞增 |
| **靈敏度** | 低（最遲鈍） | 高（最靈敏） | 中 |
| **反應速度** | 慢 | 快 | 中 |
| **適用場景** | 長期趨勢、支撐壓力 | 短線交易、MACD計算 | 較少使用 |
| **雜訊過濾** | 強 | 較弱 | 中 |

**關鍵差異**：
- SMA 所有天數權重相同 → 對支撐壓力反應更貼合
- EMA 今日價格權重最高、昨天次高、遞減 → 反應快但雜訊多
- WMA 權重線性遞減 → 介於 SMA 和 EMA 之間

## 實戰應用

### 1. 趨勢判斷

- EMA 向上 → 價格呈上升趨勢，考慮買進
- EMA 向下 → 價格呈下降趨勢，考慮賣出
- 價格在 EMA 之上 → 多方格局
- 價格在 EMA 之下 → 空方格局

### 2. 黃金交叉與死亡交叉

使用兩條不同週期的 EMA：

| 交叉訊號 | 定義 | 解讀 |
|----------|------|------|
| **黃金交叉** | 短期EMA向上穿越長期EMA（如10EMA突破30EMA） | 多方訊號，趨勢向上 |
| **死亡交叉** | 短期EMA向下穿越長期EMA（如10EMA跌破30EMA） | 空方訊號，趨勢向下 |

### 3. MACD 指標的核心計算

MACD（Moving Average Convergence Divergence）就是基於 EMA 計算的：
- **DIF** = 12日EMA − 26日EMA
- **MACD** = DIF 的 9日EMA
- 這是 EMA 最經典的應用場景

### 4. 短線交易偏好 EMA

短線/當沖交易者偏好 EMA，原因：
- 盤中價格變化快速，需要靈敏指標
- EMA 比 SMA 更早反應轉折
- 可搭配 [[K線型態]] 做確認

### 5. 參數選擇建議

| 交易風格 | 建議 EMA 參數 |
|----------|--------------|
| 極短線/當沖 | 5EMA、10EMA、20EMA |
| 波段操作 | 10EMA、20EMA、60EMA |
| 中長期投資 | 20EMA、60EMA、120EMA |

## 注意事項

### EMA 的三大陷阱

1. **大幅震盪失真**：市場出現劇烈波動時，EMA 計算值可能嚴重偏離，假訊號增多
2. **加密貨幣不適用**：價格波動過大的資產，EMA 鈍化和假突破更頻繁
3. **盤整區間洗刷**：與 SMA 一樣，盤整時均線頻繁交叉，容易被來回停損

### EMA 與 SMA 的選擇原則

- **看支撐壓力**：用 SMA，因為它貼合歷史價格共識
- **看轉折速度**：用 EMA，因為它更快反應趨勢變化
- **MACD 計算**：必須用 EMA（定義如此）
- **短線判斷**：偏好 EMA
- **長期趨勢**：SMA 和 EMA 差異不大
