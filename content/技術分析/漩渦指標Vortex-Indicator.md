# 漩渦指標 Vortex Indicator (VI)

> 從流體力學漩渦概念轉化的趨勢能量指標，VI+與VI-的開口擴大才是真趨勢，糾纏就是盤整

## 核心概念

漩渦指標（Vortex Indicator, VI）由 Etienne Botes 和 Douglas Siepman 於 2010 年發表，靈感來自流體力學中的漩渦流現象。與大多數只看收盤價的指標不同，VI 關注「波幅的連續性」，測量價格向上移動的能量（VI+）與向下移動的能量（VI-）之間的距離。

**核心邏輯**：
- **VI+（正向漩渦）**：今日最高價與昨日最低價的距離 → 多頭力道
- **VI-（負向漩渦）**：今日最低價與昨日最高價的距離 → 空頭力道
- 兩者除以真實波幅（TR）做標準化，讓不同價位的股票可以比較

**VI vs DMI 的關係**：VI 和 DMI 概念相似，都是判斷多空趨勢的方向性指標。DMI 用方向移動（+DM/-DM），VI 用漩渦移動（VM+/VM-），計算邏輯不同但目標一致。VI 的優勢是計算更簡潔、反應更直接。

## 計算方法

1. **漩渦移動量（Vortex Movement）**
   - VM+ = |今日最高價 - 昨日最低價|（多頭力道）
   - VM- = |今日最低價 - 昨日最高價|（空頭力道）

2. **真實波幅（True Range）**
   - TR = max(H-L, |H-C[1]|, |L-C[1]|)

3. **週期累加（預設14期）**
   - Sum_VM+ = 過去N期的VM+總和
   - Sum_VM- = 過去N期的VM-總和
   - Sum_TR = 過去N期的TR總和

4. **標準化計算**
   - VI+ = Sum_VM+ / Sum_TR
   - VI- = Sum_VM- / Sum_TR

5. **差額線**
   - VI 差額 = VI+ - VI-

## 實戰應用

### 標準用法：交叉訊號
- **買進**：VI+ 由下往上穿過 VI-（多頭漩渦占優）
- **賣出**：VI+ 由上往下穿過 VI-（空頭漩渦占優）
- ⚠️ 缺點：盤整時會頻繁交叉產生假訊號

### 進階用法：閾值確認
為過濾假突破，設定有效區間：
- **多頭確認**：VI+ > 1.1 且 VI- < 0.9 → 買方力道「壓倒性」勝利
- **空頭確認**：VI- > 1.1 且 VI+ < 0.9 → 賣方力道壓倒性勝利

### 背離訊號
- **頂部背離**：價格創新高但 VI+ 未同步創高 → 主力拉高出貨徵兆
- **底部背離**：價格創新低但 VI- 未同步創高 → 賣壓力道衰竭

### 與 MACD 的差異
| 比較項目 | Vortex Indicator | MACD |
|---------|-----------------|------|
| 計算基礎 | 價格高低點幾何關係 | 均線收斂發散 |
| 反應速度 | 較快 | 較慢 |
| 對轉折敏感度 | 高 | 中 |
| 盤整表現 | 糾纏假訊號多 | 柱狀圖縮小 |

## 注意事項

1. **盤整是最大剋星**：VI+ 和 VI- 在盤整時像 DNA 雙螺旋一樣糾纏，頻繁交叉產生假訊號，必須搭配 ADX 或均線過濾
2. **閾值確認比單純交叉可靠**：不要看到交叉就進場，等 VI+ 和 VI- 開口顯著擴大再行動
3. **參數建議**：預設14期適中，短線可用7-10期增加靈敏度，長線可用21-28期減少雜訊
4. **不能單獨使用**：VI 只判斷趨勢方向和能量，不判斷超買超賣，必須搭配振盪指標或支撐壓力使用
5. **假突破過濾是最大價值**：VI+ 未創高但價格創高 = 假突破真拉回，這是 VI 最強的應用場景
