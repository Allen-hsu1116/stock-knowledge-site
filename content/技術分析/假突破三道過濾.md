# 假突破三道過濾

> 假突破是散戶最常踩的陷阱，三道過濾機制可以大幅降低假突破率。

## 核心概念

假突破 = 價格短暫突破支撐/壓力線後又回到原區間，常見於：
- 散戶追突破進場 → 主力出貨
- 量能不足的突破
- 盤整末期假動作

### 三道過濾機制

**第一道：量能確認**
- 突破必須帶量：成交量 > 20 日均量 1.5 倍
- 無量突破 = 假突破機率 70% 以上

**第二道：K線確認**
- 收盤價站穩壓力線上方（非只是觸及）
- 站穩 3 天以上更可靠
- 長紅K突破比十字星突破更可靠

**第三道：時間確認**
- 突破後 3 天內不回落 = 有效突破
- 5 天內創新高 = 趨勢確立
- 反彈回壓力線獲支撐 = 最強買點

### 真突破 vs 假突破

| 特徵 | 真突破 | 假突破 |
|---|---|---|
| 量能 | 放量突破 | 縮量突破 |
| K線 | 長紅實體 | 長上影線 |
| 時間 | 3天站穩 | 1-2天回落 |
| 均線 | 多頭排列 | 均線糾結 |
| 籌碼 | 法人買超 | 散戶追價 |

## 實戰應用

1. **突破進場前檢查三道過濾**：量能 + K線 + 時間都通過才進場
2. **假突破反手操作**：假突破後回落 = 反向訊號
3. **搭配均線排列**：多頭排列下的突破更可靠
4. **停損設在突破點下方**：假突破停損距離短

## 注意事項

- 三道過濾同時通過機率極低，寧可錯過也不要硬做
- 重大消息日的突破不一定可靠（跳空缺口不適用）
- 大盤弱勢時個股突破失敗率更高
