# 假突破三道過濾

> 假突破是散戶最常踩的陷阱，三道過濾機制可以大幅降低假突破率。

## 核心概念

假突破（False Breakout）是主力製造的誘多陷阱，散戶看到突破進場後，股價迅速回落至盤整區間下方。三道過濾機制可以大幅降低被假突破騙到的機率。

### 三道過濾機制

**第一道：量能確認（帶量過濾）**
- 突破必須帶量：成交量 > 20 日均量 1.5 倍
- 無量突破 = 假突破機率 70% 以上
- 量能不足的突破缺乏動能

**第二道：K線確認（趨勢線過濾）**
- 收盤價站穩壓力線上方（非只是觸及）
- 突破是否穿越主要趨勢線或頸線達 3% 以上，盤中穿越不算有效
- 站穩 3 天以上更可靠
- 長紅K突破比十字星突破更可靠

**第三道：時間確認（打底過濾）**
- 突破前是否有足夠的底部型態（W底、頭肩底、圓形底等），沒有底部型態的突破缺乏支撐
- 突破後 3 天內不回落 = 有效突破
- 5 天內創新高 = 趨勢確立
- 反彈回壓力線獲支撐 = 最強買點

### 真突破 vs 假突破

| 特徵 | 真突破 | 假突破 |
|---|---|---|
| 量能 | 放量突破 | 縮量突破 |
| K線 | 長紅實體 | 長上影線 |
| 時間 | 3天站穩 | 1-2天回落 |
| 均線 | 多頭排列 | 均線糾結 |
| 籌碼 | 法人買超 | 散戶追價 |

## 實戰應用

1. **突破進場前檢查三道過濾**：量能 + K線 + 時間都通過才進場
2. **三道全過 = 真突破機率極高**，可進場；只過兩道 = 觀察，小部位試單；只過一道或全沒過 = 假突破機率大，避開
3. **假突破反手操作**：假突破後回落 = 反向訊號
4. **搭配均線排列**：多頭排列下的突破更可靠
5. **停損設在突破點下方**：假突破停損距離短

## 注意事項

- 三道過濾同時通過機率極低，寧可錯過也不要硬做
- 假突破常發生在主力高檔出貨期，利用「量滾量」吸引散戶追價
- 重大消息日的突破不一定可靠（跳空缺口不適用）
- 大盤弱勢時個股突破失敗率更高
- 籌碼面驗證可進一步提高判斷準確度
