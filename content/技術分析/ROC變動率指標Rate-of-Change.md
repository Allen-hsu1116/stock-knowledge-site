# ROC變動率指標 Rate of Change

> 把漲跌幅畫成曲線——ROC 是最純粹的動能溫度計，但盤整時它只會在 0 軸旁抖

## 核心概念

ROC（Rate of Change）價格變動率指標，嚴格來說它不算分析性指標，它就是我們常說的「漲跌幅變化」。只是技術指標版可以設定任意 n 天的漲跌幅，方便觀察一段時間以來的趨勢變動幅度。

ROC 被歸類為動量指標或速度指標，以 0 線為中心上下波動，衡量價格動量的強弱。

### 計算公式

```
ROC = [(收盤價 - n個交易日前的收盤價) ÷ (n個交易日前的收盤價)] × 100
```

### 參數 n 的選擇

| 交易風格 | n 值 | 特性 |
|----------|------|------|
| 超短線 | 5~9 | 反應極快，假訊號多 |
| 短線 | 12~14 | 預設值，平衡型 |
| 波段 | 20~30 | 反應慢，過濾雜訊 |
| 長線 | 50~200 | 觀察長期趨勢動能 |

n 越小，ROC 對價格變化反應越快但越敏感。n 越大，ROC 反應越慢但信號較有長期意義。

**taindicators 建議**：n 的參數一般採用 **12 天及 25 天**做為間隔週期。

### ROC 的數學特性

ROC 計算上**只受今天價格和 n 天前價格影響**，中間的 (n-1) 天完全不影響結果。這意味著：
- 如果今天或 n 天前剛好是劇烈波動日，ROC 會產生劇烈跳動
- n 越小，這種跳動效應越大
- 這跟均線不同（均線是 n 天平均，每天權重相同）

### 沒有固定範圍

ROC 沒有上下限（不像 KD 的 0~100 或 RSI 的 0~100）。因為股價漲跌幅理論上無限，長期更是如此。判斷「過熱」或「過冷」需參考歷史區間或搭配其他指標。

## 實戰應用

### 一、0 軸判讀法

| ROC 位置 | 意義 | 判讀 |
|----------|------|------|
| 0 軸之上 | 價格比 n 天前高 | 多方動能，上升趨勢 |
| 0 軸之下 | 價格比 n 天前低 | 空方動能，下降趨勢 |
| 遠離 0 軸 | 動能強 | 趨勢明確 |
| 在 0 軸附近徘徊 | 盤整 | 不宜用 ROC 交易 |
| 向 0 軸靠攏 | 動能衰減 | 趨勢可能減緩或反轉 |

### 二、0 軸交叉訊號

- ROC 向上穿越 0 軸 → 買進訊號
- ROC 向下跌破 0 軸 → 賣出訊號
- n 值越小，信號越早出現但可靠性越低
- n 值越大，信號越晚出現但可靠性越高

### 三、背離訊號

- **頂背離**：價格創新高但 ROC 沒創新高 → 漲勢趨緩，上漲動能減弱，頭部正在形成
- **底背離**：價格創新低但 ROC 沒創新低 → 跌勢趨緩，下跌動能減弱，底部正在形成

**taindicators 補充**：背離訊號是 ROC 較有價值的應用。當股價創新高但 ROC 未配合上升（不能創新高），代表上漲動能減弱；反之當股價創新低但 ROC 未配合下降（不能創新低），顯示下跌動能減弱。

**注意**：ROC 背離的實質意義不算太大，畢竟它只是兩個時間點的價格變動程度比較，不是多因子分析。

### 四、超買超賣判斷

taindicators 指出 ROC 也有超買超賣的用法：
- ROC 上升到極高位置 → 超買，產生賣出訊號
- ROC 下降到極低位置 → 超賣，產生買入訊號

但因為 ROC 沒有固定範圍，何謂「極高」「極低」需要參考歷史區間來判斷。

### 五、趨勢同步確認

- ROC 與股價從低位同步上升 → 短期有望觸底反彈或繼續上漲趨勢
- ROC 與股價從高位同時下降 → 短期警惕做頭回落或繼續下跌趨勢

### 六、ROC + 其他指標組合

ROC 的角色是「輔助圖表」，不是獨立交易系統：

- **ROC + 均線**：均線判趨勢方向，ROC 確認動能強弱
- **ROC + MACD**：MACD 確認趨勢，ROC 確認動能是否到位
- **ROC + 布林通道**：布林通道判波動，ROC 判動能方向
- **ROC + RSI**：RSI 判超買超賣，ROC 判動能方向
- **ROC + PSY心理線**：ROC 看動能強弱，PSY 看市場情緒方向，兩者角度不同但互補

## 注意事項

### 盤整時假訊號最多

ROC 在 0 軸上下運動時容易產生盤局現象，此時會產生大量干擾信號。盤整時 ROC 的唯一功能是「確認盤整」，不是交易。

### 等權重的缺陷

ROC 對最近價格和 n 期前價格給予同等權重，但中間的價格走勢完全被忽略。這可能導致：
- n 天前有一根長紅/長黑，今天的 ROC 就會被扭曲
- 例如 n=50，50 天前漲停，今天就算平盤 ROC 也會大幅跳動

### 不是預測工具

市場先生總評：ROC 是一種最簡單、容易理解的技術指標類型，它在運算上資訊輔助的意義，遠遠大於它作為趨勢判斷的意義。

ROC 告訴你的是「過去 n 天的漲跌幅度」，不是未來的走勢。動能能否延續，跟標的特性、時機點等因素有關。

### ROC vs MTM 選擇建議

- 做單一商品交易：MTM（差值）就夠用
- 做多商品比較：ROC（百分比）更適合，因為已標準化
- 兩者形狀幾乎一樣，選用哪個差異不大
- MT4/MT5 的 Momentum 本質上是 ROC 的 100 偏移版本
