# 平均K線Heikin-Ashi

> 用平均價格繪製的平滑K線圖，過濾市場噪音讓趨勢一目了然，是趨勢追隨者的好幫手。

## 核心概念

Heikin Ashi（平均K線）源自日文「平均足」，是一種使用平均值計算開高低收的K線圖表類型。與標準K線不同，Heikin Ashi 使用當前和前一個交易時段的數據來計算四個價格值，因此能過濾掉日常價格波動的噪音，讓趨勢更清晰可見。

### 計算公式

- **haOpen** =（前一個K線的 haOpen + 前一個K線的 haClose）÷ 2
- **haClose** =（當前K線的 Open + High + Low + Close）÷ 4
- **haHigh** = Max（當前 High, haOpen, haClose）
- **haLow** = Min（當前 Low, haOpen, haClose）

關鍵特點：haOpen 位於前一個K線的實體中間點，haClose 是四價平均。這使得 Heikin Ashi 不反映實際價格，而是反映平均價格的走勢。

### 與標準K線的差異

| 特性 | Heikin Ashi | 標準K線 |
|------|-------------|---------|
| 趨勢清晰度 | 更清晰，噪音少 | 有較多雜訊 |
| 進出場精確度 | 較低 | 較高 |
| 假突破過濾 | 有效過濾 | 容易受騙 |
| 缺口判斷 | 不適用 | 適用 |
| 實際價格 | 不顯示 | 顯示 |

## 實戰應用

### 五種典型訊號

1. **強勢多頭**：連續綠色長實體、無下影線 → 趨勢強勁，持股續抱
2. **多頭轉弱**：綠色K線出現下影線 → 上升動量減弱，開始考慮減碼
3. **強勢空頭**：連續紅色長實體、無上影線 → 趨勢強勁，空單續抱
4. **空頭轉弱**：紅色K線出現上影線 → 下跌動量減弱，開始考慮回補
5. **盤整期**：十字線或小實體交替出現 → 市場猶豫，等待突破方向

### 操作策略

**趨勢追隨策略**：
- 在Heikin Ashi 圖上確認連續同色K線（3根以上）→ 趨勢確立
- 搭配均線確認方向：價格在均線上方 + 連續綠色K線 = 做多
- 出現反向影線時開始警覺，但不立即反向操作

**反轉辨識策略**：
- 連續綠色K線後出現帶下影線的綠K → 多頭動量衰減
- 連續紅色K線後出現帶上影線的紅K → 空頭動量衰減
- 搭配 [[RSI相對強弱指標]] 背離或 [[MACD指標實戰判讀]] 交叉確認反轉

**假突破過濾**：
- 在短時間週期交易波動性較大的標的時，Heikin Ashi 可過濾支撐壓力的假突破
- 標準K線看似突破，但 Heikin Ashi 仍維持同色 → 可能為假突破
- 搭配 [[支撐壓力]] 關鍵位做雙重確認

### 程式交易實作注意

在 MultiCharts 等平台回測時，不能直接用 Open/High/Low/Close 變數，因為 Heikin Ashi 圖表上顯示的是轉換後的值。需自建 haOpen、haClose、haHigh、haLow 四個函數，才能做到「所視即所得」。

## 注意事項

1. **不反映實際價格**：Heikin Ashi 顯示的是平均價格，不能用於精確的進出場點位計算。停損停利仍需參考標準K線的實際價格
2. **延遲性**：因為使用前一期數據計算，訊號有天然延遲，不適合極短線當沖的精確進出
3. **不適用缺口分析**：Heikin Ashi 平滑了價格，缺口被抹除，無法做缺口相關策略
4. **搭配使用效果最佳**：單獨使用 Heikin Ashi 容易錯過精確進場點，建議搭配標準K線或 [[K線型態]] 做最終確認
5. **連續同色不代表永遠持續**：相同顏色的K線越多，趨勢結束的可能性也越大，需提高警覺
