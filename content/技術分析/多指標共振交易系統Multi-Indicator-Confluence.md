# 多指標共振交易系統 Multi-Indicator Confluence

> 單一指標勝率40-50%，三個獨立指標共振勝率提升至65-70%，關鍵在於從不同類別選指標避免冗餘

## 核心概念

共振交易（Confluence Trading）是指多個獨立的技術訊號在同一價格區域同時指向同一方向，產生高機率的進出場訊號。核心原則：**訊號越多指向同一方向，交易成功的機率越高**。

### 共振的機率提升

| 指標數量 | 大約勝率 | 備註 |
|---------|---------|------|
| 單一指標 | 40-50% | 隨機程度高 |
| 兩個指標共振 | 55-60% | 顯著提升 |
| 三個指標共振 | 65-70% | 最佳平衡點 |
| 四個以上 | 70-75% | 邊際效益遞減 |

### 四大指標類別（避免冗餘）

選擇指標時，**必須從不同類別中各選一個**，同類別的指標會產生冗餘訊號：

| 類別 | 功能 | 適用指標 |
|------|------|---------|
| **趨勢指標** | 判斷方向 | 均線（SMA/EMA）、MACD、ADX |
| **動量指標** | 判斷強弱 | RSI、KD/KDJ、CCI |
| **成交量指標** | 確認真偽 | OBV、成交量均線、MFI |
| **波動率指標** | 判斷區間 | 布林通道、ATR、肯特納通道 |

**冗餘組合**（避免）：RSI + KD + CCI（三個都是動量指標）
**有效組合**（推薦）：MACD（趨勢）+ RSI（動量）+ OBV（成交量）

### MACD + RSI + KDJ 三指標共振

這是最經典的共振組合之一，涵蓋趨勢、動量、時機三個維度：

| 指標 | 負責維度 | 核心訊號 | 盲點 |
|------|---------|---------|------|
| MACD | 趨勢方向 | 零軸上下判多空、金叉死叉 | 盤整時頻繁假訊號 |
| RSI | 動量強弱 | 超買超賣、背離、50分水嶺 | 強趨勢中長期停留在超買/超賣區 |
| KDJ | 進場時機 | J值極端判斷超買超賣、KD交叉 | 假交叉多，需趨勢過濾 |

### 共振評分系統

對每個交易設定打分（1-5分），根據分數決定部位大小：

| 因素 | 滿足條件 |
|------|---------|
| 大時間框架趨勢方向一致 | +1 |
| 回測到關鍵支撐/壓力位 | +1 |
| 多個技術位重疊在同一區域 | +1 |
| 價格行為訊號（K線型態） | +1 |
| 指標確認 | +1 |

- **5分**：絕佳交易 → 最大部位（2%風險）
- **4分**：良好交易 → 標準部位（1.5%風險）
- **3分**：一般交易 → 縮小部位（1%風險）
- **2分**：弱交易 → 跳過或最小部位（0.5%）
- **1分**：不交易 → 跳過

## 實戰應用

### 策略一：趨勢追蹤進場

**條件**：
1. MACD 在零軸上方（確認上升趨勢）
2. RSI 回測至 40-50 區間後反彈（淺回檔）
3. KDJ 在超賣區金叉（K < 20 穿過 D）

**進場**：三個條件同時滿足
**停損**：近期波動低點下方或 2× ATR
**停利**：RSI 達到 70 或 MACD 出現背離

### 策略二：反轉進場

**條件**：
1. RSI 背離（價格創新低但 RSI 沒有）
2. KDJ 在超賣區金叉（J < 0 後穿過）
3. MACD 柱狀圖由負轉正（動能轉向）

**進場**：三個條件確認後
**停損**：背離低點下方
**停利**：前波壓力或斐波那契擴展位

### 策略三：突破確認

**條件**：
1. 價格突破關鍵壓力位
2. MACD 在訊號線上方（趨勢確認突破方向）
3. RSI > 60（動量強勁但未超買）
4. KDJ 的 K 值 > 80（突破強度）
5. 成交量明顯放大

**進場**：突破K線收盤確認或次根K線拉回
**停損**：突破位下方
**停利**：等幅測量或下一個壓力位

### 策略四：布林通道 Squeeze 突破共振

**條件**：
1. 布林通道帶寬收縮至第10百分位以下（Squeeze）
2. 收盤突破上軌或下軌
3. OBV 趨勢方向一致
4. MACD 訊號線交叉確認

**進場**：突破確認後
**停損**：布林通道中軌
**停利**：對面軌道或 2× ATR

### 多時間框架共振

不同時間框架負責不同任務：

| 時間框架 | 用途 | 建議指標 |
|---------|------|---------|
| 週線 | 大趨勢方向 | 均線排列、MACD |
| 日線 | 交易偏多/偏空 | MACD + RSI |
| 4小時 | 進場時機 | RSI + KDJ |
| 1小時 | 精確進場點 | KDJ、K線型態 |

**規則**：大時間框架趨勢永遠優先。順著週線方向操作，用日線找進場區間，用小時間框架精確進場。

### 當訊號衝突時

如果 MACD 說多但 RSI 超買且 KDJ 死叉：
- **不要強迫交易**——訊號衝突代表方向不明
- **縮小部位**如果非要交易
- **等待解決**——最終會有一方勝出
- **查看更大時間框架**尋找主導趨勢

## 注意事項

### 五大常見錯誤

1. **指標冗餘**：RSI + KD + CCI 三個動量指標同時用，幾乎不增加資訊量
2. **過度優化**：不斷調整參數來fit歷史數據，實盤一定崩
3. **忽略市場狀態**：共振在趨勢市有效，在盤整市訊號會衝突
4. **確認偏誤**：只看到支持自己方向的訊號，忽略反對訊號
5. **分析癱瘓**：指標太多永遠不進場 = 錯過機會

### 關鍵原則

- **3-4個共振指標是甜蜜點**，超過5個反而造成決策癱瘓
- **永遠用成交量作為第四確認**——沒有量能的指標訊號較弱
- **回測後再實盤**——共振系統必須經過歷史驗證
- **簡單勝複雜**——更多指標 ≠ 更好結果
- 共振的價值主要來自**過濾壞交易**，不是提升進場時機
- 進場前先問：這筆交易有幾分共振？3分以下不進場
