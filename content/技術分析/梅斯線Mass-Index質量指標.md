# 梅斯線 Mass Index 質量指標

> 用高低價差波幅的「擴張→收縮」循環捕捉趨勢反轉的指標，27 閾值穿越搭配 9 日均線方向是核心訊號，風險係數在震盪指標中最小

## 核心概念

**梅斯線 (Mass Index)** 由 Donald Dorsey 開發，核心邏輯是：當高低價差（波幅）劇烈擴張到極限後收縮，趨勢反轉的機率大增。它不判斷方向，只判斷「反轉的可能性」，方向需搭配均線確認。

Mass Index 是所有區間震盪指標中**風險係數最小**的一個——它不預測方向，只告訴你「波幅極端擴張了，反轉可能在附近」。

### 計算方式

1. DIF = High - Low（每日高低價差）
2. AHL = DIF 的 9 天 EMA
3. BHL = AHL 的 9 天 EMA
4. MI = Σ(AHL/BHL, N)，N 一般為 25 天

本質：AHL/BHL 是波幅的「放大倍率」，當倍率持續偏高並累積，Mass Index 就會上升。N=25 代表觀察 25 天的波幅倍率累積值。

### 參數調整

| 參數 | 預設 | 調整版 | 說明 |
|------|------|--------|------|
| 累積天數 N | 25 | 60（長週期） | 減少假訊號，適合波段操作 |
| 搭配均線 | 9 日 MA | 25 日 MA | 配合長週期，過濾短期雜訊 |
| 穿越閾值 | 27 | 可依市場調整 | 波動大的市場可能需提高 |

## 實戰應用

### 1. 反轉訊號——「凸出部份」27→26.5 穿越法

這是 Mass Index **唯一核心訊號**：

**步驟**：
1. Mass Index 向上穿越 27 → 波幅極端擴張，反轉可能性增加
2. Mass Index 隨後回落跌破 26.5 → 確認波幅開始收縮
3. 看當時 9 日均線方向決定反轉方向：
   - **9 日均線上升中** → 多頭反轉下跌（多頭力竭）
   - **9 日均線下跌中** → 空頭反轉上漲（空頭力竭）

### 2. 低於 25 ——不具投資機會

Mass Index 低於 25 代表波幅狹窄、缺乏波動，這類股票短期內不太會有大行情。可以把 MI > 25 當作選股過濾條件。

### 3. 搭配策略

- **Mass Index + [[均線判斷\|9日均線]]**：基本搭配，均線方向判斷反轉方向
- **Mass Index + [[SAR拋物線指標Parabolic-SAR\|SAR]]**：MI 確認反轉，SAR 給進出場點位
- **Mass Index + [[ADX趨勢強度過濾盤整\|ADX]]**：MI 抓反轉時機，ADX 確認趨勢強度變化
- **Mass Index + [[布林通道Bollinger-Bands三軌八型態\|布林通道]]**：兩者都跟波幅有關，布林通道收窄+MI 突破 27 共振更可靠

### 4. 程式交易腳本（TradeScript 參考）

```
input: Length(9), MassSum(25), Band(27), Trigger(26.5);
Vars: AHL(0), BHL(0), Mass(0);

AHL = XAverage(Range, Length);
BHL = XAverage(AHL, Length);
if BHL <> 0 then Value1 = AHL / BHL;
Mass = Summation(Value1, MassSum);

// 買進條件：MI 穿越27後跌破26.5 + 9日均線下跌中
// 賣出條件：MI 穿越27後跌破26.5 + 9日均線上升中
```

台指期 30 分 K 回測（2004-2014）：訊號稀少但勝率偏高，適合搭配其他策略使用。

## 注意事項

- **訊號稀少**：27→26.5 穿越不是天天發生，在盤整期可能長期沒訊號——這是優點也是缺點
- **只判斷反轉不判斷方向**：沒有 9 日均線搭配，Mass Index 無法告訴你該買還是該賣
- **參數敏感**：預設 25/27/26.5 適合一般市場，波動大的個股或期貨可能需要調整閾值
- **假凸出**：有時 MI 突破 27 是因為單日極端波動（如除權息、重大消息），不一定是趨勢反轉訊號，需看後續是否持續
- **長週期參數更穩定**：MBA 智庫建議將累積天數改為 60、均線改為 25 日，過濾假訊號效果更好
- **與其他反轉指標共振**：單獨使用勝率有限，搭配[[背離Divergence進階實戰|背離]]、[[威科夫方法|Wyckoff Spring]] 等反轉確認更可靠
