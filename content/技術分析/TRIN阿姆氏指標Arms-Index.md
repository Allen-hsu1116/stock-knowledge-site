# TRIN阿姆氏指標 Arms Index

> 用漲跌家數比對漲跌成交量比，偵測資金流向的「市場體溫計」——TRIN低於1是多方資金集中，TRIN高於1是空方資金主導，極端值反轉是進出場訊號

## 核心概念

**TRIN**（Short-Term TRading INdex），又稱**阿姆氏指標**（Arms Index），由Richard W. Arms Jr.於1967年開發，是將**漲跌家數比**與**漲跌成交量比**相除得出的比率指標。

TRIN的核心邏輯是：不僅看「多少股票漲跌」，還要看「漲跌的股票背後有多少資金在推動」。一個市場如果1000家漲、500家跌，但漲的股票成交量只佔20%，跌的佔80%，那表面看是多方佔優，實際上空方資金力量更強——TRIN能捕捉這種差異。

### 公式

```
TRIN = (上漲家數 / 下跌家數) ÷ (上漲成交量 / 下跌成交量)
```

- 上漲家數（Advances）：當日收漲的股票數量
- 下跌家數（Declines）：當日收跌的股票數量
- 上漲成交量（Advancing Volume）：漲股的總成交量
- 下跌成交量（Declining Volume）：跌股的總成交量

### 與一般超買超賣指標的最大差異

| 特性 | 一般超買超賣指標 | TRIN |
|------|-----------------|------|
| 中界線 | 0 或 50 | **1.0** |
| 超買區域 | 數值偏高 | 數值**偏低**（< 0.5） |
| 超賣區域 | 數值偏低 | 數值**偏高**（> 2.0） |
| 方向 | 與市場同向 | 與市場**反向** |

⚠️ **TRIN的超買超賣定義與一般指標相反！** TRIN極低=市場超買（過度樂觀），TRIN極高=市場超賣（過度恐慌）。

## 實戰判讀

### 基本判讀原則

| TRIN數值 | 解讀 | 市場狀態 |
|----------|------|----------|
| TRIN < 0.5 | 漲股數量和成交量都遠超跌股 | 極度樂觀（超買），留意回調 |
| TRIN 0.5-0.8 | 多方資金集中 | 健康多頭 |
| TRIN ≈ 1.0 | 漲跌家數比與漲跌量比相當 | 多空均衡 |
| TRIN 1.0-1.5 | 空方資金稍強 | 偏空但未失控 |
| TRIN > 2.0 | 跌股數量和成交量都遠超漲股 | 極度恐慌（超賣），留意反彈 |

### 進階判讀：四種經典情境

**情境一：TRIN < 0.5（極度超買）**
- 漲股家數多、漲股量也大 → 市場過度樂觀
- 但如果持續維持低檔，代表多方力量強勁
- **對策**：不要急著做空，等TRIN從低位反彈突破0.5才考慮減碼

**情境二：TRIN > 2.0（極度超賣）**
- 跌股家數多、跌股量也大 → 市場恐慌拋售
- 極端值出現後常伴隨反彈
- **對策**：不要急著接刀，等TRIN從高位回落跌破1.5才考慮進場

**情境三：指數漲但TRIN > 1.0（表面多頭實質空頭）**
- 漲的家數少但漲的量大（大型股拉抬）
- 跌的家數多但跌的量少
- 這是**假多頭**的典型訊號
- **對策**：減碼或避開中小型股

**情境四：指數跌但TRIN < 1.0（表面空頭實質多頭）**
- 跌的家數少但跌的量大（大型股殺盤）
- 漲的家數多但漲的量少
- 大型股修正結束後，中小型股可能接棒
- **對策**：注意大型股止穩後的進場機會

### 均線平滑應用

原始TRIN波動劇烈，建議搭配均線使用：

| 週期 | 用途 | 超買/超賣參考值 |
|------|------|-----------------|
| 4-5日EMA | 短線交易 | 超買<0.7、超賣>1.5 |
| 21日EMA | 中期趨勢（最常用） | 超買<0.7、超賣>1.1 |
| 55日EMA | 長期趨勢 | 視市場調整 |

### 確認訊號SOP

1. **底部確認**：TRIN衝高超賣區 → 等TRIN回落跌破超賣線 → 確認底部
2. **頭部確認**：TRIN跌破超買區 → 等TRIN反彈突破超買線 → 確認頭部
3. **背離確認**：指數創新高但TRIN未創新低 → 頭部背離 → 留意反轉
4. **極端值反轉**：TRIN > 3.0 通常是恐慌性拋售 → 1-3日內可能反彈

## 台股應用注意事項

1. **台股適用性**：TRIN原設計針對NYSE，台股需調整超買超賣閾值。建議先觀察3-6個月的歷史TRIN分布，找出適合的區間。

2. **加權 vs 櫃買分開計算**：加權指數TRIN和櫃買指數TRIN可能不同步，分開觀察更精準。

3. **搭配[[技術分析/騰落指數ADL判讀|騰落指數]]**：ADL看累計趨勢，TRIN看每日資金流向，兩者搭配判斷大盤更完整。

4. **搭配[[技術分析/McClellan-Oscillator麥克萊倫震盪指標|麥克萊倫震盪指標]]**：McClellan Oscillator是ADL的動能版本，TRIN是量價比率版本，互補性強。

5. **急跌行情的侷限**：市場出現猛烈利空直接暴跌時，TRIN訊號會落後。這時需搭配VR成交量指標、BR情緒指標輔助。

6. **半對數刻度**：建議使用半對數（log）刻度看TRIN，因為0.5→1.0和1.0→2.0的變動幅度相同（都是100%），等距刻度會讓高位值看起來不成比例。

7. **順趨勢操作**：TRIN是反向指標，但要在主趨勢方向使用——多頭市場中偏用超賣訊號做多，空頭市場中偏用超買訊號減碼。

## 與其他市場寬度指標的比較

| 指標 | 計算基礎 | 優勢 | 弱勢 |
|------|----------|------|------|
| TRIN | 漲跌家數比÷漲跌量比 | 同時考量家數和成交量 | 原始值波動大，需平滑 |
| ADL | 漲跌家數累計 | 看長期趨勢 | 不考量成交量 |
| ADR | N日漲跌家數比 | 簡單直觀 | 不考量成交量 |
| McClellan Oscillator | 漲跌家數EMA差 | 動能變化明確 | 不考量成交量 |
| McClellan Summation | McClellan Oscillator累計 | 中長期趨勢 | 不考量成交量 |

**TRIN是唯一同時考量漲跌家數和成交量的市場寬度指標**，這是它的核心差異化價值。

## 注意事項

- ⚠️ **超買超賣定義相反**：TRIN極低=超買（過度樂觀），TRIN極高=超賣（過度恐慌），這是新手最常搞錯的地方
- **極端值不等於反轉**：TRIN > 3.0可能持續，等確認訊號才進場
- **不同市場不同閾值**：NYSE、NASDAQ、台股的超買超賣區間不同，必須回測調整
- **急跌行情TRIN會落後**：V型反轉和閃崩時TRIN參考價值降低
- **單獨使用效果有限**：搭配趨勢指標（均線、ADX）和成交量指標（OBV、VR）效果更佳
- **TRIN看的是「資金流向哪裡」而非「指數去哪裡」**：指數漲但TRIN>1代表資金流向空方，指數跌但TRIN<1代表資金流向多方
