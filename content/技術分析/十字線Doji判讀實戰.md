# 十字線Doji判讀實戰

> 十字線是市場多空僵持的快照，開盤≈收盤代表動能暫停，但絕非自動反轉保證——位置、成交量、確認訊號三維度缺一不可。

## 核心概念

十字線（Doji）的定義特徵是**開盤價與收盤價幾乎相同**，實體極小或消失，形成十字狀。它代表盤中多空激烈交戰，但最終雙方勢均力敵，回到起點。

**為什麼十字線重要**：
- 出現在趨勢末端 → 可能是動能衰竭的早期預警
- 出現在關鍵價位 → 與支撐壓力共振，訊號可靠度大增
- 多根連續出現（群集） → 比單一根更可靠的反轉訊號

## 十字線五大型態

| 型態 | 特徵 | 市場意義 |
|------|------|----------|
| **小十字線** | 上下影線極短 | 盤整觀望，交投清淡，無方向性 |
| **長腳十字線** | 上下影線都長 | 多空激烈爭奪，波動大但未分勝負，大變化前兆 |
| **蜻蜓線（T字線）** | 長下影線，無上影線 | 低檔湧現強大買盤，底部見底訊號；高檔出現則買方力盡 |
| **墓碑線（倒T字線）** | 長上影線，無下影線 | 高檔遭遇沉重賣壓，頂部反轉訊號；低檔出現則賣方力盡 |
| **四價十字線** | 開=高=低=收 | 漲跌停鎖死或極冷門股，需額外過濾 |

**實戰經驗**：
- 蜻蜓線出現在下跌末端 + 縮量 → 見底反轉成功率最高
- 墓碑線出現在上升末端 + 放量 → 頂部反轉成功率最高
- 長腳十字線 = 消息面驅動的劇烈波動 → 等待方向確認

## 十字線 vs 紡錘線

兩者外觀相似但訊號嚴格程度不同：

| | 十字線 | 紡錘線 |
|---|---|---|
| 實體 | 極小或消失，開≈收 | 小實體，但開≠收 |
| 意義 | 市場完全未做方向選擇 | 略有方向性推進 |
| 確認要求 | 需更明確的確認K線或區間突破 | 確認條件相對寬鬆 |

**判斷訣竅**：實體不夠小 → 更接近紡錘線 → 不宜用十字線的嚴格確認規則操作。

## 十字線經典組合型態

### 早晨之星（底部反轉）

1. 大黑K（空頭主導）
2. 跳空十字線（跌勢受阻，多空平衡）
3. 大紅K深入第一天實體（多方反攻確認）

若十字線為長腳或蜻蜓線 → 成功率更高。

### 黃昏之星（頂部反轉）

1. 大紅K（多頭主導）
2. 跳空十字線（漲勢受阻，多空平衡）
3. 大黑K吞噬第一天漲幅（空方反攻確認）

若十字線為墓碑線 → 極強賣出訊號。

### 續盤十字線（中繼整理）

出現在強勢趨勢中段的回檔或反彈 → 主力洗盤/換手 → 原趨勢繼續。

## 十字線群集（Doji Cluster）

**關鍵進階概念**：連續2-3根十字線出現在同一價位區域，比單一根十字線可靠得多。

- 多根群集 = 市場反覆拒絕突破 = 動能持續收縮
- TMGM 原創研究：USDJPY 2024/10 在阻力位連續3根十字線群集，下一根K線即確認反轉
- 群集期間成交量未擴大 → 強化買方動能耗盡判斷

## 實戰應用

### 五步確認法（Confluence 交易法）

1. **識別十字線**：確認實體相對近期K線明顯偏小（太小則是紡錘線）
2. **判斷趨勢位置**：趨勢末端 vs 趨勢中段（前者衰竭，後者暫停）
3. **檢查關鍵價位**：前波高低點、支撐壓力、趨勢線、均線 → 共振提升可靠度
4. **等待確認訊號**：
   - 下一根K線方向性確認
   - 十字線高低點突破
   - 成交量確認
   - RSI/MACD共振
5. **設置進出場**：
   - 看漲：Buy Stop 設在十字線高點上方
   - 看跌：Sell Stop 設在十字線低點下方
   - 停損：十字線波幅作為失效地圖

### 位置+量+確認 三維判斷

| 維度 | 高檔十字線 | 低檔十字線 |
|------|-----------|-----------|
| 位置 | 上升趨勢末端 | 下降趨勢末端 |
| 成交量 | 放量 → 反轉機率高 | 縮量 → 底部訊號 |
| 均線/支壓 | 搭配壓力線 → 看跌 | 搭配支撐線 → 看漲 |
| 次日確認 | 跌破十字線低點 → 減碼 | 帶量紅K突破高點 → 佈局 |

### 風險管理

- 十字線高低點 = 自然失效地圖（想法被證明錯誤的價位）
- 單一根十字線假訊號率高，必須搭配確認才進場
- 止損設在十字線影線極端點之外

## 注意事項

1. **十字線≠自動反轉**：最常見的誤解。同一根十字線可能是反轉也可能是續盤，必須由下一根K線證明
2. **收紅收黑不重要**：實體極小，顏色無意義；位置和影線長度才是關鍵
3. **忽略周圍價格行為是致命的**：逆勢出現在無關鍵位置的十字線 = 低品質雜訊
4. **過度依賴單根十字線**：假訊號率極高，應追求多根群集或多指標共振
5. **強趨勢中的十字線**：極端強勢多頭中，個股十字線容易被資金輕易化解
6. **不等待確認就進場**：這是頻繁觸發停損的主因
