# TRIX三重指數平滑移動平均指標

> 三次EMA平滑消除雜訊，TRIX是MACD的更流暢版本——零軸判多空、交叉找時機、背離抓反轉，但盤整時一樣會被洗

## 核心概念

TRIX（Triple Exponential）由Jack Hutson於1980年發表，概念很直覺：對收盤價連續取三次指數移動平均（EMA），然後計算變動率。三次平滑的結果讓移動平均線極度平滑，能有效消除設定期間內的價格波動雜訊，最大程度減少主力「騙線」行為的干擾。

**計算步驟**：
1. AX = N日收盤價的EMA
2. BX = N日AX的EMA
3. CX = N日BX的EMA（三次平滑完成）
4. TRIX = (當日CX - 前一日CX) / 前一日CX

TRIX值以零為中軸上下波動，正値代表動能向上、負値代表動能向下。

### TRIX vs MACD

兩者都是長期趨勢指標，但TRIX線條更加流暢，過濾雜訊能力更強。代價是反應更慢——TRIX比MACD更落後。

## 實戰應用

### 1. 零軸判多空（長線）

- TRIX在零軸上方 → 股價處於漲勢
- TRIX在零軸下方 → 股價處於跌勢

三次平滑均線的轉折點剛好對應TRIX穿越零軸的時機。缺點：有明顯落後性，等到TRIX觸零軸時股價已經漲或跌了一段。

### 2. 信號線交叉找時機（短線）

取一條TRIX的移動平均線（如10日TRIX）作為確認線：
- 9日TRIX由下往上突破10日TRIX → 買進確認
- 9日TRIX由上往下跌破10日TRIX → 賣出確認

比零軸法更早出現訊號，但準確性較差，尤其在橫向整理期。

### 3. 背離抓反轉

- **頂背離**：股價創新高，TRIX沒有創新高 → 強烈賣出訊號
- **底背離**：股價創新低，TRIX沒有創新低 → 強烈買入訊號

背離在強勢行情中較可靠。高檔只需一次頂背離即可確認反轉，低檔通常需要多次底背離。

### 4. 用在短頻率時間框架

TRIX在日線圖上落後明顯，但放在五分鐘、十分鐘、十五分鐘線上，準確性會明顯改善。長線投資者用日線、短線交易者用分鐘線，各取所需。

## 注意事項

1. **盤整剋星**：TRIX不適用於盤整行情，會產生大量假訊號。搭配ADX過濾盤整是必要的
2. **落後指標**：三次平滑換來的是極度流暢，但也代表反應更慢，絕不能拿來抄底摸頂
3. **長線定位**：TRIX最適合穩健型長期投資者，頻繁交易者會被手續費吃掉利潤
4. **訊號確認**：交叉訊號必須搭配其他指標（成交量、ADX、支撐壓力）確認，不宜單獨使用
5. **參數選擇**：預設9日/10日是常見組合，可依交易週期調整，但核心是短線確認長線方向
