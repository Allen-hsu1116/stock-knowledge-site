# 多時間框架分析

> 大週期看方向、中週期判趨勢、小週期找進場——用三層濾網過濾雜訊，讓你在對的方向做對的交易

## 核心概念

多時間框架分析（Multi Time Frame Analysis, MTF）是指透過多個時間週期來分析同一金融商品，從宏觀到微觀建立一致性判斷。

很多人對同一檔股票看多看空意見不同，其實兩邊都沒錯，只是談論的週期不同。日線看空但月線看多，可能是月線多頭中的日線回檔——這就是多時間框架要解決的問題。

**核心原則：長線保護短線**
長期趨勢難以扭轉，若長期趨勢為多方架構，短線操作就應以偏多思考為核心。

### 三重時間框架分析法

| 角色 | 功能 | 波段操作 | 當沖操作 |
|------|------|----------|----------|
| 🔭 **大週期** | 看位置、定方向 | 月線/週線 | 4小時/1小時 |
| 🎯 **中週期** | 判斷趨勢、找區間 | 日線 | 15分鐘/30分鐘 |
| ⚡ **小週期** | 尋找進場點、精確停損 | 60分鐘/4小時 | 5分鐘/1分鐘 |

### 4-6 倍法則：選擇時間框架

Rayner 提出的簡單法則：**下層時間框架 × 4~6 = 上層時間框架**

| 交易時間框架 | 上層參考（×4） | 更上層參考（×4） |
|---|---|---|
| 5分K | 30分K | 2小時K |
| 15分K | 1小時K | 4小時K |
| 1小時K | 4小時K | 日線 |
| 4小時K | 日線 | 週線 |
| 日線 | 週線 | 月線 |

新手建議從 **2 個時間框架** 開始，熟練後再加到 3 個。超過 3 個容易陷入分析癱瘓。

### 不同操作風格的時間框架組合

| 操作風格 | 組合 | 用途 |
|---|---|---|
| 當沖經典 | 15M / 1H / 4H | 4H:趨勢 1H:型態 15M:進場 |
| 當沖剝頭皮 | 5M / 15M / 1H | 1H:趨勢 15M:型態 5M:執行 |
| 波段標準 | 1H / 4H / 日線 | 日線:趨勢 4H:型態 1H:進場 |
| 波段延伸 | 4H / 日線 / 週線 | 週線:趨勢 日線:型態 4H:執行 |
| 混合非傳統 | 1H / 日線 / 週線 | 週線:週期 日線:關鍵位 1H:精確進場 |

### 為何需要多時間框架？

- **避免方向誤判**：15分鐘的多頭上漲，可能只是4小時級別的回調
- **過濾假訊號**：低層級時間框架充滿噪音，高層級時間框架是天然的過濾器
- **提高盈虧比**：用大週期趨勢方向入場、小週期精確停損，盈虧比可達5倍以上
- **精確進場時機**：高層級找方向、低層級找時機，組合起來進場更精準
- **建立交易信心**：多個時間框架同時支持你的判斷，減少猶豫和提前出場

## 實戰應用

### Trend → Setup → Entry 三步系統

**Step 1：高層級 — 定義趨勢**
- 用市場結構判斷：更高高更高低（多頭）、更低高更低低（空頭）
- 也可以用 50 期均線輔助：價格在上方=多頭，在下方=空頭
- 標記主要支撐壓力位

**Step 2：中層級 — 找到型態**
- 在趨勢方向中找拉回（Pullback）機會
- 辨識常見拉回型態：多頭旗形、空頭旗形、對稱三角、矩形整理
- 等待拉回到達關鍵支撐/壓力位

**Step 3：低層級 — 確認進場**
- 等待價格行為訊號（K線反轉型態、帶量突破）
- 確認成交量
- 設定停損：多頭放在最近 swing low 下方，空頭放在最近 swing high 上方

### 堆疊支撐壓力位（Stacked Levels）

**這是多時間框架分析中最強大的進階技巧。**

當支撐或壓力位在多個時間框架中重疊時，就是「堆疊位」。堆疊位的反應機率遠高於單一時間框架的支撐壓力位，這是高機率交易區域。

### KD 三重確認經典範例

1. 月線級別 KD 黃金交叉 → 長期多方格局確認
2. 週線級別 KD 也黃金交叉 → 中期偏多
3. 日線級別 KD 黃金交叉 → 短期轉強買入訊號

三個級別共振，勝率最高。這就是 [[操作策略/三重濾網交易系統]] 的核心邏輯。

### 型態跨時間框架確認

高層級時間框架的型態比低層級的更有意義：
1. **高層級型態**（日線/週線）：頭肩頂、雙底、三角型態 → 反映更大範圍的市場情緒
2. **低層級型態**（1H/4H）：5分鐘的雙底 → 可能只是噪音

**實戰作法**：發現日線的型態 → 不要直接進場 → 切到低層級等待突破確認

## 常見錯誤

1. **從最低層級開始分析**：應該從最高層級往下推
2. **使用太多時間框架**：2-3 個就夠了，太多反而分析癱瘓
3. **逆著高層級趨勢操作**：月線空頭卻在日線做多，勝率極低
4. **挑選時間框架來確認偏見**：這不叫分析，叫自我欺騙
5. **疊加太多指標**：時間框架已經夠複雜了，每個層級一個指標就好
6. **時間框架不匹配操作策略**：當沖看日線沒意義，波段看5分K噪音太多
7. **週期差距不能太大**：看月線配1分鐘線沒意義

## 注意事項

- **不同級別訊號矛盾時，以大級別為準**
- **不是所有型態都值得交易**：只有和高層級趨勢一致的型態才有價值
- **成交量是跨時間框架確認的關鍵**
- **必須大量回測**：每個人的交易習慣不同，適合的時間框架組合也不同
- **不一定要用多時間框架**：有些成功交易者只專注單一週期也能穩定獲利，多時間框架是工具不是必要條件
- **市場波動可能讓停損被觸發**：合理的風險管理比完美的進場更重要
- **耐心等待堆疊位**：多時間框架支撐壓力重疊的區域是最佳交易位置
