# 島型反轉 Island Reversal

> 由左右兩個跳空缺口夾成的孤立K線區塊，預示趨勢強烈反轉，但僅代表小趨勢折返而非大趨勢改變

## 核心概念

島型反轉（Island Reversal）是一種反轉型態，出現在價格的高檔或低檔位置。其特徵是股價先跳空形成一個缺口，經過短期盤整後，再以反向跳空缺口結束盤整，兩個缺口之間的K線群就像被孤立的島嶼，因此得名。

### 型態結構

1. **左側缺口**：股價跳空離開原有趨勢方向（上漲時向上跳空，下跌時向下跳空）
2. **島嶼區間**：在缺口之上（或之下）進行短期盤整，通常1-2週
3. **右側缺口**：股價以反向跳空回到原有趨勢方向，完成反轉

### 兩種型態

| 型態 | 位置 | 含義 | 操作 |
|------|------|------|------|
| **頭部島型** | 上漲趨勢末端 | 多方後繼無力，空方乘機反轉 | 減碼或賣出 |
| **底部島型** | 下跌趨勢末端 | 空方耗盡，多方接管 | 加碼或買進 |

## 實戰應用

### 成因解析

頭部島型的形成邏輯：
1. 投資人看好個股但股價已高，不計價位搶進 → 向上跳空缺口
2. 大部分看好的投資人都已進場，量能萎縮 → 盤整區間
3. 主力與先行者獲利了結，龐大賣壓 → 向下跳空缺口
4. 晚進場的投資人套牢 → 後續下跌趨勢

### 成交量確認

- **左側缺口**：通常伴隨放量，顯示資金急迫進場
- **島嶼區間**：量能萎縮，顯示動能不足
- **右側缺口**：可能放量或量縮，關鍵在缺口是否回補

### 騙線識別

島型反轉是廣為人知的型態，主力可能利用投資人對型態的預期進行騙線：
- **頭部島型完成後不跌反漲**：軋空盲目相信型態的散戶
- **判斷真偽的方法**：
  - 跳空缺口越大且持續放量、積極換手 → 越真實
  - 反向跳空缺口被快速回補 → 騙線訊號
  - 右側缺口未被回補 → 趨勢反轉確認

### 搭配指標

- **成交量**：右側缺口需要帶量確認
- **均線**：跌破 MA20 / MA60 作為輔助確認
- **[[跳空缺口Gap-Theory]]**：島型反轉本質是雙缺口結構

### 實戰案例

**牧德（3563）底部島型反轉 → 頭部島型反轉**：
- 7/25-26 跳空上漲，26日成交量為平日4倍
- 之後進入短期整理，上漲動能消失
- 9/6 跌破MA20、10日跌破MA60
- 9/11 開盤跳空下跌 → 形成島型反轉 → 後續走跌至10月

**聯亞（3081）騙線案例**：
- 10/31 收盤384.5元，11/1 跳空上漲至394元，量達平日10倍
- 整理後11/23跳空下跌至364元，形成頭部島型
- 但後續不跌反漲，最高來到408元 → 典型騙線

**日月光投控（3711）底部島型反轉**：
- 2022年10/11 因美國升息議題跳空下殺逾9%
- 兩週盤整後股價觸底反彈上攻
- 形成兩邊缺口的底部島型反轉 → 股價大漲60.1%

## 注意事項

1. **僅代表小趨勢反轉**：島型反轉形成的時間較短，只能呈現「投資人短暫的過度預期被打回原形」的現象，不代表大趨勢的反轉
2. **騙線風險高**：因為型態廣為人知，主力容易利用投資人預期操作騙線
3. **缺口回補是關鍵**：右側缺口如果被回補，型態即失效，可能只是整理而非反轉
4. **需要量價確認**：單純看型態不夠，必須搭配成交量與其他指標確認
5. **搭配時間框架**：日線島型比小時線更可靠，週線島型反轉意義更大
