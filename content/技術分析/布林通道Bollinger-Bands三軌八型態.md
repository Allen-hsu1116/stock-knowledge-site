# 布林通道 Bollinger Bands 三軌八型態

> 布林通道是均線 ± 2 個標準差構成的動態通道，95.4% 的價格落在通道內；帶寬收窄（Squeeze）是大行情前兆，W底M頭型態確認反轉，8大型態決定操作方向。

## 核心概念

### 三軌定義
- **上軌**：中軌 + 2 倍標準差（壓力線）
- **中軌**：20 日移動平均線（趨勢線）
- **下軌**：中軌 - 2 倍標準差（支撐線）

### 為什麼是2倍標準差？
根據統計學常態分佈理論，約95%的數據會落在平均值正負2個標準差範圍內。因此價格突破上下軌時，是相對極端事件，值得特別關注。

### 八大型態

| 型態 | 特徵 | 意涵 | 操作 |
|---|---|---|---|
| 1. 擴張上 | 帶寬擴大+沿上軌走 | 強勢多頭 | 沿中軌做多 |
| 2. 擴張下 | 帶寬擴大+沿下軌走 | 強勢空頭 | 沿中軌做空 |
| 3. 縮口上 | 帶寬收窄+站中軌上 | 變盤向上 | 突破上軌做多 |
| 4. 縮口下 | 帶寬收窄+站中軌下 | 變盤向下 | 跌破下軌做空 |
| 5. 沿軌上 | 沿上軌行走 | 強勢趨勢 | 中軌做多 |
| 6. 沿軌下 | 沿下軌行走 | 弱勢趨勢 | 中軌做空 |
| 7. 窄幅盤 | 帶寬極窄 | 大變盤前 | 等突破方向 |
| 8. 假突破 | 突破後回到通道內 | 誘多/誘空 | 反向操作 |

### 布林擠壓（Bollinger Squeeze）
- 帶寬收縮到近期最窄 = 變盤前夕（暴風雨前的寧靜）
- 搭配 ADX < 20 確認盤整
- 突破方向搭配成交量確認
- 通道擴張幅度越大 = 趨勢力道越強

## 參數設定進階

| 交易風格 | 建議期間 | 建議標準差 | 適用情境 |
|----------|----------|------------|----------|
| 短線當沖 | 10–15 | 1.8–2.0 | 高頻訊號，快速反應 |
| 中期波段 | 20（標準） | 2.0（標準） | 最普遍設定，一個月交易日 |
| 長期投資 | 50 | 2.5 | 過濾短期雜訊，判斷長期趨勢 |

- **約翰·布林格本人推薦 20,2**：兼顧敏感度與穩定性
- **標準差倍數越小**（1.5倍）：通道越窄，訊號越頻繁，適合盤整區間短線
- **標準差倍數越大**（2.5-3.0倍）：通道越寬，訊號越稀少但極端，適合捕捉重大轉折
- **沒有「最好」的參數，只有「最適合」的參數**：建議回測找出最適合自己策略的組合

## 布林通道軌道型態（OANDA 分類）

除8大型態外，OANDA將布林通道型態分為**同向型態**與**異向型態**：

### 同向型態（三軌同方向）
- **三軌向上**：多頭力量穩健，穩定上漲，回檔較淺
- **三軌向下**：空頭力量穩健，逐步下跌，反彈較弱
- **三軌水平**：狹幅震盪，獲利空間太小，觀望為主

### 異向型態（軌道開合變化）
- **三軌張口**：短期情緒小爆發，可看作趨勢打開訊號
- **三軌收口**：盤面進入盤整，等待新方向出現
- **張口型喇叭口**：情緒急劇爆發+大幅波動，但訊號出現時行情常已走掉大半
- **收口型喇叭口**：極端行情後的修正階段，不建議操作
- **緊口型喇叭口**：水平通道突然收窄，像彈簧瞬間壓縮，後續將爆發但方向難掌握

## 實戰應用

### 逆勢 vs 順勢：兩派用法（市場先生整理）

布林通道的核心假設是「95%的價格會落在2倍標準差通道內」，由此衍生出兩派完全不同的操作邏輯：

| | 逆勢派（均值回歸） | 順勢派（突破追蹤） |
|---|---|---|
| **邏輯** | 價格偏離均值終將回歸 | 突破極端值代表新趨勢發生 |
| **訊號** | 觸上軌賣、觸下軌買 | 突破上軌追多、跌破下軌追空 |
| **適用** | 盤整行情 | 趨勢行情 |
| **參數傾向** | 短週期(20MA)、標準差2倍或更大 | 長週期(50MA以上)、通道偏窄 |
| **統計基礎** | 短期內價格趨向常態分配 | 利用厚尾現象獲利 |

⚠️ **關鍵差異**：逆勢派用短週期是因為短期價格更符合常態分配（震盪隨機），順勢派用長週期是因為長期突破趨勢的區間才有參考價值。**搞混兩派就會虧錢。**

### 基礎策略
1. **均值回歸策略**：觸上軌賣、觸下軌買（僅限盤整盤，搭配K線反轉型態確認）
2. **趨勢追蹤（帶狀行走 Walking the Bands）**：沿上軌/下軌走時，中軌是加碼點；強勢趨勢中價格可能長期貼著軌道走
3. **帶寬指標**：帶寬擴大 = 趨勢啟動，帶寬收縮 = 盤整
4. **中軌支撐壓力**：上升趨勢中中軌是動態支撐；下跌趨勢中中軌是動態壓力；中軌得失是判斷短期趨勢改變的關鍵

### 進階策略

#### W底型態確認底部
1. 價格首次跌破下軌 → 第一個低點
2. 價格反彈至中軌附近但未能站上
3. 價格再次下跌，但第二個低點收在下軌之內，未破前低

→ **「腳在通道外，身在通道內」** = 下跌動能衰竭，潛在買入時機

#### M頭型態警示頭部
1. 價格首次突破上軌 → 第一個高點
2. 價格回檔至中軌附近
3. 價格再次上攻，但第二個高點收在上軌之內，未能再創新高

→ **「頭在通道外，身在通道內」** = 上漲力道減弱，潛在賣出時機

#### 假突破與假跌破
- 價格短暫突破上軌或跌破下軌後，下一根K線迅速回到通道內
- 通常是假突破/假跌破，趨勢往往朝相反方向發展
- **判斷要點**：看後續K線是否迅速回到通道內 + 成交量是否萎縮確認

### 搭配其他指標的黃金組合

| 組合指標 | 組合策略 | 應用優勢 |
|----------|----------|----------|
| RSI | 價格觸及布林下軌+RSI<30超賣 → 買進訊號更可靠；反之亦然 | 確認超買/超賣區間，避免強趨勢中逆勢摸頭探底 |
| MACD | 通道收窄後價格向上突破+MACD黃金交叉 → 強力多頭訊號 | 判斷趨勢方向與動能，為突破訊號提供方向指引 |
| 成交量 | 價格突破布林上下軌必須伴隨成交量放大 → 才是真突破 | 確認突破有效性，無量突破難以持久 |
| ADX | Squeeze時ADX<20確認盤整 → 突破方向更可靠 | 過濾假突破，確認趨勢強度 |
| 肯特納通道 | 布林通道收窄至肯特納通道內 = Squeeze訊號 | 雙通道確認變盤，勝率更高 |

### 多頭訊號與空頭訊號（OANDA）

**多頭訊號：**
- K線自下而上攻擊，穿越下軌線時 → 買進訊號
- K線自下而上攻擊，穿越中軌線時 → 加碼買進訊號
- K線落在中軌與上軌之間，且仍為多頭向上趨勢 → 繼續做多

**空頭訊號：**
- K線落在中軌與上軌之間，並自上而下摜破中軌 → 賣出訊號
- K線落在中軌與下軌之間，且仍為空頭趨勢 → 繼續做空

## 注意事項

- 布林通道是趨勢工具不是轉折工具
- 強勢趨勢可能「貼著上/下軌走很久」，此時逆勢操作是螳臂擋車
- 參數 20,2 是最常用但可調整（短線用 10,1.8；長線用 50,2.5）
- 盤整盤用均值回歸，趨勢盤用趨勢追蹤，搞錯就虧
- **誤把觸碰當訊號**：碰到上軌就放空、碰到下軌就做多是新手最常犯的錯
- **忽略中軌線**：中軌線是多空分水嶺，價格在其上方或下方代表不同趨勢格局
- **單一時間框架侷限**：只看日線容易一葉障目，建議週線判大趨勢、日線找進場點
- **跳空缺口限制**：開盤跳空大漲大跌會使帶寬突然放大，此時通道喪失參考價值（先天限制）
- 沒有任何指標是完美的，布林通道必須與趨勢指標、動能指標、成交量結合使用
