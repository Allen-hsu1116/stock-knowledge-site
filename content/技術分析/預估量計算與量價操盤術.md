# 預估量計算與量價操盤術

> 開盤15分鐘乘上8倍就能預估全天成交量——量比價先行，預估量讓你在收盤前就搶到強勢股。

## 核心概念

### 為什麼要算預估量？

「量比價先行」——成交量是先行指標，當量有異常行為時，代表價格可能出現異常發展。但成交量在收盤後才看得到，那時黃花菜都涼了。

**預估量**就是在盤中即時計算全天可能成交量，搶在強勢股發動前進場卡位。

### 楊雲翔的量價操盤術背景

楊雲翔縱橫股海30+年，經歷：
- 1989年投入股市，23歲賺進百萬
- 1990年台股暴跌1萬點，吐光所有獲利
- 1999年資產達1,000多萬（使用融資槓桿）
- 2000年網路泡沫，融資斷頭，負債300萬
- 破產後領悟：「市場中絕不說謎，就是成交的量和價」
- 花5年研究量價，發展出「特殊均線長短波段策略」+「短線當沖、隔日沖策略」
- 用這兩個策略翻身，賺回5,000萬元
- 2018年開30萬獨立帳戶實驗，2年報酬率560%+

## 實戰應用

### 預估量計算方法

**核心公式：預估量 = 每15分鐘累積量 × 倍數**

台股全日慣性量能呈現**凹洞型態**——開盤量大、中間量縮、尾盤量增。因此倍數隨時間遞減：

| 時間 | 預估成交量倍數 | 說明 |
|------|----------------|------|
| 9:00～9:15 | 約8倍 | 開盤量能最大，倍數最高 |
| 9:15～9:30 | 約6倍 | 逐漸縮減 |
| 9:30～10:00 | 約4倍 | 量能開始降低 |
| 10:00～10:30 | 約3倍 | 中盤量能 |
| 10:30～11:00 | 約2.2倍 | 接近折半 |
| 11:00～12:00 | 約1.8倍 | 午盤低迷 |
| 12:00～12:30 | 約1.5倍 | 略有回升 |
| 12:30～13:00 | 約1.32倍 | 尾盤量能增加 |
| 13:00～13:30 | 約1倍 | 收盤確認 |

**精進方法**：將時間切分為5分鐘，預估量可以更精準。更進階的做法是每分鐘或每個跳動點預估一次。

### 選股操作流程

```
步驟1：開盤15分鐘，計算預估量
       預估量 = 開盤15分鐘累積量 × 8

步驟2：比較預估量 vs 前一日收盤量/5日均量
       預估量 >> 前日量 → 成交量爆增 → 強勢股候選

步驟3：確認價的方向
       價漲 + 量增 → 做多候選
       價跌 + 量增 → 做空候選

步驟4：確認籌碼方向
       法人買超 + 價漲量增 → 做多確認
       法人賣超 + 價跌量增 → 做空確認

步驟5：進場操作
       當沖 or 隔日沖
```

### 實戰案例：玉晶光（2016/9/17）

- 平日交易量約1,300多張
- 開盤15分鐘爆出約1,000張 → 預估量 = 1,000 × 8 = 8,000張
- 實際收盤量約7,200張（預估誤差約11%）
- 當日股價漲停作收
- 關鍵訊號：預估量遠大於前日量，代表有大量資金進場

### 預估量的局限性

| 局限 | 說明 |
|------|------|
| 開盤量可能失真 | 開盤前15分鐘可能受到隔夜消息影響，量能偏大 |
| 凹洞型態不固定 | 不同個股的量能分布不同，8倍是平均值 |
| 需要搭配價格方向 | 預估量只告訴你「有量」，不告訴你方向 |
| 需要搭配籌碼分析 | 大量可能是主力出貨，需要確認法人方向 |

## 注意事項

- **預估量是輔助工具**：必須搭配價格方向和籌碼分析一起使用
- **倍數是參考值**：不同個股的量能分布不同，需要自行觀察調整
- **開盤量大不代表全天量大**：有些個股開盤量大後量能急縮，預估量會偏高
- **凹洞型態的規律**：台股多數個股遵循「開盤量大→中間量縮→尾盤量增」的規律
- **盤中持續修正**：不要只看開盤15分鐘的預估，要持續修正預估量
