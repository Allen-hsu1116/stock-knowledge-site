# 道氏理論 Dow Theory

> 技術分析的鼻祖，用三種趨勢（主要、次級、微波）和六大原則定義市場方向，是所有趨勢判斷的基礎框架。

## 核心概念

道氏理論由查爾斯·道（Charles Dow）在1900年代初期提出，最初發表於《華爾街日報》社論。雖然道氏本人從未將這些想法整理成完整理論，但後人將其觀點歸納成「道氏理論」，是現代技術分析的基石。

### 六大基本原則

1. **市場行為包容消化一切**：所有影響市場的因素（經濟、政治、心理）都已反映在股價指數中，是效率市場假說的雛形

2. **市場存在三種趨勢**：
   - **主要趨勢（Primary Trend）**：持續1年以上的長期趨勢，即牛市或熊市
   - **次級趨勢（Secondary Trend）**：持續3週至3個月的中期修正，方向與主要趨勢相反，折返幅度約為主要趨勢的1/3至2/3
   - **微波（Minor Trend）**：持續數天至3週的短期波動，被視為日常噪音

3. **主要趨勢分三個階段**：
   - **牛市三階段**：累積期 → 參與期 → 分配期
   - **熊市三階段**：分配期 → 恐慌期 → 累積期

4. **兩個指數必須相互確認**：道氏使用工業指數和鐵路指數（現在改為運輸指數），兩者必須同步創新高或新低才能確認趨勢。台股對應：加權指數和櫃買指數是否同步

5. **成交量必須確認趨勢**：
   - 上升趨勢：價漲量增、價跌量縮
   - 下降趨勢：價跌量增、價漲量縮
   - 成交量是輔助確認指標，趨勢方向由價格決定

6. **趨勢持續直到明確反轉**：趨勢有慣性，不會輕易改變方向，順勢而為直到出現明確反轉訊號

### 趨勢的定義

| 趨勢類型 | 定義 |
|---------|------|
| 上升趨勢 | 一系列更高的高點（HH）和更高的低點（HL） |
| 下降趨勢 | 一系列更低的高點（LH）和更低低點（LL） |
| 橫盤趨勢 | 高點和低點沒有明確方向 |

### 主要趨勢的三個階段

**牛市（上升趨勢）**：

1. **累積期（Accumulation）**：利空出盡，聰明錢開始進場，大眾仍悲觀，成交量低
2. **參與期（Public Participation）**：趨勢明朗，大眾開始追價，成交量放大，價格大幅上漲
3. **分配期（Distribution）**：大眾瘋狂搶進，聰明錢開始出貨，成交量可能放大但上漲動能減弱

**熊市（下降趨勢）**：

1. **分配期（Distribution）**：聰明錢開始出貨，大眾仍樂觀
2. **恐慌期（Panic）**：大眾恐慌拋售，成交量急增
3. **累積期（Accumulation）**：悲觀到極點，聰明錢又開始進場

### 趨勢反轉確認

**上升趨勢反轉**：
1. 價格跌破前一個低點（第一個更低低點）
2. 反彈未能突破前一個高點（第一個更低高點）
3. 再次跌破前一個低點確認反轉

**下降趨勢反轉**：
1. 價格突破前一個高點（第一個更高高點）
2. 回檔未跌破前一個低點（第一個更高低點）
3. 再次突破前一個高點確認反轉

## 實戰應用

### 1. 大方向判斷
先判斷主要趨勢方向，順勢操作。牛市做多、熊市做空或空手，不要逆勢而為。

### 2. 多指數確認
台股實戰：觀察加權指數和櫃買指數是否同步。如果加權創新高但櫃買沒有，趨勢可能不健康。

### 3. 成交量驗證
量價配合是趨勢健康的確認條件。價漲量增、價跌量縮 = 上升趨勢健康。價漲量縮 = 上升動能減弱。

### 4. 階段判斷
辨識目前處於累積期、參與期或分配期：
- **累積期**：成交量低，價格波動小，是長線進場點
- **參與期**：成交量放大，趨勢明確，是波段操作黃金期
- **分配期**：成交量可能大但價格上漲乏力，是減碼時機

### 5. 次級趨勢的利用
在主要趨勢方向上，利用次級趨勢的折返進場：
- 牛市中的次級下跌是買點
- 熊市中的次級反彈是空點
- 次級趨勢折返幅度常見50%回檔

### 6. 與其他技術分析搭配
道氏理論判斷大方向，搭配其他指標找進出場點：
- 道氏判斷趨勢 → [[均線判斷]] 找支撐壓力
- 道氏判斷階段 → [[量價關係九種型態高低檔判讀]] 確認量價
- 道氏確認反轉 → [[K線型態]] 找反轉K線

## 注意事項

### 道氏理論的局限

1. **訊號滯後**：等到確認訊號出現時，往往已經錯過一段行情。道氏理論是確認工具，不是預測工具
2. **過於主觀**：趨勢的判斷仍有主觀成分，不同人可能對同一走勢有不同解讀
3. **不提供具體進出場點**：只判斷方向，需要搭配其他指標找操作點
4. **忽略微波**：日常波動被視為噪音，但短線交易者需要關注
5. **兩指數確認可能過慢**：有時一個指數已經給出訊號，但另一個遲遲不確認，錯過最佳進場時機

### 常見誤區

- **過早判斷反轉**：看到一次更低高點就認為趨勢反轉，但可能只是次級趨勢
- **忽略成交量**：只看價格不看量，可能誤判趨勢健康度
- **在微波層級過度交易**：把日常波動當成趨勢變化，頻繁進出
- **混淆次級趨勢和主要趨勢反轉**：牛市中的10-20%回檔可能只是次級趨勢

### 台股特性

- 台股因為漲跌幅限制（10%、15%），次級趨勢的幅度可能比成熟市場更規律
- 法人影響力大，累積期和分配期的成交量特徵更明顯
- 櫃買指數和加權指數的背離是重要的趨勢健康度判斷依據
