# 月線季線均線週期實戰

> 月線（20MA）和季線（60MA）是中期趨勢的關鍵判斷基準，股價與月線季線的關係揭示了不同週期投資者的持倉成本和心理防線。

## 核心概念

### 均線週期與交易日對應

因為每週有 5 個交易日，每月約 20 個交易日，每季約 60 個交易日，所以常用均線週期可分為：

| 分類 | 均線週期 | 名稱 | 投資者涵蓋範圍 |
|------|----------|------|----------------|
| 短期 | 5 | 周線 | 過去一週買入者 |
| 短期 | 10 | 雙周線 | 過去兩週買入者 |
| 中期 | 20 | 月線 | 過去一個月買入者 |
| 中期 | 60 | 季線 | 過去一季買入者 |
| 長期 | 120 | 半年線 | 過去半年買入者 |
| 長期 | 240 | 年線 | 過去一年買入者 |

**核心邏輯**：均線代表的是「過去一段時間買入者的平均成本」，價格在均線之上代表這些投資者賺錢，之下代表虧損。成本線的力量來自投資者的心理防線。

### 計算方式

移動平均線（MA）=（收盤價₁ + 收盤價₂ + ... + 收盤價ₙ）/ N

## 實戰應用

### 策略一：多頭排列選股

當 5日 > 10日 > 20日 > 60日 均線由上而下排列時（多頭排列），代表所有週期投資者都在賺錢，這是強勢股的特徵。

- **多頭排列**：5MA > 10MA > 20MA > 60MA → 多方強勢，偏多操作
- **空頭排列**：5MA < 10MA < 20MA < 60MA → 空方強勢，偏空或觀望

### 策略二：月線回測買點

股價回測 20 日均線（月線）獲得支撐反彈，是中期趨勢仍然健康的確認，可作為加碼或進場點。

- 上升趨勢中，股價回測月線不破 → 均線支撐有效，可加碼
- 回測月線跌破但 3 日內站回 → 假跌破，仍偏多

### 策略三：季線牛熊分界

60 日均線（季線）是中期趨勢的重要分水嶺：

| 訊號 | 意涵 |
|------|------|
| 股價站上 60MA | 中期偏多 |
| 股價跌破 60MA | 中期偏空 |
| 60MA 上揚 | 中期趨勢向好 |
| 60MA 下彎 | 中期趨勢向壞 |

季線的斜率方向比價格是否站上更重要：季線上揚時即使價格短暫跌破，也多半是修正而非反轉。

### 策略四：支撐壓力互換

當股價跌破月線後反彈，月線從支撐變為壓力。反之，股價突破月線後回測，月線從壓力變為支撐。

這個「角色互換」現象的心理學基礎是：在月線附近買入的投資者，當價格跌破後被套牢，回到成本線時傾向賣出（解套賣壓），形成壓力。

### 價格與均線關係速判

| 位置 | 意涵 |
|------|------|
| 股價在所有均線之上 | 多方強勢，所有週期買入者都賺錢 |
| 股價在所有均線之下 | 空方強勢，所有週期買入者都虧損，可能有踩踏風險 |
| 股價在短期均線之上、長期均線之下 | 短期反彈但中期仍弱，需確認是否反轉 |

### 黃金交叉與死亡交叉

- **黃金交叉**：短週期均線向上穿過長週期均線 → 短期買方力量強大，上升趨勢訊號
- **死亡交叉**：短週期均線向下穿過長週期均線 → 短期賣方力量強大，下降趨勢訊號

月線與季線的黃金交叉/死亡交叉特別重要，因為影響的是中線投資者的成本結構。

## 注意事項

- **均線是落後指標** — 信號出現時趨勢可能已經走了一段，月線季線尤為明顯
- **月線季線適合中長期投資人** — 短線交易者參考 5 日、10 日即可
- **盤整市假訊號多** — 均線在盤整行情中頻繁交叉，需搭配 ADX 或布林通道寬度過濾
- **均線斜率比價格穿越更重要** — 上揚的均線代表趨勢健康，不論價格是否短暫跌破
- **多條均線綜合判斷** — 只看一條均線容易誤判，多條均線的排列和斜率才能看清全貌
