# 威廉碎形 Williams Fractal

> 用5根K線辨識局部高低點的極簡指標——向上碎形突破做多、向下碎形跌破做空，搭配鱷魚線過濾假訊號是最高勝率組合

## 核心概念

威廉碎形（Williams Fractal）由 Bill Williams 開發，是鱷魚線交易系統的核心進出場訊號指標。碎形用來標記市場的**局部極值點**（local extremes），也就是短期高點和短期低點。

### 碎形的定義

**向上碎形（上分形/賣出碎形）**：
- 中間K線的最高價 > 左邊2根K線的最高價 且 > 右邊2根K線的最高價
- 即連續5根K線中，中間那根的最高價最高

**向下碎形（下分形/買入碎形）**：
- 中間K線的最低價 < 左邊2根K線的最低價 且 < 右邊2根K線的最低價
- 即連續5根K線中，中間那根的最低價最低

數學表示：
```
向上碎形(t) = 1 if Ht > H(t+i), i = -2,-1,1,2
向下碎形(t) = 1 if Lt < L(t+i), i = -2,-1,1,2
```

### 有效碎形 vs 無效碎形

**這是碎形實戰中最關鍵的概念**：

- **有效碎形**：向上碎形高於紅色齒線（Teeth），或向下碎形低於紅色齒線
- **無效碎形**：碎形出現在齒線與唇線之間或齒線下方（多頭時）

**無效碎形變有效碎形**：原本無效的碎形，經過幾天紅色齒線下降/上升後，碎形超過了新位置的齒線，就變成有效碎形。Bill Williams 特別強調：這個機制很重要，因為齒線是移動的，一個原本在齒線上方的向上碎形可能因為齒線上升而變成無效，反之亦然。

**重要規則**：採用的碎形應以**最新出現的碎形**為準，新碎形出現後之前的碎形不可再用。這表示碎形訊號有時效性，舊的碎形一旦被新的覆蓋就失去意義。

## 實戰應用

### 碎形作為進場訊號

**做多**：
1. 鱷魚線多頭排列（綠>紅>藍）
2. 價格突破最近的**有效向上碎形**高點
3. 收盤價必須高於綠線（Lips）
4. 成交量≥前一日1倍

**做空**：
1. 鱷魚線空頭排列（藍>紅>綠）
2. 價格跌破最近的**有效向下碎形**低點
3. 收盤價必須低於綠線（Lips）
4. 成交量配合

### 碎形+鱷魚線組合策略

這是 Bill Williams 混沌交易法的核心：
- **鱷魚線判方向**：三線排列決定多空
- **碎形判時機**：突破有效碎形決定進出場
- **兩者缺一不可**：只有碎形沒有方向=假突破，只有方向沒有碎形=進場點不明確

組合勝率：
- 單獨碎形：約55%-60%
- 碎形+鱷魚線：約68%-72%
- 碎形+鱷魚線+[[Gator-Oscillator蓋特震盪指標|Gator Oscillator]]：約78%-85%

### 停損設定

- 多單停損：向下碎形低點下方1-2點
- 空單停損：向上碎形高點上方1-2點
- 目標價：參考前一個碎形區間（價格擺動幅度）

### 多時間框架碎形

- 週線碎形：主要趨勢方向
- 日線碎形：波段進出場
- 4H碎形：短線確認

大週期碎形比小週期碎形更可靠，建議用大週期碎形判方向、小週期碎形找進場。

## 注意事項

### 碎形的五大陷阱

1. **盤整期碎形密集**：三線纏繞時碎形頻繁出現但大多無效，容易被洗
2. **無效碎形誤判**：在齒線附近的碎形容易被突破又回跌，必須嚴格判斷有效性
3. **滯後2根K線**：碎形需要右邊2根K線確認，訊號出現時已經晚了2根K線
4. **假突破碎形**：突破碎形後立刻回落，尤其在震盪市中
5. **過度依賴碎形**：碎形只是局部極值標記，不具預測功能，必須搭配趨勢指標

### 使用建議

- **1H以下碎形太密**：過濾效率差，建議≥4H或日線
- **碎形必須搭配鱷魚線**：單獨使用勝率不夠高
- **新碎形覆蓋舊碎形**：永遠只看最新的有效碎形
- **用週線碎形定方向**：避免被日內雜訊干擾
- **等待右邊2根K線確認**：不要提前預判碎形形成
