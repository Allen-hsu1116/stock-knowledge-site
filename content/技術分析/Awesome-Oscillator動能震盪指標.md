# Awesome Oscillator動能震盪指標

> Bill Williams的動能溫度計——SMA(5)減SMA(34)用中位數算，零軸上多下空、碟形看續航、雙峰抓反轉，但固定參數是雙面刃

## 核心概念

Awesome Oscillator（AO）由Bill Williams開發，是其混沌交易系統的核心指標之一。AO計算5期簡單移動平均與34期簡單移動平均的差值，但與MACD不同的是：**AO使用價格中位數（High+Low）/2**，而非收盤價。

**公式**：AO = SMA(Median, 5) – SMA(Median, 34)

AO以柱狀圖呈現，綠色代表比前一根高、紅色代表比前一根低。沒有固定刻度，沒有超買超賣區間——這是AO與RSI/KD的本質差異。

### Bill Williams的設計哲學

5期和34期來自斐波那契數列。5是動能的短期感知器，34是趨勢的中期方向儀。AO的本質是：**短期動能相對中期趨勢的偏離程度**。

## 實戰應用

### 1. 零軸交叉（最基本但最不可靠）

- AO從下方穿越零軸 → 買入訊號（5期動能開始超越34期）
- AO從上方穿越零軸 → 賣出訊號

**警告**：這是AO三種訊號中最不可靠的！單獨使用會產生大量假進場。必須搭配其他過濾工具。

### 2. 碟形訊號（Saucer）——看續航

- **看漲碟形**：AO在零軸上方，連續三根柱狀圖中間低兩邊高 → 動能暫歇後再起，續漲訊號
- **看跌碟形（Inverted Saucer）**：AO在零軸下方，連續三根柱狀圖中間高兩邊低 → 續跌訊號

碟形訊號本質是動能減速後重新加速，不是反轉而是延續。

### 3. 雙峰訊號（Twin Peaks）——抓反轉

- **看漲雙峰**：AO在零軸下方出現兩個低點，第二個比第一個高，且兩者之間未穿越零軸 → 空方動能減弱，買入訊號
- **看跌雙峰**：AO在零軸上方出現兩個高點，第二個比第一個低，且兩者之間未穿越零軸 → 多方動能減弱，賣出訊號

雙峰訊號本質是動能峰值遞減，暗示趨勢即將反轉。

### 4. 背離訊號——最強反轉訊號

- 價格創新高但AO沒有 → 看跌背離
- 價格創新低但AO沒有 → 看漲背離

在重要支撐壓力位出現背離時尤其有效，建議搭配成交量指標確認。

### 5. AO + SAR組合策略

- 買入：SAR在價格下方 + AO在零軸上方且最後一根綠色
- 賣出：SAR在價格上方 + AO從上方穿越零軸
- 停損設在SAR標記價位，出場在SAR換邊時

### 6. AO + Alligator搭配

Alligator（鱷魚指標）判趨勢方向，AO在趨勢確認後找進場點位，這是Bill Williams的原始設計用法。

## 注意事項

1. **固定參數不能改**：5期和34期是寫死的，這意味著你無法依市況調整。高波動市場可能需要更長期間，低波動可能需要更短，但AO不讓你改
2. **短時間框架雜訊多**：低於M30的時間框架假訊號暴增，建議H1以上使用
3. **盤整市無用**：AO在盤整中頻繁穿越零軸，產生大量假訊號，搭配ADX或Alligator過濾
4. **零軸交叉最弱**：三種訊號中零軸交叉最不可靠，碟形和雙峰的勝率較高
5. **碟形訊號陷阱**：在不活躍市場中，碟形可能只是小幅波動，不是真正的續航訊號
6. **不做超買超賣判斷**：AO沒有固定區間，不能告訴你市場是否過熱或過冷
7. **背離需量確認**：背離訊號搭配成交量驗證才更可靠
