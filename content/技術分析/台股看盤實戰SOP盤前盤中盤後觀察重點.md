# 台股看盤實戰SOP盤前盤中盤後觀察重點

> 從盤前國際股市、期貨開盤，到盤中熱力圖與騰落線，再到盤後回顧，建立系統化的看盤流程，讓你在開盤前就做好準備，盤中不被情緒干擾。

## 核心概念

### 看盤的本質：規劃而非預測

看盤的目的不是預測漲跌，而是為所有可能性做好準備。就像規劃旅行一樣，再怎麼準備都會有突發狀況，但有了規劃就能在第一時間做出正確判斷，避免被市場情緒干擾而做出不理智的選擇。

條件式思考：「如果 A 條件成立，我就怎麼做；如果 A 條件不成立，我就用另外一種做法。」這比單純看好看壞更務實。

### 看盤三階段

1. **盤前**（08:00-09:00）：國際消息 → 亞股走勢 → 期貨開盤 → 摩台指
2. **盤中**（09:00-13:30）：大盤指數 → 騰落線 → 熱力圖 → 領漲領跌類股
3. **盤後**（13:30 之後）：回顧驗證 → 調整隔日計畫

## 實戰應用

### 一、盤前準備（08:00-09:00）

#### 1. 國際股市連動觀察

**必看指標（按重要性排序）**：
- **費城半導體**：對台股短線影響最大
- **韓國股市**：比日股對台股影響更大，韓股多為電子類股與台股相似
- **日經指數**：亞洲最早開盤（07:30），觀察國際資金動向
- **台積電 ADR**：漲跌 3% 以上對隔天台股有影響；但若早盤已大漲、ADR 跟漲，則隔天反應有限

**連動判斷邏輯**：
- 台股盤整 + 美股突破 → 台股可能補漲
- 台股撐在月線 + 美股跌破季線 → 小心台股追跌
- 台積電早盤沒動 + ADR 晚間大漲大跌 → 隔天台股會反應
- 台積電早盤已大漲 + ADR 跟漲 → 隔天影響有限（已反應）

**看盤必備自選股清單**：
- 小那斯達克（NQ）期貨：與台股連動最大的指數期貨，交易時間長達 23 小時
- 黃金：避險工具，黃金異常漲價代表大戶開始避險
- 日幣：避險貨幣之一，日幣異常升值代表避險資金流入
- 韓國股市（08:00 開盤）
- 台積電 ADR
- 中國 A50 期貨（盤中觀察）

#### 2. 日韓股市開盤（08:00）

- 日經指數與韓股綜合在早上 8 點開盤
- 日韓股市率先消化國際消息，可從中推估台股開盤方向
- 特別注意：韓國股市指數距離平盤的空間，與台股有很大關聯

#### 3. 台指期貨開盤（08:45）

比大盤早 15 分鐘，搶先展開多空拉鋸。**不要貿然進場搶先卡位**，而是觀察當天力道：

**觀察重點**：
- **委買委賣掛單**：大行情時掛單會非常積極
- **口差（委託買賣差距）**：開盤前口差達 2000 以上，偏多趨勢盤機率大幅增加
- **摩台指走勢**：摩台指在新加坡交易所交易，無漲跌幅限制，參與者多為外資法人大戶，是國際資金對台股態度的領先指標

**摩台指判讀**：
- 摩台指開平盤上 → 往下探 → 法人偏空 → 台股可能跟跌
- 摩台指開平盤下 → 往上衝 → 法人偏多 → 台股可能跟漲
- 摩台指與加權指數走勢對照：若摩台指先跌、加權指數卻開高，可預期加權指數將跟跌

### 二、盤中觀察（09:00-13:30）

#### 1. 大盤指數 vs 扣除台積電指數

| 觀察對象 | 適用情境 | 觀察目的 |
|----------|----------|----------|
| 加權指數 | 操作期貨、選擇權 | 知道大多數投資人的看法 |
| 扣除台積電指數 | 操作個股為主 | 看到更真實的非台積電市場強弱 |

用技術分析抓出支撐壓力與趨勢，初步定調多空，這是第一步也是最重要的。

#### 2. 騰落線（Advance-Decline Line）

**重要性**：騰落線是中小型股的領先指標。

**多轉空訊號**：
- 指數還撐著 → 騰落線已下彎 → 多數股票已轉弱 → **多轉空前兆**
- 指數創新高 → 騰落線掛掉 → 大盤沒幾天也跟著掛

**空轉多訊號**：
- 大盤還在跌 → 騰落線已不破底 → 可能外資賣台積電壓低指數 → **準備反彈**

#### 3. 熱力圖（Heatmap）

**三大參數**：
- **顏色**：紅漲綠跌（台股），顏色深淺反映漲跌幅度
- **亮度**：漲跌幅越大顏色越深（漲停＝最深紅，跌停＝最深綠）
- **面積**：依市值或成交金額排列，面積越大對市場影響越大

**兩種排序方式**：

| 排序方式 | 用途 | 適合誰 |
|----------|------|--------|
| 市值排序 | 看大盤漲跌由哪些個股貢獻 | 波段投資人 |
| 成交金額排序 | 找出資金青睞的強勢族群 | 當沖投資人 |

**不同時段的熱力圖用法**：
- **盤前**：看前一天成交額排序 + 國際財經局勢 + 法人買賣超
- **開盤 10 分鐘**：掃一次熱力圖，看有無主流族群進駐
- **盤中**：留意資金流向是否轉變，某族群早盤冷清、午盤灌資金 = 可能有突發題材
- **收盤**：分析資金結構 + K線 + 量能變化

**熱力圖判讀族群行情**：
- 某類股一片綠 + 零星一兩檔紅 → 個股利多題材
- 某類股整片紅 → 族群行情，留意延續性

#### 4. 近期領漲領跌類股

把近期最熱的題材相關股票抓出來看：
- 領漲股開始走弱 → 原本趨勢準備改變
- 領漲股持續強勢 → 趨勢延續
- 可以細分產業鏈（如 AI：SERVER、散熱、設備、CoWoS）

### 三、盤後回顧

1. **回顧盤前規劃 vs 實際走勢**：預測準不準？哪裡判斷錯了？
2. **熱力圖盤後分析**：當日資金結構、帶量上漲但未成焦點的個股
3. **法人買賣超**：外資、投信、自營商動向
4. **調整隔日計畫**：根據今天的觀察修正明天的條件式規劃

### 四、期貨大額交易人 & 外資留倉（參考用）

- 可偶爾參考，但**不要只看這個數據做判斷**
- 近年參考性下降：大額交易人空單被軋、外資多單但指數下跌的例外情況越來越多
- 需搭配其他數據綜合判斷

## 注意事項

### 熱力圖常見陷阱

1. **看到紅色別急著追漲**：可能是「拉積盤」，台積電獨撐大局，中小型股反而量能低迷
2. **格子大 ≠ 漲得多**：面積是按市值或成交金額排列，跟漲跌幅無關
3. **流動性陷阱**：漲幅大的小型股可能成交量很低，進得去出不來
4. **盤後熱力圖是靜態的**：無法反映收盤後的基本面、消息面變化

### 看盤常見錯誤

1. **只看加權指數**：忽略扣除台積電的真實市場強弱
2. **只看期貨留倉**：近年參考性降低，需要搭配其他指標
3. **忽略騰落線**：指數還在漲但騰落線下彎是多轉空前兆
4. **盤前不做規劃**：開盤後才隨波逐流，容易被情緒帶著走
5. **只看好消息**：規劃時必須同時準備上漲和下跌的應對方案

### 條件式交易規劃

看盤的最終目標不是變成預言家，而是建立「如果...就...」的條件式規劃：
- 如果多方條件都成立 → 進場做多
- 如果多方條件不成立 → 不進場或反向操作
- 如果盤勢發展不如預期 → 已經有退場方案
