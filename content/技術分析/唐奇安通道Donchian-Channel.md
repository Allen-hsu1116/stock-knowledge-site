# 唐奇安通道 Donchian Channel

> 海龜交易法的核心指標，用N日最高最低價畫出通道，突破上軌做多、跌破下軌做空，簡單到只剩兩個函式但勝率偏低，盤整市場假突破是最大敵人

## 核心概念

唐奇安通道由理查·唐奇安（Richard Donchian）創造，是海龜交易法的核心基礎指標。它用最近N根K線的最高價和最低價畫出一條通道，本質上是一個最純粹的趨勢追蹤工具。

### 計算公式

- **上通道** = 最近 N 日最高價（Highest High）
- **下通道** = 最近 N 日最低價（Lowest Low）
- **中間線** = (上通道 + 下通道) / 2

預設 N = 20，與海龜交易法的20日突破系統一致。

### 核心邏輯

多頭的本質就是「一高還有一高高」，空頭就是「一低還有一低低」。唐奇安通道把這個邏輯量化成兩個函式：
- `Highest(High, N)`：取最近N根K棒最高點
- `Lowest(Low, N)`：取最近N根K棒最低點

這就是所謂的 **HHLL 系統**——簡單到只有兩行，但背後是完整的趨勢追蹤哲學。

## 實戰應用

### 1. 趨勢判斷

| 價格位置 | 判斷 | 動作 |
|---------|------|------|
| 價格在中間線之上震盪，突破上通道 | 上升趨勢確立 | 考慮做多 |
| 價格在中間線之下震盪，跌破下通道 | 下降趨勢確立 | 考慮做空 |
| 價格在通道內來回震盪 | 盤整格局 | 不進場 |

### 2. 突破交易策略

- **做多訊號**：收盤價突破上通道（N日最高價）→ 買進
- **做空訊號**：收盤價跌破下通道（N日最低價）→ 賣出
- **停損**：進場後跌破下通道停損（做多）；突破上通道停損（做空）
- **停利**：可用中間線作為移動停利參考

### 3. 海龜交易法的應用

海龜交易法在唐奇安通道基礎上加入：
- **雙重時間框架**：20日突破做進場、10日突破做出場
- **ATR 停損**：配合ATR設定動態停損
- **金字塔加碼**：每上漲0.5倍ATR加碼一單位
- **部位控制**：單一市場風險不超過帳戶2%

### 4. 烏龜湯策略（Turtle Soup）

坊間有反過來操作的策略——**烏龜湯策略**：
- 當價格突破N日高點後，多數交易者會跟進做多
- 烏龜湯策略反而在此時做空，賭突破是假突破
- 適合盤整市中的假突破操作

### 5. 參數設定

| 參數 | 預設值 | 用途 |
|------|--------|------|
| N（上軌） | 20 | 計算N日最高價 |
| N（下軌） | 20 | 計算N日最低價 |
| 偏移 | 1 | 向前偏移1根K線 |

- **短週期（5-10日）**：更靈敏，假突破更多
- **長週期（20-55日）**：更穩定，海龜法用20/55日
- **建議**：短週期搭配長週期MA確認大方向

### 6. 搭配其他指標

| 搭配指標 | 用途 |
|---------|------|
| 移動平均線 MA | 確認趨勢方向，價格突破上通道+在MA之上＝強勢做多 |
| RSI | 確認超買超賣，減少假突破訊號 |
| ADX | 過濾盤整，ADX>25趨勢明確才用唐奇安通道 |
| ATR | 動態停損停利，配合海龜加碼邏輯 |
| 成交量 | 突破時必須帶量確認，否則可能是假突破 |

## 注意事項

### 最大缺陷：盤整市假突破

唐奇安通道在盤整市場中會產生大量假突破訊號，這是它最大的致命傷。因為通道只看最高最低價，盤整時價格在通道邊緣來回穿越，每次穿越都是一個假訊號。

**解法**：
1. **搭配 ADX 過濾**：ADX < 20 盤整市不進場
2. **帶量突破確認**：突破時成交量要大於20日均量
3. **收盤價確認**：用收盤價而非盤中價判斷突破
4. **大週期過濾**：短週期通道進場前，確認長週期趨勢方向一致

### 勝率偏低的真相

純唐奇安通道突破策略的勝率通常只有30-40%，但靠的是**高報酬比**（趨勢行情獲利遠大於盤整虧損）。Van K. Tharp 建議改善方式：在突破訊號前加入**價格序列進場架構**（如波動率收縮後突破、相對強勢等）。

### 其他注意事項

- **缺乏方向性確認**：通道只告訴你突破，不告訴你趨勢強度
- **只考慮極值**：只用最高最低價，忽略收盤價分布和成交量
- **訊號延遲**：作為趨勢追蹤指標，進場永遠不是最低點
- **參數敏感度**：不同參數在不同市場表現差異很大，需要針對標的調整
