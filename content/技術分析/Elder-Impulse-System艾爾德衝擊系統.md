# Elder Impulse System 艾爾德衝擊系統

> 用一條均線看趨勢、一個柱狀圖看動量，三色K棒讓你一眼判斷進場謹慎出場快速

## 核心概念

Elder Impulse System 由 Alexander Elder 設計，收錄於《Come Into My Trading Room》。核心思想：**辨識趨勢加速或減速的拐點**。系統結合兩個指標：

1. **13 日 EMA**：判斷趨勢方向
2. **MACD 柱狀圖**（12,26,9）：判斷動量方向

兩者同向上 → 衝擊力道強；兩者不同向 → 警訊。

### 三色K棒規則

| 顏色 | 條件 | 意義 |
|------|------|------|
| 🟢 綠色K棒 | EMA(13) 上升 **且** MACD-Histogram 上升 | 多方完全掌控——趨勢+動量同步向上 |
| 🔴 紅色K棒 | EMA(13) 下降 **且** MACD-Histogram 下降 | 空方完全掌控——趨勢+動量同步向下 |
| 🔵 藍色K棒 | 條件不符綠色或紅色 | 多空訊號混亂——不進場 |

### 計算邏輯

```
綠色K棒: EMA(13) > 前一期EMA(13)  AND  MACD-Histogram > 前一期MACD-Histogram
紅色K棒: EMA(13) < 前一期EMA(13)  AND  MACD-Histogram < 前一期MACD-Histogram
藍色K棒: 以上條件均不滿足
```

## 實戰應用

### 1. 多時間框架操作（核心精髓）

Elder 建議的交易時間框架體系：

- **交易時間框架** = 中期（例如日線）
- **長期時間框架** = 中期 × 5（例如週線）
- **長期趨勢判斷**：價格在 65 日 EMA 上方，或 MACD(1,65,1) > 0

**規則**：只在長期趨勢方向的訊號才有效！

- 週線上升 → 只接受日線綠色K棒訊號（做多）
- 週線下降 → 只接受日線紅色K棒訊號（做空）
- 逆趨勢方向的訊號 → 忽略

### 2. 進場策略

**做多**：
1. 確認長期趨勢向上（MACD(1,65,1) > 0 或價格 > 65 EMA）
2. 等待日線出現連續綠色K棒群
3. 第一根綠色K棒出現時進場

**做空**：
1. 確認長期趨勢向下
2. 等待日線出現連續紅色K棒群
3. 第一根紅色K棒出現時進場

### 3. 出場策略

Elder 名言：**「進場謹慎、出場快速」**（Enter cautiously, exit fast）

- 綠色K棒轉為藍色或紅色 → 立即考慮出場
- 不等第二根紅色K棒——動量減速就是出場訊號

### 4. 過濾假訊號

- **做多過濾**：Impulse System 不是全綠 → 不做多
- **做空過濾**：Impulse System 不是全紅 → 不做空
- 用 Impulse System 確認其他技術型態（旗形、支撐壓力、斐波那契等）

### 5. 預判反轉

- 趨勢進行中出現藍色K棒 → 動量開始減速
- 連續藍色K棒 → 反轉可能性升高
- 搭配價格型態（如旗形收斂末端）提高判斷準確度

## 注意事項

- **短期工具**：Impulse System 設計用來捕捉相對短期的價格波動，不適合長期持有策略
- **多時間框架必須遵守**：逆大趨勢的訊號是假訊號的主要來源
- **藍色K棒不代表趨勢結束**：可能只是暫時減速，需搭配價格行為確認
- **MACD 參數固定**：預設 MACD(12,26,9) 不建議隨意更改
- **進場慢出場快**：這是專業交易的精髓——業餘者進場快出場慢，專業者恰好相反
