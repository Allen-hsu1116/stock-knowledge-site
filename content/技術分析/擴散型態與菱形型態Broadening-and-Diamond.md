# 擴散型態與菱形型態Broadening-and-Diamond

> 擴散型態（喇叭口）反映多空分歧加劇的頂部反轉訊號；菱形型態由擴散+收斂組合而成，是罕見但信號強烈的反轉型態。

## 核心概念

### 擴散型態（Broadening Formation / Megaphone）

擴散型態又稱喇叭口、擴大形。與收斂三角形相反，價格波動範圍逐漸擴大——高點越來越高、低點越來越低，上下趨勢線呈發散狀態。

**市場心理**：多空雙方力量激烈對抗，分歧不斷加大，市場不確定性升高，通常是趨勢即將反轉的前兆。

### 菱形型態（Diamond Pattern）

菱形由兩部分組成：**前半段擴散**（高點越來越高、低點越來越低）+ **後半段收斂**（波動逐漸縮小），整體形成菱形外觀。相對罕見但信號可靠性高。

| 型態 | 位置 | 方向 | 可靠性 |
|------|------|------|--------|
| 擴散頂 | 上升趨勢末期 | 看跌 | 中等 |
| 直角擴散三角形 | 頂部/底部 | 看跌/看漲 | 中等 |
| 菱形頂 | 長期上升頂部 | 看跌反轉 | 高 |
| 菱形底 | 長期下降底部 | 看漲反轉 | 高 |
| 菱形延續 | 趨勢中段 | 延續原趨勢 | 中等 |

## 實戰應用

### 擴散型態操作

1. **識別**：至少2個上升高點+2個下降低點，趨勢線發散
2. **確認**：成交量隨波動擴大而增加
3. **進場**：價格跌破下方趨勢線時做空/賣出
4. **停損**：設在最近高點上方
5. **目標**：擴散形態的最大垂直高度

### 菱形頂操作

1. **識別**：前半段擴散（2~3個高點低點）→ 後半段收斂
2. **確認**：右半部收斂至少2次觸及邊緣後才突破
3. **進場**：跌破菱形下邊界時賣出/做空
4. **停損**：設在菱形最高點上方
5. **目標**：突破點向下延伸菱形最大高度

### 菱形底操作

1. **識別**：長期下跌後出現擴散→收斂的菱形
2. **確認**：帶量突破上邊界
3. **進場**：突破後買入
4. **停損**：設在菱形最低點下方
5. **目標**：突破點向上延伸菱形總高度

### 成交量判讀

| 階段 | 擴散型態 | 菱形型態 |
|------|----------|----------|
| 形成初期 | 量增（分歧大） | 量增（分歧大） |
| 中段 | 量持續放大 | 量逐步萎縮 |
| 突破時 | 量能不一定 | **必須放量** |

## 注意事項

1. **擴散型態假突破頻繁**：需等待完整型態+突破確認，不要過早進場
2. **菱形罕見但可靠**：出現時要特別重視，統計上反轉成功率較高
3. **菱形右半部必須收斂**：如果只有擴散沒有收斂，不是菱形
4. **時間框架影響**：日線以上的菱形可靠性高於小時間框架
5. **搭配指標**：RSI背離、MACD死叉等可輔助確認
6. **擴散型態很少出現在底部**：底部通常對應收斂（共識形成），擴散代表分歧
