# 分價量表判讀

> 透過價量分佈辨識支撐壓力、主力成本區和出貨訊號

## 核心概念

分價量表（Price Level Volume Table）是將一段時間內每個成交價位的成交量、成交筆數列出，以直觀方式呈現價量分佈的工具。它揭示了K線圖看不到的資訊——**量在哪個價位**，而不只是**總量多少**。

分價量表的核心價值在於：它把「時間序列」的量價關係轉換為「價格分佈」的量價關係，讓你直接看到籌碼在哪裡堆積。

## 實戰應用

### 分價量表的核心欄位

| 欄位 | 說明 | 實戰意義 |
|------|------|----------|
| 價格 | 每個成交價位 | 支撐壓力定位 |
| 成交量 | 該價位的總成交量 | 籌碼堆積量 |
| 成交筆數 | 該價位的成交筆數 | 交易頻率 |
| 均量/筆 | 成交量 ÷ 成交筆數 | 大單vs小單判斷 |
| 佔比 | 該價位成交量佔比 | 關鍵價位辨識 |

### 均量/筆：大單辨識的關鍵

均量/筆 = 成交量 ÷ 成交筆數，是辨識大單進場的核心指標：

| 均量/筆 | 解讀 | 可能含義 |
|---------|------|----------|
| 偏高（>市場平均2倍） | 大單進場 | 主力或法人 |
| 接近平均 | 正常交易 | 散戶為主 |
| 偏低 | 碎單交易 | 散戶追價或停損 |

**注意**：主力拆單會讓均量/筆不明顯，需搭配 [[分點進出追蹤關鍵分點]] 交叉驗證。

### 五大實戰判讀

**1. 支撐壓力辨識**
- 大量區 = 支撐/壓力區：分價量表中成交量最大的價位
- 上方大量區 = 壓力帶（套牢區）
- 下方大量區 = 支撐帶（成本區）
- 關鍵：股價穿越大量區的方向決定是支撐還是壓力
- 搭配 [[支撐壓力]] 使用

**2. 主力成本區判斷**
- 主力進場價位通常伴隨大筆成交（均量/筆數大）
- 底部區間的量能密集帶 + 均量/筆偏高 = 主力成本區
- 主力成本區是未來支撐的重要參考
- 搭配 [[主力勝率與區間均價交叉驗證]] 估算主力成本

**3. 出貨訊號辨識**
- 高價區出現大量、大筆成交 → 主力可能在高位派發
- 高價區出現大量、但均量/筆小 → 散戶追價、主力拆單出貨
- 對比：低價區大單 vs 高價區散單 → 典型的主力出貨型態
- 搭配 [[主力出貨辨識]] 綜合判斷

**4. 突破確認**
- 股價突破壓力帶時，看壓力帶的量是否被消化
- 突破後回測壓力帶（壓力轉支撐）：
  - 回測量縮 → 量能已消化 → 真突破
  - 回測帶量 → 量能未消化 → 可能假突破
- 搭配 [[真突破確認三原則帶量收盤3天]] 驗證

**5. 區間換手判讀**
- 盤整期間分價量表呈現均勻分佈 → 健康換手
- 盤整期間集中在單一價位 → 可能有主力控盤
- 集中在上方 → 壓力重
- 集中在下方 → 支撐強
- 搭配 [[成交量確認原則Volume-Confirmation]] 判斷換手品質

### 分價量表 + 籌碼面交叉驗證

| 分價量表訊號 | 籌碼面驗證 | 結論 |
|-------------|-----------|------|
| 底部大單進場 | 外資/投信連續買超 | 主力進場確認 ✅ |
| 高價區散單追價 | 法人買超減少或翻賣 | 主力出貨警告 ⚠️ |
| 突破帶量消化壓力 | 千張大戶增加 | 真突破確認 ✅ |
| 突破量縮未消化 | 融資增加、大戶減少 | 假突破風險 ❌ |

### 分價量表與 Market Profile 的關聯

分價量表是 [[Market-Profile市場輪廓TPO]] 的基礎數據：
- Market Profile 將分價量按時間分佈呈現（TPO圖）
- 分價量表是簡化版，只看量的分佈不看時間
- 兩者結合使用效果更佳：分價量表看「多少量」，Market Profile 看「何時成交」

### 不同時間框架的分價量表

| 時間框架 | 用途 | 注意事項 |
|---------|------|----------|
| 日分價量表 | 日內交易、當沖 | 只看當天，需連續觀察 |
| 週分價量表 | 波段交易 | 覆蓋一週的籌碼分佈 |
| 月分價量表 | 中長期趨勢 | 最重要的支撐壓力參考 |
| 季分價量表 | 長期成本區 | 主力長期成本參考 |

## 注意事項

1. **時間框架選擇**：日分價量表只看當天，波段交易者應看週或月分價量表
2. **除權息調整**：除權息後的價格需要調整，否則會出現虛假的大量區
3. **流動性限制**：成交量太小時分價量表意義有限，容易被單一大單扭曲
4. **主力拆單**：主力可以將大單拆成多筆小單，讓均量/筆不明顯，需搭配分點追蹤
5. **不要單獨使用**：分價量表要搭配K線、均線、籌碼面一起判斷
6. **高價股稀疏**：高價股成交筆數少，分價量表容易出現斷層，解讀需謹慎
