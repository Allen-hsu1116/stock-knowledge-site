# ATR平均真實波幅 Average True Range

> 衡量市場波動程度的無方向性指標，波動大停損大、波動小停損小，是動態停損停利和部位控制的基礎工具

## 核心概念

ATR（Average True Range，平均真實波幅）由技術分析大師 J. Welles Wilder Jr. 於1978年在《New Concepts in Technical Trading Systems》中提出，是最重要的波動性指標之一。

**關鍵特徵**：
- **無方向性**：ATR 只衡量波動大小，不判斷多空方向
- **真實波幅**：考慮了跳空缺口，比單純高低價差更準確
- **絕對數值**：ATR 是價格單位（如台股的「點」或「元」），不是百分比

### True Range（TR）計算

TR 取以下三者的最大值：
1. 當日最高價 − 當日最低價（正常日內波動）
2. |當日最高價 − 前日收盤價|（向上跳空）
3. |當日最低價 − 前日收盤價|（向下跳空）

### ATR 計算公式

```
ATR = [(前一日ATR × (n-1)) + 當日TR] / n
```

預設參數 n=14，即過去14日的平均真實波幅。ATR 使用指數移動平均（EMA式）而非簡單移動平均（SMA）來計算。

### 參數設定

- **標準參數**：14日（Wilder 原始設定）
- **短線交易**：可縮短至7-10日，提高靈敏度
- **長線投資**：可拉長至20-22日，過濾雜訊
- **均線搭配**：建議搭配20期均線作為趨勢參考

## 實戰應用

### 1. 動態停損設定（ATR Stop）

ATR 最核心的應用——根據波動大小自動調整停損距離：

| 交易風格 | 停損距離 | 說明 |
|----------|----------|------|
| 極短線（當沖） | 0.5-1倍ATR | 波動小時停損近 |
| 短線 | 1-1.5倍ATR | 平衡波動與風險 |
| 波段 | 1.5-2倍ATR | 給予更多呼吸空間 |
| 長線 | 2-3倍ATR | 避免被正常回檔掃出 |

**核心原則**：波動大時停損設遠一點，波動小時停損設近一點。用固定比例停損是散戶最常見的錯誤——不同波動的商品不能用同一個停損距離。

### 2. 移動停利（Trailing Stop）

- **均線+ATR法**：停利線 = 均線 ± N×ATR（做多減、做空加）
- **價格+ATR法**：停利線 = 最高/低價 ± N×ATR
- **ATR通道法**：以20EMA為中軸，±2ATR為上下軌，破軌出場

### 3. ATR通道突破策略

- 中軸：20EMA
- 上軌：20EMA + 2×ATR
- 下軌：20EMA − 2×ATR
- 突破上軌做多、跌破下軌做空
- 搭配量能確認提高勝率

短線變化版：用5分K + 21MA，停損設1/4 ATR，移動停利為20MA + 固定點數。

### 4. 部位控制（Position Sizing）

ATR 可用於計算每筆交易的合理部位大小：

```
部位大小 = 帳戶風險金額 / (ATR × 倍數)
```

例如：帳戶100萬，單筆風險2%（2萬），ATR=3點，停損2倍ATR=6點，則部位=20000/6≈3333股。

### 5. 加碼策略

- 賺1倍ATR後觸發第一次加碼
- 限定加碼總風險（如本金最多虧損不超過2%）
- 先小單測試方向，正確再放大部位

### 6. 波動性判斷

- ATR 上升 → 波動放大，趨勢可能正在啟動
- ATR 下降 → 波動縮小，可能進入盤整
- ATR 極低 → 醞釀突破（布林通道收縮同理）
- ATR 極高 → 波動可能即將收斂

## 注意事項

### ATR 五大盲點

1. **滯後性**：基於歷史價格計算且有平滑效果，訊號反應較慢
2. **無方向性**：只告訴你波動多大，不告訴你方向，必須搭配趨勢指標
3. **低波動市場失效**：盤整時ATR極低，停損距離太近容易被掃
4. **絕對數值限制**：高價股ATR大、低價股ATR小，不能跨股票直接比較（需用ATR% = ATR/收盤價）
5. **跳空缺口影響**：TR計算包含跳空，但停損成交價可能跳過預設點位

### ATR vs 其他波動指標

| 指標 | 特色 | 適用場景 |
|------|------|----------|
| ATR | 絕對波動幅度 | 停損停利設定、部位控制 |
| 布林通道 | 相對波動帶寬 | 突破/收縮判斷 |
| 歷史波動率 | 年化百分比 | 選擇權定價、比較不同標的 |
| VIX | 市場恐慌程度 | 大盤避險時機 |

### 實戰搭配建議

- **ATR + ADX**：ADX過濾盤整（ADX<25不交易），ATR設定停損距離
- **ATR + 均線**：均線判方向，ATR設停損停利
- **ATR + 支撐壓力**：支撐壓力找進場點，ATR決定停損距離
- **ATR + SuperTrend**：SuperTrend本身就是ATR的應用
- **ATR% 比較波動**：ATR÷收盤價，可以跨股票比較波動程度
