# Stochastic RSI隨機RSI

> RSI的加速版——對RSI再做一次隨機計算，靈敏度暴增但雜訊也暴增，必須搭配趨勢指標過濾才有實戰價值

## 核心概念

Stochastic RSI（隨機RSI）由 Tushar Chande 和 Stanley Kroll 在1994年提出，是將隨機震盪指標的計算方式應用在 RSI 值上。

簡單說：**先算 RSI，再對 RSI 做隨機計算**。標準 RSI 衡量價格變化的相對強弱，Stochastic RSI 則衡量 RSI 在一段時間內的相對位置——RSI 創新高時 StochRSI=1，RSI 創新低時 StochRSI=0。

### 計算公式

```
StochRSI = (RSI - MinRSI) / (MaxRSI - MinRSI)
```

- RSI：時間週期內的 RSI 值
- MinRSI：時間週期內 RSI 的最低值
- MaxRSI：時間週期內 RSI 的最高值

有些版本會乘以100，使範圍從0-1變成0-100。

### 參數設定

| 參數 | 預設值 | 說明 |
|------|--------|------|
| RSI長度 | 14 | 計算RSI的時間週期 |
| 隨機長度 | 14 | 計算隨機指標的時間週期 |
| K平滑 | 3 | %K平滑期數 |
| D平滑 | 3 | %D平滑期數 |

### 14日 StochRSI 基準值

| StochRSI值 | 意義 |
|-----------|------|
| 1.0（100） | RSI 在14日內創新高 |
| 0.8（80） | RSI 接近14日內高點 |
| 0.5（50） | RSI 在14日高低範圍中間 |
| 0.2（20） | RSI 接近14日內低點 |
| 0.0（0） | RSI 在14日內創新低 |

## 實戰應用

### 一、超買超賣判斷

- **StochRSI > 0.8（80）** → 超買
- **StochRSI < 0.2（20）** → 超賣

⚠️ 關鍵差異：StochRSI 的超買超賣門檻比標準 RSI 更極端（RSI用70/30，StochRSI用80/20）

**趨勢中的用法**：
- 上升趨勢 → 找 StochRSI 跌至0.2以下的超賣進場點
- 下降趨勢 → 找 StochRSI 升至0.8以上的超買進場點（做空）
- **絕對不要用 StochRSI 逆趨勢交易**

### 二、%K與%D交叉訊號

- %K 上穿 %D → 看漲訊號
- %K 下穿 %D → 看跌訊號

與 KD 指標的交叉邏輯相同，但因為是對 RSI 做隨機計算，訊號更頻繁也更敏感。

### 三、背離訊號

- **正背離（底背離）** + StochRSI 回升至0.2以上 → 可能出現上升趨勢
- **負背離（頂背離）** + StochRSI 回落至0.8以下 → 可能出現下降趨勢

背離訊號比單純超買超賣更可靠，因為背離代表動能正在減弱。

### 四、與標準 RSI 的搭配

Stochastic RSI 不是要取代 RSI，而是**互補**：

| 使用場景 | 用 RSI | 用 StochRSI |
|---------|--------|-------------|
| 趨勢強弱判斷 | ✅ 更穩定 | ❌ 太靈敏 |
| 精確進出場時機 | ❌ 太慢 | ✅ 更靈敏 |
| 長期分析 | ✅ 適合 | ❌ 雜訊多 |
| 短期交易 | ❌ 不夠靈敏 | ✅ 更適合 |
| 鈍化期間 | ❌ 鈍化嚴重 | ✅ 仍然有波動 |

## 注意事項

### 雙重距離的危險
Stochastic RSI 是「指標的指標」，距離價格**兩步之遙**：
- RSI 距離價格一步
- StochRSI 距離價格兩步

距離越遠，與實際價格走勢脫節的風險越高。

### 雜訊地獄
- 速度比標準 RSI 快很多
- 產生更多訊號 = 更多好訊號 + 更多壞訊號
- **單獨使用等於自殺**，必須搭配趨勢指標過濾

### 盤整市的陷阱
- 盤整市中 StochRSI 會在超買超賣之間快速來回
- 頻繁的交叉訊號都是假訊號
- **必須搭配 [[ADX趨勢強度過濾盤整]] 或 [[一目均衡表Ichimoku-Kinko-Hyo]] 判斷趨勢方向**

### 趨勢過濾是生存關鍵
- 上升趨勢中只做多（StochRSI 超賣是買點）
- 下降趨勢中只做空（StochRSI 超買是賣點）
- 沒有明確趨勢時不要使用 StochRSI

### 與 KD 指標的差異

| 特徵 | KD指標 | Stochastic RSI |
|------|--------|----------------|
| 計算基礎 | 價格的最高最低 | RSI 的最高最低 |
| 靈敏度 | 中等 | 很高 |
| 鈍化問題 | 有 | 更嚴重 |
| 適用場景 | 通用 | 短線精確時機 |
