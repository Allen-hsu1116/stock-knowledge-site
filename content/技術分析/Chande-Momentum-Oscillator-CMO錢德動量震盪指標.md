# 錢德動量震盪指標 CMO (Chande Momentum Oscillator)

> 測量「純動量」的震盪指標，未經平滑處理，比 RSI 更敏感，適合抓轉折與背離。

## 核心概念

**錢德動量震盪指標（Chande Momentum Oscillator, CMO）** 由 Tushar Chande 於 1994 年在《The New Technical Trader》一書中提出，是與 Stanley Kroll 共同開發的指標（兩人還共同開發了 [[Chande-Kroll-Stop錢德克羅停損指標]]）。

CMO 的核心特色：**計算分子同時包含上漲日和下跌日的數據**，不像 RSI 只用上漲日的比例，因此被稱為「純動量」指標。

### 計算公式

```
CMO = 100 × (Su - Sd) / (Su + Sd)
```

其中：
- **Su** = 指定期間內上漲日漲幅的總和
- **Sd** = 指定期間內下跌日跌幅的絕對值總和
- 預設期間：**14 或 20**

CMO 數值範圍：**-100 到 +100**

### 計算步驟

1. 計算指定期間內每日收盤價變動
2. 上漲日的漲幅加總 → Su
3. 下跌日的跌幅加總（取絕對值）→ Sd
4. 代入公式計算 CMO 值

### 與 RSI 的根本差異

| 特性 | CMO | RSI |
|------|-----|-----|
| 分子 | Su - Sd（淨動量） | Su / (Su + Sd)（上漲比例） |
| 範圍 | -100 到 +100 | 0 到 100 |
| 平滑 | **無**（原始動量） | 有（指數平滑） |
| 敏感度 | **較高** | 較低 |
| 超買超賣 | ±50 | 70/30 |
| 中心線 | 0 | 50 |

CMO 因為沒有平滑處理，**反應更快但也更容易產生雜訊**。這是一把雙面刃：在趨勢明確時能更早捕捉轉折，但在盤整時假訊號更多。

## 實戰應用

### 1. 超買超賣判斷

- **CMO > +50** → 超買區，注意可能回調
- **CMO < -50** → 超賣區，注意可能反彈
- **CMO 接近 0** → 動量中性，盤整格局

**實戰要點**：在強趨勢中，CMO 可以長期停留在 ±50 以上，不要因為超買就急著做空。

### 2. 零線交叉

- CMO 由負轉正（突破 0） → **多頭動量啟動**
- CMO 由正轉負（跌破 0） → **空頭動量啟動**
- 零線是比超買超賣更穩定的方向判斷基準

### 3. 訊號線交叉

在 CMO 上疊加一條 **9-10 期移動平均線**作為訊號線：

- CMO 向上突破訊號線 → **買進訊號**
- CMO 向下跌破訊號線 → **賣出訊號**
- 訊號線交叉可以過濾部分雜訊

### 4. 背離交易

CMO 的背離訊號比 RSI 更早出現（因為沒有平滑延遲）：

- **頂背離**：價格創新高，CMO 未創新高 → 空頭背離，看跌
- **底背離**：價格創新低，CMO 未創新低 → 多頭背離，看漲

### 5. 趨勢強度判斷

- |CMO| > 50 → 強趨勢
- |CMO| 在 20-50 → 中等趨勢
- |CMO| < 20 → 弱趨勢/盤整

CMO 的絕對值越大，趨勢越強。這可以用來搭配 [[ADX趨勢強度過濾盤整]] 做雙重確認。

### 6. CMO + 其他指標組合

**CMO + 均線趨勢過濾**：
- 均線向上 + CMO 突破 0 → 趨勢確認做多
- 均線向下 + CMO 跌破 0 → 趨勢確認做空

**CMO + 支撐壓力**：
- 價格在支撐區 + CMO 底背離 → 強力買進訊號
- 價格在壓力區 + CMO 頂背離 → 強力賣出訊號

**CMO + [[MACD指標實戰判讀]]**：
- CMO 先轉向 + MACD 後確認 → 雙重確認進場
- CMO 背離 + MACD 背離 → 強力反轉訊號

### 7. CMO 自身型態分析

CMO 本身也可以做技術分析：

- CMO 的雙頂/雙底 → 價格反轉預警
- CMO 的趨勢線突破 → 價格即將跟進
- CMO 的頭肩頂/底 → 較強的反轉訊號

## 注意事項

### ⚠️ 盤整市假訊號多

CMO 沒有平滑處理，在盤整市中會頻繁穿越零線和超買超賣區。**必須搭配趨勢判斷工具**（均線、ADX、[[鱷魚線Alligator-Indicator]]）過濾。

### ⚠️ 強趨勢中超買超賣失效

跟所有震盪指標一樣，CMO 在強趨勢中可以長期停留在 ±50 以上。超買不代表要跌，超賣不代表要漲。

### ⚠️ 期間選擇影響很大

- 短期（5-10）：極度敏感，適合極短線交易，但雜訊最多
- 中期（14-20）：平衡敏感度和穩定度，最常用的設定
- 長期（30-50）：較穩定，適合波段交易

### ⚠️ 與隨機指標的區別

CMO 比 Stochastic 更敏感但更不平滑。如果需要穩定的訊號，Stochastic 可能更適合；如果需要早期捕捉轉折，CMO 更有優勢。
