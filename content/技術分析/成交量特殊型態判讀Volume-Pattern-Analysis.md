# 成交量特殊型態判讀Volume-Pattern-Analysis

> 成交量的特殊形狀是主力行為的指紋——窒息量是起漲前的寂靜、凹洞量是換手完成的確認、遞增量是多方的攻擊號角、U型量是主力吸籌的腳印

## 核心概念

成交量不只是「量大量小」的二分法。不同型態的量能變化蘊含不同的市場訊息，就像醫生看心電圖——不只是心跳快慢，波形變化才是診斷關鍵。

**六大特殊型態**：
1. **窒息量**（地量）— 量萎縮到極致，變盤在即
2. **凹洞量**（量坑）— 短暫量縮後恢復，換手確認
3. **遞增量**（量潮）— 量逐日放大，多方攻擊
4. **U型量**（量谷）— 量緩縮再緩增，主力吸籌
5. **脈衝量**（單日爆量）— 一閃而過的量能異常
6. **量滾量**（換手量）— 量維持高位但價格不動，籌碼交換

型態的意義取決於出現位置：同樣是量縮，底部窒息量是機會、頭部量縮是危機。

## 窒息量（地量）

### 定義與判斷標準

成交量萎縮到近期最低水平，通常低於20日均量的50%，甚至出現近3-6個月的新低量。

| 判斷條件 | 標準 |
|----------|------|
| 量能水準 | < 20日均量的50% |
| 持續時間 | 連續3日以上低量 |
| 換手率 | 大盤<0.3%，個股<0.5% |

### 位置決定意義

| 位置 | 意義 | 操作 |
|------|------|------|
| 長期底部 | 最後洗盤，浮額清洗完畢 | 觀察帶量突破進場 |
| 下跌中繼 | 只是暫時休息，下跌未止 | 續抱空單或觀望 |
| 盤整末期 | 方向即將選擇 | 等待帶量方向確認 |
| 上漲中繼 | 換手完成，籌碼穩定 | 多頭續抱 |

### 實戰要點

1. **窒息量不是進場訊號**，是「準備進場」的訊號——真正的買點是窒息量之後的帶量長紅
2. **窒息量持續天數越多，後續爆發力越大**——7天以上的窒息量比3天的更有意義
3. **大盤窒息量比個股窒息量更可靠**——個股可能因為冷門而常態性低量
4. **窒息量 + [[籌碼面分析/籌碼七指標選股法|籌碼七指標選股法]] = 最佳進場時機**

### 窒息量量化判斷：今週刊補充

今週刊補充了窒息量與凹洞量的底部判讀實戰角度：

- **窒息量的量化定義**：量能只有近期最大成交量的**十分之一**，買賣雙方都縮手，量能快速萎縮
- **窒息量出現的時機**：通常發生在空方無力再殺、價格跳空之後，窒息量出現時，距離底部多半不遠
- **換手率角度**：單日換手率超過10%是市場過熱，反過來要找低點就要趁換手率極低時進場
- **實戰案例**：F-譜瑞（4966）在2014年5月，成交量從3千多張萎縮至不到300張出現窒息量，隨後量暴增至4千多張拉長紅，股價從260多元最高漲至435元

### 窒息量三大陷阱

- **冷門股常態低量** ≠ 窒息量：換手率長期低於0.3%的股票，低量是常態不是訊號
- **停板鎖住低量** ≠ 窒息量：漲停鎖死時量縮是因為買不到，不是意願低
- **除息前後量縮** ≠ 窒息量：制度性因素造成的量縮不具分析意義

## 凹洞量（量坑）

### 定義

成交量從正常水平突然萎縮到極低再恢復，形成「凹洞」形狀。與窒息量的區別：持續時間較短（3-5日），前後有明確的量能基準線。

### 識別四要素

1. 前後成交量維持在均量水平
2. 中間出現明顯量能凹陷
3. 凹洞期間成交量 < 5日均量
4. 凹洞後量能快速恢復

### 型態分類

| 型態 | 出現位置 | 意義 |
|------|----------|------|
| 底部凹洞 | 長期下跌後 | 主力悄悄吸籌，浮額洗盡 |
| 途中凹洞 | 上升趨勢中回測 | 換手完成，攻擊再起 |
| 頭部凹洞 | 高檔盤整中 | 主力不願追高，動能衰退 |

### 凹洞量 vs 窒息量：今週刊補充

今週刊特別區分了兩者的差異：

- **窒息量**是跟「近期最大成交量」相比，萎縮到十分之一
- **凹洞量**是跟「前後交易日」相比呈現萎縮，形成凹洞
- 凹洞量比窒息量**更能表現市場從谷底走向復甦**的過程，因為它代表量能先萎縮再恢復的完整循環
- 窒息量代表買賣雙方都縮手（絕對低量），凹洞量代表量能短暫休憩後重新進場（相對低量）

### 實戰操作

- **凹洞量底部進場法**：凹洞量結束後第一根帶量長紅進場
- **凹洞量確認法**：凹洞期間K線實體縮小、低點未破前低 → 有效換手
- **凹洞量陷阱**：凹洞後繼續量縮下跌 → 不是換手而是出貨
- **底部確認的完整序列**：末跌段價跌量急縮 → 形成窒息量或凹洞量 → 等待打底完成 → 出現價漲量增才能確認反轉

## 遞增量（量潮）

### 定義

成交量隨價格上漲逐日放大，每一根K線的成交量都大於前一日，是多方攻擊最健康的量能型態。

### 三種遞增量型態

| 型態 | 每日增量 | 特徵 | 力道 |
|------|----------|------|------|
| 溫和遞增 | 5-10% | 穩健多頭 | 續航力最強 |
| 加速遞增 | 20-30% | 攻擊波 | 注意量能可持續性 |
| 爆量遞增 | 50%以上 | 短線噴出 | 可能形成頭部 |

### 遞增量與均量線的關係

遞增量是推動均量線排列的核心力量：

1. **5日均量上穿20日均量** → 短線攻擊啟動
2. **5日均量上穿60日均量** → 中線趨勢確立
3. **三條均量線多頭排列** → 長多格局確認

參考：[[量價8大口訣與均量線三層次判讀Volume-Price-8-Rules-and-VMA|量價8大口訣與均量線三層次判讀]]

### 遞增量衰減訊號（多頭警報）

- 遞增幅度開始縮小（第1天增50%→第3天只增5%）→ 攻擊力道衰竭
- 價格上漲但量能開始遞減 → 量價背離，參考[[背離Divergence進階實戰|背離進階實戰]]
- 單日爆量後隔日量能急縮 → 換手失敗

## U型量（量谷）

### 定義

成交量從高水平逐漸萎縮到谷底再逐漸放大，形成U型。與凹洞量的區別：萎縮和恢復過程更緩慢、更對稱。

### 四階段識別

1. **左側遞減**：量能從高水平逐步萎縮（約5-10日）
2. **底部穩定低量**：量能維持在低水平（約3-7日）
3. **右側遞增**：量能逐步恢復（約5-10日）
4. **左右對稱**：萎縮和恢復的速度大致相當

### 位置意義

| 位置 | 意義 | 確認條件 |
|------|------|----------|
| 底部U型 | 主力緩慢吸籌 | 右側放量突破頸線 |
| 上升途中U型 | 中繼換手 | 右側量能超過左側 |
| 頭部U型 | 主力緩慢出貨 | 右側價格無法創新高 |

### 與型態學的對應

- 底部U型量 + [[技術分析/頭肩頂與頭肩底Head-and-Shoulders|頭肩底]] = 高勝率買點
- 頭部U型量 + [[技術分析/頭肩頂與頭肩底Head-and-Shoulders|頭肩頂]] = 高勝率賣點
- 途中U型量 + [[整理型態旗形三角旗形楔形三角形|旗形]] = 順勢加碼點

## 脈衝量（單日爆量）

### 定義

單日成交量突然暴增到平日的3倍以上，隔日迅速恢復正常水平。一閃而過的量能異常。

### 位置決定意義

| 位置 | 意義 | 後續觀察 |
|------|------|----------|
| 長期底部 | 主力搶進，利多刺激 | 觀察後續量能是否持續 |
| 上升初期 | 突破起漲，換手積極 | 利多，可追價 |
| 上升末期 | 主力對倒出貨 | 利空，準備出場 |
| 下跌途中 | 恐慌性拋售 | 可能是底部訊號 |

### 實戰要點

1. **脈衝量隔日量能回歸是正常現象**，不要因為隔日量縮就恐慌
2. **關鍵是脈衝量之後的3-5日走勢**：
   - 量縮不跌 = 多方控盤 ✅
   - 量縮下跌 = 出貨完成 ❌
3. 參考[[高檔爆量與底部爆量判讀|高檔爆量與底部爆量判讀]]判斷脈衝量的位置意義

## 量滾量（換手量）

### 定義

在一段期間內，成交量持續維持在均量以上，但價格變動不大（<5%），主力在特定價位區間反覆買賣吸收浮額。

### 識別方法

1. 連續5日以上成交量 > 5日均量
2. 價格波動 < 5%
3. 常出現在底部或突破前的盤整區

### 主力兩種目的

| 目的 | 特徵 | 後續走勢 |
|------|------|----------|
| 吸貨 | 量滾量期間低點逐步墊高 | 後續帶量突破上漲 |
| 出貨 | 量滾量期間高點逐步降低 | 後續跌破支撐下跌 |

### 判斷方法

1. **看分點買賣超**：主力買超 = 吸貨，參考[[籌碼面分析/主力勝率與區間均價交叉驗證|主力勝率與區間均價]]
2. **看融資變化**：融資增加 = 散戶進場 = 主力可能在出貨
3. **看K線型態**：低點墊高 = 多頭換手，高點降低 = 空頭換手

## 量能型態的完整序列判讀

### 底部反轉的量能序列

```
長期下跌 → 窒息量出現
    → 帶量長紅（脈衝量或遞增量）
        → 凹洞量或U型量（換手）
            → 量滾量或遞增量（攻擊）
```

### 頭部反轉的量能序列

```
上升趨勢 → 量能開始衰減
    → 價格創新高但量縮（量價背離）
        → 爆量長黑或脈衝量後下跌
            → 反彈量縮（凹洞量但右側不放量）
```

### 組合勝率參考

| 組合 | 勝率 | 備註 |
|------|------|------|
| 底部窒息量 + 帶量突破 | 70-80% | 最可靠的底部訊號 |
| 凹洞量 + 帶量長紅 | 65-75% | 需配合籌碼面確認 |
| U型量 + 頸線突破 | 60-70% | 需配合型態學確認 |
| 量滾量 + 主力買超 | 70-80% | 最穩健的吸貨訊號 |
| 脈衝量 + 高檔長黑 | 75-85% | 強烈頭部訊號 |

## 注意事項

1. **型態必須配合位置判斷**——同樣的量能型態在底部和頭部意義完全相反
2. **量能型態需要籌碼面確認**——量滾量看分點、窒息量看法人動向、脈衝量看內外盤比
3. **大盤量能型態比個股更可靠**——個股可能受消息面影響而出現異常量
4. **量能型態是「準備」訊號不是「進場」訊號**——永遠等價格確認才行動
5. **冷門股的量能型態意義有限**——日成交量太低的股票（<500張），量能型態容易被單一主力操控
