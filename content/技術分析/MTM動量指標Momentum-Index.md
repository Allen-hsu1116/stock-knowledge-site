# MTM動量指標 Momentum Index

> 價格漲得越快動量越大、漲得越慢動量衰減——動量線穿過中心線就是趨勢轉向的預告

## 核心概念

MTM（Momentum Index）動量指標由蓋特雷（H. M. Gartley）於 1935 年在《股票市場的獲利（Profits in the Stock Market）》首次提出。它是衡量價格變化速度的最原始指標——不搞平滑、不搞加權，就是今天跟 n 天前的價差。

### 計算公式

```
MTM = Ct - Ct-n
```

- **Ct** = 當日收盤價
- **Ct-n** = n 個交易日前的收盤價
- **n** = 一般設 10 或 12 天

MT4/MT5 上的 Momentum 版本用比率計算：`Momentum = (Ct / Ct-n) × 100`，以 100 為中心線。兩種算法邏輯相同，只是表達方式不同。

### 動量守恆原理

MTM 基於物理學的動量守恆概念：
- 價格變化速度加快 → 趨勢可能持續
- 價格變化速度放緩 → 趨勢可能反轉
- 即使價格還在上漲，只要漲幅縮小，動量指標就會先反轉下降

**關鍵特性：動量指標領先價格變動顯示趨勢轉換。**

## 實戰應用

### 一、基本用法：中心線交叉

MTM 以 0 線（或 MT4/MT5 上的 100 線）為中心：

| 訊號 | 判讀 | 操作 |
|------|------|------|
| 動量線從下穿越中心線 | 價格趨勢轉向多方 | 做多 |
| 動量線從上穿越中心線 | 價格趨勢轉向空方 | 做空 |
| 動量線在極值徘徊 | 趨勢將延續 | 續抱 |
| 動量線向中心線靠攏 | 動能衰減 | 準備出場 |

### 二、MTM + MA 當沖

做當沖時搭配短週期移動平均線（5 MA 或 10 MA）：

- **做多**：MTM 在中心線之上 + 價格在 MA 之上
- **做空**：MTM 在中心線之下 + 價格在 MA 之下
- 兩個條件同時成立才進場

### 三、MTM 背離訊號

- **頂背離**：價格創新高但 MTM 沒創新高 → 漲勢趨緩，注意反轉
- **底背離**：價格創新低但 MTM 沒創新低 → 跌勢趨緩，注意反轉

### 四、參數調整

| 交易風格 | n 值 | 效果 |
|----------|------|------|
| 超短線當沖 | 5~8 | 反應極快，假訊號多 |
| 日內/短線 | 10~12 | 預設值，平衡型 |
| 波段操作 | 20~30 | 反應慢，訊號可靠度高 |

## 注意事項

### 只看價格不看其他因素

MTM 只考慮價格的時間差，完全忽略量能、市場新聞、總經環境。遇到突發事件時，MTM 完全沒輒。

### 沒有上下限

不像 KD 有 0~100、RSI 有 0~100 的範圍，MTM 的值沒有上下限，判斷「過熱」或「過冷」全憑經驗和主觀判斷。

### 盤整時假訊號多

和所有趨勢型指標一樣，盤整時 MTM 會在中心線附近來回穿越，頻繁進出場只有被巴的份。

### MTM vs ROC 差異

- MTM = Ct - Ct-n（差值），單位是價格
- ROC = (Ct - Ct-n) / Ct-n × 100（百分比），單位是百分比
- 兩者形狀幾乎相同，ROC 多了百分比標準化，方便跨商品比較
- MT4/MT5 的 Momentum = (Ct / Ct-n) × 100，本質上就是 ROC 的變形

### 不能單獨使用

MTM 的角色是「動能溫度計」，不是獨立的交易系統。一定要搭配趨勢指標（均線、MACD）或量能指標（OBV、成交量）一起使用。
