# 趨勢反轉判斷Trend-Reversal

> 趨勢從多頭轉為空頭、或從空頭轉為多頭的轉折點，是交易者獲利最大的進場時機，但也是最容易誤判的陷阱。

## 核心概念

趨勢反轉是指市場從多頭行情轉換為空頭行情，或從空頭行情轉變為多頭行情的過程。掌握趨勢反轉之所以重要，是因為在趨勢剛開始反轉時進場，可以取得最好的成本價位；若等行情已經漲跌一大段後才進場，利潤空間減少且容易一進場就回檔。

## 實戰應用

### 5種判斷趨勢反轉的核心要領

#### 1. K線型態判斷

透過觀察K線型態確認行情是否反轉：
- **頭肩頂/頭肩底**：最經典的反轉型態，出現在上漲趨勢中預示反轉為下跌趨勢；出現在下跌趨勢中預示反轉為上漲趨勢
- **雙頭/雙底（M頭/W底）**：某個區間的價位在反覆測試中無法被打破，可能形成趨勢反轉
- **多頭吞噬/空頭吞噬**：組合K棒中最可靠的反轉訊號
- **早晨之星/黃昏之星**：三根K線構成的經典轉折型態

→ 詳見 [[技術分析/K線組合型態反轉訊號]]、[[技術分析/型態學總論Chart-Patterns-Overview]]

#### 2. 支撐壓力位突破判斷

支撐和壓力位是市場參考的重要價格區域：
- 下跌趨勢中，價格跌到某一支撐位後跌破失敗，下跌趨勢可能反轉為上漲趨勢
- 上升趨勢中，價格漲到某一壓力位後突破失敗，上升趨勢可能反轉為下跌趨勢
- 關鍵：支撐壓力位的角色互換（支撐變壓力、壓力變支撐）

→ 詳見 [[技術分析/支撐壓力]]、[[技術分析/支撐壓力互換與真假突破判斷]]

#### 3. RSI超買超賣判斷

RSI指標用於判斷市場是否處於超買或超賣狀態：
- RSI進入超買區（通常超過70）→ 價格上漲過度，可能即將回檔
- RSI進入超賣區（低於30）→ 價格下跌過度，可能出現反彈
- 注意：強趨勢中RSI可能長期處於超買/超賣區，不能單獨作為反轉依據

→ 詳見 [[技術分析/RSI相對強弱指標]]、[[技術分析/背離Divergence進階實戰]]

#### 4. 移動平均線交叉判斷

短期均線穿越長期均線被視為趨勢反轉的徵兆：
- **黃金交叉**：短期均線向上穿越長期均線，潛在上漲趨勢
- **死亡交叉**：短期均線向下穿越長期均線，潛在下跌趨勢
- MACD本質上也是均線的運用，觀察零軸與信號線交叉判斷趨勢反轉

→ 詳見 [[技術分析/均線判斷]]、[[技術分析/MACD指標實戰判讀]]、[[技術分析/葛蘭碧八大法則Granville-8-Rules]]

#### 5. 趨勢反轉策略

| 策略 | 說明 | 適用情境 |
|------|------|----------|
| 趨勢線突破 | 走勢突破或跌破趨勢線 | 明確趨勢中的反轉 |
| 箱型突破 | 尋找盤整區的突破 | 大跌後底部盤整 |
| 交叉策略 | 快速均線突破慢速均線 | 中長期趨勢轉換 |
| 頭底型態 | M頭/W底/頭肩頂底突破頸線 | 大型反轉型態 |
| 多指標共振 | 以上4種方法同時出現訊號 | 高機率反轉確認 |

## 趨勢反轉會失敗嗎？

趨勢反轉當然有可能失敗，也就是所謂的「假反轉」。增加判斷準確度的方法：

1. **多時間框架確認**：日K出現反轉訊號，確認週K是否也出現同樣訊號
2. **量能確認**：反轉時必須有量的配合，无量反轉大概率是假訊號
3. **均線確認**：是否同時跌破/站上年線、季線、月線
4. **指標共振**：多個指標同時出現反轉訊號，可信度更高
5. **基本面與消息面**：研究經濟環境、新聞資訊，判斷市場整體情緒

## 注意事項

1. **反轉訊號需要確認**：永遠等下一根K線確認，不要在訊號未確認時就進場
2. **假反轉是常態**：趨勢中的回檔或反彈不等於反轉，需區分「趨勢中的調整」和「趨勢的改變」
3. **大盤環境優先**：大盤系統性崩跌時，90%的個股反轉訊號會失效
4. **多指標共振才可靠**：單一指標的反轉訊號勝率有限，需要至少2-3個指標同時確認
5. **時間框架越大越可靠**：週K線的反轉訊號比日K線更可靠
6. **量價配合是必要條件**：没有量能配合的反轉，大概率只是趨勢中的技術性調整
