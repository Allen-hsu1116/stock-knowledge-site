# 均線扣抵值判讀

> 均線扣抵值讓你提前看穿均線即將上揚或下彎——扣抵值下降均線自然上揚，扣抵值上升均線自然下彎，蹺蹺板原理一秒判斷支撐壓力力道強弱

## 核心概念

### 什麼是均線扣抵

均線扣抵是判斷均線未來方向的最實用工具。移動平均線的計算原理是：新一天的均線值 = 移除最舊一天的收盤價 + 加入最新一天的收盤價。因此，**即將被移除的那一天的收盤價**就是「扣抵值」。

以5日均線為例：
- 今天5MA = (Day1 + Day2 + Day3 + Day4 + Day5) / 5
- 明天5MA = (Day2 + Day3 + Day4 + Day5 + Day6) / 5
- **Day1**就是明天的5MA扣抵值

### 三種判斷結果

| 明日收盤價 vs 扣抵值 | 均線方向 | 意義 |
|---|---|---|
| 新收盤價 > 扣抵值 | 均線向上 | 多方格局延續 |
| 新收盤價 = 扣抵值 | 均線持平 | 趨勢可能轉折 |
| 新收盤價 < 扣抵值 | 均線向下 | 空方格局延續 |

### 蹺蹺板原理（快速記憶法）

把扣抵值想成蹺蹺板的左側、股價和均線方向想成右側：

- **左側扣抵值由高往低下降** → 右側股價容易上升（均線上揚）
- **左側扣抵值由低往高上升** → 右側股價容易下降（均線下彎）

這就是扣抵值的核心邏輯：**你不需要知道明天收盤價多少，只要觀察扣抵值的趨勢，就能預判均線方向**。

## 實戰應用

### 1. 判斷支撐壓力力道強弱

當股價由下往上接近均線（測試壓力）：
- **壓力疲弱**：扣抵值未來趨勢向下 → 只要股價不暴跌，均線自然會下彎，股價自動突破壓力
- **壓力強勁**：扣抵值未來趨勢向上 → 股價上漲力道要強過扣抵值上升幅度才能突破

當股價由上往下接近均線（測試支撐）：
- **支撐強勁**：扣抵值未來趨勢向下 → 均線自然上揚，形成強力支撐
- **支撐疲弱**：扣抵值未來趨勢向上 → 均線自然下彎，支撐容易被突破

### 2. 提前預判均線交叉（黃金交叉/死亡交叉）

**預判黃金交叉**：
- 短期均線扣抵值呈下降趨勢 + 長期均線扣抵值呈上漲/盤整/下降較緩 → 黃金交叉機率高
- 原因：短期均線受扣抵值下降影響會加速上揚，長期均線受扣抵值上升影響會減速或下彎

**預判死亡交叉**：反向判斷即可

### 3. 不同週期扣抵值的實戰用法

- **5MA/10MA扣抵**：短線操作，判斷極短線支撐壓力力道，扣抵位置逐漸降低代表短線支撐強
- **20MA扣抵**：波段操作核心，月線扣抵低位階區域是中線由空轉多關鍵
- **60MA扣抵**：中長線判斷，季線扣抵值趨勢影響未來1-2個月走勢
- **240MA扣抵**：長線多空分界，年線扣抵值趨勢決定長期格局

### 4. 搭配籌碼面提高勝率

均線扣抵判斷出黃金交叉後，不一定要馬上進場：
- 搭配法人連續買超 → 進場做多勝率大幅提高
- 法人無連續性買超 → 可能只是短暫反彈，股價快速回檔
- 籌碼面比技術指標更適合搭配，因為技術指標反應太快可能一日翻多，法人買超有連續性

### 5. 台股實戰案例

**元太（8069）由空轉多**：
- 2023下半年跌破季線走空
- 橫盤整理後均線走平，扣抵值進入低位階
- 2024年初站上季線，季線扣抵低位區域自然上彎形成支撐
- 在季線附近進場170元，沿上彎季線爬升到260元，跌破季線220元出場，獲利30%

**國際中橡（2104）多層均線扣抵判斷**：
- 20MA扣抵值29.4元但股價24.9元，扣抵值未來有半個月急跌 → 壓力疲弱容易突破
- 60MA扣抵值26.3元，未來2個月扣抵值盤整+上漲 → 壓力強勁一次突破機率低
- 操作：先突破20MA → 碰到60MA壓力回檔 → 20MA扣抵值快速下降形成支撐 → 撐過2個月後突破60MA

## 注意事項

- **扣抵值是機率工具不是水晶球**：預測的都只是機率，絕對會有不如預期的走勢
- **走勢與扣抵預測相反時要特別留意**：代表不尋常事件發生，可能是底部翻轉的抄底訊號或暴跌前兆
- **扣抵值不只適用均線**：所有以週期計算的技術指標（KD、RSI、MACD等）都能用扣抵邏輯預判方向
- **黃金交叉不一定會漲**：空頭趨勢中短期反彈可能造成黃金交叉但股價不漲反跌，必須確認均線多頭排列才能進場
- **長週期扣抵值要耐心等待**：60MA以上扣抵值影響期長達1-2個月，不要急於一時突破
- **台股漲跌幅限制影響**：10%漲跌幅限制下，隔天收盤價有時無法超過扣抵值（例如需要漲12.88%但上限10%），均線方向無法一日反轉
