# 底部訊號綜合判讀Bottom-Signal-Confluence

> 窒息量出現只是「準備」訊號，融資清洗只是「接近」底部，真正的底部確認需要量價、籌碼、技術指標三重共振

## 核心概念

底部判斷是投資最困難也最有價值的技能。單一指標的底部訊號可靠度有限，但當多個獨立指標同時出現底部訊號時，勝率大幅提升。底部判斷的核心邏輯是：**市場極度悲觀時，所有賣方都已出場，任何微小的買力都能推動價格反轉。**

## 七大底部訊號指標

### 1. 窒息量與凹洞量（量價面）

**窒息量**：成交量萎縮到近期最大量的十分之一，買賣雙方都縮手。

- 量化判斷：< 20日均量的50%，甚至3-6個月新低量
- 換手率角度：大盤 < 0.3%，個股 < 0.5%
- 出現時機：空方無力再殺、價格跳空之後
- **窒息量不是進場訊號**，是「準備進場」訊號——真正買點是窒息量後的帶量長紅

**凹洞量**：跟前後交易日相比呈現萎縮，形成「凹洞」。

- 比窒息量更能表現市場從谷底走向復甦
- 凹洞量代表量能先萎縮再恢復的完整循環
- 凹洞量後第一根帶量長紅是進場點

**底部確認序列**：末跌段價跌量急縮 → 窒息量/凹洞量出現 → 打底完成 → 價漲量增確認反轉

參考：[[成交量特殊型態判讀Volume-Pattern-Analysis|成交量特殊型態判讀]]

### 2. 融資餘額（籌碼面）

融資餘額是散戶情緒溫度計，底部區的融資特徵：

- **大盤融資餘額 < 1,500億**：底部開始浮現
- **大盤融資餘額 > 3,500億**：市場過熱
- **融資使用率 < 1%**：非常健康（如台積電）
- **融資使用率 > 60%**：高風險警戒

空頭末期的融資清洗過程：
1. 融資餘額從高點持續減少 → 散戶停損出場
2. 融資維持率逼近130%警戒區 → 斷頭賣壓積聚
3. 融資餘額不再創低 → 浮額清洗完畢
4. 稍有利多就反彈 → 主力開始進場

參考：[[大盤融資餘額判讀]]、[[融資維持率大盤判讀與市場恐慌反轉訊號]]

### 3. KD值黃金交叉（技術面）

- KD值低於20進入低檔區
- K值由下往上突破D值（黃金交叉）→ 積極買進訊號
- 搭配DMI指標：+DI突破-DI且ADX轉折向上 → 多頭確認

參考：[[KD指標隨機指標]]

### 4. 本益比（基本面）

- 本益比倒數 = 外部股東報酬率
- 本益比越低，股價越被低估
- **兩大陷阱**：公司虧損算不出本益比、長期低本益比可能是缺乏競爭力
- 搭配股價淨值比、現金股利殖利率綜合判斷

參考：[[本益比進階判讀PE-Ratio-Advanced]]

### 5. 國安基金進場（政策面）

- 歷史經驗：2000年網路泡沫、2008年金融海嘯，大盤跌破10年線時進場
- 國安基金不一定買在最低點，但拉長2-3年看都是相對低點
- 跟著國安基金進場是賺多賠少

### 6. 市場情緒極度悲觀（心理面）

- 報章雜誌一片看衰
- 利空消息不斷
- 政府救市動作頻繁（減稅等）
- 巴菲特名言：「在他人貪婪時恐懼，在他人恐懼時貪婪」

### 7. 價量關係九種型態判讀

底部區的價量特徵：

| 型態 | 位置 | 意義 | 操作 |
|------|------|------|------|
| 價漲量增 | 底部盤整期 | 主力吃貨 | 鎖股觀察 |
| 價平量增 | 底部區 | 主力逢低吃貨 | 觀察反彈 |
| 價跌量增 | 末跌段 | 最後恐慌賣壓 | 注意止跌訊號 |
| 價跌量縮 | 末跌段 | 窒息量/凹洞量 | 等待價漲量增確認 |
| 價平量縮 | 末跌段 | 止跌現象 | 易形成底部 |

參考：[[量價關係九種型態高低檔判讀]]

## 底部確認的完整框架

### 三重共振法

**量價面 + 籌碼面 + 技術面**同時出現底部訊號，勝率最高：

1. **量價面**：窒息量或凹洞量出現 + 帶量長紅確認
2. **籌碼面**：融資餘額大幅清洗至低水位 + 融資使用率低
3. **技術面**：KD值低檔黃金交叉 + DMI多頭轉折

### 底部進場SOP

```
Step 1: 觀察窒息量/凹洞量出現（量價面）
Step 2: 確認融資餘額清洗至低水位（籌碼面）
Step 3: 等待帶量長紅確認反轉（量價面）
Step 4: KD值低檔黃金交叉確認（技術面）
Step 5: 分批進場，設定停損在窒息量低點下方
```

### 底部判斷的常見錯誤

1. **單一指標進場**：只看到窒息量就進場，可能繼續下跌
2. **忽略位置**：窒息量出現在下跌中繼而非底部
3. **過早進場**：凹洞量後不等帶量確認就買入
4. **忽略基本面**：本益比長期偏低可能是公司真的沒競爭力

## 注意事項

1. **底部是區間不是點**——不要執著於買到最低點，而是買在底部區間
2. **量能型態是「準備」訊號**——永遠等價格確認才行動
3. **融資餘額有落後性**——T日交易，T+1才公布
4. **國安基金進場不代表立即反彈**——拉長時間看才是相對低點
5. **市場情緒指標是定性非定量**——需要經驗判斷，不能機械化操作
