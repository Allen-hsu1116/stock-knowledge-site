# Force Index 力量指標

> Alexander Elder 設計的量價振盪指標，結合價格變動方向、幅度與成交量，衡量多空力量並捕捉趨勢反轉點。

## 核心概念

Force Index（力量指標／勁道指數）由 Alexander Elder 在《以交易為生》（1993）提出，公式極簡但涵義深刻：

```
Force Index = 成交量 ×（今日收盤價 − 昨日收盤價）
```

三個要素一次打包：
- **方向**：漲為正、跌為負、平為零
- **幅度**：價差越大，力量越強
- **成交量**：量越大，走勢慣性越大

未經平滑的原始值波動劇烈，實戰中一律用 **EMA 平滑**：
- **2天 EMA**：短期敏感指標，抓進出場點
- **13天 EMA**：中期趨勢指標，判斷多空主控權
- **90天 EMA**：長期趨勢方向確認

## 實戰應用

### 一、零線判斷多空主控

| 狀態 | Force Index 位置 | 涵義 |
|------|-----------------|------|
| 多頭主控 | 13天 EMA > 0 | 多方力量勝出 |
| 空頭主控 | 13天 EMA < 0 | 空方力量勝出 |
| 無明確趨勢 | 13天 EMA 在零線附近 | 停止使用順勢指標的警訊 |

### 二、2天 EMA 短線進出場（三重濾網核心）

這是 Force Index 最經典的用法，Elder 的三重濾網交易系統第二層：

1. **多頭進場**：順勢指標確認上升趨勢 → 等 Force Index 2天 EMA 跌破零線 → 買單掛在高價稍上方
2. **空頭進場**：順勢指標確認下降趨勢 → 等 Force Index 2天 EMA 升破零線 → 空單掛在低價稍下方
3. **停損設定**：多頭停損設在交易日或前一日最低價稍下方（取低者）

**核心心法**：在弱勢中買進、在強勢中賣出。不要在力量最強時追價，而是在拉回力量衰退時進場。

### 三、背離訊號（最重要）

| 背離類型 | 價格表現 | Force Index 表現 | 訊號強度 |
|---------|---------|-----------------|---------|
| 看漲背離（多頭背離） | 創新低 | 底部墊高 | ⭐⭐⭐ 強烈買進 |
| 看跌背離（空頭背離） | 創新高 | 頭部下滑 | ⭐⭐⭐ 強烈賣出 |

背離適用於 2天 EMA 和 13天 EMA，但 **13天 EMA 的背離訊號更可靠**，代表中期趨勢的重要轉折。

### 四、趨勢確認與量價配合

- Force Index 持續創新高 → 多頭勁道十足，上升趨勢應繼續
- Force Index 持續創新低 → 空頭勁道十足，下降趨勢應繼續
- **價格變動但 Force Index 走平** → 量價背離，趨勢即將反轉的警訊
- **突破時 Force Index 飆升** → 帶量突破，確認有效
- **突破時 Force Index 無反應** → 無量突破，可能失敗

### 五、加碼策略

上升趨勢中，每當 Force Index 2天 EMA 跌破零線 → 加碼多頭部位
下降趨勢中，每當 Force Index 2天 EMA 升破零線 → 加碼空頭部位

### 六、平倉時機

| 交易週期 | 平倉條件 |
|---------|---------|
| 短線 | 2天 EMA 翻正→賣出；翻負→回補 |
| 長線 | 13天 EMA 改變方向 或 Force Index 與價格背離 |

## 與其他量價指標比較

| 指標 | 優勢 | 劣勢 |
|------|------|------|
| **Force Index** | 有零線基準，力量大小可量化；背離訊號清晰 | 滯後指標，信號偏慢 |
| OBV | 累積量概念，趨勢線突破有用 | 無零線，力量大小難量化；背離不如 FI 清楚 |
| MFI | 超買超賣判斷 | 在力量分析表現不突出 |

**結論**：三者中 Force Index 最實用——有零線、能量化力量、背離清晰。OBV 的突破策略在 K 線圖就看得見。

## 注意事項

1. **滯後性**：Force Index 使用歷史價量數據再取 EMA，訊號可能延遲數根K線。突破後指標才反彈，此時價格可能已遠離進場點
2. **不適合單獨使用**：必須搭配順勢指標（如 EMA、MACD）判斷大方向，Force Index 負責抓進出場時機
3. **數值過大問題**：成交量乘以價差可能產生極大數值，可除以最近收盤價正規化
4. **盤整市失效**：零線附近徘徊時訊號不明確，應暫停使用順勢策略
5. **背離判斷主觀性**：背離需要經驗判斷，不同人可能看到不同背離型態
6. **短週期過度敏感**：2天 EMA 在波動劇烈的標的上頻繁穿越零線，產生大量假訊號
