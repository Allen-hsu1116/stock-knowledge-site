# Force Index 力量指標（勁道指數）

> 結合「價格變動方向 × 成交量」三維度衡量多空力量，2天 EMA 找進出場點，13天 EMA 確認中長期趨勢，背離是強烈轉折訊號。

## 核心概念

Force Index（力量指標/勁道指數）由 Alexander Elder 博士設計，發表於 1993 年著作《以交易為生》（Trading for a Living）。它是一個振盪指標，結合三項市場資訊：

1. **價格變動方向**：收盤價漲跌
2. **價格變動幅度**：漲跌幅大小
3. **成交量**：市場參與程度

### 公式

```
Force Index = 當日成交量 × (當日收盤價 - 前日收盤價)
```

- 收盤價上漲 → Force Index 為正值（多頭力量）
- 收盤價下跌 → Force Index 為負值（空頭力量）
- 收盤價不變 → Force Index 為零
- 價格變動越大、成交量越大 → Force Index 數值越大（力量越強）

### 短期 vs 中期

| 期間 | 用途 | 特性 |
|------|------|------|
| **2天 EMA** | 短期進出場點 | 非常敏感，適合找買賣點 |
| **13天 EMA** | 中長期趨勢確認 | 較平滑，顯示多空主控權 |
| 90天 EMA | 長期趨勢 | 參考用 |

## 實戰應用

### 1. 短期操作（2天 EMA）

**上升趨勢中找買點**：
- 當 2天 EMA 跌破零線 → 代表短期回調，是買進時機
- 買單設在當天高價稍上方，停損設在當天或前一天最低價稍下方

**下降趨勢中找空點**：
- 當 2天 EMA 向上穿越零線 → 代表短期反彈，是放空時機
- 空單設在最近低價稍下方，停損設在最高價稍上方

**加碼策略**：
- 上升趨勢中，2天 EMA 每跌破零線 → 加碼多頭
- 下降趨勢中，2天 EMA 每穿越零線 → 加碼空頭

### 2. 中期判斷（13天 EMA）

- **13天 EMA 在零線上方** → 多頭主控
- **13天 EMA 在零線下方** → 空頭主控
- **13天 EMA 徘徊零線附近** → 無明確趨勢，停止使用順勢指標

### 3. 背離訊號（最強烈的轉折訊號）

| 背離類型 | 價格 | Force Index | 訊號 |
|---------|------|-----------|------|
| 多頭背離 | 創新低 | 底部墊高 | 強烈買進 |
| 空頭背離 | 創新高 | 頭部下滑 | 強烈賣出 |

### 4. 突破確認

- 突破伴隨成交量放大 → Force Index 飆升確認突破有效
- 突破缺乏成交量 → Force Index 未確認，突破可能失敗

### 5. 三重濾網交易系統

Force Index 2天 EMA 是三重濾網系統的理想第二層濾網指標：
- 第一層：長期趨勢判斷（如 13週 EMA）
- 第二層：Force Index 2天 EMA 找短期進出場點
- 第三層：具體進場時機

## 注意事項

1. **滯後指標**：Force Index 使用歷史價量計算，信號可能延遲；突破後可能需數個 K 棒才反應
2. **數值無界**：不像 RSI 有固定範圍（0~100），Force Index 可無限上升或下降，判讀需依賴零線和歷史比較
3. **需搭配順勢指標**：Elder 強調單一指標不足以交易，需搭配趨勢判斷工具
4. **成交量影響大**：放量長紅/長黑會產生極端值，需注意異常值的干擾
5. **與 OBV、MFI 的比較**：Force Index 有標準0線（比 OBV 更直觀），MFI 主要看超買超賣（力的分析較弱），三者中 Force Index 最適合衡量力量
