# 乖離率 BIAS

> 股價與均線的距離百分比——乖離越大，回歸均線的機率越高，但乖離過大不等於一定反轉

## 核心概念

乖離率（Bias Ratio, BIAS）衡量目前股價偏離移動平均線的程度，以百分比表示。核心原理是**均值回歸**：股價就像遛狗理論中的小狗，無論跑多遠終究會跑回主人（均線）身邊。

**公式**：
```
乖離率 = (當前價格 - N日移動平均線) / N日移動平均線 × 100%
```

- **正乖離**：股價在均線上方，乖離率 > 0，代表過去 N 日買進者處於獲利狀態
- **負乖離**：股價在均線下方，乖離率 < 0，代表過去 N 日買進者處於虧損狀態
- 乖離率也可視為過去 N 日的**平均報酬率**（把均線當作平均成本）

**常用參數**：
| 參數 | 用途 | 特性 |
|------|------|------|
| BIAS6 | 短線判斷 | 波動大，訊號頻繁 |
| BIAS12 | 中短線 | 較平滑 |
| BIAS20 / BIAS24 | 中線（最常用） | 平衡穩定性與靈敏度 |
| BIAS60 / BIAS72 | 長線 | 波動小，訊號稀少但可靠 |

⚠️ 乖離率一定要搭配**同天期均線**判讀：20 日乖離率搭配 20 日均線，60 日乖離率搭配 60 日均線。

## 實戰應用

### 1. 正負乖離判斷買賣方向

- **正乖離過大 → 偏賣出**：市場超買，股價遠離均線，修正壓力上升，避免追高
- **負乖離過大 → 偏買入**：市場超賣，股價遠離均線，反彈機會增加，避免殺低

### 2. 歷史區間法判斷「過大」

**沒有絕對的乖離率標準**，每檔股票股性不同。正確做法：

1. 回顧個股近 2 年的乖離率分布範圍
2. 找出歷史高點與低點的乖離率數值
3. 當前乖離率超出歷史區間時，視為「過大」

**台股實戰閾值參考**（Ultima Markets 建議）：
| 週期 | 正乖離警戒 | 負乖離警戒 |
|------|-----------|-----------|
| 短期 | +6% | -6% |
| 中期 | +8% | -8% |
| 長期 | +10% | -10% |

**台股大盤特殊訊號**：年線（240日）負乖離超過 20%，歷史經驗後一個月上漲機率 100%。

### 3. 多時間框架乖離率交叉確認

- 短期 BIAS6 + 中期 BIAS20 + 長期 BIAS60 三個乖離率同時極端 → 訊號最強
- 短期乖離極端但中長期正常 → 可能只是短線波動，訊號較弱

### 4. 乖離率搭配其他指標

- **BIAS + RSI**：乖離率過大 + RSI 超買/超賣 → 雙重確認反轉
- **BIAS + KD**：乖離率極端 + KD 鈍化 → 強勢股可能持續走強，不要急著反向操作
- **BIAS + 布林通道**：乖離率過大 + 價格觸及布林通道外軌 → 回歸機率極高
- **BIAS + 均線扣抵**：乖離率極端 + 均線即將扣抵高價區 → 均線可能下彎加速收斂

### 5. 強勢股乖離率特徵

強勢股（主升段）的乖離率可能長期維持在正區且數值偏高，這時候：
- 不要因為正乖離大就急著做空
- 觀察乖離率是否**開始收斂**（從高點往下掉）才是轉弱訊號
- 搭配成交量判斷：放量長紅的正乖離可持續，縮量創高的正乖離危險

## 注意事項

### 三大不適用情境

1. **緩漲緩跌或長期盤整的個股**：乖離率變動不明顯，參考價值低
2. **市值太小的個股**：容易被主力操縱，乖離率失真，大型股較準確
3. **趨勢強勁的主升段/主跌段**：乖離率可能長期維持極端值，單獨使用會過早反向操作

### 五大誤區

1. **乖離過大 ≠ 一定反轉**：只是機率升高，強勢股可以維持高乖離很長時間
2. **用固定閾值套所有股票**：每檔股票股性不同，必須用歷史區間判斷
3. **單獨使用乖離率進出場**：乖離率是延遲指標，必須搭配其他指標確認
4. **忽略均線方向**：均線上彎時的正乖離和均線下彎時的正乖離意義完全不同
5. **短期乖離率過度解讀**：BIAS6 波動太大，容易產生假訊號，中長期更可靠

### 乖離率 vs 均線乖離的本質

乖離率本質上是把「均線與價格的距離」量化成百分比。它的底層邏輯和 [[均線判斷]] 一樣是均值回歸，但乖離率的好處是**把視覺上的距離變成具體數字**，可以做歷史比較和跨股比較。

### 7. 乖離率 + 布林通道進階搭配

當乖離率與布林通道搭配找進場時機：
1. 注意布林通道是否正在收斂（Squeeze）
2. 觀察布林通道可能往上或往下擴散的方向
3. 短期均線由下往上穿越長期均線（黃金交叉）+ 乖離率由負轉正 → 多頭進場訊號加強

### 8. 乖離率 + KDJ 進階搭配

- 乖離率確認價格是否超買/超賣
- KDJ評估價格將上漲或下跌的趨勢方向
- 當負乖離極端 + KDJ 從超賣區（J值突破0）向上翻轉 → 強力買進訊號
