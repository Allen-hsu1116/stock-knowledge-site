# 乖離率 BIAS

> 乖離率把「股價離均線多遠」量化成百分比，用來判斷超漲、超跌與均值回歸機會；但強趨勢裡它會一路極端，硬反向操作就是把臉湊上去給市場打。

## 核心概念

乖離率（Bias Ratio, BIAS）衡量目前價格相對於 N 日移動平均線的偏離程度，本質是把「均線距離」變成可比較的百分比。

```text
BIAS(N) = (收盤價 - N日移動平均線) / N日移動平均線 × 100%
```

- **正乖離**：股價高於均線，代表短期漲幅高於平均成本，市場偏熱。
- **負乖離**：股價低於均線，代表短期跌幅低於平均成本，市場偏冷。
- **零軸附近**：股價貼近均線，沒有明顯偏離，通常不是好訊號來源。
- **核心假設**：價格偏離均線太遠後，未來有較高機率回到均線附近，也就是 [[操作策略/均值回歸策略|均值回歸策略]]。

常見參數不是拿來拜神的，週期不同意義不同：

| 參數 | 用途 | 特性 |
|---|---|---|
| 6日 BIAS | 短線/隔日沖觀察 | 靈敏但雜訊多，容易被洗臉 |
| 12日 BIAS | 短波段 | 比 6 日穩定，適合搭配 KD/RSI |
| 20日/24日 BIAS | 中短波段主軸 | 對應月線，是台股常用判斷週期 |
| 60日/72日 BIAS | 中線趨勢偏離 | 訊號少但意義較大，適合看波段過熱或超跌 |
| 240日 BIAS | 大盤長線溫度 | 用於判斷年度級別極端超跌/過熱 |

## 實戰應用

### 1. 先看均線方向，再看乖離大小

乖離率不能脫離均線方向判讀，否則就是拿溫度計當方向盤，白癡但很多人會做。

- **均線上彎 + 正乖離擴大**：多頭趨勢加速，偏多但追價風險升高。
- **均線上彎 + 正乖離收斂**：漲勢降溫，若跌破短均線要警覺轉弱。
- **均線下彎 + 負乖離擴大**：空頭趨勢加速，不要只因「跌很多」就接刀。
- **均線下彎 + 負乖離收斂**：空方力道減弱，等站回均線或量價確認再說。

可搭配 [[技術分析/均線判斷|均線判斷]] 與 [[技術分析/均線扣抵值判讀|均線扣抵值判讀]]：如果均線即將扣抵高價區，均線容易下彎，正乖離收斂會更快；反之扣抵低價區，均線自然上彎，負乖離反彈更容易形成。

### 2. 歷史區間法：每檔股票自己校準

FM Studio、市場先生與 StockFeel 都強調：乖離率沒有全市場通用的魔法數字。不同股性、產業、波動度，正常乖離範圍完全不同。

實戰校準流程：

1. 拉出個股近 1～2 年 BIAS6、BIAS20、BIAS60。
2. 標記過去重要高點與低點時的乖離率。
3. 找出「高檔常見正乖離區」與「低檔常見負乖離區」。
4. 只在接近歷史極端值時提高警戒，不要每天看到 +3%、-3% 就像看到鬼。
5. 將警戒值分成三層：觀察、減碼/試單、強制風控。

參考區間可先用：

| 週期 | 偏熱觀察 | 過熱警戒 | 偏冷觀察 | 超跌警戒 |
|---|---:|---:|---:|---:|
| 6日 | +3% | +6% | -3% | -6% |
| 12日 | +5% | +8% | -5% | -8% |
| 20/24日 | +8% | +12% | -8% | -12% |
| 60/72日 | +10% | +15% | -10% | -15% |

這只是起始模板，不是聖旨。台積電和妖股用同一組閾值，市場會用帳戶餘額教你做人。

### 3. 三段式操作 SOP

**A. 負乖離找反彈，不是無腦抄底**

條件：
- BIAS20 或 BIAS60 進入個股歷史低檔區。
- K 線出現止跌訊號，例如長下影線、吞噬、W底雛形。
- 成交量不再放大殺低，或出現恐慌爆量後縮量止穩。
- RSI/KD 從超賣區翻揚，而不是繼續鈍化向下。

操作：
1. 第一筆只試單，不要一把梭。
2. 站回 5日或10日線再加碼。
3. 反彈到 20日均線或 BIAS 回到零軸附近先減碼。
4. 若跌破前低，立即停損，別在那邊「長期投資」自我催眠。

**B. 正乖離控風險，不是立刻放空**

條件：
- BIAS20/BIAS60 進入歷史高檔區。
- 股價遠離均線但量能縮小，追價力道下降。
- K 線出現長上影、爆量黑K、跌破短均線。
- RSI/MACD 出現背離，或 [[技術分析/布林通道Bollinger-Bands三軌八型態|布林通道]] 上軌外漲不動。

操作：
1. 持股者先調整停利線，不急著清倉。
2. 跌破短均線或 BIAS 由高檔轉折收斂，分批減碼。
3. 只有在趨勢轉弱確認後才考慮反向，不要看到正乖離就手癢放空。

**C. 零軸穿越看趨勢切換**

- BIAS由負轉正，且價格站上均線：短線由弱轉強。
- BIAS由正轉負，且價格跌破均線：短線由強轉弱。
- 零軸附近來回穿越：盤整雜訊，乖離率參考價值下降。

### 4. 多指標搭配

- **BIAS + [[技術分析/RSI相對強弱指標|RSI]]**：負乖離極端 + RSI 超賣翻揚，反彈訊號較強；正乖離極端 + RSI 背離，過熱警訊較強。
- **BIAS + [[技術分析/KD指標隨機指標|KD]]**：KD 在超買/超賣區鈍化時，不要只靠 BIAS 反向；等 KD 轉折再動作。
- **BIAS + [[技術分析/布林通道Bollinger-Bands三軌八型態|布林通道]]**：價格衝出外軌又出現極端正乖離，適合追蹤停利；跌破下軌又極端負乖離，等收回下軌再試反彈。
- **BIAS + [[技術分析/成交量確認原則Volume-Confirmation|成交量確認原則]]**：量縮上漲的高正乖離比放量上漲更危險；恐慌爆量後負乖離收斂才有止跌味道。
- **BIAS + [[技術分析/背離Divergence進階實戰|背離]]**：價格創高但 BIAS 未創高，動能衰退；價格創低但 BIAS 未創低，跌勢可能減速。

### 5. 台股選股/看盤用法

1. **強勢股風控**：持有主升段股票時，用 BIAS20 監控是否過熱；正乖離進入歷史前 10% 區間後，提高停利線。
2. **跌深反彈池**：篩 BIAS20/60 低於歷史第 10 百分位、成交量由放大轉縮、價格不再破底的股票。
3. **大盤溫度計**：加權指數 240 日乖離過度負乖離時，通常代表市場悲觀到誇張；但仍要搭配融資餘額、法人籌碼與市場寬度，不是看到低乖離就喊歐印。
4. **避免追高**：短線急漲股若 BIAS6、BIAS12、BIAS20 同時進入高檔區，除非量能持續攻擊，不然追進去通常是替前面的人結帳。

## 注意事項

1. **強趨勢會讓 BIAS 長期極端**：主升段可長期正乖離，主跌段可長期負乖離；過早反向是乖離率最大的坑。
2. **股性差異極大**：高波動小型股正常乖離可能是大型權值股的兩倍以上。
3. **參數不能亂配**：20日BIAS就看20日均線，別拿6日BIAS去判斷季線壓力，這叫跨頻亂鬥。
4. **乖離率是結果，不是原因**：它告訴你價格偏離程度，不告訴你為什麼偏離；原因要從籌碼、基本面或事件催化找。
5. **盤整區容易假訊號**：零軸附近反覆穿越時，BIAS 只是噪音放大器。
6. **必須有停損**：負乖離買進後若續破前低，代表均值回歸假設失效，先跑再說，別跟市場談戀愛。
