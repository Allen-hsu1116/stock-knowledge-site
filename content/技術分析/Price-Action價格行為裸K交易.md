# Price-Action價格行為裸K交易

> 不靠指標，只看價格本身：從市場結構、關鍵水平位和K線型態讀懂多空博弈，是所有技術分析的源頭。

## 核心概念

Price Action（價格行為）是一種僅根據價格本身變動來進行分析和交易的方法，又稱「裸K交易」。核心理念：市場的一切資訊最終都會反映在價格的走勢上，所有技術指標的計算基礎都離不開價量數據，因此掌握了 Price Action，就等於掌握了所有技術分析的源頭。

### 為何 Price Action 是市場的「原點」

- 技術指標（如移動平均線、RSI）是「加工食品」：將原始價格數據通過公式計算，方便但可能失真或延遲
- Price Action 是「生鮮食材」：直接觀察價格的每一個跳動、每一根K線的形態，解讀市場最原始的供需關係
- 日本投資界名言：「市場的事情要問市場」——價格走勢本身就是市場最真實的表達

## 實戰應用

### 三大核心要素

**1. 市場結構（Market Structure）——判斷趨勢方向**

道氏理論為基礎：
- **上升趨勢**：更高的高點（HH）+ 更高的低點（HL）
- **下降趨勢**：更低的高點（LH）+ 更低的低點（LL）
- **盤整區間**：價格在固定高低點之間來回波動

順勢而為是提高勝率的關鍵第一步。相關概念可參考 [[BOS與CHoCH市場結構轉變]] 和 [[SMC聰明錢概念Smart-Money-Concepts]]。

**2. 關鍵水平位（Key Levels）——尋找攻防戰場**

- **支撐位**：價格下跌至此時買盤承接，理解為「地板」
- **壓力位**：價格上漲至此時賣壓增加，理解為「天花板」
- **趨勢線**：連接低點或高點的斜線，提供動態支撐壓力
- **供需區**：機構交易者集中下單的價格區域，通常是劇烈波動的起點

價格行為交易者不隨意進場，耐心等待價格來到「高勝率區域」才尋找信號。詳細支撐壓力技巧見 [[支撐壓力]]。

**3. K線型態（Candlestick Patterns）——識別進場信號**

#### Pin Bar（長影線K線）
- 特徵：實體很小，一根影線是另一方向影線的3倍以上
- 支撐位 + 長下影線 Pin Bar → 買方強力反擊 → 看漲反轉
- 壓力位 + 長上影線 Pin Bar → 看跌反轉

#### Engulfing Pattern（吞噬型態）
- 特徵：第二根K線實體完全包覆前一根，且顏色相反
- 下降趨勢 + 陽線吞噬陰線 → 看漲吞噬 → 強烈買入信號
- 上升趨勢 + 陰線吞噬陽線 → 看跌吞噬 → 強烈賣出信號

#### Inside Bar（內包線）
- 特徵：第二根K線高低點完全被前一根（母線）包覆
- 市場暫時猶豫和盤整
- 操作：等待突破母線高點或低點，作為趨勢延續或反轉確認

更多K線型態見 [[K線型態]] 和 [[K線組合型態反轉訊號]]。

### 四步交易流程

1. **多時框分析**：大週期（日線/4H）判斷市場結構和趨勢方向
2. **識別關鍵水平位**：在大週期圖標示支撐、壓力、趨勢線
3. **等待價格觸及 + 信號出現**：切換到小週期（1H/15min），等待回測關鍵位時出現高勝率K線型態
4. **制定交易計畫**：明確定義進場點、停損點、停利點

相關多時框分析技巧見 [[多時間框架分析]]。

## 注意事項

1. **化繁為簡≠隨便**：Price Action 不是不分析，而是用最直接的方式分析。仍需要嚴謹的市場結構判斷和關鍵位識別
2. **主觀性較高**：不像指標有明確數值，支撐壓力的畫法和K線型態的認定存在主觀差異，需要大量練習建立一致性
3. **需要經驗累積**：裸K交易依賴交易者的經驗和盤感，新手直接上陣容易誤判
4. **搭配成交量更準確**：純看價格有局限，搭配 [[成交量確認原則Volume-Confirmation]] 可以提升訊號可靠度
5. **不適合所有市場**：流動性差或被操縱的個股，價格行為可能失真
6. **趨勢反轉確認需耐心**：不要看到一根Pin Bar就立刻進場，需等待收盤確認或次日開盤驗證
