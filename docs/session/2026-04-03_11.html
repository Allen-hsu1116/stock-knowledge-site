核心知識

夏普比率 (Sharpe Ratio)

是什麼：衡量「承擔單位風險能獲得多少超額報酬」的指標，由威廉·夏普於1966年提出。

公式：

夏普比率 = (投資報酬率 - 無風險利率) / 標準差

怎麼解讀：

  • 夏普值越高越好
  • 數值 0.8 代表：要創造 8% 報酬需承受 10% 波動風險
  • > 1：優秀（承擔1單位風險獲得超過1單位超額報酬）
  • 0.5-1：良好
  • < 0.5：普通
  • < 0：差（報酬低於無風險利率）

實例：

  • 情境：股票年化報酬 8%、無風險利率 2%、標準差 10%
  • 計算：(8% - 2%) / 10% = 0.6
  • 解讀：每承擔 1% 波動風險，獲得 0.6% 超額報酬

適用時機：

  • 選基金時比較績效（同類型基金選夏普值高的）
  • 評估投資組合風險報酬效率
  • 注意：適合低波動投資組合

限制：

  • 用總標準差衡量風險，把「上漲波動」也算成風險
  • 假設報酬呈正態分佈，忽略厚尾風險
  • 基於歷史數據，對未來預測不一定準確

索提諾比率 (Sortino Ratio)

是什麼：「只看虧損風險的夏普比率」，只計算負報酬的標準差，不把上漲波動視為風險。

公式：

索提諾比率 = (期間平均報酬 - 最低可接受報酬) / 下行標準差

怎麼解讀：

  • 數值通常比夏普比率高（因為分母只有下行風險）
  • 越高代表風險調整後的報酬越好
  • 專注於投資人真正擔心的「虧損風險」

和夏普比率的差異：

指標風險計算方式適用情境
夏普比率總標準差（上漲+下跌都算）低波動投資組合
索提諾比率只算下行標準差（只算虧損）高波動、盈虧不對稱策略

適用時機：

  • 勝率低、大賺小賠的策略（如選擇權賣方）
  • 勝率高、小賺大賠的策略
  • 期貨基金績效評估
  • 高波動投資組合

實例：

  • 情境：某策略年化報酬 15%，無風險利率 2%，總標準差 20%，下行標準差 12%
  • 夏普比率：(15% - 2%) / 20% = 0.65
  • 索提諾比率：(15% - 2%) / 12% = 1.08
  • 解讀：若考慮整體風險，每單位風險獲得 0.65% 超額報酬；若只考慮虧損風險，每單位下行風險獲得 1.08% 超額報酬

使用注意：

  • 取樣時間要夠長（包含多頭空頭）
  • 流動性差的資產可能讓數值失真
  • 獲利與虧損樣本都要充足

其他相關績效指標

馬克比率 (MAR Ratio)：

  • 基於風險調整後的最大回撤
  • 公式：年化報酬率 / 最大回撤

斯特林比率 (Sterling Ratio)：

  • 最大回撤與標準差之比

使用建議：

  • 不要只看單一指標
  • 夏普比率 + 索提諾比率搭配使用
  • 注意取樣期間和數據品質

實戰應用：如何用這些指標選基金/策略

**步

步驟一：篩選
1. 用夏普比率初步篩選（> 0.5 為門檻）
2. 同類型基金比較，選夏普值較高的

步驟二：深入分析
1. 若策略波動大，改看索提諾比率
2. 檢查取樣期間（至少 3-5 年，包含多空）
3. 看最大回撤是否可接受

步驟三：風險管理
1. 夏普比率下降時注意（風險增加或報酬下降）
2. 用滾動窗口追蹤變化
3. 結合其他風險指標（VaR、CVaR）

心得：

夏普比率是投資界的老牌子，但它的「上漲也算風險」邏輯確實有問題。對股票操作來說，索提諾比率更貼近現實——我們只在意賠錢的風險，上漲波動那是好事。

不過兩個指標通常是正相關，不用糾結選哪個。日常篩選先用夏普比率（數據好取得），深入分析再看索提諾比率。

最實用的場景是：
1. 基金選擇：同類基金選夏普值高的
2. 策略評估：用索提諾比率評估高波動策略
3. 追蹤變化：夏普比率下滑就是警訊

記得：這些都是「風險調整後」的報酬指標，不是絕對報酬。一個策略夏普值 2.0 但年報酬 5%，另一個夏普值 0.8 但年報酬 20%，要看你的風險承受度選。
