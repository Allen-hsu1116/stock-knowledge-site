一、DMI 是什麼？

DMI（Directional Movement Index）動向指標，由威爾斯·威爾德（J. Welles Wilder）開發——他就是 RSI 指標的發明者。

核心功能：判斷市場趨勢的「方向」和「強度」

適合週期：中長線、波段操作，不適合短線當沖

為什麼重要：很多技術指標只告訴你方向，但不告訴你趨勢夠不夠強。DMI 同時解決這兩個問題。

二、DMI 三條線怎麼看？

線條名稱功能數值範圍
**+DI**上升動向指數上漲趨勢強度0-100
**-DI**下降動向指數下跌趨勢強度0-100
**ADX**平均動向指數趨勢強度（不管多空）0-100

三條線的關鍵意義

+DI（藍線）：代表多頭力量

  • 數值越高，漲勢越明確
  • 下跌趨勢中通常在 20 以下徘徊

-DI（橘線）：代表空頭力量

  • 數值越高，跌勢越明確
  • 上升趨勢中通常在 20 以下

ADX（粉線）：代表趨勢強度

  • 不論多空，只要趨勢強，ADX 就會往上

三、DMI 實戰判讀方法

1. +DI 與 -DI 位置關係（判斷方向）

情況解讀
+DI 在 -DI 上方多方趨勢較強，價格容易上漲
-DI 在 +DI 上方空方趨勢較強，價格容易下跌
+DI 向上穿越 -DI**黃金交叉**，空方轉多方
-DI 向上穿越 +DI**死亡交叉**，多方轉空方

實例：

  • 鴻海（2317）技術圖：+DI 在 -DI 上方時，股價呈上升趨勢
  • 當 -DI 向上穿越 +DI 出現「死亡交叉」，股價反轉向下

2. ADX 數值判斷（確認強度）

ADX 數值趨勢狀態操作建議
< 20盤整，無明確趨勢觀望，或用區間策略
20-25趨勢萌芽開始關注，準備進場
25-30趨勢形成趨勢交易可以進場
30-50趨勢明確順勢持有
> 50趨勢極強（可能過熱）注意反轉風險

重要：ADX 只顯示強度，不顯示方向！多頭和空頭趨勢強時 ADX 都會上升。

3. 三線綜合判斷

多方訊號組合：
1. +DI 向上穿越 -DI（黃金交叉）
2. ADX 同步上升（> 25）
3. 確認：上升趨勢動能增強，可以考慮做多

空方訊號組合：
1. -DI 向上穿越 +DI（死亡交叉）
2. ADX 同步上升（> 25）
3. 確認：下跌趨勢動能增強，可以考慮做空

盤整訊號：
1. +DI 和 -DI 頻繁交叉
2. ADX 在 20 以下徘徊
3. 確認：沒有明確趨勢，避免趨勢策略

四、DMI 背離訊號

頂背離（賣出訊號）

情況：價格創新高，但 +DI 和 ADX 卻下滑
意義：上升趨勢強度正在減弱，可能即將到頂
操作：注意獲利了結時機

底背離（買進訊號）

情況：價格創新低，但 -DID 卻下滑，ADX 停止上升
意義：下跌趨勢強度正在減弱，可能即將反轉向上
操作：注意進場時機

五、公式計算（理解原理即可）

DMI 計算步驟：

1. 計算 DM（動向變動值）
   +DM = 今日最高價 - 昨日最高價（若 < 0 則 = 0）
   -DM = 昨日最低價 - 今日最低價（若 < 0 則 = 0）
   若 +DM > -DM，則 -DM = 0；反之亦然

2. 計算 TR（真實波幅）
   TR = max(今日最高價 - 昨日收盤價,
           昨日收盤價 - 今日最低價,
           今日最高價 - 今日最低價)

3. 計算 DI 值（週期 N 通常為 14）
   +DI = (N 日 +DM 平均值 / N 日 TR 平均值) × 100
   -DI = (N 日 -DM 平均值 / N 日 TR 平均值) × 100

4. 計算 ADX
   DX = |((+DI) - (-DI)) / ((+DI) + (-DI))| × 100
   ADX = DX 的 N 日平均值

參數預設值建議調整
ADX 平滑度14使用預設即可
DI 長度14使用預設即可

七、DMI 優缺點

優點

1. 同時判斷方向和強度：一次看兩件事
2. 適合中長線：波段操作的好工具
3. 圖形直觀：三條線易於判讀
4. 過濾假訊號：搭配 ADX 可以過濾盤整區的假交叉

缺點

1. 落後指標：對價格變化反應較慢
2. 盤整區鈍化：+DI 和 -DI 頻繁交叉，參考價值降低
3. 不適合短線：經過多次平滑，對價格變化不敏感
4. 需要搭配：單用容易誤判，建議搭配 RSI 或 MACD

八、DMI 實戰策略

策略一：趨勢確認進場

做多條件：
1. +DI 向上穿越 -DI
2. ADX > 25 且向上
3. 可搭配股價突破前高確認

做空條件：
1. -DI 向上穿越 +DI
2. ADX > 25 且向上
3. 可搭配股價跌破前低確認

策略二：過濾盤整

避開盤整：

  • ADX < 20 時，+DI 和 -DI 的交叉訊號可能不斷出現
  • 此時趨勢不明，避免使用趨勢策略
  • 可改用區間交易策略（如布林通道）

策略三：背離操作

頂背離賣出：

  • 股價創新高
  • +DI 和 ADX 卻下滑
  • 注意獲利了結

底背離買進：

  • 股價創新低
  • -DI 卻下滑，ADX 停止上升
  • 注意反轉機會

九、DMI 搭配其他指標

搭配 RSI

原因：RSI 對價格變化敏感，DMI 較平滑
用法：

  • DMI 確認趨勢方向和強度
  • RSI 確認超買超賣
  • 例：+DI 穿越 -DI，ADX > 25，RSI > 50 → 做多訊號加強

搭配 MACD

用法：

  • DMI 確認趨勢存在（ADX > 25）
  • MACD 確認動能方向
  • 兩者同向時訊號更可靠

搭配均線

用法：

  • DMI 確認趨勢（ADX > 25）
  • 價格站上/跌破關鍵均線
  • 例：股價突破 20 日均線 + ADX > 25 + +DI > -DI → 做多

十、實戰案例

案例：台光電（2383）

技術圖觀察：
1. ADX 在 20 以下：盤整狀態，+DI 和 -DI 頻繁交叉
2. ADX 向上攀升突破 25：趨勢開始形成
3. +DI 向上穿越 -DI，ADX 同步拉升：多方動能增強
4. 股價開始走高
5. ADX 反轉向下：趨勢動能減弱，進入整理

案例：群創（3481）

技術圖觀察：
1. +DI 位於 -DI 上方：股價呈上升趨勢
2. -DI 向上穿越 +DI（死亡交叉）：股價反轉向下
3. +DI 再度向上穿越 -DI（黃金交叉）：股價重回上升

十一、DMI 使用注意事項

1. 不要單獨使用

DMI 是落後指標，需要搭配其他指標確認。

  • RSI：確認超買超賣
  • MACD：確認動能
  • K 線型態：確認突破
  • 成交量：確認量價

2. 注意 ADX 的方向

不是只看 ADX 數值，要看 ADX 趨勢：

  • ADX 上升：趨勢正在增強
  • ADX 下降：趨勢正在減弱
  • ADX 走平：盤整中

3. 盤整區的假訊號

盤整時 +DI 和 -DI 會頻繁交叉，此時要看 ADX：

  • ADX < 20：忽略 DI 交叉訊號
  • ADX > 25：交叉訊號才可靠

4. 參數調整

  • 短線可調低參數（如 9 日），但雜訊增加
  • 長線可調高參數（如 20 日），但訊號延遲
  • 使用預設 14 日是最常見的選擇

心得

DMI 是一個被低估的指標，很多人只知道 MACD、RSI，但 DMI 的獨特價值在於：它同時告訴你方向和強度。

ADX 的設計很巧妙——不管多空，只要趨勢強就上升。這意味著你可以用它來過濾假突破：當 ADX < 20 時，股價突破可能是假的；當 ADX > 25 且上升時，突破才可能是真的。

DMI 最適合波段操作，因為它的設計本來就是抓趋势。當沖用這個指標會非常痛苦，因為計算經過多次平滑，反應太慢。

核心心法：

  • +DI/-DI 告訴你「往哪走」（方向）
  • ADX 告訴你「走得有多快」（強度）
  • 兩者結合，才能判斷「值不值得跟」

這週（技術分析週）學到的 DMI 指標，加上之前學的 KD、MACD、布林通道、趨勢線，整個技術分析的武器庫越來越完整了。

一、交易對手風險核心概念

什麼是交易對手風險？

定義：交易的另一方不履行合約義務的風險，屬於信用風險的一種，也稱為違約風險。

白話說明：就像玩牌或下棋，就算你技術再好、勝算再大，對手可能輸了不認帳或掀桌子。這種風險與投資內容好壞無關，是來自對手的信用問題。

為什麼重要？

2008年雷曼兄弟事件：

  • 雷曼曾是美國第四大投資銀行
  • 全球員工超過26,000人
  • 發生信用危機後投資人血本無歸

二、常見交易對手風險類型

金融商品風險高低排序

商品風險情境風險程度
銀行定存金額超過存款保險且銀行倒閉低（有存款保險）
保險公司保單保險公司倒閉無人承接中低

  • 健身房會員：健身房倒閉，會費拿不回來
  • 公司應收帳款：客戶跳票收不回來
  • 太陽能電板投資：政府不再承諾購電

報酬與風險的對價關係

核心邏輯：風險越高，報酬越高

高收益債（垃圾債券）
↑ 高風險 = 高報酬

實例：

  • 信用評級差的借款人 → 銀行收取較高利率
  • 信用評級好的借款人 → 銀行收取較低利率

降低風險五大原則

1. 與大型安全的金融機構往來

  • 銀行、保險選大品牌

2. 選擇大品牌券商

  • 全球排名靠前的優先

3. 選擇規模大的金融產品

  • 不碰冷門或客制化商品

4. 債券投資選擇信評良好標的

  • 不為報酬率犧牲信用安全

5. 評估高報酬投資的對手風險

  • 先想：對手是誰？會不會違約？

五、中央對手方（CCP）機制

CCP是什麼？

中央對手方（Central Counterparty, CCP）：清算所擔任所有交易者的對手方，買方的賣方、賣方的買方。

傳統模式：A ←→ B（雙方直接交易）
             ↓
CCP模式：    A ←→ CCP ←→ B
             （CCP在中間承接風險）

CCP四大功能

功能說明
合約更替CCP成為所有交易的法定對手方
擔保交收任一方違約，CCP先賠償另一方
淨額清算多筆交易軋差，減少資金移動
保证金制度防範對手方風險

CCP的優點

1. 匿名交易：不用擔心對手是誰
2. 風險重新分配：解決連鎖違約問題
3. 清算效率提升：多邊淨額清算
4. 監管透明：掌握交易結算數據

CCP的問題

問題說明
道德風險交易商可能擴大風險，反正CCP會擔保
大而不能倒CCP本身成為系統性風險來源
順周期效應市場波動時要求追加保证金，加劇波動
跨境監管困難各國監管標準不一

2008年金融危機後的改革

G20決議：

  • 標準化OTC衍生品必須通過CCP清算
  • 提高透明度
  • 加強風險管理

選擇券商的標準

優先考量：
1. 全球排名前20
2. 受多國監管
3. 資本充足率高
4. 歷史悠久（經歷過金融危機）
5. 投資者保護基金涵蓋

避開：

  • 沒聽過的小型券商
  • 只在一國註冊
  • 資本規模小
  • 優惠過度誇張

投資級債券（BBB以上）：

  • 違約率 < 0.5%
  • 適合保守型投資人
  • 報酬率較低

  • 違約率 3%~10%
  • 需要分散投資
  • 注意信用評級變化

場內交易（交易所）：

  • 有CCP擔保
  • 違約風險低

場外交易（OTC）：

  • 雙邊清算風險
  • 需要評估對手信用
  • 避免單一對手集中

心得

交易對手風險是很多人忽略的隱形風險。大家常關注「這檔股票會漲嗎？」卻很少問「和我交易的對手會不會違約？」

市場先生的比喻很生動——玩牌時對手可能輸了不認帳甚至掀桌子。這種風險和你的技術無關，和對手的品格有關。

學到幾個重要觀念：

1. 「信用就是錢」：信用評級不是虛的，它決定了資金成本和借貸能力
2. 風險溢價的本質：高報酬 = 高風險的補償，不是免費午餐
3. CCP不是萬能：中央清算所解決了雙邊違約風險，但自己變成「大而不能倒」的系統性風險
4. 監管一直在追風險：2008年後大力推CCP清算，但問題只是被轉移，不是消失

對股票操作的啟發：

  • 選擇交易平台時，優先選大型、有信譽的券商
  • 投資債券時，信用評級比票面利率更重要
  • OTC衍生品要特別小心對手風險
  • 融資融券時要看券商的財務狀況

一、交易日誌四步驟框架

1. 買進後記錄

是什麼：完成交易後立即記錄完整資訊，作為日後檢討基礎。

怎麼做：
1. 記錄買進日期、張數、股票名稱
2. 寫下買進理由（關鍵：必須是「符合策略條件」，而非「覺得會上漲」）
3. 截圖技術分析圖型，標註：停損點、買點、支撐壓力、技術判斷

實例：

  • 情境：台積電 2330
  • 買進理由：「股價突破前高 650 元，成交量放大，5 日均線穿越 20 日均線，符合突破策略」
  • 停損設在 635 元（-2.3%）
  • 截圖標註：突破點、頸線、成交量

為什麼重要：

  • 「我覺得會上漲」vs「符合買點才買」是本質差異
  • 主觀預測容易賠錢，機械化執行才是成功之路

2. 持有期間規則設定

是什麼：下單後立即制定賣出規則，避免盤中情緒干擾。

怎麼做：
1. 時間停損：「在 X 月 X 日，如果價格沒到 X 元，我就賣出」
2. 空間停損：百分比停損法（如 -5%）、均線停損法（跌破 20MA）
3. 計算最大可能虧損（預留跳空風險）
4. 設定看盤時間：開盤 9:00-9:30？尾盤 13:00-13:30？盤中逢高？
5. 停利點：帳面獲利超過 X 元或到達 X 價位
6. 加碼點：穿越 X 價格或帳面獲利 X 元再加碼
7. 極端狀況對策：「大盤暴跌千點，我的 Plan B 是什麼？」

適用時機：每次下單後立刻設定，不要盤中思考

實例：

  • 買進台積電 650 元
  • 時間停損：持有最長 10 個交易日
  • 空間停損：跌破 635 元（-2.3%）立即出場
  • 最大虧損預估：設 620 元為極端停損（-4.6%）
  • 停利：到達 700 元獲利了結
  • 加碼：突破 670 元加買一張

3. 賣出時記錄

是什麼：記錄實際出場資訊，與買進理由對照。

怎麼做：
1. 記錄賣出時間、股價、張數
2. 計算獲利/虧損金額與百分比
3. 寫下賣出理由（與買進理由對照）
4. 截圖技術線型，標註賣出點位

實例：

  • 買進理由：「突破前高」
  • 賣出理由：「跌破停損 635 元」→ 理由一致，執行正確
  • 若賣出理由變成「覺得會跌」→ 理由不一致，需檢討

關鍵觀念：

  • 寧願錯過或少賺，都不應該成為盤中主觀判斷者
  • 規則是死的？不，規則可以透過檢討修正，但盤中必須遵守

4. 月度檢討（最重要！）

是什麼：過一個月後再回頭檢討，善用馬後炮找出盲點。

怎麼做：
1. 賣得太早？買得太晚？
2. 策略哪裡需要改進？
3. 停損時間該提早或延後？
4. 停損方式調整？更寬或更窄？
5. 買進與賣出理由一致嗎？
6. 調整每次下的資金比例？
7. 勝率多少？
8. 觀察自己的個性，調整買的方式
9. 持股有效率嗎？
10. 賺錢了，行為做對嗎？（賺錢 ≠ 行為正確，可能是運氣）
11. 賠錢了，行為做錯嗎？（賠錢 ≠ 行為錯誤，需具體分析）

實例：

  • 檢讨發現：10 次交易中有 3 次「賣在最低點，反彈就飛走」
  • 原因：停損設太窄（-3%），遇到正常波動就被洗出
  • 改進：將停損從 -3% 放寬到 -5%，或採用「兩根 K 棒跌破才停損」

二、交易倦怠與心態恢復

連續虧損後的心理調適

是什麼：連續虧損後最容易陷入「報復性交易」，導致更大虧損。

怎麼做：
1. 接受虧損是成本：將虧損視為交易的必要成本，而非失敗
2. 設定每日最大虧損上限：如單日虧損超過 2% 就停止交易
3. 交易冷卻時間：連虧 3 筆後強制休息一天
4. 單筆風險控制 1-2%：永不讓單筆虧損超過總資金的 1-2%
5. 交易前情緒檢測三問：

  • 我現在是否冷靜？

實例：

  • 連輸 5 筆，情緒開始浮躁
  • 當下反應：「再下一筆贏回來！」→ 這是報復性交易前兆
  • 正確做法：關掉交易軟體，出門散步，明天再戰

交易倦怠恢復方法

是什麼：長期交易後精神耗盡，判斷力下降，需要主動恢復。

怎麼做：
1. 承認倦怠是正常的：連續交易數週/數月後，倦怠是生理反應
2. 設定交易日誌情緒記錄：每天記錄心情（1-10 分），連續 3 天低於 5 分就休息
3. 強制休假機制：每獲利 X% 或每交易 X 天，強制休息一天
4. 培養交易外興趣：運動、閱讀、社交，讓大腦有其他刺激
5. 減少交易頻率：從日內交易降級為波段，降低心理壓力

實例：

  • 交易日誌心情連續一週都在 4 分以下
  • 症狀：不想看盤、進場隨便、停損不執行
  • 解方：強制暫停交易一週，轉做長線或研究，等心情回到 7 分以上

每天保持積極態度

是什麼：連續虧損後保持積極，是新交易不被負面情緒影響的關鍵。

怎麼做：
1. 每天用積極態度面對市場，即使前一天虧損
2. 拋開自我意識：接受出錯在所難免，虧損交易可能比盈利多
3. 不要為了交易而交易：市場形勢不對就等待，連續兩週不做交易也沒關係
4. 避免沮喪情緒：交易是漫長旅途，不是快速致富之路

適用時機：每天開盤前、連續虧損後、重大虧損後

三、交易檢討框架

交易檢討記錄範本

是什麼：標準化檢討格式，確保每次檢討都有價值。

怎麼做：
記錄以下欄位：

  • 交易日期
  • 股票名稱/代碼
  • 交易策略名稱
  • 進場點位、出場點位
  • 部位大小
  • 進場原因（技術面/基本面/市場情緒）
  • 出場原因（停損/停利/技術信號改變）
  • 交易結果（獲利/虧損金額）
  • 檢討結果：

實例：

交易日期：2024-06-13
股票名稱：台積電 2330
交易策略：均線發散策略
進場點位：$650.00
出場點位：$635.00（停損出場）
部位大小：1 張
進場原因：5MA 穿越 20MA，放量突破
出場原因：跌破停損 635 元
交易結果：虧損 $1,500
檢討結果：
- 失敗因素：進場時大盤弱勢，忽略大盤風險
- 改進建議：未來先看大盤方向，大盤弱勢則不進場

看錯但行情回來的處理

是什麼：停損後行情又回來，最容易產生「早知道不要停損」的後悔。

怎麼做：
1. 停損是對的：接受停損是交易成本，而非錯誤
2. 觀察市場：停損後繼續監控，確認原策略是否仍適用
3. 重新進場：若市場重新符合策略，可以低風險點再進場
4. 調整停損距離：根據波動性優化停損位置，避免頻繁被洗出

實例：

  • 停損價 635 元，被洗出後股價跌到 630 元再反彈到 680 元
  • 錯誤心態：「早知道不要停損！」
  • 正確心態：「停損是成本。現在價格回來了，符合進場條件，可以再進場。」

四、賺賠心態管理

核心觀念

賺錢 ≠ 行為正確：可能是運氣
賠錢 ≠ 行為錯誤：可能是機率波動

怎麼做：
1. 獲利時：檢討行為是否符合策略，而非慶祝
2. 虧損時：檢討行為是否遵守紀律，而非自責
3. 設目標：年度目標與大盤比較，月度績效詳細記錄
4. 獎勵進步：適當獎勵盈利，但勿揮霍資本
5. 激勵學習：賠錢時當作學習機會，避免負面情緒

實例：

  • 年度目標：報酬率 12%，大盤 10% → 成功跑贏市場
  • 賺賠比：平均獲利 $2,000，平均虧損 $1,000 → 賺賠比 2:1
  • 勝率 40% 但賺賠比 2:1 → 期望值仍為正

五、成功交易者七步驟

培養紀律的完整框架

怎麼做：
1. 記錄所有行為：買進、持有、賣出全程記錄
2. 制定明確規則：停損、停利、時間點具體化
3. 持續檢討改進：每月檢討，修正規則
4. 接受虧損是成本：不追求 100% 勝率
5. 減少情緒影響：縮小部位，降低心理壓力
6. 保持交易一致性：專注特定市場／策略，不隨波逐流
7. 設定漸進目標：每年提升 10% 獲利能力，逐步成長

心得

交易日誌是 90% 投資人做不到的事，但這恰恰是贏家與輸家的分水嶺。

關鍵不在於「記錄」，而在於「紀律」。多數人可以記錄一週，做不到持續一年。但量變引發質變，持續寫交易日誌 1-2 年，大量經驗沈澱，才能在某個瞬間頓悟進入下一個層次。

週日補強這塊特別重要。風險管理不只是停損停利，心理建設和交易日誌才是支撐整個系統的地基。地基不穩，再好的策略都會被情緒摧毀。

  • 市場中性策略
  • 配對交易
  • Beta 中性
  • 均值回歸

核心知識

統計套利基本概念

是什麼：統計套利（Statistical Arbitrage / StatArb）是利用資產之間的統計關係和長期均衡，進行同時做多和做空操作，期待價格回歸正常水平並從中獲利的策略。

核心原理：

  • 當兩個相關資產的價格關係偏離長期均衡時，就會產生交易機會
  • 價格最終會回歸均值（均值回歸現象）
  • 不是無風險套利，是基於統計機率的交易

與高頻交易的差異：

  • 統計套利屬於「中頻策略」
  • 持倉時間從幾小時到幾天
  • 不是毫秒級的高頻交易

市場中性策略 Market Neutral

是什麼：市場中性策略是統計套利的一種，目標是讓投資組合的 Beta 值接近 0，完全消除市場系統性風險。

怎麼做：
1. 多空部位金額相等：做多了多少金額，就做空多少金額
2. Beta 相等：確保多空部位的 Beta 相同或相近
3. 賺取 Alpha：多空對沖後的報酬就是超額報酬

實例：

  • 情境：投資組合 100 萬元
  • 操作：50 萬做多台積電（Beta 1.2），50 萬做空聯電（Beta 1.2）
  • 結果：大盤漲跌 1%，組合理論上不受影響，只有多空部位的 Alpha 差異產生報酬

優點：

  • 不受大盤漲跌影響
  • 空頭市場也能獲利
  • Sharpe Ratio 通常較高

缺點：

  • 報酬率較低（要跟市場對沖）
  • 交易成本很重要（來回都要交易費）
  • 極端事件風險大
  • 實際操作難度高

四種套利類型

1. 市場中性套利

  • 同時持有多空部位
  • Beta 近似為零
  • 從價格差異中獲利

2. 跨市場套利

  • 同一資產在不同市場有價差
  • 低買高賣賺取差價
  • 實例：台積電 ADR vs 台股現貨

3. 跨資產套利

  • 金融資產與標的資產的價差
  • 實例：期貨與現貨價差
  • 當價差偏離合理範圍時進場

4. ETF 套利

  • ETF 價格與淨值差異
  • 當折溢價過大時套利
  • 需要申購贖回機制

配對交易實戰步驟

是什麼：配對交易是最常見的統計套利方式，找出兩檔高度相關的股票，當價差偏離時進場。

怎麼做：

步驟一：選擇配對股票

  • 同產業或類似的公司
  • 營運模式相近
  • 歷史價格高度相關
  • 實例：百事可樂 vs 可口可樂、台積電 vs 聯電

步驟二：計算價差 Spread

價差 = 股票A價格 - (對沖比率 × 股票B價格)
  • 對沖比率用線性回歸計算
  • 確保價差序列平穩

步驟三：平穩性檢定（ADF 檢定）

  • 使用 Augmented Dickey-Fuller test
  • 檢定價差是否為平穩序列
  • p-value < 0.05 才能使用

步驟四：設定進出場標準

  • 進場：價差超過 ±2 標準差
  • 出場：價差回到均值
  • 停損：價差持續擴大超過 ±3 標準差

實例：

  • 情境：LAD 和 TTM 兩檔汽車股
  • 觀察：兩股價格歷史上高度相關
  • 進場：當價差偏離 2 標準差
  • 出場：價差收斂回均值

Z-Score 計算與應用

是什麼：Z-Score 衡量當前價距離均值多少個標準差。

公式：

Z-Score = (目前價差 - 均值) / 標準差

怎麼用：

  • Z > 2：價差顯著偏高 → 做空價差（賣A買B）
  • Z < -2：價差顯著偏低 → 做多價差（買A賣B）
  • Z ≈ 0：價差接近均值 → 平倉出場

統計套利三大風險

1. 執行風險

  • 價差可能不會回歸
  • 均值回歸假設失效
  • 模型判斷錯誤導致虧損

2. 模型風險

  • 統計關係可能改變
  • 過去相關不代表未來相關
  • 需要持續監控與更新

3. 競爭壓力

  • 統計套利策略普及
  • 利潤空間被壓縮
  • 需要更快的執行速度

統計套利 vs 傳統套利

特性統計套利傳統無風險套利
風險有風險無風險（理論上）
持倉時間幾小時到幾天幾秒到幾分鐘
報酬來源均值回歸價格差異
機會頻率較多較少
競爭程度中等非常激烈

適用時機

適合使用統計套利：

  • 資產有明確歷史相關性
  • 價差偏離均值超過 2 標準差
  • 能忍受短期浮虧
  • 有能力進行多空操作

不適合使用統計套利：

  • 單一資產面臨特殊事件
  • 相關性被破壞（如政策變化）
  • 無法做空或成本過高
  • 市場極端波動期間

多/空股票策略 130/30

是什麼：多空策略的一種，多頭佔 130%、空頭佔 30%。

怎麼做：
1. 用 100% 資金做多看好的股票
2. 額外借 30% 資金做空看壞的股票
3. 多空搭配降低市場風險

與市場中性的差異：

  • 130/30 有淨多頭部位（100%）
  • 市場中性多空完全對沖（50/50 或 100/100）
  • 130/30 報酬較高但風險也較高

實務注意事項

1. 交易成本是關鍵：套利利潤通常很薄，來回交易費可能吃掉獲利

2. 需要高速執行：價差機會稍縱即逝，執行速度很重要

3. 持續監控相關性：兩資產的歷史關係可能改變

4. 資金規模：大資金才能有效分散多檔配對

5. 停損設定：價差不回歸時要有出場機制

心得

統計套利和市場中性策略是機構投資人的重要工具，但對散戶來說實用性有限。主要原因：

1. 交易成本太高：散戶手續費高，來回吃掉大部分利潤
2. 資金規模不夠：無法同時操作多檔配對分散風險
3. 執行速度劣勢：與機構比沒有速度優勢

但這些概念仍然值得一學：

  • 了解套利機制有助於理解市場運作
  • 均值回歸概念可以應用到其他策略
  • Beta 中性的概念對風險管理很重要

對散戶來說，更實際的應用可能是：

  • 不做純套利，但用配對交易的邏輯篩選股票
  • 關注相關股票的價差變化，尋找相對便宜的標的
  • 理解機構操作邏輯，避免成為套利的受害者

記住：市場上沒有穩賺不賠的策略，套利也不例外。價差不回歸、相關性被破壞、極端事件都可能造成虧損。

第 5 回合學習 - 交易系統設計

學習時間：05:04

文章：

  • 什麼是交易系統？如何打造一套勝利的交易系統？
  • 建立一套自已專屬的交易系統

涵蓋主題：

  • 交易系統基本概念
  • 交易系統六大組成要素
  • 交易規則設計實戰
  • 回測驗證
  • 系統交易 vs 主觀交易
  • 交易系統優化方法
  • 成功交易者心態與習慣

核心知識

交易系統基本概念

是什麼：一套規則和程序的組合，決定什麼時候買、什麼時候賣，基於市場分析、歷史數據和技術指標制定的策略。

為什麼需要交易系統：

  • 人是感性的，容易受情緒影響
  • 系統幫助客觀評價投資標的合理價位
  • 避免追高殺低的壞習慣
  • 控制投資風險，適當分散配置
  • 回饋投資績效，穩固投資信念

怎麼做：
1. 先檢視自己的投資性格（風險承受度、時間週期）
2. 建立規則化的進出場條件
3. 嚴格執行，不受情緒干擾

交易系統六大組成要素

是什麼：完整交易系統需要涵蓋六大核心要素，缺一不可。

怎麼做：

1. 市場選擇（買什麼）

  • 選擇你熟悉的市場（台股、美股、期貨、選擇權）
  • 建立適合自己投資屬性的資源池
  • 定期汰換資源池個股

實例：

  • 情境：波段交易者
  • 操作：選擇台股50成分股作為核心池，每月檢視一次
  • 適用時機：有時間研究基本面者

2. 進場訊號（何時買）

  • 明確定義買入條件
  • 可以是技術訊號、基本面訊號或籌碼訊號

實例：

  • 情境：均線突破策略
  • 操作：20日均線黃金交叉60日均線時進場
  • 台積電案例：2024年1月週K圖月季線黃金交叉，進場買入

3. 出場訊號（何時賣）

  • 可以是獲利出場或停損出場
  • 必須事先設定，不能臨場決定

實例：

  • 情境：均線策略出場
  • 操作：20日均線死亡交叉60日均線時出場
  • 適用時機：波段持有，不適合當沖

4. 停損設定（怎麼停）

  • 保護資本的最重要一環
  • 設定在進場當下，不是虧損後才想

實例：

  • 情境：突破策略停損
  • 操作：買入後停損設在突破K棒的低點
  • 風險控制：單筆交易風險不超過總資金的2%

5. 部位管理（買多少）

  • 決定每筆交易投入多少資金
  • 避免單一交易造成重大傷害

實例：

  • 情境：總資金100萬
  • 操作：單筆交易風險2%，停損5%，則部位 = 2% ÷ 5% = 40萬
  • 進階：根據波動度調整部位大小

6. 心態與紀律（怎麼執行）

  • 嚴格遵守自己設定的規則
  • 持續學習，不斷改進

實例：

  • 情境：連續虧損三次
  • 操作：暫停交易一週，檢討策略是否失效
  • 適用時機：任何交易系統都需要搭配心態管理

交易規則設計實戰

是什麼：清晰且易於執行的具體規則，指導何時進出場。

盤整區間突破策略範例：

定義盤整區間：股價在相對狹窄範圍內波動一段時間，買賣雙方力量大致平衡。

進場規則：

  • 買入：價格突破盤整區間上限（買方力量占優）
  • 賣出：價格跌破盤整區間下限（賣方力量較強）

進階優化（回調確認法）：
1. 等待突破後回調
2. 確認原盤整區間上緣成為支撐
3. 在支撐附近買入

怎麼做：
1. 識別盤整區間：至少10個交易日，波動範圍小於10%
2. 等待突破：價格收盤突破區間上緣
3. 回調確認：價格拉回測試上緣不破
4. 進場買入：在支撐確認後進場
5. 設定停損：設在突破K棒低點

實例：

  • 情境：某股票在100-105元盤整兩週
  • 操作：突破105元後，回調至105元附近獲支撐，進場
  • 停損：設在102元（突破K棒低點）
  • 適用時機：波段操作，不適合極短線

優點：

  • 過濾假突破訊號
  • 提供更好的進場價格
  • 停損點更接近進場點，提高賺賠比

回測：驗證系統有效性

是什麼：用歷史數據檢驗交易策略是否有效。

兩種方式：

1. 人工回測

  • 查看歷史K線圖，模擬交易系統操作
  • 記錄超過100次交易的勝率、最大回撤、賺賠比
  • 缺點：耗時、容易受主觀判斷影響
  • 適用：複雜策略如SMC

2. 程式化回測

  • 將策略寫成程式碼
  • 自動在大量歷史數據上運行
  • 優點：快速、精確、可調整參數
  • 適用：規則明確的策略

怎麼做：
1. 定義進出場規則（可程式化）
2. 準備歷史數據（至少3年以上）
3. 執行回測，記錄每筆交易
4. 計算關鍵指標：勝率、賺賠比、最大回撤、夏普值
5. 評估是否值得實戰

實例：

  • 情境：均線交叉策略回測
  • 操作：使用台股10年日K數據，測試20/60日均線交叉
  • 可能結果：勝率45%，賺賠比1.8，最大回撤25%

注意：過去績效不代表未來，但回測是評估策略的必要起點。

系統交易 vs 主觀交易

系統交易（Systematic Trading）：

  • 完全依靠演算法或規則
  • 所有買賣訊號由程式或明確條件觸發
  • 交易者不需臨場判斷
  • 優點：紀律性強、避免情緒干擾
  • 缺點：特殊行情下可能僵化

主觀交易（Discretionary Trading）：

  • 依靠交易者經驗、判斷力與市場感覺
  • 靈活應對突發狀況
  • 優點：彈性大，可臨機應變
  • 缺點：容易受情緒影響，紀律難維持

比較表：

面向主觀交易系統交易量化交易
決策方式經驗、直覺、盤感明確規則數學模型、程式
規則化程度低中高極高
可否回測幾乎無法可以大規模回測
情緒影響高低幾乎零
技術需求低中高
適合的人盤感強、反應快喜歡紀律喜歡數據、程式

關鍵區別：

  • 系統交易重點在「紀律規則化」
  • 量化交易 = 系統交易 + 數學建模/程式化

實務建議：很多專業交易者會混合使用，以系統交易做核心，再搭配主觀判斷調整部位。

交易系統優化方法

是什麼：根據回測和實際交易結果，持續調整系統以適應市場變化。

調整時機：

  • 連續出現虧損
  • 表現明顯低於預期
  • 市場條件發生顯著變化
  • 發現更有效的技術指標

優化方向：
1. 更改入場條件（更嚴格或更寬鬆）
2. 調整出場策略（停利停損）
3. 引入新的過濾條件
4. 調整部位管理規則

注意陷阱：

  • 避免過度優化（Overfitting）
  • 參數不要調到完美符合歷史數據
  • 樣本外測試驗證

怎麼做：
1. 記錄每筆交易的理由和結果
2. 月度檢討：哪些成功、哪些失敗
3. 識別模式：什麼情況下策略失效
4. 調整規則：只改變一個變數
5. 重新回測驗證

成功交易者的心態與習慣

持續學習：

  • 市場是動態環境，不斷變化
  • 更新知識庫和技能
  • 學習新策略、新工具
  • 深入市場心理學

自律與耐心：

  • 設立明確規則並嚴格遵守
  • 不讓情緒影響交易決策
  • 等待完全符合策略的機會
  • 耐心等待進場點和結果

實例：

  • 情境：看到強勢股大漲想追
  • 自律操作：檢查是否符合進場條件，不符合就等待
  • 不自律操作：追高買入，結果被套

心得

這回合學了「交易系統設計」，這是一個整合性的主題。之前學了很多技術分析、籌碼面、風險管理，但這些都只是零件；交易系統就是要把這些零件組裝成一輛能開的車。

最關鍵的體悟是：沒有交易系統，再好的技術分析都沒用。你可以會看K線、會判斷籌碼、懂風險管理，但如果進場靠感覺、出場看心情、停損捨不得，那最後還是賠錢。交易系統就是把所有知識「規則化」，讓執行變得像刷牙一樣自動化。

幾個重點：
1. 六要素缺一不可：市場選擇、進場訊號、出場訊號、停損設定、部位管理、心態紀律。少一樣就不是完整系統。
2. 用盤整突破舉例很清楚：盤整區間突破後的回調確認法，既過濾假突破，又提供更好進場點。這個策略本身就很完整了。
3. 回測是驗證的關鍵：不要憑感覺認為策略有效，要實際用歷史數據驗證。100次以上才有參考價值。
4. 系統 vs 主觀不是對立：很多專業交易者是混合使用。核心用系統，關鍵時刻用主觀判斷調整。
5. 系統是活的：市場會變，系統也要優化。但優化不是每虧一次就改規則，而是要有系統性的檢討機制。

最後一句話總結：交易系統就是把「我知道」變成「我做到」的橋樑。知道均線黃金交叉是買點沒用，要寫成規則、設定停損、管理部位、嚴格執行，才是交易系統。

第 6 回合學習 - 類股輪動判斷與主流類股掌握

學習時間：06:14

文章：

  • 從類股輪動，找出主流強勢股的標準流程
  • 類股輪動是什麼？資金將流回電子股？
  • 強勢股是什麼？深入解析強勢股的五大特徵、篩選方法與投資策略

涵蓋主題：

  • 類股輪動
  • 相對強度（RS）
  • 主流類股判斷
  • 板塊趨勢分析

一、類股輪動基本概念

是什麼：類股輪動是指市場資金在不同產業板塊之間輪流移動的現象，某個產業漲到一定程度後，資金轉向另一個產業繼續漲。

核心邏輯：

  • 資金效率最大化：主力（法人、外資）無法同時拉抬所有股票，需要輪動來讓資金更有效運用
  • 景氣循環驅動：不同產業在經濟週期不同階段表現不同
  • 消息面與題材：政策、事件驅動特定類股上漲

類股 vs 概念股差異：

  • 類股：有統一客觀定義，以產業類別分類（如電子、金融、鋼鐵、營建）
  • 概念股：偏向主觀認定（如蘋果概念股、電動車概念股、AI 概念股）

二、類股輪動由誰決定？

主力操盤：類股輪動主要由主力資金（外資、投信、自營商）決定，不是散戶。

觀察重點：

  • 法人籌碼流向：三大法人買賣超方向
  • 消息面同步：法人研究報告、產業新聞、報價變化
  • 同行共識：同業法人選股標準相似，會有同步化現象

三、如何判斷主流類股？（相對強度法）

核心方法：找比大盤強的強勢類股

判斷標準：
1. 大盤漲時更強：大盤漲 1%，該類股漲 5% 以上
2. 大盤跌時抗跌：大盤跌時跌幅小於大盤
3. 均線多頭排列：類股線型必須也是多頭排列，不能只是跌深反彈

相對強度指標（RS, Relative Strength）：

  • 衡量個股或類股相對於大盤的表現強弱
  • 計算方式：個股漲跌幅 ÷ 大盤漲跌幅
  • RS > 1：跑贏大盤（強勢）
  • RS < 1：落後大盤（弱勢）

四、找出主流類股的標準流程

步驟一：觀察類股排行

  • 使用看盤軟體查看「類股漲跌幅排行」
  • 找出當日、近一週、近一月的強勢類股

步驟二：確認線型結構

  • 該類股是否均線多頭排列？
  • 是否突破整理區間？
  • 是否量價配合（上漲量增、下跌量縮）？

步驟三：比較大盤強度

  • 大盤多頭時，類股是否更強？
  • 大盤空頭時，類股是否抗跌？
  • 相對位置是否合理（不能是跌深反彈）

步驟四：篩選類股內強勢股

  • 在強勢類股中找漲幅最大的前幾檔
  • 確認均線多頭排列
  • 確認法人主力買超
  • 確認籌碼集中（大戶持股增加）
  • 確認基本面成長（營收、毛利率、營業利益率）

步驟五：剔除不良標的

  • 營收沒成長的剔除
  • 無法人主力照顧的剔除
  • 籌碼散亂的剔除

五、類股輪動實戰觀察重點

量價關係判斷：

  • 上漲量增：健康上漲，買盤積極
  • 上漲量縮：可能上漲無力，注意
  • 下跌量增：賣壓沉重，可能轉弱
  • 下跌量縮：正常回檔，可觀察

類股轉弱訊號：

  • 類股指數跌破重要均線（20 日、50 日）
  • 類股內強勢股開始補跌
  • 法人賣超連續增加
  • 成交量異常放大但價格不漲（出貨）

類股輪動時間週期：

  • 短線資金輪動：觀察期間約一個月
  • 景氣循環輪動：可能持續數季到數年
  • 注意：類股輪動趨勢不一定持續數月，可能本月強、下月就弱

六、板塊與主流的關鍵觀念

核心洞察：

  • 單一股票大漲：可能是某個主力在炒
  • 一群同性質股票大漲：極可能是一群主力一起在炒

實戰意義：

  • 觀察類股板塊漲跌比看個股漲跌更重要
  • 類股同漲準確性比單一個股高
  • 投資風險相對較低（買整個強勢板塊）

風險分散：

  • 不只買一檔強勢股，可買板塊內 2-3 檔
  • 分散個股風險，保留板塊趨勢紅利

七、類股輪動交易策略

低基期選股法：

  • 在多頭走勢中，選已做出底部型態的類股
  • 法人開始買超但還沒漲太多
  • 風險低：之前沒漲太多，修正時抗跌
  • 可能接棒往上漲

進場時機：

  • 類股相對強度轉強（RS 持續 > 1）
  • 類股突破整理區間
  • 法人買超連續增加
  • 均線多頭排列確認

出場時機：

  • 類股相對強度轉弱
  • 法人賣超連續
  • 類股指數跌破關鍵支撐
  • 成交量異常但價格不漲

八、實例說明

情境：電子類股轉強，想判斷是否為主流

判斷步驟：
1. 比較電子類股指數與大盤指數
2. 大盤漲 1%，電子類股漲 3% → RS = 3（強勢）
3. 確認電子類股均線多頭排列
4. 確認法人買超集中在電子類股
5. 篩選電子類股內強勢個股

注意：

  • 不是所有電子股都會漲，要篩選強勢子產業
  • 例如半導體、IC 設計可能比面板更強

九、季節性類股輪動

電子類股：每年第三季（9-11 月）為傳統旺季，晶片、AI、5G 需求旺

營建類股：配合政策利多、房市週期

金融類股：利率升降、政策開放

原物料類股：配合國際原物料價格週期

防疫生技類股：疫情爆發、政策題材

心得

核心知識

一、主力是誰？六大主力分類

是什麼：主力是指擁有大量籌碼、足以影響短期股價的人或機構

六大主力類型：
1. 政府：國安基金（黑天鵝事件護盤）
2. 外資：影響大盤的主力
3. 投信：法人主力
4. 自營商：券商自營部門
5. 市場派：市場大戶
6. 公司派：公司內部人

實務分類：

  • 操控個股的主力：三大法人與市場大戶，多著墨於小股本股票，可透過券商分點買賣超追蹤
  • 影響大盤的主力：外資、券商、政府基金，特殊情況才會出現

二、主力洗盤五大手法

1. 短線拉回洗盤

是什麼：主升段常見的洗盤方式，追漲停買進後連續壓回4-5天

怎麼做：
1. 觀察第一防守線（長紅K線最低點）
2. 不跌破第一防守線 → 強勢整理，3-6天內拉長紅繼續上漲
3. 跌破第一防守線但守住第二防守線 → 較長時間震盪整理
4. MACD判斷：紅柱縮短再變長（強勢），或紅柱轉綠柱再轉紅柱（震盪）

實例：

  • 情境：佰研（3205）短線拉回守住第一防守線
  • 操作：股價拉回量縮、不跌破第一防守線時進場
  • 時間：3-6天內拉出長紅

適用時機：主升段追漲停後遭遇拉回時判斷是否續抱

2. K線洗盤（上影線洗盤）

是什麼：利用長上影線製造「避雷針」假象嚇跑散戶

怎麼做：
1. 觀察是否守住第一防守線
2. 若守住第一防守線 → 續抱
3. 後續股價突破「突破線」（上影線最高點）→ 新一波行情啟動

實例：

  • 情境：利勤（4426）出現長上影線配合綠色巨量
  • 判斷：看似避雷針，但股價未跌破第一防守線
  • 操作：等股價穿越突破線再確認

適用時機：主升段出現大量長上影線時判斷真偽

3. 起漲前的甩轎

是什麼：最兇猛的洗盤方式，盤整區突然長黑跌破再長紅吞噬

怎麼做：
1. 第一防守線被長黑跌破 → 先出場
2. 觀察後續是否止跌
3. 長紅吞噬長黑最高點 → 買回訊號

實例：

  • 情境：宏佳騰（1599）守住47.3第一防守線後突然長黑跌破
  • 操作：長黑跌破時先出場，不要賭它是甩轎
  • 買回條件：股價突破長黑最高點時買回

風險警示：不要去賭長黑是甩轎，萬一往下噴出會大賠

三、主力買進四種模式

1. 中線波段上漲：階梯式放量突破

是什麼：主力緩慢推高股價的建倉方式

特徵：

  • 與營收、財報、利多消息搭配
  • 技術面呈現箱型區間上漲
  • 每次區間整理平台不重疊
  • 箱底越墊越高、箱頂越墊越高

實例：

  • 情境：主力在換手過程中低買高賣降低持倉成本
  • 價格：箱型整理區間約10-15%波動
  • 時間：每次整理約1-3週

2. 長線波段上漲：量能萎縮、高檔橫盤

是什麼：高殖利率成長股常見的建倉模式

特徵：

  • 低檔爆大量緩慢推升
  • 高檔長期橫盤
  • 橫盤中量能急速萎縮（低於1000張）
  • 多間券商合作買超

實例：

  • 情境：籌碼面徹底洗盤
  • 時間：橫盤可能持續數月
  • 量能：萎縮至1000張以下

3. 底部回升：法人買盤急增、爆量過季線

是什麼：股價經過半年以上整理後的起漲模式

特徵：

  • 基本面充斥中立、利空消息
  • 股價創新低使散戶信心受打擊
  • 最後打壓洗盤創造最佳吃貨條件
  • 法人買超張數或持股比率上升
  • 技術面底部爆新高量起漲

買點判斷：
1. 「法人買超張數或持股比率」增加
2. 「技術面底部爆新高量起漲」

4. 法人認錯回補買盤

是什麼：吃「法人豆腐」的特殊買點

特徵：

  • 法人因績效壓力賣出持股
  • 股價洗盤時用時間掃走沒耐心的人
  • 特定主力券商大力吃貨形成「籌碼換手訊號」
  • 原本賣出的法人發現看錯方向後回補

實例：

  • 情境：突發利空事件、大盤劇烈波動
  • 操作：觀察主力券商逆勢承接
  • 預期：法人發現看錯後買盤回補力道又急又猛

四、出貨 vs 洗盤辨別（五個前兆）

怎麼分辨：洗盤與出貨的關鍵差異

出貨五前兆：

1. 成交量異常放大

  • 股價高位盤整時成交量突然放大但未創新高

2. 股價疲弱震盪頻繁

  • 高位震盪但反彈無法突破前高

3. 量價背離

  • 價格上漲但成交量萎縮

4. 大單頻繁賣出、小單接手

  • 主力拆單出貨

5. 利多消息頻繁但股價不漲

  • 財報亮眼、政策利多卻不推升股價

五、籌碼分析三大面向

1. 三大法人

怎麼做：

  • 觀察「上市外資買賣超金額」
  • 觀察「上市投信買賣超金額」
  • 觀察「上櫃外資買賣超金額」
  • 觀察「上櫃投信買賣超金額」
  • 確認股票是否在排行榜上
  • 確認外資、投信持股比率是否增加

實務：自營商通常略過不看（沒有明顯買賣趨勢）

2. 大戶持股

判斷標準：

  • 高價位股票：400張以上為大戶，50張以下為散戶
  • 中價位股票：1000張以上為大戶，100張以下為散戶

好現象：

  • 大戶持股比率上升
  • 散戶持股比率下降
  • 表示籌碼集中在大戶手上

3. 主力分點

是什麼：大量買賣超的券商分點

怎麼做：

  • 透過看盤軟體觀察特定券商買賣狀況
  • 可觀察1天、1週、30天、90天內的買賣張數
  • 追蹤哪個券商買最多、賣最多

六、實戰應用：跟隨主力的策略

如何判斷買點位置

成本參考：

  • 在主力成本尚未拉開20%處買進
  • 漲勢的80%~90%處獲利了結就是成功
  • 漲勢的50%處賣出也是成功

實例：

  • 情境：主力成本10元，目標20元
  • 買點：12元以下（成本未拉開20%）
  • 賣點：16-18元（漲勢的60%-90%）

出場時機

見好就收原則：

  • 沒有人能從最低點買進、最高點賣出
  • 避開高處不勝寒的環境
  • 把行情看完告一段落即可

心得

主力洗盤這檔事，教科書教的那套「五波上升、三波下降」現在已經不太適用了。台股早已是法人的市場，散戶如果只用技術指標分析很容易被當韭菜割。

最有用的概念是「防守線」：

  • 第一防守線守住 = 短線洗盤3-6天會結束
  • 第二防守線守住 = 較長時間震盪，看MACD綠柱轉紅柱

但要記住：「長黑跌破第一防守線時先出場，不要賭它是甩轎」。這句話很重要，萬一真的是往下噴出，會賠很慘。

另外「量價背離」和「利多不漲」這兩個出貨前兆我之前沒特別注意，以後看到股價在高檔震盪、利多消息頻傳卻不漲，就要警戒是不是主力在倒貨。

核心知識

ETF基本概念與優勢

是什麼：ETF（Exchange Traded Fund，指數股票型基金）是追蹤特定市場指數的一籃子股票組合，結合股票的靈活性與基金的分散性。

為什麼適合散戶：
1. 分散風險：單一股票爆雷可能血本無歸，ETF已分散風險
2. 成本低廉：總費用率極低，有些甚至不到0.3%
3. 交易彈性高：可在股市自由買賣，即時報價
4. 資訊透明：持股組成與比例公開透明
5. 免盯盤：被動投資工具，選對方向不需常盯盤

實例：

  • 投資0050（元大台灣50）＝一次買下台灣市值前50大績優股
  • 包含台積電、聯發科、鴻海等龍頭股
  • 大幅減少「踩雷」機率

ETF配息機制

配息計算公式：

每單位現金股利 × 持有單位數 ＝ 可領取配息金額

實例：

  • ETF發放現金股利2.3元
  • 持有2,000股 → 可獲得：2.3 × 2,000 = 4,600元

海外ETF稅務注意：

  • 美國ETF適用30%股息預扣稅
  • 投資人領取的是扣稅後的實際配息

除息與填息流程

除息（配息當天）：

  • ETF價格自動下調，等於配出現金股利的金額
  • 例：除息前50元，除息1元後開盤價49元
  • 這不是下跌，是配息金額從基金資產扣除

填息（價格回升）：

  • 除息後價格慢慢回到除息前水位
  • 稱為「完成填息」

重要觀念：

  • ETF總報酬＝價差＋配息
  • 不要只看價格變化，要看總報酬

配息頻率選擇

四種配息頻率：

類型適合投資人
月配息需要穩定現金流
季配息平衡現金流與成本
半年配息長期資產累積
年配息追求資本增值

選擇建議：

  • 月配型：適合退休族、需要現金流支付生活開銷
  • 季配型：最常見，定期定額好規劃
  • 年配型：成長型ETF常見，收益再投入

怎麼做：

  • 跌到10%→再加一次
  • 跌到15%→再來一次

策略三：長期持有

怎麼做：

  • 設下「5年不賣、10年才看」的目標

實例：

  • 即使遇到經濟危機、戰爭、疫情

關鍵：時間是風險最好的解藥

問題：話題性強的ETF波動高、持股集中

問題：高殖利率≠高報酬，可能犧牲成長潛力
正確做法：看「總報酬」＝價差＋配息，綜合觀察年報酬率、波動度、成分股調整頻率

一、類股輪動核心觀念

什麼是類股輪動？

定義：類股輪動 = 市場上不同產業輪流成為主流的現象。

  • A產業漲到一定程度 → 換B產業漲
  • 又稱「權值股輪漲」，因為每個類股都有領頭羊（權值股）

為什麼會輪動？
1. 資金效率運作：主力（外資、法人）資金有限，無法同時拉抬所有股票
2. 題材與景氣循環：不同產業在不同時期有各自的利多題材
3. 季節性因素：如電子股旺季在Q3、寒假推食品觀光類股

三、相對強度指標RS實戰

RS是什麼？

定義：Relative Strength，相對強度。表達個股相對於大盤的強弱程度。

公式：RS = 個股價格 / 大盤指數

數值意義：

  • RS上升 = 個股比大盤強（漲得多或跌得少）
  • RS下降 = 個股比大盤弱
  • 年RS 95 = 過去一年表現為全市場前5%

為什麼重要？

  • 找出真正強勢股（漲時比大盤凶、跌時比大盤抗）
  • 不被假象迷惑（大盤跌5%，個股跌2%看起來跌，實際上RS是上升的）

RS計算方式比較

方式公式適用情境
傳統RS個股價格 / 大盤價格長期趨勢分析
RSS個股漲幅% - 大盤漲幅%短期動態分析

實例計算：

  • 台積電 +1.9%，大盤 +0.4%
  • RSS = 1.9% - 0.4% = +1.5%（比大盤強1.5%）
  • 累積RSS上升 + 成交量放大 → 強勢股訊號

RS三大應用

1. 找帶頭領漲股

多頭市場：

  • 找出RS最高的一群股票
  • 在突破點進場

反彈或回升：

  • 指數由反彈轉回升時
  • 領導股會以最快速度脫離底部
  • 它們早已領先打好底部，RS逐漸提高

Mark Minervini實證：
> 「我最終的績效來自於RS 90以上的股票，買進的股票RS最低不小於70。」
>

2. 挑選放空標的

注意：不要隨便空高RS的股票！

  • RS很高 = 有實際強度
  • 未見顯著頭部前，空它的風險大

可放的時機：

  • 明確頭部型態出現
  • 最後一波突破失敗
  • RS開始從高檔下滑

3. 研究市場現況

觀察重點：
1. 高RS股突破是否順利？突破的檔數多嗎？
2. 有多少大型股在高RS名單裡？
3. 強度最高的一群股票走勢是否集體變差？

判斷：高RS股集體變差 → 大盤可能開始轉壞。

1. RS只是濾網，不是買了就等收割的策略

3. 不適合抄底或逆向操作
4. 不特別適合長投或存股（除非一路強到變長期投資）

五、板塊與主流的觀念

核心觀念：

  • 一檔股票大漲 → 可能是某個主力在炒
  • 一群同屬性股票大漲 → 極可能是一群主力一起在炒

結論：

  • 觀察類股板塊漲跌 > 只看個股漲跌
  • 類股同漲準確性比只漲個股來得高
  • 投資風險性也相對低

實戰做法：
1. 先找出強勢類股
2. 在類股中找最強個股
3. 買最好的股票（最強勢、均線多頭排列）
4. 未來較容易上漲
