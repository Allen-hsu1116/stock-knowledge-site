一、DMI 是什麼？

DMI（Directional Movement Index）動向指標，由威爾斯·威爾德（J. Welles Wilder）開發——他就是 RSI 指標的發明者。

核心功能：判斷市場趨勢的「方向」和「強度」

適合週期：中長線、波段操作，不適合短線當沖

為什麼重要：很多技術指標只告訴你方向，但不告訴你趨勢夠不夠強。DMI 同時解決這兩個問題。

二、DMI 三條線怎麼看？

線條名稱功能數值範圍
**+DI**上升動向指數上漲趨勢強度0-100
**-DI**下降動向指數下跌趨勢強度0-100
**ADX**平均動向指數趨勢強度（不管多空）0-100

三條線的關鍵意義

+DI（藍線）：代表多頭力量

  • 數值越高，漲勢越明確
  • 下跌趨勢中通常在 20 以下徘徊

-DI（橘線）：代表空頭力量

  • 數值越高，跌勢越明確
  • 上升趨勢中通常在 20 以下

ADX（粉線）：代表趨勢強度

  • 不論多空，只要趨勢強，ADX 就會往上

三、DMI 實戰判讀方法

1. +DI 與 -DI 位置關係（判斷方向）

情況解讀
+DI 在 -DI 上方多方趨勢較強，價格容易上漲
-DI 在 +DI 上方空方趨勢較強，價格容易下跌
+DI 向上穿越 -DI**黃金交叉**，空方轉多方
-DI 向上穿越 +DI**死亡交叉**，多方轉空方

實例：

  • 鴻海（2317）技術圖：+DI 在 -DI 上方時，股價呈上升趨勢
  • 當 -DI 向上穿越 +DI 出現「死亡交叉」，股價反轉向下

2. ADX 數值判斷（確認強度）

ADX 數值趨勢狀態操作建議
< 20盤整，無明確趨勢觀望，或用區間策略
20-25趨勢萌芽開始關注，準備進場
25-30趨勢形成趨勢交易可以進場
30-50趨勢明確順勢持有
> 50趨勢極強（可能過熱）注意反轉風險

重要：ADX 只顯示強度，不顯示方向！多頭和空頭趨勢強時 ADX 都會上升。

3. 三線綜合判斷

多方訊號組合：
1. +DI 向上穿越 -DI（黃金交叉）
2. ADX 同步上升（> 25）
3. 確認：上升趨勢動能增強，可以考慮做多

空方訊號組合：
1. -DI 向上穿越 +DI（死亡交叉）
2. ADX 同步上升（> 25）
3. 確認：下跌趨勢動能增強，可以考慮做空

盤整訊號：
1. +DI 和 -DI 頻繁交叉
2. ADX 在 20 以下徘徊
3. 確認：沒有明確趨勢，避免趨勢策略

四、DMI 背離訊號

頂背離（賣出訊號）

情況：價格創新高，但 +DI 和 ADX 卻下滑
意義：上升趨勢強度正在減弱，可能即將到頂
操作：注意獲利了結時機

底背離（買進訊號）

情況：價格創新低，但 -DID 卻下滑，ADX 停止上升
意義：下跌趨勢強度正在減弱，可能即將反轉向上
操作：注意進場時機

五、公式計算（理解原理即可）

DMI 計算步驟：

1. 計算 DM（動向變動值）
   +DM = 今日最高價 - 昨日最高價（若 < 0 則 = 0）
   -DM = 昨日最低價 - 今日最低價（若 < 0 則 = 0）
   若 +DM > -DM，則 -DM = 0；反之亦然

2. 計算 TR（真實波幅）
   TR = max(今日最高價 - 昨日收盤價,
           昨日收盤價 - 今日最低價,
           今日最高價 - 今日最低價)

3. 計算 DI 值（週期 N 通常為 14）
   +DI = (N 日 +DM 平均值 / N 日 TR 平均值) × 100
   -DI = (N 日 -DM 平均值 / N 日 TR 平均值) × 100

4. 計算 ADX
   DX = |((+DI) - (-DI)) / ((+DI) + (-DI))| × 100
   ADX = DX 的 N 日平均值

六、DMI 參數設定

參數預設值建議調整
ADX 平滑度14使用預設即可
DI 長度14使用預設即可
ADX 門檻通常設 25-30量化通告建議設 30

MT4/MT5/TradingView 預設都是 14，一般不需要調整。

七、DMI 優缺點

優點

1. 同時判斷方向和強度：一次看兩件事
2. 適合中長線：波段操作的好工具
3. 圖形直觀：三條線易於判讀
4. 過濾假訊號：搭配 ADX 可以過濾盤整區的假交叉

缺點

1. 落後指標：對價格變化反應較慢
2. 盤整區鈍化：+DI 和 -DI 頻繁交叉，參考價值降低
3. 不適合短線：經過多次平滑，對價格變化不敏感
4. 需要搭配：單用容易誤判，建議搭配 RSI 或 MACD

八、DMI 實戰策略

策略一：趨勢確認進場

做多條件：
1. +DI 向上穿越 -DI
2. ADX > 25 且向上
3. 可搭配股價突破前高確認

做空條件：
1. -DI 向上穿越 +DI
2. ADX > 25 且向上
3. 可搭配股價跌破前低確認

策略二：過濾盤整

避開盤整：

  • ADX < 20 時，+DI 和 -DI 的交叉訊號可能不斷出現
  • 此時趨勢不明，避免使用趨勢策略
  • 可改用區間交易策略（如布林通道）

策略三：背離操作

頂背離賣出：

  • 股價創新高
  • +DI 和 ADX 卻下滑
  • 注意獲利了結

底背離買進：

  • 股價創新低
  • -DI 卻下滑，ADX 停止上升
  • 注意反轉機會

九、DMI 搭配其他指標

搭配 RSI

原因：RSI 對價格變化敏感，DMI 較平滑
用法：

  • DMI 確認趨勢方向和強度
  • RSI 確認超買超賣
  • 例：+DI 穿越 -DI，ADX > 25，RSI > 50 → 做多訊號加強

搭配 MACD

用法：

  • DMI 確認趨勢存在（ADX > 25）
  • MACD 確認動能方向
  • 兩者同向時訊號更可靠

搭配均線

用法：

  • DMI 確認趨勢（ADX > 25）
  • 價格站上/跌破關鍵均線
  • 例：股價突破 20 日均線 + ADX > 25 + +DI > -DI → 做多

十、實戰案例

案例：台光電（2383）

技術圖觀察：
1. ADX 在 20 以下：盤整狀態，+DI 和 -DI 頻繁交叉
2. ADX 向上攀升突破 25：趨勢開始形成
3. +DI 向上穿越 -DI，ADX 同步拉升：多方動能增強

技術圖觀察：
1. +DI 位於 -DI 上方：股價呈上升趨勢
2. -DI 向上穿越 +DI（死亡交叉）：股價反轉向下
3. +DI 再度向上穿越 -DI（黃金交叉）：股價重回上升

十一、DMI 使用注意事項

1. 不要單獨使用

DMI 是落後指標，需要搭配其他指標確認。

  • RSI：確認超買超賣
  • MACD：確認動能
  • K 線型態：確認突破
  • 成交量：確認量價

2. 注意 ADX 的方向

不是只看 ADX 數值，要看 ADX 趨勢：

  • ADX 上升：趨勢正在增強
  • ADX 下降：趨勢正在減弱
  • ADX 走平：盤整中

3. 盤整區的假訊號

盤整時 +DI 和 -DI 會頻繁交叉，此時要看 ADX：

  • ADX < 20：忽略 DI 交叉訊號
  • ADX > 25：交叉訊號才可靠

4. 參數調整

  • 短線可調低參數（如 9 日），但雜訊增加
  • 長線可調高參數（如 20 日），但訊號延遲
  • 使用預設 14 日是最常見的選擇

心得

DMI 是一個被低估的指標，很多人只知道 MACD、RSI，但 DMI 的獨特價值在於：它同時告訴你方向和強度。

ADX 的設計很巧妙——不管多空，只要趨勢強就上升。這意味著你可以用它來過濾假突破：當 ADX < 20 時，股價突破可能是假的；當 ADX > 25 且上升時，突破才可能是真的。

DMI 最適合波段操作，因為它的設計本來就是抓趋势。當沖用這個指標會非常痛苦，因為計算經過多次平滑，反應太慢。

核心心法：

  • +DI/-DI 告訴你「往哪走」（方向）
  • ADX 告訴你「走得有多快」（強度）
  • 兩者結合，才能判斷「值不值得跟」
