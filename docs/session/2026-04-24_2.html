風險報酬比×期望值×破產風險

一、風險報酬比（Risk-Reward Ratio, RRR）

### 是什麼 單筆交易中，預期獲利與可能虧損的比例。簡單說就是「每冒1塊錢的風險，有機會賺幾塊錢」。這是風險管理最基礎也最重要的數學工具。

計算公式

風險報酬比 = 潛在報酬 ÷ 潛在風險
           = (目標價 - 進場價) ÷ (進場價 - 停損價)

怎麼做：
1. 根據技術分析確定進場價（例如支撐位買入）
2. 在關鍵支撐下方設定停損價（例如前波低點下方）
3. 根據上方壓力位或型態滿足點設定目標價
4. 代入公式計算，若結果 < 1.5 則放棄交易

實例：

  • 情境：台積電（2330）
  • 進場價：800元（突破前高後回踩確認支撐）
  • 停損價：780元（前波低點下方，風險 = 20元）
  • 目標價：860元（下一個壓力區，報酬 = 60元）
  • 風險報酬比 = 60 ÷ 20 = 3（1:3）
  • 判讀：1:3 是相當健康的比例，值得進場

不同市況的風險報酬比設定

市況建議風報比原因
趨勢行情1:3 ~ 1:4順勢利潤空間大，停損只需設在趨勢反轉關鍵點
盤整行情1:1.5 ~ 1:2價格跑不遠，務實設目標
高波動行情1:2 ~ 1:3停損要放寬避免被洗，報酬也要相應拉高
消息事件1:3 以上小停損搏大獲利，看錯小賠出場

不同交易風格的風報比

風格建議風報比特點
剝頭皮1:1 ~ 1:1.5靠高勝率+高頻率獲利
當沖1:1.5 ~ 1:3當日完成，風報比均衡
波段1:2 ~ 1:5持倉數天至數週，追求較高報酬比
長線1:3 ~ 1:10持倉數週至數月，捕捉大趨勢

適用時機：每一筆交易進場前都必須計算，無例外

不適用：如果你連停損都不設，那算風報比也沒意義

二、風險報酬比與勝率的關係

### 是什麼 風險報酬比和勝率是一對舞伴，兩者配合才能決定長期損益。單獨看任何一個都會誤判。

損益兩平勝率公式

損益兩平勝率 = 1 ÷ (1 + 報酬/風險)

不同風報比對應的最低勝率：

風險報酬比損益兩平勝率實際建議勝率
1:150%55%以上
1:233.3%40%以上
1:325%30%以上
1:516.7%20%以上
1:0.566.7%75%以上（極危險）

怎麼做：
1. 回測或統計你的策略歷史勝率
2. 根據勝率反推你需要的最低風報比
3. 若策略勝率40%，則風報比至少要1:1.5才能長期獲利
4. 進場前同時評估勝率和風報比，兩者缺一不可

實例：

  • 情境：假設你的波段策略歷史勝率35%
  • 計算：損益兩平風報比 = (1÷35%) - 1 = 1.86，至少需要1:1.86
  • 操作：只接受風報比1:2以上的交易
  • 結果：10筆交易，贏3.5筆（每筆賺2R），輸6.5筆（每筆賠1R）
  • 淨利 = 3.5×2 - 6.5×1 = 0.5R，長期正期望值

關鍵洞察

  • 高勝率+低風報比 = 危危險：賺9次小錢，一次大虧全吐回去
  • 低勝率+高風報比 = 安全：即使10次對3次也能賺錢，心理壓力反而小
  • 多數新手只追勝率，忽略風報比，這是長期虧損的主因

三、交易期望值（Trading Expectancy）

### 是什麼 大量交易後，平均每筆交易預期賺多少或虧多少。正期望值才能長期獲利，這是風險管理的終極判斷標準。

計算公式

期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損)

怎麼做：
1. 從交易日誌統計最近50筆以上的交易
2. 計算實際勝率、平均獲利金額、平均虧損金額
3. 代入公式算期望值
4. 若為正值 → 策略可持續；若為負值 → 必須修正或放棄

實例：

  • 情境A：勝率60%，平均賺1萬，平均虧1.5萬
  • 情境B：勝率35%，平均賺3萬，平均虧1萬

期望值四大陷阱

1. 樣本太少：少於30筆交易的統計沒意義，隨機性太大
2. 存活者偏差：只算還在交易的策略，沒算已經炸掉被淘汰的
3. 過度配適：回測期望值很高但實盤衰減30-50%
4. 忽略交易成本：手續費、滑價、稅金會吃掉期望值

適用時機：每週/每月定期計算，作為策略是否繼續執行的判斷依據

四、破產風險（Risk of Ruin）

### 是什麼 你的交易資金全部歸零的機率。這是風險管理中最底層的問題——不是「能賺多少」，而是「會不會死」。

破產風險公式

R = ((1-Edge) ÷ (1+Edge))^n

其中：
- Edge = 2×勝率 - 1（你的優勢）
- n = 你能承受的連續虧損次數（本金 ÷ 單筆風險金額）

簡化版經驗公式：

破產風險 ≈ ((1-勝率)÷勝率)^(本金單位數)

怎麼做：
1. 計算你的策略Edge（優勢）
2. 計算你能承受多少次連續虧損（本金 ÷ 每筆風險2%）
3. 代入公式或用線上計算器（如Myfxbook Risk of Ruin Calculator）
4. 目標：破產風險 < 1%（越小越好）

實例：

  • 情境A：本金100萬，每筆風險2%（2萬），勝率45%，風報比1:2
  • 情境B：本金100萬，每筆風險1%（1萬），勝率40%，風報比1:2.5

降低破產風險的五個方法

1. 降低單筆風險：從2%降到1%，可承受的連續虧損次數翻倍
2. 提高風報比：從1:2提升到1:3，同樣勝率下Edge增加
3. 提高勝率：加入更多過濾條件，只做高品質的交易
4. 增加本金：本金越大，能承受的連虧次數越多
5. 停止交易：連虧達到一定程度時強制暫停，避免情緒化操作

適用時機：建立任何交易系統前都必須計算破產風險，>5%的系統不值得執行

五、風險與報酬的分布關係（Howard Marks視角）

### 是什麼 Howard Marks在《投資最重要的事》中提出：高風險不代表一定高報酬，而是**報酬的分布範圍更廣**。風險越大，最好和最壞的結果都更極端。

三個關鍵認知

1. 高風險≠高報酬：高風險投資必須提供更高的「潛在」報酬來吸引資金，但不代表「一定」實現
2. 報酬是鐘形分布：風險越高，分布越廣。一個高風險投資可能成為下一個微軟，也可能變壁紙
3. 追逐導致溢酬縮水：越多人追捧高風險標的，風險溢酬反而越低，因為價格已被提前推高

怎麼做：
1. 設定「目標年化報酬率」，而非追求「最高報酬」
2. 根據目標報酬決定承擔多少風險（分布範圍）
3. 給自己足夠的投資時間，讓報酬分布收斂（時間越長，虧損機率越低）
4. 永遠記住：虧損20%需要賺25%才能回本，控制虧損比追求報酬更重要

實例：

  • 情境：投資100萬
  • 組合A（低目標報酬）：年報酬分布 -5%~17%，長期收斂趨近8%
  • 組合B（高目標報酬）：年報酬分布 -10%~30%，長期收斂趨近15%
  • 第一年虧損20%（100→80），第二年要賺25%才回本
  • 第一年賺25%（100→125），第二年虧20%就全吐回去（125→100）
  • 結論：報酬變動越大，越難穩定獲利。設定合理目標+拉長時間 = 最佳策略

六、風險報酬比實戰SOP

### 進場前四步檢查 1. **畫出支撐壓力**：標出進場價、停損價、目標價 2. **計算風報比**：至少1:2以上才考慮進場 3. **評估勝率**：根據技術型態+籌碼確認，估計這筆交易的勝率 4. **計算期望值**：期望值為正才進場 ### 進場後三步管理 1. **移動停損**：價格往有利方向移動1R後，停損移到成本價 2. **分批停利**：到達1:2時出50%，剩下用移動停利追 3. **絕不移遠停損**：虧損時擴大停損是最大禁忌 ### 週末覆盤 1. 統計本週所有交易的實際風報比 2. 比對「計畫風報比」vs「實際風報比」 3. 找出差距原因（提早出場？貪心沒跑？停損被移遠？） 4. 下週修正

心得

今天的主題雖然基礎，但越基礎的東西越容易被忽略。風險報酬比這個概念我之前已經學過很多相關的（凱利公式、ATR停損停利、2%資金管理），但今天重新整理後發現一個關鍵連結：風報比×勝率=期望值，期望值決定破產風險，這是一條完整的風控邏輯鏈。

Howard Marks的「風險是報酬分布的寬度」這個觀點特別有意思。我們常說「高風險高報酬」，但正確的理解是「高風險=報酬分布更廣=你可能大賺也可能大賠」。這跟塔雷伯的黑天鵝效應異曲同工——極端事件讓分布的尾部比常態分布更肥，所以靠常態分布算出來的風險模型永遠低估真實風險。

另外一個重要體悟：風報比不是越高越好。1:10的風報比聽起來很爽，但如果你設的目標價根本到不了，那勝率可能低到整體期望值是負的。真正要追求的是「風報比和勝率的最佳平衡點」，而不是單一指標的極端化。

最後，破產風險這個概念應該是每個交易者建立系統前第一個算的數字。不是「我能賺多少」，而是「我會不會死」。連這個都沒算過就上場，跟蒙眼開車沒兩樣。

資金曲線交易策略×交易自我破壞心理

一、資金曲線交易策略（Equity Curve Trading）

### 是什麼 資金曲線交易是一種**用你自己的帳戶淨值曲線來決定是否繼續交易**的方法。核心想法很簡單：當你的資金曲線在均線之上＝策略有效，繼續交易；當資金曲線跌破均線＝策略失效（或你狀態不佳），停止交易或縮小部位。

為什麼需要它

  • 任何策略都有「失效期」，即使長期正期望值的策略也可能連虧3-6個月
  • 人在連虧時容易做出最差的決策：加碼翻本、隨意更改策略、報復性交易
  • 資金曲線提供一個客觀的機制，強迫你在糟糕時候降速或停止

三種資金曲線策略

策略1：均線停止法（Equity Curve Moving Average）

怎麼做：
1. 計算資金曲線的N日均線（常用20日或30日）
2. 當資金曲線在均線之上 → 正常交易（100%部位）
3. 當資金曲線跌破均線 → 停止交易或縮小部位至50%
4. 當資金曲線重新站回均線之上 → 恢復正常交易

實例：

  • 情境：你的帳戶從100萬開始，20日均線目前在98萬
  • 操作：資金曲線在100萬（>98萬均線）→ 正常交易
  • 連虧後：資金曲線跌到96萬（<98萬均線）→ 停止新交易，只持有現有部位
  • 反彈後：資金曲線回到99萬（>98.5萬均線）→ 恢復交易
  • 關鍵數字：20日均線是最常見的參數，太短（5日）太敏感、太長（60日）太遲鈍

適用時機：趨勢型策略、有明確回撤週期的系統

策略2：回撤停止法（Drawdown Stop）

怎麼做：
1. 設定最大允許回撤（如10%或15%）
2. 從資金高點計算回撤幅度
3. 回撤到達門檻 → 停止交易
4. 等待資金曲線從低點回升一定比例（如5%）後才重新開始

實例：

  • 情境：帳戶高點120萬，設定最大回撤15%（即102萬為停損線）
  • 操作：帳戶跌到102萬 → 停止所有新交易
  • 恢復：帳戶從低點（如100萬）回升5%到105萬 → 重新開始50%部位交易
  • 進階：回撤10%→部位減半，回撤15%→完全停止，三級制更平滑

適用時機：所有策略，特別是波動大的系統

策略3：資金曲線動能法（Equity Momentum）

怎麼做：
1. 計算資金曲線的動能（如最近5日 vs 前5日的變化）
2. 動能為正（資金曲線上升中）→ 正常交易
3. 動能為負（資金曲線下降中）→ 縮小部位或停止
4. 搭配RSI或MACD分析資金曲線本身

實例：

  • 情境：最近5日資金曲線從100萬漲到103萬（動能+3%）
  • 操作：動能為正 → 正常部位進場
  • 反轉：最近5日從103萬跌到100萬（動能-2.9%）→ 部位減半
  • 進階：對資金曲線跑MACD，MACD柱狀圖翻正才恢復全部位

適用時機：進階交易者，已有穩定策略基礎的人

資金曲線交易的爭議與限制

#### 反對派的質疑 1. **過度擬合風險**：選擇什麼均線參數（10日？20日？30日？）本身就是過度配適的溫床 2. **樣本外效果存疑**：Build Alpha 的 David Bergstrom 做了大量隨機測試，發現**多數情況下資金曲線交易無法顯著改善長期報酬** 3. **錯過最佳反彈**：連虧後停止交易，正好錯過了策略回歸均值的反彈期——這往往是策略最賺錢的時候 4. **自我實現預言**：如果很多人用相同的資金曲線停止法，集體停止交易可能造成流動性問題 #### 支持派的論點 1. **心理價值大於數學價值**：資金曲線停止法的最大價值不是提升報酬，而是**防止交易者在最差的時候做最差的事** 2. **降低MDD**：雖然不見得提升報酬，但回測顯示確實能降低最大回撤15-25% 3. **強制紀律**：把「何時停止」的決定交給規則而非情緒

實務建議

  • 不要把它當魔法：資金曲線交易不是聖杯，它是一個風控工具
  • 簡單就好：用20日均線或10%回撤停止，不要過度優化參數
  • 搭配而非取代：它應該是你整體風控體系的一部分，不是唯一機制
  • 半部位優於全停：跌破時減半部位比完全停止更實際（完全停止容易錯過反彈）

二、交易自我破壞心理（Trading Self-Sabotage）

### 是什麼 交易自我破壞是指交易者**明知正確做法卻故意做相反的事**——連虧後加碼、有計畫卻不執行、獲利時急著跑、虧損時死抱不放。這不是智商問題，而是深層心理模式在作祟。

自我破壞四種類型

類型1：否定信念型（Negative Self-Belief）

症狀：

  • 明明有好的交易計畫，卻在進場前突然「覺得不對」而取消
  • 連續獲利後反而焦慮，開始亂做交易把利潤還回去
  • 潛意識覺得自己「不配賺錢」或「好運一定會結束」

怎麼做：
1. 識別你的否定信念：寫下「我對金錢的信念」（如「錢很難賺」、「有錢人都不好人」）
2. 挑戰每個信念：問「這是事實還是我以為的事實？」
3. 建立新信念：用具體行為取代，如「我按照計畫執行就是值得獲利的人」
4. 追蹤證據：每次按計畫獲利後記錄下來，逐漸累積正面反證

實例：

  • 情境：你的策略連續3筆獲利，帳戶創新高
  • 自我破壞：潛意識「不可能一直贏」，下一筆進場時刻意選最差的標的、加大部位
  • 修正：獲利後寫下「我今天的操作完全按照計畫，這不是運氣」→ 建立「獲利是能力的結果」新信念

類型2：活在過去型（Living in the Past）

症狀：

  • 反覆回想過去的虧損交易，無法專注當下
  • 用過去的方法在已經改變的市場中操作
  • 「上次這樣做賠錢了，所以這次不要做」→ 錯過好機會

怎麼做：
1. 完成過去交易的「告別儀式」：寫下虧損交易的完整覆盤，包括學到的教訓
2. 覆盤完畢後，在交易日誌標記「已結案」
3. 每次想翻舊帳時，翻到「已結案」標記提醒自己已經處理過了
4. 用「當下市場條件」而非「上次結果」做決策

實例：

  • 情境：上次在800點買台積電結果跌到750，停損出場
  • 自我破壞：這次台積電又到800，不管任何條件直接不敢買
  • 修正：檢查「這次的條件跟上次一樣嗎？」（均線位置？法人動向？大盤環境？）→ 條件不同就該有不同決策

類型3：完美主義型（Perfectionism）

症狀：

  • 等待「完美」的進場點，結果永遠不進場
  • 獲利1%就想「還可以更好」，結果等到變成虧損
  • 過度分析，10個指標9個說買、1個說賣就不敢動

怎麼做：
1. 接受「不完美進場也能賺錢」：回測你的策略，用各種不完美條件進場，看看結果
2. 設定「足夠好」門檻：3個核心條件滿足2個就進場，不必等5個全滿
3. 分批進場取代完美進場：先用50%部位，確認方向再加碼
4. 追蹤「等待完美」的代價：記錄每次因為等完美而錯過的利潤

實例：

  • 情境：KD黃金交叉、MACD柱狀體翻正、均線多頭排列、量能放大4個條件
  • 自我破壞：4個條件只出現3個就不敢進場，結果股價漲了15%
  • 修正：設定「3個條件滿足=進場」，4個全滿=加碼，永遠不會有5個全滿的完美時刻

類型4：缺乏技能偽裝型（Skills Deficit in Disguise）

症狀：

  • 以為自己有交易計畫，但計畫其實模糊到無法執行
  • 「我都知道了」但其實從未真正回測過自己的策略
  • 把自我破壞當藉口，掩蓋策略本身有問題的事實

怎麼做：
1. 檢查你的交易計畫是否「具體到可以程式化」：如果無法寫成程式碼，就不夠具體
2. 做客觀回測：不要用「我覺得這招有用」，要用數據說話
3. 區分「心理問題」和「策略問題」：連續虧損時，先檢查策略邏輯是否還成立
4. 承認不足：策略有問題就修改策略，不要用「心態不好」來逃避

實例：

  • 情境：你的策略過去3個月連續虧損
  • 自我破壞：告訴自己「我心理素質太差了」，繼續用同樣策略
  • 修正：先回測策略在當前市場環境下是否還有效→如果無效，改策略；如果有效但自己執行偏差太大→用程式交易

克服自我破壞五步框架

1. 覺察（Awareness）：記錄每次偏離計畫的操作，不評判，只觀察
2. 歸類（Classification）：用上面四種類型找出你的破壞模式
3. 系統化（Systematize）：把交易計畫寫成清單，進場前逐項打勾
4. 問責（Accountability）：找交易夥伴或公開交易日誌，讓別人可以監督
5. 小勝累積（Small Wins）：一次只改一個行為，連續執行21天成為習慣

自我破壞 vs 正常虧損的區分

特徵正常虧損自我破壞
原因策略的正常回撤明知不該做卻做了
情緒平靜，按計畫執行焦焦慮、衝動、事後懊悔
頻率符合策略的歷史虧損率遠超策略的預期虧損率
可控性已設停損，虧損可控未設停損或沒執行停損
修復不需要修復，繼續執行需要找出心理根源

適用時機：

  • 適用：任何交易者，尤其是反覆犯同樣錯誤的人
  • 不適用：策略本身有問題時（不要用心理問題掩蓋策略問題）

心得

資金曲線交易這招我以前一直覺得很酷——用自己的淨值曲線來控制要不要交易，好像很科學。但深入研究後發現，這東西的數學效果遠不如心理效果。Build Alpha 的大量隨機測試基本上打了「資金曲線交易能提升報酬」一巴掌，多數情況下改善不顯著，甚至有些情境會更差（因為你正好在策略回歸均值時停止交易）。

但它的心理價值是真实的：人在連虧時做決策的品質是災難級的。資金曲線停止法提供了一個「不依賴意志力」的強制機制——你不需要在情緒最差的時候做出「停止交易」的決定，因為規則已經幫你決定了。這跟之前學過的「紀律系統化：把心理問題變成流程問題」完全一致。

交易自我破壞四種類型，我認為最陰險的是「完美主義型」。因為表面上看起來很正向——追求完美嘛！但實際上是恐懼的偽裝：害怕犯錯、害怕虧損、害怕不夠好，所以用「等待完美時機」來逃避行動。這在散戶裡面超常見——分析半天、指標看一堆、最後什麼都沒做。分析癱瘓（Analysis Paralysis）的代價，往往比「不完美進場」更大。

最後，把資金曲線管理和自我破壞心理結合來看，就會得出一個清晰的框架：用資金曲線停止法防止外在的連虧擴大，用心理覺察框架防止內在的自我破壞。 一個管「何時該停」，一個管「為什麼你停不下來」。兩個一起用，才是完整的風險管理。

過度交易×報復性交易×交易倦怠

一、過度交易（Overtrading）

### 是什麼 過度交易是指**進行不屬於交易計畫一部分的交易**——量太多、頻率太高、理由不夠充分。不是「交易次數多就是過度交易」，而是「計畫外的交易太多」才是問題。 ### 過度交易兩種型態 1. **系統性過度交易**：交易規模過大（單筆部位超過風險上限） 2. **頻率性過度交易**：交易次數過多（看到波動就想進場）

怎麼判斷自己是否過度交易

  • 你今天的交易有多少筆是計畫內的？如果超過 50% 是臨時起意，已經過度交易
  • 你是否在平倉後馬上想反方向進場？
  • 你是否覺得「不進場就錯過了什麼」？
  • 你的交易頻率是否跟市場波動度不成比例？

怎麼做：五道防線

1. 設定每日進場上限

  • 日內交易：每日最多 3-5 筆（新倉到平倉算 1 次）
  • 波段交易：每週最多 2-3 筆新倉
  • 達到上限後，系統或自己強制停止下單

2. 設定交易間隔時間

  • 連續兩筆虧損後，至少休息 30 分鐘
  • 平倉後至少等 5 分鐘才能再次進場
  • 用冷卻時間避免衝動交易

3. 設定每日虧損上限

  • 每日最大虧損：總資金的 3-5%
  • 達到上限當天完全停止交易
  • 隔天重新評估，不把隔天的額度借來用

4. 交易前三問

  • 這筆交易在我的計畫內嗎？
  • 如果這筆虧損，我能冷靜接受嗎？
  • 我是因為機會還是因為無聊/焦慮想進場？

5. 記錄每一筆計畫外交易

  • 用交易日誌標記「計畫內」vs「計畫外」
  • 一個月後統計：計畫外交易的勝率和報酬 vs 計畫內
  • 數據會說話，看到計畫外交易績效多差，自然就會收斂

實例

  • 情境：某交易日大盤波動劇烈，你早上做了 3 筆當沖，2 虧 1 賺，淨虧 8,000 元
  • 錯誤操作：覺得「今天行情大，多做幾筆就能賺回來」，下午又做了 4 筆，結果全虧，當日總虧 35,000 元
  • 正確操作：達到每日進場上限 3 筆後停止，虧 8,000 元收工，保留實力明天再戰

適用時機

  • 任何交易者在任何市場都需要這些防線
  • 特別是新手前 6 個月，強制執行比自律更有效
  • 用程式或 APP 設定硬限制，不靠意志力

二、報復性交易（Revenge Trading）

### 是什麼 虧損後產生「想要馬上把虧損賺回來」的強烈衝動，導致衝動交易、過度交易、加大部位賭一把。這是交易者最容易犯的心理錯誤，也是從小虧變大虧的頭號凶手。 ### 報復性交易的心理機制 1. **虧損 → 憤怒/不甘心** → 想要立刻翻本 2. **加大部位** → 期望一筆賺回來 3. **判斷失準** → 情緒主導決策 4. **更大虧損** → 更強烈的報復衝動 5. **惡性循環** → 爆倉或信心崩潰

怎麼做：四層防禦系統

第一層：心理認知（預防）

  • 把虧損視為「交易成本」，不是「個人失敗」
  • 接受：市場沒有 100% 勝率，頂尖交易員也會虧損
  • 問自己：「我現在的決策是基於策略，還是因為情緒？」
  • 如果答案是後者，立刻停手

第二層：強制冷卻（物理中斷）

  • 連續兩筆虧損 → 休息 30 分鐘
  • 單日虧損達上限 → 當天不交易
  • 使用自動停損機制或 APP 限制下單功能
  • 離開電腦/手機，物理上遠離交易

第三層：資金管理（損害控制）

  • 單筆虧損不超過總資金 1-2%
  • 降低槓桿（10 倍 → 3-5 倍）
  • 連虧期間自動縮小部位（虧 5% 之後，部位減半）

第四層：交易日誌（事後修正）

  • 記錄每次報復性交易的觸發點、情緒狀態、結果
  • 分析模式：什麼情境最容易觸發？連虧幾筆後爆發？
  • 找到自己的「報復觸發線」，下次碰到提前預防

實例

  • 情境：帳戶 100 萬，上午做了 2 筆交易各虧 1 萬，累計虧 2 萬
  • 錯誤操作：不甘心，下午用 5 倍槓桿大部位進場，想一次賺回來，結果又虧 8 萬，當日總虧 10 萬
  • 正確操作：上午虧 2 萬後觸發冷卻機制，休息 30 分鐘，下午只做計畫內的小部位交易，當日總虧控制在 3 萬以內

適用時機

  • 任何連虧情境都需要警覺
  • 特別是當你感覺「一定要賺回來」的念頭浮現時
  • 當沖交易者風險最高（日內多次進出，觸發機率大）

三、交易倦怠（Trading Burnout）

### 是什麼 長期高壓交易導致的心理與生理耗竭狀態。症狀包括：對市場失去興趣、判斷力下降、不想看盤、即使有明確訊號也不想進場、交易績效持續下滑但無力改善。

倦怠三階段

第一階段：壓力累積期

  • 交易時間過長，每天盯盤超過 6 小時
  • 連續虧損或長期績效不佳
  • 睡眠品質下降，焦慮增加

第二階段：效能下降期

  • 判斷力明顯下滑，該進的不進、該出的不出
  • 開始忽略交易計畫
  • 對市場新聞和分析感到厭煩

第三階段：完全倦怠期

  • 完全不想看盤
  • 即使有明確機會也無動力進場
  • 交易變成一種折磨而非專業

怎麼做：五步恢復系統

1. 設定交易時間上限

  • 日內交易：每天不超過 4 小時（開盤 2 小時 + 收盤 1 小時 + 午盤 1 小時）
  • 波段交易：每天不超過 1 小時看盤
  • 其餘時間做研究、覆盤、或完全休息

2. 建立交易以外的興趣

  • 培養非金融相關的愛好（運動、音樂、閱讀）
  • 讓大腦有機會從市場模式切換到其他模式
  • 避免「人生只剩下交易」的單一認同

3. 規律作息與運動

  • 固定睡眠時間，至少 7 小時
  • 每週運動 3 次以上（有氧+重訓）
  • 運動能有效降低壓力荷爾蒙（皮質醇）

4. 深呼吸與冥想

  • 交易前做 2 分鐘深呼吸（4 秒吸、7 秒持、8 秒吐）
  • 每天冥想 5-10 分鐘，專注呼吸
  • 這不是玄學，是科學：深呼吸能激活副交感神經，降低心率，恢復理性思考

5. 定期「市場斷食」

  • 每月至少 1 個週末完全不看盤、不看財經新聞
  • 每季至少 1 週完全遠離市場
  • 回來後視角會更清晰，很多之前忽略的東西會浮現

實例

  • 情境：某交易者連續 3 個月每週交易 5 天、每天盯盤 8 小時，績效從穩定獲利變成連續虧損
  • 診斷：進入第二階段倦怠，判斷力下降，開始忽略交易計畫
  • 解方：強制休息 1 週，每天運動 1 小時，完全不碰市場。回來後縮短盯盤時間到 4 小時，績效逐步回升

適用時機

  • 任何感覺「不想看盤」的時候都是警訊
  • 連續交易超過 3 個月無休息的交易者
  • 績效突然下滑但策略沒有改變時，先懷疑是心理問題

四、恐懼與貪婪的深層解析

恐懼的兩面刃

  • 恐懼虧損 → 過早停損，被正常波動洗出場
  • 恐懼錯過（FOMO）→ 追高進場，在最高點買到
  • 恐懼成功 → 連續獲利後不敢進場，怕打破連勝

恐懼的辨識：當你猶豫要不要進場時，問自己「我是害怕虧損，還是這筆交易不符合計畫？」如果是前者，用小倉位克服；如果是後者，本來就不該進場。

貪婪的心理根源

  • 不是簡單的「想賺更多」，而是對自己能力的過度自信 + 對市場的輕視
  • 總是傾向於相信有利於自己的資訊，忽略不利信號
  • 獲利時不願出場，虧損時不願認賠

貪婪的辨識：當你已經達到目標價但想「再等等」，問自己「我的計畫是什麼？我什麼時候設定了這個目標？」堅守計畫，不要因為貪心而改變已經做好的決策。

自信心的正確位置

  • 適度自信：相信自己的系統和計畫，市場波動不動搖
  • 過度自信：覺得自己比市場聰明，可以預測轉折
  • 自信不足：稍有波動就懷疑自己，頻繁更改計畫

平衡點：索羅斯做空英鎊是自信（基於深入分析），但他在判斷錯誤時也會認賠出場。自信是「相信自己的判斷過程」，不是「相信自己的判斷一定對」。

五、心理資本（Psychological Capital）

### 是什麼 除了金錢資本，交易者還有「心理資本」——專注力、紀律、判斷力、情緒穩定度。心理資本和金錢一樣，會消耗也需要補充。

心理資本消耗的情境

  • 連續虧損：每虧一次，心理資本下降一級
  • 高波動市場：需要更多專注力來判斷
  • 多標的同時操作：注意力分散
  • 重大生活壓力：影響交易時的心理狀態
### 心理資本的補充方法 1. **休息**：最直接的補充方式 2. **小勝**：用小部位做一筆有把握的交易，重建信心 3. **覆盤**：回顧過去的成功交易，找回正確的感覺 4. **學習**：閱讀、研究新策略，重新激發熱情 5. **社交**：與其他交易者交流，知道不是只有自己遇到困境

實例

  • 情境：連虧 5 筆後，心理資本接近耗盡，第六筆明顯機會卻不敢進場
  • 錯誤：硬著頭皮用大部位進場想「一次回本」
  • 正確：用 1/4 部位做一筆高勝率交易，賺到小勝後逐步恢復部位

心得

過度交易、報復性交易、交易倦怠，這三個根本就是同一條路上的三個路標。先是過度交易（太勤勞），虧損後變成報復性交易（太衝動），長期下來變成交易倦怠（太疲憊）。解法說穿了也不複雜：少做、休息、照計畫走。

但知道不等於做到，這就是為什麼「系統化」比「自律」可靠。與其靠意志力控制自己不做報復性交易，不如用 APP 鎖住下單功能；與其靠自律控制盯盤時間，不如設鬧鐘提醒自己關電腦。把心理問題變成流程問題，讓規則代替情緒做決定。

特別有感的是「心理資本」這個概念。以前只知道資金管理，沒想過注意力、判斷力也是會消耗的資源。連虧的時候不是不夠努力，是大腦已經累了，就像體力耗盡還硬跑馬拉松一樣，成績只會更差。該休息就休息，不是偷懶，是在保護最珍貴的資產——清醒的腦袋。

一、交易計畫（Trading Plan）為什麼重要

### 是什麼 交易計畫是一份**書面文件**，詳細記錄你的交易目標、策略、風控規則、進出場條件。它不是「我大概知道要怎麼做」，而是**具體到可以交給別人執行**的操作手冊。 ### 為什麼90%的交易者沒有交易計畫 1. **覺得不需要**：以為靠盤感就能獲利 2. **不知道怎麼寫**：沒人教過，不知道要包含什麼 3. **寫了也不遵守**：這才是最大的問題——計畫存在但執行力為零 4. **怕被限制**：覺得計畫會限制靈活性

沒有交易計畫的後果

情境有計畫沒計畫
連續虧損3筆按計畫減碼或暫停報復性加碼翻本
看到突發消息檢查是否符合進場條件衝動追進
達到目標價按計畫分批出場貪心不賣，結果回吐
停損被觸發平靜接受，等待下一個機會不甘心移遠停損，小虧變大虧

核心觀念：交易計畫不是預測未來，而是決定你在各種情境下要怎麼做。市場不可控，但你的反應可控。

二、交易計畫七大要素

要素1：交易動機與目標

怎麼做：
1. 寫下你為什麼要交易（賺生活費？財務自由？學習？刺激？）
2. 設定SMART目標：

  • Specific（具體）：不是「賺大錢」，而是「12個月內年化報酬率15%」

實例：

  • ❌ 錯誤目標：「今年要賺100萬」
  • ✅ 正確目標：「未來12個月，使用波段策略，目標年化報酬率20%，最大回撤控制在15%以內，每週至少回測一次策略有效性」

要素2：時間投入

怎麼做：
1. 計算你每天/每週可以投入多少時間看盤和研究
2. 根據時間選擇交易風格：

可投入時間適合的交易風格
每天4小時以上當沖、短線交易
每天1-2小時波段交易
每週2-3小時長線投資、ETF定期定額
不固定不適合主動交易，改用被動投資

3. 交易風格決定了你的進出場頻率、停損距離、風報比要求

要素3：交易策略定義

怎麼做：
1. 選擇1-2個主要策略（不要同時用5種策略）
2. 明確定義每個策略的：

  • 進場條件（幾個條件要同時滿足）

實例——波段突破策略：

  • 進場條件：股價突破20日高點 + 量能放大1.5倍 + MACD柱狀體翻正
  • 停損設定：突破K線低點下方1%或ATR×1.5（取大者）
  • 停利設定：1:2風報比分批出場，50%在1:2出，50%用移動停損追
  • 適用市況：明確趨勢盤
  • 不適用：盤整區間、財報週、重大消息前

工具衝突的解決：

  • 當不同指標給出矛盾訊號時，制定優先順序
  • 例如：基本面篩選 → 技術面進場時機 → 籌碼面確認
  • 不是每個訊號都要等到，3個條件滿足2個就進場

要素4：風險管理規則

怎麼做：
1. 單筆風險上限：總資金的1-2%
2. 單日最大虧損：總資金的3-5%
3. 最大同時持有部位數：3-5個
4. 連虧暫停規則：連虧3筆 → 部位減半，連虧5筆 → 暫停1天
5. 每筆交易最低風報比：1:2以上

實例：

  • 本金100萬
  • 單筆最大虧損：1%（1萬元）
  • 單日最大虧損：3%（3萬元），達到後當天停止交易
  • 最多同時持有4個部位
  • 連虧3筆 → 部位從1%降到0.5%
  • 每筆交易風報比 < 1:2 → 不進場

要素5：倉位管理與盈虧比

怎麼做：
1. 決定每筆交易的風險金額（如1%）
2. 根據停損距離反算部位大小：

部位大小 = 風險金額 ÷ (進場價 - 停損價)

實例：

  • 本金100萬，單筆風險1%＝1萬元
  • 台積電進場價800元，停損780元，停損距離20元
  • 部位大小 = 10,000 ÷ 20 = 500股
  • 買入金額 = 500 × 800 = 40萬元（佔總資金40%）
  • 風報比1:3，目標價860元，預期獲利 = 500 × 60 = 3萬元

不同風險等級的部位調整：

風險等級單筆風險說明
低風險（明確趨勢）2%趨勢明確、籌碼配合
中風險（一般行情）1%正常進場
高風險（消息面、反轉）0.5%不確定性高

要素6：出場條件

怎麼做：
出場比進場更重要，必須事先定義所有出場情境：

1. 停損出場：觸及停損價，無條件出場
2. 停利出場：達到目標價，分批或全部出場
3. 移動停損出場：價格往有利方向移動1R後，停損移到成本價
4. 時間出場：持倉超過N天未達目標，減碼或出場
5. 訊號反轉出場：出現與進場訊號相反的訊號（如黃金交叉進場→死亡交叉出場）
6. 結構破壞出場：支撐/壓力結構被破壞，邏輯不再成立

實例：

  • 進場後3天未漲超過2% → 移動停損縮緊到成本價
  • 持倉超過5天未達目標價的50% → 出場一半
  • 出現與進場相反的MACD訊號 → 全部出場
  • 永遠不移遠停損（這是鐵律）

要素7：覆盤機制

怎麼做：
1. 每日覆盤（5分鐘）：今天做了什麼交易？有沒有遵守計畫？
2. 每週覆盤（30分鐘）：本週勝率、風報比、期望值是否達標？
3. 每月覆盤（1小時）：策略是否需要調整？有沒有新的市場變化？
4. 每季覆盤（2小時）：整體績效檢討，交易計畫是否需要大改？

三、交易日誌系統（Trading Journal）

### 是什麼 交易日誌是交易者的「病歷本」——記錄每一筆交易的進出場理由、情緒狀態、市場環境，用來找出自己的弱點和模式。

交易日誌必填七個欄位

1. 進場策略：這筆交易用什麼策略？為什麼選這個標的？
2. 進場價格與時間：什麼價格進場、什麼時間
3. 停損價與停利價：進場前就要設定好，不是進場後才想
4. 部位大小：買多少股？佔總資金多少%？
5. 出場價格與理由：什麼價格出場？是停損出場、停利出場、還是中途改策略？
6. 損益金額與比例：這筆交易賺/賠多少？佔總資金多少%？
7. 心情與反思：進場時的心情、持有期間的心情、出場時的心情

四、計畫執行力——為什麼計畫寫了卻不遵守

五大執行障礙與對策

障礙1：恐懼進場

症狀：條件都滿足了，但就是不敢按下去
對策：
1. 分批進場：先用50%部位進場，確認方向再加碼
2. 設定硬性規則：條件滿足3個就必須進場至少50%
3. 記錄「沒進場的代價」：每次因為恐懼沒進場，記錄錯過的利潤

障礙2：貪心不止盈

症狀：到達目標價卻想「再等等」，結果利潤全吐回去
對策：
1. 分批出場：目標價到就先出50%，剩下的用移動停損追
2. 預設出場指令：用限價單或移動停損單，讓系統幫你出場
3. 覆盤時看「沒出場的代價」：統計有多少次到了目標價沒出，最後反而虧損出場

障礙3：不甘心停損

症狀：停損到了卻移遠停損，小虧變大虧
對策：
1. 進場時就設定好停損單，不要依賴手動停損
2. 把停損視為「交易成本」，不是「失敗」
3. 事後覆盤：統計「移遠停損後的結果」，90%以上是更慘

障礙4：隨意更改策略

症狀：短線虧了就改長線，長線虧了又改短線
對策：
1. 每個策略至少回測100筆交易再評估
2. 策略的有效性需要時間驗證，連虧3-5筆不代表策略失效
3. 用交易日誌區分「策略問題」和「執行問題」

障礙5：情緒化加碼

症狀：虧損後加大部位想翻本，或獲利後過度自信加碼
對策：
1. 固定部位大小，不因情緒改變
2. 加碼只按計畫：突破後加碼，不是虧損後加碼
3. 連虧時自動縮小部位（連虧3筆 → 部位減半）

六、交易日誌 vs 交易計畫的關係

兩者是互補的：

  • 交易計畫是「應該怎麼做」——進場前就定好的規則
  • 交易日誌是「實際怎麼做了」——進場後的紀錄和反思
  • 覆盤是「計畫和實際的差距」——找出為什麼沒遵守計畫

怎麼做：
1. 每筆交易前：翻開交易計畫，確認進場條件是否滿足
2. 每筆交易後：在交易日誌記錄實際操作，標注是否遵守計畫
3. 每週覆盤：比較計畫與實際的差異，找出執行問題
4. 每月評估：策略是否需要修改（修改策略，不是修改紀錄）

交易日誌三個層次的反思

第一層：這筆交易本身

  • 進場理由是否充分？
  • 停損停利是否按計畫執行？
  • 風報比是否達標？

第二層：執行過程

  • 有沒有中途改策略？
  • 情緒是否影響了決策？
  • 計畫外交易有多少？

第三層：策略有效性

  • 最近20筆交易的勝率和風報比是否合理？
  • 這個策略在當前市況下是否還有效？
  • 是否需要調整策略參數？

心得

交易計畫和交易日誌，聽起來是最無聊的主題，但可能是風險管理裡最被低估的武器。

我之前學過的東西——停損五法、部位控制、心理管理——全部都可以寫進交易計畫裡。但知道和做到之間，隔了一條巨大的鴻溝。交易計畫就是把鴻溝上架一座橋：把「我知道應該怎麼做」變成「我實際做了什麼」，然後透過交易日誌和覆盤，找出橋上哪裡斷了。

最有感的是「不交易清單」。大多數人只關注「什麼時候要交易」，但真正赚钱的是「什麼時候不要交易」。財報前不交易、連虧後減量、狀態差就不看盤——這些「不做」比「做」更重要。就像Howard Marks說的，不犯錯比做對更重要。

另一個關鍵洞察是：交易日誌的價值不在紀錄，而在覆盤。很多人寫了日誌但從來不看，那等於白寫。覆盤的三個層次很好用：第一層看交易本身，第二層看執行過程，第三層看策略有效性。很多人只在第一層打轉，卻沒發現問題出在第二層（沒遵守計畫）或第三層（策略本身失效了）。

最後，計畫執行力的五個障礙，我覺得最難克服的是「恐懼進場」。條件都滿足了但不敢按下去，這不是知識問題而是心理問題。分批進場和硬性規則（條件滿足3個就必須進場）是兩個最實用的解法。不是「我覺得可以進場了」，而是「規則說我必須進場」——把主觀判斷變成客觀執行。
