一、內部人定義與申報規範

誰是內部人？

定義：對公司營運有深度了解且擁有決策權的人士

分類範圍
內部人董事、監察人、經理人、持股超過10%大股東
關係人配偶、未成年子女、利用他人名義持有者、法人董事代表人及其配偶、未成年子女

為什麼重要？ 他們比外人更了解公司營運狀況，交易行為被市場高度關注。

申報轉讓規範

證券交易法規定：內部人賣股必須事前申報

  • 申報時間：賣出前需申報
  • 轉讓期間：申報日起3日後至1個月內
  • 每日限制：未超過1萬股者免逐日公告
  • 禁售期：取得身份後6個月內不得轉讓

二、申報轉讓五種方式與股價影響

關鍵結論：不是所有申讓都是利空！

轉讓方式對股價影響解讀
**贈與**✅ 偏多或中性課稅基準為當天收盤價，大股東會選在股價低檔贈與節稅
**信託**⚪ 中性財務規劃或員工獎勵，對股價無明顯影響
**盤後配對/鉅額**⚪ 中性偏多不影響市場賣壓，可能是策略股東承接
**指定人交易**⚪ 中性特定人承接，可能是引進策略夥伴
**一般交易**❌ 利空直接從市場賣出，需高度警戒

詳細解析

1. 贈與 — 低檔布局的隱性利多

是什麼：內部人將股票贈與配偶、子女等

為什麼偏多？

  • 贈與稅以贈與當天收盤價計算
  • 大股東會選在股價相對低檔時贈與以節稅
  • 代表大股東認為目前股價偏低，未來有上漲空間

實例：

  • 某董監事在股價50元時申報贈與子女
  • 課稅基準：50元 × 贈與股數
  • 若股價漲到100元，省下的贈與稅非常可觀

判讀要點：看到「贈與」申報，反而可能是低檔布局訊號

2. 信託 — 中性財務規劃

是什麼：股票所有權移轉給受託人（銀行）管理

怎麼做：
1. 內部人與銀行簽訂信託契約
2. 股票所有權移轉給受託人
3. 受託人依約定管理運用

適用情境：

  • 公司發放限制員工權利新股
  • 個人資產配置規劃
  • 遺產稅規劃

對股價影響：無明顯多空，中性看待

3. 盤後配對交易 / 鉅額交易 — 中性偏多

是什麼：盤後透過鉅額交易方式轉讓

為什麼中性偏多？

  • 不會在集中市場造成賣壓
  • 買方通常是法人或策略投資人
  • 可能是大股東節稅（轉為投資公司）
  • 可能是引進策略股東

實例：致伸（4915）

  • 2014年5月：私募基金申報轉讓54,578張（約13%股本）
  • 方式：盤後鉅額交易
  • 結果：私募基金完成投資後，由長線外資承接
  • 股價表現：申報後兩個月大漲50%

判讀要點：鉅額交易不代表看壞後市，要看承接方是誰

4. 指定人交易 — 中性

是什麼：將股票轉讓給符合條件的特定人

特定人範圍：

公司類型特定人範圍
上市以同一價格受讓之公司全體員工
上櫃/興櫃證券自營商及公司全體員工
未上市櫃公發對公司財務業務有充分了解、具資力且非公開招募認購者

適用情境：

  • 引進策略股東
  • 員工激勵
  • 上下游供應商/客戶合作

對股價影響：中性，視承接方目的而定

5. 一般交易 — 最需警戒的利空

是什麼：內部人直接在集中市場賣出持股

為什麼是利空？

  • 直接從市場拋售，造成實質賣壓
  • 內部人可能知道營運出狀況
  • 業績地雷未爆前先行出場

實例：奇偶（3356）

  • 2014年2月至8月：董事長與配偶多次申報轉讓
  • 每次200-500張，全是「一般交易」
  • 8月中旬Q2財報：淨利減少三成，EPS創14季新低
  • 股價從213元跌至98.5元，腰斬超過50%

實例：國巨（2327）

  • 2018年7月：董娘以鉅額交易出脫1.2萬張
  • 套現金額：約120億元
  • 股價：從1310元高點，一年跌至200多元
  • 市場教訓：「勿忘國巨」成為董娘賣股經典案例

三、內部人買超訊號判讀

董監持股增加的意義

是什麼：董監事、大股東持股比例上升

怎麼判讀：

情境解讀
空頭市場 + 內部人加碼對公司營運有信心，可能是底部訊號
營收成長 + 內部人減碼營收可能見高點，需警戒
PB跌至歷史低點 + 內部人買超大股東抄底，波段進場點

實用數據：

  • 上市櫃平均董監持股比例：約24%
  • 眾數：約19%
  • 股本越小，董監持股比例通常越高

選股策略：大股東抄底

策略條件：
1. 最近一個月董監持股增加，或
2. 最近一個月內部人持股比例增加，或
3. 近一日關鍵券商合計買超

進場訊號：

  • PB跌至近六年低點
  • 成交量佔大盤比暴增
  • 股價上漲但漲幅不大（<5%）

回測結果（過去7年，停損停利各10%）：勝率與訊號數量表現良好

四、實戰操作心法

看到申報轉讓怎麼辦？

第一步：確認轉讓方式

  • 贈與/信託 → 中性偏多，不用緊張
  • 盤後配對/鉅額 → 中性，看承接方
  • 一般交易 → 高度警戒

第二步：評估申報張數

  • 單次小額（<總股本1%）→ 可能是理財需求
  • 大額或密集申報 → 需要深入了解原因

第三步：檢查公司基本面

  • 營收是否在高點？
  • 獲利是否下滑？
  • 是否有利空未爆？

第四步：觀察是否多位內部人同步賣出

  • 單一人賣出 → 可能是個人理財
  • 多位內部人密集申報 → 集體看壞，盡早退場

內部人買超怎麼跟？

確認訊號：
1. 查看公開資訊觀測站「董監持股變動」
2. 觀察千張大戶持股變化
3. 確認股價是否仍在低檔區

進場時機：

  • 內部人買超 + 股價低檔盤整
  • 內部人買超 + PB接近歷史低點
  • 內部人買超 + 法人同步買超

風險控制：

  • 不要因為內部人買超就重押
  • 設好停損，基本面惡化就出場
  • 內部人買超是參考，不是保證

心得

這回合學了一個很實用的籌碼面知識：不是所有內部人申報轉讓都是壞事，關鍵在於轉讓方式。

以前看到「董娘申報轉讓」的新聞標題就覺得要跌了，但其實：

  • 贈與/信託 → 可能是節稅規劃，反而是低檔訊號
  • 鉅額交易 → 接手方可能是外資或策略股東
  • 一般交易 → 這才是要警戒的

另外「勿忘國巨」這個案例印象深刻，董娘在高點出脫1.2萬張，股價從1310跌到200多。這告訴我：大股東比誰都清楚公司真實狀況，他們賣在高點不是巧合。

反過來說，當大股東在空頭市場加碼，或PB跌到歷史低點時買超，可能就是「抄底」的好時機。但前提是要確認是「哪一種申報方式」和「基本面是否真的沒問題」。

最後一句話總結：內部人動向是參考，不是聖旨。聽其言不如觀其行，但也要看他是怎麼做的。
