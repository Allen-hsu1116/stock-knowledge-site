基本面分析：配息率、盈餘再投資率與保留盈餘

第 4 回合學習 - 基本面分析：配息率、盈餘再投資率與保留盈餘

時間：03:22

文章：

  • 什麼是股票的配息率(Payout Ratio)？公式如何計算、怎麼查詢？
  • 盈餘再投資比率｜選股便利店
  • 保留盈餘是什麼？與股利的關係？盈餘負數或增加代表什麼？

一、配息率 Payout Ratio

1. 配息率是什麼？

配息率（Payout Ratio）= 公司將盈餘中拿出多少比例配發給股東。

公式：配息率 = 每股股利 ÷ 每股盈餘（EPS）× 100%

跟殖利率不同：

  • 配息率：公司賺的錢，多少比例拿出來配（公司視角）
  • 殖利率：投資人買股的市價，能得到多少比例股利（投資人視角）

2. 配息率怎麼判斷？

不是越高越好，也不是越低越好，配息率反映的是公司的政策與計畫：

配息率範圍可能代表
低（0~40%）成長型公司，盈餘拿去再投資（如 Amazon、波克夏）
中（40~80%）穩健型公司，兼顧配息與再投資
高（80~100%）成熟型公司，投資機會少，盈餘配回股東
超過100%動用過去保留盈餘配息，需警覺是否可持續

配息率超過100%的原因：公司可能有先前未發放的保留盈餘，在某次一併發放，導致當期股利超過當期盈餘。

3. 實戰判斷

情境：A公司 EPS 10元，配發8元現金股利

  • 配息率 = 8 ÷ 10 = 80%
  • 剩餘2元為保留盈餘，公司拿去再投資

情境：B公司 EPS 5元，配發6元現金股利

  • 配息率 = 6 ÷ 5 = 120%
  • 超過100%！公司可能動用過去累積的保留盈餘，需檢查能否持續

4. 配息率高低選股邏輯

高配息率 ≠ 好公司：

  • 台灣投資人偏好穩定配息，但這反而代表投資人對公司的不信任感
  • 如果公司能有效運用資金，讓公司自己投資創造複利更好
  • 巴菲特的波克夏從不配息，但長期績效驚人

低配息率 ≠ 壞公司：

  • 成長型公司把盈餘再投資，配息率低是正常的
  • 關鍵是保留盈餘能否創造更高的未來價值

巴菲特標準：企業每保留1元的盈餘，就要能在未來至少提升1元的市場價值。

5. 查詢管道

  • 國內：Goodinfo!台灣股市資訊網
  • 國外：Morningstar

二、盈餘再投資率（盈再率）

1. 盈再率是什麼？

盈餘再投資率（Reinvestment Ratio），由洪瑞泰在《巴菲特選股魔法書》提出。

公式：盈再率 =（固資₄ + 長投₄ - 固資₀ - 長投₀）÷（淨利₁ + 淨利₂ + 淨利₃ + 淨利₄）

= 近4年增加的固定資產及長期投資 ÷ 近4年稅後淨利加總

意義：近4年資本支出佔盈餘的比例。

  • 比例低 → 不需要增加太多資本支出就能持續創造獲利（省油好車）
  • 比例高 → 需要更多資本支出才能維持獲利（耗油大車）

採用4年：因為一般產業的景氣循環時間，求比較基礎一致。

2. 盈再率判斷標準

盈再率判斷
< 40%偏好，低資本支出就能維持獲利
40~80%中性，需搭配ROE判斷
> 80%偏高，資本支出吃重，景氣反轉風險大

3. 高ROE + 低盈再率 = 黃金組合

洪瑞泰認為這是最理想的選股組合：

  • 高ROE = 公司效能好（跑得快）
  • 低盈再率 = 不耗油（省油）
  • 好開又不耗油的車，當然是好選擇

雙面刃：高資本支出雖然風險大，但也可能是未來成長的機會。若公司現金流足以支應，就不需要舉債或增資。

4. 實戰案例

太醫（4126）：

  • 104年以前盈再率低於50%
  • 銅鑼新廠啟用後，盈再率飆升到139%（固定資產大增 + 折舊吃掉獲利）
  • 之後營收回升、資本支出不再大增，盈再率回到負值（健康狀態）

台積電（2330）：

  • 半導體屬高資本支出產業
  • 109年資本支出5057億（遠高於100年的1825億）
  • 但獲利能力持續提升，盈再率反而降到40%以下
  • 結論：高資本支出但hold得住！

邦特（4107）：

  • 101~109年盈再率都低於40%
  • 每年自由現金流皆為正
  • 資本支出相對獲利相當健康，用自有資金就能支應

5. 巴菲特六點選股原則

1. 大型股
2. 穩定的獲利能力
3. 高ROE + 低負債比
4. 良好的經營團隊
5. 簡單且可以理解的企業
6. 出價（便宜才出手）

6. 賣股三原則

1. 壞了：基本面不如預期、體質改變
2. 貴了：本益比超過歷史區間（書中建議>40倍，但可依個股歷史區間調整）
3. 換更好的：投資組合定期淘弱留強

三、保留盈餘 Retained Earnings

1. 保留盈餘是什麼？

保留盈餘 = 公司將未分配的獲利保留下來，供未來發放股利或投資使用。

公式：保留盈餘 = 期初保留盈餘 + 淨利潤 - 向股東分配的股利

  • 盈餘為正 → 保留盈餘
  • 盈餘為負 → 累積虧損

2. 台灣保留盈餘三種類

類型說明
法定盈餘公積公司法規定，每年保留10%稅後盈餘
特別盈餘公積公司自提，用來吸收未來可能的虧損
未分配盈餘提撥完上述後的剩餘，可自由運用

注意：未分配盈餘要繳5%的未分配盈餘稅。

3. 保留盈餘使用方式

1. 以股利形式分配給股東
2. 進行投資或推出新產品
3. 用於併購或投資
4. 用於股票回購
5. 用於償還債務

4. 保留盈餘高低判斷

過低：

  • 缺乏現金，景氣不佳時周轉困難
  • 難以擴張規模、錯過投資機會
  • 需要透過債務或增資募集資金，成本更高

過高：

  • 資金運用效率可能低
  • 如果公司ROE僅5%，投資人自己運用能達10%，不如配回股東
  • 管理層有義務有效率地運用保留的資金

為負值：

  • 代表公司虧損
  • 成長初期的企業可理解，但要確認資金充足
  • 營運良好的企業突然轉負，要嚴格檢視

5. 保留盈餘與股價的關係

巴菲特：「每1元的保留盈餘，應該要能為投資人創造1元的市場價值。」

計算邏輯：

  • 投入1元保留盈餘 → 未來EPS成長0.1元
  • 以本益比15倍 → 預期股價上漲1.5元
  • 1元保留盈餘創造1.5元市值，符合巴菲特標準

6. Amazon 實戰案例

Amazon 從不配息，獲利全部保留再投資：

  • 2009年保留盈餘佔股東權益僅3.3%
  • 2019年保留盈餘佔股東權益達50.3%
  • 營收從148億成長到2805億美元
  • 原有股東不需持續拿錢出來，靠公司盈餘再投資就能撐起成長

結論：保留盈餘成功運用的案例，但並非每間公司都能如此。

7. 核心觀念

配息是果、盈餘是因：

  • 注意力應放在盈餘，而非配息是否穩定
  • 有穩定成長的盈餘，自然帶來整體資產價值的成長
  • 配息只是決定錢留在公司名下還是轉給股東個人運用（左口袋與右口袋的差別）

心得

這回合學了三個互相關聯的基本面指標：配息率、盈再率、保留盈餘，剛好形成一個完整的「盈餘分配」判斷體系。

配息率告訴你公司「配多少」，盈再率告訴你公司「投資多少」，保留盈餘告訴你「累積多少」。三者互相勾稽：配息率低 → 保留盈餘高 → 盈再率可能高（如果公司把錢拿去蓋廠房）或低（如果公司只是把錢放在帳上）。

最核心的判斷邏輯是巴菲特那句話：保留1元盈餘，至少要創造1元市值。這是檢驗公司管理層資金運用效率的終極標準。如果公司配息率很高，代表管理層認為自己無法有效運用資金；如果配息率很低但ROE持續上升，代表管理層值得信任。

盈再率這個指標特別實用，把「耗油率」的概念引入選股，比單看ROE更能區分出哪些公司是「輕鬆賺錢」、哪些是「用力才賺得到」。高ROE+低盈再率的黃金組合，值得放在自己的選股清單裡。

文章：

  • 自由現金流量是什麼？FCF公式和查詢方法？5檔ETF分享！
  • 怎麼判斷股票估值高低？自由現金流收益率

一、自由現金流（FCF）是什麼？

是什麼：公司真正能自由運用的現金。營業現金流扣掉資本支出後，剩下的就是 FCF。巴菲特 1980 年股東信說：「我們偏愛那些產生現金而非消化現金的公司。」

公式：

FCF = 營業現金流 − 資本支出

台灣財報替代算法：

FCF = 營業現金流 + 投資現金流（投資現金流通常是負值）

白話解釋：就像你的薪水扣掉房租水電和必要投資（買電腦、進修），剩下的就是你可以自由花的錢。公司也是一樣，營業賺來的現金扣掉買設備、建廠房的支出，剩的就是自由現金流。

二、FCF 三種計算方法

方法 1：從現金流量表直接算（最常用最簡單）

FCF = 營業現金流 − 資本支出
或
FCF = 營業現金流 + 投資現金流

怎麼做：
1. 打開 Goodinfo 或公開資訊觀測站
2. 找到「現金流量表」
3. 抓營業活動現金流量和投資活動現金流量
4. 兩者相加就是 FCF

實例：

  • 台積電（2330）2025Q3：營業現金流約 4,500 億，投資現金流約 −3,800 億
  • FCF = 4,500 + (−3,800) = 700 億（正值，代表有自由現金可運用）

方法 2：從損益表和資產負債表推算

FCF = EBIT × (1 − 稅率) + 折舊攤銷 − 營運資金變動 − 資本支出

適用情境：想深入了解獲利品質，或現金流量表資料不完整時

方法 3：三大報表綜合計算

FCF = 淨利 + 利息費用 × (1 − 稅率) + 折舊攤銷 − 營運資金變動 − 資本支出

適用情境：沒有 EBIT 數據時的替代算法

三、FCF 利潤率（FCF Margin）

是什麼：衡量公司把營收轉化為自由現金流的效率

公式：

FCF Margin = FCF ÷ 營收（Revenue）

判斷標準：

  • > 5%：值得留意
  • 10～15%：相當不錯
  • < 5%：資本密集型公司可能在此區間，需搭配其他指標
  • 逐年穩步成長：有大幅成功潛力的信號
  • 同業相比異常下降：財務狀況惡化的警訊

怎麼做：
1. 查公司近 5 年 FCF 和營收
2. 計算每年 FCF Margin
3. 看趨勢：穩定上升 → 好公司；逐年下降 → 小心
4. 跟同業比較，看是否有異常

實例：

  • A 公司：營收 1,000 億，FCF 150 億 → FCF Margin = 15%（優秀）
  • B 公司：營收 800 億，FCF 20 億 → FCF Margin = 2.5%（資本密集或效率低）

四、FCF 正值 vs 負值判斷

FCF 為正值

代表的意義：營運現金能支付資本支出，不需靠借貸維持營運

但要注意：FCF 正不一定安全！要細看營業現金流的來源：

  • ✅ 正常：營業現金流穩定正值 + 資本支出合理 → 健康公司
  • ❌ 危險：營業現金流為負，但變賣資產讓投資現金流為正 → 表面好看但不可持續

怎麼做：
1. 看 FCF 是正的
2. 再看營業現金流是否也是正的
3. 看投資現金流是否主要是資本支出（而非變賣資產）
4. 三項都正常 → 放心

FCF 為負值

短期為負：可能正在擴廠、添購設備，未來有機會帶來更大收益 → 不一定壞

長期為負：營運現金流無法支撐投資需求，必須靠借貸或增資補洞 → 高風險

怎麼做：
1. 看連續幾年的 FCF 走勢
2. 若只有 1-2 年為負，看資本支出是否用在擴產（查公司公告）
3. 若連續 3 年以上為負，直接跳過
4. 注意「燒錢慘業」：面板、LED、太陽能、DRAM 這些產業天然 FCF 低

五、每股自由現金流 vs EPS：照妖鏡功能

是什麼：把 FCF 除以股數，得到每股 FCF，用來跟 EPS 比較

公式：

每股 FCF = FCF ÷ 總股數

為什麼重要：EPS 可以被會計手法美化（折舊、應收帳款、非現金收入），但 FCF 比較難造假

照妖鏡情境：

指標EPS每股 FCF判斷
正正正✅ 健康公司
正正負⚠️ 盈餘品質差，可能靠會計美化
正高低⚠️ 資本支出吃掉現金，或應收帳款暴增
負負負❌ 不碰

實例：

  • A 公司：EPS 2.45 元，每股 FCF 0.85 元 → FCF 遠低於 EPS
  • 可能原因：① 瘋狂擴廠吃現金 ② 存貨堆積 ③ 應收帳款暴增 ④ 毛利被壓縮
  • 進一步查證：看資本支出增幅、存貨周轉率、應收帳款天數

適用時機：大型穩定企業最有參考價值。小型股收入低、盈餘變動大，每股 FCF 意義有限。

六、自由現金流收益率（FCF Yield）：估值利器

是什麼：用 FCF 來衡量股票的「實質收益率」，類似債券的殖利率概念

公式：

FCF Yield = 每股 FCF ÷ 每股股價

或

FCF Yield = FCF ÷ 企業價值（EV）

白話解釋：你花 100 元買這股票，公司每年真正能自由運用的現金能幫你賺多少？這比本益比更真實，因為 FCF 不受會計手法影響。

判斷標準：

  • > 8%：非常有吸引力，可能被低估
  • 5～8%：合理偏便宜
  • 3～5%：合理價位
  • < 3%：偏貴或資本效率低
  • 負值：不適用，公司現金流為負

怎麼做：
1. 查公司 FCF（Goodinfo 或財報）
2. 查當前股價和流通股數
3. 算每股 FCF
4. 除以股價得到 FCF Yield
5. 跟同業比較，跟十年期公債殖利率比較

實例：

  • 公司 X：每股 FCF 4 元，股價 50 元 → FCF Yield = 8%（很有吸引力）
  • 公司 Y：每股 FCF 1.5 元，股價 100 元 → FCF Yield = 1.5%（偏貴）
  • 若十年期公債殖利率 4%，FCF Yield > 4% 的股票才值得考慮（風險溢價）

適用時機：

  • ✅ 成熟穩定的企業（有穩定 FCF）
  • ✅ 價值投資篩選
  • ❌ 高成長股（FCF 可能为負但公司正在高速擴張）
  • ❌ 燒錢慘業（FCF 長期為負）

七、FCF 選股實戰五步驟

怎麼做：

1. 篩選 FCF 正值：先排除連續 3 年 FCF 為負的公司
2. 看 FCF 趨勢：近 5 年是否穩定成長或持平，大幅波動的要小心
3. 算 FCF Margin：> 5% 留意，> 10% 優秀
4. 算 FCF Yield：> 5% 值得深入研究
5. 驗證營業現金流品質：確認 FCF 正值是來自本業賺錢，而非變賣資產

查詢管道：

  • 台股：Goodinfo 台灣股市資訊網 → 現金流量表
  • 美股：Morningstar → Valuation 頁面

心得

FCF 根本就是財報的照妖鏡。EPS 可以靠折舊方法、應收帳款認列來美化，但 FCF 騙不了人——現金就是現金，有就是有，沒有就是沒有。最可怕的組合是 EPS 很漂亮但 FCF 很難看，這種公司八成是在燒錢維持表面獲利。

FCF Yield 這個指標特別實用，它把 FCF 和股價連結起來，讓你一眼看出「花這個價錢買，每年實質能回多少現金」。比本益比更真實，因為本益比用的是會計盈餘，FCF 用的是真實現金。

不過 FCF 也不是萬能的，高成長股在擴張期 FCF 通常為負，這時候硬用 FCF Yield 篩選會錯過好公司。關鍵是要判斷 FCF 為負的原因：是暫時擴張投資，還是長期結構性燒錢？前者是機會，後者是地雷。

朝股票操作大師邁進中 📈## 第 5 回合學習 - 基本面分析：DDM 股利折現模型與 PEG 本益成長比

時間：05:22

文章：

  • 股利折現模型是什麼？公式如何計算？用來估價時的注意事項整理
  • 如何用股息折現模型 DDM 計算合理價格
  • 看懂本益成長比 PEG：本益成長比查詢與公式
  • 進化後的本益比｜本益成長比選股策略

一、DDM 股利折現模型（Dividend Discount Model）

1. DDM 是什麼？

是什麼：用未來領到的股利折現回現在，計算股票的內在價值。核心思想是——股票的真正價值，等於未來所有股利的現值總和。

白話解釋：買股票就像買一台印鈔機，每年幫你印出股利。DDM 就是算這台印鈔機「現在值多少錢」。未來的 1 塊不如今天的 1 塊值錢（貨幣時間價值），所以要把未來的股利「打折」回現在。

2. 三種 DDM 公式

公式一：單期 DDM

V₀ = (D₁ ÷ (1+r)) + (P₁ ÷ (1+r))
  • V₀：股票當前價值
  • D₁：一年後的股利
  • P₁：一年後的股票價格
  • r：折現率（預期報酬率）

適用情境：只打算持有一年、快速評估

公式二：多期 DDM

V₀ = Σ [Dₙ ÷ (1+r)ⁿ] + [Pₙ ÷ (1+r)ⁿ]

適用情境：預計持有多年，每年股利可能不同

公式三：高登成長模型（最常用）

V₀ = D₁ ÷ (r - g)
  • V₀：股票合理價
  • D₁：明年預估股利
  • r：折現率（預期報酬率）
  • g：股利年成長率

適用情境：股利持續穩定成長的成熟公司

3. 高登成長模型計算實例

怎麼做：

1. 確認公司穩定配息且股利有成長
2. 取得明年預估股利 D₁
3. 設定折現率 r（一般 8%~12%）
4. 預估股利成長率 g（保守估計）
5. 代入公式：合理價 = D₁ ÷ (r - g)
6. 比較合理價與市價，加上安全邊際

實例：

  • 情境：某穩定配息公司，今年配息 3 元，預估成長率 5%
  • D₁ = 3 × 1.05 = 3.15 元
  • 折現率 r = 10%
  • 合理價 = 3.15 ÷ (10% - 5%) = 3.15 ÷ 0.05 = 63 元
  • 若目前股價 50 元 → 低於合理價，被低估 20.6%
  • 若加 20% 安全邊際 → 買進價 = 63 × 0.8 = 50.4 元，剛好接近

另一個實例：

  • 情境：中華電（2412），今年現金股利 4.8 元，預估成長率 3%
  • D₁ = 4.8 × 1.03 = 4.944 元
  • 折現率 r = 8%
  • 合理價 = 4.944 ÷ (8% - 3%) = 4.944 ÷ 0.05 = 98.88 元
  • 若股價 110 元 → 高於合理價，偏貴

4. 二段式 DDM（更精進的版本）

一段式假設永遠同一成長率，不現實。二段式把成長分兩期：

怎麼做：

1. 第一段：未來 N 年（通常 5~10 年）的高成長率 g₁
2. 第二段：N 年之後的永久成長率 g₂（通常等於 GDP 成長率或通膨率，約 2%~5%）
3. 分別折現兩段的股利再加總

公式：

V₀ = Σ[Dₙ ÷ (1+r)ⁿ] + [D_{N+1} ÷ (r - g₂)] ÷ (1+r)ᴺ

實例：

  • 情境：某公司今年股息 2 元
  • 第一段（1~10年）：成長率 8%，折現率 10%
  • 第二段（11年後）：高原成長率 4%
  • 第一年股息 = 2 × 1.08 = 2.16
  • 第十年股息 = 2 × 1.08¹⁰ = 4.315
  • 第十一年股息 = 4.315 × 1.04 = 4.488
  • 第二段終值 = 4.488 ÷ (10% - 4%) = 74.79
  • 把第一段每年折現 + 第二段終值折現回現在 = 合理價

5. DDM 三大注意事項

注意一：只能用在穩定配息的成熟公司

不配息的成長股（如早期台積電、Google、Amazon）不能用 DDM。這些公司要用 DCF（自由現金流折現）。

注意二：成長率 g 的影響超級大

g 從 5% 變成 6%，合理價可能差 20% 以上。一定要保守估計：

  • 歷史 10 年年化成長率 12% → 實際用 7%~8%
  • 參考營收成長 + 淨利成長 + 庫藏股回購的綜合效果

注意三：折現率 r 越高越保守

  • 一般用 8%~10%（接近長期股市年化報酬率）
  • 高風險公司用 12%
  • r 越高 → 合理價越低 → 越保守

6. DDM vs DCF 比較

面向DDMDCF
現金流來源股利自由現金流（FCF）
適用公司穩定配息的成熟公司所有公司（含不配息的）
資料取得容易，股利數字明確較複雜，要算 FCF
估價結果偏保守較有彈性
會漏掉的公司不配息的成長股不會

二、PEG 本益成長比（Price-to-Earnings Growth Ratio）

1. PEG 是什麼？

是什麼：本益比的進階版，把公司未來獲利成長性納入考量。解決了「本益比高到底是股價貴，還是成長性被算進去」的問題。

白話解釋：本益比 30 倍看起來很貴，但如果公司每年 EPS 成長 30%，那 30 倍其實很合理。PEG 就是把這個關係量化。

2. PEG 公式

PEG = 本益比(PE) ÷ 預估盈餘成長率(%)

注意：成長率的百分比數字直接除，不用換成小數。例如成長率 20% 就是除以 20。

3. PEG 判斷標準

PEG 值意義動作
< 0.66嚴重低估強力買進
0.66 ~ 0.75低估買進
0.75 ~ 1.0略微低估可考慮
= 1.0合理不買不賣
1.0 ~ 1.2略微高估觀望
> 1.2高估考慮賣出

核心原則：PEG = 1 表示本益比與成長率匹配，股價合理。

4. PEG 計算實例

怎麼做：

1. 查公司本益比（從 Goodinfo 或券商軟體）
2. 預估未來 EPS 成長率（建議用 3~5 年預估，不要只看 1 年）
3. 代入 PEG = PE ÷ 成長率%
4. 對照判斷標準
5. 同時看 PE 和 PEG，不要只看一個

實例一：台積電（2330）

  • 本益比 22 倍
  • 預估未來 EPS 成長率 18%
  • PEG = 22 ÷ 18 = 1.22
  • 判斷：略微高估，成長性已被反映在股價中

實例二：某成長股

  • 本益比 30 倍（看起來很貴！）
  • 預估 EPS 成長率 40%
  • PEG = 30 ÷ 40 = 0.75
  • 判斷：低估！雖然本益比高，但成長性更強

實例三：某價值股

  • 本益比 12 倍（看起來很便宜！）
  • 預估 EPS 成長率 5%
  • PEG = 12 ÷ 5 = 2.4
  • 判斷：高估！便宜是因為沒有成長性

5. PEG 三大陷阱

陷阱一：PEG 不是越小越好

PEG 太小（如 0.2）可能代表：

  • 公司獲利起伏太大，某一年的高成長不可持續
  • 市場知道某些風險你不知道
  • 公司可能先盛後衰

陷阱二：只能用在成長股

前提是公司 EPS 必須是正的、且預估會成長。如果 EPS 負的或衰退，PEG 沒意義。

陷阱三：成長率估計是關鍵

跟 DDM 一樣，PEG 最難的就是正確預估成長率：

  • 不要只用過去 1 年的數據
  • 建議用 3~5 年的預估或歷史年化
  • 保守估計，不要把短期高成長當常態

6. PEG + PE 雙指標選股法

怎麼做：

1. 篩選 PE < 20 的股票（排除太貴的）
2. 計算 PEG，篩選 PEG < 1 的（成長性被低估）
3. 交叉驗證：

  • PE 低 + PEG 低 → 價值成長股，最理想

適用時機：

  • 熊市撿便宜成長股的最佳工具
  • 多頭市場時大部分好公司 PEG 都 > 1，要有耐心等
  • 搭配產業趨勢判斷，不要單看數字

心得

這回合學了兩個估值武器：DDM 和 PEG，一個對付穩定配息的成熟公司，一個對付成長股。兩個共同的核心難題都是「正確預估未來」——DDM 要猜股利成長率，PEG 要猜 EPS 成長率。

以前覺得本益比 30 倍就是貴，現在看 PEG 才知道，如果 EPS 年成長 40%，30 倍本益比其實是被低估的。反過來，本益比 12 倍看起來便宜，但如果 EPS 只成長 5%，PEG = 2.4，其實是貴的。這就是為什麼彼得林區說「本益比相同的公司，成長性高的那家遠比成長性低的那家值得買」。

DDM 則是更保守的工具，特別適合中華電、中鋼這種穩定配息的定存股。但要注意，DDM 算出來的合理價通常偏低，在多頭市場可能永遠等不到進場點。所以安全邊際不是加越多越好——太保守等於永遠不進場，要取得平衡。

兩個工具的共通心法：寧可估錯在保守那邊，也不要因為樂觀預估而買貴了。

朝股票操作大師邁進中 📈## 第 6 回合學習 - 基本面分析：利息保障倍數、固定費用保障倍數與商譽減值

時間：06:22

文章：

  • 利息保障倍數是什麼？公式如何計算？如何用它來判斷公司好壞？
  • 商譽是什麼？能用來評估企業風險？
  • 商譽減值是什麼？對財報有什麼影響？

一、利息保障倍數（Times Interest Earned, TIE）

1. 利息保障倍數是什麼？

是什麼：衡量公司支付利息的能力，告訴你「公司賺的錢是利息的幾倍」。倍數越高，代表償債能力越強，公司越不容易因為付不出利息而倒閉。

公式：

利息保障倍數 = 稅前息前利潤（EBIT）÷ 利息費用
EBIT = 稅前淨利 + 利息費用

白話解釋：公司賺 500 萬、利息要付 100 萬，保障倍數就是 5 倍。代表公司賺的錢可以覆蓋 5 次利息，安全邊際很大。

2. 利息保障倍數判斷標準

倍數判斷意義
> 5 倍✅ 安全每付 100 元利息，背後至少有 500 元以上淨利
3~5 倍⚠️ 普通能付利息但空間有限
2~3 倍🔶 警戒稍有波動就可能付不出利息
< 2 倍🚨 危險流動比率<200% + 速動比率<100% + TIE<2 = 幾乎借不到錢
負值❌ 非常危險公司虧損，可能來自淨利為負

怎麼做：
1. 從損益表找 EBIT（稅前息前利潤）和利息費用
2. EBIT ÷ 利息費用 = 利息保障倍數
3. 至少看 3~5 年趨勢，持續下降就是紅旗
4. 跟同產業比較（不同產業標準差異大）
5. 搭配流動比率和速動比率一起看

實例：

  • 情境：A 公司 EBIT 2.5 億，利息費用 0.5 億
  • TIE = 2.5 ÷ 0.5 = 5 倍 ✅
  • 情境：B 公司 EBIT 1.2 億，利息費用 0.8 億
  • TIE = 1.2 ÷ 0.8 = 1.5 倍 🚨 非常危險

3. 利息保障倍數三大注意事項

注意一：負債比率高 ≠ TIE 低

  • 負債比裡面可能大部分是應付帳款（不需付息）
  • 要看負債結構，不是只看總負債

注意二：適合評估高資本支出公司

  • 電子業、製造業、運輸業：會借錢投資設備，TIE 很有意義
  • 零售業：現金周轉快、不太借錢，TIE 參考價值低

注意三：能還利息 ≠ 好公司

  • TIE 只看償債風險，不看獲利能力
  • 台灣高鐵曾經 TIE 很差，不是因為不賺錢，是帳上現金周轉不過來
  • 好公司 TIE 高不代表未來一定好，爛公司 TIE 低不代表不賺錢

適用時機：

  • ✅ 評估高負債企業的償債風險
  • ✅ 價值投資撿菸屁股時確認不會倒
  • ✅ 評估公司債（債券投資）品質
  • ❌ 尋找優質成長股（TIE 不是重點）

二、固定費用保障倍數（Fixed Charge Coverage Ratio, FCCR）

1. 固定費用保障倍數是什麼？

是什麼：利息保障倍數的進階版，不只看利息，還把租賃費用等固定支出一起算進去。更能反映公司的真實償債壓力。

公式：

固定費用保障倍數 =（EBIT + 租借債務）÷（全部利息費用 + 租借債務）

白話解釋：很多公司不只有利息壓力，還有租店面、租設備的固定支出。TIE 只看利息，但這些租賃費用一樣是每個月要付的錢。FCCR 把所有固定支出一起算，比 TIE 更全面。

2. FCCR vs TIE 比較

指標包含項目適用情境
TIE只看利息費用一般製造業、少租賃的公司
FCCR利息 + 租賃費用零售業、航空業、大量租賃的公司

3. FCCR 判斷標準

  • 至少要 ≥ 1，否則公司連固定支出都付不起
  • > 1.5 倍才算相對安全
  • 越高越好

怎麼做：
1. 先算 EBIT
2. 找到租賃費用（資產負債表上的「租賃負債」或附註揭露）
3. 代入公式計算
4. 跟同業比較、看趨勢

實例：

  • 情境：某連鎖餐飲集團，EBIT 8 億，利息費用 1 億，租賃費用 3 億
  • TIE = 8 ÷ 1 = 8 倍（看起來很安全！）
  • FCCR =（8 + 3）÷（1 + 3）= 11 ÷ 4 = 2.75 倍
  • 真正的安全邊際比 TIE 看到的少很多，因為租賃是隱形的固定支出

適用時機：

  • ✅ 零售、餐飲、航空等大量租賃的產業
  • ✅ 公司有大量經營租賃時（IFRS 16 已把租賃入帳，更能算）
  • ❌ 無租賃的公司（FCCR = TIE，沒有額外資訊）

三、商譽（Goodwill）與商譽減值

1. 商譽是什麼？

是什麼：公司併購另一家公司時，付出的溢價。買價超過被收購公司淨資產的部分，就是商譽。

公式：

商譽 = 收購價 − 被收購公司淨資產公允價值

白話解釋：你花 150 億買一家淨值只有 120 億的公司，那多付的 30 億就是商譽。這 30 億代表你認為這家公司的品牌、客戶關係、技術等無形資產值這個錢。但問題是——你真的買對了嗎？

2. 商譽的特點

特點說明
來源只來自併購溢價
能否獨立買賣❌ 不能，只能跟著公司走
是否攤銷台灣（IFRS）：不攤銷，每年做減值測試
破產時價值歸零，無法變現

3. 商譽高低判斷

怎麼做：
1. 查資產負債表上的商譽金額
2. 計算商譽佔總資產比例：商譽 ÷ 總資產
3. 商譽佔比 > 20% 就要警覺
4. 商譽佔比 > 50% 代表公司一半的「資產」都是併購溢價，風險極高

實例：

  • 迪士尼併購福斯影業：商譽暴增約 490 億美元
  • 松下併購三洋：收購價 6,600 億日元，淨資產僅 1,420 億日元，溢價 5,180 億日元。後來認列 2,500 億日元商譽減損
  • AOL 併購時代華納：商譽 1,280 億美元，2002 年認列 542 億美元商譽減值，全年虧損 987 億美元

4. 商譽減值（Goodwill Impairment）

是什麼：公司每年對商譽做減值測試，如果發現併購時買貴了，就要認列減值損失。

觸發條件：

  • 每年定期測試（至少一次）
  • 總體經濟惡化
  • 競爭加劇
  • 法規變動
  • 關鍵人員離開
  • 現金流量持續下降

對財報三大表的影響：

報表影響
損益表減值金額列為當期費用，淨利減少
資產負債表商譽金額減少，股東權益可能減少
現金流量表非現金支出，加回營業現金流（不影響真實現金）

關鍵概念：商譽減值不影響現金！只是會計上的認列。但損益表上會很難看。

5. 商譽減值 vs 商譽攤銷

項目商譽減值商譽攤銷
方式每年測試，有減損才認列按固定年限每年折減
台灣法規IFRS 規定使用此法不採用
美國法規GAAP 主要採用2014 年後私營公司可選
優點符合實際狀況（價值不是線性消失）簡單明確，逐年反映
缺點管理層可能不願認列（打臉自己）年限設定困難

6. 商譽實戰四步驟

怎麼做：

1. 看商譽佔比：商譽 ÷ 總資產，> 20% 警覺、> 50% 高風險
2. 看併購歷史：公司過去的併購是否有創造造價值？併購後營收/獲利是否成長？
3. 看減值頻率：頻繁減值 = 併購決策品質差；偶爾一次可能只是認列
4. 用「有形資產」重算淨值：把商譽從資產中扣除，重新算本淨比，看公司真實體質

實例：

  • 情境：某公司總資產 500 億，其中商譽 200 億，負債 300 億
  • 帳面淨值 = 500 - 300 = 200 億（看起來 OK）
  • 扣除商譽後淨值 = 500 - 200 - 300 = 0（根本沒有真實淨值！）
  • 如果公司破產，商譽歸零，股東什麼都拿不回來

查詢工具：

  • 台股：Goodinfo → 資產負債表 → 無形資產項下
  • 美股：Morningstar → Financials → Balance Sheet

心得

這回合學了三個風險評估工具：利息保障倍數、固定費用保障倍數、商譽減值。三個工具本質上都在回答同一個問題：「這家公司會不會出事？」

利息保障倍數是入門款，告訴你公司能不能付利息。但它的問題是只看利息——對於那些租了一堆店面、設備的公司來說，租賃費用可能比利息還大，TIE 就會失真。這時候 FCCR 才能反映真正的償債壓力。連鎖餐飲、零售業這些重租賃的行業，FCCR 比 TIE 有意義得多。

商譽這東西更有意思了。表面上它是「資產」，但仔細想想，它其實是「以前併購買貴了的紀錄」。一間公司商譽佔總資產 50%，意思就是這間公司一半的「資產」都是以前多付的錢。如果公司破產，這些商譽直接歸零，股東可能什麼都拿不到。AOL 認列 542 億美元商譽減值那個案例太經典了，網路泡沫時靠高估值併購，泡沫破了才發現買貴了，一次認列直接虧 987 億。

最實用的做法是：看到商譽高的公司，把商譽從資產中扣除重算淨值。如果扣完之後淨值變成負的，這間公司的「真實體質」可能遠不如帳面上看的那麼健康。搭配利息保障倍數和 FCCR 一起看，就能建立一套完整的「公司不會倒」防線。

一、營業槓桿度（Degree of Operating Leverage, DOL）

1. DOL 是什麼？

是什麼：衡量公司營運仰賴固定成本的程度。當營收變動 1% 時，營業利益（EBIT）會變動多少百分比。DOL 越高，代表固定成本佔比越大，獲利對營收的敏感度越高。

公式：

DOL = 營業利益變動率 ÷ 營收變動率
    = (營收 − 變動成本) ÷ 營業利益
    = (營業利益 + 固定成本) ÷ 營業利益
    = (EBIT + F) ÷ EBIT

白話解釋：一家餐廳的房租、設備折舊、基本人事費是固定成本，食材是變動成本。如果房租佔比很高（DOL 高），那營收增加 10% 時，因為固定成本不會增加，多出來的營收幾乎全部變成利潤，利潤增幅可能高達 30-50%。反過來說，營收下降 10% 時，利潤也會暴跌 30-50%。這就是槓桿——放大獲利，也放大虧損。

2. DOL 判斷標準

DOL 值判斷意義
1~2 倍低槓桿服務業、買賣業，固定成本佔比低，獲利穩定但成長爆發力弱
2~5 倍中等槓桿一般製造業，獲利與營收同步變動
5~10 倍高槓桿重資本製造業（半導體、鋼鐵），獲利彈性大
> 10 倍極高槓桿接近損益兩平點，微幅營收成長就能讓獲利暴增，但反轉也極慘
負值⚠️ 虧損中公司營業利益為負，DOL 無意義

關鍵概念：DOL 越高 ≠ 越好或越差

  • 高 DOL 是雙面刃：景氣好時獲利暴增，景氣差時獲利暴跌
  • 低 DOL 是穩定牌：獲利波動小，但也缺乏成長爆發力
  • 投資邏輯：在景氣谷底找高 DOL 公司，景氣反轉時獲利增幅最大

3. DOL 計算實戰

怎麼做：
1. 從損益表抓營業收入和營業利益
2. 估算固定成本（人事費用 + 折舊 + 攤銷）或用簡易公式
3. DOL = (營業利益 + 固定成本) ÷ 營業利益
4. 至少看 3~5 年趨勢，DOL 上升代表固定成本佔比增加
5. 跟同產業比較才有意義

實例一：低 DOL 公司（零售業）

  • 營收 500 億，變動成本 400 億，固定成本 50 億
  • 營業利益 = 500 - 400 - 50 = 50 億
  • DOL = (50 + 50) ÷ 50 = 2 倍
  • 營收增加 10% → 營業利益增加約 20%

實例二：高 DOL 公司（半導體設備廠）

  • 營收 100 億，變動成本 40 億，固定成本 50 億
  • 營業利益 = 100 - 40 - 50 = 10 億
  • DOL = (10 + 50) ÷ 10 = 6 倍
  • 營收增加 10% → 營業利益增加約 60%！

實例三：接近損益兩平的公司

  • 營收 100 億，變動成本 40 億，固定成本 58 億
  • 營業利益 = 100 - 40 - 58 = 2 億
  • DOL = (2 + 58) ÷ 2 = 30 倍！
  • 營收增加 5% → 營業利益增加約 150%（從 2 億變 5 億）
  • 但營收減少 5% → 營業利益直接轉虧

4. 岱稜（3303）實戰案例

  • 2018 年營業利益大幅成長後，DOL 仍有 9.13（年報數據）
  • 2019Q4 開出新產能，營收微幅成長時，獲利成長幅度遠大於營收增幅
  • 這就是高 DOL 的威力：越過損益兩平點後，進入「快速拉升區」
  • 投資邏輯：在產能開出前、股價回落時提前布局，等營收起飛時享受獲利暴增

注意：年報的 DOL 數字可能跟自己算的有落差，因為年報對固定/變動成本的定義可能不同。Solid Investing 部落格用三種方式計算岱稜的 DOL，結果分別是 3.85、5 和 6，都低於年報的 9.13。所以 DOL 是方向性參考，不必執著於絕對數字。

5. DOL 與損益兩平點的關係

關鍵洞察：DOL 在損益兩平點附近會趨近無限大。

怎麼做：
1. 計算損益兩平營收 = 固定成本 ÷ (1 - 變動成本率)
2. 當前營收越接近損益兩平點，DOL 越高
3. 越過損益兩平點後，DOL 會快速下降（因為利潤開始累積）
4. 投資時機：在營收剛越過損益兩平點時進場，此時獲利彈性最大

適用時機：

  • ✅ 景氣循環谷底找反轉股
  • ✅ 新產能開出前提前布局
  • ✅ 評估公司獲利對營收的敏感度
  • ❌ 不適合用來找穩定成長股
  • ❌ 虧損公司 DOL 無意義

二、財務槓桿度（Degree of Financial Leverage, DFL）

1. DFL 是什麼？

是什麼：衡量公司使用負債融資的程度。當營業利益（EBIT）變動 1% 時，每股盈餘（EPS）會變動多少百分比。DFL 越高，代表利息支出佔比越大，EPS 對 EBIT 的敏感度越高。

公式：

DFL = EBIT ÷ (EBIT − I)
    = EPS 變動率 ÷ EBIT 變動率

其中 I = 利息費用

若有特別股：

DFL = EBIT ÷ (EBIT − I − D_pf ÷ (1 − T))

白話解釋：公司借了 100 億，每年利息 5 億。如果 EBIT 是 10 億，DFL = 10 ÷ (10 - 5) = 2 倍。代表 EBIT 增加 10%，EPS 會增加 20%。但 EBIT 減少 10%，EPS 也會減少 20%。利息就像一個放大器，把 EBIT 的變動放大到 EPS 上。

2. DFL 判斷標準

DFL 值判斷意義
1 倍無財務槓桿沒有負債或利息極低，EPS 變動 = EBIT 變動
1~1.5 倍低槓桿利息佔比低，財務風險小
1.5~2.5 倍中等槓桿一般製造業常見範圍
> 2.5 倍高槓桿利息佔比高，財務風險大，EBIT 小幅下滑就可能吃掉全部利潤
趨近無限大⚠️ 極度危險EBIT ≈ 利息費用，幾乎付不出利息

3. DFL 計算實戰

怎麼做：
1. 從損益表找 EBIT 和利息費用
2. DFL = EBIT ÷ (EBIT − I)
3. 至少看 3~5 年趨勢
4. 搭配利息保障倍數（TIE = EBIT ÷ I）一起看
5. DFL 和 TIE 互為倒數關係：DFL = TIE ÷ (TIE - 1)

實例一：低 DFL 公司（台積電）

  • EBIT 1.5 兆，利息費用 200 億
  • DFL = 1,500 ÷ (1,500 - 20) = 1,500 ÷ 1,480 ≈ 1.01 倍
  • 利息幾乎不影響 EPS，財務風險極低

實例二：高 DFL 公司（高負債營建股）

  • EBIT 10 億，利息費用 7 億
  • DFL = 10 ÷ (10 - 7) = 10 ÷ 3 ≈ 3.33 倍
  • EBIT 減少 30%，EPS 就會減少約 100%（直接轉虧）

實例三：「先收後付」的零售業

  • 表面負債比很高，但大量是應付帳款（不需付息）
  • 實質利息費用很低，DFL 不高
  • Costco、沃爾瑪：負債比 > 60%，但 DFL 可能只有 1.1~1.3
  • 結論：負債比高 ≠ DFL 高，要看負債結構

4. DFL 與公司財務槓桿策略

公司使用財務槓桿的三大理由：
1. 減少自有資金壓力：設備投資回收期長，借錢可以分散風險
2. 加速擴張：用借來的錢搶市佔、買設備
3. 稅賦優勢：利息支出可以抵稅，降低實質稅率

但 DFL 過高的風險：
1. EBIT 下降時，EPS 跌幅被放大
2. 還不出利息可能導致債務違約
3. 景氣循環產業在高點借太多錢，谷時就是災難

適用時機：

  • ✅ 評估高負債企業的財務風險
  • ✅ 判斷 EPS 對營運波動的敏感度
  • ✅ 搭配 TIE 和流動比率做完整償債評估
  • ❌ 沒有負債的公司（DFL = 1，沒資訊量）

三、總槓桿度（Degree of Total Leverage, DTL）

1. DTL 是什麼？

是什麼：結合營業槓桿和財務槓桿的總效果。當營收變動 1% 時，EPS 會變動多少百分比。DTL 就是 DOL 和 DFL 的乘積。

公式：

DTL = DOL × DFL
    = (EBIT + F) ÷ (EBIT − I)
    = EPS 變動率 ÷ 營收變動率

白話解釋：營收變動先被 DOL 放大成 EBIT 變動，再被 DFL 放大成 EPS 變動。兩層放大效果相乘，就是 DTL。一家 DOL = 3、DFL = 2 的公司，DTL = 6，代表營收增加 1%，EPS 會增加 6%。

2. DTL 判斷標準

DTL 值判斷代表意義
1~2 倍低總槓桿服務業、無負債公司，EPS 穩定但成長彈性小
2~5 倍中等總槓桿一般製造業，適度放大效果
5~10 倍高總槓桿重資本+高負債，獲利波動劇烈
> 10 倍極高總槓桿接近損益兩平+高利息，景氣反轉即虧損

3. DTL 計算實戰

怎麼做：
1. 先算 DOL 和 DFL
2. DTL = DOL × DFL
3. 分別評估營業風險和財務風險
4. 找出哪一層槓桿是主要風險來源

實例：某半導體設備廠

  • DOL = 5（固定成本佔比高）
  • DFL = 1.8（適度負債）
  • DTL = 5 × 1.8 = 9
  • 營收增加 10% → EPS 增加約 90%
  • 營收減少 10% → EPS 減少約 90%
  • 主要風險來源：營業槓桿（DOL 5 遠大於 DFL 1.8 的影響）
  • 對策：關注營收趨勢而非財務結構

實例：某高負債零售集團

  • DOL = 1.5（固定成本佔比低）
  • DFL = 4（高利息支出）
  • DTL = 1.5 × 4 = 6
  • 營收增加 10% → EPS 增加約 60%
  • 營收減少 10% → EPS 減少約 60%
  • 主要風險來源：財務槓桿（DFL 4 是主要放大器）
  • 對策：關注償債能力和利息保障倍數

4. 三大槓桿度的投資應用

應用一：景氣反轉股篩選

怎麼做：
1. 找 DOL > 5 的公司（高營業槓桿）
2. DFL < 2（財務風險可控）
3. 營收開始觸底反彈
4. DTL 高但來自 DOL 而非 DFL → 景氣反轉時 EPS 爆發力最大
5. 岱稜就是典型案例：DOL 9.13 + 低 DFL = 營收微增、獲利暴增

應用二：風險評估

怎麼做：
1. 計算 DTL，找出總體風險大小
2. 拆解 DOL 和 DFL，定位風險來源
3. DOL 高 → 關注營收趨勢、產能利用率、景氣位置
4. DFL 高 → 關注利息保障倍數、現金流、到期債務
5. 兩者都高 → 景氣好時 EPS 瘋狂成長，但反轉時是核彈級災難

應用三：損益兩平點策略

怎麼做：
1. 找接近損益兩平點的公司（DOL 極高）
2. 確認營收正在回升（不是繼續惡化）
3. DFL 低（不會因為利息壓力而倒閉）
4. 進場時機：營收剛越過損益兩平點
5. 出場時機：DOL 開始顯著下降（利潤已經穩定累積）

5. 三大槓桿度常見陷阱

陷阱一：DOL 不是常數

  • DOL 在不同營收水準下會大幅變化
  • 接近損益兩平點時 DOL 極高，遠離時 DOL 下降
  • 不要用過去的 DOL 預測未來，要根據當前營收水準重新計算

陷阱二：固定/變動成本劃分不精確

  • 年報的劃分方式可能跟你自己算的不同
  • 人工成本部分固定、部分變動，難以精確切割
  • DOL 是方向性指標，不必執著於精確數字

陷阱三：忽略產業特性

  • 製造業天然 DOL 高，服務業天然 DOL 低
  • 跨產業比 DOL 沒意義
  • 同產業內比較，看誰的 DOL 在改善（下降 = 固定成本佔比降低 = 風險降低）

查詢工具：

  • 台股年報：公開資訊觀測站 → 年報 → 營運槓桿度（部分公司有揭露）
  • Goodinfo：損益表可自行計算
  • 財報狗：可查財務槓桿相關指標
  • 自行計算：從損益表估算固定成本（人事 + 折舊 + 攤銷）

心得

三大槓桿度本質上是在回答一個問題：「公司的獲利有多容易被放大？」

DOL 說的是「營運端的放大器」——固定成本佔比越高，營收的些微波動就會被放大成利潤的劇烈波動。這解釋了為什麼半導體設備廠在景氣反轉時 EPS 能暴增好幾倍，也解釋了為什麼景氣下行時這些公司 EPS 跌幅也特別慘。岱稜的案例太生動了——DOL 9.13 意味著營收每增加 1%，利潤增加 9%，這種公司在產能開出、營收起飛的時刻就是金礦。

DFL 說的是「融資端的放大器」——利息費用越高，EBIT 的波動就被更大地放大到 EPS 上。但 DFL 有個很關鍵的陷阱：負債比高不代表 DFL 高。Costco 負債比超過 60%，但大部分是應付帳款，利息支出極低，DFL 可能只有 1.1。真正讓 DFL 飆高的是高利息的金融負債，不是營運產生的應付帳款。

最實用的投資邏輯是這個：在景氣谷底找高 DOL + 低 DFL 的公司。高 DOL 確保景氣反轉時獲利彈性最大，低 DFL 確保不會因為利息壓力在谷底倒閉。這種組合就是「不死+爆發力」的黃金配比。反過來說，高 DOL + 高 DFL = 核彈，景氣好時 EPS 瘋狂成長，景氣差時直接爆炸，這種公司只能在景氣明確上行時短暫持有。
