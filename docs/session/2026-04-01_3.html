核心知識

券資比定義

是什麼：券資比 = 融券餘額 ÷ 融資餘額，反映市場多空情緒與潛在軋空力道。

怎麼看：

  • 券資比 > 30%：被大量放空，市場看空情緒濃
  • 券資比突然暴增：需特別關注，可能是散戶過度看空、軋空前兆
  • 券資比從高檔回落：融券開始回補，股價可能止跌回升

券資比與股價走勢

反向指標特性：

  • 券資比上升 = 散戶看空 → 但過度看空往往是反向訊號
  • 融券必須回補 = 這些空單是未來的買盤
  • 軋空行情 = 股價不跌反漲，融券戶被迫回補推升股價

關鍵判斷：
1. 股價上漲 + 融券增加 = 散戶放空、主力拉抬 → 潛在軋空
2. 股價下跌 + 融資增加 = 散戶接刀、主力出貨 → 風險高
3. 股價低檔 + 融資暴增 = 可能主力進場 → 起漲訊號
4. 股價高檔 + 融券暴增 = 可能主力放空 → 回檔訊號

融資融券搭配主力買賣超

低檔融資暴增 + 主力買超：

  • 判斷：主力用融資提高槓桿進場
  • 操作：跟進做多，等融資與主力買超減少時出場

高檔融券暴增 + 主力賣超：

  • 判斷：主力看空，用融券放空
  • 操作：避開或跟進放空，等融券回補時平倉

軋空行情識別

軋空三要素：
1. 股價上漲：股價走勢與融券方向相反
2. 融券增加：散戶不斷放空
3. 券資比攀升：空單累積，未來回補力道強

實例：

  • 情境：十銓（4967）從 85 元起漲，融券持續增加
  • 操作：股價價量齊揚 + 融券上升 = 軋空行情，與融券對做做多
  • 結果：融券被迫回補，股價易漲難跌

融資融券的起漲與起跌訊號

股價起漲訊號：

  • 股價在低檔
  • 融資餘額開始上升
  • 主力同步買超
  • 融資暴增日：主力進場日

股價起跌訊號：

  • 股價在高檔
  • 融資餘額創新高不減
  • 散戶融資追價
  • 融資不減 + 股價滞漲 = 出場訊號

案例一：緯創（3231）融資起漲

  • 股價：30-40 元低檔
  • 融資：開始上升
  • 結果：股價大漲至 150 元
  • 警訊：150 元時融資還在創新高 → 留意回檔

案例二：廣達（2382）融券起跌

  • 股價：8 月來到近 300 元波段高點
  • 融券：來到波段新高
  • 結果：股價回檔至 200 元
  • 訊號：200 元時融券下降 → 融券回補，股價反彈

案例三：鴻海（2317）主力融資進場

  • 日期：3/12
  • 融資暴增：33,654 張

  • 判斷：低檔融資暴增 = 主力進場

案例四：大亞（1609）主力融券放空

  • 出場：融資與主力買超減少 → 股價回檔

操作要點

不要做的：

  • 股價高檔 + 融資創新高 ≠ 追價，是出場訊號
  • 股價多頭趨勢 ≠ 放空，永遠猜不到高點
  • 只看券資比高低 ≠ 進場依據，要看「突然提高」

要做的：

  • 低檔融資暴增 + 主力買超 → 跟進做多
  • 高檔融券暴增 + 主力賣超 → 避開或放空
  • 價量齊揚 + 融券上升 → 軋空行情，與融券對做
  • 券資比突然提高 → 關注，可能是飆股前兆

心得

這回合深入學習了券資比的進階應用，關鍵在於「券資比突然提高」才是訊號，不是看絕對數值。而且融資融券不能單獨看，要搭配主力買賷超一起判斷——低檔融資暴增 + 主力買超才是主力進場，高檔融券暴增 + 主力賣超才是主力看空。

最實用的概念是「軋空行情」：散戶放空、主力拉抬，股價不跌反漲，融券戶被迫回補推升股價。這就是為什麼「券資比高的股票容易上漲」——那些空單都是未來的買盤。
