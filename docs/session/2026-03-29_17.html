一、樣本外測試 (OOS) 核心概念

什麼是樣本外測試？

定義：將歷史數據劃分為兩部分

  • 樣本內 (In-Sample)：用於策略開發、參數優化
  • 樣本外 (Out-of-Sample)：用於驗證策略穩健性

核心目的：檢驗策略在「未見過」的數據上是否有效

為什麼重要？三大理由

1. 避免過度擬合：策略在歷史數據完美 ≠ 在未來有效
2. 評估泛化能力：好的策略應在不同市場環境保持穩定
3. 提高可靠性：全面了解策略優缺點

三種測試方法

方法原理適用情境
**固定分割法**前 70% 樣本內、後 30% 樣本外數據量大、市場環境穩定
**滾動窗口法**固定長度窗口滾動，每次重新測試模擬實盤動態調整
**交叉驗證法**分多個子集，輪流作驗證集全面評估穩健性

二、OOS 驗證流程七步驟

步驟 1：數據集劃分

怎麼做：

  • 歷史數據按時間順序分割
  • 樣本內約 70-80%，樣本外約 20-30%
  • 樣本外時間必須晚於樣本內

實例：

  • 2010-2022 年：樣本內（參數優化用）
  • 2023-2025 年：樣本外（驗證用）

關鍵：樣本外數據開發過程中「絕對不能」偷看

步驟 2：樣本內策略開發

怎麼做：
1. 在樣本內數據建構策略邏輯
2. 測試不同參數組合
3. 根據回測結果改進策略

注意：

  • 不要過度追求完美擬合
  • 參數越多越容易過度擬合
  • 關注策略複雜度

步驟 3：樣本外數據驗證

怎麼做：
1. 凍結策略參數（不能再調整！）
2. 用樣本外時間範圍運行回測
3. 記錄績效指標

關鍵：驗證的是「預測能力」，不是歷史表的現

步驟 4：績效指標評估

關鍵指標對照：

指標用途警訊
**夏普比率**風險調整後收益樣本外 < 樣本內 20% 以上
**最大回撤**最大虧損幅度樣本外回撤更大
**勝率**交易成功比例樣本外勝率大幅下降
**盈虧比**盈利/虧損交易比樣本外盈虧比惡化

過度擬合跡象：樣本外績效明顯差於樣本內

步驟 5：風險管理與壓力測試

怎麼做：
1. 模擬極端市場（股災、金融危機）
2. 觀察策略抗風險能力
3. 制定應對措施

壓力測試情境：

  • 2008 金融海嘯情境
  • 2020 疫情熔斷情境
  • 2022 升息熊市情境

步驟 6：迭代優化（如果需要）

判斷標準：

  • 樣本外夏普比率 > 1.0：可接受
  • 樣本外夏普比率 < 0.5：需要優化

優化方向：

  • 簡化策略邏輯
  • 減少參數數量
  • 嘗試不同指標組合
  • 每次迭代嚴守 OOS 原則

步驟 7：模擬交易監控

為什麼需要：回測無法模擬：

  • 交易成本（手續費、稅）
  • 滑價（實際成交價差）
  • 市場衝擊（大單影響）

實戰建議：

  • 模擬交易至少 1-3 個月
  • 密切監控策略績效
  • 根據實際狀況微調

三、過度擬合的識別與避免

過度擬合四大跡象

1. 樣本內績效完美：勝率 90%+、夏普比率 3+
2. 樣本外績效驟降：樣本外表現差樣本內一大截
3. 參數過多：超過 5-7 個可調參數
4. 規則過於複雜：一堆例外條件、特殊規則

避免過度擬合五大方法

方法具體做法
**簡化策略**減少指標數量，避免複雜規則
**正則化**參數優化時加入懲罰項，限制複雜度
**交叉驗證**K-fold 驗證，評估不同數據子集表現
**控制優化範圍**不在歷史數據上過度調整
**納入交易成本**回測時強制計入手續費、滑價

數據洩露陷阱

定義：不小心讓樣本外數據「污染」開發過程

常見錯誤：

  • 在樣本外看過數據後微調參數
  • 用全體數據選擇最佳參數，再分樣本內外驗證
  • 樣本外多次迭代調整

嚴格原則：樣本外數據「只用一次」驗證，看過就不能再調！

四、實戰案例分析

案例一：移動平均線交叉策略

策略規則：

  • 短期均線向上穿越長期均線 → 買入
  • 短期均線向下穿越長期均線 → 賣出

OOS 測試流程：
1. 數據分割：2015-2021 樣本內，2022-2024 樣本外
2. 樣本內優化：測試參數組合 (5,20)、(10,30)、(20,50)
3. 選擇最佳：(10,30) 在樣本內表現最佳
4. 樣本外驗證：凍結參數，在 2022-2024 數據測試

結果判讀：

  • 樣本內夏普比率：1.8
  • 樣本外夏普比率：0.9
  • 判定：績效差距大，可能過度擬合

優化方向：

  • 加入趨勢濾鏡（避免震盪盤假訊號）
  • 減少參數優化範圍
  • 考慮交易成本

案例二：RSI 指標策略

策略規則：

  • RSI < 30 超賣區 → 買入
  • RSI > 70 超買區 → 賣出

發現問題（透過 OOS 測試）：

  • 熊市情境：樣本外最大回撤 -45%
  • 牛市情境：樣本外勝率 55%
  • 震盪市：樣本外勝率 68%

優化方案：

  • 動態倉位調整：熊市降低倉位
  • 趨勢濾鏡：加入均線判斷趨勢
  • ATR 止損：用波動度動態設定止損

五、回測平台實務要點

交易成本必須納入

計算方式：

  • 手續費：0.1425%（買+賣）
  • 證交稅：0.1%（賣出時）
  • 滑價：估算一檔價差

實例：

  • 交易成本約 0.3-0.5%
  • 日內策略每日交易，一年成本可能 > 50%
  • 忽略成本會導致回測績效虛高

績效報告必看指標

指標意義基準
**夏普比率**每單位風險的報酬> 1.0 可接受
**最大回撤**最大虧損幅度< 20% 較安全
**勝率**成功交易比例40-60% 正常
**盈虧比**平均盈利/平均虧損> 1.5 較理想
**交易次數**樣本足夠性> 50 次才有統計意義

六、常見錯誤與迷思

錯誤一：樣本外反覆調整

錯誤做法：樣本外績效不好 → 調整參數 → 再驗證 → 再調整

正確做法：樣本外只看一次，不好就回到樣本內重新設計

錯誤二：忽略交易成本

迷思：「反正回測賺錢，實盤一定賺」

現實：高頻策略每天交易，手續費可能吃掉所有利潤

錯誤三：過度追求高勝率

迷思：「勝率 80% 才是好策略」

現實：

  • 勝率 40% + 盈虧比 3:1 → 期望值 +20%
  • 勝率 80% + 盈虧比 0.2:1 → 期望值 -4%（虧損！）

錯誤四：參數越多越好

迷思：「加更多條件，績效一定更好」

現實：每個參數都在「學習」歷史數據的噪音，不是學習真正的規律

心得

回測這件事，說穿了就是「對過去作弊」。樣本外測試的重要性在於：防止你真的騙了自己。

三大關鍵收穫：

1. 樣本外只看一次：這是鐵律。看過就不能當作「未來數據」了。很多人在這步驟犯錯，反覆調整直到樣本外也漂亮——這不是驗證，這是自欺欺人。

2. 交易成本是殺手：一個看起來年化 50% 的策略，加上手續費、滑價，可能只剩 10%，甚至虧損。回測時不計成本，實盤就是驚喜變驚嚇。

3. 簡單策略更有優勢：參數越多，過度擬合風險越高。寧可要一個簡單但穩定的策略，也不要一個複雜但在樣本外崩潰的策略。

另外，壓力測試是很多人忽略的環節。策略在正常市況可能 OK，但在股災、熔斷時會怎樣？不用親身經歷，用歷史數據模擬就知道了。

最後，模擬交易這步驟不能省。回測再完美，也無法模擬真實市場的滑價、流動性不足、心理壓力。先模擬 1-3 個月，確認策略在「接近真實」的環境下有效，再投入真金白銀。
