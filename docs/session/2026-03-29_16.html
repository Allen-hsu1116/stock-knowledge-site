核心知識

KD鈍化基本概念

是什麼：KD值長期停留在超買區（80以上）或超賣區（20以下），導致原本的買賣訊號失效。

為什麼會鈍化：

  • 當股價持續創高，收盤價一直高於前幾天價格，KD值被持續拉高，卡在80以上
  • 當股價持續下跌，KD值被持續拉低，卡在20以下

鈍化的本質：市場在告訴你「現在的行情太強了，我已經無法判斷了，別再只看我了！」

高檔鈍化順勢操作策略

定義：K值連續三根站在80以上 = 高檔鈍化

傳統誤解：教科書說KD>80是賣點 → 錯！那是強勢股的特徵

羅威心法：

  • 高檔鈍化 = 強者恆強
  • KD能>80，代表股票處於強勢
  • 回檔不破前波低點，將來還會創新高
  • 不是賣點，反而是找買點的好時機

實戰操作：

情境判斷條件操作
高檔鈍化中K值連續3根以上>80觀望或沿均線持有
鈍化結束+回檔KD跌破80+回檔不破前低長線保護下找買點
鈍化結束+死亡交叉K線跌破D線+跌破支撐明確賣出訊號

實例：

  • 宜特(3289)：2014/1/8週KD高檔鈍化到2017/7/28，股價從33.7元漲到91.9元，漲近3倍
  • 可成(2474)：2013/5/31結束週KD鈍化，月KD向上保護，盤整後從約131元漲到289元

低檔鈍化反轉判斷

定義：K值連續三根站在20以下 = 低檔鈍化

意義：股價越來越弱勢，空頭格局

操作提醒：

  • 低檔鈍化 ≠ 買點！
  • 空頭格局中，「低檔鈍化」反彈不過前波高點，會繼續創新低
  • 反彈要找空點

反轉訊號：

  • 低檔鈍化結束
  • K線向上突破D線（黃金交叉）
  • 股價突破關鍵壓力位
  • 三者同時出現 = 買入訊號

KD鈍化三大應對策略

策略一：結合趨勢指標確認

原理：鈍化本身就是「趨勢非常強」的訊號，應搭配趨勢指標而非超買超賣指標

搭配方式：

指標判斷條件意義
均線股價在所有均線之上+多頭排列確認強勢上漲趨勢
MACDKD高檔鈍化+MACD紅柱持續放大上漲動能強勁

策略二：尋找鈍化結束訊號

高檔鈍化結束賣點：

  • KD已在高檔鈍化一陣子
  • K線向下跌破D線（死亡交叉）
  • 股價跌破關鍵支撐位
  • 同時出現 = 非常明確的賣出訊號

低檔鈍化結束買點：

  • KD在低檔鈍化一段時間
  • K線向上突破D線（黃金交叉）
  • 股價突破關鍵壓力位
  • 三者同時出現 = 明確買入訊號

策略三：觀察價格行為提前預警

高檔鈍化期間：

  • 出現爆量收黑K → 買方力量衰竭警訊
  • 長上影線 → 上攻無力
  • 即使KD還在高檔，提高警覺

低檔鈍化期間：

  • 出現爆量收紅K → 賣方力量耗盡
  • 長下影線 → 支撐強勁
  • 可開始尋找反彈機會

長線保護短線概念

定義：大時間框架趨勢保護小時間框架操作

實戰應用：

  • 月KD向上時，週KD高檔鈍化結束後的回檔 = 只是回檔，不是轉空
  • 週KD向下時，日KD低檔鈍化的反彈 = 只是反彈，不是轉多

時間框架對照表：

母框架趨勢子框架鈍化操作方向
月KD向上週KD高檔鈍化結束回檔找買點
月KD向下週KD低檔鈍化結束反彈找空點

KD鈍化+濾網策略（進階）

核心問題：KD>80做多、KD<20做空是「追漲殺跌」策略，震盪盤會「多空雙巴」

濾網原理：

  • 震盪行情：高低點差距小
  • 趨勢行情：高低點差距大

艾比策略：

  • KD鈍化策略要避免在「震盪行情」進單
  • 使用ATR或高低點濾網過濾震盪盤
  • 高低點差距小時 → 不下單
  • 高低點差距大時 → 才進場

反向思考：

  • 直觀：價格高低點差距大才下單 → 趨勢已走完才進場，沒賺頭
  • 正確：高低點差距小時不下單 → 避開震盪，保留資金給趨勢行情

策略失效與優化

失效徵兆：

  • 權益數長期不再創高
  • 持平時間超過回測時的「沒賺錢時段」
  • 市場波動率改變

優化方向：

  • 核心（KD+濾網）不調整
  • 加入追蹤止損：讓反轉時更快出場
  • 加入加碼：金字塔加碼放大獲利
  • 重新優化參數

參數調整原則：

  • 原本止損距離遠 → 縮短，因為有追蹤止損
  • 原本沒有TP → 加上合理的TP
  • 加碼手數：越加越小（金字塔型）

心得

KD鈍化的關鍵在於「框架轉換」：

1. 不要用鈍化的指標繼續判斷：KD卡在80以上時，它的超買訊號已經失效，這時候反而要看趨勢方向。

2. 鈍化是強勢的證明：能夠連續3根以上K線站在80以上，這本身就是強者恆強的表現。羅威的觀點徹底顛覆了「KD>80就該賣」的教條。

3. 層級思維很重要：長線保護短線的概念，讓你在週KD高檔鈍化結束時不會恐慌殺出，而是等待月KD方向的確認。

4. 價格永遠是老大：當指標鈍化失效時，回歸最原始的價格行為——爆量、影線、突破——這些才是真實的訊號。

5. 濾網是紀律的具象化：高低點差距濾網的本質，是用數字定義「什麼行情不做」，避免在震盪盤中被雙巴。這比人性判斷可靠多了。

這回合學到最重要的一點：技術指標不是萬能的，每個指標都有它的「盲區」。KD鈍化就是它的盲區，而這個盲區本身卻是最強趨勢的特徵。 認識盲區，才能跳脫框架。
