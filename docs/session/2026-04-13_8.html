2B 法則與支撐壓力互換

第 8 回合學習 - 2B 法則與支撐壓力互換

  • 如何在道氏理論中使用2B法則抓到趨勢轉折

2B 法則核心概念

什麼是 2B 法則

由 Victor Sperandeo 在《專業投機原理》中提出，用來判斷趨勢反轉的技術法則。核心邏輯：當價格突破前高（或前低）後，無法持續並迅速回轉，代表趨勢可能反轉。

本質就是捕捉「假突破」——市場試探前高前低失敗，就是反轉訊號。

2B 法則兩種情境

上升趨勢中的 2B（看空訊號）：
1. 價格突破前高，創下新高點
2. 短時間內無法繼續上漲
3. 迅速回跌，跌破前高位置
4. → 趨勢可能反轉向下，進場做空

下降趨勢中的 2B（看多訊號）：
1. 價格跌破前低，創下新低點
2. 短時間內無法繼續下跌
3. 迅速反彈，漲破前低位置
4. → 趨勢可能反轉向上，進場做多

關鍵條件：假突破的認定

不是所有突破再跌破都叫 2B！ 必須符合：

1. 幅度要夠小 — 突破前高前低後，超出幅度不能太大（一般 1~2% 以內），否則可能是真突破後的正常回檔
2. 時間要夠短 — 突破後迅速回轉，通常 1~3 根 K 線內就跌破（漲破）前高（前低），拖太久就不算
3. 要確認 K 線收線 — 不是盤中瞬間刺一下就算，要等那根 K 線收盤確認

實例判斷：

  • ✅ 某股票前高 100 元，盤中衝到 101 元，當天收盤回到 99.5 元 → 符合 2B
  • ❌ 某股票前高 100 元，漲到 108 元盤整三天才跌回 99 元 → 不是 2B，是真突破後回檔

2B 法則 vs 123 法則

比較123 法則2B 法則
入場時機較晚（需確認新低/新高）較早（假突破即入場）
準確度較高較低（可能只是盤整）
風險較小較大
本質高低點不再上移/下移且確認高低點不再上移/下移（更早判斷）

2B 是 123 的特殊形態，可以看作 123 法則的早期版本。

順勢 2B vs 逆勢 2B

順勢 2B（可用）：

  • 上升趨勢中，價格跌破支撐區但迅速收回 → 買入機會
  • 下降趨勢中，價格突破壓力區但迅速回落 → 賣出機會

逆勢 2B（不要用）：

  • 上升趨勢中，價格突破壓力區但迅速跌破 → 不要做空！大趨勢仍在上升
  • 下降趨下降趨勢中，價格跌破支撐區但迅速漲破 → 不要做多！大趨勢仍在下降

原則：順勢 2B 才操作，逆勢 2B 只當警示。

2B 法則停損設定

2B 法則的停損非常明確：

  • 上升趨勢做空：停損設在突破時的新高點（假突破的最高點）
  • 下降趨勢做多：停損設在突破時的新低點（假突破的最低點）

優勢：停損點明確、距離近、風險可控。如果抓到反轉，利潤遠大於停損。

實例：

  • 某股前高 150 元，假突破到 152 元後回落 → 做空進場 150 元，停損 152 元（風險 2 元）
  • 如果趨勢真的反轉下跌到 130 元 → 獲利 20 元，風報比 1:10

2B 法則失效條件

只有一個：確認突破前高（或前低）。

如果價格突破前高後沒有迅速回轉，而是繼續往上走，2B 就失效了，不要硬做。

支撐壓力互換角色

這是 2B 法則的延伸應用：

原則：當支撐被跌破後，該支撐變成壓力；當壓力被突破後，該壓力變成支撐。

2B 與支撐壓力互換的關係：
1. 前高是壓力區 → 價格突破前高（壓力突破）→ 壓力應轉為支撐
2. 但 2B 發生：價格迅速跌回前高之下 → 壓力沒有成功轉為支撐 → 代表突破無效
3. 此時前高依然是壓力，甚至壓力更強（因為套牢了突破追價的人）

實戰意義：

  • 2B 失敗 = 支撐壓力互換失敗 = 原壓力更強
  • 2B 成功 = 確認反轉 = 前高前低角色即將互換

上落盤整市的 2B 應用

Victor Sperandeo 特別指出：盤整市中 2B 法則可以連續出現、連續獲利。

邏輯：

  • 盤整區有明確頂部和底部
  • 價格測試頂部 → 假突破 → 2B 做空
  • 價格測試底部 → 假突破 → 2B 做多
  • 只要盤整繼續，2B 訊號會反覆出現

注意：盤整一旦突破（真突破），2B 策略要停止，改用突破追價策略。

心得

2B 法則看似簡單（突破又回來就做），但魔鬼在細節裡。最關鍵的就是「假突破」的認定——幅度要小、時間要短，這兩個條件缺一不可。很多人看到突破後回跌就興沖沖進場，結果只是正常回檔，被真突破打臉。

順勢 2B 和逆勢 2B 的區分也很重要。上升趨勢中看到壓力區假突破就想做空，這是逆勢操作，大趨勢還在上升啊！逆勢 2B 只能當警示，不能當進場訊號。這跟之前學的「順勢交易」原則一致。

停損設計是 2B 最漂亮的地方——停損就設在假突破的極端點，距離近、風險小，如果真抓到反轉，風報比極佳。這就是「小虧大賺」的具體實現。

最後，2B 法則跟支撐壓力互換的關係很值得深究。假突破本質上就是「支撐壓力互換失敗」，這個角度讓我對價格行為的理解又深了一層。
