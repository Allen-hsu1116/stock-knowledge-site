一、集保庫存週變化的核心邏輯

籌碼流動方向決定股價方向

是什麼：集保庫存（股權分散表）是臺灣集中保管結算所每週公布的資料，顯示每檔股票的持股分布。核心原則只有一條：籌碼從弱者流到強者手上 → 偏多；從強者流到弱者手上 → 偏空。

怎麼看：
1. 定義大戶：百張以上持股者（建議用百張而非千張，因為千張級距過粗，高價股如大立光一張 200 萬，很少有人能持千張）
2. 定義散戶：十張以下持股者
3. 每週六看資料：集保所通常週六中午公布，最遲週一早上九點前
4. 觀察方向：大戶增＋散戶減 → 利多；大戶減＋散戶增 → 利空

實例：

  • 中鋼（2002）9/28 集保資料：千張大戶持股 58.31%，零股持有者僅 0.33%
  • 中砂（1560）連兩週大戶持股增加＋散戶持股減少，為全市場籌碼偏多最顯著的股票
  • 光耀（3666）用千張級距看大戶在衰退，但切到百張級距後反而看出大戶仍在增加 → 千張級距誤判

適用時機：

  • ✅ 中小型股（市值 > 10 億、年均量 > 1000 萬，避開殭屍股）
  • ❌ 高價股用千張級距會失真
  • ❌ 大型權值股如台積電，法人持股 6-7 成，大戶定義失靈

二、大戶錯殺股——反轉點捕捉策略

大戶錯殺股篩選法

是什麼：當大戶持股已連續下降到新低，但本週突然反轉增加，同時散戶持股減少，代表大戶「認錯回頭買」，這種股票很可能處於底部反轉點。

怎麼做：
1. 篩選條件：

  • 上週大戶持股創新低（近 N 週最低）

2. 排序方式：照區間跌幅排序（預設 20 日），跌越多排越前面
3. 進場時機：確認連續兩週大戶增＋散戶減再進場，避免一週假訊號
4. 搭配基本面：大戶錯殺股只是籌碼面訊號，要再搭配營收、獲利等基本面確認

實例：

  • 富邦媒（8454）：大戶持股節節敗退、股價跌跌不休，但某週突然大戶增＋散戶減，隨後股價反轉向上
  • 中砂（1560）：連兩週大戶持股增加比例＋散戶持股減少比例，為全市場數據差額最大的籌碼偏多股

適用時機：

  • ✅ 跌深反彈行情，找底部反轉點
  • ✅ 搭配技術面支撐位置進場
  • ❌ 不適合追高，因為大戶持續買通常股價也在高檔

三、籌碼凌亂股——放空訊號辨識

籌碼凌亂股篩選法

是什麼：大戶持續減持、散戶持續增加，且散戶持股創新高，代表籌碼從強者流向弱者，股價易跌難漲，是放空或迴避的訊號。

怎麼做：
1. 篩選條件：

  • 本週大戶減少

2. 排序方式：照區間漲幅排序，漲越多但籌碼越亂，越值得放空
3. 進場方式：股價異常上揚但籌碼面不佳 → 等待股價轉弱時進場做空
4. 停損設定：前高上方 3-5% 設停損

實例：

  • 科嶠（4542）：股價暴衝，但大戶減、散戶增且散戶持股創新高 → 籌碼面不佳，股價異常上揚，後續容易回跌

適用時機：

  • ✅ 股價大漲但籌碼面惡化的個股
  • ✅ 搭配融資使用率偏高的個股，放空勝率更高
  • ❌ 不適合在法人持續買超的大型股使用

四、內部大戶理論——簡哲隆實戰心法

內部大戶定義與計算

是什麼：真正的贏家 8 成是公司的大股東或高層（非法人大戶）。「內部大戶持股變化」= 集保股權分散表大戶增減 − 三大法人同時期的持股增減。

怎麼做：
1. 大戶定義依股價調整：

  • 股價 > 50 元：持股超過 400 張就是大戶

2. 關注訊號：

  • 內部大戶單週爆買占股本 2% 以上 → 特別關注

3. 進場邏輯：

  • 內部大戶不會作價，股價不會立即反映 → 耐心等待

4. 排除大型權值股：台積電、聯發科等法人持股比率 6-7 成，不適用

實例：

  • 簡哲隆靠內部大戶理論搭配基本面，年賺 30%
  • 某檔股票內部大戶續買且營收攀高 → 進場後不到 1 個月賺 14%
  • 基本面派投資人搭上內部大戶選股後，整體報酬成長快 10 倍

適用時機：

  • ✅ 中小型股（內部大戶佔比才有意義）
  • ✅ 搭配基本面利多一起判斷
  • ❌ 大型權值股法人持股比重太高
  • ❌ 不要只看一週，要持續追蹤累積買超占股本比例

五、實戰操作流程整合

每週集保庫存檢查四步驟

怎麼做：
1. 週六看資料：集保所公保所公佈後，篩選大戶增＋散戶減的股票
2. 雙重確認：至少連兩週同方向才算有效訊號
3. 分類篩選：

  • 大戶錯殺股（底部反轉）→ 做多

4. 搭配驗證：技術面支撐壓力 + 基本面營收獲利 + 法人買賣超

重要注意事項：

  • 千張級距過粗，百張級距才能看出有意義的變化
  • 999 張的人被千張級距剔除，但實質上只差 1 張，百張級距就不會漏掉
  • 大戶連續買超但股價沒動是正常的（內部大戶不會作價），要耐心等
  • 單純看「大戶增就買」勝率 > 50% 但容易被燙到，必須搭配其他面向

心得

集保庫存這東西，我之前學過「集保庫存定義與用途」跟「股權分散表判讀方法」跟「大戶持股比例籌碼判斷」，但今天這幾篇才是真正把「怎麼用」講清楚的。

幾個重點心得：

1. 百張 vs 千張的選擇差很多。千張級距一刀切太粗，高價股根本沒幾個人能持千張，光耀那個案例就是經典——千張看在減，百張看在增，方向完全相反。以後看集保庫存，一律用百張級距。

2. 大戶錯殺股是個很漂亮的反轉訊號。不是找「大戶持續買」（那通常股價已在高檔），而是找「大戶之前一直賣到新低，突然回頭買」。這個轉折點才是真的有價值的進場時機。

3. 內部大戶理論是個殺招。「集保大戶增減 − 三大法人增減 = 內部大戶增減」，這個計算方式把法人干擾剔除，能看到真正公司派在幹嘛。大股東身體比嘴巴誠實——他們敢砸幾千萬進去，一定有我們不知道的好事。

4. 但最關鍵的是：籌碼面不能單獨用。不管是大戶錯殺股還是籌碼凌亂股，單用勝率剛過五成，被燙一次就痛得要死。「基本面為盾、籌碼面為矛」這句話記住了——籌碼面幫你找到目標，基本面幫你確認能不能進場。
