安全邊際是什麼？

定義：安全邊際（Margin of Safety）是股票內在價值與股票市場價格的差距。當差距越大，越能降低投資虧損的風險。

生活比喻：上班時間9點，通勤需要30分鐘，但你會提前40-60分鐘出門，多出來的10-30分鐘就是「安全邊際」——用來應對無法預見的風險（塞車、道路施工、公車誤點等）。

巴菲特名言：
> 「安全邊際就是計算一個東西值多少錢，然後以較低的價格把它買下來。」

為什麼安全邊際重要？

問題核心：你評估的內在價值可能會錯！

如果估錯了，看似便宜的價格可能反而貴。安全邊際的作用就是：即使估錯，你仍然有餘地。

安全邊際怎麼計算？

公式：

  • 安全邊際 = 股票內在價值 - 股票市場價格
  • 買進價格 = 內在價值 × 80%（保留 20% 安全邊際）

四種情況：
1. 價值 >> 價格：安全邊際很大
2. 價值 > 價格：安全邊際較小
3. 價值 = 價格：安全邊際不存在
4. 價值 < 價格：沒有安全邊際，不應買進

安全邊際要抓多少？

保守型30-50%願意少買，但求安全
一般型20-30%巴菲特建議至少 20%
�極型10-20%接受較高風險

重要提醒：安全邊際並非越大越好。邊際太大 → 錯過很多機會 → 總報酬可能更低。

什麼時候會有安全邊際？

股價變便宜的兩種原因：

1. 群眾恐慌（好機會）：例如 2008 年金融海嘯，好公司價格腰斬。這時要有勇氣和眾人不同。

2. 公司出事（要判斷）：意外、負面消息、財報不如預期。必須確認：影響是「永久性」還是「暫時性」。

實際操作步驟

1. 評估股票內在價值（參考資產、利潤、股息、公司發展前景）
2. 計算安全買進價格（安全買進價格 = 內在價值 × 80%）
3. 耐心等待股價跌到安全價格區間

心得

安全邊際的概念很直覺：就像買東西打折時才出手，但你要先知道那個東西「真正值多少」。

幾個重要點：
1. 估價能力是基本功：如果連內在價值都估不準，安全邊際就沒意義
2. 耐心比技術更重要：知道要買多少是一回事，能夠等到那個價格出現又是另一回事
3. 安全邊際大不一定是好事：要反問「為什麼別人看不出來？」
4. 能力圈原則：要評估自己看得懂的東西，這樣估價才有意義
