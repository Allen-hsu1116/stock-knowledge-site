當沖三種操作模式

Tick流（極短線）

是什麼：賺取股價跳動檔位的價差，幾秒到幾分鐘內完成交易。

怎麼做：
1. 觀察五檔和即時成交明細
2. 看到價格跳動方向確認後進場
3. 跳1-2檔就獲利了結
4. 沒有連續性走勢立刻停損

實例：

  • 情境：台積電股價在600元，五檔顯示買盤力道強
  • 操作：600元進場，看到600.5、601連續成交，601元出場
  • 獲利：0.5檔 = 0.5元/股 = 500元/張（扣手續費約賺300-400元）

適用時機：盤中波動大、流動性好的股票；不適合開盤前30分鐘（波動太大不好抓）

一波流（突破交易）

是什麼：抓大單進場時的突破瞬間，跟著主力一起衝。

怎麼做：
1. 監控成交明細，看到大單（數百張以上）連續敲進
2. 確認五檔買盤也同步增加
3. 果斷追進（不要猶豫，猶豫就輸了）
4. 關鍵：沒有連續性走勢必須果斷停損

實例：

  • 情境：某股票在100元附近盤整，突然看到連續5筆以上100張大單敲進
  • 操作：100.5元追進，看到101、101.5持續成交，抱到102元出場
  • 如果進場後價格停滯、沒有繼續衝 → 立刻停損出場

適用時機：有主力作手操作的股票；不適合量太小的股票（容易被假突破）

趨勢流（日內波段）

是什麼：抓當日一段時間的趨勢，可能持倉幾十分鐘到整天。

怎麼做：
1. 開盤觀察今日多空方向
2. 等待回調不破關鍵價位時進場
3. 設好停損點（通常2-3%）
4. 抱住直到趨勢反轉或收盤前出場

實例：

  • 情境：台股指數開高走穩，電子股表現強勢
  • 操作：選擇強勢股如台積電，回調到平盤附近進場
  • 停損：跌破開盤價1%停損
  • 停利：抱到收盤或漲幅達3%

適用時機：趨勢明確的日子；不適合盤整盤（容易被左右打臉）

當沖停損三種設定方式

方式1：絕對金額停損

怎麼做：設定每天最大可接受虧損金額，達到就停手。

實例：

  • 每日設定最大虧損5,000元
  • 第一筆虧3,000元 → 還能繼續
  • 第二筆再虧2,000元 → 累計5,000元，今日收手

優點：簡單明確，容易執行
缺點：沒考慮股價波動特性

方式2：百分比停損

怎麼做：設定股價下跌一定百分比就停損。

實例：

  • 進場價100元，設定3%停損
  • 跌到97元 → 觸發停損，出場

建議比例：

  • 極短線：1-2%
  • 日內波段：2-3%
  • 不建議超過5%（回本難度大增）

方式3：觸價單停損

怎麼做：預先設定觸價設定觸價單，價格一到自動停損。

實例：

  • 進場價100元
  • 同時設定97元觸價賣出單
  • 股價跌到97元 → 系統自動賣出

優點：不用盯盤、避免人性猶豫
缺點：可能被插針後彈回（停損在最低點）

當沖資金配置策略

順勢單（追突破）

資金分配比例：3:2:1

  • 第一筆：30%資金試單
  • 確認方向對：加碼20%
  • 漲勢確立：最後加碼10%

實例：預計投入60萬

  • 第一筆：18萬（試水溫）
  • 確認突破：加12萬
  • 漲勢確立：加6萬

逆勢單（抓轉折）

資金分配比例：1:2:3

  • 第一筆：少量試單（10%）
  • 確認轉折：加碼20%
  • 趨勢確立：大舉加碼30%

為什麼反過來：逆勢風險較大，先用小單試，確認判斷正確再加碼。

當沖交易紀律

五大紀律

1. 大單短進短出，小單贏到最後 - 不要因為賭錯方向死扛
2. 投降輸一半 - 虧損時一定要減碼
3. 不要有執念 - 籌碼判讀比感覺重要
4. 強勢股多單抱緊，空單有賺就跑
5. 弱勢股多單有賺就跑，空單抱緊

最佳交易時段

黃金時段：09:00 - 10:30

  • 波動大、機會多
  • 最適合兼職者

避開時段：

  • 09:00-09:30（開盤30分鐘）：波動劇烈，最高最低點常出現在這
  • 13:00-13:30（收盤30分鐘）：容易反向、無法沖銷
  • 13:25-13:30：搓尾盤機制，風險高

手機當沖技巧

如何接近電腦下單速度

1. 網路訊號要穩定（4G/5G正常）
2. 熟悉下單介面（不要下單時還要找按鈕）
3. 預先設定好快速下單功能

手機當沖核心能力

  • 五檔觀察：看買賣價的單量變化
  • 成交明細：看大單進出方向
  • 果斷執行：看到訊號立刻動作

關鍵：手機當沖要做到「看成交明細同時出手」，才能打在下一檔價位。

當沖心態建立

五大基本心態

1. 多空雙向都要會 - 不只做多，也要會放空
2. 一定要設停損點 - 開盤前就設好
3. 不過度交易 - 等待最有利機會
4. 資金充足防違約 - 不要在漲跌停附近交易
5. 賺超過5%要收手 - 持盈保泰

心得：

今天這三篇文章補足了之前學習的細節。之前學過「當沖三大交易時段」和「停損停利比例設定」，但今天學到了更具體的操作模式和心態建立。

幾個關鍵收穫：

1. 一波流的停損心法很重要：進場後「沒有連續性走勢就果斷停損」— 這要練到變反射動作。很多人輸在「凹單」，想說再等一下看看，結果小賠變大賠。

2. 資金配置有學問：順勢單和逆勢單的資金配置完全相反。順勢單要「金字塔加碼」，逆勢單要「倒金字塔加碼」。這邏輯很合理：順勢單方向對了才加碼，逆勢営要確認轉折才敢加碼。

3. 手機當沖不是弱勢：只要練好五檔和成交明細的觀察力，手機也能做得跟電腦一樣快。關鍵是「下單不要猶豫」，看到訊號就要出手。

4. 回本難度的表格很震撼：虧50%要漲100%才能回本。這告訴我們：停損永遠比幻想「會漲回來」重要。

5. 「賺超過5%要收手」這個紀律很實際：很多人口袋不夠深卻想天天大賺，結果一波反向就賠光。持盈保泰才是長久之道。

  • 主力洗盤、甩轎手法

洗盤 vs 變盤：怎麼分辨？

洗盤甩轎是什麼

主力為了降低持倉成本，讓低價買進、意志不堅定的散戶拋出股票，同時讓散戶持股的平均價位升高，從中獲取暴利。

洗盤 vs 變盤的關鍵差異

特徵洗盤變盤（出貨）
目的清洗浮動籌碼，嚇跑散戶清倉出貨
手法快速、連續性下跌、跌破支撐溫和下跌，「溫水煮青蛙」
K線特徵漲停打開拉長下影線再收漲停緩慢下跌，不知不覺深套
成交量先大後小（洗完縮量）高檔放量後量縮

三種主力洗盤手法

1. 高檔放量滯漲

是什麼：行情拉升到一定高度後，主力慢慢出貨，但散戶不耐久盤會賣出換股，行情維持橫向盤整。

怎麼判斷：

  • 股價在高檔盤整，成交量放大但不漲
  • 主力高檔換手、低檔接回增加籌碼
  • 盤整結束後主力會再拉一波

注意：如果波段漲幅已大，放量滯漲可能是主力真的在出貨，要看波段漲幅和成交量變化判斷。

實例：

  • 情境：某股票從 100 元漲到 150 元，開始在高檔盤整，成交量放大
  • 判斷：如果之前沒出現天量大陽線，且主力籌碼沒大改變，可能是洗盤
  • 操作：觀察盤整結束後的方向再決定

2. 高檔雙陰線洗盤

是什麼：股價一波上漲後，主力在前期頭部附近反向賣出兩天，製造洗盤陷阱。

K線特徵：

  • 前面沒有天量大陽線或吸引跟風走勢
  • 突然高位收陰
  • 第二天低開低走，收出近似光腳陰線

更狠的版本：

  • 起漲前先出現帶量長黑棒，嚇出猶豫散戶
  • 隨後大陽線連續拉抬，甚至連續漲停

實例：

  • 情境：股價從 80 元漲到 120 元，突然出現兩根陰線跌到 110 元
  • 判斷：看均線是否仍多頭排列、技術指標是否仍多方
  • 操作：如果 5/12/21 日均線多頭排列未變，可視為洗盤

3. 邊拉邊出洗盤兼出貨

是什麼：主力不是一次拉到出貨價位，而是在接近出貨價位時減緩上升速率，邊拉升邊出貨。

盤面特徵：

  • 股價每次創新高後就回調
  • 回調結束後又創新高
  • 散戶以為在洗盤，主力卻在不知不覺中出貨完畢

判斷訊號：

  • 高檔放量：主力在出貨
  • 出貨完畢後：高檔量縮（明顯結束訊號）

實例：

  • 情境：股價從 100 元漲到 150，再漲到 155、158、160，每次創高都回調
  • 判斷：看成交量是否高檔放量後量縮
  • 操作：高檔量縮時要警覺，可能主力已出貨完畢

投資者因應之道（四個判斷要點）

要點 1：看均線支撐

判斷準則：

  • 行情剛起漲：回檔不跌破 12 日均線 = 洗盤
  • 漲幅超過 20%：回檔不跌破 21 日均線 = 洗盤
  • 5/12/21/58 日均線多頭排列未變 = 多頭行情未變

實例：

  • 股價從 50 元漲到 65 元（漲 30%），回檔到 60 元守穩 21 日線
  • 判斷：均線多頭排列，可視為洗盤
  • 操作：持股續抱或拉回加碼

要點 2：看成交量變化

洗盤的成交量特徵：

  • 洗盤剛開始：成交量放大（膽小、性急的人出逃）
  • 洗盤過程：成交量由大到小縮量
  • 洗盤結束：量縮到極致

K線特徵：

  • 常出現長上影線
  • 拉回幅度有限

要點 3：看技術指標（KD + MACD）

買進訊號：

  • 日線 KD 交叉由下往上穿越 40
  • MACD 柱狀由黑翻紅
  • 兩者同時具備，對應 K 線可買進

賣出訊號：

  • 日線 KD 交叉由上往下跌破 70
  • MACD 柱狀由紅翻黑
  • 兩者同時具備，對應 K 線可賣出

實例：

  • 情境：某股票 KD 在 30 位置交叉向上，MACD 柱狀翻紅
  • 判斷：指標由多轉空確認，可視為洗盤結束
  • 操作：買進時機

要點 4：看籌碼變化

判斷準則：

  • 千張持股比例變化
  • 外資持股變化
  • 高檔放量或高檔創新高沒量 = 警訊

操作心法：

  • 漲停打開時：不急於出場
  • 跌停打開往上拉時：不盲目接盤
  • 細心觀察成交量變化

心得

主力洗盤是散戶最常被洗出場的關鍵時刻。文章說得好：「行情剛啟動不久，手中股票剛出現小賺，只要行情波動一下立刻想獲利出場」——這就是散戶的致命傷。

幾個重點：
1. 均線是關鍵防線：12 日線、21 日線是洗盤的判斷基準，跌破這些防線才要警覺
2. 成交量是說真話的：洗盤過程量會縮，出貨過程量會放大，這個特徵很難造假
3. 技術指標雙確認：KD 穿越位置（40/70）加上 MACD 柱狀翻色，比單一指標可靠
4. 心態最重要：了解持有股票的內在價值，才能撐過主力的嚇唬手法

這篇跟之前學過的「真假突破判斷」可以互補——洗盤往往發生在突破後的整理區間，兩者搭配著看更能判斷主力的真實意圖。

  • 一文認識順勢交易、逆勢交易，你屬於哪一派？
  • 想嘗試逆勢交易卻無從下手？逆勢交易前這 3 個特點你得先了解

核心知識

一、順勢交易與逆勢交易定義

順勢交易（右側交易）：

  • 「弱者恆弱、強者恆強」— 買的高，賣的更高
  • 確認行情階段性高點（或低點）已出現，在高點（或低點）的「右側」進場
  • 等到底部確立、趨勢確認才進場

逆勢交易（左側交易）：

  • 「物極必反、均值回歸」— 買低、賣高
  • 在行情潛在高點（或低點）的「左側」進場
  • 假設行情乖離到某種程度時會反轉

簡單理解：在「Ｖ」型走勢中，左側買入是逆勢，右側買入是順勢。

二、順勢交易實戰要點

是什麼：追隨已確認的趨勢，突破後才追漲，高點後才賣出。

怎麼做：
1. 確認趨勢：觀察均線斜率向上、價格突破關鍵位置
2. 進場時機：等待突破或回確認支撐後進場
3. 加碼策略：趨勢正確時逐步加倉，但前期低成本部位要掩護後面加倉
4. 出場時機：確認高點出現後才賣出

實例：

  • 情境：S&P500 2020年3月23日反轉
  • 順勢操作：等3/23後確認底部、T+n日沒再出現低點才進場
  • 風險：可能一次噴出錯過最佳進場點

優點：

  • 不會錯過大趨勢，單筆獲利極高（一次獲利可抵5-20次虧損）
  • 不會頻繁交易
  • 策略不易被市場淘汰

缺點：

  • 勝率低，洗刷次數多
  • 需忍受長期大幅回落
  • 若交易週期短，滑價影響大
  • 長則數年無大趨勢，需多市場交易

三、逆勢交易實戰要點

是什麼：假設均值回歸，判斷行情乖離到某種程度時做反向交易。

怎麼做：
1. 判斷乖離：使用技術指標（如布林通道、乖離率）判斷超買/超賣
2. 分批進場：越跌越買，分批建立部位（絕對不能一次All in）
3. 設停損：預設可接受的虧損範圍
4. 等待回歸：行情如預期反轉時獲利

實例：

  • 情境：股市從高點大跌20%，判斷超跌
  • 逆勢操作：分批買入權值股或ETF（0050、QQQ）
  • 加碼策略：每跌5%加碼一次，控制總部位
  • 風險：可能「越抄越心痛」，如2020年原油崩跌

優點：

  • 進在有利位置，勝率較高
  • 不易有滑價
  • 獲利頻率高，績效相對穩定
  • 可以建立比較好的成本

缺點：

  • 交易週期短、獲利幅度小，手續費成本高
  • 連續大虧損不易恢復
  • 平均單筆獲利低，靠交易次數撐大獲利
  • 需要一直盯盤，資金管理能力要求高

四、如何選擇適合自己的方式

順勢適合：

  • 一般投資者
  • 無法頻繁盯盤的人
  • 想要抓住大趨勢的人

逆勢適合：

  • 機構法人
  • 有時間盯盤、能做預判的人
  • 資金控管能力強的人

五檔資訊基礎概念

什麼是五檔？

定義：離目前成交價最接近的5個價位，分別有多少買家和賣家

組成：

  • 委買價（內盤）：買方掛單價格，從高到低排列
  • 委賣價（外盤）：賣方掛單價格，從低到高排列
  • 掛單張數：每個價位上的等待成交數量

tick 概念：

  • 股價跳動單位，例如 100-500 元股價的 tick 為 0.1 元
  • 110 元往上漲一檔是 110.1，往下跌一檔是 109.9
  • 對當沖來說，1 tick 的價差影響巨大

內盤與外盤

內盤（委買價成交）

是什麼：成交在「委買價」，計入內盤

代表意義：

  • 賣方較為積極，願意降價求售
  • 賣方看跌，急著出脫股票
  • 內盤成交量大 → 股價下跌機率高

外盤（委賣價成交）

是什麼：成交在「委賣價」，計入外盤

代表意義：

  • 買方較為積極，願意追價買進
  • 買方看多，無論如何都要買到
  • 外盤成交量大 → 股價上漲機率高

內外盤比

計算方式：

  • 內盤比 = 內盤成交量 ÷ 總成交量
  • 外盤比 = 外盤成交量 ÷ 總成交量

判讀：

  • 內盤比 > 外盤比：賣壓沉重，偏空
  • 外盤比 > 內盤比：買盤積極，偏多

注意：內外盤比合計不一定剛好 100%，因為有極少數成交價不落入委買/委賣價

2. 找出大量賣單聚集的價位 → 可能是阻力位

一、台股交易成本結構

是什麼：每次買賣股票都要付出的「隱形成本」，很多人忽略這筆費用對報酬率的影響。

成本組成：

項目買進賣出
手續費0.1425%0.1425%
交易稅00.3%
當沖稅-0.15%

怎麼做：
1. 一般交易總成本 = 0.1425% + 0.1425% + 0.3% = 0.585%
2. 當沖交易總成本 = 0.1425% + 0.1425% + 0.15% = 0.435%
3. 快速估算：交易金額 × 0.585（或 0.435）= 總成本

實例：

  • 情境：買進 1 張 100 元股票後賣出
  • 計算：100,000 × 0.585% = 585 元成本
  • 結論：股價必須上漲超過 0.6 元（100.6 元以上）才算獲利

二、當沖 Tick 打平成本表

是什麼：不同價區的股票，因為 Tick 跳動單位不同，當沖回本的難度也不同。

Tick 是什麼：股價的最小跳動單位，不同價區有不同規定：

股價範圍Tick 單位
10 元以下0.01 元
10~50 元0.05 元
50~100 元0.1 元
100~500 元0.5 元
500~1000 元1 元
1000 元以上5 元

怎麼做：
1. 選擇 Tick 較大的價區（如 100~150 元），1 Tick = 0.5 元
2. 若手續費 6 折，成本約 480 元，跳 1 Tick (0.5 元) 賺 500 元就獲利
3. 避免低價股當沖：10 元以下股票 3 Tick 才回本

實例：

  • 情境：台積電 120 元，手續費 6 折
  • 成本：120,000 × 0.435% × 60% = 313 元（約）
  • 跳動：1 Tick = 0.5 元 = 500 元
  • 結論：只要跳 1 Tick (120 → 120.5)，就足以打平成本

最佳當沖價區（手續費 6 折）：

  • 100~150 元：1 Tick 就回本
  • 50~100 元：2 Tick 就回本
  • 避開：低於 30 元的股票（需要 3~5 Tick）

月成交量定位可談折扣
500 萬以下小散戶5 折就不錯

是什麼：頻繁交易會被成本吃掉報酬率。

實例：

  • 情境：年報酬率 6%，每月買賣 1 次

  • 每次交易成本 0.5%，一年 12 次 = 6% 成本
  • 結論：6% 報酬 - 6% 成本 = 0% 實際報酬

怎麼做：
1. 長期投資：買賣次數少，成本低
2. 短線交易：必須有「高勝率+高賺賠比」才值得
3. 當沖：交易稅減半（0.15%），是高頻交易的唯一可行方式
