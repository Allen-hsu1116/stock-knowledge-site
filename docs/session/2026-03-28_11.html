KD指標基本概念

是什麼：KD指標（隨機震盪指標）由喬治·萊恩於1950年代創造，用來判斷股價在特定時間內的相對高低位置，找轉折點。

公式原理：

  • RSV（未成熟隨機值）= (今日收盤價 - 近N天最低價) ÷ (近N天最高價 - 近N天最低價) × 100
  • K值（快線）= 昨天K值 × 2/3 + 今天RSV × 1/3
  • D值（慢線）= 昨天D值 × 2/3 + 今天K值 × 1/3

怎麼做：
1. 先計算RSV，知道今天收盤價相對過去N天高低的位置
2. RSV做加權移動平均得到K值（快速反應）
3. K值再做加權移動平均得到D值（慢速平滑）

數值範圍：0-100，50是中線，>80是超買區，<20是超賣區

是什麼：KD參數決定敏感度，參數愈小愈敏感但假訊號多，參數愈大愈穩定但反應慢。

怎麼做（依交易週期）：

交易週期建議參數優勢注意事項

中長期投資(21,3)或(28,3)過濾短期雜訊反應遲緩，需耐心等待

  • 長期投資大型股：用(14,3)，過濾雜訊

超買超賣區判斷

怎麼做：
1. 觀察KD值是否進入20以下或80以上
2. 確認K線與D線的交叉方向
3. 配合成交量確認（量價背離要小心）

適用時機：

實例：

  • 某股票KD在15，開始向上穿越D線→考慮進場
  • KD在85，開始向下穿越D線→考慮出場

黃金交叉與死亡交叉

是什麼：K線（快線）與D線（慢線）的交叉訊號。

怎麼做：

黃金交叉（買進訊號）：
1. K線由下向上穿越D線
2. 確認出現在超賣區（KD<20附近）更強

4. 進場後設停損在近期低點下方

死亡交叉（賣出訊號）：
1. K線由上向下穿越D線
2. 確認出現在超買區（KD>80附近）更強
3. 配合量縮確認
4. 出場後觀察是否繼續下跌

  • KD從18往上穿過D線，K值到25，出現黃金交叉→買進訊號
  • KD從82往下穿過D線，K值到75，出現死亡交叉→賣出訊號

KD鈍化判斷

是什麼：KD值長期停留在高檔或低檔，失去參考價值。

高檔鈍化：

  • 判斷：連續3根以上K線都在80以上
  • 意義：股價可能很強，不要急於賣出
  • 拉回不破前波低點，價格有機會創新高

低檔鈍化：

  • 判斷：連續3根以上K線都在20以下
  • 意義：股價可能很弱，不要急於買進
  • 反彈沒過前波高點，價格可能繼續創新低

怎麼做：
1. 高檔鈍化時等拉回到支撐再考慮買進
2. 低檔鈍化時等突破壓力再考慮進場
3. 配合成交量判斷（量縮鈍化要小心）

實例：

  • KD連續5天都在85以上→高檔鈍化，不要急著賣
  • KD連續4天都在15以下→低檔鈍化，不要急著買

KD背離判斷

是什麼：價格與KD指標走勢不一致，是反轉前兆。

高檔背離（負背離）：

  • 現象：價格創新高，KD卻沒創新高
  • 意義：上漲動能減弱，可能反轉下跌
  • 操作：賣出訊號

低檔背離（正背離）：

  • 現象：價格創新低，KD卻沒創新低
  • 意義：下跌動能減弱，可能反轉上漲
  • 操作：買進訊號

怎麼做：
1. 先確認價格是否創新高/新低
2. 檢查KD是否同步創高/低
3. 不同步就是背離
4. 配合其他指標確認（RSI、MACD）

實例：

  • 股價從100漲到110創高，KD從90降到85沒創高→高檔背離，賣出
  • 股價從80跌到70創低，KD從15升到20沒創低→低檔背離，買進

KD搭配其他指標

搭配RSI：

  • 用途：強化超買超賣訊號
  • 方法：KD黃金交叉 + RSI從低檔回升→更可靠的買進訊號
  • 實例：KD低於20且交叉向上，RSI也從30回升→雙重確認

搭配移動平均線：

  • 用途：確認趨勢方向
  • 方法：KD黃金交叉出現在股價站穩均線之上更準
  • 實例：KD金叉+股價在20日均線之上→順勢操作

搭配MACD：

  • 用途：濾除雜訊，確認中長期動能
  • 方法：KD買進訊號 + MACD柱狀體翻紅→趨勢轉強
  • 實例：KD金叉+MACD DIF向上→提高勝率

KD指標優缺點

優點：

  • 操作簡單，新手友善
  • 反應靈敏，擅長震盪盤
  • 可與多種指標搭配

缺點：

  • 趨勢行情容易鈍化
  • 過於靈敏，假訊號多
  • 參數設定影響大

適用時機：

  • 震盪整理盤最有效
  • 配合其他指標使用
  • 不適合單獨使用

心得

KD指標是我一直聽過但沒深入學過的經典技術指標，這次終於補上這塊。最關鍵的學習點是「參數調整」：

1. 不要用預設參數套用所有情況：短線用(5,3)、波段用(9,3)、長線用(14,3)，這個觀念讓我理解為什麼有時候KD怎麼看都對不上實際走勢。

2. 鈍化不是壞事：高檔鈍化反而代表強勢，不要急著賣；低檔鈍化代表弱勢，不要急著買。這跟直覺相反，但市場就是這樣，強者恆強。

3. 背離是重要訊號：價格創高但KD沒創高，這種「動能減弱」的訊號比單純看交叉更有價值。

4. 不要單獨使用：KD是落後指標，只能參考過去，要配合趨勢類指標（均線、MACD）使用。

這個指標特別適合震盪盤，遇到趨勢行情就要小心鈍化。下次看盤時會特別注意KD是否進入鈍化區，以及是否有背離現象。
