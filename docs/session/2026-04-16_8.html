一、移動停損（Trailing Stop）實戰策略

1. 移動停損定義與核心機制

是什麼：移動停損是一種隨價格有利方向自動調整停損點位的風控策略，停損點只往有利方向移動、不往不利方向退讓，觸發時鎖定已獲利潤。

怎麼做：
1. 設定追蹤距離：用百分比或固定點數設定停損點與當前價的距離（例如5%或10元）
2. 價格上漲時：停損點自動上移，維持與最高價的固定距離
3. 價格下跌時：停損點不動，維持在已上移的位置
4. 觸發出場：當價格回跌到停損點即自動賣出，鎖住利潤

實例：

  • 情境：台積電(2330)在600元買進，設定5%移動停損
  • 初始停損：570元（600×0.95）
  • 股價漲至660元新高 → 停損自動上移至627元（660×0.95）
  • 股價回跌至627元 → 觸發賣出，鎖定每股27元利潤（4.5%）
  • 若用固定停損570元 → 要多虧57元差額才能出場

適用時機：趨勢明確的波段行情；不適合盤整震盪市（會被頻繁洗出）

2. 移動停損 vs 固定停損比較

是什麼：固定停損設一個死價格，移動停損設一個會動的保護線。

怎麼做：

  • 固定停損：進場時設好停損價，不論股價漲多少都不動
  • 移動停移動停損：獲利後停損點跟著往上跑，自動保護浮動利潤
  • 混合策略：進場先用固定停損控管初始風險，獲利超過一定幅度後切換成移動停損

實例：

  • 情境：某股100元買進
  • 固定停損：設92元，股價漲到120再跌回92才出場，中間28元利潤全吐
  • 移動停損5元距離：股價漲到120時停損已到115元，跌到115就出場，至少賺15元
  • 混合法：先設固定停損92元，漲到108元以上後切換移動停損3%

適用時機：固定停損適合盤整短線；移動停損適合趨勢波段；混合法最靈活

3. 移動停損幅度設定三大考量

是什麼：追蹤距離設太大太小都不行，要根據三個因素來拿捏。

怎麼做：
1. 標的波動性：波動大的標的（小型股、加密貨幣）設寬一點（10-20%），穩定大型股設窄一點（3-8%）
2. 操作時間週期：

  • 當沖：不太適合移動停損，微波動就會洗掉

3. 個人風險偏好：穩健型設窄、追求大波段型設寬，但不能脫離標的實際波動特性

實例：

  • 情境：台積電日波動約±2%，設5%移動停損可以過濾正常震盪
  • 情境：某小型生技股日波動±5%，設5%太緊會被頻繁洗出，至少要8-10%
  • 錯誤示範：比特幣日波動±10%，卻設2%移動停損 → 必定反覆止損

適用時機：每次設定移動停損前都要做這三項評估，不能一個參數走天下

4. 移動停損搭配技術分析三種方法

是什麼：不用死板的百分比，改用技術指標動態決定停損位置。

怎麼做：
1. 均線移動停損法：以10日或20日均線作為移動停損線，跌破均線才出場
2. 布林通道停損法：多頭時以布林下軌作為停損參考，跌破下軌出場
3. ATR倍數停損法：以ATR（平均真實波幅）的2-3倍作為追蹤距離，波動大時自動放寬

實例：

  • 情境：台積電多頭走勢，20日均線在615元
  • 均線法：停損設在615元下方，跌破即出場，均線每天上移停損也跟著上移
  • 布林法：下軌在605元，跌破605出場
  • ATR法：ATR=15元，設2倍ATR=30元追蹤距離，最高價660元時停損在630元

適用時機：趨勢明確的波段操作，比固定百分比更貼合標的實際波動

二、左側交易 vs 右側交易

1. 左側交易：逆勢抄底的藝術

是什麼：在股價還在下跌時買進、還在上漲時賣出，即「在轉折點的左邊」操作，又稱逆勢交易。

怎麼做：
1. 基本面判斷極端區域：用P/E、P/B等估值指標判斷是否被嚴重低估
2. 技術面找超跌訊號：乖離率異常（如-10%以下）、RSI極度超賣（<20）
3. 分批進場降低風險：不預測最低點，分3-5批越跌越買
4. 設好最大停損：每批進場都設停損，看錯時認賠出場

實例：

  • 情境：某績優股從100元跌至50元，P/E從20倍降至10倍，產業前景不變
  • 操作：55元買第1批（1/3資金），52元買第2批，50元以下買第3批
  • 停損：若跌破45元（基本面惡化證據）則全數出場
  • 目標：回升至80元合理估值，獲利60%

適用時機：市場極度恐慌、個股被錯殺時；不適合基本面正在惡化的股票

2. 右側交易：順勢追漲的紀律

是什麼：在股價已經反轉向上時才買進、已經反轉向下時才賣出，即「在轉折點的右邊」操作，又稱順勢交易。

怎麼做：
1. 等待確認訊號：突破關鍵技術位（前高、均線金叉、頸線）才進場
2. 成交量確認：突破時必須伴隨量能放大，否則可能是假突破
3. 嚴格停損：進場同時設停損在前波低點或突破價下方2-3%
4. 移動停損保護利潤：獲利後切換移動停損讓利潤奔跑

實例：

  • 情境：某股從50元低點反彈，突破60元頸線
  • 操作：60.5元突破確認後進場，停損設57元（約5%），停利看前波高點75元
  • 若漲到70元後回測，移動停損已拉至65元，不會被洗掉
  • 勝率較高但單次利潤較小（買得貴、賣得早）

適用時機：趨勢初升段或中升段；盤整市會被反覆假突破修理

3. 左側右側混合策略：左側預判＋右側確認

是什麼：結合兩者優點，左側找機會區間，右側確認進場時機。BNF後期和小手川隆都採用此法。

怎麼做：
1. 左側預判：基本面或技術面判斷股價已到極端區域，開始觀察
2. 等待止跌訊號：不急著買，等K線出現止跌型態（錘子線、量縮長下影線）
3. 右側確認進場：出現明確反轉訊號（突破前波高點、均線黃金交叉）才正式建倉
4. 分批加碼：確認後逐步加碼，不是一次All-in

實例：

  • 情境：某股從100跌至55元，P/E僅8倍，開始關注
  • 左側預判：55-60元是可能底部區間
  • 等待：觀察是否出現止跌K線（如帶長下影線的錘子線）
  • 右側確認：股價突破63元（近期反彈高點）且量能放大 → 正式進場
  • 停損：跌破58元（左側預判區間下緣）出場

適用時機：想在相對低點進場但又不想承受純左側的套牢風險時；適合有耐心的波段交易者

4. BNF vs cis：左側右側的極致實踐

是什麼：日本兩位傳奇交易員的操作哲學，分別代表左側和右側的極致。

怎麼做：

  • BNF（左側代表）：

1. 利用乖離率判斷超跌（如日線乖離-10%以下）
2. 市場極端恐慌時逢低分批進場
3. 不預測明確轉折，靠分散買進降低風險
4. 強調耐心等待，時間是左側交易者的朋友

  • cis（右側代表）：

1. 等待技術突破訊號（前高突破、放量）
2. 趨勢確認後果斷加碼
3. 嚴格停損，趨勢反轉立即退出
4. 紀律與時機，遠勝預測與情緒

實例：

  • BNF風格：2008年金融海嘯，日經期貨暴跌，乖離率達-15%，BNF分批買進，指數從7000反彈至10000，大賺3000點
  • cis風格：某股放量突破前高，cis當天買進5%資金，隔天繼續漲加碼至20%，移動停損設5%，跌破即出場

適用時機：

  • BNF式左側：市場極端恐慌、估值極度偏低、有長線持有耐心
  • cis式右側：趨勢明確、短線靈活、紀律嚴格

心得

移動停損跟左側右側交易本質上都在解決同一個問題：如何在「讓利潤奔跑」和「保護已有獲利」之間取得平衡。

移動停損最關鍵的設定就是那個追蹤距離——設太小被洗掉、設太大等於沒設。比起死背百分比，用ATR或均線來動態調整更貼合實際，這個觀念要記住。

左側右側的選擇也一樣，不是非A即B。純左側抄底套牢風險太高，純右側追高利潤空間太小。最務實的做法是BNF後期的「左側預判、右側確認」——先用基本面和估值圈出機會區間，再等技術面給確認訊號才動手。這樣既不用承受純逆勢的心理壓力，也不會追在高點。

說到底，交易的核心矛盾就是「想買便宜」vs「想買確定」。左側買便宜但不確定，右側買確定但不便宜。混合策略是在這兩端之間找一個自己睡得著覺的位置。

一、開盤八法：15分鐘判斷當日盤勢

1. 開盤八法核心邏輯

是什麼：用開盤後15分鐘（9:05、9:10、9:15）的三個時段漲跌組合，判斷當日多空力道與行情走勢。統計上準確度相當高，是短線和當沖交易者的必備工具。

怎麼做：
1. 觀察三個時段：

  • A = 9:05收盤 vs 昨收（漲或跌）

2. 組合判讀：三個漲跌組合共8種（2³=8），每種對應不同盤勢
3. 關鍵時間：今日高低點多半發生在10:00~11:30之間，開盤15分鐘的力道決定了盤勢基調

實例：

  • 9:05漲、9:10漲、9:15漲 → 連三漲（第一法），開盤強勢但短線賣壓將出籠
  • 9:05跌、9:10跌、9:15跌 → 連三跌（第六法），利空或巨大賣壓，反彈不站上9:15則後市不樂觀

適用時機：每日開盤必做，是當沖判斷當日盤勢的第一步

2. 八種開盤型態完整解析

第一法：漲漲漲（連三紅）

  • 常見情境：反彈或多頭起漲或假突破
  • 開大漲（利多刺激）：高檔盤堅，多方操作較能獲利
  • 開小漲（無利多）：10點前跌破9:15則開高走低，尾盤小漲
  • 操作：以9:10收盤為基準點，拉回不破可作多

第二法：漲漲跌

  • 常見情境：多頭上攻中段
  • 開大漲：9:15下跌量若大於9:10上漲量，跌破9:05機率高，佈空
  • 開小漲：多頭格局整理盤，9:15收盤為多空點，佈多較有利
  • 操作：以9:05收盤為作多基準點

第三法：漲跌跌（一高兩低）

  • 常見情境：今日有上檔賣壓
  • 開小漲：三個時段都是壓力區，突破作多、跌破做空
  • 開大漲：盤中無法再突破9:15，尾盤多為小漲
  • 操作：壓力區明確，突破追多、跌破追空

第四法：跌漲漲（一低兩高）

  • 常見情境：波段跌幅末端或橫向震盪，有低接買盤
  • 開小跌：10:00前不能突破9:05，尾盤拉高機會大，持股可續抱
  • 開小跌但10:00前突破9:15且突破開盤價：盤中持續拉抬機率很高
  • 操作：底部有買盤支撐，可作多

第五法：跌跌漲（兩低一高）

  • 常見情境：波段跌幅中段
  • 開小跌：反彈力道，短多操作
  • 開大跌：10:00~11:30前未能突破站穩昨收，尾盤收黑機率高
  • 操作：9:10收盤不破可為低接進場點

第六法：跌跌跌（連三黑）

  • 常見情境：波段跌幅尾端
  • 開小跌：反彈宜減碼，9:15低點不破可短多
  • 開大跌：反彈突破9:05代表有買盤，明日可能還有高點，多單可續抱
  • 操作：極度弱勢，除非大跌反彈站上9:05否則偏空

第七法：跌漲跌（震盪型）

  • 常見情境：波段漲幅後的高檔震盪
  • 開小跌：9:15為多空線，突破短多、跌破短空
  • 開大跌：9:10為做多方基準，破9:05反向作空
  • 操作：多空膠著，以短打為原則

第八法：漲跌漲（震盪型）

  • 常見情境：多空短兵相接
  • 開大漲：帶量突破前高作多
  • 開小漲：帶量跌破9:10作空，量平觀望
  • 操作：振幅不大，觀望為主或短多

實例：

  • 情境：大盤昨收20000點，9:05收20100（漲）、9:10收20060（跌）、9:15收20080（漲）
  • 判讀：第八法「漲跌漲」，多空短兵相接
  • 操作：觀察盤中帶量突破20100作多，或帶量跌破20060作空

適用時機：每日9:15後即可判斷，搭配VWAP和成交量確認更準確

3. 開盤八法的實戰注意事項

是什麼：開盤八法不是聖杯，有幾個重要的使用限制和搭配方式。

怎麼做：
1. 開盤15分鐘的量能是關鍵：如果開盤量能極大（比前日同時段多50%以上），開盤型態的參考性更高
2. 11:30前的高低點往往是當日高低點：統計上大部分投資人在11:30前已完成當日主要買賣
3. 盤中突發消息會推翻開盤判斷：開盤八法只代表「早盤力道」，不是「全天保證」
4. 搭配VWAP使用：開盤八法判方向、VWAP判進場點，兩者結合勝率更高
5. 開大漲vs開小漲的判斷：開大漲（>1%）和開小漲（<0.5%）的同型態可能有完全不同的後續走勢

實例：

  • 第一法「漲漲漲」開大漲：利多消息支撐，高檔盤堅，可作多
  • 第一法「漲漲漲」開小漲：無利多消息卻連漲15分鐘，10點前跌破9:15就開高走低

適用時機：每次開盤後快速判斷，但不應機械式執行，需搭配消息面和量能

二、VWAP標準差通道當沖進階策略

4. VWAP標準差通道（VWAP Bands）原理

是什麼：在VWAP線上下方加上1倍和2倍標準差構成的通道，類似布林通道，但基準線是VWAP而非MA。能更精確判斷當日股價的正常波動範圍和極端位置。

怎麼做：
1. 計算方式：以VWAP為中心，計算當日所有成交價的標準差，上下各加1σ和2σ
2. 統計意義：

  • 68%的成交價落在VWAP±1σ內

3. 三條線構成：VWAP中線 + VWAP+1σ上軌 + VWAP-1σ下軌（進階可加2σ線）
4. 每日歸零：跟VWAP一樣，每天開盤重新計算

實例：

  • 情境：台積電VWAP在600元，1σ=5元，2σ=10元
  • VWAP+1σ=605元、VWAP+2σ=610元
  • VWAP-1σ=595元、VWAP-2σ=590元
  • 股價在595-605元之間波動是正常的（68%機率），超過610或低於590是極端（5%機率）

適用時機：當沖交易中判斷股價偏離程度；不適合波段操作（每日歸零）

5. VWAP標準差通道三大實戰策略

是什麼：用VWAP標準差通道判斷真假突破、找進出場點、識別反轉訊號。

怎麼做：

策略一：1σ通道趨勢跟隨
1. 股價在VWAP上方且沿+1σ上行 → 強勢多頭，回測VWAP時作多
2. 股價在VWAP下方且沿-1σ下行 → 強勢空頭，反彈至VWAP時作空
3. 股價在VWAP與±1σ之間來回 → 盤整，不操作或高拋低吸

策略二：2σ極端反轉
1. 股價觸及+2σ且量能萎縮 → 多頭極端，假突破機率高，可短空
2. 股價觸及-2σ且量能萎縮 → 空頭極端，假跌破機率高，可短多
3. 觸及2σ但帶量突破 → 真突破，順勢追進

策略三：VWAP+支撐壓力共振
1. 當VWAP剛好與日K線的支撐/壓力重疊 → 共振點，支撐/壓力效果加倍
2. 例如：某股日K前波支撐在100元，當日VWAP也剛好在100元附近 → 超強支撐
3. 回測此共振點時進場，停損設在共振點下方1-2%

實例：

  • 策略一：台積電9:30開盤後股價持續在VWAP上方且沿+1σ上行，10:00回測VWAP不破 → 作多進場
  • 策略二：某股11:00急拉至VWAP+2σ但量能只有前波60% → 假突破，短空進場，目標VWAP
  • 策略三：聯發科日K支撐在800元，當日VWAP在802元 → 回測800-802區間作多，停損795元

適用時機：

  • 策略一 → 趨勢明確的早盤（9:30-10:30）
  • 策略二 → 極端走勢出現時（盤中任何時間）
  • 策略三 → VWAP與技術面關鍵價位重疊時

6. 法人VWAP執行邏輯與散戶對策

是什麼：理解法人如何使用VWAP，就能預判法人的買賣節奏，站在法人同一邊。

怎麼做：

法人執行邏輯：
1. 法人目標不是買在最低點，是「不買得比市場平均貴」
2. 程式交易設定以VWAP為基準分批執行
3. 股價低於VWAP → 法人買盤積極（優於平均成本）
4. 股價遠高於VWAP → 法人買進力道減弱
5. 這就是VWAP形成強大支撐壓力的根本原因——法人程式在VWAP下方不斷買進

散戶對策：
1. 跟法人同方向：股價回測VWAP不破時作多，因為法人程式正在VWAP下方接單
2. 判斷真假突破：股價突破VWAP但受阻於+2σ且量能不足 → 法人不追價，假突破
3. 避開法人賣壓區：股價在VWAP上方乖離過大時，法人不會追高，散戶追高容易被洗
4. VWAP斜率判斷法人態度：VWAP斜率向上（均價持續上升）→ 法人持續買進；VWAP斜率平緩 → 法人觀望

實例：

  • 情境：某AI概念股9:00開盤後急漲3%，VWAP從100元緩步上升到102元
  • 法人態度：VWAP斜率向上，法人正在分批買進
  • 散戶操作：回測102元（VWAP）不破時作多，停損設100.5元（VWAP-1.5σ）
  • 反例：股價衝到108元（VWAP+3σ）但VWAP仍在102元緩升 → 乖離過大，法人不追，散戶追高風險極高

適用時機：每日當沖，尤其是法人買超明顯的個股，VWAP支撐效果更強

三、當沖選股進階：三指標+進階技巧整合

7. 當沖選股三指標整合SOP

是什麼：成交量、波動度、題材三個基礎篩選條件，加上籌碼面、關鍵價位、市場氛圍三個進階條件的完整選股SOP。

怎麼做：

基礎三指標（必備）：
1. 成交量：日均2000張以上，確保進出順暢。量太小會滑價、出不去、容易被操控
2. 波動度：日均振幅>1.5%，用ATR量化或肉眼觀察走勢圖。沒波動就沒利潤
3. 題材：新聞題材、產業趨勢、財報公告。有題材才有資金集中，才有連動效應

進階三條件（加分）：
1. 籌碼面：三大法人連續買超 → 帶動關注和當沖機會；當沖比例高的標的 → 短線資金集中，波動更大
2. 關鍵價位：突破支撐壓力 → 追價或殺出；整數關卡 → 心理面效果
3. 市場氛圍：大盤強勢 → 熱門族群更容易爆量；大盤疲弱 → 找逆勢抗跌或題材獨強的標的

完整SOP步驟：
1. 開盤前：打開成交量排行，挑前段班（>2000張）
2. 篩選題材：是否有新聞或產業趨勢支撐
3. 確認波動度：查看近幾日振幅或ATR
4. 加入進階條件：法人買超、接近關鍵價位、大盤氛圍配合
5. 縮小到3-5檔觀察名單，開盤後用VWAP和開盤八法判斷進場時機

實例：

  • 步驟1：成交量排行前30名
  • 步驟2：AI概念股族群今日有美國AI政策利多
  • 步驟3：篩出ATR>2%的5檔股票
  • 步驟4：其中3檔外資連續3日買超，且接近前波高點
  • 步驟5：3檔列入觀察，開盤用VWAP判斷進場

適用時機：每日盤前必做，是當沖成功的第一道關卡

8. 新手三大選股陷阱複習

是什麼：學吧整理的新手最常犯的選股錯誤，每個都是當沖失敗的主因。

怎麼做（避免方法）：

陷阱一：選冷門股

  • 新手以為冷門股「安全」，但成交量小代表滑價嚴重、出場困難、容易被操控
  • 對策：只選日均量2000張以上的標的

陷阱二：只看新聞標題

  • 看到新聞利多就衝進去，結果股價不動——「紙上利多」，沒有資金進場就沒有行情
  • 對策：題材必須配合成交量放大，有量才有價

陷阱三：忽略波段位置

  • 股票已漲一大段卻因當日強勢追高，結果主力在高檔出貨，散戶變「接刀俠」
  • 對策：即使當沖也要看支撐壓力和位階，不追高檔已漲多的股票

實例：

  • 陷阱一實例：某生技股日均量只有300張，進場後掛單半小時才成交，出場時找不到買方
  • 陷阱二實例：某公司宣布利多但成交量沒放大，股價開高走低，追進的散戶全被套
  • 陷阱三實例：某股已從50漲到120元（漲140%），當日再漲5%追進，結果隔天大跌8%

適用時機：每次選股前都檢查這三個陷阱

心得

今天學的開盤八法和VWAP標準差通道，剛好補完了我之前在當沖策略中缺的兩塊拼圖：「怎麼快速判斷今天的盤」和「用法人視角找進場點」。

1. 開盤八法的核心價值是「快速分類」。8種開盤型態就像8個劇本大綱，15分鐘就能判斷今天是趨勢盤還是震盪盤、是多頭佔優還是空頭佔優。但最關鍵的不是記住8種組合，而是理解背後的邏輯——「開大漲」和「開小漲」即使同型態，後續走勢可能完全不同。這跟之前學的盤勢屬性辨識（趨勢盤、震盪盤、誘多誘空盤）是互補的——開盤八法是快速分類工具，盤勢屬性是深度判斷工具。

2. VWAP標準差通道是布林通道的當沖版。之前學過布林通道用在波段操作，但日內交易用MA當基準線有個致命問題——MA的計算沒有考慮成交量。VWAP把成交量納入計算，代表的是「當天所有人的真實平均成本」，這就是為什麼法人用它——因為法人的目標就是「買得不比平均貴」。加了標準差通道後，不只看成本線，還能看極端位置，假突破的判斷更精確了。

3. 法人的VWAP執行邏輯是最有價值的洞察。知道法人在VWAP下方積極買進、在VWAP上方追價意願降低，就能理解為什麼VWAP會形成那麼強的支撐——不是什麼神秘力量，是法人程式交易在背後撐腰。站在法人同一邊操作，勝率自然提高。

4. 開盤八法+VWAP是當沖的黃金組合。開盤八法告訴你「今天偏多還是偏空」，VWAP告訴你「在哪個價位進場最安全」。先判方向、再找點位，這是系統化交易的基礎。以前學的「多看少做、先判斷再進場」這個原則，有了這兩個工具就能更具體地執行。

5. 選股SOP比進場技巧更重要。很多人花大量時間研究進場訊號，卻忽略了一開始就選錯股票——冷門股、沒波動、沒題材，再好的進場訊號也沒用。「有量、有波動、有題材」這三個基礎條件篩完後，再加上籌碼面和關鍵價位，把觀察名單縮小到3-5檔，專注觀察比分散火力有效得多。

一、交易系統三層次：策略、風控、心理

1. 交易系統的定義與為什麼需要

是什麼：交易系統是一套完整的規則和程序，告訴你什麼時候買、什麼時候賣、買多少、賣多少、錯了怎麼辦。它不只是技術指標的組合，而是策略+風控+心理的三位一體。

怎麼做：
1. 沒有系統 vs 有系統的差異：

  • 沒有系統：憑感覺進出場、聽消息做決策、虧了凹單、賺了就跑

2. 系統化的核心價值：排除情緒干擾、可重複執行、可回測驗證、可持續優化
3. 期望值公式：期望值 = （平均獲利 × 勝率）+（平均虧損 × 敗率）- 交易成本。期望值為正，長期才能賺錢；期望值為負，再好的運氣也會虧光

實例：

  • 沒有系統的人：看到新聞說AI股看好，就追高買進，跌了不知道該不該賣，最後凹單虧20%
  • 有系統的人：設定「MACD金叉+均線多排+法人連續買超3日」才進場，停損5%，停利看移動停損，長期勝率55%、賺賠比2:1 → 期望值 = 2×0.55 + (-1)×0.45 = 0.65，每筆平均賺0.65單位

適用時機：任何交易者都需要交易系統，不管是當沖、波段還是長投；沒有系統的交易等於賭博

2. 交易系統三層次架構

是什麼：完整的交易系統包含三個層次，缺一不可：交易策略（進出場規則）、風險控管（資金管理）、交易心理（紀律執行）。

怎麼做：

第一層：交易策略（進出場濾網）

  • 進場條件：什麼訊號出現才進場（如均線多排+法人買超+量增）
  • 出場條件：停損點（原始停損）、停利點（移動停損或目標價）、時間出場（持倉超過N日）
  • 濾網機制：不符合條件的訊號不進場，過濾掉盤整假訊號
  • 林恩如四法寶：均線（判斷多空）+ 趨勢線（判斷趨勢是否確立）+ 型態（判斷底部或頭部）+ 成交量（判斷是否有大人在照顧）

第二層：風險控管（資金管理）

  • 單筆風險：每筆交易最大虧損不超過總資金的1-2%
  • 部位控制：同時持有5-10檔分散風險，單一標的不超過總資金的20%
  • 連續虧損熔斷：連續虧損3筆或累計虧損超過6%，暫停交易1週冷靜
  • 2%+6%規則：單筆虧損不超過2%，單月虧損不超過6%

第三層：交易心理（紀律執行）

  • 紀律比策略更重要：有正期望值的策略但執行不了，跟沒有策略一樣
  • 四大投資紀律：該停損就停損、小賠大賺才長久、耐心等報酬、嚴守進出場規則
  • 交易日誌：記錄每筆交易的進場理由、持有期間心情、出場理由、事後檢討
  • 認知偏誤管理：確認偏誤（只看支持自己判斷的資訊）、過度自信（覺得自己比市場聰明）、損失厭惡（虧了不願認賠）

實例：

  • 只有策略沒有風控：勝率60%、賺賠比1:1，但單筆虧損不設限，一次虧50%就把之前10次獲利全吐回去
  • 只有策略和風控沒有心理：設定5%停損但到了5%不願意賣，心想「再等等看」，結果虧到20%
  • 三層齊全：策略決定進出場、風控決定虧多少、心理確保你執行下去

適用時機：建立交易系統的第一步就是先確認三層都到位，缺一不可

二、系統化交易 vs 主觀交易

3. 系統交易與主觀交易的差異

是什麼：系統交易靠規則、主觀交易靠判斷，兩者各有優缺點，可以混合使用。

怎麼做（比較）：

面向系統交易主觀交易
決策依據明確規則（如突破20日均線買進）經驗、判斷、盤感
紀律性高，照規則走低，容易受情緒影響
回測能力可回測驗證難以量化回測
彈性低，規則固定高，可臨場應變
適合對象喜歡數據、不想被情緒干擾反應快、盤感強、能承受壓力
技術門檻中高（需設計規則或寫程式）低（有交易平台就能開始）

混合模式（推薦）：

  • 以系統交易做核心（進出場規則、停損停利）
  • 以主觀判斷做調整（部位大小、突發事件應對）
  • 例如：系統訊號說進場，但今天有重大利空新聞，主觀判斷減少部位或不進場

實例：

  • 純系統交易：「MACD金叉+成交量放大2倍」→ 自動買進，停損5%，停利看移動停損
  • 純主觀交易：「感覺今天盤勢偏多，AI股有題材」→ 進場，虧了覺得「再等一下」
  • 混合模式：系統訊號確認進場時機，主觀判斷決定部位大小（如今天不確定性高，只用半倉）

適用時機：新手建議先從系統交易開始建立紀律，累積經驗後再加入主觀判斷

三、打造交易系統五步驟

4. 步驟一：市場分析與標的選擇

是什麼：決定你要在什麼市場、什麼標的上交易，這是系統的地基。

怎麼做：
1. 選市場：台股、美股、期貨、選擇權、外匯——每個市場特性不同
2. 選時間框架：當沖（1-5分K）、波段（日K）、長投（週K）——決定你的交易節奏
3. 選標的特徵：

  • 當沖：流動性高（日均量>5000張）、波動度夠（日均振幅>2%）、有題材性

4. 林恩如四大法寶選股：

  • 均線：站上20MA偏多，跌破20MA偏空

實例：

  • 當沖選股：篩選日均量>5000張 + 日均振幅>2% + 當日有題材的股票
  • 波段選股：20MA向上 + 法人連續買超3日 + 營收YoY>10%的股票
  • 長投選股：殖利率>4% + EPS連續3年成長 + 負債比<50%的股票

適用時機：建立系統前先決定市場和時間框架，不要什麼都做

5. 步驟二：設定進場規則

是什麼：明確定義什麼條件出現才進場，避免「感覺可以買」的模糊判斷。

怎麼做：
1. 簡單規則優於複雜規則：規則越簡單越容易執行，3-5個條件即可
2. 多維度確認：技術面+籌碼面+基本面至少兩面共振才進場
3. 林恩如的系統化選股SOP：

  • 大盤多空判斷：紅燈（大盤在20MA和100MA之上）→ 偏多操作

4. FinLab的被動投資SOP：

  • 用量化指標篩選（如PB<1、ROE>15%、營收成長>10%）

實例：

  • 波段進場規則：大盤紅燈 + 個股站上20MA + MACD金叉 + 外資連續買超3日 → 買進
  • 當沖進場規則：開盤30分鐘觀察 + VWAP上方 + 量增突破前高 → 追多
  • 長投進場規則：PB<歷史均值25分位 + EPS連續3年成長 + 殖利率>4% → 定期買進

適用時機：每次進場前對照規則清單，不符合條件就不進場

6. 步驟三：設定出場規則

是什麼：出場比進場更重要，沒有出場規則的交易系統是不完整的。

怎麼做：

停損五時機（林恩如版）：
1. 股價跌破20MA
2. 跌破上升趨勢線
3. 跌破波段前低
4. M頭型態完成
5. 股價創新低又破底

停利三方式：
1. 固定目標價停利：到達目標價就出場（如+10%）
2. 移動停損停利：停損點隨價格上移，保護已有利潤
3. 訊號反轉停利：進場訊號的反向訊號出現時出場

時間出場：

  • 進場後N日未達目標 → 出場（避免資金卡住）
  • 當沖不留倉：13:25前必須出場

實例：

  • 進場60元，停損57元（5%），停利看移動停損（追蹤高點回撤5%出場）
  • 進場後20日未漲超過3% → 時間出場，換股操作
  • 股價跌破20MA → 趨勢改變，停損出場

適用時機：每次進場前就要同時設定停損和停利，不是進場後才想

7. 步驟四：回測驗證

是什麼：用歷史數據驗證你的交易系統是否有效，是上線前的必經之路。

怎麼做：
1. 人工回測：看歷史K線圖，模擬你的系統在過去的表現

  • 優點：簡單直覺，適合複雜策略（如型態判斷）

2. 程式化回測：用Python或回測平台自動化驗證

  • 優點：快速、客觀、可大規模測試

3. 關鍵績效指標：

  • 勝率：>40%即可（搭配高賺賠比）

實例：

  • 回測「MACD金叉+20MA之上+量增2倍」策略，2020-2025年台積電
  • 結果：勝率48%、賺賠比2.3:1、期望值+0.62、最大回撤-15%
  • 結論：期望值為正，可以上線，但需要搭配嚴格停損

四大陷阱（複習）：

  • 過擬合：回測超好但實盤虧損 → 用Out-of-Sample驗證
  • 前視偏誤：用到未來資訊 → 確保只用當時已知數據
  • 忽略成本：手續費+稅+滑價 → 至少算0.6%
  • 策略失效：市場結構改變 → 定期監控績效

適用時機：任何策略上線前都必須回測，回測不過的不要用真錢

8. 步驟五：紀律執行與持續優化

是什麼：再好的系統，執行不了就等於零。紀律是交易系統的最後一哩路。

怎麼做：

紀律執行四原則：
1. 進場前寫下交易計畫：進場理由、停損點、停利點、部位大小，白紙黑字
2. 停損設了就不改：進場前設好的停損是經過理性思考的，盤中改停損是情緒化決策
3. 交易日誌：每筆交易記錄進出場理由、心情、結果，每週回顧
4. 連續虧損冷卻機制：連虧3筆或累計虧損6%，暫停1週

持續優化三步驟：
1. 每月回顧：統計勝率、賺賠比、期望值，與回測結果對比
2. 策略失效判斷：連續3個月期望值為負，或最大回撤超過歷史回測的2倍 → 策略可能失效
3. 調整但不過度優化：微調參數（如從20MA改為22MA），但不要大幅改變系統邏輯

贏家 vs 輸家思維對比：

  • 贏家在乎風控優先，輸家只看有利可圖
  • 贏家用交易系統獲利，輸家用想像與情緒交易
  • 贏家注重長期持續賺錢，輸家只在乎當下
  • 贏家確認策略有正期望值，輸家用少數經驗以偏概全

實例：

  • 交易日誌範例：2026/04/16 買進台積電600元，理由：MACD金叉+20MA之上+法人買超，停損570（5%），停利看移動停損。結果：持倉5天賣出620元（+3.3%），出場理由：移動停損觸發。心情：進場時有點緊張但按計畫執行，出場時小賺但符合系統規則。
  • 連虧冷卻：本週連續3筆停損出場 → 下週暫停交易，重新檢視系統是否有問題

適用時機：每天交易後寫日誌，每週回顧，每月檢討系統績效

四、從零開始建立交易系統的SOP

9. FinLab三階段SOP：學習→實驗→實戰

是什麼：FinLab提出的投資SOP，將投資分為三個階段：基礎知識建立、策略實驗（模擬）、投資實戰。

怎麼做：

階段一：基礎知識建立（1個月）

  • 目標：對投資有基本感覺，不需要樣樣精通
  • 方法：每天花10分鐘看一篇文章，了解K線、技術指標、財報關鍵數據
  • 重點：被動投資優先於主觀投資——用少數指標機械式操作，不帶感情
  • 關鍵認知：每個財務名詞就是一個數字，數字大於或小於多少就買進賣出
  • 禁忌：不要急著實戰！新手在行情大好時進場最容易賠錢

階段二：策略實驗（1個月到半年）

  • 目標：找到至少5種你滿意的被動選股策略
  • 方法：用模擬交易或回測驗證每種策略的績效
  • 選股策略條件：同時持有5-10檔以上、週期性換股（如7天、20天、60天）
  • 重要原則：不要自創方法！先用別人驗證過的策略，自創風險極大
  • 禁忌：還沒驗證就不要用真錢

階段三：投資實戰

  • 目標：用小資金驗證策略在真實市場的表現
  • 方法：先用總資金的10-20%開始，確認策略穩定後再逐步加碼
  • 關鍵：虧錢了是策略不好，不是你不好——換策略就好
  • 紀律：嚴格執行停損，單筆不超過2%

實例：

  • 階段一：看FinLab的PE/PB選股教學，了解本益比和淨值比怎麼用
  • 階段二：用FinLab回測「PB<1 + ROE>15% + 營收成長>10%」策略，回測5年績效
  • 階段三：用10萬元開始實戰，每月檢討績效，3個月後確認穩定再加碼到50萬

適用時機：完全新手從零開始建立交易系統的最佳路徑

10. 交易系統常見錯誤與對策

是什麼：建立交易系統過程中最容易犯的錯誤，以及如何避免。

怎麼做：

錯誤一：系統太複雜

  • 症狀：用了10個指標、5個濾網、3個出場條件，結果從來不進場
  • 對策：系統越簡單越好，3-5個條件就夠了。林恩如的四法寶（均線+趨勢線+型態+量）就是極簡系統
  • 實例：好的系統「20MA之上+MACD金叉+量增」，三個條件就夠了

錯誤二：不停修改參數

  • 症狀：看到虧損就改參數，20MA改成22MA，MACD改成14改成12
  • 對策：參數固定後至少執行100次交易再評估，不要頻繁修改
  • 實例：設定20MA就執行3個月，看100次交易的績效再決定是否調整

錯誤三：回測過度優化

  • 症狀：回測年化50%，實盤卻虧損
  • 對策：保留30%數據做Out-of-Sample測試，參數用整數不用小數
  • 實例：回測用2018-2022年數據優化，用2023-2025年數據驗證

錯誤四：沒有交易日誌

  • 症狀：虧了不知道為什麼虧，賺了不知道為什麼賺
  • 對策：每筆交易記錄進出場理由、心情、結果，每週回顧
  • 實例：交易日誌格式：日期 | 標的 | 方向 | 進場價 | 出場價 | 理由 | 心情 | 檢討

錯誤五：忽略交易成本

  • 症狀：回測年化30%，扣掉手續費+稅+滑價只剩5%
  • 對策：回測一定要設定交易成本（台股：買0.1425%+賣0.1425%+證交稅0.3%=約0.6%）
  • 實例：當沖策略每天交易5次，手續費就吃掉3%利潤，必須確認扣除成本後期望值仍為正

適用時機：建立交易系統時逐項檢查，每個錯誤都可能讓你的系統從獲利變虧損

心得

今天學的交易系統三層次架構，是之前所有學習內容的「統合框架」。之前學了技術分析、基本面、籌碼面、當沖、波段、風控等各種工具，但一直缺少一個把所有工具組織起來的系統。今天的內容補上了這個缺口：

1. 策略、風控、心理三層缺一不可。這跟之前學的Tom Basso排序一致——心理最重要、風控其次、策略排最後。有最好的策略但執行不了（心理崩潰），跟沒有策略一樣。有好的心理但沒有風控，一次黑天鵝就可能歸零。三層都到位才能長期存活。

2. 系統化交易的核心是「可重複」。不是每次都賺，而是每次都按規則做。贏家思維那篇文章說得很好：「沒有策略的交易方式，究竟可以這樣賺多久？是一個月、一年、三年，還是一輩子？」靠運氣賺的錢最終會靠實力虧回去，只有正期望值的系統才能持續獲利。

3. 林恩如的四法寶（均線+趨勢線+型態+量）是極簡系統的最佳範例。四個條件就夠了，不需要十個指標。均線判斷多空、趨勢線判斷趨勢確立、型態判斷底部或頭部、成交量判斷有沒有人在照顧。這四個條件簡單、明確、可執行，而且每個都有明確的定義，不會有「感覺好像」的模糊地帶。

4. FinLab的三階段SOP很實用：學習→實驗→實戰。很多新手跳過前兩個階段直接實戰，結果用真金白銀繳學費。正確的做法是先用模擬交易驗證策略至少100次，確認期望值為正才用真錢。這跟之前學的「回測」概念一致——回測不過的策略不要用真錢。

5. 交易日誌是交易系統的靈魂。沒有交易日誌，你永遠不知道自己為什麼賺、為什麼虧。每一筆交易都要記錄進場理由、持有期間心情、出場理由、事後檢討。每週回顧、每月總結，才能持續優化系統。這跟之前學的交易日誌四步驟（買進→持有→賣出→月度檢討）完全一致。

6. 系統交易和主觀交易不是對立的，是互補的。系統交易處理規則內的情況（佔70-80%），主觀判斷處理規則外的突發事件（佔20-30%）。最好的方式是「系統為主、主觀為輔」——進出場規則固定不變，但部位大小和特殊情況的應對可以主觀判斷。

一、道氏理論突破交易策略

1. 突破進場策略

是什麼：當股價突破道氏理論定義的「最新高價標準」時進場做多，這是順勢交易中最明確的進場方式。

怎麼做：
1. 確認趨勢：先用道氏理論確認目前處於上漲趨勢（高價與低價持續上移）
2. 等待突破：當股價突破前波高點（最新高價標準）時進場
3. 設停損：突破進場的停損設在前波低點下方，或突破點下方2-3%
4. 確認量能：向上突破必須伴隨成交量放大，否則可能是假突破

實例：

  • 情境：台積電(2330)在550-580區間整理兩週，形成道氏理論的高價標準580元
  • 操作：當股價帶量突破580元時進場做多，停損設在565元（前波低點下方）
  • 若突破後量能不足（成交量低於20日均量），則觀望不進場

適用時機：趨勢明確時（高價低價持續上移），不適合盤整無方向時

2. 回調買進策略（比突破進場更優）

是什麼：突破後等待價格回調到突破位置再進場，這是比「突破直接進場」勝率更高的策略。

怎麼做：
1. 確認突破：先等股價突破前波高點（不進場，只確認方向）
2. 等待回調：突破後價格回落到突破價位附近時進場
3. 確認支撐：回調時價格不能跌破突破點（前阻力變支撐）
4. 進場條件：回調到突破點附近 + 出現止跌訊號（如K線止跌、量縮）時進場

實例：

  • 情境：聯發科(2454)從900元突破前波高點920元，之後回調到920-930元附近
  • 操作：在925元附近出現量縮止跌時進場，停損設在910元（跌破前支撐）
  • 盈虧比：突破進場在920元進場，停損890元（風險30元）；回調進場在925元進場，停損910元（風險15元），風險少一半但目標相同

適用時機：趨勢明確的回調，不適合V型反轉（不會給回調機會）

3. 突破後獲利目標設定：N字波動法

是什麼：利用道氏理論的N字波動（上漲-回調-再上漲）來設定停利目標，把第一波上漲的幅度等量投射到突破後的位置。

怎麼做：
1. 測量第一波：找出突破前的上漲波段幅度（從低點到高點的價差）
2. 投射目標A：突破價位 + 第一波幅度 = 停利目標A
3. 投射目標B：回調低點 + 第一波幅度 = 停利目標B（更保守）
4. 分批出場：到達目標A出1/2，到達目標A的1.272倍（斐波那契擴展）再出1/2

實例：

  • 情境：某股從100元漲到120元（第一波上漲20元），回調到110元，然後突破120元
  • 目標A：120 + 20 = 140元
  • 目標B：110 + 20 = 130元（更保守目標）
  • 操作：突破120元進場，130元出1/3，140元出1/3，剩下用移動停損保護

適用時機：趨勢明確的波段行情，不適合盤整或劇烈波動行情

二、突破後反轉判斷與假突破應對

4. 假突破判斷三原則

是什麼：突破後價格反向回落，判斷是真突破還是假突破的方法。

怎麼做：
1. 量能確認：真突破必須帶量（成交量>20日均量1.5倍以上），無量突破大概率是假突破
2. 收盤確認：突破當天收盤價必須站在突破點之上，盤中突破不算數
3. 持續確認：突破後連續3天收盤價都在突破點之上，才是真突破

實例：

  • 情境：某股盤中突破100元壓力，但收盤99元
  • 判斷：假突破，不進場
  • 情境2：帶量突破100元，連續3天收盤101、102、103元
  • 判斷：真突破，可以回調時進場

適用時機：每次突破都必須做假突破確認，不分多空

5. 突破後反轉的道氏理論判斷

是什麼：當突破後價格跌回突破點以下，用道氏理論判斷是暫時回調還是趨勢反轉。

怎麼做：
1. 關鍵判斷：看最新低價是否下移
2. 低價未下移：只是暫時回調，上漲趨勢可能持續 → 持有或加碼
3. 低價下移：趨勢可能反轉 → 減碼或出場
4. 圖表型態輔助：回調時若出現頭肩頂、雙重頂等反轉型態，確認反轉機率大增

實例：

  • 情境：股價從100突破110後回到105元
  • 判斷：如果前波低點是95元，而回調低點在100元以上（低價未下移），趨勢仍在上漲
  • 判斷2：如果回調跌破95元（低價下移），趨勢可能反轉

適用時機：突破後回調時判斷是否要繼續持有

三、順勢交易系統建立SOP

6. 順勢交易核心三原則

是什麼：順勢交易（Trend Following）是跟隨市場趨勢方向進行交易，在上漲趨勢中做多、下跌趨勢中做空，不做預測只做確認。

怎麼做：
1. 確認趨勢方向：用均線排列（多頭排列：5MA>20MA>60MA）或道氏理論（高價低價持續上移）確認
2. 順勢進場：趨勢向上只做多不做空，趨勢向下只做空不做多
3. 預設停損停利：進場前就設定好出場點，不因情緒改變

三不原則：

  • 不預測趨勢何時結束（等訊號確認）
  • 不逆勢操作（趨勢向上不找空點）
  • 不因恐懼提前出場（讓停損機制執行）

實例：

  • 情境：台積電20MA向上，股價在20MA之上 → 確認多頭趨勢
  • 操作：只在股價回調到20MA附近時買進，停損設在20MA下方2%
  • 心態：不猜頭部，不因「漲太多了」而提前賣出，讓移動停損保護獲利

適用時機：趨勢明確時；盤整期不適合順勢交易，應觀望或用區間策略

7. 均線時間週期搭配策略

是什麼：不同投資期間搭配不同均線時間週期，短期用5MA/10MA、中期用20MA/60MA、長期用120MA/240MA，並用長短均線交叉確認趨勢。

怎麼做：
1. 選擇時間週期：根據你的投資期間選擇對應均線

  • 當沖/短線（1天~3個月）：5MA + 10MA

2. 雙均線確認：短期均線在長期均線之上=多頭，之下=空頭
3. 訊號權衡：小週期訊號快但雜訊多，大週期訊號慢但可靠
4. 實務做法：用大週期判斷方向（60MA是否向上），用小週期找進場點（5MA回調到20MA附近）

實例：

  • 波段操作：看60MA判斷中期趨勢方向，看20MA找進場時機
  • 台積電60MA向上（中期多頭）→ 等股價回調到20MA附近 → 出現止跌K線 → 進場做多
  • 停損：跌破20MA的2-3%

適用時機：任何時間框架的交易，但必須先確定你的投資期間再選均線

8. 圖表型態輔助突破判斷

是什麼：在道氏理論的趨勢判斷基礎上，用矩形和三角形型態輔助判斷突破方向。

怎麼做：

矩形型態：
1. 特徵：高價與低價平行來回波動，形成盤整區間
2. 進場：等突破矩形上緣（阻力線）時進場做多
3. 確認：突破必須帶量，否則觀望

上升三角形：
1. 特徵：水平阻力線 + 上升支撐線，低價逐步上移
2. 進場：突破水平阻力線時進場
3. 目標：突破後的目標價 = 突破點 + 三角形底邊高度

實例：

  • 情境：某股在90-100元形成矩形整理3週
  • 操作：帶量突破100元時進場做多
  • 目標：100 + (100-90) = 110元
  • 停損：跌破95元（矩形中軸下方）

適用時機：趨勢中的盤整期，等型態突破再進場，比盤整中頻繁交易更安全

心得

今天的道氏理論突破回調策略跟之前的「突破回踩確認策略」有關聯但更深入。之前學的是回踩的三條件（籌碼峰支撐+均線共振+板塊指數共振），今天學的是道氏理論框架下的突破策略，兩者可以互補：

1. 回調買進比突破買進更好。這個結論跟之前學的一致，但今天有了更量化的比較——同樣的目標價，突破進場風險30元，回調進場風險15元，盈虧比直接翻倍。這不是理論，是數學：進場點越靠近支撐，風險越小，盈虧比越好。

2. N字波動設定停利目標是今天的新收穫。之前學停利大多用移動停損或固定比例，N字波動法提供了一個更客觀的目標：第一波漲幅多少，第二波就至少漲多少。這跟斐波那契擴展的概念一致（1.0倍和1.272倍目標），但更直覺好算。

3. 假突破三確認（量能+收盤+持續3天）是鐵律。之前學過真假突破四原則，今天又從道氏理論的角度驗證了同樣的道理——盤中突破不算數，收盤價才是真正的突破。這個觀念太重要了，散戶最常犯的錯就是看到盤中突破就追，結果收盤跌回來被套。

4. 道氏理論的低價是否下移是判斷趨勢延續vs反轉的關鍵。這個概念很實用——不用猜頭部，只要看低價是否持續上移就好。低價上移=趨勢延續，低價下移=可能反轉。簡單、明確、可執行，比一堆技術指標湊在一起更可靠。

5. 順勢交易的「三不原則」——不預測趨勢結束、不逆勢操作、不因恐懼提前出場。這三條看似簡單，實際上是散戶最難做到的。「漲太多了」是最危險的想法，因為你是在預測趨勢結束，但趨勢可能還會繼續很久。讓停損機制做決定，不要讓恐懼做決定。

1. 均線糾結的核心意義

是什麼：短、中、長期均線靠攏在一起，代表不同週期的投資人持股成本都差不多，盤整越久累積的量能越大，一旦發動就是波段大行情。

怎麼做（理解邏輯）：
1. 盤整5-6天：5日均線走平
2. 盤整10-12天：10日均線也走平
3. 盤整更久：20日、60日、甚至120日、240日均線全部靠攏
4. 越多條均線糾結：代表盤整時間越長，累積能量越大，突破後的力道越強
5. 口訣：突破均線糾結的向上紅K線是起漲的開始，跌破均線糾結的向下黑K線是起跌的開始

  • 鴻海（2317）2月底帶量長紅K棒突破均線糾結，5日線率先上彎反應，股價沿5日線墊升，一個月獲利近50%
  • 東森（2614）週線圖盤整超過一年，突破均線糾結後短短一個多月漲超過250%

適用時機：尋找波段飆股的進場點，盤整越久機會越大

步驟一：日常選股觀察

  • 每天收盤後看當天強勢股
  • 在強勢股中找「放量上漲 + 突破均線糾結 + 中長紅K線」的股票

  • 第2天或第3天如果沒有大漲，都還可以進場
  • 不用急著第一天就追

步驟三：飆漲停的應對

  • 如果起漲第2天就飆漲停 → 不追漲停板
  • 等第1段飆漲後回檔再漲時切入
  • 賺續勢第2段飆漲（大多數飆股都有2段飆漲期）

步驟四：停損設定

  • 以進場當天K線最低點為停損

  • 用「智慧K線交易法」操作

  • 每天下午1:20看交易情形

  • 再次出現上漲紅K線 + 收盤突破前一天K線最高點

步驟八：循環操作

  • 上漲一段後若再次盤整，均線再度糾結
  • 同樣把握均線糾結再次突破向上的做多機會

實例：

  • 情境：某股5日、10日、20日、60日均線糾結2個月，某天帶量長紅突破
  • 操作1：隔天進場做多，停損設進場K線最低點

  • 操作3：漲一段後跌破前K低點 → 1:20市價賣出
  • 操作4：回檔仍多頭，紅K突破前高 → 再進場做多
  • 操作5：漲完又盤整均線糾結 → 等再次突破

適用時機：所有均線糾結突破的波段操作，是完整的循環操作系統

3. 向下跌破做空八步操作法

2. 第2天或第3天如果沒有連續大跌，仍可進場做空
3. 起跌第2天跌停鎖死 → 不追空，等第1段飆跌後反彈再跌時切入
4. 停損設進場K線最高點，小黑K線則以反彈5%為停損
5. 每天收盤沒突破前一天K線最高點 → 續抱空單
6. 下午1:20確認收盤突破前一天K線最高點 → 市價回補
7. 回補後反彈若仍空頭趨勢，黑K跌破前低 → 再進場做空
8. 下跌一段後再次盤整均線糾結 → 把握再次跌破的做空機會

實例：

  • 裕日車（2227）8月初跌破盤整區間後持續下挫，再次盤整後1月初又跌破，跌幅擴大

適用時機：均線糾結向下跌破時，適合有融券或期貨操作能力的交易者

二、均線糾結進階選股三招

4. 避免KD/MACD背離的假突破

是什麼：股價突破均線糾結但技術指標出現背離，容易是假突破、騙線。

怎麼做：
1. 判斷背離：股價帶量創新高，但KD或MACD沒有創新高 → 頂背離
2. 頂背離 + 均線糾結突破 = 假突破機率極高，不要追
3. 指標同步創高 + 均線糾結突破 = 真突破，可以進場

實例：

  • 假突破：新日興（3376）帶量突破均線糾結，但KD和MACD沒有創高 → 後續股價回檔
  • 真突破：保瑞（6472）帶量突破均線糾結，KD和MACD同步創高 → 後續有一波漲幅

適用時機：每次看到均線糾結突破時都要同時檢查KD和MACD是否同步創高，背離就避開

5. 挑選長期均線走多的個股

是什麼：均線糾結突破要成功，中長期趨勢必須配合——月線和季線維持多方，突破才有續航力。

怎麼做：
1. 突破均線糾結時，月線（20日線）仍在下彎 → 中期趨勢偏空，突破容易失敗
2. 月線和季線都維持多方趨勢 → 中長期趨勢支撐，突破後短期均線率先反映形成多頭排列
3. 短期均線帶動中長期均線 → 形成均線多頭排列，股價持續上漲

實例：

  • 失敗：材料-KY（4763）4月帶量突破均線糾結，但月線仍下彎 → 股價不易扭轉空方，回檔繼續下探
  • 成功：旺矽（6223）中長期均線維持多方，突破均線糾結後短期均線率先反映，多頭排列帶動股價續漲

適用時機：選股時要檢查月線和季線方向，不是只看短期均線突破就進場

6. 搭配籌碼面搭上主力順風車

是什麼：均線糾結突破時搭配法人買賣超，確認有主力資金支撐，突破更有續航力。

怎麼做：
1. 股價還沒突破均線糾結時，觀察投信或外資是否開始連續買超
2. 投信連續買超 + 股價突破均線糾結 → 主力已在布局，突破更可靠
3. 突破後法人繼續買超 → 這個突破有機會帶來一波多頭行情
4. 突破後法人轉賣 → 可能是假突破或行情已結束

實例：

  • 旺矽（6223）：股價突破均線糾結前，投信已連續買超；突破後投信繼續買 → 股價持續上漲
  • 藝舍-KY（2724）：突破均線糾結後外資持續買超，大盤漲多回檔但股價不受影響，漲幅超過50%

適用時機：均線糾結突破時搭配籌碼面做第二層確認，技術面+籌碼面共振才進場

三、均線糾結方向判斷兩大方法

7. 區間突破辨識法

是什麼：最直觀的方法——均線糾結就是盤整區間，突破上緣偏多、跌破下緣偏空。

怎麼做：
1. 找出均線糾結期間的價格區間上下緣
2. 突破上緣 → 假設行情會走出去，進場做多
3. 跌破下緣 → 進場做空
4. 如果是假突破跌回來 → 再做反向
5. 核心思維：不能因為擔心假突破就不進場，否則真突破時也會錯過

適用時機：保守型交易者，等確認突破再進場；缺點是進場點比較晚，利潤較少

8. 順勢預判法（推薦）

是什麼：在均線糾結還沒突破前，先順著長期趨勢方向找進場點，提前卡位獲利更大。

怎麼做：
1. 判斷長期趨勢：用季線（60MA）方向判斷——季線往上=多頭、往下=空頭
2. 順勢找進場點：

  • 多頭趨勢中：在盤整區內找相對低點做多

3. 不用等突破再進場：提前卡位獲利空間更大
4. 背著人性做交易：不要預設立場，趨勢沒結束前先假設會持續
5. 太早猜頭會讓你賺不到趨勢錢：一路上漲中一路猜頭的人，不可能賺到那一段趨勢

實例：

  • 大盤從底部上漲一大段後橫向盤整4個月，均線糾結
  • 60MA方向往上 → 先假設盤整後會往上走
  • 在盤整區內找進場點做多
  • 盤整結束後行情往上發動 → 起漲點就已在場內

風險管理搭配：
1. 進場設停損（盤整區下緣或1/2 ATR）
2. 用長均線當移動停利（如40MA或60MA）
3. 虧損控制在本金2%以內
4. 停損設定考慮波動：波動大停損大、波動小停損小（用ATR計算）

適用時機：積極型交易者，願意承受盤整期間的時間成本換取更大獲利空間

四、短線動能突破飆股選股法

9. 帶量突破策略核心邏輯

是什麼：股價創新高 + 成交量爆大量 = 市場資金明顯集中推升，是最經典的動能交易進場訊號。

怎麼做（選股條件）：

核心條件：
1. 股價創新高：創近20日/60日/120日新高，20日新高最常用
2. 成交爆大量：成交量 > 近5日均量的3倍，3倍是明顯閾值

10. 動能突破兩種進場方式

是什麼：帶量突破後的進場方式分為「積極追價」和「保守回測」兩種。

怎麼做：

方式一：積極追價買進

  • 股價突破前波高點後立即追價
  • 突破通常發生在開盤後30分鐘至1小時內
  • 觀察成交量是否持續增加
  • 停損：跌破前波高點下方0.5%-1%（或4-8個Tick）

方式二：等待回測再進場（推薦）
