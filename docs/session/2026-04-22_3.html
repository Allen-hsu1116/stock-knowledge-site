籌碼面分析：主力出貨手法辨識×洗盤vs出貨判斷×防守線系統

主力出貨，是每個散戶最怕遇到的事——你以為是在搭順風車，結果你上車的那一刻，主力已經準備下車了。辨識主力出貨，是籌碼面分析中最實戰也最關鍵的技能。搞懂主力怎麼出貨，你才知道什麼時候該跑。

一、主力出貨的四種基本手法

是什麼

主力出貨 = 主力把手中大量籌碼賣給散戶的過程。因為主力持股量極大，不可能一次賣完（那會直接崩盤），所以必須用各種手法讓散戶「自願」接盤。

怎麼做 — 四種出貨手法對照：

手法特徵K線表現成交量散戶最容易犯的錯
**拉高出貨**邊拉邊賣，股價持續創新高帶下影線的中大陽線暴量，每天有大量追漲停，以為還會更高
**震盪出貨**高位橫盤震盪，小幅創高一張一合，起伏拉鋸中量，無法放大以為是盤整蓄勢，等突破
**打壓出貨**橫向K線上影線拉高，對準買盤打壓上影線+橫盤量能萎縮搶反彈，以為跌深了
**反彈出貨**跌深後小反彈，趁反彈出清剩餘30日均線下方反彈反彈量小以為底部到了，抄底

實例：

  • 拉高出貨：某股從50元漲到80元過程中，主力邊拉邊賣。散戶看到連續長紅就追進，殊不知每根長紅的成交量暴增是因為主力在倒貨。爆量不漲或爆量收長上影線，就是拉高出貨的訊號。
  • 震盪出貨：股價在75-80元區間來回震盪2-3週，每次「突破」80元就回落，散戶以為是突破蓄勢，其實主力在每次反彈時加大賣出力度。
  • 急拉型出貨：長期整理後突然急拉創新高，散戶以為是新波段行情開始，結果爆量之後迅速反轉下殺。創新高那根K線的低點被跌破就是假突破確認。

二、四種拉高出貨模式（黃韋中分類）

是什麼

技術分析大師黃韋中在《主控戰略成交量》中將主力出貨分為四種模式，每種模式的型態特徵和出場時機不同，是辨識出貨的經典框架。

模式一：反轉型出貨

特徵：

  • 多頭上漲過程中，先出現正反轉低點
  • 股價以中長紅或跳空創新高
  • 這段強勢上漲 = 拉高出貨段

確認訊號：
1. 滿足型態學測量目標或波浪理論預測價位
2. 爆量止漲
3. 拉回過程出現「價跌量增」
4. 跌破拉高出貨段的正反轉低點 → 確認多頭反轉

怎麼做：

  • 爆量長紅的最低點設為「防守線」
  • 跌破防守線 → 立即出場，不戀戰
  • 股本小的股票：主力在拉高階段已出7-8成，壓低出貨時很快出清，反彈幅度小
  • 股本大的股票：反彈幅度約1/2以上，可以利用反彈減碼

模式二：解套型出貨

特徵：

  • 前波高點出貨後股價反轉下跌
  • 跌到支撐位開始反彈，反彈幅度超過1/2
  • 反彈走勢相當強勁、價量齊揚
  • 約7成的人會誤以為多頭仍在

怎麼做：

  • 解套型出貨只適宜短多單，嚴設停損
  • 反彈到前波高點附近 = 出清時機
  • 最大陷阱：散戶在反彈最強的時候才開始積極作多，結果主力剛好出完

模式三：末升段出貨

特徵：

  • 主升段行情後進入三角型態整理
  • 整理過程成交量「量能退潮」
  • 末升段上漲時主力搭配利多消息「邊拉邊出」

確認訊號：

  • 超過型態學測量目標或波浪理論預測價位後
  • 爆量止漲
  • 破壞短線多頭支撐

怎麼做：

  • 末升段出貨結束後，股價不是進入整理就是盤頭反轉
  • 通常會出現小5波下跌走勢
  • 出現爆量不漲 + 破壞支撐 → 全數賣出

模式四：急拉型出貨

特徵：

  • 出貨前有很長時間的整理期（跟末升段的三角整理不同）
  • 整理結束後急拉創新高，讓人以為是大波段行情開始
  • 常利用大盤逃命波或解套型出貨的時機發動

怎麼做：

  • 關鍵在創新高時的價量觀察
  • 創新高那根K線（通常是價漲量增的中長紅K線）的低點 = 關鍵防守線
  • 攻擊K線的低點不該被跌破，跌破 = 假突破確認
  • 跌破當天立即賣出，退場觀望

三、洗盤 vs 出貨：最關鍵的辨識

是什麼

洗盤和出貨是主力操作的兩面，看起來很像（都是股價下跌+成交量變化），但本質完全不同：洗盤是為了「洗掉不堅定的散戶」，出貨是為了「把籌碼賣給散戶」。判斷錯誤的代價極大——把洗盤當出貨會錯過大行情，把出貨當洗盤會被套在山頂。

核心判斷原則：位置決定一切

出貨時股價在高位，洗盤時股價在低位或相對低位。

這是唯一的黃金法則。脫離位置談洗盤出貨都是耍流氓。

洗盤 vs 出貨對照表

判斷維度洗盤出貨
**位置**低檔或主升段中繼高檔或末升段
**成交量**縮量下跌（量縮價跌）放量不漲或價跌量增
**防守線**不跌破第一防守線跌破所有防守線
**均線**長期均線仍多頭排列長期均線開始下彎
**K線**長黑後迅速收復長黑後無力收復
**利多消息**無特殊利多利多頻傳但股價不漲
**主力動向**主力未大量賣出主力大量賣超
**時間**2-5天完成洗盤持續性出貨

防守線系統

第一防守線：起漲長紅K線的最低點

  • 股價回測不跌破 = 強勢整理，3-6天內會再拉長紅
  • 跌破但收盤站回 = 仍在洗盤範圍

第二防守線：前期支撐或重要均線（20MA或60MA）

  • 跌破第一防守線但不破第二防守線 = 較長時間的震盪整理
  • MACD由紅柱轉綠柱，等綠柱再轉紅柱時突破

怎麼做：
1. 買進後立即標記第一防守線（起漲長紅最低點）
2. 收盤跌破第一防守線 → 先出場觀望
3. 如果隨後長紅吞噬長黑 = 甩轎洗盤，可重新進場
4. 跌破第一防守線後，觀察是否守住第二防守線
5. 跌破所有防守線 = 不是洗盤，是出貨或趨勢反轉

四、主力洗盤三種手法

手法一：短線拉回洗盤

特徵：主升段中股價回調4-5天，散戶失去耐心出場後，主力再拉長紅

辨識：

  • MACD紅柱縮短後再變長 = 洗盤結束
  • 股價不跌破第一防守線
  • 3-6天內完成

實例：某股從50元起漲到60元，出現5根長紅後回測55元（第一防守線），3天後再拉長紅突破65元。散戶在55元附近被洗出場，錯過後續行情。

手法二：K線洗盤

特徵：主升段出現長上影線（看似避雷針），搭配大量，散戶以為是高檔反轉訊號

辨識：

  • 觀察第一防守線是否守住
  • 如果後續股價穿越上影線最高點（突破線） = 確認是洗盤
  • 關鍵：收盤價守住第一防守線，盤中跌破但收盤站回也算

實例：利勤（4426）出現白色上影線+綠色巨量，看起來完全像避雷針。但後續股價沒有跌破第一防守線，反而穿越突破線開啟新行情。散戶都懂的避雷針，主力當然也懂，所以會反向利用。

手法三：起漲前的甩轎

特徵：股價長期盤整，突然長黑跌破盤整區，散戶停損出場後，股價反轉向上

辨識：

  • 長黑跌破盤整區後突然停止下跌
  • 之後長紅吞噬長黑 = 確認甩轎

怎麼做：

  • 不要賭長黑是甩轎！ 如果遇到往下噴出就完了
  • 先停損出場，等後續股價突破長黑最高點再買回
  • 寧可少賺一段，也不要冒著大賠的風險

五、出貨五道紅旗判斷系統

是什麼

結合量價、籌碼、技術三個維度，建立出貨判斷的五道紅旗。紅旗越多，出貨機率越高。

五道紅旗：

紅旗判斷標準權重
**量價背離**股價創高但成交量萎縮，或爆量不漲★★★★★
**利多不漲**利好消息頻傳但股價滯漲或反跌★★★★
**主力賣超**法人或大戶連續賣超3天以上★★★★★
**防守線跌破**跌破起漲長紅最低點★★★★
**均線破壞**長期均線（30MA/60MA）下彎或死亡交叉★★★

怎麼做：

  • 1-2面紅旗：提高警覺，觀察為主
  • 3面紅旗：開始減碼，設好停損
  • 4面以上紅旗：果斷出場，不要猶豫

實例：

  • 某股連續3天利好消息（營收創高、法人買超、新聞看好），但股價從100元只漲到102元，成交量從5000張暴增到15000張
  • 紅旗1：量價背離 ✅（爆量不漲）
  • 紅旗2：利多不漲 ✅（三個利多只漲2%）
  • 紅旗3：法人賣超 ✅（外資連續3天賣超）
  • 紅旗4：防守線 ✅（跌破起漲長紅最低點98元）
  • 結論：4面紅旗，果斷出場

六、主力作價六部曲

是什麼

主力操作一檔股票的完整循環：吃→脫→洗→拉→誘→嘎

六部曲詳解：

階段特徵散戶心理正確動作
**1. 吃貨**市場渾然不覺、利空不斷、爆量不跌、長黑不見低恐懼、不敢買開始觀察，不急進場
**2. 脫底**急速連續長紅（低檔四紅K）、讓賣在低檔者買不回來後悔賣掉、想追又不敢突破確認後進場
**3. 洗盤**大量鎖碼後拉高回測、重挫2-3天或盤整8-10天恐慌、以為行情結束守住防守線，不輕易出場
**4. 拉抬**迅雷不及掩耳快拉猛漲、法人進場、散戶不知道發生什麼來不及上車、開始注意主升段加碼機會
**5. 誘空**漲幅已大、裝死誘空、長紅長黑交錯、天量出現以為還會跌、不敢進開始減碼、設移動停損
**6. 嘎空**利多不斷、連續長紅（高檔三紅K）、媒體一致看好追高、畫大餅全數出場、不戀戰

怎麼做：

  • 在「吃貨」階段開始追蹤（爆量不跌是重要訊號）
  • 在「脫底」確認後進場（突破前高）
  • 在「洗盤」時守住防守線（不輕易出場）
  • 在「拉抬」時加碼（主升段）
  • 在「誘空」開始減碼（天量是警訊）
  • 在「嘎空」時全數出場（利多不斷=出貨訊號）

七、出貨 vs 洗盤實戰決策樹

決策流程

股價下跌 → 位置判斷
├── 高檔（已大漲30%以上）
│   ├── 量價背離（價漲量縮或爆量不漲）→ 出貨 ✅
│   ├── 利多不漲 → 出貨 ✅
│   ├── 主力賣超 → 出貨 ✅
│   └── 跌破防守線 → 確認出貨，立即出場
├── 中段（主升段回調）
│   ├── 守住第一防守線 → 洗盤機率高
│   ├── 跌破第一防守線但守住第二防守線 → 觀察
│   └── 跌破第二防守線 → 可能出貨或趨勢反轉
└── 低檔（底部區域）
    ├── 爆量不跌 → 吃貨訊號
    ├── 長黑後長紅吞噬 → 甩轎洗盤
    └── 持續破底 → 真實下跌，不是洗盤

實戰SOP

1. 每次買進後立即標記防守線（起漲長紅最低點）
2. 收盤跌破防守線 → 先出場
3. 出場後觀察：

  • 長紅吞噬長黑 → 甩轎洗盤 → 重新進場

4. 高檔出現5面紅旗 → 不看防守線，直接出場

心得

主力出貨手法的學習，讓我對「為什麼散戶總是接到最後一棒」有了更深的理解。幾個核心體悟：

1. 洗盤和出貨的判斷，位置是唯一真理——在低檔的下跌，要勇敢守住；在高檔的下跌，要果斷出場。這句話看似簡單，但實戰中散戶最容易犯的錯就是「高檔把出貨當洗盤，低檔把洗盤當出貨」。防守線系統就是幫你量化這個判斷：跌破第一防守線先出場，長紅吞噬再回來，寧可少賺也不要大賠。

2. 主力出貨不是一次完成的——從拉高出貨到震盪出貨到打壓出貨到反彈出貨，是一個遞進過程。散戶最常被騙的是「解套型出貨」：跌深反彈超過1/2，7成的人以為多頭還在，結果剛好是主力最後的出貨機會。這種反彈只適合短多，嚴設停損。

3. 「萬般拉抬只為出貨」這六個字要刻在腦子裡——主力作價六部曲的最後一步就是嘎空兼出貨，利多不斷、連續長紅、媒體一致看好的時候，就是最危險的時候。散戶在嘎空階段最興奮，但那正是主力在最後倒貨的時刻。記住：當所有人都說會再漲的時候，就是該跑的時候。

籌碼面分析：期貨正逆價差×外資期貨部位判讀×價差收斂策略

期貨與現貨之間的價差，是市場情緒最真實的溫度計。而外資在台指期的部位，更是全球散戶最幸運的公開數據——台灣是少數每日公布法人多空部位的國家。搞懂正逆價差的本質、外資期貨部位的贏家邏輯、以及價差收斂的實戰策略，是籌碼面分析不可或缺的一塊拼圖。

一、正價差與逆價差

是什麼

價差 = 期貨價格 − 現貨價格

  • 正價差（Contango）：期貨 > 現貨，價差為正
  • 逆價差（Backwardation）：期貨 < 現貨，價差為負

例如：大盤指數 12,850，台指期 12,800 → 逆價差 50 點。台指期 12,900，大盤 12,850 → 正價差 50 點。

為什麼會產生價差

1. 持有成本理論：持有現貨有利息、倉儲等成本，期貨需補償賣方這些成本，所以理論上期貨略高於現貨（正價差是常態）
2. 市場情緒：多頭市場投資人搶買期貨→正價差擴大；空頭市場避險資金湧入期貨放空→逆價差擴大
3. 除權息效應：6-8月除權息旺季，期貨提前反應除息扣點，形成「假性逆價差」。例如現貨 10,000 點，預期除息 200 點，期貨可能直接報 9,800 點→逆價差 200 點。這不是看空，是數學必然

怎麼判讀

關鍵原則：台指期約 70% 時間處於逆價差，這是常態不是看空訊號！

1. 先判斷常態範圍：查詢該商品歷史價差的平均值和標準差，只有「遠超常態」的價差才有分析價值
2. 看變化趨勢而非絕對數字：

  • 指數下跌 + 逆價差由大縮小 → 期貨抗跌，行情可能止跌反轉

3. 扣除除息因素：6-8月的逆價差要扣除除息點數才是「真實逆價差」
4. 結算日必然收斂：期貨最後結算價 = 現貨最後30分鐘均價，價差歸零

實例

  • 除息季假性逆價差：2024年7月，加權指數 24,000，台指期 23,600，逆價差 400 點。但當年除息預估扣 350 點，真實逆價差僅 50 點→市場其實偏中性
  • 恐慌性逆價差：2008年金融海嘯，逆價差瞬間拉大到 200-300 點（已扣息），代表市場極度恐慌，避險資金湧入
  • 過熱正價差：多頭末端正價差異常擴大→市場亢奮過度，法人開始賣期貨買現貨套利

遠月合約的逆價差

越遠月份的合約，逆價差越大，這是常態。例如當月 12,976，隔月 12,951，3月後 12,883，9月後 12,358。這不代表遠月看空，而是除息和持有成本的累積效應。

1. 看淨額變化比絕對值重要：外資從淨空單1萬口轉為淨多單1萬口→方向翻多；但1萬口多單本身不能判斷強弱

1. 外資操作工具多元：台指期、摩台指、選擇權、借券，只看台指期會漏掉大量資訊
2. 資訊時間差：期貨部位下午3點公布，散戶最快隔天開盤跟單→進場比外資慢一天，效益大約只剩外資的一半
3. 資金要充足：外資可以滿倉4萬口多單熬過2週17%的下跌，散戶做不到→2008年10月外資4萬口多喉，指數2週從4800跌到3900，帳面虧損幾乎賠掉2年獲利
4. 2011年後效果衰退：台指期外資部位2011年前非常準確，2012年後效果下降，可能因外資玩法改變或市場量能衰退

實例

三、價差收斂策略

是什麼

利用「結算日價差必然歸零」的特性，在價差異常大時建立部位，賺取價差收斂的利潤。

怎麼做

策略一：逆價差長線做多
1. 觀察台指期逆價差是否遠超常態（扣除除息因素後仍異常大）
2. 在遠月合約建立多單→成本比現貨便宜
3. 即使現貨不漲，價差收斂本身就是利潤
4. 適用情境：非空頭市場，行情盤整或上漲

策略二：正價差套利（法人專用）
1. 當正價差遠超持有成本
2. 賣出期貨 + 買入等值現貨
3. 結算時價差收斂，賺取超額價差
4. 散戶資金門檻較高，但理解邏輯可避免追高殺低

策略三：價差背離預判反轉
1. 指數下跌 + 逆價差由大縮小→期貨抗跌→行情即將止跌反轉
2. 指數創高 + 正價差持續縮小→多頭力竭→行情即將反轉
3. 這是「價差背離」訊號，比單純看正逆價差更有預測力

實例

  • 除息季逆價差布局：7月除息季逆價差300點，扣除除息200點後真實逆價差100點→在遠月合約做多，成本比現貨低100點，即使現貨盤整，結算收斂即可賺100點×200元=2萬元/口
  • 恐慌逆價差反轉：2020年3月COVID恐慌，逆價差急遽擴大到200+點，但隨後期貨開始抗跌（逆價差縮小）→這是止跌反轉的先行訊號

適用時機與注意事項

  • 適用：非空頭市場的逆價差布局、價差背離預判反轉、除息季真實逆價差計算
  • 不適用：空頭市場逆價差可能持續擴大（趨勢 > 收斂）、近月合約即將結算（時間太短）
  • 風險：做空者換月時會放空在更低價格，獲利減少；逆價差看似便宜但市場可能已反映未來看法

四、正逆價差×外資部位×現貨三維驗證

整合判讀框架

情境期貨價差外資期貨部位外資現貨判斷
強烈偏多正價差擴大淨多單增加買超多頭趨勢確認

多頭末端正價差縮小淨多單減少買超縮小趨勢力竭
避險升溫逆價差擴大淨空單增加賣超空方主導

查詢工具

  • 期交所：
  • 財經M平方：價差曲線圖
  • 玩股網、鉅亨網：正逆價差即時數據
  • 券商APP：大部分都有內建價差計算功能

心得

正逆價差這東西，最常見的誤解就是「逆價差=看空、正價差=看多」，這觀念大錯特錯。台指期70%時間都在逆價差，如果逆價差就是看空，那台股早該崩盤了。關鍵是「扣除除息因素後的真實逆價差」以及「價差的變化趨勢」，絕對數字本身意義不大。

外資期貨部位更是散戶的寶藏——全球很少有國家這麼大方每天公布法人多空部位。但2011年後效果衰退是事實，外資工具從台指期擴展到摩台、選擇權、借券，只看台指期會被誤導。最實用的還是「三維驗證」：期貨價差+外資期貨部位+外資現貨買賣超，三者同向才值得信賴。

是什麼

大盤融資餘額 = 整個台股市場所有投資人透過融資（向券商借錢）買入股票後，尚未償還的借款總金額。它反映了市場中「槓桿資金」的活躍程度，是衡量散戶集體情緒的關鍵指標。

四階段判讀法

市場階段指數位置融資餘額變化意涵
多頭初期低檔反彈溫和增加健康資金回流，聰明錢進場

1. 看變化趨勢，不看絕對數字：融資餘額連續5天增加或減少才有意義
2. 對照指數位置：同樣是融資增加，低檔是好事，高檔是壞事
3. 看增速而非總量：融資餘額「急遽暴增」比「溫和增加」更危險
4. 結合融資維持率：當整體維持率逼近140%-150%警戒區，斷頭賣壓隨時爆發

  • 確認趨勢持續性（指數漲但融資不增 = 虛胖行情）

二、融資維持率：比餘額更重要的壓力測試計

是什麼

整戶融資維持率 = (融資買入股票市值 / 融資金額) × 100%。衡量融資部位的健康狀況，低於130%觸發追繳。

怎麼做

1. 追繳門檻130%：維持率低於130%，券商發出追繳通知
2. 補繳期限T+2：必須在2個交易日內補繳保證金，將維持率拉回166%以上
3. 斷頭T+3：未補繳則第3個交易日開盤強制賣出
4. 市場系統性風險：整體維持率逼近140%-150%時，大盤若再重挫將引發連鎖斷頭

實例

  • 融資買進100元股票，自備款4萬、融資6萬
  • 維持率 = 100,000 / 60,000 = 166.67%（安全）
  • 股價跌到78元：維持率 = 78,000 / 60,000 = 130%（觸發追繳）
  • 股價跌到70元：維持率 = 70,000 / 60,000 = 116.67%（低於130%，斷頭）

迷思一：融資使用者全是散戶？

錯！ 主力、公司派、對沖基金也用融資鎖碼。個股融資暴增未必是散戶追高，可能是主力積極布局。

迷思二：融資增加就要賣？

錯！ 多頭趨勢中融資溫和增加是正常現象。過早賣出會錯失大段漲幅。關鍵看「增加速度」是否與指數漲幅脫鉤。

四、大小外資期貨部位拆解

是什麼

外資是一個「群體」而非個體，裡面包含幾十個機構。把外資拆成「大外資」和「小外資」，能看到不同時間框架的市場訊號。

拆解方法

小外資 = 外資總部位 − 五大特定法人部位

五大特定法人是市場規模最大的五位法人（通常就是最大的五家外資），扣除後剩下的小外資反應更靈敏。

怎麼做

1. 大外資看中期趨勢：部位大、調整慢，持倉方向代表中期看法
2. 小外資看短線動能：部位小、調整快，轉向比大外資更靈敏
3. 雙外資共振最可靠：大外資+小外資同向偏多 → 行情偏多；同向偏空 → 行情偏空
4. 大小外資背離要小心：大外資偏多但小外資翻空 → 短線有壓力；大外資偏空但小外資翻多 → 短線有支撐

  • 但拆解後：五大外資（大外資）其實偏多，空單來自小外資
  • 小外資靈敏翻空是短期避險，大外資中期偏多格局未變
  • 結果：短期回檔後行情繼續上漲，小外資被軋空

  • 判斷外資空單增加是「真看空」還是「避險/短線調整」
  • 確認行情持續性（大小外資同向 = 高確定性）
  • 短線擇時（小外資轉向領先大外資）

不適用時機

  • 單獨看外資總部位做決策（大小外資可能方向相反）
  • 過度解讀單日變化（至少觀察3-5天趨勢）

1. 先看現貨方向：現貨才是真金白銀，期貨可能是避險
2. 期貨空單+現貨買超 = 避險非看空：外資持有大量現貨部位，用期貨空單保護下檔風險
