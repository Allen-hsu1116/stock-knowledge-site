法說會是什麼？

是什麼：法人說明會（Earnings Call），公司向法人投資機構說明營運狀況與未來展望的會議。台股一年至少一次，美股每季一次。

與股東會的差異：

  • 法說會：法人限定，散戶只能觀看、不能提問
  • 股東會：只要是股東就能參加

法說會三大觀察重點

1. 管理團隊回答態度

是什麼：觀察公司高層回答問題的風格，判斷真實意圖。

怎麼做：
1. 先了解該公司管理團隊的「慣性風格」（保守 vs 樂觀）
2. 比較本次回答與平時風格的差異
3. 交叉驗證：口頭說法 vs 實際資本支出

判斷邏輯：

  • 保守團隊突然樂觀 → 真實情況可能比說的更好
  • 樂觀團隊突然保守 → 真實情況可能比說的更差

實例：

  • 公司說「前景看好」，但資本支出預算大幅刪減 → 小心，可能是假樂觀
  • 公司說「保守看待」，但正在積極擴廠、增加研發 → 可能是藏拙

適用時機：長期追蹤特定公司的投資人，累積足夠的觀察紀錄後最有用。

2. 新產品資訊與產業消息

是什麼：法說會常會揭露新產品、新訂單、新市場等利多消息。

怎麼做：
1. 記錄新產品/新訂單的具體內容
2. 評估對營收/獲利的貢獻時間點與金額
3. 比對市場原本預期：超出預期還是低於預期？

判斷標準：

  • 超出預期 → 利多，股價可能上漲
  • 符合預期 → 利多出盡，可能不漲反跌
  • 低於預期 → 利空，股價可能下跌

實例：

  • 台積電法說會宣布 2026 年營收看增近 30%（超出預期）→ 股價上漲
  • 大立光 2016 Q2 法說會毛利率衝上 66.37%，隔天股價漲停（大漲 310 元）

適用時機：短線投資人可在法說會前佈局，長線投資人用來調整持股策略。

3. 時事話題看法

是什麼：分析師會針對當下熱門議題提問（疫情、戰爭、原物料上漲等）。

怎麼做：
1. 記錄公司對重大事件的應對策略
2. 評估衝擊程度：一次性還是持續性？
3. 觀察公司是否有具體應對方案

實例：

  • 疫情期間：公司如何應對供應鏈中斷？有無備援方案？
  • 記憶體價格大漲：公司如何控管成本？能否轉嫁給客戶？

適用時機：重大事件發生時，判斷公司抗風險能力。

法說會流程（美股）

時間：約 1 小時

三大階段：
1. 安全港聲明（Safe Harbor）：免責聲明，說明預測可能與實際結果有差異
2. 財報報告（約 10 分鐘）：上一季財務狀況 + 下一季展望
3. Q&A 時間（重頭戲）：分析師提問，公司回答

台股 vs 美股差異：

  • 美股：每季一次（配合季財報）
  • 台股：一年至少一次

法說會對股價的影響

時機點：

  • 法說會通常在盤後舉行
  • 利多/利空消息在「隔天開盤」反應在股價上

短線操作策略：
1. 法說會當天觀察股價走勢
2. 強勢 → 可能有利多，可考慮買進認購權證
3. 弱勢 → 可能有利空，可考慮買進認售權證
4. 隔天開盤後視情況獲利了結

實例數據（大立光）：

  • 法說會隔日上漲機率：不高
  • 但盤中波動幅度大：平均上漲 3.3%，下跌 -3.1%
  • 適合用權證做當沖套利

長線投資策略：

  • 關注公司經營方向、長期展望
  • 驗證基本面判斷是否正確
  • 調整持股或加減碼

法說會解讀三大心法

心法一：聽其言，觀其行

是什麼：不要只聽公司說什麼，要看公司做什麼。

怎麼做：
1. 記錄法說會宣示的目標（營收成長、擴廠等）
2. 下一季法說會檢視達成狀況
3. 長期追蹤：說到做到的公司值得信賴

紅旗警訊：

  • 公司一直說「看好未來」，但資本支出持續縮減
  • 公司說「沒問題」，但高層持續賣股

心法二：比較預期與實際

是什麼：股價反應的是「預期落差」，不是消息本身。

怎麼做：
1. 法說會前先了解市場預期（營收、毛利率、EPS 等）
2. 法說會後比較：實際公布 vs 市場預期
3. 超出預期 → 利多；低於預期 → 利空

實例：

  • 公司宣布獲利成長 20%，但市場預期成長 30% → 利空，股價可能下跌
  • 公司宣布獲利下滑 10%，但市場預期下滑 20% → 利多，股價可能上漲

心法三：關注 Q&A 中的「意外」

是什麼：真正有價值的資訊常藏在 Q&A 環節。

怎麼做：
1. 觀察分析師問了什麼「尖銳問題」
2. 注意公司如何「迴避」或「轉移」焦點
3. 迴避的問題 = 可能是公司的痛點

實例：

  • 分析師追問某產品線獲利，公司一直轉移話題 → 該產品線可能有問題
  • 分析師問庫存問題，公司說「一切正常」但存貨周轉天數持續上升 → 小心

心得

法說會是「聽公司說真話還是場面話」的最佳時機。

重點不是公司說了什麼，而是：
1. 說法和做法是否一致：口頭樂觀 but 沒有擴廠計畫 = 假樂觀
2. 預期落差：超預期利多，不如預期利空，市場反應的是「驚喜」
3. Q&A 的迴避：分析師追問但公司閃躲的，通常是有問題的地方

短線投資人可以用法說會前後的波動賺價差，但對長線投資人來說，法說會是「驗證基本面判斷」的關鍵時刻。持續追蹤同一家公司的法說會，就能累積對管理團隊的判斷力。

最後提醒：法說會是工具，不是聖旨。公司有義務維持市場信心，投資人有義務保持懷疑。
