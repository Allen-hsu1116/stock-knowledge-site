核心知識

現金殖利率基本概念

是什麼：現金股利除以股價的比值，用來衡量投入資金預期每年能拿回的股息比例。

公式：

現金殖利率(%) = 現金股利(元) / 股價(元)

實例：

  • 台積電股價 100 元，配息 3 元 → 殖利率 3%
  • 殖利率 4% 意思：用 100 萬買進，預期每年領到 4 萬元股息

適用時機：僅適用於「盈餘穩定、配息穩定」的成熟企業，不適用於成長股或衰退股。

高殖利率股篩選 4 指標（CMoney）

篩選條件：
1. 現金殖利率 > 4%
2. 近 3 年現金股利不曾衰退
3. 負債比 < 40%
4. 速動比 > 200%

為什麼要這些條件：

  • 殖利率 > 4%：高於台股平均（約 3.74%）
  • 配息不衰退：確認公司獲利穩定，不是一次性配息
  • 負債比 < 40%：財務安全邊際
  • 速動比 > 200%：短期償債能力充足

實例：

  • 某公司殖利率 8%、負債比 60%、速動比 80% → 財務體質差，高殖利率可能是陷阱
  • 某公司殖利率 5%、近 3 年配息穩定、負債比 30% → 值得深入研究

殖利率 4 大迷思（市場先生）

迷思 1：殖利率 = 報酬率

真相：配息時股價會除息，總資產不變。

  • 領 3 萬股息 → 股價下跌 3 萬元價值
  • 真正報酬來自股價上漲，不是股息

迷思 2：過去配息 = 未來配息

真相：殖利率是用過去股利計算，未來不一定相同。

  • 某公司去年配 5 元，今年可能只配 2 元
  • 必須確認公司盈餘穩定

迷思 3：配息越高越好

真相：高配息可能代表公司無法有效運用資金。

  • 成長股（Google、Amazon）殖利率是 0%，因為把錢拿去再投資
  • 巴菲特波克夏殖利率 0%，因為資金在他手上效率更高

迷思 4：高殖利率 = 好公司

真相：高殖利率可能是陷阱：

  • 成長性公司 → 股價高、殖利率低（正常）
  • 成熟公司 → 股價合理、殖利率高（正常）
  • 衰退公司 → 股價跌、殖利率暴增（陷阱！）

高殖利率 3 大陷阱

陷阱 1：賺股利賠價差

特徵：

  • 公司衰退中，股價持續下跌
  • 殖利率看起來很高（因為分母下跌）
  • EPS 不穩定、營收衰退、毛利率下降

避開方法：檢查 EPS 是否穩定、營收是否衰退、毛利率是否下降

陷阱 2：一次性配息

特徵：

  • 公司賣土地、賣子公司
  • 當期 EPS 暴增、業外收入大增
  • 明年沒有這種好事

避開方法：

  • 檢查 EPS 是否突然暴增
  • 檢查業外收入佔比是否異常

陷阱 3：增資配息

特徵：

  • 公司辦現金增資跟投資人要錢
  • 然後配息給投資人
  • 配息率 > 100%（配得比賺得多）

避開方法：

  • 配息率 = 現金股利 / EPS，若超過 100% 要小心
  • 檢查營業現金流、自由現金流是否為正值

殖利率高低判斷標準

與大盤比較：

  • 台股（0050）殖利率歷史區間：2.2% ~ 3.8%
  • 美股（S&P500）殖利率歷史區間：1.3% ~ 2.2%
  • 高於平均 → 稱為高殖利率

與同產業比較：

  • 同產業公司互相比較
  • 同類型公司（如金融股、電信股）互相比較

重要觀念：

  • 殖利率高不一定好，可能隱藏風險
  • 殖利率低不一定差，可能是成長股

實際應用流程

Step 1：篩選候選股

  • 殖利率 > 4%
  • 近 3 年配息穩定或成長
  • 負債比 < 40%
  • 速動比 > 200%

Step 2：檢查是否為陷阱

  • EPS 是否穩定？
  • 營收是否衰退？
  • 毛利率、營業利益率是否下滑？
  • 業外收入是否暴增？
  • 配息率是否 > 100%？

Step 3：評估公司體質

  • 營業現金流是否為正？
  • 自由現金流是否為正？
  • 公司是否處於成熟穩定期？

Step 4：決策

  • 體質好、配息穩定 → 可納入投資組合
  • 體質差、配息不穩 → 避開

心得

殖利率這東西，表面看起來很簡單，實際上陷阱一堆。

最重要的觀念：殖利率不是報酬率。很多人以為殖利率 5% 就是每年賺 5%，錯。配息時股價會除息，左手換右手而已。真正要賺錢，得靠股價上漲或公司長期成長。

另一個關鍵：高殖利率可能是衰退股。股價跌 → 分母變小 → 殖利率變高。這種公司往往是夕陽產業或經營出問題，領了股利賠了價差。

所以篩選條件很重要：

  • 殖利率 > 4% 只是門檻
  • 配息穩定才是關鍵
  • 財務體質要健康（負債比、速動比）
  • 要排除一次性配息、增資配息的假高殖利率

最後，別忘了高成長股通常殖利率很低甚至是 0%，因為錢都拿去再投資了。如果只看殖利率選股，會錯過所有高成長的好公司。

一句話總結：殖利率只能用來評估「成熟穩定配息」的公司，其他都是陷阱。
