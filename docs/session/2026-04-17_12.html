選擇權避險三策略：Protective Put、Collar、Delta Neutral

一、保護性賣權（Protective Put）——為持股買保險

是什麼：持有現股的同時，買入相同標的的賣權（Put），為持股建立下跌保護。就像幫房子買地震險——平常每年繳保費，地震來了理賠，沒地震就當花錢買心安。

公式：
> Protective Put = 持有股票 + 買入 Put

怎麼做：

1. 確定保護需求：

  • 你持有多少股票？每 100 股需要 1 張 Put

2. 選擇 Put 履約價：

  • 平價 Put（ATM）：履約價＝現價，保護最完整但最貴

3. 計算保護成本：

  • 每季保護成本 ≈ 持股市值的 1.5-3%（ATM）或 0.5-1.5%（OTM 5%）

4. 到期決策：

  • 股價上漲：Put 過期作廢，損失權利金（保險費），但股票獲利不受限

實例：

  • 情境：持有 1 張台積電（2330），成本 800 元，現價 900 元
  • 保護：買入 1 張履約價 850 的 Put（價外 5.6%），權利金 15 點
  • 成本：15 × 2,000 = 30,000 元（約持股市值的 1.7%）
  • 最大虧損鎖定：900 - 850 + 15 = 65 點（約 7.2%）
  • 如果台積電跌到 700：沒有 Put 虧 200 點（22%），有 Put 只虧 65 點（7.2%）
  • 如果台積電漲到 1000：Put 過期，淨獲利 100 - 15 = 85 點

適用時機：

  • ✅ 財報公布前、重大事件前（選舉、升息）
  • ✅ 長期持有不想賣但擔心短期風險
  • ✅ 帳面獲利豐厚想鎖定利潤
  • ❌ 長期持續買：保險費會吃掉利潤（年化 2-6%）
  • ❌ 已經套牢想靠 Put 回本：Put 只能止損不能回本

二、Collar 領口策略——零成本避險

是什麼：持有現股 + 買入 Put（保護下跌）+ 賣出 Call（放棄上漲超過某個價位的利潤）。用賣 Call 收到的權利金來支付買 Put 的成本，達到「零成本」或「極低成本」的避險效果。

公式：
> Collar = 持有股票 + 買入 Put（下方保護）+ 賣出 Call（上方封頂）

怎麼做：

1. 設定保護區間：

  • 下方保護：選擇 Put 履約價（通常價外 5-10%）

2. 建構零成本 Collar：

  • 賣出 Call 收到的權利金 ≈ 買入 Put 付出的權利金

3. 到期損益計算：

  • 最大虧損 = 買入價 - Put 履約價 + 淨權利金成本

實例（台積電 Collar）：

  • 持有 1 張台積電，現價 900 元
  • 買入 850 Put（保護 5.6% 下方），權利金 15 點，成本 30,000 元
  • 賣出 950 Call（封頂 5.6% 上方），權利金 14 點，收入 28,000 元
  • 淨成本 = 30,000 - 28,000 = 2,000 元（接近零成本）

三種到期情境：

  • 跌到 800：Put 發揮保護，你以 850 賣出，最大虧損 = (900-850)×2000 + 2000 = 102,000（約 5.7%）
  • 漲到 1000：Call 被行使，你以 950 賣出，最大獲利 = (950-900)×2000 - 2000 = 98,000（約 5.4%）
  • 維持 900 附近：兩者都過期，淨成本 2,000 元，股票仍在手上

Collar 的動態調整（Rolling）：

  • 股價大漲逼近 Call 履約價 → Roll Up：買回原 Call 認賠，賣出更高履約價的 Call
  • 股價大跌觸及 Put 履約價 → 可以賣出 Put 獲利了結，重新買入更低履約價的 Put
  • 到期前調整可以保留股票持有權，但會增加交易成本

適用時機：

  • ✅ 長期持有大量股票不想賣（稅務考量）
  • ✅ 度假期間無法盯盤
  • ✅ 重大不確定事件前（選舉、升息、財報）
  • ❌ 看好大幅上漲空間（Collar 會封頂獲利）
  • ❌ 短期交易（Collar 適合 1-3 個月的保護期）

三、Delta Neutral 中性避險——消除方向風險

是什麼：透過調整多空部位，讓投資組合的總 Delta 值接近零，使組合價值不受標的價格小幅波動的影響。賺的不是方向，而是波動率或時間價值。

核心概念：

  • Delta = 選擇權價格對標的價格變動的敏感度
  • Call 的 Delta 介於 0 到 +1 之間
  • Put 的 Delta 介於 -1 到 0 之間
  • Delta Neutral = 組合總 Delta ≈ 0

怎麼做：

1. 計算組合總 Delta：

  • 總 Delta = Σ（每個部位的 Delta × 部位數量 × 乘數）

2. 建立對沖部位：

  • 如果總 Delta > 0（偏多）→ 賣出期貨或買入 Put 來對沖

3. 動態再平衡：

  • Delta 會隨標的價格變動而改變（Gamma 效應）

實例（台指期貨 + 選擇權 Delta Neutral）：

  • 持有 1 口小台指多單（Delta = 1，每點 50 元）
  • 買入 2 口價平 Put（Delta = -0.5 各）
  • 組合 Delta = 1 + (-0.5 × 2) = 0
  • 台指漲 100 點：小台賺 5,000，Put 虧約 5,000 → 損益接近零
  • 台指跌 100 點：小台虧 5,000，Put 賺約 5,000 → 損益接近零
  • 但波動率上升時：Put 的 Vega 正值讓整體獲利

Delta Neutral 的三種獲利來源：

1. Vega 交易（波動率）：

  • 預期 IV 上升 → 買入 Straddle/Strangle + 對沖 Delta

2. Theta 交易（時間價值）：

  • 賣出跨式/勒式 + 動態對沖 Delta

3. Gamma Scalping：

  • 買入 ATM Straddle（Gamma 正值）

Delta Neutral 的四大限制：

1. 調整頻繁：Delta 不是固定的，隨價格和時間變化，需要不斷再平衡
2. 交易成本：每次調整都有手續費和滑價，頻繁對沖會侵蝕利潤
3. Gamma 風險：價格劇烈波動時 Delta 快速變化，對沖可能瞬間失效
4. 忽視長期目標：過度專注短期 Delta 調整，可能忽略長期趨勢判斷

適用時機：

  • ✅ 大型部位需要避險（法人、大戶）
  • ✅ 波動率交易（賺 IV 變化而非方向）
  • ✅ 高波動期保護既有部位
  • ❌ 散戶小部位：交易成本太高，不划算
  • ❌ 不了解 Greeks 的人：Delta Neutral 只是表面中性，Gamma 和 Vega 風險仍在

五、散戶避險策略選擇 SOP

情境一：持有大量現股，擔心短期大跌

  • 最佳選擇：Collar 策略（零成本、保護完整、封頂可接受）
  • 替代選擇：Protective Put（如果不想封頂上漲）
  • 不建議：Delta Neutral（散戶操作成本太高）

情境二：長期持有，偶爾需要短期保護

  • 最佳選擇：Protective Put（只在財報、選舉等不確定事件前買入）
  • 持有期間：1-2 週，事件過後賣出
  • 成本控制：只買價外 5-10% 的 Put，降低保險費

情境三：帳面獲利豐厚，想鎖定利潤但不賣股

  • 最佳選擇：Collar 策略（鎖住下方、放棄部分上方、零成本）
  • 特別適合：長期持有數年的股票，賣出會產生大量資本利得稅
  • 替代選擇：直接部分減碼（如果稅務不是考量）

情境四：想要從波動率賺錢

  • 最佳選擇：Delta Neutral + 波動率交易
  • 前提條件：深入了解 Greeks、有即時交易系統、交易成本極低
  • 散戶替代：買賣 VIX 相關 ETF（簡單但追蹤誤差大）

心得

選擇權避險是風險管理的進階武器，今天學了三種：Protective Put 是最直覺的「買保險」，Collar 是最精明的「免費保險」，Delta Neutral 是最專業的「消除方向風險」。

Protective Put 最容易理解：花錢買保險，你希望永遠用不到。但保險費是實實在在的成本——ATM Put 年化成本可能高達 6-12%，長期買下來你等於每年繳 6-12% 的「保護費」。所以它最適合短期事件避險，而不是長期持有。

Collar 最聰明的地方是「用別人的錢買保險」。賣出 Call 的權利金支付 Put 的成本，幾乎零成本就獲得了下方保護。代價是放棄上方超過 Call 履約價的利潤。這其實是一個很合理的取捨——如果你本來就打算在某个價位減碼，那為什麼不順便用它來支付保險費？Collar 最適合的情境是：你持有大量獲利股票，短期擔心回調，但長期不想賣出。

Delta Neutral 是法人專屬的精密武器。散戶最好不要碰——光是每日動態調整的交易成本就足以把利潤吃光。但理解 Delta Neutral 的概念很重要：它讓你知道「方向不是唯一可以賺的東西」，波動率和時間價值也是獲利來源。這個認知會改變你對選擇權的理解。

三種策略對散戶的建議很明確：Protective Put 用於短期事件避險，Collar 用於長期部位保護，Delta Neutral 留給法人和高頻交易者。 不要因為學了高級工具就想去用高級工具，簡單的工具用得好，遠比複雜的工具用不好更安全。
