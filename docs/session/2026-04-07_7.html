毛利率、營業利益率、淨利率選股實戰

第 7 回合學習 - 毛利率、營業利益率、淨利率選股實戰

文章：

  • 看懂3率 輕鬆找出賺錢好公司 - 股魚
  • 三率三升：毛利率、營業利益率、淨利率 - Yahoo財經

核心知識

三率定義與計算

毛利率（GPM）

是什麼：營業收入扣除營業成本後的利潤占比，反映產品競爭力與成本控制能力。

公式：

毛利率 = (營業收入 - 營業成本) / 營業收入 × 100%

意義：

  • 毛利率越高 → 產品附加價值越高、成本控制越好
  • 毛利率較同業高 → 公司在產業中有競爭優勢
  • 代工業毛利率通常很低（毛三到四 = 3~4%），需用資產週轉率彌補

營業利益率（OPM）

是什麼：營業毛利扣除營業費用後的利潤占比，反映公司整體經營能力與本業獲利能力。

公式：

營業利益率 = 營業利益 / 營業收入 × 100%
營業利益 = 營業收入 - 營業成本 - 營業費用

營業費用包括：租金、水電、薪資、行銷費用、研發費用、折舊等

意義：

  • 營業利益率越高 → 經營管理能力越好
  • 營業利益率為負數 → 公司本業虧損，營運有困難
  • 這是最重要的指標，因為業外收入會干擾淨利率判斷

淨利率（NPM）

是什麼：稅後淨利占營收的比例，代表公司最終真正賺到的錢。

公式：

淨利率 = 稅後淨利 / 營業收入 × 100%
稅後淨利 = 營業收入 - 營業成本 - 營業費用 + 業外損益 - 稅金

業外損益包括：處分資產收益、轉投資收益、匯兌損益等

注意：

  • 淨利率突然飆高可能是賣土地、賣廠房等一次性收益
  • 需確認業外損益是否為經常性

三率的重要性排序

股魚觀點：營業利益率 > 淨利率 > 毛利率

為什麼營業利益率最重要？
1. 毛利率容易被調整（成本與費用之間移動）
2. 淨利率受業外干擾（賣地、匯兌等一次性收益）
3. 營業利益率最能反映「本業真實獲利能力」

實例對照：

情境毛利率營業利益率判讀
A電子公司40%+-81.6%毛利看起來好，但本業虧損嚴重
B瓦斯公司穩定穩定毛利與營益率差距小，體質穩健

三率三升選股實戰

是什麼：三率同時上升，代表公司營運全面改善。

怎麼判斷：
1. 毛利率上升 → 產品競爭力提升或成本下降
2. 營業利益率上升 → 經營效率提升
3. 淨利率上升 → 整體獲利能力增強

實例：鴻海 2023 Q3

  • 毛利率：6.16% → 6.66%（YoY 提升）
  • 營業利益率：2.78% → 2.99%（YoY 提升）
  • 淨利率：2.22% → 2.79%（YoY 提升）
  • 股價從跌破百元重返百元關卡

三率選股三步驟

步驟 1：基本門檻篩選

條件：

  • 毛利率、營業利益率、淨利率 都 > 0（至少不賠錢）
  • 本業收入占比 > 80%

步驟 2：連續性檢驗

檢查最近 5 年財報：

  • 每年本業經營、稅後淨利都不能虧損
  • 一個完整產業循環約 5 年，需驗證景氣好壞都能獲利

步驟 3：產業特性調整

不同產業毛利率特性：

產業類型毛利率特性分析重點
代工製造毛三到四（3-4%）不應嫌棄，看資產週轉率
IC 設計通常較高產品區隔度高，汰換頻繁
固定資產重較難提升看產能利用率
研發型企業通常較高看研發費用是否有效轉換

常見陷阱

陷阱 1：只看 EPS 成長

問題：淨利可能來自處分資產等一次性收益

實例：公司賣土地讓淨利大增，隔年回歸原形

對策：優先看營業利益率，確認本業獲利

陷阱 2：毛利率好看但營業利益率差

問題：成本與費用在報表上調整的結果

實例：A公司毛利率 40%+，但營業利益率 -81.6%

對策：三率一起看，毛利率與營業利益率的差距要注意

陷阱 3：忽略產業特性

問題：用統一標準比較不同產業

對策：

  • 代工業毛利率本來就低，要看週轉率
  • IC設計毛利率本來就高，要看能否維持
  • 同產業比較才公平

實戰操作步驟

怎麼做：

1. 查財報：上公開資訊觀測站或 Yahoo股市查損益表
2. 計算三率：

  • 毛利率 = (營業收入 - 營業成本) / 營業收入

3. 檢查本業占比：營業利益 / (營業利益 + 業外收入) > 80%
4. 看連續 5 年：確認沒有虧損年度
5. 對比同業：同產業比較才準

適用時機：

  • 選股初步篩選
  • 法報公布後檢查持股體質
  • 三率三升的轉機股
  • 本益比偏低時的成長股

EBITDA 指標補充

是什麼：稅前息前折舊攤銷前獲利

公式：

EBITDA = 稅後淨利 + 利息 + 稅 + 折舊與攤銷

用途：

  • 評估公司核心獲利能力
  • 排除資本結構、稅率、折舊政策差異
  • 適合比較資本支出重的產業（如電信、公用事業）

心得

這次學到最重要的一點：營業利益率比毛利率和淨利率更值得關注。

之前我都覺得毛利率高就是好公司，現在看來太天真了。毛利率高但營業利益率差，可能是成本與費用在報表上調整的結果，或者公司本業根本在賠錢。

股魚的「三步驟選股法」很實用：
1. 三率都 > 0 + 本業占比 > 80%
2. 連續 5 年不虧損
3. 同產業比較

另外學到一個重要觀念：代工業毛利率「毛三到四」不是問題，因為他們靠資產週轉率賺錢。這點我之前完全沒想到，只會嫌毛利率低。

「三率三升」是很好的篩選標準，三個指標同時上升代表公司營運全面改善。鴻海那個例子很清楚，Q3 三率三升後股價就重返百元。

以後看財報，優先看營業利益率的變化，再看本業收入占比，不要被 EPS 成長迷惑。業外收入可能是賣地賣廠房，一次性收益沒意義。
