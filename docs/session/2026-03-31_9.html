核心知識

PEG（本益成長比）- 成長股評估利器

是什麼：將本益比除以盈餘成長率，用來判斷成長股是否被低估或高估的指標。創始人為英國投資大師吉姆·史萊特，後被彼得·林區推廣。

怎麼算：

PEG = 本益比 ÷ 盈餘成長率（%）

例：股價 100 元，EPS 5 元，預估成長率 10%
→ 本益比 = 100 ÷ 5 = 20
→ PEG = 20 ÷ 10 = 2

判斷標準：

PEG解讀操作
< 0.66嚴重低估買進良機
0.66-0.75低估買進
≈ 1合理觀望
> 1.2高估考慮賣出

實例：

  • NVIDIA 本益比 50，成長率 40% → PEG = 50÷40 = 1.25（合理）
  • 可口可樂 本益比 25，成長率 5% → PEG = 25÷5 = 5（高估）

三大注意事項：
1. PEG 不是越小越好：可能選到獲利起伏太大的公司
2. 只用於成長股：必須是 EPS 成長的公司
3. 成長率預估要保守：企業成長不可能永遠相同

ROE（股東權益報酬率）- 巴菲特最愛

是什麼：公司用股東的錢賺錢的效率。巴菲特多次強調這是他最看重的指標之一。

怎麼算：

ROE = 稅後淨利 ÷ 股東權益

例：A 公司資本額 100 萬，獲利 20 萬 → ROE = 20%
例：B 公司資本額 1000 萬，獲利 50 萬 → ROE = 5%

判斷標準：

ROE解讀
< 5%資金使用效率低
5%-10%普通，穩定型企業
10%-15%中等偏上，具吸引力
> 15%高效率，需確認可持續性

產業基準：

  • 科技/軟體：15%-30%（輕資本，獲利強）
  • 傳統製造：5%-12%（重資本，毛利低）
  • 金融業：8%-15%（高槓桿，風控重要）
  • 公用事業：5%-10%（現金穩，成長慢）

ROE 高不代表好：
1. 可能來自一次性獲利（賣資產）
2. 可能靠高槓桿堆出來（高負債）
3. 可能成長性已停滯（成熟企業）

EPS vs ROE：

  • EPS = 稅後淨利 ÷ 流通股數（每股獲利能力）
  • ROE = 稅後淨利 ÷ 股東權益（股東資金效率）
  • EPS 高但 ROE 低 = 公司賺錢沒效率

ROA（資產報酬率）- ROE 的互補指標

是什麼：公司運用所有資產創造獲利的能力，包含負債。

怎麼算：

ROA = 稅後淨利 ÷ 總資產
    = 稅後淨利 ÷（股東權益 + 負債）

進階公式（更精準）：

ROA =［稅後淨利 + 利息×（1-稅率）］÷ 總資產

利息有抵稅效果，這公式能更精準計算。

ROE vs ROA 差異：

指標公式衡量適用產業
ROE淨利÷股東權益股東資金效率一般產業
ROA淨利÷總資產整體資產效率高負債產業

ROA 合理值：一般 6%-7% 為合理水準

什麼情況用 ROA：

  • 負債比率高的產業（金融、重工、航運）
  • 想知道公司「舉債效率」
  • ROE 和 ROA 差距大 = 靠槓桿

杜邦分析 - 拆解 ROE 三因子

是什麼：將 ROE 拆解成三個因子，深入分析公司獲利來源。

公式：

ROE = 淨利率 × 資產周轉率 × 槓桿比率
    =（淨利/營收）×（營收/總資產）×（總資產/股東權益）

三因子解析：
1. 淨利率（Profit Margin）：獲利能力

  • 高淨利率 = 成本控制好、定價能力強

2. 資產周轉率（Asset Turnover）：營運效率

  • 高周轉率 = 資產運用效率高

3. 槓桿比率（Leverage）：財務結構

  • 高槓桿 = 靠舉債放大報酬

杜邦分析實戰應用：

  • ROE 高 + 淨利率高 = 產品有競爭力（好）
  • ROE 高 + 槓桿高 = 靠借錢堆出來（風險）
  • ROE 高 + 周轉率高 = 營運效率好（佳）
  • ROE 低 + 淨利率低 + 槓桿高 = 問題公司

投資決策應用：

  • 巴菲特偏好：高淨利率 + 穩定 ROE
  • 避開：高槓桿推高的 ROE
  • 最佳組合：淨利率↑ + 周轉率↑ + 槓桿合理

實戰選股五步驟

Step 1：看長期 ROE

  • 連續 5-10 年 ROE > 15%
  • 觀察趨勢是否穩定或上升

Step 2：用杜邦分析拆解

  • 確認 ROE 來源是否健康
  • 避開靠高槓桿的公司

Step 3：用 PEG 判斷估值

  • PEG < 1 = 被低估
  • 搭配本益比一起看

Step 4：ROE 和 ROA 比對

  • ROE > ROA 太多 = 槓桿高
  • 高負債產業看 ROA 更準

Step 5：產業橫向比較

  • 與同業 3-5 家比較
  • ROE > 產業平均 = 有優勢

心得

基本面分析不是看單一指標就好，而是要看「指標背後的意義」。

PEG 教會我：高本益比不代表貴，要看成長性。NVIDIA 本益比 50，但成長率 40%，PEG 1.25 還算合理。這解釋了為什麼成長股本益比一直很高但股價還能漲。

ROE 教會我：巴菲特看重的不是「公司賺多少」，而是「公司用股東的錢賺多少效率」。兩家公司賺一樣的錢，資本小的那家更厲害。但 ROE 高要注意來源——是經營好還是借錢借出來的？

杜邦分析教會我：拆解 ROE 才能看到公司真正的實力。淨利率高代表產品強，周轉率高代表管理好，槓桿高代表風險大。同樣 20% ROE，來源不同、風險差很多。

ROA vs ROE 教會我：高負債產業要用 ROA 看，因為 ROE 會被槓桿扭曲。金融、航運這種負債比高的，ROA 比 ROE 更能反映真實效率。

最後，這些指標都要「長期看」+「跟同業比」。單一年度的數字可能被操弄或異常，看 5-10 年趨勢才靠譜。
