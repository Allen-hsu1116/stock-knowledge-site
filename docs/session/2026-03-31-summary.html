1. 本益比（P/E）估值實戰

核心概念：衡量「多久能回本」的估值指標。股價除以每股盈餘（EPS），代表買進後需要多少年才能賺回本金。

公式：

本益比 = 股價 ÷ 每股盈餘（EPS）

三種判斷標準：

方法怎麼做注意事項
縱向比較法與公司歷史本益比區間比較商業模式要穩定
橫向比較法與產業及同業比較龍頭享有溢價
PEG Ratio本益比 ÷ 預估EPS成長率PEG < 1 低估，PEG > 1.2 高估

三大陷阱：
1. 低本益比 ≠ 便宜：景氣循環股高峰時本益比最低，但往往是賣點
2. 高本益比 ≠ 昂貴：高成長預期可支撐高估值
3. 虧損公司無法計算：改用PBR、PSR、殖利率

實例計算：

  • 台積電股價1035元，近4季EPS合計50.51元
  • 本益比 = 1035 ÷ 50.51 = 20.49倍
  • 意義：假設未來EPS不變，需要20.49年回本

2. 本益成長比（PEG）選股法

公式：

PEG = 本益比 ÷ 盈餘成長率（%）

判斷標準：

PEG狀態操作
< 0.66大幅低估買進良機
0.66-0.75低估可買進
≈ 1合理觀望
> 1.2高估考慮賣出

實例比較：

  • A公司：本益比30倍，成長率35% → PEG = 30 ÷ 35 = 0.86（有吸引力）
  • B公司：本益比20倍，成長率15% → PEG = 20 ÷ 15 = 1.33（可能高估）

吉姆·史萊特原則：

  • 淨利成長率15% → 合理本益比15倍
  • 淨利成長率20% → 合理本益比20倍
  • 淨利成長率30% → 合理本益比30倍

三大注意事項：
1. PEG不是越小越好（可能選到獲利起伏太大的公司）
2. 只適用於成長股（EPS必須成長）
3. 成長率預估要保守（用長期數據，不用單一年度）

3. ROE、ROA 與杜邦分析

ROE（股東權益報酬率）：

ROE = 稅後淨利 ÷ 股東權益

判斷標準：

ROE解讀
< 5%資金使用效率低
5%-10%普通，穩定型企業
10%-15%中等偏上，具吸引力
> 15%高效率，需確認可持續性

ROE四大陷阱：
1. 一次性獲利推高ROE：檢查業外損益
2. 財務槓桿提高ROE：對比ROA，差距大=靠槓桿
3. 股東權益下降造成假象：檢查股東權益變動表
4. 保留盈餘增加但ROE下降：公司資金運用效率降低

ROA（資產報酬率）：

ROA = 稅後淨利 ÷ 總資產
  • 適合評估高負債產業（金融、航運）
  • 合理值：6%-7%

杜邦分析三因子：

ROE = 淨利率 × 資產週轉率 × 權益乘數
    = (淨利/營收) × (營收/總資產) × (總資產/股東權益)
因子意義高代表什麼
淨利率獲利能力產品有競爭力、成本控制好
資產週轉率營運效率資產運用效率高
權益乘數財務槓桿舉債經營，風險高

實戰判斷：

  • ROE高 + 淨利率高 = 產品有競爭力（好）
  • ROE高 + 槓桿高 = 靠借錢堆出來（風險）
  • ROE高 + 週轉率高 = 營運效率好（佳）

5. 自由現金流（FCF）選股

公式：

自由現金流 = 營業現金流 - 資本支出

巴菲特名言：「我們偏愛那些產生現金，而非消化現金的公司。」

FCF正負數意義：

情況解讀
FCF為正（營業現金流 > 投資支出）企業真正賺到現金，可發股息、還債、投資
FCF為正（賣資產讓FCF轉正）變賣資產非長久之計，要警覺
FCF為負（擴廠、買設備）未來可能帶來更多收入，不一定是壞事
FCF為負（長期負數）企業過度燒錢，財務危機風險

EPS增長但FCF下降四大警訊：

情況原因觀察重點
瘋狂投資擴張CapEx大增FCF + 投資現金流是否成長
庫存堆積現金卡在存貨庫存是否增加但銷量沒跟上
應收帳款問題客戶延遲付款應收帳款是否增加
供應商要求更快付款現金流出加快應付帳款變化

6. 波特五力分析

五力框架：

力量定義高度威脅
現有競爭者強度產業內同業競爭競爭者多、規模相近、產品差異低
潛在進入者威脅新競爭者進入可能性資本門檻低、技術要求不高
替代品威脅不同產品滿足相同需求替代品價格性能比佳、轉換成本低
買方議價能力客戶議價能力買方集中、單一買方採購量大、產品標準化
供應商議價能力供應商議價能力供應商集中、供應產品獨特、轉換成本高

五力評分法：

  • 1-2分：該力量弱，對企業有利
  • 3分：該力量中等
  • 4-5分：該力量強，對企業不利

7. DCF現金流量折現法

核心概念：把公司未來產生的自由現金流，全部折現到現在的價值總和。

公式：

DCF = Σ (未來n期現金流 ÷ (1+折現率)^n年)

三大要素：
1. 自由現金流（CF）：公司真正可自由運用的資金
2. 折現率（r）：通常用WACC，代表機會成本
3. 期間（n）：一般算5-15年

適用時機：

  • ✅ 現金流穩定、可預測的公司
  • ❌ 景氣循環股、新創公司、科技公司、生技公司

DDM（股利折現模型）：

V0 = D1 ÷ (r - g)
D1 = 明年預估股利
r = 折現率
g = 股利成長率
  • 適用：穩定配息的成熟企業
  • 不適用：不配息的公司（Google、Amazon）

8. ROIC 投入資本報酬率

公式：

ROIC = EBIT × (1-稅率) ÷ 投入資本

判斷標準：

  • ROIC > WACC：創造超額報酬，資本使用效率佳
  • ROIC = WACC：沒有創造超額報酬
  • ROIC < WACC：破壞公司價值，資本使用效率差

vs ROE vs ROA：

指標分子分母特點
ROICEBIT × (1-稅率)投入資本排除槓桿與業外，最全面
ROE稅後淨利股東權益會被槓桿影響
ROA稅後淨利總資產會被短期負債影響

9. EPS 每股盈餘計算與判斷

公式：

基本 EPS = 稅後淨利 ÷ 流通在外普通股數
稀釋 EPS =（稅後淨利 - 特別股股利）÷（普通股 + 潛在稀釋股）

EPS選股四大條件：
1. EPS相近時，選本益比低的（更便宜）
2. 同類股中，選EPS較高的（獲利能力強）
3. 看過往EPS趨勢是否正成長（排除假成長）
4. 選EPS穩定的公司（波動大風險高）

三大陷阱：
1. 業外收入虛增EPS：賣土地、賣設備等一次性收入
2. 庫藏股讓EPS變大：股數減少，EPS上升但獲利沒變
3. 短期景氣波動：訂單暴增讓EPS短期走高

調整後EPS：

調整後EPS =（稅後淨利 - 特別股股利 ± 業外損益調整）÷ 股數

11. Altman Z-Score 財務預警模型

公式：

Z-Score = 1.2A + 1.4B + 3.3C + 0.6D + 1.0E
A = 營運資金/總資產
B = 留存收益/總資產
C = EBIT/總資產
D = 公司市值/總負債
E = 營業收入/總資產

三大區間：

Z-Score區間意義
> 2.99安全區財務健康，破產風險低
1.81-2.99灰色區可能有財務問題
< 1.81遇險區破產風險高，應避開

實戰應用：

  • 2008年金融海嘯前，整體股市Z-Score中位數只有1.81
  • 準確預測了危機，準確度達82%-94%

12. 成長股 vs 價值股選股差異

價值股特徵：

  • 本益比偏低
  • 股價淨值比偏低
  • 殖利率較高（>5%）
  • 基本面穩健

成長股特徵：

  • 營收成長率YoY > 15%
  • EPS連續多年成長
  • 高本益比（市場給予溢價）
  • 處於成長期的新興產業

配息股特徵：

  • 配息穩定（連續多年）
  • 殖利率合理（4-6%）
  • 現金流穩健

ETF績效比較（2008-2020）：

  • 成長型ETF（IUSG）：多頭表現最佳，2008年跌幅最深
  • 價值型ETF（VTV）：表現平穩，空頭抗跌
  • 配息型ETF（DVY）：介於兩者之間

13. 營收財報公布時程

台股財報公布：

  • 每月10日前：公布上月營收
  • 每季結束後30天內：公布季財報
  • 年度結束後3個月內：公布年報

營收成長率三種類型：

  • YoY（年增率）：與去年同月比較 — 最常用，避開淡旺季
  • QoQ（季增率）：與上一季比較
  • MoM（月增率）：與上個月比較

動能判斷：

  • 短期營收年增率 > 長期營收年增率 → 業績動能轉強
  • 短期營收年增率 < 長期營收年增率 → 業績動能轉弱
