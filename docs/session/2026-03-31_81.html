減資是什麼

是什麼：減資（Capital Reduction）是指公司減少資本額（股本），讓流通在外的股數變少。減資後投資人持股變少，但股價會等比例上升，總市值維持不變。

核心概念：

  • 減資是「調整」股價，不是「漲」股價
  • 減資前後，投資人持有的總市值不變
  • 減資會影響股東權益結構（與股票分割不同）

三種減資方式

1. 一般減資（彌補虧損減資）

是什麼：公司直接註銷部分流通股票，用來打消累積虧損。

怎麼做：
1. 公司累積虧損嚴重，每股淨值低於法定標準
2. 董事會決議減資比例（如減資20%）
3. 註銷部分股票，減少股本
4. 每股淨值提升（因分母變小）

實例：

  • 情境：某公司股價120元，總股數1000股，每股淨值只剩8元
  • 操作：宣布減資20%，1000股變800股
  • 結果：新股價 = 120 ÷ (1-20%) = 150元
  • 每股淨值從8元提升到10元，恢復信用交易資格

適用時機：

  • 公司連年虧損，需要美化財報
  • 每股淨值低於10元，避免取消信用交易
  • 每股淨值低於5元，避免變成全額交割股

風險：

  • 通常代表公司營運有問題
  • 減資後若經營策略沒改善，問題依然存在
  • 股價可能繼續下跌

2. 現金減資

是什麼：公司把多餘現金以「退還股款」名義發還給股東，同時減少股本。

怎麼做：
1. 公司帳上現金太多，且短期內沒有投資計畫
2. 董事會決議現金減資比例及每股退還金額
3. 股東收到現金（不用課所得稅）
4. 股數減少，股價調整

實例：

  • 情境：某公司股價120元，宣布現金減資20%，每股退還2元
  • 操作：1000股減為800股，每股退還2元
  • 結果：

適用時機：

  • 成熟產業，獲利穩定
  • 帳上現金太多，沒有擴廠計畫
  • 大股東想要節稅（現金減資不用課所得稅）

優點：

  • 大股東可以合法節稅（股利要課稅，現金減資不用）
  • 提升每股盈餘（EPS）和股東權益報酬率（ROE）
  • 改善財務結構

風險：

  • 可能暗示公司缺乏成長機會
  • 帳上現金減少，降低應對風險的能力
  • 不要把退還股款和股利混為一談

3. 庫藏股減資

是什麼：公司用帳上現金買回自家股票，然後註銷。

怎麼做：
1. 公司認為股價被低估
2. 用現金在市場買回自家股票
3. 將買回的股票註銷（不是放著當庫藏股）
4. 流通股數減少，EPS 提升

實例：

  • 情境：Dell 2003年花22.9億美元買回自家股票
  • 操作：買回後註銷，流通股數從27.26億股減至26.44億股
  • 結果：EPS提升，股價通常會上漲（市場解讀為公司看好自己）

適用時機：

  • 公司認為股價被低估
  • 想要提升每股價值
  • 防禦惡意併購

優點：

  • 對股價有正面激勵效果
  • 傳遞「公司看好自己」的訊號
  • 提升EPS和ROE

風險：

  • 公司花了帳上現金，這原本也是股東的錢
  • 若買回後股價仍下跌，公司虧損

減資對股價的計算公式

一般減資公式

減資後新股價 = 減資前收盤價 ÷ (1 - 減資比例)

現金減資公式

減資後新股價 = (股價 - 退還現金) ÷ (1 - 減資比例)

減資的優缺點總結

### 優點 1. **節稅**：現金減資不用課所得稅 2. **財報變好看**：EPS、ROE提升，每股淨值增加 3. **股價提升**：股本變小，股價調升 ### 缺點 1. **暗示財務風險**：可能被解讀為公司沒有成長性 2. **無實質獲利提升**：股價是「調整」上去，不是「漲」上去 3. **負債比提高**：現金減少，應對風險能力下降

投資人如何判斷減資好壞

關鍵問題：
1. 公司為什麼要減資？（彌補虧損 vs. 現金太多）
2. 減資後營運策略有沒有改變？
3. 公司基本面是否良好？

判斷標準：

減資類型好壞判斷
一般減資（彌補虧損）通常偏負面，代表公司營運有問題
現金減資（成熟產業）可能偏正面，代表公司現金充足、財務穩健
庫藏股減資通常偏正面，代表公司看好自己

注意：

  • 現金減資後，不要把退還股款算進殖利率
  • 減資是行政流程，關鍵在「為什麼減資」
  • 要回歸公司基本面判斷

心得

減資這件事，表面上看起來好像股價「變高」了，但實際上投資人的總資產根本沒變。很多人會被「股價上漲」這個假象騙了，以為自己賺到了。

真正要關注的是：公司為什麼要減資？

如果是因為連年虧損、每股淨值太低被迫減資，這公司可能爛到骨子裡，減資只是粉飾太平。這種減資後股價通常會繼續跌，因為根本問題沒解決。

如果是成熟產業、現金太多沒地方花，選擇現金減資把錢還給股東，這反而是好現象——代表公司財務穩健，不亂投資。長榮海運就是一個好例子。

庫藏股減資通常是公司覺得自己被低估了，用真金白銀買回自家股票。這種訊號比較正面，但要確認公司是真的有錢而不是舉債買股。

最後，現金減資退還的錢不是股利，不要把它算進殖利率裡面。這種「假殖利率」陷阱要小心。
