盈餘品質分析

第 4 回合學習 - 盈餘品質分析

核心知識

盈餘品質（Earnings Quality）

是什麼：判斷公司帳面盈餘能否真正轉化為現金的能力，衡量獲利的「含金量」。

為什麼重要：

  • 損益表採「應計制」，收入發生就認列，不管錢有沒有進來
  • 現金流量表是「實收實付制」，有錢進出才記錄
  • 盈餘品質差的公司，帳面賺錢但現金沒進來，可能周轉出問題

怎麼判斷：
1. 營業現金流 ÷ 淨利：比值接近 1 代表盈餘品質好
2. 營業現金流長期高於淨利 → 盈餘品質佳
3. 營業現金流長期低於淨利 → 可能做帳或收款有問題

實例：

  • 某公司 EPS 5 元，營業現金流每股 4.8 元 → 盈餘品質 96%，算不錯
  • 某公司 EPS 3 元，營業現金流每股 0.5 元 → 盈餘品質 17%，危險訊號

現金流量表四大分類

1. 營業現金流（Operating Cash Flow）

是什麼：公司本業營運實際賺進來的現金。

怎麼看：

  • 正值 = 公司營運有現金流入（正常）
  • 負值 = 營運持續燒錢（通常是新創或虧損期）

公式：稅後淨利 + 折舊攤銷 + 營運資金變動（應收帳款、存貨、應付帳款）

實例：

  • 台積電 2023 營業現金流 1.2 兆元，淨利 1.1 兆元 → 現金轉換能力強
  • 某公司淨利 10 億，但應收帳款增加 8 億 → 營業現金流可能只剩 2 億

2. 投資現金流（Cash Flow from Investing）

是什麼：公司買設備、廠房、投資其他公司的現金流。

怎麼看：

  • 負值 = 公司在擴張（正常現象，代表把錢投入未來）
  • 正值 = 公司在變賣資產（通常不是好事，除非是處分非核心資產）

實例：

  • 特斯拉 2015-2020 投資現金流每年負數十億美元 → 持續擴廠
  • 某公司投資現金流突然轉正大額 → 可能賣土地救急，要查清楚

3. 融資現金流（Cash Flows from Financing）

是什麼：公司跟股東、銀行借錢或還錢的現金流。

怎麼看：

  • 正值 = 公司在借錢、增資（可能是擴張，也可能是缺錢）
  • 負值 = 公司在還債、發股利、買回庫藏股（通常是成熟公司）

實例：

  • 新創公司融資現金流長期正值 → 靠募資存活
  • 成熟公司融資現金流負值，發股利 → 財務健康的表現

4. 自由現金流（Free Cash Flow）

是什麼：公司扣除必要投資後，真正可自由運用的現金。

公式：營業現金流 + 投資現金流（投資通常是負的，所以是相減）

怎麼看：

  • 正值 = 公司有餘錢可發股利、併購、買庫藏股
  • 長期負值 = 資金缺口，可能需要借錢或增資

實例：

  • 特斯拉 2024 自由現金流 96 億美元 → 轉正，財務體質改善
  • 某公司自由現金流連三年負值 → 危危險，遲早要融資

不同階段公司的現金流特徵

公司階段營業現金流投資現金流融資現金流自由現金流
財務穩健成熟公司正（穩定流入）負（合理投資）負（還債發股利）正

現金流量表看公司健康的快速檢查

好公司的特徵：
1. 營業現金流長期正值，且與淨利接近
2. 自由現金流長期正值
3. 投資現金流負值（持續投資），但金額合理
4. 融資現金流負值或小幅正值（不頻繁募資）

危險訊號：
1. 營業現金流長期低於淨利 → 盈餘品質差
2. 自由現金流連三年負值 → 資金周轉風險
3. 投資現金流突然大額正值 → 可能賣資產救急
4. 融資現金流持續正值且金額大 → 持續跟市場要錢

實戰步驟

Step 1：計算盈餘品質

盈餘品質 = 營業現金流 ÷ 稅後淨利
  • > 1.0：非常健康，現金轉換能力強
  • 0.8-1.0：正常範圍
  • 0.5-0.8：稍差，要查原因
  • < 0.5：危險，可能作帳或收款有問題

Step 2：追蹤自由現金流

查看最近 3-5 年的自由現金流：

  • 全正：健康
  • 偶爾負：可能是一次性投資，可接受
  • 連續負：危險

Step 3：交叉驗證三大報表

損益表資產負債表現金流量表檢查重點
淨利增加應收帳款大增營業現金流沒跟上可能塞貨給客戶，還沒收到錢
淨利增加存貨大增營業現金流下降可能產品賣不動
淨利穩定應收帳款穩定營業現金流穩定健康的公司

心得

盈餘品質這概念比我想像的重要。以前只看 EPS，覺得有賺就好，現在才知道「賺多少」不等於「拿到多少現金」。

現金流量表是三張報表裡最難造假的，因為是真金白銀進出。以後看財報，營業現金流和自由現金流要列為第一順位檢查。

特斯拉的例子很有趣，早期燒錢階段現金流全負，靠市場信念支撐融資，後來成功轉正。這種公司風險很高，但若賭對了報酬驚人。不過我目前還是偏好財務穩健的公司，自由現金流要長期正值才安心。

一句話：EPS 是面子，現金流是裡子。面子可以妝，裡子假不了。
