籌碼面分析：作帳行情×結帳行情×集團作帳

每季季底（3月、6月、9月、12月），台股市場都會出現一種特殊現象——法人為了績效結算而拉抬或出脫股票，這就是「作帳行情」和「結帳行情」。搞懂誰在作帳、誰在結帳、什麼時候該上車什麼時候該下車，是籌碼面分析中非常實戰的技能。

一、投信作帳行情

是什麼

投信基金經理人的績效決定其生存——績效差隨時被換掉，績效好才能吸引資金流入和獲得高額分紅。因此每到季底，投信會拼命拉抬手中持股，讓帳面數字好看。

關鍵差異：投信要的是「帳面上漲」的數字，不是真正賣出實現獲利。只要基金淨值在結算日那天是高的，績效評比就好看。

投信作帳三大特徵

1. 時間點明確

  • 季底結算：3月、6月、9月、12月底
  • 重要性：年底 > 半年底 > 季底
  • 年底動作最大（影響年終獎金和來年募集資金）

2. 選股邏輯：獨特性原則

  • 投信經理人會拉抬「別人沒有但我有」的股票
  • 同類型基金都持有的股票（如台積電、鴻海）不會是作帳標的
  • 股本大的股票拉不動，中小型股才是作帳目標
  • 篩選條件：同類型前幾名基金 → 找出「我有你沒有」的持股 → 持股比例較高的中小型股

3. 掛單手法：假單吸籌

  • 看多時：在外盤掛大單，製造壓力假象 → 散戶嚇出 → 投信低價吸籌 → 吸夠撤單 → 股價飆漲
  • 看空時：在內盤掛大單，製造支撐假象 → 散戶搶進 → 投信偷偷倒貨

實戰操作步驟

Step 1：5-6月開始觀察投信持股變化（年底作帳半年前就開始布局）

  • 用 MoneyDJ 查詢同類型基金排行前 3-5 名
  • 找出「我有你沒有」的持股

Step 2：確認投信持股比例

  • 總投信持股 <10%：還在吸籌期，可以跟進
  • 總投信持股 10-15%：作帳期，注意風險
  • 總投信持股 >15%：⚠️ 危險區，隨時可能結帳出貨

Step 3：10月以後出現的投信突然連續買超 → 可能是「王牌持股」作帳

  • 篩選條件：投信連續買超 + 持股比例不高 + 漲幅不超過30%

Step 4：結算日前 1-2 週是最危險的時間

  • 作帳拉抬完畢後必定結帳
  • 結帳訊號：投信買超轉為賣超

二、自營商結帳行情

跟投信作帳的關鍵差異

項目投信作帳自營商結帳
目的帳面績效好看落袋為安、實現獲利
手法拉抬持股股價賣出獲利持股
對股價影響上漲下跌
關注指標投信持股比例自營商持股變化
結果帳面數字漂亮白花花的鈔票入袋

自營商結帳時間表

  • 每季第一個月（1月、4月、7月、10月）：自營商大量買進，布局新季度
  • 每季第二個月：持股股價開始反映
  • 每季最後一個月（3月、6月、9月、12月）：⚠️ 自營商逢高調節，結帳出場

實戰操作

搭轎時機：每季第一個月的自營商大買股 + 股價橫向整理 = 進場時機
下車時機：進入每季最後一個月 = 果斷出場

口訣：投信作帳要搭車，自營商結帳要下車。

三、集團作帳行情

是什麼

集團公司透過旗下公司交叉持股，或在媒體釋出利多消息，拉抬集團內個股股價，美化帳面價值。比投信作帳更難預測，因為沒有像投信持股比例那樣的公開資訊。

常見作帳集團

集團旗下代表股作帳特徵
華新麗華華新、華新科、精星、佳邦電動車題材加持，5月底常有利多
國巨國巨、奇力新、至鴻被動元件景氣循環
聯電聯電、聯詠中小型轉投資
台塑台塑、南亞、台化傳產集團穩定作帳
鴻海鴻海、群創、廣宇權值股帶動
統一統一、統一超消費穩定型

集團作帳辨識四步驟

1. 歷史觀察法

  • 查過去3-5年該集團在季底/年底的股價表現
  • 找出集團作帳的時間規律和主攻個股

2. 交叉持股追蹤

  • 集團內公司互相持有對方股票
  • A公司買B公司股票 → 兩者股價連動
  • 當集團開始作帳，旗下所有股票容易同步上漲

3. 利多消息配合

  • 集團作帳常配合利多消息釋出
  • 「5月底公布特斯拉相關利多」= 精心安排的作帳時間點
  • ⚠️ 利多出盡 = 作帳結束

4. 題材催化

  • 有題材的集團股更容易作帳成功
  • 沒有基本面支撐的作帳，最終會被打回原形

四、作帳行情 vs 結帳行情：完整對照表

維度投信作帳自營商結帳集團作帳
動機績效排名、獎金落袋為安、實現獲利美化財報、集團形象
方向拉抬股價賣出持股拉抬集團股
時間季底前1-2週季底最後1個月年底10-12月
標的中小型股為主股性活潑的股票集團旗下個股
風險結算後反手賣出結帳完股價回落作帳後回歸基本面
持股上限單一基金10%無明確限制交叉持股

五、作帳行情實戰SOP

進場時機

1. 投信作帳：5-6月開始追蹤投信持股變化，找出「我有你沒有」的中小型股
2. 自營商搭車：每季首月（1/4/7/10月）追蹤自營商新增持股
3. 集團作帳：10月開始觀察歷史作帳集團的股價變化

出場時機（最重要！）

1. 投信結帳訊號：投信買超轉賣超 → 立即出場
2. 自營商結帳：進入每季最後一個月 → 果斷下車
3. 集團作帳結束：利多出盡 + 集團股轉弱 → 出場
4. 絕對出場：季底結算日前的最後一週，不管賺賠都要考慮出場

避險原則

  • 不要在季底最後一週才進場：這是結帳潮的高峰期
  • 總投信持股 >15% 的股票不要追高：隨時可能結帳
  • 沒有基本面支撐的作帳股不要長抱：作帳結束後股價會回歸基本面
  • 自營商跟單原則：跟進要快、下車更要快，自營商操作周期短

心得

作帳行情是台股獨有的籌碼面現象，本質是法人為了自身利益而製造的短期行情。三個重點：

1. 投信作帳要搭車，自營商結帳要下車——這兩句話就是精華。投信要的是帳面數字，所以會拉抬；自營商要的是實現獲利，所以會賣出。同一檔股票，投信在拉、自營商在賣，你要知道跟誰。

2. 作帳行情的本質是零和遊戲的末端——法人拉抬的每一分錢，最終都要有人買單。作帳完畢後的結帳潮是必然的，不要幻想股價會永遠停留在高點。總投信持股超過15%就是危險訊號，這跟之前的學習「投信持股>15%警戒」完全吻合。

3. 集團作帳比投信作帳更難捉摸——沒有公開的持股比例可以追蹤，只能靠歷史經驗和交叉持股分析。但集團作帳的好處是規模更大、持續時間可能更長，如果能在10月就識別出作帳集團，獲利空間更大。

外資期貨未平倉×選擇權籌碼分析

期貨和選擇權的未平倉量（Open Interest, OI）是籌碼面分析中最高級的技能之一。不同於現貨的法人買賣超，期貨和選擇權的未平倉量直接透露了市場主力對未來方向的看法，而且因為槓桿更大，這些信號往往比現貨更靈敏、更有前瞻性。

一、期貨未平倉量（Open Interest）基礎

是什麼

期貨未平倉量 = 市場上尚未平倉了結的期貨合約口數。每一口多單必然對應一口空單，所以多頭未平倉 = 空頭未平倉。

關鍵概念：未平倉量不是「誰做多多少、誰做空多少」的淨值，而是「市場上還有多少資金押注在裡面」。

未平倉量變化的三種情境

情境價格成交量未平倉解讀
多方進場上漲增加增加新多資金進場，趨勢力道持續
多方出場上漲增加減少多方獲利了結，空方停損，偏中性
空方進場下跌增加增加新空資金進場，趨勢力道持續
空方出場下跌減少減少空方回補，多方停損，偏中性

怎麼做：
1. 每天收盤後到期交所查詢三大法人期貨未平倉量
2. 追蹤外資淨部位變化（多單-空單），連續3天同方向加碼就是明確信號
3. 對照價格走勢：如果指數上漲但外資淨多單減少，趨勢可能即將反轉

實例：

  • 2021年5月台股大跌，外資期貨空單從-37519口減少至-15524口，外資在低點回補空單
  • 結論：外資期貨空單大幅增加後，指數止跌回升時外資回補，確認底部

二、大外資 vs 小外資：拆解外資部位

是什麼

「外資」不是一個人，而是幾十個外資機構的集合。把外資拆成「大外資（五大特定法人）」和「小外資」，判斷力道完全不同。

大外資 vs 小外資的差異

項目大外資（五大特定法人）小外資
定義部位最大的前5名外資外資總部位減去五大外資
部位大小極大，佔外資部位50-70%較小，但更靈活
操作風格中長期布局短線進出
信號意義中期趨勢方向短線方向變化
反應速度慢（部位太大，無法短線進出）快（部位小，靈活轉向）

怎麼做：
1. 到期交所查詢「大額交易人未沖銷部位結構表」，取得五大特定法人部位
2. 小外資部位 = 外資總部位 − 五大特定法人部位
3. 看短線變化：參考小外資的翻多翻空時機
4. 看中期方向：參考大外資的持倉方向

實例：

  • 小外資在2021年5月大跌後比大外資更快翻多，領先指數反彈
  • 大外資部位方向不變（仍偏空），說明中期趨勢尚未扭轉
  • 當大小外資同時翻多時，行情偏多看待的勝率最高

雙外資共振策略

  • 大外資偏多 + 小外資偏多：強力偏多訊號，可以順勢做多
  • 大外資偏多 + 小外資偏空：中期仍偏多，但短線可能回檔
  • 大外資偏空 + 小外資偏空：強力偏空訊號，做空為主
  • 大外資偏空 + 小外資偏多：中期仍偏空，短線可能有反彈但不可追

三、期貨未平倉量與正逆價差

是什麼

  • 正價差：期貨價格 > 現貨價格（期貨比現貨貴）
  • 逆價差：期貨價格 < 現貨價格（期貨比現貨便宜）

台指期特殊現象

台指期大部分時間呈現逆價差，原因：
1. 除權息旺季（6-9月），期貨價格已扣除預期除息點數
2. 持有現貨的法人做空期貨避險，壓低期貨價格

怎麼做：
1. 不要用正逆價差直接判斷多空，正價差不代表看空，逆價差不代表看多
2. 重點看結算日：正逆價差會趨近於零，這是套利機會
3. 結算日前3天，觀察外資期貨淨部位的變化方向，比正逆價差更有參考價值

實例：

  • 外資大量買進現貨同時做空期貨 → 逆價差擴大 → 這是避險，不是看空
  • 外資大量買進期貨多單 → 逆價差收斯甚至轉正價差 → 這才是真正看多

四、選擇權未平倉量（OI）判讀

是什麼

選擇權未平倉量 = 尚未結算的Call和Put合約口數，按履約價分布。每個履約價都有對應的未平倉量，形成一個「支撐壓力地圖」。

以賣方角度解讀

選擇權未平倉量以賣方角度解讀，原因：賣方承擔較高保證金和風險，通常是法人或大戶。

  • Call OI大量區 = 賣Call的人在這個價位看不漲 → 壓力區
  • Put OI大量區 = 賣Put的人在這個價位看不跌 → 支撐區

怎麼做：
1. 每天收盤後，查詢台指選擇權各履約價的Call和Put未平倉量
2. 找出Call OI最大的履約價 = 上檔壓力
3. 找出Put OI最大的履約價 = 下檔支撐
4. 觀察支撐壓力區是否每天移動：如果壓力區從18000移到18500，代表多方力道增強

實例：

  • 台指在17500附近，Call OI最大量在18000（壓力），Put OI最大量在17000（支撐）
  • 交易區間就在17000-18000之間
  • 如果Put OI最大量從17000上移到17200，支撐上移，偏多看待

未平倉量增加 vs 減少的意義

變化解讀
Call OI增加新的賣方進場看不漲，壓力增強
Call OI減少賣方回補（看漲出場），壓力減弱
Put OI增加新的賣方進場看不跌，支撐增強
Put OI減少賣方回補（看跌出場），支撐減弱

五、Put Call Ratio（PCR）實戰判讀

是什麼

PCR = Put未平倉量 ÷ Call未平倉量

有兩種計算方式：

  • 成交量PCR：Put成交量 ÷ Call成交量（反映當日交易偏向）
  • 未平倉量PCR：Put未平倉量 ÷ Call未平倉量（反映累積部位偏向）

PCR判讀：反指標！

PCR是散戶反指標，原因：選擇權市場以散戶為主，散戶往往在錯誤的時間做錯誤的事。

PCR數值散戶行為實際意義
PCR > 1.2大量買Put防跌散戶過度恐慌 → **偏多訊號**
PCR 1.0-1.2正常偏空中性偏多觀察
PCR 0.8-1.0正常偏多中性
PCR < 0.8大量買Call追漲散戶過度樂觀 → **偏空訊號**

怎麼做：
1. 每天盤後計算PCR數值
2. PCR極端值（>1.3或<0.7）是更強的反向信號
3. 結合外資期貨部位判斷：如果PCR極端偏多（散戶恐慌）+ 外資淨多單增加 = 強力做多訊號

實例：

  • 2020年3月COVID崩盤時，PCR飆升至1.5以上，散戶瘋狂買Put避險
  • 結果大盤在散戶最恐慌時見底反彈
  • 相反，2021年初散戶瘋狂買Call，PCR跌到0.6以下，大盤隨後回檔

PCR的成交量版 vs 未平倉版

版本優點缺點
成交量PCR反應即時，日內可用波動大，容易被短線交易扭曲
未平倉PCR累積部位，更穩定反應較慢，需要連續觀察趨勢

建議：短線看成交量PCR，中長線看未平倉PCR的趨勢變化。

六、三大法人選擇權部位判讀

是什麼

期交所每天公布外資、投信、自營商在選擇權市場的Call和Put未平倉部位。法人選擇權部位的解讀邏輯與全市場不同——看法人是以買方還是賣方的角度。

外資選擇權部位判讀

外資動作解讀
買Call（做多Call）看多大盤，願意支付權利金賺取上漲利潤
賣Call（做空Call）看不漲或溫和看空，收取權利金
買Put（做多Put）看空大盤或避險，支付權利金
賣Put（做空Put）看不跌或溫和看多，收取權利金

怎麼做：
1. 到期交所查詢「三大法人選擇權買賣權分計」資料
2. 觀察外資的Call和Put淨部位變化
3. 外資淨買Call增加 + 淨賣Put增加 = 強力偏多
4. 外資淨買Put增加 + 淨賣Call增加 = 強力偏空

實例：

  • 外資Sell Put大量增加 → 外資不看跌，大盤有支撐
  • 外資Buy Put大量增加 → 外資在做空或避險，大盤可能下跌

籌碼面分析：融券回補×軋空行情×內部人持股

融券回補和內部人持股變動，是籌碼面分析中兩個極具實戰價值的主題。融券回補是「被動式」的籌碼壓力——不管放空者願不願意，時間到了就必須買回；內部人持股變動則是「主動式」的籌碼訊號——公司高層用真金白銀表態對未來的看法。兩者搭配使用，可以提前捕捉股價反轉的時機。

一、融券強制回補機制

是什麼

融券回補 = 放空者必須買回股票還給券商。當公司需要確認股東名冊時（股東會、除權息、現金增資），融券者必須在停止過戶日前6個營業日（含）前完成回補，否則券商會以市價強制買回，不管價格多高。

為什麼必須回補：融券借來的股票，原股東（融資買進者）仍登記為股東。當公司需要確認「誰才是真正的股東」來配發股利或開股東會時，融券者必須把股票還回去，否則會出現重複計算股東的問題。

融券強制回補的四大觸發時機

觸發原因發生時間回補期限停券天數
股東常會每年3-4月停止過戶日前6個營業日約3-7天
除息每年6-9月為主停止過戶日前6個營業日約3-7天
除權不一定停止過戶日前6個營業日約3-7天
現金增資不定期停止過戶日前6個營業日約3-7天

重要：臨時股東會「不需要」融券回補（法規例外條款）。

怎麼做：
1. 查詢台灣證券交易所「停券預告表」→ 確認回補日期
2. 路徑：證交所網站 → 交易資訊 → 融資融券與可借券賣出額度 → 停券預告表
3. 計算：停止過戶日往前推6個營業日 = 最後回補日
4. 在最後回補日前至少2-3天就完成回補，不要壓最後一刻

實例：

  • 某公司公告除息停止過戶日為7月21日
  • 往前推6個營業日 = 7月15日（含）為最後回補日
  • 7月9日至7月15日為回補期間
  • 7月16日至7月21日為停券期間，無法融券賣出

一年多次回補的陷阱

台股目前一年最多可配息4次（每季配息），加上股東會和除權：

  • 卜蜂（1215）：一年有3次融券回補（股東會4/15、除息6/29、除權8/19）

結論：經常需要停券的標的，不適合長期放空。平均放空2個月就要被迫回補，交易成本極高。

二、軋空行情（Short Squeeze）

是什麼

軋空 = 融券者被迫在高位回補，推升股價，股價上漲又迫使更多融券者停損回補，形成惡性循環。

軋空發生的三大條件

條件1：融券餘額高

  • 融券餘額佔成交量比例越高，回補壓力越大
  • 融券餘額 > 日均量的3倍以上 → 軋空風險極高

條件2：流動性低

  • 日均成交量小，融券者搶著買回時買不到
  • 冷門股 + 高融券 = 軋空最佳溫床

條件3：接近最後回補日

  • 越接近最後回補日，融券者的時間壓力越大
  • 特定人士可能故意在最後回補日前哄抬股價

軋空行情的運作機制

融券餘額高 → 接近回補日 → 部分融券者開始回補 → 股價上漲
→ 更多融券者停損回補 → 股價進一步上漲 → 特定人士趁機拉抬
→ 融券者被迫以更高價格回補 → 股價飆漲 → 軋空完成

怎麼做（搭軋空順風車）：
1. 每月追蹤融券餘額變化，篩選融券餘額偏高且即將回補的股票
2. 確認日均成交量，融券餘額/日均量 > 3倍是危險訊號
3. 在最後回補日前1-2週觀察股價是否開始異常上漲
4. 搭車進場時機：最後回補日前5-10個營業日
5. 出場時機：回補日當天或前一天就出場，不要戀戰

實例：

  • 某中小型股日均量500張，融券餘額2000張
  • 融券餘額/日均量 = 4倍，軋空風險極高
  • 最後回補日前一週，股價從50元漲到55元（+10%）
  • 回補日前3天，股價加速漲到62元（+24%）
  • 回補日後融券歸零，股價回落到58元
