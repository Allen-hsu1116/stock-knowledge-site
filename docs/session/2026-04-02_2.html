均值回歸核心概念

是什麼：價格偏離平均值太遠後，終究會回歸到正常水準的現象。就像橡皮筋，拉太緊會彈回來。

為什麼有效：統計學上，95.4% 的數值會落在 2 個標準差內。價格極端偏離平均時，回歸的機率很高。

兩種交易哲學：

  • 動能突破策略：追漲殺跌，相信趨勢會延續
  • 均值回歸策略：低買高賣，相信價格會回歸平均

布林通道：均值回歸的實戰工具

組成公式

中軌 = 20 MA（20日移動平均線）
上軌 = 20 MA + 2 倍標準差
下軌 = 20 MA - 2 倍標準差
帶寬 = (上軌 - 下軌) / 中軌

參數可調整：週期可用 60 日，標準差可用 1.5 倍或 3 倍

統計意義

  • 1 倍標準差：涵蓋 68.2% 的價格
  • 2 倍標準差：涵蓋 95.4% 的價格
  • 3 倍標準差：涵蓋 99.8% 的價格

實務意義：價格大多時間在通道內，碰到上下軌時回歸中軌的機率高

逆勢交易策略（區間盤整用）

適用時機：市場橫盤震盪，沒有明確趨勢

操作訊號

訊號解讀操作
價格由下往上穿過下軌下跌力道消退，可能反轉買進訊號
價格由下往上穿過中軌上漲力道足夠加碼做多
價格由上往下跌破上軌上漲力道消退，可能反轉放空訊號
價格由上往下跌破中軌下跌力道足夠加碼作空

區間交易實例

情境：台積電一年區間 450-550 元

  • 450 元（區間底部）→ 買進
  • 550 元（區間頭部）→ 賣出

布林通道實戰：

  • 下軌碰到反彈 → 買進，目標中軌或上軌
  • 上軌碰到反轉 → 放空，目標中軌或下軌

順勢交易策略（趨勢行情用）

適用時機：帶寬由窄轉寬，價格突破通道

趨勢判斷

多頭趨勢形成：
1. 帶寬由窄開始擴大
2. 價格沿著上軌持續前進
3. 下跌力道強勁

空頭趨勢形成：
1. 帶寬由窄開始擴大
2. 價格沿著下軌持續前進
3. 下跌力道強勁

雙布林通道進階技巧

設定方式：

  • 短天期：20 MA + 2 倍標準差
  • 長天期：60 MA + 3 倍標準差

使用邏輯：

  • 95.4% 價格在 2 個標準差內
  • 99.8% 價格在 3 個標準差內
  • 超過短天期上下軌，以長天期為目標終點

趨勢判斷：

  • 短天期中軌在長天期中軌之上 → 偏多
  • 短天期中軌在長天期中軌之下 → 偏空

搭配 RSI 指標

是什麼：用 RSI 確認超買超賣

怎麼做：

  • RSI 接近 80（超買）+ 價格碰到上軌 → 賣出訊號更強
  • RSI 接近 20（超賣）+ 價格碰到下軌 → 買進訊號更強

風險與注意事項

最大風險：遇到趨勢行情

均值回歸最怕趨勢突破：

  • 價格突破區間上緣 → 繼續大漲
  • 價格跌破區間下緣 → 繼續大跌

解決方案：

  • 嚴格停損
  • 搭配選擇權或期貨避險
  • 觀察帶寬變化，帶寬擴大時停止逆勢操作

交易成本問題

  • 均值回歸交易次數多
  • 手續費佔獲利比例高
  • 必須維持高勝率才能獲利

不適用的股票

  • 財務有問題的公司：價格可能不回歸
  • 持續創新高/新低的股票：均值回歸失效

實戰心法

何時用逆勢：帶寬穩定、價格在通道內震盪
何時用順勢：帶寬由窄轉寬、價格突破通道
關鍵指標：帶寬變化是判斷行情類型的關鍵

切記：

  • 沒有 100% 準確的指標
  • 均值回歸是「機率」，不是「確定」
  • 單一指標不夠，要搭配其他工具

心得

均值回歸策略的核心精神很簡單：「物極必反」。但實戰最難的是判斷什麼時候是「極端」，什麼時候是「還會更極端」。

布林通道把這個概念具體化了：用統計學的 2 個標準差定義「極端」。95.4% 的價格會在通道內，這給了我們一個明確的操作框架。

但這策略有個致命弱點：遇到真正的趨勢行情會被打臉。價格突破上軌後繼續噴、跌破下軌後繼續殺，這時候「均值回歸」就變成「賠錢」。

所以重點是：
1. 會判斷行情類型：盤整用逆勢，趨勢用順勢
2. 嚴格停損：錯了就認賠
3. 不要單押一個指標：搭配 RSI、成交量、均線一起看

均值回歸是很好的「區間交易」工具，但不是萬靈丹。用對時機才是關鍵。

量價關係核心概念

為什麼成交量重要？

「新手看價、老手看量」——這句話有道理。成交量反映的是參與者的決心：

  • 成交量大 = 多空看法分歧、籌碼混亂
  • 成交量小 = 多空看法一致、籌碼穩定

關鍵觀念：成交量一定是買張數等於賣張數。看盤軟體的紅綠色只是方便辨識，不代表「買進量」或「賣出量」。

六種量價關係詳解

1. 價漲量增

是什麼：股價上漲，成交量也增加。多頭趨勢最健康的型態。

怎麼判斷：

  • 低檔盤整突破 + 量增 = 確認漲勢健康
  • 高檔異常大量 + 長黑K棒 = 小心主力出貨

實例：

  • 情境：股價 50 元盤整三個月，突破 55 元壓力，成交量從日均 1,000 張放大到 3,000 張
  • 操作：可追價買進，設停損於突破點下方（如 52 元）
  • 危險訊號：股價在 100 元高檔，單日爆量 10,000 張（日均 2,000 張），收長黑K → 可能是出貨

適用時機：

  • ✅ 低檔突破壓力區
  • ✅ 主升段上漲確認
  • ❌ 高檔異常大量（小心反轉）

2. 價漲量縮

是什麼：股價上漲但成交量縮減。分兩種情況：量價背離 vs 量縮上漲。

怎麼區分：

特徵量價背離量縮上漲
K 線強弱紅K越來越弱K線強勢
成交量漸漸縮小漸漸縮小
含義追價資金減少賣方不足、籌碼鎖定
後勢可能下跌或盤整容易加速上漲

量縮上漲的操作意義：

  • 情境：想買 500 張，賣方只拋出 400 張 → 成交 400 張 + 鎖漲停
  • 代表：持有者不想賣、主力高度控盤
  • 周線/月線量縮上漲 = 長線主力看好，值得跟進

適用時機：

  • ✅ 周線/月線級別的量縮上漲（主力長線佈局）
  • ✅ 漲停鎖死造成的量縮（籌碼穩定）
  • ❌ 高檔價漲量縮 + 紅K變弱（背離訊號）

3. 價跌量縮

是什麼：股價下跌，成交量也縮減。多頭趨勢下的正常回檔。

怎麼判斷：

  • 正常洗盤：回檔幅度小、量縮 → 賣壓不大
  • 危險訊號：回檔幅度大、量縮 → 沒買盤接手

實例：

  • 情境：股價從 100 元漲到 120 元，拉回到 115 元，成交量從 3,000 張降到 1,500 張
  • 判斷：正常洗盤，可在 115 元附近承接
  • 危險：從 120 元跌到 105 元還量縮 → 沒人想買，續跌機率高

適用時機：

  • ✅ 多頭趨勢中的量縮回檔（買點）
  • ✅ 橫向盤整的洗盤（價平量縮）
  • ❌ 大幅度價跌量縮（可能續跌）

4. 價跌量增

是什麼：股價下跌但成交量放大。多頭趨勢中最危險的訊號。

怎麼判斷：

  • 高檔出現 = 主力出貨、多頭獲利了結
  • 連續下跌後突然量增 = 可能有人進場承接（底部訊號）

實例：

  • 情境：股價從 80 元漲到 150 元，某天跌到 140 元，成交量爆出 8,000 張（日均 2,000 張）
  • 操作：先出場觀望，等盤整結束再重新進場
  • 底部訊號：已跌三個月，某天量增跌到 60 元 → 可能是底部承接

適用時機：

  • ❌ 多頭高檔價跌量增（強烈賣訊）
  • ✅ 連續下跌後價跌量增（注意底部承接）

5. 價平量增

是什麼：股價持平但成交量放大。通常有人在收購籌碼或偷偷出貨。

怎麼判斷：

  • 低檔價平量增 = 主力吸籌，待表態後上漲
  • 高檔價平量增 + 多根上影線 = 主力偷偷出貨

實例：

  • 情境：股價在 50 元盤整，日均量 500 張，連續幾天量增到 2,000 張但股價沒動
  • 判讀：有人在吸貨，可提前佈局
  • 危險：高檔盤整 + 上影線 + 量增 → 出貨中

適用時機：

  • ✅ 低檔價平量增（主力吸籌）
  • ❌ 高檔價平量增 + 上影線（出貨訊號）

6. 價平量縮

是什麼：股價持平，成交量萎縮。通常出現在急漲後的籌碼沉澱。

怎麼判斷：

  • 急漲後價平量縮 = 主力洗盤、等待散戶離場
  • 盤整結束時量縮 = 即將變盤

實例：

  • 情境：股價從 60 元急漲到 90 元，接著在 85-90 元盤整兩週，成交量從 5,000 張縮到 1,000 張
  • 判讀：籌碼沉澱中，等突破再進場
  • 操作：突破 90 元 + 量增 → 追價

適用時機：

  • ✅ 急漲後的籌碼沉澱（等待突破）
  • ✅ 盤整期末的量縮（變盤前兆）

多頭趨勢量價關係總結表

量價關係多頭正常多頭異常操作建議
價漲量增✅ 漲勢健康⚠️ 高檔異常大量小心出貨低檔可追
價漲量縮⚠️ 量縮上漲（籌碼穩定）❌ 量價背離（追價減弱）看K線強弱
價跌量縮✅ 正常回檔洗盤❌ 大幅回檔無買盤小幅回檔可買
價跌量增❌ 頭部出貨訊號⚠️ 連跌後可能見底多頭趨勢快逃
價平量增⚠️ 主力吸籌❌ 高檔出貨低檔可跟
價平量縮✅ 籌碼沉澱—等待突破

實戰操作要訣

買點判斷

1. 量縮回檔買進：多頭趨勢中，股價拉回量縮時買進
2. 價平量增關注：低檔盤整時量增價平，可提前佈局
3. 突破量增確認：突破壓力區 + 價漲量增 = 追價

賣點判斷

1. 高檔價跌量增：多頭趨勢最危險訊號，優先出場
2. 高檔異常大量：爆量長黑K棒 = 主力出貨
3. 量價背離：價漲量縮 + K線轉弱 = 趨勢反轉

每日監控重點

異常訊號警覺：

  • 價跌量增出現在多頭趨勢
  • 高檔爆量長黑K棒
  • 價漲量縮但紅K越來越弱

心得

量價關係是技術分析的基礎中的基礎，但也是最實用的工具之一。今天學到最重要的一個觀念是：量縮上漲和量價背離的區分——關鍵在K線強弱。同樣是價漲量縮，K線強勢就是籌碼鎖定、K線變弱就是追價無力，這個差別決定了是要追還是要逃。

另一個重點是「異常訊號」的監控。多頭趨勢中，價跌量增是最危險的訊號，代表主力在出貨。以前只知道「價跌量縮」是回檔，現在知道要同時看「回檔幅度」來判斷是洗盤還是真的沒人要。

量價關係不是單獨使用的，要搭配：
1. K 線型態（長黑、長紅、上下影線）
2. 價格位置（高檔、低檔）
3. 趨勢方向（多頭、空頭）

最後，「量比價先行」這句話要記住——觀察量的變化，往往能提前發現趨勢轉折。操作策略上，量縮時買進比較不容易套牢，量增時買進如果遇到量縮拉回就會套牢好幾天。這個節奏感要培養。

跳空缺口是什麼

定義：相鄰兩根 K 線之間出現的「價格真空帶」，代表這個價格區間沒有任何人交易。

形成原因：市場發生特殊利多或利空，投資人在開盤前就積極掛單買進或賣出，使價格直接大幅開高或開低。

基本判斷：

  • 向上跳空 = 強勢多頭訊號
  • 向下跳空 = 強勢空頭訊號

四種缺口型態

1. 普通缺口（區域缺口）

是什麼：出現在盤整區間內的跳空，沒有明顯趨勢方向。

特徵：

  • 缺口真空區不大
  • 容易在短時間內（1-2 天）回補
  • 較無支撐、壓力參考作用

怎麼做：忽略，不具交易價值。

2. 突破缺口

是什麼：股價跳空突破盤整區間，確立趨勢方向。

特徵：

  • 向上突破伴隨成交量放大
  • 缺口短期內不會被回補
  • 形成支撐（向上突破）或壓力（向下突破）

怎麼做：
1. 確認突破缺口：價格突破盤整區 + 成交量放大 + 缺口未被回補
2. 進場時機：

  • 積極：缺口出現當天進場

3. 停損設在缺口下方

實例：

  • 情境：台積電在 600-620 區間盤整一個月
  • 突破：某天開盤直接跳空到 630，成交量比前一日增加 50%
  • 操作：在 630-635 區間進場，停損設在 618（缺口下緣）

適用時機：盤整區間明確突破時使用。

3. 逃逸缺口（測量缺口）

是什麼：趨勢進行中出現的跳空，表示趨勢尚未結束。

特徵：

  • 出現在突破缺口之後
  • 多頭行情中可能出現複數個
  • 不容易被回補

怎麼做：
1. 確認已經有突破缺口
2. 逃逸缺口出現時可以加碼
3. 持續持有直到出現竭盡缺口

適用時機：明顯趨勢進行中，想要加碼時使用。

4. 竭盡缺口

是什麼：趨勢末端的跳空，代表力道耗盡，即將反轉。

特徵：

  • 成交量暴增（主力出貨或散戶恐慌）
  • 2-3 天內快速回補缺口
  • 常出現在末升段或末跌段

怎麼做：
1. 警訊：缺口 + 巨量 + 快速回補
2. 多頭竭盡：準備做空或賣出持股
3. 空頭竭盡：準備抄底或回補空單

實例：

  • 情境：某股從 100 漲到 180，某天跳空開高到 195
  • 警訊：成交量是平常 3 倍，收盤價卻跌回 185
  • 隔天：股價跌回 178，缺口被回補
  • 判斷：這是竭盡缺口，準備反轉

適用時機：趨勢進行一段時間後出現的跳空要特別留意。

缺口回補的判斷

回補缺口是什麼：股價回到跳空的價格區間，填補空白。

判斷準則：

情況涵義
向上跳空後 2-3 天內回補假突破，後續看空
向上跳空突破後回補多頭力道結束，可能反轉
向下跳空後 2-3 天內回補假跌破，後續看多
向下跳空跌破後回補空頭力道結束，可能反轉

島狀反轉

是什麼：左右兩邊都有跳空缺口，中間的 K 線像島嶼一樣。

形成條件：

  • 向上跳空後，價格無法續創新高
  • 成交量萎縮
  • 接著向下跳空下跌

怎麼做：
1. 識別島狀反轉型態
2. 確認 2-3 天內缺口沒有被回補
3. 這是強勁反轉訊號，可以順勢操作

適用時機：不常出現，但發生時是明確反轉訊號。

實戰注意事項

排除假缺口：

  • 除權、除息造成的缺口 ≠ 真正缺口
  • 減資造成的缺口 ≠ 真正缺口
  • 這些不反映群眾心理，不適合用缺口戰法

判斷依據：
1. 缺口大小：越大代表走勢越強
2. 成交量：突破缺口要放量，竭盡缺口卻是巨量後無力
3. 回補時間：2-3 天內回補通常是假訊號

心得

跳空缺口這主題剛好補上了我對「趨勢確認」和「反轉判斷」的知識缺口（雙關）。

四種缺口的判斷關鍵在於「位置」和「成交量」：

  • 位置告訴你這是哪種缺口（盤整中 = 普通、突破區間 = 突破、趨勢中 = 逃逸或竭盡）
  • 成交量告訴你力道（突破要放量、竭盡是暴量後無力）

最實用的是「2-3 天內回補 = 假訊號」這條規則，可以過濾掉很多假突破。另外島狀反轉雖然不常見，但一出現就是明確反轉訊號，值得記住。

以前只會看「有缺口就當支撐壓力」，現在知道要分類型態，操作上會更精準。

核心知識

一、除權息基本概念

是什麼：公司將獲利以股票或現金方式分配給股東，股價會在除權息日向下調整。

除息 vs 除權：

  • 除息：發放現金股利（配息）
  • 除權：發放股票股利（配股）

怎麼算參考價：
1. 除息參考價 = 前一日收盤價 - 現金股利
2. 除權參考價 = 前一日收盤價 ÷（1 + 配股率）

  • 配股率 = 股票股利 ÷ 10（股票面額）

3. 除權息參考價 =（股價 - 現金股利）÷（1 + 配股率）

實例：

  • 情境：玉山金 2022/7/28 除權息，前一日收盤 29.35 元，配發現金 0.67 元 + 股票 0.674 元
  • 計算：(29.35 - 0.67) ÷ (1 + 0.0674) = 26.87 元
  • 實際參考價：26.85 元（符合升降單位 0.05）

二、除權息三大操作策略

策略一：公告後買進，除權息前賣出（不參加除權息）

是什麼：在公司公告高股利後進場，利用高殖利率題材帶動的股價上漲賺價差，除權息前賣出，不領股利。

怎麼做：
1. 3月年報公布後，留意公司股利政策
2. 挑選殖利率高、基本面佳的個股
3. 股利公告後買進，觀察融券回補潮
4. 除權息前 1-2 週賣出，賺價差

實例：

  • 情境：某公司公告配發 4 元現金股利，殖利率 5%
  • 操作：公告後股價從 75 元漲到 82 元，除息前賣出
  • 獲利：9.3% 價差，省去股利稅負

適用時機：

  • 高所得稅率族群（稅率 > 30%）
  • 大盤位置偏高，擔心貼息風險
  • 個股波段漲幅已大

策略二：除權息日當天買進，填息時賣出

是什麼：利用高所得稅率投資人在除權息前賣股節稅，造成除權息日股價相對低點，當天買進等填息。

怎麼做：
1. 挑選過往除權息當日上漲機率高的個股
2. 除權息日開盤觀察，找買點進場
3. 短線當日收盤賣出，或等填息過程中賣出

實例：

  • 情境：某高殖利率股除息參考價 90 元
  • 操作：除息日開盤 89.5 元買進，收盤 91.5 元賣出
  • 獲利：2.2% 當日價差

適用時機：

  • 短線交易者
  • 大盤多頭趨勢明確
  • 個股基本面強、填息機率高

策略三：除權息後買進，賺股利又賺價差

是什麼：除權息後，挑選中長期營運趨勢向上的個股，既領股利又賺後續漲勢。

怎麼做：
1. 篩選基本面佳、獲利成長的公司
2. 除權息後觀察股價表現，等回測再進場
3. 中長期持有，等填息甚至續漲

實例：

  • 情境：台積電除息後股價回測 560 元
  • 操作：560 元買進，持有至填息 575 元並續漲到 600 元
  • 獲利：股利 3.5 元 + 價差 40 元 = 7.7%

適用時機：

  • 中長期投資人
  • 公司營運展望佳
  • 產業趨勢向上

三、除權息時程表（一定要懂！）

關鍵日期：

日期名稱意義
T 日最後買進日這天買進才能領股利
T+1 日除權息交易日股價向下跳空，開始除權息
T+2 日最後過戶日公司開始造冊確認股東名冊
T+3~T+7停止過戶期間5 天造冊期間
T+8 日除權息基準日完成股東名冊，符合資格者可領股利

重點：

  • 想領股利，重點是「買進時間點」，跟抱多久無關
  • 最後買進日買進，隔天除權息日股價就會跳空

四、稅務成本（參加除權息前必算！）

二代健保補充保費

門檻：單檔個股、單次股利（現金+股票）超過 2 萬元

費率：2.11%

實例：

  • 持有 10 張股票，每股配 1 元現金 + 1 元股票
  • 股利總額：(1+1) × 1000 股 × 10 張 = 20,000 元
  • 二代健保：20,000 × 2.11% = 422 元
  • 實領：19,578 元

股利所得稅

兩種方式：
1. 合併課稅：股利併入綜所稅，享 8.5% 抵減稅額（上限 8 萬）
2. 分離課稅：股利單獨課 28%

怎麼選：

  • 稅率 20% 以下 → 合併課稅較划算
  • 稅率 40% 以上 → 分離課稅較划算
  • 報稅前先試算！

五、判斷是否參加除權息

三大評估要素：

1. 基本面：公司營運狀況、發展前景
2. 殖利率：至少 3% 才有參與價值
3. 市場環境：多空氛圍、大盤位置

額外觀察：

  • 歷史填權息表現
  • 近年除息前波段漲幅
  • 股價與 10 日線乖離率

避險技巧：

  • 若高殖利率股除息前漲幅已大，等回測再進場
  • 觀察殖利率是否來到歷史區間低檔

心得

除權息操作是台股特色，每年 6-8 月是旺季。三種策略各有適用情境，最重要的是搞懂自己要什麼：要價差還是要股利？

高所得稅率的人，策略一（除息前賣出）可能更划算，因為領股利要被課兩次稅（二代健保 + 所得稅），如果又貼息就虧大了。

除權息日買進那招蠻有意思的，利用富人賣股節稅創造的賣壓，在相對低點進場。但這招要挑「歷史填息機率高」的股票，不然買到貼息股就哭了。

最後，稅務成本真的要算。二代健保門檻是「單檔、單次」2 萬元，分散持股或分批參與除息可以規避。報稅時也要試算合併 vs 分離哪個划算，不要傻傻地多繳稅。

核心知識

進出場策略四種類型

是什麼：根據進場和出場是否分批，可分為四種策略組合。

四種類型：

類型進場出場特點
點進點出單次全進單次全出最簡單，全有全無
點進面出單次全進分批出場降低持倉壓力
面進點出分批進場單次全出起始風險低
面進面出分批進場分批出場最複雜但最穩

適用時機：

  • 新手 → 點進點出（最直覺）
  • 震盪盤 → 點進面出（降低壓力）
  • 確信度高 → 面進點出（逐步加碼）

點進面出（分批出場）

是什麼：一次買進，但分批在不同價位賣出，兼顧獲利速度與獲利長度。

怎麼做：
1. 設定多個出場目標價（例如 4 個價位）
2. 第一目標達成 → 出場 1/4 部位
3. 第二目標達成 → 再出場 1/4
4. 以此類推，最後 1/4 抱到最終目標
5. 每次出場後可移動停損到進場價（保本）

實例：

  • 情境：台積電 500 元買進 4 張
  • 操作：
  • 結果：平均賣出價 550 元，比一次性賣在 520 元多賺 30 元/張

優點：

  • 減少情緒壓力（早期獲利入袋）
  • 提高勝率（部分部位出在好價位）
  • 更有耐心持有長線

缺點：

  • 損益比降低（提前出場少賺）
  • 可能整體利潤變少

適用策略：

  • ✅ 起漲起跌型策略（C 點策略）
  • ✅ 震盪行情
  • ❌ 追漲殺跌型策略（高損益比策略不適合）

分批進場 vs 單筆買進

是什麼：進場時的兩種策略選擇，風險與報酬的取捨。

分批買進怎麼做：
1. 找到好公司，設定目標價
2. 股價跌到目標價 → 買第一批
3. 續跌 5-10% → 買第二批
4. 再跌 → 再買（越跌越買）
5. 不往上加碼（只在下跌時加碼）

單筆買進怎麼做：
1. 深入分析公司，確定是好公司
2. 價格遠低於內在價值 → 一次買進大量
3. 把這筆當作獨立投資，不加碼
4. 若續跌 → 檢查基本面，沒變壞就續抱

實例比較：

項目分批買進單筆買進
例子100 元開始買，分 3 批100 元一次買進
平均成本較低（攤平效果）較高
風險較低（分散時間）較高
報酬較低較高（若判斷正確）
適合者一般投資人深入分析者

適用時機：

  • 分批買進 → 對公司了解不深、不確定性高
  • 單筆買進 → 深入分析、高確信度、安全邊際大

實戰心法：選擇策略的關鍵

考慮因素：
1. 性格：你能承受波動嗎？喜歡短線還是長線？
2. 策略類型：追漲殺跌 vs 起漲起跌
3. 市場狀態：趨勢明顯 vs 震盪刷盤
4. 時間：有時間盯盤 vs 加班族

關鍵原則：

  • 「投資第一看風險，第二看流通性，最後才看獲利。」
  • 勝率與賠率要平衡，不能只追求其中一個
  • 多策略組合自然達成「面進面出」效果

避免的錯誤：

  • 買進奇怪或不穩定的標的
  • 虧損後才變成「長期投資」（套牢）
  • 限制交易次數可減少判斷錯誤（巴菲特：一輩子只能交易 20 次）

心得：

這篇讓我對「分批進出場」有更完整的理解。之前只知道「不要一次 all in」，但沒想過出場也可以分批。

點進面出的核心價值在於降低心理壓力，讓你能抱住長線。這點很實際——很多時候不是不知道要抱，而是波動太大讓你抱不住。分批出場就像「過程中吃點小點心」，讓你能堅持減肥計畫一樣。

另外，文章提到「追漲殺跌型策略不適合分批出場」這點很關鍵，因為這類策略本來就是靠高損益比獲利，提前出場會削弱優勢。這讓我想到：策略要搭配適合的進出場方式，不是什麼策略都能用同一套。

「投資第一看風險」這句話要記住。很多人（包括我）常先看獲利，這是本末倒置。

當沖三種操作模式

Tick流（極短線）

是什麼：賺取股價跳動檔位的價差，幾秒到幾分鐘內完成交易。

怎麼做：
1. 觀察五檔和即時成交明細
2. 看到價格跳動方向確認後進場
3. 跳1-2檔就獲利了結
4. 沒有連續性走勢立刻停損

實例：

  • 情境：台積電股價在600元，五檔顯示買盤力道強
  • 操作：600元進場，看到600.5、601連續成交，601元出場
  • 獲利：0.5檔 = 0.5元/股 = 500元/張（扣手續費約賺300-400元）

適用時機：盤中波動大、流動性好的股票；不適合開盤前30分鐘（波動太大不好抓）

一波流（突破交易）

是什麼：抓大單進場時的突破瞬間，跟著主力一起衝。

怎麼做：
1. 監控成交明細，看到大單（數百張以上）連續敲進
2. 確認五檔買盤也同步增加
3. 果斷追進（不要猶豫，猶豫就輸了）
4. 關鍵：沒有連續性走勢必須果斷停損

實例：

  • 情境：某股票在100元附近盤整，突然看到連續5筆以上100張大單敲進
  • 操作：100.5元追進，看到101、101.5持續成交，抱到102元出場
  • 如果進場後價格停滯、沒有繼續衝 → 立刻停損出場

適用時機：有主力作手操作的股票；不適合量太小的股票（容易被假突破）

趨勢流（日內波段）

是什麼：抓當日一段時間的趨勢，可能持倉幾十分鐘到整天。

怎麼做：
1. 開盤觀察今日多空方向
2. 等待回調不破關鍵價位時進場
3. 設好停損點（通常2-3%）
4. 抱住直到趨勢反轉或收盤前出場

實例：

  • 情境：台股指數開高走穩，電子股表現強勢
  • 操作：選擇強勢股如台積電，回調到平盤附近進場
  • 停損：跌破開盤價1%停損
  • 停利：抱到收盤或漲幅達3%

適用時機：趨勢明確的日子；不適合盤整盤（容易被左右打臉）

當沖停損三種設定方式

方式1：絕對金額停損

怎麼做：設定每天最大可接受虧損金額，達到就停手。

實例：

  • 每日設定最大虧損5,000元
  • 第一筆虧3,000元 → 還能繼續
  • 第二筆再虧2,000元 → 累計5,000元，今日收手

優點：簡單明確，容易執行
缺點：沒考慮股價波動特性

方式2：百分比停損

怎麼做：設定股價下跌一定百分比就停損。

實例：

  • 進場價100元，設定3%停損
  • 跌到97元 → 觸發停損，出場

建議比例：

  • 極短線：1-2%
  • 日內波段：2-3%
  • 不建議超過5%（回本難度大增）

方式3：觸價單停損

怎麼做：預先設定觸價設定觸價單，價格一到自動停損。

實例：

  • 進場價100元
  • 同時設定97元觸價賣出單
  • 股價跌到97元 → 系統自動賣出

優點：不用盯盤、避免人性猶豫
缺點：可能被插針後彈回（停損在最低點）

順勢單（追突破）

資金分配比例：3:2:1

  • 第一筆：30%資金試單
  • 確認方向對：加碼20%
  • 漲勢確立：最後加碼10%

實例：預計投入60萬

  • 第一筆：18萬（試水溫）
  • 確認突破：加12萬
  • 漲勢確立：加6萬

資金分配比例：1:2:3

核心知識

一、順勢交易與逆勢交易定義

順勢交易（右側交易）：

  • 「弱者恆弱、強者恆強」— 買的高，賣的更高
  • 確認行情階段性高點（或低點）已出現，在高點（或低點）的「右側」進場
  • 等到底部確立、趨勢確認才進場

逆勢交易（左側交易）：

  • 「物極必反、均值回歸」— 買低、賣高
  • 在行情潛在高點（或低點）的「左側」進場
  • 假設行情乖離到某種程度時會反轉

簡單理解：在「Ｖ」型走勢中，左側買入是逆勢，右側買入是順勢。

二、順勢交易實戰要點

是什麼：追隨已確認的趨勢，突破後才追漲，高點後才賣出。

怎麼做：
1. 確認趨勢：觀察均線斜率向上、價格突破關鍵位置
2. 進場時機：等待突破或回確認支撐後進場
3. 加碼策略：趨勢正確時逐步加倉，但前期低成本部位要掩護後面加倉
4. 出場時機：確認高點出現後才賣出

實例：

  • 情境：S&P500 2020年3月23日反轉
  • 順勢操作：等3/23後確認底部、T+n日沒再出現低點才進場
  • 風險：可能一次噴出錯過最佳進場點

優點：

  • 不會錯過大趨勢，單筆獲利極高（一次獲利可抵5-20次虧損）
  • 不會頻繁交易
  • 策略不易被市場淘汰

缺點：

  • 勝率低，洗刷次數多
  • 需忍受長期大幅回落
  • 若交易週期短，滑價影響大
  • 長則數年無大趨勢，需多市場交易

三、逆勢交易實戰要點

是什麼：假設均值回歸，判斷行情乖離到某種程度時做反向交易。

怎麼做：
1. 判斷乖離：使用技術指標（如布林通道、乖離率）判斷超買/超賣
2. 分批進場：越跌越買，分批建立部位（絕對不能一次All in）
3. 設停損：預設可接受的虧損範圍
4. 等待回歸：行情如預期反轉時獲利

實例：

  • 情境：股市從高點大跌20%，判斷超跌
  • 逆勢操作：分批買入權值股或ETF（0050、QQQ）
  • 加碼策略：每跌5%加碼一次，控制總部位
  • 風險：可能「越抄越心痛」，如2020年原油崩跌

優點：

  • 進在有利位置，勝率較高
  • 不易有滑價
  • 獲利頻率高，績效相對穩定
  • 可以建立比較好的成本

缺點：

  • 交易週期短、獲利幅度小，手續費成本高
  • 連續大虧損不易恢復
  • 平均單筆獲利低，靠交易次數撐大獲利
  • 需要一直盯盤，資金管理能力要求高

四、如何選擇適合自己的方式

順勢適合：

  • 一般投資者
  • 無法頻繁盯盤的人
  • 想要抓住大趨勢的人

逆勢適合：

  • 機構法人
  • 有時間盯盤、能做預判的人
  • 資金控管能力強的人

逆勢交易的資產選擇順序：
1. 指數型ETF（最安全）
2. 大型權值股、藍籌股（台積電、蘋果）
3. 一般個股（風險最高）

關鍵提醒：

  • 逆勢交易較違背人性（「別人恐懼時，我貪婪」）
  • 順勢交易較符合人性，但會慢一步
  • 兩者都對，關鍵在：總經/產業分析能力 + 部位控制 + 資金控管 + 心理素質

逆勢交易常見錯誤：
1. 在第一個看似轉折的地方就把資金用光
2. 過度預測行情，行情不如預期時產生超額虧損
3. 買到會倒的公司

順勢交易常見錯誤：
1. 部位頭重腳輕，一修正就跌出成本價
2. 洗刷過程中受不了停損出場

心法：

  • 股票漲跌無法掌握，但風險可以自己控制
  • 做好部位控制，才不會一看錯就被洗出場

五檔資訊基礎概念

什麼是五檔？

定義：離目前成交價最接近的5個價位，分別有多少買家和賣家

組成：

  • 委買價（內盤）：買方掛單價格，從高到低排列
  • 委賣價（外盤）：賣方掛單價格，從低到高排列
  • 掛單張數：每個價位上的等待成交數量

tick 概念：

  • 股價跳動單位，例如 100-500 元股價的 tick 為 0.1 元
  • 110 元往上漲一檔是 110.1，往下跌一檔是 109.9
  • 對當沖來說，1 tick 的價差影響巨大

內盤與外盤

內盤（委買價成交）

是什麼：成交在「委買價」，計入內盤

代表意義：

  • 賣方較為積極，願意降價求售
  • 賣方看跌，急著出脫股票
  • 內盤成交量大 → 股價下跌機率高

外盤（委賣價成交）

是什麼：成交在「委賣價」，計入外盤

代表意義：

  • 買方較為積極，願意追價買進
  • 買方看多，無論如何都要買到
  • 外盤成交量大 → 股價上漲機率高

內外盤比

計算方式：

  • 內盤比 = 內盤成交量 ÷ 總成交量
  • 外盤比 = 外盤成交量 ÷ 總成交量

判讀：

  • 內盤比 > 外盤比：賣壓沉重，偏空
  • 外盤比 > 內盤比：買盤積極，偏多

注意：內外盤比合計不一定剛好 100%，因為有極少數成交價不落入委買/委賣價

判斷市場氣氛

怎麼做：
1. 觀察買賣雙方掛單數量比例
2. 買方掛單明顯多於賣方 → 上漲潛力
3. 賣方掛單明顯多於買方 → 下跌壓力

識別關鍵價位

怎麼做：
1. 找出大量買單聚集的價位 → 可能是支撐位
2. 找出大量賣單聚集的價位 → 可能是阻力位
3. 這些價位是主力可能防守或攻擊的位置

預測短期走勢

怎麼做：
1. 高價位買單突然增加 → 可能即將上漲
2. 低價位賣單突然增加 → 可能即將下跌
3. 掛單快速變化要注意主力動向

把握入場時機
