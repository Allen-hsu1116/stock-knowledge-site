一、區間震盪操作策略

適用情境：大盤無明確趨勢，指數呈現上下邊界來回震盪

核心觀念：高出低進的區間交易邏輯，不追高殺低

VIXTWN 恐慌指數應用：

  • 是什麼：台指選擇權隱含波動率，反映市場對未來 30 日波動預期
  • 怎麼用：

1. VIXTWN 突破 30 以上 → 高恐慌 → 進場布局
2. VIXTWN 回落至 15-18 → 低恐慌 → 減碼
3. VIXTWN 與加權指數呈負相關

  • 實例：
  • 數據：VIXTWN 突破 30 進場，3 個月後平均上漲 13.7%，6 個月後平均上漲 19.7%

二、技術面抗跌股篩選

核心邏輯：題材是誘因，技術面抗跌力才是切入關鍵

篩選條件：

條件一：大盤破底但個股未破底

  • 顯示個股相對大盤更具支撐力
  • 資金未撤出，甚至有護盤跡象
  • 實例：2025 年初至 4/9 大盤恐慌下殺，驊宏資、華經、寶一、晟田皆未隨大盤破底

條件二：站穩關鍵均線

  • 月線（20MA）：短中期趨勢轉強
  • 季線（60MA）：具備波段反彈潛力
  • 股價有效站回 = 買盤力道強

進出場策略：

  • 依據道氏理論趨勢通道操作
  • 找出技術抗跌股 + 確認題材支撐 → 短波段操作

Tick流：

  • 捕捉股價微小波動
  • 每次進出場間隔數秒或數分鐘
  • 適用於頻繁震盪且成交量活躍的標的

一波流：

  • 掌握當日震盪最大的短期趨勢
  • 追蹤成交量變化與大單流向
  • 操作標的通常為熱門族群與題材股

日內波：

  • 研判個股有機會出現長紅K或長黑K才進場
  • 通常等趨勢確立後進場，接近收盤時出場
  • 非常注重選股策略，需基本面/技術面/籌碼面篩選

四、日內交易各時段特點

開盤前：

  • 觀察昨日美股收盤、盤前海外期貨走勢
  • 美期貨盤、日韓股開盤、台指期開盤
  • 決定偏多或偏空操作方向

開盤後（9:00-9:30）：

  • 市場劇烈反應消息、短線籌碼變化
  • 「快攻時間」，短進短出機會多
  • 部分強勢股會快速搶攻漲停板

早盤（9:30-10:30）：

  • 標的走勢逐漸明朗
  • 技術型態與市場動態提供操作方向

中盤（10:30-12:30）：

  • 成交量下降，波動縮小
  • 宜減少出手次數

尾盤（12:30-13:30）：

  • 當沖交易者準備出場
  • 波段交易者下定決心進場
  • 成交量擴增，股價波動加劇
  • 注意拉尾盤或下殺變化

五、短線關鍵技術指標

當日開盤價：

  • 股價在開盤價上方 = 紅K棒
  • 股價在開盤價下方 = 黑K棒
  • 開盤價為日內走勢多空分界線
  • 行情強勁的股票開盤後會迅速脫離開盤價

當日均價線：

  • 當天的平均交易成本
  • 價格維持在上方 = 多方趨勢明確，逢拉回可買進
  • 價格維持在下方 = 空頭佔優勢，逢反彈可放空

日K線關鍵點位：

  • 近期高低點、昨日收盤價易形成支撐或壓力
  • 突破或跌破後易產生趨勢

日K線重要技術指標：

  • 均線、布林通道
  • 上揚的均線容易成為支撐
  • 下彎的均線則易成為壓力

實戰案例 - 凱美（2375）：

  • 開盤不久後成交量顯著增加
  • 股價突破昨日高點 → 進場時機
  • 或待拉回不破均價線確認下檔有撐後再布局
  • 最終收在漲停價

實戰案例 - 聯光通（4903）：

  • 跳空開高但遇 20 日均線反壓
  • 早盤攻高無力後跌落均價線
  • 反彈至均價線上但量能未放大
  • 走勢延續性轉弱 → 不適合作多，可逢彈放空

六、勝率與賺賠比

勝率：獲勝機率

  • 10 次交易中有 7 次賺錢 → 勝率 70%

賺賠比：承擔每單位風險能獲得多少報酬

  • 公式：平均獲利 ÷ 平均虧損
  • 例：平均獲利 30 元，平均虧損 10 元 → 賺賠比 = 3

期望值公式：

期望值 = (賺賠比 × 勝率) - (1 - 勝率)

實例：

  • 勝率 40%，賺賠比 3
  • 期望值 = 3 × 0.4 - 0.6 = 0.6
  • 每次交易平均獲利 0.6 元 → 正期望值策略

七、波段操作核心觀念

股票 ≠ 房地產：

  • 房地產長期往上，股票不一定
  • 股票可能被淘汰、下市
  • 20 年前的前 50 大股跟現在不一樣

三大原則：
1. 不要隨便賣：下跌時亂賣容易賣在低點
2. 保留現金：下跌時不要加碼攤平
3. 學會停損：越早停損賠越少

賠錢時不加碼攤平：

  • 越買賠越多，攤也攤不平
  • 看好某檔股票 → 轉空時先賣出，等止跌再逢低買回

40 日均線控盤法：

  • 40 日均線 ≈ 2 個月交易日
  • 主力吃貨平均需 2 個月
  • 怎麼用：

1. 均線往上 → 做多
2. 均線往下 → 做空或空手
3. 跌破 40 日均線 → 停損出場

  • 實例 - 台積電：跌破 40 日均線後一路往下，沒有持有必要
  • 實例 - 聚積：跌破 40 日均線後持續下跌，應在漲過均線時持有

八、期貨留倉六大注意事項

1. 只在賺錢時留倉

  • 賠錢時留倉 = 賭隔天會賺回來
  • 如果繼續跌 → 越賠越大
  • 一開始賠錢就該停損

2. 事先設定出場點

  • 包含日盤及夜盤
  • 進場前先設停損
  • 獲利後設停利
  • 不要讓賺錢的單變賠錢

3. 跳空避險

  • 用選擇權買方鎖住獲利
  • 期貨做多 → 加買選擇權 BUY PUT 避險
  • 期貨做空 → 加買選擇權 BUY CALL 避險
  • 用賺的錢買選擇權
  • 週選擇權比月選擇權便宜

4. 接受行情可能走不出去

  • 行情總是盤整很久才有一段趨勢
  • 留倉做多遇到盤整 → 買反向選擇權避險
  • 行情下跌 → 期貨賠但選擇權賺
  • 行情上漲 → 期貨賺比選擇權賠更多

5. 順勢留倉 + 慎選進場點

  • 行情偏多 → 留多單
  • 行情偏空 → 留空單
  • 進場點不好 → 反彈就停損
  • 例：1200 點跌幅中，200-300 點震幅合理，但短線進場點差會被掃出

6. 停損不要設太小

  • 20-30 點太小，現在行情波動 200-300 點
  • 建議用 ATR 的 1/4 或 1/5 設定停損
  • 例：ATR 219 點 → 1/5 約 44 點，設 50 點停損
  • 海龜交易法則：停損設很大，用大資金守住風險控管

九、均線波段交易實戰

N 日均線波段交易：

  • 價格跌破 N 日均線 → 進場做空
  • 價格突破 N 日均線 → 進場做多
  • 持續抱波段直到反向訊號出現

30 分K + 均線操作：

  • 用 60MA 作為移動停利基準
  • 設定與均線保持一段距離（如 40 點）
  • 漲過均線進場做多
  • 移動停利線會跟著部位走
  • 行情打到移動停利線 → 獲利了結出場

設定建議：

  • 不要設定完全剛好的條件
  • 用 60MA 時，不要設定碰到 60MA 就出場
  • 行情可能只是回檔後又恢復
  • 保持一段距離避免被掃出去

心得

今天學的是操作策略，三篇文章各有側重：

聯合理財網那篇給了我一個很實用的恐慌指數策略。VIXTWN 突破 30 進場的數據很漂亮——3 個月平均漲 13.7%、6 個月平均漲 19.7%。這種「高恐慌時進場」的邏輯跟巴菲特說的「別人恐懼時貪婪」一樣，但這篇給了具體數字。另外「大盤破底個股不破底」這個篩選條件也很實用，直接告訴你哪些股票有主力在護盤。

玉山證券那篇把短線交易拆解得很清楚。三種模式（Tick流、一波流、日內波）各有適用場景，賺賠比也不同。最關鍵的是日內時段分析——開盤 30 分鐘是「快攻時間」，中盤要減少出手，尾盤一小時才是真正的決戰點。這種時段劃分對實際操作很有幫助。

WinSmart 那篇則是一針見血地指出「股票 ≠ 房地產」這個誤區。股票不會一直漲，還可能下市。40 日均線控盤法簡單實用——均線往上做多、往下做空或空手。期貨留倉六個注意事項也都很實際，尤其是「只在賺錢時留倉」和「跳空用選擇權避險」這兩點。

最後：波段操作的精髓就是「順勢持有到轉向」。但判斷轉向是難點，40 日均線提供了一個客觀標準。短線交易則要看時段、看價量、看技術位，快攻時間要果斷，中盤要忍手。
