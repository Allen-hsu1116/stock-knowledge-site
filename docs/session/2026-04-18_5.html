K線三招判讀法——顏色、實體、影線的實戰解讀

是什麼：K線（蠟燭線）由開盤價、收盤價、最高價、最低價四個價格組成。讀K線不是看一根猜漲跌，而是用三個維度拆解多空力道：顏色判方向、實體判強度、影線判阻礙。

怎麼做：

1. 顏色判方向：

  • 紅K（收 > 開）＝多方勝利，黑K（收 < 開）＝空方勝利

2. 實體判趨勢強度：

  • 實體長（影線佔整體長度 < 20%）→ 趨勢明確，延續性高

3. 影線判多空交戰過程：

  • 長實體 + 短影線 → 趨勢延續性高（強勢）

實例：

  • 情境：某股今天收紅K，開100收105，最高110最低99
  • 實體長度 = 5元，上影線 = 5元，下影線 = 1元，總長度 = 11元
  • 影線佔比 = (5+1)/11 = 54.5% > 50% → 多空激烈交戰，趨勢不明
  • 上影線5元遠大於下影線1元 → 賣壓遠大於買盤支撐
  • 結論：雖然是紅K，但不是強勢紅K，上方壓力大，追高風險高

適用時機：每天看K線時的基本功。不要只看漲跌（顏色），要拆解實體和影線比例判斷力道。

16種K線型態——6種必懂的實戰型態

是什麼：K線型態太多記不完，但以下6種是實戰中出現頻率最高、訊號最明確的。

怎麼做：

1. 大紅K（長陽線）：

  • 特徵：實體長、影線極短（< 20%）

2. 大黑K（長陰線）：

  • 特徵：實體長、影線極短

3. 鎚子線（紅K鎚子）：

  • 特徵：實體短、無上影線、下影線 ≥ 實體2倍

4. 墓碑線（倒鎚黑K）：

  • 特徵：實體短、無下影線、上影線 ≥ 實體2倍

5. 十字線（Doji）：

  • 特徵：開盤價 ≈ 收盤價，有長上下影線

6. T字線 / 倒T字線：

  • T字線（開=高=收，長下影線）：開盤被殺後買盤大量接手拉回。低檔出現＝空頭結束

實例：

  • 情境：某股從200元連跌5天至170元，第6天出現鎚子線（開170收171.5，最低165，下影線5元）
  • 判讀：下跌末端出現鎚子線，空方殺到165但被多方拉回171.5
  • 操作：隔天若站穩172元以上，可進場做多，停損設165
  • 反例：同樣的鎚子線如果出現在股價從100漲到200之後，那不是鎚子線是「吊人線」，是賣訊

適用時機：每根K線收盤後判讀。但單一根K線訊號有限，必須搭配位置（高檔/低檔）和前後K線組合判斷。

W底與M頭——型態學最實用的兩把刀

是什麼：W底（雙重底）和M頭（雙重頂）是型態學中最常見、最可靠的轉折型態。不是看到W就買、看到M就賣，關鍵在「頸線突破/跌破」才是確認訊號。

怎麼做：

1. W底（看多）：

  • 形成過程：急跌 → 反彈（左腳）→ 回落量縮（右腳）→ 突破頸線量增

2. M頭（看空）：

  • 形成過程：急漲 → 回落（左肩）→ 反彈量縮無法過高（右肩）→ 跌破頸線帶量下殺

3. 假突破判斷：

  • 突破頸線但3天內跌回頸線以下 → 假突破，不要追

實例：

  • W底實例：某股從100元跌到80元反彈至90元（左腳+頸線），再回落至82元（右腳），帶量突破90元頸線
  • 買進價：90~91元（頸線附近）
  • 目標價：90 + (90-80) = 100元
  • 停損：跌破87元（右腳低點）
  • M頭實例：某股從80元漲到100元回落至90元（左肩+頸線），反彈至98元（右肩，量縮不過高），帶量跌破90元頸線
  • 賣出價：89~90元（頸線附近）
  • 目標價：90 - (100-90) = 80元
  • 做空停損：站回93元以上

適用時機：日K或週K級別的波段操作。不適合：分鐘K級別（雜訊太多），或不成熟的W/M型態（只有一個高/低點）。

1. 單一均線法（最簡單實用）：

  • 根據操作週期只保留一條均線：

2. 黃金交叉與死亡交叉（雙均線法）：

  • 黃金交叉：短期均線向上穿越長期均線 → 買進訊號

3. 多空排列判斷大趨勢：

實例：

  • 操作：股價回測20MA附近（如從110回落到102~103元附近）不破 → 進場做多

支撐壓力線——用K線高低點找買賣點

是什麼：支撐線是價格停止下跌並反彈的水平，壓力線是價格停止上漲並回落的水平。供需關係的產物——低檔買方多於賣方產生支撐，高檔賣方多於買方產生壓力。

怎麼做：

1. 畫支撐線：

  • 在K線圖上找至少2波的低點相連，向右延伸

2. 畫壓力線：

  • 在K線圖上找至少2波的高點相連，向右延伸

3. 支撐壓力互換原則：

  • 突破壓力線後，壓力線變成支撐線（上方壓力解除，回測有撐）

4. 突破確認三原則：

  • 帶量突破（量比 > 1.5）

5. 交易策略：

  • 支撐線買進：價格回測支撐線有守 → 買進，停損設支撐線下方1~2%

實例：

  • 情境：某股近3個月在80元（支撐）到95元（壓力）區間震盪
  • 操作1：股價回落至80~81元附近，前4次都撐住 → 買進，停損78元
  • 操作2：股價反彈至93~95元，前3次都過不去 → 賣出，目標80元
  • 操作3：股價帶量突破95元，3天站穩97元以上 → 回測95元時加碼買進（壓力變支撐）
  • 操作4：股價跌破80元，3天收在78元以下 → 反彈到80元時賣出（支撐變壓力）

適用時機：所有時間框架的交易。支撐壓力是技術分析最基礎也最實用的工具。不適合：單邊趨勢行情（支撐壓力不斷被突破，畫線意義有限）。

心得

這回合補強了本週最缺的「技術分析」基礎——K線判讀、型態學、均線和支撐壓力。說實話，這些東西我之前學當沖和選擇權的時候一直在用，但沒有系統性地整理過。

幾個重要的認知更新：

1. K線不是只看顏色：這大概是散戶最常犯的錯。紅K不代表一定看漲，如果影線佔比超過50%，那根紅K其實是「多方險勝」，不是強勢。墓碑線就是最好的例子——開盤後拉一大段，結果被空方打到收黑，雖然有上影線代表過程中多方有攻擊，但結果是空方贏了。

2. 鎚子線 vs 吊人線是同一根K線：差別在位置。低檔叫鎚子線（看多），高檔叫吊人線（看空）。這提醒我一個重要原則：同樣的型態在不同位置，意義完全相反。不能死背型態不考慮位置。

3. 支撐壓力互換是核心心法：這東西很多書都提過，但實戰中真的很有用。突破壓力後回測就是買點，跌破支撐後反彈就是賣點——這比任何指標都直觀。關鍵是要確認「真突破」，帶量 + 收盤站上 + 3天確認，三個條件缺一不可。

4. 均線只要一條就夠：以前看圖都掛6條均線，花花綠綠的反而看不清楚。現在明白根據操作週期選一條，配合支撐壓力線使用，比看均線交叉準多了。均線是落後指標，等交叉出現已經漲/跌一大段，用單一均線判斷「股價在均線上還是下」才是最直接的趨勢判斷。

5. W底和M頭的目標價算法很實用：頸線到腳/肩的距離等幅測量，這給了一個具體的出場目標，不是憑感覺亂設。搭配支撐壓力互換原則，可以制定完整的進出場計畫。
