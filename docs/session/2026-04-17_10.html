風險管理

第 10 回合學習 - 風險管理

時間：15:30

  • 關於選擇權的風險係數Delta，一些你需要知道的事

Delta 中性避險（Delta Neutral Hedging）

是什麼：透過調整選擇權與現貨/期貨部位，讓整體組合的 Delta 值接近零，消除方向性風險，讓你不靠猜漲跌也能賺錢。

怎麼做：
1. 計算部位總 Delta：每個部位的 Delta × 口數 × 乘數（台股選擇權乘數為 50）

  • 例：1 口小台多單 Delta = +1，2 口價平賣權 Delta = -0.5 × 2 = -1，總 Delta = 0

2. 建立對沖部位：用反向 Delta 工具抵消風險

  • 持有多頭部位 → 買 Put 或賣 Call 來降低 Delta

3. 動態再平衡：市場變動後 Delta 會偏離零，需定期調整（每日或每週）

  • 標的上漲 → 總 Delta 變正 → 賣出部分現貨或平倉部分 Call

實例：

  • 情境：持有 1 口小台期貨多單（Delta = +1），加權指數 20,000 點
  • 避險：買進 2 口 20,000 Put（價平，Delta 各 -0.5），總 Delta = 1 + (-0.5×2) = 0
  • 指數跌到 19,500：小台虧 250 點 × 50 = -12,500，Put 賺約 125 點 × 2 × 50 = +12,500，幾乎抵消
  • 也可以只買 1 口價外 Put（如 19,500 履約價），不是完全避險而是「保險型避險」

適用時機：

  • 大事件前不想猜方向（選舉、FOMC、財報）→ Delta 中性保平安
  • 長期持股短期擔憂 → 不賣股，用選擇權對沖方向風險
  • 不適合：懶得盯盤的人（需要動態調整）、小資金（交易成本會吃掉利潤）

風險：

  • Gamma 風險：標的大幅波動時 Delta 快速偏離，調整不及就虧
  • 交易成本：頻繁再平衡的滑點和手續費會侵蝕利潤
  • Theta 損耗：買選擇權避險要付時間價值

Gamma Scalping（Gamma 剝頭皮交易）

是什麼：利用做多 Gamma 的部位（如買進跨式 Straddle），在標的價格上下波動時，透過動態調整 Delta 來高拋低吸，從波動本身獲利，不靠方向。

怎麼做：
1. 建倉：買進平值 Call + 平值 Put（Long Straddle），初始 Delta 接近零，Gamma 為正
2. 價格上漲時：Call 變實值 Delta 上升，Put 變虛值 Delta 接近零 → 總 Delta 變正

  • 調整：平倉部分獲利的 Call，或加買便宜的 Put，讓 Delta 回到零

3. 價格回落時：Put 變實值，Call 變虛值 → 總 Delta 變負

  • 調整：平倉部分獲利的 Put，或加買便宜的 Call，讓 Delta 回到零

4. 反覆循環：每次調整都在「高拋低吸」，獲利逐步沉澱
5. Vega 對對沖：賣出遠月平值選擇權來對沖 Vega 風險（近月 Gamma 大、遠月 Vega 大）

實例：

  • 情境：台指期 20,000 點，買進 1 口 20,000 Call + 1 口 20,000 Put，成本各 300 點，總成本 600 點 × 50 = 30,000 元
  • 漲到 20,300：Call 漲到 500（+200），Put 跌到 150（-150），組合浮盈 +50 點 × 50 = +2,500
  • 調整：平倉 0.3 口 Call 鎖定利潤，或加買 Put 讓 Delta 回零
  • 跌回 20,000：之前調整讓 Delta 回零，Put 回漲的利潤又入袋
  • 每次波動都能收割一次，但如果行情一潭死水，Theta 每天吃掉你權利金

適用時機：

  • 預期大震盪但不知道方向（如財報季、央行會議前）
  • 低波動率環境建倉（IV 低時買選擇權便宜）
  • Gamma/Theta 比值高時（每承受 1 元時間損耗，能賺回更多 Gamma 利潤）

不適用：

  • 低波動盤整市（Theta 會慢慢把你吃死）
  • 沒時間盯盤的人（需要頻繁調整 Delta）
  • 不懂 Greeks 的人（這是進階策略）

風險：

  • Theta 成本：每天時間價值損耗是固定支出，波動不夠大就虧
  • Vega 風險：IV 下降時整體部位貶值，需要遠月賣權對沖
  • Gamma 在價平最大，遠離價平後效果遞減

GEX（Gamma Exposure）造市商避險籌碼判讀

是什麼：GEX 是市場上所有選擇權部位的 Gamma 加總，反映造市商為了避險必須在期貨市場做的動作。正 GEX = 造市商「低買高低買高賣」穩定市場；負 GEX = 造市商「追漲殺跌」加速波動。

怎麼做：
1. 判斷 GEX 正負：

  • +GEX（正 Gamma 曝險）：造市商賣 Call + 賣 Put 為主 → 指數上漲時賣期貨壓盤，下跌時買期貨撐盤 → 市場黏滯

2. Zero Gamma 關鍵點位：GEX 從正轉負的價位 = 多空分水嶺

  • 指數在 Zero Gamma 上方 → +GEX 區 → 安心做做賣方（鐵兀鷹、價差）

3. GEX vs OI 差異：

  • OI（未平倉量）是靜態累積數據，包含殭屍部位，反應遲鈍

實例：

  • 情境：加權指數 20,500，GEX 數據顯示 Zero Gamma 在 20,200
  • 指數在 20,500（+GEX 區）：造市商低買高賣 → 市場狹幅震盪 → 適合鐵兀鷹收租
  • 指數跌破 20,200（-GEX 區）：造市商從支撐者變砸盤者 → 可能出現長黑 → 轉做買方或空單
  • 2020 年 3 月新冠崩盤：大量 Buy Put 湧入 → 造市商被迫 Sell Put → -GEX → 被迫放空期貨 → 加速暴跌

適用時機：

  • 每日開盤前查看 GEX 圖，判斷今天盤勢屬性
  • 大事件後判斷市場是否進入負 GEX 區（恐慌訊號）
  • 搭配 P/C Ratio 和 OI 做多維度籌碼判讀

不適用：

  • GEX 數據取得需要付費工具或自行計算
  • GEX 是即時動態的，過期數據無效

心得

Delta 中性避險跟 Gamma Scalping 是選擇權風控的兩面：前者是防守，後者是進攻。Delta 中性讓你在不想猜方向時保住部位，Gamma Scalping 則是在波動中主動收割利潤。

關鍵認知：Delta 中性不是一次性的，是動態的。市場在動，Delta 就在動，你不調整就等於在裸露方向性風險。這跟很多人以為「對沖完就安全了」的觀念完全不同。

GEX 這個概念更狠——它讓你看穿造市商的底牌。正 GEX 時造市商幫你穩盤，你安心收租；負 GEX 時造市商也在砸盤，你還在做賣方就是找死。這比單看 OI 高明太多，OI 告訴你哪裡有牆，GEX 告訴你牆會不會倒。

三個策略的風控共同點：都需要持續監控和動態調整。靜態避險在選擇權世界裡不存在，這是跟股票操作最大的差異。在股票世界你可以買了放著，但在選擇權世界，不管不顧等於慢性自殺。
