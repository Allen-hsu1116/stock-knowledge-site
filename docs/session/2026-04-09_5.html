判斷多空的五大口訣

雙均線判斷行情多空

是什麼：
用兩條均線（短天期+長天期）判斷當前趨勢強弱

怎麼做：
1. 日K圖上設定兩條均線（例：5MA + 20MA，或10MA + 40MA）
2. 觀察K線位置：

  • 強多盤：價格同時在短均和長均上方

實例：

  • 廣達(2382)：日K價格在5MA和20MA上方，紅K較多，屬於強多盤
  • 技嘉(2376)：價格在所有均線上方，賺錢天數比例較高
  • 光寶科(2301)：跌破短均後代表轉弱

適用時機：

  • 做當沖前必須先看日K，確認目前趨勢位置
  • 強多盤時做多勝算較高，強空盤時做空勝算較高

今開昨收定多空

是什麼：
用「今日開盤價」和「昨日收盤價」兩個關鍵價位，判斷盤中即時多空

怎麼做：
1. 用今日開盤價判斷：

  • 紅K定義：收盤價 > 開盤價

2. 用昨日收盤價判斷：

  • 價格在昨收之上 → 今天上漲

實例：

  • 景碩(3189)：漲幅2.23%，價格在昨收之上
  • 華邦電(2344)：開盤往上跳高，價格在昨收之上，屬於強勢盤
  • 廣明(6188)：一根長紅K配一根長黑K，跌破今開也跌破昨收，雙重偏弱

適用時機：

  • 盤中即時判斷多空方向
  • 短線進出場點確認
  • 判斷當天K線強弱

三個關鍵價位觀察

核心概念：
盤中要隨時掌握三個價格：昨天收盤價、今天開盤價、現在價格

判斷邏輯：
1. 價格跌破昨收 → 今天偏弱
2. 價格跌破今開 → 當下走弱（開高走低的跡象）
3. 價格在昨收和今開之下 → 確認偏弱格局

強中透弱的識別：

  • 第一根紅K：收在開盤價之上 → 偏強
  • 隔天黑K：在昨收之上，但跌破今開 → 開高走低，強中透弱
  • 第三根黑K：價格在昨收之下 → 完全走弱

籌碼判斷行情多空

是什麼：
觀察市場買賣力道和大戶籌碼方向，判斷盤中多空

怎麼看：
1. 買賣力道：市場上全部買盤 vs 全部賣盤的加總比較

  • 買盤 > 賣盤 → 紅色柱狀體，買盤較強

2. 大戶籌碼：觀察大戶是用追價成交還是追賣成交

  • 追價成交 → 買盤較強

實例：

  • 買賣力道和大戶籌碼都出現紅色 → 價格容易往上爬
  • 兩者都呈現綠色 → 價格容易往下跌

心得

這回合學到的是當沖的核心判斷框架，特別是「今開昨收定多空」這個方法非常實用。

三個關鍵收穫：

1. 先看日K再看盤中：很多人做當沖只看分K，忽略了日K的大方向。先確認價格在均線上方或下方，決定順勢方向，勝算自然提高。

2. 開盤價是當天的多空分界線：這個觀念很簡單但很強大。紅K = 收 > 開，黑K = 收 < 開。盤中只要跌破開盤價，就要警覺可能收黑K。

3. 籌碼顏色確認方向：買賣力道和大戶籌碼同時翻紅或翻綠，是很好的方向確認訊號。單一指標可能有雜訊，但兩者同步時可信度更高。
