核心知識

配對交易（Pairs Trading）是什麼？

定義：一種市場中性策略，同時操作兩個高度相關的標的，一邊做多、一邊做空，透過價差收斂或發散獲利，不受市場漲跌影響。

核心概念：

  • 市場上漲時：多頭部位賺錢、空頭部位賠錢，淨效果取決於相對表現
  • 市場下跌時：空頭部位賺錢、多頭部位賠錢，同樣取決於相對表現
  • 無論市場漲跌，只要兩者價差收斂就能獲利

起源：1980年代中期由摩根士丹利技術分析研究人員提出，利用統計和技術分析尋找市場利潤。

配對交易兩種獲利方式

1. 價差收斂獲利（主流方式）

是什麼：兩個高度相關標的短期價差放大，長期會回歸一致，趁價差異常時進場。

怎麼做：
1. 選擇兩個高度相關的標的（如可口可樂 vs 百事可樂）
2. 當價差放大到歷史極端時
3. 放空價格偏高的標的 + 做多價格偏低的標的
4. 等待價差收斂，平倉獲利

實例：

  • 可口可樂（KO）和百事可樂（PEP）長期走勢高度相關
  • 當 KO 相對 PEP 價格偏高時
  • 操作：放空 KO、做多 PEP
  • 等待兩者價格差距縮小

2. 價差發散獲利（變化型）

是什麼：選擇兩個相關但有明顯強弱之分的標的，買強賣弱，長期賺取強弱差異。

怎麼做：
1. 選擇兩個相關但強弱不同的標的
2. 買進強勢標的 + 放空弱勢標的
3. 上漲時：強的漲多，弱的漲少
4. 下跌時：強的跌少，弱的跌多
5. 無論漲跌都能賺到強弱價差

實例：

  • 做多台積電（2330）+ 放空台指期（代表整體大盤）
  • 台積電長期表現優於大盤
  • 無論市場漲跌，只要台積電相對大盤強就能獲利

注意：這不是傳統主流的配對交易，是作為多空對沖的一種模式。

配對交易的重要特性

1. 需要兩倍資金，獲利卻只有價差

影響：

  • 同時做多、做空，需要兩份資金
  • 只賺取兩者之間的價差，報酬相對較小
  • 扣除手續費、交易稅、融券利息後，利潤更薄

解決方式：

  • 大多使用期貨等衍生性金融商品
  • 槓桿放大後，0.2%價差放大10倍 = 2%報酬
  • 若策略20天收斂，20天獲利2%

2. 價差收斂時間不確定

風險：

  • 可能1天收斂，也可能數月不收斂
  • 時間越長，機會成本越高
  • 收斂時間不可控

3. 價差不一定會收斂

風險：

  • 歷史相關性不代表未來
  • 其中一檔可能面臨特殊事件（公司出問題）
  • 相關性發生變異，策略可能失敗

4. 價差可能繼續發散

風險：

  • 預期收斂但實際發散
  • 若使用槓桿，可能保證金不足被迫停損
  • 一次意外就可能造成永久性損傷

配對交易如何執行？

步驟一：選擇配對標的

原則：

  • 選擇高度相關的股票
  • 考慮流動性（小股票難以放空）
  • 選擇有規模的企業

常見配對：

配對類型範例
同業競爭可口可樂 vs 百事可樂
同業競爭福特汽車 vs 通用汽車
同業競爭Visa vs Mastercard
類似產業台積電 vs 大盤指數

一籃子配對：

  • 做多一籃子股票 + 放空另一籃子股票
  • 分散單一公司風險
  • 確保收斂可能性提高
  • 但價差放大的機會減少

步驟二：判斷價差進場時機

方法一：歷史經驗判斷

怎麼做：
1. 統計歷史價差變化
2. 找出適合進場的價差大小
3. 當價差達到歷史極端時進場

方法二：布林通道判斷

怎麼做：
1. 計算兩檔股票的價格比例
2. 使用布林通道（如 200MA，2.3 標準差）
3. 當價格比例穿越通道上緣後向下突破 → 建立配對部位
4. 價格比例回歸中線 → 平倉

實例（Tesla vs GM）：

  • 2017/6/26：TSLA/GM 比例到達通道上緣
  • 建立：賣出 TSLA $377.5 x 100股，買入 GM $34.5 x 1100股
  • 2017/9/27 平倉：TSLA 跌到 $341，GM 漲到 $40.58
  • 獲利：$3,650 + $6,688 = $10,338

方法三：共整合檢定（進階）

怎麼做：
1. 計算兩檔股票價差序列
2. 使用 ADF 單根檢定確認定態性
3. 若價差序列定態，代表具有共整合關係
4. 用均值 ± 標準差設定進出場門檻

實例（長榮 vs 陽明）：

  • 形成期：計算價差序列的 μ 均值與 σ 標準差
  • 開倉條件：價差超過 μ ± 1.5σ
  • 平倉條件：價差回到 μ ± 0.2σ
  • 停損條件：價差突破 μ ± 2.5σ

步驟三：風險管理

風險控制要點：

風險控制方式
價差不收斂設定時間限制，超過期限停損
價差持續發散設定停損點（如 μ ± 2.5σ）
保證金不足控制槓桿倍數，預留緩衝
單一公司風險使用一籃子股票分散

配對交易的優缺點

優點

優點說明
對沖系統風險市場大跌時，空頭部位獲利抵消多頭虧損
不需判斷方向無論市場漲跌都能獲利
勝率高價差收斂是高機率事件
損益曲線平滑無劇烈的大賺大賠（無槓桿時）

缺點

缺點說明
盈虧比極低小賺大賠，一次大賠可能抵消多次小賺
資金成本高需要兩倍資金，交易成本也是兩倍
執行困難需要程式能力、資料源、數學運算能力
放空成本高小股票難以放空，融券利息高
槓桿風險為了提高報酬常使用槓桿，風險放大

配對交易的脆弱性

關鍵特性：

  • 高勝率、小賺大賠
  • 類似選擇權賣方策略
  • 一次意外可能造成嚴重傷害
  • 容錯率極低

市場先生觀點：

  • 配對交易是「脆弱性」策略
  • 遭遇意外一次，可能永久性損傷
  • 若要使用，只佔資產一小部分
  • 不建議重押

實戰範例：長榮 vs 陽明配對交易

情境：2021年1月至3月，海運股波動劇烈

設定：

  • 形成期：2020年1月至12月
  • 交易期：2021年1月至3月
  • 開倉門檻：μ ± 1.5σ
  • 平倉門檻：μ ± 0.2σ
  • 停損門檻：μ ± 2.5σ

策略邏輯：
1. 價差上穿 μ + 1.5σ → 做空配對（賣出陽明、買進長榮）
2. 價差回到 μ + 0.2σ → 平倉
3. 價差下穿 μ - 1.5σ → 做多配對（買進（買進陽明、賣出長榮）
4. 價差回到 μ - 0.2σ → 平倉
5. 價差突破 μ ± 2.5σ → 緊急停損

結果檢討：

  • 策略有兩次最大回檔跌破8%
  • 海運股2021年上半年波動極大
  • 形成期參數可能失靈
  • 建議用 rolling 方式動態調整 μ 和 σ

心得

配對交易是個很酷的概念：「不用猜市場方向，只要猜兩個標的會不會回到正常關係」。

最有價值的是「市場中性」的概念。傳統投資要賺錢必須市場上漲，但配對交易可以對沖掉系統風險。大盤大跌時，做多部位賠錢但放空部位賺錢，淨效果取決於相對表現。

但這策略有個致命傷：盈虧比極低。小賺多次，一次大賠可能全吐回去。就像選擇權賣方，平常穩穩賺時間價值，遇到黑天鵝就爆炸。

實戰上有幾個困難點：
1. 找到高度相關且長期維持相關性的標的很難
2. 放空小股票成本高、流動性差
3. 需要程式能力和數學統計能力
4. 價差收斂時間不確定，機會成本難估

TEJ 的實作文章提到用共整合檢定來確認配對關係，這是更嚴謹的做法。單純看相關係數可能被誤導，因為相關性會隨時間變化。
