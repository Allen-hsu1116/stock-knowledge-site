1. K線型態學實戰

核心概念：透過大量歷史統計彙整，用眼睛辨識K線組合成的圖形樣態，判斷市場趨勢方向。

反轉型態：底部型態（買進訊號）：

型態判斷要點目標價公式停損設定
W底第2次低點高於第1次，突破頸線需放量頸線 + (頸線 - 底部)跌破頸線3%
頭肩底頭低於兩肩，突破頸線需放量頸線 + (頸線 - 底部)跌破頸線

反轉型態：頭部型態（賣出訊號）：

型態判斷要點目標價公式停損設定
M頭兩高點差不多等高，跌破頸線確認頸線 - (頭部 - 頸線)突破頸線3%
頭肩頂頭高於兩肩，跌破頸線確認頸線 - (頭部 - 頸線)突破頸線

單根K線型態：

看漲型態特徵操作
錘形線短實體+長下影線，下降趨勢底部隔日開高收紅確認
多頭吞噬紅K完全吞沒前一根綠K型態完成後進場
早晨之星長紅K+短K線+長綠K組合第三根K線收盤越高越強

型態學六大誤區：
1. 型態強行對號入座（忽略圖形不完整）
2. 忽略成交量確認（無量突破=假突破）
3. 單一型態孤立判斷（忽略大環境）
4. 忽略時間與比例對稱性（型態失衡）
5. 忽視關鍵支撐壓力位
6. 過度依賴歷史重複性

2. 布林通道實戰操作

三條線組成：

  • 中軌 = 20日移動平均線（MA20）
  • 上軌 = 中軌 + 2 × 標準差
  • 下軌 = 中軌 - 2 × 標準差

兩大操作派別：

派別核心邏輯適用市場參數建議
逆勢觀點95%價格落在通道內，回歸均值盤整理N=20, K=2
順勢觀點突破後沿通道發展趨勢明確N更長，通道更窄

十種布林通道型態：

型態特徵操作方式
布林收縮上下軌極度收窄等待突破方向確認
布林擴張上下軌大幅張開順勢追進
上軌觸碰價格觸及或突破上軌強勢持有/震盪賣出
下軌觸碰價格觸及或跌破下軌強勢做空/震盪買進
W型底部下軌附近形成W底第二底+動量指標底背離
M型頂部上軌附近形成M頂第二頂+動量指標頂背離
走廊運行價格緊貼軌道運行順勢持有
布林反轉觸軌後反彈盤整理適用
假突破短暫突破後回歸反向操作
收縮性突破通道收窄後帶量突破高勝率順勢訊號

實戰心法：

  • 通道收縮後的突破是高勝率訊號
  • 強勢股會沿上軌跑，不要隨便摸頂
  • 布林通道是波動率指標不是方向指標

3. ATR 真實波動範圍

TR計算公式（三種情況取最大值）：
1. 當日最高價 - 當日最低價
2. |當日最高價 - 前日收盤價|
3. |當日最低價 - 前日收盤價|

ATR停損設定：

停損點 = 買進價 - k × ATR
k值：波動大用大k，波動小用小k（常用1.5-2倍）

棘輪止損法：

  • 持倉期間每增加1倍ATR獲利，上調停損
  • 公式：新停損 = 最近低點 + (持倉天數 × 0.05 × ATR)

海龜交易法則（資金管理）：

建倉單位 = (帳戶資金 × 1%) ÷ (點值 × ATR)
持倉上限：4個單位

4. SAR拋物線指標

核心概念：Stop and Reverse，提供停損與反轉位置。

公式：

多方SAR = SAR昨 + AF × (最高價n日內 - SAR昨)
空方SAR = SAR昨 + AF × (SAR昨 - 最低價n日內)
AF起始值：0.02，每創新高/新低加0.02，最大0.2

實戰三原則：
1. 日K比小週期準確（雜訊少）
2. 適合趨勢盤，不適合盤整盤
3. 參數可調整快慢（AF越大越敏感）

搭配指標：

搭配用途
移動平均線均線多頭排列+SAR翻多
MACD黃金交叉+SAR翻多
布林通道通道收斂=盤整理，不用SAR

SAR優缺點：

優點缺點
自帶停損停利盤整理必虧
趨勢吃到底落後指標
客觀明確參數敏感
反手機制小週期失效

5. 支撐壓力判斷

三種找支撐壓力的方法：

方法做法特點
影線實體法觀察K棒跌不破/漲不破的價位最直觀
連線法至少2波低點/高點連線越多點越有力
跳空法K棒之間的空白區間強力支撐壓力

突破後的反應：

  • 支撐跌破 → 變成壓力
  • 壓力突破 → 變成支撐

找買賣點策略：

策略操作
支撐線買點價格觸及支撐不跌破，確認後買進
壓力線賣點價格觸及壓力無法突破，確認後賣出

6. MACD 指標實戰應用

組成：

  • DIF（快線）= EMA(12) - EMA(26)
  • DEA（慢線/訊號線）= EMA(DIF, 9)
  • MACD柱狀圖 = DIF - DEA

四大買點：

買點定義適用時機
B1 突破買點均線走平/上彎，價格突破均線趨勢反轉初期
B2 支撐買點均線上揚，價格拉回測試均線反彈多頭趨勢中回檔（最常用）
B3 假跌破買點均線上揚，價格短暫跌破後快速收回主力洗盤時
B4 反彈買點價格急跌造成負乖離過大空頭急跌搶反彈

背離訊號：

  • 頂背離：價格創新高但MACD未創新高 → 賣出訊號
  • 底背離：價格創新低但MACD未創新低 → 買進訊號

參數調整：

用途參數
短線(5, 13, 8)
標準(12, 26, 9)
長線(24, 52, 18)

7. RSI 相對強弱指標

公式：

RSI = 買方力道 ÷ (買方力道 + 賣方力道) × 100

判斷標準：

RSI數值市場狀態操作
> 70超買區留意回檔
< 30超賣區可能反彈
≈ 50多空平衡無明確方向

Failure Swing（反轉訊號）：

  • 看漲：RSI跌破30後反彈，拉回不破30，創新高確認
  • 看跌：RSI突破70後跌回，拉回不破70，創新低確認

RSI 50分水嶺：

  • RSI > 50：偏多，50是支撐
  • RSI < 50：偏空，50是壓力

鈍化問題：

  • 高檔鈍化：RSI > 70但價格持續上漲 → 不要急著放空
  • 低檔鈍化：RSI < 30但價格持續下跌 → 不要急著抄底

8. 趨勢線繪製與實戰

繪製五大要點：
1. 選對點位（低點連支撐，高點連壓力）
2. 畫線規則（實體對實體，影線對影線）
3. 斜度要求（30-70度之間）
4. 方向確認（上升趨勢：高點也要上移）
5. 驗證次數（越多K棒連成越有力）

123法則（趨勢反轉確認）：
1. 趨勢線被突破
2. 價格不再創新高（或不創新低）
3. 價格跌破前低（或突破前高）

趨勢通道：

  • 上升通道：下軌買進，上軌減倉
  • 下降通道：上軌賣出，下軌平倉

9. OBV 能量潮指標

公式：

  • 今日收 > 昨日收 → OBV = 前日OBV + 今日成交量
  • 今日收 < 昨日收 → OBV = 前日OBV - 今日成交量

四大交易訊號：

訊號判斷操作
頂背離股價創新高，OBV未創新高賣出
底背離股價創新低，OBV未創新低買進
OBV急升買盤已大量累積注意反轉
OBV急跌大量賣盤湧出觀望

核心觀念：「量先價行」——成交量是股市元氣，股價只是表象。

10. 成交量的主力行為判讀

主力進出場特徵：

  • 主力進場：低檔爆大量，股價劇烈波動
  • 主力出貨：高檔爆大量，長上影線或大黑K
  • 主力洗盤：量縮下跌，股價進入箱型整理

成交量節點：

  • 高量節點（阻力區）：價格暫停、進入震盪
  • 低量節點（順暢區）：價格快速穿越

特殊量能型態：

型態定義意義
凹洞量成交量突然縮到極低後恢復底部反轉機會
窒息量成交量<最大量10%賣壓耗盡
高檔爆量四大指標先賣再說

11. CCI 順勢指標

公式：

CCI = (TP - MA) / (0.015 × MD)
TP = (最高價 + 最低價 + 收盤價) / 3

三大區間：

區間CCI數值市場狀態操作
超買區> +100強勢上漲關注反轉
常態區-100 ~ +100震盪整理用其他指標
超賣區< -100弱勢下跌關注反轉

CCI六大秘訣：
1. 圍繞地線震盪後向上突破 → 轉強
2. 上穿天線後達300左右不能持續 → 短線出貨
3. 跌破-100再突破+100 → 洗盤結束
4. 常態區運行一個月後突破+100 → 大膽買入
5. 突破+100後快速回落不破+100 → 最佳買點
6. 穿越+100後持續上揚 → 強者恆強

12. MTM 動量指標

公式：

MTM = 當日收盤價 - n日前收盤價（n通常=10或12）

運用方法：

  • MTM從下方穿過中心線（零軸）→ 買進訊號
  • MTM從上方穿過中心線（零軸）→ 賣出訊號
  • MTM在高點徘徊 → 強勢行情持續
  • MTM在低點徘徊 → 弱勢行情持續

當沖應用：搭配短週期均線（5MA或10MA）

  • 價格站上均線 + MTM為正 → 多方確認
  • 價格跌破均線 + MTM為負 → 空方確認

13. 威廉分形與鱷魚線

威廉分形：

  • 向上分形（賣出）：中間K棒高點高於左右各2根K棒
  • 向下分形（買入）：中間K棒低點低於左右各2根K棒
  • 需第6根K棒確認

鱷魚線：

  • 嘴唇線（綠線）：5日線，位移3根K棒
  • 牙齒線（紅線）：8日線，位移5根K棒
  • 下顎線（藍線）：13日線，位移8根K棒

鱷魚狀態判斷：

  • 睡覺：三線糾纏 → 市場盤整（不交易）
  • 甦醒：三線發散 → 趨勢形成
  • 進食：三線明顯發散且方向一致 → 強勢趨勢

組合策略：
1. 確認鱷魚甦醒（三線發散）
2. 價格突破分形高點 → 進場
3. 停損設在分形低點

15. 乖離率 BIAS

公式：

乖離率 = (當前價格 - 移動平均線) / 移動平均線 × 100%

參數設定參考值：

均線週期負乖離買點正乖離賣點
6日-3%+3.5%
12日-4.5%+5%
24日-7%+8%
72日-11%+11%

實戰應用：

  • 負乖離過大 + K線反轉訊號 → 買進
  • 正乖離過大 + 量價背離 → 賣出

三大誤解：
1. 乖離過大不等於立即反轉
2. 價格回到均線不等於獲利
3. 不能單獨使用，需搭配KDJ、布林通道
