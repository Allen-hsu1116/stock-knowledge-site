一、杜邦分析法進階：五因子模型

之前學過三因子杜邦分析（淨利率×資產週轉率×權益乘數），這次深入五因子模型，拆得更細、看得更透。

三因子 vs 五因子

三因子模型：
> ROE = 淨利率 × 總資產週轉率 × 權益乘數

五因子模型：
> ROE = 稅務負擔 × 利息負擔 × 營業利潤率 × 總資產週轉率 × 權益乘數

五因子把「淨利率」再拆成三塊：

  • 稅務負擔 = 淨利 ÷ 稅前淨利（EBT）→ 反映有效稅率，越接近1稅負越輕
  • 利息負擔 = 稅前淨利（EBT）÷ 營業利潤（EBIT）→ 反映利息侵蝕程度，越接近1利息壓力越小
  • 營業利潤率 = 營業利潤（EBIT）÷ 營業收入 → 核心業務獲利能力

五因子怎麼用？

三因子只能告訴你「淨利率下降了」，但五因子能精確歸因：

問題現象三因子診斷五因子精確定位
ROE下降淨利率下降營業利潤率下降？利息負擔加重？稅務優惠到期？
ROE穩定三因子都OK可能利息負擔掩蓋了營業利潤率下滑
高ROE靠槓桿權益乘數高利息負擔比率低＝利息吃掉大量利潤

實戰案例：同樣ROE 20%，天差地遠

指標科技巨人（A公司）零售龍頭（B公司）
淨利率25%2%
資產週轉率0.8次5.0次
權益乘數1.0（無負債）2.0（有負債）
ROE20%20%

A公司：高利潤驅動，技術壁壘強，財務風險極低，但成長可能受限
B公司：高週轉＋適度槓桿驅動，薄利多銷，容錯空間小，對景氣敏感

投資啟示：同樣的ROE數字，背後的經營故事完全不同。巴菲特偏好A公司這種「高淨利率＋低槓桿」的模式，因為品質更高、風險更低。

杜邦分析五大侷限

1. 依賴會計數據：折舊方法、存貨計價、營收認列時間點都可能被「化妝」
2. 忽略無形資產：品牌、專利、客戶關係在資產負債表上看不到，軟體/生技公司被嚴重低估
3. 必須同業比較：重工業週轉率1.0可能是頂尖，超市1.0可能快倒閉
4. 時間序列更重要：單一時間點只是快照，5-10年趨勢才有意義
5. 無法反映現金流：ROE是權責發生制，不等於實際收到的現金，必須搭配現金流量表

二、DCF現金流量折現法進階

DCF核心公式

> DCF現值 = Σ [未來第n期現金流 ÷ (1+折現率)^n]

三個關鍵元素：

  • 自由現金流（FCF）：公司真正可自由運用的資金＝營業現金流－資本支出
  • 折現率（r）：通常用WACC（加權平均資本成本），代表投資人的機會成本
  • 期間（n）：一般估算5-15年，越遠越不準

為什麼用FCF不用EPS？

因為EPS沒考慮：

  • 折舊攤銷：以前花的投資攤在帳上，但現金已經花掉了
  • 資本支出：必須要花的投資，不花未來就沒有盈餘

FCF才是公司「真正賺到口袋裡的錢」。

計算步驟

1. 預測未來5-15年FCF：不確定性越高，估算期間越短
2. 計算終值（TV）：預測期結束時的價值

  • 永續法：TV = FCF ÷ (折現率 - 成長率)

3. 設定折現率（WACC）：股東期望報酬率＋債權人成本，加權平均
4. 計算淨現值（NPV）：所有現金流折現後加總＝企業價值
5. 比較股價：NPV vs 當前股價，低於＝低估可買

實例計算

某公司未來10年每年FCF 10元，10年後終值125元，折現率10%：

DCF = 10÷1.1 + 10÷1.1² + ... + 10÷1.1¹⁰ + 125÷1.1¹⁰
= 9.1 + 8.3 + 7.5 + 6.8 + 6.2 + 5.6 + 5.1 + 4.7 + 4.2 + 3.9 + 48.2
= 109.6元

如果當前股價 < 109.6元 → 低估，年化報酬率 > 10%
如果當前股價 > 109.6元 → 高估，年化報酬率 < 10%

DCF適用與不適用

適用：

  • 現金流穩定可預測的公司（公用事業、消費必需品）
  • 債券估值（報酬確定）
  • 價值投資人評估內在價值

不適用：

  • 週期性公司：獲利大起大落，難以預測
  • 初創公司：沒有歷史數據，未來全猜
  • 科技/生技公司：長期虧損後可能一飛衝天，也可能歸零
  • 私募公司：財務數據難以取得

DCF的優缺點

優點缺點
考慮時間價值，最完整的估價法預測未來現金流本身就是猜測
不受市場情緒影響，較客觀終值難以準確估計
可分析不同類型投資，得出一致估值WACC折現率難以精確計算
計算IRR比較不同機會需要大量財務數據，耗時

市場先生的名言：「查理蒙格說過：『我知道巴菲特都是這樣估價，但我從來沒有見他動手算過。』」→ 真正好的投資機會，隨便估一下就知道值不值得，不需要精算到小數點。

DCF vs 其他估價法

本益比法其實是DCF的簡化版：

  • 本益比法：EPS × 合理本益比倍數，假設未來多年盈餘不變，沒考慮時間價值
  • DCF：考慮了成長率變化、折現率、終值，完整性更高但更複雜

三、波特五力分析：投資人視角

五力分析不只是商學院工具，更是投資人判斷「這個產業值不值得投資」的利器。

五力拆解

1. 現有競爭者的威脅

  • 競爭者越多越激烈
  • 產業成長慢＝搶市佔＝價格戰
  • 產品差異化低＝容易價格競爭
  • 退出壁壘高（資產難變現、合約約束）＝即使虧錢也不走，競爭更慘烈

投資啟示：避開同質化高、競爭者多的紅海產業。找有差異化、品牌壁壘的公司。

2. 新進入者的威脅
進入壁壘越高，既有業者越安全：

  • 規模經濟（大廠成本優勢）
  • 品牌忠誠度
  • 資本需求（半導體設備動輒數百億）
  • 管制與許可（金融、電信、醫藥牌照）
  • 通路掌控
  • 學習曲線（經驗累積的優勢）

投資啟示：高進入壁壘＝護城河。台積電的資本支出和技術門檻就是超級壁壘。

3. 供應商的議價能力

  • 供應商越少、產品越獨特，議價能力越強
  • 轉換成本高（換供應商要重新認證）＝被綁住
  • 供應商前向整合威脅（自己跳下來做終端產品）

投資啟示：如果一家公司嚴重依賴單一供應商，風險很大。分散供應鏈是競爭優勢。

4. 買方的議價能力

  • 買方集中度高、採購量大＝壓價能力強
  • 產品標準化＝容易比價
  • 轉換成本低＝客戶隨時可走
  • 資訊透明＝客戶知道成本

投資啟示：蘋果的供應商就是典型案例，被買方壓價到毛利極薄。反之，產品獨特、客戶分散的公司議價能力強。

5. 替代品的威脅

  • 替代品性價比更高＝威脅大
  • 轉換成本低＝容易切換
  • 電子書威脅紙書、串流威脅有線電視、AI威脅傳統顧問

投資啟示：投資前要問「這個產品/服務未來會不會被取代？」

五力縱橫軸分析法

橫軸（買方vs賣方議價力）：判斷產業利潤空間

  • 買方強＋賣方強＝利潤被兩面夾殺
  • 買方弱＋賣方弱＝公司有定價權

縱軸（新進者vs替代品）：判斷利潤能否持續

  • 進入壁壘高＋替代品少＝利潤穩定可持續
  • 進入壁壘低＋替代品多＝利潤隨時被侵蝕

最佳投資組合：買方弱、賣方弱、進入壁壘高、替代品少＝最肥的產業

五力分析的投資實戰清單

投資一家公司前，用五力快速檢視：

1. 產業競爭：同業幾家？市佔率集中度？有價格戰嗎？
2. 進入壁壘：新玩家容易進來嗎？需要多少資本？有法規保護嗎？
3. 供應商議價：原料來源集中嗎？能輕易換供應商嗎？
4. 買方議價：客戶分散還是集中？產品獨特性高嗎？
5. 替代品威脅：有沒有不同產業但功能類似的替代品？

台股實例：

  • 台積電：進入壁壘極高（資本支出＋技術）、買方議價力中等（客戶分散）、替代品威脅低→五力強
  • 面板廠：競爭激烈（少數大廠價格戰）、進入壁壘高但退出也難、買方議價力強（大客戶）、替代品威脅低→利潤薄
  • 食品股：競爭者多、產品差異化低、買方議價力強→五力弱

心得

今天三個主題串在一起，剛好形成一套完整的基本面分析框架：

杜邦分析是「拆解ROE」，告訴你一家公司賺錢的品質如何。同樣20%的ROE，高淨利率驅動和靠槓桿驅動，風險天差地遠。五因子模型比三因子更精準——當你發現ROE下降，五因子能告訴你是核心業務變差、利息吃掉利潤、還是稅務優惠到期。

DCF是「計算內在價值」，把「這家公司到底值多少錢」從感覺變成數字。但最深刻的一句話是蒙格的「我從沒看過巴菲特動手算DCF」——真正好的投資機會，不用精算就知道便宜。DCF的價值不在於算出精確數字，而在於強迫你思考三件事：FCF是否真實？成長率是否合理？折現率是否反映風險？

五力分析是「判斷產業好不好」，是巴菲特護城河概念的源頭。投資不是只看公司自己，要看整個產業的利潤結構。五力分析告訴你：利潤是被競爭吃掉、被供應商拿走、被買方壓縮、被新進者搶走、還是被替代品消滅。

三個工具搭配使用：杜邦分析看品質 → DCF算價值 → 五力判產業，這就是完整的基本面分析框架。
