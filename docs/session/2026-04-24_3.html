金字塔加碼法×分批進出策略×部位管理

一、金字塔加碼法（Pyramiding）核心概念

### 是什麼 金字塔加碼法是一種**分批建立部位**的策略：隨著價格往有利方向移動，逐步加碼，但每次加碼的量越來越小，形成「底部大、頂部小」的金字塔結構。核心精神是**「只有在市場證明你對的時候，才增加部位」**。 ### 為什麼要金字塔加碼？ 1. **控制風險**：一開始不重押，看錯時損失有限 2. **順勢放大獲利**：趨勢確認後才加大部位，讓獲利奔跑 3. **避免攤平陷阱**：跟「越跌越買」的攤平完全相反——是「越漲越加碼」

傑西·李佛摩的名言：「買進時要測試市場，市場證明你是對的時候，才應該加碼，而不是在市場證明你錯誤時不斷攤平成本。」

二、兩種金字塔策略

### 正金字塔買入法（逢低加碼） **越跌買越多，越漲買越少**

怎麼做：
1. 預設總投入金額（例：10萬元買A股票）
2. 第一筆：股價在目標區間時買40%（4萬）
3. 第二筆：股價下跌5%時買30%（3萬）
4. 第三筆：股價再下跌5%時買20%（2萬）
5. 第四筆：股價再下跌5%時買10%（1萬）

實例：

  • 情境：看好台積電，目前600元，預計投入50萬
  • 600元買20萬（40%）→ 570元買15萬（30%）→ 540元買10萬（20%）→ 510元買5萬（10%）
  • 平均成本約561元，比一口氣在600元買50萬的均成本低很多

適用時機：

  • 左側交易、價值投資
  • 看好標的但不知道會跌多深
  • 長期投資分批建立部位

風險：可能「越攤越平」——如果公司基本面真的出問題，跌越多買越多只會死更慘

### 倒金字塔加碼法（順勢加碼 / Pyramiding） **越漲買越多（遞減式加碼），越跌買越少**

怎麼做：
1. 第一筆（試單）：小部位進場，佔總部位的50%
2. 第二筆（確認加碼）：趨勢確認後加碼30%
3. 第三筆（最後加碼）：趨勢持續強勁再加20%
4. 每次加碼金額遞減，確保均價離停損點夠遠

實例：

  • 情境：看好聯發科突破800元壓力區
  • 800元試單買5張（50%）→ 830元突破確認加碼3張（30%）→ 860元續強再加2張（20%）
  • 平均成本約816元，如果回檔到810元停損，最大虧損可控
  • 停損邏輯：第一筆停損790元，加碼後停損上移到810元（保本停損）

適用時機：

  • 右側交易、趨勢追蹤
  • 突破後追蹤強勢股
  • 短線、波段操作

優勢：看錯時只損失小部位（試單），看對時部位在趨勢確認後放大

三、金字塔加碼的數學原理

Risk-at-Market vs. 總部位

關鍵概念：金字塔加碼的目標是增加總部位，但維持風險不變

計算公式：

  • 初始單位大小 U1 = (帳戶餘額 × 風險%) / (進場價 - 停損價)

實例：

  • 帳戶100萬，風險1%（1萬元）
  • 股價100元，停損90元（風險10元/股）
  • 第一筆：買100股（10,000元曝險，1,000元風險）

加碼後停損上移：

  • 股價漲到110元，加碼50股
  • 新均價 = (100×100 + 110×50) / 150 = 103.33元
  • 維持1,000元風險 → 每股風險 = 1,000 / 150 = 6.67元
  • 新停損 = 103.33 - 6.67 = 96.67元
  • 部位增加50%，但總風險沒變！

正金字塔（4-3-2-1）：數學上最合理

  • 第一筆最大，均價最低
  • 小幅回檔不會讓整體部位轉虧
  • 適合大多數人

倒金字塔（1-2-3-4）：危險

  • 最後加碼最多，均價被拉高
  • 小幅回檔就可能讓整體部位從賺轉虧
  • 只有在極度確認趨勢時才考慮

四、停損管理：金字塔的生死線

加碼後停損上移原則

怎麼做：
1. 第一筆進場：設停損（例：7%）
2. 第一次加碼：停損上移到「整體部位損益兩平點」
3. 第二次加碼：停損再上移到「保護一定利潤」的位置
4. 目標：確保最壞情況下，總風險不超過最初的風險額度

實例：

  • 100元買進，停損93元（7%）
  • 漲到110元加碼，停損上移到103元（保本）
  • 漲到120元再加碼，停損上移到115元（保護5%利潤）
  • 如果從120元回檔到115元出場，整體仍獲利

免費交易（Free Trade）概念

是什麼：當第一筆的浮動利潤 ≥ 第二筆的最大可能虧損時，第二筆加碼就是「免費」的

怎麼計算：

  • 第一筆：60,000元買BTC，停損58,000元
  • BTC漲到64,000元，浮動利潤4,000元
  • 第二筆：64,000元加碼，停損62,000元，最大虧損2,000元
  • 4,000 ≥ 2,000 → 這筆加碼是「免費」的（即使加碼失敗，整體仍損益兩平）

適用時機：波動大的標的（加密貨幣、期貨），在趨勢明確時使用

五、三種加碼觸發方式

### 1. 固定間隔加碼（Fixed Interval） **怎麼做**：每漲5%加碼一次，不論時間
  • 例：100元買→105元加碼→110.25元再加碼→115.76元最後加碼
  • 優點：機械化，沒有情緒干擾
  • 缺點：盤整時可能頻繁觸發
### 2. 波動調整加碼（Volatility-Adjusted） **怎麼做**：用ATR（平均真實範圍）決定加碼間隔
  • 例：ATR=3元，每次漲2倍ATR（6元）才加碼
  • 低波動時：加碼間隔小，可以更快建立部位
  • 高波動時：加碼間隔大，避免被震出場
  • 優點：自動適應市場波動
### 3. 動能確認加碼（Momentum-Based） **怎麼做**：必須有技術訊號確認才加碼
  • RSI > 60 才向上加碼，RSI < 40 才向下加碼
  • MACD柱狀圖擴大確認動能
  • 成交量高於均量確認機構參與
  • 優點：過濾弱趨勢，只在強勢時加碼

三種方式比較：

  • 機械化程度：固定間隔 > 波動調整 > 動能確認
  • 適應性：動能確認 > 波動調整 > 固定間隔
  • 推薦：波動調整 + 動能確認雙重過濾

六、分批出場（倒金字塔賣出法）

### 是什麼 跟加碼相反——**越漲賣越多，越跌賣越少**。鎖定利潤同時保留部分部位參與後續行情。 ### 怎麼做 1. 第一目標價到達：賣出40% 2. 第二目標價到達：賣出30% 3. 第三目標價到達：賣出20% 4. 最後10%：用移動停損保護，讓利潤奔跑

實例：

  • 持有台積電10張，成本500元
  • 550元賣4張（40%）→ 鎖定主要獲利
  • 600元賣3張（30%）→ 再鎖定一部分
  • 650元賣2張（20%）→ 大部分已出場
  • 最後1張（10%）→ 設移動停損，看能跑多遠

適用時機：

  • 達到目標價但不想一次清空
  • 擔心回檔但又不確定會不會續漲
  • 任何需要降低曝險的時候

七、金字塔加碼的常見錯誤

### 1. 每筆交易都金字塔加碼 **錯在哪**：不是每個走勢都值得加碼。金字塔只適用於**高信念、強趨勢**的標的 **正確做法**：只對通過你所有篩選條件的標的使用金字塔加碼 ### 2. 用融資加碼 **錯在哪**：槓桿+金字塔 = 災難。回檔時不只虧損本金，還要還融資利息 **正確做法**：只用自有資金金字塔加碼，絕對不加槓桿 ### 3. 忽略總部位大小 **錯在哪**：加碼加到部位佔整體投資組合過高，一次回檔就重傷 **正確做法**：設定總部位上限（例：單一標的不超過投資組合20%） ### 4. 倒金字塔加碼（越漲買越多） **錯在哪**：均價被拉到接近市價，小幅回檔就從賺轉虧 **正確做法**：永遠用正金字塔或遞減式加碼（4-3-2-1或50-30-20） ### 5. 不設停損就加碼 **錯在哪**：加碼後沒有調整停損，等於把風險加倍 **正確做法**：每次加碼都要重新計算整體部位的停損點

八、金字塔 vs. 攤平：為什麼加碼贏家而非輸家

比較項目金字塔加碼（加碼贏家）攤平（加碼輸家）
進場邏輯趨勢確認才加碼越跌越買
均價變化均價略升（但有利潤保護）均價下降
最大風險有限（停損上移保護）無限（不知道會跌多深）
心理壓力低（越加越有信心）高（越加越焦慮）
適用情境趨勢追蹤、波段操作價值投資、長期定投

核心差異：金字塔加碼是「市場證明我對，我才加碼」；攤平是「我覺得我對，市場還沒證明我就加碼」

九、實戰SOP：金字塔加碼五步法

Step 1：設定總部位上限

  • 單一標的最大投入：投資組合的15-20%
  • 預先決定分幾次加碼（建議3-4次）

Step 2：第一筆試單

  • 金額：總部位的40-50%
  • 停損：依策略設定（技術停損或百分比停損）
  • 確認訊號：突破、均線黃金交叉、量能放大等

Step 3：加碼觸發條件

  • 價格漲幅觸發：每漲5-7%加碼一次
  • 技術訊號觸發：突破前高、回測支撐不破、MACD金叉等
  • 動能確認：成交量放大、RSI維持60以上

Step 4：加碼金額遞減

  • 第二筆：30%
  • 第三筆：20%
  • 加碼金額絕不超過前一筆

Step 5：停損上移

  • 第一次加碼後：停損移到損益兩平點
  • 第二次加碼後：停損移到保護利潤位置（至少保住3-5%獲利）
  • 任何時候：總風險不超過原始風險額度

心得

金字塔加碼法看起來簡單，但魔鬼在細節裡。最關鍵的其實不是「怎麼加碼」，而是「什麼時候不該加碼」。

很多人犯的錯是什麼都想加碼——每個看起來對的部位都想要金字塔。但金字塔加碼只適合強趨勢、高信念的標的。盤整盤用金字塔加碼，只會被來回洗刷，手續費都虧掉。

另一個重要體悟：金字塔加碼的數學原理非常優雅——增加部位但維持風險不變。這跟直覺相反，很多人以為加碼就等於加風險，但如果你同時把停損上移到保本點，其實風險是下降的。這就是金字塔加碼最強大的地方。

還有那個「免費交易」的概念也很實用——當第一筆的浮動利潤已經足夠覆蓋第二筆的可能虧損時，加碼就是零風險的。這種時候不加碼才真的可惜。

最後，倒金字塔加碼（越漲買越多）是新手最常犯的錯誤。那種「越看越對、越對越買」的感覺太爽了，但一次回檔就從天堂掉到地獄。永遠記得：底部最大、頂部最小，這才是金字塔的正確形狀。
