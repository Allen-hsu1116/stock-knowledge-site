一、投信作帳行情核心邏輯

投信為什麼要作帳？

投信（投資信託公司）是基金背後的操盤者，靠「投資績效」生存。績效差 → 基金規模縮 → 被換掉。所以每到結算日，基金經理人會拚命拉抬手中持股，讓帳面績效好看。

關鍵差異：投信要的是「帳面上的數字」，不是「落袋為安的現金」。

投信基金經理人看重的是基金評比排名，只要持有的股票價格上漲，基金淨值就上升，評比就好。不用把股票賣掉，帳面上股價漲就好，最好一直漲到季底。

作帳時間軸

時間點重要性拉抬幅度
年底（12月）⭐⭐⭐最大
半年底（6月）⭐⭐中等
季底（3/6/9/12月）⭐較小
  • 作帳行情通常出現在月中至月底（3月、6月、9月、12月）
  • 年底最重要：攸關年終獎金，動作最大
  • 半年底次之：年中分紅季
  • 季底最小：只影響季排名

二、投信作帳三大操作手法

手法一：假單吸籌

投信看多時，在外盤整數價位掛大單（假賣單），讓散戶以為有壓力過不去，提前賣出 → 投信趁機在低價吸籌碼 → 籌碼吸夠後撤單 → 股價飆漲。

投信看空時，在內盤掛大單（假買單），讓散戶以為有支撐搶進 → 投信偷偷倒貨。

怎麼辨識：

  • 觀察五檔掛單：突然在整數價位出現異常大單（與平常掛單量差很大）
  • 該大單被吃掉後又重新掛出 → 典型假單特徵
  • 成交量沒有真正放大但掛單量異常 → 假單機率高

手法二：選「我有你沒有」的股票

基金經理人互相競爭排名，拉抬邏輯：

  • A基金持有甲、乙股；B基金持有乙、丙股
  • A基金會拉抬甲股（自己有、別人沒有），這樣只有A績效變好
  • 如果拉抬乙股，A、B績效一起好，等於白忙

實戰篩選步驟：
1. 查同類型基金排行前幾名（如國內股票型中小型股前10名）
2. 比較各基金持股，找出「只有一家投信重倉、其他投信沒買」的股票
3. 避開台積電、鴻海等幾乎每家都買的大型股 → 股本太大也不好拉

手法三：王牌股最後衝刺

10月以後才開始進場的投信買超，可能是「王牌持股」——最後績效衝刺的法寶。

篩選條件：

  • 投信近12天連續買超，量還不小
  • 目前持股比例還不高（< 10%）
  • 漲幅還沒超過30%（漲太多不好再拉）
  • 有題材加持更容易形成拉抬

三、投信持股比例警戒線

15% 總投信持股警戒線

  • 金管會規定：單一基金持一檔股票不得超過淨資產10%
  • 總投信持股超過15% → 開始留意是否在出貨
  • 超過20% → 高度警戒，隨時可能結帳

實例：

  • 立積（4968）：投信持股從23%降到接近0%，股價從200多元跌到不到100元
  • 台耀（4746）：以往投信持股約15%就回落，2018年突破20%後開始出脫，股價盤整後下跌

怎麼做：
1. 查詢個股的「投信持股比率」（可從公開資訊觀測站、GoodInfo、XQ等查）
2. 持股 < 10%：安全區，投信可能還在買
3. 持股 10-15%：觀察區，注意買超是否減速
4. 持股 > 15%：警戒區，投信可能開始倒貨
5. 持股 > 20%：危%：危險區，隨時可能結帳

四、自營商結帳行情（跟投信完全相反！）

核心差異

投信作帳自營商結帳
目的美化帳面績效落袋為安拿獎金
手法買進拉抬股價賣出實現獲利
時間季底前拉抬季底前賣出
關鍵「未實現」數字越大越好「已實現」獲利越多越好

自營商操盤人要的是績效獎金，老闆發獎金看的是「你幫公司實際賺了多少」，不是帳面上的未實現獲利。所以季底前自營商會逢高調節，把浮盈變成真金白銀。

自營商結帳時間軌跡

1. 季初第1個月（1/4/7/10月）：自營商大買進場 → 個股股價橫盤整理時是進場時機
2. 季中第2個月（2/5/8/11月）：股價拉一波
3. 季末第3個月（3/6/9/12月）：自營商逢高調節 → 散戶要下車！

實戰操作：

  • 在每季第1個月，追蹤自營商新增持股張數最大的個股
  • 該月股價橫向整理 → 順勢進場搭轎
  • 第2個月股價拉升 → 續抱
  • 進入第3個月 → 獲利了結出場，不要幻想自營商會繼續推高

五、集團作帳行情

是什麼

集團公司透過旗下公司交叉持股，或在媒體釋出利多消息，拉抬股價美化帳面。

  • 旗下公司股價輪動上漲
  • 常搭配媒體利多消息（如打入特斯拉供應鏈等）
  • 比投信作帳更難捉摸，沒有公開持股比例可參考

作帳行情出場時機

  • 投信持股比率突破15% → 開始警戒
  • 投信從連續買超轉為賣超 → 立即出場

心得

投信作帳跟自營商結帳根本是同一個時間點的兩股相反力量，一個在買一個在賣，搞不清楚就會被兩邊夾殺。最實用的概念是：投信追求「帳面數字好看」，自營商追求「錢落袋為安」，所以同一檔股票如果同時被投信大買和自營商大賣，那個季底震盪會非常劇烈。

另外「我有你沒有」的邏輯很精闢——投信之間是競爭關係，拉抬別人也有買的股票等於做白工。這個觀念可以反過來用：如果一檔股票被很多投信同時重倉，那它反而不太可能是作帳標的，因為拉了大家一起爽。

持股比例15%警戒線也是實戰好用的指標，比單純看買賣超張數更有參考價值，因為它反映了投信的「可操作空間」。持股太高就拉不動了，而且隨時可以結帳。
