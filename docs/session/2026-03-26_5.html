配對交易核心概念

什麼是配對交易

是什麼：一種市場中性策略，同時操作兩個（或兩組）高度相關的投資標的，一多一空，消除系統性風險，賺取兩者之間的價差。

核心邏輯：

  • 不在乎市場漲跌方向
  • 跟波動、相關性有關
  • 用多空對沖消除系統風險

實例：

  • 可口可樂 vs 百事可樂 — 兩者走勢高度相關
  • 長榮 vs 陽明 — 海運股同產業、高相關
  • 台積電 vs 台指期貨 — 做多強勢股、放空大盤

兩種獲利方式

1. 價差收斂（主流方式）

是什麼：認為兩標的長期走勢相似，短期價差放大是暫時現象，等價差收斂獲利。

怎麼做：
1. 找出兩個高度相關的標的
2. 當價差放大時，放空偏高的、做多偏低的
3. 等價差收斂後平倉

實例：

  • 情境：可口可樂與百事可樂歷史價差約 2%，現在擴大到 5%
  • 操作：放空可口可樂、買進百事可樂
  • 獲利：價差回歸 2% 時平倉，賺 3% 價差

2. 價差發散（對沖變化型）

是什麼：選擇一強一弱的相關標的，買強放弱，長期賺取強弱差距。

實例：

  • 情境：認為台積電長期會優於整體台股
  • 操作：做多台積電期貨、放空台指期貨
  • 獲利：台積電漲 5%、大盤漲 2%，賺 3% 差價

進出場時機判定

價差的統計方法

怎麼計算價差：

  • 使用線性迴歸找出兩標的的 alpha 與 beta 係數
  • 價差 = 標的B - alpha - beta × 標的A
  • 檢定價差序列是否定態（ADF 檢定）

進場條件（以 TEJ 實例）：

區間操作
價差 > μ + 1.5σ做空配對（賣B買A）
價差 < μ - 1.5σ做多配對（買B賣A）
價差回到 μ ± 0.2σ平倉出場
價差突破 μ ± 2.5σ緊急停損

實例數值：

  • 形成期計算出 μ（均值）和 σ（標準差）
  • 例如：價差均值 0.02%，標準差 0.8%
  • 進場門檻：0.02% + 1.5 × 0.8% = 1.22%
  • 出場門檻：0.02% + 0.2 × 0.8% = 0.18%

形成期 vs 交易期

怎麼設定：

  • 形成期：例如 12 個月，用來計算 μ、σ、beta
  • 交易期：例如 3 個月，用形成期參數執行策略
  • 避免前視偏誤 — 交易期不能看到未來資料

標的選擇原則

選擇條件

必備條件：
1. 高度相關 — 產業相近或替代品
2. 價差序列定態 — 確認有共整合關係
3. 可放空 — 流動性足夠、融券或期貨可行

實例配對：

  • 同產業：福特 vs 通用汽車
  • 同類股：Visa vs Master
  • 同族群：長榮 vs 陽明（海運）
  • 指數對沖：台積電 vs 台指期

不能選什麼

錯誤配對：

  • 不相干的標的 — 系統風險不同，無法對沖
  • 例如：外匯貨幣對 vs 單一股票

風險管理

四大風險

1. 價差可能不收斂

  • 相關性可能隨時間改變
  • 其中一家公司可能出現特殊事件

2. 槓桿放大風險

  • 單次獲利小（可能只 0.2%）
  • 必須用槓桿才有合理報酬
  • 槓桿後 0.2% 變 2%，但風險也放大

3. 盈虧比極低

  • 特質：高勝率、小賺大賠
  • 類似選擇權賣方
  • 一次意外可能吃掉多年利潤

4. 資金成本兩倍

  • 需要買進 + 放空兩份資金
  • 手續費、交易稅、融券利息加倍

風險控制方法

停損設定：

  • 價差突破 μ ± 2.5σ 立即平倉
  • 時間停損：超過 N 天不收斂就出場

資金控制：

  • 只用小部分資金
  • 不重押單一配對
  • 分散多組配對降低單一風險

優缺點總結

優點

1. 對沖系統風險 — 市場大跌也能穩定
2. 與市場方向無關 — 漲跌震盪都能做
3. 勝率高 — 理論上收斂機率大
4. 損益曲線平滑 — 無劇烈波動（無槓桿時）

缺點

1. 盈虧比低 — 小賺大賠類型
2. 資金效率低 — 兩倍資金賺微小價差
3. 相關性不保證 — 可能永久發散
4. 執行難度高 — 需要程式、資料源、數學能力
5. 放空成本高 — 融券利息、流動性限制

心得

配對交易是典型的「高勝率、小賺大賠」策略——聽起來很穩，但一次意外就能吃掉多年利潤。跟選擇權賣方一樣，是容錯率很低的策略。

這策略的核心假設是「價差終將收斂」，但沒人能保證。長榮和陽明的例子就很清楚：2021 年上半年波動極大，形成期的參數可能完全不適用交易期。市場先生說得對，這是「脆弱性」策略，一次意外就重傷。

實務上幾個重點：
1. 參數要用 rolling 更新，不能死守歷史數值
2. 停損必須嚴格，價差發散到 μ ± 2.5σ 要果斷出場
3. 槓桿要謹慎，寧願賺少也不要被斷頭
4. 這策略不適合一般人，是避險基金和機構法人的戰場

如果要實作，我會選擇：

  • 只用總資金 5-10% 在配對交易
  • 選擇流動性最好的標的（期貨優於股票）
  • 用程式自動執行，避免人性干擾
  • 每組配對都有明確停損點和時間上限
