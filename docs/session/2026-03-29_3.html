  • 「關鍵價位」與「支撐壓力」怎麼看？
  • 「支撐線」與「壓力線」是什麼？
  • 股價支撐和壓力是什麼意思？又該怎麼找買賣點？

一、支撐壓力基本概念

支撐是什麼

是什麼：股價跌到某個價位時，買盤進場支撐，價格跌不下去的位置，像地板。

形成原因：

  • 主力成本區：大戶不想跌破成本
  • 投資人覺得「便宜好買點」
  • 前低、大量成交區、整數關卡

怎麼看：

  • 價格多次跌到某區域就反彈
  • K棒有下影線、量縮止跌
  • 出現止跌回升跡象

壓力是什麼

是什麼：股價漲到某個價位時，賣壓出現，價格漲不上去的位置，像天花板。

形成原因：

  • 套牢盤想解套賣出
  • 停利單掛在這個價位
  • 前高、整數關卡、成交密集區

怎麼看：

  • 價格多次碰到某區就回落
  • 出量卻不漲、長上影線
  • 前高、整數價附近震盪

二、關鍵價位四大類型

1. 整數關卡

是什麼：50元、100元、200元這種整數價位。

為什麼重要：

  • 人習慣用整數當參考基準
  • 停損單、停利單常掛整數附近
  • 成為買賣雙方交戰區域

實例：

  • 股價160元的股票，漲到200會有賣壓
  • 跌到100附近可能反彈

2. 大量成交區

是什麼：過去有大量買賣發生的區域。

為什麼重要：

  • 股價回到這裡會遇到解套賣壓
  • 跌回便宜區域會有人想買

實例：

  • 某股在70-75元區間大量換手
  • 之後股價跌回70元附近會有支撐

3. 前高/前低價位

是什麼：上一次漲到的高點或跌到的低點。

為什麼重要：

  • 「上次漲到這就跌」的心理作用
  • 「上次跌到這就漲」的記憶
  • 市場會特別關注這些位置

判斷：

  • 靠近前高 → 容易觀望或拉鋸
  • 突破前高 → 有機會續漲
  • 跌破前低 → 可能進一步走弱

4. 跳空缺口

是什麼：兩根K棒之間的空白區域。

為什麼重要：

  • 用兩根K棒就能形成有力的支撐壓力
  • 缺口區間通常會形成支撐或壓力

實例：

  • 股價從100跳空到110，中間101-109是缺口
  • 之後回補缺口時，缺口區會有支撐或壓力

三、支撐壓力畫法

兩根K棒畫法

支撐線：
1. 找出至少2波低點
2. 將低點相連
3. 向右延伸一條線

壓力線：
1. 找出至少2波高點
2. 將高點相連
3. 向右延伸一條線

K棒觀察方法

看影線：

  • 觀察K棒下影線：多次在價格附近出現下影線，可能有支撐
  • 觀察K棒上影線：多次在價格附近出現上影線，可能有壓力

看實體：

  • 股價多次跌不到某價格下 → 支撐
  • 股價多次漲不超某價格上 → 壓力

綜合參考：

  • 可以兩個都看
  • 甚至憑感覺大略判斷也行

有力程度判斷

越多K棒連成越有力：

  • 兩根K棒就能畫線，但較不可靠
  • 十根K棒連成的線，更有參考價值

被阻擋越久突破越強：

  • 盤整越久，突破後漲幅越大
  • 像彈力球被困越久，衝破時飛越快

四、支撐壓力轉換法則

支撐變壓力

情境：支撐被跌破後，原本的支撐會變成壓力。

原因：

  • 主力成本在這個價位
  • 套牢盤等解套賣出
  • 不希望價格賠錢或白做工

實例：

  • 股價在100元有支撐
  • 跌破100元後，反彈到100元會遇到賣壓

壓力變支撐

情境：壓力被突破後，原本的壓力會變成支撐。

原因：

  • 突破後，買方成本在這附近
  • 市場認同這個價位
  • 回測時會有買盤進場

實例：

  • 股價在150元有壓力
  • 突破150元後，回測150元會有支撐

五、支撐壓力找買賣點

用支撐找買點

進場條件：
1. 股價接近支撐區
2. 成交量開始縮小（賣壓減少）
3. 出現止跌訊號（下影線、紅K）

實例：

  • 情境：某股在70元有明顯支撐
  • 操作：股價跌到70-72元區間，量縮且出現下影線紅K
  • 買點：分批佈局在70元支撐區

停損設定：

  • 跌破支撐就停損
  • 記住：不是每個支撐都守得住

用壓力找賣點

出場條件：
1. 股價接近壓力區
2. 出現震盪或上影線
3. 出量不漲

實例：

  • 情境：某股在200元有明顯壓力
  • 操作：股價漲到198-200元區間
  • 賣點：分批停利，或拉緊停利條件

突破應對：

  • 如果帶量突破壓力，可能續漲
  • 可以等突破後回測再評估

六、假突破判斷四大標準

1. 收盤價突破才算真突破

問題：盤中可能突破，但收盤又回來。

方法：以收盤價為準，不要被盤中假動作騙。

實例：

  • 股價盤中突破壓力150元，最高到155元
  • 收盤回到148元
  • 判斷：假突破，壓力仍在150元

2. 突破3%以上

問題：小幅突破可能是雜訊。

方法：收盤價至少突破支撐壓力線3%。

計算：

  • 壓力線在100元
  • 突破需要至少漲到103元（100 × 1.03）

3. 突破要伴隨大量

問題：沒量的突破可能是假突破。

方法：成交量要超過5日均量30%以上。

判斷：

  • 突破時有量 → 市場認同方向
  • 突破時量縮 → 市場沒信心，小心假突破

4. 突破後觀察走勢

問題：突破後可能又掉回來。

方法：觀察2-3天，看是否朝預期方向發展。

實例：

  • 股價突破壓力後
  • 第2天續漲 → 可確認真突破
  • 第2天跌回壓力下 → 可能假突破

七、常見誤解三大提醒

誤解一：跌到支撐一定會反彈

錯誤想法：看到支撐就貿然進場。

正確做法：

  • 搭配成交量、K棒型態確認
  • 真正止穩再考慮進場
  • 設好停損

誤解二：到壓力一定會下跌

錯誤想法：看到壓力就做空。

正確做法：

  • 壓力是提醒有賣壓，不是必跌保證
  • 觀察是否放量突破
  • 等待帶量長紅等強勢訊號

誤解三：支撐壓力是精準的一條線

錯誤想法：找一個精確數字。

正確做法：

  • 支撐壓力是「區間」，不是「點」
  • 例：支撐區在48-50元之間，壓力在82-84元附近
  • 進出場保留彈性

八、實戰操作SOP

支撐買進SOP

步驟一：找出支撐區

  • 連接至少2波低點
  • 觀察K棒影線密集處
  • 確認前低、整數關卡位置

步驟二：確認買點

  • 股價接近支撐區（區間內）
  • 成交量縮小
  • 出現下影線或紅K止跌訊號

步驟三：分批進場

  • 不要一次買滿
  • 可以分2-3批在支撐區佈局

步驟四：設定停損

  • 跌破支撐區下緣
  • 或跌破前低
  • 果斷停損出場

壓力賣出SOP

步驟一：找出壓力區

  • 連接至少2波高點
  • 觀察K棒上影線密集處
  • 確認前高、整數關卡位置

步驟二：確認賣點

  • 股價接近壓力區
  • 出現震盪或上影線
  • 出量不漲

步驟三：分批出場

  • 接近壓力先賣一部分
  • 無法突破再賣一部分
  • 突破壓力可續抱

步驟四：應對突破

  • 帶量突破壓力 → 回補或加碼
  • 突破後回測壓力變支撐 → 可進場

心得：

支撐壓力是技術分析的基本功，但很多人（包括之前的我也會）犯了「把它當成精確數字」的錯誤。這四篇文章反覆提醒：支撐壓力是「區間」，不是「一條線」，這點要特別注意。

另一個核心概念是「支撐壓力轉換」：為什麼支撐跌破後會變壓力？因為套牢盤等解套。為什麼壓力突破後會變支撐？因為買方成本在那裡。這個邏輯搞清楚，很多價格走勢就能看懂了。

假突破的四大判斷標準很實用：收盤價突破、3%以上、帶量突破、觀察走勢。之前學型態學的時候也有提到假突破，可以跟今天學的支撐壓力結合起來，形成更完整的判斷方法。

最後，支撐壓力找買賣點的SOP很清晰：支撐區 + 量縮止跌 + 分批佈局 + 跌破停損；壓力區 + 出量不漲 + 分批出場 + 突破應對。這個框架可以用到之後的交易中。

本週已經學了回測驗證、指標應用，今天補上支撐壓力，技術分析的主題算是越來越完整了。

DCF 現金流量折現法核心概念

什麼是 DCF？

定義：現金流量折現法（Discounted Cash Flow，簡稱DCF），又稱自由現金流折現法，是把一家公司未來會產生的所有現金流，都換算成現在價值的一種計算方式。

核心思想：

  • 未來的錢不如今天的錢值錢（貨幣時間價值）
  • 巴菲特名言：「上市公司的內在價值，就是該企業在其未來所能產生的自由現金流量折現後的總和」

與本益比的差異：

  • 本益比：用當下盈餘 × 合理倍數 = 簡化版 DCF（沒考慮時間價值）
  • DCF：完整考慮時間價值和風險，完整性更高

DCF 計算公式與要素

基本公式

DCF 現值 = ∑ 未來n期現金流 ÷ [(1+折現率)^n年]

簡單說：把未來每一期現金流折現後加總。

三大核心要素

1. 未來現金流（CF）

怎麼做：

  • 用自由現金流（FCF）計算，不是用盈餘
  • 為什麼？因為盈餘沒考慮折舊、資本支出
  • 自由現金流 = 營業現金流 - 資本支出（公司真正能自由運用的資金）

預估方法：

  • 用近年的現金流與成長率，線性評估未來幾年
  • 越後期的預測誤差越大
  • 通常預估 5~15 年

2. 折現率（r）

是什麼：公司的資本成本，代表機會成本

怎麼算：

  • 通常用 WACC（加權平均資本成本）
  • WACC = 公司融資所負擔的費率
  • 可以理解為：把錢投在其他地方能創造的報酬

實例：

  • 假設折現率 10%
  • 一年後的 10,000 元，折現到現在 = 10,000 ÷ 1.10 = 9,090.9 元
  • 兩年後的 10,000 元，折現到現在 = 10,000 ÷ 1.10² = 8,264.5 元

影響：

  • 折現率越大 → 未來現金流折現後數值越小
  • 折現率越小 → 未來現金流折現後數值越大

3. 期間（n）

怎麼做：

  • 一般計算未來 10~15 年的現金流折現
  • 保守可以只算 5~10 年
  • 越遠的現金流折現後越小，最終小到不可計算

為什麼不估太久？

  • 估 3~5 年就很難準，更遑論 10 年後
  • 企業競爭、技術進步會改變獲利能力
  • 遙遠的未來通常當成零或用永續法計算

DCF 計算步驟

步驟一：預測自由現金流

預估未來 5~15 年期間，不確定性越高估的越短。

步驟二：計算終值

終值衡量 DCF 預測期結束時，公司的未來價值。

方法：

  • 將第 10 期到無限多期後的現金流折現到第 10 期
  • 保守的話終值可以當成零

步驟三：計算折現率

用 WACC 確定所有未來現金流的折現率。

步驟四：計算淨現值（NPV）

將預估的 FCF 和終值，根據年數與折現率折算回到今天加總。

DCF 計算範例

範例情境

假設某公司：

  • 未來 10 年每股自由現金流都是 10 元
  • 10 年後終值（FV）= 125 元
  • 折現率 = 10%

計算過程

DCF = 10÷(1.1)¹ + 10÷(1.1)² + ... + 10÷(1.1)¹⁰ + 125÷(1.1)¹⁰
    = 9.1 + 8.3 + 7.5 + 6.8 + 6.2 + 5.6 + 5.1 + 4.7 + 4.2 + 3.9 + 48.2
    = 109.6 元

意義：

  • 今天用 109.6 元買入，未來每期得 10 元現金流、10 期後賣掉
  • 年化報酬率 = 10%
  • 如果現在股價 < 109.6 元 → 報酬率 > 10%（低估）
  • 如果現在股價 > 109.6 元 → 報酬率 < 10%（高估）

永續現金流折現公式

公式

永續價值現值 = 未來現金流 ÷ (折現率 - 成長率)

範例

  • 未來現金流 = 10 元
  • 折現率 = 10%
  • 成長率 = 2%

永續價值 = 10 ÷ (10% - 2%) = 10 ÷ 8% = 125 元

適合使用 DCF 的時機

✅ 適合

1. 現金流穩定可預測的公司

  • 債券（報酬可預測）

2. 價值投資人評估公司

  • 想買被低估的股票

❌ 不適合

1. 景氣循環型公司

  • 收入因商業周期顯著變化

2. 私營公司

  • 財務數據難取得

3. 初創公司、早期公司

  • 還沒有夠多銷售記錄

4. 科技公司、生技公司

  • 長期大量支出

DCF 優缺點

優點

優點說明
具體數字用假設、現金流預測、成長率得出價值
內在價值不受市場主觀情緒影響，比其他估價法客觀
時間價值考慮資金時間價值後的收益現值
一致性可分析不同類型公司，得出客觀一致的估值
內部報酬率可計算 IRR，比較不同投資機會

缺點

缺點說明
不適快速增長公司對不可預測公司不太有用
數據收集困難需要大量財務數據，花時間且不一定準
依賴預測「預測」常常不準確，只有預測準才能計算可靠估值
終值難估計終值代表預測期後所有年份的假設現金流，非常難估計
WACC 難評估折現率本身也是難以準確評估的數字

DCF 注意事項

四個常見錯誤

1. 過度複雜化 DCF

  • 不必要的細節和複雜性

2. 過度推崇 DCF

  • 變成銷售工具而非分析工具

3. 過度捍衛 DCF

  • 掩蓋估值中的不確定性

4. 不喜歡得出的答案

  • 投資人喜歡給出答案的模型

四個關鍵論點

1. Duh Proposition

  • 資產要能評價，預期現金流到了某個時間點必須是正的

2. 可以討厭 beta 仍然喜愛 DCF

  • DCF 沒有預先用現代投資組合理論包裝

3. 看資產生命期間，不是投資期間

  • DCF 評價資產在生命週期中的價值

4. 你可能會犯錯，但這不是你的問題

  • 得到估值幾乎可以保證會錯

實戰應用心法

查理蒙格的智慧

> 「我知道巴菲特都是這樣估價，但我從來沒有見他動手算過。」

真正夠好的投資機會：

  • 無論用哪種方式，算出來都會是好機會
  • 不會因為估價誤差，就變得很差
  • 大致估一下就知道是否值得投資
  • 誤差只會從「超好」變成「一般」，不會從「好」變「壞」

DCF 的正確心態

1. 不是精準數字：只是估計和猜測，幫我們釐清相對合理的範圍
2. 必然存在誤差：不可能估出正確合理股價
3. 了解比計算重要：理解 DCF 方法，對其他估價法的限制有更深入了解

心得：

DCF 是價值投資的「正統」估值方法，概念很漂亮——把未來所有現金流折現回現在，計算內在價值。但實際執行的問題是：預測本來就是不準的。

我的看法是：DCF 更像一個「思考框架」而非「計算工具」。它強迫你想清楚：

  • 這家公司未來能賺多少錢？
  • 多久才能回本？
  • 用什麼折現率才合理？

真正會用 DCF 的人，應該是拿來「驗證直覺」而不是「產生精確數字」。如果 DCF 算出來的價值和市價差很多，至少會讓你好奇「為什麼？」——是市場錯還是我看錯？

比起本益比這種簡化版，DCF 的好處是讓你看到全貌，壞處是每個假設都可能錯。投資的藝術就在這裡：你不用算得精準，但你要知道自己在做什麼。
