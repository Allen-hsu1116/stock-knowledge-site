OBV 能量潮指標 —— 從成交量看趨勢

是什麼

OBV（On-Balance Volume）又稱能量潮指標、人氣指標，是少數用「成交量」來計算的技術指標。由葛蘭維（Joe Granville）在 1960 年代提出，核心概念是「量先價行」——成交量比股價更早反映趨勢。

為什麼特別？

  • 大多技術指標（MA、MACD、KD、RSI）都以「價格」計算
  • OBV 以「成交量」計算，能看出主力、法人的進出軌跡
  • 可以快速辨識「有人氣」的熱門股票

怎麼算

公式很簡單：

  • 今日收盤價 > 昨日收盤價 → 今日 OBV = 昨日 OBV + 今日成交量（正值）
  • 今日收盤價 < 昨日收盤價 → 今日 OBV = 昨日 OBV - 今日成交量（負值）
  • 今日收盤價 = 昨日收盤價 → 今日 OBV = 昨日 OBV（不計算）

結果：OBV 是累積值，會畫成一條趨勢線，跟 K 線圖疊在一起看。

OBV 四大交易訊號

一、背離訊號（最重要）

頂背離（賣出訊號）：

  • 股價創新高，OBV 沒有創新高
  • 代表買盤力道衰退，賣壓正在增強
  • 後續可能反轉下跌

實例：

  • 情境：英業達（2356）股價從 100 元漲到 120 元創高，但 OBV 從 500 萬下降到 400 萬
  • 解讀：價格看似強勢，但量能跟不上，主力正在出貨
  • 操作：考慮逢高賣出或減碼

底背離（買進訊號）：

  • 股價創新低，OBV 沒有創新低
  • 代表賣盤力道減弱，買盤正在承接
  • 後續可能反轉向上

二、同步訊號（趨勢確認）

多頭同步（加碼訊號）：

  • OBV 線緩慢上升，股價同步上漲
  • 代表買盤穩健，中長期看好
  • 操作：可以考慮加碼或續抱

實例：

  • 情境：台積電（2330）OBV 從 1000 萬緩步上升到 1500 萬，股價從 500 元漲到 600 元
  • 解讀：量價齊揚，漲勢健康
  • 操作：續抱或逢低加碼

空頭同步（賣出訊號）：

  • OBV 線緩慢下降，股價同步下跌
  • 代表賣壓持續，中長期看淡
  • 操作：賣出或場外觀望

三、急速變化訊號（反轉警訊）

OBV 急速上升（賣出警訊）：

  • 代表買盤全力湧進，市場過熱
  • 主力不會幫大家抬轎，上漲已經乏力
  • 後續可能情勢扭轉，考慮逢高賣出

OBV 急速下跌（觀望訊號）：

  • 代表大量賣盤湧出，轉為空方趨勢
  • 股價止跌後通常還會盤整一段時間
  • 不要急著搶反彈，等 OBV 在低檔形成齒狀曲線再考慮進場

實例：

  • 情境：英濟（3294）OBV 急速上升後見高點，股價創高後反轉下跌
  • 錯誤操作：下跌過程中一直抄底
  • 正確操作：等 OBV 跌到低點後，觀察是否形成底部型態再進場

四、累積訊號（支撐壓力）

OBV 累積高點 = 壓力區：

  • OBV 長期累積後的大波段高點，會形成壓力
  • 股價在這區域容易反轉下跌
  • 一旦突破壓力區，後續漲勢更強勁

OBV 累積低點 = 支撐區：

  • OBV 長期累積後的大波段低點，會形成支撐
  • 股價在這區域容易止跌回穩
  • 一旦跌破支撐區，後續跌勢更猛

OBV 搭配均線實戰

怎麼做

均線設定：20 MA（月線）或 60 MA（季線）

多頭確認：

  • OBV 從底部上升 + 股價在均線之上
  • 判斷：多方力道強勢，可以進場或加碼

空頭確認：

  • OBV 從頂部下降 + 股價在均線之下
  • 判斷：空方力道強勢，應該觀望或做空

實例：

  • 情境：某股票 OBV 從 -500 萬上升到 0，同時股價突破 20 MA（從 90 元到 95 元）
  • 解讀：量能由負轉正，股價突破月線，多方訊號
  • 操作：可以考慮進場，停損設在跌破 20 MA

OBV 三大基本假設

1. 市場看法分歧愈大，成交量愈大

  • 多空看法不一致時，買賣熱絡，量能放大
  • 多空看法一致時，買賣冷清，量能萎縮

2. 重力原理

  • 股價上漲需要大量買盤支撐（主力進場）
  • 股價下跌不必耗費能量，自然有恐慌賣壓

3. 慣性原理

  • 熱門股票的人氣會持續一段時間
  • 冷門股票的成交量和股價會持續萎縮

OBV 使用注意事項

缺點一：落後指標

  • OBV 是根據歷史數據計算，不能預測未來
  • 只能作為警示訊號，不能單獨當作進出場依據

缺點二：不適合低量股

  • 成交量太低的股票，OBV 反應不明顯
  • 容易被利多利空消息影響波動

缺點三：不能單獨使用

  • OBV 原理簡單，容易被主力做量操控
  • 必須搭配其他技術指標（均線、KD、MACD）一起判斷

適用情境

  • 適合：高成交量的熱門股、大盤指數
  • 不適合：低成交量的冷門股、價格震盪劇烈的商品（如加密貨幣）

心得

OBV 是「量先價行」的具體實現，核心邏輯很直覺：「量是價的燃料」。

之前學過「量價關係六種型態」，OBV 把成交量累積成一條趨勢線，讓量價關係更視覺化，不需要每天盯著成交量柱狀圖猜測。

關鍵收穫：
1. 背離是最強訊號：股價和 OBV 走勢不一致時，OBV 通常比較準，因為量不會騙人
2. 急速變化是警訊：OBV 急升不是好事（過熱），急跌後也不要急著搶反彈（會盤整）
3. 累積高低點 = 支撐壓力：這個概念可以跟之前學的「支撐壓力判斷」搭配使用
4. 不能單獨用：OBV 只是輔助指標，要搭配均線、KD、MACD 確認趨勢
5. 熱門股才準：低量股的 OBV 參考價值很低，要選成交量大、有法人進出的股票

實戰應用：

  • 可以用 OBV 找「有人氣」的股票：OBV 持續上升 = 主力在買
  • 用背離判斷轉折點：頂背離 = 出貨訊號，底背離 = 進場機會
  • 搭配均線確認多空：OBV 上升 + 股價在均線上 = 多頭

OBV 和之前學的「均線扣抵」、「KD 指標」一樣，都是技術分析的基本功，持續累積這些工具，選股時就能過濾掉更多假訊號。
