核心知識

當沖選股三大條件

是什麼：三個最基本的篩選標準，幫你快速挑出適合當沖的股票。

怎麼做：
1. 成交量：每日至少 2000 張以上
2. 波動度：當日振幅要夠，才有價差空間
3. 題材：要有市場關注的焦點，才會有資金湧入

為什麼重要：

  • 成交量太小 → 進出困難、滑價嚴重、價格容易被操弄
  • 沒波動 → 沒利潤空間，白費力氣
  • 沒題材 → 只是零星單子，行情撐不久

成交量篩選實戰

是什麼：確保股票有足夠流動性，進出順暢。

怎麼做：
1. 打開成交量排行榜
2. 挑選前段班的股票（每日 2000 張以上）
3. 確認成交量的「品質」——是散戶單還是主力單

成交量大 vs 小的差異：

項目成交量大成交量小
價格合理，滑價少不理想，容易被坑
出場順暢卡住賣不掉
操縱不容易被單一大戶操控幾筆大單就能拉殺

實例：

  • 冷門股成交量只有幾百張，進場後發現沒人接，只能一路降價砍出
  • 熱門股成交量上萬張，掛單秒成交，進出都有依據

適用時機：所有當沖操作的第一步篩選

波動度判斷方法

是什麼：確認股票當天「夠不夠活躍」，有沒有價差空間。

怎麼做：
1. 肉眼觀察：看走勢圖，線型上下起伏明顯才適合
2. 當日振幅：高低點差距越大越好
3. ATR 指標：用「平均真實波幅」量化波動度

實例：

  • 股價整天在 100~101 之間震盪 → 波動度不足，不適合當沖
  • 股價從 100 衝到 105 又殺回 98 → 波動度充足，可以操作

適用時機：

  • 大盤熱鬧時，熱門族群波動度會放大
  • 大盤冷清時，選股要更嚴格

題材連動效應

是什麼：市場關注的焦點會吸引資金集中，帶動整個族群的股票。

題材類型：
1. 新聞題材：政府政策補助、重大利多
2. 產業趨勢：AI、電動車、半導體等熱門產業
3. 財報公告：營收大增、重大合作

怎麼做：
1. 問自己：今天市場的熱門題材是什麼？
2. 確認：這檔股票有沒有在題材之內？
3. 判斷：是主流核心股，還是跟風的小弟？

實例：

  • AI 熱潮時，台積電、伺服器、散熱、記憶體族群都會跟著放量
  • 主流核心股（如台積電）波動更穩定，跟風股（二線股）波動更劇烈

適用時機：選股時優先挑有題材的股票

進階選股技巧：籌碼面

是什麼：觀察法人與主力的短線布局，判斷資金流向。

怎麼做：
1. 三大法人連續買超：常常帶動市場關注，增加當沖機會
2. 當沖熱門標的：成交量裡有很大比例來自當沖，代表短線資金集中

實例：

  • 外資連續 3 天買超某股 → 該股可能成為當沖熱門標的
  • 投信作帳行情期間 → 相關個股波動度放大

適用時機：想提高勝率時，加上籌碼面篩選

進階選股技巧：關鍵價位

是什麼：市場大多數人都在關注的價格位置，突破或跌破會引發大動作。

關鍵價位類型：

  • 支撐壓力
  • 前一高點、低點
  • 整數關卡（如 100 元、200 元）

怎麼做：
1. 標出關鍵價位
2. 等待突破或跌破確認
3. 突破關鍵價位 → 往往帶動追價，出現急漲
4. 跌破關鍵價位 → 容易引發恐慌，造成急殺

實例：

  • 某股壓力在 150 元，突破後瞬間衝到 160 元
  • 某股支撐在 100 元，跌破後一路殺到 90 元

適用時機：避免「追在高點、砍在低點」

進階選股技巧：市場氛圍與族群輪動

是什麼：單一股票的表現往往和大盤、族群息息相關。

怎麼做：
1. 大盤強勢：熱門族群的個股更容易爆量上攻
2. 大盤疲弱：選股要更謹慎，找「逆勢抗跌」或「題材獨強」的標的
3. 族群效應：若某產業整體上整體上漲，龍頭股和二線股往往會一起活絡

實例：

  • 大盤大漲 200 點，AI 族群全面上攻 → 龍頭股、二線股都是當沖標的
  • 大盤下跌，某題材股逆勢上漲 → 可能是主力在作價

適用時機：判斷當天大盤氛圍後，調整選股策略

當沖三種操作模式

是什麼：根據交易節奏，當沖可分為 Tick流、一波流、日內波三種。

特性Tick流一波流日內波
賺賠比低中高
出手次數多中少
資金效率高中低
操作核心短進短出當前行情走勢盤前選股
是否留倉否否可
平均獲利<1%1~5%>5%

怎麼做：
1. Tick流：捕捉微小波動，每次進出場間隔數秒到數分鐘
2. 一波流：掌握當日震盪最大的短期趨勢，追蹤成交量與大單流向
3. 日內波：研判有機會出現長紅/長黑K棒才進場，可轉波段持有

實例：

  • Tick流：股價在 100~100.5 元間快速來回，每次賺 0.2 元
  • 一波流：股價從 100 衝到 103，抓住這段 3% 的行情
  • 日內波：開盤前選定某強勢股，當天漲 7%，留倉繼續賺波段

適用時機：

  • Tick流：頻繁震盪、成交量活躍的標的
  • 一波流：市場熱門族群與題材股
  • 日內波：基本面、技術面、籌碼面都篩選過的強勢股

日內交易時段分析

是什麼：台股在不同時段有不同特性，懂得順勢而為才能把握機會。

時段時間特性操作建議
盤前8:45~9:00觀察期貨走勢、調整策略做好心理準備
開盤9:00~9:30劇烈反應消息、快攻時間短進短出機會多
早盤9:30~10:30走勢逐漸明朗技術型態確認後進場
中盤10:30~12:30成交量下降、波動縮小減少出手次數
尾盤12:30~13:30當沖出場、波段進場注意拉尾盤或下殺

實例：

  • 開盤 30 分鐘內強勢股快速搶攻漲停 → 錯過就別追
  • 中盤股價在均價線上下震盪 → 觀望為主
  • 尾盤突然拉尾盤 → 注意是否主力在作價

適用時機：根據時段調整操作策略

短線技術指標實戰

是什麼：用來判斷進出場時機的技術工具。

常用指標：
1. 當日開盤價：多空分界線，上方紅K、下方黑K
2. 當日均價線：平均交易成本，上方偏多、下方偏空
3. 日K線關鍵點位：近期高低點、昨日收盤價
4. 日K線均線：上揚均線是支撐，下彎均線是壓力

實戰案例：

  • 凱美開盤後量增突破昨日高點 → 可進場
  • 聯光通跳空開高遇月線反壓 → 跌回均價線下，不適合作多

適用時機：

  • 開盤價和均價線是日內多空判斷核心
  • 日K線均線是波段位置判斷依據

勝率與賺賠比計算

是什麼：判斷一套交易策略是否值得執行的核心數字。

公式：

  • 勝率 = 獲利次數 / 總交易次數
  • 賺賠比 = 平均獲利 / 平均虧損
  • 期望值 = (賺賠比 × 勝率) − (1 − 勝率)

實例：

  • 勝率 40%、賺賠比 3
  • 期望值 = 3 × 0.4 − 0.6 = 0.6
  • 代表每次交易預期獲利 0.6 元

怎麼做：
1. 記錄每次交易的獲利與虧損
2. 計算勝率與賺賠比
3. 期望值為正才繼續使用該策略

適用時機：檢討交易策略是否有效

當沖新手三大陷阱

陷阱一：選冷門股

  • 冷門股成交量小，滑價嚴重、出場困難
  • 是當沖的大地雷

陷阱二：只看新聞標題

  • 看到利多就衝進去，結果股價不動
  • 題材要配合成交量才有行情

陷阱三：忽略波段位置

  • 股票已經漲一大段，當天強勢就追高
  • 結果主力在高檔出貨，變成「接刀俠」

心得

當沖選股其實就是一個篩選流程：先看成交量、再看波動度、最後看題材。這三個條件缺一不可，選錯股票，技術再好也沒用。

最關鍵的觀念是「不要跟冷門股玩」。成交量小代表沒人跟你玩，進去就被坑。寧可錯過熱門股的一段行情，也不要去冷門股送死。

另外，當沖有三種模式：Tick流、一波流、日內波。新手建議從「一波流」開始，捕捉當日主要趨勢，而不是頻繁進出。日內波則需要更強的選股能力，適合有經驗的人。

最後，一定要懂「時段」。開盤 30 分鐘是快攻時間，波動大但風險也大；中盤是觀望期，減少出手；尾盤要注意拉尾盤或下殺的陷阱。順著時段的特性操作，才不會被市場節奏搞亂。
