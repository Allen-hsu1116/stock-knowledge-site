基本面分析：本益比評價法

第 2 回合學習 - 基本面分析：本益比評價法

本益比基本概念

本益比是什麼？

定義：本益比 = 股價 ÷ 每股盈餘（EPS）

意義：買進股票後，多久可以回本？

實例：

  • A公司股價100元，EPS 5元 → 本益比 = 20倍 → 需要20年回本
  • B公司股價100元，EPS 10元 → 本益比 = 10倍 → 需要10年回本
  • B公司相對便宜，因為回本時間較短

本益比與投資報酬率

本益比的倒數 = 投資報酬率

  • 本益比 10倍 → 報酬率 10%
  • 本益比 20倍 → 報酬率 5%
  • 本益比 100倍 → 報酬率 1%

本益比兩種計算方式

預估本益比（Forward P/E）

怎麼算：股價 ÷ 預估未來EPS

用途：前瞻性評估，反映市場預期

注意：

  • 預估EPS可能不準確
  • 公司可能低估/高估預期
  • 法人預估也不一定準

實例：

  • 公司預估下季EPS 2元，財報出來3元 → 優於預期，股價上漲
  • 公司預估EPS 5元，財報出來2元 → 不如預期，股價下跌

歷史本益比（Trailing P/E）

怎麼算：股價 ÷ 過去4季EPS合計

優點：客觀、數據明確

限制：假設未來EPS和過去一樣，但實際不一定

建議：使用近4季EPS合計，比去年年度EPS更能反映近期變化

實例（台積電 2025/6/13）：

  • 股價：1035元
  • 近4季EPS：50.51元（9.56+12.55+14.45+13.95）
  • 本益比：20.49倍

本益比使用限制

不適用情況

1. EPS為負或接近零：本益比算不出來或失真
2. 獲利波動大：營建業、原物料股
3. 時賺時賠：景氣循環股
4. 高速成長期：Amazon、Tesla早期EPS接近零但股價高

替代方案：

  • 股價現金流量比（P/CF）
  • 股價營收比（P/S）
  • 現金流量折現法（DCF）

比較基準

本益比不能跨產業比較，要比就跟：

  • 同產業競爭對手
  • 公司歷史區間

本益成長比 PEG

為什麼需要PEG？

本益比沒考慮「成長性」。

實例：

  • A公司：本益比12倍，年成長率5%
  • B公司：本益比12倍，年成長率20%
  • 表面上兩家公司本益比相同，但B公司成長性高，投資價值更高

PEG計算公式

公式：PEG = 本益比 ÷ 預估淨利成長率

實例：

  • A公司：12 ÷ 5 = 2.4倍
  • B公司：12 ÷ 20 = 0.6倍

PEG判斷標準

PEG值意義操作建議
< 0.66大幅低估買進訊號
0.66-0.75低估可考慮買進
= 1合理持有
> 1.2高估可考慮賣出

成長性與合理本益比

吉姆·史萊特原則：

  • 淨利成長率 15% → 合理本益比 15倍
  • 淨利成長率 20% → 合理本益比 20倍
  • 淨利成長率 30% → 合理本益比 30倍

本益比河流圖

什麼是河流圖？

將不同時間的股價與本益比繪製在同一張表，形成類似河流的樣子。

怎麼看：

  • 股價在河流偏上方 → 本益比處於歷史高點
  • 股價在河流偏下方 → 本益比處於歷史低點

判斷：

  • 本益比在歷史低點 + 公司營運正常 → 可能是好買點
  • 本益比在歷史高點 + 成長性有限 → 可能太貴

本益比實戰應用

判斷股價貴便宜

步驟：
1. 確認公司營運穩定、EPS可預測
2. 計算歷史本益比區間
3. 找出同產業龍頭股的本益比作參考
4. 預估未來EPS成長性
5. 計算PEG判斷是否低估

計算合理買進價

公式：合理股價 = 預估EPS × 合理本益比

實例：

  • 公司合理本益比15倍，便宜本益比12倍
  • 預估未來EPS 2元
  • 合理價：15 × 2 = 30元
  • 便宜價：12 × 2 = 24元

常見誤區

1. 低本益比≠便宜：可能產業前景差、獲利衰退
2. 高本益比≠貴：可能是成長股、市場看好
3. 只用歷史EPS：忽略未來成長性
4. 忽略產業比較：不同產業本益比差異大

心得

本益比是最常見的估值工具，但很多人用錯了。關鍵在於：

1. 本益比要看產業和成長性：成長股本益比可以高，但要用PEG來判斷是否合理
2. 預估EPS比歷史EPS重要：過去不等於未來，要評估公司成長性
3. 河流圖是好工具：一眼看出本益比是否在歷史高點或低點
4. PEG比純本益比更完整：把成長性納入考量，避免「便宜買到爛公司」

最實用的是合理價計算：預估EPS × 合理本益比 = 可以進場的價格。這比憑感覺買賣科學多了。
