一、盈餘品質是什麼？為什麼重要？

核心觀念：$1盈餘 ≠ $1現金。

會計制度採用「應計基礎」（Accrual Basis），公司帳面上的盈餘包含了大量非現金項目，這些叫應計項目（Accruals）。包括：

  • 應收帳款增加（東西賣了但錢還沒收到）
  • 折舊攤銷（非現金費用）
  • 存貨增加（買了但還沒賣出）

盈餘品質（Earnings Quality）就是在問：這家公司的盈餘，有多少是真的現金流入？還是只是帳面上的數字遊戲？

二、Sloan 應計項目異象（Accrual Anomaly）

發現者：Richard Sloan（1996年）

Sloan 在 1996 年的論文發現了一個驚人的事實：

> 應計項目高的公司，未來股價表現差；應計項目低的公司，未來股價表現好。

這個現象被稱為「應計項目異象」，是市場中最被廣泛研究且持久的市場異常現象之一。

歷史回測數據

  • 1962-2001年（美股市場，40年）：買入最低應計項目公司 + 做空最高應計項目公司 → 年均複合報酬率 18%，同期 S&P 500 僅 7.4%
  • 台股回測（曹壽民等，2008）：台股也存在應計項目高估現象
  • 全球市場（LaFond，17國研究）：應計項目異象是全球性現象

為什麼會存在？

盈餘 fixation 假說（Earnings Fixation Hypothesis）：
1. 投資人過度關注「盈餘」這個單一數字
2. 忽略了盈餘中的「現金流」和「應計項目」兩個組成部分的差異
3. 現金流部分是未來盈餘的更好預測因子，應計項目的持續性較差
4. 結果：投資人對高應計項目公司過度樂觀（高估），對低應計項目公司過度悲觀（低估）

三、Sloan Ratio 三種計算法

方法一：Sloan Ratio（現金流量法）

> Sloan Ratio = (Net Income − CFO − CFI) / Total Assets

  • CFO = 營業現金流
  • CFI = 投資現金流
  • Total Assets = 總資產

判讀標準：

Sloan Ratio判斷
-10% ~ +10%✅ 安全區 — 盈餘品質良好
-25% ~ -10% 或 +10% ~ +25%⚠️ 警告區 — 應計項目累積中
< -25% 或 > +25%🔴 危險區 — 盈餘很可能由應計項目組成

注意：如果 Sloan Ratio 連續多季或多年處於危險區，那幾乎可以確定公司在玩會計遊戲。

方法二：資產負債表應計比率（Balance Sheet Accrual Ratio）

步驟：
1. 計算淨營業資產（NOA）：
> NOA = (流動資產 − 現金) + (非流動資產) − (流動負債 − 短期借款) − (非流動負債)

2. 計算應計項目：
> BS Accruals = 當期 NOA − 前期 NOA

3. 計算比率：
> BS Accrual Ratio = BS Accruals / 平均 NOA

方法三：Sloan 原始公式（學術版）

> Accruals = [(Δ流動資產 − Δ現金) − (Δ流動負債 − Δ短期借款 − Δ應付所得稅) − 折舊費用] / 前期總資產

Δ = 與去年同期的變動數

方法四：財報狗簡易法

> (營業現金流 − 淨利) / 總資產

或更直觀的：
> 營業現金流 / 淨利

這個方法最簡單，但 Sloan 原始公式的選股效果更好（回測顯示排除高應計項目後，投資組合報酬比未調整前高出 2% 以上，夏普值也大幅改善）。

四、Sloan Ratio 實戰選股策略

策略一：排除法（最簡單、最實用）

怎麼做：
1. 每年5月（所有公司年報出爐後），計算所有股票的 Sloan Ratio
2. 排除 Sloan Ratio > 10% 的公司（約占整體的 30%）
3. 在剩餘的 70% 中做選股

回測效果：

  • 排除高應計項目公司後，每年預期報酬提高約 2%
  • 夏普值大幅改善
  • 不需要做空，只需「不買」爛的

策略二：多空策略（機構級）

怎麼做：
1. 將所有股票按 Sloan Ratio 分成十等分（Decile）
2. 做多最低應計項目的 10%（第十組）
3. 做空最高應計項目的 10%（第一組）
4. 每年5月再平衡

回測效果（1966-2003年美股）：

  • 年化報酬：7.5%
  • 年化波動：10.26%
  • 最大回撤：-35.58%
  • 夏普比率：0.34

策略三：結合其他指標（最推薦）

Sloan Ratio 不是單獨使用的指標，而是過濾器！

實戰 SOP：
1. 用 Sloan Ratio 過濾掉盈餘品質差的公司（Sloan Ratio > 10%）
2. 在剩餘股票中用其他選股條件篩選：

  • PEG < 1（成長合理價）

3. 每年5月重新篩選

五、應計項目的紅旗訊號

以下情況出現時，要特別警覺：

1. 盈餘跳增但 Sloan Ratio 也跳增 → 盈餘成長是假的，靠應計項目撐出來的
2. 營收成長率高於產業平均 + Sloan Ratio 也高於產業平均 → 可能是通路塞貨
3. 高資產負債表應計項目 → 公司快速擴張資產基礎，可能來自併購或過度投資
4. 高應計項目的公司通常有高 ROE → 但這個 ROE 是虛的，不可持續
5. 低應計項目的公司 ROE 低於平均 → 分析師可能低估了，反而是撿便宜的機會
6. 低應計項目的公司在縮減資產 → 可能是分拆、回購或大額減損，這些通常是好信號

六、應計項目異象的兩種類型

Detzel, Schabel & Strauss（2017）發現，應計項目異象其實是兩個不同的現象：

類型來源性質市場情緒關係
投資相關應計項目併購、資本支出、擴張風險溢價與市場情緒負相關
非投資應計項目應收帳款、存貨、遞延收入錯誤定價與市場情緒正相關

實戰意義：

  • 非投資應計項目的異象更像是「錯誤定價」，可以用來選股獲利
  • 投資相關應計項目的異象更像是「風險溢價」，不一定是套利機會
  • 優先關注非投資應計項目（應收帳款暴增、存貨暴增）

七、異象的衰減與復甦

  • 1962-2002年：異象穩定存在，顯著超額報酬
  • 2003年後：隨著分析師現金流預測普及，異象有所減弱
  • 2008年後：金融海嘯後，盈餘品質信號強勁回歸

原因：分析師開始提供現金流預測，市場對應計項目的定價效率提高了。但高應計項目公司通常是小盤股、流動性差、交易成本高，機構不願意碰，所以異象仍存在。

八、台股實戰案例

假設情境：某電子股

項目數值
淨利50億
營業現金流20億
投資現金流-5億
總資產400億

Sloan Ratio = (50 − 20 − (−5)) / 400 = 35 / 400 = 8.75%

✅ 在安全區（< 10%），但接近警告邊界，值得持續追蹤。

如果營業現金流只有 5 億呢？

Sloan Ratio = (50 − 5 − (−5)) / 400 = 50 / 400 = 12.5%

⚠️ 進入警告區！盈餘有大量應計項目支撐，品質堪慮。

心得

這個 Sloan Ratio 真的是個被低估的武器。大多數投資人只看 EPS 和營收成長率，但完全不關心這些盈餘背後有多少是真的現金。這就像一個人月薪 10 萬但 7 萬都是欠條——看起來收入很高，實際口袋裡沒幾塊錢。

最實用的方式就是把 Sloan Ratio 當成過濾器：不需要做空，只要不買 Sloan Ratio > 10% 的公司，就能每年多賺 2%。這個動作幾乎零成本，但長期效果顯著。

另外一個重要洞察：高應計項目的公司通常也有高 ROE。所以如果只看 ROE 選股，很可能選到「看起來很會賺但其盈餘品質很差」的公司。這解釋了為什麼有些 ROE 很高的公司股價就是不漲——市場最終會看穿這些「紙上富貴」。

結合之前學過的 F-Score、FCF 選股、Beneish M-Score，現在可以組成一個多層盈餘品質檢查：
1. Sloan Ratio < 10%（應計項目正常）
2. F-Score ≥ 7（財務趨勢改善）
3. FCF為正且FCF Margin > 5%（真的有現金進來）
4. Beneish M-Score < -1.78（沒有造假跡象）

這四道關卡全過的公司，盈餘品質才真正靠譜。

一、P/S股價營收比（Price-to-Sales Ratio）— 本益比不能用的時候，靠它估值

是什麼

股價營收比（P/S Ratio，又稱市銷率、PSR）= 市值 ÷ 年營業收入，或等價地 = 每股股價 ÷ 每股營收。

告訴你：投資人願意為每1元營收支付多少元。

由Kenneth L. Fisher（成長股大師Philip Fisher的兒子）在1984年《Super Stocks》一書中提出，專門用來識別被低估的股票。

為什麼需要P/S？本益比不能用的三種情境

情境一：公司虧損中

  • EPS為負 → 本益比算不出來或為負值
  • 例子：Amazon 1997-2003年連續虧損，PE完全失靈
  • 但營收永遠為正 → P/S可以正常計算

情境二：獲利波動劇烈

  • 景氣循環股盈餘大起大落，PE忽高忽低
  • 營收相對穩定 → P/S判斷更可靠

情境三：高成長公司大量再投資

  • 成長型公司把利潤全部再投資，EPS極低
  • PE可能幾百倍，看不出估值意義
  • P/S直接看市場給多少「營收溢價」

P/S計算公式

> P/S = 市值 ÷ 年營業收入
> P/S = 每股股價 ÷ 每股營收

怎麼做：
1. 取得公司最新市值（股價 × 在外流通股數）
2. 取得近四季或年度營業收入
3. 兩者相除即得P/S

實例：

  • Amazon 2003年：市值約140億美元，年營收約75億美元 → P/S ≈ 1.9
  • Amazon 1999年（泡沫高峰）：P/S曾接近40倍 → 嚴重高估
  • Amazon 成熟期：P/S穩定在1.5~3倍之間

P/S判讀 — 沒有絕對標準，只能相對比較

關鍵原則：P/S沒有像PE那樣的「合理區間」，因為不同產業的淨利率天差地遠。

產業淨利率合理P/S範圍原因
軟體SaaS20-40%5-15倍淨利率高，1元營收利潤多
半導體15-30%3-8倍高毛利但資本支出大
消費品5-15%1-3倍穩定但利潤薄
零售/電商1-5%0.3-1倍利潤極薄但週轉快
生技（虧損中）N/A看管線價值沒有營收或利潤為負

怎麼做：
1. 先確認產業的合理P/S範圍
2. 與同業比較：同產業同淨利率的公司互相比較
3. 與自身歷史比較：P/S在歷史區間的位置
4. 搭配淨利率一起看

P/S與淨利率的關係 — 為什麼不能跨產業比較

P/S和淨利率之間有一個重要的理論關係：

> P/S ≈ 本益比 × 淨利率

推導：P/S = 股價/營收 = (股價/EPS) × (EPS/營收) = PE × 淨利率

實例：

  • A公司：淨利率20%、PE 15倍 → P/S = 15 × 20% = 3倍
  • B公司：淨利率3%、PE 15倍 → P/S = 15 × 3% = 0.45倍

同樣PE 15倍，A公司P/S是3倍、B公司P/S是0.45倍——差距6.7倍！所以跨產業比P/S毫無意義。

P/S的五大使用陷阱

陷阱一：P/S低≠便宜

  • 低P/S可能是因為市場正確地看壞公司前景
  • 營收衰退的公司，低P/S只是在反映未來營收將下滑
  • 一定要搭配營收成長率一起看

陷阱二：P/S不考慮利潤率和負債

  • 兩家P/S相同的公司，淨利率差5倍，估值意義完全不同
  • 高負債公司的低P/S可能是因為債務風險壓抑了市值

陷阱三：營收可以灌水

  • 本益比用的是稅後淨利，已扣掉所有費用
  • P/S用的是營收，公司可以透過總額法認列灌水
  • 一定要確認營收認列方式（總額法 vs 淨額法）

陷阱四：併購會扭曲營收

  • 併購後營收暴增，P/S突然下降，但這不代表變便宜
  • 併購前的P/S和併購後的P/S不可直接比較

陷阱五：成長性才是關鍵

  • P/S用的是過去營收，但決定股價的是未來預期
  • 衰退中的公司P/S低，是因為營收持續萎縮
  • 成長中的公司P/S高，是因為市場預期營收會大幅增長

P/S實戰選股SOP

怎麼做（三步篩選法）：

1. 第一步：與同業比較P/S

  • 找出同產業、淨利率相近的公司

2. 第二步：確認營收成長率

  • 營收年增率 > 15% + P/S低於同業 → 值得深入研究

3. 第三步：交叉驗證其他指標

  • P/S搭配PE：如果P/S低但PE合理 → 獲利能力正常，可能被低估

適用時機：

  • 成長型公司（虧損中但有營收）→ P/S是首選估值指標
  • 景氣循環股（盈餘波動大）→ P/S比PE更穩定
  • 搭配PE和PBR使用，形成完整的估值框架

二、可轉換公司債（Convertible Bond, CB）— 進可攻退可守的投資工具

是什麼

可轉換公司債（CB）是企業發行的債券，同時具備「債券」和「股票」雙重特性：

  • 債券特性：定期領息、到期還本，下檔有保護
  • 股票特性：可在約定條件下轉換為發行公司的股票，上檔有參與

核心邏輯：股價下跌時，持有到期領回本金+利息（抗跌）；股價大漲時，轉換成股票賺取價差（跟漲）。

CB三大核心要素

1. 轉換價格（Conversion Price）

  • 可轉債轉換為股票的約定價格
  • 發行時就確定，不會隨意改變
  • 除非發生除權息、減資、增資等事件才會調整

2. 轉換比率（Conversion Ratio）

  • 一張可轉債可以換多少股股票
  • 公式：轉換比率 = 可轉債面額（10萬元）÷ 轉換價格

實例：

  • 轉換價格20元 → 可換 100,000 ÷ 20 = 5,000股 = 5張股票
  • 轉換價格50元 → 可換 100,000 ÷ 50 = 2,000股 = 2張股票
  • 轉換價格100元 → 可換 100,000 ÷ 100 = 1,000股 = 1張股票

3. 轉換價值（Conversion Value）

  • 可轉債轉換成股票後的市值
  • 公式：轉換價值 = 股價 × 轉換比率

實例：

  • 轉換價格50元，可換2,000股
  • 當股價60元時：轉換價值 = 60 × 2,000 = 120,000元
  • 當股價40元時：轉換價值 = 40 × 2,000 = 80,000元

CB的定價邏輯 — 債券底線 + 股票選擇權

可轉債的價值 = max(債券價值, 轉換價值) + 選擇權時間價值

三種狀態：

狀態條件CB行為投資策略
深度價外股價遠低於轉換價格CB像純債券，價格接近債券底線領息為主，等待反轉
價平附近股價接近轉換價格CB同時受債券和股票影響最佳投資區間
深度價內股價遠高於轉換價格CB像股票，價格跟隨股價已有轉換利潤，注意贖回風險

距離比例 — CB投資時機的關鍵指標

> 距離比例 = 股價 ÷ 轉換價格

最佳投資區間：距離比例 0.8 ~ 0.95

  • 距離比例 0.5：股價要漲100%才能到轉換價格，難度太高
  • 距離比例 0.8：股價再漲25%就到轉換價格，CP值高
  • 距離比例 0.95：股價接近轉換價格，上檔空間近
  • 距離比例 1.3：股價已超過轉換價格30%，要小心公司執行贖回條款

怎麼做：
1. 找出有發行可轉債的公司
2. 計算距離比例 = 現股價 ÷ 轉換價格
3. 篩選距離比例在0.8-0.95的標的
4. 進一步分析公司基本面和產業前景
5. 基本面看好 + 距離比例適中 → 買進CB

轉換溢價率 — CB是否值得買的量化判斷

> 轉換溢價率 = (CB市價 - 轉換價值) ÷ 轉換價值 × 100%

實例：

  • CB市價 110,000元，轉換價值 100,000元
  • 轉換溢價率 = (110,000 - 100,000) ÷ 100,000 × 100% = 10%

判讀：

  • 溢價率 0-5%：CB接近轉換價值，接近「價內」，投資價值高
  • 溢價率 5-15%：適度溢價，為選擇權付出的合理代價
  • 溢價率 15-30%：溢價偏高，需要更大的股價漲幅才能回本
  • 溢價率 > 30%：嚴重溢價，投資價值低

CB投資的四大陷阱

陷阱一：虛假的低轉換價格

  • 公司股價5元，轉換價格6元，看起來很近
  • 但如果基本面差，股價可能繼續跌到3元、2元
  • 轉換價格再低，也要搭配基本面分析

陷阱二：過高的轉換溢價率

  • 溢價率超過15%，股價要漲很多才有轉換價值
  • 等於花很多錢買一個可能不值得的選擇權

陷阱三：即將到期的遠價外CB

  • 剩餘時間短 + 股價遠低於轉換價格 = 時間價值快歸零
  • 除非有事件催化，否則只能當普通債券持有到期

陷阱四：忽略信用風險

  • CB本質上還是債券，公司倒閉就拿不回本金
  • 投資前必須確認發行公司的財務狀況
  • 信用評等低的公司，CB的債券保護力也低

CB對股價的影響 — 轉換價格是天花板

當股價超過轉換價格時，可轉債投資人會轉換成股票並賣出，形成短期賣壓。

怎麼做：
1. 查詢公司可轉債的轉換價格和到期日
2. 當股價接近轉換價格時，預期賣壓增加
3. 轉換價格往往成為短期壓力區（天花板效應）
4. 股價突破轉換價格後，可轉債大量轉換會稀釋EPS

適用時機：

  • 看到中小型股股價在特定價位遇到壓力 → 查是否有可轉債轉換壓力
  • 公司發行可轉債的目的：利率低+不用還債（轉換後股本膨脹）
  • 財務吃緊的公司可能炒作股價讓CB轉換 → 對股票投資人不利

CB除權息調整機制

轉換價格不是一成不變的。當公司除權息時，轉換價格會自動調整：

> 新轉換價格 = 原轉換價格 × (1 - 現金股利 ÷ 每股時價)

實例：

  • 轉換價格100元，公司每股配息3元，除息前股價100元
  • 新轉換價格 = 100 × (1 - 3/100) = 97元
  • 除息後股價變97元，轉換價格也變97元 → 投資人權益獲得保護

重要提醒：

  • 重設條款（股價下跌時重新設定更低的轉換價格）已經不再使用
  • 目前市面上的CB都沒有重設條款
  • 轉換價格只會因除權息等事件調整，不會因股價下跌而自動下修

CB投資策略總結

策略一：價外CB + 基本面看好 = 低風險高報酬

  • 距離比例0.8-0.95，公司基本面佳
  • 最壞情況：持有到期拿回本金+利息
  • 最好情況：股價上漲，轉換賺價差

策略二：價內CB = 股票替代品

  • 距離比例>1.0，CB價格跟隨股價
  • 優點：下檔有債券保護
  • 缺點：溢價已反映，上漲空間有限

策略三：CB套利

  • 當CB市價低於轉換價值時，買入CB同時放空股票
  • 獲利 = 轉換價值 - CB市價 - 交易成本
  • 需要考慮流動性和融券成本

心得

今天學了兩個主題，一個是估值工具箱裡的P/S Ratio，一個是進可攻退可守的可轉債。

P/S最核心的概念是：當本益比不能用時，P/S是你的替代方案。虧損中的公司、高成長公司、景氣循環股——這些PE失靈的場景，P/S依然有效。但P/S最大的陷阱就是「低P/S=便宜」的直覺，因為不同產業的淨利率天差地遠，軟體公司P/S 5倍可能比零售公司P/S 0.5倍還便宜。記住 P/S ≈ PE × 淨利率 這個公式，就能理解為什麼跨產業比P/S沒意義。

Kenneth Fisher提出P/S的初衷，其實就是為了抓到Amazon這種「營收在長但利潤被再投資吃掉」的公司。Amazon在2003年P/S約1.9倍，看PE完全失靈，但P/S清楚顯示：市場只給每1元營收1.9倍溢價，在成長型公司中算便宜。回頭看這就是Amazon最好的買點之一。

可轉債則是另一個維度的思考。以前只知道CB是「債券+股票」，但距離比例這個概念太實用了：股價÷轉換價格 = 0.8~0.95是最佳投資區間。這等於告訴你：當股價離轉換價格還有5%~20%的距離時，是風險報酬比最好的時機。2020年疫情期間那個航運CB的案例太經典了——股價4.72元、轉換價格10.4元、距離比例0.45，看起來很遠，但基本面分析+週期判斷+CB的債券保護，等於用有限的風險賭一個大反轉。

轉換價格是股價天花板這個概念也很重要。以後看到中小型股在特定價位遇到壓力，第一個要查的就是有沒有可轉債轉換壓力。公司為了不還債會想辦法讓股價突破轉換價格，這對股票投資人可能不一定是好事。

估值工具箱到現在已經很完整了：PE看成熟股、PBR看循環股和金融股、PEG看成長股、P/S看虧損股和高成長股、DCF算內在價值、DDM看穩定配息股。每個工具有適用場景，用錯工具比不用工具更危險。

一、商譽Goodwill：併購溢價的白話文就是「買貴了」

是什麼

商譽 = 收購價 - 被併購公司淨資產公允價值

白話文：你花150億買一家淨值120億的公司，那30億就是商譽。商譽不是「品牌好聲譽高」，而是「你買貴了30億」。

怎麼做：商譽判讀五步法

1. 打開資產負債表，找無形資產

  • 台股：商譽藏在「無形資產」項目內，要再看附註拆分

2. 看商譽佔總資產比例

  • 低於5%：正常，併購便宜（如桂盟5306，商譽佔總資產2%）

3. 併購後追蹤總資產週轉率

  • 總資產週轉率 = 營收 ÷ 總資產

4. 檢查商譽變動趨勢

  • 商譽突然大增 → 新併購案，查併購標的和價格

5. 查看財報附註

  • 無形資產的組成：商譽多少、客戶關係多少、電腦軟體多少

實例

  • 如興(4414)慘案：花108.6億併購中國玖地，淨資產僅48.9億，商譽59.5億佔併購金額55%。併購後總資產週轉率一路下滑，2022年爆發財務危機被打入全額交割。國發基金以18.6元投資14.8億全數慘賠。
  • 桂盟(5306)成功案：花10億併購，淨資產9.53億，商譽僅0.47億（佔總資產2%）。併購後資產週轉率持平，現金佔資產近20%，營運穩健。

適用時機

  • 任何有併購活動的公司都必須看商譽
  • 高商譽公司用PBR選股會失真（商譽膨脹淨值）
  • 併購型公司（如鴻海、聯發科）每季必查商譽變動

二、商譽減值Goodwill Impairment：認列「買貴了」的代價

是什麼

商譽減值 = 公司承認當初併購付出過多溢價，現在要一次性認列損失。等於管理層承認「我們買貴了」。

怎麼做：商譽減值判讀三步驟

1. 確認減值觸發事件

  • 總體經濟不佳（如網路泡沫、金融海嘯）

2. 看減值對三大報表的影響

  • 損益表：減值金額列為當期費用，淨利減少（但非現金支出）

3. 評估減值背後的意義

  • 小額減值：正常調整，可能反映產業變化

實例

  • AOL收購時代華納：2000年以1,280億美元商譽收購，網路泡沫後2002年認列542億美元商譽減值，全年虧損987億美元，華爾街史上最大虧損之一。AOL股價從67美元跌至24美元。
  • 台股實務：公司原本稅後淨利100元，商譽減值80元後帳面淨利剩20元，但配息仍可能根據100元發放（因為80元非真實現金流出）。

適用時機

  • 每年財報公布季注意商譽減值公告
  • 併購後1-3年是減值高風險期（蜜月期結束，真實績效浮現）
  • 產業景氣反轉時，高商譽公司首當其衝

三、商譽減值 vs 商譽攤銷：法規差異

是什麼

兩種處理商譽的會計方法：

  • 商譽攤銷：把商譽當消耗性資產，按年限每年折減
  • 商譽減值：每年做減值測試，認為價值減損才一次性認列

怎麼做：台灣 vs 美國法規差異

項目台灣（IFRS）美國（GAAP）
攤銷不允許上市公司不允許，私營公司可選10年直線法
減值每年至少一次每年至少一次
減值方法收入法或市場法一步法或兩步法
實務影響商譽長期掛帳不動同左，但私營公司會逐年攤銷

實例

  • 台灣公司商譽常年不動，直到一次性大減值，投資人看到EPS突然暴減才驚覺
  • 如果改為攤銷制，每年固定認列費用，投資人能更早察覺問題
  • 反對攤銷的論點：併購價值非線性消失，強制攤銷也不合理

適用時機

  • 讀台股財報時，不要以為商譽不動就沒問題
  • 併購後2-3年商譽未減值不代表安全，可能只是管理層不願認列
  • 對比同產業有攤銷制度的外國公司，能更真實反映併購績效

四、無形資產Intangible Assets：商譽的鄰居

是什麼

無形資產 = 商譽 + 其他可識別無形資產（專利、商標、版權、特許權、客戶關係、電腦軟體等）

關鍵區分：

  • 可識別無形資產：能獨立分離轉賣（專利、商標）→ 會攤銷
  • 不可識別無形資產：無法分離轉賣（商譽）→ 不攤銷，做減值測試

怎麼做：無形資產檢查三招

1. 看無形資產佔總資產比例

  • 低於10%：正常

2. 拆分商譽 vs 其他無形資產

  • 其他無形資產（專利、客戶關係）有明確年限，會攤銷歸零

3. 追蹤攤銷費用對EPS的影響

  • 攤銷是非現金費用，壓低帳面EPS但不影響現金流

實例

  • 如興(4414)：無形資產81億中，商譽60億+客戶關係22億，商譽佔無形資產73%，佔總資產32%，現金只佔12%。無法變現的資產遠大於可變現資產。
  • 桂盟(5306)：商譽0.47億佔總資產2%，現金23.7億佔總資產20%。資產品質健康。

適用時機

  • 看資產負債表時，無形資產大增必查組成
  • 科技股、生技股常有大量研發資本化（也是無形資產），要辨識是研發還是商譽
  • 用PBR選股時，高無形資產公司的PBR會失真

五、商譽實戰選股避雷SOP

四步避雷流程

1. 篩選：商譽佔總資產 > 15%的公司列入觀察名單
2. 驗證：查併購後總資產週轉率是否改善
3. 確認：看財報附註，無形資產中商譽佔比
4. 決策：

  • 商譽佔總資產 < 5% + 週轉率持平或上升 → 通過

贏家的詛咒Winner's Curse

競爭併購下的贏家，通常付出過高代價，成為未來的輸家。併購溢價越高，商譽越大，未來減值風險越大。

心得

商譽這東西，名字聽起來很正面，實際上就是「買貴了的金額」。最毒的是它不影響現金流，所以公司可以一直掛在那裝沒事，直到有一天大減值，EPS突然暴減，散戶才驚覺被坑。

如興那個案例真的是經典教材：商譽佔總資產32%，現金才12%，等於資產裡三成是無法變現的東西。而且併購後總資產週轉率一路下滑，代表資產變大但賺錢能力沒跟上，這種併購就是用股東的錢去買一個虛胖的資產負債表。

反觀桂盟，商譽才佔2%，現金佔20%，這才是健康的資產結構。同樣是併購，一個買出禍害，一個買出綜效，差別就在於有沒有買貴。

以後看到無形資產大增的公司，第一步就是去附註拆開看商譽佔多少。商譽超過總資產15%的，直接打個問號；超過30%的，除非有非常強的成長邏輯，不然基本可以跳過。

一、稀釋每股盈餘（Diluted EPS）— 你看到的EPS可能是假的

是什麼

稀釋每股盈餘 = 假設所有「可能變成股票」的金融商品全部轉換後，每股真正能分到的盈餘。

公式：稀釋EPS = 稅後淨利 ÷（普通股在外流通股數 + 稀釋股）

稀釋股來源：

  • 可轉換公司債（CB）：債券持有人把債換成股票
  • 員工股票選擇權（ESO）：員工行使認股權
  • 認股權證（Warrant）：持有人行使認購權
  • 可轉換特別股：特別股轉換成普通股

怎麼做：稀釋EPS判讀SOP

1. 查損益表：在「基本每股盈餘」旁邊會列出「稀釋每股盈餘」
2. 計算稀釋比率：((稀釋EPS ÷ 基本EPS) - 1) × 100%
3. 判讀：

  • 稀釋比率 0~-3%：正常範圍，影響輕微

實例：

  • 信邦(3023)2024Q2：基本EPS 3.9，稀釋EPS 3.86 → 稀釋比率 -1.02%（輕微，可接受）
  • 特斯拉2020年：基本EPS和稀釋EPS差距突然擴大 → 原因是大量員工股權獎勵、可轉債、認股權證發行

可轉債稀釋的計算邏輯

當可轉債轉換時：
1. 分子變化：省下可轉債的稅後利息費用（因為不用付息了）
2. 分母變化：新增的普通股股數

怎麼做：
1. 查財報附註「可轉換公司債」區段
2. 找到轉換價格和尚未轉換的債券面額
3. 潛在新增股數 = 尚未轉換面額 ÷ 轉換價格
4. 稅後利息費用加回分子
5. 重新計算稀釋EPS

實例：

  • 某公司發行可轉債面額10億元，轉換價格50元
  • 潛在新增股數 = 10億 ÷ 50 = 2,000萬股
  • 年利息費用 = 10億 × 2% = 2,000萬，稅後 = 2,000萬 × (1-20%) = 1,600萬
  • 如果原本稅後淨利5億、流通在外1億股
  • 基本EPS = 5億 ÷ 1億 = 5.0
  • 稀釋EPS = (5億 + 1,600萬) ÷ (1億 + 2,000萬) = 5.16億 ÷ 1.2億 = 4.3
  • 稀釋比率 = (4.3 ÷ 5.0 - 1) = -14% 🚨 嚴重稀釋！

反稀釋效應

不是所有潛在股份都會稀釋EPS。如果可轉債轉換後，省下的稅後利息加回分子，讓每股盈餘反而上升，這稱為「反稀釋」——此時該可轉債不應納入稀釋EPS計算。

判斷標準：只有「具稀釋性」的潛在股份才納入計算。如果納入後稀釋EPS反而上升，就是反稀釋，不計入。

二、財報窗飾（Window Dressing）— 合法範圍內的財報化妝術

是什麼

財報窗飾是指在財務報表編製完成之前，運用會計準則允許的判斷空間，改善或美化財務報表的行為。不是做假帳，而是在規則內「化妝」。

五大窗飾動機

1. 逃稅

  • 故意漏開發票，逃漏營業稅或營所稅
  • 貨款收入不入帳，轉為股東往來
  • 無進貨事實但索取假進貨發票增加成本
  • 高毛利率時，將帳面存貨轉列成本出帳

2. 爭取或維持銀行貸款

  • 將次年度收入提前到今年入帳，讓本年度轉虧為盈
  • 美化流動比率、負債比率以符合銀行貸款條件

3. 維持股價或核准增資

  • 處分閒置土地或低成本高市價的資產給關係企業
  • 提前催收應收帳款、延後支付應付票據
  • 攤提損失年限延長、高估存貨價值
  • 利用關係企業買高賣低

4. 所得平坦化（Income Smoothing）

  • 營收高的年度，減少盈餘認列
  • 營收低的年度，增加盈餘認列
  • 目的：降低投資人對獲利波動的認知風險

5. 沖大澡（Big Bath）

  • 在盈餘差的年度，把所有未來可能的損失和費用一次認列
  • 以後年度就不用再認列這些費用，盈餘自然好看
  • 這是最危險也最常見的窗飾手法

窗飾四大技巧

技巧一：高估營收、低列費用（膨脹本期盈餘）

  • 提前認列營收：未來仍須提供勞務卻先認列收益
  • 虛構營收：集團公司間相互購買產品，沒有真正銷售
  • 非營業收入列為營收：投資收益、資產處分收入列為營業收入
  • 低列費用：延遲報廢過時存貨、減少呆帳提列、延長折舊年限

技巧二：策略性調整收支認列時點

  • 今年不好看 → 衝量：年底加速出貨、暫緩設備維修/廣告/研發支出
  • 今年已無望 → 留給明年：把未來支出提前在本期認列，做一次性特別支出
  • 併購美化：合併前暫時隱藏營收，合併後補上，美化併購績效

技巧三：創造業外收益

  • 出售市價高於帳面價值的資產
  • 處分投資認列利得
  • 一次性資產出售美化當季EPS

技巧四：策略性歸類

  • 將營業費用轉列為業外支出（膨脹營業利益）
  • 將業外收益列為營業收入（膨脹營收規模）
  • 一年內到期的長期負債不轉列流動負債（美化流動比率）

博達案窗飾手法拆解

2004年博達案是台股最經典的窗飾案例：

1. 設立子公司：在海外成立5家子公司，表面上是「銷貨代理商」
2. 假銷貨給子公司：虛增營收和應收帳款
3. 出售應收帳款：賣斷給銀行並回存，虛增帳面現金、提高流動比率
4. 創造衍生性金融商品：把應收帳款包成CLN賣給海外投資者，變現

財報效果：營收增加（業績亮眼）→ 利潤增加（獲利佳）→ 流動資產增加（流動性好）→ 資產增加（成長性好）

但全都是假的。

投資人辨識窗飾的七大檢查法

怎麼做：

1. 營收與現金流背離：營收成長但營業現金流停滯或下降 → 🚨
2. 應收帳款週轉天數持續增加：可能提前認列或虛增營收
3. 存貨週轉率突然下降：壓貨或需求下滑
4. 季末營收異常暴增：最後一個月佔全季超過40% → 可能衝量
5. 業外收入佔比異常：超過稅前淨利30% → 可能賣資產美化
6. 流動比率突然改善：可能是延後支付應付款或提前收回應收款
7. 關係人交易佔比過高：超過50% → 虛增營收風險極高

適用時機：每季財報公布後，用七項檢查跑一遍持股。3項以上觸發 → 🚨 高度警戒。

三、沖大澡（Big Bath）— 最狡猾的盈餘管理手法

是什麼

在公司盈餘本來就差的年度，管理層把所有未來可能發生的損失和費用一次性全部認列，讓當期虧損看起來非常慘烈（大洗澡），但未來幾年的盈餘會因為不需要再認列這些費用而顯得「大幅成長」。

為什麼這樣做

管理層邏輯：

  • 今年反正是爛年，再多爛一點無所謂
  • 把明年的費用提前到今年認列
  • 明年開始「輕裝上陣」，盈餘大幅成長
  • 市場看到盈餘成長 → 股價上漲 → 管理層拿到績效獎金

投資人受害：

  • 在「虧損最慘」時恐慌賣出（其實是被放大了）
  • 在「盈餘大幅成長」時追高買入（其實是前期洗掉費用後的反彈）

沖大澡五種常見手法

1. 大額資產減損

  • 一次性認列商譽減損、固定資產減損
  • 金額可能佔淨值10-30%
  • 減損後資產基數降低，未來折舊減少，利潤自然上升

2. 一次性重組費用

  • 宣布組織重組，大量認列遣散費、資產處分損失
  • 未來幾年不再有這些費用

3. 存貨大量報廢

  • 一次性把過時存貨全部報廢認列損失
  • 未來存貨成本降低，毛利率上升

4. 提前認列壞帳

  • 大幅提高壞帳提列比率
  • 未來壞帳費用減少

5. 研發支出一次性費用化

  • 把原本資本化的研發支出全部一次性費用化
  • 未來攤銷壓力大幅減少

怎麼做：辨識沖大澡的SOP

1. 看減損金額佔淨值比例：> 10% 可能是沖大澡
2. 看減損原因：一次性vs持續性
3. 看管理層更換時機：新CEO上任第一年最常沖大澡（把帳做爛，以後都是自己的功勞）
4. 看未來指引：如果減損後公司同時發布樂觀展望 → 🚨 典型沖大澡特徵
5. 追蹤後續盈餘：如果減損後盈餘連續2-3季大幅成長 → 驗證沖大澡判斷

實例判讀：

情境可能是沖大澡可能是真的虧損
新CEO上任首年認列大額減損✅ 高度可能❌
減損後立即發布樂觀展望✅ 典型特徵❌
減損原因明確且合理❌✅ 可能真實
產業整體下行同步減損❌✅ 可能真實
單一公司獨特大額減損✅ 高度可能❌

沖大澡後的投資策略

策略一：沖大澡後買進

  • 識別出沖大澡後，等恐慌賣壓結束
  • 在減損認列後的低點買進
  • 未來盈餘「大幅成長」帶動股價回升

策略二：辨識真假沖大澡

  • 真沖大澡：公司本業沒變差，只是會計數字被洗掉 → 未來盈餘成長可信
  • 假沖大澡：公司本業真的在惡化，減損只是如實反映 → 買進就是接刀

適用時機：看到公司認列大額減損時，不要恐慌，先判斷是沖大澡還是真的基本面惡化。

四、CFRA七大財務騙局 — 從窗飾到詐欺的完整光譜

窗飾是「合法範圍內化妝」，但有些公司會跨過界線變成「財務騙局」（造假）。

七大騙局

1. 營業收入入帳時間或品質有問題：提前認列、尚未交付就認列、銷售給關係人
2. 列記虛擬的營業收入：無實質交易的營收、借款列為營收、投資收入列為營收
3. 利用一次性利得虛灌盈餘：出售低估資產、資產重新分類創造盈餘
4. 將本期費用移轉至後期：放慢攤銷速度、未沖銷毀損資產、將營業成本資本化
5. 未依法列計或不當短列負債：不承認費用與相關負債、釋出有問題的準備金
6. 將本期營收移轉至後期：延後認列已實現收入（隱藏獲利）
7. 將未來費用以特別支出移轉至本期：沖大澡手法

容易出現騙局的公司特徵

  • 過去高成長但目前已趨緩（維持假象）
  • 瀕臨破產但苟延殘喘（製造假象）
  • 股權集中度極高且從未被查核
  • 董監事質押比率>50%且持續增加
  • 頻頻更換會計師或財務主管
  • 年年轉投資年年虧損，又年年增資
  • 關係人資金往來或背書保證過多

心得

今天三個主題串起來，就是一套完整的「財報防詐騙技能」：

稀釋EPS告訴你：你看到的EPS可能是被高估的。可轉債、員工認股權這些潛在股份一旦轉換，股本膨脹，每股盈餘就會被吃掉。信邦的例子看起來只稀釋1%，但那是因為可轉債規模小；如果公司大量發行可轉債（像科技股常做的），稀釋比率可能超過10%。最實用的判斷：用PE估值時一定要用稀釋EPS，否則你以為本益比15倍，實際可能是17倍。

財報窗飾告訴你：財報數字可以在合法範圍內被「化妝」。窗飾不是造假，但投資人如果看不出妝有多厚，就會被誤導。博達案是台股經典教材——表面上看營收、利潤、流動性、成長性四項全優，但全都是靠子公司交易堆出來的假象。最關鍵的辨識方法就是：營收成長要跟營業現金流同步。如果營收漲30%但營業現金流只漲5%，裡面一定有鬼。

沖大澡是所有窗飾手法裡面最狡猾的。本質上就是管理層在「爛年」把未來的費用全塞進去，讓未來幾年的盈餘自動好看。新CEO上任第一年最愛幹這種事——把帳做爛是前任的鍋，以後的成長全是自己的功勞。投資人要學會分辨：減損是「一次性洗掉」（未來會好轉）還是「基本面真的在惡化」（買了就是接刀）。

最深刻的體會：財報分析不是看數字，而是看數字背後的動機。管理層為什麼要這樣認列？為什麼要在這個時間點改變會計政策？為什麼要在年底衝量？動機比數字更重要。CFRA的七大騙局本質上都是同一個邏輯：把好的提前認、把壞的延後認、或者把壞的一次性全部認掉讓未來變好。看透了這個邏輯，大部分的窗飾手法都逃不過你的眼睛。
