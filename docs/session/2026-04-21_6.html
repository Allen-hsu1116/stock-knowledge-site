第 6 回合學習 - 基本面分析：金融資產三分类实戰、權益法認列、長期股權投資判讀

時間：12:22

文章：

  • 看懂財報「透過損益按公允價值衡量之金融資產」— 夏綠蒂聊投資
  • 買雞排懂長期股權投資 — 法說狗

一、金融資產三分类 — 你投資的公司，把別人的股票放哪裡？

是什麼

IFRS 9上路後，公司持有的金融資產分為三類，分類方式決定了「別人的股價漲跌，會不會影響這家公司的EPS」：

分類簡稱價值變動去哪影響EPS？適用情境
透過損益按公允價值衡量FVPL損益表（本期淨利）✅ 影響短期交易目的
透過其他綜合損益按公允價值衡量FVOCI其他綜合損益（OCI）❌ 不影響長期持有+收息
攤銷後成本衡量AC攤銷後成本（利息法）❌ 不影響持有至到期（僅債券）

怎麼做：判斷公司把投資放在哪個科目

步驟一：打開資產負債表，找「透過損益按公允價值衡量之金融資產」

步驟二：看附註揭露，確認投資標的是哪些公司、持股多少

步驟三：追蹤這些標的公司的股價走勢，預估對EPS的影響

實例：

  • 欣陸(3703) 持有高鐵(2633) 65,588張，分類為FVPL
  • 東元(1504) 同樣持有高鐵，但分類為FVOCI

FVPL vs FVOCI — 同一檔股票，兩種命運

這是投資人最容易忽略的關鍵差異：

FVPL（透過損益）：

  • 被投資公司股價漲 → 持有公司EPS增加（即使沒賣出）
  • 被投資公司股價跌 → 持有公司EPS減少（即使沒賣出）
  • 每季都要依收盤價重新評價

FVOCI（透過其他綜合損益）：

  • 被投資公司股價漲跌 → 只影響OCI，不影響EPS
  • 賣出時，差額直接調整保留盈餘，不重分類至損益
  • 只有「股利收入」會影響EPS

投資人實戰應用

怎麼做：

1. 找出持股中FVPL的標的：看財報附註，哪些投資被分類為FVPL
2. 追蹤這些標的的股價走勢：被投資公司股價變動直接影響持股公司的EPS
3. 計算預估影響：持有股數 × (本季末收盤價 - 上季末收盤價) = 預估評價利益/損失
4. 在財報公布前提前判斷EPS方向：如果FVPL標的股價大漲，持股公司EPS可能超預期

適用時機：每季財報公布前，特別是集團股、控股公司、壽險股

二、權益法認列 — 轉投資的「看得到吃不到」問題

是什麼

當公司持有另一家公司20%~50%股權，且具有重大影響力時，必須採用權益法認列投資損益。

權益法核心：按持股比例認列被投資公司的淨利，不管被投資公司有沒有發放股利。

三種持股比例的認列方式

持股 < 20%：

  • 只認列股利收入（現金股利進損益表）
  • 公允價值變動依FVPL/FVOCI分類處理
  • 不認列被投資公司的盈虧

持股 20%~50%（權益法）：

  • 按持股比例認列被投資公司的淨利為「投資收益」
  • 被投資公司賺錢 → 母公司投資收益增加、長期投資增加
  • 被投資公司虧錢 → 母公司投資損失增加、長期投資減少
  • 被投資公司發放股利 → 母公司視為成本回收，長期投資減少

持股 ≥ 50%（合併報表）：

  • 完全控制，必須編製合併報表
  • 內部交易須消除
  • 歸屬母公司損益 + 非控制權益 = 合併總損益

權益法的「看得到吃不到」問題

問題核心：母公司認列了被投資公司的盈虧，但可能一毛錢現金都沒收到。

實例：

  • A公司持有B公司30%股權
  • B公司今年賺100億 → A公司認列投資收益30億
  • B公司不發股利 → A公司一毛現金都沒收到
  • A公司損益表顯示投資收益30億，但營業現金流沒有增加

怎麼做（投資人判讀）：

1. 看營業現金流 vs 淨利的差距：如果淨利遠大於營業現金流，可能就是權益法認列的投資收益在「灌水」
2. 看投資收益的現金股利佔比：投資收益中的現金股利才是真正收到的錢
3. 看被投資公司的盈餘品質：被投資公司本身如果淨利遠大於營業現金流，母公司的投資收益品質也差

適用時機：分析集團股、控股公司時，一定要看合併報表的現金流量表，不要只看母公司損益表

三、長期股權投資判讀 — 轉投資是金雞母還是錢坑？

長期投資佔總資產比例 — 第一道濾網

怎麼做：

1. 打開資產負債表
2. 找到「長期投資」或「採權益法之長期股權投資」
3. 計算長期投資 ÷ 總資產 × 100%

判讀標準：

  • 長期投資佔總資產 < 15%：影響有限，看母公司財報即可
  • 長期投資佔總資產 15%~30%：有一定影響，需注意投資收益品質
  • 長期投資佔總資產 > 30%：🚨 轉投資是公司核心，必須看合併報表

三種長期投資類型判讀

類型一：垂直整合型（金雞母）

  • 投資上下游公司，確保原料供應或通路
  • 合併報表營收成長率高於母公司
  • 投資收益穩定且來自本業相關
  • 判斷：✅ 轉投資是策略的一環

類型二：多角化型（看情況）

  • 投資不相關的產業，分散風險或追求新成長
  • 需要看個別轉投資的獲利品質
  • 如果轉投資持續虧損 → 🚨 管理層可能在做多餘的事
  • 判斷：⚠️ 需要逐一檢視

類型三：十字交叉型（錢坑）

  • 轉投資虧損連連、業外損益波動大
  • 長期投資佔比高 + 業外虧損 > 本業利益
  • 沒有合併報表可看 → 🚨 直接避開

權益法投資收益的四大陷阱

陷阱一：虛盈實虧

  • 認列了投資收益，但被投資公司沒發股利
  • 母公司EPS增加，但現金流沒增加
  • 驗證方式：投資收益 ÷ 來自被投資公司的現金股利，比率越大越虛

陷阱二：減損地雷

  • 被投資公司虧損嚴重，母公司必須認列投資損失
  • 如果持股比例高，損失可能吃掉本業利潤
  • 驗證方式：看長期投資減損損失的歷史紀錄

陷阱三：交叉持股

  • A公司持有B公司，B公司又持有A公司
  • 互相認列盈虧，形成虛胖的EPS
  • 驗證方式：看是否有環狀持股結構

陷阱四：轉投資黑箱

  • 持股低於20%的轉投資，不需要編合併報表
  • 投資人只看到業外虧損，不知道虧在哪
  • 驗證方式：看附註揭露的轉投資明細，查個別公司的營運狀況

四、壽險業的FVOCI地雷 — 淨值大幅波動的真相

是什麼

壽險公司持有大量金融資產，大部分分類在FVOCI。當市場大幅下跌時，這些資產的未實現損失不會進EPS，但會進OCI，直接壓低淨值。

壽險業的特殊性

  • 壽險公司持有大量債券和股票，分類在FVOCI
  • 債券殖利率上升 → 債券價格下跌 → OCI大額未實現損失
  • 未實現損失 + 未分配盈餘為負 → 金管會規定不能發股利
  • 2022年多家壽險公司淨值大幅下降甚至轉負，就是因為FVOCI的未實現損失

投資人判讀

怎麼做：

1. 看壽險公司的「其他權益—備供出售金融資產未實現損益」
2. 看金額是否接近或超過1個股本
3. 如果OCI累計為大額負數 → 淨值被壓低，未來回升時可能有大量回沖利益

適用時機：投資金融股時，一定要看權益變動表的OCI累計金額

心得

今天學的主題串起來，就是在回答一個問題：你投資的公司，把錢放在哪裡？又把別人的股票怎麼分類？

金融資產三分类看起來是會計問題，但對投資人來說是實實在在的EPS預測工具。欣陸和東元都持有高鐵，但因為分類不同，高鐵股價上漲對欣陸的EPS有直接影響，對東元卻完全沒有。這意味著如果你持有欣陸，你可以追蹤高鐵的股價走勢，在財報公布前就預估欣陸的EPS變動方向——這就是資訊優勢。

權益法認列的「看得到吃不到」問題也是投資人常踩的坑。母公司認列了30億投資收益，損益表看起來很漂亮，但營業現金流沒有增加半毛錢。這種「帳面利潤」在景氣反轉時會瞬間變成投資損失，而且因為沒有現金流支撐，公司可能配不出股利。以後看集團股的財報，一定要同時看「投資收益」和「來自被投資公司的現金股利」，兩者的差距就是「虛盈」的部分。

長期投資佔總資產超過30%的公司，必須看合併報表，這個判斷標準太實用了。神基和精元的案例清楚顯示：只看母公司營收年增率跟股價完全不相關，但看合併營收年增率就高度相關。轉投資多的公司，母公司財報根本不能反映真實獲利能力。

壽險業的FVOCI地雷也值得警惕。2022年壽險股淨值大幅縮水，就是因為債券價格下跌導致未實現損失灌進OCI。雖然不影響EPS，但壓低淨值會限制股利發放，也讓投資人對公司體質產生疑慮。反過來說，當市場回升時，這些未實現損失會回沖，淨值和EPS都可能大幅改善——這是逆向投資的機會。

最核心的判斷框架：FVPL的投資看EPS波動，FVOCI的投資看淨值波動，權益法的投資看現金流品質，合併報表看真實獲利。 四個角度缺一不可。

時間：13:22

文章：

  • 淺析IFRS 16對企業評價之影響 - TEJ台灣經濟新報

一、IFRS 16 是什麼 — 為什麼投資人必須懂

是什麼

IFRS 16 是 2019 年生效的新租賃準則，取代舊的 IAS 17。核心改變：承租人幾乎所有租賃都要「上表」，不能再把營業租賃藏在表外。

舊制 IAS 17 把租賃分成兩類：

  • 融資租賃：資產負債表認列資產和負債
  • 營業租賃：只認列租金費用，資產負債表完全看不到

新制 IFRS 16：不分融資或營業，承租人一律認列「使用權資產」和「租賃負債」。只有兩種豁免：短期租賃（12個月以下）和低價值資產（全新時價值 USD 5,000 以下）。

為什麼重要

過去很多公司（特別是航空、零售、物流）大量使用營業租賃，把龐大的租賃承諾藏在表外。兩家同產業的公司，一家自購資產、一家租用資產，舊制下財報完全無法比較。IFRS 16 就是讓這些「隱形負債」現形。

二、使用權資產 Right-of-Use Asset — 你租的東西變成你的資產

是什麼

使用權資產 = 承租人取得「使用租賃資產的權利」，按公報規定入帳為資產。

怎麼計算

使用權資產 = 租賃負債原始金額 + 租賃開始日前已付的租金 - 租賃誘因 + 原始直接成本 + 除役復原成本現值

實例

阿嘉公司租店面，每年年初付租金 50 萬，租期 5 年，增額借款利率 3%，原始直接成本 6 萬，租賃誘因 1 萬，除役成本 10 萬（5 年後復原店面）：

  • 租賃負債 = 500,000 × 3.717（利率 3%，4 期期初年金現值）= 1,858,500
  • 使用權資產 = 1,858,500 + 500,000 + 60,000 - 10,000 + 86,300 = 2,494,800

後續衡量

  • 使用權資產按成本模式：成本 - 累計折舊 - 累計減損（跟固定資產一樣）
  • 折舊採直線法，按租賃期間分攤

三、租賃負債 Lease Liabilities — 你承諾付的租金變成負債

是什麼

租賃負債 = 未來所有租金給付的現值，按適當利率折現。

怎麼計算

1. 確定每期租金給付
2. 選折現率：優先用出租人隱含利率，無法得知時用承租人增額借款利率
3. 計算租金給付現值 = 租賃負債入帳金額

後續衡量 — 利息法（前高後低）

  • 每期支付租金 = 償還本金 + 利息費用
  • 利息費用 = 期初租賃負債餘額 × 折現率
  • 償還本金 = 支付租金 - 利息費用

阿嘉公司第二年：支付租金 500,000，利息費用 = 1,858,500 × 3% = 55,755，償還本金 = 500,000 - 55,755 = 444,245

關鍵特性：利息費用前高後低。初期租賃負債餘額大，利息費用高；後期餘額小，利息費用低。這導致租賃總費用（折舊 + 利息）在租賃期間呈現前高後低。

四、IFRS 16 對三大報表的影響

資產負債表

  • 資產增加：認列使用權資產
  • 負債增加：認列租賃負債
  • 影響：負債比率上升、資產週轉率下降、權益不變（初始）

損益表

  • 舊制（營業租賃）：租金費用全部列為營業費用，直線法平均分攤
  • 新制（IFRS 16）：折舊費用（營業成本）+ 利息費用（財務成本）
  • EBITDA 上升：因為租金費用不再全數列為營業費用，折舊和利息都不算進 EBITDA
  • 費用前高後低：折舊是直線法平均，但利息前高後低，合計費用前期高於後期

現金流量表

  • 舊制：租金全數列為營業現金流出（CFO）
  • 新制：本金償還列為融資現金流出（CFF），利息支出列為 CFO 或 CFF
  • 影響：CFO 上升（租金不再全部從營業現金流扣），CFF 下降（新增融資還本流出）

五、投資人實戰：IFRS 16 後的財務比率修正

5.1 EBITDA 修正 — 不要被膨脹的 EBITDA 騙了

是什麼：IFRS 16 讓 EBITDA 虛增，因為租金費用不再從 EBITDA 扣除。

怎麼做：
1. 找到財報附註的租賃相關費用（折舊 + 利息）
2. 計算 Adjusted EBITDA = EBITDA - 租賃折舊 - 租賃利息 + 原始租金費用
3. 或更簡單：EBITDA - 租賃相關的非現金加回項目

實例：

  • 某零售公司 EBITDA = 50 億，其中使用權資產折舊 = 15 億，租賃利息 = 5 億
  • 舊制下租金費用約 = 20 億（折舊 + 利息的合計近似值）
  • 調整後 EBITDA = 50 - 15 - 5 + 20 = 50 億（這就是真實的獲利能力）
  • 如果直接用 50 億 EBITDA 算 EV/EBITDA，會低估公司估值倍數

5.2 負債比率修正 — 舊制下隱藏的負債

是什麼：IFRS 16 讓負債比率大幅上升，特別是大量使用營業租賃的行業。

怎麼做：
1. 比較 IFRS 16 前後的負債/權益比
2. 舊制分析時，手動加回營業租賃承諾的現值
3. 租賃承諾現值通常在財報附註的「承諾及或有事項」可找到

實例：

  • 某航空公司舊制負債/權益比 = 2.0
  • IFRS 16 後認列租賃負債 500 億 → 負債/權益比 = 4.5
  • 如果只看舊制數字，會以為槓桿風險較低

5.3 ROA / ROE 修正

是什麼：使用權資產讓總資產膨脹，ROA 下降；租賃負債讓權益在初期不變但後期可能受費用前高後低影響。

怎麼做：
1. ROA = 淨利 /（總資產 + 使用權資產），IFRS 16 後分母變大，ROA 自動下降
2. 不要直接比較 IFRS 16 前後的 ROA，要看調整後的一致數字
3. ROE 短期受費用前高後低影響，初期 EPS 被壓低

5.4 CFO 修正

是什麼：IFRS 16 讓營業現金流虛增，因為租金的本金償還部分移到融資活動。

怎麼做：
1. CFO 調整 = CFO - 利息費用中的租賃利息（如果列在 CFO）+ 全部租金費用
2. 更簡單的做法：看「調整後營業現金流」= CFO + 租賃本金償還 - 租賃利息
3. 或直接看「自由現金流」= CFO - 資本支出（含使用權資產增加）

六、IFRS 16 對企業評價的影響

市場法（EV/EBITDA）

  • 分子 EV 增加（因為要扣除的淨負債多了租賃負債）
  • 分母 EBITDA 增加（租金不再從 EBITDA 扣除）
  • EV/EBITDA 比率變大或變小，取決於公司的具體情況
  • 實務建議：用 Adjusted EV/EBITDA，將 EV 扣除租賃負債、EBITDA 也扣除租賃相關影響

收益法（DCF）

  • FCFF 增加（EBIT 增加 + 折舊增加，兩者加回使 FCFF 增加）
  • WACC 下降（租賃負債讓負債比率上升，負債權重增加拉低 WACC）
  • 評估出的 EV 會增加
  • 但理論上扣除新增的淨負債後，股東權益價值不變
  • 陷阱：淨負債只算剩餘租賃期間的現值，沒算租賃到期後續約的現金流出 → 可能高估

七、IFRS 16 的投資人實戰檢查清單

看財報時必做的 5 件事：

1. 找使用權資產和租賃負債：在資產負債表的「非流動資產」和「非流動負債」找
2. 看附註的租賃揭露：折舊金額、利息費用、租賃期間、未來租金給付
3. 調整 EBITDA：不要直接用報表上的 EBITDA 算估值倍數
4. 調整負債比率：IFRS 16 前的數字已經不能直接用
5. 注意費用前高後低：租賃初期的 EPS 會被壓低，不代表獲利能力變差

高影響行業：

  • 航空業：飛機多為租賃，影響最大
  • 零售業：店面大量租賃
  • 物流業：倉庫和車輛租賃
  • 餐飲業：展店靠租賃

心得

IFRS 16 這東西，說白了就是：過去公司可以透過「租」代替「買」來美化財報，現在不行了。

這對投資人的影響是全方位的。你以為公司的 EBITDA 很高？可能是因為租金費用不再從 EBITDA 扣除了。你以為負債比率安全？可能是租賃負債還沒上表。你以為營業現金流很健康？可能是租金的本金償還移到了融資活動。

最實用的收穫是「費用前高後低」這個特性。利息費用在租賃初期最高，意味著剛開始大量租賃的公司，前幾年的 EPS 會被壓低。這不代表公司變差了，只是會計處理造成的。反過來說，如果一家公司剛好處於租賃後期，利息費用很低，EPS 看起來很好，但也不代表獲利能力真的提升了。這就是為什麼同一間公司 IFRS 16 前後的 EPS 不能直接比較。

另外，TEJ 那篇提到的評價陷阱也很值得警惕：DCF 估出來的 EV 變大了，理論上扣掉新增的租賃負債後股東權益價值不變，但實務上因為只算了剩餘租賃期間的負債現值，沒考慮租約到期後續約的現金流出，所以評價結果可能被高估。這告訴我：用 DCF 估值時，對於大量租賃的公司，一定要額外處理租約到期後的現金流量。

以後看財報，第一步就是找「使用權資產」和「租賃負債」這兩個項目。有這兩個數字，才知道公司的真實負債水準和資產結構。沒看到這兩個數字之前，所有的財務比率都是半成品。

一、現金流量表 — 三大報表裡最難造假的那張

為什麼現金流量表最重要

三大報表各有分工：

  • 損益表：看公司帳面上賺多少（應計制，不等於真金白銀）
  • 資產負債表：看公司某個時間點的家當和負債（靜態快照）
  • 現金流量表：看公司現金真正的流入流出（實收實付制，最難造假）

市場先生說得好：「投資菜鳥看損益表，老鳥都看現金流量表。」原因很簡單——損益表可以透過應收帳款、遞延收入、折舊方法等手段「化妝」，但現金流量表記錄的是真金白銀的進出，要造假難度極高。

現金流量表 vs 損益表的關鍵差異

項目損益表現金流量表
基礎應計制（權責發生制）現金制（實收實付制）
收入認列合約簽完就認列真正收到現金才認列
折舊每月認列費用非現金支出，加回
應收帳款認列為收入扣掉（錢還沒收到）
應付帳款認列為成本加回（錢還沒付）

實例：某公司本月簽了1000萬合約，客戶3個月後才付款

  • 損益表：立即認列1000萬收入
  • 現金流量表：0元（現金還沒進來）

這就是為什麼一家公司損益表可以很漂亮，但現金流量表可能很難看——因為「賺到的錢」不等於「收到的錢」。

二、間接法回推邏輯 — 從淨利到真實現金的調整

間接法是什麼

台灣大部分企業的現金流量表都採用「間接法」編製。間接法的邏輯是：從稅後淨利出發，把「沒有實際現金流進出的項目」加減回去，最後得到營業活動的淨現金流。

回推步驟

1. 稅後淨利（起點）
2. 加回折舊與攤銷：損益表上認列了費用，但現金早就付過了，不是本期現金支出
3. 加回處分資產損失（或扣減處分資產利益）：帳面損失不是現金流出，帳面利益不是現金流入
4. 調整營運資金變動：

  • 應收帳款增加 → 扣減（錢還沒收到）

5. 最後得到營業活動現金流量

怎麼做：看間接法的重點項目

第一步：看折舊攤銷加回金額

  • 折舊金額越大 → 公司固定資產越多（重資本行業）
  • 如果折舊金額遠大於資本支出 → 公司可能不再投資，吃老本

第二步：看應收帳款變動

  • 應收帳款大幅增加（扣減項目變大）→ 客戶沒付錢，盈餘品質差
  • 應收帳款減少 → 收款效率提升，盈餘品質好

第三步：看存貨變動

  • 存貨大幅增加 → 賣不出去或備貨過多，現金被卡住
  • 存貨減少 → 銷售暢旺或去化順利

第四步：看應付帳款變動

  • 應付帳款增加 → 付款條件改善或拖延付款，暫時增加現金
  • 應付帳款減少 → 付款速度加快或供應商要求現金交易

實例判讀：

項目健康公司危險公司
稅後淨利100億100億
折舊攤銷加回30億30億
應收帳款增加5億（微量）40億（暴增）
存貨增加3億（微量）25億（暴增）
應付帳款減少2億15億（供應商不信任）
營業現金流120億 ✅50億 ⚠️

兩家公司損益表都是100億淨利，但現金流天差地遠。危險公司看起來賺100億，實際現金只進來50億——剩下的50儻全卡在應收帳款和存貨裡。

三、三大現金流判讀 — 正負號的排列組合

現金流三大分類

營業現金流（CFO）：本業經營產生的現金流入/流出

  • 正值：本業有現金流入，公司靠本業賺錢
  • 負值：本業在燒現金，可能是虧損或應收帳款收不回來

投資現金流（CFI）：投資相關的現金流入/流出

  • 正值：變賣資產（通常是一次性的）
  • 負值：資本支出、擴廠、購買設備（常態性的，代表公司持續投資）

融資現金流（CFF）：籌資相關的現金流入/流出

  • 正值：借錢、增資、發行公司債（向外部要錢）
  • 負值：還債、發股利、買回庫藏股（把錢還給外部）

八大型態判斷企業體質

三大現金流的正負號組合，可以判斷企業處於什麼階段、體質如何：

型態CFOCFICFF代表意義判斷
1+-+本業賺錢、持續投資、向外部籌資🟡 成長擴張期，需要外部資金支持擴張
2+--本業賺錢、持續投資、償還債務發股利🟢 **最健康**，成熟穩定企業的典型型態
3++-本業賺錢、變賣資產、償還債務發股利🟡 縮編期，賣資產還債，本業尚穩
4+++本業賺錢、變賣資產、向外部籌資🔴 異常，本業賺錢卻賣資產又借錢？
5--+本業燒錢、持續投資、向外部籌資🟡 **初創期**，靠外部資金支撐成長（特斯拉早期）
6-++本業燒錢、變賣資產、向外部籌資🔴 **危險**，三面燒錢靠外部輸血
7-+-本業燒錢、變賣資產、償還債務🔴 **衰退期**，賣資產還債，本業持續惡化
8---本業燒錢、持續投資、償還債務🔴 **極度危險**，現金全方位淨流出

各型態深入判讀

型態2（+,-,-）：最健康的現金流模式

  • 營業現金流為正：本業穩定賺錢
  • 投資現金流為負：持續投資未來（買新設備、擴廠）
  • 融資現金流為負：還債、發股利、買回庫藏股
  • 代表：台積電、中華電等穩定獲利公司

型態5（-,-,+）：初創期典型

  • 營業現金流為負：本業還沒賺錢
  • 投資現金流為負：大量資本支出
  • 融資現金流為正：靠VC、IPO、銀行借款支撐
  • 代表：早期特斯拉、新創公司
  • 關鍵：看融資現金流能否持續撐到轉正

型態6（-,+,+）：最危險的組合之一

  • 本業燒錢、還在變賣資產、還要借錢
  • 代表：公司本業持續虧損，靠賣資產和借錢續命
  • 這是最典型的「殭屍企業」型態

型態1（+,-,+）：成長擴張期

  • 本業已經賺錢，但投資需求大於本業現金流
  • 需要外部融資支持擴張
  • 代表：高速成長期的公司
  • 關鍵：看融資是股權融資（可接受）還是債務融資（需注意）

怎麼做：現金流型態判讀SOP

1. 拿到現金流量表，先看三大現金流的正負號
2. 對照八大型態表，判斷公司處於什麼階段
3. 連續看3-5年的現金流型態變化，判斷趨勢
4. 型態2是目標，型態6和8要避開

實例：特斯拉的現金流演進

  • 2015-2017年：型態5（-,-,+），本業燒錢、大量投資、靠融資
  • 2018年：型態1（+,-,+），本業轉正但投資仍大、繼續融資
  • 2019-2020年：型態2（+,-,-），本業穩定、持續投資、開始還債
  • 2021年後：型態2持續，投資現金流流出減少（工廠蓋完），自由現金流大幅成長

實例：面板雙虎的現金流

  • 景氣好時：型態2（+,-,-），本業賺錢、投資、還債
  • 景氣差時：型態7（-,+,-），本業虧損、被迫賣資產、還要還債
  • 現金流型態隨景氣循環劇烈波動，這就是循環股的特徵

四、三大現金流比率 — 現金流量表的量化指標

1. 現金流量允當比率（Cash Adequacy Ratio）

公式：近5年平均營業現金流 ÷ 近5年平均（資本支出 + 存貨增加 + 現金股利）

判讀：

  • 比率 > 1：公司本業賺的現金足以支應所有投資和配息需求，自給自足 ✅
  • 比率 < 1：公司本業賺的不夠花，需要外部融資，依賴度 ⚠️
  • 比率長期 < 0.5：公司嚴重依賴外部資金 🔴

實例：

  • 台積電(2330)：近5年現金流量允當比率 > 1.5 → 本業賺的錢遠超過投資需求
  • 某面板廠：近5年現金流量允當比率 < 0.5 → 本業現金流不夠，靠融資撐

2. 現金再投資比率（Cash Reinvestment Ratio）

公式：（營業現金流 - 現金股利）÷（固定資產 + 長期投資 + 其他資產 + 營運資金）

判讀：

  • 7%~11%：正常範圍，投資與配息平衡
  • > 15%：公司大量再投資，成長期特徵
  • < 5%：投資不足，可能進入衰退期或缺乏成長機會

3. 債務保障比率（Debt Coverage Ratio, DCR）

公式：營業現金流 ÷（短期負債 + 長期負債）

判讀：

  • DCR > 0.3：營業現金流足以覆蓋30%以上的總負債，償債能力強
  • DCR 0.15~0.3：正常範圍
  • DCR < 0.1：營業現金流不足以覆蓋10%的總負債，償債壓力大

適用時機：搭配流動比率、速動比率一起看，評估公司償債能力

五、現金流品質檢查 — 營業現金流 vs 淨利

為什麼要對比營業現金流和淨利

損益表上的淨利是「應計制」，現金流量表上的營業現金流是「現金制」。兩者的差距越大，代表盈餘品質越差。

營業現金流對淨利比率

公式：營業現金流 ÷ 稅後淨利

判讀：

  • 比率 > 1.2：盈餘品質優良，現金收入大於帳面利潤 ✅
  • 比率 0.8~1.2：正常範圍，帳面利潤和現金收入大致相符
  • 比率 0.5~0.8：盈餘品質存疑，可能有大量應收帳款未收回 ⚠️
  • 比率 < 0.5：嚴重警訊，帳面賺的錢一半以上沒收到現金 🔴
  • 比率 < 0：營業現金流為負，本業在燒現金 🔴🔴🔴

五大紅旗：營業現金流與淨利背離

1. 淨利成長但營業現金流下降：可能靠應收帳款衝營收，錢沒進來
2. 營業現金流主要來自折舊加回：折舊不是現金流入，只是帳面調整
3. 應收帳款增加幅度遠超過營收成長：放寬信用條件衝營收
4. 存貨暴增：賣不出去或過度備貨，現金被卡住
5. 遞延收入大幅減少：提前認列未來收入，透支明天的現金

實戰對照：台積電 vs 面板廠

台積電(2330) 2023年：

  • 稅後淨利：約3,800億
  • 營業現金流：約5,000億
  • 比率：1.32 ✅ 營業現金流 > 淨利，折舊加回後現金非常充沛

某面板廠 2023年：

  • 稅後淨利：約200億
  • 營業現金流：約50億
  • 比率：0.25 🔴 帳面賺200億，現金只進來50億，大量卡在應收帳款和存貨

六、自由現金流（FCF）的進階判讀

FCF三種計算方式（複習）

1. 最簡單：FCF = 營業現金流 + 投資現金流
2. 較精確：FCF = 營業現金流 - 資本支出（只扣維持性投資）
3. 最完整：FCF = 營業現金流 - 維持性資本支出 - 變動性資本支出

FCF的季節性與波動

  • 資本支出通常集中在Q3-Q4（建廠、買設備），導致FCF在這兩季偏低
  • 營收通常Q4最高（年底效應），導致FCF在Q1偏高
  • 判讀FCF時要看全年或至少4季移動平均，不要只看單季

FCF為負的兩種情況

情況一：擴張期投資（可以接受）

  • 營業現金流為正且持續成長
  • 投資現金流為負但金額大是因為資本支出
  • FCF暫時為負，但營業現金流持續成長會很快轉正
  • 代表：台積電2022-2023年大量投資海外廠

情況二：燒錢期（危險）

  • 營業現金流為負
  • 投資現金流也為負
  • 靠融資現金流續命
  • 代表：早期新創公司，如果營業現金流一直無法轉正，就是殭屍企業

心得

現金流量表是三大報表裡我之前看得最少的，但今天深入後才意識到：這張表才是檢驗盈餘品質的終極武器。

損益表會騙人，但現金流量表很難騙人。同樣100億淨利，一家公司營業現金流120億，另一家只有50億，盈餘品質天差地遠。前者是真的賺到錢進了口袋，後者有一半還卡在應收帳款和存貨裡。

八大型態這個框架太實用了。以後拿到一家公司的現金流量表，第一步就是看三大現金流的正負號組合，立刻判斷公司在什麼階段。型態2是健康成熟企業的特徵，型態5是初創期，型態6和8是地雷——這個判斷邏輯比任何單一指標都直觀。

最深刻的體會是間接法的回推邏輯。以前看現金流量表，只看營業現金流的數字，從來沒有往回推去看應收帳款和存貨的變動。但這兩個項目才是盈餘品質的關鍵——應收帳款暴增代表客戶沒付錢，存貨暴增代表貨賣不出去，這兩個問題在損益表上都看不出來，只有現金流量表的間接法回推才能看到。

還有營業現金流對淨利比率，這個簡單到不行的指標，卻能一眼看出一家公司是不是在「紙上賺錢」。比率>1.2代表現金收入大於帳面利潤（好），比率<0.5代表帳面利潤的一半以上沒收到現金（壞）。以後每季財報出來，這個比率一定要算。

現金流量表的三個比率（允當比率、再投資比率、債務保障比率）也補上了之前學財報分析時缺的一塊。允當比率>1代表公司自給自足，不用靠外部融資；再投資比率7-11%是健康範圍；債務保障比率>0.3代表償債能力強。這三個數字加上八大型態，現金流分析算是完整了。
