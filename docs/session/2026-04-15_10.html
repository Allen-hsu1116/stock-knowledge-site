一、Put/Call Ratio（P/C Ratio）完整實戰

P/C Ratio 是什麼

是什麼：將選擇權賣權（Put）的未平倉量除以買權（Call）的未平倉量，再乘以 100。台灣市場習慣以 100 為基準點，P/C Ratio > 100 代表 Put 未平倉量大於 Call，P/C Ratio < 100 代表 Call 未平倉量較多。

公式：

P/C Ratio = Put 未平倉量 ÷ Call 未平倉量 × 100

兩種版本：

  • 成交量版：Put 成交量 ÷ Call 成交量 × 100 → 反映當日交易熱度，受當沖影響大，波動大
  • 未平倉量版（主流）：Put 未平倉量 ÷ Call 未平倉量 × 100 → 反映留倉意願，趨勢性強，訊號穩定

為什麼要從賣方角度解讀：

  • 選擇權買方只需付權利金（如 2,000~5,000 元），賣方需繳保證金（如 30,000~50,000 元）
  • 賣方留倉代表更高的資金承諾與風險承擔
  • Put 未平倉量大增 = 大量資金願意「押注大盤不會跌破某個價位」
  • 賣方部位比買方投機更具參考價值

怎麼用：
1. 每天盤後查看期交所公布的 P/C Ratio（約 15:00 更新）
2. 觀察 P/C Ratio 的趨勢變化，而非單日絕對值
3. 搭配支撐壓力表和法人籌碼一起判斷

實例（2026/04/14 最新數據）：

  • P/C Ratio = 165.4%（+24.51%），Put 未平倉 141,290 口，Call 未平倉 85,423 口
  • 解讀：Put 未平倉遠大於 Call，市場賣方偏向「看不跌」，支撐強勁
  • 但 165.4% 屬於極端高值，需警惕反向風險

P/C Ratio 多空判讀四區間

怎麼判斷：

P/C Ratio 區間市場意涵賣方解讀實務建議
> 120極度偏多Sell Put 遠大於 Sell Call，支撐強勁搭配法人籌碼驗證，警惕散戶過度樂觀
100~120溫和偏多Put 略多於 Call，市場有支撐順勢偏多操作，注意 P/C Ratio 是否下滑
80~100溫和偏空Call 略多於 Put，上方有壓力短線保守，留意洗盤或真正轉空
< 80極度偏空Sell Call 遠大於 Sell Put，壓力沉重反向思考：極端值可能是反轉訊號

台股長期均值：2020~2025 年統計，P/C Ratio 長期均值約在 85~95 之間，不宜死守 100 為唯一分界。

P/C Ratio 作為反向指標

是什麼：當 P/C Ratio 出現極端值時，往往是市場即將反轉的警示訊號。

怎麼判斷：
1. P/C Ratio 極高（> 130）：散戶大量 Sell Put，市場過度樂觀 → 若指數已在高檔，需警戒回檔
2. P/C Ratio 極低（< 70）：散戶大量 Sell Call，市場過度悲觀 → 可能是底部反轉訊號
3. 歷史回測：P/C Ratio 突破 130 後 5 個交易日內，指數回檔（跌幅 > 1%）機率約 65%
4. P/C Ratio 低於 70 後 5 個交易日內，指數反彈（漲幅 > 1%）機率約 60%

歷史案例：

  • 2020/3 COVID 崩盤：P/C Ratio 跌至約 65，極度恐慌 → 隨後 V 型反轉，半年漲 40%
  • 2021/7 航運高點：P/C Ratio 升至 135+，散戶大量 Sell Put → 一個月修正 1,500 點
  • 2022/10 全球升息底部：P/C Ratio 再度跌至 70 以下 → 數月反彈行情

適用時機：極端值區間參考價值最高，正常區間（80~120）指引性有限

二、選擇權支撐壓力表實戰判讀

支撐壓力表是什麼

是什麼：透過選擇權各履約價的未平倉量分布，判斷市場認為指數「不容易漲破哪裡、跌破哪裡」。站在賣方角度思考——賣方敢留倉代表有信心指數不會到達該價位。

怎麼看：
1. 支撐：Put 未平倉量最大的履約價 = 多方防守區域（賣方認為不會跌破）
2. 壓力：Call 未平倉量最大的履約價 = 空方防守區域（賣方認為不會漲破）
3. 兩個關鍵履約價之間的距離：

  • 距離窄 → 盤整格局

OP 凱文進階技巧：看「變化量最大」的履約價

是什麼：不看未平倉量最大的履約價，而是看未平倉量「變化最大」的履約價，因為它更貼近目前市況。

怎麼做：
1. 比對前一日和當日的未平倉數據
2. 找出 OI 增加最多的履約價 → 這是新進場的賣方部位，支撐/壓力意義更強
3. OI 減少最多的履約價 → 賣方已撤守，該價位的支撐/壓力弱化

實例：

  • 情境：大盤 17,300 點，Call 最大 OI 在 17,500（40,000 口），Put 最大 OI 在 17,000（35,000 口）
  • 但 Call OI 變化最大的是 17,600（新增 8,000 口），Put OI 變化最大的是 16,900（新增 6,000 口）
  • 判斷：新的賣方防守區在上移到 17,600 和下移到 16,900，比靜態最大 OI 更貼近當下市況

支撐壓力不是絕對

關鍵認知：

  • 大部分時候指數在支撐壓力區間內遊走
  • 但碰到重大消息、波動放大、法人做買方時，可能直接衝出區間
  • 漲破壓力或跌破支撐時 → 必須做相對應處理（停損、部位調整、順勢操作）
  • 不要有「碰到支撐就不會跌破」的迷思

三、選擇權未平倉量與成交量的價量關係

未平倉量是趨勢的燃料

是什麼：選擇權未平倉量增加代表多空雙方歧見加大，一方勝出時會形成趨勢加速。未平倉量減少代表輸家認賠、贏家了結，原有趨勢可能結束。

怎麼判斷：

指數未平倉成交量買方賣方判斷
上漲增加增加獲利加碼虧損攤平強勢軋空
上漲增加減少獲利加碼虧損攤平多頭持續
上漲減少增加獲利了結認賠出場多頭反轉
上漲減少減少獲利了結認賠出場多頭趨緩
下跌增加增加虧損攤平獲利加碼強勢走空
下跌增加減少虧損攤平獲利加碼空頭持續
下跌減少增加認賠出場獲利了結空頭反轉
下跌減少減少認賠出場獲利了結空頭趨緩

核心原則：

  • 趨勢明確時，成交量與未平倉量應同步增加
  • 成交量與未平倉量同步減少 → 對當前行情不利，可能反轉
  • 上升趨勢中 OI 增加 = 多頭買進 + 空頭賣出 → 空頭被軋時被迫回補推高價格
  • 下降趨勢中 OI 增加 = 空頭拋空 + 多頭承接 → 多頭停損退出加速下跌

實例：

  • 情境：大盤連續上漲 3 天，未平倉量增加 5,000 口，成交量也同步放大
  • 判斷：趨勢有燃料支撐，強勢軋空行情
  • 反之：大盤上漲但未平倉量減少 3,000 口 → 多頭反轉警訊，贏家開始離場

OI 增減的四種情境

怎麼做：
1. 新買方 + 新賣方 → OI 增加（新契約，趨勢燃料增加）
2. 新買方 + 舊買方退場 → OI 不變（換手，燃料不變）
3. 舊賣方退場 + 新賣方進場 → OI 不變（換手，燃料不變）
4. 舊買方退場 + 舊賣方退場 → OI 減少（平倉，燃料減少）

適用時機：判斷趨勢是否能持續、是否有反轉跡象

四、P/C Ratio 搭配法人籌碼判讀法

誰在 Sell Put / Sell Call 決定訊號可信度

是什麼：P/C Ratio 只告訴你 Put 和 Call 的總量比例，但「誰在買、誰在賣」才是決定訊號可信度的關鍵。

怎麼做：
1. 查看三大法人（外資、投信、自營商）的選擇權留倉數據
2. 外資是台股選擇權最大參與者，其方向對 P/C Ratio 影響最大
3. 自營商部位多為避險需求，方向性參考價值較低
4. 全市場未平倉扣除三大法人 = 散戶部位（散戶常扮演「燃料」角色）

四種籌碼組合判讀：

情境P/C Ratio外資方向解讀操作
1> 120（偏高多）外資 Sell Put法人看好，支撐可信偏多操作
2> 120（偏高多）外資 Sell Call法人看空，散戶推高 P/C Ratio警惕反向風險
3< 80（偏空）外資 Sell Call法人看空，壓力真實偏空操作
4< 80（偏空）外資 Sell Put法人偏多，散戶恐慌可能是底部反轉

實例：

  • 情境：P/C Ratio = 140%（極度偏多），但外資 Sell Call 淨額 2 萬口
  • 判斷：散戶大量 Sell Put 推高 P/C Ratio，法人看法相反
  • 操作：不要被 P/C Ratio 誤導，反而要警戒回檔

未平倉口數淨額

是什麼：Put 未平倉量 − Call 未平倉量，相較於 P/C Ratio 的比值，淨額能更直觀看出多空差距的「絕對規模」。

怎麼用：

  • 淨額為正 → Put OI 較多（偏多訊號）
  • 淨額為負 → Call OI 較多（偏空訊號）
  • 搭配 P/C Ratio 趨勢一起看，一個看比值、一個看規模

五、P/C Ratio 與 VIX 的搭配判斷

VIX 是什麼

是什麼：台指選擇權隱含波動率指數（VIX），反映市場對未來波動的預期。VIX 俗稱「恐慌指數」——股市緩漲急跌，下跌時 VIX 飆升，上漲時 VIX 走低。

P/C Ratio + VIX 四象限判斷

象限P/C RatioVIX市場氛圍操作建議
1> 100（偏多）低（< 20）溫和多頭，波動小Sell Put 或多頭價差，收割時間價值
2> 100（偏多）高（> 25）多空激戰，波動大Buy Call 或多頭價差，控制風險
3< 100（偏空）高（> 25）恐慌偏空，波動大Sell Call 或空頭價差，但注意反轉風險
4< 100（偏空）低（< 20）溫和偏空，波動小Sell Call 或空頭價差，收割時間價值

實例：

  • 情境：P/C Ratio = 65，VIX = 35
  • 判斷：極度恐慌偏空，但極端值可能反轉
  • 操作：不追空，反而留意 Buy Call 機會

適用時機：判斷選擇權策略方向、評估市場情緒

心得

這回合學到的選擇權籌碼面分析，把之前散散學到的 Put/Call Ratio、選擇權支撐壓力、未平倉量串成了一個完整的判斷體系。

最核心的認知翻轉是：P/C Ratio 要從賣方角度解讀。以前直覺覺得 Put 未平倉多 = 很多人看跌 = 偏空，結果剛好相反——Put 未平倉多代表很多人「Sell Put」（看不跌），這些人押了龐大保證金在「指數不會跌破某個價位」上，所以是偏多訊號。這個邏輯搞清楚後，整個 P/C Ratio 的判讀就通了。

另一個重要收穫是「OP 凱文的變化量技巧」。大部分人只看 OI 最大的履約價當支撐壓力，但市場是動態的——賣方每天都在調整部位，看 OI 變化最大的履約價才能真正抓到「當下」的市況。靜態看 17,500 有 4 萬口 Call OI，但今天新增最多的是 17,600，那 17,600 才是新的壓力防線。

法人籌碼拆解更是關鍵——P/C Ratio 165% 看起來超偏多，但如果外資在 Sell Call，那這個偏多是散戶撐出來的假象。反過來說，P/C Ratio 偏低但外資 Sell Put，可能是散戶恐慌造成的好買點。數字會騙人，但籌碼組成不會。

最後，未平倉量是「趨勢燃料」這個比喻太精準了。上漲 + OI 增加 = 強勢軋空（空頭還在加碼，隨時被軋）；上漲 + OI 減少 = 多頭反轉（贏家跑路了）。以後看選擇權不只看 P/C Ratio，還要看 OI 增減方向，才能判斷趨勢還有沒有燃料。
