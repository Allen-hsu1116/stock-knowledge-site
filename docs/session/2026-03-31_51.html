核心知識

營業槓桿 (DOL, Degree of Operating Leverage)

是什麼：衡量公司使用固定成本的程度。銷售量變動1%時，EBIT（息前稅前盈餘）會變動多少百分比。

公式：

DOL = (EBIT + F) / EBIT = 邊際貢獻 / 營業利益
  • F = 固定成本（含折舊）
  • EBIT = 息前稅前盈餘

怎麼做：
1. 從損益表找出營業利益（EBIT）
2. 計算固定成本（折舊、租金、薪資等不隨銷量變動的費用）
3. 代入公式計算 DOL

實例：

  • A公司 EBIT = 1000萬，固定成本 = 1500萬
  • DOL = (1000 + 1500) / 1000 = 2.5
  • 意思：銷售量增加10%，EBIT會增加25%（10% × 2.5）

產業特性：

  • 高DOL：製造業、半導體、鋼鐵（資本密集，固定成本高）
  • 低DOL：餐飲服務業、零售業（變動成本高，固定成本低）

適用時機：

  • 評估公司營運風險：DOL越高，營運風險越大
  • 預測獲利波動：高DOL公司，營收波動時獲利波動更劇烈
  • 景氣判斷：景氣好時高DOL公司獲利爆發力強，景氣差時受傷也重

財務槓桿 (DFL, Degree of Financial Leverage)

是什麼：衡量公司舉債程度對EPS的影響。EBIT變動1%時，EPS會變動多少百分比。

公式：

DFL = EBIT / (EBIT - I)
  • I = 利息費用
  • 若有特別股：DFL = EBIT / (EBIT - I - D_pf/(1-T))

怎麼做：
1. 從損益表找出 EBIT 和利息費用
2. 代入公式計算 DFL
3. DFL > 1 代表有使用財務槓桿

實例：

  • B公司 EBIT = 2000萬，利息費用 = 500萬
  • DFL = 2000 / (2000 - 500) = 1.33
  • 意思：EBIT增加10%，EPS會增加13.3%（10% × 1.33）
  • 但EBIT減少10%，EPS也會減少13.3%

風險警示：

  • DFL越高，財務風險越大
  • EBIT下降時，高DFL公司的EPS會被「放大傷害」
  • 若 EBIT < 利息費用，公司可能陷入財務困境

適用時機：

  • 評估公司財務風險：DFL越高，破產風險越大
  • 比較同業：同產業中DFL較高者風險較大
  • 「先收後付」產業例外：超市、大賣場現金流充裕，高DFL風險較可控

總槓桿 (DTL, Degree of Total Leverage)

是什麼：結合營業槓桿與財務槓桿，衡量銷售量變動對EPS的總體影響。

公式：

DTL = DOL × DFL

怎麼做：
1. 先算 DOL
2. 再算 DFL
3. 兩者相乘得到 DTL

實例：

  • C公司 DOL = 2，DFL = 1.5
  • DTL = 2 × 1.5 = 3
  • 意思：銷售量增加10%，EPS增加30%
  • 銷售量減少10%，EPS減少30%

適用時機：

  • 整體風險評估：DTL越高，公司整體風險越大
  • 選股過濾：避開DTL過高且現金流不足的公司
  • 景氣敏感度：高DTL公司對景氣週期更敏感

實戰判斷要點

高槓桿公司的特徵：

  • 固定成本占比高（製造業、半導體）
  • 負債比率高
  • DOL + DFL 都高 = DTL爆炸

低風險高槓桿的例外：

  • 超市、大賣場、金融支付業
  • 「先收後付」模式，現金流充裕
  • 即使負債高，利息支付壓力也相對可控

選股應用：
1. 景氣復甦初期 → 找高DOL公司（獲利彈性大）
2. 景氣下行 → 避開高DTL公司
3. 利率上升期 → 避開高DFL公司（利息負擔加重）
4. 現金流差的公司 → 即使DTL不高也要小心

心得

槓桿是雙面刃這道理我懂，但今天學到的是「拆解」——同樣是高槓桿，背後原因不同，風險意義也不同。

DOL高代表固定成本重，這種公司在景氣好時賺很兇，景氣差時也很痛。DFL高代表舉債多，利息壓力大。兩個加起來的DTL才是總體風險。

最有意思的是「先收後付」產業的例外。以前看到高負債比就覺得危險，但像超市、大賣場這類公司，現金先進來、貨款後付，帳上現金充裕，高負債比反而是營運模式的自然結果，不是財務風險。

這讓我想到，看財報不能只看數字，要理解背後的商業模式。同樣是負債，有的公司是「借錢擴張」，有的是「營運特性造成」，判斷完全不同。
