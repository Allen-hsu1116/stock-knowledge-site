動能交易核心概念

什麼是動能交易？

核心邏輯：「強者恆強、弱者恆弱」——表現好的股票，未來也會表現好；表現差的，未來也會表現差。

為什麼有效？

  • 動能出現 = 資金進駐 = 可能吸引更多資金跟進
  • 失去動能 = 資金暫時不感興趣 = 上漲機率低
  • 等有動能才進場 = 資金效率最高

適用市場：股票、期貨、外匯、ETF 都適用

二、進場與出場策略

進場方式

方式一：積極追價

怎麼做：
1. 股價突破前波高點後直接追價
2. 觀察成交量是否續增
3. 最佳進場時機：開盤後30分鐘至1小時內

適用對象：盤感好、反應快的交易者

方式二：回測關鍵價進場

怎麼做：
1. 確認有效突破前波高點
2. 等待股價拉回至均價線或前波高點
3. 確認均價線不破後再掛單買進

注意：不要先掛單在均價線等！因為拉回有跌破均價線的風險

適用對象：追求風險報酬比的交易者

停損設定

對應方式一（追價進場）：

  • 停損點：前波高點下方 0.5%~1%
  • 或 4~8 tick（股價跳動單位）

對應方式二（回測進場）：

  • 停損點：跌破均價線

停利方式

tick流、一波流（短線當沖）：

怎麼做：
1. 追價買進強勢股後
2. 觀察 5~15 分鐘內是否繼續上漲
3. 若未繼續上漲、買單不連續、成交量未放大 → 可能失去動能 → 立刻出場

保護策略：獲利超過交易成本 5~10 tick 後，把停損單掛在成本價上方 2~3 tick，確保本次交易一定獲利

日內波、短波段交易者：

怎麼做：
1. 盡可能持有強勢股
2. 中盤（11點~12點）：陸續減碼
3. 收盤前（13點~13點30分）：陸續減碼
4. 若最終鎖漲停 → 可改波段操作

三、動能投資策略類型

1. 價格動能策略

怎麼做：

  • 比較今日漲幅與過去特定週期的平均
  • 例如：今日漲幅超過前3個月收盤價平均 → 視為有動能 → 進場

2. MA交叉動能策略

怎麼做：

  • 設置兩條均線：短週期快線 + 長週期慢線
  • 快線向上穿越慢線 = 黃金交叉 = 強勁上漲動能
  • 快線向下穿越慢線 = 死亡交叉 = 強勁下跌動能

3. 成交量動能策略

怎麼做：

  • 成交量增加 = 動能出現
  • 量增價漲 = 確認趨勢

4. 量價策略

核心：有價要有量，有量要有價

怎麼做：

  • 不單獨看成交量或價格
  • 價量配合才是真動能

5. 反轉策略

怎麼做：

  • 觀察 RSI、乖離率等指標
  • 價格過度乖離 = 反轉力道出現
  • 極度樂觀或悲觀 = 反轉動能

  • 盤整階段容易頻繁停損

五、實戰要點總結

動能交易三大核心

1. 選股：找有動能的股票（創新高、爆大量、均線多頭排列）
2. 進場：追價或回測買進，看個人風格
3. 出場：停損嚴格、停利依交易風格調整

適用時機

適合：

  • 明顯多頭或空頭市場
  • 有明確趨勢的股票
  • 成交量活躍的標的

不適合：

  • 盤整震盪市場
  • 無量下跌或上漲
  • 基本面惡化的股票

風險控制

1. 假突破：停損要嚴格執行
2. 盤整磨損：減少交易頻率，等待明確趨勢
3. 基本面風險：搭配基本面篩選，避免追逐沒有營運支撐的股票

心得

動能交易的核心就是「追強不追弱」，但這「追」要有方法、有紀律。今天的學習讓我理解了幾個關鍵：

第一，動能不是亂追。要有篩選條件：創新高、爆大量、均線多頭排列，這些都是資金進駐的訊號。不是漲了就追，而是要確認「真的有動能」。

第二，進場方式要看個性。積極的人直接追價，保守的人等回測。但回測買進不是「掛單等」，而是「確認不破再買」——這是關鍵差別。

第三，停利方式要看交易風格。tick流5~15分鐘沒續漲就跑；波段可以抱到中盤或收盤。重點是：獲利後要移動停損，鎖住獲利。

第四，盤整是動能交易的殺手。盤整期會一直假突破，一直停損。所以動能策略要在「有趨勢」的市場用，盤整期要縮手或用其他策略。

第五，動能策略有滯後性。今天表現好的股票，要等一段時間才能買進，可能錯過最佳進場點。但也因為這個滯後，有時候反而能吃到更大的波段。

這跟之前學的當沖、波段操作可以結合：動能選股 + 當沖時段 + 波段持有。重點是「動能只是選股方法，操作策略還是要有自己的節奏」。
