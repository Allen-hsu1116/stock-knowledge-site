洗盤 vs 出貨辨別實戰：主力手法拆解與對應策略

一、洗盤與出貨的本質差異

洗盤是什麼

是什麼：主力在拉抬股價過程中，故意製造賣壓和恐慌，讓意志不堅定的散戶拋出股票，然後在低檔承接，達到降低成本、集中籌碼的目的。洗盤的核心目的是「清洗浮額」——把不穩定的籌碼洗出去，讓後續拉升更輕鬆。

出貨是什麼：主力在高檔將手中的籌碼逐步賣給散戶，實現獲利。出貨的核心目的是「落袋為安」——把帳面利潤變成真金白銀。

兩者關鍵差異

判斷維度洗盤出貨
目的清洗浮額、集中籌碼清倉獲利了結
手法特徵快速、猛烈、跌破重要支撐溫和、緩慢、溫水煮青蛙
成交量下跌量縮、反彈放量下跌放量、反彈量縮
均線支撐跌破3日內站回跌破後無法收復
均線排列多頭排列未破壞均線轉為空頭排列
融資變化融資減少（散戶被洗出）融資增加（散戶接手）
千張大戶持續增加或持平開始減少
持續時間5~15 個交易日數週到數月

怎麼區分：不要只看單一指標，要綜合量價、均線、籌碼三面向交叉驗證。最核心的判斷是——主力在買還是在賣。

二、主力洗盤四大手法拆解

手法一：開高殺低洗盤法

是什麼：主力把股價推升到高點後突然大量賣出，製造恐慌性賣壓，讓散戶以為行情結束而停損出場，主力趁機在低檔承接。

怎麼辨識：
1. 早盤股價大幅開高，吸引散戶追進
2. 盤中突然出現大單殺出，股價急速下跌
3. 成交量在下跌時暴增，但很快被大量買單承接
4. 股價在低檔止穩後緩步回升
5. 收盤留下長下影線（如漲停打開後再收漲停）

實例：

  • 情境：某股開盤漲 5%，10 點後突然湧現大賣單殺到平盤
  • 散戶恐慌停損，但盤後看籌碼發現主力仍在買超
  • 隔天股價跳空開高，洗盤完成後重新攻擊

對應策略：

  • 如果均線多頭排列未破、主力仍在買超 → 不急著出場
  • 如果跌幅超過 5% 且跌破 21 日均線 → 提高警覺，設好停損

手法二：盤整洗盤法

是什麼：股價在特定區間長時間橫向盤整，讓不耐久盤的散戶失去耐心而出場。主力慢慢收集籌碼，達到清洗浮額的目的。

怎麼辨識：
1. 股價在一個價格區間內上下震盪（如 50~55 元）
2. 成交量逐漸萎縮
3. 時間可能長達 2~8 週
4. 均線開始糾結但不轉空
5. 融資餘額緩步下降（散戶不耐煩離場）

對應策略：

  • 在盤整區間下緣附近買進，上緣附近賣出（區間操作）
  • 或等待帶量突破盤整區上緣時追進（突破確認法）
  • 不要在盤整區中間進場，容易被兩面巴

手法三：跌停洗盤法

是什麼：主力在開盤時直接把股價打到跌停，製造極端恐慌。散戶看到跌停慌忙賣出，主力在跌停價掛買單承接散戶拋出的籌碼，然後用自己的大單打開跌停。

怎麼辨識：
1. 開盤直接跌停或盤中快速殺到跌停
2. 跌停掛單量不大，且很快就被打開
3. 打開跌停後股價快速拉升，留下長下影線
4. 成交量在跌停被打開時急速放大
5. 主力當日買超量明顯大於賣超量

對應策略：

  • 如果股票基本面良好、均線多頭排列 → 跌停可能是洗盤，不要恐慌賣出
  • 如果跌停鎖死到收盤、隔天繼續跌 → 不是洗盤，是真的出貨

手法四：高檔雙陰線洗盤

是什麼：股價一波上漲後，主力在前期頭部附近反向賣出兩天，製造連續兩根黑 K 線的洗盤陷阱。散戶以為頭部成形而賣出，主力洗完後以大陽線甚至連續漲停拉抬。

怎麼辨識：
1. 前面沒有出現天量大陽線（主力還沒出貨）
2. 突然高位收陰，第二天低開低走收光腳陰線
3. 但均線仍為多頭排列，技術指標未完全轉空
4. 第三天可能出現帶量大陽線吃掉前兩天跌幅

對應策略：

  • 如果前期沒有明顯出貨跡象、均線未破 → 不急著殺出
  • 如果第三天帶量反轉向上 → 洗盤確認，可加碼
  • 如果雙陰線後繼續下跌跌破均線 → 可能是真的變盤，設停損

三、主力出貨三大手法拆解

手法一：邊拉邊出

是什麼：主力不是一步把股價拉到目標價再出貨，而是在接近出貨價位時，減緩上升速率，一邊小量拉抬、一邊大量出貨。散戶以為主力在洗盤，洗完還會漲，結果主力已經不知不覺出貨完畢。

怎麼辨識：
1. 股價每次創新高後就回調，回調結束後又創新高（反覆震盪走高）
2. 高檔成交量明顯放大（主力大筆出貨）
3. 回調時量也放大（散戶跟著賣）
4. 主力買賣超數據：雖然股價在上漲，但主力開始轉為小賣或淨賣超
5. 結束訊號：高檔從放量轉為量縮 → 主力出貨完畢，不再需要拉抬

實例：

  • 情境：某股從 50 元漲到 80 元後，進入 78~85 元區間震盪
  • 每次創新高 85 元就回落到 80 元，再攻到 86 元又回落
  • 散戶覺得是「高檔洗盤」，結果盤後主力連續 5 日淨賣超
  • 當高檔量從 5000 張縮到 1000 張 → 主力已經出完

對應策略：

  • 高檔震盪 + 主力轉賣超 → 減碼，不要以為是洗盤
  • 高檔量縮 → 立即出場，主力已經撤退

手法二：高檔放量滯漲

是什麼：股價在高檔區域成交量暴增，但股價漲不動甚至微跌。這是最經典的出貨訊號——量很大，但買盤推不上去，因為主力在大量倒貨。

怎麼辨識：
1. 成交量暴增到 20 日均量的 2~3 倍
2. 股價卻只漲 1~2% 甚至收黑
3. 留下長上影線（避雷針）
4. 內盤成交比例明顯高於外盤
5. 主力賣超量佔成交量 20% 以上

實例：

  • 某股股價 100 元，成交量從日均 2000 張暴增到 6000 張
  • 但收盤只漲 0.5 元，留長上影線
  • 盤後主力賣超 1500 張（佔成交量 25%）
  • 判斷：高檔放量滯漲 = 出貨警報

對應策略：

  • 高檔放量滯漲 = 最強烈的出貨訊號
  • 搭配主力賣超、融資增加確認 → 立即出場

手法三：拉高出貨（誘多出貨）

是什麼：主力用少量資金快速拉升股價突破前高，製造突破假象吸引散戶追進，然後在高位大量出貨。跟「假突破」邏輯一致。

怎麼辨識：
1. 股價快速突破前高或壓力區
2. 突破時量放大，但隨後量急速萎縮
3. 突破後 1~3 天股價快速跌回壓力區下方
4. 散戶在突破時追進，結果被套在最高點

對應策略：

  • 不要看到突破就追，要等突破後 2~3 天確認「站穩」再進場
  • 突破後如果出現長上影線或量價背離 → 假突破機率高

四、洗盤 vs 出貨四步驟判斷法

步驟一：看成交量

洗盤的量：

  • 下跌量縮（主力不賣，散戶小賣）
  • 反彈放量（主力低接後拉升）
  • 整體呈現「遞減」趨勢

出貨的量：

  • 下跌放量（主力大賣）
  • 反彈量縮（沒人願意追價）
  • 高檔量暴增但價漲不動

步驟二：看均線支撐

洗盤的均線：

  • 短期均線（5 日、10 日）可能跌破，但 3 日內站回
  • 中長期均線（21 日、60 日）維持多頭排列
  • 股價回檔不超過起漲幅度的 38.2%（斐波那契）

出貨的均線：

  • 跌破 21 日均線後無法收復
  • 中長期均線開始走平或下彎
  • 均線從多頭排列轉為空頭排列

步驟三：看籌碼變化

洗盤的籌碼：

  • 千張大戶持續增加或持平
  • 融資減少（散戶被洗出）
  • 主力買賣超仍為淨買超或小賣超
  • 大戶買賣力偏多

出貨的籌碼：

  • 千張大戶明顯減少
  • 融資增加（散戶接手）
  • 主力轉為淨賣超且金額擴大
  • 大戶買賣力由正轉負

步驟四：看技術指標

洗盤的技術指標：

  • RSI 從超買區回落但未跌破 30
  • KD 高檔死亡交叉但很快黃金交叉
  • MACD 柱狀圖收斂但不跌破零軸

出貨的技術指標：

  • KD 由上往下跌破 70 + MACD 柱狀由紅翻黑 → 賣出訊號
  • RSI 跌破 50 進入空方區
  • 布林通道股價跌破中軌且中軌下彎

綜合確認公式

洗盤確認 = 量能萎縮(30~50%) + RSI 底背離 + MACD 收斂 + 布林下軌支撐

四個條件中出現三個以上 → 洗盤可能性超過 80%。

五、散戶對應策略 Checklist

懷疑洗盤時（不想被甩下車）

1. 先看均線排列：21 日、60 日均線是否仍多頭向上？是的話 → 不急著出場
2. 看成交量：下跌是否量縮？是的話 → 洗盤機率高
3. 看主力籌碼：主力是否仍在買超或小賣超？是的話 → 不必恐慌
4. 看回檔幅度：是否超過起漲幅度的 38.2%？超過的話 → 可能不是洗盤
5. 設好停損：以 21 日均線為防線，跌破且 3 日內無法站回 → 出場

懷疑出貨時（不想被套牢）

1. 先看量價關係：高檔放量滯漲？或下跌放量？是的話 → 出貨警報
2. 看主力籌碼：主力連續 3 日以上淨賣超？是的話 → 立即減碼
3. 看融資變化：融資持續增加？是的話 → 散戶在接刀
4. 看千張大戶：大戶持股連續 2 週以上減少？是的話 → 籌碼流向散戶
5. 看均線支撐：是否跌破 21 日均線且無法收復？是的話 → 趨勢反轉

特殊情境：邊拉邊出最難判斷

怎麼處理：

  • 邊拉邊出的特徵是「股價還在創新高，但主力已在賣超」
  • 不要被股價創新高迷惑，重點是看主力在做什麼
  • 如果主力連續賣超但股價還在漲 → 可能是邊拉邊出
  • 這時候用移動停損跟著股價走，一旦跌破移動停損就出場
  • 不要預設高點，因為邊拉邊出的結束點是「主力出完那一刻」，散戶無法精確預知

心得

洗盤 vs 出貨，這大概是散戶最頭痛的問題了。你在車上，主力把你甩下去；你不在車上，主力把你騙上車。怎麼做都不對。

但今天學完後，我覺得最核心的判斷邏輯其實很簡單：主力在買就是洗盤，主力在賣就是出貨。問題是散戶盤中看不到主力動態，盤後看到又慢了一拍。所以用「量價」和「均線」做第一道過濾，再用盤後籌碼做確認，是比較實際的做法。

OANDA 那篇文章給了一個很好的公式：「洗盤確認 = 量能萎縮 30~50% + RSI 底背離 + MACD 收斂 + 布林下軌支撐，四個條件三個以上成立，洗盤可能性超過 80%。」這比之前那種「感覺好像在洗盤」的模糊判斷好太多了。以後遇到股價回檔，就套這個四條件檢查，不要憑感覺。

凱基那篇最讓我印象深刻的是「邊拉邊出」這個手法。這個最陰險——股價還在創新高，散戶覺得是洗盤洗完會繼續漲，結果主力已經在偷偷出貨了。判斷的關鍵是「高檔量縮」——如果之前高檔量很大但突然量縮了，代表主力已經不需要拉抬了，出貨完畢。這個訊號比「主力轉賣超」更早出現，因為主力可以在「小量買超」的掩護下大量賣出。

另一個重點是洗盤的時間窗口：台股通常 5~15 個交易日。如果回檔超過 3 週還沒止穩，那大概率不是洗盤而是真的趨勢反轉。設定一個時間上限，不要無限等待「洗完會漲」，這是紀律。

最後，散戶永遠要記得一句話：「萬般拉抬只為出貨」。主力拉股票不是做慈善，拉到目標價就是要出。所以在高檔的時候，寧可少賺也不要被套。均線守不住就走，不要賭它是洗盤。

主力作線與騙線手法實戰辨識

一、試撮騙線：盤前最危險的假訊號

試撮是什麼

是什麼：試撮（模擬撮合）是台股開盤前（8:30~9:00）及收盤前（13:25~13:30）揭露的模擬成交價格，不會真正成交，但會顯示在看盤軟體上。

主力怎麼用試撮騙人：
1. 盤前掛假買單→試撮漲停：開盤前掛大量買單讓股價呈現漲停，散戶看到以為有利多，開盤追價買進，主力趁機用漲停價倒貨
2. 盤前掛假賣單→試撮跌停：開盤前掛大量賣單讓股價呈現跌停，散戶恐慌殺出，主力趁低接貨
3. 關鍵手法：在 9:00 開盤前取消假單，完全不成交，但已成功影響散戶情緒

防範方法：
1. 絕對不要因試撮價格下單：試撮只對掛假單的人有用，對一般人毫無意義
2. 等開盤5分鐘後再觀察：真正的買盤會持續，假的會消失
3. 注意暫緩開盤機制：證交所在開盤前1分鐘（8:59~9:00），如果模擬價格漲跌超過3.5%，或取消/變更委託數量達30%以上，會暫緩2分鐘開盤——這就是用來防範主力掛假單的

實例：

  • 某股盤前試撮漲停價50元，散戶看到後開盤追進，結果開盤價只有47元，盤中一路跌到45元
  • 主力用0成本（因為開盤前取消委託不成交）就騙到散戶追高

緩漲/緩跌試撮

是什麼：盤中如果模擬成交價與前5分鐘加權平均價差距超過3.5%，證交所會暫緩撮合2分鐘。

散戶要注意：

  • 緩漲試撮期間只能掛限價ROD單，不接受市價單、IOC、FOK
  • 暫緩前的市價ROD未成交單會自動刪除
  • 2分鐘冷靜期→用集合競價撮合→恢復逐筆撮合

二、真假突破辨識實戰

真突破 vs 假突破的核心差異

是什麼：真突破是有基本面、消息面、資金面支撐的價格突破，假突破是主力刻意拉抬製造的假象，沒有實質利多佐證，後續漲勢無以為繼。

怎麼判斷真假突破：

1. 量能確認：

  • 真突破：突破時成交量放大1.5倍以上，資金願意追價

2. 站穩確認：

  • 真突破：突破後連續3天收盤價站穩在突破點之上

3. 回踩測試：

  • 真突破：回踩壓力變支撐，不跌破突破點

4. 基本面支撐：

  • 真突破：有實質利多（營收成長、產業趨勢、新產品等）

實例：

  • NVIDIA 2020年6月~2021年7月在區間盤整，突破時伴隨虛擬貨幣大漲的利多（GPU需求暴增），量價齊揚，這是真突破
  • 反之如果某冷門股突然突破長期壓力，但沒有任何利多消息，量能也沒放大，大概率是假突破

突破交易的時間對應法則

經驗法則：

  • 想賺1個月波段 → 前面至少盤整3個月
  • 想賺半年波段 → 前面最好盤整1~2年
  • 想賺1年以上大波段 → 前面要盤整5年以上

為什麼：盤整越久，籌碼沉澱越徹底，突破後的爆發力越強。「沉寂多年突然爆發」的股票，是突破交易最理想的標的。

最適合突破交易的標的：

  • 原本冷門、低檔盤整很久的股票
  • 突然有重大利多（如產業反轉、新技術突破）
  • 特徵：冷門股+利多爆發=反應劇烈

不適合突破交易的標的：

  • 大型權值股（如台積電、鴻海），體量太大，單一利多難以推動大漲
  • 已經大漲過的熱門股，再突破的空間有限

三、主力騙線三大手法拆解

手法1：長上影線避雷針騙局

是什麼：主力在主升段中故意拉出長上影線（看起來像避雷針），搭配大量成交量，讓散戶以為是高檔反轉訊號而恐慌賣出，主力洗到籌碼後再拉一波。

怎麼辨識：
1. 觀察第一防守線（長紅K線最低點）：如果收盤價守在第一防守線之上，就不是出貨
2. 觀察突破線（上影線最高點）：後續股價如果穿越突破線，確認是洗盤而非出貨
3. 盤中跌破 vs 收盤收回：盤中即使跌破第一防守線，但收盤收回，仍然算守住

實例：

  • 利勤（4426）出現白色長上影線+綠色巨量，看起來完完全全是教科書說的避雷針。但後續股價都沒有跌破第一防守線（即使盤中跌破，收盤也收回），最後穿越突破線，開啟新一輪上漲行情
  • 關鍵：一般散戶都懂的東西主力當然也懂，主力會反向運用

手法2：長黑跌破盤整區甩轎

是什麼：股價在盤整區守了好幾天，突然一根長黑灌破盤整區，嚇得所有人停損出場。結果停止下跌後，又用長紅往上拉，開啟主升段。

怎麼應對：
1. 不要賭是甩轎：長黑灌破盤整區時，乖乖停損出場，萬一是真的往下噴就完了
2. 確認後再買回：觀察後續股價是否吞噬長黑最高點，確認是甩轎再買回
3. 犧牲小虧避免大賠：停損後再追回，中間的價差就是「保險費」

實例：

  • 宏佳騰（1599）股價守在第一防守線47.3盤整，突然長黑灌破，散戶恐慌出場。結果停止下跌後長紅往上拉，吞噬長黑最高點，確認是起漲前甩轎

手法3：試撮假單操縱開收盤價

是什麼：主力利用試撮制度，在開盤前或收盤前掛大量假單，製造漲停或跌停的假象，影響散戶判斷後在開盤前取消。

防範三招：
1. 完全不看試撮價格：試撮對散戶來說只有風險沒有價值
2. 開盤後觀察5分鐘：真買盤會持續，假買盤會瞬間消失
3. 注意「暫緩開盤」訊號：如果出現暫緩開盤，代表有人在搞鬼，更要冷靜

四、CIS 主力思維：散戶為何永遠輸

順勢操作勝率最高

  • 持續上漲的股票大概率會再漲，持續下跌的股票大概率會再跌
  • 「回檔買進」是錯誤觀念：稍微下跌時買進是一種逆勢操作
  • 買進的股票如果下跌，應該果斷賣掉，不要想「等反彈」

攤平是最差勁的技巧

  • 明知失敗卻加碼買進，應該承認錯誤果斷停損
  • CIS 從100萬日圓起家到2300億日圓，核心原則就是「不攤平」
  • 投資者不能避免損失，但必須避免巨大的損失

三隻泥鰍理論

  • 第1隻泥鰍：市場剛開始注意到新趨勢，少數人發現，獲利空間最大
  • 第2隻泥鰍：趨勢被更多人注意，仍可獲利但空間縮小
  • 第3隻泥鰍：所有人都知道了，散戶湧入，幾乎沒有獲利空間
  • 實戰應用：一發現有獲利機會就馬上行動，不要等媒體報導才進場

市場恐慌時就是機會

  • CIS 基本上不避險，承擔風險就是要追求利潤
  • 恐懼之時就是機會來臨之時（與巴菲特「別人恐懼我貪婪」理念相同）
  • 股價上漲創新高時，CIS 會欣然買進；一般散戶反而擔心是不是要下跌了

心得

主力作線最可怕的地方不是手法多高明，而是散戶太容易被教科書綁架。教科書說「避雷針要跑」，主力就故意做避雷針來洗你；教科書說「跌破支撐要停損」，主力就故意長黑灌破來甩你下車。

防範作線的核心不是記更多口訣，而是建立自己的防守線紀律：守住第一防守線就續抱，跌破就出場，不要猜、不要賭。被洗出去小虧一次，總比遇到真出貨大賠要好。

試撮騙線更是零成本的騙術——主力掛假單不用花一毛錢就能影響散戶情緒。面對這種騙術最好的方法就是完全不看試撮價格，等開盤5分鐘後真正的買盤力道才是真的。

CIS 的三隻泥鰍理論也很有啟發：散戶永遠在追第3隻泥鰍，因為第3隻泥鰍是媒體告訴你的。但第3隻泥鐵已經沒肉了。想賺錢，要搶第1隻——趁沒人注意時就進場，不要等新聞報導才動。

朝股票操作大師邁進中 📈## 第 12 回合學習 - 主力對倒交易與虛假成交量辨識

時間：12:30

文章：

  • 操縱股價的風險有什麼？法律規範一次看
  • 主力出貨的前兆：如何識別市場中的隱藏信號
  • 遇到這些情況別被甩下車！揭露主力洗盤、甩轎手法
  • 主力習慣在這2個時間點偷偷倒貨

一、主力對倒交易是什麼

是什麼：主力對倒（Wash Trading）是指操控者利用自己掌控的多個帳戶，相互買賣同一檔股票，製造出虛假的交易活絡假象。左手賣、右手買，籌碼始終在自己手裡打轉，但成交量卻被灌水，讓散戶誤以為「這檔股票很熱門」。

怎麼做到的：
1. 多帳戶對敲：主力開設數十個人頭帳戶，A帳戶掛賣單、B帳戶掛買單，撮合後籌碼仍在自己手中
2. 相對委託：與約定好的對手方，同時以約定價格申報買進和賣出，讓成交價格被鎖定在想要的位置
3. 沖洗買賣：連續申報買進又賣出，製造交易量放大的假象，吸引散戶跟進

法律層面：這些行為在台灣證券交易法第155條明確禁止，構成操縱股價罪，刑責為3年以上10年以下有期徒刑，併科1,000萬至2億元罰金。若不法利得超過1億元，刑責加重至7年以上有期徒刑。

實例：

  • 情境：某小型股日成交量長期僅500張，某天突然放大到5,000張，但股價僅微漲0.5%
  • 判斷：大量成交但價格不動，極可能是對倒交易製造的虛假量能
  • 操作：這時候千萬不要因為「量放大」就追進，因為這些量可能是假的

二、對倒交易五種辨識特徵

特徵一：量增價不動或微動

怎麼看：
1. 對照成交量與漲跌幅——如果某天成交量是5日均量的3倍以上，但漲跌幅不到1%，就要懷疑
2. 觀察連續3天的量價關係——如果連續放量但股價始終在狹窄區間（1~2%）震盪，對倒機率高
3. 用技術軟體看「大單買賣比」——如果大單買超和賣超金額相近，代表大戶兩邊掛單自買自賣

實例：

  • 某股票5日均量1,000張，今日突然成交5,000張，漲幅僅0.3%
  • 大單買超2,500張、大單賣超2,400張，幾乎打平
  • 判讀：大戶對倒製造量能假象，真實買盤遠不如帳面數字

特徵二：買賣家數差異極小

怎麼看：
1. 查看證交所「交易家數」資料——如果買方家數和賣方家數差值極小（例如買方12家、賣方11家），且集中在少數分點
2. 正常熱門股買方可能有50~100家以上，如果一檔股票量大但交易家數少，就是異常
3. 對照「買賣家數差」指標——買賣家數差接近零，代表多空力量「太對稱」，不符合自然交易規律

實例：

  • 某股今日成交3,000張，但買方僅15家、賣方僅14家
  • 同時，某單一分點買超1,200張、另一分點賣超1,150張
  • 判讀：籌碼集中在極少數帳戶之間流動，對倒可能性極高

特徵三：分點買賣超高度對稱

怎麼看：
1. 盤後查看「券商分點買賣超」資料——如果買超前3名和賣超前3名的張數極為接近（差距<10%），就是高度對稱
2. 對照歷史資料——這種「完美對稱」是否反覆出現在相同分點？如果是，幾乎可以確定是對倒
3. 追蹤分點歷史績效——對倒分點的操作績效通常極差（因為不是真正在賺錢，而是在製造量能）

實例：

  • A分點買超800張、B分點賣超780張，數字幾乎鏡像
  • 連續5個交易日，A和B分點的買賣超持續高度對稱
  • 判讀：兩個分點極可能是同一主力的人頭帳戶

特徵四：特定時間帶量但無續航力

怎麼看：
1. 觀察盤中走勢——對倒量能通常集中在開盤（9:00-9:30）或收盤前（13:00-13:30），主力在這兩個時段大量掛單製造量能
2. 對照「主力習慣倒貨的2個時間點」：

  • 9:00-10:30 發勢盤：法人搶第一時間布局，真偽混雜

3. 如果某檔股票在特定時段突然放量（佔全天量能60%以上），但其餘時間完全沒有量，要懷疑是對倒

實例：

  • 某股9:00-9:15成交2,000張（佔全天70%），之後6小時僅成交800張
  • 且開盤15分鐘的成交價格波動極小（高低差不到0.5%）
  • 判讀：開盤大量對倒拉出「量能放大」的假象，之後真實買盤薄弱

特徵五：內盤成交比例異常

怎麼看：
1. 對比「內盤成交比例」——如果某天內盤成交佔比突然從正常的40-50%飆升到70-80%，代表大部分成交是「主動賣出」
2. 但如果股價卻不跌反漲，這矛盾現象就是對倒的典型特徵——主力一邊自賣自買製造內盤量，一邊用小量資金拉高股價
3. 用「內外盤比+量價背離」雙重確認——內盤大於外盤但價漲 = 警訊

實例：

  • 某股內盤成交2,100張（70%）、外盤900張（30%），但股價上漲1.5%
  • 正常情況：內盤>外盤應該是下跌，現在卻上漲
  • 判讀：主力對倒製造內盤量（假裝有人在賣），同時用小量真金拉價格

三、對倒交易防範四大原則

原則一：量能要拆開看「真實量」vs「對倒量」

怎麼做：
1. 把每日成交量扣除「大單買賣對稱量」——如果買超前3和賣超前3的張數加起來佔成交量的50%以上，且高度對稱，真實量可能只有帳面量的40-50%
2. 用「交易家數」估算——一檔股票如果日成交3,000張但只有15個買方，平均每家買200張，集中度極高
3. 對照5日和20日均量——如果單日量能是5日均量的3倍以上，但5日均量本身沒有同步上升，代表這天的量可能是「做」出來的

原則二：不追「量突然放大」但「價格不動」的股票

怎麼做：
1. 設定篩選條件：成交量>5日均量×2 且 漲跌幅<±1% = 警訊股
2. 這類股票的「量」很可能是對倒做出來的，真實買盤不足以推動價格
3. 寧可錯過也不要追——因為一旦主力停止對倒，量能會瞬間萎縮，股價可能崩跌

原則三：追蹤分點歷史績效辨識人頭帳戶

怎麼做：
1. 選定目標股後，回測買超前3名分點在這檔股票過去3個月的操作紀錄
2. 如果某分點每次買超後，股價都沒有明顯上漲（甚至下跌），這個分點很可能是「對倒用」的人頭帳戶
3. 反之，如果某分點買超後股價穩定上漲，才是真正的「主力進場」訊號
4. 建立觀察清單：將疑似人頭帳戶的分點記錄下來，長期追蹤

原則四：結合多重指標交叉驗證

怎麼做：
1. 量價背離確認：成交量放大但OBV（能量潮指標）不升反降 = 量是假的
2. KD指標確認：如果KD鈍化在高檔但成交量暴增，可能是主力邊對倒邊出貨
3. RSI背離確認：股價創新高但RSI不創新高 + 成交量異常放大 = 高度警訊
4. 千張大戶持股變化：如果成交量放大但千張大戶持股數量沒有增加，代表量能不是來自真正的大戶

四、對倒交易 vs 正常大量成交 比較表

心得

主力對倒這東西，說穿了就是左手賣右手買，看起來很忙其實籌碼根本沒換人。散戶最容易被騙的就是「量能放大」這個訊號——看到量暴增就以為有人在搶，結果搶的是主力自己人。

辨識對倒最核心的邏輯就一句話：量要真，價要跟，家數要多，分點要散。任何一個條件不滿足，就要懷疑。

最實用的判斷方法其實是看分點買賣超的對稱性——如果買超第一和賣超第一的數字幾乎一樣，那八成是對倒。這在盤後資料就能查到，不用什麼高深技術。

另外一個很實用的觀察是「量增價不動」——如果一檔股票突然量放大3倍以上，但漲跌幅不到1%，這就跟路上有人大聲喊「快來搶」但店裡沒人排隊一樣，量是假的。

以後看到放量股，先查三件事：交易家數多少、分點對不對稱、量價有沒有背離。這三關都過了，量才是真的。

  • 股票換手率是什麼？5分鐘看懂股票換手率太高太低的優缺點

一、換手率定義與計算

換手率是什麼

是什麼：換手率（Turnover Rate）又稱週轉率，是指一段時間內股票成交量佔流通股數的百分比。它反映的是「籌碼流動的速度」——有多少比例的持股在這段期間內換了主人。這是真金白銀堆出來的數字，做不了假。

計算公式：

換手率 = (成交量 ÷ 流通股數) × 100%

為什麼換手率比成交量更重要：
成交量是絕對數字，但不同市值的股票成交量無法直接比較。例如 10 億市值和 100 億市值的股票，日成交額都是 1 億，活躍度天差地遠。換手率把市值差異標準化了，讓你可以跨股比較。

1%~3%正常活躍度，冷門股可觀察，不急進
3%~7%相對活躍，開始有人氣進入觀察名單
7%~10%活活躍股，廣受市場關注主力可能正在進場
10%~15%高度活躍，主力運作跡象看清意圖再動，觀察為主
15%~25%異常活躍，可能是出貨搭配 K 線型態判斷，收黑要警覺
> 25%極端異常，出貨機率極高收黑線＝主力出貨，不要碰
> 70%死亡換手率次日大跌機率 8 成以上，立刻離場

三、換手率搭配股價位置的六種情境

情境 1：低檔 + 換手率 5%~10% + 放量收紅

判斷：主力吸籌，強烈買入訊號

怎麼做：
1. 確認股價在 20 日均線附近或剛突破
2. 換手率落在 5%~10%，成交量明顯大於前幾日
3. 收盤價接近日高，K 棒實體長
4. 下小單試探（目標倉位的 20%），停損設 5%~8%

實例：

  • 某股長期在 25~28 元盤整，20 日均線在 26.5 元
  • 突然一天成交量暴增 3 倍，換手率 7.2%，收盤 29.5 元（接近日高 30 元）
  • 操作：買入 20% 倉位，停損設 27 元（約 8.5%）

情境 2：盤整期 + 換手率穩定 3%~6%

判斷：洗盤格局，主力高出低進吸收浮動籌碼

怎麼做：
1. 觀察股價在盤整區間內小幅震盪
2. 換手率維持一定水準（3%~6%），代表有人持續換手
3. 等待「量縮後再放量突破」的加碼點
4. 突破時分批加碼（2~3 次），每次加碼後重設停損

實例：

  • 某股在 40~45 元區間盤整 2 週，日均換手率 4.5%
  • 某天量縮至換手率 1.8%（洗盤尾聲），隔日放量突破 45 元，換手率 8%
  • 操作：突破當天加碼至 60% 倉位，停損上移至 42 元

情境 3：高檔 + 換手率 > 20% + 收黑

判斷：主力出貨，應減碼或出場

怎麼做：
1. 股價已上漲一段（從起漲點漲 50% 以上）
2. 單日換手率飆破 20%，但收盤收黑或長上影線
3. 立即減碼 50% 或全部出場
4. 如果還想觀察，至少把停損移到成本價

實例：

  • 某股從 30 元漲到 65 元，某天換手率高達 23%，收盤跌 3% 收黑
  • 操作：立即出場至少一半，嚴格停損在 60 元

情境 4：高檔 + 換手率 > 25% + 收黑 K 線

判斷：主力明確出貨訊號，不要碰

怎麼做：
1. 換手率 > 25% 加上收黑，幾乎可以確認主力在倒貨
2. 手上有部位的全部出場
3. 沒有部位的絕對不要進場接刀

實例：

  • 某股連續上漲後，某天換手率 28%，K 線收長黑
  • 操作：全數出場，後續不管怎麼跌都不接

情境 5：高位橫盤 + 換手率 > 25%

判斷：主力正在高位持續出貨

怎麼做：
1. 股價在高檔橫盤但換手率居高不下
2. 代表主力每天慢慢倒，散戶每天慢慢接
3. 避開這類股票，就算盤整看起來穩定也不要進

情境 6：死亡換手率 > 70%

判斷：持股人幾乎全部換過一輪，次日大跌機率 8 成以上

怎麼做：
1. 不論漲跌，立刻離場
2. 就算次日漲停也是賭博，勝率極低
3. 這種情況常見於小型股被炒作的極端案例

四、換手率實戰 SOP 五步驟

步驟 1：篩股

  • 過濾 20 日均線方向：只追蹤股價在均線上方或剛突破的
  • 排除高檔弱勢股
  • 優先追蹤低檔盤整但「近期換手突然放大」的股票
  • 每日或每週更新 watchlist

步驟 2：量價確認

  • 看當日換手率是否落在 5%~10%
  • 確認收盤價接近日高（K 棒實體長）
  • 收紅且放量＝強烈訊號
  • 收黑或收中段＝訊號弱，等確認

步驟 3：洗盤判斷

  • 盤整期換手率穩定在 3%~6%＝主力洗盤
  • 等待量縮→再放量突破的加碼點
  • 分批加碼，每次重新評估停損

步驟 4：高檔警示

  • 單日換手率 > 20%＝提高警覺
  • 收黑＝立即減碼或出場
  • 收紅但量能持續放大＝可以抱但要嚴格移動停損

步驟 5：每日檢核

  • 有無高檔巨量訊號？→ 評估減碼
  • 有無放量突破且收紅？→ 可續抱或加碼
  • 是否觸及停損？→ 依規則執行
  • 寫交易日誌記錄

五、新股換手率判斷

關鍵數字：新股上市首日換手率 > 52%

判斷邏輯：
1. 首日換手率超過 52%：主力搶到足夠籌碼
2. 之後幾天換手率持續偏高，股價連續收紅：主力正在建倉
3. 可以盯準這種股票，後續可能出現一波行情
4. 但如果首日換手率偏低（< 30%），代表主力沒興趣，後續表現通常不佳

六、換手率不要買的三種股票

籌碼九階段進階判讀與散戶對策

之前學過「籌碼九階段論：接護拉換養軋出套殺」的基本定義，但光知道名詞不夠，重點是判斷自己站在哪個階段，以及每個階段散戶該做什麼。

一、九階段快速定位 Checklist

階段股價位置籌碼特徵散戶可見訊號操作方向
1.接底部盤整或緩跌大戶買超、散戶賣超、融資減成交量萎縮、內部人持股增加🟡 觀察名單
2.護底部築底大戶持續買、股價不跌跌不下去的感覺、融資持續減🟡 開始研究
3.拉初升段主力買超放大、成交量放大帶量長紅突破、法人開始買🟢 進場點
4.換中段整理主力換手、部分買超停滯高檔震盪、融資微增🟡 持股續抱
5.養主升段主力買超減少、融資大增利多頻出、媒體曝光、散戶搶進🟠 提高警覺
6.軋末升段主力買超停滯、融券暴增消息見報率大增、社群瘋傳🔴 準備出場
7.出頭部區主力開始賣超、融資成長法人還在追捧但股價上不去🔴 出場
8.套初跌段主力賣超、千張大戶減少開高走低、上影線、大戶賣散戶買🔴 絕對不接
9.殺主跌段散戶恐慌賣超、多殺多跌停、融資斷頭、恐慌性殺出⚫ 放空或觀望

二、關鍵階段深入判斷

階段 5「養」→ 階段 6「軋」的轉折判斷

這是最容易上當的階段。主力在「養」的時候，會製造出漂亮的線型和消息面。

怎麼判斷已經進入「養」的階段：
1. 融資連續 5 日以上增加，且增幅大於法人買超
2. 主力買超開始減少，但股價還在漲（靠散戶推上去的）
3. 消息面突然大量出現（之前都沒人講，突然一堆人推薦）
4. 成交量放大但漲幅縮小（量價背離的徵兆）

實例：

  • 某檔股票從 50 元漲到 120 元，法人買超從前期的每日 2000 張降到 500 張
  • 同時融資餘額從 5000 張暴增到 15000 張
  • 電視名嘴開始推薦、PTT 討論度暴增
  • → 這就是「養」的末段，距離「軋」不遠了

階段 7「出」的辨識重點

五個出貨確認訊號：
1. 主力分點連續 3 日以上賣超
2. 融券開始回補（空手被軋完了）
3. 利多不斷但股價卻漲不動（甚至開高走低）
4. K 線出現長上影線或長下影線（偏弱訊號）
5. 千張大戶持股比例開始下降

實例（國巨 2327 的教訓）：

  • 2017-2018 年被動元件大行情，國巨從 100 多漲到 1310
  • 在軋空段，主力就開始慢慢出貨
  • 消息面仍然一片大好（缺貨、漲價），但外資借券賣出不斷增加
  • 軋空結束後，主力更是用力出貨
  • 散戶看到利多就衝進去，結果全被套在高點

階段 8「套」→ 階段 9「殺」的避雷方法

被套的特徵：
1. 消息面利多不斷（主力還在放消息掩護出貨）
2. K 線有長上影線與下影線，或直接長黑
3. 股價盤頭且位於短期均線（5MA、10MA）下方
4. 大戶賣超、散戶買超（籌碼由集中轉分散）

殺的觸發：

  • 主力出貨完畢，不再護盤
  • 融資斷頭潮啟動，多殺多
  • 跌停鎖死，想跑都跑不掉

三、散戶對策：每個階段該怎麼做

在「接、護」階段——跟主力一起撿便宜

怎麼做：
1. 用籌碼K線或集保庫存查「千張大戶持股」連續 3 週增加
2. 確認融資餘額在減少（散戶在賣）
3. 等待帶量長紅突破（進入「拉」的階段）再進場
4. 不要在「接」的階段就進場——你可能會等很久

在「拉」的階段——最佳進場時機

怎麼做：
1. 法人連續買超 + 帶量突破關鍵價位
2. 設停損在突破K線的低點
3. 目標看「養」的階段（等融資開始大增時就要小心）

在「養、軋」的階段——準備下車

怎麼做：
1. 設移動停損（如 5MA 或前低）
2. 不要再追高加碼
3. 開始注意主力分點是否轉為賣超
4. 看到「消息面大好但股價不漲」→ 立即出場

在「出、套、殺」的階段——絕對不要碰

怎麼做：
1. 如果還沒買：不要接刀
2. 如果已經被套：認賠出場，不要等反彈
3. 如果想放空：等反彈再空，不要直接空在跌停

誤區錯誤認知正確理解

主力連買=無風險主力買就跟可能是誘多吸籌，需搭配量價確認

五、實戰 SOP：判斷股票所處階段

1. 看千張大戶持股趨勢（連續 3 週以上）→ 判斷籌碼集中或分散
2. 看融資餘額變化 → 散戶進出方向
3. 看主力分點買賣超 → 是否有連續性
4. 看法人買賣超 vs 借券賣出 → 外資是真買還是避險
5. 看消息面 vs 股價走勢 → 利多不漲=出貨，利空不跌=吸籌
6. 對照九階段表 → 確認目前位置
7. 依位置決定操作 → 接護觀察、拉時進場、養軋警覺、出套殺避開

心得

籌碼九階段之前學過基本定義，但這次深入到「怎麼判斷自己站在哪裡」，差別很大。就像知道紅綠燈的顏色定義，跟知道現在是紅燈還是綠燈——後者才能決定要不要過馬路。

最大的體悟是：「養」的階段是最危險的。因為那時候一切都看起來很美好——線型漂亮、利多不斷、親友推薦——但這恰恰就是主力設計好的陷阱。散戶要在「拉」的時候進場、在「養」的時候警覺、在「出」的時候離場，這才是跟著主力走而不是被主力養。

另一個重點是籌碼面不能單獨用。主力買超但K線破底，那就是矛盾訊號；融資大增但股價還在漲，代表散戶進來了但主力還沒走完。任何單一指標都不能當作聖杯，九階段定位要搭配量價和技術面一起看才準。
