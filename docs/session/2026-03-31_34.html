股利折現模型(DDM)

第 17 回合學習 - 股利折現模型(DDM)

時間：12:45
文章：

  • 股利折現模型是什麼？公式如何計算？用來估價時的注意事項整理
  • 如何用股息折現模型DDM計算合理價格

核心知識

股利折現模型(DDM)是什麼？

是什麼：一種股票估值方法，將未來預期發放的股利，用適當的折現率折算成現值，來估算股票的內在價值。

核心概念：股票的真正價值 = 未來所有股利的現值總和。因為「今天的錢比未來的錢更值錢」，所以未來的股利要「打折」才能換算成今天的價值。

適用對象：

  • 穩定配息的成熟公司
  • 股利連續成長的企業
  • 不適合成長股或不配息的公司

三種DDM公式

1. 單期股利折現模型

公式：V₀ = D₁÷(1+r) + P₁÷(1+r)

適用時機：只打算持有一年時使用。

是什麼：把「一年後拿到的股利」和「一年後賣掉的股價」折現到今天。

實例：

  • 預期明年股利 D₁ = 3元
  • 預期一年後股價 P₁ = 110元
  • 折現率 r = 8%
  • 內在價值 V₀ = 3÷1.08 + 110÷1.08 = 2.78 + 101.85 = 104.63元

2. 多期股利折現模型

公式：V₀ = D₁÷(1+r)¹ + D₂÷(1+r)² + ... + Dₙ÷(1+r)ⁿ + Pₙ÷(1+r)ⁿ

適用時機：預計持有多年，且能預測每一年的股利。

實例：

  • 預期3年內股利分別為：D₁=2元、D₂=2.5元、D₃=3元
  • 第三年底預期股價 P₃ = 120元
  • 折現率 r = 10%
  • V₀ = 2÷1.1 + 2.5÷1.21 + 3÷1.331 + 120÷1.331
  • V₀ = 1.82 + 2.07 + 2.25 + 90.15 = 96.29元

3. 高登成長模型（最常用）

公式：V₀ = D₁ ÷ (r - g)

適用時機：股利穩定成長的公司，最實用的公式。

參數說明：

  • D₁ = 明年預估股利
  • r = 折現率（預期報酬率），通常用8%-12%
  • g = 股利年成長率

實例：

  • 明年預估股利 D₁ = 3元
  • 折現率 r = 8%
  • 股利成長率 g = 5%
  • 內在價值 V₀ = 3 ÷ (8% - 5%) = 3 ÷ 0.03 = 100元

判斷方式：

  • 若目前股價 = 110元，內在價值 = 100元 → 股價被高估10元
  • 若目前股價 = 90元，內在價值 = 100元 → 股價被低估，有投資價值

DDM vs DCF 差異

項目DDM（股利折現）DCF（現金流折現）
折現標的未來股利未來自由現金流
適用公司穩定配息的成熟公司所有公司（含不配息的）
計算難度較簡單較複雜
保守程度較保守較寬鬆
缺點會漏掉不配息的好公司變數多，預測可能不準

關鍵差異：DDM是DCF的一種特殊形式，差別在於用「股利」還是「自由現金流」當折現標的。

DDM的三大缺點

缺點1：只能評估穩定配息的公司

問題：不配息的公司（如Google、Amazon、波克夏）無法用DDM估價。

解法：改用DCF模型，用自由現金流替代股利。

缺點2：需要大量假設

問題：成長率、折現率都是「猜」的，只要假設錯誤，算出的價值就會差很多。

實例：

  • 同一家公司，成長率估5% vs 10%，合理價可能差一倍
  • 折現率用8% vs 12%，結果也會差很大

缺點3：過去配息不等於未來配息

問題：

  • 公司可能賣土地、賣事業體，當年股利暴增（一次性的）
  • 公司政策可能改變（如台積電2013年前固定配3元，後來才調高）
  • 配息來源可能是業外收入，不是常態獲利

解法：看「配息率」是否穩定，而非只看股利金額。

實務操作：如何保守使用DDM

1. 成長率要保守

怎麼做：

  • 過去10年成長率12% → 預估7%就好
  • 分段預估：前5年7%、後5年5%、之後0%

實例：

  • 營收成長2% + 淨利成長2% + 庫藏股貢獻3% = 合計7%
  • 配息率30%且穩定 → 7%成長率合理

2. 折現率要合理

一般範圍：

  • 8%-10%：相當於股市長期年化報酬率
  • 12%：公司風險較高或想更保守時使用

原則：折現率越高，算出的合理價越低（越保守）。

3. 加上安全邊際

怎麼做：算出合理價後，打8折再買進。

實例：

  • 合理價 = 100元
  • 進場價 = 100 × 80% = 80元
  • 多出20元的緩衝空間，降低評估錯誤的風險

兩段式DDM進階用法

概念：將股利成長分成兩階段

  • 第一階段：前10年的成長率（較高）
  • 第二階段：第11年後的「高原成長率」（較低）

實例：

  • 目前股利 = 1元
  • 第一階段（前10年）成長率 = 7%
  • 第二階段（第11年後）成長率 = 5%
  • 折現率 = 10%
  • 合理價 = 24.54元

計算邏輯：

  • 第1年股利 = 1 × 1.07 = 1.07，現值 = 1.07 ÷ 1.1 = 0.97
  • 第2年股利 = 1.07 × 1.07 = 1.14，現值 = 1.14 ÷ 1.21 = 0.94
  • ...依此類推，加總所有現值

使用DDM的注意事項

1. 小心價值陷阱

股價下跌 + 獲利衰退 = 不是便宜，是陷阱

怎麼判斷：

  • 檢查公司獲利能力是否真的衰退
  • 看配息率是否異常暴增（可能來自業外收入）

2. 檢查配息率穩定度

配息率 = 股利 ÷ EPS

正常範圍：30%-80%（依產業而定）

警訊：

  • 配息率 > 100% → 動用老本，不健康
  • 配息率劇烈波動 → 公司政策不穩定

3. 只適合簡單企業

適合：成立多年、業務單純、獲利穩定的公司

不適合：新創公司、成長股、景氣循環股

心得

DDM 比 DCF 更保守，但也更受限。它的核心假設是「股利是股東唯一能拿到的現金」，這在美股尤其有問題——很多公司用庫藏股代替配息，DDM完全無法反映這些公司的價值。

高登公式 V₀ = D₁ ÷ (r - g) 看起來簡單，但魔鬼藏在細節裡：
1. 成長率 g 不能大於等於 r，否則公式失效（分母變零或負數）
2. 永續成長的假設在現實中很少成立
3. 一個參數差 1%，結果可能差 10% 以上

實務上，我會這樣用 DDM：

  • 只用在股息貴族（連續配息25年以上）
  • 成長率抓歷史平均值打7折
  • 折現率至少10%
  • 合理價再打8折當買進價

這方法很保守，可能會讓你「沒東西可買」。但寧可錯過，不可做錯。耐心等待是價值投資的基本功。

核心知識

現金流量折現法（DCF）是什麼？

是什麼：DCF（Discounted Cash Flow）是把公司未來會產生的所有自由現金流，折現回今天的價值總和。巴菲特說過：「上市公司的內在價值，就是該企業在其未來所能產生的自由現金流量折現後的總和。」

為什麼用自由現金流而不用盈餘：

  • 盈餘沒有考慮折舊（以前花掉的投資攤在本期帳上）
  • 盈餘沒有考慮資本支出（未來必須要花的投資）
  • 自由現金流才能真正反映公司可以自由運用的資金

DCF 公式與計算步驟

公式：

DCF 現值 = ∑ 未來n期現金流 ÷ [(1+折現率) ^n年]

三大核心元素：

1. 未來現金流（CF）

  • 用自由現金流（FCF）來計算
  • 通常預估未來 5~15 年
  • 越遠期的預測誤差越大
  • 保守做法：只算近 10 年，之後當成零

2. 折現率（r）

  • 通常用加權平均資本成本（WACC）
  • 代表投資人的機會成本
  • 例如：投資人把錢放別處能賺 7%，這個投資也至少要 7%
  • 折現率越高，未來現金流折現後越小

現值計算例子：

  • 折現率 5%，一年後 10,000 元的現值
  • 10,000 ÷ (1 + 0.05)¹ = 9,523.81 元
  • 兩年後的 10,000 元現值 = 10,000 ÷ (1.05)² = 9,070.3 元
  • 越遠的現金流，現值越小

3. 期間（n）

  • 一般計算未來 10~15 年
  • 保守可以只算 5~10 年
  • 數學上越遠期開越多次方，數值變得很小

DCF 計算實例

情境：某公司未來 10 年每股自由現金流都是 10 元，10 年後終值 125 元，折現率 10%

計算：

DCF = 10÷1.1 + 10÷1.21 + 10÷1.331 + ... + 10÷2.5937 + 125÷2.5937
DCF = 9.1 + 8.3 + 7.5 + 6.8 + 6.2 + 5.6 + 5.1 + 4.7 + 4.2 + 3.9 + 48.2
DCF = 109.6 元

解讀：

  • 今天用 109.6 元買進，年化報酬率就是 10%
  • 如果現在股價低於 109.6 元，報酬率會高於 10%
  • 如果現在股價高於 109.6 元，報酬率會低於 10%

永續現金流折現

公式：

永續價值現值 = 未來現金流 ÷ (折現率 - 成長率)

實例：

  • 現金流 10 元，折現率 10%，成長率 2%
  • 永續現值 = 10 ÷ (10% - 2%) = 10 ÷ 0.08 = 125 元

用途：用於計算終值（預測期結束後到無限期的現金流）

DCF 適用時機

適用：

  • 公司現金流量穩定且可預測（如公用事業、穩定配息股）
  • 債券（報酬可預測）
  • 你對公司未來盈餘、成長性有相當掌握
  • 價值投資人評估公司內在價值

不適用：

  • 景氣循環股（收入變化大）
  • 私營公司（財務數據難取得）
  • 初創公司、早期公司（沒有足夠銷售記錄）
  • 科技公司、生技公司（長期支出大、現金流不可預測）

DCF 優缺點

優點：

  • 用具體數字計算內在價值，不受市場情緒影響
  • 考慮資金的時間價值
  • 可用來分析不同類型的公司
  • 幫助計算內部報酬率（IRR）比較投資機會
  • 比本益比、股價淨值比等方法更完整

缺點：

  • 對快速成長、不可預測的公司不適用
  • 需要大量財務資料，計算複雜
  • 依賴對未來現金流的預估，預測常常不準
  • 終值（FV）非常難以估計
  • WACC 折現率也是一個難以準確評估的數字

DCF vs 本益比估價法

本益比估價法的問題：

  • 假設未來每年盈餘都一樣（簡化）
  • 沒有考慮折現率（金錢時間價值）
  • 本益比倍數通常根據歷史經驗或同業比較，參雜市場情緒

DCF 更完整：

  • 可以精確預估每一年的現金流
  • 考慮機會成本（折現率）
  • 計算複雜但完整性更高

查理蒙格的觀點：
「我知道巴菲特都是這樣估價，但我從來沒有見他動手算過。」

解讀：真正好的投資機會，無論用哪種方法算，都會是好機會。大致估一下就知道是否值得投資，誤差只會讓它從「超好」變成「普通」，不會變成「爛」。

實際應用步驟

Step 1：預測自由現金流

  • 從財報抓取近幾年 FCF
  • 估算未來成長率
  • 預測未來 5~15 年的 FCF

Step 2：計算終值

  • 方法一：永續現金流公式
  • 方法二：保守估計，終值當零

Step 3：計算折現率

  • 用 WACC 公式
  • 或用簡化版：無風險利率 + 風險溢酬
  • 一般用 8%~12%

Step 4：折現加總

  • 把每一期 FCF 折現
  • 加上終值折現
  • 得到企業價值

Step 5：比較股價

  • DCF 估值 > 當前股價 → 可能低估
  • DCF 估值 < 當前股價 → 可能高估
  • 思考差異原因，評估買進時機

重要提醒

1. 沒有任何估價法是精準的：DCF 只是幫助釐清合理範圍，必然存在誤差
2. 預測不準是正常的：現金流預測本身就是猜測，不確定性越高越要保守
3. 了解公司比計算更重要：去讀財報、了解公司營運，比精算數字更有價值
4. DCF 幫助思考內在價值：讓你思考公司的現金流品質、時間價值、長期眼光

心得

DCF 是估值方法中最「硬派」的一種——它逼你思考公司到底值多少錢，而不是市場覺得它值多少錢。但計算複雜、預測困難，實務上用的人反而比本益比法少。

我覺得最關鍵的不是計算本身，而是它背後的邏輯：
1. 自由現金流才是真正的錢，盈餘可以是會計數字
2. 未來的錢不如今天的錢值錢，所以要折現
3. 投資要看長期，不是 1~2 年的短期投機

查理蒙格那句話很經典——巴菲特用 DCF 邏輯思考，但他從不算。真正好的公司，隨便估都知道便宜。DCF 的價值在於幫你建立估值的「框架」和「直覺」，而不是精算出來的數字。

對我來說，DCF 是一個「檢查點」：當我看到一家現金流穩定的公司，先用本益比、股價淨值比快速估一下，如果覺得便宜，再想想用 DCF 的角度會不會也便宜。不一定要真的算，但腦中要有這個框架。

不適合用 DCF 的公司：科技股、生技股、初創公司、景氣循環股。這些公司的現金流太難預測，算出來的數字意義不大。反而穩定配息的公用事業、成熟產業龍頭，DCF 會是很好的工具。

營業利益率

是什麼

營業利益率 = 營業利益 ÷ 營業收入 × 100%

營業利益 = 營業收入 − 營業成本 − 營業費用

每 1 元營收中，公司本業能賺多少錢。最能代表企業「本業獲利能力」的指標。

怎麼判斷

不是越高越好！ 營業利益率超過 100% 反而是異常（可能是費用為負、或利益暴增）。

判斷標準：

1. 穩定性：最近 6 個月營業利益率皆 > 15%，代表穩定
2. 趨勢：與公司自身過去相比是否成長
3. 同業比較：在同產業中是否較高

選股三方式

  • 最近 x 個月營業利益率皆 > x%（如：最近 6 個月 > 15%）
  • 同產業中營業利益率較高的公司
  • 搭配毛利率、淨利率一起看，避免單一指標失真

實例

異常案例：2021 Q3 晟德營益率 151.32%、吉祥全 114.39%，數值失真，需注意營運是否穩定。

正常案例：長榮 66.99%、陽明 65.5%、京城銀 90.77%，這些才是真正高獲利公司。

毛利率

是什麼

毛利率 =（營業收入 − 營業成本）÷ 營業收入 × 100%

毛利 = 營業收入 − 營業成本

代表公司產品的獲利能力，毛利率越高 = 成本控制越好、產品附加價值越高。

毛利率多少合理？

依產業差異很大：

產業平均毛利率
IC 設計40-50%+
光學鏡頭龍頭66%+
代工產業3-4%（毛三到四）

判斷方式：

1. 跟自己比：與過去幾年毛利率比較，是否成長
2. 跟同業比：在同產業中是否領先

毛利率變化因素

  • 價格優勢：品牌力、產品力帶來的附加價值
  • 成本優勢：規模經濟、議價能力
  • 外來因素：原物料、匯率波動

低毛利一定不好嗎？

不一定。鴻海毛利率低，但擁有：

  • 低成本營運模式
  • 交貨速度與品質
  • 專利技術領先

關鍵：低毛利公司需要更大營收規模才能創造相同淨利。

營收、毛利率、營業利益率 選股心法

實戰步驟

Step 1：每月 10 日看營收

  • 營收由衰退轉成長，或逐月成長 → 進入候選名單

Step 2：看最新一季財報

  • 毛利率較去年同期成長 ✓
  • 營業利益率較去年同期成長 ✓
  • 兩者都成長 → 優先列入

Step 3：確認趨勢

  • 查近期新聞、法人報告
  • 確認產業報價是否向上

Step 4：看籌碼面

  • 外資連買 3 個交易日以上
  • 代表有延續性買盤

Step 5：看估值

  • 本益比 < 10 倍算便宜
  • 殖利率 > 5% 算有吸引力

為什麼要看三個指標？

  • 營收成長：最即時，但不代表真的賺錢
  • 毛利率成長：產品獲利能力提升
  • 營業利益率成長：本業經營效率提升

如果營收成長但毛利率下降，可能是「殺價競爭」或「合併帶來的虛胖」。

心得

今天學的是「獲利能力三劍客」：營收、毛利率、營業利益率。

以前只會看 EPS，現在知道 EPS 是結果，毛利率和營業利益率才是「過程」。毛利率看的是產品好不好賺、營業利益率看的是公司會不會經營。

一個重要的觀念：高毛利不一定好，低毛利不一定差。重點是跟誰比——跟自己過去比、跟同業比。鴻海毛利只有個位數，照樣是世界級企業。相反，有些公司毛利率很高但營業利益率很低，代表費用失控、經營不善。

實戰上，每個月 10 日是關鍵時間點，要抓營收成長的公司，再進一步看毛利率和營業利益率是否同步成長。三者都成長才代表公司真的在變好。

一、現金股利 vs 股票股利

現金股利（配息）

是什麼：公司將盈餘以「現金」形式配發給股東，稱為「配息」，發放過程叫「除息」。

怎麼算：

  • 配息 1 元 = 每股可領 1 元現金
  • 公式：現金股利 = 配息金額 × 持有股數
  • 例：持有 1 張（1000 股），配息 0.67 元 → 可領 670 元

公司訊號：公司現金充足、短期無急迫投資需求，通常處於成熟期。

股票股利（配股）

是什麼：公司將盈餘以「股票」形式配發給股東，稱「配股」，發放過程叫「除權」。

怎麼算：

  • 台股面額 10 元，配股金額要除以 10
  • 公式：股票股利（股數）= 配股金額 ÷ 10 × 持有股數
  • 例：持有 1 張，配股 0.67 元 → 可領（0.67÷10）×1000 = 67 股

公司訊號：公司對現金有需求（擴廠、研發），處於成長期，保留現金擴張。

二、股利發放率（配息率）

是什麼

公司從盈餘中提撥股利的比例，反映公司的股利政策。

怎麼算

公式：股利發放率 = 每股股利 ÷ 每股盈餘（EPS）× 100%

實例：

  • 台泥 2020 年 EPS 4.22 元，現金股利 3.5 元
  • 配息率 = 3.5 ÷ 4.22 × 100% = 82.94%

判斷要點

1. 配息率超過 100%：代表動用過去保留盈餘，需留意是否為一次性
2. 配息率穩定：公司獲利穩定，可預測性高
3. 配息率波動大：獲利不穩定，或公司有特殊政策

三、股利來源三種類型

1. 保留盈餘（最常見）

公司歷年賺的錢，還沒分配出去的累積盈餘。

2. 法定盈餘公積

公司每年獲利需提撥 10% 作為法定盈餘公積。

  • 只能用超過資本額 25% 的部分配股
  • 公司沒虧損才能配發

3. 資本公積

來自資產重估獲利、股票溢價發行、出售固定資產溢價等。

  • 注意：配息來源非本業獲利，要小心公司是否只是為了維持股價

配股 vs 配息透露的訊號

情況可能解讀
主要配現金股利公司成熟穩定，現金充足，短期投資需求低
主要配股票股利公司成長期，需要現金擴張
配息率穩定 60-80%穩健配息公司，獲利可預測
配息率超過 100%可能動用老本，要查原因
公積配股非本業獲利，留意公司真實獲利能力

不同階段的股利政策

1. 成長期：配股為主、配息率低，保留現金擴張
2. 成熟期：配息為主、配息率高，現金充裕
3. 衰退期：可能維持高配息率，但要看是否吃老本

門檻

  • 單次股利（現金+股票）超過 2 萬元要扣 2.11%
  • 超過 1,000 萬以 1,000 萬計算

避開方法

1. 分散持股：同一家公司股利不超過 2 萬
2. 選季配息/半年配：單次領取金額較低
3. 投資 KY 股或 DR：海外所得不扣二代健保
4. ETF 選收益平準金配息：不計入股利所得

六、實際應用：選股時如何用股利政策篩選

穩健存股族

找什麼：

  • 配息率穩定（60-80%）
  • 主要發放現金股利
  • 連續多年配息不中斷
  • 配息來自保留盈餘（非公積）

避開什麼：

  • 配息率忽高忽低
  • 經常公積配股

一、現金股利 vs 股票股利

是什麼：

  • 現金股利（配息）：公司將盈餘以「現金」發放給股東，發放行為稱為「除息」
  • 股票股利（配股）：公司將盈餘以「股票」發放給股東，發放行為稱為「除權」

怎麼判斷公司發哪種：

公司階段常見股利政策原因
成長期股票股利為主需要現金擴廠、買設備，保留現金
成熟期現金股利為主投資需求減少，多餘現金回饋股東
衰退期可能不發股利獲利下降，保留現金度過難關

實例：

  • 台積電：成熟期公司，採取季配息（現金股利）
  • 成長型科技股：常發股票股利，保留現金研發擴充

二、股利計算方式

現金股利計算

公式：持有股數 × 每股配息金額 = 可領現金

實例：

  • 情境：持有台泥 1 張（1000 股），配息 3.5 元
  • 計算：1000 股 × 3.5 元 = 3,500 元現金

股票股利計算

公式：股票股利金額 ÷ 面額(10元) × 持有股數 = 可領股數

關鍵：配股 1 元 = 每 1 股可獲得 0.1 股股票

實例：

  • 情境：持有元大金 1 張，配股 0.2 元
  • 計算：0.2 ÷ 10 × 1000 = 20 股（多拿到 20 股）

除權後股價計算：

  • 公式：除權前股價 ÷ (1 + 股票股利 ÷ 10)
  • 實例：股價 29.25 元，配股 0.3 元
  • 除權後股價：29.25 ÷ (1 + 0.03) = 28.40 元

重要觀念：發放股票股利前後，總市值不變，只是股數變多、每股價值變少。

三、股利發放率

是什麼：公司從盈餘中提撥股利的比例

公式：股利發放率 = 每股股利 ÷ EPS × 100%

實例：

  • 台泥 2020 年 EPS 4.22 元，現金股利 3.5 元
  • 股利發放率：3.5 ÷ 4.22 × 100% = 82.94%

判斷標準：

發放率解讀
> 70%高發放率，公司成熟穩定，保留較少盈餘
40-70%中等發放率，平衡成長與股東回饋
< 40%低發放率，公司可能處於成長期或保守

注意：

  • 發放率過高可能影響公司成長動能
  • 發放率過低可能代表公司前景不佳或過度保守

四、股利來源判斷

三大來源：
1. 保留盈餘：公司當期或累積獲利，最正常的股利來源
2. 法定盈餘公積：公司每年獲利提撥 10%，累積部分可配股
3. 資本公積：資產重估獲利、股票溢價發行、出售資產溢價等

公積配股好嗎？

  • 公積配股 = 股利來源「不是」本業獲利
  • 可能代表：公司獲利不佳，為了維持股價才用公積配股
  • 判斷：查看公司近年獲利狀況，若連續虧損卻發股利，要小心

五、二代健保補充保費

門檻：單次股利超過 2 萬元（同公司、同發放日）

費率：2.11%

計算方式：

  • 現金股利：直接以金額計算
  • 股票股利：分配股數 × 10 元面額

實例：

  • 情境：持有 A 公司 10 張，配股 5 元
  • 股票股利：5 ÷ 10 × 10000 = 5000 股
  • 課稅金額：5000 × 10 = 50,000 元
  • 二代健保費：50,000 × 2.11% = 1,055 元

避免方式：
1. 分散持股，單一公司股利控制在 2 萬元以下
2. 選擇季配息或半年配的股票（台積電、0050 等）
3. 投資 DR/KY 股（海外所得，不課二代健保）

六、從股利政策判斷公司體質

好的股利政策特徵：

  • 發放率穩定（不大幅波動）
  • 股利來源主要為保留盈餘
  • 現金股利為主（成熟公司）
  • 配合公司成長階段調整

警示訊號：

  • 連續虧損卻發股利（可能是公積配股）
  • 發放率突然大幅提高（可能為了撐股價）
  • 只發股票股利不發現金（公司缺現金）
  • 股利發放率超過 100%（動用老本）

心得

股利政策是判斷公司成熟度的重要指標。現金股利代表公司有穩定現金流，股票股利則代表公司需要現金擴充。投資時不能只看殖利率高低，要看股利來源是否健康、發放率是否穩定。

特別要注意公積配股的情況，這代表公司可能用「老本」發股利，而非本業賺錢，這種公司長期投資風險較高。

另外，二代健保補充保費是台股投資的隱形成本，分散持股或選擇季配息可以避免。這也是為什麼越來越多 ETF 強調「收益平準金」作為配息來源，因為不會被課二代健保。

核心知識

殖利率計算方式

是什麼：股息與股價的比率，反映持有股票可獲得的現金回報。

計算公式：

  • 現金股息殖利率（%）= 現金股息 ÷ 股價
  • 總殖利率（%）= 現金股息殖利率 + 股票股利殖利率

實例：

  • 情境：台積電（2330）去年配息 11 元，年均價 516 元
  • 計算：11 ÷ 516 = 2.13%
  • 意義：投資 51.6 萬買一張，每年可領 1.1 萬現金股息

台股參考值：

  • 台股平均殖利率：約 4%
  • 理想目標：5%~6% 穩定殖利率

高殖利率三大陷阱

陷阱一：股利發放不穩定

是什麼：今年配很多、明年配很少，甚至不配。

怎麼判斷：
1. 查過去 5 年配息紀錄
2. 計算配息變異係數（標準差÷平均）
3. 變異係數 > 30% 代表不穩定

實例：

  • 情境：營建股今年殖利率 15%
  • 問題：建案完工一次認列，明年可能沒獲利
  • 判斷：避開「一次性暴利」造成的假高殖利率

陷阱二：賺了股息賠了價差

是什麼：殖利率變高是因為股價下跌，不是股息增加。

怎麼判斷：
1. 比較近 3 年殖利率變化
2. 檢查同期股價走勢
3. 若殖利率變高、股價變低 → 價值陷阱警訊

實例：

  • 情境：某股票去年殖利率 3%，今年變 8%
  • 檢查：股價從 100 跌到 40（殖利率變高是因為股價跌）
  • 判斷：公司可能基本面惡化，股價下跌不是買點

陷阱三：配息來自一次性收益

是什麼：賣資產、賣子公司、處分土地等一次性獲利配息。

怎麼判斷：
1. 看財報「營業外收支」項目
2. 檢查是否有大額處分資產
3. 看配息是否來自本業獲利

實例：

  • 情境：某公司今年配 10 元，但 EPS 只有 3 元
  • 問題：配息遠高於 EPS，一定有一次性收益
  • 判斷：明年配息可能腰斬甚至歸零

四大高殖利率選股策略

策略一：高現金股利連續發放股

適合誰：追求穩定現金流的投資人（存股族）

核心邏輯：公司願意且有能力持續分享獲利

篩選條件：

條件參數意義
現金股利近 5 年平均 > 3 元有穩定盈餘分配能力
連發年數連續 5 年發放現金股利配息穩定
股利成長連續 3 年增加營運成長、管理層有信心
發放率5 年內 3 年 > 70%願意與股東分享獲利

實例篩選：
1. 開啟選股工具，設「連續 5 年配息」
2. 篩選「近 5 年平均現金股利 > 3 元」
3. 再過濾「股利連續 3 年成長」
4. 結果可能：中華電、台塑化、統一超等成熟企業

策略二：高殖利率快速填息股

適合誰：退休族、以股利為主要收入者

核心邏輯：配息後股價快速回到除息前水準，避免賺股息賠價差

篩選條件：

條件參數意義
殖利率近 3 年平均 > 5%高於市場平均
填息率近 3 年填息成功率 100%每年都完成填息
填息天數平均 < 60 天填息速度快

實例：

  • 情境：某股票近 3 年殖利率 6%，每年都填息，平均 45 天
  • 操作：除息前買入，領股息後等待填息
  • 風險：過去填息不代表未來一定填息

策略三：價值低估高殖利率股

適合誰：價值型投資人

核心邏輯：高殖利率 + 低本益比 + 低股價淨值比 = 被低估的好股票

篩選條件：

條件參數意義
殖利率近 3 年平均 > 5%現金流優於定存
股價淨值比< 1 倍或 < 1.5 倍股價低於淨值或接近
本益比< 15 倍股價相對便宜

注意事項：

  • 股價淨值比 < 1 可能是公司營運狀況差
  • 本益比低可能是市場看壞
  • 需搭配基本面分析，確認不是價值陷阱

策略四：具成長動能的高股息股

適合誰：能承受較大波動的投資人

核心邏輯：避開成熟期衰退股，選擇「配息成長 + EPS 成長」的標的

篩選條件：

條件參數意義
股利成長連續 3 年增加願意分享成長果實
EPS 年增率> 10%獲利持續成長
營業利益成長率近 3 年皆 > 5%本業獲利成長

風險：成長股波動較大，除息後可能不填息

高殖利率股票進出場策略

進場時機

方法一：估值合理時進場

  • 殖利率高於市場/產業平均時進場
  • 注意：確認不是基本面惡化造成的假高殖利率

方法二：除息後走勢回穩時買入

  • 除息後股價通常會有賣壓
  • 等股價穩定後再進場，減少稅務損失

方法三：市場低迷時分批進場

  • 市場恐慌時，殖利率超過歷史平均的標的優先
  • 分批投入，降低買在相對高點風險

方法四：定期定額買入

  • 每月固定金額投入高殖利率股票或 ETF
  • 平滑成本，減少擇時壓力

方法五：股利政策變化前提前布局

  • 法說會、財報公告後，推測股利政策
  • 在股利宣布前進場

出場時機

方法一：殖利率下降至不具吸引力

  • 當殖利率低於市場平均 → 逐步賣出
  • 公司調降股利 → 考慮出場

方法二：基本面惡化

一、現金股利 vs 股票股利

是什麼：

  • 現金股利：公司把盈餘以「現金」發放給股東，又稱「配息」或「除息」
  • 股票股利：公司把盈餘以「股票」配發給股東，又稱「配股」或「除權」

怎麼做：
1. 查公司公告的股利政策（公開資訊觀測站）
2. 判斷是現金股利還是股票股利
3. 計算可領金額

  • 股票股利：公司成長期、需要現金擴充

二、股利計算方式

現金股利公式：

現金股利 = 每股配息金額 × 持有股數

股票股利公式：

股票股利（股數）= 配股金額 ÷ 股票面額 × 持有股數

（註：台股面額通常為10元，但2014年後公司可自訂面額）

實例：

  • 台泥（1101）配發現金股利3.5元，EPS為4.22元
  • 股利發放率 = 3.5 ÷ 4.22 × 100% = 82.94%

怎麼做：
1. 到公開資訊觀測站查詢公司股利分配
2. 用公式計算可領金額
3. 注意：股票股利會讓股數增加，現金股利直接入帳

三、股利發放率

是什麼：公司從盈餘中提撥多少比例作為股利

公式：

股利發放率 = 每股股利 ÷ EPS × 100%

怎麼判斷：

發放率範圍可能意義
80%以上公司成熟，投資需求低
50-80%平衡配息與成長
50%以下公司有成長需求，保留盈餘再投資

實例：

  • 台泥2020年：EPS 4.22元，現金股利3.5元，發放率82.94%
  • 台積電：發放率約70-80%，季配息

注意：美股只計算現金股利，不計入股票股利

四、股利來源（品質判斷關鍵）

三種來源：
1. 保留盈餘：公司歷年賺的錢未分配部分（最好）
2. 法定盈餘公積：每年獲利提撥10%（次佳）
3. 資本公積：資產重估獲利、股票溢價發行（要小心）

怎麼做：
1. 查公司股利來源（公開資訊觀測站）
2. 看是否來自「保留盈餘」或「公積」

實例判斷：

  • 若公司用「資本公積」配股 → 可能是獲利不佳，為維持股價才配發
  • 要追問：為什麼本業賺不到錢還要配息？

適用時機：

  • 存股族要特別注意：穩定配息的公司，股利來源是否健康

五、公司生命週期與股利政策

怎麼做：根據公司所處階段，判斷股利政策是否合理

階段現金需求股利政策
草創期高不發股利
成長期高股票股利為主
成熟期低現金股利為主
衰退期不一定視情況

判斷要點：

  • 成長期公司發現金股利 → 可能缺乏投資機會，要注意
  • 成熟期公司發股票股利 → 可能現金不足，要檢查現金流

六、二代健保補充保費

是什麼：單次股利超過2萬元要課2.11%補充保費

怎麼做：
1. 計算單次股利是否超過2萬
2. 若超過，扣2.11%補充保費

實例：

  • 持有5張富邦金，每股配息4.25元
  • 股利 = 4.25 × 5,000 = 21,250元
  • 超過2萬，扣二代健保 = 21,250 × 2.11% = 448元
  • 匯款手續費 = 10元
  • 實際入帳 = 21,250 - 448 - 10 = 20,792元

避免方式：

  • 選擇季配息或半年配息的股票（單次金額較低）
  • 分散持股到不同家人帳戶
  • 投資ADR、KY股（海外所得不課二代健保）

七、配息頻率的考量

是什麼：公司一年配息幾次

三種方式：
1. 年配息：一年一次（最常見）
2. 半年配息：一年兩次（如0050）
3. 季配息：一季一次（如台積電）

季配息的好處：
1. 避免單次超過2萬元，省二代健保
2. 穩定配息吸引基金投資
3. 融券回補支撐股價（除權息時強制回補）

怎麼選：

  • 存股族：季配息可分散二代健保成本
  • 短線交易：配息頻率影響不大

3. 公司是否處於成熟期
4. 現金股利 vs 股票股利比例

  • 公司用資本公積配股 → 可能獲利不佳
