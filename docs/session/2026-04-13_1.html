核心知識

一、標準樞紐點 Pivot Point

是什麼：由 J.W. Wilder（RSI 開發者）設計，用前一天的高價、低價、收盤價計算出 7 條線（1 個 PP + 3 條阻力 + 3 條支撐），作為當天的支撐壓力參考。

計算方式：

  • PP（樞紐點） =（昨日高價 + 昨日低價 + 昨日收盤價）÷ 3
  • R1 = PP × 2 − 昨日低價
  • R2 = PP +（昨日高價 − 昨日低價）
  • R3（HBOP） = R1 +（昨日高價 − 昨日低價）
  • S1 = PP × 2 − 昨日高價
  • S2 = PP −（昨日高價 − 昨日低價）
  • S3（LBOP） = S1 −（昨日高價 − 昨日低價）

怎麼做：
1. 每天開盤前用前一日數據計算所有水平線
2. 價格在 S1～R1 之間震盪 → 區間交易：S1 買、R1 賣
3. 突破 R1 → 目標看 R2；跌破 S1 → 目標看 S2
4. 突破 R3（HBOP）→ 趨勢啟動，轉為順勢做多
5. 跌破 S3（LBOP）→ 趨勢啟動，轉為順勢做空

實例：

  • 某股昨日高價 105、低價 98、收盤 102
  • PP = (105 + 98 + 102) ÷ 3 = 101.67
  • R1 = 101.67 × 2 − 98 = 105.33
  • S1 = 101.67 × 2 − 105 = 98.33
  • 當天價格若在 98.33～105.33 之間震盪 → 區間操作
  • 若突破 105.33 → 看向 R2 = 101.67 + 7 = 108.67

適用時機：

  • 日線 Pivot → 短期當沖、日內交易
  • 週線 Pivot → 波段交易、中長期趨勢
  • 月線 Pivot → 中長期趨勢判斷
  • ⚠️ 每條線僅限當天/當週/當月有效，隔期歸零重算

二、樞紐點雙模式交易系統

是什麼：Pivot Point 系統包含兩種模式 — Reaction（逆勢）模式和 Trend（順勢）模式。

逆勢模式（區間交易）：
1. 價格到達 S1 → 買進，停利目標 R1
2. 價格到達 R1 → 賣出，停利目標 S1
3. 若突破 S1 或 R1，可在 S2 或 R2 採取相同策略

順勢模式（突破交易）：
1. 價格突破 HBOP（R3）→ 順勢做多
2. 價格跌破 LBOP（S3）→ 順勢做空
3. 這是趨勢啟動的訊號，要果斷跟單

關鍵判斷：

  • 價格在 PP 以上 → 市場偏多
  • 價格在 PP 以下 → 市場偏空
  • 離 PP 愈遠的 Level（R3、S3），拉回力量愈強 → 逆勢撈頂撈底的區域

適用時機：

  • 震盪盤 → 逆勢模式在 S1/R1 區間操作
  • 突破盤 → 順勢模式在 HBOP/LBOP 追單
  • 不要在趨勢盤用逆勢模式，也不要在震盪盤用順勢模式

三、卡瑪利拉樞紐點 Camarilla Pivot Points

是什麼：由 Nick Stott 在 1989 年開發，是標準樞紐點的改良版，特色是產生更密集的支撐阻力線，特別適合日內交易和剝頭皮（Scalping）。

計算方式：

  • R4 = Close + (High − Low) × 1.1 ÷ 2
  • R3 = Close + (High − Low) × 1.1 ÷ 4
  • R2 = Close + (High − Low) × 1.1 ÷ 6
  • R1 = Close + (High − Low) × 1.1 ÷ 12
  • S1 = Close − (High − Low) × 1.1 ÷ 12
  • S2 = Close − (High − Low) × 1.1 ÷ 6
  • S3 = Close − (High − Low) × 1.1 ÷ 4
  • S4 = Close − (High − Low) × 1.1 ÷ 2

怎麼做：
1. R3 / S3 反轉策略：價格觸及 R3 → 找賣出訊號；價格觸及 S3 → 找買進訊號
2. 突破策略：價格突破 R3 → 目標 R4；突破 S3 → 目標 S4
3. 假突破辨識：價格短暫突破 R3/S3 但迅速拉回 → 反向操作
4. 剝頭皮：在 R1/R2、S1/S2 之間小幅度來回交易

實例：

  • 某股昨日高價 110、低價 100、收盤 105
  • Range = 110 − 100 = 10
  • R3 = 105 + 10 × 1.1 ÷ 4 = 105 + 2.75 = 107.75
  • S3 = 105 − 10 × 1.1 ÷ 4 = 105 − 2.75 = 102.25
  • 當天若漲到 107.75 附近 → 觀察是否反轉，搭配 K 線型態確認
  • 若直接突破 107.75 → 目標看 R4 = 105 + 10 × 1.1 ÷ 2 = 110.5

適用時機：

  • 日內交易（最適合）、剝頭皮交易
  • 波動大的標的（原油、外匯主流貨幣對）
  • ⚠️ 不適合長期波段，因為水平線僅限當天有效

四、樞紐點五種算法比較

算法特點最適用場景
標準算法最常見，(H+L+C)÷3通用，所有市場
斐波那契算法加入 Fib 比率（0.382、0.618）趨勢追蹤確認
伍迪算法加入開盤價，更重視近期價格短線交易
卡瑪利拉算法8 條線更密集，乘以 1.1 係數當沖/剝頭皮
德馬克算法根據開盤收盤關係選擇公式預測當日高低點

怎麼做：
1. 新手先用標準算法入門
2. 日內交易者切換到卡瑪利拉 → 更精準的進出場點
3. 波段交易者可用斐波那契算法 → 確認趨勢中的 Fib 回調位
4. 多種算法同時顯示 → 重疊區域支撐壓力更強

適用時機：

  • 台股當沖 → 卡瑪利拉算法（線更密集）
  • 期貨波段 → 標準算法 + 斐波那契算法
  • 外匯短線 → 伍迪算法（加入開盤價）

五、樞紐點搭配其他指標實戰

搭配 RSI：
1. 價格到達 R3 且 RSI 超買（>70）→ 強烈賣出訊號
2. 價格到達 S3 且 RSI 超賣（<30）→ 強烈買進訊號
3. 價格在 PP 附近且 RSI 中性 → 等待方向明確

搭配布林通道：
1. 價格同時到達 R3 且觸及布林上軌 → 反轉機率極高
2. 價格突破 R3 且突破布林上軌 → 強趨勢確認
3. 卡瑪利拉 S3/R3 與布林帶重疊 → 更可靠的支撐壓力

搭配均線：
1. 價格在均線上方 + 突破 PP → 做多勝率更高
2. 價格在均線下方 + 跌破 PP → 做空勝率更高
3. 均線方向確認趨勢，樞紐點確認進出場位

實例：

  • 台積電（2330）昨日高 980、低 950、收 970
  • PP = (980+950+970)÷3 = 966.7
  • R1 = 966.7×2 − 950 = 983.3
  • S1 = 966.7×2 − 980 = 953.3
  • 當天若 5MA 向上 + 價格在 966.7 上方 → 偏多操作
  • RSI 到 70 且價格到 983.3 → 考慮部分出場
  • 停損設在 S1 953.3 下方

心得

樞紐點這東西妙在哪？妙在它是「客觀的」。不像均線有參數可以調、不像 RSI 有人看 14 有人看 9，Pivot Point 的公式是死的——同一個昨日數據，所有人算出來的線一模一樣。這意味著什麼？意味著市場上用這工具的人，都在看同一個價位。人多看的地方，支撐壓力自然就強。

但問題也在这里：正因為大家都知道，所以 R1/S1 的支撐壓力常常被「假突破」——主力知道散戶在哪裡設停損，專門去掃。所以單獨用 Pivot Point 是不夠的，一定要搭配其他指標確認。

卡瑪利拉版本更有意思，乘以 1.1 的係數讓線更密集，對當沖族來說簡直是量身定做。R3/S3 是關鍵——碰到就反轉，突破了就跟單。但老實說，這個「1.1」係數到底是怎麼來的？Nick Stott 從沒公開過推導過程，這讓我有點不爽。不過回測數據是有效的，那就先用吧，反正市場不講道理，只講結果。

最後一個體悟：Pivot Point 每天重算，這是它的優點也是缺點。優點是自動適應最新價格結構，缺點是你不能用昨天的線看明天的走勢。搭配週線和月線 Pivot 一起用，才能看更大格局。
