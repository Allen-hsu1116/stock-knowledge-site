一、Piotroski F-Score 財務健康評分

1. F-Score 是什麼？

是什麼：由芝加哥大學教授 Joseph Piotroski 於 2002 年提出的選股指標，用 9 個基本面條件幫股票打分數，每符合一項得 1 分，總分 0~9 分。分數越高代表公司基本面越健康。

白話解釋：F-Score 就像公司的健康檢查表，9 個項目各打勾或打叉，勾越多代表公司體質越好。Piotroski 的研究發現，在低本益比股票中，F-Score 高的股票報酬率明顯優於 F-Score 低的。

2. F-Score 九大指標詳解

獲利性（4 項）：

編號指標條件分數
F1資產報酬率（ROA）ROA > 0符合得 1 分
F2ROA 變化今年 ROA > 去年 ROA符合得 1 分
F3營業現金流營業現金流 > 0符合得 1 分
F4營業現金流 vs 淨利營業現金流 > 稅後淨利符合得 1 分

安全性（3 項）：

編號指標條件分數
F5長期負債變化今年長期負債 < 去年符合得 1 分
F6流動比率變化今年流動比率 > 去年符合得 1 分
F7股份稀釋今年沒有發行新股符合得 1 分

成長性（2 項）：

編號指標條件分數
F8毛利率變化今年毛利率 > 去年符合得 1 分
F9資產周轉率變化今年資產周轉率 > 去年符合得 1 分

3. 九大指標的投資意義

獲利性四項解讀：

  • F1（ROA > 0）：最基本的——公司有沒有在賺錢？ROA 為正代表公司用資產創造了利潤
  • F2（ROA 成長）：不只要賺錢，還要比去年賺更多——獲利能力有沒有進步
  • F3（營業現金流 > 0）：賺的錢有沒有真的進來？帳面有盈餘但現金流為負 = 紅旗
  • F4（營業現金流 > 淨利）：盈餘品質的終極檢驗——現金流大於會計利潤，代表利潤有真實現金支撐

安全性三項解讀：

  • F5（長期負債減少）：公司有沒有在還債？負債減少 = 財務結構改善
  • F6（流動比率增加）：短期償債能力有沒有改善？流動比率上升 = 更安全
  • F7（沒有發行新股）：公司有沒有稀釋股東權益？增資會攤薄 EPS，不增資代表公司用自有資金就夠

成長性兩項解讀：

  • F8（毛利率提升）：公司定價能力或成本控制有沒有進步？毛利率上升 = 競爭力增強
  • F9（資產周轉率提升）：公司用資產創造營收的效率有沒有提高？周轉率上升 = 經營效率改善

4. F-Score 判斷標準

F-Score判斷意義
8~9 分優質股基本面全面健康，值得深入研究
6~7 分良好大部分指標過關，少數需要關注
4~5 分普通有改善空間，需搭配其他指標
2~3 分偏弱多數指標不及格，投資風險較高
0~1 分危險基本面嚴重惡化，建議避開

5. F-Score 實戰五步驟

怎麼做：

1. 從低本益比或高股息股中篩選：F-Score 最大的威力在於從便宜的股票中挑出基本面好的。如果只在高價成長股中篩選，F-Score 效果不明顯
2. 計算 F-Score：逐項檢查 9 個指標，計算總分
3. 篩選 F-Score ≥ 8 的股票：8 分以上才是 Piotroski 認為值得投資的
4. 看趨勢：比較今年和去年的 F-Score，上升趨勢比單一年份的高分更有意義
5. 搭配其他指標：F-Score 不單獨使用，搭配 ROIC、本益比、股價淨值比一起看

適用時機：

  • ✅ 價值投資篩選：從低 PB 或低 PE 股中挑出基本面好的
  • ✅ 避雷：F-Score 持續下降的公司，基本面正在惡化
  • ❌ 成長股篩選：高成長股的 F-Score 可能因為增資（F7=0）而偏低
  • ❌ 金融股：F-Score 原始設計針對製造業，金融業的負債結構不同

查詢工具：

  • 美股：Gurufocus（直接提供 F-Score）
  • 台股：財報狗（付費功能）、自行計算
  • 自行計算：從 Goodinfo 取得財報數據，逐項打分

6. F-Score 的限制

限制一：不適合金融業和景氣循環股

  • 金融業的負債結構不同（存款就是負債），F5（長期負債減少）可能誤判
  • 景氣循環股在谷底時 F-Score 很低，但可能正是買點

限制二：只看方向不看幅度

  • 毛利率從 60% 降到 59.9%，F-Score 給 0 分
  • 毛利率從 5% 升到 5.1%，F-Score 給 1 分
  • 但 0.1% 的變動可能只是雜訊

限制三：沒有考慮產業差異

  • 不同產業的毛利率、周轉率、負債比率天生就不同
  • 跨產業比較 F-Score 可能失真

限制四：回測效果有限

  • 市場先生回測台股，F-Score 選股長期略優於大盤，但不顯著
  • 單獨使用效果不明顯，建議搭配其他指標

二、Altman Z-Score 破產預測模型

1. Z-Score 是什麼？

是什麼：由紐約大學教授 Edward Altman 於 1968 年提出的財務預警模型，用 5 個財務比率的加權計算，判斷公司在 2 年內破產的可能性。準確度介於 72%~94%。

白話解釋：Z-Score 就像公司的「破產預警器」。分數越低，破產風險越高。Altman 研究了數百家破產和未破產的公司，找出最能預測破產的 5 個財務比率，加上不同權重，算出一個綜合分數。

2. Z-Score 計算公式

公開上市公司（製造業）公式：

Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 0.999×X5

X1 = 營運資金 ÷ 總資產 =（流動資產 − 流動負債）÷ 總資產
X2 = 保留盈餘 ÷ 總資產
X3 = EBIT ÷ 總資產
X4 = 股東權益市值 ÷ 總負債
X5 = 營業收入 ÷ 總資產

私營公司（非製造業）公式：

Z'' = 6.56×X1 + 3.26×X2 + 6.72×X3 + 1.05×X4

（省略 X5，不使用營業收入/總資產）

3. Z-Score 判斷標準

公開上市公司：

Z-Score 範圍判斷意義
Z > 2.99安全區破產風險極低
1.8 < Z < 2.99灰色區無法明確判斷，需進一步分析
Z < 1.8危險區破產風險很高

私營公司：

Z-Score 範圍判斷
Z > 2.9安全區
1.23 < Z < 2.9灰色區
Z < 1.23危險區

4. 五大變數的投資意義

X1：營運資金 ÷ 總資產（流動性）

  • 反映公司短期償債能力和資產變現能力
  • X1 高 = 流動資產充足，短期沒有周轉問題
  • X1 低或為負 = 營運資金不足，可能面臨償債壓力
  • 權重 1.2：流動性在破產預測中重要但不是最關鍵

X2：保留盈餘 ÷ 總資產（累積獲利能力）

  • 反映公司歷年累積的利潤
  • X2 高 = 公司長期以來持續獲利，累積了大量保留盈餘
  • X2 低或為負 = 公司長期虧損或剛成立不久
  • 權重 1.4：累積獲利能力比流動性稍重要
  • 新創公司 X2 通常偏低（保留盈餘少），不代表一定會破產

X3：EBIT ÷ 總資產（獲利能力）

  • 反映公司用資產創造利潤的能力
  • X3 高 = 資產效率高，本業獲利能力強
  • X3 低或為負 = 本業虧損，靠資產吃老本
  • 權重 3.3：最高權重！ 獲利能力是預測破產最關鍵的因素

X4：股東權益市值 ÷ 總負債（財務結構）

  • 用市值而非帳面值衡量股東權益
  • X4 高 = 市場對公司有信心，負債相對低
  • X4 低 = 市值遠低於負債，市場已經在定價破產風險
  • 權重 0.6：市場信心有一定參考價值，但不如獲利能力重要

X5：營業收入 ÷ 總資產（資產周轉率）

  • 反映公司用資產創造營收的效率
  • X5 高 = 資產周轉快，營運效率高
  • X5 低 = 資產閒置或過度投資
  • 權重 0.999：周轉率重要但不是破產預測的最關鍵因素

5. Z-Score 實戰計算實例

情境：某製造業上市公司財務數據

  • 流動資產：150 億，流動負債：100 億
  • 總資產：500 億
  • 保留盈餘：80 億
  • EBIT：60 億
  • 股東權益市值：300 億
  • 總負債：200 億
  • 營業收入：800 億

計算：

  • X1 =（150 − 100）÷ 500 = 0.10
  • X2 = 80 ÷ 500 = 0.16
  • X3 = 60 ÷ 500 = 0.12
  • X4 = 300 ÷ 200 = 1.50
  • X5 = 800 ÷ 500 = 1.60
  • Z = 1.2×0.10 + 1.4×0.16 + 3.3×0.12 + 0.6×1.50 + 0.999×1.60
  • Z = 0.12 + 0.224 + 0.396 + 0.90 + 1.60
  • Z = 3.24 → 安全區

另一個情境：某財務困難公司

  • 流動資產：80 億，流動負債：120 億
  • 總資產：400 億
  • 保留盈餘：-10 億（累積虧損）
  • EBIT：5 億
  • 股東權益市值：40 億
  • 總負債：360 億
  • 營業收入：200 億

計算：

  • X1 =（80 − 120）÷ 400 = −0.10
  • X2 = −10 ÷ 400 = −0.025
  • X3 = 5 ÷ 400 = 0.0125
  • X4 = 40 ÷ 360 = 0.111
  • X5 = 200 ÷ 400 = 0.50
  • Z = 1.2×(−0.10) + 1.4×(−0.025) + 3.3×0.0125 + 0.6×0.111 + 0.999×0.50
  • Z = −0.12 + (−0.035) + 0.041 + 0.067 + 0.50
  • Z = 0.453 → 危險區，破產風險極高

6. Z-Score 的限制

限制一：原始公式適用製造業

  • 服務業、金融業的資產結構不同
  • 非製造業應使用 Z'' 公式（省略 X5）
  • 金融業完全不適用（負債定義不同）

限制二：權重可能不適用台股

  • 原始權重是根據 1960 年代美國製造業數據推導
  • 50 多年前的美國製造業跟現在的台股差異很大
  • 但作為方向性參考仍有價值

限制三：不考慮景氣循環

  • 景氣循環股在谷底時 Z-Score 可能很低
  • 但景氣反轉後 Z-Score 會快速回升
  • 低 Z-Score 不一定等於會破產，可能只是暫時困境

限制四：使用會計數據，更新較慢

  • 財報每季才更新一次
  • 公司可能在一季內就從安全區滑入危險區
  • Z-Score 有時間落差

限制五：X4 使用市值

  • 股價大幅波動會直接影響 Z-Score
  • 空頭市場時所有公司 X4 都下降，Z-Score 偏低
  • 可能過度反應市場情緒

7. F-Score 與 Z-Score 搭配使用

組合邏輯：

F-ScoreZ-Score判斷
高高✅ 基本面健康且財務安全，最佳選擇
高低⚠️ 基本面好但財務風險高，需看 Z-Score 低的原因
低高⚠️ 財務安全但基本面惡化，可能是景氣循環谷底
低低🚨 基本面差且財務風險高，避開

最佳使用方式：
1. 先用 Z-Score 排雷：Z < 1.8 的公司直接跳過
2. 再用 F-Score 篩選：Z > 2.99 且 F-Score ≥ 8 的公司優先考慮
3. 搭配 ROIC > WACC、本益比 < 合理價等條件縮小範圍

查詢工具：

  • 美股：Gurufocus（同時提供 F-Score 和 Z-Score）
  • 台股：Goodinfo + 自行計算
  • 台股 Z-Score：也可參考 CMoney、財報狗

心得

F-Score 和 Z-Score 這兩個指標，一個看「健康度」、一個看「生存風險」，剛好互補。

F-Score 最實用的地方是它把 9 個基本面條件簡化成 0~9 分的快篩。不需要計算複雜的財務模型，只要逐項檢查就能打分。但它的限制也很明顯：只看方向不看幅度，毛利率從 60% 掉到 59.9% 和從 5% 升到 5.1% 都是 0 分 vs 1 分，但經濟意義天差地遠。所以 F-Score 更適合當「初步篩選器」，通過之後還是要深入分析每個項目的變化幅度。

Z-Score 的核心洞察是：破產預測最重要的變數是獲利能力（X3 權重 3.3 最高），而不是流動性或周轉率。這跟直覺相反——大部分人以為公司破產是因為「周轉不靈」，但 Altman 的研究告訴我們，根本原因是「不賺錢」。一家公司如果 EBIT 持續為正，即使流動比率偏低，也還有時間調整；但如果 EBIT 為負，就算流動比率很高，也是在燒錢等死。

兩個指標搭配使用的邏輯很清晰：先用 Z-Score 排雷（Z < 1.8 直接跳過），再用 F-Score 找好公司（F ≥ 8 優先考慮）。這比單用任何一個指標都更完整。不過要記住，這兩個模型都是 1960-2000 年代的美股數據，套用到台股需要調整權重，至少方向性判斷還是有效的。
