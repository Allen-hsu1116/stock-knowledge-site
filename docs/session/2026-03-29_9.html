一、類股輪動核心觀念

什麼是類股輪動？

定義：類股輪動 = 市場上不同產業輪流成為主流的現象。

  • A產業漲到一定程度 → 換B產業漲
  • 又稱「權值股輪漲」，因為每個類股都有領頭羊（權值股）

為什麼會輪動？
1. 資金效率運作：主力（外資、法人）資金有限，無法同時拉抬所有股票
2. 題材與景氣循環：不同產業在不同時期有各自的利多題材
3. 季節性因素：如電子股旺季在Q3、寒假推食品觀光類股

二、類股輪動實戰五步驟

步驟1：觀察當日強勢類股排行

做法：
1. 開盤後查看類股漲跌幅排行
2. 找出當日漲幅前5名的類股
3. 記錄：漲幅%、成交量、是否上漲量增

判斷重點：

  • 漲幅要真實（不是跌深反彈）
  • 成交量必須放大（有主力資金進駐）

步驟2：確認類股線型架構

做法：
1. 點選強勢類股看線型圖
2. 判斷是否為「多頭排列」
3. 與大盤比較相對位置

關鍵條件：

條件多頭架構空頭架構
均線排列短>中>長短<中<長
趨勢方向向上向下
是否可買✅ 是❌ 只是跌深反彈

實例：大盤已多頭排列，某類股還是空頭排列 → 當日漲幅再強也只是一日行情。

步驟3：篩選類股中強勢個股

做法：
1. 在強勢類股中找當日漲幅最大的個股
2. 檢查技術面：均線多頭排列
3. 檢查基本面：營收成長、毛利率成長
4. 檢查法人面：外資、投信是否在買
5. 檢查籌碼面：大戶持股比例、籌碼集中度

剔除標準：

  • 營收沒成長 → 剔除
  • 無法人主力關注 → 剔除
  • 籌碼分散 → 剔除

留下的股票：強勢類股中最強勢的股票 = 主流中的主流。

輪動結束訊號：
1. 主流類股漲幅已大（超過30-50%）
2. 散戶大量追進（融資大增）
3. 法人開始賣超該類股

停損位置：

三、相對強度指標RS實戰

RS是什麼？

定義：Relative Strength，相對強度。表達個股相對於大盤的強弱程度。

公式：RS = 個股價格 / 大盤指數

數值意義：

  • RS上升 = 個股比大盤強（漲得多或跌得少）
  • RS下降 = 個股比大盤弱
  • 年RS 95 = 過去一年表現為全市場前5%

為什麼重要？

  • 找出真正強勢股（漲時比大盤凶、跌時比大盤抗）
  • 不被假象迷惑（大盤跌5%，個股跌2%看起來跌，實際上RS是上升的）

RS計算方式比較

方式公式適用情境
傳統RS個股價格 / 大盤價格長期趨勢分析
RSS個股漲幅% - 大盤漲幅%短期動態分析

實例計算：

  • 台積電 +1.9%，大盤 +0.4%
  • RSS = 1.9% - 0.4% = +1.5%（比大盤強1.5%）
  • 累積RSS上升 + 成交量放大 → 強勢股訊號

RS三大應用

1. 找帶頭領漲股

多頭市場：

  • 找出RS最高的一群股票
  • 在突破點進場

反彈或回升：

  • 指數由反彈轉回升時
  • 領導股會以最快速度脫離底部
  • 它們早已領先打好底部，RS逐漸提高

Mark Minervini實證：
> 「我最終的績效來自於RS 90以上的股票，買進的股票RS最低不小於70。」
>
> 研究顯示：最強飆股在突破第一個整理區域時，平均RS評分是87分。

2. 挑選放空標的

注意：不要隨便空高RS的股票！

  • RS很高 = 有實際強度
  • 未見顯著頭部前，空它的風險大

可放的時機：

  • 明確頭部型態出現
  • 最後一波突破失敗
  • RS開始從高檔下滑

3. 研究市場現況

觀察重點：
1. 高RS股突破是否順利？突破的檔數多嗎？
2. 有多少大型股在高RS名單裡？
3. 強度最高的一群股票走勢是否集體變差？

判斷：高RS股集體變差 → 大盤可能開始轉壞。

四、RS指標實戰篩選流程

類股RS計算

公式：類股指數收盤價 / 加權指數收盤價

XQ腳本：

value1 = GetSymbolField("TSE.TW", "收盤價", "D");
rss = CLOSE / value1;
rssav = average(rss, 10);  // 10日移動平均

RS突破訊號判斷

買進訊號：
1. RS指標突破移動平均線
2. RS與均線差距開始拉大
3. 成交量同步放大

確認步驟：
1. RS突破均線 → 注意
2. RS拉開差距 + 量放大 → 進場訊號
3. 這個類股/個股可能成為新的主流

RS使用注意事項

重要提醒：
1. RS只是濾網，不是買了就等收割的策略
2. 還需要技術面、基本面、籌碼面綜合判斷
3. 不適合抄底或逆向操作
4. 不特別適合長投或存股（除非一路強到變長期投資）

五、板塊與主流的觀念

核心觀念：

  • 一檔股票大漲 → 可能是某個主力在炒
  • 一群同屬性股票大漲 → 極可能是一群主力一起在炒

結論：

  • 觀察類股板塊漲跌 > 只看個股漲跌
  • 類股同漲準確性比只漲個股來得高
  • 投資風險性也相對低

實戰做法：
1. 先找出強勢類股
2. 在類股中找最強個股
3. 買最好的股票（最強勢、均線多頭排列）
4. 未來較容易上漲

心得

這回合補強了類股輪動的實戰細節，特別是RS指標的應用方式。之前對RS的概念比較模糊，現在清楚知道：

1. RS是相對勝率的概念 - 不是看絕對漲跌，而是看相對於大盤的表現。大盤跌5%個股跌2%，RS其實是上升的。

2. 板塊思維比個股思維重要 - 一群同屬性股票一起漲，比單一股票上漲更可靠。以後要看類股輪動，不是追單一題材股。

3. 高RS股是領頭羊 - 這是Mark Minervini驗證過的方法，最強飆股在突破前RS平均是87分。不要空高RS的股票，除非明確頭部出現。

4. RSS（漲幅差）適合短線 - 傳統RS適合長期趨勢分析，RSS適合當日或當週的短期動態判斷。

類股輪動要搭配季節性因素和總經環境來看，這部分可以再深入研究。
