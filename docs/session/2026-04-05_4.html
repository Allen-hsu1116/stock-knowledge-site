借券賣出餘額基礎

  • 借券餘額 = 借入股票尚未歸還的總量
  • 借券賣出餘額 = 已在市場賣掉的借入股票數量
  • 只有法人機構可以借券，散戶無法使用
  • 外資四大工具：期貨（多空）、選擇權（多空）、現貨（做多）、借券賣出（做空）

減少-獲利回補（股價跌時）空方

借券賣出餘額前 20 名：

  • 外資佈局一季以上的標的

重要注意事項

  • 借券不一定只有放空目的（避險、套利、法規要求）
  • 借券餘額 = 潛在空方力道（借了還沒賣）
  • 借券賣出餘額 = 已實現賣壓（借了已經賣掉）
  • 三指標齊備才準：外資賣超 + 跌破均線 + 借券賣出增加
