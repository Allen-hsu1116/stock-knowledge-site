核心概念

籌碼分析是什麼

是什麼：籌碼就是股票，籌碼分析就是觀察市場上籌碼的分布及流向，推測股價未來可能發展的方向。

為什麼重要：主力、大戶資金多，影響股價能力大。跟著他們的腳步，有機會搭上拉抬股價的順風車。

三大觀察重點：
1. 誰在買賣
2. 買賣多少
3. 持續多久

三大法人買賣

法人分類

外資：

  • 定義：台灣以外的外國機構投資者
  • 特點：三大法人中資金最雄厚
  • 影響力：對大盤影響最大

投信：

  • 定義：投資信託公司，基金公司募集投資人資金
  • 特點：由基金經理人代為操作
  • 影響力：對個股影響較大

自營商：

  • 定義：國內券商，用公司自有資金買賣
  • 特點：不幫客戶買賣，用自己的錢
  • 影響力：可能短線進出

法人買賣超判讀

買超意義：法人買進 > 賣出，可能是上漲訊號

賣超意義：法人賣出 > 買進，可能是下跌徵兆

連續買賣超天數：天數越長，代表法人越看好，較有機會出現波段行情

主力買賣超

主力定義

是什麼：擁有很多籌碼、短線可能因為買進賣出影響股價的市場參與者

六大類型：

  • 政府：國安基金等
  • 外資：三大法人之一
  • 投信：三大法人之一
  • 自營商：三大法人之一
  • 市場派：專門炒作的主力
  • 公司派：公司內部的籌碼

影響個股的主力：三大法人、市場派、公司派，多著墨於小股本股票

影響大盤的主力：外資、政府，特殊情況才出現（如國安基金護盤）

主力買賣超判讀

公式：每日買超前15名買超合計 - 每日賣超前15名賣超合計

紅色（正值）：買超 > 賣超，買方力道強

綠色（負值）：賣超 > 買超，賣方力道強

注意：中小型股可能出現「股價漲、主力賣超」的反向走勢，這是有主力介入的特徵！

反向走勢原因：

  • 籌碼夠集中或股票冷門
  • 主力左手掛賣單，右手往上買進
  • 買張數 < 賣張數，不增加持股
  • 用漲勢吸引買盤，慢慢倒貨

大戶籌碼

千張大戶持股

定義：持股 > 1000張的股東

觀察重點：這檔股票有多少比例在外資、大戶、董監手上

籌碼集中：籌碼在少數人手中 = 股價易漲難跌

籌碼分散：籌碼分散在散戶 = 股價易跌難漲

資券變化（融資融券）

融資融券定義

融資：

  • 定義：向券商借錢買股票
  • 意義：投資人預期未來股價上漲（看多）
  • 風險：股價下跌可能面臨追繳、斷頭

融券：

  • 定義：向券商借股票來賣，之後買回股票還給券商
  • 意義：投資人預期近期股價走弱（看空）
  • 風險：股價上漲可能面臨軋空

資券判讀

散戶指標：操作融資融券的幾乎都是散戶

看多訊號：融資增加 + 融券減少

看空訊號：融資減少 + 融券增加

六大籌碼集中度指標

分類

絕對指標（前三項）：具備先天優勢
1. 千張大戶持股
2. 內部人持股
3. 佔股本比重

相對指標（後三項）：趁早發現集中趨勢
4. 籌碼集中
5. 主力買賣超
6. 買賣家數差

千張大戶持股

公式：持有 > 1000張股東持股合計 / 股本（張數）

標準：

  • < 40%：籌碼分散，不利炒作
  • 40%～70%：理想的交易對象
  • > 80%：過度集中，外資不願參與，難炒出行情

實例：

  • 某股本10億的股票
  • 千張大戶持股合計5億（5000張）
  • 千張大戶持股比率 = 5000/100000 = 5%

重要：同樣比率，持有人數越少，炒作動機越強

內部人持股比率

公式：內部人持股合計 / 股本（張數）

內部人定義：董事、監察人、經理人、大股東及關係人

特點：需受主管機關監管，買賣要事先申報

判讀：

  • > 40%：高水位，可能有基本面支撐，但欠缺炒作動機
  • < 40%：可能有人為炒作動機

注意：這個指標沒有絕對好壞，看分析角度

佔股本比重

公式：（區間買超前15名買超張數合計 - 區間賣超前15名賣超張數合計）/ 股本（張數）

標準：

  • > 3%（60天內）：有特定券商在收集籌碼
  • 大型股：3%金額很大（上億），構成炒作動機
  • 小型股：3%金額可能小，需看絕對金額

注意：市值沒超過上億，不構成炒作動機

籌碼集中度

公式：（區間買超前15名買超張數合計 - 區間賣超前15名賣超張數合計）/ 區間成交量

特點：最快反映籌碼集中趨勢，但受成交量影響

標準：

  • 60天集中度 > 5%：佳
  • 120天集中度 > 3%：佳
  • 1天集中度 > 20%：可能有特定人在收集

實例：

  • 某股60天成交量100萬張
  • 買超前15名合計買超6萬張
  • 賣超前15名合計賣超1萬張
  • 籌碼集中度 = (60000-10000)/1000000 = 5%

主力買賣超（相對指標）

公式：每日買超前15名買超合計 - 每日賣超前15名賣超合計

特點：與股價表現正相關

股價漲 + 主力買超：正常走勢

股價漲 + 主力賣超：100%有主力介入，主力在倒貨

搭配指標6使用：買賣家數差

買賣家數差

公式：今天有買進該股票的券商家數 - 今天有賣出該股票的券商家數

意義：

  • 賣出家數 > 買進家數：多數人將股票賣給少數人 = 籌碼趨向集中
  • 買進家數 > 賣出家數：少數人將股票賣給多數人 = 籌碼趨向分散

注意：不能只看一天，需配合「主力買賣超」綜合分析

綜合分析範例

情境：某股股價上漲，主力買賣超卻呈現賣超

可能情況：

  • 主力左手掛賣單、右手往上買進
  • 買張數 < 賣張數，不增加持股
  • 利用漲勢吸引散戶進場
  • 慢慢倒貨給散戶

判斷方式：搭配「買賣家數差」

  • 賣出家數 > 買進家數 = 多數人賣給少數人 = 籌碼集中
  • 可能是主力在洗盤，不是真正出貨

盤中籌碼分析工具

大小單（即時）

用途：盤中即時呈現資金流向

判讀：

  • 大單買進：主力買盤
  • 大單賣出：主力賣壓
  • 小單：散戶情緒

觀察重點：

  • 集中度下降 = 主力撤出訊號

心得

籌碼面分析補足了技術面和基本面的不足，讓我們能看到「誰」在買賣。關鍵幾個心得：

1. 千張大戶持股 40-70% 是黃金區間

  • 太低（< 40%）籌碼太散

2. 股價漲 + 主力賣超 = 有鬼

  • 這是100%有主力介入的訊號

3. 60天籌碼集中度 > 5%、120天 > 3%

  • 這是主力布局的標準

4. 融資融券是散戶指標

  • 融資增加 + 融券減少 = 散戶看多

5. 六大指標分兩類使用

  • 絕對指標（前3個）：先天優勢
