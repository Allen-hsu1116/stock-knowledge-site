操作策略：突破回調進場法（Pullback Entry）

突破交易最經典的兩難：追突破怕假突破，等回調怕不回調直接噴走。今天要把「突破後回調進場」（Pullback Entry）這個操作策略拆乾淨，從進場邏輯到停損停利，一次搞懂。

一、為什麼回調進場比追突破更好？

核心邏輯

道氏理論的基礎：上漲趨勢定義為「高點更高、低點也更高」。當股價突破前波高點後，若價格回落到突破價位附近再獲得支撐，代表「阻力線變成支撐線」——角色反轉完成，這才是最安心的進場點。

追突破的問題：

  • 假突破機率高：統計上約40-60%的突破會失敗
  • 停損距離大：追在最高點，停損設在突破點下方，風險報酬比差
  • 心理壓力大：買在最高附近，一有風吹草動就想跑

回調進場的優勢：

  • 假突破已被過濾：等回調到突破點還撐住，代表突破是真的
  • 停損距離小：停損設在突破點下方一點點，風險極小
  • 風險報酬比佳：進場點離停損近，離目標遠

實例說明

情境：某股票前波高點100元，整理3個月後帶量突破到110元。

  • 追突破進場：買在110元，停損設99元（跌破前高），風險11元。目標130元，報酬20元。風險報酬比 = 20:11 ≈ 1.8:1
  • 回調進場：等股價從110元回到102元（突破點100元附近獲得支撐），買在102元，停損設98元（有效跌破突破點），風險4元。目標130元，報酬28元。風險報酬比 = 28:4 = 7:1

結論：同一檔股票、同一個趨勢，回調進場的風險報酬比是追突破的3.8倍。

二、回調 vs 反轉：最關鍵的判斷

這是回調進場法成敗的關鍵——你以為是回調，結果是反轉，那就完蛋了。

判斷標準

判斷維度回調（Pullback）反轉（Reversal）
幅度前波漲幅的5%-38.2%超過前波漲幅的61.8%
量能回調時量縮回調時放量
均線守住20MA或50MA跌破50MA且站不回
趨勢結構低點更高、高點更高跌破前波低點
時間幾天到2-3週超過3週仍未恢復

斐波那契61.8%法則

核心原則：如果回調幅度超過前波漲幅的61.8%，大概率不是回調而是反轉。

怎麼算：
1. 找到這波漲勢的起漲點（A）和最高點（B）
2. 計算漲幅 = B - A
3. 61.8%回撤位 = B - (漲幅 × 0.618)
4. 如果股價跌破這個位置，視為反轉而非回調

實例：

  • 起漲點80元，最高點120元，漲幅40元
  • 61.8%回撤位 = 120 - (40 × 0.618) = 120 - 24.72 = 95.28元
  • 如果股價跌到95元以下還不止跌，大概率已經反轉

三個關鍵回調位

  • 38.2%回撤：強勢回調，代表買盤很強，最好的進場機會
  • 50%回撤：中性回調，最常見的回調深度
  • 61.8%回撤：最後防線，跌破就當反轉處理

三、回調進場三種方法

方法一：趨勢線回調進場

是什麼：畫一條連接回調低點的上升趨勢線，等價格回到趨勢線附近再買。

怎麼做：
1. 確認股價處於上升趨勢（高點更高、低點更高）
2. 連接至少2個回調低點畫趨勢線
3. 等股價第3次以上回測趨勢線
4. 觀察K線型態：出現長下影線、吞噬型態、晨星等止跌訊號
5. 確認成交量縮小（回調量縮是正常的）
6. 進場買入

停損：設在趨勢線下方1-2%

實例：

  • 台積電從500漲到600，期間3次回測上升趨勢線
  • 第3次回測時在530元附近出現長下影線，量縮
  • 買在532元，停損設525元（趨勢線下方1%）
  • 目標600元，風險報酬比 = 68:7 ≈ 9.7:1

方法二：均線回調進場

是什麼：等股價回測關鍵均線（20MA、50MA）再買進。

怎麼做：
1. 確認均線方向向上（多頭排列）
2. 等股價回到均線附近
3. 觀察均線是否提供支撐（K線觸碰均線後反彈）
4. 搭配RSI或KD確認超賣
5. 進場買入

不同均線的意義：

  • 20MA：短線回調，適合短波段操作（持股5-20天）
  • 50MA：中線回調，適合波段操作（持股1-3個月）
  • 120MA：長線回調，適合中長線操作（持股3-12個月）

停損：設在均線下方2-3%

實例：

  • 某股沿著20MA上漲，股價從80漲到95後回到83（接近20MA在82）
  • 在83元出現紅K反彈，RSI回到40附近
  • 買在83元，停損設80元（20MA下方2%）
  • 目標100元，風險報酬比 = 17:3 ≈ 5.7:1

方法三：突破回踩進場（最推薦）

是什麼：股價突破關鍵壓力後，等它回踩到壓力線（現在變支撐）再進場。這是今天學的最核心策略。

怎麼做：
1. 找出關鍵壓力位（前波高點、整數關卡、長期橫盤區上緣）
2. 等股價帶量突破這個壓力位
3. 不要追！ 等股價回調到突破點附近
4. 觀察突破點是否提供支撐（壓力變支撐）
5. 確認支撐有效後進場

確認支撐有效三原則：

  • 價格碰到突破點後反彈（K線有下影線）
  • 成交量在回調時縮小
  • 回調幅度不超過突破點下方1-2%

停損：設在突破點下方1-2%（有效跌破代表假突破）

實例：

  • 某股長期在45-50元區間盤整3個月
  • 某天帶量突破50元，最高漲到55元
  • 等了2天，股價回到51元（50元突破點附近）
  • 觀察51元有撐，買在51元
  • 停損設49元（跌破50元突破點1%），風險2元
  • 目標60元（區間高度5元 × 1:1以上），風險報酬比 = 9:2 = 4.5:1

四、盤整時間 vs 突破幅度：經驗法則

這是一個非常實用的經驗法則，幫你預判突破後可能有多大行情：

盤整時間 → 預期波段：

  • 盤整3個月以上 → 可能賺1個月的波段
  • 盤整1-2年 → 可能賺半年的波段
  • 盤整5年以上 → 可能賺1年以上的大波段

邏輯：盤整越久，累積的潛在買盤越多，一旦突破，力道越強。

最適合突破交易的標的：

  • ✅ 冷門股突然有利多（如航運股疫情爆發、AI概念股需求暴增）
  • ✅ 長期盤整後基本面出現轉折（虧轉盈、營收暴增）
  • ✅ 沉寂多年突然有新題材

不適合突破交易的標的：

  • ❌ 大型權值股（蘋果、波克夏這種很難突然暴漲）
  • ❌ 已經連漲很久的股票（突破後容易是末升段）
  • ❌ 沒有基本面支撐的純技術突破

五、真假突破判斷

### 真突破特徵 1. **帶量突破**：成交量放大到5日均量的2-3倍以上 2. **收盤站穩**：收盤價在突破點上方（不是盤中刺一下就掉下來） 3. **連續3天站穩**：連續3個交易日收盤在突破點上方 4. **基本面配合**：有利多消息或營收成長支撐 ### 假突破特徵 1. **量增但漲不動**：放量卻只小漲或收長上影線 2. **隔日立刻跌回**：突破隔天就跌回區間內 3. **無基本面支撐**：純技術面突破，沒有利多消息 4. **大戶出貨**：突破當天法人賣超、千張大戶減少

六、完整操作SOP：突破回調進場法

Step 1：選股（盤後作業）

  • 找出正在盤整或有突破跡象的股票
  • 條件：盤整3個月以上、量能萎縮到極致、均線糾結
  • 搭配籌碼面：法人近期買超、千張大戶增加

Step 2：設定觀察名單

  • 把突破候選股加入追蹤
  • 設定突破價位提醒（壓力點上方1-2%）
  • 準備好進場計畫

Step 3：等待突破

  • 突破必須帶量（量大於5日均量2倍）
  • 收盤確認站穩突破點
  • 不要追！

Step 4：等待回調

  • 突破後耐心等股價回到突破點附近
  • 回調時量能要縮小
  • 回調幅度不超過前波漲幅的38.2%（最強）到50%（可接受）

Step 5：確認進場

  • 觀察突破點是否提供支撐
  • K線出現止跌訊號（長下影線、吞噬、晨星）
  • 搭配指標：RSI回到40-50附近、KD黃金交叉
  • 進場！

Step 6：停損停利

  • 停損：突破點下方1-2%
  • 停利：前波漲幅的1:1以上，或用追蹤停利（每創新高就將停損上移）
  • 加碼：只在賺錢時加碼，每次回調到均線獲支撐時加碼

七、玉山證券短線動能突破策略補充

選股條件

  • 股價創近20日/60日/120日新高
  • 成交量大於5日均量的3倍
  • 搭配：突破上關價、盤中由黑翻紅、一紅吃三黑、突破糾結均線、OBV突破
### 進場方式（兩種） 1. **積極型**：突破當下追價，開盤30分鐘到1小時內是關鍵時段 2. **保守型（推薦）**：等突破後回測均價線或前波高點，確認不破再買

停損方式

  • 積極進場：跌破前波高點下方0.5%-1%（4-8 tick）
  • 保守進場：跌破均價線即停損

停利方式

  • 當沖型（tick流/一波流）：5-15分鐘內未繼續上漲即出場；獲利超過5-10 tick後用智慧單保本
  • 短波段型：抱到中盤（11-12點）或收盤前（13-13:30）再陸續減碼；鎖漲停則改波段操作

心得

回調進場法本質上就是「讓市場先幫你驗證突破是真的」。很多人看到突破就興奮地追進去，結果被假突破電得不要不要的。等回調不是怕，是穩。

最核心的概念是「壓力變支撐」——如果一個價位曾經是很多人想賣的壓力，突破後變成很多人想買的支撐，那代表市場的共識已經改變了，這時候進場勝率最高。

另外盤整時間和突破幅度的關係很值得記住。冷門股沉寂越久，一旦有利多爆發，力道越恐怖。這也解釋為什麼航運股在疫情時能漲那麼兇——已經冷門到沒人關注了，突然需求暴增，所有空手的人都想買，結果就是爆量突破後直接起飛。

最後，回調 vs 反轉的判斷是生死線。61.8%法則很實用：回調超過前波漲幅的61.8%就不是回調了，是反轉。這時候不要再幻想「只是回調」，該跑就跑。
