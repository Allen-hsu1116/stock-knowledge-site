行為金融學基礎概念

是什麼：行為金融學認為投資人並非完全理性，會受到自身偏見影響產生認知錯誤，導致錯誤決策。它與效率市場假說相對，解釋為什麼市場會出現非理性波動。

與市場有效理論的關係：

  • 市場有效理論：假設投資人完全理性，價格反映所有資訊
  • 行為金融學：認為人是情感動物，行為常受情緒驅使
  • 實務上：市場既非完全有效也非完全無效，需要根據市場特性調整策略

怎麼做：
1. 了解自己容易犯哪些認知偏誤
2. 記錄交易決策，回顧哪些偏誤影響了你
3. 建立檢查清單，在重大決策前逐一核對
4. 將情緒與行動隔離，學會管理情緒

十大行為金融偏誤詳解

一、心理帳戶（Mental Accounting）

是什麼：人們會把客觀等價的支出或收益，在心理上歸類到不同帳戶，影響消費決策。

實例：

  • 薪水歸類到「勤勞帳戶」→ 精打細算
  • 年終獎金歸類到「獎勵帳戶」→ 輕鬆花費
  • 發票中獎歸類到「天上掉下的禮物」→ 更容易衝動花費

同樣是 1 萬元，但在不同心理帳戶中有不同的花費態度。用信用卡比用現金更容易花錢，因為兩者分屬不同心理帳戶。

怎麼做：
1. 檢視自己的花費習慣：如果用信用卡比現金更容易花錢，代表兩者在不同心理帳戶
2. 主動建立帳戶：儲蓄帳戶、買房帳戶、財務自由帳戶，分別存錢
3. 運用心理帳戶的影響，克制支出
4. 投资時把所有錢視為同一帳戶，不要因為來源不同而有不同風險態度

適用時機：制定投資計畫、理財規劃時

二、從眾行為（Herd Mentality / 羊群效應）

是什麼：人們喜歡模仿大多數人的行為，認為大群體不太可能出錯。

實例：

  • 分析師推薦某股票，即使公司財務不健全，投資人仍追捧
  • 最近漲多的標的容易被追捧，很多人說不清為什麼上漲就買入
  • 投資人喜歡按字母順序選股，因為列表是按字母排列的

怎麼做：
1. 「人多的地方不要去」是基本原則
2. 如果容易被他人影響，減少資訊接觸，少看電視報紙
3. 跟群眾做相反的事需要勇氣，在不重押的前提下多嘗試幾次
4. 建立自己的判斷框架，不要只依賴他人意見

適用時機：市場熱點出現、眾人一致看好/看壞時

三、損失規避（Loss Aversion）

是什麼：人們對損失的痛苦感受是獲利喜悅的 2 倍以上。損失 100 元的痛苦 > 找到 100 元的喜悅。

實例：

  • 為了避免「真正」虧損的痛苦，投資人會繼續持有虧損股票（套牢）
  • 只願意在舊標的加減碼，不願關注新標的（即使更有利）
  • 只出售有獲利的股票，不出售虧損的股票
  • 不願意以低於成本的價格出售，儘管繼續持有風險更大

怎麼做：
1. 理解風險承受能力是獲利基礎，但不能想著一點風險都不承擔
2. 你更該害怕財務目標無法達成，而不只是害怕虧損
3. 面對虧損時，問自己：「如果我是空手，還會想買這檔股票嗎？」
4. 設定明確停損點，事前決定虧多少就出場

適用時機：停損決策、是否繼續持有虧損部位時

四、沉沒成本（Sunk Costs）

是什麼：已經發生、無法改變的成本。人們常因為「不想浪費」而做出不理性決策。

實例：

  • 買了電影票，電影不好看仍硬看完 → 其實應該離場做更有意義的事
  • 投資虧損股票但不想賣，「都已經投下去了不該退場」
  • 任何時間點，持有股票與沒持有（假設立刻賣掉），資產總值一樣

怎麼做：
1. 做決策時「歸零」，不考虑過去虧損金額
2. 「永遠假設自己是空手」：如果你是空手會想買嗎？會就持有，不會就賣出
3. 過去成本不該影響決策，只考慮「接下來最好的機會是什麼」
4. 定期檢視持股：如果今天空手，還會買這些股票嗎？

適用時機：虧損部位決策、是否加碼/減碼時

五、賭徒謬誤（Gambler's Fallacy）

是什麼：人們根據過去一系列事件結果，認為隨機事件發生的可能性會不同。例如連續出黑球就覺得紅球快來了。

實例：

  • 股票持續上漲 → 投資人認為快崩盤，決定賣出
  • 股票持續下跌 → 投資人認為快上漲，決定持有
  • 反過來也是謬誤：持續上漲覺得會繼續漲、持續下跌覺得會繼續跌

事實是：每個事件都是獨立的，過去不決定未來。

怎麼做：
1. 不要常常猜漲猜跌，養成說「我不知道」的習慣
2. 仔細思考事件發生的實際過程，不要用偶然機率或迷信下決策
3. 建立數據導向的判斷方式，而非依賴「感覺」
4. 理解趨勢可能持續也可能反轉，需要更多分析而非簡單推論

適用時機：判斷趨勢是否持續、是否逆勢操作時

六、控制的錯覺（Illusion of Control）

是什麼：人們高估自己控制事件的能力，把希望寄託在迷信或其他心理因素上。

實例：

  • 買樂透簽自己的幸運號碼（生日等），認為有助中獎
  • 基本面分析者研究越深入，越相信自己的判斷
  • 把長期投資當作信仰，忽略其他問題
  • 每天殺進殺出、勇於決策、堅定停損 → 聽起來不錯，但真的比較好嗎？

怎麼做：
1. 持有一組分散投資部位（如大盤指數型 ETF），和自己的選股比較績效
2. 承認「市場比我聰明」，不要過度自信
3. 對自己選股或方法高度狂熱的人，保持距離
4. 建立客觀評估機制，定期檢視自己的判斷準確率

適用時機：深入研究某標的後、對自己判斷很有信心時

七、選擇悖論（Paradox of Choice）

是什麼：過多選擇反而讓人無法做出明智決策，受到錨定效應、框架效應影響。

實例：

  • 市場上有數千檔股票可投資，每分鐘都有買賣機會
  • 太多選擇自由讓投資人無所適從，做選擇時隨意挑選
  • 放棄思考，只想問：「什麼可以買？」「什麼會漲？」
  • 新的金融產品不斷推出，越新真的越好嗎？

怎麼做：
1. 價值投資者只投資自己理解的少數公司
2. 巴菲特建議普通投資人用最簡單方式：長期投資 ETF，不需要選股、擇時
3. 限制自己的投資範圍，只看自己熟悉的領域
4. 選擇「老東西」而非追逐新產品，簡單策略往往更有效

適用時機：選股決策、是否追蹤太多標的時

八、確認偏誤（Confirmation Bias）

是什麼：人們選擇性收集對自己有利的資訊，忽略不利或矛盾的資訊。

實例：

  • 持有 A 公司股票，忽略該公司的不利消息
  • 只專注某些財經分析師的意見，忽略其他人的意見

-「不要跟股票談戀愛」就是要避免確認偏誤

怎麼做：
1. 主動尋找反對意見，問「什麼情況下我的判斷會錯？」
2. 記錄自己的判斷和結果，定期回顧哪些假設是錯的
3. 不要只看認同自己的分析師或資訊來源
4. 把「做功課」定義為找反例，而非找支持證據

適用時機：研究股票、接受分析師建議時

九、錨定偏見（Anchoring Bias）

是什麼：人們做決定時過分依賴已有的訊息或第一個看到的訊息。

實例：

  • 先看到 1200 元 T 恤，再看到 100 元 T 恤 → 覺得 100 元很便宜
  • 如果只看到 100 元 T 恤，可能不會覺得便宜
  • 問「3 個月前台積電股價會到多少？」你會先看今天股價，然後基於此做假設

怎麼做：
1. 以量化數據替代假設，對抗錨定偏差
2. 參考的「錨」越多越好，不要只根據一個錨點下判斷
3. 問自己：「如果沒有這個參考點，我會怎麼判斷？」
4. 多角度評估：基本面、技術面、總體面分別看，而非只看一個面向

適用時機：評估股價是否合理、設定價格目標時

十、情緒（Emotions）

是什麼：情緒會影響金融決策，是直覺的快速評估。恐懼和貪婪是最大的情緒敵人。

實例：

  • 牛市時：投資人紛紛進場，希望能獲利（貪婪）
  • 熊市時：投資人不敢進場，避免損失（恐懼）
  • 情緒會衍生出其他類型的偏見

怎麼做：
1. 盡快感受情緒起伏，記住它
2. 把「行動」和「情緒」隔離，決策時不被情緒帶著走
3. 開始投資後，記住第一次損益變動時的情緒，之後試著掌握
4. 優秀的投資者都是最佳的情緒管理者

適用時機：牛市/熊市極端情緒時、大賺大賠後

行為金融學的投資實踐

塞勒基金的實戰策略

核心理念：利用人類對訊息的「過度反應」和「反應遲鈍」兩個行為偏差，進行反方向投資。

實例：

  • Fuller & Thaler Fund 自成立以來年均收益超越指數 3.2%
  • 策略相對簡單，但長期有效
  • 不需要複雜模型，關鍵是理解市場參與者的行為偏誤

怎麼做：
1. 觀察市場對訊息的反應：過度反應或反應不足
2. 過度反應時：考慮反向操作
3. 反應遲鈍時：考慮順勢加碼或等待市場修正
4. 保持策略簡單，複雜不一定好

市場有效性判斷框架

兩個關鍵問題：
1. 你進行投資的市場多有效？（發達市場通常更有效）
2. 訊息多快被市場反映？反應是過度還是不夠？

實例：

  • 美國大型股市場較有效，主動投資較難獲取超額報酬
  • 新興市場可能存在較多非理性波動，主動投資機會較多
  • 1986 年挑戰者號失事：Morton Thiokol 股價在調查報告發布前就大跌，市場「知道」問題在哪

步驟二：建立檢查清單

  • 在重大決策前，逐一核對是否受到偏誤影響
  • 問自己：「我是空手的話會怎麼做？」

步驟三：情感隔離

  • 記住第一次損益變動時的情緒
  • 學習把行動和情緒分開

步驟四：保持簡單

  • 不要追逐複雜策略
  • 簡單策略往往更有效

步驟五：持續學習

  • 閱讀行為金融學相關書籍和文章
  • 反思自己的投資決策過程

心得

這篇文章把行為金融學的十大偏誤整理得很清楚，每個偏誤都有實例和應對方法。最重要的是「市場有效理論 vs 行為金融學」的框架：

市場既非完全有效，也非完全無效。投資人既非完全理性，也非完全非理性。關鍵是理解：
1. 你投資的市場多有效？
2. 訊息反映多快？是過度還是不足？

塞勒基金的實戰成績很有說服力：簡單的「過度反應/反應遲鈍」反向策略，年均超越指數 3.2%。複雜不一定好，理解人性弱點比複雜模型更重要。

最後一句話很深刻：「做投資，既要了解自己，又要了解他人。」
