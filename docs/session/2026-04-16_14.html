一、ORB開盤區間突破策略完整解析

1. ORB策略核心邏輯

是什麼：Opening Range Breakout（開盤區間突破），由Toby Crabel在1990年著作《Day Trading with Short Term Price Patterns and Opening Range Breakout》中提出，Larry Williams靠此策略一年從1萬變100萬聲名大噪。核心邏輯是：開盤後一段時間內的價格區間代表多空力量的初步交戰結果，突破這個區間代表當日方向已定，動能會延續。

怎麼做：
1. 定義開盤區間：開盤後30分鐘（9:00-9:30）的最高價與最低價，作為區間上下緣
2. 突破上緣做多：價格突破區間高點 → 買進做多
3. 跌破下緣做空：價格跌破區間低點 → 賣出做空
4. 收盤平倉：當天必須出場，不留倉過夜

實例：

  • 情境：台指期開盤9:00-9:30，最高點20050、最低點19980
  • 區間：19980-20050（70點區間）
  • 9:35價格突破20050 → 做多，停損設19980（區間下緣），目標看20120
  • 若9:40跌破19980 → 做空，停損設20050（區間上緣）

適用時機：趨勢明確的日子效果最好；盤整日會頻繁假突破造成虧損

2. ORB三大版本與參數設定

是什麼：ORB經過多年改良，有三種波動度計算方式，決定區間的大小和進場敏感度。

怎麼做：

版本一：Larry Williams版（最經典）

  • 基準價 = 當日開盤價
  • 波動度 = 前一日True Range（真實區間）
  • 做多線 = 開盤價 + 乘數 × 前日True Range
  • 做空線 = 開盤價 - 乘數 × 前日True Range
  • 乘數通常設0.3-0.5，越小越敏感（越早進場但假突破越多）

版本二：Sheldon Knight版

  • 波動度 = 前N日的最高價與最低價之差
  • N通常設3-5日
  • 適合波動較大的市場，區間比Williams版寬

版本三：Toby Crabel版（原始版）

  • 波動度 = 前N日的 min(開盤價-最低價, 最高價-開盤價) 的平均值
  • 只取開盤價到極值的較短邊，區間更窄
  • 適合波動較小的市場，進場更早但假突破風險更高

實例：

  • 台指期前日True Range = 150點，乘數0.3
  • Williams版做多線 = 開盤價 + 0.3 × 150 = 開盤價 + 45點
  • Williams版做空線 = 開盤價 - 0.3 × 150 = 開盤價 - 45點
  • 乘數0.3 vs 0.5：0.3進場更早（45點就進場）但假突破更多；0.5進場更晚（75點才進場）但勝率更高

適用時機：

  • Williams版 → 通用性最強，適合大多數情況
  • Knight版 → 波動大的市場（如期貨、加密貨幣）
  • Crabel版 → 波動小的市場（如大型股、穩定型ETF）

3. ORB策略三大改良法

是什麼：原始ORB在台指近年績效走下坡，主因是假突破太多、大行情太少。三種改良方式可以提升勝率和獲利。

怎麼做：

改良一：二次突破法（回踩再攻）

  • 第一次突破開盤區間不進場，等待價格回落
  • 第二次突破同一方向才進場
  • 原理：假突破通常一次就結束，二次突破代表真正的動能
  • 實例：9:35突破20050（第一次）→ 不進場 → 9:50回落到20020 → 10:05再次突破20050（第二次）→ 進場做多

改良二：假突破反向操作

  • 辨識假突破：突破後5-15分鐘內回落到區間內
  • 假突破做多失敗 → 反向做空
  • 假突破做空失敗 → 反向做多
  • 實例：9:35突破20050做多，9:40回落到20040 → 假突破確認 → 9:45跌破19980反向做空

改良三：VWAP趨勢濾網（台灣碩士論文改良）

  • 加入VWAP判斷趨勢方向：價格在VWAP上方只做多，下方只做空
  • 加入ATR動態停損：停損距離 = k × ATR，而非固定區間
  • 實證：改良後的ORB在台指期2016-2025年回測，風險調整後績效優於原始版本

適用時機：

  • 改良一 → 假突破頻繁的盤整市
  • 改良二 → 假突破多的趨勢初期
  • 改良三 → 需要趨勢濾網的全面改良版

4. ORB策略的致命缺陷與應對

是什麼：ORB不是萬能的，了解它的缺陷才能正確使用。

怎麼做：

缺陷一：假突破太多

  • 原因：高頻交易和法人開盤試單造成大量假突破
  • 應對：使用二次突破法或VWAP濾網過濾

缺陷二：市場結構改變

  • 原因：2003年後美國走向近24小時電子交易，開盤價重要性大降
  • 應對：ORB在日內交易仍有價值，但不能依賴單一策略

缺陷三：台指盤性變黏

  • 原因：程式交易普及後，ORB策略被太多人使用，套利空間消失
  • 應對：加入獨特濾網（籌碼面、VWAP、成交量確認）提升差異化

缺陷四：策略有賞味期限

  • 原因：任何公開策略都會因為使用人數增加而失效
  • 應對：定期監控策略績效，連續虧損超過歷史最大值時停止使用

實例：

  • 2009年前ORB在台指期績效優異，之後因為太多人使用而逐漸失效
  • 冠軍交易法（ORB變形）也曾熱賣，最終不敵虧損消聲匿跡

適用時機：了解缺陷後，ORB仍可作為當沖策略的基礎框架，但必須改良

二、回測績效評估指標完整解析

5. 回測績效五大火眼金睛

是什麼：程式交易回測報表有二三十個數據，但真正重要的只有五個——它們告訴你策略能不能用、風險有多大、會不會破產。

怎麼做：

指標一：資金曲線圖（第一眼看這個）

  • 不是數據，是圖表——直觀顯示策略整體表現
  • 斜率代表獲利速度、鋸齒幅度代表穩定度
  • 兩次創高間隔越短越好
  • 看曲線圖中賠錢的時段，對應當時行情，了解策略適合什麼盤勢

指標二：MDD（最大回撤）

  • 定義：資金曲線從最高點到最低點的最大回撤金額
  • 實務鐵律：「MDD就是拿來破的」——實際MDD通常是回測的1.5-2倍
  • 原因：回測是完美環境（無滑價、無延遲），實盤有滑價、浮動點差、網路延遲
  • 判斷標準：MDD佔總資金超過30%就不建議使用

指標三：風報比（風險報酬比）

  • 公式：總淨盈利 / MDD
  • 風報比越高越好
  • 判斷標準：回測期間1年以上，風報比>3算不錯，>5算優秀
  • 注意：風報比會隨回測時間拉長而上升（淨利潤累積），所以不能跨期間比較
  • MT5報表中的「採收率」（Recovery Factor）就是風報比

指標四：連續虧損次數

  • 最多連續虧損：策略歷史上最慘的連續虧損次數，實際可能破紀錄
  • 平均連續虧損：日常交易的心理預期基準
  • 心理影響：連續虧損是交易者最痛苦的時期，如果策略勝率太低（<25%），即使長期獲利也很難堅持

指標五：交易單數

  • 最低標準：一個月至少2次，一年至少24次，十年至少240次
  • 單數太少 → 統計意義不足，回測績效不可靠
  • 單數太多 → 順勢策略單數多獲利困難，需要加入濾網

實例：

  • 好策略：年化報酬20%、MDD 10%、風報比2、最多連虧5次、年交易60次
  • 差策略：年化報酬50%、MDD 40%、風報比1.25、最多連虧12次、年交易10次（單數太少，統計不可靠）

適用時機：每次看回測報表，先看資金曲線圖，再看MDD和風報比，最後看連虧次數和交易單數

6. 進階績效指標：夏普、索提諾、卡瑪、獲利因子

是什麼：五大火眼金睛之外的四個進階指標，用來更精確地衡量策略品質。

怎麼做：

夏普比率（Sharpe Ratio）

  • 公式：（年化報酬率 - 無風險利率）/ 年化波動率
  • 衡量：每承擔一單位總風險，獲得多少超額報酬
  • 判斷標準：< 0.5 差、0.5-1.0 普通、1.0-2.0 良好、> 2.0 優秀
  • 缺陷：把上漲波動和下跌波動同等對待，但投資者只討厭下跌波動

索提諾比率（Sortino Ratio）

  • 公式：（年化報酬率 - 無風險利率）/ 下行波動率
  • 與夏普的差異：分母只用下行波動率（只算虧損的波動），上漲波動不計入
  • 判斷標準：< 1 差、1-2 良好、> 2 優秀
  • 優勢：更符合投資者真實感受——上漲的波動是好事，不應懲罰

卡瑪比率（Calmar Ratio）

  • 公式：年化報酬率 / 最大回撤
  • 衡量：每承擔一單位最大回撤風險，獲得多少報酬
  • 判斷標準：< 0.2 差、0.2-0.5 合理、> 0.5 優秀
  • 特點：對極端風險的懲罰最重——MDD越大，卡瑪比率越低
  • 與風報比的差異：風報比用總淨利潤/MDD（金額比），卡瑪用年化報酬率/MDD%（比率比）

獲利因子（Profit Factor）

  • 公式：總獲利金額 / 總虧損金額
  • 衡量：每虧1元能賺多少元
  • 判斷標準：< 1 不具投資價值、1-1.5 有潛力但風控待優化、1.5-2 穩健、> 2 非常吸引人
  • 特點：最直覺的策略品質指標，Profit Factor = 2 代表賺的錢是虧的2倍

尾部比率（Tail Ratio）

  • 公式：報酬分布95%分位數 / 5%分位數
  • 衡量：極端獲利與極端虧損的比率
  • 判斷標準：> 1 良好（極端獲利大於極端虧損）、≈ 1 平衡、< 1 風險高
  • 用途：評估黑天鵝事件的策略韌性

實例：

  • 策略A：年化報酬30%、MDD 15%、夏普1.5、索提諾2.0、卡瑪2.0、PF 1.8
  • 策略B：年化報酬50%、MDD 35%、夏普1.0、索提諾1.2、卡瑪1.43、PF 1.4
  • 分析：策略A風險調整後報酬更好（夏普、索提諾、卡瑪都更高），策略B只是賺得多但風險也大

適用時機：比較不同策略時，不要只看年化報酬率，要看風險調整後指標（夏普、索提諾、卡瑪）

7. ORB策略搭配凱利公式的資金管控

是什麼：用凱利公式計算ORB策略的最適下注比例，但要小心「理論最適」和「實際安全」的巨大差距。

怎麼做：

凱利公式基本版：

  • f* = (bp - q) / b
  • b = 賠率（平均獲利/平均虧損）
  • p = 勝率
  • q = 1 - p

實例：

  • ORB策略回測：勝率40%、平均虧損15點、平均獲利40點
  • 賠率 b = 40/15 = 2.67
  • 凱利比例 = (2.67×0.4 - 0.6) / 2.67 = 0.467 / 2.67 = 17.5%
  • 半凱利（更安全）= 8.75%

最大區間虧損的制約：

  • 回測最大區間虧損400點 = 80,000元（大台每點200元）
  • 為了不被追繳保證金（維持率>80%），至少需要16萬資金操作一口
  • 16萬操作一口大台 = 10倍槓桿
  • 但凱利公式說可以操作9口（80倍槓桿）——這是矛盾的！

矛盾原因：

  • 凱利公式用的是「平均」獲利和虧損，但現實中每筆盈虧高低起伏很大
  • 最大虧損遠大於平均虧損，全壓凱利比例可能一次就破產

實務建議：
1. 用1/4凱利到1/2凱利，不要用全凱利
2. 以最大虧損為基礎計算安全倉位，而不是用平均虧損
3. 預留至少2倍MDD的資金作為安全邊際

適用時機：任何策略上線前都要做凱利公式計算，但實際下注要用保守版本

心得

今天學了兩個看似不相關的主題：ORB開盤區間突破策略和回測績效評估指標，但它們之間有深層聯繫——ORB是策略開發的起點，績效評估是策略驗證的終點。

1. ORB教我的最重要一課：沒有永遠有效的策略。Larry Williams靠ORB一年從1萬變100萬，但同一個策略在台指近年績效走下坡。原因不是策略不好，是市場結構改變了——高頻交易讓開盤假突破變多、程式交易讓套利空間消失。這告訴我，任何策略都有「賞味期限」，定期監控績效、及時停損策略本身，比開發新策略更重要。

2. 回測報表的第一眼不是看賺多少，是看虧多少。艾比女王說得好：「賺錢都是沒問題的，有問題時都會發生在賠錢的時候。」所以第一眼看資金曲線圖（直觀判斷），第二眼看MDD（最大回撤），第三眼看風報比（獲利與風險的比值）。年化報酬率只是裝飾品，MDD和風報比才是定生死的核心。

3. 夏普比率vs索提諾比率，後者更實用。夏普把上漲波動和下跌波動一視同仁，但誰會討厭上漲？索提諾只計算下行波動，更符合投資者真實感受。一個策略如果大賺小虧（上漲波動大但下跌波動小），夏普比率可能不高但索提諾比率很高——這種策略其實很棒，只是夏普比率沒反映出來。

4. 凱利公式的理論與現實差距。凱利公式算出17.5%的最適下注比例，但最大虧損告訴你至少要16萬操作一口（10倍槓桿），而凱利又說可以操作9口（80倍槓桿）。這個矛盾的核心是：平均值不等於極端值。回測的平均虧損是15點，但最大虧損是400點——如果你用平均值來計算下注比例，一次極端虧損就可能把你炸出市場。所以實務上永遠用半凱利甚至1/4凱利，並以最大虧損為基礎計算安全倉位。

5. ORB的三種改良剛好對應了之前學的三個概念：二次突破法=假突破過濾（之前學的2B法則和突破確認）、假突破反向操作=逆勢策略（之前學的ADX<20用逆勢）、VWAP濾網=法人視角判斷（之前學的VWAP量價加權均價）。所有策略都是互相連接的，學得越多，改良的武器庫就越豐富。

一、VWAP的核心定位：不是技術指標，是流動性結構

1. VWAP vs 傳統均線的關鍵差異

是什麼：VWAP（成交量加權平均價格）把「成交量」納入計算，而非只看價格的算術平均。它代表當日所有參與者的「真實平均持倉成本」。

怎麼做：
1. 理解計算邏輯：VWAP = Σ(價格 × 成交量) / Σ(總成交量)，成交量大的價位對VWAP影響更大
2. 辨別使用時機：MA適合波段和長線趨勢；VWAP專為日內交易設計，每日開盤歸零重算
3. 看懂數字背後意義：VWAP不是「技術指標」，它是「流動性結構」——代表主流資金的平均心態價位

實例：

  • A股票在100元成交2,000股，在101元只成交100股
  • 傳統均線可能算出100.5元，但VWAP會算出約100.04元
  • 因為大部分資金都在100元成交，VWAP更貼近「真實成本」

適用時機：當沖判斷日內多空、法人成本參考；不適合日線以上週期

2. VWAP適合與不適合的使用情境

是什麼：VWAP在不同盤勢下效果差異極大，必須辨識使用時機才能提高勝率。

怎麼做：

最適用時機：
1. 開盤第一小時（08:45～09:45）：VWAP迅速拉出「早盤資金中心區」，極具指標性
2. 盤整震盪盤：VWAP常成為主力回補/反壓的分界線
3. 多空轉折時刻：穿越VWAP通常伴隨趨勢切換跡象

不適合時機：
1. 趨勢爆發盤（一日單邊2%以上）：VWAP會「失效延遲」，一直被遠遠甩開，不能當支撐壓力
2. 超低成交量窒息盤：VWAP跳動很小，無指標意義
3. 開盤前15-30分鐘：累積數據不足，VWAP不穩定

實例：

  • 2024年某日台積電開盤後30分鐘VWAP在580元附近震盪，此時參考價值低
  • 等到09:45之後成交量累積足夠，VWAP穩定在582元，此時回測582元才是有效支撐

適用時機：日內交易為主；波段交易請改用MA

二、VWAP三大實戰策略進階版

1. VWAP趨勢跟隨法（回測支撐）

是什麼：股價在VWAP之上時，VWAP成為動態支撐；等待股價回測VWAP不破時進場做多。

怎麼做：
1. 確認多空方向：股價穩定在VWAP之上 → 多方強勢；在VWAP之下 → 空方強勢
2. 不追高：觀察股價拉升後等待回測VWAP
3. 進場確認：股價回測VWAP時出現下影線或紅K棒，且量能沒有異常萎縮
4. 停損設定：跌破VWAP超過0.3%-0.5%（依個股波動度調整），且5分鐘內無法站回

實例：

  • 某股開盤後強勢上漲，VWAP穩定在150元
  • 股價一度拉到153元後回落至150.2元
  • 出現帶下影線的紅K，量能正常 → 在150.5元附近進場做多
  • 停損設149.3元（跌破VWAP 0.5%），目標152-153元

2. VWAP乖離反轉法（抓轉折）

是什麼：當股價短時間內急速拉升、遠離VWAP線（乖離率過大）時，通常會有回歸均值的壓力，可嘗試短空或等待反轉。

怎麼做：
1. 觀察乖離程度：股價偏離VWAP超過1.5-2倍標準差（可用VWAP Bands輔助判斷）
2. 確認反轉訊號：出現爆量長黑K或上影線，且成交量開始萎縮，無力再創新高
3. 短空進場：目標價設定在VWAP線附近
4. 嚴設停損：突破前高即停損（逆勢操作風險較高）

實例：

  • 某股10:00急拉至VWAP上方2%以上（VWAP在100元，股價衝到103元）
  • 出現長上影線，量能開始萎縮
  • 在102.5元短空，目標100.5元（VWAP附近），停損103.5元（突破前高）

3. VWAP穿越確認法（多空轉折）

是什麼：價格與VWAP線的交叉是日內趨勢轉變的關鍵訊號，必須搭配量能確認。

怎麼做：
1. 黃金交叉（看漲）：價格從VWAP下方帶量突破VWAP線
2. 死亡交叉（看跌）：價格從VWAP上方放量跌破VWAP線
3. 確認條件：突破後至少站穩2根5分K線，且回踩VWAP確認支撐
4. 假突破判斷：穿越VWAP時成交量萎縮 → 可能是假突破，需警惕

實例：

  • 某股09:30起股價持續在VWAP下方（VWAP=200元）
  • 10:15帶量突破VWAP，5分K站穩200元以上
  • 10:25回踩200.5元確認支撐 → 進場做多
  • 若突破時量能只有均量的60% → 假突破機率高，不追

三、VWAP標準差通道（VWAP Bands）進階應用

1. VWAP Bands的計算與意義

是什麼：在VWAP線的上下方加上1倍或2倍標準差，構成當日股價的常態分佈區間。68%的成交價落在一倍標準差內，95%落在兩倍標準差內。

怎麼做：
1. 上方2倍標準差受阻：股價突破VWAP但受阻於上軌道 → 多頭力道已達極限，容易劇烈回檔
2. 下方2倍標準差反彈：股價跌破VWAP但在下軌道獲得支撐 → 空方力道耗盡，可能反彈
3. VWAP + 支撐區重疊：當股價回測VWAP且剛好與下方支撐區重疊 → 「共振點」，勝率大幅提高

實例：

  • VWAP=100元，+1σ=101.5元，+2σ=103元，-1σ=98.5元，-2σ=97元
  • 股價衝到103元（+2σ）但無量無法站穩 → 多頭極限訊號，可考慮短空
  • 股價跌到97元（-2σ）出現下影線 → 空方耗盡，可嘗試短多

四、Anchored VWAP：指定起點的進階武器

1. Anchored VWAP定義與用途

是什麼：傳統VWAP固定從開盤開始計算，但Anchored VWAP允許你指定任意時間點作為起點，重新累積成交量加權平均價。它能捕捉「特定事件後資金的真實平均進場成本」。

怎麼做：
1. 選擇Anchor點：以重要事件發生時間為起點（如CPI公布、FOMC會議、財報公布、重大利多利空消息）
2. 從Anchor開始計算VWAP：重新累積成交量與價格
3. 解讀Anchored VWAP：如果事件後股價在Anchored VWAP之上 → 事件後買盤偏多；之下 → 賣壓偏重
4. 搭配傳統VWAP交叉確認：兩個VWAP重疊確認時勝率更高

實例：

  • 某日13:30 Fed公布利率決策，股價隨即大幅波動
  • 從13:30開始計算Anchored VWAP，假設算出610元
  • 如果股價持續在610元以上 → Fed公布後買盤主導，偏多操作
  • 如果跌破610元 → Fed公布後賣壓佔優，偏空操作

2. Anchored VWAP三大應用場景

怎麼做：
1. 重大新聞事件：CPI、FOMC、非農就業等數據公布後，從公布時間開始計算，看資金真實流向
2. 缺口跳空：從缺口發生的K線開始計算Anchored VWAP，判斷缺口後資金的真實成本
3. 趨勢轉折點：從前一波高低點開始計算，追蹤新趨勢的資金成本結構

適用時機：事件驅動型交易、缺口後方向判斷、趨勢轉折確認；不適合一般盤整行情（傳統VWAP即可）

五、法人VWAP執行邏輯與散戶對策

1. 法人為什麼用VWAP執行

是什麼：法人（外資、投信）管理大筆資金，目標不是「買在最低點」，而是「不買得比市場平均貴」。許多法人的自動化交易演算法以VWAP為基準分批執行。

怎麼做：
1. 理解法人行為：股價低於VWAP → 法人買盤積極（優於平均成本建倉）；股價遠高於VWAP → 買進力道減弱
2. 跟隨法人方向：觀察股價在VWAP附近的反應，等於觀察法人的成本防線
3. 避開法人出貨區：股價遠高於VWAP且量能萎縮 → 法人可能已經完成建倉，進入出貨階段

實例：

  • 外資欲買進5,000張台積電
  • 當日VWAP=580元
  • 演算法設定：股價<580元時加大買進速度；股價>583元時減少買進
  • 散戶對策：在580元附近VWAP回測時跟隨法人力道做多

心得

VWAP不是單純的技術指標，它的本質是「流動性結構」，代表主流資金的真實平均成本。這跟RSI、MACD這類振盪指標完全不同——VWAP告訴你的是「錢在哪裡」，而不是「超買超賣」。

最讓我印象深刻的是Anchored VWAP這個概念。傳統VWAP固定從開盤計算，在跳空或重大事件後會「失真」，但Anchored VWAP允許從任意時間點開始計算，直接捕捉事件後的資金真實成本。這對於Fed決策日、CPI公布日、財報公告日這種高波動情境，根本是自帶透視鏡。

另外，法人VWAP執行邏輯也給了我一個重要啟發：當你在VWAP附近進場，本質上你是在跟法人站在同一條成本線上。這不是什麼玄學，是簡單的經濟學——法人要控制執行成本，自然會在VWAP附近形成支撐壓力。理解這個邏輯，比單純看線圖有效得多。

最後要注意VWAP的局限性：趨勢爆發盤和窒息盤中VWAP會失效。沒有任何指標是萬能的，VWAP也不例外。正確的做法是先判斷盤勢屬性，再決定是否使用VWAP，而不是無腦套用。

一、ICT 2022 交易模型：四條件進場系統

是什麼：ICT（Inner Circle Trader）開發的一套系統化交易模型，原本設計給新手快速上手，結合流動性、市場結構、失衡區與時間四大條件，在高勝率時機進場。

四個條件怎麼判斷：

1. 流動性獵取（Liquidity Raid/Purge）

  • 價格突破關鍵高點或低點後快速拉回

2. 市場結構轉變 CHoCH（Change of Character）

  • 下降趨勢中，前一個高點被突破 → 空翻多

3. 失衡區 FVG（Fair Value Gap）

  • 價格快速移動時留下的三根 K 線不重疊區域

4. Killzone 時間

  • 亞洲時段：夏令 08:00-12:00 / 冬令 09:00-13:00（容易盤整，用來觀察）

進場公式：

  • 進場點：FVG 上緣（做空）或下緣（做多）
  • 止損點：流動性獵取的極值點（最高點上方 / 最低點下方）
  • 止盈點：下一個流動性點位或下一個 FVG
  • 目標賺賠比：至少 2:1

實戰流程：
1. 在 Killzone 時段觀察
2. 發現流動性獵取 → 等待
3. 出現 CHoCH 確認趨勢轉變 → 等待
4. 出現 FVG → 在 FVG 邊緣進場
5. 止損設在極值點，止盈看下一個流動性位

適用時機：

  • ✅ 有明確趨勢且出現流動性獵取的市場
  • ✅ Killzone 時段內發生的訊號
  • ❌ 亞洲時段的盤整行情（流動性不足）
  • ❌ 沒有 CHoCH 就貿然進場

二、OB 訂單塊進階：高品質 OB 三濾網

是什麼：訂單塊（Order Block）是機構投資者的成本區，即市場突破前的最後幾根反向 K 線。當價格回到 OB 區域，聰明錢很可能再次進場。

看漲 OB：突破前的最後幾根下跌 K 線（機構買入區）
看跌 OB：跌破前的最後一根上漲 K 線（機構賣出區）

三個濾網判斷高品質 OB：

濾網 1：需要突破（BOS/CHoCH）

  • OB 出現後必須伴隨強勢突破，確認機構已進場
  • 突破越強，OB 效力越大
  • 沒有突破的 OB 不算數

濾網 2：出現 FVG

  • OB + FVG 同時出現 = 雙重確認聰明錢進場
  • OB 是成本區，FVG 是未成交訂單區
  • 兩者共振 = 高品質進場點

濾網 3：流動性獵取

  • 價格先獵取散戶止損 → 快速反轉 → 形成高品質 OB
  • 結合濾網 1+2+3 = 最強進場訊號

OB 交易策略三步驟：
1. 觀察市場反應：價格回到 OB 後，觀察是否有反彈跡象（引線、小時間框架 CHoCH）
2. 設定止損：設在 OB 最外側（加一點彈性空間），被突破就沒理由繼續抱單
3. 設定止盈：保守看最近流動性點位（前高/前低），積極看更大時間框架的 DOL

OB 陷阱識別：

  • 前一個 OB 還沒被測試，就出現新的 OB → 新 OB 可能是陷阱
  • 這種「假 OB」其實是流動性陷阱，聰明錢會先獵取再推動價格
  • 實例：下跌中產生 OB-1 和 OB-2，OB-1 還沒回測 → OB-2 可能是陷阱 → 價格突破 OB-2 上緣後才繼續下跌

OB 三種畫法：
1. 突破前最後一根反向 K 棒（最精簡）
2. 突破前最後幾根連續反向 K 棒（包含整個區域）
3. 突破前所有 K 棒都包含進去（最寬鬆）

  • 關鍵：關注 OB 中點，如果有效 OB，價格通常不會突破中點

推進塊（Mitigation Block）：

  • 當價格回到 OB 並再次出現 BOS 形成新 OB → 疊加的 OB 稱為推進塊
  • 代表機構持續加碼的跡象，是多頭/空頭力道強烈的訊號

心得

ICT 2022 模型最讓我印象深刻的是它的「四條件全部滿足才進場」邏輯。很多人（包括我之前）看到一個 FVG 或 OB 就急著進場，結果被打停損。這個模型強制要求你等待：先有流動性獵取（確認聰明錢在動），再有 CHoCH（確認趨勢轉變），再有 FVG（提供好價格），最後還要在對的時間（Killzone）。四個條件缺一不可，這就是過濾假訊號的關鍵。

OB 三濾網也是同樣的道理——不是每個 OB 都有效。最常見的錯誤就是把「還沒被測試的 OB」當成有效 OB，結果進場後才發現那是聰明錢設的流動性陷阱。以後看到 OB，我會先問三個問題：有突破嗎？有 FVG 嗎？有流動性獵取嗎？三個都通過才算高品質。

台股的 Killzone 對應亞洲時段，開盤 30 分鐘和收盤前是流動性最活躍的時段。搭配 ICT 2022 模型，我會特別在 09:00-09:30 和 13:00-13:30 觀察流動性獵取和 CHoCH 的出現。
