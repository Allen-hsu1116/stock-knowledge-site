調整時機：何時該動作？

黃金法則：

  • 股價距離短邊履約價 3% 以內，或
  • 短邊 Delta 觸及 25

實例：

  • RUT 上漲，短 Call 距離 5.01%，Delta 15 → 不急著調整
  • 短 Put 距離 9.47%，Delta 更低 → 先觀察
  • 若短邊 Delta 達 25 或距離 <3% → 執行調整

調整策略一：Roll Up / Roll Down（攻擊型）

是什麼：將獲利邊 roll up/down，收割部分利潤後重新建立更窄的 Iron Condor

怎麼做：
1. 股價上漲時 Roll Up Put 價差：

  • Buy to close 現有 Put 價差（獲利入袋）

2. 股價下跌時 Roll Down Call 價差：

  • Buy to close 現有 Call 價差（獲利入袋）

實例：

  • PLTR 上漲，Put Spread 因 Delta 降低獲利，Call Spread 因 Delta 上升虧損
  • 執行 Roll Up：將 Put Spread 從 980-1000 移到 1010-1030
  • 結果：Iron Condor 寬度變窄，Delta 從 -45 降到 -34，額外收入 $550 權利金

優點：

  • 收入增加（額外權利金）
  • Delta 降低

缺點：

  • 新的短邊更靠近價平，反向波動風險增加
  • 若持續單邊走势，仍會虧損

適用時機：

  • 交易初期
  • 風險參數還在可控範圍
  • 波動率環境穩定

調整策略二：增加非威脅邊的合約數（攻擊型）

是什麼：在獲利邊增加更多合約，提高收入同時降低 Delta

怎麼做：
1. 股價上漲威脅 Call 邊 → 在 Put 邊增加合約數
2. 從 10 口增加到 20 口（達到完整配置）
3. 確保新加的價差賣價 > $1（值得承擔額外風險）

實例：

  • RUT 上漲，Put 邊獲利
  • 增加 10 口 Put Spread
  • Delta 從 -45 降到更中性，Theta 從 82 升到 112
  • 獲利潛力增加 $1,350

優點：

  • Delta 大幅降低
  • 收入潛力增加最多

缺點：

  • 下檔風險翻倍
  • Vega 暴露增加（雖然 Vega/Theta 比率略降）

適用時機：

  • 交易初期，正在建立完整配置
  • 對方向有信心時

調整策略三：加買 debit spread（防守型）

是什麼：在受威脅邊加買 debit spread（call spread），形成「鼠耳」獲利區

怎麼做：
1. 股價上漲威脅 Call 邊 → Buy bull call spread
2. 使用短邊履約價作為 debit spread 的短邊
3. 選擇價外 2-3 個履約價作為長邊

實例：

  • RUT 上漲，Short Call 在 1240
  • 買入 1240/1260 Bull Call Spread
  • 成本 $440，Delta 從 -45 降到 -24
  • Vega 從 -524 降到 -454

優點：

  • Delta 和 Vega 都降低
  • 不移動短邊履約價
  • 創造獲利區間

缺點：

  • 花錢買 debit spread，降低收入潛力
  • 若股價進入鼠耳區但離到期還有 >2-3 天，仍會虧損
  • 不要被獲利區圖形誤導

適用時機：

  • 市場過度延伸，回調機率高
  • 泡動率低（debit spread 較便宜）
  • 不想增加下檔風險時

調整策略四：加買 Long Call/Put（防守型）

是什麼：直接買入 long call（或 long put）避險

怎麼做：
1. 股價上漲威脅 Call 邊 → Buy long call
2. 選擇同到期日或下個月到期日
3. 履約價接近短邊或略為價外

實例 A（同月到期）：

  • 買入與 Short Call 同履約價的 Long Call
  • 成本 $960，Delta 從 -45 降到 -22
  • Vega 大幅降低

實例 B（下月到期，更價外）：

  • 買入下月更價外的 Long Call
  • 成本 $550（較便宜）
  • 保護效果較弱

優點：

  • Vega 顯著降低（對波動率上升有保護）
  • Delta 快速降低
  • 若持續走勢，獲利圖形漂亮

缺點：

  • 成本較高
  • 時間價值會衰減

適用時機：

  • 市場快速移動，需要緊急降低 Delta
  • 泡動率低時（long option 便宜）
  • 需要時間思考下一步

調整策略五：Calendar Spread（防守型）

是什麼：在短邊履約價加賣下個月的同一履約價選擇權

怎麼做：
1. 識別受威脅的短邊履約價
2. Buy 下個月同履約價的 Call/Put
3. Sell 當月同履約價的 Call/Put（形成 calendar）
