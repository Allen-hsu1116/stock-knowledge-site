外資操控台股的四大工具

1. 期貨 - 多空雙向操作工具
2. 選擇權 - 多空雙向操作工具
3. 現貨 - 只能作多的工具
4. 借券賣出 - 只能做空的工具

借券賣出是投資人平常比較少注意的指標，但當外資開始偏空操作時，借券賣出餘額會持續增加。

借券相關名詞釐清

四個關鍵名詞

名詞定義比喻
**借券成交**今日投資人從市場上借來的券之數量今日跟市場借來的子彈
**借券餘額**每日借券數量的累積量，扣除已歸還的券借子彈的累積量，不需要的會還回去
**借券賣出**今日確實把借來的券賣出了今日把子彈打出去的數量
**借券賣出餘額**每日借券賣出的累積量，扣除已回補的券把子彈打出去的累積量

借券餘額 vs 借券賣出餘額

重點：借券餘額 ≠ 借券賣出餘額

  • 借券賣出餘額更重要：因為是實際賣出的數量
  • 借券人借入股票，不一定馬上賣出
  • 兩者走勢通常高度相關

搭配外資現貨買賣超的進階判讀

情境一：借券賣出增加 + 現貨買超 = 避險、不看好多方

解讀：外資左手買進、右手偷偷布局空單

  • 不看好多方行情

  • 兩隻手動作同向
  • 大行情即將出現
  • 做多的朋友要盡快撤退

操作：手上有股票應減碼，更積極的投資人可考慮放空。

借券不是只有放空目的

重要提醒：借券餘額提高，不絕對表示法人想做空！

可能原因：

  • 避險需求
  • 套利操作
  • 融資性借券

結論：單一指標做進出依據期望值不理想，需搭配其他空方指標。

法人不能放空的規定

法律規定：法人不能放空，只能做多頂多保留現金。

借券用途：法人可以利用「借券」達到放空目的。

空頭走勢時：防禦性股票（如中華電2412）會有資金流入。

注意事項

剛開放借券的股票

失真情況：剛開放借券時，基準點都是0，無法判斷借券賣出餘額是增加或減少。

對策：剛開放借券的股票，借券解讀會失真，要特別留意。

心得

借券賣出餘額是一個很強的「領先指標」。外資擁有第一手資訊，當他們提前用借券賣出布局空單時，往往在新聞出來之前股價就已經開始跌了。

關鍵是：借券賣出餘額增加 + 外資同步賣超 = 危險訊號。這代表外資兩手同向做空，大行情要來了。

但如果借券賣出增加卻搭配外資買超，可能只是避險，不用太緊張。

實戰上，可以用「外資連3日賣超 + 股價跌破季線 + 借券賣出連3日增加」當作空方選股條件，找出外資正在狙擊的股票清單。
