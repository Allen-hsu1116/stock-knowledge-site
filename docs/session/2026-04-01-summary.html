2. 融資融券實戰

融資（借錢買股）：

  • 上市：自備40%可買100%
  • 上櫃：自備50%可買100%
  • 槓桿約2.5倍，年利息6-7%

融券（借股賣出）：

  • 自備90%保證金
  • 最大虧損無上限
  • 風險：軋空、強制回補

維持率計算：

維持率 = 股票市值 ÷ 借款金額 × 100%
安全線：166%以上
追繳線：130%（低於此數字會被追繳）
斷頭線：低於130%，T+3強制賣出

券資比判斷：

券資比 = 融券餘額 ÷ 融資餘額 × 100%
< 10%：正常範圍
10-30%：需要注意
> 30%：高危險區，軋空風險大

3. 千張大戶持股分析

持股比例判斷：

比例意義
< 40%籌碼太分散，不利炒作
40%-70%最佳區間，易漲難跌
> 80%太集中，不易炒出行情

千張大戶 vs 散戶關係：

  • 千張持股上升 + 總股東人數下降 → 籌碼集中，主力吃貨
  • 千張持股下降 + 總股東人數上升 → 籌碼分散，主力出貨

4. 外資期貨與選擇權未平倉

大外資 vs 小外資：

類型特性操作意義
大外資部位大，無法短線進出看「中期趨勢」（1-3個月）
小外資部位小，操作靈活看「短線變化」（1-2週）

判斷邏輯：

  • 大外資淨多單 + 小外資翻多 → 行情偏多
  • 大小外資都翻空 → 大行情反轉警訊

選擇權未平倉解讀：

  • Call OI 增加 → 賣方看「不漲」，該履約價為壓力區
  • Put OI 增加 → 賣方看「不跌」，該履約價為支撐區

5. 庫藏股分析

公司買回庫藏股三大理由：
1. 轉讓股份給員工
2. 作為特別股或公司債轉換來源
3. 維護公司信用及股東權益（護盤）

判斷庫藏股利多還是利空：

情境執行目的效果
過去半年漲幅 >10%轉讓員工通常較好
過去半年跌幅 >40%護盤可能反彈
執行後1個月仍跌 >5%—警訊！未來2年難好

執行率是關鍵：

  • 執行率 = 實際買回 ÷ 預定買回
  • 執行率差（<50%）→ 可能只是「口水護盤」
  • 執行率高（>80%）→ 公司真的在買

6. 融資使用率與斷頭風險

融資使用率：

融資使用率 = 實際融資餘額 ÷（流通股數 × 25%）

風險分級：

使用率風險判斷
0-20%低風險籌碼健康

50-80%高風險下跌10%可能引發斷頭潮
> 80%極高風險人踩人行情隨時可能發生

斷頭價計算：

  • 融資買進100元，自備40元，借款60元
  • 斷頭線 = 60 × 1.3 = 78元
  • 下跌幅度 = (100 - 78) ÷ 100 = 22%

9. 分點籌碼追蹤

關鍵分點追蹤法：
1. 觀察某分點的歷史績效
2. 記錄這個分點在哪裡買進、哪裡賣出
3. 如果低檔買進、高檔賣出 → 關鍵分點
4. 當關鍵分點開始買進 → 考慮跟進

地緣券商判讀：

  • 公司地址附近的券商分點
  • 可能是公司派（董監事、大股東）的下單管道
  • 地緣券商在低檔買進 → 可能有內線

隔日沖券商名單：

  • 凱基-松山、元大-土城永寧
  • 國票-敦北法人、富國-建邦
  • 開盤4%為分水嶺，高於4%可續抱，低於4%先出

10. 借券賣出餘額判讀

借券餘額 vs 借券賣出餘額：

  • 借券餘額 = 借入但未賣出的股票
  • 借券賣出餘額更重要 = 實際賣出的數量

判讀方法：

  • 借券賣出餘額上升 → 看空者增加，股價可能下跌
  • 借券賣出餘額下降 → 空方回補，股價可能上漲

搭配外資現貨買賣超：

  • 借券賣出增加 + 現貨買超 = 避險、不看好多方
  • 借券賣出增加 + 現貨賣超 = 大行情要來（危險）

14. 主力洗盤手法

短線拉回洗盤：

  • 第一防守線：長紅K線最低點
  • 第二防守線：前一波低點或重要支撐
  • 判斷：不跌破第一防守線 → 強勢整理

K線洗盤（避雷針洗盤）：

  • 出現長上影線（看似避雷針）
  • 守住第一防守線
  • 後續突破上影線高點 → 洗盤完成

起漲前甩轎：

  • 最兇猛的洗盤方式
  • 長黑跌破盤整區
  • 隨即止穩不續跌
  • 長紅吞噬長黑最高點 → 確認甩轎

應對心法：

  • 不要賭它是洗盤，先設停損
  • 洗盤後的買點是「突破前高」
  • 真正的洗盤不會跌破第二防守線

18. 券資比進階應用

券資比公式：

券資比 = 融券餘額 ÷ 融資餘額 × 100%

判斷標準：

  • 券資比 > 30%：被大量放空，市場看空情緒濃
  • 券資比突然暴增：需特別關注，可能是軋空前兆

軋空行情識別：
1. 股價上漲 + 融券增加 = 散戶放空、主力拉抬
2. 股價下跌 + 融資增加 = 散戶接刀、主力出貨
3. 股價低檔 + 融資暴增 = 可能主力進場
4. 股價高檔 + 融券暴增 = 可能主力放空
