營業利益率分析

第 21 回合學習 - 營業利益率分析

時間：15:49
文章：

  • 營業利益率的解讀與應用｜EBC金融集團

核心知識

營業利益率（Operating Profit Margin）

是什麼：企業扣除「直接成本」和「間接成本（營業費用）」後，真正從本業賺到的利潤佔營收的比例。是衡量「本業獲利能力」的核心指標。

公式：

營業利益率 = 營業利益 ÷ 營業收入 × 100%

怎麼算：

  • 營業利益 = 營業收入 - 銷貨成本 - 營業費用
  • 營業費用包括：管理費用、行銷費用、研發費用等

實例：

  • 某電子公司營收 300 萬，營業利益 60 萬
  • 營業利益率 = 60 ÷ 300 × 100% = 20%
  • 意義：每賺 100 元營收，本業淨賺 20 元

損益表三率比較

指標扣除項目反映什麼
毛利率只扣直接成本（原料、製造成本）生產效率、議價能力
營業利益率扣直接成本 + 營業費用（薪資、租金、行銷）本業經營能力
稅後淨利率扣所有成本 + 稅最終真實獲利

為什麼要看營業利益率，不能只看毛利率？

因為毛利率沒算到營業費用。一家公司可能毛利高，但砸大錢打廣告、請太多人，最後本業反而沒賺錢。

實例：

  • A 公司：毛利率 40%，但營業費用佔 35%，營業利益率只剩 5%
  • B 公司：毛利率 25%，營業費用控制好只佔 10%，營業利益率 15%
  • B 公司的本業獲利能力反而更好

營業利益率判斷標準

好壞怎麼看：
1. 跟同業比：不同產業差異很大，不能跨業比較
2. 跟自己比：看趨勢，上升好、下降要警覺
3. 跟標準比：一般來說 >10% 算不錯，>20% 算優秀

選股實戰應用

「連續高營業利益率」選股策略：
1. 連續 5 季以上營業利益率 > 20%
2. 或連續 7 季以上營業利益率 > 24%
3. 篩選出的公司具備穩定的本業獲利能力

進階分析技巧：

毛利率 vs 營業利益率的變化：

  • 毛利率升 + 營業利益率降 → 產品競爭力強，但管理有問題（費用失控）
  • 毛利率降 + 營業利益率升 → 管理改善，彌補了產品競爭力下滑

營業利益率 vs 淨利率的差異：

  • 營業利益率高 + 淨利率低 → 可能有高利息支出、業外虧損、稅負問題
  • 要查財報看「業外損益」和「利息費用」

營業利益率不是越高越好

過高的風險：
1. 可能壓低成本 → 產品品質下降、客戶流失
2. 可能砍研發 → 長期競爭力衰退
3. 可能過度集權 → 企業彈性不足
4. 吸引競爭者 → 價格戰壓縮利潤

負數代表什麼：

  • 營業利益率 < 0 → 本業虧損
  • 這種公司沒有投資價值，除非是轉機股或成長股（燒錢換成長）

完整分析步驟

Step 1：計算營業利益率

  • 從損益表找：營業收入、營業利益
  • 公式：營業利益率 = 營業利益 ÷ 營業收入 × 100%

Step 2：產業對比

  • 找同業平均
  • 高於平均 = 經營效率好
  • 低於平均 = 要找原因

Step 3：趨勢分析

  • 看 5~8 季的變化
  • 上升趨勢好、下降趨勢要警覺
  • 單季變化要查是否有一次性因素

Step 4：三率交叉驗證

  • 毛利率 → 看產品力
  • 營業利益率 → 看本業經營能力
  • 淨利率 → 看最終真實獲利
  • 三率之間的變化關係，會說出公司的故事

三率三升

是什麼：毛利率、營業利益率、淨利率同時上升。代表公司從產品到管理到最終獲利都在改善，是基本面轉好的強訊號。

實例（鴻海）：

  • 毛利率從 6% 升到 8%
  • 營業利益率從 2% 升到 4%
  • 淨利率從 1.5% 升到 3%
  • 三率同時上升 → 代表產品組合優化、費用控制得當、業外也改善

心得

營業利益率是損益表裡最被低估的指標。很多人只看 EPS、淨利，但忽略了本業到底賺不賺錢。

毛利率高不代表真的好，可能費用一堆。淨利率高也可能是賣土地、賣股票賺的。真正要看的，是營業利益率——這才是一家公司「本業」真正的賺錢能力。

用「連續高營業利益率」選股，比單純看 EPS 更能抓到穩定獲利的公司。EPS 可能被業外影響，營業利益率騙不了人。

之後看財報，三率要一起看：

  • 毛利率 → 產品競爭力
  • 營業利益率 → 經營能力
  • 淨利率 → 最終真實獲利

三率之間的變化關係，才是真正會說話的地方。

核心知識

本益成長比（PEG Ratio）

是什麼：用來衡量「成長型股票是否昂貴或便宜」的指標，將公司未來獲利成長性納入考量，解決本益比無法反映成長性的問題。

公式：

PEG = 本益比（PE）÷ 盈餘成長率（%）

怎麼算：
1. 查詢個股本益比（PE）
2. 預估未來稅後淨利成長率（建議用5年平均或預估）
3. 本益比除以成長率（百分比直接除，不用換算小數）

實例：

  • 情境：A公司股價100元，預估EPS為10元，預估稅後淨利成長率5%
  • 計算：本益比 = 100 ÷ 10 = 10 倍
  • PEG = 10 ÷ 5 = 2
  • 判斷：PEG > 1.2，股價高估

PEG 判斷標準

PEG 數值股價合理性操作建議
> 1.2 倍高估賣出
= 1 倍合理觀望
< 0.75 倍低估買進
< 0.66 倍嚴重低估強力買進

適用時機：

  • ✅ 用於評估「成長股」是否被低估
  • ✅ 公司本益比為正、EPS 成長中
  • ✅ 盈餘成長率用長期數據（5年以上）估算
  • ❌ 不適用於衰退產業或獲利不穩定的公司

PEG vs 本益比（PE）

指標用途差異
本益比（PE）評估股價便宜或昂貴未考慮成長性
本益成長比（PEG）評估成長股便宜或昂貴將成長性納入考量

為什麼成長股要用 PEG：

  • 成長股本益比通常偏高（20-50倍）
  • 單看 PE 會覺得股價昂貴，錯失買點
  • PEG 將成長率納入考量，更能反映真實價值

PEG 選股四大陷阱

1. PEG 不是越小越好

  • PEG 極小可能代表公司獲利起伏大
  • 這類公司股價波動劇烈，風險高
  • 要看獲利是否穩定成長

2. 只能用於成長股

  • 前提：公司必須是成長股
  • 本益比為正、EPS 成長中
  • 用在衰退股會誤判

3. 可能選到衰退公司

  • 歷史成長率不代表未來
  • 預估盈餘成長率很困難
  • 需要持續追蹤公司基本面變化

4. 可能錯失投資機會

  • 台股 PEG < 0.75 或 < 0.66 的時機不多
  • 堅持用 PEG 選股可能錯過優良標的
  • 建議搭配其他指標使用

實戰應用步驟

Step 1：篩選成長股

  • EPS 連續3年以上成長
  • 營收年增率 > 10%
  • 產業前景看好

Step 2：計算 PEG

  • 查詢本益比（Goodinfo!台灣股市資訊網）
  • 預估未來5年平均盈餘成長率
  • 計算 PEG = PE ÷ 成長率%

Step 3：綜合判斷

  • PEG < 0.75：買進訊號
  • 同時看本益比是否合理（< 20倍較安全）
  • 檢查財報是否健康（負債比、現金流）

Step 4：持續追蹤

  • 每季檢視盈餘成長率是否變化
  • 成長率下滑要重新評估 PEG
  • 設定停損點

PEG 查詢管道

台股：

  • Goodinfo! 台灣股市資訊網（查本益比，自行估算成長率）
  • 個股財務報表（損益表看稅後淨利成長率）

美股：

  • Morningstar（Morningstar.com）
  • Yahoo Finance

心得

PEG 這個指標我之前有聽過，但沒深入了解它的判斷標準。原來 < 0.75 才算低估，這個數字比我想像的更嚴格。而且關鍵是「預估未來盈餘成長率」——這才是最難的部分，因為沒人能準確預測未來。

作者說得對：本益比假定「未來獲利相同」，PEG 假定「未來成長率相同」，但這兩個假設都不是現實。所以 PEG 更像是一個思考框架，而不是精準的計算公式。

實戰上，我會：
1. 先確認公司是成長股（EPS 連續成長）
2. 用過去5年平均成長率作為保守預估
3. PEG < 0.75 才考慮買進
4. 搭配本益比、現金流一起看，不單用一個指標

PEG 在台股適用時機不多（低於0.75的機會少），所以把它當成「確認訊號」而非「主要選股工具」比較實際。看到好公司時，再用 PEG 確認一下是否有便宜可撿。
