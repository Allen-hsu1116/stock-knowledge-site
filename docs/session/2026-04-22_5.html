籌碼面分析：自營商避險部位判讀×假外資識別×法人買超股價背離

三大法人裡面，外資和投信的籌碼分析已經很成熟了，但自營商這塊一直是散戶最容易踩雷的區域。看到自營商大買就追進去，結果隔天被倒貨；看到外資買超中小型股就以為有行情，結果是假外資隔日沖。這些坑，今天一個一個填起來。

一、自營商避險部位：最容易被誤讀的籌碼

是什麼

自營商的買賣超分為兩種：自行買賣和避險部位。從2014年12月起，證交所把這兩種分開公告，但很多網站和App仍然合併顯示，這是散戶踩雷的最大來源。

自行買賣：自營交易員基於自身判斷進出的部位，代表券商對個股的看法。
避險部位：券商發行權證後，為了避免權證部位的風險暴露，被迫在現貨市場買進（認購權證）或賣出（認售權證）對應股票。這不是自營商自己想要買的，是被迫買的。

為什麼自營商要避險

邏輯很簡單：
1. 券商發行認購權證 → 投資人買進權證（看多）
2. 如果股票上漲 → 投資人賺錢，券商賠錢
3. 券商為了不賠錢 → 買進同價值的現股避險
4. 這樣股票漲 → 權證賠的部分被現股賺的部分Cover掉
5. 券商穩賺權利金和時間價值

關鍵：避險買盤是「被迫的」，不代表自營商看好這檔股票。

怎麼做 — 辨識自營商避險 vs 自行買賣

1. 查詢證交所T86報表：分別看「自行買賣」和「避險」兩欄
2. 避險部位大增 = 隔日沖訊號：

  • 避險部位大量買進 → 代表有權證主力大量買進認購權證

3. 自行買賣部位才有參考價值：

  • 自行買賣連續3天買超 → 可能值得關注

實例：

  • 某股收盤後公告：自營商避險買超800張，自行買賣賣超50張
  • 隔天開盤：自營商避險賣超750張 → 開高走低
  • 如果你只看「自營商合計買超750張」就進場 → 隔天被倒貨

自營商避險 = 隔日沖券商的一種

權證主力可以利用自營商的避險機制：
1. 大量買進某股的認購權證
2. 自營商被迫在現股買進避險 → 推升股價
3. 權證主力趁股價拉高時賣出權證和現股
4. 隔天自營商賣出避險部位 → 股價回落

怎麼做 — 避開自營商避險地雷：
1. 收盤後查個股的權證成交明細 → 如果有人大量買進權證 → 當天的自營商買超很可能是避險
2. 隔天開盤9:00-9:30是自營商避險出脫的高峰期 → 不要在早盤追高
3. 如果要買這檔股票，等到9:30之後再決定 → 等避險賣壓消化完畢

二、自營商自行買賣部位：短線高手的進出紀錄

自營商的交易特徵

特徵說明
資金規模三大法人中最小，單家5-50億
交易員人數每家十幾到幾十位
單一交易員部位2,000萬～數億元
操作風格極短線，當沖、隔日沖為主
風控嚴格單筆虧損超過上限就砍，不戀戰
追求目標絕對報酬（不管市況好壞都要賺）
投資標的大中小型股都有，偏好中小型波動大的

自營商的風控機制

每個交易室都有嚴格的停損規範：

  • 總虧損不能超過10%
  • 單筆交易有最大虧損限制
  • 風控金額一到 → 交易員無法繼續交易，嚴重的丟飯碗

這導致的結果：自營商的交易風格非常短線，行情一有風吹草動就砍部位。散戶看到的數據永遠是落後的——今天看到買超，可能明天就賣超了。

自營商參考價值評估

部位類型參考價值原因
自行買賣-連續買超★★★代表自營商看好，但週期極短
自行買賣-單日買超★可能只是當沖，隔天就反轉
避險部位-買超★★★★★（反向）隔天必賣，是反向指標
避險部位-賣超★★★（反向）認售權證被大量買進

結論：自營商的避險部位比自行買賣部位更有參考價值，但用法是反向使用——避險大買 = 隔天賣壓，不要跟進。

三、假外資識別：藏在法人群體裡的隔日沖

是什麼

「假外資」= 本土資金透過外資券商下單，在三大法人統計中歸類為「外資」，但操作風格完全是隔日沖或短線炒作。

為什麼要用外資身分：
1. 外資的交易稅率較低（某些情況下）
2. 透過外資券商可以隱藏真實身分
3. 利用市場對「外資買超」的過度反應來套利

怎麼做 — 假外資辨識三法

方法一：看股本大小

  • 外資買超的標的如果股本<50億 → 高機率是假外資
  • 真正的海外共同基金、ETF偏好大型權值股（台積電、聯發科等）
  • 中小型股的外資買超 → 多數是假外資或避險基金短線操作

方法二：看操作模式

  • 真外資：連續數週到數月同方向買超，持股穩定累積
  • 假外資：一天大買、隔天大賣，進出頻繁
  • 在個股的法人買賣超歷史中，如果外資常出現「隔日沖」模式 → 假外資機率極高

方法三：看外資券商分點

  • 透過外資券商分點可以追蹤是哪一家外資在下單
  • 常見的假外資特徵：某特定外資券商分點反覆在個股進行隔日沖
  • 真正的海外基金通常透過大摩、高盛、瑞銀等大型外資券商

實例：

  • 某中小型股（股本30億）外資連續2天買超2,000張
  • 股價從50元漲到55元（+10%）
  • 第3天外資賣超2,500張 → 股價跌到48元
  • 這就是典型的假外資隔日沖：利用散戶看到「外資買超」就追進的慣性來套利

假外資應對策略

1. 看到外資買超中小型股 → 先查歷史進出紀錄：這檔股票的外資過去有沒有頻繁隔日沖的紀錄
2. 外資買超+股本<50億 → 保守看待：除非投信也同步買超（土洋同買更可信）
3. 外資買超+隔天就賣超 → 直接當隔日沖處理：不要跟單
4. 外資買超+股本>500億 → 比較可信：大型股的外資買超通常是真外資

四、法人買超股價背離：籌碼面最危險的訊號

是什麼

法人買超股價背離 = 法人持續買超，但股價不漲甚至下跌。這是籌碼面分析中最反直覺但也最重要的訊號之一。

三種背離情境

情境一：外資買超 + 股價不漲

  • 可能是避險基金在做多現股+做空期貨的套利
  • 也可能是外資在不同標的間輪動（賣A買B）
  • 判斷方式：查外資期貨部位。期貨空單增加+現貨買超 = 避險，非真看多

情境二：投信買超 + 股價下跌

  • 可能是投信在底部抄底，但市場賣壓更大
  • 也可能是投信被迫買進（基金有最低持股比例要求）
  • 判斷方式：看投信持股比例。如果已經超過10%還在買 → 可能在攤平；如果是4-6%開始買 → 比較像是抄底布局

情境三：三大法人同買 + 股價不漲

  • 這是最危險的訊號
  • 法人都在買但股價不動 → 代表有更大的賣壓在消化法人的買盤
  • 可能是公司派出貨、大戶調節、或主力出貨
  • 判斷方式：查分點進出和千張大戶持股。如果法人買超但千張大戶減少 → 大戶趁法人接盤時出場

怎麼做 — 背離判讀SOP

1. 出現法人買超但股價連續3天不漲 → 進入警戒
2. 查對應的籌碼數據：

  • 外資買超不漲 → 查期貨部位、查股本大小（假外資？）

3. 背離持續5天以上 → 大機率是陷阱

  • 法人買超是實實在在的金錢，但股價不漲代表供給（賣壓）大於需求（買盤）

4. 背離結束的方式：

  • 賣壓耗盡 → 股價突破向上（利好兌現）

實例：

  • 某股連續5天投信買超3,000張，但股價從100元跌到95元
  • 查千張大戶：大戶持股連續2週減少
  • 結論：大戶趁投信接盤出貨 → 投資人不宜跟單
  • 後續：投信轉為賣超，股價加速跌到80元

五、自營商×假外資×背離 整合判讀框架

籌碼面「偵訊」五問

每次看到法人買超，先問自己五個問題：

問題是的話否的話
1. 這是自行買賣還是避險？自行買賣才有意義避險 = 隔日沖訊號
2. 這是真外資還是假外資？股本>50億+連續買超較可信股本<50億+隔日沖=假外資
3. 股價有沒有跟著法人方向走？法人買超+股價漲=正常法人買超+股價不漲=背離警訊
4. 大戶和散戶怎麼動？大戶增+散戶減=籌碼健康大戶減+散戶增=籌碼惡化
5. 這個買超持續多久了？連續5天以上較可靠1-2天可能只是調節

實戰口訣

「自營商避險隔日賣，假外資隔日沖，法人買超不漲要看大戶」

三個最常見的籌碼面陷阱：
1. 自營商避險大買 → 隔天賣壓（不要跟）
2. 假外資買超中小型股 → 隔日沖出貨（不要跟）
3. 法人買超但股價不漲 → 大戶出貨中（不要接）

心得

自營商避險和假外資，一個是「被迫買進」，一個是「假裝買進」，兩者都會製造虛假的買盤信號，讓散戶以為有大資金進場而追進。這是籌碼面分析中最容易被忽略的兩個盲區。

幾個核心體悟：

1. 自營商避險是籌碼面的「假訊號源」——看到自營商大買，第一反應不應該是「券商看好」，而是「是不是權證避險？」。避險買盤100%隔天會賣，本質上就是隔日沖券商的變形。最有效的辨識方式是收盤後查個股的權證成交明細：如果有人大量買進認購權證，那當天的自營商買超就是避險，不是真買盤。

2. 假外資是台股特有的「偽裝術」——本土資金穿著外資的外衣，利用散戶對「外資買超」的過度信任來套利。辨識的關鍵在於「股本」和「操作模式」：真外資買大型股、連續性強；假外資買中小型股、隔日沖頻繁。以後看到外資買超中小型股，第一件事就是查這檔股票的外資歷史進出紀錄。

3. 法人買超股價背離是「籌碼面最危險的訊號」——因為它表面上看起來是利多（法人在買！），但實際上是有人在更大的力道倒貨。這種背離如果持續5天以上，大機率是陷阱。判斷的關鍵是看千張大戶和分點進出：如果法人買超但大戶在減碼，那就是大戶趁法人接盤時出場，千萬不要當那個接最後一棒的人。

籌碼面分析：鎖定鎖單效應×流通籌碼×新股蜜月期

鎖定鎖單效應是籌碼面中非常獨特又實戰性極強的概念——當一檔股票的流通籌碼被人為鎖住時，供需失衡會造成股價劇烈波動。搞懂鎖單效應、流通量與新股蜜月期的關係，才能在籌碼面分析中抓住別人看不到的機會。

一、鎖定鎖單效應（Lock-up Effect）

是什麼

鎖單效應是指：當大量持股因法令或約定無法在市場上賣出時，實際流通籌碼大幅減少，導致供需失衡——需求不變但供給銳減，股價容易大漲或劇烈波動。

鎖單的三種來源

1. 法定鎖單（IPO鎖定期）

  • 新上市（櫃）股票，董監事、大股東、員工持股依法須鎖定一段期間不能賣出
  • 台股規定：IPO後6個月內，董監事持股不得轉讓
  • 這段期間，真正在市場上流通的籌碼只是總股數的一小部分

2. 基金鎖單（人為鎖單）

  • 部分操守不佳的基金經理人與上市公司高層約定，在一定期間內不賣出持股
  • 這種鎖單是灰色地帶，屬於非法利益輸送
  • 早期台股市場較常見，現在監管趨嚴已大幅減少

3. 自我鎖單（策略性鎖單）

  • 大股東、公司派基於看好後市或護盤需要，主動不賣出
  • 這種鎖單沒有法規約束，隨時可能解鎖
  • 判斷方式：觀察內部人持股比率是否持續偏高

鎖單效應對股價的影響

怎麼做：
1. 計算實際流通量：浮動股數 = 流通在外股數 - 鎖定股數 - 限制性股票
2. 評估供需失衡程度：流通量越小，鎖單效應越強，股價越容易暴漲暴跌
3. 追蹤解鎖時間表：鎖定期結束前的1-2週是風險最高的時刻，大量籌碼即將釋出

實例：

  • 某新上市公司股本10億元（1萬張），IPO只釋出20%（2,000張），其餘80%被董監事和創投鎖定6個月
  • 市場上真正可交易的只有2,000張，買盤只要集中幾百張就能把股價推上去
  • 6個月解鎖後，8,000張籌碼湧入市場，供給暴增4倍，股價往往大跌

適用時機：

  • 新股上市櫃前5天（蜜月期）— 最經典的鎖單效應場景
  • 大股東申報轉讓前夕 — 鎖定籌碼即將解鎖
  • 公司實施庫藏股期間 — 庫藏股鎖定等同減少流通量

二、流通籌碼分析：四層股數結構

是什麼

理解鎖單效應的前提是搞清楚「股數」有四個層次，每層的數字差異就是鎖單效應的來源。

四層股數結構

層次名稱定義實戰意義
第一層授權股（額定股本）公司依法可發行的最大股數稀釋上限，未來增資空間
第二層已發行股數實際已發行的全部股數包含庫藏股，大於流通在外
第三層流通在外股數市場上所有流通的股數（扣除庫藏股）EPS、每股淨值的計算基礎
第四層浮動股數一般大眾可交易的股數真正影響供需的數字

關鍵公式

浮動股數 = 流通在外股數 - 大股東長期持股 - 限制性股票 - 鎖定股數

怎麼做：
1. 查財報附註：找額定股數、已發行股數、庫藏股
2. 查集保股權分散表：看千張大戶持股比率，估算大股東鎖定數量
3. 計算浮動比率：浮動股數 ÷ 流通在外股數，比率越低，鎖單效應越強
4. 比對歷史變化：浮動比率突然下降代表有人在鎖單收集籌碼

實例：

  • 台積電（2330）流通在外約2,593萬張，外資持股約70%，千張大戶集中度極高
  • 浮動比率約30-40%，真正在市場上每天交易的可能只有幾萬張

三、新股蜜月期籌碼特性

是什麼

新股上市櫃前5天沒有漲跌幅限制，加上鎖單效應導致流通籌碼極少，容易產生「蜜月期」行情。這是鎖單效應最極致的展現。

蜜月期三階段

第一階段：蜜月期（上市前5天）

  • 無漲跌幅限制，股價可能暴漲數倍
  • 流通籌碼極少，買盤集中就能推升
  • 承銷價通常低估，確保上市首日有價差

第二階段：穩定期（第6天起恢復漲跌幅）

  • 恢復10%漲跌幅限制
  • 鎖單效應仍在，但波動收斂
  • 觀察第一天有沒有留下長紅K棒

第三階段：解鎖期（6個月後）

  • 董監事鎖定解除，大量籌碼湧出
  • 供給暴增，股價通常面臨壓力
  • 這是蜜月期最危險的時刻

抽新股三原則

怎麼做：
1. 一抽：有價差的新股抽籤都參加（興櫃轉上市、現金增資）
2. 二賣：抽到股票後，一天不漲就賣
3. 三無關：賣出後這檔股票的漲跌跟你無關

實例：

  • 2014年力智（6719）承銷價589元，上市第一天收盤808元，最高價971元
  • 抽到籤的報酬率37%，第一天搶買的反而可能買在高點
  • 蜜月期結束後回歸基本面，很多新股半年後跌破承銷價

適用時機：

  • 新股申購抽籤 — 低風險套利機會
  • 蜜月期搶買 — 高風險高報酬，但需要嚴格停損
  • 解鎖期前放空 — 需注意融券回補風險

四、籌碼集中度六大指標（進階版）

是什麼

前面學過基礎的籌碼集中度，現在加入鎖單效應的維度，讓集中度判讀更精準。

六大指標分類

絕對指標（先天優勢）：

1. 千張大戶持股比率

  • 公式：持有1,000張以上股東持股合計 ÷ 股本

2. 內部人持股比率

  • 公式：內部人持股合計 ÷ 股本

3. 佔股本比重

  • 公式：（區間買超前15名合計 - 區間賣超前15名合計） ÷ 股本

相對指標（趨勢追蹤）：

4. 籌碼集中度

  • 公式：主力買賣超張數 ÷ 區間成交量

5. 主力買賣超

  • 公式：每日買超前15名合計 - 每日賣超前15名合計

6. 買賣家數差

  • 公式：今日買進券商家數 - 今日賣出券商家數

鎖單效應×集中度實戰組合

怎麼做：
1. 先看絕對指標確認「先天體質」（千張大戶比率 + 內部人比率）
2. 再看相對指標確認「趨勢方向」（集中度 + 主力買超 + 買賣家數差）
3. 最後計算浮動比率確認「鎖單程度」（流通量越少，效應越強）

實例：

  • 某中小型股千張大戶比率55%（中等）、內部人持股25%（適中）
  • 60天買超佔股本4%（接近5%門檻）、籌碼集中度7%（超過5%門檻）
  • 買賣家數差持續為負（多數人賣給少數人）
  • 結論：先天體質中等，但趨勢方向明確朝集中，浮動籌碼正在被收集

適用時機：

  • 選股初篩 — 先看絕對指標過濾
  • 進場確認 — 相對指標趨勢向上才買
  • 出場判斷 — 相對指標反轉即賣

心得

鎖單效應是籌碼面分析中最被低估的概念之一。大多數人只看法人買賣超，卻忽略了「真正在市場上流通的籌碼有多少」這個根本問題。

一檔股本10億的股票，如果80%被鎖定，實際流通量只有2億。一個法人買超5,000萬，在2億流通量裡是25%的影響力，但在10億總量裡只有5%。同一筆買超，效應差了5倍。

新股蜜月期就是鎖單效應的極致展演——流通籌碼極少 + 無漲跌幅限制 + 承銷價低估 = 暴漲幾乎必然。但解鎖期一到，籌碼供給暴增，蜜月變噩夢。

今天的學習讓我把之前學的千張大戶、籌碼集中度、主力買賷超全部串起來了——先看鎖單結構決定效應強度，再看集中度趨勢決定進出場時機。這才是籌碼面分析的完整框架。
