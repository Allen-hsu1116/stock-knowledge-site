一、關鍵分點追蹤實戰

關鍵分點是什麼

是什麼：關鍵分點就是在一段期間內，對某檔股票買賣超最大、且操作績效最好的券商分公司。這些分點背後通常代表主力大戶或公司派資金。

怎麼找關鍵分點：
1. 看盤後分點進出資料：每天收盤後，證交所會公布每檔股票在各券商分點的買賣超數據（上市→bsr.twse.com.tw，上櫃→櫃買中心）
2. 篩選買超最大前幾名：找出買超張數異常大的分點
3. 回測該分點歷史績效：觀察這個分點過去在這檔股票的操作紀錄——是否能在低檔大買、高檔大賣
4. 建立追蹤清單：將績效好的分點記錄下來，持續觀察後續進出

實例：

  • 網龍（3083）在 2019/8/21，「兆豐-景美」突然買超 557 張，之後股價出現明顯上漲波段
  • 在相對高點時，「兆豐-景美」開始出脫持股，精準掌握波段高低點
  • 2019/9 初，「兆豐-景美」再度進場買超，後續又有一波漲幅
  • 結論：兆豐-景美就是網龍的關鍵分點，未來再進場就可留意

關鍵分點四大特性：
1. 買賣點位特殊，有獨到之處
2. 精準掌握買入和賣出時機（低買高賣）
3. 對特定公司股價掌握能力強
4. 不一定有大量買賣，但投資效益驚人

地緣券商追蹤法

是什麼：公司地址附近的券商分點，如果出現大量買超，可能是公司派（內部人）在佈局。

怎麼做：
1. 查詢公司登記地址
2. 搜尋公司地址附近的券商分點
3. 追蹤這些地緣券商的買賣超動態
4. 如果地緣券商出現異常大買超 → 合理懷疑公司派進場

實例：

  • 為升（2231）公司位於彰化縣福興鄉彰鹿路附近
  • 鹿港市區的券商分點如果出現大買超 → 可能是公司內部人士進場
  • 新光鋼（2031）公司在新北三重，「元大-士林」和「統一-三重」兩個分點在低檔買入、高點賣出，距離公司非常近，判斷為地緣券商

多主力辨識與分點合併觀察

是什麼：一檔股票可能同時有多個主力在不同分點操作，需要合併觀察才能看清全貌。

怎麼做：
1. 單一分點追蹤法：找出 Top 3 買超分點，觀察是否連續多日買超
2. 分點合併觀察法：同一券商多個分點合計（如凱基-台北 + 凱基-松山 + 凱基-信義），可能是同一主力分散下單
3. 外資券商合併法：摩根大通、美林等外資券商，可能是同一外資法人
4. 主力對作辨識：當買超前幾名和賣超前幾名同時出現大量，表示有多空對作

實例：

  • 鴻海（2317）2024/3/11 出量大漲，Top 3 主力分點連續大買，中間雖有賣超一兩天但數量不大，股價持續飆漲
  • 東哥遊艇（8478）解除處置後暴跌，凱基-台北賣超 526 張、摩根大通賣超 514 張，兩家占當天成交量四成以上

適用時機：籌碼面分析選股、短線追蹤主力動向、判斷股價續漲或反轉

二、隔日沖券商辨識與應對

隔日沖是什麼

是什麼：隔日沖是「今天買、明天賣」的短線操作手法。主力用龐大資金將股價拉抬甚至拉到漲停，隔天趁開高時全部倒貨給散戶，快速賺取價差。

隔日沖兩大手法：

手法一：拉漲停
1. 主力鎖定當天強勢股，大量買進
2. 將股價拉到 7%~9% 甚至漲停鎖死
3. 散戶看到漲停排隊跟進
4. 隔天開高，散戶繼續追漲
5. 主力趁開高全部出貨，散戶接在高點
6. 股價反轉下跌，甚至跌破起漲點

手法二：權證+股票策略
1. 主力同時大量買進權證和標的股票
2. 自營商因風控規定必須買進股票避險
3. 兩方推力下股價上漲
4. 隔天主力全部賣出賺取價差
5. 一般投資人難以察覺這種手法

隔日沖券商名單

主要隔日沖券商：

  • 區域性分點：嘉義幫（如 XX-嘉義）、虎尾幫（如 XX-虎尾）
  • 外資券商：美林、摩根大通
  • 特定券商：凱基-松山、凱基-台北、元大-土城永寧、國票-敦北法人、富國-建邦
  • 次要隔日沖券商：統一-士林、群益金鼎-大安、中國信託-忠孝

辨識要點：

  • 看到這些券商名稱不代表一定是隔日沖
  • 關鍵判斷：該分點買超是否高居第一名且與第二名差距明顯
  • 買進量不多 → 隔天造成反轉的機率較小
  • 買進量異常大（佔成交量 10% 以上）→ 隔日沖機率高

隔日沖應對三招

第一招：4% 開盤分水嶺

  • 隔天開盤高於 4%：股價強勁，可續抱觀察
  • 隔天開盤低於 4% 且成交量迅速擴大：先獲利出場

第二招：尾盤再決定

  • 隔日沖賣壓通常在早盤釋放完畢
  • 盤中波動大，建議在尾盤（13:00~13:30）再做買賣決定
  • 等隔日沖籌碼沉澱後再進場

第三招：看你的持有成本

  • 持有成本低：按原計畫停損停利即可，不用被隔日沖影響
  • 短期籌碼集中 + 上升趨勢（5日、20日籌碼集中度 > 0 或特定大戶連續買）：隔日沖出場後趨勢多半會延續，不必急著賣
  • 長期投資人：隔日沖跟你無關，照自己的策略走

隔日沖進化手法：

  • 近年隔日沖越來越被注意，主力也變聰明
  • 可能隔 2 日、3 日才賣出
  • 甚至隔天繼續拉漲停再出貨
  • 不要預設立場認為隔日沖一定隔天賣

實例：

  • 穎漢（4562）6/28、7/1 兩天強拉漲停
  • 7/1 盤後發現凱基-松山與元大-土城永寧買超張數異常多
  • 7/2 早盤一度拉到近 7%，隨後隔日沖倒貨，收盤爆大量黑 K
  • 亞翔（6139）7/1 漲停，凱基台北買超第一名且為第二名的兩倍
  • 7/2 開高後下挫，典型隔日沖模式

三、千張大戶持股比例判讀

千張大戶持股比例判斷

是什麼：透過集保庫存的股權分散表，觀察持股 1000 張以上的大戶持股比例變化，判斷籌碼是集中還是分散。

怎麼做：
1. 到 TDCC（集保結算所）網站查詢個股的股權分散表
2. 重點觀察「千張以上持股比例」和「50 張以下持股比例」的變化
3. 兩者互為鏡像：千張增加 = 散戶減少，千張減少 = 散戶增加

判讀標準：

  • 低於 40%：籌碼流向散戶，行情易跌難漲
  • 40%~70%：籌碼集中度適中，最容易拉抬
  • 70%以上：高度集中，大戶看好可能續漲，但也可能不易出貨
  • 80%以上：過度集中，容易操控股價但出貨風險大

實例：

  • 聯電：外資+董監持股從 56.3%（4月，股價 15.55 元）→ 58.1%（7月，股價 22.35 元），籌碼集中股價上漲
  • 9 月董監增加持股 19,385 張，多頭趨勢仍在
  • 800-1000 張的中實戶持股比例增加 + 50 張以下散戶比例下降 = 籌碼往大戶集中

適用時機：中長期選股、存股篩選、判斷股價趨勢方向

千張大戶持股追蹤五步驟

1. 查集保股權分散表：每週更新，觀察千張以上和 50 張以下比例變化
2. 觀察連續性：一週變化不夠，要看連續 2~4 週的趨勢
3. 搭配三大法人數據：千張增加如果主要是外資，參考價值更高
4. 搭配技術面：千張增加 + 股價站上均線 = 更強的買進訊號
5. 設定警戒：千張比例從 70% 以上急速下降 → 大戶可能在出貨

四、籌碼面分析常見誤區

六大誤區

1. 法人買超 ≠ 股價必漲：法人可能為避險而買進，需要看是否搭配股價上漲
2. 主力連買 ≠ 沒有風險：可能是拉高出貨（誘多吸籌），須留意量價配合
3. 籌碼集中 ≠ 絕對利多：過度集中反而壓抑股價波動，出貨時下跌更猛
4. 短線波動不代表籌碼面失效：籌碼面適合觀察中短期趨勢，日內波動勿過度解讀
5. 高融券不一定看空：高融券 + 股價不跌反漲 → 可能引發軋空行情
6. 只看籌碼不看技術圖形：主力連買但 K 線破底 → 保持警覺，避免單一面向判斷

籌碼面實戰對照表

  • 投信連續多日買超 → 中小型股具題材或業績 → 留意進場點，搭配均線支撐操作
  • 外資忽然大幅買進 → 權值股可能進入攻擊波段 → 留意成交量是否配合放大
  • 融資暴增 + 股價橫盤 → 可能為誘多 → 觀察是否有主力出貨跡象
  • 大戶持股穩步增加 → 籌碼集中 → 可考慮逢低佈局，設好停損
  • 主力買超異常放大但股價不漲 → 拉高出貨訊號 → 多觀察，不急著跟進

心得

這回合學到的關鍵分點追蹤和隔日沖辨識，其實是籌碼面分析中非常實用的兩塊。分點進出數據是台股獨有的透明資訊——證交所每天公布所有券商分點的買賣明細，這在其他市場是看不到的，等於是給散戶一雙看穿主力底牌的眼睛。

但重點來了：看到關鍵分點大買就無腦跟？看到隔日沖進場就怕？都不對。關鍵分點要「回測績效」才有參考價值——一個分點如果歷史上在這檔股票都是低買高賣，那它再進場就值得留意；但如果只是偶爾出現一次大買，可能只是散戶湊巧集中，沒有跟單意義。

隔日沖更是如此——不是所有凱基-松山或元大-土城永寧的買超都是隔日沖，要看買超量是否佔成交量異常大的比例。而且隔日沖也在進化，會隔兩天三天才出貨，甚至繼續拉漲停再倒貨，所以不要死背「隔日沖=隔天賣」。

最實用的收穫是地緣券商的概念——公司地址附近的券商如果出現異常大買超，很可能是公司派在佈局，這個訊號的勝率比一般分點高很多，因為公司派對自家營運狀況最清楚。下次選股時，我會把地緣券商追蹤納入籌碼面分析的 SOP 裡。
