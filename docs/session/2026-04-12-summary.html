1. OBV 能量潮指標實戰

核心概念:

  • OBV 將成交量累積化為趨勢線，核心邏輯「量先價行」
  • 四大訊號：頂背離（賣）、底背離（買）、多頭同步（加碼）、空頭同步（觀望）
  • OBV 急速上升 = 買盤過度集中，鋸狀曲線轉向才是明確訊號

實戰應用:

訊號判斷操作
股價創高 + OBV 沒創高頂背離分批出場
股價創低 + OBV 沒創低底背離準備進場
OBV 急升後鋸狀轉下警戒逢高賣出

注意: 不適用冷門股、低量股、當沖短線

2. 波段交易進階技巧

核心概念:

  • 四大策略：突破/跌破（右側）、盤整區間操作、反轉行情、回撤加碼
  • 量價四型態：價漲量增（多頭）、價跌量縮（洗盤）、價漲量縮（背離）、價跌量增（空頭）
  • 大成交量區間帶可作為支撐壓力判斷

實戰重點:

  • 盤整區間操作最適合新手，突破後要換策略
  • 反轉操作風險最高，要等確認後再進場
  • 箱型理論適用：大型股、多頭環境、順勢做多

3. 多重時間框架分析

核心概念:

  • 大週期看位置、中週期判趨勢、小週期找入場
  • 時間框架比例建議 1:4 或 1:6
  • 用小週期停損 + 大週期停利，盈虧比可拉到 5-10 倍以上

實戰設定:

交易風格大週期中週期小週期
波段交易日線4小時1小時
當沖交易1小時15分鐘5分鐘

注意: 不要看太多時間框架，固定三個即可

4. 突破回踩確認策略

核心概念:

  • 突破後等回踩確認，避免假突破，勝率更高
  • 六步驟：識別盤整 → 等突破 → 找回踩 → 執行交易 → 設停損停利 → 監控

回踩低吸三條件:
1. 下方有獲利籌碼峰作為依託
2. 結合趨勢線或均線共振
3. 板塊指數時間共振（大盤不好時 80% 股票都不好）

注意: 回踩不會每次出現，有時價格直接續漲

5. 主力鎖碼股判斷技巧

核心概念:

  • 三大指標：主力買賷超（連續正數）、買賣家數差（連續負數）、籌碼集中度（上升）
  • 黃金組合：三者同時出現 = 大戶集中吃貨

融資鎖碼三特徵:
1. 融資使用率 > 20%（大戶用融資鎖碼）
2. 融資暴增（短時間突然增加）
3. 籌碼趨向集中（區分主力鎖碼 vs 散戶攤平）

四步驟選股: 買賣家數差七成為負 + 融資近20日暴增10% + 融資使用率 > 20% + 股價在融資成本線上
回測績效: 上漲機率 77.8%，平均報酬 19.26%

6. 股息貴族與高股息投資策略

核心概念:

  • 股息貴族：S&P 500 成分股中連續 25 年增加配息
  • 股息國王：連續 50 年以上增加配息（更嚴格）
  • DRIP 股息再投資實現複利效應

三種選股策略:
1. 殖利率排行法（殖利率 3-8%）
2. 配息成長法（連續 10 年配息成長 > 5%）
3. 價值低估法（合理價 vs 當前價有安全邊際）

三大風險: 高殖利率陷阱、配息來自借債（配息率 > 100%）、股息刪減

7. 配對交易策略

核心概念:

  • 市場中性策略：做多一邊、放空另一邊，賺價差收斂或發散
  • 兩種獲利方式：價差收斂（主流）、價差發散（變化型）
  • 需要兩倍資金但只賺價差，報酬相對小

進場判斷三方法:
1. 歷史經驗判斷
2. 布林通道判林通道判斷（價格比例穿越通道後回歸）
3. 共整合檢定（進階，ADF 單根檢定確認定態性）

致命傷: 盈虧比極低（小賺大賠），類似選擇權賣方策略，一次意外可能永久性損傷

8. K 線組合型態實戰判斷

核心概念:

  • 多根 K 線組合比單根 K 線訊號更可靠，觀察「多空轉化過程」
  • 三根組合：早晨之星（空轉多）、黃昏之星（多轉空）、紅三兵、黑三兵
  • 兩根組合：多頭吞噬、空頭吞噬、烏雲罩頂、貫穿型態、母子型態

心理解讀: 趨勢 → 猶豫 → 反轉（三階段轉化過程）
注意: 在支撐壓力關鍵位置出現型態才可靠，確認成交量配合

9. ROE/ROA 獲利能力分析 + 杜邦分析法

核心概念:

  • ROE 衡量用股東資本賺錢的能力；ROA 衡量用全部資產賺錢的能力
  • 同樣的 ROE，背後的故事可能完全不同
  • 杜邦分析拆解：ROE = 淨利率 × 總資產週轉率 × 權益乘數

三種 ROE 推升來源:

來源代表評價
淨利率推升公司體質變好✅ 最理想
週轉率推升可能薄利多銷⚠️ 要留意
權益乘數推升高槓桿假象🚨 風險增加

選股標準: 最近 10 年 ROE > 8%、最近 3 年 ROE > 15%、高負債產業用 ROA > 6-7%

10. EPS 每股盈餘 + 三率三升 + 營收年增率

核心概念:

  • EPS 灌水三大陷阱：業外收益、庫藏股買回、景氣循環假象
  • 基本 EPS vs 稀釋 EPS 差距 > 10% 要用稀釋 EPS

三率三升: 毛利率、營業利益率、稅後淨利率同步上升 = 最強基本面訊號

  • 三率同步上升才是真成長，單一率上升可能是假象

營收年增率三大陷阱: 基期效應、成長品質（營收增+毛利降=殺價搶單）、單一指標盲點

11. 多維度選股系統：三合一整合

核心概念:

  • 基本面選「買什麼」→ 籌碼面確認「誰在買」→ 技術面決定「何時買」→ 風險評估「買多少」→ 持續追蹤「何時賣」

三面訊號衝突處理:

衝突處理方式
基本面好 + 技術面弱長投可加碼，波段要等
技術面強 + 基本面差可短打但停損要嚴
籌碼面好 + 技術面未發動放觀察名單，等突破
三面都好最佳進場時機

12. 漲停板操作 + ETF 投資配置

核心概念:

  • 漲停能追的條件：突破型、有量、籌碼乾淨（無隔日沖）、均線支撐
  • 絕對不要追：連續多日漲停、隔日沖券商大量買超、無量漲停

  • 衛星部位 20-30%（主題型 ETF）

13. 分批進場 vs 單筆買進 + 隔日沖券商判斷

核心概念:

  • 分批進場 ≠ 定期定額，資金邏輯完全不同
  • 歷史回歷史回測：單筆投入勝率 75-100%（台股 0050、台積電）
  • 分批進場的「報酬率陷阱」：分母要用總資金，不是已投入金額

隔日沖券商名單: 凱基-松山、凱基-台北、元大-土城永寧、國票-敦北法人、富國-建邦
應對三招: 隔天開盤 4% 分水嶺、尾盤再行動、長期投資者不受影響

14. 權證操作實戰

核心概念:

  • 權證 = 負和遊戲，券商賺價差 + 時間價值
  • 最大陷阱：隱含波動率調降（BIV 下降直接虧損 5%+）
  • 五大選權證原則：價平價內、到期日長、流動性好、買賣價差小、實質槓桿高

避坑: 選隱波穩定不降的券商、避開價外權證、設好出場時間不長抱、資金控制在小額

15. 融券強制回補 + Put Call Ratio

核心概念:

  • 三大強制回補時機：股東會前、除權息前、現金增減資
  • 軋空行情四指標：融券張數、融券佔成交量比、券資比 > 30%、大戶持股 > 40%

PCR 當散戶指標:

  • PCR > 140% = 散戶極度恐慌 = 可能底部
  • PCR < 100% = 散戶極度樂觀 = 可能頭部
  • 搭配大額交易人持倉做反向操作，訊號更可靠

16. 鐵兀鷹 Iron Condor 選擇權策略

核心概念:

  • 買權空頭價差 + 賣權多頭價差 = 不預測方向，賭區間盤整
  • Delta 中立比距離中立更重要
  • 獲利 50% 平倉是鐵律（時間價值衰減前半快、後半慢）

三種被貫穿應對: 小台避險、Rolling 調倉、轉倉換時間
適用時機: VIX 高位時建倉、盤整市、週三新合約後建倉

17. 蝶式價差 Butterfly Spread + 減資操作策略

核心概念（蝶式）:

  • 三個等距履約價、四口選擇權，風險有限（=建倉成本）
  • 用「利潤壓縮」換「風險控制」，是跨式的安全版
  • Butterfly vs Condor：確定中心用蝶式，確定區間用兀鷹

核心概念（減資）:

  • 三種減資：一般（打消虧損）、現金（退還股款）、庫藏股（回購註銷）
  • 非現金減資非常危險：每 4 檔就有 1 檔虧 50%+，浴火重生機率僅 15%
  • 減資後股價調升 ≠ 賺錢（只是數學結果）

18. DMI 趨向指標與 ADX 趨勢強度

核心概念:

  • +DI 衡量買方力道、-DI 衡量賣方力道、ADX 衡量趨勢強度（不看方向）
  • ADX < 20 = 盤整，不做順勢交易；ADX > 25 = 趨勢形成，可進場
  • ADX 突破 25 + +DI/-DI 交叉 = 高勝率進場訊號

六大交易訊號:
1. +DI 上穿 -DI + ADX 上升 → 最強做多
2. +DI 跌破 -DI + ADX 上升 → 最強做空
3. ADX 從低檔轉折向上 → 趨勢正在形成
4. ADX 從高檔回落 → 趨勢減弱，開始減碼
5. +DI/-DI 背離 → 趨勢即將反轉
6. +DI/-DI 頻繁交叉 + ADX < 20 → 盤整，不操作
