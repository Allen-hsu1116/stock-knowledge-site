一、選擇權雙買策略（跨式與勒式）實戰

1. 跨式策略（Straddle）：同履約價雙買

是什麼：同時買入相同到期日、相同履約價（價平）的Call和Put，預期市場將出現大幅波動但方向不明。最大虧損為雙邊權利金全歸零，獲利理論上無限。

怎麼做：
1. 進場條件：預期短時間內會有大漲大跌（如重大經濟數據公布、選舉、政策宣布）
2. 選擇履約價：買在價平（ATM），Call和Put履約價相同
3. 成本計算：總成本 = Call權利金 + Put權利金（如Call 206點 + Put 198點 = 404點，每點50元 = 20,200元）
4. 損益兩平點：上方BEP = 履約價 + 總權利金，下方BEP = 履約價 - 總權利金
5. 出場時機：行情突破損益兩平點後有足夠利潤即可平倉，不一定要持有到結算

實例：

  • 2016美國總統大選：台指9200點，買入9200 Call（111點）+ 9200 Put（104點），總成本215點
  • 選舉開票過程劇烈震盪，收盤前平倉：Call賣17點、Put賣289點
  • 結果：-94 + 185 = 82點淨利（約4,100元/組），獲利主因是Put大漲（川普當選恐慌）

適用時機：有明確重大事件、預期波動率將上升、方向不確定時
不適用：盤整行情、已無重大事件催化、VIX已在高位

2. 勒式策略（Strangle）：不同履約價雙買

是什麼：同時買入相同到期日、不同履約價（均價外）的Call和Put。成本比跨式低，但需要更大的行情才能獲利。

怎麼做：
1. 進場條件：同跨式，但想降低成本、願意接受需要更大波動才能獲利
2. 選擇履約價：Call買在價外（履約價>現價），Put買在價外（履約價<現價）
3. 成本比較：勒式比跨式便宜（價外權利金較低），但獲利門檻更高
4. 損益兩平區間：比跨式更寬，行情必須走得更遠才能獲利

適用時機：預期有超大行情、想降低進場成本、可接受更低勝率
不適用：行情幅度可能不夠大時（勒式比跨式更容易虧損）

3. Delta值與雙買策略的獲利原理

是什麼：Delta衡量標的每變動1點對選擇權價格的影響。價平Call的Delta≈0.5，價平Put的Delta≈-0.5。雙買策略的獲利關鍵是「Delta不對稱放大」。

怎麼做：
1. 初始狀態：跨式部位淨Delta≈0（Call+0.5 + Put-0.5 = 0），理論上方向中立
2. 行情啟動：假設行情上漲，Call逐漸進入價內（Delta從0.5→0.6→0.68），Put進入價外（Delta從-0.5→-0.4→-0.32）
3. 獲利原理：行情繼續走遠，Call的Delta越來越大（越接近1），Put的Delta越來越小（越接近0），Call價格上漲的幅度遠大於Put價格下跌的幅度
4. 關鍵理解：越靠近價平處，Delta變化率最大（Gamma最大）；越遠離價平，Delta變化趨緩

適用時機：理解雙買策略為什麼能獲利，避免「行情漲了但我還虧錢」的困惑

4. IV Crush：雙買策略的最大殺手

是什麼：事件結束後隱含波動率（IV）急速崩跌，導致選擇權權利金大幅縮水的現象。即使方向判斷正確，IV Crush也可能讓你虧錢。

怎麼做（防範方法）：
1. 不要在事件前才進場：事件前IV最高，進場成本最貴，事件後IV Crush殺傷力最大
2. 提前至少2-3天進場：在IV還沒飆到最高時布局，事件結束後IV Crush的影響較小
3. 事件結束後盡快平倉：IV Crush在事件後24-48小時最劇烈
4. 用Vega評估IV風險：Vega越大，對IV變動越敏感

IV Crush量化判斷：

  • IV Percentile > 70% → 進場成本太高，IV Crush風險極大
  • IV Percentile < 30% → 成本合理，可考慮進場
  • 簡單規則：如果大家都已經在討論這個事件了，IV大概率已經太高了

適用時機：每次考慮用雙買策略賭事件行情前，都要先評估IV是否已過高

5. 雙買策略風險管理四大原則

原則一：設定最大虧損比例 — 單次雙買部位虧損不超過總資金的2-3%
原則二：時間停損優先 — 進場後3-5個交易日若無大行情→考慮減碼或平倉
原則三：避免在已知事件前追高 — 若真想做，提前1-2週布局
原則四：事件後果斷平倉 — 事件結果出爐後24小時內平倉

6. 隱含波動率 vs 歷史波動率

  • IV > HV → 選擇權偏貴，偏向賣方
  • IV < HV → 選擇權偏便宜，偏向買方

7. VIX指數的四大實戰應用

應用一：判斷大盤多空
應用二：當沖加減碼判斷
應用三：選擇權買賣方切換 — VIX高位偏賣方、VIX低位偏買方
應用四：遠月選擇權風險警示 — 裸賣遠月選擇權超級危險

8. 波動率微笑與偏斜

  • 負偏斜（Smirk）：價外賣權IV > 價外買權IV → 股票指數市場常見
  • 偏斜越陡 → 市場越擔心暴跌

9. 利用波動率偏斜設計選擇權策略

核心原則：永遠賣出高IV端、買入低IV端

  • 偏斜陡峭時 → 用信用用信用價差（賣出高IV端）
  • 偏斜平坦時 → 用借方價差（買入方向性部位）

心得

今天學的選擇權波動率交易，讓我對之前學的Greeks和選擇權策略有了更深層的理解：

1. 雙買策略的核心矛盾：你需要大行情，但大行情來之前IV已經先漲了。選擇權超人那個2024台灣大選的案例太經典了——大家預期選舉有大行情，瘋狂買Call+Put避險，結果IV飆高讓進場成本超貴，選後行情不夠大+IV暴跌，直接虧損102點。這告訴我：如果大家都知道的事件，就不應該在事件前才做買方。

2. IV Crush是選擇權買方最容易被忽略的殺手。很多人以為「方向對了就能賺」，但選擇權的價值不只看方向，還看波動率。這就是Vega風險——之前學Greeks的時候知道Vega衡量波動率敏感度，但今天才真正理解Vega在實戰中的殺傷力。

3. 波動率偏斜是選擇權市場的地形圖。OP凱文那篇寫得太好了——偏斜不是抽象概念，而是直接告訴你「哪裡貴、哪裡便宜」。永遠賣出高IV端、買入低IV端。

4. VIX和IV/HV的比較是選擇權交易的羅盤。VIX高→選擇權貴→做賣方；VIX低→選擇權便宜→做買方。這兩個簡單的比較，就能幫你決定當下應該做買方還是賣方。

5. Delta不對稱放大是雙買獲利的數學本質。行情啟動後，贏的那一端Delta越來越大，輸的那一端Delta越來越小，兩者不再互相抵銷，獲利加速。

6. 裸賣遠月選擇權是自殺行為。遠月選擇權Vega值更大，一旦波動率飆漲，虧損遠大於近月。做賣方就要做近月、快進快出、嚴格停損。
