一、海龜交易法則核心體系

1. 海龜交易法四大原則

是什麼：理查德·丹尼斯（Richard Dennis）1983年創立的完整交易系統，核心是「交易員是培養的，不是天生的」，海龜學員四年年均複利80%。

怎麼做：
1. 掌握優勢：找到期望值為正的交易策略，長期才能創造正回報
2. 管理風險：控制風險守住陣地，否則等不到創造成果的一天
3. 堅定不移：唯有堅定不移執行策略，才能真正獲得系統成效
4. 簡單明瞭：從長久看，簡單的系統比複雜的系統更有生命力

實例：

  • 情境：海龜學員被教導「不要預測市場，從機率角度做交易」
  • 操作：即便不知道單筆交易結果，只要長期期望值為正就持續執行；連續虧損是系統的一部分，不是停止的理由

適用時機：建立任何交易系統的底層哲學，無論當沖、波段還是長期投資

2. 海龜交易法六大決策要素

是什麼：一個完整交易系統必須涵蓋的所有決策面向，缺一不可。

怎麼做：
1. 市場（買賣什麼）：選擇流動性足夠、趨勢明確的市場，不要在成交量太少的市場交易
2. 頭寸規模（買賣多少）：這是最重要的決策！大多數新手單筆冒太大風險，大幅增加破產機會
3. 入市（何時買賣）：系統產生明確的進場信號，包含價位和市場條件
4. 止損（何時退出虧損頭寸）：必須在建立頭寸前預先設定退出點位
5. 離市（何時退出贏利頭寸）：任何不說明贏利頭寸離市的系統都不是完整系統
6. 策略（如何買賣）：大帳戶要考慮進出場的市場衝擊成本

實例：

  • 情境：假設做多10口黃金期貨，在350美元買進，停損設在320美元
  • 操作：買進與停損差距30美元 × 10口 × 100盎司 = 30,000美元風險；若總資金100萬美元，單筆風險3%，仍在可接受範圍
  • 若總資金僅50萬美元，單筆風險6%，就超過安全範圍

適用時機：每次建立新交易策略時，用這六個要素逐項檢查是否完備

二、海龜交易四大實戰原則

3. 風險控制2%法則

是什麼：單筆損失控制在總資金的2%以內，這是海龜交易法對風險管理的核心要求。

怎麼做：
1. 計算單筆最大虧損 = 停損距離 × 部位大小 × 每點價值
2. 確認此虧損 ≤ 總資金的2%
3. 如果超過2%：要嘛增加本金（入金），要嘛縮小部位（大台換小台）
4. 波動大時特別注意：行情振幅4-500點時，停損20點太容易被掃；停損100點又佔10萬本金的5%

實例：

  • 情境：本金10萬元，小台指數停損設100點
  • 操作：100點 × 200元/點 = 20,000元虧損，佔10萬的20%，遠超2%
  • 解法：要嘛本金加到100萬（2萬/100萬=2%），要嘛縮小停損到10點（2,000元=2%）
  • 關鍵認知：同樣停損100點賠5,000元，本金越大損失比例越小；入金是為了降低損失比例，不是為了加碼

適用時機：每筆交易前必做計算，特別是波動大的市場環境

4. ATR動態停損設定

是什麼：用ATR（平均真實範圍）來設定停損，讓停損距離隨市場波動自動調整。

怎麼做：
1. 計算近期ATR（日均最大波動幅度）
2. 停損設為 ATR 的 1/4 到 1/3
3. 停損至少要 ≥ 1-2根5分K的距離
4. 近期一根5分K至少60點，停損20點根本不夠用

實例：

  • 情境：最近平均震幅400點
  • 操作：1/4 ATR = 100點停損，1/3 ATR = 133點停損
  • 錯誤示範：只設20-40點停損，一根5分K都還沒跑完就被掃出場
  • 常見悲劇：「我看到行情上漲做多，一下就停損、再做多又停損」→ 檢查發現停損只設40點

適用時機：所有交易，特別是波動加大時必須動態調整停損距離

三、雙海龜分批出場策略

5. 「210」分批出場法

是什麼：先買2個單位，到某條件先出1個單位獲利了結，剩下1個單位在另一條件出場。2→1→0的概念。

怎麼做：
1. 進場買2個單位（2口期貨或2張股票）
2. 第一條件觸發：出掉一半（1個單位），鎖住部分獲利
3. 第二條件觸發：出掉剩下一半（1個單位）
4. 即使第二個部位賠錢也沒關係：一賺一賠相抵，整體不賠
5. 如果行情繼續走，第二個部位還能放大獲利

實例：

  • 情境：做空台指期，行情持續往下
  • 操作：先在相對低點獲利一半出場；若行情反彈，另一半停損出場，整體仍不虧
  • 若行情繼續向下：另一半在更低點獲利了結，放大整體獲利
  • A轉情境：行情往上後反轉，若不先出一半，最後可能回到成本價一毛沒賺；先出一半至少保住部分利潤

適用時機：所有波段交易和當沖，特別是無法判斷行情持續性時

6. 雙海龜線判斷出場時機

是什麼：用兩條不同參數的海龜線（區間突破線）來判斷分批出場的時機，短線出第一批、長線出第二批。

怎麼做：
1. 海龜線邏輯：取最近N根K棒的高低點，加一段緩衝，作為移動停利線
2. 設定短線參數（例如6根K棒+25點）：觸發時先出場一半部位
3. 設定長線參數（例如12根K棒+60點）：觸發時出場剩餘部位
4. 日K長線參數（例如2根日K+40點）：價格不跌破最近2根日K低點+40點就不出場
5. 先用歷史走勢驗證參數有效性，再實戰使用

實例：

  • 情境：5分K做多台指，短線參數6根K棒+25點，長線參數12根K棒+60點
  • 操作：進場後行情往上，短線海龜線觸發→出掉一半獲利8,140元；剩餘一半繼續抱，長線海龜線觸發→再出場，整筆獲利81,400元
  • 極短線玩家：參數設2根K棒+2點，行情稍有反轉就出場，節奏快但抓高點能力強
  • 中長線玩家：用日K移動停利，2根日K+40點，過濾盤中雜訊

適用時機：想要分批出場但不知道何時出第一批、何時出第二批時

7. 參數選擇核心原則

是什麼：沒有對錯的參數，只有「適不適合你」的參數。

怎麼做：
1. 先了解自己的投資習性和目標（短線？中線？長線？）
2. 設定好參數後，先回頭看歷史走勢驗證效果
3. 極短線：參數小（2根K棒+2點），反應快但交易頻繁
4. 中長線：參數大（12根K棒+60點），過濾雜訊但可能錯過短線機會
5. 最佳解：長短並用，同時設短線和長線參數，短線先出一半、長線抱另一半

實例：

  • 12根K棒+60點：對有些人是長線，對另一些人是短線
  • 2根K棒+2點：極短線，中長線投資人根本跟不上節奏
  • 錯誤：中長線投資人設2根K棒+2點，整天被洗來洗去

適用時機：建立任何參數化交易系統時

四、海龜交易心理核心

8. 思考機率而非預測

是什麼：海龜交易法的核心心態——不試圖預測市場走勢，而是從機率角度做交易。

怎麼做：
1. 接受「單筆交易結果不可知」，但長期期望值為正
2. 連續虧損是系統的一部分，不是停止執行的理由
3. 波動大時：入金降低損失比例，不是加碼拚翻身
4. 遇到假突破：修改系統（如站穩週線24MA三到四週才確認），寧可績效略落後，也要避免假突破損失
5. 「交易現在、思考機率、避免預測、對自己的交易負責」

實例：

  • 情境：有交易者為了等待訊號可以3-6個月不進場
  • 操作：交易系統顯示此時進場沒優勢，沒優勢就不玩
  • 選擇週線24MA/48MA的原因：訊號不常發生、不容易產生矛盾、需要極大耐心
  • 這是一般散戶做不到的事

適用時機：交易心理建設，特別是連續虧損後想要放棄或報復性交易時

9. 分批試單與資金管理

是什麼：海龜交易法在資金管理上的實戰應用——分批進場、分批出場、預留現金。

怎麼做：
1. 進場和出場都分批進行
2. 向下加碼：資產下跌10%、20%、30%時分批買進
3. 向上加碼：回檔10%時加碼，但部位逐漸減少
4. 永遠保留20%現金防黑天鵝
5. 根據市場相對高低點位階調整資金水位

實例：

  • 情境：使用ETF槓桿做多兩倍
  • 操作：相比期貨高槓桿，ETF不用擔心結算日，能在市場中更長時間參與
  • 向下攤平風險存在，但根據市場位階做適當處理
  • 核心邏核心邏輯：寧可高點利潤減少，也不要高檔大額部位大幅虧損

適用時機：資金管理策略制定，特別是想要系統化進出場的投資人

心得

海龜交易法之所以被稱為「交易系統的聖杯」，不是因為它多麼神祕或複雜，恰恰相反——是因為它夠簡單、夠完整、夠紀律。

最讓我印象深刻的有兩點：

第一，「210分批出場法」的概念太實用了。以前學過分批出場，但海龜的版本更直觀——先出一半鎖利，剩下一半不管賺賠都不影響整體。這解決了散戶最大的問題：方向看對但最後一毛沒賺到。A轉、V轉都不怕，至少一半的利潤已經落袋。

第二，參數選擇的核心認知：「沒有對錯的參數，只有適不適合你」。這句話適用於所有交易系統。很多人花了大量時間找「最佳參數」，但真正該做的是先搞清楚自己是什麼類型的交易者，再選適合的參數。而且長短並用的思路也很棒——短線先出一半，長線抱另一半，不用在「停利太早」和「抱到虧」之間二選一。

海龜交易法四年的年均複利80%不是靠預測神準，而是靠「正期望值 + 嚴格風險控制 + 紀律執行」。說到底，交易的本質就是這三件事。
