單一選擇權操作 — 四大基礎策略

是什麼

單一部位是所有選擇權策略的基石，分為四種基本操作：

操作方向最大獲利最大風險
Buy Call（買進買權）看漲無限權利金
Buy Put（買進賣權）看跌履約價 - 權利金權利金
Sell Call（賣出買權）看跌或盤整權利金無限
Sell Put（賣出賣權）看漲或盤整權利金履約價 - 權利金

怎麼做

Buy Call 使用時機：

  • 預期標的物價格上漲
  • 將買進標的物決策遞延
  • 降低直接放空的風險

Buy Put 使用時機：

  • 預期標的物價格下跌
  • 將賣出標的物的決策遞延
  • 降低直接作多的風險

Sell Call 使用時機：

  • 預期標的物價格稍跌或盤整
  • 為已持有標的物設定賣點
  • 為已持有標的物提供有限度避險

Sell Put 使用時機：

  • 預期標的物價格稍漲或盤整
  • 為欲持有標的物設定買點
  • 為已放空標的物提供有限度避險

價差組合策略 — 風險有限的進階操作

一、垂直價差（執行價格價差）

是什麼：
同時買賣到期日相同但履約價不同的相同標的買權或賣權。

怎麼做 — 買權多頭價差（Bull Call Spread）：

  • 使用時機：看好市場，但認為短期大為短期大漲不易，屬於小漲格局
  • 操作方式：買進低履約價 Call + 賣出高履約價 Call（同到期日）
  • 最大獲利：兩履約價差額 - 投資成本
  • 最大風險：投資成本（買進 Call 權利金 - 賣出 Call 權利金）

實例：

  • 情境：大盤指數 5,200 點，認為會漲但不超過 5,500 點
  • 操作：
  • 成本：200 - 100 = 100 點（5,000 元）
  • 最大獲利：(5,500 - 5,200) - 100 = 200 點（10,000 元）
  • 盈虧比：1:2

怎麼做 — 賣權空頭價差（Bear Put Spread）：

  • 使用時機：看空市場，但認為短期大跌不易
  • 操作方式：買進高履約價 Put + 賣出低履約價 Put（同到期日）

二、時間價差（Calendar Spread）

是什麼：
同時買賣履約價相同但到期日不同的相同標的選擇權。

怎麼做：

  • 買進遠月選擇權 + 賣出近月選擇權

優點：

  • 起始投資金額較少
  • 具財務槓桿效果
  • 適合盤整格局

三、跨式組合（Straddle）

是什麼：
同時買進（或賣出）相同到期日與履約價的 Call 與 Put。

買進跨式（Long Straddle）：

  • 使用時機：預期大波動，但不確定方向
  • 操作：買進 Call + 買進 Put（同履約價、同到期日）
  • 最大風險：權利金總和
  • 最大獲利：無限（單邊）

賣出跨式（Short Straddle）：

  • 使用時機：預期盤整，波動度低
  • 操作：賣出 Call + 賣出 Put（同履約價、同到期日）
  • 最大獲利：權利金總和
  • 最大風險：無限

實例 — 賣出跨式：

  • 情境：大盤在 5,300 點盤整
  • 操作：
  • 總權利金：225 點（11,250 元）
  • 結算價 5,300 點：兩者都不履約，賺 11,250 元
  • 結算價 6,000 點：Call 被履約，損失 700 點，損益 = 225 - 700 = -475 點（-23,750 元）

保證金計算：

  • 取 Call、Put 保證金較大者 + 較少方權利金市值
  • 例：Call 保證金 26,250 + Put 權利金市值 5,000 = 31,250 元

四、勒式組合（Strangle）

是什麼：
同時買進（或賣出）相同到期日但履約價不同的 Call 與 Put。

買進勒式（Long Strangle）：

  • 使用時機：預期大波動，想降低成本
  • 操作：買進價外 Call + 買進價外 Put
  • 成本比跨式低，但需要更大波動才能獲利

實例 — 買進勒式：

  • 情境：大盤在 5,200 點，預期大波動
  • 操作：
  • 總成本：660 點（33,000 元）
  • 結算價 4,600 點：Call 不執行，Put 獲利 700 點，損益 = 700 - 660 = 40 點（2,000 元）
  • 結算價 5,200 點：兩者都價外，虧損 660 點

五、轉換/逆轉組合（Conversion/Reversal）

是什麼：
同時買賣相同到期日、相同履約價的 Call 與 Put。

轉換組合：

  • 買進 Put + 賣出 Call = 合成賣出標的
  • 使用時機：持有股票，想要鎖住利潤

逆轉組合：

  • 買進 Call + 賣出 Put = 合成買進標的
  • 起始投資金額較少，具財務槓桿效果
  • 若與標的資產產生價差可進行套利

實戰操作心法

心法一：單一部位是基礎

  • 所有複雜策略都是單一部位的組合
  • 先搞懂 Buy Call、Buy Put、Sell Call、Sell Put
  • 再學價差組合

心法二：價差組合降風險

  • 垂直價差：限制最大虧損
  • 時間價差：利用時間價值差異
  • 跨式/勒式：針對波動度交易

心法三：賣方策略要配保護

  • 賣出單一部位風險無限
  • 用價差組合把風險「框住」
  • 例：賣出 5,300 Call + 買進 5,500 Call = 風險被限制在 200 點

心法四：保證金要精算

  • 價差部位保證金較低
  • 但不代表風險較低
  • 仍要設定停損、控制部位

心得

選擇權價差組合是「風險管理」和「成本控制」的藝術。之前學過單一買方、賣方策略，但單一部位的風險不是太大（買方）就是無限（賣方），價差組合就是把風險「框住」的方法。

關鍵收穫：
1. 垂直價差 = 風險框住：用一買一賣把風險鎖在履約價差內，適合「看漲但不敢追」或「看跌但怕反彈」
2. 跨式/勒式 = 波動度交易：不看方向、只看波動度，大波動買跨式、小波動賣跨式
3. 時間價差 = 吃時間價值差異：近月衰減快、遠月衰減慢，賣近買遠賺價差
4. 保證金不是風險：價差部位保證金低，但風險仍在，仍要設停損
5. 賣方要配保護：賣出單一部位風險無限，加一個買進部位就能把風險框住

和之前學習的連結：

  • 「選擇權賣方策略」強調風險無限，價差組合就是把風險框住的方法
  • 「Greeks 希臘字母」中 Delta、Gamma、Vega、Theta 都可以透過價差組合精細控制
  • 「風險管理三層次架構」中的部位控制，價差組合是實現方式之一

實戰應用：

  • 小漲格局用 Bull Call Spread，不用買方單壓
  • 盤整時賣跨式收權利金，但要設停損
  • 大事件前買跨式賭波動，方向不明也能獲利
  • 永遠不要裸賣（不配保護的賣方）

價差組合是進階技巧，需要熟悉 Greeks 和風險管理，但這才是選擇權的真正威力：不只是賭方向，而是建構多維度的交易系統。
