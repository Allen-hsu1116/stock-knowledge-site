操作策略：多空轉換操作法×123法則趨勢反轉×洗盤辨識應對×均線進場時機

一、均線進場時機判斷：不追高的核心哲學

是什麼

均線進場法的核心思想：乖離率是最大的敵人。股價偏離均線太遠時，回檔機率極高；在均線附近買進，風險最低、勝率最高。

怎麼做

1. 大盤環境判斷先行

  • 大盤環境好（均線多頭排列）：用短均線進場（5MA、10MA）
  • 大盤環境差（均線糾結或空頭）：用長均線進場（月線、季線、半年線）
  • 判斷標準：20MA、60MA、120MA 是否上揚 + K棒是否在均線上方

2. 進場時機三原則

原則說明實例
均線上彎才買均線下彎代表趨勢向下，是壓力不是支撐5MA上彎+股價觸及5MA=買點
量縮回測才買上漲趨勢中回測均線，量縮=主力沒跑鴻海2024/4/1量縮回測5日線，後續漲5%
不追正乖離乖離率過大=追高，回檔機率遠大於續漲主力1億資金10%報酬=1000萬，散戶10萬資金10%=1萬

3. 20MA 進出場系統（林恩如極簡法）

  • 進場：股價站上週20MA → 買進
  • 出場：股價跌破週20MA → 賣出
  • 每週五檢視：週20MA價位每週變動，需定期檢視
  • 核心紀律：進場理由消失就出場，不憑感覺買賣

4. 四線多排進場法（朱家泓）

  • 5日線、10日線、20日線、60日線全部向上排列=四線多排
  • 四線多排最容易延續漲勢
  • 進場訊號：收盤價站回5日線 + 紅K實體漲幅2%以上
  • 回檔條件：不能跌破月線（20MA），跌破=多頭力道不足

實例

  • 鴻海2024/3/11起漲後，每次量縮回測5日線都是買點（3/14、4/1、4/25、5/21、6/27）
  • 4/16跌破月線=停損出場合理；4/24出量站上月線後，4/25量縮回測月線=再次買進
  • 萬潤2024年8月超跌到半年線附近，8/7反彈站回半年線=相對低點買進

適用時機

  • 順勢交易者，不適合想要抄底摸頂的人
  • 大盤環境好時用短均線、環境差時用長均線
  • 搭配籌碼面（主力買賣超、波段大戶券商）確認更準確

二、123法則：趨勢反轉確認系統

是什麼

123法則出自維克多·斯波朗迪《專業投機原理》，用三個條件判斷趨勢是否反轉。比單純看趨勢線突破更嚴謹，減少假突破的誤判。

怎麼做

123法則三個條件（判斷上升趋势反轉）：

條件1：上升趨勢線被向下跌破

  • 趨勢線至少要有2個觸點，跌破代表趨勢暫止

條件2：上升過程中不再創新高（試探前高但不破位）

  • 價格接近前高但無力突破，代表多頭力道衰減
  • 連接前低和試探後低點的斜率最好不超過30度（超過30度可能不是回調而是反轉）

條件3：價格向下跌破先前短期回檔低點

  • 這是進場訊號，確認反轉成立

重要：三個條件的順序可以打亂！

  • 可以是213、321等順序
  • 也可以沒有第2步（V形反轉）
  • 但必須完成第3步才算確認

進場點判斷：

情境突破位置
有第2步前高和第2步試探高點之間形成的低點
沒有第2步（V形）前高和前高之前緊鄰的低點

判斷下降趋势反轉（做多）：反向操作

1. 下降趨勢線被向上突破
2. 下跌過程不再創新低
3. 價格向上突破前期反彈高點 → 做多進場

實例

  • 台股2024年4月股災跌至17,306點後，6月26日突破前高22,414壓力線=123法則第3步確認
  • 虹冠電（3257）底部完成後進入初升段，5/10/20/60均線多排=趨勢反轉確認
  • 當123法則與反轉型態（頭肩底、W底）同時出現時，優先按反轉型態處理，因為型態學可以推算目標價

適用時機

  • 趨勢反轉的確認，不適用於盤整盤
  • 搭配均線、成交量一起判斷更可靠
  • 123法則完成後只代表短線反向運行，大趨勢反轉需要更多確認

三、洗盤辨識與應對：不被主力甩下車

是什麼

洗盤是主力在拉抬股價過程中，故意製造震盪讓意志不堅的散戶出場，然後在低檔承接籌碼，為後續大漲做準備。辨識洗盤vs真正下跌，是波段操作最關鍵的技能之一。

怎麼做

1. 洗盤七大特徵

#特徵說明數據標準
1緩漲急跌慢慢盤上後突然下殺，製造恐慌上涨3-5天 vs 急跌1-2天
2下跌量縮下跌時成交量萎縮=主力沒跑較20日均量減少30-50%
3跌破均線但快速收復故意破線嚇散戶，2-5天內站回3天內收復=強支撐
4均線仍多頭排列長期均線（30MA、60MA）維持上揚30MA和60MA不能下彎
5維持在10MA之上健康洗盤10MA是重要支撐跌破也快速站回或貼著量縮整理
6成交量遞減浮額減少，籌碼逐漸穩定洗盤末期較20日均量減少60-80%
7帶量向上突破洗盤結束的明確訊號量能較前期增加20-50%

2. 洗盤三階段量化指標

階段成交量變化量化標準持續時間主要特徵
初期：恐慌性賣出較20日均量暴增50-100%大量拋售後被快速承接1-3天急跌有強力買盤
中期：量能萎縮較前期高點減少30-50%日均量持續低於20日均量5-20天觀望情緒濃厚
末期：極度萎縮較20日均量減少60-80%量縮價穩，為突破做準備2-7天浮籌基本清洗完畢

3. 四指標洗盤確認公式

洗盤確認 = 量能萎縮(30-50%) + RSI底背離 + MACD收斂 + 布林下軌支撐

四個條件中出現三個以上 → 洗盤可能性超過80%

4. 三種洗盤手法辨識

手法操作方式辨識特徵應對策略
開高殺低拉高後大量賣出製造恐慌長上影線+爆量+融資減少不追高，等量縮止穩再買
盤整洗盤長時間區間整理消磨耐心低位長期盤整+量能遞減耐心持有或低接，設好停損
跌停洗盤開盤直接跌停嚇出散戶跌停後快速打開+主力敲進觀察跌停打開速度和量能

5. 洗盤 vs 真正下跌：五道紅旗

如果出現以下特徵，不是洗盤而是真正下跌：

1. 量價背離：價漲量縮、價跌量增
2. 利多不漲：好消息不漲反跌
3. 主力賣超：盤後籌碼主力持續出貨
4. 跌破防守線：起漲長紅最低點被跌破
5. 均線破壞：30MA、60MA轉為下彎

實例

  • 台股2024年7-8月大跌：8/2-8/5恐慌性下跌，但萬潤8/6跌破半年線後8/7快速反彈站回=洗盤特徵
  • 洗盤初期8/2-8/5成交量暴增（恐慌性賣出），中期量能萎縮，末期量縮價穩後帶量突破=典型三階段
  • 融資餘額下降=主力達到清洗浮額目的

適用時機

  • 波段操作中遇到回檔，需要判斷是洗盤還是真正反轉
  • 搭配123法則：如果洗盤後跌破前低=真正反轉，應停損出場
  • 盤整洗盤法適合有耐心的投資人，不適合短線操作者

四、大盤環境×個股進場：四維度整合策略

是什麼

單一指標不足以判斷進場時機。將大盤環境、基本面、技術面、籌碼面四個維度整合，建立完整的進場決策框架。

怎麼做

1. 大盤環境判斷（先看環境再選股）

大盤狀態判斷方式操作策略適合均線
多頭（均線多排）5MA>10MA>20MA>60MA積極操作5MA、10MA
盤整（均線糾結）均線交叉纏繞觀望或小部位20MA、60MA
空頭（均線空排）5MA<10MA<20MA<60MA保守操作只看120MA以上

2. 基本面篩選（選什麼股票）

  • 遛狗理論：股價（狗）最終回歸基本面（主人）
  • 選基本面良好+未來展望正面的公司
  • 好公司在超跌時逢低買進，比追強勢股更安全

3. 技術面進場（何時買進）

  • 大盤好 → 5MA/10MA回測買進
  • 大盤差 → 20MA/60MA回測才考慮
  • 量縮回測均線=買點，量增跌破均線=停損

4. 籌碼面確認（誰在買）

  • 波段大戶券商沒出貨 → 還有上漲波段
  • 隔日沖券商大量進出 → 不要追高
  • 三大法人同步買超 → 確認訊號最強

  • 每次進場前的系統性檢查清單
  • 大盤環境決定操作節奏，基本面決定選股方向，技術面決定進場時機，籌碼面確認方向

心得

這回合學了均線進場、123法則、洗盤辨識和四維度整合策略，幾個核心收穫：

1. 不追高是進場的第一原則。買在均線附近，風險最低。主力1億資金賺10%=1000萬，散戶10萬資金賺10%=1萬，追高被洗出去的損失完全不成比例。

2. 123法則比趨勢線突破更可靠，因為它要求三個條件同時滿足。但順序可以打亂，最重要的是第3步完成才算確認。跟型態學（W底、頭肩底）疊加時，型態學優先，因為可以推算目標價。

3. 洗盤辨識的量化公式很實用：量能萎縮30-50% + RSI底背離 + MACD收斂 + 布林下軌支撐，四個出現三個以上就是洗盤機率80%。但五道紅旗（量價背離、利多不漲、主力賣超、破防守線、均線破壞）出現任何一個就要小心。

4. 大盤環境決定操作策略。環境好用短均線、環境差用長均線，這不是什麼高深道理，但很多人在多頭市場用長均線進場（太慢）、空頭市場用短均線進場（太急），剛好做反。

5. 四維度整合是系統化交易的基礎：大盤→基本面→技術面→籌碼面，每個維度都要通過才能進場。這比單看任何一個指標的勝率高很多。

操作策略：金字塔加碼法×移動停利停損系統×分批進出場策略

進場和出場一樣重要，但很多人只研究「怎麼買」，不研究「怎麼加碼」和「怎麼出場」。今天要把分批進出場的兩大核心系統搞懂：金字塔加碼法和移動停利停損。

一、金字塔加碼法：分批進場的資金管理藝術

是什麼

金字塔加碼法（Pyramid Trading）是一種分批買入的投資策略，根據股價走勢調整每次買入的數量，形成如金字塔般的部位結構。核心概念：不是一次all in，而是分批建立部位，用資金分配來控制風險。

正金字塔 vs 倒金字塔：兩種完全不同的邏輯

正金字塔買法（逢低加碼）：

  • 越跌買越多，越漲買越少
  • 適合左側交易、價值投資
  • 目的：降低平均成本
  • 風險：可能越攤越平，標的持續下跌就完蛋

倒金字塔買法（順勢加碼）：

  • 越漲買越多，越跌買越少（先試單，確認趨勢才加碼）
  • 適合右側交易、趨勢交易
  • 目的：讓資金主要投入在趨勢已確認的區間，極大化獲利
  • 風險：成本會越墊越高，反轉時回檔幅度大

怎麼做

1. 正金字塔買法操作步驟（逢低加碼型）

  • 確認標的長期趨勢向上（好公司、龍頭股、ETF）
  • 設定總投入金額（例如某檔股票預計投入10萬）
  • 分批買入，價格越低買越多：
  • 每一批都設停損點，跌破則不繼續加碼

實例：

  • 台積電在600元開始分批買進，預計投入100萬
  • 600元買40萬（4張），570元買30萬（5張多），540元買20萬（3張多），510元買10萬（2張）
  • 平均成本約562元，比一次600元買進成本低約6.3%
  • 前提：台積電這種長期趨勢向上的公司才適合

2. 倒金字塔買法操作步驟（順勢加碼型）

  • 先用小資金試單（10-20%）
  • 確認趨勢向上才加碼，越漲加越多：
  • 看錯方向時，因為初始部位小，停損成本低

實例：

  • 某股突破50元壓力，先用1萬試單
  • 確認站穩50元後，漲到52.5元加碼2萬
  • 漲到55元趨勢確立，加碼3萬
  • 漲到57.5元強趨勢，加碼4萬
  • 平均成本約54.8元，主要部位建立在趨勢確認後

傑西·李佛摩的核心原則

> 「買進時要測試市場，市場證明你是對的時候，才應該加碼，而不是在市場證明你錯誤時不斷攤平成本。」

這就是倒金字塔加碼法的哲學——賺錢才加碼，虧錢不加碼。

等注加碼法：簡化版金字塔

如果覺得正/倒金字塔太複雜，可以用「等注加碼」：

  • 每次加碼的金額固定（例如每次2萬）
  • 優點：每一注的盈虧固定，不受大小注影響
  • 缺點：賺得比正金字塔少（因為一開始沒下大注）
  • 彌補方式：「快速加碼」——縮短加碼間隔，讓部位快速建立

金字塔加碼的五大陷阱

1. 現金拖累：長期投資設定價格區間過大，結果只買到10%，90%現金閒置
2. 越攤越平：正金字塔用在持續下跌的爛公司，越買越深套
3. 加碼後反轉：倒金字塔加到滿倉後趨勢反轉，成本高回檔大
4. 看對買太少：正金字塔一開始買太少，後面加碼不到就起飛了
5. 頻繁被洗：加碼後被震盪洗出，交易成本侵蝕利潤

金字塔加碼法選擇決策樹

你的交易風格推薦方式理由
價值投資（長線）正金字塔逢低加碼好公司跌越便宜，成本越低越好
趨勢交易（波段）倒金字塔順勢加碼趨勢確認才下重注，控制假突破風險
不確定方向等注加碼每次固定金額，簡單可控
期貨/選擇權只用倒金字塔有到期日和槓桿，虧錢加碼=找死

二、移動停利停損系統：讓獲利奔馳的出場藝術

是什麼

移動停利停損（Trailing Stop），也稱追蹤停止，是一種出場點會隨股價有利方向移動，但不會反向退縮的出場機制。核心概念：當市場對你有利時鎖住利潤，當市場對你不利時保持原停損不動。

傳統停損停利 vs 移動停利停損

比較傳統停損停利移動停利停損
設定方式固定價格（如100元買、92元停損108元停利）從最高點拉回一定比例出場
調整方式不會自動調整，需手動修改自動跟著價格上漲而上調
獲利鎖定要嘛停利要嘛停損，二選一可以在「賺了又回吐」的中間地帶出場
適合市場穩定小幅波動趨勢明確、波動較大

怎麼做

1. 百分比移動停利法（最常用）

  • 設定從最高點拉回X%即出場
  • 停損點只往上移動，不會往下

計算方式：

  • 買進價：100元
  • 設定：從最高點拉回5%出場
  • 漲到120元 → 停損點 = 120 × 0.95 = 114元
  • 續漲到150元 → 停損點 = 150 × 0.95 = 142.5元
  • 從150元跌到142.5元 → 觸發出場，獲利42.5元

實例：

  • 台積電500元買進，設定10%移動停利
  • 漲到600元，停損點540元
  • 漲到650元，停損點585元
  • 從650跌回585元出場，獲利85元（17%），比固定停利在550元出場多賺35元

2. ATR移動停利法（進階版，最精準）

  • 利用ATR（真實平均波動區間）設定移動停損
  • 好處：自動適應標的波動，波動大停損寬、波動小停損窄

計算方式：

  • 買進後最高價 - N倍ATR = 停損價
  • N通常取2-3倍

實例：

  • 某股ATR=3元，買進後最高漲到110元
  • 2倍ATR停損 = 110 - (3 × 2) = 104元
  • 3倍ATR停損 = 110 - (3 × 3) = 101元
  • 波動大的股票用3倍ATR避免被洗，波動小用2倍更貼近

3. Chandelier Exit（吊燈出場法）

  • 進階版ATR移動停利
  • 公式：前N日最高價 - N倍ATR
  • 常用參數：22日最高價 - 3倍ATR
  • 比%數更科學，因為考慮了標的實際波動

4. 搭配技術指標的動態出場

  • 跌破10日均線出場（短線）
  • 跌破20日均線出場（波段）
  • 跌破布林通道中軌出場
  • SAR翻轉出場
  • 優點：出場點跟著趨勢動態調整，不是固定%數

移動停利停損的參數設定建議

操作週期建議移動停利%ATR倍數調整頻率
當沖不適合移動停利——
短線（5-20天）3-5%1.5-2倍每日調整
波段（1-3個月）8-15%2-3倍每週檢視
中長線（3-12個月）15-25%3-4倍每月檢視

重要原則：設定不要太小，否則正常回檔就會被洗出場。觀察標的過去3個月的正常波動範圍，停損%數要大於正常回檔幅度。

移動停利停損的適用情境

✅ 適合：

  • 趨勢明確的市場（多頭或空頭排列）
  • 無法長期盯盤的交易者
  • 容易受情緒影響該出不出的人
  • 波段操作想要「讓獲利奔馳」的策略

❌ 不適合：

  • 橫盤震盪的市場（會被來回洗）
  • 波動太小的標的（獲利還沒超過啟動門檻）
  • 當沖交易（需要更敏感的出場方式）

三、OCO二擇一單：停損停利同時設定的實戰工具

是什麼

OCO（One Cancels the Other）是一種「二擇一」的下單方式，同時掛出停利單和停損單，其中一邊成交，另一邊自動取消。

怎麼做

操作流程：
1. 下好母單（限價或市價買進）
2. 成交後，自動掛出停利單（一般單）＋停損停損單（MIT觸價單）
3. 停利成交 → 停損自動取消；停損觸發 → 停利自動取消

實例：

  • 100元買進某股
  • OCO設定：停利110元、停損95元
  • 情境A：漲到110元停利出場，95元停損單自動取消
  • 情境B：跌到95元停損出場，110元停利單自動取消
  • 好處：不用盯盤，進場後就讓系統自動管理

台股券商OCO功能名稱對照

  • 玉山證券：「智慧下單」
  • 國票證券：「雲端智慧單」
  • 凱基證券：「智慧單」
  • 統一期貨：「e指發」
  • 康和期貨：「全都賺」

OCO + 移動停利的進階組合

1. 進場後先用OCO設定初始停損停利
2. 當獲利超過移動停利啟動門檻後，取消OCO改掛移動停利
3. 讓獲利持續擴大，直到移動停利觸發出場

四、完整分批進出場SOP

進場（金字塔加碼）

步驟1：確認大盤環境（多頭→積極，盤整→觀望，空頭→保守）

步驟2：選擇加碼方式

  • 趨勢明確 → 倒金字塔順勢加碼
  • 價值低估 → 正金字塔逢低加碼
  • 不確定 → 等注加碼

步驟3：設定總投入金額和分批比例

  • 單一標的不超過總資金的20%
  • 至少分3批，最多5批
  • 每批之間的價差或時間要事先規劃

步驟4：第一批進場同時設定OCO停損停利

步驟5：第二批之後的加碼條件要明確（價格到多少才加、量多少）

出場（移動停利停損）

步驟1：進場後設定初始停損（成本下方2-5%或1.5倍ATR）

步驟2：獲利超過5%後，啟動移動停利機制

  • 短線：3-5%移動停利
  • 波段：8-15%移動停利

步驟3：移動停利啟動後，取消原本的OCO停利單（停損仍保留或由移動停利取代）

步驟4：移動停利只往上不往下，耐心讓獲利奔馳

步驟5：移動停利觸發 → 出場，不戀戰

分批出場策略

不只進場可以分批，出場也可以：

  • 第一目標（1:1風報比）：賣出50%部位，鎖定基本獲利
  • 第二目標（1.5:1或2:1）：賣出30%部位
  • 剩餘20%：用移動停利跑，吃到最後一段
  • 好處：不會全部賣在太早或太晚，心理壓力小

心得

今天學的東西其實是在回答一個最核心的問題：「我該一次買好還是分批買？賺了以後什麼時候賣？」

金字塔加碼法的精髓不在於「正」還是「倒」，而在於你有沒有事先規劃好資金分配。多數散戶的問題就是一次all in，然後套牢了不知道該加碼還是認賠。如果一開始就分批，每次的決策壓力都小很多。

傑西·李佛摩那句話真的值得背起來：「市場證明你是對的時候，才應該加碼，而不是在市場證明你錯誤時不斷攤平成本。」倒金字塔加碼法的本質就是這個——用小錢試錯，用大錢賺趨勢。

移動停利停損最大的價值不是「賺更多」，而是克服人性。很多人賺了30%不賣，結果回吐到-10%才不甘心地出場。移動停利就是幫你在「賺了又回吐」的時候果斷出場，至少保住一部分利潤。

最後，OCO單和移動停利的組合，是台股投資人應該善用的工具。進場後設好OCO，就不用盯盤；獲利後切換移動停利，就能讓利潤繼續跑。這不是偷懶，是把情緒從交易中移除，讓紀律取代感覺。

海龜交易法則（Turtle Trading System）是交易史上最著名的公開系統之一。1983年，理查德·丹尼斯用這套系統培訓了13個幾乎沒有交易經驗的新手，結果四年內年化複利報酬率達80%。核心哲學很簡單：交易不需要天賦，需要的是一套完整的規則加上鐵的紀律。

今天要把海龜交易法則從頭到尾拆乾淨：市場選擇、部位計算、進場、停損、出場、加碼，每個環節都有明確的公式和數字。

一、海龜交易法則的核心哲學

四大原則

#原則說明
1掌握優勢找到期望值為正的交易策略，長期才能創造正回報
2管理風險控制風險、守住陣地，否則等不到創造成果的那天
3堅定不移唯有堅定執行策略，才能真正獲得系統成效
4簡單明瞭從長久看，簡單的系統比複雜的系統更有生命力

完整交易系統的六個決策

1. 市場 — 買賣什麼（選擇流動性好的市場）
2. 部位規模 — 買賣多少（根據波動性計算）
3. 進場 — 何時買賣（突破系統）
4. 停損 — 何時退出虧損部位（2N停損）
5. 出場 — 何時退出獲利部位（10日/20日反向突破）
6. 策略 — 如何買賣（限價單、市價單等）

二、唐奇安通道突破：海龜的進場系統

是什麼

唐奇安通道（Donchian Channel）是最古老的突破系統之一，由理查德·唐奇安發明。海龜交易法使用兩套基於唐奇安通道的突破系統：

系統一：20日突破（短線系統）

進場規則：

  • 價格突破前20日最高價 → 做多（買進1單位）
  • 價格跌破前20日最低價 → 做空（賣出1單位）

過濾規則（重要！）：

  • 如果上一次20日突破導致了獲利交易，則這次突破信號被忽略
  • 目的：避免重複進入已經走完的趨勢，減少假突破
  • 如果突破信號被忽略，可以在55日突破時進場（自動保險突破點 Failsafe Breakout）

怎麼判斷上次突破是否成功：

  • 如果在10日出場之前，突破日之後的價格與部位方向相反波動了2N（2倍ATR），則視為失敗的突破
  • 失敗的突破不會使下一次突破被忽略

系統二：55日突破（長線系統）

進場規則：

  • 價格突破前55日最高價 → 做多（買進1單位）
  • 價格跌破前55日最低價 → 做空（賣出1單位）

過濾規則：

  • 所有55日突破都必須接受，不管上次突破是成功還是失敗
  • 目的：確保不會錯過大型趨勢

雙系統配置建議

  • 激進配置：100%資金用系統二（55日突破）
  • 平衡配置：50%系統一 + 50%系統二
  • 保守配置：100%系統一（20日突破）

實例

台積電 2024年案例：

  • 3月11日突破前20日高點595元 → 系統一進場做多
  • 如果上次20日突破已獲利，忽略此信號，等55日突破
  • 4月突破前55日高點620元 → 系統二進場做多（不受過濾限制）

適用時機

  • 趨勢明確的市場最適合，盤整市容易被假突破反覆洗出
  • 適合流動性好的標的（大型股、期貨、ETF），不適合冷門股
  • 台股0050、台積電、聯發科等大型權值股最適合

三、N值與部位管理：海龜的風險控制核心

N值是什麼

N值就是ATR（Average True Range，真實平均波動區間），海龜用N來衡量市場波動性，並以此計算部位大小和停損距離。

公式：

  • True Range = max(H-L, H-前收盤, 前收盤-L)
  • N = True Range的20日指數移動平均

部位計算（最核心的公式）

海龜的部位管理原則：任何一筆交易的風險不超過帳戶淨值的2%

單位部位公式：

單位 = 帳戶淨值 × 1% / (N × 每點價值)

怎麼算：
1. 確定帳戶淨值（例如100萬）
2. 計算1%風險金額 = 1,000,000 × 1% = 10,000元
3. 找出標的的N值（ATR）
4. 計算每單位的合約價值
5. 單位數 = 10,000 / (N × 每點價值)

實例：

  • 帳戶淨值：100萬元
  • 台積電股價：600元，N值（ATR）= 15元
  • 1%風險 = 10,000元
  • 單位部位 = 10,000 / 15 ≈ 666股 ≈ 0.67張
  • 最大4單位 = 0.67 × 4 ≈ 2.66張

重點：波動大的標的（N值大），每單位買的少；波動小的標的（N值小），每單位買的多。這就是「波動大少買，波動小多買」的海龜精神。

加碼規則

  • 在突破時建立1個單位
  • 之後每上漲0.5N加碼1個單位
  • 最多加碼到4個單位

加碼實例（原油）：

  • N = 1.20，55日突破 = 28.30
  • 第1單位：28.30（進場價）
  • 第2單位：28.30 + 0.5 × 1.20 = 28.90
  • 第3單位：28.90 + 0.5 × 1.20 = 29.50
  • 第4單位：29.50 + 0.5 × 1.20 = 30.10

加碼實例（黃金）：

  • N = 2.50，55日突破 = 310
  • 第1單位：310.00
  • 第2單位：310.00 + 0.5 × 2.50 = 311.25
  • 第3單位：311.25 + 0.5 × 2.50 = 312.50
  • 第4單位：312.50 + 0.5 × 2.50 = 313.75

四、停損系統：2N停損與雙重損失

標準2N停損

  • 多頭停損：進場價 - 2N
  • 空頭停損：進場價 + 2N
  • 每個單位風險 = 2N = 帳戶淨值的2%

加碼後停損調整：

  • 每增加一個單位，前面所有單位的停損往上移0.5N
  • 最終全部部位的停損 = 最近一個單位的進場價 - 2N

原油加碼停損實例：

單位進場價原始停損調整後停損
第1單位28.3025.90—
第2單位28.90—26.50
第3單位29.50—27.10
第4單位30.10—27.70

加碼4單位後，全部停損在27.70（最近單位30.10 - 2×1.20 = 27.70）

雙重損失（Whipsaw）備選策略

  • 更積進但更精準的停損方式
  • 停損設在0.5N（帳戶風險0.5%），比標準2N更緊
  • 如果被停損出場，但價格回到進場價，重新建立部位
  • 好處：最大4單位的總風險仍不超過2%（4 × 0.5% = 2%）
  • 壞處：更容易被洗出場，勝率較低，但獲利交易的平均報酬更高

雙重損失停損實例：

單位進場價停損（0.5N）
第1單位28.3027.70
第2單位28.9028.30
第3單位29.5028.90
第4單位30.1029.50

五、出場系統：10日/20日反向突破

系統一出場

  • 多頭出場：價格跌破前10日最低價 → 全部平倉
  • 空頭出場：價格突破前10日最高價 → 全部平倉

系統二出場

  • 多頭出場：價格跌破前20日最低價 → 全部平倉
  • 空頭出場：價格突破前20日最高價 → 全部平倉

出場為什麼這麼難

海龜系統出場可能是整個系統最難執行的部分：

> 「你可能要看著10%到30%的利潤變成小幅虧損。在趨勢中期，可能要看著80%到100%的利潤回檔30%到40%。」

這就是海龜系統的核心矛盾：

  • 大多數突破不會形成趨勢（勝率低，大約20-40%）
  • 少數大趨勢賺的錢要彌補所有小虧損
  • 如果過早出場，就會錯過讓利潤奔跑的大趨勢
  • 如果不出場，就要忍受巨大的利潤回吐

雙海龜分批出場法（台股實戰改良）

原版海龜是全部平倉，但台股實戰可以改良為分批出場：

「210」策略：

  • 買2個單位
  • 短線海龜線出場1個單位（鎖定部分利潤）
  • 長線海龜線出場1個單位（讓利潤奔跑）
  • 最差情況：第一單位賺、第二單位停損 → 整體不虧

海龜線設定：

  • 短線：N根K棒高低點 + 緩衝（例如5分K 6根 + 25點）
  • 長線：M根K棒高低點 + 緩衝（例如5分K 12根 + 60點，或日K 2根 + 40點）
  • 緩衝大小根據波動調整：波動大→緩衝大，波動小→緩衝小

六、海龜法則的勝率與期望值

關鍵數字

  • 勝率：約20-40%（大多數交易虧損）
  • 賺賠比：約3:1到5:1（少數大趨勢彌補多數小虧損）
  • 年均報酬：海龜計畫期間年化複利80%
  • 最大回撤：可達30-50%

為什麼勝率低但能賺錢

海龜系統的邏輯：用很多小虧損換少數大獲利

如果勝率30%、平均虧損2N、平均獲利6N：

  • 10次交易：7次虧損 = -14N，3次獲利 = +18N
  • 淨結果 = +4N
  • 期望值為正，長期穩定獲利

勝率迷思

> 「90%的獲利交易」這種說法極具誤導性。勝率的百分比根本沒有意義。一個勝率25.9%的系統可能比勝率41.1%的系統獲利更好——如果它的賺賠比夠高。

  • 日均成交量大於5,000張（滑價小）
  • 有明顯趨勢性的標的（0050、台積電、聯發科等）
  • 避免盤整股和消息面主導的投機股

完整SOP：台股海龜交易流程

Step 1：選股

  • 從0050成分股或大型股篩選
  • 計算20日和55日最高/最低價
  • 設定突破提醒

Step 2：計算部位

  • N值 = 20日ATR
  • 單位 = 帳戶淨值 × 1% / N
  • 最大4單位，單一標的不超過帳戶20%

Step 3：進場

  • 突破20日高點 → 系統一進場（檢查上次突破是否獲利）
  • 突破55日高點 → 系統二進場（無過濾）
  • 日間突破即進場，不等待收盤

Step 4：加碼

  • 每上漲0.5N加碼1單位
  • 最多4單位

Step 5：停損

  • 標準：進場價 - 2N
  • 加碼後：全部停損移至最近單位進場價 - 2N

Step 6：出場

  • 系統一：跌破前10日最低價 → 全部平倉
  • 系統二：跌破前20日最低價 → 全部平倉
  • 或用雙海龜分批出場：短線出50%、長線出50%

心得

海龜交易法則給我最大的震撼是它的完整性。大多數人學交易只學「進場」，但海龜把六個決策（市場、部位、進場、停損、出場、策略）全部量化，每個環節都有明確的公式，沒有任何主觀判斷的空間。

最核心的概念是「N值部位管理」：波動大少買，波動小多買。這顛覆了很多人「看好就重押」的直覺。用N值計算部位，讓每筆交易的風險都控制在2%以內，不管標的波動大還是小。這才是真正的風險管理——不是「感覺」風險大小，而是用數字量化風險。

另一個啟發是勝率迷思。海龜系統勝率只有20-40%，但靠少數大趨勢賺的錢彌補所有小虧損。這代表大多數時候你在虧錢，但少數幾次讓你大賺。這需要極強的心理素質——連續虧損10次還能堅持系統的人，才是真正能賺錢的人。

雙重損失（Whipsaw）策略很巧妙：0.5N的緊停損，被洗出場就重新進場。看起來很矛盾（不是才說不要過早出場嗎？），但邏輯是：用更小的虧損試錯，被洗了就再來一次，反正每次只虧0.5%。這給了紀律不夠強的人另一個選擇。

最後，出場是最難的。看著80%的利潤回吐到40%還不出場，這違反人性。但海龜的出場邏輯很清楚：趨勢不結束就不出場，用10日/20日反向突破來確認趨勢結束。不用預測頂部，讓市場告訴你什麼時候該走。雙海龜分批出場是很好的改良——先鎖定一部分利潤，剩下的讓它跑，心理壓力小很多。

一、左側交易vs右側交易：你到底是哪一派？

是什麼

左側交易和右側交易是兩種根本不同的操作哲學，差異不在「賺不賺得了」，而在你「什麼時候進場、什麼時候出場」。搞清楚自己屬於哪一派，比學任何技術指標都重要——因為左右側混用，是散戶最常犯的致命錯誤。

左側交易（逆勢交易）：在趨勢反轉「之前」進場。股價還在跌就開始買，股價還在漲就開始賣。代表人物：巴菲特、日本BNF（小手川隆）。

右側交易（順勢交易）：在趨勢「已經確認」後才進場。股價突破才買，股價跌破才賣。代表人物：威廉·江恩、日本cis。

怎麼做

1. 左側交易的操作邏輯

核心思想：預判市場反轉，逆勢佈局。

  • 買進時機：股價跌到合理估值以下，分批買入
  • 賣出時機：股價漲超過合理估值，分批賣出
  • 不需要精確判斷最低點，靠分批攤平成本
  • 停損概念：左側交易者通常不設傳統停損，因為下跌就是加碼機會

操作實例：

  • 某股從100元跌到50元，基本面良好
  • 左側交易者在55-50元分批買入
  • 分三批：55元買30%、52元買40%、50元買30%
  • 平均成本52.3元
  • 預期回升到80元，獲利空間52.7%
  • 風險：如果續跌到30元，套牢22.3元（42.7%虧損）

2. 右側交易的操作邏輯

核心思想：確認趨勢才進場，不預測、只跟隨。

  • 買進時機：股價突破關鍵壓力、均線金叉、趨勢確認
  • 賣出時機：股價跌破支撐、均線死叉、趨勢反轉
  • 停損概念：停損是右側交易的核心，趨勢錯了就認賠
  • 停利概念：右側交易者不主動停利，讓趨勢帶著走

操作實例：

  • 同一檔股票，右側交易者等股價突破60元壓力線才買進
  • 買在62元，停損設58元（跌破壓力線），風險4元
  • 趨勢持續到80元，獲利18元（29%）
  • 獲利空間比左側少，但風險也小很多（4元vs42.7%套牢）

4. 折衷派：左側預判＋右側確認（BNF後期進化）

這是很多成功交易者最終演化出的策略：

  • 左側用來「找機會區間」——基本面分析、估值判斷，鎖定潛在標的
  • 右側用來「確認進場時機」——等待技術訊號出現，才實際下單
  • 好處：避免左側的「套牢深淵」和右側的「追高被洗」

操作實例：

  • 基本面分析發現某股合理價值80元，目前50元，左側預判「被低估」
  • 但不急著買，等股價出現止跌訊號（K線反轉、量能縮減）才右側確認進場
  • 可能在55元進場，比純左側的50元高，但比純右側的60元低

左右側混用的致命錯誤

市場先生指出的最常見問題：

  • 用價值投資（左側）進場，卻設停損（右側做法）→ 低點被洗出場，然後看股價漲回去
  • 用順勢交易（右側）進場，卻過早停利（左側做法）→ 吃不到趨勢中後段的大利潤

核心原則：

  • 左側交易者：不要設停損，不要過早停利。下跌是加碼機會，上漲是分批出場機會
  • 右側交易者：一定要設停損，不要過早停利。趨勢不結束就不出場

適用時機

  • 左側適合：資金大、風險承受高、能長期等待、基本面分析能力強的人
  • 右側適合：資金小、風險承受低、需要快速周轉、技術分析能力強的人
  • 折衷派最推薦：左側找機會＋右側確認進場，兼顧成本和安全性
  • 市場環境：波動大的市場右側佔優，波動小的市場左側佔優
  • 風險承受度是選擇標準：承受度高→左側，承受度低→右側

二、箱型理論（Darvas Box Theory）：視覺化的趨勢交易系統

是什麼

箱型理論由Nicolas Darvas提出——一個職業舞者，在18個月內把1萬美元變成200萬美元。核心概念極度簡單：股價在「箱子」裡整理，突破箱頂就是買點，跌破箱底就是出場點。 不需要複雜指標，只需要「看出箱子在哪裡」。

箱型的形成

三個必要條件：
1. 固定高點與低點：價格在特定區間來回震盪，形成明顯的上下界線
2. 持續數天到數週：箱型持續時間越長，突破後的動能越強
3. 成交量逐漸萎縮：買賣雙方僵持，量能收斂=能量正在累積

箱型判定方法：

  • 箱頂（壓力線）：連續3個交易日不突破前高
  • 箱底（支撐線）：連續3個交易日不跌破前低
  • Darvas原文：連續3次觸及同一價位無法突破，即可確立箱型邊界

怎麼做

1. 箱型理論完整操作SOP

Step 1：選股

  • 成交量充足的股票（冷門股箱型無參考價值）
  • 成長股或熱門產業（需要動能突破箱型）
  • 股價來到歷史高價或52週高點附近（上升趨勢中的箱型最有意義）
  • 只在牛市操作（Darvas明確表示熊市不交易）

Step 2：找出箱型範圍

  • 畫法：找到一個高峰（箱頂）和一個低谷（箱底）
  • 用K線圖而非折線圖（需要看到最高價和最低價）
  • 箱型上下緣是「大概範圍」非精確數字，可抓0.3%誤差範圍
  • 箱型越窄越好（停損範圍小）
  • 箱型持續越久越有參考價值

Step 3：設定進場點

  • 在箱型上緣上方一點點設限價買單
  • 例：箱型30-40元，在41元設限價單
  • 向上突破必須伴隨成交量放大，否則可能是假突破

Step 4：設定停損點

  • 初始停損設在箱型下緣下方
  • 例：箱型30-40元，停損設29元
  • 最大損失=箱型上下緣的距離（40-30=10元）
  • 進場當下就同時設好停損單

Step 5：箱型向上堆疊

  • 突破後進入新箱型，把停損點上移到新箱底下緣
  • 這就是「移動停損」的概念
  • 出現箱型向上堆疊=做對了！

Step 6：出場

  • 跌破新箱型下緣→出場
  • 不要隨意止盈！順勢交易低勝率，小賺就出場長期必定虧損

2. 箱型理論三階段

階段狀態投資人行為
建立箱型價格區間明確形成觀望，等待方向出現
突破箱頂突破上緣並放量進場訊號，趨勢啟動
跌破箱底價格跌破下緣停損出場或觀望

3. 進階操作：箱型內區間交易

除了Darvas的突破做多法，也有其他門派在箱型內做區間交易：

  • 在箱型下緣附近買進
  • 在箱型上緣附近賣出
  • 適合盤整市，但風險是突破方向不確定
  • 建議先觀察1-2個箱型區間的走勢方向，再順勢操作

4. 箱型理論的停損系統

停損類型位置使用時機
初始停損買進價位下一檔（箱底下緣下方）進場當下立即設定
移動停損新箱型下緣下方箱型向上堆疊後調整
優化停損新箱型上緣下方（而非下緣）想縮小虧損範圍時

操作實例：

  • 某股在30-40元箱型整理3個月
  • 突破40元到45元後進入45-55元新箱型
  • 第一箱型進場41元，初始停損29元
  • 第二箱型成立後，停損上移到44元（新箱底下緣下方）
  • 如果優化，可上移到54元（新箱頂下方）
  • 最終跌到55元跌破新箱底→出場，獲利14元（34%）

箱型理論的限制

1. 只適用牛市做多：熊市中箱型向下突破的機率高，做多會被反复停損
2. 支撐壓力線判斷主觀：每個人畫的箱型可能不同
3. 假突破問題：盤中突破不代表收盤也能站穩，建議用收盤價確認
4. 滑價成本高：順勢追價策略的通病，突破時買價可能比預期高
5. 勝率低：30-40%勝率是常態，靠少數大獲利彌補多數小虧損
6. 箱型範圍太大：上下緣差距過大時，停損距離太遠，風險過高

適用時機

  • 最適合：牛市中的成長股，有基本面支撐，成交量充足
  • 不適合：熊市、冷門股、波動太小的牛皮股
  • 搭配基本面：選基本面正在轉好的公司，再透過箱型理論做進出場
  • 搭配成交量：向上突破量增確認，向下跌破量縮確認

三、左側×右側×箱型：三維度操作策略整合

操作策略選擇決策樹

市場狀態建議策略理由
牛市＋成長股右側交易＋箱型理論順勢做多，箱型突破進場
牛市＋價值股左側交易＋分批加碼逢低買入好公司，等待價值回歸
盤整市左側交易或箱型區間操作右側在盤整市會被反復洗出
空頭市右側交易做空或空手觀望左側抄底可能越抄越深

箱型理論的左右側視角

箱型理論本質上是右側交易（順勢突破），但可以融合左側思維：

純右側箱型操作：

  • 等突破箱頂才買進
  • 停損設箱底下緣
  • 優點：確定性高
  • 缺點：成本較高、假突破風險

左側預判＋右側箱型確認：

  • 基本面分析找到被低估的股票（左側）
  • 等股價形成箱型整理（等待市場消化）
  • 突破箱頂確認進場（右側）
  • 優點：買在趨勢起漲點附近，成本比純右側低
  • 缺點：需要更多耐心等待箱型形成

操作實例：

  • 基本面分析發現某AI概念股合理價值100元，目前60元
  • 左側預判：這檔股票被低估，列入觀察名單
  • 等待2個月，股價在55-65元箱型整理
  • 某天帶量突破65元箱頂→右側確認進場
  • 停損設54元（箱底下緣下方）
  • 風險11元（65-54），預期目標100元，風報比=35:11≈3.2:1

心得

左側vs右側這個主題，最大的收穫不是「哪個比較好」，而是搞清楚自己是哪一派，然後貫徹到底。市場先生那句話真的打臉很多人：用價值投資進場卻設停損，或用順勢交易進場卻過早停利，最後得到兩邊的缺點，優點完全沒得到。這就是多數散戶的寫照——想左又想右，結果四不像。

BNF和cis的對比很有啟發性。BNF靠乖離率在恐慌中進場，160萬日圓變200億；cis靠技術突破嚴格停損，成為日本最紀律的個人交易員。兩個人方法完全相反，但都賺到爆。這證明沒有「唯一正確」的方法，只有「適合你」的方法。但關鍵是：一旦選了，就要貫徹那套方法的完整邏輯，不能混搭。

折衷派「左側預判＋右側確認」是我目前最認同的方式。用基本面找到被低估的好公司，但不要急著買，等市場出現止跌訊號或箱型突破才進場。這樣既不會套牢太深，也不會追高被洗。

箱型理論則是一個很好的「進出場視覺化工具」。它的核心價值不在「預測」，而在「提供明確的進出場依據和風險控制」。箱型上緣以上買、下緣以下賣，邏輯簡單到不會因為情緒誤判。Darvas說的 "I let the market tell me what to do" 就是這個意思——不猜方向，等市場告訴你。

最後，箱型理論跟之前學的海龜交易法有異曲同工之妙：都是順勢突破系統、都有低勝率高風報比特性、都靠嚴格停損控制風險。差別在於海龜用唐奇安通道的20日/55日高低點當進出場，箱型理論用肉眼可見的支撐壓力區間。海龜更量化，箱型更直覺，兩者可以互補驗證。

操作策略：動能交易×均值回歸兩大策略流派

操作策略最根本的二元對立：動能交易相信「強者恆強」，均值回歸相信「物極必反」。這兩種策略各有適用場景，搞懂什麼時候用哪種，比死守一種重要一百倍。

一、動能交易（Momentum Trading）

是什麼

基於「趨勢將持續」的核心假設，買入價格走勢強勁的資產，在趨勢反轉或動能減弱時賣出。哲學就是「強者恆強，弱者恆弱」。

核心原理

市場短期並非完全效率，投資者的群體心理（追漲殺跌、FOMO）會導致價格趨勢在一定時間內自我強化，創造出可供利用的交易機會。

怎麼做

第一步：篩選具備動能的標的
1. 價格接近52週新高
2. 股價位於MA50和MA200之上，均線多頭排列
3. 成交量異常放大（價漲量增）
4. 相對強度（RS）遠超大盤

第二步：技術指標確認進場

  • MA黃金交叉：短期MA向上穿越長期MA
  • RSI持續>70：動能交易反其道，RSI維持70以上視為極強動能確認（不是超買賣出）
  • MACD柱狀體在0軸之上持續放大：多頭動能強勁
  • ADX>25：趨勢強度確認，數值越高趨勢越強

第三步：嚴格出場與風控

  • 停損：固定百分比（如-8%）或跌破關鍵均線
  • 移動停損：停損點隨股價上價上漲提高，鎖定獲利
  • 停利：預期目標（如20%）或動能衰退信號出現時分批出場

實例

  • 情境：台積電（2330）突破前波高點680元，成交量放大1.5倍，RSI維持在72
  • 操作：在685元進場，停損設在650元（跌破MA20），移動停利設在5%回撤
  • 預期：若動能持續，目標看750元；若RSI從高檔回落+量價背離，分批出場

適用時機

  • ✅ 趨勢明確的市場（多頭或空頭皆可）
  • ✅ 營收成長+法人買超的強勢股
  • ❌ 盤整市場（無明確方向，會被多空雙巴）
  • ❌ 市場反轉點（動能崩潰風險最高）

動量崩潰（Momentum Crashes）

致命風險：動量崩潰是動能交易最恐怖的敵人。

  • 2009年3-5月：過去表現好的組合上漲8%，表現差的組合上漲163%，動量策略淨損失-155%
  • 發生時機：市場持續下跌後準備回升的時間點
  • 根本原因：動量策略本質是「買低Beta、空高Beta」，當市場反轉回升時，策略擁有極大負Beta值
  • 熊市特徵：動量策略在熊市呈現「賣買權」報酬型態——上漲時賺有限，反轉時虧無限

防範方法：
1. 加入波動率預測因子（VIX指標）
2. 熊市中降低動量策略曝險
3. 設定絕對停損，不要硬扛

二、均值回歸交易（Mean Reversion Trading）

是什麼

基於「價格偏離均值後終將回歸」的統計假設，在超買時賣出、超賣時買入。哲學是「物極必反」。

核心原理

AR(1)過程中，當φ（自回歸係數）的絕對值小於1，時間序列具有靜止性（stationarity），會回歸均值。φ越接近0，均值回歸越強；φ=1則是隨機走勢，不會回歸。

怎麼做

方法一：單一標的均值回歸
1. 計算歷史均值和標準差
2. 價格跌破均值-2σ時買入
3. 價格回到均值時賣出
4. 停損設在均值-3σ（承認均值回歸失效）

方法二：配對交易（Pairs Trading）
1. 找到兩檔高相關性的標的（如台積電vs聯電）
2. 用共整合ADF檢定確認兩者價差具均值回歸性
3. 當價差偏離均值+2σ以上：賣出相對強勢股、買入相對弱勢股
4. 價差回歸均值時平倉獲利

方法三：布林通道反向操作
1. 僅適用於共整合關係的配對標的
2. 殘差觸及布林上軌→賣出symbol1、買入symbol2
3. 殘差觸及布林下軌→買入symbol1、賣出symbol2
4. 殘差回到中軌→平倉

實例

  • 情境：台積電（2330）和聯電（2303）歷史價差均值5元、標準差2元
  • 操作：某日價差擴大到11元（均值+3σ），賣出台積電同時買入聯電
  • 預期：價差回歸5元附近時平倉，每組獲利約6元
  • 停損：價差繼續擴大到15元（均值+5σ），承認結構性改變

ADF檢定實戰

判斷步驟：
1. 用Python的statsmodels庫跑ADF檢定
2. 看p-value：小於0.05→拒絕隨機走勢假設→可能具均值回歸性
3. 看ADF統計量：小於5%臨界值→靜止過程→均值回歸成立

共整合ADF檢定（配對交易用）：
1. 對兩個時間序列做線性回歸
2. 對殘差跑ADF檢定
3. p-value<0.05→兩者存在共整合關係→配對交易可行

適用時機

  • ✅ 盤整市場（價格在區間內震盪）
  • ✅ 高相關性配對標的
  • ✅ 價格明顯偏離歷史均值
  • ❌ 趨勢明確的市場（均值會持續移動，回歸失效）
  • ❌ 結構性改變（公司基本面質變）

2. 不追高，等回調到MA20或布林中軌附近再進場
3. 本質上是「左側等回調+右側確認動能」的折衷派操作

心得

動能交易和均值回歸就像是武俠小說裡的兩大門派——一個是剛猛的降龍十八掌，一個是綿密的太極拳。多數人死守一派，結果在錯的市場用錯的策略，被揍得滿地找牙。

最核心的教訓是：先判斷市場狀態，再決定用什麼策略。 ADX>25用動能，ADX<20用均值回歸。這比任何單一策略都重要。

動量崩潰是動能交易最可怕的事——市場反轉時，你買的低Beta股不漲，你空的高Beta股噴出去，兩邊挨巴掌。2009年那個-155%的慘案，就是活生生的教材。所以在熊市末期，動能策略要大幅降倉或直接關掉。

均值回歸的陷阱是「這次不一樣」——當基本面結構性改變時，均值真的不會回來。所以停損一定要設，共整合關係也會斷裂，不能死等。

最實用的可能是混合策略：用動能選股（找強勢股），用均值回歸擇時（等回調進場）。這就是之前學的「突破回調進場法」的底層邏輯——本質上就是動能+均值回歸的混血兒。

操作策略：跳空缺口交易策略×多時間框架分析MTF

今天學兩個操作策略的核心工具：跳空缺口是K線圖上最直觀的訊號之一，多時間框架分析則是所有專業交易員都在用的「看大局」方法。一個是進出場的觸發器，一個是方向判斷的框架，組合起來就是完整的交易決策系統。

一、跳空缺口交易策略（Gap Trading）

是什麼

跳空缺口（Gap）是指K線圖上相鄰兩根K棒之間出現的價格真空地帶——今天最低价比昨天最高价还高（向上跳空），或今天最高价比昨天最低价还低（向下跳空），中間完全沒有交易發生。缺口就像市場情緒的放大鏡，直接反映投資人對突發消息的集體反應。

四大缺口類型

1. 普通缺口（Common Gap）

是什麼：出現在盤整區間的無意義缺口，像市場打了個噴嚏。

怎麼判斷：

  • 出現在無趨勢的橫盤區
  • 成交量無明顯變化
  • 通常幾天內就會回補

怎麼做：

  • 不需要特別操作，當市場雜訊處理
  • 唯一用途：在寬幅震盪行情中，普通缺口「必然回補」的特性可用來做區間交易——出現普通缺口後，等回補就是目標價

實例：

  • 某股在45-55元區間震盪，突然跳空開在53元（普通缺口）
  • 預期：幾天內回到50元附近回補缺口
  • 操作：在53元賣出或放空，目標50元

適用時機：寬幅盤整區間的差價交易，不適合趨勢追蹤

2. 突破缺口（Breakaway Gap）——含金量最高的缺口！

是什麼：股價在重要支撐壓力區盤整很久後，以跳空方式突破。代表新趨勢的起跑槍響。

怎麼判斷：

  • 出現在突破長期盤整區、壓力線或支撐線的位置
  • 伴隨巨大成交量（關鍵！）
  • 短期內不容易被回補
  • 缺口本身會變成新的支撐（向上突破）或壓力（向下突破）

怎麼做：

  • 進場：突破缺口形成當天收盤時輕倉進場，或隔幾天確認站穩缺口上方再買
  • 停損：設在缺口下緣——強勢突破缺口會形成支撐，跌破回補代表突破力道不如預期
  • 加碼：如果缺口沒回補，在5日均線附近穩定2-3天，果斷加倉

實例：

  • 某股在80-100元箱型整理3個月，某天跳空開在105元，成交量暴增3倍
  • 這是突破缺口，新趨勢啟動
  • 進場：105元買進1/3部位
  • 停損：設99元（缺口下緣附近）
  • 確認：3天後站穩108元，5日均線支撐，加碼至2/3部位
  • 目標：用箱型高度（100-80=20元）估算，目標約120元

適用時機：長期盤整後的突破點，配合大量確認最可靠

3. 持續缺口/逃逸缺口（Runaway/Continuation Gap）

是什麼：出現在已確立趨勢中途的缺口，代表趨勢力道強勁，還會繼續跑。

怎麼判斷：

  • 出現在明確上升或下降趨勢的中段
  • 成交量平穩或溫和放大（不像突破缺口那樣爆量）
  • 不一定會回補

怎麼做——測量缺口估算目標價：
1. 找到趨勢起點A到持續缺口出現點B的距離
2. 從持續缺口位置起算，加上同樣的距離，就是粗略的目標價C
3. 公式：C = B + (B - A)

實例：

  • 某股從50元起漲，在70元出現持續缺口
  • 起點到缺口的距離 = 70 - 50 = 20元
  • 目標價 = 70 + 20 = 90元
  • 操作：已在車上的抱緊，沒上車的可以在缺口附近回調時小量進場

適用時機：趨勢明確時的加碼確認和目標估算，不適合新手追高

4. 竭盡缺口（Exhaustion Gap）——反轉的強烈警訊！

是什麼：趨勢末端的最後煙火，燦爛卻短暫，通常是反轉的前兆。

怎麼判斷：

  • 出現在長期漲勢或跌勢的末段
  • 成交量異常放大（最瘋狂的時刻）
  • 缺口在極短時間內（幾天）就被迅速回補
  • 回補後，趨勢反轉的可能性非常高

怎麼做：

  • 進場：耐心等待缺口被回補確認後，才考慮反向操作
  • 停損：設在近期高點（做空）或低點（做多）之外，非常嚴格
  • 操作對象：漲幅或跌幅已大，且出現量價背離的個股

實例：

  • 某股從50元漲到150元，某天跳空開在155元（竭盡缺口），爆量
  • 3天後跌回148元，缺口被回補
  • 這是趨勢反轉的強烈訊號
  • 操作：在148元附近放空，停損設156元（近期高點上方），目標看130元

島狀反轉：竭盡缺口的進階版——先向上跳空（竭盡），盤整幾天後再向下跳空（突破），中間的K線像孤島。這是最強烈的反轉型態之一。

適用時機：趨勢末段的反轉操作，技術難度極高，新手不要輕易抄底摸頂

普通缺口盤整區無變化非常高區間交易目標價
突破缺口突破關鍵位爆量低順勢追進，停損設缺口下緣
持續缺口趨勢中段溫和較低估算目標價，抱緊或小量加碼
竭盡缺口趨勢末端爆量非常高且迅速等回補後反向操作

「缺口必補」的真相

不是鐵律，是傾向：

  • 普通缺口、竭盡缺口：回補機率極高
  • 突破缺口、持續缺口：不回補才是強的證明
  • 向上跳空後被回補 = 多頭力道減弱，可能是假突破
  • 向下跳空後被回補 = 空頭力道緩解，可能是假跌破

缺口交易台股SOP

1. 每日開盤前掃描：預估開盤跳空超過2%的個股
2. 辨識缺口類型：看位置（盤整中？趨勢中？趨勢末端？）+ 成交量
3. 突破缺口：輕倉追進，停損設缺口下緣
4. 持續缺口：估算目標價，調整停利
5. 竭盡缺口：等回補確認後反向操作
6. 普通缺口：忽略或做區間交易

二、多時間框架分析（Multi-Timeframe Analysis, MTF）

是什麼

多時間框架分析是指透過多個時間週期來分析同一個金融商品，通常由較大的時間框架開始（看大局），逐步縮小到執行交易的時間框架（找進場點）。核心口訣：大週期看方向、中週期判趨勢、小週期找進場。

為什麼需要多時間框架

只看單一週期的問題：

  • 15分鐘看到的多頭上漲，可能只是4小時級別的回調——方向判斷完全相反
  • 單一週期容易被雜訊誤導，連續停損
  • 無法判斷當前走勢在大格局中的位置

多時間框架的好處：

  • 拉高盈虧比——用大週期的方向+小週期的停損，盈虧比可達5倍以上
  • 更清晰且宏觀地分析市場
  • 避免被小週期的雜訊洗出場

三重時間框架分析法

常用框架組合：

  • 波段操作：月線 → 週線 → 日線
  • 短線波段：週線 → 日線 → 60分線
  • 當沖操作：日線 → 60分線 → 15分線
  • 極短線：4小時 → 15分 → 1分鐘

框架比例原則：相鄰兩個時間框架的比例建議在3-6倍之間，太近沒意義，太遠會混亂。

怎麼做

Step 1：大週期看方向（戰略層）

目的：確定市場的大方向，是偏多還是偏空？

  • 看均線排列：MA20 > MA60 > MA120 = 多頭架構
  • 看KD方向：月KD或週KD黃金交叉 = 偏多
  • 看價格位置：在長期均線之上 = 偏多
  • 看趨勢線：長期上升趨勢線未被跌破 = 偏多

Step 2：中週期判趨勢（戰術層）

目的：確認中期的趨勢狀態，是趨勢進行中還是即將反轉？

  • 看結構：高點更高、低點更高 = 上升趨勢
  • 看MACD：零軸之上 = 多頭趨勢
  • 看成交量：量價配合確認趨勢
  • 看型態：是否有頭肩頂/底、雙頂/底等反轉型態

Step 3：小週期找進場（執行層）

目的：找到精確的進場點和停損點

  • 等待回調到支撐位：如MA20、前波低點
  • 等待K線反轉訊號：如鎚子線、吞噬線
  • 等待指標轉強：如KD黃金交叉、RSI脫離超賣
  • 設定停損：放在小週期的支撐位下方，距離極小

實例：波段操作的多時間框架

情境：台積電（2330）

月線（大週期）：

  • MA20（月）持續上升，價格站在月MA20之上
  • 月KD黃金交叉
  • 結論：長期偏多架構

週線（中週期）：

  • 週MA5 > MA10 > MA20，均線多排
  • 週MACD在零軸之上，柱狀體放大
  • 結論：中期趨勢向上進行中

日線（小週期）：

  • 股價從680元回調到650元（接近MA20支撐）
  • 日KD在20附近即將黃金交叉
  • K線出現鎚子線止跌訊號
  • 結論：進場點出現！

操作：

  • 進場：652元（MA20附近+K線止跌確認）
  • 停損：645元（跌破MA20+近期低點下方），風險7元
  • 目標：700元（週線壓力突破目標），報酬48元
  • 盈虧比 = 48:7 ≈ 6.9:1

實例：當沖操作的多時間框架

4小時（大週期）：

  • 趨勢突破強高點，回踩起漲點收看漲PB線
  • 結論：4H級別偏多

15分鐘（中週期）：

  • 出現多空轉換，下方有未緩解的需求區
  • 結論：找需求區進場

1分鐘（小週期）：

  • 優化需求區，精確掛單
  • 停損設在需求區下方+點差
  • 第一停利：上方供給區（盈虧比15:1）
  • 第二停利：2倍趨勢延續（盈虧比30:1）

多時間框架的陷阱

1. 大中小週期相悖

  • 大週期偏多、中週期偏空、小週期偏多 → 方向混亂
  • 解法：以大週期方向為主，等中週期也轉向再進場

2. 週期拉太開

  • 月線+5分鐘 → 資訊斷層太大，無法銜接
  • 解法：相鄰框架比例3-6倍

3. 過度交易

  • 小週期太多雜訊，每個訊號都進場 → 連續停損
  • 解法：只做大週期方向一致的交易

4. 分析癱瘓

  • 看太多週期反而猶豫不決
  • 解法：固定3個框架，不要超過

長線保護短線的核心原則

玉山證券的文章總結得好：長期趨勢難以扭轉，因此當長期趨勢為多方架構，在較短週期內應以偏多思考為操作核心。

月線KD黃金交叉 + 週線KD黃金交叉 + 日線KD黃金交叉 = 三重共振最強買點

任何一個框架方向不一致，就要降倉或觀望。

三、缺口交易×多時間框架：組合實戰

突破缺口+多時間框架確認

最強的進場組合：
1. 週線：長期趨勢偏多（MA多排+KD金叉）
2. 日線：出現帶量突破缺口，突破長期盤整區
3. 60分線：缺口後回調不破缺口下緣，出現止跌K線
4. 進場：60分線止跌點，停損設缺口下緣

竭盡缺口+多時間框架確認反轉

1. 週線：趨勢已走很長，出現量價背離
2. 日線：出現爆量竭盡缺口，3天內回補
3. 60分線：跌破缺口後持續下跌，出現更低的高點和更低的低點
4. 確認反轉後反向操作

心得

跳空缺口這東西，以前只覺得是K線圖上的空白，沒想到背後有這麼多學問。四種缺口類型，四種完全不同的操作策略——普通缺口是雜訊，突破缺口是起跑，持續缺口是加速，竭盡缺口是煞車。搞錯類型就搞錯操作，把竭盡缺口當突破缺口追進去，就是站在趨勢末端的接盤俠。

「缺口必補」這句話要修正：不是所有缺口都會補，突破缺口和持續缺口不回補才是強的證明。如果一個向上突破的缺口很快被回補，那大概率是假突破。這個認知反轉很重要——以前可能看到缺口回補覺得「正常」，其實是突破失敗的訊號。

多時間框架分析是所有專業交易員的基本功，但多數散戶只用一個時間框架看盤。柴豬那個例子很震撼：4H看多、15分找需求區、1分鐘精確掛單，盈虧比15-30倍。關鍵在於用大週期的方向換取確定性，用小週期的停損換取盈虧比。同樣一筆交易，只看日線的人可能盈虧比3:1，加上多時間框架可以拉到10:1以上。

最實用的組合是：突破缺口+多時間框架確認。週線方向偏多，日線出現帶量突破缺口，60分線回調不破缺口，三重確認後進場。這就是把之前學的箱型理論、海龜突破法、均線進場法全部串起來——缺口是訊號觸發器，多時間框架是方向濾網，兩者結合就是勝率和盈虧比兼顧的交易系統。

一、開盤八法：15分鐘看懂今日盤勢

是什麼

開盤八法是台股市場老前輩的實戰經驗統計，透過觀察開盤後三個時間點（9:05、9:10、9:15）的指數漲跌，組合出8種可能的盤勢型態，藉此判斷當日多空方向和操作策略。

三個觀察時段：

  • A段：9:05收盤 vs 前日收盤（判斷開盤強弱）
  • B段：9:10收盤 vs 9:05收盤（判斷延續或反轉）
  • C段：9:15收盤 vs 9:10收盤（判斷確認或震盪）

每段只有漲或跌兩種可能，3段組合起來就是2³=8種型態，所以叫「開盤八法」。

怎麼做

第一法：漲漲漲（連續上攻型）

時段比較方向
9:05vs 前收漲
9:10vs 9:05漲
9:15vs 9:10漲

盤勢解析：開盤後連續上攻，短線賣壓會出籠。以9:10的收盤價為基準點不能跌破，拉回不破可作多。

  • 開大漲：有利多消息刺激，多半能開高震盪盤堅在高檔，多方操作較能獲利
  • 開小漲：無利多卻連漲15分鐘，10點前跌破9:15就守穩不了，多半開高走低，尾盤只成小漲

操作：短多可進場，停損設9:10低點下方。9:15後觀察是否帶量突破前高。

第二法：漲漲跌（多頭中段型）

時段比較方向
9:05vs 前收漲
9:10vs 9:05漲
9:15vs 9:10跌

盤勢解析：10分鐘連漲後拉回，多頭短衝後暫時無力上攻。觀察量能是否異常。

  • 開大漲：9:15下跌的成交量不應比9:10上漲的量大，否則跌破9:05機率高，偏向佈空
  • 開小漲：常在日K多頭格局時出現，盤中整理，9:15的收盤價可作為多空線，佈多較能獲利

操作：以9:05收盤為支撐，不跌破可偏多操作。

第三法：漲跌跌（上檔賣壓型）

時段比較方向
9:05vs 前收漲
9:10vs 9:05跌
9:15vs 9:10跌

  • 開小漲：拉回後三個時段都是壓力區，突破作多，跌破做空
  • 開大漲：拉回後盤中無法再突破9:15，尾盤多半小漲，偶爾拉尾盤

9:15vs 9:10漲

  • 開小跌：10:00前突破9:15且突破開盤價，盤中持續拉抬機率很高

操作：這是多方最佳進場時機！以9:10收盤價為支撐，不跌破就偏多操作。

  • 開大跌：以9:10為基準做多線，破9:05反向作空，量平觀望

二、日內交易時間帶策略

為什麼時間帶重要

台股交易時間9:00-13:30，共4.5小時。但成交量並非均勻分佈——開盤15分鐘和收盤15分鐘的成交量，佔全天成交量的30-40%。不同時間帶有不同特性，理解這些特性才能在對的時間做對的事。

六大交易時間帶

1. 開盤期（9:00-9:15）：最關鍵的15分鐘

特性：

  • 成交量最大、波動最劇烈
  • 反映隔夜消息面和國際股市影響
  • 散戶掛單集中成交的時段
  • 開盤八法的觀察窗口

操作策略：

  • 開盤八法判斷盤勢方向
  • 當沖者最重要的進場時段
  • 不建議在9:00-9:05追價（波動太大、滑價風險高）
  • 等到9:10-9:15確認方向再進場更穩妥

實例：

  • 開盤八法第四法（跌漲漲）：9:05開低，9:10翻紅，9:15繼續漲 → 9:15後偏多進場
  • 停損設9:10收盤價下方，目標看前日高點

2. 盤中主戰場（9:30-11:30）：趨勢確認期

特性：

  • 開盤情緒消化後，趨勢逐漸明朗
  • 最高點和最低點多半在10:00-11:30之間出現
  • 當沖的主要獲利時段
  • 量能穩定，技術指標參考性高

操作策略：

  • 一波流交易者的主要操作時段
  • 量能變化是關鍵：帶量突破前高作多、帶量跌破前低作空
  • 10:00前高低點通常是當日重要支撐壓力
  • 中盤（11:00-11:30）開始觀察是否需要減碼

實例：

  • 9:30突破9:15高點且帶量 → 確認偏多，一波流進場
  • 10:30量能開始萎縮且無法創新高 → 考慮減碼
  • 11:00出現更高高點但量縮 → 可能是當日高點附近
