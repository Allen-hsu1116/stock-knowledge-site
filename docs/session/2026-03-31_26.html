核心知識

股利折現模型（DDM）

是什麼：用未來會領到的股利，折現成現在的價值，來估算股票的內在價值。簡單說就是「這檔股票未來會發給我多少股利？這些錢現在值多少？」

核心概念：股票價格是市場供需的結果，不一定反映真實價值。股票的真正價值取決於公司持續經營能力，而這可以從股利發放看出來。

高登成長模型公式

公式：V₀ = D₁ ÷ (r - g)

各變數代表：

  • V₀：股票當前價值（內在價值/合理價）
  • D₁：明年預估股利（今年股利 × (1 + 成長率)）
  • r：折現率/預期報酬率（透過 CAPM 計算）
  • g：股利年成長率

怎麼做：
1. 找出公司今年股利
2. 預估未來股利成長率（保守估計）
3. 決定你的預期報酬率（r，通常用 CAPM 或自己設定）
4. 代入公式計算內在價值
5. 比較內在價值與市價，決定是否買進

實例：

  • 情境：某公司今年股利 2.85 元，預估成長率 5%，你要求報酬率 8%
  • 計算：D₁ = 2.85 × 1.05 = 3 元
  • 內在價值：V₀ = 3 ÷ (8% - 5%) = 3 ÷ 0.03 = 100 元
  • 判斷：如果市價 90 元 → 低於內在價值 → 可能被低估，值得考慮

DDM 的三種公式變形

#### 1. 單期模型 **公式**：V₀ = (D₁ ÷ (1+r)) + (P₁ ÷ (1+r))

適用時機：只打算持有股票一年，預估一年後的股利和賣出價格。

#### 2. 多期模型 **公式**：V₀ = D₁÷(1+r)¹ + D₂÷(1+r)² + ... + Dₙ÷(1+r)ⁿ + Pₙ÷(1+r)ⁿ

適用時機：預計持有多個時期，需要預測每一年的股利。

#### 3. 兩階段/多階段成長模型 **適用時機**：公司股利成長分不同階段。
  • 例如：前 5 年成長 10%，之後成長 5%，最後零成長
  • 比單一固定成長率更貼近現實

怎麼做：
1. 先預測高成長期（5-10 年）的股利
2. 用永續公式計算成熟期的終值
3. 將所有現金流折現加總

DDM 適用的公司類型

適合：

  • 穩定配息的成熟企業（公用事業、電信、銀行）
  • 股利政策穩定的公司
  • 獲利可預測的公司

不適合：

  • 成長型公司（配息不穩或完全不配息）
  • 不配息的公司（Google、Amazon、波克夏）
  • 股利政策劇烈變動的公司
  • 透過庫藏股回購代替配息的美股

DDM 的三大缺點

缺點 1：只能評估穩定配息的公司

  • 成長股通常不配息或配息不穩
  • 美股很多用庫藏股代替現金股利

缺點 2：需要大量假設

  • 未來成長率 g 是猜的
  • 折現率 r 也是猜的
  • 任何假設錯誤，結果就差很大

缺點 3：股利不一定反映真實獲利

  • 一次性的業外收入（賣土地）會讓股利暴增
  • 公司政策可能改變（台積電 2013 年前固定配 3 元，與獲利脫鉤）

DDM vs DCF 比較

項目DDMDCF
現金流來源股利自由現金流
適用範圍所有公司穩定配息公司
計算難度較簡單較複雜
估價結果較保守較靈活

怎麼選：

  • 公司穩定配息 → 用 DDM
  • 公司不配息或配息不穩 → 用 DCF

實戰注意事項

#### 1. 小心價值陷阱 股價下跌 + 獲利能力衰退的公司，DDM 算出來的「內在價值」可能是假的。

2. 一定要加安全邊際

  • 算出內在價值 100 元，不要 100 元就買
  • 等 80-90 元再買，給自己留 10-20% 的容錯空間

3. 檢查配息率是否穩定

  • 配息率（股利 ÷ EPS）應該落在穩定區間
  • 忽高忽低代表公司股利政策不穩

4. 預估要保守

  • 過去成長 10% ≠ 未來也會 10%
  • 更實際的做法：高成長 5 年 → 中成長 5 年 → 零成長
  • 沒有企業能永遠成長
#### 5. 耐心等待 DDM 算出來的內在價值通常偏保守，多頭市場時會覺得「沒東西可買」。這很正常，要耐心等價格回到合理區間。

心得

之前學了 DCF 用自由現金流估價，現在加上 DDM 用股利估價，工具箱更完整了。

DDM 的核心洞察很簡單：股票的價值就是未來所有股利的折現總和。但實務上，這個模型對台股存股族特別有用，因為台股很多公司（中華電、台積電、金融股）都有穩定的配息政策。

不過要小心幾個坑：
1. 成長股完全不適用 — 你不會用 DDM 去估特斯拉
2. 假設差一點，結果差很多 — g 從 5% 變 4%，內在價值可以差 20%
3. 安全邊際是必須的 — 模型只是估計，不是真理

我覺得最好的用法是：DDM 當作參考下限，DCF 當作上限，兩者之間就是合理價格區間。這樣既能抓到保守值，又不會錯過成長潛力。

本益成長比（PEG）是什麼？

是什麼：用來衡量「成長型股票」是否昂貴或便宜的指標。由英國投資大師吉姆・史萊特（Jim Slater）發明，彼得・林區推廣發揚。

為什麼需要：本益比（PE）只考慮公司當下獲利，沒考慮未來成長性。成長股因為預期未來獲利會大幅成長，本益比通常偏高，如果只用 PE 評估會覺得「太貴」而錯失投資機會。PEG 把成長率納入考量，更適合評估成長股。

PEG 計算公式

公式：

PEG = 本益比（PE）÷ 盈餘成長率（%）

計算步驟：
1. 取得本益比：股價 ÷ EPS
2. 取得盈餘成長率：建議用長期數據（過去5年或預估未來5年）
3. 相除得出 PEG

實例：

  • 情境：A公司股價100元，今年預估EPS為10元，預估稅後淨利成長率為20%
  • 本益比：100 ÷ 10 = 10倍
  • PEG：10 ÷ 20 = 0.5
  • 判斷：PEG < 0.75，股價被低估，可能是買進機會

注意：盈餘成長率的百分比數字直接相除（不是除以0.2，是除以20）

PEG 數值判斷標準

PEG 倍數股價狀態操作建議
> 1.2高估考慮賣出
= 1合理觀望
< 0.75低估買進
< 0.66嚴重低估強力買進

核心邏輯：PEG = 1 代表股價合理反映成長預期；低於1代表市場低估了成長性；高於1代表市場過度樂觀。

本益比 vs 本益成長比

項目本益比（PE）本益成長比（PEG）
用途評估股價便宜或昂貴評估成長股便宜或昂貴
考量因素當下獲利能力當下獲利 + 未來成長性
適用股票所有股票成長股
限制不考慮成長性需要正確預估成長率

什麼時候用 PEG：

  • 公司處於快速成長期
  • 本益比偏高但成長性也高
  • 想評估成長股是否被市場錯估

PEG 選股四大注意事項

1. PEG 不是越小越好

PEG 太小可能代表：

  • 公司獲利起伏大、不穩定
  • 股價可能先盛後衰
  • 單一年度數據波動造成誤判

實例：

  • 某公司去年EPS暴增300%，本益比30倍，PEG = 0.1
  • 看起來超便宜，但如果這300%是一次性獲利（賣資產），明年EPS回到正常水準
  • 真實PEG可能變成3倍以上，根本不便宜

2. PEG 只能用來分析成長股

前提條件：

  • 公司必須是成長股（前景看好）
  • 本益比必須為正值（不能是虧損股）
  • EPS 必須是成長的
  • 成長率要用長期數據估算

不適用：

  • 虧損公司（EPS為負）
  • 獲利衰退的公司
  • 成熟穩定的價值股（用本益比更適合）

3. PEG 可能選到衰退的公司

問題：PEG 用的是「預估」成長率，如果預估錯誤：

  • 預估成長20%，實際衰退10%
  • 計算時 PEG = 0.5，實際可能 > 2

解決方法：

  • 用歷史數據驗證成長率
  • 多種情境分析（樂觀、保守、悲觀）
  • 不要只看單一年度預估

4. PEG 可能錯失投資機會

台股中 PEG < 0.75 的時機並不多，如果堅持一定要買到 PEG < 0.66：

  • 可能長期空手
  • 錯過很多優質成長股
  • 長期報酬可能不如持有

實務建議：PEG 只是參考指標之一，不要當唯一標準。

實戰步驟

Step 1：確認是成長股

  • 公司所處產業是否有成長空間
  • 營收、EPS 是否連續多年成長
  • 公司是否有競爭優勢

Step 2：計算 PEG

  • 從財報網站查本益比（Goodinfo、財報狗）
  • 估算未來3-5年盈餘成長率
  • PEG = PE ÷ 成長率%

Step 3：交叉驗證

  • 同時看本益比是否合理
  • 檢查成長率來源（歷史數據？法人預估？）
  • 對照同業的 PEG 水準

Step 4：綜合判斷

  • PEG < 0.75 + 基本面佳 = 買進
  • PEG = 1 左右 = 持續觀察
  • PEG > 1.2 = 考慮賣出或避開

台股查詢管道

  • Goodinfo 台灣股市資訊網：可查詢個股本益比，再自行計算 PEG
  • 財報狗：付費版有 PEG 指標篩選功能
  • 計算方式：查到本益比後，預估盈餘成長率，自行計算 PEG

心得

PEG 是本益比的「成長版」，解決了「成長股本益比偏高不敢買」的問題。但核心難點在於「如何正確預估成長率」——這跟算命差不多難。

實戰上有幾個重點：
1. PEG 只能用在成長股，不要硬套在價值股或虧損股上
2. 成長率預估要保守，市場先生的建議是盡可能保守評估
3. 不要單看 PEG，要搭配本益比、產業分析一起看
4. 台股 PEG < 0.66 的機會不多，不要太執著於找到「完美便宜價」

最後，彼得・林區推廣的這套方法，本質是「用成長性來合理化本益比」。成長股本益比20倍看起來貴，但如果每年成長20%，PEG = 1 就是合理價。這個邏輯可以記住。
