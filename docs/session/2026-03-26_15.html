一、Covered Call 掩護性買權

是什麼

Covered Call（掩護性買權） = 持有現貨（股票/ETF）＋賣出相同標的的買權（Sell Call）

白話說：持有股票的同時，賣出 Call 選擇權收取權利金，就像「用股票收租」。

怎麼做

操作步驟：
1. 手上持有現貨部位（如 300 萬的 ETF）
2. 賣出對應規模的 Call 選擇權（Short Call）
3. 設定履約價在價平或略價外
4. 收取權利金，等待到期

設定要點：

  • 履約價：選略價外，權利金中時間價值比例較高
  • 到期日：短期（流動佳、時間價值衰減快）vs 長期（不用頻繁換倉）
  • 規模：選擇權曝險 ≤ 現貨規模（例：300 萬現貨 → 最多賣 4 口台指 Call @ 75 萬/口）

實例

情境：持有 C 公司股票成本 $25，賣出履約價 $27 的 Call，權利金 $1

三種結果：

股價結果損益
大漲（>$27）被履約，股票被買走賺價差 $2 + 權利金 $1 = $3（少賺）
緩漲（$25-$27）不履約，繼續持有賺價差 + 權利金 $1
下跌（<$25）不履約，持有股票虧損虧損 - 權利金 $1（小虧）

優缺點

優點：

  • 盤整緩漲：賺權利金
  • 大漲：仍獲利（但少賺）
  • 下跌：權利金彌補部分虧損

缺點：

  • 大漲時獲利有上限
  • 大跌時依然會虧損
  • 需要持續換倉（選擇權會到期）

適用標的

  • 全市場型 ETF（波動低）
  • 大型價值股（穩定）
  • 大型牛皮股（很少暴漲）

不適合：成長股、中小型股（容易暴漲）

適用時機

  • 判斷股價高檔，不易大漲
  • 盤整市場
  • 長期持有部位，短期不動

關鍵心法

風險本質：承擔「大漲錯失獲利」的風險，換取權利金收入。就像開保險公司——穩定收保費，但偶爾要賠大錢。

二、Straddle 與 Strangle（跨式與勒式）

是什麼

這是波動率策略——不預測漲跌方向，只預測「波動大小」。

跨式 Straddle：買 / 賣同一履約價的 Call + Put
勒式 Strangle：買 / 賣不同履約價的 Call + Put

四種策略總覽

策略組成使用時機風險特性
Long Straddle買 Call + 買 Put（同履約價）預期大波動損失有限，成本高
Long Strangle買 Call + 買 Put（不同履約價）預期大波動成本較低，需要更大波動
Short Straddle賣 Call + 賣 Put（同履約價）預期盤整勝率高，風險無限
Short Strangle賣 Call + 賣 Put（不同履約價）預期盤整勝率高，風險更大

Long Straddle 買進跨式

使用時機：

  • 重大政策/經濟數據發布前（利率、失業率）
  • 重要選舉前
  • 重大意外後（可能繼續崩盤或大反彈）
  • 盤整末期（三角收斂快突破）

優點：

  • 不用賭方向，漲跌都賺
  • 大波動報酬可觀
  • 風險可預期（最大虧損 = 權利金）
  • 不需高保證金

缺點：

  • 成本是單邊兩倍
  • 雙倍時間價值衰減

實戰技巧：短期持有幾天就平倉，不必抱到結算。

Short Straddle 賣出跨式

使用時機：

  • 預期進入盤整
  • 成交量低，預期小漲小跌

優點：

  • 勝率高，天天收時間價值
  • 盤整市場很常見

缺點：

  • 大漲大跌風險無限
  • 資金需求高
  • 太安逸容易忽視風險，一次意外就畢業

流動性風險：極端大跌時賣權變深價內，可能難以平倉。

Straddle vs Strangle 差異

核心差異：履約價相同 vs 不同

StraddleStrangle
履約價Call 和 Put 同一履約價Call 和 Put 不同履約價
成本較高較低
獲利門檻波動較小即可獲利需要更大波動
風險（賣方）風險較集中風險更分散但範圍更大

進階關聯

Short Straddle + 價外買 Call + 價外買 Put = Iron Condor（鐵兀鷹）
Short Strangle + 價外買 Call + 價外買 Put = Butterfly（蝴蝶）

實戰心法

買方策略：重大事件前佈局，事件後平倉賺波動率上升。
賣方策略：越接近結算，時間價值收越快，抱久一點（但看到危險要跑）。

近年變化：大家都事件前買雙買，事件後停損 → 隱波事件前上升、事件後下降。
→ 實務改為：更早買進，事件前平倉；事件後改賣方賺權利金。

心得

Covered Call 是「用股票收租」的概念，適合長期持有部位想增加額外報酬的人。核心是：願意犧牲大漲的獲利潛力，換取穩定的權利金收入。這策略最重要的是標的選擇——選那種不會暴漲的牛皮股，不然會哭。

Straddle/Strangle 是「不賭方向、賭波動」的思路。買方適合事件前佈局（賭大波動），賣方適合盤整期收時間價值（像開保險公司）。賣方策略最怕的是「太安逸」——平常穩穩賺，一次意外就畢業。

這幾個策略跟之前學的 Iron Condor 和 Theta 都有關聯：

  • Iron Condor = Short Strangle + 保護性買方
  • Covered Call = 現貨 + Short Call（損益特徵接近 Short Put）
  • Theta 衰變是賣方策略的利潤來源
