核心知識

順勢交易（Trend Following）

是什麼：一種追隨市場趨勢進行買賣的策略，核心思想是「市場價格有持續性」，上漲趨勢會繼續上漲，下跌趨勢會繼續下跌。

怎麼做：
1. 辨識市場趨勢（用趨勢線、均線等工具）
2. 確定進場和出場點（突破支撐/壓力時進場）
3. 設定停損點（入場時就設好）
4. 追蹤並調整策略（移動停損、加減碼）

實例：

  • BTC 價格突破壓力區時，跟著市場價格做多
  • 價格跌破支撐區時，順勢做空

適用時機：

  • 趨勢明確時（多頭排列或空頭排列）
  • 不適合盤整期（假突破多、容易被洗掉）

均線順勢交易

是什麼：利用均線作為順勢指標，均線往上就做多，往下就做空或空手。

怎麼做：
1. 畫出均線（如 40 日均線）
2. 均線往上 → 做多
3. 均線往下 → 做空或空手
4. 跌破均線時停損出場

實例：

  • 台積電跌破 40 日均線後一路往下 → 沒有持有必要，應該空手
  • 聚積漲過 40 日均線的這段才是應該持有的時間

適用時機：

  • 判斷行情轉向
  • 適用於大部分股票
  • 主力吃貨平均用 2 個月，40 日均線約等於 2 個月交易日

KD、RSI 順勢交易

是什麼：將 KD、RSI 從逆勢指標改為順勢指標，以 50 作為基準點。

怎麼做：
1. KD 或 RSI 大於 50 → 做多
2. KD 或 RSI 小於 50 → 做空
3. 出場可用黃金交叉/死亡交叉，或原本的超買超賣方式

實例：

  • 傳統用法：20 以下超賣買進，80 以上超買賣出（逆勢）
  • 順勢用法：指標大於 50 做多，小於 50 做空

適用時機：

  • 想用震盪指標做順勢交易時
  • 不想追高殺低時

亞當理論十大原則

是什麼：順勢交易的核心心法，強調「看到上漲就做多，看到下跌就做空」。

十大原則：
1. 賠錢的部位，絕不加碼或攤平
2. 開始交易或加碼時要設置停損點
3. 除非朝著交易相同方向，否則絕不取消或移動停損點
4. 絕對不讓損失擴大
5. 任何一筆交易絕不讓自己虧損交易資金 10% 以上
6. 不猜頭摸底，讓市場自己去走
7. 不逆勢操作，除非有強烈證據顯示反轉已發生
8. 保持彈性，世上沒有絕對的事
9. 交易不順時縮手休息
10. 認清自己想從市場上得到什麼

波段操作六大技巧

是什麼：介於短線和長期投資之間的策略，持股數天至數週，捕捉價格短期波動。

技巧一：只在賺錢時留倉

  • 賠錢就不應該留倉，應該直接停損
  • 一開始賠錢就不留倉，避免損失擴大

技巧二：事先設定出場點

  • 進場前先設停損
  • 獲利後設停利條件
  • 日盤、夜盤都要設

技巧三：跳空避險

  • 留倉時用選擇權買方鎖住獲利
  • 期貨做多 → 加買選擇權 BUY PUT 避險
  • 期貨做空 → 加買選擇權 BUY CALL 避險

技巧四：接受行情走不出去

  • 行情總是盤整跟趨勢交錯
  • 遇到盤整可用選擇權避險

技巧五：順勢留倉、慎選進場點

  • 行情偏多留多單，偏空留空單
  • 避免在差的位置進場（如跌深反彈區）

技巧六：停損不要設太小

  • 用平均波動的 1/5 來設定
  • 例如波動 300 點，停損設 60 點
  • 海龜法則：停損設很大，但只賠 1%

期貨波段操作三種策略

策略一：N 日均線波段交易

  • 價格跌破均線 → 做空進場
  • 價格突破均線 → 做多進場
  • 設定停損用 ATR 的 1/4 或 1/5

策略二：30 分 K + 均線操作

  • 用 60MA 作為移動停利參考
  • 移動停利與 60MA 保持一段距離（如 40 點）
  • 不要設定完全貼著均線，容易被掃出去

策略三：30 分 K + 海龜停利

  • 以區間 K 棒高低點往外加點數當移動停利
  • 例如 8 根 K 棒區間加 120 點
  • 配合獲利拉回保本出場

雙策略應對盤整：

  • 一個部位放短線（當沖）
  • 另一個部位放長線（留倉）
  • 短線沒獲利就不留倉
  • 長線用較大停損抱住

順勢交易的三大困難

1. 頻繁的假突破

  • 台指期常有假突破，突破後短時間反轉
  • 解法：等確認訊號、設合理停損

2. 需要耐心等待

  • 趨勢形成需要時間
  • 盤整期要忍受波動
  • 解法：培養耐心、不要急躁進場

3. 過度依賴技術分析

  • 可能忽略基本面、籌碼面
  • 台指期是淺碟型市場，易受消息影響
  • 解法：結合多種分析方法

心得

順勢交易和波段操作的核心都是「不要跟趨勢對著幹」。亞當理論說得直白：看到上漲就做多，看到下跌就做空。

波段操作最關鍵的是「賺錢才能留倉」。很多人賠錢時抱著希望明天會回來，結果越賠越多。正確做法是一開始就設好停損，賠錢就砍，賺錢才留。

期貨波段的三種策略其實都圍繞著一個核心：用均線或區間當作趨勢判斷工具，設定合理的停損停利，讓獲利持續奔跑。雙策略對付盤整盤也很實用——短線沒賺就跑，長線用大停損抱住。

最後，停損不要設太小。波動 300 點卻設 20 點停損，根本是找死。用 1/5 的波動當停損，給行情一點呼吸空間，才不會被正常震盪洗掉。
