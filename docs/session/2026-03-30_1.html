學習時間：01:03

文章：

型態學核心概念

什麼是型態學？

是什麼：透過大量歷史統計彙整，用眼睛辨識K線組合成的圖形樣態，判讀市場趨勢方向的技術分析方法。

三大觀察點：
1. 波段趨勢方向：下一個波段往哪走
2. 漲跌斜率與速度：價格發展的快慢
3. 目標價測量：估算漲跌空間滿足點

反轉型態：底部型態（買進訊號）

W底（雙重底）

是什麼：價格跌一段後反彈，再跌到前次低點附近，突破前高形成的W型態。

判斷要點：

  • 第2次低點通常高於第1次低點
  • 突破下降趨勢線 + 突破頸線
  • 成交量明顯放大才算確認
    頸線 ──────────────── 突破點 ★
       \        /      \
        \      /        \ 目標價 = 頸線 + (頸線-底部)
         \    /          \
          \  /            \
           \/              \
           底部              \

目標價計算：

  • 公式：目標價 = 頸線 + (頸線 - 底部)
  • 實例：頸線100元，底部80元 → 目標價 = 100 + (100-80) = 120元

進場時機：
1. 突破頸線確認後進場
2. 回測頸線獲支撐可加碼

停損設定：跌破頸線3%或前低

頭肩底

是什麼：低檔位置形成的「頭」低於「兩肩」的反轉型態。

結構組成：

  • 左肩：第一波下跌反彈
  • 頭：跌破前低後回升
  • 右肩：第三波下跌但動能趨緩

兩個買入訊號：
1. 跌破下降趨勢線
2. 突破頸線（需放量確認）

目標價計算：

  • 公式：目標價 = 頸線 + (頸線 - 底部)
  • 實例：頸線100元，底部80元 → 目標價120元

反轉型態：頭部型態（賣出訊號）

M頭（雙重頂）

是什麼：價格上漲到一定價位後回跌，再漲到前波高點附近，第二次下跌跌破頸線。

判斷要點：

  • 兩個高點差不多等高
  • 低點連線就是「頸線」
  • 跌破頸線 = 趨勢反轉
    頭部 ────/\────
              \  /\  
               \/  \ 頸線
                    \──────
                     \      目標價 = 頸線 - (頭部-頸線)

目標價計算：

  • 公式：目標價 = 頸線 - (頭部 - 頸線)
  • 實例：頸線100元，頭部120元 → 目標價 = 100 - (120-100) = 80元

三個賣出訊號：
1. 跌破上升趨勢線
2. 跌破頸線
3. 反彈測頸線失敗再跌

頭肩頂

是什麼：高檔位置形成的「頭」高於「兩肩」的反轉型態。

結構組成：

  • 左肩：獲利了結後回檔
  • 頭：續創新高後回檔
  • 右肩：上漲動能減弱，低於頭部

三個賣出訊號：
1. 跌破上升趨勢線
2. 跌破頸線
3. 反彈回測頸線後續跌

K線單根型態：看漲型態

錘形線（Hammer）

特徵：短實體 + 長下影線，位於下降趨勢底部

含義：盤中賣壓強，但買壓最終推升價格回升

實戰操作：

  • 位置：下降趨勢末端
  • 確認：隔日開高收紅
  • 進場：確認日收盤前
  • 停損：錘形線最低價

倒錘形（Inverse Hammer）

特徵：短實體 + 長上影線 + 短下影線

含義：賣壓不足以推動下跌，買盤重新控制市場

多頭吞噬（Bullish Engulfing）

特徵：兩根K線，紅短實體被綠長實體完全吞沒

含義：開盤低但收盤高，買家明顯獲勝

實戰操作：

  • 確認：第二根K線收紅且實體大
  • 進場：型態完成後
  • 停損：第二根K線最低點

穿刺線（Piercing Line）

特徵：長紅K + 長綠K，開盤有跳空缺口，收盤高於前一天中點

含義：買壓強勁，價格被推高至前一天中點以上

早晨之星（Morning Star）

特徵：三根K線組合 - 長紅K + 短K線 + 長綠K

含義：賣壓消退，看漲趨勢來臨

實戰要點：

  • 中間的「星」通常有跳空缺口
  • 第三根K線收盤越高，訊號越強

白色三兵（Three White Soldiers）

特徵：連續三根綠色長K線，開盤收盤都高於前一日

含義：下降趨勢後的強烈看漲訊號，買壓穩步上升

K線單根型態：看跌型態

上吊線（Hanging Man）

特徵：與錘形線形狀相同，但出現在上升趨勢頂部

含義：盤中有明顯拋壓，買家失去控制

射擊之星（Shooting Star）

特徵：短實體 + 長上影線，出現在上升趨勢頂部

含義：開盤走高但收盤僅略高於開盤，如流星墜地

空頭吞噬（Bearish Engulfing）

特徵：小綠實體被大紅實體吞沒，出現在上升趨勢頂部

含義：價格高峰，市場即將下滑

黃昏之星（Evening Star）

特徵：長綠K + 短K線 + 長紅K

含義：上升趨勢反轉，第三根K線越大越強

三隻烏鴉（Three Black Crows）

特徵：連續三根紅色長K線，影線很短或不存在

含義：賣壓連續三天超過買壓，下降趨勢開始

烏雲蓋頂（Dark Cloud Cover）

特徵：兩根K線，紅K開盤高於綠K，收盤低於綠K中點

含義：空頭接管，價格大幅走低

整理型態

十字星（Doji）

特徵：開盤價 ≈ 收盤價，實體極短

含義：買賣雙方掙扎，未分勝負

實戰操作：

  • 單獨出現：中性訊號
  • 搭配反轉型態：確認訊號增強

紡錘線（Spinning Top）

特徵：短實體 + 等長上下影線

含義：市場猶豫不決，盤整或休整期

下降三法（Falling Three Methods）

特徵：長紅K + 三根小綠K + 長紅K，綠K被覆蓋在紅K範圍內

含義：多頭無力扭轉趨勢，空頭延續

上升三法（Rising Three Methods）

特徵：長綠K + 三根小紅K + 長綠K，紅K被覆蓋在綠K範圍內

含義：賣壓存在，但買家仍保持控制

型態學六大常見誤區

1. 型態強行對號入座

問題：主觀套用型態，忽略圖形不完整

正確做法：

  • 確認型態要素是否完整
  • 檢查頸線是否明確
  • 時間對稱性是否合理

2. 忽略成交量確認

問題：只看圖形突破，忽略量能

正確做法：

  • W底突破頸線：必須爆量
  • M頭跌破頸線：需有量能
  • 無量突破 = 假突破，觀望等待

3. 單一型態孤立判斷

問題：只依型態決策，忽略大環境

正確做法：

  • 結合整體趨勢方向
  • 參考技術指標（RSI、MACD）
  • 注意消息面與資金動能

4. 忽略時間與比例對稱性

問題：型態過於急促或變形

正確做法：

  • W底：兩低點間距要合理
  • 頭肩型：左右肩要對稱
  • 比例失衡的型態可信度低

5. 忽視關鍵支撐/壓力位

問題：型態成形但遇前波壓力區

正確做法：

  • 檢查頸線位置有無前波壓力
  • 注意跳空缺口位置
  • 結合技術位置判讀

6. 過度依賴歷史重複性

問題：認為歷史會完全複製

正確做法：

  • 市場結構、資金流向不同會影響表現
  • 每次型態要重新評估當時背景

K線型態 + 技術指標搭配

型態 + 成交量

應用：所有反轉型態

  • W底突破頸線：爆量確認買盤進場
  • M頭跌破頸線：放量確認賣壓
  • 無量突破：假突破，觀望

型態 + RSI

應用：低檔布局、過熱判斷

  • W底 + RSI < 30：低檔反轉機率高
  • M頭 + RSI > 70：超買，反轉機率大

型態 + MACD

應用：確認動能支持

  • W底突破 + MACD黃金交叉：強力買進訊號
  • M頭跌破 + MACD死亡交叉：空頭確認

型態 + 均線

應用：確認中線趨勢

  • 突破頸線 + 站上季線：趨勢轉強
  • 跌破頸線 + 跌破長期均線：空頭確認

心得

型態學最怕「看圖說故事」——看到什麼就硬套成型態。這篇整理的重點在於：

1. 成交量是關鍵：沒有放量確認的突破，十有八九是假的。這跟之前學過的「量價關係」完全吻合。

2. 目標價公式要記住：

  • 反轉向上：目標 = 頸線 + (頸線 - 底部)

這比憑感覺猜漲跌有依據多了。

3. 不要單獨用型態：要搭配均線、RSI、MACD一起看。型態是方向，指標是確認。

4. 時間對稱性很重要：左肩3週，右肩最好不要只有3天。失衡的型態容易失敗。

5. 假突破/假跌破：最常出現在型態未確認 + 無量突破的情況。寧可等確認後進場，不要搶在型態完成前。

單根K線型態（錘形線、吞噬等）是型態學的基石，多根K線組合（W底、頭肩型）是進階。先搞懂單根，再玩組合。

布林通道是什麼

定義：由 John Bollinger 在 1980 年代發明的技術指標，用三條線組成的帶狀通道顯示價格波動範圍。

三條線組成：

  • 中軌：20 日移動平均線（20MA），反映價格整體趨勢
  • 上軌：中軌 + 2 個標準差（壓力線）
  • 下軌：中軌 - 2 個標準差（支撐線）

公式計算：

中軌 = 20 日收盤價總和 ÷ 20
上軌 = 中軌 + 2 × 標準差
下軌 = 中軌 - 2 × 標準差

統計原理：根據常態分配，95% 的價格會落在 2 個標準差範圍內，只有 5% 機率會跑到通道外。

兩大操作派別

1. 逆勢觀點（均值回歸）

核心邏輯：價格會回歸均值，95% 機率落在通道內，上下軌是極端價格。

適用：短週期、盤整市場

參數：N=20, K=2 或更大標準差倍數

操作方式：

  • 價格觸及上軌 → 考慮賣出
  • 價格觸及下軌 → 考慮買進
  • 價格回歸中軌時平倉

2. 順勢觀點（突破追勢）

核心邏輯：當大趨勢發生時，股價會大幅偏離舊有通道區間，沿通道上沿或下沿發展。

適用：長週期、趨勢明確市場

參數：N 更長（如 50），通道更窄

操作方式：

  • 價格突破上軌且通道擴張 → 順勢做多
  • 價格跌破下軌且通道擴張 → 順勢做空

六種基本買賣訊號

訊號判斷操作
股價由下往上穿越下線可能短期反轉觀察
股價由下往上穿越中線可能加速向上買進訊號
股價在中線與上線之間多頭行情持有多單
股價由上往下跌破上線上漲趨勢結束賣出訊號
股價由上往下跌破中線可能下跌賣出訊號
股價在中線與下線之間空頭行情持有空單

十種布林通道型態

1. 布林通道收縮（Bollinger Squeeze）

特徵：上下軌極度收窄，波動性降至低點，市場進入盤整。

實戰操作：

  • 通道持續收窄 → 關注突破方向
  • 價格帶量突破上軌 → 順勢做多
  • 價格帶量跌破下軌 → 順勢做空
  • 止損設在突破起點或中軌

注意：收縮本身無法判斷突破方向，需等待確認。

2. 布林通道擴張（Bollinger Expansion）

特徵：上下軌大幅張開，波動性急劇增加，伴隨強勢趨勢啟動。

實戰操作：

  • 確認價格趨勢方向（中軌之上做多、之下做空）
  • 價格穩定在中軌之上且通道擴張 → 買入訊號
  • 價格穩定在中軌之下且通道擴張 → 賣出訊號

3. 上軌觸碰或突破

特徵：價格觸及或突破上軌，顯示超買或強勢。

實戰判斷：

  • 強勢上漲趨勢：價格持續沿上軌運行 → 趨勢延續，可持有
  • 震盪或趨勢末期：觸及上軌 → 可能回調，賣出或做空時機

注意：不能簡單將上軌突破等同於買入訊號，需結合趨勢判斷。

4. 下軌觸碰或突破

特徵：價格觸及或跌破下軌，顯示超賣或強勢下跌。

實戰判斷：

  • 強勢下跌趨勢：價格持續沿下軌運行 → 趨勢延續
  • 震盪或下跌末期：觸及下軌 → 可能反彈，買進時機

5. W型底部（W-Bottom）

特徵：在下軌附近形成「W」雙底，第二個底部高於第一個。

實戰操作：

  • 第一個底部觸及下軌
  • 第二個底部更高 + 動量指標（RSI/MACD）底背離
  • 價格突破頸線（兩個低點之間的反彈高點）→ 做多訊號

6. M型頂部（M-Top）

特徵：在上軌附近形成「M」雙頂，第二個頂部低於第一個。

實戰操作：

  • 第一個頂部觸及上軌
  • 第二個頂部更低 + 動量指標頂背離
  • 價格跌破頸線 → 做空或離場訊號

7. 走廊運行（Walking the Bands）

特徵：價格緊貼上軌或下軌持續運行，趨勢極端強勁。

實戰操作：

  • 沿上軌運行 → 強烈上漲，順勢持有
  • 沿下軌運行 → 強烈下跌，順勢做空
  • 不要逆勢猜頂摸底
  • 價格顯著脫離軌道回調至中軌 → 動能可能減弱

8. 布林反轉（Bollinger Bounce）

特徵：價格觸及上下軌後像彈簧般快速反彈。

實戰操作：

  • 常見於盤整市場
  • 觸及下軌後反彈 → 買進訊號
  • 觸及上軌後回落 → 賣出訊號
  • 結合成交量或 RSI 確認
  • 不適用於強勢趨勢（容易假突破）

9. 布林假突破（Bollinger Fakeout）

特徵：價格短暫突破通道邊界後迅速反向回歸。

識別方法：

  • 以較小動能（縮量）刺穿軌道
  • 無法維持，快速收盤於通道內
  • 原突破方向力量衰竭

實戰操作：

  • 假突破後結合 K 線反轉信號（Pin Bar、吞沒）
  • 順勢反向操作
  • 目標看向通道另一側或中軌

10. 收縮性突破（Breakout After Squeeze）

特徵：通道持續收窄後，價格帶量強勢突破。

實戰操作（高勝率訊號）：

  • 等待通道極度收縮
  • 確認帶量突破方向
  • 順勢入場
  • 止損設在突破起點或中軌

參數設定實戰

預設參數：N=20, K=2（創始人設計，市場最常用）

參數調整參考：

交易時間N 參數K 參數特性
短線10-202-2.5敏感度高，訊號多
日線（預設）202平衡敏感度與穩定性
周線及長線20-502-3減少噪音，方便趨勢判斷

核心建議：使用預設參數 20 和 2，原因：
1. 避免過度擬合
2. 市場最多人使用的參數，會自我強化

布林通道優缺點

優點：

  • 三條線都可作為交易訊號參考
  • 清楚反映安全震盪區間
  • 可用於設定停損停利點位
  • 結合均線和標準差，資訊豐富

缺點：

  • 市場劇烈震盪時可能無法及時反應
  • 短週期下帶寬變化大，參考價值降低
  • 開盤跳空會造成帶寬突然放大
  • 無法判斷未來價格走勢方向（需搭配基本面）
  • 不能單獨使用，需搭配其他指標（MACD、RSI、成交量）

實戰心法

趨勢判斷

  • 價格在中軌與上軌之間 → 多頭趨勢
  • 價格在中軌與下軌之間 → 空頭趨勢
  • 中軌方向向上 → 上漲趨勢
  • 中軌方向向下 → 下跌趨勢

入場時機

  • 上下軌收窄時入場 → 降低浮虧風險
  • 等待突破確認 → 順勢操作
  • 結合成交量確認 → 提高勝率

風險管理

  • 不逆勢交易（以中軌判斷趨勢）
  • 價格突破下軌不追空（潛在反彈風險）
  • 觸及上軌不追高（可能回調）
  • 搭配基本面判斷未來方向

心得

布林通道的精華在於「波動性」的概念。它不是告訴你價格「會」往哪裡走，而是告訴你價格「現在」相對於平均的位置。

最實用的場景是：
1. 通道收縮後的突破：這是高勝率訊號，市場從盤整理轉趨勢
2. 走廊運行：強勢股會沿著上軌跑，不要隨便摸頂
3. 假突破辨識：縮量突破後快速回歸通道，是反向操作機會

關鍵心法：布林通道是「波動率指標」不是「方向指標」。它告訴你價格是否偏離均值，但方向要靠趨勢（中軌方向）和基本面來判斷。

另外，不要隨便改參數。20 和 2 是市場共識，共識越強，指標越有效。

第 3 回合學習：技術分析：ATR 真實波動範圍

學習時間：03:23

文章：

  • ATR指標完整教學 - 量化通 QuantPass
  • ATR指標教學：止損與部位計算 - OANDA Lab
  • ATR 損失規避策略 - TEJ台灣經濟新報

核心知識

ATR 基本概念

是什麼：ATR（Average True Range，真實平均波動區間）是由技術分析大師 J. Welles Wilder 於 1970 年代開發的波動性指標，用來衡量價格在一段時間內的「真實」波動幅度。

為什麼叫「真實」：因為 TR（True Range）計算時納入了跳空因素，比單純用最高價減最低價更能反映實際波動。

TR（True Range）計算公式

三種情況取最大值：
1. 當日最高價 - 當日最低價
2. |當日最高價 - 前日收盤價|（跳空高開）
3. |當日最低價 - 前日收盤價|（跳空低開）

實例：

  • 情境：某股票前日收盤 100 元，今日最高 108 元、最低 105 元
  • TR = max(108-105=3, |108-100|=8, |105-100|=5) = 8 元
  • 因為跳空高開，TR 比日內波動 3 元更大

ATR 公式

計算方式：

ATR = [(前一天 ATR × (n-1)) + 當天 TR] / n

參數設定：

  • 預設週期：14 天
  • 建議搭配 20 日均線觀察

實戰應用

一、波動性停損

怎麼做：
1. 進場時記錄當時 ATR 值
2. 停損點 = 買進價 - k × ATR
3. k 值依波動度調整（波動大用大 k，波動小用小 k）

實例：

  • 情境：台積電 600 元買進，ATR = 15 元
  • 設定：k = 2（保守）
  • 停損點 = 600 - 2×15 = 570 元
  • 含義：股價跌破 570 元才停損，避免被正常波動震出

棘輪止損法（移動停損）：

  • 持倉期間，每增加 1 倍 ATR 獲利，上調停損
  • 公式：新停損 = 最近低點 + (持倉天數 × 0.05 × ATR)
  • 目的：鎖定浮動盈利，讓利潤奔跑

實例：

  • 情境：GBPUSD 在 1.30598 做多，ATR = 0.0072
  • 初始停損：1.30598 - 1×ATR = 1.29878
  • 持倉 20 天後，價格上漲
  • 調整後停損：1.29878 + (20×0.05×0.0072) = 1.30598
  • 若途中形成新低點 1.29822，則停損 = 1.29822 + (20×0.05×ATR)

二、資金管理（海龜交易法則）

怎麼做：
1. 計算每個標的「美元計價波動性」= 點值 × ATR
2. 建倉單位 = 帳戶資金 × 1% ÷ 美元計價波動性

實例：

  • 帳戶：200,000 美元
  • GBPUSD：ATR = 0.0106，點值 = 10 美元/點 → 波動性 = 1060 美元
  • NZDUSD：ATR = 0.0053，點值 = 10 美元/點 → 波動性 = 530 美元
  • GBPUSD 建倉單位 = 2000 ÷ 1060 = 1.88 手
  • NZDUSD 建倉單位 = 2000 ÷ 530 = 3.77 手
  • 結果：波動性大的標的部位小，波動性小的部位大

三、布林通道 + ATR 策略（TEJ 實測）

策略規則：
1. 買進：收盤價跌破布林下軌
2. 加碼：跌破下軌 + 未跌破停損點（停損點 = 收盤價 - k×ATR）
3. 賣出：突破布林上軌
4. 停損：跌破停損點

回測結果（台積電 2022-2023）：

  • 實驗組（有 ATR 停損）：夏普比率 0.9，最大回撤 2.8%
  • 對照組（無停損）：夏普比率 0.1，最大回撤 7.14%
  • 結論：ATR 停損有效控制風險

ATR 優缺點

優點：

  • 真實反映波動，納入跳空因素
  • 適用多種市場和時間週期
  • 新手也能理解
  • 可用於停損、停利、部位計算

缺點：

  • 低波動市場效果差（訊號不明顯）
  • 基於歷史數據，有滯後性
  • 不提供方向訊號（只告訴你波動多大）
  • 需要搭配其他指標使用

適用時機：

  • 波動較大的市場
  • 趨勢行情
  • 需要動態調整停損時

不適用時機：

  • 極度平穩的盤整市
  • 極短線當沖（波動太小）

心得

之前學了一堆停損方法（K 線停損、均線停損、支阻停損），但都沒講到「停損要設多遠」這個關鍵問題。ATR 解答了這個問題：停損距離 = k × ATR，讓停損隨市場波動自動調整，而不是憑感覺設固定百分比。

最實用的是棘輪止損法，隨著價格往有利方向移動，逐步上調停損，既能保護盈利又不會太早被洗出場。這比死板板的「賺 10% 就停利」靈活多了。

TEJ 的回測數據很有說服力：有 ATR 停損的夏普比率從 0.1 拉到 0.9，最大回撤從 7% 降到 3%。風險調整後的報酬才是真理，不是只看賺多少。

下次要試著把 ATR 停損整合到現有的選股策略裡。布林通道買進、ATR 設停損，這個組合可以研究。

SAR拋物線指標核心概念

什麼是SAR指標？

是什麼：SAR（Stop and Reverse）指標由技術分析大師Wells Wilder發明，是一種趨勢追蹤指標，透過拋物線形狀的點位顯示停損與反轉位置。

核心邏輯：

  • Stop：停損出場
  • Reverse：反手做空（或做多）
  • 指標會自動追蹤價格，提供移動停利參考

發明者背景：Wells Wilder還發明了ATR、RSI、ADX、DMI等知名指標，SAR是其重要發明之一。

SAR指標公式與計算

拋物線公式

買進訊號（多方SAR）：

SAR今 = SAR昨 + AF × (最高價n日內 - SAR昨)
  • SAR起始點：n日內最低價
  • AF起始值：0.02，每創新高加0.02，最大0.2

賣出訊號（空方SAR）：

SAR今 = SAR昨 + AF × (SAR昨 - 最低價n日內)
  • SAR起始點：n日內最高價
  • AF起始值：0.02，每創新低加0.02，最大0.2

參數設定

n值（通常10）：

  • n值越大，SAR距離K線越遠，越不容易觸及
  • 長週期適合波段操作，短週期適合當沖

AF加速因數：

  • 預設0.02，最高0.2
  • AF越大→圖形越貼近價格→越敏感
  • AF越小→圖形越遠離價格→越不敏感
  • 調整為0.05會跟得更緊、反應更快

實例：

  • 股價100元，SAR在95元（距離5%）
  • AF=0.02，行情往上，SAR慢慢跟上到97元
  • AF=0.05，SAR會更快跟上到98元附近

SAR指標解讀方法

基本判讀

多方訊號（紅點）：

  • 紅點出現在K線下方→上升趨勢
  • 紅點接觸到K線→多單出場訊號（翻空）

空方訊號（綠點）：

  • 綠點出現在K線上方→下降趨勢
  • 綠點接觸到K線→空單出場訊號（翻多）

多空翻轉判斷

翻多進場：

  • 價格向上突破綠色SAR線（空方拋物線）
  • 綠點變紅點，SAR轉到K線下方

翻空進場：

  • 價格向下跌破紅色SAR線（多方拋物線）
  • 紅點變綠點，SAR轉到K線上方

SAR指標實戰操作

進場三步驟

步驟一：等待翻轉訊號

  • 行情下跌後，等待價格突破綠色SAR
  • 撈底式進場：在下跌過程中持續觀察，看到翻多才進場

步驟二：進場設停損

  • 多單進場後立刻設停損
  • 停損參考：可設在SAR位置或根據ATR設定（如ATR的20-25%）
  • 實例：最近波動240點，停損設50-60點

步驟三：移動停利

  • 行情走出去，SAR隨價格上升
  • 紅點持續跟進，直到價格拉回打到SAR才出場

避開盤整陷阱

問題：SAR在盤整盤會不斷翻轉，造成連續停損

解決方法一：延遲啟動停利

  • 設定條件：移動停利價格超過進場成本價後才開始移動
  • 例子：股票100元買進，停利線等到價格超過100元才開始跟進

解決方法二：區間K棒高低點

  • 用3根K棒高低點加5%當作出場條件
  • 比單純用SAR更穩定，避免盤整區被掃出場

SAR指標三大使用原則

原則一：日K比小週期準

實務經驗：

  • 日K層級SAR較可靠
  • 1分K、5分K因雜訊多，SAR容易被假突破
  • 小週期應優先看K棒型態，SAR為輔

原則二：適合趨勢盤，不適合盤整盤

特性：

  • 趨勢明確時，SAR能「一波吃到底」
  • 盤整時會反覆翻轉，造成連續虧損
  • 需搭配趨勢判斷工具過濾盤整

原則三：參數可調整快慢

靈敏度調整：

  • AF=0.01：較慢，適合長波段
  • AF=0.02（預設）：標準設定
  • AF=0.05：較快，適合短線或期貨當沖

SAR搭配其他指標

搭配移動平均線MA

方法：

  • 均線多頭排列+SAR翻多→做多
  • 均線空頭排列+SAR翻空→做空
  • 均線糾結盤整時→不依SAR訊號進場

實例：

  • 5日線>10日線>20日線（多頭排列）
  • SAR從空翻多→進場做多
  • 做多後SAR翻空→只做「多單出場」，不做空

搭配MACD

方法：

  • MACD金叉+SAR翻多→買進
  • MACD死叉+SAR翻空→賣出
  • 利用MACD確認趨勢方向，SAR提供進出點

搭配布林通道

方法：

  • 通道寬幅收斂→盤整盤，暫停使用SAR
  • 通道寬幅放大→趨勢啟動，開始使用SAR
  • 用布林通道判斷是否進入趨勢盤

SAR實戰案例

股票波段案例（鴻海）

進場：

  • 行情下跌，SAR綠點接近K線
  • 價格突破綠色SAR，紅點出現在K線下方
  • 成本約在起漲點附近

持有：

  • 紅色SAR點隨價格上升
  • 行情橫盤整理時，停利線保持距離（用成本價當門檻）
  • 不參與盤整區的進出場

出場：

  • 價格拉回打到紅色SAR
  • 或用區間K棒高低點+5%設定

實績：

  • 風險約2萬元
  • 獲利約24萬元
  • 賺賠比10倍

期貨當沖案例（台指期）

設定：

  • 週期：1分K
  • AF參數：調為0.05（跟得更緊）
  • 停損：參考ATR的20-25%，約50-60點
  • 移動停利：6根K棒高低點+50點

操作：

  • 價格突破綠色SAR→進場做多
  • 設定初始停損
  • 行情走出去後，停利線開始跟進
  • 盤整區不進出，抱住部位
  • 最終獲利2萬6千元

SAR指標優缺點

優點

1. 自帶停損停利：不用手動計算，指標自動追蹤
2. 趨勢吃到底：在明顯趨勢中能完整抱住波段
3. 客觀明確：翻紅翻綠一目了然，沒有模糊空間
4. 反手機制：出場後自動反手，不錯過反方向行情

缺點

1. 盤整必虧：橫盤時會反覆翻轉，連續停損
2. 落後指標：必須等價格突破才翻轉，無法買在最低點
3. 參數敏感：不同參數效果差異大，需調整
4. 小週期失效：短線太多雜訊，SAR容易誤判

心得

SAR指標的核心不是參數或公式，而是「停損+反手」的交易邏輯。這跟之前學過的「移動停利」概念一樣，但SAR更極端——永遠持有部位，多空不斷翻轉。

這讓我想到一個關鍵問題：如果永遠持有部位，那什麼時候該停手？

答案很清楚：SAR不適合盤整盤。文章中提到「雙八盤」會讓SAR不斷翻轉，這時候應該用其他條件過濾，而不是硬跟SAR訊號進出。結合之前學的布林通道、均線排列，就能判斷現在是不是趨勢盤。

另外，WINSMART文章提到一個聰明做法：「等移動停利價超越成本價才開始移動」。這招可以直接避開盤整區被洗出場的問題——盤整時你還在成本附近震盪，停利線根本不動。

最後，SAR的「反手」概念很重要。不是「停損出場就算了」，而是「停損出場+反手做空（或做多）」。這樣就算第一次看錯方向，後面行情反轉時你已經在對的方向了。這種「永遠保持部位」的思維，跟「選擇性持有」很不一樣，需要更大的心理資本來承受連續翻轉的虧損。

第 5 回合學習：技術分析：支撐壓力判斷

學習時間：05:43

文章：

  • K棒如何看出支撐壓力？看懂它讓你先贏一半！
  • 「支撐線」與「壓力線」是什麼？檢視方法介紹

核心知識

一、支撐壓力基本定義

是什麼：

  • 支撐（Support）：價格跌不破的價位，像地板一樣托住價格
  • 壓力（Resistance）：價格漲不破的價位，像天花板一樣壓住價格

形成原因：

  • 有一群人或主力的成本在這個價位
  • 不希望價格攻破，否則會賠錢或白費工
  • 也是市場心理關卡：「一百大關」、「兩百大關」

供需關係：

  • 支撐：買方 > 賣方，價格容易上升
  • 壓力：賣方 > 買方，價格容易下跌

二、如何找出支撐壓力

方法一：K棒影線與實體判斷

怎麼做：
1. 影線法：觀察K棒下影線，找出價格跌不破的價位
2. 實體法：觀察K棒實體，找出價格漲不破的價位
3. 綜合法：兩個都看，找出上下都有阻擋的區間

實例：

  • 情境：某股票在12.65元附近，K棒都沒有低於這個價格
  • 判斷：12.65元是支撐
  • 情境：某股票27元附近，K棒都沒超越這個價格
  • 判斷：27元是壓力

方法二：兩根K棒連線法

怎麼做：
1. 找到至少2波的低點（畫支撐線）
2. 找到至少2波的高點（畫壓力線）
3. 將這些點連線，向右延伸

有力程度判斷：

  • 越多K棒連成的支撐壓力越有力
  • 觸及次數越多越可靠
  • 市場共識越強

適用時機：快速判斷、波段操作

方法三：跳空缺口法

怎麼做：
1. 找出K棒之間的空白區間
2. 跳空下方的缺口可能形成支撐
3. 跳空上方的缺口可能形成壓力

實例：

  • 情境：兩根K棒之間有明顯空白

一、MACD 是什麼？

定義：MACD（Moving Average Convergence / Divergence）全名為「指數平滑異同移動平均線」，由美國技術分析大師 Gerald Appel 於 1979 年提出。

核心原理：利用兩條速度不同的指數移動平均線（EMA）計算差離值，判斷市場趨勢變化與動能強弱。

預設參數：(12, 26, 9)

  • 快速 EMA：12 日
  • 慢速 EMA：26 日
  • 訊號線：9 日 EMA

二、MACD 指標組成

MACD 線（快線 / DIF）

是什麼：短期 EMA 與長期 EMA 的差值，反映短期價格變化的敏感度。

公式：

DIF = EMA(12) - EMA(26)

怎麼解讀：

  • DIF > 0：短期均線高於長期均線，上漲趨勢，多方主導
  • DIF < 0：短期均線低於長期均線，下跌趨勢，空方主導

訊號線（慢線 / DEA）

是什麼：DIF 的 9 日 EMA，用於平滑快線波動，捕捉較穩定的長期趨勢。

公式：

DEA = EMA(DIF, 9)

怎麼解讀：

  • 反應速度較慢，作為訊號確認線
  • DIF 穿越 DEA 時產生買賣訊號

MACD 柱狀圖

是什麼：DIF 與 DEA 的差值，直觀反映市場動能強弱。

公式：

MACD 柱狀圖 = DIF - DEA
或
MACD 柱狀圖 = 2 ×（DIF - DEA）  // 部分軟體使用

怎麼解讀：

  • 柱狀圖 > 0：DIF 高於 DEA，上漲動能增強
  • 柱狀圖 < 0：DIF 低於 DEA，下跌動能增強
  • 柱狀圖放大：動能增強，趨勢加速
  • 柱狀圖縮小：動能減弱，趨勢可能轉折

三、黃金交叉與死亡交叉

黃金交叉（買進訊號）

是什麼：DIF 由下往上穿越 DEA，或柱狀圖由負轉正。

怎麼做：
1. 確認 DIF 向上突破 DEA
2. 觀察柱狀圖是否同步由負轉正
3. 搭配成交量和趨勢線確認
4. 若交叉點在 0 軸以上，後續漲勢更明顯

實例：

  • 情境：台股某檔股票 DIF = 0.5，DEA = 0.3
  • 操作：DIF 穿越 DEA 形成 黃金交叉，柱狀圖翻正，股價 50 元
  • 進場：50 元買進，停損設 48 元（-4%）
  • 目標：前波段高點 58 元（+16%）

適用時機：

  • 多頭市場初期或中段
  • 趨勢明確時
  • 搭配支撐壓力位確認

死亡交叉（賣出訊號）

是什麼：DIF 由上往下穿越 DEA，或柱狀圖由正轉負。

怎麼做：
1. 確認 DIF 向下跌破 DEA
2. 觀察柱狀圖是否同步由正轉負
3. 若交叉點在 0 軸以下，後續跌幅更明顯
4. 考慮減碼或出場

實例：

  • 情境：DIF 從 0.8 跌至 0.2，DEA 維持在 0.4，形成死亡交叉
  • 柱狀圖：從 +0.4 變成 -0.2（翻負）
  • 操作：股價 60 元時減碼五成，跌破 55 元全出

適用時機：

  • 空頭市場初期
  • 趨勢明確向下時
  • 搭配壓力位確認

心得

MACD 是技術分析的基礎指標，但很多人只用黃金交叉和死亡交叉，忽略了更重要的「動能」判斷。這次學習讓我有幾個體會：

1. 柱狀圖比交叉更重要：黃金交叉發生時，如果柱狀圖還在縮小，這個訊號可能很弱。真正強勢的多頭，柱狀圖應該要放大。

2. 背離是反轉前的預警：頂背離和底背離比交叉更早發出訊號，但要搭配其他指標確認，因為單獨背離也可能失敗。

3. 趨勢優先，MACD 其次：統一期貨文章提到一個重要觀念——MACD 是判斷「趨勢強度」，不是判斷「趨勢改變」。先用均線確定多空方向，再用 MACD 找進場點，這個順序不能顛倒。

4. 0 軸是多空分水嶺：柱狀圖在 0 軸之上和之下，代表的市場結構完全不同。黃金交叉在 0 軸以上，通常漲勢更明顯；死亡交叉在 0 軸以下，跌幅更大。

5. 盤整理想不要用：MACD 在趨勢行情好用，但在盤整時會不斷交叉，進出場都很尷尬。這時改用 RSI 或 KD 會更好。

這次學習補足了技術分析拼圖的重要一塊。MACD 雖然是老指標，但動能判斷的核心概念到現在還是很實用。重點是用對地方——趨勢明確時用 MACD 找進場，盤整時改用震盪指標，這樣才能發揮各個指標的優勢。
