一、現金流量表間接法 — 從淨利回推真實現金

是什麼

間接法是全球超過95%公司編製現金流量表的方法。它不是從零開始記錄每筆現金進出，而是從損益表的「稅後淨利」出發，逐步調整回推到「營業活動現金流」。

核心邏輯：淨利是「應計基礎」（帳面上賺多少），營業現金流是「現金基礎」（口袋裡真正進多少），兩者的差額就是「非現金項目」和「營運資金變動」。

間接法調整步驟

怎麼做：

1. 起點：稅後淨利（來自損益表）
2. 加回非現金支出：折舊、攤銷（帳面上扣掉了但沒有真正花錢）
3. 加回非營業損失：處分資產損失（已經在投資現金流處理）
4. 扣除非營業收益：處分資產利益（不屬於營業活動）
5. 調整營運資金變動：

  • 應收帳款增加 → 扣除（錢還沒收到）

公式記憶法：
> 營業現金流 = 淨利 + 折舊攤銷 ± 應收帳款變動 ± 存貨變動 ± 應付帳款變動 ± 其他調整

為什麼間接法比直接法重要

比較直接法間接法
列出項目逐筆現金收支從淨利調整
優點直觀，看得到每筆現金流揭示淨利和現金流的差異
缺點編製成本高不直觀，需要理解調整邏輯
全球使用率<5%>95%
投資人價值知道收了多少錢**知道淨利和現金流差在哪裡**

間接法的核心價值在於：它清楚告訴你，帳面利潤和真實現金之間的落差是什麼原因造成的。應收帳款暴增？存貨堆積？還是折舊佔比太高？這些資訊直接法看不到，間接法一目了然。

實例：間接法拆解

某公司損益表顯示淨利 10 億，但營業現金流只有 6 億。差在哪？

調整項目金額說明
稅後淨利+10億起點
折舊與攤銷+3億非現金支出，加回
處分資產利益-1億非營業收益，扣除
應收帳款增加-4億錢還沒收到，扣除
存貨增加-3億錢壓在庫存，扣除
應付帳款增加+1億錢還沒付，加回
**營業現金流****6億**最終結果

關鍵發現：淨利10億但現金只進來6億，差距4億。其中應收帳款增加4億是最大元凶——公司帳面上賺了10億，但有4億還掛在帳上沒收到現金。這就是「盈餘品質」的問題。

二、營業現金流對淨利比率 — 盈餘品質快篩

是什麼

營業現金流對淨利比率 = 營業現金流 ÷ 稅後淨利

這個比率告訴你：公司每賺1元淨利，實際收到多少現金。

怎麼做：
1. 從現金流量表找到「營業活動現金流量」
2. 從損益表找到「稅後淨利」
3. 營業現金流 ÷ 稅後淨利 = 比率
4. 觀察至少3-5年的趨勢

判讀標準

比率區間意義判斷
> 1.2現金比利潤多✅ 盈餘品質極佳，折舊大或應付帳款管理好
1.0~1.2現金約等於利潤✅ 正常健康
0.7~1.0現金略少於利潤⚠️ 應收帳款或存貨可能增加
0.5~0.7現金明顯少於利潤🚨 盈餘品質差，可能美化帳面
< 0.5現金遠少於利潤🚨🚨 嚴重警訊，利潤可能有大量水分

為什麼比率可以大於1

因為折舊和攤銷是非現金支出，在計算淨利時被扣除，但在現金流量表上被加回。所以：

  • 重資本產業（半導體、面板）：折舊大 → 比率可能 > 1.5
  • 輕資產產業（軟體、諮詢）：折舊小 → 比率通常接近 1.0

怎麼做（調整版）：

  • 重資本產業：比率 > 1.0 是正常的，不要誤以為現金太多
  • 輕資產產業：比率 < 0.8 就要警覺

實例判讀

  • 台積電(2330)：營業現金流 8,000億 ÷ 淨利 5,000億 = 1.6 → ✅ 折舊大是主因，盈餘品質極佳
  • 某面板廠：營業現金流 200億 ÷ 淨利 500億 = 0.4 → 🚨 帳面賺500億但現金只進來200億，300億去哪了？多半是應收帳款暴增和存貨堆積
  • 某軟體公司：營業現金流 15億 ÷ 淨利 12億 = 1.25 → ✅ 輕資產且現金回收能力好

適用時機：每季財報公布後，第一時間算這個比率。如果連續2季低於0.7，立即深入調查應收帳款和存貨。

三、現金流量八大型態 — CFO/CFI/CFF正負號組合判讀

是什麼

把營業現金流(CFO)、投資現金流(CFI)、融資現金流(CFF)的正負號組合起來，可以判斷公司的經營階段和體質。三個數字各有正負，理論上有8種組合（2³），對應8種不同的企業體質。

八大型態詳解

型態一：CFO(+) CFI(-) CFF(-) — 🟢 成熟穩健型

  • 營業現金流正：本業賺錢
  • 投資現金流負：持續投資擴產
  • 融資現金流負：還債、發股利、買回庫藏股

判讀：最健康的型態。本業賺的錢足以支應投資，還有餘額回饋股東和還債。台積電、中華電就是典型。

投資啟示：長期持有的好標的，FCF穩定，適合存股。

型態二：CFO(+) CFI(-) CFF(+) — 🟡 成長擴張型

  • 營業現金流正：本業開始賺錢
  • 投資現金流負：大量資本支出
  • 融資現金流正：本業賺的不夠投資，需要借錢或增資

判讀：公司正處於快速成長期，投資需求超過本業現金流，需要外部資金。特斯拉2018-2020年就是這個型態。

投資啟示：要看投資是否能帶來未來回報。如果投資報酬率低於資金成本，長期會摧毀股東價值。

型態三：CFO(+) CFI(+) CFF(-) — 🟡 收縮防禦型

  • 營業現金流正：本業還在賺錢
  • 投資現金流正：賣資產、收回投資
  • 融資現金流負：還債、發股利

判讀：公司在收縮規模，賣掉資產來還債和發股利。可能是成熟期企業不再擴張，也可能是策略轉型。

投資啟示：如果本業穩定只是不再擴張 → 穩定收息股。如果本業正在衰退靠賣資產撐 → ⚠️ 警覺。

型態四：CFO(+) CFI(+) CFF(+) — 🟠 異常型

  • 營業現金流正：本業賺錢
  • 投資現金流正：賣資產
  • 融資現金流正：借錢/增資

判讀：本業賺錢還要賣資產又要借錢？除非有大額併購或特殊投資計畫，否則這種型態很異常。

投資啟示：🚨 必須深入查原因。可能的情況：大額併購、買回庫藏股、支付特別股利。

型態五：CFO(-) CFI(-) CFF(+) — 🔴 燒錢擴張型

  • 營業現金流負：本業燒錢
  • 投資現金流負：還在投資
  • 融資現金流正：完全靠外部資金

判讀：新創公司的典型型態。本業還沒賺錢就大舉投資，完全靠融資活著。特斯拉2015-2017年就是這個型態。

投資啟示：高風險高回報。如果燒錢換市佔率成功（如Amazon、特斯拉），回報驚人；如果失敗，血本無歸。

型態六：CFO(-) CFI(+) CFF(+) — 🔴 困境求生型

  • 營業現金流負：本業燒錢
  • 投資現金流正：賣資產求現
  • 融資現金流正：還要借錢

判讀：最危險的型態之一。本業不賺錢，賣資產撐著，還要借錢度日。

投資啟示：🚨 除非有明確轉機，否則避開。

型態七：CFO(-) CFI(-) CFF(-) — ⚫ 全面流出型

  • 營業現金流負：本業燒錢
  • 投資現金流負：還在投資
  • 融資現金流負：還在還債

判讀：三個現金流都在流出，現金水位持續下降。如果沒有現金儲備撐著，很快會面臨資金危機。

投資啟示：🚨🚨 極度危險。除非是暫時性的大量投資期，否則不碰。

型態八：CFO(-) CFI(+) CFF(-) — ⚫ 衰退型

  • 營業現金流負：本業燒錢
  • 投資現金流正：賣資產
  • 融資現金流負：還債

判讀：本業不賺錢，靠賣資產和還債度日。公司正在縮減規模，但如果本業無法轉正，最終會把資產賣光。

投資啟示：🚨 除非你確信本業即將轉正，否則避開。

八大型態速查表

型態CFOCFICFF企業階段投資判斷
一+--成熟穩健🟢 長期持有
二+-+成長擴張🟡 看投資報酬
三++-收縮防禦🟡 看本業是否衰退
四+++異常型🟠 查原因
五--+燒錢擴張🔴 高風險高回報
六-++困境求生🔴 避開
七---全面流出⚫ 極度危險
八-+-衰退型⚫ 避開

實戰判讀SOP

怎麼做：
1. 打開現金流量表，找出CFO、CFI、CFF三個數字
2. 判斷正負號，對照八大型態表
3. 觀察至少3-5年的型態變化趨勢
4. 如果型態發生變化（例如從型態二變成型態一），確認原因

型態變化的意義：

  • 型態二→型態一：從成長期進入成熟期，本業現金流足以支應投資 ✅
  • 型態一→型態三：停止投資開始賣資產，可能進入衰退 ⚠️
  • 型態五→型態二：本業開始賺錢了，轉虧為盈 ✅
  • 型態一→型態五：本業突然虧損，嚴重警訊 🚨

適用時機：每次看財報時，第一時間判斷現金流型態，再看細項。

四、現金流量表與三大報表的交叉驗證

為什麼不能只看現金流量表

現金流量表是「照妖鏡」，但不是萬能的。它必須和損益表、資產負債表交叉驗證，才能看出完整的真相。

交叉驗證五步法

第一步：營業現金流 vs 淨利

  • 營業現金流/淨利比率 > 1.0 → 盈餘品質佳
  • 營業現金流/淨利比率 < 0.7 → 盈餘品質差，可能有美化帳面

第二步：應收帳款變動 vs 營收成長

  • 應收帳款成長率 > 營收成長率 → 收款速度變慢，危險
  • 應收帳款成長率 < 營收成長率 → 收款效率提升，健康

第三步：存貨變動 vs 營收成長

  • 存貨成長率 > 營收成長率 → 銷售不暢，庫存堆積
  • 存貨成長率 < 營收成長率 → 銷售暢旺，庫存管理好

第四步：投資現金流 vs 資本支出趨勢

  • 資本支出持續增加 + 營收成長 → 擴張期，正面
  • 資本支出持續增加 + 營收持平或下降 → 投資效率差，警覺
  • 資本支出大幅減少 + 賣資產 → 收縮或衰退

第五步：融資現金流 vs 債務趨勢

  • 持續借錢 + 本業虧損 → 🚨 借錢度日，隨時可能周轉不靈
  • 持續還債 + 本業賺錢 → ✅ 財務體質改善中
  • 大額增資 + 本業虧損 → ⚠️ 稀釋股東權益

企業生命週期現金流特徵

階段CFOCFICFF典型公司
初創期--+新創、生技研發
快速成長期+-+擴張中的科技公司
成熟穩健期+--台積電、中華電
衰退期-++/-傳統產業轉型

心得

現金流量表間接法是我之前一直覺得「很複雜看不懂」的部分，但今天拆解後才發現，它其實是在回答一個最根本的問題：帳面上賺的錢，真的進口袋了嗎？

間接法的回推邏輯非常精妙：從淨利出發，先把折舊攤銷加回（因為沒有真正花錢），再把應收帳款增加扣掉（因為錢還沒收到），把存貨增加扣掉（因為錢壓在庫存裡），把應付帳款增加加回（因為錢還沒付出去）。每一步調整都在告訴你：淨利和現金之間的差距，到底是什麼原因造成的。

營業現金流對淨利比率是個超實用的快篩工具。以後看財報，第一件事就是算這個比率——如果連續低於0.7，不用看其他指標，先去查應收帳款和存貨。台積電比率1.6看起來嚇人，但那是因為折舊大，不是問題；面板廠比率0.4看起來「只有」0.4的差距，但背後是3億的應收帳款收不回來，這才是致命的。

八大型態的框架太清晰了。型態一是「理想情人」——本業賺錢、持續投資、還能回饋股東；型態五是「潛力股」——本業還在燒錢但瘋狂投資，靠融資活著，賭對了回報驚人，賭錯了歸零；型態六和七是「地雷」——本業不賺錢還在賣資產借錢，遲早炸掉。但最實用的不是單一型態判斷，而是型態的變化趨勢。一家公司從型態五進化到型態二再到型態一，就是經典的成長股路徑；反過來從型態一退化到型態三再退化到型態八，就是經典的衰退股路徑。

最後一個深刻體會：現金流量表是三大報表裡最難造假的。損益表可以靠提前認列收入、延後認列費用來美化；資產負債表可以靠關係人交易來灌水；但現金流量表記錄的是真金白銀的進出，要造假得真的把錢搬來搬去。所以巴菲特說「利潤只是概念，現金才是事實」，這句話放在間接法的拆解裡，每一個調整項目都在證明這句話。

一、併購套利（Merger Arbitrage）— 用概率思維做投資

是什麼

併購套利，又稱風險套利（Risk Arbitrage），是指當市場上有公司宣布併購時，套利者買入被收購公司的股票，在併購完成時賺取「殘餘價差」。

核心邏輯：併購消息宣布後，被收購公司股價會迅速上漲接近收購價，但因為併購尚未完成（股東投票、監管審查、融資條件等不確定性），股價不會完全等於收購價，這段差價就是套利空間。

兩種併購類型的操作方式

類型一：現金併購（Cash Deal）

  • 操作：做多目標公司股票
  • 預期：併購完成時，股價收斂到現金收購價
  • 例：A公司宣布以每股100元現金收購B公司，B公司股價漲到95元，殘餘價差5元

類型二：股票併購（Stock Deal）

  • 操作：做多目標公司股票 + 放空收購方股票
  • 預期：兩者股價因併購完成收斂至約定的換股比例
  • 例：A公司以1股換1.5股B公司，買B賣A，鎖定換股比例的價差

併購套利的風險報酬計算

怎麼做：

1. 確認收購價與當前股價的殘餘價差
2. 估算併購完成的機率
3. 估算併購失敗時的股價回落幅度
4. 計算期望報酬

實例：

  • 收購方宣布以30%溢價現金收購目標公司
  • 消息公布後目標公司股價上漲26%，殘餘價差4%
  • 假設併購完成機率95%，失敗機率5%
  • 失敗時股價可能下跌20%

期望報酬 = 95% × 4% + 5% × (-20%) = 3.8% - 1% = 2.8%

年化報酬率取決於併購完成時間：

  • 4個月完成 → 年化約 8.4%
  • 8個月完成 → 年化約 4.2%
  • 1年完成 → 年化約 2.8%

併購套利的風險特徵

關鍵特徵：高機率小賺、低機率大虧（負偏態分佈）

  • 大部分時間：併購成功，小賺殘餘價差
  • 少數時候：併購失敗，股價暴跌20-30%

這種分佈跟賣選擇權很像——穩定收小錢，偶爾大虧一次。因此併購套利基金會同時佈局多筆交易，分散風險。

併購套利的系統性風險

Mitchell & Pulvino（2001）研究發現：

  • 市場平穩或上漲時：併購套利報酬與大盤相關性接近0（低相關）
  • 市場大幅下跌時：併購套利報酬與大盤正相關且同步下滑

HFRI併購套利指數（1989-2013）：

  • 平均年化報酬約8.2%
  • 資訊比率約2.0
  • 2008年金融海嘯最大回撤約8.3%

結論：併購套利在正常市況下是不錯的分散工具，但在極端市況下會跟大盤一起跌。

二、巴菲特的併購套利四問 — 實戰分析框架

巴菲特在1988-1989年股東信中分享了他的套利原則。他強調：只參與已公開宣布的交易，不根據謠言做交易。

四個核心問題

問題一：宣布事件實際發生的可能性有多大？

影響併購成功率的关键因素：

  • 友善併購 vs 惡意併購：友善併購成功率高（管理層支持），惡意併購失敗率高（毒丸計畫、代理權爭奪）
  • 收購溢價水平：溢價越高，目標公司股東越願意接受
  • 收購方與目標方規模：大收小吃成功率高，小收大風險高
  • 監管審查：反壟斷法規、國家安全考量
  • 融資條件：槓桿收購失敗風險更大
  • 分手費條款：分手費越高，雙方完成併購的動機越強

問題二：你的資金會被佔用多久？

  • 併購完成時間從幾個月到幾年不等
  • 資金被佔用越久，年化報酬率越低
  • 用年化報酬率評估，不是看絕對價差

問題三：有多大可能出現更好的結果？

  • 競價者出現（更高收購價）= 更好的結果
  • 評估是否有其他潛在買家
  • 競價者出現時，套利報酬會大幅增加

問題四：如果併購失敗，會怎樣？

  • 通常股價會回落到併購宣布前的水準
  • 可以用選擇權限制下行風險
  • 例：買進遠期買權，最大損失就是權利金

三、特殊事件投資法（Special Situation Investing）

是什麼

葛林布雷（Joel Greenblatt）在《你也可以成為股市天才》中介紹的投資法，核心邏輯是：在公司發生特殊事件時，市場往往無法正確定價，這就是超額報酬的來源。

特殊事件包括：

  • 併購收購（M&A）
  • 分拆（Spin-off）
  • 破產重整（Bankruptcy/Reorganization）
  • 清算（Liquidation）
  • 股份回購（Share Buyback）
  • 資產出售（Asset Sale）

核心原則：下檔風險優先

特殊事件投資法不是追求最大報酬，而是追求下檔風險有限的機會：

1. 投資只要不虧錢，無論大賺小賺結果都是好的
2. 獲利會自己照顧自己
3. 時間是好公司的朋友，壞公司的敵人

怎麼確認下檔有限？

  • 管理團隊有強烈動機持續經營（持股比例高、認股權證多）
  • 資產價值遠高於股價（資產大於市值）
  • 有明確的催化劑（事件即將發生，股價會回歸價值）

六種特殊事件投資類型

類型一：公司分拆（Spin-off）

母公司將子公司獨立上市，分配股票給母公司股東。

為什麼分拆是好機會：

  • 分拆後新公司通常規模較小，華爾街分析師不跟蹤
  • 機構投資者被迫賣出（規模不符投資限制、指數成分調整）
  • 大量賣壓造成股價低估
  • 分拆公司管理層有強烈動機做好（薪酬與績效掛鉤）

分拆投資原則：
1. 關注被分拆的公司，而非母公司
2. 等待分拆後1-3個月的賣壓期
3. 確認管理層持股比例高
4. 被分拆公司越不被市場關注，機會越大

類型二：併購套利

前面已詳細介紹。

類型三：破產重整

公司進入破產程序後，債權人可能獲得重整後公司的新股。

為什麼破產重整是好機會：

  • 市場對破產公司避之不及，股價可能嚴重低估
  • 重整完成後，新公司往往負債大幅減少、營運改善
  • 但風險極高，需要深入分析資產負債表

類型四：清算

公司決定清算資產，將剩餘價值分配給股東。

清算投資原則：
1. 計算清算價值 = 流動資產 - 總負債（NCAV法）
2. 如果股價遠低於清算價值，就有安全邊際
3. 關注清算時間表，資金被佔用越久報酬率越低

類型五：股份回購

公司回購自家股票，減少流通股數，提升EPS。

回購投資原則：

  • 回購原因比回購本身重要：股價低估時回購是好事，股價高估時回購是浪費
  • 管理層是否同步買進？如果是，信心更強
  • 回購後EPS提升多少？計算稀釋效果

類型六：資產出售

公司出售非核心資產或虧損部門。

資產出售投資原則：
1. 出售後的公司價值 > 出售前（去除拖累的部門後更專注）
2. 出售所得現金的用途（還債、配息、再投資）
3. 出售價格是否合理

催化劑的重要性

特殊事件投資法強調催化劑（Catalyst）：

  • 沒有催化劑：股價可能長期低估，資金效率差
  • 有催化劑：股價會因事件發生而快速回歸價值

常見催化劑：

  • 併購完成日
  • 分拆上市日
  • 重整計畫批准日
  • 清算分配日
  • 法人說明會
  • 財報公布

特殊事件投資的操作原則

原則一：好公司長期投資，一般公司短線操作

  • 特殊事件造成的安全邊際讓你有買進機會
  • 如果公司本身品質好 → 低估買入，高估賣出（長期）
  • 如果公司品質一般 → 低估買入，合理價賣出（短線）

原則二：尋找催化劑

  • 買進前就要確認催化劑是什麼
  • 沒有催化劑的低估股只是價值陷阱

原則三：確認管理層動機

  • 管理層持股比例越高，與股東利益越一致
  • 分拆後新公司管理層是否有認股權證？
  • 重整後新管理層是否有激勵機制？

原則四：使用長期選擇權（LEAPS）放大報酬

葛林布雷的一個獨門技巧：

  • 如果一家公司股價嚴重低估，它的長期買權（LEAPS）也被嚴重低估
  • 例：內在價值60元的股票，現價40元，潛在報酬50%
  • 但如果買履約價60元的長期買權，一旦股價回到60元以上，買權報酬是以倍數計算
  • 風險：買權可能到期作廢，損失全部權利金

台股的特殊事件投資機會

台股併購案相對少見，但特殊事件投資仍有機會：

  • 減資：現金減資（利多）、彌補虧損減資（利空）
  • 分拆上市：子公司獨立上市，母公司股東可能分配新股
  • 資產處分：出售土地、大樓等業外收入
  • 董監改選：改選行情前的布局
  • 庫藏股執行：公司宣布買回庫藏股期間

心得

併購套利和特殊事件投資法，跟之前學的所有基本面分析都不一樣——這不是在分析公司「值多少錢」，而是在分析「事件發生的機率有多大」。

巴菲特四問是這整套方法論的精華：(1) 事件完成的機率、(2) 資金佔用時間、(3) 有沒有更好的結果、(4) 失敗會怎樣。這四個問題本質上就是期望值計算的框架，把投資決策從「感覺」變成「數學」。

併購套利的負偏態特徵很值得警惕——大部分時間小賺，偶爾大虧。這跟賣選擇權的風險結構一模一樣，所以分散是必須的，不能把全部資金壓在單一併購案上。HFRI的數據告訴我們年化8.2%的報酬加上2.0的資訊比率看起來不錯，但2008年8.3%的最大回撤提醒我們：系統性風險來的時候，併購套利也跑不掉。

葛林布雷的特殊事件投資法最打動我的是「催化劑」的概念。之前學價值投資，總覺得找到低估股就夠了，但低估不代表會漲——沒有催化劑的低估只是價值陷阱。分拆是最經典的催化劑：機構被迫賣出、分析師不跟蹤、管理層有動機做好，三重力量把股價壓低，然後又三重力量讓它回歸價值。

LEAPS搭配特殊事件的用法很巧妙，但也極度危險。買長期買權的風險是100%損失權利金，這跟併購套利的風險結構恰好相反——併購套利是高機率小賺低機率大虧，LEAPS是低機率大賺高機率小虧。兩者組合使用才能平衡風險。

台股的併購案不多，但減資、分拆、董監改選、資產處分這些特殊事件每年都有。關鍵是建立一個事件日曆，追蹤哪些公司即將發生什麼事，然後用巴菲特四問評估期望值。這是一個需要耐心和紀律的策略，不適合天天操作，但一年找到2-3個好機會就夠了。
