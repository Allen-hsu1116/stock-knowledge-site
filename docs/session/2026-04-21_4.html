一、股價淨值比（PBR）基礎 — 你花的錢，買到多少帳面資產？

是什麼

股價淨值比（Price-Book Ratio，簡稱PB、PBR）= 每股股價 ÷ 每股淨值

告訴你：你花多少錢，買到公司帳面上價值1元的資產。

怎麼做：
1. 從財報找到每股淨值（股東權益 ÷ 在外流通股數）
2. 用當前股價除以每股淨值
3. 判讀結果，與同業、歷史區間比較

實例：

  • A公司股價12元，每股淨值10元 → PBR = 12÷10 = 1.2倍
  • B公司股價8元，每股淨值10元 → PBR = 8÷10 = 0.8倍（股價低於淨值）

每股淨值是什麼？

每股淨值 = （總資產 - 總負債 - 特別股價值）÷ 在外流通股數

就是公司清算後，理論上每股可以拿回的錢。但注意，這是會計價值，不等於真實價值。

每股淨值 vs 真實價值的落差：

  • 土地：帳面成本可能遠低於市價（低估）
  • 設備：帳面價值可能遠高於二手市場拍賣價（高估）
  • 品牌專利：大多沒列在資產負債表上（嚴重低估）
  • 應收帳款：不一定收得回來（高估）

PBR區間判讀

PBR區間意義判斷
PBR < 0.5嚴重折價需要深入調查為何如此便宜，可能是價值陷阱
PBR 0.5~0.8低估如果基本面沒問題，可能有撿便宜的機會
PBR 0.8~1.0接近淨值合理偏低，安全邊際較大
PBR 1.0~1.5合理溢價市場認為公司有成長性
PBR 1.5~3.0高溢價市場期待高成長，需驗證
PBR > 3.0極度溢價除非是高速成長股，否則風險極高

適用時機：PBR<1不代表便宜，PBR>1不代表貴，必須搭配ROE和產業特性判斷。

二、PBR vs 本益比PE vs ROE — 三個指標各有所長

三指標定位比較

指標計算衡量什麼適用產業不適用產業
本益比PE股價÷EPS花多少年回本獲利穩定的成長股虧損股、循環股
股價淨值比PBR股價÷每股淨值花多少倍價格買帳面資產循環股、金融股、資產股輕資產科技股、品牌股
ROE淨利÷股東權益資本運用效率所有產業淨值為負的公司

為什麼循環股要用PBR不用PE？

景氣循環股（鋼鐵、航運、水泥、金融）的EPS波動劇烈：

  • 景氣好時EPS暴漲 → PE看起來很低 → 其實是高點，不該買
  • 景氣差時EPS暴跌甚至虧損 → PE很高或無法計算 → 其實是低點，應該買

PBR以淨值為基準，淨值相對穩定，不會因為單季獲利暴起暴落而大幅波動，所以更適合循環股。

實例：

  • 長榮(2603)：2021年EPS暴漲，PE極低（看起來便宜），但其實是航運景氣高峰
  • 如果用PBR看，長榮在景氣谷時PBR可能跌到0.5倍以下，那才是真正的低點

PBR與ROE的關係 — 杜邦公式的延伸

ROE = 淨利率 × 資產週轉率 × 權益乘數

而 PBR 和 ROE 之間有一個重要的邏輯關係：

> ROE高的公司，PBR通常也高（市場願意給溢價）
> ROE低的公司，PBR通常也低（市場不願意給溢價）

這告訴我們：PBR低不代表便宜，如果ROE也低，那是「便宜有便宜的道理」。真正值得投資的是PBR低但ROE高的公司——市場尚未發現其價值。

PB-ROE選股框架：
1. 篩選ROE > 15%的公司
2. 在ROE高的公司中，找PBR相對低的
3. 交叉驗證：PBR低是因為ROE暫時下降（被低估），還是ROE永久性下降（價值陷阱）

三、PBR的四種保守計算法 — 清算價值的精確估算

市場先生提出了四種越來越保守的PBR計算法，從最寬鬆到最嚴格：

### 方法一：傳統PBR > PBR = 股價 ÷ 每股淨值（總資產 - 總負債 - 特別股）

最常見但最不精確，因為總資產包含了難以變現的無形資產和商譽。

### 方法二：有形資產淨值PBR > 有形資產淨值 = 總資產 - 總負債 - 特別股 - 無形資產 > PBR = 股價 ÷ 每股有形資產淨值

扣除商譽、專利、品牌等無形資產，只看「摸得到」的資產。對科技股、品牌股影響很大（無形資產占比高）。

### 方法三：流動資產淨值PBR > 流動資產淨值 = 流動資產 - 總負債 - 特別股 > PBR = 股價 ÷ 每股流動資產淨值

只看現金、短期投資、存貨、應收帳款等容易變現的資產，扣掉全部負債。更保守，但更接近真實清算價值。

### 方法四：淨現金PBR（最保守） > 每股淨現金 = （現金和短期投資 - 總負債 - 特別股）÷ 在外流通股數 > PBR = 股價 ÷ 每股淨現金

完全不看存貨和應收帳款，只算口袋裡的現金扣掉負債。這是「公司明天關門，股東能拿回多少」的極端估算。

實例：

  • 某公司：股價50元，每股淨值60元，每股有形資產淨值45元，每股流動資產淨值30元，每股淨現金20元
  • 傳統PBR = 50÷60 = 0.83（看起來便宜）
  • 有形PBR = 50÷45 = 1.11（其實沒那麼便宜）
  • 流動PBR = 50÷30 = 1.67（開始偏貴）
  • 淨現金PBR = 50÷20 = 2.5（一點都不便宜！）

適用時機：

  • 一般篩選：用傳統PBR即可
  • 深入分析價值陷阱：用有形PBR或流動PBR
  • 極端保守估算清算價值：用淨現金PBR
  • 控股公司、資產股：用流動PBR或淨現金PBR最有意義

四、PBR河流圖 — 一眼看出股價在歷史區間的哪個位置

是什麼

PBR河流圖是將歷年股價和PBR對應的倍數區間視覺化，用不同顏色標示出PBR區間，一眼判斷股價是相對高點還是低點。

怎麼看：
1. 河流圖下方（紫色/藍色）= PBR低 = 股價相對便宜
2. 河流圖上方（橘色/紅色）= PBR高 = 股價相對昂貴
3. 河流圖中間（綠色/黃色）= PBR合理區間

三種河流圖型態判讀

型態一：上升趨勢（成長股）

  • 河流圖逐漸往上走，PBR區間整體上移
  • 代表市場持續給予更高溢價，公司成長性被認可
  • 台積電就是典型：PBR從1-2倍區間一路上升到3-5倍以上
  • 注意：PBR持續上升不代表不能買，但要確認ROE是否支撐

型態二：上下震盪（景氣循環股）

  • 河流圖在固定區間來回波動
  • 代表PBR有規律的高低循環
  • 台泥、中鋼是典型：PBR在0.8-1.8倍之間震盪
  • 操作策略：PBR跌到歷史低檔區間買進，漲到歷史高檔區間賣出

型態三：穩定平坦（成熟穩定型）

  • 河流圖幾乎水平，PBR長期在固定區間
  • 代表公司獲利穩定，市場給予的溢價變動不大
  • 中華電是典型：PBR長期在2.5-3.5倍區間
  • 操作策略：PBR跌到區間下緣買進，漲到上緣賣出

  • 快速判斷個股目前股價在歷史PBR區間的位置
  • 景氣循環股的買賣點參考

五、PBR的五大陷阱 — 低PBR不一定是便宜

陷阱一：PBR<1就是便宜？

錯誤認知：PBR<1代表股價低於淨值，應該撿便宜。

真相：

  • 面板股（友達、群創）長期PBR<1，但股價持續下跌
  • PBR低可能是因為市場不看好這家公司，而不是被低估
  • 淨值不等於清算價值：設備可能根本賣不到帳面價格

怎麼做：PBR<1時，先確認：
1. ROE是否穩定為正？
2. 營收是否持續成長？
3. FCF是否為正？
4. 產業是否處於衰退期？

三個Yes+不是衰退產業 → 可能是真便宜
任何一個No → 大概率是價值陷阱

陷阱二：無形資產占比高時PBR失真

情境：科技公司品牌價值、專利、演算法不在資產負債表上，淨值嚴重低估。

實例：

  • Google的品牌和演算法價值數千億美元，但帳面上看不到
  • 用PBR看Google會覺得「超貴」（PBR極高），但實際上Google的獲利能力遠超帳面資產

怎麼做：科技股、品牌股、軟體股不要用PBR估值，改用PE或PEG。

陷阱三：景氣循環股的PBR高低判斷要反過來看

情境：循環股在景氣高峰時EPS最高，PE看起來最低——這時候反而該賣。

正確用法：

  • PBR跌到歷史低檔區間 + 產業開始復甦 → 買進
  • PBR漲到歷史高檔區間 + 產業開始過熱 → 賣出
  • 不要只看PBR數字，要看PBR在歷史區間的位置

實例：

  • 航運股在2020年PBR跌到0.4倍 → 景氣谷底，事後證明是買點
  • 航運股在2021年PBR漲到3倍以上 → 景氣高峰，事後證明是賣點

陷阱四：淨值為負的公司PBR無法計算

情境：長期虧損的公司，累積虧損超過股本，每股淨值為負值。

怎麼做：

  • 淨值為負時PBR失去意義，改看營收成長率、現金流量
  • 淨值為負代表公司已經資不抵債，除非有明確的轉機，否則應避開

陷阱五：資產減損讓淨值突然縮水

情境：公司進行商譽減損、固定資產減損，淨值突然大幅下降，PBR反而變高。

怎麼做：

  • 看PBR趨勢，不要只看單一時間點
  • 如果淨值突然大幅變動，先確認是否有資產減損或除權息
  • 減損後的淨值反而更接近真實價值（因為擠掉了水分）

六、PBR實戰選股SOP

怎麼做（4步篩選法）：

1. 第一步：用PBR快速篩選

  • PBR < 1.0（傳統PBR）初步入選

2. 第二步：搭配ROE確認品質

  • ROE > 10%（至少）

3. 第三步：確認產業適用性

  • 適用PBR選股：金融股、資產股、景氣循環股、控股公司

4. 第四步：用PBR河流圖找買點

  • 看目前PBR在歷史區間的位置

心得

之前學過本益比河流圖，今天加上PBR河流圖，整個估值工具箱更完整了。

最深刻的體會是：PBR和PE不是替代關係，而是互補關係。PE看的是「這家公司能賺多少錢」，PBR看的是「這家公司值多少家當」。穩定獲利的公司用PE估，獲利不穩定的循環股用PBR估——用錯工具比不用工具更危險。

市場先生的那四種保守PBR算法讓我大開眼界。傳統PBR只是最寬鬆的估算，真正要算清算價值，至少要用有形資產淨值版，甚至流動資產淨值版。很多看起來PBR<1的「便宜股」，用流動資產淨值一算，PBR可能超過1.5——根本不便宜。

PBR最大的陷阱就是「低PBR=便宜」的直覺。面板股PBR長期低於1，不代表可以撿便宜，而是市場在告訴你「這些資產根本不值帳面上那個數字」。真正值得關注的是PBR低+ROE高的組合——ROE證明這些資產確實能創造利潤，PBR低只是市場暫時忽略了。

最後，PBR河流圖對循環股的操作太實用了。台泥、中鋼這種股票，PBR在固定區間來回跑，跌到歷史低檔就是買點，漲到歷史高檔就是賣點——不需要猜行情，只要看PBR在河流圖的位置就夠了。
