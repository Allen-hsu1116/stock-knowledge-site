2026-03-31 09:09 - 成長股 vs 價值股選股差異

文章：

  • 價值股、成長股、配息股是什麼？三種類型的ETF績效表現如何？
  • 價值股vs.成長股，如何避雷？

核心知識

價值股（Value Stock）

是什麼：價值相對價格被低估的股票，公司未來獲利能力高於目前股價反映的價值。

怎麼判斷：
1. 本益比偏低：低於同業或歷史平均
2. 股價淨值比偏低：P/B < 1 或低於產業平均
3. 殖利率較高：現金股利殖利率 > 5%
4. 基本面穩健：獲利能力、現金流正常

實例：

  • 情境：中華電信（2412），本益比約 12 倍，殖利率約 5%
  • 判斷：股價淨值比低、配息穩定、現金流穩健，符合價值股特徵

常見產業：金融、工業、公用事業、傳產

適用時機：

  • 空頭市場或經濟不確定時
  • 長期存股、追求穩定現金流
  • 風險趨避者

陷阱：

  • 「便宜」不代表會漲，可能公司正在走下坡
  • 單看本益比、淨值比可能忽略未來獲利惡化

成長股（Growth Stock）

是什麼：具有高成長潛能的股票，營收、獲利增長速度快於整體市場和產業。

怎麼判斷：
1. 營收成長率：YoY > 15%
2. EPS 成長率：連續多年成長
3. 高本益比：市場給予溢價，反映未來成長預期
4. 產業前景：處於成長期的新興產業

實例：

  • 情境：台積電（2330），營收 YoY 成長 20%+，本益比約 15-20 倍
  • 判斷：半導體景氣向上、技術領先、獲利持續成長，符合成長股特徵

常見產業：資訊科技、生技醫療、電動車、AI 相關

適用時機：

  • 多頭市場、經濟擴張期
  • 追求資本利得、可承受較高波動
  • 願意承擔較高風險

陷阱：

  • 「昂貴」不代表會跌，可能公司真的優秀
  • 高本益比可能反映市場過度樂觀，成長不如預期時修正更慘

配息股（Dividend Stock）

是什麼：穩定配發現金股利的股票，股價波動較小。

怎麼判斷：
1. 配息穩定：連續多年配息
2. 殖利率合理：4-6% 區間
3. 現金流穩健：營業現金流足以支應股利
4. 盈餘分配率：60-80% 區間

實例：

  • 情境：中華電信每年配息約 4.5 元，股價約 110 元，殖利率約 4.1%
  • 判斷：電信業現金流穩定、配息可持續

適用時機：

  • 退休族、保守投資人
  • 追求穩定現金流

重要觀念：

  • 配息是「左手換右手」，股價會扣除配息金額
  • 高股息可能是盈餘配發率高，要注意公司再投資能力

三種股票 ETF 績效比較

美股 ETF 歷史績效（2008-2020）：

  • 成長型 ETF（IUSG）：多頭表現最佳，2008 年跌幅最深
  • 價值型 ETF（VTV）：表現平穩，空頭抗跌
  • 配息型 ETF（DVY）：介於兩者之間

關鍵數據：

  • 2020 年成長股表現明顯優於價值股
  • 2008 年金融海嘯，成長股跌幅最大

利差反轉與價值股表現

是什麼：長短期公債利差反轉（短期利率 > 長期利率）常是經濟衰退前兆。

歷史經驗：

  • 利差反轉時，價值股往往表現優於成長股
  • 2022 年底美國長短期利差擴大至 -1.6%，但成長股仍強勢

怎麼用：

  • 當利差反轉出現，考慮增加價值股配置
  • 降低成長股比重，因為經濟衰退時成長股修正更深

降低踩雷機率

價值股踩雷風險：

  • 買到「便宜但正在衰退」的公司
  • 金融股可能受銀行業危機影響

成長股踩雷風險：

  • 買到「昂貴但成長不如預期」的公司
  • 高本益比修正時跌幅更慘

避雷方法：
1. 價值股：檢查產業前景、避開高危機產業（如 2023 年美國中小型銀行）
2. 成長股：檢查成長是否可持續、估值是否過度膨脹
3. 分散配置：同時持有價值股、成長股，對沖風格風險

心得

這回合學到了價值股、成長股、配息股的本質差異。關鍵洞察：

1. 不要被標籤綁架：價值股不是「便宜就好」，成長股不是「貴就對」。核心是判斷公司未來獲利能力是否被市場低估或高估。

2. 追求的本質：

  • 價值股投資人追求「高勝率」——便宜買好公司

3. 環境決定風格：利差反轉時，價值股相對抗跌；多頭時，成長股漲更多。要根據總體經濟調整配置。

4. 最重要的觀念：「別為了買股票，而尋找一個合理的藉口。」——先搞懂公司，再決定分類，不是反過來。

第 23 回合學習 - 營收財報公布時程與解讀

文章：

  • 上市櫃每月營收何時公布？
  • 價值投資行事曆與台股營收、財報公布時間表

台股財報公布時程總覽

是什麼：台股上市櫃公司依法必須定期公布營收和財報，這些是投資人追蹤公司營運狀況最即時的依據。

怎麼做：記住以下關鍵日期：

報告類型公布期限
月營收次月10日前（遇假日順延）
Q1季報5月15日前
Q2季報8月14日前
Q3季報11月14日前
年報次年3月31日前

實例：

  • 情境：想了解某公司Q1表現
  • 時程：Q1結束是3/31，季報最晚在5/15前公布
  • 操作：在5月初關注該公司季報發布，用EPS成長率篩選績優股

適用時機：每月10日前後是營收公布期，可關注營收創新高或年增率大幅成長的個股。

月營收數據解讀

是什麼：月營收報告包含本月營收、去年同月營收、累計營收、增減金額、增減百分比（YoY年增率）。

怎麼做：
1. 計算月增率(MoM)：（本月營收 - 上月營收）/ 上月營收 × 100%
2. 計算年增率(YoY)：（當月營收 - 去年同期營收）/ 去年同期營收 × 100%
3. 年增率比月增率更有參考價值（月增率受淡旺季、工作天數影響）

實例：

  • 情境：某公司6月營收50億，去年同期40億
  • 計算：YoY = (50-40)/40 × 100% = 25%
  • 判斷：年增率25%，營收成長趨勢向上，股價展望可能偏多

適用時機：每月營收公布日（10日前後），追蹤持股或關注標的的營運變化。

年度行事曆關鍵節點

是什麼：掌握財報公布節奏，提前布局或避開風險。

怎麼做：

月份關注重點
1/10前一年12月營收（可知去年全年營收狀況）
3/31年報及Q4財報公布截止（得知去年全年EPS，可推估股利）
5/15Q1季報截止（找出Q1業績成長股）
6-8月除權息旺季
8/14Q2季報截止（確認Q2業績是否延續成長）
11/14Q3季報截止（前三季EPS可推估全年EPS）

實例：

  • 情境：想布局除權息行情
  • 時程：6-8月是除權息旺季，7月是高峰
  • 操作：在5-6月關注高股息個股的股利政策，選擇殖利率佳的標的

適用時機：中長期投資人，提前規劃年度投資策略。

財報未公布的後果

是什麼：公司若無法如期公布財報，將面臨處罰。

怎麼做：
1. 月營收未公布：罰鍰24萬-240萬元
2. 季報/年報難產：可能被停止交易
3. 停止交易6個月未改善：股票下市，40天後變廢紙

實例：

  • 情境：某公司連續延遲公布季報
  • 判斷：財務可能有問題，內控有疑慮
  • 操作：避開這類公司，列入觀察名單

適用時機：篩選股票時，優先排除財報延遲或未公布的公司。

心得

財報公布時間表是投資人的行事曆，每個月10日前後關注營收，5/15、8/14、11/14、3/31關注季報年報。年增率(YoY)比月增率(MoM)更有參考價值，因為月增率會受淡旺季、工作天數影響。

財報是落後指標，反映的是「過去」的表現，但對於判斷公司營運趨勢仍然重要。若公司連財報都交不出來，基本上就不用考慮了——內控有問題的公司不值得冒險。

第 24 回合學習 - 股東權益變動表解讀

文章：

  • 讀得懂的財務報表03｜股東權益表
  • 誰偷吃了你的批薩?請看權益變動表

股東權益變動表是什麼

是什麼：呈現公司在一段特定期間內，股東權益各項目的增減變動情況。是財報四大表中最容易被忽略的一張，但對股東來說卻是最有關係的一張表。

四大表的關係：

  • 資產負債表：看公司體質
  • 損益表：看公司賺錢能力
  • 股東權益變動表：看股東權益剩什麼
  • 現金流量表：看現金進出

會計恆等式：資產 = 負債 + 權益 → 權益 = 資產 - 負債（又稱「淨值」）

股東權益六大組成項目

1. 股本（法定資本）

是什麼：已發行股數 × 面額（台灣通常每股面額 10 元）

怎麼看：

  • 注意「已發行股數」vs「核定股本」vs「流通在外股數」
  • 核定股本：公司向政府註冊的可發行上限
  • 已發行股數：實際發行並收足股款的股份
  • 流通在外股數：已發行股數 - 庫藏股（EPS、每股淨值的分母）

2. 資本公積

是什麼：股東投資超過面額的部分，以及資產重估增值、處分資產利益等

怎麼看：

  • 股本溢價：股東願意用超過面額的價格買股票
  • 超額程度越高，代表股東對公司評價越高

功用：

  • 可彌補虧損
  • 可發放股利
  • 警訊：若公司動用資本公積調節虧損或發股利，多半意味營運有狀況

3. 保留盈餘

是什麼：公司累計賺取、尚未分配給股東的盈餘

公式：期初保留盈餘 + 本期淨利（-本期淨損） - 分派股利 = 期末保留盈餘

怎麼看：

  • 保留盈餘佔股東權益比率越高越好
  • 代表公司靠經營累積財富，而非靠新股東挹注
  • 若持續虧損，就叫「累積虧損」

4. 庫藏股

是什麼：公司從市場買回自己的股票，是權益的減項

怎麼看：

  • 等於資本暫時退還給股東
  • 買回成本列為權益減項
  • 流通在外股數 = 已發行股數 - 庫藏股

5. 累計其他綜合損益

是什麼：不經過損益表、但會影響權益的項目

常見來源：

  • 匯率變動造成的未實現損益
  • 證券價格波動
  • 資產重估增值

6. 非控制權益（少數股權）

是什麼：合併報表中，母公司未持有 100% 股權的子公司，由其他人持有的少數股權

重要分析重點

保留盈餘佔股東權益比率

是什麼：判斷股東財富是靠經營累積還是靠新股東挹注

怎麼算：保留盈餘 ÷ 股東權益 × 100%

判斷標準：

  • 比率高 → 公司靠賺錢累積財富，分食利潤的人沒變多
  • 比率低 → 財富來自新股東，分食利潤的人變多了

心得

股東權益變動表就像是在追蹤「披薩怎麼被分配」的過程。資產負債表告訴你披薩有多大，損益表告訴你披薩有沒有變大，但只有股東權益變動表能讓你知道：披薩被切成幾片、有沒有人多帶朋友來分食、公司是把錢留下來做料還是退還給股東。

最重要的指標是「保留盈餘佔股東權益比率」。一間公司如果保留盈餘佔比高，代表它是靠賺錢慢慢變大的，不是靠不斷跟股東要錢。

第 25 回合學習 - 盈餘品質指標分析

文章：

  • 財報分析 第34章 盈餘品質指標
  • 觀察「盈餘品質指標」，輕鬆避開作假帳的地雷股

什麼是盈餘品質？

定義：盈餘品質（Earnings Quality）是指用過去盈餘來預估未來盈餘的可靠程度。品質高表示公司盈餘真實反映經營績效，品質差則表示盈餘數字可能有問題。

為什麼重要：

  • 博達科技掏空案後，投資人開始注意到盈餘數字可以動手腳
  • 高品質盈餘大部分是現金組成，有持續性，能預測未來
  • 低品質盈餘可能隱藏財務問題或作假帳風險

盈餘品質分析步驟

怎麼做：
1. 確認公司使用的會計方法是否適當
2. 查閱會計師查核報告書（是否有保留意見）
3. 確認公司之前是否有重編財報（重編是警訊）
4. 計算公司相關的營運風險
5. 利用七項指標進行量化分析

七大盈餘品質指標

1. 存貨指標

公式：存貨指標 = 營收增加百分比 - 存貨增加百分比

判斷標準：

  • 指標 > 0：存貨控管適當，盈餘會增加 ✓
  • 指標 < 0：存貨累積太快，可能有滞銷風險 ⚠️

2. 應收款項指標

公式：應收款項指標 = 營收增加百分比 - 應收款項增加百分比

判斷標準：

  • 指標 > 0：應收款項催收良好，產品銷售佳 ✓
  • 指標 < 0：可能用賒銷衝業績，或收款困難 ⚠️

3. 毛利指標

公式：毛利指標 = 毛利增加百分比 - 營收增加百分比

判斷標準：

  • 指標 > 0：獲利能力超過營收成長 ✓
  • 指標 < 0：成本控制可能有問題 ⚠️

4. 銷管費用指標

公式：銷管費用指標 = 營收增加百分比 - 銷管費用增加百分比

判斷標準：

  • 指標 > 0：銷管費用未因營收增加而暴增 ✓
  • 指標 < 0：費用增加太快，可能有資源浪費 ⚠️

5. 備抵壞帳指標

公式：備抵壞帳指標 = 備抵壞帳增加百分比 - 應收款項增加百分比

判斷標準：

  • 指標 > 0：備抵壞帳提列充足 ✓
  • 指標 < 0：備抵壞帳可能提列不足，有修飾財報嫌疑 ⚠️

6. 研發費用指標

公式：研發費用指標 = 研發費用增加百分比 - 同業研發費用增加百分比

判斷標準：

  • 指標 > 0：研發投入高於同業，有利長期發展（但不宜過高）
  • 指標 < 0：研發投入不足，可能影響競爭力

7. 員工生產力指標

公式：員工生產力指標 =（本期員工平均營收 - 前期）/ 前期

判斷標準：

  • 指標 > 0：員工生產力成長，人力素質提升 ✓
  • 指標 < 0：生產力下降，可能管理有問題 ⚠️

心得

這套指標最有價值的地方在於「不只看數字，還看數字之間的關係」。單看 EPS 成長可能被騙，但如果：

  • 營收成長但應收帳款暴增 → 可能是賒銷衝業績
  • 營收成長但存貨暴增 → 可能賣不出去
  • 營收成長但毛利下降 → 可能削價競爭

這些「質」的問題，單看 EPS 看不出來。
