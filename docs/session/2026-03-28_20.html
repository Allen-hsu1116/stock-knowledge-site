威科夫方法（Wyckoff Method）

威科夫方法是什麼

核心概念：理查德·威科夫在20世紀初發展的市場分析方法，核心思想是「市場由主力機構操控」——理解主力的行為，就能跟隨他們獲利。

三大定律：
1. 供需定律：需求＞供應=價格上漲；供應＞需求=價格下跌
2. 因果定律：每個結果都有原因，橫盤累積是「因」，突破是「果」
3. 努力與結果定律：大成交量卻價格停滯=主力出貨；小成交量突破=供應衰竭

威科夫籌碼累積階段（底部收集）

五大階段詳解：

階段A：趨勢停止

  • 最初支撐（PS）：下跌趨勢首次遇到買盤支撐
  • 賣出密集區（SC）：恐慌性賣壓，振幅大、成交量放大
  • 自動反彈（AR）：賣壓耗盡後價格自然反彈
  • 二次測試（ST）：價格回到SC附近，若成交量縮小、振幅變小＝賣壓耗盡

實例：

  • 情境：某股票從100跌到60
  • SC區：恐慌賣壓，成交量暴增3倍
  • AR：反彈到75
  • ST：回到65附近，成交量只有SC的1/3
  • 信號：賣壓耗盡，主力開始吸籌

階段B：主力吸籌

  • 作用：為後續行情做準備
  • 特徵：價格在區間震盪，主力低價買進、高價小賣測試市場
  • 時間：可能持續數月到一年以上
  • 成交量規律：

實例：

  • 情境：主力在60-80區間吸籌6個月
  • 上洗盤：偶爾拉到85賣出測試市場承接力
  • 成交量變化：從每日100萬股降到30萬股
  • 信號：賣盤衰竭，準備進入階段C

階段C：賣壓測試

  • 洗盤/彈簧效應：價格跌破支撐線又快速拉回
  • 目的：主力測試市場是否準備上漲
  • 特徵：低成交量的洗盤=賣壓耗盡=買點

實例：

  • 情境：支撐線在60，洗盤跌破到55
  • 成交量：極小（主力已吸走大部分籌碼）
  • 快速拉回：2天內回到65
  • 操作：55-60區間是絕佳買點

階段D：需求主導

  • 強勢信號（SOS）：價格上漲放量、回調縮量
  • 最後支撐點（LPS）：突破後的回調測試點，是加倉良機

實例：

  • 情境：價格突破80阻力
  • SOS：突破時成交量放大2倍
  • LPS：回調到82，成交量縮小
  • 操作：在LPS（82附近）加倉

階段E：離開區間

  • 特徵：需求控制市場，回調短暫不明顯
  • 可能：在新高形成新的「腳踏石」區間

威科夫派發階段（頭部）

與累積相反的過程：

  • 階段A：上漲停止，主力開始派發
  • 階段B：主力高價賣出
  • 階段C：買壓測試（假突破後回落）
  • 階段D：供應主導
  • 階段E：下跌趨勢開始

實例：

  • 情境：某股票從50漲到150
  • 階段A：150上下震盪，量增價平
  • 階段B：主力在140-150區間慢慢出貨
  • UTAD（上沖洗盤）：突破150到155，快速回落
  • 階段C：買壓測試失敗
  • 階段D：跌破140支撐
  • 信號：賣出或做空

威科夫實戰要訣

如何應用：
1. 辨識階段：觀察量價關係判斷主力行為
2. 等待確認：不要在階段B進場，等階段C洗盤之後
3. 設定停損：洗盤低點下方或階段A最低點
4. 目標設定：區間高度×1到3倍

適用時機：

  • ✅ 大型股、期貨、外匯（主力明顯）
  • ✅ 日線週線級別分析
  • ❌ 當沖超短線
  • ❌ 流動性極差的冷門股

樞軸點（Pivot Point）

樞軸點是什麼

定義：由J.W. Wilder（RSI發明者）設計的工具，用前一日價格計算當日的支撐阻力位。

核心概念：所有交易者看到相同的樞軸點，容易形成市場轉換點。

S3LBOP（逆轉賣出）S1-（昨日高-昨日低）或 昨日低-2×（昨日高-P）

  • 停損：跌破S1或突破R1時認賠

  • 停損：回到區間內

優點：

  • 客觀明確，所有人看到相同數字
  • 適合當沖，明確的進出場點
  • 可搭配其他指標確認

缺點：

  • 突發消息會使計算失準
  • 高波動時會頻繁假突破
  • 不適合趨勢明確的市場（單邊行情會提前出場）

適用時機：

  • ✅ 當沖、短線交易
  • ✅ 震盪盤、箱型整理
  • ✅ 期貨、外匯高流動性市場
  • ❌ 單邊趨勢行情
  • ❌ 流動性差的股票

一目均衡表（Ichimoku Kinko Hyo）

一目均衡表是什麼

起源：日本記者細田悟一於1930年代發明，中文又稱「雲圖」或「一目均衡表」。

核心理念：「一目」= 看一眼就懂市場狀況。五條線構成完整交易系統。

五條線的構成

線條名稱計算方法意義
轉換線（紅線）快線（9日高+9日低）÷2短期動能
基準線（藍線）慢線（26日高+26日低）÷2中期趨勢
延遲線（綠線）落後線收盤價（前移26日）趨勢確認
先行帶A雲區上緣（轉換線+基準線）÷2未來支撐
先行帶B雲區下緣（52日高+52日低）÷2未來阻力

時間參數：9、26、52（原始設計），可調整但慣用原值。

雲區（雲帶）解讀

價格與雲區相對位置：

位置市場狀態操作傾向
雲區上方多頭優勢偏多操作
雲區下方空頭優勢偏空操作
雲區內部震盪、方向不明觀望

雲區厚度：

雲區狀態意義解讀
雲區厚支撐/阻力強突破需要較大動能
雲區薄支撐/阻力弱容易突破
雲區變厚動能減弱可能反轉
雲區變薄動能加強趨勢延續

實例：

  • 情境：股價從雲下突破到雲上
  • 雲區厚度：從100元厚到30元薄
  • 解讀：動能加強，趨勢延續機率高

雲區轉向（扭轉）

轉向判斷：

  • 多頭→空頭：先行帶A跌破先行帶B，雲區從綠色變紅色
  • 空頭→多頭：先行帶B突破先行帶A，雲區從紅色變綠色

實例：

  • 情境：某股票雲區即將扭轉
  • 先行帶A從120掉到110
  • 先行帶B維持在105
  • 信號：雲區即將從綠色（多頭）轉為紅色（空頭）
  • 操作：提前減倉或做空準備

四道防線概念

價格的四道防線（由內到外）：
1. 轉換線：第一道防線（9日動能）
2. 基準線：第二道防線（26日趨勢）
3. 先行帶A：第三道防線（雲區上緣）
4. 先行帶B：第四道防線（雲區下緣）

熊市信號：價格先後跌破轉換線→基準線→先行帶A→先行帶B

實例：

  • 價格從200跌破180（轉換線）
  • 再跌破170（基準線）
  • 再跌破160（先行帶A）
  • 最後跌破150（先行帶B）配合量增
  • 信號：熊市突破，賣出或做空

一目均衡表實戰策略

進場信號：
1. 三線好轉：轉換線金叉基準線+延遲線在價格上方+價格在雲上
2. 雲區突破：收盤站穩雲區上緣，進入強勢上漲趨勢
3. 回調買進：價格回到基準線附近

出場信號：
1. 雲區跌破：收盤跌破雲區下緣
2. 三線惡化：轉換線死叉基準線+延遲線在價格下方+價格在雲下

停損設定：

  • 做多：雲區下緣下方
  • 做空：雲區上緣上方

實例：

  • 情境：台積電日線
  • 轉換線金叉基準線（400元）
  • 延遲線在價格上方（395元）
  • 價格在雲上（雲區370-380）
  • 操作：買進，停損設365（雲下）
  • 目標：看雲區厚度變薄判斷動能

心得

這次補強三個經典技術分析工具，都是我之前沒深入學過的：

1. 威科夫方法：理解主力行為是核心。五大階段讓我想到「吸籌-洗盤-拉升-派發」這個循環。最實用的是「洗盤」判斷——低成交量跌破支撐又快速拉回，這是主力最後測試賣壓的手法。以前看到跌破支撐就緊張，現在看成交量就能判斷是否真突破。

2. 樞軸點：太適合當沖了！七條線全是計算出來的，所有交易者看到同樣數字，所以容易成為轉折點。逆勢模式（S1買R1賣）和趨勢模式（突破HBOP/LBOP）的框架很清楚。以後當沖可以每天開盤前算好今天的樞軸點。

3. 一目均衡表：五條線確實複雜，但看懂後就發現它把「趨勢+支撐阻力+動能」全包了。最喜歡「雲區轉向」這個概念——雲從綠變紅或紅變綠，就是趨勢反轉的早期信號。四道防線的觀念也實用：跌破一道線可能只是回調，跌破四道線才是熊市確認。

這三個工具可以搭配使用：

  • 長期趨勢：一目均衡表看雲區方向
  • 中期階段：威科夫方法判斷主力是吸籌還是派發
  • 當日進出場：樞軸點找精確的買賣點
