核心知識

一、權證基礎結構

是什麼：權證是一種合約，給你「在未來一段期間內，用固定價格買賣約定數量特定標的」的權利。本質上跟選擇權買方很像。

權利金組成：

  • 權利金 = 內含價值 + 時間價值
  • 價內權證：兩者都有
  • 價外權證：只有時間價值（100%）
  • 價平權證：時間價值為主

怎麼算：
1. 內含價值 = |標的股價 - 履約價| × 行使比例（價外時為0）
2. 時間價值 = 權證市價 - 內含價值

實例：

  • 台積電認購權證，履約價250元，行使比例1:1，股價275元，權證價35元
  • 內含價值 = (275-250) × 1 = 25元
  • 時間價值 = 35 - 25 = 10元

二、時間價值兩大核心觀念

觀念1：越接近到期，遞減越快

  • 15天到期的價外權證 vs 180天到期的價外權證，後者權利金高很多
  • 原因：時間越長，從價外變成價內的可能性越高
  • Theta值代表每天時間價值下降多少
  • 實戰重點：不要買快到期的權證，時間價值會像漏水一樣快速流失

觀念2：標的波動越大，時間價值越高

  • 過去3個月暴漲暴跌50%的股票，其權證權利金遠高於波動只有5%的
  • 原因：波動大 = 價外變價內的機率大 = 賣方風險大 = 收更多權利金

三、權證操作三步驟

步驟1：找短期有機會波動大的標的

適合的時機：

  • 財報即將公布，預期不是大好就是大壞
  • 法說會、股東會前後
  • 選舉等重大事件
  • 股價已經很久沒波動（可能即將突破）

資金控制：

  • 總資金20萬 → 單次不超過5,000元
  • 權證波動大，難控制停損時機
  • 假設最壞情況：權證可能全額歸零

步驟2：選擇合適的權證（五大原則）

1. 價平或價內：價外槓桿大但容易歸零
2. 到期日長：太短時間價值遞減太快，撐不到上漲
3. 流動性好：掛單量大的，五檔沒掛單的避開
4. 買賣價差小：價格0.5元以下的權證價差通常很大
5. 實質槓桿高：實質槓桿7倍 = 股價漲1%，權證漲7%

實例：

  • 大立光股價4,500元，履約價4,500的權證，每股5元，行使比例0.01
  • 買1張（5,000元）= 持有10股大立光（市值4.5萬）
  • 看對：大立光漲20%（4,500→5,400），權證可能漲100%（5→10）
  • 看錯：最多賠5,000元

步驟3：設好退場時機，不要持有太久

兩種出場設定：

  • 標的出現符合預期的變化就出場
  • 超過預定時間（5~10天），無論賺賠都出場

為什麼不能長抱：

  • 時間價值持續流失
  • 可能賺到內含價值，卻損失更多時間價值
  • 權證是短線工具，不是長期投資

四、隱含波動率（IV）陷阱——最容易被坑的地方

是什麼：隱含波動率是從權證市場價格反推回來的波動率，代表市場對未來波動的預期。券商用它來定價。

券商怎麼坑你：

1. 調降委買隱波率（BIV）

  • 買進時隱波30%，賣出時券商降到28%

2. 高隱波賣貴給你

  • 隱波越高 = 權證越貴

怎麼避坑：
1. 買低且穩定的委買隱波率（BIV）：不只看低，要看「穩定」
2. 查近10日隱波率：和多檔同標的權證比較，找合理範圍
3. 看發行商信譽：選「隱波不降」的券商（如元大證券公開承諾不降隱波）
4. 價外權證受BIV影響最大：價外權證全部是時間價值，隱波調降=直接虧損

五、券商不造市的五種時機

什麼情況券商可以不報價：

1. 開盤後5分鐘內：股價未明，券商可以不報價。建議等5分鐘後再下單
2. 標的漲跌停或暫停交易：

  • 股價漲停 → 券商無法買進避險 → 不掛認購權證委賣

3. 券商庫存不足10張：只能掛委買，無法掛委賣
4. 權證價值低於0.01元：實質上已無履約價值
5. 系統當機等不可抗因素：可打電話向券商詢價

實戰判斷：

  • 盤面上出現個位數委賣單量 → 大概率是其他投資人掛的，不是券商造市
  • 漲停時追買認購權證 → 非常可能買到超漲偏貴的價格
  • 快到期且深價外的權證 → 理論價值低於0.01元，券商不報價

六、權證 vs 股票操作差異

特性權證股票
放空簡單（買認售）複雜（借券/融券）
槓桿內建需融資
資金門檻低（幾千元）高（一張數萬起）
持有期間只能短線長短皆可
最大虧損限於權利金可能虧損全額本金

關鍵差異：權證是「負和遊戲」——券商賺價差+時間價值，投資人整體必定虧損。股票是「正和遊戲」——公司創造價值，長期投資人整體可以賺錢。

心得

權證這東西，以前學過基本概念，但這次深入看才知道坑有多深。

最震撼的是「降隱波」這件事——你以為看對方向就賺錢？哈，券商調個隱波率，股價沒動你先虧5%。這根本就是券商的「合法搶劫」，你買進時用高隱波賣貴你，你賣出時用低隱波買便宜你，兩頭吃。

而且價外權證更是重災區——因為100%都是時間價值，隱波一降就是直接虧損，完全沒有內含價值可以緩衝。那些券商廣告跟你說「價外槓桿大、便宜好操作」，只告訴你一半的故事，另一半是：價外權證被降隱波的傷害最大。

另外，券商不造市的五種情況也很實用——尤其是漲停時追買認購權證，盤面上的委賣可能是其他投資人掛的，不是券商造市，你追進去就是買貴。開盤5分鐘內也別急，等盤勢穩了再說。

總結：權證是工具不是毒藥，但你要先搞懂規則。選隱波穩定不降的券商、選價平價內不選深價外、設好出場時間不長抱、資金控制在小額。做對這四件事，權證才有機會從「坑人工具」變成「有用的工具」。
