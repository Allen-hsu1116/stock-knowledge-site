重點筆記

賣方視角解讀 OI

  • 為什麼看賣方：賣方風險大、資金多、留倉久，解讀方向更可信
  • 資金門資金門檻：賣方要預繳保證金（可能幾千萬到上億）
  • 風險不對稱：買方最大虧損 = 權利金；賣方最大虧損 = 理論上無限
  • 實例：17300 PUT 有 25,781 口 OI → 代表 25,781 口 SELL PUT，莊家認為不會跌破 17300

OI 支撐壓力判斷

  • Call 最大 OI = 壓力區（莊家認為漲不過）
  • Put 最大 OI = 支撐區（莊家認為跌不破）
  • 查詢方式：期交所官網或 optree.tw

Put/Call Ratio 關鍵數字

  • > 100%：Put OI > Call OI，偏多頭
  • < 100%：Call OI > Put OI，偏空頭
  • 接近 115%：強勢多頭
  • 跌破 75%：強勢空頭

上漲增加增加強勢軋空
上漲減少增加多頭反轉
下跌增加增加強勢走空

倒莊效應

  • 是什麼：價格突破 OI 最大區，賣方被強制停損，造成價格加速突破
  • 突破壓力：Call 賣方倒莊，被迫買回 Call 止損
  • 跌破支撐：Put 賣方倒莊，被迫賣出 Put 止損
