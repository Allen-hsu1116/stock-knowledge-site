一、庫藏股是什麼？

定義：公司用自己的資金，從市場買回已發行的股票。這些股票不再對外流通。

特性：

  • 沒有投票權
  • 沒有股利分派權
  • 公司解散時不能變現
  • 庫藏股總價值不能超過公司資本的 5%

會計處理：資產負債表的股東權益項目中列為「減項」。

總發行股數公式

總已發行股數 = 流通在外股票 + 庫藏股

EPS 計算、股利分配只算「流通在外股票」，不計入庫藏股。

二、公司買回庫藏股的三大原因（法令規定）

1. 轉讓股份給員工

  • 作為員工認股權或激勵制度的股票來源
  • 留住優秀人才

2. 作為股權轉換來源

  • 給附認股權證公司債、可轉換公司債使用

3. 維護公司信用及股東權益

  • 俗稱「護盤」
  • 避免股價過度下跌

三、大股東買回庫藏股的「真正」四大理由

理由1：硬撐股價保質押

  • 大股東用股票質押借錢，股價跌會影響借款額度
  • 用公司錢護盤，保護自己的利益

理由2：用公司錢幫特定人出貨

  • 安排特定時間、價格，用公司資金買進
  • 特定人士趁機賣出大量持股

理由3：防止惡意併購

  • 提升大股東股權比例
  • 拉高股價讓併購成本增加

理由4：節稅效果

  • 庫藏股減資代替發放現金股利
  • 對大股東有節稅效果（台灣現金股利要課稅）

四、庫藏股對股價的實際影響（數據驗證）

實驗背景

  • 使用 2009-2012 年台股資料
  • 2009 年為漲勢、2011 年為跌勢

結論1：執行庫藏股期間買進，平均不賺錢

年度報酬率表現
2009年獲利集中（大盤上漲）
2011年大幅虧損（大盤下跌）

重點：庫藏股效果受大盤行情影響很大，不是穩定獲利方法。

結論2：股價上漲時，「轉讓股份予員工」效果較好

條件：過去半年漲幅 > 10%

類型2年後獲利100%以上比例
轉讓股份予員工30% (15家/50家)
維護公司信用及股東權益10% (5家/50家)

結論3：股價大跌時，「維護公司信用及股東權益」效果較好

條件：過去半年跌幅 > 40%

類型2年後獲利100%以上比例
維護公司信用及股東權益40%
轉讓股份予員工20%

結論4：庫藏股「急救無效」是危險訊號

定義：執行庫藏股後 1 個月內仍下跌 5% 以上

後果：

  • 未來 2 年上漲機率 < 50%
  • 倒賠 20% 的機率高達 25%
  • 若 1 年內觀察，虧損機率可能 > 60%

實例對比：

  • ✅ 可口可樂：營收成長 + 庫藏股加持 = 股價長期上漲
  • ❌ 宏達電：多次宣告庫藏股但執行率差 + 業績下滑 = 股價持續下跌

五、如何判斷庫藏股是利多還是利空？

正面訊號

條件原因
公司基本面良好本業賺錢，買回是加分
股價被低估公司用閒置資金投資自己
執行率高公司真的用行動展現信心
單次或少次宣告訊號效應強

負面訊號

條件原因
公司業績下滑可能只是護盤、美化財報
多次宣告庫藏股投資人產生「資訊疲乏」，效應遞減
執行率差口水護盤，騙投資人進場
股價已經在高點買貴了，無法提升股東價值
舉債買回財務槓桿過高，增加風險

關鍵檢查點

1. 執行率：預定買回 vs 實際買回的比例
2. 買回目的：轉讓員工 vs 維護權益
3. 股價位置：高點還是低點
4. 公司基本面：營收、獲利是否穩定
5. 過去紀錄：是否多次宣告但執行率差

Step 1：查詢庫藏股公告

  • 頻繁宣告庫藏股的公司
  • 業績持續下滑卻一直護盤
  • 用舉債資金買回庫藏股

心得

這篇研究讓我對庫藏股有了更清醒的認識。

以前覺得「公司買回股票 = 看好前景 = 利多」，但數據告訴我這是錯的。

關鍵體悟：
1. 動機比行為重要：同樣是買回庫藏股，為了員工激勵和為了護盤，長期效果差很多
2. 執行率是照妖鏡：很多公司喊得大聲，執行率卻很差，這種就是純粹的口水護盤
3. 急救無效是死亡訊號：如果公司買回庫藏股後一個月還跌 5%，基本可以刪自選股了
4. 台股 vs 美股：美國庫藏股是為了節稅和提升股東價值，台灣很多是為了護盤和幫大股東出貨

實務上：

  • 看到庫藏股公告，先查執行率和目的
  • 股價大跌後的「維護權益」庫藏股，反而比高點的更有參考價值
  • 不要單純因為庫藏股就買進，這不是買進理由
