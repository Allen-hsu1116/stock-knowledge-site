散戶四大致命心理偏誤——知道不等於做到

是什麼：行為金融學研究發現，散戶虧錢的主要原因不是技術差，而是心理偏誤。損失厭惡、處置效應、過度自信、FOMO（錯失恐懼）這四個偏誤，像寄生蟲一樣嵌在人類大腦裡，光「知道」無法消除，必須用系統化的方法對抗。

怎麼做：

1. 損失厭惡（Loss Aversion）：

  • 核心機制：虧1萬元的痛苦，大約是賺1萬元快樂的2-2.5倍

2. 處置效應（Disposition Effect）：

  • 核心機制：傾向過早賣出盈利資產，過久持有虧損資產

3. 過度自信偏誤（Overconfidence Bias）：

  • 核心機制：高估自己的判斷能力和資訊優勢，認為自己比他人更厲害

4. FOMO（Fear Of Missing Out，錯失恐懼）：

  • 核心機制：害怕錯過行情而衝動追高，把「潛在獲利」當成「已擁有但即將失去的東西」

適用時機：每天交易前和交易中，都要覺察自己的情緒狀態。特別是連續獲利後（過度自信高發期）和連續虧損後（報復性交易高發期）。

交易旅程框架——交易日誌不只是記帳

是什麼：多數人的交易日誌只記錄「買什麼、賣什麼、賺多少賠多少」，這跟記帳本沒兩樣。真正有效的交易日誌，應該按照「交易旅程」框架，把交易過程拆成三個階段，每個階段記錄不同內容，才能找出你的交易弱點在哪裡。

怎麼做：

1. 交易前階段（Pre-Trading）——準備與計畫：

  • 記錄內容：

2. 交易中階段（Trading）——執行與管理：

  • 記錄內容：

3. 交易後階段（Post-Trading）——回顧與改進：

  • 記錄內容：

實例：

  • 交易前記錄：今天大盤偏多，某股5MA>20MA>60MA多頭排列，昨帶量突破前高105元，預計106元進場，停損102元（前波高點下方），目標115元（壓力區）
  • 交易中記錄：開盤108元追進（偏高3元，因為開盤急漲），盤中回測106元時想加碼但猶豫沒加（恐懼情緒），尾盤110元收盤
  • 交易後記錄：+2元（+1.85%），偏離計畫（追高3元），加碼機會沒把握（恐懼），下次應在計畫價位掛單不追市價

適用時機：每筆交易都應記錄。短線交易者可簡化（只記關鍵節點），波段交易者應詳細記錄持有期間的心理變化。

損失厭惡的三種投資病症——你中了幾個？

是什麼：損失厭惡在實戰中會表現為三種具體的「病症」，每種都有不同的破壞力和對應解法。

怎麼做：

1. 處置效應（過早賣盈、過久抱虧）：

  • 症狀：賺3%就跑了，虧20%還在凹

2. 沉沒成本謬誤（因已投入而不願放棄）：

  • 症狀：「我已經虧了這麼多，現在賣太可惜」「再補一點攤平就好」

3. 錨定效應（買入價成為神聖參考點）：

  • 症狀：所有決策圍繞「回本」而非資產的未來價值

適用時機：每次持有虧損部位時，檢查自己是否中了這三種病症。最簡單的測試：問「如果我現在沒有這檔股票，我會買嗎？」

情緒覺察四步法——用情緒當訊號，不當主人

是什麼：情緒不是敵人，是訊號。研究顯示情緒是理性決策不可或缺的一部分。關鍵不是「消除情緒」，而是「覺察情緒→解讀訊號→理性決策」。

怎麼做：

1. 覺察：注意到自己正在經歷什麼情緒

  • 「我現在很興奮因為股票大漲」

2. 解讀：這個情緒在告訴我什麼？

  • 興奮 → 可能有獲利機會，但也可能FOMO

3. 評估：這個情緒和當下的決策有關嗎？

  • 興奮是因為基本面確實好？還是因為別人在賺？

4. 承諾：做出決定，不論是否交易

  • 「我注意到了我的興奮，但我沒有足夠的研究，所以我不交易」

實例：

  • 情境：某AI股突然大漲8%，你很興奮想追
  • 覺察：「我現在非常興奮，心跳加速」
  • 解讀：「這個興奮來自FOMO——別人都在賺，我怕錯過」
  • 評估：「我對這家公司沒有深入研究，不知道估值是否合理」
  • 承諾：「我注意到了我的FOMO，但因為缺乏研究，我不會追高。我會把它加入觀察名單，等回檔再研究」

適用時機：每次想做出不在計畫中的交易決策時。特別是「想追高」「想凹單」「想報復性交易」這三種情境。

交易日誌範本——7個必填欄位

是什麼：一個簡潔但完整的交易日誌範本，不需要花太多時間寫，但能捕捉最關鍵的資訊。

怎麼做：

1. 日期和時間：什麼時候買/賣的
2. 標的和方向：做多還是做空，哪檔股票
3. 進出場策略：本筆交易策略是什麼（波段追突破？支撐反彈？逆勢抄底？）
4. 進場理由：為什麼進場（技術面+基本面+籌碼面，至少兩個面向確認）
5. 進出場價格和部位大小：買價、賣價、張數、佔總資產比例
6. 損益金額和比例：單筆損益金額、損益百分比、佔整體資產的損益百分比
7. 心情起伏和教訓：持有期間的心理變化、想改策略的時刻、如果重來會怎麼做

進階做法：

  • 需要覆盤的交易另外開文件，把連結貼到表格裡
  • 定期統計：哪個市場勝率最高？哪種策略盈虧比最好？哪種情緒最容易導致虧損？
  • 月度回顧：統計當月勝率、平均盈虧比、最大單筆虧損、最常犯的錯誤

實例：

  • 日期：4/15，標的：台積電，方向：做多
  • 策略：波段追突破，進場理由：帶量突破前高1050+外資連3買+5MA>20MA
  • 進場：1060元×5張（佔總資產15%），停損：1020元，目標：1150元
  • 出場：1130元（5MA跌破出場），損益：+70元/張×5張=+35,000元（+6.6%）
  • 心情：漲到1100時想提前出場（處置效應），但20MA仍在向上所以續抱。教訓：移動停利比主觀出場好，下次繼續用

適用時機：每筆交易完成後都記錄。不記錄的交易等於白做——你無法改進你沒有測量的東西。

心得

這回合補強了交易中最容易被忽略、但影響最大的領域——心理偏誤和交易日誌。說實話，技術分析和基本面分析我已經學了一大堆，但回頭看，真正讓散戶虧大錢的從來不是「看錯方向」，而是「做錯決策」。看對方向但殺在最低點、看錯方向但死不認賠——這些都是心理偏誤的傑作。

幾個深刻體悟：

1. 損失厭惡是所有偏誤的根源：處置效應、沉沒成本、錨定效應，這三個表面上看起來不同的病症，底層都是損失厭惡——虧錢的痛苦是賺錢快樂的2倍以上。所以人會做出各種不理性的行為來「避免認賠」，結果反而虧更多。理解了這一點，就能理解為什麼「系統化交易」這麼重要——因為系統沒有損失厭惡，系統只看規則。

2. FOMO的四個歷史案例太經典了：網路泡沫、比特幣、Tesla、GameStop，劇本都一樣——社群瘋傳→散戶追高→崩跌→後進者巨虧。而且每次FOMO的時候，人都覺得「這次不一樣」，結果每次都一樣。「預先驗屍法」是我覺得最實用的工具：假設一年後虧50%，是什麼原因？如果你回答不出來，就不要進場。

3. 交易日誌的核心是「行為正確性」：以前覺得交易日誌就是記賺賠，但這只是表面。真正重要的是「我有沒有按照策略執行」。賺錢但中途改了策略=運氣好但行為錯誤，長期必虧。虧錢但完全按照策略=行為正確但結果不好，可以從策略層面改進。這個區分太重要了——「靠運氣贏來的，最後還是會憑實力輸回去」。

4. 情緒覺察四步法比壓抑情緒有效：以前學的交易心理學都說「控制情緒」「不要恐懼不要貪婪」，說得好像情緒是開關一樣。但事實上情緒是訊號——興奮告訴你可能有機會、焦慮告訴你風險可能過大、焦躁告訴你在不對的市況硬做。關鍵是覺察→解讀→評估→承諾，而不是假裝沒感覺。

5. 交易旅程框架解決了「不知道要記什麼」的問題：交易前記計畫、交易中記執行、交易後記回顧，三個階段各有重點。這比「想到什麼寫什麼」有效太多。特別是交易中記錄「是否按照計畫執行」，這是日後改進交易系統最重要的數據。

分點進出——追蹤主力行蹤的X光機

是什麼：每筆股票交易都在某個券商的某個分點發生，交易所每天傍晚公布所有分點的買賣超數據。透過追蹤特定分點的進出，就能掌握「誰在買、誰在賣」，比看法人數據更精細。

怎麼做：

1. 理解分點進出的本質：

  • 每個券商分點 = 該分公司所有客戶的合計買賣超

2. 找關鍵分點的方法：

  • 先找個股近5-10日買超最大的分點

3. 地緣券商——公司附近的分點特別重要：

  • 公司員工通常在公司附近券商開戶

4. 關鍵分點的判斷標準：

  • 買賣力 > 60% = 強力訊號（該分點當日對這檔股票的買賣超佔成交量60%以上）

實例：

  • 情境：2019年8月21日，網龍（3083）「兆豐-景美」突然買超557張
  • 追蹤：該分點進場後股價明顯上漲，高點時兆豐-景美開始出脫持股
  • 9月初兆豐-景美再度進場，後續又有一波漲幅
  • 結論：兆豐-景美是網龍的關鍵分點，未來再度大買時可以跟進

適用時機：籌碼面分析必備工具，特別是中小型股（大戶更容易在分點上留下痕跡）。不適合：大型權值股（分點太分散，難以追蹤單一大戶）

隔日沖券商辨識——知道誰在「今天買明天賣」

是什麼：隔日沖大戶用龐大資金在當天將股價拉抬甚至拉到漲停，隔天趁開高時全部倒出賺取價差。散戶如果在尾盤追高買進，隔天就被當提款機。辨識隔日沖券商是避開陷阱的第一步。

怎麼做：

1. 主要隔日沖券商名單：

本土隔日沖分點：

  • 凱基-台北、凱基-松山

外資隔日沖代表：

  • 美林（Merrill Lynch）

區域性分點：

  • XX-嘉義、XX-虎尾等（小型區域分點突然出現大量買超也要注意）

2. 隔日沖的兩種手法：

手法一：拉漲停策略

  • 當天抓強勢股，大單掃貨拉到漲停鎖死

手法二：權證策略

  • 大量買進認購權證 + 同時買進標的股票

3. 隔日沖三大派別（權證小哥分類）：

群益幫：順勢砸重金

  • 分點：海山、丹鳳、上新莊、大興

美好大：手法靈活多變

  • 來自富邦台南分點

元大自動幫：電腦監控超精準

  • 用電腦下單，監控到單筆500萬以上買單就立刻進場

4. 遇到隔日沖的應對策略：

如果你是波段投資人：

  • 隔日沖跟你無關，照自己策略走就好

如果你不小心追高了：

  • 隔天開盤4%是分水嶺：開高超過4%且量能正常 → 可以觀察；開高不到4%且成交量迅速放大 → 先獲利出場

如果你想跟單隔日沖：

  • 買相同標的股票但不同檔權證（避免隔天賣時好價位被搶走）

實例：

  • 情境：東哥遊艇（8478）2022年4月18日解除處置後摜殺跌停
  • 盤後抓到元兇：隔日沖大戶凱基-台北賣超526張、摩根大通賣超514張
  • 兩家券商佔當天總成交量40%以上
  • 4/19股價再摜殺第二根跌停至453元
  • 教訓：看到隔日沖券商大買且股價已大漲，不要追高

適用時機：每天盤後檢查個股分點進出時必查。特別是當天漲停或大漲的中小型股，必須確認是否有隔日沖介入。不適合：大型權值股（隔日沖資金量拉不動）

借券賣出餘額——外資隱形做空工具

是什麼：借券賣出是外資做空的隱形管道。跟融券不同，借券是向券商借股票來賣，不受融券券源限制，也不受平盤以下不能放空的限制。借券賣出餘額上升代表看空的人越多，下降代表空方在回補。

怎麼做：

1. 借券 vs 融券的差異：

| 比較項目 | 融券 | 借券 |
|----------|------|------|
| 借券來源 | 券商融資融券部 | 證交所借券系統 |
| 放空限制 | 平盤以下不能放空 | 無此限制 |
| 費用 | 融券利率約0.1% | 借券費率約3-7% |
| 回補期限 | 股東會前強制回補 | 無強制回補期限 |
| 使用者 | 散戶、本土法人 | 主要為外資 |
| 隱蔽性 | 融券餘額公開透明 | 借券餘額需另外查詢 |

2. 借券賣出餘額的判讀方法：

看趨勢不看絕對值：

  • 借券餘額持續增加 = 空方在加碼，看空力道增強

搭配現貨買賣超判斷：

  • 借券賣出增加 + 外資現貨賣超 = 雙重看空訊號 ⚠️⚠️

看佔股本比例：

  • 借券賣出餘額佔股本 > 3% = 高度警戒

3. 借券賣出三種情境判讀：

情境一：借券增加 + 股價下跌 = 空方勝利

  • 空方在做空且股價確實在跌，空方趨勢確立

情境二：借券增加 + 股價上漲 = 潛在軋空

  • 空方在加碼但股價不跌反漲

情境三：借券減少 + 股價止跌 = 空方撤退

  • 空方在回補，賣壓減輕

4. 查詢工具：

  • Goodinfo：借券賣出餘額排名

實例：

  • 情境：某股借券賣出餘額從2,000張增加到8,000張（佔股本3.5%），外資現貨同步賣超
  • 判讀：雙重看空訊號，空方力道強
  • 操作：不抄底，等借券餘額開始減少且股價出現反轉K線再考慮進場
  • 如果借券繼續增加到12,000張但股價開始反彈不跌 → 潛在軋空行情，空方可能撐不住

適用時機：追蹤外資動向時必查。特別是外資現貨賣超的個股，一定要看借券數據確認是否是「真看空」還是「避險」。不適合：借券餘額極少的股票（數據雜訊大）

做空的三種工具與成本比較

是什麼：做空不是只有「融券放空」一種方式，台股有三種做空工具：當沖做空、融券做空、借券做空。不同工具的成本和風險差異極大，選錯工具會讓你白做甚至虧更多。

怎麼做：

借券做空：

2. 做空的核心原則：

實例：

  • 做空方式選擇：

分點追蹤的價值在於「比三大法人更精細」。法人數據告訴你外資整體買超還是賣超，但分點數據可以告訴你「哪個分點的大戶在進出」。特別是地緣券商的概念——公司附近的券商分點突然大買，很可能是內部人提前知情在布局，這種訊號比任何技術指標都早。

隔日沖券商的辨識是散戶必備的自保技能。幾個重要認知：

1. 隔日沖不只隔天賣：越來越多人會看盤後分點數據，所以隔日沖也變聰明了，可能隔2-3天才出貨，甚至隔天繼續拉漲停。不要預設立場，發現隔日沖介入就要提高警覺。

2. 隔日沖三大派別的跟單策略不同：群益幫最容易跟（單量大、手法規律），元大自動幫最難跟（電腦下單比人快）。如果真要跟單，買同標的但不同檔權證，避免被降隱波和搶價位。

3. 隔日沖的4%分水嶺：隔天開盤如果漲不到4%且成交量迅速放大，就要先出場。這是判斷隔日沖是否在出貨的簡單有效方法。

借券賣出餘額的判讀更讓我重新理解了外資做空的方式。以前以為外資做空就是賣現股，但其實借券才是外資更隱蔽的做空管道。借券不受平盤以下放空限制、沒有強制回補期限，是外資長期做空的利器。最關鍵的判斷是「借券+現貨」要一起看——借券增加+現貨賣超=真看空，借券增加+現貨買超=可能只是避險。

最後，做空三種工具的成本比較是個很實用的知識。借券做空年費7%，對散戶來說成本高到不太划算。散戶做空應該優先用當沖（最便宜）或融券（中間），借券留給外資大戶去玩。

整體來說，這回合把籌碼面的「微觀分析」補齊了——從三大法人的宏觀視角，深入到分點追蹤的微觀視角，再到隔日沖和借券的隱蔽管道。籌碼面分析從此不再只是看法人數字，而是能追蹤到具體的大戶行蹤。

MDD（Maximum Drawdown）最大回撤——你最慘會虧多少

是什麼：MDD 指的是在特定期間內，帳戶淨值從歷史最高點跌到隨後最低點的最大跌幅百分比。它告訴你「從任意時間點進場後，最慘會遇到多大的虧損」。

怎麼做：

1. 計算回撤（Drawdown）：每次淨值創新高，就從新高點計算到隨後最低點的跌幅

  • 回撤 = (峰值淨值 - 谷底淨值) / 峰值淨值 × 100%

2. 找最大回撤：比較期間內所有回撤，取最大值即為 MDD

3. 計算恢復所需漲幅：

  • 跌10% → 需漲11%回本

實例：

  • 情境：帳戶從5萬成長到10萬（高點），跌到7萬
  • 回撤 = (10萬 - 7萬) / 10萬 = 30%
  • 之後回升到12萬（新高），又跌到9萬
  • 第二次回撤 = (12萬 - 9萬) / 12萬 = 25%
  • MDD = 30%（取最大值）

適用時機：評估任何策略或投資組合的風險承受度。MDD > 30% 就要警惕，超過50%基本是災難。

MDD 四大誤區——不要用錯MDD

是什麼：多數人對MDD有四個常見誤解，用錯了會導致過度槓桿或誤判風險。

怎麼做（避開誤區）：

1. 誤區一：用歷史MDD預估未來

  • 歷史MDD只是過去的紀錄，未來MDD可能更大

2. 誤區二：以為設停損就能改善MDD

  • MDD常是連續多筆虧損累積的，不是單筆造成

3. 誤區三：MDD只看極端值

  • 只看MDD會忽略「常態回撤」有多深

4. 誤區四：拿MDD做槓桿最佳化

  • 用MDD反推最大槓桿 = 在跟老天爺對賭

實例：

  • 情境：策略回測MDD為20%，你用5倍槓桿（理論最大虧損=100%）
  • 這代表只要MDD稍微超過歷史值就爆倉
  • 安全做法：用2~3倍槓桿，留出緩衝空間

適用時機：任何使用MDD做決策的時候——資金配置、槓桿設定、策略篩選。

回撤持續時間（Drawdown Duration）——跌多久才能爬回來

是什麼：從淨值跌破高點到重新創新高所需的時間長度。MDD告訴你跌多深，Duration告訴你跌多久。

怎麼做：

1. 計算方式：從淨值開始回落的那天，到淨值恢復並創新高的那天，中間的交易日數
2. 搭配MDD使用：

  • 淺回撤 + 長持續 → 策略在陰跌，慢性失血

3. Calmar Ratio（年化報酬/MDD）：

  • Calmar > 1.0 → 每承受1%最大回撤，獲得>1%年化報酬，不錯

實例：

  • 情境：策略年化報酬15%，MDD為25%
  • Calmar = 15% / 25% = 0.6
  • 另一個策略年化報酬12%，MDD為10%
  • Calmar = 12% / 10% = 1.2
  • 雖然報酬較低，但第二個策略的風險效率更好

適用時機：比較策略好壞時，不要只看報酬率，也要看Calmar Ratio和回撤持續時間。

追蹤回撤（Trailing Drawdown）vs 固定回撤——Prop Firm的風控邏輯

是什麼：追蹤回撤的底線會隨帳戶盈利新高而上移；固定回撤的底線始終鎖定初始資金。

怎麼做：

1. 固定回撤：

  • 底線 = 初始資金 × (1 - MDD限制)

2. 追蹤回撤：

  • 底線 = 當前最高淨值 × (1 - MDD限制)

3. 實戰應用原則：

  • 用追蹤回撤邏輯管理自己的帳戶：每創新高就重設停損線

實例：

  • 情境：100萬本金，設定追蹤MDD 15%
  • 帳戶成長到130萬（新高），底線 = 130萬 × 85% = 110.5萬
  • 如果跌破110.5萬，強制清倉檢討
  • 即使從130萬回跌到110.5萬，仍保留了10.5萬的獲利

適用時機：任何有紀律的交易者都該用追蹤回撤邏輯管理自己的風險底線，不一定要參加Prop Firm才用。

策略壓力測試——你的策略撐得住嗎

是什麼：在回測基礎上，主動模擬最壞情境，驗證策略在極端條件下是否還能存活。

怎麼做：

1. 成本壓力測試：

  • 在回測手續費之外，額外加0.1%的滑價成本

2. MDD壓力測試：

  • 將歷史MDD乘以1.5~2倍，看帳戶是否存活

3. 多情境壓力測試：

  • 模擬2008金融海嘯、2020疫情崩盤、2022升息空頭

4. 連虧壓力測試：

  • 用蒙地卡羅模擬或歷史數據，找出策略最長連虧序列

實例：

  • 情境：某波段策略回測年化20%、MDD 18%、勝率55%
  • 成本壓力測試：加0.1%滑價後年化降到14%、MDD升到22%
  • MDD壓力測試：2倍MDD = 44%，帳戶從100萬跌到56萬
  • 連虧壓力測試：最長連虧7筆 × 每筆虧3% = 21%
  • 結論：策略可接受，但建議單筆風險壓到2%以內

適用時機：每個策略上線前、定期回顧時、市況出現重大變化時。

高Sharpe + 適度槓桿 > 高報酬低Sharpe

是什麼：一個Sharpe比率高（風險效率好）但絕對報酬低的策略，搭配適度槓桿，往往比一個高報酬但Sharpe低的策略表現更好。

怎麼做：

1. 核心邏輯：

  • Sharpe = 每單位風險的超額報酬

2. 實務操作：

  • 找Sharpe > 1.5的策略

3. 注意事項：

  • 槓桿有融資成本，會吃掉部分獲利

實例：

  • 策略A：年化8%、Sharpe 2.0、MDD 5%
  • 策略B：年化20%、Sharpe 0.8、MDD 35%
  • 策略A加2倍槓桿：年化16%、Sharpe 2.0（理論值）、MDD 10%
  • 結果：加槓桿的A比B報酬低一點，但風險只有B的三分之一

適用時機：有穩定高Sharpe策略時，用槓桿放大報酬比追求高報酬低Sharpe策略更安全。不適合沒有回測數據或Sharpe < 1的策略。

心得

MDD這東西我之前只停留在「知道它是什麼」的層面，今天深挖才發現用錯MDD的人太多了——包括以前的我。最關鍵的領悟是：MDD不是用來算最佳槓桿的，是用來算安全邊際的。 你拿歷史MDD反推能開多少倍槓桿，本質上就是在賭「未來不會比過去更慘」——這在市場裡就是找死。

另一個收穫是追蹤回撤的概念。以前我只會設固定停損線，但追蹤回撤更符合實際需求：賺了錢之後，停損線要跟著上移，不然等於讓獲利裸奔。這個邏輯不只Prop Firm在用，個人交易者也該用。

最後，高Sharpe + 槓桿 > 高報酬低Sharpe 這個觀念，跟之前學的Half Kelly完全打通了——先確保策略的風險效率好，再用科學化的槓桿放大，而不是追求高報酬然後被大回撤嚇出場。整條邏輯線串起來了。

Covered Call 掩護性買權——用股票收租的被動收入策略

是什麼：在持有現貨（股票或ETF）的基礎上，賣出相同標的的買權（Sell Call），收取權利金增加額外收益。本質是「犧牲大漲時的獲利潛力，換取權利金收入」。

怎麼做：

1. 建倉步驟：

  • 持有現貨部位（如0050 ETF或台積電）→ 賣出略為價外的Call → 收取權利金

2. 四種到期情境判讀：

  • 大漲：有獲利但少賺，短時間暴漲時權利金無法彌補少賺的幅度 ⚠️

3. 履約價與到期日選擇：

  • 履約價：選略為價外（OTM），能收取相對較高的時間價值

4. 適合的標的特性：

  • 股價長期持平或向上

實例：

  • 你持有0050 ETF成本100元，賣出履約價105元的Call，收取權利金1.5元
  • 到期時0050在102元：Call不會被執行，你賺2元價差+1.5元權利金=3.5元
  • 到期時0050在108元：Call被執行，你以105元元賣出，賺5元價差+1.5元權利金=6.5元（但單純持有可賺8元，少賺1.5元）
  • 到期時0050在95元：Call不會被執行，你虧5元-1.5元權利金=淨虧3.5元（比單純持有少虧1.5元）

適用時機：

  • ✅ 預期短期不會大漲（盤整或緩漲）
  • ✅ 長期持有現貨部位，想額外增加收益
  • ✅ 波動率偏高時（權利金收更多）
  • ❌ 預期短期大漲（會少賺很多）
  • ❌ 不打算長期持有的現貨（不如直接做Short Put）

Iron Condor 鐵兀鷹——不猜漲跌的穩定收金機

是什麼：鐵兀鷹是由上方一組Call Credit Spread（買權空頭價差）和下方一組Put Credit Spread（賣權多頭價差）組成的中性策略。只要結算時指數落在兩組價差之間的區間，就能全收權利金。風險有限、獲利也有限，是賣方策略中最穩健的選擇之一。

怎麼做：

1. 建倉組成（以台指期16000點為例）：

  • 上方：賣出16300 Call收35點 + 買入16400 Call付15點 = 淨收20點

2. 盈虧結構：

  • 結算在15700~16300之間：全收40點權利金（最大獲利）

3. 三個實戰秘訣：

秘訣一：Delta中立 vs 跨距對稱

  • 最簡單做法：上下跨距對稱（如距現貨各300點）

秘訣二：部位快被貫穿時的補救

  • 方法A：小台避險——快被貫穿時建同方向小台，對沖選擇權虧損

秘訣三：選擇合適的區間寬度

  • 寬區間（如上下各500點）：勝率高但權利金少

實例：

  • 情境：台指期在20000點，預期一個月內震盪區間19500~20500
  • 建建倉：賣出20500/20600 Call Spread + 賣出19500/19400 Put Spread
  • 收取權利金共80點
  • 結算在19500~20500：賺80點×50元=4,000元
  • 結算超過20600或低於19400：最大虧損=(100-80)×50=1,000元
  • 如果大盤漲到20480快要貫穿：平倉下方Put Spread獲利30點，Rolling到20500建新Put Spread收25點，最大虧損降到(100-30-25-25)=20點×50=1,000元→500元

適用時機：

  • ✅ 預期盤整震盪（最適合）
  • ✅ 支撐壓力明確的區間
  • ✅ IV偏高時建倉（權利金收更多）
  • ❌ 預期大趨勢行情
  • ❌ 重大事件前（FOMC、選舉，波動會打破區間）

Iron Butterfly 鐵蝴蝶——壓縮區間、高權利金的中性策略

是什麼：鐵蝴蝶是鐵兀鷹的「極端版」——上下兩組價差的賣方履約價都設在現價（At-the-Money），形成「中間胖、兩邊窄」的蝴蝶形損益圖。最大獲利比鐵兀鷹高很多，但獲利區間極窄，勝率較低。

怎麼做：

1. 建倉組成（以台指期16000點為例）：

  • 賣出16000 Call + 賣出16000 Put（兩口都價平，權利金最高）

2. 與鐵兀鷹的關鍵差異：

  • 鐵兀鷹：獲利區間寬（600點），勝率高（70-85%），但最大獲利低

3. 盈虧結構：

  • 最大獲利：結算價正好=中心履約價，所有賣方部位到期無價值，只扣買方成本

4. 實戰轉換技巧：

  • 鐵兀鷹被貫穿時，可以Rolling獲利邊到現價，變成鐵蝴蝶

5. 使用時機：

  • 市場波動性低，預期價格幾乎不動

實例：

  • 情境：台指期20000點，你預期一週內不會大漲大跌
  • 建倉：賣出20000 Call + Put，買入20100 Call + 19900 Put
  • 淨收權利金150點（比鐵兀鷹高很多）
  • 結算在19900~20100：全收150點×50=7,500元
  • 結算正好20000：最大獲利
  • 結算超過20100或低於19900：最大虧損=(100-150)=-50點，等於你反而倒賺（因為權利金收太多）

適用時機：

  • ✅ 極度盤整、幾乎不動的市況
  • ✅ 事件後IV回落期
  • ✅ 短期到期（1-2週）
  • ❌ 任何有方向性的市場
  • ❌ 重大事件前後

是什麼：Covered Call、Iron Condor、Iron Butterfly三種策略都適合盤整市，但風險收益結構完全不同，必須根據市況和持倉選擇。

2. 情境選擇指南：

情境一：長期持有0050，預期短期盤整
→ 用Covered Call，每月賣價外Call收權利金，如用股票收租

情境二：沒有現貨，預期指數區間震盪
→ 用Iron Condor，設寬區間高勝率，穩穩收權利金

情境三：事件剛結束，IV正在回落，預期超窄幅盤整
→ 用Iron Butterfly，收高額權利金，但要注意Gamma風險

情境四：Iron Condor快要被貫穿
→ Rolling獲利邊，可轉成Iron Butterfly降低最大虧損

3. 共同風險管理原則：

  • 三種策略都怕「短期大漲」，建倉前要確認沒有重大事件在到期前

心得

這回合把選擇權的「中性策略三兄弟」一次搞清楚了。之前學Gamma Scalping是靠波動賺錢的買方策略，今天學的三種都是靠「時間流逝」賺錢的賣方策略——本質上是「當保險公司」，收取權利金，只要不出大事就穩穩賺。

幾個重要認知更新：

1. Covered Call不是選擇權策略，是現貨管理策略：它的核心前提是「你本來就要長期持有現貨」。在這個前提下，賣Call只是在額外收租。如果你不想持有現貨，那Covered Call跟Short Put完全一樣，不如直接做Short Put更簡單。這個區分很重要——很多人搞不清楚自己到底是在做現貨管理還是在做選擇權投機。

2. Iron Condor的Rolling技巧太實用了：被貫穿不是世界末日，把獲利邊平倉鎖住利潤，然後在更遠處或現價附近重建新部位，可以有效降低最大虧損。這比死抱著等結算好太多了。而且Rolling還可以轉成Iron Butterfly——如果大盤真的回來了，反而會大賺。

3. Iron Butterfly是高報酬高風險版本：同樣是中性策略，鐵蝴蝶收的權利金比鐵兀鷹高很多，但獲利區間極窄。這就像「保險公司收到很高的保費，但理賠條件很寬鬆」——平時很爽，出事時虧很大。不適合新手，但老手可以在特定時機（事件後IV回落、極度盤整）使用。

4. 三種策略的共同弱點是「短期大漲」：Covered Call怕大漲少賺，Iron Condor和Iron Butterfly怕大漲貫穿。所以建倉前一定要檢查到期前有沒有重大事件（FOMC、財報季、選舉），有就不要做。這跟之前學的VIX四象限是同一個邏輯——VIX高位時建倉收權利金，VIX低位時不要做（權利金太少）。

5. 鐵兀鷹的Delta中立比跨距對稱更重要：當市場偏多時，同跨距的Put Delta比Call小，如果用跨距對稱建倉，整體部位其實偏空。用Delta對稱建倉才能真正中立。這個細節很多人忽略，但長期下來會影響勝率。

整體來說，這三種中性策略搭配之前學的Gamma Scalping（買方策略），形成了完整的選擇權策略框架：盤整時用賣方策略收時間價值，波動大時用買方策略靠Gamma賺錢。選擇權不再是「只會做買方」或「只會做賣方」，而是根據市況靈活切換。

布林通道（Bollinger Bands）——用統計學框住股價的波動範圍

是什麼：布林通道是由 John Bollinger 發明的技術指標，結合移動平均線和統計學的標準差概念，用三條軌道線把股價的波動範圍「框」起來。通道涵蓋95.4%的K線資料，讓你一眼看出股價目前是在正常範圍內波動，還是已經偏離到極端位置。

怎麼做：

1. 三條軌道線構成：

  • 通道中軌 = 20MA（20日移動平均線，代表平均成本）

2. 五種基本訊號判讀：

多頭訊號（3種）：

  • K線自下而上穿越下軌線 → 股價觸及統計極端低位，買進訊號

空頭訊號（2種）：

  • K線自上而下摜破中軌線 → 股價跌破平均成本，賣出訊號

3. 八種通道型態判讀：

同向型態（三軌同方向移動）：

  • 三軌向上移動：多頭力量穩健，盤面穩定上漲，回檔較淺

異向型態（三軌不同方向移動）：

  • 三軌張口：短期情緒小爆發，上漲或下跌趨勢打開的訊號

4. 實戰操作策略：

策略一：通道內低買高賣（震盪盤）：

  • 股價觸及下軌 → 觀察是否出現反轉K線（鎚子線、十字線）→ 買進

策略二：帶寬收縮後的突破（Squeeze）：

  • 帶寬收縮到極窄 → 波動率極低 → 像彈簧被壓到極限

策略三：通道中軌當多空分界線（趨勢盤）：

  • 股價站上中軌 → 偏多操作

實例：

  • 情境：某股20MA=100元，標準差=3元
  • 上軌=106元，中軌=100元，下軌=94元
  • 股價從95元觸及下軌94元附近，出現鎚子線 → 買進訊號
  • 買進價95元，停損91元（跌破下軌3元）
  • 股價反彈穿越中軌100元 → 加碼買進
  • 股價觸及上軌106元附近，出現上影線 → 減碼或出場
  • 反例：股價從108元跌回跌破中軌100元 → 多頭失守，賣出

適用時機：

  • ✅ 震盪盤中低買高賣（通道帶寬適中、三軌水準運行）
  • ✅ 帶寬極窄後的突破行情（Squeeze策略）
  • ✅ 判斷股價是否偏離正常範圍
  • ❌ 跳空大漲/大跌的行情（帶寬突然放大，通道失去參考價值）
  • ❌ 單邊趨勢行情中逆勢操作（觸及上軌就做空、觸及下軌就做多，在趨勢盤中會被輾過去）

布林通道的缺點與限制：

  • 使用均線週期過短時，帶寬變化劇烈，訊號不穩定
  • 跳空開盤會造成帶寬突然放大，通道暫時失效
  • John Bollinger 本人建議搭配其他技術指標使用，不要單獨依賴布林通道做決策
  • 布林通道是「事後確認」工具，不是「事前預測」工具——張口型喇叭口出現時，行情往往已走了一大半

斐波那契回撤（Fibonacci Retracement）——用黃金比例找支撐壓力

是什麼：斐波那契回撤是基於斐波那契數列比例關係的技術分析工具。當價格經過一輪上漲或下跌後，回撤（反方向修正）的幅度往往落在特定的黃金比例位置（38.2%、50%、61.8%），這些位置就是天然的支撐和阻力。跟其他技術指標不同，斐波那契回撤具有前瞻性——可以在行情發生前就畫好線，提前部署。

怎麼做：

1. 五個關鍵比例：

  • 23.6%：淺回調，強趨勢中的小修正

2. 畫線方法：

上漲趨勢中找支撐：

  • 找出這波上漲的起點（最低價）和終點（最高價）

下跌趨勢中找阻力：

  • 找出這波下跌的起點（最高價）和終點（最低價）

3. 回撤公式：

  • 回撤水平 = 最高價 -（最高價 - 最低價）× 回撤比率

4. 實戰交易策略：

策略一：回撤支撐買進法（多頭）：

  • 前提：趨勢明確向上

策略二：回撤阻力賣出法（空頭）：

  • 前提：趨勢明確向下

策略三：斐波那契擴展（Fibonacci Extension）找目標價：

  • 回撤找進場點，擴展找出場點

5. 關鍵比例的實戰可靠性排序：

  • 38.2%：第一次回調最容易在這裡找到支撐/阻力，可靠性最高

6. 使用注意事項：

  • 修正幅度超過23.6%時，斐波那契回撤才真正發揮效用

實例：

  • 情境：某股從80元漲到120元後開始回調
  • 38.2%回撤位：120 - (120-80) × 0.382 = 104.72元
  • 50%回撤位：120 - (120-80) × 0.5 = 100元
  • 61.8%回撤位：120 - (120-80) × 0.618 = 95.28元
  • 操作1：股價回到105元附近，出現鎚子線+量縮 → 試單買進，停損94元
  • 操作2：如果105元沒撐住，回到100元附近 → 加碼買進，停損94元
  • 操作3：如果跌破95.28元 → 趨勢可能反轉，停損出場
  • 擴展目標價：127.2%擴展 = 80 + (120-80) × 1.272 = 130.88元

適用時機：

  • ✅ 趨勢明確的回調找進場點（跟之前學的支撐壓力線概念互補）
  • ✅ 搭配布林通道：回撤位恰好在布林通道下軌附近 → 雙重支撐確認
  • ✅ 設定停損停利的參考點位
  • ❌ 盤整行情中沒有明確趨勢時（起點終點難認定）
  • ❌ 單獨使用不搭配其他指標（勝率不夠高）

布林通道 + 斐波那契回撤的雙重確認策略

是什麼：單一指標的勝率有限，把布林通道和斐波那契回撤結合使用，當兩者在同一個價位區間發出訊號時，勝率大幅提升。

怎麼做：

1. 雙重支撐買點：

  • 先用斐波那契回撤畫出關鍵回調位（38.2%、50%、61.8%）

2. 雙重壓力賣點：

  • 先用斐波那契回撤畫出反彈阻力位

3. 帶寬收縮 + 斐波那契回撤：

  • 布林通道帶寬極窄（Squeeze）+ 價格在斐波那契61.8%回撤位附近

實例：

  • 情境：某股從80元漲到120元後回調，布林通道中軌=100元、下軌=95元
  • 斐波那契61.8%回撤位=95.28元，布林通道下軌=95元
  • 兩者幾乎重合 → 95元附近是超強支撐區
  • 股價回到96元，出現鎚子線 + 量縮 → 雙重確認買進
  • 停損：跌破93元（布林通道下軌下方2元）
  • 目標：布林通道上軌105元或斐波那契127.2%擴展位131元

適用時機：所有需要精確找進出場點位的交易。雙重確認的勝率比單一指標高出很多。

心得

布林通道和斐波那契回撤，一個是用統計學框住價格範圍，一個是用黃金比例找關鍵位置，兩者本質不同但互補性極強。

布林通道最實用的不是「碰到上下軌就反向操作」，而是「帶寬收縮後的突破」（Squeeze）。帶寬極窄代表市場正在醞釀方向選擇，一旦帶寬放大就是行情啟動的訊號。這跟之前學的均線糾結突破是同一個邏同一個邏輯——盤整越久、波動越小，突破後力道越大。布林通道只是用統計學的方式把這個概念量化了。

斐波那契回撤最令人驚奇的是它的「前瞻性」。大多數技術指標都是落後的（均線、MACD都是看過去的數據），但斐波那契回撤可以在行情還沒發生之前就畫好支撐阻力線。38.2%是第一道防線、61.8%是最後防線，這個框架給了交易者一個明確的「在哪等、在哪砍」的決策依據。

但兩個工具都有致命的缺點：布林通道在跳空行情中失效，斐波那契回撤的起點終點選擇有主觀性。所以作者和實戰者都強調「不要單獨使用」——這跟之前學的「多重背離共振」和「均線糾結三道過濾」是同一個核心觀念：單一訊號勝率不夠，多重確認才能提高勝率。

雙重確認策略是今天最大的收穫：當布林通道下軌和斐波那契61.8%回撤位重合時，那個價位就是超強支撐。這種「不同分析框架得出同一個結論」的現象，在技術分析中是最可靠的訊號之一。以後找進場點，我會先畫斐波那契回撤線，再套上布林通道，找兩者重合的區間——這比單看任何一個指標都精準。

OBV能量潮指標——把成交量變成趨勢訊號的X光機

是什麼：OBV（On Balance Volume，能量潮指標）由喬·葛蘭維（Joe Granville）在1960年代發明，是一種把「成交量」累積成趨勢線的技術指標。它的核心假設是「量先價行」——資金的進出會領先價格變動，OBV就是把資金流向視覺化的工具。

怎麼做：

1. 計算方式超簡單：

  • 今日收盤 > 昨日收盤 → 今日OBV = 前日OBV + 今日成交量

2. 三個核心假設：

  • 市場看法分歧：當投資人對股價看法越不一致，買賣越熱絡，成交量越大；看法一致時成交量萎縮

3. 四種訊號判讀：

背離訊號（最核心用法）：

  • 價創新高但OBV沒創新高 → 頂背離，上漲缺乏資金支持，隨時可能下跌

同步訊號：

  • OBV緩慢上升 + 股價同步上漲 → 資金穩定流入，多頭趨勢健康

急速變化訊號：

  • OBV急速上升 → 買盤大量湧入，但主力不會幫大家抬轎，可能即將反轉

累積訊號：

  • OBV長期累積的高點 → 壓力區，突破後漲勢更強

實例：

  • 情境：某股從100元漲到130元創新高，但OBV線沒有創新高（量能沒跟上）
  • 判讀：頂背離，上漲缺乏資金支持
  • 操作：減碼或設移動停利，等OBV確認新高再加碼
  • 反例：某股從100元跌到70元創新低，但OBV沒有創新低（低檔有買盤承接）
  • 判讀：底背離，可能即將反彈
  • 操作：觀察是否出現反轉K線（鎚子線、十字線），確認後試單做多

適用時機：

  • ✅ 波段操作確認趨勢方向
  • ✅ 判斷突破是否有效（OBV同步創高 = 真突破）
  • ✅ 背離預警趨勢反轉
  • ❌ 盤整震盪行情（OBV雜訊多，容易誤判）
  • ❌ 成交量極小的冷門股（OBV數據失真）
  • ❌ 超短線當沖（OBV是累積指標，反應較慢）

OBV進階實戰——搭配均線與型態的雙重確認

是什麼：OBV單獨使用勝率不夠高，必須搭配其他技術指標做雙重確認。最實用的三種搭配方式：OBV+均線、OBV+K線型態、OBV+支撐壓力。

怎麼做：

1. OBV + 均線（20MA或60MA）：

  • OBV從底部上升 + 股價站上均線 → 多方啟動，偏多操作

2. OBV + K線型態確認突破：

  • 股價帶量突破壓力線 + OBV同步創新高 → 真突破，勝率極高

3. OBV + 支撐壓力區間：

  • OBV也會形成支撐和壓力，跟股價一樣可以畫趨勢線

實例：

  • 情境：某股在100-110元區間盤整2個月，某天帶量突破110元壓力
  • 檢查1：股價突破110元 ✅
  • 檢查2：OBV同步創新高 ✅ → 真突破機率極高
  • 進場：111元買進，停損108元（支撐區下方）
  • 反例：同樣突破110元，但OBV沒有創新高 → 假突破風險高，不進場或小部位試探

適用時機：所有需要確認突破真假的交易。特別是均線糾結突破、支撐壓力突破、型態突破時，用OBV確認量能是否跟上。

OBV的五大盲點與避坑指南

是什麼：OBV雖然好用，但有五個常見的陷阱，不知道的話會被騙。

怎麼做：

1. 假突破誤導：

  • OBV一口氣突破前高，但只是短線大單衝進來，很快就消失

2. 無法區分買賣結構：

  • OBV只看「漲就加、跌就減」，不區分大單小單、主力散戶
