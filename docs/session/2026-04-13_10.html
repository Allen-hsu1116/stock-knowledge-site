核心知識

一、和諧型態是什麼？

是什麼：和諧型態（Harmonic Patterns）是結合幾何價格結構與斐波那契比例的進階技術分析工具。透過精確的斐波那契回撤和擴展比例，辨識市場中重複出現的幾何型態，預測價格反轉點。每一種型態都有嚴格的斐波那契比例規則，型態不完美就不交易。

歷史淵源：

  • 1930 年代：H.M. Gartley 在《Profits in the Stock Market》首次提出 Gartley 222 型態
  • 1990 年代：Larry Pesavento 加入斐波那契比例，出版《Fibonacci Ratios and Pattern Recognition》
  • 2000 年代：Scott Carney 出版《Harmonic Trading》，創造 Bat、Crab、Shark 等型態

五個關鍵點位：

  • X：型態起點
  • A：第一個價格端點
  • B：XA 的回撤點
  • C：AB 的回撤點
  • D：型態完成點 = 潛在反轉區（PRZ, Potential Reversal Zone）

核心原則：
1. 每一條腿的斐波那契比例必須符合特定範圍
2. 型態結構必須對稱且乾淨
3. D 點是行動區——等待反轉確認再進場
4. 不完美的型態直接跳過，寧可錯過不可做錯

適用市場：所有市場（外匯、股票、商品、加密貨幣），所有時間框架（高時間框架更可靠）

二、ABCD 基礎型態

是什麼：所有和諧型態的基礎，由四個點位 ABCD 組成。AB 和 CD 是同方向的推動浪，BC 是回撤浪。視覺上像字母「N」或閃電形。

斐波那契規則：

  • BC 回撤 AB 的 0.382～0.886（理想值 0.618）
  • CD 擴展 BC 的 1.13～2.618
  • AB ≈ CD（對稱原則）

怎麼做：
1. 找到 ABC 三個點：AB 是趨勢脈衝，BC 是回撤
2. 量測 BC 回撤比例（例如落在 0.618）
3. 從 C 點開始，預期 D 點出現在 BC 的 1.618 擴展位
4. 同時用 AB = CD 對稱原則交叉確認 D 點
5. 在 D 點附近等反轉 K 線確認後進場

三、Gartley 加特利型態

斐波那契規則：

  • AB 回撤 XA：必須在 0.618（最嚴格的條件）
  • BC 回撤 AB：0.382～0.886
  • CD 擴展 AB：1.13～1.618
  • D 點回撤 XA：0.786（Gartley 的標誌性比例）

關鍵：D 點不超過 X 點——這是區別 Gartley 和 Butterfly 的關鍵。

四、Butterfly 蝴蝶型態

與 Gartley 的關鍵差異：D 點超過 X 點。意味著市場在完成型態前走出了比前一波更遠的距離，代表更深的超賣/超買，反轉力道更強。

斐波那契規則：

  • AB 回撤 XA：0.786（Butterfly 的標誌性比例）
  • D 點擴展 XA：1.27（D 超過 X 點！）

五、Bat 蝙蝠型態

斐波那契規則：

  • AB 回撤 XA：0.382～0.5（比 Gartley 的 0.618 更淺）
  • D 點回撤 XA：0.886（Bat 的標誌性比例，比 Gartley 的 0.786 更深）

六、Crab 螃蟹型態

斐波那契規則：

  • AB 回撤 XA：0.382～0.618
  • CD 擴展 AB：2.618～3.618（最極端的擴展）
  • D 點擴展 XA：1.618（Crab 的標誌性比例）

七、和諧型態的交易策略五步驟

1. 型態辨識：先找到 XA 推動浪，量測 AB 回撤 XA 的比例判斷型態
2. PCZ 確認：多個斐波那契比例重疊的區域更可靠
3. 反轉確認：價格到達 PCZ 後，等反轉 K 線確認（錘子線、晨星、吞噬型態）
4. 進場與停損：確認後進場，停損設在 D 點稍遠處或 X 點下方
5. 目標設定：AD 的 0.382、0.618 回撤，或 A 點/C 點

八、和諧型態的優缺點

優點：
1. 提供明確的進場點、停損點、目標價
2. 適用所有市場和時間框架
3. 基於數學比例，減少主觀判斷
4. 盈虧比極佳（特別是 Crab 和 Butterfly）

缺點：
1. 型態辨識困難——需要大量練習
2. 主觀選點問題——不同人可能畫出不同型態
3. 型態可能中途失敗
4. 訊號稀少——完美型態不常出現
5. 不容易程式化——自動辨識困難

常見錯誤：
1. 硬湊型態
2. 比例不符仍進場
3. 不等確認就進場
4. 忽略大趨勢

心得

和諧型態是之前學的斐波那契回撤的「究極進化版」。把多段趨勢的斐波那契比例結合起來，形成一個完整的幾何結構，從「看一條線」進化到「看一幅地圖」。

最關鍵的體悟：AB 回撤 XA 的比例決定了是哪種型態。0.618 = Gartley，0.786 = Butterfly，0.382～0.5 = Bat。你不需要記住所有比例，只要先量 AB 回撤 XA，就能快速分類。

四種型態的 D 點位置比較非常重要：Gartley 的 D 在 XA 的 0.786（最淺），Bat 的 D 在 0.886（更深），Butterfly 的 D 在 1.27（超過 X），Crab 的 D 在 1.618（遠超 X）。同樣是看漲反轉，Crab 的進場點比 Gartley 低很多——但正因如此，Crab 的盈虧比是四者中最好的。

但和諧型態最大的陷阱就是「硬湊」。人腦天生喜歡找規律，看到 M 或 W 就想塞進某種型態。解法很簡單：嚴格遵守斐波那契比例，差一點就不算。寧可少做幾筆完美型態，也不要做一堆「差不多」的型態——「差不多」在市場裡就是「差很多」。

最後，和諧型態跟之前學的波浪理論有異曲同工之妙。波浪理論看的是趨勢的結構（推動波 + 調整波），和諧型態看的是趨勢的幾何比例（XA + AB + BC + CD）。兩者結合使用：波浪理論判斷「現在在哪一波」，和諧型態判斷「反轉點在哪裡」，互相補充，勝率更高。
