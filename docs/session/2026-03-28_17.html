一、通膨對投資的影響

什麼是通膨？

定義：整體物價持續上升，導致貨幣購買力下降。

現實案例：

  • 過去100元買焢肉便當，現在可能要120元
  • 台灣2024年CPI年增2.2%，食物類上漲3.7%

通膨原因：
1. 消費需求增加 — 經濟繁榮、薪資增長，消費力提升
2. 生產成本上升 — 原物料、運輸、人工成本上漲
3. 貨幣發行過多 — 央行過度發行，錢變多但東西沒變多

實質報酬計算

公式：實質報酬 = 名目報酬 - 通膨率

實例：

  • 投資報酬率5%，通膨率2%
  • 實質報酬 = 5% - 2% = 3%

台股實例：

  • 2004-2024年台灣加權指數：從6,319點漲到23,035點
  • 平均年漲幅6.83%（名目報酬）
  • 年均通膨1.36%
  • 實質報酬約5.47%

通膨對資產的影響

政策利率：

  • 通膨過高 → 央行升息 → 抑制消費投資
  • 通膨過低 → 央行降息 → 刺激經濟

債券：

  • 升息 → 新債券利率更高 → 舊債券價格下跌
  • 降息 → 債券價格上漲

股票：

  • 通膨提高企業借貸成本，壓縮獲利
  • 具成本轉嫁能力的企業（強勢股）股價仍可上漲
  • 高通膨環境要避開高負債公司

二、央行升息降息對股市的影響

升息是什麼？

定義：央行提高基準利率，讓借錢變貴、存錢變划算。

影響層面：

  • 貸款利率上升 → 房貸、信貸負擔增加
  • 存款利率上升 → 放在銀行更划算
  • 投資意願下降 → 股市資金流出

降息是什麼？

定義：央行調降基準利率，讓借錢變便宜、存錢變不划算。

一碼是多少：

  • 一碼 = 0.25%（25個基點）
  • 台灣央行最小單位是半碼 = 0.125%

影響層面：

  • 貸款利率下降 → 拿錢投資更划算
  • 存款利率下降 → 把錢拿出來消費投資
  • 投資意願上升 → 股市資金流入

升息 vs 降息對照表

項目升息降息
央行目的抑制通膨刺激經濟
借貸成本上升下降
存款利率上升下降
股市利空（資金流出）利多（資金流入）
債券價格下跌價格上漲
房市抑制熱度刺激買氣
匯率貨幣升值貨幣貶值

升息降息對股市的具體影響

為什麼降息推升股市：
1. 借貸成本下降 → 可運用資金增加
2. 機會成本降低 → 存款利率低，投資股市更划算
3. 要求報酬率下降 → 原本要7%才投資，現在5%就願意

但不是絕對：

  • 升息時如果經濟景氣好，股市仍可能上漲
  • 降息時如果經濟衰退，股市仍可能下跌
  • 還要看當時的市場景氣、匯率等因素

美國聯準會（Fed）決策關鍵

FOMC會議：每年8次例行會議討論利率決策

降息時機判斷：
1. 通膨年增率穩定朝2%目標前進
2. 經濟成長趨緩或進入衰退期
3. 失業率上升

2024年Fed利率動態：

  • 9月：降息兩碼（4.75%-5%）
  • 11月：降息一碼（4.50%-4.75%）

三、總體經濟指標與股市關聯

GDP與股市

定義：國內生產毛額，衡量一國經濟活動總值。

關聯性：

  • GDP成長 = 經濟基本面好 = 股市可能上漲
  • GDP衰退 = 經濟基本面差 = 股市可能下跌

注意：

  • 股市是「領先指標」，通常領先GDP 3-6個月
  • GDP公布有時間落差，實際操作要參考其他指標

PMI（採購經理人指數）

定義：製造業景氣的領先指標。

判讀方式：

  • PMI > 50：景氣擴張（利多）
  • PMI < 50：景氣收縮（利空）

子指標：

  • 新訂單
  • 生產量
  • 就業
  • 存貨
  • 供應商交貨
  • 六個月商業展望（重要領先指標）

CPI（消費者物價指數）

定義：衡量通膨的主要指標。

判讀：

  • CPI年增率 > 3%：高通膨，央行可能升息
  • CPI年增率 < 1%：通縮風險，央行可能降息
  • 目標區間：2%左右

美國重要經濟數據

對台股影響大的數據：

  • 非農就業人數（每月第一週五）
  • CPI公布（每月中旬）
  • PCE物價指數（Fed偏好指標）
  • Fed利率決議（FOMC會議）

四、投資人實戰應對策略

高通膨環境選股

避開：

  • 高負債公司（借貸成本大增）
  • 毛利低的企業（難轉嫁成本）
  • 現金流差的企業

優先：

  • 具成本轉嫁能力的龍頭股
  • 高毛利率企業
  • 擁有定價權的品牌公司

升息週期策略

資產配置：

  • 提高現金、短債比重
  • 減少高估值成長股（對利率敏感）
  • 關注金融股（利差擴大受惠）

選股方向：

  • 低負債、高現流的企業
  • 受惠升息的金融業
  • 必需消費品（較不受影響）

降息週期策略

資產配置：

  • 增加股票、債券比重
  • 減少現金部位（利率低）
  • 可考慮REITs（不動產投資信託）

選股方向：

  • 成長股（對利率敏感，降息受惠）
  • 高負債但成長性強的企業（借貸成本下降）
  • 房地產相關類股

總經觀察要點

每週追蹤：
1. 美國初領失業金人數
2. Fed官員談話
3. 原油、大宗商品價格

每月追蹤：
1. 美國非農就業數據
2. 美國CPI、PCE
3. 台灣外銷訂單、工業生產

每季追蹤：
1. GDP成長率
2. 企業財報季
3. 央行利率會議

心得

這週終於補上了一個大缺口——總體經濟對股市的影響。之前學了一堆技術分析、籌碼面、風險管理，但卻忽略了最根本的：央行政策和經濟數據如何影響股市。

幾個關鍵認知更新：
1. 升息降息不是絕對利空利多，要看當時經濟景氣。升息時如果經濟旺，股市照樣漲。
2. 實質報酬才是真的。投報率5%但通膨2%，實際只賺3%，這觀念要深植腦海。
3. 總經數據是領先指標——股市領先GDP，PMI領先股市。不要等GDP公布才反應，要看PMI和訂單數據。

核心知識

一、選股四大方法

基本面選股

是什麼：從公司財報找體質優良、財務健全、成本控管好的公司

怎麼做：
1. 看三大報表：資產負債表、損益表、現金流量表
2. 計算三率：毛利率、營業利益率、淨利率
3. 比較YoY（年增率）、QoQ（季增率）

實例：

  • 情境：台積電毛利率50%、營業利益率40%，YoY成長15%
  • 判斷：成本低、獲利穩、成長動能強

適用時機：長期投資、價值型投資人

技術面選股

是什麼：用技術指標找進出場時機

怎麼做：
1. 學習指標：MA均線、K線、MACD、KD、RSI、布林通道
2. 反覆練習看圖、模擬操作
3. 用指標篩選標的和進場時機

實例：

  • 情境：股價突破20日均線，KD黃金交叉
  • 操作：進場買進，停損設均線下方5%

適用時機：短線交易、波段操作

籌碼面選股

是什麼：追蹤法人、大戶的買賣動態

怎麼做：
1. 觀察三大法人（外資、投信、自營商）進出
2. 追蹤千張大戶持股變化
3. 跟單順勢操作、搭大戶便車

實例：

  • 情境：外資連買5日、投信連買3日、千張大戶增加10%
  • 操作：籌碼集中，可考慮跟單

適用時機：中短期操作、題材股

消息面選股

是什麼：從財經新聞找投資機會

怎麼做：
1. 關注公司重大消息（新產品、新訂單、新產能）
2. 分析消息對股價的影響
3. 判斷消息真假和時效

實例：

  • 情境：台積電宣布3奈米量產，訂單能見度達2年
  • 操作：消息公布前佈局，消息揭露後獲利了結

適用時機：事件驅動、題材操作

二、兩大選股流程

由下而上分析法（Bottom-up）

是什麼：從個股研究開始，向上追溯到產業、總體經濟

怎麼做：
1. 選定觀察清單：用EPS、本益比、股利穩定度、營業利益率篩選
2. 研究公司：主要產品、核心技術、關鍵客戶、競爭對手
3. 閱讀財報：三表重要比率、營收追蹤、法說會
4. 延伸上下游：了解供貨夥伴、下游客戶、議價能力
5. 加入總經：景氣循環、利率變化、匯率走勢

實例：

  • 情境：研究台積電
  • 步驟：財報→產品（晶圓代工）→客戶（蘋果、輝達）→競爭對手（三星）→上下游（ASML）→總經（半導體景氣）

適用時機：對個股研究有興趣、時間充裕者

由上而下分析法（Top-down）

是什麼：從總體經濟出發，找明星產業，再到個股

怎麼做：
1. 總經分析：利率、CPI、GDP、PMI、景氣燈號
2. 產業分析：找未來成長產業（如AI、電動車）
3. 產業內選股：選龍頭或競爭優勢公司

實例：

  • 情境：觀察AI趨勢
  • 步驟：總經（降息寬鬆）→產業（AI伺服器需求）→個股（輝達、台積電、廣達）

適用時機：喜歡大方向思考、想掌握趨勢者

三、產業研究工具

產業價值鏈資訊平台

是什麼：證交所和櫃買中心建立的免費平台

怎麼用：
1. 網址：https://ic.tpex.org.tw
2. 選產業類別（共30類）
3. 閱讀產業解說（從上游到下游）
4. 點擊供應鏈環節查詢個股

實例：

  • 情境：研究紡織業
  • 操作：點進紡織業→閱讀從石化原料到成衣的供應鏈→查詢相關個股

產業公協會網站

是什麼：產業的官方組織，提供專業報告和數據

怎麼用：
1. 搜尋「該產業 + 公會/協會」
2. 找月報、年報、產銷統計
3. 取得第一手進出口數據

實例：

  • 情境：研究工具機產業
  • 操作：進工具機公會網站→看MA雜誌→查進出口數據

Goodinfo選股工具

是什麼：免費網站，整合基本面、技術面、籌碼面數據

怎麼用：
1. 網址：https://goodinfo.tw/tw/index.asp
2. 點「類股一覽」
3. 選產業（39種分類）
4. 用指標排序篩選（如營收年增率）

實例：

  • 情境：找紡織股中短期業績最強者
  • 操作：類股一覽→紡織→按單月營收年增率排序

MoneyDJ個股介紹

是什麼：快速認識一檔個股的網站

怎麼用：
1. Google搜尋「MoneyDJ + 股票名稱」
2. 閱讀公司背景、業務、產品組合
3. 了解產業上下游、競爭對手

實例：

  • 情境：第一次看到「台燿」
  • 操作：搜尋「MoneyDJ 台燿」→了解做PCB、客戶是誰、競爭對手

四、公司估值方法

本益比（P/E）

是什麼：股價除以每股盈餘，衡量股價是否合理

怎麼算：本益比 = 股價 / EPS

實例：

  • 情境：台積電股價500元，EPS 40元
  • 計算：本益比 = 500 / 40 = 12.5倍
  • 判斷：低於產業平均15倍，可能被低估

股價淨值比（P/B）

是什麼：股價除以每股淨值，衡量股價相對帳面價值

怎麼算：股價淨值比 = 股價 / 每股淨值

實例：

  • 情境：某股價100元，每股淨值80元
  • 計算：P/B = 100 / 80 = 1.25倍
  • 判斷：大於1表示市場給予溢價

企業價值倍數（EV/EBITDA）

是什麼：從公司營運角度評估股價

怎麼算：EV/EBITDA = （市值 + 負債 - 現金）/ 息前稅前折舊攤銷前盈餘

實例：

  • 情境：適合資本密集產業（如半導體）
  • 判斷：可排除資本結構影響，比較不同公司

五、經營表現衡量指標

股東權益報酬率（ROE）

是什麼：公司用股東資金賺錢的能力

怎麼算：ROE = 淨利 / 股東權益

實例：

  • 情境：公司淨利100億，股東權益500億
  • 計算：ROE = 100 / 500 = 20%
  • 判斷：ROE越高越好，巴菲特喜歡ROE > 15%的公司

資產報酬率（ROA）

是什麼：公司用資產賺錢的能力

怎麼算：ROA = 淨利 / 總資產

實例：

  • 情境：公司淨利100億，總資產1000億
  • 計算：ROA = 100 / 1000 = 10%
  • 判斷：適合資產輕重比較

六、「三新二益」選股心法

是什麼：分析師陳威良提出的飆股選股心法

三新：
1. 新產品：公司推出新產品
2. 新訂單：接到新的大訂單
3. 新產能：新產線開出

二益：
1. 意料外利多：市場未預期的利多消息
2. 實質財報利益：真正反映在財報上的獲利

實例：

  • 情境：鼎炫-KY今明年完成收購兩家中國企業
  • 判斷：一年貢獻EPS 10元（實質財報利益）、收購（新訂單）

七、景氣循環股 vs 防守型產業

景氣循環股

特徵：受總體環境影響大，營運有週期性

產業：航空、汽車、金融、鋼鐵、營建、原物料

怎麼操作：
1. 在景氣谷底進場
2. 在景氣高峰出場
3. 週期結束前獲利了結

防守型產業

特徵：受景氣影響小，營運穩定

產業：民生必需品、公用事業、電信、醫療

怎麼操作：
1. 長期持有收股息
2. 在市場恐慌時加碼
3. 不用擔心景氣循環

八、貨幣政策對股市的影響

降息環境

是什麼：央行降息，市場資金寬鬆

影響：

  • 市場流動資金增加
  • 科技股、成長股受青睞
  • 風險性資產漲幅大

選股方向：成長股、科技股、小型股

升息環境

是什麼：央行升息，市場資金緊縮

影響：

  • 市場流動資金減少
  • 資金流向保守穩健股票
  • 高估值成長股壓力大

選股方向：價值股、金融股、高股息股

心得

這回合補強了產業分析和選股方法，這塊之前確實比較薄弱。由下而上 vs 由上而下兩種方法，其實可以混用——先觀察總經趨勢抓產業方向，再用基本面工具深入個股。
