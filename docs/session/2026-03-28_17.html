核心知識

一、非農就業數據（Non-Farm Payrolls, NFP）

是什麼：每月第一個週五公布的美國就業報告，包含非農就業人口變動、失業率、勞動參與率、平均時薪四大核心指標，是全球金融市場最受關注的經濟數據。

四大核心指標：
1. 非農就業人口變動（NFP）：農業以外行業新增就業人數

  • 正值：經濟擴張訊號

2. 失業率：失業人口 ÷ 勞動力人口

  • U-3：一般失業率（新聞報導使用）

3. 勞動參與率：有工作意願者佔成年人口比例

  • 下降可能代表「放棄求職」而非失業改善

4. 平均時薪：工資水準變化

  • 年增率 > 4%：可能觸發 Fed 預防性升息

怎麼做：
1. 公布時間：每月第一個週五，台灣時間 20:30（夏令）或 21:30（冬令）
2. 關注「實際值 vs 預期值」差距
3. 不要只看單一數據，要綜合判斷

實例：

  • 情境：NFP 公布 +25 萬人，預期 +15 萬人，時薪年增率 4.2%
  • 判斷：經濟過熱，升息預期升溫
  • 操作：美元可能升值，EUR/USD 可能下跌

適用時機：

  • 美股、美債、外匯交易者必看
  • Fed 決策前後特別重要
  • 避開公布後 5-30 分鐘的高波動區間

二、失業率深入解讀

是什麼：美國失業率分為 U-1 到 U-6 六個層級，市場通常只看 U-3，但 U-6 更能反映真實就業情況。

失業率分級：

指標定義
U-1持續 15 週以上失業者的比例
U-2遭解雇或暫時結束工作者的比例
U-3一般失業率（新聞報導使用）
U-4U-3 + 放棄求職者
U-5U-4 + 準備工作但無法工作者
U-6U-5 + 希望全職但被迫兼職者（廣義失業率）

關鍵判讀技巧：
1. 失業率下降 + 勞動參與率上升 = 真正好轉
2. 失業率下降 + 勞動參與率下降 = 假性改善（放棄求職）
3. 失業率上升 + 勞動參與率上升 = 經濟復甦徵兆（更多人找工作）

實例：

  • 情境：失業率 4.2% 維持穩定，但勞動參與率從 62.6% 降到 62.4%
  • 判斷：表面穩定，實際可能有人放棄求職
  • 觀察：U-6 失業率是否同步下降

三、初領失業救濟金人數（Initial Jobless Claims）

是什麼：每週四公布，統計上週初次申請失業救濟金的人數，是勞動市場的「領先指標」，可預測非農數據。

怎麼做：
1. 公布時間：每週四台灣時間 20:30（夏令）或 21:30（冬令）
2. 觀察趨勢：連續上升或下降比單週數字重要
3. 對照預期值：高於預期 = 經濟放緩訊號

與非農的關係：

  • 初領失業金人數上升 → 非農可能疲弱
  • 初領失業金人數下降 → 非農可能強勁
  • 每週數據可提前判斷月度非農趨勢

實例：

  • 情境：連續 4 週初領失業金人數從 20.5 萬上升到 25 萬
  • 判斷：就業市場降溫，非農數據可能低於預期
  • 操作：Fed 可能偏鴿派，美元承壓

數據解讀要點：

  • 初領失業金：領先指標，波動大
  • 連續領取失業金：同時指標，較穩定
  • 正常範圍：~20-25 萬人/週
  • 超過 30 萬人：勞動市場明顯惡化

四、數據交易實戰策略

數據公布前：
1. 不要猜方向，波動可能高達 50-100 點
2. 先下單風險極高，容易被假突破洗掉
3. 觀察市場預期與前值差距

數據公布後（5-30 分鐘內）：
1. 觀察初始方向與市場反應
2. 警惕「先漲後跌」或「先跌後漲」的假突破
3. 等待短期趨勢確認再進場

停損停利設定：
1. 事件交易必設停損，波動可能達 50-100 點
2. 停損設定 20-30 點（外匯）
3. 停利目標 50-100 點

風險控制：

  • 單筆風險 ≤ 2% 帳戶
  • 數據公布前減倉或觀望
  • 避免過度槓桿

五、關鍵數據追蹤時程

每週追蹤：

  • 週四：初領失業救濟金人數

每月追蹤：

  • 第一週週五：非農就業數據（NFP）
  • 月中：CPI 通膨數據
  • 月底：PCE 物價指數

Fed 會議時程：

  • FOMC 利率決議：每年 8 次
  • 會議紀要：會議後三週公布

心得：

這次學到的美國經濟數據體系比想像中更龐大。非農不只是「一個數字」，而是包含 NFP、失業率（U-1 到 U-6）、勞動參與率、平均時薪四大核心，還有每週的初領失業金作為領先指標。

關鍵洞察：
1. 失業率下降不等於經濟好轉 — 要同時看勞動參與率，放棄求職的人不算失業
2. U-6 更能反映真實情況 — 包含被迫兼職者，是「真正的失業率」
3. 初領失業金是領先指標 — 每週追蹤，可提前預判非農

實戰上，數據公布前後 30 分鐘是高風險區間，假突破頻繁。更好的策略是：

  • 公布前觀望或減倉
  • 公布後等 5-10 分鐘確認方向
  • 搭配技術分析找進場點

這些總經數據對 Fed 決策有直接影響，而 Fed 決策又左右美元、美債、美股走向。掌握數據解讀，就是掌握市場脈搏。
