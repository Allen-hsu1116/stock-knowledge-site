頭肩型態轉折判斷

是什麼：成功率最高的反轉型態（研究統計約 83%），捕捉「趨勢反轉」的關鍵訊號。

頭肩頂（多頭終章）：

  • 位置：上漲末端，見頂訊號
  • 結構：左肩（放量衝高）→ 頭部（創新高但量略減）→ 右肩（量縮無力突破）
  • 頸線：連接左肩和頭部的低點
  • 確認訊號：跌破頸線 + 放量

頭肩底（多頭序曲）：

  • 位置：下跌末端，底部反轉
  • 結構：左肩（短暫反彈被打回）→ 頭部（創新低但跌幅收斂）→ 右肩（未創新低）
  • 頸線：連接兩個反彈高點
  • 確認訊號：突破頸線 + 放量

怎麼做：
1. 等待完整型態（右肩必須長出來才算成立）
2. 確認成交量變化（右肩量要比左肩小）
3. 等頸線突破/跌破後才進場（不要猜頭摸底）
4. 放量突破/跌破才進場（沒量可能是假訊號）

目標價計算：

  • 頭肩頂目標跌幅 = 頭部高點 - 頸線
  • 頭肩底目標漲幅 = 頸線 - 頭部低點

成交量在轉折判斷的關鍵角色

是什麼：成交量是驗證價格變化的「測謊器」，華爾街名言：「股市充斥騙子，只有成交量不說謊」。

量價關係判斷：

量價關係意義操作方向
價漲量增多方強勢，健康上漲可跟進
價漲量縮上漲無力，可能是假突破觀望
價跌量增警訊！恐慌殺盤快撤
價跌量縮空頭持續，還沒到底觀望

假突破與假跌破識別

假突破特徵：
1. 突破時成交量不足（沒人願意追價）
2. 突破後立刻拉回頸線以下
3. 右肩成交量沒有明顯縮小（表示主力還在試單）

假跌破特徵：
1. 跌破頸線 3% 以內又拉回
2. 跌破時量能不足
3. 收盤價沒有有效站穩頸線下方

防範技巧：
1. 等「收盤價」確認，不盤中衝動
2. 跌破後等反彈不過頸線再進場（保守做法）
3. 搭配大盤方向：大盤弱勢時，突破成功率會下降
4. 觀察量的持續性：單日放量不算，要連續 2-3 日

轉折點的成交量四大關鍵

1. 右肩量縮是關鍵：頭肩頂右肩成交量明顯小於左肩 → 買氣疲乏
2. 突破/跌破要放量：頸線突破時必須放量（比前 5 日均量多 30% 以上）
3. 觀察量的持續性：單日放量不代表趨勢確立，要連續 2-3 日量能維持
4. 警惕爆量長黑/長紅：高檔爆量長黑 → 主力出貨；低檔爆量長紅 → 主力進場

心得

三個重要體悟：

1. 型態只是骨架，量能才是血肉。頭肩型態成功率高，是因為它反映了市場情緒從貪婪到恐懼（或反過來）的完整過程。
2. 右肩量縮是型態成立的關鍵證據。如果右肩量沒縮，表示主力還在試單或洗盤，型態可能失敗。
3. 假突破/假跌破的防範在於「等」。不要盤中衝動，等收盤確認、等放量、等回測不破。

實戰應用方向：

  • 看到頭肩型態，先看右肩量有沒有縮
  • 突破頸線時，檢查成交量是否比均量多 30% 以上
  • 不搶第一個進場點，等確認後再進場
