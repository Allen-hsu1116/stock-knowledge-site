核心知識

一、交易心理壓力管理

是什麼：在高波動市場中保持理性決策的能力，包括恐懼控制、貪婪管理、自信心建立三個層面。

1. 恐懼對交易的影響

是什麼：恐懼是最常見、也最具破壞力的情緒，通常在市場劇烈波動時出現。

怎麼做：
1. 辨識恐懼徵兆：過早停損、錯過入場時機、不敢追勢
2. 區分合理停損與情緒停損：前者依計畫，後者因恐慌
3. 建立停損紀律：設定好停損位後，執行計畫不動搖

實例：

  • 情境：買進 EUR/USD，設好停損
  • 價格短暫下探觸及停損後迅速反彈
  • 情緒反應：因恐懼提前平倉，錯失後續獲利
  • 正確做法：停損位要設在合理波動區間外

恐懼的代價：

  • 過早停損 → 白白承受可避免的損失
  • 不敢進場 → 錯過良機，追高時風險已大增

2. 貪婪對交易的影響

是什麼：貪婪是從獲利走向虧損的「隱形殺手」。

危害表現：
1. 過度加倉：覺得行情會一直漲，不斷加碼
2. 該止盈不止盈：希望再多賺一點，結果利潤回吐
3. 忽視風險信號：只看獲利可能，忽略反轉徵兆

怎麼做：
1. 設定合理目標：進場前先想好出場條件
2. 分批止盈：不要一次全出，但要有出場計畫
3. 嚴守加碼紀律：只在計畫中規劃好的加碼點位加碼

3. 交易自信心

是什麼：決定能否長期走下去的關鍵，是「定海神針」。

為什麼重要：

  • 自信的交易者能堅持計畫，不因波動動搖
  • 缺乏自信會頻繁修改計畫、猶豫不決

如何建立：
1. 深思熟慮的判斷：每一次入場出場都有理由
2. 紀律執行：照計畫操作，累積成功經驗
3. 複盤改進：從失敗中學習，而非沉浸在懊悔中

案例：索羅斯 1992 年做空英鎊

  • 當時幾乎無人敢做空英鎊
  • 他憑藉對宏觀經濟的深入理解，堅信英鎊被嚴重高估
  • 不受外界質疑干擾，果斷進場
  • 最終獲利 11 億美元

4. 高壓環境保持冷靜

實用方法：

方法一：交易計畫是情緒穩定器

  • 規定進場時機、出場條件、可承受虧損、目標收益
  • 有計畫就不會被情緒牽著走
  • 關鍵是「執行計畫」——很多人不是沒計畫，而是不照計畫做

方法二：深呼吸與冥想

  • 深呼吸技巧：吸氣 4 秒 → 憋住 4 秒 → 吐氣 4 秒，重複練習
  • 效果：降低心率、緩解壓力、恢復理性思考
  • 冥想：每天幾分鐘專注呼吸，排除雜念
  • 效果：提升專注力、增強心理韌性

方法三：複盤從失敗中成長

  • 每次虧損後認真複盤：

1. 資訊獲取是否完整？
2. 技術判斷是否正確？
3. 執行力是否到位？
4. 是否有情緒干擾？

  • 找出問題，變成改進方向
  • 失敗不可怕，可怕的是重複同樣的錯誤

二、風險調整績效評估指標

是什麼：衡量「承擔單位風險能獲得多少報酬」的指標，用於評估策略或投資組合的績效。

1. 夏普比率（Sharpe Ratio）

是什麼：衡量投資組合每單位風險所獲得的超額報酬。

公式：

夏普比率 = (投資組合平均報酬率 - 無風險利率) ÷ 報酬率標準差

實例計算：

  • 0050 自成立以來平均年化報酬：11%
  • 定存利率（無風險利率）：1.7%
  • 標準差：0.23（23%）

夏普比率 = (0.11 - 0.017) ÷ 0.23 = 0.403

怎麼解讀：

  • 夏普率 0.403 = 承受 1% 波動風險，可獲得 0.403% 報酬
  • 反過來說：想得到 4.03% 報酬，預期過程有 10% 上下波動

使用原則：
1. 夏普比率越高，承擔相同風險獲得的報酬越高
2. 要在相同時間區間比較才有意義
3. 數值高不一定好，要看報酬/波動組合

限制：

  • 把「正報酬的波動」也算進風險
  • 對於「大賺小賠」或「小賺大賠」的不對稱策略，評估失真

2. 索提諾比率（Sortino Ratio）

是什麼：改進版夏普比率，只考慮「下行風險」（負報酬的風險）。

公式：

索提諾比率 = (期間內平均報酬 - 最低可接受報酬) ÷ 虧損時的標準差

核心邏輯：

  • 投資人不介意「正報酬的波動」（大賺是好事）
  • 只在意「負報酬的風險」（虧損才是風險）
  • 因此只計算下行標準差，剔除上行波動

與夏普比率的差異：

指標風險計算方式特點
夏普比率總標準差（正負報酬都算）原始數據，最常用
索提諾比率下行標準差（只算負報酬）專注虧損風險

什麼情況用索提諾比率較好：
1. 勝率低、大賺小賠的策略（如趨勢追蹤）
2. 勝率高、小賠大賺的策略（如賣期權）
3. 績效走勢獲利與虧損時期特性不對稱

數值閱讀：

  • 索提諾比率數值通常比夏普比率更高
  • 兩者通常正比關係：夏普高，索提諾也高
  • 兩者擇一觀察即可，多數情況看夏普比率

3. 使用注意事項

時間長度：

  • 建議以長期為參考依據
  • 需包含多頭、空頭與各種市場狀況
  • 參考期限太短，無法看出真正風險

流動性問題：

  • 低流動性資產（小盤股、私募）可能讓風險數值失真
  • 實際風險比數值顯示的更高

樣本充足性：

  • 獲利與虧損樣本都要充足
  • 少數幾筆大賺可能拉高整體報酬
  • 關鍵問題：績效未來能否重複複製？

心得

風險管理兩大支柱：心理素質與績效評估。

心理層面，恐懼和貪婪是交易最大的敵人。很多時候虧損不是因為看錯方向，而是因為情緒失控：恐懼讓人過早停損或不敢進場，貪婪讓人加倉過度或該止盈不止盈。解決之道很簡單：有計畫、執行計畫、事後複盤。

績效評估方面，夏普比率和索提諾比率告訴我們：報酬率 alone 沒有意義，要看「承擔多少風險換來多少報酬」。索提諾比率的改良思路很有意思——我們確實不在意大賺時的波動，只在意虧損時的風險。對於不對稱績效的策略，索提諾比率更貼近真實感受。

最後，複盤是成長的關鍵。失敗不可怕，重複同樣的錯誤才可怕。每次虧損都是學習機會，找出問題在哪裡，下次改進，這才是成熟交易者的心態。
