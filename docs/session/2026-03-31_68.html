成長股 vs 價值股：核心定義與判斷方法

一、成長股（Growth Stock）

是什麼：市場認定具有高速潛在獲利成長空間的企業，股價因市場預期而享有較高估值。即使當下虧損或營收不高，股價仍可能極高。

怎麼判斷：
1. 營收成長率 ≥ 20% — 一般標準是維持雙位數增速才算「高速成長」
2. 高本益比 — 估值倍數可達幾十倍甚至百倍
3. 低或無股息 — 盈利多用於再投資，不重視配息
4. 重視未來成長性 — 評價建立在「未來獲利預期」而非當下

實例：

  • 台積電：年殖利率約 2%，保留盈餘率 60%，ROE 約 25%
  • 年獲利率估算：2% + 60% × 25% = 17%
  • 關鍵在於成長持續性而非股息

適用產業：資訊科技、醫療保健、新興產業

風險：若預期成長未實現，高股價可能只是「太貴」，不是「有潛力」

二、價值股（Value Stock）

是什麼：股價低於企業當下價值，被市場低估的股票。通常基本面穩健、現金流強勁、配息穩定。

怎麼判斷：
1. 低本益比、低本淨比 — 相對便宜的估值倍數
2. 高殖利率 — 通常有穩定配息（4%以上）
3. 低成長率 — 營收可能個位數甚至零成長
4. 強勁現金流 — 獲利穩定但市場給予較低評價

實例：

  • 中華電信：2023 年殖利率 6.15%，股價上漲 0.62%
  • 資本投資率趨近 0（淨利幾乎全配給股東），ROE 約 10%
  • 年獲利率估算：5% + 0% = 5%

適用產業：金融、工業、公用事業、傳統產業

適用時機：

  • 利差反轉時（短期債券殖利率 > 長期債券）
  • 經濟前景不確定、通膨升息環境
  • 成長股表現過分強勢後的修正期

本淨比 vs 本益比：

  • 本益比：適用於獲利穩定的公司
  • 本淨比：適用於營收波動劇烈、景氣循環股

EV/EBITDA：

  • 扣除折舊費用，更適合評估創業前期、大量資本支出的公司
  • 如小型新興科技股

四、DCF 估算法：計算預期年獲利率

公式：

年獲利率 = 殖利率(1) + 股價成長率(2)
股價成長率(2) = 保留盈餘率 × ROE

注意：DCF 是估算，實際投資需考量市場環境、競爭態勢、科技趨勢等因素。

五、不同經濟環境的選擇

成長股優勢環境：

  • 低利率、資金寬鬆
  • 經濟擴張期
  • 科技創新浪潮（如 AI、5G）
  • 2008 金融海嘯後的長多頭

價值股優勢環境：

  • 升息循環、通膨壓力
  • 利差反轉（衰退前兆）
  • 經濟前景不確定
  • 銀行業危機時避開金融比重高的價值股基金

六、避雷心法

成長股陷阱：

  • 預期成長未實現 → 股價只是「太貴」
  • 本益比百倍但營收沒跟上 → 泡沫
  • 獲利都用於擴張，沒有配息保護

價值股陷阱：

  • 股價「低估」可能是反映未來衰退
  • 產業夕陽化 → 所謂便宜只是「價值陷阱」
  • 營運狀況日漸衰落 → 不是被低估，是真的變差

核心心法：
> 「成長股看未來，價值股看當下。但兩者都需要正視基本面，因子投資只是輔助。」

心得

這次學到最實用的就是 DCF 估算公式：殖利率 + 保留盈餘率 × ROE。這公式讓我可以用一套方法快速比較成長股和價值股的預期報酬。

之前我總覺得高殖利率股票比較穩，但其實要看人生階段。年輕時可以承受風險，追求成長股的 17% 預期報酬；快退休時，價值股的 5% 穩定配息反而更適合。

另外一個關鍵：價值股不是「越便宜越好」，要避開價值陷阱。股價低可能是市場領先反映衰退，不是撿便宜。成長股也是，高本益比要有成長支撐，不然就是泡沫。
