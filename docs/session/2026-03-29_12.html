波浪理論核心概念

### 「是什麼」 波浪理論（艾略特波浪理論）由艾略特於1930年代提出，認為市場價格走勢呈現「5波上升、3波下降」的規律循環，反映投資人情緒的集體變化。

核心重點：不是預測高點，而是掌握情緒循環節奏。

八波結構詳解

推動波（上升五波）

波浪名稱特性成交量市場情緒
波浪1初升波趨勢萌芽，利多不明顯低懷疑、保守
波浪2回檔波吃掉波浪1大部分漲幅，但不跌破起點比波浪1小悲觀
波浪3主升波**最強最長**，突破波浪1高點是買進訊號最大樂觀轉強
波浪4整理波橫盤或小回檔，底部高於波浪1高點縮減觀望
波浪5末升波情緒最高潮，技術指標常背離大但量價背離過熱

修正波（下降三波）

波浪特性市場誤判
波浪A初次下跌多數人以為只是暫時回檔
波浪B假反彈「多頭陷阱」，讓人以為多頭回來了
波浪C全面下跌真正的修正，破壞力最強

波浪理論三鐵律

「怎麼做」— 判斷波浪是否正確的關鍵規則：

1. 波浪2不可能完全回撤波浪1 — 波浪2的底不會跌破波浪1的起點
2. 波浪3不可能是驅動波中最短的 — 如果你的數浪中波浪3比波浪1和5都短，數法就錯了
3. 波浪4的低點不能與波浪1頂點重疊 — 波浪4的底要在波浪1的高點之上

實例判斷：

  • 情境：某股票波浪1從100漲到120（漲20點），波浪2回檔到105
  • 驗證：105 > 100，符合鐵律1 ✓
  • 波浪3從105漲到145（漲40點），比波浪1長，符合鐵律2 ✓
  • 波浪4回檔到128，比波浪1高點120還高，符合鐵律3 ✓

費波南希比例計算

上升五波計算

波浪計算方式
波浪2回檔波浪1漲幅的 **38.2%～78.6%**
波浪3幅度波浪1的 **1.618倍、2.62倍、或4.25倍**（最常見是1.618倍）
波浪4回檔波浪3的 **23.6%或38.2%**
波浪5幅度若波浪3延伸：波浪5 = 波浪1 或 波浪1 × 1.618

修正波計算

波浪計算方式
波浪B反彈波浪A的 **38.2%～78.6%**
波浪C幅度波浪A的 **1.382倍或1.618倍**

邪惡第五波

### 「是什麼」 波浪理論中最危險的一段，**第五波可能是最長的延伸波**，突破主升段波浪3，讓人以為多頭還沒結束，但隨後會迎來強烈反轉。 ### 「為什麼邪惡」 1. **脫離常態** — 第五波可能走得比波浪3還強還長 2. **情緒最高** — 散戶、擦鞋童都進場，市場一片樂觀 3. **技術背離** — 量價背離、技術指標背離早已出現 4. **反轉猛烈** — ABC修正波會快速吃掉前面漲幅

「怎麼做」— 邪惡第五波操作策略

識別訊號：

  • 市場普遍亢奮，利多新聞滿天飛
  • 成交量放大但股價漲幅有限（量價背離）
  • 技術指標（RSI、MACD）出現頂背離
  • 散戶大量進場（街上人人談股票）

操作原則：
1. 不追高 — 第五波雖然可能延伸，但風險報酬比已惡化
2. 逐步減碼 — 波段部位分批出場，不一次賣完
3. 嚴設停損 — 把停損設在前一波段重要轉折點
4. 避開槓桿 — 不要在末升段放大槓桿

波浪理論實戰應用

進場時機

最佳進場點：波浪2結束、波浪3突破時

「怎麼做」：
1. 觀察波浪1結束後的回檔
2. 用費波南希比例抓波浪2可能支撐（0.382、0.5、0.618）
3. 等波浪2回檔結束，波浪3突破波浪1高點時進場
4. 停損設在波浪2的低點下方

實例：

  • 情境：台積電從500漲到600（波浪1），回檔到550（波浪2的0.5回檔）
  • 操作：在550附近觀察，若突破600，進場買進
  • 停損：設在540（波浪2低點下方）

第二進場點：波浪5初期

風險較高，但波浪3沒抓到時可用：

1. 只有在波浪5突破波浪3時才進場
2. 停損要更嚴格，設在波浪4低點
3. 隨時準備出場，因為波浪5隨時可能結束

出場時機

波浪5或修正波C波階段：

  • 技術指標背離時逐步減碼
  • 不要貪戀最後一段漲幅
  • 現金為王，等待下一個循環

波浪理論三大注意事項

### 1. 不能單獨使用 波浪理論基於市場情緒，**必須搭配其他指標**：
  • 成交量確認
  • 技術指標（RSI、MACD）背離
  • 總體經濟數據
  • 恐懼貪婪指數
### 2. 波浪數法有主觀性 同一張圖，不同分析師可能數出不同的波浪。實務上要：
  • 對照多種時間週期（日線、週線、月線）
  • 驗證是否符合三鐵律
  • 等待確認訊號再行動
### 3. 延伸波難以預測 波浪3或波浪5都可能出現延伸波，讓幅度難以預測。處理方式：
  • 用移動停利，讓獲利跟著走
  • 不要想抓最高點，分批出場

波浪理論優缺點

### 優點
優點說明
趨勢框架提供完整的市場循環視角
情緒解讀幫助理解市場心理變化
費波南希比例客觀的支撐壓力參考
### 缺點
缺點說明
主觀性高不同人數法可能不同
事後驗證常常是「事後諸葛」才能確認波浪
不一定發生並非所有走勢都符合5波3波模式

心得

波浪理論最有價值的概念不是「預測」而是「判斷市場所處階段」。知道現在在哪一波，比猜未來會怎麼走更重要。

實戰上，我會特別注意幾個點：
1. 波浪3是最佳進場點 — 突破波浪1高點時進場，風險報酬比最好
2. 邪惡第五波要小心 — 市場一片樂觀時反而要準備撤退
3. 三鐵律是驗證工具 — 如果數出來的波浪違反三鐵律，代表數法錯了

但波浪理論最大的問題是事後驗證，很多時候只有行情走完才能確認「喔！原來那個是波浪3」。所以實務上我會搭配：

  • 成交量確認（波浪3量最大）
  • 技術背離（波浪5常見背離）
  • 移動停利（不猜頭，讓市場告訴我何時出場）

最後，邪惡第五波這個概念提醒我：最後的瘋狂往往最致命。當散戶都在談股票、擦鞋童都給明牌時，反轉就不遠了。

核心知識

一、波浪理論七種進場時機（多頭市場）

1. 第二波結束時進場買進

是什麼：在第2波回檔結束時進場，準備賺取完整的第3波漲幅

怎麼做：
1. 確認左邊走勢已完成A-B-C回檔修正波
2. 確認緊接著的第1波可細分為5個上升小波
3. 觀察第2波回檔幅度：可能是第1波的0.382、0.5或0.618處
4. 用c波≒a波來找尋回檔低點進場

停損設定：

  • 設在比第1波起點低一點的地方
  • 第2波不會跌破第1波的起點（波浪理論三鐵律之一）

獲利目標：

  • 第3波整段漲幅（通常是最長的一波）

實例：

  • 情境：某股票從100元漲到120元（第1波），回檔到110元（第2波，回檔50%）
  • 操作：在110元附近進場
  • 停損：設在98元（低於第1波起點）
  • 目標：賺取完整第3波，假設第3波漲幅等於第1波的1.618倍，目標約142元

2. 在第三波之中進場買進

是什麼：唯一「不等待低點」的進場時機，在第3波進行中跳空突破第1波高點時加碼

怎麼做：
1. 確認第2波已結束且進場作多無誤
2. 等待股價接近第1波頂點
3. 觀察是否跳空開高突破第1波頂點
4. 跳空當天進場加碼

停損設定（兩種方法）：
1. 設在略低於第1波頂點（因為第3波進行中不會回頭）
2. 設在略低於前一天高點

獲利目標：

  • 第3波整段漲幅，利潤最豐富且來得快
  • 第3波漲幅至少與第1波相等

注意事項：

  • 如果跳空當天就回補，須重新檢視波浪辨認是否有錯誤
  • 第3波爆發力最強

實例：

  • 情境：第1波高點120元，股價跳空開在122元
  • 操作：122元進場加碼
  • 停損：設在118元（低於第1波高點）
  • 風險：如果當天回補跌破120元，代表波浪判斷錯誤

3. 在第四波將結束時進場買進

是什麼：在第4波修正尾聲進場，準備賺取第5波漲幅

怎麼做：
1. 確認已完成強勁的第3波（有跳空、比第1波長）
2. 觀察第4波低點是否高於第1波高點（波浪鐵律）
3. 尋找第4波結束點的5個線索：

  • 第4波低點可能是第3波回檔0.382處

停損設定：

  • 設在比第1波高點低一點的位置

獲利目標：

  • 第5波漲幅
  • 第5波長度可能：等於第1波，或=第1波起點至第3波高點的0.618倍

實例：

  • 情境：第1波高點120元，第3波高點180元，第4波回檔到145元
  • 進場：145元附近（第4波結束點）
  • 停損：118元（低於第1波高點）
  • 目標：145 + (180-100)×0.618 = 194元（第5波目標）

4. 在第五波高點賣出（放空）

是什麼：在第5波結束時放空，賺取A-B-C修正波的跌幅

怎麼做：
1. 確認上升五波已完成
2. 用MACD背離輔助判斷第5波是否結束
3. 觀察是否有竭盡缺口出現
4. 等待竭盡缺口被回補或跳空下跌

停損設定（三種方法）：
1. 第3波漲幅以上一點
2. 第1波到第3波漲幅的0.618倍以上一點
3. 第5波高點以上一點

獲利目標：

  • A-B-C波跌幅
  • 至少回檔到前上升五波的0.382處或第4波低點

注意事項：

  • 有摸頭探底之嫌，停損難設
  • A波跌幅沒有固定標準，目標區最難確認
  • B波反彈可能超過第5波高點（不規則修正）

5. 在B波結束時賣出（放空）

是什麼：在修正波的B波反彈結束時放空，賺取C波跌幅

怎麼做：
1. 先算A波是三波下跌或五波下跌
2. A波五波段：B波反彈幅度是A波的0.382、0.5或0.618倍，不超過第5波高點
3. A波三波段：B波反彈幅度約為A波的1.236～1.382倍，可能超過第5波高點

停損設定：

  • A波五波段：設在第5波高點以上
  • A波三波段：設在A波跌幅×1.382以上

獲利目標：

  • C波跌幅
  • C波跌幅至少約略與A波相同

實例：

  • 情境：第5波高點200元，A波從200跌到160（五波段）
  • B波反彈預期：160 + (200-160)×0.618 = 185元附近
  • 操作：185元附近放空
  • 停損：202元
  • 目標：160元以下（C波至少跌到與A波相同）

6. 在鋸齒型修正波之後下單

是什麼：鋸齒型修正波的C波完成後進場

怎麼做：
1. 確認鋸齒型修正波（5-3-5結構）
2. C波要分為五波
3. 各種指標呈現極度超賣或超買
4. 確認鋸齒型修正波是第2波、第4波或B波

注意事項：

  • 鋸齒型修正波可能是第2波、第4波或B波
  • 如果是B波，最好場外觀望（B波最容易賠錢）
  • C波跌幅可能是A波等長，或A波×1.618或×2.618

7. 在三角形整理區作單

是什麼：在收斂三角形整理中作單，等待突破

怎麼做：
1. 確認價位帶愈來愈窄
2. 每一波幅度約為前一波的0.618倍
3. 三角形只出現在第4波或B波
4. 在三角形e波進場
5. 等待突破，通常會有跳空

停損設定：

  • 設在三角形c波以下

獲利目標：

  • 三角形整理突破後的漲跌幅度
  • 至少為三角形整理內最寬的距離

二、波浪理論計算實戰（費波南希比例）

上升五波計算公式

波浪計算方式
第1波基準波，作為計算基礎
第2波回檔第1波的38.2%～78.6%
第3波可能為第1波的1.618倍、2.62倍或4.25倍
第4波回檔第3波的23.6%或38.2%或50%
第5波若第3波≥1.618倍第1波：第5波=第1波或第1波×1.618或×2.62

下降三波計算公式

波浪計算方式
A波基準波
B波回檔A波的38.2%～78.6%，或延伸至A波×1.272～1.382
C波可能為A波×1.382或A波×1.618

三、邪惡第五波識別

是什麼：第5波產生很長的延伸波，幅度甚至超越第3波，隨後迎來強烈反轉

特徵：
1. 市場已大漲很長時間
2. 市場情緒高漲，新手紛紛進場
3. 第5波有延伸波（只在1、3、5波出現一次）
4. 漲幅可能超過第3波

風險：

  • 雖然利潤可能很大，但不一定能即時脫身
  • 後面的A-B-C修正波會殺得很兇

應對策略：
1. 用MACD背離輔助判斷第5波是否結束
2. 觀察竭盡缺口
3. 嚴設停損：第3波高點以上
4. 不要追高

四、波浪理論實戰三大注意事項

1. 波浪鐵律

  • 第2波不會跌破第1波起點
  • 第3波不會是最短的一波
  • 第4波低點不會低於第1波高點

2. 延伸波特性

  • 延伸波只會出現一次
  • 通常出現在第3波（主升段）
  • 出現延伸波代表走勢非常強勁

3. 不能單獨使用

  • 波浪理論依賴市場情緒
  • 必須搭配其他技術指標
  • 建議搭配：K線、型態學、成交量、MACD
  • 可參考恐懼貪婪指數判斷市場位階

五、波浪理論最佳進場時機

最佳時間：第3波剛形成

  • 第1波難以發現
  • 第2波結束後、第3波即將突破時進場
  • 第3波形成後有後續趨勢可賺

第二選擇：第5波初期

  • 一定要在第5波超過第3波時進場
  • 風險較高，第5波隨時可能結束反轉
  • 嚴設停損

心得

波浪理論最困難的是「數浪」——辨認現在是哪一波。這篇文章最可貴的是提供了七種具體的進場時機，每種都有明確的：
1. 先決條件（如何確認波浪位置）
2. 注意事項（觀察重點）
3. 停損位置（控制風險）
4. 獲利目標（設定停利）
