市場寬度指標概述

是什麼：市場寬度指標（Market Breadth Indicators）是一組衡量「整體市場參與度」的指標，不同於價格指數只反映少數權值股的漲跌，寬度指標告訴你「漲的是少數還是多數」。這是判斷大盤健康度的關鍵視角。

核心邏輯：

  • 健全的多頭市場：大多數個股都上漲，百花齊放
  • 虛假的多頭：只有少數權值股在撐指數，多數個股下跌
  • 台股特性：台積電佔大盤權重超過30%，漲跌1%影響指數約50點，極易出現「賺指數賠個股」的拉G盤

怎麼做：

一、ADL 騰落指數（Advance/Decline Line）

是什麼：累計每日上漲家數減去下跌家數，形成一條連續線。不看市值權重，每支股票平等對待——只管漲跌，不管漲幅大小或股本。

公式：

ADL(t) = ADL(t-1) + 上漲家數 - 下跌家數

平盤的股票不列入計算。

怎麼讀：

1. ADL 與大盤同向上升：真多頭，多數個股參與上漲，做多个股勝率高
2. ADL 下降但大盤上升（背離）：拉G盤，只有權值股在漲，中小型股沒參與，做多个股勝率低
3. ADL 連續下降：整體市場弱勢，下跌家數持續多於上漲家數

實戰策略——ADL MA5/MA60 交叉法：

  • 黃金交叉（MA5 突破 MA60）：市場寬度轉強，適合做多中小型股
  • 死亡交叉（MA5 跌破 MA60）：市場寬度轉弱，個股勝率低，適合持有0050或空手

三種配置策略回測結果：

  • ADL多方做個股 + 空方持有0050：近年績效優於一直持有0050
  • ADL空方空手：能躲過大跌，但長期錯過指數上漲
  • ADL空方硬做個股：長期績效近乎白工

實例：

  • 情境：大盤從16000漲到18000，但ADL從+500一路降到-200
  • 解讀：這800點的漲幅幾乎是台積電等權值股貢獻的，中小型股大多下跌
  • 操作：不要因為大盤漲就追個股，改持有0050或權值股

重要發現：台股ADL長期呈負值趨勢！這意味著即使在指數上漲的年份，下跌家數也經常多於上漲家數。這驗證了一個殘酷的事實：台股長期是「少數權值股在上漲、多數個股在下跌」。

適用時機：

  • 判斷是否適合做中小型股
  • 辨識拉G盤（賺指數賠個股）
  • 不適合：用來選個股（寬度指標是大盤工具）

二、ADR 漲跌比率（Advance/Decline Ratio）

是什麼：上漲家數除以下跌家數的比率，反映當日市場多空力量對比。與ADL不同，ADR看的是「比率」而非「累積」。

公式：

ADR = 上漲家數 / 下跌家數

怎麼讀：

1. ADR = 1：上漲家數等於下跌家數，多空均衡
2. ADR > 1.5：多方佔優勢，市場參與度高
3. ADR < 0.67：空方佔優勢，市場普遍下跌
4. ADR > 2：超買區，短期可能反轉
5. ADR < 0.5：超賣區，短期可能反短期可能反彈

鐘擺原理：ADR的基礎是鐘擺原理——當一方力量過大時，物極必反，向反方向擺動的拉力越強。人氣過度高漲→大跌；人氣低迷→反彈。

ADR的向上偏斜問題：

  • ADR在>1之後會快速上升（例如900漲100跌→ADR=9）
  • 但ADR<1時圖形都在0~1之間徘徊
  • 修正版：ADR = 上漲家數 / (上漲家數 + 下跌家數)，數值在0~1之間
  • 不過修正後與OBOS完全相同，所以一般仍用原公式

實際使用：

  • 通常取10日或15日移動平均（ADR(10)或ADR(15)），避免單日波動
  • 台股數據可在 CMoney 等平台直接查詢

實例：

  • 情境：上市900家，某日600漲250跌50平
  • ADR = 600/250 = 2.4，市場超買
  • 如果連續3天ADR>2，注意短期回檔風險

適用時機：

  • 判斷短期市場超買超賣
  • 搭配ADL使用：ADR看短期、ADL看長期
  • 不適合：ADR高檔反轉不代表股價也在高檔反轉

三、OBOS 超買超賣指標

是什麼：上漲家數減去下跌家數的差值，數值在0上下波動。比ADR更穩定，不會向上偏斜。

公式：

OBOS = 上漲家數 - 下跌家數
今日ADL = 前一日ADL + 今日OBOS

所以 OBOS 是 ADL 的「每日增量」。

怎麼讀：

  • OBOS > 0：上漲家數多於下跌家數，多方佔優
  • OBOS < 0：下跌家數多於上漲家數，空方佔優
  • OBOS 持續為負：即使指數上漲，市場內部其實很弱

實例：

  • 台股從8月開始OBOS持續小於0（上漲家數均值從未超過下跌家數均值）
  • OBOS在10月初領先大盤先做頭→提前預警轉折

適用時機：

  • 短期市場多空判斷
  • 提前預警大盤轉折
  • 不適合：長期趨勢判斷（用ADL更合適）

四、McClellan Oscillator 麥克連指標（MCL）

是什麼：ADL的進階版本，對淨漲跌家數（OBOS）做指數平滑移動平均，取19日EMA和39日EMA的差值。比原始ADL更平滑、訊號更清晰，是最常見的市場寬度指標之一。

公式：

DIF = 上漲家數 - 下跌家數
AD = DIF 的 19日指數移動平均（EMA）
BD = DIF 的 39日指數移動平均（EMA）
MCL = AD - BD

怎麼讀：

1. 常態行情（MCL在-25到+25之間）：

  • MCL上漲至+25~+35超買區，反轉跌破+25 → 短期賣出訊號

2. 極端行情（MCL衝出超買超賣區）：

  • MCL向上超越+35 → 極端多頭，不必急賣！通常還會持續2-3週

3. 極端行情的轉折判斷：

  • 背離確認：MCL高點越來越低但指數還在創高 → 多頭衰竭

實例：

  • 情境：MCL從-30反彈突破-25 → 短期買進訊號
  • 情境：MCL衝到+50（超越+35極端區）→ 不急賣，等MCL形成一頂比一頂低的背離走勢再出場
  • 情境：MCL連續3天從+40降到+30到+25 → 多頭力道減弱，準備離場

MCL vs ADL 比較：

  • ADL是累積值，長期趨勢明確但短期波動大
  • MCL是平滑後的差值，短期訊號更清晰
  • 兩者搭配使用：MCL找短期進出場、ADL看長期寬度趨勢

適用時機：

  • 判斷市場短期超買超賣
  • 辨識極端行情的持續與轉折
  • 搭配背離確認大盤轉折
  • 不適合：個股操作（寬度指標都是大盤工具）

四指標整合：寬度分析實戰框架

怎麼做：

1. 每日檢查清單：

  • 看 OBOS：今天淨漲跌家數是正還是負

2. 決策樹：

  • ADL多方 + MCL常態偏低 → 積極做個股

3. 與之前學的籌碼面交叉驗證：

  • ADL多方 + 外資買超 → 真多頭，強力做多

心得

市場寬度指標打開了一個全新的視角。之前學技術分析都是在看「價格怎麼動」，但寬度指標看的是「市場的靈魂健不健康」。

幾個關鍵收穫：

1. 台股ADL長期負值的殘酷真相：即使在指數年年創高的年代，下跌家數也多於上漲家數。這說明台股的「賺指數賠個股」不是偶爾發生，而是常態。做個股的人如果沒看寬度，根本不知道自己在逆風飛行。

2. ADL空方期間硬做個股=白工：FinLab的回測數據很清楚——ADL MA5

3. MCL極端行情的順勢操作：這跟之前學的RSI鈍化概念很像——指標進入極端區不是反手操作的時機，而是順勢加碼的時機。差別在於MCL告訴你極端行情通常還有2-3週，給你一個時間框架參考。

4. 寬度指標+籌碼面的雙重驗證：ADL多方+外資買超是真多頭，ADL空方+外資買超是權值股獨角戲。這種組合判斷比單看任何一個指標都更準確，也跟之前學的法人操作慣性可以對接。

5. 最實用的應用：每天看一眼上漲家數/下跌家數比，就知道今天適不適合做個股。如果上漲家數<下跌家數，即使大盤漲，也別追中小型股。
