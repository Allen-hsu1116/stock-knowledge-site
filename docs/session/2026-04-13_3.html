核心知識

一、Parabolic SAR 拋物線指標

是什麼：由 RSI 開發者 J.W. Wilder 設計的順勢操作系統。SAR = Stop And Reverse（停止並轉向），兼具停損與停利功能。指標以點狀構成拋物線，跟隨價格移動，當價格穿過 SAR 點時即為反轉訊號。

計算公式：

  • SAR（今日） = SAR（昨日）+ AF × [EP（昨日）− SAR（昨日）]
  • EP（區間極值）：上升波段取到昨日為止的最高價；下跌波段取到昨日為止的最低價
  • AF（加速因子）：初始值 0.02，每創一次新高/新低加 0.02，上限 0.2

核心設計原理：像砲彈飛行一樣——SAR 以拋物線加速追趕價格，當上升力道竭盡，價格就會跌破 SAR 值。AF 愈大，SAR 追價愈快，反轉訊號愈早出現。

怎麼做：
1. 趨勢判斷：SAR 點在 K 線下方 → 多頭；SAR 點在 K 線上方 → 空頭
2. 反轉訊號：收盤價跌破下方 SAR → 多翻空；收盤價突破上方 SAR → 空翻多
3. 移動停損：持多單時，SAR 點就是移動停損點，SAR 每天跟著上移
4. 反手操作：反轉當天即可平倉並反手（同一天多空一起操作）
5. 參數調整：AF 步長 0.02（預設）→ SAR 距離適中；0.01 → SAR 離價格遠，反轉慢；0.05 → SAR 貼近價格，反轉快但假訊號多

實例：

  • 某股處於上升波段，SAR = 45，EP = 50，AF = 0.06
  • 隔日 SAR = 45 + 0.06 × (50 − 45) = 45.3
  • 若再創新高 52 → AF 增為 0.08 → SAR = 45.3 + 0.08 × (52 − 45.3) = 45.84
  • 當某天收盤價跌破當日 SAR 值（例如 SAR = 48.5，收盤 47.8）→ 多翻空
  • 反轉日 SAR 值 = 前一波 EP = 52（前波最高價變成上方最大壓力）

適用時機：

  • 趨勢明確的市場（最適合）
  • ⚠️ 盤整震盪市極易出現頻繁反轉假訊號，必須搭配 ADX 或均線過濾
  • 當沖：5 分 K 搭配 SAR 做移動停損
  • 波段：日 K 搭配 SAR 做趨勢追蹤

二、SAR 搭配其他指標實戰

搭配 ADX（趨勢強度）：
1. ADX > 25（趨勢明確）→ 啟用 SAR 做順勢交易
2. ADX < 20（盤整）→ 停用 SAR，改用區間策略
3. 這是最關鍵的搭配，否則盤整市會被 SAR 來回巴

搭配均線：
1. 價格在 20MA 上方 + SAR 在價格下方 → 做多，SAR 當移動停損
2. 價格在 20MA 下方 + SAR 在價格上方 → 做空，SAR 當移動停損
3. 均線確認方向，SAR 確認進出場

搭配 MACD：
1. MACD 金叉 + SAR 翻到 K 線下方 → 雙重做多訊號
2. MACD 死叉 + SAR 翻到 K 線上方 → 雙重做空訊號
3. 兩者衝突 → 不進場，等待一致

實例：

  • 台積電（2330）ADX = 32，趨勢明確
  • 價格站上 20MA（960），SAR 翻到 K 線下方（SAR = 950）
  • 進場做多，停損設 SAR = 950
  • 隨後 SAR 每日上移：950 → 955 → 962 → 970
  • 某天收盤 968 跌破 SAR 970 → 多單出場，反手做空

適用時機：SAR 當作移動停損工具（這是它最強的用途），而非獨立進場訊號

三、CMF 佳慶資金流量指標

是什麼：由 Marc Chaikin 開發的資金流向指標，衡量一段時間內的買賣壓力。核心假設：收盤價在當日高低區間上半部 + 成交量放大 = 買壓強；收盤在下半部 + 量放大 = 賣壓強。

計算公式：
1. 資金流量乘數（CLV） = [(Close − Low) − (High − Close)] ÷ (High − Low)

  • 收盤在最高價 → CLV = +1

2. 資金流量 = CLV × Volume
3. CMF = n 期資金流量總和 ÷ n 期成交量總和（常用 n = 20 或 21）
4. CMF 值在 +1 到 −1 之間，實際上多在 ±0.3 之內

怎麼做：
1. 零線穿越：CMF 從負轉正 → 買進訊號；CMF 從正轉負 → 賣出訊號
2. 超買超賣：CMF > +0.3 → 超買，可能回檔；CMF < −0.3 → 超賣，可能反彈
3. 趨勢確認：上升趨勢中 CMF 持續 > 0 → 趨勢可靠；CMF 轉負 → 警告
4. 背離訊號：股價創新高但 CMF 沒創新高 → 頂背離，即將反轉
5. 避免假訊號：用 ±0.05 代替零線當閾值，減少頻繁交叉

實例：

  • 某股近 20 日 CMF = +0.15 → 買壓持續
  • 股價從 80 漲到 95，但 CMF 從 +0.2 降到 +0.05 → 背離！量能跟不上漲幅
  • 可能是主力在出貨，散戶在追高 → 準備減碼
  • 反之：股價跌到 60 但 CMF 從 −0.2 升到 −0.05 → 賣壓減弱，可能築底

適用時機：

  • 判斷趨勢是否有資金支撐（量價配合的量化版）
  • 搭配趨勢線突破確認
  • ⚠️ CMF 不考慮跳空缺口，跳空日資料會失真
  • ⚠️ 不建議單獨使用，搭配同系列的 A/D 線或蔡金擺動指標效果更好

五、唐奇安通道 Donchian Channel

是什麼：由「海龜交易法則」之父 Richard Donchian 開發的通道指標，是最簡單的突破交易系統。由三條線組成：上通道 = n 日最高價、下通道 = n 日最低價、中線 =（上通道 + 下通道）/ 2。

計算公式：

  • 上通道 = max（過去 n 日最高價）
  • 下通道 = min（過去 n 日最低價）
  • 中線 =（上通道 + 下通道）/ 2
  • 預設 n = 20（海龜交易法則原始參數）

怎麼做：
1. 突破做多：價格突破上通道 → 做多，停損設中線或下通道
2. 突破做空：價格跌破下通道 → 做空，停損設中線或上通道
3. 趨勢判斷：價格在中線上方震盪 → 偏多；中線下方 → 偏空
4. 波動性觀察：通道變寬 → 波動加大；通道變窄 → 盤整壓縮，即將突破
5. 海龜加碼法：突破上通道後每上漲 0.5 倍 ATR 加碼一次，最多加 4 次

實例：

  • 某股過去 20 日最高 120、最低 105
  • 上通道 = 120，下通道 = 105，中線 = 112.5
  • 當天最高價 121 → 突破上通道 → 做多
  • 停損設中線 112.5（風險 8.5 元，約 7%）
  • 若持續上漲，通道上移 → 移動停損跟著下通道上移

海龜交易法則的完整參數：

  • 系統一：20 日突破進場，10 日反向突破出場
  • 系統二：55 日突破進場，20 日反向突破出場
  • 停損：進場後 2 倍 ATR 以下出場
  • 加碼：每上漲 0.5 倍 ATR 加碼，最多 4 單位

適用時機：

  • 趨勢明確的市場（突破系統的共通特性）
  • ⚠️ 盤整市會頻繁假突破，勝率偏低（海龜法勝率僅約 30-40%）
  • 中長期波段 > 當沖（20 日參數本來就不是為日內設計）
  • 搭配 ATR 停損和 ADX 過濾盤整

六、唐奇安通道進階：雙系統與過濾器

雙系統策略：
1. 短周期系統（20/10）：靈敏，抓短期趨勢，但假突破多
2. 長周期系統（55/20）：穩定，抓大波段，但進場晚
3. 實戰：兩個系統同時運行，短系統捕捉小波段，長系統捕捉大趨勢
4. 資金分配：短系統 40% + 長系統 60%

過濾假突破：
1. 量能確認：突破上通道當日成交量 > 20 日均量 1.5 倍 → 真突破
2. ADX 過濾：ADX > 25 才啟用突破系統，ADX < 20 停用
3. 收盤確認：不以盤中突破為準，等收盤價確認突破才進場
4. N 日確認法：突破後連續 2-3 日站穩通道上方才進場（減少假突破但進場較晚）

實例：

  • 聯發科（2454）20 日最高 = 1350，ADX = 28
  • 盤中突破 1350 到 1365，成交量 1.8 倍均量
  • 等收盤確認：收盤 1358 > 1350 → 真突破
  • 進場做多 1358，停損設 2 倍 ATR（假設 ATR = 25）→ 停損 1358 − 50 = 1308
  • 目標：持有至跌破 10 日最低價出場

適用時機：所有突破交易策略的底層邏輯，簡單但需要嚴格的資金管理

心得

這回合學了三個指標，剛好涵蓋三個面向：SAR 是「時間+價格」的趨勢追蹤系統、CMF 是「量價關係」的資金流向判斷、唐奇安通道是「價格區間」的突破系統。

SAR 最有趣的地方在於它的「加速因子」設計——AF 只會增大不會縮小，這意味著 SAR 會愈來愈急迫地追趕價格，就像砲彈的拋物線軌跡。這設計太天才了：趨勢剛啟動時 AF 小，SAR 給價格充分的空間發展；趨勢成熟後 AF 變大，SAR 快速收斂，鎖定利潤。但缺點也很明顯——盤整市會被來回巴到懷疑人生，所以 ADX 過濾是必須的。SAR 最實用的定位其實是「移動停損」，而不是獨立進場訊號。

CMF 填補了我之前學 OBV 的不足。OBV 只看收盤價漲跌來累加成交量，但 CMF 更精細——它看的是收盤價在當日高低區間的「位置」。同樣是上漲日，收在最高價和收在區間中間，OBV 看不出差異，但 CMF 能分辨。不過 CMF 的致命缺陷是不處理跳空缺口，在台股這種常常跳空開高的市場，使用時要特別小心。

唐奇安通道的本質就是一個問答：「過去 N 日的價格範圍是多少？突破了沒有？」就這麼簡單。但海龜交易法則用這個最簡單的系統，搭配嚴格的資金管理（1-2% 風險、0.5 ATR 加碼、2 ATR 停損），在 1980 年代創造了驚人的績效。這再一次證明：交易系統不需要複雜，需要的是紀律和資金管理。勝率低沒關係（30-40%），只要賺的時候賺夠多，虧的時候虧得少，長期就是正期望值。

三個指標有一個共通點：單獨用都有嚴重缺陷，必須搭配其他工具。SAR 配 ADX，CMF 配趨勢線，唐奇安通道配 ATR 和 ADX。指標是武器，但沒有一把武器是萬能的。
