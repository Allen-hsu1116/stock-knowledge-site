地雷股的真相

是什麼：財報可能被美化，光看財報數字看不出問題。地雷股在出事前幾年，公司治理問題已經變得愈來愈嚴重。

關鍵人物：控制股東（大股東、董事長或兼總經理）的誠信問題是核心。

葉銀華教授的「公司治理9大警訊指標」

能夠提早兩年以上發現財報警訊：

1. 控制股東直接持股比率持續下降或直接持股太低

  • 大股東一直賣股票，對公司沒信心
  • 持股太低 = 利益與公司綁得不夠深

怎麼判斷：

  • 查年報中的「股權分散表」
  • 追蹤董監持股變化（公開資訊觀測站）
  • 連續幾季下降就要警戒

2. 董事會趨於家族與內部化

  • 獨立董事被邊緣化或全部辭職
  • 董事會變成「自己人」的俱樂部
  • 控制股東權力過大，缺乏制衡

怎麼判斷：

  • 查董事會成員名單
  • 獨董是否超過1/3？有沒有專業背景？
  • 獨董突然全部辭職 = 立刻逃命訊號

3. 大股東、機構投資人持續賣股票

  • 內部人比外人更知道公司問題
  • 機構法人撤守通常有原因

怎麼判斷：

  • 追蹤法人買賣超（每週公佈）
  • 大股東申報轉讓（公開資訊觀測站）

4. 董事、監察人、財務主管多人辭職與更換會計師

  • 尤其4月更換會計師要特別小心（財報簽核前）
  • 關鍵人員離職 = 看到問題不想背鍋

怎麼判斷：

  • 查「重大訊息」中的董監辭任公告
  • 會計師變動公告（注意原因）

5. 無效率的轉投資與設立許多投資公司

  • 設立一堆紙上公司
  • 轉投資項目看不懂、與本業無關
  • 可能是掏空資產的管道

怎麼判斷：

  • 查財報的「採權益法之長期股權投資」
  • 子公司、轉投資公司數量異常多
  • 轉投資損益與本業不成比例

6. 存在許多重大且異常的關係人交易

  • 買貴賣便宜給關係企業
  • 關係人交易 = 利益輸送的管道

怎麼判斷：

  • 查財報附註的「關係人交易」
  • 交易金額異常大、頻率異常高
  • 交易條件是否合理？（價格、利率、期間）

7. 控制股東與董事會股權質押比率過高

  • 質押股票換現金 = 大股東缺錢
  • 質押比超過40%，有一半會變地雷股

怎麼判斷：

  • 查「董監事質押情形」（公開資訊觀測站）
  • 質押比 = 質押股數 / 持股數
  • 超過30%就要警戒，超過40%要避開

8. 公司經常發布重訊澄清媒體報導

  • 澄清太頻繁 = 媒體一直挖到問題
  • 海外子公司或控制股東有債信問題

怎麼判斷：

  • 追蹤重大訊息公告頻率
  • 注意澄清的內容是否避重就輕
  • 海外子公司狀況（是否財務不透明）

9. 虛增盈餘、激進認列營收與假造財報的傾向

  • 營收成長與同業差異大
  • 應收帳款激增但現金沒進來
  • 存貨異常增加

怎麼判斷：

  • 營收成長率 vs 同業平均
  • 應收帳款週轉率惡化
  • 存貨週轉率惡化
  • 現金流量表與損益表矛盾

雷浩斯的「三個必看指標」

更簡單的判斷方法，三個都符合 = 避開：

1. 獲利持續轉差

是什麼：獲利連續下滑，可能趁機「洗大澡」（一次認列大額資產減損）

怎麼判斷：

  • 連續四季EPS下滑
  • ROE、ROA連續下降
  • 公司趁機認列巨額減損（如TPK一次認列55元資產減損）

實例：

  • 某公司連續四季EPS為：2.5→1.8→0.9→-3.2（負債加上洗大澡）
  • 警訊：不要期待「壞消息出盡」，可能還有

2. 長短期金融負債比超過45%

是什麼：向銀行借款佔總資產比例過高，財務結構脆弱

怎麼算：

長短期金融負債比 = (短期借款 + 長期借款 + 應付公司債) / 總資產

判斷標準：

  • 低於30%：安全
  • 30-45%：警戒
  • 超過45%：危險，要避開

注意：金融業、租賃業不適用此指標（產業特性）

實例：

  • A公司總資產100億，短期借款20億，長期借款30億
  • 金融負債比 = (20+30)/100 = 50% → 危險

3. 董監事質押比高

是什麼：董監事拿股票去抵押借款

官方說法：「董監事個人理財」
真相：「董監事個人理財有問題，或資金用途有問題」

怎麼算：

董監事質押比 = 質押股數 / 董監事持股數

判斷標準：

  • 0%：最安全
  • 超過30%：警戒
  • 超過40%：有一半會變地雷股，避開
  • 超過80-90%：直接避開

為什麼危險：

  • 董監事和你站在不同邊，利益不一致
  • 可能拿借到的錢去炒作股票
  • 股價下跌會被斷頭，引發骨牌效應

實例（光洋科）：

  • 獲利持續轉差 ✓
  • 金融負債比超過45% ✓
  • 董監事質押比89%以上 ✓
  • 結果：坦誠作假帳

心得

「財報漂亮，也可能是地雷股」這句話是核心。研究20年的教授都說，光看財報數字看不出問題，因為「數字始終來自於人性」，財報可以被美化。

葉銀華教授的9大警訊最有價值的地方在於：這些都是「事前」徵兆，能提早兩年以上發現問題。不是等新聞爆出來才跑，而是在公司治理出問題的階段就能察覺。

雷浩斯的三指標更實用：獲利轉差 + 金融負債比 > 45% + 董監質押比高，三個都符合，直接避開。這是懶人版的「避雷針」。

最值得記住的數字：董監質押比超過40%，有一半會變地雷股。這不是機率問題，是生存問題。投資市場不需要去賭那50%的好運氣。

核心原則：董監事如果跟你站在不同邊（質押股票、持股下降、套利輸送），就不要期待他們會幫你賺錢。他們可能正在想辦法掏空公司。
