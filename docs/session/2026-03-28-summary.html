📊 今日核心知識提煉

1. 財報三大表實戰閱讀

核心概念:

  • 三大報表關係：損益表說故事（赚多少）→ 現金流量表驗證（真的收到錢嗎）→ 資產負債表盤點（剩下什麼）
  • 閱讀順序：損益表（獲利能力）→ 現金流量表（獲利品質）→ 資產負債表（財務結構）

實戰要點:

報表關鍵指標警訊
資產負債表流動比率>2、負債比率<60%存貨週轉率異常
損益表毛利率穩定、業外<20%獲利品質低
現金流量表營業現金流>稅後淨利應收帳款未收回

3. 技術分析指標綜合

OBV 能量潮指標:

  • 運用：價漲量增、價跌量縮時上升，代表資金流入
  • 背離：價格創新高但 OBV 未創高 → 出貨訊號

ATR 停損停利:

  • 動態調整停損距離 = ATR × 倍數（常用 1.5-2 倍）
  • 優點：反映當下波動度，高波動時自動放寬停損

KD 指標:

  • 黃金交叉：K 值由下往上穿越 D 值
  • 死亡交叉：K 值由上往下穿越 D 值
  • 高檔鈍化（K>80）：可能續涨也可能反轉
  • 低檔鈍化（K<20）：可能續跌也可能反轉

MACD 指標:

  • 黃金交叉：DIF 由下往上穿越 MACD
  • 死亡交叉：DIF 由上往下穿越 MACD
  • 柱狀體：紅柱縮小→多頭力道減弱，綠柱縮小→空頭力道減弱

布林通道:

  • 上軌 = 中軌 + 2×標準差
  • 下軌 = 中軌 - 2×標準差
  • 突破上軌 → 強勢、跌破下軌 → 弱勢
  • 通道收斂 → 變盤前兆

4. 波浪理論核心

八波結構: 上升五波（1-5）+ 下降三波（A-C）

波浪三鐵律:
1. 波浪2不跌破波浪1起點
2. 波浪3不是最短的
3. 波浪4不跌破波浪1高點

邪惡第五波: 第五波可能比第三波還長，隨後反轉劇烈

5. 選擇權策略

Iron Condor 鐵兀鷹:

  • 同時賣出 CALL 價差 + 賣出 PUT 價差
  • 不預測方向，賭股價在區間內震盪
  • 獲利來源：Theta 時間價值衰減
  • 風險管理：虧損達權利金 2 倍停損

調整策略:

  • Delta 中立調整：用期貨或選擇權調整 Delta 接近 0
  • 移倉調整：將虧損部位移到更遠月份

6. 總體經濟指標

經濟數據影響股市:

數據好消息好消息對股市的影響
GDP 強經濟好可能延後降息→股市跌
GDP 弱經濟差可能加速降息→股市漲

關鍵指標:

  • 領先指標（建築許可、ISM 新訂單）：預測 6-12 個月
  • 同時指標（工業生產、就業）：反映當下
  • 落後指標（失業時間、消費者物價）：確認趨勢

📈 技術分析補強

威科夫方法

  • 四階段：累積（主力吃貨）→ 標記（主力拉抬）→ 分配（主力出貨）→ 標記下跌

樞軸點（Pivot Points）

  • 計算：P = (H+L+C)/3
  • 支撐：S1 = 2P - H、S2 = P - (H-L)
  • 壓力：R1 = 2P - L、R2 = P + (H-L)
