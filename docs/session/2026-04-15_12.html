籌碼技術面整合進出場策略

一、為什麼要整合籌碼面與技術面？

核心問題：單看籌碼面或單看技術面，都會有盲區。

  • 只看籌碼面：盤後資料，你知道的時候主力可能已經跑了；匯撥、隔日沖可以讓籌碼數據失真
  • 只看技術面：技術訊號可能被主力操縱（騙線、試撮），不知道背後是誰在買誰在賣
  • 整合使用：籌碼告訴你「誰在動」，技術面告訴你「動到哪了」，兩者交叉驗證，勝率大幅提升

比喻：基本面是主人的方向，技術面是小狗跑的路線，籌碼面是主人拉牽繩的力道。三個一起看，才能判斷小狗接下來往哪跑。

二、主力投信同調 vs 不同調 — 郭勝實戰心法

是什麼

「市井股神」郭勝的核心方法：觀察主力買賣超與投信買賣超是否同步，判斷個股後勢方向。

怎麼做

同調情況（主力 + 投信同步）：
1. 兩者同步買超 → 後市明確向上，可進場
2. 兩者同步賣超 → 後市明確向下，應避開或放空

不同調情況（主力 + 投信反向）：
1. 高檔主力賣、投信買 → 主力倒貨給法人，危險！跌破前波低點就出清
2. 低檔主力買、投信賣 → 主力接走投信停損籌碼，通常底部不遠，可留意

實例

  • 情境：中興電（1513）漲多後，主力開始連續賣超，投信卻仍大買
  • 判讀：主力倒貨給投信的嫌疑，提高警覺，暫不動作
  • 操作：觀察前波低點能否守住，守不住就出場

適用時機

  • 中小型股特別適用（投信主力影響力大）
  • 大型權值股以看外資為主
  • 不適用：成交量極小的冷門股，主力買賣超數據參考性低

三、底底高型態 + 籌碼確認 — 多頭防護策略

是什麼

「底底高」＝每次回檔低點都比前一次高，代表多頭結構完好。搭配籌碼面確認，判斷是否繼續持有或出場。

怎麼做

1. 畫出每次回檔的低點連線，確認是否愈墊愈高
2. 確認千張大戶持股比例是否穩定或上升
3. 確認融資餘額是否未暴增（散戶未大量進場）
4. 三個條件同時成立 → 維持多頭部位
5. 任何一個條件被破壞 → 提高警覺，設好停損

實例

  • 情境：怡利電、康舒等，股價在年線上方不遠，底底高型態
  • 籌碼：大戶持股穩定增加，融資未暴增
  • 操作：逢拉回布局，停損設在前波低點下方

適用時機

  • 中長期波段操作
  • 不適用：盤整無方向市場，底底高型態不明確

四、周線技術指標 + 籌碼驗證 — 陳榮華實戰法

是什麼

技術派達人陳榮華的方法：用周線圖找趨勢方向，用 KD + MACD + DMI 三指標翻多確認進場時機，再用千張大戶持股變化與股息推算目標價交叉驗證。

怎麼做

進場四步驟：
1. 周線圖找前波高低點 → 計算下一波目標區
2. 三指標翻多確認 → KD 黃金交叉 + MACD 柱狀圖翻正 + DMI +DI 向上穿越 -DI
3. 籌碼確認 → 千張大戶持股比例上升 + 散戶比例下降
4. 股息推算合理價 → 預估 EPS × 配息率 ÷ 殖利率 = 合理價/低估價/高估價

出場條件：

  • 股價到達高估價區間（目標價滿足）
  • 技術指標轉空（KD 死亡交叉 + MACD 翻黑）
  • 籌碼鬆動（大戶開始減碼）

實例

  • 情境：台半（3139），預估 EPS 6.77 元，配息率 70%
  • 計算：平均配息 4.57 元
  • 籌碼：千張大戶從 37.5% 升到 51.4%，散戶從 27.4% 降至 14.7%
  • 結論：三個面都指向 139-146 元為目標區，90 元以下為便宜價

適用時機

  • 波段操作（持有期間數週至數月）
  • 不適用：當沖或極短線，周線反應太慢

五、籌碼面常見誤區 — 避開陷阱

1. 法人買超 ≠ 股價必漲

  • 陷阱：看到外資大買就追進，結果外資是避險佈局
  • 怎麼判斷：看外資買超同時是否在期貨佈空單，若是避險就不會推升現貨
  • 對策：搭配期貨未平倉量一起看

2. 主力連買 ≠ 沒有風險

  • 陷阱：主力連續買超 5 天，但可能是「邊拉邊出」或「對倒」
  • 怎麼判斷：主力買超但股價不漲反跌 = 出貨訊號
  • 對策：籌碼 + 量價交叉驗證，買超要搭配價漲量增才有意義

3. 籌碼集中 ≠ 絕對利多

  • 陷阱：大戶持股 80% 以上，股價卻不動如山
  • 原因：籌碼太集中反而缺乏流動性，沒人願意追高
  • 對策：適度分散（60-70% 大戶持有）比過度集中更健康

4. 只看籌碼不看技術圖形

  • 陷阱：主力連買但 K 線破底，相信籌碼忽略技術面
  • 結果：主力可能已經開始用匯撥轉移籌碼，數據失真
  • 對策：籌碼與技術面背離時，以技術面為準，提高警覺

5. 高融券 ≠ 一定看空

  • 陷阱：融券大增就放空，結果被軋空
  • 判斷：融券大增 + 股價不跌反漲 = 軋空行情啟動
  • 對策：看券資比 + 股價走勢，股價在均線上方 + 高融券 = 潛在軋空標的

心得

學了這麼多籌碼面的主題，今天終於把籌碼和技術面「怎麼搭在一起用」講清楚了。

最核心的觀念其實很簡單：籌碼告訴你「誰在動」，技術面告訴你「動到哪了」。只看一面就像瞎子摸象，摸到腿說是柱子，摸到尾巴說是繩子。郭勝的「主力投信同調/不同調」判斷法很實用，尤其高檔主力賣投信買這個訊號，散戶最容易被騙——因為投信大買的新聞會讓你覺得安全，但主力才是真正推動股價的力量。

陳榮華的周線三指標 + 籌碼驗證法也很紮實，把目標價用三個面向交叉算出來，當三個方法算出的數字接近時，準確度最高。這比單靠一個指標進出場穩多了。

最後那五個誤區真的要刻在腦子裡，尤其是「法人買超 ≠ 股價必漲」和「只看籌碼不看技術圖形」，這兩個是最常犯的錯。籌碼是落後指標，等你看到的時候主力可能已經換了下一齣戲。所以一定要搭配技術面的即時訊號一起判斷。

一、籌碼集中度六指標完整解析

什麼是籌碼集中度

是什麼：籌碼集中度衡量的是一段時間內，主力籌碼的買賣力道佔成交量的比例。集中度越高，代表籌碼越集中在少數人（主力）手上，股價越容易出現大行情。

公式：

籌碼集中度(%) = (買超前15名合計 − 賣超前15名合計) ÷ 區間成交量 × 100%

核心觀念：主力買超張數如果只佔成交量的一小部分，就算持續買進，對股價的影響力也有限。所以必須看「佔成交量的比例」，而不是只看絕對張數。

六指標分類：絕對指標 vs 相對指標

怎麼做：籌碼面分析可分為三大絕對指標和三大相對指標，六個搭配使用才能完整判斷。

絕對指標一：千張大戶持股比率

是什麼：持有 1000 張以上股東的持股合計 ÷ 股本（張數）。

判斷標準：

  • 40%~70%：黃金區間，最容易拉抬
  • < 40%：籌碼分散，不易上漲
  • > 70%：過度集中，流動性不足
  • > 80%：極度集中，操控風險高

怎麼用：
1. 每週六查集保股權分散表
2. 觀察千張大戶持股比率的「連續變化趨勢」
3. 連續 2~4 週大戶增加 + 散戶減少 → 偏多
4. 連續 2~4 週大戶減少 + 散戶增加 → 偏空

實例：

  • 聯電（2303）2020/5~6月：千張大戶從 56.3% 上升到 58.1%，同期股價從 15.55 元漲到 22.35 元
  • 中鋼（2002）：千張大戶持股 58.31%，零股持有者僅 0.33%，籌碼高度集中

絕對指標二：內部人持股比率

是什麼：董事、監察人、經理人、大股東及其關係人的持股合計 ÷ 股本。

判斷標準：

  • > 40%：高水位，欠缺人為炒作動機，但公司有強烈動機好好經營
  • < 20%：低水位，可能缺乏公司派支撐

怎麼用：
1. 查詢個股內部人持股（公開資訊觀測站）
2. 內部人持股增加 → 公司派看好自家前景
3. 搭配「內部大戶理論」：集保大戶增減 − 三大法人增減 = 內部大戶增減

注意：內部人持股高不代表股價會漲——可能是公司派不願釋出籌碼，導致流動性不足。

絕對指標三：區間買賣超佔股本比重

是什麼：(區間買超前15名合計 − 區間賣超前15名合計) ÷ 股本(張數)。

判斷標準：

  • 60 天內佔股本 > 3%：籌碼明顯集中到特定券商
  • 對大型股：3% 的股本是相當大的金額，代表大規模佈局
  • 對中小型股：3% 可能只是幾千萬，構成炒作動機有限

怎麼用：
1. 先看 60 天和 120 天的佔股本比重
2. 大型股 > 3% = 嚴肅看待
3. 中小型股要換算市值，超過 1 億以上才有炒作動機
4. 佔股本比重持續上升 → 主力持續佈局

相對指標一：籌碼集中度(%)

是什麼：(買超前15名合計 − 賣超前15名合計) ÷ 區間成交量 × 100%。

判斷標準（依天期不同）：

天期集中度門檻意義
1 天> 20%可能有特定人在收集籌碼
5 天> 10%短線籌碼集中
60 天> 5%中期籌碼明顯集中
120 天> 3%長期籌碼有集中

怎麼用：
1. 初次接觸一檔股票，先看 60 天和 120 天的集中度
2. 短線操作者看 1、3、5 天，天期越短門檻越高
3. 5 日集中度 > 20 日集中度 → 主力短期買進力道大於長期，表示「近期加速買超」
4. 120 天集中度太高要小心：可能主力出不了貨，你也出不了貨

實例：

  • 中鼎（9933）2022/2/14 之後：5 日籌碼集中度 > 20 日籌碼集中度，主力近期加速買超，後續股價上漲約 20%
  • 聯電 2020/5~6月：籌碼集中度從賣方（綠色）轉到買方（紅色），最高超過 20%，主力買超張數達萬張以上

相對指標二：主力買賣超

是什麼：每日買超前 15 名合計 − 每日賣超前 15 名合計。

怎麼用：
1. 紅色（正數）= 主力當日偏買；綠色（負數）= 主力當日偏賣
2. 與股價表現呈正相關：股價漲時買超 > 賣超，股價跌時賣超 > 買超
3. 但中小型股有陷阱：股價一路漲，主力買賣超卻一直呈現賣超 → 主力用甜美漲勢誘多出貨
4. 必須搭配「買賣家數差」一起看才有意義

關鍵陷阱：

  • 主力買賣超連續正數 + 股價上漲 = 看起來很好
  • 但如果賣出的都是「贏家」（績效好的券商），可能是在獲利了結，後勢未必樂觀
  • 反過來：買進的都是贏家，後勢可期

相對指標三：買賣家數差

是什麼：當日有買進該股票的券商家數 − 當日有賣出該股票的券商家數。

怎麼用：
1. 負數（綠色）= 賣家數 > 買家數 = 籌碼往少數人集中
2. 正數（紅色）= 買家數 > 賣家數 = 籌碼往多數人分散
3. 最佳組合：主力買賣超連續正數 + 買賣家數差連續負數 = 少數主力正在集中買進籌碼
4. 危險組合：主力買賣超連續負數 + 買賣家數差連續正數 = 多數散戶在接，少數主力在出

進場三條件：
1. 股價緩漲，區間漲幅不超過 20%
2. 主力買賣超連續呈現正數（連續好幾天買超）
3. 買賣家數差連續呈現負數（連續好幾天賣家 > 買家）

注意：漲幅超過 20% 後，上述條件參考價值下降，因為漲多自然吸引更多交易者加入，籌碼分布變複雜。

二、買盤贏家/輸家判讀——籌碼K線進階技巧

贏家與輸家是什麼

是什麼：在籌碼K線中，每個券商分點都有歷史操作績效。「贏家」是在這檔股票上反覆操作都能獲利的分點，「輸家」則是長期虧損的分點。

怎麼判斷：
1. 進入籌碼K線個股頁面
2. 點擊分點進出資料
3. 觀察買方和賣方的分點，看哪些是贏家、哪些是輸家
4. 買盤贏家多 = 進場的資金來自績效好的分點，後勢可期
5. 賣盤贏家多 = 獲利出清，這些績效好的分點在賣出，要警戒

贏家/輸家判讀四種情境

情境買方賣方解讀操作
1贏家多輸家多好分點買、差分點賣 → 強力偏多可進場
2輸家多贏家多差分點買、好分點賣 → 強力偏空減碼或出場
3贏家多贏家多好分點對做 → 方向不明觀望，看誰量大
4輸家多輸家多差分點互相交易 → 意義不大不宜以此判斷

怎麼做：
1. 盤後查看個股分點進出
2. 識別買方和賣方的贏家/輸家比例
3. 情境 1（買盤贏家多）是最佳進場訊號
4. 情境 2（賣盤贏家多）要特別小心——績效好的分點在出場，他們通常比散戶更早看到反轉

實例：

  • 某股某日盤後資料：買超前 3 名都是歷史績效佳的贏家分點，賣超前 3 名都是輸家
  • 判斷：贏家進場、輸家退場 → 偏多訊號
  • 反之：如果買超第一名是輸家分點，賣超第一名是贏家分點 → 偏空訊號

三、籌碼K線四個獲勝公式整合

公式一：主力成本線 + 趨勢軌道

怎麼做：
1. 畫出主力成本線（40 日主力加權平均成本）
2. 疊加趨勢軌道（股價運行通道）
3. 股價在成本線之上 + 趨勢軌道向上 → 偏多
4. 股價跌破成本線 + 趨勢軌道下彎 → 偏空

公式二：散戶買賣超 + 主力買賣超

怎麼做：
1. 同時觀察散戶和主力的買賣超
2. 散戶賣超 + 主力買超 → 籌碼從弱手流向強手，偏多
3. 散戶買超 + 主力賣超 → 籌碼從強手流向弱手，偏空

公式三：籌碼多空趨勢 + 多空趨勢線

怎麼做：
1. 畫出多空趨勢線
2. 疊加籌碼多空趨勢
3. 趨勢向上 + 籌碼趨勢同向 → 可以追
4. 趨勢向上 + 籌碼趨勢反向 → 不要追，籌碼面不支撐

核心問題：「趨勢出來了但股價也漲多了，可以追嗎？」→ 看籌碼多空趨勢是否同向，同向才可以追。

公式四：權證多空 + 可轉債轉換

怎麼做：
1. 權證多空：股價暴漲暴跌時，留意是否因權證市場交易行為觸發（隔日沖居多）
2. 可轉債轉換：股價接近轉換價或償還日時，留意異常轉換是否為套利行為
3. 期貨大戶：留意個股期貨的十大交易人淨部位是否異常

四、籌碼集中度過高的陷阱

一年集中度太高，不要參與！

是什麼：120 天籌碼集中度如果長期處於極高水準（如 > 15%），可能有兩種情況：

1. 沒有下文：主力出不了貨，你進去了也出不了——流動性陷阱
2. 公司在進行不為人知的事情：如併購、經營權之爭，買了當大股東是不會賣的

怎麼判斷：

  • 120 天集中度 > 10% + 成交量萎縮 → 流動性陷阱，不要碰
  • 120 天集中度 > 10% + 成交量放大 + 有消息面 → 可能是併購或經營權之爭

區間周轉率搭配：

  • 60 天區間周轉率 > 50%：主力在做激情行情，誘騙散戶進場
  • 如果你看到不買，隔天可能跳空漲——但追進去可能就是高點
  • 正確做法：列入觀察，等回測支撐再進場

五、實戰操作流程：籌碼集中度選股 SOP

Step 1：初次篩選

1. 用籌碼選股功能，篩選 60 天籌碼集中度 > 5%、120 天 > 3% 的股票
2. 排除年集中度過高（> 10%）的股票
3. 確認成交量足夠（日均量 > 500 張）

Step 2：確認三指標

1. 主力買賣超連續正數（至少 3 天以上）
2. 買賣家數差連續負數（籌碼往少數人集中）
3. 5 日集中度 > 20 日集中度（近期加速買超）

Step 3：驗證3：驗證贏家/輸家

1. 打開分點進出資料
2. 看買方是否贏家多、賣方是否輸家多
3. 贏家買 + 輸家賣 = 強力偏多訊號

Step 4：搭配技術面進場

1. 確認股價漲幅 < 20%（超過 20% 參考價值下降）
2. 找支撐位置（均線、前低、主力成本線）
3. 在支撐附近進場，停損設支撐下方 3~5%

Step 5：出場判斷

1. 賣盤出現贏家分點 → 獲利了結訊號
2. 主力買賣超由正轉負 → 主力轉向
3. 5 日集中度 < 20 日集中度 → 近期買超力道減弱
4. 股價跌破主力成本線 → 出場

心得

這回合把「籌碼集中度」六指標從概念到實戰串起來了。之前學過千張大戶持股、主力買賣超、買賣家數差這些個別指標，但一直沒搞清楚它們之間的「相對 vs 絕對」關係和搭配邏輯。現在才明白：絕對指標看的是「籌碼結構」（誰持有多少），相對指標看的是「籌碼流向」（最近在往哪裡走），兩者缺一不可。

最實用的收穫是「5 日集中度 > 20 日集中度」這個判斷。以前只看籌碼集中度的絕對值，不知道短期和長期的相對關係這麼重要。5 日 > 20 日代表主力「近期」在加速買超，這比長期慢慢買的訊號更強烈、更即時。中鼎那個案例就很清楚：2/14 之後 5 日集中度一路超過 20 日，後續漲了 20%。

買盤贏家/輸家是這回合最讓我興奮的概念。以前看分點進出只知道「誰買得多、誰賣得多」，不知道還可以追蹤「這個分點歷史上在這檔股票是賺錢還是賠是賺錢還是賠錢」。贏家買 + 輸家賣 = 最強的偏多訊號，比單純看主力買賣超可靠得多。因為主力買賷超只告訴你「前 15 名合計是買是賣」，但贏家/輸家告訴你「進場的是高手還是菜鳥」。

最後，120 天集中度過高是個很容易被忽略的陷阱。看到集中度高就覺得「主力在佈局」？不一定。集中度太高可能代表籌碼被鎖死，沒人願意交易，你進去了也賣不掉。這跟之前學到的「千張大戶 > 80% 過度集中」風險一致——集中度要有，但不能太高，40%~70% 才是黃金區間。

一、外資期貨與現貨同步判斷

外資為什麼要操作期貨？

是什麼：外資在台股市場是最大的法人參與者，佔台股成交比重超過5成。外資不只買賣現股（現貨），也大量操作台指期貨和選擇權。期貨部位是外資的「底牌」——看多看空，先在期貨佈局，再到現貨市場執行。

為什麼外資要同時操作現貨和期貨：
1. 避險需求：外資持有大量台股現貨，為了防止下跌造成損失，會在期貨市場建立空單避險。現貨跌→期貨空單賺，兩邊對沖。
2. 降低衝擊成本：外資要賣出大量權值股時，直接賣會越賣越低。所以先在期貨建立空單，再慢慢賣現貨，期貨獲利彌補現貨虧損。
3. 方向性佈局：外資看好台股時，會先在期貨建立多單（因為期貨槓桿高、流動性好、進出快），確認方向後再買現貨。

期貨現貨四種同步組合判斷

怎麼判斷：

現貨期貨含義操作建議
買超多單增加看多訊號，外資全面做多偏多操作，留意進場點
買超空單增加避險訊號，外資買現貨但怕跌中性偏多，但要有防守準備
賣超多單增加矛盾訊號，可能小外資在做短线觀察為主，不急著跟單
賣超空單增加看空訊號，外資全面做空偏空操作或空手觀望

最關鍵的判斷：

  • 現貨買超 + 期貨空單增加 ≠ 看空，而是避險！外資持有大量現股，怕短期回檔所以用期貨空單避險
  • 現貨賣超 + 期貨空單增加 = 真正看空，外資全面撤離
  • 現貨買超 + 期貨多單增加 = 強烈看多，外資全力做多
  • 現貨賣超 + 期貨多單增加 = 矛盾，可能是小外資短线操作

實例：

  • 2021/11 外資期貨空單從 15,579 口增加到 19,767 口，同期現貨持續賣超 → 全面看空訊號，後續大盤下跌
  • 2024/04/19 外資單日賣超 857 億（歷史次高），同日期貨空單也大增 → 確認看空，大盤暴跌 774 點

外資期貨未平倉量的正確解讀

是什麼：期貨未平倉量（OI）代表市場上還沒有平倉的契約數量。每一口多單一定對應一口空單，所以多頭OI = 空頭OI。

怎麼看外資期貨OI：
1. 淨額比絕對值重要：外資期貨多空淨額的「變化方向」比「絕對數字」更有參考價值
2. 連續3-5日同方向才值得參考：單日變動可能是調整部位，連續多日同方向才代表趨勢
3. 結算日效應：每月第三個週三是台指期結算日，結算前OI會大幅減少，這是正常現象
4. 正逆價差：台指期大部分時間呈逆價差（期貨價格 < 現貨價格），尤其在除權息季節更明顯，不代表偏空

常見誤區：

  • 看到外資期貨空單增加就恐慌 → 可能只是避險，不代表看空
  • 看到外資期貨空單減少就以為偏多 → 可能只是到期結算，不是主動回補
  • 只看外資總量不拆大小外資 → 大外資和小外資操作邏輯完全不同

二、大外資 vs 小外資：拆解外資期貨數據

為什麼要拆大外資和小外資

是什麼：「外資」不是一個人，而是一群機構的加總。裡面有大外資（摩根大通、高盛等大型機構）和小外資（規模較小的外資或假外資）。兩者操作邏輯完全不同。

大外資特性：

  • 資金部位超級大（數百億到上千億）
  • 操作週期很長（以季甚至年為單位）
  • 部位笨重，不會頻繁調整
  • 期貨空單多半是避險部位
  • 偏好大型權值股（台積電、聯發科等）

小外資特性：

  • 資金部位較小（幾十億以內）
  • 操作週期短（以天或週為單位）
  • 部位靈活，頻繁調整
  • 期貨部位可能是方向性操作
  • 可能操作中小型股（甚至假外資）

怎麼拆：

  • 外資總量 − 五大特定法人 = 小外資
  • 五大特法（身份碼L）包含國安基金、退撫基金、勞退基金、勞保基金、郵政儲金
  • 大外資看長線趨勢，小外資看短線方向

實戰判斷：
1. 大外資持續做多 + 小外資也做多 → 強烈偏多訊號
2. 大外資做多（長線避險空單）+ 小外資做空 → 可能是短线波動，中長線仍偏多
3. 大外資轉空 + 小外資也空 → 強烈偏空訊號，要警覺
4. 大外資空單增加但小外資多單增加 → 觀望，可能是避險部位調整

三、籌碼七指標選股法（CMoney 珍妮實戰）

七大籌碼特徵同時出現 = 準備發動

是什麼：當一檔股票同時出現7大籌碼特徵，代表主力已經完成佈局，準備發動行情。這7個指標從不同角度確認籌碼流向，同時滿足越多項，上漲機率越高。

指標1：券商重押股（重押比例 > 30%）

是什麼：某間券商一天內急買特定股票，買超張數佔當天成交量超過30%。

怎麼用：

  • 重押比例 = 單一券商買超張數 ÷ 當日成交量 × 100%
  • 重押比例 > 30% = 有人極度看好後勢，急著搶貨
  • 可能掌握外人不知道的情報（營收利多、接獲大單等）

實例：

  • 聯詠（3034）：港商麥格理重押 1,081 張，佔成交量 32%，4天漲 9.56%

指標2：TOP 1 買超異常（超額買超 > 4倍）

是什麼：買最多的券商買超量，是賣最多的券商賣超量的4倍以上。

怎麼用：

  • 超額買超 = TOP 1 買超 ÷ TOP 1 賣超
  • 超額買超 > 4倍 = 買方力道遠大於賣方，買盤強勁
  • 結合量價判斷：超額買超 + 量增紅K = 更強訊號

實例：

  • 中美晶（5483）：元大經紀買超 1,890 張 vs 富邦民生賣超 224 張，超額買超 8.4 倍，3天漲 4.7%

指標3：短線籌碼集中（1日集中度 > 20%）

是什麼：前15名買超券商合計買超量 − 前15名賣超券商合計賣超量 = 籌碼集中。籌碼集中度 = 籌碼集中 ÷ 總成交量。

怎麼用：

  • 籌碼集中度 > 20% = 主力看好程度很高，容易出現波段行情
  • 連續2天以上籌碼集中 = 更可靠的訊號
  • 籌碼集中的同時，籌碼集中度也在增加 = 加速上漲前兆

實例：

  • 超眾（6230）：11/25 籌碼集中度 21.33%，7天大漲 43.57%

指標4：大戶買散戶賣（連3週）

是什麼：集保庫存顯示大戶（千張以上）持股連續3週增加，同時散戶（50張以下）持股連續3週減少。

怎麼用：

  • 籌碼從弱手流向強手 = 上漲趨勢
  • 連續3週大戶增 + 散戶減 = 趨勢確認，不是單週雜訊
  • 搭配技術面均線多頭排列 = 勝率更高

實例：

  • 凌巨（8105）：連3週大戶增散戶減，1個月股價從 12.9 元漲到 24.6 元，漲幅 90.69%

指標5：主力連買（連買 > 3天，買超 > 1000張）

是什麼：主力（前15大券商買超合計）連續買超3天以上，且累計買超超過1000張。

怎麼用：

  • 主力連買 = 有資金實力的人正在佈局
  • 買超 > 1000張 = 不是小打小鬧，是有意圖的大動作
  • 主力連買 + 股價仍在低檔 = 最佳進場時機

實例：

  • 華新科（2492）：主力連買3天，8天漲 26.9%

指標6：外資連買（連買 > 3天，買超 > 1000張）

是什麼：外資連續買超3天以上，且累計買超超過1000張。

怎麼用：

  • 外資三大習性：買大牌（權值股）、買優良股（重基本面）、連續動作（一直買或一直賣）
  • 外資連買3天 = 趨勢可能啟動
  • 搭配股性：外資連買在大型股參考價值高，中小型股可能是假外資

實例：

  • 智邦（2345）：外資連買3天，2個月大漲 65.65%

指標7：投信連買（連買 > 3天，買超 > 1000張）

是什麼：投信連續買超3天以上，且累計買超超過1000張。

怎麼用：

  • 投信偏愛中小型股（股本小、容易拉抬）
  • 投信連買有績效壓力驅動，更容易形成波段
  • 櫃買指數強於加權指數時，投信連買標的勝率更高

實例：

  • 萬潤（6187）：投信連買3天，2天漲 6.17%

七指標綜合判斷

怎麼用：
1. 不需要7個指標全部滿足，滿足越多項勝率越高
2. 優先順序：籌碼集中 > 主力連買 > 大戶買散戶賣 > 外資連買 > 投信連買 > 券商重押 > TOP1買超異常
3. 籌碼集中 + 主力連買 + 大戶買散戶賣 = 三指標同時出現，就是強烈進場訊號
4. 搭配技術面確認：均線多頭排列 + 量增 + 籌碼七指標 = 最強進場點

注意事項：

  • 七指標是「短線選股」工具，進場後仍需設停損
  • 籌碼數據是落後資訊（盤後才公布），隔天開盤可能已有一段漲幅
  • 不要只看單一指標就進場，至少要有3個以上指標同時滿足

四、期貨價量未平倉三角判斷

價格 + 成交量 + 未平倉量 三角關係

是什麼：期貨市場中，價格趨勢的延續需要「燃料」，而未平倉量（OI）就是趨勢的燃料。價格、成交量、未平倉量三者必須一起看，才能判斷趨勢是否健康。

怎麼判斷：

期貨價格成交量未平倉盤面表現判讀
上升增加增加價漲量增OI增**強勢偏多**，新資金進場推升，趨勢健康
上升增加減少價漲量增OI減**偏中性**，多方獲利了結、空方停損出場
上升減少減少價漲量縮OI減**偏中性偏弱**，上漲力道減弱，趨勢可能結束
下跌減少減少價跌量縮OI減**偏中性**，多方停損、空方獲利了結
下跌增加增加價跌量增OI增**強勢偏空**，新空方進場，趨勢加速
下跌增加減少價跌量增OI減**偏中性**，多方停損出場、空方獲利了結

核心原則：

  • 趨勢 + OI增加 = 趨勢有燃料，可持續
  • 趨勢 + OI減少 = 趨勢燃料不足，可能反轉
  • 上升 + OI增加 = 多頭加碼 + 空頭加碼 → 空頭被軋時被迫回補，推升價格
  • 下降 + OI增加 = 空頭加碼 + 多頭承接 → 多頭停損退出，加速下跌

結算日注意：

  • 每月第三個週三是台指期結算日
  • 結算前OI會大幅減少（這是正常換倉，不是趨勢反轉）
  • 結算後重新觀察OI變化才有參考價值
  • 正逆價差在結算日會趨近於零

心得

這回合把外資期貨現貨同步判斷搞清楚了，最大的認知翻轉是：外資期貨空單增加 ≠ 看空。以前看到新聞說「外資期貨空單大增2萬口」就覺得天要塌了，但其實大部分時候外資期貨空單是避險部位——外資持有幾千億台股現貨，不避險才是神經病。真正要看的是「期貨空單增加 + 現貨賣超」這個組合，那才是全面看空。

大外資和小外資的拆解也很關鍵。大外資像大象，轉身慢但力道大；小外資像猴子，靈活但影響短。看外資期貨數據不看大小拆解，就像把大象和猴子混在一起稱體重，數字沒有參考價值。以後看外資期貨數據，要先問：這是大外資的長線避險部位調整，還是小外資的短線方向性操作？

籌碼七指標選股法是這回合最實用的工具。核心邏輯是：當多個籌碼指標同時發出訊號，代表「有人的資金正在集中進場」。但珍妮提醒得很對——籌碼數據是落後的，盤後才公布，所以看到訊號時股價可能已經動了。最好的做法是先用七指標篩選標的，再用技術面找進場點，而不是看到訊號就無腦衝進去。

最後，價量OI三角判斷是把期貨市場的語言翻譯成白話文。上漲 + OI增加 = 多空都在加碼，空頭遲早被軋；上漲 + OI減少 = 多頭在跑了，趨勢快結束。這個邏輯不只適用於期貨，也適用於選擇權的未平倉量分析，跟之前學的P/C Ratio完全串起來了。

一、融資維持率計算與追繳機制

融資維持率是什麼

是什麼：融資維持率 = 股票現值 ÷ 融資金額 × 100%，是衡量融資投資人「還有多少安全邊際」的指標。台股規定，融資維持率低於 130% 就會觸發追繳。

怎麼計算：
1. 融資買進一張 100 元股票，自備款 4 萬、融資 6 萬
2. 買進時維持率 = 10 萬 ÷ 6 萬 = 166.7%
3. 股價跌到 78 元時，維持率 = 7.8 萬 ÷ 6 萬 = 130%（觸發追繳門檻）
4. 股價跌到 70 元時，維持率 = 7 萬 ÷ 6 萬 = 116.7%（已低於 130%）

追繳三種處置：

情境處理方式結果
補到 166% 以上補足保證金到安全水位解除追繳狀態
補到 130%~166%維持率過門檻但未解除追繳日後再低於 130% 當日必須補錢，否則隔日斷頭
仍低於 130%不補或補不夠T+3 開盤強制賣出（斷頭）

實例：

  • 小明融資買進一張 100 元股票，融資 6 萬元
  • 股價跌到 75 元，維持率 = 7.5 萬 ÷ 6 萬 = 125%，收到追繳通知
  • T+1 和 T+2 兩天內必須處理
  • 如果選擇補繳保證金到 166%（即補 2.5 萬以上），就解除追繳
  • 如果不處理，T+3 券商以市價強制賣出

適用時機：使用融資交易必須隨時監控維持率，大跌時要預先準備資金或設好停損

斷頭價反向計算法

是什麼：買進前就算好「跌到多少會被斷頭」，避免意外收到追繳令。

怎麼算：

  • 斷頭價 = 融資金額 × 130% ÷ 持有張數
  • 例：融資買進一張 100 元股票，融資 6 萬
  • 斷頭價 = 6 萬 × 1.3 = 7.8 萬 → 股價 78 元
  • 換句話說，從 100 元跌 22%（到 78 元）就觸發追繳

實例：

  • 某上市股 50 元，融資成數 6 成，融資金額 = 50,000 × 60% = 30,000 元
  • 斷頭價 = 30,000 × 1.3 ÷ 1,000 = 39 元
  • 從 50 元跌到 39 元（跌幅 22%）就觸發追繳
  • 如果同時持有多檔股票，要用「整戶維持率」計算，強勢股可補弱勢股的維持率

適用時機：融資進場前必算；持有融資部位時每日收盤後確認維持率

二、融資斷頭潮與市場系統性風險

融資斷頭潮是什麼

是什麼：當大量投資人在相似價位融資買進同一檔股票，股價大跌後，這些融資戶幾乎同時收到追繳通知。若多數人無法補繳，券商在同一天大量拋售，造成股價不理性暴跌，這就是「斷頭潮」。

發生條件：
1. 個股或大盤融資餘額處於歷史高檔
2. 股價從高點急跌 20% 以上
3. 大量融資戶維持率跌破 130%
4. 券商集中拋售 → 股價再跌 → 更多人斷頭（惡性循環）

實例：

  • 2025/4/7 川普關稅衝擊，台股單日暴跌 2,065.87 點（跌幅 9.7%），創史上最大單日跌幅
  • 大量融資戶同時收到追繳通知，隔天開盤斷頭賣壓湧現
  • 許多個股開盤直接跌停，融資戶被砍在「阿呆谷」（恐慌低點）
  • 隨後幾天市場冷靜，股價反彈，但被斷頭的人已經無法回本

阿呆谷現象

是什麼：斷頭潮時大量股票被市價拋售，賣在恐慌最低點，事後股價反彈，這個低點被稱為「阿呆谷」。

為什麼會發生：
1. 券商 T+3 開盤直接以市價賣出，不會幫你找好價格
2. 當天所有斷頭股同時拋售，買盤嚴重不足
3. 跌停板無法成交的股還會延後賣出，可能賣在更低價

如何避免被砍在阿呆谷：
1. 主動停損：不要等收到追繳令才處理，維持率接近 140% 就應考慮減碼
2. 提前補繳：收到追繳通知後立刻補繳，不要拖到最後一刻
3. 避免集中持股：融資不要 all-in 單一個股，避免整戶維持率同時跌破門檻
4. 設定斷頭價警報：預先算好斷頭價，接近時就先處理

適用時機：大盤急跌時、個股利空暴跌時、融資使用率偏高的股票

三、大盤融資餘額判讀實戰

大盤融資餘額四象限判斷

是什麼：大盤融資餘額代表全市場融資買進的總金額，是觀察散戶情緒與市場槓桿水位的指標。重點不在絕對數字，而在「變化趨勢」與「指數的對應關係」。

四象限判斷法：

情境指數位置融資變化市場意涵操作建議
多頭初期低檔反彈溫和增加健康資金回流持股續抱
多頭末期歷史高檔急遽暴增散戶瘋狂追高減碼、提高警覺
空頭初期高檔回落開始減少信心動搖多殺多風險升高
空頭末期低檔盤整持續減至低水位籌碼清洗乾淨留意落底反彈訊號

怎麼做：
1. 每日觀察大盤融資餘額變化（證交所官網、看盤軟體、MacroMicro）
2. 對照加權指數位置，判斷目前處於哪個象限
3. 特別注意「價漲資增」（正常）vs「價跌資增」（危險）vs「價漲資減」（虛胖）
4. 當融資餘額創歷史新高搭配指數高檔 → 大概率即將反轉

實例：

  • 2024 年台股在 2 萬點附近時，融資餘額持續創新高，散戶瘋狂進場
  • 隨後出現修正，融資戶大量斷頭出場
  • 當融資餘額被清洗到相對低檔時，才是比較安全的進場時機
  • 價漲資減的情況，表示上漲靠的是法人或自有資金，籌碼相對穩定

融資維持率作為市場壓力計

是什麼：整戶融資維持率是比融資餘額更敏銳的市場風險指標。當大盤平均維持率逼近 140%~150% 警戒區，代表市場槓桿壓力升高。

怎麼判斷：
1. 維持率 > 166%：市場健康，槓桿風險低
2. 維持率 150%~166%：正常範圍，但需留意個股風險
3. 維持率 140%~150%：警戒區，系統性風險升溫
4. 維持率 < 140%：危險區，隨時可能爆發斷頭潮

實例：

  • 2025/4/7 台股暴跌 9.7%，大盤融資維持率瞬間跌到 130% 以下
  • 數萬融資戶同時收到追繳令，T+2 前必須補錢或斷頭
  • 這種系統性風險是「所有融資戶同時受害」，無法靠選股完全規避

適用時機：每日收盤後查詢大盤維持率；大盤急跌時特別關注；作為整體倉位風控的參考

四、融資使用率避雷判斷

個股融資使用率

是什麼：融資使用率 = 該股實際融資餘額 ÷ 該股融資限額 × 100%。融資使用率越高，代表散戶介入程度越深。

怎麼判斷：
1. 使用率 < 20%：散戶極少，籌碼乾淨
2. 使用率 20%~50%：散戶溫和介入，尚可接受
3. 使用率 50%~80%：散戶大量進場，籌碼凌亂風險上升
4. 使用率 > 80%：散戶過度集中，一旦反轉賣壓沉重
5. 使用率近 100% 或暫停融資：極度危險，券商已停止融資買進

實例：

  • 某熱門股融資使用率從 30% 暴增到 85%，同期股價從 50 元漲到 120 元
  • 這代表散戶大量用融資追高，一旦反轉，融資斷頭賣壓極重
  • 反之，某績優股融資使用率長期維持在 15%~25%，代表散戶少、籌碼穩定

避雷三大原則：
1. 融資使用率暴增的股票不要追：使用率一個月內從低檔暴增超過 30 個百分點，通常是散戶追高
2. 融資使用率超過 70% 時減碼：散戶過度集中，反轉時賣壓會很重
3. 融資使用率快速下降代表清洗中：如果是因為斷頭而出場，後續可能出現反彈

適用時機：選股時查詢個股融資使用率；大盤急跌時檢查持股融資使用率；避免追融資使用率暴增的股票

五、整戶維持率與多股融資管理

整戶維持率

是什麼：當你同時融資持有多檔股票，券商計算的是「整戶維持率」而非單股維持率。整戶維持率 = 所有融資股票市值總和 ÷ 所有融資金額總和。

怎麼用：
1. 強勢股的獲利可以補弱勢股的維持率缺口
2. 例：融資持有 A 股（獲利 30%）和 B 股（虧損 20%），整戶維持率可能仍在安全區
3. 但如果兩檔同時下跌，整戶維持率會比單股更快跌破 130%

風險管理三原則：
1. 融資持股不要超過 3 檔：分散風險但不要過度分散，否則管理困難
2. 相關性要低：不要同時融資持有同一產業的股票，否則系統性風險時全倒
3. 預留補繳資金：融資金額的 20% 作為預備金，隨時可補維持率

實例：

  • 小明融資持有台積電 2 張（成本 500）和聯發科 1 張（成本 900）
  • 大盤急跌，兩檔同時下跌 15%
  • 整戶維持率從 166% 降到 141%，仍在安全區但已進入警戒
  • 如果再多跌 5%，整戶維持率就會跌破 130%，收到追繳令

適用時機：融資持有多檔股票時；大盤系統性風險評估時；融資部位管理

心得

融資維持率這東西，說穿了就是「借錢炒股的生死線」。130% 不是隨便訂的——它代表你的股票市值只要跌個 22%，就從天堂掉進地獄。更可怕的是斷頭潮，2025 年 4 月那波川普關稅，一天跌 9.7%，幾萬人同時被追繳，券商 T+3 開盤直接市價砸出來，賣在阿呆谷的散户根本來不及反應。

最實用的三個數字要記住：78 元（100 元融資買進的斷頭價）、130%（追繳門檻）、166%（安全水位）。融資前先算好斷頭價，不是等到收到追繳令才開始想怎麼辦。

大盤融資餘額跟融資維持率是兩個層次的事：融資餘額告訴你「市場槓桿有多大」，融資維持率告訴你「這些槓桿有多危險」。四象限判斷法很實用——價漲資增是正常多頭，價漲資減是虛胖，價跌資增是危險訊號，價跌資減到極低才是底部訊號。

最後，融資使用率超過 70% 的股票千萬別追，那是散戶提款機，不是你的提款機。

一、董監改選行情是什麼

核心概念

是什麼：依《公司法》規定，董監事任期不得超過 3 年，每 3 年須在股東會改選。改選前，公司派為保住經營權必須回補持股，市場派為搶奪經營權大量買進股票，雙方搶籌碼大戰推升股價，形成「董監改選行情」。

為什麼會有行情：
1. 公司派回補壓力：董監事持股不足法定成數，必須在停止過戶日前買夠股票才能保住席位
2. 市場派搶權誘因：公司體質好但董監持股低，市場派只需買少量股票就能取得經營權
3. 委託書徵求：雙方都會徵求委託書，增加市場買盤需求
4. 外部搭轎資金：察覺搶籌碼動作的主力也會跟進

二、董監改選時間軸

關鍵時程

怎麼算：
1. 股東會召開：通常在每年 6 月
2. 停止過戶日：股東會前約 2 個月（約 4 月中）
3. 最後過戶日：停止過戶日前一天
4. 融券最後回補日：停止過戶日前第 6 個營業日

行情發動時程：

階段時間特色
潛伏期前一年 12 月公司派默默開始吃貨，股價變動不明顯
發酵期當年 1-2 月市場派開始布局，量能逐步放大
高潮期3-4 月停止過戶日前，雙方搶籌碼最激烈
衰退期停止過戶日後改選底定，賣壓出籠，行情結束

實例：假設某公司 6/13 舉行股東會，從年初到 4/13（股東會造冊截止日）這段期間，是有籌碼面動能的時機點。

三、董監改選選股四大條件

條件一：董監持股比例偏低

怎麼判斷：
1. 查看公開資訊觀測站：每月 15 日前公布董監持股資料
2. 上市櫃平均董監持股比例約 24%，眾數落在 19% 附近
3. 持股低於 10%：公司派籌碼不多，容易被市場派超越，搶權機率高
4. 持股 5-15%：FinLab 回測顯示這個區間改選行情效果最好

實例：

  • 董監持股 5% 以下：公司派幾乎沒有防禦，像大門敞開，市場派輕鬆搶進（如日月光吃矽品）
  • 董監持股 40% 以上：公司派穩如泰山，外部人介入機會極低

條件二：董監持股比前次改選年下降

怎麼做：
1. 查詢該公司 3 年前改選年的董監持股比例
2. 比較目前持股比例是否下降
3. 持股比例下降越多 → 回補壓力越大 → 行情動能越強

為什麼重要：如果董監持股比 3 年前少，代表這段期間大股東一直在賣，到了改選年被迫要買回來，這個「被迫回補」就是行情的燃料。

條件三：股價淨值比偏低 + 股本較小

怎麼篩：
1. 股價淨值比 < 1.3：公司股價便宜，資產價值被低估
2. 股本 < 20 億（< 200,000 張）：小股本易拉抬，市場派只需投入較小資金即可快速提高持股
3. 負債比率 < 50%（非金融業）：沒人想買來幫忙還債
4. 營業利益率 > 0：本業至少要能賺錢

實例：FinLab 回測顯示，多因子條件（股本<20億 + 董監持股5-15% + PB<1.3 + 負債比<50% + 營業利益率>0 + 近6月董監有買進）近 10 年共出現 46 次，勝率 82%，持有 3 個月平均獲利 15%，中位數 11%。

條件四：公司具備亮點題材

三大題材方向：
1. 資產重估題材：公司持有大量土地或不動產，資產價值遠超帳面（如大同的「大同電鍋」資產重估）
2. 轉投資價值：轉投資的子公司或關聯企業具龐大潛在價值
3. 專利收入題材：公司擁有持續性的龐大專利收入來源

四、董監改選行情操作策略

進場時機

怎麼做：
1. 前一年 12 月：開始追蹤今年要改選的公司名單（每 3 年改選一次，可從 3 年前名單推算）
2. 1-2 月：觀察董監持股變化、法人買超、融資券變化
3. 3 月初：在停止過戶日前 1-2 個月進場，這是黃金布局期
4. 進場點設定 10% 停損：FinLab 回測顯示，設 10% 停損可過濾多數虧損交易，幾乎保留所有獲利情況

出場時機

三種出場訊號：
1. 停止過戶日後出場：改選底定，搶籌碼動能消失，賣壓出籠
2. 董監改選結果出爐：無論誰贏，行情都會結束
3. 停損出場：進場後跌幅超過 10%，果斷出場

三大注意事項

1. 改選年名單怎麼找：

  • 公司每 3 年改選一次，可從 3 年前有改選的公司名單推算
  • 公開資訊觀測站可查詢股東會資訊明細
  • 篩選股東會類別為「常會」且有舉辦董監改選的企業

2. 股東臨時會的奇襲：

  • 近年市場派不只在常會搶權，也會發動股東臨時會提前改選
  • 「大同條款」上路後，市場派奇襲更容易發生
  • 要留意大股東動向與公司派防禦動作

3. 董監質押比也要看：

  • 董監質押比持續走高 → 董監個人不看好或資金有問題
  • 質押比高 + 持股比低 = 公司派最脆弱，市場派最容易介入

五、董監持股比例判讀

每月追蹤方法

怎麼做：
1. 每月 15 日後查看公開資訊觀測站的董監持股資料
2. 正常情況下，董監持股比例變化不大
3. 出現大幅變動時列入觀察：

  • 大幅增加 → 公司派看好或正在為改選做準備

搭配其他籌碼面指標

組合判斷：
1. 營收好 + 董監持股下降：營收可能就是相對高峰，大股東趁好時機出脫
2. 空頭格局 + 董監持續加碼：高層對公司營運有信心，可能是底部訊號
3. 董監持股低 + 法人買超增加 + 改選年：典型董監改選行情發動訊號

六、FinLab 回測數據參考

雙因子策略（不佳）

  • 條件：股本 < 20 億 + 全體董事（不含獨董）不足法定股數
  • 結果：平均有獲利但勝率不高，輸大盤

多因子策略（優秀）

  • 條件：股本 < 20 億 + 董監持股 5-15% + PB < 1.3 + 負債比 < 50% + 營業利益率 > 0 + 近 6 月董監有買進
  • 10 年回測共 46 次，勝率 82%
  • 持有 3 個月平均獲利 15%，中位數 11%
  • 設 10% 停損可大幅優化策略體質

心得

董監改選行情是少數可以事先預測的「事件驅動型」行情，因為改選時間表是公開的（每 3 年一次），籌碼面的變化有跡可循。關鍵在於搶在市場派和公司派之前就先布局，而不是等新聞大肆報導才跟進——那時候往往已經是行情尾聲了。

這個策略最吸引人的地方是「有人被迫要買」這個邏輯。公司派不買就失去經營權，市場派不買就搶不到，所以雙方的買盤是確定的，我們只是搭便車。但要注意停止過戶日後的出場紀律——改選底定後，搶籌碼的買盤消失，賣壓會反手出來。

另外，董監持股比例的每月追蹤本身就是很好的籌碼面工具，不限於改選年使用。平時看到董監持股異常增減，就是一個值得注意的訊號。

董監設質比與違約交割：籌碼面隱藏風險判讀

一、董監事股票質押（設質）實戰判讀

股票質押是什麼

是什麼：股票質押又稱股票貸款，就是把持有的股票當抵押品，向金融機構借錢。借款額度最高為股票市值的 60%，質押維持率必須保持在 130% 以上，否則會被斷頭（強制賣出股票）。

質押維持率計算：

質押維持率 = 股票市值 ÷ 借款金額 × 100%

實例：

  • 市值 100 萬的股票質押，借出 60 萬元
  • 維持率 = 100 萬 ÷ 60 萬 × 100% = 166%
  • 股災來襲，市值剩 70 萬 → 維持率 = 70 ÷ 60 = 116% < 130%
  • 銀行要求 3 天內補繳保證金，否則第 4 天開盤強制賣出全部股票

利率比較：

  • 銀行股票質押：年利率 2%~5%
  • 券商股票質押：年利率 6%~7%
  • 銀行利率較低但流程較慢，券商撥款快但利率高

董監事設質比判讀——地雷股預警系統

是什麼：董監事把手中股票拿去質押借款，如果質押比例過高，代表公司治理出現嚴重警訊。證交所法規明言：「上市上櫃公司，其董監事多將持股質押以求護盤，但股價下跌時，又再大肆借貸力守股價，惡性循環之結果導致公司財務公司財務惡化，損害投資大眾權益。」

怎麼查：
1. 進入公開資訊觀測站（mops.twse.com.tw）
2. 選「基本資料」→「董監大股東持股、質押、轉讓」→「董監事持股餘額明細資料」
3. 輸入公司代號，查看每位董監事的「選任時持股數」「目前持股數」「設質股數」「設質股數佔持股比例」

設質比判讀四區間：

設質比風險等級意涵操作建議
< 30%低風險正常理財行為不必過度擔心
30%~50%中度風險董監事資金需求偏大留意財報異常
50%~70%高度風險護盤壓力大，股價下跌惡性循環認真評估是否避開
> 70%極度危險隨時可能斷頭引爆賣壓強烈建議迴避

怎麼做：
1. 每月追蹤設質比變化：不只看絕對值，更要看趨勢——設質比逐月攀升 = 警訊
2. 搭配控制股東持股趨勢：控制股東持股逐漸降低 + 設質比上升 = 雙重警報
3. 設質比高 + 股價下跌 = 死亡螺旋：股價跌 → 維持率不足 → 銀行追繳 → 董監事必須賣股補保證金 → 更多賣壓 → 股價再跌
4. 檢查財報：設質比高時，一定要查財報是否有異常（現金流、負債比、關係人交易等）

實例：

  • 情境：某公司董監事設質比從 30% 三個月內飆升至 65%
  • 同時控制股東持股比例從 25% 降至 18%
  • 財報顯示營業現金流連續兩季為負
  • 判斷：三重警訊同時出現，高機率是地雷股，強烈建議迴避

控制股東持股降低的雙重解讀

是什麼：控制股東是指持股超過 50% 或能實質控制董事會的股東。他們對公司最了解，持股變化是重要訊號。

正面解讀：

  • 控制股東持股穩定或增加 = 看好公司前景，不願稀釋利潤
  • 這是最強的籌碼面偏多訊號之一

負面解讀（兩種風險）：
1. 持股逐漸降低：代表控制股東看壞前景，開始出脫持股減少損失
2. 持股很低但控制董事會：公司治理警訊——控制股東風險很低，卻能做決策，容易犧牲外部股東權益謀求私利

怎麼做：

  • 持股降低 + 設質比上升 = 最危險的組合，趕快跑
  • 持股低但控制力強 = 檢查公司治理品質，看是否有掏空風險

二、違約交割機制與市場風險判讀

違約交割是什麼

是什麼：股票成交後，買方在 T+2 上午 10:00 前交割帳戶餘額不足，導致扣款失敗。這不是「帳虧」，而是法律上的違約行為。

後果嚴重程度排序：
1. 信用破產：違約紀錄計入聯徵中心，影響信用卡、房貸、信貸申請
2. 帳戶註銷：3 個營業日內未清償，證券帳戶被註銷
3. 違約金：券商可收成交金額 7% 為上限的違約金
4. 強制賣出：券商直接市價賣出違約股票，虧損部分繼續追討
5. 5 年開戶禁令：違約未結案且未滿 5 年，所有券商不得開戶
6. 刑事責任：情節重大影響市場秩序，3 年以上 10 年以下有期徒刑，得併科 1,000 萬至 2 億元罰金

下錯單怎麼辦：

  • 還沒收盤：開現股當沖反向賣出抵銷，或賣其他持股籌錢
  • 已經收盤：趕快借錢（親友周轉）完成 T+2 交割，T+1 再賣出股票還錢

違約交割的市場層面影響

是什麼：當違約交割金額暴增時，代表市場出現系統性壓力。歷史數據顯示，違約交割金額飆升常出現在股市大跌後。

歷史案例：

  • 2022/7/11：單一投資人當沖違約交割 6.6 億元，佔當日違約金額 95%
  • 2024/8/2：台股重挫 1004 點，10 家券商通報違約交割 2.3 億元，共 13 戶

怎麼判讀違約交割數據：
1. 查詢管道：證交所「違約公告專區」（twse.com.tw），每日公布違約交割金額
2. 單日違約金額超過 5 億：市場壓力明顯，短線可能有更多賣壓
3. 違約戶數突然暴增：從常態的個位數跳到兩位數以上 → 散戶資金鏈出問題
4. 集中在特定個股：某檔股票違約金額異常大 → 可能引發券商強制賣出該股的連鎖效應

違約交割 vs 融資斷頭：兩種不同的系統性風險

關鍵差異：

面向融資斷頭違約交割
觸發原因融資維持率 < 130%T+2 帳戶餘額不足
影響範圍使用融資的個股任何股票交易都可能發生
處理方式券商 T+3 強制賣出券商立即處分 + 法律追訴
市場衝擊大面積融資斷頭 = 多殺多大面積違約 = 券商賣壓 + 信用緊縮
可預測性可從融資水位預判較難預測，突發性強

怎麼做：

  • 大盤大跌後，同時觀察融資減少速度和違約交割金額
  • 兩者同時飆升 = 系統性風險最高峰，不搶反彈
  • 融資減少但違約金額正常 = 融資斷頭接近尾聲
  • 違約金額持續偏高但融資已穩定 = 散戶資金鏈仍緊張，保守操作

三、內部人申報轉讓：董監事賣股的提前預警

內部人轉讓申報制度

是什麼：董監事、經理人、大股東（持股 10% 以上）等內部人，賣出股票前必須向主管機關「事前申報」，申報日起 3 日內才能轉讓。每月 5 日前須申報上月持股變動。

每日最大可轉讓股數計算：

  • 公司已發行股份總數 3,000 萬股以下：按千分之二計算
  • 超過 3,000 萬股之部分：按千分之一計算
  • 或按申報日前 10 個營業日平均每日成交量的 50% 計算
  • 取兩者較高者為當日最大可轉讓數

怎麼用：
1. 每天查看公開資訊觀測站的「內部人股權轉讓事前申報」
2. 如果出現大量董監事申報轉讓 → 內部人在賣股，這是最直接的賣出訊號
3. 注意申報轉讓的「數量」和「頻率」：

  • 偶爾小量轉讓 = 正常理財需求

實例：

  • 情境：某公司三位董事同時在 5 天內申報轉讓合計 5,000 張
  • 同時公司股價在相對高檔，本益比超過 30 倍
  • 判斷：內部人大舉出脫，後市看淡，應跟著減碼或出場

內部人申報轉讓 vs 一般賣超

差異：

  • 一般法人賣超：可能是換股操作、避險，不一定是看空
  • 內部人申報轉讓：內部人對公司最了解，他們賣股的參考價值遠高於法人
  • 「身體比嘴巴誠實」——內部人大買就是利多，大賣就是利空，不需要猜測動機

但要注意：

  • 內部人賣股可能有個人理財需求（買房、繳稅等），不代表一定看空公司
  • 要看「持續性」和「數量」——偶發性小量賣出不需過度解讀
  • 搭配設質比一起看：內部人賣股 + 設質比上升 = 最強烈的看空訊號

四、三重風險交叉判讀 Checklist

董監設質比 + 內部人轉讓 + 違約交割 綜合判斷

怎麼做：
1. 每月第一步：查公開資訊觀測站董監設質比
2. 每日第二步：查看內部人股權轉讓事前申報
3. 大跌後第三步：查證交所違約交割公告
4. 交叉判讀：

設質比內部人轉讓違約交割綜合判斷操作
< 30%無異常正常安全正常操作
30~50%偶發小量正常輕度警戒留意但不需動作
50~70%多人申報偏高高度警戒考慮減碼
> 70%大量申報暴增極度危險立即出場

適用時機：

  • ✅ 選股時的風險篩選（先剔除設質比過高的公司）
  • ✅ 大跌後的系統性風險評估
  • ✅ 持股健康度定期檢查
  • ❌ 不適合當作唯一的進場訊號（要搭配基本面和技術面）

心得

這回合學到的「董監設質比」和「違約交割」，是籌碼面分析中常被忽略的兩個風險指標。大部分人在看籌碼只看法人買賣超、主力進出，卻忘了最根本的問題：董監事自己對公司有沒有信心？

設質比過高就是最明確的「不信任票」。董監事把股票拿去借錢，而且比例超過 50%，代表他們自己的財務狀況都不健康了，還在護盤維持股價——這種護盤是被迫的，不是自願的。一旦股價跌破維持率 130% 的警戒線，銀行就會強制賣出質押的股票，形成「越跌越賣、越賣越跌」的死亡螺旋。公司法第 197 條之 1 的立法意旨講得很白：董監質押護盤 → 股價下跌 → 再借貸護盤 → 惡性循環 → 公司財務惡化。

違約交割的部分，以前我以為這只是「散戶自己不小心」的問題，但從市場層面看，大面積違約交割意味著散戶資金鏈斷裂。2022 年那個 6.6 億違約的案例太誇張——一個人當沖搞到違約交割 6.6 億，這不是「不小心」，是賭徒心態。但從市場風險角度，這種事件會引發券商強制賣出的連鎖效應，對股價造成額外壓力。

最實用的收穫是那個「三重風險交叉判讀 Checklist」。以前只會單獨看設質比或違約交割，但把它們組合起來看，判斷力就強多了。設質比高 + 內部人申報轉讓 + 違約交割暴增 = 三重警報同時響，這種股票碰都不要碰。以後選股時，我會把設質比加入篩選條件——超過 50% 的直接剔除，不管基本面多好看。

一、融券回補力道計算與判讀

融券回補力道是什麼

是什麼：融券回補力道 = 融券餘額 ÷ 5 日成交均量。這個比率衡量的是「如果所有融券都要在同一天回補，需要消耗幾天的成交量」。比率越高，回補對股價的推升力道越強。

怎麼算：
1. 查詢個股融券餘額（張數）：證交所「融資融券餘額」頁面
2. 計算近 5 日成交均量（張數）
3. 回補力道 = 融券餘額 ÷ 5 日均量

實例：

  • A 股融券餘額 5,000 張，5 日均量 1,000 張 → 回補力道 = 5（需要 5 天成交量才能消化）
  • B 股融券餘額 3,000 張，5 日均量 3,000 張 → 回補力道 = 1（1 天就能消化，壓力小）

判讀標準：

  • 回補力道 > 3：高軋空風險，回補買盤可能推升股價
  • 回補力道 1~3：中等壓力
  • 回補力道 < 1：回補壓力低，軋空行情機率小

適用時機：接近股東會回補日、除權息回補日時，用此指標篩選有軋空潛力的個股

二、真假軋空辨別四步驟

為什麼要辨別真假軋空

是什麼：高券資比不一定代表軋空行情。有些融券增加是「避險鎖價差」行為，不是真的看空，這種就不會產生軋空動能。

怎麼判斷：
1. 查公司是否發行可轉換公司債：如果近期有 CB（轉換價低於市價），融券可能是套利鎖價差，不是真看空
2. 查是否有現金增資：認股價低於市價時，融券鎖價差是正常操作
3. 搭配大戶持股與主力買賣超：如果大戶持股增加、主力買超，但融券也增加，代表散戶在放空、主力在做多 → 真軋空機率高
4. 看融券增加的來源：如果主力也在放空，那可能是真看空；如果融券全來自散戶，被軋空機率高

實例：

  • 創惟(6104) 2022/1-2月：大戶持股人數增加 + 主力買超 + 融券大增（散戶放空）→ 券資比 > 30% → 散戶被軋空
  • 某股融券增加 + 同時有 CB 發行 → 假軋空，融券是鎖價差

適用時機：看到高券資比個股想跟單做多時，先確認是不是真軋空再進場

三、融券回補日期計算實戰

回補日怎麼算

是什麼：法規規定，停止過戶日前第 6 個營業日為最後回補日。回補日之前 4 個營業日開始停券（不能融券賣出）。

怎麼算：
1. 確認停止過戶起始日（公司公告）
2. 往前推 6 個營業日 = 最後回補日
3. 再往前推 4 個營業日 = 開始停券日

實例：

  • 停止過戶起始日 8/16（一）
  • 最後回補日 = 8/16 往前推 6 個營業日 = 8/8
  • 停券期間 = 8/9 ~ 8/15（共 4 個營業日不能融券）
  • 8/16 之後恢復融券交易

關鍵提醒：

  • 臨時股東會「不需要」融券回補
  • 一年多次配息的股票（如台積電），一年最多 5 次回補，頻繁回補增加放空成本
  • 證交所「停券預告表」可直接查詢回補日

適用時機：融券放空前務必確認回補日；做軋空行情要鎖定即將回補的高券資比股

四、軋空行情操作三條件

什麼情況下軋空行情最猛烈

是什麼：軋空行情不是每次高券資比都會發生，需要三個條件同時滿足，力道才夠強。

三條件：
1. 券資比 > 30%：融券餘額佔融資餘額超過三成，代表放空比重高，潛在買盤大
2. 回補力道 > 3：融券餘額超過 5 日均量的 3 倍，回補時供不應求
3. 籌碼集中在大戶：千張大戶持股比例高，浮額少，拉抬容易

怎麼操作：
1. 每年 3~4 月股東會旺季前，篩選高券資比個股
2. 搭配大戶持股變化、主力買賣超方向
3. 在最後回補日前 2~3 週進場，回補日前 1 週出場（避免回補後賣壓）
4. 停損設在進場價下方 5%

實例：

  • 建準(2421) 2023 年：券資比上升 + 基本面轉佳 + 大盤偏多 → 引發軋空行情
  • 卜蜂(1215) 111 年有 3 次回補：04/15（股東會）、06/29（除息）、08/19（除權）

適用時機：每年股東會旺季（3-5月）和除權息旺季（6-8月）是軋空行情高發期

不適用時機：大盤趨勢明確走空時，即使高券資比也難逆勢軋空

心得

這回合把融券回補的「力道量化」搞清楚了。以前只會看券資比高低，但券資比沒考慮成交量，融券 5000 張搭配日均量 500 張 vs 日均量 5000 張，完全是不同等級的壓力。回補力道 = 融券餘額 ÷ 5 日均量這個公式簡單但好用，以後篩選軋空股可以用這個當第一關。

真假軋空的辨別也很關鍵。以前看到高券資比就想跟單做多，結果碰到 CB 鎖價差的假軋空被巴。以後看到高券資比，先查：有沒有 CB？有沒有增資？主力是買還是賣？四個確認做完再進場。

另外臨時股東會不用回補這點要注意，不是所有股東會都會觸發回補行情。

一、籌碼七指標選股法完整解析

七大籌碼指標是什麼

是什麼：將七個籌碼面指標組合成選股條件，當一檔股票同時滿足越多指標，短線發動的機率越高。這不是單一指標的判斷，而是多維度籌碼確認。

七大指標一覽：

指標條件意涵
1. 券商重押股單一券商買超佔成交量 > 30%有「人」極度看好，可能掌握利多
2. TOP1 超額買超TOP1 買超 ÷ TOP1 賣超 > 4 倍買方力道遠大於賣方，推升力道強
3. 短線籌碼集中1 日籌碼集中度 > 20%前 15 名買超力道集中，主力作多明確
4. 大戶買散戶賣大戶連 3 週成長 + 散戶連 3 週衰退籌碼從弱手流向強手
5. 主力連買主力連買 > 3 天，買超 > 1000 張主力持續加碼，波段行情機率高
6. 外資連買外資連買 > 3 天，買超 > 1000 張外資波段操作，後續推升力道強
7. 投信連買投信連買 > 3 天，買超 > 1000 張投信認養股，中線偏多

指標 1：券商重押股

是什麼：某間券商突然急買特定股票，買超張數佔當天成交量 30% 以上。

怎麼做：
1. 每天收盤後查看分點進出資料
2. 計算各券商買超張數佔成交量比例
3. 重押比例 > 30% = 高度關注
4. 回測該券商歷史重押績效——是否每次重押後都有波段行情

實例：

  • 港商麥格理重押聯詠（3034）1081 張，佔當天成交量 3393 張的 32%
  • 4 天後聯詠大漲 9.56%
  • 麥格理是外資券商，大買通常代表外資有內部資訊或強烈看好

適用時機：短線選股、發現主力異常進場時

指標 2：TOP1 超額買超

是什麼：買超最多的券商與賣超最多的券商，兩者買賣量的比值超過 4 倍。

怎麼算：

超額買超 = TOP1 買超券商買超張數 ÷ TOP1 賣超券商賣超張數

怎麼做：
1. 查看個股當日分點進出
2. 找出買超第一名和賣超第一名的張數
3. 計算超額買超比值
4. 比值 > 4 = 買方力道遠大於賣方，股價上漲機率高

實例：

  • 中美晶（5483）最大買超券商元大經紀買超 1890 張
  • 最大賣超券商富邦-民生賣超僅 224 張
  • 超額買超比 = 1890 ÷ 224 = 8.4 倍
  • 3 天後中美晶漲 4.7%

是什麼：衡量前 15 名買超券商的買超力道，是否集中在少數主力手中。

怎麼算：

籌碼集中 = 前 15 名買超量 - 前 15 名賣超量
籌碼集中度(%) = 籌碼集中 ÷ 總成交量 × 100%

怎麼做：
1. 計算個股每日籌碼集中度
2. 籌碼集中度 > 20% = 主力作多意願強烈
3. 觀察連續性：連續 2 天以上籌碼集中度偏高更可靠
4. 搭配股價位置判斷：低檔集中比高檔集中更有爆發力

實例：

  • 超眾（6230）11/25 籌碼集中度 21.33%，7 天大漲 43.57%
  • 關鍵：低檔籌碼突然集中，代表主力在低檔大量吃貨

指標 4：大戶買散戶賣

是什麼：透過集保股權分散表觀察，大戶（千張以上）持股連續增加，散戶（50 張以下）持股連續減少。

怎麼做：
1. 每週六查看集保股權分散表更新
2. 觀察千張以上持股比例變化（連續 3 週增加）
3. 同時觀察 50 張以下持股比例變化（連續 3 週減少）
4. 兩者同時滿足 = 籌碼從散戶流向大戶，股價易漲難跌

實例：

  • 凌巨（8105）連續 3 週大戶持股增加、散戶持股減少
  • 10/26 以開盤價 12.9 元進場
  • 11/23 以收盤價 24.6 元出場
  • 1 個月大漲 90.69%

指標 5：主力連買

是什麼：主力（前 15 名券商買賣超合計）連續 3 天以上買超，且買超張數超過 1000 張。

怎麼做：
1. 每天收盤後查看主力買賣超數據
2. 篩選主力連買 > 3 天的個股
3. 確認買超張數 > 1000 張（量太小的沒意義）
4. 觀察主力是否創近期買超新高
5. 搭配股價位置判斷：低檔主力連買勝率更高

實例：

  • 華新科（2492）3/2 發現主力已連買 3 天
  • 3/3 開盤 22.05 元進場
  • 持有至 3/14 收盤價 28 元
  • 8 天報酬率 26.9%

2. 確認買超張數 > 1000 張
3. 外資三大特性：

  • 買大牌（偏好權值股、大型股）

4. 在外資剛開始連買時進場，勝率最高

2. 大戶買賣力由負轉正 → 進場做多
3. 停損設在進場價下方 3~5%（或前一根 K 棒低點）
4. 停利設在下一個壓力位或移動停損（回跌 5% 出場）
5. 持股期間每天追蹤七指標變化，指標轉弱時減碼

三、籌碼面技術面整合進出場策略

為什麼要整合籌碼面和技術面

核心邏輯：

  • 籌碼面告訴你「誰在買」和「誰在賣」
  • 技術面告訴你「什麼時候買」和「什麼價格買」
  • 單用籌碼面：知道主力在買，但不知道最佳進場點
  • 單用技術面：看到突破訊號，但不知道是不是主力在出貨
  • 籌碼面 + 技術面 = 知道「買什麼」+「何時買」+「何時賣」

整合策略一：籌碼確認 + 技術進場

怎麼做：
1. 籌碼面篩選：七指標中滿足 3 個以上
2. 技術面等進場點：

  • 股價突破下降趨勢線 → 等待回踩確認再進場

3. 停損設定：突破前低或均線下方 2~3%

實例：

  • 某股主力連買 5 天 + 投信連買 3 天 + 籌碼集中度 18%
  • 但股價仍在盤整區間，均線糾結
  • 等待突破盤整區上緣且量增 → 進場
  • 停損設在盤整區下緣

整合策略二：九階段判斷 + 技術出場

怎麼做：
1. 用籌碼九階段判斷股價目前處於哪個階段
2. 不同階段對應不同的技術面操作：

九階段技術面操作
1. 接等待底部型態完成（W底、頭肩底）
2. 護小量佈局，停損設在前低
3. 拉突破下降趨勢線進場，停損設在趨勢線
4. 換均線多頭排列時加碼，停損設在 20MA
5. 養減碼至半數，停損設在前高（支撐變壓力不成立則出場）
