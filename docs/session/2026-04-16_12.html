一、交易計畫書：每一筆交易前的必填單

1. 為什麼需要交易計畫書？

是什麼：交易計畫書是一筆交易啟動前，把所有進出場條件、風險管理、預期目標全部寫下來的文件。不是寫在腦海裡，是白紙黑字寫下來。

怎麼做：
1. 交易前寫，交易中不改：這是最核心的原則。計畫書在進場前完成，進場後任何條件都不允許事後修改——尤其是停損點
2. 強制回答五個問題：每一筆交易都必須回答以下五個問題，少一個就不進場

  • 我為什麼要做這筆交易？（理由）

3. 寫下來的目的不是預測，是紀律：很多人覺得「計畫趕不上變化」，但交易計畫書的價值不在於預測準確，在於防止你在壓力下做出衝動決策

實例：

  • 錯誤：看到台積電大漲3%，腦中閃過「應該會繼續漲」，直接追進5張，沒想停損停利 → 漲到4%反轉跌到-2%，不知道該不該賣，最後凹單虧8%
  • 正確：先寫計畫書「進場理由：帶量突破前高，均線多排。目標：+5%。停損：-3%。盈虧比：1.67:1。部位：2張。」→ 按計畫執行，不猶豫

適用時機：每一筆交易都要寫，不管大小。如果你覺得寫計畫太麻煩，那代表這筆交易你還沒想清楚，不應該進場

2. 交易計畫書六大欄位完整架構

是什麼：一個完整的交易計畫書應該包含六大欄位，每個欄位都有明確要填寫的內容。

怎麼做：

欄位一：趨勢方向確認

  • 交易標的名稱/代碼：______
  • 週線級別趨勢：多頭/空頭（EMA60 vs EMA120判斷）
  • 日線級別趨勢：多頭/空頭
  • 趨勢斜率方向：12點到6點鐘方向，斜率越陡趨勢越強
  • 對應策略：多頭趨勢→順勢做多或回調買進；空頭趨勢→觀望或做空；盤整→不操作或逆勢區間交易

欄位二：進場理由

  • 做多/做空？開倉/平倉？
  • 進場理由（具體到可驗證）：如「帶量突破前高850元且站穩3天」、「RSI從超賣區回升且MACD金叉」
  • 預計進場價格：______

欄位三：出場目標

  • 如果方向正確，何時停盈出局？（具體條件，不是「感覺漲夠了」）
  • 預估停盈價格：______

欄位四：停損底線

  • 如果方向錯誤，何時止損出局？（具體條件）
  • 預估止損價格：______

欄位五：盈虧比計算

  • 盈虧比 = (停盈價 - 進場價) / (進場價 - 止損價) = ______
  • 盈虧比 > 3 才進場（低於3:1的交易不值得冒險）

欄位六：部位計算

  • 最大可承受虧損金額：______（建議不超過總資金1-2%）
  • 最多可持有頭寸 = 最大虧損金額 / (進場價 - 止損價) = ______張

實例：

  • 標的：聯發科(2454)，週線多頭、日線多頭，趨勢斜率2點鐘方向
  • 進場理由：帶量突破前高950元，回測945不破，5日均線上方
  • 進場價：950元
  • 停盈：前波壓力1030元（+8.4%）
  • 停損：跌破920元（-3.2%）
  • 盈虧比 = 80 / 30 = 2.67:1（接近3:1，可接受）
  • 最大虧損：1萬元（總資金100萬的1%）
  • 部位：10000 / 30 = 333股 ≈ 0.33張 → 不足1張，需調整為1張（虧損放大到3000×30=9萬，超過1%）→ 改為停損935元，虧損15元/股，10000/15=666股，仍不足1張 → 改為最大虧損2%（2萬），20000/15=1333股≈1張

適用時機：每筆交易進場前完成，交易過程中只看計畫書執行，不憑感覺

二、進場前檢查清單（Trading Checklist）

3. 大環境檢查：三道總體濾網

是什麼：在選股之前，先確認大環境是否適合進場。大環境不對，個股再好也容易被拖累。

怎麼做：

濾網一：全球股市趨勢

  • VT（Vanguard全世界股票ETF）是否在季線（60日均線）上方？
  • VT在季線上方 → 全球股市偏多頭，可以做多個股
  • VT在季線下方 → 全球股市偏空頭，減少做多部位或觀望

濾網二：台股加權指數趨勢

  • 加權指數是否在季線上方？
  • 在季線上方 + 均線多頭排列 → 做多勝率高
  • 在季線下方 + 均線空頭排列 → 做多勝率低，應觀望或做空

濾網三：市場情緒指標

  • VIX指數是否異常飆高？（>30代表恐慌）
  • 大盤成交量是否萎縮？（低於20日均量=量縮，難有行情）
  • 外資期貨未平倉口數方向？（淨多單=偏多，淨空單=偏空）

實例：

  • 2024年4月：VT在季線上方✅、台股加權在季線上方✅、VIX正常✅、外資淨多單✅ → 四項全過，適合做多
  • 2022年6月：VT跌破季線❌、台股加權在季線下方❌、VIX飆到30以上❌ → 三項不過，不宜做多

適用時機：每日盤前必做，只需5分鐘；三項都不過的日子，不要做任何新單

4. 個股進場前十大確認項目

是什麼：通過大環境濾網後，對個股進行的逐項檢查。林昇的10項基本面條件 + 技術面 + 籌碼面三層確認。

怎麼做：

第一層：基本面篩選（10項條件，林昇散戶贏家法）
1. 營收年增率 > 0（公司營收在成長）
2. 毛利率 > 20%（有基本獲利能力）
3. 營業利益率 > 10%（本業有賺錢）
4. 稅後淨利率 > 8%（最終獲利品質好）
5. ROE > 15%（股東權益報酬率夠高）
6. 負債比 < 60%（財務結構健康）
7. 本益比 < 25（未過度溢價）
8. 殖利率 > 2%（有基本回報）
9. 夏普值 > 0（風險調整後報酬為正）
10. 法人持股比例穩定或上升（外資/投信持續買進）

第二層：技術面確認
1. 四線多排：5日、10日、20日、60日均線全部向上
2. K線結構：頭頭高、底底高（多頭結構）
3. 位階判斷：底部初升段最佳，避免末升段追高
4. 成交量：帶量突破或量價齊揚

第三層：籌碼面確認
1. 外資/投信連續3日以上買超
2. 大戶持股比例增加
3. 融資使用率未過高（<30%）
4. 券資比合理（<30%偏多）

實例：

  • 通過全部三層確認的股票：台積電(2330) 營收YoY+30%✅、毛利率58%✅、ROE 32%✅、四線多排✅、外資連續5日買超✅ → 三層全過，高信心進場
  • 只通過部分的股票：某營建股 營收YoY-5%❌、ROE 8%❌、但技術面帶量突破✅、投信買超✅ → 基本面不過，不進場

適用時機：每次選股時逐項檢查，至少通過7項以上才列入觀察名單；三層都通過的才是操作標的

5. 盈虧比與部位計算實戰

是什麼：進場前必須計算的兩個數字——盈虧比決定值不值得做，部位大小決定虧多少。

怎麼做：

盈虧比計算：
1. 公式：(目標價 - 進場價) / (進場價 - 停損價)
2. 盈虧比 ≥ 3:1 才進場（例外：勝率>60%的策略可用2:1）
3. 盈虧比 < 2:1 絕對不進場，不管訊號多好看

部位大小計算：
1. 公式：最大虧損金額 / (進場價 - 停損價) = 可買股數
2. 最大虧損金額 = 總資金 × 1%（保守）或 2%（積極）
3. 計算結果不足1張時：調整停損距離或放棄這筆交易

實例：

  • 總資金100萬，1%法則 = 單筆最大虧損1萬
  • 進場價50元，停損48元（2元風險），目標56元（6元利潤）
  • 盈虧比 = 6/2 = 3:1 ✅
  • 部位 = 10000/2 = 5000股 = 5張
  • 驗算：5張虧2元 = 10000元（正好1%）✅
  • 反例：進場價600元（台積電），停損590元（10元風險），目標620元（20元利潤）
  • 盈虧比 = 20/10 = 2:1（偏低）❌
  • 部位 = 10000/10 = 1000股 = 1張
  • 驗算：1張虧10元 = 10000元 ✅，但盈虧比只有2:1不值得
  • 解決：等更好的進場點（如580元進場），或等回檔再進場

適用時機：每筆交易進場前必算，不計算不進場

三、交易日記與覆盤系統

6. 交易日記七大必填欄位

是什麼：交易日記是交易計畫書的「事後對照版」，記錄實際發生的情況，用來覆盤和改進策略。

怎麼做：

欄位一：進出場策略

  • 這筆交易一開始的策略是什麼？做多/做空？短線/波段？追突破/等回調？
  • 核心原則：交易中不得更改策略。原本做短線的不能因為套牢就改成長投

欄位二：進出場價格

  • 實際買進價格、賣出價格
  • 與計畫書的預設價格比較：是否有滑價？是否追高？

欄位三：買賣數量與加碼比例

  • 初始部位多少？中間有加碼嗎？加碼比例？
  • 加碼是否按計畫執行？是否違反「只加碼獲利部位」原則？

欄位四：交易時間

  • 進場時間、出場時間
  • 持倉多久？是否超出預期？

欄位五：進出場理由

  • 進場理由是什麼？出場理由是什麼？
  • 出場是按計畫停損/停利？還是情緒化出場？

欄位六：損益金額與比例

  • 這筆交易賺/賠多少？佔該筆資金多少%？佔整體資產多少%？
  • 用比例而非金額，才能跨市場比較

欄位七：心情起伏

  • 交易過程中情緒有什麼變化？
  • 有沒有想改策略的衝動？最後有沒有忍住？
  • 記錄「差點犯錯但忍住了」的時刻，這是進步的證據

7. 覆盤四步驟：從日記中提煉改進方向

是什麼：交易日記寫了不看的話等於白寫，覆盤是從日記中找出策略弱點並改進的過程。

怎麼做：

步驟一：分類統計

  • 將所有交易分為「賺錢」和「賠錢」兩類
  • 分別統計：勝率、平均獲利、平均虧損、盈虧比、期望值
  • 用公式驗證：期望值 = 平均獲利 × 勝率 - 平均虧損 × (1-勝率)

步驟二：找出賠錢交易共同特徵

  • 賠錢的交易有什麼共同點？
  • 是不是都在特定盤勢（震盪盤）進場？
  • 是不是都違反了某條紀律（如沒設停損、逆勢加碼）？
  • 是不是都在特定時段（如開盤30分鐘）進場？

步驟三：找出賺錢交易共同特徵

  • 賺最多的交易有什麼共同點？
  • 是不是都在趨勢明確時順勢操作？
  • 是不是都嚴格按計畫停損停利？
  • 是不是都選對了產業方向？

步驟四：制定改進行動

  • 根據步驟二和三的發現，寫下1-3條具體改進行動
  • 例如：「震盪盤不做多」、「進場前一定算盈虧比」、「外資連續賣超3日以上減碼至半倉」

實例：

  • 月覆盤發現：本月20筆交易，勝率45%，平均獲利5%，平均虧損3%
  • 期望值 = 5%×0.45 - 3%×0.55 = 2.25% - 1.65% = 0.6%（正期望值但偏低）
  • 賠錢交易共同特徵：8筆虧損中有6筆是在ADX<20的震盪盤中進場
  • 改進行動：ADX<20時不進場做多，只做區間交易或不操作

適用時機：每週小覆盤（10分鐘），每月大覆盤（30分鐘）；覆盤比交易本身更重要

四、波段交易資金配置原則

8. 林昇的波段資金配置法

是什麼：散戶贏家林昇的實戰資金配置方法——波段交易資金不超過總投資部位的20%。

怎麼做：
1. 核心配置（80%）：長期投資部位，ETF（0050/006208）、高殖利率股、美元定存等，追求穩定現金流
2. 波段配置（20%）：波段交易部位，用基本面+籌碼面篩選個股，積極操作
3. 波段部位的風險控制：單一個股不超過波段部位的30%（即總資金的6%）
4. 對沖工具：用期貨和選擇權做風險對沖，尤其是大盤轉弱時
5. 新手建議：先從ETF開始，累積經驗後再增加波段交易比重

實例：

  • 總資金100萬
  • 核心部位80萬：0050 40萬、高殖利率股30萬、美元定存10萬
  • 波段部位20萬：最多選3-5檔個股操作
  • 單一個股上限6萬（20萬×30%）
  • 單筆最大虧損1萬（總資金1%）→ 單一個股虧損控制在6萬×17%以內

適用時機：年度資產配置時設定，每季檢視一次比例是否需要調整

9. 波段交易四大策略整合

是什麼：林昇在波段交易中使用的四種策略，從選股到進場的完整流程。

怎麼做：

策略一：確認全球與台灣趨勢偏多

  • VT在季線上方 + 台股加權指數在季線上方 → 做多勝率高
  • 任一指標在季線下方 → 減少做多部位

策略二：基本面量化選股

  • 用10項條件篩選（營收成長率、毛利率、ROE、負債比等）
  • 篩出100-200檔後，用夏普值和法人持股比例做第二層過濾
  • 最終保留20-30檔觀察名單

策略三：籌碼面確認進場

  • 觀察法人、大戶的籌碼動向
  • 外資或投信連續買超 + 大戶持股增加 → 籌碼面偏多
  • 夏普值 > 0 → 風險調整後報酬為正

策略四：技術面擇時進場

  • 四線多排確認 + 回後買上漲進場
  • 停損設在進場K線低點或5%固定停損
  • 出場按朱家泓邏輯：遇壓力看突破與否、未過前高看盤整或反轉

實例：

  • Step1：VT在季線上方✅、台股在季線上方✅ → 大環境偏多
  • Step2：用10項條件篩出180檔，夏普值>0 + 投信連續買超 → 保留25檔
  • Step3：25檔中籌碼面最強的5檔列為觀察名單
  • Step4：等技術面「回後買上漲」訊號出現再進場

適用時機：每週做一次完整四步驟檢視，盤中只需追蹤步驟三和四

心得

今天學的交易計畫書和進場前檢查清單，看似「老生常談」，但回顧之前學過的所有策略——海龜交易法、朱家泓回後買上漲、當沖八大步驟、程式交易系統——它們成功的共同前提都是「有計畫、守紀律」。今天終於把這個「前提」系統化了。

1. 交易計畫書的核心不是預測，是防止犯錯。霍華馬克思說「不只要試著做對的事，還要大力強調不去做錯的事」。計畫書的價值不是讓你預測準確，而是在你面對價格波動、情緒起伏時，有一個冷靜時寫下的決策可以回頭看。腦中的計畫在壓力下會變形，白紙黑字的計畫不會。

2. 盈虧比≥3:1是一個嚴格但必要的門檻。TheMarketMemo的交易流程表把盈虧比放在最核心的位置——盈虧比<3就不進場。這跟之前學的海龜交易法2%風險控制一致：如果你每筆只冒1-2%的風險，但潛在利潤有3-6%，就算勝率只有30-40%，長期期望值仍然是正的。盈虧比<2:1的交易就像賭博，就算方向對了也賺不多，方向錯了就虧大了。

3. 林昇的20%波段資金配置很務實。很多人做波段交易就把全部資金壓進去，結果一個閃失就大虧。20%的上限確保即使波段部位全部虧光（最極端情況），你還有80%的核心部位撐著。而且波段交易風險本來就比ETF高，個股一旦下跌要回到前高比ETF難得多。

4. 交易日記的「心情起伏」欄位最被低估。七仔財經筆記的觀點很好——交易=「投資知識」+「心理學」。很多人只記錄進出場價格和損益，但真正影響績效的是那些「差點犯錯的時刻」和「衝動改策略的念頭」。把心情起伏記下來，覆盤時才能區分「策略有問題」還是「執行有問題」。

5. 覆盤四步驟是交易日記的靈魂。日記寫了不看的話等於白寫。最實用的做法是每月統計勝率和盈虧比，然後找出虧損交易和盈利交易的共同特徵。這跟程式交易的回測邏輯一致——用數據說話，不要憑感覺判斷策略好不好。

6. 「不因情緒改變計畫」這個勾選欄位，是最重要也最難做到的。交易計畫書最後那個確認簽名，看似形式主義，但它的心理作用是：你在冷靜狀態下承諾了一件事，當壓力來襲時，這個承諾會幫你守住底線。這跟之前學的「交易冷卻時間」概念一致——在情緒高點不做決策，回頭看計畫書。

一、區間交易（Range Trading）核心概念與實戰

1. 區間交易定義與市場環境判斷

是什麼：在價格明確的支撐與阻力之間反覆低買高賣的策略。統計上，金融市場有 70-80% 的時間處於區間震盪狀態，而非單邊趨勢——這意味著區間交易的機會遠多於趨勢交易。

怎麼做：
1. 識別區間：價格在同一高點（阻力）和同一低點（支撐）之間來回至少2次觸及且反彈
2. 確認區間有效性：支撐和阻力各至少被測試2次以上，且價格未有效突破（收盤價未站穩區間外）
3. 判斷市場狀態：用ADX指標確認——ADX<20代表趨勢極弱、市場處於盤整；布林通道收縮（Squeeze）也暗示區間成形
4. 區間寬度評估：用ATR衡量區間寬度，區間至少要有1.5倍ATR以上的寬度才值得交易（太窄的區間交易成本吃掉利潤）

實例：

  • 情境：台積電（2330）在580-620元之間震盪3週，支撐580被測試3次、阻力620被測試2次
  • 判斷：ADX=15（趨勢極弱），布林通道收縮，ATR=15元，區間寬度40元（2.67倍ATR）→ 有效區間
  • 操作規劃：580附近做多、620附近做空

適用時機：盤整市（ADX<20、均線糾結、布林通道收縮）；不適合趨勢市（ADX>25、均線發散）

2. 區間交易兩大操作法

是什麼：區間交易不是只有「支撐買、阻力賣」這一種做法，還有「突破交易法」——兩種方法對應不同的風險偏好。

怎麼做：

方法一：區間內支撐阻力交易（逆勢打法）
1. 價格觸及支撐區 → 做多，停損設在支撐下方1-2%或1個ATR
2. 價格觸及阻力區 → 做空，停損設在阻力上方1-2%或1個ATR
3. 目標：價格回到區間中間或對側邊界
4. 搭配K線反轉型態確認：支撐區出現錘子線、牛市孕育型 → 做多訊號更可靠
5. 搭配震盪指標確認：RSI<30（超賣）+ 價格在支撐區 → 做多；RSI>70（超買）+ 價格在阻力區 → 做空

方法二：區間突破交易（順勢打法）
1. 價格帶量突破阻力 → 做多，停損設在突破點下方1個ATR
2. 價格帶量跌破支撐 → 做空，停損設在跌破點上方1個ATR
3. 等待回測確認：突破後回測區間邊界不破 → 更安全的進場點
4. 突破確認條件：成交量明顯放大（至少1.5倍均量）+ 2-3根K線收在區間外

實例：

  • 方法一：某股在100-110元區間，價格跌到100元附近出現錘子線+RSI=25 → 做多，停損97元（3%），目標108元
  • 方法二：某股帶量突破110元阻力，成交量放大2倍，2根K線收在111-112元 → 回測110元不破後做多，停損108元，目標120元

適用時機：

  • 方法一 → 區間明確且寬度足夠時（>1.5倍ATR）
  • 方法二 → 區間末期、突破訊號明確時

3. 區間交易K線反轉確認技巧

是什麼：單靠價格觸及支撐/阻力就進場勝率不高，搭配K線反轉型態可大幅提升可靠性。

怎麼做：

支撐區反轉訊號（做多）：
1. 錘子線：實體小、下影線為實體2-3倍 → 價格突破錘子線高點時買進
2. 牛市孕育型：長黑K後接一根小K線，小K線完全在長黑K範圍內 → 突破長黑K高點時買進
3. 隨機指標確認：%D低於20 + 上述反轉型態 → 訊號可靠度大幅提升

阻力區反轉訊號（做空）：
1. 射擊之星：實體小、上影線為實體2-3倍 → 價格跌破射擊之星低點時賣出
2. 熊市孕育型：長紅K後接一根小K線 → 跌破長紅K低點時賣出
3. 隨機指標確認：%D高於80 + 上述反轉型態 → 訊號可靠度大幅提升

重要原則：如果反轉型態出現在區間中間而非邊界附近，且隨機指標不在超買超賣區（%D在20-80之間），則該訊號不可靠，不應交易。

實例：

  • 支撐區（100元）出現錘子線（低點98、收盤100、實體0.5元、下影線1.5元）+ KD的%D=18 → 突破100.5元時買進，停損97.5元（錘子線低點下方）
  • 阻力區（110元）出現射擊之星（高點112、收盤110、實體0.5元、上影線2元）+ KD的%D=82 → 跌破109.5元時賣出，停損112.5元

適用時機：每次區間交易進場前都應等待K線反轉確認，不要只看價格觸及支撐/阻力就進場

4. 區間交易風險管理：突破怎麼辦

是什麼：區間交易最大的風險是「區間被突破」，停損和應對策略決定你的存活率。

怎麼做：

停損設定三原則：
1. 固定ATR停損：停損設在支撐/阻力外1個ATR（如ATR=15元，支撐100元 → 停損85元）
2. 百分比停損：停損設在區間邊界外2-3%（如支撐100元 → 停損97-98元）
3. 結構性停損：停損設在最近的波段結構低點/高點外

突破後的應對：
1. 假突破處理：價格突破區間後1-2天內回到區間內 → 原區間仍有效，可繼續區間交易，但區間邊界可能需要微調
2. 真突破處理：價格帶量突破且2-3根K線收在區間外 → 區間結束，停損被觸發，改用趨勢策略
3. 突破後回測：真突破後等回測區間邊界（支撐變阻力或阻力變支撐）不破 → 新的順勢進場點

ATR的輔助判斷：

  • ATR持續下降 → 區間正在收縮，突破越來越近，減少區間交易頻率
  • ATR突然放大 → 可能即將突破，觀望為主
  • ATR穩定 → 區間穩固，可放心交易

實例：

  • 某股在100-110元區間，ATR=8元
  • 做多停損設92元（支撐-1ATR），做空停損設118元（阻力+1ATR）
  • 若價格帶量突破110元且ATR從8放大到12 → 判定真突破，停損出場，等回測110元不破後做多

適用時機：每次區間交易都必須預設突破的應對方案，不能只想區間內的情況

二、ORB 開盤區間突破策略實戰

5. ORB策略核心邏輯

是什麼：Opening Range Breakout（開盤區間突破），由Larry Williams推廣的經典當沖策略，利用開盤後一段時間內形成的價格區間，突破後順勢進場。特別適合期貨和股票當沖。

怎麼做：

標準ORB策略：
1. 定義開盤區間：通常取開盤後30分鐘（9:00-9:30）的最高價和最低價
2. 突破做多：價格突破開盤區間上緣 → 做多
3. 突破做空：價格跌破開盤區間下緣 → 做空
4. 停損設定：固定點數（如台指期100點）或區間寬度的50%
5. 當日平倉：13:30前全部平倉，不留倉過夜

Larry Williams變形版ORB：
1. 計算前日振幅：前日高點 - 前日低點 = value1
2. 計算突破帶：UP = 開盤價 + value1 × N；DN = 開盤價 - value1 × N（N為可調整參數，通常0.3-0.5）
3. 突破進場：價格突破UP做多、跌破DN做空
4. 兩大關鍵因子：基準價（開盤價或昨日高低均價）+ 波動度（昨日振幅或ATR）

實例：

  • 標準ORB：台指期9:00-9:30區間為19500-19580，9:35價格帶量突破19580 → 做多，停損19530（區間中點），目標19650
  • Williams版ORB：昨日振幅200點，今日開盤19500，N=0.4 → UP=19580、DN=19420，突破19580做多

適用時機：當沖交易、開盤後30分鐘觀察區間、9:30後突破進場；不適合盤整無方向的日子

注意事項：

  • N值越大（如0.5），過濾越多假突破，但進場點越差
  • N值越小（如0.2），進場點好但假突破多
  • 建議用ATR替代前日振幅，ATR更穩定

6. ORB策略搭配支撐壓力的加強版

是什麼：將ORB與日K級別的支撐壓力結合，提升突破的有效性判斷。

怎麼做：
1. 預先標記關鍵價位：盤前標記前日高低點、近期支撐壓力位、均線位置
2. ORB突破方向與大級別一致：ORB向上突破 + 日K級別在支撐位附近 → 做多勝率更高
3. ORB突破方向與大級別矛盾：ORB向上突破但日K級別在壓力位附近 → 小心假突破，停損設緊
4. 搭配成交量確認：突破時成交量至少1.5倍均量 → 有效突破；量縮突破 → 假突破機率高

實例：

  • 強訊號：台指期ORB向上突破19580，且日K級別在5日均線附近獲得支撐 → 做多信心高
  • 弱訊號：台指期ORB向上突破19580，但日K級別面臨20000整數關卡壓力 → 停損設緊（30點）

適用時機：每次ORB進場前都應對照日K級別的支撐壓力，不要只看開盤區間

三、多策略組合設計：降低回撤的終極方案

7. 為什麼需要多策略組合

是什麼：單一策略不管多好，都有失效期。多策略組合是利用不同策略之間的低相關性或負相關性，讓一個策略虧損時另一個策略賺錢，從而平滑整體資金曲線、降低最大回撤。

怎麼做：
1. 核心原理：馬可維茲投資組合理論——相同報酬率下風險最低、相同風險下報酬率最高的組合才是效率組合
2. 關鍵指標：策略間相關係數越低越好（理想值<0.3），負相關最好
3. 組合效果：2個不相關策略組合 → 波動降低約30%；3個不相關策略 → 降低約45%
4. 風險分散公式：組合風險 ≈ 單策略風險 × √(1/策略數) × (1+平均相關係數)

實例：

  • 單一趨勢策略：年化報酬20%、最大回撤-35%
  • 趨勢策略+區間策略組合：年化報酬18%、最大回撤-18%（回撤大幅降低）
  • 趨勢策略+區間策略+籌碼策略三合一：年化報酬19%、最大回撤-12%

適用時機：任何有2種以上策略的交易者都應考慮組合；只用1種策略的人風險最集中

8. 五大策略類型與相關性分析

是什麼：不同類型的策略在不同的市場環境下表現不同，了解它們的相關性才能設計出好的組合。

怎麼做（五種策略特性）：

策略一：趨勢策略（順勢）

  • 獲利環境：趨勢明確（佔市場30%時間）
  • 虧虧損環境：盤整震盪（佔市場70%時間）
  • 特性：低勝率（30-40%）、高賺賠比（3:1以上）

策略二：區間/逆勢策略

  • 獲利環境：盤整震盪（佔市場70%時間）
  • 虧損環境：趨勢明確（佔市場30%時間）
  • 特性：高勝率（60-70%）、低賺賠比（1:1至2:1）

策略三：籌碼策略

  • 獲利環境：法人動向明確、主力操作
  • 虧損環境：籌碼凌亂、假外資、隔日沖
  • 特性：與技術面策略相關性低，因為基於不同數據源

策略四：通道/突破策略

  • 獲利環境：區間突破後的趨勢展開
  • 虧損環境：假突破
  • 特性：與趨勢策略正相關、與區間策略負相關

策略五：均線/移動停損策略

  • 獲利環境：趨勢持續
  • 虧虧損環境：頻繁穿越均線的震盪市
  • 特性：與趨勢策略高度正相關

相關性矩陣：

趨勢1.0-0.50.20.60.8
區間-0.51.00.1-0.3-0.4

結論：趨勢+區間是最佳互補組合（相關性-0.5），籌碼策略與其他策略相關性最低（0.1-0.2），是理想的第三策略

適用時機：設計多策略組合時，先用相關性矩陣篩選，避免組合內策略高度正相關

9. 多策略組合配置實戰三模型

10. 多策略組合管理三大原則

是什麼：有了多策略組合不等於從此穩賺，管理不當一樣會出問題。

怎麼做：

原則一：策略失效監控
1. 每月檢視每個策略的滾動夏普比率（近3個月）
2. 若某策略夏普比率連續2個月<0 → 該策略暫停，資金分配到其他策略
3. 若某策略夏普比率連續3個月<0 → 該策略永久移除，重新研發替代策略

原則二：動態再平衡
1. 每季根據市場環境調整策略權重
2. 趨勢市來臨時（ADX>25、均線發散）→ 增加趨勢策略權重至50%
3. 盤整市來臨時（ADX<20、均線糾結）→ 增加區間策略權重至50%
4. 但單一策略權重不超過60%，避免過度集中

原則三：相關性定期檢查
1. 每半年計算策略間的相關係數
2. 若兩個策略相關係數>0.7 → 實質上是同一種策略，保留績效較好的那個
3. 策略組合至少需要2個相關性<0.3的策略才算有效分散

實例：

  • 1月：趨勢策略夏普=1.2、區間策略夏普=0.8 → 正常
  • 4月：趨勢策略夏普=-0.3、區間策略夏普=1.5 → 趨勢策略暫停，區間策略加大
  • 7月：趨勢策略恢復（夏普=0.5）→ 重新啟用，但權重降低

心得

今天學的區間交易和多策略組合，讓我對「全天候操作系統」有了更清晰的藍圖：

1. 區間交易是趨勢交易的最佳互補。市場70-80%的時間在盤整，但大部分散戶只會趨勢策略，在盤整中被雙巴。區間交易教會我一個重要的思維轉換：盤整不是「沒行情」，而是「另一種行情」。支撐買、阻力賣，配合K線反轉型態和KD/RSI確認，勝率可以達到60-70%。但風險也很明確——一旦區間被突破，停損必須果斷執行。

2. ORB開盤區間突破是當沖的利器。Larry Williams用這個策略創造了驚人績效，核心邏輯很簡單：開盤30分鐘的區間代表了多空初期的博弈結果，突破方向往往代表當日主流方向。關鍵是N值的設定——太小吃假突破、太大錯過行情。我會想用ATR替代前日振幅，因為ATR更穩定、不受單日異常波動影響。

3. K線反轉確認是區間交易的保險。只看價格觸及支撐/阻力就進場太魯莽，搭配錘子線、射擊之星、孕育型等K線型態，再加上KD/RSI超買超賣確認，可以把勝率從50%提升到65%以上。這跟之前學的「多維度確認」概念一致——單一訊號不可靠，多重確認才可靠。

4. 多策略組合是降低回撤的數學解。不是感覺會降低，是數學上一定會降低——只要策略間相關性夠低。趨勢策略和區間策略的相關性約-0.5，這意味著一個虧損時另一個很可能獲利。組合後最大回撤可以從35%降到15-20%，這對心理素質的要求大幅降低——不用再忍受腰斬般的痛苦。

5. 組合管理的核心是「動態再平衡」。不是設好比例就永遠不動，而是根據市場環境動態調整。趨勢市加大趨勢策略、盤整市加大區間策略，就像開車一樣——直路踩油門、彎路踩煞車。但單一策略權重不超過60%，這是避免「押錯邊」的保險。

6. 策略失效監控比策略本身更重要。再好的策略也會失效（市場結構改變、太多人使用同一策略），關鍵是「失效時要及時停損退出」。每月檢視夏普比率、連續2個月<0就暫停、3個月<0就淘汰——這個規則比任何單一策略都重要，因為它保護你的整體組合不被單一失效策略拖垮。
