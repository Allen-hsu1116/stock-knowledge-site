Gamma Scalping——不押方向、靠波動賺錢的進階技術

是什麼：Gamma Scalping 是一種選擇權進階策略，透過買入跨式組合（Long Straddle）+ 動態調整 Delta Neutral，在標的價格來回波動中不斷「高拋低吸」獲利。不管漲跌，只要波動夠大就能賺錢。

怎麼做：

1. 建倉：買入平值 Call + 平值 Put，構成 Long Straddle

  • 選擇到期前 1-3 個月的合約（Gamma 够大、Theta 成本可控）

2. 調整 Delta Neutral 的四階段：

  • 階段一（標的上漲）：Call 變實值 Delta 升至 0.5-1，Put 變虛值 Delta 降至 -0.5~0，組合總 Delta 變正。→ 平倉部分獲利 Call 或買入便宜的 Put，重新 Delta 中性

3. 調整閥值設定（何時調 Delta）：

  • 方法一：以時間週期（如每 30 分鐘、每日收盤）

4. 成本控制：用 Gamma/Theta 比值衡量性價比

  • 分子 Gamma = 獲利源泉（價格波動帶來的收益）

實例：

  • 情境：台指期在 20000 點附近，預期將有大幅波動（如 FOMC 會議前）
  • 操作：買入 20000 點 Call + Put 各 1 口，總成本約 500 點權利金
  • 指數漲至 20200：平倉 0.3 口 Call（獲利約 120 點），或加買 Put
  • 指數回落至 20000：平倉部分 Put（獲利約 80 點），或加買 Call
  • 來回 4 次調整，累積 scalping 利潤約 400 點
  • 最差情況：指數不動如山，每天損失 Theta，最終權利金歸零

適用時機：

  • ✅ 預期大幅波動但方向不明（財報季、FOMC、選舉前）
  • ✅ 隱含波動率在低位（買入成本低）
  • ❌ 盤整無波動的死水盤（Theta 吃光利潤）
  • ❌ 隱含波動率已高（建倉成本太貴）

Vega 對沖——Gamma Scalping 的隱形風險

是什麼：Long Straddle 同時有正 Gamma 和正 Vega。Gamma Scalping 靠 Gamma 賺錢，但 Vega 暴露是風險——如果波動率下降，即使價格在波動，組合仍可能虧錢。Vega 對沖就是消除這個風險。

怎麼做：

1. 核心原理：Gamma 和 Vega 雖然同向（買權都為正），但期限結構不同

  • 近月期權：Gamma 大、Vega 小（到期日近，波動率影響小）

2. 對沖操作：賣出遠月平值期權（Sell Far ATM Option）

  • 只損失小部分 Gamma（遠月 Gamma 本來就小）

3. 不完美之處：波動率期限結構問題

  • 黑天鵝事件：近月 IV 飆升遠大於遠月，Vega 對沖不完美

實例：

  • 情境：買入近月（1個月到期）20000 Straddle，Vega = 80
  • 操作：賣出遠月（3個月到期）20000 Call，Vega = -60
  • 淨 Vega = 20（大幅降低，但未完全消除）
  • 好處：波動率下跌時虧損從 80 點/1% 降到 20 點/1%
  • 代价：犧牲了約 15% 的 Gamma（遠月賣方吃掉的）

適用時機：

  • ✅ 波動率可能下降的行情（建倉後 IV 回落）
  • ✅ 長期持有 Gamma Scalping 部位時
  • ❌ 預期波動率還會上升（Vega 正值對你有利，不需要對沖）

隱含波動率（IV）與 VIX 指數實戰判讀

是什麼：隱含波動率（Implied Volatility, IV）是從選擇權市場價格反推出來的波動率，反映市場對未來波動的預期。VIX 指數是整體選擇權市場 IV 的加權平均，被稱為「恐慌指數」。

怎麼做：

1. IV 與歷史波動率（HV）的區別：

  • HV = 過去的波動（年化標準差），只看後照鏡

2. VIX 與行情的四種組合：

  • VIX漲 + 行情漲 = 軋空行情！最兇猛的多頭，Call Put 權利金同漲

3. VIX 當沖實戰判斷：

  • VIX 一路走高 → 今日走單邊行情，幅度大，當沖加碼獲利部位

4. 裸賣遠月選擇權為什麼超危險：

  • 遠月 Vega 值遠大於近月（波動率影響更大）

實例：

  • 情境：台指 VIX 長期均值約 19.5
  • VIX = 12（極低）：買方好時機，權利金便宜；賣方風險高（隨時可能飆升）
  • VIX = 30（偏高）：賣方好時機，權利金貴；買方成本高
  • 2015/4/24 指數站上萬點軋空行情：VIX 大漲 + 指數大漲，Call Put 同漲
  • FOMC 前一天 VIX = 25：會後決策出爐 IV 從 25 降到 18，賣方大賺 Theta + Vega，買方被 Double Kill

適用時機：

  • ✅ 每次操作選擇權前都要看 IV/VIX（就像開車看時速表）
  • ✅ 用 IV 高低決定做買方還是賣方
  • ✅ 用 VIX 走勢輔助當沖判斷盤勢
  • ❌ 不要把 VIX 當方向指標（VIX 只衡量波動程度，不指明方向）

Delta Neutral 策略體系——從基礎到進階

是什麼：Delta Neutral 是讓整體選擇權部位的 Delta 值接近 0，不偏多也不偏空，靠波動率而非方向獲利。跨式（Straddle）、勒式（Strangle）都是基於此原理。

怎麼做：

1. Delta 值核心概念：

  • Call Delta：0 ~ 1（價平約 0.5，深價內趨近 1）

2. 配平 Delta 的方法：

  • 等量買入同履約價的 Call + Put → 自動接近 Delta 中性

3. Gamma 的角色：

  • Gamma = Delta 對標的價格的敏感度（Delta 的變化率）

實例：

  • 情境：買入 20000 點 Call（Delta=0.5, Gamma=0.002）+ 20000 點 Put（Delta=-0.5, Gamma=0.002）
  • 建倉時總 Delta = 0（中性）
  • 指數漲至 20100：Call Delta → 0.6，Put Delta → -0.4，總 Delta = +0.2（偏多）
  • 調整：買入 0.2 口 Put 或賣出 0.2 口 Call → 重新 Delta 中性

適用時機：

  • ✅ 不確定方向但預期大波動
  • ✅ 想靠波動率交易而非方向交易
  • ❌ 不願意頻繁調整的人（Delta Neutral 需要動態維護）
  • ❌ 手續費吃重的帳戶（每次調整都有交易成本）

心得

Gamma Scalping 是我目前學到最「帥」的選擇權策略——不押方向，靠波動賺錢，聽起來像交易界的「不管漲跌我都賺」。但現實是殘酷的：這策略的成本是 Theta，每次調整有手續費，Vega 風險也沒辦法完美對沖。本質上是用「時間成本」換取「波動收益」。

幾個關鍵認知更新：
1. Vega ≠ Gamma：做 Gamma Scalping 時最大的誤區是忽略 Vega。Gamma 是加速度（價格變動的快慢），Vega 是路程（波動率的高低）。波動率下降時，即使價格在波動，你仍然可能虧錢——這點很多人沒意識到
2. 遠月賣方超危險：遠月 Vega 值大 + 流動性差 = 波動率飆升時跑不掉。這解釋了為什麼很多人覺得「賣遠月收時間價值很安全」，結果一次黑天鵝賠掉幾年利潤
3. VIX 四象限比單純看漲跌有用：VIX漲+行情漲=軋空行情，這是最被低估的組合。散戶通常只看指數漲跌，忽略波動率傳遞的信號
4. Gamma/Theta 比值是核心 KPI：選擇權交易不是只有買賣，要算性價比。Gamma/Theta 高的合約才是好合約
