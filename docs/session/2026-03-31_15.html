盈餘品質分析

第 21 回合學習 - 盈餘品質分析

文章：

  • 觀察「盈餘品質指標」，輕鬆避開作假帳的地雷股
  • 財報分析 第34章 盈餘品質指標

核心知識

一、盈餘品質是什麼？

定義：用過去盈餘來預估未來盈餘的能力。盈餘品質高，代表公司盈餘真實可靠、可持續，能作為投資決策依據。

為什麼重要：

  • 博達科技掏空案後，投資人發現盈餘數字可以動手腳
  • 只看 EPS 不夠，要檢查「盈餘與其他營業項目的關係」
  • 高品質盈餘 = 大部分是現金組成 + 有持續性 + 可預測

二、盈餘品質分析步驟

五步驟判斷法：
1. 確認會計方法是否適當 — 是否頻繁更換會計政策？
2. 查閱會計師查核報告書 — 是否有保留意見？
3. 確認公司是否重編財報 — 過去有無財報重編紀錄？
4. 計算公司營運風險 — 財務結構是否穩健？
5. 量化指標分析 — 計算七大盈餘品質指標

三、七大盈餘品質指標（全部「正數」較好）

指標 1：存貨指標

公式：

存貨增加% = (本年存貨 - 前兩年平均存貨) ÷ 前兩年平均存貨
營收增加% = (本年營收 - 前兩年平均營收) ÷ 前兩年平均營收
存貨指標 = 營收增加% - 存貨增加%

判斷：

  • 指標 > 0：存貨控管適當，盈餘品質佳
  • 指標 < 0：存貨累積速度 > 銷售成長，可能有滞銷問題

實例：

  • 情境：某公司營收成長 20%，存貨增加 15%
  • 計算：存貨指標 = 20% - 15% = +5%（正向）
  • 若存貨增加 30%：指標 = 20% - 30% = -10%（負向，存貨堆積過快）

警訊：存貨增加遠高於營收成長 → 可能銷售不順、產品滯銷

指標 2：應收款項指標

公式：

應收款項增加% = (本年應收款項 - 前兩年平均應收款項) ÷ 前兩年平均應收款項
應收款項指標 = 營收增加% - 應收款項增加%

判斷：

  • 指標 > 0：應收款項催收良好，產品銷售佳
  • 指標 < 0：應收款項增加 > 營收成長 → 可能用賒銷衝業績

實例：

  • 情境：公司營收成長 15%，應收帳款增加 25%
  • 計算：應收款項指標 = 15% - 25% = -10%（負向）
  • 解讀：可能放宽信用政策衝業績，壞帳風險增加

警訊：應收帳款異常增加 → 可能財報美化、客戶關聯交易

指標 3：毛利指標

公式：

毛利增加% = (本年營業毛利 - 前兩年平均營業毛利) ÷ 前兩年平均營業毛利
毛利指標 = 毛利增加% - 營收增加%

判斷：

  • 指標 > 0：獲利能力成長超過營收成長，盈餘品質佳
  • 指標 < 0：獲利能力跟不上營收成長 → 成本控制有問題

實例：

  • 情境：營收成長 20%，毛利成長 25%
  • 計算：毛利指標 = 25% - 20% = +5%（正向）
  • 若毛利只成長 10%：指標 = 10% - 20% = -10%（負向）

警訊：營收成長但毛利衰退 → 可能削價競爭、成本失控

指標 4：銷管費用指標

公式：

銷管費用增加% = (本年銷管費用 - 前兩年平均銷管費用) ÷ 前兩年平均銷管費用
銷管費用指標 = 營收增加% - 銷管費用增加%

判斷：

  • 指標 > 0：銷管費用未因營收增加而暴增，管理效率佳
  • 指標 < 0：銷管費用增加 > 營收成長 → 可能資源浪費

實例：

  • 情境：營收成長 30%，銷管費用增加 20%
  • 計算：銷管費用指標 = 30% - 20% = +10%（正向）
  • 若銷管費用增加 40%：指標 = 30% - 40% = -10%（負向）

警訊：銷管費用暴增但營收沒跟上 → 管理效率差、亂花錢

指標 5：備抵壞帳指標

公式：

備抵壞帳增加% = (本年備抵壞帳 - 前兩年平均備抵壞帳) ÷ 前兩年平均備抵壞帳
備抵壞帳指標 = 備抵壞帳增加% - 應收款項增加%

判斷：

  • 指標 > 0：備抵壞帳提列充足，財報保守誠實
  • 指標 < 0：備抵壞帳準備不足 → 可能有修飾財報嫌疑

實例：

  • 情境：應收款項增加 20%，備抵壞帳增加 25%
  • 計算：備抵壞帳指標 = 25% - 20% = +5%（正向）
  • 若備抵壞帳只增加 10%：指標 = 10% - 20% = -10%（負向）

警訊：應收增加但備抵壞帳沒跟上 → 可能低估壞帳風險、美化盈餘

指標 6：研發費用指標

公式：

研發費用增加% = (本年研發費用 - 前兩年平均研發費用) ÷ 前兩年平均研發費用
研發費用指標 = 研發費用增加% - 同業研發費用增加%

判斷：

  • 指標 > 0：企業研發費用高於同業，雖短期不利盈餘，但長期有競爭力
  • 指標 < 0：研發投入低於同業 → 可能犧牲長期競爭力換短期盈餘

實例：

  • 情境：某科技公司研發費用增加 15%，同業平均增加 10%
  • 計算：研發費用指標 = 15% - 10% = +5%（正向）

注意：研發費用也不能過高，要看產業特性

指標 7：員工生產力指標

公式：

員工生產力指標 = (本期員工平均營收 - 前期員工平均營收) ÷ 前期員工平均營收
員工平均營收 = 公司總營收 ÷ 員工人數

判斷：

  • 指標 > 0：員工生產力成長，人力素質提升，有利盈餘
  • 指標 < 0：員工生產力下降 → 可能人員膨脹或效率低落

實例：

  • 情境：去年員工平均營收 500 萬，今年 550 萬
  • 計算：員工生產力指標 = (550 - 500) ÷ 500 = +10%（正向）

四、綜合判斷法

評分方式：

  • 7 個指標中，正數越多 → 盈餘品質越好
  • 建議至少 5 個以上指標為正數才考慮投資

紅旗警示：

  • 存貨、應收款項同時異常增加 → 可能虛增營收
  • 備抵壞帳明顯不足 → 可能低估壞帳風險
  • 毛利指標連續為負 → 成本控制有問題

五、實戰應用

適用時機：

  • 選股時排除盈餘品質差的公司
  • 財報公佈後檢視公司體質
  • 發現公司盈餘與股價走勢背離時

不適用時機：

  • 新創公司（前兩年無財務資料）
  • 產業劇烈變動期（基準期失真）
  • 併購重整中的公司（財務數字不具比較性）

心得

盈餘品質指標是一套很實用的「財報健康檢查表」。

它的核心邏輯是：盈餘不是獨立存在的數字，而是和其他財務項目有合理關聯。如果某個項目（如存貨、應收帳款）的變動脫離了營收成長的合理範圍，就要警覺。

最實用的是前五個指標：存貨、應收款項、毛利、銷管費用、備抵壞帳。這五個指標只要認真算一遍，就能抓出很多「看起來賺錢但其實有問題」的公司。

關鍵心得：

  • 指標全正不一定好公司，但指標多負一定有問題
  • 重點是「連續性」——連續兩三年指標異常比單一年更重要
  • 要和同業比較，不能只看自己

之後看財報時，先算這七個指標再決定要不要深入研究。
