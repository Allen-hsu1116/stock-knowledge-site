一、順勢加碼 vs 逆勢加碼

1. 順勢加碼（正確做法）

是什麼：前面的單子已經獲利時，才沿著同方向再加碼。行情上漲時加碼做多，行情下跌時加碼做空。

怎麼做：
1. 核心原則：只加碼獲利的部位——這是加碼的黃金法則
2. 進場邏輯：初始單有浮盈 → 趨勢確認 → 在回調或突破關鍵位時加碼
3. 風險控制：浮盈增加保證金維持率，加碼幾乎不可能爆倉
4. 出場邏輯：追蹤停損隨價格上移，保護已有獲利

實例：

  • 情境：台積電(2330)在600元建立多頭部位，漲到630元出現回調到620元
  • 操作：620元回調支撐確認後加碼，停損統一設610元（前低點下方）
  • 900點趨勢實例：初始1單位賺900點 + 第一次加碼賺600點 + 第二次加碼賺300點 = 總計1800點，是單持初始部位的2倍

適用時機：波段趨勢明確時，搭配順勢策略效果最好；盤整行情不適用

優缺點：

  • ✅ 風險低、小賠大賺、長期期望值高
  • ❌ 勝率低（30-40%）、獲利慢、有獲利回吐的「雙倍痛苦」

2. 逆勢加碼（攤平，高風險）

是什麼：前面的單子虧損時，反而加碼——行情下跌時加碼做多（攤平成本），行情上漲時加碼做空。

怎麼做（如果你硬要做）：
1. 必須設停損：不是凹單，是事先計算好最大可承受虧損
2. 計算總風險：所有加碼單的潛在虧損總和不得超過資金的5-10%
3. BE點思維：加碼後計算損益兩平點，知道價格回到哪裡就能打平

實例：

  • 情境：500元買入1張，跌到400元再加碼1張
  • BE點 = (500×1 + 400×1) / 2 = 450元
  • 若400元加碼2張：BE點 = (500×1 + 400×2) / 3 = 433元
  • 風險：若價格繼續跌到300元，1張加碼虧200元、2張加碼虧400元

適用時機：嚴格說不建議。若使用，必須有明確停損且可接受該虧損金額

優缺點：

  • ✅ 勝率高（60-70%）、心理感覺較舒服
  • ❌ 容易凹單越攤越平、風險不可控、一次大行情可能爆倉

3. 順勢 vs 逆勢期望值比較

是什麼：評估加碼方式好壞的終極標準不是勝率，是期望值。

怎麼做：
1. 順勢交易期望值：勝率30%，每次賺500元或虧100元 → 期望值 = 500×0.3 - 100×0.7 = 80元/次
2. 逆勢交易期望值：勝率70%，每次賺100元或虧100元 → 期望值 = 100×0.7 - 100×0.3 = 40元/次
3. 結論：勝率高不代表賺得多，賺賠比才是決定期望值的關鍵

適用時機：每次評估策略時都應計算期望值，而非只看勝率

二、加碼量三種模式

4. 正金字塔加碼（推薦）

是什麼：每次加碼的量遞減，越加越小。初始倉位最大，後續加碼逐步縮小。

怎麼做：
1. 初始倉位：1單位
2. 第一次加碼：0.7單位（乘以0.7）
3. 第二次加碼：0.49單位（乘以0.7）
4. 第三次加碼：0.34單位（乘以0.7）
5. 搭配追蹤停損，整體風險可控

實例：

  • 情境：100萬資金，初始投入20萬，第一次加碼14萬，第二次9.8萬，第三次6.9萬
  • 總投入：50.7萬，但最新加碼部位最小
  • 好處：行情反轉時獲利回吐最少，因為越高的價位部位越小

適用時機：順勢加碼的最佳搭檔，適合怕獲利回吐的交易者

5. 倒金字塔加碼（危險）

是什麼：每次加碼量遞增，越加越大。這就是馬丁格爾策略的加碼方式。

怎麼做：
1. 初始：1單位
2. 第一次加碼：2單位
3. 第二次加碼：4單位
4. 第三次加碼：8單位

風險：

  • 順勢用倒金字塔：趨勢稍微反轉就虧掉前面所有利潤
  • 逆勢用倒金字塔（攤平）：價格一去不回頭就爆倉

適用時機：不建議。逆勢攤平如果硬要做，至少要設總停損

6. 固定交易量加碼

是什麼：每次加碼量與初始單相同，簡單直接。

怎麼做：
1. 初始：1單位
2. 每次加碼：1單位
3. 搭配追蹤停損，每張單給相同的追蹤停損

適用時機：固定點數停損策略時自然產生；簡單好執行

三、BE點（損益兩平點）計算與應用

7. BE點是加碼的核心概念

是什麼：所有加碼單合併計算後的均價，也就是整體部位損益兩平的價格。

怎麼做：
1. 公式：BE點 = (手數1×價格1 + 手數2×價格2 + ...) / 總手數
2. 實務意義：不用一筆一筆算盈虧，直接看市價與BE點的距離就知道整體盈虧
3. 加碼決策：加碼會改變BE點，順勢加碼會墊高BE點（多單），逆勢加碼會拉低BE點

實例：

  • 情境：600元買1張台積電，630元加碼1張，650元再加碼1張
  • BE點 = (600+630+650) / 3 = 626.7元
  • 停損設在BE點下方3% = 607.9元，確保即使反轉也不虧太多

適用時機：每次加碼後都要重新計算BE點，據此調整停損

四、條件式加碼進階

8. 單純條件式加碼

是什麼：只要技術訊號出現就直接加碼，不看價格距離。

怎麼做：
1. 首次出現黃金交叉 → 買進第一張
2. 第二次出現黃金交叉 → 買進第二張
3. 第三次出現黃金交叉 → 買進第三張

風險：盤整時可能連續在差不多的價位出現交叉訊號，變成在同一個價格疊加部位

適用時機：趨勢明確時；盤整時容易過度加碼

9. 條件式＋距離加碼（推薦）

是什麼：加碼必須同時滿足「技術訊號」和「與上一張單拉開足夠距離」兩個條件。

怎麼做：
1. 首次黃金交叉 → 買進第一張（600元）
2. 第二次黃金交叉 + 價格必須高於上一張單至少3% → 才加碼（620元以上）
3. 第三次黃金交叉 + 價格必須高於第二張單至少3% → 才加碼（640元以上）

實例：

  • 盤整時黃金交叉出現在605元，距離上一張單600元只有0.8% → 不加碼
  • 趨勢展開後黃金交叉出現在625元，距離上一張600元有4.2% → 加碼

適用時機：所有條件式加碼都應加入距離條件，避免盤整中過度加碼

五、加碼時機四大黃金位

10. 最佳加碼進場點

是什麼：刀疤老二提出的加碼時機選擇，決定了加碼策略的成敗。

怎麼做：
1. 趨勢確認後的第一次回調：多頭趨勢中第一次拉回支撐位 → 最佳加碼點
2. 突破關鍵阻力後的回測：帶量突破前高後回測不破 → 加碼做多
3. 突破支撐後的回測：帶量跌破前低後反彈不過 → 加碼做空
4. 均線回踩確認：價格回踩20MA或60MA獲得支撐 → 加碼做多

實例：

  • 情境：聯發科(2454)從800元突破前高830元，回測835元不破
  • 操作：835元加碼做多，停損設825元（回測前高下方），目標880元
  • 風報比：10:1（風險10元，目標45元）

適用時機：搭配正金字塔加碼，在低風險高回報的位置加碼

六、加碼常見三大錯誤

11. 避開這三個坑

是什麼：刀疤老二指出的加碼三大致命錯誤。

怎麼做（避免方法）：
1. 逆勢加碼（最大殺手）：虧損部位絕不加碼，這是李佛摩的核心教訓
2. 過早加碼：趨勢尚未確立就急於擴大倉位 → 等趨勢確認後才加碼
3. 情緒化加碼：恐懼錯過行情（FOMO）或想要快速翻本 → 建立加碼SOP，不憑感覺加碼

實例：

  • 錯誤：600元買進，跌到580元「攤平」再加1張，跌到560元又加2張 → BE點586元，價格繼續跌到500元虧損86元×3=258元
  • 正確：600元買進，漲到630元回調620元加碼 → 即使跌回600元也只是獲利回吐，不會虧到本金

適用時機：每次想加碼前都要問自己「這是獲利加碼還是虧損攤平？」

心得

今天學的加碼哲學雖然之前有學過金字塔加碼，但這次深入到「方向」和「量」兩個維度，收穫很大：

1. 順勢加碼vs逆勢加碼不是選擇題，是生死題。逆勢加碼（攤平）的本質是放任風險擴大、賺錢時急著收，這跟所有正期望值系統背道而馳。順勢加碼雖然有獲利回吐的雙倍痛苦，但那是「正確的痛苦」——你在對的方向承受波動，而不是在錯的方向越陷越深。

2. 正金字塔加碼是最務實的選擇。越加越小不是因為不夠勇敢，而是因為趨勢越走越遠時繼續前進的機率在降低。這跟「承認不確定性」的思維一致——你對近端有信心，對遠端要保持敬畏。

3. BE點是加碼的靈魂。不管加了幾張單，全部簡化成一個均價來看盈虧。這讓複雜的多單管理變得清晰——停損就設在BE點下方一定比例，不用一筆一筆算。

4. 條件式加碼一定要加距離條件。單純看訊號加碼，盤整時會在同一價格附近越加越多，風險全部堆在一起。加了距離條件後，只有在真正展開的行情才會加碼，盤整時自動過濾。

5. 加碼前提：你已經能穩定執行停損。刀疤老二說得好——加碼是為了放大成功，不是彌補失敗。如果你連單筆停損都執行不好，加碼只會加速你的死亡。
