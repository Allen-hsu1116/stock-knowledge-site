OBV 能量潮指標基本概念

是什麼：OBV（On-Balance Volume）是將成交量累積化的一種技術指標，用來判斷買賣力道與資金流向。華爾街名言：「股市裡充斥著各式各樣的騙子，只有成交量是唯一例外」——OBV 就是把成交量變成「趨勢訊號」的工具。

怎麼做：
1. 股價上漲日 → 把當天成交量加進 OBV
2. 股價下跌日 → 把當天成交量減掉 OBV
3. 股價不變日 → OBV 不動
4. 每日累積形成一條「OBV 線」

實例：

  • 情境：昨天 OBV 是 10,000；今天股價上漲，成交量是 2,000
  • 計算：今天 OBV = 10,000 + 2,000 = 12,000
  • 解讀：能量條上升，買氣增強

適用時機：

  • 適合波段趨勢判斷，不適合超短線當沖
  • 成交量大的股票效果較好，冷門股容易有雜訊
  • 作為輔助指標，搭配 K 線、均線、支撐壓力一起看

OBV 四種交易訊號

1. 背離訊號

是什麼：股價走勢與 OBV 走勢方向不一致，是趨勢可能反轉的警訊。

怎麼做（頂背離 - 賣出訊號）：
1. 觀察股價創新高
2. 檢查 OBV 是否同步創新高
3. 若 OBV 沒有創新高 → 頂背離，上漲缺乏資金支持
4. 考慮逢高賣出

怎麼做（底背離 - 買進訊號）：
1. 觀察股價創新低
2. 檢查 OBV 是否同步創新低
3. 若 OBV 沒有創新低 → 底背離，低檔有買盤承接
4. 考慮進場

實例：

  • 情境：某股票股價從 100 元漲到 120 元再漲到 130 元
  • OBV 表現：第一次高點 OBV 達 50,000，第二次 130 元時 OBV 只有 45,000
  • 解讀：頂背離，上漲動能不足，主力可能在出貨
  • 操作：考慮減碼或賣出

適用時機：找轉折點時使用，特別是股價經過一段漲勢或跌勢後

2. 同步訊號

是什麼：OBV 與股價同方向移動，表示趨勢健康。

怎麼做（上升同步）：
1. OBV 線緩慢上升
2. 股價同步上漲
3. 判斷：行情穩健往上，中長期看好
4. 操作：可以考慮加碼待漲

怎麼做（下降同步）：
1. OBV 線緩慢下降
2. 股價同步下跌
3. 判讀：行情緩跌，中長期看淡
4. 操作：賣出或手持現金觀望

適用時機：確認趨勢是否健康時使用

3. 急速變化訊號

是什麼：OBV 快速上升或下降，是趨勢可能反轉的強烈訊號。

怎麼做（急速上升 - 賣出訊號）：
1. 觀察 OBV 急速上升
2. 判斷：買分已經累積許多人，主力不會幫大家抬轎
3. 警訊：鋸狀曲線有掉頭向下跡象
4. 操作：考慮逢高賣出

怎麼做（急速下跌 - 觀望訊號）：
1. 觀察 OBV 急速下跌
2. 判斷：大量賣盤洶湧而出，轉為空方趨勢
3. 等待：OBV 在底部形成齒狀曲線
4. 操作：才考慮進場做短期反彈

實例：

  • 情境：某股票 OBV 一週內從 30,000 飆升到 80,000
  • 解讀：短線資金大量湧入，可能是主力拉高出貨的徵兆
  • 操作：不追高，甚至考慮減碼

適用時機：極端行情判斷，避免追高殺低

4. 累積訊號

是什麼：OBV 長期累積形成的高點或低點，常成為支撐壓力區。

怎麼做（累積高點突破）：
1. 找出 OBV 長期累積的高點（壓力區）
2. 觀察股價是否突破這個壓力
3. 若突破 → 後續漲勢更強勁有力

怎麼做（累積低點跌破）：
1. 找出 OBV 長期累積的低點（支撐區）
2. 觀察股價是否跌破這個支撐
3. 若跌破 → 後續跌勢更猛

適用時機：判斷支撐壓力位置時使用

OBV 實戰應用三招

1. 搭配 K 線型態判斷真假突破

是什麼：用 OBV 驗證 K 線突破是否可靠。

怎麼做：
1. 觀察 K 線突破壓力區
2. 檢查 OBV 是否同時創新高
3. OBV 創高 → 突破可靠，有量支撐
4. OBV 沒創高 → 可能是假突破，小心陷阱

實例：

  • 情境：股價從 50 元突破 55 元壓力
  • OBV 表現：若 OBV 也同步突破前高，表示資金真的進來
  • 操作：可以進場；若 OBV 沒跟上，保守觀望

適用時機：突破交易時驗證訊號可靠性

2. 搭配支撐壓力觀察領先訊號

是什麼：OBV 的支撐壓力有時會領先股價反應。

怎麼做：
1. 在 OBV 上畫趨勢線（跟畫股價趨勢線一樣）
2. 觀察 OBV 是否率先突破或跌破
3. OBV 突破下降趨勢線 → 資金回流訊號，股價可能跟進
4. OBV 跌破上升趨勢線 → 資金撤出訊號，股價可能下跌

適用時機：盤整區間判斷突破方向時使用

3. 搭配均線 MACD 多重驗證

是什麼：OBV 加上其他技術指標，提高準確度。

怎麼做：
1. OBV 上升 + 均線黃金交叉 → 成功率更高
2. OBV 下跌 + MACD 死亡交叉 → 特別小心
3. 多重驗證比單看 OBV 更可靠

適用時機：任何進出場決策時都可以搭配使用

OBV 四大盲點與陷阱

是什麼：OBV 不是萬能，使用時要注意四個陷阱。

1. 假突破陷阱

怎麼做：

  • OBV 一口氣突破，可能只是短線大單衝進來
  • 要確認突破是否持續，不要盲目追進

2. 無法區分買賣結構

怎麼做：

  • OBV 只反映資金流向的大方向
  • 不知道背後是誰在買誰在賣
  • 需搭配籌碼面分析（法人、主力買賣超）

3. 盤整時容易失真

怎麼做：

  • 股價長期在區間震盪時，OBV 也會上下震盪
  • 訊號混亂，不適合單獨使用
  • 等待明確趨勢再判斷

4. 容易受消息面影響

怎麼做：

  • 突發利多/利空會讓成交量暴增
  • OBV 可能急拉或急跌
  • 必須過濾消息面因素，看本質

OBV 使用三大建議

1. 不要單獨使用

怎麼做：OBV 是輔助工具，一定要跟價格型態、均線、MACD 等一起看。

2. 先找背離訊號

怎麼做：

  • 最簡單的用法就是「股價與 OBV 背離」
  • 股價創高但 OBV 沒跟上 → 保守
  • 股價創低但 OBV 未創新低 → 可能是低檔轉折

3. 適合波段不適合當沖

怎麼做：

  • OBV 是累積成交量，適合波段趨勢
  • 超短線用 OBV 容易落後反應
  • 當沖建議用 Tick 量或分時成交量

心得

OBV 指標的核心精神很酷：「成交量是唯一不說謊的」。價格可以被人為操控，但成交量需要真金白銀來堆疊。

以前以為 OBV 就是成交量，今天才理解差別：成交量只看「單日」，OBV 是「累積」起來的趨勢。這讓我想起主力作盤的邏輯——主力吸貨時，股價可能盤整，但 OBV 會悄悄上升，資金在偷偷流入。

最實用的是「背離」這招：股價創高但 OBV 沒跟上，就是主力在告訴你「快跑了」。這種訊號比技術面單純看股價更早發現問題。

不過 OBV 的盲點也要記住：盤整時容易失真，消息面會干擾。所以 SKILL.md 強調的「多重驗證」很重要——OBV 搭配均線、MACD、籌碼面一起看，才能過濾掉雜訊。

最後，累積高點和低點可以當支撐壓力看，這點我以前沒想過。OBV 不是只有一條線，它的歷史高點低點也有參考價值。
