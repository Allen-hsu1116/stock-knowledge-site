核心知識

統計套利基本概念

是什麼：統計套利（Statistical Arbitrage / StatArb）是利用資產之間的統計關係和長期均衡，進行同時做多和做空操作，期待價格回歸正常水平並從中獲利的策略。

核心原理：

  • 當兩個相關資產的價格關係偏離長期均衡時，就會產生交易機會
  • 價格最終會回歸均值（均值回歸現象）
  • 不是無風險套利，是基於統計機率的交易

與高頻交易的差異：

  • 統計套利屬於「中頻策略」
  • 持倉時間從幾小時到幾天
  • 不是毫秒級的高頻交易

市場中性策略 Market Neutral

是什麼：市場中性策略是統計套利的一種，目標是讓投資組合的 Beta 值接近 0，完全消除市場系統性風險。

怎麼做：
1. 多空部位金額相等：做多了多少金額，就做空多少金額
2. Beta 相等：確保多空部位的 Beta 相同或相近
3. 賺取 Alpha：多空對沖後的報酬就是超額報酬

實例：

  • 情境：投資組合 100 萬元
  • 操作：50 萬做多台積電（Beta 1.2），50 萬做空聯電（Beta 1.2）
  • 結果：大盤漲跌 1%，組合理論上不受影響，只有多空部位的 Alpha 差異產生報酬

優點：

  • 不受大盤漲跌影響
  • 空頭市場也能獲利
  • Sharpe Ratio 通常較高

缺點：

  • 報酬率較低（要跟市場對沖）
  • 交易成本很重要（來回都要交易費）
  • 極端事件風險大
  • 實際操作難度高

四種套利類型

1. 市場中性套利

  • 同時持有多空部位
  • Beta 近似為零
  • 從價格差異中獲利

2. 跨市場套利

  • 同一資產在不同市場有價差
  • 低買高賣賺取差價
  • 實例：台積電 ADR vs 台股現貨

3. 跨資產套利

  • 金融資產與標的資產的價差
  • 實例：期貨與現貨價差
  • 當價差偏離合理範圍時進場

4. ETF 套利

  • ETF 價格與淨值差異
  • 當折溢價過大時套利
  • 需要申購贖回機制

配對交易實戰步驟

是什麼：配對交易是最常見的統計套利方式，找出兩檔高度相關的股票，當價差偏離時進場。

怎麼做：

步驟一：選擇配對股票

  • 同產業或類似的公司
  • 營運模式相近
  • 歷史價格高度相關
  • 實例：百事可樂 vs 可口可樂、台積電 vs 聯電

步驟二：計算價差 Spread

價差 = 股票A價格 - (對沖比率 × 股票B價格)
  • 對沖比率用線性回歸計算
  • 確保價差序列平穩

步驟三：平穩性檢定（ADF 檢定）

  • 使用 Augmented Dickey-Fuller test
  • 檢定價差是否為平穩序列
  • p-value < 0.05 才能使用

步驟四：設定進出場標準

  • 進場：價差超過 ±2 標準差
  • 出場：價差回到均值
  • 停損：價差持續擴大超過 ±3 標準差

實例：

  • 情境：LAD 和 TTM 兩檔汽車股
  • 觀察：兩股價格歷史上高度相關
  • 進場：當價差偏離 2 標準差
  • 出場：價差收斂回均值

Z-Score 計算與應用

是什麼：Z-Score 衡量當前價距離均值多少個標準差。

公式：

Z-Score = (目前價差 - 均值) / 標準差

怎麼用：

  • Z > 2：價差顯著偏高 → 做空價差（賣A買B）
  • Z < -2：價差顯著偏低 → 做多價差（買A賣B）
  • Z ≈ 0：價差接近均值 → 平倉出場

統計套利三大風險

1. 執行風險

  • 價差可能不會回歸
  • 均值回歸假設失效
  • 模型判斷錯誤導致虧損

2. 模型風險

  • 統計關係可能改變
  • 過去相關不代表未來相關
  • 需要持續監控與更新

3. 競爭壓力

  • 統計套利策略普及
  • 利潤空間被壓縮
  • 需要更快的執行速度

適用時機

適合使用統計套利：

  • 資產有明確歷史相關性
  • 價差偏離均值超過 2 標準差
  • 能忍受短期浮虧
  • 有能力進行多空操作

不適合使用統計套利：

  • 單一資產面臨特殊事件
  • 相關性被破壞（如政策變化）
  • 無法做空或成本過高
  • 市場極端波動期間

多/空股票策略 130/30

是什麼：多空策略的一種，多頭佔 130%、空頭佔 30%。

怎麼做：
1. 用 100% 資金做多看好的股票
2. 額外借 30% 資金做空看壞的股票
3. 多空搭配降低市場風險

與市場中性的差異：

  • 130/30 有淨多頭部位（100%）
  • 市場中性多空完全對沖（50/50 或 100/100）
  • 130/30 報酬較高但風險也較高

實務注意事項

1. 交易成本是關鍵：套利利潤通常很薄，來回交易費可能吃掉獲利

2. 需要高速執行：價差機會稍縱即逝，執行速度很重要

3. 持續監控相關性：兩資產的歷史關係可能改變

4. 資金規模：大資金才能有效分散多檔配對

5. 停損設定：價差不回歸時要有出場機制

心得

統計套利和市場中性策略是機構投資人的重要工具，但對散戶來說實用性有限。主要原因：

1. 交易成本太高：散戶手續費高，來回吃掉大部分利潤
2. 資金規模不夠：無法同時操作多檔配對分散風險
3. 執行速度劣勢：與機構比沒有速度優勢

但這些概念仍然值得一學：

  • 了解套利機制有助於理解市場運作
  • 均值回歸概念可以應用到其他策略
  • Beta 中性的概念對風險管理很重要

對散戶來說，更實際的應用可能是：

  • 不做純套利，但用配對交易的邏輯篩選股票
  • 關注相關股票的價差變化，尋找相對便宜的標的
  • 理解機構操作邏輯，避免成為套利的受害者

記住：市場上沒有穩賺不賠的策略，套利也不例外。價差不回歸、相關性被破壞、極端事件都可能造成虧損。
