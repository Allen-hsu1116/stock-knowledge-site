訂單流分析（Order Flow）

是什麼：訂單流分析是觀察市場中即時買賣指令和成交數據的方法，比傳統技術分析更微觀，直接揭示「誰在主導價格」和「資金流向」。傳統技術分析看歷史價格（滯後），訂單流看即時供需壓力（領先）。

怎麼做：

1. 核心觀念——聰明錢 vs 投機錢

  • 傳統分析把市場分為買方和賣方

2. 三大訂單流工具

  • Footprint Chart（足跡圖）：拆解每根K線內部結構，顯示每個價格水平的買方和賣方主動成交數量。左側顯示已成交賣單、右側顯示已成交買單。能精確識別「誰在主導價格」。

3. 判斷真突破 vs 假突破

  • 真突破特徵：突破關鍵價位時，Footprint Chart 顯示大量累積 Delta（主動買賣差），代表有堅決資金在推動價格

4. 限價單 vs 市價單

  • 限價單：等待特定價格才成交，構成市場的「牆壁」（支撐/壓力）

5. Spoofing（假單）風險

  • 交易者在特定價位下假委託，即將成交前取消

適用時機：

  • 日內交易、短線進出場時機判斷
  • 辨識突破真假
  • 不適合：長期投資、低流動性股票（訂單流需要足夠成交量才有效）

Volume Profile（成交量分布圖）

是什麼：Volume Profile 是一種垂直顯示成交量分布的工具，與傳統水平量能圖不同，它以價格為縱軸、成交量為橫軸，直接告訴你「在哪個價位成交最多」——也就是市場的「成本中心」。

怎麼做：

1. 四大核心元素

  • POC（Point of Control，主要成交價）：成交量最高的價格，橫條最長的那個價位。代表市場最大共識點，價格在此附近行為對判斷反轉或延續至關重要。

2. 高成交量區（HVN）vs 低成交量區（LVN）

  • HVN（High Volume Node）：成交量高於平均的價位，買賣雙方大量交易的區域。價格在此停留時間長，代表「公認價格區域」

3. 三種市場定律（VPA 三定律）

定律一：回歸均值

  • 大多數時間，價格會在 VA 內來回震盪

定律二：回檔支撐

  • 上升趨勢中，價格回到 VA 或 POC 時出現支撐

定律三：反彈阻力

  • 下跌趨勢中，價格反彈到 VA 或 POC 時遭遇阻力

4. 判斷趨勢形態（正三角 vs 倒三角）

  • 正三角：下方 HVN 大於上方 HVN → 上漲趨勢中，股價向上尋找新的公認價格

5. 突破判斷四要素

  • 突破當下要放量（超過平均量 50% 以上）

6. 三種 VP 顯示模式

  • Fixed Range：框選特定時間範圍，適合回顧

適用時機：

  • 找底部型態的 Pivot Point（比頸線更早的進場點）
  • 區間逆勢操作（VAH 做空、VAL 做多）
  • 選擇權履約價挑選（POC 和 VA 是天然的支撐壓力參考）
  • 確認突破真假
  • 不適合：直接用 VP 預測趨勢方向（VP 是確認指標，不是預測指標）

多維度驗證模型（訂單流 + VP + 結構分析）

是什麼：將訂單流信號、Volume Profile 和市場結構分析結合，形成三層驗證的完整交易系統，擺脫對單一指標的依賴。

怎麼做：

1. 訂單流設定進出場條件：Footprint Chart 或 Bookmap 上的特定主力資金行為作為觸發條件
2. Volume Profile 決定風險範圍：用 VP 找出 VA 和 LVN，停損設在 LVN 之外。一旦突破 LVN，確認市場結構已根本改變
3. 結合不同時間框架：大週期看位置+中週期判結構+小週期找入場，避免被單一時間框架的雜訊誤導

實例：

  • 大週期（日線）：VP 顯示 POC 在 550 元，VA 在 530-570
  • 中週期（60分線）：股價從 540 漲到 565，接近 VAH
  • 小週期（5分線）：Footprint 顯示 565 附近主動買量大於賣量，Delta 正值持續擴大
  • 進場：565 元突破 VAH 時做多，停損設在 LVN 下方 525 元
  • 邏輯：大週期位置確認 + 中週期結構確認 + 小週期訂單流確認 = 三層驗證

適用時機：

  • 需要高勝率進場點的波段交易
  • 突破交易的確認
  • 不適合：純看 VP 不看結構就進場（VP 是配角，結構是主角）

心得

訂單流和 Volume Profile 是一種完全不同的看市場方式。傳統技術分析看的是「價格做了什麼」，訂單流看的是「誰在背後推動價格」——這從滯後指標變成了領先視角。

幾個關鍵收穫：

1. POC 是市場的「磁鐵」：不管價格跑到哪，遲早會被 POC 拉回來。這不是玄學，是成本心理——大部分人的成本在那裡，價格偏離成本太遠就會產生「打折了」或「套太久了」的情緒反應。

2. HVN 和 LVN 的切換才是重點：HVN 是「認同區」（價格會停留），LVN 是「不認同區」（價格快速通過）。突破發生在 LVN，支撐壓力在 HVN。記住：價格在 LVN 移動很快、在 HVN 移動很慢。

3. VP 是配角不是主角：這點很重要。VP 告訴你「成本在哪裡」，但不告訴你「價格要去哪裡」。必須搭配趨勢分析和型態分析，VP 才能發揮最大價值。先判趨勢，再用 VP 確認進出場點。

4. 正三角 vs 倒三角：這個判斷形態的方法很直覺——底部放量比頂部大就是上漲趨勢，反之就是趨勢衰竭。配合之前學的籌碼面（主力買超+買賣家數差），可以從成本結構的角度確認趨勢健康度。

5. 假突破的訂單流特徵：突破後 Delta 迅速轉負、主動買量枯竭——這比看 K 線型態更早發現假突破。不過台股散戶要用 Footprint 需要券商提供 Level 2 數據，門檻比較高。
