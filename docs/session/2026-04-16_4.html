一、逆勢交易策略完整解析

1. 逆勢交易核心原理

是什麼：基於「物極必反、均值回歸」的假設，認為價格偏離均值過遠時終將回歸，因此在超漲時做空、超跌時做多。

怎麼做：
1. 找到均值：確認標的的合理價格區間（如中鋼30-40元、台灣50的長期均值），用歷史均值或移動均線作為基準
2. 判斷偏離程度：用乖離率（BIAS）、RSI超買超賣、布林通道位置等指標，衡量價格偏離均值的程度
3. 在極端偏離時進場：價格偏離均值超過2個標準差時進場，方向與趨勢相反
4. 設定嚴格停損：逆勢交易最大的風險是「趨勢不回歸」，停損必須設在均值繼續遠離的位置

實例：

  • 情境：中鋼（2002）歷史均值約35元，因鋼鐵需求暴增漲到48元，乖離率超過25%
  • 操作：在48元附近放空，停損設50元（約4%），目標回到40元（約17%獲利）
  • 風報比：17%:4% = 4.25:1，划算

適用時機：

  • ✅ 盤整市、有明確均值回歸特性的標的（大型股、ETF、原物料）
  • ❌ 單邊趨勢市（逆勢做空會被軋死）、小型股（均值不明確）

2. 逆勢交易三大操作法

是什麼：逆勢交易不是只有「猜頭摸底」，有三種系統化的操作方法。

怎麼做：

方法一：超買超賣反轉法
1. 用RSI或KD判斷超買（RSI>70、KD K值>80）→ 準備做空
2. 用RSI或KD判斷超賣（RSI<30、KD K值<20）→ 準備做多
3. 等指標從極端值開始反轉（RSI從70回落、KD從80往下交叉）才進場，不要在極端值還在上升時就進場

方法二：布林通道回歸法
1. 價格觸及布林通道上軌（2σ以上）→ 超漲，準備做空
2. 價格觸及布林通道下軌（-2σ以下）→ 超跌，準備做多
3. 進場後目標看回到中軌（均值），不貪心等穿越中軌

方法三：支撐壓力區間反向法
1. 在盤整區間內，價格觸及壓力區 → 做空
2. 價格觸及支撐區 → 做多
3. 停損設在區間外1-2%，確認突破/跌破後認賠

實例：

  • 方法一：某股RSI從85回落到75且KD死亡交叉 → 進場做空，停損設RSI重返85對應的價位
  • 方法二：台灣50（0050）跌破布林下軌 -2.5σ → 進場做多，目標回到中軌，停損設 -3σ
  • 方法三：某股在100-110元區間盤整三週 → 110做空（停損112）、105做多（停損103），每次賺3-5%

適用時機：

  • 方法一 → 波動較大的個股或期貨
  • 方法二 → 有均值回歸特性的ETF或大型股
  • 方法三 → 明確盤整區間的股票

3. 順勢 vs 逆勢：何時用哪一種？

是什麼：順勢和逆勢不是非此即彼，而是根據市場狀態切換使用。

怎麼做（判斷框架）：

市場狀態策略選擇原因
明確上升趨勢（均線多頭排列、ADX>25）順勢做多趨勢會延續，逆勢做空會被軋
明確下降趨勢（均線空頭排列、ADX>25）順勢做空趨勢會延續，逆勢接刀會受傷
區間盤整（均線糾結、ADX<20）逆勢操作沒有趨勢可跟，價格在區間來回
趨勢末端（背離出現、量價背離）逆勢操作趨勢動能衰退，反轉機率增加
劇烈波動（VIX飆高）逆勢操作極端情緒往往過度反應

實例：

  • 2020年3月疫情崩盤：VIX飆到80以上 → 逆勢做多（極度恐慌往往過度反應）
  • 2023年AI熱潮：台積電均線多頭排列、ADX=35 → 順勢做多
  • 2024年大盤在16000-17500盤整 → 逆勢操作（高拋低吸）

適用時機：每次進場前都先判斷市場狀態，再決定用順勢還是逆勢策略

4. 逆勢交易風險管理四大鐵律

是什麼：逆勢交易的致命風險是「趨勢不回歸」，所以風控比順勢更嚴格。

怎麼做：
1. 停損比順勢更嚴格：順勢停損可以設5-10%，逆勢停損必須設2-3%，因為你在跟趨勢對做
2. 單筆虧損上限：逆勢交易單筆最大虧損不得超過總資金的1%（順勢可以放寬到2%）
3. 不凹單：逆勢交易被套住絕不加碼（那叫攤平），停損就是停損
4. 趨勢確認後立即切換：當ADX從<20突破到>25，表示趨勢開始了，此時逆勢策略要停止

實例：

  • 錯誤：在50元逆勢做空某股，漲到52元沒停損想「再等等」，漲到58元被迫停損，虧損16%
  • 正確：50元做空，停損51.5元（3%），被停損後承認「這不是盤整，是趨勢」，改用順勢策略

適用時機：每次逆勢交易都要遵守，沒有例外

二、程式交易系統建立完整指南

5. 程式交易是什麼？與人工交易的本質差異

是什麼：用程式碼或自動化工具執行投資流程（選股、分析、下單），排除人性的恐懼與貪婪。

怎麼做（差異比較）：

面向程式交易人工交易
交易依據僅依據程式設定的客觀標準主觀判斷，受情緒影響
交易時間24小時全自動僅限空餘時間
監控能力同時監控數百檔股票最多同時看5-10檔
回測能力快速回測數年數據只能憑記憶和主觀回顧
紀律嚴格執行，不會恐懼或貪婪容易凹單、追高殺低
彈性突發事件無法應變可根據新聞靈活調整

實例：

  • 程式交易的優勢：設定「MACD金叉+成交量放大2倍」的自動選股，每分鐘掃描全市場，人不可能做到
  • 程式交易的弱點：2020年3月疫情突發，程式無法理解「疫情」這個非結構化資訊，人類交易者可以

適用時機：有明確策略規則時用程式交易；需要判斷非結構化資訊時用人工

6. 程式交易五大入門策略

是什麼：最適合用程式交易執行的五種策略類型。

怎麼做：

策略一：趨勢策略（順勢交易）

  • 邏輯：EMA均線交叉、MACD金叉死叉、ADX趨勢強度
  • 程式優勢：能24小時監控多市場、趨勢出現時立刻進場
  • 適合商品：期貨、外匯、大型股

策略二：動能策略

  • 邏輯：汰弱留強，追蹤動能最強的標的
  • 常用指標：MTM動量指標、ATR波動度、RSI強弱度
  • 程式優勢：快速計算全市場動能排名，即時換股

策略三：逆勢策略

  • 邏輯：BIAS乖離率、SAR拋物線、Pivot樞紐點
  • 程式優勢：嚴格執行停損（人容易凹單）、在超漲超跌時自動進場

策略四：籌碼策略

  • 邏輯：券資比、大戶持股數、法人買賣超
  • 程式優勢：即時監控法人進出，大戶開始買進時自動掛單

策略五：基本面策略

  • 邏輯：現金流量表、資產負債表、毛利率變化
  • 程式優勢：快速篩選符合條件的股票，定期定額執行

實例：

  • 趨勢策略：程式設定「台指期5分K EMA12上穿EMA26且ADX>20」→ 自動進場做多，停損設30點
  • 籌碼策略：程式設定「外資連續3日買超且金額>5億」→ 自動買進該股

適用時機：有明確規則的策略都適合程式化；模糊判斷（如「感覺要漲」）不適合

7. 程式交易回測系統建立五步驟

是什麼：回測是程式交易的核心，用歷史數據驗證策略是否有效。

怎麼做：

步驟一：導入股價資料

  • 選擇資料來源：TEJ API、永豐Shioaji、XQ全球贏家、Yahoo Finance
  • 設定日期區間和標的（如「2018-2025台積電2330」）
  • 確保資料品質：除權息調整、沒有缺值

步驟二：建立策略函式（initialize）

  • 設定回測環境：滑價模式、手續費（台股0.1425%買賣各一次）
  • 設定初始變數：交易日計數器、是否持倉標記
  • 範例：設定手續費0.285%（買+賣）、固定滑價1個Tick

步驟三：撰寫交易邏輯（handle_data）

  • 每個交易日執行的判斷邏輯
  • 進場條件：技術指標訊號、成交量條件、籌碼條件
  • 出場條件：停損、停利、時間出場
  • 下單指令：買進/賣出多少股

步驟四：績效分析（analyze）

  • 視覺化策略績效曲線
  • 計算關鍵指標：年化報酬率、最大回撤、夏普比率、勝率
  • 與大盤比較（benchmark）

步驟五：執行回測（run_algorithm）

  • 設定回測參數：起始日、結束日、初始資金、交易頻率（日/分鐘）
  • 輸出結果包含：每日損益、持倉紀錄、訂單紀錄、績效指標

實例：

  • 用zipline回測台積電買進持有策略（2017-2025）
  • 初始資金100萬，第一天買進1張台積電，最後一天賣出
  • 回測結果：累積報酬率+180%、最大回撤-25%、夏普比率0.85

適用時機：任何策略上線前都必須回測，回測不過的策略不要用真錢

8. 程式交易常見陷阱與對策

是什麼：程式交易不是萬能的，有幾個致命陷阱必須避開。

怎麼做：

陷阱一：過擬合（Overfitting）

  • 症狀：回測績效超好（年化50%+），但實盤虧損
  • 原因：參數過度針對歷史數據優化，未來沒有同樣的行情
  • 對策：用Walk-Forward驗證、保留30%數據做Out-of-Sample測試

陷阱二：前視偏誤（Look-Ahead Bias）

  • 症狀：回測績效不合理地好
  • 原因：用到「未來才知道」的資訊（如當日收盤價在開盤時就用了）
  • 對策：確保每筆交易只用「當時已知」的資訊，zipline會自動延遲一天成交

陷阱三：忽略交易成本

  • 症狀：回測小賺，實盤大虧
  • 原因：手續費、滑價、證交稅沒有計入
  • 對策：回測一定要設定手續費（0.1425%×2+證交稅0.15%）和合理滑價（至少1 Tick）

陷阱四：策略失效不自知

  • 症狀：策略上線後前三個月還行，之後越來越差
  • 原因：市場結構改變、策略被太多人使用（套利空間消失）
  • 對策：每月監控策略績效，設定「連續虧損N次自動停止」的熔斷機制

實例：

  • 過擬合：RSI參數從14調到11.3最好，但11.3只對那段歷史有效 → 用整數參數（12或14）
  • 忽略成本：高頻策略每天交易10次，回測年化30%扣掉成本剩5% → 不值得做

適用時機：回測完成後、上線前，逐項檢查這四個陷阱

心得

今天學的逆勢交易和程式交易，一個是「策略方向」的補完，一個是「執行工具」的升級，兩個加起來讓我的操作體系更完整了：

1. 逆勢交易不是逆天，是逆「過度反應」。之前學的大部分是順勢策略（海龜、均線順勢、MACD順勢），但市場只有30%時間在趨勢中，70%時間在盤整。在盤整中用順勢策略會被雙巴，這時候逆勢策略才是正解。關鍵是判斷「現在是趨勢還是盤整」——用ADX和均線斜率判斷，ADX<20且均線糾結就用逆勢，ADX>25且均線發散就用順勢。

2. 逆勢交易的停損比進場更重要。逆勢交易最怕的不是判斷錯方向，而是「趨勢不回歸」。一個20元的停損在順勢交易可能還行，但在逆勢交易就是致命的——因為你在跟趨勢對做，多拖一秒風險就大一秒。2-3%的嚴格停損是逆勢交易的保命符。

3. 程式交易的核心價值是「紀律」和「監控」。不是說程式比人聰明，而是程式不會恐懼、不會貪婪、不會凹單。對我來說最有價值的兩個應用：一是自動停損（人不會猶豫），二是同時監控全市場（人只能看幾檔）。但程式的弱點是不能處理突發事件，所以最好是人機協作——程式執行規則內的策略，人處理規則外的判斷。

4. 回測是程式交易的照妖鏡。任何策略上線前都要過回測這關，但回測過了不代表實盤就能賺——過擬合、前視偏誤、忽略成本，這三個坑一個踩到就完了。最實際的做法是：回測參數用整數（不要微調到小數點），成本算滿（手續費+滑價+稅），保留30%數據做驗證。

5. 順勢+逆勢+程式 = 全天候操作系統。趨勢市用順勢程式自動追蹤，盤整市用逆勢程式高拋低吸，突發事件用人腦判斷。三個工具各有擅場，組合起來就能在任何市場狀態下都有對應策略。

一、底部反轉操作策略：朱家泓五步驟實戰法

1. 四線多排選股法

是什麼：用5日線、10日線、20日線、60日線的排列方向判斷股票是否處於多頭趨勢，篩選出趨勢明確向上的標的。

怎麼做：
1. 檢查均線方向：5日線、10日線、20日線、60日線全部向上排列（短期在中期上方、中期在長期上方），代表短中長期趨勢一致走多
2. 排除假多頭：若股價短線上漲但月線或季線仍在下彎，這不是真正的多頭，均線下彎會形成壓力拖累股價
3. 確認K線結構：除了均線方向，還要檢視K線是否符合「頭頭高、底底高」——後續高點比前一個高點更高、低點也比前一個低點更高

實例：

  • 情境：某AI概念股股價從80元上漲到95元，但20日均線仍在下彎
  • 判斷：雖然短期上漲，但中期均線向下壓制，不是真正的四線多排，暫時觀望
  • 正確案例：虹冠電（3257）5日線、10日線、20日線、60日線全部向上，且K線呈現頭頭高底底高 → 符合四線多排條件，可列入觀察名單

適用時機：每日選股必做，是所有波段操作的第一道濾網；均線糾結或下彎時不適用

2. 位階判斷：底部完成 vs 高檔風險

是什麼：同樣是四線多排，股價位階不同，風報比天差地遠。底部完成的初升段勝率最高，高檔的主升段末升段風險最大。

怎麼做：
1. 底部完成、初升段（最佳進場位階）：股價剛從底部翻揚，上方仍有主升段和末升段的漲幅空間，勝率最高
2. 主升段（仍可操作）：股價已有一段漲幅，但仍可沿5日線操作，嚴守停損
3. 末升段（風險最高）：股價已漲1倍以上，四線雖然多排但乖離過大，隨時可能反轉
4. 位階篩選原則：瞄準50-100元區間的中低價股，一旦轉強，漲幅空間比800元以上的高價股更大

實例：

  • 好位階：某股從50元底部剛翻揚到55元，四線多排剛形成 → 初升段進場，風報比佳
  • 差位階：某股從200元漲到500元，四線多排但乖離極大 → 末升段，追進風險極高

心得

今天學的朱家泓底部反轉操作法和當沖進出場紀律，核心觀念都指向同一件事：紀律比技術更重要。

2. 四線多排是選股濾網，不是進場訊號。均線全部向上排列只是告訴你「這檔股票目前趨勢向上」，不代表現在就是進場點。進場點要等「回後買上漲」的訊號出現。很多人看到四線多排就追進去，結果買在回檔起跌點，這是搞混了「選股」和「擇時」。

3. 位階判斷決定風報比。同樣是四線多排，底部初升段的風報比可能是3:1以上，末升段可能只剩1:1。朱家泓建議瞄準50-100元區間的中低價股，因為這些股票一旦轉強，漲幅空間遠大於已經漲了3-5倍的高價股。這跟之前學的「價值投資安全邊際」概念一致——不是找便宜的，是找風報比好的。

4. 五種停損方法不是選一種就好，是根據情境選擇。K線停損適合短線、趨勢停損適合波段、均線停損適合沿均線操作、固定比例適合新手、支撐壓力停損適合有明確關鍵位的標的。但不管選哪種，「設好就不改」是鐵律。

5. 當沖出場比進場更靠紀律。「進場靠技巧，出場靠紀律」這句話太真實了。很多人進場分析半天，出場卻憑感覺——賺一點就跑、虧了就凹。三種出場方式（固定比例、指標訊號、時間管理）可以組合使用，最重要的是「設了就執行」。
