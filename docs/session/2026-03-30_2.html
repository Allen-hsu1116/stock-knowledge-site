RSI 指標基本概念

RSI 是什麼

定義：RSI（Relative Strength Index）相對強弱指標，由技術分析大師 J. Welles Wilder Jr. 於 1978 年提出，用來衡量一段時間內「買方力道」與「賣方力道」的相對強弱。

核心邏輯：

  • RSI 越高 → 買方力道強，市場可能過熱
  • RSI 越低 → 賣方力道強，市場可能過冷
  • 數值範圍：0 ~ 100

RSI 計算公式

公式：

買方力道 = 過去 N 日漲幅加總 / N
賣方力道 = 過去 N 日跌幅加總 / N
RSI = 買方力道 / (買方力道 + 賣方力道) × 100

計算範例：
假設過去 5 天漲跌幅：+6%、+3%、-10%、+1%、-2%

  • 買方力道 = (6+3+1)/5 = 2
  • 賣方力道 = (10+2)/5 = 2.4
  • RSI = 2 / (2+2.4) × 100 = 45.45

RSI 50 的意義：

  • RSI > 50：買方力道較強，偏多
  • RSI < 50：賣方力道較強，偏空
  • RSI ≈ 50：多空平衡，盤整

RSI 基礎應用

一、超買超賣判斷

標準定義：

RSI 數值市場狀態判斷方式
RSI > 70超買區市場可能過熱，留意回檔
RSI < 30超賣區市場可能過冷，可能反彈
RSI ≈ 50多空平衡行情整理，無明顯方向

實戰應用：

  • 超買區不代表「立刻賣出」，而是提醒「行情可能過熱」
  • 超賣區不代表「立刻買進」，而是提醒「可能出現反彈」
  • 不同市場可調整標準：

二、RSI 背離（Divergence）

定義：價格走勢與 RSI 指標走勢不同步，通常代表市場即將反轉。

看漲背離（底部背離）：

  • 價格創新低，但 RSI 沒有創新低，反而低點墊高
  • 代表空方力道減弱，可能出現反彈
  • 實例：股價跌至 80 元，RSI 低點從 20 升到 25，代表空頭力道衰竭

看跌背離（頂部背離）：

  • 塑價創新高，但 RSI 沒有創新高，反而高點下降
  • 代表多方力道不足，可能出現回檔
  • 實例：股價漲至 150 元，RSI 從 75 降到 70，代表上漲動能不足

實戰技巧：

  • 背離後等待 RSI 穿越 50 分水嶺再做確認
  • 設好停損點，避免反轉未成立

三、RSI 黃金交叉與死亡交叉

設定方式：使用兩條不同週期的 RSI（如 6 日 RSI 與 14 日 RSI）

黃金交叉：短週期 RSI 向上突破長週期 RSI → 短期多頭力道超過長期 → 可能上漲

死亡交叉：短週期 RSI 向下跌破長週期 RSI → 短期空頭力道強於長期 → 可能下跌

注意：RSI 交叉勝率不高，震盪盤易出現假訊號，建議搭配均線、MACD 使用

RSI 高階應用

四、Failure Swing 反轉訊號

核心概念：RSI 原創者 Wilder 提出的進階用法，結合「超買超賣」＋「型態學」判斷更精確的反轉趨勢。

看漲 Failure Swing（買進訊號）：
1. RSI 跌破 30（進入超賣區）
2. RSI 反彈超過 30
3. RSI 拉回，但依然在 30 以上（未再跌破超賣區）
4. RSI 創新高 → 確認反轉，買進

看跌 Failure Swing（賣出訊號）：
1. RSI 上漲超過 70（進入超買區）
2. RSI 跌回 70 以下
3. RSI 稍微拉回，但未超過 70（保持在超買區以下）
4. RSI 創新低 → 確認反轉，賣出

實戰價值：

  • 比單純超買超賣更可靠
  • 透過「回踩確認」趨勢成立
  • 符合順勢交易思維，勝率較高

五、RSI 50 分水嶺

核心概念：RSI 50 是多空轉換的關鍵分水嶺。

應用方式：

情境RSI 狀態操作提示
上升趨勢RSI 回落到 50 附近後反彈多頭延續，50 是支撐
下跌趨勢RSI 跌破 50行情轉弱，可能續跌
震盪盤RSI 在 50 附近反覆穿越整理區間，無明確方向

實戰技巧：

  • RSI > 50 時，價格走勢向上；若拉回碰到 50 後反彈，可視為買點
  • RSI 跌破 50 時，考慮獲利了結，可能形成下跌趨勢
  • 搭配超買超賣、背離訊號一起觀察

RSI 鈍化問題

高檔鈍化與低檔鈍化

高檔鈍化：

  • RSI 進入超買區（>70），價格仍持續上漲
  • 常見於強勢多頭行情
  • 若過早賣出，會錯失後續上漲

低檔鈍化：

  • RSI 進入超賣區（<30），價格仍持續下跌
  • 常見於強勢空頭行情
  • 若過早買進，會被持續套牢

解決方法：
1. 調整參數：將超買超賣區間改為 80/20 或 90/10
2. 搭配其他指標：觀察均線、MACD、支撐壓力位確認趨勢
3. 使用 Failure Swing 或 RSI 50：避免過早進出場

RSI 缺點與限制

四大限制

1. 不考慮時間先後順序

  • RSI 只看一段時間內的漲跌幅總和
  • 漲跌先後順序不影響數值
  • 範例：前 3 天大跌、後 10 天大漲，RSI 可能仍顯示空頭壓力

2. 容易受極端數據影響

  • 單日大漲或暴跌會拉扯 RSI 數值
  • 極端值可能讓整體結果失去代表性

3. 無法反映長期趨勢

  • 14 日 RSI 屬於短中期指標
  • 無法完整反映數月的趨勢變化

4. 鈍化問題

  • 強勢行情中，RSI 可能長時間停留在極端區域
  • 不應單獨作為進出場依據

RSI 參數設定建議

常見週期

週期特性適用族群
6 日 RSI訊號敏感、假訊號多短線交易者
14 日 RSI平衡性高（預設）波段交易者
30-60 日 RSI趨勢明顯、反應慢長線投資者

依市場特性調整

市場建議設定
股票市場14 日 RSI，區間 70/30
外匯市場9 日或 14 日 RSI，區間可調整為 60/40
加密貨幣6 日 RSI，區間 90/10

搭配建議

RSI + 移動平均線：用 RSI 判斷多空，均線確認趨勢
RSI + MACD：同時觀察動能與趨勢，減少假訊號
多週期 RSI：同時觀察 6 日與 14 日 RSI 提高準確度

心得

RSI 是技術分析的「溫度計」：能快速判斷市場溫度，但不應單獨使用。

核心價值：

  • 超買超賣：提醒行情可能過熱或過冷
  • 背離訊號：預警趨勢可能反轉
  • Failure Swing：更可靠的反轉確認
  • 50 分水嶺：判斷多空分界

實戰要點：
1. 不要 RSI > 70 就賣、< 30 就買，要看整體趨勢
2. 高檔鈍化時不要急著放空，低檔鈍化時不要急著抄底
3. Failure Swing 比「超買超賣」更可靠
4. RSI 50 分水嶺是很好的多空確認點
5. 務必搭配均線、MACD、支撐壓力位使用

比 KD 強的地方：RSI 考慮「漲跌幅度」，KD 只考慮「收盤價位置」
比 KD 弱的地方：RSI 不考慮時間先後，KD 的交叉訊號更明確

第 7 回合學習：技術分析：趨勢線繪製與實戰技巧

學習時間：09:18

文章：

  • 趨勢線構成的趨勢通道的使用方法 - OANDA Lab

核心知識

趨勢線定義

是什麼：連接兩個以上價格高點或低點形成的直線，用來判斷市場趨勢方向。

三種類型：

  • 上升趨勢線：連接一連串低點，向右上傾斜
  • 下降趨勢線：連接一連串高點，向右下傾斜
  • 水平趨勢線：連接價格波動的高點或低點，標示盤整區間

趨勢線繪製五大要點

是什麼：正確繪製趨勢線的核心規則，確保趨勢線有效。

怎麼做：

1. 選對點位

  • 上升趨勢：找相對低點連線
  • 下降趨勢：找相對高點連線
  • 至少連接三個點才有參考價值

2. 畫線規則

  • K線實體對實體，影線對影線，不能混用
  • 趨勢線不能穿破任何K線實體
  • 若被突破，需重新畫線直到滿足條件

3. 斜度要求

  • 適合斜度：30-70度之間
  • 斜度太大：上升過快不易維持
  • 斜度太小：趨勢不明顯

4. 方向確認

  • 上升趨勢：高點也要上移（不只低點上移）
  • 下降趨勢：低點也要下移（不只高點下移）
  • 否則是假趨勢線

5. 驗證次數

  • 被驗證次數越多越可靠
  • 多條趨勢線選被測試最多的一條

實例：

  • 情境：台積電從500元上漲到600元
  • 低點序列：500 → 520 → 540 → 560
  • 畫線：連接這四個低點，斜度約45度
  • 高點確認：580 → 600 → 620（同步上移）
  • 判斷：有效上升趨勢線

適用時機：任何有明顯趨勢的股票或期貨，不挑盤面、不挑週期。

趨勢線的支撐阻力功能

是什麼：趨勢線不只判斷方向，還能作為支撐和壓力位。

怎麼做：

上升趨勢線作為支撐

  • 價格回調到趨勢線附近，可能反彈上漲
  • 靠攏趨勢線時，配合K線反轉形態準備買進

下降趨勢線作為壓力

  • 價格反彈到趨勢線附近，可能受阻下跌
  • 靠攏趨勢線時，配合K線反轉形態準備賣出

實例：

  • 情境：鴻海上升趨勢線支撐在120元
  • 操作：價格回調到120元附近，觀察是否有買盤進場
  • 進場條件：趨勢線附近出現陽線吞噬或早晨之星
  • 停損設置：跌破趨勢線3%以上

適用時機：確認趨勢存在後，作為進出場點位參考。

趨勢線突破判斷

是什麼：當價格突破趨勢線時，可能代表趨勢反轉或加速。

怎麼做：

突破確認標準

  • 以收盤價為準，不是盤中價
  • 突破幅度需達3%以上才確立
  • 最好伴隨放量

突破後行為

  • 上升趨勢線被跌破 → 趨勢反轉，考慮賣出
  • 下降趨勢線被突破 → 趨勢反轉，考慮買進
  • 突破後可能回測 → 原壓力變支撐，原支撐變壓力

假突破判斷

  • 突破後迅速回到線內
  • 成交量未放大
  • 突破後沒有續漲/續跌

實例：

  • 情境：某股下降趨勢線壓力在100元
  • 突破：收盤價突破100元達103元（3%突破）
  • 成交量：突破當日成交量比平日大2倍
  • 後續：回測100元獲支撐，確認突破有效
  • 操作：100元附近可考慮買進

適用時機：趨勢運行一段時間後的突破訊號。

趨勢通道應用

是什麼：用趨勢線和平行線構建通道，擴展單一趨勢線的應用。

怎麼做：

上升通道畫法
1. 先畫上升趨勢線（連接低點）
2. 找到趨勢線兩低點之間的高點
3. 通過該高點畫平行線，形成上軌

下降通道畫法
1. 先畫下降趨勢線（連接高點）
2. 找到趨勢線兩高點之間的低點
3. 通過該低點畫平行線，形成下軌

通道交易策略

  • 上升通道：下軌買進或加倉，上軌考慮減倉
  • 下降通道：上軌賣出或加空，下軌考慮平倉
  • 通道突破：配合123法則確認趨勢變化

平移技巧

  • 平行線被突破 → 將通道平移一次
  • 尋找短時目標點位
  • 趨勢加速時重新畫通道

實例：

  • 情境：台積電上升通道，下軌580元，上軌620元
  • 操作：價格回調到580元（下軌）買進
  • 目標：上軌620元考慮減倉
  • 停損：跌破下軌或上升趨勢線

適用時機：趨勢明確、價格在通道內規律波動時。

趨勢線與123法則

是什麼：趨勢線突破後的趨勢確認規則。

怎麼做：

123法則三步驟
1. 趨勢線被突破
2. 價格不再創新高（上升趨勢）或不創新低（下降趨勢）
3. 價格跌破前一個低點（上升趨勢）或突破前一個高點（下降趨勢）

交易應用

  • 三步都完成 → 趨勢反轉確認
  • 僅完成前兩步 → 可以開始佈局
  • 第三步完成 → 加碼或進場

實例：

  • 情境：某股上升趨勢線被跌破
  • 第一步：趨勢線突破確認
  • 第二步：價格反彈未創新高（從100元到95元就回頭）
  • 第三步：價格跌破前低90元
  • 判斷：趨勢反轉確立，可以做空

適用時機：趨勢線突破後的趨勢確認。

趨勢線使用注意事項

主觀性問題

  • 不同人畫的趨勢線可能不同
  • 解決：多點連接、選被驗證最多的線

時間框架搭配

  • 不同週期趨勢可能矛盾
  • 日線上升、60分鐘線可能是下降
  • 解決：用長期趨勢判斷方向，短期趨勢找進場點

不要單獨使用

  • 趨勢線是方向參考，不是進出唯一依據
  • 搭配：成交量、均線、RSI、K線型態
  • 趨勢線被突破不代表趨勢馬上反轉

假訊號處理

  • 趨勢線可能產生假突破
  • 解決：等收盤價確認、看突破幅度、觀察成交量
  • 可搭配濾網條件：價格必須在線外停留2-3根K線

趨勢線與均線的差異

是什麼：釐清兩種工具的不同用途和適用場景。

比較項目趨勢線均線
性質手動繪製自動計算
參考基準高低點連線平均收盤價
靈活性隨市場調整固定週期計算
主觀性較高較低
適用場景趨勢判斷、支撐壓力趨勢平滑、動態支撐

實例：

  • 情境：股價在MA20上方運行
  • 均線：自動顯示支撐，無需手動
  • 趨勢線：需手動連接回調低點，可能與MA20接近但略有差異
  • 搭配使用：均線確認中期趨勢，趨勢線找精確進場點

心得

趨勢線看起來簡單（不就是畫條線？），但魔鬼在細節裡。最大的陷阱是主觀性——每個人對高低點的解讀不同，畫出來的線不一樣，得出的結論也可能矛盾。

這篇學到的實戰重點：

1. 三個點才有意義：兩個點畫出來的趨勢線太隨機，至少三個點才能確認趨勢有效。這跟我之前學型態學時強調的「觸碰次數」概念一致。

2. 斜度30-70度：這是個實用的經驗法則。太陡的趨勢線（大於70度）很難維持，就像跑得太快容易喘不過氣；太平的趨勢線（小於30度）又太弱，一個小波動就被突破。

3. 通道是進階版：單一趨勢線給你方向，通道給你進出場點位。上升通道下軌買、上軌賣，這種規則化的操作比憑感覺好多了。

4. 123法則確認反轉：趨勢線突破不等於趨勢反轉，要三步都完成才算。這跟之前學的「假突破」概念呼應——單純看到價格穿過趨勢線就反手，很容易被巴。

5. 不要單獨使用：趨勢線要搭配成交量、K線型態、其他技術指標。之前學的背離訊號（RSI背離、MACD背離）都可以在趨勢線突破時當輔助確認。

趨勢線最大的價值是讓你「一眼」看出市場方向，不需要複雜計算。但也因為這樣，它需要練習——多畫、多驗證、多比較不同人的畫法，才能培養出「看一眼就知道怎麼畫」的直覺。

OBV 能量潮指標基本概念

是什麼

OBV（On-Balance Volume）又稱能量潮指標、人氣指標，由喬葛蘭維（Joe Granville）於 1960 年代提出。是一種結合「成交量」與「價格」的技術分析工具。

核心理念：「量先價行」——成交量是股市的元氣，股價只是表象特徵。

怎麼計算

公式：

  • 今日收盤價 > 昨日收盤價 → 今日 OBV = 前一日 OBV + 今日成交量
  • 今日收盤價 < 昨日收盤價 → 今日 OBV = 前一日 OBV - 今日成交量
  • 今日收盤價 = 昨日收盤價 → 今日 OBV = 前一日 OBV

實例：

  • 第 1 天：收盤價 100 元，成交量 1000 張，OBV = 1000
  • 第 2 天：收盤價 102 元（漲），成交量 800 張，OBV = 1000 + 800 = 1800
  • 第 3 天：收盤價 98 元（跌），成交量 1200 張，OBV = 1800 - 1200 = 600

OBV 四大交易訊號

1. 背離訊號（最重要）

頂背離（賣出訊號）：

  • 股價創新高，但 OBV 沒有創新高
  • 價量背離，隨時可能下跌
  • 實例：股價從 100 元漲到 120 元（創新高），OBV 卻從 5000 降到 4500（未創新高）→ 賣出訊號

底背離（買進訊號）：

  • 股價創新低，但 OBV 沒有創新低
  • 量能已經不願意再殺，反彈機會大
  • 實例：股價從 100 元跌到 80 元（創新低），OBV 卻從 2000 升到 2500（未創新低）→ 買進訊號

2. 同步訊號

多頭同步（加碼訊號）：

  • OBV 線緩慢上升 + 股價同步上漲
  • 行情穩健，中長期看好
  • 實例：台積電股價從 500 漲到 520，OBV 也從 10000 穩定上升到 12000 → 可加碼

空頭同步（賣出訊號）：

  • OBV 線緩慢下降 + 股價同步下跌
  • 行情疲弱，中長期看淡
  • 實例：股價從 100 跌到 90，OBV 從 5000 降到 4000 → 應賣出或觀望

3. 急速變化訊號

OBV 急速上升（賣出訊號）：

  • 買盤已大量累積，主力不會幫散戶抬轎
  • 上漲已經乏力，情勢即將扭轉
  • 實例：OBV 從 5000 一天跳升到 8000，股價卻只小漲 2% → 注意反轉

OBV 急速下跌（觀望訊號）：

  • 大量賣盤湧出，轉為空方趨勢
  • 不要輕易搶反彈
  • 實例：OBV 從 8000 降到 3000，股價跌停 → 觀望，等止跌後再說

搶反彈時機：

  • OBV 急跌後，在底部形成齒狀曲線
  • 可以考慮進場做短期反彈
  • 實例：OBV 從高點急跌到低點後，不再創新低且開始橫盤 → 短線可試

4. 累積訊號

累積高點（阻力區）：

  • OBV 長期累積後的大波段高點
  • 股價在此區域遇到強大上升壓力
  • 一旦突破，漲勢將更強勁

實例：

  • 某股 OBV 在 5000-8000 區間累積 3 個月，股價在 80-90 元盤整
  • 突破 90 元後，OBV 也突破 8000 → 強勁上漲可期

累積低點（支撐區）：

  • OBV 長期累積後的大波段低點
  • 股價在此區域遇到極強支撐
  • 一旦跌破，跌勢將更猛

OBV 進階實戰：均線搭配

搭配 20MA 或 60MA

多方確認：

  • OBV 從底部上升 + 股價在均線之上 = 多方力道強勢
  • 實例：股價 50 元，20MA 在 48 元（均線之上），OBV 從低點翻揚 → 多頭確立

空方確認：

  • OBV 從頂部下降 + 股價在均線之下 = 空方力道強勢
  • 實例：股價 45 元，20MA 在 47 元（均線之下），OBV 從高點下滑 → 空頭確立

OBV 使用注意事項

適用情境

  • 成交量較大的股票（熱門股）
  • 中長期趨勢判斷
  • 主力、法人進出場判斷

不適用情境

  • 成交量太低的股票（冷門股）
  • 價格震盪劇烈的市場（如加密貨幣）
  • 極短線當沖（OBV 是落後指標）

重要提醒

  • OBV 是落後指標，依靠歷史數據研判
  • 不具有預測未來行情的能力
  • 必須搭配其他技術指標使用（均線、MACD、RSI 等）
  • 不要單獨使用 OBV 做決策

OBV 優缺點

優點

  • 簡單易懂：已將複雜數據圖像化
  • 反映真實：量價關係一目了然
  • 提前預警：背離訊號可提前發現趨勢反轉
  • 辨識主力：可判斷主力進出場

缺點

  • 落後指標：無法預測未來
  • 不適合冷門股：成交量低時失真
  • 不適合震盪盤：價格波動大時判讀困難
  • 無價格維度：只看量不看價格波動幅度

心得

OBV 是相對簡單但實用的量價指標，核心觀念是「量先價行」——在股價還沒轉向之前，成交量往往已經先透露端倪。

最重要的是「背離」訊號：股價創新高但 OBV 沒跟著創新高，代表買盤力道已經不足；股價創新低但 OBV 沒跟著創新低，代表殺盤力道已經耗盡。這兩個訊號比單純看股價更能提前預警。

不過要記住，OBV 是落後指標，不能單獨使用。實戰上應該搭配均線、MACD 或其他指標一起看。另外，這個指標更適合成交量穩定的熱門股，冷門股的 OBV 容易失真。

整體來說，OBV 是「量價關係」的視覺化工具，讓投資人不用再盯著成交量表看，一眼就能看出買賣力道的強弱。在判斷主力動向、股價趨勢是否健康（價量是否同步）方面，是很好用的輔助工具。

成交量與主力行為

主力大戶的運作邏輯

主力進出場的特徵訊號：

  • 主力進場：低檔爆大量，股價劇烈波動
  • 主力出貨：高檔爆大量，長上影線或大黑K
  • 主力洗盤：量縮下跌，股價進入箱型整理

主力吃貨到出貨的完整週期：
1. 低價區間：放出利空消息製造恐慌，散戶賣出、主力吃貨
2. 橫盤震盪：量縮整理，主力慢慢吃貨，籌碼乾淨
3. 主升段：媒體放利多，主力抬價造市，散戶追進
4. 賣出高峰：高檔爆大量，主力倒貨給散戶，形成頭部
5. 下跌段：散戶套牢、認賠殺出，主力落井下石加速崩跌

實例：

  • 智邦（2345）：長期底部橫盤整理後爆大量，波段行情啟動
  • GIS-KY（6456）：86 元起漲，拉至 130 元後高檔出貨，跌回 90 元

成交量節點與支撐壓力

什麼是成交量節點？

定義：

  • 價格區間內累積大量成交的位置
  • 代表多空情緒平衡的支點

兩種節點類型：

節點類型特徵對股價影響
高量大節點成交量密集價格暫停、進入震盪
低量小節點成交量稀疏價格快速穿越

怎麼用：
1. 畫出價格區間的成交量分佈圖
2. 找出高量節點（阻力區）和低量節點（順暢區）
3. 價格接近高量節點：準備震盪或反轉
4. 價格接近低量節點：快速穿越機率高

特殊成交量型態

1. 凹洞量

定義：成交量突然縮到非常低，接著又恢復

出現位置：

  • 大盤指數底部
  • 個股洗盤結束

判斷方法：
1. 觀察成交量是否突然縮到極低（比前幾盤小很多）
2. 縮量後立即放量
3. 配合股價型態判斷

操作意義：凹洞量大盤出現，代表反轉機會

2. 窒息量（換手率極低）

定義：成交量不到 20 日內最大成交量的 10% 以下

與凹洞量的差異：

  • 凹洞量：與前後盤相比呈現萎縮
  • 窒息量：與最大成交量相比極低

出現位置：空方窮途末路、價格跳空下跌之後

操作意義：

  • 股本小的股票：籌碼已清洗乾淨
  • 股本大的股票：股價已低到沒人願意賣
  • 有足夠量進來：股價容易反轉向上

3. 高檔爆量四大指標

指標 1：長黑爆量

  • 實務：先賣再講，不問為什麼

指標 2：高檔爆量收上影線

  • 實務：股價陷入盤整

指標 3：高檔爆量多次上影線

  • 實務：主力偷偷出貨，賣壓變重

指標 4：高檔爆量後下跌

  • 實務：趨勢反轉

核心原則：高檔爆量都非好事，先賣再講

5 均量判定法（複習加強）

核心定義：

  • 成交量 > 5 均量 = 有量
  • 成交量 < 5 均量 = 無量

為什麼用 5 均量：

  • 一週 5 個交易日
  • 代表一週的平均成交水準
  • 適用於日線、週線、月線

實戰判斷：

  • 有量：價格有波動，值得操作
  • 無量：價格停滯，觀望為宜

價平量增 vs 價平量縮

價平量增

出現位置：股價持平，但成交量突然放大

低檔意義：有人在收購籌碼，等待主力表態

高檔意義：

  • 出現多根上影線
  • 成交量明顯放大
  • 主力偷偷出貨
  • 賣壓變重後轉為下跌

價平量縮

出現位置：股價持平，成交量萎縮

常見情境：連續急漲後，籌碼混亂

主力行為：花時間沉澱籌碼，等待散戶離場

後續發展：籌碼乾淨後，進入下一波上漲

心得

今天深入看了成交量的主力行為判讀，最有價值的觀念是「成交量節點」和「特殊量能型態」。

主力行為的辨識不再只是看「量增」或「量縮」，而是要看「誰在買、誰在賣」。高檔爆大量 = 主力倒貨給散戶；低檔爆大量 = 主力吃貨準備起漲。這個邏輯要記住。

凹洞量和窒息量是之前沒學過的。凹洞量是「突然縮量又放量」，窒息量是「量能極低到 10% 以下」。這兩個型態都出現在底部區間，是「賣壓耗盡」的訊號。

高檔爆量四大指標很實用。不用猜為什麼爆量，先賣就對了。長黑爆量 = 跑；上影線爆量 = 盤整或下跌。簡單明確。

操作重點：
1. 找低檔爆大量的股票（主力吃貨）
2. 判斷成交量節點（避開高量阻力區）
3. 高檔爆量先賣，不要問為什麼

核心知識

CCI順勢指標基本概念

是什麼：CCI（Commodity Channel Index）順勢指標，由唐納德·蘭伯特於1980年創造，是一種震盪指標，用來測量價格偏離平均價格的程度，判斷超買超賣狀態。

最大特色：CCI沒有上下限（波動於正無窮到負無窮之間），不會像KDJ、RSI那樣出現鈍化現象，特別適合研判短期內暴漲暴跌的非常態行情。

CCI指標計算公式

公式：

CCI = (TP - MA) / (0.015 × MD)

TP（典型價格）= (最高價 + 最低價 + 收盤價) / 3
MA = TP的N日移動平均（通常用14日或20日）
MD = TP的N日平均絕對偏差

怎麼做：
1. 計算典型價格TP = (最高價 + 最低價 + 收盤價) / 3
2. 計算TP的N日移動平均MA
3. 計算平均絕對偏差MD
4. 代入公式得出CCI值

實例：

  • 某股票今日最高100、最低90、收盤98
  • TP = (100 + 90 + 98) / 3 = 96
  • 假設20日MA為95，MD為3
  • CCI = (96 - 95) / (0.015 × 3) = 22.2

CCI三大區間判斷

是什麼：CCI以+100和-100為關鍵分界線，將市場分為三個區間。

怎麼做：

區間CCI數值市場狀態操作建議
超買區CCI > +100強勢上漲、非常態波動關注反轉訊號，可做多
常態區-100 ~ +100震盪整理用其他指標研判
超賣區CCI < -100弱勢下跌、非常態波動關注反轉訊號，可做空

實例：

  • 情境：某股票CCI從下向上突破+100
  • 操作：中短線買入，若有大成交量配合更可靠
  • 預期：股價進入強勢上漲階段

CCI六大進出場訊號

怎麼做：

1. 突破+100買進：CCI從下向上突破+100，股價進入非常態區間，買進訊號
2. 跌破+100賣出：CCI從上向下跌破+100，強勢可能結束，逢高賣出
3. 跌破-100觀望：CCI從上向下跌破-100，進入尋底過程，持幣觀望
4. 突破-100少量買進：CCI從下向上突破-100，探底可能結束，逢低少量買進
5. 常態區用其他指標：CCI在-100到+100之間，改用KDJ、W&R等指標研判
6. 遠離區間看強度：CCI遠離+100或-100時，代表趨勢強度

CCI背離訊號

是什麼：當CCI曲線方向與股價K線方向相反時，稱為背離。

頂背離（賣出訊號）：

  • 是什麼：股價創新高，CCI反而在高點形成一峰比一峰低
  • 怎麼做：

1. CCI在+100以上的高位
2. 股價創新高但CCI沒有創新高
3. 這是強烈賣出訊號

  • 實例：股價從50漲到60再漲到65，CCI卻從+180降到+150再降到+120，頂背離

底背離（買進訊號）：

  • 是什麼：股價創新低，CCI反而在低位形成一底比一底高
  • 怎麼做：

1. CCI在-100以下的低位
2. 股價創新低但CCI沒有創新低
3. 需要多次確認，只能戰略建倉或短線

  • 實例：股價從40跌到35再跌到30，CCI卻從-120升到-100再升到-80，底背離

重要原則：頂背離準確性高於底背離，需要多次底背離才能確認

CCI曲線形態研判

是什麼：CCI曲線也會形成M頭、W底等反轉形態。

怎麼做：

形態位置訊號操作
M頭+100以上高位由強轉弱，即將下跌賣出
三重頂+100以上高位強勢結束，大跌將至賣出
W底-100以下低位由弱轉強，即將反彈少量買進
三重底-100以下低位底部確認買進

實例：

  • 某股票CCI在+200附近形成M頭
  • 頸線跌破後，預估跌幅 = 頭部到頸線的距離
  • 操作：及時賣出

CCI實戰操作要點

選股技巧：
1. 選擇CCI在-220到-400的股票
2. 股價創新低但CCI沒有創新低（底背離）
3. 當CCI上穿-100時是最後買入點

加碼時機：
1. CCI到+100時可加碼買進
2. +100以上開始回頭跌破+100賣出
3. 第二次穿過+100通常會創新高
4. 若指標背離再跌破+100，堅決賣出

參數設定：使用原始參數14日，週期越長越準確（週線、月線）

CCI六大秘訣

秘訣一：CCI圍繞地線震盪後向上突破，表示股價轉強，可尋機介入

秘訣二：CCI上穿天線後達300左右，若不能持續上升，是短線出貨機會

秘訣三：從上往下跌破-100再突破+100，是洗盤結束、拉升開始的訊號

秘訣四：CCI在常態區運行一個月以上，突破+100後又回落，但5日後再次突破+100且成交量放大，可大膽買入（主力洗盤結束）

秘訣六：CCI向上穿越+100後持續上揚，強者恆強，後續漲幅驚人

CCI優缺點分析

優點：

  • 無上下限，不會鈍化
  • 適合非常態行情（暴漲暴跌）
  • 可識別趨勢反轉訊號
  • 參數簡單（14日）

缺點：

  • 常態區間（-100到+100）幾乎無效
  • 無法給出具體交易訊號
  • 需要搭配其他指標使用
  • 波動小的市場容易產生假訊號

MACD趨勢共振CCI與MACD同時金叉，強買訊號

心得：

CCI這指標挺有意思的，跟KDJ、RSI最大的不同就是「不限範圍」這一點。不會鈍化這特性對於抓暴漲暴跌行情很好用，但在常態震盪盤就變成廢物了。

1. 不要在常態區用CCI - -100到+100之間它幾乎沒用，要切換成KDJ或RSI

3. 週期越長越準 - 週線比日線可靠，日線比小時線可靠
4. 必須搭配其他指標 - 單獨用CCI無法給出具體交易點

最重要的是：CCI是針對「非常態」設計的，常態行情就要換工具。工具選錯比沒工具還慘。

第 10 回合學習：技術分析：MTM 動量指標

學習時間：16:59

文章：

  • MTM指標完整教學，MTM動量指標參數設定與優缺點分析，當沖怎麼用

核心知識

MTM 動量指標基本概念

是什麼：MTM（Momentum Index）動量指標，由蓋特雷（H. M. Gartley）於1935年提出，用來測量價格變化速度的技術指標。

原理：基於「動量守恆定律」，觀察價格變化的速度和慣性來判斷趨勢走向。

  • 價格變化速度加快 → 趨勢可能持續一段時間
  • 價格變化速度放緩 → 趨勢可能反轉

MTM 公式計算

公式：

MTM = Ct - Ct-n
  • Ct：當日收盤價
  • Ct-n：n 個交易日前的收盤價
  • n：通常設為 10 或 12 天

實例：

  • n = 10 時，MTM = 當日收盤價 - 10 天前收盤價
  • 例：當日收盤 105 元，10 天前收盤 100 元
  • MTM = 105 - 100 = +5（正值表示上漲動能）

MTM 指標運用方法

1. 穿越中心線訊號

買進訊號：

  • 動量線從下方穿過中心線（零軸）
  • 意味著價格和趨勢轉向多方
  • 可以考慮做多

賣出訊號：

  • 動量線從上方穿過中心線（零軸）
  • 意味著價格和趨勢轉向空方
  • 可以考慮做空

實例：

  • 情境：某股票 MTM 從 -3 上升到 +1，穿越零軸
  • 操作：買進訊號，可在確認穿越後進場

2. 極值判斷

高點徘徊：

  • MTM 在高點徘徊
  • 代表市場處於強勢行情
  • 上漲趨勢可能持續

低點徘徊：

  • MTM 在低點徘徊
  • 代表市場處於弱勢行情
  • 下跌趨勢可能持續

實例：

  • 情境：MTM 連續 5 天維持在 +8 以上高檔
  • 判斷：強勢行情，上漲趨勢持續中

MTM 當沖實戰應用

怎麼做：

1. 搭配均線使用：

  • 短週期均線：5 MA 或 10 MA

2. 多方判斷：

  • 價格在均線之上

3. 空方判斷：

  • 價格在均線之下

實例：

  • 情境：台積電股價 500 元，5MA 在 495 元，MTM = +5
  • 操作：股價站上 5MA，MTM 為正，確認多方
  • 進場：買進多單
  • 出場：MTM 跌破零軸或價格跌破 5MA 時出場

參數設定建議

標準設定：

  • MTM（動量線）：n = 10 或 12
  • 中心線：0（零軸）

當沖設定：

  • 短週期：n = 5 或 10
  • 搭配 5MA 或 10MA 使用

MTM 優缺點分析

優點：
1. 計算方式簡單，容易理解
2. 可廣泛應用不同金融商品
3. 能檢測市場趨勢方向
4. 幫助掌握趨勢波段

缺點：
1. 只考慮過去價格走勢，忽略市場新聞和經濟環境
2. 重大事件時可能無法準確預測
3. 對市場整體趨勢預測能力有限

MTM 搭配其他指標

適合搭配：

  • 移動平均線（MA）：確認趨勢方向
  • RSI 或 KD：判斷超買超賣
  • 成交量：確認動能強弱

組合策略：
1. MTM 穿越 + 價格站上均線 → 買進
2. MTM 穿越 + RSI 不在超買區 → 增加 confidence
3. MTM 穿越 + 成交量放大 → 確認動能

適用時機

適合：

  • 趨勢明確的市場
  • 波段交易
  • 當沖（搭配短週期均線）

不適合：

  • 盤整理區間
  • 橫盤震盪市場
  • 重大消息事件前後

心得

MTM 動量指標是個簡單直觀的指標，概念就是「測量價格變化速度」。比起其他複雜的指標，MTM 公式就是「現在價格 - n天前價格」，簡單到不行。

但這種簡單也是雙面刃。計算簡單，但訊號可能會被噪音干擾。所以我認為 MTM 更適合當作輔助指標，不是主要決策依據。用來「確認」趨勢方向，而不是「發現」趨勢轉折。

當沖應用方面，MTM 搭配短週期均線（5MA/10MA）的方式很實用：

  • 價格站上均線 + MTM 為正 → 多方確認
  • 價格跌破均線 + MTM 為負 → 空方確認

這種雙重驗證比單用 MTM 靠譜多了。

之前學 ROC 變動率指標，概念跟 MTM 很像，都是在測量價格變化速度。差別在 ROC 用百分比，MTM 用絕對值。實戰上我覺得百分比可能更直觀（比如上漲 5% vs MTM = +5），但原理相通。

下一個要學的候補：CDP 逆勢操作系統，這個看起來是當沖專用的支撐壓力計算方式。

第 11 回合學習：威廉分形與鱷魚線交易策略

學習時間：18:09

文章：

  • Bill Williams所開發的分形理論(Fractal Theory) - OANDA Lab
  • 置換移動平均線是什麼？介紹DiNapoli與鱷魚線指標（Alligator） - OANDA Lab
  • 鱷魚指標詳解 —— 威廉斯鱷魚指標 | 萊特金融
  • The Williams Fractals Swing Trading Strategy | Medium

核心知識

威廉分形 (Williams Fractal)

是什麼：由比爾·威廉姆斯開發的技術指標，用於識別市場的轉折點（高點和低點），是混沌交易理論的一部分。

怎麼做：
1. 定義：分形由5根K棒組成

  • 向上分形（賣出分形）：中間那根K棒的高價高於左右兩側各2根K棒的高價

2. 確認條件：5根K棒完全形成後，分形才會被確認（第6根K棒出現時才能確定）
3. 標記方式：向上分形顯示朝上箭頭，向下分形顯示朝下箭頭

實例：

  • 情境：某股票在60-65元區間震盪，出現分形
  • 操作：當價格突破向下分形的高點66元時，確認突破成功可進場做多；停損設在分形低點62元

適用時機：

  • 用來畫趨勢線和支撐壓力線
  • 作為波段進場的突破訊號
  • 作為停損點設定的參考位置
  • 需配合鱷魚線或其他指標過濾假訊號

鱷魚線 (Alligator Indicator)

是什麼：由比爾·威廉姆斯開發的趨勢追蹤指標，由三條不同週期的平滑移動平均線組成，用來判斷市場狀態（趨勢或盤整）。

怎麼做：
1. 三條線組成：

  • 嘴唇線（綠線）：5日線，向未來位移3根K棒

2. 計算價格：採用中位價（最高價+最低價）/2，而非收盤價
3. 判讀狀態：

  • 鱷魚睡覺：三線糾纏在一起，市場盤整

實例：

  • 多頭排列：綠線 > 紅線 > 藍線，且價格在所有線上方
  • 空頭排列：藍線 > 紅線 > 綠線，且價格在所有線下方
  • 進場時機：三線從糾纏開始發散時

適用時機：

  • H1、H4、D1 時間框架（較短時間框架噪音太多）
  • 趨勢初期的進場訊號
  • 判斷盤整與趨勢的轉換點
  • 作為動態支撐壓力參考

鱷魚線與分形組合策略 (Profitunity System)

是什麼：比爾·威廉姆斯的完整交易系統，結合鱷魚線判斷趨勢方向，分形確認進場點。

怎麼做 - 買入步驟：
1. 確認鱷魚正在甦醒：綠線向上穿過紅線，紅線向上穿過藍線
2. 價格在所有鱷魚線上方
3. 找到最近的向上分形（突破阻力）
4. 等待價格突破向上分形的高點
5. 進場做多，停損設在最近向下分形的低點下方

怎麼做 - 賣出步驟：
1. 確認鱷魚正在甦醒：綠線向下穿過紅線，紅線向下穿過藍線
2. 價格在所有鱷魚線下方
3. 找到最近的向下分形（跌破支撐）
4. 等待價格跌破向下分形的低點
5. 進場做空，停損設在最近向上分形的高點上方

實例：

  • 情境：某股票價格180元，鱷魚線多頭排列，最近向上分形高點182元，向下分形低點175元
  • 操作：價格突破182元時進場做多，目標184元以上（2倍風險），停損設175元
  • 報酬風險比：目標獲利/(182-175)=7元風險，目標2倍風險=14元獲利目標

適用時機：

  • 趨勢形成初期
  • 避免在鱷魚睡覺（盤整）時交易
  • 分形形成的K棒不應觸及鱷魚線（否則視為無效訊號）

不同時間框架的參數調整

是什麼：鱷魚線參數可根據時間框架調整，以適應不同的交易風格。

怎麼做：

時間框架下顎參數牙齒參數嘴唇參數用途
H4~D1（標準）週期13,位移8週期8,位移5週期5,位移3中長期波段
H1（調整）週期21,位移13週期13,位移8週期8,位移5日內交易
M5（快速）週期34,位移21週期21,位移13週期13,位移8過濾噪音

實例：

  • 原始參數（13,8,5）適用於日線波段
  • 在小時圖上用（21,13,8）可減少假訊號
  • 在5分鐘圖上用（34,21,13）可過濾更多噪音

適用時機：

  • 時間框架越短，參數應越大以過濾噪音
  • 時間框架越長，可用標準參數
  • 調整前需在歷史數據上測試驗證

保本位設定與平倉時機

是什麼：使用鱷魚下顎線作為趨勢邊界，動態調整停損位置。

怎麼做：
1. 保本時機：當下顎線（藍線）高於進場價（多頭）或低於進場價（空頭）時，將停損移至保本
2. 手動平倉時機：

  • 鱷魚進入睡眠狀態（三線開始糾纏）

3. 獲利目標：建議設為停損距離的2倍

實例：

  • 多頭部位進場價180元，原始停損175元
  • 當藍線上升到181元時，將停損改為180元（保本）
  • 獲利目標設在180 + 2×(180-175) = 190元

適用時機：

  • 趨勢追蹤交易中
  • 避免獲利回吐
  • 作為動態停利依據

心得

威廉分形和鱷魚線是比爾·威廉姆斯混沌交易系統的核心工具，這兩個指標搭配使用非常巧妙：

優點：
1. 視覺直觀：分形直接標示高低點，鱷魚線清楚顯示市場狀態
2. 趨勢辨識能力強：鱷魚線能清楚區分盤整與趨勢，避免在盤整時進場
3. 進場點明確：分形突破提供精確的進場訊號
4. 停損點清晰：分形的低點/高點直接作為停損參考

缺點：
1. 落後性：分形要等第6根K棒才能確認，有落後
2. 盤整時假訊號多：鱷魚睡覺期間不應交易，但很多交易者會忍不住
3. 參數需調整：不同時間框架需要調整參數，沒有萬用設定

實戰重點：

  • 鱷魚睡覺時不要交易！這是最重要的紀律
  • 分形突破後要確認價格站穩，避免假突破
  • 配合成交量或其他指標（如AO動量震盪指標）過濾訊號
  • 保本位的設定是保護獲利的關鍵

這套系統適合波段交易和趨勢追蹤，不適合當沖。在台股市場可以應用在權值股和期貨上，但要注意台股的跳空風險。
