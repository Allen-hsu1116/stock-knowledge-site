一、肯特納通道是什麼

是什麼：由切斯特·肯特納（Chester Keltner）於1960年在《如何在商品中賺錢》書中提出的技術指標，透過計算價格波動幅度建立通道來評估市場趨勢、強度和波動性。

核心概念：結合移動平均線（EMA）與平均真實波幅（ATR）形成三條線的通道，用來判斷趨勢方向與波動範圍。

二、肯特納通道公式與結構

三條線組成

中軌（Middle Band）：

  • 公式：EMA 20（20日指數移動平均線）
  • 意義：反映整體趨勢方向

上軌（Upper Band）：

  • 公式：EMA 20 + ATR × M
  • 意義：壓力線，M通常設為2

下軌（Lower Band）：

  • 公式：EMA 20 - ATR × M
  • 意義：支撐線，M通常設為2

ATR（平均真實波幅）計算

TR（真實波幅）取以下三個值的最大值：
1. 今日最高價 - 今日最低價
2. |今日最高價 - 昨日收盤價|
3. |今日最低價 - 昨日收盤價|

ATR：TR的算術平均（通常取14日或20日）

通道寬度變化：

  • 波動度高 → ATR放大 → 通道變寬
  • 波動度低 → ATR縮小 → 通道變窄

三、四種基本交易策略

1. 多頭順勢交易

怎麼做：
1. 均線斜率往上（中軌角度上揚）
2. 價格由下往上突破上通道
3. 買進多單

實例：

  • 情境：台積電股價在590元附近，中軌（EMA 20）呈現上揚角度
  • 操作：價格從585元突破上軌595元時進場，目標看610元
  • 停損：設在進場K線低點583元

適用時機：趨勢明確向上，突破上軌表示動能強勁

2. 多頭逆勢交易（拉回買進）

怎麼做：
1. 均線斜率往上（多頭趨勢）
2. 價格回調觸及或跌破下通道
3. 等待轉陽（從陰線轉陽線）時買進

實例：

  • 情境：鴻海股價在上升趨勢，中軌角度向上
  • 操作：價格從140元回跌觸及下軌132元，K線轉陽後在134元進場
  • 停損：設在下軌下方130元
  • 目標：回到中軌140元或上軌148元

適用時機：上升趨勢中的回調買點，風險較低

3. 空頭順勢交易

怎麼做：
1. 均線斜率往下（中軌角度下降）
2. 價格由上往下跌破下通道
3. 進場空單

實例：

  • 情境：聯發科股價在下跌趨勢，中軌明顯下彎
  • 操作：價格從950元跌破下軌920元時放空
  • 停損：設在進場K線高點955元
  • 目標：看900元或880元

適用時機：趨勢明確向下，跌破下軌表示下跌動能強

4. 空頭逆勢交易（反彈放空）

怎麼做：
1. 均線斜率往下（空頭趨勢）
2. 價格反彈觸及或突破上通道
3. 等待轉陰時放空

實例：

  • 情境：大盤在下跌趨勢，中軌向下
  • 操作：期貨從18000反彈觸及上軌18200，K線轉陰後在18150放空
  • 停損：設在上軌上方18250
  • 目標：回到中軌17900或下軌17600

適用時機：下跌趨勢中的反彈賣點

四、當沖實戰策略（平均K線 + 肯特納通道）

進場條件

步驟①確認趨勢：

  • 觀察肯特納通道角度
  • 角度上揚 → 上升趨勢
  • 角度持平 → 不交易

步驟②第一次進場（逢低買進）：

  • 平均K線在中軌（EMA）上方
  • 從陰線轉陽線時進場
  • 停損：前一根K線低點

步驟③第二次進場（突破跟進）：

  • 對突破近期上軌的高價下限價停損單（Stop Limit Order）
  • 乘著突破力道獲利

出場條件

停損設定：

  • 設在進場K線的前一根K線低價

停利設定：

  • 價格範圍的1.5倍
  • 例：停損10pips → 停利15pips（盈虧比1:1.5）
  • 或當平均K線轉陰時全部平倉

時間週期建議：

  • 當沖使用1小時圖表
  • 波段可用日線圖

五、肯特納通道與布林通道的差異

特性肯特納通道布林通道
通道寬度計算ATR（平均真實波幅）標準差
對極端價格反應較平滑，不易被單根K棒影響較敏感，容易被極端值拉寬
適用情境趨勢明確的市場波動劇烈或盤整市場
假突破次數較少較多
信號穩定性較穩定較頻繁

實戰心得：肯特納通道用ATR計算通道寬度，比布林通道用標準差更穩定，少了單根極端K棒的干擾。

六、優缺點分析

優點

1. 對趨勢變化反應快速：價格突破上下軌可迅速識別趨勢
2. 適用多種市場：股票、期貨、外匯都適用
3. 通道寬度動態調整：ATR會隨波動度自動調整
4. 信號相對穩定：比布林通道的假突破少

缺點

1. 盤整市場假信號：波動小時通道過窄，容易誤判
2. 滯後性：依賴移動平均線計算，有延遲
3. 劇烈行情跟不上：快速變動時信號反應慢

七、進階應用技巧

1. 趨勢判斷

上漲趨勢：

  • 價格持續在中軌之上
  • 突破上軌 → 強勢上漲

下跌趨勢：

  • 價格持續在中軌之下
  • 跌破下軌 → 強勢下跌

2. 波動幅度判斷

通道變寬：

  • 市場波動加大
  • 可能有大行情

通道變窄：

  • 波動減弱
  • 可能進入盤整

3. 搭配其他指標

搭配RSI：

  • 通道突破時，RSI確認是否超買/超賣
  • 防止假突破

搭配移動平均線：

  • 價格突破上軌且在MA之上 → 強買入
  • 價格跌破下軌且在MA之下 → 強賣出

搭配平均K線：

  • 中軌角度向上 + 平均K線轉陽 → 多單進場
  • 中軌角度向下 + 平均K線轉陰 → 空單進場

八、參數設定建議

標準設定：

  • EMA週期：20日
  • ATR週期：14日或20日
  • ATR倍數：2倍（可調整1.5-2.5）

調整原則：

  • EMA越長 → 指標越落後，信號越少
  • EMA越短 → 反應越快，雜訊越多
  • 倍數越大 → 通道越寬，突破越少但品質越高
  • 倍數越小 → 通道越窄，突破越多但假突破也多

心得

肯特納通道的核心在於「用ATR決定通道寬度」，這比布林通道用標準差更穩定。因為標準差容易被單根極端K棒拉寬，而ATR計算的是一段時間的平均波動，相對平滑。

實戰應用上，我覺得最有價值的是「四種策略搭配趨勢判斷」：
1. 順勢交易適合趋势明確時，突破上下軌跟進
2. 逆勢交易適合在趨勢中等待回調後進場，風險較低

關鍵是「觀察中軌角度」：角度明顯上揚或下彎才值得交易，角度平坦就別碰。

另一個實用技巧是「搭配時間週期」：小時圖做當沖、日線圖做波段。大週期確認方向，小週期找進場點。

跟之前學的布林通道相比，肯特納通道的「假突破較少」是最大優勢。如果你發現布林通道的突破信號太多、被巴來巴去，試試肯特納通道可能會改善。
