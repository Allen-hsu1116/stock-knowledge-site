一、MACD波段進出場實戰策略

1. MACD黃金交叉與死亡交叉進出場

是什麼：MACD快線（DIF）與慢線（MACD線/DEA）的交叉訊號，用來判斷波段趨勢的轉折點。

怎麼做：
1. 零軸上方金叉（最強訊號）：快線在零軸上方由下往上穿過慢線 → 多頭趨勢延續或加速，可進場做多
2. 零軸下方金叉（弱訊號）：快線在零軸下方上穿慢線 → 可能只是空頭反彈，需搭配其他指標確認
3. 零軸下方死叉（最強訊號）：快線在零軸下方由上往下跌破慢線 → 空頭趨勢延續，應迴避或做空
4. 零軸上方死叉（弱訊號）：快線在零軸上方跌破慢線 → 可能只是多頭回檔，不一定是大跌

實例：

  • 情境：台積電(2330)日線MACD在零軸上方，DIF由下往上穿越DEA
  • 操作：金叉當天收盤價600元進場，停損設在前波低點580元（約3.3%），停利目標設630元（5%）
  • 若柱狀體持續放大（紅柱變長），代表多頭動能加強，可加碼或持續持有

適用時機：波段趨勢明確時（均線多頭排列或空頭排列），不適合盤整行情

2. MACD背離波段反轉判斷

是什麼：股價走勢與MACD指標方向不一致，暗示趨勢動能減弱，可能即將反轉。

怎麼做：
1. 頂背離（多頭結束訊號）：股價創新高，但MACD快線或柱狀體沒有創新高 → 多頭力道衰退
2. 底背離（空頭結束訊號）：股價創新低，但MACD快線或柱狀體沒有創新低 → 空頭力道減弱
3. 確認條件：背離出現後，需等MACD柱狀體由紅轉綠（頂背離）或由綠轉紅（底背離）才算確認

實例：

  • 情境：某股股價從100元漲到120元新高，但MACD快線從8降到5
  • 操作：頂背離出現 → 開始減碼或設移動停損，當柱狀體由正轉負時全部出場
  • 停損：若股價再創新高且MACD也創新高，則背離失效

適用時機：趨勢末端（非趨勢中段），搭配成交量確認更準確

3. MACD搭配均線提升勝率

是什麼：單用MACD容易在盤整出現假訊號，搭配均線可以過濾掉大部分假訊號。

怎麼做：
1. 加EMA99趨勢濾網：股價在EMA99上方時出現金叉，勝率較高；股價在EMA99下方時出現金叉，可能是假訊號
2. 搭配季線（60日MA）：股價在季線上方 + MACD金叉 → 多頭訊號可靠；股價在季線下方 + MACD死叉 → 空頭訊號可靠
3. 看柱狀體再決定：柱狀體由負轉正（金叉前兆）→ 準備進場；柱狀體由正轉負（死叉前兆）→ 準備出場

實例：

  • 情境：聯發科(2454)股價在季線上方，MACD柱狀體由負轉正後2天出現金叉
  • 操作：金叉當天進場，停損設季線下方3%，停利看柱狀體縮小到零軸附近時減碼
  • 避開陷阱：若股價在季線附近糾纏且MACD頻繁交叉，則觀望不操作

適用時機：日線或週線等大週期，小時線雜訊太多不建議

二、MACD波段交易三大陷阱與對策

4. 震盪盤假訊號陷阱

是什麼：盤整行情中MACD快線慢線反覆交叉，金叉死叉頻繁出現但都是假訊號。

怎麼做：
1. 先看均線斜率：MA20斜率平緩（角度<15度）→ 盤整機率高，不操作
2. 看柱狀體振幅：柱狀體在零軸附近小幅波動（振幅<0.5%）→ 不操作
3. 看成交量：金叉出現但量縮 → 假訊號機率高；金叉+放量 → 真突破機率高

實例：

  • 情境：某股在50-55元區間盤整2週，MACD反覆金叉死叉3次
  • 對策：全部忽略，等價格帶量突破55元且MACD柱狀體明顯放大才進場

適用時機：遇到盤整行情時必用，避免在盤整中反覆虧損手續費

5. 滯後指標延遲進場

是什麼：MACD是滯後指標，等金叉出現時股價往往已經漲了一段，進場點不理想。

怎麼做：
1. 看柱狀體提前卡位：柱狀體由負開始縮小（綠柱變短）→ 動能正在轉強，可以開始觀察準備進場，不用等金叉完全形成
2. 搭配K線型態：出現早晨之星、吞噬型態等反轉訊號 + MACD柱狀體縮小 → 比金叉更早進場
3. 分批進場：柱狀體縮小時先進1/2部位，金叉確認時再加碼1/2

實例：

  • 情境：某股從80元跌到70元，出現長下影線，MACD綠柱從-2縮小到-0.5
  • 操作：先進場1/2部位（70元），金叉確認時（72元）再加碼1/2
  • 停損：跌破前波低點68元出場

適用時機：想提高進場效率、不想等金叉確認太晚進場時

三、平均K線+移動均線波段交易法

6. 平均K線(Heikin Ashi)波段操作法

是什麼：平均K線是將傳統K線經過平均計算改良的指標，更容易判斷趨勢的發生、持續與結束。趨勢中容易連續出現同色K線，轉折時顏色會切換。

怎麼做：
1. 設定指標：日K線圖 + 平均K線 + 30日簡單移動平均線(SMA)
2. 確認趨勢：SMA30角度向上 → 多頭趨勢；SMA30角度向下 → 空頭趨勢；SMA30持平 → 盤整不操作
3. 多頭進場：連續紅K中出現一根黑K（回檔），等待黑K後再出現紅K時進場做多
4. 多頭出場：連續紅K轉為黑K時獲利了結（或移動停損）
5. 停損：跌破進場時最近的K線低點

實例：

  • 情境：某股SMA30向上，平均K線連續5根紅K後出現1根黑K，接著再出現紅K
  • 操作：紅K當天收盤價45元進場做多，停損設黑K低點43元（約4.4%）
  • 持有時機：只要平均K線持續紅K就持有，出現黑K就出場或移動停損

適用時機：波段趨勢明確時（SMA30斜率明斜率明顯），趨勢弱時容易出現假訊號需觀望

四、波段操作選股與時機判斷

7. 波段操作標的選擇四大原則

是什麼：不是所有股票都適合波段操作，選對標的才能提高勝率。

怎麼做：
1. 趨勢性要好：選擇行情走勢平穩、方向穩定的標的，避免震盪劇烈的個股
2. 流通量要夠：日均成交量至少5000張以上，確保進出場不會滑價
3. 優先選大盤指數或產業ETF：個股容易受單一因素影響，大盤指數或龍頭股（如台積電）更適合波段操作
4. 避開主力操控明顯的小型股：容易被主力洗盤或出貨，波段操作難度大增

實例：

  • 好標的：台積電(2330)日均量3萬張、趨勢穩定、流動性佳 → 波段操作勝率高
  • 差標的：某營建股日均量500張、震盪劇烈、主力明顯 → 波段操作容易被洗

適用時機：每次波段操作前都要先確認標的是否符合條件

8. 波段操作五大事件驅動策略

是什麼：利用「發酵時間較長的事件」進行波段布局，事件不會短期結束，進出場不用爭分奪秒。

怎麼做：
1. 升降息週期：美聯儲開始升息週期 → 做多美元；通膨觸頂回落時出場
2. 產業突破性技術：如AI技術突破 → 做多相關產業ETF（非單一個股）
3. 供需失衡：生產週期長的產品（晶片、農產品）供需失衡時做多；出場時機依生產週期評估
4. QE/QT政策：QE時做多黃金、比特幣等保值資產；QT時反向操作
5. 技術面多頭格局：盤整越久後突破前高的標的，敢買強勢股，判斷是否為「發酵較長時間的事件」

實例：

  • 情境：2022年3月美國開始升息，美元指數從98開始飆升
  • 操作：升息週期開始後做多美元指數，到10月CPI觸頂回落時出場
  • 獲利：美元指數上漲約15%，若開10倍槓桿可獲利150%

適用時機：有明確的長期事件驅動時，不適合短線波動行情

心得

今天週四的主題是操作策略，我學到了幾個重要的波段操作觀念：

1. MACD不是萬能的，但搭配均線就變強了：單用MACD金叉死叉在盤整行情會被洗到懷疑人生，但加一條EMA99或季線當趨勢濾網，勝率可以大幅提升。這個「濾網」的概念很重要——不是每個訊號都要做，只在趨勢明確時做。

2. 柱狀體比交叉更早訊號：等金叉出現往往已經漲一段了，但柱狀體從負轉正（或從正縮小）可以更早察覺動能變化。分批進場的方式很實用——柱狀體縮小時先進半倉，金叉確認再加碼。

3. 平均K線波段法簡單有效：只要看SMA30角度和平均K線顏色就能操作，連上班族都能用（看日線早晨判斷進出場）。但關鍵是「趨勢弱時不操作」，這跟MACD盤整不操作的邏輯一致。

4. 波段操作的核心不是技術，是選對標的：趨勢性好的大盤ETF或龍頭股才是波段操作的好朋友，小型股容易被主力操控，波段操作反而更難。

5. 事件驅動才是波段操作的大肉：升降息週期、產業技術突破、供需失衡這些「長期發酵的事件」，才是波段操作最容易賺大錢的時機。不用爭分奪秒，但要有耐心等大趨勢。
