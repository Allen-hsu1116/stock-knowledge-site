一、蝶式價差策略（Butterfly Spread）

是什麼

蝶式價差是一種風險有限、獲利也有限的盤整策略，由三種不同履約價的四口選擇權組成。

怎麼做

買進蝶式價差建構步驟：
1. 買進一組多頭價差（買低履約價 Call + 賣高履約價 Call）
2. 買進一組空頭價差（賣低履約價 Call + 買高履約價 Call）
3. 多頭價差的較高履約價 = 空頭價差的較低履約價（中間共享同一履約價）
4. 兩組價差履約價間距必須相等

實例（大盤22,000點）：

  • 買進 21,800 Call，支付 515 點
  • 賣出 22,000 Call，收取 410 點（賣兩口）
  • 買進 22,200 Call，支付 315 點
  • 淨支出：515 - 410×2 + 315 = 10 點（$500）

最大獲利與風險

最大獲利：結算價正好落在中間履約價（22,000點）

  • 最大利潤 = 多頭價差最大利潤 + 空頭價差最大利潤
  • 實例：$9,500

最大虧損：結算價超過最高或低於最低履約價

  • 最大虧損 = 淨權利金支出 = $500

損益兩平點：

  • 高點：22,190點
  • 低點：21,810點
  • 獲利區間：380點

適用時機

  • 預期盤整，到期日不會大漲或大跌
  • 想控制風險但又希望獲利的保守策略

二、時間價差策略（Time/Calendar Spread）

是什麼

時間價差是利用不同到期日選擇權的時間價值流失速率差異獲利，核心概念是「買長賣短」——買進遠期選擇權、賣出近期選擇權。

怎麼做

買進時間價差建構步驟：
1. 買進遠期選擇權（例如8月到期），支付權利金
2. 賣出近期選擇權（例如7月到期），相同履約價，收取權利金
3. 近期到期時平倉遠期合約

實例（履約價21,800）：

  • 買進 8月 21,800 Call，支付 725 點
  • 賣出 7月 21,800 Call，收取 505 點
  • 淨成本：(725 - 505) × 50 = $11,000

最大獲利情境

結算價剛好等於履約價時：

  • 近期選擇權到期無價值，權利金全收
  • 遠期選擇權時間價值流失有限

實例最大獲利：

  • 賣出7月收505點 + 遠期殘值235點 - 買入725點 = 15點（$750）

風險與注意事項

  • 最大風險：遠期支付 - 近期收取 = 淨成本
  • 結算後需手動平倉遠月合約

三、價差單四大類型

是什麼

價差單是「一賣一買」的組合策略，賣出選擇權收權利金，同時在更價外的履約價買入保險。

四種類型

名称英文建構方式適用時機
買權多頭價差Call Debit Spread (CDS)買低履約價Call + 賣高履約價Call看漲，預期會突破某點位
賣權多頭價差Put Credit Spread (PS)賣高履約價Put + 買低履約價Put看跌不破某點位
買權空頭價差Call Credit Spread (CS)賣低履約價Call + 買高履約價Call看漲不過某點位
賣權空頭價差Put Debit Spread (PDS)買高履約價Put + 賣低履約價Put看跌，預期會跌破某點位

損益計算實例（100點價差）

實例：賣16000 Call收20點，買16100 Call付5點

  • 最大獲利：收權利金 - 付權利金 = 20 - 5 = 15點
  • 最大虧損：價差 - 淨權利金 = 100 - 15 = 85點
  • 損平點：履約價 + 淨權利金 = 16000 + 15 = 16015

保證金計算

  • 保證金 = 價差 × 50
  • 100點價差只需 $5,000
  • 50點價差最低只需 $2,500

重要注意事項

必須使用複式下單功能，不可分開建倉！

  • 分開建倉：風險無法綁定，極端行情可能被強制平倉
  • 正確做法：使用APP「複式單」或電腦「多次IOC」功能

四、鐵兀鷹策略（Iron Condor）

是什麼

鐵兀鷹是由「買權空頭價差」+「賣權多頭價差」組成的雙邊賣方策略，適合盤整格局，是風險有限的中立策略。

怎麼做

建構步驟（大盤16,000點為例）：
1. 上方：賣出16300 Call收35點，買入16400 Call付15點 → Call Spread收20點
2. 下方：賣出15700 Put收38點，買入15600 Put付18點 → Put Spread收20點
3. 總權利金收入：40點

獲利區間

  • 結算價落在15700~16300區間：全拿40點權利金
  • 結算價超過16300或跌破15700：開始虧損
  • 最大虧損：100 - 40 = 60點

三大實戰秘訣

秘訣一：Delta中立 vs 跨距中立

  • 跨距中立：上下對稱距離（如各300點）
  • Delta中立：選擇Delta值相近的履約價
  • 例：市場偏多時，Put Spread可能要縮短距離以對齊Delta

秘訣二：被貫穿時的應對

  • 小台避險：貫穿前建小台同方向單抵消虧損
  • Rolling轉倉：平掉獲利邊，在更遠處重建價差

秘訣三：時間換取空間

  • 被貫穿時可轉倉到下個到期日
  • 用更多時間期待價格拉回

Delta中性調整實例

假設16300的Call Spread賣方Delta是0.2：

  • 若市場偏多，同距離15700的Put Delta可能是-0.15（不對稱）
  • 調整：Put Spread改用15750，讓Delta接近-0.2
  • 結果：上方300點，下方250點（不對稱跨距）

五、鐵蝴蝶策略（Iron Butterfly）

是什麼

鐵蝴蝶是鐵兀鷹的變形版本，中間履約價是同一個，獲利區間更窄但最大獲利更高。

建構方式

  • 賣出中間履約價的Call和Put（兩口）
  • 買入更高履約價的Call + 買入更低履約價的Put（保險）

優缺點

優點：

  • 比賣出跨式風險有限
  • 保證金需求低（兩組100點價差只需$10,000）

缺點：

  • 損益兩平點範圍縮減
  • 收取時間價值不如裸賣跨式明顯

六、玉蜥蜴策略（Jade Lizard）

是什麼

玉蜥蜴是「賣出勒式 + 一邊保險」的三腳策略，是鐵兀鷹少一隻腳的版本。

怎麼做

建構方式：

  • 賣出Call + 賣出Put（勒式）
  • 只買一邊的保險（買Call或買Put，擇一）

使用時機

  • 預期盤整，但擔心單邊風險
  • 考量股市「緩漲急跌」特性，可只買Put邊保險

優缺點比較

策略保險費用風險獲利潛力
勒式無無限（單邊）最高
玉蜥蜴一邊單邊有限中
鐵兀鷹兩邊雙邊有限較低

七、風險管理核心要點

1. 保證金控制

  • 價差單保證金 = 履約價差 × 50
  • 100點價差 = $5,000/組
  • 可用小資金操作大部位

2. 複式下單強制要求

  • 絕對禁止分別建倉買賣兩口
  • 必須使用複式單功能同時綁定
  • 極端行情分開建倉可能被強制平倉

3. Delta中性調整

  • 不只看跨距點數，要看Delta值
  • 市場情緒偏多/偏空時調整不對稱跨距

4. 被貫穿時的應對

策略優點缺點
小台避險完全抵消虧損來回掃造成摩擦成本
Rolling轉倉用時間換空間不一定拉回

最實用的是鐵兀鷹三大秘訣：Delta中立、小台避險、Rolling轉倉。這些都是實戰中會遇到的問題，文章給了具體的應對方法。
