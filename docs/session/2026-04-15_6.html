一、大戶買賣力與散戶買賣力判讀

大戶買賣力是什麼

是什麼：大戶買賣力是盤中即時籌碼指標，透過成交單的大小（特大單、大單、中單、小單）來判斷當下是主力還是散戶在主導行情。這是「盤中就能看到」的籌碼數據，不需要等盤後。

怎麼算：

  • 大戶買賣力 =（買進特大單成交量＋買進大單成交量）−（賣出特大單成交量＋賣出大單成交量）
  • 散戶買賣力 = 買進小單成交量 − 賣出小單成交量
  • 這是從開盤到當下的「累計值」，不是某一瞬間的數值

大小單定義（以股價區分）：

  • 以台積電股價 400 元為例：
  • 不同價格的股票，大小單的金額門檻不同

怎麼用：
1. 大戶買賣力由負轉正 → 主力開始偏多，可考慮進場
2. 大戶買賣力持續為正且擴大 → 主力持續加碼，趨勢偏多
3. 大戶買賣力由正轉負 → 主力轉向偏空，注意出場或放空
4. 特大單、大單、中單差額皆為負，只有小單為正 → 主力正在倒貨，散戶接刀

實例：

  • 技嘉某日盤中，大戶買賣力由負轉正的時刻，就是不錯的進場時機
  • 群創 2022/11/18 盤中，特大單、大單、中單差額全部為負，只有小單為正 = 主力倒貨、散戶接刀
  • 智原 2022/11/18 創波段新高，但大戶買賣力開始轉賣超，且賣超張數接近近期大量 → 先出部分部位

適用時機：當沖進出場時機判斷、短線多空判斷、主力動向即時追蹤

二、大單與散單方向判斷三種盤勢

狀況一：大單散單方向相反——趨勢盤

是什麼：當大單（主力）和散單（散戶）方向完全相反時，行情通常由大單決定方向，而且會走出一波趨勢。

怎麼判斷：

  • 大單一直買、散單一直賣 → 盤勢通常收最高，把握機會加碼做多
  • 大單一直賣、散戶一直接 → 盤勢通常收最低，不要跟散戶站在同一邊

實例：

  • 某日台指走勢一路做多向上，大單能量全是紅色（做多），散單能量全是綠色（做空）
  • 大單和散営完全分歧，結果盤勢由大單主導，收在當日最高點附近
  • 這種盤勢是最順的趨勢盤，大單方向就是當日方向

核心原則：大單散單完全分歧時，永遠跟大單走，不要跟散戶站同一邊。

狀況二：大單散單同方向——轉折盤

是什麼：當大單和散單同方向（一起做多或一起做空），行情可能即將出現轉折。此時要特別注意哪一邊能量先縮小。

怎麼判斷：
1. 早盤大單和散單同方向（如一起做多）→ 做好下車準備
2. 觀察哪邊能量先開始縮小 → 縮小的那邊代表力道減弱
3. 大單能量開始變小 → 趨勢可能即將轉折，準備出場或反向操作

實例：

  • 早盤高點出現前，大單和散單同向做多
  • 但大單能量開始縮小時，台指也跟著下跌
  • 如果已經進場做多單，在大單散單同方向時就要做好結利準備

核心原則：大單散單同方向 = 轉折訊號。誰先縮小力道，趨勢就會轉向。

狀況三：漲跌幅區間窄小——盤整盤

是什麼：當指數漲跌幅很小、區間窄小時，大單散單的訊號可靠性降低，這種盤最好不要重倉進場。

怎麼判斷：
1. 大單能量持續增加，但指數卻沒有跌破前低 → 能量背離，可能反轉向上
2. 尾盤散單翻紅在軋大單 → 反轉跡象更明顯
3. 但整體來說，盤整盤試單的虧損要用停損控制，沒有下大注的空間

核心原則：盤整盤最好保持空手或小部位試探，停損一定要設。

盤勢大單散單關係操作策略
趨勢盤完全分歧跟大單走，加碼做同一邊
轉折盤同方向準備反向或出場，看誰先縮小
盤整盤不明顯或背離保持空手或小部位試探，嚴設停損

三、盤中主力出貨五大蛛絲馬跡

線索一：異常掛單行為

是什麼：主力出貨時不會直接掛超級大賣單（太明顯），而是用更隱蔽的方式。

怎麼判斷：

賣單牆（大單壓制）：

  • 在特定價位（尤其是壓力區）老是掛著大量賣單
  • 不一定成交，但光看到那堵「牆」就會打擊買方信心
  • 作用：壓制股價方便主力在更低價位慢慢出貨

螞蟻搬家式小單：

  • 把大賣單拆成很多筆小單，持續不斷賣出
  • 每筆看起來沒什麼，但累積起來量很驚人
  • 特徵：特定價位附近持續有幾十張幾十張的賣單成交，尤其成交在內盤

實例：

  • 某股在 100 元壓力區附近，賣單牆 500 張持續掛著不成交
  • 同時下方 99~100 元之間，每隔幾分鐘就有 10~30 張的內盤成交
  • 結果：股價始終無法突破 100 元，主力在 99 元區間慢慢出貨

線索二：壓力區過不去或假突破

怎麼判斷：

撞天花板：

  • 股價一次又一次想突破壓力區（前高、整數關卡、重要均線）
  • 每次快碰到就湧現賣單打回，收盤留長上影線（避雷針）
  • 代表有人在壓力區倒貨

假突破真拉回：

  • 主力先用資金把股價推過壓力區，製造突破假象
  • 散戶看到突破興奮追進
  • 沒多久股價快速跌回壓力區下方，甚至比突破前還低
  • 突破時量放大但回跌量也不小 = 典型誘多出貨

實例：

  • 某股前高 80 元，盤中一度拉升到 82 元突破
  • 散戶追進後，股價快速回落到 78 元，收盤長上影線
  • 隔天繼續下跌，追突破的散戶全部套牢

適用時機：判斷是否要追價突破、識別主力誘多手法

線索三：量價背離——漲沒力、跌爆量

怎麼判斷：

價漲量縮（上漲沒力氣）：

  • 股價還在漲，但成交量越來越小
  • 甚至創新高了，買氣卻比之前弱
  • 像爬坡爬到沒力氣，隨時會停下來
  • 可能是主力趁還有人想買，悄悄把股票賣給他們

價跌量增（下跌爆量）：

  • 股價下跌，成交量反而放大
  • 表示有人急著賣（可能是恐慌，也可能是主力大倒貨）
  • 尤其是跌破支撐位時放量，賣壓極重

實例：

  • 某股從 50 元漲到 65 元過程中，成交量從 5000 張縮減到 2000 張
  • 創新高但量不配合 = 價漲量縮 = 上漲力道衰竭
  • 隔天一根長黑跌破 60 元，成交量爆增到 8000 張 = 價跌量增 = 主力倒貨

線索四：內外盤氣氛反轉

怎麼判斷：

外盤成交減少、內盤成交增加：

  • 成交在外盤（買方追價）的比例越來越少
  • 成交在內盤（賣方殺出）的比例越來越多
  • 表示想接手的人變保守了，想脫手的人越來越急

賣單掛得比買單多很多：

  • 在關鍵壓力價位附近，掛賣的單子數量遠遠超過掛買的
  • 形成一道厚厚的「賣壓牆」
  • 可能是主力壓著股價方便出貨

注意：主力也會故意掛假單（掛大賣單嚇散戶，但隨時可能撤單），所以不能只看掛單，要搭配實際成交和價量變化一起判斷。

線索五：領頭羊率先軟掉、資金搬家

怎麼判斷：

領頭羊先跑：

  • 同族群中最強的龍頭股，突然開始漲不動、回檔
  • 甚至率先跌破重要支撐
  • 就算同族群其他股票還在漲，也要小心
  • 主力通常先賣漲最多、流動性最好的龍頭股

資金從高處流到低處：

  • 市場資金明顯從漲多的股票，流到還趴在地上很久沒漲的股票
  • 健康的類股輪動是正常的
  • 但如果發生在大盤氣氛怪異 + 漲多股票出現其他出貨跡象時 = 主力在高檔出貨後把錢移到避風港

心得

這回合學到的「大戶買賣力」和「大單散單對作」，是籌碼面分析中最即時的工具。之前學的分點進出、法人買賣超都是「盤後資料」，看到時已經慢了一天。但大戶買賣力是「盤中就能看」的數據，等於是即時透視主力動向。

最核心的觀念是：大單散單方向相反時，永遠跟大單走。這句話講起來簡單，但實戰中很多人看到股價在上漲、散戶也在搶進，就跟著追，結果大單一直在賣。盤後看籌碼才發現主力倒了一堆貨，但已經來不及了。大戶買賣力的好處就是讓你盤中就能看到這個分歧。

另一個重要收穫是「三種盤勢判斷」。以前看盤只看價格和成交量，但加上大單散單方向後，可以更精準判斷今天是趨勢盤、轉折盤還是盤整盤。趨勢盤跟大單加碼、轉折盤準備反向、盤整盤不重倉——這三種對應策略簡單明確，比之前憑感覺看盤強多了。

主力出貨的五個蛛絲馬跡也很有實戰價值。其中「螞蟻搬家式小單出貨」和「假突破真拉回」這兩個最容易上當。散戶看到突破就追，結果主力早就拆成小單在那邊慢慢倒貨了。以後遇到突破，先看大戶買賣力是否配合、內外盤比例是否正常，再決定要不要追。

最後，大戶買賣力不是萬能的——盤整盤時訊號不可靠、主力也會拆單避開大單偵測。所以還是要搭配技術面和價量一起看，不能只靠一個指標就進場。
