一、投信跟單實戰策略

投信為什麼最適合散戶跟單

是什麼：三大法人中，投信的操作特性最適合散戶追蹤——操作週期偏中短線、偏好中小型股、買超有連續性、持股資訊相對透明。

為什麼投信比外資和自營商更適合跟單：
1. 外資：資金太龐大、操作週期太長（可能好幾年）、標的以權值股為主，散戶跟單時窗不明確
2. 自營商：進出極快速、追求絕對報酬、可接受虧損極小、部位遠比外資投信少，加上「避險」部位混入買賣超數據，參考性低
3. 投信：操作週期適中（數週到數月）、偏好中小型股（股價波動大）、有法規持股上限（10%）、有績效排名壓力、季底作帳行情明確

怎麼做：
1. 每天收盤後查看投信買賣超排行（證交所、Goodinfo、CMoney 都有）
2. 篩選「投信買超占比高」的股票（投量比 > 10%）
3. 觀察投信是否「連續買超」（至少連買 2 天以上）
4. 確認股價是否在「相對低檔」（低檔爆發力高、風險低）
5. 同時檢查「主力是否也買超」（主投同步買超勝率最高）
6. 設定出場條件：投信持股比率接近 10~15% 時提高警覺

實例：

  • 緯創（3231）：3月3日股價 32.65 元，投信+主力同步買超，截至4月12日漲至 45.15 元，漲幅 38%
  • 宜鼎（5289）：3月15日股價 236.5 元，主投同步買進，4月12日漲至 322 元，漲幅 36%
  • 華城（1519）：4月6日股價 73.4 元，主投同步買進，4月12日漲至 96.8 元，漲幅 32%
  • 國巨（2327）：2017~2018年投信一路買進，股價突破 1,000 元大關

投量比：投信買超占成交量比重

是什麼：投量比 = 投信買賣超張數 ÷ 成交量張數 × 100%，衡量投信對一檔股票的「關照程度」。

怎麼做：
1. 計算每日投量比
2. 投量比 > 10%（連續 2 天更好）= 值得觀察
3. 投信買超占成交量高 + 股價在低檔 = 高爆發力標的
4. 投量比連續高 + 主力同步買超 = 最佳進場時機

適用時機：篩選投信剛開始認養的股票、判斷投信是否「認真」在買

投信持股比率警戒線

是什麼：法規規定，單一基金投資任一公司股票不得超過該公司已發行股份總數的 10%，所經理全部基金合計也不得超過 10%。但多檔基金合計可能超過 10%。

怎麼判斷：

  • 投信持股 < 5%：初期階段，還有空間
  • 投信持股 5~10%：中期階段，持續觀察
  • 投信持股接近 10%：警戒區，可能面臨「買盤力竭」
  • 投信持股 > 15%：高風險區，隨時可能「由買轉賣」

實例：

  • 截至 2019/6/12，台股 1,700 多檔中只有 11 檔投信比率超過 10%，最高不超過 20%
  • 投信持股比率來到 15% 時，買盤力竭風險大增

適用時機：判斷投信認養股的出場時機、評估投信繼續買超的空間

二、外資認養股判斷與成本線實戰

外資認養股

是什麼：當外資對單一個股持股超過 10% 以上，該股票被視為外資認養股，外資的買賣行為會對股價產生明顯影響。

外資操作特性：
1. 資金量三大法人中最大，佔台股成交量 40~60%
2. 以長期投資為主，重視基本面和產業結構
3. 偏好大型權值股（如台積電，外資持股長期 > 70%）
4. 不大會碰規模小、流動性低的公司
5. 操作容易跟隨國際趨勢變動——台灣屬新興市場，國際資金撤出時影響更大

怎麼做：
1. 查詢個股的外資持股比例（證交所、Goodinfo）
2. 外資持股 > 10% = 外資認養股，後續籌碼動向值得追蹤
3. 觀察外資是否連續買超——連續幾天買超通常有波段上升趨勢
4. 連續幾天賣超——通常有波段下跌趨勢
5. 注意外資在期貨/選擇權的佈局，比股票的短線參考價值更高

實例：

  • 台積電（2330）：外資持股長期超過 70%，2016~2018年外資連續買超 2 年，推升股價走出超級大多頭
  • 2022年外資在台股提款逾 1 兆，加權指數下跌 22.4%

外資成本線

是什麼：統計外資在某檔股票或期貨上的平均成本，透過累積一段時間的買賣成交量及金額回推出平均成本。

怎麼用：
1. 在看盤軟體上開啟外資成本線
2. 股價在成本線之上 → 外資已賺錢，預期傾向繼續加碼
3. 股價跌破成本線 → 外資已虧錢，預期傾向停損，形成賣壓
4. 外資持股比例越高（>10%），成本線參考價值越大

注意事項：

  • 取的週期長度不同，算出來的成本線位置就不同
  • 成本線不是絕對多空指標，僅供參考
  • 外資不是神仙，也會看錯、也會認賠殺出
  • 外資撤資不一定代表不看好，可能是國外有更吸引的標的

外資假外資判斷

怎麼辨識假外資：
1. 中小型個股（股本 < 50 億）出現外資買超 → 可能是假外資
2. 外資在該股過去常常隔日沖（一天大買、一天大賣）→ 假外資機率高
3. 真正的外資共同基金、ETF 偏好大型股，不會頻繁進出中小型股

實戰對策：

  • 大型權值股看外資 → 參考價值高
  • 中小型股出現外資買超 → 先確認是否假外資再決定是否跟單

三、主投同步買超策略

主投買 = 主力 + 投信同步買超

是什麼：當主力（前15名券商分點買賣超）和投信同時買超同一檔股票，這是籌碼面最強的做多訊號之一。

怎麼做：
1. 每天用軟體篩選「投信買超%高 + 主力買超」的股票
2. 確認股價在低檔（低檔爆發力強）
3. 觀察投量比是否 > 10%
4. 進場後持續追蹤主投籌碼變化
5. 出場條件：主力轉賣超 + 投信也轉賣 = 上漲行情結束

八大追蹤要點：
1. 投信第1天大買 → 隔天容易強勢跳空開高，值得留意
2. 觀察投信是否持續性買超 → 容易走波段
3. 投量比 > 10%（連續 2 天更好）→ 值得觀察
4. 投信連買 2 天 + 投量比 > 10% + 股價低檔 → 可考慮進場
5. 投信買超張數極少 → 沒吸引力，不建議跟單
6. 投信小賣 → 看主力是否持續買超，主力買股價仍有支撐
7. 主投買 → 容易發動攻擊
8. 主投賣 → 容易走偏空行情，下跌速度快

反向訊號：

  • 股價在高檔 + 投信下車 + 主力轉賣超 → 跟著下車
  • 投信買但主力拚命倒貨 → 不妙，不要跟

適用時機：波段選股、短中線進場判斷、籌碼面確認

四、三大法人跟單優先順序與風險

跟單優先順序

投信 ★★★ > 外資 ★★ > 自營商 ★（最不建議）

法人適合跟單程度原因
投信★★★操作有連續性、週期適中、標的明確、持股透明、季底作帳行情可預期
外資★★資金大但週期長、短期訊號不明確、跟單有一點效但不顯著
自營商★進出太快、避險部位混入數據、散戶拿到的是落後資訊

跟單風險提醒

1. 外資買超 ≠ 股價必漲：外資不是神仙，也會認賠殺出
2. 投信連買 ≠ 無風險：可能是為了搶績效追價，非理性買進
3. 自營商避險部位混入數據：看到的買超可能只是權證避險，非看好標的
4. 籌碼資訊是落後的：盤後才公布，散戶永遠慢一步
5. 單一法人訊號不夠：要主投同步、要搭配技術面確認

心得

這回合整理了法人跟單的實戰策略，核心結論很清楚：投信是最適合散戶跟單的法人，但不是無腦跟，是有條件的跟。

投量比 > 10% 這個指標很實用——它告訴你投信不是隨便買買，而是「認真」在買。以前我只看投信買超張數，但沒考慮佔成交量的比例，結果有些投信買超看起來很多，但其實佔成交量很小，代表投信只是小試水溫，不值得跟。投量比解決了這個問題。

另一個關鍵是「主投同步」的概念。投信買 + 主力也買 = 勝率最高。反過來說，投信買但主力拼命倒 = 大概率會悲劇。這跟之前學到的「買賣家數差」概念一致——籌碼面要看的不只是誰在買，還要看「誰還在賣」。

外資的部分，市場先生說得很坦白：「有一點效，但沒有這麼顯著有效。」這種誠實的結論比那些「跟著外資保證賺」的標題黨文章有價值多了。外資的問題在於週期太長、進出反覆，散戶拿到的又是落後資訊，所以外資數據更適合拿來判斷大方向，而不是找進場點。

最後，投信持股比率 10~15% 的警戒線也要記住——投信不是無限子彈，到達法規上限就買不了了，「買盤力竭」就是出場訊號之一。
