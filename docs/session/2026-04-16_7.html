一、美林投資時鐘：經濟週期與資產輪動框架

1. 美林投資時鐘四階段模型

是什麼：美林證券2004年提出的投資理論，用GDP經濟成長率和CPI通貨膨脹率將經濟週期分為四階段，每個階段有表現最佳的資產類別和行業類別。回測1973-2004年超過30年數據驗證。

怎麼做：

階段一：通貨再膨脹（Reflation/衰退期）

  • 經濟狀態：GDP成長停滯 ↓、通膨下降 ↓
  • 市場特徵：產能過剩、企業獲利差、央行降息刺激經濟
  • 最佳資產：債券（年化報酬9.8%）
  • 最佳行業：必需消費品（+13.3%）、金融（+11.0%）、非必需消費品（+8.9%）
  • 最差行業：能源（-12.8%）、電信（-10.2%）

階段二：復甦（Recovery）

  • 經濟狀態：GDP成長加速 ↑、通膨持續下降 ↓
  • 市場特市場特徵：寬鬆政策見效、企業獲利大幅上升、央行仍維持寬鬆
  • 最佳資產：股票（年化報酬19.9%）
  • 最佳行業：非必需消費品（+3.8%）、電信（+3.7%）、科技（+3.3%）
  • 最差行業：醫藥（-4.5%）、能源（-4.4%）

階段三：過熱（Overheat）

  • 經濟狀態：GDP成長仍在高位 ↑、通膨開始上升 ↑
  • 市場特徵：產能受限、央行升息、債券表現差
  • 最佳資產：大宗商品（年化報酬19.7%）
  • 最佳行業：科技（+4.7%）、工業（+4.3%）、能源（+4.2%）
  • 最差行業：非必需消費品（-5.8%）、材料（-3.6%）

階段四：滯脹（Stagflation）

  • 經濟狀態：GDP成長下降 ↓、通膨持續上升 ↑
  • 市場特徵：企業獲利下降、工資上漲、央行維持高利率
  • 最佳資產：現金（股票-11.7%、債券-1.9%）
  • 最佳行業：能源（+14.7%）、醫藥（+11.6%）、公用事業（+6.4%）
  • 最差行業：科技（-12.5%）、非必需消費品（-8.9%）

實例：

  • 2020年3月疫情爆發 → 衰退期：美聯儲急降息至0，債券大漲
  • 2020下半年-2021年 → 復甦期：經濟重啟、股市大漲、科技股領漲
  • 2021年底-2022上半年 → 過熱期：通膨升溫、大宗商品暴漲、央行開始升息
  • 2022下半年 → 滯脹期：經濟放緩但通膨仍高、股債雙殺、現金為王

適用時機：作為年度資產配置的宏觀框架，每季檢視一次所處階段；不適合用來做短線進出場判斷

重要限制：

  • 外部衝擊會打亂循環（如2008金融海嘯、2020疫情）
  • 最難的是判斷「目前處於哪個階段」以及「週期轉折點」
  • 2008年後美林時鐘曾長期失靈（股債雙牛、大宗商品不漲）
  • 景氣是景氣，股價是股價——價格反映預期，不等於當前經濟狀態

2. 美林時鐘台股實戰對應

是什麼：將美林時鐘的四階段對應到台股的產業結構和操作策略。

怎麼做：

通貨再膨脹/衰退期 → 做多防禦型標的

  • 台股對應：金融股（官股銀行）、高殖利率股（中華電、中鋼）、債券型ETF
  • 操作：降低股票部位至30-50%，加碼債券型ETF和現金
  • 台灣央行降息時，金融股雖然利差縮小但防禦性仍佳

復甦期 → 重壓成長股

  • 台股對應：半導體（台積電、聯發科）、電子零組件、科技ETF
  • 操作：股票部位拉高至70-90%，選擇營收成長率最高的產業
  • 復甦期是股市最容易賺大錢的階段

過熱期 → 轉向原物料和能源

  • 台股對應：塑化股（台塑四寶）、鋼鐵股（中股（中鋼、豐興）、航運股
  • 操作：開始減碼成長股，轉進原物料族群
  • 注意央行升息腳步，升息末期要加速減碼

滯脹期 → 現金為王

  • 台股對應：保留現金、買進美元定存、短天期債券
  • 操作：股票部位降至20%以下，只保留高殖利率防禦股
  • 這是股市最容易虧大錢的階段，寧可空手也不要硬做

實例：

  • 2022年台股滯脹期：加權指數從18600跌到12600（-32%），科技股首當其衝
  • 若在滯脹期持有現金或美元定存（利率4-5%），反而穩穩賺

適用時機：每年初做年度資產配置時先判斷所處階段；每月用GDP和CPI數據追蹤是否轉折

二、相對強度RS指標：找出強勢股和強勢類股

3. RS相對強度指標定義與計算

是什麼：衡量個股相對於大盤或基準指數的強度，將強度量化為0-100的排名。RS=95代表過去一年該股表現為全市場前5%強。這不是RSI（相對強弱指標），是完全不同的指標。

怎麼做：
1. 計算公式：RS = 個股收盤價 / 大盤收盤價（或類股指數/加權指數）
2. 量化排名：將全市場股票的RS值排序，0-100分位數化
3. 期間選擇：常用年RS（250日）、季RS（60日）、月RS（20日）
4. 觀察趨勢：RS值的變化方向比絕對數值更重要——RS持續上升代表相對大盤越來越強

實例：

  • 年RS=95的股票 → 過去一年漲幅排全市場前5%
  • 年RS=30的股票 → 過去一年漲幅排全市場後30%
  • 重點不是RS的絕對值，是RS的變化趨勢——RS從50上升到70比RS一直在90但下降更值得注意

適用時機：選股濾網的第一步，先篩出RS>70的股票再進一步分析

與RSI的差異：

  • RS（相對強度）= 個股vs大盤的相對表現排名
  • RSI（相對強弱指標）= 個股自身漲跌力道的擺盪指標
  • 兩者完全不同，不要混淆

4. RS指標三大應用實戰

是什麼：尼克萊根據《超級績效》作者Mark Minervini的方法論，提出的RS三大實戰應用。

怎麼做：

應用一：找帶頭領漲股
1. 在多頭市場中，用RS篩選出排名前10%的股票（RS>90）
2. 在底部轉折時，找出RS快速提升的股票——它們已領先打好底部
3. 再用技術面（均線多頭排列、杯柄型態）和基本面（營收成長）做第二層過濾
4. Minervini實證：「最強飆股在向上突破第一個整理區域時，平均RS評分是87分」

應用二：挑選放空標的
1. 不要隨便做空RS高的股票——RS高代表有真實強度
2. 等待明確頭部型態出現（M頭、頭肩頂）
3. 或是最後一波突破失敗（假突破）
4. 衰敗的高RS股才是放空標的——RS從90降到50以下 + 技術面反轉

應用三：研究市場現況
1. 觀察高RS股的突破是否順利——如果高RS股集體突破失敗，大盤可能轉壞
2. 突破的檔數多寡——越多高RS股成功突破，多頭越健康
3. 大型股在高RS名單的比例——大型股強度提升通常代表趨勢可持續
4. 2022 Q1尼克萊觀察到高RS股集體走勢變差，及早辨認出大盤轉壞

實例：

  • 應用一：2023年AI熱潮初起，緯創（3231）RS從40快速拉升至85，技術面均線多頭排列 → RS領漲股篩選成功
  • 應用二：某股RS從92降到55，出現M頭型態 → 衰敗的高RS股，可放空
  • 應用三：2022年3月高RS股集體跌破支撐 → 大盤轉壞預警

適用時機：

  • 找領漲股 → 多頭初期和中期
  • 找放空股 → 空頭初期
  • 研究市場 → 任何時候，作為大盤健康度指標

注意事項：

  • RS只是一道濾網，不是買進訊號——還要搭配技術面、基本面、籌碼面
  • 不適合抄底或逆向操作
  • 不適合長投或存股（除非RS持續維持高位）
  • 買進強勢股仍要注意位階和循環位置

5. RS搭配類股輪動：找出下一波主流

是什麼：用RS指標觀察類股指數相對大盤的強弱變化，找出資金正在流入的產業，在類股輪動中搶先卡位。

怎麼做：
1. 計算類股RS：類股指數收盤價 / 加權指數收盤價
2. 繪製RS趨勢圖：觀察RS線是否突破其移動平均線
3. 確認訊號：RS突破均線且差距拉大 + 成交量同步放大 → 該類股可能成為新主流
4. 類股內選股：從轉強的類股中，用個股RS排名找出最強勢的1-3檔股票
5. 多維度確認：技術面（均線多頭排列）+ 基本面（營收成長、毛利率提升）+ 籌碼面（法人買超）

實例：

  • 情境：半導體類股RS突破20日均線，且成交量放大2倍
  • 操作：從半導體類股中找出年RS>80的個股（如台積電、聯發科），再確認法人連續3日買超
  • 進場：回檔不破5日均線時買進

台股類股輪動觀察重點（楚狂人心法）：
1. 板塊>個股：一群同屬性股票大漲比單一個股大漲可靠得多
2. 比大盤強才是真強：大盤漲1%，該類股漲5%以上才算強勢
3. 位階要同步：大盤多頭排列時，類股也需多頭排列才可靠；類股空頭排列但當日大漲，可能只是跌深反彈
4. 低基期+法人買+底部完成：多頭走勢中選還沒漲太多的低基期股，風險低又可能接棒輪漲

適用時機：每週檢視一次類股RS排名，找出正在轉強的產業；盤中即時觀察當日強勢類股

三、類股輪動操作策略整合

6. 類股輪動四步驟SOP

是什麼：結合美林時鐘的宏觀框架和RS指標的微觀篩選，形成完整的類股輪動操作SOP。

怎麼做：

步驟一：判斷經濟週期階段

  • 看GDP成長率方向（↑或↓）
  • 看CPI通膨率方向（↑或↓）
  • 對照美林時鐘四象限，確認當前階段
  • 決定大方向：股票加碼or減碼？債券or現金？

步驟二：鎖定當期強勢產業

  • 用類股RS排名找出RS正在上升的產業
  • 確認類股RS突破均線且量增
  • 交叉比對美林時鐘推薦的行業（如復甦期→科技、電信）

步驟三：類股內選最強個股

  • 用個股年RS篩選（RS>80優先）
  • 確認技術面均線多頭排列
  • 確認基本面營收成長、毛利率提升
  • 確認籌碼面法人買超、大戶持股增加

步驟四：設定進出場

  • 進場：回後買上漲（回檔不破前低和月線，站回5日均線）
  • 停損：跌破進場當天K線低點或5%固定停損
  • 出場：RS開始下降且跌破月線，或遇到壓力未過前高

實例：

  • 2023年4月（復甦期）：GDP ↑、CPI ↓ → 美林時鐘推薦股票
  • 半導體類股RS突破均線且量增 → 鎖定AI半導體族群
  • 類股內年RS最高：緯創(3231)RS=85、廣達(2382)RS=82
  • 緯創技術面均線多頭排列 + 營收YoY+30% + 投信連續買超 → 進場
  • 進場後沿5日均線操作，RS跌破70且跌破月線時出場

適用時機：每月初做一次完整的四步驟檢視，盤中只需追蹤步驟二和三

7. 類股輪動三大常見錯誤

是什麼：操作類股輪動時最容易犯的三個錯誤，每個都可能讓你賺不到輪動的利潤。

怎麼做（避免方法）：

錯誤一：追已經漲一大段的強勢類股

  • 症狀：看到新聞報導某類股大漲才去追，結果買在末端
  • 對策：用RS判斷「正在轉強」而非「已經很強」——RS從50突破到70比RS一直在90更有價值
  • 實例：AI股已經漲3個月才看到新聞追進，結果買在高點

錯誤二：忽略類股位階

  • 症狀：看到當日漲幅最大就認為是主流，但可能是跌深反彈
  • 對策：大盤多頭時，類股也要多頭排列才算真強勢；空頭排列但當日大漲 = 一日行情
  • 實例：某鋼鐵股當日漲7%，但均線空頭排列 → 跌深反彈，隔天就跌回去

錯誤三：只看個股不看類股

  • 症狀：選到弱勢類股中的強勢股，大盤修正時照跌不誤
  • 對策：先選類股、再選個股——板塊力量大於個股力量
  • 實例：金融股整體弱勢時，選了體質最好的金融股也一樣賠錢

適用時機：每次操作類股輪動前都要檢查這三個錯誤

8. 美林時鐘的致命缺陷與修正方法

是什麼：美林時鐘不是萬能的，有幾個致命缺陷必須認知，才能正確使用。

怎麼做：

缺陷一：外部衝擊打亂循環

  • 2008金融海嘯 → 美林時鐘失靈，股債雙殺
  • 2020疫情 → 從復甦直接跳回衰退，然後又因撒錢跳到過熱
  • 修正：外部衝擊時放棄美林時鐘框架，改用「現金+避險」策略

缺陷二：難以判斷轉折點

  • 事後看很清楚，當下很難確認「現在是復甦還是過熱」
  • 修正：不要試圖精確判斷轉折點，而是「漸進調整」——從復甦到過熱時，每月減碼5%股票、加碼5%原物料

缺陷三：2008年後長期失靈

  • QE導致股債雙牛，大宗商品不漲——完全違背美林時鐘預測
  • 修正：美林時鐘在「正常經濟循環」中有效，在「非正常政策干預」下失效——央行QE時要額外考慮流動性因素

缺陷四：價格≠經濟

  • 市場先生的核心觀點：「景氣是景氣，股價是股價」
  • 股價反映預期，往往領先經濟6個月——等你確認復甦期，股市可能已經漲完了
  • 修正：用債券殖利率曲線和股市領先指標輔助判斷，不要只看GDP和CPI

實例：

  • 2009年3月股市觸底反彈，但GDP仍在衰退——股市領先經濟6個月
  • 如果等GDP確認復甦才買股票，已經錯過最大的漲幅

適用時機：使用美林時鐘時永遠要保持懷疑，搭配市場領先指標（殖利率曲線、ISM指數、領先指標綜合指數）做雙重確認

心得

今天學的類股輪動策略，讓我對「如何在對的時間做對的產業」有了完整的框架：

1. 美林時鐘是宏觀框架，不是交易訊號。它告訴你「復甦期要買股票、過熱期要買大宗商品、滯脹期要抱現金」，但最難的是判斷現在到底處於哪個階段。而且外部衝擊會直接打亂整個循環——2008年和2020年都是活生生的例子。所以正確的用法是：用美林時鐘決定大方向（股票vs債券vs現金的比例），但進出場還是要靠技術面和籌碼面。
