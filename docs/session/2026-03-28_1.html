核心知識

一、三大財務報表概覽

是什麼：財務報表是公司向投資人報告經營成果的正式文件，包含三大表：資產負債表、損益表、現金流量表。分季報（每3個月）和年報（第四季總結）。

時間範圍差異：

  • 資產負債表：特定時間點的「快照」（如2026/3/31當天的狀態）
  • 損益表：一段時間內的經營成果（如2026年第一季，共3個月）
  • 現金流量表：一段時間內的現金流入流出

二、資產負債表實戰

是什麼：顯示公司「某一時間點」的財務狀況，公式為：資產 = 負債 + 股東權益

結構解析：

類別流動/非流動代表項目
資產流動資產現金、應收帳款、存貨
資產非流動資產不動產、廠房設備、無形資產
負債流動負債短期借款、應付帳款（一年內還）
負債非流動負債長期借款、公司債（一年以上）
股東權益-股本、資本公積、保留盈餘

怎麼看：

1. 償債能力檢查：

  • 流動比率 = 流動資產 ÷ 流動負債（>2 較安全）

2. 資本結構評估：

  • 負債比率 = 總負債 ÷ 總資產

3. 資產使用效率：

  • 存貨周轉率 = 營業成本 ÷ 平均存貨

實例：

  • 情境：某公司流動資產 500 萬、存貨 200 萬、流動負債 300 萬
  • 流動比率 = 500 ÷ 300 = 1.67（尚可）
  • 速動比率 = (500 - 200) ÷ 300 = 1.0（剛好達標）

適用時機：買股前先看公司是否快破產、負債是否過高、現金是否足夠。

三、損益表實戰

是什麼：顯示公司「一段時間內」賺了多少錢，從營業收入一路扣到稅後淨利。

公式串連：

營業收入
- 營業成本
= 毛利
- 營業費用（銷售、管理、研發）
= 營業利潤
+ 營業外收入
- 營業外支出
= 稅前利潤
- 所得稅
= 稅後淨利

關鍵名詞：

項目定義觀察重點
營業收入賣東西賺的錢成長趨勢
營業成本直接生產成本成本控制
毛利收入 - 成本毛利率是否穩定
營業費用管銷研費用是否有效率
營業外收支非核心業務不要占比太高
稅後淨利最終獲利EPS 計算基礎

怎麼看：

1. 毛利率變化：毛利 ÷ 營業收入

  • 上升：定價能力增強或成本下降

2. 營業費用率：營業費用 ÷ 營業收入

  • 太高要檢查是否管理無效率

3. 業外收支占比：營業外收支 è 稅前利潤

  • >20% 要小心，可能本業不賺錢

實例：

  • 情境：某公司營業收入 1000 萬、營業成本 600 萬、營業費用 200 萬
  • 毛利 = 1000 - 600 = 400 萬，毛利率 = 40%
  • 營業利潤 = 400 - 200 = 200 萬，營業利益率 = 20%

適用時機：評估公司賺錢能力、成長性、獲利品質。

四、現金流量表實戰

是什麼：追蹤「現金是否真的進帳」，因為損益表的獲利可能只是帳面上有、實際沒收到錢。

三大活動：

活動內容健康狀態
營業現金流本業賺到的現金正數且穩定成長
投資現金流買設備、投資其他公司成長期常為負數（正在擴張）
融資現金流借錢、發股票、發股利看公司生命週期

怎麼看：

1. 營業現金流 vs 稅後淨利：

  • 營業現金流 > 稅後淨利：獲利品質好（錢真的進來了）

2. 自由現金流：

  • FCF = 營業現金流 - 資本支出

3. 連續燒錢危機：

  • 如果連續幾季營業現金流為負，公司可能在吃老本

實例：

  • 情境：某公司稅後淨利 1000 萬，營業現金流 600 萬
  • 營業現金流 / 稅後淨利 = 60%
  • 代表有 40%（400 萬）的利潤只是帳面上，錢還沒收到

適用時機：確認公司是否真的收到錢、是否需要一直借錢維持營運。

五、三大報表關係

核心邏輯：
1. 損益表告訴你：公司「說」賺了多少
2. 現金流量表告訴你：公司「真的」拿到多少現金
3. 資產負債表告訴你：這一切之後，公司「剩下」多少家產

閱讀順序建議：
1. 先看損益表：公司有沒有賺錢？成長性如何？
2. 再看現金流量表：錢真的進來了嗎？還是只有帳面數字？
3. 最後看資產負債表：公司會不會倒？財務結構是否健康？

常見警示訊號：

  • 損益表漂亮但現金流量表營業現金流為負 → 可能做假帳或收款有問題
  • 應收帳款持續增加但營業收入沒成長 → 可能有呆帳風險
  • 存貨持續增加但毛利率下降 → 可能滯銷

六、財報閱讀實戰步驟

怎麼做：

1. 選定一家公司：挑一家你有興趣的公司，實際去下載財報

2. 看趨勢不只看單點：

  • 拿近 4-8 季的財報比較

3. 與同業比較：
-毛利率、營業利益率在同業中的位置

  • 是否顯著高於/低於同業？

4. 閱讀附註：

  • 財報後面的附註有寶藏

5. 不要線性外推：

  • 過去 3 年 EPS 成長不代表明年一定成長

七、財報的限制

要注意：
1. 財報是歷史，不是未來：數字反映過去發生的事
2. 不同人的解讀可能不同：同樣的數字，樂觀悲觀看法不同
3. 會計原則可能被調整：折舊年限改變就能影響獲利
4. 大型股財報已被充分分析：很難發現別人不知道的資訊

實例：

  • 情境：某公司過去 3 年 EPS 分別為 1.5、2.0、2.5 元
  • 錯誤思考：明年 EPS 一定 3.0 元
  • 正確思考：要看成長動能是否持續、產業環境是否變化

心得

財報三大表看似各自獨立，其實緊密關聯。損益表是「說故事」，告訴你公司賺多少；現金流量表是「驗證」，確認錢真的進來了；資產負債表是「家底盤點」，告訴你公司值多少。

最實用的閱讀方式是先看損益表確認公司有沒有賺錢，再看現金流量表確認獲利品質，最後用資產負債表確認公司不會倒。這三張表互相勾稽，任何一張有問題都可能是警示訊號。

財報是基本功，能減少「無知帶來的損失」。不用期待看財報能撿到便宜股，但至少能避開地雷股。
