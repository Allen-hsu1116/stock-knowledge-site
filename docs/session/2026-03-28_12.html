MACD 指標基本概念

MACD 是什麼

全名：平滑異同移動平均線（Moving Average Convergence & Divergence）

是由誰提出：1970 年代美國技術分析大師 Gerald Appel 提出

用途：利用快慢線的交錯，判斷股價走勢的轉折，辨識趨勢方向與變化時機

特點：屬於「趨勢跟隨」指標，適合順勢操作

MACD 組成與公式

三大組成元素

元素計算方式說明
**DIF（快線）**EMA(12) - EMA(26)短期與長期 EMA 差值，反應較快
**DEM（慢線）**DIF 的 9 日 EMADIF 的平滑值，反應較慢
**柱狀圖**(DIF - DEM) × 2快線與慢線差值

預設參數 (12, 26, 9)

  • 12：快線 EMA 週期
  • 26：慢線 EMA 週期
  • 9：訊號線 EMA 週期

參數調整建議：

  • 短線交易：(5, 13, 8) 更敏感
  • 長線交易：(24, 52, 18) 更平滑

MACD 四大買賣訊號

1. 黃金交叉「買進」

是什麼：DIF（快線）向上穿越 DEM（慢線），或柱狀圖由負轉正

怎麼做：
1. 確認柱狀圖由負轉正
2. 觀察 DIF 線向上穿過 DEM 線
3. 搭配成交量放大更可靠

實例：

  • 情境：台積電股價從 550 元止跌回升
  • 操作：MACD 柱狀圖由負轉正，DIF 向上穿越 DEM，買進
  • 驗證：成交量同步放大，訊號更可信

適用時機：空頭反轉多頭時，靠近 0 軸的黃金交叉

2. 死亡交叉「賣出」

是什麼：DIF（快線）向下穿越 DEM（慢線），或柱狀圖由正轉負

怎麼做：
1. 確認柱狀圖由正轉負
2. 觀察 DIF 線向下穿過 DEM 線
3. 搭配成交量確認

實例：

  • 情境：聯發科股價從 900 元開始下跌
  • 操作：MACD 柱狀圖由正轉負，DIF 向下穿越 DEM，賣出
  • 驗證：成交量放大，確認趨勢反轉

適用時機：多頭反轉空頭時，靠近 0 軸的死亡交叉

3. 0 軸多空判斷

是什麼：0 軸是多空分界線

怎麼做：

  • MACD 在 0 軸之上：盤面偏多，主要考慮做多
  • MACD 在 0 軸之下：盤面偏空，主要考慮做空

實例：

  • 情境：大盤 MACD 柱狀圖維持在 0 軸之上
  • 判斷：市場多頭情緒占主導，偏多操作
  • 注意：必須維持一段時間才算有效，曇花一現不算

適用時機：判斷大盤或個股多空方向

4. 柱狀體伸長縮短

是什麼：柱狀體長短反映多空力量對比

怎麼做：

  • 0 軸之上，柱體伸長：多頭力量增強
  • 0 軸之上，柱體縮短：多頭力量減弱
  • 0 軸之下，柱體伸長：空頭力量增強
  • 0 軸之下，柱體縮短：空頭力量減弱

實例：

  • 情境：股價上漲但柱體開始縮短
  • 判斷：多頭動能減弱，可能回檔
  • 操作：短線波段考慮出場

適用時機：判斷趨勢動能變化

MACD 鈍化判斷

高檔鈍化

是什麼：股價創新高，但 DIF 線沒有創新高，開始震盪走平

意義：上漲動能減弱，買方力量削減，下跌趨勢可能到來

怎麼做：
1. 觀察股價創波段新高
2. MACD DIF 卻沒有創新高
3. DIF 在高檔區震盪
4. 考慮獲利了結、找出場點

實例：

  • 國產（2504）5 月股價創新高，但 MACD 震盪
  • 形成高檔鈍化
  • 後續股價回檔

低檔鈍化

是什麼：股價創新低，但 DIF 線沒有創新低，開始走平或回升

意義：下跌動能減弱，賣方力量耗盡，回升機率提高

怎麼做：
1. 觀察股價創波段新低
2. MACD DIF 卻沒有創新低
3. DIF 在低檔區震盪
4. 考慮進場做多

實例：

  • 祥碩（5269）股價兩波下跌，MACD 第二次出現鈍化
  • 後續回升速度快，有波段行情

MACD 背離實戰

多頭背離（底背離）

是什麼：股價創新低，但 MACD 沒有創新低，反而向上

意義：空頭行情即將到底，高機率反轉向上

怎麼做：

  • 觀察股價創波段新低
  • DIF 線與 DEM 線反而向上
  • 綠柱狀體開始縮短
  • 視為買進訊號

實例：

  • 倉佑（1568）4 月股價比 2 月低
  • 但 MACD 快線沒有創新低
  • 後續股價逐漸起漲

空頭背離（頂背離）

是什麼：股價創新高，但 MACD 沒有創新高，反而轉下

意義：多頭行情即將到頂，高機率回檔

怎麼做：

  • 觀察股價創波段新高
  • DIF 線與 DEM 線反而轉下
  • 紅柱狀體開始縮短
  • 視為賣出訊號

實例：

  • 玖鼎電力（4588）4 月股價比 3 月高
  • 但 MACD 快線沒有創新高
  • 形成空頭背離，後續股價下跌

柱狀體背離

多頭背離（綠柱）：

  • 股價創新低，綠柱卻縮短
  • 賣壓減弱，股價反轉向上機率高

空頭背離（紅柱）：

  • 股價創新高，紅柱卻縮短
  • 買盤減弱，股價反轉向下機率高

實例：

  • 世紀鋼（9958）股價波段新低，綠柱縮短
  • 產生柱狀體多頭背離，股價止跌回升

MACD 實戰注意事項

滯後性問題

是什麼：MACD 從均線出發，繼承均線的滯後性

影響：

  • 金叉或死叉出現時，股價已經開始運行
  • 信號有點慢，可能錯失部分利潤

解決方法：

  • 搭配其他領先指標（如 KD）
  • 觀察柱狀體邊沿線方向變化

盤整行情假訊號

是什麼：MACD 屬於趨勢型指標，盤整時頻繁交叉

影響：

  • 難以明確指示方向
  • 容易進出場過頻，造成損益不穩

解決方法：

  • 盤整期間降低 MACD 權重
  • 轉用震盪型指標如 RSI 或 KD

遠離 0 軸的交叉要謹慎

怎麼做：

  • 0 軸之上的死亡交叉：可能只是修正，不一定反轉
  • 0 軸之下的黃金交叉：可能只是反彈，不一定反轉
  • 靠近 0 軸的交叉更可信

MACD 優缺點總結

優點

1. 直觀好上手：新手也能快速判斷趨勢
2. 適用多種商品：股票、期貨、外匯、加密貨幣
3. 適用多種週期：日線、週線、小時線皆可
4. 訊號明確：黃金交叉、死亡交叉清楚易懂
5. 功能多元：判斷趨勢、動能、背離

缺點

1. 滯後性：信號比價格慢一拍
2. 盤整失效：震盪盤容易假訊號
3. 參數敏感：不同參數結果差異大
4. 不預測未來：只能追蹤歷史走勢

心得：

MACD 是技術分析中最經典的指標之一，優點是訊號明確、容易理解，但缺點是滯後性明顯。實戰上，我覺得最有價值的是「背離」判斷——當股價創新高但 MACD 沒有，這種動能減弱的訊號比單純的金叉死叉更有參考價值。

另外一個重點是「鈍化」與「背離」的搭配。MACD 高檔背離遇到 KD 高檔鈍化，這種雙重確認可以讓出場時機更精準，不用擔心「跑太慢」。

最後，MACD 在盤整市容易產生假訊號，這時候應該改用震盪型指標，或者乾脆暫停交易。不要硬要在盤整市用趨勢型指標，這是很多人會犯的錯。
