核心知識

一、麥克連震盪指標 McClellan Oscillator（MCL）

是什麼：由 Sherman 和 Marian McClellan 夫婦開發的市場廣度指標（Breadth Indicator），用來判斷整體市場的參與度是否健康。核心邏輯：一個健全的多頭市場，不應該只有少數權值股在撐場面，而應該是大多數個股都在上漲。

計算公式：
1. DIF = 上涨家數 − 下跌家數（每日淨漲跌家數）
2. AD = DIF 的 19 日 EMA（短期廣度動能）
3. BD = DIF 的 39 日 EMA（長期廣度動能）
4. MCL = AD − BD

本質：MCL 就是把「漲跌家數差」當作 MACD 來用——19 日 EMA 是快線、39 日 EMA 是慢線，兩者的差就是振盪值。只不過 MACD 算的是價格，MCL 算的是「市場參與廣度」。

怎麼做：

1. 超買超賣判斷

  • MCL > +25～+35：超買區，短期市場過熱

2. 零線穿越

  • MCL 向上穿越 0 → 短期廣度轉正，市場偏多

3. 極端行情處理

  • MCL > +35：極端多頭行情，不要急著賣！通常還會持續 2～3 週

4. 背離訊號

  • 指數創新高但 MCL 沒創新高 → 市場廣度跟不上，多頭不健康

實例：

  • 台股指數從 20000 漲到 21000 創新高，但 MCL 從 +30 降到 +15
  • 這是「頂背離」：指數靠少數權值股撐，多數個股已經不跟了
  • 操作：開始減碼，不要追高
  • 停損：指數突破 21000 且 MCL 同步創高 → 背離失敗，回補

適用時機：

  • ✅ 判斷大盤趨勢的健康程度（最核心用途）
  • ✅ 搭配 ADL（累積漲跌家數）和指數一起看
  • ⚠️ MCL 是大盤指標，不能用在個股
  • ⚠️ 極端行情中不要逆勢操作

二、麥克連總和指標 McClellan Summation Index（MSI）

是什麼：MSI 是 MCL 的累積版本——把每天的 MCL 值加總起來。MCL 是短線廣度指標，MSI 是中長期廣度指標。

計算方式：

  • MSI（今日）= MSI（昨日）+ MCL（今日）
  • 就是 MCL 的累積和

怎麼做：
1. MSI > 0 → 中長期廣度偏多，市場趨勢向上
2. MSI < 0 → 中長期廣度偏空，市場趨勢向下
3. MSI 超賣區：-1000～-1250 → 中長期底部區
4. MSI 超買區：+1000～+1250 → 中長期頭部區
5. MSI 從負轉正 → 中長期買進訊號
6. MSI 從正轉負 → 中長期賣出訊號

MCL vs MSI 比較：

特性MCL（震盪指標）MSI（總和指標）
週期短期中長期
波動大（像 MACD）小（像累積線）
用途抓短期轉折判斷大趨勢方向
零線穿越短期多空分界中長期多空分界
極端值±25～±35±1000～±1250

實例：

  • 台股 MSI 從 -800 開始回升，穿越 0 → 中長期趨勢由空轉多
  • 同時 MCL 從 -30 反彈到 +15 → 短期廣度也改善
  • MCL + MSI 雙重確認 → 加碼做多
  • 停損：MSI 跌破 0

適用時機：

  • ✅ 判斷大盤中長期趨勢方向
  • ✅ 搭配 MCL 用：MCL 抓短線進出場、MSI 確認大方向
  • ⚠️ 台股適用時需用上市＋上櫃的漲跌家數來計算

三、ARBR 情緒指標

是什麼：ARBR 由兩個指標組成——AR 人氣指標和 BR 意願指標，用來衡量市場的買賣氣勢和情緒狀態。

AR 人氣指標：以「開盤價」為基準，衡量當日從開盤到收盤的買賣力道對比。

計算公式：

  • AR = SUM（最高價 − 開盤價, N）÷ SUM（開盤價 − 最低價, N）× 100
  • N = 26（預設）
  • 分子：開盤後往上推的力道（買方力量）
  • 分母：開盤後往下推的力道（賣方力量）

BR 意願指標：以「昨日收盤價」為基準，衡量今天相對於昨天收盤的買賣意願。

計算公式：

  • BR = SUM（MAX(0, 最高價 − 前日收盤), N）÷ SUM（MAX(0, 前日收盤 − 最低價), N）× 100
  • N = 26（預設）
  • 分子：今天超過昨天收盤的幅度（買方意願）
  • 分母：今天低於昨天收盤的幅度（賣方意願）

怎麼做：

AR 判讀：
1. AR = 100 → 買賣氣勢均衡，持股安全
2. AR 80～120 → 盤整行情，無大波動
3. AR < 50 → 超賣區，可買進
4. AR > 150 → 超買區，注意回檔，賣出時機
5. AR 在頭部和底部常具有領先能力

BR 判讀：
1. BR 70～150 → 正常盤整區
2. BR > 300 → 超買區，股價隨時回檔，賣出時機
3. BR < 50 → 超賣區，股價隨時反彈，買進時機
4. 正常情況下 BR 值 > AR 值

AR + BR 組合判讀：
1. AR 和 BR 同步上升 → 買氣增強，偏多操作
2. AR 和 BR 同步下降 → 賣氣增強，偏空操作
3. BR 急速上升但 AR 小跌或盤整 → 逢高出貨（BR 過熱但 AR 沒跟上，代表只有意願沒有人氣）
4. BR 比 AR 低時 → 可逢低買進（罕見現象，代表意願低於人氣，市場過度悲觀）
5. AR 和 BR 急速下降 → 股價已近高峰，獲利了結

實例：

  • 情境：某股 AR 從 120 降到 45（超賣區），BR 從 200 降到 40（超賣區）
  • 雙重超賣 → 強烈買進訊號
  • 進場後停損：AR 跌破 30 或 BR 跌破 30
  • 目標：AR 回到 80～100（正常區）
  • 情境：某股 AR 在 160（超買），BR 在 350（超買），但 BR 急升而 AR 停滯
  • 代表：買進意願很強但開盤後推升力道不足 → 逢高減碼
  • 這種「BR 熱 AR 冷」的背離是出貨訊號

適用時機：

  • ✅ 判斷市場情緒的極端值
  • ✅ 搭配使用效果最佳（不要單看 AR 或 BR）
  • ⚠️ BR 在除權息時前日收盤價需調整，否則數據失真
  • ⚠️ AR 不反映跳空缺口的強弱，只反映開盤後的日內波動
  • ⚠️ 個股使用時效果不如大盤穩定

四、PSY 心理線指標（Psychological Line）

是什麼：PSY 是最簡單的市場情緒指標，計算 N 天內上漲天數佔比。邏輯：當大多數人都在買（上漲天數多），市場可能過熱；當大多數人都在賣（下跌天數多），市場可能過冷。

計算公式：

  • PSY =（N 日內上漲天數 ÷ N）× 100
  • 常用 N = 12（短線）或 24（中線）
  • 範圍：0～100

怎麼做：

1. 超買超賣判斷

  • PSY > 75（或 80）→ 超買，市場過度樂觀

2. 50 線多空分界

  • PSY > 50 → 多方市場（上漲天數多於下跌天數）

3. 極端值反轉

  • PSY 接近 100 → 幾乎天天漲，極度樂觀，隨時反轉

4. PSY 線型態判讀

  • PSY 本身的圖形型態也有效

實例：

  • 情境：某股過去 12 天有 10 天上漲 → PSY = 10/12 × 100 = 83.3
  • 超買區！市場過度樂觀
  • 操作：不追價，等 PSY 回落到 50 以下再考慮
  • 停損：如果 PSY 持續在 75 以上超過 5 天 → 可能是強趨勢，不要輕易做空
  • 情境：某股過去 24 天只有 4 天上漲 → PSY = 4/24 × 100 = 16.7
  • 超賣區！市場過度悲觀
  • 操作：開始關注，搭配 RSI 或 KD 確認底部
  • 買進時機：PSY 從 20 以下回升突破 25

適用時機：

  • ✅ 判斷市場情緒的極端狀態
  • ✅ 搭配 RSI、MACD 使用效果更好
  • ❌ 不適合單獨使用（只看上漲天數比例，不考慮漲跌幅度）
  • ❌ 高波動市場容易失真（漲 0.1 元也算上漲日）
  • ❌ 強趨勢中 PSY 會長期停在超買或超賣區，不要逆勢操作

五、三大情緒/廣度指標比較

指標類型計算基礎適用最強用途
MCL市場廣度漲跌家數差的 EMA 差大盤判斷指數漲勢是否健康
ARBR市場情緒開盤/收盤價相對高低價個股+大盤判斷買賣氣勢的極端
PSY市場心理上漲天數比例個股+大盤判斷情緒過熱或過冷

組合使用建議：
1. 大盤層級：MCL 判斷廣度健康度 + MSI 判斷中長期方向 + PSY 判斷情緒極端
2. 個股層級：ARBR 判斷買賣氣勢 + PSY 判斷情緒 + KD/RSI 判斷超買超賣
3. 三指標共振：MCL 超買 + ARBR 超買 + PSY 超買 → 三重確認，反轉機率極高

實戰組合實例：

  • 台股 MCL 從 +30 回落到 +10（廣度減弱）
  • ARBR 中 AR 從 150 降到 110（買氣降溫）
  • PSY 從 85 降到 60（情緒冷卻）
  • 三個指標同步降溫 → 大盤短期偏空，減碼操作
  • 停損：MCL 反彈突破前高 + AR 重新站上 120 → 降溫失敗，回補

心得

這回合學了三個「非價格」維度的指標——它們不直接看股價漲跌，而是看市場的「參與度」和「情緒」。這跟之前學的所有技術指標完全不同：KD、RSI、MACD、布林通道都在看「價格的變化」，但 MCL 看「有多少股票在漲」、ARBR 看「開盤後的買賣力道」、PSY 看「漲跌天數比例」。

MCL 最有價值的地方是：它能告訴你指數的漲勢是否「貨真價實」。指數創新高但 MCL 沒跟上 = 只有權值股在撐場面 = 這個新高不健康。這跟之前學的 ADL（累積漲跌家數）概念一脈相承，但 MCL 用 EMA 平滑後更穩定、訊號更清晰。而且 MCL 的「極端行情」概念很有意思——MCL 衝破 ±35 不要逆勢，因為市場像高速行駛的車，不可能瞬間停車，至少還會跑 2～3 週。這個「加速度」的比喻比單純說「超買就賣」精確多了。

ARBR 的設計邏輯很巧妙：AR 以開盤價為基準（開盤後的買賣力量對比），BR 以昨收為基準（今天相對昨天的意願）。兩者的差異在於：AR 反映的是「當日內部」的多空力道，BR 反映的是「相對前日」的買賣意願。最關鍵的組合訊號是「BR 熱 AR 冷」——BR 急升但 AR 停滯 = 想買的人很多但開盤後推不上去 = 有人在出貨。這種背離比單一指標超買超賣更有參考價值。

PSY 是三個裡面最簡單的——數上漲天數而已。但簡單的東西反而容易被忽略。它的價值在於「量化情緒」：12 天內漲了 10 天（PSY = 83）和漲了 6 天（PSY = 50），市場氣氛完全不同。但 PSY 的致命缺陷也最明顯：漲 0.1 元和漲 10 元對 PSY 來說都一樣是「上漲日」，完全不考慮幅度。所以一定要搭配考慮幅度的指標（RSI、MACD）一起用。

最後一個體悟：市場廣度指標和情緒指標，本質上是在回答一個問題——「這波行情，大家都在跟嗎？」如果只有少數股票在漲、買氣集中在開盤瞬間、上漲天數比例看似高但幅度不大，那這個行情就是「虛胖的」。反過來，如果廣度健康、氣勢均衡、情緒尚未過熱，那就是可以安心持有的行情。這三個指標從不同角度回答同一個問題，交叉驗證後的結論才可靠。
