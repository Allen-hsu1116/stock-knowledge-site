重點筆記

均線扣抵值速查

  • 扣抵值：即將被移除的收盤價位置（如 5MA 扣抵 = 5 天前收盤價）
  • 判斷公式：新收盤價 > 扣抵值 → 均線上揚；新收盤價 < 扣抵值 → 均線下彎
  • 蹺蹺板記憶法：左邊（扣抵值）下降 → 右邊（股價）容易上升
  • 支撐壓力預判：扣抵值趨勢向下 → 壓力疲弱；扣抵值趨勢向上 → 壓力強勁
#### 葛蘭碧八大法則速查 **買進訊號**：
  • ① 突破：均線由降轉升，股價突破均線
  • ② 假跌破：跌破均線後隨即回到均線上，均線仍上升
  • ③ 支撐：股價在均線上，回檔未跌破均線就反彈
  • ④ 抄底：股價急跌遠離均線後開始反彈

賣出訊號：

  • ⑤ 跌破：均線由升轉降，股價跌破均線
  • ⑥ 假突破：突破均線後隨即跌破，均線仍下降
  • ⑦ 反壓：股價在均線下，反彈無法突破均線
  • ⑧ 反轉：股價急漲遠離均線後開始下跌

均線糾結操作

  • 判斷：多條均線糾結在一起（相差 0.5% 以內）
  • 買點：帶量突破糾結區 + 均線向上發散
  • 停損：突破 K 棒最低點
  • 諺語：盤久必漲或盤久必跌

多頭/空頭排列

  • 多頭排列：K 線 > 5MA > 10MA > 20MA > 60MA（所有人獲利）
  • 空頭排列：K 線 < 5MA < 10MA < 20MA < 60MA（所有人虧損）
### 核心觀念 1. **扣抵值預判均線方向**：用已知數據預判未來均線走向 2. **葛蘭碧法則系統化買賣**：8 種訊號涵蓋趨勢反轉、回檔、極端乖離 3. **均線糾結抓波段**：糾結越久，發散時力道越強 4. **季線保命法則**：季線（60MA）是中期多空分水嶺，跌破就走 ### 實務應用 今天學的均線進階應用比基礎均線更有實戰價值。扣抵值是最大收穫，以前只知道均線是支撐壓力，現在看扣抵值就能預判均線會上揚還是下彎。葛蘭碧八大法則把均線操作系統化了，特別是「假跌破」和「假突破」這兩個訊號。均線糾結抓波段的方法跟三角收斂很像，都是壓縮後的爆發。最實用的是季線保命法則：跌破季線就要走，這是風險管理的核心。

重點筆記

道氏理論六大法則速查

法則 1：價格反映一切

  • 不用追逐消息面，價格已經反映了
  • 利多不漲 → 賣訊；利空不跌 → 買訊

法則 2：市場有三種趨勢

  • 長期趨勢：1 年以上 → 波段操作
  • 中期趨勢：3 週～3 個月 → 回檔買進
  • 短期趨勢：數小時～3 週 → 當沖、極短線

法則 3：長期趨勢有三階段

  • 承接期：股價溫吞、量縮、主力佈局
  • 大眾參與期：股價急漲、媒體報導、突破追進
  • 出貨期：利多滿天飛、散戶搶進、準備出場

法則 4：指數必須相互驗證

  • 台股：加權 + 電子類股指數 + 金融類股指數
  • 美股：S&P 500 + 道瓊 + NASDAQ + 費半
  • 趨勢背離 → 警訊

法則 5：成交量必須確認趨勢

  • 價漲量增 → 多頭健康
  • 價漲量縮 → 買盤不足
  • 價跌量縮 → 惜售（多頭）
  • 價跌量增 → 殺盤（空頭）

法則 6：趨勢持續直到反轉訊號出現

  • 不要猜頭猜底，順勢操作
  • 高點降低 + 低點降低 → 空頭
  • 高點墊高 + 低點墊高 → 多頭
### 核心觀念 1. **不要猜頭底** → 順勢操作才是王道 2. **成交量說真話** → 價格會騙人，量不會 3. **指數相互驗證** → 單一訊號不可靠 4. **三階段理論** → 當散戶都知道這檔股票好時，通常已經是尾聲 ### 實務應用 道氏理論是技術分析的「基本功」，很多後來的理論（波浪理論、形態學）都是從這裡延伸出來的。最實用的是「三階段理論」判斷股票生命週期，以及「成交量確認趨勢」來過濾假訊號。這些觀念看似簡單，但大部分散戶都做不到——總是想抄底、想賣在最高點、看到新聞就追。道氏理論提醒我們：**尊重趨勢，不要自以為聰明。**

重點筆記

借券賣出餘額基礎

  • 借券餘額 = 借入股票尚未歸還的總量
  • 借券賣出餘額 = 已在市場賣掉的借入股票數量
  • 只有法人機構可以借券，散戶無法使用
  • 外資四大工具：期貨（多空）、選擇權（多空）、現貨（做多）、借券賣出（做空）

四種組合判讀速查

借券賣出現貨解讀訊號強度
增加買超不看好多方（被迫買進）中
增加賣超大空頭（雙手做空）強
減少-潛在軋空力道多方
減少-獲利回補（股價跌時）空方
#### 實戰選股：外資狙擊目標 **篩選條件**： 1. 外資連續 3 日賣超 2. 股價跌破季線 3. 借券賣出連 3 日增加

借券賣出餘額前 20 名：

  • 外資佈局一季以上的標的
  • 增加金額超過 20 億 → 重點關注
  • 這些股票是多單避開、空單可跟的標的

重要注意事項

  • 借券不一定只有放空目的（避險、套利、法規要求）
  • 借券餘額 = 潛在空方力道（借了還沒賣）
  • 借券賣出餘額 = 已實現賣壓（借了已經賣掉）
  • 三指標齊備才準：外資賣超 + 跌破均線 + 借券賣出增加
### 核心觀念 1. **借券賣出是外資的「真心話」**：現貨買賣超可能被動，借券賣出是主動行為 2. **四種組合判讀**：借券賣出增加 + 現貨買超 = 不看好多方（陷阱） 3. **前 20 名觀察**：提前發現外資正在狙擊的股票 4. **搭配其他指標**：借券不一定是純放空，必須搭配外資賣超、技術面破位一起看 ### 實務應用 借券賣出餘額是外資的「真心話」——現貨買賣超可能被動（法規要求），但借券賣出是主動行為。這次學到最重要的概念是「四種組合判讀」，特別是「借券賣出增加 + 現貨買超 = 不看好多方」這個陷阱。外資被迫買現貨時，用借券賣出避險，這種情況代表後市其實不妙。

另外「借券賣出餘額前 20 名」是很實用的工具，可以提前發現外資正在狙擊的股票。但要記得借券不一定是純放空，也可能有避險、套利等目的，所以必須搭配外資賣超、技術面破位一起看才準。

下次看到借券賣出餘額大增，先查外資現貨動向，再確認是否跌破關鍵均線，三個條件齊備才是真正的空方訊號。

涵蓋主題

  • 選擇權未平倉量（OI）定義與賣方視角解讀
  • Call 最大 OI = 壓力，Put 最大 OI = 支撐
  • OI 增減判斷趨勢動能
  • Put/Call Ratio 判讀多空
  • OI 通道實戰應用
  • 倒莊效應與軋空

重點筆記

賣方視角解讀 OI

  • 為什麼看賣方：賣方風險大、資金多、留倉久，解讀方向更可信
  • 資金門資金門檻：賣方要預繳保證金（可能幾千萬到上億）
  • 風險不對稱：買方最大虧損 = 權利金；賣方最大虧損 = 理論上無限
  • 實例：17300 PUT 有 25,781 口 OI → 代表 25,781 口 SELL PUT，莊家認為不會跌破 17300

OI 支撐壓力判斷

  • Call 最大 OI = 壓力區（莊家認為漲不過）
  • Put 最大 OI = 支撐區（莊家認為跌不破）
  • 查詢方式：期交所官網或 optree.tw

Put/Call Ratio 關鍵數字

  • > 100%：Put OI > Call OI，偏多頭
  • < 100%：Call OI > Put OI，偏空頭
  • 接近 115%：強勢多頭
  • 跌破 75%：強勢空頭
#### OI 增減判斷動能
指數未平倉成交量判斷
上漲增加增加強勢軋空
上漲減少增加多頭反轉
下跌增加增加強勢走空
下跌減少增加空頭反轉

倒莊效應

  • 是什麼：價格突破 OI 最大區，賣方被強制停損，造成價格加速突破
  • 突破壓力：Call 賣方倒莊，被迫買回 Call 止損
  • 跌破支撐：Put 賣方倒莊，被迫賣出 Put 止損
### 核心觀念 1. **OI 用賣方視角解視角解讀**：賣方（莊家）敢押的位置是市場共識 2. **Call 最大 OI = 壓力，Put 最大 OI = 支撐**：找到震盪區間 3. **OI 增加 = 趨勢有燃料**：OI 減少 = 燃料耗盡，準備反轉 4. **倒莊 = 大行情**：突破 OI 通道 = 賣方被迫停損 = 加速行情 ### 實務應用 這次學到選擇權未平倉的實戰判讀，核心觀念是「用賣方視角解讀」。賣方（莊家）資金大、風險大、留倉久，他們敢押的位置就是市場共識。

最實用的是「OI 通道」概念：把每日最大 Call OI 和 Put OI 畫成通道，K 線通常在裡面震盪。突破通道 = 倒莊 = 大行情。

這跟之前學的「借券賣出餘額」互補：

  • 借券賣出餘額看的是外資做空動向
  • OI 看的是整體市場（包括散戶和法人）的支撐壓力共識

兩個一起看，更能判斷外資和莊家的真實意圖。

重點筆記

趨勢線畫法核心原則

上升趨勢線：

  • 連接上升趨勢中至少 2 個低點
  • 向右延伸，確認線沒有穿破任何 K 線實體
  • 買點：股價回測趨勢線時買入，停損設在趨勢線下方 3%

下降趨勢線：

  • 連接下降趨勢中至少 2 個高點
  • 向右延伸，確認線沒有穿破任何 K 線實體
  • 賣點：股價反彈到趨勢線時賣出或做空

趨勢線畫法 7 大注意事項

1. 一致性原則：同一條線要麼都用實體高點，要麼都用影線高點
2. 不穿破原則：趨勢線不能穿破任何 K 線實體
3. 斜度適中：建議 30-70 度之間，太陡易失效，太平力道不足
4. 趨勢確認：畫上升趨勢線要確認高點也在上移
5. 不必從最高/最低點開始：可以從趨勢中段開始畫
6. 優先選擇驗證次數最多的線：被價格測試次數越多越可靠
7. 配合 K 線型態進場：趨勢線只是告訴你位置，進場要配合 K 線反轉型態

支撐壓力有力程度

支撐壓力類型有力程度原因
多根 K 棒連成⭐⭐⭐⭐⭐多次測試，市場共識強
跳空缺口⭐⭐⭐⭐天然斷層，心理關卡
兩根 K 棒⭐⭐力道較弱，容易假突破

突破確認四大原則

1. 收盤價突破才有效：盤中突破不算，要收盤價站上趨勢線
2. 突破 3% 以上：收盤價要突破趨勢線 3% 以上，避免雜訊
3. 突破要伴隨大量：成交量要超過 5 日均量 30% 以上
4. 突破後要觀察：突破後觀察 1-3 天，確認股價朝預期方向發展

支撐壓力互換角色

  • 支撐跌破變壓力：支撐區有買方成本，跌破後買方被套牢，反彈時會想賣出解套
  • 壓力突破變支撐：壓力區有賣方成本，突破後賣方會停損，回檔時空方回補形成支撐

趨勢通道實戰

  • 上升通道操作：下軌位置進場多單，上軌位置考慮出場
  • 下降通道操作：上軌位置進場空單，下軌位置考慮出場
  • 通道突破處理：使用 123 法則確認趨勢反轉，或重新畫更陡的平行線

核心觀念

1. 畫線有規矩：不能亂畫，要遵守 7 大原則，尤其是「不穿破 K 線實體」
2. 越多次測試越可靠：被測試越多的線越有力，盤整越久的突破越猛
3. 突破要確認：收盤價 + 3% + 大量，三個條件都要滿足
4. 角色互換要記住：支撐跌破變壓力，壓力突破變支撐
5. 線要跟著市場變：趨勢線被突破就要重畫，沒什麼好糾結的

實務應用

趨勢線和支撐壓力線是技術分析的基礎中的基礎，但也是最實用的工具。很多人嫌它簡單就忽略了，結果畫了半天都畫錯。

這次學到最重要的是「畫線 7 大原則」，特別是「不穿破 K 線實體」和「斜度適中」這兩點。以前以為趨勢線隨便畫畫就好，現在看來是有嚴格規範的。

「突破四大原則」也很實用：收盤價 + 3% + 大量 + 觀察。這四個條件可以有效過濾假突破，避免被市場誘騙。

最後是「支撐壓力互換」這個概念，以前只知道「壓力突破變支撐」，但不知道原因。現在理解了：是因為套牢盤和停損盤的心理因素。這個理解讓我對市場心理更有感覺。

第 7 回合學習 - 趨勢線畫法實戰

學習時間：2026-04-05 08:09

文章來源

  • 趨勢線｜趨勢線怎麼畫？上升（下降）趨勢線是壓力與支撐？
  • 趨勢線構成的趨勢通道的交易策略
  • 上升/下降趨勢線怎麼畫？趨勢線畫法及app軟體推薦

涵蓋主題

  • 趨勢線基礎概念（支撐與壓力判斷）
  • 上升趨勢線與下降趨勢線畫法
  • 畫趨勢線六大注意事項
  • 趨勢通道構成與實戰操作
  • 趨勢線實戰應用（買賣點判斷、變盤信號）
  • 趨勢線突破處理（假突破判斷）

重點筆記

趨勢線基礎概念

  • 上升趨勢線：連接低點，判斷支撐
  • 下降趨勢線：連接高點，判斷壓力
  • 核心邏輯：低點墊高 + 高點墊高 = 多頭；高點降低 + 低點降低 = 空頭

畫趨勢線六大注意事項

1. 實體與影線要一致：不能混用實體和影線
2. 不能穿破 K 線實體：穿過任何 K 線實體就無效
3. 斜度建議 30-70 度：太陡易突破，太平力道不足
4. 趨勢線表示趨勢，不能確認趨勢：要搭配其他指標
5. 畫上升趨勢線要配合高點上移：低點墊高 + 高點墊高才對
6. 用驗證次數最多的那條：被測試越多越可靠

趨勢通道實戰

上升通道：

  • 先畫上升趨勢線（連接低點）
  • 再畫平行線穿過高點
  • 下軌買入，上軌賣出

下降通道：

  • 先畫下降趨勢線（連接高點）
  • 再畫平行線穿過低點
  • 上軌做空，下軌出場

趨勢線突破處理

假突破判斷：

  • 突破後 3 天內無法站穩 = 假突破
  • 突破時要放量才可靠
  • 收盤價突破比盤中突破可靠

變盤信號：

  • 空轉多：突破下降趨勢線 + 回調確認 + 原壓力變支撐
  • 多轉空：跌破上升趨勢線 + 反彈確認 + 原支撐變壓力

核心觀念

1. 畫線有規矩：六大原則要遵守，特別是「不穿破實體」和「斜度適中」
2. 通道比單線好用：明確的進出場點位，上軌賣、下軌買
3. 高點低點要搭配：畫上升趨勢線時高點也要墊高才對
4. 變盤要等確認：突破後要等回調，原壓力變支撐才是真變盤
5. 不要單獨使用：搭配成交量、K 線型態一起看才可靠

實務應用

趨勢線看起來簡單，但這些細節做好了才能真正發揮作用。最關鍵的是：

1. 不要混用實體和影線：新手常犯錯誤，一條線要麼都用實體，要麼都用影線
2. 斜度很重要：太陡很快被突破，太平支撐壓力效果差，30-70 度最實用
3. 通道比單線好用：可以給出明確進出場點位
4. 變盤要等確認：突破後要等回調確認，原壓力變支撐才是真的變盤
5. 不要單獨使用：要搭配其他指標確認

畫趨勢線不是畫爽的，是要能在實戰中找到進出場點位的。

第 8 回合學習 - 多時間框架分析實戰

學習時間：2026-04-05 09:09

文章來源

  • 多重時間框架掌握順勢交易 - 玉山證券
  • 技術分析基礎教學_多重時間框架 - 柴豬交易群

涵蓋主題

  • 多時間框架定義與核心概念
  • 三重時間框架分析法
  • 不同交易類型的時間框架選擇
  • 多時間框架的優點（拉高盈虧比）
  • 多時間框架的缺點與陷阱
  • 「長線保護短線」實戰操作
  • 多時間框架與單一週期的選擇

重點筆記

多時間框架定義

  • 是什麼：透過多種時間週期來辨識股價行情趨勢的一種交易技術
  • 核心概念：「長線保護短線」——長期趨勢難以扭轉，若長期趨勢為多方架構，短週期操作應以偏多思考為核心

三重時間框架分析法

  • 框架結構：大週期看位置 → 中週期判斷趨勢 → 小週期尋找入場位置
  • 常用組合：
  • 實例（茂林-KY 4935）：
#### 不同交易類型的時間框架選擇
交易類型主週期（找點位）輔助週期（看趨勢）
波段操作日線週線、月線
日內交易1小時4小時、日線
當沖操作5分鐘15分、30分

多時間框架的優點

  • 拉高盈虧比：用小週期進場、小週期止損，目標是大週期的趨勢延續，盈虧比可達 5 倍以上
  • 實例（外匯交易）：15倍盈虧比，1%風險可獲利 15%
  • 更宏觀分析市場：避免單一週期誤判趨勢方向

多時間框架的缺點與陷阱

  • 常見錯誤：大中週期判斷不準 → 連續止損；小週期雜訊太多 → 誤導進場時機
  • 怎麼避免：

1. 不使用相距太大的時間框架（如月線配 5 分鐘）
2. 新手用標準三重框架回測
3. 確認三個週期趨勢一致才進場

  • 時間框架相悖處理：大週期多、中週期空 → 等中週期轉多再進場

「長線保護短線」實戰操作

  • 步驟：月線 KD 黃金交叉 → 週線 KD 黃金交叉 → 日線 KD 黃金交叉進場
  • 關鍵判斷：

核心觀念

1. 大週期定方向，小週期找點位：這是多時間框架的核心精神
2. 時間框架不要跨太遠：月線配 5 分鐘會精神分裂，標準做法是相鄰時間框架
3. 三個週期要同向：大週期多、中週期空，不要急著進場，等中週期也轉多
4. 盈虧比是最大優勢：用小週期止損、吃大週期波段，盈虧比隨便都 5 倍起跳
5. 雜訊是最大敵人：小週期雜訊多，確認中週期結構完整再切小週期找精確進場點

實務應用

多時間框架這招其實就是「不要只看一個視窗」的概念。很多人只盯著日線做波段，結果被週線的回調洗出場；或者只看 15 分鐘做當沖，卻沒發現 4 小時已經到壓力區。

這招的核心精神很好理解：大週期定方向，小週期找點位。但實戰上有幾個坑要注意：

1. 時間框架不要跨太遠：月線配 5 分鐘這種組合會讓你精神分裂，因為兩個週期的趨勢方向可能完全不同。標準做法是相鄰時間框架，比如日/4H/1H 或 1H/15m/5m。

2. 三個週期要同向：如果大週期多、中週期空，不要急著進場。等中週期也轉多再說。這就是「長線保護短線」的真正意思——不是說你可以逆勢做短線，而是要等小週期跟上大週期的方向。

3. 盈虧比是這招最大優勢：用小週期止損、吃大週期波段，盈虧比隨便都 5 倍起跳。柴豬那個例子更誇張，15 倍到 30 倍。前提是你要耐得住性子等大週期的趨勢延續。

4. 雜訊是最大敵人：小週期雜訊多，容易被假突破洗掉。解決方法是確認中週期結構完整（比如回踩起漲點、供需區明確），再切小週期找精確進場點。

總結一句話：多時間框架讓你「看得遠、進得準、出得巧」，但前提是你要耐得住性子等三個週期同向。

第 9 回合學習 - DCF 股利折現模型實戰

學習時間：2026-04-05 10:09

文章來源

  • 股利折現模型是什麼？公式如何計算？
  • 現金流量折現法是什麼？如何使用 DCF 來做股票估值

涵蓋主題

  • 股利折現模型（DDM）核心概念
  • 高登成長模型公式
  • DCF 現金流量折現法
  • DDM vs DCF 差異比較
  • 安全邊際實戰應用
  • 價值陷阱警示

重點筆記

DDM 高登成長模型

公式：V₀ = D₁ ÷ (r - g)

  • V₀：股票合理價（內在價值）
  • D₁：明年預估股利
  • r：折現率（預期報酬率，用 CAPM 計算）
  • g：股利成長率

實例：明年股利 3 元，折現率 8%，成長率 5%
內在價值 = 3 ÷ (8% - 5%) = 3 ÷ 0.03 = 100 元

如果現在股價 110 元 → 高估 10 元，不買。

DDM 使用限制

只適合：

  • 股利發放穩定的成熟企業
  • 獲利可預測的公用事業、銀行、電信股

不適合：

  • 不發股利的成長股（Google、Amazon）
  • 股利忽高忽低的景氣循環股
  • 用股票回購代替配息的美股

DCF 現金流量折現法

公式：DCF = Σ [CFₙ ÷ (1+r)ⁿ]

計算步驟：
1. 預測自由現金流（5-15 年）
2. 計算終值（第 10 期後到永續）
3. 計算折現率（WACC）
4. 加總淨現值

實例：10 年每年現金流 10 元，終值 125 元，折現率 10%
DCF = 10÷1.1 + 10÷1.1² + ... + 10÷1.1¹⁰ + 125÷1.1¹⁰ = 109.6 元

代表用 109.6 元買進，年化報酬率 = 10%。

DDM vs DCF 比較

項目DDMDCF
現金流來源股利自由現金流
適用範圍只適合穩定配息股適用所有公司
計算難度較簡單較複雜
保守程度更保守較中性

關鍵理解：DDM 是 DCF 的一種特例，把「股利」當現金流。

安全邊際實戰

公式：買進價 = 內在價值 × (1 - 安全邊際)

實例：DCF 算出 100 元，安全邊際 20%
買進價 = 100 × 0.8 = 80 元

為什麼要打折？估價可能錯誤 + 未來不確定性 + 犯錯空間。

價值陷阱警示

什麼是價值陷阱：股價大跌 + 獲利能力也大跌
DCF 算出來「便宜」，但其實是衰退股。

避開方法：

  • 檢查配息率是否穩定
  • 確認獲利能力沒有結構性衰退
  • 觀察產業競爭態勢

核心觀念

1. 算出來的數字都是假的：巴菲特從不動手算 DCF
2. DCF 是思考工具不是計算工具：重點是理解現金流結構
3. 安全邊際是投資的保險：算出來的價值要打折再買
4. 保守估計成長率：不要線性外推高成長率 forever
5. DDM 適合存股，DCF 適合深入了解：不同場景用不同工具

實務應用

DCF 和 DDM 是估值方法中理論最完整的，但「算出來的數字都是假的」。巴菲特說：「我從來沒見他動手算過。」——因為真正好的投資機會，用什麼方法算都是好機會。

重點不是算出一個精確數字，而是透過這個過程：
1. 理解公司的現金流結構
2. 思考未來的成長性和風險
3. 建立「時間就是金錢」的直覺

實戰上，把 DCF 當作「思考工具」而非「計算工具」。對於穩定配息的存股標的（如中華電、台塑），用 DDM 快速估一下合理價；對於想深入了解的公司，才會花時間做完整 DCF。

最重要的是：算出來的價值要打折再買。安全邊際是投資的保險。

第 10 回合學習 - 高殖利率選股策略實戰

學習時間：2026-04-05 11:09

文章來源

  • 策略選股工具實戰－高殖利率選股策略
  • 【股市入門】殖利率是什麼？股息殖利率怎麼算？高殖利率股票有哪些？

涵蓋主題

  • 殖利率基礎定義與計算
  • 高殖利率三大陷阱
  • 四種高殖利率選股策略
  • 進出場時機判斷

重點筆記

殖利率計算公式

是什麼：股息與股價的比率，反映持有股票可獲得的現金回報率。

怎麼算：

現金股息殖利率（%）＝ 現金股息 ÷ 股價
股票股利殖利率（%）＝ 股票股息 ÷ 股價
總殖利率（%）＝ 現金殖利率 + 股票殖利率

實例：

  • 台積電（2330）年配息 11 元，年均價 516 元
  • 殖利率 = 11 ÷ 516 = 2.13%
  • 投資一張（51.6 萬），年領現金股息 1.1 萬

高殖利率三大陷阱

陷阱 1：股利發放不穩定

  • 問題：股利忽高忽低，難以預測收入
  • 識別方法：查詢近 5 年配息紀錄，避開營建、航運等景氣循環股

陷阱 2：賺股息賠價差

  • 問題：殖利率高是因為股價下跌，不是股息增加
  • 判斷方式：檢查股價下跌原因，是否基本面惡化

陷阱 3：配息來自一次性收益

  • 問題：出售子公司、不動產等一次性獲利，明年沒有
  • 識別方法：查看財報「營業外收支」項目

四種高殖利率選股策略

策略 1：高現金股利連續發放股

  • 篩選條件：現金股利近 5 年平均 > 3 元、連續 3 年增加、股利發放率 > 70%
  • 適合：追求穩定收益的存股族

策略 2：高殖利率快速填息股

  • 篩選條件：現金殖利率近 3 年平均 > 5%、填息成功率 100%
  • 適合：退休族群、以股利收入為主的投資人

策略 3：價值低估高殖利率股

  • 篩選條件：現金殖利率 > 5%、股價淨值比 < 1.5 倍、本益比 < 15 倍
  • 適合：價值型投資人

策略 4：成長動能高股息股

  • 篩選條件：現金股利連續 3 年增加、EPS 年增率 > 10%、營業利益成長率 > 5%
  • 適合：能承受股價較大波動的投資人

進出場時機

進場時機：
1. 估值合理時：殖利率高於市場/產業平均
2. 除息後走勢回穩
3. 市場低迷時分批
4. 定期定額買入
5. 股利政策變化前

出場時機：
1. 殖利率下降或公司調降股利
2. 基本面惡化：EPS 下滑、自由現金流轉負
3. 填息完成後
4. 達到收益目標
5. 替代性投資更優
