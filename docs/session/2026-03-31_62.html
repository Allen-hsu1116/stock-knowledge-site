核心知識

一、股利政策四大類型

1. 剩餘股利政策（Residual Dividend Approach）

是什麼：公司優先滿足投資需求，有剩餘才發股利。

怎麼做：
1. 確定公司的最佳資本結構
2. 確定下一年度的資金需求量
3. 按最佳資本結構，計算需增加的股東權益數額
4. 稅後利潤先滿足資金需求，剩餘才發現金股利

實例：

  • 情境：台積電每年需大量資本支出擴廠研發
  • 操作：2022年 EPS 29.4元，配息11元，盈餘分配率約37%
  • 判斷：盈餘分配率低是因為錢要投入下一輪資本支出

適用時機：成長期公司、資本密集產業
不適用時機：想吸引存股族的公司、成熟穩定產業

2. 穩定股利額政策（Constant Dividend）

是什麼：每年發放固定金額的股利，不隨盈餘波動。

怎麼做：
1. 設定一個可持續發放的固定股利金額
2. 不論當年獲利好壞，維持相同配息
3. 預留安全邊際，設定不宜過高

實例：

  • 情境：兆豐金近五年穩定配息
  • 操作：2022年 EPS 1.89元，配息1.4元，盈餘分配率74%
  • 判斷：金融業成熟穩定，配息率接近八成

適用時機：成熟產業、公用事業、金融業
優點：

  • 給市場穩定訊息，降低不確定性
  • 吸引追求穩定現金流的長期投資人

注意：設定金額不能太高，避免獲利不佳時無力支付

3. 固定股利率政策

是什麼：每年按固定比例從稅後利潤支付股利。

怎麼做：
1. 設定固定配息率（如60%、80%）
2. 每年按該比例計算股利金額
3. 股利金額隨獲利波動

實例：

  • 情境：某公司設定配息率70%
  • 操作：EPS 10元時配7元，EPS 5元時配3.5元
  • 判斷：金額波動大，市場可能解讀為不穩定

適用時機：極少公司採用
缺點：股利波動大，傳遞不穩定訊息給市場

4. 正常股利加額外股利政策

是什麼：固定發放一筆「正常股利」，獲利佳時再加發「額外股利」。

怎麼做：
1. 設定一個固定的「正常股利」金額
2. 評估當年度獲利狀況
3. 獲利佳、資金充裕時加發額外股利

實例：

  • 情境：某公司設定正常股利2元
  • 操作：正常年度配2元，獲利佳年度配2元+1元額外股利
  • 判斷：給投資人保底收入，景氣好還有驚喜

適用時機：獲利波動大、有保留盈餘可運用的公司
優點：平衡穩定與彈性

二、股利政策六大影響因素

1. 契約約束

是什麼：借款合約可能限制公司發放股利。

怎麼判斷：

  • 查看長期借款合約條款
  • 注意是否有「償還借款前不得發放股利」等限制
  • 注意是否要求維持一定流動比率

實例：公司有銀行借款，合約規定流動比率需維持1.5以上，大量配息可能影響比率。

2. 法律約束

是什麼：公司法規定的利潤分配順序、資本充足性要求。

怎麼判斷：

  • 公司需先彌補虧損、提撥法定盈餘公積
  • 股利分配來自保留盈餘，不能動用資本
  • 台股限制：配息來源限於保留盈餘

3. 現金充裕性約束

是什麼：發現金股利必須有足夠現金。

怎麼判斷：

  • 檢查現金流量表
  • 營業現金流是否穩定正值
  • 現金部位是否充足
  • 注意：EPS高不代表有現金（可能有大量應收帳款）

實例：某公司EPS 10元，但應收帳款占8元，實際現金只有2元，無法配息8元。

4. 投資機會

是什麼：投資機會多的公司倾向保留盈餘再投資。

怎麼判斷：

  • 公司是否處於成長期？
  • 是否有擴廠、研發、併購計畫？
  • 資本支出佔營收比例是否逐年增加？

實例：台積電每年資本支出數千億，盈餘分配率自然較低。

5. 資本成本與償債能力

是什麼：配息影響公司資本結構和償債能力。

怎麼判斷：

  • 配息後負債比率是否上升？
  • 流動比率、速動比率是否維持安全水準？
  • 公司是否有足夠儲備應對景氣波動？

6. 信息傳遞

是什麼：股利政策傳遞公司前景訊息給市場。

怎麼判斷：

  • 增加配息：公司對未來有信心
  • 減少配息：可能獲利下滑或需保留資金
  • 穩定配息：營運穩定、財務健康

注意：市場會解讀股利變化，需考慮投資人反應。

三、盈餘分配率（配息率）

公式

盈餘分配率 = 每股股利 ÷ 每股盈餘（EPS）× 100%

與殖利率的差異：

  • 盈餘分配率：看公司賺多少、配多少
  • 殖利率：看股價、配多少
公式分子分母
盈餘分配率每股股利EPS
殖利率每股股利股價

如何判斷盈餘分配率

高配息率（70-90%）：

  • 代表公司進入成熟期
  • 不需要大量資本支出
  • 適合存股族

低配息率（30-50%）：

  • 公司處於成長期
  • 需要保留盈餘再投資
  • 適合成長型投資人

異常高配息率（>100%）：

  • 可能動用過去保留盈餘
  • 獲利衰退但維持配息
  • 需注意是否可持續

心得

股利政策不是「配越多越好」這麼簡單。這次學習最大的收穫是理解了「配息率高低反映公司發展階段」這個概念。

成長期的公司配息率低是合理的——台積電需要持續投入資本支出才能維持領先地位，把錢配掉反而害了公司。成熟期的公司配息率高也是合理的——金融業、公用事業已經不需要大量投資，把錢還給股東才是對的。

投資人要問自己的是：我要的是現金流還是資本增值？ 要現金流就找配息率60%以上的成熟公司；要資本增值就找配息率低但成長潛力大的公司。兩者都要？那是在做夢。

另外，盈餘分配率 > 100% 是警訊，代表公司可能動用過去保留盈餘或在獲利衰退時硬配，這種配息不可持續，追高殖利率會被套。
