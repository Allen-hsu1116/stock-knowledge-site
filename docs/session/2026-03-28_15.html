成交量基本概念

是什麼：成交量指某一時段內具體的交易數量，是判斷股市走勢的重要指標。

怎麼看：

  • 個股成交量：台股以「張」為單位（1張=1000股）
  • 大盤成交量：以「成交金額」為主
  • 期貨成交量：以「口」為單位

重要認知：成交量的背後，同時有一組買方與一組賣方。因此：

  • 成交量大 = 買方多，賣方也多（多空看法嚴重分歧）
  • 成交量小 = 買方少，賣方也少（多空看法一致）

流動性風險：

  • 60日平均成交金額低於100萬的標的要小心
  • 成交量過低可能導致想賣賣不掉，或被迫賣在差價位

九種量價關係完整解析

一、趨勢盤的量價關係（正常現象）

1. 價漲量增 — 多頭健康訊號

是什麼：股價上漲，成交量同時增加

多頭情境：

  • 價格突破壓力區，伴隨成交量增加
  • 投資人積極追價，買盤氣勢增
  • 後勢看好，漲勢健康

實例：

  • 情境：股價從50元突破60元壓力區，成交量從500張增至800張
  • 操作：確認突破後可追進，符合多頭趨勢

注意事項：

  • 在相對高點出現「異常大量」+「長黑K棒」+「跌破短期均線」= 主力出貨訊號
  • 可能是獲利了結，趨勢反轉

2. 價跌量縮 — 多頭正常回檔

是什麼：股價下跌，成交量同時減少

多頭情境：

  • 正常回檔，賣壓不大
  • 可能是主力洗盤（測試支撐）
  • 回檔幅度小 + 成交量縮減 = 後續上漲機率高

實例：

  • 情境：股價從100元回檔至95元，成交量從1000張縮減至400張
  • 操作：量縮回檔時買進，不易套牢

注意事項：

  • 若回檔「幅度很大」+ 成交量縮減 = 可能沒有買盤接手
  • 需確認型態結構是否被破壞

二、量價背離（異常訊號）

3. 價漲量縮 — 兩種結果

是什麼：股價上漲，成交量減少

情境A：量價背離（空頭訊號）

  • 價格上漲過程中，成交量逐漸縮小
  • 紅K越來越弱，追價資金減少
  • 後續可能下跌或進入盤整

實例：

  • 情境：股價從80元漲至100元，但成交量從1000張縮減至300張
  • 操作：高檔出現此現象，應減碼觀望

情境B：量縮上漲（多頭訊號）

  • 價格上漲，成交量縮小，但K線強勢
  • 成因：賣方不足，想買買不到
  • 特徵：主力高度控盤、籌碼穩定

實例：

  • 情境：股價漲停鎖死，成交量反而萎縮
  • 判讀：持有者不想賣出，後續看漲

周線/月線應用：

  • 周線或月線的「量縮上漲」是相當棒的量價關係
  • 代表資金雄厚的主力長期鎖住籌碼
  • 出現「窒息量」後容易加速上漲

4. 價跌量增 — 轉折訊號

是什麼：股價下跌，成交量增加

情境A：高檔出貨（空頭訊號）

  • 高檔震盪區間出現「價跌量增」
  • 多頭獲利了結出場
  • 常是波段跌勢的開始

實例：

  • 情境：股價在120元高檔區，突然下跌至115元，成交量爆增至2000張
  • 操作：主力出貨訊號，應出場觀望

情境B：底部承接（多頭訊號）

  • 連續下跌後出現「價跌量增」
  • 有人進場承接，行情可能距離底部不遠
  • 可能是反轉向上訊號

實例：

  • 情境：股價從50元跌至30元，連續下跌後，某日下跌至28元但成交量放大至1500張
  • 判讀：買盤進場承接，可關注反轉機會

三、盤整盤的量價關係

5. 價平量增 — 籌碼收集或出貨

是什麼：股價持平，成交量突然增加

低檔情境：

  • 有人在收購籌碼
  • 待主力表態後，有望展開上漲

實例：

  • 情境：股價維持在50-52元區間，某日成交量從300張增至800張
  • 判讀：主力收購，可關注後續突破

高檔情境：

  • 盤整區間出現多根上影線
  • 成交量明顯放大
  • 主力偷偷出貨，最終可能下跌

6. 價平量縮 — 籌碼沉澱

是什麼：股價持平，成交量萎縮

情境：

  • 連續急漲後的整理
  • 籌碼混亂，主力需花時間沉澱
  • 等待沒信心的散戶離場後，才有機會進入下一波上漲

實例：

  • 情境：股價從80元急漲至100元後，在95-100元區間盤整，成交量從1000張縮減至200張
  • 判讀：籌碼沉澱期，待量縮結束後關注突破

7. 價漲量平 — 動能不足

是什麼：股價上漲，成交量持平

情境：

  • 趨勢向上，但動能不見得能長期持續
  • 容易進入盤整期

8. 價跌量平 — 趨勢不明確

是什麼：股價下跌，成交量持平

情境：

  • 有一定程度的持續換手
  • 趨勢陷入不明確的盤整

9. 價平量平 — 觀望氣氛

是什麼：股價與成交量都持平

情境：

  • 買賣雙方處於觀望氣氛
  • 不確定性高

實戰操作要點

買進時機

最佳買點：量縮回檔

  • 多頭趨勢中，量縮對應股價拉回位置
  • 低買高賣，在量縮時買進不易套牢
  • 若在量增時買進，剛好遇到量縮拉回會套牢

確認條件：

  • 價漲量增（主升段）→ 價跌量縮（洗盤）→ 確認支撐後買進
  • 回檔幅度小、成交量縮減 = 賣壓不大

出場時機

出場訊號：價跌量增（高檔）

  • 高檔震盪或漲勢末端出現「價跌量增」
  • 多頭獲利了結，波段跌勢開始

出場訊號：價漲量縮（高檔背離）

  • 高檔出現價漲量縮
  • 追價資金減少，趨勢可能反轉

注意假訊號：

  • 頭部有時會用「價跌量縮」結束
  • 讓散戶以為是短線洗盤，結果套牢

常見迷思與陷阱

迷思一：成交量大代表很多人買

錯誤：成交量大 = 買方多 + 賣方也多

正確解讀：

  • 多空看法嚴重分歧
  • 籌碼混亂
  • 需搭配價格趨勢判斷

迷思二：量縮上漲一定是壞事

錯誤：價漲量縮不一定是背離

正確解讀：

  • 需看K線強度
  • 漲停鎖死的量縮上漲是多頭訊號
  • 周線/月線量縮上漲更是主力控盤特徵

陷阱：頭部價跌量縮

現象：

  • 頭部有時用「價跌量縮」結束
  • 散戶誤以為是洗盤，結果套牢

辨別方法：

  • 搭配其他技術指標（K線型態、均線支撐）
  • 確認型態結構是否被破壞
  • 觀察是否有跌破關鍵支撐

主力炒股模型中的量價關係

以主力炒股模型 A～H 階段為例：

  • A～B 階段：底部翻轉向上，「價漲量增」
  • C～E 階段：洗盤區，呈現「價平量縮」或「價跌量縮」
  • F～G 階段：主升段上漲，「價漲量增」
  • G 階段：末升段可能「價漲量增」
  • H 階段：頭部出貨，容易「價跌量增」

心得

「量比價先行」這句話確實有道理。量價關係就像市場的「體溫計」——能看出資金是在進場還是退場。

最實用的三個判斷：
1. 量縮回檔是買點：多頭趨勢中，量縮對應的是洗盤回調
2. 高檔價跌量增要注意：主力可能在出貨
3. 窒息量後可能加速：主力鎖碼、量縮上漲是強勢訊號

不過量價關係只是參考，100%準確的指標不存在。還要搭配：

  • K線型態
  • 均線支撐壓力
  • 籌碼面（法人、主力動向）
  • 整體市場趨勢

新手看價、老手看量，這句不是說量比價重要，而是說老手會深入一層去看資金流向。成交量是市場的「熱度計」，但解讀要看趨勢位階——同樣的量價關係，在低檔和高檔意義完全不同。
