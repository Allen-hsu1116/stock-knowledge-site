一、合併報表 vs 個體報表 — 你看的是誰的數字？

是什麼

上市櫃公司通常不是一家公司單打獨鬥，而是一個集團——母公司加上一堆子公司。財報因此有兩種版本：

  • 合併財務報表（Consolidated）：把母公司和所有子公司的資產、負債、收入、費用全部加總，當作一個經濟體來呈現。集團內部交易會被消除，不會左手賣右手虛增營收。
  • 個體財務報表（Parent company only）：只看母公司自己的數字，不含子公司。母公司對子公司的投資用「權益法」認列（只認持股比例的損益）。

為什麼重要：IFRS上路後，上市櫃公司每季只公布合併財報，個體財報一年才編一次。投資人日常看的是合併報表，但合併數字可能「虛胖」。

怎麼做：兩個關鍵指標辨識虛胖

指標一：母公司營收佔合併營收比重
> 母公司營收 ÷ 合併營收 × 100%

  • 比重越高（>80%）→ 公司主要靠本業賺錢
  • 比重越低（<50%）→ 公司靠轉投資子公司賺錢，營收數字可能灌水

指標二：歸屬母公司稅後純益佔總稅後純益比重
> 歸屬母公司稅後純益 ÷ 合併稅後純益 × 100%

  • 比重越高（>90%）→ 利潤多數進母公司口袋
  • 比重越低 → 利潤被非控制權益（少數股東）瓜分

實例：

  • 台積電(2330)：母公司營收佔比 98.34%，歸屬母公司純益佔比 99.79% → 數字扎實，本業為主
  • 某控股公司：母公司營收佔比 20%，歸屬母公司純益佔比 30% → 合併營收看起來很大，但80%來自子公司，70%利潤被少數股東拿走

適用時機：每次看合併報表時，都要看這兩個比重。如果母公司佔比持續下降，代表公司越來越依賴轉投資，獲利品質可能在惡化。

非控制權益（少數股權）怎麼看

合併報表會把子公司100%的數字都併進來，即使母公司只持股60%。剩下40%屬於其他股東的部分，在報表上稱為「非控制權益」。

怎麼做：
1. 看綜合損益表下半段，有「歸屬於母公司業主」和「歸屬於非控制權益」兩行
2. 看資產負債表的股東權益，有「歸屬於母公司業主」和「非控制權益」
3. 非控制權益佔比越大，合併數字與母公司實際可分配的差距越大

實例：

  • 甲公司持有B公司80%，B公司賺100元
  • 合併損益表顯示淨利100元
  • 但甲公司只享有80元，另外20元歸非控制權益
  • 投資人只看合併淨利會高估甲公司真正可分配的利潤

適用時機：計算EPS、ROE等指標時，一定要用「歸屬於母公司業主」的數字，不要用合併總數。

二、關係人交易辨識 — 地雷股的預警系統

是什麼

關係人交易（Related Party Transactions）是指公司與其關係人之間的進銷貨、資金借貸、資產買賣等交易。關係人包括：母子公司、同一集團企業、董監事及其近親控制的公司等。

為什麼危險

關係人交易最容易成為掏空公司、虛增營收、利益輸送的管道：

  • 虛增營收：母公司出貨給海外子公司，子公司還沒賣給客戶，母公司就認列營收
  • 利益輸送：以不合理的高價向關係人採購，或以不合理的低價賣給關係人
  • 資金掏空：公司資金借貸給關係人、替關係人背書保證

怎麼做：關係人交易檢查SOP

第一步：查財報附註「關係人交易」區段

  • 在公開資訊觀測站，每季財報的附註都會揭露關係人交易
  • 重點看：進貨金額、銷貨金額、應收帳款、應付帳款、資金借貸

第二步：計算關係人交易佔比

  • 關係人進貨佔總進貨比例
  • 關係人銷貨佔總銷貨比例
  • 關係人應收帳款佔總應收帳款比例

判讀標準：

  • 關係人交易佔比 < 20%：正常範圍
  • 關係人交易佔比 20%~50%：需要注意，了解交易條件是否合理
  • 關係人交易佔比 > 50%：🚨 高度警戒，公司可能嚴重依賴關係人

第三步：看關係人交易價格是否合理

  • 關係人進貨單價 vs 市場價格
  • 關係人銷貨單價 vs 市場價格
  • 如果價格明顯偏離市場價 → 可能利益輸送

第四步：追蹤關係人應收帳款

  • 關係人應收帳款佔比過高 → 可能虛增營收，錢收不回來
  • 關係人應收帳款週轉天數異常增加 → 可能拖延付款

實例：

  • 某公司合併營收100億，但關係人銷貨佔60億（60%）→ 🚨 營收嚴重依賴關係人
  • 進一步查：關係人應收帳款週轉天數從60天增加到180天 → 🚨 錢很可能收不回來
  • 再查：關係人銷貨毛利只有5%，非關係人銷貨毛利20% → 🚨 利益輸送確認

關係人交易八大紅旗

1. 關係人銷貨佔比超過50%：營收嚴重依賴關係人，真實性存疑
2. 關係人應收帳款週轉天數暴增：可能虛增營收收不回來
3. 關係人進貨單價高於市場價：可能透過進貨利益輸送
4. 關係人銷貨單價低於市場價：可能透過銷貨利益輸送
5. 公司替關係人背書保證金額過大：可能掏空公司資產
6. 關係人資金借貸金額異常：可能挪用公司資金
7. 董監事質設比率高：大股東財務吃緊，可能有掏空動機
8. 子公司過多且業務複雜：管理困難，容易藏污納垢

適用時機：投資前必查，尤其是集團股、控股公司、海外子公司多的公司。

三、財報附註 — 藏在三大報表背後的秘密

為什麼要看附註

市場先生說得好：「財報的三大報表及各種財務比率數字只是基本功，但已經足以減少無知帶來的損失，而更深入閱讀財報附註，則是能增加決策品質。」

附註是財報最厚、最多人跳過、但最重要的一部分。三大報表的數字是「結果」，附註才是「原因」。

財報附註必看六大項目

1. 重大會計政策與會計估計變更

  • 公司改變折舊年限、存貨計價方法、收入認列時點
  • 會計政策變更可能大幅影響盈餘數字
  • 如果公司突然延長折舊年限 → 可能在美化盈餘

怎麼做：
1. 看附註「重大會計政策」區段
2. 比較前後期是否有一致性
3. 變更時要問：這個變更對淨利影響多少？是合理調整還是人為美化？

2. 關係人交易揭露

  • 前面已詳細說明，這是附註中最重要的一塊
  • 每季都要追蹤關係人交易金額和佔比的變化

3. 或有事項與承諾

  • 或有負債：正在訴訟中、可能要賠償的金額
  • 承諾事項：已簽約但尚未執行的重大合約、資本支出承諾
  • 這些數字不在資產負債表上，但可能對公司造成重大影響

怎麼做：
1. 找附註「承諾及或有事項」區段
2. 看有沒有重大訴訟（金額佔淨值的比例）
3. 看資本支出承諾是否超出公司現金流能力

實例：

  • 某公司資本支出承諾500億，但年度FCF只有200億 → 可能需要大量舉債或增資
  • 某公司有專利侵權訴訟，求償金額佔淨值30% → 🚨 潛在重大風險

4. 資產減損

  • 商譽減損、固定資產減損、長期投資減損
  • 減損金額直接認列損益，可能讓單季EPS暴減
  • 減損後的資產價值反而更接近真實（擠掉水分）

怎麼做：
1. 看附註「資產減損」區段
2. 看減損原因是什麼：一次性還是持續性？
3. 一次性減損可能反而創造買點（利空出盡）
4. 持續性減損代表產業或公司基本面在惡化

5. 營運部門資訊（區隔報告）

  • 公司不同業務線或地區的營收、利潤分別是多少
  • 可以看出公司真正靠什麼賺錢

怎麼做：
1. 找附註「營運部門資訊」區段
2. 看各部門的營收和營業利益
3. 判斷公司成長動能來自哪個部門
4. 如果主要利潤來自非核心部門 → 本業可能正在衰退

6. 重大交易與期後事項

  • 資產處分、併購、分割
  • 財報截止日之後發生的重大事件
  • 可能改變對公司未來的評估

適用時機：每季財報公布後，至少把附註的這六個項目看一遍。花30分鐘看附註，可能幫你避開幾十萬的損失。

四、地雷股八大警訊 — 財報之外的紅旗

從財報數字看出來的警訊

1. 關係人交易異常：佔比過高、價格偏離市場、應收帳款週轉天數暴增
2. 流動比率持續下降：低於100%代表短期償債困難
3. 速動比率低：扣掉存貨後，變現能力更差
4. 利息保障倍數低於5倍：賺的錢不夠付利息，低於2倍極度危險
5. 存貨週轉率異常下降：存貨越來越多，可能賣不出去

從財報之外看出來的警訊

6. 財務長無預警離職：除了董事長和總經理，最了解公司內部狀況的人，突然離職一定是出事了
7. 更換會計師頻繁：法令規定4年更換一次是正常的，但如果更換頻率過高或換到愈來愈小的事務所 → 🚨
8. 董監事質設比率高：超過60-70%，代表大股東自己財務吃緊，有掏空動機

綜合判斷方法

怎麼做：
1. 每季財報公布後，先看三大報表數字
2. 再看附註六大項目
3. 最後交叉檢查八大警訊
4. 如果3個以上警訊同時出現 → 🚨 立即避開

適用時機：投資前必查。特別是低PBR、高殖利率的「便宜股」，警訊可能更多。

心得

今天學的三個主題，剛好串成一套「財報閱讀進階技能」：

合併報表vs個體報表，本質上是在回答「這些數字到底是誰的？」。以前看合併營收1000億覺得很厲害，但拆開來看，如果母公司只貢獻200億，剩下800億是子公司灌進來的，而且其中30%的利潤屬於少數股東，那投資人真正能享有的遠比表面數字少。CMoney那兩個指標太實用了——「母公司營收佔比」和「歸屬母公司純益佔比」，以後每次看合併報表都要算。

關係人交易辨識，就是地雷股的預警系統。博達、力霸這些經典地雷股，事後看都有關係人交易異常的特徵——關係人銷貨佔比超過50%、應收帳款收不回來、毛利率異常偏低。問題是大部分投資人根本不看附註，只看EPS和殖利率，自然看不到這些紅旗。

財報附註，是三大報表的「說明書」。數字會說謊，但附註會說實話。會計政策變更、或有事項、減損原因、部門利潤來源——這些才是決策品質的關鍵。市場先生那句話很到位：資料庫篩選不出來的資訊，就在附註裡。

最深刻的體會：看財報的順序不是從數字開始，而是從附註開始。先看附註了解數字背後的故事，再看三大報表的數字才有意義。數字只是結果，附註才是原因。

一、營運槓桿度（DOL, Degree of Operating Leverage）

是什麼

營運槓桿度衡量的是：銷售量變動 1% 時，息前稅前盈餘（EBIT）會變動多少百分比。

公式：
> DOL = (營收 − 變動成本) ÷ 營業利益 = (營業利益 + 固定成本) ÷ 營業利益

或者用變動率表示：
> DOL = ∆EBIT/EBIT ÷ ∆Q/Q

怎麼做

1. 從財報拆出固定成本與變動成本：固定成本 = 人事費用 + 折舊 + 攤銷（最保守估計）；變動成本 = 原物料 + 直接人工
2. 計算 DOL：DOL = (營業利益 + 固定成本) ÷ 營業利益
3. 判讀數值：

  • DOL = 1.x～2.x：一般穩定獲利製造業的正常範圍

實例

  • 岱稜（3303）2018 年：年報 DOL = 9.13，意即營收增加 1%，營業利益增加約 9%。這麼高的原因是剛越過損益兩平點，固定成本高但營業利益基期低
  • 自行計算版：用固定成本 = 人事 + 攤銷 + 折舊 = 6.47 億，營業利益 = 2.27 億，DOL = (2.27 + 6.47) ÷ 2.27 ≈ 3.85
  • 差異原因：年報對固定/變動成本的定義可能不同（可能把更多成本歸為固定），但兩種算法都指向「相對高槓桿」的結論

適用時機

  • 最佳時機：企業剛開出新產能、越過損益兩平點時 — 微幅營收成長帶來大幅獲利成長，這是提前布局的黃金時機
  • 不適用：企業營業利益為負時（DOL 失去意義）、服務業/買賣業（固定成本低，DOL 本來就小）
  • 注意：DOL 是基於當期數字計算的，不是絕對值，是方向性指標

二、財務槓桿度（DFL, Degree of Financial Leverage）

是什麼

財務槓桿度衡量的是：EBIT 變動 1% 時，每股盈餘（EPS）會變動多少百分比。

公式：
> DFL = EBIT ÷ (EBIT − 利息費用)

如果有特別股：
> DFL = EBIT ÷ (EBIT − I − D_pf/(1−T))

其中 I = 利息費用，D_pf = 特別股股利，T = 稅率

怎麼做

1. 從損益表抓出利息費用（營業外支出中的利息支出）
2. 計算 DFL：DFL = EBIT ÷ (EBIT − I)
3. 判讀：

  • DFL ≈ 1.0～1.2：低財務槓桿，利息負擔輕，安全

實例

  • A 公司：EBIT = 3,200 千元，利息 = 1,600 千元 → DFL = 3,200 ÷ 1,600 = 2.0。EBIT 減少 10%，EPS 減少 20%
  • 「先收後付」產業例外：超市、大賣場、支付產業帳上常有大量浮存現金，即使負債權益比高，現金流充足下風險不大

適用時機

  • 一定要看：負債比高、利息覆蓋倍數低於 3 倍的公司
  • 不適用：無負債的公司（DFL = 1，沒有分析價值）
  • 搭配看：負債權益比 + 利息覆蓋倍數，DFL 只是補充驗證

三、總槓桿度（DTL, Degree of Total Leverage）

是什麼

總槓桿度 = 營運槓桿 × 財務槓桿，衡量的是：銷售量變動 1% 時，EPS 會變動多少百分比。

公式：
> DTL = DOL × DFL

或者：
> DTL = (營業利益 + 固定成本) ÷ (營業利益 − 利息費用)

怎麼做

1. 分別算出 DOL 和 DFL
2. 兩者相乘 = DTL
3. 判讀：

  • DTL < 3：低風險，銷售波動對 EPS 影響溫和

實例

  • A 公司：DOL = 1.75，DFL = 2.0 → DTL = 3.5。銷售增加 10%，EPS 增加 35%
  • B 公司：DOL = 0.3，DFL = 2.0 → DTL = 0.6。銷售增加 10%，EPS 增加 6%
  • A 公司 EPS 彈性遠大於 B 公司 — 多頭市場 A 大賺，空頭市場 A 也大賠

適用時機

  • 多頭找高 DTL：景氣上行期，高總槓桿的公司 EPS 爆發力最強
  • 空頭找低 DTL：景氣下行期，低總槓桿的公司防禦力最好
  • 最佳組合：高 DOL + 低 DFL — 用營運槓桿吃景氣紅利，但不用財務槓桿冒倒閉風險

四、實戰選股：DOL + DFL 組合策略

四象限判讀

低 DFL（利息輕）高 DFL（利息重）
**高 DOL**（固定成本重）🟢 **黃金組合**：景氣好 EPS 爆發，景氣差也不會因利息倒閉🔴 **危險組合**：景氣好大賺，景氣差 EBIT 不夠付利息就倒
**低 DOL**（固定成本低）🟡 **防禦組合**：穩定但不性感，適合存股🟠 **投機組合**：靠借錢放大報酬，現金流一斷就完蛋

最佳實戰策略

1. 景氣復甦期：找高 DOL + 低 DFL 的公司（剛過損益兩平點、沒什麼負債）
2. 景氣擴張期：高 DOL + 中 DFL 可以接受，但要設停損
3. 景氣收縮期：全部轉低 DOL + 低 DFL，防禦為主
4. 避開：高 DOL + 高 DFL = 定時炸彈，一次衰退就歸零

計算步驟

1. 從財報抓出：營業利益（EBIT）、固定成本（人事+折舊+攤銷）、利息費用
2. DOL = (EBIT + 固定成本) ÷ EBIT
3. DFL = EBIT ÷ (EBIT − 利息)
4. DTL = DOL × DFL
5. 對照景氣位置，決定投資方向

心得

三種槓桿度本質上是在回答一個問題：這家公司離「死」有多遠？

DOL 告訴你：營收往下掉多少，EBIT 就會歸零（損益兩平點在哪）
DFL 告訴你：EBIT 往下掉多少，EPS 就會被利息吃光
DTL 告訴你：兩者乘起來，離「銷售下滑 → EPS 歸零」的距離

以前只看負債比和利息覆蓋倍數，那只是 DFL 的視角。但 DOL 的視角更重要 — 一家沒有負債的公司，如果固定成本極高、營業利益極薄，同樣是高風險。岱稜的案例就非常清楚：年報 DOL 9.13，自行估計也有 3.85～5，都是「高槓桿」，但這跟借錢無關，純粹是因為重資本製造業的特性。

最實用的判斷框架就是四象限：高 DOL + 低 DFL 是景氣復甦期的最佳標的 — 營運槓桿讓它吃得到成長紅利，低財務槓桿讓它不怕衰退時付不出利息。反過來，高 DOL + 高 DFL 就是在走鋼索，Bill Hwang 的千億美元爆倉、LTCM 的 30 倍槓桿倒閉，都是血淋淋的教材。

以後選股除了看 ROE、負債比，一定要加算 DOL 和 DFL，看清楚這家公司的「獲利彈性」和「死亡距離」。

一、股東權益變動表 — 四大報表裡最常被忽略、但最會藏雷的那張

是什麼

股東權益變動表（Statement of Changes in Equity）記錄一個會計期間內，股東權益各科目的增減變化。三大報表告訴你公司「賺多少、欠多少、剩多少」，但權益變動表告訴你「剩下的那些，怎麼來的、怎麼分的」。

股東權益的五大科目

1. 股本（Common Stock）：發行股數 × 面額10元，代表股東最初投入的資本
2. 資本公積（Additional Paid-in Capital）：股票發行時超過面額的溢價部分，以及庫藏股交易超過面額的部分。是「股東多出的資本」
3. 保留盈餘（Retained Earnings）：企業歷年獲利累積，扣除配息後的剩餘

  • 法定盈餘公積：公司法規定提撥10%，累計超過股本就不用再提

4. 其他權益（Other Equity）：

  • 國外營運機構財報換算匯兌差額（隨匯率波動，影響不大）

5. 庫藏股（Treasury Stock）：公司買回自家股票，是權益的減項

怎麼做：權益變動表檢查SOP

第一步：看權益變動表的「乾淨度」

蕭老師提出了一個很實用的判斷標準：愈專注本業的企業，權益變動表愈乾淨。

  • 大立光(3008)：權益變動只有「提列法定盈餘、配發現金股利、綜合損益」三項，非常乾淨
  • 鴻海(2317)：多了「股份基礎給付」和「非控制權益增減」，也算簡潔
  • 樂陞(3662)：現金增資、可轉債轉換、各種權益操作一大堆 → 🚨 權益變動表太複雜就是警訊

怎麼做：
1. 打開權益變動表
2. 數一下當期有多少項權益變動事件
3. 除了「本期淨利、配息、法定盈餘公積」這三項基本款之外，如果超過5項 → 提高警覺
4. 如果有「資本公積彌補虧損」「可轉債轉換」「現金增資」等多項同時出現 → 🚨

第二步：檢查「其他權益」中的未實現損益

備供出售金融資產的累計未實現損益，是最容易被忽略的地雷。

怎麼做：
1. 看權益變動表中「其他權益—備供出售金融資產未實現損益」的累計金額
2. 如果累計虧損接近甚至超過1個股本 → 🚨 未來一旦實現，EPS會重傷
3. 如果累計獲利很大 → 未來處分時會推升EPS，但要注意公司是否靠處分資產撐獲利

實例：

  • 潤泰全球(2915)2016年前3季：備供出售未實現利益5.7億，但累計帳上虧損高達184.5億，近2個股本 → 🚨 隨時可能認列大額虧損
  • 福懋興業(1434)同期：累積未實現利益292.3億，近2個股本 → 未來處分時EPS會大增

第三步：檢查「保留盈餘」是否真實

保留盈餘為正不代表公司真的累積了獲利。公司可以用「資本公積彌補虧損」，讓保留盈餘從負轉正——這是「以東牆補西牆」美化財報的手法。

怎麼做：
1. 看保留盈餘的「未分配盈餘」是否為正
2. 如果為正，再看資本公積是否大幅減少（被拿來彌補虧損了）
3. 保留盈餘從負轉正 + 資本公積同步減少 = 美化財報，不是真正的獲利改善

第四步：追蹤股本變動來源

股本增加不一定來自現金增資，也可能來自可轉換公司債轉換、員工股票選擇權等。

怎麼做：
1. 看權益變動表中股本增加的原因
2. 如果可轉債轉換股本的比例高 → 股本膨脹會稀釋EPS
3. 如果頻繁現金增資 → 公司可能現金流有問題

實例：

  • 樂陞(3662)2016年：現金增資2.1億 + 可轉債轉換670萬股本；2015年整年度現金增資680萬 + 可轉債轉換2,261萬股本 → 權益操作頻繁，警訊

適用時機

  • 每季財報公布後，花10分鐘掃一遍權益變動表
  • 投資前必查，尤其是集團股、金融股、轉投資多的公司
  • 如果三大報表看起來沒問題但股價持續下跌 → 一定要查權益變動表

二、減資三方式 — 不是所有減資都一樣

減資是什麼

減資（Capital Reduction）= 減少公司資本額（股本），讓流通在外股數變少。減資後投資人持股變少，但股價會相應調升，總市值不變。

三種減資方式比較

方式一：一般減資（彌補虧損減資）

目的：打掉累積虧損，美化每股淨值，避免信用交易被取消或變成全額交割股

怎麼做（公司端）：
1. 董事會通過減資案
2. 股東會決議
3. 送金管會審查
4. 註銷部分流通在外股票，用股本抵銷累積虧損

股價計算：減資後股價 = 原股價 ÷ (1 - 減資比例)

實例：某公司股價20元，減資50%（1000股→500股）

  • 減資後股價 = 20 ÷ (1 - 50%) = 40元
  • 原市值 = 1000股 × 20元 = 20,000元
  • 減資後市值 = 500股 × 40元 = 20,000元（不變）

投資判斷：

  • ⚠️ 這類減資通常是「不得不做」——公司虧損到淨值快低於10元
  • 減資後如果營運沒改善，問題還在，股價可能繼續跌
  • 統計數據：彌補虧損減資的公司，短期股價通常下跌

方式二：現金減資

目的：公司現金太多且無投資計畫，把現金退還給股東

怎麼做（公司端）：
1. 董事會通過現金減資案
2. 股東會決議
3. 將部分股本以現金退還股東（每股退還X元）
4. 同時註銷相應股數

股價計算：減資後股價 = (原股價 - 每股退還現金) ÷ (1 - 減資比例)

實例：東和鋼鐵(2006)2021年現金減資33%

  • 減資前收盤價43.6元，每股退還3.3元
  • 減資後股價 = (43.6 - 3.3) ÷ (1 - 33%) = 60.2元
  • 原持有1張（1,000股）→ 減資後持有670股 + 3,300元現金

為什麼大股東愛現金減資：

  • 領現金股利要課稅（二代健保2.11%、所得稅）
  • 現金減資是「退還股款」，定義上不是股利，不用課稅
  • 大股東透過現金減資可以拿回大量現金又避稅

投資判斷：

  • ✅ 獲利穩定、現金充沛的公司現金減資 → 利多，股價通常短期上漲
  • ⚠️ 現金減資後負債比提高、現金不足 → 潛在風險
  • ⚠️ 公司無投資計畫 = 可能產業成長性不足
  • 統計數據：現金減資的績優公司，宣告後短期股價多上漲

方式三：庫藏股減資

目的：公司認為股價被低估，買回自家股票並註銷，提升每股盈餘

怎麼做（公司端）：
1. 董事會決議買回庫藏股（須說明買回目的、價格區間、數量）
2. 在市場上用公司現金買回自家股票
3. 買回後註銷股票，減少流通在外股數

股價計算：庫藏股減資不調整股價（因為是用市價買的）

對投資人的影響：

  • 股數不變（只有賣出庫藏股的股東股數才減少）
  • 流通在外總股數減少 → EPS提升 → 股價通常助漲
  • 但公司花了帳上現金買股票，這原本也是投資人的錢

投資判斷：

  • ✅ 公司覺得股價低估 + 本業穩定 → 利多
  • ⚠️ 公司買回庫藏股只是為了護盤、替大股東解套 → 利多有限
  • ✅ 統計數據：庫藏股減資的績優公司，短期股價多上漲

三種減資方式對股價影響的統計

依據2007~2016年台股統計：

  • 現金減資的績優公司：宣告後短期股價上漲
  • 庫藏股減資的績優公司：宣告後短期股價上漲
  • 彌補虧損減資的公司：短期股價通常下跌

結論：減資本身是中性行為，關鍵看減資原因背後的公司體質。

金管會2022年新設減資門檻

1. 減資超過50%時，會計師需附相關說明（合理性、必要性、對財務結構影響）
2. 投保中心需逐案審核減資案，在股東會上表達關心
3. 通過減資決議後1年內不得再進行現金增資

減資後的零股處理

減資後如果持股不滿1張（1,000股），零股部分須以零股交易方式出售。例如原持有1張，減資30%後剩700股，這700股須以零股賣出（盤中零股交易或盤後定盤交易）。

三、增資對股價的影響 — 股本膨脹的代價

增資是什麼

增資 = 增加公司資本額（股本），讓流通在外股數增加。增資後股價會相應調降，總市值不變。

增資方式

1. 現金增資：公司向股東或市場募集資金，發行新股

  • 員工認股、原股東優先認購、公開申購

2. 盈餘轉增資（股票股利）：把保留盈餘轉成股本，配發股票股利

  • 對投資人來說：股數增加但每股價值降低，總市值不變

3. 資本公積轉增資：把資本公積轉成股本，無償配發新股

  • 通常用「資本公積—股本溢價」來轉

增資對股價的影響

  • 短期：現金增資通常是利空（稀釋EPS、市場解讀為公司缺錢）
  • 長期：如果增資是為了擴廠、研發等生產力投資 → 長期利多
  • 最糟的增資：為了還債、補虧損而增資 → 本業沒改善，股本卻膨脹

增資 vs 減資的判讀邏輯

事件短期影響背後原因投資判斷
現金減資利多公司現金多✅ 如果本業穩定
庫藏股減資利多股價低估✅ 如果本業穩定
彌補虧損減資利空公司虧損⚠️ 除非有明確轉機
現金增資利空公司缺錢⚠️ 除非是擴產投資
盈餘轉增資中性配股票股利看公司成長性

心得

今天學的權益變動表，真的是四大報表裡最被低估的一張。大部分投資人只看損益表的EPS和營收，頂多看一下資產負債表的負債比，權益變動表根本跳過。但蕭老師那句話太精準了：「愈專注本業的企業，權益變動表愈乾淨。」大立光只有三項變動，樂陞洋洋灑灑一大堆——這種對比就是最直覺的「好公司 vs 地雷股」判斷法。

最讓我震驚的是「備供出售金融資產未實現損益」這個科目。潤泰全球累計未實現虧損184.5億，近2個股本——這個數字不會出現在損益表上，但一旦實現就是核彈級的衝擊。還有「資本公積彌補虧損」讓保留盈餘從負轉正的把戲，以前根本不知道有這招。

減資的部分，最重要的結論是：減資本身是中性行為，關鍵是減資的原因。現金減資=公司現金多（好事）、庫藏股減資=公司覺得股價低估（好事）、彌補虧損減資=公司虧損嚴重（壞事）。統計數據也證實了這個判斷：前兩者短期股價上漲，後者短期下跌。

現金減資的大股東避稅問題也值得深思。對小股東來說，現金減資和現金股利效果差不多（都是拿到現金），但大股東透過現金減資可以完全避開股利所得稅——這制度上的漏洞讓大股東有動機選擇現金減資而非配息，小股東要意識到這個遊戲規則的不對等。

以後看財報的流程要改：不只看三大報表，一定要加看權益變動表。花10分鐘掃一遍權益變動的項目數量、其他權益的累計金額、股本變動來源，就能避開很多隱藏的地雷。

一、盈餘管理（Earnings Management）— 財報數字的化妝術

是什麼

盈餘管理不是做假帳，而是在會計準則允許的範圍內，透過會計政策選擇、會計估計變動、交易時點安排等手段，讓財報數字呈現管理層期望的樣貌。

核心特徵：

  • 不改變企業實際獲利總額，只改變獲利在不同期間的分布
  • 利用手中的「判斷空間」影響會計數據
  • 最終目的是管理層自身利益最大化

盈餘管理的三種手法

手法一：會計政策選擇與變更

會計準則給企業多種選擇，管理層可以選擇對自己最有利的：

  • 存貨計價方法：FIFO讓通膨時期毛利好看，加權平均法讓數字平穩
  • 折舊方法：直線法vs加速折舊法，影響每年折舊費用
  • 研發支出資本化vs費用化：資本化讓當期費用減少、利潤增加
  • 收入認列時點：提早認列收入vs延後認列收入

手法二：應計項目操縱

透過調整「應計項目」（還沒收到現金但已認列的收入，或還沒付現金但已認列的費用）：

  • 放寬信用政策衝營收 → 應收帳款暴增
  • 延後認列費用 → 應付帳款累積
  • 提前認列收入 → 遞延收入減少
  • 操控壞帳提列比率 → 呆帳費用變化

手法三：實質經濟活動操縱

不只改會計數字，連實際交易都操控：

  • 年底衝營收：給經銷商大量折扣壓貨，明年再退貨
  • 出售資產認列業外收益：賣土地、賣設備，讓當季EPS達標
  • 減少研發支出：砍研發費讓當季利潤達標，但犧牲長期競爭力
  • 操控資本支出時點：延後或提前資本支出，影響折舊費用

投資人如何辨識盈餘管理

怎麼做（八道紅旗檢查法）：

1. 應收帳款成長率遠高於營收成長率：可能放寬信用條件衝營收
2. 存貨週轉率突然下降：可能壓貨給經銷商，或需求正在下滑
3. 營業現金流持續低於淨利：帳面賺錢但現金沒進來，應計項目可能有問題
4. 一次性項目頻繁出現：經常有「業外收入」美化淨利
5. 會計政策突然變更：特別是接近年底或財測可能未達標時
6. 研發資本化比率突然上升：把本該費用化的支出資本化，美化利潤
7. 關係人交易金額異常增加：可能透過關係人虛增營收
8. 遞延收入大幅減少：可能提前認列未來收入

實例判讀：

紅旗可能的手法判讀方式
應收帳款增加30%，營收只增10%放寬信用條件衝營收應收帳款週轉天數是否增加
存貨增加50%，營收只增15%壓貨或需求下滑存貨週轉率是否下降
營業現金流/淨利 < 0.7應計項目膨脹連續3季觀察

會計政策變更換手法美化盈餘附註揭露+前後期比較

適用時機：每季財報公布後，用八道紅旗跑一遍持股。3個以上紅旗同時出現 → 🚨 高度警戒。

二、存貨計價方法 — 同樣的存貨，不同的毛利

三種主要方法

方法一：先進先出法（FIFO）

假設最早買進的存貨先賣出。

> 通膨環境下：舊成本低 → 銷貨成本低 → 毛利高；期末存貨接近市價（偏高）

怎麼做：
1. 建立存貨進貨明細（批次、數量、單價）
2. 銷貨時，從最早的進貨批次開始扣減
3. 期末存貨由最近進貨的成本組成

實例：

  • 第一批進貨100件@10元，第二批進貨100件@12元
  • 銷貨150件：先扣第一批100件@10元，再扣第二批50件@12元
  • 銷貨成本 = 100×10 + 50×12 = 1,600元
  • 期末存貨 = 50×12 = 600元

方法二：加權平均法（台灣最常用）

把所有存貨的加權平均成本當作銷貨成本和期末存貨成本。

> 加權平均單位成本 = （期初存貨成本 + 本期進貨成本）÷（期初存貨數量 + 本期進貨數量）

怎麼做：
1. 計算加權平均單位成本
2. 銷貨成本 = 銷貨數量 × 加權平均單位成本
3. 期末存貨 = 期末數量 × 加權平均單位成本

實例：

  • 期初存貨100件@10元=1,000元，本期進貨100件@12元=1,200元
  • 加權平均單位成本 = (1,000+1,200) ÷ (100+100) = 11元
  • 銷貨150件：銷貨成本 = 150×11 = 1,650元
  • 期末存貨 = 50×11 = 550元

方法三：後進先出法（LIFO，台灣IFRS禁止使用）

假設最後買進的存貨先賣出。

> 通膨環境下：新成本高 → 銷貨成本高 → 毛利低；期末存貨偏低（老成本）

重要：IFRS已禁止使用LIFO，台股投資人不需擔心LIFO問題，但看美股財報時仍需注意。

三種方法在通膨環境下的比較

影響FIFO加權平均法LIFO
銷貨成本低（用舊成本）中等高（用新成本）
毛利最高中等最低
期末存貨價值最接近市價中等偏低（舊成本）
稅負最高中等最低
營業現金流較低（稅繳多）中等最高（稅繳少）

存貨計價方法變更的紅旗

IFRS規定：存貨計價方法變更屬於會計政策變更，必須：
1. 在附註中揭露變更原因
2. 追溯重編前期損益

投資人警覺信號：

  • 公司突然從FIFO改為加權平均法，或反過來 → 可能是為了調整毛利率
  • 特別是在毛利率剛好卡在分析師預期邊緣時變更 → 🚨 高度可疑
  • 變更後毛利率大幅改善，但營業現金流沒有同步改善 → 數字美化

適用時機：每次看財報附註時，確認存貨計價方法是否與前期一致。

三、存貨跌價損失與回升利益 — 景氣循環股的超前指標

是什麼

IFRS規定存貨必須以「成本與淨變現價值孰低法」評價。當存貨的成本高於淨變現價值（預估售價−估計完工成本−估計銷售費用），必須認列存貨跌價損失；當淨變現價值回升時，可以在原認列減損範圍內認列回升利益。

公式：
> 淨變現價值（NRV） = 預估售價 − 估計完工尚需投入成本 − 估計銷售費用

存貨跌價損失的雙面性

電子業的存貨跌價：產品生命周期短，跌價後幾乎不可能回升 → 真正的損失
傳統產業的存貨跌價：原物料價格有週期性，跌價後可能回升 → 暫時性損失，日後可能認列回升利益

怎麼做（投資人判讀SOP）：

1. 看損益表的營業成本明細：存貨跌價損失或回升利益都在營業成本中
2. 區分產業特性：原物料相關（鋼鐵、航運、石化）vs 產品相關（電子、手機）
3. 追蹤原物料價格走勢：判斷存貨跌價是暫時還是永久
4. 計算存貨跌價對EPS的影響：存貨跌價損失金額 ÷ 在外流通股數

實戰案例：原物料循環股的超前佈局

新光鋼（2031）案例：

  • 2015年鐵礦砂大跌，新光鋼認列存貨跌價損失2.96億元
  • 毛利率跌到-9%（存貨跌價吃掉毛利）
  • 2016年鐵礦砂大漲，新光鋼認列存貨回升利益3.05億元
  • 毛利率回升到15%
  • 股價從12元漲到24元，翻倍

第一銅（2009）案例：

  • 銅價下跌時認列存貨跌價損失，毛利率為負
  • 銅價回升時認列存貨回升利益，毛利率轉正
  • 股價跟著毛利率走動

投資策略：存貨跌價反轉法

怎麼做：
1. 篩選原物料相關產業，目前毛利率被存貨跌價損失壓低的公司
2. 確認本業沒有嚴重惡化（營收穩定、非關係人交易正常）
3. 追蹤原物料價格是否開始反轉向上
4. 原物料反轉 + 存貨回升利益 + 本業回穩 = 三重加分效果

適用時機：原物料價格底部反轉時，這是超前佈局的黃金時機。

四、資本化vs費用化 — 財報數字的核心操縱點

是什麼

一筆支出，公司可以選擇：

  • 資本化：記為資產，未來逐年折舊/攤銷 → 當期費用少、利潤高
  • 費用化：當期全額認列費用 → 當期費用高、利潤低

同樣的支出，不同的會計處理，當期EPS可能天差地遠。

哪些支出有「灰色地帶」

支出類型會計處理灰色地帶
辦公室租金費用化無爭議
員工薪資費用化無爭議
購買廠房設備資本化無爭議
**研發支出**研究階段→費用化；發展階段→可資本化**最大爭議**
**軟體開發成本**技術可行前→費用化；可行後→可資本化**有操作空間**
**借款利息**一般→費用化；建造期間→可資本化**有操作空間**
**設備維修**日常維修→費用化；延長壽命的改良→資本化**有操作空間**

研發支出資本化 — 最大的灰色地帶

IFRS規定研發支出分為兩個階段：

  • 研究階段：必須費用化（未來效益不確定）
  • 發展階段：符合嚴格條件可資本化（技術可行、有商業價值、有意圖完成等）

怎麼判斷資本化是否過度：
1. 看研發資本化比率 = 資本化研發支出 ÷ 總研發支出
2. 與同業比較：如果同業資本化比率20%，公司突然拉到50% → 🚨
3. 看資本化金額佔淨利的比例：如果超過50% → 利潤品質存疑
4. 看攤銷速度：如果資本化後攤銷年限拉很長 → 可能拖延費用認列

實例：

  • A公司研發支出100億，資本化20億（20%）→ 合理範圍
  • B公司研發支出100億，資本化60億（60%）→ 過度積極，美化利潤
  • 如果B公司淨利50億，其中60億是資本化延後認列的費用 → 實質上本業虧損

資本化利息 — 隱藏的利息費用

建造期間的借款利息可以資本化，計入資產成本而非當期費用。

投資人判讀：
1. 看財報附註的「資本化利息」金額
2. 資本化利息 ÷ 總利息支出 = 利息資本化比率
3. 比率越高，代表越多利息被「藏」在資產裡，當期損益表看起來更好
4. 建廠期間資本化利息正常，但完工後持續資本化 → 🚨

過度資本化的風險

短期看：利潤美化，EPS好看
長期看：
1. 資產虛胖 → 未來折舊/攤銷壓力大
2. 如果資本化的資產無法產生預期效益 → 必須認列資產減損
3. 減損一次性認列 → 未來EPS暴減

實例：

  • 某軟體公司開發新產品，資本化80%研發支出
  • 產品上市後銷售不佳，必須認列大量減損
  • 減損當季EPS暴跌，股價腰斬

投資人防範SOP

怎麼做（五步檢查法）：

1. 看現金流量表：營業現金流 vs 淨利 — 如果淨利持續遠高於營業現金流 → 應計項目膨脹，資本化可能是原因之一
2. 看研發資本化比率：與同業比較，超過同業平均值太多 → 🚨
3. 看無形資產攤銷：如果攤銷金額佔營業利益比例超過30% → 過去資本化的壓力正在顯現
4. 看資產減損頻率：經常性減損代表過去資本化過度
5. 看附註的會計政策揭露：資本化標準是否比同業更寬鬆

適用時機：分析科技股、生技股時，研發資本化比率是必看指標。

五、收入認列操縱 — 盈餘管理最直接的管道

五種常見手法

手法一：提前認列收入

  • 還沒交付商品就認列收入
  • 給經銷商大量折扣壓貨，月底衝營收
  • 提前開立發票認列收入

怎麼辨識：
1. 季末營收突然暴增（最後一個月佔全季40%以上）
2. 應收帳款週轉天數增加
3. 退貨率突然上升

手法二：總額法vs淨額法

  • 總額法：按交易總金額認列收入（如平台按GMV認列）
  • 淨額法：只按佣金/手續費認列收入（如平台只認列手續費收入）
  • IFRS 15規定：只有「主要義務人」才能用總額法，「代理人」只能用淨額法

實例：

  • 某電商平台GMV 100億，手續費5億
  • 總額法認列收入100億，毛利率5%（看起來像薄利多銷的零售商）
  • 淨額法認列收入5億，毛利率100%（真實面貌是平台服務商）
  • 兩者ROE相同，但營收規模差20倍！

手法三：通道填塞（Channel Stuffing）

  • 季末大量出貨給經銷商，超過正常需求
  • 下季再接受退貨或提供折讓
  • 實質上沒有最終銷售

怎麼辨識：
1. 季末營收暴增後，下季初營收驟降
2. 經銷商庫存異常增加
3. 退貨率在下季異常偏高

手法四：建構式收入（Built-to-Stock）

  • 超出客戶實際需求生產並認列收入
  • 把存貨塞到通路商，但風險報酬尚未移轉

手法五：多元素合約的收入分配

  • 一筆合約包含硬體+軟體+服務，總價如何分配到各元素？
  • 把更多收入分配到當期可認列的元素 → 提前認列
  • IFRS 15要求按各元素單獨售價比例分配

投資人收入品質檢查SOP

怎麼做：

1. 營收與現金流對比：營收成長率 vs 營業現金流成長率 — 差距過大 → 🚨
2. 應收帳款週轉天數趨勢：持續增加 → 可能提前認列或放寬信用
3. 遞延收入變化：大幅減少 → 可能提前認列未來收入
4. 季末營收佔比：最後一個月佔全季超過40% → 可能衝量
5. 退貨率趨勢：退貨率突然上升 → 可能是通路塞貨

適用時機：每季財報公布後，用五項指標檢查收入品質。

心得

今天的主題串起來，就是在回答一個核心問題：財報上的數字，有多少是真實的？

盈餘管理不是做假帳，而是在規則允許的範圍內「化妝」。化妝不犯法，但投資人必須看得出來妝有多厚。三種手法各有巧妙：會計政策選擇是「換粉底」，應計項目操縱是「打光影」，實質經濟活動操縱是「整型」。

存貨計價方法的影響比想像中大。同樣一批存貨，FIFO在通膨時期讓毛利最高、稅也繳最多；加權平均法讓數字平穩。但最重要的是：公司如果突然變更存貨計價方法，就是在換化妝品品牌，投資人一定要看附註確認。

存貨跌價損失對原物料循環股是個超前指標，這個最實用。新光鋼和第一銅的案例清楚顯示：當毛利率被存貨跌價損失壓到負數，但原物料價格開始反轉時，存貨回升利益+本業回穩的雙重效果可以讓EPS和股價暴漲。這種機會不常有，但一出現就是大波段。

資本化vs費用化是科技股和生技股最常見的陷阱。研發資本化比率突然從20%飆到60%，就是在把當期費用「搬」到未來去攤，讓當期EPS好看。但搬走的費用不會消失，遲早要認列攤銷或減損。最簡單的驗證方法：看營業現金流是否跟得上淨利。如果淨利年年成長但營業現金流停滯，裡面一定有鬼。

收入認列操縱是最直接的：把明天的收入搬到今天，把今天的費用推到明天。IFRS 15的五步法收入認列模型雖然規範更嚴，但管理層仍有判斷空間——特別是在多元素合約的收入分配、變動對價的估計、履約義務的拆分上。投資人能做的就是交叉驗證：營收成長率要跟營業現金流成長率同步，應收帳款週轉天數不能持續惡化，季末營收不能異常暴增。

最後，盈餘管理的核心邏輯是：不改變企業真實價值，只改變獲利在不同期間的分布。所以長期投資人其實不太需要怕盈餘管理——時間拉長，數字會回歸真實。但短線交易者如果看不懂化妝術，就會在EPS最高點買進、在減損認列時賣出，完美反向操作。

一、DDM 股利折現模型（Dividend Discount Model）

是什麼

用未來領到的股利折現回現在，算出股票的內在價值。核心邏輯：股票的價值 = 未來所有股利的現值總和。

三種變體

1. 單期模型：V₀ = D₁/(1+r) + P₁/(1+r)

  • 只看一年，適合短期持有估算
  • 問題：P₁（賣出價）本身就需要估計

2. 多期模型：V₀ = Σ[Dₜ/(1+r)ᵗ] + Pₙ/(1+r)ⁿ

  • 看多年，每年股利可以不同
  • 問題：年數越多，預估越不準

3. 戈登成長模型（Gordon Growth Model）：V₀ = D₁/(r-g)

  • 最常用，假設股利以固定成長率 g 永遠成長
  • 前提：r > g（折現率必須大於成長率，否則公式無意義）

實例計算

假設某公司：

  • 明年預估股利 D₁ = 3 元
  • 折現率 r = 8%
  • 股利成長率 g = 5%

內在價值 = 3 / (8% - 5%) = 3 / 0.03 = 100 元

如果目前股價 110 元 → 高估 10 元，不該買
如果目前股價 80 元 → 低估 20 元，值得買

零成長特例

當 g = 0 時：V₀ = D/r

  • 適用於股利不成長的穩定公司（像債券一樣）
  • 例如：D = 5 元、r = 8% → V₀ = 5/0.08 = 62.5 元

DDM 三大缺點

缺點 1：只能評估穩定配息的公司

  • 不配息的公司（Google、Amazon、波克夏）完全無法用 DDM 估價
  • 早期成長股也不適用

缺點 2：假設條件太多

  • 成長率 g 是猜的，g 變 1%，合理價可以差幾十%
  • D₁ = 3, r = 8%, g = 5% → V = 100
  • D₁ = 3, r = 8%, g = 4% → V = 75（差 25%！）

缺點 3：一次性收益會扭曲

  • 某年賣土地配特別股利 → 不是常態
  • 必須排除一次性業外收入，用常態股利計算

DDM vs DCF 比較

比較DDMDCF
現金流來源股利自由現金流（FCF）

計算難度較簡單較複雜
風險漏掉不配息的好公司變數多，估錯風險大

實戰使用原則

1. 只用在財務簡單、配息穩定的成熟企業
2. 預估盡量保守：過去成長 10%，寧可估 5% 成長 5 年再遞減
3. 搭配安全邊際：算出合理價後打 8 折才進場
4. 不要當唯一估價方法：搭配 PE、PBR、DCF 交叉驗證
5. 檢查配息率是否穩定：盈餘與配息比例異常的要避開

二、戈登成長模型進階：二段式與多段式

二段式成長模型

現實中公司不會永遠以同一個速度成長。二段式模型把成長分成兩階段：

公式：
V₀ = Σ[D₀×(1+g₁)ᵗ/(1+r)ᵗ] (t=1~n) + Dₙ₊₁/[(r-g₂)×(1+r)ⁿ]

  • g₁：第一階段高成長率（例如前 5 年成長 15%）
  • g₂：第二階段穩定成長率（例如之後永續成長 3%）
  • n：高成長期年數

實例：
某公司今年股利 2 元，預期前 5 年成長 12%，之後穩定成長 4%，折現率 10%

第一階段（高成長 5 年）：

  • D₁ = 2×1.12 = 2.24
  • D₂ = 2.24×1.12 = 2.51
  • D₃ = 2.51×1.12 = 2.81
  • D₄ = 2.81×1.12 = 3.15
  • D₅ = 3.15×1.12 = 3.53
  • 現值 = 2.24/1.1 + 2.51/1.21 + 2.81/1.33 + 3.15/1.46 + 3.53/1.61 ≈ 10.34

第二階段（永續成長）：

  • D₆ = 3.53×1.04 = 3.67
  • 終值 = 3.67/(0.10-0.04) = 61.17
  • 終值現值 = 61.17/1.61 ≈ 37.99

內在價值 ≈ 10.34 + 37.99 = 48.33 元

三個關鍵假設

1. r > g₂：永續成長率必須小於折現率（否則公式發散）
2. g₂ 接近 GDP 成長率：沒有公司能永遠以 10% 成長，永續成長率通常用 2-4%
3. 過渡期處理：高成長到穩定成長之間的過渡期可以加入第三階段（三段式）

戈登公式反推預期報酬率

公式變形：r = (D₁/P₀) + g

這告訴我們：預期報酬率 = 股息殖利率 + 股利成長率

例如：台積電股價 600，明年股利 15 元，成長率 8%
r = (15/600) + 8% = 2.5% + 8% = 10.5%

→ 投資人預期年報酬率約 10.5%

三、WACC 加權平均資本成本（Weighted Average Cost of Capital）

是什麼

DCF 估值法中，最主流的折現率計算方式。把公司的兩種資金成本（債權 + 股權）加權平均，得到一個客觀的折現率。

公式

WACC = [D/(E+D) × Rd × (1-t)] + [E/(E+D) × Re]

  • E = 股東權益市值
  • D = 有息負債（銀行借款 + 公司債）
  • Rd = 稅後債權成本 = 利息費用 × (1-t) ÷ 有息負債
  • Re = 股權成本 = Rf + β × (Rm - Rf)（CAPM 模型）
  • t = 公司所得稅率

各組成部分詳解

1. 債權成本 Rd（稅後）

  • 銀行借款利率 + 公司債殖利率的加權平均
  • 為什麼要乘 (1-t)？因為利息可以抵稅，實際成本比名義利率低
  • 例如：借款利率 5%、稅率 20% → 稅後成本 = 5% × (1-0.2) = 4%

2. 股權成本 Re（CAPM）

  • Re = 無風險利率 + β × (市場報酬率 - 無風險利率)
  • 無風險利率：通常用美國 10 年期國債殖利率（約 4-5%）
  • β（Beta）：個股相對大盤的波動度
  • 市場風險溢酬（Rm - Rf）：歷史平均約 5-7%

3. 權重 D/(E+D) 和 E/(E+D)

  • 用市值計算，不是帳面價值
  • 高槓桿公司：債權比重大，WACC 可能較低（因為債權成本 < 股權成本）
  • 但槓桿太高會增加破產風險

實例計算

A 公司：

  • 有息負債 D = 1200 萬（銀行借款 800 萬利率 5%、公司債 400 萬利率 3%）
  • 股東權益 E = 500 萬
  • 稅率 t = 35%
  • 無風險利率 = 1.28%
  • 市場期望報酬 = 8%
  • β = 1

Step 1：計算 Rd

  • 利息費用 = 800×5% + 400×3% = 40 + 12 = 52 萬
  • Rd = 52 × (1-35%) ÷ 1200 = 33.8 ÷ 1200 ≈ 2.82%

Step 2：計算 Re

  • Re = 1.28% + 1 × (8% - 1.28%) = 1.28% + 6.72% = 8%

Step 3：計算 WACC

  • WACC = (1200/1700) × 2.82% + (500/1700) × 8%
  • = 0.706 × 2.82% + 0.294 × 8%
  • = 1.99% + 2.35%
  • ≈ 4.34%

→ DCF 估價時用 4.34% 作為折現率

WACC 高低代表什麼

WACC意義實例
低（3-5%）穩定低風險，未來現金流價值高台積電、中華電
中（6-8%）一般風險多數成熟企業
高（10%+）高風險，未來現金流要大打折新創、生技、景氣循環股

WACC 越高 → 折現後的內在價值越低
WACC 越低 → 折現後的內在價值越高

WACC 使用陷阱

1. Beta 不穩定：新公司歷史短，Beta 沒參考價值
2. 市場風險溢酬是估的：不同人用不同數字，結果差很多
3. 資本結構會變：今天 D/E 比率不等於明天的
4. 稅率變動：不同國家、不同年度稅率不同
5. 不要拿一家公司的 WACC 去估另一家：每家公司風險不同

四、Fed Model 股債比價模型

是什麼

比較股票的盈餘殖利率（Earnings Yield = 1/PE）與 10 年期公債殖利率，判斷股市整體是偏高還是偏低。

核心公式：Earnings Yield - 10Y Treasury Yield = 股債利差

三種判斷

情境比較判斷
盈餘殖利率 > 公債殖利率股票殖利率比債券高股票偏低，偏買
盈餘殖利率 = 公債殖利率兩者相當股市合理估值
盈餘殖利率 < 公債殖利率股票殖利率比債券低股票偏高，偏賣

實例計算

假設 S&P 500 遠期本益比 = 20，10 年期美國公債殖利率 = 4.5%

  • 盈餘殖利率 = 1/20 = 5%
  • 股債利差 = 5% - 4.5% = +0.5%

→ 股票只比公債多 0.5% 的殖利率，風險溢酬偏低，股市偏貴

如果本益比 = 15：

  • 盈餘殖利率 = 1/15 = 6.67%
  • 股債利差 = 6.67% - 4.5% = +2.17%

→ 股票溢酬充足，股市偏便宜

台股應用

Fed Model 原本是美國市場的概念，但同樣邏輯可以套用到台股：

  • 用台股指數的遠期本益比倒數當盈餘殖利率
  • 用台灣 10 年期公債殖利率當基準
  • 當台股盈餘殖利率大幅高於公債殖利率時，偏向買股
  • 當利差縮小甚至翻負時，偏向買債

Fed Model 的五大問題

問題 1：只在特定時期有效

  • 歷史回測只在 1921-1928 和 1987-2000 有效
  • 其他大部分時間，股債殖利率的相關性很弱
  • 2000 年後幾乎完全失效

問題 2：名義利率 vs 實質利率

  • Fed Model 比較的是「名義」盈餘殖利率和「名義」公債殖利率
  • 沒有考慮通膨因素
  • 高通膨時期，名義盈餘可能被通膨灌水，造成誤判

問題 3：永遠給出買進訊號

  • 幾乎從不顯示股市偏貴
  • 被華爾街券商用來推銷股票
  • 2020 年 12 月 Fed 主席 Powell 也用 Fed Model 為高估值辯護

問題 4：只適用大盤，不適用個股

  • Fed Model 只能判斷整體股市估值
  • 無法用來判斷個股是否值得買

問題 5：低利率環境失靈

  • QE 時代公債殖利率被壓到極低
  • 盈餘殖利率幾乎一定高於公債殖利率
  • 模型永遠顯示「股票便宜」，失去判斷意義

改良版：實質利差模型

Fed Model 的改良版，考慮通膨因素：

實質盈餘殖利率 = 名義盈餘殖利率 - 預期通膨率
實質公債殖利率 = 名義公債殖利率 - 預期通膨率
實質利差 = 實質盈餘殖利率 - 實質公債殖利率

這個版本在高通膨時期判斷更準確。

Fed Model 實戰定位

用途適合不適合
大盤長期配置方向✅-
個股估值-❌
短線擇時-❌
股債資產配置比例✅ 參考用-
精確進出場點位-❌

結論：Fed Model 是大方向參考工具，不是精準估值器。當股債利差極端時（例如超過 ±2%），才有比較大的參考價值。平時利差不大時，不用太認真。

心得

今天一次搞定了估值三大支柱：DDM、WACC、Fed Model。這三個剛好是估值的三個層次——

DDM 是最底層的：用股利折現來算合理價，簡單粗暴，但只能用在穩定配息的公司。戈登公式的核心就是 D₁/(r-g)，看起來簡單，但 g 的估計差 1% 就能讓合理價差 25% 以上，這就是為什麼安全邊際那麼重要——你估不準的東西，就得留餘地。

WACC 是中間層的：算出客觀的折現率，讓 DCF 估值有根據。債權成本 × (1-t) 這個稅盾效果很實用，高稅率的公司舉債反而比股權便宜。但 CAPM 算出來的 Re 也是估的，Beta 和市場風險溢酬都會變，WACC 本身就有誤差。

Fed Model 是最宏觀的：看股債之間的相對吸引力，決定大方向。但它最大的問題是——幾乎永遠說「股票該買」，這跟華爾街的利益一致，跟散戶的風險意識不一致。低利率時代它就徹底失靈了。

三個模型都有共同的問題：你輸入垃圾，就輸出垃圾。DDM 的 g、WACC 的 β 和風險溢酬、Fed Model 的通膨調整，全部都是估計值。所以估值不是算一個數字，而是算一個區間，然後用安全邊際來保護自己估計錯誤的風險。

戈登公式反推的那個版本 r = D₁/P₀ + g 其實很好用——不用估內在價值，只要看市場價格隱含的預期報酬率是多少，就能判斷目前股價是否合理。這個思路比硬算合理價更實用。

一、Piotroski F-Score（皮爾托斯基分數）— 基本面體檢的九項快篩

是什麼

F-Score 是芝加哥大學教授 Joseph Piotroski 提出的選股指標，用9個基本面條件替股票打分數，每符合一項得1分，總分0~9分。專門用來從低本益比、低股價淨值比的「價值股」中，篩出基本面真正好的公司。

核心邏核心邏輯：單看一個指標（如低PE）容易被騙，但9個面向同時過關的公司，體質差的機率很低。

九大指標詳解

【獲利性】4項

1. ROA > 0：資產報酬率為正 → 公司最基本的賺錢能力
2. 今年ROA > 去年ROA：獲利能力沒衰退，最好還在成長
3. 營業現金流 > 0：公司真的有現金進帳，不是帳面好看
4. 營業現金流 > 稅後淨利：現金比會計利潤多 → 獲利品質好，不是靠應計項目灌水

【安全性】3項

5. 長期負債/總資產 比去年下降：財務槓桿在降，風險在減
6. 流動比率比去年上升：短期償債能力變強
7. 去年沒有發行新股：公司沒有靠稀釋股權籌錢 → 現金流沒有吃緊

【成長性】2項

8. 毛利率比去年上升：成本控制能力在進步，或產品有定價權
9. 資產週轉率比去年上升：用同樣的資產創造更多營收，營運效率提升

怎麼做：F-Score計算SOP

步驟一：取得今年度和去年度的財報數據

  • 資產報酬率（ROA）
  • 營業現金流
  • 稅後淨利
  • 長期負債/總資產比率
  • 流動比率
  • 股數變動
  • 毛利率
  • 資產週轉率

步驟二：逐項判斷，符合給1分，不符合0分

步驟三：加總得到F-Score

6~7基本面良好✅ 可以關注
8~9基本面極強✅✅ 值得深入研究，特別是股價偏低時

F-Score的原始論文發現

Piotroski 的2000年論文發現：

  • 在低PBR的價值股中，F-Score高的股票年報酬率比F-Score低的高出7.5%
  • F-Score 8~9分的低PBR股票，3年持有期平均年報酬約15%
  • F-Score 0~1分的低PBR股票，3年持有期平均年報酬約-5%

F-Score的最佳使用場景

最適合：從低本益比、低股價淨值比的「便宜股」中，篩出基本面真正好的
不適合：高成長股（本來ROA就低、負債就高、現金流就不穩）

怎麼做（實戰SOP）：
1. 先用低PE或低PBR篩出一批「看起來便宜」的股票
2. 用F-Score幫每家公司打分數
3. 只保留F-Score ≥ 6的標的
4. 再用產業分析、護城河判斷進一步篩選

F-Score的五大限制

1. 單獨使用效果有限：市場先生回測台股，F-Score選股長期只略優大盤，不是大勝
2. 產業差異大：金融股、科技股、循環股的F-Score分佈不同，不應跨產業直接比較
3. 只看方向不看幅度：ROA從1%變2%和從10%變20%都得1分，但意義天差地遠
4. 不能精確擇時：F-Score告訴你體質好不好，不告訴你現在該不該買
5. 分數會變動：去年的9分今年可能變5分，必須每年重新計算

二、EVA經濟附加價值（Economic Value Added）— 真正為股東創造了多少價值？

是什麼

EVA = 稅後淨營業利潤（NOPAT）− 投入資本 × 加權平均資本成本（WACC）

告訴你：公司扣除所有資本成本（包括股權成本！）後，真正為股東創造了多少價值。

為什麼比傳統指標好：

  • EPS和ROE沒有扣除股權成本，一家ROE 8%的公司，如果股東期望10%的報酬，其實是在摧毀股東價值
  • EVA把股權的機會成本算進去，只有EVA為正的公司，才是真正在幫股東賺錢

公式拆解

EVA = NOPAT − TC × WACC

  • NOPAT（Net Operating Profit After Tax）：稅後淨營業利潤
  • TC（Total Capital）：投入資本總額
  • WACC：加權平均資本成本

怎麼做：EVA計算SOP

步驟一：計算NOPAT
1. 從損益表找到EBIT（營業利益）
2. EBIT × (1−有效稅率) = NOPAT
3. 如果有遞延稅款變動，要加回

步驟二：計算投入資本TC
1. 股東權益 + 有息負債（銀行借款+公司債）
2. 或總資產 − 無息流動負債（應付帳款、應付費用等）

步驟三：計算WACC（前面已學過）

步驟四：計算EVA = NOPAT − TC × WACC

實例計算

台積電假設數據：

  • EBIT = 8,000億
  • 稅率 = 20%
  • 股東權益 = 20,000億
  • 有息負債 = 3,000億
  • WACC = 8%

Step 1：NOPAT = 8,000 × (1−0.2) = 6,400億
Step 2：TC = 20,000 + 3,000 = 23,000億
Step 3：資本成本 = 23,000 × 8% = 1,840億
Step 4：EVA = 6,400 − 1,840 = 4,560億（正值，大幅為股東創造價值）

某面板廠假設數據：

  • EBIT = 200億
  • 稅率 = 20%
  • 股東權益 = 3,000億
  • 有息負債 = 4,000億
  • WACC = 10%

Step 1：NOPAT = 200 × 0.8 = 160億
Step 2：TC = 3,000 + 4,000 = 7,000億
Step 3：資本成本 = 7,000 × 10% = 700億
Step 4：EVA = 160 − 700 = −540億（負值，正在摧毀股東價值）

EVA判讀標準

EVA意義判斷
EVA > 0 且逐年上升公司持續為股東創造價值✅ 最佳投資標的
EVA > 0 但逐年下降創造價值的能力在衰退⚠️ 需要關注原因
EVA = 0公司剛好打平資本成本⚠️ 不創造也不摧毀價值
EVA < 0公司在摧毀股東價值🚨 除非有明確轉機，否則避開

EVA vs 會計利潤 — 為什麼ROE高不等於為股東賺錢？

關鍵差異：會計利潤不扣除股權資本成本

實例：

  • A公司：ROE 12%，但股東期望報酬率15% → EVA為負，看似賺錢實則虧
  • B公司：ROE 8%，但股東期望報酬率5%（低風險公用事業） → EVA為正，真正為股東創造價值

結論：ROE > WACC 才是真正為股東創造價值。ROE很高但WACC更高 = 假繁榮。

EVA的四大會計調整

思騰思特公司建議對會計數字做160多項調整，最常見的4項：

1. 研發支出資本化：會計上研究階段費用化，但EVA認為研發是長期投資，應資本化後逐年攤銷
2. 商譽不攤銷：IFRS已不再攤銷商譽，EVA贊同這個做法
3. 遞延稅款調整：把遞延所得稅加入資本，避免稅務策略扭曲真實績效
4. 營業租賃資本化：把租賃當作隱性負債，加入資本計算

投資人簡化做法：不用做160項調整，最關鍵的是把研發支出資本化 + 把營業租賃資本化，這兩項影響最大。

MVA（市場附加價值）— EVA的市場對應指標

MVA = 公司市值 − 投入資本總額

MVA是市場對公司未來EVA的折現預期：

  • MVA > 0：市場預期公司未來會持續創造正EVA
  • MVA < 0：市場預期公司未來EVA為負（摧毀價值）

EVA vs MVA 的關係：

  • EVA是當期已實現的價值創造（落後指標）
  • MVA是市場對未來EVA的預期（領先指標）
  • EVA持續為正 → MVA通常為正
  • EVA持續為負 → MVA可能為負

EVA估價法 — 用EVA算內在價值

公司內在價值 = 投入資本 + 未來EVA的折現值

公式：
> V = TC + Σ[EVAₜ / (1+WACC)ᵗ] (t=1~∞)

這跟DCF的邏輯一樣，只是用EVA取代FCF：

  • DCF：V = Σ[FCFₜ / (1+WACC)ᵗ]
  • EVA：V = TC + Σ[EVAₜ / (1+WACC)ᵗ]

兩者數學上等價，但EVA法的好處是：

  • 直接看出公司是在創造還是摧毀價值
  • EVA為正的公司，內在價值一定大於投入資本
  • EVA為負的公司，內在價值小於投入資本

EVA的優缺點

優點缺點
扣除所有資本成本，反映真實經濟利潤計算複雜，需要多項會計調整
解決ROE沒扣除股權成本的盲點WACC本身是估計值，誤差會傳導到EVA
可以用來評估任何公司（不限於穩定配息）歷史EVA不代表未來EVA
EVA上升趨勢是股價上漲的最根本驅動力不適合短期交易，更適合長期投資判斷
高盛、所羅門美邦等華爾街機構推崇一般散戶計算門檻高

三、F-Score × EVA 雙指標實戰組合

為什麼要搭配使用

F-Score告訴你：公司基本面「方向」對不對（9項有幾項改善）
EVA告訴你：公司「真正」賺不賺錢（扣除資本成本後有沒有剩）

F-Score高分 + EVA為正 = 最強組合

EVA > 0（創造價值）EVA < 0（摧毀價值）

今天學了兩個互補的基本面工具：F-Score是「快速體檢」，EVA是「深度驗血」。

F-Score最棒的地方是簡單粗暴——9個條件，每個1分，加起來就是總分。不需要估算未來、不需要假設折現率，純粹看過去一年基本面有沒有改善。Piotroski的論文數據也很清楚：在低PBR的價值股中，F-Score高的年報酬比F-Score低的高出7.5%——這不是小數字。

但市場先生的回測也提醒了我：F-Score單獨使用只略優大盤，不是什麼神奇公式。問題出在哪？出在F-Score只看「方向」不看「幅度」。ROA從1%變2%和從10%變20%都得1分，但後者的重要性是前者的十倍。所以F-Score是篩選工具，不是決策工具——用來快速排除爛公司，而不是用來精確選股。

EVA的概念讓我大開眼界。以前只知道ROE高就好，但EVA告訴我：ROE 12%但如果股東期望15%，你其實在虧錢。這就像一家餐廳營業額100萬，但租金+食材+人事成本110萬——營業額看起來不錯，實質在燒錢。可口可樂、IBM、沃爾瑪這些大企業都在用EVA管理體系，不是沒道理的。

EVA跟DCF數學上等價，但EVA的直覺更強——直接告訴你「這家公司是在創造還是摧毀價值」。DCF算出來一個內在價值數字，你還要跟股價比才知道便不便宜；EVA算出來正的就是好、負的就是壞，判斷門檻低很多。

一、Graham Number 葛拉漢數值

是什麼

Graham Number 是價值投資之父班傑明·葛拉漢提出的股價合理上限估算公式，綜合本益比（PE）和股價淨值比（PB）兩個指標，幫助防禦型投資者判斷「這檔股票最貴應該值多少錢」。

公式

Graham Number = √(22.5 × EPS × BVPS)

其中：

  • 22.5 = PE上限15 × PB上限1.5
  • EPS = 每股盈餘
  • BVPS = 每股淨值（建議用有形淨值，排除商譽和無形資產）

推導邏輯

葛拉漢認為防禦型投資人不應買：

  • PE > 15 倍的股票
  • PB > 1.5 倍的股票

所以：PE × PB ≤ 15 × 1.5 = 22.5

代數推導：

P/EPS × P/BVPS ≤ 22.5
P² / (EPS × BVPS) ≤ 22.5
P ≤ √(22.5 × EPS × BVPS)

怎麼用

1. 計算 Graham Number：找到公司的EPS和BVPS，套入公式
2. 與市價比較：

  • 市價 < Graham Number → 股價被低估，值得關注

3. 搭配七項過濾標準（見下方）

實例

假設某公司：

  • EPS = 3.5 元
  • BVPS = 25 元

Graham Number = √(22.5 × 3.5 × 25) = √(1968.75) = 44.37 元

如果市價 35 元 → 低於 Graham Number，可能被低估
如果市價 55 元 → 高於 Graham Number，偏貴

葛拉漢防禦型投資七項標準

項目條件說明
1. 規模年營收 > 市場平均避免小公司風險
2. 財務強度流動比率 > 2、長期負債 < 淨流動資產免於破產
3. 獲利穩定性過去10年無虧損穩定獲利
4. 股利紀錄連續20年配息一致的股利政策
5. 獲利成長10年EPS成長 > 1/3獲利成長性
6. 本益比PE ≤ 15不買太貴
7. 股價淨值比PB ≤ 1.5不買太貴

第6項和第7項的乘積就是 Graham Number 的核心。

NCAV 淨流動資產價值（更保守的估價法）

葛拉漢還有一個更激進的估價方法——NCAV（Net Current Asset Value）：

NCAV = 流動資產 - （總負債 + 特別股）
每股NCAV = NCAV ÷ 總股數

買進條件：在 NCAV 的 66%（2/3）價格買入

歷史回測：1930-1932年間，以NCAV 66%買入，30年平均年報酬 20%

不適用產業：資產少的公司（軟體、服務業）、景氣循環股

五大限制

1. 只適用EPS為正且有形淨值為正的公司——虧損公司不能用
2. 不包含成長假設——對成長型公司效用不大
3. 景氣循環股不適用——建議用3-5年平均常態化數字
4. 無形資產多的公司會被低估——軟體、服務業
5. PE×PB≤22.5只是概測法——不是鐵律，要依市場調整

二、IRR 內部報酬率

是什麼

IRR（Internal Rate of Return）是讓一項投資的淨現值（NPV）= 0 的折現率，代表這項投資的年化報酬率。

核心概念

NPV = 0 時的折現率 = IRR

意思是：所有現金流入的現值 = 所有現金流出的現值，這個折現率就是IRR。

計算方式

IRR 無法用公式直接求解，必須用逼近法（迭代法）計算，Excel 有內建函數：

  • IRR()：等間隔現金流
  • XIRR()：可指定每筆現金流的日期，直接算年化報酬率

實例

投資10萬元，5年後回收20萬：

  • 單筆投入 → 年化報酬率 = 7.2%
  • 每年投入1萬（定期定額）→ IRR = 12.3%

同樣10萬投入、20萬回收，定期定額的IRR比單筆高很多，因為每筆資金投入時間不同。

IRR vs 其他報酬率指標

指標適用情境特點
ROI單筆投入單筆回收不考慮時間
CAGR最初投入最終回收年均複合成長
IRR多筆不同時間現金流考慮時間價值
TWRR基金經理人績效排除資金進出影響
MWRR投資人實際報酬同IRR

IRR 三大缺點

1. 多重IRR問題：現金流正負交替多次時，可能出現多個IRR，失去判斷意義
2. 再投資假設不合理：IRR假設所有現金流入都以IRR再投資，但實際上可能只能放定存
3. 短期高IRR不代表好投資：IRR高但總獲利可能很少

IRR + NPV 搭配使用

條件解讀
IRR > 資金成本，NPV > 0✅ 值得投資
IRR > 資金成本，NPV ≈ 0⚠️ 邊際投資，需觀察風險
IRR < 資金成本，NPV < 0❌ 不值得投資

IRR判斷效率，NPV判斷價值，兩者搭配最準確。

三、MIRR 修正內部報酬率

為什麼需要 MIRR

IRR 假設所有正現金流都以 IRR 再投資，這太樂觀了。實務上：

  • 收回的現金可能只能放定存（2-5%）
  • 不可能每筆資金都以20%再投資

MIRR 修正了這個問題：正現金流用實際再投資利率計算、負現金流用資金成本折現。

公式邏輯

MIRR = (FV正現金流 / PV負現金流)^(1/n) - 1
  • FV正現金流：所有收入以再投資利率計算到期末的終值
  • PV負現金流：所有支出以資金成本折現到第0期的現值
  • n：投資期間

實例

投資案：初期投入 -1,000,000，3年回收 400,000、500,000、300,000

假設IRRMIRR（再投資8%，資金成本6%）
結果13.1%10.8%

MIRR 比 IRR 保守，更接近真實報酬率。

Excel 計算

=MIRR(現金流範圍, 資金成本, 再投資利率)
=MIRR({-10000, 400, 500, 300}, 6%, 8%)

IRR vs MIRR 比較

項目IRRMIRR
再投資假設以IRR再投資（過度樂觀）以實際利率再投資
多重解問題現金流正負交替時可能有多個解永遠只有一個解
資金成本不考慮負現金流用資金成本折現
結果偏高（樂觀）較保守（貼近現實）
適用情境簡單現金流、初步評估複雜現金流、精確決策

投資決策實務建議

1. 簡單投資（單筆進出）：用 CAGR 或年化報酬率即可
2. 多筆現金流（儲蓄險、定期定額）：用 IRR 初步判斷
3. 精確評估（企業投資、不動產）：用 MIRR + NPV 雙重確認
4. 門檻報酬率：IRR/MIRR > WACC 才值得投資

心得

Graham Number 看似簡單（就是 √(22.5×EPS×BVPS)），但背後的邏輯很清楚：PE和PB的乘積不超過22.5，等於同時限制了估值倍數和淨值倍數，是一種雙重安全邊際。

最值得學的是葛拉漢的七項標準——不是單看一個指標就買，而是規模、財務強度、獲利穩定性、股利紀錄、成長性、估值全部過關才出手。這跟之前學的Piotroski F-Score九大指標異曲同工，都是多面向篩選。

NCAV更是狠——直接在淨流動資產打66折買入，等於免費拿到固定資產。這在台股可能機會不多，但危機入市時值得掃描。

IRR/MIRR這組比較實用。IRR最大的坑就是「再投資假設」——假設所有回收的錢都能以同一報酬率再投入，這在現實中根本不可能。MIRR修正了這點，用實際再投資利率計算，結果更保守也更真實。

實戰建議：

  • 評估儲蓄險、定期定額用IRR就夠了（XIRR更好，可指定日期）
  • 評估企業投資案或不動產，用MIRR+NPV雙確認
  • Graham Number當快速篩選器，通過的再深入研究，不要當唯一判斷
