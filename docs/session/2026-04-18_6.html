本益比河流圖（PE River Chart）——圖像化估值利器

是什麼：把不同倍數的本益比 × EPS 畫成多條線，形成像河流一樣的帶狀圖，讓你一眼看出目前股價在歷史本益比區間的相對位置，判斷是高估還是低估。

怎麼做：

1. 計算河流上下緣

  • 河流上緣股價 = EPS × 歷史最高本益比

2. 判讀三步驟

  • 看位置：股價落在河流偏下緣=低估，偏上緣=高估

3. 選股實戰用法

  • 股價在綠色區間（低估→合理）→ 關注買進機會

實例：

  • 台積電（2330）2025年第20週近四季EPS=45.25元
  • 歷史本益比區間13倍~23倍
  • 13倍本益比股價=588元（河流下緣）
  • 23倍本益比股價=1,041元（河流上緣）
  • 當時股價999元→本益比約22倍→落在紅色與橘色區間之間→相對歷史偏高

適用時機：

  • ✅ 適用：獲利穩定的成熟股、產業龍頭、穩定成長股（如台積電、中華電）
  • ❌ 不適用：景氣循環股（本益比與股價反向）、獲利不穩定股、快速成長股
  • 注意：河流上下緣會隨時間更新，歷史區間是動態的，不能當絕對規則

常見陷阱：

  • 景氣循環股在獲利高峰時本益比反而低（盈餘衰退前夕），河流圖下緣反而是賣點
  • 河流圖的區間會被「最佳化」，歷史最高本益比創新高時整張圖會重畫，不能把最低格當絕對買點
  • 不同平台畫法不同（Goodinfo用週線、財報狗用月線、富果用5年區間6等分），結論可能不同

查詢工具：

  • Goodinfo：https://goodinfo.tw/StockInfo/ShowK_ChartFlow.asp
  • 財報狗：https://statementdog.com（搜尋股票代號→PE Band）
  • 玉山證券富果帳戶：內建本益比河流圖（5年區間、6色分區）

現金流量表實戰判讀——看穿公司「真有錢」還是「帳面賺」

是什麼：記錄企業在一段時間內「實際現金進出」的報表。損益表告訴你「帳面賺多少」，現金流量表告訴你「真正收到多少錢」。巴菲特最重視的報表就是這張。

怎麼做：

1. 三大現金流拆解

  • 營業活動現金流（OCF）：本業真正賺進來的現金，正數且穩定增長=健康

2. 自由現金流（FCF）計算與判讀

  • FCF = 營業活動現金流 + 投資活動現金流

3. 三大關鍵比率

  • 現金流量允當比率 = 近5年平均OCF / 近5年平均（資本支出+存貨增加+現金股利）

4. 損益表 vs 現金流量表交叉驗證

  • EPS成長但OCF下降→假賺錢，可能應收帳款暴增

實例：

  • 台積電2024年現金及約當現金餘額2.12兆，佔總資產40%
  • OCF持續強勁+適度投資活動流出（擴廠）+籌資活動流出（配息）=成熟穩健型現金流模式
  • 若某公司OCF連續3年為負+靠籌資活動（舉債、增資）維持→倒閉風險高

適用時機：

  • 每季財報公布時必查，確認公司「真有錢」
  • 特別適用：高負債公司（確認償債能力）、成長股（確認擴張有實質現金支撐）、配息股（確認股利來自本業而非借錢）
  • 間接法編制，不用搞懂每個回推項目，抓三大現金流的趨勢和正負即可

不同生命週期的現金流特徵：

  • 初創期：OCF負、ICF大負（投資擴張）、FCF正（靠募資）→靠外部資金活
  • 成長期：OCF轉正但不大、ICF仍負（持續擴張）、FCF波動大→開始自給自足
  • 成熟期：OCF穩定正、ICF適度負、FCF負（穩定配息）→最健康的模式
  • 衰退期：OCF下降、ICF可能轉正（變賣資產）、FCF波動→危險信號

心得

本益比河流圖真的是個好東西，比單看本益比數字直覺太多了。但最關鍵的認知是：河流圖是動態的，歷史最高最低本益比會跟著時間更新，所以你今天看到的「低估區」，三年後可能整條河都不一樣了。把它當參考工具，不要當交易系統的硬規則。

現金流量表最大的價值在於交叉驗證損益表。單看EPS很容易被騙——應收帳款暴增、折舊操弄、一次怔回沖，這些在損益表上都是漂亮的數字，但現金流量表會誠實告訴你：錢到底有沒有進來。一個簡單的判斷：OCF長期大於淨利的公司，獲利品質一定好。

以後看財報的順序應該是：先看現金流量表確認「真有錢」→ 再看損益表了解「賺多少」→ 最後看資產負債表確認「家底多厚」。現金為王不是喊假的。

指標背離——趨勢反轉的早期預警系統

是什麼：當技術指標（MACD、RSI、KD等）的走勢方向與價格走勢不一致時，稱為「背離」。背離是趨勢即將反轉或調整的重要信號，因為它代表市場的「內在動能」和「表面價格」已經脫節。

怎麼做：

1. 頂背離（高檔背離）— 賣出訊號：

  • 價格創新高，但技術指標沒有跟著創新高，反而下降

2. 底背離（低檔背離）— 買進訊號：

  • 價格創新低，但技術指標沒有跟著創新低，反而上升

3. 背離確認三步驟：

  • 第一步：價格出現明顯的 higher high 或 lower low

4. 背離的致命陷阱：

  • 假背離：背離出現後價格繼續原方向走，常見於強趨勢中

實例：

  • 情境：某股從100元漲到150元，又漲到160元創新高
  • 但RSI從75降到65（沒跟著創高）→ 頂背離
  • 操作：不要立刻做空，等出現長黑K或跌破短期均線確認後再進場
  • 停損：如果價格再次創新高並帶量突破，認錯出場

適用時機：趨勢末端的反轉判斷。不適合：強趨勢中途（連續背離很常見，太早進場會被輾過去）

四大指標背離的差異與選擇

是什麼：MACD、RSI、KD、量價四種背離各有特性，適用場景不同，選對指標做背離分析才能提高勝率。

怎麼做：

1. MACD背離 — 最穩定、最常用：

  • 看MACD柱狀圖（MACD - Signal）的峰/谷變化

2. RSI背離 — 最靈敏、但假訊號多：

  • 正背離（底背離）：價格創低，RSI不創低反而上升

3. KD背離 — 適合短線、反應快：

  • 正背離：價格創低，KD不創低

4. 量價背離 — 最真實的底層訊號：

  • 價漲量縮：上漲動能不足，多頭走不遠

實例：

  • 某股日K出現MACD頂背離 + 量價背離（價漲量縮）= 雙重確認賣訊，勝率大幅提升
  • 某股RSI出現底背離，但MACD沒有底背離 + 成交量持續萎縮 = 單一訊號，暫時觀望

適用時機：多重背離確認是最佳實務。至少兩個指標同時出現背離，勝率才夠高。單一背離太容易出現假訊號。

9種量價關係——高檔低檔位置決定一切

是什麼：量價關係不是只有「價漲量增=好、價漲量縮=壞」這麼簡單。同一種量價組合出現在低檔和高檔，意義可能完全相反。

怎麼做：

1. 價漲量增：

  • 低檔：健康上漲，買盤積極追價，後市看漲 ✅

2. 價漲量縮：

  • 漲勢初期+漲停鎖死：籌碼集中，多頭掌控 ✅

3. 價跌量縮：

  • 小幅回檔+量縮：正常洗盤，賣壓不大，支撐有效 ✅

4. 價跌量增：

  • 高檔：多頭獲利了結，波段跌勢開始 ⚠️

5. 價平量增：

  • 低檔：主力悄悄吸籌，待表態後上漲 ✅

6. 價平量縮：

  • 急漲後：籌碼沉澱期，主力等散戶離場後再攻 ✅

7. 價漲量平：

  • 上升力道減弱，後市看跌 ⚠️

8. 價跌量平：

  • 跌勢趨緩，可能進入盤整，但後續續跌機率高 ⚠️

9. 價平量平：

  • 市場死水，不要操作，等方向出來再說 🚫

實例（綜合判讀）：

  • 某股長期在80~95元區間盤整，某天價平量增（成交量暴增3倍但股價沒動）→ 主力吸籌
  • 一週後帶量突破95元壓力，價漲量增 → 確認主力表態，進場做多
  • 漲到120元後開始價漲量縮 → 背離訊號，開始減碼
  • 某天價跌量增+長黑K跌破115元 → 主力出貨確認，全數出場

適用時機：每天看盤必備的基本判斷框架。特別是「位置決定意義」這個核心觀念——同一種量價組合在高檔和低檔意義完全不同。

量價背離的進階應用——從背離到反轉的時間框架

是什麼：量價背離出現後，不是馬上反轉。理解從背離到反轉的過程，才能精準抓到進出場時機。

怎麼做：

1. 背離三階段：

  • 階段一（背離出現）：價量走勢分歧，警訊出現，但不動作

2. 量先價行原則：

  • 成交量的變化通常領先價格1~3天

3. 多指標背離共振：

  • 最強訊號：量價背離 + MACD背離 + RSI背離 同時出現

實例：

  • 情境：某股在160元，RSI出現頂背離（價格創高但RSI下降），同時量價背離（價漲量縮），MACD柱狀圖也在下降
  • 判斷：三重背離共振，反轉機率極高
  • 操作：不立刻做空，等出現長黑K跌破155元支撐線 → 進場做空或清多
  • 停損：站回160元以上
  • 若只出現RSI背離，但量和MACD都正常 → 單一訊號，觀望為主

適用時機：波段操作的反轉判斷。多指標背離共振的勝率遠高於單一背離。

心得

背離這東西，知道的人多，用得好的人少。最大的問題是「太早進場」——看到背離就興奮地反手做，結果趨勢繼續走，你被輾過去。背離是「預警」，不是「進場訊號」。從背離到真正反轉，中間可能還有幾天甚至幾週的差距。正確做法是：背離出現→提高警覺→等確認K線→再行動。

量價關係最核心的認知是「位置決定意義」。價跌量增在高檔是死亡訊號，在低檔是底部訊號。很多人背了9種量價關係卻忽略了位置判斷，結果在錯誤的位階做出錯誤的解讀。

多重背離共振是今天最重要的收穫。單一背離勝率不夠高（可能只有50-60%），但如果量價背離+MACD背離+RSI背離同時出現，反轉機率可以拉到80%以上。這就像多個證人同時指認同一個嫌疑犯，可信度自然大增。

最後一個體悟：量先價行是真的。成交量的異常變化（暴增或萎縮）通常比價格反轉早1-3天。所以看盤的時候，如果量能出現異常，就算價格還沒動，也要開始警戒了。

波段操作完整框架——從選股到出場的系統化方法

是什麼：波段操作（Swing Trading）是介於當沖和長期投資之間的策略，持股數天到數週，目標是捕捉一段完整的中期趨勢波段。不需要盯盤，適合有正職的投資人。

怎麼做：

1. 波段選股四原則：

  • 趨勢性好：選擇有明確方向（多頭或空頭）的標的，盤整股不碰

2. 五種高勝率波段情境：

  • 升降息循環：美聯儲升息/降息週期持續半年到一年，做多美元或做空受影響標的，直到通膨觸頂或就業好轉

3. 進場時機判斷：

  • 趨勢跟隨法：股價帶量突破重要均線（如20MA或60MA）+ 均線開始上彎 → 進場做多

4. 出場時機判斷：

  • 停利：到達壓力區或預估目標價，觀察K線是否出現轉折訊號

實例：

  • 情境：2022年初美國開始升息週期
  • 操作：做多美元指數，因為升息週期通常持續半年以上
  • 出場：9月CPI年增率從9%開始下降，通膨觸頂回落，結束做多
  • 結果：美元指數從96漲到114，漲幅約15%
  • 如果做個股波段：台積電在200元盤整3個月後帶量突破210元，5MA > 20MA多頭排列確認，進場做多，目標看前波高點240元，停損設200元（盤整區下緣）

適用時機：有明確趨勢和基本面驅動力的標的。不適合：盤整震盪股、消息面主導的題材股、流通量太小的個股

波段操作停損停利五法——朱家泓40年實戰精華

是什麼：波段操作最大的敵人不是看錯方向，而是看錯了不認賠。朱家泓用近40年實戰經驗歸納出5種停損設定法和5種必須停損的情境，以及完整的停利邏輯。

怎麼做：

1. 五種停損設定法：

① K線停損法：

  • 做多：以進場當天K線最低點為停損點

② 趨勢停損法：

  • 回檔買上漲：停損設在轉折波低點

③ 均線停損法：

  • 以進出依據的均線設停損

④ 固定比例停損法：

  • 以買進價的2%~10%為停損點

⑤ 支撐壓力停損法：

  • 做多時：關鍵支撐跌破即停損

2. 五種必須停損的情境（絕對停損）：

  • 盤整區布局多單，股價跌破盤整區 → 立刻停損

3. 停利邏輯：

  • 短線：以靠近股價最近的壓力為初步獲利目標

實例：

  • 情境：你在200元突破盤整區時買進某股，設定5種停損法中最適合的
  • K線停損：進場當天最低198元，停損設198元
  • 趨勢停損：盤整突破買進，停損設突破點200元
  • 均線停損：20MA在195元，停損設跌破195元
  • 固定比例停損：200 × 5% = 10元，停損設190元
  • 支撐停損：盤整區下緣195元，停損設跌破195元
  • 選擇：波段操作用趨勢停損（200元跌破）或均線停損（195元），最保守也最合理
  • 停利：目標看前高220元，到達後觀察是否帶量突破，若量縮不過則出場

適用時機：每筆波段交易都必須在進場前就設定停損停利。不設停損進場等於沒有安全網走鋼絲。

價值投資的四層安全邊際——從葛拉漢到巴菲特的進化

是什麼：安全邊際（Margin of Safety）是價值投資的核心概念，但多數人只理解第一層（價格低於價值），不知道還有三層更重要的安全邊際。從葛拉漢到巴菲特，安全邊際的概念經歷了重大進化。

怎麼做：

1. 第一層：價差安全邊際（葛拉漢）：

  • 核心思想：用比內在價值便宜的價格買入，價差就是保護

2. 第二層：成長安全邊際（現代價值投資）：

  • 核心思想：公司未來的成長性才是真正的安全邊際

3. 第三層：能力圈安全邊際（巴菲特/蒙格）：

  • 核心思想：只投資自己深度理解的領域，超出能力圈就沒有安全邊際

4. 第四層：市場先生安全邊際（行為紀律）：

  • 核心思想：市場先生（Mr. Market）像鐘擺一樣在樂觀和悲觀之間擺盪，價值投資者利用這種非理性

實例：

  • 台積電1月21日高點673元
  • 從高點下殺20% = 538元（理論買點）
  • 實際最低到575元，538元買不到
  • 調整為下殺15% = 572元，在575元附近分批進場
  • 更精細做法：每跌8%買一點（673 → 619 → 569 → 523...）
  • 如果完全等不到：定期定額買入，用時間換取平均成本

適用時機：長期投資選股。不適合：短線交易（安全邊際需要時間發酵）、投機題材股（沒有內在價值可估算）

五位大師選股條件——從葛拉漢到巴菲特的量化篩選

是什麼：把五位價值投資大師的選股邏輯量化成具體的財務條件，可以直接用來篩選台股。

怎麼做：

1. 班哲明·葛拉漢——價值低估型：

  • 近5年EPS > 1元（連續穩定獲利）

2. 彼得·林區——成長股型：

  • 近5年平均負債比 < 30%（財務穩健）

3. 麥克·墨菲——研發創新型：

  • 近12季營業利益率平均 > 10%（成本控制佳）

4. 馬克·約克奇——高毛利穩定型：

  • 近12季毛利率平均 > 20%（定價能力強）

5. 華倫·巴菲特——護城河價值型：

  • 近12季毛利率平均 > 20%（市場護城河）

實例：

  • 以台積電為例，用巴菲特條件檢驗：
  • 結論：台積電是偉大的公司，但以巴菲特的嚴格標準（PB<1.2），多數時候不算便宜。巴菲特的條件更適合篩選被市場低估的傳產股或金融股

適用時機：每季財報公布後，用這些條件篩選台股，建立觀察名單。不適合：題材炒作股（財務條件通常不符合）、景氣循環股（獲利波動大，條件會失真）

價值投資進出場策略——買賣的紀律框架

是什麼：價值投資不是「買了就放著」，進出場同樣需要系統化的紀律。以下三種進場和四種出場策略，讓價值投資不只是選股，更是一套完整的操作框架。

怎麼做：

1. 三種進場策略：

① 分批買入（定時定額法）：

  • 將資金分為多批，按固定時間間隔（每月/每季）買入

② 估值買入（低本益比/低PB法）：

  • 當本益比低於歷史均值，或PB接近1時進場

③ 市場恐慌買入（逆向投資法）：

  • 市場恐慌拋售時，優質股被錯殺，是最好的買入時機

2. 四種出場策略：

① 分批賣出：

  • 達到目標價後分批出場（如每漲5%賣1/3）

② 再平衡出場：

  • 當某檔股票佔比超過目標配置（如超過30%），減碼回歸原比例

③ 達成目標出場：

  • 設定20-30%的獲利目標，達到後出場

④ 基本面惡化出場：

  • 公司基本面出現重大負面變化（獲利大幅下降、管理層問題、護城河消失）

實例：

  • 你在2020年3月疫情崩盤時用估值買入法買進台積電260元
  • 設定目標：獲利30%出場（目標價338元）
  • 2020年底台積電漲到450元，超過目標
  • 執行分批賣出：420元賣1/3，450元賣1/3，480元賣1/3
  • 平均賣出價約450元，獲利約73%
  • 如果不設紀律：可能450還想等500，結果2021年回跌到400就坐立難安

適用時機：所有價值投資的進出場決策。特別是基本面惡化出場，很多人因為「捨不得」而抱著爛公司不放，最後虧更多。

心得

這回合把本週最缺的兩塊補上了——波段操作的實戰框架和價值投資的完整體系。

波段操作最核心的認知是：波段操作不是「看技術指標進出」這麼簡單。它的本質是「捕捉有明確驅動力的中期趨勢」。升降息循環、產業技術突破、供需失衡——這些才是波段的根本，技術指標只是輔助確認進出場時機的工具。很多人反過來，只看技術指標不管基本面驅動力，結果在盤整市被來回停損。

朱家泓的停損五法很實用，特別是「絕對停損」五條規則。其中「跌幅超過10%絕對停損」和「逆勢單反轉立刻停損」這兩條，是保命的鐵律。很多人在虧損超過10%之後開始凹單，然後越凹越深。10%是紅線，沒有商量空間。

價值投資這塊，最深刻的認知是安全邊際的四層結構。多數人只知道第一層（買便宜），但巴菲特的進化是從「買便宜的菸蒂股」轉向「買有護城河的偉大公司」。第二層（成長安全邊際）才是現代價值投資的核心——你用現在的價格買到的是未來的成長空間。第三層（能力圈）更關鍵，不在你能力圈內的公司，再便宜都不要碰，因為你無法判斷它是否真的被低估。

五位大師的量化條件也值得收藏。巴菲特的條件最嚴格（PB<1.2），這在台股很多好公司都不符合（台積電PB通常3-5倍），說明巴菲特的純量化條件在台股需要調整。但核心邏輯不變：高毛利率、高ROE、穩定現金流、合理估值——這四個條件拿來篩台股，仍然能濾出一堆好公司。

最後一個體悟：價值投資的出場策略裡，「基本面惡化出場」是最容易被忽略但最重要的。很多人在基本面變壞時還在用「長期投資」安慰自己，結果抱著一家正在衰退的公司越虧越多。價值投資的「長期持有」前提是公司基本面持續健康，一旦基本面惡化，長期持有就變成長期套牢。

均線糾結——盤整結束前的最後信號

是什麼：當短、中、長期均線（如5MA、20MA、60MA）糾纏在一起，彼此距離極近、方向不明時，稱為「均線糾結」。這代表不論短線、中線、長線的投資人，持股成本都差不多，股價正在盤整等待方向選擇。均線糾結不是壞事，反而是一個潛在的大行情前兆——盤整越久，突破後的力道越大。

怎麼做：

1. 理解均線糾結的本質：

  • 均線 = 平均成本，均線糾結 = 所有人成本差不多

2. 兩種判斷方向的方法：

方法一：區間突破法

  • 當股價帶量突破盤整區上緣 → 偏多操作

方法二：順勢交易法（推薦）

  • 先看長期均線（如60MA或120MA）的方向判斷大趨勢

實例：

  • 情境：某股盤整4個月，5MA、20MA、60MA糾結在一起，但60MA持續向上
  • 判斷：長期趨勢向上，盤整結束後往上突破機率大
  • 操作：在盤整區下緣附近找買點進場，停損設在盤整區下緣下方
  • 結果：帶量突破後沿5MA上漲，一個月漲超過30%

適用時機：所有時間框架，但日K和週K效果最好。不適合：均線雖然接近但方向仍然明確（沒有真正糾結）的情況。

均線糾結突破的三道過濾——避免假突破

是什麼：突破均線糾結不代表一定會漲，「今日突破、明日跌破」的假突破是散戶最大的陷阱。三大過濾條件可以大幅提高真突破的勝率。

怎麼做：

1. 過濾條件一：打底確認

  • 用2個低點畫一條上升趨勢線

2. 過濾條件二：突破下降趨勢線

  • 用2個高點畫一條下降趨勢線

3. 過濾條件三：帶量突破

  • 突破當天成交量必須明顯放大（至少均量1.5倍以上）

實例：

  • 情境：某股5MA、20MA、60MA糾結，某天漲5%突破
  • 過濾1：前波低點80、85，底部墊高 ✅
  • 過濾2：突破前波高點95的下降趨勢線 ✅
  • 過濾3：當日成交量為月均量3倍 ✅
  • 三個條件全滿 → 真突破機率極高，進場做多
  • 反例：同樣突破，但量只有均量的80% → 假突破風險高，觀望

適用時機：每次看到均線糾結突破訊號時都必須過濾。不過濾直接進場的勝率大約只有50%，過濾後可提升到70%以上。

均線糾結進階選股——多方策略三招

是什麼：找到均線糾結的股票後，不是每檔都會變飆股。需要搭配其他指標過濾，篩出真正有潛力的標的。

怎麼做：

1. 第一招：避免KD/MACD背離的股票

  • 突破均線糾結時，如果KD或MACD出現頂背離 → 容易假突破

2. 第二招：挑選中長期均線走多的個股

  • 突破均線糾結時，如果月線（20MA）或季線（60MA）仍在下彎 → 中期趨勢偏空

3. 第三招：搭配籌碼面確認

  • 在均線糾結期間，觀察法人買賣超

實例：

  • 情境：旺矽（6223）3月均線糾結
  • 檢查1：突破時MACD同步上升，無背離 ✅
  • 檢查2：60MA持續向上 ✅
  • 檢查3：突破前投信連續買超 ✅
  • 三招全過 → 突破後沿5MA上漲，波段獲利超過20%

適用時機：篩選均線糾結突破股時的三道品質檢驗。三招全過的標的勝率最高，少一招就要降低部位或觀望。

均線糾結選股量化條件

是什麼：用具體的量化條件篩選均線糾結即將突破的股票，而不是憑感覺看圖。

怎麼做：

1. 基本篩選條件：

  • 股價大於週線（5MA）、月線（20MA）、季線（60MA）

2. 進階過濾條件（排除不良標的）：

  • 排除地雷股：Z-Score < 1.81

3. 加強訊號條件（提高勝率）：

  • 量能暴增：當日成交量 > 月均量3倍

4. 動能突破策略（短線飆股篩選）：

  • 股價創近20日/60日/120日新高

實例：

  • 篩選結果：某股股價85元，5MA=83、20MA=82、60MA=80，均線距離在5%以內
  • 當日漲幅3.5%，成交量為月均量4倍
  • 投信近5日連續買超
  • 60MA持續向上
  • 結論：符合均線糾結突破 + 帶量 + 法人買超 + 長期均線走多 → 高機率真突破

適用時機：每日收盤後用選股工具篩選，納入觀察名單，隔天盤中確認進場。

均線糾結的進場與出場策略

是什麼：找到均線糾結突破的標的後，進場和出場的時機與方式決定最終獲利。

怎麼做：

1. 兩種進場方式：

積極型：突破當下追價

  • 股價帶量突破前波高點時直接追進

保守型：回測確認後進場

  • 等股價有效突破後，拉回測試突破點或均價線

2. 出場策略：

停利：

  • tick流/一波流：股價在5~15分鐘內未繼續上漲、買單不連續、量能未放大 → 立即出場

停損：

  • 以進場當天帶量紅K的最低點為停損點

實例：

  • 情境：某股帶量長紅K突破均線糾結，進場價100元，當天最低97元
  • 積極進場：100元追進，停損設96元（前波高點下方1%）
  • 保守進場：等回測99元確認5MA支撐後買進，停損設96元
  • 停利：股價沿5MA上漲至120元，某天爆量（週轉率12%）但收長黑K → 主力出貨訊號，出場
  • 若股價跌破97元 → 假突破確認，停損出場

適用時機：所有均線糾結突破的交易。關鍵原則：進場前就設好停損，不要進場後才想停損設在哪裡。

均線糾結 + ATR 停損停利系統

是什麼：ATR（Average True Range，平均真實範圍）是衡量個股波動度的指標。用ATR設定停損停利，可以根據個股波動特性動態調整，而不是用固定的百分比。

怎麼做：

1. ATR基礎計算：

  • ATR通常取14天的平均真實範圍

2. ATR停損設定：

  • 長線操作：停損設在進場價 - 1/2 ATR

3. ATR移動停利：

  • 進場後，用均線（如40MA）作為移動停利線

4. 風險控制鐵律：

  • 不管停損設多遠，單筆虧損控制在本金2%以內

實例：

  • 情境：某股ATR=5元（14天平均日波動5元），進場價100元
  • 長線停損：100 - 5/2 = 97.5元（1/2 ATR）
  • 短線停損：100 - 5/4 = 98.75元（1/4 ATR）
  • 部位計算：本金100萬，2%風險=2萬
  • 移動停利：股價沿40MA上漲，某天收盤跌破40MA → 出場

適用時機：所有需要設定停損停利的交易。特別適合：不同波動度的標的需要差異化停損距離時。

心得

均線糾結這東西，我以前覺得就是「均線黏在一起很亂不知道看什麼」，現在才知道這反而是最值得關注的訊號。盤整越久，代表浮額洗得越乾淨，突破後的力道越大——「橫有多長，豎有多高」這句話不是唬人的。

幾個重要認知更新：

1. 順勢交易法比突破法更好：多數人等突破才進場，但其實看長期均線方向就可以提前在盤整區內找進場點。長期均線向上就在盤整區下緣找買點，向下就找空點，這樣突破後你已經在車上了，不用追高。但前提是要設好停損——盤整區下緣就是天然的停損點。

2. 假突破的三道過濾太重要了：打底+突破趨勢線+帶量，三個條件缺一不可。以前看到均線糾結+漲5%就想衝，結果隔天就跌回來，因為沒量也沒打底。帶量長紅K突破才是真突破的必要條件，沒量的突破就是主力製造的騙線。

3. KD/MACD背離是假突破的死亡交叉：突破均線糾結但KD或MACD頂背離，這種「價格創高但動能沒跟上」的組合，假突破機率非常高。反過來說，如果突破時KD和MACD同步創高，那真突破的勝率就大增。這跟之前學的「多重背離共振」是一體兩面——背離共振看空，指標共振看多。

4. ATR停損比固定百分比合理太多：台積電日波動1%用5%停損跟小型股日波動5%用5%停損，合理度天差地遠。用ATR動態調整停損距離，波動大的股票停損設寬一點，波動小的設窄一點，再配合本金2%風險控制來決定部位大小，這才是科學化的部位管理。跟之前學的凱利公式和蒙地卡羅模擬是同一套邏輯——先算風險再決定下注大小。

5. 暴量出場法值得記住：股價大漲後突然爆出歷史天量，週轉率超過10%，這就是主力在出貨的訊號。散戶常看到爆量就以為是「主力進場」，其實大漲後的爆量更可能是「主力出場」。

ROE杜邦分析——拆解公司獲利品質的X光機

是什麼：ROE（股東權益報酬率）告訴你公司用股東的錢賺了多少，但單看ROE數字會誤判。杜邦分析把ROE拆成三個因子：淨利率、資產週轉率、權益乘數，讓你看出ROE高的「原因」是什麼——是真正會賺錢，還是靠借錢撐出來的。

怎麼做：

1. 杜邦公式拆解：

  • ROE = 淨利率 × 資產週轉率 × 權益乘數

2. 三大因子的意義與判讀：

淨利率高（每100元營收賺很多）：

  • 代表公司有定價能力和成本控制能力

資產週轉率高（每1元資產產生很多營收）：

  • 代表公司營運效率高、市場佔有率大

權益乘數高（負債比高）：

  • 代表公司大量舉債來放大獲利

3. 七種杜邦分析組合判斷公司類型：

| 類型 | ROE趨勢 | 淨利率 | 週轉率 | 權益乘數 | 判斷 |
|------|---------|--------|--------|----------|------|
| A最佳 | 向上 | 向上 | 向上 | 提高 | 穩健高成長，三引擎齊發 |
| B次佳 | 向上 | 向上 | 向上 | 降低 | 獲利成長+去槓桿，最健康 |
| C好轉 | 向上 | 向上 | 平平 | 提高 | 獲利改善但槓桿增加 |
| D隱憂 | 向上 | 向上 | 向下 | 提高 | 獲利好但資產效率下降+槓桿增加 |
| E成熟 | 平平 | 緩下 | 向上 | 平平 | 獲利穩定但成長趨緩 |
| F轉差 | 向下 | 緩下 | 平平 | 平平 | 獲利開始衰退 |
| G危險 | 向下 | 向下 | 緩下 | 提高 | 三引擎都出問題+槓桿增加 |

  • 兩間ROE差不多，但川湖的品質遠優於全國電子

ROE杜邦分析的三大誤區——一知半解會吃大虧

是什麼：很多人看完ROE就以為自己懂了價值投資，結果踩進三個致命誤區。

怎麼做：

1. 誤區一：以為看ROE就是價值投資：

  • ROE是分析的起點，不是買賣指標

2. 誤區二：淨值減損會讓ROE虛高：

  • 淨值（股東權益）減少 → 權益乘數變大 → ROE看起來變高

3. 誤區三：不區分公司類型，用同一標準評判：

  • 高淨利率型公司（如台積電）和低淨利率高週轉型公司（如全國電子）不能放在一起比ROE

實例：

  • 情境：某公司ROE從15%跳升到25%，看起來很棒
  • 杜邦拆解：淨利率從5%降到4.5%，週轉率從1.5降到1.3，權益乘數從2倍升到4.3倍
  • 判讀：ROE提升完全來自槓桿翻倍（負債比從50%升到77%），獲利能力其實在下降
  • 結論：這不是好公司變更好，而是好公司開始借更多錢來維持ROE → 風險大增

適用時機：每次看到ROE數字時，都先做杜邦拆解再下結論。

資產負債表實戰判讀——看懂公司的「家底」

是什麼：資產負債表是公司的「財務健康報告卡」，記錄某一時間點公司擁有什麼（資產）、欠了什麼（負債）、股東真正擁有多少（股東權益）。公式：資產 = 負債 + 股東權益。

怎麼做：

1. 看資產結構——錢花在哪裡：

  • 流動資產（一年內可變現）：現金、有價證券、應收帳款、存貨

2. 看負債結構——欠了多少錢：

  • 流動負債（一年內要還）：應付帳款、短期借款、應付費用

3. 看股東權益——股東真正擁有多少：

流動比率與速動比率——公司會不會周轉不靈？

是什麼：流動比率和速動比率是衡量公司短期償債能力的兩個關鍵指標，判斷公司「會不會突然還不出錢」。

怎麼做：

1. 流動比率 = 流動資產 / 流動負債：

  • 意義：每1元短期負債，有多少元短期資產可以還

2. 速動比率 = 速動資產 / 流動負債：

  • 速動資產 = 流動資產 - 存貨（更嚴格的話再扣預付費用）

3. 為什麼要扣存貨：

  • 存貨是最難變現的流動資產

4. 產業特性影響判斷標準：

  • 輕資產公司（如Facebook/Meta）：流動和速動比率都很高，因為幾乎沒有存貨也沒有大量負債

5. 三大警訊：

  • 流動比率和速動比率同時下降趨勢 → 可能在「以短支長」（用短期借款做長期投資），極度危險

實例：

  • Costco速動比率0.45，遠低於100%標準 → 看起來危險？
  • 某製造業公司流動比率1.5、速動比率0.6 → 流動比率偏低、速動比率嚴重不足
  • 台灣高鐵：有足夠獲利能力，但債務壓力沉重導致周轉困難，最後債務重組

適用時機：篩選投資標的時必查。特別是高負債公司、景氣循環股、營建股，這兩個比率是避開地雷股的第一道防線。

損益表三率趨勢分析——毛利率、營利率、淨利率的交叉判讀

是什麼：損益表上最核心的三個利潤率：毛利率、營業利益率、淨利率。單看一期的數字意義有限，看趨勢變化才能判斷公司體質是在變好還是變壞。

怎麼做：

1. 毛利率 =（營收 - 營業成本）/ 營收：

  • 代表產品的定價能力和成本控制

2. 營業利益率 = 營業利益 / 營收：

  • 營業利益 = 營收 - 營業成本 - 營業費用（管銷費用、研發費用）

3. 淨利率 = 稅後淨利 / 營收：

  • 淨利率和營利率的差距 = 業外收支+稅

4. 三率趨勢交叉判讀六種情境：

情境一：三率同升（最佳）

  • 毛利率↑ 營利率↑ 淨利率↑ → 產品競爭力提升+費用控制良好

情境二：毛利率升、營利率降（警示）

  • 產品賺更多，但費用吃掉利潤 → 管理效率下降或過度擴張

情境三：毛利率平、淨利率暴增（陷阱）

  • 本業沒變好，但業外收入大增（賣地、賣股、匯兌收益）

情境四：毛利率降、營利率降（衰退）

  • 產品競爭力下降+費用失控 → 公司體質正在惡化

情境五：毛利率穩定、營利率穩定、淨利率穩定（穩健）

  • 公司營運穩定，沒有大驚喜也沒有大驚嚇

情境六：毛利率暴增暴降（循環股特徵）

  • 景氣循環股的毛利率會隨產品價格大幅波動

最震撼的認知是同樣的ROE，背後的品質天差地遠。川河和全國電子ROE都是24%左右，但川河是靠29%的淨利率驅動，全國電子是靠3.6%的淨利率加上3倍週轉率和2.16倍槓桿硬撐出來的。如果只看ROE數字就買全國電子，景氣反轉時高槓桿會讓ROE暴跌，而川河的高淨利率型ROE卻相對穩定。這就是杜邦分析的價值——不是告訴你ROE多少，而是告訴你ROE從哪裡來。

另外幾個重要收穫：

1. 淨值減損會讓ROE虛高：這個陷阱太陰險了。股東權益變小→權益乘數變大→ROE看起來變高，但其實公司沒有變得更會賺錢，只是股東的錢變少了。用杜邦分析一眼就能看穿：淨利率沒變、週轉率沒變、只有權益乘數在漲→這是假ROE。

2. 流動比率和速動比率的產業差異：Costco速動比率0.45看起來很危險，但它是「先收後付」模式，應付帳款等同於無息借款，根本沒有還款壓力。所以看比率不能只看數字，要看產業特性和商業模式。

3. 損益表三率交叉判讀比單一指標有用太多：毛利率上升但營利率下降，代表費用失控；毛利率穩定但淨利率暴增，代表業外收入不可持續。這些只用單一指標是看不出來的，必須三個一起看、看趨勢、看差距。

4. 景氣循環股要在毛利率最低點買、最高點賣：這和一般股票完全相反。一般股票是獲利好時買、獲利差時賣（追蹤趨勢），但循環股的獲利巔峰就是景氣高峰，接下來就是下坡路。這跟之前學的「本益比河流圖不適用於景氣循環股」是同一個邏輯——景氣高峰時EPS最高、本益比反而低（因為市場預期獲利會回落），這時候買就套在山頂。
