分價量表 Volume Profile 完整實戰指南

一、分價量表是什麼？

定義：分價量表（Volume Profile）是將成交量以「水平橫條」顯示在對應價格上的指標。傳統成交量告訴你「何時」有量，分價量表告訴你「何價」有量。

核心差異：

  • 傳統成交量 → 垂直柱狀圖，看「時間軸」的量
  • 分價量表 → 水平橫條圖，看「價格軸」的量

二、四大核心元素

1. 控制點 POC（Point of Control）

是什麼：整段價格區間內成交量最集中的那個價格點，橫條最長的那根。

怎麼看：

  • POC = 市場的「重心」或「共識價」
  • 買賣雙方在此價位交易最活躍，認同度最高
  • 像磁鐵一樣吸引價格靠攏

實例：

  • 情境：台積電在 900-950 區間盤整一個月
  • 操作：POC 落在 925，之後價格每次回測 925 都獲得支撐 → 925 就是主力成本區

注意：POC 不一定是最強支撐壓力，最強的常出現在 VAH/VAL（多空攻防轉換點）

2. 價值區 VA（Value Area）

是什麼：圍繞 POC，集中了總成交量 70% 的價格範圍。

怎麼用：

  • VAH（Value Area High）= 價值區高點
  • VAL（Value Area Low）= 價值區低點
  • 價格在 VA 內 = 市場平衡、盤整
  • 價格突破 VAH 或跌破 VAL = 新趨勢可能啟動

實例：

  • 情境：聯發科 VA 範圍 1300-1400
  • 操作：價格在 1300-1400 區間震盪 → 在 VAL（1300）附近買入，VAH（1400）附近賣出
  • 停損：VAL 下方 1-2%，約 1274-1287

3. 高成交量節點 HVN（High Volume Node）

是什麼：除了 POC 外，其他成交量突出的價格區域。

怎麼看：

  • 代表過去發生過密集換手 = 市場的「加油站」「休息區」
  • 價格回測 HVN → 容易盤整，形成支撐或壓力
  • 就像歷史戰役的重要據點，攻守雙方重兵佈防

實例：

  • 情境：鴻海在 165 有一個明顯的 HVN
  • 操作：股價從 180 回檔到 165 附近 → 觀察是否止跌，搭配 K 線反轉訊號進場
  • 停損：HVN 下方 2%，約 161.7

4. 低成交量節點 LVN（Low Volume Node）

是什麼：成交量非常稀疏的價格區域，圖表上像「峽谷」「真空帶」。

怎麼用：

  • 市場在這些價位沒停留，買賣雙方沒共識，價格快速穿越
  • 像滑過冰面一樣快速通過
  • LVN = 「不公平」的價格區
  • 價格進入 LVN → 容易加速突破或下跌

實例：

  • 情境：某股在 50-55 之間有 LVN（真空帶）
  • 操作：股價突破 55 後進入 LVN → 上方無明顯賣壓，容易快速拉升到下一個 HVN（如 60）
  • 停損：跌破突破點 55

三、三種分佈型態判讀

1. D 形分佈 — 平衡市

特徵：對稱鐘形曲線，像字母 D

怎麼做：
1. POC 位於中央，買賣勢均力敵
2. 在 VAL 買入、VAH 賣出（逆勢區間操作）
3. 密切關注價格往哪個方向突破 VA

適用時機：盤整市、箱型走勢

2. P 形分佈 — 軋空行情

特徵：成交量集中在下方，上方有細長尾巴

怎麼做：
1. 下方 HVN（P 的身體）= 強勁支撐
2. 初期大量買盤推升價格 → 快速拉升 → 上方追價不足
3. 價格回測下方 HVN = 加碼買點

適用時機：上漲趨勢中的回檔買點

3. b 形分佈 — 殺多行情

特徵：成交量集中在上方，下方有細長尾巴

怎麼做：
1. 上方 HVN（b 的身體）= 強大壓力
2. 高位大量換手（可能主力出貨）→ 快速下跌
3. 價格反彈至上方的 HVN = 放空點

適用時機：下跌趨勢中的反彈放空

四、三大實戰策略

策略一：價值區邊緣逆勢交易

怎麼做：
1. 確認市場處於盤整（D 形分佈）
2. 價格跌至 VAL → 等待止跌 K 線（錘子線、晨星）→ 買入
3. 價格漲至 VAH → 等遭遇阻 K 線（流星線、烏雲罩頂）→ 放空
4. 停損：VAL/VAH 下方/上方 1-2%
5. 目標：POC 或對向 VA 邊緣

實例：

  • 情境：某股 VA = 80-90，POC = 85
  • 買入：價格到 81 出現錘子線 → 買入，停損 79，目標 85-90
  • 放空：價格到 89 出現流星線 → 放空，停損 91，目標 85-80

策略二：LVN 真突破確認

怎麼做：
1. 找到 HVN 盤整區 + 上方有 LVN 真空帶
2. 股價帶量突破 HVN 上緣 → 進入 LVN
3. LVN 區域賣壓極小 → 價格容易快速拉升
4. 進場：突破後順勢追價，或等回踩不破再進
5. 停損：HVN 下方

實例：

  • 情境：某股在 100-105（HVN）盤整，105-115 為 LVN
  • 操作：帶量突破 105 → 順勢買入，目標看 115 以上（下一個 HVN）
  • 停損：104 以下

策略三：POC 動態支撐壓力

怎麼做：
1. 上升趨勢中：前一波盤整區的 POC = 回檔支撐
2. 下降趨勢中：前一波盤整區的 POC = 反彈壓力
3. 價格回測 POC 不破 + 再度放量 = 進場訊號
4. 停損：POC 破損 1-2%

實例：

  • 情境：台積電前波盤整區 POC 在 900，目前從 950 回檔
  • 操作：回測 900 不破 + 放量上攻 → 加碼買入
  • 停損：890 以下

五、分價量表與其他指標的黃金組合

指標組合應用方式效果
VP + 移動均線價格回測 50MA/200MA 且該位置 = HVN 或 POC雙重支撐，高勝率買點

VP + ATR用 ATR 突破策略 + VP 確認 POC 上方無重大阻力提高突破成功率

六、Volume Profile 三大市場定律

1. 回歸均值：價格偏離 POC 太遠 → 會回到 VA 區域
2. 回檔支撐：上升趨勢中，價格回到 VA/POC → 吸引買盤形成支撐
3. 反彈阻力：下跌趨勢中，價格反彈至 VA/POC → 投資者成本區賣壓

八、風險與限制

  • 滯後性：基於過去成交數據，突發事件可能使舊支撐壓力失效
  • 週期選擇：不同週期會產生完全不同的圖形，選錯週期 = 分析失準
  • 不能單獨使用：VP 是確認性工具，需搭配趨勢分析或型態分析

心得

分價量表就像拿到了市場的 X 光片 — 你不只看到價格在動，還看到背後的籌碼結構在哪裡堆積、哪裡是真空。以前只知道「這裡好像有支撐」，現在知道為什麼有支撐：因為這裡是 POC，是主力的成本區。

最實用的三個概念：
1. VAH/VAL 邊緣操作 — 盤整市的最實用工具，不用猜頭猜尾
2. LVN 真空帶 — 突破後進入 LVN = 無阻力區，容易加速，這個概念跟之前學的「假突破判斷」可以完美結合
3. P 形 / b 形分佈 — 一眼判斷多空結構，不用再糾結「現在是多還是空」

跟之前學的支撐壓力判斷、量價關係可以整合：分價量表提供了更精確的支撐壓力定位，不像傳統方法只能粗略畫線。以後選股分析可以加上 VP 確認關鍵價位。
