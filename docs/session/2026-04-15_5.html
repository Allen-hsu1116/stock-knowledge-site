一、借券賣出餘額實戰判讀

借券與借券賣出的關鍵區別

是什麼：借券（Security Lending）是將手上的股票借出去賺利息，也可以借別人的股票賣出來賺差價。但「借券」≠「借券賣出」——借入證券後不一定馬上賣出，借券成交不等於借券放空。

四個關鍵名詞：
1. 借券成交：當日投資人向券商借入的證券數量
2. 借券餘額：借入的證券尚未還回去的餘額（扣除前一日歸還的券）
3. 借券賣出：借券人借出股票後，把股票實際賣出了
4. 借券賣出餘額：每日借券賣出的累積量，扣掉前一日已回補買回的證券

怎麼用：
1. 借券賣出餘額減少 → 可能代表多頭訊號（空方回補）
2. 借券賣出餘額增加 → 可能代表空頭訊號（放空壓力增大）
3. 但要搭配外資持股數據綜合判斷——有時借券增加是外資避險，股價可能不跌反升

實例：

  • 萬海、陽明借券餘額雙暴增：陽明遭法人賣超且借券上升，反映市場對航運基本面擔憂；萬海借券增加但法人買超，呈現「多空雙壓」矛盾
  • 這種情況要進一步看：借券增加是避險操作還是看空？搭配外資持股方向判斷

借券 vs 融券差異實戰

是什麼：兩者都是「借股票來賣」的交易方式，但交易細節和風險不同。

怎麼選擇：

項目融券借券
保證金股價90%股價140%
交易方向單向（借來必須直接賣出）雙向（可借入也可借出）
用途只能賣出可賣出、履約、還券、避險
強制回補股東會/除權息必須回補不用強制回補，補償出借人權益即可
提前召回出借人不能提前召回出借人可提前召回（1~6個工作天）
適合操作散戶放空、短線做空法人避險、套利策略、中長期布局

實戰判斷：

  • 想短線放空 → 融券較方便（保證金低、操作直接）
  • 想做避險或套利 → 借券較彈性（不必馬上賣出、不受強制回補限制）
  • 遇到股東會前 → 融券必須強制回補可能引發軋空，借券不受影響

借券賣出餘額搭配外資持股判讀法

怎麼做：
1. 觀察個股借券賣出餘額變化趨勢（連續增加或減少）
2. 同步查看外資持股變化
3. 四種組合判讀：

  • 借券賣出餘額增 + 外資買超 → 外資避險（持股做多但借券放空避險），股價可能續漲

實例：

  • 情境：某檔股票借券賣出餘額連續5日增加，但外資同步買超
  • 判斷：外資在持有現股的同時借券賣出，是典型的避險操作（如持有現股+賣出看跌選擇權的保護性策略）
  • 操作：不必恐慌，反而可留意外資買超是否持續

適用時機：判斷借券增加是「真放空」還是「避險操作」，避免誤判訊號

二、融資斷頭潮系統性風險判斷

融資維持率計算與斷頭機制

是什麼：融資是向券商借錢買股票（上市股借6成、自備4成），當股價下跌使融資維持率低於130%，券商會發出追繳令，T+2日內未補錢就會被斷頭（強制賣出）。

怎麼算：

  • 融資維持率 = 股票現值 ÷ 融資金額 × 100%
  • 融資買進時維持率 = 166%（10萬÷6萬）
  • 追繳線 = 130%（股價跌到買進價的78%時觸發）
  • 斷頭簡算：買進價 × 0.78 = 追繳價格

實例：

  • 融資買進一張100元台積電，自備40萬、融資60萬
  • 維持率166%（100萬÷60萬）
  • 股價跌到78元 → 維持率78,000÷60,000=130%，收到追繳令
  • 也就是股價跌22%就會被追繳
  • 連續2根跌停（100×0.9×0.9=81元）就逼近追繳邊緣

追繳後三種應對

怎麼做：
1. 補到166%以上：撤銷追繳令，恢復正常狀態
2. 補到130%~166%：不會被斷頭，但追繳令仍在，之後只要盤中再跌破130%，必須「當日」補錢，否則隔天直接斷頭（不再給2天寬限期）
3. 不補錢：T+3日開盤被市價強制賣出，砍在阿呆谷

融資斷頭潮的形成與辨識

是什麼：當大量融資投資人在相近價格買進，股價大跌時這些人同時面臨追繳和斷頭，券商同一天拋售形成「多殺多」，就是融資斷頭潮。

怎麼判斷：
1. 融資使用率過高：個股融資使用率超過50%就要警戒，超過80%極度危險
2. 大盤融資餘額佔市值比：正常1.5%~2%，超過2.1%表示市場槓桿過高
3. 連續跌停：個股連續2根跌停就會觸發追繳，3根以上可能引發斷頭潮
4. 成交量異常爆增：下跌時成交量暴增3倍以上，可能是斷頭賣壓
5. 融資單日減少超過10%：斷頭潮正在發生的明確訊號

實例（2025年4月台股歷史事件）：

  • 4/7~4/9 三日重挫3,906點（-18.3%），創台股歷史最大三日跌幅
  • 融資三日蒸發750億元（-25%），單日最大減少330億元創歷史新高
  • 4/7成交量1,661億（恐慌窒息量）→ 4/8暴增至5,780億（斷頭量）→ 4/9維持5,900億
  • 融資餘額從3,000億降至2,250億，回到2022年水位
  • 融資比率從2.1%降至1.5%，回到歷史安全水位

融資斷頭潮三階段特徵

怎麼做：
1. 初始階段：高融資比例個股率先被追繳，市場選擇性賣壓
2. 加速階段：斷頭擴散至各類股，成交量爆增3倍以上，跌幅擴大
3. 尾聲階段：融資減少速度逐漸放緩（日減少<5%），高量換手結束

斷頭潮後的市場模式：
1. 技術性反彈：斷頭結束初期常出現2~5日反彈
2. 二次探底：反彈後常二次探底，但成交量和跌幅通常小於首次
3. W型底部：完整的斷頭後通常形成W底，第二個底通常高於第一個底

融資使用率排行榜避雷

是什麼：融資使用率不是全部流通股數的百分比，而是總流通股數25%上限中，實際由融資買進的比例。融資使用率超過50%意味著超過12.5%的流通股在融資，一旦下跌10%就可能引發融資逃難潮。

怎麼避雷：
1. 投資前先查該股融資使用率（Yahoo股市→籌碼→資券變化）
2. 融資使用率 > 50%：高風險，下跌時賣壓會特別重
3. 融資使用率 > 80%：極度危險，一有風吹草動就是死亡螺旋
4. 台積電融資使用率約0.4%，非常健康——高市值、法人為主的股票融資率通常很低
5. 虧損股、小型股的融資使用率通常偏高，這類股票要特別小心

實例：

  • 2025年3月台股融資使用率前10名最低也有68.92%，前3名超過80%
  • 這類股票下跌10%就可能觸發10%流通股的融資賣壓，引發人踩人的死亡螺旋

大盤融資水位判斷系統性風險

怎麼做：
1. 融資比率（融資餘額÷市值）：

  • > 2.0%：市場槓桿偏高，系統性風險上升

2. 融資減少速度：

  • 單日減少 > 10%：斷頭潮正在發生

3. 融資餘額與指數的關係：

  • 融資餘額高位 + 指數下跌初期 → 斷頭潮可能才剛開始，不要急著抄底

歷史實證：

  • 2020年疫情：融資減少高峰3/19，指數同日見底8,681點，一個月後上漲22.3%
  • 2015年中國股災：融資減少高峰8/24，指數8/25見底7,410點，一個月後上漲17.5%
  • 2008年金融危機：融資減少高峰9/16，指數延遲至10/22才見底4,089點，一個月後上漲11.2%

適用時機：判斷大盤是否接近底部、評估系統性風險程度、決定是否可以開始分批進場

心得

這回合把借券和融資斷頭潮這兩個籌碼面的「進階工具」搞清楚了。之前學過借券賣出餘額的基本概念，但這次更深入理解了「借券≠放空」這個核心觀念。借券餘額增加可能是避險操作而非看空，這跟之前學到的自營商避險買超邏輯一致——永遠要問「背後動機是什麼」，不能只看數字表面。

借券賣出餘額搭配外資持股判讀是這回合最實用的收穫。四種組合（借券增+外資買、借券增+外資賣、借券減+外資買、借券減+外資賣）把模糊的借券數據變成了可操作的訊號。以前看到借券增加就覺得是壞消息，但其實外資避險的借券增加搭配外資買超，反而是偏多訊號。

融資斷頭潮的部分，2025年4月的那次歷史事件太震撼了——三日蒸發750億融資、單日減少330億創歷史新高。但文章最核心的觀點是：融資斷頭潮雖然恐怖，但「融資大幅減少往往與市場底部形成有關」。關鍵是判斷斷頭潮處於三階段的哪一段——初始期不要抄底、加速期更要避開、尾聲期才是機會。

最值得記住的一個數字：融資比率1.5%是歷史安全水位。當大盤融資比率降到這個水準，代表系統性風險已大幅降低，大規模斷頭潮接近尾聲。以後遇到大跌，先看融資比率，不要急著接刀。
