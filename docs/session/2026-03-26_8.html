核心知識

跳空缺口是什麼

是什麼：K線圖中兩根相鄰K棒之間，有一段價格區間沒發生任何交易，形成空白缺口。代表市場在此價位沒有成交，是強勢表態。

為什麼重要：跳空缺口是強勢買進或賣出訊號，對判斷支撐壓力、趨勢強弱、趨勢轉折有重要參考價值。

四種缺口型態

1. 普通缺口

是什麼：發生在盤整區間內，沒有明顯趨勢方向。

特徵：

  • 跳空價格真空區不大
  • 容易在1-2天內回補
  • 沒有支撐壓力參考作用
  • 也稱「區域缺口」

怎麼做：忽略它。對技術分析意義不大，不要因為這種缺口進場。

2. 突破缺口

是什麼：股價向上或向下突破盤整區間時出現的跳空缺口，是趨勢強勢表態。

判斷標準：

  • 伴隨成交量放大
  • 突破盤整區的壓力線或支撐線
  • 2-3天內沒被回補

怎麼做 - 向上突破缺口：
1. 確認：跳空突破壓力線 + 成交量放大
2. 進場時機：

  • 激進做法：跳空當天進場

3. 停損：設在缺口下方

怎麼做 - 向下突破缺口：
1. 確認：跳空跌破支撐線
2. 操作：做空或賣出持股
3. 注意：向下缺口不需要成交量放大來驗證

實例：

  • 情境：某股票在100-110元盤整一個月
  • 突破：開盤跳空到115元，成交量倍增
  • 操作：確認缺口未在3天內回補，於拉回至115元附近進場，停損設112元（缺口下方）

注意：向上跳空缺口如果2-3天內被回補，視為「假突破」；向下跳空缺口被回補，視為「假跌破」。

3. 逃逸缺口

是什麼：趨勢進行中（上漲或下跌），盤整後再出現的跳空缺口，代表趨勢尚未結束。

判斷標準：

  • 出現在突破缺口之後
  • 成交量持續或放量
  • 缺口不容易被回補

怎麼做：
1. 辨識：突破缺口後一波上漲，然後盤整，再出現跳空
2. 操作：如果是多頭，可以繼續持有；在預估價位來臨時限價賣出
3. 注意：一段多頭行情可能出現複數逃逸缺口

實例：

  • 情境：股票從100元突破缺口到115元上漲
  • 過程：漲到130元後盤整幾天
  • 逃逸：再跳空到138元，確認為逃逸缺口
  • 操作：繼續持有，等壓力區或目標價再賣

4. 竭盡缺口

是什麼：趨勢末段出現的跳空缺口，代表多空力道耗盡，通常暗示行情即將反轉。

判斷標準：

  • 出現在末升段或末跌段
  • 成交量放大但價格不漲（或不續跌）
  • 2-3天內缺口快速被回補

怎麼做 - 多頭竭盡缺口：
1. 辨識：已大漲一段時間 + 跳空 + 成交量爆大量 + 價格不涨反跌
2. 操作：賣出持股，可酌量做空
3. 停損：如果誤判（不是竭盡缺口），停損設在缺口上方

怎麼做 - 空頭竭盡缺口：
1. 辨識：已大跌一段時間 + 跳空開低 + 成交量放大 + 價格不再創低
2. 操作：開始布局做多
3. 注意：下跌突破缺口不需要成交量驗證，但竭盡缺口需要

實例：

  • 情境：股票從100元漲到200元
  • 竭盡：開盤跳空到210元，爆大量但收盤跌回205元
  • 隔天：再跌回200元，缺口被回補
  • 操作：確認竭盡缺口，賣出持股

缺口與成交量的關係

缺口類型成交量表現操作意義
突破缺口成交量放大確認趨勢啟動，可進場
逃逸缺口成交量持續趨勢仍強，可持有
竭盡缺口成交量爆大量但價格停滯趨勢反轉，應出場

關鍵觀察：

  • 突破缺口：成交量增加初期
  • 竭盡缺口：成交量增加但價格推升無力（投資人不願追高）

島狀反轉

是什麼：股價跳空上漲後，在高檔無法創新高，接著再向下跳空下跌，圖形上形成「島嶼」形狀。

判斷標準：

  • 左邊有向上跳空缺口
  • 右邊有向下跳空缺口
  • 中間高點像海上島嶼

怎麼做：
1. 辨識：已有一段漲幅 + 跳空開高 + 幾天後跳空開低
2. 操作：做空好機會（發生在大跌初期）
3. 注意：島狀反轉可能出現假訊號，需等2-3天確認

哪些缺口不適合使用缺口戰法

不適合的缺口：
1. 除權息缺口：公司發放紅利造成，非市場心理反映
2. 減資缺口：股數減少導致價格跳空，與技術面無關

注意：

  • 除權息缺口被回補稱為「填權息」
  • 能快速填權息的股票，代表基本面強勁

實戰操作總結

進場時機：

  • 突破缺口 + 成交量放大 + 2-3天未回補 = 買進訊號
  • 保守做法：等拉回測試缺口再進場

出場時機：

  • 竭盡缺口出現 = 快速減碼
  • 缺口被回補 + 技術指標（RSI/KDJ/MACD）死亡交叉 = 出場

停損設定：

  • 向上突破缺口進場：停損設在缺口下方
  • 突破缺口被回補（假突破）：立即停損

心得

缺口戰法的核心是「判斷缺口類型」，四種缺口長得一樣，但意義完全不同。重點在於：

1. 位置決定性質：同樣是跳空，在盤整區是「普通缺口」（無視），在突破區是「突破缺口」（進場），在趨勢中段是「逃逸缺口」（持有），在末段是「竭盡缺口」（出場）。

2. 成交量是關鍵：突破缺口的成交量是用來確認趨勢強度；但竭盡缺口的成交量放大反而是警訊，代表主力在出貨。

3. 時間驗證一切：2-3天內是否回補，是判斷缺口真假的最重要依據。假突破和假跌破就是這樣辨識的。

4. 實戰建議：不要看到跳空就追，要等2-3天確認缺口未回補再進場。雖然會錯過一點利潤，但避開假突破的風險更划得來。

這套缺口服邏輯跟葛蘭碧八大法則有點像，都是在找均線和價格的關係，只是缺口更強調「價格真空帶」的力量。可以兩套併用：缺口確認趨勢，均線找進出場點。

核心知識

融資操作策略

是什麼：向券商借錢買股票，用4成資金買到10成股票（上市），或5成資金買到10成股票（上櫃）。

怎麼做：
1. 開立信用交易帳戶（需證券戶滿3個月、年成交筆數10筆以上）
2. 自備4成資金（上市）或5成資金（上櫃）
3. 在下單軟體選擇「融資」選項
4. 設定停損機制避免斷頭

實例：

  • 情境：看好台積電從500元上漲，但只有20萬資金
  • 操作：融資買進，自備20萬可買到50萬股票
  • 獲利：若股價漲10%，獲利5萬（投資報酬率25%，而非現股的10%）
  • 風險：若股價跌10%，虧損5萬（投資報酬率-25%）

適用時機：

  • ✅ 看好短期上漲、想放大獲利
  • ✅ 資金不足但想參與行情
  • ✅ 當沖交易（資券當沖）
  • ❌ 長期持有（利息成本6~7%太高）
  • ❌ 新手或不熟悉槓桿風險

融券放空策略

是什麼：向券商借股票賣出，等股價下跌後買回還給券商，賺取價差。

怎麼做：
1. 開立信用交易帳戶
2. 準備90%保證金（股價的9成）
3. 在下單軟體選擇「融券」選項
4. 等股價下跌後「回補」買回股票還券商

實例：

  • 情境：看壞某股從100元下跌
  • 操作：融券賣出，付90元保證金
  • 獲利：若跌到50元，買回成本50元，赚50元
  • 風險：若漲到120元，需花120元買回，加上保證金90元，共虧20元

適用時機：

  • ✅ 看壞短期下跌
  • ✅ 打帶跑策略（做空比做多更快）
  • ❌ 長線持有（下跌時間短，上漲時間長）
  • ❌ 成交量低的股票（可能借不到券）
  • ❌ 接近除權息/股東會（強制回補風險）

軋空判斷與風險

是什麼：放空者預期股價下跌，結果股價上漲，被迫高價買回股票還券商，進一步推升股價。

怎麼判斷軋空風險：
1. 檢查融券餘額比例：融券餘額高於10%要小心
2. 檢查股票流動性：成交量太低容易軋空
3. 檢查回補日期：接近除權息、股東會前更危險

實例：

  • 情境：GME軋空事件，2021年初
  • 操作：散戶集體買進，逼迫做空機構高價買回
  • 結果：股價從17元飆到483元，做空機構虧損數十億

適用時機：

  • ✅ 放空前檢查融券餘額
  • ✅ 設定停損避免軋空
  • ❌ 不要執著於放空邏輯，邏輯正確但被軋空是常見悲劇
  • ❌ 不要陷入「莊家盯著我」的陰謀論

填息波段操作策略

是什麼：在除息日買進高股息股票，賺取填息價差。

怎麼做：
1. 篩選「快速填息常勝軍」：過去10年平均在2個月內填息
2. 檢查當年殖利率：殖利率越高越好（>5%）
3. 確認大盤環境：必須是多頭市場
4. 在除息日當天買進

實例：

  • 情境：微星2377，2021年配6.1元股利
  • 操作：除息前一天收盤122.5元，殖利率約5%
  • 結果：2天完成填息，一個月內從119元走到147元，波段漲幅超過20%

適用時機：

  • ✅ 大盤多頭欣欣向榮
  • ✅ 過去10年平均2個月內填息的股票
  • ✅ 當年殖利率高（>5%）
  • ❌ 空頭市場（填息常勝軍也可能被波及）
  • ❌ 第一次除息的股票（沒有填息紀錄）

融資斷頭與融券回補

是什麼：

  • 斷頭：融資維持率低於130%，券商強制賣出你的股票
  • 回補：融券在特定日期前必須買回股票還券商

斷頭怎麼處理：
1. 維持率計算：（融資擔保品 + 保證金）÷ 股票現值
2. 低於166%券商會通知補保證金
3. 低於130%券商會強制賣出（斷頭）
4. 避免方法：設定停損，控制槓桿

回補重要日期：

  • 股東常會召開前2-3個月
  • 除權息日前
  • 現金增資時

放空心法三大建議

1. 放空不宜長線作戰

  • 下跌快速但短暫，上漲緩慢但持久
  • 打帶跑，見好就收

2. 放空不要執著於邏輯

  • 邏輯正確但被軋空是常見悲劇
  • 看對方向但被保證金追繳強制平倉

3. 放空不要陷入幻想

  • 不要覺得「莊家在盯著我的空單」
  • 保持理性，該停損就停損

心得：

融資融券是股市的「雙面刃」：放大獲利但同樣放大風險。

最讓我印象深刻的是「獲利一倍，風險無限」這句話。放空的風險真的比做多可怕——做多是股價跌到0為止，但做空是股價上漲沒有上限。GME軋空事件就是最好的警示。

填息波段操作這招很聰明：不是為了領股息，而是賺取填息價差。關鍵是選對股票（快速填息常勝軍）和對的時機（多頭環境）。這招勝率高但報酬率有限，適合保守型投資人。

關於融資，雖然槓桿很誘人，但6~7%的年利息實在太高，長期持有根本不划算。所以融資適合的是「短線看好」的情境，不是「長期投資」。
