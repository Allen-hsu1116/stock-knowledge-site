核心知識

一、停損停利比例設定

是什麼：停利與停損的金額或百分比比例，決定你的盈虧結構。

為什麼重要：

  • 停利設太少：賺3次賠1次就打平（如賺1千、停損3千）
  • 停利設太遠：根本等不到，只有一直被停損的份

怎麼做：
1. 建議比例：1:2 或 1:3（停損1份：停利2-3份）
2. 實例：停損500元，停利設1500元（1:3）
3. 若擔心停利太遠觸不到，可縮小金額但維持比例

  • 原本：虧1千、賺3千 → 改成：虧5百、賺1千5

注意：範圍不能縮太小

  • 台股跳動檔位：20元股票，一檔0.05元
  • 若設停損100元（約20檔），很容易被正常波動洗出去

二、分批停利策略

是什麼：分兩次以上賣出，保留部分部位續抱。

適用情境：

  • 擔心賣太早，後續還有行情
  • 不確定會漲到哪，想先鎖定部分利潤

怎麼做：
1. 前提：至少買進2張（或零股2股以上）
2. 實例：

  • 20元買進2張，停利設25元

3. 關鍵：第二張的停損要設在「打平或小賺」

  • 最差設在進場價20元（打平出場）

心態：先賣的是「確定的獲利」，留的是「潛在的空間」

三、移動停損（Trailing Stop）

是什麼：停損點跟著價格移動，兼具停損和移動停利效果。

為什麼用：

  • 解決「賺小賠大」問題
  • 方向正確時，停損點會往上移，變成移動停利

怎麼做（K棒低點法）：
1. 多單：停損設在「最近N根K棒低點」（如N=5）
2. 空單：停損設在「最近N根K棒高點」
3. 效果：

  • 價格上漲 → 低點跟著上移 → 停損變停利

進階版：用 ATR（平均真實區間）

  • 停利：進場價 + N × ATR
  • 停損：進場價 - M × ATR
  • 優點：自動適應波動度，不用固定點數

ATR 實例：

  • ATR = 150點，設定停利5倍、停損2倍
  • 多單進場18000，停利 = 18000 + 5×150 = 18750
  • 停損 = 18000 - 2×150 = 17700

四、時間出場策略

是什麼：不靠價格，而是用時間來決定出場。

適用情境：
1. 當沖收盤前出場

  • 語法：if time > 1320 then sell next bar at market

2. 結算日前強制出場

  • 不想轉倉，直接在結算日前平倉

3. 進場後N根K棒 + 損益兩平出場

  • 進場超過5根K棒且已有獲利 → 掛損益兩平單

五、停利三種設定方法

方法說明實例
**數字停利**固定金額或百分比賺8%出場、賺3000元出場
**支撐/壓力停利**技術位置買進後漲到壓力區賣出
**進場理由消失**技術訊號反轉因突破買進，跌破支撐就賣

可自由搭配：停損設「進場理由消失」，停利設「賺8%」

心得

停利比停損更難，因為要對抗「貪婪」。停損是認賠，停利是「放棄更多可能」——這需要勇氣。

今天學到最有價值的是 分批停利 和 移動停損用 ATR：

  • 分批停利解決了「賣太早後悔」和「等太晚變賠」的兩難
  • ATR 讓停損停利自動適應波動，比固定點數更科學

還有一個重要概念：停損停利比例 1:2 或 1:3。這不是隨便定的數字，而是數學期望值。如果勝率50%，賺3賠1，期望值 = 0.5×3 - 0.5×1 = 1（正期望）。但多數人是反過來：賺1賠3，期望值 = 0.5×1 - 0.5×3 = -1（負期望），難怪賠錢。
