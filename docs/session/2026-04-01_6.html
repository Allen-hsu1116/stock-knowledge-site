一、期貨未平倉基本概念

是什麼：期貨未平倉口數（Open Interest, OI）是指尚未到期且未平倉的期貨契約口數，代表市場上多空雙方還在場內的資金。

怎麼看：
1. 期貨是零和市場：有人做多一口，就有人做空一口
2. 多頭未平倉 = 空頭未平倉（永遠相等）
3. 未平倉量增加 → 新資金進場（多空對決加劇）
4. 未平倉量減少 → 資金離場（多空一方認輸出場）

實例：

  • 情境：台指期未平倉從 15 萬口增加到 18 萬口
  • 解讀：有新資金進場，趨勢動能增強
  • 若同時價格上漲 → 多方主導，偏多
  • 若同時價格下跌 → 空方主導，偏空

二、價格、成交量、未平倉三要素判讀

是什麼：結合「價格方向」「成交量」「未平倉量」三個維度，判斷趨勢強弱。

怎麼判斷：

期貨價格成交量未平倉盤面解讀
上升增加增加價漲量增偏多，OI增加代表多空加碼，趨勢持續
上升增加減少價漲量增但OI減，多方獲利了結+空方停損，偏中性
上升減少減少價漲量縮，力道減弱，偏中性
下跌增加增加價跌量增偏空，OI增加代表空方加碼，趨勢持續
下跌增加減少價跌量增但OI減，多方停損+空方獲利了結，偏中性
下跌減少減少價跌量縮，空方力道減弱，偏中性

實例：

  • 情境：台指期上漲 150 點，成交量放大，未平倉增加 5,000 口
  • 解讀：多方積極進場，空方也不認輸持續加碼，多空對決
  • 操作：趨勢有望延續，可偏多操作

三、大外資 vs 小外資（實戰關鍵！）

是什麼：外資其實不是單一整體，可以拆分成「大外資」和「小外資」。

計算方式：

  • 五大外資：期交所公布的「五大特定人」，通常是外資中部位最大的
  • 小外資 = 外資總部位 - 五大外資部位

怎麼用：

類型特性操作意義
大外資部位大，無法短線進出看「中期趨勢」（1-3個月）
小外資部位小，操作靈活看「短線變化」（1-2週）

實例：

  • 情境：大外資淨多單 3 萬口，小外資剛翻多
  • 解讀：中期偏多（大外資撐腰），短線也轉多
  • 操作：大小外資同步偏多，行情偏多機率高

注意：

  • 若大小外資都翻空 → 大行情反轉警訊
  • 若大小外資方向分歧 → 以大外資中期方向為主

四、外資期貨未平倉實戰判讀

是什麼：外資台指期的多空淨口數，判斷外資對大盤的看多或看空。

怎麼看：
1. 淨多單（正數）：外資偏多，看漲大盤
2. 淨空單（負數）：外資偏空，看跌大盤
3. 加碼變化：看「口數變化」而非絕對數字

實例：

  • 情境：外資台指期淨空單從 -25,000 口加碼到 -35,000 口
  • 解讀：外資加碼空單 10,000 口，看空力道增強
  • 警惕：結算日可能壓低結算

關鍵數字：

  • 外資淨空單 > 3 萬口 → 空方力道強
  • 外資淨多單 > 3 萬口 → 多方力道強
  • 變化 > 5,000 口 → 有意義的加減碼

不是 100% 準確：

  • 外資籌碼只是參考，不是聖杯
  • 外資可能有避險需求，不一定代表看空
  • 需搭配其他指標綜合判斷

五、選擇權未平倉判讀

是什麼：選擇權未平倉量透露市場對支撐壓力的共識。

怎麼解讀：

項目解讀方式
Call OI 增加賣方看「不漲」，該履約價為壓力區
Put OI 增加賣方看「不跌」，該履約價為支撐區

實例：

  • 情境：17500 Call 有 5 萬口未平倉，17000 Put 有 4 萬口未平倉
  • 解讀：17500 是上檔壓力區，17000 是下檔支撐區
  • 操作：指數接近 17500 時留意賣壓，接近 17000 時留意支撐

六、Put/Call Ratio

是什麼：Put未平倉量 / Call未平倉量，判斷整體市場多空氛圍。

怎麼用：

  • P/C Ratio > 1：Sell Put > Sell Call，莊家偏多（認為有支撐）
  • P/C Ratio < 1：Sell Call > Sell Put，莊家偏空（認為有壓力）

實例：

  • 情境：P/C Ratio 從 0.8 升到 1.2
  • 解讀：選擇權賣方從偏空轉偏多，大盤支撐增強
  • 操作：偏多操作相對安全

七、三大法人選擇權未平倉

是什麼：外資、投信、自營商在選擇權市場的部位。

怎麼看：

  • 外資 Sell Call 大增 → 看不漲，偏空
  • 外資 Sell Put 大增 → 看不跌，偏多
  • 搭配期貨未平倉一起看更準確

八、實戰整合判讀

步驟：
1. 先看外資期貨淨口數 → 判斷多空方向
2. 再看大小外資分歧 → 判斷短中期是否一致
3. 輔以選擇權 P/C Ratio → 確認市場氛圍
4. 找出未平倉大量區 → 畫出支撐壓力帶

注意事項：

  • 外資籌碼不是聖杯，準確率約 60-70%
  • 需搭配技術面（K線、均線）、基本面一起判斷
  • 結算日前的加減碼變化更重要
