籌碼面分析：主力成本線判讀×法人成本價推算×成本區進出場策略

知道主力在哪裡買的，比知道主力買了多少更重要。因為「成本」決定了主力的盈虧平衡點，而盈虧平衡點決定了主力會在什麼價位防守、什麼價位獲利了結、什麼價位斷頭出場。主力成本線就是一條告訴你「大戶的成本在哪裡」的線，搞懂這條線，等於拿到了主力的底牌。

一、什麼是成本線？

是什麼

成本線 = 某段時間內，考量了「交易量」與「身分別」後的加權平均成本。與均線不同，均線只看收盤價的平均，不考慮誰買了多少；成本線則把每個身份（融資、投信、外資、主力）的買賣量和價格一起算進去，更真實反映特定群體的持倉成本。

成本線 vs 均線差異：

維度均線成本線
計算基礎收盤價平均成交價×成交量加權平均
身分別不區分區分融資/投信/外資/主力
交易量不考慮加權計算
意義市場平均價格特定群體的平均成本

四種成本線

成本線使用者適用週期特性
**融資成本線**散戶短線（日～週）最貼近股價，反應快
**投信成本線**投信基金中線（月～季）中期趨勢判斷
**外資成本線**外資機構長線（季～年）大波段方向
**主力成本線**主力大戶依參數設定最核心的籌碼判斷

二、主力成本線：40MA操作法

是什麼

專職操盤手獨孤求敗提出，以40日移動平均線作為「主力成本線」。邏輯：台股周轉率全球最高，投資人偏短線，40天大約是主力從吃貨到拉抬的一個完整循環，因此40MA最能代表主力的平均進場成本。

怎麼做 — 40MA操作法

均線上揚時：
1. 股價站上40MA → 買進
2. 停損設在40MA位置
3. 用移動停利：不設固定目標價，出場點跟著均線往上移
4. 股價跌破40MA → 賣出
5. 重複直到均線下彎

均線下彎時：
1. 股價跌破40MA → 做空或觀望
2. 股價站回40MA → 回補空單
3. 重複直到均線翻揚

實例 — 微星（2014年）：

  • 2014/3/12 收盤價29.2元站上40MA（27.65元），買進
  • 停損設27.65元（虧損風險約5%）
  • 持股過程中獲利20%不出場、漲到40元不出場
  • 移動停利設在均線位置，給價格回檔空間
  • 2014/7/29 跌破40MA（45.98元）出場
  • 獲利：(45.98-29.2)/29.2 = 57.5%
  • 之後40MA下彎，不再進場

盤整區的變通方法

問題：股價進入橫向整理，K線在40MA上下來回穿越，頻繁進出會耗損資金。

解法：
1. 發現進入盤整區時，停損改設在「盤整區下緣再低一些」
2. 不要在盤整區裡面進進出出
3. 等行情突破盤整區上緣後，再恢復用40MA控盤
4. 「該搭船的時候就搭船，不會有人堅持把車子開進海裡」

適用時機：

  • 主力有在著墨的熱門股效果最好
  • 沒量、沒特定主導勢力的冷門股不適用
  • 盤整行情需要變通，不要死守單一方法

三、法人成本線：短中長期判讀框架

融資成本線（短線）

是什麼：散戶的加權平均買進成本。融資是散戶指標，因為散戶沒有龐大資金做長期布局，通常是短期買賣。

怎麼做：
1. 融資成本線非常貼近股價走勢
2. 正常情況下股價往上、融資慢慢堆疊，成本線跟著上移
3. 突破訊號：過去沒有融資買，突然融資大增 + 突破月線或融資成本線 + 成交量放大 → 有機會往上一波
4. 跌破訊號：股價跌破融資成本線 → 散戶套牢，因為融資槓桿2.5倍，跌破成本線代表散戶開始虧損，可能引發斷頭賣壓

實例 — 百和：

  • 過去無融資買進，某天突然融資大增
  • 同時突破月線和融資成本線
  • 成交量放大
  • 後續展開一波上漲行情

實例 — 敬鵬：

  • 融資成本線跟隨股價上升
  • 股價突然大跌，跌破融資成本線
  • 散戶因槓桿2.5倍，跌破成本即虧損
  • 引發追繳和斷頭壓力，股價進一步下跌

投信成本線（中線）

是什麼：投信基金的加權平均買進成本。投信布局多為中長期，以月為單位（3個月、半年進出）。

怎麼做：
1. 投信成本線是中線投資人判斷進出場的主要依據
2. 投信持股比例五段判讀：

持股比例解讀行動
0%-4%低檔區，後續有機會增持觀察是否開始加碼
4%-6%低檔升溫區可開始布局
6%-10%升溫區，各路人馬積極買超順勢操作
10%-15%高檔區，注意反轉風險開始減碼
>15%危危險區，隨時可能結帳果斷出場

3. 投信殺盤特性：一旦開始賣，不會停在15%直接砍到10%，後續降到6%和4%才回穩
4. 搭配投信成本線：股價在成本線之上 → 投信賺錢，偏多操作；跌破成本線 → 投信套牢，偏空

外資成本線（長線）

是什麼：外資的加權平均買進成本。外資布局以年為單位，是大波段操作的代表。

怎麼做：
1. 股價在外資成本線之上 → 外資賺錢 → 繼續加碼 → 推升股價
2. 股價跌破外資成本線 → 外資套牢 → 停損快狠準 → 連鎖賣壓
3. 前提：外資持股比例需超過10%，成本線才有參考價值

  • 外資持股 <10%：外資不是主要籌碼，成本線意義不大

實例 — 敦吉（2459）：

  • 外資長期認養，持股超過10%
  • 股價長期在外資成本線之上運行
  • 每次拉回觸及成本線都是買點（外資會防守成本線）
  • 一旦跌破成本線且外資轉為賣超 → 趨勢反轉

重要原則：乖離過大不要追高！股價和成本線拉開一段距離後，不要再加碼進場。

四、主力作多成本線：XQ量化計算法

是什麼

XQ全球贏家平台提供的主力作多成本線指標，用更精確的量化公式計算主力的平均進場成本。

計算公式

主力作多成本 = Σ(主力買張 × 日均價) / Σ(主力買張)

其中：
- 日均價 = (開盤 + 最高 + 最低 + 收盤) / 4
- 期間預設40天
- 主力買張 = GetField("主力買張")

公式拆解：
1. 每天計算：主力買張 × 當日均價 = 當日主力作多金額
2. 累計40天的作多金額 / 累計40天的買張 = 主力平均成本

怎麼做 — 三種應用情境

情境1：判斷拉回止跌點

  • 股價上漲後拉回
  • 接近主力成本線時，看主力是否仍在買超
  • 主力仍買超 → 主力在成本區加碼 → 成本線有支撐 → 偏多
  • 主力轉賣超 → 主力不防守成本區 → 可能繼續跌

情境2：判斷反彈滿足點

  • 股價大跌後反彈
  • 反彈到主力成本線附近 → 這是主力套牢者的解套區
  • 套牢的主力可能趁反彈出場 → 成本線是反彈的天花板
  • 除非主力重新翻多買超，否則反彈到成本線就是出場點

情境3：判斷主力是否持續作多

  • 股價拉回至成本線
  • 主力買超（持續加碼）→ 主力仍看好 → 成本線是買點
  • 主力大賣超 → 主力要清倉 → 成本線是最後出場機會

實例

  • 某熱門股主力40天內累計買超1萬張，成本線在50元
  • 股價從60元拉回到51元接近成本線
  • 觀察主力近3日仍持續買超 → 主力在50元附近加碼 → 51元是買點
  • 反之如果主力近3日轉為賣超 → 主力在50元附近棄守 → 51元不是支撐，不要接刀

適用與不適用

適用不適用
主力有著墨的熱門股沒量的冷門股
有特定主導勢力的個股沒有特定主力主導的股票
中大型股本、流動性佳小型股、主力不明確

五、成本線進出場整合策略

是什麼

結合四種成本線（融資、投信、外資、主力），建立一個多維度的進出場判斷框架。不同週期的投資人看不同的成本線，但最強的訊號是「多條成本線同時確認」。

怎麼做 — 三步驟策略

Step 1：確認你的投資週期

投資週期主要參考成本線輔助成本線
短線（1-5天）融資成本線主力成本線（短參數20日）
中線（1-3個月）投信成本線主力成本線（40日）
長線（6個月以上）外資成本線主力成本線（60-120日）

Step 2：成本線之上做多，之下做空

基本原則跟均線一樣，但成本線比均線更準確，因為它考慮了身分別和成交量：

  • 股價在主力成本線之上 + 主力買超 → 做多
  • 股價在主力成本線之下 + 主力賣超 → 做空或觀望
  • 股價在主力成本線之上 + 主力賣超 → 可能出貨，減碼
  • 股價在主力成本線之下 + 主力買超 → 可能在抄底，觀察

Step 3：多條成本線共振確認

最強的訊號是「多條成本線同時指向同一個方向」：

訊號強度條件行動
★★★★★外資+投信+主力成本線全部在股價下方強力做多
★★★★外資+投信成本線在下方，融資成本線在上方偏多，散戶套牢
★★★只有主力成本線在下方中性偏多觀察
★★融資成本線在下方，法人成本線在上方散戶賺法人套，不太妙
★外資+投信+主力成本線全部在股價上方強力做空或觀望

成本線五大陷阱

1. 乖離過大不要追

  • 股價遠離成本線時，不要加碼追高
  • 正乖離過大 → 獲利了結壓力
  • 負乖離過大 → 可能反彈，但也可能繼續崩

2. 成本線不是精確數字

  • 成本線是「推估值」，因為用的是日均價而非實際成交均價
  • 主力可能盤中低接、收盤前拉高，實際成本比算出來的更低
  • 成本線的「區間」比「精確數字」更重要

3. 主力可能分批進出

  • 主力不是一次買完，成本線是加權平均
  • 主力可能在100元買1000張、在80元買2000張
  • 平均成本87元，但實際上主力80元的部位更多
  • 關注成本線的「移動方向」比「絕對位置」更重要

4. 成本線對冷門股無效

  • 沒有明確主力著墨的股票，成本線是雜訊
  • 成交量太小的股票，成本線計算失真
  • 只適用有量、有主力主導的熱門股

5. 多身份成本線可能矛盾

  • 外資成本線在下方（偏多）、投信成本線在上方（偏空）
  • 這代表外資賺錢、投信套牢
  • 需要判斷誰是主要籌碼驅動者，而非簡單取平均

六、成本線×其他指標實戰組合

組合一：成本線 + 主力買賣超

股價位置主力動向判斷行動
成本線之上主力買超主力持續加碼做多做多
成本線之上主力賣超主力在獲利區出貨減碼/出場
成本線附近主力買超主力在成本區加碼防守買點
成本線附近主力賣超主力不防守成本區觀望/做空
成本線之下主力買超主力套牢但加碼攤平觀察是否止跌
成本線之下主力賣超主力認賠出場做空或避開

組合二：成本線 + 千張大戶持股

1. 千張大戶增加 + 股價在主力成本線之上 → 大戶和主力同步做多，最強訊號
2. 千張大戶減少 + 股價跌破主力成本線 → 大戶和主力同步出場，最弱訊號
3. 千張大戶增加但股價跌破成本線 → 大戶被套，觀察是否加碼

組合三：成本線 + 技術面支撐壓力

  • 主力成本線與技術面支撐（前低、均線、趨勢線）重疊 → 雙重支撐，支撐力道更強
  • 主力成本線與技術面壓力重疊 → 雙重壓力，突破難度更高
  • 成本線在壓力位之上 → 壓力可能失效（主力成本在更高位置，不會在壓力位出貨）

心得

主力成本線這東西，本質就是把「主力在哪裡買的」量化成一條線。看起來簡單，但背後的邏輯非常重要：

1. 40MA主力成本線是最簡潔的版本——獨孤求敗用40天移動平均線代表主力成本，操作邏輯就是「站上買、跌破賣」，簡單到不行。但越簡單的方法越需要紀律執行。微星那個案例，從29.2買進到45.98出場，中間漲到40元時「忍住不賣」才是最難的。這跟之前學的「移動停利」概念完全一致：不要想賣在最高點，讓利潤奔跑，用均線當移動停利。

2. 法人成本線的三層結構是關鍵——融資成本線看短線散戶、投信成本線看中線基金、外資成本線看長線外資。三條線就像三個時間尺度的「溫度計」，各自量測不同群體的持倉狀態。最危險的情況是：三條線都在股價上方（所有人套牢），最安全的情況是：三條線都在股價下方（所有人都賺錢）。這個框架把之前學的法人持股、融資券判讀全部串起來了。

3. XQ的主力作多成本線公式提供了量化計算的思路——主力買張×日均價 / 主力買張，這個公式任何人都可以自己算，不一定要用特定軟體。但更重要的是公式背後的邏輯：當股價拉回到成本線附近時，看主力是「加碼防守」還是「認賠出場」，這才是成本線最核心的應用——它不只是告訴你成本在哪，而是告訴你主力對這個成本的態度。

4. 成本線是「推估值」不是「精確值」——用日均價代替實際成交價，用公開資料推算主力行為，這些都有誤差。但投資不需要精確，需要的是方向正確。成本線提供的是一個「大約在哪裡」的參考，配合主力買賣超方向和技術面支撐壓力，三重確認才最可靠。

籌碼面分析：多維度籌碼整合SOP×看法歧異度×實戰選股流程

學了這麼多籌碼面指標——三大法人、融資融券、千張大戶、分點進出、成本線、期貨未平倉、選擇權OI——如果不把它們整合成一個可執行的流程，等於手上有滿把的武器卻不知道先拿哪一把。今天要把之前所有籌碼面知識串起來，建立一個從大盤到個股、從日線到週線、從法人到大戶的完整SOP。

一、看法歧異度（Belief Dispersion）：籌碼分析的理論基礎

是什麼

看法歧異度 = 市場參與者之間對股價看法的分歧程度。當歧異度高，趨勢難以持續；當歧異度低（籌碼集中到看法一致的群體手上），趨勢更容易延續。

這不是玄學，是有學術基礎的：

  • Yu (2011)：投資人看法歧異度越高，均值回歸越快越強
  • Atmaz & Basak (2018)：看法分歧導致股價反轉
  • Li et al. (2022)：看法異質性與價格反轉正相關

籌碼集中的數學意義

周依晴 (2020) 的台股研究發現：

持股級別增加時對報酬的影響影響期間
10張以下（散戶）**負面**月、季、年報酬都受害
100-200張（中實戶）**正面**月、季、年報酬都受益
1000張以上（大戶）**正面**季、年報酬受益

核心結論：短期看中實戶跟散戶，長期看大戶。中實戶增加→月報酬正面；散戶增加→所有期間報酬都負面。

怎麼做

1. 選股時優先找「千張大戶增加+散戶減少」的股票——這代表看法歧異度降低，籌碼從分歧走向一致
2. 避開「散戶暴增」的股票——不管基本面多好，散戶一窩蜂進場就代表歧異度升高，趨勢難持續
3. 中實戶（100-200張）是短線領先指標——中實戶比千張大戶更靈敏，月報酬影響最快顯現

實例：

  • 某股千張大戶持股從45%上升到52%，同時1-5張散戶人數從15,000人降到12,000人
  • 看法歧異度降低 → 籌碼往看法一致的群體集中 → 趨勢容易延續
  • 反之：散戶人數暴增到25,000人 → 歧異度飆高 → 即使股價在漲，隨時可能反轉

適用時機：每週六9:00集保股權分散表更新後，優先篩選大戶增加+散戶減少的個股

二、大盤籌碼面 vs 個股籌碼面：兩個層次不可混為一談

是什麼

大部分教學把大盤籌碼和個股籌碼混在一起講，但兩者的觀察重點完全不同：

層次觀察對象核心指標更新頻率用途
**大盤**整體市場資金方向三大法人現貨買賣超、大盤融資餘額、外資期貨淨部位、借券賣出餘額每日判斷大環境多空
**個股**特定股票的籌碼分布個股三大法人買賣超、千張大戶持股、分點進出、融資融券餘額、成本線每日/每週找進出場時機

怎麼做 — 大盤籌碼每日SOP

Step 1：下午4:00查三大法人現貨買賣超（證交所官網）

  • 外資買超+投信買超+自營商買超 = 三大法人同步買超 → 大盤偏多
  • 外資賣超+投信買超 = 大型股偏空、中小型股偏多 → 選中小型操作

Step 2：查大盤融資餘額變化

  • 融資連續5天增加+指數在高檔 → 散戶FOMO，危險
  • 融資連續5天減少+指數在低檔 → 籌碼沉澱，接近底部

Step 3：查外資期貨淨部位（期交所）

  • 外資淨多單連續3天增加 → 中期偏多
  • 外資淨空單連續3天增加 → 中期偏空
  • 搭配現貨看：期貨多+現貨買超 = 真偏多；期貨空+現貨買超 = 避險

Step 4：查借券賣出餘額（證交所）

  • 借券賣出餘額連續5天增加 → 外資偏空
  • 借券賣出餘額減少+外資現貨買超 → 外資翻多

怎麼做 — 個股籌碼每日SOP

Step 1：查個股三大法人買賣超（證交所T86報表）

  • 投信連續5天買超 → 投信波段行情啟動
  • 外資連續3天買超+成交量放大 → 權值股波段行情
  • 自營商買超 → 短線參考價值低，除非連續大量

Step 2：查個股融資融券餘額

  • 融資暴增+法人賣超 = 籌碼從強者轉弱者 → 危險
  • 融資減少+法人買超 = 籌碼從弱者轉強者 → 利多
  • 融券暴增+即將回補 → 軋空候選股

Step 3：查分點進出（證交所/櫃買中心）

  • 找出買超最大的前5名分點
  • 對照「關鍵分點清單」：如果是績效好的分點 → 跟單
  • 對照「隔日沖分點」：如果是凱基松山、元大土城 → 不要追高

Step 4（每週六）：查集保股權分散表

  • 千張大戶持股比例變化：連續3週增加 → 籌碼集中
  • 1-5張散戶人數變化：連續3週減少 → 散戶退場
  • 中實戶（100-200張）變化：增加 → 短線偏多訊號

Step 5：計算主力成本線

  • 用XQ公式或40MA估算主力成本
  • 股價在成本線之上+主力買超 → 偏多
  • 股價接近成本線+主力加碼 → 買點
  • 股價跌破成本線+主力賣超 → 賣點

三、六大籌碼陷阱：常見誤區修正

陷阱一：法人買超≠股價必漲

真相：法人買超可能是避險布局。外資買現貨+空期貨=避險，不是看多。必須搭配期貨部位一起看。

怎麼做：出現法人買超時，同時查期貨部位。期貨多+現貨買超=真看多；期貨空+現貨買超=避險。

陷阱二：主力連買≠沒有風險

真相：主力連續買超可能是「拉高出貨」的前期動作。邊拉邊買，到高檔一次倒出。

怎麼做：看買超的「位置」。低檔連買=吃貨；高檔連買+爆量不漲=出貨訊號。搭配量價和防守線判斷。

陷阱三：籌碼集中≠絕對利多

真相：籌碼太過集中也可能壓抑股價波動。千張大戶持股>80%，流動性極差，想賣賣不掉。

怎麼做：千張大戶40-70%是最佳區間。太低（<30%）沒主力；太高（>80%）沒流動性。

陷阱四：高融券≠看空

真相：高融券反映市場懷疑情緒，但若股價不跌反漲，反而可能引發軋空行情。高融券是軋空的燃料。

怎麼做：高融券+即將回補+內部人加碼=軋空候選股。高融券+股價持續下跌=空方趨勢確立。

陷阱五：只看籌碼不看技術圖形

真相：主力連買但K線破底，應保持警覺。籌碼與技術走勢背離=某一方在騙人。

怎麼做：籌碼面+技術面雙重確認。主力買超+股價站上均線=偏多；主力買超+股價破底=陷阱或主力被套。

陷阱六：短線波動過度解讀

真相：籌碼面分析適合觀察中短期趨勢，對日內價格震盪不適用。單日數據可能只是調節，連續3-5天才值得參考。

怎麼做：至少觀察3天趨勢，不要因為單日法人賣超就恐慌。

四、籌碼面實戰選股五步流程

是什麼

把所有籌碼面指標整合成一個由大而小、由粗而精的五步選股流程。這不是一個單一策略，而是一個「過濾框架」——每一步都在縮小候選股範圍，最後留下的就是籌碼面最健康的股票。

Step 1：大盤環境判斷（每日）

目的：確認大環境是否適合做多

條件偏多中性偏空
外資現貨買賣超連續3天買超買賣互見連續3天賣超
外資期貨淨部位淨多單增加區間震盪淨空單增加
大盤融資餘額低檔溫和增加正常波動高檔急增
借券賣出餘額減少平穩連續5天增加

至少3個條件偏多才進場選股。2個偏多+1個中性=保守操作。2個以上偏空=空手觀望。

Step 2：法人篩選（每日）

目的：找出法人正在布局的股票

篩選條件：
1. 投信連續3天買超且累計買超>500張
2. 外資連續3天買超且累計買超>1000張
3. 三大法人合計連續5天淨買超

加分項：

  • 投信持股<10%（還在吸籌期）
  • 外資持股>10%且持續增加（外資認養）
  • 投信+外資同時買超（土洋同買=強烈訊號）

Step 3：大戶籌碼確認（每週六）

目的：確認籌碼是否從散戶轉向大戶

集保股權分散表篩選：
1. 千張大戶持股比例連續2週增加
2. 1-5張散戶人數連續2週減少
3. 中實戶（100-200張）人數增加

量化條件（研究回測勝率77.8%）：

  • 千張大戶持股在40-70%區間
  • 大戶持股市值佔比上升
  • 散戶人數月減幅>5%

Step 4：成本線與分點驗證（每日）

目的：確認主力態度和進場位置

成本線判斷：

  • 股價在主力成本線之上 → 偏多
  • 股價接近成本線+主力買超 → 買點
  • 股價遠離成本線（乖離>15%） → 不追高

分點驗證：

  • 買超分點是「績效好的關鍵分點」 → 加分
  • 買超分點是「隔日沖券商」 → 扣分
  • 買賣力>60% → 強力訊號

Step 5：出場條件設定（提前規劃）

目的：進場前就決定何時出場

出場訊號（任一觸發即出場）：
1. 投信連續3天賣超（投信翻空）
2. 跌破起漲長紅最低點（防守線失守）
3. 千張大戶持股連續2週減少（籌碼鬆動）
4. 股價跌破主力成本線+主力賣超（主力認賠）
5. 融資暴增+法人賣超（散戶接最後一棒）

五、不同市場情境的籌碼策略

情境一：多頭市場

籌碼特徵：

  • 外資持續買超、期貨淨多單增加
  • 融資溫和增加（健康資金回流）
  • 千張大戶持股穩定

操作策略：

  • 選股偏重投信連續買超的中小型股
  • 搭配成本線在買點附近進場
  • 寬設停損，用移動停利鎖住獲利

情境二：盤整市場

籌碼特徵：

  • 法人買賣互見、期貨部位區間震盪
  • 融資餘額變化不大
  • 千張大戶持股持平

操作策略：

  • 選股重千張大戶增加+散戶減少的個股
  • 操作週期縮短，有賺就跑
  • 注意作帳行情（季底前1-2月進場、季底前1週出場）

情境三：空頭市場

籌碼特徵：

  • 外資持續賣超、期貨淨空單增加
  • 融資餘額急增後急減（斷頭清洗）
  • 千張大戶持股下降（大戶也跑了）

操作策略：

  • 空手為主，不要跟空頭對作
  • 觀察融資清洗進度：融資餘額降到歷史低檔+外資空單開始回補 → 底部訊號
  • 反彈只做短多，嚴設停損
  • 高融券+即將回補的個股可考慮軋空行情

六、籌碼面+技術面+基本面 三合一確認

為什麼需要三合一

  • 基本面告訴你「買什麼」——公司值不值得投資
  • 籌碼面告訴你「誰在買」——有沒有大資金認同你的判斷
  • 技術面告訴你「何時買」——進出場時機

三者缺一不可：

  • 只看基本面：可能買在主力出貨前
  • 只看籌碼面：可能買到基本面爛的炒作股
  • 只看技術面：可能被假突破騙進去

三合一高勝率進場條件

面向條件確認度
基本面營收成長YoY>10%、ROE>15%、EPS成長★★★★
籌碼面投信連續買超+千張大戶增加+成本線之上★★★★★
技術面均線多頭排列+MACD黃金交叉+量價配合★★★★

兩個面向滿足 → 正常操作
一個面向滿足 → 觀望為主

心得

今天把之前所有籌碼面學習做了一個系統化的整合。幾個核心體悟：

1. 看法歧異度是籌碼分析的靈魂——之前學了這麼多指標，但一直沒有回答一個根本問題：為什麼籌碼集中股價就會漲？答案是Belief Dispersion。當籌碼集中到看法一致的群體（大戶），歧異度降低，趨勢容易延續；當籌碼散到看法分歧的群體（散戶），歧異度升高，均值回歸加速。這個理論框架把所有籌碼面指標串起來了：千張大戶增加=歧異度降低，散戶暴增=歧異度升高。以後看到集保股權分散表，不再是「大戶多就買」，而是理解背後的數學邏輯。

2. 大盤和個股是兩個戰場——大盤籌碼判斷大環境多空，個股籌碼找進出場時機。兩者不可混為一談。大盤偏空時，個股籌碼再好也要保守操作；大盤偏多時，個股籌碼好才能積極進場。這就是「先看天氣再出門」的道理。

3. 五步選股流程是可複製的框架——從大盤環境→法人篩選→大戶確認→成本線驗證→出場設定，每一步都在過濾，最後留下籌碼面最健康的股票。這不是一個保證賺錢的公式，而是一個「提高勝率的流程」。紀律執行這個流程，比憑感覺買賣好太多。

4. 出場條件比進場條件更重要——五步流程的最後一步是出場設定，這是故意放在進場流程裡的。進場前就決定何時出場，才能避免「賺了捨不得賣、賠了不甘心停損」的人性弱點。五個出場訊號任一觸發就走，不需要等到全部出現——等全部出現時已經賠一大段了。

籌碼面分析：融資斷頭連鎖效應×市場底部判斷×籌碼集中度六大指標

一、融資斷頭連鎖效應（Cascade Effect）

是什麼

融資斷頭連鎖效應是指：當股價下跌觸發融資追繳→投資人無法補繳→券商強制賣出→賣壓進一步壓低股價→觸發更多融資追繳，形成「多殺多」的惡性循環。

連鎖效應三階段

第一階段：觸發期（1-2天）

  • 外界因素點火：政策利空、國際股市崩跌、黑天鵝事件
  • 股價連續下跌，融資維持率逼近130%
  • 關鍵數字：股價跌約22%（0.78倍買進價）即觸發追繳

第二階段：擴散期（3-5天）

  • 券商發出追繳令，投資人無法補繳
  • 券商市價拋售不計成本，形成額外賣壓
  • 賣壓壓低更多個股，觸發新一輪追繳
  • 成交量爆增（2025/4/8成交量5,779億，為前日的3.5倍）

第三階段：結束期（5天後）

  • 融資水位降至安全區，被迫賣出完畢
  • 市場逐漸止穩，法人開始低接
  • 底部形成的先決條件：融資賣壓釋放完畢

時間融資減少金額減少比率後續走勢

融資斷頭後的底部判斷三信號

信號一：融資維持率接近或跌破130%

  • 當大盤融資維持率逼近130%，代表斷頭賣壓已出籠
  • 歷史經驗：融資維持率跌破130%後，下跌壓力大幅減輕
  • 這是「危機入市」的條件之一

信號二：融資減少金額開始遞減

  • 連續2-3天融資減少，但金額逐日縮小
  • 代表被迫賣出的力道正在衰竭
  • 第一天減136億→第二天284億（加劇）→第三天開始遞減→底部接近

信號三：法人籌碼開始分化

  • 外資由賣轉買（尤其上櫃市場）
  • 投信持續加碼持股
  • 自營商避險部位縮減
  • 法人開始承接融資斷頭的浮額，代表他們認為已到合理買點

融資斷頭後的「資金回流期」布局策略

  • 第5-10個交易日：資金回流至防禦型高息股（金融、電信）
  • 第15-20個交易日：資金回流至基本面穩健的龍頭股
  • 第25-30個交易日：資金回流至超跌且基本面未受損的成長股

避開高融資風險股三條件

1. 融資餘額占流通股數比例超過15%
2. 近期融資減少未達融資增加的30%（潛在斷頭風險）
3. 流動性差且股價已跌破多條均線

關注法人與融資「背離」的個股

  • 外資持續買超 + 融資水位穩定 = 最佳承接標的
  • 投信與自營商同步買超的防禦型龍頭
  • 融資用途為長期投資（非當沖）的高息股

二、借券賣出餘額 vs 借券餘額

兩者差異

  • 借券餘額：借入股票尚未歸還的總量（包含尚未賣出的部分）
  • 借券賣出餘額：借入的股票中，已在市場中賣出的數量

關鍵差異：借券人借入股票不一定馬上賣出，所以借券餘額 ≠ 借券賣出餘額。兩者之間的差額代表「借了但還沒賣」的部分。

借券的真正用途（不是只有看空）

1. 放空：借券賣出，低檔回補
2. 避險：外資持有現股，借券賣出避險（不等於看空）
3. 套利：期現貨價差套利、ETF套利
4. 資金暫泊：外資匯入資金後，必須依申報目的投入股市，但看好時機未到，先借券暫時持有

借券賣出實戰選股策略

三條件同時符合：
1. 外資連續3日賣超
2. 借券賣出連3日增加
3. 股價跌破季線

實例：2022年1月富邦媒(8454)

  • 外資連續賣超3天
  • 股價跌破所有均線
  • 借券賣出餘額連日升高
  • 本益比高達80倍
  • 同業網家(8455)也跌破月線、季線

→ 減碼或放空的合理標的

借券判讀陷阱

  • 借券餘額增加 ≠ 一定看空（可能是避險或套利）
  • 應搭配其他空方指標交叉驗證（外資賣超、均線、融資變化）
  • 單一指標做為進出依據，期望值不理想

三、籌碼集中度六大指標實戰

籌碼集中的股票有「易漲難跌」的特性。六大指標分為兩類：

絕對指標（判斷籌碼是否已集中）

1. 千張大戶持股比率

公式：持有1,000張以上股東的持股合計 ÷ 股本（張數）

  • < 40%：籌碼太分散，不利炒作
  • 40%～70%：理想區間，籌碼適度集中
  • > 80%：過度集中，市場參與者不願進場，反而不易起漲

實戰重點：同樣的持股比率，持有「人數」越少，炒作動機越強。

2. 內部人持股比率

公式：內部人持股合計 ÷ 股本（張數）

內部人 = 董事、監察人、經理人、大股東及關係人

  • > 40%：高水位，公司派掌握籌碼，但可能欠缺人為炒作動機
  • < 15%：低水位，需注意公司派是否持股信心不足
  • 雙面性：從籌碼面看，內部人持股太高不利炒作；從基本面看，內部人持股高代表經營者有信心

3. 佔股本比重

公式：（區間買超前15名買超張數 − 區間賣超前15名賣超張數）÷ 股本（張數）

  • 60天統計區間，佔股本比重 > 3% = 有主力在收集籌碼
  • 大型股3%是很大金額，小型股3%可能只有幾千萬
  • 換算市值必須超過1億，才構成主力炒作動機

相對指標（判斷籌碼是否開始集中）

4. 籌碼集中度

公式：（區間買超前15名買超 − 區間賣超前15名賣超）÷ 區間成交量

  • 60天集中度 > 5% 為佳
  • 120天集中度 > 3% 為佳
  • 1天集中度 > 20% 才認定有特定人收集
  • 成交量越小，可靠度越低

5. 主力買賣超

公式：每日買超前15名買超合計 − 每日賣超前15名賣超合計

  • 與股價正相關：上漲時買超 > 賣超，下跌時賣超 > 買超
  • 反向走勢才是關鍵：股價漲但主力賣超 = 100%有主力介入，左手賣右手買
  • 主力可能在漲勢中「倒貨」給散戶

6. 買賣家數差

公式：今日買進該股的券商家數 − 今日賣出該股的券商家數

  • 賣出家數 > 買進家數 = 「多數人把股票賣給少數人」= 籌碼趨向集中
  • 不能只看一天，必須搭配「主力買賣超」綜合分析
  • 家數統計與張數統計互補，缺一不可

六大指標綜合分析SOP

步驟一：先看絕對指標

  • 千張大戶持股40-70%
  • 內部人持股適中（15-40%）
  • 佔股本比重 > 3%（60天）

步驟二：再看相對指標

  • 籌碼集中度60天 > 5%
  • 主力買賣超與股價走勢是否背離
  • 買賣家數差是否趨向集中

步驟三：最終確認

  • 四個以上指標同時指向籌碼集中 = 高機率有行情
  • 絕對指標具備先天優勢 + 相對指標出現集中趨勢 = 最佳進場時機

心得

融資斷頭連鎖效應是市場最恐怖的惡性循環機制，但也是最有規律可循的底部信號。歷史數據清楚告訴我們：融資大減不是末日，而是去槓桿的痛苦過程。當融資維持率跌破130%、融資減少金額開始遞減、法人籌碼分化出現——三個信號同時出現時，就是準備進場的時機。

籌碼集中度六大指標則是從另一個角度切入：不是看「誰在賣」（融資斷頭），而是看「誰在買」。當多數人把股票賣給少數人、千張大戶持股上升、主力買超與股價同步——這就是籌碼集中的經典特徵，易漲難跌。

兩個觀點合在一起就是完整的籌碼面戰略：大跌時看融資斷頭判斷底部，底部確認後看籌碼集中度找標的。這才是籌碼面分析的完整拼圖。

籌碼面分析：土洋對作辨識與實戰策略

土洋對作，是台股市場中最常見也最困擾散戶的籌碼現象——外資和內資（投信、自營商、政府基金）看法分歧，一邊買一邊賣，散戶夾在中間被兩面巴。搞懂土洋對作的本質、誰勝誰負的判斷邏輯、以及如何在這種混亂中找到方向，是籌碼面分析的進階必修課。

一、土洋對作是什麼

是什麼

「土洋對作」指本土資金（投信、自營商、政府基金）與外資在同一檔股票或整體市場上方向相反——一個買、一個賣。

  • 土：本土資金 = 投信 + 自營商 + 政府基金（國安基金、退撫等）
  • 洋：外資 = 海外基金、ETF、避險基金、假外資

為什麼會出現土洋對作

1. 投資邏輯不同：外資看全球資金流向（美元指數、國際政經），投信看台灣個股基本面和題材
2. 資金屬性不同：外資資金大但受國際資金流動影響，投信資金小但有地緣優勢
3. 時間框架不同：外資可能因為全球資產配置調節而賣出台股（非看壞個股），投信則是基於產業研究而買進
4. 避險需求：外資賣出現貨可能是為了避險或調節期貨部位，不等於看壞

對散戶的影響

土洋對作時，市場方向不明，最常出現「雙巴盤」——今天大漲、明天大跌，散戶被兩邊打臉。這就是「仙人打架，凡人遭殃」。

二、土洋對作六種型態

型態一：外資賣＋投信買（最常見）

情境：外資因全球資金調節賣出，投信基於基本面持續買進

怎麼判斷誰會贏：
1. 看籌碼在誰手中：外資持股高（>20%）→ 外資主導；投信持股高 → 投信主導
2. 看產業前景：成長產業 → 投信通常對；循環衰退產業 → 外資通常對
3. 看股價位置：低檔區投信買 → 勝率高；高檔區投信買 → 可能是作帳行情

實例：

  • 同欣電（6271）：2021年5月外資大賣14,287張，投信買超9,070張，當月低點117元從此沒見過，股價後來衝上280元 → 投信贏
  • 友達（2409）：2021年6月投信大買13.6萬張，外資狂賣43.1萬張，股價從26.8元一路跌至16.25元 → 外資贏

型態二：外資買＋投信賣

情境：外資開始回補，投信獲利了結或停損出場

怎麼判斷：
1. 看投信賣出原因：是獲利了結（好訊號）還是停損出場（壞訊號）
2. 看外資買進力道：連續3日以上買超且金額遞增 → 趨勢反轉
3. 看股價反應：外資買但股價不漲 → 還在吸收投信賣壓；外資買且股價止穩 → 賣壓消化完畢

實例：

  • 智原（3035）：股價由投信買上來，也由投信賣下去，外資接手買進，雖然沒有拉升股價但已看到止穩現象
  • 嘉澤（3533）：投信10月中旬持續調節，外資反手買進，股價回到前波高點，本益比18倍不高

型態三：外資賣＋政府基金買（護盤）

情境：外資因國際因素大賣，政府基金進場護盤

判斷重點：

  • 護盤是「擋跌不擋漲」，只能減緩跌幅，不能改變趨勢
  • 如果外資賣超持續且金額大（>500億/日），護盤撐不住
  • 護盤買的標的以權值股為主（台積電、金融股），不是題材股
  • 操作建議：護盤期間不追高，等外資賣超縮小再進場

型態四：外資買＋自營商賣

情境：外資看多佈局，自營商短線獲利了結

判斷重點：

  • 自營商本來就是短進短出，賣出不一定看壞
  • 外資買進如果是連續且帶量，參考價值遠高於自營商
  • 操作建議：跟外資方向，忽略自營商短線進出

型態五：三大法人同買（最強訊號）

情境：外資、投信、自營商同步買超

判斷：

  • 三大法人同買 = 高度看好後市，勝率最高
  • 但要注意金額：如果合計買超金額不到50億，力道不夠
  • 如果連續3日以上三法同買，趨勢性極強

型態六：三大法人同賣（最弱訊號）

情境：外資、投信、自營商同步賣超

判斷：

  • 三大法人同賣 = 高度看壞後市
  • 但如果是結算日、除息旺季的技術性賣超，不一定是看壞
  • 需搭配期貨未平倉量判斷：現貨賣＋期貨空單增加 = 真看空

三、土洋對作三大判斷準則（楊惠宇方法）

準則一：籌碼在誰手中

邏輯：誰握有最大部位的籌碼，誰就能主導股價方向

怎麼做：
1. 查外資持股比例（證交所/集保可查）
2. 外資持股 > 20%：外資有定價權，偏向跟外資
3. 外資持股 < 5%：外資影響力有限，偏向跟投信
4. 中間地帶（5%-20%）：看第二個準則

實例：

  • 長榮（2609）：法人持股29%，外資占26%，外資連6賣時股價持續受壓 → 籌碼在外資手上，跟外資

準則二：產業是否有前景

邏輯：如果產業長期趨勢明確，短期的法人分歧反而是買點

怎麼做：
1. 確認產業是否處於成長趨勢（AI、高速傳輸、電動車等）
2. 產業趨勢明確 + 投信買超 = 高勝率跟單
3. 產業前景不明 + 外資賣超 = 避開

實例：

  • 智原（3035）：IP矽智財產業是主流，投信賣完外資接手 → 產業前景支撐，最終止穩
  • 威鋒（6756）：USB4.0高速傳輸是趨勢，投信賣超1,111張、外資買進1,806張，股價接近歷史新高

準則三：股價是否合理

邏輯：合理的股價 + 法人買超 = 安全邊際；不合理的股價 + 法人賣超 = 加速下跌

怎麼做：
1. 計算當前本益比與產業合理本益比區間
2. 本益比低於產業中位數 + 外資買超 = 價值被低估，安全
3. 本益比高於產業中位數 + 投信買超 = 可能在作帳，要小心
4. 注意：每個產業的本益比標準不同，不可跨產業比較

實例：

  • 嘉澤（3533）：獲利連年成長，本益比18倍不高 → 外資一買就回神
  • 長榮（2609）：EPS 45元，本益比不到3倍 → 極度低估，但外資持續賣出使股價承壓

四、外資行為背後的關鍵：美元指數

是什麼

外資在台股的買賣行為，與美元指數有強烈相關性，這是理解土洋對作最重要的底層邏輯。

怎麼判斷

美元指數走多頭：

  • 熱錢流出新興市場
  • 外資賣超台股現貨
  • 外資台指期空單留倉
  • 此時投信買超是逆風而行，勝率較低

美元指數走空頭：

  • 熱錢流入新興市場
  • 外資買超台股現貨
  • 外資台指期多單留倉
  • 此時投信買超是順風而行，勝率較高

實例

2023年美元指數突破短均糾結區轉為多頭，對應台股8/2日下跌319點，外資台指期從連續多單留倉轉向連續空單布局，此後外資持續賣超台股。

操作建議

1. 每天看美元指數走勢（DXY），這是外資行為的前置指標
2. 美元轉弱 + 外資開始買超 = 進場訊號
3. 美元轉強 + 外資開始賣超 = 減碼訊號
4. 土洋對作時，如果美元趨勢明確，跟著趨勢那一方走

五、外資重押訊號：三現象確認

當土洋對作持續一段時間後，如何判斷外資是否「動真格」？三個同時出現的現象：

1. 外資留倉部位居三大法人之冠

只比部位大小，不計方向。外資留倉量 > 投信 + 自營商合計，代表外資投入的資金量最大。

2. 外資未平倉量突暴增超過月均量2倍

正常情況下外資留倉量相對穩定，突然暴增2倍以上 = 異常信號，代表外資加大押注。

3. 外資多單留倉與空單留倉走勢一增一減（開口擴大）

不是多空都加碼，而是一邊加碼、另一邊減碼，代表外資方向明確。

同時符合三種現象 = 外資重押趨勢，此時土洋對作的勝負即將揭曉，跟外資方向。

六、土洋對作實戰SOP

步驟一：確認土洋對作存在

1. 查看三大法人買賣超
2. 外資和投信/自營商方向相反持續3日以上 = 土洋對作

步驟二：判斷籌碼主導權

1. 查外資持股比例（>20%外資主導，<5%投信主導）
2. 查美元指數趨勢（多頭偏外資賣、空頭偏外資買）
3. 查期貨未平倉量（外資重押方向）

步驟三：確認產業與估值

1. 產業是否處於成長趨勢
2. 股價本益比是否合理
3. 有無基本面催化劑（法說會、新產品、營收成長）

步驟四：決定策略

情境策略
外資主導+產業前景佳等外資轉買再進場
投信主導+產業前景佳跟投信買，設嚴格停損
外資主導+產業前景差避開或放空
投信主導+產業前景差不碰
三法同買+產業前景佳積極進場
三法同賣空手觀望

步驟五：停損設定

  • 土洋對作期間波動大，停損要設在支撐線下方2-3%
  • 如果籌碼主導方翻轉（外資從賣轉買或從買轉賣），也要相應調整部位

七、土洋對作五大陷阱

陷阱一：外資賣超 ≠ 看壞

外資可能因為全球資產配置調節、避險需求、期貨結算操作而賣出，不代表看壞個股或台股。曾有外資「低賣高買」的情況——先賣出籌措資金追其他市場強勢股，賺夠了再把資金挪回來。

陷阱二：投信買超 ≠ 看好

投信買超可能是「作帳行情」（季底拉抬績效）、也可能是「被迫買進」（基金有最低持股限制70%），不一定是真的看好。

陷阱三：政府基金護盤 ≠ 底部

政府基金護盤只能減緩跌勢，不能改變趨勢。如果外資持續大賣，護盤買的部位最終也會變成套牢籌碼。

陷阱四：假外資混淆

本土資金透過外資券商下單，看起來是外資但操作邏輯像散戶。辨識方法：中小型股出現外資買超且過去常隔日沖 = 假外資。

陷阱五：結算日外資數據失真
