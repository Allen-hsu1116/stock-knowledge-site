核心知識

融券強制回補的5大原因

是什麼：融券放空後，某些情況下券商會強制你買回股票還券，這就是「強制回補」。

5大原因：
1. 除權息強制回補 — 停止過戶日前6個交易日，融券必須回補
2. 股東會強制回補 — 開股東會前也要回補（通常5-6月高峰）
3. 現金增資 — 公司辦現金增資時強制回補
4. 減資 — 公司減資時強制回補
5. 下市 — 股票下市時強制回補

實例：

  • 情境：某檔股票股東會在6/15，停止過戶日是5/30
  • 操作：融券必須在5/22前回補（停止過戶日前6個交易日）
  • 時間點：3-4月就開始有強制回補壓力，因為往前推60天

適用時機：

  • 放空前務必查「除權息日程」和「股東會日期」
  • 5-6月是融券回補高峰期，這段時間放空風險最高

券資比：軋空風險的關鍵指標

是什麼：券資比 = 融券餘額 ÷ 融資餘額，代表放空者佔融資者的比例。

怎麼判斷：

  • 券資比 < 10%：正常範圍，風險較低
  • 券資比 10-30%：需要注意，觀察主力動向
  • 券資比 > 30%：高危險區，軋空風險大增

實例：

  • 情境：創惟(6104) 在 2022/1/7-2/11 期間
  • 數據：券資比超過30%，融券激增，大戶持股增加，主力買超
  • 結果：融券被軋空，股價持續上漲

適用時機：

  • 放空前一定要查券資比
  • 券資比 > 30% + 主力買超 = 不要放空

軋空行情的形成機制

是什麼：大量放空者被迫買回股票，造成股價急速上漲的現象。

形成過程：
1. 空頭部位累積：散戶看壞後世，融券增加
2. 投機者進場：大戶發現放空過多，開始買進
3. 價格上漲：買盤推升股價
4. 強制回補連鎖反應：

  • 融券維持率不足被券商強制回補

5. 價格急速上漲：大量回補單推升股價

實例：

  • 情境：雙鴻(3324)、創惟(6104) 2025年案例
  • 數據：低檔融券激增、基本面佳（營收創新高、本益比低）
  • 結果：股價上漲，融券被迫回補，形成軋空

避開被軋空的三個關鍵指標

怎麼做：放空前一定要看這三個指標

1. 大戶持股比例

  • 大戶持股增加 → 可能有內線利多 → 不要放空
  • 大戶持股減少 → 才考慮放空

2. 主力買賣超

  • 主力買超 + 融券增加 → 散戶在放空，主力在做多 → 軋空前兆
  • 主力賣超 + 融券增加 → 才是合理的放空訊號

3. 券資比

  • 券資比 > 30% → 高風險，避開
  • 券資比 < 10% → 相對安全

實例判斷：

情境：某股票融券大增
檢查：大戶持股 ↑ + 主力買超 + 券資比 > 30%
結論：這是散戶在放空，大戶在做多 → **被軋空機率極高，千萬不要放空**

軋空行情的操作策略（做多角度）

適用時機：想搭軋空順風車時

怎麼做：
1. 找標的：融券激增 + 基本面佳 + 低檔
2. 確認訊號：大戶持股增加 + 主力買超
3. 等待起漲：股價突破整理區，融券開始回補
4. 設停損：跌破近期低點出場

風險：

  • 軋空行情爆發快，進場要快
  • 已經大漲一段的軋空股，追高風險大
  • 要有停損紀律
