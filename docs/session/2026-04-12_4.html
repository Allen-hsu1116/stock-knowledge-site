核心知識

突破回踩策略（Breakout-Pullback）

是什麼：價格突破阻力或支撐後，回到突破點進行「回踩測試」，確認突破有效後才進場。這是技術派交易者最推崇的高勝率入場法則。

核心邏輯：

  • 突破後，原本的阻力會變成新的支撐（反之亦然）
  • 回踩確認突破有效性，避免假突破陷阱
  • 等待回踩才進場，風險更低、勝率更高

突破回踩六步驟實戰

步驟一：識別盤整區間

怎麼做：
1. 找出明確的支撐位和阻力位
2. 確認價格在一個相對狹窄的區間內波動
3. 用水平線標出盤整區域

實例：

  • 某股在 100-110 元區間震盪三週
  • 標記支撐 100、阻力 110
  • 這是「突破回踩之前的寧靜時刻」

步驟二：等待突破出現

怎麼做：
1. 觀察價格突破阻力或跌破支撐
2. 此時「不要急著進場」
3. 警惕假突破，等待確認

注意：

  • 突破發生時，原本支撐變阻力、阻力變支撐
  • 這只是形態剛出現，還不成熟

步驟三：尋找回踩機會

怎麼做：
1. 觀察價格是否回到突破點
2. 確認價格在原阻力/支撐位獲得支撐
3. 出現反轉訊號（如看漲 K 線）

看漲突破回踩原理：

  • 買盤湧入 → 價格上漲 → 觸發買入止損單
  • 空頭開始賣出 → 價格略微回落
  • 價格回到支撐位 → 看漲交易者再次進場 → 價格繼續上漲

步步驟四：執行交易

怎麼做：
1. 確認回踩成功（價格在原阻力位止穩）
2. 出現反轉訊號（K 線、量能放大）
3. 進場做多或做空

實例：

  • 某股突破 110 阻力後，回踩到 110 附近
  • 在 110 出現看漲吞沒線 + 量能放大
  • 進場價：111
  • 這是高勝率的入場點

步驟五：設置停損與停利

怎麼做：
1. 停損設在回踩點下方 2-3%
2. 停利設在前波高點或阻力位
3. 確保盈虧比至少 2:1

實例：

  • 進場價：111
  • 停損：108（約 2.7%）
  • 停利：120（約 8%）
  • 盈虧比：8% ÷ 2.7% ≈ 3:1

步驟六：監控交易

怎麼做：
1. 持續觀察價格動態
2. 記錄交易表現
3. 檢討停損停利是否適當

關鍵問題：

  • 停損是否留出足夠空間讓價格完成回踩？
  • 停利是否設得太保守？
  • 每次覆盤都是成長機會

回踩低吸買點三條件

條件一：下方有獲利籌碼峰作為依託

怎麼做：
1. 查看籌碼分佈圖
2. 找出獲利籌碼集中的價位
3. 回踩到籌碼峰附近 = 有支撐

條件二：結合趨勢線或均線共振

怎麼做：
1. 畫出趨勢支撐線（連接低點）
2. 或參考重要均線（MA5/MA13/MA144）
3. 回踩同時觸及趨勢線 + 均線 = 強支撐

實例：

  • 某股市場平均成本 17.57 元
  • MA13 均線位置 17.61 元
  • 回踩到 17.5-18 元區間 = 低吸買點

條件三：板塊指數時間共振

怎麼做：
1. 參考對應板塊指數
2. 確認板塊指數也處於支撐位或拐點
3. 個股 + 板塊同時止跌 = 高勝率

重要性：

  • 大盤不好時，80% 的股票都不好
  • 結合時間維度，勝率提高不是一點半點

突破回踩的圖表形態

1. 盤整區間

特徵：價格在狹窄區間波動，支撐阻力明確

怎麼做：

  • 用水平線標出支撐阻力
  • 等待突破和回踩確認

2. 通道形態

特徵：價格在略微傾斜的狹窄範圍內波動

注意：

  • 斜向通道也可能出現突破回踩
  • 識別難度較高，需要更多練習

3. 楔形形態

特徵：支撐線和阻力線斜率不同，逐漸收斂

怎麼做：

  • 收斂 = 趨勢可能減弱或反轉
  • 楔形突破後常伴隨回踩

4. 三角形形態

上升三角形：

  • 水平阻力 + 逐漸抬高的低點
  • 偏向看漲突破

下降三角形：

  • 水平支撐 + 逐漸降低的高點
  • 偏向看跌突破

對稱三角形：

  • 兩條收斂的趨勢線
  • 方向不明確，視為「觀望訊號」

  • 突破阻力後，最終上漲至 72 美元
  • 漲幅近 400%

關鍵：

  • 識別盤整區間（15 美元阻力）

心得

突破回踩是技術分析的經典戰法，核心概念很簡單：「突破後別急，等回踩確認」。

以前常犯的錯誤是看到突破就想追，結果常常被假突破騙。現在知道要等價格回到突破點測試，確認支撐有效才進場。這樣雖然可能錯過一些直接續漲的機會，但勝率會高很多。

最實用的是「三條件共振」：籌碼峰 + 趨勢線/均線 + 板塊指數。以前只看價格支撐，忽略了籌碼和大環境的配合。現在做回踩低吸會先查籌碼分佈，再看均線和板塊指數是否同步。

另外一個重點是「時間維度共振」。大盤不好時，80% 的股票都不好，這句話要記住。不要想著「輕大盤重個股」，那是給自己找麻煩。
