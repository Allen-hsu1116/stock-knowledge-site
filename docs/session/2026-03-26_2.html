核心知識

動能投資核心概念

是什麼：相信「強者恆強、弱者恆弱」，只有當股票出現動能（資金入場、價格強勢）時才是最佳進場時機。

為什麼有效：

  • 有動能 = 有資金關注 = 更容易繼續上漲
  • 沒動能 = 資金暫時不感興趣 = 上漲機會低、效率差

五大動能策略類型

1. 價格動能策略

怎麼做：
1. 計算過去一段時間的漲幅（如過去20天）
2. 篩選漲幅超越平均的股票
3. 進場追蹤強勢股

實例：

  • 情境：台積電過去20天上漲12%
  • 判斷：同期大盤上漲5%，台積電動能強
  • 操作：進場追蹤，停損設5%

適用時機：明顯多頭市場，避免盤整期

2. 均線交叉動能策略（MA交叉）

怎麼做：
1. 設定兩條均線：快速線（如5日）、慢速線（如20日）
2. 快速線向上穿越慢速線 = 黃金交叉 = 買進訊號
3. 快速線向下穿越慢速線 = 死亡交叉 = 賣出訊號

實例：

  • 情境：聯發科5日均線在380元，20日均線在365元
  • 判斷：5日均線剛突破20日均線 = 黃金交叉
  • 操作：買進，停損設20日均線下方3%（約354元）

適用時機：趨勢明確時有效，盤整時會頻繁假訊號

3. 成交量動能策略

怎麼做：
1. 觀察成交量異常放大（如5日均量翻倍）
2. 確認價格同步上漲（量價齊揚）
3. 進場後觀察量能是否持續

實例：

  • 情境：某股平時日成交量2,000張，今天爆量至8,000張
  • 判斷：量增3倍+價漲6% = 強勁動能
  • 操作：進場，若隔日量縮價跌則快速出場

適用時機：突破關鍵價位時，成交量是不可缺少的確認

4. 量價策略

怎麼做：
1. 要求「有價要有量，有量要有價」
2. 價漲但量縮 = 動能不足 = 不進場
3. 量增但價跌 = 出貨嫌疑 = 不進場

實例：

  • 情境A：鴻海漲停，成交量10萬張（平日3萬張）= 量價齊揚 ✓
  • 情境B：某股漲5%，但成交量只有平日的60% = 動能不足 ✗
  • 操作：只選擇量價齊揚的標的

適用時機：所有動能操作都應該用此原則過濾

5. 反轉策略（逆向動能）

怎麼做：
1. 觀察RSI或乖離率判斷是否過度乖離
2. RSI > 80 = 超買 = 可能反轉向下
3. 乖離率過大（如+15%）= 短期回檔機率增加

實例：

  • 情境：某股連漲10天，RSI來到88
  • 判斷：極度超買，動能即將耗盡
  • 操作：不追高，甚至考慮反向操作

適用時機：行情過熱時，作為「何時不追」的判斷

TEJ動能策略實戰數據

回測設定（2019-2025）：

  • 股票池：市值前200大（確保流動性）
  • 動能計算：過去20天報酬率
  • 持股：每月月初買入動能前5強
  • 部位：每檔20%（分散風險）

回測結果：

  • 年化報酬率：28.26%
  • 年化波動度：37.25%
  • 最大回檔：超過30%

關鍵心得：
1. 波動大，需要強心臟
2. 獲利來自少數幾次大趨勢（如2021陽明海運）
3. 適合能承受高風險的投資者

動能投資的優缺點

優點：

  • 捕捉強勢股，資金效率高
  • 操作簡單明瞭
  • 適用多種市場（股票、期貨、ETF）

缺點：

  • 盤整期頻繁進出，交易成本高
  • 入場時已漲一段，風險較大
  • 容易忽略基本面
  • 假突破會造成虧損

如何降低風險：

  • 只在市值前200大選股（降低被操控風險）
  • 分散持股（至少5檔）
  • 嚴格停損（5-10%）
  • 避開盤整期（用均線判斷是否有趨勢）

實戰檢查清單

進場前確認：

  • [ ] 是否有明確趨勢？（非盤整）
  • [ ] 價量是否齊揚？
  • [ ] 是否符合股票池條件？（市值前200）
  • [ ] 停損點設好了嗎？

出場時機：

  • 動能消失（量縮價跌）
  • 停損觸發
  • RSI進入極端區域（超買）
  • 月底再平衡（賣出動能轉弱的）

心得

動能投資的核心就是「不要猜底，要追確認的趨勢」。這跟之前學的突破交易有點像，但更強調「量」和「持續性」。

最有趣的是TEJ的回測數據：年化28%報酬率很吸引人，但37%波動度就讓人抖抖的。關鍵在於，這種策略是「大賺小賠多次」，靠少數幾次大趨勢把整體拉上去。所以心理建設很重要——要能承受多次小虧損，才能抓住那幾次大行情。

另外，動能策略最怕的就是盤整，因為會一直被洗出去。所以判斷「現在是不是盤整」很重要，可以用之前學的均線糾結或區間判斷來輔助。
