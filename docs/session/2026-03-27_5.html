核心知識

交易日誌四步驟框架

是什麼：一套結構化的交易記錄方法，包含買進、持有、賣出、檢討四個階段，幫助交易者從錯誤中學習、建立紀律。

怎麼做：
1. 買進後記錄：日期、張數、股票、理由、技術分析截圖（含停損、支撐壓力）
2. 持有期間記錄：設定出場規則（停損點、停利點、加碼點）、記錄心情變化
3. 賣出時記錄：時間、價格、損益、賣出理由（與買進理由對照）
4. 檢討（一個月後）：賣太早？買太晚？策略改進？勝率計算？

實例：

  • 情境：買進台積電 100 元，預期突破
  • 操作：截圖標記買進點 100 元、停損 95 元、壓力 110 元
  • 持有期間：每日記錄心情「今天跌到 97 很緊張，想賣但堅持規則」
  • 賣出：105 元出場，賺 5%
  • 檢討：停利設 110 卻沒等到，下次可提高耐心

適用時機：所有交易都需要，中短線交易者尤其重要；當沖不適合逐筆寫，可改用交易統計

買進理由的關鍵差異

是什麼：區分「主觀覺得會上漲」vs「符合買點策略才買」的心態差異。

怎麼做：
1. 寫下買進理由時，問自己：「這是預測還是策略？」
2. 策略型理由：「股價突破盤整區上緣 100 元，成交量放大，符合突破買進規則」
3. 預測型理由（危險）：「我覺得這檔股票會上漲」
4. 訓練自己成為「機械化交易者」，不是「主觀交易者」

實例：

  • ❌ 錯誤：「這檔感覺會漲，最近新聞很熱」
  • ✅ 正確：「股價站上 20 日均線，且均線向上，符合順勢策略買進條件」

適用時機：每次買進前都要問自己這個問題

持有期間規則設定

是什麼：下單後立即訂好的出場規則，避免盤中情緒決策。

怎麼做：
1. 時間停損：「在 3/30 如果沒到 105 元就賣出」
2. 空間停損：「跌破 95 元（-5%）停損出場」
3. 最大可能虧損：假設大盤突然崩跌，這筆交易最大虧損是多少？是否能承受？
4. 看盤時間：「只看開盤 9:00-9:30 和尾盤 13:00-13:30」
5. 停利設定：「獲利超過 10% 或到達 110 元停利」
6. 加碼條件：「突破 105 元再買 1 張」

實例：

  • 買進 100 元，1 張
  • 停損：95 元（最大虧損 5,000 元）
  • 停利：110 元（獲利 10,000 元）
  • 時間：持有最多一週
  • 加碼：突破 105 元可加碼 1 張

適用時機：下單後立即設定，不要盤中才想

極端狀況思考

是什麼：預先設想「最壞情況」的應對方案，避免突發事件時手足無措。

怎麼做：
1. 問自己：「如果大盤突然崩跌 500 點，這檔股票跌停賣不掉怎麼辦？」
2. 問自己：「跌破停損價但賣不掉時，我要認賠還是等反彈？」
3. 預先寫下答案，作為極端狀況的操作依據
4. 「永遠都要為任何狀況做好準備，否則你的交易就沒有永遠」

實例：

  • 情境：買進 100 元，停損 95 元
  • 極端狀況：開盤直接跳空到 90 元
  • 預設答案：若跌停賣不掉，等反彈到開盤價附近再出場（不追殺）

適用時機：持有期間每天都要記住

檢討的十大問題

是什麼：一個月後對照技術線型，用馬後炮視角找出改進點的問題清單。

怎麼做：
1. 賣太早？ 原本應該等到哪裡才賣？
2. 買太晚？ 錯過哪些訊號？
3. 策略改進：什麼條件應該加入策略？
4. 停損時間：該提早還是延後？
5. 停損範圍：該放寬還是縮小？
6. 理由一致性：買進理由和賣出理由是否一致？
7. 資金比例：每筆下多少比例才適合？
8. 勝率計算：這策略的勝率多少？
9. 個性調整：這策略適合自己的個性嗎？
10. 持股效率：資金是否有效運用？

實例：

  • 檢討發現：10 次交易有 7 次賣太早，都只賺 3-5%，被洗出後又續漲 15%
  • 改進：停利改用移動停利，讓獲利跑更遠

適用時機：每筆交易結束後一個月，截圖當時、一週後、一個月後的走勢

賺錢不代表行為正確

是什麼：用運氣賺的錢，最終會憑實力輸回去；要區分「運氣」和「能力」。

怎麼做：
1. 賺錢時問自己：「我真的做對了嗎？」
2. 檢查：是否按策略執行？還是偏離策略但剛好運氣好？
3. 賠錢時問自己：「我真的做錯了嗎？」
4. 檢查：策略執行是否正確？賠錢可能是機率問題，不是策略問題
5. 「靠運氣贏來的，最後還是會憑實力輸回去」

實例：

  • ✅ 做對但賠錢：策略說買進 A 點、停損 B 點，確實執行，但股價真的跌破 B 點
  • ❌ 做錯但賺錢：策略說等突破才買，結果提前買進，剛好運氣好股價續漲

適用時機：每次檢討都要問這兩個問題

交易策略中途不要改

是什麼：進場前想好的策略，在持有期間不能隨意更改。

怎麼做：
1. 進場前先想好：長線還是短線？突破還是拉回？做多還是做空？
2. 持有期間，如果價格走勢不如預期，不要改變策略
3. ❌ 錯誤示範：原本做短線，結果跌了就改口「沒關係我長期持有」
4. 一旦中途改策略，出場點就完全不同，會不知道何時該賣
5. 「當局者迷」：拿著虧損股票時，你會找理由說服自己不賣

實例：

  • 計畫：短線追突破，買進 100 元，預期漲到 110 元
  • 錯誤：跌破 98 元，心想「這公司基本面不錯，我就長期持有吧」
  • 後果：跌破 90 元、80 元...虧損擴大
  • 正確：按計畫停損出場，下次再找機會

適用時機：每次持有期間，盤中不要改策略

損益比例計算方法

是什麼：不只看單筆損益金額，還要計算占資產的比例，找出最會賺錢的市場。

怎麼做：
1. 紀錄這筆交易賺賠多少錢
2. 計算這筆交易占「該資產」的報酬率（例如：加密貨幣這筆賺 50%）
3. 計算這筆交易占「整體資產」的報酬率（例如：加密貨幣只占整體 5%，這筆對整體貢獻 2.5%）
4. 比較不同市場的績效，找出勝率高的市場

實例：

  • 總資產 100 萬
  • 分配：台股 60%、美股 30%、加密貨幣 10%
  • 台股這筆賺 10%，占台股部位 10% → 占總資產 6%
  • 加密這筆賺 100%，占加密部位 100% → 占總資產 10%
  • 結論：加密這筆賺更多，但風險也高；可調整加密部位

適用時機：每筆交易都要計算，找出最佳配置

加碼比例：左側 vs 右側交易

是什麼：根據交易策略（左側、右側）決定不同的加碼方式。

怎麼做：
1. 右側交易（短線）：從大到小加碼

  • 一開始丟大部位，後面少量加碼

2. 左側交易（長線）：從小到大加碼

  • 一開始小部位，後面慢慢加碼

實例：

  • 右側（突破追價）：100 元買 5 張，103 元買 2張，均價約 100.8 元
  • 左側（定期定額）：每月買 1 張，長期平均成本

適用時機：短線順勢用右側加碼，長期投資用左側加碼

心情起伏紀錄的重要性

是什麼：紀錄價格波動時的心理變化，找出自己容易犯錯的情緒觸發點。

怎麼做：
1. 記錄：「看到價格從 105 跌到 100，我很緊張，想賣掉」
2. 記錄：「看到價格突破 110，我覺得會續漲，想加碼」
3. 之後檢討時對照：當時的情緒判斷是否正確？
4. 找出情緒模式：什麼情況下容易衝動？

實例：

  • 發現：每次價格跌破波段低點，我就會恐慌賣出
  • 檢討：事後看有一半是假跌破，之後都反彈
  • 改進：設定好停損價，跌破再賣，不要因恐慌提前出場

適用時機：持有期間有強烈情緒時記錄

心得

今天學的是交易日記的寫作方法，這個主題在之前的 topicsCovered 裡有「復盤交易檢討方法」，但今天學到的是更完整的交易日誌框架。

四大核心收穫：

1. 交易日記不是流水帳：很多人以為只是記錄買賣價格，但其實更重要的是「買進理由」、「持有期間規則」和「檢討」。90%以上投資人忽略的是持有期間的記錄——這才是紀律的關鍵。

2. 下單後立即設規則：不要在盤中思考停損停利，要在下單後馬上設定好。盤中的情緒會影響判斷，規則先訂好，盤中只要「執行」就好。這點我覺得很重要，因為盤中決策很容易被恐懼和貪婪綁架。

3. 一個月後才檢討：不要馬上檢討，要等一個月。因為「馬後炮」是最好用的——你可以清楚看到之後的走勢，知道當時的決策是否正確。這個時間差是交易日記的精髓。

4. 賺錢不代表做對：這句話超重要。「靠運氣贏來的，最後會憑實力輸回去」。交易日記的價值在於區分「運氣」和「能力」，只有按策略執行的獲利才是真正的能力。

最後華生的這句話我很喜歡：「規則並非是死的，當你認為某一條規則不再適用，你只能下次改進」。這就是交易日記的循環——記錄、執行、檢討、改進、再記錄。紀律不是天生的，是透過這個循環練出來的。

一、槓桿控制與風險管理

槓桿是什麼？

定義：用少量資金（保證金）控制更大交易部位的機制，本質上是向經紀商借錢放大投資。

計算公式：

  • 槓桿倍數 = 交易總價值 ÷ 所需保證金
  • 保證金比例 = 所需保證金 ÷ 交易總價值 × 100%

例子：

  • 帳戶 10,000 元，10 倍槓桿
  • 可控制 100,000 元部位
  • 價格上漲 10% → 獲利 10,000 元（報酬率 100%）
  • 價格下跌 10% → 虧損 10,000 元（全部歸零）

  • < 1,000 美元：新手 10:1，中級 20:1，老手 50:1
  • 1,000-5,000 美元：新手 5:1，中級 10:1，老手 20:1
  • > 5,000 美元：新手 3:1，中級 5:1，老手 10:1

槓桿交易六大風險

1. 放大損失：價格不利變動 1% 可能清空帳戶
2. 追加保證金（Margin Call）：帳戶低於維持保證金被迫平倉
3. 市場快速波動：跳空、快市瞬間穿價
4. 利息成本：隔夜持倉的 Swap 費用
5. 過度自信：高槓桿誘惑做出不理性決策
6. 滑價風險：市價單成交價與預期價格不一致

槓桿風險管理六招

1. 設定停損單

  • 原則：進場同時設好停損
  • 實例：10 倍槓桿，停損設在入場價下 3%
  • 結果：最大虧損控制在保證金的 30%

2. 使用止盈單

  • 鎖定獲利，避免獲利回吐變虧損
  • 建議：報酬風險比至少 1:2

3. 頭寸大小管理

  • 公式：頭寸 = 帳戶 × 風險比例 ÷ (入場價 - 停損價)
  • 實例：帳戶 100,000 元，單筆風險 2%，停損距離 5%
  • 頭寸 = 100,000 × 2% ÷ 5% = 40,000 元

4. 多樣化分散

  • 不要全壓單一標的
  • 建議：單一標的不超過總部位 20%

5. 定期監控調整

  • 根據價格變動調整停損
  • 移動停損保護獲利

6. 持續學習

  • 了解市場新聞與事件
  • 掌握波動變化即時調整策略

二、連續虧損後的心理調整

交易贏家五大原則

原則1：預測行情也要滾動式修正

  • 法人目標價也不一定準
  • 永遠不要把目標價放第一優先
  • 進場前先確定自己能承擔的風險
  • 進場後持續驗證看法是否正確

原則2：進場與出場有憑有據

  • 技術分析是「可能性」不是「保證」
  • 合理計算風險、規劃策略、設好停損
  • 養成好習慣就不會賠大錢

原則3：重複正確的習慣

  • 愈簡單的操作原則愈要長期遵守
  • 要忍受枯燥——紀律是生存關鍵
  • 檢討與記錄交易，盡量不重複犯錯

原則4：容許虧損與失敗

  • 允許依交易系統執行後的虧損
  • 但不能違反「有憑有據」原則
  • 追求完美、不接受停損的人不適合股市

原則5：耐心等待一記好球

  • 巴菲特：「錢會從沒有耐心的人手中轉移到有耐心的人手中」
  • 好的進場需要時間等待
  • 長線思維沒有，短線波動更難掌握

連續虧損恢復三階段

第一階段：止血
1. 停止交易，冷靜下來
2. 檢視虧損原因：策略問題？心態問題？市場問題？
3. 不要急著翻本——急著翻本只會虧更多

第二階段：診斷
1. 回顧交易日記
2. 找出共同錯誤模式
3. 區分「策略虧損」vs「心態虧損」

第三階段：復盤
1. 調整策略或心態
2. 模擬帳戶驗證新策略
3. 小資金重新開始
4. 恢復信心後再放大部位

心理素質關鍵提醒

  • 技術再強也無法預測市場下一步
  • 每筆交易進場前都要先了解風險
  • 永遠別做重壓
  • 心理因素是長期獲利的關鍵差異
  • 長期持續獲利的人「思考方式」跟輸家截然不同

心得

風險管理有兩個層次：技術層面和心理層面。

槓桿控制是技術層面——有明確的公式和建議倍數。重點是：

  • 新手用低倍槓桿（5:1 以下）
  • 單筆風險控制在帳戶 2% 以內
  • 永遠設好停損再進場

心理恢復是更難的層面。連續虧損後最容易犯的錯：
1. 急著翻本
2. 加碼攤平
3. 忽略停損

「容許虧損與失敗」這個觀念很重要——不是所有虧損都是錯誤，按系統執行後的虧損是成本的一部分。怕的是為了「不想認輸」而做出違反系統的操作。

今天複習了槓桿計算和風險控制方法，加上心理層面的恢復策略，整體來說風險管理的核心就是：
技術上控制損失，心理上接受損失。

這兩件事缺一不可。只會算停損但心裡不接受虧損，還是會在關鍵時刻停損失敗；只會說「接受虧損」但不懂槓桿風險，一個大波動就被掃出場。

核心知識

一、凱利公式是什麼

定義：由貝爾實驗室約翰·凱利（John Kelly）於1956年提出的數學公式，解決「在正期望值的機會中，應該投入多少資金」這個核心問題。

目標：最大化長期資本的幾何增長率（複利），而非單次期望值。

核心價值：

  • 平衡獲利潛力與破產風險
  • 避免過度投注導致的毀滅性虧損
  • 避免投注不足錯失增長機會

二、公式形式與計算

基礎形式（二元結果）

公式：f* = (bp - q) / b

變數說明：

  • f*：最佳投注比例（投入資金的多少百分比）
  • b：淨賠率（每投入1元，獲勝時額外贏得多少）
  • p：勝率（成功機率）
  • q：敗率（失敗機率）= 1 - p

實戰形式（基於歷史數據）

公式：K% = W - [(1-W) / R]

變數說明：

  • W：獲勝機率（從歷史交易計算）
  • R：贏/輸比率（平均獲利金額 / 平均虧損金額）

部分損失形式

公式：f* = p/a - q/b

適用情境：當損失不是全部本金時

變數說明：

  • p：勝率
  • q：敗率
  • a：失敗時的損失率（例：虧損25%，a=0.25）
  • b：成功時的收益率（例：獲利20%，b=0.2）

三、實例演練

例1：不公平硬幣

情境：

  • 擲出正面機率 60%（勝率 p=0.6）
  • 擲出反面機率 40%（敗率 q=0.4）
  • 贏的賠率 1:1（淨賠率 b=1）

計算：
f* = (1 × 0.6 - 0.4) / 1 = 0.2

結論：投入資金的 20%

例2：股票投資

情境：

  • 100萬資金
  • 60%機率上漲20%（收益 b=0.2）
  • 40%機率下跌25%（損失 a=0.25）

計算：
f = 0.6/0.25 - 0.4/0.2 = 2.4 - 2 = 0.4

結論：投入 40萬（40%）

例3：分級債券

情境：

  • 價格 0.8 元
  • 漲到 0.95（+18.75%）或跌到 0.75（-6.25%）
  • 漲跌機率各 50%

計算：
f = 0.5/0.0625 - 0.5/0.1875 = 8 - 2.67 = 5.33

結論：可加槓桿投入 533%（實務需考慮槓桿成本）

四、凱利公式的假設與限制

核心假設

1. 準確的機率與賠率：必須準確知道勝率和賠率
2. 可重複的獨立試驗：機會可無限次或大量重複
3. 資金可無限分割：可精確按比例投入
4. 利潤再投資：利潤複利計入本金
5. 長期視角：不保證短期結果

主要風險

1. 估計錯誤風險

  • 高估勝率或賠率 → 過度投注 → 破產風險大增
  • 人類容易過度自信
  • 歷史數據不一定預測未來

2. 高波動性

  • 全凱利可能帶來劇烈資金波動
  • 可能經歷深度回撤（從高點下跌）
  • 波動性本身會侵蝕複利增長

3. 市場動態性

  • 真實市場勝率和賠率會變動
  • 不同資產間存在相關性
  • 基礎公式未考量相關性

4. 黑天鵝風險

  • 無法預測極低機率但影響巨大的事件
  • 金融危機、戰爭、疫情等

  • 根據市場變化調整策略
  • 記錄交易驗證預期假設

八、實戰注意事項

1. 參數估計方法

勝率 (W)：

  • 分析最近 40-60 筆交易記錄
  • 計算方式：獲利交易次數 / 總交易次數

盈虧比 (R)：

  • 平均每次獲利金額 / 平均每次虧損金額
  • 例：平均賺 3,000 / 平均賠 1,500 = 2.0

2. 避免常見錯誤

錯誤一：過度自信

  • 以為自己知道精確的勝率和賠率
  • 實際上很難有把握

錯誤二：忽略市場變化

  • 用歷史數據預測未來
  • 市場狀態會改變

錯誤三：忽視交易成本

  • 手續費、滑價會影響實際勝率
  • 小額交易影響更大

錯誤四：誤用公式

  • 全凱利對大多數人太激進
  • 應使用分數凱利

3. 心理建設

紀律的重要性：

  • 連續虧損時要減少投注（公式要求）
  • 判斷有優勢時要敢於重注（公式建議）
  • 人性往往與此相反

索普的建議：

  • 21點賭局中，從不下注超過總籌碼的 2%
  • 投資中，從不讓單一頭寸風險超過總資金的 2%

九、與風險管理的整合

整合框架

1. 先用凱利計算理論最優
2. 乘以保守係數（0.25-0.5）
3. 設定硬性上限（20-25%）

5. 定期檢討參數估計

與2%規則的關係

  • 凱利關注「投入比例」
  • 2%規則關注「單次風險」
  • 兩者可互補使用

心得

凱利公式是個很精妙的數學工具，但它的「阿基里斯之踵」就在於：你必須準確知道勝率和賠率。這在賭場（如21點算牌）還算可行，但在股票市場？難如登天。

歷史勝率70%不代表下次也70%，市場在變、對手在變、條件在變。所以實務上，我把凱利公式的計算結果當參考用，而不是鐵律。算出來說可以投入40%？我只投10-15%。

最實用的洞見是：
1. 有優勢才下注，沒優勢就不玩：公式算出負數或零就是叫你不要進場
2. 優勢越大，才值得投入越多：不是每次都一樣的倉位
3. 永遠不要全押：即使你有絕對把握，市場也會有黑天鵝

巴菲特說用桶接金子，但他說的是「金子」——確定的高品質機會。普通人看到的是黃銅，卻拿桶去接，下場就是破產。

結論：凱利公式告訴我的不是「怎麼算」，而是「怎麼想」——有優勢、敢重注、懂控制。這才是朝股票操作大師邁進的核心心法。
