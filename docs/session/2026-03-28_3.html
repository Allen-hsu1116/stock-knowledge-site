核心知識

一、期貨未平倉量（OI）基本概念

是什麼：期貨未平倉口數，就是尚未平倉的契約口數。多頭等於空頭（零和市場），代表市場上的「燃料」。

怎麼看：
1. 期交所官網 → 交易資訊 → 三大法人 → 期貨
2. 每日 15:30 後公布當日數據
3. 永豐期貨「盤後快訊」有整理好的圖表

重點公式：

  • 多空淨額 = 多方 - 空方
  • 未平倉變化 = 當日多空淨額
  • 累計部位 = 前一日未平倉 + 當日變化

二、價格、成交量、未平倉量三維判讀

是什麼：結合三個指標判斷盤勢偏多或偏空。單看價格不夠，要配合量能和 OI 才能確認趨勢力道。

怎麼做：

期貨價格成交量未平倉盤面解讀
上升增加增加**偏多** — 價漲量增，新多空進場，趨勢續航力強
上升增加減少**中性** — 多方獲利了結、空方停損，力道減弱
上升減少減少**中性** — 價漲量縮，續航力弱
下跌增加增加**偏空** — 價跌量增，新空單進場，趨勢延續
下跌增加減少**中性偏空** — 空方獲利了結、多方停損
下跌減少減少**中性** — 價跌量縮

實例：

  • 情境：台指期從 18500 漲到 18650，OI 從 15 萬口增至 16 萬口
  • 判讀：價漲 + OI 增加 = 新資金進場，多方攻擊力道強
  • 操作：可順勢做多，停損設在前低

適用時機：日線、週線判斷趨勢方向；不適合當沖（變化太快）

三、法人期貨部位判讀

是什麼：追蹤外資、投信、自營商的期貨淨部位，判斷大戶多空意向。

三大法人差異：

法人部位特性判讀重點
外資部位最大、變化慢長線趨勢風向球
投信部位小、偏現貨較少用期貨避險
自營商部位中等、變化快短線調節、造市功能

怎麼做：
1. 到期交所官網查詢「三大法人期貨未平倉」
2. 計算外資淨部位 = 多方 - 空方
3. 觀察連續 3-5 日的變化趨勢
4. 對照同期現貨買賣超

實例：

  • 情境：外資連續 5 日加碼期貨空單，從 -15000 口增至 -20000 口
  • 判讀：外資看空後市，壓低結算機率提高
  • 操作：偏空思考，但需對照現貨部位（可能是避險非看空）

適用時機：結算前三天特別重要；日線波段操作參考

關鍵提醒：

  • 外資期貨部位 ≠ 真正看法（可能是避險）
  • 要同步看「期貨 + 現貨」部位變化
  • 大外資（五大特法）和小外資行為可能不同

四、避險 vs 投機部位判斷

是什麼：法人期貨部位不一定是「看空」或「看多」，可能是避險用途。判斷錯誤會被誤導。

判斷方法：

情境期貨部位現貨部位解讀
外資空單增加-增加現貨買超增加**避險** — 現貨多頭 + 期貨空單保護
外資空單增加-增加現貨賣超增加**投機/看空** — 雙向看空
外資多單增加+增加現貨賣超增加**套利** — 可能有套利空間
外資多單增加+增加現貨買超增加**投機/看多** — 雙向看多

怎麼做：
1. 每日查看「三大法人買賣超」（現貨）
2. 同步查看「三大法人期貨未平倉」
3. 計算「期貨 + 現貨」淨曝險
4. 判斷是避險還是真實看法

實例：

  • 情境：外資現貨買超 50 億，期貨空單增加 5000 口（約 50 億市值）
  • 判讀：期貨空單是避險部位，不是看空
  • 操作：不用因為「空單增加」就偏空思考

適用時機：長線波段操作、結算日判斷

五、小外資部位指標

是什麼：大外資（五大特法）部位大、調整慢；小外資部位小、調整快。小外資更能反映短期看法。

怎麼做：
1. 外資總期貨部位 - 五大特法總部位 = 小外資部位
2. 觀察小外資淨部位變化
3. 對照短期（1-3 日）行情走勢

實例：

  • 情境：外資總空單 +20000 口，五大特法空單 +25000 口
  • 計算：小外資 = 20000 - (-25000) = 小外資實際是「多單 +5000 口」
  • 判讀：小外資反而看多，跟大外資不同調
  • 操作：小外資短線較準，可參考其方向

適用時機：短線、當沖判斷

六、期貨避險策略（法人角度）

是什麼：法人持有大量現貨時，會用期貨反向部位降低風險。理解這點才不會誤判「空單增加」。

兩種避險方式：

方式情境操作
空頭避險持有現貨、怕跌做空期貨保護
多頭避險未來要買、怕漲做多期貨鎖價

怎麼做（法人視角）：
1. 持有大量股票時，用期貨空單對沖
2. 避險比例 = 期貨市值 / 現貨市值
3. 成本極低（手續費 < 0.01%）
4. 隨時可解除（比賣股票方便）

實例：

  • 情境：法人持有台積電 100 億市值股票
  • 操作：做空 100 億市值台指期（約 3000 口）
  • 效果：股票跌 5% = 現貨賠 5 億，期貨賺 5 億，總損益 = 0
  • 解除：期貨平倉只需一秒，比賣股票快

適用時機：法人操作；散戶資金太小不適合複製

七、法人期貨部位的三大誤區

誤區一：外資空單增加 = 股市要跌

  • 事實：可能是避險，不是真正看空
  • 正確做法：對照現貨部位變化

誤區二：只看今日數據

  • 事實：單日變化可能只是調節
  • 正確做法：觀察 3-5 日趨勢

誤區三：把「籌碼」當「預測」

  • 事實：籌碼是參考，不是 100% 準確
  • 正確做法：配合技術面、基本面綜合判斷
