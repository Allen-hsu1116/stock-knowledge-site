凱利公式（Kelly Criterion）——科學化部位控制

是什麼：凱利公式是一個數學公式，告訴你「在有正期望值的機會中，應該投入總資金的多少比例」，目標是最大化長期資金的幾何增長率（複利），而不是單次下注的期望值。

怎麼做：

1. 基礎公式（二元結果）：f* = (bp - q) / b

  • f* = 建議投入資金比例

2. 投資版公式（考慮損益率）：f* = p/a - q/b

  • a = 虧損時的損失率（如虧25%，a=0.25）

3. 交易回測版公式：K% = W - (1-W)/R

  • W = 歷史勝率（獲利交易次數/總交易次數）

實例：

  • 情境：你有100萬，分析某股票60%機率上漲20%，40%機率下跌25%
  • 計算：f* = 0.6/0.25 - 0.4/0.2 = 2.4 - 2.0 = 0.4
  • 結果：凱利公式建議投入40萬（100萬×40%）
  • 注意：這是「全凱利」，實務上風險偏高

另一個實例：

  • 回測策略：100次交易，勝率45%，平均獲利200元，平均虧損100元
  • 盈虧比 R = 200/100 = 2
  • K% = 0.45 - (1-0.45)/2 = 0.45 - 0.275 = 0.175
  • 建議每次交易投入本金的17.5%

適用時機：已有穩定回測數據的交易策略，用來決定每次交易應投入多少資金。不適合沒有歷史數據的新策略。

Half Kelly 與 Fractional Kelly——降低波動的實務做法

是什麼：全凱利（Full Kelly）是理論最佳比例，但實際應用時波動極大、回撤極深。Half Kelly（半凱利）就是只用凱利公式計算值的50%，Quarter Kelly只用25%。

怎麼做：

1. 為什麼要用 Half Kelly：

  • 全凱利的回撤可能高達50-70%，心理壓力極大

2. 實務建議：

  • 保守型：用 Quarter Kelly（25%）或更低

3. 超過全凱利的後果：

  • 超過凱利比例下注，長期增長率反而下降

實例：

  • 凱利公式算出 f* = 40%
  • Half Kelly：投入 40% × 0.5 = 20%
  • Quarter Kelly：投入 40% × 0.25 = 10%

適用時機：任何使用凱利公式的場景，都建議至少用 Half Kelly。尤其是參數估計不確定時，更應該保守。

凱利公式的致命限制與避坑指南

是什麼：凱利公式看起來完美，但有6個關鍵限制，不理解的話用下去會虧到脫褲。

怎麼做：

1. 估計錯誤風險（最致命）：

  • 勝率 p 和盈虧比 R 都是從歷史數據推估的

2. 高波動與深回撤：

  • 全凱利的最大回撤可能超過50%

3. 市場動態性：

  • 凱利假設 p 和 b 是固定的，但市場隨時在變

4. 黑天鵝風險：

  • 凱利公式基於已知機率分佈，無法處理極端事件

5. 相關性問題：

  • 基礎凱利公式假設每次下注獨立

6. 心理與行為挑戰：

  • 連續虧損時，凱利要求減少下注，但人會想翻本

實例：

  • 你回測某策略勝率60%、盈虧比2:1，凱利算出投入35%
  • 但實際上市場環境變了，真實勝率可能只有50%
  • 50%勝率×2:1盈虧比的凱利值 = 25%
  • 你用35%下注，超過了真實凱利值，長期會虧損
  • 用Half Kelly（17.5%）就安全很多，即使勝率估計有誤差也能承受

適用時機：使用凱利公式前，務必先評估這6個限制。如果你無法確定勝率和盈虧比的準確度，就不要用全凱利。

蒙地卡羅模擬回撤估算（Monte Carlo Drawdown Estimation）

是什麼：用回測的盈虧數據，透過隨機重抽樣產生數千條模擬權益曲線，然後統計每條曲線的最大回撤，得出未來可能面臨的回撤範圍和機率。告訴你「未來最壞可能有多慘」。

怎麼做：

1. 準備數據：收集回測或實盤的逐筆交易盈虧記錄（至少50-100筆）
2. 隨機抽樣：從盈虧記錄中隨機抽取n筆（允許重複），組成一條新的模擬權益曲線
3. 重複模擬：重複上述步驟5000-10000次，產生5000-10000條模擬路徑
4. 計算指標：對每條路徑計算最大回撤、最終報酬率、夏普值等
5. 統計分析：

  • 將5000個最大回撤值做成直方圖

6. 判斷基準：

  • 95% CI 最大回撤 < 你的心理承受線（如-30%）→ 策略可用

實例：

  • 某策略回測100筆交易，最大回撤-15%
  • 蒙地卡羅模擬5000次，95% CI最大回撤 = -35%
  • 代表：即使策略邏輯正確，未來有5%的機率遇到超過-35%的回撤
  • 如果你的承受線是-25%，這個策略需要調整（降低部位或加過濾條件）
  • 反之，如果回測最大回撤-15%，但95% CI是-20%，表示回撤風險可控

適用時機：任何有回測數據的交易策略，在投入實盤前都應該做蒙地卡羅模擬。特別是當你想知道「策略的回撤可能有多嚴重」時。

連檢定（Runs Test）——驗證蒙地卡羅的前提

是什麼：蒙地卡羅模擬假設每次交易盈虧是獨立的，但實際上很多策略的盈虧有相關性（連續虧損或連續獲利）。連檢定用來驗證這個獨立性假設是否成立。

怎麼做：

1. 計算連（Run）的數量：

  • 將每筆交易結果分為「獲利」和「虧損」

2. 計算Z分數：

  • Z = (實際連數 - 期望連數) / 連數的標準差

3. 實務判斷：

  • Z在-1.96到+1.96之間：蒙地卡羅模擬結果可參考

實例：

  • 某策略100筆交易，55筆獲利、45筆虧損
  • 觀察到的連數 = 42個
  • 期望連數 = 2×55×45/(55+45) + 1 = 50個
  • 連數標準差 ≈ 4.9
  • Z = (42 - 50) / 4.9 = -1.63
  • |Z| < 1.96，獨立性假設成立，蒙地卡羅模擬結果可靠

適用時機：做蒙地卡羅模擬之前，先做連檢定確認獨立性。如果Z分數異常，模擬結果的參考價值會打折。

心得

凱利公式這東西，知道的人很多，用得好的人很少。問題不在公式本身，而在輸入參數。你以為自己勝率60%，結果實際只有50%，凱利就叫你下太多，長期反而虧。這就是為什麼Half Kelly是實務上的黃金標準——承認自己估不準，留點安全邊際。

蒙地卡羅模擬更實用。回測最大回撤-15%？那是歷史路徑剛好走了一條幸運的路。蒙地卡羌跑5000次告訴你95% CI是-35%，這才是你真正要心理準備的數字。超過承受線就縮部位，別鐵齒。

連檢定是蒙地卡羅的前置作業，很多人跳過這步直接跑模擬，結果策略有連續虧損傾向（Z<-1.96），模擬出來的回撤反而低估了實際風險。這種「看起來安全其實不安全」的陷阱，就是忽略獨立性假設的代價。

總結一句：凱利公式告訴你「理論上要下多少」，Half Kelly告訴你「實際上該下多少」，蒙地卡羅告訴你「最壞可能有多慘」，連檢定告訴你「你的模擬結果能不能信」。四個工具串在一起，才是完整的風險管理框架。
