核心知識

股利折現模型 (DDM) 是什麼

是什麼：用未來領到的股利，折現回現在，計算股票的內在價值。

核心概念：股票價格是市場供需的表現，不一定反映真正價值。真正的價值來自公司持續經營能力，而這可以從逐年發放的股利看出——因為股利發放與公司經營業績相關，是公司的正現金流。

適用對象：長期穩定發放股利的成熟公司（如中華電、台塑、可口可樂等）。

不適用對象：不發股利的成長股（Google、Amazon、波克夏）、股利不穩定的公司。

三種公式變體

1. 單期股利折現模型

公式：V0 = (D1 ÷ 1+r) + (P1 ÷ 1+r)

適用時機：只想持有股票一年，計算短期內在價值。

變數說明：

  • V0：股票當前價值
  • D1：一年後預估股利
  • P1：一年後預估股價
  • r：折現率（預期報酬率）

實例：

  • 假設：想持有台積電一年，預期股利 12 元，一年後股價 600 元，預期報酬率 8%
  • 計算：V0 = (12 ÷ 1.08) + (600 ÷ 1.08) = 11.11 + 555.56 = 566.67 元
  • 判斷：若現在股價低於 566.67 元，就值得買進

2. 多期股利折現模型

公式：V0 = D1/(1+r)¹ + D2/(1+r)² + ... + Dn/(1+r)ⁿ + Pn/(1+r)ⁿ

適用時機：預計持有股票多年，每年股利可能不同。

變數說明：

  • D1, D2, ..., Dn：各年度預估股利
  • Pn：持有期結束時的預估股價
  • r：折現率

實例：

  • 假設：持有某股票三年，預期股利分別為 5、6、7 元，三年後賣出價 200 元，預期報酬率 10%
  • 計算：V0 = 5/1.1 + 6/1.21 + 7/1.331 + 200/1.331 = 4.55 + 4.96 + 5.26 + 150.26 = 165.03 元
  • 判斷：股價低於 165 元可考慮買進

3. 高登成長模型（最常用）

公式：V0 = D1 ÷ (r - g)

適用時機：股利持續成長、穩定的成熟公司。

變數說明：

  • V0：股票內在價值（合理價）
  • D1：明年預估股利（= 今年股利 × (1 + 成長率)）
  • r：折現率（預期報酬率），通常用 CAPM 計算或取 8-12%
  • g：預估股利成長率

實例：

  • 假設：某公司股價 110 元，今年股利 2.86 元，預期成長率 5%，預期報酬率 8%

DDM vs DCF 差異

項目股利折現模型 (DDM)現金流量折現 (DCF)
現金流來源股利自由現金流
適用範圍只能用於穩定配息公司可評估更多公司類型
計算難度較簡單較複雜
估價結果較保守較寬鬆
缺點錯過不配息的成長股現金流預測不一定準

選擇建議：

  • 穩定配息的成熟股 → 用 DDM
  • 不配息或配息不穩定 → 用 DCF

四大缺點

1. 只能評估穩定配息公司

問題：無法評估不配息的公司（Google、Amazon、波克夏），或配息不穩定的公司。

原因：這些公司把盈餘投入資本支出或股票回購，而非發放現金股利。

解決：改用 DCF，用自由現金流代替股利。

2. 需要許多假設條件

問題：公式中的「成長率」、「折現率」都是預測值，稍有誤差就會讓估價結果偏差很大。

實例：

  • 預期成長率 10% vs 實際 5% → 內在價值可能差一倍
  • 預期報酬率 8% vs 10% → 結果完全不同

應對：採用保守估計，不要假設無止盡成長。

3. 過去股利不一定反映未來

問題：

  • 一次性收益：賣土地、賣事業體造成股利暴漲，但未來不會持續
  • 政策性配息：台積電 2013 年前固定配 3 元，與獲利脫鉤

應對：觀察配息率是否穩定，排除一次性收益。

4. 小心價值陷阱

問題：股價下跌可能是因為獲利能力衰退，而非被低估。

實例：

  • 某公司股價從 100 跌到 50，DDM 算出內在價值 60 元
  • 看似便宜，但其實公司競爭力已衰退，未來股利會大減
  • 真正的內在價值可能是 30 元

應對：不只看股利，還要看獲利能力、產業地位是否健全。

五大實戰注意事項

1. 小心價值陷阱

怎麼做：檢查公司獲利能力是否衰退，不能只看股價下跌就認為便宜。

2. 預留安全邊際

怎麼做：

  • 算出內在價值後，打 8-9 折再買進
  • 例如：內在價值 100 元 → 等 80-90 元再買

原因：估價可能錯誤，預留空間可以降低風險。

3. 檢查配息率是否穩定

怎麼做：配息率（股利/EPS）應該落在一個穩定區間（如 60-80%）。

異常情況：

  • 配息率 > 100% → 可能動用過去盈餘或借錢配息
  • 配息率 < 30% → 公司可能需要資金擴張

4. 預估數字盡可能保守

怎麼做：

  • 過去成長率 10% → 預估未來 5-10%，而非直接用 10%
  • 成長不可能永遠持續 → 假設成長 5 年後放緩

實例：

  • 過去股利成長 10%
  • 保守預估：前 5 年成長 10%，後 5 年成長 5%，之後零成長

5. 耐心等待

現實：DDM 算出的內在價值通常偏低，多頭時期常覺得「沒東西可買」。

應對：這是正常的。價值投資需要耐心等待價格回落到合理區間。

心得

這模型最大的問題是「假設太多」，成長率和折現率都是猜的，輸入垃圾就輸出垃圾。不過它的核心概念很值得學——股票的價值來自未來現金流的折現，這是所有估值模型的基礎。

實務上，DDM 適合用來評估「中華電信」這類股息穩定的存股標的，對成長股完全沒用。我會把它當成一個參考工具，但不會只依賴它做決策。

關鍵是要保守估計，預留安全邊際。與其精算出「合理價」，不如確保自己「不買貴」。
