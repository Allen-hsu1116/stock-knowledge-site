核心知識

期貨未平倉量（Open Interest, OI）基本概念

是什麼：期貨市場尚未結清的合約總數量，每日收盤後公布。一口未平倉合約包含一個買方和一個賣方。

怎麼算：

  • 新建倉：OI +1（買賣雙方都是新進場）
  • 平倉出場：OI -1（買賣雙方都平倉）
  • 換手：OI 不變（原持倉者賣給新進場者）

未平倉量 vs 成交量差異

指標定義特性
未平倉量（OI）尚未平倉的期貨合約總數累計至到期日，反映資金存量
成交量（Volume）當日成交的合約數量每日歸零重算，反映交易熱度

實例說明：

  • 第一天：A 買入 7 口，B 賣出 7 口 → 成交量 7，OI 7
  • 第二天：A 賣出 3 口給 C → 成交量 3，OI 仍是 7（合約易手，OI 不變）

OI 與價格關係解讀（實戰核心）

怎麼判斷多空趨勢：

OI 變化價格變化市場解讀操作意義
OI 增加價格上漲多頭趨勢強新資金進場，趨勢可能延續
OI 增加價格下跌空頭趨勢強空方加碼，可能續跌
OI 減少價格上漲趨勢可能轉弱多方獲利了結，需謹慎
OI 減少價格下跌趨勢可能轉弱空方回補，可能反彈
OI 不變價格震盪市場缺乏方向等待明確訊號

實例：

  • 情境：台指期從 18000 上漲到 18200，OI 從 15 萬口增加到 18 萬口
  • 解讀：新資金進場做多，多頭趨勢強
  • 操作：可考慮順勢做多，停損設 18000 以下

三大法人未平倉判讀

三大法人是誰：

  • 外資（外國機構投資人）：資金量最大，影響力最強
  • 投信（國內投信基金）：相對穩定，偏好中長期
  • 自營商（券商自營部門）：操作靈活，短線為主

多空淨額計算：

外資未平倉口數淨額 = 外資多單未平倉口數 - 外資空單未平倉口數
  • 正值（如 +5000 口）：外資偏多
  • 負值（如 -8000 口）：外資偏空

怎麼看法人動向：

  • 查詢管道：期交所網站 → 三大法人 → 區分各期貨契約
  • 更新時間：每日 15:00 後公布
  • 參考網站：玩股網、聚財網、財經 M 平方

散戶反指標

是什麼：散戶未平倉 = 全體未平倉 - 三大法人未平倉

怎麼用：

  • 散戶大幅偏多（如多空比 > 60%）→ 反向偏空，可能是市場反轉信號
  • 散戶大幅偏空（如多空比 < 40%）→ 反向偏多，可能是底部信號

實例：

  • 情境：小台指散戶多空比 70%，散戶普遍看多
  • 解讀：散戶作為反指標，可能暗示市場反轉下跌
  • 操作：可考慮與散戶反向，或至少減少多單部位

注意事項：
1. 散戶反指標不一定每次都準確
2. 搭配法人動向一起觀察更可靠
3. 極端值（> 70% 或 < 30%）參考價值較高

法人期貨部位三大誤區

誤區一：法人數字 100% 正確

  • 外資有全球佈局，期貨部位可能是避險而非投機
  • 不能只看單一數字，要搭配整體資金流向

誤區二：正逆價差直接判斷多空

  • 台指期多數時間呈逆價差（因除息扣點）
  • 不代表市場偏空，要觀察價差變化趨勢

誤區三：只看單日變化

  • 單日加碼空單可能是調整部位，不代表看空
  • 要看連續數日的變化趨勢

實戰操作策略

策略一：追蹤外資連續動向

  • 連續 3 天以上加碼多單 → 可能偏多
  • 連續 3 天以上加碼空單 → 可能偏空
  • 配合大盤走勢確認

策略二：法人與散戶反向操作

  • 外資偏多 + 散戶偏空 → 多頭力道更強
  • 外資偏空 + 散戶偏多 → 空頭壓力更大

策略三：結算日觀察

  • 結算前一日：觀察法人是否移倉
  • 結算日：正逆價差會趨近於零
  • 大額部位可能影響結算價

策略四：OI 突破訊號

  • 價格創新高 + OI 增加 → 多頭續強
  • 價格創新高 + OI 減少 → 多頭力道減弱，注意反轉
