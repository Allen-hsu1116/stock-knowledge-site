楔型型態實戰：上升楔型與下降楔型

核心知識

一、楔型型態基本定義

是什麼：楔型是由兩條同方向收斂的趨勢線構成的整理型態，價格在越來越窄的區間內震盪，最終突破趨勢線選擇方向。與三角形不同，楔型兩條線都傾斜（三角形一條平、一條斜）。

繪製三原則：
1. 連結波峰與波谷形成兩條趨勢線
2. 兩條線同方向且收斂，不可平行（平行叫通道）
3. 兩條線在不遠處有明顯交叉點

關鍵特徵：楔型形成期間，成交量逐步萎縮；突破時必須帶量，否則假突破機率大增。

二、上升楔型（Rising Wedge）— 看空訊號

是什麼：兩條上升趨勢線收斂，高點和低點都不斷墊高，但高點上升斜率比低點慢，導致區間越來越窄。不管出現在多頭還是空頭趨勢中，最終方向都是看空。

兩種情境：

情境 A：多頭趨勢中的反轉型上升楔型

  • 出現在一段上漲後的頂部
  • 高點雖然還在創高，但幅度越來越小，多頭動能衰竭
  • 跌破下方趨勢線後，大概率反轉向下

情境 B：空頭趨勢中的延續型上升楔型

  • 出現在下跌趨勢的反彈波中
  • 反彈只是空頭獲利了結或多頭的逃命波
  • 跌破下方趨勢線後，空頭繼續主導

怎麼做（做空）：
1. 確認楔型成型：至少有兩個高點和兩個低點，兩線收斂
2. 等待跌破：價格跌破下方支撐趨勢線
3. 等回踩確認：跌破後價格回測原支撐線（支撐轉壓力），再進場做空
4. 停損設在前波高點上方：如果價格創新高，代表型態失敗
5. 目標價：楔型起始處的最低點到突破點的垂直距離，往下等幅投射

實例：

  • 情境：台積電從 650 上漲到 720 後開始整理，高點 725→728→730，低點 700→710→718，兩線收斂形成上升楔型
  • 操作：跌破 718 支撐線後回踩 718 做空，停損設 731（前高上方），目標 718 - (730-700) = 688

三、下降楔型（Falling Wedge）— 看多訊號

是什麼：兩條下降趨勢線收斂，高點和低點都不斷下移，但低點下降斜率比高點慢，區間越來越窄。不管出現在多頭還是空頭趨勢中，最終方向都是看多。

兩種情境：

情境 A：空頭趨勢中的反轉型下降楔型

  • 出現在一段下跌後的底部
  • 低點雖然還在破底，但幅度越來越小，空頭動能衰竭
  • 突破上方趨勢線後，大概率反轉向上

情境 B：多頭趨勢中的延續型下降楔型

  • 出現在上漲趨勢的回檔波中
  • 回檔只是多頭獲利了結或空頭的逃命波
  • 突破上方趨勢線後，多頭繼續主導

怎麼做（做多）：
1. 確認楔型成型：至少有兩個高點和兩個低點，兩線收斂
2. 等待突破：價格突破上方壓力趨勢線
3. 等回踩確認：突破後價格回測原壓力線（壓力轉支撐），再進場做多
4. 停損設在前波低點下方：如果價格創新低，代表型態失敗
5. 目標價：楔型起始處的最高點到突破點的垂直距離，往上等幅投射

實例：

  • 情境：聯發科從 1350 回跌，高點 1300→1280→1265，低點 1200→1190→1185，兩線收斂形成下降楔型
  • 操作：突破 1265 壓力線後回踩 1265 做多，停損設 1184（前低下方），目標 1265 + (1300-1185) = 1380

四、楔型與三角型態的關鍵差異

特徵楔型三角型態
兩條趨勢線同方向傾斜一條平、一條斜
上升楔型看空（反直覺！）上升三角形看多
下降楔型看多（反直覺！）下降三角形看空
突破方向與楔型傾斜方向相反上升三角形往上、下降三角形往下

新手最常犯的錯：看到「上升」楔型就直覺做多，看到「下降」楔型就做空。完全相反！ 上升楔型看空，下降楔型看多，這是反直覺的關鍵。

五、楔型搭配道氏理論的順勢交易法

是什麼：利用道氏理論判斷主趨勢方向，再在楔型整理結束時順勢進場。

怎麼做：
1. 先確認主趨勢符合道氏理論（多頭：高點高、低點高；空頭：高點低、低點低）
2. 在主趨勢的回檔/反彈中識別出楔型
3. 兩種進場方式：

  • (A) 突破進場：楔型突破趨勢線時直接進場

4. 停損設在楔型另一端之外

實例：

  • 情境：大盤在多頭趨勢中出現下降楔型（回檔整理）
  • 操作(A)：突破楔型上緣 18500 時進場做多，停損 18200
  • 操作(B)：在楔型形成前的高點 18800 附近掛單做多

六、楔型交易的量價確認與假突破防範

量價確認三原則：
1. 楔型形成期：成交量必須逐步萎縮，如果量能不縮，型態可能不成立
2. 突破瞬間：必須放量突破，量能不夠的突破是假突破的高危險群
3. 突破後：突破後回踩時量能縮小，代表賣壓減弱，確認支撐/壓力轉換成功

假突破防範：

  • 等收盤價突破，不要看到盤中穿越就進場
  • 突破後等回踩確認（支撐壓力互換），再進場
  • 如果突破後 3 根 K 線內跌回楔型內，視為假突破

楔型失敗的演化：如果價格往預期反方向突破（上升楔型往上破），可能演變成單向通道，開始一段新趨勢。此時不要硬做原方向，要順著新突破方向操作。

心得

楔型最坑人的地方就是名字跟方向反直覺——上升楔型看空、下降楔型看多。這種反直覺設計就是市場用來收割新手的陷阱，搞不清楚的人看到「上升」就追多，結果剛好接到最後一棒。

另一個重點是楔型的時間週期——週期越大的楔型，突破後的行情越猛。日線級別的楔型整理可能持續 2-4 週，突破後的行情往往能跑 5-10%；週線級別的楔型整理可能 1-3 個月，突破後行情更大。所以不要只看小週期的楔型，大週期的楔型才是真正的肥肉。

最後，楔型跟之前學過的旗型、三角旗型是同一族群的整理型態，差別在於旗型是平行通道、三角旗型一平一斜、楔型是同斜收斂。把這三個放一起記，就能快速分類任何整理型態了。
