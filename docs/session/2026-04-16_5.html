一、剝頭皮交易（Scalping）核心概念

1. 剝頭皮交易的定義與核心邏輯

是什麼：在極短時間內（數秒到數分到數分鐘）頻繁進出市場，從微小價格波動中累積利潤的超短線交易策略。與當。與當沖不同，剝頭皮更極端——持倉更短、頻率更高、單筆目標更小。

核心公式：剝頭皮 = 高頻率 × 小獲利 × 嚴格停損

怎麼做：
1. 時間框架：使用1分鐘（M1）或5分鐘（M5）K線圖，少數進階者搭配15分鐘圖確認整體趨勢
2. 單筆目標：外匯市場2-10 pip利潤；台股則是1-3個Tick（0.5-1.5元）
3. 交易頻率：每日數十到數百次（外匯），台股因交易成本較高，每日約5-20次
4. 持倉時間：數秒到數分鐘，絕不過夜

實例：

  • 外匯：EUR/USD在1.1050-1.1055區間震盪，每次在1.1050買進、1.1053賣出，每筆賺3 pip，一天做50次 = 150 pip
  • 台股：台積電(2330)在600-601元區間來回，每次在600元買進、600.5元賣出，每筆賺0.5元（1 Tick），扣掉手續費約打平，需要2-3 Tick才有淨利

適用時機：高流動性、低點差（外匯）或低交易成本（台股當沖證交稅減半）的標的；不適合低流動性或高交易成本的市場

2. 剝頭皮 vs 當沖 vs 波段：三種交易風格比較

是什麼：三種交易風格在持倉時間、頻率、目標獲利上有根本差異，選擇哪種取決於你的性格、資金和時間。

怎麼做（比較表）：

項目剝頭皮 Scalping當沖 Day Trading波段 Swing
持倉時間數秒～數分鐘數分鐘～數小時數天～數週
每日交易次數數十～數百次數次～數十次1-3次
目標獲利/筆1-3 Tick1-5%5-30%
所需時間投入非常高（全程盯盤）高中
隔夜風險無通常無有
交易成本影響極大中低
心理壓力極高高中
適合性格極度冷靜、反應極快有紀律、可盯盤有耐心、可等

實例：

  • 剝頭皮者：每天在台積電5分K線上來回操作15次，每次賺1-2個Tick，日獲利0.3-0.5%
  • 當沖者：每天做2-3筆交易，捕捉日內趨勢段，每筆賺1-3%，日獲利2-5%
  • 波段者：一週做1筆交易，持倉5-10天，賺5-15%

適用時機：

  • 剝頭皮 → 有全職盯盤時間、反應極快、心理素質極強
  • 當沖 → 有盯盤時間、有基本技術分析能力
  • 波段 → 上班族、無法盯盤、有耐心

3. 三線移動平均線剝頭皮策略（OANDA實戰法）

是什麼：利用高價移動平均線（2日）、低價移動平均線（2日）和簡單移動平均線（40日）三條線構成的通道，在30分鐘線上進行剝頭皮交易。

怎麼做：

指標設定：

  • SMA(40)：以收盤價計算，判斷中期趨勢方向
  • MA_High(2)：以高價計算，作為通道上軌和獲利出場點
  • MA_Low(2)：以低價計算，作為通道下軌和進場點

做多步驟：
1. 步驟一：確認趨勢：MA_High和MA_Low形成的通道位於SMA(40)上方，且SMA(40)角度向上 → 上升趨勢確認
2. 步驟二：進場：K線觸碰到MA_Low（通道下軌）時立即買進，目視即可不用等收盤
3. 步驟三A：獲利出場：K線觸碰到MA_High（通道上軌）時獲利平倉，通常1-2根K線內完成
4. 步驟三B：停損：K線觸碰到SMA(40)時停損（趨勢可能反轉），或1-2根K線內無法獲利則不論損益平倉

做空步驟：反之，通道在SMA(40)下方 + SMA(40)角度向下 → K線觸碰MA_High時做空

實例：

  • 情境：台指期30分K線，SMA(40)角度向上，通道在SMA上方
  • 操作：K線回踩觸碰MA_Low時買進，1-2根K線後觸碰MA_High平倉，獲利約15-20點
  • 一個趨勢中可重複操作3-5次，累積60-100點

適用時機：趨勢明確時（SMA角度明顯向上或向下），SMA持平時不操作；30分鐘線為主，不建議用更短週期（雜訊太多）

4. 剝頭皮四大策略

是什麼：四種不同情境下適用的剝頭皮操作方法。

怎麼做：

策略一：趨勢剝頭皮（最安全）

  • 在強勢趨勢中只順勢操作
  • 上升趨勢中等待短暫回調至EMA支撐位後做多
  • 停損設在EMA下方3-5 pip，停利看前高附近
  • 適合：初學者

策略二：支撐阻力突破剝頭皮

  • 價格突破關鍵支撐/阻力後順向進場
  • 停損設在突破位下方3-5 pip
  • 需要Price Action經驗判斷真假突破
  • 適合：有經驗者

策略三：新聞剝頭皮（極高危險）

  • 利用重大經濟數據發布後的瞬間波動
  • 需要極低延遲的執行環境
  • 滑點和點差會急劇擴大
  • 適合：專業交易者，新手絕對不要碰

策略四：區間剝頭皮（Range Scalping）

  • 橫盤整理區間內，支撐位買進、阻力位賣出
  • 反覆操作直到區間被突破
  • 適合：波動較小、流動性充足的時段

實例：

  • 趨勢剝頭皮：台積電5分K線沿EMA20上升，每次回踩EMA20就買進，反彈1-2個Tick就賣出
  • 區間剝頭皮：某股在100-102元區間整理，100買102賣，一天做5次，每次賺2元

適用時機：

  • 趨勢剝頭皮 → 趨勢明確時
  • 突破剝頭皮 → 關鍵價位突破時
  • 新聞剝頭皮 → 重大數據公布時（極度危險）
  • 區間剝頭皮 → 盤整整理時

5. 剝頭皮風險管理五大鐵律

是什麼：剝頭皮因為高頻率和高槓桿，風控比一般交易更嚴格。

怎麼做：

1. 嚴格停損：每筆交易必須預設停損，建議3-5 pip（外匯）或1-2個Tick（台股），單筆最大虧損不超過帳戶資金的0.5-1%
2. 每日最大虧損限制：設定每日最大虧損（如帳戶3%），達到即停止當日交易，避免「情緒性連性連敗」毀掉帳戶
3. 風報比管理：即使剝頭皮風報比通常較低（1:1至1:2），也必須確保長期期望值為正，避免為追求勝率而放寬停損
4. 避免過度槓桿：剝頭皮常用較大倉位+高槓桿，連敗時爆倉風險極高，初學者從最小倉位開始
5. 避開新聞時間：重大數據公布前後5分鐘，滑價和點差急劇急劇擴大，剝頭皮者的噩夢

實例：

  • 正確：10萬資金，每筆交易風險1%（1000元），每天最多虧3%（3000元），連虧3天就停手一週
  • 錯誤：10萬資金，每筆交易風險5%（5000元），連虧3筆就虧15000元（15%），心態崩潰

適用時機：每次剝頭皮交易都要遵守，沒有例外

6. 剝頭皮交易在台股的適用性分析

是什麼：台股的交易成本結構和外匯不同，剝頭皮在台股的可行性需要特別評估。

怎麼做（成本分析）：

台股當沖交易成本：

  • 買進手續費：0.1425%（可議價到0.05-0.08%）
  • 賣出手續費：0.1425%（可議價到0.05-0.08%）
  • 當沖證交稅：0.15%（賣出時）
  • 合計單次交易成本：約0.45%（未議價）或0.25%（已議價）

成本對剝頭皮的影響：

  • 100元股票：1 Tick = 0.5元 = 0.5%，交易成本0.25-0.45% → 扣完成本淨賺0.05-0.25%
  • 500元股票：1 Tick = 1元 = 0.2%，交易成本0.25-0.45% → 扣完成本淨虧0.05-0.25%
  • 600元股票（如台積電）：1 Tick = 0.5元 = 0.083%，交易成本0.25-0.45% → 虧定了

結論：

  • 台股剝頭皮只適合中低價股（50-150元），因為Tick跳動佔比相對大
  • 高價股不適合剝頭皮，成本吃光利潤
  • 手續費議價是關鍵——一定要談到0.05-0.08%
  • 需要至少2-3 Tick的波動才有淨利

實例：

  • 可行：80元股票，1 Tick = 0.5元 = 0.625%，交易成本0.25% → 淨賺0.375%/筆，一天做10次 = 3.75%
  • 不可行：600元台積電，1 Tick = 0.5元 = 0.083%，交易成本0.25% → 淨虧0.167%/筆

適用時機：只在低價高流動性標的上嘗試，且手續費已議價；高價股改用當沖一波流或波段

二、股市季節性效應實戰運用

7. 全球股市季節性效應統計

是什麼：根據1950年以來的美股、1988年以來的歐股、1971年以來的亞太股市統計，全球股市確實存在統計上顯著的季節性模式。

怎麼做（統計結果）：

全球共同規律：
1. 4月是表現最好的月份：美股平均+1.67%、歐股平均+2.43%、亞太平均+2.0%，統計上顯著
2. 9月是唯一全球共跌的月份：美股平均-0.8%、歐股平均-1.57%、亞太平均-1.0%，統計上顯著
3. 11月到次年1月通常是年度最強時期：包含「聖誕老人紅包行情」（最後5個交易日+前2個交易日）
4. 6-8月是歐洲最差月份：6月-0.67%、8月-1.23%

台股特有規律：
1. 元旦開紅盤：單日上漲機率高達70%
2. 元月效應不存在：整個1月台股上漲機率只有30%，平均報酬率為負
3. 端午變盤：第二季度末下跌機率增加（5-6月）
4. 中秋變盤：秋天回落機率增加（9月）
5. 星期一效應：1996年後台股存在顯著的星期一效應，集中在第三和第四個星期一

實例：

  • 4月做多策略：統計上4月全球股市上漲機率最高，台股4月歷史平均也是正報酬
  • 9月避險策略：9月全球共跌，考慮減碼或買進Put保護

適用時機：作為輔助判斷工具，不當作唯一依據；季節性解釋力只有3-6%，不能單獨預測股市

8. Sell in May 效應與台股對應策略

是什麼：「Sell in May and go away」是英文諺語，統計上5-10月是美股收益最差的6個月，11月至次年4月是最好的6個月。

怎麼做：
1. 美股Sell in May：5月開始減碼高風開始減碼高風險部位，轉向防禦性標的（公用事業、必需消費品）；11月開始重新布局成長股
2. 台股對應：5-6月是除權息旺季開始前，加上繳稅季（5月報稅、6月賣股繳稅），資金面壓力大；7-9月除息除權旺季，股價因除息自然回落
3. 台股農曆年效應：農曆年前資金回流（紅包行情）、年後開紅盤上漲機率高
4. 但注意：Sell in May不是每年都準，只是統計上機率較高。2023年5月台股就大漲，因為AI題材發酵

實例：

  • 統計策略：每年5月1日減碼至50%部位，11月1日加碼至100%部位，回測2000-2022年台股，年化報酬比買進持有高2-3%
  • 反例：2023年5月賣出AI股會錯過大行情，所以季節性是輔助不是絕對

適用時機：作為年度資金配置的參考，不要機械式執行；要搭配當年基本面和題材面判斷

9. 總統週期效應與台股選舉行情

是什麼：美國總統任期4年循環中，股市表現有固定模式——後兩年表現遠優於前兩年（724% vs 273.1%的累積報酬），第三年行情最好。

怎麼做：

總統週期四年模式：
1. 第一年（上任年）：總統「上任三把火」，但經濟陣痛和壓力對股市構成拖累，表現通常普通
2. 第二年（期中選舉年）：為地方選舉做準備，開始推動有利政策，但市場仍有不確定性
3. 第三年（選舉年前一年）：行情最好的一年！統計顯示從期中選舉年末到選舉年前最後一個季度，是美股表現最強的時期。除1939年二戰爆發外，第三年幾乎都是正報酬
4. 第四年（大選年）：執政黨為連任做利好，但若「聖誕老人行情」未出現，可能預示空頭

台股對應：
1. 美國總統第三年 → 台股通常也受惠全球做多氛圍
2. 台灣自己的選舉行情 → 總統大選前一年（選舉年）通常有政策作多
3. 兩國選舉週期疊加時效果最強

實例：

  • 2023年（拜登第三年）：美股大漲，台股受AI題材也大漲 → 第三年效應驗證
  • 2024年（大選年）：美國總統大選+台灣總統大選年，政策作多

適用時機：每年初判斷當年是總統週期的第幾年，作為年度投資計畫的宏觀框架

10. 星期效應與日內效應實戰運用

是什麼：股市在一週的特定日子有統計上的差異表現，台股還有特有的星期一效應。

怎麼做：

星期效應：
1. 星期一效應（台股）：1996年後台股第三、第四個星期一跌幅較大，原因是散戶在假日收集資訊後發現券商建議不當，週一集中賣出
2. 星期五效應：美股週五上漲機率較高（週末前平倉空單），台股週五也有類似現象
3. 運用方式：週三、週四進場做多，避開第三第四個週一的大波動

日內效應（台股）：
1. 9:00-9:30 開盤震盪期：波動最大，適合剝頭皮但不適合一般當沖
2. 9:30-10:30 趨勢形成期：方向逐漸明朗，是較穩定的進場時機
3. 10:30-12:00 量縮期：波動縮小，減少操作
4. 12:30-13:30 尾盤期：當沖者出場、拉尾盤或殺尾盤

實例：

  • 避開第三週一：統計上第三週一台股下跌機率較高，可以考慮週五先減碼，週一再接回
  • 尾盤效應：13:00-13:25是當沖出場時間，常出現「拉尾盤」（主力作價）或「殺尾盤」（停損單湧出）

適用時機：作為進出場時機的輔助判斷，不要單獨依賴星期效緒做決策

11. 季節性效應的正確認知：輔助而非主導

是什麼：季節性效應雖然統計上顯著，但解釋力只有3-6%，不能單獨預測股市走勢。

怎麼做（正確態度）：
1. 季節性是機率工具，不是預測工具：4月上漲機率較高不代表每年4月都會漲，只是「歷史上4月漲的次數比較多」
2. 搭配基本面使用：如果4月統計上易漲 + 當年有明確利多（如AI題材）→ 做多信心更強
3. 用來調整部位，不是用來做方向：9月統計上易跌 → 不是做空，而是減碼或加強停損
4. 當季節性與基本面矛盾時，相信基本面：2023年5月Sell in May但AI題材大爆發 → 基本面勝出
5. 回歸模型的擬合度只有3-6%：意味著季節性只解釋了3-6%的股市波動，剩下94-97%由其他因素決定

實例：

  • 正確用法：4月統計易漲 + 當年第一季營收成長 + 法人買超 → 三重確認做多
  • 錯誤用法：因為9月統計易跌就做空，結果9月有大題材反而大漲 → 季節性被推翻

適用時機：作為多維度判斷中的一個維度，佔權重5-10%即可，不要讓季節性主導你的投資決策

心得

今天學了兩個看似不相關但其實很搭的主題：剝頭皮交易和季節性效應。一個是「微觀極短線」，一個是「宏觀時間規律」，但它們有共同的核心——都是在利用市場的統計性重複模式獲利。

1. 剝頭皮在台股的可行性比外匯低很多。台股的交易成本結構（手續費+證交稅）讓Tick級別的獲利幾乎被成本吃光，只有中低價股（50-150元）的Tick跳動佔比夠大，才有剝頭皮的空間。高價股如台積電，1 Tick只有0.083%，扣掉0.25%成本直接虧損。所以如果要做剝頭皮，標的選擇是第一關鍵。

2. 三線移動平均線剝頭皮策略很聰明。用高價MA和低價MA構成通道，通道在SMA上方就做多、在下方就做空，K線碰到通道邊界就進出場。最妙的是「1-2根K線不獲利就平倉」——這不是傳統的停損邏輯，而是「時間停損」，如果進場後價格沒有立即往預期方向走，就代表判斷可能錯了，果斷出場。這跟之前學的「固定時間出場策略」概念一致。

3. 季節性效應最反直覺的結論是「4月最好、9月最差」。4月全球股市平均上漲1.67-2.43%，9月平均下跌0.8-1.57%，這個規律在美股、歐股、亞太股市都成立。但最關鍵的認知是——季節性只解釋3-6%的波動，剩下94-97%是其他因素。所以季節性是「錦上添花」的工具，不是「雪中送炭」的工具。

4. 台股的星期一效應很有趣。散戶在假日收集資訊、發現券商建議不當，週一集中賣出——這種行為金融學解釋比純統計更有說服力。這告訴我，統計效應背後一定要有行為邏輯支撐才值得重視，否則可能只是數據挖掘（Data Mining）。

5. 季節性的正確用法是「調整部位」，不是「改變方向」。9月統計易跌不代表要做空，而是「多單停損設緊一點、部位小一點」。這跟所有風控原則一致——不確定性高的時候減少暴露，不確定性低的時候增加暴露。季節性只是告訴你「什麼時候不確定性比較高」。

6. 總統週期第三年是最值得記住的宏觀規律。統計1833-2011年，第三年行情最好，幾乎只有1939年例外。這個規律如果跟台灣選舉行情疊加，效果更強。2027年如果是美國總統第三年+台灣無選舉，那就是全球做多年——但這是預測，要等當時基本面確認才有意義。
