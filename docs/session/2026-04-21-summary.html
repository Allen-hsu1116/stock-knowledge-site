1. 杜邦分析進階：五因子模型

核心概念:

  • 三因子只能告訴你「淨利率下降」，五因子能精確歸因：是營業利潤率下降？利息負擔加重？稅務優惠到期？
  • 五因子模型：ROE = 稅務負擔 × 利息負擔 × 營業利潤率 × 總資產週轉率 × 權益乘數
  • 同樣ROE 20%，高淨利率驅動（科技巨人）vs 高週轉+槓桿驅動（零售龍頭），風險天差地遠
  • 巴菲特偏好「高淨利率＋低槓桿」模式，品質更高、風險更低

五大侷限: 依賴會計數據、忽略無形資產、必須同業比較、時間序列更重要、無法反映現金流

2. DCF現金流量折現法進階

核心概念:

  • 三要素：FCF（真正可自由運用的資金）、折現率（通常用WACC）、期間（5-15年）
  • FCF > EPS，因為EPS沒考慮折舊攤銷和資本支出
  • 蒙格名言：「我從沒看過巴菲特動手算DCF」——真正好的投資機會，不用精算就知道便宜
  • DCF適用：現金流穩定的公司；不適用：週期性、初創、科技/生技公司

3. 波特五力分析：投資人視角

核心概念:

  • 五力縱橫軸：橫軸（買方vs賣方議價力）判斷利潤空間；縱軸（新進者vs替代品）判斷利潤持續性
  • 最佳投資組合：買方弱、賣方弱、進入壁壘高、替代品少
  • 台積電五力強（進入壁壘極高、替代品威脅低）；面板廠五力弱（競爭激烈、買方議價力強）
  • 投資前用五力快速檢視：產業競爭、進入壁壘、供應商議價、買方議價、替代品威脅

4. FCF進階實戰：三種計算法與四大警訊

核心概念:

  • 三種計算：現金流量表直算法（最常用）、損益表+資產負債表推算法（深入分析）、三大報表綜合法（無EBIT時用）
  • FCF收益率 = 每股FCF ÷ 股價，>8%嚴重低估、5-8%合理偏低、<3%偏貴
  • FCF利潤率 = FCF ÷ 營收，>10%優秀、5-10%正常、<5%效率差
  • FCF是EPS的「照妖鏡」：EPS成長但FCF下降 = 紅旗

四大警訊: EPS成長但FCF下降、FCF為正但營業現金流為負、FCF長期為負、FCF Margin逐年下降

5. 配息品質分析：配息率 + FCF保障倍數

核心概念:

  • 配息率 40-70%最理想、>100%是糖衣毒藥
  • FCF配息保障倍數 >1.5倍安全、<1.0倍危險、<0是借錢發股息
  • 高殖利率 + 配息率110% = 吃老本，不可持續
  • 5步檢查法：看配息率→看FCF保障→看連續配息年數→看配息成長→交叉驗證EPS vs FCF

6. PBR股價淨值比與河流圖

核心概念:

  • PBR和PE是互補關係，不是替代：PE看「能賺多少」，PBR看「值多少家當」
  • 循環股用PBR不用PE（因為EPS波動劇烈，PBR以穩定的淨值為基準）
  • 四種保守PBR算法：傳統→有形資產→流動資產→淨現金（越來越保守）
  • PBR河流圖三型態：上升趨勢（成長股）、上下震盪（循環股）、穩定平坦（成熟型）
  • PBR低+ROE高 = 被低估；PBR低+ROE低 = 大概率價值陷阱

7. 合併報表 vs 個體報表 + 關係人交易 + 財報附註

核心概念:

  • 兩個關鍵指標：母公司營收佔合併營收比重、歸屬母公司純益佔比
  • 非控制權益佔比越大，合併數字與母公司實際可分配的差距越大
  • 關係人交易八大紅旗：銷貨佔比>50%、應收帳款週轉天數暴增、進銷貨價格偏離市場等
  • 財報附註六大必看：會計政策變更、關係人交易、或有事項、資產減損、營運部門資訊、期後事項

8. 營運/財務/總槓桿度（DOL/DFL/DTL）

核心概念:

  • DOL：銷售量變動1%時EBIT變動多少%（固定成本越高DOL越大）
  • DFL：EBIT變動1%時EPS變動多少%（利息越高DFL越大）
  • DTL = DOL × DFL：銷售量變動1%時EPS變動多少%
  • 四象限策略：高DOL+低DFL = 景氣復甦期黃金組合；高DOL+高DFL = 定時炸彈

9. 股東權益變動表 + 減資/增資

核心概念:

  • 「愈專注本業的企業，權益變動表愈乾淨」——大立光只有三項，樂陞洋洋灑灑
  • 備供出售金融資產未實現損益是最會藏雷的科目
  • 減資三方式：現金減資（利多）、庫藏股減資（利多）、彌補虧損減資（利空）
  • 現金減資是大股東避稅工具（退還股款不課稅 vs 股利要課稅）

10. 盈餘管理 + 存貨計價 + 資本化vs費用化

核心概念:

  • 盈餘管理三手法：會計政策選擇、應計項目操縱、實質經濟活動操縱
  • 存貨計價：FIFO通膨時毛利最高、加權平均法平穩、LIFO台灣禁止
  • 存貨跌價損失是原物料循環股的超前指標（新光鋼案例）
  • 資本化vs費用化：研發支出資本化是最大灰色地帶，資本化比率突然上升 = 美化利潤
  • 簡單驗證：看營業現金流是否跟得上淨利

11. DDM股利折現模型 + WACC + Fed Model

核心概念:

  • 戈登公式：V₀ = D₁/(r-g)，g估計差1%合理價差25%以上→安全邊際極重要
  • 戈登反推：r = D₁/P₀ + g（預期報酬率 = 股息殖利率 + 成長率）
  • WACC = 債權成本×(1-t)×權重 + 股權成本×權重
  • Fed Model幾乎永遠說「股票該買」，低利率時代完全失靈，只能當大方向參考

12. F-Score + EVA經濟附加價值

核心概念:

  • F-Score九大指標（獲利性4項+安全性3項+成長性2項），≥6分基本面良好
  • EVA = NOPAT - 投入資本 × WACC，EVA>0才是真正為股東創造價值
  • ROE > WACC才是真正賺錢，ROE高但WACC更高 = 假繁榮
  • 最佳組合：F-Score ≥ 6 + EVA > 0 = 鑽石股

13. Graham Number + NCAV + IRR/MIRR

核心概念:

  • Graham Number = √(22.5 × EPS × BVPS)，PE×PB不超過22.5的雙重安全邊際
  • NCAV = 流動資產 - 總負債，在NCAV的66%買入，歷史30年平均年報酬20%
  • IRR假設以IRR再投資（過度樂觀），MIRR用實際再投資利率（更貼近現實）
  • IRR判斷效率，NPV判斷價值，兩者搭配最準確

14. 金融資產三分类 + 權益法 + 長期股權投資

核心概念:

  • FVPL（透過損益）：被投資公司股價漲跌直接影響EPS
  • FVOCI（透過其他綜合損益）：股價漲跌只影響淨值不影響EPS
  • 權益法「看得到吃不到」：認列投資收益但可能一毛現金都沒收到
  • 長期投資佔總資產>30%必須看合併報表

15. IFRS 16 租賃會計

核心概念:

  • 核心改變：承租人幾乎所有租賃都要「上表」（使用權資產+租賃負債）
  • 費用前高後低（利息前期高），初期EPS被壓低不代表獲利能力變差
  • EBITDA虛增、CFO虛增、負債比率上升——IFRS 16前的數字已不能直接用
  • 看財報第一步：找使用權資產和租賃負債

16. 現金流量表深度判讀 + 八大型態

核心概念:

  • 現金流量表是三大報表中最難造假的
  • 間接法回推：從淨利調整回真實現金，應收帳款暴增和存貨暴增是最大警訊
  • 營業現金流對淨利比率：>1.2品質佳、0.5-0.7品質存疑、<0.5嚴重警訊
  • 八大型態：型態2（+,-,-）最健康、型態6和8最危險
  • 關鍵是型態變化趨勢，不是單一時間點

17. ROIC + ROIC vs WACC + 殖利率曲線倒掛 + PEG

核心概念:

  • ROIC = NOPAT ÷ 投入資本，不受財務槓桿影響，比ROE更真實
  • 經濟利差 = ROIC - WACC：>0創造價值、<0摧毀價值
  • 殖利率曲線倒掛是經濟衰退最強領先指標，倒掛到衰退約5-22個月
  • PEG = PE ÷ 成長率，<0.66低估、=1合理、>1.2高估

18. 併購套利 + 特殊事件投資法

核心概念:

  • 巴菲特四問：完成機率、資金佔用時間、更好結果可能性、失敗後果
  • 併購套利風險結構：高機率小賺、低機率大虧（負偏態），必須分散
  • 特殊事件投資法核心：下檔風險優先 + 催化劑必不可少
  • 六類特殊事件：分拆（最佳機會）、併購、破產重整、清算、回購、資產出售

19. GARP策略（合理價格買成長）

核心概念:

  • GARP = Growth At a Reasonable Price，彼得林奇13年年化29.2%
  • 核心指標：PEG ≤ 1 + ROE ≥ 15% + 營收年增 > 5%
  • 林奇六分類：快速增長型是最愛，穩定增長型是核心
  • PB-ROE交叉選股：價值前20%+成長前20%，回測年化超額13.9%

20. 盈餘品質 + Sloan應計項目異象

核心概念:

  • Sloan Ratio = (淨利 - CFO - CFI) / 總資產，>10%盈餘品質差
  • 應計項目高的公司未來股價表現差，低的表現好（40年回測年化18% vs 大盤7.4%）
  • 最實用方式：排除Sloan Ratio > 10%的公司，每年多賺2%
  • 高應計項目公司通常也有高ROE（虛的），低應計項目公司ROE低於平均（被低估）

21. P/S股價營收比 + 可轉換公司債

核心概念:

  • P/S = PE × 淨利率，跨產業比P/S毫無意義
  • P/S適用：虧損中公司、高成長公司、景氣循環股（PE失靈的場景）
  • 可轉債距離比例 = 股價 ÷ 轉換價格，0.8~0.95是最佳投資區間
  • 轉換價格是股價天花板，可轉債大量轉換會稀釋EPS

22. 商譽Goodwill + 稀釋EPS + 財報窗飾 + 沖大澡

核心概念:

  • 商譽 = 買貴了的金額，佔總資產>15%高度警戒、>30%紅色警戒
  • 併購後總資產週轉率下滑 = 商譽可能是負擔（如興案例 vs 桂盟案例）
  • 稀釋EPS：用PE估值時必須用稀釋EPS而非基本EPS
  • 財報窗飾：合法範圍內化妝，最關鍵辨識法是營收成長要跟營業現金流同步
  • 沖大澡：爛年把未來費用全塞進去，新CEO上任最愛幹這種事
