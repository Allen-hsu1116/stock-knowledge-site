核心知識

股利折現模型（DDM）是什麼？

是什麼：一種絕對估值方法，用未來領到的股利折現回現在，計算股票的內在價值。

基本概念：股票的價值 = 未來所有股利的現值總和。把未來每年會收到的股利，用適當的折現率換算成今天的價值。

適用對象：長期穩定發放股利的成熟公司（如中華電信、台塑、中鋼等存股標的）。

三種 DDM 公式

1. 單期股利折現模型

適用：只想持有一年的短期投資

公式：

V0 = (D1 ÷ (1+r)) + (P1 ÷ (1+r))

變數說明：

  • V0：股票當前價值
  • D1：一年後的股利
  • P1：一年後的股價
  • r：折現率（必要報酬率）

實例：

  • 假設折現率 8%
  • 預計一年後股利 3 元，股價 105 元
  • 計算：V0 = (3 ÷ 1.08) + (105 ÷ 1.08) = 2.78 + 97.22 = 100 元

2. 多期股利折現模型

適用：預計持有多年，需要預測每一年的股利

公式：

V0 = D1÷(1+r)¹ + D2÷(1+r)² + ... + Dn÷(1+r)ⁿ + Pn÷(1+r)ⁿ

特點：每年股利可能不同，需要逐一預測再折現加總。

缺點：預測期數越多，不確定性越高。

3. 高登成長模型（最常用）

適用：股利穩定成長的公司

公式：

V0 = D1 ÷ (r - g)

變數說明：

  • V0：股票合理價
  • D1：明年預估股利
  • r：折現率（必要報酬率）
  • g：股利年成長率

關鍵限制：r 必須大於 g，否則公式無效。

實例計算：

  • 某公司明年股利 D1 = 3 元
  • 折現率 r = 8%
  • 股利成長率 g = 5%
  • 計算：V0 = 3 ÷ (8% - 5%) = 3 ÷ 0.03 = 100 元

判斷：

  • 若市價 110 元 → 高估 10 元，不買
  • 若市價 80 元 → 低估 20 元，可買

DDM 的三大缺點

缺點 1：只能評估穩定配息公司

問題：無法用在不配息的成長股（如 Google、Amazon、波克夏）

解決：改用現金流量折現法（DCF），用自由現金流代替股利。

缺點 2：需要大量假設

問題：公式中的 r、g 都是預估值，一旦假設錯誤，結果偏差很大。

敏感度分析：

  • g=5% 時，V0=100 元
  • g=4% 時，V0=75 元（差 25%）
  • g=6% 時，V0=150 元（差 50%）

結論：成長率 g 是最敏感的變數，預估要保守。

缺點 3：股利可能被扭曲

問題：

  • 一次性業外收益讓股利暴增（賣土地、賣事業）
  • 公司政策改變（台積電 2013 年前固定配 3 元）
  • 股利與獲利脫鉤

判斷方法：檢查配息率是否穩定，通常應落在合理區間（40%-80%）。

DDM vs DCF 差異

項目DDMDCF
折現標的未來股利未來自由現金流
適用範圍穩定配息公司所有公司
計算難度較簡單較複雜
估價結果較保守較寬鬆

關係：DDM 是 DCF 的特例，用股利當現金流。

實戰使用技巧

1. 避開價值陷阱

價值陷阱：股價大跌、獲利也大跌的公司，DDM 算出來的「便宜價」其實不便宜。

判斷：檢查獲利能力是否衰退，不是只看股價下跌。

2. 預留安全邊際

做法：

  • 算出合理價 100 元
  • 打 8 折進場 → 80 元買進
  • 預留 20% 錯誤空間

3. 檢查配息來源

判斷：配息率是否穩定？配息是否來自本業獲利？

危險信號：

  • 配息率突然暴增（可能發現金資本公積）
  • 配息率超過 100%（借錢配息）

4. 成長率要保守估

實務做法：

  • 不要假設成長率永遠不變
  • 分段估計：前 5 年成長 10%，後 5 年成長 5%，之後零成長
  • 越遠的未來，成長率要越保守

5. 耐心等待

心理建設：

  • DDM 算出來的合理價通常偏低
  • 多頭市場很難找到符合標準的股票
  • 等待是價值投資的必修課

實例演練

案例：中華電信

已知：

  • 今年股利：4.5 元
  • 近 5 年股利成長率：約 2%
  • 預期報酬率（折現率）：7%

計算：

V0 = 4.5 ÷ (7% - 2%) = 4.5 ÷ 0.05 = 90 元

判斷：

  • 若市價 105 元 → 高估，不買
  • 若市價 75 元 → 低估 16.7%，有安全邊際

注意：

  • 電信股成長率低，g 通常接近 0
  • 適合用 DDM 評估
  • 但要確認股利政策是否穩定

案例：台積電

問題：

  • 台積電 2013 年前固定配 3 元，跟獲利脫鉤
  • 用當時股利算 DDM 會嚴重低估
  • 2017 年後改變政策，股利隨獲利調整

啟示：

  • 公司政策改變會讓歷史股利數據失效
  • 用 DDM 前要確認股利政策穩定

適用時機

適合用 DDM 的情況：

  • 穩定配息的成熟企業（電信、公用事業、銀行）
  • 存股型投資人
  • 想要保守估價

不適合用 DDM 的情況：

  • 不配息的成長股（科技業、新創）
  • 股利政策不穩定的公司
  • 處於虧損的公司

心得

DDM 是最保守的估值方法，它的核心假設很簡單：股票的價值就是你未來能拿到的現金（股利）折現回來的總和。

但這個方法的致命缺陷是「假設太多」——預期報酬率要假設、成長率要假設，而這兩個數字差一點點，算出來的合理價就差很多。

實務上我會這樣用 DDM：

1. 只用在股利政策穩定的公司：像是中華電、台塑、中鋼這種存股標的
2. 成長率寧可低估：如果過去成長 10%，我會用 5% 甚至更低來算
3. 一定要留安全邊際：合理價打 8 折才是進場價
4. 搭配其他指標交叉驗證：不要只信 DDM 一個數字

最有價值的一點是：DDM 教你思考「股票到底是什麼」——不是螢幕上的數字，而是未來現金流的折現。這個概念比公式本身更重要。
