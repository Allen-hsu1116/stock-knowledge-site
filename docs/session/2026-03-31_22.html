核心知識

一、殖利率定義與計算

是什麼：殖利率是衡量股票配息報酬率的指標，類似銀行利息的概念。

現金殖利率公式：

現金殖利率（%）= 每股現金股利 ÷ 每股股價 × 100%

怎麼算：
1. 找到公司宣布的現金股利金額（每股）
2. 除以當時股價
3. 乘以 100% 得到百分比

實例：

  • 情境：A 公司配發現金股利 4 元，股價 60 元
  • 計算：4 ÷ 60 × 100% = 6.67%
  • 意思：投資 60 元買一股，每年可領 4 元股利，報酬率 6.67%

股票股利怎麼算：

  • 股票股利單位是「元」，但投資人拿到的是「股」
  • 公式：可獲得股數 = 股票股利 × 持有股數 ÷ 10 元（票面價值）
  • 實例：玉山金配股票股利 0.1 元，持有 1 張（1000 股）

二、還原殖利率（更精確算法）

是什麼：當公司同時配現金股利和股票股利時，需要用還原殖利率才能反映真實價值。

怎麼算：
1. 計算除權息參考價：

   除權息參考價 = (除權除息前1天收盤價 - 現金股利) ÷ (1 + 股票股利÷10)
   ```


2. 計算股票股利價值：



   股票股利價值 = 除權息參考價 × 拿到的配股數
   ```


3. 計算總股利價值：



   總股利價值 = 股票股利價值 + 現金股利
   ```


4. 計算還原殖利率：



   還原殖利率 = 總股利價值 ÷ (股價 × 持有股數) × 100%
   ```


實例：



    

  • 玉山金 3/13 收盤價 29.6 元，配現金 1.2 元、股票 0.1 元
    • 

  • 除權息參考價：(29.6 - 1.2) ÷ (1 + 0.1÷10) = 28.12 元
    • 

  • 股票股利價值：28.12 × 10 股 = 281.2 元
    • 

  • 總股利價值：281.2 + 1.2 × 1000 = 1481.2 元
    • 

  • 還原殖利率：1481.2 ÷ 29650 × 100% = 4.99%
    •

四、高殖利率的四大陷阱



陷阱 1：賺股利、賠價差



是什麼：殖利率高可能是因為股價下跌，不是公司配息好。



怎麼判斷：



    

  • 檢查股價走勢：近一年是否持續下跌？
    • 

  • 比較歷史殖利率：今年突然變高就要小心
    • 

  • 查看填息機率：過去是否常填息？
    • 




實例：



    

  • 情境：B 股票去年股價 100 元，配息 4 元，殖利率 4%；今年股價跌到 50 元，殖利率變成 8%
    • 

  • 判斷：殖利率變高是因為股價腰斬，不是好事
    • 




陷阱 2：公司配息來源不佳



是什麼：公司配息不是來自本業獲利，而是賣資產或一次性收入。



怎麼判斷：



    

  • 查財報：營業利益 vs 稅後淨利
    • 

  • 若稅後淨利 >> 營業利益，可能有一次性收入
    • 

  • 查新聞：公司是否賣土地、處分資產？
    • 




實例：



    

  • 情境：營建股今年大賺，殖利率 10%+
    • 

  • 判斷：可能是建案完工一次認列，明年可能沒有
    • 

  • 操作：避開一次性收入配息的公司
    • 




陷阱 3：公司運用資金效率低



是什麼：高配息可能代表公司沒有成長機會，只能把錢還給股東。



怎麼判斷：



    

  • 看產業：成熟產業（電信、金融）通常高配息
    • 

  • 看成長性：公司是否在擴張？研發投入多少？
    • 

  • 看同業：同產業公司配息率多少？
    • 




適用時機：



    

  • 成長股：低配息正常，公司要留錢擴張
    • 

  • 成熟股：高配息合理，但要配息穩定
    • 




陷阱 4：未填權息等於沒賺



是什麼：除權息後股價會下跌，必須「填權息」才算真正獲利。



怎麼判斷：



    

  • 查歷史填息天數：過去平均幾天填息？
    • 

  • 查填息機率：過去幾年成功填息的比例？
    • 

  • 填息機率 > 80%、天數 < 30 天 = 較好
    • 




實例：



    

  • 情境：C 公司配息 5 元，除息前股價 50 元，除息後變 45 元
    • 

  • 判斷：如果股價沒回到 50 元，雖然領了 5 元股利，但股票也跌了 5 元
    • 

  • 操作：選填息機率高、填息天數短的股票
    • 




五、選股實務要點



高殖利率股選股 4 檢查：

1. 配息來源：是否來自本業獲利？

2. 填息能力：歷史填息機率多少？平均幾天？

3. 配息穩定度：連續幾年配息？金額穩定嗎？

4. 股價走勢：殖利率高是否因為股價下跌？



搭配指標：



    

  • EPS（每股盈餘）：獲利能力
    • 

  • 配息率：配息 / EPS，看公司願意發多少
    • 

  • 股價淨值比：是否買太貴
    • 

  • 歷史配息紀錄：至少看 5 年
    •

  • 聯詠（3034）：5 年平均殖利率 9.23%，填息機率 100%，平均 66 天填息
    • 

  • 矽創（8016）：5 年平均殖利率 6.97%，填息機率 100%，平均 17 天填息

心得



殖利率不是越高越好，這點真的要記住。高殖利率背後可能是股價下跌、一次性收入配息、或公司沒有成長機會。



關鍵學習：

1. 還原殖利率比單純「現金股利÷股價」更準確，尤其是配股票股利時

2. 填息能力比殖利率高低更重要——沒填息等於沒賺

3. 要看配息來源，本業獲利配息才穩定

4. 高殖利率可能代表公司缺乏成長機會，要綜合判斷



操作原則：



    

  • 不要只看殖利率排行買股
    • 

  • 檢查：配息來源 + 填息機率 + 配息穩定度 + 股價趨勢
    • 

  • 成長股低配息正常，成熟股高配息合理
