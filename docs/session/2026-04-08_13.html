一、分點進出基礎概念

分點進出是什麼？

定義：分點進出是指每間券商的各個分點當日所有股票的買賣紀錄。

關鍵點：

  • 台灣證交所每天傍晚公布每一間券商分點的買賣紀錄
  • 資料詳細到「哪一天、哪檔股票、哪個價位、買進或賣出幾張」
  • 可以從中追蹤主力大戶的動向

為什麼重要：

  • 找出關鍵主力的進出軌跡
  • 判斷大戶是在佈局還是出貨
  • 跟著聰明錢走，提高勝率

二、如何從分點進出找關鍵主力

關鍵券商識別

是什麼：一段期間內買超最多的券商分點，對股價有影響力。

怎麼判斷：
1. 看當日買超排行，第一名買超量是否特別突出
2. 觀察該分點歷史績效（勝率、選股能力）
3. 追蹤該分點進出與股價的關聯性

實例：

  • 網龍（3083）2019/8/21 「兆豐-景美」突然買超 557 張
  • 之後股價明顯上漲
  • 高點時「兆豐-景美」開始出脫持股
  • 9 月初再次進場，又有一波漲幅

地緣券商

是什麼：公司附近的券商分點，公司派或內部人士可能在此進出。

怎麼用：
1. 查詢公司所在地（如：彰化縣福興鄉）
2. 找出公司附近的券商分點
3. 觀察這些分點是否有異常買超
4. 可能是公司派或內部人士的籌碼動向

注意：地緣券商大量買超可能是利多提前佈局。

三、分點指標實戰

三大分點指標

1. 分點進出（買賣超）

公式：買賣超 = 買進張數 - 賣出張數

怎麼看：

  • 連續三天買超 → 籌碼收集中
  • 連續三天賣超 → 籌碼出逃中
  • 單日爆量買超 → 可能是主力進場

2. 分點買賣力

公式：買賣力 = 買賣超 ÷ 買賣合 × 100%

買賣合：買進張數 + 賣出張數

怎麼看：

  • 買賣力 > 60% → 該分點強力站在買方
  • 買賣力 < -60% → 該分點強力站在賣方
  • 買賣力接近 100% 或 -100% → 分點幾乎全是單邊操作

實例：

  • 北極星（6550）2022/9/22 凱基-台北
  • 買賣力 99.63%，幾乎全部是買進
  • 成交力 37.05%，佔個股成交量三成以上
  • 當天開盤即漲停

3. 分點成交力

公式：成交力 = 買賣超 ÷ 個股成交量 × 100%

怎麼看：

  • 成交力 > 20% → 該分點是當日主力
  • 成交力 < 10% → 該分點影響力有限

注意：成交力高不代表一定漲，要搭配股價走勢看。

系統券商 vs 指定券商 vs 追蹤券商

類型用途實戰場景
系統券商看整體類型（外資、投信、自營商）觀察法人動向
指定券商鎖定單一分點追蹤特定主力
追蹤券商自建券商組合監控隔日沖名單
自選券商大範圍搜尋掃描主力買了哪些股

四、隔日沖券商識別

隔日沖是什麼？

定義：今天買進、隔天賣出的短線操作手法。

核心邏輯：

  • 大資金拉抬股價（甚至拉到漲停）
  • 隔天開高時賣出持股
  • 雖然價差不大（可能 3-5%），但量大、勝率高

隔日沖兩大手法

1. 拉漲停策略

怎麼做：
1. 選當天強勢股
2. 大量買進，拉到漲停鎖死
3. 散戶看到漲停會跟進排隊
4. 隔天開高時賣出

特徵：

  • 隔天早盤開高
  • 09:30 後開始賣壓出籠
  • 可能收黑 K 或留下影線

2. 權證+股票策略

怎麼做：
1. 大量買進權證和標的股票
2. 自營商因為風險控制必須買進股票避險
3. 兩方買盤推升股價
4. 隔天全部賣出

為什麼有效：自營商「必須」避險，是制度性買盤。

  • 買超量是否遠超第二名
  • 買進張數佔成交量比例
  • 該分點歷史是否常有隔日沖紀錄

  • 凱基-台北買超第一名，買超張數是第二名兩倍
  • 判斷：隔日沖機率很高
  • 7/2 早盤拉高 7%
  • 過了 09:30 股價下挫
  • 最終爆量收黑

五、如何應對隔日沖

手中持股遇到隔日沖

情境一：短期籌碼集中 + 上升趨勢

  • 如果 5 日、20 日籌碼集中度 > 0
  • 且股價在上升趨勢中
  • 隔日沖籌碼移出後，趨勢多半會繼續
  • 不必急著出場

情境二：持有成本低

  • 按原本計畫停損停利
  • 不需要被隔日沖影響

情境三：開盤判斷

  • 開盤 > 4% → 股價強勁，可續抱
  • 開盤 < 4% 且成交量迅速擴大 → 先獲利出場

要跟單隔日沖嗎？

不建議原因：

  • 散戶資金不如大戶
  • 無法像大戶一樣影響股價
  • 資訊不對稱

如果真的要跟：
1. 選中小型股（流動性高）
2. 見好就收，快進快出
3. 不長期持有，最長 3-5 個交易日
4. 交割款項要充足（T+2 扣款）

三個實戰建議

1. 尾盤再做決定

  • 盤中波動大

2. 不要預設立場

  • 隔日沖可能隔 2 日、3 日才賣

3. 長期投資人不受影響

  • 隔日沖是短線操作

心得

分點進出是籌碼面分析的核心工具，可以從中看出主力的蛛絲馬跡。關鍵在於：

1. 找對關鍵分點：不是所有分點都重要，要找買超量突出、歷史績效好的分點。

2. 理解隔日沖邏輯：隔日沖不是壞事，他們只是快進快出的短線客。關鍵是識別他們、理解他們的操作模式，然後做出對應的判斷。

3. 不要被名稱嚇到：看到「隔日沖券商」不要恐慌，要看：

  • 買超量是否異常

4. 地緣券商是公司派風向球：公司附近的券商大量買超，可能是有利多消息提前佈局。

5. 分點買賣力 > 60% 是強力訊號：代表該分點幾乎是單邊操作，意圖明確。
