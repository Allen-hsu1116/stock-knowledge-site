當沖選股三個必要條件

1. 流動性佳

是什麼：股票要能順利買進賣出，不會卡住

怎麼做：
1. 日成交金額 > 5,000萬（基本門檻）
2. 日成交量 > 2,000張（更實用的篩選標準）
3. 個人當沖金額不超過該股日成交金額的 1-2%（避免流動性風險）

實例：

  • 你要當沖 100萬 → 選日成交金額 > 2,000萬的股票
  • 你要當沖 1,000萬 → 選日成交金額 > 2億的股票

為什麼重要：

  • 量太小 → 價格不理想、賣不掉、容易被大戶操控
  • 量夠大 → 進出順暢、價格合理、行情真實

2. 波動性大

是什麼：股票當日高低點差距要夠大，才有價差可賺

怎麼做：
1. 計算日均波動：看當日振幅或近幾日平均高低點差距
2. 用 ATR（平均真實振幅）指標量化
3. 肉眼觀察：走勢圖上下起伏明顯 vs 一條線

實例：

  • 漲跌幅排行：查當日、昨日、30日內排行
  • 週轉率排行：週轉率越高 = 越熱門
  • 理想波動：日均 3% 以上（扣掉交易成本 0.3-0.4% 後還有空間）

警語：

  • 台積電、中華電信 → 波動小，不適合當沖
  • ETF → 分散風險，波動較個股小，也不理想

3. 有題材

是什麼：市場關注的焦點，能吸引短線資金進場

怎麼做：
1. 新聞題材：政府政策、公司利多
2. 產業趨勢：AI、電動車、半導體熱門族群
3. 財報公告：營收大增、重大合作

實例：

  • AI 熱潮 → 台積電動，伺服器、散熱、記憶體族群跟著放量
  • 找「有量、有波動、有題材」的標的

適用時機：

  • 大盤熱鬧 → 熱門族群波動放大
  • 大盤冷清 → 選股要更嚴格，避免「有量但不動」

當沖進場時機

進場三步驟

步驟一：判斷今天適不適合進場

怎麼做：
1. 觀察大盤/個股有沒有穩定的漲跌方向（有方向 = 市場有共識）
2. 檢查成交量有沒有放大（個股日成交量 > 2,000張）
3. 看有沒有明顯熱門題材或族群

判斷準則：

  • 方向明確 + 成交熱絡 + 資金集中 → 適合出手
  • 盤整無方向 + 成交清淡 → 今天不操作也沒關係

步驟二：三個進場圖形

圖形一：上升趨勢線

  • 是什麼：股價低點越來越高，連成一條上升線
  • 怎麼做：股價接近趨勢線時買進
  • 警訊：跌破趨勢線 → 可能反轉，考慮賣出

圖形二：壓力變支撐

  • 是什麼：股價突破壓力線後站穩，壓力線變成支撐線
  • 怎麼做：在靠近支撐線的價格進場
  • 為什麼有效：突破後壓力線變成市場共識的支撐

圖形三：假跌破

  • 是什麼：股價跌破支撐線，但很快拉回來
  • 怎麼做：拉回穩住後，在支撐線附近買進
  • 注意：要確認「穩住」才是真的轉強

步驟三：進場前的自我提問

問題一：我這次進場的理由？

  • 走勢亂七八糟、看不懂 → 不出手
  • 看不懂，就不出手

問題二：我設定好停損了嗎？

  • 進場前先想好「萬一錯了怎麼處理」
  • 停損點要具體（例如：跌 1% 就出場）

心法：

  • 不求買最低、賣最高
  • 抓到一段你看得懂的走勢，在合理位置進出

當沖出場策略

方法一：固定停損／停利比例

怎麼做：

  • 漲 2% 就賣出（停利）
  • 跌 1% 就出場（停損）

為什麼有效：

  • 避免猶豫不決、捨不得賣
  • 避免貪心硬撐

方法二：設定出場訊號

訊號一：突然爆大量卻沒再漲 → 主力可能出貨
訊號二：明顯長黑K → 可能轉弱，考慮減碼
訊號三：多頭走勢中忽然轉震盪 → 減碼觀望

方法三：時間管理

怎麼做：

  • 進場後 20 分鐘走勢沒變動 → 出場觀望
  • 最晚 13:25 一定要出場

最佳交易時間：

  • 開盤前半小時（9:00-9:30）：波動大，但難度高
  • 盤中（9:30-12:30）：型態明確，相對好判斷
  • 12:30 後：不建議入場（風險高）

當沖交易六大注意事項

注意一：確認資金上限

  • 千萬不要無本當沖
  • 單筆虧損控制在總資金 0.5% 以內
  • 單次投入不超過總資金 20%

注意二：專注少數股票

  • 新手最多 3 檔以內
  • 太多檔容易顧此失彼

注意三：設停利停損

  • 虧 10% → 需要獲利 11.1% 才能回本
  • 虧 50% → 需要獲利 100% 才能回本
  • 停損遠比賺錢重要

注意四：把握交易時間

  • 9:00-12:30 是最佳時段
  • 12:30 後不建議入場
  • 一天最多操作 3 趟

注意五：避免過度交易

  • 一天走勢最多 1-3 次轉折
  • 沒趨勢、沒波動 = 付出成本而已
  • 達到目標就收手，明天還有行情

注意六：了解期望值

  • 當沖要賺錢，勝率要 60-70% 以上
  • 大多數人賠錢賠時間
  • 心態正確才能長久

新手常見錯誤

錯誤一：沒有進出計畫

  • 問題：看到漲就追、跌了不停損
  • 解決：每一筆進場前先設定目標價與停損點

錯誤二：賺一點就跑，賠錢就跑不動

  • 問題：小賺大賠
  • 解決：紀律比賺多更重要

錯誤三：沒觀察股票習性

  • 問題：每檔股票漲跌節奏不同
  • 解決：專注幾檔熟悉的標的，熟悉它的節奏

錯誤四：選冷門股

  • 問題：量小、滑價嚴重、出場困難
  • 解決：成交量 > 2,000張

錯誤五：只看新聞標題

  • 問題：利多新聞不等於股價會動
  • 解決：題材要配合成交量

錯誤六：忽略波段位置

  • 問題：股票已漲一大段還追高
  • 解決：即使當沖也要看支撐壓力

進階選股技巧

技巧一：籌碼面

  • 三大法人連續買超 → 帶動市場關注
  • 當沖熱門標的 → 短線資金集中，波動度更高

技巧二：關鍵價位

  • 突破關鍵價位 → 帶動追價，急漲
  • 跌破關鍵價位 → 引發恐慌，急殺
  • 注意：支撐壓力、前高前低、整數關卡

技巧三：市場氛圍

  • 大盤強勢 → 熱門族群容易爆量上攻
  • 大盤疲弱 → 找「逆勢抗跌」或「題材獨強」的標的
  • 族群效應 → 整個產業上漲，龍頭股和二線股一起活絡

心得

這幾篇文章把當沖選股和進出場時機講得很清楚，核心就三件事：有量、有波動、有題材。

但最讓我印象深刻的是「交易成本」這件事。一買一賣成本 0.3-0.4%，如果股票日均波動只有 2%，扣掉成本後根本沒利潤空間。難怪市場先生說「日均波動 3% 以上才是理想標的」。

還有一點很關鍵：當沖不是每天都要操作。看不懂就不出手，這比硬上更重要。新手最常犯的錯就是賺小賠大，停損紀律比抓點位更重要。

最後，進場前問自己兩個問題：
1. 我這次進場的理由是什麼？
2. 我停損點設在哪裡？

這兩個問題沒想清楚就不要進場。

一、區間震盪操作策略

適用情境：大盤無明確趨勢，指數呈現上下邊界來回震盪

核心觀念：高出低進的區間交易邏輯，不追高殺低

VIXTWN 恐慌指數應用：

  • 是什麼：台指選擇權隱含波動率，反映市場對未來 30 日波動預期
  • 怎麼用：

1. VIXTWN 突破 30 以上 → 高恐慌 → 進場布局
2. VIXTWN 回落至 15-18 → 低恐慌 → 減碼
3. VIXTWN 與加權指數呈負相關

  • 實例：
  • 數據：VIXTWN 突破 30 進場，3 個月後平均上漲 13.7%，6 個月後平均上漲 19.7%

二、技術面抗跌股篩選

核心邏輯：題材是誘因，技術面抗跌力才是切入關鍵

篩選條件：

條件一：大盤破底但個股未破底

  • 顯示個股相對大盤更具支撐力
  • 資金未撤出，甚至有護盤跡象
  • 實例：2025 年初至 4/9 大盤恐慌下殺，驊宏資、華經、寶一、晟田皆未隨大盤破底

條件二：站穩關鍵均線

  • 月線（20MA）：短中期趨勢轉強
  • 季線（60MA）：具備波段反彈潛力
  • 股價有效站回 = 買盤力道強

進出場策略：

  • 依據道氏理論趨勢通道操作
  • 找出技術抗跌股 + 確認題材支撐 → 短波段操作

Tick流：

  • 捕捉股價微小波動
  • 每次進出場間隔數秒或數分鐘
  • 適用於頻繁震盪且成交量活躍的標的

一波流：

  • 掌握當日震盪最大的短期趨勢
  • 追蹤成交量變化與大單流向
  • 操作標的通常為熱門族群與題材股

日內波：

  • 研判個股有機會出現長紅K或長黑K才進場
  • 通常等趨勢確立後進場，接近收盤時出場
  • 非常注重選股策略，需基本面/技術面/籌碼面篩選

四、日內交易各時段特點

開盤前：

  • 觀察昨日美股收盤、盤前海外期貨走勢
  • 美期貨盤、日韓股開盤、台指期開盤
  • 決定偏多或偏空操作方向

開盤後（9:00-9:30）：

  • 市場劇烈反應消息、短線籌碼變化
  • 「快攻時間」，短進短出機會多
  • 部分強勢股會快速搶攻漲停板

早盤（9:30-10:30）：

  • 標的走勢逐漸明朗
  • 技術型態與市場動態提供操作方向

中盤（10:30-12:30）：

  • 成交量下降，波動縮小
  • 宜減少出手次數

尾盤（12:30-13:30）：

  • 當沖交易者準備出場
  • 波段交易者下定決心進場
  • 成交量擴增，股價波動加劇
  • 注意拉尾盤或下殺變化

五、短線關鍵技術指標

當日開盤價：

  • 股價在開盤價上方 = 紅K棒
  • 股價在開盤價下方 = 黑K棒
  • 開盤價為日內走勢多空分界線
  • 行情強勁的股票開盤後會迅速脫離開盤價

當日均價線：

  • 當天的平均交易成本
  • 價格維持在上方 = 多方趨勢明確，逢拉回可買進
  • 價格維持在下方 = 空頭佔優勢，逢反彈可放空

日K線關鍵點位：

  • 近期高低點、昨日收盤價易形成支撐或壓力
  • 突破或跌破後易產生趨勢

日K線重要技術指標：

  • 均線、布林通道
  • 上揚的均線容易成為支撐
  • 下彎的均線則易成為壓力

實戰案例 - 凱美（2375）：

  • 開盤不久後成交量顯著增加
  • 股價突破昨日高點 → 進場時機
  • 或待拉回不破均價線確認下檔有撐後再布局
  • 最終收在漲停價

實戰案例 - 聯光通（4903）：

  • 跳空開高但遇 20 日均線反壓
  • 早盤攻高無力後跌落均價線
  • 反彈至均價線上但量能未放大
  • 走勢延續性轉弱 → 不適合作多，可逢彈放空

六、勝率與賺賠比

勝率：獲勝機率

  • 10 次交易中有 7 次賺錢 → 勝率 70%

賺賠比：承擔每單位風險能獲得多少報酬

  • 公式：平均獲利 ÷ 平均虧損
  • 例：平均獲利 30 元，平均虧損 10 元 → 賺賠比 = 3

期望值公式：

期望值 = (賺賠比 × 勝率) - (1 - 勝率)

實例：

  • 勝率 40%，賺賠比 3
  • 期望值 = 3 × 0.4 - 0.6 = 0.6
  • 每次交易平均獲利 0.6 元 → 正期望值策略

七、波段操作核心觀念

股票 ≠ 房地產：

  • 房地產長期往上，股票不一定
  • 股票可能被淘汰、下市
  • 20 年前的前 50 大股跟現在不一樣

三大原則：
1. 不要隨便賣：下跌時亂賣容易賣在低點
2. 保留現金：下跌時不要加碼攤平
3. 學會停損：越早停損賠越少

賠錢時不加碼攤平：

  • 越買賠越多，攤也攤不平
  • 看好某檔股票 → 轉空時先賣出，等止跌再逢低買回

40 日均線控盤法：

  • 40 日均線 ≈ 2 個月交易日
  • 主力吃貨平均需 2 個月
  • 怎麼用：

1. 均線往上 → 做多
2. 均線往下 → 做空或空手
3. 跌破 40 日均線 → 停損出場

  • 實例 - 台積電：跌破 40 日均線後一路往下，沒有持有必要
  • 實例 - 聚積：跌破 40 日均線後持續下跌，應在漲過均線時持有

八、期貨留倉六大注意事項

1. 只在賺錢時留倉

  • 賠錢時留倉 = 賭隔天會賺回來
  • 如果繼續跌 → 越賠越大
  • 一開始賠錢就該停損

2. 事先設定出場點

  • 包含日盤及夜盤
  • 進場前先設停損
  • 獲利後設停利
  • 不要讓賺錢的單變賠錢

3. 跳空避險

  • 用選擇權買方鎖住獲利
  • 期貨做多 → 加買選擇權 BUY PUT 避險
  • 期貨做空 → 加買選擇權 BUY CALL 避險
  • 用賺的錢買選擇權
  • 週選擇權比月選擇權便宜

4. 接受行情可能走不出去

  • 行情總是盤整很久才有一段趨勢
  • 留倉做多遇到盤整 → 買反向選擇權避險
  • 行情下跌 → 期貨賠但選擇權賺
  • 行情上漲 → 期貨賺比選擇權賠更多

5. 順勢留倉 + 慎選進場點

  • 行情偏多 → 留多單
  • 行情偏空 → 留空單
  • 進場點不好 → 反彈就停損
  • 例：1200 點跌幅中，200-300 點震幅合理，但短線進場點差會被掃出

6. 停損不要設太小

  • 20-30 點太小，現在行情波動 200-300 點
  • 建議用 ATR 的 1/4 或 1/5 設定停損
  • 例：ATR 219 點 → 1/5 約 44 點，設 50 點停損
  • 海龜交易法則：停損設很大，用大資金守住風險控管

九、均線波段交易實戰

N 日均線波段交易：

  • 用 60MA 時，不要設定碰到 60MA 就出場

核心知識

價值投資核心理念

是什麼：找出被低估的好公司，用便宜或合理的價格買進，長期持有等待價格回歸價值。

核心思想：

  • 價格 ≠ 價值，但長期而言價格會回歸內在價值
  • 巴菲特名言：「價格是你付出的，價值是你得到的」
  • 目標：找到好公司 + 用低於價值的價格買進

適合誰：

  • 願意長期持有（以年為單位）
  • 能夠耐心等待買點
  • 願意做功課研究公司

內在價值評估三大方法

方法一：清算價值法（Net Net）

是什麼：評估公司清算後的剩餘價值，在遠低於清算價時買進。

怎麼做：
1. 計算公司資產 - 負債 = 清算價值
2. 清算價值再打 5~6 折作為買進價
3. 等待股價跌到目標價買進

實例：

  • 公司清算後估計有 10 億價值
  • 打 5 折 = 5 億市值時買進
  • 適合：營運差但資產豐富的公司

注意：現代好機會稀少，虧損公司清算價值會逐年下降

方法二：現金流折現法（DCF）

是什麼：用公司未來能創造的現金流，折現計算當下的內在價值。

怎麼做：
1. 預估公司未來每年的自由現金流
2. 用折現率（常用 10%）折現到現在
3. 加總所有折現後的現金流
4. 內在價值打 7~8 折作為買進價

實例：

  • 公司預估每年每股自由現金流 10 元
  • 折現率 10%，計算 10 年
  • 內在價值 = 10÷1.1¹ + 10÷1.1² + ... + 10÷1.1¹⁰ = 61.4 元
  • 安全邊際打 7 折 = 43 元買進

注意：成長型公司要考慮成長率，但成長率估計誤差會被放大

方法三：簡化估價指標

本益比（PE）：

  • 公式：股價 ÷ EPS
  • 判斷：<12 偏便宜，15 可接受，>20 偏貴
  • 注意：不同產業標準不同

本益成長比（PEG）：

  • 公式：本益比 ÷ 預估成長率
  • 判斷：PEG < 1 偏便宜
  • 優點：考慮成長性

股價淨值比（PB）：

  • 公式：股價 ÷ 每股淨值
  • 判斷：<1 偏便宜
  • 適合：資產型公司

現金殖利率：

  • 公式：現金股利 ÷ 股價
  • 判斷：>5% 偏高
  • 注意：股息來自獲利，是更保守的評估

安全邊際概念

是什麼：內在價值與買進價格的差距，用來降低投資風險。

怎麼做：
1. 估出內在價值（例如 100 元）
2. 打折作為買進價（例如打 8 折 = 80 元）
3. 打折幅度 = 安全邊際

實例：

  • 內在價值 100 元
  • 安全邊際 20%（打 8 折）
  • 買進價 80 元
  • 好處：估錯也只少賺 20 元，風險可控

注意：安全邊際越大越安全，但可能錯過機會

投資護城河五大類型

是什麼：公司獨特的競爭優勢，讓競爭對手難以打破。

1. 無形資產

  • 品牌（可口可樂）
  • 專利（藥廠）
  • 特許執照

2. 轉換成本

  • 用戶換廠商要付出極大代價
  • 例：企業軟體、銀行帳戶

3. 網絡效應

  • 越多人用越好用
  • 例：FB、微信支付

4. 成本優勢

  • 生產程序優化
  • 地點優勢
  • 特有資產

5. 規模優勢

  • 規模經濟
  • 市場獨佔

真假護城河判斷：能否轉為「訂價權」——對客戶調漲售價或對供應商壓低成本

價值陷阱是什麼？

是什麼：以為買到便宜股，但股價持續下跌，永遠不回歸價值。

怎麼避開：
1. 股價下跌要問「為什麼」
2. 判斷是一次性影響還是長期衰退
3. 長期獲利能力衰退 = 不要買
4. 尋找「明顯的機會」，而非模稜兩可

實例：

  • 本益比低但營運持續惡化
  • 高股息但獲利能力下降
  • 淨值比低但資產變現困難

價值投資六步流程

Step 1：確定策略類型

  • 低價買平凡公司（清算價值法）
  • 合理價買優質公司（現代派）← 推薦

Step 2：選股——建立口袋名單

  • 分析公司體質：獲利、負債、股利
  • 閱讀財報、理解產業
  • 分散到 5~10 檔股票
  • 問自己：證交所關閉 10 年，你還願意持有嗎？

Step 3：估價——乘上安全邊際

  • 選擇估價方法（本益比、DCF 等）
  • 算出內在價值
  • 打 7~8 折作為買進價

Step 4：等待——耐心等機會

  • 等價格跌到目標價
  • 好機會常在市場恐慌時出現

Step 5：買進——耐心持有

  • 價格回歸價值需要時間
  • 不要期待快速獲利

Step 6：持有或賣出

  • 定期檢驗公司體質
  • 價值改變 → 果斷認賠
  • 價格不合理高估 → 分批賣出

進出場策略

進場時機

分批買入：

  • 資金分成多批，定期定額買入
  • 例：每月買 1/12 的資金
  • 適合：無法判斷最佳買點

估值買入：

  • 本益比低於歷史均值
  • 股價淨值比接近 1
  • 適合：有能力評估價值

市場恐慌買入：

  • 非經濟因素造成恐慌
  • 優質股被錯殺
  • 適合：有膽識逆勢

出場時機

分批賣出：

  • 達到目標價分批賣
  • 適合：不確定是否高點

再平衡：

  • 股票權重過高時賣出
  • 維持多元配置

達成獲利目標：

  • 設立 20%~30% 獲利目標
  • 達到就分批出場

基本面惡化：

  • 獲利能力大幅下降
  • 管理層出問題
  • → 立即賣出

心得

這篇學的是價值投資，跟之前學的當沖、波段完全是不同路數。當沖拼的是速度和技巧，價值投資拼的是耐心和判斷力。

幾個關鍵體會：

1. 安全邊際是保命符——寧願少賺也不要虧損，這是巴菲特的核心原則。估價再準也會錯，打折買進就是給自己留退路。

2. 台股參數要調整——直接套用大師參數可能會錯過很多好標的。電子股本益比高一點（<20）還能接受，重點是成長性和護城河。

3. 護城河比數字重要——財務指標會變，但護城河是長期的。有品牌、有轉換成本、有網絡效應的公司，才值得長期持有。

4. 價值陷阱很危險——便宜的股票可能有便宜的理由。股價下跌要先問「為什麼」，不要看到低本益比就衝進去。

5. 要像老闆一樣思考——「如果證交所關閉 10 年，你還願意持有嗎？」這個問題很強。如果你不想當這家公司的老闆，就別買它的股票。

價值投資的核心不是「買什麼」，而是「用什麼價格買」。好公司 + 好價格 = 好投資。但「好價格」需要等待，這就是最考驗耐心的地方。

一、當沖標的選擇黃金法則

是什麼：不是所有股票都適合當沖，選錯標的是「10個當沖9個輸」的主因。

怎麼做：
1. 成交量門檻：

  • 股價 < 100 元：日成交量要 > 10,000 張

2. 波動要夠大：當日振幅要能 cover 成本（手續費 + 交易稅）
3. Tick 大小是關鍵：

  • 最小 tick 升幅就能獲利的價位最適合當沖

實例：

  • 90 元股票漲停（10%）：從 90 → 99，隔著 90 個 tick
  • 100 元股票漲停（10%）：從 100 → 110，只隔 20 個 tick
  • 同樣漲停，100 元更容易達到！
  • 100-149 元區間：tick 為 0.5，最小升幅 1 tick 就能獲利

適用時機：選股時優先篩選，避開量能不足、波動太小的股票。

二、當沖三大交易時段特性

是什麼：台股 9:00-13:30，不同時段有不同操作邏輯。

怎麼做：

時段時間特性適合誰
開盤9:00-10:00量大、波動大、方向不明老手、上班族趕時間
盤中10:00-13:00型態明確、相對穩定專職當沖客主戰場
尾盤13:00-13:30時間短、易被操控不建議進場

實例：

  • 情境：早上 9:15 開盤後量能爆出
  • 錯誤操作：看到大單就追價，結果來回洗盤被停損
  • 正確操作：等 9:30 後方向確立，再順勢進場

適用時機：

  • 上班族：開盤時段快速操作
  • 專職者：盤中時段耐心等待型態

三、當沖停損停利比例設定

是什麼：設定合理的風險報酬比，是當沖存活關鍵。

怎麼做：
1. 基本比例：虧損:獲利 = 1:2 以上
2. 設定方式：

  • 進場時就設好停損停利點

3. 避免的錯誤設定：

  • 1:10 比例太貪心，很難達到

實例：

  • 買進價：100 元
  • 停損設：99.5 元（虧 0.5 元）
  • 停利設：101 元以上（賺 1 元以上）
  • 風險報酬比 = 1:2

適用時機：每筆交易進場前必設，進場後嚴格執行。

是什麼：波段操作是利用「發酵較長時間的事件」進行投資，勝率高且不用盯盤。

技巧 2：產業突破性技術

  • 例：ChatGPT 出現 → 做多 AI 產業 ETF
  • 出場：價格突破前高後或財報公告前

技巧 4：QE/QT 與保值商品

  • 印鈔時期（QE）→ 買入黃金、比特幣
  • 出場：政策方向改變時

技巧 5：技術面多頭格局

  • 長期盤整後突破前高 → 順勢買入

是什麼：波段操作不需盯盤，但風險管理仍不能少。

1. 停損位：設在支撐位以下，控制在可接受範圍

3. 停盈位：接近壓力位時鎖定利潤
4. 紀律性：

  • 制定交易計劃（進場點、停損、停盈）

實例：

  • 買進：技術面突破前高，進場價 100 元
  • 停損：95 元（支撐位以下）

動能交易核心概念

什麼是動能交易？

核心邏輯：「強者恆強、弱者恆弱」——表現好的股票，未來也會表現好；表現差的，未來也會表現差。

為什麼有效？

  • 動能出現 = 資金進駐 = 可能吸引更多資金跟進
  • 失去動能 = 資金暫時不感興趣 = 上漲機率低
  • 等有動能才進場 = 資金效率最高

適用市場：股票、期貨、外匯、ETF 都適用

五、實戰要點總結

動能交易三大核心

1. 選股：找有動能的股票（創新高、爆大量、均線多頭排列）
2. 進場：追價或回測買進，看個人風格
3. 出場：停損嚴格、停利依交易風格調整

適用時機

適合：

  • 明顯多頭或空頭市場
  • 有明確趨勢的股票
  • 成交量活躍的標的

不適合：

  • 盤整震盪市場
  • 無量下跌或上漲
  • 基本面惡化的股票

風險控制

1. 假突破：停損要嚴格執行
2. 盤整磨損：減少交易頻率，等待明確趨勢
3. 基本面風險：搭配基本面篩選，避免追逐沒有營運支撐的股票

心得

動能交易的核心就是「追強不追弱」，但這「追」要有方法、有紀律。今天的學習讓我理解了幾個關鍵：

第一，動能不是亂追。要有篩選條件：創新高、爆大量、均線多頭排列，這些都是資金進駐的訊號。不是漲了就追，而是要確認「真的有動能」。

第二，進場方式要看個性。積極的人直接追價，保守的人等回測。但回測買進不是「掛單等」，而是「確認不破再買」——這是關鍵差別。

第三，停利方式要看交易風格。tick流5~15分鐘沒續漲就跑；波段可以抱到中盤或收盤。重點是：獲利後要移動停損，鎖住獲利。

第四，盤整是動能交易的殺手。盤整期會一直假突破，一直停損。所以動能策略要在「有趨勢」的市場用，盤整期要縮手或用其他策略。

第五，動能策略有滯後性。今天表現好的股票，要等一段時間才能買進，可能錯過最佳進場點。但也因為這個滯後，有時候反而能吃到更大的波段。

這跟之前學的當沖、波段操作可以結合：動能選股 + 當沖時段 + 波段持有。重點是「動能只是選股方法，操作策略還是要有自己的節奏」。

  • 選擇權 PUT 從 146 元漲到 2400 元（16 倍）

適用時機：

  • 大盤持續下跌、多空循環時
  • 外資期貨空單 > 3 萬口時
  • 外資選擇權從多單翻空單時

籌碼面判斷空頭訊號

是什麼：透過外資、自營商的期貨選擇權部位，判斷行情轉折。

怎麼做：
1. 外資期貨空單：超過 3 萬口 = 大空頭訊號
2. 外資選擇權加總：BC + BP 加總，負值 = 外資做空
3. 自營商 + 外資同步做空：大跌機率極高

實例：

  • 7/18 外資選擇權翻空，大盤在 23398 點歷史高點
  • 兩週內崩跌至 19830 點
  • 自營商 BP 金額一天增加 7 億（歷史新高）

適用時機：

  • 每日追蹤期貨選擇權籌碼
  • 外資從多翻空 = 轉折點
  • 自營商 + 外資同步 = 大跌訊號

反彈空 vs 追空

是什麼：空頭市場不追空，要等反彈再空。

怎麼做：
1. 不追空：破底空容易被反彈掃停損（反彈可能 300-600 點）
2. 反彈空：空在轉折右側，降低風險

判斷方式：

型態學：

  • 早盤先拉升 → 拉抬無力 → 轉頭向下 = 空點
  • 高點不過前高 = 放空訊號

均線判斷：

  • 用 5 分 K + 60MA 或 3MA
  • 均線從上揚（紅）轉下彎（綠）= 進場時機

實例：

  • 8/1 放空：大盤反彈約 420 點，在高點進場
  • 8/2 放空：當天跌 1000 點，隔天再跌 2000 點

適用時機：

  • 波動大的環境（停損要設更大）
  • 短線操作可用 3MA 抓轉折

停損停利設定

是什麼：放空也需要嚴格的停損停利，保護資金。

怎麼做：

停損：

  • 根據當日波動（ATR）的 20-25%
  • 例：波動 800 點 → 停損 200 點
  • 設在前高之上，避免被掃

移動停利：

  • 進場後沒打到停損 → 啟動移動停利
  • 例：最近 3 根 K 棒低點 + 150 點
  • 機械式往下跟，保護已實現獲利

加碼條件：

  • 反彈至少 100 點以上
  • 均線出現轉彎訊號
  • 停損設在新轉折高點上方

實例：

  • 第一筆單停損 200 點
  • 反彈 100 點後加碼第二筆
  • 總虧損仍控制在 130 點內

適用時機：

  • 大行情可加碼至 4 口單
  • 波動縮小時停損要跟著縮（400 點波動 → 100 點停損）

實戰案例

案例一：獲利 70 萬

  • 進場點：均線下彎、型態高點不過前高
  • 加碼：反彈超過 100 點 + 3MA 轉彎
  • 最大持倉：4 口單
  • 出場：移動停利觸發

案例二：獲利 26 萬

  • 型態：逐步降低的高點
  • 進場：第一個均線下彎點
  • 加碼：反彈 100 點
  • 獲利：400 多點

心得

這篇把「放空」講得很清楚。關鍵是不追空，等反彈再空。這跟之前學的「不追高」是一樣的道理——追的都容易被洗出去。

籌碼面判斷很實用：外資期貨空單 > 3 萬口、選擇權從多翻空，這些都是明確訊號。還有自營商 + 外資同步做空 = 大跌。這些以後看盤要特別注意。

停損用 ATR 的 20-25% 是好方法，會隨波動自動調整。移動停利用「3 根 K 棒低點 + 150 點」也很實際，不用盯盤也能保護獲利。

空頭市場不是只能被動挨打，學會放空反而能逆勢獲利。但這風險也大，一定要嚴格執行停損。

突破交易核心概念

是什麼：避開盤整區間，等股價突破壓力後再進場，抓住趨勢起漲點。

為什麼有效：

  • 盤整區間是市場共識形成的價格帶
  • 突破代表新資金進場、市場願意用更高價格持有
  • 基於統計學和心理學原理

判斷真假突破四原則

原則一：以收盤價為準

怎麼做：

  • 盤中跌破不算跌破，收盤跌破才叫真跌破
  • 盤中追空容易遇到長下影線反彈，陷入兩難

為什麼重要：收盤價是多空雙方一天博弈的結果，最具參考價值

原則二：連續三天有效

怎麼做：

  • 連續三天的收盤都跌破支撐，才認定為真跌破
  • 連續三天的收盤都突破壓力，才認定為真突破
  • 若跌破兩天後第三天反彈，視為假突破

實例：

  • 情境：某股票跌破支撐100元
  • 判斷：第一天收盤99元、第二天收盤98.5元、第三天收盤98元 → 真跌破
  • 反例：第一天99元、第二天98元、第三天100.5元 → 假跌破

原則三：回測原方向（非必要）

怎麼做：

  • 跌破後小反彈測試，再往下續跌
  • 突破後小拉回測試，再往上續攻

效果：有回測的型態效果更好，但沒有也不妨礙真突破判斷

原則四：要有一定幅度

怎麼做：

  • 指數：連續三天跌破支撐，且超過 0.5%
  • 股票：連續三天突破壓力，且超過 3%

為什麼需要幅度：

  • 避免小波動被誤判為突破
  • 確保突破有實質力道

真假突破實戰判斷

量能觀察

怎麼做：

  • 真突破：量增，顯示資金願意追價
  • 假突破：放量卻漲不動，可能是主力拉抬出貨

跌破支撐後的反應

怎麼做：

  • 跌破支撐但隔日量能急縮 → 可能是假跌破，市場不願賣
  • 放量下跌 → 可能是真跌破，主力抽身

盤整時間與波段大小的關係

經驗法則：

想賺的波段需要的盤整時間
一個月至少 3 個月以上
半年1～2 年
一年以上5 年以上

為什麼：

  • 消息很即時，基本面變化一出現股價就會先反應
  • 長期沒有利多的公司，突然爆出好消息（虧轉盈、業績大成長）爆發力最強

最適合突破交易的標的：

  • 平常沒人關注的冷門股
  • 一有利多就能吸引市場目光
  • 例：疫情爆發時的航運股、AI 爆發時的 NVIDIA

不適合突破交易的標的：

  • 巨無霸公司（蘋果、波克夏）
  • 體量太大，單一利多難以引爆股價
  • 適合長期持有，不適合賺突破快錢

出場時機判斷

技術面出場

判斷原則：

  • 每次回檔都沒破前低 → 上漲趨勢還在，續抱
  • 連續跌破前波低點 → 動能減弱，可能反轉，該走

實例：

  • 情境：NVIDIA 2021 年從 $200 漲到 $330
  • 出場訊號：12 月底連續兩次跌破前波低點
  • 操作：獲利了結，不要戀戰

基本面/消息面出場

怎麼做：

  • 漲勢要看「資金」跟「故事」有沒有延續
  • 圖形破壞 + 消息不再 = 反轉機率高
  • 利多消失就該走

NVIDIA 案例：

  • 漲勢動力：拆股 + 比特幣大漲
  • 出場時機：比特幣價格開始走弱、跌破前波低點

實戰操作建議

進場：

  • 滿足第一原則（收盤價為準）即可進場
  • 雖然勝率會降低，但不會錯過大行情
  • 可等確認再進場，真突破後的漲勢往往是一段完整趨勢

停損：

  • 突破失敗要果斷停損
  • 假突破的特徵：量增卻漲不動、跌破後量能急縮

停利：

  • 觀察前低是否被跌破
  • 搭配基本面消息判斷故事是否結束

心得

突破交易這招其實很直覺：等在那個點位，突破了再追。但魔鬼在細節裡——假突破這個坑太常見了，一掉進去就是追高被套。

四原則裡面，最實用的是「收盤價為準」和「幅度要夠」。盤中追突破很容易被洗出去，因為盤中波動大、主力會做價。等收盤確認才進場，雖然可能少賺一點，但勝率會高很多。

幅度這點也很有意思——指數要 0.5%、股票要 3%。這差異反映了波動特性：股票波動本來就比指數大，所以標準要更嚴格。

另外一個關鍵點：突破交易要看的不只是技術面。文章提到 NVIDIA 和 TESLA 的案例，它們的大波段都是「基本面質變 + 技術面突破」的共振。純技術突破沒有故事支持，很容易變假突破。

最後是出場：連續跌破前低就該走。這個原則簡單但有效。很多人賺到錢捨不得走，結果抱著抱著就變虧損。設定好出場規則，該走就走，不要戀戰。

金字塔加碼法

是什麼：一種分批進場的策略，股價上漲時逐步加碼，而不是一次買滿。核心是「賺錢時才加碼，虧損時絕不加碼」。

為什麼這樣做：

  • 一開始用小部位試單，錯了損失有限
  • 趨勢確認後再加碼，提高勝率
  • 平均成本會比一次買進高，但風險可控

怎麼做：
1. 先判斷標的和進場點（技術突破或價值合理）
2. 第一筆進場：投入預計總資金的 20-40%（試單）
3. 股價上漲突破關鍵點位（如箱型、旗型、三角形突破）後加碼
4. 每次加碼金額遞減（如第一筆 10 萬、第二筆 5 萬、第三筆 3 萬）
5. 設好停損點，破了就出場

適用時機：

  • 適合：波段操作、趨勢明確的股票
  • 不適合：盤整整理、趨勢不明確的股票

正金字塔 vs 倒金字塔

正金字塔（越跌買越多）：

  • 用於：價值投資、左侧交易
  • 特點：下跌時分批加碼，降低平均成本
  • 風險：可能「越攤越平」，公司體質變化時大虧
  • 適合：長期看好的穩健公司

倒金字塔（越漲買越多）：

  • 用於：短線交易、趨勢交易
  • 特點：上漲確認後才加碼，一開始試單
  • 風險：成本會越來越高，反轉時虧損放大
  • 適合：技術面突破、動能強的股票

怎麼選擇：

  • 信心程度高、向下風險有限 → 正金字塔（一開始買多）
  • 不確定性高、需要確認趨勢 → 倒金字塔（一開始試單）

李佛摩的加碼心法

核心原則：
1. 突破關鍵點再加碼：不要在股價尚未明確突破之前過度期待
2. 主要部位先建立：一開始用較小金額建立持股
3. 避免攤平損失：股價下滑時不要加碼，這是攤平不是加碼
4. 保留現金：有現金才能把握市場出現的好機會

心態管理：

  • 保持冷靜，不要因為過度期待而做出草率決定
  • 專注於市場分析，避免分心
  • 保持健康的生活習慣，減少壓力

金字塔投資法的陷阱

現金拖累問題：

  • 長期投資設定過大價格區間，可能大部分時間持有現金
  • 例如：等台股指數跌到 5000 點才買滿，結果長期在萬點以上
  • 解決：設定合理的價格區間，不要過大

錯誤示範：

  • 買了就跌、跌了就加碼 → 這是攤平，不是金字塔
  • 一開始就買滿，沒有保留加碼空間
  • 加碼點位不明確，憑感覺加碼

實戰建議

部位配置原則：

  • 單一標的金字塔比例是對該標的的投入比例，不是總資金
  • 例如：總資金 100 萬，預計投資 A 股 10 萬，金字塔的 40% 是 4 萬

進場前先想好：
1. 這筆投資總共要投入多少？（部位大小）
2. 分幾次進場？每次多少比例？
3. 加碼條件是什麼？（突破哪個價位或技術型態）
4. 停損點在哪裡？

風險控制：

  • 試單失敗停損，損失有限
  • 加碼後反轉，整體成本比一次買進高，但部位可控
  • 永遠保留現金，不要全倉

心得

李佛摩的金字塔加碼法跟一般想的「越跌買越多」完全相反。他的邏輯是：只有在賺錢的時候才加碼。

這跟「攤平」是天壤之別：

  • 攤平：虧錢時加碼，想降低成本，結果越陷越深
  • 金字塔加碼：賺錢時加碼，趨勢確認才擴大部位

這背後的邏輯是：市場告訴你對了，才下更大注。一開始試單就是承認自己可能看錯，所以用小部位測試。這是謙卑的操作方式。

市場先生的文章提醒了一個重點：金字塔投資法的百分比是「對單一標的的投入比例」，不是總資金。這點很多人會搞混。另外現金拖累是長期投資的大問題，如果價格區間設太大，可能大部分時間都在持有現金。
