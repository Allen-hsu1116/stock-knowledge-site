當沖選股三個必要條件

1. 流動性佳

是什麼：股票要能順利買進賣出，不會卡住

怎麼做：
1. 日成交金額 > 5,000萬（基本門檻）
2. 日成交量 > 2,000張（更實用的篩選標準）
3. 個人當沖金額不超過該股日成交金額的 1-2%（避免流動性風險）

實例：

  • 你要當沖 100萬 → 選日成交金額 > 2,000萬的股票
  • 你要當沖 1,000萬 → 選日成交金額 > 2億的股票

為什麼重要：

  • 量太小 → 價格不理想、賣不掉、容易被大戶操控
  • 量夠大 → 進出順暢、價格合理、行情真實

2. 波動性大

是什麼：股票當日高低點差距要夠大，才有價差可賺

怎麼做：
1. 計算日均波動：看當日振幅或近幾日平均高低點差距
2. 用 ATR（平均真實振幅）指標量化
3. 肉眼觀察：走勢圖上下起伏明顯 vs 一條線

實例：

  • 漲跌幅排行：查當日、昨日、30日內排行
  • 週轉率排行：週轉率越高 = 越熱門
  • 理想波動：日均 3% 以上（扣掉交易成本 0.3-0.4% 後還有空間）

警語：

  • 台積電、中華電信 → 波動小，不適合當沖
  • ETF → 分散風險，波動較個股小，也不理想

3. 有題材

是什麼：市場關注的焦點，能吸引短線資金進場

怎麼做：
1. 新聞題材：政府政策、公司利多
2. 產業趨勢：AI、電動車、半導體熱門族群
3. 財報公告：營收大增、重大合作

實例：

  • AI 熱潮 → 台積電動，伺服器、散熱、記憶體族群跟著放量
  • 找「有量、有波動、有題材」的標的

適用時機：

  • 大盤熱鬧 → 熱門族群波動放大
  • 大盤冷清 → 選股要更嚴格，避免「有量但不動」

當沖進場時機

進場三步驟

步驟一：判斷今天適不適合進場

怎麼做：
1. 觀察大盤/個股有沒有穩定的漲跌方向（有方向 = 市場有共識）
2. 檢查成交量有沒有放大（個股日成交量 > 2,000張）
3. 看有沒有明顯熱門題材或族群

判斷準則：

  • 方向明確 + 成交熱絡 + 資金集中 → 適合出手
  • 盤整無方向 + 成交清淡 → 今天不操作也沒關係

步驟二：三個進場圖形

圖形一：上升趨勢線

  • 是什麼：股價低點越來越高，連成一條上升線
  • 怎麼做：股價接近趨勢線時買進
  • 警訊：跌破趨勢線 → 可能反轉，考慮賣出

圖形二：壓力變支撐

  • 是什麼：股價突破壓力線後站穩，壓力線變成支撐線
  • 怎麼做：在靠近支撐線的價格進場
  • 為什麼有效：突破後壓力線變成市場共識的支撐

圖形三：假跌破

  • 是什麼：股價跌破支撐線，但很快拉回來
  • 怎麼做：拉回穩住後，在支撐線附近買進
  • 注意：要確認「穩住」才是真的轉強

步驟三：進場前的自我提問

問題一：我這次進場的理由？

  • 走勢亂七八糟、看不懂 → 不出手
  • 看不懂，就不出手

問題二：我設定好停損了嗎？

  • 進場前先想好「萬一錯了怎麼處理」
  • 停損點要具體（例如：跌 1% 就出場）

心法：

  • 不求買最低、賣最高
  • 抓到一段你看得懂的走勢，在合理位置進出

當沖出場策略

方法一：固定停損／停利比例

怎麼做：

  • 漲 2% 就賣出（停利）
  • 跌 1% 就出場（停損）

為什麼有效：

  • 避免猶豫不決、捨不得賣
  • 避免貪心硬撐

方法二：設定出場訊號

訊號一：突然爆大量卻沒再漲 → 主力可能出貨
訊號二：明顯長黑K → 可能轉弱，考慮減碼
訊號三：多頭走勢中忽然轉震盪 → 減碼觀望

方法三：時間管理

怎麼做：

  • 進場後 20 分鐘走勢沒變動 → 出場觀望
  • 最晚 13:25 一定要出場

最佳交易時間：

  • 開盤前半小時（9:00-9:30）：波動大，但難度高
  • 盤中（9:30-12:30）：型態明確，相對好判斷
  • 12:30 後：不建議入場（風險高）

當沖交易六大注意事項

注意一：確認資金上限

  • 千萬不要無本當沖
  • 單筆虧損控制在總資金 0.5% 以內
  • 單次投入不超過總資金 20%

注意二：專注少數股票

  • 新手最多 3 檔以內
  • 太多檔容易顧此失彼

注意三：設停利停損

  • 虧 10% → 需要獲利 11.1% 才能回本
  • 虧 50% → 需要獲利 100% 才能回本
  • 停損遠比賺錢重要

注意四：把握交易時間

  • 9:00-12:30 是最佳時段
  • 12:30 後不建議入場
  • 一天最多操作 3 趟

注意五：避免過度交易

  • 一天走勢最多 1-3 次轉折
  • 沒趨勢、沒波動 = 付出成本而已
  • 達到目標就收手，明天還有行情

注意六：了解期望值

  • 當沖要賺錢，勝率要 60-70% 以上
  • 大多數人賠錢賠時間
  • 心態正確才能長久

新手常見錯誤

錯誤一：沒有進出計畫

  • 問題：看到漲就追、跌了不停損
  • 解決：每一筆進場前先設定目標價與停損點

錯誤二：賺一點就跑，賠錢就跑不動

  • 問題：小賺大賠
  • 解決：紀律比賺多更重要

錯誤三：沒觀察股票習性

  • 問題：每檔股票漲跌節奏不同
  • 解決：專注幾檔熟悉的標的，熟悉它的節奏

錯誤四：選冷門股

  • 問題：量小、滑價嚴重、出場困難
  • 解決：成交量 > 2,000張

錯誤五：只看新聞標題

  • 問題：利多新聞不等於股價會動
  • 解決：題材要配合成交量

錯誤六：忽略波段位置

  • 問題：股票已漲一大段還追高
  • 解決：即使當沖也要看支撐壓力

進階選股技巧

技巧一：籌碼面

  • 三大法人連續買超 → 帶動市場關注
  • 當沖熱門標的 → 短線資金集中，波動度更高

技巧二：關鍵價位

  • 突破關鍵價位 → 帶動追價，急漲
  • 跌破關鍵價位 → 引發恐慌，急殺
  • 注意：支撐壓力、前高前低、整數關卡

技巧三：市場氛圍

  • 大盤強勢 → 熱門族群容易爆量上攻
  • 大盤疲弱 → 找「逆勢抗跌」或「題材獨強」的標的
  • 族群效應 → 整個產業上漲，龍頭股和二線股一起活絡

心得

這幾篇文章把當沖選股和進出場時機講得很清楚，核心就三件事：有量、有波動、有題材。

但最讓我印象深刻的是「交易成本」這件事。一買一賣成本 0.3-0.4%，如果股票日均波動只有 2%，扣掉成本後根本沒利潤空間。難怪市場先生說「日均波動 3% 以上才是理想標的」。

還有一點很關鍵：當沖不是每天都要操作。看不懂就不出手，這比硬上更重要。新手最常犯的錯就是賺小賠大，停損紀律比抓點位更重要。

最後，進場前問自己兩個問題：
1. 我這次進場的理由是什麼？
2. 我停損點設在哪裡？

這兩個問題沒想清楚就不要進場。
