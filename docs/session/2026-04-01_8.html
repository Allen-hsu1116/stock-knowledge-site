一、融資使用率風險判斷

融資使用率是什麼

融資使用率 = 實際融資餘額 ÷（流通股數 × 25%）

  • 融資上限是流通股數的 25%
  • 使用率超過 50%，等於 12.5% 流通股在融資
  • 使用率越高，斷頭風險越大

怎麼判斷融資風險

步驟一：看融資使用率

  • 0-20%：低風險，籌碼健康
  • 20-50%：中等風險，需留意
  • 50%以上：高風險，下跌10%就可能引發斷頭潮
  • 80%以上：極高風險，人踩人行情隨時可能發生

步驟二：計算斷頭價位

  • 維持率 = 股票市值 ÷ 融資借款
  • 斷頭線 = 借款 × 1.3（維持率 130%）
  • 下跌幅度 =（買進價 - 斷頭價）÷ 買進價

實例：

  • 情境：融資買進台積電 1000 元
  • 融資借款：60 萬（六成）
  • 自備款：40 萬（四成）
  • 斷頭價：60 萬 × 1.3 = 78 萬
  • 下跌幅度：（100-78）÷ 100 = 22%

融資使用率高的意義

多頭時：

  • 融資是上漲燃料，相同資金買到更多股票
  • 買盤力道更強

空頭時：

  • 引發死亡螺旋，融資戶被迫殺出
  • 下跌 10% 就可能引發逃難潮
  • 10% 融資股票開始賣 → 帶動現股投資人停損 → 人踩人

實戰應用

避開高融資使用率個股：

  • 大盤在高點時，優先避開使用率 >50% 的股票
  • 前十大高融資使用率股票要特別小心

判斷籌碼健康度：

  • 台積電融資使用率約 0.4%，非常健康
  • 使用率 <1% 表示籌碼穩定

二、董監持股比例分析

董監持股是什麼

董監持股比例 = 董事、監察人持有股數 ÷ 公司股本

  • 反映公司高層對企業前景的信心
  • 持股比例高 = 利益與公司綁在一起
  • 賣出需申報（每日賣出 10 張以上要申報）

怎麼用董監持股選股

步驟一：看持股比例高低

  • 持股高於法定標準 = 管理層有信心
  • 持股持續增加 = 正在佈局
  • 持股突然下降 = 可能是高點訊號

步驟二：看設質比例

  • 持股高 + 設質低 = 真正看好
  • 持股高 + 設質高 = 等於沒持股（股票拿去抵押了）

步驟三：搭配股價走勢

  • 董監增加 + 股價盤整半年 = 潛力股
  • 等帶量突破盤整區高點再進場

實戰應用

選股條件：
1. 董監持股連續 3 個月增加
2. 設質比例為 0
3. 股價盤整半年以上
4. 等帶量突破再進場

出場時機：

  • 日週轉率 >10% 時出場觀望
  • 不用等到董監持股下滑才賣

警訊判斷：

  • 營收好但董監開始賣 → 可能是高峰
  • 空頭格局董盤還加碼 → 高層有信心
