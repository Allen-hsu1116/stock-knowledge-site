操作策略：網格交易Grid Trading×剝頭皮交易Scalping

今天學兩種完全不同的操作策略：網格交易是「懶人神器」——設好參數就讓機器人幫你低買高賣；剝頭皮交易是「極限操作」——在數秒到數分鐘內頻繁進出，從微小波動中累積利潤。一個靠耐心，一個靠速度，但共同點是：都不需要預測市場方向。

一、網格交易（Grid Trading）

是什麼

網格交易是一種「將價格切成好幾等分，像網格一樣」的自動化交易策略。當標的價格到達網格的某一條線時，就根據規則買或賣，規律地進行低買高賣，賺取每一格之間的價差利潤。

核心公式：網格交易 = 區間設定 + 自動低買高賣 + 每格小利潤累積

怎麼做

Step 1：設定價格區間（最高價/最低價）

  • 根據歷史波動範圍，設定價格上下限
  • 價格超出區間 → 網格暫停運作
  • 價格低於最低價 → 滿手持倉，帳面虧損
  • 價格高於最高價 → 滿手現金，踏空上漲

實例：

  • 觀察某標的過去3個月價格在80-120元區間震盪
  • 設定最高價125元、最低價75元（留一些緩衝）

Step 2：設定網格數與間隔方式

  • 等差網格：每格間隔固定數字（如100、110、120、130）
  • 等比網格：每格間隔固定比例（如100、110、121、133）

Step 3：計算單網格利潤

核心原則：單網格利潤必須大於雙邊手續費！

  • 單網格利潤建議：0.5%左右
  • 手續費計算：一買一賣各收一次，若手續費0.1%，雙邊就是0.2%
  • 所以0.5%的單格利潤扣除0.2%手續費後，實賺0.3%
  • 如果單格利潤太低（低於雙邊手續費），每筆交易都在虧錢

Step 4：設定總投資額與觸發價格

  • 總投資額決定每格的資金分配
  • 格數越多 → 每格資金越少 → 資金使用率越低
  • 觸發價格：如果當前價格不在理想位置，可設定觸發價格讓網格延後啟動

Step 5：設定止盈止損

  • 止盈止盈價格：價格高於此價 → 網格自動終止（保護利潤）
  • 止損價格：價格低於此價 → 網格自動終止（避免更大虧損）

完整操作實例：

  • 標的：台積電，目前價格600元
  • 區間設定：最低500元、最高700元
  • 網格數：20格（等差）
  • 每格間隔：(700-500)/20 = 10元
  • 單格利潤：10/600 ≈ 1.67%（扣除0.3%手續費後實賺1.37%）
  • 總投資額：50萬元
  • 每格資金：50萬/20 = 2.5萬元 ≈ 0.42張
  • 運作：股價每跌10元買0.42張，每漲10元賣0.42張
  • 理想情境：若一個月來回震盪5次，利潤 = 5 × 10元 × 0.42張 × 1000股 = 21,000元

網格交易五大優點

#優點說明
1震盪行情中獲利市場70%時間在震盪，網格交易正好是震盪市的剋星
2避免情緒影響機器人自動執行，不會因為恐慌或貪婪偏離策略
324小時運作加密貨幣市場全年無休，網格可以不間斷套利
4分批買賣降風險資金分N等份，不會一次重壓
5設定簡單上下限+格數+金額，三個參數就能開始

網格交易五大缺點

#缺點說明
1資金使用率低格數越多，每格資金越少，大量資金閒置
2超出範圍暫停單邊趨勢超出區間就停擺，低於最低價滿手套牢
3單邊趨勢賺不到最多上漲時分批賣出，吃不到最大漲幅
4無法即時調整區間設好後不能改，想改只能關掉重開
5依賴交易平台需要有網格機器人的交易所

網格交易三大風險

1. 標的持續下跌

  • 網格會持續買入，平均成本下降但帳面虧損擴大
  • 解法：選擇不會歸零的優質標的、設定止損

2. 誤用網格交易

  • 明確趨勢向上 → 不該用網格（會提前賣出踏空）
  • 明確趨勢向下 → 不該用網格（會越買越套）
  • 解法：只在震盪市用網格，趨勢市用其他策略

3. 交易平台風險

  • 交易所倒閉、詐騙平台
  • 解法：選擇有監管的大型交易所

適用時機

  • ✅ 震盪行情（網格的天堂）
  • ✅ 不想盯盤的投資人
  • ✅ 容易受情緒影響的交易者
  • ✅ 加密貨幣市場（7×24小時運作）
  • ❌ 單邊趨勢明確的市場
  • ❌ 波動太小的標的（格利潤不夠付手續費）

二、剝頭皮交易（Scalping）

是什麼

剝頭皮交易（Scalping），又稱「搶帽子交易」，是一種在極短時間內頻繁進出市場、從微小價格波動中累積利潤的超短線交易策略。典型持倉時間從數秒到數分鐘，每次目標僅2-10個點差（pip）的利潤。

核心公式：剝頭皮 = 高頻率 × 小獲利 × 嚴格停損

關鍵特徵：

  • 不追求單筆大獲利，靠重複執行小型盈利交易累積收益
  • 降低敞口風險：在市場時間極短，減少不利事件衝擊的機會
  • 小幅波動比大幅波動更容易把握，發生頻率也更高

四大剝頭皮策略

策略一：趨勢剝頭皮（Trend Scalping）——最適合初學者

是什麼：在強勢趨勢中，只順著趨勢方向做極短線交易。

怎麼做：
1. 用EMA或MA確認趨勢方向（如200MA向上=偏多）
2. 等待價格短暫回調到EMA支撐位
3. 順勢進場做多
4. 目標：前高附近（5-10 pip）
5. 停損：3-5 pip

實例：

  • EUR/USD處於上升趨勢，200MA持續向上
  • 價格從1.1050回調到1.1030（EMA20支撐）
  • 在1.1031做多，停損1.1027（4 pip），停利1.1039（8 pip）
  • 風報比 2:1，持倉時間約3-5分鐘

策略二：支撐/阻力突破剝頭皮——需要Price Action經驗

是什麼：等待價格突破關鍵支撐或阻力位後，順向進場。

怎麼做：
1. 找出明確的支撐/阻力位
2. 等待價格突破（搭配成交量確認）
3. 突破確認後立即進場
4. 停損設在突破位以下3-5 pip
5. 停利5-10 pip

實例：

  • 某股在50元整數關卡整理3天
  • 帶量突破50元到50.5元
  • 在50.3元進場做多，停損49.8元（5 tick），停利51.3元（10 tick）
  • 持倉時間1-3分鐘

策略三：範圍剝頭皮（Range Scalping）——盤整市專用

是什麼：在橫盤整理區間內，於支撐位買進、阻力位賣出，反覆操作。

怎麼做：
1. 確認市場處於橫盤（MA持平、ADX<20）
2. 在支撐位附近買進
3. 在阻力位附近賣出
4. 停損設在支撐/阻力位外3-5 pip
5. 反覆操作直到區間被突破

實例：

  • EUR/USD在1.1000-1.1050區間震盪
  • 在1.1005買進，停損1.0995（10 pip），停利1.1045（40 pip）
  • 或更短線：1.1005買、1.1020賣（15 pip），反覆操作
  • 適合波動較小、流動性充足的時段

策略四：新聞剝頭皮——風險極高，新手勿碰

是什麼：利用重大經濟數據發布後的瞬間波動進場。

風險：

  • 滑點巨大（點差可能瞬間擴大10倍）
  • 需要極低延遲的執行環境
  • 方向可能一秒反轉
  • 不建議新手嘗試

OANDA三均線剝頭皮法（台股當沖適用）

是什麼：利用3條移動平均線形成的「帶狀通道」進行剝頭皮交易的策略，特別適合30分鐘K線。

設定：

  • SMA(40)：確認中期方向
  • 高價MA(2)：帶狀上緣（出場/賣出信號）
  • 低價MA(2)：帶狀下緣（進場/買入信號）

怎麼做（做多）：
1. 確認2日線帶狀在SMA(40)上方 → 趨勢向上，固定做多
2. K線觸碰低價MA(2) → 進場做多
3. K線觸碰高價MA(2) → 獲利平倉
4. 停損：K線觸碰SMA(40) → 趨勢可能反轉
5. 替代停損：1-2根K線內未獲利 → 不論損益平倉

實例：

  • 某股30分鐘圖，SMA(40)持續向上，帶狀在上方
  • K線回調觸碰低價MA → 進場
  • 1-2根K線後觸碰高價MA → 平倉獲利
  • 趨勢穩定時可在同一趨勢中重複多次

特點：

  • 不需考慮收盤價，目視觸碰即進場
  • 已有持倉時不再建第二個新倉
  • 趨勢持續時可反覆進場，一個趨勢多次套利

剝頭皮風險管理四鐵律

1. 嚴格設定停損

  • 每筆交易必須預先設定停損
  • 停損範圍：3-5 pip
  • 單筆最大虧損：不超過帳戶資金的0.5-1%

2. 每日最大虧損限制

  • 設定每日最大虧損金額（如帳戶3%）
  • 達到即停止當日交易
  • 避免情緒性連敗毀掉帳戶

3. 風險報酬比管理

  • 即使風報比低（1:1至1:2），也要確保長期期望值為正
  • 不因追求更高勝率而放寬停損

4. 避免過度槓桿

  • 剝頭皮常用較大倉位+高槓桿
  • 連敗時追加保證金風險極高
  • 初學者從最小倉位開始

適用時機

  • ✅ 高波動+高流動性的時段（台股開盤9:00-9:30、歐美重疊時段）
  • ✅ 有時間全程盯盤的交易者
  • ✅ 紀律性極強、反應速度極快的人
  • ✅ 低點差的交易環境
  • ❌ 有正職工作的人
  • ❌ 容易情緒化交易的人
  • ❌ 重大新聞發布前後5分鐘
  • ❌ 點差高的冷門標的

組合應用：網格+剝頭皮的混合策略

概念：用網格交易處理大部分震盪行情，用剝頭皮交易處理極短線的機會。

操作方式：
1. 在震盪區間設定網格交易（自動低買高賣）
2. 當出現突發事件或明確突破時，手動用剝頭皮策略加碼
3. 網格處理「日常波動」，剝頭皮處理「異常波動」

實例：

  • 台積電在580-620元區間，設定網格自動套利
  • 某天突然傳出AI利多，開盤跳空到625元
  • 網格已到上限暫停，手動剝頭皮追多
  • 在626元進場，632元出場，6 pip利潤
  • 網格+剝頭皮兩頭賺

心得

網格交易和剝頭皮交易，一個是懶人策略，一個是拚命策略，但本質上都是「不預測方向，只利用波動」。

網格交易最大的價值在於把情緒從交易中移除。設好參數就不管它，讓機器人幫你低買高賣。在震盪市中，這可能是最輕鬆的獲利方式。但致命傷是單邊趨勢——跌破最低價滿手套牢，漲破最高價滿手現金踏空。所以網格交易的成敗，80%取決於選對區間。區間設太窄容易被突破，設太寬資金使用率太低，這中間的平衡很關鍵。

剝頭皮交易的本質是用速度換利潤。IG那篇文章說得很直白：研究顯示交易頻率越高的交易者只會越快虧錢。這不是嚇人，是統計事實。但確實有少數頂尖剝頭皮交易者能穩定獲利，他們的共同特徵是：鐵的紀律、極快的執行力、嚴格的每日虧損上限。多數人做不到這三點，所以剝頭皮對大多數人來說是賠錢策略。

OANDA的三均線剝頭皮法很有趣——用高價MA和低價MA形成帶狀通道，在通道下緣買、上緣賣，1-2根K線內完成交易。這比純粹盯著1分鐘K線亂槍打鳥有系統多了。核心是「只順趨勢方向做，不逆勢」，這一點跟之前學的「順大逆小」原則完全一致。

最後，兩種策略的組合應用是個有趣的思路。網格處理日常波動，剝頭皮處理突發機會，兩者互補。但關鍵還是那句話：先判斷市場狀態，再選擇策略。 震盪市用網格，趨勢初段用剝頭皮追，趨勢中段用移動停利跑，這才是完整的操作體系。

操作策略：交易系統設計完整框架×回測績效評估×前向優化WFO

前面八個回合學了各種操作策略——突破回調、均線進場、金字塔加碼、海龜交易、箱型理論、動能均值回歸、缺口交易、開盤八法、網格剝頭皮——但這些都只是「武器」。今天要學的是「武器系統」：怎麼把這些策略組裝成一套完整的交易系統，怎麼測試它到底能不能用，怎麼評估它的績效好壞。沒有系統的交易，就像拿著一堆好武器卻沒有作戰計畫，最後還是被打趴。

一、交易系統六層架構

是什麼

交易系統不是「一個指標」，是一套從選股到出場的完整決策框架。海龜交易法則的創始人理查德·丹尼斯說過：「交易不需要天賦，需要的是一套完整的規則加上鐵的紀律。」這套規則，就是交易系統。

六層架構

第一層：市場選擇（買賣什麼）

  • 選擇流動性充足的市場和標的
  • 台股：日均成交量>2000張、市值>100億
  • 期貨：主力合約、流動性充足的月份
  • 不碰冷門股、不碰消息面主導的投機股
  • 不同市場特性不同策略：股票用波段、期貨用順勢、選擇權用賣方

第二層：部位規模（買賣多少）

  • 單筆風險不超過總資金的1-2%（海龜用1%）
  • 公式：部位大小 = (總資金 × 風險%) ÷ (ATR × 停損倍數 × 每點價值)
  • 波動大的標的少買，波動小的標的多買
  • 相關性高的標的不能同時重倉（避免一損俱損）
  • 最大持倉不超過總資金的20-25%（單一標的）

第三層：進場訊號（何時買賣）

  • 明確的進場條件，不含主觀判斷
  • 可以是技術指標（MACD金叉、突破20日高點）、型態（W底突破頸線）、籌碼面（外資連3日買超）
  • 關鍵：進場訊號必須可以量化、可以回測、可以重複執行
  • 好的進場訊號：條件明確、觸發機率適中（太頻繁=雜訊多，太少=機會少）

第四層：停損出場（何時認賠）

  • 進場前就要設好停損，不是進場後才想
  • 七種停損方法：K線停損、趨勢線停損、均線停損、固定比例停損、支撐壓力停損、移動停損、ATR動態停損
  • 絕對停損五條：跌幅>10%、破盤整區、趨勢反轉、逆勢單反轉、情緒失控
  • 停損是保命符，不是認輸牌

第五層：停利出場（何時獲利了結）

  • 固定百分比停利（如目標20%）
  • 移動停利（從最高點回撤X%出場）
  • 技術停利（跌破MA20、MACD死叉）
  • 分批出場（50%+30%+20%三批法）
  • 讓利潤奔跑，不要過早停利

第六層：心態紀律（怎麼執行）

  • 交易日誌：買進理由、持有期間心情、賣出理由、月度檢討
  • 覆盤四步驟：進場理由是否充分→持有期間是否遵守紀律→出場是否果斷→事後看是否正確
  • 2%+6%規則：單筆虧損不超過2%，月虧損不超過6%
  • 虧損後強制冷靜：連續虧損3次暫停交易1天

怎麼做：交易計畫書模板

一份完整的交易計畫書應該包含以下內容：

1. 市場與時間框架

  • 交易市場：台股上市櫃
  • 時間框架：日線為主、60分線為輔
  • 操作週期：波段（持股5-30天）

2. 進場規則

  • 多頭排列（MA5>MA20>MA60）+ MACD金叉 + 成交量>5日均量1.5倍
  • 或：突破20日高點 + 回調不破突破點 + 量縮回調確認

3. 出場規則

  • 停損：進場價-2ATR 或 跌破MA20
  • 停利：目標價1:2風報比 或 移動停利8%

4. 部位管理

  • 單筆風險：1%
  • 最大持倉：5檔
  • 相關性：同類股不超過2檔

5. 例外規則

  • 重大消息前減倉50%
  • VIX>30暫停新倉位
  • 連虧3次暫停1天

6. 覆盤機制

  • 每週覆盤一次
  • 每月計算績效指標
  • 連續虧損超過6%暫停交易檢討

二、回測績效評估：七大量化指標

是什麼

回測是檢驗交易系統是否有效的關鍵步驟。但回測績效好不代表未來一定好——你需要知道怎麼評估回測結果，以及回測的陷阱在哪裡。

七大核心指標

1. Sharpe Ratio（夏普比率）——每單位風險的報酬

公式：Sharpe = (年化報酬率 - 無風險利率) ÷ 年化標準差

判讀：

  • <1：差（風險調整後報酬不足）
  • 1-2：好（大多數基金在此區間）
  • >2：優秀（每承受1%風險，獲得2%以上超額報酬）

實例：

  • 策略A：年化報酬25%，標準差30%，無風險利率3%
  • 策略B：年化報酬18%，標準差12%，無風險利率3%
  • 策略B雖然報酬較低，但每單位風險的報酬遠高於策略A

陷阱：Sharpe同時計算上行和下行波動，但投資人只在意下行風險。高報酬+高波動和低報酬+低波動可能得到相同的Sharpe，但持有體驗完全不同。

2. Sortino Ratio（索提諾比率）——只看下行風險

公式：Sortino = (年化報酬率 - 無風險利率) ÷ 下行標準差

判讀：

  • <1：風險過高或回報不足
  • 1-2：良好
  • >2：非常優秀

與Sharpe的差異：

  • Sharpe分母用全部標準差（上行+下行）
  • Sortino分母只用下行標準差（只計算負報酬的波動）
  • 投資人不在意上行波動（漲越多越好），只在意下行風險
  • Sortino比Sharpe更貼近投資人真實感受

實例：

  • 策略年化報酬15%，無風險利率3%，總標準差20%，下行標準差12%
  • Sharpe = (15%-3%) ÷ 20% = 0.60
  • Sortino = (15%-3%) ÷ 12% = 1.00
  • Sortino比Sharpe高，代表上行波動大但下行風險可控——這是好的策略特徵

3. Calmar Ratio（卡爾馬比率）——年化報酬vs最大回撤

公式：Calmar = 年化報酬率 ÷ 最大回撤（取絕對值）

判讀：

  • <0.2：差（回報不足以補償回撤風險）
  • 0.2-0.5：合理
  • >0.5：優秀（每承受1%最大回撤，獲得0.5%以上年化報酬）

實例：

  • 策略年化報酬20%，最大回撤30%
  • Calmar = 20% ÷ 30% = 0.67 → 優秀
  • 另一策略年化報酬10%，最大回撤15%
  • Calmar = 10% ÷ 15% = 0.67 → 同樣優秀，但風險更低

為什麼重要：MDD是投資人最痛苦的指標。跌50%需要漲100%回本。Calmar直接告訴你「每承受1%的回撤痛苦，能換多少報酬」。

4. Profit Factor（盈利因子）——總獲利÷總虧損

公式：Profit Factor = 所有獲利交易的總利潤 ÷ 所有虧損交易的總虧損

判讀：

  • <1：虧損策略（不具投資價值）
  • 1-1.5：勉強獲利（風險控制需優化）
  • 1.5-2：穩健策略
  • >2：非常吸引人

實例：

  • 100筆交易：60筆獲利共賺120萬，40筆虧損共賠60萬
  • Profit Factor = 120 ÷ 60 = 2.0 → 非常好
  • 如果獲利50筆共賺80萬，虧損50筆共賠60萬
  • Profit Factor = 80 ÷ 60 = 1.33 → 勉強獲利，需要改善

5. 最大回撤 MDD（Maximum Drawdown）——最痛苦的跌幅

公式：MDD = (波峰 - 波谷) ÷ 波峰

判讀：

  • <10%：保守型策略
  • 10-20%：穩健型策略
  • 20-30%：積極型策略
  • >30%：高風險策略

6. Win Rate（勝率）× Avg Win/Avg Loss（平均賺賠比）

期望值公式：期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損)

兩種獲利模式：

  • 高勝率+低賠比：如剝頭皮交易，勝率70%×平均賺0.5% - 敗率30%×平均賠1% = 0.05%
  • 低勝率+高賠比：如海龜交易，勝率30%×平均賺6% - 敗率70%×平均賠2% = 0.4%

關鍵洞察：勝率不重要，期望值才重要。一個勝率30%的系統如果賺賠比夠高，可以比勝率70%的系統更賺錢。

7. Tail Ratio（尾部比率）——極端情況的韌性

公式：Tail Ratio = 95百分位報酬 ÷ 5百分位報酬（取絕對值）

判讀：

  • >1：良好（極端獲利大於極端虧損）
  • ≈1：平衡（上行和下行風險相當）
  • <1：危險（極端虧損大於極端獲利）

實例：

  • 策略95百分位報酬+8%，5百分位報酬-4%
  • Tail Ratio = 8 ÷ 4 = 2.0 → 良好
  • 代表最好的一天賺的是最差一天賠的2倍

三、回測陷阱與品質門檻

七大量化投資陷阱

1. 倖存者偏差（Survivorship Bias）

  • 只看現在還存在的標的，忽略已下市的
  • 回測時只看0050成分股現在的成員，忽略被剔除的
  • 解法：使用包含下市股的完整資料庫

2. 前瞻偏差（Look-Ahead Bias）

  • 用了未來才知道的資訊做回測
  • 例如：用財報公布日的收盤價交易，但財報是盤後才公布的
  • 解法：信號T日產生，T+1日才能執行（shift(1)）

3. 故事偏差（Storytelling Bias）

  • 事後找到理由解釋歷史走勢
  • 看到結果再編故事，但未來不會重演同一個故事
  • 解法：先有假設再回測，不是先看結果再找理由

4. 過度配適（Overfitting）

  • 參數調到完美契合歷史資料，但對未來完全無效
  • 例如：最佳參數是MA17.3天，四捨五入到MA17——這就是過度配適
  • 解法：Bonferroni校正（多重檢定調整）、參數高原選擇

5. 交易成本（Transaction Cost）

  • 回測忽略手續費、滑價、稅金
  • 台股：證交稅0.3% + 手續費0.1425% = 單邊0.4425%
  • 當沖：證交稅減半0.15% + 手續費0.1425% = 單邊0.2925%
  • 解法：回測時加0.5%交易成本（保守估計）

6. 極端值（Extreme Values）

  • 少數極端獲利交易拉高整體績效
  • 例如：10年回測中2次暴賺覆蓋98次小虧
  • 解法：去除最極端5%的交易，看績效是否依然穩定

7. 做空成本（Short Cost）

  • 回測假設做空沒有成本，但現實有融券利息、強制回補風險
  • 解法：做空策略加上2-5%年化借券成本

回測品質門檻（Quality Gate）

五層20項檢查清單：

第一層：資料品質

  • [ ] 資料是否包含下市股
  • [ ] 是否有前瞻偏差
  • [ ] 資料是否經過除權息調整
  • [ ] 是否包含完整交易成本

第二層：策略邏輯

  • [ ] 進場條件是否可量化
  • [ ] 出場條件是否明確
  • [ ] 停損是否設在進場前
  • [ ] 是否有合理的參數範圍

第三層：績效指標

  • [ ] Sharpe > 1（扣除交易成本後）
  • [ ] MDD < 25%
  • [ ] Profit Factor > 1.5
  • [ ] 交易次數 > 100（統計顯著性）

第四層：穩健性

  • [ ] 參數微調±20%績效是否穩定（參數高原測試）
  • [ ] 不同市場環境是否都能獲利
  • [ ] 去除極端5%交易後績效是否穩定
  • [ ] 是否通過Walk-Forward驗證

第五層：實盤可行性

  • [ ] 滑價估計是否合理
  • [ ] 流動性是否足夠（日均量>策略需求量10倍）
  • [ ] 策略容量是否超過市場承受範圍
  • [ ] 是否有合理的執行延遲

四、前向優化（Walk-Forward Optimization, WFO）

是什麼

前向優化是回測的進階版，解決「過度配適」這個最致命的問題。傳統回測在全部歷史資料上找最佳參數，WFO則是模擬真實交易：用過去資料優化參數，用未來資料驗證績效，然後滾動前進。

為什麼傳統回測不可靠

傳統回測的問題：
1. 在2010-2024年的資料上找最佳參數
2. 用同一段資料驗證（自己考自己）
3. 最佳參數可能完美契合歷史噪音，對未來完全無效
4. 看到Sharpe 3.0的回測結果就以為找到了聖杯——結果實盤Sharpe只有0.5

WFO的解法：

  • 用2010-2015年的資料優化參數
  • 用2016年的資料驗證績效（這是模型從未看過的資料）
  • 然後滾動到2011-2016優化、2017驗證
  • 重複直到2024年
  • 把所有「驗證期」績效拼起來，才是真實績效

怎麼做

Step 1：設定滾動窗口

  • 訓練窗口（In-Sample）：5年
  • 驗證窗口（Out-of-Sample）：1年
  • 滾動步長：1年

Step 2：執行WFO

  • 訓練期2010-2014 → 驗證期2015
  • 訓練期2011-2015 → 驗證期2016
  • 訓練期2012-2016 → 驗證期2017
  • ...
  • 訓練期2019-2023 → 驗證期2024

Step 3：計算WFO效率（WFE）

  • WFE = 驗證期Sharpe ÷ 訓練期Sharpe
  • WFE > 0.5：策略穩健，可以實盤
  • WFE 0.3-0.5：策略尚可，需要改善
  • WFE < 0.3：策略過度配適，不能用

Step 4：參數高原選擇

  • 不選最高峰的參數（過度配適風險最高）
  • 選擇高原區的參數（周圍參數績效也穩定）
  • 例如：MA17.3天是最高峰，但MA15-25天績效都差不多 → 選MA20（高原中心，更穩健）

WFO的限制

1. 窗口選擇偏差：訓練窗口太短抓不到完整市場週期，太長包含過時資料
2. 市場環境變化：WFO對環境劇變（如金融海嘯）反應滯後
3. 計算量大：比傳統回測慢數十倍
4. 仍不能保證未來：只是比傳統回測更可靠，不是水晶球

回測到實盤的衰減

實盤績效通常比回測差30-50%，原因：

  • 滑價：回測用收盤價，實盤有滑價
  • 流動性：回測假設無限流動性，實盤有限
  • 心理因素：回測不會恐懼貪婪，實盤會
  • 環境變化：市場微結構持續變化

漸進式部署：
1. 紙上交易1-3個月（驗證邏輯）
2. 小部位實盤3-6個月（驗證執行）
3. 逐步放大部位（每2週增加10-20%）
4. 全部位運行（持續監控WFE）

五、系統交易vs主觀交易

系統交易（Systematic Trading）

定義：所有進出場依照固定規則執行，不需要臨場判斷

優點：

  • 紀律性強，避免情緒干擾
  • 可回測驗證，量化績效
  • 一致性高，同樣條件同樣決策
  • 可同時監控多市場多標的

缺點：

  • 特殊行情下可能僵化
  • 需要程式能力或量化工具
  • 前期投入時間多

主觀交易（Discretionary Trading）

定義：依靠交易者經驗、判斷力和盤感做決策

優點：

  • 彈性大，臨機應變
  • 門檻低，有交易平台就能開始
  • 能處理系統無法預見的特殊情況

缺點：

  • 容易受情緒影響
  • 難以回測驗證
  • 紀律難維持
  • 績效依賴個人狀態

最佳實踐：混合模式

很多專業交易者採用混合模式：

  • 核心部位用系統交易（佔70-80%資金）
  • 衛星部位用主觀交易（佔20-30%資金）
  • 系統交易做趨勢追蹤和波段操作
  • 主觀交易處理特殊機會和系統無法捕捉的行情

心得

今天學的主題是「交易系統的系統化」——把前面學的零散策略，組裝成一套可以重複執行的完整框架。

六層架構最核心的概念是：市場選擇、部位規模、進場訊號、停損出場、停利出場、心態紀律，缺一不可。很多人只研究進場訊號（什麼指標交叉、什麼型態突破），卻忽略了部位管理和出場策略，結果就是「會買不會賣，買對也賠錢」。進場只佔整個系統的1/6，剩下的5/6才是決定長期能不能活下來的關鍵。

回測績效評估七大指標給了我一個量化框架。以前看策略績效只看「報酬率」，現在看的是Sharpe、Sortino、Calmar、Profit Factor、MDD、勝率×賠率、Tail Ratio這七個維度。最反直覺的是勝率不重要——海龜交易勝率只有20-40%，但靠高賠率照樣賺大錢。一個勝率70%但賠率0.5的策略，期望值可能比勝率30%但賠率3的策略還差。

前向優化（WFO）是最讓我震撼的概念。以前覺得回測績效好就等於策略好，但WFO告訴我：在歷史資料上找到的最佳參數，很可能是過度配適的產物。 真正可靠的策略，不是最高峰的參數，而是高原區的參數——周圍參數績效都差不多，代表策略對參數不敏感，這才是真正穩健的。WFE > 0.5才是實盤可用的門檻。

最後，回測到實盤30-50%的衰減是殘酷的事實。回測Sharpe 2.0，實盤可能只有1.0。所以回測Sharpe不到1.5的策略，根本不用考慮實盤。這個數字讓我對「紙上談兵」有了更清醒的認識。

一、程式交易系統化：讓策略自己跑

是什麼

程式交易就是「讓程式去自動執行你的投資想法」。市場先生的定義最直白：你想的事情會賺錢，電腦就幫你賺；你想的事情會賠錢，電腦就幫你賠，而且還是穩定的賠。

核心公式：程式交易 = 策略邏輯程式化 + 回測驗證 + 自動執行

優勢1：克服人性弱點（最大價值）

  • 人會恐懼、會貪婪、會猶豫、會拖延停損、會過早停利
  • 程式不會——該買就買、該賣就賣、該停損就停損

優勢2：24小時全自動 + 同時監控大量數據

  • 不需要盯盤，程式幫你看
  • 可以同時監控數十到數百檔股票

優勢3：回測驗證

  • 同樣的策略，人腦只能憑感覺說「好像可以」

二、交易心理五大真相：馬克·道格拉斯的核心框架

是什麼

馬克·道格拉斯（Mark Douglas）是全球最知名的交易心理輔導師，他的兩本書《交易心理分析》和《紀律的交易者》被奉為交易心理學聖經。Tom Basso那句名言最經典：「如果一切重來，我會把順序完全顛倒——投資心理最重要，其次是風險管控，最後才是選什麼市場買賣。」

五大基本事實（The Five Fundamental Truths）

事實1：任何事情都可能發生

  • 市場的下一刻是漲是跌，沒有人能100%確定
  • 即使所有指標都指向看漲，下一秒也可能暴跌
  • 接受不確定性，是交易心理的起點
  • 實例：2020年2月美股歷史新高，所有指標偏多，3月就崩了33%

事實2：要賺錢不必知道下一步會有什麼變化

  • 你不需要預測市場才能賺錢
  • 你只需要在機率對你有利的時候下注
  • 海龜交易勝率只有20-40%，照樣年化80%
  • 關鍵：期望值為正，不是預測正確

事實3：定義優勢的任何一組變數產生的盈虧都是隨機分布的

  • 同樣的進場訊號，可能這次賺、下次賠
  • 連續虧損不代表策略失效，連續獲利也不代表策略無敵
  • 這就是為什麼需要至少100筆交易才能統計顯著
  • 實例：勝率60%的策略，連續虧損5次的機率是1.2%——看起來低，但長期交易一定會遇到

事實4：優勢只不過是顯示一件事情發生的概率高於另一件事情

  • 所謂的「優勢」不是穩贏，只是機率稍高
  • MACD金叉做多，可能勝率55%，但這55%就是你相對於50%的優勢
  • 不要把「優勢」當「確定」
  • 5%的機率優勢，長期複利下來就是巨大財富

事實5：市場中的每一刻都具有獨一無二的性質

  • 今天的盤勢和昨天不一樣，即使K線長得一樣
  • 歷史會重複，但不會完全相同
  • 不要用昨天的經驗套用今天
  • 每次交易都是全新的開始

七大紀律原則（The Seven Principles of Consistency）

原則1：我客觀地看出自己的優勢

  • 不主觀臆測，用數據和回測確認優勢
  • 我的優勢是什麼？勝率多少？賺賠比多少？期望值為正嗎？

原則2：我事先界定每一筆交易的風險

  • 進場前就設好停損，不是進場後才想
  • 單筆風險1-2%，這是鐵律

原則3：我完全接受風險，並願意放棄交易

  • 如果不能接受停損，就不要進場
  • 停損是成本，不是失敗
  • 接受風險 = 進場後不再猶豫

原則4：我根據自己的優勢行動，毫無保留或猶豫

  • 訊號出現就進場，不猶豫、不等待「更好的價格」
  • 猶豫的本質是不接受風險
  • 程式交易就是這個原則的終極實現

原則5：市場讓我賺錢時，我會酬謝自己

  • 獲利了結後，給自己獎勵
  • 正向強化紀律行為
  • 不要「賺了又想更多」，最後吐回去

原則6：我不斷檢視自己犯錯的可能性

  • 我可能是錯的，市場是對的
  • 檢視自己的偏見、盲點
  • 定期覆盤，每週至少一次

原則7：我了解這些長期成功原則絕對必要，因此從來沒有違反過

  • 紀律不是有時候遵守，是永遠遵守
  • 一次違反就可能造成致命虧損
  • 長期成功的基礎是100%的紀律，不是99%

交易心理的四大敵人

敵人1：不願意制定規則

  • 散戶最常見的問題：不想被規則綁住
  • 沒有規則 = 隨機交易 = 長期必輸
  • 解法：寫下交易計畫書，白紙黑字

敵人2：不能負責

  • 虧損時怪市場、怪主力、怪政府
  • 不承認虧損是自己的決策造成的
  • 解法：交易日誌記錄每一筆決策

敵人3：沉迷於隨機的報酬

  • 偶爾大賺就以為自己找到了聖杯
  • 隨機報酬造成的自信是最危險的
  • 解法：看期望值，不看單筆盈虧

敵人4：外在控制與內在控制

  • 以為可以控制市場（外在控制錯覺）
  • 或者完全放棄控制（內在控制無力）
  • 解法：承認只能控制自己（部位、停損、紀律），不能控制市場

紀律系統化：把心理問題變成流程問題

核心概念：不要靠意志力守紀律，靠系統守紀律。

1. 交易前：預設條件清單

  • 大盤環境是否適合進場？
  • 進場條件是否全部滿足？
  • 停損價是否已設定？
  • 部位大小是否在風險限制內？
  • 沒有全部打勾 → 不進場

2. 交易中：執行規則

  • 程式交易自動執行（最佳方案）
  • 手動交易：OCO單鎖定停損停利
  • 不盯盤（盯盤會觸發情緒反應）

3. 交易後：覆盤流程

  • 進場理由是否充分？
  • 持有期間是否遵守紀律？
  • 出場是否果斷？
  • 事後看是否正確？
  • 紀律得分：1-10分自我評量

4. 每月檢討：系統性評估

  • 月度Sharpe、MDD、勝率
  • 紀律違反次數
  • 最大單筆虧損
  • 是否需要調整策略參數

三、程式交易×交易心理：終極組合

為什麼兩者缺一不可

程式交易解決的是「執行問題」：該買就買、該賣就賣、該停損就停損，不被情緒干擾。

交易心理解決的是「信念問題」：你必須真正相信「任何事情都可能發生」，才能真正接受停損；你必須真正相信「盈虧是隨機分布的」，才能在連續虧損時不修改策略。

兩者的關係：

  • 程式交易是紀律的「硬體」——它替你執行
  • 交易心理是紀律的「軟體」——它讓你相信並堅持
  • 沒有程式交易，你的心理信念撐不過連續虧損
  • 沒有交易心理，你會在程式虧損時手動關掉它

實戰組合SOP

Step 1：建立信念（交易心理）

  • 讀完《交易心理分析》和《紀律的交易者》
  • 寫下自己的五大基本事實認知
  • 確認自己真正接受「虧損是成本」

Step 2：量化策略（回測驗證）

  • 將策略邏輯程式化
  • 回測至少5年歷史資料
  • Walk-Forward Optimization確認穩健性
  • WFE > 0.5才考慮實盤

Step 3：自動執行（程式交易）

  • 串接券商API自動下單
  • 設定風控參數
  • 小部位試跑1-3個月
  • 確認執行無誤後逐步放大

Step 4：紀律維護（交易心理+程式交易）

  • 每週覆盤一次
  • 程式運行時不手動干預（除非風控觸發）
  • 連續虧損時不關掉程式，而是檢視WFE
  • 每月計算紀律得分

Step 5：持續優化

  • 每季重新跑WFO
  • 參數高原選擇，不選最高峰
  • 心理覆盤：是否有衝動干預程式？
  • 技術覆盤：策略邏輯是否需要更新？

心得

今天的兩個主題看似一硬一軟，但本質上是在解決同一個問題：怎麼確保你的策略真正被執行。

程式交易最大的價值不是「賺更多」，而是「克服人性」。市場先生那句話太精準了：你想的事情會賺錢，電腦就幫你賺；你想的事情會賠錢，電腦就幫你賠，而且還是穩定的賠。程式不會在該停損的時候猶豫，不會在該加碼的時候害怕，不會在該持有的時候過早停利。它就像一個沒有感情的執行者，只做你告訴它的事。

但程式交易有個致命前提：你必須在程式虧損時不手動關掉它。 這就是交易心理的戰場。馬克·道格拉斯的五大基本事實，每一條都在對抗人性：

1. 「任何事情都可能發生」——對抗確定性偏誤
2. 「不必知道下一步」——對抗預測強迫症
3. 「盈虧隨機分布」——對抗結果歸因錯誤
4. 「優勢只是機率」——對抗過度自信
5. 「每刻獨一無二」——對抗經驗主義陷阱

最震撼的是Tom Basso那句話：「如果一切重來，我會把順序完全顛倒。」多數人花了90%時間研究策略（選什麼市場），9%時間研究風控（怎麼管理風險），1%時間研究心理（怎麼執行）。但正確的順序應該反過來——心理最重要，風控其次，策略最後。

這跟之前學的海龜交易法則完全呼應：海龜四大原則的第一條就是「掌握優勢」，但最難執行的是第三條「堅定不移」。海龜的勝率只有20-40%，連續虧損10次是常態，如果沒有鐵的心理信念，你一定會在第七次虧損時放棄——然後錯過接下來的大行情。

最後，程式交易和交易心理的組合，本質上就是把心理問題變成流程問題。不要靠意志力守紀律——意志力是有限資源，用完就沒了。靠系統守紀律：程式自動執行、OCO單鎖定停損、覆盤流程強制檢討。紀律不是一種美德，是一套可執行的系統。

2. 強化紀律 — 寫下來的東西比腦袋裡的念頭可靠 100 倍
3. 可覆盤改進 — 有白紙黑字才知道自己是哪個環節出問題
4. 正循環 — 遵守計畫賺到錢 → 更相信計畫 → 更遵守紀律

真實教訓：TMNT Trader 分享，當天交易超過 5 筆就會爆走，所以設了「日交易上限 5 筆」的硬規則。這是用真金白銀堆出來的教訓。

二、交易計畫書七大模組

模組 1：自我定位

是什麼：先搞清楚自己是誰、適合什麼。

怎麼做：
1. 交易動機 — 寫下來：為什麼交易？賺錢？刺激？學習？
2. 可投入時間 — 每天能看盤多久？盤中還是只能盤後？
3. 風險承受度 — 單筆最大虧損佔總資金多少 %？連續虧損多少 % 要停手？
4. 性格特質 — 急性子適合短線，慢性子適合波段

實例：

  • 情境：白天有正職，只能看盤後資料
  • 操作：選擇波段操作策略，用日線週期，不碰當沖
  • 具體數字：單筆最大虧損 2%，連虧 8% 暫停一週

模組 2：市場選擇

是什麼：決定交易什麼商品、什麼市場。

怎麼做：
1. 商品類型 — 股票、期貨、選擇權、ETF？
2. 產業聚焦 — 科技股、傳產、金融？
3. 流動性篩選 — 日均量至少多少？（台股建議 > 1000 張）
4. 波動度匹配 — 波動太大受不了？波動太小沒利潤？

實例：

  • 情境：資金 50 萬，不想做期貨
  • 操作：專注台股中型股（市值 50-500 億），日均量 > 500 張
  • 具體數字：只做股價 30-300 元的股票，低於 30 元不碰（流動性差），高於 300 元不碰（買不了幾張）

模組 3：進場策略

是什麼：明確定義什麼條件下買進/放空。

怎麼做（把學過的策略選一套為主、一套為輔）：
1. 主要策略 — 例：均線糾結突破法（技術面）
2. 輔助策略 — 例：法人連買 3 日 + 融資減少（籌碼面）
3. 多重確認 — 至少 2 個維度同時指向同一方向才進場
4. 進場時機 — 什麼時間帶進場？開盤 15 分鐘不追、尾盤不追？

實例：

  • 情境：均線糾結 + 法人買超雙重確認
  • 操作：5/10/20/60MA 糾結後帶量突破 + 投信連買 3 日 + 千張大戶增加
  • 具體數字：量比 > 2（帶量突破）、投信買超 > 100 張、千張大戶比例週增 > 2%

模組 4：出場策略（最重要！）

是什麼：明確定義什麼條件下賣出/回補。出場比進場重要。

怎麼做（四層出場系統）：

第一層：初始停損

  • 進場同時就設好，不可事後調整
  • 方法：ATR 停損（2N）或支撐壓力停損
  • 例：進場價 100 元，ATR = 2 元，停損 = 100 - 2×2 = 96 元

第二層：移動停利

  • 方向正確後，停損線跟著移動
  • 方法：Chandelier Exit（前 5 日最高 - 3ATR）或跌破近 N 根 K 棒低點
  • 例：股價漲到 110，前 5 日高點 108，ATR = 2，移動停利 = 108 - 6 = 102

第三層：目標停利

  • 風報比達到目標即部分出場
  • 方法：分批出場 50% + 30% + 20%
  • 例：風報比 1:3 時出 50%，1:5 時出 30%，剩下 20% 用移動停利

第四層：時間出場

  • 進場後 N 根 K 棒沒動靜就出場
  • 方法：BarsSinceEntry > 5 且未獲利 → 損益兩平出場
  • 例：進場 5 天了還在成本附近晃 → 掛進場價停損單保護

實例：

  • 情境：100 元買進，ATR = 2
  • 初始停損：96 元（2ATR）
  • 移動停利：股價到 108 後，移動停利 = 102 元
  • 目標停利：112 元出 50%（風報比 3:1），120 元出 30%（風報比 5:1）
  • 時間出場：5 天沒動 → 100 元損益兩平出場

模組 5：部位控制

是什麼：每次買多少？最多同時持有幾個部位？

怎麼做：
1. 單筆風險 — 每筆交易最大虧損不超過總資金的 1-2%
2. 總部位上限 — 同時持有不超過 N 個部位（建議 3-5 個）
3. 同產業上限 — 同產業不超過總部位的 30%（分散風險）
4. 加碼規則 — 只在賺錢時加碼（李佛摩原則），不加攤平

部位計算公式：

買進張數 = (總資金 × 單筆風險%) ÷ (進場價 - 停損價) ÷ 1000

實例：

  • 情境：總資金 100 萬，單筆風險 2%，進場 100 元，停損 96 元
  • 計算：1,000,000 × 2% ÷ (100 - 96) ÷ 1000 = 5 張
  • 具體數字：最多買 5 張，虧損上限 20,000 元（2 萬）

模組 6：交易紀律清單

是什麼：進場前、持倉中、出場後各有一套檢查流程。

怎麼做：

進場前檢查（5 項全過才進場）：
1. ✅ 符合進場策略條件？
2. ✅ 停損價設好了？
3. ✅ 部位大小算好了？
4. ✅ 風報比 ≥ 1.5？
5. ✅ 今天已交易次數 < 上限？

持倉中檢查（每天盤後）：
1. ✅ 停損/停利條件有變嗎？
2. ✅ 需要移動停利嗎？
3. ✅ 基本面/籌碼面有變嗎？
4. ✅ 情緒狀態正常嗎？（生氣、興奮時不交易）

出場後檢查（每筆交易必做）：
1. ✅ 有遵守計畫嗎？
2. ✅ 如果沒遵守，原因是什麼？
3. ✅ 這筆交易學到什麼？

模組 7：覆盤系統

是什麼：交易日誌 + 定期檢討，持續改進。

怎麼做：
1. 交易日誌 — 每筆交易記錄：進場理由、出場理由、盈虧、情緒狀態
2. 週檢討 — 每週回顧：勝率、平均盈虧比、最大回撤
3. 月檢討 — 每月檢視：策略是否需要調整？哪個模組出問題？
4. 年度健檢 — 每年大盤點：整體報酬率 vs 大盤、Sharpe 值、最大回撤

交易日誌七個必填欄位（之前學過的）：
1. 進場日期/時間
2. 進場理由（策略觸發條件）
3. 進場價/停損價/目標價
4. 出場日期/時間
5. 出場理由（停損/停利/時間出場/其他）
6. 盈虧金額與%
7. 情緒狀態與教訓

反轉交易策略（Reversal Trading）實戰全攻略

一、反轉策略核心概念

是什麼：識別市場現行趨勢即將結束、新趨勢即將開始，在轉折點附近進場的交易方法。與順勢交易不同，反轉策略是「逆勢而為」——在最高點附近賣出，在最低點附近買入。

反轉 vs 回調——搞錯會死：

特徵反轉 (Reversal)回調 (Retracement)
定義主要趨勢方向的根本改變主趨勢中的暫時反向修正
持續時間長期（數週到數月）短期（數天）
影響開啟全新行情原趨勢繼續前進
交易意涵平倉+建反向倉順勢加碼好時機

怎麼判斷：關鍵差異在「61.8%法則」——回調通常不超過前一波段的61.8%，超過就可能是反轉。

二、道氏理論判斷趨勢反轉

是什麼：用高價與低價的「上移/下移」來定義趨勢，是最樸素但最可靠的趨勢判斷法。

道氏趨勢定義：

  • 上漲趨勢：高價與低價持續上移
  • 下跌趨勢：高價與低價持續下移

反轉訊號：

  • 上漲→下跌：高價未再上移 + 低價下移（雙下移）
  • 下跌→上漲：低價未再下移 + 高價上移（雙上移）

怎麼做（三步判斷法）：
1. 黃色訊號：上漲趨勢中，最新高價未能超越前高 → 趨勢可能生變，但不急著動作
2. 確認訊號：最新低價跌破前低 → 高價下移+低價下移，反轉機率大增
3. 行動訊號：配合K線型態（頭肩、M頭等）跌破頸線 → 進場

實例：

  • 某股從50元漲到100元，創高後回檔到85元，反彈只到95元（未創高，黃色訊號），接著跌破85元（確認訊號），此時道氏理論判定上漲趨勢結束
  • 停損設在95元（反彈高點上方），風報比至少1:2

適用時機：日線以上的中長線趨勢判斷，短線雜訊太多不適用

三、頂部反轉四大K線型態

1. 頭肩頂（Head and Shoulders Top）

是什麼：三個山峰，中間最高，左右較低。最經典的頂部反轉型態。

怎麼做：
1. 識別左肩、頭部、右肩的形成
2. 畫頸線（連接兩個低點）
3. 等待跌破頸線進場做空
4. 目標價 = 頸線價位 − (頭部最高價 − 頸線價位)

實例：

  • 某股頭部在120元，頸線在100元，頭到頸線距離20元
  • 跌破100元進場做空，目標價80元（100−20），停損設105元
  • 風報比 20:5 = 4:1

2. 雙重頂（M頭 / Double Top）

是什麼：價格兩次上漲到幾乎相同高度後回落，上方壓力極大。

怎麼做：
1. 確認兩個頂部價格差異在3%以內
2. 跌破前次回調低點（頸線）進場
3. 目標價 = 頸線 − (頂部 − 頸線)

實例：

  • 某股兩次觸及80元後回落，頸線在70元
  • 跌破70元做空，目標價60元（70−10），停損73元

3. 黃昏之星（Evening Star）

是什麼：三根K線組合——長陽線 + 十字星/小實體 + 長陰線深入第一根陽線實體。

怎麼做：
1. 確認三根K線完整出現
2. 第三根陰線至少深入第一根陽線的50%
3. 下一根開盤即可進場做空
4. 停損設在十字星最高點上方

4. 空頭吞噬（Bearish Engulfing）

是什麼：一根大陰線完全包覆前一根陽線實體，賣方力量爆發。

怎麼做：
1. 確認陰線實體完全吞噬前一根陽線
2. 陰線量能要大於陽線（量能確認）
3. 下一根開盤做空
4. 停損設在陰線最高點上方1-2%

四、底部反轉四大K線型態

1. 頭肩底（Inverse Head and Shoulders）

是什麼：倒立的頭肩頂，三個谷底，中間最低。最強的底部反轉訊號。

怎麼做：
1. 識別左肩、頭部、右肩
2. 突破頸線進場做多
3. 目標價 = 頸線 + (頸線 − 頭部最低價)
4. 停損設在右肩低點下方

實例：

  • 某股頭部在40元，頸線在55元，距離15元
  • 突破55元做多，目標價70元（55+15），停損50元
  • 風報比 15:5 = 3:1

2. 雙重底（W底 / Double Bottom）

是什麼：價格兩次跌到幾乎相同低點後反彈，下方支撐極強。

怎麼做：
1. 確認兩個底部價格差異在3%以內
2. 突破前次反彈高點（頸線）進場
3. 目標價 = 頸線 + (頸線 − 底部價)

3. 早晨之星（Morning Star）

是什麼：長陰線 + 十字星/小實體 + 長陽線深入第一根陰線實體。黎明來臨。

怎麼做：
1. 三根K線完整出現
2. 第三根陽線至少深入第一根陰線的50%
3. 下一根開盤做多
4. 停損設在十字星最低點下方

4. 多頭吞噬（Bullish Engulfing）

是什麼：一根大陽線完全吞噬前一根陰線實體。買方力量強勢回歸。

怎麼做：
1. 確認陽線實體完全吞噬前一根陰線
2. 陽線量能要大於陰線
3. 下一根開盤做多
4. 停損設在陽線最低點下方1-2%

五、多指標交叉驗證——反轉策略的核心武器

單一訊號不可靠，多重確認才進場。

底部反轉三重驗證法

步驟：
1. K線訊號：低點附近出現多頭吞噬或W底
2. RSI背離：價格創新低，RSI未創新低（看漲背離）
3. 量能確認：反彈上漲時成交量明顯放大
4. 進場確認：突破頸線或突破短期MA（如5MA或10MA）
5. 停損設定：設在W底最低點下方

實例：

  • 某股從60元跌到35元，在35-38元形成W底
  • RSI從25反彈到30（底背離），反彈量能是下跌量的1.5倍
  • 突破42元頸線進場做多，停損34元，目標49元（42+7）
  • 三重確認風報比 7:8，勝率從30%提升到50%+

頂部反轉三重驗證法

步驟：
1. K線訊號：高點附近出現空頭吞噬或M頭
2. RSI背離：價格創新高，RSI未創新高（看跌背離）

操作策略：信用價差Credit Spread選擇權收租策略

選擇權有四個基本動作：買Call、賣Call、買Put、賣Put。裸賣方的問題是風險無限，但加上一腳保險，風險就變成可控。信用價差（Credit Spread）就是「收權利金同時買保險」的收租策略，是選擇權操作中最實用、最穩健的策略之一。今天要把四種價差策略拆乾淨，重點放在兩種信用價差——Bull Put Spread和Bear Call Spread。

一、選擇權價差策略全覽：四種組合一次搞懂

核心概念

選擇權價差策略就是在同一個標的、同一個到期日，對不同履約價的Call或Put進行一買一賣的組合。目的很簡單：用一腳消除另一腳的缺點。

  • 買方的缺點：時間價值衰減，勝率低
  • 賣方的缺點：風險無限（或極大）

價差策略 = 買方降低成本 + 賣方限制風險 = 可控利潤、可控風險、高勝率

四種價差策略對照

策略方向權利金主角使用時機類型
買權多頭價差 Bull Call Spread看漲淨支出（借方）買進Call預期小漲Debit Spread
賣權多頭價差 Bull Put Spread看不跌淨收入（貸方）賣出Put預期不跌/小漲Credit Spread
賣權空頭價差 Bear Put Spread看跌淨支出（借方）買進Put預期小跌Debit Spread
買權空頭價差 Bear Call Spread看不漲淨收入（貸方）賣出Call預期不漲/小跌Credit Spread

記憶口訣：

  • Credit Spread（貸方價差）：建倉時收錢 → 賺時間價值 → 勝率高 → 收租策略
  • Debit Spread（借方價差）：建倉時付錢 → 賺方向 → 勝率低 → 方向策略

二、Bull Put Spread（賣權多頭價差）：看不上漲的收租神器

是什麼

Bull Put Spread是信用價差中最受歡迎的策略。賣出一口較高履約價的Put，同時買入一口更低履約價的Put，到期日相同。 建倉時收到淨權利金，只要標的不跌到賣出的履約價，權利金全入袋。

白話翻譯：我賭股票不會跌破某個價位，如果跌破，虧損有上限。

怎麼做

建倉步驟：
1. 賣出較高履約價的Put（收權利金，這是主角）
2. 買入更低履約價的Put（付權利金，這是保險）
3. 淨收權利金 = 賣Put收的 - 買Put付的

損益計算：

  • 最大獲利 = 淨收權利金（標的收在賣出Put履約價以上，兩口都作廢）
  • 最大虧損 =（賣出履約價 - 買入履約價）× 50 - 淨收權利金（台指期每點50元）
  • 損平點 = 賣出Put履約價 - 淨收權利金點數

實例：

  • 台股指數17,500點，看好不會大跌
  • 賣出17,400 Put，收150點
  • 買入17,200 Put，付50點
  • 淨收權利金 = 150 - 50 = 100點（5,000元）
  • 最大獲利 = 100點 = 5,000元
  • 最大虧損 = (17,400 - 17,200) × 50 - 100 × 50 = 10,000 - 5,000 = 5,000元
  • 損平點 = 17,400 - 100 = 17,300點

情境分析：

  • 結算17,500以上：兩口Put都作廢，淨賺100點（5,000元）✅
  • 結算17,300：損平，不賺不賠
  • 結算17,200以下：最大虧損100點（5,000元）❌
  • 結算17,400-17,200之間：部分虧損，線性遞減

保證金計算

台指選擇權信用價差保證金 = 價差 × 50元

以上例：價差 = 17,400 - 17,200 = 200點，保證金 = 200 × 50 = 10,000元

但實際上，台灣期交所對信用價差有保證金優惠，保證金遠低於裸賣，這就是信用價差最大的優勢之一。

適用時機

  • ✅ 多頭市場或盤整市場，判斷不會跌破支撐
  • ✅ 想要穩定收權利金（時間價值站在你這邊）
  • ✅ 不想承擔裸賣的無限風險
  • ❌ 看大跌（裸買Put或Bear Put Spread更適合）
  • ❌ 波動率極低（權利金太少不值得收）

選擇履約價的原則

賣出Put履約價選擇：

  • 選在支撐位以下1-2個標準差的位置
  • 離現貨價越遠，勝率越高但權利金越少
  • 一般建議delta在-0.15到-0.30之間（約70-85%勝率）

買入Put履約價選擇：

  • 與賣出Put的價差決定最大虧損
  • 價差越大，保費越低但最大虧損越大
  • 價差越小，保費越高但最大虧損越小
  • 一般建議價差200-400點（台指期）

三、Bear Call Spread（買權空頭價差）：看不上漲的收租策略

是什麼

Bear Call Spread是看空或看不漲時的信用價差策略。賣出一口較低履約價的Call，同時買入一口更高履約價的Call，到期日相同。 建倉時收到淨權利金，只要標的不漲破賣出的履約價，權利金全入袋。

怎麼做

建倉步驟：
1. 賣出較低履約價的Call（收權利金，這是主角）
2. 買入更高履約價的Call（付權利金，這是保險）
3. 淨收權利金 = 賣Call收的 - 買Call付的

損益計算：

  • 最大獲利 = 淨收權利金（標的收在賣出Call履約價以下，兩口都作廢）
  • 最大虧損 =（買入履約價 - 賣出履約價）× 50 - 淨收權利金
  • 損平點 = 賣出Call履約價 + 淨收權利金點數

實例：

  • 台股指數17,500點，看不上漲

買入Call履約價選擇：

  • 價差200-400點為宜
  • 價差越大，最大虧損越大但保費越低
  • 價差越小，最大虧損越小但保費越高

四、Debit Spread（借方價差）：方向明確時的精打細算

Bull Call Spread（買權多頭價差）

是什麼：看漲但認為漲幅有限時，買進低履約價Call + 賣出高履約價Call，降低權利金成本。

建倉步驟：
1. 買進較低履約價的Call（付權利金，主角）
2. 賣出較高履約價的Call（收權利金，降低成本）
3. 淨支出 = 買Call付的 - 賣Call收的

損益計算：

  • 最大獲利 =（賣出履約價 - 買入履約價）× 50 - 淨支出
  • 最大虧損 = 淨支出（兩口Call都作廢時）
  • 損平點 = 買入Call履約價 + 淨支出點數

實例：

  • 台股指數17,500點，看好小漲
  • 買入17,400 Call，付180點
  • 賣出17,600 Call，收60點
  • 淨支出 = 180 - 60 = 120點（6,000元）
  • 最大獲利 = (17,600 - 17,400) × 50 - 120 × 50 = 10,000 - 6,000 = 4,000元
  • 最大虧損 = 120點 = 6,000元
  • 損平點 = 17,400 + 120 = 17,520點

與Credit Spread的差異：

  • 建倉時付錢（不是收錢），時間價值站在你對面
  • 方向性更強：必須漲到損平點以上才開始賺
  • 勝率較低（約35-45%），但賺賠比更好
  • 不需要保證金（因為是淨支出）

Bear Put Spread（賣權空頭價差）

是什麼：看跌但認為跌幅有限時，買進高履約價Put + 賣出低履約價Put，降低權利金成本。

建倉步驟：
1. 買進較高履約價的Put（付權利金，主角）
2. 賣出較低履約價的Put（收權利金，降低成本）
3. 淨支出 = 買Put付的 - 賣Put收的

損益計算：

  • 最大獲利 =（買入履約價 - 賣出履約價）× 50 - 淨支出
  • 最大虧損 = 淨支出
  • 損平點 = 買入Put履約價 - 淨支出點數

與Bear Call Spread的差異：

  • 建倉時付錢，時間價值衰減不利
  • 方向性更強：必須跌到損平點以下才開始賺
  • 勝率較低，但賺賠比更好

六、信用價差實戰五步SOP

Step 1：判斷大盤環境和趨勢

  • 多頭市場 → Bull Put Spread為主
  • 空頭市場 → Bear Call Spread為主
  • 盤整市場 → 兩邊都可以收租（Iron Condor）
  • 不確定 → 不做，等待明確訊號

Step 2：選擇到期日

  • 週選（5-7天到期）：權利金時間價值衰減最快，但容錯空間小
  • 雙週選（12-14天）：時間價值衰減適中，推薦初學者
  • 月選（30天左右）：容錯空間大，但時間價值衰減慢
  • 建議：建倉時選30-45天到期，這段時間theta衰減開始加速

Step 3：選擇履約價

  • Bull Put Spread：賣出Put選在支撐位下方1-2個標準差，delta約-0.15到-0.30
  • Bear Call Spread：賣出Call選在壓力位上方1-2個標準差，delta約0.15到0.30
  • 價差：200-400點（台指期），控制最大虧損在可承受範圍

Step 4：建倉和管理

  • 建倉後設定停損：虧損達到最大獲利的2倍就出場
  • 帳面獲利達50%以上可考慮提前平倉（鎖定利潤）
  • 距離到期7-10天，如果已賺70%以上，提前平倉（避免Gamma風險）

Step 5：Rolling策略（展延）

  • 如果標的接近賣出的履約價，不要等被穿價
  • 選項1：提前停損出場（虧損可控）
  • 選項2：Roll out（展延到更遠的到期日，延長時間讓標的回到安全區）
  • 選項3：Roll up/down（調整履約價，移到更遠的價外）
  • 重要：Rolling不是免費的，只是把虧損延後，必須嚴格執行停損

七、信用價差的五大陷阱

1. 裸賣的誘惑

  • 信用價差最大獲利是收到的權利金，裸賣看起來收更多
  • 但裸賣風險無限，一次黑天鵝就歸零
  • 千萬不要裸賣選擇權！ 加買一腳保險是鐵律

2. 末日效應（Gamma風險）

  • 到期前3-5天，如果標的在賣出履約價附近，Gamma急劇增大
  • 小幅波動就會造成巨大損益變化
  • 解法：距離到期7-10天就平倉，不要撐到最後一天

3. IV崩塌（Vega風險）

  • Credit Spread在IV高時建倉最划算（賣貴）
  • 但如果IV急劇下降（例如財報公布後），權利金快速縮水
  • 好消息：你收的權利金也縮水，但已經收到了，所以IV下降對你有利
  • 壞消息：如果你想提前平倉，收到的權利金變少，利潤空間縮小

4. 流動性風險

  • 冷門履約價的價差可能很大，買賣價差侵蝕利潤
  • 選擇成交量充足的履約價（台指期主要履約價）
  • 避免價外超過3個履約價的位置

5. 過度自信的勝率

  • Credit Spread勝率看似65-80%，但一次大虧損可能吃掉5-10次小獲利
  • 如果最大虧損是最大獲利的3倍（賺賠比1:3），勝率80%的期望值：
  • 解法：控制賺賠比在1:1以上，或勝率在70%以上且賺賠比1:1.5以上

心得

信用價差是選擇權操作中「最像房東收租」的策略——你收權利金，時間流逝幫你賺錢，不用預測方向，只要預測「不會到某個價位」。這比裸買方（勝率只有30-40%）舒服太多了。

但最核心的教訓是那篇文章說的：千萬不要裸賣選擇權！ 加買一腳保險的成本很低，但消除了無限風險。Bull Put Spread賣17,400 Put買17,200 Put，保險成本可能只有50點，但把最大虧損從「理論上無限」變成「最多200點減100點權利金」。這50點的保險費，是選擇權最值得花的錢。

Credit Spread和Debit Spread的選擇，本質上是「高勝率低賠比」vs「低勝率高賠比」的抉擇。這跟之前學的動能交易vs均值回歸是同一個邏輯——Credit Spread是均值回歸思維（大部分時間收租金），Debit Spread是動能思維（看對方向大賺）。我個人更傾向Credit Spread，因為時間站在你這邊，心理壓力小很多。

Rolling策略是信用價差的雙面刃——它可以延長時間讓你「等回來」，但也可能越滾越大。鐵律是：如果虧損超過最大獲利的2倍，不要Roll，直接停損出場。這跟之前學的海龜交易法則「2N停損」是同一個精神——虧損可控比獲利可期更重要。

最後，信用價差最適合的市場環境是盤整市——這正好是方向性策略（突破、趨勢追蹤）最痛苦的時候。市場70%的時間在盤整，Credit Spread就是為這70%設計的武器。搭配之前學的ADX判斷趨勢強度：ADX<20用Credit Spread收租，ADX>25用趨勢追蹤策略，這就是完整的操作框架。

價格行為交易（Price Action）：裸K交易的藝術與科學

所有技術指標的本質都是價格的「加工品」。移動平均線是價格的平均、RSI是價格的動能、MACD是價格的差值。那為什麼不直接看價格本身？Price Action（價格行為交易）就是這個問題的答案——拋開所有指標，只用裸K線圖解讀市場，直接和多空對話。

一、Price Action 核心哲學：價格是因，指標是果

為什麼裸K交易比指標交易更即時？

指標的延遲本質：所有技術指標都是用歷史價格計算出來的。MA需要N根K線、RSI需要14期、MACD需要12/26/9。當指標給出訊號時，價格已經走了一段。就像開車只看後照鏡——知道剛剛過了什麼，但看不到前面。

Price Action的領先性：K線型態是即時的。當一根Pin Bar在壓力位形成時，你立刻知道這裡有賣壓。不用等MA交叉、不用等RSI背離——價格本身已經告訴你了。

比喻記憶法

指標交易Price Action
本質加工食品生鮮食材
時效看後照鏡開車看擋風玻璃開車
圖表五顏六色線條乾淨的裸K圖
訊號滯後，可能錯過最佳點領先，直接反映市場情緒

二、Price Action 三大核心要素

要素一：市場結構（Market Structure）——判斷方向

市場結構是Price Action的骨架，告訴你「現在往哪裡走」。

三種市場結構：

1. 上升趨勢：Higher High（更高的高點）+ Higher Low（更高的低點）

  • 實例：台積電從500→550（HH）→520（HL）→600（HH）→560（HL）→680（HH）

2. 下降趨勢：Lower Low（更低的低點）+ Lower High（更低的高點）

  • 實例：某股從200→180（LL）→190（LH）→160（LL）→170（LH）→140（LL）

3. 盤整區間：價格在明確支撐壓力間來回

  • 操作：區間下緣買、上緣賣，或等待突破

結構轉換的關鍵訊號：

  • 上升趨勢轉空：價格跌破前一個Higher Low → 結構破壞
  • 下降趨勢轉多：價格突破前一個Lower High → 結構反轉
  • Equal High / Equal Low：當趨勢中出現「相等高點」或「相等低點」，代表動能開始減弱，是潛在反轉的早期預警

要素二：關鍵水平位（Key Levels）——尋找戰場

四種關鍵水平位：

1. 支撐位（Support）：價格下跌至此，買方進場阻止繼續下跌

  • 作用：像地板，價格踩到就彈起來

2. 壓力位（Resistance）：價格上漲至此，賣方進場阻止繼續上漲

  • 作用：像天花板，價格撞到就壓回去

3. 支撐壓力互換（Role Reversal）：壓力突破後變支撐、支撐跌破後變壓力

  • 這是Price Action交易者最愛的進場位置

4. 供需區（Supply & Demand Zones）：機構大單集中的區域

  • 供給區（Supply Zone）：大量賣單聚集的區域，價格快速下跌的起點

繪製關鍵水平位的實戰步驟：
1. 打開日線圖或4小時圖
2. 縮放到可以看到3-6個月的價格範圍
3. 標記所有價格曾反應過的水平（至少2次觸碰）
4. 合併相近的價格水平為區間
5. 觀察價格每次觸及這些水平的反應

要素三：K線型態（Candlestick Patterns）——確認訊號

K線型態是Price Action的「進場觸發器」，但要記住：型態必須出現在關鍵水平位才有意義。隨便出現的型態只是噪音。

三、六大Price Action交易訊號

1. Pin Bar（長影線／鎚子線）——最強反轉訊號

是什麼：一根K線，實體很小，但有一根非常長的影線（影線長度≥實體的2-3倍以上）。

怎麼看：

  • 看漲Pin Bar：長下影線出現在支撐位 → 買方強力反擊，賣壓被吸收
  • 看跌Pin Bar：長上影線出現在壓力位 → 賣方強力反擊，買壓被吸收

怎麼做：
1. 在關鍵支撐位等待看漲Pin Bar出現
2. 進場點：Pin Bar收盤價附近
3. 停損點：Pin Bar影線最低點下方2-3個tick
4. 停利點：下一個壓力位，或風險報酬比1:2以上

實例：

  • 情境：台積電回測500元支撐位，出現長下影線Pin Bar，影線從510到490（20元），實體505-510（5元）
  • 操作：510元進場，停損487元（影線下方3元），目標540元（前波高點附近）
  • 風險：23元，報酬：30元，風報比1:1.3（不理想，等更好的位置）

適用時機：趨勢回調到支撐/壓力位時最有效。趨勢盤中Pin Bar勝率最高，盤整盤中意義較小。

2. Inside Bar（內包線／孕線）——盤整後的突破

是什麼：後一根K線的高低點完全被前一根K線（母線）包住。

怎麼看：

  • 代表市場暫時猶豫，多空力量達到平衡
  • 像壓縮的彈簧，突破方向就是接下來的方向

怎麼做：
1. 在上升趨勢中出現Inside Bar → 等待向上突破母線高點做多
2. 在下降趨勢中出現Inside Bar → 等待向下突破母線低點做空
3. 進場點：突破母線高/低點
4. 停損點：母線的另一端
5. 停利點：風險報酬比1:2以上

實例：

  • 情境：聯發科在800元附近的上升趨勢中出現Inside Bar，母線790-810，內包線795-805
  • 操作：突破810元做多，停損790元，目標850元（前波高點）
  • 風險：20元，報酬：40元，風報比1:2

適用時機：強趨勢中的回調盤整，Inside Bar是趨勢延續的最佳入場點。盤整區間中出現Inside Bar方向不明確，避免操作。

3. Fakey Pattern（假突破）——機構誘多的陷阱反轉

是什麼：Inside Bar之後，先突破母線方向，但迅速反向收回。這是裸K交易最強大的訊號之一。

怎麼看：

  • 機構用假突破觸發散戶的停損/追單，然後反向操作
  • 看漲Fakey：向下假跌破後迅速收回 → 強烈做多訊號
  • 看跌Fakey：向上假突破後迅速收回 → 強烈做空訊號

怎麼做：
1. 識別Inside Bar
2. 等待價格突破Inside Bar方向
3. 觀察價格是否迅速收回Inside Bar範圍
4. 確認收回後進場，停損設在假突破的極端點

實例：

  • 情境：某股支撐位100元，出現Inside Bar（95-105），然後跌破95元到92元，但當天收回98元（在95元以上）
  • 操作：98元進場做多，停損92元（假突破最低點下方），目標115元
  • 風險：6元，報酬：17元，風報比1:2.8

適用時機：關鍵水平位出現的Fakey勝率極高，是最值得等待的訊號。但出現頻率低，需要耐心。

4. Engulfing Bar（吞噬型態）——動能反轉

是什麼：第二根K線的實體完全包住前一根K線的實體，且顏色相反。

怎麼看：

  • 看漲吞噬：下跌趨勢中，大陽線包住小陰線 → 買方力量壓倒性反擊
  • 看跌吞噬：上升趨勢中，大陰線包住小陽線 → 賣方力量壓倒性反擊

怎麼做：
1. 在支撐位等待看漲吞噬
2. 進場點：吞噬K線收盤價附近
3. 停損點：吞噬K線的極端點（看漲吞噬的低點下方）
4. 停利點：風險報酬比1:2以上或下一個壓力位

實例：

  • 情境：鴻海在120元支撐位，前一根小陰線122-118，後一根大陽線128-117完全吞噬
  • 操作：128元進場，停損116元，目標148元（前波高點）
  • 風險：12元，報酬：20元，風報比1:1.7

適用時機：趨勢末段出現吞噬型態最有效。盤整盤中吞噬意義不大。

5. 2-Bar Reversal（雙K反轉）——溫和版吞噬

是什麼：第二根K線「遲到」第一根K線的實體部分（不需要完全吞噬），方向相反。

怎麼看：

  • 比吞噬型態更常見，但訊號稍弱
  • 看漲：陰線後出現陽線，陽線收盤深入陰線實體50%以上
  • 看跌：陽線後出現陰線，陰線收盤深入陽線實體50%以上

怎麼做：
1. 在關鍵水平位出現2-Bar Reversal
2. 進場點：第二根K線收盤價
3. 停損點：第二根K線極端點下方
4. 確認：搭配成交量放大更有意義

適用時機：趨勢中的回調結束處。比吞噬型態更頻繁出現，適合捕捉溫和反轉。

6. Hikkake（日文「引掛」＝陷阱）——假突破的另一種形式

是什麼：由2-3根K線組成的假突破型態。先突破Inside Bar方向，但隨後3根K線內反向突破。

怎麼看：

  • 第1根：母線
  • 第2根：Inside Bar
  • 第3根：嘗試突破Inside Bar（假突破方向）
  • 第4-6根：反向突破母線（真正方向）

怎麼做：
1. 識別Inside Bar
2. 等待第3根K線突破Inside Bar
3. 觀察接下來3根K線是否反向突破母線
4. 反向突破母線時進場

適用時機：低波動盤整後的突破。與Fakey相似但更正式的定義。

四、Price Action 交易流程：三步公式

最好的Price Action交易 = 趨勢（Trend）+ 位置（Level）+ 訊號（Signal）

Step 1：判斷市場結構（趨勢）

  • 上升趨勢：只做多
  • 下降趨勢：只做空
  • 盤整：等突破或做區間交易
  • 永遠順趨勢操作，不要在上升趨勢中做空

Step 2：等待價格來到關鍵水平位（位置）

  • 支撐位（做多）或壓力位（做空）
  • 支撐壓力互換區
  • 供需區
  • 50%回撤區（斐波那契0.5附近）

為什麼位置這麼重要？

  • 在隨機位置出現的Pin Bar可能只是噪音
  • 在支撐位出現的Pin Bar是高勝率訊號
  • 同一個訊號，位置決定勝率

Step 3：確認K線型態（訊號）

  • Pin Bar在支撐位 → 做多
  • Engulfing在壓力位 → 做空
  • Fakey在關鍵水平位 → 反向操作
  • Inside Bar在趨勢中 → 順勢突破

五、多時間框架分析：Price Action的時間維度

Price Action不是只看一個時間框架。空間（市場結構）+ 時間（多時間框架）= 完整的判斷。

三重時間框架法則

時間框架作用範例（波段交易）
大框架判斷趨勢方向週線/日線
中框架找關鍵水平位日線/4小時
小框架找進場訊號4小時/1小時

實戰流程：
1. 週線圖看大趨勢（上升/下降/盤整）
2. 日線圖畫支撐壓力位
3. 4小時圖等K線型態出現
4. 1小時圖精確進場

盤整區間的小框架機會

  • 日線看盤整，但在4小時框架中可能有小型上升趨勢
  • 小框架的交易者可以利用這些機會操作
  • 但要注意：小框架趨勢可能隨時被大框架的突破打破

六、Price Action vs 技術指標：不是對立，是互補

比較Price Action技術指標
訊號性質領先/即時滯後
圖表乾淨簡潔可能雜亂
主觀性較高，需經驗較低，客觀
學習曲線初期陡峭初期平緩
適用性所有市場通用不同市場差異大

最佳實踐：Price Action為主 + 1-2個指標為輔

  • 用Price Action判斷方向和進場時機
  • 用成交量確認突破有效性
  • 用200MA判斷長期趨勢方向
  • 用RSI確認超買超賣

這就是所謂的「共振」（Confluence）：多個訊號指向同一方向，勝率大幅提升。

七、Price Action 實戰五步驟

1. 清理圖表：移除所有指標，只留裸K線。強迫大腦專注價格本身
2. 辨識關鍵價位：在日線或4小時圖畫出支撐壓力區
3. 學習3-5個核心型態：Pin Bar、Engulfing、Inside Bar、Fakey，不需要更多
4. 大量回測：手動翻歷史圖表，找型態出現在關鍵位時的後續走勢
5. 小倉位實戰：模擬交易建立信心後，用最小真實倉位開始

八、停損停利：R倍數管理法

Price Action交易者常用R倍數（Risk Multiples）管理停損停利：

  • 1R = 進場到停損的距離（你的風險）
  • 停利目標：

移動停利規則：
1. 價格進入1R-2R區間 → 停損移到進場價（盈虧平衡）
2. 價格進入2R-3R區間 → 停損移到1R位置（鎖定1R利潤）
3. 每進入新的盈利區間 → 停損上移一個區間，始終保持1R風險

心得

學Price Action最有趣的地方是：它讓你重新用「價格本身」來看市場，而不是透過指標的濾鏡。就像學攝影的人最後都會從自動模式回到全手動——因為只有自己掌控光圈和快門，才能真正理解光線。

六種訊號裡面，我覺得Fakey是最值得深入研究的。它本質上是在讀懂「市場的陷阱」——當你看到假突破，你不是被騙的那個人，而是看到陷阱在哪裡的人。這跟主力出貨和洗盤的邏輯是一樣的：散戶被假突破騙進去，Price Action交易者看到假突破反手操作。

但說實話，Price Action最大的挑戰不是學不會，而是「太簡單以至於不信」。很多人學了之後還是忍不住加一堆指標，因為裸K圖看起來「太空了」。但越是乾淨的圖表，越能看清楚市場在說什麼。

最後一點：型態必須出現在關鍵水平位才有效。這句話值得寫在電腦前面。隨機出現的Pin Bar毫無意義，但支撐位的Pin Bar就是金礦。位置決定勝率，型態只是觸發器。

操作策略：配對交易Pairs Trading實戰策略

配對交易是量化交易中最經典的市場中立策略。不預測漲跌，只賺兩檔股票之間的價差收斂。聽起來簡單，但魔鬼全在「怎麼找配對」和「怎麼判斷收斂時機」。

一、配對交易核心邏輯

是什麼

配對交易（Pairs Trading）是一種統計套利策略：找兩檔歷史走勢高度相關的股票，當它們的價差偏離歷史常態時，做空相對偏貴的、做多相對偏便宜的，等價差回到均值時平倉獲利。

核心假設：相關股票的價差具有均值回歸特性——偏離終將收斂。

為什麼有效

1. 市場中立：大盤漲跌跟你無關，你只賺價差
2. 確定性高：不需要預測方向，只需要判斷「價差會回到正常範圍」
3. 勝率穩定：統計上均值回歸的勝率在60-70%
4. 風險可控：最大虧損就是價差繼續擴大，但同產業股票很難長期背離

跟一般交易的差異

維度一般交易配對交易
預測方向要預測漲或跌不預測方向
大盤風險有幾乎沒有
獲利來源股價變動價差收斂
最大風險單邊走勢反轉價差持續擴大（共整合關係破裂）
適合環境趨勢市震盪市

二、怎麼找配對：三步篩選法

Step 1：同產業配對（最基礎）

原則：選同產業、同業務性質的股票，它們受相同因素驅動。

台股經典配對：

  • 金融股：國泰金（2882）vs 富邦金（2881）
  • 鋼鐵股：中鋼（2002）vs 東和鋼鐵（2006）
  • 航運股：長榮（2603）vs 陽明（2609）
  • 紡織股：遠東新（1402）vs 新纖（1409）
  • 光學股：大立光（3008）vs 玉晶光（3406）

為什麼要同產業：

  • 營收受同樣的產業週期影響
  • 法人對同產業股票的評價邏輯一致
  • 突發事件對同產業影響相似（價差會自然收斂）

Step 2：相關性檢驗

計算相關係數（Correlation Coefficient）：

  • 用過去60-120個交易日的收盤價計算皮爾森相關係數
  • 篩選標準：相關係數 > 0.8 才值得考慮
  • 計算方式：Excel的CORREL函數，或Python的np.corrcoef()

實例：

  • 國泰金vs富邦金近120日相關係數 = 0.87 ✅
  • 台積電vs鴻海近120日相關係數 = 0.45 ❌（不同產業，不可靠）

Step 3：共整合檢驗（最關鍵）

相關性高不代表價差會均值回歸。共整合（Cointegration） 才是配對交易的靈魂。

是什麼：兩個非平穩的時間序列（股價），如果它們的線性組合是平穩的，就稱為共整合。白話說：兩檔股票的價差長期會在一個穩定範圍內震盪。

ADF檢定（Augmented Dickey-Fuller Test）：

  • 用Python的statsmodels.tsa.stattools.adfuller()檢驗價差序列
  • p-value < 0.05：拒絕虛無假說，價差是平穩的，配對有效
  • p-value > 0.05：價差不平穩，配對無效，不要交易

實例：

  • 國泰金vs富邦金價差ADF檢定p-value = 0.02 ✅（共整合成立）
  • 中鋼vs台積電電價差ADF檢定p-value = 0.38 ❌（共整合不成立，即使相關性看似OK）

重要提醒：相關性≠共整合。相關性只看「同漲同跌」，共整合看的是「價差會回來」。很多配對看起來相關，但價差越拉越大，這種配對會讓你賠錢。

三、價差計算與進出場時機

價差的計算方式

配對交易的「價差」不是簡單的A股價減B股價。要用對沖比率（Hedge Ratio） 加權。

對沖比率怎麼算：

  • 用普通最小平方法（OLS）回歸：Price_A = β × Price_B + ε
  • β就是對沖比率
  • Python：statsmodels.api.OLS(price_A, price_B).fit().params[0]

價差公式：

Spread = Price_A - β × Price_B

Z-Score標準化：

Z-Score = (Spread - Mean) / Std
  • Mean和Std用過去60-120個交易日計算
  • Z-Score告訴你目前價差偏離均值幾個標準差

進場時機

訊號操作邏輯
Z-Score > +2做空A、做多BA相對B偏貴，等收斂
Z-Score < -2做多A、做空BA相對B偏便宜，等收斂

為什麼是2個標準差：

  • 常態分佈下，超過2σ的機率約5%
  • 統計上這已經是「異常偏離」，大概率會收斂
  • 保守一點可以設2.5σ甚至3σ，勝率更高但機會更少

出場時機

訊號操作
Z-Score回到0全部平倉（價差完全收斂，獲利最大）
Z-Score回到±0.5部分平倉（先鎖定部分利潤）
Z-Score反向超過±2.5止損（共整合可能已經破裂）

止損設定

最危險的情況：共整合關係破裂，價差永遠不收斂。

止損規則：
1. Z-Score超過±3：立刻止損，共整合已失效
2. 持倉超過20個交易日價差未收斂：考慮減倉
3. 持倉超過40個交易日：強制平倉，這筆交易失敗
4. 產業發生結構性變化（如合併、拆分、政策改變）：立刻出場

四、資金分配與部位控制

對沖比例

做多和做空的金額要相等，才能達到市場中立。

計算方式：

  • 多頭部位金額 = 空頭部位金額
  • 例：做多次國泰金100萬元，做空富邦金也100萬元
  • 股數 = 金額 / 股價

Beta調整版（進階）：

  • 用Beta係數調整部位，讓兩邊的市場風險完全抵消
  • A股部位 × Beta_A = B股部位 × Beta_B
  • 例：國泰金Beta=0.9、富邦金Beta=1.1
  • 做多國泰金110萬 × 0.9 = 做空富邦金90萬 × 1.1 = 99萬市場風險

單筆風險控制

  • 單筆配對交易風險不超過總資金的2%
  • 最大持倉配對數量：3-5組（分散風險）
  • 不同產業的配對才算分散（同產業的多組配對實際上是同一個風險）

五、台股配對交易實戰SOP

盤後作業（每日）

1. 更新價差序列：下載收盤價，計算當日Spread和Z-Score
2. 監控持倉配對：檢查Z-Score是否接近出場點
3. 掃描新機會：觀察歷史配對的Z-Score是否觸發進場條件
4. 重新檢驗共整合：每週跑一次ADF檢定，確認配對仍然有效

盤中操作

1. 進場：以VWAP或開盤30分鐘均價執行，避免追價
2. 出場：觸及目標Z-Score即平倉，不留戀
3. 止損：觸及止損Z-Score立刻出場，不猶豫

選股→配對→交易 完整流程

Step 1：選產業 → 找出營收穩定、受同因素驅動的產業
Step 2：找配對 → 同產業中選2檔股票
Step 3：算相關性 → 60-120日相關係數 > 0.8
Step 4：跑ADF → p-value < 0.05
Step 5：算對沖比率 → OLS回歸求β
Step 6：監控Z-Score → 等 ±2 觸發
Step 7：進場 → 多空同時建倉
Step 8：出場 → Z-Score回到0或止損

六、配對交易五大陷阱

陷阱1：相關性陷阱

  • 問題：相關性高但共整合不成立，價差會越拉越大
  • 實例：2021年航運股長榮和陽明相關係數高達0.92，但長榮基本面遠優於陽明，價差從10%擴大到40%以上
  • 應對：一定要跑ADF檢定，不只看相關係數

陷阱2：共整合破裂

  • 問題：過去共整合不等於未來共整合。公司基本面發生結構性變化時，配對關係會突然失效
  • 實例：2020年疫情時，許多歷史配對因為產業受影響程度不同而破裂
  • 應對：每週重新跑ADF，p-value > 0.05就停止交易這組配對

陷阱3：做空成本

  • 問題：台股做空有限制（平盤以下不能融券賣出），融券借券有成本
  • 實例：融券手續費0.08%、借券費年利率1-16%，長期持有成本很高
  • 應對：用期貨或選擇權替代做空，或選擇當沖（當日沖銷無融券限制）

陷阱4：收斂時間不確定

  • 問題：價差偏離2σ後，可能1天收斂，也可能3個月才收斂
  • 實例：國泰金vs富邦金2019年Q3價差偏離2σ，花了45天才收斂
  • 應對：設定最大持倉天數（20-40天），超過強制出場

陷阱5：過度最佳化

  • 問題：回測時不斷調整參數讓績效看起來很好，但實盤完全不行
  • 實例：用特定時間段的數據調出最佳Z-Score門檻，換個時間段就虧損
  • 應對：Walk-Forward驗證，用前70%數據找參數，後30%驗證

七、進階配對策略

1. 多因子配對

不只用價格做配對，還加入：

  • 營收成長率差異
  • 本益比差異
  • 法人買賣超差異

當基本面差異偏離歷史常態時，配合價格Z-Score一起進場，勝率更高。

2. 動態對沖比率

對沖比率β不是固定的。用滾動回歸（Rolling OLS），每週重新計算β，適應最新的市場關係。

3. 配對+趨勢過濾

  • 大盤處於明確趨勢時：配對交易勝率下降（個股分化加劇）
  • 大盤震盪時：配對交易勝率最高（個股同漲同跌，價差容易收斂）
  • 用ADX或VIX判斷大盤狀態：ADX < 25或VIX > 20時，配對交易最適合

心得

配對交易是量化策略裡面最「反直覺」的一種——你不需要預測漲跌，只需要判斷「兩檔股票的關係是否偏離了常態」。這跟一般交易思維完全不同。

最核心的概念是共整合≠相關性。很多人只看相關係數就覺得可以配對，結果價差越拉越大虧到哭。相關性只告訴你「同漲同跌」，共整合才告整合才告訴你「價差會回來」。ADF檢定是配對交易的生死線，不跑就進場是在賭博。

台股做配對交易最大的困難是做空成本。融券有平盤限制、借券費用高、還有強制回補的問題。解法是用期貨或選擇權替代做空端，雖然有basis risk（基差風險），但成本和靈活度好很多。

最後，配對交易最怕的就是「共整合破裂」。過去20年都共整合的配對，可能因為一次併購、一次政策改變就永遠失效了。所以每週重新驗證ADF不是偷懶的問題，是生存的問題。

操作策略：保護性賣權Protective Put×保護性買權Protective Call×持股避險實戰

之前學了Covered Call（掩護性買權）和Iron Condor（鐵兀鷹），今天要把選擇權避險的另一面補齊：保護性賣權和保護性買權。這兩個策略的核心邏輯就一句話——幫你的現貨部位買保險。

一、保護性賣權（Protective Put）：持股避險神器

是什麼

保護性賣權 = 持有股票 + 買入同一標的賣權（Put）

本質就是幫你的持股買保險。你長期看好這檔股票，但短期擔心下跌風險，不想賣掉持股，於是買一個Put來鎖住下檔風險。

公式：保護性賣權 = 持有股票 + 買入賣權（Put）

怎麼做

Step 1：確認持股避險需求

不是任何時候都需要避險。以下情境才值得花權利金買保險：

  • 持有大型部位，短期有重大事件（選舉、財報、政策變動）
  • 市場波動加劇，VIX飆升，但你不願意賣出持股
  • 技術面出現反轉訊號，但基本面仍然看好
  • 已經有大額未實現獲利，想要鎖住利潤

Step 2：選擇履約價和到期日

履約價選擇保護程度權利金成本適用情境
價平（ATM）100%保護最高極度看空短期走勢
價外5%95%保護中等溫和避險
價外10%90%保護較低成本考量優先
價外15-20%80-85%保護最低災難保險，防大跌

到期日選擇：

  • 短期避險：1-2個月（權利金低但時間價值消耗快）
  • 中期避險：3-6個月（平衡保護和成本）
  • 長期避險：6個月以上（成本高但保護期長）

Step 3：計算最大虧損和損益兩平

最大虧損 = （股票買進價 - Put履約價） + Put權利金

損益兩平點 = 股票買進價 + Put權利金

實例：

  • 持有台積電1張（1000股），買進均價600元
  • 擔心短期下跌，買入1口履約價580元的Put，到期日1個月後
  • Put權利金：每單位8元，1口（1000股）= 8,000元

到期損益分析：

  • 股價漲到650元：Put作廢，損失8,000元權利金，股票賺50,000元，淨賺42,000元
  • 股價維持600元：Put作廢，損失8,000元權利金
  • 股價跌到550元：行使Put以580元賣出，股票虧20,000元 + 權利金8,000元 = 最大虧損28,000元
  • 如果沒有買Put：股價跌到550元，虧50,000元 → Put幫你省了22,000元

Step 4：到期前管理

  • 股價上漲超過預期：可以賣出Put回收部分權利金（仍有時間價值），或讓Put到期作廢
  • 股價下跌接近履約價：可以提前行使Put鎖定賣出價格，或在市場上賣出Put獲利了結
  • 到期日接近但風險仍在：展期（Roll）——賣出即將到期的Put，買入更遠月的Put，延長保護期

保護性賣權 vs 直接停損

比較保護性賣權直接停損
下檔保護有限（履約價以下）精確（停損價）
上檔參與完全參與完全參與
成本權利金無
彈性可以展期、調整履約價一次決定
適用情境看長做多但怕短期下跌趨勢破壞時出場

二、保護性買權（Protective Call）：做空避險神器

是什麼

保護性買權 = 做空股票 + 買入同一標的買權（Call）

跟保護性賣權鏡像相反——你做空某檔股票，但擔心股價反彈造成虧損，於是買一個Call來鎖住上檔風險。

公式：保護性買權 = 做空股票 + 買入買權（Call）

怎麼做

操作邏輯：

  • 做空某檔股票（融券或期貨空單）
  • 同時買入同一標的的Call
  • 如果股價如預期下跌 → Call作廢，做空獲利
  • 如果股價反彈上漲 → Call獲利彌補做空虧損

最大虧損 = （Call履約價 - 股票放空價） + Call權利金

損益兩平點 = 股票放空價 - Call權利金

實例：

  • 做空某股1000股，放空價100元
  • 買入履約價105元的Call，權利金3元，成本3,000元

到期損益分析：

  • 股價跌到80元：Call作廢，做空賺20,000元，扣掉權利金3,000元，淨賺17,000元
  • 股價維持100元：Call作廢，損失3,000元權利金
  • 股價漲到120元：行使Call以105元回補，做空虧5,000元 + 權利金3,000元 = 最大虧損8,000元
  • 如果沒有買Call：股價漲到120元，虧20,000元 → Call幫你省了12,000元

保護性買權的經典應用場景

2020年3月新冠暴跌後做空：

  • 空頭趨勢明確，但反彈也可能很猛烈
  • 做空同時買Call，即使遇到V型反彈也不會被軋死
  • 2008年金融海嘯做空也一樣——趨勢向下但單日大漲頻繁，有Call保護才能抱得住空單

三、保護性策略的損益圖形理解

保護性賣權損益結構

保護性賣權的到期損益圖形，長得跟單純買入Call一模一樣：

  • 股價大漲：獲利無限（股票漲多少賺多少，Put作廢）
  • 股價在損益兩平點附近：小虧或小賺
  • 股價大跌：虧損有上限（最多虧到履約價以下的保護停止）

這就是為什麼保護性賣權被稱為「合成的買入Call」——它用股票+Put組合出了跟買Call一模一樣的損益結構。

保護性買權損益結構

保護性買權的到期損益圖形，長得跟單純買入Put一模一樣：

  • 股價大跌：獲利空間大（做空賺多少就是多少，Call作廢）
  • 股價在損益兩平點附近：小虧或小賺
  • 股價大漲：虧損有上限（最多虧到履約價以上的保護停止）

合成策略的概念

現貨+選擇權組合等效於
持有股票 + 買入Put買入Call（看多，下檔保護）
做空股票 + 買入Call買入Put（看空，上檔保護）
持有股票 + 賣出Call賣出Put（看多，收取權利金）
做空股票 + 賣出Put賣出Call（看空，收取權利金）

這就是選擇權的買賣權平價關係（Put-Call Parity）。理解這個概念，你就能用不同工具組合出想要的損益結構。

四、保護性策略的權利金成本考量

權利金的隱性成本

保護性策略最大的代價就是權利金。這筆錢就像保險費——不管用不用得到，錢已經花了。

台股選擇權權利金估算：

  • 台指選擇權：價外5%的Put，1個月到期，大約指數的1-2%
  • 個股選擇權/權證：視波動率而定，通常2-5%
  • 如果一年到頭都在買保護性Put，年化成本可能高達10-20%

怎麼降低權利金成本：

1. 只在不確定性高的時期買保險：財報季、選舉前、重大政策發布前
2. 選擇價外履約價：價外5-10%的Put權利金比價平便宜很多，犧牲少部分保護換取低成本
3. 用Covered Call補貼Put成本：同時賣出Covered Call收取權利金，用來支付Put的權利金，這就是Collar策略（領口策略）
4. 選擇較長的到期日：時間價值不是線性消耗，3個月的Put不會是1個月的3倍

Collar策略（領口策略）：成本更低的避險

公式：Collar = 持有股票 + 買入Put（下檔保護）+ 賣出Call（上檔限制）

  • 買Put花權利金，賣Call收權利金
  • 如果Call權利金 ≈ Put權利金，整體成本接近零
  • 代价：上檔獲利空間被封頂（Call履約價以上賺不到）

實例：

  • 持有台積電600元
  • 買入580元Put，權利金8元（花8,000元）
  • 賣出650元Call，權利金7元（收7,000元）
  • 淨成本：8,000 - 7,000 = 1,000元

到期損益：

  • 股價 < 580元：Put保護，最大虧損 = （600-580）+ 1 = 21元/股
  • 股價 580-650元：正常獲利，扣除1,000元淨成本
  • 股價 > 650元：Call被行使，最大獲利 = （650-600）- 1 = 49元/股

Collar策略的取捨：用上檔空間換取下檔保護，成本接近零。適合「不想花大錢買保險，也不介意上檔封頂」的投資人。

五、保護性策略的台股實戰考量

台股避險工具比較

工具避險方向流動性成本適用標的
台指選擇權Put大盤下跌極佳透明0050、大型權值股
個股認售權證個股下跌一般較高有發行的個股

融券放空個股下跌差借券費+平盤下限制所有融資券股

台指選擇權避險SOP

情境：持有300萬台股部位，擔心大盤短期下跌

1. 計算Beta值：假設投組Beta = 1.2
2. 避險口數 = 投組市值 × Beta / （指數 × 契約乘數）

  • 3,000,000 × 1.2 / （20,000 × 50）= 3.6口 → 買4口Put

3. 選擇履約價：價外5%（指數20,000 → 履約價19,000）
4. 選擇到期日：1-2個月後
5. 買入4口19,000 Put，支付權利金約30,000-50,000元
6. 到期管理：大盤大跌→Put獲利補償現貨虧損；大盤穩定→Put作廢，權利金是保險費

保護性策略的常見錯誤

錯誤1：一年到頭都在買保險

  • 權利金成本會吃掉大部分獲利
  • 只在VIX飆升或重大事件前買保險
  • 好比車險：你不會一年365天都買全險

錯誤2：履約價選太接近價平

  • 價平Put權利金最貴，保護範圍等於停損
  • 如果只是怕大跌，價外10-15%的Put就夠了
  • 價外Put就像高自負額保險：平時小跌自己承擔，大跌才理賠

錯誤3：忘記時間價值消耗

  • 選擇權有時間衰減（Theta），每過一天權利金就少一點
  • 如果避險事件沒發生，到期時Put歸零
  • 解法：提前在時間價值還有剩餘時賣出，回收部分權利金

錯誤4：忽略隱含波動率的影響

  • VIX高時Put權利金暴漲，此時買保險最貴
  • 最需要保險的時候，保費也最貴——這是避險的矛盾
  • 解法：在VIX低時提前佈局遠月Put，成本更低

心得

保護性賣權和保護性買權，說白了就是「幫你的部位買保險」。這個比喻太精準了——你買車險不是因為你希望出車禍，而是因為萬一出事你承受不起。同樣的邏輯，買Put不是因為你看空，而是因為你長線看多但短期承受不起大跌。

最關鍵的洞察是Put-Call Parity——持有股票+買入Put的損益結構，跟直接買入Call一模一樣。這意味著，如果你想參與上漲但怕下跌，你有兩種選擇：直接買Call（槓桿較大，不需要持股），或者持股+買Put（保護現有部位）。兩種方式的最終損益結構是一樣的，差別在於資金效率和是否已持有現貨。

Collar策略（領口）是我之前沒想到的好東西。用賣Call的權利金來支付買Put的權利金，整體成本接近零。代价是上檔被封頂，但如果你本來就打算在目標價出場，那賣出目標價的Call等於是「提前收到出場的錢」，根本不虧。這對已經有大額未實現獲利的部位來說，是極度實用的鎖利工具。

最後，保護性策略最常犯的錯誤就是「一年到頭都在買保險」。權利金是確定性成本，但保護效果是不確定的。如果一年花10-20%的權利金在避險上，長期報酬率就被吃掉了。正確的做法是：只在VIX低且有不確定性事件的時候買保險，平常就用停損來管理風險。

操作策略：反轉交易進階——道氏理論反轉確認×多指標三重驗證系統

之前學了反轉交易的基礎概念（頂部底部反轉型態、反轉vs回調判斷、五大鐵律），今天要進階到實戰層面：如何用系統化的方法確認反轉，而不是靠感覺猜頂猜底。核心框架是道氏理論反轉三步法+K線型態+技術指標背離的三重驗證系統——三道確認缺一不可，大幅降低假反轉的誤判率。

一、道氏理論反轉確認：最客觀的趨勢反轉判斷法

是什麼

道氏理論對趨勢的定義極度簡潔：

  • 上漲趨勢：高點更高、低點也更高（高價低價連續上移）
  • 下跌趨勢：高點更低、低點也更低（高價低價連續下移）

反轉的定義同樣明確：

  • 從上漲轉下跌：高價與低價都未能上移，且低價跌破前波低點（下移）
  • 從下跌轉上漲：高價與低價都未能下移，且高價突破前波高點（上移）

怎麼做——反轉確認三步法

步驟1：趨勢減速信號（黃色警示）

在上升趨勢中，如果最新高點或低點未能上移，代表趨勢減速。這是「黃色警示」，不是立即反轉——後方高點若突破最新高點，趨勢仍可延續。

判斷方式：

  • 前波高點120元，最新反彈只到115元 → 高點未能上移 = 黃色警示
  • 此時不要急著做空，只是提高警覺

步驟2：反轉確認信號（紅色確認）

當高價下移且低價也下移時，才是真正的趨勢反轉確認。

判斷方式：

  • 黃色警示後，股價跌破了前波低點（例如從108元跌破105元）
  • 此時「高價下移+低價下移」雙重確認 = 道氏理論反轉成立

步驟3：型態學驗證（強化確認）

道氏理論的反轉訊號，常與頭肩頂/底、雙重頂/底等型態同時出現。如果兩者共振，反轉機率更高。

判斷方式：

  • 道氏反轉確認 + 頭肩頂頸線跌破 = 最強反轉訊號
  • 道氏反轉確認 + M頭頸線跌破 = 次強反轉訊號
  • 單獨道氏反轉（無型態配合）= 仍需謹慎，可能是假反轉

實例

情境：台積電從500元漲到680元後可能反轉

  • 步驟1：680元後反彈只到660元（高點未能上移）→ 黃色警示
  • 步驟2：股價跌破前波低點630元（高價下移+低價下移）→ 反轉確認
  • 步驟3：K線圖形成頭肩頂，頸線在630元 → 跌破630元時型態+道氏雙重確認
  • 操作：630元跌破時做空，停損設645元（頸線上方2%），目標看580元（頭部680到頸線630的距離50元，630-50=580元）

適用時機

  • 日線以上級別的反轉判斷（週線更可靠）
  • 不適用於太短的時間框架（小時線雜訊太多）
  • 搭配型態學使用時準確度最高

二、移動平均線判斷反轉：趨勢的溫度計

是什麼

移動平均線是趨勢的「溫度計」——上漲趨勢中，價格在均線上方推移，均線是支撐；當均線轉平甚至下彎，且價格跌破均線，代表趨勢可能反轉。

怎麼做

1. 均線由支撐轉壓力的三階段

階段均線狀態價格位置意義
趨勢進行中均線上揚價格在均線上方均線是支撐，趨勢健康
趨勢減速均線走平價格與均線交錯多空拉鋸，方向不明
趨勢反轉均線下彎價格跌破均線均線變壓力，反轉機率高

2. 均線參數選擇

  • 20MA或25MA：與價格交會頻率適中，適合判斷短期趨勢反轉
  • 40MA：代表中期趨勢，跌破40MA通常代表較大級別的反轉
  • 60MA（季線）：波段操作的重要多空分界線

3. 進場判斷

當均線轉為橫向，價格在均線上下反覆穿越時，就要提高警覺。當均線明確下彎，且道氏理論高價低價同時下移時，就是反轉進場時機。

實例

  • 某股沿著25MA上漲，25MA持續上揚 → 趨勢健康
  • 股價從100元跌到90元反彈，25MA開始走平 → 黃色警示
  • 股價再次跌破90元，25MA下彎 → 反轉確認進場做空
  • 停損設92元（均線上方），目標看80元（前波支撐）

三、背離訊號：反轉的領先指標

是什麼

背離（Divergence）是指價格走勢與技術指標出現相反方向——價格創新高但指標沒有，或價格創新低但指標沒有。這代表趨勢的「引擎」正在減速，雖然車子還在前進，但動力已經不足，隨時可能停車甚至倒退。

怎麼做

1. 頂背離（看跌背離）

定義：價格創新高，但RSI/MACD/KD的高點反而比前一次低

背離的本質：

  • 價格雖然上漲，但上漲幅度正在減少
  • 新的多頭力道減弱，買盤後繼無力
  • 已有獲利者開始停利出場
  • 持有未實現損失的投資者停損賣單 + 持有未實現利益的投資者停利賣單 = 賣壓累積

RSI頂背離判斷：

  • 前波高點價格100元，RSI 75
  • 本次高點價格105元，RSI只有68
  • 價格創新高但RSI更低 = 頂背離確認

MACD頂背離判斷：

  • 前波高點MACD柱狀體+0.5
  • 本次高點MACD柱狀體只有+0.2
  • 短期EMA與長期EMA的差距縮小 = 上升力度減弱

2. 底背離（看漲背離）

定義：價格創新低，但RSI/MACD/KD的低點反而比前一次高

KD底背離判斷：

  • 前波低點價格80元，KD低點15
  • 本次低點價格75元，KD低點20
  • 價格創新低但KD更高 = 底背離確認，下跌動能減緩

3. 隱形背離（趨勢延續背離）

這是比較少人知道的高階用法：

  • 隱形頂背離：價格未創新高，但RSI/MACD創新高 → 雖然價格沒漲，但內部動能很強，突破機率高
  • 隱形底背離：價格未創新低，但RSI/MACD創新低 → 雖然價格沒跌，但內部空方動能強，跌破機率高

背離的陷阱

  • 背離只是「預兆」，不代表一定反轉
  • 強趨勢中背離可以連續出現2-3次才真正反轉
  • 不能只靠背離進場，必須搭配道氏理論和K線型態

四、多指標三重驗證系統：反轉交易的終極確認法

是什麼

單一指標判斷反轉的誤判率很高。多指標三重驗證系統要求三個維度同時確認才進場，大幅提高勝率。

怎麼做

三重驗證框架：

第一重：道氏理論結構確認

  • 高價低價同時下移（或上移）= 趨勢結構反轉
  • 這是最客觀的確認，沒有主觀判斷空間

第二重：K線型態確認

  • 頂部反轉：頭肩頂頸線跌破、M頭確認、空頭吞噬、黃昏之星
  • 底部反轉：頭肩底頸線突破、W底確認、多頭吞噬、早晨之星

第三重：技術指標背離確認

  • RSI背離、MACD背離、KD背離至少出現一個
  • 量價背離（價漲量縮或價跌量增）也可作為第三重

進場規則：

  • 三重全部確認 → 高信心進場（勝率預估60-70%）
  • 兩重確認 → 小部位試單（勝率預估40-50%）
  • 只有一重確認 → 不進場，繼續觀察

實例：頂部反轉做空

情境：某股從50元漲到85元

第一重驗證（道氏理論）：

  • 85元後反彈只到80元（高點未能上移）→ 黃色警示
  • 跌破前波低點72元（高價下移+低價下移）→ 反轉確認 ✅

第二重驗證（K線型態）：

  • K線形成頭肩頂：左肩78元、頭部85元、右肩80元
  • 頸線在72元，跌破72元 → 頭肩頂確認 ✅
  • 目標價 = 72 - (85-72) = 72 - 13 = 59元

第三重驗證（指標背離）：

  • 價格從78元漲到85元（創新高），但RSI從72降到65 → 頂背離 ✅
  • MACD柱狀體從+0.8縮小到+0.3 → 動能衰退 ✅

操作：

  • 三重全部確認 → 72元跌破時做空
  • 停損設74元（頸線上方約3%），風險2元
  • 目標59元，報酬13元
  • 風險報酬比 = 13:2 = 6.5:1

實例：底部反轉做多

情境：某股從120元跌到70元

第一重驗證（道氏理論）：

  • 70元後反彈到80元，回檔只到73元（低點未能下移）→ 黃色警示
  • 突破前波高點80元（高價上移+低價上移）→ 反轉確認 ✅

第二重驗證（K線型態）：

  • K線形成W底：左底70元、右底73元
  • 頸線在80元，突破80元 → W底確認 ✅
  • 目標價 = 80 + (80-70) = 90元（若右底73元則為80 + (80-73) = 87元，取保守值87元）

第三重驗證（指標背離）：

  • 價格從75元跌到70元（創新低），但KD從18升到22 → 底背離 ✅
  • 成交量在下跌時萎縮，反彈時放量 → 量價配合 ✅

操作：

  • 三重全部確認 → 80元突破時做多
  • 停損設72元（W底最低點下方），風險8元
  • 目標87-90元，報酬7-10元
  • 風險報酬比 = 7:8 到 10:8 ≈ 0.9:1 到 1.25:1

注意：底部反轉的風報比通常不如頂部反轉，因為底部形成時間長、反彈初期力道不確定。建議分批進場，先用小部位試單。

五、反轉交易五大鐵律（複習+強化）

之前學過五大鐵律，今天結合三重驗證系統重新強化：

鐵律1：等待確認，不預測

  • 不要在第一個反轉跡象就進場
  • 三重驗證至少完成兩重才考慮進場
  • 等頸線突破/跌破才是最安全的進場點

鐵律2：嚴格停損，保命第一

  • 做空停損設在頸線上方2-3%
  • 做多停損設在頸線下方或型態最低點下方
  • 反轉交易本質上是逆勢操作，停損比順勢交易更重要

鐵律3：日線以上級別操作

  • 反轉訊號在日線以上才有參考價值
  • 小時線、15分線的反轉太多雜訊
  • 週線反轉訊號最可靠，月線反轉是超大級別

鐵律4：道氏理論辨識，非主觀預測

  • 用高價低價的上移下移來客觀判斷
  • 不要「感覺」要反轉了就進場
  • 道氏理論是量化標準，消除主觀偏誤

鐵律5：型態+背離+道氏三重確認

  • 單一確認的勝率不夠高
  • 三重確認才是進場的充分條件
  • 寧可錯過，不可做錯

六、反轉交易 vs 回調：最後一次判斷框架

判斷維度回調（不進場做空）反轉（進場做空）
道氏理論高價低價仍在上移高價低價同時下移
均線長期均線仍上揚，價格在均線上方均線下彎，價格跌破均線
K線型態無反轉型態頭肩頂、M頭等型態出現
指標背離無背離或隱形背離（趨勢延續）RSI/MACD/KD出現頂背離
成交量回調量縮（正常）反彈量縮+下跌放量（不正常）
幅度未超過前波漲幅的38.2%超過61.8%且持續下跌

心得

今天把反轉交易從「感覺可能要反轉了」提升到「三重驗證系統確認反轉」，這是質的飛躍。

最核心的體悟是：道氏理論的「高價低價同時下移」才是反轉的客觀定義，不是什麼「感覺漲不動了」這種主觀判斷。在看到高價未能上移時只是黃色警示，必須等到低價也下移才是紅色確認。這中間的等待就是紀律的考驗——很多人在黃色警示時就急著做空，結果被最後一波上漲掃出去。

背離訊號的解釋讓我更清楚它的本質：RSI背離代表「上漲幅度正在減少」，MACD背離代表「短期EMA和長期EMA的差距在縮小」，這不是什麼神秘的預測，而是動能衰退的客觀觀察。但背離只是預兆，不是確認——強趨勢中背離可以出現2-3次都不反轉，所以背離必須搭配道氏和型態一起判斷。

三重驗證系統的價值在於強迫你等待足夠的確認。單一指標出訊號時你會想「這次可能是真的」，但三重驗證要求你冷靜下來，等所有條件都滿足。這不是降低效率，是降低誤判。寧可少做幾次對的，也不要多做幾次錯的——因為反轉交易做錯的代價很大（逆勢操作+沒有趨勢保護）。

最後，底部反轉的風報比確實不如頂部反轉。頂部反轉通常有明確的頸線和頭部距離可以算目標價，底部反轉則因為反彈初期力道不確定，目標價較難精確估算。所以底部反轉建議用分批進場，頂部反轉可以一次進場。

類股輪動策略（Sector Rotation Strategy）

什麼是類股輪動？

是什麼：資金在不同產業類股之間流動，形成「先漲A、再漲B、後漲C」的接力現象。當一個類股漲幅已大、資金獲利了結後，會流向下一個相對便宜、有題材的類股，如此循環。

為什麼會輪動：
1. 資金溢出效應：大資金從已漲多的類股撤出，轉進相對便宜類股
2. 景氣循環驅動：不同產業在景氣不同階段有不同表現（美林投資時鐘）
3. 題材催化：政策、事件驅動資金快速遷移（如AI題材、降息題材）
4. 估值收斂：漲多類股估值偏高、未漲類股估值吸引，資金自然再分配

類股輪動三種操作流派

一、景氣輪動法（Macro Sector Rotation）

怎麼做：根據景氣位置判斷應配置哪些類股

景氣階段利率方向最強類股操作方向
復甦期利率谷底上升科技、工業、原物料加碼成長股
擴張期利率上升能源、原物料持有成長+價值
趨緩期利率高峰防禦型（公用事業、必需消費）減碼成長股
衰退期利率下降公用事業、債券現金+防禦

實例：

  • 情境：2024年7月美股AI類股（輝達為首）從高點回落
  • 操作：大資金從半導體撤出，轉進建商股（D.R. Horton等），建商股7月中突破年高後一路漲到9月
  • 判斷：景氣從「擴張期」轉向「趨緩期」，資金從高成長類股轉進受惠降息的營建類股

適用時機：中長期投資者、ETF配置者、3個月以上的波段操作

二、族群股短線輪動法（Intra-sector Rotation）

怎麼做：觀察盤中強勢產業，在龍頭股帶動下，利用補漲股時間差操作

1. 找出當日最強產業：看盤中各產業漲跌幅排行
2. 辨識龍頭股：產業中漲幅最大、量能最足的個股
3. 選擇操作標的：

  • 積極型：直接追價龍頭股

4. 設定停損：龍頭股前波高點下方0.5%~1%

實例：

  • 情境：航運族群某日最強，龍頭中櫃（2613）開盤到9:10漲6%
  • 操作A：直接追價中櫃，9:30到10:00再漲到7%以上觸漲停
  • 操作B：觀察中櫃漲停後，資金外溢到同產業志信（2611）10點後漲5%、遠雄港（5607）10:30後拉抬
  • 關鍵：龍頭漲停=資金外溢起點，補漲股的進場時間在龍頭鎖死後15-30分鐘

適用時機：當沖、短線1-3天操作

三、相對輪動圖法（RRG, Relative Rotation Graph）

怎麼做：用RS-Ratio和RS-Momentum兩維度判斷類股相對強弱位置

RRG四象限判讀：

象限RS-RatioRS-Momentum意義操作
右上（領先）>100>100且上升相對強且動能增強**加碼**
右下（減弱）>100<100且下降相對強但動能衰退**減碼/觀察**
左下（落後）<100<100且下降相對弱且動能衰退**避開**
左上（改善）<100>100且上升相對弱但動能改善**準備進場**

輪動路徑：左下→左上→右上→右下→左下，類股沿此路徑移動

實例：

  • 情境：半導體類股從左上象限移向右上（RS-Ratio從<100轉為>100）
  • 操作：在左上象限開始佈局，進入右上象限後加碼，移向右下時開始減碼
  • 判斷：搭配美林時鐘確認景氣階段，雙重驗證類股方向

適用時機：波段操作、ETF輪動、2週到3個月的中線操作

類股輪動實戰SOP

Step 1：判斷大盤環境

  • 景氣燈號>38（紅燈區）：偏多，選攻擊型類股
  • 景氣燈號16-38（黃藍燈區）：中性，防禦+攻擊並行
  • 景氣燈號<16（藍燈區）：偏空，選防禦型類股或債券

Step 2：篩選強勢類股

  • 日/週/月漲幅排行前三名產業
  • RRG位於左上或右上象限的類股
  • 產業RSI從<30回升或從50向上穿越65

Step 3：確認領先-落後關係

  • 比較產業間RSI走勢，找出「領先產業走強→落後產業即將跟漲」的配對
  • 例：航運RSI>65時，半導體可能即將接棒

Step 4：選股進場

  • 從強勢類股中選龍頭股或補漲股
  • 用籌碼面（法人買超、大戶持股）+技術面（突破、均線多排）雙重確認
  • 設定停損：前波低點或均線支撐

Step 5：輪動出場

  • 當類股從RRG右上移向右下，或RSI>80出現背離
  • 獲利了結後，資金轉進下一個左上象限類股
  • 如景氣燈號轉弱（>38趨緩），全數轉入防禦型或債券

類股輪動五大陷阱

1. 輪動順序不固定：上次是航運→半導體，下次不一定，每次景氣週期類股順序可能不同
2. 輪動速度差異大：有時1-2週就完成，有時3-6個月，過早切換會兩面挨耳光
3. 假輪動（假突破）：類股看似轉強，實為反彈後繼續下跌，需量價確認
4. 流動性風險：小型類股輪動可能因流動性不足無法及時進出
5. 過度交易：頻繁切換類股，手續費和滑價吃掉利潤，建議2-4週評估一次

類股輪動這東西，說穿了就是「資金是流動的」這件事的具體化。之前學過美林投資時鐘、RS相對強度，現在看類股輪動剛好是把這兩個工具串起來——景氣位置決定大方向，RS-Ratio/RS-Momentum決定哪個類股值得切入。

最實用的反而是那個族群股短線操作法。龍頭鎖漲停→資金外溢到補漲股，這個節奏感很清楚，當沖的話15-30分鐘內就要判斷。但RRG四象限對波段操作更有價值，左上象限開始佈局、右上象限加碼、右下象限減碼，這個邏輯跟之前學的威科夫吸籌-派發週期有異曲同工之妙。

陷阱部分要特別注意「上次有效的輪動順序下次不一定有效」，這就是過度配適的問題。輪動的本質是資金流動，不是死板的公式，每次都要重新觀察確認。

價格行為交易 Price Action — 裸K交易法完整解析

一、Price Action 的核心哲學

是什麼：Price Action（價格行為交易）是只根據價格本身的變動來分析和交易的方法，不依賴任何技術指標，又稱「裸K交易」。核心理念：市場的一切資訊最終都反映在價格走勢上。

為什麼有效：
1. 所有技術指標的源頭都是價格 — 指標是加工食品，Price Action 是生鮮食材
2. 零和賽局的生存之道 — 外匯等零和市場，關鍵是比別人更早判斷多空誰勝出，而這場拉鋸最直接體現在K線上
3. 自我實現的預言 — 越多機構依據相同 Price Action 信號交易，信號有效性越高
4. 假象也是最好的訊號 — 從價格走勢中產生的假突破、誘多誘空，本身也能成為反向操作的高勝率訊號

二、裸K交易三大核心要素

1. 市場結構 Market Structure — 判斷趨勢方向

是什麼：用道氏理論判斷市場整體方向，是 Price Action 的骨架。

怎麼做：
1. 看K線的高低點排列方式
2. 上升趨勢：不斷創出「更高的高點 HH」和「更高的低點 HL」
3. 下降趨勢：不斷創出「更低的高點 LH」和「更低的低點 LL」
4. 盤整區間：在一個固定高低點之間來回波動

實例：

  • 情境：台積電從 550 → 600 → 580 → 650 → 630 → 700，每次回檔低點（580、630）都高於前次低點，每次高點（600、650、700）都創新高 → 上升趨勢確認，只找做多機會
  • 反之：若 700 → 650 → 680 → 600 → 630 → 550，高點遞減低點也遞減 → 下降趨勢，只找做空或空手

適用時機：每次看圖的第一步，先判斷市場結構再決定方向。在上升趨勢中找買點，下降趨勢中找賣點。

2. 關鍵水平位 Key Levels — 尋找攻防戰場

是什麼：市場多空曾激烈交鋒的價位，未來價格再觸及時極可能引發重要反應。是 Price Action 的戰場。

四種關鍵水平位：

類型說明比喻
支撐位 Support價格下跌至此，買方增強阻止續跌地板
壓力位 Resistance價格上漲至此，賣方增強阻止續漲天花板
趨勢線 Trend Lines上升連接低點、下降連接高點的斜線動態支撐壓力
供需區 Supply/Demand機構集中下單的價格區域劇烈波動的起點

怎麼做：
1. 在日線圖或週線圖上，找出價格多次反彈或受阻的價位
2. 標記水平支撐壓力線
3. 畫趨勢線（至少連接2個以上的低點或高點）
4. 識別供需區（價格劇烈波動的起點區域）

實例：

  • 情境：某股在 100 元附近3次觸底反彈 → 100 元是強支撐
  • 操作：當股價再次回到 100 元附近時，不是立刻買，而是等待 Price Action 信號確認

適用時機：只在高勝率區域（關鍵水平位附近）進場，不要在毫無參考的區間隨意交易。

3. K線型態 Candlestick Patterns — 識別進場信號

是什麼：當價格來到關鍵水平位時，K線型態如同戰場號角，提供具體進場信號。是 Price Action 的觸發器。

(1) Pin Bar 長影線K線

特徵：實體很小，有一根非常長的影線（長度是另一方向影線的3倍以上）

怎麼做：
1. 在支撐位出現長下影線 Pin Bar → 看漲信號，買方強力反擊
2. 在壓力位出現長上影線 Pin Bar → 看跌信號，賣方強力反擊
3. 進場：Pin Bar 收盤後進場
4. 停損：設在 Pin Bar 影線極端點外（看漲 Pin Bar 停損設最低點下方幾個 tick）
5. 停利：設在下一個關鍵壓力位，或風險報酬比 1:2 以上

實例：

  • 情境：台積電跌到 550 元支撐區，出現一根實體僅 2 元但下影線長 20 元的 Pin Bar
  • 操作：550 元買進，停損 547 元（最低點下方），停利看 580 元壓力位，風報比約 10:3

適用時機：只在關鍵水平位的 Pin Bar 才有效，盤整區中間出現的 Pin Bar 不可靠。

(2) Engulfing Pattern 吞噬型態

特徵：第二根K線的實體完全包覆前一根K線的實體，且顏色相反

怎麼做：
1. 看漲吞噬：下降趨勢中，一根長紅K線完全包覆前一根黑K線 → 強烈買入信號
2. 看跌吞噬：上升趨勢中，一根長黑K線完全包覆前一根紅K線 → 強烈賣出信號
3. 進場：吞噬K線收盤後進場
4. 停損：設在吞噬K線的極端點外
5. 停利：下一個關鍵水平位

實例：

  • 情境：某股在 80 元支撐區，前一天收黑 79-81 元，第二天收長紅 78-85 元，完全包覆 → 看漲吞噬
  • 操作：85 元收盤價附近買進，停損 77.5 元，停利看 95 元壓力

適用時機：在支撐壓力位的吞噬型態最可靠；盤整中間的吞噬型態無方向性，不操作。

(3) Inside Bar 內包線

特徵：第二根K線的高低點完全被前一根K線（母線 Mother Bar）包覆

怎麼做：
1. 代表市場暫時進入猶豫和盤整狀態
2. 等待價格突破母線高點 → 做多信號
3. 等待價格跌破母線低點 → 做空信號
4. 停損設在母線另一端之外

實例：

  • 情境：上升趨勢中，母線高點 620 低點 600，第二天形成 Inside Bar 高 615 低 605
  • 操作：價格突破 620 時買進，停損設 598 元（母線低點下方）

適用時機：趨勢中的 Inside Bar 代表蓄勢待發，突破方向通常順著原趨勢。盤整區的 Inside Bar 無方向性。

三、Price Action 完整交易策略 — 四步流程

怎麼做：

1. 多時框分析：先在較大時間週期（日線、週線）判斷市場結構和趨勢方向
2. 識別關鍵水平位：在大週期圖上標示支撐壓力、趨勢線、供需區
3. 等待價格觸及 + 信號出現：切到小週期（60分、15分），等待價格回到關鍵位且出現 Pin Bar / 吞噬 / Inside Bar
4. 制定交易計畫：明確進場點、停損點、停利點後才下單

實例 — 完整操作流程：

  • 情境：日線圖台積電上升趨勢中，550 元是前波支撐
  • 第一步：日線確認上升趨勢（HH + HL）
  • 第二步：標記 550 元支撐位
  • 第三步：切到60分線，等股價拉回 550 附近，出現長下影 Pin Bar
  • 第四步：550.5 元買進，停損 547 元，停利 575 元（下一壓力位），風報比 1:8.3

四、Price Action vs 技術指標比較

比較維度Price Action技術指標
信號時效性領先或同步，即時反映滯後，基於歷史價格計算
圖表簡潔度極簡，只有K線複雜，多指標可能矛盾
主觀性較高，需經驗累積較低，信號客觀
學習曲線初期陡峭但適用性廣初期平緩但精通難
適用市場所有市場通用不同市場表現差異大

重要原則：Price Action 不代表完全不用指標。許多成功交易者會用 1-2 條均線（如 20EMA、50EMA）輔助判斷動態趨勢，但決策核心仍是價格在關鍵位的行為。指標是「輔助」不是「主角」。

五、Price Action 假象辨識 — 最重要的進階技巧

是什麼：主力刻意製造的假突破、誘多誘空行為，本身就是可操作的反向訊號。

怎麼做：
1. 假突破識別：價格短暫越過支撐壓力後迅速掉回 → 主力誘多或誘空
2. 操作方式：假突破掉回後，順著反方向進場
3. 實例：某股在 100 元壓力位，價格衝到 103 元但收盤回到 99 元，形成長上影線 → 假突破誘多，做空進場

適用時機：在重要支撐壓力位的假突破最可靠，配合量能萎縮（假突破無量）更確認。

1. 在非關鍵位置找信號 — 沒有支撐壓力參考的 Pin Bar 或吞噬型態不可靠

心得

Price Action 最大的啟發是「回歸本質」。我之前學了一堆指標 KD、RSI、MACD、布林通道，結果有時指標互相矛盾反而不知道怎麼操作。Price Action 告訴我們：這些指標的源頭都是價格，與其看加工後的二手資料，不如直接看原始的價格行為。

最實用的概念是「三要素組合」：市場結構定方向 → 關鍵水平位找戰場 → K線型態扣板機。這三個缺一不可。很多人只看 Pin Bar 或吞噬型態就進場，但如果不看位置（是不是在支撐壓力位）和方向（是不是順著大趨勢），勝率會很低。

另一個收穫是「假象也是訊號」這個觀點。OANDA 文章說得很好——市場的假突破本身就是主力意圖的暴露，看懂假象就能反過來利用它。這跟之前學的威科夫 Spring（彈簧效應）和 UTAD（上沖假突破）完全呼應，本質上都是在說同一件事：主力製造的假象，反而是最好的反向操作時機。

最後，Price Action 不是完全排斥指標，而是「指標為輔、價格為主」。這個定位很重要，不然容易走極端。一條 20EMA 當動態支撐壓力參考，配合裸K的型態判讀，其實是很務實的做法。
