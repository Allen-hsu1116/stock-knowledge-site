道氏理論六大法則速查

法則 1：價格反映一切

  • 不用追逐消息面，價格已經反映了
  • 利多不漲 → 賣訊；利空不跌 → 買訊

法則 2：市場有三種趨勢

  • 長期趨勢：1 年以上 → 波段操作
  • 中期趨勢：3 週～3 個月 → 回檔買進
  • 短期趨勢：數小時～3 週 → 當沖、極短線

法則 3：長期趨勢有三階段

  • 承接期：股價溫吞、量縮、主力佈局
  • 大眾參與期：股價急漲、媒體報導、突破追進
  • 出貨期：利多滿天飛、散戶搶進、準備出場

法則 4：指數必須相互驗證

  • 台股：加權 + 電子類股指數 + 金融類股指數
  • 美股：S&P 500 + 道瓊 + NASDAQ + 費半
  • 趨勢背離 → 警訊

法則 5：成交量必須確認趨勢

  • 價漲量增 → 多頭健康
  • 價漲量縮 → 買盤不足
  • 價跌量縮 → 惜售（多頭）
  • 價跌量增 → 殺盤（空頭）

法則 6：趨勢持續直到反轉訊號出現

  • 不要猜頭猜底，順勢操作
  • 高點降低 + 低點降低 → 空頭
  • 高點墊高 + 低點墊高 → 多頭
### 核心觀念 1. **不要猜頭底** → 順勢操作才是王道 2. **成交量說真話** → 價格會騙人，量不會 3. **指數相互驗證** → 單一訊號不可靠 4. **三階段理論** → 當散戶都知道這檔股票好時，通常已經是尾聲 ### 實務應用 道氏理論是技術分析的「基本功」，很多後來的理論（波浪理論、形態學）都是從這裡延伸出來的。最實用的是「三階段理論」判斷股票生命週期，以及「成交量確認趨勢」來過濾假訊號。這些觀念看似簡單，但大部分散戶都做不到——總是想抄底、想賣在最高點、看到新聞就追。道氏理論提醒我們：**尊重趨勢，不要自以為聰明。**
