量價關係核心概念

為什麼成交量重要？

「新手看價、老手看量」——這句話有道理。成交量反映的是參與者的決心：

  • 成交量大 = 多空看法分歧、籌碼混亂
  • 成交量小 = 多空看法一致、籌碼穩定

關鍵觀念：成交量一定是買張數等於賣張數。看盤軟體的紅綠色只是方便辨識，不代表「買進量」或「賣出量」。

六種量價關係詳解

1. 價漲量增

是什麼：股價上漲，成交量也增加。多頭趨勢最健康的型態。

怎麼判斷：

  • 低檔盤整突破 + 量增 = 確認漲勢健康
  • 高檔異常大量 + 長黑K棒 = 小心主力出貨

實例：

  • 情境：股價 50 元盤整三個月，突破 55 元壓力，成交量從日均 1,000 張放大到 3,000 張
  • 操作：可追價買進，設停損於突破點下方（如 52 元）
  • 危險訊號：股價在 100 元高檔，單日爆量 10,000 張（日均 2,000 張），收長黑K → 可能是出貨

適用時機：

  • ✅ 低檔突破壓力區
  • ✅ 主升段上漲確認
  • ❌ 高檔異常大量（小心反轉）

2. 價漲量縮

是什麼：股價上漲但成交量縮減。分兩種情況：量價背離 vs 量縮上漲。

怎麼區分：

特徵量價背離量縮上漲
K 線強弱紅K越來越弱K線強勢
成交量漸漸縮小漸漸縮小
含義追價資金減少賣方不足、籌碼鎖定
後勢可能下跌或盤整容易加速上漲

量縮上漲的操作意義：

  • 情境：想買 500 張，賣方只拋出 400 張 → 成交 400 張 + 鎖漲停
  • 代表：持有者不想賣、主力高度控盤
  • 周線/月線量縮上漲 = 長線主力看好，值得跟進

適用時機：

  • ✅ 周線/月線級別的量縮上漲（主力長線佈局）
  • ✅ 漲停鎖死造成的量縮（籌碼穩定）
  • ❌ 高檔價漲量縮 + 紅K變弱（背離訊號）

3. 價跌量縮

是什麼：股價下跌，成交量也縮減。多頭趨勢下的正常回檔。

怎麼判斷：

  • 正常洗盤：回檔幅度小、量縮 → 賣壓不大
  • 危險訊號：回檔幅度大、量縮 → 沒買盤接手

實例：

  • 情境：股價從 100 元漲到 120 元，拉回到 115 元，成交量從 3,000 張降到 1,500 張
  • 判斷：正常洗盤，可在 115 元附近承接
  • 危險：從 120 元跌到 105 元還量縮 → 沒人想買，續跌機率高

適用時機：

  • ✅ 多頭趨勢中的量縮回檔（買點）
  • ✅ 橫向盤整的洗盤（價平量縮）
  • ❌ 大幅度價跌量縮（可能續跌）

4. 價跌量增

是什麼：股價下跌但成交量放大。多頭趨勢中最危險的訊號。

怎麼判斷：

  • 高檔出現 = 主力出貨、多頭獲利了結
  • 連續下跌後突然量增 = 可能有人進場承接（底部訊號）

實例：

  • 情境：股價從 80 元漲到 150 元，某天跌到 140 元，成交量爆出 8,000 張（日均 2,000 張）
  • 操作：先出場觀望，等盤整結束再重新進場
  • 底部訊號：已跌三個月，某天量增跌到 60 元 → 可能是底部承接

適用時機：

  • ❌ 多頭高檔價跌量增（強烈賣訊）
  • ✅ 連續下跌後價跌量增（注意底部承接）

5. 價平量增

是什麼：股價持平但成交量放大。通常有人在收購籌碼或偷偷出貨。

怎麼判斷：

  • 低檔價平量增 = 主力吸籌，待表態後上漲
  • 高檔價平量增 + 多根上影線 = 主力偷偷出貨

實例：

  • 情境：股價在 50 元盤整，日均量 500 張，連續幾天量增到 2,000 張但股價沒動
  • 判讀：有人在吸貨，可提前佈局
  • 危險：高檔盤整 + 上影線 + 量增 → 出貨中

適用時機：

  • ✅ 低檔價平量增（主力吸籌）
  • ❌ 高檔價平量增 + 上影線（出貨訊號）

6. 價平量縮

是什麼：股價持平，成交量萎縮。通常出現在急漲後的籌碼沉澱。

怎麼判斷：

  • 急漲後價平量縮 = 主力洗盤、等待散戶離場
  • 盤整結束時量縮 = 即將變盤

實例：

  • 情境：股價從 60 元急漲到 90 元，接著在 85-90 元盤整兩週，成交量從 5,000 張縮到 1,000 張
  • 判讀：籌碼沉澱中，等突破再進場
  • 操作：突破 90 元 + 量增 → 追價

適用時機：

  • ✅ 急漲後的籌碼沉澱（等待突破）
  • ✅ 盤整期末的量縮（變盤前兆）

實戰操作要訣

買點判斷

1. 量縮回檔買進：多頭趨勢中，股價拉回量縮時買進
2. 價平量增關注：低檔盤整時量增價平，可提前佈局
3. 突破量增確認：突破壓力區 + 價漲量增 = 追價

賣點判斷

1. 高檔價跌量增：多頭趨勢最危險訊號，優先出場
2. 高檔異常大量：爆量長黑K棒 = 主力出貨
3. 量價背離：價漲量縮 + K線轉弱 = 趨勢反轉

每日監控重點

異常訊號警覺：

  • 價跌量增出現在多頭趨勢
  • 高檔爆量長黑K棒
  • 價漲量縮但紅K越來越弱

心得

量價關係是技術分析的基礎中的基礎，但也是最實用的工具之一。今天學到最重要的一個觀念是：量縮上漲和量價背離的區分——關鍵在K線強弱。同樣是價漲量縮，K線強勢就是籌碼鎖定、K線變弱就是追價無力，這個差別決定了是要追還是要逃。

另一個重點是「異常訊號」的監控。多頭趨勢中，價跌量增是最危險的訊號，代表主力在出貨。以前只知道「價跌量縮」是回檔，現在知道要同時看「回檔幅度」來判斷是洗盤還是真的沒人要。

量價關係不是單獨使用的，要搭配：
1. K 線型態（長黑、長紅、上下影線）
2. 價格位置（高檔、低檔）
3. 趨勢方向（多頭、空頭）

最後，「量比價先行」這句話要記住——觀察量的變化，往往能提前發現趨勢轉折。操作策略上，量縮時買進比較不容易套牢，量增時買進如果遇到量縮拉回就會套牢好幾天。這個節奏感要培養。
