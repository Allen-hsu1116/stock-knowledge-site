核心知識

融資（向券商借錢買股）

是什麼：看好股票上漲但資金不足時，向券商借錢買股，用槓桿放大獲利。

怎麼做：
1. 開立信用交易帳戶（需證券戶滿3個月、年滿20歲、成交筆數滿10筆）
2. 在下單軟體選擇「融資買進」
3. 自備4成資金（上市股）或5成資金（上櫃股），券商借6成或5成
4. 設好停損點，避免斷頭

實例：

  • 情境：台積電股價100元，想買1張
  • 操作：自備4萬元，券商借6萬元，共10萬元買進1張
  • 槓桿效果：2.5倍槓桿，漲10%賺1萬（報酬率25%），跌10%虧1萬（虧損率25%）
  • 風險：跌到78元以下時維持率低於130%，會被追繳

適用時機：

  • 短期看好、想放大獲利
  • 資金不足但看好標的
  • 不適合長期持有（年利率6-7%很高）

融券（向券商借股票放空）

是什麼：看壞股票下跌時，向券商借股票賣出，等跌了再買回還券賺差價。

怎麼做：
1. 開立信用交易帳戶
2. 在下單軟體選擇「融券賣出」
3. 準備9成保證金（融券賣出金額的90%）
4. 注意強制回補日（股東會、除權息前）
5. 設好停損，避免被軋空

實例：

  • 情境：某股股價100元，看壞會跌
  • 操作：付9萬元保證金，融券賣出1張（10萬元）
  • 獲利計算：跌到80元時買回還券，賺2萬（10萬-8萬）
  • 手續費：借券費0.8‰（賣出金額的0.08%）

適用時機：

  • 看壞某股票會下跌
  • 注意：成交量太低的股票借不到券
  • 避開強制回補日前後

維持率與斷頭風險

是什麼：券商控管風險的指標，低於130%會被追繳、斷頭。

怎麼算：

  • 融資維持率 = 股票市值 ÷ 融資金額
  • 融券維持率 = (擔保品 + 保證金) ÷ 股票市值
  • 整戶維持率 = (融資股票市值 + 融券擔保品 + 保證金) ÷ (融資金額 + 融券股票市值)

實例：

  • 情境：融資買進100元股票1張，融資6萬、自備4萬
  • 初始維持率：10萬÷6萬 = 166%
  • 危險區：跌到78元，維持率 = 7.8萬÷6萬 = 130%（開始追繳）
  • 斷頭：T日維持率低於130% → T+2日收盤前補錢 → T+3日開盤強制賣出

如何增加維持率：
1. 償還融資金（直接還錢）
2. 增加擔保品（轉入其他股票）
3. 信用部位了結（賣出部分持股）
4. 新增信用部位（買低波動股票稀釋比重，風險高）

軋空風險（融券最怕的情況）

是什麼：放空後股票大漲，被迫高價買回還券，虧損放大。

發生條件：

  • 融券餘額高（空單多）
  • 股票流動性差（買不到券）
  • 接近強制回補日（被迫買回）

實例：

  • 情境：融券賣出100元股票1張
  • 軋空發生：股票暴漲到150元
  • 虧損：強制買回還券，虧5萬（150-100）×1000股
  • 還要加計手續費、借券費

避險方式：

  • 放空前確認融券餘額不能太高
  • 設嚴格停損（虧損超過X%就回補）
  • 避開接近強制回補日的時間點

心得

融資融券是放大風險和報酬的雙面刃。重點收穫：

1. 維持率130%是生死線 — 低於這條線就會被追繳，2天內不補錢就被斷頭。這解釋了為什麼很多人融資買股，股價大跌時會被強制出場。

2. 融資的隱形成本很高 — 年利率6-7%，長期持有利息會吃掉獲利。短線操作才適合。

3. 融券風險比融資更難控制 — 股價下跌有限、上漲無限，放空的虧損沒有天花板。而且還有強制回補的時間壓力。

4. 實務操作建議：

  • 融資：用在短期看好、流動性高的股票

這次學到的是「工具的使用方式」，但更重要的是「風險控制」。融資融券不是不能碰，而是要知道自己在玩什麼。新手不建議碰，先搞懂再說。
