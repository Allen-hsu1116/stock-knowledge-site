一、錨定效應

是什麼

人類做決策時，會過度依賴「第一個看到的數字」或「容易取得的參考點」，即使這個數字跟決策毫無關聯。

實驗證明：

  • 題目：1×2×3×4×5×6×7×8 = ?（正確答案 40,320）
  • 從小數字開始算，受測者中位數猜 512
  • 題目倒過來 8×7×6×5×4×3×2×1，中位數猜 2,250
  • 同一個答案，因為起點不同，猜測差了 4 倍

投資中的三大陷阱

陷阱一：買進價格變成心魔

怎麼發生：

  • 買進價格 100 元，跌到 80 元 → 不賣（不甘心）
  • 跌到 60 元 → 繼續抱（更不甘心）
  • 漲回 98 元 → 趕快賣（終於解套！）

問題：決策應該看「公司未來前景」，不是「我買多少錢」。

實例：

  • 特斯拉從 1,000 美元跌到 600 多，一堆人覺得便宜去抄底
  • 特斯拉一天從 563 漲到 673（漲 19%），沒上車的覺得「太貴」不敢追
  • 同樣是 600 多，心態完全不同 — 這就是錨定效應

陷阱二：52 週高低點的迷思

研究發現（George et al., 2004）：

  • 買進「接近 52 週高點」的股票 → 會賺錢
  • 賣出「遠離 52 週高點」的股票 → 會賺錢

為什麼跟直覺相反：

  • 一般人覺得「接近高點風險高」、「遠離高點比較便宜」
  • 因為大家都這樣想 → 接近高點時買盤縮手、賣壓出籠
  • 有好消息時，股價沒辦法一次漲到位
  • 消息確實發酵後，股價持續上漲 → 反而產生正報酬

陷阱三：歷史價格框架

台股案例：

  • 台股 1990 年 2 月 12 日歷史高點 12,682 點
  • 跌了 70%，三十年後 2020 年 7 月才突破
  • 這個數字三十年來被媒體不斷提起
  • 從經濟學角度：這數字完全沒意義（指數不含股息、無法跨期比較）
  • 但投資人會把這當「錨點」→ 接近時就不敢買

如何克服錨定效應

方法一：反向思考

怎麼做：
1. 看到一個價格，先暫停
2. 問自己：「為什麼這個價格可能是錯的？」
3. 列出至少 3 個「反向理由」
4. 再做決定

實例：

  • 看到股價 52 週新高，想「太貴不買」
  • 反向思考：為什麼會創新高？有什麼利多？法人怎麼看？
  • 可能發現：創新高是因為基本面轉好

方法二：多面向資訊來源

怎麼做：

  • 不要只看價格
  • 加入基本面（營收、獲利、產業前景）
  • 加入籌碼面（法人買賣、融資券）
  • 加入技術面（均線、量價關係）

實例：

  • 歷史低點 50 元，現在 55 元
  • 錨定效應：「接近歷史低點，好便宜」
  • 檢查基本面：營收連續三季下滑、毛利率下降
  • 結論：便宜有原因，不要買

方法三：建立多元策略

怎麼做：

  • 不要把所有資金壓在單一策略
  • 區間交易 + 順勢交易 + 價值投資
  • 不同行情用不同策略

二、心理帳戶偏誤

是什麼

諾貝爾經濟學獎得主 Richard Thaler 提出：人們會把錢分到不同的「心理帳戶」，對待相同金錢卻用不同態度。

核心觀念：「錢就是錢，不管來源或用途，價值都一樣」

三個生活實例

實例一：意外之財

情境：

  • 小明年終獎金平常 20,000 元，今年拿到 1,000,000 元
  • 決定拿 800,000 買車（不在 5 年規劃內）
  • 心態：「剩下 200,000 還是比平常多十倍」

問題：憑運氣得來的財富，容易憑實力輸掉。

實例二：演唱會門票實驗

情境 A：到會場發現口袋 5,000 元現金不見了

  • 88% 選擇「再買票看演唱會」

情境 B：到會場發現 5,000 元門票不見了

  • 只有 46% 選擇「再買票」

分析：

  • 兩種情況都是「損失 5,000 元價值」
  • 但情境 B 的人覺得「演唱會帳戶已經用掉了」
  • 同樣的錢，因為分在不同帳戶，決策完全不同

實例三：手機換購

策略 A：原價 20,000，優惠價 16,000（保留舊機）

  • 44% 選擇

策略 B：原價 20,000，舊機折抵 4,000（交出舊機）

  • 56% 選擇

分析：

  • 兩個方案實際上一樣（都花 16,000，都沒有舊機）
  • 但策略 B 讓人覺得「舊手機有價值」，罪惡感較低

投資中的心理帳戶

案例：0050 獲利挪去賠錢

情境：

  • X 先生大量持有 0050 ETF，獲利不少
  • 覺得某個股超跌，是「價值投資」好機會
  • 把 0050 的 20% 挪去買個股
  • 個股跌 10%，X 先生說：「這是 0050 賺的錢，賠掉沒關係」

問題：

  • 錢就是錢，賠 10% 就是賠 10%
  • 心理帳戶只是增加「虧損容忍度」，不是真的沒賠

心理帳戶的正確用法

好的用法：專款專用

怎麼做：

  • 把收入分成：生活費、儲蓄、投資、娛樂
  • 「投資帳戶」的錢不輕易動用
  • 「娛樂帳戶」賠掉不心痛

實例：

  • 年終獎金 100,000 元
  • 50,000 進「投資帳戶」→ 長期持有
  • 30,000 進「旅遊帳戶」→ 可以花掉
  • 20,000 進「緊急預備金」→ 不動用

壞的用法：虧損合理化

要避免：

  • 「這筆錢本來就是賭博用的」→ 風險失控
  • 「這是 0050 賺的」→ 賠掉沒關係
  • 「這筆錢放著也是放著」→ 隨便亂買

四、實戰應用

錨定效應實戰檢核表

買進前問自己：

  • [ ] 我是否因為「買進價」而捨不得賣？
  • [ ] 我是否因為「52 週新高」而不敢買？
  • [ ] 我是否因為「歷史低點」而覺得便宜？
  • [ ] 我是否只看價格，沒看基本面？

破解方法：

  • 買進價 ≠ 公司價值
  • 52 週新高 ≠ 風險高（可能動能續漲）
  • 歷史低點 ≠ 便宜（可能公司有問題）

心理帳戶實戰檢核表

投資決策前問自己：

  • [ ] 我是否把這筆錢標記為「可以賠掉的錢」？
  • [ ] 我是否因為「這是獲利來的」而風險更高？
  • [ ] 我是否用不同標準對待不同來源的錢？

破解方法：

  • 所有資金統一管理
  • 計算整體風險係數
  • 用同樣的紀律對待每一筆錢

心得

今天學的是投資心理學的兩大核心偏誤，這些東西書本上寫得很清楚，但實際操作時超難避免。

錨定效應最狠的地方在於：你根本不知道自己被錨定了。買進價格、52 週高低點、歷史價格，這些數字就像隱形的錨，把你綁住。我以前也常犯「解套就想賣」的錯誤，覺得「終於回本了」，結果賣完股價就噴上去。現在回頭看，就是被買進價錨定，沒去想公司未來會怎樣。

心理帳戶也很有意思。把錢分成「投資用的」、「玩玩的」、「賺來的」、「存的」，聽起來很合理。但問題是「0050 賺的錢賠掉沒關係」這種心態，實際上就是讓自己對虧損更寬容，反而更容易賠錢。

關鍵學習：
1. 錨定效應會讓你不敢買在高點、不敢賣在低點 → 反向思考、多面向分析
2. 心理帳戶會讓你對不同來源的錢用不同標準 → 統一管理、同樣紀律
3. 12 種偏誤不是獨立的，會互相影響 → 定期檢視自己的決策

這些偏誤是人性的弱點，知道不等於能做到。但至少，下次做決策時，我會問自己：「我是不是被錨住了？」「我是不是在玩心理帳戶的遊戲？」
