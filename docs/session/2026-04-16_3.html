一、當沖八大步驟建立短線策略

1. 步驟一：選定短線交易模式

是什麼：根據你的性格、資金和時間，選擇適合的當沖操作模式。

三種模式比較：

模式賺賠比出手次數資金效率操作核心是否留倉
Tick流低多高短進短出否
一波流中中中當前行情走勢否
日內波高少低盤前選股可

怎麼做：
1. Tick流：捕捉股價微小波動，每次進出場間隔數秒到數分鐘，平均獲利1%以下，積小成大。適合頻繁震盪、成交量活躍的標的。
2. 一波流：掌握當日震盪最大的短期趨勢，單筆獲利1%~5%。須追蹤成交量變化與大單流向，操作熱門族群與題材股。
3. 日內波：研判個股有機會出現長紅/長黑K棒才進場，單筆獲利5%以上，甚至可轉為波段持有。非常注重選股策略。

實例：

  • Tick流：台積電（2330）在9:00-9:15震盪區間來回，每跳1個Tick賺0.5元，一日操作20次小賺積沙成塔
  • 一波流：AI概念股開盤帶量突破昨日高點，追進後抱到波段高點出場，單筆賺3%
  • 日內波：凱美（2375）股價創波段新高，早盤進場後收漲停，隔日可選擇繼續持有賺波段

適用時機：

  • Tick流 → 盤勢震盪、量能充沛的個股
  • 一波流 → 有明確題材驅動的熱門股
  • 日內波 → 盤前已做足功課、基本面技術面籌碼面三面共振

2. 步驟二：制定選股策略

是什麼：用系統化方法在開盤前鎖定當日操作標的，避免盤中盲目追價。

怎麼做：
1. 事件關聯法：找出政策發布、產業趨勢變化、經濟指標、財報公布等事件與特定類股漲跌的關聯，未來類似事件發生時快速反應
2. 動能篩選法：開盤後即時篩選波動幅度5%以上的股票進行觀察，選擇強勢標的進行短線操作
3. 多維度條件組合：綜合周轉率排名、成交量排名、法人買賣超等條件，篩選人氣陡升、資金流動性高的標的

實例：

  • 美國通過AI晶片補貼政策 → 隔日台股AI晶片族群（如創意、力旺）成為當沖熱門標的
  • 開盤15分鐘篩選漲幅>5%且成交量>5日均量2倍的股票 → 鎖定當日強勢股

適用時機：每日盤前必做，是當沖成功的第一道關卡

3. 步驟三：瞭解日內交易各區段特點

是什麼：台股盤中不同時段有不同特性，順勢而為才能將時機轉為獲利契機。

怎麼做：
1. 開盤前（8:45-9:00）：觀察美股收盤、海外期貨、日韓股開盤、台指期開盤，決定偏多或偏空操作
2. 開盤後（9:00-9:30）：市場劇烈反應消息，短線交易者的「快攻時間」，強勢股會在此時搶攻漲停
3. 早盤（9:30-10:30）：走勢逐漸明朗，技術型態提供操作方向依據，是較穩定的進場時機
4. 中盤（10:30-12:30）：成交量下降、波動縮小，宜減少出手次數
5. 尾盤（12:30-13:30）：當沖者準備出場、波段者可能進場，成交量擴增，注意拉尾盤或下殺

實例：

  • 9:00開盤某AI股帶量跳空漲5% → 快攻時間果斷進場
  • 10:30後盤面縮量橫盤 → 此時進場容易被洗，等待尾盤方向
  • 13:00大單敲進權值股拉尾盤 → 留意主力作價意圖

適用時機：每日交易時段選擇，不同時段用不同策略

4. 步驟四：短線交易技術指標實戰

是什麼：用技術指標應對快速變化的價格走勢，提高進出場判斷準確度。

五大關鍵指標：

怎麼做：
1. 當日開盤價：股價在開盤價上方=紅K多方佔優，在下方=黑K空方佔優。強勢股會迅速脫離開盤價。
2. 當日均價線（VWAP）：當天平均交易成本，價格維持在上方=多頭趨勢明確逢低買進，維持在下方=空頭優勢逢彈放空。
3. 日K線關鍵點位：近期高低點、昨收等易形成支撐壓力，突破/跌破後易產生趨勢。
4. 日K線技術指標：上揚均線易成支撐，下彎均線易成壓力，避免追高殺低。
5. VWAP（進階武器）：法人看的是VWAP而非MA。股價回測VWAP有守=法人護盤訊號；放量跌破VWAP=主力出貨警訊。

實例：

  • 凱美（2375）開盤後量增突破昨日高點 → 進場；或等拉回不破均價線再布局
  • 聯光通（4903）跳空開高遇月線反壓，攻高無力跌落均價線，量能未放大 → 不適合作多，可逢彈放空

適用時機：盤中即時判斷，搭配選股結果使用

5. 步驟五至八：勝率賺賠比、交易成本、風險控制、交易檢討

勝率與賺賠比：

  • 期望值 = 賺賠比 × 勝率 - 賠率
  • 勝率40%、賺賠比3 → 期望值 = 3×0.4 - 1×0.6 = 0.6（正期望值，長期可獲利）

交易成本：

  • 手續費買賣各0.1425%（可議價）+ 當沖證交稅0.15%（賣出時）
  • 100~150元區間股票1 Tick跳0.5元，手續費有優惠時1 Tick就能打平成本

風險控制：

  • 明確最大可接受虧損（如總資金10%）
  • 虧損接近上限立即停止交易
  • 結合技術指標或自動停損減少主觀情緒

交易檢討：

  • 每天整理交易記錄（進出場原因、判斷依據、結果）
  • 每周/月回顧策略是否仍適用
  • 參與交易社群補足盲點

二、高階盤感訓練：辨識當日「盤勢屬性」與對應策略

1. 趨勢盤（單邊大漲或大跌）

是什麼：開盤後股價一路向上或向下，幾乎沒有太深回檔，均線呈現完美發散排列，成交量穩定放大。

怎麼做：
1. 確認今日是趨勢盤（開盤30分鐘走勢方向明確、量價齊揚）
2. 股價回測日內均價線（VWAP）有撐 → 最佳加碼或上車點
3. 停利點放寬，讓利潤奔跑，直到尾盤趨勢改變或跌破均價線才出場
4. 切忌「猜頭摸底」（漲多去放空、跌多去接刀）

實例：

  • 某半導體股9:00開盤帶量跳空+3%，9:30回測VWAP不破 → 加碼進場，抱到13:00收盤前出場，單筆賺8%

適用時機：大盤強勢或個股有重大利多/利空發酵時

不要用：盤勢不明、量能不足時硬做趨勢盤策略會被雙巴

2. 震盪盤（區間洗盤）

是什麼：股價在特定上下區間內來回震盪，均線糾結，突破前高馬上軟腳，跌破前低又瞬間拉起。

怎麼做：
1. 確認今日是震盪盤（開盤30分鐘內價格來回穿越均價線）
2. 運用布林通道或早盤支撐壓力區間，找出區間上下緣
3. 打到壓力區試單做空，回測支撐區試單做多（逆勢打法）
4. 獲利目標縮小，有賺就走，絕不貪心戀戰

實例：

  • 某電子股在105-110元區間震盪 → 110做空、105做多，每次賺2-3個Tick就出場

適用時機：大盤狹幅震盪、缺乏明確方向時

不要用：當突破已確認為趨勢盤時仍用震盪盤策略會被單邊軋出去

3. 誘多/誘空盤（極端洗盤反轉）

是什麼：早盤看似強勢帶量突破，吸引散戶追價後主力反手倒貨；或早盤急殺破底後又直線拉抬過高。殺傷力最強的盤勢。

怎麼做：
1. 發現追高突破卻瞬間跌回關鍵頸線以下，且出現爆量長黑 → 立刻砍單停損
2. 進階老手可順著主力反轉方向反手操作
3. 新手只要果斷停損不凹單、甚至空手觀望，就贏過80%散戶

實例：

  • 某股開盤帶量突破前高120元，散戶追進後主力反手倒貨，5分鐘跌回115元 → 在119元認賠出場，虧損1元/tick而非死抱到跌停

適用時機：每天都要有防範意識，特別是開盤30分鐘

不要用：新手不要在誘多誘空盤中嘗試反手操作，光是停損就夠了

4. 開盤前30分鐘的「觀盤藝術」

是什麼：9:00-9:30是主力籌碼交換最劇烈的時間，正確觀盤可以判斷當日盤勢屬性。

怎麼做：
1. 新手在這段時間「多看少做」
2. 觀察大盤預估成交量、權值股動向、觀察名單的委買委賣力道
3. 確認今天傾向哪一種盤勢後，9:30之後再選擇適合的策略進場
4. 避免一開盤就衝動耗盡子彈

實例：

  • 9:00-9:30觀察到三大法人大單敲進AI股，VWAP穩定上移 → 判定趨勢盤，9:30後順勢進場做多
  • 9:00-9:30觀察到個股在均價線上下來回穿越 → 判定震盪盤，採取高出低進策略

適用時機：每天開盤必做，是決定當日操作策略的關鍵

心得

今天學的是當沖的系統化架構，從八大步驟到盤勢屬性辨識，核心觀念是「不要用同一套策略打所有的盤」。很多人當沖失敗不是因為不懂技術分析，而是因為不管今天什麼盤都同一套打法——趨勢盤用震盪策略（太早下車）、震盪盤用趨勢策略（被雙巴）、誘多盤凹單不止損。

最值得記住的是開盤前30分鐘的觀盤藝術：多看少做，先判斷盤勢再進場。這跟以前學到的「固定時間出場策略」和「交易冷卻時間」概念一致——給自己冷靜判斷的空間，比急著出手更重要。

另外VWAP比MA更適合當沖判斷法人動向，這個概念要記住：法人看的是VWAP，散戶看的是MA，用法人視角看盤才能減少被洗盤出場的次數。
