一、多時間框架分析（MTF）核心概念

1. 三重時間框架分析法

是什麼：用三個不同時間週期分析同一檔商品，大週期看位置、中週期判斷趨勢、小週期尋找進場點，是專業交易者最常用的分析框架。

怎麼做：
1. 大週期看位置：用最長的時間框架判斷市場目前所處的大環境（多頭、空頭、盤整），決定偏多或偏空
2. 中週期判斷趨勢：用中間的時間框架確認趨勢方向，找出支撐壓力位和可能的轉折點
3. 小週期找進場點：用最短的時間框架精確定位進出場點，設定停損停利

常用時間框架組合：

  • 波段交易者：月→週→日
  • 日內交易者：日→60分→15分
  • 當沖交易者：60分→15分→5分（或1分）

實例：

  • 大週期（日K）：台積電日K線在均線多頭排列，趨勢偏多 → 決定只做多
  • 中週期（60分K）：60分K線出現回測支撐且有看漲K棒 → 確認進場區域
  • 小週期（15分K）：15分K線出現突破前高且量能放大 → 精確進場點

適用時機：每次交易前必做，從大週期看到小週期，三層確認後才進場

2. 為什麼要使用多時間框架？

是什麼：單看一個時間框架容易被雜訊誤導，多時間框架能提供宏觀視角，避免「只見樹木不見森林」。

怎麼做：

優點：
1. 拉高盈虧比：用大週期的趨勢方向判斷，用小週期的停損點進場，停損小但目標看大週期的幅度，盈虧比可達5:1以上
2. 更清晰判斷趨勢：15分K線看似多頭，但切到4小時線可能只是大空頭中的一次反彈，多時間框架讓你看到全貌
3. 避免假訊號：小週期的突破若沒有大週期趨勢支持，假突破機率高

缺點與風險：
1. 大中週期判斷錯誤：如果大週期趨勢判斷錯，小週期進場也會連續停損
2. 小週期雜訊過多：1分K線雜訊非常多，容易被假訊號洗出場
3. 時間框架相悖：大週期偏多但小週期偏空時，容易猶豫不決

實例：

  • 好例子：4小時K線突破前高回踩確認支撐 → 15分K線出現反轉K棒 → 1分K線精確進場，停損設在1分K的支撐下方，目標看4小時級別的幅度，盈虧比15:1
  • 壞例子：只看15分K線覺得多頭要進場，但4小時K線是空頭趨勢的反彈 → 進場後被大趨勢壓回去

適用時機：所有交易風格都適用，波段交易者用日/週/月，當沖交易者用60分/15分/5分

3. 柴豬三重框架實戰案例

是什麼：交易者柴豬的實戰方法，用4小時→15分→1分的三重框架做日內交易，盈虧比可達15:1到30:1。

怎麼做：
1. 4小時K線（大週期）：觀察趨勢方向，找出突破的關鍵位置和供需區
2. 15分鐘K線（中週期）：確認趨勢延續或反轉，找出多空轉換點和供需區
3. 1分鐘K線（小週期）：優化供需區位置，精確掛單進場，停損設在供需區下方

進場步驟：
1. 大週期確認方向（4H突破前高+回踩確認 → 偏多）
2. 中週期找進場區域（15M出現多空轉換+供需區）
3. 小週期精確進場（1M優化供需區+掛單）
4. 停損設在供需區下方（加點差）
5. 第一目標設在上方供給區（盈虧比15:1），第二目標設在兩倍趨勢延續（盈虧比30:1）

實例：

  • 歐美4小時K線突破強高點，回踩起漲點收看漲PB → 大週期偏多
  • 15分K線出現多空轉換，下方有未緩解的供需區 → 中週期找進場區
  • 1分K線優化供需區，掛單進場 → 停損在供需區下方，目標盈虧比15:1
  • 結果：回踩供需區後一路上漲，第一目標平一半倉位，第二目標全平，賺15倍風險金額

適用時機：日內交易和外匯交易；台股可改用日K→60分K→15分K的框架

二、短線K線週期選擇與盤型辨識

4. K線週期選擇原則

是什麼：不同的交易風格對應不同的K線週期，選錯週期會讓你看到不正確的訊號。

怎麼做：

交易風格建議週期特點
當沖5分K以下（1分K、5分K）訊號多、雜訊也多，需要快速反應
波段30分K以上（60分K、日K）訊號少但穩定，不需要盯盤
中段15分K介於當沖和波段之間
趨勢判斷日K一定要看，決定偏多偏空

關鍵原則：
1. 趨勢判斷永遠先看大週期：不管做什麼風格，先看日K判斷趨勢方向，再到小週期找進場點
2. K線週期越大越看不出型態：30分K以上的型態會被壓縮，看不出短期轉折
3. 5分K以下才看得到短線型態：一底比一底高、頭頭低這些型態在5分K才清楚

實例：

  • 當沖台指期：5分K看型態 + 1分K精確進場 + 日K判斷大方向
  • 波段操作台股：日K判斷趨勢 + 60分K找支撐壓力 + 15分K精確進場

適用時機：每次開盤前先看大週期，盤中用小週期操作

5. 三種基本盤型辨識

是什麼：短線操作的第一步是辨識當日盤型——多方、空方、盤整，不同盤型用不同策略。

怎麼做：

多方型態：

  • 特徵：一底比一底高，且不斷創新高
  • 操作：拉回找買點，沿趨勢做多
  • 出場：跌破最後一個轉折低點（末升低），多方結束

空方型態：

  • 特徵：一頭比一頭低，且不斷創新低
  • 操作：反彈找空點，沿趨勢做空
  • 出場：站上最後一個轉折高點（末跌高），空方結束

盤整型態：

  • 特徵：高低點在一個區間內震盪，沒有明確方向
  • 操作：減少出手，或用區間交易法高拋低吸
  • 注意：盤整盤最容易雙巴，追高殺低必虧

進階盤型——竹馬盤：

  • 特徵：前段出現兩個高點像馬耳朵，中段出現馬身體
  • 含義：容易殺尾盤，下午盤要注意出場

實例：

  • 多方型態：台指期5分K線低點從17800→17850→17900不斷墊高，高點也持續創新高 → 偏多操作，拉回17850附近買進
  • 空方型態：高點從18000→17950→17900持續降低 → 偏空操作，反彈17950附近做空

適用時機：每天開盤後用5分K線辨識當日盤型，9:30後方向通常更明朗

6. 轉折高低點與進出場判斷

是什麼：短線操作中，轉折高低點（Swing High/Low）是判斷趨勢延續和結束的關鍵，也是設定停損停利的基準。

怎麼做：

轉折低點（Swing Low）的運用：
1. 多方趨勢中，轉折低點不斷墊高 → 繼續看多
2. 跌破最後一個轉折低點（末升低）→ 多方結束，出場
3. 進場做多時，停損設在最近的轉折低點下方

轉折高點（Swing High）的運用：
1. 空方趨勢中，轉折高點不斷降低 → 繼續看空
2. 站上最後一個轉折高點（末跌高）→ 空方結束，出場
3. 進場做空時，停損設在最近的轉折高點上方

移動停利機制：
1. 做多時：每出現新的轉折低點，停利點跟著上移到新轉折低下方
2. 做空時：每出現新的轉折高點，停利點跟著下移到新轉折高上方
3. 不需要盯盤也能執行：可使用條件單自動設定

實例：

  • 做多台指期在17900進場，停損設在前一個轉折低17860下方5點（17855）
  • 行情上漲到17980，出現新的轉折低17940 → 移動停利上移到17935
  • 行情繼續漲到18050，新轉折低18010 → 移動停利上移到18005
  • 最終行情跌破18010，在18005出場，獲利105點

適用時機：所有短線交易，轉折高低點是最自然的停損停利基準

三、短線交易期望值與加碼策略

7. 建立正期望值交易策略

是什麼：短線交易不追求每筆都對，而是追求長期期望值為正——勝率×賺的金額 > 賠率×賠的金額。

怎麼做：

兩種建立正期望值的方法：

方法一：提高勝率

  • 找高勝率的盤型重複做：底底高轉多、頭頭低轉空
  • 只做熟悉的盤型，不做看不懂的
  • 勝率60%、賺賠比1:1 → 期望值 = 0.6×1 - 0.4×1 = 0.2（正期望值）

方法二：拉開賺賠比

  • 每次賠的少、賺的多，即使勝率不到50%也能賺錢
  • 停損固定（如ATR的1/5），停利放大（如轉折高低點或固定點數）
  • 勝率40%、賺賠比3:1 → 期望值 = 0.4×3 - 0.6×1 = 0.6（正期望值）

實例：

  • 策略A：勝率70%，賺1賠1 → 期望值0.4
  • 策略B：勝率35%，賺3賠1 → 期望值0.05
  • 策略C：勝率40%，賺3賠1 → 期望值0.6 ← 最佳

關鍵認知：技術分析是機率的概念，沒有保證100%對的訊號。建立正期望值策略，然後重複執行，才是長期獲利的正道。

適用時機：每次制定交易計畫時，都要計算期望值，確認是正期望值才執行

8. 短線加碼策略：順勢加碼拉大獲利

是什麼：WINSMART的短線加碼策略——只在賺錢時加碼，用小風險換取大波段利潤，是拉開賺賠比的關鍵方法。

怎麼做：

加碼兩大原則：
1. 賺錢才加碼：永遠只在部位有浮盈時加碼，絕不攤平虧損
2. 堅守風控：不管加了幾口單，總風險控制在設定範圍內

加碼設定範例：
1. 初始部位：2口
2. 加碼條件：每獲利10點加1口，最多加到6口
3. 停損設定：ATR的1/5（如當日ATR=200點，停損40點）
4. 移動停利：用轉折高低點設定自動出場
5. 分批出場：獲利100點先出1口鎖定利潤，其餘用轉折點繼續抱

實戰案例：

  • 台指期夜盤出現W底型態，進場2口多單在17950
  • 行情上漲到17960（+10點），自動加碼1口
  • 繼續上漲到17970（+20點），再加碼1口
  • 總共4口多單，初始風險80點（16000元）
  • 行情一路漲到18200，跌破轉折低出場
  • 總獲利：超過15萬元
  • 賺賠比：1:9以上

適用時機：趨勢明確時使用；盤整時不加碼

四、多時間框架進場策略整合

9. MTF進場三步驟實戰SOP

是什麼：將多時間框架分析與短線K線型態結合，形成系統化的進場判斷流程。

怎麼做：

步驟一：大週期定方向（日K或4小時K）
1. 看均線排列：多頭排列（5>10>20>60）→ 偏多操作；空頭排列 → 偏空操作
2. 看趨勢線：價格在上升趨勢線上方 → 偏多；在下降趨勢線下方 → 偏空
3. 看關鍵位置：前波高低點、整數關卡、供需區位置
4. 確認方向後，只做順方向的交易

步驟二：中週期找進場區域（60分K或15分K）
1. 找出大週期供需區在中週期的對應位置
2. 確認中週期趨勢是否與大週期一致
3. 找出中週期的支撐壓力位
4. 觀察是否有反轉K棒型態（吞沒、早晨之星等）

步驟三：小週期精確進場（5分K或1分K）
1. 在中週期供需區附近，切到小週期觀察
2. 等待小週期出現確認訊號（突破前高、反轉K棒、量能放大）
3. 進場同時設定停損（設在小週期轉折低點下方）
4. 移動停利用小週期轉折高低點追蹤

實例（台股）：
1. 日K：台積電均線多頭排列，股價在5日均線上方 → 偏多
2. 60分K：股價回測595元（前波支撐），出現下影線 → 進場區域確認
3. 15分K：595元附近出現量增紅K突破前高597元 → 進場做多
4. 停損：593元（15分K轉折低下方）
5. 目標：605-610元（日K前波高點附近）

適用時機：每次進場前都要走完三步驟，不跳步

10. 多時間框架常見錯誤與修正

是什麼：使用多時間框架時最容易犯的三個錯誤，每個都可能導致連續停損。

怎麼做（避免方法）：

錯誤一：時間框架相悖時強行進場

  • 症狀：日K偏空但15分K出現反彈訊號就進場做多 → 被大趨勢壓回去
  • 修正：三個時間框架方向必須一致才進場。大週期偏空時，小週期做多只能做極短線（Tick流），不做波段

錯誤二：頻繁切換時間框架

  • 症狀：15分K做多被洗出場，切到1分K想重新進場，又被洗 → 反覆停損
  • 修正：進場前決定好用哪個時間框架操作，中途不換。如果15分K的停損被觸發，代表判斷錯了，不切到1分K凹單

錯誤三：小週期雜訊當趨勢

  • 症狀：1分K線出現一根紅K就認為趨勢反轉 → 其實只是短期波動
  • 修正：趨勢判斷用大週期，進場精確度用小週期。1分K只用来找進場點，不用來判斷趨勢方向

實例：

  • 錯誤示範：日K台指期在季線下方（偏空），但5分K出現三根紅K就進場做多 → 進場後5分鐘就被壓回，停損出場
  • 正確做法：日K偏空 → 當天只做空或不操作；如果5分K出現反彈訊號，最多做極短線（1-2個Tick），停損設極小

適用時機：每次用多時間框架分析時都要檢查這三個錯誤

心得

今天學的多時間框架分析（MTF）是操作策略的「放大鏡+望遠鏡」組合——望遠鏡看大方向，放大鏡找精確進場點。這跟之前學的所有技術分析工具都有關聯，但MTF解決的是「在哪個時間尺度上使用這些工具」的問題。

1. 三重框架是進場前的必做功課。以前我可能只看日K線就進場，或者只看5分K線就進場，兩個極端都有問題——只看日K進場點不精確，停損大；只看5分K容易被雜訊洗出場。MTF的核心是「大週期定方向，小週定進場點」，這樣停損可以設在小週期的轉折點（很小），但目標看大週期的幅度（很大），盈虧比自然拉高。

2. 柴豬的實戰案例讓我理解了盈虧比15:1是怎麼做到的。關鍵不是勝率高，而是停損小（設在1分K的供需區下方）但目標大（看4小時級別的幅度）。這跟之前學的「正期望值策略」完全一致——勝率可能只有35-40%，但賺賠比夠大，長期就是正期望值。

3. 轉折高低點是最自然的停損停利基準。之前學過很多停損方法（K線停損、均線停損、ATR停損、固定比例停損），但轉折高低點有一個獨特優勢——它是市場自己畫出來的，不是人為設定的。多方趨勢中每個轉折低點都是天然的停損/停利點，不需要猜測市場會走到哪裡，只要「不跌破轉折低就抱住」就好。

4. K線週期選擇取決於交易風格，但趨勢判斷永遠先看大週期。當沖看5分K以下，波段看30分K以上，這是共識。但不管做什麼風格，都一定要先看日K判斷大方向——這是最容易被忽略的步驟。很多人看5分K覺得多頭就進場，結果日K是空頭趨勢的反彈，進場就被壓回去。
