核心知識

一、三率的定義與公式

毛利率（Gross Profit Margin）

是什麼：每 1 元營收中，扣除直接成本後，還剩下多少錢。

公式：毛利率 =（營業收入 - 營業成本）/ 營業收入

怎麼解讀：

  • 反映產品本身的獲利能力
  • 代表公司產品或服務的競爭力
  • 高毛利率 = 成本控制好 或 產品有定價優勢

實例：

  • 台積電毛利率約 50%，代表每賣 100 元，成本 50 元，毛利 50 元
  • 鴻海毛利率約 6-7%，代工產業利潤極低（俗稱「毛三到四」）

營業利益率（Operating Profit Margin, OPM）

是什麼：每 1 元營收中，扣除直接成本 + 間接成本（營業費用）後，還剩下多少錢。

公式：營業利益率 = 營業利益 / 營業收入

  • 營業利益 = 營業收入 - 營業成本 - 營業費用
  • 營業費用 = 薪資、租金、水電、行銷費用、研發費用等

怎麼解讀：

  • 反映「本業經營能力」
  • 衡量公司控制費用、管理效率的能力
  • 營業利益率為負 = 本業賠錢，營運有困難

稅後淨利率（Net Profit Margin）

是什麼：每 1 元營收中，扣除所有成本、費用、稅金、業外損益後，真正賺到的錢。

公式：稅後淨利率 = 稅後淨利 / 營業收入

怎麼解讀：

  • 企業最終真正賺到的錢
  • 需注意「業外損益」佔比，避免被一次性收入誤導

二、毛利率 vs 營業利益率：差異判讀

核心差異

指標毛利率營業利益率
扣除項目只扣直接成本（原料、製造）扣直接成本 + 營業費用
反映能力產品競爭力、成本控制經營管理能力、費用控制
適用場景判斷產品本身是否賺錢判斷公司整體營運效率

毛利率高、營業利益率低 = 費用控制差

意義：產品本身很賺錢，但公司花太多錢在行銷、管理、研發等費用上。

怎麼判斷：
1. 計算「毛利率 - 營業利益率」的差距
2. 差距越大，代表營業費用佔比越高
3. 與同業比較，如果差距明顯高於同業，代表費用控制不佳

實例：

  • A 公司：毛利率 40%，營業利益率 5%
  • B 公司：毛利率 30%，營業利益率 10%
  • 解讀：A 公司產品較賺錢，但 B 公司經營效率更好

投資判斷：

  • 毛利率高、營業利益率低 → 公司可能處於擴張期（砸錢搶市佔），或管理效率差
  • 需進一步看「營業費用明細」，判斷是研發投資還是浪費

毛利率低、營業利益率高 = 營運效率好

意義：產品利潤薄，但公司靠精實管理擠出利潤。

怎麼判斷：

  • 毛利率低但穩定
  • 營業利益率與毛利率差距小
  • 營業費用控制得很好

實例：鴻海

  • 毛利率約 6-7%（代工利潤薄）
  • 但靠規模經濟、費用控制，營業利益率仍能維持 2-3%
  • 解讀：低成本競爭策略，靠量取勝

三、三率三升：選股指標

什麼是「三率三升」

毛利率、營業利益率、稅後淨利率三者同步上升。

意義：

  • 毛利率上升 → 產品競爭力增強或成本下降
  • 營業利益率上升 → 經營效率提升
  • 淨利率上升 → 整體獲利能力改善

怎麼用：
1. 在選股軟體篩選「連續 2-3 季三率三升」的公司
2. 排除業外收益造成的淨利率上升（看業外損益佔比）
3. 與同業比較，找出產業中表現最穩定的公司

解讀：三率皆較前一季及去年同期上升，代表產品組合優化、費用控制改善。

四、營業利益率選股三種方法

方法一：穩定性篩選

怎麼做：最近 6 個月（或 2 季）營業利益率皆大於特定百分比（如 10-15%）。

適用時機：想找營運穩定的公司，避開暴起暴落的股票。

實例：

  • 設定條件：連續 4 季營業利益率 > 10%
  • 結果：找出經營穩定、本業賺錢的公司

方法二：同業比較

怎麼做：在同一產業中，挑選營業利益率較高的公司。

適用時機：已經鎖定某產業，想找出該產業中經營能力最好的公司。

注意：

  • 不同產業營業利益率差異很大
  • 不能拿台積電（IC設計）跟鴻海（代工）比
  • 一定要「同產業比較」才有意義

方法三：搭配毛利率、淨利率一起看

怎麼做：三率一起分析，避免單一指標失真。

檢查要點：
1. 毛利率趨勢：產品競爭力
2. 營業利益率趨勢：經營效率
3. 淨利率趨勢：整體獲利能力

紅旗警示：

  • 毛利率穩定，但營業利益率下降 → 費用失控
  • 營業利益率穩定，但淨利率暴增 → 業外收益（可能一次性）
  • 三率波動劇烈 → 獲利不穩定，風險較高

  • 營業利益率超過 100% 可能是計算失真（營業利益或營業費用為負）
  • 觀察連續性比看單季數值更重要

心得

以前看財報只會看 EPS 和淨利率，沒想過毛利率和營業利益率之間的差距可以這麼有用。「毛營差」其實就是營業費用佔比，差距越大代表公司砸越多錢在行銷、管理、研發。

兩個判斷方向：
1. 高毛利低營益率 → 可能是成長期公司砸錢搶市佔，或管理效率差
2. 低毛利高營益率 → 精實管理、規模經濟，像鴻海這種代工模式

「三率三升」是個好用的選股指標，代表公司獲利能力全面改善。但要小心業外收益造成的假象，淨利率突然飆高要先查業外損益來源。

之後看財報要多注意毛利率和營業利益率的差距變化，看看公司是不是費用失控，還是在投資未來。
