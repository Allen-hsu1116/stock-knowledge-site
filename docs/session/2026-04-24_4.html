停利策略：用「出場」思考代替「停利」思考

停利是什麼？

停利是設置一個價格當作停利點，當價格漲到停利點時就賣出，目的是鎖定報酬、落袋為安。

三種停利方法

1. 固定百分比停利法

是什麼：設定一個固定百分比（如10%、20%、30%），獲利達到就賣出。

怎麼做：
1. 進場時同時設定停利百分比
2. 當股價漲到目標百分比，系統自動或手動賣出
3. 百分比根據個人風險承受度調整

實例：

  • 在100元買進A公司，設定20%停利
  • 股價漲到120元時賣出，獲利20%

常見錯誤：用「固定金額」停利（如漲10元就停利）是錯的！

  • 加權指數8000點時漲5%=400點
  • 加權指數15000點時漲5%=750點
  • 如果都用漲400點停利，等於在8000點漲5%停利，在15000點漲2.67%就停利了
  • 必須用百分比，不能用金額

適用時機：短線交易、有明確獲利目標的策略。不適合長期成長股。

2. 移動停利法（Trailing Stop）

是什麼：停利點會跟著股價上升而自動上移，但股價下跌時停利點不動，回檔到設定比例就出場。

怎麼做：
1. 進場後設定回檔百分比（如10%）
2. 股價上漲時，停利點 = 最高價 × (1 - 回檔%)
3. 股價從高點回檔到停利點時，自動賣出
4. 回檔百分比根據個股波動度調整

實例：

  • 在100元買進，設定10%移動停利
  • 股價漲到120元，停利點 = 120 × 0.9 = 108元
  • 股價從120元下跌到108元時賣出，仍獲利8%
  • 如果股價繼續漲到150元，停利點 = 150 × 0.9 = 135元

適用時機：趨勢明確的波段行情。不適合震盪盤（容易被洗出場）。

3. 價值分析停利法（出場策略）

是什麼：根據股票的估值來決定出場時機，而不是根據自己的損益。

怎麼做：
1. 進場時計算股票的合理價值（本益比、股價淨值比、DCF等）
2. 低於價值時買進，回到合理價或超過合理價時賣出
3. 出場判斷基於「進場理由是否消失」，而非「賺了多少」

實例：

  • A公司本益比8倍時買進（極度低估），合理本益比15倍
  • 當本益比回到15倍時賣出，不管賺了30%還是50%
  • 如果進場理由是「下季財報優於預期」，那財報出來沒優於預期就要出場

適用時機：價值投資、中長線操作。不適合短線或無法估值的標的。

停利的關鍵洞見：用「出場」思考代替「停利」

核心觀念：

  • 傳統停利只看自己的損益，但市場不會因為你想賺10%就只漲10%
  • 回測數據顯示：加上停利後，勝率微升但盈虧比下降，整體報酬往往下降
  • 正確做法：問自己三個問題

1. 當初買進的理由是什麼？ → 這是你的目標和方向
2. 進場的理由合理嗎？ → 必須容許邏輯反例的存在，可以被驗證
3. 進場的理由消失了嗎？ → 理由消失就出場，不管賺賠

不合理進場理由（無法驗證反例）：

  • ❌ 朋友推薦 → 朋友不推薦了才算？
  • ❌ 最近漲很多 → 不漲了就算？
  • ❌ 新聞說有趨勢 → 新聞不報了就算？

合理進場理由（可驗證反例）：

  • ✅ 下季財報優於預期 → 財報沒優於預期就出場
  • ✅ 公司價值遠低於價格 → 價格超過價值或價值下降就出場
  • ✅ 經濟趨勢長期向上 → 趨勢改變就出場

風險平價（Risk Parity）：平均風險而非平均資金

風險平價是什麼？

風險平價是一種資產配置策略，核心思想是「平均風險，而非平均資金」。波動大的資產部位放小，波動小的資產部位放大，讓每類資產對整體組合的風險貢獻相等。

為什麼傳統配置有問題？

傳統60/40股債配置的陷阱：

  • 美股波動率約15%，債券約5%
  • 雖然資金60%在股票、40%在債券
  • 但股票對組合波動的貢獻高達90%以上！
  • 本質上等於「90%的風險來自60%的部位」
  • 風險高度集中在股票上，根本沒有分散到

數學驗證：

  • 股票波動2.5%，債券0.5%，相關係數0.3
  • 60/40組合波動 = 2.47%
  • 股票風險貢獻 = 94.7%，債券 = 5.3%
  • 差距懸殊！

風險平價怎麼做？

簡易版：反波動率配置

怎麼做：
1. 計算每個資產的歷史波動率（標準差）
2. 部位權重 ∝ 1/波動率（波動越大，部位越小）
3. 歸一化讓權重加總 = 100%

實例：

  • 特斯拉年化波動率40%，通用汽車年化波動率20%
  • 反波動率權重：特斯拉 = 1/40 = 0.025，通用 = 1/20 = 0.05
  • 歸一化：特斯拉 = 0.025/0.075 = 33%，通用 = 0.05/0.075 = 67%
  • 雖然通用佔了2/3資金，但兩者對組合的風險貢獻相等

進階版：等風險貢獻（ERC）

怎麼做：
1. 建立協方差矩陣Σ（考慮資產間的相關性）
2. 設定目標：每個資產的風險貢獻 wᵢ(Σw)ᵢ 相等
3. 用數值方法（如二階錐算法）求解最適權重
4. 高度相關的資產會被視為類似風險來源，自動降低配置

實例：

  • 同時持有科技股A和科技股B（高度相關），加上金融股C
  • ERC會自動降低A和B的配置，因為它們相關性高，風險來源重疊
  • 這是1/N等權重做不到的

風險平價的代價與解法

問題：低波動資產配置多 → 整體收益偏低

解法：加槓桿

  • 風險平價組合的夏普率通常比傳統配置高
  • 但絕對報酬偏低
  • 透過槓桿線性放大，可以達到目標報酬
  • 因為夏普率高，即使加槓桿，風險調整後的表現仍優於傳統配置

實例：

  • 風險平價組合年化報酬5%，波動5%，夏普率1.0
  • 傳統60/40年化報酬8%，波動12%，夏普率0.67
  • 將風險平價組合加2倍槓桿 → 年化報酬10%，波動10%，夏普率仍1.0
  • 報酬更高、風險更低

風險平價的三大限制

1. 歷史數據有效性：波動率和相關性來自歷史數據，未來可能不同
2. 利率上升環境不利：大量配置債券，升息時表現差
3. 協方差矩陣不穩定：危機時相關性趨近1，分散效果消失

個股操作的風險平價應用

風險平價不只適用於股債配置，也能用在股票組合：

  • 「偷雞要用小部位」：波動大的投機股，部位一定要小
  • 同時持有穩定股（如中華電）和投機股（如航運股）
  • 穩定股可以放較大部位，投機股放小部位
  • 讓兩者對組合的風險貢獻相近

心得

停利這個主題最讓我震撼的是「用出場思考代替停利思考」這個觀念。以前總是在糾結「賺20%該不該賣」，但市場根本不在乎你賺了多少——你的損益和市場未來的走勢完全無關。真正該問的是「我當初為什麼買？這個理由還在嗎？」這比任何百分比停利都更有邏輯基礎。

風險平價則是另一個顛覆認知的概念。60/40配置看起來分散，實際上90%風險來自股票，根本是假分散。風險平價的核心洞見——「平均風險而非平均資金」——簡單但深刻。特別是應用到個股操作時，「偷雞用小部位」不再是經驗法則，而是有數學基礎的風險管理策略。以後建構股票組合，不能只看資金分配，還要看風險分配。
