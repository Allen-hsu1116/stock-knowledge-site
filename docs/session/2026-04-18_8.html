技術分析補強：KD指標與RSI指標進階實戰

第 17 回合學習 - 技術分析補強：KD指標與RSI指標進階實戰

時間：16:30

文章：

  • KD指標實戰攻略：掌握買賣時機的關鍵技術｜OANDA Lab
  • RSI 相對強弱指數是什麼？計算公式、參數設定、應用與缺點說明｜雷司紀
  • RSI指標參數怎麼設？最強戰法「RSI指標+均線」賺賠比1：10｜WinSmart

KD指標——震盪盤的最強武器

是什麼：KD指標（隨機指標，Stochastic Oscillator）由George Lane發明，利用RSV（未成熟隨機值）的加權移動平均，衡量目前股價在過去一段期間內的相對高低位置。K值反應快、D值反應慢，兩者交叉與位置判斷超買超賣。

怎麼做：

1. 計算邏輯：

  • RSV =（今日收盤價 - 最近N天最低價）/（最近N天最高價 - 最近N天最低價）× 100

2. 超買超賣判斷：

  • K值 > 80 且 D值 > 80 → 超買區，股價可能過熱

3. 黃金交叉與死亡交叉：

  • 黃金交叉：K值由下而上穿越D值（低檔區交叉更可靠）→ 買進訊號

4. KD鈍化——最重要的實戰認知：

  • 高檔鈍化：KD > 80 連續3天以上 → 不是要賣，反而是超強勢表現

5. KD參數調整——根據交易週期選擇：

| 交易週期 | 建議參數 | 優勢 | 注意事項 |
|----------|----------|------|----------|
| 當沖/極短線 | (5,3) 或 (7,3) | 反應快速 | 假訊號多，需嚴格停損 |
| 波段操作 | (9,3) 或 (14,3) | 敏感度與穩定性平衡 | 大型股用14，成長股用9 |
| 中長期投資 | (21,3) 或 (28,3) | 過濾短期雜訊 | 反應遲緩，需耐心 |

6. 依市場環境調整：

  • 牛市（多頭）：用(9,3)或(14,3)，避免提早賣出；高檔鈍化不急於做空

實例：

  • 情境：某股KD(9,3)在20以下黃金交叉，同時股價站上20MA
  • 操作：進場做多，停損設在交叉點的K線低點下方
  • 若KD在高檔80以上連續5天鈍化 → 不做空，順勢續抱或等拉回不破前低再加碼
  • 反例：KD在20以下鈍化 → 不要急著抄底，等KD回到20以上且股價站上5MA再考慮

適用時機：

  • ✅ 震盪盤中高拋低吸（KD最佳發揮場景）
  • ✅ 搭配均線、MACD確認趨勢方向的交叉訊號
  • ✅ 低檔黃金交叉確認反彈啟動
  • ❌ 單邊趨勢行情（KD鈍化，訊號失真）
  • ❌ 不搭配其他指標單獨使用（勝率不夠）

RSI指標——比KD更穩定的擺盪指標

是什麼：RSI（相對強弱指數，Relative Strength Index）由J. Welles Wilder Jr.在1978年發明，衡量一段時間內「買方力道」與「賣方力道」的相對強弱，數值在0到100之間。跟KD不同，RSI不只看收盤價的相對位置，還考量了漲跌的「幅度」，更能反映真實的多空力道。

怎麼做：

1. 計算邏輯：

  • 買方力道 = 過去N日漲幅加總 / N日

2. 超買超賣判斷：

  • RSI > 70 → 超買區，市場可能過熱

3. RSI 50 分水嶺——比超買超賣更實用：

  • RSI > 50 → 多方力道較強，市場偏多

4. RSI背離——跟之前學的背離框架完全相通：

  • 看漲背離：價格創新低，RSI沒有創新低（低點墊高）→ 底部反轉訊號

5. RSI高階應用——Failure Swing（反轉失敗）：

  • 這是Welder本人提出的高階用法，比單純超買超賣可靠得多

看漲Failure Swing：

  • RSI跌破30（進入超賣）

看跌Failure Swing：

  • RSI突破70（進入超買）

6. RSI鈍化與應對：

  • 高檔鈍化：RSI長期維持70以上，常見於強勢多頭

實例：

  • 情境：某股RSI14從35跌破30進入超賣，反彈到38，再拉回只到32（沒有再破30），然後RSI衝到55
  • 判讀：看漲Failure Swing完成，底部確認
  • 操作：在RSI突破38的前高時進場做多，停損設在RSI跌破30的價格低點
  • 反例：RSI跌破30後反彈到35，又跌回28（再次破30）→ Failure Swing失敗，不進場

適用時機：

  • ✅ 震盪盤的超買超賣判斷
  • ✅ 背離預警趨勢反轉（搭配MACD背離做雙重確認）
  • ✅ Failure Swing做精確的反轉進場
  • ✅ RSI 50分水嶺判斷多空趨勢
  • ❌ 單邊趨勢行情（RSI鈍化）
  • ❌ 不搭配趨勢指標單獨使用

RSI + 均線「順大逆小」策略——賺賠比1:10的秘密

是什麼：RSI是擺盪指標（沒有方向性），均線是趨勢指標（有方向性），兩者互補性極強。核心邏輯是「順大逆小」——用均線判斷大方向（順大），用RSI在回調時找進場點（逆小）。

怎麼做：

1. 第一步：用均線判斷多空方向：

  • 5MA > 20MA（多頭排列）→ 只做多，不做空

2. 第二步：用RSI找進場時機（順大逆小）：

  • 均線向上 + RSI從超賣區或50以下轉強 → 做多進場點

3. 第三步：停損停利設定：

  • 停損：本金的2%（固定風險）

4. RSI四色溫度計——視覺化RSI強弱：

  • RSI > 70：亮紅色（最強，超買/尬空）

實例：

  • 情境：某股5MA > 20MA多頭排列，均線向上，股價回測20MA附近
  • RSI從28（亮綠色）回升到35（暗綠色）→ 轉強點
  • 操作：在RSI轉強且股價在20MA附近時進場做多
  • 停損：本金2%或20MA下方1-2%
  • 移動停利：收盤跌破20MA出場
  • 賺賠比：停損1-2% vs 潛在獲利10-20% = 賺賠比1:5到1:10

適用時機：

  • ✅ 有明確趨勢的股票或期貨
  • ✅ 波段操作（持股數天到數週）
  • ✅ 想要高賺賠比的進場策略
  • ❌ 盤整雙巴行情（均線和RSI都會失效）
  • ❌ 沒有趨勢的冷門股

KD + RSI + MACD 三指標共振——最高勝率的進場策略

是什麼：KD和RSI都是擺盪指標，適合震盪盤但趨勢盤會鈍化；MACD是趨勢指標，適合趨勢盤但反應慢。三個指標同時發出訊號時，勝率最高。

怎麼做：

1. 多頭共振進場條件：

  • KD在低檔（< 30）黃金交叉

2. 空頭共振進場條件：

  • KD在高檔（> 70）死亡交叉

3. 指標衝突時的判斷：

  • KD看多 + RSI看多 + MACD看空 → 中等訊號，偏多但需確認趨勢

4. 不同市況下的指標選擇：

  • 震盪盤：KD + RSI為主（擺盪指標最適合）

實例：

  • 情境：某股經歷一輪下跌後
  • KD(9,3)在18黃金交叉 ✅
  • RSI14從25回升到40 ✅
  • MACD柱狀圖連續3根縮小並即將翻正 ✅
  • 三重共振 → 高勝率做多進場
  • 反例：KD黃金交叉但RSI仍在50以下且持續下降、MACD柱狀圖仍為負 → 只有一個指標支持，觀望

適用時機：

  • ✅ 關鍵轉折點的進場確認
  • ✅ 提高單一指標訊號的可靠性
  • ✅ 波段操作的進場時機判斷
  • ❌ 不適合頻繁出訊號的震盪盤（會過度交易）

KD指標搭配其他指標的實戰組合

是什麼：KD最強的用法不是單獨使用，而是跟其他指標搭配做交叉驗證。

怎麼做：

1. KD + RSI：強化超買超賣訊號

  • KD黃金交叉 + RSI從30以下回升 → 雙重確認買進訊號

2. KD + 均線（MA）：確認趨勢方向

  • KD黃金交叉 + 股價站穩20MA → 順勢做多，勝率更高

3. KD + MACD：濾除雜訊、確認中長期動能

  • KD黃金交叉 + MACD柱狀圖翻紅 → 中長期趨勢轉強確認

實例：

  • 情境：某股KD(9,3)黃金交叉在25
  • 檢查1：RSI同時從28回升到35 ✅
  • 檢查2：股價站穩20MA ✅
  • 檢查3：MACD柱狀圖即將翻正 ✅
  • 三重確認 → 進場做多，停損設在近期低點下方

適用時機：每次KD出現交叉訊號時，至少再用一個指標做確認。越多指標同時確認，勝率越高。

心得

KD和RSI這兩個擺盪指標，說實話我之前只停留在「超買就賣、超賣就買」的層面，但今天深入學了才發現這是最危險的用法。在趨勢盤中，KD高檔鈍化可以鈍到80以上連續10幾天，如果你一看到超買就做空，會被趨勢輾過去。

幾個重要認知更新：

1. 鈍化不是反轉訊號，是強勢/弱勢訊號：高檔鈍化 = 超強勢多頭，不要做空；低檔鈍化 = 極弱勢空頭，不要抄底。這跟之前學的「趨勢盤中不要猜頭摸底」完全一致。正確做法是：鈍化期間順勢操作，等鈍化結束後（拉回不破前低/反彈不過前高）再判斷方向是否改變。

2. RSI的Failure Swing比超買超賣好用太多：Welder本人提出這個高階用法，比單純看70/30更可靠。核心邏輯是：RSI進入超賣區後反彈，再拉回卻沒有再破超賣線——這代表空方力道已經減弱到無法再創新低，是比單純超賣更強的反轉確認。之前學的「背離確認三步驟」也是同一個邏輯——不要在第一個訊號出現就進場，等確認再動。

3. RSI 50分水嶺是被低估的工具：多數人只看70/30，但其實50才是多空的真正分界。RSI在50以上 = 多方力道強，回落到50不破 = 加碼點；跌破50 = 多頭轉弱。這比超買超賣更常用、更可靠。

4. 「順大逆小」策略把均線和RSI的互補性發揮到極致：均線給方向（不讓你在空頭中做多），RSI找時機（在回調時進場而不是追高）。這個框架解決了RSI最致命的缺點——沒有方向性。搭配均線後，RSI只在正確的方向上操作，勝率自然大幅提升。

5. KD參數要根據市況調整：以前都用固定(9,3)，但現在知道牛市要用長參數(14,3)避免過早賣出，熊市要用短參數(5,3)快速反應。這跟之前學的「不同盤勢用不同策略」是同一個核心觀念——沒有一套參數走天下。

6. 三指標共振是最高勝率的進場方式：KD + RSI + MACD同時發出做多訊號，勝率可以拉到70%以上。但這種機會不常出現，需要耐心等待。這跟之前學的「多重背離共振」策略完全一致——越多不同類型的指標同時確認，訊號越可靠。

整體來說，這回合把技術分析中兩個最基礎也最常用的擺盪指標從「表面用法」深入到「實戰用法」，跟之前學的背離確認、均線趨勢判斷、OBV量價確認完全串在一起。至此，技術分析的指標體系已經相當完整：趨勢指標（MA/MACD）+ 擺盪指標（KD/RSI）+ 量價指標（OBV/A/D）+ 波動指標（布林通道），加上支撐壓力線和斐波那契回撤，整個框架可以互相確認、互相過濾。

時間：17:30

文章：

  • 營收成長率是什麼？YoY、QoQ、MoM 怎麼算？成長率計算完整教學！
  • 營收成長率是什麼？只看YoY？最完整的營收成長率解析
  • 月營收成長率策略應用 - TEJ台灣經濟新報
  • 營收成長率 YoY 是什麼？

一、營收成長率三種計算方式

是什麼：營收成長率是衡量公司在一段期間內營收增減的指標，分為年增率（YoY）、季增率（QoQ）、月增率（MoM），是最即時的基本面追蹤工具——台股每月10日前公布上月營收，比季財報早得多。

怎麼算：
1. 年增率 YoY =（當期營收－去年同期去年同期營收）÷ 去年同期營收 × 100%

  • 例：2026年3月營收12億，2025年3月營收10億 → YoY = (12-10)/10 = +20%

2. 季增率 QoQ =（當季營收－上季－上季營收）÷ 上季營收 × 100%
3. 月增率 MoM =（當月營收－上月營收）÷ 上月營收 × 100%

YoY最重要，原因有二：

  • 產業淡旺季：旅遊業暑假旺、玩具業Q4旺，QoQ和MoM容易被季節性扭曲
  • 月份天數差異：2月只有28天vs1月31天，差2-3天可能就是10%的誤差

適用時機：

  • YoY：任何時候的首選，消除季節性干擾
  • MoM：僅適合觀察「趨勢方向」，數值本身參考性低
  • QoQ：國外美股常用（因只有季報），台股較少用

二、三種YoY層級：年營收YoY、季營收YoY、月營收YoY

是什麼：同樣是年增率，但用不同時間尺度的營收來計算，敏感度不同。

怎麼做：
1. 年營收YoY：全年營收vs去年全年，最穩定但最慢（一年才能看一次）
2. 季營收YoY：今年Q1 vs 去年Q1，一季看一次，兼顧時效與穩定
3. 月營收YoY：今年3月 vs 去年3月，最即時但波動大

實戰用法：

  • 連續2-3季季營收YoY都維持類似成長率 → 成長非偶然，可信度高
  • 例如：某公司Q1 YoY +15%、Q2 YoY +18%、Q3 YoY +16% → 穩定成長確認
  • 但若只有1季YoY暴增，要小心是否為一次性訂單

適用時機：

  • 中長線投資人：看季營收YoY確認趨勢
  • 短線交易人：看月營收YoY抓轉折點
  • 年營收YoY：年報出來時做最終確認

三、累計營收年增率

是什麼：從年初累計到目前的營收，與去年同期累計營收比較的成長率。比單月YoY更穩定，因為累計效應會平滑掉單月波動。

怎麼做：
1. 例如4月公布的累計營收 = 1月+2月+3月+4月營收
2. 去年同期累計 = 去年1-4月營收
3. 累計YoY =（今年累計－去年累計）÷ 去年累計 × 100%

實例：

  • 某公司1-4月累計營收50億，去年同期45億 → 累計YoY = +11.1%
  • 若單月4月YoY只有+3%，但累計YoY仍有+11.1% → 整體趨勢仍健康，4月只是稍微降溫

適用時機：每月營收公布後第一時間看累計YoY，比單月數字更有參考價值

四、營收成長率三種選股訊號

是什麼：用營收成長率的「變化方向」來選股，不是看絕對數值大小，而是看成長率的加速度。

怎麼做：

1. 成長增加（加速度正）：營收YoY從+10% → +15% → +22%

  • 市場需求正在加速擴大

2. 衰退減少（減速收斂）：營收YoY從-20% → -12% → -5%

  • 公司營運衰退正在收斂，可能接近谷底

3. 由負轉正（趨勢反轉）：營收YoY從-8% → -3% → +5%

  • 公司轉型成功、取得新訂單、產品開始受市場青睞

實例：

  • 某IC設計公司YoY：1月-5%、2月+2%、3月+8%、4月+15% → 由負轉正且加速，可能是新產品放量
  • 某塑化股YoY：1月+30%、2月+25%、3月+18%、4月+10% → 成長減速，可能景氣循環到頂

嚴謹篩選條件：設定「連續3個月以上月營收YoY > 10%」，過濾掉曇花一現的公司

五、TEJ回測實證：月營收YoY+MoM雙指標策略

是什麼：TEJ用2013-2021年台股全上市櫃公司回測，以月營收的YoY和MoM雙重確認做進出場。

策略邏輯：
1. 計算YoY的5月移動平均、MoM的10月移動平均
2. 進場：當月YoY > YoY 5月均線 且 當月MoM > MoM 10月均線
3. 出場：當月YoY < YoY 5月均線 且 當月MoM < MoM 10月均線
4. 買進時機：營收公布隔天買進（避免未來資料偏誤）

回測結果：

六、營收成長率判讀三大陷阱

1. 營收成長率≠淨利成長率

2. 營收成長率不是線性的

  • 怎麼做：用連續多季YoY觀察趨勢，不要外推假設未來維持同樣成長率

3. 營建股營收成長率不具參考性

  • 怎麼做：營建股看推案量和銷售率，不看營收YoY
