一、籌碼九階段論：接→護→拉→換→養→軋→出→套→殺

什麼是籌碼九階段論

是什麼：一檔個股從興起到幻滅，在籌碼面上會經歷九個固定階段，形成一個「S 曲線」——籌碼先從散戶流向大戶（聚集），再從大戶流向散戶（發散）。掌握股價處於哪個階段，就能決定「什麼股票要觀察、什麼股票要進場、什麼股票要出場、什麼股票要做空」。

怎麼做：每檔股票每天觀察其籌碼數據（主力買賣超、融資融券、千張大戶、法人進出），判斷目前處於九階段的哪一段，然後對應不同的操作策略。

九階段詳細判讀

階段 1：接（低檔承接）

是什麼：股價跌到主力認為便宜的價位，大戶開始逢低承接。

特徵：
1. 大戶買超，但主力不見得買超（大戶≠主力，大戶可能是公司派）
2. 散戶賣超（耐心不足、停損出場）
3. 融資減少（散戶退場）
4. 成交量不大（主力低調吸貨）
5. 股價仍在下降趨勢，但跌勢漸緩
6. 關鍵：股價下跌，但內部人持股增加

操作：此時不宜進場，因為股價可能還會繼續跌。但可以列入觀察名單，等待後續階段。

實例：國巨（2327）在 2017 年初，股價在低檔盤整，大戶持續小量買超，融資持續下降，內部人持股增加——這就是「接」的階段。

階段 2：護（積極護盤）

是什麼：主力買超一陣子後，股價跌到目標價，開始更積極布局。

特徵：
1. 股價多日未再破底
2. 區間主力買超金額愈來愈大
3. 成交量比以前大
4. 主力不只逢低承接，也開始追價買進

操作：可以開始小量佈局，設好停損。這個階段的風險是主力可能護盤失敗、股價再度破底。

階段 3：拉（拉升吃貨）

是什麼：主力在低檔買不到籌碼後，開始追價買進。此時市場還沒有利多消息，股價莫名上漲。

特徵：
1. 股價開始明顯上漲
2. 主力持續買超
3. 消息面還沒有利多（市場覺得「莫名奇妙就漲了」）
4. 股價比大盤強
5. 主力又拉又殺，洗出短線籌碼，但累計買超持續上升

操作：這是做多最愛的進場點之一——「從拉到換」的過程。如果前面有「接」和「護」的階段，從「拉」進場的勝率更高。

關鍵判斷：如果「拉」之後沒有出現「換」（市場不認同），就見好就收；如果出現「換」，就可以抱牢。

階段 4：換（籌碼換手）

是什麼：紙包不住火，主力持續買超和股價強勢表現引起市場注意。基本面利多開始出現，法人、代操、投資高手等先知先覺者進場，成交量放大，籌碼快速換手。

特徵：
1. 法人買超
2. 官股券商、高手券商、前十大綜合券商總公司都買超
3. 千張大戶開始增加
4. 股價比大盤強
5. 融資減少
6. 大戶買超、散戶賣超

操作：這是最佳持股階段。多方合力推升股價，千張大戶增加、散戶退場——籌碼從弱手流向強手。

實例：當一檔股票從代操到外資、投信到官股都一起進場買，就是進入「換」的階段。

階段 5：養（養券養空）

是什麼：股價漲到一定程度，散戶追不下手。主力故意示弱，讓短多下車，也讓空頭覺得高點已過開始做空。

特徵：
1. 股價漲幅已大
2. 股價轉弱（主力刻意壓回）
3. 融券開始增加
4. 主力小賣超（假裝出場，其實是養空）
5. 融資與融券同步增加

操作：此階段要特別小心。看似主力出場，其實是養空。關鍵判斷：如果接下來股價沒有跌破防守線，融券又持續增加，後續可能出現「軋空」行情。

階段 6：軋（軋空拉升）

是什麼：豬養肥了就是要殺。主力在「養」的階段累積了空單後，開始拉高股價，準備軋空出貨。

特徵：
1. 消息見報率大增（利多消息不斷）
2. 社群媒體常曝光（甚至有「老師」、網紅推薦）
3. 融券大增（空頭被鎖定）
4. 主力買超停滯（開始準備出貨）
5. 成交量增加（散戶追價、空頭回補）

操作：此階段股價加速上漲，看似最甜美，但主力已經準備出貨。已持有者設移動停損續抱，新進場者風險極高。

階段 7：出（主力出貨）

是什麼：好消息不斷、股價持續上漲後，空頭回補推升股價。股價接近主力目標價，主力開始出場。

特徵：
1. 主力開始賣超
2. 融券開始回補（軋空尾聲）
3. 融資成長（散戶追價進場）
4. 消息面充斥利多（但股價漲不動了）
5. 法人仍在追捧，但股價上不去或上去撐不久

操作：這是散戶最危險的階段。利多消息滿天飛，但股價已經漲不動。看到主力賣超 + 融資增加 = 出貨警報，應該減碼或出場。

實例：國巨（2327）在軋空時主力就開始出貨。軋空結束、消息面大好時，其實主力更是用力出貨。

階段 8：套（套牢散戶）

是什麼：主力出貨初期，股價隨消息面開高，但常開高走低，留下影線。剛進場的投資人處於套牢狀態。

特徵：
1. 消息面利多不斷（但已經是「利多不漲」）
2. 主力賣超
3. K 線有較長上影線或長黑，相對大盤偏弱
4. 股價盤頭，位於短期均線下方
5. 千張大戶開始減少
6. 大戶賣超、散戶買超

操作：此階段如果還持有，應該盡快出場。利多不漲 + 千張大戶減少 + 散戶買超 = 明確的出場訊號。

階段 9：殺（下殺取量）

是什麼：主力出貨出得差不多，或現有價位出不掉，開始下殺取量，把剩餘籌碼全部拋出。各方勢力爭相獲利了結。

特徵：
1. 股價明顯下跌
2. 融資反而增加（散戶想撿便宜，結果愈撿愈便宜）
3. 所有勢力（主力、法人、公司派、高手、官股）都站在賣方
4. 空單急劇減少（已經沒有做空價值）
5. 先前累計的買超加速釋出

操作：此階段不應搶反彈。融資增加表示散戶還在接刀，下跌動能未止。等融資明顯減少、大戶重新買超，才可能進入新的「接」階段。

九階段對應操作策略表

階段特徵操作策略
1. 接大戶買、散戶賣、融資減、量縮列入觀察，不宜進場
2. 護股價不破底、主力買超加大小量佈局，設停損
3. 拉股價明顯上漲、主力持續買做多進場點，觀察後續有無「換」
4. 換法人進場、大戶買、散戶賣最佳持股階段，抱牢
5. 養股價轉弱、融券增、主力小賣小心養空，觀察防守線
6. 軋利多不斷、融券大增、主力買超停滯設移動停損，新進場風險高
7. 出主力賣超、融資增、利多不漲減碼或出場
8. 套開高走低、大戶賣、散戶買盡快出場
9. 殺全面下跌、融資反增、各方賣超不搶反彈，等待新循環

二、法人籌碼轉折判讀

外資連續買超的兩大難題

難題 1：假外資判斷

怎麼判斷：

  • 假外資資金有限，更重視資金效率，操作持續期間較短
  • 假外資偏好中小型股（市值小、用少量資金就能影響股價）
  • 今天買、隔天賣 = 典型假外資（隔日沖）
  • 真外資偏好中大型股（考量流動性風險）

結論：即使是假外資，一樣具備參考性——因為假外資背後也是具備資金實力的投資人。關鍵是判斷其操作持續性。

難題 2：外資還會繼續買嗎？

核心觀念：「外資連續買超 5 日」只是選股方法之一，不代表外資一定會繼續買，更不代表股價一定會漲。

怎麼做：
1. 外資連續買超只是起點，要搭配技術面和價格位置判斷
2. 觀察外資買超金額的趨勢——遞增（持續加碼）vs 遞減（買盤力竭）
3. 外資買超 + 股價站上均線 = 進場勝率更高
4. 外資買超 + 股價破底 = 不要跟，外資可能認賠

法人籌碼轉折五種訊號

訊號 1：買超遞增 → 遞減（買盤力竭）

  • 外資/投信連續買超，但買超金額逐日遞減
  • 意義：法人仍在買，但力道減弱，買盤即將結束
  • 操作：買超力道減弱時，逐步減碼

訊號 2：買超 → 轉賣超（多空轉折）

  • 法人從連續買超突然轉為賣超
  • 意義：法人翻空，籌碼面由多轉空
  • 操作：賣超金額超過前 3 日平均買超的 50%，就要警戒

訊號 3：賣超 → 轉買超（空翻多）

  • 法人從連續賣超突然轉為買超
  • 意義：法人可能認為底部已到，開始回補
  • 操作：觀察是否連續買超 2~3 日確認，再決定是否跟進

訊號 4：法人對作（多空分歧）

  • 外資買超、投信賣超（或反過來）
  • 意義：法人對後市看法分歧
  • 操作：看誰量大誰主導——外資量大偏多、投信量小影響有限

訊號 5：法人合買（多空一致）

  • 外資 + 投信 + 自營商同步買超
  • 意義：三大法人一致看多，短期續漲機率高
  • 操作：合買訊號可作為進場確認，但要注意自營商避險部位是否干擾

大戶散戶對作判讀

是什麼：透過盤中即時的「大戶買賣力」與「散戶買賣力」，判斷當下的多空趨勢由誰主導。

怎麼做：
1. 開盤觀察大戶 vs 散戶的買賣力道
2. 大戶買力 > 散戶賣力 → 大戶主導，股價容易續漲
3. 散戶買力 > 大戶賣力 → 散戶推升，容易被大戶倒貨
4. 盤中持續追蹤，大戶出貨 + 散戶接刀 = 階段 7~8 的「出」和「套」

實例：

  • 華通（2313）2/23 開盤大戶就開始出貨，股價從 82.3 跌至最低 76.6
  • 散戶盤中持續買超，但對股價無明顯影響
  • 大戶的買賣超對股價影響遠大於散戶

大戶持股比例判斷：

  • 低於 40%：籌碼分散，不易拉抬
  • 40%~70%：黃金區間，最容易操作
  • 70%以上：過度集中，流動性不足
  • 80%以上：極度集中，操控風險高

心得

這回合學到的「籌碼九階段論」根本就是主力操盤的劇本。以前看籌碼數據只知道「大戶買超好、大戶賣超不好」，但不知道股價在同一個籌碼狀態下，可能處於完全不同的階段。

最關鍵的認知是：同樣是「大戶買超」，在「接」的階段和「換」的階段意義完全不同。「接」的時候大戶低調吃貨，你跟了可能要等很久；「換」的時候多方合力推升，才是最佳持股階段。同樣「主力賣超」，在「養」的階段可能是假動作（養空），在「出」的階段就是真出貨。

另一個重要教訓是「利多不漲 = 出貨」。階段 7（出）和階段 8（套）時，消息面都是利多，但股價漲不動甚至開高走低。這就是散戶最容易被套的地方——看到利多就以為還會漲，結果主力已經在倒貨了。

法人轉折判讀的部分，最實用的是「買超遞增→遞減」這個訊號。以前只看法人買超或賣超，沒注意買超金額的趨勢。遞增代表加增代表加碼中，遞減代表買盤力竭——這個觀察維度比單純看買超賣超更有價值。

下次看一檔股票的時候，先問自己：它現在在哪個階段？這個問題比「主力買超還是賣超」更重要。
