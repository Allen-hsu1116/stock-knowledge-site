核心知識

企業經營效率是什麼？

是什麼：企業在各個經營流程（賣出商品、收錢、付錢）的速度與效率。效率好的企業能更快把貨賣出去、更快拿到現金運用。

為什麼重要：效率差的企業資金會積壓在存貨或應收帳款，無法有效運用，甚至可能周轉不靈。

存貨周轉率（Inventory Turnover）

是什麼：公司在一段時間內商品平均銷售的次數，反映存貨流動性和資金使用效率。

公式：

  • 存貨周轉率（次/年）= 營業成本 ÷ 平均存貨
  • 平均存貨 = （期初存貨 + 期末存貨）÷ 2
  • 存貨周轉天數 = 365天 ÷ 存貨周轉率

怎麼做：
1. 從財報找到營業成本（損益表）
2. 從資產負債表找到期初、期末存貨
3. 計算平均存貨後代入公式

實例：

  • 情境：某公司營業成本100萬，平均存貨20萬
  • 計算：存貨周轉率 = 100 ÷ 20 = 5次/年
  • 解讀：一年能清空存貨5次，約73天賣完一批存貨

數值解讀：

情況存貨周轉率高存貨周轉率低
意義銷售快、資金效率好存貨賣不出去、資金積壓
風險太高可能缺貨、錯失訂單跌價損失、存貨過期

注意事項：

  • 必須與「同業」或「公司前期」比較，各產業標準不同
  • 金融業、營建業不適用此指標（金融業存貨是錢，營建業存貨是土地）
  • 製造業、電子業、零售業、百貨業適用

適用時機：

  • ✅ 評估製造業、零售業的經營效率
  • ✅ 觀察公司存貨是否有長期增加趨勢
  • ✅ 搭配營業現金流、應收帳款一起看
  • ❌ 不適用金融業、營建業

應收帳款周轉率（Receivables Turnover Ratio）

是什麼：公司一年內收回帳款的次數，衡量公司回收應收帳款的速度。

公式：

  • 應收帳款周轉率（次）= 銷貨淨額 ÷ 平均應收帳款
  • 平均應收帳款 = （期初應收帳款 + 期末應收帳款）÷ 2
  • 應收帳款周轉天數 = 365天 ÷ 應收帳款周轉率

怎麼做：
1. 從損益表找到銷貨淨額（營業收入）
2. 從資產負債表找到期初、期末應收帳款
3. 計算平均應收帳款後代入公式

實例：

  • 情境：A公司銷貨淨額80萬美元，年初應收帳款6.4萬，年底7.2萬
  • 計算：平均應收帳款 = (6.4 + 7.2) ÷ 2 = 6.8萬美元
  • 應收帳款周轉率 = 80 ÷ 6.8 = 11.76次
  • 應收帳款周轉天數 = 365 ÷ 11.76 ≈ 31天

數值解讀：

情況周轉率高周轉率低
意義收款快、客戶信用好收款慢、呆帳風險高
天數天數短（<45天為佳）天數長、資金滯留

注意事項：

  • 各產業特性不同，要與同業比較
  • 關注「趨勢變化」比絕對數字重要
  • 應收帳款和存貨是財報最容易造假的地方

適用時機：

  • ✅ 評估公司收款能力和客戶品質
  • ✅ 觀察公司是否有帳款回收問題
  • ✅ 搭配存貨周轉率一起看現金循環
  • ❌ 不適用主要收現金的企業（如餐飲業）

現金轉換循環（Cash Conversion Cycle）

是什麼：從付錢買貨到收到現金所需的天數，整合存貨、應收帳款、應付帳款的綜合指標。

公式：

  • 現金轉換循環 = 存貨周轉天數 + 應收帳款周轉天數 - 應付帳款周轉天數

怎麼做：
1. 計算存貨周轉天數（賣貨要幾天）
2. 計算應收帳款周轉天數（收錢要幾天）
3. 計算應付帳款周轉天數（付錢給供應商要幾天）
4. 相加減得到現金轉換循環天數

實例：

  • 情境：NIKE vs Under Armour 比較
  • NIKE：存貨88天 + 收現45天 - 付現46天 = 87天
  • UA：存貨101天 + 收現33天 - 付現?天

數值解讀：

  • 天數越短越好：表示現金週轉效率高，資金積壓時間短
  • 天數太長：企業需要更多營運資金，財務風險較高

實戰應用

如何判斷公司經營效率好壞？

步驟一：與同業比較

  • 同產業的周轉率才有比較意義
  • 例如：零售業周轉率通常比製造業高

步驟二：觀察趨勢變化

  • 本期 vs 前期：變好還是變差？
  • 存貨周轉率上升 = 銷售能力變好
  • 應收帳款周轉率上升 = 收款能力變好

步驟三：搭配現金流檢查

  • 營業現金流是否穩定？
  • 應收帳款增加是否合理？（配合營收成長）

步驟四：檢查是否有造假風險

  • 存貨、應收帳款是財報最易造假的地方
  • 若周轉率異常波動要特別注意

警訊信號

狀況可能問題
存貨周轉率持續下降產品滯銷、市場需求減弱
應收帳款周轉天數暴增客戶付款困難、可能認列呆帳
營收成長但存貨周轉率下降成長可能是假的（堆在存貨）
營收成長但應收帳款周轉率下降成長可能是假的（錢還沒收到）

心得

這幾個指標串在一起，其實就是在看「企業的錢轉一圈要多久」。

存貨周轉率看的是「貨多久能賣掉」，應收帳款周轉率看的是「賣掉後多久收到錢」，加起來就是企業從「備貨→賣貨→收錢」的效率。

實際看公司時，我會特別注意：
1. 趨勢變化：比絕對數字重要，周轉率持續下降就是警訊
2. 與同業比較：每個產業特性不同，單看數字沒意義
3. 搭配現金流：周轉率變差 + 營業現金流變差 = 大問題
4. 造假風險：存貨和應收帳款是最容易動手腳的地方

最後，企業經營效率好，不一定代表值得投資（毛利、成長性也很重要）。但經營效率差的企業，財務風險通常比較高，這點要記住。
