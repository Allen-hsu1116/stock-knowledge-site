核心知識

一、鋸齒形（Zigzag）調整波

是什麼：鋸齒形是波浪理論中最常見的調整波型態，由 ABC 三個波動組成。A 浪是推動波（5 波結構），B 浪是調整波，C 浪是推動波（5 波結構）。整體走勢像鋸齒一樣「急跌—反彈—再跌」。

結構特徵：

  • A 浪 = 衝擊波或楔形（5 波下跌）
  • B 浪 = 任何調整波型態（鋸齒形、平台形、三角形皆可）
  • C 浪 = 衝擊波或楔形（5 波下跌）
  • B 浪不會超過 A 浪的起點（這是鐵律）

出現位置：

  • 推動波的第 2 波和第 4 波（最常見）
  • 楔形的 1、3、5 浪（作為附次級浪，與主要趨勢同向！）
  • 平台形的 A 浪（與主要趨勢同向）
  • 三角形的子波

怎麼做（實戰操作）：
1. 識別鋸齒形：A 浪是 5 波下跌 → B 浪反彈但不超過 A 浪起點 → C 浪又是 5 波下跌
2. 在 C 浪完成時進場：C 浪幅度 ≈ A 浪幅度（或 C = A × 1.618、C = A × 2.618）
3. 用 C 浪的 5 波細分結構確認末端：C 浪本身也是衝擊波，可在 C-5 完成時買入
4. 停損設在 C 浪低點下方
5. 目標：下一波推動波的全段漲幅

實例：

  • 某股第 1 浪從 50 漲到 80（漲幅 30），第 2 浪開始調整
  • A 浪從 80 跌到 65（跌 15），B 浪反彈到 72（不超過 80），C 浪從 72 跌到 55（C ≈ A 的 1.13 倍）
  • C 浪完成時 RSI 超賣 + 出現錘子線 → 在 55 附近買入
  • 停損：50 下方（第 1 浪起點，鐵律：第 2 浪不跌破第 1 浪起點）
  • 目標：第 3 浪整段漲幅，至少等於第 1 浪的 30 元

適用時機：

  • 上升趨勢中的第 2 波回檔（深幅調整，常回測 0.618）
  • 下跌趨勢中的反彈放空
  • ⚠️ B 浪中交易最容易賠錢——不確定性最高，最好等 C 浪完成

二、雙重鋸齒形與三重鋸齒形

是什麼：當一個鋸齒形調整完畢但市場仍需更多時間整理，會出現兩個或三個鋸齒形連接。雙重鋸齒形 = W + X + Y；三重鋸齒形 = W + X + Y + X + Z。

結構：

  • W 浪 = 鋸齒形
  • X 浪 = 任何調整波（連接用）
  • Y 浪 = 鋸齒形
  • Z 浪 = 鋸齒形（三重時才有）

怎麼做：
1. 當你以為鋸齒形 C 浪結束了，結果又出現一段下跌 → 可能是雙重鋸齒形
2. 不要急著進場，等整個 W-X-Y 結構完成
3. Y 浪的 C 浪完成時才是真正的買點
4. 雙重鋸齒形性質與單一鋸齒形相同，只是調整更深、時間更長

實例：

  • 台股指數從 20000 跌到 18500（W 浪鋸齒形），反彈到 19200（X 浪），再跌到 18000（Y 浪鋸齒形）
  • 整個雙重鋸齒形完成 → 在 18000 附近買入
  • 停損：17500 下方
  • 目標：回到 20000 以上

適用時機：

  • 替代單一鋸齒形出現（當單一鋸齒形不足以完成調整時）
  • 大級別的調整波中較常出現

三、平台形（Flat）調整波

是什麼：平台形同樣由 ABC 三波組成，但與鋸齒形不同——A 浪是 3 波結構（調整波），B 浪幾乎完全回補 A 浪，C 浪是推動波。整體走勢接近「橫向波動」，調整幅度比鋸齒形淺。

四種平台形型態：

1. 常態平台（Regular Flat）：

  • B 浪終點 ≈ A 浪起點
  • C 浪終點 ≈ A 浪終點
  • 像矩形整理

2. 非常態/擴張平台（Irregular/Expanded Flat）：

  • B 浪超越 A 浪起點（創新高或新低！）
  • C 浪也超越 A 浪終點
  • 最容易出現的平台形型態
  • B 浪至少要回補 A 浪的 90% 以上

3. C 浪超級擴張平台：

  • C 浪遠超過 A 浪的終點
  • 極端行情中出現

4. 順勢/強勢平台（Running Flat）：

  • B 浪超越 A 浪起點
  • C 浪未到達 A 浪終點
  • 趨勢極度強勢時出現

怎麼做（實戰操作）：
1. 判斷 A 浪結構是關鍵：A 浪是 3 波 → 整體是平台形；A 浪是 5 波 → 整體是鋸齒形
2. A 浪三波下跌後，B 浪反彈可達 A 浪的 90%～123.6%
3. 若 A 浪是三波 → B 浪可能超過第 5 浪高點（不規則修正）
4. 在 C 浪 5 波完成時反向操作

實例：

  • 某股第 3 浪高點 100，A 浪三波跌到 90，B 浪反彈到 102（超過前高！擴張平台）
  • C 浪從 102 跌到 88（超過 A 浪終點 90）
  • C 浪 5 波完成 + RSI 背離 → 在 88 買入
  • 停損：85 下方
  • 目標：第 5 浪漲幅

適用時機：

  • 推動波的第 4 波（常見型態：第 2 波鋸齒形，第 4 波平台形）
  • 橫向盤整時間長但幅度淺
  • ⚠️ 擴張平台的 B 浪創新高會讓人誤以為趨勢繼續——這是最危險的陷阱！

四、三角形（Triangle）調整波

是什麼：由 ABCDE 五個波組成的收斂型態，每一波都是鋸齒狀（雙鋸齒結構）。E 波最常出現三角形。三角形是調整波，原則上只出現在第 4 波或 B 波。

四種三角形模式：
1. 收縮三角形（Converging）：高點降低、低點升高，逐漸收斂
2. 障礙三角形（Barrier）：B-D 呈水平線，另一邊收斂
3. 擴張三角形（Expanding）：高點升高、低點降低，逐漸擴張
4. 持續三角形（Running）：整體偏離主要趨勢方向

結構特徵：

  • 每一波幅度約為前一波的 0.618 倍
  • 每一波多分三波進行
  • E 波可能在 A-C 連線上結束、提前結束、或稍微超出

怎麼做（實戰操作）：
1. 三角形通常是事後才確認：進行到一半才發現是三角形
2. E 波完成時進場：這是最佳進場點
3. 突破確認：價格突破 C 浪高點（或跌破 C 浪低點）+ 常伴隨跳空缺口
4. 目標計算：突破後的漲跌幅至少 = 三角形最寬處的距離
5. 三角形完成 = 大級別波浪即將結束的訊號：如果三角形在第 4 波，第 5 波完成後趨勢即告一段落

實例：

  • 台積電第 3 浪高點 1000 後，第 4 波進入三角形整理
  • A = 920、B = 970、C = 930、D = 960、E = 935
  • 三角形最寬處 = 970 − 920 = 50
  • E 波完成後向上突破，跳空越過 960（D 浪高點）
  • 目標 = 935 + 50 = 985（第 5 波目標）
  • 第 5 波完成後 → 趨勢即將反轉，準備 ABC 修正

適用時機：

  • 第 4 波或 B 波的整理區
  • ⚠️ 不會出現在第 2 波（鐵律）
  • 三角形完成 = 「最後一波」的警報器
  • 在三角形內操作利潤小、風險低，但手續費可能是負擔

五、複合修正（Combination）

是什麼：把多個調整波連接在一起的複雜結構。分兩大類：

1. 鋸齒類複合修正：

  • 雙重鋸齒形（W-X-Y）
  • 三重鋸齒形（W-X-Y-X-Z）
  • 性質與鋸齒形相同，只是規模更大

2. 平台形複合修正：

  • 雙重三波（Double Three = W-X-Y）
  • 三重三波（Triple Three = W-X-Y-X-Z）
  • 性質與平台形相同，是橫向持平的修正
  • 只會出現一處鋸齒狀
  • W、Y、Z 是作用波（Action wave），X 是反應波（Reaction wave）

怎麼做：
1. 遇到複雜的橫向整理 → 先歸類為「平台形或複合修正」
2. 不需要把每一個子波都算清楚——把持平部分整體看待即可
3. 等整個複合修正完成後再進場
4. 雙重鋸齒 = 取代鋸齒形；雙重三波 = 取代平台形

實例：

  • 道瓊指數 1937-1942 年的第 4 波 = 長達約 15 年的持平修正波
  • 分析為平台形 + 三角形的複合修正
  • 完成後展開超級第 5 浪上升

適用時機：

  • 當調整波看起來太複雜、無法用單一型態解釋時
  • 大級別調整中常出現
  • ⚠️ 不需過度分析細部結構，抓大方向即可

六、波浪理論七大操作法則（實戰總整理）

法則一：第二波結束時進場買進

  • 先決條件：ABC 調整完成 + 出現 5 波上升的第 1 浪
  • 第 2 波回檔目標：0.382、0.5、0.618
  • 停損：第 1 浪起點下方
  • 目標：第 3 浪全段漲幅

法則二：第三波之中進場買進（唯一不抄底的進場法）

  • 先決條件：第 2 波結束確認 + 價格跳空突破第 1 浪高點
  • 停損①：第 1 浪高點下方；停損②：前一天高點下方
  • 目標：第 3 浪全段漲幅（至少等於第 1 浪）
  • ⚠️ 跳空當天回補 = 波浪辨認可能錯誤

法則三：第四波將結束時進場

  • 五個買點指引：

1. 第 4 波低點 > 第 1 波高點（鐵律）
2. 第 4 波回檔 = 第 3 波的 0.382
3. 第 4 波長度 ≈ 第 2 波長度
4. 第 4 波回檔到 3-4 波低點附近
5. 第 4 波鋸齒形時，C ≈ A

  • 買點愈聚集 → 信心愈高
  • 停損：第 1 波高點下方
  • 目標：第 5 波漲幅

法則四：第五波高點放空（最危險的操作）

  • 確認訊號：竭盡缺口出現 + 被回補
  • 第 5 波目標價：

① = 第 1 浪漲幅
② = 第 1 浪到第 3 浪漲幅的 0.618 倍

  • ⚠️ 竭盡缺口未回補前，可能不是第 5 浪而是第 3 浪延長——此時放空是自殺
  • 停損：第 5 波高點上方一點
  • 目標：ABC 修正的 0.382 回檔位

法則五：B 波結束時放空（關鍵在 A 波結構）

  • A 浪 5 波下跌 → B 浪不會超過第 5 浪高點
  • A 浪 3 波下跌 → B 浪可能超過第 5 浪高點（不規則修正）
  • 停損：A 浪最低 + A 浪跌幅 × 1.382 以上

法則六：鋸齒形之後進場

  • 鋸齒形可能是更大級別的第 2、4 或 B 波
  • ⚠️ 如果是 B 波 → 不進場（B 波中交易最危險）
  • C 浪完成時各指標極度超賣 → 反向操作
  • C 浪可能 = A 浪、A × 1.618、A × 2.618

法則七：三角形整理區操作

  • 三角形通常是進行一半才被發現
  • E 波完成時進場
  • 突破 C 浪高/低點 + 跳空 = 確認訊號
  • 目標 = 三角形最寬處距離
  • ⚠️ 三角形完成 → 後面只剩最後一波

盈虧比要求：進場前計算賺賠比 ≥ 3 才進場，否則繼續觀望。

心得

這回合把波浪理論最讓人頭痛的部分——調整浪——整個拆解了一遍。之前學的「八大波浪結構」只是骨架，這次才是真正的血肉。

最核心的體悟：A 浪的結構決定一切。A 浪是 5 波還是 3 波，決定了後面 B 浪和 C 浪的行為模式。A 浪 5 波 = 鋸齒形（深幅調整，B 浪反彈有限）；A 浪 3 波 = 平台形（淺幅調整，B 浪可能創新高）。這個判斷做對了，後面的操作就有依據；做錯了，就會在擴張平台的 B 浪創新高時以為趨勢繼續，結果 C 浪一殺下來直接被抬出去。

三角形是另一個大收穫——它不只是整理型態，更是「最後一波」的警報器。當第 4 波出現三角形，第 5 波完成後趨勢就要反轉了。這跟之前學的「竭盡缺口」和「MACD 背離」可以互相印證：三角形完成 → 第 5 波啟動 → 第 5 波出現竭盡缺口 + MACD 背離 → 確認趨勢反轉。三重確認比單一訊號可靠太多了。

操作法則中，最實用的是「法則三：第四波將結束時進場」——五個買點指引愈聚集，信心愈高。這概念跟之前學的「斐波那契群聚」一模一樣：多種方法指向同一個價位，那個價位就值得信賴。波浪理論的鐵律（第 4 波不跌破第 1 波高點、第 2 波不跌破第 1 波起點）則提供了客觀的停損位，不用再憑感覺設停損。

最後一個教訓：永遠不要在 B 波中交易。這是所有操作法則中最一致的警告。B 浪的不確定性最高——你不知道它是鋸齒形的 B、平台形的 B、還是三角形的 B，每一種的行為完全不同。等 C 浪完成再進場，少賺一點但活得久。
