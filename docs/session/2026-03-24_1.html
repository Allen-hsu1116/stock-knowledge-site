20:06 - 財報會計陷阱辨識（基本面分析）

文章：預先看出財報地雷－企業作假帳的6特徵、6警訊

企業作假帳的五大動機

是什麼：企業為何要冒險做假帳？理解動機有助於提早警覺。

五大動機：

1. 順利上市上櫃 — 為了符合上市櫃標準，美化財報
2. 掩飾本業衰退 — 怕財報難看導致股價下跌或銀行抽銀根
3. 避免大股東破產 — 大股東股票質押成數高（>60%），股價不能跌
4. 操縱股價 — 先低價買進，再假帳拉抬股價，最後高賣
5. 掏空公司 — 把公司資產搬進自己口袋

實例：

  • 情境：揚華(4703)假交易案
  • 操作：找不存在或呆滯品存貨，用人頭公司循環交易，營造營收成長假象
  • 判斷：應收帳款帳齡超過3個月、存貨天數不合理

適用時機：質押成數>60%的公司，股價不易跌但一跌如山崩

財報假帳六大特徵

是什麼：透過財報數字的異常模式，識別假帳可能性。

特徵1：過高的應收帳款天數

是什麼：應收帳款週轉天數遠高於同業。

怎麼做：
1. 計算應收帳款週轉天數 = 應收帳款 ÷ (營收 ÷ 365)
2. 與同業比較，若超過3個月（~90天）需警惕
3. 查看應收帳款帳齡分析表

實例：

  • 情境：揚華(4703)
  • 數字：應收帳款週轉天數異常偏高
  • 判斷：假交易利潤藏在應收帳款裡銷不掉

適用時機：所有公司，特別是營收快速成長但應收帳款同步暴增時

特徵2：過低的存貨天數

是什麼：平均銷貨日數不合理低於同業，可能是假交易虛增營收。

怎麼做：
1. 計算存貨週轉天數 = 存貨 ÷ (銷貨成本 ÷ 365)
2. 正常公司存貨約30-60天
3. 若低於10天甚至5天，需警惕

實例：

  • 情境：仕欽(6232) 2005-2007
  • 數字：存貨天數 8.12天 → 6.80天 → 3.39天
  • 同時：應收帳款天數 135天 → 175天 → 205天
  • 判斷：「應收帳款過高 + 存貨過低」= 假帳機率90%

適用時機：與同業比較時發現存貨天數異常偏低

特徵3：過高的不動產、廠房及設備

是什麼：應收帳款突然消失，不動產大幅增加，可能透過高價購資產消化假帳。

怎麼做：
1. 觀察應收帳款科目變化
2. 比較不動產週轉率與同業
3. 若應收帳款突然大降、不動產大增，注意

實例：

  • 情境：公司以高於市價買資產
  • 操作：假應收帳款轉入「不動產、廠房及設備」
  • 判斷：不動產週轉率較低於同業

適用時機：應收帳款科目異常變化時

特徵4：過高的長期投資

是什麼：透過高價投資未上市櫃公司消化應收帳款。

怎麼做：
1. 查看長期投資科目
2. 若投資未上市公司 >50% 股權，看商譽科目
3. 若投資 <50%，看「按權益法之投資」科目

實例：

  • 情境：樂陞(3663)併購案
  • 數字：出事後打掉約26億元商譽及其他無形資產
  • 判斷：未上市櫃公司價格難推算，容易藏假

適用時機：公司頻繁併購未上市公司時

特徵5：過高的現金

是什麼：現金很多但同時借款很多，現金不能動用（被限制）。

怎麼做：
1. 看現金及約當現金金額
2. 看長短期借款金額
3. 對比利息收入 vs 利息費用
4. 若現金高於8個月營運、借款也很高 = 異常

實例：

  • 情境：博達(2398) 2002年
  • 數字：現金47億、借款92億、利息收入0.19億、利息費用4.08億
  • 判斷：手上現金很多卻不還借款，現金可能被限制

適用時機：現金很多但同時借款很多的矛盾狀況

特徵6：過高的雜項資產

是什麼：其他應收款、預付款項、存出保證金等科目金額異常。

怎麼做：
1. 查看資產負債表的「其他資產」
2. 比較預付款項與同業
3. 金額很高且同業沒有 = 警訊

適用時機：雜項資產佔比異常高時

假帳六大非財務警訊

警訊1：老闆誠信被質疑

是什麼：市場耳語或媒體報導老闆誠信問題。

怎麼做：
1. 搜尋公司老闆的新聞
2. 查公開資訊觀測站的董監質押情形
3. 質押成數>80% = 很缺錢

實例：

  • 情境：和旺(5505) 2015
  • 數字：董監質押從14% → 100%
  • 判斷：質押逐年增加 = 需錢孔急

警訊2：經營階層變動

是什麼：財務長、獨立董事、董事接連離職。

怎麼做：
1. 注意財務長離職訊息
2. 注意獨董請辭訊息
3. 離職順序：財務長 → 獨董 → 一般董事 → 董事長

適用時機：公司負債比高 + 財會主管離職

警訊3：股價異常飆漲

是什麼：股價漲幅遠超同業且無基本面支撐。

怎麼做：
1. 比較同業股價表現
2. 技術與規模相近的公司，股價不應差太多
3. 漲到「玉山高度」= 可能炒股

警訊4：市場利多消息不斷

是什麼：公司頻繁發布利多，常澄清「不予置評」。

怎麼做：
1. 查公開資訊觀測站的「重大訊息」
2. 若利多澄清連續多次 = 公司自己放的
3. 目的：炒股票

實例：

  • 情境：仕欽(6232)
  • 操作：對外宣稱手機訂單大增、大廠有意入主
  • 結果：頻繁發布更正或澄清，最後證明子虛烏有

警訊5：借殼後初期階段

是什麼：借殼上市後3-4年內，營收驚人成長但現金流為負。

怎麼做：
1. 查公司是否借殼上市
2. 借殼後營收連續成長 >50% = 警訊
3. 營業現金流連續為負 = 更大警訊

實例：

  • 情境：揚華(4703)前身「金美克能」
  • 數字：2012借殼後，營收從6.3億 → 14.7億 → 29.2億（成長91% → 136% → 98%）
  • 同時：營業現金流都是負的

適用時機：投資借殼3年內的公司

警訊6：產業不好獲利卻異常

是什麼：公司技術普通、產業景氣差，但獲利穩定超前。

怎麼做：
1. 研究產業景氣
2. 比較同業獲利
3. 技術普通的公司在景氣差時不該賺錢

實例：

  • 情境：雅新(2418) 印刷電路板
  • 數字：當時景氣差，雅新每年穩定賺EPS 2元多
  • 結果：董事長承認虛增營收作假帳

假帳偵測心法

是什麼：結合財報特徵與非財務警訊的偵測方法。

怎麼做：
1. 先看老闆質押：>80% = 小心
2. 再看應收帳款天數：>90天 = 警訊
3. 看存貨天數：<10天且與同業差距大 = 警訊
4. 看現金與借款：雙高 = 現金可能被限制
5. 看營業現金流：營收成長但現金流為負 = 怪
6. 看利多頻率：澄清「不予置評」太多 = 放消息炒股
7. 看借殼歷史：借殼後3-4年內營收暴增 = 警訊
8. 研究產業：產業差卻賺錢 = 可能有假

適用時機：每次選股都要做這些基本查核

心得：

這篇是「大會計師教你從財報數字看懂經營本質」的精華摘錄，非常實用。台灣每2-3年就有上市櫃公司財報造假案件，從揚華、雅新、博達到和旺，手法大同小異：虛增營收 → 應收帳款爆增 → 找方法消化（不動產、長投、限制性現金）。

最關鍵的偵測點：「應收帳款天數過高 + 存貨天數過低」= 假帳機率90%。這是「低級錯誤」，但多數假帳公司都會犯。

另一個重點：假帳通常做了2-3年才被抓到。所以避雷的關鍵是提前發現徵兆，而不是相信媒體利多。市場利多不斷、公司澄清「不予置評」的，反而是最危險的。

投資前必查：
1. 董監質押成數（>80% 小心）
2. 應收帳款天數（>90天）
3. 存貨天數（<10天）
4. 現金與借款雙高
5. 借殼後3年內的營收成長是否異常
6. 產業景氣 vs 公司獲利是否矛盾

這些都是「免費」資訊，查一下就能避開很多地雷。懶得查的下場就是踩雷。

21:06 - 企業掏空手法辨識（基本面分析）

文章：如何掏空一家公司、關係人交易查核之審計品質缺失懲戒案例介紹

企業掏空的定義

是什麼：公司大股東或管理層以不合法手段轉移上市公司資源，損害中小股東利益。

關鍵特點：不同於盜竊公司資產，掏空者會讓各項安排符合法律手續，使其更加隱蔽。

掏空手法初級班

是什麼：結構簡單，透過關係人交易轉移利潤。

常見手法：

1. 訂單過水 — A公司賺100萬的訂單，改由老闆控制的B公司接單再轉包給A，B賺70萬、A賺30萬
2. 指定進貨 — 重要原料由老闆私人公司代理，以集團整合為由強制採購
3. 指定發包 — 擴廠、買資產都指定向老闆公司購買，中間賺一手
4. 成立投資部門 — 投資老闆私人公司，資金轉化為私人花費
5. 關係人高價買資產 — 董事會通過以高於市價向老闆或關係人買土地、房屋
6. 擔保抵押 — A公司為B公司借款擔保，掏空B後A負擔賠償
7. 代養私人公司 — 老闆私人公司的人事費用掛在上市公司
8. 認購關係人債券 — 老闆私人公司發債，上市公司認購
9. 庫藏股護盤 — 公司買庫藏股，大股東趁機出脫
10. 資產灌水超貸 — 高估資產價格向自家金融機構借貸
11. 指定買賣仲介費 — 公司買特定金融商品，仲介費進私人口袋
12. 直接轉帳 — 以業主往來名義將公司現金轉入老闆海外帳戶

實例：

  • 情境：中強(1441)
  • 操作：對無力付款的子公司大量銷貨
  • 結果：帳面有營收有獲利，但關係人應收帳款收不回來

適用時機：上市公司有大量關係人交易時

掏空手法中級班

是什麼：牽涉多家公司或海外公司，需要專業人士指導。

常見手法：

1. 虛增營收 — 銷貨給海外子公司，再要求供應商買回存貨，形成循環銷貨
2. 切割金雞母 — 有價值部門獨立成子公司，透過增資稀釋母公司股權
3. 金融工具套利 — 發行海外可轉債(ECB)，認股權由大股東認購，股價上漲獲利
4. 海外子公司隱匿 — 利潤留在海外子公司，購買不記名債券後盜賣

實例：

  • 海外子公司現金購入不記名債券
  • 企業主盜賣資產，現金轉入個人帳戶

適用時機：公司有大量海外子公司或複雜金融操作時

掏空手法高級班（博達案）

是什麼：結合國際投資銀行、衍生性金融商品，手法複雜難辨。

手法1：信用連結債券(CLN)虛增銀行存款

怎麼做：
1. 在維京群島設立紙上公司「北亞金融」
2. 由北亞金融委託法國興業銀行發行8500萬美元CLN
3. 博達海外假代理商匯入8500萬美元至博達銀行帳戶
4. 博達與首都銀行簽約，以存款為擔保購買CLN

結果：

  • 財報：應收帳款下降、現金存款增加
  • 真相：現金早就被凍結，博達重整時被銀行扣除
  • 8500萬美元不知所蹤

手法2：應收帳款融資+零息債券

怎麼做：
1. 在維京群島設立紙上公司「AIM Global」
2. 以9折出售五大人頭客戶的應收帳款給CBA銀行
3. AIM Global買下這批應收帳款，發行零息債券
4. 博達以CBA存款為擔保指示CBA買入債券

結果：

  • 財報：應收帳款下降、銀行存款增加
  • 真相：存款被限定，博達違約時銀行直接扣除

手法3：海外可轉債套利

怎麼做：
1. 在維京群島設立Best Focus、Fernvale二家公司
2. 由博達背書保證，向羅伯銀行、首都銀行借款5000萬美元
3. 二家公司買下博達發行的無擔保海外可轉債
4. 博達發行可轉債成功後，5000萬美元需回存銀行並限制動用

操作：

  • 股價上漲至17.4元，高於轉換價15.08元
  • 子公司全部轉換為普通股拋售
  • 所得5314萬美元：部分匯回博達、部分回流葉素菲私人帳戶
  • 向羅伯銀行借款4000萬美元未還，隨博達重整被沒入

手法4：為海外人頭公司擔保借款

怎麼做：
1. 設立Addie公司
2. 博達存入1000萬美元作為擔保
3. 建華銀行海外子公司借款1000萬美元給Addie
4. 博達簽承諾書：Addie無法還款時銀行可扣取存款

結果：

  • 財報：1000萬美元定存
  • 真相：名存實亡的現金，博達重整時被銀行取走

掏空防範警訊

是什麼：投資人應注意的掏空徵兆。

三大現象：

1. 大股東一股獨大

  • 單一股東對公司施以絕對控制

2. 內部監管不健全

  • 股東大會只有大股東參加

3. 關係人交易異常

  • 交易條件較優厚（收付款期間相同、條件彼此關聯）

查核案例（關係人交易）：

  • 情境：A公司總部大樓承租人B公司
  • 警訊：應收帳款帳齡較長、交易條件優厚、辦公處所同一
  • 會計師缺失：僅以口頭答覆和聲明書認定非關係人

實例：

  • 情境：出售機器設備給關係人C公司
  • 警訊：

關係人交易查核重點

怎麼做：

1. 識別關係人

  • 查看財報附註的關係人揭露

2. 評估交易條件

  • 比較與公平交易的條款是否相當

3. 查核佐證資料

  • 會議紀錄、契約文件

適用時機：

  • 公司有大量關係人交易
  • 關係人交易占營收/進貨比例過高
  • 關係人應收帳款帳齡過長

心得：

掏空手法從初級到高級，複雜度差異很大，但核心邏輯一致：把上市公司資源搬進大股東口袋。博達案是經典的「高級班」案例，葉素菲透過國際投資銀行、衍生性金融商品，把簡單的掏空包裝成複雜的金融操作。

最精妙的設計是「信用連結債券」和「零息債券」手法：在財報上看到的是「現金增加+應收帳款下降」，帳面數字變漂亮，但實際上現金早就被凍結。投資人根本無從得知。

這讓我想到博達案的教訓：
1. 現金很多但同時借款很多 = 現金可能被限制
2. 利息收入太低 = 現金可能不能動用或已被質押
3. 海外子公司太多太好賺 = 利潤可能留在海外
4. 複雜金融商品 = 可能是掏空的包裝

博達案後，台灣訂了很多公司治理規範，但國際化和金融創新讓掏空手法更難察覺。投資人能做的：
1. 避免單一大股東持股過高的公司

22:06 - 會計師查核報告判讀（基本面分析）

文章：【財報教學】2 種查帳方式與 4 種查核意見...原來，最重要的財報是「它」！

查帳兩種方式：核閱 vs 查核

是什麼：會計師對財報的審查深度不同，影響財報可信度。

核閱（Q1-Q3季報）

是什麼：會計師取得公司自結數字，在辦公室核算有沒有問題。

怎麼做：
1. 公司提供自結報表數字
2. 查帳員核算數字是否合理
3. 針對兩期金額差異大的科目詢問分析
4. 可能調整公司給的數據

特點：

  • 只坐在辦公室按計算機
  • 不會盤點、不會發函證
  • 可信度較低

適用時機：Q1、Q2、Q3季報（金融保險業例外）

查核（Q4年報）

是什麼：除了核算數字外，還會盤點、發函證等實地查核。

怎麼做：
1. 盤點 — 年底親自到工廠盤點存貨、設備、現金（含海外據點）
2. 銀行函證 — 直接向銀行確認公司帳戶餘額（不經手公司員工）
3. 律師函證 — 向委任律師詢問訴訟官司、勝敗機率
4. 關係人函證 — 向關係企業確認應收帳款、存出保證金

實例：

  • 情境：會計師查核某公司
  • 操作：直接向銀行窗口確認帳戶餘額，不經過公司員工
  • 目的：避免公司偽造銀行對帳單

適用時機：Q4年報（所有公司）、Q2（金融保險業）

四種會計師查核意見

1. 無保留意見：財報沒問題

是什麼：會計師認為財報在所有重大方面可信，投資人可放心參考。

怎麼判斷：

  • 查核報告寫「允當表達」
  • 沒有任何但書或強調事項

實例：

  • 正常公司的年報都是無保留意見
  • 這是基本門檻，不代表公司體質好

適用時機：投資前提，拿到「非無保留意見」的公司直接跳過

2. 保留意見：某方面有疑慮

是什麼：財報某方面存在疑慮，但整體還算可信。

兩種情況：
1. 重大不實表達（非廣泛） — 某科目有問題但不影響整體
2. 無法取得足夠證據（非廣泛） — 某部分查不到但影響有限

關鍵判斷：

  • Q1-Q3季報的保留意見：常見，因為只是核閱，很多子公司沒仔細查帳
  • Q4年報的保留意見：大有問題！所有子公司都查了還有保留 = 事情大條

實例：

  • 情境：英瑞-KY(1592) 110Q1
  • 數字：會計師出具「保留意見—繼續經營能力存在重大不確定性」
  • 結果：無法如期公告財報，停止上市買賣

適用時機：Q4年報出現保留意見 = 立即警覺

3. 否定意見：財報整體不可信

是什麼：會計師已取得足夠證據，但發現財報有重大錯誤或違反會計原則。

怎麼判斷：

  • 影響是「重大且廣泛」
  • 非常少見，一出現就是大事

實例：

  • 情境：勝華(2384) 103Q3
  • 數字：會計師出具「否定式核閱報告」
  • 原因：
  • 結果：Q4找不到會計師簽證，無法繳交財報，下市

適用時機：看到否定意見 = 準備跑

4. 無法表示意見：查不到

是什麼：會計師無法取得足夠資訊審核，可能是公司給的資料不完整。

怎麼判斷：

  • 公司阻撓查核
  • 資料不完整或無法驗證
  • 影響「重大且廣泛」

適用時機：公司不配合查核 = 心裡有鬼

為什麼金融業Q2也是查核：

  • 掌握大量民眾資金（保費、存款）
  • 現金會再投資（股票、債券）
  • 擔心投資虧損或挪用
  • 主管機關要求更嚴格監督

Q4年報資訊最精確

是什麼：年報的查帳方式是「查核」，可信度最高。

為什麼：
1. 查核方式最嚴謹 — 盤點、發函證都做
2. 會計師一次調整入帳 — 如退休金估算
3. 獨家資訊只在年報揭露 — 如投資性不動產公允價值

怎麼做：

  • 投資決策以年報資訊為主
  • 季報用來追蹤趨勢，不做重大決策依據

實戰應用：怎麼看會計師查核報告

怎麼做：
1. 到公開資訊觀測站 — 輸入股票代碼 → 電子書 → 財務報告
2. 翻到會計師查核報告 — 在財報前幾頁
3. 看意見類型：

  • 「無保留意見」= 正常

4. 看強調事項段 — 可能有繼續經營疑慮、重大訴訟等

實例應用：

  • 情境：篩選股票時
  • 步驟：先查會計師意見，非無保留意見的直接剔除
  • 理由：會計師都覺得有問題，一般投資人沒必要冒險

適用時機：

  • 每次選股都要先查會計師意見
  • 收到財報時第一件事就是看查核報告
  • Q4年報的意見最關鍵

心得：

這篇補上了一個重要觀念：財報可信度要看「查帳方式」。Q1-Q3季報是核閱，會計師坐在辦公室按計算機，沒盤點、沒發函證；Q4年報是查核，會計師親自去工廠盤點、直接問銀行帳戶餘額。這兩者的可信度差很多。

關鍵點：
1. Q4年報最準 — 投資決策以年報為主
2. Q4出現保留意見 = 大問題 — 季報保留意見常見，年報保留意見少見
3. 否定意見、無法表示意見 = 直接跳過 — 沒必要跟會計師的看法對作

英瑞-KY和勝華的案例很有教育意義：會計師的意見是「提前警告」。英瑞-Q1保留意見後，無法交財報就停交易；勝華-Q3否定意見後，Q4找不到會計師就下市。投資人如果會看查核報告，其實有時間跑。

操作建議：

  • 選股前先查會計師意見（公開資訊觀測站）
  • 非無保留意見的直接剔除
  • Q4年報的意見最關鍵
  • 收到財報第一件事：翻到會計師查核報告

這是免費的「專家意見」，不用白不用。

23:06 - 管理階層品質評估（基本面分析）

文章：巴菲特最推崇的8大企業家特質、好的CEO能幫公司賺錢...股票才會漲！「4特質」

為什麼要評估管理階層

是什麼：投資就是投人，CEO直接決定企業能做多大、走多遠。

巴菲特說過：「這輩子從來沒有遇過跟壞人打交道能做成好生意的。」

關鍵點：

  • 企業失敗案例中，只有一小部分是因為決策者才能有限
  • 更多是因為「方向或心術不正」導致的失敗
  • 有能力但心術不正的經營者，比沒能力更危險

適用時機：投資任何公司前，先評估經營階層品質

巴菲特推崇的CEO評估標準

是什麼：巴菲特評估CEO的兩大核心標準。

標準1：相對回報率而非絕對回報率

怎麼做：
1. 不要只看CEO任期內股價漲多少
2. 要比同期大盤表現（如S&P 500）
3. 比同期同業表現
4. 計算「超額報酬」= 公司報酬 - 基準報酬

實例：

  • 情境：巴菲特推崇的8位企業家
  • 數字：平均回報超過同期S&P 500指數20倍
  • 對比：Jack Welch帶領GE的回報「只有」S&P的3.3倍

適用時機：評估任何CEO的績效

標準2：誠信與資本配置能力

怎麼做：
1. 查大股東質押成數（>80% = 小心）
2. 查公司資金運用效率（ROE、自由現金流）
3. 查併購紀錄（是創造價值還是毀滅價值）
4. 查買回紀錄（低買高賣還是高買低賣）

適用時機：投資前必查

優秀CEO的8大特質（巴菲特推崇）

特質1：效率決定生死

是什麼：不是追求KPI效率，而是透過授權激發內驅力。

怎麼做：
1. 避免中心化管理（所有決策都要總部同意）
2. 充分授權，讓員工發揮創造力
3. 讓部門自給自足，無法自給的立即處理

實例：

  • 情境：大都會廣播公司 vs 哥倫比亞廣播公司
  • 大都會：去中心化管理，總部沒有層層官僚，收視率第一
  • 哥倫比亞：中心化管理，光總裁+副總裁就42人，處處需要總部同意
  • 結果：皮划艇打敗大船

適用時機：評估公司組織效率，看是否「人治」還是「制度」

特質2：聚焦資本配置

是什麼：CEO最重要的工作是資本配置，每一塊錢都要有明確報酬。

怎麼做：
1. 不投資「會發光」的標的，投資「能穩定創造現金」的資產
2. 不做年度預測，不靠願景決策
3. 像會計師一樣看公司，像資本家一樣決策
4. 每筆投資都要經得起時間與數字檢驗

實例：

  • 情境：Tom Murphy（大都會廣播公司CEO）
  • 策略：只收購能穩定創造現金的資產
  • 結果：成為美國最賺錢的媒體公司之一

適用時機：評估CEO的併購、投資決策紀錄

特質3：綁定股東利益

是什麼：CEO利益與股東一致，不是為了自己薪酬。

怎麼判斷：
1. 查CEO持股比例（太少 = 激勵不足）
2. 查薪酬結構（固定薪資 vs 績效獎金 vs 股票選擇權）
3. 查是否長期持有（賣股 vs 持有）
4. 查是否有「金降落傘」（離職金過高）

適用時機：評估管理階層是否「同在一艘船」

特質4：高效運用槓桿

是什麼：不是不敢用槓桿，是知道何時用、如何用。

怎麼做：
1. 用槓桿買「穩定現金流」的資產
2. 不用槓桿做高風險擴張
3. 保持安全邊際，預留現金度過寒冬
4. 稻盛和夫：「企業會遇到各種問題，要留足夠現金生存」

實例：

  • 情境：大眾影院公司CEO Dick Smith
  • 策略：售出租回（sale-leaseback），賣影院給投資人再租回經營權
  • 結果：手上有充足現金面對市場波動

適用時機：評估公司財務槓桿是否適當

特質5：勇於逆向前行

是什麼：在市場恐慌時買進，在市場狂熱時保持冷靜。

怎麼做：
1. 不問未來會變怎樣，問現在該做什麼
2. 像撲克玩家：手握資本，等待市場錯價
3. 時機成熟就出手，不成熟就讓資金留在口袋
4. 承擔風險但不盲目賭博

實例：

  • 情境：華盛頓郵報CEO凱瑟琳·格雷厄姆
  • 背景：經濟蕭條，同行拋售資產
  • 操作：逆勢買入電視台、移動通信公司、教育公司
  • 結果：新業務創造公司一半收入和利潤

適用時機：看CEO在危機時的反應

特質6：數據思維制勝

是什麼：用數據做決策，不靠直覺或願景。

怎麼做：
1. 決策前看數據，不憑感覺
2. 追蹤關鍵指標並持續優化
3. 數據驅動而非KPI驅動

適用時機：評估CEO是否有數據思維習慣

特質7：永遠關注現金流

是什麼：現金是企業的血，沒有現金就沒有未来。

怎麼做：
1. 檢視自由現金流的穩定性
2. 現金流為負要看原因（擴張還是虧損）
3. 利潤不等於現金，有現金才能活下去
4. 預留至少3年收入中斷的現金準備

實例：

  • 情境：湛廬文化
  • 策略：CEO要求「沒有收入也能維持3年」的現金儲備
  • 結果：疫情期間沒裁員、沒減薪，正常運作

適用時機：評估公司的現金部位是否健康

特質8：首席執行官就是首席投資官

是什麼：CEO做的每一個決策都是投資決策。

怎麼做：
1. 不喊口號，把自由現金流最大化
2. 不賺錢的部門敢於果斷處理
3. 把價值還給股東（買回或配息）
4. 不為了「偉大」而擴張，為了「報酬」而配置

實例：

  • 情境：Bill Anders接手General Dynamics
  • 操作：接管龐大、繁複、效率低的軍工集團
  • 策略：大幅出售不賺錢或不熟悉的業務，把現金拿來回購股票
  • 結果：市值倍增，股東回報成為事實

適用時機：看CEO是否有「投資人思維」

好CEO的4個核心特質（實戰版）

1. 決策邏輯透明、紀律清晰

是什麼：不追求成長神話，釐清每一筆資金的最佳用途。

怎麼判斷：

  • CEO是否花時間談「願景」？太多願景 = 缺乏紀律
  • 是否清楚說明資金用途和預期報酬？
  • 是否有明確的買回/配息政策？

2. 不迷信願景與熱潮

是什麼：投資報酬率高、現金流穩定的資產，勝過熱門題材。

怎麼判斷：

  • 查併購紀錄：是買熱門題材還是穩定現金流？
  • 查業務組合：是追逐風口還是深耕本業？
  • 查研發方向：是炒作題材還是務實創新？

3. 擅長處置不賺錢的部門

是什麼：不拖延、不戀棧，清理資產、回饋股東。

怎麼判斷：

  • 查公司歷史：有沒有果斷出售不賺錢部門的紀錄？
  • 查虧損部門：拖了多久才處理？
  • 查買回紀錄：股價低時買回還是高時買回？

4. 偏好沉默與低調

是什麼：不需要話術說服市場，報酬會說話。

怎麼判斷：

  • 查媒體曝光：是否頻繁上媒體談「願景」？
  • 查法說會：是說明成果還是畫大餅？
  • 查澄清頻率：「不予置評」太多 = 可能放消息炒股

偽CEO特質（避雷）

是什麼：這些特質代表CEO可能有問題。

1. 頻繁談願景、很少談報酬

警訊：

  • 法說會都在畫大餅
  • 很少提到股東報酬
  • 頻繁發布利多消息

2. 併購紀錄不亮眼

警訊：

  • 併購多但商譽減損多
  • 併購後業績不如預期
  • 愛買熱門題材公司

3. 股價高時買回、股價低時發行

警訊：

  • 大漲後宣布買回庫藏股
  • 股價低時宣布現增
  • 選擇權在股價高時大量執行

4. 現金流與獲利脫節

警訊：

  • EPS很好但自由現金流為負
  • 營收成長但應收帳款暴增
  • 大量併購導致現金流惡化

實戰應用：如何評估管理階層

怎麼做：

1. 查CEO背景

  • Google CEO姓名 + 公司名稱

2. 查薪酬結構

  • 年報有董監酬勞揭露

3. 查持股變動

  • 內部人持股明細看大股東是否持續減持

4. 查併購紀錄

  • 過去5年併購案

5. 查買回紀錄

  • 過去買回庫藏股的價格和時機

6. 查語錄和新聞

  • 法說會紀錄

適用時機：每次選股都要做這些查核，跟查財報一樣重要

成功CEO的3個共同特質（補充）

1. 強大的學習力

是什麼：終身學習，不斷迭代認知。
