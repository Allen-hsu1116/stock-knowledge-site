一、分點籌碼分析

什麼是分點籌碼？

定義：證券商在各處所設立的分公司，證交所每天會公告每檔股票在各個分點的進出價位與張數。

與法人籌碼的差異：

  • 法人籌碼：研究外資、投信、自營商的進出
  • 分點籌碼：研究本土分點上的大戶與主力

為什麼分點籌碼可能更強？

分點上的大戶往往具有特殊身分：

  • 大股東、公司經理人、高階主管
  • 董監經理人的親戚摯友
  • 公司派作手、市場派炒手

這些人對公司股價的敏感度比法人高很多，通常是第一手進場布局，之後才輪到法人來抬轎。同樣的，他們也是跑最快的一群，總在行情最熱時倒貨給法人。

二、法人籌碼的三大特性

1. 外資 — 大盤漲跌的主導者

特性：

  • 買賣超金額最大，主導大盤走勢
  • 創新高時外資站在買方
  • 從買轉賣、或從賣轉買是重要轉折訊號

實戰範例：

  • 2022年11月底外資從賣超轉買超，台股大幅反彈
  • 1月3號外資大賣228億，台股拉出長黑K

怎麼做：
1. 到 Yahoo 股市或證交所查看三大法人買賣超
2. 觀察外資的連續變化（連續買超或連續賣超）
3. 注意「從買轉賣」或「從賣轉買」的轉折點

2. 投信 — 左側交易者

特性：

  • 喜歡在台股下跌時逢低買進、逢高賣出
  • 低買高賣的操作手法

實戰範例：

  • 2020大空頭時投信一路買進
  • 等到台股漲到相對高漲時，把底部買的股票賣出

3. 自營商 — 高檔調節者

特性：

  • 逢低買進、逢高賣出
  • 當價格上漲時會不斷調節
  • 高檔時幾乎都是賣超

三、兩個被忽視的強大指標

1. 大戶持股比率

定義：持有量超過1000張的投資人持股比率（實務上可視個股調整，如華映50張就算大戶，大立光5張就算大戶）

怎麼判讀：

  • 比率持續上升 → 大戶準備拉抬
  • 比率持續下降 → 大戶在倒貨

實戰操作：
1. 查詢集保戶股權分散表
2. 追蹤千張大戶持股比率變化
3. 比率連續上升時可考慮跟進

2. 集保戶數

定義：每個人只有一個集保戶，所以集保戶數 = 持有該股票的人數

怎麼判讀：

  • 集保戶數持續上升 → 股票散落在散戶手中，主力可能在倒貨
  • 集保戶數持續下降 → 有人不斷從市場買入，主力吸收股票準備拉抬

黃金組合：

  • 大戶持股比率上升 + 集保戶數下降 = 主力吸籌，準備上攻
  • 大戶持股比率下降 + 集保戶數上升 = 主力倒貨，風險增加

選股步驟

2. 將前幾名列入觀察名單
3. 注意：自營商通常買ETF，比較不需要看

實戰範例：

  • 群光（2385）：投信連續買超 + 外資站買方，股價翻倍
  • 長榮（2603）：投信連續買超 + 外資站買方，股價往上拉

均線控盤

怎麼做：
1. 打開技術線圖，加上均線
2. 均線斜率往上（紅色）抱住股票
3. 均線斜率往下（綠色）考慮出場

關鍵：

  • 籌碼連續買超 + 均線往上 = 抱住波段
  • 這種股票容易走連續大段走勢

五、期貨盤中籌碼

1. 委買委賣口差（買賣力道）

定義：統計市場上全部的買盤及賣盤口數

怎麼判斷：

  • 委買口數 > 委賣口數 → 買盤強，容易漲
  • 委賣口數 > 委買口數 → 賣盤強，容易跌

實戰：

  • 委買5000口、委賣2000口 → 買盤多出3000口，看多

2. 成交筆差

定義：統計追多成交筆數 vs 追空成交筆數（累積制，從開盤累積到收盤）

怎麼判斷：

  • 追多成交筆數 > 追空成交筆數 → 價格往上拉抬
  • 追空成交筆數 > 追多成交筆數 → 價格往下跌

實戰：

  • 開盤至今追多1000筆、追空300筆 → 追多700筆，市場不斷追高

進場判斷關鍵

同向訊號：

  • 委買口差大 + 成交筆差大 = 買盤強，行情上漲
  • 委賣口差大 + 成交筆差大 = 賣盤強，行情下跌

數據門檻：數據要到達至少1000口以上，才較容易走趨勢盤

六、散戶期貨籌碼（反指標）

定義：三大法人以外的即為散戶，三大法人口數加總 × (-1) = 散戶數據

怎麼判斷：

  • 散戶做多 → 通常被收割，行情反向下跌
  • 散戶做空 → 通常被收割，行情反向上漲

實戰範例：

  • 散戶在行情大漲時幾乎都放空
  • 外資賣超200多億時散戶才開始做多，結果從上面抱下來
  • 最後兩天外資回頭大買，台股上漲600點，散戶又反空

七、實戰策略：進場加碼走

核心邏輯

「看錯就走，看對加碼」

風險管理：

  • 最大部位：例如一次一口，上限兩口
  • 初始停損：例如40點（視波動調整）
  • 停利條件：例如賺120點就停利
  • 移動停利：用33%日ATR做移動停利
  • 加碼間距：例如每40點加碼一口

賺賠比：40點虧損換120點獲利 = 1:3

做多範例

1. 判斷行情從綠轉紅，第一個紅點進場設停損
2. 行情走出去，第二個紅點加碼
3. 到達獲利滿足點，第三個紅點停利出場

案例：2萬多風險換取25800元獲利

做空範例

1. 行情走弱，第一點進場設停損
2. 程式自動找加碼點及移動停利
3. 到達獲利滿足點停利出場

案例：兩筆交易各賺2萬多，合計48400元

心得：

今天深入研究了分點籌碼和盤中籌碼，發現這兩塊是籌碼分析的進階版。分點籌碼能看到本土主力、大股東的動向，這些人往往比法人更早進場、更早出場。而大戶持股比率和集保戶數這兩個指標，簡單但強大，不用每天盯盤也能看出主力是在吸貨還是倒貨。

盤中籌碼的委買口差和成交筆差，對於期貨當沖很有幫助。重點是兩個數據要同向，且數據要到達1000口以上才容易走趨勢盤。這讓我想到以前常犯的錯：看到小數據就以為有趨勢，結果只是雜訊。

散戶籌碼當反指標很有趣，但重點不是跟散戶對做，而是理解市場情緒。當散戶都做多時，行情往往快反轉了。
