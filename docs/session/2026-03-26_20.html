核心知識

VWAP 成交量加權平均價格

是什麼：VWAP (Volume Weighted Average Price) 是用成交量和價格計算的加權平均價，反映當日市場的「真實平均成本」。與傳統均線不同，VWAP 加入成交量權重，更能反映真實交易成本。

公式：

VWAP = Σ((開盤價+最高價+最低價+收盤價)/4 × 成交量) / Σ總成交量

怎麼算（Excel 實例）：
1. 計算每根 K 棒的典型價格：(最高價 + 最低價 + 收盤價) / 3
2. 乘以成交量得到加權價格
3. 累加所有加權價格，除以累計成交量

實例：

  • 09:30 成交 100 股，典型價 $50 → 加權 $5,000
  • 09:45 成交 200 股，典型價 $49 → 加權 $9,800
  • 累計：1500 股，加權總額 $71,700
  • VWAP = $71,700 / 1,500 = $47.8

VWAP 核心特性

每日歸零：VWAP 從開盤開始計算，每日收盤後歸零，隔天重新計算。

為什麼重要：

  • 傳統均線容易被跳空干擾，價格失真
  • VWAP 只計算當天，不受隔夜跳空影響
  • 更精確反映「當日」多空平均成本

三大優勢：
1. 考慮成交量，反映真實成本（高量大價對 VWAP 影響大）
2. 每日重算，對當沖最精準
3. 機構法人常用做交易執行基準

兩大限制：
1. 滯後指標 — 反應較慢，盤中後半段更慢
2. 不適合多日趨勢分析 — 只看當天

VWAP 實戰應用

一、判斷多空方向

怎麼做：

  • 股價在 VWAP 上方 → 多頭有利，平均成本在下方
  • 股價在 VWAP 下方 → 空頭有利，平均成本在上方
  • 股價接近 VWAP → 多空平衡

實例：

  • 台指期現價 23,200，VWAP = 23,150
  • 判斷：價格在 VWAP 上方 50 點，多頭佔優
  • 操作偏多

二、支撐壓力判斷

怎麼做：
1. 價格回測 VWAP 彈起 → VWAP 是支撐，可試多
2. 價格跌破 VWAP 無法站回 → VWAP 變壓力，偏空操作
3. 等待確認：不是碰到就進，要看「彈起」或「跌破站不回」

實例：

  • 價格從 23,300 回測 VWAP 23,150，接著反彈到 23,200
  • VWAP 確認為支撐
  • 進場點：回測 VWAP 彈起時
  • 停損：跌破 VWAP 下方 30 點

三、VWAP 回調策略

怎麼做：
1. 找價格大幅波動的股票
2. 計算 VWAP
3. 等價格回落到 VWAP 附近
4. 順原方向進場（原漲→做多，原跌→做空）
5. 停損設 VWAP 略下/略上

實例：

  • 早盤股價從 100 元漲到 108
  • VWAP 目前在 104
  • 等價格回落到 104~104.5
  • 進場做多，停損 103.5
  • 目標：回測後恢復上漲

四、VWAP 通道策略

怎麼做：
1. 在 VWAP 上下各畫 2 個標準差通道
2. 價格碰上緣 → 趨漲，可能反轉做空
3. 價格碰下緣 → 超跌，可能反彈做多
4. 停損設通道外側

關鍵：通道寬度用標準差，比固定百分比更科學。

VWAP + 其他指標組合

VWAP + 均線

組合方式：

  • VWAP：當日平均成本
  • 5MA/10MA：短期趨勢
  • VWAP 上穿均線 → 短期轉多
  • VWAP 下穿均線 → 短期轉空

VWAP + RSI

組合方式：
1. 價格遠離 VWAP 且 RSI 超買（>70）→ 注意反轉
2. 價格跌破 VWAP 且 RSI 超賣（<30）→ 可能反彈
3. 用 VWAP 找位置，用 RSI 找時機

期貨當沖三種進場方式

這部分補充「期貨當沖技巧」實務：

1. 測底拉回買進

怎麼做：

  • 在「下跌後上漲」的走勢中
  • 等價格拉回測底
  • 買在「起漲點」

實例：

  • 台指期早盤殺到 23,100，之後回升
  • 等價格回測 23,100~23,120 區間
  • 站穩後進場，停損 23,080

2. 突破買進

怎麼做：

  • 價格突破區間高點或箱型上緣
  • 方向確認後進場
  • 注意：可能買在高點

實例：

  • 區間整理 23,100~23,200
  • 突破 23,200 時進場
  • 停損：跌破區間上緣

3. 突破回測買進

怎麼做：

  • 等突破後回測原壓力變支撐
  • 較安全但「回測不一定發生」

實例：

  • 突破 23,200 後等回測
  • 回測不破 23,200 再進場
  • 停損：跌破 23,180

關鍵提醒：回測不一定發生，強勢盤可能直接飛，要能接受「沒買到」。

期貨當沖加碼條件

四大核心：

條件說明實例
加碼間距小20~30 點加碼，不是 100 點大台每 20 點加一筆
出場出得快獲利滿足點全出，不追到尾賺 160 點就出
停利跟得近加碼多，停利要近，風險才穩移動停利跟 20 點
守住成本全部出場時計算總損益，守住最大風險不賠超過初始本金的 2%

停損怎麼設：

  • 不是固定數字，要看波動
  • 用 ATR（平均震幅）的 1/4 或 1/5
  • 例：ATR = 200 點 → 停損 40~50 點

VWAP vs 移動均線 vs MVWAP

指標特性適用時機
VWAP每日歸零，量價加權當沖、日內
移動均線多日平均，只看價格波段、趨勢
MVWAP移動版 VWAP，不歸零多日留倉

心得

VWAP 是當沖交易的「真實成本線」，比傳統均線更精準，因為它用成交量加權，反映實際交易成本。這就是為什麼機構法人愛用 — 能知道自己買得比平均好還是差。

最實用的三個場景：
1. 判多空：價格在 VWAP 上下，直接告訴你誰佔優
2. 找支撐壓力：當日最乾淨的支壓，不用翻歷史
3. 進場時機：回測 VWAP 彈起 → 好的進場點

跟期貨當沖技巧結合，VWAP 可以取代「電話線」或「主力成本線」當作進場參考。三種進場（測底拉回、突破、突破回測）都可以用 VWAP 當基準。

缺點是盤中後半段 VWAP 會變鈍（累積太多資料），這時候要搭配其他指標（RSI、趨勢線）一起判斷。記住：VWAP 是工具，不是聖杯。
