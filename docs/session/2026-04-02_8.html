突破交易策略

第 8 回合學習 - 突破交易策略

突破交易核心概念

是什麼：避開盤整區間，等股價突破壓力後再進場，抓住趨勢起漲點。

為什麼有效：

  • 盤整區間是市場共識形成的價格帶
  • 突破代表新資金進場、市場願意用更高價格持有
  • 基於統計學和心理學原理

判斷真假突破四原則

原則一：以收盤價為準

怎麼做：

  • 盤中跌破不算跌破，收盤跌破才叫真跌破
  • 盤中追空容易遇到長下影線反彈，陷入兩難

為什麼重要：收盤價是多空雙方一天博弈的結果，最具參考價值

原則二：連續三天有效

怎麼做：

  • 連續三天的收盤都跌破支撐，才認定為真跌破
  • 連續三天的收盤都突破壓力，才認定為真突破
  • 若跌破兩天後第三天反彈，視為假突破

實例：

  • 情境：某股票跌破支撐100元
  • 判斷：第一天收盤99元、第二天收盤98.5元、第三天收盤98元 → 真跌破
  • 反例：第一天99元、第二天98元、第三天100.5元 → 假跌破

原則三：回測原方向（非必要）

怎麼做：

  • 跌破後小反彈測試，再往下續跌
  • 突破後小拉回測試，再往上續攻

效果：有回測的型態效果更好，但沒有也不妨礙真突破判斷

原則四：要有一定幅度

怎麼做：

  • 指數：連續三天跌破支撐，且超過 0.5%
  • 股票：連續三天突破壓力，且超過 3%

為什麼需要幅度：

  • 避免小波動被誤判為突破
  • 確保突破有實質力道

真假突破實戰判斷

量能觀察

怎麼做：

  • 真突破：量增，顯示資金願意追價
  • 假突破：放量卻漲不動，可能是主力拉抬出貨

跌破支撐後的反應

怎麼做：

  • 跌破支撐但隔日量能急縮 → 可能是假跌破，市場不願賣
  • 放量下跌 → 可能是真跌破，主力抽身

盤整時間與波段大小的關係

經驗法則：

想賺的波段需要的盤整時間
一個月至少 3 個月以上
半年1～2 年
一年以上5 年以上

為什麼：

  • 消息很即時，基本面變化一出現股價就會先反應
  • 長期沒有利多的公司，突然爆出好消息（虧轉盈、業績大成長）爆發力最強

最適合突破交易的標的：

  • 平常沒人關注的冷門股
  • 一有利多就能吸引市場目光
  • 例：疫情爆發時的航運股、AI 爆發時的 NVIDIA

不適合突破交易的標的：

  • 巨無霸公司（蘋果、波克夏）
  • 體量太大，單一利多難以引爆股價
  • 適合長期持有，不適合賺突破快錢

出場時機判斷

技術面出場

判斷原則：

  • 每次回檔都沒破前低 → 上漲趨勢還在，續抱
  • 連續跌破前波低點 → 動能減弱，可能反轉，該走

實例：

  • 情境：NVIDIA 2021 年從 $200 漲到 $330
  • 出場訊號：12 月底連續兩次跌破前波低點
  • 操作：獲利了結，不要戀戰

基本面/消息面出場

怎麼做：

  • 漲勢要看「資金」跟「故事」有沒有延續
  • 圖形破壞 + 消息不再 = 反轉機率高
  • 利多消失就該走

NVIDIA 案例：

  • 漲勢動力：拆股 + 比特幣大漲
  • 出場時機：比特幣價格開始走弱、跌破前波低點

實戰操作建議

進場：

  • 滿足第一原則（收盤價為準）即可進場
  • 雖然勝率會降低，但不會錯過大行情
  • 可等確認再進場，真突破後的漲勢往往是一段完整趨勢

停損：

  • 突破失敗要果斷停損
  • 假突破的特徵：量增卻漲不動、跌破後量能急縮

停利：

  • 觀察前低是否被跌破
  • 搭配基本面消息判斷故事是否結束

心得

突破交易這招其實很直覺：等在那個點位，突破了再追。但魔鬼在細節裡——假突破這個坑太常見了，一掉進去就是追高被套。

四原則裡面，最實用的是「收盤價為準」和「幅度要夠」。盤中追突破很容易被洗出去，因為盤中波動大、主力會做價。等收盤確認才進場，雖然可能少賺一點，但勝率會高很多。

幅度這點也很有意思——指數要 0.5%、股票要 3%。這差異反映了波動特性：股票波動本來就比指數大，所以標準要更嚴格。

另外一個關鍵點：突破交易要看的不只是技術面。文章提到 NVIDIA 和 TESLA 的案例，它們的大波段都是「基本面質變 + 技術面突破」的共振。純技術突破沒有故事支持，很容易變假突破。

最後是出場：連續跌破前低就該走。這個原則簡單但有效。很多人賺到錢捨不得走，結果抱著抱著就變虧損。設定好出場規則，該走就走，不要戀戰。
