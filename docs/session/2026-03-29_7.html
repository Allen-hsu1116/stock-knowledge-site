核心知識

一、主力是誰？六大主力分類

是什麼：主力是指擁有大量籌碼、足以影響短期股價的人或機構

六大主力類型：
1. 政府：國安基金（黑天鵝事件護盤）
2. 外資：影響大盤的主力
3. 投信：法人主力
4. 自營商：券商自營部門
5. 市場派：市場大戶
6. 公司派：公司內部人

實務分類：

  • 操控個股的主力：三大法人與市場大戶，多著墨於小股本股票，可透過券商分點買賣超追蹤
  • 影響大盤的主力：外資、券商、政府基金，特殊情況才會出現

二、主力洗盤五大手法

1. 短線拉回洗盤

是什麼：主升段常見的洗盤方式，追漲停買進後連續壓回4-5天

怎麼做：
1. 觀察第一防守線（長紅K線最低點）
2. 不跌破第一防守線 → 強勢整理，3-6天內拉長紅繼續上漲
3. 跌破第一防守線但守住第二防守線 → 較長時間震盪整理
4. MACD判斷：紅柱縮短再變長（強勢），或紅柱轉綠柱再轉紅柱（震盪）

實例：

  • 情境：佰研（3205）短線拉回守住第一防守線
  • 操作：股價拉回量縮、不跌破第一防守線時進場
  • 時間：3-6天內拉出長紅

適用時機：主升段追漲停後遭遇拉回時判斷是否續抱

2. K線洗盤（上影線洗盤）

是什麼：利用長上影線製造「避雷針」假象嚇跑散戶

怎麼做：
1. 觀察是否守住第一防守線
2. 若守住第一防守線 → 續抱
3. 後續股價突破「突破線」（上影線最高點）→ 新一波行情啟動

實例：

  • 情境：利勤（4426）出現長上影線配合綠色巨量
  • 判斷：看似避雷針，但股價未跌破第一防守線
  • 操作：等股價穿越突破線再確認

適用時機：主升段出現大量長上影線時判斷真偽

3. 起漲前的甩轎

是什麼：最兇猛的洗盤方式，盤整區突然長黑跌破再長紅吞噬

怎麼做：
1. 第一防守線被長黑跌破 → 先出場
2. 觀察後續是否止跌
3. 長紅吞噬長黑最高點 → 買回訊號

實例：

  • 情境：宏佳騰（1599）守住47.3第一防守線後突然長黑跌破
  • 操作：長黑跌破時先出場，不要賭它是甩轎
  • 買回條件：股價突破長黑最高點時買回

風險警示：不要去賭長黑是甩轎，萬一往下噴出會大賠

三、主力買進四種模式

1. 中線波段上漲：階梯式放量突破

是什麼：主力緩慢推高股價的建倉方式

特徵：

  • 與營收、財報、利多消息搭配
  • 技術面呈現箱型區間上漲
  • 每次區間整理平台不重疊
  • 箱底越墊越高、箱頂越墊越高

實例：

  • 情境：主力在換手過程中低買高賣降低持倉成本
  • 價格：箱型整理區間約10-15%波動
  • 時間：每次整理約1-3週

2. 長線波段上漲：量能萎縮、高檔橫盤

是什麼：高殖利率成長股常見的建倉模式

特徵：

  • 低檔爆大量緩慢推升
  • 高檔長期橫盤
  • 橫盤中量能急速萎縮（低於1000張）
  • 多間券商合作買超

實例：

  • 情境：籌碼面徹底洗盤
  • 時間：橫盤可能持續數月
  • 量能：萎縮至1000張以下

3. 底部回升：法人買盤急增、爆量過季線

是什麼：股價經過半年以上整理後的起漲模式

特徵：

  • 基本面充斥中立、利空消息
  • 股價創新低使散戶信心受打擊
  • 最後打壓洗盤創造最佳吃貨條件
  • 法人買超張數或持股比率上升
  • 技術面底部爆新高量起漲

買點判斷：
1. 「法人買超張數或持股比率」增加
2. 「技術面底部爆新高量起漲」

4. 法人認錯回補買盤

是什麼：吃「法人豆腐」的特殊買點

特徵：

  • 法人因績效壓力賣出持股
  • 股價洗盤時用時間掃走沒耐心的人
  • 特定主力券商大力吃貨形成「籌碼換手訊號」
  • 原本賣出的法人發現看錯方向後回補

實例：

  • 情境：突發利空事件、大盤劇烈波動
  • 操作：觀察主力券商逆勢承接
  • 預期：法人發現看錯後買盤回補力道又急又猛

四、出貨 vs 洗盤辨別（五個前兆）

怎麼分辨：洗盤與出貨的關鍵差異

出貨五前兆：

1. 成交量異常放大

  • 股價高位盤整時成交量突然放大但未創新高

2. 股價疲弱震盪頻繁

  • 高位震盪但反彈無法突破前高

3. 量價背離

  • 價格上漲但成交量萎縮

4. 大單頻繁賣出、小單接手

  • 主力拆單出貨

5. 利多消息頻繁但股價不漲

  • 財報亮眼、政策利多卻不推升股價

五、籌碼分析三大面向

1. 三大法人

怎麼做：

  • 觀察「上市外資買賣超金額」
  • 觀察「上市投信買賣超金額」
  • 觀察「上櫃外資買賣超金額」
  • 觀察「上櫃投信買賣超金額」
  • 確認股票是否在排行榜上
  • 確認外資、投信持股比率是否增加

實務：自營商通常略過不看（沒有明顯買賣趨勢）

2. 大戶持股

判斷標準：

  • 高價位股票：400張以上為大戶，50張以下為散戶
  • 中價位股票：1000張以上為大戶，100張以下為散戶

好現象：

  • 大戶持股比率上升
  • 散戶持股比率下降
  • 表示籌碼集中在大戶手上

3. 主力分點

是什麼：大量買賣超的券商分點

怎麼做：

  • 透過看盤軟體觀察特定券商買賣狀況
  • 可觀察1天、1週、30天、90天內的買賣張數
  • 追蹤哪個券商買最多、賣最多

六、實戰應用：跟隨主力的策略

如何判斷買點位置

成本參考：

  • 在主力成本尚未拉開20%處買進
  • 漲勢的80%~90%處獲利了結就是成功
  • 漲勢的50%處賣出也是成功

實例：

  • 情境：主力成本10元，目標20元
  • 買點：12元以下（成本未拉開20%）
  • 賣點：16-18元（漲勢的60%-90%）

出場時機

見好就收原則：

  • 沒有人能從最低點買進、最高點賣出
  • 避開高處不勝寒的環境
  • 把行情看完告一段落即可

心得

主力洗盤這檔事，教科書教的那套「五波上升、三波下降」現在已經不太適用了。台股早已是法人的市場，散戶如果只用技術指標分析很容易被當韭菜割。

最有用的概念是「防守線」：

  • 第一防守線守住 = 短線洗盤3-6天會結束
  • 第二防守線守住 = 較長時間震盪，看MACD綠柱轉紅柱

但要記住：「長黑跌破第一防守線時先出場，不要賭它是甩轎」。這句話很重要，萬一真的是往下噴出，會賠很慘。

另外「量價背離」和「利多不漲」這兩個出貨前兆我之前沒特別注意，以後看到股價在高檔震盪、利多消息頻傳卻不漲，就要警戒是不是主力在倒貨。

最後，籌碼分析要長時間觀察才有效果，不是看一天兩天就能下結論。重點是追蹤「法人買賣超」和「大戶持股比率」這兩個指標。
