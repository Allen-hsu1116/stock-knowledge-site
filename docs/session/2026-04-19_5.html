威科夫方法（Wyckoff Method）總覽

是什麼：威科夫方法是 Richard D. Wyckoff（1873-1934）在 1930 年代提出的技術分析框架，核心概念是追蹤「聰明錢」（Smart Money / Composite Man）的吸籌與派發過程。他是技術分析五大巨頭之一（與道氏、甘氏、艾略特、美林齊名），方法至今 90+ 年仍被機構交易員使用。

為什麼重要：大多數散戶賠錢是因為跟聰明錢對做——在吸籌末期恐慌賣出，在派發末期追高買入。威科夫方法教你辨識主力行為，讓你在正確的時間站在正確的一邊。

威科夫三大法則

法則一：供需法則（Law of Supply and Demand）

是什麼：價格上漲 = 需求大於供給；價格下跌 = 供給大於需求；價格橫盤 = 供需平衡。

怎麼做：
1. 觀察價格行為判斷供需力量消長
2. 上漲時量增 = 需求真實；上漲時量縮 = 需求虛弱
3. 下跌時量增 = 供給強大；下跌時量縮 = 供給枯竭

實例：

  • 情境：某股票從 100 元跌到 60 元後盤整
  • 判斷：如果下跌到 60 附近量能萎縮（供給枯竭），反彈量能放大（需求進場），符合供需轉換，可能是吸籌

適用時機：所有時間框架，是威科夫方法的基礎思維

法則二：因果法則（Law of Cause and Effect）

是什麼：趨勢不會憑空出現，必須先有「因」（盤整區間的吸籌/派發），才有「果」（後續的上漲/下跌）。而且「果」與「因」成正比——大因生大果，小因生小果。

怎麼做：
1. 用點數圖（Point & Figure Chart）計算目標價

  • 盤整區間的橫向柱數 = 「因」的大小

2. 三種目標計算法

  • 經典法：目標 = LPS/LPSY 價格 + 計算結果

3. 盤整時間越長，後續趨勢越大

  • 3 個月的吸籌區 → 可能產生 20-30% 的漲幅

實例：

  • 情境：台積電在 550-600 區間盤整 4 個月
  • 操作：用 P&F 圖計算橫向柱數，假設 20 柱 × 每格 5 元 = 目標 100 元距離
  • 從 LPS（約 570）起算 → 目標價 670 元

適用時機：盤整區間的目標價估算，幫助決定「值不值得等」

法則三：努力與結果法則（Law of Effort vs Result）

是什麼：成交量是「努力」，價格變動是「結果」。兩者應該同方向——大量應該推動大價格變動。如果出現背離，就是轉折預警。

怎麼做：
1. 收斂（Convergence）= 健康趨勢

  • 放量上漲 = 努力大、結果大 → 趨勢健康

2. 背離（Divergence）= 轉折預警

  • 放量卻漲不動 = 努力大、結果小 → 供給正在消化需求，可能派發

實例：

  • 情境：聯發科從 800 漲到 1200，但在 1150-1200 出現連續三天大量（成交量比平均多 50%）但只漲了 2%
  • 判斷：努力大、結果小 = 供給正在進場 → 派發訊號

適用時機：判斷趨勢是否延續或即將反轉，是威科夫最實用的法則

威科夫吸籌示意圖（Accumulation Schematic）五階段

Phase A：停止下跌

關鍵事件：

  • PS（Preliminary Support，初步支撐）：下跌趨勢中首次出現放量反彈，表示有買方開始進場，但大多數人認為趨勢會繼續
  • SC（Selling Climax，賣出高潮）：恐慌性拋售，巨量長黑K或長下影線。賣方用盡全力，但被買方全部吃掉
  • AR（Automatic Rally，自動反彈）：SC 後空手買方湧入 + 空方回補，自然反彈。建立盤整區間的上緣
  • ST（Secondary Test，二次測試）：回測 SC 低點，量能縮小，確認賣壓枯竭

實例：

  • 情境：鴻海從 180 跌到 120，某天出現歷史天量（SC），隔天反彈到 140（AR），一週後再回到 125 附近但量縮（ST）
  • 判斷：Phase A 完成，進入吸籌區間

Phase B：吸籌（主力默默吃貨）

核心：大戶在區間內反覆低買高賣小量加碼，每次 ST 都是大戶進場的機會。

特徵：

  • 多次 ST 測試底部，每次量能遞減
  • 價格在區間內上下震盪，高點和低點逐漸收斂
  • 大戶無法一次買夠，所以必須分批在類似價格重複進場
  • 可能持續數週到數月

實戰注意：Phase B 最漫長也最考驗耐心，不要急著進場，等 Phase C 的 Spring 訊號

Phase C：Spring（彈簧效應）— 最佳買點

是什麼：主力刻意打穿區間底部，製造假跌破，目的是：
1. 逼出最後一批意志不堅的持有者
2. 誘空方進場
3. 主力趁機大量吃貨

怎麼做：
1. Spring 三種型態

  • 1 號 Spring：跌破底部後立即反彈，量能極小 → 最強訊號

2. Spring 確認三條件

  • 跌破區間底部時量能極小（供給枯竭）

3. 進場時機

  • 保守型：Spring 後回測不破前低（LPS）再進場

實例：

  • 情境：某股在 50-60 區間盤整 2 個月，突然跌到 48（Spring），但量能是近 2 個月最小，隔天回到 52
  • 操作：Spring 確認！在 52-53 小量進場，停損設在 47（Spring 低點下方）
  • 目標：用 P&F 計算，預估目標 75-80

Phase D：需求接管

關鍵事件：

  • SOS（Sign of Strength，力量初現）：帶量突破區間上緣或阻力線，確認主力已完成吸籌
  • LPS（Last Point of Support，最後支撐點）：SOS 後的回測，量能萎縮，確認支撐有效。這是最後的上車機會
  • BU（Back-Up，回檔）：同 LPS，但更強調回測到前期阻力變支撐的位置

實例：

  • 情境：上述股票 Spring 後從 53 漲到 62（SOS），回測到 57 不破且量縮（LPS）
  • 操作：57 進場，停損 52，目標 75-80

Phase E：突破啟動

核心：價格正式離開盤整區間，進入 Mark-up（上漲）階段。開始形成更高的高點和更高的低點。

威科夫派發示意圖（Distribution Schematic）五階段

派發是吸籌的鏡像，結構相同但方向相反。

Phase A：停止上漲

  • PSY（Preliminary Supply，初步供給）：上漲趨勢中首次出現放量停滯
  • BC（Buying Climax，買入高潮）：恐慌性追高，巨量長紅K或長上影線。買方用盡全力
  • AR（Automatic Reaction，自動回落）：BC 後自然回落，建立區間下緣
  • ST（Secondary Test，二次測試）：回測 BC 高點，量能縮小

Phase B：派發（主力默默出貨）

  • 主力在區間內分批賣出
  • SOW（Sign of Weakness，弱勢初現）：跌破區間中線，供給開始壓過需求
  • 多次測試區間頂部和底部

Phase C：UTAD（派發後上沖）— 最強賣出訊號

是什麼：主力刻意拉穿區間頂部，製造假突破，目的是：
1. 吸引追多散戶進場
2. 誘空方停損
3. 主力趁機大量出貨

確認條件：

  • 突破區間頂部但無法維持（假突破）
  • 突破時量能可能大但隨即枯竭
  • 快速回落到區間內

實例：

  • 情境：某股在 100-120 盤整 3 個月，突然衝到 125，散戶瘋搶，但隔天直接回到 118
  • 判斷：UTAD！主力在 122-125 大量出貨，這是極度危險的訊號
  • 操作：立即減碼或全部出場，不要抱有任何幻想

Phase D：供給接管

  • LPSY（Last Point of Supply，最後供給點）：反彈越來越弱，高點越來越低
  • 確認派發完成

Phase E：下跌啟動

  • 進入 Mark-down（下跌）階段

威科夫五步交易法

步驟 1：判斷市場趨勢

  • 用大盤指數判斷整體方向
  • 決定是否應該進場、做多還是做空

步驟 2：選擇與趨勢一致的強勢股

  • 多頭選比大盤強的股票（漲幅大、跌幅小）
  • 空頭選比大盤弱的股票
  • 不確定就跳過，不要勉強

步驟 3：確認股票有足夠的「因」

  • 用 P&F 圖估算盤整區間的潛在目標
  • 只選目標滿足你最低獲利要求的股票
  • 大因 = 大果，盤整越久越值得關注

步驟 4：確認股票準備移動

  • 用九大買入/賣出測試（Wyckoff's Nine Tests）確認
  • 吸籌區：Spring + 量縮 + 回測不破 = 可以進場
  • 派發區：UTAD + 量增卻漲不動 = 該出場了

步驟 5：配合大盤轉折時機進場

  • 75% 的個股與大盤同步
  • 在大盤也出現轉折訊號時進場，勝率最高
  • 設好停損並追蹤移動

再吸籌 vs 再派發的區別

再吸籌（Re-accumulation）：

  • 出現在上升趨勢中段
  • 區間較小，時間較短
  • 仍然有更高的高點和更高的低點
  • 量能逐漸萎縮後再突破

再派發（Re-distribution）：

  • 出現在下跌趨勢中段
  • 反彈越來越弱
  • 量能在反彈時放大但在下跌時縮小
  • 看起來像底部但其實是下跌中繼

區別方法：看大週期趨勢方向 + 量價行為。如果大趨勢向上，區間盤整後大概率是再吸籌；反之亦然。

吸籌 vs 派發快速對照表

心得

威科夫方法最讓我震撼的是它的「Composite Man」概念——把市場所有參與者的行為當成一個有策略的操作者來分析。這不是陰謀論，而是分析框架。你不需要真的有一個主力在操控，你只需要理解供需力量的對比。

三個法則裡面，努力與結果法則是日常最實用的。每次看到「大量不漲」或「縮量大跌」，現在知道這不是隨機的——這是供需力量轉換的訊號。以前只會用 RSI 或 MACD 背離，現在多了量價背離這個更根本的判斷工具。

Spring 和 UTAD 是威科夫方法最精華的部分。Spring 告訴你「假跌破 = 最佳買點」，UTAD 告訴你「假突破 = 最後逃命點」。這完全翻轉了散戶的直覺——散戶看到跌破就恐慌賣出，看到突破就追高買入，而威科夫告訴你這正是主力設計的陷阱。

結合之前學的 ICT 模型（CHoCH + FVG + Killzone），威科夫提供了更高層次的結構框架。ICT 是微觀進場工具，威科夫是宏觀方向判斷。兩者結合：先用威科夫判斷目前處於吸籌/派發哪個階段，再用 ICT 找精確進場點。

目標價計算用 P&F 圖是威科夫的特色，但實務上斐波那契擴展和區間垂直投影更方便。我傾向用多種方法交叉驗證——如果 P&F、斐波那契 1.618 和區間垂直投影都指向類似目標，信心就很高。

營運效率指標概述

是什麼：營運效率指標衡量企業在日常經營循環中「賣貨→收錢→付錢」的速度。速度越快，代表公司資金運用效率越好、現金流越健康。這組指標是基本面分析中「經營能力」的核心，跟獲利能力（ROE/ROA）、償債能力（流動比率）並列三大財報分析支柱。

企業經營循環：
1. 向供應商進貨 → 存貨增加、應付帳款增加
2. 把商品賣給客戶 → 存貨減少、應收帳款增加
3. 客戶付款 → 應收帳款減少、現金增加
4. 付錢給供應商 → 應付帳款減少、現金減少

效率好的公司：賣貨快、收錢快、付錢慢 = 現金留身上最久
效率差的公司：賣貨慢、收錢慢、付錢快 = 現金流吃緊

一、存貨周轉率（Inventory Turnover）

是什麼：公司在一段時間內（通常一年），存貨被賣出再補貨的次數。次數越高，代表存貨銷售速度越快、資金積壓在存貨上的時間越短。

公式：

存貨周轉率（次/年） = 營業成本 ÷ 平均存貨
平均存貨 =（期初存貨 + 期末存貨）÷ 2
存貨周轉天數 = 365 ÷ 存貨周轉率

怎麼做：

1. 計算與判讀

  • 存貨周轉率 5 次 = 一年清空庫存 5 次

2. 存貨周轉率高低判斷

| 數值 | 意義 | 風險 |
|------|------|------|
| 偏高 | 銷售順利、經營效率好 | 太高可能備貨不足，接到大單沒貨賣，甚至供應鏈斷鏈 |
| 偏低 | 存貨積壓、銷售不順 | 倉儲成本增加、跌價損失、過期報廢風險 |
| 持續下降 | 銷售能力衰退或備貨策略改變 | 可能是看漲原物料而囤貨，也可能是賣不出去 |
| 持續上升 | 銷售能力提升 | 要注意是否備貨不足 |

3. 產業差異極大，只能同業比或自己比

  • 電子業/半導體：存貨跌價快，周轉率很重要。群聯（8299）因為 NAND Flash 報價循環，低價庫存就是黃金

4. 存貨是黃金還是垃圾？看報價趨勢

  • 報價上漲趨勢：低價庫存 = 黃金（群聯案例：NAND 報價逐季上逐季上漲，滿手低價庫存就是獲利保證）

5. 實戰判讀四步驟

  • 步驟一：看存貨周轉率趨勢（上升或下降？連續幾季？）

實例：

  • 情境：群聯（8299）2023 Q3 存貨周轉天數從 60 天拉長到 90 天
  • 表面看：效率變差？
  • 深入看：NAND Flash 報價正在從谷底反轉向上，公司趁低價大量備貨
  • 結果：2024 年 NAND 報價累計上漲 60%，低價庫存效益爆發，獲利大增
  • 教訓：存貨周轉率下降 + 報價上漲 = 備貨策略正確，不是效率變差

適用時機：

  • 製造業、電子業、半導體、零售業——存貨跌價風險大的行業
  • 觀察公司經營效率的長期趨勢
  • 不適合：營建業、金融業、服務業

二、應收帳款周轉率（Receivables Turnover Ratio）

是什麼：公司在一年內收回應收帳款的次數。次數越高，代表客戶付款速度越快、收帳效率越好、呆帳風險越低。

公式：

應收帳款周轉率（次/年）= 銷貨淨額 ÷ 平均應收帳款
平均應收帳款 =（期初應收帳款 + 期末應收帳款）÷ 2
應收帳款周轉天數（DSO）= 365 ÷ 應收帳款周轉率

怎麼做：

1. 計算與判讀

  • 應收帳款周轉率 12 次 = 一年收帳 12 次

2. 應收帳款周轉率高低判斷

| 數值 | 意義 | 風險 |
|------|------|------|
| 偏高 | 收帳效率好、客戶信用佳、公司對下游有議價力 | 可能信用條件太嚴，流失客戶 |
| 偏低 | 收帳效率差、客戶還款慢、呆帳風險高 | 現金流吃緊，可能造假 |
| 突然大幅上升 | 可能虛增營收（塞貨給客戶） | 應收帳款暴增但現金沒進來 |
| 持續拉長 | 下游客戶財務狀況惡化 | 最終可能變呆帳 |

3. 應收帳款是財報最容易造假的科目之一

  • 造假手法：虛增營收 → 應收帳款跟著增加 → 但現金沒進來

4. 對下游議價力的判讀

  • DSO 持續縮短 = 公司對客戶議價力增強，要求更快付款

5. 實戰判讀三步驟

  • 步驟一：看 DSO 趨趨勢（縮短還是拉長？持續幾季？）

實例：

  • 情境：某公司營收年增 30%，但應收帳款年增 60%，DSO 從 45 天拉長到 60 天
  • 表面看：營收成長很好
  • 深入看：應收帳款增速是營收的兩倍，錢沒收回來！
  • 進一步確認：營業現金流只增加 5% ≠ 營收增加 30%
  • 判判斷：紅旗！可能是在塞貨或給客戶過長的信用條件衝營收

適用時機：

  • 所有需要給客戶信用條件的B2B企業
  • 辨識財報造假的重要工具
  • 不適合：現金交易為主的零售業（應收帳款本來就很少）

三、應付帳款周轉率（Payables Turnover Ratio）

是什麼：公司在一年內支付供應商貨款的次數。跟存貨和應收帳款相反——應付帳款周轉率越低越好（代表付款天數越長，現金留在身上越久）。

公式：

應付帳款周轉率 = 進貨 ÷ 平均應付帳款
本期進貨 = 銷貨成本 - 期初存貨 + 期末存貨
應付帳款周轉天數 = 365 ÷ 應付帳款周轉率

怎麼做：

1. 應付帳款周轉天數越長 = 對供應商議價力越強

  • NIKE：付現天數逐期遞增 → 議價力越來越強

2. 三大周轉率的關聯

  • 存貨周轉天數：從進貨到賣出的天數

四、現金轉換循環（Cash Conversion Cycle, CCC）

是什麼：企業從「付錢買原料」到「收回客戶貨款」的完整天數。CCC 越短，現金使用效率越高，企業需要的外部融資越少。

公式：

CCC = 存貨周轉天數 + 應收帳款周轉天數 - 應付帳款周轉天數

怎麼做：

1. CCC 越短越好

  • CCC = 0：公司完全不需要自有資金就能運轉

2. 縮短 CCC 的三種方法

  • 方法一：降低存貨周轉天數（賣貨更快）

3. NIKE vs UA 的 CCC 比較

  • NIKE：CCC 從 102 天降到 87 天（穩定改善）

實例：

  • 情境：某公司存貨周轉天數 60 天 + 應收帳款周轉天數 45 天 - 應付帳款周轉天數 90 天
  • CCC = 60 + 45 - 90 = 15 天
  • 解讀：公司從付錢到收錢只要 15 天，現金周轉很快
  • 如果改成：應付帳款周轉天數只有 30 天 → CCC = 60 + 45 - 30 = 75 天
  • 解讀：現金周轉壓力大增，可能需要短期借款

適用時機：

  • 評估企業整體營運效率
  • 比較同業的現金周轉能力
  • 判斷企業是否需要大量外部融資
  • 最適合：零售業、製造業等需要管理存貨和帳期的行業

五、總資產周轉率與固定資產周轉率

總資產周轉率：

總資產周轉率 = 營業收入 ÷ 平均總資產
  • 衡量企業用全部資產創造營收的效率
  • 持續上升 = 經營活動熱絡；持續下滑 = 營收成長不如資產擴充，可能進入成熟期末端

固定資產周轉率：

固定資產周轉率 = 營業收入 ÷ 平均固定資產
  • 衡量不動產、廠房、設備的使用效率
  • 偏低 = 產能閒置；偏高 = 產能充分利用

適用時機：

  • 重資產行業（製造業、運輸業）特別重要
  • 品牌型/輕資產公司（如 NIKE）這個指標相對不重要

六、九大營運效率指標一覽

指標公式方向意義
存貨周轉率營業成本÷平均存貨越高越好存貨銷售速度
存貨周轉天數365÷存貨周轉率越短越好存貨賣出所需天數

應收帳款周轉天數(DSO)365÷應收帳款周轉率越短越好收回帳款天數

七、營運效率實戰篩選框架

怎麼做：

1. 第一步：CCC 排名篩選

  • 在同產業中找 CCC 最短的公司

2. 第二步：趨勢確認

  • 存貨周轉天數：連續 3 季縮短 = 銷售動能增強

3. 第三步：交叉驗證

  • 營運效率指標 + 營業現金流：效率好 + 現金流正 = 真的好

4. 第四步：紅旗偵測

  • 營收成長 > 20% 但 DSO 拉長 > 15% → 可能塞貨

心得

營運效率指標是基本面分析裡面最被散戶低估的一組數據。大部分人看財報只看 EPS 和營收，但 EPS 可以被應收帳款灌水、營收可以被塞貨膨脹——營運效率指標是拆穿這些把戲的照妖照妖鏡。

幾個關鍵收穫：

1. 存貨是黃金還是垃圾，看報價趨勢：群聯的案例太經典了。存貨周轉天數從 60 天拉到 90 天，表面看效率變差，但 NAND Flash 報價正在從谷底翻升，低價庫存就是利潤保證。如果只看數字不看產業背景，會誤判。這跟之前學的「營收成長率三大陷阱」是同一個道理——數字的表面意義和背後真相可能完全相反。

2. 應收帳款是財報造假的溫床：市場先生特別強調這點，應收帳款和存貨是資產負債表上最容易造假的科目。辨識方法是「應收帳款增速 vs 營收增速」加上「營業現金流驗證」。這跟之前學的現金流量表判讀可以無縫銜接——營業現金流才是最真實的獲利品質指標。

3. CCC 是企業的「現金體檢報告」：CCC 把存貨、應收、應付三個指標整合成一個數字，非常直覺。CCC < 0 的公司等於在用供應商的錢周轉（亞馬遜就是這種模式），現金流壓力極小。以後選股可以把 CCC 排名當第一道濾網。

4. 跟之前學的 ROE 杜邦分析連結：杜邦分析把 ROE 拆成淨利率 × 資產周轉率 × 權益乘數。今天學的總資產周轉率和固定資產周轉率就是杜邦分析裡的「資產周轉率」這一環。高周轉率型公司（如零售業）靠的是薄利多銷，高淨利率型公司（如台積電）靠的是高毛利，兩種路徑都能創造高 ROE，但風險特徵完全不同。

5. 實戰應用：以後讀財報不只看 EPS，一定要同時看存貨周轉天數和 DSO 的趨勢。EPS 漂亮但 DSO 拉長的公司，我會特別警覺。這比任何技術指標都能更早發現公司基本面的惡化。

乖離率（BIAS）與 MACD 進階實戰

之前學了 RSI、KD、布林通道、OBV、DMI、一目均衡表、型態學、威科夫等指標，但有兩個最基礎也最常用的指標一直沒有深入探討：乖離率（BIAS） 和 MACD 的進階應用。今天把這兩個補起來，它們是日常看盤最常用的工具之一。

一、乖離率（BIAS）

是什麼

乖離率衡量的是「股價偏離均線的程度」，用百分比表示。核心邏輯來自「遛狗理論」：不管狗（股價）跑多遠，最終都會回到主人（均線）身邊。乖離率越大，股價回歸均線的機率越高。

公式：

乖離率 = (當前價格 − N日移動平均) ÷ N日移動平均 × 100%

怎麼做

1. 正乖離與負乖離判讀

情境乖離率意義操作參考
股價在均線上方正乖離市場超買，回檔壓力增大避免追高，考慮減碼
股價在均線下方負乖離市場超賣，反彈機會增加避免殺低，考慮買進
乖離率接近 0股價在均線附近多空均衡等待方向明朗

實例：

  • 麗臺（2465）2024年初從20日均線附近起漲，月線乖離率從 8.3% 一路飆到 62.94%
  • 解讀：正乖離持續擴大 = 漲勢極度超買，回檔修正的機率急遽升高
  • 實際結果：股價隨後出現明顯回檔

2. 乖離率多少算大？跟自己比！

核心原則：乖離率沒有絕對標準，要跟自己的歷史數據比較。

每檔股票的股性不同，波動程度不同：

  • 台積電（2330）：20日乖離率歷史範圍約 ±8%~10%
  • 小型股：20日乖離率可能到 ±20% 以上
  • 鴻海（2317）：近兩年 20日乖離率落在 8.16% 到 -10.07% 之間

OANDA 提供的參考基準（台股一般適用）：

期間負乖離（買進參考）正乖離（賣出參考）
6日-3%+3.5%
12日-4.5%+5%
24日-7%+8%
72日-11%+11%

重點：以上只是粗略參考，實際操作一定要回看個股的歷史乖離率範圍！

3. 乖離率實戰四步驟

步驟一：選定均線週期

  • 短線操作：6日或12日乖離率
  • 波段操作：24日（月線）乖離率
  • 中長線：72日（季線）乖離率

步驟二：回看個股歷史乖離率範圍

  • 找出過去1-2年該股正乖離率和負乖離率的極值
  • 以正乖離率前 10% 作為超買警示區
  • 以負乖離率前 10% 作為超賣警示區

步驟三：結合均線趨勢方向

  • 均線向上 + 正乖離擴大 → 強勢續漲，暫不急賣
  • 均線向上 + 正乖離極大 → 回檔機率高，考慮減碼
  • 均線向下 + 負乖離擴大 → 弱勢續跌，暫不急買
  • 均線向下 + 負乖離極大 → 反彈機率高，考慮搶短

步驟四：搭配其他指標確認

  • 乖離率 + KD：負乖離極大 + KD 黃金交叉 → 強力買進訊號
  • 乖離率 + RSI：正乖離極大 + RSI 超買（>70）→ 強力賣出訊號
  • 乖離率 + 布林通道：正乖離擴大 + 觸碰布林上軌 → 超買確認
  • 乖離率 + 成交量：正乖離極大 + 暴量長紅 → 可能最後一擊

實例：

  • 情境：技嘉（2376）股價在20日均線上方，20日乖離率達到 +15%（歷史上限約 +12%）
  • 判斷：正乖離超過歷史範圍，回檔機率極高
  • 操作：考慮減碼或設緊停利，不要追高
  • 如果同時 KD 在 80 以上死亡交叉 → 更強烈賣出訊號

4. 乖離率的三大使用限制

限制一：不適用波動幅度小的個股

  • 緩漲緩跌或長期盤整的股票，乖離率變動不明顯
  • 這類股票的乖離率永遠在 ±3% 以內，沒有參考價值

限制二：不適用市值太小的個股

  • 小型股容易被主力操縱，乖離率可能人為拉大或壓低
  • 市值較大的權值股更符合市場一般狀況，乖離率更準確

限制三：乖離率是滯後指標

  • 單獨使用容易誤判：強勢股可以長期維持高正乖離
  • 必須搭配其他指標綜合判斷

5. 台股年線負乖離的特殊規律

歷史經驗：台股年線（240日）負乖離若超過 20%，後一個月上漲機率接近 100%。

這是因為年線代表過去一年的平均成本，當股價跌到年線以下超過 20%，代表極度超賣，幾乎所有持有者都虧損，反彈力量非常強。

適用時機：

  • 大盤或大型股的長線底部判斷
  • 搭配市場寬度指標（ADL、MCL）確認
  • 不適合：短線交易（年線乖離率變動太慢）

二、MACD 進階實戰

基礎回顧

MACD（Moving Average Convergence Divergence，平滑異同移動平均線）由 Gerald Appel 在 1970 年代提出，是最廣泛使用的趨勢指標之一。

三大組成：

  • DIF 快線 = 12日EMA − 26日EMA（短期動能）
  • DEM 慢線（訊號線） = DIF 的 9日EMA（中期趨勢確認）
  • 柱狀圖（Histogram） = DIF − DEM（動能強度）

MACD 進階應用一：零軸上下方的交叉意義

很多人只知道「黃金交叉買進、死亡交叉賣出」，但交叉發生在零軸上方還是下方，意義完全不同：

交叉位置黃金交叉死亡交叉
**零軸上方**多頭市場中動能加強，上漲行情延續或加速多頭趨勢中的回撤或轉變，不宜過度樂觀
**零軸下方**空頭趨勢中的反彈，或趨勢逐漸轉向多頭空頭市場中動能加強，下跌延續或加速

實戰重點：

  • 零軸上方的黃金交叉勝率遠高於零軸下方
  • 零軸下方的死亡交叉代表空方加速，不要輕易抄底
  • 零軸下方的黃金交叉可能只是反彈，不要當成反轉

實例：

  • 情境：台積電 MACD 在零軸上方發生黃金交叉
  • 判斷：多頭趨勢延續，動能加速，加碼或續持
  • 如果是零軸下方的黃金交叉 → 可能只是跌深反彈，不追多

MACD 進階應用二：參數調整——沒有萬用聖杯

預設參數 (12, 26, 9) 的由來：1970年代一週交易6天，12天≈半個月，26天≈一個月。這組參數設計初衷是日線級別的中長期趨勢。

不同交易風格的參數選擇：

參數組合適用風格特性風險
(12, 26, 9)中期波段、日線趨勢平衡性佳，適用性廣對短線反應慢
(5, 35, 5)短線交易、當沖反應極快，捕捉早期趨勢假訊號多，盤整市頻繁停損
(6, 13, 5)極短線、高波動市場更快反應假訊號更多
(24, 52, 9)長線投資、週線圖過濾雜訊，穩定性高訊號嚴重滯後
(50, 100, 20)年度趨勢、月線圖判斷牛熊轉換極度遲鈍

參數調整原則：

  • 縮短週期 → 提高靈敏度，但假訊號增加
  • 加長週期 → 增強穩定性，但滯後性增加
  • 沒有萬用聖杯，必須根據交易標的和週期回測驗證
  • 切換參數時保持回測習慣，確認在歷史數據中有效再應用到實盤
  • 避免過度優化（Curve-Fitting）：一組參數只在某段行情完美 ≠ 好參數

MACD 進階應用三：提升黃金/死亡交叉勝率

單獨使用 MACD 交叉訊號勝率有限，以下是提升勝率的三道過濾：

過濾一：加入長期趨勢線（EMA 99）

  • 黃金交叉發生在 EMA 99 之上 → 多頭市場中的動能加強，勝率更高
  • 黃金交叉發生在 EMA 99 之下 → 可能只是空頭反彈，勝率低
  • 死亡交叉發生在 EMA 99 之下 → 空頭加速，不要抄底

過濾二：搭配價格突破關鍵位

  • 黃金交叉 + 價格突破壓力線 → 強烈多頭共識
  • 黃金交叉 + 價格仍在區間盤整 → 可能是假訊號

過濾三：搭配成交量確認

  • 黃金交叉 + 放量突破 → 可靠的買進訊號
  • 黃金交叉 + 縮量 → 動能不足，謹慎看待
  • 死亡交叉 + 放量下跌 → 可靠的賣出訊號

MACD 進階應用四：背離訊號的參數選擇

看漲背離（底背離）：

  • 股價創新低（Lower Low），MACD 柱狀圖或 DIF 卻創更高的低點（Higher Low）
  • 空方動能衰竭，潛在底部反轉訊號

看跌背離（頂背離）：

  • 股價創新高（Higher High），MACD 卻創更低的高點（Lower High）
  • 多方動能衰退，潛在頭部反轉訊號

背離的參數選擇技巧：

  • 找大級別背離：使用標準 (12, 26, 9) 或長線參數 (24, 52, 9)
  • 短線參數產生的背離太頻繁，假訊號多
  • 長線參數過濾短期雜訊，讓真正的大級別動能轉折更清晰

實例：

  • 情境：聯發科股價從 800 跌到 650，MACD(12,26,9) 柱狀圖的底部卻從 -15 上升到 -8
  • 判斷：看漲背離！股價雖然還在跌，但空方動能在衰竭
  • 操作：等待股價止穩 + MACD 黃金交叉確認，再進場做多
  • 如果使用 (5,35,5) 參數 → 背離訊號更早但更不可靠，可能出現多次假背離

MACD 進階應用五：三大使用陷阱

陷阱一：迷信萬能參數

  • (12, 26, 9) 不是聖杯，只是適合日線波段交易的預設值
  • 不同商品、不同週期需要不同參數
  • 台積電日線的參數 ≠ 比特幣15分鐘線的參數

陷阱二：在盤整行情中濫用交叉訊號

  • MACD 是趨勢指標，天敵是盤整
  • 盤整時快慢線頻繁交叉 = 大量假訊號（Whipsaw）
  • 解決方案：搭配 ADX 指標，ADX < 20-25 時代表盤整，暫停使用 MACD 交叉訊號
  • 跟之前學的 DMI 完全對接：ADX 趨勢強度判斷 + MACD 趨勢方向判斷

陷阱三：過度優化歷史數據

  • 回測找到一組「完美參數」≠ 未來也有效
  • 好的參數應該在多段不同行情中都有不錯表現
  • 穩健性 > 完美性

三、乖離率 × MACD 雙指標共振策略

把今天學的兩個指標組合起來，可以形成更強的共振訊號：

買進共振訊號

1. 負乖離極大 + MACD 底背離 + 黃金交叉

  • 24日乖離率來到歷史低點（例如 -10% 以下）

2. 負乖離回歸 + MACD 零軸上方黃金交叉

  • 負乖離率從極端值回到接近 0

賣出共振訊號

1. 正乖離極大 + MACD 頂背離 + 死亡交叉

  • 24日乖離率來到歷史高點（例如 +15% 以上）

2. 正乖離擴大 + MACD 柱狀圖由正轉負

  • 正乖離率持續擴大

盤整過濾

  • 當 ADX < 20 且乖離率在 ±3% 以內波動 → 盤整行情
  • 此時 MACD 交叉訊號失效，乖離率也沒有參考價值
  • 策略：暫停使用這兩個指標，或改用區間交易策略

心得

乖離率和 MACD 是技術分析入門最先接觸的兩個指標，但也最容易因為「太簡單」而被忽視其深度。今天深入學了之後，幾個關鍵收穫：

1. 乖離率的「跟自己比」原則：這是我之前最常犯的錯——看到乖離率 +10% 就覺得超買，但每檔股票的歷史範圍不同。鴻海的 ±10% 可能是極端值，但小型股 ±20% 都還算正常。以後一定要先查個股的歷史乖離率範圍，再判斷是否超買超賣。

2. MACD 交叉的零軸位置決定一切：同樣是黃金交叉，零軸上方和零軸下方的意義天差地遠。零軸上方的黃金交叉是多頭加速，零軸下方的黃金交叉可能只是跌深反彈。這個觀念之前學 RSI 50 分水嶺的時候也有類似的概念，但 MACD 的零軸意義更明確——它代表短期動能是否大於長期動能。

3. 參數不是聖杯，回測才是王道：(12, 26, 9) 是 1970 年代的產物，但現在的市場結構已經不同。(5, 35, 5) 在短線更靈敏，(24, 52, 9) 在長線更穩定。重點是要根據自己的交易風格和標的去調整，然後回測驗證，不要迷信任何一組「最佳參數」。

4. 乖離率 + MACD 的共振策略很實用：單獨用乖離率只能知道「偏離程度」，單獨用 MACD 只能知道「動能方向」。兩者結合就變成「超買/超賣 + 動能確認」，勝率大幅提升。再加上之前學的 ADX 過濾盤整，三個指標形成完整的多空判斷系統。

5. 年線負乖離超過 20% 的歷史規律：這是台股特有的經驗法則，而且後一個月上漲機率接近 100%。這個訊號大概幾年才出現一次（通常在股災底部），但出現時就是極佳的長線進場時機。以後要在大盤大跌時留意這個訊號。
