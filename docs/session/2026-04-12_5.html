主力鎖碼股判斷技巧

第 5 回合學習 - 主力鎖碼股判斷技巧

核心知識

主力是誰？

定義：具備龐大資金實力，能夠影響股價波動的一群人，包含三大法人、公司派、市場大戶。

為什麼要觀察主力：

  • 股票在大戶手上 → 利空不跌
  • 股票在散戶手上 → 利多不漲
  • 跟著主力走 → 搭上順風車

觀察主力的三大指標

1. 主力買賣超

是什麼：當天主力的淨買入或淨賣出金額。

怎麼看：

  • 正數（連續數日）→ 主力持續布局，股價上漲機率高
  • 負數（連續數日）→ 主力在出貨，股價下跌風險高

實例：

  • 鴻海（2317）2024年3月連續大買14天
  • 股價從100.5元漲到175元，漲幅74.1%

2. 買賣家數差

是什麼：買方券商家數減去賣方券商家數。

怎麼看：

  • 家數差 > 0 → 買方多於賣方，籌碼分散，不易上漲
  • 家數差 < 0 → 買方少於賣方，籌碼集中，容易上漲

邏輯：

  • 少數買家（大戶）買走多數賣家的籌碼
  • 籌碼集中在大戶手上，利於拉抬

3. 籌碼集中度

是什麼：（買超前15名券商張數 - 賣超前15名券商張數）÷ 成交量

怎麼看：

  • 短天期集中度 > 長天期集中度 → 近期主力正在買入
  • 集中度上升 → 籌碼趨向集中，利於上漲

主力鎖碼的黃金組合

條件組合：
1. 主力買賷超：連續數日正數
2. 買賣家數差：連續數日負數
3. 籌碼集中度：上升

意義：

  • 股票正在往少數人手上集中
  • 主力看好持續買入
  • 這是良好的進場時機

注意：

  • 漲幅已大不追高
  • 籌碼開始分散時要警覺

融資鎖碼手法

主力為什麼用融資鎖碼？

原因：

  • 融資成本一年約 6%，但潛在報酬可能遠大於成本
  • 資金效率運用：4成自備款買到完整股票
  • 鎖住融資額度後，散戶無法用融資買進
  • 籌碼相對穩定，利於股價上漲

注意：可融資張數不可超過發行數量的25%

判斷融資鎖碼股的三特徵

特徵一：融資使用率 > 20%

怎麼判斷：

  • 市場約七成個股融資使用率低於20%
  • 中小型股起漲前融資使用率達2成以上 → 有大戶鎖碼跡象
  • 高度鎖碼股在底部融資使用率甚至超過50%

實例：

  • 揚明光（3504）2017年底部融資使用率超過50%
  • 後續股價翻倍

特徵二：融資暴增

怎麼判斷：

  • 融資在短時間內突然暴增 → 比較可能是主力行為
  • 散戶不太可能一次重壓這麼多
  • 散戶常逆勢操作（股價下跌慢慢攤平）
  • 主力不太會這樣做

特徵三：籌碼趨向集中

怎麼判斷：

  • 符合前兩點後，還要搭配籌碼越來越集中
  • 才能確認融資由主力布局，而非散戶介入

區分主力鎖碼 vs 散戶攤平：

  • 主力鎖碼：融資暴增 + 籌碼集中 → 未來翻倍行情
  • 散戶攤平：融資增加 + 籌碼分散 → 後勢疲弱

四步驟找出主力融資鎖碼股

步驟一：買賣家數差七成為負

設定：

  • 預設：最近60天買賣家數差七成為負
  • 可調整：近10天買賣家數差6成為負

意思：買家數少於賣家數，籌碼集中在大戶

步驟二：融資近20日暴增10%

設定：

  • 預設：近20日融資暴增10%
  • 可調整：近10日融資暴增2%

意思：出現主力用融資鎖碼的徵兆

步驟三：融資使用率 > 20%

設定：

  • 預設：融資使用率 > 60%
  • 可調整：調降至20%讓濾網寬鬆

意思：避免剔除太多股票

步驟四：股價在融資成本線之上

意思：趨勢偏多

關鍵：行情偏多時，融資鎖碼股漲勢更剽悍

風險提醒

融資鎖碼股的特性：

  • 籌碼被主力鎖定 → 股價上去很快
  • 主力開始出貨 → 股價下去也很快

操作要點：

  • 密切留意主力動向
  • 主力開始賣超時要跟著出場
  • 不要戀戰

心得

主力鎖碼股的判斷其實就是跟著「有錢人」走。他們有資訊優勢、資金優勢，我們散戶能做的就是觀察他們的腳印。

最有價值的是「買賣家數差」這個指標。以前只知道看主力買賣超，現在知道要搭配家數差。買家少賣家多 = 大戶集中買入 = 籌碼集中 = 容易上漲。這個邏輯很清楚。

融資鎖碼的概念也很有趣。一般認為融資是散戶指標，融資多=籌碼渙散。但主力反而用融資鎖碼，把融資額度用完後散戶就買不到了。這就是為什麼要同時看「融資使用率」和「籌碼集中度」——單看融資會誤判。

四步驟選股法很實用：買賣家數差負數 + 融資暴增 + 融資使用率20%以上 + 股價在融資成本線上。下次試著用籌碼K線的選股功能跑一次。

風險要記住：主力鎖碼股漲得快，跌得也快。主力一跑要跟著跑，不要幻想「長線投資」。
