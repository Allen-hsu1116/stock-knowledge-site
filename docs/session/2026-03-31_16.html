Altman Z-Score 財務預警模型

第 22 回合學習 - Altman Z-Score 財務預警模型

Altman Z-Score 是什麼

是什麼：一個用來評估公司財務健康狀況、預測破產風險的綜合指標。由紐約大學教授 Edward Altman 在 1968 年發表，透過 5 個財務比率加權計算出一個分數。

為什麼重要：

  • 2008 年金融海嘯前，Altman 計算出整體股市 Z-Score 中位數只有 1.81，準確預測了危機
  • 研究顯示準確度達 82%-94%，能幫助投資人避開地雷股

Z-Score 計算公式

公式：Z-Score = 1.2A + 1.4B + 3.3C + 0.6D + 1.0E

五個變數：

變數計算方式意義
A營運資金／總資產衡量流動性，反映短期償債能力
B留存收益／總資產衡量累積利潤，反映獲利持久性
C息稅前利潤（EBIT）／總資產衡量資產盈利能力，最重要權重（3.3）
D公司市值／總負債衡量財務結構穩定性
E營業收入／總資產衡量資產週轉效率

實例計算（假設某公司）：

  • A = (流動資產100億 - 流動負債60億)／總資產200億 = 0.2
  • B = 留存收益30億／總資產200億 = 0.15
  • C = EBIT 20億／總資產200億 = 0.1
  • D = 市值500億／總負債80億 = 6.25
  • E = 營收150億／總資產200億 = 0.75

Z-Score = 1.2×0.2 + 1.4×0.15 + 3.3×0.1 + 0.6×6.25 + 1.0×0.75 = 5.615（安全區）

三大判斷區間

🟢 安全區（Safe Zone）：Z-Score > 2.99

  • 公司財務狀況健康
  • 破產風險極低

🟡 灰色區（Grey Zone）：1.81 ≦ Z-Score ≦ 2.99

  • 公司可能在未來兩年內出現財務問題
  • 需要進一步研究

🔴 遇險區（Distress Zone）：Z-Score < 1.81

  • 公司財務狀況岌岌可危
  • 破產風險很高，應避開

實戰案例

蘋果 AAPL：

  • gurufocus 顯示 Z-Score = 5.3（安全區）
  • 過去 13 年最低曾跌到 1.96，最高 10.64
  • 結論：財務狀況非常健康

Tesla TSLA：

  • 目前 Z-Score 落在安全區
  • 過去曾跌到負數（早期財務危機）
  • 結論：已從高風險轉為安全

AT&T：

  • 電信龍頭但 Z-Score 落在遇險區
  • 顛覆「大公司 = 安全」的直覺
  • 結論：大型股不等於財務健康

  • 數值下降代表財務惡化
  • 同產業互相比較更有意義

適用時機

什麼情況用：

  • 篩選股票時，快速排除地雷股
  • 股災期間，檢視持股是否安全
  • 長期投資前，評估公司財務體質
  • 金融海嘯來臨前，作為風險警示

什麼情況不要用：

  • 新創公司、尚未獲利的成長股（Z-Score 偏低但可能是正常現象）
  • 金融業（公式不適用於金融機構）
  • 單獨使用（應搭配其他指標綜合判斷）

Z-Score 的限制

不適用的公司類型：

  • 金融機構（銀行、保險公司）
  • 新創公司、尚未獲利的成長股
  • 能源、航運等高波動產業（Z-Score 通常偏低）

注意事項：

  • 原版公式針對製造業設計
  • 2012 年有更新版 Z-Score Plus，適用範圍更廣
  • 應搭配產業特性調整解讀

心得

這個指標最厲害的地方在於：它在 2008 年金融海嘯前就發出警告了。當時股市在創新高，但 Altman 算出整體市場 Z-Score 中位數只有 1.81，代表一半的公司財務不健康。結果隔年真的崩盤。

實戰上有幾個關鍵點：
1. C（EBIT／總資產）權重最高：獲利能力是最重要的財務指標
2. D（市值／負債）：這項說明為什麼高股價的公司看起來更安全——市值墊高了分數
3. 趨勢比數值重要：要看 Z-Score 是上升還是下降

AT&T 的案例很有教育意義。直覺會覺得電信龍頭很穩，結果 Z-Score 在遇險區。這說明「大公司」不等於「財務健康」。很多時候大公司背負龐大負債，反而是隱形危機。

應用策略：

  • 選股時先用 Z-Score 過濾，避開 < 1.81 的公司
  • 長期投資標的，Z-Score 應該 > 2.99
  • 持股定期檢查 Z-Score 趨勢，下降時要警惕

這個指標不是萬能的，但作為第一道防線很有效。用 Charlie Munger 的話說：「成功來自於知道該避開什麼。」

核心知識

營收成長率（Revenue Growth Rate）

是什麼：比較現在和特定時期營業收入的成長百分比，判斷企業是否在擴張。

三種類型：
1. YoY（年增率）：與去年同月/同季比較 — 最常用，可避開淡旺季影響
2. QoQ（季增率）：與上一季比較 — 受季節性影響大
3. MoM（月增率）：與上個月比較 — 波動最大

怎麼計算：

營收年增率 YoY = (本月營收 / 去年同月營收 - 1) × 100%

實例：

  • 情境：台積電2021年1月營收1267.49億，2020年1月營收1036.83億
  • 計算：(1267.49 / 1036.83 - 1) × 100% = 22.2%
  • 判斷：年增22.2%，成長動能強

三種 YoY 細分：
1. 短期營收年增率：近3個月營收總和 vs 去年同期
2. 長期營收年增率：近12個月營收總和 vs 去年同期
3. 累積營收年增率：今年累積營收 vs 去年同期累積

動能判斷：

  • 短期 > 長期 → 業績動能轉強
  • 短期 < 長期 → 業績動能轉弱

適用時機：

  • 台股每月10號前公布上月營收，比財報更早取得
  • 用來預估下一季財報表現
  • 營收成長穩定 → EPS 成長機率高

EPS成長率

是什麼：每股盈餘（EPS）的年增率，代表公司為股東賺錢能力的成長幅度。

怎麼計算：

EPS年增率 = (本期EPS / 去年同期EPS - 1) × 100%

EPS公式：

EPS = 稅後淨利 / 在外流通普通股股數

實例：

  • 情境：三竹(8284) 2022年Q3 EPS年增率69.7%
  • 判斷：單季EPS年增率在近4季EPS年增率上方 → EPS成長動能向上

EPS選股判斷標準

好公司的EPS特徵：
1. EPS長期穩定：營運穩定，財務可預測
2. EPS持續向上成長：有成長動能

EPS與股價關係：

  • EPS持續成長 → 股價通常長期上漲
  • EPS持續下降 → 股價通常長期下跌

EPS成長要觀察的原因：

  • 營收成長？（最健康）
  • 新產品/大訂單？（一次性或長期？）
  • 營運效率改善？（毛利率提升或費用降低）
  • 股票回購？（減少股本推升EPS）
  • 業外收入？（不可持續）

EPS選股四大陷阱

陷阱1：EPS成長來自股票回購

  • 徵兆：EPS成長但營收沒成長
  • 做法：比較EPS成長率 vs 營收成長率，兩者應該同步

陷阱2：獲利與現金流落差

  • 徵兆：EPS高但營業現金流低
  • 做法：檢查「營業現金流 ÷ 稅後淨利」比率，應長期 > 100%

陷阱3：稀釋EPS差異大

  • 徵兆：稀釋EPS比一般EPS低很多
  • 做法：比較兩者差異，差異大代表未來可能股本膨脹

陷阱4：忽高忽低的EPS

  • 徵兆：EPS波動劇烈
  • 做法：跳過這類公司，除非對該產業有深入研究

EPS搭配四大財務指標

1. EPS + 本益比（P/E）

  • 用途：評估股價是否昂貴
  • 注意：成長股本益比常高（30~50倍），但不要「花太多錢買成長」

2. EPS成長 + 營收成長

  • 用途：確認成長來自本業而非回購或業外
  • 做法：兩者應該同步成長

3. EPS + 營業現金流

  • 用途：評估EPS品質
  • 判斷：營業現金流 ÷ 稅後淨利 > 100% → EPS品質佳

4. EPS + ROE

  • 用途：評估保留盈餘的運用效率
  • 判斷：EPS成長同時，ROE應持平或上升

實戰選股流程

Step 1：篩選營收成長

  • 用年增率YoY，避開季節性干擾
  • 營收連續3個月以上成長 → 關注

Step 2：確認EPS同步成長

  • EPS成長率與營收成長率是否同步？
  • 不同步要查原因（回購？業外？）

Step 3：檢查EPS品質

  • 營業現金流 / 稅後淨利 > 100%
  • 稀釋EPS與一般EPS差異 < 10%

Step 4：評估ROE趨勢

  • EPS成長同時，ROE是否持平或上升？
  • ROE下降 → 保留盈餘運用效率變差

Step 5：判斷本益比合理性

  • 成長股本益比可以高，但不能「太貴」
  • 用預估未來EPS計算本益比較準確

心得：

營收成長率和EPS成長率是最基本也最重要的成長判斷工具。但這篇學到最關鍵的概念是：EPS成長不等於公司變好。

成長可能來自：

  • 真正的營運成長（最健康）
  • 股票回購（只是把配息換成回購）
  • 業外收入（不可持續）
  • 營運效率改善（較少見）

所以單看EPS成長是不夠的，必須搭配：

  • 營收成長確認是本業成長
  • 營業現金流確認獲利品質
  • ROE確認資金運用效率

另外一個實用點是短期vs長期營收年增率的動能判斷：短期 > 長期表示動能轉強。這個可以用來判斷股價轉折點。
