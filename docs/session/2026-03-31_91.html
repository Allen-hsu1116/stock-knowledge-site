核心知識

營業現金流 vs 稅後淨利

是什麼：

  • 稅後淨利是損益表上的數字，採用「應計基礎」，代表公司「應該」賺到的錢
  • 營業現金流是現金流量表上的數字，採用「現金基礎」，代表公司「實際」收進戶頭的錢
  • 兩者差異就在於「帳面上有沒有真正收到錢」

為什麼會有差異：
1. 應收帳款：已認列營收但還沒收到錢
2. 存貨變動：買了貨還沒賣掉
3. 應付帳款：已認列費用但還沒付錢
4. 折舊攤銷：損益表扣了，但實際沒有現金流出

實例：

  • 情境：A公司出貨10億給客戶，客戶要求三個月後付款
  • 損益表：認列10億營收，淨利增加
  • 現金流量表：0元（因為還沒收到錢）
  • 結論：公司帳面賺錢，但現金流入是零

營業現金流對淨利比（獲利含金量）

是什麼：

  • 公式：營業現金流 ÷ 稅後淨利 × 100%
  • 意義：公司每賺1元淨利，實際收到多少現金

怎麼判斷：

數值意義解讀
>100%現金比淨利多優質公司，折舊攤銷加回或應付增加
80%~100%正常範圍獲利含金量高，盈餘與現金一致
60%~80%稍有落差需關注應收帳款或存貨是否增加
<60%警訊獲利可能有水分，需深入檢查原因

為什麼會超過100%：

  • 重資產公司：折舊攤攤銷多（損益表扣掉，但沒有現金支出）
  • 預收款多：先收錢再提供服務
  • 應付帳款增加：延後付款，現金留在手上

為什麼會低於60%：

  • 應收帳款暴增：貨賣了錢沒收到
  • 存貨積壓：貨買了賣不出去
  • 可能是財報美化、虛增營收

實戰案例

台積電（2330）：

  • 營業現金流對淨利比：長年130%以上
  • 原因：嚴格控管應收帳款和存貨
  • 解讀：獲利含金量極高，是「名符其實」的現金獲利

研華（2395）：

  • 營業現金流對淨利比：長年70%以上
  • 特性：工業電腦電腦龍頭，帳面獲利與現金流一致

使用方法

方法1：檢查是否為正數

  • 營業現金流正值：公司營運有持續流入現金
  • 營業現金流負值：公司現金持續流出，小心資金斷鏈

方法2：看比值是否達80%

  • 計算：營業現金流 ÷ 稅後淨利
  • 可用季報、年報或近4季累積數字
  • 目標：80%以上代表獲利紮實

方法3：觀察趨勢

  • 比值逐年上升：公司管理能力提升
  • 比值逐年下降：可能應收帳款或存貨問題
  • 突然大跌：可能有一次性事件或財報問題

搭配其他指標

每股營業現金流：

  • 公式：營業現金流 ÷ 在外流通股數
  • 用途：衡量公司每股能產生多少現金
  • 適用：高資本支出公司、高折舊攤銷公司

為什麼要看每股營業現金流：

  • 有些公司EPS被折舊攤銷影響很大
  • 每股營業現金流更能反映真實獲利能力
  • 例：重資產產業（電信、運輸、公用事業）

危險訊號

低獲利含金量要檢查什麼：
1. 應收帳款週轉率是否下降
2. 存貨週轉率是否下降
3. 是否有大額業外收入（賣資產）
4. 是否有關係人交易異常

什麼情況要避開：

  • 營業現金流長期為負
  • 淨利為正但營業現金流為負
  • 比值長期低於50%
  • 比值突然從80%掉到50%

心得

營業現金流對淨利比這個指標很實用，直接檢查公司「賺的錢是不是真的有進戶頭」。很多公司帳面獲利漂亮，但現金流很差——這種就是危險訊號。

台積電是很好的範例：長年130%以上代表不只獲利紮實，還因為折舊加回讓現金流比淨利更高。這種公司財務體質健康，不容易出現資金問題。

以後看財報要養成習慣：看淨利的同時，一定要檢查營業現金流對淨利比。80%以下是警訊，60%以下要深入了解原因。這可以幫我避開那些「帳面好看但現金很差」的地雷股。
