VaR 風險值核心概念

定義

是什麼：VaR (Value at Risk) 風險值，是在特定期間內、特定機率下，持有某一資產或投資組合所預期可能發生的最大損失。

白話解釋：用一個數字告訴你「我有 X% 的信心，未來 N 天內損失不會超過 V 元」。

實例：

  • 某部位在 99% 信賴水準下之每日風險值為 100 萬元
  • 意思是：平均在 100 個交易日中，只有 1 日的損失會超過 100 萬元

VaR 三要素

要素說明實例
信賴水準多少信心保證不會超過95%、99%
持有期間多長時間內的風險1 天、10 天
金額最大損失數值100 萬元

怎麼設定：

  • 信賴水準：金融機構常用 99%，個人可用 95%
  • 持有期間：短線交易用 1 天，波段用 5-10 天
  • 巴塞爾協定要求銀行以 10 個營業日為持有期限

VaR 三種計算方法

方法一：變異數-共變異數法

是什麼：假設報酬率為常態分配，用標準差計算風險值。又稱 Delta Normal 法。

優點：

  • 計算效率高、速度快
  • 容易解釋
  • 易於處理增額風險值

缺點：

  • 無法處理厚尾問題（低估極端風險）
  • 需要估計波動度與相關係數
  • 不適合非線性資產（如選擇權）

計算公式：

每日盈餘風險 (DEAR) = |z| × σ × 部位價值

其中：
- z = 信賴水準對應的 z 值（95% = 1.65，99% = 2.33）
- σ = 標準差

實例計算（以台積電為例）：

  • 部位：100 萬元
  • 日報酬率標準差：2%
  • 信賴水準：99%（z = 2.33）

DEAR = 2.33 × 2% × 1,000,000 = 46,600 元

意思：有 99% 信心，明天最大損失不超過 46,600 元

方法二：歷史模擬法

是什麼：用過去歷史資料直接模擬，排序後取分位數。

怎麼做：
1. 收集過去 N 天的報酬率資料（如 100 天）
2. 將報酬率由小到大排序
3. 找出第 5 個百分位（95% 信賴水準）的報酬率
4. 乘以部位價值即為 VaR

優點：

  • 不需假設報酬分配
  • 不需估計波動度和相關係數
  • 能考量厚尾分配及極端事件

缺點：

  • 完全依賴特定歷史資料
  • 未能考量市場結構改變
  • 資料期過短會導致估計偏差

實例：

  • 100 天歷史報酬率，由小到大排序
  • 95% VaR = 第 5 小的報酬率（100 × 5% = 5）
  • 若第 5 小的報酬率是 -3.5%，部位 100 萬
  • VaR = 3.5% × 1,000,000 = 35,000 元

方法三：蒙地卡羅模擬法

是什麼：假設股價遵循隨機過程（如幾何布朗運動），透過大量隨機模擬產生情境。

怎麼做：
1. 建立價格走勢的隨機過程模型
2. 估計參數（平均數、波動度）
3. 模擬數千次路徑（如 1000 次、10000 次）
4. 產生報酬率分配，找出分位數

優點：

  • 可以包含任何風險因子分配
  • 能處理非線性資產
  • 模型設定具彈性

缺點：

  • 電腦資源需求高
  • 隨機過程設定錯誤會有模型風險
  • 極端值未能包含於分配中

絕對 VaR vs 相對 VaR

相對風險值 (Relative VaR)

定義：投資標的相對於報酬率均值的風險值。

公式：

相對 VaR = |-z| × σ × 部位價值

絕對風險值 (Absolute VaR)

定義：投資標的相對於 0 的風險值（考慮平均報酬）。

公式：

絕對 VaR = (|-z| × σ - mean) × 部位價值

差異說明：

  • 相對 VaR：假設平均報酬為 0，單純看波動
  • 絕對 VaR：考慮實際平均報酬，更精確

VaR 的關鍵限制

限制一：厚尾問題

是什麼：常態分配假設低估了極端事件發生的機率。

實際狀況：

  • 股市報酬率常有厚尾現象
  • 極端虧損發生機率比常態分配預測更高
  • 用 VaR 會低估實際風險

限制二：只告訴你不會超過多少

關鍵陷阱：VaR 告訴你 95% 不會超過的損失，但沒告訴你剩下的 5% 會賠多少。

解決方法：搭配 CVaR（條件風險值）或壓力測試使用

限制三：歷史不一定重複

問題：歷史模擬法假設未來會像過去。

VaR 在個人交易的實戰應用

應用一：部位控制

怎麼用：
1. 設定你能承受的最大單日虧損（如 2%）
2. 計算標的標準差（如 3%）
3. 反推最大部位

實例：

  • 帳戶 100 萬，能承受單日虧損 2 萬（2%）
  • 標的日標準差 3%，99% z 值 = 2.33
  • 最大部位 = 20,000 ÷ (2.33 × 3%) ≈ 286,000 元

心得：

VaR 是金融機構的標準風險指標，但對個人交易者也很有用。核心概念很簡單：用一個數字量化「可能的虧損」。

最實用的是變異數-共變異數法，只要有標準差就能算。缺點是它假設常態分配，低估了極端事件的機率——這剛好是交易者最該擔心的部分。

VaR 最大的陷阱是它只告訴你「95% 不會超過多少」，但沒說剩下 5% 會賠多少。這就是為什麼金融海嘯時一堆 VaR 模型失靈——尾部風險被嚴重低估。

對個人交易者來說，VaR 最實際的用法是反推部位：先決定能承受多少虧損，再計算最大部位。這比憑感覺下單科學多了。
