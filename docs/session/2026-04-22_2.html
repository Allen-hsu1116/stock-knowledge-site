核心知識

一、違約交割 Default Settlement Failure

是什麼：股票成交後，在 T+2 早上 10:00 前交割帳戶餘額不足，導致扣款失敗。

兩種類型：

  • 買方違約：T+2 早上 10:00 前帳戶錢不夠（最常見）
  • 賣方違約：交割期限內交不出股票

怎麼做 — 違約交割六層後果：
1. 信用記錄：計入個人信用評分，影響房貸、信用卡、信貸
2. 帳戶註銷：通報後 3 個營業日未清償，證券帳戶被註銷
3. 違約金：券商可收最高成交金額 7% 的違約金
4. 強制賣股：券商直接賣出違約股票償債，不夠繼續追討
5. 5 年禁開戶：違約未結案且未滿 5 年，所有券商不得開戶或交易
6. 刑事責任：情節重大影響市場秩序，最高 3～10 年有期徒刑，得併科 1,000 萬～2 億罰金

實例：

  • 2022/7/11：單一投資人當沖違約 6.6 億元，佔當日總違約金額 7 億的 95%
  • 2024/8/2：台股重挫 1,004 點，10 家券商通報 13 戶違約交割 2.3 億

下錯單補救：

  • 收盤前發現 → 現股當沖反向賣出抵銷，或賣其他持股籌錢
  • 收盤後發現 → 趕快籌錢，T+1 立刻賣出股票還款

適用時機：每個用信用交易的人都必須了解，這是底線知識。行情劇烈波動時特別容易觸發。

二、融資維持率 Margin Maintenance Ratio

是什麼：衡量融資帳戶擔保品是否足夠的比率，低於 130% 就會被追繳。

公式：

融資維持率 = 股票現值 / 融資金額
整戶擔保維持率 = (融資擔保證券市值 + 原融券擔保價款及保證金) / (原融資金融 + 融券證券市值) × 100%

怎麼做 — 維持率三段判讀：
1. 166% 以上：安全區，初始融資維持率即為 166%（100%/60%）
2. 130%～166%：警戒區，券商盯著你，跌破 130% 當日就要補錢
3. 低於 130%：危險區，券商發出追繳令，2 日內補到 166% 以上才撤銷

實例 — 台積電：

  • 買進價 815 元，融資自備 40% = 32.6 萬，融資借款 60% = 48.9 萬
  • 維持率 166% = 81.5 萬 / 48.9 萬
  • 股價跌到 635 元以下（約買價 78 折）→ 維持率跌破 130% → 收到追繳令
  • 跌幅約 22% 就觸發追繳，這就是融資的風險

追繳令三階段處理：
1. 2 日內補到 166% 以上 → 撤銷追繳令，回歸正常
2. 2 日內只補到 130%～166% → 不強制斷頭，但追繳令仍在，再跌破 130% 當日必須補錢
3. 3 日內未補到 130% → T+3 開盤強制賣股（斷頭）

適用時機：使用融資的交易者必懂。大盤重挫時，整戶維持率是觀察斷頭賣壓的關鍵指標。

三、斷頭連鎖效應 Forced Liquidation Cascade

是什麼：融資維持率跌破門檻 → 券商強制賣股 → 賣壓加劇 → 更多帳戶維持率跌破 → 更多斷頭，形成惡性循環。

怎麼做 — 連鎖效應三階段：
1. 觸發期：大盤或個股急跌，融資維持率跌破 130%，追繳令大量發出
2. 擴散期：被追繳者無力補錢或選擇不補，T+3 強制賣股 → 賣壓推動股價續跌 → 觸發更多追繳
3. 結束期：融資餘額大幅下降（散戶被洗出場）或政府/主力進場護盤，賣壓耗盡

實例：

  • 2024/8/2 台股重挫 1,004 點 → 違約交割 2.3 億 → 若持續下跌恐引發連鎖斷頭
  • 歷史經驗：融資餘額高的個股，單日跌停可能引發連續跌停（融資斷頭排隊）

觀察指標 — 如何判斷斷頭壓力大小：

  • 融資使用率：融資餘額 / 融資限額，超過 30% 就要小心
  • 融資維持率分布：看整戶維持率在 130%～140% 附近的帳戶數量，越多危險
  • 融資餘額增減趨勢：融資持續增加 + 股價下跌 = 籌碼從強者轉弱者，後續壓力大

適用時機：

  • 大盤重挫時，觀察融資維持率判斷斷頭壓力
  • 個股融資使用率高時，避開或做好避險
  • 逆向思考：斷頭賣壓耗盡後，可能是撿便宜的好時機

四、融資量×融券量×借券量 籌碼面三指標

是什麼：信用交易三個核心指標，分別反映散戶看多、散戶看空、法人看空的程度。

怎麼做 — 三指標判讀邏輯：

指標使用者增加代表實戰意義
融資量散戶為主散戶看多法人賣超+融資增加=籌碼弱手，股價偏跌
融券量散戶為主散戶看空融券暴增=潛在軋空燃料，回補日有軋空行情
借券賣出餘額法人為主法人看空借券賣出增加=外資避險或看空，減少=偏多

實例：

  • 某股融資量大增 + 主力賣超 → 籌碼從大戶轉散戶 → 股價下跌機率高
  • 某股融券量暴增 + 即將回補 → 回補日前 5～10 天可能是軋空行情
  • 借券賣出餘額從 60 增加到 80 → 看空力量增加，股價偏跌

重要區別：

  • 借券餘額 ≠ 借券賣出餘額（借到不一定賣）
  • 借券賣出餘額才是真正做空的量
  • 融券利率約 0.04%～0.08%（很低），借券利率由出借人決定（可達 3%）

適用時機：選股時交叉驗證籌碼面，不要只看價量。融資增加+法人賣超是最危險的組合。

五、違約交割×斷頭 散戶地雷完整避險 SOP

是什麼：結合違約交割和融資斷頭的完整風險管理流程。

怎麼做 — 五步避險：
1. 下單前確認帳戶餘額：確保 T+2 扣款前帳戶有足夠資金
2. 設定自動停損：用融資的斷頭機制當自動停損，但不要讓它真正觸發
3. 監控整戶維持率：每天收盤後計算，150% 以下就開始警戒
4. 大盤重挫時降部位：維持率在 140% 附近就主動減碼，不要等到 130% 被追繳
5. 避免融資買高波動股：跌幅 22% 就追繳，高波動股很容易觸發

實例：

  • 用融資買台積電 815 元 → 自備款 32.6 萬
  • 設定主動停損線：維持率 150% 對應股價約 733 元（跌 10%）
  • 股價到 733 元就主動賣出，比等到 635 元被斷頭好得多
  • 主動賣出 vs 被斷頭：主動賣出可以選價格，斷頭是開盤市價全倒

適用時機：每個使用信用交易的人都應該有這個 SOP。不融資的人也該懂，因為大盤斷頭壓力會影響所有持股。

心得

今天學了違約交割和融資維持率的實戰判讀，說真的，這是那種「最好永遠不要用到但必須懂」的知識。

幾個關鍵體悟：

1. 融資是雙面刃，跌幅 22% 就追繳：以台積電 815 元為例，跌到 635 元就觸發，這距離其實不遠。很多人以為融資只是「借錢放大報酬」，但忽略了虧損也被放大，而且是被動放大的——你不是自己決定停損，是券商幫你決定。

2. 違約交割的代價遠超想像：不只是罰錢，信用記錄影響 5 年以上，情節重大還有刑責。2022 年那個 6.6 億的案例，一個人當沖失控就搞出這種規模，真的是玩命。

3. 斷頭連鎖效應是系統性風險：大盤重挫時，融資維持率低於 130% 的帳戶全部被追繳，沒補到就 T+3 強制賣 → 賣壓更重 → 更多斷頭。這不是個股問題，是市場結構問題。懂這個邏輯，在暴跌時才不會慌，甚至可以逆向等待斷頭壓力耗盡後進場。

4. 主動停損永遠優於被動斷頭：被斷頭是開盤市價全倒，完全無法控制價格。在維持率 150% 就主動減碼，比等到 130% 被追繳好太多。這個差異，就是專業和業餘的分界線。
