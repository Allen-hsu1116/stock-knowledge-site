配息穩定度判斷

第 46 回合學習 - 配息穩定度判斷

文章：

  • 至少連15年配息、殖利率4％以上！投資達人10檔存股名單公開

核心知識

一、配息穩定度四大判斷條件

是什麼：用來篩選能長期穩定配息的存股標的，確保股利收入不中斷。

怎麼做：

條件標準目的
連續配息年數≧10年確認經歷完整景氣循環
殖利率下限近10年≧4%（嚴格版≧6%）確保配息水準穩定
盈餘配發率≧50%（嚴格版≧70%）公司願意與股東分享獲利
淨利率近10年均為正確認公司賺錢能力

實例：

  • 情境：篩選台股存股清單
  • 操作：用 Goodinfo!台灣股市資訊網篩選「連續10年配息且殖利率≧6%」
  • 結果：全台上市櫃約2000檔，符合條件僅約16檔

適用時機：存股族篩選標的、評估高殖利率股票的真實品質

二、配息率與配息來源

是什麼：配息率＝現金股利÷EPS，代表公司把賺的錢分給股東的比例。

怎麼做：

1. 判斷公司是否大方：配息率≧70%表示公司願意分享獲利
2. 觀察異常情況：配息率>100%代表公司動用其他資金來源

股利三種法定來源：

  • 去年度盈餘（正常來源）
  • 保留盈餘（過去累積的獲利）
  • 資本公積（最不建議動用）

實例：

  • 情境：遠傳(4904)與台灣大(3045)
  • 操作：連續11年與5年配息率>100%
  • 原因：動用保留盈餘、公積來維持高殖利率
  • 判斷：對小股東並非壞事，但要留意是否可持續

適用時機：評估公司股利政策的穩定性與誠意

三、賣股時機判斷

是什麼：存股不是買了就放著不管，需要設定停損條件。

怎麼做：

情況操作
單一年殖利率偏低不急著賣，給1年緩衝
連續2年殖利率≦2年期定存考慮停損
經營者換人重新評估股利政策是否改變

實例：

  • 情境：永信建(5508)近10年僅2020年殖利率5.13%低於6%
  • 操作：不需因單一年度殖利率下降就賣股
  • 結果：近10年平均殖利率10.24%，仍具參考價值

適用時機：當持股殖利率或配息率出現異常變化時

四、買入價格判斷

是什麼：買入價決定殖利率高低，同一檔股票在不同價格買進，殖利率差異很大。

怎麼做：

1. 設定買入價上限：近5年年均平均值
2. 設定目標買價：近5年年均最低價（可遇不可求）
3. 計算殖利率：股利÷買入價

實例：

  • 情境：新巨(2420)，2022年現金股利3元
  • 年均價42.4元 → 殖利率7.08%
  • 近5年年均價36.96元 → 殖利率8.12%
  • 近5年年均最低價33.3元 → 殖利率9.0%

適用時機：尋找合適的存股買點、評估當前價位是否划算

五、左側交易策略

是什麼：在市場恐慌下跌時分批買進，愈跌愈買。

怎麼做：

1. 等待時機：全市場恐慌性下跌（可能數年一次）
2. 分批進場：不要一次買滿
3. 設定群組：在券商App把存股標的設為一個群組便於追蹤

存股三要素：

  • 耐心：等待危機來臨
  • 勇氣：逆勢進場
  • 現金：手頭要有資金

適用時機：市場大跌時分批承接優質存股

心得

這回合學到配息穩定度的判斷框架，重點在於「連續性」和「品質」——連續10年配息且殖利率維持一定水準，才是真正穩定的存股標的。

幾個關鍵收穫：

1. 配息率>100%不一定是壞事：公司可能動用保留盈餘來配息，這代表公司有意願維持股利政策，但要看能否持續。

2. 單一年度異常不用急著賣：給1年緩衝期，連續2年殖利率低於定存才考慮停損。這種耐心是存股族必備的素質。

3. 買入價決定殖利率：同一檔股票在不同價位買進，殖利率差異可達2%。存股不是「什麼價格都可以買」，要等到合理價位。

4. 興櫃股票風險較高：像云光(3633)這種興櫃14年、每年獲利配息的公司，流動性低、財報透明度較差，想投資要更謹慎。
