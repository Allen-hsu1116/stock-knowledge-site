一、交易對手風險核心概念

什麼是交易對手風險？

定義：交易的另一方不履行合約義務的風險，屬於信用風險的一種，也稱為違約風險。

白話說明：就像玩牌或下棋，就算你技術再好、勝算再大，對手可能輸了不認帳或掀桌子。這種風險與投資內容好壞無關，是來自對手的信用問題。

為什麼重要？

2008年雷曼兄弟事件：

  • 雷曼曾是美國第四大投資銀行
  • 全球員工超過26,000人
  • 發生信用危機後投資人血本無歸

五、中央對手方（CCP）機制

CCP是什麼？

中央對手方（Central Counterparty, CCP）：清算所擔任所有交易者的對手方，買方的賣方、賣方的買方。

傳統模式：A ←→ B（雙方直接交易）
             ↓
CCP模式：    A ←→ CCP ←→ B
             （CCP在中間承接風險）

CCP四大功能

功能說明
合約更替CCP成為所有交易的法定對手方
擔保交收任一方違約，CCP先賠償另一方
淨額清算多筆交易軋差，減少資金移動
保证金制度防範對手方風險

CCP的優點

1. 匿名交易：不用擔心對手是誰
2. 風險重新分配：解決連鎖違約問題
3. 清算效率提升：多邊淨額清算
4. 監管透明：掌握交易結算數據

CCP的問題

問題說明
道德風險交易商可能擴大風險，反正CCP會擔保
大而不能倒CCP本身成為系統性風險來源
順周期效應市場波動時要求追加保证金，加劇波動
跨境監管困難各國監管標準不一

六、實戰應用

選擇券商的標準

優先考量：
1. 全球排名前20
2. 受多國監管
3. 資本充足率高
4. 歷史悠久（經歷過金融危機）

避開：

  • 沒聽過的小型券商
  • 只在一國註冊
  • 資本規模小
  • 優惠過度誇張

債券投資實務

投資級債券（BBB以上）：

  • 違約率 < 0.5%
  • 適合保守型投資人
  • 報酬率較低

投機級債券（BB以下）：

  • 違約率 3%~10%
  • 需要分散投資
  • 注意信用評級變化

衍生品交易注意

場內交易（交易所）：

  • 有CCP擔保
  • 違約風險低

場外交易（OTC）：

  • 雙邊清算風險
  • 需要評估對手信用
  • 避免單一對手集中

心得

交易對手風險是很多人忽略的隱形風險。大家常關注「這檔股票會漲嗎？」卻很少問「和我交易的對手會不會違約？」

市場先生的比喻很生動——玩牌時對手可能輸了不認帳甚至掀桌子。這種風險和你的技術無關，和對手的品格有關。

學到幾個重要觀念：

1. 「信用就是錢」：信用評級不是虛的，它決定了資金成本和借貸能力
2. 風險溢價的本質：高報酬 = 高風險的補償，不是免費午餐
3. CCP不是萬能：中央清算所解決了雙邊違約風險，但自己變成「大而不能倒」的系統性風險
4. 監管一直在追風險：2008年後大力推CCP清算，但問題只是被轉移，不是消失

對股票操作的啟發：

  • 選擇交易平台時，優先選大型、有信譽的券商
  • 投資債券時，信用評級比票面利率更重要
  • OTC衍生品要特別小心對手風險
  • 融資融券時要看券商的財務狀況
