融資回補力道計算與真假軋空辨別四步驟

一、軋空的核心機制：為什麼融券回補會推升股價？

軋空（Short Squeeze）是什麼

是什麼：做空投資人預期股價下跌而融券放空，但股價不跌反漲，放空者被迫買回股票回補，大量回補買盤推升股價，又迫使更多放空者回補，形成正回饋循環——這就是軋空。

軋空的三個必要條件：
1. 融券餘額很高：做空比例越大，回補時的買盤力量越強
2. 股票缺乏流動性：籌碼集中在少數人手上，回補時找不到股票賣，價格被推升
3. 接近融券強制回補日：台股特有機制，股東會、除權息前必須強制回補，時間壓力迫使空頭搶買

台股 vs 美股差異：

  • 台股：有「融券強制回補」機制，股東會/除權息前 6 個營業日是最後回補日，不回補就違約交割
  • 美股：沒有強制回補機制，軋空主要靠保證金追繳（Margin Call）觸發

軋空的正回饋循環

怎麼運作：
1. 股價上漲 → 融券投資人開始虧損
2. 虧損擴大 → 維持率下降，可能被追繳或強制回補
3. 強制回補 → 買盤湧入 → 股價進一步上漲
4. 更多融券被追繳 → 更多買盤 → 股價續漲
5. 循環直到融券餘額大幅消化或股價到達不合理高點

實例：

  • 美股 GameStop（GME）2021/1：股價從 20 美元飆升至 347 美元，放空機構損失近 200 億美元
  • 台股：每年 3~4 月股東會前的融券強制回補潮，高券資比個股容易出現軋空行情

二、融資回補力道計算與判讀

券資比：軋空行情的關鍵指標

是什麼：券資比 = 融券餘額 ÷ 融資餘額 × 100%，衡量一檔股票的放空比例相對於做多的比例。

怎麼算：

券資比 = 融券餘額 ÷ 融資餘額 × 100%

判讀標準：

  • 券資比 < 20%：正常範圍，放空壓力不大
  • 券資比 20%~30%：開始留意，放空比例偏高
  • 券資比 > 30%：軋空訊號！太多人放空，只要多方拉抬就容易觸發軋空
  • 券資比 > 50%：極端高值，軋空行情一觸即發

實例：

  • 創惟（6104）2022/1~2 月：券資比超過 30%，大戶持股增加 + 主力買超 → 散戶放空被大戶軋空，股價持續上漲
  • 某高券資比股券資比 80.17%，後續股價暴漲 68.6%

融資回補力道計算

是什麼：估算融券回補時會產生多少買盤力道，判斷軋空行情的潛在強度。

怎麼算：
1. 回補張數：查看融券餘額（證交所每日公布）
2. 佔日均量比例：回補張數 ÷ 日均成交量 × 100%
3. 回補天數：回補張數 ÷ 日均成交量 = 需要多少交易日才能消化

判讀：

  • 回補張數佔日均量 < 20%：回補力道弱，軋空影響有限
  • 回補張數佔日均量 20%~50%：回補力道中等，可能推升股價 3~5%
  • 回補張數佔日均量 > 50%：回補力道強，軋空行情潛力大
  • 回補張數 > 日均量（比例 > 100%）：極端情況，一天消化不完，多日軋空行情

實例：

  • 情境：某股融券餘額 5,000 張，日均成交量 3,000 張
  • 佔日均量比例 = 5,000 ÷ 3,000 = 166.7%
  • 回補天數 = 5,000 ÷ 3,000 ≈ 1.67 天
  • 判斷：融券回補力道極強，需要近 2 天才能消化全部回補買盤
  • 股價可能因回補潮上漲 10% 以上

融券強制回補日期計算

怎麼算：
1. 查詢公司公告的停止過戶日（股東會/除權息）
2. 往前推算 6 個營業日 = 最後回補日
3. 停券時間：最後回補日到停止過戶日之間（約 3~7 天）

實例：

  • 某公司除權息日公告，停止過戶日為 7/21
  • 往前推 6 個營業日 = 7/13（最後回補日）
  • 7/13 之前必須完成回補，否則券商強制市價買回

三大強制回補時機：
1. 股東會：每年 3~6 月密集，常會前 2~3 個月公告停止過戶
2. 除權息：6~8 月密集，一年可能配息 4 次 = 4 次強制回補
3. 現金增資：不定期，CB 轉換也會觸發

重要：臨時股東會不須融券回補！

三、真假軋空辨別四步驟

核心觀念：「十個軋空九個假」

是什麼：高券資比不代表一定會軋空。陳威良老師根據 20 年看盤經驗指出，十個高券資比個股中，大約只有一個會出現真正的軋空行情。大部分高券資比是「假的」——背後有其他原因。

第一步：確認融券增加的原因

真軋空的融券特徵：
1. 散戶看壞而放空 → 股價卻不跌反漲 → 被主力軋空
2. 搭配大戶持股增加 + 主力買超 → 印證大戶做多
3. 融券持續增加但股價不跌 → 空頭逐漸被鎖定

假軋空的常見原因：
1. 可轉換公司債（CB）套利：投資人買 CB 同時融券放空現股鎖價差，這種融券不會因股價上漲而急著回補，因為 CB 也跟著漲，套利空間沒變
2. 現金增資套利：認股價低於市價，融券放空鎖價差，同理不會急著回補
3. 避險放空：持有現股同時融券放空部分部位避險，不是真的看空
4. 主力自己融券：主力左手買右手融券，製造高券資比假象吸引散戶跟進做多，之後反手出貨

怎麼做：
1. 查看個股近期是否有 CB 發行或現金增資公告
2. 如果有 → 高券資比很可能是套利放空，不是真軋空
3. 如果沒有 → 進入第二步驗證

實例：

  • 某股券資比 60%，看起來很高
  • 但查公告發現近期有 CB 發行，轉換價低於當前股價 15%
  • 判斷：大部分融券是 CB 套利部位，不會因股價上漲而恐慌回補
  • 結論：假軋空，不要追多

第二步：搭配大戶持股與主力動向

怎麼做：
1. 查看大戶持股比例變化（集保股權分散表）
2. 查看主力買賣超數據（CMoney 籌碼 K 線）
3. 大戶持股增加 + 主力買超 + 融券增加 = 散戶在放空、大戶在買 → 真軋空機率高
4. 大戶持股減少 + 主力賣超 + 融券增加 = 主力出貨、散戶接刀 → 假軋空

實例：

  • 創惟（6104）：大戶持股增加 + 主力買超 + 券資比 > 30% → 散戶放空被軋，股價續漲
  • 反例：某股券資比 50%，但大戶持續減持 + 主力賣超 → 主力在出貨，散戶的融券可能只是「最後一里路」的空單，不會被軋

第三步：觀察股價是否抗跌

怎麼做：
1. 真軋空的核心特徵：股價易漲難跌
2. 大盤下跌時，該股相對抗跌（甚至逆勢上漲）
3. 融券持續增加，但股價就是不跌 → 多方力道強，空頭正在被「養」
4. 反之：融券增加 + 股價也下跌 → 放空有道理，不是軋空格局

判斷方式：

  • 5 日均線上揚 + 融券增加 → 偏多軋空格局
  • 5 日均線下彎 + 融券增加 → 放空有理，不會軋空
  • 股價在支撐區附近 + 融券增加 → 可能是主力養空，準備軋空

第四步：計算回補時間壓力

怎麼做：
1. 查詢該股最近的強制回補日（股東會/除權息前 6 個營業日）
2. 距離回補日越近 → 回補壓力越大 → 軋空力道越強
3. 計算回補張數佔日均量比例（如前面所述）
4. 黃金時機：回補日前 2~3 週開始觀察，回補日前 1 週是軋空最可能發生的時段

距離最後回補日回補壓力軋空機率操作建議
> 30 天低低觀察，不急
15~30 天中中開始佈局，小量進場
7~15 天高高主要進場時段

四、軋空行情操作三條件

條件一：券資比 > 30% 且非套利放空

怎麼做：
1. 確認券資比超過 30%
2. 排除 CB 套利和現金增資套利
3. 確認是散戶「真的看空」而放空

條件二：大戶持股增加 + 主力買超

怎麼做：
1. 大戶持股比例近期增加（集保股權分散表）
2. 主力近 5 日買超
3. 多方力道 > 空方力道

條件三：股價站上關鍵均線且抗跌

怎麼做：
1. 股價站上 5 日均線（短線偏多）
2. 20 日均線上揚（中期趨勢向上）
3. 大盤下跌時該股相對抗跌或逆勢上漲

三個條件同時滿足 → 軋空行情機率高，可考慮做多
任一條件不滿足 → 觀望，不要賭軋空

軋空行情出場時機

1. 融券餘額大幅減少：空頭已經認輸回補，軋空動能消退
2. 股價出現爆量長黑：主力趁軋空行情出貨
3. 融資餘額暴增：散戶搶進 = 接刀人出現
4. 利多消息見報率大增：媒體開始報導飆股 = 階段 6~7，主力準備出場
5. 移動停損出場：不預設目標價，用移動停損跟著趨勢走

五、融券放空避雷指南

哪些股票不適合融券放空

1. 一年多次配息的股票：一年最多 4 次除息 + 股東會 + 除權，強制回補次數太多，平均放空 2 個月就要回補
2. 高券資比個股：券資比 > 30% 的股票，容易被大戶鎖定軋空
3. 大戶持股增加 + 主力買超的個股：大戶做多你做空，被軋是遲早的事
4. 流動性差的股票：日均量 < 500 張的股票，回補時找不到賣方，容易被高價成交
5. 接近股東會/除權息的股票：強制回補日前 2 週不要放空

融券放空風控

1. 停損設定：進場價上方 5~8% 設停損（做空虧損是無限的，必須嚴格控制）
2. 部位控制：單一股票放空部位不超過總資金的 5%
3. 提前回補：不要壓到最後回補日，至少提前 2 週開始規劃
4. 替代方案：如果想長期看空，考慮用股票期貨或買進賣權（Put），不受強制回補限制

心得

這回合把軋空行情從「聽過但不太懂」變成「有具體框架可以判斷」。最關鍵的認知翻轉是：高券資比 ≠ 軋空。以前看到券資比超過 30% 就覺得會軋空，結果十個裡面九個不會。原因很簡單——很多融券不是「真的看空」，而是 CB 套利、現增套利、避險放空，這些融券不會因為股價上漲而恐慌回補，所以不會形成軋空的正回饋循環。

真假軋空辨別四步驟是這回合最實用的框架：先排除套利放空、再看大戶和主力方向、觀察股價抗跌性、計算回補時間壓力。以前只看券資比一個指標，現在四個維度交叉驗證，勝率應該會高很多。

另一個重要收穫是「回補力道計算」。以前只知道融券餘額很高，但不知道怎麼量化回補對股價的影響。用「回補張數佔日均量比例」來計算，就能判斷回補潮需要幾天消化、對股價推升力道有多大。回補張數超過日均量 100% 的股票，軋空行情才有可能持續多天。

最後，市場先生那句話很實在：「做空是高風險的交易。」股價最多跌到零（做空獲利有限），但可以無限上漲（做空虧損無限）。融券放空前一定要算好券資比、查清楚 CB 和現增公告、設好停損。不然被軋一次，幾個月的薪水就沒了。
