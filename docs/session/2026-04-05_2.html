均線扣抵值速查

  • 扣抵值：即將被移除的收盤價位置（如 5MA 扣抵 = 5 天前收盤價）
  • 判斷公式：新收盤價 > 扣抵值 → 均線上揚；新收盤價 < 扣抵值 → 均線下彎
  • 蹺蹺板記憶法：左邊（扣抵值）下降 → 右邊（股價）容易上升
  • 支撐壓力預判：扣抵值趨勢向下 → 壓力疲弱；扣抵值趨勢向上 → 壓力強勁
#### 葛蘭碧八大法則速查 **買進訊號**：
  • ① 突破：均線由降轉升，股價突破均線
  • ② 假跌破：跌破均線後隨即回到均線上，均線仍上升
  • ③ 支撐：股價在均線上，回檔未跌破均線就反彈
  • ④ 抄底：股價急跌遠離均線後開始反彈

賣出訊號：

  • ⑤ 跌破：均線由升轉降，股價跌破均線
  • ⑥ 假突破：突破均線後隨即跌破，均線仍下降
  • ⑦ 反壓：股價在均線下，反彈無法突破均線
  • ⑧ 反轉：股價急漲遠離均線後開始下跌

均線糾結操作

  • 判斷：多條均線糾結在一起（相差 0.5% 以內）
  • 買點：帶量突破糾結區 + 均線向上發散
  • 停損：突破 K 棒最低點
  • 諺語：盤久必漲或盤久必跌

多頭/空頭排列

  • 多頭排列：K 線 > 5MA > 10MA > 20MA > 60MA（所有人獲利）
  • 空頭排列：K 線 < 5MA < 10MA < 20MA < 60MA（所有人虧損）
