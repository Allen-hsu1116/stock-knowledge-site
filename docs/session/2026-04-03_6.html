交易日誌四步驟實戰法

是什麼：交易日誌不只是買賣資金紀錄，而是幫助穩定情緒、嚴守規則、減少凹單的完整系統。

怎麼做：

步驟一：買進後立刻紀錄
1. 買進日期、張數、股票名稱
2. 買進理由 — 必須是「股票已到達買點」，不是「我覺得會上漲」
3. 技術分析圖截圖 — 標註停損點、買點、支撐壓力

步驟二：持有期間設定規則（90%人沒做到）
1. 時間停損：在 X月X日，若價格沒到 X 元就賣出
2. 空間停損：設定百分比停損或均線停損
3. 最大可能虧損：預估極端狀況下的虧損
4. 看盤時段：開盤 9:00-9:30？尾盤 13:00-13:30？
5. 停利點：帳面獲利超過 X 元或到達 X 價位
6. 加碼點：穿越 X 價格或帳面獲利 X 元再買一張
7. 保險計畫：如果大盤狂跌導致個股瞬間穿過停損，該怎麼辦？
8. 每日心情記錄：買進後每天寫下當下感受

**步
步驟三：賣出時的紀錄
1. 賣出時間
2. 賣出股價 × 張數
3. 獲利/虧損金額
4. 賣出理由（與買進理由對照）
5. 技術線型截圖（在K棒上標註賣出點位）

步驟四：檢討（最多人忽略的一步）

  • 過一個月後再做檢討（用馬後炮看更清楚）
  • 賣太早？買太晚？
  • 策略哪裡需要改進？
  • 停損時間該提早或延後？
  • 停損方式要更寬或更窄？
  • 買進與賣出理由一致嗎？
  • 勝率多少？
  • 持股夠有效率嗎？
  • 賺錢了，如何複製？行為做對了嗎？
  • 賠錢了，如何避免？行為做錯了嗎？

實例：

  • 情境：買進台積電 100 元
  • 操作：買進理由「突破前高且成交量放大」、停損設 95 元（-5%）、停利設 110 元（+10%）、時間停損設 10 個交易日、每日收盤後記錄心情
  • 一個月後檢討：股價到 108 就回落，是否停利點設太遠？下次可以分批停利

適用時機：每一筆交易都應該寫，不管大額小額

交易日誌六大內容架構

是什麼：用 5W1H 架構組織交易日誌，讓記錄更完整、更有系統。

怎麼做：

1. 市場現況觀察

  • 趨勢面：指數漲跌、產業輪動、技術型態
  • 數據面：經濟指標（非農就業、失業金人數、消費者信心、PMI）
  • 問自己：這值不值得進場？可能是什麼原因？下次遇到會進場嗎？

2. 最近學到的投資新知

  • 讀完一本投資書的心得
  • 新學會的理論或指標
  • 整理成筆記默寫在日誌中
  • 例如：「亞當理論」— 看到什麼趨勢就做什麼交易，不作其他聯想

3. 買進策略

  • A 價位買進多少單位
  • B 價位再買進多少單位
  • C 價位又再買進多少單位
  • 預計達到多少獲利？
  • 可承受多少虧損幅度？

4. 賣出策略

  • 停利點設定
  • 停損點設定
  • 時間出場條件
  • 特殊狀況應對（如熔斷、黑天鵝事件）

5. 買賣回顧

  • 績效總體檢
  • 發現自己的交易模式
  • 找出需要改進的地方
  • 例如：發現自己太沉迷盯盤，無形中付出太多手續費

6. 給自己的叮嚀

  • 用來穩定情緒的話
  • 例如：「不要頻繁下單」
  • 例如：「股市漲時賺金錢，跌時賺經驗」

實例：

  • 情境：2020 年疫情崩盤
  • 操作：市場現況寫下「美股連續熔斷」、賣出策略寫「靜待市場反彈」、給自己的叮嚀寫「這是我唯一能做的」
  • 結果：冷靜度過風暴，後來價格回調還小有獲利

適用時機：每次買賣都寫、每週回顧一次

交易日誌關鍵心法

核心觀念：

  • 「我覺得股票會上漲」VS「股票已經到達買點」— 後者才是紀律交易的思維
  • 規則的力量：讓盤中不用思考，只要照著開盤前設定的規則操作
  • 賺錢 ≠ 行為做對（可能是運氣）、賠錢 ≠ 行為做錯（要具體分析）
  • 成功交易者：記下所有行為，並知道有所不為

常見錯誤：

  • 只有買賣資金紀錄（那是 Excel 就能做到的）
  • 盤中思考要不要賣（應該在買進時就設定好）
  • 馬上檢討（應該過一個月再檢討，用馬後炮看更清楚）
  • 寫完不回顧（日誌是拿來檢討的，不是寫完就放著）

實用技巧：

  • 買進理由截圖上面要標註停損、買點、支撐壓力
  • 每天記錄心情，才不會忘記當下的感受
  • 可以替每天股價變化拍照或截圖
  • 如果盤中確信明天會上漲但規則說今天賣 → 下次訂規則時加但書

心得

交易日誌這東西，以前覺得是「功課」，現在覺得是「武器」。不是寫給老師看的，是寫給自己的。

最關鍵的是第二步「持有期間設定規則」— 90%的人沒做到，但這才是讓你不用盯盤的關鍵。買完就訂好停損停利時間價位，盤中只要照規則走，不用思考。這才是紀律交易者的做法。

還有一個重點：買進理由不能是「我覺得會漲」，要寫「符合買點的條件」。前者是主觀交易，後者是機械化交易。成功交易者不是靠預測，是靠規則。

最後，一個月後再檢討這個觀念很妙。馬後炮的好處是看得很清楚：哪裡賣太早、哪裡買太晚、策略要怎麼調。沒有交易日誌，這些都不會知道。
