跳空缺口交易策略

跳空缺口交易策略

缺口是市場情緒的放大鏡——突破缺口找趨勢，竭盡缺口找反轉，分辨類型比缺口大小更重要

核心概念

跳空缺口（Gap） 是K線圖上兩根相鄰K棒之間出現的價格真空地帶，代表一段完全沒有交易的價格區間。它的出現意味著市場參與者的情緒發生了劇烈變化——多空某一方以壓倒性的力量直接跳過了某個價格區間。

缺口的形成原因：

  • 隔夜消息：公司財報、產業政策、經濟數據
  • 突發事件：戰爭、天災、黑天鵝事件
  • 市場情緒共振：投資人集體形成強烈共識

四大缺口類型

類型出現時機成交量意義回補機率
----------------------------------------
普通缺口盤整區間無明顯變化趨勢不明，參考價值低非常高
突破缺口突破盤整/趨勢線明顯放大新趨勢的開始，含金量最高低
持續缺口趨勢中途平穩或溫和放大趨勢加速確認，可估算目標價較低
竭盡缺口趨勢末端爆出巨量趨勢即將反轉的警示非常高且迅速

重要辨別原則

  • 缺口大小 ≠ 趨勢強度：一個帶量中等大小的突破缺口，趨勢強度可能遠大於沒有成交量支撐的巨大竭盡缺口
  • 關鍵在類型和成交量，不是缺口的絕對大小
  • 只有日線以上的跳空缺口才有分析和操作價值，日內/小時線缺口參考價值低

實戰應用

策略一：跟隨突破缺口（順勢交易）

這是勝率最高的缺口交易策略：

  • 進場時機：突破缺口形成當天，或後續幾天確認站穩缺口之上時買入
  • 停損設定：缺口下緣（向上突破）或上緣（向下突破）
  • 邏輯：強勢突破缺口會形成新的支撐/壓力，跌破代表突破失敗
  • 適用對象：突破長期盤整區的強勢股

策略二：捕捉竭盡缺口反轉（逆勢交易）

進階策略，風險較高但報酬也大：

  • 進場時機：確認價格反轉並「封閉」缺口後才進場
  • 停損設定：設在近期高點/低點之外，必須嚴格
  • 確認條件：爆量跳空 + 幾天內迅速回補 = 竭盡缺口確認
  • 適用對象：漲幅或跌幅已大，且出現量價背離的個股

持續缺口目標價估算

持續缺口又稱「測量缺口」：

  • 從趨勢起點到持續缺口的距離 ≈ 從持續缺口到趨勢終點的距離
  • 可用來粗略估算目標價位
  • 當第二個持續缺口出現時，要警覺可能是竭盡缺口

島狀反轉識別

最強烈的反轉型態之一： 1. 先向上跳空（竭盡缺口） 2. 中間盤整幾天，K線像海上的孤島 3. 再向下跳空（突破缺口） 4. 代表多空情勢根本性逆轉

注意事項

「缺口必補」不是鐵律

  • 普通缺口、竭盡缺口：回補機率很高
  • 突破缺口、持續缺口：不回補才是強的證明
  • 如果一個向上突破缺口很快被回補 → 大概率是假突破

缺口回補的訊號意義

  • 向上跳空後被回補：多頭力道減弱，可能反轉
  • 向下跳空後被回補：空頭力道趨緩，可能止跌

向上跳空卻下跌的陷阱

  • 很可能是竭盡缺口特徵（開高走低）
  • 主力利用開盤跳高的機會大量出貨
  • 強烈反轉警訊，千萬不要追高

缺口不能單獨使用

必須搭配：

  • 成交量：帶量缺口才有意義
  • 均線（MA）：確認趨勢方向
  • RSI：判斷超買超賣
  • 整體市場環境和基本面

週期選擇

只有日線以上的跳空缺口有操作價值：

  • ✅ 日線、週線、月線缺口：有分析價值
  • ❌ 分鐘線、小時線缺口：參考價值低
