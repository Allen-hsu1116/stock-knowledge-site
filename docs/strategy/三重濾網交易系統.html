三重濾網交易系統

Elder的經典多時間框架交易架構：潮汐定方向、波浪找時機、漣漪確認進場

核心概念

三重濾網交易系統 (Triple Screen Trading System) 是 Alexander Elder 在《Trading for a Living》（《操作生涯不是夢》）中提出的交易方法。

Elder 用海洋的三種波動來比喻市場：

比喻市場對應時間框架指標類型
---------------------------------
潮汐 Tide主趨勢週線趨勢型（MA、MACD）
波浪 Wave中期趨勢日線震盪型（KD、RSI）
漣漪 Ripple短期波動盤中（60分K）突破型
潮汐是長線主趨勢，漲潮時海水愈來愈近海岸，退潮時愈來愈遠。波浪是潮汐中的一波波起伏。漣漪是波浪碰到地形的浪花，漂亮但對大方向無關緊要。

三重濾網的核心邏輯：不要逆著潮汐操作，在波浪中找時機，從漣漪確認進場。

實戰應用

第一重濾網：掌握主要趨勢（週線）

  • 用趨勢型指標：MA多頭排列、MACD黃金交叉
  • 目的：決定只做多還是只做空
  • 週線MA多頭排列 → 只找做多機會，不做空
  • 週線MA空頭排列 → 只找做空機會，不做多

第二重濾網：找進場時機（日線）

  • 用震盪指標：KD、RSI
  • 目的：在主趨勢方向的拉回中找進場點
  • 主趨勢多方 + KD超賣區（K<20, D<30）= 拉回買點
  • 主趨勢空方 + KD超買區 = 反彈空點
  • 也可用 Elder-Ray牛熊力量指標 的空頭力竭訊號

第三重濾網：確認進場點（盤中/60分K）

  • 用突破型策略：突破近期新高、帶量突破
  • 目的：從短期波動確認進場時機
  • 價格突破60分K近期高點 → 確認進場

實際建構方法

在XQ系統上： 1. 前兩重用選股腳本（週線+日線），每日自動執行 2. 第三重用策略雷達（60分鐘線），盤中即時警示 3. 三重濾網串成一個跨頻率的交易策略

選股腳本範例：

  • 週線：移動平均線多頭排列（Close > MA6）
  • 日線：KD進入超賣區（K<20, D<30）
  • 60分K：突破近期10期新高

為什麼這樣設計？

市場老手常說：

  • 不要逆勢操作
  • 用長線保護短線
  • 看長作短

三重濾網交易系統把這些口語化的原則，系統化為可執行的交易規則。

注意事項

  • 三重濾網不等於三個指標：是三個「層級」的判斷，每個層級用不同類型的指標
  • 不要跳過第一重：不看大方向直接用KD做交易是散戶常見錯誤
  • 前兩重必須是日線以上：受系統限制，無法用日線+60分+1分K構成三重濾網
  • 指標類型要匹配：第一重用趨勢型、第二重用震盪型、第三重用突破型，不要混淆
  • 三重都通過才進場：任何一重不通過就不做，寧可錯過不要做錯
