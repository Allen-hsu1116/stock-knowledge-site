CAN SLIM 選股法

CAN SLIM 選股法

William O'Neil 研發的七字母成長股選股系統，結合基本面與技術面，從 1953 年以來的飆股中歸納出最大漲幅前的共同特徵。AAAI 評為 1998-2009 年最佳投資策略。

核心概念

CAN SLIM 是 O'Neil 在《How to Make Money in Stocks》一書中提出的選股方法，每個字母代表一個篩選維度：

C — Current Quarterly Earnings（當季獲利）

  • 最近一季 EPS 年增率至少 25%（理想 20-50%+）
  • 獲利加速成長更是正面訊號
  • 歷史數據：飆股在起漲前，當季獲利平均成長 70%
  • 實例：Dell 74%/108% → 587%；Apple 350%/300% → $12→$202

A — Annual Earnings（年度獲利）

  • 近三年 EPS 逐年成長
  • 近五年複合年成長率至少 24%
  • 年度 ROE ≥ 17%
  • 實例：Wal-Mart 年增 43% → 11,200%；Priceline EPS $0.96→$2.03 → 股價 3 倍

N — New（新事物）

  • 95% 的飆股有「新」：新產品、新服務、新管理、新市場、或產業變革
  • 也指股價創新高——98% 散戶不敢買新高股，但「看似太高的通常會更高」
  • 實例：Cisco 網路設備 → 75,000%；McDonald's 連鎖加盟 → 1,100%

S — Supply and Demand（供需）

  • 95% 飆股在主升段流通股數 < 2,500 萬股，平均僅 1,180 萬股，中位數 460 萬股
  • 公司持續實施庫藏股是加分項
  • 流通籌碼少 → 少量買盤就能推升股價

L — Leader or Laggard（領先或落後）

  • 買產業龍頭，不買落後股
  • 相對強度（RS）排名 ≥ 80（即過去一年贏過 80% 的股票）
  • 歷史數據：500 檔飆股起漲前平均 RS = 87
  • 實例：Home Depot RS 領先 → 1,766%；IBM 落後 → 跌 33%

I — Institutional Sponsorship（法人持股）

  • 需要有法人支撐（基金、銀行、壽險），特別是近幾季加碼
  • 用累積/分配（A/D）評等衡量法人動向
  • 過度持有是警訊：當所有法人都在裡面時，可能已太晚
  • 反面教材：Xerox 1974 年法人仍加碼，最終慘賠

M — Market Direction（大盤方向）

  • 3/4 的股票跟著大盤走，逆勢操作勝率低
  • O'Neil 將大盤分三狀態：確認上升趨勢 / 壓力下上升 / 修正中
  • 透過 S&P 500 和 NASDAQ 的價量判斷大盤頭部
  • 只在確認上升趨勢中進場

實戰應用

完整 CAN SLIM 交易流程

1. 篩選：用七個條件逐一過濾，任一條件不滿足就跳過 2. 找買點：等股價從「基底」（base）突破，最常見型態是杯柄型態（cup-with-handle），至少 7 週 3. 設停損：買進價下方 7-8%，無例外 4. 判大盤：確認大盤處於上升趨勢才進場 5. 選龍頭：同產業選 RS 最高的那檔

進場確認清單

  • [ ] 最近一季 EPS YoY ≥ 25%（或月營收 YoY ≥ 20%）
  • [ ] 近三年 EPS 逐年成長
  • [ ] 有新產品/新題材/新題材發酵
  • [ ] 流通籌碼不過度龐大
  • [ ] RS 排名前 20%（贏過 80% 個股）
  • [ ] 法人近幾季持續加碼
  • [ ] 大盤處於上升趨勢
  • [ ] 股價從基底帶量突破

注意事項

1. 七個條件必須全用：O'Neil 強調不能用「喜歡的部分」，跳過停損的都賠錢 2. 不是動能投資：CAN SLIM 找的是基本面強（獲利加速）+ 技術面突破的股票，不是純追價 3. 7-8% 停損不可妥協：這是系統的核心防禦機制，很多批評者都因為沒用停損才失敗 4. 新高恐懼症：98% 散戶不敢買創新高的股票，但飆股一定會創新高——要克服這個心理障礙 5. 法人過度持有反而危險：當一檔股票被所有法人抱滿，出事時的賣壓會非常可怕 6. 實戰績效爭議：O'Neil 自己的兩檔共同基金表現平庸，理論和實際操作有落差——回測好看不代表實盤能複製 7. 大型股不適用：流通股數條件排除了大部分大型股，在大型股主導的市場中可能錯過機會 8. 杯柄型態至少 7 週：不要把幾天的盤整當基底，真正的基底需要時間成形
