多時間框架分析

用多種時間週期辨識行情趨勢，長線保護短線，大方向對了再找進場點

核心概念

多重時間框架（Multi TimeFrame, MTF）是透過多種時間週期來辨識股價行情趨勢的交易技術。

很多人對同一檔股票看多看空意見不同，其實兩邊都沒錯，只是談論的週期不同。日線看空但月線看多，可能是月線多頭中的日線回檔——這就是多時間框架要解決的問題。

核心原則：長線保護短線 長期趨勢難以扭轉，若長期趨勢為多方架構，短線操作就應以偏多思考為核心。

經典操作範例

1. 月線級別 KD 黃金交叉 → 多方格局確認 2. 週線級別 KD 也黃金交叉 → 中期也偏多 3. 日線級別 KD 黃金交叉 → 短期轉強買入訊號

三個級別共振，勝率最高。

重點：不能用日線操作當沖（1天才1根K棒），時間框架要匹配操作策略。

實戰應用

Step 1：判斷大方向（最高層級時間框架）

  • 用趨勢型指標（MA、MACD）判斷主趨勢方向
  • 主趨勢往上 → 只找做多機會
  • 主趨勢往下 → 只找做空機會

Step 2：找進場時機（中間層級時間框架）

  • 用震盪指標（KD、RSI）找超買超賣
  • 主趨勢多方 + KD 超賣區 = 拉回買點
  • 主趨勢空方 + KD 超買區 = 反彈空點

Step 3：確認進場點（最短層級時間框架）

  • 用突破型策略確認進場
  • 價格突破近期高點、帶量突破等
  • 也可以搭配 Elder-Ray牛熊力量指標 確認空頭力竭

多時間框架三重確認法

這就是 三重濾網交易系統 的核心邏輯——三個層級的時間框架分別用不同類型的指標，互相驗證。

注意事項

  • 不要逆著大方向操作：月線空頭卻在日線做多，勝率極低
  • 時間框架要匹配：當沖看日線沒意義，波段看5分K噪音太多
  • 三個級別不必同時發出訊號：大方向確認後，等中短期出現機會即可
  • 不同級別訊號矛盾時，以大級別為準
  • 多時間框架不是看越多級別越好：三個級別就夠了，太多反而混亂
