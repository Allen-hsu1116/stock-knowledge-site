海龜交易法則 Turtle Trading

海龜交易法則 Turtle Trading

最經典的順勢交易系統：20日突破進場、ATR停損、金字塔加碼，40年來仍然有效。

核心概念

1983年，理查德·丹尼斯和威廉·埃克哈特做了一個實驗：招募13名新手，每人100萬美元，用一套完全機械化的交易系統操作。四年後這些「海龜」創造了年均80%的複利報酬，震驚全世界。

海龜系統的核心精神：不判斷、不預測、不猶豫。當市場突破過去N天的價格區間，立刻開倉。所有規則都是機械化的，沒有任何主觀判斷空間。

雙系統設計

系統一（短期）系統二（長期）
---------
進場20日最高價突破55日最高價突破
出場10日反向突破20日反向突破
特性靈敏、訊號多過濾雜訊、大趨勢
系統一容易進出頻繁但捕捉短期趨勢，系統二過濾掉較多假突破但進場較晚。建議兩個系統同時運作，分配不同資金比例。

實戰應用

進場規則

做多：當日收盤價突破過去20個交易日最高價 → 隔日開盤買進 做空：當日收盤價跌破過去20個交易日最低價 → 隔日開盤賣出

突破確認：

  • 必須是收盤突破，不是盤中突破
  • 隔日開盤執行，避免盤中假突破
  • 突破當天帶量更可靠

ATR停損系統

海龜交易法則的停損以ATR（Average True Range）為基礎：

  • ATR(20) = 過去20日真實波幅的平均值
  • 每週調整一次停損點
  • 2倍ATR停損：最大虧損 = 總資金的2%
  • 1倍ATR停損：最大虧損 = 總資金的1%
  • 0.5倍ATR停損：停損較小，可加碼更多單位

金字塔加碼

海龜系統的核心優勢：在獲利的部位加碼，而不是在虧損的部位攤平。

  • 每上漲0.5倍ATR → 加碼一個單位
  • 最多加碼4次（共5個單位）
  • 每次加碼後重新調整停損點到最新進場價 - 2倍ATR
  • 絕不在虧損部位加碼（這是亞當理論第一原則）

部位大小計算

海龜系統的部位大小由ATR決定，不是固定金額：

  • 單位大小 = 帳戶淨值的1% ÷ (ATR × 每點價值)
  • 波動大的商品 → 單位小
  • 波動小的商品 → 單位大
  • 目的：讓每個部位的風險一致

注意事項

台股適用性調整

1. 20日突破可能太短：台股淺碟市場假突破多，建議用55日系統或加上帶量確認 2. 漲跌幅限制影響：台股10%漲停限制，突破後可能買不到，需搭配限價單策略 3. 週轉率問題：系統一進出頻繁，手續費侵蝕明顯，建議以系統二為主 4. 流動性篩選：只操作日均量足夠的股票，避免突破後無法成交

心理挑戰

  • 連續小虧：盤整期可能連虧10幾次，每次小賠2%，需要極大紀律
  • 大賺時不跑：趨勢行情中要忍住不提早獲利了結，讓利潤奔跑
  • 機械化執行：所有訊號必須無條件執行，不能因為「感覺不對」而跳過

為什麼40年仍有效？

  • 趨勢追蹤不預測市場，通過自然反應自成體系
  • 風險管理方法學讓每次虧損可控
  • 市場永遠會有趨勢，只是什麼時候出現不知道
  • 多數人無法堅持機械化執行，這本身就是護城河
