券資比判讀

券資比判讀

券資比是融券餘額除以融資餘額的比值，30%為多空分水嶺，高券資比暗示軋空行情可能啟動

核心概念

券資比 = 融券餘額 ÷ 融資餘額 × 100%

這個比率反映市場多空力量對比：

  • 券資比 > 30%：看空的人較多，散戶放空意願強
  • 券資比 < 30%：看多的人較多，散戶偏多
  • 以往70%以上才算高券資比，近年40%以上就會引起注目

背景知識：

  • 融資：向券商借錢買股票，自備40%，券商出60%，通常視為散戶動向指標
  • 融券：向券商借股票賣出，自備90%保證金，看空工具
  • 兩者都是信用交易，具有槓桿風險

實戰應用

1. 高券資比的軋空行情邏輯

高券資比出現的典型流程： 1. 股價已上漲數日，散戶覺得到高點 2. 散戶開始融券放空，券資比攀升 3. 主力可能刻意賣出造成下跌，誘使更多散戶加碼做空 4. 券資比提高到一定程度 5. 主力大量買進或釋出利多，散戶被迫停損回補 6. 股價易漲難跌，形成軋空行情

操作策略：當券資比偏高且法人持續買超時，站在主力同一邊做多，等待軋空。

2. 券資比搭配其他指標

搭配指標判斷方式
--------------------
外資買賣超高券資比 + 外資持續買超 = 強力軋空訊號
投信買賣超高券資比 + 投信加碼 = 中線軋空潛力
融資餘額變化融資減 + 融券增 = 散戶多翻空，留意反轉
股價位置低檔高券資比比高檔更有軋空力道

4. 實實戰案例

穎漢(4562)案例：

  • 股價從5月開始上攻，從十幾元翻倍大漲
  • 券資比在6月暴增，散戶認為到了高點開始放空
  • 外資持續買超
  • 融券做空的散戶被迫回補
  • 股價具備易漲難跌特性，上攻至80.1元

注意事項

  • 主力也會使用融資/融券：不要假設融資一定是散戶、融券一定是散戶
  • 單一指標不足：券資比需搭配基本面、法人籌碼、技術面綜合判斷
  • 高券資比≠一定軋空：可能主力也看空，需確認法人動向
  • 券資比會變化：需追蹤趨勢，不是看單一時點
  • 信用交易槓桿風險：融資融券都有放大風險，停損要嚴格
