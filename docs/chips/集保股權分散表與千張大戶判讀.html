集保股權分散表與千張大戶判讀

集保股權分散表是追蹤大戶籌碼動向的週度數據，千張大戶持股比例變化是判斷籌碼集中與分散的核心指標

核心概念

什麼是集保股權分散表

由臺灣集保結算所（TDCC）每週六上午 9 點公告，記錄每檔股票各持股級距的人數、股數及佔集保庫存比例。資料範圍為前一週最後營業日的集保登摺餘額（不含公告當日週五的買賣資料）。

持股級距分為 15 級：1-999、1,000-5,000、5,001-10,000⋯⋯到 1,000,001 以上，分別對應散戶、中實戶、大戶的分布。

千張大戶的定義

持有某檔股票超過 1,000 張（100 萬股）的投資人，極可能是公司內部人（董事、監察人、經理人）或大股東、機構法人。

注意：不到千張的內部人不會被列入千張大戶，400 張以上的中實戶也具有參考價值。

籌碼混亂 vs 籌碼集中

狀態大戶散戶股價表現
----------------------------
籌碼集中比例上升比例下降易漲難跌
籌碼混亂比例下降比例上升易跌難漲

實戰應用

千張大戶持股比例變化判讀

1. 連續3週千張大戶持股比例上升 → 大戶持續進場，偏多訊號 2. 連續3週千張大戶持股比例下降 → 大戶持續退場，偏空訊號 3. 變動幅度太小（<0.5%） → 可忽略，屬正常調節 4. 變動幅度大（>1%） → 需特別留意，可能有大戶進出

400張大戶 — 比千張更靈敏

400-999 張級距的大戶又稱「中實戶」，比千張大戶更靈敏：

  • 千張大戶進場可能分批、時間拉長
  • 400 張大戶動作更即時，適合短線參考
  • 建議同時觀察 400 張和千張級距的變化

總股東人數 — 反向指標

  • 總股東人數下降 → 散戶退場、籌碼集中，偏多
  • 總股東人數急升 → 散戶湧入、籌碼分散，偏空
  • 「人少的地方才有機會」——人數減少代表籌碼往大戶集中

注意事項

1. 千張大戶不是絕對 — 大戶也可能看錯或只是調節持股，變動幅度大才需特別留意 2. 股價高低影響判斷 — 不同股票股價差異大，1,000 張台積電和 1,000 張低價股的金額天差地遠，應考慮持有金額而非僅看張數 3. 資料有遞延 — 週六公告的是上週四收盤後的資料，最長可能有 2 天延遲 4. 集保庫存不包含全部 — 質押、借券中的股票不計入集保庫存，可能低估大戶持股 5. 籌碼集中 ≠ 一定漲 — 大戶持股集中只是條件之一，仍需搭配基本面、技術面判斷
