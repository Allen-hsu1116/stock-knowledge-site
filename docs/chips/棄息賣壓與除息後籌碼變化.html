棄息賣壓與除息後籌碼變化

大戶為避稅在除息前賣出股票，形成棄息賣壓；除息後法人籌碼變化決定填息成功與否。

核心概念

棄息（棄權）：大股東或大戶為避免股利所得稅，在除息交易日之前把股票賣出，除息後再買回。這個動作產生的賣出壓力稱為「棄息賣壓」或「棄權賣壓」。

為什麼大戶要棄息？

  • 股利所得需繳稅（所得稅率可能高達20-40%）
  • 大戶持股量大，稅金驚人
  • 賣出再買回，股價差額（跌掉的股利部分）反而是資本損失可抵稅
  • 稅務考量下，棄息比參與除息更划算

除息後兩股力量對決： 1. 棄息賣壓：大戶賣出造成股價下跌壓力 2. 逢低買盤：除息後股價走低，吸引不參與除息的投資人進場 3. 這兩股力量交互作用，決定除息後股價走勢

實戰應用

棄息賣壓時間序列

1. 除息前5-10天：部分大戶開始提前減碼 2. 除息前1-3天：棄息賣壓最明顯，成交量放大 3. 除息日當天：股價跳空下跌（扣除股利），開盤價=參考價 4. 除息後1-3天：逢低買盤進場或法人持續賣超，決定方向 5. 除息後1個月內：觀察填息進度，法人籌碼動態是關鍵

法人除息後籌碼判讀

  • 外資除息後持續買超：偏多訊號，看好後市，填息機率高
  • 外資除息後轉賣超：偏空訊號，可能是棄息或看壞後市
  • 投信除息後賣超：短線獲利了結，需觀察是否持續
  • 自營商除息後買超：需區分自行買賣vs避險盤

操作策略

1. 觀察棄息賣壓vs逢低買盤強弱：除息前3天看成交量與法人動向 2. 大戶棄息後回補時機：若除息後大戶快速回補，代表只是避稅操作 3. 搭配法人連續買超：外資或投信連續買超+除息行情=雙重偏多訊號 4. 設定填息目標：除息前就設定填息多少後賣出，避免貪心

注意事項

  • 棄息賣壓不一定造成下跌，若逢低買盤夠強，股價反而上漲
  • 中小型股棄息賣壓影響較大，大型權值股影響較小
  • 外資的棄息行為需區分：真外資長線調節vs假外資隔日沖
  • 除息後1個月是關鍵觀察期，超過30天未填息要考慮停損
  • 稅制變化會影響棄息行為強度，注意政策動態
