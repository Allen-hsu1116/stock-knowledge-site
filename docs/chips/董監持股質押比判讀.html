董監持股質押比判讀

董監持股質押比判讀

董監持股比越高質押比越低，公司籌碼越安定

核心概念

董監事和三大法人是對公司最了解的內行人。如果公司真的有前景，這些人應該會想持有。透過觀察董監持股比和質押比，可以判斷籌碼安定度與公司風險。

計算公式

  • 董監持股比 = 董監事持股股數 / 總股數 × 100%
  • 董監持股質押比 = 董監持股質押張數 / 總股數 × 100%

判斷標準

  • 董監和法人持股比 > 33%：✅ 合格
  • 董監持股質押比 < 33%：✅ 合格

實戰應用

董監持股比判讀

持股比解讀
--------------
高且穩定經營者有信心，籌碼安定
逐年下降經營者逐漸淡出，需觀察原因
法定最低水位只為了符合法規，無額外承諾感

質押比判讀

質押比解讀
--------------
< 33%正常理財需求，不需過度擔心
33-50%需開始留意，觀察趨勢
> 50%高風險警訊，董監利益與股東脫節
> 70%極度危險，一旦股價大跌可能被迫斷頭

五種申報轉讓方式判讀

方式多空影響解讀
----------------------
贈與中性偏多可能是節稅規劃，對公司無不良影響
信託中性財產規劃，對公司無不良影響
盤後配對交易中性需觀察可能引進新股東或策略夥伴
指定人交易中性需觀察可能拉攏供應商或客戶
一般交易⚠️ 偏空大股東在市場上賣出，可能公司將轉壞

質押比高的三種情境

1. 大型投資計畫：如鴻海郭台銘質押買夏普、投資美國 → 有明確用途，不需擔心 2. 財務困難：大股東需要錢周轉 → 高風險，股價下跌可能引發斷頭賣壓 3. 槓桿過高：質押借錢再加碼股票 → 最危險，正反饋循環

注意事項

  • 質押比高不一定壞：要了解背後原因，可能是大型投資計畫
  • 持股比低不一定壞：前十大股東可能都是關係人，實質控制力仍強
  • 趨勢比絕對值重要：持股比逐年下降比單一年度低更值得擔心
  • 法人持股要看結構：外資佔大宗跟投信佔大宗意義不同
  • 搭配財報判斷：質押比高 + ROE 穩定（如鴻海）vs 質押比高 + 獲利下滑（如地雷股）
  • 留意質押異動：突然大幅增加質押比要特別警覺
