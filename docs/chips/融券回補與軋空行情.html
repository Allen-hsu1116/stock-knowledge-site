融券回補與軋空行情

融券必須在停止過戶日前6個營業日回補，越接近回補日越容易出現軋空行情

核心概念

融券回補是指融券放空的投資人，必須在規定期限前買回股票歸還券商。這個機制存在的原因是：融券的股票是向券商借的，當公司需要確認股東名冊時，借券人就必須還券。

融券強制回補的5大時機： 1. 股東常會 — 每年固定發生，須確認股東名冊（3~4月） 2. 除權除息 — 每年固定發生，配發股利須確認股權（5~7月高峰） 3. 現金增資 — 視個股而定，發行新股須確認股東權益 4. 減資 — 視個股而定，退還股本須確認股東 5. 合併/分割 — 視個股而定，公司結構變動須確認股東

重要例外：臨時股東會不需融券回補。

回補日計算：停止過戶日前6個營業日即為最後回補日。停券期間約3~7天，之後恢復融券交易。

台股特殊狀況：目前一年最多配息4次，加上股東會及除權，強制回補次數可能非常多。例如台積電一年就有5次融券強制回補。這類標的相對不適合融券放空。

實戰應用

1. 軋空行情判斷

越接近融券回補日，越容易出現軋空行情。邏輯：

  • 融券餘額高 → 大量投資人必須買回股票
  • 特定人士可能故意哄抬股價
  • 流動性差的股票更容易被軋空
  • 搶回補的恐慌心理推升股價

查詢工具：

  • 證交所融券餘額：https://www.twse.com.tw/zh/page/trading/exchange/MI_MARGN.html
  • 停券預告表：臺灣證券交易所 → 交易資訊 → 融資融券與可借券賣出額度

2. 低檔融券激增股篩選

篩選條件：融券增加 + 股價尚未過前高 + 基本面佳

  • 營收創新高、本益比低、基本面好的個股，融券激增後容易軋空
  • 例：雙鴻(3324) 營收成長40%、本益比<15，低檔融券激增後軋空

3. 融券回補後繼續放空的4種解法

方法優點缺點
------------------
先回補再重新放空簡單直接回補到重新放空間若大跌賺不到
股票期貨放空無回補問題不是每檔都有期貨
買進賣權(權證)無回補問題時間價值成本、Delta問題、流動性風險
找同類標的放空替代方案事件型機會不會發生在同類股

4. 操作建議

  • 不要壓最後一刻回補，提早買回避免軋空
  • 避開一年多次配息的股票放空，平均放空2個月就要回補
  • 營業員沒有義務提醒回補，自己注意最後回補日
  • 基本面好的個股不宜輕易放空，非基本面利空往往只是短線修正

注意事項

  • 軋空不一定發生：融券回補訊息會讓股價變高是經驗邏輯推論，並非適用所有股票
  • 回補行情可能是虛漲：若股價上漲速度顯著快於盈餘提升幅度，可能含軋空虛灌成分
  • 融券餘額高≠一定軋空：需搭配法人籌碼、基本面判斷
  • 流動性風險：交易量小的股票，回補時可能買不到好價位
  • 違約交割後果嚴重：超過回補日未還券，券商強制市價買回，損失慘重
