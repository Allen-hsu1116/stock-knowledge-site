Put Call Ratio 選擇權籌碼

選擇權賣權/買權比率（PCR），是判讀散戶多空情緒的反向指標，PCR越高代表散戶偏空反而看多，PCR越低代表散戶偏多反而看空。

核心概念

Put Call Ratio（PCR） 是選擇權市場的重要籌碼指標，分為兩種計算方式：

1. 成交量 PCR = Put 成交量 / Call 成交量 - PCR 越大 → Put 交易比 Call 活絡 → 市場偏空情緒濃厚 2. 未平倉量 PCR = Put 未平倉量 / Call 未平倉量 - 這是實戰中最常使用的版本 - 反向指標：因為選擇權賣方多為大戶/自營商

為什麼是反向指標？

  • 選擇權買方以散戶為主
  • 散戶習慣買 Put 做空保護、買 Call 做多投機
  • 大戶/自營商是賣方，收取權利金
  • 因此：
- PCR 越高 → 散戶大量買Put（偏空） → 大戶偏多 → 後市看多 - PCR 越低 → 散戶大量買Call（偏多） → 大戶偏空 → 後市看空

實戰應用

技巧1：搭配支撐壓力表

把每日選擇權未平倉口數和支撐壓力表的最大增量比對：

  • 最大未平倉量的 Call 履約價 → 壓力區
  • 最大未平倉量的 Put 履約價 → 支撐區

技巧2：看 Call/Put 籌碼內容組成

  • 價平附近大戶賣方多 → 區間盤整機率高
  • 價外極端大量 → 可能有極端行情
  • 區分誰在買（散戶）誰在賣（大戶）

技巧3：PCR 量化策略

  • Data1 放台指期，Data2 放未平倉 P/C ratio
  • PCR > 閾值 → 做多
  • PCR < 閾值 → 做空
  • 加上 K 棒濾網提高勝率

技巧4：結合其他籌碼指標

  • PCR 搭配外資期貨未平倉量 → 雙重確認
  • PCR 搭配三大法人現貨買賣超 → 多空共振

注意事項

1. PCR 是輔助指標，不能單獨做為進出場依據 2. 極端值才有參考意義，常態區間內訊號不明確 3. 結算日效應：接近結算日，未平倉量結構會大幅變動 4. 成交量 PCR vs 未平倉量 PCR：未平倉量版更具參考價值 5. 不同履約價的 PCR 意義不同：要區分價平、價內、價外

PCR 區間判讀參考

PCR 值散戶情緒逆向判讀
--------------------------
> 1.2極度偏空偏多
0.8-1.2中性中性
< 0.8偏多偏空

進階：看變化速率

PCR 的變化速率比絕對值更有參考價值。當 PCR 快速攀升，代表散戶恐慌加劇，可能是底部訊號。
