借券賣出餘額與外資隱形做空

借券賣出餘額是外資的「隱形空單」，當餘額持續增加時，外資偏空操作，後續常伴隨利空消息

核心概念

借券賣出（Security Lending Short Sale）是外資操控台股的第四項工具，不同於融券，借券賣出沒有平盤以下不能放空的限制，是法人專屬的做空管道。

借券 vs 融券差異

項目融券借券
------------------
借的對象券商持有股票的人（出借人）
誰能做散戶、法人法人為主
利率0.04%~0.08%出借人自訂（約3%）
平盤以下放空有限制無限制
強制回補股東會/除權息前需回補無強制回補時間
交易成本融券手續費+借券費借券利息
借券餘額 ≠ 借券賣出餘額：
  • 借券餘額：成功借出的股票總量（不一定全部賣出）
  • 借券賣出餘額：借券後實際賣出的部分（真正做空的數量）

實戰應用

三種借券賣出訊號

1. 借券賣出餘額持續增加 → 外資偏空操作，做多要謹慎 2. 借券賣出增加 + 現貨買超 → 外資多空都做（左手買進、右手偷布空單），表示不看好多方行情，被迫買進同時避險 3. 借券賣出增加 + 現貨轉賣超 → 雙手同向偏空，將出現大行情，做多應盡快撤退

找出外資狙擊目標

當外資開始全力偏空操作時，密切觀察借券賣出的標的：

  • 佈局一季以上的標的：外資要賣最兇的股票
  • 借券賣出金額前六大：超過20億的標的要多單避開
  • 外資手握第一手資訊，利空消息出來前就已經佈局完成

實際案例：宸鴻（3673）

2013年3月中過後，宸鴻借券賣出餘額持續暴增，到5月中累積15,000張。之後股價一路下跌超過30%。之後新聞「跌破眼鏡 宸鴻6月營收大減四成」，但借券賣出早已提前反應。

注意事項

1. 借券賣出是領先指標：外資的借券佈局往往在新聞出來之前就已完成，利空消息出來時外資空單已經獲利滿滿 2. 借券不等於做空：借券後可能用於避險、套利等用途，不完全是看空，需搭配現貨買賣超一起判斷 3. 外資可能多空都做：借券賣出增加但現貨也買超，表示外資在避險而非全面看空 4. 借券賣出減少是反轉訊號：當借券賣出餘額開始減少，表示外資開始回補空單，股價可能回升 5. 搭配其他指標判斷：單看借券賣出餘額不夠，需搭配外資現貨買賣超、期貨未平倉量等一起判讀
