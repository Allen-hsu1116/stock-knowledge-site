籌碼七指標選股法

籌碼七指標選股法

七個籌碼特徵同時出現，主力準備動了；指標愈多共振愈強，短線發動機率愈高

核心概念

籌碼是主力用錢買出來的，最不可能作假。一檔股票如果同時出現多個籌碼面的正面特徵，代表多方力量正在凝聚，短線發動的機率大幅提高。

七個指標從不同角度觀察籌碼流向： 1. 券商重押 → 單一券商急買程度 2. TOP 1買超異常 → 最大買方vs最大賣方力量對比 3. 籌碼集中度 → 前15大買方vs賣方的淨買超 4. 大戶買散戶賣 → 股權流向大戶還是散戶 5. 主力連買 → 主力連續進場天數 6. 外資連買 → 外資連續買超天數 7. 投信連買 → 投信連續買超天數

實戰應用

指標1：券商重押股

  • 條件：單一券商當日買張佔成交量 > 30%
  • 意義：有人極度看好後勢，可能是掌握未公開利多
  • 實戰：重押券商如果是外資總部（如港商麥格理），訊號更強

指標2：TOP 1券商買超異常

  • 條件：超額買超 = TOP 1買超 ÷ TOP 1賣超 > 4倍
  • 意義：最大買方力道遠超最大賣方，買方主導盤面
  • 實戰：超額買超8倍以上，3-5日內上漲機率極高

指標3：短線籌碼集中

  • 條件：1日籌碼集中度 > 20%
  • 計算：籌碼集中度 =（前15名買超量 - 前15名前15名賣超量）÷ 總成交量
  • 意義：市場前15大主力較看好後市，影響力足以推動波段
  • 實戰：籌碼集中度持續上升趨勢比單日高值更重要

指標4：大戶買散戶賣

  • 條件：大戶持股連3週成長 + 散戶持股連3週衰退
  • 意義：籌碼由散戶流向大戶，大戶逐步掌握籌碼主導權
  • 實戰：從集保股權分散表追蹤，大戶持股比上升是中長線偏多訊號

指標5：主力連買

  • 條件：主力連買 > 3天 + 買超 > 1000張
  • 意義：主力用大筆資金連續進場，可能掌握未知利多
  • 實戰：主力連買期間若買超創近日新高，發動後漲幅更驚人

指標6：外資連買

  • 條件：外資連買 > 3天 + 買超 > 1000張
  • 意義：外資波段操作，看對趨勢會繼續加碼
  • 外資3大習性：買大牌（權值股）、買優良股（基本面）、連續數十天一致動作
  • 實戰：外資剛開始連續大買時進場勝率最高

指標7：投信連買

  • 條件：投信連買 > 3天 + 買超 > 1000張
  • 意義：投信偏愛中小型股，追求績效，一旦進場會積極拉抬
  • 實實戰**：櫃買指數強於集中市場時，投信連買標的更有效

多指標共振策略

  • 3個指標同時出現 → 值得關注，列入觀察清單
  • 5個指標同時出現 → 短線發動機率高，可小部位試單
  • 7個指標同時出現 → 極稀有，大波段行情可能啟動

注意事項

  • 單一指標不構成訊號：任何一個指標單獨看都可能失準，必須多指標共振
  • 高集中度可能是隔日沖：券商重押+隔天賣出是隔日沖手法，要排除
  • 外資連買不等於一定漲：外資有避險部位，連買可能是被動配置
  • 投信作帳行情要小心：季底投信拉抬持股美化績效，季初可能反向賣出
  • 主力連買也可能是洗盤：連買幾天後突然反向賣出，製造籌碼鬆動假象
  • 籌碼數據是盤後才公布：進場時成本可能已經比主力高
  • 冷門股籌碼指標容易失真：成交量太低時，任何大單都會讓指標暴衝
