大外資與小外資判讀

大外資與小外資判讀

大外資看長線方向、小外資看短線靈敏度；扣掉五大特法就是小外資，短線變化更有參考價值。

核心概念

什麼是大外資與小外資？

台股的「外資」不是一個統一的主體，而是眾多海外法人機構的集合。這些法人可以依規模分為兩類：

大外資（五大特定法人）：

  • 資金部位超級大，交易週期長
  • 看法較長遠，部位笨重
  • 不會頻繁調整部位
  • 代表：摩根大通、高盛、美林等大型投資銀行

小外資（非五大特法）：

  • 資金部位較小
  • 短線調整靈活
  • 看法偏短線
  • 可能有假外資混入

為什麼要區分？

因為外資總量是大外資+小外資的加總，當兩者方向不同時，只看總量會被誤導：

  • 大外資偏多（長線佈局）+ 小外資偏空（短線避險）= 總量可能接近平衡
  • 大外資偏空（避險）+ 小外資偏多（短線搶反彈）= 總量可能接近平衡
  • 關鍵：大外資決定長線方向，小外資決定短線波動

小外資的計算方法

小外資 = 外資總量 − 五大特定法人總量

期交所每日公布三大法人期貨部位，其中「五大特定法人」的部位也會單獨列出。扣掉五大特法後，剩下的就是規模較小的外資機構。

實戰應用

一、大外資判讀重點

1. 看長線方向：大外資部位笨重，調整需要時間，一旦方向確定通常持續較久 2. 看未平倉變化趨勢：連續3-5天同方向增加，比單日大幅變動更有意義 3. 看現貨配合：大外資現貨連續買超 + 期貨多單 = 最強偏多訊號 4. 注意避險部位：大外資期貨空單可能只是避險，不代表看空

二、小外資判讀重點

1. 看短線靈敏度：小外資反應快，變化幅度大 2. 追蹤日變化：小外資每天調整，觀察短線方向 3. 搭配大外資：大小外資同方向 = 方向明確；方向相反 = 震盪整理 4. 注意假外資：小外資中可能有本土資金透過海外帳戶操作

三、大小外資交叉判讀

大外資小外資判讀
----------------------
偏多偏多強烈多頭訊號，趨勢明確
偏多偏空長多短震盪，拉回買進
偏空偏多長空短反彈，反彈賣出
偏空偏空強烈空頭訊號，趨勢明確
中性偏多短線反彈，觀察是否轉多
中性偏空短線修正，觀察是否轉空

四、跟單策略建議

跟大外資：

  • 適合波段操作、中長線投資
  • 觀察期貨未平倉趨勢，而非單日變化
  • 需要充足資金承受短期波動
  • 進場後要有耐心等待方向發酵

跟小外資：

  • 適合短線操作、當沖
  • 反應快，但要確認不是假外資
  • 進出迅速，停損要果斷
  • 不適合長線投資者參考

注意事項

  • 五大特法名單會變動：期交所的五大特法不是固定不變的
  • 假外資無法過濾：小外資中可能有本土資金混入
  • 摩台期是盲區：大外資可能同時操作摩台，台指期只看得到一部分
  • 群體非個體：外資總量是多個法人加總，不代表單一看法
  • 2011年後準確度下降：跟單效益約只有外資的一半
