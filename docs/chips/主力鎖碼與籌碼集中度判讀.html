主力鎖碼與籌碼集中度判讀

主力鎖碼與籌碼集中度判讀

籌碼集中在少數人手上，股價容易被拉抬；籌碼分散在多數人手上，股價容易陷入盤整。三指標共振（主力買超+家數差為負+集中度上升）是最佳進場訊號。

核心概念

什麼是籌碼集中度

籌碼集中度衡量一檔股票的持股是集中在少數人還是分散在多數人手上。計算方式為：一段時間內，買超前15名券商總張數減去賣超前15名券商總張數，再除以成交量。

直覺理解：

  • 籌碼集中 = 少數人掌控大量股票 → 有能力主導價格走勢
  • 籌碼分散 = 持股者眾多、想法分歧 → 容易區間盤整

觀察主力的三大指標

1. 主力買賣超：數值為正代表主力買進，連續數日正數代表持續看多布局 2. 買賣家數差：家數差 < 0 代表買家數 < 賣家數，籌碼往少數人集中（大戶進場）；家數差 > 0 代表籌碼分散 3. 籌碼集中度：短天期集中度 > 長天期集中度 → 近期主力正在買入

判斷籌碼集中的四個觀察點

  • 千張大戶持股比例持續上升 → 籌碼往大戶集中
  • 融資餘額下降但股價不跌 → 散戶在賣，有人在接（很可能是主力）
  • 成交量萎縮到極低水準、股價維持穩定 → 籌碼沉澱，浮額清洗乾淨
  • 券商分點出現連續性大量買超 → 特定主力正在吸貨

兩個以上訊號同時出現 → 值得列入觀察名單，等待突破訊號進場。

實戰應用

最佳進場訊號：三指標共振

當以下三個條件同時滿足，是教科書級的進場時機：

指標條件意義
------------------
主力買賣超連續數日正數主力持續看多
買賣家數差連續數日負數籌碼往少數人集中
籌碼集中度上升中確認主力確實看好

主力吸貨訊號組合

「法人買、主力買、散戶賣、量增價穩」= 主力吸貨的經典訊號：

  • 投信連續買超 → 法人看好基本面，開始建倉
  • 特定分點大量買超 → 主力同步進場
  • 融資餘額減少 → 散戶在賣，籌碼從散戶轉移到主力
  • 帶量小漲 → 主力壓著價格吃貨，不想讓股價過早曝光

主力集中價 vs 散戶解套價

  • 主力集中價：主力進場成本的平均價位。站上→主力獲利有動機推升；跌破→主力可能停損
  • 散戶解套價：散戶平均持股成本估算值。接近時形成解套賣壓
  • 關鍵判斷：不是怕解套賣壓，而是看「解套賣壓釋放後股價有沒有守住」。守住代表有人在下面接籌碼（通常是主力），通過測試的股票後面走得更穩

券商分點追蹤技巧

  • 特定主力或大戶通常固定在幾個分點下單
  • 常勝分點連續五天大量買超（>當日成交量3%）+ 股價未明顯拉升 = 典型主力吸貨
  • 搭配成交量和價格走勢一起看，準確度更高

注意事項

1. 漲幅已大不追高：籌碼集中的股票漲幅已大時，容易吸引追價者加入導致籌碼分散，不建議再追 2. 只看單日數據就下結論是大忌：至少觀察3-5天累積買賣超，單日法人買賣超可能只是調節或避險 3. 同樣的籌碼數據，不同量價結構解讀完全相反：主力買超+股價不漲=壓著吃貨（好事）；主力買超+股價大漲量大增=拉高出貨（壞事） 4. 主力會製造假象：A分點大量買、B分點同時賣（對敲洗盤）；或放利多引散戶進場再出貨 5. 籌碼分析不能當唯一依據：要搭配基本面，否則可能被主力短線操作誤導 6. 最適合波段操作：持股1週到3個月。當沖客有時效性問題（數據一天延遲），存股族看基本面更重要
