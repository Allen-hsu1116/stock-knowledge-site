選擇權支撐壓力表

選擇權支撐壓力表

從選擇權未平倉量找出市場的支撐壓力區

核心概念

選擇權支撐壓力表是根據市場上未平倉量（Open Interest, OI）整理而成的數據表，反映市場參與者的籌碼分布，幫助找到可能的壓力區與支撐區。

為什麼要站在賣方角度思考？

這是看支撐壓力表最重要的前提：

  • 買方留倉：最多只虧權利金，損失有限，不需要擔心未知風險
  • 賣方留倉：需要繳保證金，每看錯一點就多賠一點，沒有充足信心不會留倉

因此未平倉量聚集的履約價，代表賣方（通常是大戶/法人）認為指數不容易突破的價位。

支撐與壓力判讀

  • 支撐：賣權 Put 的最大未平倉量所在價位 → 賣方認為不會跌破
  • 壓力：買權 Call 的最大未平倉量所在價位 → 賣方認為不會漲破

實戰應用

1. 最大未平倉量 vs 變化量

  • 最大未平倉量：看絕對值最大的履約價，反映整體籌碼分布
  • 變化量最大：看當日增減最多的履約價，更貼近當下市場共識

建議兩者都要看，但變化量更能反映即時動態。

2. 支撐壓力區的實戰運用

  • 區間操作：在支撐壓力區間內來回操作
  • 突破策略：帶量突破壓力區 → 追多；跌破支撐區 → 追空
  • 選擇權策略：Iron Condor 的翼可設在最大未平倉量外側

3. 搭配其他籌碼指標

  • 搭配 Put/Call Ratio 判讀：支撐壓力區 + PCR 方向一致更可靠
  • 搭配 三大法人操作慣性：看外資期貨部位是否配合
  • 搭配 外資期貨空單：外資空單 + Call OI 大 → 壓力更強

注意事項

  • 不是碰到就不會破：支撐壓力是機率概念，不是絕對防線
  • 重大消息會打破區間：波動變大或法人做買方時可能衝出區間
  • 結算日效應：接近結算日，部位大量移轉，支撐壓力區會變動
  • 需搭配其他工具：單靠支撐壓力表容易過度簡化
  • 日內變動：盤中未平倉量會持續變化，建議看盤後確定數據
