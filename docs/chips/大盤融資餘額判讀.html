大盤融資餘額判讀

大盤融資餘額判讀

融資餘額是散戶情緒溫度計，高檔暴增是警訊，低檔清洗是底部訊號

核心概念

大盤融資餘額是整個台灣股市中，所有投資人透過融資（向券商借錢）買入股票後尚未償還的借款總金額。因為融資交易以散戶為主，大盤融資餘額常被視為「散戶指標」。

融資交易基本參數：

  • 上市股票融資成數最高60%，槓桿2.5倍
  • 融資年利率約6%
  • 信用戶資格：開戶滿3個月、近1年成交10筆以上

實戰應用

四種情境判讀法

情境指數位置融資變化操作對策
---------------------------------
多頭初期低檔反彈溫和增加健康多頭，持股續抱
多頭末期高檔高檔盤整/噴出急遽增加過熱警訊，降低持股槓桿
空頭初期高檔回落開始減少信心動搖，多殺多風險升高
空頭末期低檔盤整/破底持續減少至低水位籌碼沉澱，留意落底進場機會

三招實戰應用

1. 確認趨勢：指數上漲但融資不增反減 → 可能是少數權值股拉抬的虛胖行情，多頭基礎不穩 2. 警示風險：指數創高 + 融資歷史性暴增 + 「價漲量縮、資增」背離 → 提高警覺，適度降低持股或槓桿 3. 尋找底部：融資餘額大幅清洗後不再創低 + 低檔盤整 + 指數止跌訊號 → 可能是相對安全的進場時機

融資維持率是壓力測試

融資維持率 = (融資買入股值 / 融資金額) × 100%

  • 130%：追繳門檻（Margin Call）
  • 166%：補繳後必須回到的安全水位
  • T+2補繳，T+3斷頭強制賣出
  • 整體市場維持率逼近140%~150%時，斷頭賣壓積聚
  • 大盤重挫可能引發連鎖斷頭 → 多殺多惡性循環

注意事項

  • 融資餘額不是純反指標：多頭初期融資增加是健康現象，不是反指標
  • 融資餘額也不是純正指標：多頭末期融資暴增是警訊
  • 必須搭配指數位置判讀：同樣的融資增加，在高檔和低檔意義完全不同
  • 融資維持率比餘額更重要：維持率逼近警戒區才是真正的系統性風險
  • 個股融資與大盤融資不同：個股融資暴增可能是主力鎖單，不一定是散戶追高
  • 融資數據有落後性：T日交易，T+1才公布
