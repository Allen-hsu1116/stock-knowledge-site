融券強制回補與軋空行情

融券放空者必須在特定日期前買回股票還給券商，這個強制回補機制可能被利用來製造軋空行情，是籌碼面重要的觀察指標。

核心概念

融券回補是融券放空者買回股票還給券商的動作。法規規定在股東會、除權息、現金增資等事件前，融券必須強制回補，否則券商會以市價強制買回（可能造成嚴重虧損）。

四大強制回補時機

1. 除權 — 在外流通股數變動 2. 除息 — 需確認股東身分配息 3. 股東會 — 需確認股東名冊（3-4月最常見） 4. 現金增資 — 不定期

注意：臨時股東會不需要融券回補。

日期計算

  • 最後回補日 = 停止過戶日前6個營業日（含）
  • 停券時間約3-7天
  • 公司需在股東會前2-3個月公告停止過戶日程

範例：停止過戶日7/21 → 最後回補日7/15

一年可能多次回補

台股一年最多配息4次，加上股東會和除權，一年可能5次以上的融券強制回補。頻繁配息的標的不適合長期融券放空。

實戰應用

1. 軋空行情判斷

軋空行情發生條件：

  • 融券餘額高：回補日前融券餘額仍多
  • 流動性差：交易量小，回補買不到
  • 有心人士哄抬：知道融券必須回補，故意推升股價

判斷方法：

  • 查詢證交所融券餘額數字
  • 越接近最後回補日，軋空壓力越大
  • 融券餘額佔流通量比例越高，軋空力道越強

2. 融券回補行情的操作

做多策略：

  • 在融券回補日前1-2週買入高融券餘額個股
  • 等待融券回補推升股價
  • 回補日後留意結帳賣壓

放空者自保策略： 1. 提前回補，不要壓最後一刻 2. 改用股票期貨放空（無回補問題） 3. 買進賣權（Put）替代融券 4. 找類似標的放空

3. 查詢融券回補日

  • 臺灣證券交易所 → 交易資訊 → 融資融券與可借券賣出額度 → 停券預告表
  • 營業員沒有義務提醒回補，投資人需自行注意

注意事項

1. 強制回補是公開資訊，主力可能利用此機制故意軋空 2. 流動性差的股票回補風險更高，可能被逼在很高價位買回 3. 多次配息標的不適合長期融券放空，平均2個月就要回補一次 4. 融券維持率不足也會被強制回補，不只在特定日期 5. 借券 vs 融券差異：借券沒有平盤下放空限制，但有借券費用
