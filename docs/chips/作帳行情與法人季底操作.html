作帳行情與法人季底操作

作帳行情與法人季底操作

投信季底「作帳」拉抬股價、自營商季底「結帳」賣出獲利，兩者操作方向完全相反。

核心概念

投信作帳行情

投信基金經理人為了績效評比排名，在季底/年底拉抬持股股價，增加帳面上未實現獲利，使基金淨值上漲。投信不需要賣出股票，只要帳面上股價在漲就好。

自營商結帳行情

自營商操盤人的績效獎金建立在已實現獲利上。老闆不會用未實現數字發獎金，因此自營商季底前會將獲利了結，逢高調節。

兩者差異

投信作帳自營商結帳
---------
目的帳面未實現獲利最大化實現獲利落袋為安
手法買進推升股價賣出獲利了結
對股價影響上漲下跌壓力
時間季底前拉抬季底前調節
獎金基礎績效排名（未實現）實現損益（已實現）
報酬類型相對報酬（贏過大盤）絕對報酬（一定要賺）

集團作帳行情

集團公司透過旗下公司交叉持股或在媒體釋出利多消息拉抬股價。常見集團：華新、國巨、聯電。

實戰應用

搭乘投信作帳行情三步驟

步驟一：選擇績效好的基金

  • 同類型排行前幾名才有拉抬動機
  • 使用 MoneyDJ 理財網查詢基金排行
  • 績效差的基金沒有拉抬必要

步驟二：選擇「我有你沒有」的股票

  • 台積電、鴻海等大型股各家都有，拉抬意願低
  • 找該基金獨有、競爭對手沒有的持股
  • 中小型股是投信主要操作標的

步驟三：剛起步買進

  • 5-6 月開始篩選布局最安全
  • 10 月仍可找「王牌股」，投信最後衝刺用
  • 篩選條件：投信連續買超 + 持股比例不高 + 漲幅不超過 30%

搭乘自營商行情的節奏

1. 每季第一個月（1、4、7、10月）：觀察自營商大買的個股 2. 每季第二個月：股價順利拉了一波後，不要以為會繼續推高 3. 每季最後一個月（3、6、9、12月）：準備下車，自營商逢高調節

投信假單辨識

  • 看多假單：外盤整數價位掛大單 → 吸收籌碼 → 撤單後股價飆漲
  • 看空假單：內盤掛大單假裝支撐 → 暗中倒貨 → 股價下跌

投信持股比例警訊

  • 總投信持股超過 15%：留意是否有出貨跡象
  • 出貨完畢後持股比例急降，股價往往大跌
  • 單一投信持股通常不超過 10%（法規限制）

注意事項

  • 作帳之後必有結帳潮：搭順風車同時要注意下車時機
  • 自營商喜好：股性活潑的中小型股，常出現隔日沖
  • 股價最終回歸基本面：作帳行情只能短期帶動，基本面不夠強會被打回原形
  • 投信持股資訊是落後消息：王牌持股只能猜測，風險較高
  • 集團作帳更難捉摸：缺乏像投信持股比例的公開資訊
