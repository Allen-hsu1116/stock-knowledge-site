K線型態

K線基礎

每根K線由四個價格組成：開盤價（Open）、最高價（High）、最低價（Low）、收盤價（Close）。

解讀三要素：

  • 顏色：紅K（上漲）vs 黑K（下跌）
  • 實體長度：趨勢強度（長實體 = 趨勢明顯）
  • 影線長度：多空交戰激烈程度（長影線 = 阻力大）

單根K線型態

型態定義市場訊號
**大紅K**收盤遠高於開盤，實體長，影線<20%多頭強勢，後市看漲
**大黑K**收盤遠低於開盤，實體長，影線<20%空頭強勢，後市看跌
**十字線**開盤≈收盤，長上下影線多空勢均力敵
**鎚子線**短實體+長下影線（無上影線）下檔有支撐
**倒鎚線/墓碑線**短實體+長上影線（無下影線）上檔有壓力
**紡錘線**短實體+上下影線市場猶豫

看漲反轉型態

型態組成訊號
**錘形線**單根K線，短實體長下影線，位於下降趨勢底部拋售壓力後買壓推升，看漲
**多頭吞噬**兩根K線，短紅K被長綠K完全吞沒買家明顯獲勝，看漲
**早晨之星**三根K線：長紅-短K-長綠賣壓消退，看漲
**白色三兵**三根連續長紅K，開盤收盤均高於前日買壓穩步上升，強烈看漲

看跌反轉型態

型態組成訊號
**上吊線**單根K線，短實體長下影線，位於上升趨勢頂部拋售明顯，多頭失控，看跌
**射擊之星**單根K線，短實體長上影線，位於上升趨勢頂部流星墜地，看跌
**空頭吞噬**兩根K線，短綠K被長紅K完全吞沒價格高峰，看跌
**黃昏之星**三根K線：長綠-短K-長紅上升趨勢反轉，看跌
**三隻烏鴉**三根連續長紅K，影線短或無賣方力量連續超越買方，看跌

整理型態

型態定義市場訊號
**十字星**開盤≈收盤，短實體長影線多空掙扎，中性訊號
**紡錘線**短實體，等長上下影線市場猶豫，盤整期
**下降三法**長紅K + 三根短綠K + 長紅K趨勢延續，看跌
**上升三法**長綠K + 三根短紅K + 長綠K趨勢延續，看漲

實戰心法

  • 單根K線不構成訊號：需搭配前後K線一起判斷
  • 影線透露過程，實體透露結果：長影線 = 激烈交戰，長實體 = 明確趨勢
  • 長實體+短影線：趨勢延續性高
  • 短實體+長影線：趨勢延續性低
  • 型態位置很重要：同樣型態在高位和低位意義不同

均線判斷

均線種類

種類計算方式特點
**SMA（簡單移動平均）**收盤價算術平均權重相等，最常用
**EMA（指數移動平均）**近期價格權重較高對價格波動敏感

常用均線週期

週期名稱用途
**5MA**周線短線/當沖
**10MA**雙周線短線
**20MA**月線波段（最重要）
**60MA**季線中期
**120MA**半年線中長期
**240MA**年線長期

均線三大用法

1. 價格突破均線

  • 收盤價 > 10MA → 過去10天進場者賺錢
  • 收盤價 < 20MA → 過去20天進場者賠錢

2. 黃金交叉與死亡交叉

交叉類型定義訊號
**黃金交叉**短均線向上突破長均線看漲，多單進場訊號
**死亡交叉**短均線向下跌破長均線看跌，空單進場訊號

3. 多頭排列與空頭排列

排列型態定義訊號
**多頭排列**短均線在上，長均線在下（5>10>20>60）趨勢強勁上升
**空頭排列**短均線在下，長均線在上（5<10<20<60）趨勢疲弱下跌

均線使用注意

  • 盤整時頻繁交叉：會被來回洗刷，造成虧損
  • 均線是落後指標：反應慢半拍，短線精確度有限
  • 均線是市場成本線：價格在均線上代表過去一段時間買者賺錢
  • 動態支撐壓力：上升趨勢中均線作為支撐，下降趨勢中均線作為壓力

支撐壓力

定義

類型定義特性
**支撐線**價格停止下跌並反彈的位置買方多於賣方，價格上升
**壓力線**價格停止上漲並下跌的位置賣方多於買方，價格下跌

如何畫線

類型畫法
**支撐線**連接至少 2 波低點，向右延伸
**壓力線**連接至少 2 波高點，向右延伸

強度判斷

  • 觸及次數越多：支撐/壓力越可靠
  • 時間越久：心理認同越強
  • 成交量越大：該位置越重要

角色互換

  • 支撐線被跌破 → 轉變為壓力線
  • 壓力線被突破 → 轉變為支撐線

交易策略

情境支撐線操作壓力線操作
價格觸及考慮買進考慮賣出
價格突破賣出或觀望準備作多
價格回測買進機會賣出訊號

K棒判斷支撐壓力

1. 觀察上下影線：影線尖端常是支撐/壓力位置
2. 觀察實體密集區：大量K棒實體疊加的區間
3. 整數關卡：如 100 元、1000 點等心理關卡

實戰心法

  • 假突破要防範：突破需要確認（收盤站上、成交量放大）
  • 越多觸及越可靠：2-3 次以上測試才確認有效
  • 時間框架要匹配：日線支撐壓力比分線更重要

財報假帳偵測

假帳五大動機：

動機說明
順利上市上櫃為了符合上市櫃標準，美化財報
掩飾本業衰退怕財報難看導致股價下跌或銀行抽銀根
避免大股東破產大股東股票質押成數高（>60%），股價不能跌
操縱股價先低價買進，再假帳拉抬股價，最後高賣
掏空公司把公司資產搬進自己口袋

財報假帳六大特徵：

特徵判斷標準原理
應收帳款天數過高> 90天假交易利潤藏在應收帳款銷不掉
存貨天數過低< 10天且異於同業虛增營收但存貨沒增加
不動產過高週轉率低於同業應收帳款轉入不動產消化
長期投資過高未上市櫃公司投資多難驗證的資產藏假
現金過高現金高但借款也高現金可能被限制（博達案）
雜項資產過高其他應收款、預付款高雜項科目藏假

假帳非財務警訊：

警訊判斷方式
老闆誠信被質疑質押成數 > 80% = 很缺錢
經營階層變動財務長、獨董、董事接連離職
股價異常飆漲漲幅遠超同業且無基本面支撐
利多消息不斷頻繁發布利多、澄清「不予置評」太多
借殼後初期借殼 3-4 年內營收暴增但現金流為負
產業差獲利好技術普通、產業景氣差卻賺錢

假帳偵測心法：
1. 先看老闆質押：> 80% = 小心
2. 再看應收帳款天數：> 90天 = 警訊
3. 看存貨天數：< 10天且與同業差距大 = 警訊
4. 看現金與借款：雙高 = 現金可能被限制
5. 看營業現金流：營收成長但現金流為負 = 怪
6. 看利多頻率：澄清「不予置評」太多 = 放消息炒股
7. 看借殼歷史：借殼後 3-4 年內營收暴增 = 警訊
8. 研究產業：產業差卻賺錢 = 可能有假

企業掏空識別

掏空定義：公司大股東或管理層以不合法手段轉移上市公司資源，損害中小股東利益。

掏空手法分級：

級別特點常見手法
初級班簡單關係人交易訂單過水、指定進貨、高價買資產、代養私人公司、庫藏股護盤
中級班牽涉多家公司/海外虛增營收、切割金雞母、金融工具套利、海外公司隱匿
高級班博達案等級CLN 虛增存款、應收帳款融資、海外可轉債套利、擔保借款

博達案四大手法：
1. 信用連結債券（CLN）虛增銀行存款
2. 應收帳款融資 + 零息債券
3. 海外可轉債套利
4. 為海外人頭公司擔保借款

掏空防範警訊：

警訊觀察重點
大股東一股獨大董事會、股東會形同虛設
內部監管不健全獨立董監無法發揮監督功能
關係人交易異常交易條件優厚、租金高於行情、應收票據未如期兌現

關係人交易查核重點：
1. 識別關係人：財報附註的關係人揭露、實質關係人（姻親、同一辦公處所）
2. 評估交易條件：比較與公平交易的條款、收付款期間是否異常
3. 查核佐證資料：會議紀錄、契約文件、應收帳款帳齡分析

會計師查核報告判讀

核閱 vs 查核：

項目核閱（季報）查核（年報）
盤點❌✅ 年底親自盤點
銀行函證❌✅ 直接向銀行確認
律師函證❌✅ 詢問訴訟官司
關係人函證❌✅ 確認應收帳款
可信度較低最高

財報申報期限：

季度一般公司金融保險業
Q15/15（核閱）5/30（核閱）
Q28/14（核閱）8/14（查核）
Q311/14（核閱）11/29（核閱）
Q43/31（查核）3/31（查核）

四種會計師查核意見：

意見類型判斷行動
無保留意見財報沒問題✅ 正常投資
保留意見某方面有疑慮⚠️ Q4 出現 = 小心
否定意見財報整體不可信🚫 快跑
無法表示意見查不到🚫 快跑

實戰要點：

  • Q4 年報最精確，投資決策以年報資訊為主
  • Q4 出現保留意見 = 大問題（季報保留意見常見，年報少見）
  • 否定意見、無法表示意見 = 直接跳過
  • 收到財報第一件事：翻到會計師查核報告

管理階層品質評估

好 CEO 八大特質：

特質內容判斷方式
效率決定生死去中心化管理、充分授權總部官僚程度
聚焦資本配置每塊錢都要明確報酬併購績效、買回紀錄

高效運用槓桿知道何時用、如何用槓桿買穩定現金流資產
勇於逆向前行危機時買進、狂熱時冷靜CEO 在危機時的反應

永遠關注現金流預留至少 3 年收入中斷的現金自由現金流穩定性
CEO = CIO每個決策都是投資決策是否喊口號還是把自由現金流最大化

偽 CEO 特質（避雷）：

警訊表現
頻繁談願景法說會都在畫大餅，很少談報酬
併購紀錄不亮眼併購多但商譽減損多

1. 查 CEO 背景（Google 負面新聞、過往績效）
2. 查薪酬結構（固定 vs 績效 vs 選擇權）
3. 查持股變動（持續賣出 = 不看好）
4. 查併購績效（創造價值還是毀滅價值）
5. 查買回時機（低買還是高買）
6. 查媒體曝光（太多願景 = 缺乏紀律）

籌碼面分析

三大法人特性與操作

法人資金規模操作偏好參考價值實戰要點
**外資**40-60%成交大型權值股、中長期高（續買率74%）創新高時站在買方，從買轉賣/從賣轉買是重要轉折
**投信**~5%成交中小型股、波段中（留意季底作帳）左側交易者，下跌時買進、高檔時賣出
**自營商**1-2%成交短線、避險低（進出快）高檔時幾乎都是賣超，買ETF為主

外資連續買超判斷：

  • 建議觀察 3-5 天確認趨勢
  • 買賣超金額至少要達「億」級別才有參考價值
  • 選擇大型股（外資偏好權值股）
  • 注意「假外資」：操作期間短、偏好中小型股、隔日沖

最佳進場點：外資「連賣轉買」的轉折點，但不要第一天就進場，觀察 5 天確認

法人同步買超訊號：

同步情況訊號強度意義
三大法人同步買超最強市場高度共識
外資+投信同步買超強中長期看好
僅外資買超中需觀察連續性
僅自營商買超弱可能短線操作

融資融券基礎

融資（向券商借錢買股）：

  • 自備 40%（上市）/ 50%（上櫃）
  • 槓桿約 2.5 倍
  • 年利率 6-7%
  • 維持率 130% 是生死線

融券（向券商借股票放空）：

  • 準備 90% 保證金
  • 借券費 0.8‰
  • 強制回補日：股東會、除權息前
  • 軋空風險：股價上漲無限，虧損無天花板

融資維持率計算：

維持率 = 股票市值 ÷ 融資借款
斷頭線 = 借款 × 1.3
下跌幅度 = (買進價 - 斷頭價) ÷ 買進價

實例：融資買進 100 元股票，借款 60 元，斷頭價 78 元，下跌 22% 就斷頭

券資比進階應用

定義：券資比 = 融券餘額 ÷ 融資餘額

判斷標準：

券資比狀態風險
< 10%正常低
10-30%需注意中
> 30%高危險區軋空風險大增

軋空行情識別：
1. 股價上漲 + 融券增加 = 散戶放空、主力拉抬
2. 券資比突然提高 = 關注，可能是飆股前兆
3. 價量齊揚 + 融券上升 = 軋空行情

融資融券搭配主力買賣超：

  • 低檔融資暴增 + 主力買超 = 主力進場
  • 高檔融券暴增 + 主力賣超 = 主力看空

千張大戶持股分析

定義：持有某股票超過 1,000 張的大戶

查詢管道：
1. 集保中心官網（每週六早上 9:00 更新）
2. 神秘金字塔網站
3. 籌碼 K 線 APP

判斷標準：

比例意義
< 40%籌碼太分散，不易漲
40-70%最佳區間，易漲難跌
> 80%太集中，炒作空間小

實戰技巧：

  • 大戶持股增加 + 散戶減少 = 主力吸貨
  • 大戶持股減少 + 散戶增加 = 主力出貨
  • 連續 3 週同方向變動才值得注意
  • 變動幅度 > 2% 才有意義

外資期貨與選擇權未平倉

大外資 vs 小外資：

類型特性操作意義
大外資部位大，無法短線進出看中期趨勢（1-3個月）
小外資部位小，操作靈活看短線變化（1-2週）

期貨未平倉實戰判讀：

  • 淨多單 > 3 萬口 → 多方力道強
  • 淨空單 > 3 萬口 → 空方力道強
  • 變化 > 5,000口 → 有意義的加減碼

選擇權未平倉判讀：

項目解讀
Call OI 增加賣方看「不漲」，該履約價為壓力區
Put OI 增加賣方看「不跌」，該履約價為支撐區
P/C Ratio > 1莊家偏多（認為有支撐）
P/C Ratio < 1莊家偏空（認為有壓力）

庫藏股判讀

公司買回庫藏股的三大官方理由：
1. 轉讓股份給員工 — 可能偏多
2. 作為特別股或公司債轉換來源 — 中性
3. 維護公司信用及股東權益 — 可能只是護盤

關鍵判斷：

轉讓方式股價影響解讀
贈與/信託中性偏多可能是低檔布局、節稅規劃
盤後配對/鉅額中性偏多不影響市場賣壓
指定人交易中性特定人承接
一般交易利空直接市場拋售，需警戒

執行率是關鍵：

  • 執行率 < 50% = 可能是「口水護盤」

董監改選行情

定義：公司派和市場派在股東會前搶奪籌碼，導致股價易漲難跌的現象。公司法規定董監事任期最長3年，每3年改選一次。

為什麼會漲：

  • 公司派：怕經營權被搶走，持股不賣甚至回補
  • 市場派：想取得經營權，從市場不斷買進
  • 籌碼被鎖住只進不出，供需失衡，股價自然上漲

董監改選行情三階段：

階段時間特徵散戶策略
默默吃貨期改選前一年Q4甚至更早大股東默默吃貨，股價不明顯上漲太早介入，時間成本高
蜜月期每年12月到隔年3月（行情高峰）雙方搶籌碼最激烈，股價漲勢凌厲**最佳搭轉時機**
回歸基本面4月停止過戶後改選落幕，敗選方可能釋出籌碼提防股價大幅修正

實例：

  • 國票金(2889)：2010年10月起漲，2011年2月從10元漲到17.6元，漲幅80%
  • 三陽(2206)：2014年改選，累積漲幅超過100%，但經營權底定後跌回起漲點

容易出現董監改選行情的四大條件：

條件判斷標準原理
董監持股偏低<20%（容易引起覬覦），<10%（高風險）公司派防衛門檻太低
董監持股質押比率過高>50%現金不足，容易被市場派鎖定
股價偏低且具潛在價值股價淨值比<1，有資產、現金部位高取得經營權成本低
小型股股本<20萬張，非集團、非家族企業介入成本低，籌碼易操控

  • 經營權之爭結束後，敗選方會賣股票，股價可能快速回檔
  • 這是「事件交易」，事件結束就要跑

當沖（Day Trading）

定義: 當天買進當天賣出，不留倉過夜

申請條件:

  • 年滿 20 歲
  • 開戶滿 3 個月
  • 近 1 年成交 10 筆以上

優勢:

  • 時間短，不用長時間等待結果
  • 門檻低，不用負擔股票本金（只需手續費 + 證交稅）
  • 降低心理負擔，避免收盤後負面新聞影響
  • 不用耗時研究公司基本面
  • 證交稅減半：0.3% → 0.15%

風險:

  • 交易頻繁，手續費成本高
  • 需要盯盤，上班族不方便
  • 高槓桿（約 290 倍），可能誤把投資當賭場
  • 需有波動幅度的股票才划算

適合對象: 不喜歡研究公司、想要快速知道結果、有時間盯盤的人

紀律: 一天操作次數不用多，有賺就好，維持穩定紀律

隔日沖

定義: 今天買進，隔天賣出

特點:

  • 風險限制在兩天內
  • 適合盤中有波動的商品
  • 波動幅度要高於交易成本才划算

波段操作

定義: 股價低檔買進、高檔賣出，持有幾天到數週

進出場判斷:

  • 支撐位買進、阻力位賣出
  • 搭配技術分析形態學
  • 輔以基本分析、布林通道等指標

優勢:

  • 摩擦成本比當沖少
  • 交易壓力比當沖低
  • 成功率較高，是新手適合的入門策略

缺點:

  • 沒有固定規則，高低點判斷眾說紛紜
  • 賺錢的時間成本較大

適合對象: 新手菜鳥想從交易入手

買進長期持有（價值投資）

定義: 長線策略，不理會短期波動，持有長期一定上漲的資產

流程: 基本分析 → 估值 → 買進 → 長期持有 → 獲利賣出

優勢:

  • 風險小、勝率高（前提是分析能力足夠）
  • 有機會創造巨大報酬

缺點:

  • 基本分析時間成本很大
  • 需要估值能力和財報解讀能力

適合對象: 有分析能力、能接受慢慢致富的人

指數型投資（被動投資）

定義: 投資指數 ETF（如 0050、標普 500），不需要選擇單一公司

優勢:

  • 不需要基本分析能力
  • 不用判斷買賣時機
  • 透過複利效應取得不錯報酬
  • 巴菲特建議普通人投資指數型基金

缺點:

  • 報酬率較中庸

適合對象: 分析能力不好、沒時間分析的人

資金配置建議

新手建議:

  • 大部分資金：指數型投資（勝率高、風險低）
  • 少量資金：波段交易（判斷自己適不適合投機）
  • 有經驗後再嘗試當沖或集中選股

投機心法:

  • 80/20 法則：市場上 80% 的錢被 20% 的人贏走
  • 永遠要注意停損停利控制風險
  • 投機很看天賦和運氣，少不了大量練習

實戰操作步驟

進場前規劃：
1. 確定進場價格
2. 計算停損價（進場價 × 0.95 或 K 線低點）
3. 計算停利價（進場價 × 1.2 或前高壓力）
4. 決定倉位大小（單筆風險不超過總資金 2%）

進場後執行：
1. 立即設定停損單（條件單或手動監控）
2. 股價上漲至最高點後，啟動移動停利（回檔 10% 出場）
3. 收盤跌破停損價 → 次日立即出場，不抱幻想

注意事項：

  • 停損點不得更改，否則等於沒設
  • 順勢交易為主，高勝率位置進場
  • 收盤跌破停損價要立即出場，要有壯士斷腕決心

風險管理

停損方法（7 種）

方法操作方式優點缺點適用場景
**K 線停損法**以進場日 K 線最低點為停損簡單明確盤整易觸發短線、當沖
**趨勢停損法**停損設在轉折波低點順勢操作需識別轉折趨勢明確行情
**均線停損法**以進場均線價為停損客觀標準震盪易被洗均線操作者
**固定比例停損法**買進價 × (1 - 停損%)最簡單難適應波動新手
**支阻停損法**跌破關鍵支撐停損符合技術邏輯需判斷關鍵位技術分析者
**移動停損法（N K 棒）**最近 N 根 K 棒最低點兼具停損停利需持續監控波段操作
**ATR 動態停損法**進場價 ± N × ATR反映當下波動計算較複雜高波動標的

絕對停損原則（5 種必須停損）：
1. 跌破盤整區 → 絕對停損
2. 趨勢反轉確認 → 絕對停損
3. 虧損超過 10% → 絕對停損
4. 逆勢單反轉 → 立刻停損
5. 關鍵支撐跌破 → 絕對停損

停損核心原則：

  • 華爾街標準：單筆不超過 10%
  • 一般建議：固定 5% 停損
  • 進場前就設好，不得更改
  • 收盤跌破停損價 → 次日開盤立即出場

停利方法（4 種）

方法操作方式適用場景
**固定百分比停利**達到目標百分比出場短線（10-20%）
**移動停利（回檔比例）**從最高點回檔 X% 出場波段、趨勢
**壓力位停利**到達技術壓力位觀察轉折短線、前高附近
**乖離率停利**乖離率超過閾值分批出場波段、防止過熱

停利建議：

  • 短線：10-20% 固定停利或回檔 5-10%
  • 中線：30-50% 或乖離率 15-25%
  • 移動停利：從最高點回檔 10% 出場

倉位管理（5 種方法）

1. 固定百分比法

倉位大小 = 總資金 × 風險比例 / (進場價 - 停損價)
  • 單筆風險：1-2% 總資金
  • 適用：長期穩健增長型交易者

2. ATR 波動率法

倉位大小 = (帳戶資金 × 風險%) / (ATR × 停損倍數 × 每點價值)
  • 高波動自動縮小倉位
  • 適用：追求與市場波動同步

3. 凱利公式（Kelly Criterion）

f* = (p × b - q) / b
  • f* = 最佳下注比例，p = 勝率，q = 敗率，b = 盈虧比
  • 實務建議：使用「半凱利」更安全
  • 適用：勝率與盈虧比穩定的策略

4. 金字塔加碼法

  • 趨勢確認後初始倉位
  • 獲利後回調時加碼
  • 每次加碼金額遞減
  • 適用：趨勢跟隨策略

5. 固定金額法

  • 每筆交易固定風險金額
  • 計算簡單但帳戶波動後不靈活
  • 適用：小型帳戶

避險策略

核心觀念

  • 避險 ≠ 消除風險，是將傷害控制在可接受範圍
  • 避險有成本（交易成本 + 機會成本）
  • 事前避險成本遠低於事後避險

事前 vs 事後避險

類型特點成本
事前避險根據風險預判，事先配置對抗資產低
事後避險意外發生後才減碼或調整高（易追高殺低）

何時該避險

  • 進場理由「可能改變」→ 考慮避險
  • 進場理由「沒變」→ 不要因恐慌打破紀律

何時不該避險

  • 只因價格下跌或恐慌 → 不該避險
  • 常常想避險 → 策略本身有問題

對沖交易策略

對沖定義：建立與現貨相反的部位，降低整體風險

三種對沖類型

類型說明適用情境
完全對沖現貨 + 等量反向期貨消除所有價格風險
部分對沖只對沖部分部位保留部分上漲空間
動態對沖根據市況調整比例需持續監控

期貨對沖實例

  • 持有台積電 100 張 + 放空台積電期貨 100 張
  • 股價跌 5%：現貨虧 5%，期貨賺 5% → 損益兩平
  • 股價漲 5%：現貨賺 5%，期貨虧 5% → 損益兩平

期權對沖策略

策略操作效果
保護性認沽持股 + 買 Put鎖定最大虧損，保留上漲空間
遮蓋性買權持股 + 賣 Call收權利金降成本，限制上漲獲利
領口策略持股 + 買 Put + 賣 Call鎖定上下價格區間

對沖注意事項

  • 基差風險：期貨與現貨價差變動
  • 保證金追繳：期貨虧損需補錢
  • 流動性風險：小標的可能流動性不足

  • 虧損：18 × $100 = $1,800
  • 淨獲利：$600（月報酬率 12%）

VaR 與 CVaR 風險量化

VaR（風險價值）

項目說明
**定義**特定期間、特定信賴水準下的最大預期損失
**公式**VaR = Z × σ × √T × V（Z=信賴水準z值、σ=波動度、T=天數、V=部位價值）
**三種計算方法**變異數-共變異數法、歷史模擬法、蒙地卡羅模擬法
**限制**不考慮尾部損失、不滿足次可加性

CVaR（條件風險價值）

項目說明
**定義**超過 VaR 後的平均損失（期望尾端損失）
**公式**CVaR = E[Loss \Loss > VaR]
**優勢**考慮尾端損失嚴重度、滿足一致性風險測度

VaR vs CVaR 比較

特性VaRCVaR
關注點閾值損失尾端平均損失
次可加性❌ 不滿足✅ 滿足
尾部資訊不考量充分考量
適合用途日內監控、風險限額資本配置、壓力測試

投資組合 VaR 計算

組合 VaR = √(Σ DEAR_i² + 2×Σ(ρ_ij × DEAR_i × DEAR_j))
DEAR = |-α| × σ × 價值（99% 信賴水準 α = 2.33）

實務要點：

  • 台股常有肥尾現象，變異數-共變異數法可能低估風險
  • VaR 搭配 CVaR、壓力測試使用
  • 定期回測驗證模型準確度

策略核心邏輯適用時機
**價格動能**過去漲的還會漲明顯多頭市場

  • 選錯策略就是虧損

金字塔加碼策略

正金字塔 vs 倒金字塔：

方法操作適用時機風險
正金字塔越跌買越多長期趨勢向上可能「越買越深」
倒金字塔越漲買越多多頭行情確認成本較高、假突破風險
等注加碼每次固定金額最穩健獲利比正金字塔少

核心心法：
1. 只在賺錢時加碼（市場證明你是對的）
2. 突破關鍵點再加碼
3. 設好停損保命
4. 只賠最後一碼（加碼後出場價設在前一碼上方）

阮慕驊SOP：底部30% → 漲10%加20% → 再漲10%加10%

箱型理論（Darvas Box）

核心概念: 股價在「箱型區間」內整理震盪，突破箱頂代表新趨勢啟動。

操作方法：

  • 順勢：突破箱型上緣 → 進場，停損設箱型下緣或上緣下一檔
  • 逆勢：箱型下緣買進、上緣賣出（不建議一般投資人）

假突破判斷：

  • 突破但成交量沒放大 → 可能假突破
  • K線收盤價沒真正突破 → 可能假突破

關鍵體悟: 勝率只有30-40%，但盈虧比好。不要預設「合理」獲利就停利，讓箱型告訴你什麼時候出場。

葛蘭碧八大法則

四大買點：

買點定義適用時機
**B1 突破買點**均線走平/上彎，價格突破均線趨勢反轉初期
**B2 支撐買點**均線上揚，價格拉回測試均線反彈多頭趨勢中回檔（最常用）
**B3 假跌破買點**均線上揚，價格短暫跌破後快速收回主力洗盤時
**B4 反彈買點**價格急跌造成負乖離過大空頭急跌搶反彈

四大賣點：

  • S1 跌破賣點：均線下彎，價格跌破均線
  • S2 壓力賣點：均線下彎，價格反彈到均線無法突破（最重要）
  • S3 假突破賣點：均線下彎，價格短暫突破後又跌破
  • S4 反轉賣點：價格急漲造成正乖離過大

核心心法: B2、B3是最實用的買點（趨勢確認後進場），均線方向比價格位置更重要。

跳空缺口操作

四種缺口型態：

缺口定義操作
**普通缺口**盤整區間內，無明顯趨勢忽略
**突破缺口**突破盤整區，成交量放大進場訊號
**逃逸缺口**趨勢進行中，再出現跳空繼續持有
**竭盡缺口**趨勢末段，爆量但價格停滯出場訊號

判斷關鍵: 2-3天內是否回補。突破未回補 = 真突破；快速回補 = 假突破。

選擇權策略核心

賣方策略核心：

  • 勝率高（85%+），但偶爾毀滅性虧損
  • 賣價外選擇權賺時間價值
  • 資金要充足（一口至少5-10萬）
  • 絕對不能加碼攤平

Iron Condor（鐵兀鷹）：

  • 同時賣出CALL價差單 + PUT價差單
  • 不預測漲跌，賭股價在區間內震盪
  • Delta中立建倉，獲利來自Theta衰減
  • 最大獲利 = 淨權利金，最大虧損 = 價差寬度 - 淨權利金
  • 風險管理：虧損達權利金2倍停損

Theta時間價值：

  • 價平選擇權Theta最大
  • 越接近到期，Theta衰減越快
  • 結算週是時間價值消失最快的時候
  • 賣方賺Theta：平均每天收10點左右

Covered Call（掩護性買權）：

  • 持有現貨 + 賣出Call選擇權
  • 「用股票收租」的概念
  • 適合：盤整緩漲市場、長期持有部位
  • 不適合：成長股、中小型股（容易暴漲）

Straddle vs Strangle：

策略操作使用時機風險特性
Long Straddle買Call+買Put（同履約價）預期大波動損失有限，成本高
Short Straddle賣Call+賣Put（同履約價）預期盤整勝率高，風險無限
Long Strangle買Call+買Put（不同履約價）預期大波動成本較低，需更大波動
Short Strangle賣Call+賣Put（不同履約價）預期盤整勝率高，風險更大

2% + 6% 規則（資金控管）

2% 規則：

  • 單筆風險 = 總資金 × 2%
  • 可買股數 = 2% 風險上限 ÷ (買價 - 停損價)
  • 每月初重新計算資金

6% 規則：

  • 總資金從前月末下跌6%時，立即停止當月所有交易
  • 平掉所有未平倉部位，月底前只觀望

為什麼需要：

  • 2% 規則防「鯊魚」（一次致命虧損）
  • 6% 規則防「食人魚群」（連續小虧損）
  • 連敗代表策略可能失效，需要停下來檢查

部位操作（多空雙作）

核心概念: 試水溫 + 避險，降低「看錯方向」的風險。

操作方式：
1. 看好買進1張（試水溫）
2. 股價下跌出現黑K
3. 加放空1張避險
4. 續跌：停損買單、加碼空單
5. 續漲：停損空單、加碼多單

核心體悟: 不管怎麼走，最大虧損都可控。從「大賠」變「小賺」或「小賠」。

VWAP成交量加權平均價格

公式:

VWAP = Σ(價格 × 成交量) / Σ成交量

實戰應用：

  • VWAP以上：當日買方成本區，支撐較強
  • VWAP以下：當日賣方成本區，壓力較重
  • 價格在VWAP之上 → 偏多操作
  • 價格在VWAP之下 → 偏空操作

布林通道（Bollinger Bands）

是什麼：由三條線組成的帶狀通道，中軌為 20 日均線，上下軌為中軌 ± 2 個標準差。

核心邏輯：95% 的價格會落在通道內，上下軌代表極端價格。

兩大操作派別：

派別核心邏輯適用情境
逆勢觀點價格回歸均值，上下軌是極端盤整市場
順勢觀點大趨勢時沿通道邊緣發展趨勢明確

十種型態：
1. 通道收縮：波動性低，關注突破方向
2. 通道擴張：波動性增加，趨勢啟動
3. 上軌觸碰：超買或強勢，需結合趨勢判斷
4. 下軌觸碰：超賣或強勢下跌
5. W 型底部：下軌附近形成雙底
6. M 型頂部：上軌附近形成雙頂
7. 走廊運行：緊貼軌道持續運行，趨勢強勁
8. 布林反轉：觸軌後彈簧般反彈
9. 假突破：短暫突破後迅速回歸
10. 收縮性突破：通道收窄後帶量突破

實戰要點：

  • 通道收縮後的突破是高勝率訊號
  • 走廊運行時不要逆勢猜頂摸底
  • 不要隨便改參數（20, 2 是市場共識）

ATR 動態停損

是什麼：真實平均波動區間（Average True Range），納入跳空因素的真實波動測量。

計算方式：TR 取三種情況最大值
1. 當日最高 - 當日最低
2. |當日最高 - 前日收盤|
3. |當日最低 - 前日收盤|

實戰應用：

  • 停損點 = 買進價 - k × ATR（k 通常 1-2）
  • 棘輪止損：隨價格有利方向移動，逐步上調停損
  • 資金管理：建倉單位 = 帳戶資金 × 1% ÷ 美元計價波動性

優缺點：

  • 優點：真實反映波動，適用多種市場
  • 缺點：低波動市場效果差、有滯後性、不提供方向訊號

SAR 拋物線指標

是什麼：Stop and Reverse，透過拋物線形狀的點位顯示停損與反轉位置。

核心邏輯：永遠持有部位，多空不斷翻轉。

操作方式：

  • 紅點出現在 K 線下方 → 上升趨勢，做多
  • 綠點出現在 K 線上方 → 下降趨勢，做空
  • 觸及 K 線 → 翻轉訊號

使用原則：
1. 日 K 比小週期準確
2. 適合趨勢盤，不適合盤整盤
3. 參數可調整快慢（AF=0.02 預設，0.05 較快）

搭配技巧：

  • 延遲啟動停利：移動停利價超過成本價才開始移動
  • 區間 K 棒高低點 + 5%：避開盤整陷阱

  • MACD 柱狀圖：DIF - DEA

DIF 向下跌破 DEA死亡交叉，賣出

RSI 相對強弱指標

公式：RSI = 買方力道 / (買方力道 + 賣方力道) × 100

區間判斷：

RSI < 30超賣區
RSI ≈ 50多空平衡

3. 創新高確認 → 買進

  • 突破幅度需達 3% 以上

OBV 能量潮指標

核心概念：「量先價行」—— 成交量是股市元氣，股價只是表象。

計算方式：

  • 上漲日：OBV = 前日 OBV + 今日成交量
  • 下跌日：OBV = 前日 OBV - 今日成交量

四大訊號：

訊號判斷
頂背離股價創新高，OBV 未創新高 → 賣出
底背離股價創新低，OBV 未創新低 → 買進

同步下跌OBV + 股價同步下跌 → 賣出或觀望

適用情境：成交量大的熱門股、中長期趨勢

公式：MTM = 當日收盤價 - n 天前收盤價（n 通常 10 或 12）

  • 價格跌破均線 + MTM 為負 → 空方確認
