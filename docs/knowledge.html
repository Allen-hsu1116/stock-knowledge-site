K線型態

K線基礎

每根K線由四個價格組成：開盤價（Open）、最高價（High）、最低價（Low）、收盤價（Close）。

解讀三要素： - 顏色：紅K（上漲）vs 黑K（下跌） - 實體長度：趨勢強度（長實體 = 趨勢明顯） - 影線長度：多空交戰激烈程度（長影線 = 阻力大）

單根K線型態

| 型態 | 定義 | 市場訊號 | |------|------|----------| | 大紅K | 收盤遠高於開盤，實體長，影線<20% | 多頭強勢，後市看漲 | | 大黑K | 收盤遠低於開盤，實體長，影線<20% | 空頭強勢，後市看跌 | | 十字線 | 開盤≈收盤，長上下影線 | 多空勢均力敵 | | 鎚子線 | 短實體+長下影線（無上影線） | 下檔有支撐 | | 倒鎚線/墓碑線 | 短實體+長上影線（無下影線） | 上檔有壓力 | | 紡錘線 | 短實體+上下影線 | 市場猶豫 |

看漲反轉型態

| 型態 | 組成 | 訊號 | |------|------|------| | 錘形線 | 單根K線，短實體長下影線，位於下降趨勢底部 | 拋售壓力後買壓推升，看漲 | | 多頭吞噬 | 兩根K線，短紅K被長綠K完全吞沒 | 買家明顯獲勝，看漲 | | 早晨之星 | 三根K線：長紅-短K-長綠 | 賣壓消退，看漲 | | 白色三兵 | 三根連續長紅K，開盤收盤均高於前日 | 買壓穩步上升，強烈看漲 |

看跌反轉型態

| 型態 | 組成 | 訊號 | |------|------|------| | 上吊線 | 單根K線，短實體長下影線，位於上升趨勢頂部 | 拋售明顯，多頭失控，看跌 | | 射擊之星 | 單根K線，短實體長上影線，位於上升趨勢頂部 | 流星墜地，看跌 | | 空頭吞噬 | 兩根K線，短綠K被長紅K完全吞沒 | 價格高峰，看跌 | | 黃昏之星 | 三根K線：長綠-短K-長紅 | 上升趨勢反轉，看跌 | | 三隻烏鴉 | 三根連續長紅K，影線短或無 | 賣方力量連續超越買方，看跌 |

整理型態

| 型態 | 定義 | 市場訊號 | |------|------|----------| | 十字星 | 開盤≈收盤，短實體長影線 | 多空掙扎，中性訊號 | | 紡錘線 | 短實體，等長上下影線 | 市場猶豫，盤整期 | | 下降三法 | 長紅K + 三根短綠K + 長紅K | 趨勢延續，看跌 | | 上升三法 | 長綠K + 三根短紅K + 長綠K | 趨勢延續，看漲 |

實戰心法

- 單根K線不構成訊號：需搭配前後K線一起判斷 - 影線透露過程，實體透露結果：長影線 = 激烈交戰，長實體 = 明確趨勢 - 長實體+短影線：趨勢延續性高 - 短實體+長影線：趨勢延續性低 - 型態位置很重要：同樣型態在高位和低位意義不同

均線判斷

均線種類

| 種類 | 計算方式 | 特點 | |------|----------|------| | SMA（簡單移動平均） | 收盤價算術平均 | 權重相等，最常用 | | EMA（指數移動平均） | 近期價格權重較高 | 對價格波動敏感 |

常用均線週期

| 週期 | 名稱 | 用途 | |------|------|------| | 5MA | 周線 | 短線/當沖 | | 10MA | 雙周線 | 短線 | | 20MA | 月線 | 波段（最重要） | | 60MA | 季線 | 中期 | | 120MA | 半年線 | 中長期 | | 240MA | 年線 | 長期 |

均線三大用法

1. 價格突破均線 - 收盤價 > 10MA → 過去10天進場者賺錢 - 收盤價 < 20MA → 過去20天進場者賠錢

2. 黃金交叉與死亡交叉

| 交叉類型 | 定義 | 訊號 | |----------|------|------| | 黃金交叉 | 短均線向上突破長均線 | 看漲，多單進場訊號 | | 死亡交叉 | 短均線向下跌破長均線 | 看跌，空單進場訊號 |

3. 多頭排列與空頭排列

| 排列型態 | 定義 | 訊號 | |----------|------|------| | 多頭排列 | 短均線在上，長均線在下（5>10>20>60） | 趨勢強勁上升 | | 空頭排列 | 短均線在下，長均線在上（5<10<20<60） | 趨勢疲弱下跌 |

均線使用注意

- 盤整時頻繁交叉：會被來回洗刷，造成虧損 - 均線是落後指標：反應慢半拍，短線精確度有限 - 均線是市場成本線：價格在均線上代表過去一段時間買者賺錢 - 動態支撐壓力：上升趨勢中均線作為支撐，下降趨勢中均線作為壓力

支撐壓力

定義

| 類型 | 定義 | 特性 | |------|------|------| | 支撐線 | 價格停止下跌並反彈的位置 | 買方多於賣方，價格上升 | | 壓力線 | 價格停止上漲並下跌的位置 | 賣方多於買方，價格下跌 |

如何畫線

| 類型 | 畫法 | |------|------| | 支撐線 | 連接至少 2 波低點，向右延伸 | | 壓力線 | 連接至少 2 波高點，向右延伸 |

強度判斷

- 觸及次數越多：支撐/壓力越可靠 - 時間越久：心理認同越強 - 成交量越大：該位置越重要

角色互換

- 支撐線被跌破 → 轉變為壓力線 - 壓力線被突破 → 轉變為支撐線

交易策略

| 情境 | 支撐線操作 | 壓力線操作 | |------|------------|------------| | 價格觸及 | 考慮買進 | 考慮賣出 | | 價格突破 | 賣出或觀望 | 準備作多 | | 價格回測 | 買進機會 | 賣出訊號 |

K棒判斷支撐壓力

  • 觀察上下影線：影線尖端常是支撐/壓力位置
  • 觀察實體密集區：大量K棒實體疊加的區間
  • 整數關卡：如 100 元、1000 點等心理關卡

    • 實戰心法

    - 假突破要防範：突破需要確認（收盤站上、成交量放大） - 越多觸及越可靠：2-3 次以上測試才確認有效 - 時間框架要匹配：日線支撐壓力比分線更重要

    財報假帳偵測

    假帳五大動機：

    | 動機 | 說明 | |------|------| | 順利上市上櫃 | 為了符合上市櫃標準，美化財報 | | 掩飾本業衰退 | 怕財報難看導致股價下跌或銀行抽銀根 | | 避免大股東破產 | 大股東股票質押成數高（>60%），股價不能跌 | | 操縱股價 | 先低價買進，再假帳拉抬股價，最後高賣 | | 掏空公司 | 把公司資產搬進自己口袋 |

    財報假帳六大特徵：

    | 特徵 | 判斷標準 | 原理 | |------|----------|------| | 應收帳款天數過高 | > 90天 | 假交易利潤藏在應收帳款銷不掉 | | 存貨天數過低 | < 10天且異於同業 | 虛增營收但存貨沒增加 | | 不動產過高 | 週轉率低於同業 | 應收帳款轉入不動產消化 | | 長期投資過高 | 未上市櫃公司投資多 | 難驗證的資產藏假 | | 現金過高 | 現金高但借款也高 | 現金可能被限制（博達案） | | 雜項資產過高 | 其他應收款、預付款高 | 雜項科目藏假 |

    假帳非財務警訊：

    | 警訊 | 判斷方式 | |------|----------| | 老闆誠信被質疑 | 質押成數 > 80% = 很缺錢 | | 經營階層變動 | 財務長、獨董、董事接連離職 | | 股價異常飆漲 | 漲幅遠超同業且無基本面支撐 | | 利多消息不斷 | 頻繁發布利多、澄清「不予置評」太多 | | 借殼後初期 | 借殼 3-4 年內營收暴增但現金流為負 | | 產業差獲利好 | 技術普通、產業景氣差卻賺錢 |

    假帳偵測心法：

  • 先看老闆質押：> 80% = 小心
  • 再看應收帳款天數：> 90天 = 警訊
  • 看存貨天數：< 10天且與同業差距大 = 警訊
  • 看現金與借款：雙高 = 現金可能被限制
  • 看營業現金流：營收成長但現金流為負 = 怪
  • 看利多頻率：澄清「不予置評」太多 = 放消息炒股
  • 看借殼歷史：借殼後 3-4 年內營收暴增 = 警訊
  • 研究產業：產業差卻賺錢 = 可能有假

    企業掏空識別

    掏空定義：公司大股東或管理層以不合法手段轉移上市公司資源，損害中小股東利益。

    掏空手法分級：

    | 級別 | 特點 | 常見手法 | |------|------|----------| | 初級班 | 簡單關係人交易 | 訂單過水、指定進貨、高價買資產、代養私人公司、庫藏股護盤 | | 中級班 | 牽涉多家公司/海外 | 虛增營收、切割金雞母、金融工具套利、海外公司隱匿 | | 高級班 | 博達案等級 | CLN 虛增存款、應收帳款融資、海外可轉債套利、擔保借款 |

    博達案四大手法：

  • 信用連結債券（CLN）虛增銀行存款
  • 應收帳款融資 + 零息債券
  • 海外可轉債套利
  • 為海外人頭公司擔保借款

    • 掏空防範警訊：

    | 警訊 | 觀察重點 | |------|----------| | 大股東一股獨大 | 董事會、股東會形同虛設 | | 內部監管不健全 | 獨立董監無法發揮監督功能 | | 關係人交易異常 | 交易條件優厚、租金高於行情、應收票據未如期兌現 |

    關係人交易查核重點：

  • 識別關係人：財報附註的關係人揭露、實質關係人（姻親、同一辦公處所）
  • 評估交易條件：比較與公平交易的條款、收付款期間是否異常
  • 查核佐證資料：會議紀錄、契約文件、應收帳款帳齡分析

    管理階層品質評估

    好 CEO 八大特質：

    | 特質 | 內容 | 判斷方式 | |------|------|----------| | 效率決定生死 | 去中心化管理、充分授權 | 總部官僚程度 | | 聚焦資本配置 | 每塊錢都要明確報酬 | 併購績效、買回紀錄 | | 綁定股東利益 | 利益與股東一致 | 持股比例、薪酬結構 | | 高效運用槓桿 | 知道何時用、如何用 | 槓桿買穩定現金流資產 | | 勇於逆向前行 | 危機時買進、狂熱時冷靜 | CEO 在危機時的反應 | | 數據思維制勝 | 用數據做決策 | 決策是否看數據 | | 永遠關注現金流 | 預留至少 3 年收入中斷的現金 | 自由現金流穩定性 | | CEO = CIO | 每個決策都是投資決策 | 是否喊口號還是把自由現金流最大化 |

    偽 CEO 特質（避雷）：

    | 警訊 | 表現 | |------|------| | 頻繁談願景 | 法說會都在畫大餅，很少談報酬 | | 併購紀錄不亮眼 | 併購多但商譽減損多 | | 買賣時機錯誤 | 股價高時買回、股價低時發行 | | 現金流與獲利脫節 | EPS 好但自由現金流為負 |

    評估管理階層實戰步驟：

  • 查 CEO 背景（Google 負面新聞、過往績效）
  • 查薪酬結構（固定 vs 績效 vs 選擇權）
  • 查持股變動（持續賣出 = 不看好）
  • 查併購績效（創造價值還是毀滅價值）
  • 查買回時機（低買還是高買）
  • 查媒體曝光（太多願景 = 缺乏紀律）

    籌碼面分析

    三大法人

    | 法人 | 資金規模 | 操作偏好 | 參考價值 | |------|----------|----------|----------| | 外資 | 40-60%成交 | 大型權值股、中長期 | 高（續買率74%） | | 投信 | ~5%成交 | 中小型股、波段 | 中（留意季底作帳） | | 自營商 | 1-2%成交 | 短線、避險 | 低（進出快） |

    實戰要點： - 外資連買 → 大盤上漲率70% - 投信季底（3/6/9/12月）會作帳行情 - 自營商避險買進 ≠ 看好

    融資融券

    口訣： - 資減 + 券增 + 法人買 = 利多 - 資增 + 券減 + 法人賣 = 利空

    解讀： - 融資增加 → 散戶搶進，上檔有壓力 - 融券增加 → 可能軋空行情

    特殊券商

    | 類型 | 用途 | 實戰 | |------|------|------| | 高勝率券商 | 找高手跟單 | 凱基-台北、元富-四維 | | 隔日沖券商 | 避免當韭菜 | 今天買明天賣，不要追高 | | 地緣券商 | 抓公司派動向 | 公司附近券商異常買進 = 可能利多 |

    大戶持股比率

    關鍵區間： - < 40%：股權太分散，不易漲 - 40% ~ 80%：籌碼健康集中 ✅ - > 80%：過度集中，炒作空間小

    買賣家數差

    公式： 買超券商家數 - 賣超券商家數

    判讀： - 負數 = 籌碼集中 → 有利上漲 - 正數 = 籌碼分散 → 不利上漲

    實戰心法

    正向組合： - 投信連買 + 量增 + 均線金叉 = 進場點 - 大戶持股連增 + 股價緩漲 = 佈局中

    反向陷阱： - 主力大買 + 股價不漲 = 可能出貨 - 融資暴增 + 股價橫盤 = 誘多陷阱 - 法人大賣 + 技術破位 = 趕快跑

    常見誤區

    | 誤區 | 正解 | |------|------| | 法人買超 = 必漲 | 可能是避險部位 | | 主力連買 = 沒風險 | 可能是誘多吸籌 | | 高融券 = 看空 | 可能引發軋空 |

    董監改選行情

    定義：公司派和市場派在股東會前搶奪籌碼，導致股價易漲難跌的現象。公司法規定董監事任期最長3年，每3年改選一次。

    為什麼會漲： - 公司派：怕經營權被搶走，持股不賣甚至回補 - 市場派：想取得經營權，從市場不斷買進 - 籌碼被鎖住只進不出，供需失衡，股價自然上漲

    董監改選行情三階段：

    | 階段 | 時間 | 特徵 | 散戶策略 | |------|------|------|----------| | 默默吃貨期 | 改選前一年Q4甚至更早 | 大股東默默吃貨，股價不明顯上漲 | 太早介入，時間成本高 | | 蜜月期 | 每年12月到隔年3月（行情高峰） | 雙方搶籌碼最激烈，股價漲勢凌厲 | 最佳搭轉時機 | | 回歸基本面 | 4月停止過戶後 | 改選落幕，敗選方可能釋出籌碼 | 提防股價大幅修正 |

    實例： - 國票金(2889)：2010年10月起漲，2011年2月從10元漲到17.6元，漲幅80% - 三陽(2206)：2014年改選，累積漲幅超過100%，但經營權底定後跌回起漲點

    容易出現董監改選行情的四大條件：

    | 條件 | 判斷標準 | 原理 | |------|----------|------| | 董監持股偏低 | <20%（容易引起覬覦），<10%（高風險） | 公司派防衛門檻太低 | | 董監持股質押比率過高 | >50% | 現金不足，容易被市場派鎖定 | | 股價偏低且具潛在價值 | 股價淨值比<1，有資產、現金部位高 | 取得經營權成本低 | | 小型股 | 股本<20萬張，非集團、非家族企業 | 介入成本低，籌碼易操控 |

    法定持股成數標準：

    | 資本額 | 董事持股 | 監察人持股 | |--------|----------|------------| | 3億以下 | 15% | 1.5% | | 3~10億 | 10% | 1% | | 10~20億 | 7.5% | 0.75% | | 20~40億 | 5% | 0.5% | | 40~100億 | 4% | 0.4% | | 100~500億 | 3% | 0.3% |

    FinLab 量化選股條件（回測勝率82%）：

  • 股本<200000張（小股本）
  • 董監持股占比5~15%（太低有財務風險，太高公司派穩固）
  • 股價淨值比<1.3，負債比率<50%，營業利益率>0（公司本質不差）
  • 近6個月董監增加股數>0（內部人有回補意願）

    • 回測結果：勝率82%、持有3個月平均報酬15%、建議停損10%

    實戰心法： - 第1階段太早，第2階段是蜜月期最佳進場，第3階段要跑 - 經營權之爭結束後，敗選方會賣股票，股價可能快速回檔 - 這是「事件交易」，事件結束就要跑

    土洋對作判斷

    定義：外資（洋）與投信（土）買賣方向相反，一買一賣。

    | 方法 | 說明 | |------|------| | 買賣超數量比較 | 誰量大，誰主導方向 | | 股價趨勢方向 | 股價突破 = 多方獲勝，股價跌破 = 空方獲勝 |

    大型股 vs 小型股判斷：

    | 股票類型 | 主導力量 | 關鍵籌碼 | |---------|---------|----------| | 大型股 | 外資 | 看外資買賣超 | | 小型股 | 投信 | 看投信買賣超 |

    土洋同買（最佳訊號）：外資、投信同時買超同一檔個股 = 強烈買進訊號

  • 外資多單留倉與空單留倉一增一減（開口擴大）

    • 同時符合三種現象 = 外資動真格

    當沖（Day Trading）

    定義: 當天買進當天賣出，不留倉過夜

    申請條件: - 年滿 20 歲 - 開戶滿 3 個月 - 近 1 年成交 10 筆以上

    優勢: - 時間短，不用長時間等待結果 - 門檻低，不用負擔股票本金（只需手續費 + 證交稅） - 降低心理負擔，避免收盤後負面新聞影響 - 不用耗時研究公司基本面 - 證交稅減半：0.3% → 0.15%

    風險: - 交易頻繁，手續費成本高 - 需要盯盤，上班族不方便 - 高槓桿（約 290 倍），可能誤把投資當賭場 - 需有波動幅度的股票才划算

    適合對象: 不喜歡研究公司、想要快速知道結果、有時間盯盤的人

    紀律: 一天操作次數不用多，有賺就好，維持穩定紀律

    隔日沖

    定義: 今天買進，隔天賣出

    特點: - 風險限制在兩天內 - 適合盤中有波動的商品 - 波動幅度要高於交易成本才划算

    波段操作

    定義: 股價低檔買進、高檔賣出，持有幾天到數週

    進出場判斷: - 支撐位買進、阻力位賣出 - 搭配技術分析形態學 - 輔以基本分析、布林通道等指標

    優勢: - 摩擦成本比當沖少 - 交易壓力比當沖低 - 成功率較高，是新手適合的入門策略

    缺點: - 沒有固定規則，高低點判斷眾說紛紜 - 賺錢的時間成本較大

    適合對象: 新手菜鳥想從交易入手

    買進長期持有（價值投資）

    定義: 長線策略，不理會短期波動，持有長期一定上漲的資產

    流程: 基本分析 → 估值 → 買進 → 長期持有 → 獲利賣出

    優勢: - 風險小、勝率高（前提是分析能力足夠） - 有機會創造巨大報酬

    缺點: - 基本分析時間成本很大 - 需要估值能力和財報解讀能力

    適合對象: 有分析能力、能接受慢慢致富的人

    指數型投資（被動投資）

    定義: 投資指數 ETF（如 0050、標普 500），不需要選擇單一公司

    優勢: - 不需要基本分析能力 - 不用判斷買賣時機 - 透過複利效應取得不錯報酬 - 巴菲特建議普通人投資指數型基金

    缺點: - 報酬率較中庸

    適合對象: 分析能力不好、沒時間分析的人

    資金配置建議

    新手建議: - 大部分資金：指數型投資（勝率高、風險低） - 少量資金：波段交易（判斷自己適不適合投機） - 有經驗後再嘗試當沖或集中選股

    投機心法: - 80/20 法則：市場上 80% 的錢被 20% 的人贏走 - 永遠要注意停損停利控制風險 - 投機很看天賦和運氣，少不了大量練習

    實戰操作步驟

    進場前規劃：

  • 確定進場價格
  • 計算停損價（進場價 × 0.95 或 K 線低點）
  • 計算停利價（進場價 × 1.2 或前高壓力）
  • 決定倉位大小（單筆風險不超過總資金 2%）

    • 進場後執行：

  • 立即設定停損單（條件單或手動監控）
  • 股價上漲至最高點後，啟動移動停利（回檔 10% 出場）
  • 收盤跌破停損價 → 次日立即出場，不抱幻想

    • 注意事項： - 停損點不得更改，否則等於沒設 - 順勢交易為主，高勝率位置進場 - 收盤跌破停損價要立即出場，要有壯士斷腕決心

    倉位管理（5 種方法）

    1. 固定百分比法 

    倉位大小 = 總資金 × 風險比例 / (進場價 - 停損價)
    - 單筆風險：1-2% 總資金 - 適用：長期穩健增長型交易者

    2. ATR 波動率法 

    倉位大小 = (帳戶資金 × 風險%) / (ATR × 停損倍數 × 每點價值)
    - 高波動自動縮小倉位 - 適用：追求與市場波動同步

    3. 凱利公式（Kelly Criterion） 

    f* = (p × b - q) / b
    - f* = 最佳下注比例，p = 勝率，q = 敗率，b = 盈虧比 - 實務建議：使用「半凱利」更安全 - 適用：勝率與盈虧比穩定的策略

    4. 金字塔加碼法 - 趨勢確認後初始倉位 - 獲利後回調時加碼 - 每次加碼金額遞減 - 適用：趨勢跟隨策略

    5. 固定金額法 - 每筆交易固定風險金額 - 計算簡單但帳戶波動後不靈活 - 適用：小型帳戶

    風險報酬比與期望值

    風險報酬比公式 

    風險報酬比 = 預期獲利 / 可能虧損

    最低勝率要求

    | 風險報酬比 | 所需最低勝率 | |-----------|-------------| | 1:1 | 50% | | 1:1.5 | 40% | | 1:2 | 33% | | 1:3 | 25% | | 1:5 | 17% |

    期望值公式 

    期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損)

    R 乘數概念 - R = 停損距離（1 單位風險） - 報酬 = R 的倍數 - 範例：停損 5 元是 1R，獲利 15 元是 3R

    ---

    最大回撤 MDD

    MDD 定義 

    MDD = (歷史最高點 - 最高後最低點) / 歷史最高點 × 100%

    跌幅與回升幅度對照

    | 跌幅 | 需要回升幅度 | 難度 | |------|------------|------| | -10% | +11% | 容易 | | -20% | +25% | 中等 | | -30% | +43% | 較難 | | -40% | +67% | 困難 | | -50% | +100% | 極難 | | -70% | +233% | 幾乎不可能 |

    核心結論：控制回撤比追求報酬更重要

    ---

    VaR 風險價值

    定義：在特定信心水準下，投資組合在特定時間內最大可能虧損

    三種計算方法

    | 方法 | 優點 | 缺點 | |------|------|------| | 歷史模擬法 | 簡單、不需假設分配 | 依賴歷史、無法預測極端 | | 變異數-共變異數法 | 計算快速 | 假設常態分配、忽略肥尾 | | 蒙地卡羅模擬 | 可處理複雜分配 | 計算量大、依賴參數假設 |

    VaR vs CVaR

    | 特性 | VaR | CVaR | |------|-----|------| | 關注點 | 閾值損失 | 尾端平均損失 | | 肥尾風險 | 不考量 | 充分考量 | | 適合用途 | 日內監控 | 資本配置、壓力測試 |

    避險策略

    核心觀念 - 避險 ≠ 消除風險，是將傷害控制在可接受範圍 - 避險有成本（交易成本 + 機會成本） - 事前避險成本遠低於事後避險

    事前 vs 事後避險

    | 類型 | 特點 | 成本 | |------|------|------| | 事前避險 | 根據風險預判，事先配置對抗資產 | 低 | | 事後避險 | 意外發生後才減碼或調整 | 高（易追高殺低）|

    何時該避險 - 進場理由「可能改變」→ 考慮避險 - 進場理由「沒變」→ 不要因恐慌打破紀律

    何時不該避險 - 只因價格下跌或恐慌 → 不該避險 - 常常想避險 → 策略本身有問題

    VaR 與 CVaR 風險量化

    VaR（風險價值）

    | 項目 | 說明 | |------|------| | 定義 | 特定期間、特定信賴水準下的最大預期損失 | | 公式 | VaR = Z × σ × √T × V（Z=信賴水準z值、σ=波動度、T=天數、V=部位價值） | | 三種計算方法 | 變異數-共變異數法、歷史模擬法、蒙地卡羅模擬法 | | 限制 | 不考慮尾部損失、不滿足次可加性 |

    CVaR（條件風險價值）

    | 項目 | 說明 | |------|------| | 定義 | 超過 VaR 後的平均損失（期望尾端損失） | | 公式 | CVaR = E[Loss \| Loss > VaR] | | 優勢 | 考慮尾端損失嚴重度、滿足一致性風險測度 |

    VaR vs CVaR 比較

    | 特性 | VaR | CVaR | |------|-----|------| | 關注點 | 閾值損失 | 尾端平均損失 | | 次可加性 | ❌ 不滿足 | ✅ 滿足 | | 尾部資訊 | 不考量 | 充分考量 | | 適合用途 | 日內監控、風險限額 | 資本配置、壓力測試 |

    投資組合 VaR 計算 

    組合 VaR = √(Σ DEAR_i² + 2×Σ(ρ_ij × DEAR_i × DEAR_j))
DEAR = |-α| × σ × 價值（99% 信賴水準 α = 2.33）

    實務要點： - 台股常有肥尾現象，變異數-共變異數法可能低估風險 - VaR 搭配 CVaR、壓力測試使用 - 定期回測驗證模型準確度
