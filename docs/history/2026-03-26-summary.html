1. 當沖操作策略

三大操作模式:

| 模式 | 獲利目標 | 特性 | 適合標的 | |------|----------|------|----------| | Tick流（機關槍式） | <1% | 高頻進出、資金效率高 | 震盪劇烈、成交量大 | | 一波流（狙擊槍式） | 1-5% | 抓當日最大波段 | 熱門族群、題材股 | | 日內波（大砲式） | >5% | 盤前功課要充足 | 有長紅/黑潛力股 |

選股四大原則:

  • 成交量：100元以下需破萬張、100元以上需五千張以上
  • 波動度：盤中波幅5%以上才划算
  • 最適價格區間：100-149元（1 Tick = 0.5元，手續費優惠時1 Tick可打平成本）
  • 避開冷門股

    • 日內時段特性: - 開盤（9:00-9:30）：波動最大、量能最熱，但不適合追高 - 早盤（9:30-10:30）：技術型態明朗，主戰場 - 中盤（10:30-12:30）：量縮波動小，減少出手 - 尾盤（12:30-13:30）：當沖客出場、波段客進場

    資金配置: - 順勢單（追突破）：3:2:1 分批進場 - 逆勢單（抓轉折）：1:2:3 分批進場 - 設定單日最大虧損（如5000元），達到立即停手

    停損停利心法: - 風險:獲利 = 1:2 以上 - 進場時就設好，不要等虧損才想停損 - 觸價單自動執行，不要相信意志力

    勝率與賺賠比公式: 

    期望值 = (賺賠比 × 勝率) - (1 × 賠率)
正數代表策略可獲利

    2. 波段操作策略

    三大鐵律:

    | 鐵律 | 核心觀念 | 違規後果 | |------|----------|----------| | 股票≠房地產 | 加權指數會創新高，個股可能下市 | 攤平攤到變壁紙 | | 賠錢絕對不加碼 | 下跌時逢低攤平是錯誤觀念 | 越買賠越多 | | 40日均線控盤法 | 跌破40日線 = 空手或做空 | 持有下跌波段 |

    期貨留倉六大心法:

  • 只在賺錢時留倉
  • 事先設定出場點
  • 跳空避險（期貨多 + 買PUT）
  • 接受行情走不出去
  • 順勢留倉 + 慎選進場點
  • 停損不要設太小（參考ATR的1/4-1/5）

    • 關鍵認知: 當沖拚速度和勝率，波段拚持有和賺賠比。波段更適合上班族，因為不用盯盤。

    3. 價值投資選股策略

    核心五要點:

  • 評估公司（管理層、財務、護城河）
  • 內在價值 > 價格（本益比、股價淨值比、EV/EBIT）
  • 安全邊際（內在價值×70%-80%=進場價）
  • 了解市場先生（恐慌時買進、狂熱時等待）
  • 長期持有（5-10年視角）

    • 五位投資大師選股條件:

    | 大師 | 核心條件 | 特色 | |------|----------|------| | 葛拉漢 | EPS>1元、本益比<15、流動比率>100% | 低價買平凡公司 | | 彼得林區 | 負債比<30%、本益比<15、董監持股>30% | 成長股先鋒 | | 麥克墨菲 | 營業利益率>10%、ROE>8%、研發占比>5% | 獲利與創新 | | 馬克約克奇 | 毛利率>20%、稅前純益成長>3% | 高毛利標竿 | | 巴菲特 | 毛利率>20%、ROE>8%、股價淨值比<1.2、本益比<15 | 護城河價值型 |

    避免價值陷阱: 股價下跌是「一次性原因」還是「長期獲利能力衰退」？後者無法回歸。

    4. 突破交易與拉回買進

    時間對應關係:

    | 想賺的波段 | 需要的盤整時間 | |------------|----------------| | 1個月 | 至少3個月以上 | | 半年 | 1-2年 | | 1年以上 | 5年以上 |

    真假突破判斷:

    | 真突破 | 假突破 | |--------|--------| | ✅ 成交量放大 | ❌ 放量卻漲不動 | | ✅ 有基本面利多支持 | ❌ 無實質利多 | | ✅ 回檔不破前高 | ❌ 快速回落 | | ✅ 站穩壓力區上方 | ❌ 無法再創新高 |

    突破買進 vs 拉回買進:

    | 項目 | 突破買進 | 拉回買進 | |------|----------|----------| | 進場位置 | 高點追價 | 相對低檔 | | 停損空間 | 需要大停損 | 小停損即可 | | 風險程度 | 高（易被洗出場）| 低 | | 勝率 | 較低 | 較高 |

    核心觀念: 拉回買進比突破買進好太多——用時間換空間，放棄追高的獲利機會，換取更低的風險。

    5. 動能投資策略

    五大動能類型:

    | 策略 | 核心邏輯 | 適用時機 | |------|----------|----------| | 價格動能 | 過去漲的還會漲 | 明顯多頭市場 | | 均線交叉 | 黃金交叉做多、死亡交叉做空 | 趨勢明確時 | | 成交量動能 | 量價齊揚確認強勢 | 突破關鍵價位 | | 量價策略 | 有價要有量、有量要有價 | 所有動能操作 | | 反轉策略 | RSI極端值反轉 | 行情過熱時 |

    TEJ回測數據（2019-2025）: - 年化報酬率：28.26% - 年化波動度：37.25%（心臟要強） - 最大回檔：超過30% - 關鍵：獲利來自少數幾次大趨勢

    動能 vs 均值回歸: - 動能：「強者恆強，漲會續漲」→ 趨勢行情 - 均值回歸：「漲多會跌，跌多會漲」→ 震盪行情 - 選錯策略就是虧損

    6. 金字塔加碼策略

    正金字塔（左側交易）vs 倒金字塔（右側交易）:

    | 方法 | 操作 | 適用時機 | 風險 | |------|------|----------|------| | 正金字塔 | 越跌買越多 | 長期趨勢向上的股票 | 可能「越買越深」 | | 倒金字塔 | 越漲買越多 | 多頭行情確認 | 成本較高、假突破風險 | | 等注加碼 | 每次固定金額 | 最穩健 | 獲利比正金字塔少 |

    核心心法:

  • 只在賺錢時加碼（市場證明你是對的）
  • 突破關鍵點再加碼
  • 設好停損保命
  • 只賠最後一碼（加碼後出場價設在前一碼上方）

    • 阮慕驊SOP: - 底部30% → 漲10%加20% → 再漲10%加10% - 移動停利跟著10日線

    7. 箱型理論（Darvas Box）

    核心概念: 股價在「箱型區間」內整理震盪，突破箱頂代表新趨勢啟動。

    操作方法: - 順勢：突破箱型上緣 → 進場，停損設箱型下緣或上緣下一檔 - 逆勢：箱型下緣買進、上緣賣出（不建議一般投資人）

    假突破判斷: - 突破但成交量沒放大 → 可能假突破 - K線收盤價沒真正突破 → 可能假突破

    關鍵體悟: 勝率只有30-40%，但盈虧比好。關鍵是紀律——不要預設「合理」獲利就停利，讓箱型告訴你什麼時候出場。

    8. 葛蘭碧八大法則

    四大買點:

    | 買點 | 定義 | 適用時機 | |------|------|----------| | B1 突破買點 | 均線走平/上彎，價格突破均線 | 趨勢反轉初期 | | B2 支撐買點 | 均線上揚，價格拉回測試均線反彈 | 多頭趨勢中回檔（最常用）| | B3 假跌破買點 | 均線上揚，價格短暫跌破後快速收回 | 主力洗盤時 | | B4 反彈買點 | 價格急跌造成負乖離過大，反彈回歸均線 | 空頭急跌搶反彈 |

    四大賣點: - S1 跌破賣點：均線下彎，價格跌破均線 - S2 壓力賣點：均線下彎，價格反彈到均線無法突破（最重要） - S3 假突破賣點：均線下彎，價格短暫突破後又跌破 - S4 反轉賣點：價格急漲造成正乖離過大，反轉下跌

    核心心法: B2、B3是最實用的買點（趨勢確認後進場），均線方向比價格位置更重要。

    9. 跳空缺口操作

    四種缺口型態:

    | 缺口 | 定義 | 操作 | |------|------|------| | 普通缺口 | 盤整區間內，無明顯趨勢 | 忽略 | | 突破缺口 | 突破盤整區，成交量放大 | 進場訊號 | | 逃逸缺口 | 趨勢進行中，再出現跳空 | 繼續持有 | | 竭盡缺口 | 趨勢末段，爆量但價格停滯 | 出場訊號 |

    判斷關鍵: 2-3天內是否回補。突破了未回補 = 真突破；快速回補 = 假突破。

    島狀反轉: 左邊向上跳空 + 右邊向下跳空 = 做空好機會

    10. 融資融券操作

    融資（借錢買股）: - 上市：自備40%可買100% - 上櫃：自備50%可買100% - 槓桿約2.5倍，利息6-7%/年 - 適合：短期看好、當沖 - 風險：維持率<130%斷頭

    融券（借股賣出）: - 自備90%保證金 - 最大虧損無上限（股價上漲沒有上限） - 適合：短期看空 - 風險：軋空、強制回補（除權息前、股東會前）

    填息波段操作: - 篩選快速填息常勝軍（10年平均2個月內填息） - 大盤多頭時，除息日買進 - 微星2021實例：除息後2天填息，一個月漲20%+

    放空心法:

  • 不宜長線作戰（下跌快但短暫）
  • 不要執著於邏輯（邏輯對也可能被軋空）
  • 不要陷入幻想（別覺得莊家在盯你）

    11. 當沖進場訊號

    K線順勢操作: - 連續紅K（第2根紅K）→ 做多 - 連續黑K（第2根黑K）→ 做空 - 出現反向K線就停利

    均線順勢操作: - 紅K突破且站上均線 → 做多 - 黑K跌破均線 → 做空 - 價格跌破/突破均線才出場

    停損設定: 最近兩根K棒高低點 ± 5點緩衝

    移動停利: - 達到目標（如80點）先出一半 - 剩下一半用均線移動停利 - 6根K棒高低點 + 100點

    12. 選擇權策略

    賣方策略核心: - 勝率高（85%+），但偶爾毀滅性虧損 - 賣價外選擇權賺時間價值 - 資金要充足（一口至少5-10萬） - 絕對不能加碼攤平

    Iron Condor（鐵兀鷹）: - 同時賣出CALL價差單 + PUT價差單 - 不預測漲跌，賭股價在區間內震盪 - Delta中立建倉，獲利來自Theta衰減 - 最大獲利 = 淨權利金，最大虧損 = 價差寬度 - 淨權利金 - 風險管理：虧損達權利金2倍停損、移動式平倉重複建倉

    Theta時間價值策略: - 價平選擇權Theta最大 - 越接近到期，Theta衰減越快 - 結算週是時間價值消失最快的時候 - 賣方賺Theta：平均每天收10點，3天收30點 - 買方付Theta：時間是敵人

    Covered Call（掩護性買權）: - 持有現貨 + 賣出Call選擇權 - 「用股票收租」的概念 - 適合：盤整緩漲市場、長期持有部位 - 風險：大漲時獲利有上限 - 不適合：成長股、中小型股（容易暴漲）

    Straddle vs Strangle:

    | 策略 | 操作 | 使用時機 | 風險特性 | |------|------|----------|----------| | Long Straddle | 買Call + 買Put（同履約價）| 預期大波動 | 損失有限，成本高 | | Short Straddle | 賣Call + 賣Put（同履約價）| 預期盤整 | 勝率高，風險無限 | | Long Strangle | 買Call + 買Put（不同履約價）| 預期大波動 | 成本較低，需更大波動 | | Short Strangle | 賣Call + 賣Put（不同履約價）| 預期盤整 | 勝率高，風險更大 |

    13. 資金控管：2% + 6% 規則

    2% 規則: - 單筆風險 = 總資金 × 2% - 可買股數 = 2% 風險上限 ÷ (買價 - 停損價) - 每月初重新計算資金

    6% 規則: - 總資金從前月末下跌6%時，立即停止當月所有交易 - 平掉所有未平倉部位 - 月底前只觀望、不交易

    為什麼需要: - 2% 規則只防「鯊魚」（一次致命虧損） - 6% 規則防「食人魚群」（連續小虧損） - 連敗代表策略可能失效，需要停下來檢查

    14. 部位操作（多空雙作）

    核心概念: 試水溫 + 避險，降低「看錯方向」的風險。

    情境一：股價朝預期反向走

  • 看好買進1張（試水溫）
  • 股價下跌出現黑K
  • 加放空1張避險
  • 續跌：停損買單、加碼空單
  • 續漲：停損空單、加碼多單
  • 核心：不管怎麼走，最大虧損都可控

    • 情境二：股價朝預期方向走 - 可以加碼同方向部位 - 但不要無限制加碼

    16. 權證操作實戰

    權證選擇五大關鍵:

  • 剩餘天數 > 100天
  • 價外 5-20%
  • 權證價格 0.7元左右
  • 買賣量正常，流動性佳
  • 造商品質穩定

    • 權證優缺點:

    | 優點 | 缺點 | |------|------| | 門檻低（幾千元）| 時間價值遞減 | | 槓桿效果大 | 波動性大（可能一天跌50%）| | 虧損有限（最大賠光）| 到期歸零風險 | | 無斷頭風險 | 流動性風險 |

    誰適合: 懂遊戲規則、有明確觀點、資金控管紀律、能承受波動

    不適合: 完全不懂新手、只因錢少、想長期持有、沒方向判斷

    17. VWAP成交量加權平均價格

    公式: 

    VWAP = Σ(價格 × 成交量) / Σ成交量

    實戰應用: - VWAP以上：當日買方成本區，支撐較強 - VWAP以下：當日賣方成本區，壓力較重 - 價格在VWAP之上 → 偏多操作 - 價格在VWAP之下 → 偏空操作

    VWAP回調策略: 價格拉回到VWAP附近出現止跌跡象 → 進場做多
