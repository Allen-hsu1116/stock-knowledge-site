五、回測驗證

定義：用歷史數據測試交易策略的過去表現

為什麼需要回測： - 驗證假說、排除無效策略 - 了解策略風險報酬特性 - 了解策略在不同行情下的表現 - 提高策略信心 - 減少人性對投資的影響

回測五步驟：

| 步驟 | 說明 | |------|------| | 1. 了解獲利來源 | 你在賺什麼錢？為什麼策略長期可行？ | | 2. 清楚定義策略 | 進場條件、濾網、出場條件、風險控制、資金控管 | | 3. 收集歷史數據 | 價格、指標、財務數據，時間需足夠長 | | 4. 進行回測 | 套用歷史數據模擬交易 | | 5. 結果分析 | 評估報酬率、勝率、MDD、夏普率等 |

有效策略的判斷指標： - 績效曲線從左下到右上 - 平均單筆獲利扣除成本後仍足夠 - 每年正報酬或有明確原因的負報酬年度 - MDD 不會對帳戶造成破壞性打擊

回測核心指標：

| 類型 | 指標 | |------|------| | 收益 | 總報酬率、年化報酬率、平均報酬 | | 風險 | MDD、波動率、夏普率、索丁諾比率 | | 勝率盈虧 | 勝率、盈虧比 |

回測限制： - 過去績效 ≠ 未來績效 - 過度最佳化風險（overfitting） - 交易成本與滑價影響 - 未來函數問題（用到未來資料）
