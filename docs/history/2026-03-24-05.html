怎麼做：

  1. 搜尋公司老闆的新聞
  2. 查公開資訊觀測站的董監質押情形
  3. 質押成數>80% = 很缺錢
實例： - 情境：和旺(5505) 2015 - 數字：董監質押從14% → 100% - 判斷：質押逐年增加 = 需錢孔急

警訊2：經營階層變動

是什麼：財務長、獨立董事、董事接連離職。

怎麼做：

  1. 注意財務長離職訊息
  2. 注意獨董請辭訊息
  3. 離職順序：財務長 → 獨董 → 一般董事 → 董事長
適用時機：公司負債比高 + 財會主管離職

警訊3：股價異常飆漲

是什麼：股價漲幅遠超同業且無基本面支撐。

怎麼做：

  1. 比較同業股價表現
  2. 技術與規模相近的公司，股價不應差太多
  3. 漲到「玉山高度」= 可能炒股

警訊4：市場利多消息不斷

是什麼：公司頻繁發布利多，常澄清「不予置評」。

怎麼做：

  1. 查公開資訊觀測站的「重大訊息」
  2. 若利多澄清連續多次 = 公司自己放的
  3. 目的：炒股票
實例： - 情境：仕欽(6232) - 操作：對外宣稱手機訂單大增、大廠有意入主 - 結果：頻繁發布更正或澄清，最後證明子虛烏有

警訊5：借殼後初期階段

是什麼：借殼上市後3-4年內，營收驚人成長但現金流為負。

怎麼做：

  1. 查公司是否借殼上市
  2. 借殼後營收連續成長 >50% = 警訊
  3. 營業現金流連續為負 = 更大警訊
實例： - 情境：揚華(4703)前身「金美克能」 - 數字：2012借殼後，營收從6.3億 → 14.7億 → 29.2億（成長91% → 136% → 98%） - 同時：營業現金流都是負的

適用時機：投資借殼3年內的公司
