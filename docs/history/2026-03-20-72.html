一、股票投資的八大風險類型

1. 市場風險（系統性風險） - 定義：影響整個股票市場的宏觀因素帶來的風險 - 特徵：無法透過分散投資完全消除 - 成因：經濟衰退、地緣政治衝突、央行加息等 - 影響：大部分股票價格一同下跌，優質藍籌股也難倖免 - 應對：資產配置、降低倉位、持有現金

2. 公司風險（非系統性風險） - 定義：個別公司獨有的風險 - 成因： - 管理層決策失誤 - 新產品開發失敗 - 陷入法律訴訟 - 財務報表問題 - 特徵：對整體市場影響有限 - 應對：分散投資於不同行業、不同公司

3. 流動性風險 - 定義：想賣出股票時，市場上沒有足夠買家承接 - 特徵： - 無法以合理價格迅速成交 - 需要大幅折價才能賣出 - 常見情境：每日成交量稀疏的細價股或興櫃股票 - 應對： - 選擇成交量大、流動性佳的標的 - 避免重倉低流動性股票 - 預留變現緩衝

4. 利率與貨幣風險 - 利率風險： - 利率上升 → 企業融資成本增加 → 盈利受壓 - 利率上升 → 儲蓄/債券吸引力增加 → 資金流出股市 - 貨幣風險： - 持有海外股票時，匯率波動影響最終回報 - 美股上漲但美元貶值 → 換算後報酬打折 - 應對： - 關注央行利率政策 - 考慮匯率避險策略

5. 政策與監管風險 - 定義：政府政策轉變或監管新規例對市場造成衝擊 - 實例： - 中國對教育、科網行業監管收緊 → 相關港股劇烈波動 - 產業政策變化影響特定板塊 - 應對： - 密切關注政策風向 - 評估政策對持股的潛在影響 - 避免重押政策敏感板塊

6. 信用風險 - 定義：公司財務困難無法償還債務的風險 - 特徵： - 股東權益排在債權人之後 - 公司破產清算時可能血本無歸 - 應對： - 審查公司財務報表 - 關注債務比率與現金流 - 避開高負債公司

7. 操作風險 - 定義：公司內部流程、人員及系統的失誤或失效 - 成因： - 嚴重生產事故 - 核心技術人員流失 - 內部欺詐 - 數據系統被黑客攻擊 - 影響：嚴重影響公司營運及聲譽，拖累股價 - 應對： - 評估公司治理品質 - 分散投資，避免單一公司風險

8. 資訊不對稱風險 - 定義：機構投資者和內部人員比散戶擁有更多、更及時的資訊 - 特徵： - 散戶在交易中處於不利位置 - 知情者可能先行買賣 - 散戶在消息公布後才反應，蒙受更大損失 - 應對： - 持續學習，提升認知 - 深入研究公司基本面 - 不追逐市場消息和「貼士」
