融資融券變化判讀

是什麼：融資 = 散戶借錢買股（看多）；融券 = 散戶借股賣出（看空）。融資融券餘額變化，反映散戶情緒。

怎麼做：

  1. 融資增加、融券減少 → 散戶看多
  2. 融資減少、融券增加 → 散戶看空
  3. 融資爆增 + 股價不漲 → 可能是誘多（主力出貨）
  4. 融券暴增 + 股價不跌 → 可能有軋空行情
實例： - 情境：某股票融資餘額暴增 30%，但股價沒漲 - 判斷：散戶大量湧入，但主力可能在出貨 - 警訊：注意主力對作風險

適用時機： - ✅ 判斷散戶情緒 - ✅ 搭配主力籌碼看是否對作 - ❌ 現在大戶也會用融資融券，要謹慎解讀
