六、對沖基金的主要策略

1. 股票多空策略（Long/Short Equity） - 做多看好的股票 + 做空看差的股票 - 淨部位可以是偏多、偏空或中性 - 目的：降低市場風險，獲取選股報酬

2. 市場中立策略（Market Neutral） - 做多與做空金額相等 - 淨部位接近零 - 目的：完全消除市場風險 - 報酬來源：多空部位的相對表現

3. 事件驅動策略（Event Driven） - 利用企業併購、重整、破產等事件 - 高風險高報酬 - 需要深入研究企業財務

4. 全球宏觀策略（Global Macro） - 根據全球經濟趨勢配置資產 - 可以操作股票、債券、貨幣、商品 - 需要對宏觀經濟有深入了解
