選擇權基本概念

選擇權定義

是什麼：一種契約，買方支付權利金取得「未來以約定價格買賣標的物」的權利（非義務）。賣方收取權利金，但有履約義務。

四大操作象限：

  1. 買進買權（Buy Call）：看大涨，支付權利金
  2. 賣出買權（Sell Call）：看跌或盘整，收權利金、有義務
  3. 買進賣權（Buy Put）：看大跌，支付權利金
  4. 賣出賣權（Sell Put）：看漲或盤整，收權利金、有義務
實例： - 情境：台股指數 18000 點 - 操作：買進履約價 18200 的 Call，權利金 50 點 - 成本：50 × 50 = 2500 元 - 到期後：若指數漲到 18500，獲利 = (18500-18200-50) × 50 = 12500 元 - 最大虧損：2500 元（權利金全額）

價內、價平、價外

是什麼：履約價相對於現貨價格的位置

怎麼判斷（以買權Call為例）： - 價平（ATM）：履約價 = 現貨價 - 價內（ITM）：履約價 < 現貨價（立即履約有利可圖） - 價外（OTM）：履約價 > 現貨價（立即履約會虧損）

注意：Call 和 Put 的價內/價外是相反的！ - Call 價內 = Put 價外 - Call 價外 = Put 價內

選擇權價值結構： - 權利金 = 內在價值 + 外在價值（時間價值 + 波動值） - 價外選擇權：只有外在價值，權利金較便宜 - 價內選擇權：有內在價值，權利金較貴
