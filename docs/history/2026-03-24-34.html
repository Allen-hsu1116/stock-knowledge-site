2026-03-24 · 💰 基本面分析 🧠 交易心理

心得：

這篇補上了一個基本面分析的重要維度：如何評估管理階層。過去學的多是財務數字（本益比、ROE、現金流），但公司最終是由人經營的。

巴菲特說「投資就是投人」，這話不是空話。他推崇的8位CEO，知名度遠不如Jobs、Musk、Welch，但他們的長期回報率遠超這些明星CEO。為什麼？因為他們專注在「資本配置」這件事，每一塊錢都要求明確報酬。

關鍵洞察：

  1. 好CEO不談願景，談報酬 — 願景是畫大餅，報酬是實績
  2. 好CEO不是追求成長神話 — 每一筆投資都要經得起數字檢驗
  3. 好CEO敢於逆向 — 別人恐慌時買進，別人狂熱時冷靜
  4. 好CEO保持沉默 — 報酬會說話，不需要話術
評估CEO的實戰方法：
  1. 查薪酬結構（固定 vs 績效 vs 選擇權）
  2. 查持股變動（持續賣出 = 不看好自家公司）
  3. 查併購績效（創造價值還是毀滅價值）
  4. 查買回時機（低買還是高買）
  5. 查媒體曝光（太多願景 = 缺乏紀律）
這些資訊都在年報、公開資訊觀測站可以查到。跟查財報一樣，是「免費」的盡職調查。

「投資就是投人」這句話要落實到行動，不是說說而已。
