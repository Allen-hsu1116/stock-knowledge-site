四、行為金融學核心洞見

1. 展望理論（Prospect Theory） - 核心發現： - 失去一百元的痛苦 > 得到一百元的快樂（約2倍） - 人們在面對收益時傾向風險規避 - 在面對損失時傾向風險尋求 - 對稱性：決策行為在收益和損失情境下不對稱

2. 處置效應（Disposition Effect） - 定義：投資者傾向於太快賣掉上漲股票，卻長期持有下跌股票 - 心理機制： - 賣掉上漲股票 → 鎖定收益，規避未來下跌風險 - 持有下跌股票 → 不願承認失敗，期待反彈回本 - 與理性原則矛盾： - 理性：「讓利潤奔跑，及時停損」 - 實際：「賣盈守虧」 - 應對： - 建立停損紀律，嚴格執行 - 讓獲利股票有更多上漲空間 - 不因帳面虧損就拒絕出場

3. 參考點依賴與框架效應 - 參考點依賴： - 對結果是收益還是損失的判斷，取決於選擇的參考點 - 參考點通常是買入成本或初始預期 - 框架效應： - 同樣的客觀結果，不同描述方式會改變決策 - 描述為「收益」→傾向風險規避 - 描述為「損失」→ 傾向風險尋求 - 實例： - 方案 A 能確定救活 200 人 → 多數人选擇 - 方案 C 會確定導致 400 人死亡 → 多數人不選 - 實際上 A 和 C 結果完全相同，只是描述框架不同 - 應對： - 警惕措辭方式可能帶來的心理誘導 - 評估資訊時剝離描述框架

4. 其他重要認知偏誤 - 錨定效應：過度依賴初始資訊（如股票歷史高點、買入成本） - 後見之明偏誤：事後總覺得自己早就預見了結果 - 模糊厭惡：傾向選擇風險已知（即使機率不利）的選項
