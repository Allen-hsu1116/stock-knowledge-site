優秀CEO的8大特質（巴菲特推崇）

特質1：效率決定生死

是什麼：不是追求KPI效率，而是透過授權激發內驅力。

怎麼做：

  1. 避免中心化管理（所有決策都要總部同意）
  2. 充分授權，讓員工發揮創造力
  3. 讓部門自給自足，無法自給的立即處理
實例： - 情境：大都會廣播公司 vs 哥倫比亞廣播公司 - 大都會：去中心化管理，總部沒有層層官僚，收視率第一 - 哥倫比亞：中心化管理，光總裁+副總裁就42人，處處需要總部同意 - 結果：皮划艇打敗大船

適用時機：評估公司組織效率，看是否「人治」還是「制度」

特質2：聚焦資本配置

是什麼：CEO最重要的工作是資本配置，每一塊錢都要有明確報酬。

怎麼做：

  1. 不投資「會發光」的標的，投資「能穩定創造現金」的資產
  2. 不做年度預測，不靠願景決策
  3. 像會計師一樣看公司，像資本家一樣決策
  4. 每筆投資都要經得起時間與數字檢驗
實例： - 情境：Tom Murphy（大都會廣播公司CEO） - 策略：只收購能穩定創造現金的資產 - 結果：成為美國最賺錢的媒體公司之一

適用時機：評估CEO的併購、投資決策紀錄

特質3：綁定股東利益

是什麼：CEO利益與股東一致，不是為了自己薪酬。

怎麼判斷：

  1. 查CEO持股比例（太少 = 激勵不足）
  2. 查薪酬結構（固定薪資 vs 績效獎金 vs 股票選擇權）
  3. 查是否長期持有（賣股 vs 持有）
  4. 查是否有「金降落傘」（離職金過高）
適用時機：評估管理階層是否「同在一艘船」

特質4：高效運用槓桿

是什麼：不是不敢用槓桿，是知道何時用、如何用。

怎麼做：

  1. 用槓桿買「穩定現金流」的資產
  2. 不用槓桿做高風險擴張
  3. 保持安全邊際，預留現金度過寒冬
  4. 稻盛和夫：「企業會遇到各種問題，要留足夠現金生存」
實例： - 情境：大眾影院公司CEO Dick Smith - 策略：售出租回（sale-leaseback），賣影院給投資人再租回經營權 - 結果：手上有充足現金面對市場波動

適用時機：評估公司財務槓桿是否適當

特質5：勇於逆向前行

是什麼：在市場恐慌時買進，在市場狂熱時保持冷靜。

怎麼做：

  1. 不問未來會變怎樣，問現在該做什麼
  2. 像撲克玩家：手握資本，等待市場錯價
  3. 時機成熟就出手，不成熟就讓資金留在口袋
  4. 承擔風險但不盲目賭博
實例： - 情境：華盛頓郵報CEO凱瑟琳·格雷厄姆 - 背景：經濟蕭條，同行拋售資產 - 操作：逆勢買入電視台、移動通信公司、教育公司 - 結果：新業務創造公司一半收入和利潤

適用時機：看CEO在危機時的反應

特質6：數據思維制勝

是什麼：用數據做決策，不靠直覺或願景。

怎麼做：

  1. 決策前看數據，不憑感覺
  2. 追蹤關鍵指標並持續優化
  3. 數據驅動而非KPI驅動
適用時機：評估CEO是否有數據思維習慣

特質7：永遠關注現金流

是什麼：現金是企業的血，沒有現金就沒有未来。

怎麼做：

  1. 檢視自由現金流的穩定性
  2. 現金流為負要看原因（擴張還是虧損）
  3. 利潤不等於現金，有現金才能活下去
  4. 預留至少3年收入中斷的現金準備
實例： - 情境：湛廬文化 - 策略：CEO要求「沒有收入也能維持3年」的現金儲備 - 結果：疫情期間沒裁員、沒減薪，正常運作

適用時機：評估公司的現金部位是否健康

特質8：首席執行官就是首席投資官

是什麼：CEO做的每一個決策都是投資決策。

怎麼做：

  1. 不喊口號，把自由現金流最大化
  2. 不賺錢的部門敢於果斷處理
  3. 把價值還給股東（買回或配息）
  4. 不為了「偉大」而擴張，為了「報酬」而配置
實例： - 情境：Bill Anders接手General Dynamics - 操作：接管龐大、繁複、效率低的軍工集團 - 策略：大幅出售不賺錢或不熟悉的業務，把現金拿來回購股票 - 結果：市值倍增，股東回報成為事實

適用時機：看CEO是否有「投資人思維」
