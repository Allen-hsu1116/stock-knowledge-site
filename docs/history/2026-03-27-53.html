四、凱利公式的假設與限制

核心假設

  1. 準確的機率與賠率：必須準確知道勝率和賠率
  2. 可重複的獨立試驗：機會可無限次或大量重複
  3. 資金可無限分割：可精確按比例投入
  4. 利潤再投資：利潤複利計入本金
  5. 長期視角：不保證短期結果

主要風險

1. 估計錯誤風險 - 高估勝率或賠率 → 過度投注 → 破產風險大增 - 人類容易過度自信 - 歷史數據不一定預測未來

2. 高波動性 - 全凱利可能帶來劇烈資金波動 - 可能經歷深度回撤（從高點下跌） - 波動性本身會侵蝕複利增長

3. 市場動態性 - 真實市場勝率和賠率會變動 - 不同資產間存在相關性 - 基礎公式未考量相關性

4. 黑天鵝風險 - 無法預測極低機率但影響巨大的事件 - 金融危機、戰爭、疫情等
