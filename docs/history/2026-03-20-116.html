四、其他風險量化指標

1. 標準差（Standard Deviation） - 衡量資產價格的波動程度 - 標準差越大，波動越大 - 缺點：上行波動和下行波動一視同仁

2. Beta 值 - 衡量資產相對於大盤的波動 - Beta > 1：資產比大盤波動大 - Beta < 1：資產比大盤波動小 - 缺點：只考慮相對波動，忽略絕對風險

3. Alpha 值 - 衡量資產相對於大盤的超額報酬 - Alpha > 0：資產表現優於大盤 - Alpha < 0：資產表現劣於大盤

4. 夏普比率（Sharpe Ratio） - 公式：(報酬率 - 無風險利率) / 標準差 - 衡量每單位風險的報酬 - 夏普比率越高，風險調整後報酬越好

5. 索提諾比率（Sortino Ratio） - 公式：(報酬率 - 無風險利率) / 下行標準差 - 只考慮下行風險，更符合投資人關注虧損的心理
