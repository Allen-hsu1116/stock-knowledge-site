六、MDD 的核心意義

1. 回撤恢復難度

| 回撤幅度 | 恢復所需漲幅 | |---------|------------| | 10% | 11% | | 20% | 25% | | 30% | 43% | | 40% | 67% | | 50% | 100% | | 60% | 150% | | 70% | 233% | | 80% | 400% | | 90% | 900% |

關鍵洞察： - 回撤 50% 需要漲 100% 才能回本 - 回撤越大，恢復難度呈指數級增加 - 控制回撤幅度是資金管理的核心

2. MDD 告訴投資者什麼？ - 最糟糕情況下的可能虧損 - 跌多久（回撤持續時間） - 多久能回本（恢復時間）

3. 投資績效評估 - 年化報酬相同時，MDD 較低的策略更優 - 用較小的風險換取相同的報酬
