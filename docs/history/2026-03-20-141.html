一、什麼是交易系統？

1. 定義 - 交易系統：一套明確定義的規則，告訴交易者何時進場、何時出場、何時停損停利 - 核心目的：消除主觀判斷，讓交易可以重複執行 - 關鍵特徵：可定義、可回測、可執行

2. 交易系統的組成要素

| 要素 | 說明 | 是否必須 | |------|------|---------| | 進場規則 | 何時買進 | 必須 | | 出場規則 | 何時賣出 | 必須 | | 停損規則 | 虧損多少出場 | 必須 | | 停利規則 | 獲利多少出場 | 建議 | | 倉位管理 | 每次投入多少資金 | 必須 | | 風險管理 | 單筆最大虧損、總體 MDD 上限 | 必須 |

3. 為什麼需要交易系統？ - 消除情緒：依照規則執行，不受恐懼貪婪影響 - 可重複執行：每次條件相同，操作相同 - 可回測驗證：可以用歷史數據驗證有效性 - 可持續改進：知道哪裡出問題才能修正
