六、均值回歸的力量

1. 定義與原理 - 定義：極端現象傾向回歸平均水平的自然機制 - 起源：高爾頓研究豌豆遺傳時發現 - 核心現象： - 非常高的父母，子女傾向比父母矮一些 - 非常矮的父母，子女傾向比父母高一些 - 商業應用： - 行業或公司獲得超高回報率 → 吸引競爭者 - 資本湧入 → 產能增加 → 價格下降 → 回報率回歸

2. 投資績效的均值回歸 - 發現：短期「冠軍」基金，隨後績效常回落到平均水平 - 研究證據： - 過去三年表現最好的股票（贏家組合） - 過去三年表現最差的股票（輸家組合） - 未來三年：「輸家組合」平均報酬超越市場 - 「贏家組合」表現落後於市場 - 啟示： - 不應過度迷信短期靚麗業績 - 關注長期、可持續的投資邏輯 - 風險控制能力比短期報酬更重要
