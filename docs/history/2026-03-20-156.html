六、Beta 值與 Alpha 值

1. Beta 值

定義：衡量資產相對於大盤的波動程度

解讀： - Beta = 1：與大盤同步波動 - Beta > 1：波動比大盤大（高風險高報酬） - Beta < 1：波動比大盤小（低風險低報酬） - Beta < 0：與大盤反向（如放空部位）

實務應用： - Beta 高的股票適合多頭市場 - Beta 低的股票適合空頭市場或保守配置

2. Alpha 值

定義：衡量資產相對於大盤的超額報酬

解讀： - Alpha > 0：表現優於大盤 - Alpha = 0：表現與大盤相同 - Alpha < 0：表現劣於大盤

實務應用： - Alpha 來自選股能力 - 主動型基金目標是創造正 Alpha - 被動型基金目標是追蹤大盤（Alpha = 0）
