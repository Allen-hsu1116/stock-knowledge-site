二、對沖交易 vs 套利交易（根本差異）

| 比較維度 | 對沖交易 (Hedging) | 套利交易 (Arbitrage) | |---------|-------------------|---------------------| | 核心目標 | 降低或鎖定現有頭寸的風險 | 利用定價錯誤賺取無風險利潤 | | 風險態度 | 風險規避 | 風險中性 | | 操作前提 | 已持有一個風險敞口 | 發現同一資產在不同市場存在價差 | | 利潤來源 | 減少虧損來保護資產 | 從價差中直接獲利 | | 常見例子 | 持有股票 + 買入認沽期權 | A 交易所買入 $10，B 交易所賣出 $10.05 |

關鍵差異： - 對沖 = 「避險」，保護資產 - 套利 = 「無風險獲利」，從價差賺錢
