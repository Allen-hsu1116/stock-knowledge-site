步驟二：持有期間的記錄（超過 90% 投資人沒做到）

關鍵觀念：下單完後立刻訂下規則，避免盤中思考

應該設定的規則：

時間停損點： - 在 X 月 X 號，如果價格沒到 X 元，就賣出 - 例：買進後 5 個交易日，若未突破前高，賣出

空間停損點： - 百分比停損法：虧損 7% 停損 - 均線停損法：跌破 5 日均線停損 - 支撐停損法：跌破關鍵支撐停損

最大可能虧損： - 通常設定比停損更高（因為大盤可能突然暴跌） - 例：停損設 7%，但心理準備最大虧損 10%

何時看盤決定賣出？ - 開盤 9:00-9:30？ - 尾盤 13:00-13:30？ - 盤中逢高賣出？ - 提前決定，盤中執行

停利點設定： - 股價帳面獲利超過 X 元 → 停利 - 股價到 X 價位 → 停利 - 分批停利：第一目標達成賣一半，第二目標達成全出

加碼點設定： - 股價突破 X 價格 → 再買一張 - 帳面獲利 X 元 → 再買一張 - 金字塔加碼：每次加碼金額遞減

極端狀況應對（保險）： - 大盤狂跌千點，個股瞬間穿過停損 → 該賣嗎？ - 賣不掉跌破停損 → 該怎麼辦？ - 永遠為任何狀況做好準備，否則交易就沒有永遠

記錄心情： - 每天持股都要寫 - 像日記般隨興發揮 - 記錄當下的感受 - 不用批判自己，只要記錄

為什麼記錄心情重要？ - 交易 = 投資知識 + 心理學 - 很多時候知識正確，但價格波動時控制不住 - 記錄心情才能識別自己的情緒觸發點 - 未來遇到類似情況時提醒自己不要衝動
