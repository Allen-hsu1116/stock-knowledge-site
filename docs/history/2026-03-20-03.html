二、ATR 停損策略（動態風險管理）

二、ATR 停損策略（動態風險管理）

1. ATR 固定倍數停損法 - 具體操作：停損點設在進場價 ± N × ATR - 步驟： 1. 確認進場訊號（均線交叉、趨勢線突破等） 2. 讀取進場時的 ATR 數值（如 ATR(14) = 70 pips） 3. 計算停損距離 = ATR × 倍數（常用 1.5~3 倍） 4. 做多停損價 = 進場價 - 停損距離 5. 做空停損價 = 進場價 + 停損距離 - 實例： - 黃金在 2350.00 做多，ATR(14) = 25.00 - 用 2 倍 ATR 停損 - 停損距離 = 25.00 × 2 = 50.00 - 停損價 = 2350.00 - 50.00 = 2300.00

2. 為什麼用 ATR 倍數而非固定點數？ - 高波動市場：固定 50 pips 停損容易被「洗盤」出場 - 低波動市場：固定 50 pips 停損又過寬，風險過大 - ATR 解決方案：讓停損「動態適應」當前市場波動 - 波動大 → ATR 大 → 停損自動放寬 - 波動小 → ATR 小 → 停損自動收緊

3. ATR 停損倍數選擇建議 - 1.5 倍 ATR：適合短線、當沖，停損較緊 - 2 倍 ATR：最常用，平衡風險與被洗出的機率 - 3 倍 ATR：適合波段、長線，給足空間
