二、橋水基金全天候策略（All Weather）

是什麼：Ray Dalio 設計的策略，目標是在任何經濟環境下都能穩定表現。核心是「配置風險，而不是配置資金」。

四個經濟節氣： | 經濟環境 | 受惠資產 | |---------|---------| | 成長高於預期 + 通膨高於預期 | 股票、大宗商品 | | 成長低於預期 + 通膨高於預期 | 黃金、大宗商品 | | 成長高於預期 + 通膨低於預期 | 股票、債券 | | 成長低於預期 + 通膨低於預期 | 債券 |

經典配置： - 30% 股票（SPY） - 40% 長期債券（TLT） - 15% 中期債券（IEF） - 7.5% 黃金（GLD） - 7.5% 大宗商品（DBC）

失效風險： - 股債相關性上升（同跌） - 債券波動度變大 - 利率上升環境 - 槓桿帶來的額外風險

2020新冠疫苗期間表現不佳原因： 各類資產都在劇烈波動，依照波動率越大的投資越要減碼的邏輯，被迫無差別減持所有資產。
