風險貢獻分解

投資組合變異數分解： 

V_p = Var(Σ β_ip × F_i) + Var(ε_p)
- 第一項 = 因子曝險解釋的變異數 - 第二項 = 無法解釋的變異數（特有風險）

實例： - 若投資組合 = 市場投資組合，則 Beta = 1，所有變異數都由市場因子解釋 - 若偏離市場，還會有其他因子曝險和特有風險

應用： - 經理人的技能若來自特定因子（如價值、動能），應最小化「非因子解釋」的風險 - 選股經理人：最小化因子風險，最大化選股 Alpha - 因子經理人：最大化目標因子曝險，最小化其他因子和特有風險
