四、VaR 的限制與補強

VaR 的限制：

  1. 未考量超門限損失：VaR 只告訴你最大預期損失，不告訴你超過這個損失會怎樣
  2. 不滿足次可加性：不是一致風險測度，可能出現投資組合 VaR 大於個別資產 VaR 之和
  3. 模型風險：參數估計誤差會影響結果
  4. 常態分配假設：實際市場報酬有厚尾現象
補強方法：

1. 壓力測試（Stress Testing） - 分析市場出現異於常態的極端情況 - 模擬歷史重大事件（如金融海嘯）的衝擊 - 評估投資組合在極端壓力下的表現

2. 情境分析（Scenario Analysis） - 設計特定市場情境 - 評估不同壓力情境下的潛在損失

3. 回溯測試（Backtesting） - 驗證 VaR 模型的準確性 - 比較每日真實報酬與 VaR 預測 - 調整模型參數
