一、VaR（風險價值）基礎概念

1. VaR 是什麼？ - 全名：Value at Risk（風險價值） - 定義：資產組合在持有期間內，在給定的信賴區間內，由於市場價格變動所導致的最大預期損失的數值 - 核心功能：將所有風險彙整成一個數值，讓管理階層能簡單明瞭地知道資產組合可能面臨之最大損失 - 直觀解讀：「有 95% 的信心在未來 N 天內，最大損失不會超過 V 元」

2. VaR 的應用場景 - 金融機構資本適足率計算 - 企業內部風險控管 - 資產配置的參考依據 - 投資組合風險量化

3. VaR 的歷史背景 - 1980年代末至1990年代初，美國銀行業遭受商業風險困擾 - JP 摩根公司發明 VaR 方法 - 1996年巴塞爾修正協定將 VaR 作為市場風險衡量指標 - 芝加哥商品交易所於1988年推出 SPAN 系統，成為計算投資組合保證金的全球行業標準

4. VaR 與傳統風險測度的差異 - 傳統風險測度：以標準差為代表，關注整體波動 - VaR 優勢：著重資產下行風險，特別是極端情況下的損失規模
