四、斷頭風險與槓桿控制

斷頭定義： - 保證金帳戶低於維持保證金 - 券商強制賣出資產（強制平倉） - 也稱：爆倉、強制賣出、強制平倉

斷頭後果： - 個人：損失可能超過保證金，產生債務 - 市場：多殺多惡性循環

降低斷頭風險四方法：

| 方法 | 說明 | |------|------| | 降低槓桿倍數 | 放充足保證金（建議 100% 商品價值） | | 設定停損點 | 進場前設好停損，達到就出場 | | 分散風險 | 分批進場，不要 All In | | 自行賣出 | 主動賣出提高保證金，而非被動斷頭 |

台灣期貨斷頭門檻： - 盤中：風險指標低於 25% → 立即強制平倉全部部位 - 盤後：追繳後未補錢 → 次日強制平倉
