核心內容：

#### 一、ATR 指標基礎概念

1. ATR 是什麼？

    - **全名**：Average True Range（平均真實波幅） - **發明者**：J. Welles Wilder Jr.（RSI、ADX 的發明者） - **功能**：衡量市場波動率，不是預測方向 - **核心理念**：是一把「動態的尺」，測量特定時間週期內價格的平均波動幅度 - **解讀**：
- ATR 上升 = 市場波動加劇，價格上沖下洗 - ATR 下降 = 市場趨於平靜，價格波動收斂

2. True Range（真實波幅）計算公式

    - TR 取以下三者的最大值：
1. |當期最高價 - 當期最低價| 2. |當期最高價 - 前期收盤價| 3. |當期最低價 - 前期收盤價|
    - **為什麼要這樣算？**：傳統算法忽略跳空缺口，TR 將跳空也納入考量 - **ATR**：過去 N 期 TR 的移動平均（預設 14 期）
3. ATR 參數設定
    - **預設值**：14 週期（適用大多數情況） - **短線交易**：縮短至 5-7 週期，對近期波動更敏感 - **長線投資**：放寬至 20-25 週期，平滑短期雜訊
---

#### 二、ATR 停損策略（動態風險管理）

1. ATR 固定倍數停損法

    - **具體操作**：停損點設在進場價 ± N × ATR - **步驟**：
1. 確認進場訊號（均線交叉、趨勢線突破等） 2. 讀取進場時的 ATR 數值（如 ATR(14) = 70 pips） 3. 計算停損距離 = ATR × 倍數（常用 1.5~3 倍） 4. 做多停損價 = 進場價 - 停損距離 5. 做空停損價 = 進場價 + 停損距離
    - **實例**：
- 黃金在 2350.00 做多，ATR(14) = 25.00 - 用 2 倍 ATR 停損 - 停損距離 = 25.00 × 2 = 50.00 - 停損價 = 2350.00 - 50.00 = 2300.00

2. 為什麼用 ATR 倍數而非固定點數？

    - **高波動市場**：固定 50 pips 停損容易被「洗盤」出場 - **低波動市場**：固定 50 pips 停損又過寬，風險過大 - **ATR 解決方案**：讓停損「動態適應」當前市場波動
- 波動大 → ATR 大 → 停損自動放寬 - 波動小 → ATR 小 → 停損自動收緊

3. ATR 停損倍數選擇建議

    - **1.5 倍 ATR**：適合短線、當沖，停損較緊 - **2 倍 ATR**：最常用，平衡風險與被洗出的機率 - **3 倍 ATR**：適合波段、長線，給足空間
---

#### 三、ATR 追蹤停損法（Trailing Stop）

1. 追蹤停損原理

    - **目的**：保護已有利潤，同時抓住整個趨勢 - **做多的追蹤停損**：停損點 = 當前價格高點 - N × ATR - **做空的追蹤停損**：停損點 = 當前價格低點 + N × ATR - **關鍵規則**：停損點只能往有利方向移動，絕不後退
2. 具體操作步驟
    - 進場後：設定初始停損 = 進場價 - 2 × ATR - 價格上漲創新高：停損點上移至「新高點 - 2 × ATR」 - 價格下跌未破停損：停損不動，繼續持有 - 價格跌破停損：出場，鎖定利潤
3. 實例演示
    - 100 元買進，ATR = 3 元 - 初始停損 = 100 - 2×3 = 94 元 - 漲到 110 元：停損上移至 110 - 6 = 104 元 - 漲到 120 元：停損上移至 120 - 6 = 114 元 - 回跌到 114 元：觸及停損，出場，鎖定 14% 利潤
---

#### 四、ATR 資金管理（風險一致化）

1. 核心觀念

    - 不同市場、不同時期的波動度不同 - 用固定金額下單會導致：高波動時風險大、低波動時風險小 - 解決方案：用 ATR 調整倉位，讓每筆交易風險一致
2. 倉位大小計算公式 ``` 倉位大小 = (帳戶總資金 × 風險百分比) / (ATR 值 × 停損倍數 × 每點價值) ```

3. 實例計算

    - 帳戶資金：100 萬元 - 單筆風險：2%（2 萬元） - 目前 ATR：50 元 - 停損倍數：2 倍 ATR - 每點價值：1 元 - 倉位大小 = 2 萬 / (50 × 2 × 1) = 200 股
4. 風險一致化的效果
    - 高波動市場（ATR 大）：自動減少倉位 - 低波動市場（ATR 小）：自動增加倉位 - 每筆交易的最大虧損金額都控制在預設範圍（如帳戶的 1-2%）
---

#### 五、ATR 搭配其他指標的進階策略

1. ATR + ADX 趨勢強度確認

    - **ADX（平均趨向指標）**：衡量趨勢強度（非方向） - **ADX < 25**：無趨勢或趨勢弱，盤整 - **ADX > 25 且向上突破**：趨勢正在形成 - **策略**：
1. ADX 從低點向上突破 25 2. 確認價格方向（均線之上做多，之下做空） 3. 使用 ATR 設定停損（強趨勢下 ATR 較大，需放寬停損）

**2. ATR + 布林通道突破}

    - **布林通道帶寬收窄**：稱為「擠壓」，預示大行情 - **ATR 處於相對低位**：市場平靜，可能是暴風雨前的寧靜 - **策略**：
1. 布林通道帶寬收窄 + ATR 低 2. 等待價格強勢突破布林上下軌 3. ATR 開始上揚確認突破有效性 4. 進場，用 ATR 設定停損

---

#### 六、避險策略深度解析

1. 避險的核心意義

    - **定義**：將傷害控制在可接受範圍，不是追求零風險 - **類比**：買保險一樣，有成本，但能避免極端傷害 - **關鍵觀念**：
- 避險不一定帶來獲利或減少虧損 - 長期而言，能讓極端傷害不會發生 - 即使壞事沒發生，避險依然有用（就像沒火災不代表滅火器沒用）

2. 事前避險 vs 事後避險

    - **事前避險**：
- 根據風險預判，事先配置對抗風險的資產 - 例如：配置公債、黃金作為防禦資產 - 成本低、執行容易
    - **事後避險**：
- 意外發生後才減碼或調整配置 - 成本高、容易追高殺低 - 市場已超前反應，賣在低點、買在高點

3. 避險的成本分析

| 成本類型 | 說明 | 實例 | |---------|------|------| | 交易成本 | 手續費、交易稅 | 賣出股票再買回，付出兩次成本 | | 機會成本 | 避險期間錯過的報酬 | 2020/3 減碼後錯過反彈，少賺一段 |

4. 避險容易犯的錯誤

    - **追高殺低**：大跌後賣股票、買已漲高的避險資產 - **打破紀律**：沒計畫的避險 = 打破原有策略的藉口 - **忽略機會成本**：只看到「安心」，沒算付出的代價 - **過度避險**：想完全避開風險 = 錯失市場報酬
---

#### 七、如何規劃避險策略（三大重點）

1. 衡量自己的避險能力

    - **有判斷能力**：可根據時機設計動態避險策略 - **能力不足**：做好資產配置，讓組合本身有避險效果 - **關鍵**：像主動 vs 被動投資一樣，有能力才主動，沒能力就被動
2. 判斷執行避險的時機
    - **該避險的時機**：進場理由「可能改變」
- 投資底層假設改變 - 市場結構大變化 - 法規和參與者變動 - 過去資訊有誤
    - **不該避險的時機**：只因價格下跌或恐慌
- 進場假設沒變 - 隨風吹草動就改策略，等於沒策略

3. 在原策略中預先加入避險

    - **正確做法**：一開始就把各種風險的應對規劃在內 - **錯誤做法**：遇到意外才靈機一動想避險 - **原則**：
- 避險應該是「極少發生的偶發事件」 - 如果常常想避險 → 策略本身就有問題 - 或策略不符合你的風險承受能力

---

#### 八、避險的反向思考：承受風險

1. 投資本質是承受風險換取報酬

    - 定存、短期公債：幾乎不承受波動風險，報酬低 - 股票：短期波動大，長期報酬較高 - 高報酬來自「不確定性」
2. 應該問的問題
    - **錯誤問法**：「怎麼避開所有風險？」 - **正確問法**：「我願意承受怎樣的風險？」
3. 核心結論
    - 完全避開風險 = 錯失市場報酬（機會成本） - 好的投資組合 = 能承受各種意外風險 - 讓組合能承受風險，比每次意外都避險更重要
---

怎麼應用：

實戰操作步驟（ATR 部分）：

1. **進場前設定**： - 讀取當前 ATR(14) 數值 - 決定停損倍數（短線 1.5 倍，波段 2 倍，長線 3 倍） - 計算停損距離 = ATR × 倍數 - 計算停利距離（風險回報比 ≥ 1:1.5） - 用倉位公式計算下單股數：倉位 = (總資金 × 風險%) / (ATR × 停損倍數 × 每點價值)

2. **進場後執行**： - 設定停損價（做多：進場價 - 停損距離） - 價格上漲創新高 → 啟動追蹤停損（新高點 - N × ATR） - 停損只往有利方向移動，絕不後退 - 跌破停損 → 出場，不猶豫

3. **注意事項**： - ATR 滯後，對突發極端行情反應不夠及時 - ATR 無方向性，需搭配趨勢判斷指標 - 停損倍數需回測優化，無標準答案

實戰操作步驟（避險部分）：

1. **事前規劃**： - 列出所有可能的風險情境 - 為每個風險設定應對措施 - 把避險寫進投資策略的「標準程序」

2. **執行判斷**： - 進場理由改變 → 執行預設的避險措施 - 進場理由沒變 → 不要因恐慌打破紀律 - 交易成本 vs 機會成本都要算

3. **事後檢討**： - 評估避險是否有效 - 計算付出的機會成本是否值得 - 調整策略供下次使用

---

優點：

#### ATR 策略優點：

    - **客觀性**：量化的波動數據，避免主觀猜測 - **適應性**：自動適應市場波動，高波動停損寬、低波動停損緊 - **通用性**：適用各種金融商品、各種時間週期 - **資金管理**：可精確計算倉位大小，確保風險一致 - **動態追蹤**：追蹤停損既能保護利潤又能抓住趨勢
#### 避險策略優點：
    - **事前成本低**：事先規劃比臨時應變便宜 - **保護極端風險**：避免黑天鵝事件造成毀滅性傷害 - **維持紀律**：預設避險方案避免情緒化決策 - **控制傷害範圍**：將損失控制在可接受的範圍內
---

缺點/限制：

#### ATR 策略缺點：

    - **滯後性**：基於歷史數據，對突發極端行情反應不足 - **無方向性**：只告訴波動大小，不判斷漲跌方向 - **倍數主觀**：1.5 倍 vs 2 倍 vs 3 倍需自行回測 - **極端行情風險**：ATR 值過大時，停損點可能過寬
#### 避險策略缺點：
    - **機會成本**：避險期間錯過的報酬可能很高 - **可能追高殺低**：事後避險容易賣低買高 - **打破紀律風險**：沒規劃的避險 = 破壞原有策略 - **長期報酬代價**：可能每年損失 2-3% 報酬

核心內容：

#### 一、停損點設置的三種方法

1. 百分比停損法

    - **具體操作**：根據買進股價，設定下跌多少百分比時賣出 - **步驟**：
1. 計算停損價 = 買進價 × (1 - 停損百分比) 2. 百分比建議：7-10% 是較合理的區間 3. 設定太緊（如 3%）非常容易被洗掉
    - **範例**：
- 100 元買進，10% 停損 → 停損價 90 元 - 100 元買進，20% 停損 → 停損價 80 元
    - **注意**：百分比要根據個人風險承受能力調整
2. 移動停損法
    - **具體操作**：不根據買進價，而是隨股價上漲的「最高點」計算停損 - **步驟**：
1. 買進後記錄最高價 2. 停損價 = 最高價 × (1 - 停損百分比) 3. 股價創新高 → 用新高點重新計算停損價 4. 股價回落觸及停損價 → 賣出
    - **範例**：
- 100 元買進，設定 10% 移動停損 - 漲到 150 元：停損價 = 150 × 0.9 = 135 元 - 回跌到 135 元 → 觸發停損出場，鎖住 35% 利潤
    - **優點**：既能保護利潤，又能讓獲利奔跑
3. 支撐線停損法
    - **具體操作**：跌破技術分析的支撐線時賣出 - **步驟**：
1. 畫出股價趨勢圖 2. 找出支撐線（價格多次觸及反彈的低點連線） 3. 股價跌破支撐線且沒有反彈 → 停損出場
    - **關鍵判斷**：
-「碰到支撐線反彈」= 支撐有效，續持有 -「跌破支撐線不回頭」= 支撐失守，出場

4. 基本面停損法（買進理由消失）

    - **具體操作**：不看股價數字，而是檢視「買進理由」是否還成立 - **步驟**：
1. 記錄買進這支股票的理由（如：看好營收成長、政策利多、新產品） 2. 定期檢視公司財報、新聞、產業變化 3. 買進理由消失了（公司轉壞、產業變差）→ 賣出
    - **優點**：
- 較理性，不受股市情緒影響 - 符合價值投資思維
    - **缺點**：需要持續追蹤公司狀況，適合長線投資
---

#### 二、停損點設置的優缺點分析

優點：

    - **減少情緒波動**：心裡對最壞狀況有準備，不會因下跌緊張不安 - **控制風險**：停損點是你自己設定的，可根據財務狀況決定 - **避免重大虧損**：價格到停損點就賣出，不用擔心虧損越來越大
缺點：
    - **增加交易成本**：停損點設很緊，頻繁交易產生手續費 - **錯失上漲機會**：停損後股價上漲，捨不得買回就錯過 - **忽略公司體質**：只看價格數字波動，忘記檢視公司真實價值
重要觀念：
    - 長期投資人「不設停損」不會怎樣 - 設停損短期保護資產，長期可能錯過獲利 - 關鍵是先問自己「為什麼要投資這檔股票」 - 想清楚買進理由，比設停損更重要
---

#### 三、凱利公式（Kelly Criterion）- 科學化倉位管理

1. 凱利公式是什麼？

    - **定義**：在給定勝率與盈虧比下，計算出讓長期資本成長速度最快的投資比例 - **目的**：平衡「增長」與「生存」 - **核心信念**：獲利是勝率、盈虧比與倉位大小的乘積 - **關鍵洞察**：即便策略勝率 90%，下注規模超過凱利公式邊界，長期「歸零機率」趨近 100%
2. 系統運作邏輯

倉位優化（Position Optimization）：

    - 根據回測所得的期望值與當前策略信心度 - 自動計算建議投入比例 - 公式輸出的是「應該投入多少資金」
回撤控制（MDD Management）：
    - 凱利公式的目標：將最大回撤（MDD）控制在 10% 以內 - 即使連續虧損，整體權益曲線仍具恢復力
盈虧比綁定：
    - 系統要求預期盈虧比須大於 2.0 - 盈虧比 >= 2 才能用較低勝率獲利 - 確保長線獲利效率
3. 實戰案例分析

成功用法 - 在優勢區間放大收益：

    - 場景：系統偵測到「主升攻擊」機制，8 維度共識達成強共振（8/8） - 觀察：勝率與預期盈虧比同時處於高檔 - 凱利公式計算：給出較高的建議倉位 - 結論：在統計優勢時刻重拳出擊，低風險前提下實現跳躍式增長
失敗用法 - 過度押注導致毀滅風險：
    - 場景：投資者因連續獲利產生過度自信，無視凱利上限，全倉交易 - 觀察：市場突發黑天鵝，極短線劇烈震盪觸發強制平倉 - 結論：典型「過度博弈」，忽略倉位風險，長期必輸
4. 關鍵實戰建議
    - **用半凱利（Half-Kelly）**：全額按凱利公式下注波動太大，建議只投入計算結果的「一半」 - **盈虧比 > 2**：低於 2 的策略要極高勝率才能獲利 - **不要過度最佳化**：參數調太完美 = 對歷史數據過度擬合 - **記入交易成本**：手續費、滑價都會侵蝕獲利

#### 四、資產配置與分散投資（風險管理核心）

1. 什麼是資產配置？

    - **定義**：將資金分配至不同資產類別（股票、債券、現金等） - **目的**：降低單一資產波動對整體投資組合的影響 - **核心精神**：不是預測市場，而是建立能穿越景氣循環的投資架構
2. 三種配置類型

| 類型 | 股票比例 | 債券比例 | 適合對象 | |------|---------|---------|---------| | 保守型 | 低（20-30%） | 高（70-80%） | 風險趨避者、退休族 | | 平衡型 | 中（50%） | 中（50%） | 穩健投資者 | | 積極型 | 高（70-80%） | 低（20-30%） | 年輕人、高風險承受者 |

3. 現代投資組合理論（MPT）

核心概念：

    - 在給定風險水準下，透過組合不同相關性資產 - 建構「有效邊界」（Efficient Frontier） - 使投資人獲得相對最佳的風險報酬組合
關鍵洞察：
    - 資產之間的「相關性」是關鍵 - 相關性越低，分散效果越好 - 整體組合風險 < 單一資產風險加總
4. 分散投資三個維度

資產類別：

    - 股票 vs 債券 - 美股 vs 台股 vs 新興市場 - 公債 vs 公司債
產業分散：
    - 科技、能源、醫療、金融 - 不要重押單一產業
地區分散：
    - 成熟市場（美、歐、日） - 新興市場（中國、印度、東南亞） - 不同地區經濟週期不同步
5. 策略型 vs 戰術型配置

策略型資產配置：

    - 根據長期目標設定固定比例 - 定期再平衡（如每季、每年） - 一般投資人較適合
戰術型資產配置：
    - 依市場短期變化調整比例 - 屬於主動策略 - 需要市場判斷能力
6. 再平衡的重要性

為什麼要再平衡：

    - 市場波動會讓比例偏離目標 - 例如：目標股債 60/40，股市大漲變成 75/25 - 需要賣股買債「再平衡」回 60/40
再平衡作用：
    - 維持原本風險水準 - 避免單一資產比重過高 - 建立紀律性操作機制 - 研究顯示：再平衡有助風險控制，非提升報酬
7. 新手常見錯誤
    - 過度集中投資（單一股票或產業） - 忽略再平衡 - 情緒化調整配置 - 將短期市場波動誤認為策略失敗
---

#### 五、期望值計算（交易員的「廟算」）

1. 什麼是廟算？

    - **典故**：《孫子兵法》「多算勝，少算不勝」 - **古代**：將領出征前在宗廟推演兵力、糧草、天時地利 - **現代交易**：進場前的數據回測與期望值計算 - **目的**：在開戰第一秒就決定勝負
2. 核心公式：期望值

``` 期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損) ```

範例計算：

    - 勝率 50% - 平均獲利 10,000 元 - 平均虧損 5,000 元 - 期望值 = (50% × 10,000) - (50% × 5,000) = 2,500 元
解讀：
    - 正期望值 → 系統長期有利可圖 - 負期望值 → 長期必輸 - 正期望值 + 樣本數足夠（>100 次）→ 有進場底氣
3. 廟算的兩個核心

數據回測（現代版沙盤推演）：

    - 有交易想法後，丟進歷史數據測試 - 過濾掉「看起來會賺」的垃圾靈感 - 很多想法在腦中完美，回測後 MDD 高得嚇人，勝率連 40% 都不到
計入交易成本：
    - 手續費、滑價會侵蝕獲利 - 頻繁進出導致成本大幅侵蝕 - 沒把摩擦成本算進去 = 勝算是幻覺
4. 廟算的致命盲點

過度最佳化（Curve Fitting）：

    - 為追求完美績效曲線而過度調整參數 - 等於「竄改歷史考題」 - 必須用走勢最佳化（WFO）確保策略在未知數據中依然有效
忽視最大回撤：
    - 期望值正不代表中途不會虧損 - 要確認「連續虧損後，系統是否能靠小輸大贏回歸正值」
5. 交易者的日常分配
    - 80% 時間：在安靜環境裡計算與研究 - 20% 時間：冷靜地執行計算後的結果 - 不是臨場發揮，而是事前準備
---

怎麼應用：

實戰操作步驟（停損設定）：

1. **進場前決定停損方法**： - 短線交易 → 百分比停損（5-7%）或 K 線停損 - 波段持有 → 移動停損（10-15%） - 長期投資 → 基本面停損（買進理由消失）

2. **設定停損價格**： - 百分比停損：買進價 × (1 - 停損%) - 移動停損：最高價 × (1 - 停損%) - 券商設定：選擇 Stop 類型掛單

3. **執行紀律**： - 停損點不得更改 - 跌破就出場，不抱幻想 - 出場後檢討：策略有問題還是執行有問題？

實戰操作步驟（倉位管理）：

1. **計算單筆風險**： - 決定願意承擔的風險比例（建議 1-2%） - 例如：帳戶 100 萬，單筆風險 1% = 1 萬元

2. **套用凱利公式概念**： - 半凱利（Half-Kelly）較安全 - 高信心訊號（8/8 共振）→ 可加重倉位 - 一般訊號 → 用保守倉位

3. **計算實際股數**： ``` 股數 = 單筆風險金額 / (進場價 - 停損價) 範例：風險 1 萬 / (100 - 95) = 2,000 股 ```

實戰操作步驟（資產配置）：

1. **確定投資目標**： - 短期（<3 年）→ 保守型 - 中期（3-10 年）→ 平衡型 - 長期（>10 年）→ 積極型

2. **設定目標比例**： - 例如：股票 60%、債券 30%、現金 10% - 每季檢查一次比例

3. **執行再平衡**： - 股票大漲超過 65% → 賣股買債 - 股票大跌低於 55% → 買股賣債 - 每年至少再平衡一次

實戰操作步驟（期望值計算）：

1. **記錄每筆交易**： - 勝/敗筆數 - 平均獲利金額 - 平均虧損金額

2. **計算期望值**： ``` 期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損) ```

3. **確認策略有效性**： - 期望值為正 + 樣本 > 100 → 策略可繼續 - 期望值為負 → 策略需修正或放棄 - 勝率不是唯一指標，關鍵是「小輸大贏」

優點：

停損策略優點：

    - **控制損失範圍**：最大虧損 = 停損百分比，不會血本無歸 - **減少情緒干擾**：事先決定，到價就執行，避免「再等等」心態 - **移動停損保護利潤**：隨股價上漲自動調整，鎖住獲利 - **多種方法選擇**：可根據交易風格選擇適合的方法
凱利公式優點：
    - **科學化倉位**：不是憑感覺下注，而是根據數學計算 - **最大化資本成長**：找到增長與生存的最佳平衡 - **控制最大回撤**：確保連續虧損後仍能恢復 - **搭配盈虧比**：確保「小輸大贏」的系統優勢
資產配置優點：
    - **降低整體波動**：不同資產類別相關性低，風險分散 - **穿越景氣週期**：不預測市場，建立長期架構 - **紀律性操作**：再平衡機制避免情緒決策 - **符合研究**：Brinson 等研究證實配置比例比標的選擇更重要
期望值計算優點：
    - **事前驗證策略**：進場前就知道策略是否有利可圖 - **過濾垃圾想法**：很多「感覺不錯」的想法回測後一文不值 - **建立信心**：有數據支撐，執行時更果斷
---

缺點/限制：

停損策略缺點：

    - **可能被洗出場**：停損設太緊，正常波動就觸發 - **錯失反彈**：停損後股價上漲，心理糾結要不要買回 - **基本面停損需要研究能力**：要持續追蹤公司營運 - **跳空缺口風險**：股價跳空開盤，無法在設定價出場
凱利公式缺點：
    - **假設歷史數據可信**：實際上過去勝率不代表未來 - **波動極大**：全額凱利下注波動太大，心理壓力重 - **需要大量數據**：樣本數太少會產生錯誤結論 - **忽略極端事件**：黑天鵝可能讓策略瞬間失效
資產配置缺點：
    - **報酬被稀釋**：分散降低風險，也降低潛在報酬 - **無法消除系統性風險**：全球金融危機時全部資產都會跌 - **再平衡成本**：賣買產生交易成本 - **需要紀律**：市場大漲時想「再等等」，大跌時不敢買
期望值計算缺點：
    - **樣本數要夠**：少於 100 筆結論不可靠 - **過去不代表未來**：回測績效好不代表實盤好 - **過度最佳化風險**：參數調到完美只對歷史有效 - **忽略交易成本**：手續費、滑價會讓期望值縮水

核心內容：

#### 一、停損的意義與功能

1. 停損的定義

    - **意義**：停止損失，損失到此為止，從市場取回主動權 - **功能一**：控制風險，脫離不可預測的危險 - **功能二**：保留資金，保存反敗為勝的本錢
2. 設停損的重要原則（5 點） 1. **進場前就要設好停損**：預備買股票時，除了設定買進價位，也要設好決策錯誤時認賠出場的價位 2. **停損點不得更改**：設好後不能改，否則等於沒設 3. **收盤跌破要立即出場**：做多跌破停損價要出場，做空突破停損價要出場，必須絕對遵守紀律 4. **以順勢交易為主**：為了減少停損次數，在高勝率位置進場 5. **停損不超過 10%**：以不超過 10% 為標準

#### 二、五大停損方法（具體操作方式）

1. K 線停損法

    - **做多操作**：以進場當天 K 線的最低點為停損點 - **做空操作**：以進場當天 K 線的最高點為停損點 - **步驟**：
1. 進場當天記錄 K 線高低點 2. 做多：收盤價跌破進場日 K 線最低點 → 次日開盤立即出場 3. 做空：收盤價突破進場日 K 線最高點 → 次日開盤立即出場
    - **適用情境**：短線操作、當沖交易 - **優點**：簡單明確，以進場日的實際價格區間為基準 - **注意**：K 線實體長度決定停損距離，實體長則停損空間大
2. 趨勢停損法
    - **做多時機與停損設定**：
- 回檔後買上漲：停損設在轉折波的低點 - 盤整突破買上漲：停損設在突破點
    - **做空時機與停損設定**：
- 反彈後空下跌：停損設在轉折波的高點 - 盤整跌破空下跌：停損設在跌破點
    - **步驟**：
1. 識別趨勢轉折點（前低/前高） 2. 做多停損設在最近轉折低點下方 3. 做空停損設在最近轉折高點上方
    - **適用情境**：波段操作、趨勢交易 - **關鍵**：必須能正確判斷趨勢轉折點
3. 均線停損法
    - **具體操作**：以進出依據的均線設停損 - **步驟**：
1. 明確進場依據（如：股價站上 5 日均線買進） 2. 停損設定為進場時的均線價格 3. 收盤跌破該均線價 → 出場
    - **範例**：
- 進場條件：突破 5 日均線 - 進場時 5 日均線價：92 元 - 停損價：92 元 - 收盤跌破 92 元 → 出場
    - **適用情境**：均線交易系統、短線操作 - **優點**：與進場邏輯一致，邏輯清晰
4. 固定比例停損法
    - **具體操作**：以買進價的 2%~10% 為停損點 - **步驟**：
1. 決定停損比例（依個人風險容忍度） 2. 華爾街操盤高手標準：不超過 10% 3. 一般建議：固定 5% 4. 計算停損價 = 買進價 × (1 - 停損比例)
    - **範例**：
- 買進價：100 元 - 停損比例：5% - 停損價：100 × 0.95 = 95 元 - 收盤跌破 95 元 → 出場
    - **適用情境**：所有交易風格、新手最容易執行 - **優點**：計算簡單，執行明確 - **注意**：比例要根據個股波動度調整，高波動股票停損比例要放寬
5. 支阻停損法
    - **做多操作**：關鍵支撐跌破停損 - **做空操作**：關鍵壓力突破停損 - **步驟**：
1. 識別關鍵支撐/壓力（前低、前高、密集成交區、跳空缺口） 2. 做多：支撐位下方設停損 3. 做空：壓力位上方設停損
    - **關鍵支撐類型**：
- 前低點（多次觸及反彈的價位） - 盤整區下緣 - 向上跳空缺口 - 上升趨勢線
    - **關鍵壓力類型**：
- 前高點（多次無法突破的價位） - 盤整區上緣 - 向下跳空缺口 - 下降趨勢線
    - **適用情境**：技術分析交易、突破/跌破交易

#### 三、五種絕對停損情況

定義：明顯判斷錯誤，絕對要停損的位置點，否則極易造成重大損失

1. 盤整區布局多單，股價跌破盤整區

    - **情境**：在盤整區間布局多單，預期突破向上 - **訊號**：股價跌破盤整區下緣 - **操作**：絕對停損，不抱幻想 - **原因**：跌破盤整區代表原有的整理結構被破壞，趨勢可能反轉
2. 多頭高檔趨勢反轉，空頭確認
    - **情境**：持有多單，股價在高檔區 - **訊號**：趨勢反轉訊號出現（如：頭頭低、跌破重要支撐、大量黑 K） - **操作**：多單絕對停損 - **原因**：多頭趨勢結束時，續抱只會擴大虧損
3. 空頭低檔趨勢反轉，多頭確認
    - **情境**：持有空單，股價在低檔區 - **訊號**：趨勢反轉訊號出現（如：頭頭高、突破重要壓力、大量紅 K） - **操作**：空單絕對停損 - **原因**：空頭趨勢結束時，續抱會被軋空
4. 股價跌幅超過 10%
    - **情境**：任何持倉 - **訊號**：虧損達到 10% - **操作**：絕對停損，不可再拗單 - **原因**：10% 是關鍵心理關卡，超過後心理壓力會影響判斷，且回本需要漲幅 11%，難度大增
5. 逆勢單的反轉
    - **情境**：做逆勢交易（如：多頭做空、空頭做多） - **訊號**：走勢與預期相反，且反轉確立 - **操作**：立刻停損 - **原因**：逆勢交易風險極高，反轉時不果斷出場會造成嚴重虧損

#### 四、停利概念：上漲預測獲利目標

1. 目標設定原則

    - 多頭向上會遇到上面很多壓力 - 短線操作先以靠近股價最近的壓力作為短期初步的獲利目標
2. 向上的壓力類型
    - 前高 - 長週期均線下彎 - 大量黑 K - 向下跳空缺口 - 盤整區（日線看短線壓力，週線看中長期壓力）
3. 多頭趨勢特性
    - 「見壓不是壓」：強勢多頭會突破壓力 - 上漲到預定壓力時，要觀察多空力量變化 - 出現的 K 線是否有轉折訊號 - 配合個人操作紀律（如短線守 5 日均線操作）決定是否出場
---

怎麼應用：

實戰操作步驟：

1. **進場前規劃（5 步）**： - 確定進場條件與價格 - 選擇停損方法（K 線/趨勢/均線/比例/支阻） - 計算停損價格 - 評估停損比例是否超過 10%（超過就不要進場） - 確認單筆風險不超過總資金 2%

2. **進場後執行（3 步）**： - 立即設定停損單或記錄停損價 - 每日收盤檢查是否觸發停損 - 觸發停損 → 次日開盤立即出場，不猶豫

3. **五種絕對停損檢查清單**： - 跌破盤整區 → 出場 - 多頭趨勢反轉 → 出場 - 虧損超過 10% → 出場 - 逆勢單反轉 → 出場 - 關鍵支撐跌破 → 出場

4. **停利規劃**： - 找出最近的壓力位 - 到達壓力位觀察 K 線轉折訊號 - 短線可守 5 日均線操作 - 獲利達到預期目標或超過設定目標 → 考慮出場

---

優點：

    - **方法多元**：五種停損方法適用不同交易風格與情境 - **邏輯清晰**：每種方法都有明確的計算方式與執行步驟 - **控制風險**：停損標準不超過 10%，一般用 5%，確保單筆虧損可控 - **保留資金**：停損保存反敗為勝的本錢 - **紀律明確**：五種絕對停損情況讓判斷有依據 - **與進場邏輯結合**：均線停損、趨勢停損與進場條件一致，邏輯完整
---

缺點/限制：

    - **K 線停損**：盤整區容易頻繁觸發，K 線實體長短影響停損距離 - **趨勢停損**：需要正確判斷轉折點，技術要求較高 - **均線停損**：均線種類選擇（5/10/20）影響停損位置，無標準答案 - **固定比例停損**：未考慮個股波動度，高波動股可能過早觸發 - **支阻停損**：支撐/壓力判斷主觀，不同人看法不同 - **紀律執行**：知道方法不等於能做到，執行力是關鍵 - **跳空風險**：股價跳空開盤跌破停損價，無法在設定價出場 - **心理關卡**：10% 絕對停損對許多人仍是心理障礙

核心內容：

#### 一、倉位管理基本概念

1. 什麼是倉位？

    - **定義**：投資者在某個資產上投入的資金佔總資金的比例 - **功能**：倉位大小決定每筆交易承擔的風險 - **本質**：在機會與風險之間取得平衡 - **目的**：避免單筆交易過度虧損導致資本歸零
2. 不同資產的倉位管理考量

| 指標 | 股票 | 期貨 | 外匯 | 加密貨幣 | |------|------|------|------|---------| | 典型波動率 | 年化 5-20% | 日波動 10-30% | 日波動 1-2% | 日波動 5-10% | | 建議最大風險 | 每筆不超過 2% | 每筆 1-2% | 每筆 1% | 每筆 0.5-1% | | 槓桿倍率 | 1× / 2×（融資） | 10-20× | 20-50× | 2-10× | | 停損距離建議 | 股價 5-10% | 合約 2-3% 或 20-50 點 | 20-50 pips | 幣價 5-10% |

3. 倉位管理的重要性

    - 控制每筆交易風險在可承受範圍 - 降低爆倉風險 - 讓交易者在波動市場中存活 - 在長期內最大化收益
---

#### 二、五種倉位計算策略

1. 固定百分比法（Fixed % Risk）

    - **具體操作**：每次交易僅使用帳戶總額的固定比例（如 1-2%）作為風險額 - **步驟**：
1. 計算單筆風險金額 = 帳戶總額 × 風險比例（如 100 萬 × 2% = 2 萬） 2. 計算每股風險 = 進場價 - 停損價 3. 下單股數 = 單筆風險金額 / 每股風險
    - **範例**：
- 帳戶 100 萬，風險 2% → 每筆風險 2 萬 - 進場價 100 元，停損 95 元 → 每股風險 5 元 - 下單股數 = 20,000 / 5 = 4,000 股
    - **優點**：風險與帳戶規模同步成長，易於紀律執行 - **缺點**：無法考量當前市場波動性 - **適用**：長期穩健增長型交易者
2. 固定金額法（Fixed $ Risk）
    - **具體操作**：每次交易固定金額作為風險額 - **步驟**：
1. 設定固定風險金額（如每筆 5,000 元） 2. 計算每股風險 3. 下單股數 = 固定風險金額 / 每股風險
    - **優點**：計算簡單明瞭 - **缺點**：帳戶大幅波動後風險分配不靈活 - **適用**：小型帳戶、程式化交易策略
3. ATR 波動率法
    - **具體操作**：根據 ATR 動態調整倉位大小 - **步驟**：
1. 讀取當前 ATR(14) 數值 2. 計算停損距離 = ATR × 倍數（1.5-3 倍） 3. 倉位大小 = (帳戶資金 × 風險%) / (ATR × 停損倍數 × 每點價值)
    - **優點**：自動考量市場波動性，停損更貼近行情 - **缺點**：必須正確設定 ATR 週期，高波動時過度縮小倉位可能錯失機會 - **適用**：追求與市場波動同步的波段交易
4. Kelly 公式
    - **具體操作**：根據勝率與盈虧比計算最佳下注比例 - **公式**：f* = (p × b - q) / b
- f* = 最佳下注比例 - p = 勝率 - q = 敗率（1 - 勝率） - b = 盈虧比（平均獲利 / 平均虧損）
    - **實務調整**：使用「半凱利」（Half-Kelly）更安全 - **優點**：理論上可最大化資本成長 - **缺點**：需精準估算勝率與盈虧比，易過度激進 - **適用**：資料充分、勝率與盈虧比穩定的策略
5. 金字塔加碼法（Pyramiding）
    - **具體操作**：趨勢明確時分批加倉，利潤放大 - **步驟**：
1. 趨勢確認後先建立初始倉位 2. 獲利後回調時加碼（不是虧損時加碼） 3. 每次加碼金額遞減（如第一筆 10 萬，第二筆 8 萬，第三筆 6 萬） 4. 停損移至成本價保護本金
    - **優點**：趨勢明確時可擴大利潤，避免一次重倉 - **缺點**：趨勢反轉時最後一次加碼風險最大 - **適用**：趨勢跟隨策略、波段交易
---

#### 三、風險報酬比（Risk-Reward Ratio）

1. 定義與計算

    - **定義**：單筆交易中，預期獲利和可能虧損的比例 - **公式**：風險報酬比 = 預期獲利 ÷ 可能虧損（或 報酬 ÷ 風險） - **表示方式**：以「X:1」形式表示，X 代表潛在報酬相對於風險的倍數 - **範例**：
- 進場價 100 元，停損 95 元，停利 110 元 - 可能虧損 = 5 元 - 預期獲利 = 10 元 - 風險報酬比 = 10 / 5 = 2（或 1:2，表示風險 1 報酬 2）

2. 為什麼風險報酬比重要？

    - **控管單筆虧損**：設定停損上限，防止一次失誤造成毀滅性打擊 - **確保長期獲利**：勝率 50% 配合 1:2 風險報酬比，長期仍可獲利 - **培養紀律**：進場前先設停損停利，避免情緒化交易 - **評估交易機會**：快速評估一筆交易值不值得做
3. 不同行情下的風險報酬比設定

| 行情類型 | 建議風險報酬比 | 實務範例 | |---------|--------------|---------| | 趨勢行情 | 1:3 ~ 1:4 | 明確上漲趨勢做多，停損小、目標遠 | | 盤整行情 | 1:1.5 ~ 1:2 | 區間震盪底部支撐買進，見好就收 | | 高波動行情 | 1:2 ~ 1:3 | 數據公布前後短線進場，停損較寬 | | 消息行情 | 1:3 以上 | 財報前小停損搏大獲利 |

4. 風險報酬比計算步驟

    - **步驟 1**：設定進場價、停損價、停利價 - **步驟 2**：計算可能虧損 = |進場價 - 停損價| - **步驟 3**：計算預期獲利 = |停利價 - 進場價| - **步驟 4**：風險報酬比 = 預期獲利 ÷ 可能虧損 - **步驟 5**：判斷是否符合最低標準（建議至少 1:1.5 或 1:2）
5. R 乘數概念（R Multiple）
    - **定義**：以「風險單位 R」衡量報酬的倍數 - **計算**：R = 停損距離，報酬 = R 的倍數 - **範例**：
- 停損 5 元，這是 1R - 獲利 15 元，這是 3R - 風險報酬比 = 3R / 1R = 3:1
    - **意義**：讓報酬標準化，方便跨交易比較
---

#### 四、風險報酬比與勝率的關係

1. 核心公式：期望值 ``` 期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損) ```

2. 不同組合的損益平衡

| 風險報酬比 | 所需最低勝率 | |-----------|-------------| | 1:1 | 50% | | 1:1.5 | 40% | | 1:2 | 33% | | 1:3 | 25% | | 1:5 | 17% |

3. 關鍵洞察

    - **高風險報酬比 + 低勝率**：也能獲利
- 勝率 40%，風險報酬比 1:3 - 期望值 = (40% × 3R) - (60% × 1R) = 1.2R - 0.6R = 0.6R（正值）
    - **低風險報酬比 + 高勝率**：需要極高勝率
- 風險報酬比 1:0.5，勝率需 >66% 才能保本
    - **最差情況**：低勝率 + 低風險報酬比 = 長期負期望 = 爆倉
4. 勝率與風險報酬比的平衡
    - **高勝率策略**：停利目標近，停損較寬，勝率可達 80% 以上
- 風險：一次大虧損可能抵消十次小獲利
    - **高報酬策略**：停利目標遠，停損較緊，勝率可能只有 30-40%
- 優勢：一次大獲利可抵消多次小虧損
    - **最佳化關鍵**：找到適合自己的勝率/報酬比組合
---

#### 五、倉位管理常見錯誤

1. 過度槓桿與重倉

    - 滿倉交易：一旦逆勢造成嚴重回撤 - 高槓桿放大損失：50 倍槓桿 2% 波動就會爆倉
2. 忽略停損與風險
    - 沒設停損：帳戶暴露在無限虧損風險 - 隨意移動停損：破壞風險管理紀律
3. 情緒化倉位調整
    - 虧損時加碼「補倉」：越賠越重倉 - 獲利時追漲：越賺越追高 - 「越賠越重倉、越賺越追高」= 爆倉公式
4. 忽略投資組合分散
    - 集中單一標的或單一市場 - 相關資產風險疊加
5. 過度自信與高追預期
    - 小賺就大幅加碼 - 遇到逆轉損失更多
---

#### 六、R 乘數實戰應用

1. 先設定風險再設定報酬

    - **核心概念**：先決定「我願意賠多少」，再計算「我要賺多少」 - **步驟**：
1. 進場時設定停損點 2. 停損距離視為「1 單位風險」 3. 目標報酬設定為 1.5R、2R 或 3R
    - **範例**：
- 進場價 109.07，停損 108.55 - 停損距離 = 52 pips = 1R - 停利目標 = 109.07 + (52 × 1.5) = 109.85（1.5R）

2. 使用技術分析找高報酬/低風險區域

    - **支撐阻力**：在關鍵支撐區域進場，停損設在支撐下方不遠處 - **趨勢線突破**：突破關鍵點進場，假突破小賠，真突破大賺 - **形態學**：三角形突破、頭肩底等，風險小、獲利空間大
3. 追蹤停損保護利潤
    - **目的**：達到目標後用追蹤停損爭取更大報酬 - **操作**：價格上漲創高後，停損跟著上移 - **效果**：將 1:2 的風險報酬比變成 1:3 或更高
---

怎麼應用：

實戰操作步驟（倉位管理）：

1. **進場前計算**： - 決定單筆風險比例（建議 1-2%） - 計算停損距離 - 套用公式：倉位大小 = (帳戶資金 × 風險%) / 停損距離 - 確認風險報酬比至少 1:2

2. **選擇倉位策略**： - 長期穩健 → 固定百分比法 - 追求動態適應 → ATR 波動率法 - 勝率報酬比穩定 → Kelly 公式 - 趨勢明確 → 金字塔加碼法

3. **執行紀律**： - 下單前先設停損，不輕易移動 - 虧損時不加碼，獲利時可考慮分批加倉 - 每筆交易記錄風險報酬比

實戰操作步驟（風險報酬比）：

1. **進場前評估**： - 計算風險報酬比，問自己「這筆交易值得嗎？」 - 風險報酬比低於 1:1.5 → 考慮放棄或調整進出場位置 - 避開障礙重重的路徑（中途有重大阻力）

2. **設定停損停利**： - 先設停損（風險），再設停利（報酬） - 停損設在技術分析關鍵點（支撐下方、趨勢線下方） - 停利設在合理目標（下一個阻力位）

3. **不同行情調整**： - 趨勢行情：目標 1:3 以上 - 盤整行情：目標 1:1.5 ~ 1:2 - 高波動：降低倉位，維持 1:2 以上

4. **記錄與檢討**： - 每週統計平均風險報酬比 - 檢視是否有貪心沒跑或提早出場 - 調整下一週的策略

關鍵心態建議：

1. **資金保全優先**：活下來比一時獲利更重要 2. **避免恐慌和貪婪**：堅守停損，不因虧損加碼，不因獲利追高 3. **不迷信高勝率**：勝率和風險報酬比要搭配看 4. **耐心執行計劃**：每次進出場都有明確理由與倉位規則

---

優點：

倉位管理優點：

    - **控制單筆風險**：每筆虧損可控，避免一次翻船 - **適應帳戶變化**：帳戶成長時倉位自動放大，縮小時自動縮小 - **科學化決策**：用公式取代情緒，避免過度交易 - **多策略適用**：不同風格可選擇不同倉位策略
風險報酬比優點：
    - **簡單明確**：計算容易，進場前就知道划不划算 - **確保長期獲利**：即使勝率不高，只要風險報酬比夠高就能賺錢 - **培養紀律**：強制事先設停損停利，避免情緒化交易 - **評估交易機會**：快速篩選值得做的交易
---

缺點/限制：

倉位管理缺點：

    - **固定百分比法**：無法考量市場波動性 - **ATR 波動率法**：高波動時過度縮小倉位可能錯失機會 - **Kelly 公式**：需精準估算勝率與盈虧比，實務上困難 - **金字塔加碼法**：趨勢反轉時最後加碼風險最大
風險報酬比缺點：
    - **不考量達成機率**：只看數字，不考慮實際達到停利的難度 - **中途障礙**：可能有阻力/支撐擋路，影響達成率 - **停損太緊**：容易被洗出場，提高勝率但可能降低風險報酬比 - **停利太遠**：目標脫離實際，永遠到不了 - **需要技術分析能力**：合理的停損停利需要判斷關鍵價位

#### 一、為什麼交易心理比技術更重要？

1. 成功交易者的兩個共同特質

    - 對於選股、買進、賣出都有自己的一套投資邏輯或方法 - 嚴格遵循自己設下的規定進行買賣
2. 為什麼大多數人不制定規則？
    - 不希望為自己的交易結果負責 - 一旦制定規則，失敗時就無法怪罪市場或其他因素 - 寧可不制定標準，讓自己和市場的關係保持神秘性
3. 不制定規則的後果
    - 製造真正的心理矛盾 - 沒有「定海神針」，波濤洶湧時不知該加碼、賣出還是觀望 - 永遠無法學習有效的交易之道
4. 核心信念
    - 創造一套屬於自己的投資法則，就像在股市中擁有一根定海神針 - 股市就是大眾心理博弈 - 了解別人的心理較困難，先從了解自己開始

#### 二、紀律交易者的七個成功步驟

步驟一：聚焦於你要學習的知識

    - **具體做法**：
- 選定一種交易風格（短線、波段、長線） - 專注學習該風格的相關知識 - 不要一次學太多方法，專精一種
    - **目的**：避免資訊過載，建立扎實基礎 - **常見錯誤**：短線、波段、長線都想學，每種都學不精
步驟二：處理虧損
    - **具體做法**：
- 接受虧損是交易的一部分 - 設定可承受的虧損上限 - 虧損發生時不情緒化，按計畫處理 - 反省虧損原因，而非逃避
    - **目的**：控制損失，避免虧損擴大 - **關鍵心態**：虧損不是失敗，而是學習機會
步驟三：成為一種市場行為的專家
    - **具體做法**：
- 選擇一個熟悉的市場（台股、美股、期貨、外匯） - 專注研究該市場的特性 - 了解市場參與者行為模式 - 掌握該市場的週期與節奏
    - **目的**：建立對市場的深入了解 - **要避免**：頻繁換市場，每個都學不精
步驟四：學習如何完美執行一種交易系統
    - **具體做法**：
- 選定一套交易系統（如均線交叉突破、型態交易等） - 反覆練習直到熟練 - 不中途更換系統 - 記錄每次執行的結果
    - **目的**：讓系統內化為本能反應 - **關鍵**：完美執行一套普通系統，勝過拙劣執行多套系統
步驟五：學習依據機率思考
    - **具體做法**：
- 理解「每次交易是獨立事件」 - 不因連續獲利就覺得下次一定賺 - 不因連續虧損就覺得下次一定賠 - 以長期期望值為決策依據
    - **目的**：避免賭徒謬誤，理性決策 - **數學思維**：勝率 60% 的策略，連續 10 次虧損仍可能發生
步驟六：學習客觀
    - **具體做法**：
- 區分「事實」與「觀點」 - 不讓希望或恐懼影響判斷 - 價格是唯一事實，其他都是解讀 - 盤中不做主觀判斷，按照規則執行
    - **目的**：避免情緒干擾交易決策 - **實踐方法**：把「我覺得股價會上漲」改成「股價突破壓力線，符合買進條件」
步驟七：學習監督自己
    - **具體做法**：
- 寫交易日誌記錄每次交易 - 定期回顧交易記錄 - 識別自己的行為模式與弱點 - 持續調整與改進
    - **目的**：從自己的錯誤中學習 - **關鍵**：監督是持續的過程，不是一次性的

#### 三、交易日誌四步驟完整教學

步驟一：買進後記錄什麼

必要記錄項目：

    - **買進日期**：幾月幾號，日內波動大的可記錄時間點 - **買進張數/股數**：部位大小 - **買進股票**：標的名稱與代號 - **買進理由**：（關鍵！）
- 錯誤寫法：「我覺得股票會上漲」（主觀、無標準） - 正確寫法：「股價突破盤整區上緣，成交量放大，符合突破買進策略」（客觀、有標準）
    - **技術分析截圖**：（超級重要！）
- 畫出買點、停損、支撐壓力 - 記錄技術分析判斷 - 作為日後檢討依據

為什麼買進理由這麼重要？

    - 主觀覺得會上漲 → 容易賠錢（市場不如預期時不知所措） - 客觀符合買點才買 → 訓練成為機械化交易者 - 交易者成功之路：從主觀交易者變成機械化交易者
截圖要標記什麼？
    - 進場點位 - 停損價位 - 目標價位 - 支撐壓力線 - 關鍵均線 - 當時的成交量
---

步驟二：持有期間的記錄（超過 90% 投資人沒做到）

關鍵觀念：下單完後立刻訂下規則，避免盤中思考

應該設定的規則：

時間停損點：

    - 在 X 月 X 號，如果價格沒到 X 元，就賣出 - 例：買進後 5 個交易日，若未突破前高，賣出
空間停損點：
    - 百分比停損法：虧損 7% 停損 - 均線停損法：跌破 5 日均線停損 - 支撐停損法：跌破關鍵支撐停損
最大可能虧損：
    - 通常設定比停損更高（因為大盤可能突然暴跌） - 例：停損設 7%，但心理準備最大虧損 10%
何時看盤決定賣出？
    - 開盤 9:00-9:30？ - 尾盤 13:00-13:30？ - 盤中逢高賣出？ - 提前決定，盤中執行
停利點設定：
    - 股價帳面獲利超過 X 元 → 停利 - 股價到 X 價位 → 停利 - 分批停利：第一目標達成賣一半，第二目標達成全出
加碼點設定：
    - 股價突破 X 價格 → 再買一張 - 帳面獲利 X 元 → 再買一張 - 金字塔加碼：每次加碼金額遞減
極端狀況應對（保險）：
    - 大盤狂跌千點，個股瞬間穿過停損 → 該賣嗎？ - 賣不掉跌破停損 → 該怎麼辦？ - 永遠為任何狀況做好準備，否則交易就沒有永遠
記錄心情：
    - 每天持股都要寫 - 像日記般隨興發揮 - 記錄當下的感受 - 不用批判自己，只要記錄
為什麼記錄心情重要？
    - 交易 = 投資知識 + 心理學 - 很多時候知識正確，但價格波動時控制不住 - 記錄心情才能識別自己的情緒觸發點 - 未來遇到類似情況時提醒自己不要衝動
---

步驟三：賣出時的記錄

必要記錄項目：

    - **賣出時間**：日期時間 - **賣出股價 × 張數**：賣出價金 - **獲利/虧損多少**：金額與比例 - **賣出理由**：（重要！與買進理由對照）
- 是否符合原先設定？ - 是停損出場、停利出場還是中途出場？
    - **截圖技術線型**：
- 賣出點位在 K 棒上的位置 - 當時的成交量 - 賣出後的價格變化

常見問題：預定今天賣，但盤中覺得明天會漲，可以不賣嗎？

答案：不行。

    - 問題不在於要不要賣，而是「為什麼當初規則把賣出日期訂得這麼短」 - 不該在盤中做判斷 - 寧願錯過或少賺，都不應該讓自己成為主觀交易者 - 下次交易可以把「如果看到 XX 就可以不賣」寫進規則裡
規則的力量：
    - 讓我們盤中輕鬆面對股市任何情況 - 要做的只有根據開盤前設定的規則，照著操作 - 如果真的照著做，要不紀律都難
---

步驟四：檢討（最多人忽略的一步）

關鍵觀念：過一個月後再進行檢討（不要馬上檢討）

為什麼要延後檢討？

    - 馬上檢討容易陷入當時情緒 - 隔一段時間看更客觀（馬後炮視角） - 可以清楚看到當時看不到的盲點
檢討項目：

技術面檢討：

    - 賣得太早？ - 買得太晚？ - 策略哪裡需要改進？ - 停損時間該提早或延後？ - 停損方式調整？更寬或更窄？
資金與勝率檢討：
    - 調整每次下的資金？ - 自己的勝率多少？ - 持股夠有效率嗎？
行為面檢討：
    - 買進與賣出理由一致嗎？ - 是否有衝動買賣？ - 是否有違反自己的規則？ - 情緒如何影響決策？
結果分析：
    - 賺錢了，如何複製賺錢方式？你的行為做對嗎？（賺錢 ≠ 行為做對，可能是運氣） - 賠錢了，如何避免下次再賠？賠錢 ≠ 行為做錯，需要具體分析
進階技巧：
    - 截圖買進、賣出、一個月後的股價變化 - 觀察「如果當時沒賣會怎樣」 - 找出自己的優勢模式

#### 五、交易日誌的核心價值

1. 為什麼要寫交易日誌？

    - 人的記憶短暫，容易忘記當時的思考 - 每筆交易都是寶貴經驗 - 靠運氣贏來的，最後會憑實力輸回去 - 不記錄就無法檢討，不檢討就不會進步
2. 交易日誌的三個核心功能：
    - **記錄行為**：買進、賣出、加碼的決策過程 - **制定規則**：進場前就設好停損停利，避免盤中情緒化 - **持續檢討**：一個月後回顧，找出優缺點
3. 成功交易者的特質：
    - 不是只想著要賺多少錢 - 追求把每筆交易做對 - 記下所有行為，並知道有所不為

#### 六、檢討時的反省心法

1. 反省是累積精神資本的過程

    - 虧損或失誤的交易，必須覆盤寫下交易日誌 - 不知道為何進場的交易，要深刻反省，那是自己的弱點 - 填補弱點後，會獲得自信感 - 自信是投資者最重要的精神資本
2. 如果精神資本為負值：
    - 會喪失自信感 - 無論做任何決定，都不會順利 - 陷入惡性循環
3. 如何處理虧損？
    - 把虧損當作學費 - 想想這次失誤學到了什麼 - 取代自我慚愧感 - 跟自己約定，把下次一決勝負的小禮物送給自己
4. 長期累積的效果
    - 每天、每週、每月持續觀察和反省 - 保留的股票相關紀錄超過數千筆 - 幾年累積造就現在的你
5. 重要的投資觀念
    - 股票不是命中率之爭 - 新手會問「這技法命中率多少？」 - 真正重要的是「有沒有賺錢」 - 不需要成為聰明的人 - 「你有賺錢嗎？」才是最重要的

#### 七、實戰心態建議

1. 市場環境像流動的河流

    - 沒有源頭、沒有盡頭 - 你必須自行決定：
- 是否跳進去 - 何時跳進去 - 用多大力量跳進去
    - 跳進去後，可以：
- 增加或減少力道 - 跳出來，改變方向 - 在旁觀望，市場還是會繼續流動

2. 如果看不清楚市場走向

    - 先暫時停下來觀望 - 不要強迫自己一定要操作
3. 不要中途改變策略
    - 原本打算短線追突破 - 沒有突破後安慰自己「這家公司很好，可以長期持有」 - 這跟當初策略不一樣 - 一旦中途改變，出場點就完全不一樣 - 永遠中途更改策略，一輩子注定賠錢
4. 走過完整交易過程
    - 才能知道哪些是做對的 - 哪些是做錯的 - 永遠中途更改策略，無法檢討

怎麼應用：

實戰操作步驟：

建立交易日誌系統：

1. **選擇記錄工具：** - Excel 表格：適合量化分析 - Notion 或 Evernote：適合質化記錄 - 專門 App：手機隨時記錄 - 建議：Excel + 文件連結

2. **設計記錄範本：** - 基本資料區（日期、標的、價格、損益） - 策略紀錄區（進出場策略、停損停利、加碼條件） - 心態記錄區（每天持股心情） - 檢討區（一個月後填寫）

3. **買進當天記錄：** - 截圖技術線型 - 寫下買進理由（客觀、符合策略） - 設定停損停利點 - 設定極端狀況應對方案

4. **持有期間記錄：** - 每天記錄心情 - 價格變化截圖 - 是否有想改變策略的念頭？記錄下來

5. **賣出時記錄：** - 賣出理由與買進理由是否一致 - 截圖賣出點位 - 計算損益

6. **一個月後檢討：** - 走過完整週期再回顧 - 找出做得好的地方（複製） - 找出做不好的地方（改進） - 調整策略或規則

執行紀律的具體方法：

1. **下單前寫規則：** - 不要在盤中思考 - 進場前就設好停損、停利、加碼條件 - 寫在交易日誌裡，盤中照表執行

2. **極端狀況預演：** - 如果大盤崩盤怎麼辦？ - 如果賣不掉跌破停損怎麼辦？ - 提前想好應對方案

3. **心情記錄：** - 每天記錄持股心情 - 記錄任何想改變策略的念頭 - 未來遇到類似情況時可回顧

持續改進的循環： ``` 制定規則 → 執行交易 → 記錄結果 → 一個月後檢討 → 調整規則 → 下次執行 ```

優點：

交易心理優點：

    - **建立定海神針**：有規則就不受市場波濤影響 - **避免情緒干擾**：機械化執行，減少主觀判斷 - **對自己負責**：制定規則後，失敗就不能怪市場 - **長期進步**：七步驟持續累積，終成專家
交易日誌優點：
    - **記錄經驗**：人的記憶短暫，交易日誌幫助保存經驗 - **紀律執行**：事先設好規則，盤中照表執行 - **發現問題**：透過檢討找出自己的弱點 - **複製成功**：找出賺錢的模式，持續複製 - **避免運氣陷阱**：區分運氣和實力，不把運氣當實力
---

缺點/限制：

交易心理限制：

    - **需要時間養成**：七步驟需要長期累積 - **知易行難**：知道規則不等於能做到 - **人性弱點**：貪婪、恐懼、僥倖心理難以克服 - **需要自我監督**：沒人強迫，容易懈怠
交易日誌限制：
    - **時間成本**：每次交易都要記錄，需要投入時間 - **執行門檻**：超過 90% 投資人沒做到持有期間記錄 - **記錄品質**：記錄太簡單等於沒記，記錄太詳細又太累 - **檢討延遲**：要等一個月才檢討，可能忘記當時心情 - **需要紀律**：知道要寫不等於會寫，需要持續執行

#### 一、什麼是黑天鵝事件？

1. 定義與特徵

    - **定義**：無法預測、影響重大、事後才有人解釋合理的極端事件 - **三大特徵**：
1. **意外性**：發生前幾乎無法預測 2. **影響重大**：對市場造成劇烈衝擊 3. **事後解釋**：事後總有人找到「跡象」解釋

2. 為什麼黑天鵝難以防範？

    - **市場的盲點**：大多數風險模型基於「常態分布」 - **極端事件稀有但致命**：常態分布假設極端值很少，實際上極端事件影響遠超想像 - **人類心理**：事後才覺得「早該想到」，實際上事前沒人預測到
3. 黑天鵝事件的市場影響
    - **流動性枯竭**：恐慌時買盤消失，賣單直接成交在極端低位 - **連鎖清算**：槓桿部位被強制平倉，引發更多賣壓 - **相關性飆升**：原本不相關的資產同步下跌，分散投資失效 - **定價失靈**：衍生品定價邏輯崩潰，導致連環清算爭議

#### 二、幣圈 7 次致命黑天鵝事件（歷史教訓）

案例 1：2014 年 Mt.Gox 交易所倒閉

    - **事件**：當時占全球 70% 比特幣交易量的交易所被盜 85 萬枚 BTC - **價格影響**：比特幣從 $1000 跌到 $150，跌幅 86% - **教訓**：「Not your keys, not your coins」— 資產在平台手上不等於安全 - **風險管理啟示**：
- 分散交易所，不把所有資產放在單一平台 - 大額資產移至冷錢包 - 平台風險是獨立於市場風險的另一個風險

案例 2：2017 年 94 事件（中國一刀切）

    - **事件**：中國七部委聯合發文，將 ICO 定義為非法融資，交易所限期關停 - **影響**：山寨幣單日跌 50%-80%，項目群退款退幣 - **教訓**：政策風險是黑天鵝重要來源 - **風險管理啟示**：
- 監管政策變化無法預測 - 避免重押單一市場或政策敏感資產 - 分散地區與法幣計價資產

案例 3：2018 年 11 月算力內戰

    - **事件**：BCH 社區內訌，比特幣 24 小時跌 15%，兩周跌 45% - **教訓**：所謂共識在硬核算力和利益面前瞬間瓦解 - **風險管理啟示**：
- 幣圈共識脆弱 - 算力戰爭可能導致短期大幅波動 - 持有 BTC 本身是對抗社群分裂的策略

案例 4：2020 年 312 全球流動性危機

    - **事件**：新冠疫情爆發，全球資產「換現金」，美股熔斷 - **價格影響**：
- 比特幣 3/12 從 $8000 跌到 $5500（單日 -30%） - 3/13 最低殺到 $3782（24 小時 -50%） - 山寨幣單日跌 70%-80%
    - **教訓**：危機初期比特幣不是避險資產，但修復最快 - **風險管理啟示**：
- 流動性危機時所有資產同步下跌 - 槓桿部位會被清算在極端價位 - 持有現金是度過危機的關鍵

案例 5：2021 年 519 中國全面清退挖礦

    - **事件**：中國三大協會公告清退礦場 + 馬斯克「叛變」+ DeFi 連環清算 - **價格影響**：比特幣單日跌 30%，以太坊和山寨幣腰斬 - **教訓**：三重打擊（政策 + 情緒 + 技術）同時爆發 - **風險管理啟示**：
- 多重利空疊加時，市場會「閃崩」 - 交易所宕機時無法操作 - 流動性枯竭時想賣都賣不掉

案例 6：2022 年 Luna 歸零 + FTX 暴雷

    - **Luna 事件**：
- 市值前十的幣 72 小時內歸零 - 算法穩定幣 ≠ 穩定 - 引爆三箭資本等機構連環清算
    - **FTX 事件**：
- 11/2 媒體爆料 Alameda 資產負債表全是 FTT - 11/6 CZ 宣布拋售所有 FTT，擠兌爆發 - 11/11 FTX 申請破產 - 「大而不倒」在加密世界是徹頭徹尾的謊言
    - **教訓**：機構暴雷比散戶恐慌更可怕 - **風險管理啟示**：
- 不要信任「大平台」或「大機構」 - 流動性危機會連鎖爆發 - 事前風險分散是唯一保護

案例 7：2025 年 10.11 事件

    - **事件**：川普宣布重啟對華 100% 關稅，加密資產與美股相關性達歷史高位 - **價格影響**：
- 比特幣從 $122,000 閃崩到 $101,500（單日 -17%） - 以太坊從 $4800 跌到 $3400（-30%） - 爆倉規模 191.4 億美元，超過 160 萬人爆倉
    - **特徵**：
- 做市商集體撤單 - 流動性瞬間枯竭 - 賣單直接成交在極端低位
    - **教訓**：宏觀利空可以瞬間觸發崩盤

#### 三、股市歷史崩盤教訓（1929 與 2000）

1. 1929 年華爾街股災

    - **背景**：「咆哮的二十年代」充滿樂觀 - **徵兆**：
- 標準普爾成分股平均本益比高達 32.6 倍（遠高於歷史平均） - 大量借貸買股，投機貸款超越流通貨總額
    - **結果**：
- 「黑色星期四」泡沫破裂 - 股市跌幅 89% - 4 年經濟大蕭條 - 6000 多家企業破產、5500 多家銀行倒閉
    - **教訓**：
- 極端高本益比是危險徵兆 - 槓桿貸款會放大跌幅 - 泡沫破裂後的恢復期極長

2. 2000 年網際網路科技泡沫

    - **背景**：網際網路崛起，大量資金湧入未證明獲利能力的公司 - **徵兆**：
- 只要掛「dot-com」股價就能翻倍 - 全球僅 3% 人口使用網路，但估值已達天價
    - **結果**：
- 那斯達克從 5048.62 跌到 1114.11（跌幅 78%） - 無數科技公司破產 - 僅亞馬遜、eBay 等少數企業存活
    - **教訓**：
- 估值與實際獲利能力嚴重脫節時警惕 - 「新興產業」不代表「必賺」 - 泡沫破裂會淘汰 90% 以上的公司

#### 四、當下三大黑天鵝風險來源（2026 年分析）

1. 地緣政治風險（油價 + 供應鏈）

    - **事件**：美伊衝突升溫，荷姆茲海峽通行受阻 - **影響**：
- 布蘭特原油突破 $100/桶 - 全球 20% 石油運輸經過荷姆茲海峽 - 油價每上漲 $1 → 能源生產州受益、消費者負擔增加
    - **市場反應**：
- VIX 指數飆升至接近 30 - 期貨價格倒掛（現貨高於期貨） - 亞股大跌（日經跌 4000 點、韓股跌 8%、台股盤中跌 2000 點）

2. 宏觀政策風險（聯準會 + 利率）

    - **事件**：聯準會政策效果滯後，可能在錯誤時機降息或升息 - **影響**：
- 利率政策影響所有資產估值 - 聯準會是「完全的不確定因素」
    - **市場脆弱性**：
- AI 產業帶來的變革尚未完全反映 - 信貸市場可能出現裂痕

3. 流動性風險（市場共振）

    - **特徵**：
- 多重壓力疊加（地緣政治 + AI 變革 + 信貸市場） - 各類資產同步拋售 - 做市商撤單，流動性瞬間枯竭
    - **結果**：
- 股市可能尚未完全反映風險 - 最糟糕時期可能還沒到來

#### 五、極端行情的市場特徵

1. VIX 指數飆升

    - **定義**：芝加哥期權交易所波動率指數，衡量市場恐慌程度 - **解讀**：
- VIX < 15：市場平靜 - VIX 15-25：正常波動 - VIX > 25：恐慌開始 - VIX > 30：極端恐慌
    - **價格倒掛**：現貨價格高於期貨價格，表示近期恐慌
2. 相關性飆升
    - **平常**：不同資產相關性低，分散投資有效 - **極端行情**：所有資產同步下跌，相關性趨近 1 - **教訓**：分散投資在極端行情下可能失效
3. 流動性枯竭
    - **特徵**：
- 做市商集體撤單 - 買盤消失，賣單直接成交在極端價位 - 定價邏輯失靈
    - **後果**：
- 想賣賣不掉 - 價格瞬間跳空 - 衍生品連環清算

3. 決策框架
    - 機率最高的情境作為基準假設 - 在基準假設下規劃操作策略 - 準備其他情境的應對方案 - 定期更新情報，調整機率評估

#### 七、極端行情風險控制策略

1. 事前預防（最重要）

倉位控制：

    - 極端行情時倉位要比平常低 - 設定「最大回撤防線」：帳戶虧損達 X% 強制減倉 - 預留現金度過危機
槓桿管理：
    - 極端行情前降低槓桿 - 預留保證金避免爆倉 - 不要用「極端罕見」當作不加槓桿控制的理由
分散風險：
    - 分散交易所/券商 - 大額資產移至冷錢包或多個帳戶 - 單一平台風險是獨立風險
流動性準備：
    - 持有足夠現金/穩定幣 - 預留緊急流動性度過危機 - 危機發生時想賣賣不掉
2. 危機發生時的應對

不要恐慌性拋售：

    - 流動性枯竭時賣在最低點 - 定住情緒，評估是否真的需要出場 - 如果策略沒變，不要因恐慌打破紀律
評估持有部位：
    - 哪些是受影響最大的？ - 哪些是相對抗跌的？ - 哪些應該優先減碼？
等待流動性恢復：
    - 極端行情後流動性會恢復 - 資產修復需要時間 - 比特幣在 312 危機後修復最快
3. 危機後的機會

危機入市的原則：

    - 估值跌到歷史低點 - 流動性已恢復 - 基本面沒有永久性改變 - 可以用合理價格買進好公司
歷史經驗：
    - 1929 股災後 25 年才回到高點 - 2000 科技泡沫後 15 年才回到高點 - 312 危機後比特幣數月內創新高 - FTX 暴雷後 1 年多加密市場恢復

怎麼應用：

實戰風險控制步驟：

平日準備：

1. **倉位健康管理**： - 設定最大倉位上限（如 70%） - 預留 30% 現金應對極端行情 - 槓桿部位控制在可承受爆倉的範圍

2. **平台風險分散**： - 資產分散在 2-3 個交易所/券商 - 大額資產移至冷錢包或多個帳戶 - 記住「Not your keys, not your coins」

3. **情境推演練習**： - 定期思考「如果大盤明天跌 20%，我怎麼辦？」 - 預設應對方案，不是等危機發生再想 - 寫下各種情境的出場條件

4. **情報監控**： - 關注 VIX 指數 - 關注宏觀政策變化 - 關注地緣政治風險 - 設置新聞提醒

危機發生時：

1. **第一時間評估**： - 這是什麼類型的危機？（政策、流動性、地緣政治） - 預計持續多久？ - 持有部位受多大影響？

2. **執行預設方案**： - 不是在恐慌中做決定 - 照著事先規劃的方案執行 - 如果策略條件改變，依新條件調整

3. **避免犯錯**： - 不要恐慌性拋售在流動性最差的時候 - 不要加槓桿想在崩盤中「抄底」 - 不要忽略分散風險的重要性

危機後重建：

1. **評估損失**： - 計算實際損失金額與比例 - 分析哪些部位損失最大、原因為何

2. **檢討策略**： - 風險控制方案是否有效？ - 哪些準備不足？ - 下次如何改進？

3. **等待機會**： - 危機後估值便宜 - 流動性恢復後可以逐步布局 - 用合理價格買進優質資產

#### 一、股票投資的八大風險類型

1. 市場風險（系統性風險）

    - **定義**：影響整個股票市場的宏觀因素帶來的風險 - **特徵**：無法透過分散投資完全消除 - **成因**：經濟衰退、地緣政治衝突、央行加息等 - **影響**：大部分股票價格一同下跌，優質藍籌股也難倖免 - **應對**：資產配置、降低倉位、持有現金
2. 公司風險（非系統性風險）
    - **定義**：個別公司獨有的風險 - **成因**：
- 管理層決策失誤 - 新產品開發失敗 - 陷入法律訴訟 - 財務報表問題
    - **特徵**：對整體市場影響有限 - **應對**：分散投資於不同行業、不同公司
3. 流動性風險
    - **定義**：想賣出股票時，市場上沒有足夠買家承接 - **特徵**：
- 無法以合理價格迅速成交 - 需要大幅折價才能賣出
    - **常見情境**：每日成交量稀疏的細價股或興櫃股票 - **應對**：
- 選擇成交量大、流動性佳的標的 - 避免重倉低流動性股票 - 預留變現緩衝

4. 利率與貨幣風險

    - **利率風險**：
- 利率上升 → 企業融資成本增加 → 盈利受壓 - 利率上升 → 儲蓄/債券吸引力增加 → 資金流出股市
    - **貨幣風險**：
- 持有海外股票時，匯率波動影響最終回報 - 美股上漲但美元貶值 → 換算後報酬打折
    - **應對**：
- 關注央行利率政策 - 考慮匯率避險策略

5. 政策與監管風險

    - **定義**：政府政策轉變或監管新規例對市場造成衝擊 - **實例**：
- 中國對教育、科網行業監管收緊 → 相關港股劇烈波動 - 產業政策變化影響特定板塊
    - **應對**：
- 密切關注政策風向 - 評估政策對持股的潛在影響 - 避免重押政策敏感板塊

6. 信用風險

    - **定義**：公司財務困難無法償還債務的風險 - **特徵**：
- 股東權益排在債權人之後 - 公司破產清算時可能血本無歸
    - **應對**：
- 審查公司財務報表 - 關注債務比率與現金流 - 避開高負債公司

7. 操作風險

    - **定義**：公司內部流程、人員及系統的失誤或失效 - **成因**：
- 嚴重生產事故 - 核心技術人員流失 - 內部欺詐 - 數據系統被黑客攻擊
    - **影響**：嚴重影響公司營運及聲譽，拖累股價 - **應對**：
- 評估公司治理品質 - 分散投資，避免單一公司風險

8. 資訊不對稱風險

    - **定義**：機構投資者和內部人員比散戶擁有更多、更及時的資訊 - **特徵**：
- 散戶在交易中處於不利位置 - 知情者可能先行買賣 - 散戶在消息公布後才反應，蒙受更大損失
    - **應對**：
- 持續學習，提升認知 - 深入研究公司基本面 - 不追逐市場消息和「貼士」

---

#### 二、三大投資心理陷阱

1. 情緒交易：追高殺低與恐慌性拋售

    - **追高（FOMO）**：
- 市場氣氛熾熱時，不問估值地追入 - 產生「錯失恐懼症」 - 典型高買低賣，通往虧損的捷徑
    - **殺低（恐慌）**：
- 市場逆轉時因恐懼急於拋售 - 被貪婪和恐懼支配
    - **應對**：
- 建立投資紀律與規則 - 設定停損停利 - 不因情緒做決策

2. 過度自信：頻繁交易導致高成本

    - **特徵**：
- 連續獲利後產生過度自信幻覺 - 認為自己可以準確預測市場走向 - 進行頻繁短線交易
    - **後果**：
- 增加佣金和稅費等交易成本 - 提高判斷失誤概率 - 忽視或低估風險
    - **應對**：
- 認識市場的不可預測性 - 降低交易頻率 - 以長期投資為主

3. 羊群效應：盲目跟從市場「貼士」

    - **特徵**：
- 四處打探消息 - 盲目跟從財經 KOL、朋友或討論區「貼士」 - 沒有經過獨立分析
    - **風險**：
- 無法得知「貼士」來源是否可靠 - 無法判斷背後邏輯 - 市場風向轉變時不知如何應對 - 成為最後的接盤俠
    - **應對**：
- 建立自己的投資研究方法 - 獨立思考，不盲從 - 對所有消息抱持懷疑態度

---

#### 三、三大核心認知偏誤

1. 確認偏誤（Confirmation Bias）

    - **定義**：選擇性地尋找和強化與既有觀點一致的資訊，忽略相反證據 - **投資表現**：
-買進股票後只關注利多消息 - 對利空消息選擇性失聰 - 對持股產生過度偏執態度
    - **實例**：
- 買進 A 公司後，不斷尋找 A 公司的利多消息 - 忽略影響 A 公司的利空消息 - 無法理性分析個股現有價值
    - **應對**：
- 主動尋找反對意見 - 觀點相反的資訊有助於辨識非理性認知 - 建立客觀的決策流程

2. 過度自信（Overconfidence Bias）

    - **定義**：對自身能力和判斷過於自信，高估自己的能力和知識 - **投資表現**：
- 忽視或低估風險 - 認為自己能應對各種市場情況 - 過度槓桿、過度頻繁交易 - 對投資消息誤判
    - **實例**：
- 相信 B 公司股票會持續上漲 - 忽視市場風險和警訊 - 在股價高點或市場不穩定時買進 - 全力投入單一股票或產業
    - **應對**：
- 培養「智力上的謙遜」 - 承認自己的認知侷限 - 主動尋找反駁證據

3. 損失厭惡（Loss Aversion）

    - **定義**：對「承受損失」的敏感度高於「獲得同等利益」的敏感度 - **特徵**：損失帶來的痛苦程度高於同等利益快樂程度的兩倍以上 - **投資表現**：
- 傾向採取保守投資策略 - 股價下跌超過停損點時不願出場 - 固執持有虧損股票，希望價格回升 - 逃避確認自己的損失
    - **實例**：
- 持有 C 股票價值下跌超過停損點 - 不願出場避免承受損失 - 導致進一步超額損失
    - **應對**：
- 嚴格執行停損紀律 - 接受虧損是交易的一部分 - 不讓情緒影響決策

---

#### 四、行為金融學核心洞見

1. 展望理論（Prospect Theory）

    - **核心發現**：
- 失去一百元的痛苦 > 得到一百元的快樂（約2倍） - 人們在面對收益時傾向風險規避 - 在面對損失時傾向風險尋求
    - **對稱性**：決策行為在收益和損失情境下不對稱
2. 處置效應（Disposition Effect）
    - **定義**：投資者傾向於太快賣掉上漲股票，卻長期持有下跌股票 - **心理機制**：
- 賣掉上漲股票 → 鎖定收益，規避未來下跌風險 - 持有下跌股票 → 不願承認失敗，期待反彈回本
    - **與理性原則矛盾**：
- 理性：「讓利潤奔跑，及時停損」 - 實際：「賣盈守虧」
    - **應對**：
- 建立停損紀律，嚴格執行 - 讓獲利股票有更多上漲空間 - 不因帳面虧損就拒絕出場

3. 參考點依賴與框架效應

    - **參考點依賴**：
- 對結果是收益還是損失的判斷，取決於選擇的參考點 - 參考點通常是買入成本或初始預期
    - **框架效應**：
- 同樣的客觀結果，不同描述方式會改變決策 - 描述為「收益」→傾向風險規避 - 描述為「損失」→ 傾向風險尋求
    - **實例**：
- 方案 A 能確定救活 200 人 → 多數人选擇 - 方案 C 會確定導致 400 人死亡 → 多數人不選 - 實際上 A 和 C 結果完全相同，只是描述框架不同
    - **應對**：
- 警惕措辭方式可能帶來的心理誘導 - 評估資訊時剝離描述框架

4. 其他重要認知偏誤

    - **錨定效應**：過度依賴初始資訊（如股票歷史高點、買入成本） - **後見之明偏誤**：事後總覺得自己早就預見了結果 - **模糊厭惡**：傾向選擇風險已知（即使機率不利）的選項
---

#### 五、風險管理的智慧錦囊

1. 擁抱機率思維，告別單點預測

    - **做法**：
- 不執著於猜測一個「最可能」的結果 - 思考一系列可能的情境 - 為每種情境分配發生機率
    - **實戰應用**：
- 設定樂觀、中性、悲觀三種假設 - 評估不同情況下的潛在損益 - 形成全面的風險報酬圖譜

2. 保持思想彈性，踐行貝氏更新

    - **貝氏定理核心**：
- 初始信念（先驗機率） - 根據新證據調整 - 得到更新後信念（後驗機率）
    - **實戰應用**：
- 對公司前景有初始判斷（如 60% 成長、30% 持平、10% 衰退） - 新財報、新產品、新競爭者出現 - 相應調整各情境發生機率
    - **關鍵**：
- 保持「智力上的謙遜」 - 主動尋找挑戰觀點的证据 - 定期複盤判斷依據

3. 關注決策品質，而非單次結果

    - **核心觀念**：
- 好決策未必帶來好結果（可能運氣不好） - 壞決策也可能僥倖成功（可能運氣好）
    - **評估重點**：
- 決策過程是否理性、周全 - 非僅僅看最終成敗
    - **實戰應用**：
- 撰寫投資日誌 - 記錄決策理由、預期和事後反思 - 長期累積，提高成功機率

4. 認識心智盲點，善用工具對抗偏誤

    - **做法**：
- 了解損失規避、過度自信、從眾心理等常見陷阱 - 在決策關鍵時刻保持警惕
    - **投資檢查清單**：
- 強制系統性檢查所有關鍵因素 - 是否忽略某些風險？ - 是否受情緒或偏見影響？

5. 預留安全邊際，為未知留下緩衝

    - **定義**：做決策時預留一定的緩衝空間 - **投資應用**：
- 以低於內在價值的價格買入 - 評估風險時做相對保守的假設
    - **效果**：
- 不能阻止壞事發生 - 但能在壞事發生時提供保護 - 避免災難性後果

6. 拉長視角，利用時間的力量

    - **原理**：
- 短期波動受噪音、情緒、隨機因素干擾 - 長期來看，價值終將回歸
    - **實戰應用**：
- 忽略短期市場喧囂 - 專注於企業長期價值創造 - 耐心持有優質資產 - 利用時間平滑波動、累積回報

7. 了解自身風險承受度，量身訂做策略

    - **關鍵**：
- 每個人財務狀況、投資目標、知識水平、心理素質不同 - 不存在一套適用所有人的「最佳」策略
    - **做法**：
- 深入了解自己的風險偏好和風險承受能力 - 選擇匹配的資產配置、投資風格和風險控制措施 - 不盲目模仿他人 - 不承擔超出自身承受範圍的風險

---

#### 六、均值回歸的力量

1. 定義與原理

    - **定義**：極端現象傾向回歸平均水平的自然機制 - **起源**：高爾頓研究豌豆遺傳時發現 - **核心現象**：
- 非常高的父母，子女傾向比父母矮一些 - 非常矮的父母，子女傾向比父母高一些
    - **商業應用**：
- 行業或公司獲得超高回報率 → 吸引競爭者 - 資本湧入 → 產能增加 → 價格下降 → 回報率回歸

2. 投資績效的均值回歸

    - **發現**：短期「冠軍」基金，隨後績效常回落到平均水平 - **研究證據**：
- 過去三年表現最好的股票（贏家組合） - 過去三年表現最差的股票（輸家組合） - 未來三年：「輸家組合」平均報酬超越市場 - 「贏家組合」表現落後於市場
    - **啟示**：
- 不應過度迷信短期靚麗業績 - 關注長期、可持續的投資邏輯 - 風險控制能力比短期報酬更重要

---

#### 七、建立交易 SOP 對抗認知偏誤

1. 為什麼需要交易 SOP？

    - 靠感覺做交易容易陷入認知偏誤的心理陷阱 - 建立標準化作業程序可以降低偏誤影響 - 系統性方法比主觀判斷更可靠
2. 交易 SOP 應包含的內容
    - **選股標準**：
- 明確的選股條件與篩選指標 - 避免因消息或情緒選股
    - **進場條件**：
- 技術指標觸發條件 - 基本面條件 - 量化數據標準
    - **停損規則**：
- 固定百分比停損 - 技術支撐跌破停損 - 基本面惡化停損
    - **停利規則**：
- 固定比率停利 - 移動停利 - 技術壓力位停利
    - **倉位管理**：
- 單一標的最大倉位 - 總倉位上限 - 分批進出場規則

3. 執行紀律

    - 事先制訂規則，盤中照表執行 - 不要在盤中思考，避免情緒化 - 定期檢視與修正 SOP
---

怎麼應用：

實戰操作步驟：

風險意識建立：

1. **辨識八大風險**： - 每筆投資前評估：市場風險、公司風險、流動性風險 - 關注利率與匯率變化對持股的影響 - 審查公司財務報表，降低信用風險

2. **風險分散原則**： - 資產配置：股票、債券、現金、REITs 等 - 地區分散：成熟市場與新興市場 - 行業分散：避免重押單一產業 - 公司分散：單一公司倉位不超過總資金 X%

認知偏誤對抗：

1. **確認偏誤對抗**： - 買進前列出「看空理由」 - 主動尋找反對意見 - 建立「投資檢查清單」

2. **過度自信對抗**： - 承認自己無法預測市場 - 每筆交易記錄預期與實際結果 - 計算實際勝率與盈虧比

3. **損失厭惡對抗**： - 進場前設定停損點 - 停損是保護資金，不是失敗 - 帳面虧損也是虧損，及時認賠

4. **處置效應對抗**： - 上漲股票給設定移動停利 - 下跌股票給設定強制停損 - 不因「賺到了」就急著賣

決策流程優化：

1. **貝氏更新實踐**： - 建立投資假設（如：公司未來一年成長率 60%） - 收集新證據（財報、產業新聞、競爭者動態） - 更新假設機率（成長率從 60% 調整為 40%） - 根據新機率調整倉位

2. **投資檢查清單**： - 公司基本面是否良好？ - 估值是否合理？ - 技術面是否處於有利位置？ - 有沒有忽略的風險？ - 是否受情緒影響？ - 停損停利是否設定？

3. **交易日誌記錄**： - 記錄決策理由與預期 - 記錄情緒狀態 - 定期回顧偏誤案例 - 持續改進決策流程

均值回歸應用：

1. **選擇基金經理**： - 不迷信短期績效冠軍 - 關注長期（3-5 年）績效記錄 - 評估投資邏輯是否可持續

2. **股票投資**： - 高 ROE 公司難以永遠維持超高報酬 - 理解「護城河」可以延緩均值回歸 - 避免追逐已經超高報酬的熱門股

優點：

風險意識優點：

    - **全面了解風險**：八大風險類型涵蓋絕大多數投資風險 - **對症下藥**：不同風險有不同應對策略 - **保護本金**：認識風險是保護資產的第一步
認知偏誤對抗優點：
    - **減少非理性決策**：識別偏誤就能避免犯錯 - **建立紀律**：SOP 和檢查清單強制執行理性決策 - **長期提升勝率**：改善決策品質比追求短期報酬更重要
貝氏更新優點：
    - **動態調整**：根據新證據更新判斷 - **避免僵化**：不固守舊有觀念 - **理性客觀**：系統化整合新資訊

缺點/限制：

風險意識限制：

    - **風險無法完全消除**：只能管理，無法避開 - **新風險不斷出現**：黑天鵝事件無法預測 - **過度謹慎代價**：分散過度可能拖低報酬
認知偏誤對抗限制：
    - **人性難以改變**：知道偏誤不等於能做到 - **需要長期練習**：建立新習慣需要時間 - **市場壓力干擾**：恐慌時難以保持理性
貝氏更新限制：
    - **需要持續投入**：不斷收集新資訊 - **判斷主觀性**：機率評估可能不準確 - **資訊品質**：錯誤資訊會導致錯誤更新

核心內容：

#### 一、對沖交易的基本概念

1. 對沖交易的定義

    - **意義**：在投資組合中加入負相關或低相關的資產，降低整體風險 - **目的**：不是消除風險，而是將風險控制在可接受範圍 - **核心原理**：不同資產的價格走勢不完全同步，可以相互抵消部分波動
2. 對沖的三種主要類型

| 類型 | 說明 | 適用情境 | |------|------|---------| | **完全對沖** | 持有現貨 + 放空等量期貨 | 消除所有價格風險 | | **部分對沖** | 只對沖部分部位 | 保留部分上漲空間 | | **動態對沖** | 根據市場變化調整對沖比例 | 需要持續監控 |

3. 對沖的成本與代價

    - **交易成本**：手續費、滑價 - **機會成本**：對沖期間可能錯過收益 - **基差風險**：現貨與期貨價格走勢不完全一致
---

#### 二、外匯對沖策略

1. 為什麼需要外匯對沖？

    - 持有海外資產時，匯率波動影響最終收益 - 美股上漲但美元貶值 → 換算後報酬打折 - 台灣投資人投資美股、日股、新興市場都面臨匯率風險
2. 外匯對沖方法

方法一：遠期外匯合約（Forward）

    - 與銀行簽訂合約，鎖定未來匯率 - 優點：完全鎖定匯率成本 - 缺點：需要大額資金、銀行門檻高
方法二：外匯期貨（Currency Futures）
    - 在期貨交易所買賣標準化外匯合約 - 優點：流動性高、交易方便 - 缺點：需要追繳保證金
方法三：外匯ETF
    - 購買具有匯率對沖功能的ETF - 優點：簡單方便、門檻低 - 缺點：管理費較高
3. 實務建議
    - 短期投資：可以忽略匯率波動 - 長期投資：分散配置多種貨幣資產 - 大額資金：考慮遠期合約或期貨對沖
---

#### 三、期貨對沖策略

1. 期貨對沖原理

    - 持有現貨 + 放空等量期貨 = 鎖定價格 - 價格下跌時，現貨虧損，期貨獲利 - 價格上漲時，現貨獲利，期貨虧損 - 最終結果：價格變動影響相互抵消
2. 股票期貨對沖實例

情境：持有台積電 100 張，擔心短期下跌 操作：放空台積電期貨 100 張 結果：

    - 股價下跌 5%：現貨虧損 5%，期貨獲利 5% → 損益兩平 - 股價上漲 5%：現貨獲利 5%，期貨虧損 5% → 損益兩平
3. 期貨對沖的注意事項
    - **基差風險**：期貨價格與現貨價格可能有價差 - **流動性風險**：小台到期貨可能流動性不足 - **保證金追繳**：期貨虧損需要追繳保證金 - **到期日風險**：期貨有到期日，需要換倉
---

#### 四、期權對沖策略

1. 保護性認沽期權（Protective Put）

    - **操作**：持有股票 + 買入認沽期權（Put） - **效果**：鎖定最大虧損，保留上漲空間 - **成本**：期權權利金 - **實例**：
- 持有台積電成本 600 元 - 買入 580 元履約價的 Put，權利金 10 元 - 最大虧損 = 600 - 580 + 10 = 30 元 - 股價上漲時獲利不受限

2. 遮蓋性買權（Covered Call）

    - **操作**：持有股票 + 賣出買權（Call） - **效果**：權利金收入降低成本，但限制上漲獲利 - **風險**：股價下跌時仍有虧損風險 - **實例**：
- 持有台積電成本 600 元 - 賣出 650 元履約價的 Call，收權利金 15 元 - 有效成本 = 600 - 15 = 585 元 - 股價超過 650 元時必須賣出

3. 領口策略（Collar）

    - **操作**：持有股票 + 買入 Put + 賣出 Call - **效果**：同時鎖定上漲和下跌空間 - **成本**：Put 權利金 - Call 權利金（可能接近零成本） - **適合**：希望保護現有獲利，又不想付出太高成本
---

#### 五、配對交易（Pair Trading）

1. 配對交易原理

    - 找出兩檔高度相關的股票 - 當價差偏離正常區間時 - 做多相對便宜的股票 - 同時做空相對昂貴的股票 - 等待價差回歸正常時獲利
2. 配對交易的優點
    - **市場中立**：不論大盤漲跌都能獲利 - **風險可控**：虧損來源主要是價差擴大 - **適合震盪市**：多空都能操作
3. 配對交易的風險
    - **相關性失效**：兩檔股票走勢不再同步 - **融券成本**：做空需要支付借券費用 - **流動性風險**：小股票可能買賣困難
---

#### 六、對沖基金的主要策略

1. 股票多空策略（Long/Short Equity）

    - 做多看好的股票 + 做空看差的股票 - 淨部位可以是偏多、偏空或中性 - 目的：降低市場風險，獲取選股報酬
2. 市場中立策略（Market Neutral）
    - 做多與做空金額相等 - 淨部位接近零 - 目的：完全消除市場風險 - 報酬來源：多空部位的相對表現
3. 事件驅動策略（Event Driven）
    - 利用企業併購、重整、破產等事件 - 高風險高報酬 - 需要深入研究企業財務
4. 全球宏觀策略（Global Macro）
    - 根據全球經濟趨勢配置資產 - 可以操作股票、債券、貨幣、商品 - 需要對宏觀經濟有深入了解
---

怎麼應用：

實戰操作步驟（股票期貨對沖）：

1. **確定對沖需求**： - 持有多少股票？ - 想對沖多少比例？（50%、80%、100%） - 對沖期限多久？

2. **選擇對沖工具**： - 大型權值股 → 股票期貨 - 中小型股 → 可能需要個股期貨或期權 - 海外資產 → 外匯期貨或ETF

3. **計算對沖數量**： ``` 期貨口數 = 現貨市值 / (期貨價格 × 契約乘數) ```

4. **執行與監控**： - 定期檢查基差變化 - 注意保證金需求 - 期貨到期前換倉

實戰操作步驟（保護性認沽）：

1. **評估保護需求**： - 股票持有多久？ - 最大可接受虧損是多少？ - 保護期限需要多久？

2. **選擇履約價**： - 平價（ATM）：保護成本低，但最大虧損較大 - 價外（OTM）：保護成本更低，但保護範圍較小 - 價內（ITM）：保護範圍大，但成本最高

3. **評估權利金成本**： - 權利金 / 股價 = 保護成本比例 - 成本過高可能降低報酬太多

4. **持續監控**： - 選擇權到期需要展期 - 股價大漲可考慮賣出 call 降低成本

優點：

對沖交易優點：

    - **降低風險**：價格波動對整體投資組合影響減小 - **鎖定利潤**：已實現獲利可以透過對沖保護 - **市場中立**：不論大盤漲跌都能操作 - **靈活運用**：可以選擇完全對沖或部分對沖
期貨對沖優點：
    - **成本低**：只需要保證金 - **流動性高**：大型股票期貨成交量大 - **操作簡單**：買賣方向明確
期權對沖優點：
    - **保留上漲空間**：保護性 Put 不限制上漲獲利 - **成本低廉**：遮蓋性 Call 可以零成本或正成本 - **風險明確**：最大虧損就是權利金
---

缺點/限制：

對沖交易限制：

    - **成本費用**：對沖不是免費的，需要付出代價 - **機會成本**：對沖期間可能錯過收益 - **基差風險**：現貨與衍生品價差不完全同步 - **複雜性**：需要一定金融知識
期貨對沖限制：
    - **保證金追繳**：需要準備額外資金 - **到期日限制**：需要換倉 - **對手方風險**：期貨交易所風險
期權對沖限制：
    - **權利金成本**：保護性 Put 需要付費 - **時間價值消耗**：時間越接近到期，時間價值越低 - **流動性風險**：小型股選擇權可能流動性不佳 - **複雜性**：需要理解選擇權定價

#### 一、風險管理的核心定義

1. 什麼是風險管理？

    - **定義**：識別、評估和控制潛在損失的過程 - **主要目標**：保護資金，讓交易者能存活足夠長的時間讓優勢發揮作用 - **核心觀念**：成功的交易者不是避開虧損，而是**限制虧損**
2. 為什麼風險管理重要？
    - **保護資本**：沒有資金就無法交易 - **減少情緒壓力**：事先知道風險讓波動時更容易保持冷靜 - **確保一致性**：每筆交易風險一致，回撤可控 - **改善決策**：風險受控時，恐懼和貪婪影響較小
3. 專業交易者的核心信念
    - 業餘者關注報酬，專業者關注風險 - 你無法控制市場，但可以控制願意承受多少損失

#### 二、12種風險管理核心技術

1. 倉位管理（Position Sizing）

    - **定義**：決定每筆交易分配多少資金 - **1% 規則**：單筆交易風險不超過總資金的 1-2% - **範例**：帳戶 $10,000，每筆風險最多 $100-$200 - **目的**：即使連續虧損，仍有資金繼續交易 - **公式**：
``` 倉位大小 = 總資金 × 風險比例 / (進場價 - 停損價) ```

2. 停損單（Stop-Loss Orders）

    - **定義**：預設價格觸發時自動平倉 - **功能**：限制損失，避免情緒化決策 - **設定方法**：
- 百分比停損：從買進價下跌 X% 出場 - 技術停損：跌破支撐位出場 - 波動停損：根據 ATR 調整停損距離
    - **關鍵**：進場前就設定好停損，不是進場後才想
3. 風險報酬比（Risk-Reward Ratio）
    - **定義**：預期獲利與可能虧損的比例 - **建議比例**：至少 1:2（風險 $1 報酬 $2） - **範例**：停損 $5，停利目標 $10 - **效果**：即使勝率只有 50%，長期仍可獲利 - **重要性**：確保贏家補償輸家後仍有餘
4. 多元化（Diversification）
    - **定義**：將資金分散於不同資產、產業、策略 - **目的**：單一損失不會嚴重影響整體組合 - **方法**：
- 跨資產類別：股票、債券、商品、外匯 - 跨產業：科技、能源、醫療、金融 - 跨市場：成熟市場、新興市場
    - **效果**：平滑報酬、降低波動
5. 對沖（Hedging）
    - **定義**：建立抵銷部位以降低風險 - **方法**：
- 持有股票 + 買入 Put 選擇權 - 持有股票 + 放空指數期貨 - 持有跨國資產 + 外匯避險
    - **代價**：降低潛在獲利 - **優點**：提供保險功能，對抗不可預測事件
6. 追蹤停損（Trailing Stop）
    - **定義**：停損點隨價格上漲自動調整 - **範例**：
- 買進 $50，設定 5% 追蹤停損 - 漲到 $55，停損自動調整到 $52.25 - 回跌觸及停損 → 自動出場，鎖住利潤
    - **優點**：保護利潤同時保留上漲空間 - **缺點**：震盪行情容易被洗出場
7. 每筆交易風險計算
    - **定義**：進場前計算最大可能虧損 - **公式**：
``` 風險金額 = (進場價 - 停損價) × 股數 ```
    - **目的**：確保風險在可接受範圍內 - **實務**：計算後調整股數，使風險符合計畫
8. 槓桿管理（Leverage Management）
    - **定義**：控制使用槓桿的程度 - **風險**：10x 槓桿意味 10% 跌幅 = 100% 虧損 - **建議**：
- 新手避免使用槓桿 - 有經驗者仍需謹慎，只在高勝率時使用 - 永遠搭配停損使用

9. 波動度倉位調整

    - **定義**：根據市場波動度調整倉位 - **方法**：
- 高波動期：降低風險比例（2% → 1%） - 低波動期：可維持正常風險比例
    - **目的**：避免極端行情造成重大虧損 - **ATR 應用**：波動大時縮小倉位
10. 風險資金配置
    - **定義**：只用可承受損失的資金交易 - **原則**：
- 不要用生活費、退休金、緊急預備金交易 - 只用「多餘的錢」 - 虧損不影響生活方式
    - **效果**：避免情緒化決策
11. 情境分析與壓力測試
    - **定義**：模擬各種市場狀況測試策略 - **方法**：
- 模擬市場崩盤情境 - 模擬低流動性情境 - 模擬極端波動情境
    - **目的**：找出策略弱點，事先準備應對方案 - **效果**：實際遇到極端行情不會手足無措
12. 情緒控制與紀律心態
    - **定義**：維持理性決策不受情緒影響 - **常見情緒陷阱**：
- 恐懼：過早出場、不敢進場 - 貪婪：加碼過度、抱住獲利太久 - 自我：不願認錯、報復性交易
    - **對抗方法**：
- 建立交易計畫並嚴格執行 - 信任計畫，即使不舒服也跟著做 - 長期紀律比短期直覺更重要

#### 三、風險管理計畫五步驟

步驟一：定義每筆交易最大風險

    - 建議：總資金的 1-2% - 嚴格執行，不因情緒調整
步驟二：設定每日/每週虧損上限
    - 達到上限立即停止交易 - 避免情緒化「報復性交易」
步驟三：一致使用停損
    - 進場前就設定好 - 每筆交易都有停損，無例外
步驟四：風險先於報酬
    - 只接受風險報酬比良好的交易 - 不為了「試試看」而降低標準
步驟五：定期檢討與調整
    - 分析交易記錄 - 找出需要改進的地方 - 持續優化風險策略

#### 四、Beta 係數與投組風險評估

1. Beta 是什麼？

    - **定義**：衡量投組相對於大盤的波動度 - **Beta > 1**：比大盤波動大（高風險高報酬） - **Beta < 1**：比大盤波動小（低風險低報酬） - **Beta = 1**：與大盤同步
2. Beta 實務應用
    - **範例**：Beta = 1.2 的投組
- 大盤漲 10% → 投組漲 12% - 大盤跌 10% → 投組跌 12%
    - **風險意義**：Beta 越高，波動越大 - **投組調整**：根據風險承受度調整 Beta

#### 六、風險管理的心理學層面

1. 知道 ≠ 做到

    - 許多交易者知道如何用停損 - 但恐懼、貪婪、自尊往往壓過理性 - 建立紀律需要時間與練習
2. 專業交易者的心態
    - **接受虧損**：虧損是交易的一部分 - **信任計畫**：即使不舒服也嚴格執行 - **冷靜應對**：市場波動時保持理性
3. 常見心理陷阱
    - **報復性交易**：虧損後想立刻賺回 - **過度自信**：連贏後放大風險 - **恐懼錯失症（FOMO）**：怕錯過而追高
---

怎麼應用：

實戰操作步驟：

建立風險管理系統：

1. **設定風險上限** - 每筆交易：1-2% 總資金 - 每日虧損上限：5% 總資金（觸及停止交易） - 每週虧損上限：10% 總資金（觸及休息一週）

2. **計算倉位大小** ``` 單筆風險金額 = 總資金 × 2% 每股風險 = 進場價 - 停損價 股數 = 單筆風險金額 / 每股風險 ```

3. **設定停損停利** - 進場前就設定好停損價 - 風險報酬比至少 1:2 - 使用追蹤停損保護獲利

4. **多元化配置** - 單一標的不超過總資金 20% - 單一產業不超過總資金 40% - 預留現金 10-20% 應對極端行情

5. **建立情緒控制機制** - 連續虧損 3 筆 → 暫停交易 1 天 - 單日虧損達上限 → 當日停止交易 - 交易日誌記錄情緒狀態

壓力測試練習：

1. **模擬情境**： - 如果明天大盤跌 20%，我的組合會怎樣？ - 如果我持有的股票連續跌停 3 天，我能承受嗎？

2. **檢視應對方案**： - 有沒有足夠現金度過流動性危機？ - 有沒有設定停損觸發價？ - 有沒有分散風險？

優點：

風險管理系統優點：

    - **保護資本**：確保單筆虧損不會毀掉帳戶 - **情緒穩定**：事先規劃讓交易更理性 - **長期存活**：讓策略有時間展現優勢 - **一致性**：每筆交易風險一致，結果可預測 - **可複製**：有系統可以持續改進
12 種技術整合優點：
    - **全方位覆蓋**：從倉位到情緒都有方法 - **互補效果**：多種方法組合效果更強 - **依情況調整**：可根據市場狀況選擇適合的方法

缺點/限制：

方法本身限制：

    - **停損太緊**：容易被洗出場 - **停損太寬**：單筆虧損過大 - **多元化**：可能降低整體報酬 - **對沖**：降低獲利潛力
執行層面限制：

2026-03-20 第 10 回合：風險管理 - VaR 風險價值與風險量化指標

文章：[VaR 風險價值介紹與應用](https://www.concords.com.tw/about/company/var.htm)

---

核心內容：

#### 一、VaR 風險價值的定義

1. VaR 是什麼？

    - **全名**：Value at Risk（風險價值） - **定義**：在特定信心水準下，某投資組合在特定時間內最大可能的虧損金額 - **核心概念**：「在最糟糕的情況下，我們最多會虧多少？」
2. VaR 的三個核心參數

| 參數 | 說明 | 常見設定 | |------|------|---------| | **信心水準** | 預測的可信程度 | 95%、99% | | **時間範圍** | 預測的期間 | 1 天、10 天、1 個月 | | **基礎貨幣** | 計算損失的幣別 | 本國貨幣 |

3. VaR 的解讀方式

範例：

    - VaR = 100 萬元（95%信心水準，1 天） - 意義：有 95% 的信心，明天最大虧損不超過 100 萬 - 反之：有 5% 的機率，明天虧損可能超過 100 萬
4. VaR 的計算方法

方法一：歷史模擬法

    - 收集歷史價格變動資料 - 計算每個歷史情境的損益 - 找出對應信心水準的分位數 - 優點：簡單、不需要假設分配 - 缺點：依賴歷史資料、無法預測極端事件
方法二：變異數-共變異數法
    - 假設報酬率服從常態分配 - 計算平均報酬和標準差 - 使用常態分配分位數推算 VaR - 優點：計算快速 - 缺點：假設常態分配，忽略肥尾風險
方法三：蒙地卡羅模擬
    - 設定資產分配參數 - 模擬大量可能的價格路徑 - 計算每個路徑的損益 - 找出對應信心水準的分位數 - 優點：可以處理複雜分配 - 缺點：計算量大、依賴參數假設
---

#### 二、VaR 的優點與限制

優點：

    - **易懂**：單一數字表示風險，方便溝通 - **可比較**：不同資產、策略可以用 VaR 比較風險 - **整合性**：可以整合不同風險來源 - **標準化**：銀行業廣泛使用，有標準規範
限制：
    - **不告訴極端損失**：VaR 只說「最大可能」虧損，不說「如果更糟會怎樣」 - **分配假設**：假設常態分配可能低估極端事件 - **歷史依賴**：歷史資料無法預測新的風險 - **相關性變化**：市場極端時相關性會改變
---

#### 三、條件風險價值 CVaR

1. CVaR 是什麼？

    - **全名**：Conditional VaR（條件風險價值） - **又稱**：Expected Shortfall（預期短缺） - **定義**：當損失超過 VaR 時，平均會損失多少
2. CVaR vs VaR
    - VaR 告訴你「最大可能」虧損 - CVaR 告訴你「如果情況更糟」的平均虧損 - CVaR 比 VaR 更保守，更能反映極端風險
3. 為什麼需要 CVaR？
    - VaR 假設損失是常態分配 - 但極端事件可能讓損失遠超 VaR - CVaR 考慮了 VaR 之後的風險
---

#### 四、其他風險量化指標

1. 標準差（Standard Deviation）

    - 衡量資產價格的波動程度 - 標準差越大，波動越大 - 缺點：上行波動和下行波動一視同仁
2. Beta 值
    - 衡量資產相對於大盤的波動 - Beta > 1：資產比大盤波動大 - Beta < 1：資產比大盤波動小 - 缺點：只考慮相對波動，忽略絕對風險
3. Alpha 值
    - 衡量資產相對於大盤的超額報酬 - Alpha > 0：資產表現優於大盤 - Alpha < 0：資產表現劣於大盤
4. 夏普比率（Sharpe Ratio）
    - 公式：(報酬率 - 無風險利率) / 標準差 - 衡量每單位風險的報酬 - 夏普比率越高，風險調整後報酬越好
5. 索提諾比率（Sortino Ratio）
    - 公式：(報酬率 - 無風險利率) / 下行標準差 - 只考慮下行風險，更符合投資人關注虧損的心理
---

怎麼應用：

實戰操作步驟：

1. **計算投資組合 VaR**： - 收集投資組合成分股的歷史價格 - 計算每日報酬率 - 使用歷史模擬法找出 95% 分位數 - 得出 VaR 數值

2. **設定風險限額**： - 根據 VaR 設定最大可接受虧損 - 例：1 天 95% VaR 不超過帳戶 2% - 超過限額時降低倉位

3. **監控與調整**： - 定期回顧 VaR 預測是否準確 - 實際損失超過 VaR 時檢討 - 根據市場變化調整 VaR 參數

4. **搭配其他指標**： - VaR + CVaR：了解極端風險 - VaR + 夏普比率：平衡風險與報酬 - VaR + Beta：了解相對風險

---

優點：

    - **量化風險**：將抽象風險轉為具體數字 - **可比較**：不同投資組合的風險可直接比較 - **設定限額**：可以明確設定風險上限 - **監控工具**：定期檢視風險是否超標
---

缺點/限制：

    - **假設風險**：依賴歷史分配假設 - **極端風險低估**：無法捕捉黑天鵝 - **相關性變化**：極端時相關性會改變 - **計算複雜**：需要統計軟體支援

2026-03-20 第 11 回合：風險管理 - 最大回撤 MDD 與回撤恢復難度

文章：[Max drawdown(MDD)是什麼？最大回檔幅度的4個注意事項](https://rich01.com/max-drawdown-mdd/)

文章：[交易必學最大回撤MDD！一篇搞懂定義、計算方法與應用](https://homedadpro.com/mdd/)

---

核心內容：

#### 一、MDD 最大回撤的定義

1. MDD 是什麼？

    - **全名**：Maximum Drawdown（最大回撤） - **定義**：在特定時間內，投資組合從歷史最高點跌至最低點的最大跌幅 - **意義**：直觀反映策略在最糟糕情況下可能承受的損失程度
2. MDD 計算公式

步驟一：計算回撤（Drawdown） ``` 回撤 = (投資組合最高價值 - 最高後的最低價值) / 投資組合最高價值 × 100% ```

步驟二：找出最大回撤 ``` MDD = MAX(特定時間內所有回撤) ```

3. 實際計算範例

情境：

    - 起始資金：5 萬美元 - 漲到高點：10 萬美元 - 跌到谷底：7 萬美元 - 回升高點：12 萬美元 - 再跌到：9 萬美元
計算：
    - 第一次回撤：(10 - 7) / 10 = 30% - 第二次回撤：(12 - 9) / 12 = 25% - MDD = MAX(30%, 25%) = 30%
---

#### 二、各資產類別的 MDD 參考

| 資產類別 | 歷史 MDD | 風險等級 | |---------|---------|---------| | 美國大型股 | -50.9% | 高 | | 美國小型股 | -54.0% | 高 | | 非美國已開發市場股市 | -57.1% | 高 | | 新興市場股市 | -62.7% | 極高 | | 短期美國公債 | -6.3% | 低 | | 中期美國公債 | -23.2% | 中 | | 長期美國公債 | -45.3% | 高 | | 不動產 REITs | -68.3% | 極高 | | 黃金 | -61.8% | 高 | | 商品類 | -88.7% | 極高 |

關鍵洞察：

    - 股票類 MDD 通常在 -50% 到 -65% - 債券類 MDD 較低，但長期債風險較高 - 商品類 MDD 極高，風險最大 - 資產配置可以降低整體 MDD
---

#### 三、回撤恢復難度（關鍵觀念）

1. 跌幅與回升幅度的關係

| 跌幅 | 需要回升幅度 | 難度 | |------|------------|------| | -10% | +11% | 容易 | | -20% | +25% | 中等 | | -30% | +43% | 較難 | | -40% | +67% | 困難 | | -50% | +100% | 極難 | | -60% | +150% | 幾乎不可能 | | -70% | +233% | 幾乎不可能 | | -80% | +400% | 不可能 |

2. 核心結論

    - **跌幅越大，恢復難度呈指數級增加** - 跌 50% 需要漲 100% 才能回本 - 跌 80% 需要漲 400% 才能回本 - 控制回撤比追求報酬更重要
3. 為什麼控制 MDD 至關重要？
    - 一旦 MDD 超過 50%，回本機會渺茫 - 即使 MDD 在 30-40%，也需要大行情才能恢復 - 一般建議 MDD 控制在 20% 以內
---

#### 四、MDD 的五個重要注意事項

1. MDD 數值與未來無關

    - 歷史 MDD 不代表未來 MDD - 未來可能發生更大的回撤 - 用歷史 MDD 評估風險時要保守
範例：2022 年前長期公債 MDD 約 -23%，2022 年卻創下 -45.3%

2. 極端 MDD 可能不常重複

    - 某些 MDD 來自特殊事件（如美元金本位改變） - 這些事件可能一輩子只發生一次 - 但也可能發生新的極端事件
3. 管理 MDD 的重點在控制倉位
    - 無法控制市場波動 - 但可以控制投入資金比例 - 降低倉位 = 降低 MDD
4. MDD 不一定是單筆交易造成
    - 常見誤解：設停損就能控制 MDD - 實際：MDD 可能來自連續虧損 - 即使每筆停損，連續虧損仍可能造成大回撤
5. 執行越久，越可能遇到更大的 MDD
    - 市場總可能變得更極端 - 不要假設 MDD 上限 - 保留安全邊際
---

#### 五、MDD 在 Prop Firm 的應用

1. Prop Firm 為什麼重視 MDD？

    - Prop Firm 提供資金給交易者操作 - 交易者虧損由公司承擔 - MDD 是評估交易者風險控制能力的核心指標
2. Prop Firm 的 MDD 限制

| 類型 | 說明 | 範例 | |------|------|------| | **固定 MDD** | 以初始資金計算 | 初始 10 萬，MDD 10%，淨值不得低於 9 萬 | | **追蹤 MDD** | 隨獲利上調底線 | 帳戶漲到 11 萬，新底線 = 11 × 0.9 = 9.9 萬 |

3. 觸及 MDD 限制的後果

    - 帳戶立即終止 - 所有部位強制平倉 - 交易資格取消 - 需重新付費挑戰
4. Prop Firm 如何使用 MDD 評估交易者？
    - 保護公司資金安全 - 檢驗交易者風險管理能力 - 與獲利目標搭配衡量風險報酬比
---

怎麼應用：

實戰操作步驟：

1. **計算自己的 MDD**： - 記錄每日投資組合淨值 - 找出歷史最高點 - 計算每個谷底到最高點的回撤 - 最大值即為 MDD

2. **設定 MDD 上限**： - 一般建議：個人投資 MDD 控制在 20% 以內 - Prop Firm 挑戰：通常 MDD 上限 5-10% - 超過上限時立即減倉

3. **監控 MDD 變化**： - 每週更新 MDD 數值 - 接近上限時警覺 - 定期檢視造成 MDD 的原因

4. **降低 MDD 的方法**： - **降低單筆風險**：每筆交易風險不超過帳戶 1-2% - **分散投資**：降低單一資產權重 - **動態倉位**：市場波動大時降低倉位 - **停損紀律**：嚴格執行停損，不讓虧損擴大

不同策略的 MDD 特性：

| 策略類型 | MDD 特性 | 建議 | |---------|---------|------| | 趨勢跟蹤 | MDD 較大，但獲利也大 | 降低倉位或設更寬停損 | | 均值回歸 | MDD 較小，但賺幅也小 | 可適度增加倉位 | | 高頻交易 | MDD 通常很小 | 風險主要在系統故障 | | 波段交易 | MDD 中等 | 平衡獲利與風險 |

---

優點：

    - **直觀易懂**：直接告訴你最慘會賠多少 - **風險衡量**：量化投資組合的極端風險 - **可比較**：不同策略的風險可直接比較 - **實務應用**：Prop Firm 廣泛使用，標準明確
---

缺點/限制：

#### 一、11 種常見交易失敗案例

案例 1：高槓桿爆倉

案例 2：不設停損
    - **情境**：虧損不斷擴大，最終血本無歸 - **教訓**：停損是保命線，不是選項是必須 - **建議**：進場前先設停損，進場後不修改
案例 3：停損心理障礙
    - **情境**：頻繁停損後不敢設停損，或者設了不停執行 - **教訓**：停損是成本，不是失敗 - **建議**：接受停損是交易的一部分

    - **情境**：一天交易數十次，手續費吃掉獲利 - **教訓**：交易次數多不代表獲利多 - **建議**：減少交易頻率，提高勝率

    - **情境**：虧損後急著追回，越做越輸 - **教訓**：情緒化交易必敗 - **建議**：虧損後休息，情緒穩定再操作

    - **情境**：高頻交易，手續費和滑價吃掉獲利 - **教訓**：交易成本是隱形殺手 - **建議**：計算實際獲利時要扣除所有成本

#### 二、「韭菜指標」散戶三種情緒弱點

1. 羊群效應

---

核心內容：

#### 一、什麼是交易系統？

1. 定義

    - **交易系統**：一套明確定義的規則，告訴交易者何時進場、何時出場、何時停損停利 - **核心目的**：消除主觀判斷，讓交易可以重複執行 - **關鍵特徵**：可定義、可回測、可執行
2. 交易系統的組成要素

| 要素 | 說明 | 是否必須 | |------|------|---------| | **進場規則** | 何時買進 | 必須 | | **出場規則** | 何時賣出 | 必須 | | **停損規則** | 虧損多少出場 | 必須 | | **停利規則** | 獲利多少出場 | 建議 | | **倉位管理** | 每次投入多少資金 | 必須 | | **風險管理** | 單筆最大虧損、總體 MDD 上限 | 必須 |

3. 為什麼需要交易系統？

    - **消除情緒**：依照規則執行，不受恐懼貪婪影響 - **可重複執行**：每次條件相同，操作相同 - **可回測驗證**：可以用歷史數據驗證有效性 - **可持續改進**：知道哪裡出問題才能修正
---

#### 二、系統交易 vs 主觀交易

| 比較面向 | 系統交易 | 主觀交易 | |---------|---------|---------| | **決策依據** | 明確規則 | 經驗判斷 | | **情緒影響** | 低 | 高 | | **執行一致性** | 高 | 低 | | **回測驗證** | 可以 | 困難 | | **適應性** | 低，需定期更新 | 高，可即時調整 | | **學習門檻** | 較低 | 較高 | | **時間投入** | 初期高，後期低 | 持續高 | | **適合對象** | 新手、有紀律者 | 經驗豐富、直覺強者 |

1. 系統交易的優點

    - 100% 照規則執行，不受情緒影響 - 可以回測驗證策略有效性 - 紀律容易維持 - 適合新手快速建立框架
2. 系統交易的缺點
    - 缺乏彈性，無法應對極端情況 - 市場變化時需要更新規則 - 可能被有心人利用規則
3. 主觀交易的優點
    - 可以即時應對市場變化 - 利用自己的經驗和直覺 - 靈活度高
4. 主觀交易的缺點
    - 情緒影響大，難以保持紀律 - 經驗難以傳承 - 無法回測驗證
---

#### 三、如何設計交易系統？

步驟一：選擇交易風格

    - 當沖：日内完成進出場 - 波段：持有數天到數週 - 長期投資：持有數月到數年
步驟二：定義進場規則
    - 技術指標：如均線交叉、MACD、RSI - 價格型態：如突破、拉回買進 - 基本面條件：如財報數據、產業趨勢 - 選擇 1-3 個進場條件即可，不要太複雜
步驟三：定義出場規則
    - 停損：技術支撐跌破、固定比例虧損 - 停利：技術壓力突破、固定比例獲利 - 移動停利：根據價格移動調整 - 時間停損：持有 N 天未達目標自動出場
步驟四：定義倉位管理
    - 單筆風險：不超過帳戶 1-2% - 總倉位上限：不超過 70-80% - 分批進場：根據訊號強度調整倉位
步驟五：回測驗證
    - 使用歷史數據測試 - 檢視勝率、盈虧比、最大回撤 - 確認正期望值
步驟六：持續優化
    - 執行中記錄結果 - 定期檢討規則 - 適時調整但不頻繁更換
---

#### 四、交易系統的核心原則

原則一：可重複性

    - 規則必須明確到可以寫成程式碼 - 不同人執行應得到相同結果 - 沒有模糊地帶
原則二：正期望值
    - 期望值 = (勝率 × 平均獲利) - (敗率 × 平均虧損) - 期望值必須為正，長期才能獲利 - 樣本數必須足夠（建議 > 100 次）
原則三：風險可控
    - 每筆交易風險明確 - 總體風險有限 - 最大回撤在可接受範圍
原則四：適合自己
    - 考慮時間可以投入多少 - 考慮風險承受能力 - 考慮個性特質
---

    - 風險管理是基礎，沒有風險管理一切白談 - 策略是工具，幫助找到進出場點 - 心態是執行，再好的策略執行不下去也沒用

出場條件： - 停損：跌破進場日均線 - 停利：達到壓力位 或 移動停利 10%

倉位管理： - 單筆風險：帳戶 2% - 最大倉位：帳戶 60% ```

4. **持續改進**： - 每月檢視交易日誌 - 找出規則的問題 - 微調但不要頻繁更換

優點：

    - **消除情緒**：照規則執行，不受恐懼貪婪影響 - **可驗證**：可以用歷史數據驗證有效性 - **可重複**：每次條件相同，操作相同 - **可改進**：知道哪裡出問題才能修正

    - **缺乏彈性**：無法應對規則外的情況 - **需要回測**：新手可能不會回測 - **執行困難**：知道規則和執行規則是兩回事 - **市場變化**：市場改變時規則可能失效

核心內容：

#### 一、資金曲線的類型

1. 理想資金曲線

    - **特徵**：穩定上升，波動小 - **含義**：風險控制良好，策略穩定 - **指標**：夏普比率 > 1，MDD < 20%
2. 高波動資金曲線
    - **特徵**：大漲大跌，波動劇烈 - **含義**：風險過高，可能爆倉 - **問題**：心臟受不了，容易在高點低點做錯決策
3. 向下資金曲線
    - **特徵**：持續下跌，偶爾反彈 - **含義**：策略無效或執行有問題 - **解決**：停止交易，檢討策略
4. 震盪資金曲線
    - **特徵**：上上下下，沒有明顯方向 - **含義**：策略期望值為零或接近零 - **問題**：白做工，交易成本還賠錢
---

#### 二、資金曲線管理的核心原則

原則一：穩定勝過暴利

    - 年報酬 20% 優於大賺大賠 - 長期複利的力量 - 10 年 20% 報酬 = 6.19 倍
原則二：控制回撤
    - 回撤 20% 需要 25% 漲幅恢復 - 回撤 50% 需要 100% 漲幅恢復 - 控制回撤比追求報酬更重要
原則三：資金來源管理
    - 自有資金：壓力可控 - 親友資金：壓力來自他人期望 - 借貸資金：壓力最大，利息成本高
原則四：資金效率三要素
    - **獲利能力**：策略的期望值 - **風險控制**：最大回撤、波動度 - **時間效率**：持有時間、機會成本
---

#### 三、資金三類型來源與管理

類型一：自有資金

    - **優點**：無利息成本、無壓力、完全自主 - **建議**：適合長期穩健投資
類型二：親友資金
    - **優點**：資金較大、利息可能較低 - **風險**：虧損時關係緊張、決策壓力大 - **建議**：清楚說明風險、設定虧損上限
類型三：借貸資金（槓桿）
    - **優點**：放大獲利 - **風險**：利息成本、追加保證金、爆倉風險 - **建議**：新手不建議，老手也要控制槓桿
---

#### 四、系統性風險 vs 非系統性風險

1. 系統性風險

定義：影響整個市場的風險，無法透過分散投資消除

來源：

    - 經濟衰退 - 通貨膨脹 - 利率政策 - 地緣政治 - 自然災害
特徵：
    - 所有股票都會受到影響 - 相關性在極端時趨近 1 - 分散投資無法完全消除
應對：
    - 資產配置（股票 + 債券 + 現金） - 降低倉位（保留現金） - 避險工具（期貨、選擇權、VIX）
2. 非系統性風險

定義：個別公司或產業特有的風險，可透過分散投資消除

來源：

    - 公司財務問題 - 管理層決策失誤 - 產業政策變化 - 技術被取代 - 法律訴訟
特徵：
    - 只影響特定公司或產業 - 分散投資可以有效降低
應對：
    - 持有多檔股票（10-20 檔） - 分散不同產業 - 分散不同地區
---

#### 五、風險分散的三大維度

維度一：資產類別分散

| 資產類別 | 特性 | 風險等級 | |---------|------|---------| | 股票 | 高報酬、高波動 | 高 | | 債券 | 固定收益、低波動 | 低 | | 現金 | 無風險、低報酬 | 最低 | | REITs | 不動產收益 | 中高 | | 黃金 | 避險資產 | 中 | | 加密幣 | 極高波動 | 極高 |

配置建議：

    - 保守型：股票 30% + 債券 50% + 現金 20% - 平衡型：股票 50% + 債券 30% + 其他 20% - 積極型：股票 70% + 債券 20% + 其他 10%
維度二：地區分散

地區配置建議：

    - 本國股票：40-60% - 美國股票：20-40%

#### 六、Beta 值與 Alpha 值

1. Beta 值

定義：衡量資產相對於大盤的波動程度

解讀：

    - Beta = 1：與大盤同步波動 - Beta > 1：波動比大盤大（高風險高報酬） - Beta < 1：波動比大盤小（低風險低報酬） - Beta < 0：與大盤反向（如放空部位）
實務應用：
    - Beta 高的股票適合多頭市場 - Beta 低的股票適合空頭市場或保守配置
2. Alpha 值

定義：衡量資產相對於大盤的超額報酬

解讀：

    - Alpha > 0：表現優於大盤 - Alpha = 0：表現與大盤相同 - Alpha < 0：表現劣於大盤
實務應用：
    - Alpha 來自選股能力 - 主動型基金目標是創造正 Alpha - 被動型基金目標是追蹤大盤（Alpha = 0）

1. 尾端風險

定義：極端事件發生的機率和影響

特徵：

    - 機率低但影響大 - 無法用常態分配預測 - 分散投資失效
應對：
    - 保留現金緩衝 - 設定最大虧損上限 - 購買保險（認沽期權）
2. 斷頭風險

定義：被迫在不合理價格清算部位

原因：

    - 槓桿過高 - 保證金不足 - 流動性危機
應對：
    - 控制槓桿比例 - 預留保證金緩衝 - 避免過度集中部位

2. 設停損的重要原則（5 點） 1. **進場前就要設好停損**：預備買股票時，除了設定買進價位，也要設好決策錯誤時認賠出場的價位 2. **停損點不得更改**：設好後不能改，否則等於沒設 3. **收盤跌破要立即出場**：做多跌破停損價要出場，做空突破停損價要出場，必須絕對遵守紀律 4. **以順勢交易為主**：為了減少停損次數，在高勝率位置進場 5. **停損不超過 10%**：以不超過 10% 為標準

#### 二、五大停損方法（具體操作方式）

1. K 線停損法

    - **做多操作**：以進場當天 K 線的最低點為停損點 - **做空操作**：以進場當天 K 線的最高點為停損點 - **步驟**：
1. 進場當天記錄 K 線高低點 2. 做多：收盤價跌破進場日 K 線最低點 → 次日開盤立即出場 3. 做空：收盤價突破進場日 K 線最高點 → 次日開盤立即出場
    - **適用情境**：短線操作、當沖交易
2. 趨勢停損法
    - **做多時機與停損設定**：
- 回檔後買上漲：停損設在轉折波的低點 - 盤整突破買上漲：停損設在突破點
    - **做空時機與停損設定**：
- 反彈後空下跌：停損設在轉折波的高點 - 盤整跌破空下跌：停損設在跌破點

3. 均線停損法

    - **具體操作**：以進出依據的均線設停損 - **步驟**：
1. 明確進場依據（如：股價站上 5 日均線買進） 2. 停損設定為進場時的均線價格 3. 收盤跌破該均線價 → 出場

4. 固定比例停損法

    - **具體操作**：以買進價的 2%~10% 為停損點 - **步驟**：
1. 決定停損比例（依個人風險容忍度） 2. 華爾街操盤高手標準：不超過 10% 3. 一般建議：固定 5% 4. 計算停損價 = 買進價 × (1 - 停損比例)

核心內容：

#### 一、對沖交易的定義與核心原理

1. 什麼是對沖交易？

    - **定義**：為一項可能因市場價格波動而產生虧損的資產，建立一個與其風險敞口相反的投資頭寸 - **核心目的**：
- 主要為風險管理，而非直接獲利 - 鎖定利潤或限制虧損 - 穩定投資組合價值
    - **比喻**：就像買一份保險，付出一定成本換取市場逆轉時的保障
2. 對沖如何有效降低風險？
    - **原理**：金融工具之間的負相關性或可預測的價格關係 - **實例說明**：
- 持有 100 萬港元香港藍籌股 - 擔心未來一個月股市回調 - 沽空等值恒生指數期貨合約 - 股市下跌：股票虧損，期貨獲利，相互抵銷 - 結果：降低整體投資組合波動性

3. 常用對沖工具

    - 期貨 - 期權 - 差價合約（CFD） - 遠期合約
---

#### 二、對沖交易 vs 套利交易（根本差異）

| 比較維度 | 對沖交易 (Hedging) | 套利交易 (Arbitrage) | |---------|-------------------|---------------------| | 核心目標 | 降低或鎖定現有頭寸的風險 | 利用定價錯誤賺取無風險利潤 | | 風險態度 | 風險規避 | 風險中性 | | 操作前提 | 已持有一個風險敞口 | 發現同一資產在不同市場存在價差 | | 利潤來源 | 減少虧損來保護資產 | 從價差中直接獲利 | | 常見例子 | 持有股票 + 買入認沽期權 | A 交易所買入 $10，B 交易所賣出 $10.05 |

關鍵差異：

    - 對沖 = 「避險」，保護資產 - 套利 = 「無風險獲利」，從價差賺錢

#### 三、三大對沖策略實戰教學

策略一：外匯對沖

1. 直接對沖（Direct Hedge）

    - **適用對象**：持有外幣資產的投資者、國際貿易企業 - **操作方式**：
- 持有美元多頭頭寸（如買入 USD/JPY） - 預期美元短期回調 - 建立等值美元空頭頭寸（如賣出 EUR/USD） - 兩個頭寸損益相互抵銷，鎖定風險
    - **效果**：無論美元升值或貶值，整體損益接近零
2. 利用衍生工具對沖
    - **遠期外匯合約（Forward Contracts）**：
- 與銀行簽訂合約鎖定未來匯率 - 實例：香港出口商預計 3 個月後收到 100 萬美元貨款 - 鎖定 7.80 匯率將美元兌換成港元 - 規避美元貶值風險
    - **外匯期權（Currency Options）**：
- 買入認沽期權鎖定賣出匯率上限 - 匯率不利時行使期權保護

---

策略二：期貨與期權對沖

1. 期貨對沖

    - **適用對象**：農產品生產商、航空公司、股票投資者 - **實例說明**：
- 航空公司擔心油價上漲 → 買入原油期貨 - 油價上漲：燃料成本增加，但期貨獲利彌補 - 股票投資者預期大市下跌 → 沽空指數期貨 - 股市下跌：股票虧損，期貨獲利抵銷

2. 期權對沖（Protective Put）

    - **定義**：股票投資者最直接的「保險」策略 - **操作方式**：
1. 持有某股票，現價 $100 2. 擔心下跌但不想賣出 3. 買入行使價 $95 的認沽期權（Put Option） 4. 股價跌至 $90：股票虧損 $10，期權價值上漲彌補虧損
    - **成本**：期權金（保險費用） - **效果**：將最大虧損鎖定在行使價附近
---

策略三：配對交易（Pairs Trading）

1. 核心概念

    - **性質**：市場中性策略 - **目標**：從兩個高度相關資產的「短期失衡」中獲利 - **特點**：同時對沖掉市場整體的系統性風險
2. 操作邏輯
    - **步驟一：識別配對**
- 找出兩隻歷史上價格走勢高度相關的股票 - 例如：兩家本地銀行股、兩家業務相似的科技股
    - **步驟二：發現偏離**
- 監測兩者股價的價差或比率 - 當比率偏離歷史平均水平時，出現交易機會
    - **步驟三：執行交易**
- 買入（做多）表現相對較弱的股票 - 沽空（做空）表現相對較強的股票 - 等待價差回歸正常水平

3. 配對交易實例

    - A 銀行和 B 銀行股價長期保持 1.2:1 比率 - A 銀行股價大漲，比率擴大到 1.4:1 - 操作：沽空 A 銀行 + 買入 B 銀行 - 結果：無論市場漲跌，只要價差回歸 1.2:1 就能獲利
4. 配對交易的優勢
    - 對沖掉市場系統性風險 - 不受大市漲跌影響 - 從「相對價值失衡」中獲利

#### 四、對沖交易的優點與潛在風險

優點：

1. 降低下行風險

    - 在市場動盪或預期回調時，顯著減少投資組合價值回撤 - 保護資本免受極端市場波動衝擊
2. 心理穩定性
    - 知道資產有保護，在市場恐慌時保持冷靜 - 避免因情緒化做出錯誤決策
3. 鎖定利潤
    - 持有頭寸已有可觀未實現利潤 - 不想立即平倉，可通過對沖鎖定大部分利潤 - 防止市場突然逆轉
缺點與風險：

1. 交易成本（Transaction Costs）

    - 經紀佣金 - 買賣差價（點差） - 期權金 - 這些成本直接侵蝕投資回報
2. 限制上行潛力
    - 對沖在保護虧損的同時，也限制獲取更高利潤的可能 - 股票大漲 + 買了認沽期權 → 收益扣除期權金 - 完美對沖可能完全抵銷標的資產的任何收益
3. 基差風險（Basis Risk）
    - 對沖工具與持有資產並非 100% 完美負相關 - 價格關係可能發生變化 - 導致對沖效果不佳 - 極端情況：兩邊都虧損
4. 操作複雜性
    - 期貨、期權等衍生工具具有相當複雜性 - 不當的對沖操作可能放大風險 - 需要充分理解相關工具的運作機制

#### 五、散戶如何進行對沖交易？

1. 可用工具

    - 期貨帳戶：交易指數期貨、商品期貨 - 期權帳戶：買入認沽期權保護持股 - 差價合約（CFD）：靈活做多做空各種資產 - 遠期合約：銀行提供的匯率鎖定服務
2. 散戶對沖步驟
    - **步驟一**：評估持倉風險 - **步驟二**：選擇適合的對沖工具 - **步驟三**：計算對沖比例（需要多少對沖頭寸） - **步驟四**：執行對沖操作 - **步驟五**：持續監控並適時調整
3. 注意事項
    - 對沖不是穩賺不賠策略 - 需要支付交易成本 - 槓桿工具風險較高 - 充分了解工具運作後再操作
---

#### 六、對沖基金是什麼？

1. 定義

    - 專門為高淨值人士或機構投資者服務的私募基金 - 利用對沖交易等複雜策略，在市場不論升跌的情況下爭取絕對回報
2. 現代對沖基金策略
    - 全球宏觀 - 量化交易 - 事件驅動 - 多空策略 - 配對交易
3. 與個人對沖的關係
    - 「對沖交易」是對沖基金使用的工具之一 - 現代對沖基金策略遠超單純的風險對沖 - 旨在實現高於市場平均的回報 - 伴隨較高的風險和門檻

2. **執行沽空**： - 沽空指數期貨 - 記錄開倉價格

3. **持續監控**： - 定期評估對沖效果 - 市場變化時調整對沖比例

2. **選擇對沖工具**： - 遠期合約：鎖定匯率 - 外匯期權：鎖定匯率上限/下限

3. **執行對沖**： - 與銀行簽訂合約 - 支付期權金（如適用）

4. **到期結算**： - 按約定匯率兌換 - 或放棄期權（如更有利）

核心內容：

#### 一、VaR（風險價值）基礎概念

1. VaR 是什麼？

    - **全名**：Value at Risk（風險價值） - **定義**：資產組合在持有期間內，在給定的信賴區間內，由於市場價格變動所導致的最大預期損失的數值 - **核心功能**：將所有風險彙整成一個數值，讓管理階層能簡單明瞭地知道資產組合可能面臨之最大損失 - **直觀解讀**：「有 95% 的信心在未來 N 天內，最大損失不會超過 V 元」
2. VaR 的應用場景
    - 金融機構資本適足率計算 - 企業內部風險控管 - 資產配置的參考依據 - 投資組合風險量化
3. VaR 的歷史背景
    - 1980年代末至1990年代初，美國銀行業遭受商業風險困擾 - JP 摩根公司發明 VaR 方法 - 1996年巴塞爾修正協定將 VaR 作為市場風險衡量指標 - 芝加哥商品交易所於1988年推出 SPAN 系統，成為計算投資組合保證金的全球行業標準
4. VaR 與傳統風險測度的差異
    - **傳統風險測度**：以標準差為代表，關注整體波動 - **VaR 優勢**：著重資產下行風險，特別是極端情況下的損失規模
---

#### 二、VaR 的三種計算方法

方法一：變異數-共變異數法（Delta Normal）

原理：

    - 假設風險因子為常態分配 - 報酬率之對數為常態分配 - 常態分配的線性組合仍為常態分配
計算步驟： 1. 計算股價報酬率（連續型報酬） 2. 計算投資組合變異數矩陣 3. 套用公式計算 VaR

股票投資組合 VaR 公式： ``` VaR = Z × σ × √T × V ```

    - Z：信賴水準對應的 Z 值（95% = 1.645，99% = 2.326） - σ：投資組合標準差 - T：持有期間（天） - V：投資組合價值
優點：
    - 計算效率高 - 風險因子數量多且相互獨立時適用 - 易於處理增額風險值（Marginal VaR）
缺點：
    - 無法妥善處理厚尾問題（極端事件） - 需要估計波動度及報酬相關係數 - 不適合處理選擇權等非線性資產
---

方法二：歷史模擬法

原理：

    - 利用實際歷史資料求算資產組合風險值 - 假設未來資產報酬走勢與過去大致相同 - 無需對報酬分配做任何假設
計算步驟： 1. 收集持有一段時間的歷史價格資料（如過去100天） 2. 計算每種資產的損益報酬率 3. 利用歷史報酬率模擬未來可能報酬 4. 將投資組合可能報酬由小至大排序 5. 依信賴水準找出對應百分位數

實例：

    - 100 筆損益資料 - 信賴水準 95% - 第 5 筆資料（100 × 5%）即為風險值
優點：
    - 不需對風險因子作任何假設 - 不需估計波動度與相關係數 - 可考量厚尾分配及極端事件
缺點：
    - 完全仰賴特定歷史資料集 - 極端事件可能扭曲估計結果 - 未考量市場結構改變 - 資料期過短可能導致偏差
---

方法三：蒙地卡羅模擬法

原理：

    - 假設資產價格變化符合特定隨機過程 - 股價常用幾何布朗運動（Geometric Brownian Motion） - 重複模擬產生足夠情境
隨機過程公式： ``` dS/S = μdt + σdW ```
    - S：股價 - μ：預期報酬率 - σ：波動度 - dW：布朗運動
計算步驟： 1. 建立價格走勢隨機過程，估計參數（平均數、波動度、相關係數） 2. 從標準常態分配隨機抽取 ε，產生下一期股價 3. 重複 N 次（如 1000 次），產生投資組合報酬分配 4. 依信賴水準找出對應百分位數

優點：

    - 可以包含任何風險因子分配 - 將任何複雜投資組合模組化 - 可執行壓力測試與敏感度分析 - 模型設定具彈性
缺點：
    - 隨機過程設定錯誤會產生模型風險 - 電腦資源需求高 - 極端值未能包含於分配中
---

#### 四、VaR 的限制與補強

VaR 的限制： 1. **未考量超門限損失**：VaR 只告訴你最大預期損失，不告訴你超過這個損失會怎樣 2. **不滿足次可加性**：不是一致風險測度，可能出現投資組合 VaR 大於個別資產 VaR 之和 3. **模型風險**：參數估計誤差會影響結果 4. **常態分配假設**：實際市場報酬有厚尾現象

補強方法：

1. 壓力測試（Stress Testing）

    - 分析市場出現異於常態的極端情況 - 模擬歷史重大事件（如金融海嘯）的衝擊 - 評估投資組合在極端壓力下的表現
2. 情境分析（Scenario Analysis）
    - 設計特定市場情境 - 評估不同壓力情境下的潛在損失
3. 回溯測試（Backtesting）
    - 驗證 VaR 模型的準確性 - 比較每日真實報酬與 VaR 預測 - 調整模型參數

#### 五、最大回撤（MDD）定義與計算

1. MDD 是什麼？

    - **全名**：Maximum Drawdown（最大回撤） - **定義**：在特定期間內，資產淨值從歷史最高點跌至最低點的最大跌幅 - **直觀理解**：投資過程中可能出現的最大資金虧損比例
2. MDD 計算公式
    - **步驟一**：計算所有回檔
- 回檔 = (投資組合最高價值 - 最高價值後的最低價值) ÷ 投資組合最高價值 × 100%
    - **步驟二**：找出所有回檔中最大的數值
- MDD = MAX(所有回檔)

3. 實例說明

    - 帳戶起始資金：5 萬美元 - 漲到高點：10 萬美元 - 跌至谷底：7 萬美元 - 再漲至新高：12 萬美元 - 再跌至：9 萬美元
計算：
    - 第一次回撤：(10 - 7) ÷ 10 = 30% - 第二次回撤：(12 - 9) ÷ 12 = 25% - MDD = MAX(30%, 25%) = 30%
---

#### 六、MDD 的核心意義

1. 回撤恢復難度

| 回撤幅度 | 恢復所需漲幅 | |---------|------------| | 10% | 11% | | 20% | 25% | | 30% | 43% | | 40% | 67% | | 50% | 100% | | 60% | 150% | | 70% | 233% | | 80% | 400% | | 90% | 900% |

關鍵洞察：

    - 回撤 50% 需要漲 100% 才能回本 - 回撤越大，恢復難度呈指數級增加 - 控制回撤幅度是資金管理的核心
2. MDD 告訴投資者什麼？
    - 最糟糕情況下的可能虧損 - 跌多久（回撤持續時間） - 多久能回本（恢復時間）
3. 投資績效評估
    - 年化報酬相同時，MDD 較低的策略更優 - 用較小的風險換取相同的報酬

#### 八、MDD 在 Prop Firm 考核中的應用

1. Prop Firm 風控規則

    - 每日最大虧損限制（如 5%） - 總體最大回撤限制（如 10%） - 帳戶淨值不得低於初始資金的 90%
2. 追蹤回撤 vs 固定回撤

| 類型 | 定義 | 特點 | |-----|------|------| | 固定回撤 | 以初始資金為基準 | 回撤底線固定不變 | | 追蹤回撤 | 以當前高點為基準 | 隨獲利增加而上移 |

實例比較：

    - 初始資金：10 萬美元 - 10% 固定回撤底線：9 萬美元（固定） - 10% 追蹤回撤：
- 帳戶賺到 11 萬 → 底線上移到 9.9 萬 - 帳戶賺到 12 萬 → 底線上移到 10.8 萬

3. 為什麼 Prop Firm 重視 MDD？

    - 保護公司資金安全 - 檢驗交易者風險管理能力 - 與盈利目標搭配衡量風險報酬比 - 違反回撤規則通常會終止帳戶
---

#### 九、使用 MDD 的四個重要注意事項

1. MDD 與未來無關

    - MDD 只能根據歷史得出過去最大回撤 - 未來 MDD 可能比歷史更大 - 不能用歷史 MDD 推估未來 - 例：2022年前長期公債 MDD 僅 -23.1%，2022年突破至 -45.3%
2. MDD 可能來自極端市場狀況
    - 某些 MDD 不是常態，而是特定條件 - 例：黃金 1980-2000 年的 MDD 來自金本位結束 + 石油危機 - 這些事件在未來很難複製
3. 管理 MDD 的關鍵是控制部位大小
    - 影響 MDD 的要素：持有時間、交易頻率、部位大小、市場波動 - 無法控制所有要素 - 有效方法：控制部位大小 - 透過資產配置控制風險
4. MDD 不一定是單筆交易造成
    - 迷思：設定停損就能改善 MDD - 事實：MDD 常是連續虧損造成，非單筆交易 - 停損設得越緊，越容易觸發，虧損越容易發生 - 任何策略只要執行夠久，都有機會遇到更大的 MDD
---

#### 十、VaR 與 MDD 的實務應用

VaR 應用場景：

1. 金融機構風險控管

    - 銀行資本適足率計算 - 投資組合風險限額設定 - 部門風險預算分配
2. 投資組合管理
    - 評估不同投組的風險 - 選擇風險調整後報酬最佳的配置
3. 衍生性商品
    - 選擇權風險值計算 - 期貨保證金估算
MDD 應用場景：

1. 交易策略評估

    - 比較不同策略的 MDD - 選擇報酬相同但 MDD 較低的策略
2. 基金經理人選擇
    - 年化報酬相同時，選擇 MDD 較低者 - 評估基金經理人的風險控制能力
3. Prop Firm 考核
    - 評估交易者是否能守住風險底線 - 測試風險管理紀律
4. 個人投資
    - 評估自己能承受的最大虧損 - 設定心理容忍度
---

怎麼應用：

VaR 實戰計算步驟（變異數-共變異數法）：

1. **收集資料**： - 取得持股的歷史價格資料 - 計算日報酬率

2. **計算參數**： - 報酬率平均值（μ） - 報酬率標準差（σ） - 持有期間（T）

3. **設定信賴水準**： - 95% 信賴水準 → Z = 1.645 - 99% 信賴水準 → Z = 2.326

4. **計算 VaR**： ``` VaR = Z × σ × √T × 投組價值 ```

5. **解讀結果**： - 「有 95% 的信心在未來 1 天內，最大損失不會超過 VaR 元」

MDD 實戰管理步驟：

1. **計算自己的 MDD 容忍度**： - 評估心理能承受的最大虧損 - 一般建議控制在 20% 以內

2. **設定資金管理規則**： - 單筆交易風險 1-2% - 總部位控制在可接受範圍

3. **監控帳戶淨值**： - 定期計算當前回撤 - 接近 MDD 限制時減倉

4. **策略評估**： - 回測時計算策略 MDD - MDD 超過容忍度的策略不採用

5. **建立復原機制**： - 回撤發生時的因應方案 - 分批加碼或等待恢復的決策

---

優點：

VaR 優點：

    - **簡化風險**：將複雜風險量化為單一數值 - **可比較性**：不同資產、策略的風險可以比較 - **監管標準**：巴塞爾協定認可的風險指標 - **管理工具**：可用於風險限額設定與預算分配
MDD 優點：
    - **直觀易懂**：直接告訴投資者最大虧損 - **心理準備**：幫助投資者了解最壞情況 - **策略比較**：評估策略風險調整後報酬 - **風控基準**：Prop Firm 和基金的風險標準
---

缺點/限制：

VaR 缺點：

    - **不告訴超限損失**：只知門檻，不知超過後如何 - **模型風險**：參數估計誤差影響結果 - **常態分配假設**：實際市場有厚尾現象 - **計算複雜**：蒙地卡羅法需要專業知識
MDD 缺點：
    - **歷史資料限制**：未來 MDD 可能更大 - **不預測未來**：只反映過去，不預測未來 - **計算期間依賴**：不同期間可能得到不同結果 - **停損不一定改善**：MDD 常是連續虧損造成
---

散戶賠錢的行為模式：

| 情境 | 散戶行為 | 結果 | |------|----------|------| | **賺錢時** | 有賺就跑，很少擴大部位 | 獲利有限 | | **賠錢時** | 捨不得、不認輸、不甘心 | 虧損擴大 |

    - 散戶留倉方向與趨勢方向呈**反向關係** - 市場上漲時散戶看空，市場下跌時散戶看多 - 散戶賠錢比例高達 90%~95%

#### 一、資金曲線管理

資金來源三類型：

| 類型 | 說明 | 風險 | |------|------|------| | **在庫現金** | 閒置未使用的資金 | 過多=效率不佳，過少=錯失機會 | | **資產變現** | 從持有資產變現獲得資金 | 需考慮流動性與變現效率 | | **融資融券** | 向券商借款/借股 | 高槓桿，新手應避免 |

資金配置策略：

    - **初期靜態配置**：資金較少時，預留閒置資金後全數投入 - **動態配置**：資金變多後分散至貨幣市場、債券、股市
範例配置：
    - 10%：當沖日結損益 - 20%：短期投資（5 分 K） - 30%：中期投資（小時/日 K） - 40%：長期投資（基本面分析）
資金效率三要素：

| 要素 | 定義 | 關鍵 | |------|------|------| | **充分性** | 資金使用程度 | 避免閒置過多或槓桿過高 | | **有效性** | 投入是否有效益 | 高獲利機率、高獲利率 | | **流動性** | 資金週轉效率 | 週轉越快，資金累積越快 |

---

#### 二、系統性 vs 非系統性風險

系統性風險（市場風險）：

    - 定義：整個市場都會出現的風險 - 特徵：無法透過分散投資消除 - 來源：金融風暴、經濟崩盤、天災、戰爭、政治 - 應對：分散資產（股票+債券+黃金+REITs 等）
非系統性風險（個股風險）：
    - 定義：單一股票/企業特有的風險 - 特徵：可透過分散投資消除 - 來源：財報不佳、高層舞弊、競爭對手出現 - 應對：投資多檔股票
Beta 值（衡量系統性風險）：
    - β > 1：波動比市場大（高風險高報酬） - β < 1：波動比市場小（保守穩健） - β = 1：與市場同步
Alpha 值（衡量非系統性風險）：
    - α 高：扣除系統性風險後可獲得超額報酬 - 與公司基本面、營運狀況、管理能力相關
---

#### 三、尾端風險（Tail Risk）

定義：

    - 發生機率極低，但一旦發生衝擊巨大的極端事件 - 投資收益偏離均值超過 3 個標準偏差 - 「黑天鵝中的黑天鵝」
風險測度方法： 1. **VaR（風險價值）**：給定信賴區間內最大預期損失 2. **ES（期望損失）**：超過 VaR 後的預期損失 3. **TCE（尾部條件期望）**：考慮尾部分佈的風險

應對策略：

    - 保留現金（逢低買進） - 分散資產配置 - 尾部避險策略（保護性認沽期權） - 降低槓桿
---

#### 四、斷頭風險與槓桿控制

斷頭定義：

    - 保證金帳戶低於維持保證金 - 券商強制賣出資產（強制平倉） - 也稱：爆倉、強制賣出、強制平倉
斷頭後果：
    - **個人**：損失可能超過保證金，產生債務 - **市場**：多殺多惡性循環
降低斷頭風險四方法：

| 方法 | 說明 | |------|------| | **降低槓桿倍數** | 放充足保證金（建議 100% 商品價值） | | **設定停損點** | 進場前設好停損，達到就出場 | | **分散風險** | 分批進場，不要 All In | | **自行賣出** | 主動賣出提高保證金，而非被動斷頭 |

台灣期貨斷頭門檻：

    - 盤中：風險指標低於 25% → 立即強制平倉全部部位 - 盤後：追繳後未補錢 → 次日強制平倉
台灣股票斷頭門檻：
    - 融資維持率低於 130% → 追繳通知 - T+2 日內需補至 166% 以上 - 未補足 → T+3 日強制賣出
安全保證金原則：
    - 不要只放原始保證金 - 準備超出安全範圍很多 - 新手建議：100% 商品價值保證金（零槓桿）
---

#### 五、回測驗證

定義：用歷史數據測試交易策略的過去表現

為什麼需要回測：

    - 驗證假說、排除無效策略 - 了解策略風險報酬特性 - 了解策略在不同行情下的表現 - 提高策略信心 - 減少人性對投資的影響
回測五步驟：

| 步驟 | 說明 | |------|------| | **1. 了解獲利來源** | 你在賺什麼錢？為什麼策略長期可行？ | | **2. 清楚定義策略** | 進場條件、濾網、出場條件、風險控制、資金控管 | | **3. 收集歷史數據** | 價格、指標、財務數據，時間需足夠長 | | **4. 進行回測** | 套用歷史數據模擬交易 | | **5. 結果分析** | 評估報酬率、勝率、MDD、夏普率等 |

有效策略的判斷指標：

    - 績效曲線從左下到右上 - 平均單筆獲利扣除成本後仍足夠 - 每年正報酬或有明確原因的負報酬年度 - MDD 不會對帳戶造成破壞性打擊
回測核心指標：

| 類型 | 指標 | |------|------| | **收益** | 總報酬率、年化報酬率、平均報酬 | | **風險** | MDD、波動率、夏普率、索丁諾比率 | | **勝率盈虧** | 勝率、盈虧比 |

回測限制：

    - 過去績效 ≠ 未來績效 - 過度最佳化風險（overfitting） - 交易成本與滑價影響 - 未來函數問題（用到未來資料）
---

怎麼應用：

實戰檢查清單：

槓桿控制： 1. 計算維持保證金水位 2. 保證金準備至少 2-3 倍原始保證金 3. 設定風險指標警示（如 50% 以下通知） 4. 監控盤中風險指標變化

尾端風險應對： 1. 保留 10-20% 現金部位 2. 逢低加碼策略而非追高 3. 考慮保護性認沽期權（大資金） 4. 分散資產配置

回測驗證步驟： 1. 明確定義策略規則（可程式化） 2. 收集足夠長的歷史數據（建議 5-10 年） 3. 確認數據品質與清理 4. 加入交易成本與滑價 5. 分析績效指標 6. 驗證策略邏輯是否合理

優點：

    - **系統性理解**：完整掌握風險管理框架 - **實用性高**：可直接應用於交易決策 - **量化指標**：用數據而非感覺做決策 - **保護資金**：避免斷頭、過度槓桿
---

缺點/限制：

    - **執行困難**：知道 ≠ 做到 - **回測陷阱**：過度最佳化、未來函數 - **心理障礙**：槓桿誘惑難以抗拒 - **時間成本**：數據收集與回測需要時間

第 15 回合：交易系統設計

一、什麼是交易系統？

定義：一套規則和程序的組合，幫助你決定什麼時候買、什麼時候賣。

核心目的：

    - 明確交易時機 - 控制風險 - 保持紀律 - 避免情緒化決策
為什麼需要交易系統：
    - 沒有系統 = 沒有地圖闖入叢林 - 情緒驅使的衝動決策 = 虧損 - 明確規則 = 長期致勝關鍵
---

二、交易系統三大核心要素

核心觀念：一套完整且符合科學依據的交易系統，三大核心缺一不可。

| 核心 | 說明 | 關鍵問題 | |------|------|----------| | **交易策略** | 進出場訊號 | 何時買？何時賣？ | | **資金管理** | 倉位規模定義 | 能開多大倉位？能虧多少？ | | **個人心理** | 執行的心理支柱 | 能否堅定執行？ |

---

三、交易策略：三大量化要素

開發交易策略後，需統計三大要素：

| 要素 | 說明 | 計算方式 | |------|------|----------| | **交易勝率** | 策略成功率 | 獲利次數 / 總交易次數 | | **平均盈虧比** | 平均獲利 / 平均虧損 | 不是單筆，是長期平均 | | **交易頻率** | 交易次數 | 單位時間內的交易數 |

期望值公式： ``` 期望值 = [勝率 × 平均盈虧比] - [失敗率 × 1] ```

    - 期望值必須為**正數**才值得交易 - 數值越高越好 - 勝率和盈虧比理論上越高越好，但同步提升困難
---

四、資金管理：倉位規模定義

核心問題： 1. 我們能夠開多大倉位？ 2. 我們能夠虧損多少？

金鑰原則：

    - 必須是**準確數值**，不是理論數值 - 因人而異（資金情況、心理承受能力不同） - 核心作用：長期生存、破除破產風險、盈利時擴大收益
---

五、交易金字塔：安全、期望值、績效

交易金字塔三大核心：

``` △ 績效 ╱ ╲ ╱期望值╲ ╱─────────╲ ╱ 安全 ╲ ╱─────────────╲ ```

#### 5.1 安全 = 單筆風險 × 勝率

公式：`安全 = 單筆風險 / 總資金 × 連輸次數機率`

爆倉機率標準：至少要到小數點後第 9 位（比中樂透還低）

案例： | 情境 | 勝率 | 單筆風險 | 安全邊界 | 連輸爆倉次數 | 爆倉機率 | |------|------|----------|----------|--------------|----------| | 阿特（修正前） | 80% | 30% | 20% | 5 筆 | 0.03%（太高！）| | 阿特（修正後） | 80% | 5% | 20% | 32 筆 | 0.000...（合格）|

核心洞察：單筆風險對策略影響遠大於勝率！勝率無法直接控制，但單筆風險可以！

---

六、賭場四大法寶：盈利交易規則的基礎

賭場為什麼永遠賺錢？四大盈利關鍵：

#### 6.1 期望值優勢

| 賭局 | 賭場勝率 | 盈虧比 | |------|----------|--------| | 骰子押大小 | 略高於 50%（豹子通殺） | 1:1 |

    - 只要樣本數夠大，賭場絕對賺錢 - 所有交易策略精進，目的都是「打破 50% 勝率 + 1:1 盈虧比」的隨機狀態
新手最大錯誤：過分追求勝率，犧牲盈虧比

建議：

    - 勝率落在 30-70% 都正常 - 更合理的施力點：**放大盈虧比** - 「保留參與趨勢的尾倉」永遠必要（至少 25%）
---

#### 6.2 可重複性 & 獨立事件

可重複性：

    - 策略足夠客觀、定義嚴謹 - 每次執行邏輯相同 - 盤感、小道消息、衝動交易 → 不具可重複性
獨立事件：
    - 上一次交易失敗，不影響這次的執行 - **最高指導原則**：除非有進行規則更改，否則都應依照規則執行 - 不要每做完一筆就優化，會污染交易樣本
---

#### 6.3 單筆虧損上限

賭場智慧：不接受壓身家的賭局

    - 每張賭桌都有押注上限 - 上限是賭場可承受的範圍 - 賭場從不擔心賭客贏錢
交易應用：
    - 通常建議：帳戶金額的 **1-2%** - 最少最少：可以有 50 筆交易機會 - 不要把輸贏壓在任何一筆交易
---

#### 6.4 大數法則

前面三件事成立後，賭場只需：**確保有越多場賭局越好**

關鍵洞察：

    - 策略有期望值優勢 → 大數法則是朋友 - 策略期望值不 OK → 大數法則是敵人（做越多賠越多）
反面案例：
    - 每次冒 $100 風險，固定賺 $5 就跑 - 10 次對 9 次（勝率 90%） - 數學：`9 × 5 - 100 = -55` - 這正是「小賺沾沾自喜，一次虧掉前面所有」

七、系統交易 vs 主觀交易

| 面向 | 主觀交易 | 系統交易 | 量化交易 | |------|----------|----------|----------| | 決策方式 | 經驗、直覺、盤感 | 明確規則（可人工執行） | 數學模型、程式自動判斷 | | 規則化程度 | 低，臨機應變 | 中高，規則固定 | 極高，完全程式化 | | 可否回測 | 幾乎無法 | 可用 Excel 或手工驗證 | 大規模電腦回測 | | 情緒影響 | 高 | 低 | 幾乎零 | | 技術需求 | 低 | 中 | 高 | | 彈性 | 高 | 中 | 低 | | 適合的人 | 觀察市場、反應快 | 紀律、避免情緒干擾 | 數據、程式、模型建構 |

八、建立交易系統的步驟

第 18 回合：VaR 投資組合計算與 CVaR 進階概念

文章來源：

    - [投資組合風險值 - TEJ台灣經濟新報](https://www.tejwin.com/insight/量化管理/投資組合風險值/) - [市場風險管理- VaR - 元富證券](https://www.masterlink.com.tw/esg-riskmarket/)

一、投資組合 VaR 計算實務

VaR 計算三種方法比較

| 方法 | 分配假設 | 非線性資產 | 厚尾問題 | 優點 | 缺點 | |------|----------|------------|----------|------|------| | **變異數-共變異數法** | 常態分配 | 不適合 | 無法處理 | 計算快、易解釋 | 不能處理非線性 | | **歷史模擬法** | 各種分配 | 可處理 | 可處理 | 無須假設分配 | 需大量歷史資料 | | **蒙地卡羅模擬法** | 各種分配 | 可處理 | 可處理 | 彈性高、可壓力測試 | 計算複雜、模型風險 |

投資組合 VaR 計算步驟

步驟一：計算個別標的 DEAR（每日盈餘風險） ``` DEAR = |-α| × σ × 價值 α = 信賴水準對應 z 值（99% → 2.33） σ = 報酬率標準差 ```

步驟二：計算標的間相關係數

    - 使用報酬率資料，而非價格資料 - 避免假性迴歸問題
步驟三：計算投資組合 VaR ``` 組合 VaR = √(Σ個別 VaR² + 2×Σ(相關係數× VaR_i × VaR_j)) ```

實例：四檔股票投資組合（儒鴻、台積電、國泰金、長榮）

    - 99% 絕對 VaR = $50,647.78 - 99% 相對 VaR = $52,205.86 - 意義：單一交易日最大損失有 99% 機率不超過 5.2 萬元
絕對 VaR vs 相對 VaR

| 類型 | 公式 | 說明 | |------|------|------| | **絕對 VaR** | (\|-α\| × σ – 均值) × 價值 | 相對於 0 的風險值 | | **相對 VaR** | \|−α\| × σ × 價值 | 相對於報酬率均值的風險值 |

變異數-共變異數法的限制

1. **無法衡量非線性資產** - 相關係數假設線性關係 - 選擇權等非線性商品不適用

2. **忽略厚尾問題** - 假設常態分配 - 實際市場有肥尾現象 - 可能低估風險值

解決方法：

    - 使用歷史模擬法或蒙地卡羅模擬 - 採用 CVaR（條件風險價值）補充
---

二、CVaR（條件風險價值）進階概念

CVaR 定義

    - CVaR = Conditional Value at Risk = 期望尾端損失（Expected Shortfall） - **核心問題**：當損失超過 VaR 時，預期會虧多少？ - CVaR 計算 VaR 點以下的平均損失
VaR vs CVaR 比較

| 特性 | VaR | CVaR | |------|-----|------| | **定義** | 特定信賴水準下的最大損失 | 超過 VaR 後的平均損失 | | **關注點** | 情況會變多糟糕？ | 最糟糕發生時預期虧多少？ | | **次可加性** | ❌ 不滿足 | ✅ 滿足 | | **尾部資訊** | 不考量 | 充分考量 | | **一致性風險測度** | 否 | 是 |

為什麼 CVaR 更好？

VaR 的缺陷：

    - 不考慮超過閾值後的損失嚴重度 - 不滿足次可加性（組合 VaR 可能大於個別 VaR 之和） - 無法正確反映分散風險的好處
CVaR 的優勢：
    - 考慮尾端損失的嚴重度 - 滿足次可加性，符合一致性風險測度 - 對估計誤差更敏感 - 更適合風險管理和資本配置
CVaR 公式概念

直觀理解：

    - CVaR 把 VaR 以下的損失全部加權平均 - 正態分配下：(CVaR - VaR) / VaR ≈ 15%（99% 信賴水準） - 峰度越高，差距越大
計算方法：
    - 把尾端分成 n 等分 - 計算對應 n-1 分的 VaR - 求期望值
實例（FRM 範例，n=5，常態分配）：

| 信賴區間 | 96% | 97% | 98% | 99% | |---------|-----|-----|-----|-----| | VaR | 1.7507 | 1.8808 | 2.0537 | 2.3263 | | **CVaR（平均）** | 2.003 |

---

三、風險值實務應用要點

VaR 使用時機

    - 日內風險監控 - 資本配置決策 - 風險限額設定 - 績效評估指標
VaR 使用限制
    - 歷史資料不代表未來 - 模型假設可能不符現實 - 極端事件無法預測 - 市場結構變化時失效
實務建議
    - VaR 搭配 CVaR 使用 - 定期回測驗證模型 - 壓力測試補充 VaR 不足 - 設定風險限額時保留緩衝
厚尾問題處理
    - 觀察報酬率分配 - 高峰度資產風險值需加權 - 長榮等航運股厚尾現象明顯，風險被低估
---

四、實戰應用步驟

投資組合 VaR 計算流程
