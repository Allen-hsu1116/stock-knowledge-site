K線型態

K線是技術分析的基礎語言，每一根K線都在說一個多空交戰的故事。

核心概念

每根K線由四個價格組成：開盤價（Open）、最高價（High）、最低價（Low）、收盤價（Close）。

解讀三要素：

  • 顏色：紅K（上漲）vs 黑K（下跌）
  • 實體長度：趨勢強度（長實體 = 趨勢明顯）
  • 影線長度：多空交戰激烈程度（長影線 = 阻力大）

單根K線型態

型態定義市場訊號
----------------------
大紅K收盤遠高於開盤，實體長，影線<20%多頭強勢，後市看漲
大黑K收盤遠低於開盤，實體長，影線<20%空頭強勢，後市看跌
十字線開盤≈收盤，長上下影線多空勢均力敵
鎚子線短實體+長下影線（無上影線）下檔有支撐
倒鎚線/墓碑線短實體+長上影線（無下影線）上檔有壓力
紡錘線短實體+上下影線市場猶豫

看漲反轉型態

型態組成訊號
------------------
錘形線單根K線，短實體長下影線，位於下降趨勢底部拋售壓力後買壓推升，看漲
多頭吞噬兩根K線，短紅K被長綠K完全吞沒買家明顯獲勝，看漲
早晨之星三根K線：長紅-短K-長綠賣壓消退，看漲
白色三兵三根連續長紅K，開盤收盤均高於前日買壓穩步上升，強烈看漲

看跌反轉型態

型態組成訊號
------------------
上吊線單根K線，短實體長下影線，位於上升趨勢頂部拋售明顯，多頭失控，看跌
射擊之星單根K線，短實體長上影線，位於上升趨勢頂部流星墜地，看跌
空頭吞噬兩根K線，短綠K被長紅K完全吞沒價格高峰，看跌
黃昏之星三根K線：長綠-短K-長紅上升趨勢反轉，看跌
三隻烏鴉三根連續長紅K，影線短或無賣方力量連續超越買方，看跌

K線組合型態

雙K型態：

型態組成訊號
------------------
多頭吞噬短紅K被長綠K完全吞沒買家明顯獲勝，看漲
空頭吞噬短綠K被長紅K完全吞沒價格高峰，看跌
烏雲罩頂長紅K後開高收跌，跌破紅K一半空頭訊號
貫穿型態長黑K後開低收紅，站回黑K一半多頭訊號
母子型態大K線包覆小K線趨勢暫止，等方向確認
三K型態：
型態組成訊號
------------------
晨星長黑K + 小K/十字 + 長紅K空轉多，強烈買進
夜星長紅K + 小K/十字 + 長黑K多轉空，強烈賣出
紅三兵三根連續紅K，收盤墊高趨勢延續，看漲
黑三兵三根連續黑K，收盤走低趨勢延續，看跌

實戰應用

K線影線比例判斷法

影線比例意義
---------------
影線佔整體 20% 以內阻力小，趨勢延續機率高
影線佔整體 20%-50%阻力不小，趨勢不明朗
影線佔整體 50% 以上阻力極大，多空激戰，應謹慎
實戰口訣：長實體 + 短影線 → 趨勢延續性高；短實體 + 長影線 → 趨勢延續性低

注意事項

  • 單根K線不構成訊號：需搭配前後K線一起判斷
  • 影線透露過程，實體透露結果：長影線 = 激烈交戰，長實體 = 明確趨勢
  • 型態位置很重要：同樣型態在高位和低位意義不同
  • 反轉型態要在趨勢末端才有效
  • 停損永遠設在型態極端點
  • 三K比雙K更可靠（多了確認K線）
