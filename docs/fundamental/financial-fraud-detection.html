假帳偵測心法

1. 先看老闆質押：> 80% = 小心 2. 再看應收帳款天數：> 90天 = 警訊 3. 看存貨天數：< 10天且與同業差距大 = 警訊 4. 看現金與借款：雙高 = 現金可能被限制 5. 看營業現金流：營收成長但現金流為負 = 怪 6. 看利多頻率：澄清「不予置評」太多 = 放消息炒股 7. 看借殼歷史：借殼後 3-4 年內營收暴增 = 警訊 8. 研究產業：產業差卻賺錢 = 可能有假
