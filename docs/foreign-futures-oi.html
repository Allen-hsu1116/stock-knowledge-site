外資期貨未平倉判讀

外資期貨未平倉判讀

大外資看中期方向，小外資看短線靈敏度，兩者同向偏多才是真多

核心概念

外資期貨未平倉口數是台股指數期貨市場最重要的籌碼指標之一。外資在台指期的多空部位，直接反映其對大盤方向的預期。

關鍵區分：大外資 vs 小外資

  • 大外資（五大特定人）：市場最大的五位法人部位，通常就是外資中最大的幾家。部位大，無法短線操作，期貨部位通常代表中期看法（數週到數月）
  • 小外資：外資總部位扣除五大外資後的淨額。部位較小，短線操作更靈活，對行情變化更敏感

計算方式：小外資 = 外資總未平倉淨額 − 五大外資淨額

實戰應用

大小外資同向判斷法

| 大外資 | 小外資 | 判斷 | |--------|--------|------| | 偏多 | 偏多 | 行情偏多看待，最可靠訊號 | | 偏多 | 偏空 | 中期偏多但短線有雜音，留意短期回調 | | 偏空 | 偏多 | 中期偏空但短線反彈，搶短要快 | | 偏空 | 偏空 | 行情偏空看待，最危險組合 |

核心原則：大、小外資沒有人翻空單之前，行情不太會有太大的翻轉變動。

小外資領先訊號

小外資比大外資更敏感，在一波下跌後小外資較快翻多，在一波上漲後也較快翻空。因此：

  • 想看短線變化 → 參考小外資部位
  • 想看中期趨勢 → 參考大外資部位
  • 最強訊號 = 小外資領先翻多 + 大外資維持偏多

結算日參考

外資台指期空單加碼 → 預期大盤下跌 → 結算日壓低結算機率較高 外資台指期多單加碼 → 預期大盤上漲 → 結算日拉高結算機率較高

但籌碼面與行情相關性不是 100%，需搭配其他因素綜合判斷。

常見誤區

1. 把外資當反指標：當外資空單增加但行情卻上漲時，不要急著認定外資是反指標。可能只是小外資已經翻多但大外資還沒跟上 2. 只看外資總部位：外資總部位包含幾十家機構，大小外資可能方向不同，總量會掩蓋真相 3. 忽略套利部位：外資可能同時持有現貨多單和期貨空單做套利，單看期貨空單會誤判

注意事項

  • 外資在各國交易策略多元，期貨部位不一定純粹是方向性看法
  • 五大特定人數據有落後性，盤後才公布
  • 結算日效應雖有參考價值，但不保證準確
  • 建議搭配 期貨未平倉量判讀、三大法人特性 綜合判斷
